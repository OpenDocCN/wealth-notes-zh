# “学生第一”没错，“学生第二”也对

240910

整理：公众号懒人搜索，懒人专属群分享

懒人微信：lazyhelper

今天是9月10日，是咱们国家的第40个教师节。没错，咱们中国的教师节从1985年设立，到2024年正好是第40年。在这里，先对各位人民教师说一声，教师节快乐。

今天，借着这个时机，我们就说说来自教育领域的思考。为什么要关注教育领域？一来，这往小了说，关系到自己的下一代，往大了说，关系到整个社会的未来。二来，正因为教育太重要，因此，教育领域的创新都非常谨慎。毕竟，这关系到很多人、很多家庭，乃至一个社会的未来。这也意味着，凡是在教育领域验证过的创新，大概率上都有一套经得起推敲的结论，以及相当扎实的方法。

因此，接下来要说的洞察，或许对你的行业也有启发。

咱们从今天的标题说起。学生第二与学生第一。这两个词，其实是两本书的名字。两本书的作者都是同一个人，咱们得到同学都熟悉，著名的教育家，李希贵校长。

这两本书的出版顺序，先是《学生第二》，后是《学生第一》。两本书都是李希贵校长多年从事教育改革的实践文集。换句话说，书里的观点和方法，都是经过验证，亲测有效的。

那么，这两个词又是什么意思呢？尤其是放在一起，好像有点相互矛盾。

这两个词针对的其实是不同的层面。

借用李希贵校长的话说，学生第二其实是管理学的概念，而学生第一是教育学的取向。毕竟，从事教育的人都知道学生在工作中的分量。但是，如何去实现学生为本？怎样去落实学生的中心地位？这就需要在学校的管理层面，将教师放在最重要的位置，尊重他们，相信他们，最大限度地发挥教师的积极性和创造性。

换句话说，管理工作中的学生第二，从根本上是为了更好地实现校园里的学生第一。

据说有这么个故事，不知道真假，但在教育界流传很广。说有位校长被人问，假如一个老师与一个学生发生争执，到校长室去找校长评理。请问你应该批评这个老师，还是批评学生？这位校长果断回答，批评老师，因为学生是学校的主人。

但是反过来说，假如学校只强调学生主体，老师又在什么位置上呢？事实上，假如老师状态不好，教学大概率上很难有好的效果。

你可以回想一下，所有让顾客满意的服务，从早年的海底捞，到现在的胖东来，这些公司都有一个特点，他们的服务员都很爱笑。世界上最成功的旅游管理公司之一，罗森伯斯国际集团，他们的老板叫豪尔·罗森伯斯。豪尔说，公司之所以能从一个小旅行社成长为现在的行业巨头，最关键的前提就是他们有一个经营理念，叫员工第一，顾客第二。

豪尔认为，有很多公司打着顾客至上的旗号，把所有的压力都转嫁给员工。那么员工只会带着情绪面对顾客。但最高级别的服务，应该是发自内心的。你就想象，假如我开了间餐厅，你是我的顾客，我嘴上说着顾客第一，我一切都为你着想。但同时又安排一个内心充满愤恨的厨师给你做饭，你觉得这菜吃着放心吗？

放在学校里也一样。你就想象一个场景，假如一位老师，刚刚因为分房子的事，跟学校的总务主任争得面红耳赤。5 分钟后，这位老师走进教室，开口第一句，不是讲课，而是讲起了生活中的不公，充满火药味。学生一边听课，一边战战兢兢，生怕把火惹到自己身上。尽管大家都尽量乖巧，但还是由几个倒霉的成了撒气桶。

再比如，假设一个老师，家里孩子生病，没人照看。她刚走进教室，又发现校领导正好来视察听课。她不会觉得压力就是动力，她只会觉得领导在乘人之危。哪还有什么好好上课的心情，你觉得学生的感受能好吗？

借用李希贵校长的话说，教育是塑造人的事业，塑造学生的美好人生是学校的追求。但是要知道，这一切都只能通过教师来完成。学生是在教师的耳濡目染间长大成人的，教师在课堂上、在校园里、在举手投足间，都潜移默化地影响着学生。学生亲其师，才能信其道。用幸福才能塑造幸福，用美好才能创造美好。

说白了，好的学校，应该是一个层层传导善意，层层放大善意的系统。学校盼着老师好，给老师提供一切支持。进而老师盼着学生好，把学生放在第一位。

当然，除了老师，学校还可以把后勤人员，把支持部门人员放在第一位，因为这些都是直接支持老师，或者直接服务于学生的人。换句话说，你可以把学校当成一个动力传导系统。最前面的是学生，离学生最近的是老师，老师身后是各类支持部门和学校管理层。学校注入的力量，需要经过年级、班级、老师的传导，最终才能作用到学生的身上。借用李希贵校长的话说，任何关爱都必须经过人的传递才显得真切、动人，谁都没办法改变这一点。

这也是为什么说，管理意义上的学生第二，是为了实现教育意义上的学生第一。

那么，具体到学生第一这四个字，又应该怎么做呢？我们应该怎样定义这个标准？什么才叫学生第一？是让学生考出最好的成绩？都考上好学校？

没那么复杂。这里说的学生第一，标准其实非常朴素，就是，你发自内心地，盼着学生变好。比如，李希贵校长曾经提出过一个教育目标，叫发现每一个学生的伟大之处，并让每一个学生走在通向伟大的路上。也就是，让每个学生长成属于他的最好的样子。

一旦这个目标定下来，行动就变得清晰了。

你看，要想让一个学生变好，首先他得对自己的职业有个预期吧？据说有相关组织做过调查，结果发现咱们中国人的职业幸福感是偏低的。假如不喜欢自己的工作，意味着一个人一辈子，大概有三分之二的时间不开心。

为什么会这样？因为绝大多数人直到大学毕业之前，对工作这个事都一无所知的。我还记得20年前，很多高校跟风开办生物工程专业，很多学生也抢着报名，但这个专业的就业在当时其实非常模糊，很多人毕业之后都去做了药代。

怎么解决这个问题？最好的办法之一，就是让学生尽早对职业产生感知，弄清楚自己将来想要干什么。

我们以李希贵校长推进教育改革的北京十一学校为例。首先，学校会定期发布校园机会榜，学校里的很多可以交给学生做的事，都会在这里公开招募。包括设计学校吉祥物、策划组织新年联欢活动、给学校拍摄宣传片，等等。这些事不仅能让学生感受到在学校的主场感，还能让他们摸索自己的兴趣。

其次，学校还会定期发布一个信息，叫做《你离这个职业有多远》。学校会把一个职业的大概情况、对从业者的要求、可能遇到的问题，以及未来的发展前景，都经过调研后做成海报，公布给学生。学生看了之后就大概知道，自己还应该补齐哪些能力，或者要不要换个方向。

最后，学校还会定期从校外邀请各行各业的专家，或者组织去各行各业的公司参观。为的也是让学生尽早找到适合自己的发展方向。

除了职业启蒙，学生第一还有另外一层含义，这就是，让学生在学校找到主场感，成为在学校说了算的人。

比如，食堂，这是学生反映问题的高发区。李希贵校长就在十一学校推行过一个设计，食堂有一套学生做主的淘汰机制。学校的几个食堂，承包给几个不同的餐饮公司。假如一个餐饮公司在这学期的学生投票里排名最后，就直接末位淘汰。但仍然有下一年的竞标资格。但假如中标了，第二年依然排在最后，那么这家公司在三年内都不许竞标。同时，在竞标环节，校方评审团里，必须有不少于三分之一的学生代表。你看，在食堂这个事上，是学生说了算吧？

再比如，十一学校还有个设计，叫与校长共进午餐。学校每天中午安排一位校领导值班，学生可以向值班领导提交申请。在午餐过程中，有的学生是为了反映问题，也有的是寻求建议。在这个过程中，学生会感觉到自己的想法被倾听，被重视。有了这个前提，学生就更有把想法变成现实的勇气。当然，学生假如不是为了交流想法，只是想单纯地跟校长吃个饭，也欢迎。

但不管是哪种情况，唯独有一个要求，午餐的费用必须 AA制，按照饭菜的成本付费。一来是防止有的学生纯粹为了蹭饭，二来还有一层潜台词，天下没有免费的午餐。你看，这个设计包含一层潜台词，就是学校并没有把学生当成小孩，而是当成平等的大人。

关于教育创新的细节，你可以去读读《学生第一》这本书，在得到就有电子书。尤其是身在教育一线的教师，更推荐你去看一看。

好，关于教育领域的洞察，咱们先说到这。最后，说说从中获得的一个启发。

什么是好的教育？很多人试图寻找这个问题的标准答案。但是，从今天的内容里，我们也许能看到一个线索。也许好的教育，恰恰是没有标准答案的教育。就像这两本书的书名，“学生第一”和“学生第二”，哪个是正确答案？都是正确答案。换句话说，兼容不同，也许比塑造统一标准更重要。
