

# 天津机场诡异事件，是怎么回事？

9月11日晚，天津滨海机场在没有任何异常气候的情况下，突然启动联合应急处置预案黄色等级响应，取消了所有航班的起降。这实际上就等同于关闭机场了，是前所未有的。更夸张的是，9月12日晚，天津滨海机场再次出现多架航班盘旋无法降落的情况。

那么，天津滨海机场到底发生什么了？

## 「1」
很多吃瓜群众，第一时间把这件事和2010年萧山机场的事情联系在了一起。两件事，同样是晚上，同样是关闭机场，同样是航班大面积延误，萧山机场的“长条状发光体”不明飞行物到现在也没结果，所以很多人猜测，这次是不是UFO又来了？

不过，这一次的解释是由于无人机入侵空域导致。如果这个解释是真，毫无疑问是近年来最恶劣的一起人为干扰民航安全事件，明显超出了社会治安的范畴，到了国家安全范畴。为什么这么说？因为性质严重，影响恶劣。

玩过无人机的都知道，像大疆这样的航拍无人机，在机场、油库等目标保护区内，是压根无法起飞的，人家大疆在系统里面就直接给你限制了。当然，有一些发烧友和黑飞能够破解，实现目视在禁飞区内飞行。问题在于，这种无人机一旦被发现，马上会被电子反制，予以俘获或击落。

但这次呢？无人机不仅在禁飞区飞起来了，而且无视各种反制手段，似乎有意戏耍机场一样，先是在19点左右出现，导致机场关闭，然后在22:20左右，机场宣布目视和雷达均未观测到无人机，按法规要求40分钟后接收降落飞机，但是到了23:00左右，机场又发现无人机活动！

连续两批无人机侵扰，吓得机场不敢再开放了，一直到第二天6点才重新起降飞机，关闭长达10小时。这事显然不简单。在无人机飞不起来的禁飞区，竟然飞起来了！无人机一块电池也就飞半个小时左右，这次呢？竟然持续了3个小时！还无视所有干扰设备！干扰枪都没打下来！且飞行高度不低，机场的一些低空物理驱离手段难以奏效。

最夸张的是，9月12日晚，无人机似乎觉得9月11日没玩过瘾，竟然在警方严阵以待的情况下又来了，这说明什么？说明这次压根就不是什么“黑飞误入禁飞区”，而是一次蓄意为之的挑衅行为。为了实现这一挑衅行为，这种无人机必然进行了改装，以应对机场的各种反制措施，至少有两种可能。

第一种是设定个方向，飞出去就不管了，因为不受遥控信号控制，自然不怕干扰枪，虽然这种淘宝无人机续航只有5分钟，但用来搞事情，足够了。

第二种性质就严重了，是俄乌战场上刚刚冒出来的线控无人机。咱们都知道，俄乌战场如今已经成了无人机的天下，各种FPV无人机满天飞，吓得双方都在给坦克焊铁栅栏，而双方的士兵甚至连渔网都用上了。而为了限制无人机的打击，双方都从某神秘国家采购了无人机干扰仪，有单兵背负式的，有干扰枪式的，还有干扰车，以往依靠2.4G/5.8G信道进行遥控和图传的无人机受到了很大威胁。

于是，现在俄乌战场冒出了一种线控无人机，简单来说就是拖着光纤飞行，把无线电遥控换成光纤有线遥控，这种无人机虽然飞行距离近，只有5公里远，但好处在于没有无线通信，不怕无线电监控和压制，免疫大部分干扰手段。观察这次天津机场事件，特警出动了雷达车和干扰车，都没压制侵入机场的无人机，不排除这种无人机是线控的。对于这种处心积虑的入侵行为，机场就必须谨慎应对，必须以最大的恶意来揣测其目的。

- 比如，无人机上是否携带了爆炸物？是否蓄意在航班起落期间实施攻击或撞击？
- 比如，无人机是否携带了三角锥型破胎钉？这种钉子很小，一架无人机可以携带很多，往跑道上一撒，飞机在降落时，轮胎很可能会爆掉，进而造成坠毁事故。
- 比如，无人机消失后，到底是飞走了，还是掉落在机场上了？如果飞走了还好说，如果掉在机场就麻烦了，有没有掉在跑道上？会不会被飞机发动机吸入？
- 等等等等。

“安全大于天”，毕竟涉及几十架飞机几千人，不出事则已，出事就是震惊世界，所以，机场第一时间停止了起降所有航班，不仅禁止任何航班降落可能的危险区域，而且还要对无人机入侵过的区域进行一次拉网式检查，排除任何风险因素！

当然，这样做安全了，但不可否认的是，损失也很大。从晚上7点多关闭机场开始，到晚上11点多，总共延误航班29架次，取消8架次，有32架次备降外场，有的航班在机场上方盘旋了几十圈还降不下来，共3000多名旅客受到影响。

要知道，一般飞机起飞时，不是装满油的，决定了备降机场只能在附近找，不能跑太远。这次主要是北京大兴/石家庄正定来接收航班。然而，备降机场的空域流量和停机位，也是有限的！很难短时间接收这么多飞机。结果到了22:40左右，石家庄正定机场饱和了，无法再接纳新的飞机，不得不启用济南遥墙机场与太原武宿机场。我们应该庆幸，附近的机场足够多，如果是机场密度低的地方，飞机没地方降，那可真的要出大事情的！

除此之外，这次事件给航空公司造成了极大损失。那些延误、取消和备降的航班，你要支付飞机在地面停留费用吧？你要支付机组成员的加班费吧？你要支付额外的燃油消耗吧？你要为旅客提供餐饮和住宿吧？这么算下来，每个航班都要大几十万。那么加起来就是至少3000万的损失。这笔钱该找谁索赔？你就算找到了无人机操作者，他恐怕也赔不起啊，只能自认倒霉。

更严重的是，这次事件给大家提了个醒，这么玩的成本低廉，门槛较低。由于门槛不高，咱们想一下，如果某些人掌握了无人机技术会怎样？一架自己改装的FPV无人机，成本不过几千块钱，大部分人都负担得起。门槛也很低，稍微培训一下就行。然而，这么简单的装备，危害却很大。

对速度极快的FPV无人机来说，在机场周边起飞，直冲民航客机的起飞和降落航线，重点撞击发动机、驾驶舱等位置，那立马就是一场群死群伤的大事件！而这个时间要多久呢？时机把握得好的话，30秒足矣！

30秒钟，机场就算发现了无人机，又能怎样？根本来不及做任何反应！等警方把无人机干扰枪调过来，恐怕飞机早就坠毁了。对于这种门槛极低的袭击，恐怕没有任何一个机场，能有效应对。

## 「2」
如此严重的事件，暴露出了什么问题？首先，就是机场的处置类似情况能力还不足。

说实话，中国是无人机大国，同样也是反无人机大国，随便找个警务展和防务展去看看，就会发现，做无人机反制设备的公司，足有上百家。但是呢？偏巧反无人机大国，自己的机场反无人机设备，配备却不足。特别是有了前一天教训、警方和机场都严阵以待的情况下，9月12日仍然被无人机再次入侵，最后啥也没找到，足以说明机场应对能力不足。

目前，中国机场的反无人机设施主要分为两层，外层是“防御式侦测预警系统”，在机场外侧20*50平方公里范围内部署电子围栏监控摄像头，图像与空管平台联动，确认无人机作业是否合法合规，并对非法作业人员进行抓捕、驱离等。

内层呢？是“无人机干扰系统”，一般是反无人机雷达作业车，它可实现8公里范围的无人机监测、发现和跟踪，然后通过压制无人机遥控信号和卫星定位信号的方法，迫使无人机原地降落或返航，信号压制距离可达1500米。

看到问题没？侦测预警系统的摄像头，并不是针对无人机的，而是针对操作无人机的人的，但在夜晚视野受限，很难有效发现黑飞飞手。而无人机干扰系统呢？只有“软杀伤”能力（而且这种能力很有限），缺乏“硬杀伤”能力，碰上不依赖遥控信号的无人机或者线控无人机，就抓瞎了。

所以，有必要给各大机场配上中国刚刚出口给沙特那种“寂静狩猎者”激光拦截系统了，发现黑飞无人机，直接一个照射，无人机就被烧毁了。而且因为其杀伤距离达4000米，足够覆盖机场的大部分。

第二，也证明了，无人机发现和反制之难。无人机亮相现代战争，是在阿塞拜疆和亚美尼亚的战争中，一开始大家觉得这两家是菜鸡互啄，连无人机都搞不定。可是后来，俄罗斯、北约、以色列乃至红海中的美国也拿无人机没招，这就让大家很诧异，这是为什么呢？

其实吧，主要是小无人机太难发现和摧毁了，目视几百米外就看不见，雷达也很难在低空大量杂波下，及时甄别出这么小这么慢的目标，特别是在城市中，几乎是无解，无人机在楼层间穿梭，非常难发现。你看俄罗斯作为五常之一，不还是被无人机炸了克里姆林宫的屋顶么？现在中国推出的Ku波段双极化有源相控阵雷达，倒是比较适合发现无人机，不过，就算你发现了，又能怎么样呢？

拿密集阵打倒是可以，但散落的炮弹会造成连带损伤。电子干扰？遇上做了防干扰处理（比如屏蔽罩或者线控）的无人机，一样抓瞎。至于GPS干扰和EMP？开玩笑，你敢在机场周边玩这个？恐怕无人机没掉下来，航班先掉下来了。

2022年的时候，华盛顿国家机场跑道飞入一架无人机，就导致90个航班延误，最后无人机也没抓住。看来，哪怕强如美国，遇上这种无人机入侵机场的事情，也没特别好的办法。

第三是处理之难。根据《通用航空飞行管制条例》有关规定，无人机“黑飞”，将面临行政处罚吊销执照、经营许可证、罚款等行政处罚。造成重大事故或者严重后果的，依照刑法关于重大飞行事故罪或者其他罪的规定，依法追究刑事责任。

但问题在于，这一规定，只能管住“老实人”，对于蓄意去闯禁飞区的人来说，他们敢黑飞，就证明他们觉得有把握逃脱处罚。比如，利用散件组装无人机，刷上一个开源的系统，根本没法实名制，再配上AI自主飞行控制软件和定时自毁装置以消灭作案痕迹的话，就算飞起来，又如何追查？

而且，在中国无人机圈子里面，有大量良莠不齐的人员，以挑战禁飞区作为装X的本钱，“黑飞”就像摩托党刻意炸街一样，意味着勇气以及不羁，你处罚越重，反而容易引发他们作死的欲望。对于这种人，你很难去一个个甄别，对于他们那种“打游击”的行为，更难以定点抓控。那么有关部门，就只能一刀切，但这又会伤害中国刚刚开放的低空飞行环境。

当然，如果只是飞友装X的炫耀行为，问题还不算太大。但这种试探了机场禁飞区的行为，无疑会给恐怖分子一种启发和鼓励，到时候恐怖分子在境外势力的支持和帮助下，随便找几个人，时不时在国内大型城市机场搞一下，那中国整个民航业都会有大麻烦。管吧？不好抓，如果为了规避风险而关闭机场，那整个航空就瘫痪了；不管吧？谁知道这些无人机到底会不会撞飞机？

这样一来，机场就陷入了两难境地，不论怎么做，都不是最优解。

所以，对天津这件事来说，不能因为没有造成事故，就掉以轻心，反而应该吸取教训，升级为威胁国家安全的 行为，一方面加强机场反无人机能 力，另一方面集中力量予以侦破，杀 鸡儆猴，防止更多人效仿。如果不从重处理的话，时不时给来一下，大家都没办法放心坐飞机了。
