# 如果财富归零，我会去哪里寻找好的赚钱机会？

## #小灯塔# 04

今天这篇，讲具体方法，我会告诉你，如果财富归零，要去寻找好的赚钱机会，我会去哪里找？

在开始之前，先讲几条核心底层逻辑，希望我们能够有共识，因为这些逻辑影响着如何选择机会。

- 正确认知自己，你只是想赚钱还是想做一个伟大的产品改变世界。目标不一样，做事方法不一样。对于大多数圈友来说，多赚点钱改变个人和家庭生活，应当是主要目标。

- 想赚更多钱，一点都不丢人，在生财有术不想赚更多钱才丢人。抛弃掉大多数中国人骨子里的赚钱羞耻感，是赚到钱的第一步。

- 作为大多数想赚钱的普通人，10人以下团队，包括个体，核心工作都应该是搞流量，搞流量，搞流量，重要的事情说三遍。

赚钱就是搞流量，搞流量就是赚钱。当你能力和资金不足，却开始搞自己的产品，大概率走向了失败和亏钱。

- 在目前这个时代，搞流量基本等于在大平台规模化搞内容（包括短视频和直播），搞内容的核心是找到爆款然后微调，爆过的内容一定会再爆。

- 找一个可以卖的产品很容易，做出一个产品也很容易，最难的是把产品卖出去，也就是跑通项目：流量-转化闭环验证。

起步先把最不确定最难干的事情验证好：先搞流量，再卖别人的产品，最后才是自己做产品。什么时候开始自己做产品？靠流量赚到第一桶金后，比如 100 万。

- 我寻找赚钱机会，一定不会直接做从生财有术精华帖或者风向标里看到的项目，大家都盯着的项目肯定不是好项目。

但我一定会从某一个风向标或者精华帖里去观察背后趋势，然后深入挖掘，充分拆解，举一反三，找到一个潜在市场不小、竞争不高的机会。

- 在内容平台，每一个持续更新的账号等于一个赚钱的项目，每一个投放的广告等于一个正在放大的赚钱项目。仔细研究，ta 是怎么赚钱的？

- 从零碎的情报线索，比如风向标，到一个清晰的赚钱机会，中间是深入拆解和仔细算账。
