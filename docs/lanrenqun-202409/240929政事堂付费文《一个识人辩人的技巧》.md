

# 一个识人辩人的技巧
## 240929 政事堂 顾子明说

整理：公众号懒人搜索，懒人专属群独享

懒人备注：小懒一直觉得，善良是一种能力，更是一种选择

![](bbox=[58, 211, 824, 334])

过去一周，罗翔成为了键政圈围殴的对象，各路大佬纷纷下场对他围剿。

作为一名资深的键政爱好者，看着视频中罗翔老师一本正经的讲历史政治，想象着罗翔老师看着我给他讲刑法的心中暗笑。

对于罗翔老师的内容本身，纯属政治表态，政事堂没啥想说的，也不想参与论战。但是，对他的围剿的这事儿，底层内核却是非常值得说的。

借此，可以帮助政事堂的年轻读者，或者正在教育培养孩子的中年父母，get人生旅途中一个重要的识人辩人技巧。

懂了之后，在人生旅途的重大选择时选对，可以受用一生。

但是借助罗翔事件，把这个事儿写透，既得罪人，又会挨骂，所以，这篇文章设置为收费。而且，这篇文章本身不是讨论罗翔和罗翔视频，想看相关辩论或者我态度的请绕路。

尤其是，如果你不是，或者不曾是一个善良、阳光或者积极向上的人，也请千万不要往下点开，看了这篇文章甚至有可能对你造成伤害。

在东亚这个社会上，努力，可以在一定的范围内提升自己的生活水平，但是普通人总会被一些隐性的门槛压着无法突破，若要强行渡劫，往往要冒着极大的风险，并付出极其惨痛的代价。

而现实世界上，大部分成功突破阶层和渡劫的人，往往并不是靠着冒险渡劫，而是在一些段位远高于自己的伯乐庇护下，完成了某个或者某几个阶段的跨越。

以受众面最广的《潜伏》为例，余则成出场时就是个毛头小子，如果没有吕宗方、秋掌柜、吴站长，三位老领导接力棒式的庇护与栽培，在复杂的政治斗争中，早就不知道死了多少回了，根本没机会积累经验，逐步成长起来。

绝大多数成功的人，都在自己的人生旅途中，遇到过一个甚至多个这样的伯乐，有过的成功人士，可以回忆一下。

但这种伯乐是可遇而不可求的，因为需要他同时满足两个互斥的属性：
- 精通游戏规则
- 善良

精通规则的人很难善良，善良的人也很难精通规则。若是善良，还能精通游戏规则，说明这个人非常之强大，可以强大到凭借自身的能量，在一定范围内忽视、影响甚至调整规则。这类人在市面上是并不常见，生活中一旦遇到这种伯乐，就千万不要错过。

不同于社会上常见的利益交换甚至魔鬼契约，这些历经沧海的伯乐们，他们不仅能够帮助年轻人，而且强大如他们，对年轻人也无所图。

他们甚至常常会情不自禁的将自己的经历投射到朝气蓬勃的年轻人身上，就像当年拉自己一把的老伯乐，竭力并无偿的帮助年轻人渡过自己年轻时曾遇到的坎坷，不希望他们的翻车。

类似于余则成后期已经暴露了身份，但站长仍然帮他遮掩，甚至不惜代价与后果，硬塞给他一张用钱买不到的机票，是因为站长从余则成坚定的眼神中，看到了年轻时拥有同样革命信念的自己。

早已透析了人性与政治的站长很清楚的知道，余则成如果留在大陆，上限就是军统六哥郑耀先，便替他做了人生中最重要的选择。

- 潜伏中，吴敬中对余则成如此；
- 大明王朝中，胡宗宪对高翰文如此；
- 北平无战事中，何其沧对梁经纶如此；
- 人民的名义中，高育良对祁同伟亦如此。

只不过人民的名义是现代反腐剧，为了符合主流价值观，祁同伟必须被剧情杀。

相反，无论是吴刚饰演的李达康还是陆桥山，无论你是他们的同事还是下属，都不要高兴，这种人的本质都是在利用你，你拿到的晋升与财富，都来源于跟魔鬼签订的契约。

那位被李达康器重的丁义珍，也许直到他身死异国他乡时的那一刻，才会想明白，所有命运的馈赠，都在暗中标好了价格。

至于本次罗翔事件本身，没什么好说的，他讲的历史属实不靠谱，是一个情绪化的产物。

但罗翔视频前后所引发的大争论，是一个非常好的观察标的与试金石。

35岁以下，世界观未成形的群体，怎么表态都是正常的，这个不用关注。
重点是看，35岁以上，那些事业有成的企业家或者级别较高的领导。

他们支持哪一方，都是价值观的体现，可以忽略。

但若是公开或隐晦的表达了类似于罗翔这样“心软”、“不忍”的观点，或者有些罗翔式的“抑郁”，就要重点关注，说明精通游戏规则的他们，还拥有善良的特质。

三观归三观，但这种考验一个成年人良心的公开事件并不多，本次罗翔事件前后，给了我们一个很好观察领导内心的契机。

在实权人物里面，还能做到善良的，寥寥无几，但凡拥有善良的，都是那种内心极其强大，向外源源不断输出能量的真牛人。

如果你身边有这样的领导，而你又恰巧是一个积极向上的小朋友，那么一定要紧紧的跟着他，不要藏什么心眼。

他未来给你的，大概率是你穷尽一生的努力都无法拿到的。借用尼采的一句话：

灵魂高级的人，或多或少都有一些抑郁，他们抑郁的本质，是洞悉一切后的善良。
