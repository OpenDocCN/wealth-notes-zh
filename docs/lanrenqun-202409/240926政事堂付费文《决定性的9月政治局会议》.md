

# 决定性的9月政治局会议

240926 顾子明说

整理：公众号懒人搜索，懒人专属群独享

懒人微信：lazyhelper

重磅消息：新华社北京9月26日电中共中央政治局9月26日召开会议，分析研究当前经济形势，部署下一步经济工作。

政治局一般只有4月，7月，12月召开经济主体的会议，只有类似中美贸易战或者新冠疫情这种级别，外部或者内部环境出现根本性大变化的时候，才会召开此类会议。

跟几天前金融三巨头临时加塞的会议类似，必然是要放大招的，所以，本次会议将是今年迄今为止最重要的一场政治局会议。

先看对照今年前两次会议，政治局视角的外部环境：

今年4月会议的最大困扰是“外部环境复杂性、严峻性、不确定性明显上升”。

今年7月会议的最大困扰是“当前外部环境变化带来的不利影响增多”。

今年9月会议，没有一个字的外部的困扰，变成了内部“。。等有利条件并未改变”。

这个U转的大背景，是原本预计今年一季度进入的美联储降息周期，终于在9月开启，而且是大力度的50个基点，彻底解决了中国最为棘手的外部环境问题，可以放手一博。

随后，中美经济工作组在北京取得了“富有成功”的会晤，两国关系出现实质性缓和。

这便是本次政治局会议临时加塞的时代大背景。

所以，大气候变化了，政治局的风格也出现了U转的变化。

今年4月的说法，是要增强做好经济工作的信心。

今年7月的说法，是稳定市场预期，增强社会信心，坚定发展信心，唱响中国经济光明论。

今年9月的说法，是正视困难，主动作为，有效落实存量政策，加力推出增量政策。

也就是说，整个中国经济从二季度和三季度不断加码，用嘴和“集结号”唱响光明，进入四季度，变成了挽起袖子干活，开始真正出台政策了。

所以，相较于4月会议的第一任务是鼓励民众扩大内需，7月会议的第一任务是鼓励企业家，都是“县大队区小队”级的防御性质动作，用来拼消耗，9月会议的第一任务，变成了国家队的主动出击。

并本次在第一任务中明确提出了：

- 1、要促进房地产市场止跌回稳，加大“白名单”项目贷款投放力度，降低存量房贷利率。
- 2、努力提振资本市场，大力引导中长期资金入市，打通社保、保险、理财等资金入市堵点。

房地产与资本市场，是中国最核心的两大资产，也就是说，9月的政治局会议，正式吹响了国家队主力反击的号角，中国经济完成了触底，开始进入反弹阶段。

最后，去年年底说的“三个区分开来”，由于外部环境变化，在4月和7月都没有再提，但本次会议重新提出，并要求大家“干字当头”。

对此，千万不要怀疑，本质上，这跟雅尔塔会议后，我党1945年4月20日会议的性质是一模一样的，随着外部大环境的改变，中国终于可以彻底的解放了干部与企业家的手脚，让他们放手去拼搏和扩张。

会议要求，各地区各部门要认真贯彻落实党中央决策部署，干字当头、众志成城，充分激发全社会推动高质量发展的积极性主动性创造性，推动经济持续回升向好。广大党员、干部要勇于担责、敢于创新，在攻坚克难中长本领、出业绩。要树好选人用人风向标，认真落实“三个区分开来”，为担当者担当、为干事者撑腰。要支持经济大省挑大梁，更好发挥带动和支柱作用。
