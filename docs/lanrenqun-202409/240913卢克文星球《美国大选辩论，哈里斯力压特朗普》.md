

# 美国大选辩论，哈里斯力压特朗普

任何辩论都需要遵循一个原则，辩论的目的从来都不是说服对手，辩论的目的是说服观众。

当地时间9月10日晚，哈里斯和特朗普在宾夕法尼亚州费城首次进行电视辩论。纵观整场辩论，特朗普完全沉浸在其曾经的逻辑中。查理芒格说过，“对于手里只有锤子的人看来，一切问题都是钉子。”特朗普一遍遍重复着非法移民、贸易战等老话题，并希望他能解决犯罪率、俄乌冲突、通货膨胀等新问题。

特朗普在本场辩论的观点，基本上是对2016年竞选纲领的重复，仿佛他希望将过去的内容再重复一遍，并得到相同的结果。在华盛顿邮报的调查中，90%的美国民众在辩论前就知道特朗普会说什么，简单来讲，2016年特朗普的观点和政策可谓石破天惊，然而在经历了魔幻般的八年之后，特朗普的政策已经无法让民众感到耳目一新了。非法移民、减税、贸易战都不过是陈词滥调。

当然特朗普也不是没有高光时刻。当哈里斯提出特朗普必须为骚乱分子冲进了国会山而负责的时候，特朗普立刻反怼：“你们让更多非法移民冲进了美国，你们也应该为此负责。”

但总体来讲，哈里斯的表现则更为犀利，在辩论技巧的应用上更是炉火纯青。他们两人的辩论给我一种年轻力壮的职业政客偷袭一个退休老头的感觉。其中最为绝妙的金句出现在非法移民的指控上：特朗普指控哈里斯放了大量移民进来增加了美国犯罪率，哈里斯反怼“你也是个罪犯”，随后特朗普暴怒，为自己不是罪犯而辩解，完全落入了哈里斯的圈套。

## 经济问题

在辩论论点中，经济问题是一个核心，其中双方前10分钟的交战重点，出现在减税之上。

哈里斯提出了给中小企业减税的政策，并明确攻击特朗普曾经在2018年大规模给富人减税。

哈里斯说道：“高盛表示，唐纳德·特朗普的计划将使经济陷入最糟糕的境地。沃顿商学院表示，唐纳德·特朗普的计划实际上会让赤字激增。16位诺贝尔奖获得者将他的经济计划描述为会加剧通货膨胀，到明年年中会引发经济衰退。”

这里就是哈里斯最精妙的一点，哈里斯几乎所有的反驳和论点内容都会指明出处，而且往往来自权威专家，这会增加她观点的可信度。相反，特朗普几乎所有的反驳都没有出处，给人一种“我认为”的感觉，即“我不要你觉得，我要我觉得”。在经历了专业学者从不同角度对特朗普经济计划的评价之后，哈里斯以朋友的视角诚恳地告诉大家：“我想让你知道，特朗普实际上对你没有计划，因为他更感兴趣的是保护自己，而不是照顾你。”

当特朗普反驳的时候，主持人拉了一场偏架，他试图将今天美国的通货膨胀和高昂的关税挂钩，而之所以征收关税是因为特朗普的贸易战。果不其然，特朗普直接炸开，口无遮拦地批评中国，并强调关税是为了美国攫取利益。而这明显是中了哈里斯的计。

因为早在2019年，美国国家经济研究局在调查后就得出结论，关税增加的成本几乎100%落在了美国企业和消费者身上。哥伦比亚大学教授戴维·E·温斯坦在抽查了中国海关和美国海关数据后也认为，中国并没有因为贸易战而受到影响，反而是大量美国企业因此增加成本。

美联储两位经济学家亚伦·弗拉恩和贾斯汀·皮尔斯于2019年12月底发布的一项研究佐证了以上观点，他们写道：中国可以很容易地用巴西大豆替换美国大豆，但对于美国企业来讲，中国生产的电子产品、专业消费品以及机械零部件等是难以替代的。

因为哪怕加上关税，中国产品也具备市场竞争力，这种情况下，相关企业只能选择支付更高的成本继续从中国进口，而关税增加的成本自然被美国企业所承担了。

可以说，哈里斯在这里设计了一个大陷阱，而特朗普毫不留情地钻进去了，特朗普越是强调关税的重要性，这个陷阱就越致命。

来自宾夕法尼亚州的观众亚伦在看完辩论后提到：“哈里斯完全回避了美国人民现在的生活是否比四年前更好这个问题，也没有谈及过去几年的通货膨胀问题。”

事实上，哈里斯的团队知道这个问题非常致命。所以下面哈里斯只需要重复关税和商品价格上涨的必然联系，就可以将今天美国通货膨胀这口大锅，从拜登屁股上挪走，直接扣在特朗普头上。

但鉴于关税是美国与中国进行谈判的重要手段，这张牌不到万不得已，哈里斯也不会用。

## 移民问题

在关于移民问题的辩论中，特朗普试图让全体美国人相信，那些不远万里来到美国的非法移民都是一群罪犯。他们千辛万苦来到美国就是为了摧毁美国人的美好生活。他还创造性地指出，全球的犯罪率都在下降，因为那些罪犯都到美国来了，所以美国犯罪率正在上升。

特朗普话音刚落，主持人连忙打断，提醒他：根据联邦调查局的数据，犯罪率没有上升而是下降。特朗普云淡风轻地表示，联邦调查局在撒谎。果然高手过招就是这么朴实无华，且枯燥。

这个内容上我觉得特朗普没有说错，根据美国智库的研究和调查，确实有大量帮派分子通过各种渠道来到美国。这里我们需要提醒大家，南美黑帮和一般的黑社会还不太一样，南美黑帮某种程度上是当地治理体系的一部分。

在南美，一个黑帮对于当地社区的影响力是巨大的。一些黑帮会支付教师工资并安装小型电网以此便利当地居民的生活，同时会资助社区厨房并为儿童购买圣诞礼物。这种强社会链接往往会被带到美国，让南美移民不自觉地向“组织”靠拢。

而几乎所有南美黑帮，最大的利润来源都是前往美国贩毒。在这种情况下，黑帮非常有动力，利用贩毒网络将其骨干成员送往美国站稳市场。对于南美地区的贫困人口而言，在美国犯罪是有吸引力的，毕竟最地狱的笑话是，美国的监狱条件都比他们家好。坐牢对于这些人而言根本不算什么。

同时，因为从南美来到美国的成本过高，很多本来不是帮派成员的南美平民，为了赚取路费或者为了还清欠款，不得不加入帮派，融入他们的事业之中。所以我并不认为特朗普的言论有什么问题。

看完这段后，来自亚利桑那州的观众艾伦表示：“很多调查都说犯罪率下降了，但我觉得包括我自己在内的每个人都看到犯罪率在上升。”

其实这也是美国群众的一些看法，即犯罪率数据在下降，但个人生活中并不能感觉到，也就是说统计数据与实际体验出现了较大差距。

这里我想说大家没有认识到，哈里斯及其代表的民主党只是让美国的犯罪率下降了，至于美国犯罪下降没有，他们并不在乎。而犯罪率是一种统计数字，它受限于统计手段，比如随着哈里斯将纽约州重罪金额提高到950美元，当年纽约州的重罪率直线下降。

现在民主党一边放移民进入美国，一边又大幅度降低重罪标准并对一些法律进行修改，将贩毒逐渐轻罪化。这样一来，这些进入美国的南美移民及其背后的家庭或者帮派，就成为了民主党的忠实票仓。

随后特朗普在辩论中指出，我们的选举很糟糕，很多非法移民进来，哈里斯们试图让他们投票。他们甚至不会说英语，他们甚至不知道自己在哪个国家。

## 俄乌与巴以

在巴以和俄乌问题上，双方是各有各的打算。站在民主党的立场上，乌克兰必须赢得战争胜利，而以色列需要停火。虽然哈里斯认为以色列有权反击，但其对于以色列不顾平民伤亡的行为始终是不认可的。而特朗普则认为乌克兰那边可有可无，美国没必要支援他们，反而是以色列必须存在。

特朗普率先在这个问题上开火，直接表示，他撕毁伊核协议的举动摧毁了伊朗的经济，让伊朗无力支持真主党和哈马斯，而民主党对伊朗的缓和政策帮助伊朗重整了经济，让其有能力支持上述团体并发动袭击。简单来讲，今天中东问题的核心是民主党政府对于伊朗的放纵。

天地良心，本来奥巴马政府都谈妥了伊核协议，要不是特朗普直接炸死苏莱曼尼，让伊朗内部温和派彻底完蛋，伊朗保守派也不会上台，今天中东局面根本不会这么复杂。现在特朗普反手把锅扣在哈里斯身上，毕竟阿克萨洪水确实是在拜登任期发生的，特朗普也指出，你看我的任期就没有这些问题。

哈里斯不愧是辩论老手，根本不和特朗普讨论伊朗问题，反而谈及俄乌。她聊到特朗普和朝鲜领导人和解，而今天朝鲜领导人给了俄罗斯大量援助。所谓特朗普结束俄乌冲突的计划就是让乌克兰投降，而乌克兰的失败会导致一系列无法预测的连锁反应。

这里不得不说哈里斯真的很有水平，你说我放松伊朗的政策导致了阿克萨洪水的出现，我说你对朝鲜的放纵，增长了俄罗斯的信心和实力。看起来，说的是两件事情，其实根子上是一件事情。指出特朗普访问朝鲜，本身就是对特朗普指控的回应。

但从双方的辩论中我还是注意到一些细节，那就是双方都认可以色列必须存在。无论哈里斯多么厌恶以色列对于平民的袭击，但哈里斯不敢否认以色列反击的权利。而特朗普则敢明确以挽救性命的名义，鼓动乌克兰停战，哪怕现在停战会让乌克兰损失大量领土。同样哈里斯在援助乌克兰的过程中，必须强调乌克兰对于美国整体的利益多么重要。

简单来讲，援助以色列、支持以色列不需要理由，而支持乌克兰则需要理由。双方在美国政坛中孰轻孰重，一眼可知。

## 结论

辩论结束后，竞选双方都宣布自己取得了一场辉煌的胜利，而SSRS为CNN进行的一项快速民意调查显示，哈里斯无疑是本次辩论的胜利者。在辩论开始之前，各有50%的观众分别认为特朗普与哈里斯表现更好。等到辩论结束后，63%的人认为哈里斯表现更好，认为特朗普更好的人只有37%。事实上辩论中，哈里斯利用一切技巧成功激怒了特朗普，包括表情上的无视、语言上的攻击，以及对特朗普过往黑历史的嘲讽。这成功让一部分观众对特朗普产生了厌恶。

而在辩论过程中，哈里斯提出了一系列减税政策，以及针对医疗的改革方案，让普通美国群众感同身受。在辩论开始前，43%的选民认为特朗普更了解当今普通美国人面临的问题，这个数据远超哈里斯。辩论结束后，数据出现了逆转，44%的选民认为哈里斯更了解他们的问题。与此同时，在堕胎问题进行一系列辩论之后，不少女性开始站位哈里斯。

从以上数据我们可以得出结论，哈里斯在本轮辩论中取得了无可争议的胜利，但这场胜利对于选举的影响并没有想象中的那么大。

令人难以理解的是，虽然民调普遍认为哈里斯表现更好，但82%的美国群众不会因为辩论影响他们对总统的选择，只有4%的民众承认，经过本轮辩论，他们投票意向发生了改变。

这就说明了一个更为严重的问题，那就是美国两派之间的界限已经非常清晰了。哪怕特朗普表现如此令人失望，真正改变投票意向的人数依旧是少数。这就更证明了一点，即无论你的表现如何，你无法说服对方，你只能说服那些中间派，而在现在的选举中，美国中间派越来越少，双方都越来越难以动摇。

正如《纽约时报》的评论，当一群美国人想从另外一群美国人手中拯救祖国的时候，灾难就产生了。随着中间派的不断减少，这一天正在逐渐到来。
