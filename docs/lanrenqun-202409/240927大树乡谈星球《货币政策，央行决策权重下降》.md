

# 货币政策，央行决策权重下降

最近金融三巨头说了非常多很有价值的利好政策，尤其是把此前释放的信息和这次开会释放的对比，能够看到政策决策的逻辑正在发生变化，尤其体现在货币政策。

会议信息量很大，涉及到降准、降息、针对非银机构创设互换便利工具，还有回购和再贷款，当然还有降低房地产存量贷款。会上说平均降0.5%，参考上一轮实际肯定要小于0.5%，小镇月初曾经有个判断，说未来最多两个月内落地、平均降0.3%—0.4%，不会打开跨行转按揭，准确度还可以。

今天也不准备挨个做简评了，这几天沉淀下再好好谈，今天重点谈货币政策逻辑变化。

上周美联储降息后，国内一直传闻要降息20BP，但周五央行没有降息，市场传闻落空了，于是上周末很多相关人士都在质问为什么央行还不降息，也有人反过来分析，从过去找各种央行发声来证明央行早就说了不会降息，批判市场传闻是错误的。

毕竟本月初国新办例行发布会，陆副行长和货政一把手邹司参会，专门回应了降息降准，表态就非常保守。在回答记者问题的时候，说政策调整要继续观察经济走势；另外潘行在陆家嘴论坛和接受采访时，谈到货币政策，也强调要处理好短期和长期、实体经济和金融机构、内部和外部三者的关系，明确说保持政策定力、不大放水，不大收。

而货政一把手邹司在回应降准降息可能性时，也谈到继续降低面临一些约束，重点是银行净息差收缩的问题，这跟潘行强调平衡支持实体经济和金融机构的健康性是一致的，言下之意就是不能总想着降息降准对经济的积极影响，也得考虑金融机构能不能支持。

邹司还明确说“法定存款准备金率是我们供给长期流动性的一项工具”，既然是供给长期，言下之意就是不能为了应对短期刺激就把降准用了，考虑到今年已经降了一次了，这种表态显然倾向于暂不降准。

然而，今天的会，实际做的跟前面反复强调的大相径庭。存款准备金率直接降了0.5%，一下子为市场提供流动性1万亿元，这是年内第二次降低存款准备金率，而且潘行还表示年底前还有可能进一步降准 0.25%—0.5%。

降息也一样，直接降了 20 个 BP。虽然相比美联储动辄 25、50、75 还是更保守，但是要注意中美情况不同，央行过去降息也就是 5BP、10BP，非常珍惜降息空间，相比之下，这次 20BP 已经很猛了，这就意味着未来如果继续降息，可能不会再是过去 5、10 的微调，力度会更大。

反而是降低房贷存量利率，这个并不出奇，小镇这个非金融领域的都能判断的差不多。

这背后的根本逻辑是央行对于货币政策的影响力正在下降，越来越从货币政策的决策者向执行者转变，因此央行自己也很难对货币政策进行有效预判，公开表达的观点更多是央行自己的判断，并不代表真正要执行的。

未来，货币政策的决定权在更高层级，而更高层级考虑的是整体而非金融领域，金融也要服务于大局。

这就导致央行其实很难做预期管理，但这也意味着，未来货币政策会更加灵活、更有力度，只要现实需要，过去的条条框框都可以打破，货币政策与财政政策以及其他政策的联系会更加紧密。

还有一个隐性的作用，那就是大大加大了国外机构对中国货币政策的判断难度，很难再通过分析公开信息以及部分特殊渠道去推测。

因此，未来思考预判货币政策，需要考虑更整体、更高层的维度，要跳出金融、跳出央行，考虑更高层级怎么想。

# 镇长本人回复：

从今天的会，看出更高层级的意志力影响力进一步加强。实体经济、先进制造、高质量发展对所有政策的影响权重更高了。

参考超长期特别国债也用于短期刺激，结合今天的会，这也说明更高层级更加重视应对短期，未来逆周期调节会成为重点，跨周期调节下调。
