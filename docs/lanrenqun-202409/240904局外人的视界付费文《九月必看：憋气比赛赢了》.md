# 前言

持股，持股，尽可能的持有筹码！如果你相信中国实力，做多大A，转折点已经来了。

从去年三月份硅谷银行暴雷开始，全世界都在猜美联储啥时候降息，然而美联储的嘴巴比死鸭子还硬，坚决不肯降息，但又不能不救美国的银行，于是美帝搞了一个假紧缩、真放水的玩法。

允许美国和盟友的银行向美联储用债券做抵押借款。只要流动性还在，炒高资产价格，无非是损害一下实体经济的利益，金融机构们照样能大发其财。这就是为啥美国搞紧缩，疯狂加息非但没有搞出金融危机，反倒把美国的资产价格给提起来的原因。

但大家都知道，出来混的，迟早要还。借的钱是不可以赖账的，不仅要还本金，还得连带加个高额的利息。美联储现在的利率是 5.25%—5.5% 之间，现在你告诉我，到底干啥才能保证稳定的有这笔收入？

有人说，可以炒股。

呵呵，笑死了。慢慢的出货还则罢了，一口气几万亿美元都要出来，就问你美股有可能挺得住吗？别忘了，所谓的趋势都是大资金带起来的，怎么，你一个小散真以为几千几万块钱就能左右大盘走势了？

一旦大资金都迫于还款的需要从股市上抽离，后果只有一个：产生踩踏，多杀多，杀得比 08 年次贷危机还要惨。

所以市场上的资金认为，老美会让美国资产挺住，在今年年初就宣布降息，好让那帮接了钱去搞投机的银行们胜利大逃亡。但每次老美有点降息的苗头，市场上立刻就会传出各种看空中国经济的声音。这次加息本来就是为了要对付东方大国，美联储一看到胜利在望，立刻就会咬牙坚持住。

在四月底，国际资金率先憋不住了，先到港股杀了个回马枪。但事情还是像之前那样，每次美联储先放了点软话，立刻就会被神秘的力量给拉到鹰派调性上。四月底到五月底的这波行情，大A是涨不过港股的，国际资金东流当然首选在HK探风，本身港股市场容量也小，因此拉起来也快。

五月底，美国又开始放狠话，直接改了就业和CPI数据，市场一看立刻抽身观望。随后六月、七月、八月，八月就是前几天，大A都有几根阳线，有人认为这是主力在喊话，让老乡别走。

呵呵，我以为你们想复杂了。

我之前做过很多妖股，在起来之前，莫名奇妙的突然大涨，然后又迅速回落。一次次的玩，玩到后面大家都不玩了，量都没了，主力一看试盘差不多了，外部环境也好，于是一口气拉起来，好多玩高抛低吸的就此被洗出去了。

昨天我写《人性和反人性》，有人留言冷嘲热讽，说不会有牛市的，所以我对市场看法就是瞎扯。对于这种人，你就别跟他辩论，他是大聪明，涨到天上去看多，跌到谷底看空，你全对。

现在全世界股市都在高位，难道世界经济到处一片大好，只有中国差？我之前很多次在文章里反复说，美国这波加息跟遏制通胀没有半毛钱关系，仅仅是因为俄乌战争开打以后，美国想要针对东大挑起一波金融战。

大A是从 2021 年年底开始指数一路向下的。我们赶在美联储加息之前，先捅破了房地产泡沫，再把股市给打下来了。当然了，过去的两年多里，虽然大盘比较怂，但结构性行情却酝酿了不少机会。股市、楼市不好，但债市却异常火爆，等于是用债市去留住那些爱折腾的热钱，让他们不至于流出太快。

8月份央妈直接出手，把债市给干熄火了。这就是一个信号。

8月份美联储自己跳出来主动说过去几年自己在非农数据上大肆造假，美国经济太好，因此要维持高利率的理由根本说不通。自己打自己嘴巴子这么响，又不是有病，当然是有目的，目的就是为了给自己降息找个台阶下。

美联储主席鲍威尔公开表示，货币政策要转向了，那么九月份开始降息已经成了板上钉钉的事，无非是降息多少、降息的速度有多快。这里还有个美国大选在其中搅局。

要知道美国最花钱的不是总统PK大战，而是争夺国会山上的席位。谁知道趴在帝国身上吸血的好日子不多了，因此谁也不愿意把最后的机会让给别人，能多吃一口是一口，所以现在美国政坛斗争越发激烈：以前是皇帝轮流做，明年到我家，大家都还能维持个表面的和气，毕竟日子还长；但现在谁也不看长久，只看眼前了。

特朗普之前威胁过鲍威尔，让他别在11月份之前降息，否则就收拾他。现在鲍威尔早早的这么喊话了，大概率就是美国的金融资本集团已经打算集体站队哈里斯女士。

如果是这样的话，美国经济就必须软着陆，软不了也得软。否则，在大选之前再搞出点乱子，这不就说明拜登哈里斯无能？哈里斯的总统梦也就彻底熄火，川建国乘乱崛起，说不定真的把美联储给收拾了，这后果谁敢试？

所以这段时间，美股即便跌也是跌幅有限，不可能出现断崖式暴跌，美联储会出台一系列超宽松政策。但资金做的是未来，既然看到你丫要降息，还有放水的可能，那么先早早的囤积黄金，这就是最近金价不断创新高的原因。并且资金们也会早早做好外逃的打算，抛售美元资产，到全球去找价格洼地。

之前我说过，这波大A的行情必然会跟美联储降息周期一起走，毕竟这波下跌是从美联储加息开始的。

更何况，从前我们想加快经济的流动性，启动大基建、拉起房地产就行了；现在地产基本已经没办法拉起来了，或者说今时今日这种环境下，也不适合拉起地产。至于什么时候拉，大家可以去看看当年煤炭行业是怎么整合的，然后对照一下时间节点，乐观的看，也只能保证底部相对平滑，没有那种可怕的暴跌。一轮资源的整合，往往都有国资进场收集筹码，现在筹码能算收集够了吗？

地产和地方基建几乎是同时被封住了，国家层面上，主导的就是一个针对企业设备的更新改造。这是为了避免让产业被金融资本裹挟着到人力资源更低的国家去。制造业、科技、新质生产力相关的企业都要在这个国家主导的投资方向里分一杯羹，毕竟未来是属于科技的。你得让资金愿意往这里投，那么就必须有资本市场配合。

为什么从新掌门人上市之后对IPO查的如此严厉？对存量上市企业的财务造假也是下来狠手收拾的。有朋友做一级市场的，现在都在那里不知道咋办了才好，一般般的企业根本不可能像之前那样，靠着PS大法混上大A再说。那么这反过来就逼着资金向那些真的好企业集中。

实际上，做投资的都不傻，不会看不出来你业绩造假、技术夸大。只不过投资这件事从来讲的是击鼓传花，只要能卖出更高的价格给市场，那么投垃圾反倒更好，毕竟垃圾企业更没有底线，更愿意配合。咱就是说，如果企业已经做到大疆这么成功，你觉得他会听你资本的吗？

所以我跟大家的看法还是有差别的。我们要集中资源办大事，那么股市是不是一个优选、选优的资源？用资本市场的力量，让好的企业更容易获得资金，打好科技战，这是大方向。

那么转到现在的市场上来。这几天为什么主力非要把银行打下来？你看到的是六大行屡创新高，你可能没关注的是银行板块近几年的高点是在 5 月份，大行掩护，其实很多中小银行里的资金早就加快跑路了。大家可以复盘一下上一波银行股的高位在 2019 年 4 月，后来发生了什么？银行一路跌，但科技股却涨起来了。

很多趋势在初开始发生的时候，很多人是不信，也不愿意相信，他们更愿意相信自己的所谓经验。咱就是说，你过去的经验让你发大财了吗？成为很成功的人了吗？即便是，也要多思考一下；如果不是呢？你凭什么用本来就不怎么成功的经验来指导未来？

心态打开一点。无非是时间问题，难道给个机会吧价格打下来不开心吗？

唱多绝对不是脑子有病，我自己也是满仓开干的。

先声明一点，在三四月份，我也有降低仓位的，老粉丝都该知道，这个我也公开说过，并且之前每次低吸冲高也会首选落袋为安。毕竟，我们是来市场上赚钱的，不是来市场给人当肉垫，动不动就割肉，哪怕是个大胖子，割来割去也会成骨头架子，还有命吗？所以我们一定要做确定性投资。

什么叫确定性？就是你对个股有信心，不是脑子一热随便乱选的。你要清楚股票的真实内核、大主力的心态。说个非常简单的道理：如果这里所有人都被套了，那么你肯定套不住；如果里面都是获利盘，大家都赚的盆满钵满，随时清仓跑路都无所谓，那么大概率你会被套。股票的本质就是博弈。

这段话我为什么不放在公开部分？因为我这是我最近悟出来的。中美的市场本来就不同，你看拼多多，一旦说了自己不分红、不回购，立刻暴跌；但如果他在大A这么说，有可能跌，但绝对不会跌成这个狗样子。

大A的股票本质上都是筹码，筹码能不能涨，其实跟企业到底多好关系不大。有人要问，那跟什么有关呢？

说白了，跟大势有关。管理层希望鼓励什么产业发展，那么那些行业就有机会出大牛。想想看，过去几年里一波新能源牛市猛不猛？再往前，芯片牛猛不猛？还有之前的创业牛、科创牛、中小板牛、影视牛，本质上都是被顶层设计带着走的，甚至连白酒牛也有国产替代的意思在里面。

当然了，也有点小票纯靠博弈走出来的，这就不在讨论之列。个人投资者能买到这种涨的莫名其妙的牛股，纯靠撞大运。还是那句话，靠运气得来的，迟早凭实力亏光。

一声令下，银行就是再牛逼都没办法，只能乖乖的把吃到嘴里的高利率贷款给吐出来，这在美利坚可能会发生吗？想都别想！

现在大A里其实是有两个主线的，一条是把中国资产的价格给做上去。我这么说不是没有道理的，因为房产价格大跌，消灭了大量的财富，但财富估值没了，负债还在。你不把资产做大，靠着辛辛苦苦的干活赚钱去填补窟窿，那要做到何年何月？日本之所以彻底失去了几十年，其中一个重要的原因就是泡沫破裂留下的债务问题，把小日子给彻底吸干了，搞出几十年的通缩，经济停滞不前。

当然了，日本被美国殖民了，财富被美国给吸走了，这也是重要原因。但因为日本整体资产价格长期低迷，本国投资收益低下，资金跑到美国去高效利用，你凭什么反对？我们摸着石头过河，苏联、日本、美国的石头都要摸，肯定不可能犯重复性的错误。

大家可以看到，去年胡锡进进场差不多时间，国家队开始重仓买入银行股，各种ETF也买了不少。所以我一直说，胡锡进不会无缘无故的开始炒股的，没有收到点风声，他素了一辈子，这么大年纪跑进来当韭菜，凭什么？

只不过某队资金量太大了，又不想当冤大头，人家肯定是一边砸一边吸，跟着他一起进来，大概率会被套上一阵子。

央企也会被考核市值，所以我一直说，有闲钱、多余资金的，可以买买央企破净票，因为迟早得回到净资产之上，这就是纯纯的套利。至于要等到哪一天呢？其实大可不必过于纠结，你把它当长期存款就行了，反正现在银行存款利率已经跌破2%了，无非是时间问题。

之前为什么大资金狂拉六大行、两桶油？很简单，大资金认为它被严重低估了呗，要把整体估值拉起来。当然啦，拉住这些大家伙其实花钱不多，并且还能吓唬投资者，达成砸盘低吸的目的。我之前也说过，一旦行情真的要来了，这些之前用来拉行情的票，大概率会被砸下来平衡指数，让市场不至于太疯。

那么很简单，跷跷板的另一端，之前没怎么大涨，甚至被最近这段行情给砸下来的央企国企票就是我们重点关注的对象！把中国资产价格做起来，肯定第一个要消灭破净股！

放在板块里，那就是中特估。我长线持仓的账户里，中特估只有在冲高的时候减点仓位，大跌的话立刻把减仓的股票给买回来，其实目标就是尽可能的降低持仓成本，并且保持重仓。所谓中特估，选股一定要选破净票，并且之前跌的越惨越好，当然了还必须有点业绩。

炒高资产价格，必须要用上工具人券商，所以券商是可以配的，前几天我在微博上还秀过在券商上的操作：逢低吸纳，冲高卖出。但这次券商跟之前不一样，之前金融行业还能高高在上，这次整个金融都成为献祭对象，所以再想有之前几波行情里券商那种豪迈的走势，个人觉得可能性不大。那么怎么选呢？要么选最稳当的，要么选最烂的、已经跌成狗的，你们自己去琢磨呗。

还有一个保险要特别注意，因为保险是手上资产拿得最多的。之前保险熊了那么久，都是被房地产暴雷给拖累了。但到了现在，其实也差不多了，既然国家已经出手收房了，等于是给了个底，那么保险整个行业差不多也算是软着陆成功了。个人认为，保险整体的行情走得会比券商要好，毕竟保险标的少，券商一大堆。

大A的另一个方向就是中美科技战。

新质生产力这个口号是年初开始提的，别以为就是个口号。鼓励企业搞升级改造，这就是在扩大内需，我们没有大搞基建，因为地产已经那样了。新质生产力本质上是中国特色对中国对抗欧美联合起来玩的那一手所谓“供应链去中国化”。毕竟论起人力成本，中国是比不上隔壁的印度和东南亚的。欧美扶持印度东南亚，一方面可以打击我们，另一方面也能给咱们弄一圈外敌，毕竟同行是冤家嘛。所以提出新质生产力，就是要通过新的技术大力提升生产力，让我们的成本通过科技加成、产业链供应链加成，抵消掉对方的廉价人工吸引力，起码是把上游、重要的产业牢牢控制在自己手上。那么这里的机会到底在哪里呢？

其实还是之前的那些：自动化、信息化、数字化。那么对应的就是这里的行业受益。

自动化里就是工业母机、自动化控制、机器人这些板块了，这里几大头部企业都是重点关注的对象。还有个人工智能在工业生产中的应用，这些还真的只有头部大企业才有实力去做，其余的都是噱头，所以就坚定的搞龙头策略就好了。

信息化，还是集中在通讯、通讯服务，要特别关注下物联网这个细分板块。这里的逻辑不用我解释吧，我们要用信息化技术来解决产能浪费的问题，这里未来还会有很多相关的说法，很有可能会有些相关的政策出来刺激下，反正在里面找找蹲蹲呗。

至于物联网的说法，9月份有个华为鸿蒙的全联接大会，鸿蒙要搞万物互联，这不就是物联网的事吗？那么这里是不是有很多有相关概念、跟华为扯上关系的票？重点关注啊，说不定就能抓住一枚妖股。

还有个数字化，这就不用多说了。我重点强调的还是其中数字安全细分部分。我们要跟老美打科技战，人家喜欢搞阴招，之前不是有个微软漏洞让全世界都断片了，你怎么知道这不是人家搞的提前预演呢？总之网络安全、数字安全等于是发展新质生产力的基础建设。这个板块一直没有资金炒作，也一直没起来。头部某企业老板现在跑去做了网红，天天上热搜，那老哥在回国二次上市带了一帮小伙伴，这个月定了要散伙，但一群人都被套了八年了，抗战也不过八年啊，你觉得他可能啥事都不做吗？

还是那句话，核心是大主力想要涨，概念是说出来能喊来围观群众认可买单的噱头。假如一点噱头都没，可能就要烂在主力手上，两者互为因果，希望你们能悟透这里的道道。

说起科技战，自然有芯片在里面。芯片板块我看空了几年，从今年开始看多，做了一波，但个人认为接下来还有行情。之前看空是因为大基金带头砸盘，现在看多是因为大基金它又来了。

以我个人跟大基金搞投机的经验看，大基金进来是先砸盘，再慢慢减仓，讲究的是长期持股，往往都要赚几倍的利润才可能走。建议喜欢做芯片的朋友可以去多留意大基金的动向。

大家可以去关注下最近的妖股深圳华强，这货居然连续拉了 11 个板，都没有被关小黑屋，这说明什么？有记忆的可以去复盘下 2019 年东方通讯带起来的一波行情，牛一打旗杆那是相当的生猛。有牛一冒出来，你就别太在意牛一到底是干啥的，就像东方通信成为 5G 概念龙头，然而人家根本不做 5G，但那重要吗？重要的是资金开始选择了，并且管理层也在默许了。

当然了，我没有让你们去接深圳华强的飞刀，我只是让你们注意到市场的变化。这不是我自己在那里臆想出来的，这是有实实在在的资金做出来的。接下来一段时间，各位朋友尽可能的要多关注下科技股了。

九月份最最重要的概念就是海思和鸿蒙，两场华为大会，前者是芯片的，后者是搞手机系统的。每年华为概念都会来一波，中国科技实力最强的企业也就华为了，华为是不可能上市，但跟它扯上关系的上市公司很是不少。能给华为做供应商的企业，都不会差到哪里去，如果这时候大股东有强烈的拉市值需求，借了东风，很容易就走妖了。这里要画重点了，这个月重点盯住华为概念的科技小票！

之前有人问我，通胀概念还要不要做。

如果你手上有持股，且被套了，没必要现在割肉。很简单，通胀是大势所趋，大宗价格一定会起来的，黄金也会继续创新高。但猪肉涨价，未必马上养猪的企业就涨了；黄金涨价，黄金企业也未必能涨。肯定会跟着涨的，但啥时候，只能说看主力的心情。

如果市场上没有什么好题材大家一窝蜂的做，那么主力可能会跑去做通胀，但如果市场上有热点，做通胀概念就有点不划算了。个人认为九月份有华为的因素在，通胀没啥可做的。

游戏我要重点说下。游戏因为黑神话悟空的缘故火了一把，上个月付费里我也让大家重点关注下游戏，倒不是未卜先知算出来黑神话悟空会全球大卖、搞成现象级事件。说游戏纯粹是因为游戏传媒又跌倒了底部，这个板块本身盘子比较小，拉起来也容易，往往到了低位就会各种蹭概念拉。当年炒AI，最后大涨的是游戏。游戏跟AI有什么关系？硬要扯的话任何行业都能这么扯，其实根本原因就是这个板块是个很好的筹码，跌成狗了，所有参与者都被深度套牢，随便找个借口就能拉。

所以游戏这个月还能继续做，只要看到连续大涨就要快跑，短线玩的就是心跳。还是那句话，得集中精力，别什么都想做，那样只会害了你，人的精力、金钱都是有限的！

医药、消费之类的尽可能的少碰。有华为在，九月份即便有苹果大会，也少去看果链的，除非能跟华为扯关系。
