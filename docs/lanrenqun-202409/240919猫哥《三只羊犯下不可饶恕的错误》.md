

# 三只羊犯下不可饶恕的错误

240919 猫哥

整理：公众号懒人搜索，懒人专属群分享
懒人微信：lazyhelper

正常情况下，三只羊还算是根基深厚的，线上线下至少有两方面会保它。

线上是抖音平台，毕竟三只羊是抖音好不容易扶持起来带货顶流，如果三只羊倒了，电商这块利润丰厚的项目要再扶持一个顶流出来也非易事。

线下是合肥地方，2023 年三只羊地方纳税几个亿，这可是地方财政的现金奶牛，特别是在房地产下行，地方财政困难的大背景下，这头现金奶牛重要性更为凸显。

所以，如果三只羊仅仅是犯个一般点的错误，包括不限于广告虚假宣传，产品质量暴雷等等，有线上线下两方面力量保护，三只羊都不会有太大的问题，偏偏三只羊在风口浪尖上犯下的是不可饶恕的错误，这就是作死了，所以这一次三只羊基本就是凉凉了。

上一次在辛巴手撕三只羊事件中我写三只羊文章最后给了一个结论，如果能及时改正错误未来还有一线生机，如果执迷不悟，那就真的要凉凉的。现在三只羊是真的执迷不悟。我给大家捋一捋。

辛巴手撕三只羊从更高的层面来看，主要目的还是敲打，因为当时三只羊仅仅只是注册了香港分公司，大致算是有了向境外转移资产的动机，如果敲打一下，三只羊幡然悔悟，估计也不会赶尽杀绝。

比较典型的信号就是，辛巴准备在某日直播对三只羊客户进行赔付的时候——也就是手撕三只羊刚刚要到最高潮的时候，辛巴在快手的直播间被封禁——这就是明显的信号，敲打差不多了，事情到此为止。本来这就是给三只羊留下一线生机，但是三只羊接下来是怎么做的呢？是在直播间大规模推销美诚月饼！

美诚月饼的事情远远不是擦边涉嫌虚假宣传这么简单。

美诚月饼所属企业是香港美诚食品集团有限公司，这个公司注册地是湾仔乐基中心15楼1512室，这个注册地办公室属于世侨商务有限公司，这个商务公司大致类似于国内工商代理公司。

这个工商代理公司旗下有多个代理公司，其中偏偏就有4家涉嫌帮助伊朗军方洗钱被美国财政部制裁。

也就是说，帮助香港美诚食品集团有限公司代理工商业务的这个工商代办公司与某些跨国洗钱公司有紧密的业务合作。

之前三只羊卖美诚月饼也有两年历史，但是过去卖这款月饼与现在卖是不一样的——原因很简单，今年8月三只羊注册了香港分公司。

直播电商业务流程是这样的。

三只羊在直播间卖月饼，然后销售金额就进入香港美诚公司的银行账户，然后香港美诚公司再把相应的佣金打入三只羊公司银行账户。

所以，如果是过去三只羊卖美诚月饼，那么香港美诚公司只能把佣金打入三只羊国内的银行账户，但是今年 8 月之后，香港美诚公司完全可以把这笔佣金打入三只羊在香港公司的银行账户。

考虑到这个香港美诚公司可能与跨国洗钱公司有间接关联，这就不能不让有关部门有所警惕了——三只羊这是要干什么？

前段时间对你的敲打难道毫无作用吗？

你为什么还要在这个风口浪尖时候顶风踩红线？

这里多啰嗦几句，香港公司在大陆做生意，是可以相对自由将大陆账户的资金汇入香港银行账户的，但是大陆公司要将资金汇入香港银行账户这个是要经过严格审查的。

所以三只羊拼命推销美诚月饼，资金到了美诚公司账户之后，美诚公司可以相对自由汇入到自己香港银行账户，然后再把对应给三只羊的佣金汇入到三只羊香港分公司香港银行账户。这就是一条清晰的转移资金到境外的渠道。

公司注册地在香港，生产地在大陆，宣传时声称这就是香港产品——严格意义上讲，这不是虚假宣传，事实上大部分在大陆售卖的香港产品其生产地都在大陆，如果真要在这个环节较真，香港产品就没法做了。最多三只羊在直播间有些言辞有夸大宣传之嫌，也就是按照广告法规定罚款而已。

过去辛巴与三只羊撕逼的时候，官方是冷眼旁观了，但是这一次官方却下场了：
- 人民日报点名三只羊；
- 三只羊注册地合肥高新区一把手被双开；
- 三只羊被合肥市场监督局立案调查；

这里要特别说明一下，昨天我只看到三只羊被合肥市场监督局立案调查的新闻的时候，我还认为，这是合肥方面抢到了对三只羊的办案权，三只羊还有一线生机——毕竟从地方税收的角度，地方是肯定不愿将三只羊一棒子打死的；但是后来我发现三只羊注册地合肥高新区一把手被双开的新闻之后，我发现，决定三只羊生死的已经不是地方可以决定的了。

最后再多说几句。

最近有媒体陆续爆出三只羊过去带货产品的质量有问题，简直就是触目惊心——从电吹风到梅菜扣肉，到假茅台、牛肉卷、面膜等等。

这让我很不理解。

三只羊一年的销售好几百亿，这在过去已经相当于一个国内大型连锁超市的销售额，过去这种国内大型连锁超市品控管理是非常严格的，根本不可能出现类似三只羊这种在短短两年时间就爆出如此多品类产品的严重问题。

## 这说明什么？

说明三只羊的品控部门根本就是形同虚设——据说三只羊只要对方提供质检报告就敢卖——过去国内大型连锁超市如果敢这么干，早就倒台了。

财富到一定数量级之后，有两条红线是绝对不能踩的：
- 政治错误不能犯；
- 业务基本面不能出问题；

三只羊是两条红线全部踩中，即便有天王老子护驾也同样会出事。

我们国内某些直播带货的头部主播，崛起很快，但是内涵却严重缺失，表面上看起来很光鲜，底色却是一个草台班子。

说白一点，就是对财富对资源的驾驭能力严重不足，出事翻车也就是迟早的事情。
