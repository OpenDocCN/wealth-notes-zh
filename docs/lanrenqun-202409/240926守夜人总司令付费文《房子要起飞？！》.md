

# 房子要起飞?!

240926 守夜人总司令
整理：公众号懒人搜索，懒人专属群独享
懒人微信：lazyhelper

如果把时空的跨度拉大，你就会发现：一个后发的农业社会如果要完成社会基础结构和产业体系的升级，就必须要经历三个阶段。最困难的是前两个阶段！这三个阶段都围绕一个共同的点而展开，那就是：从哪里搞钱！

第一个阶段是建立重工业体系，为农业社会开始向工业社会转型打地基。作为一个后发的农业社会，这个阶段如果不能对外殖民掠夺，就只能对内采用剪刀差的方式挤出钱来。这个模式由苏联的大列宁发明，虽然残酷但是有效，而且是唯一可行的方式。第二个阶段就是在打下地基之后，建立主体框架并补齐功能结构的过程。在这个过程中，需要建立完整的工业体系，让纯粹输出劳动力变成输出成品技术和服务。这个过程往往伴随着城市化的进程，通过房子从市民群体手中收集所需资金就成了唯一可行的方案。也正因为如此，越是规模较大的社会，到了这个阶段房地产都会起飞。

第三个阶段是精装修阶段，需要构建起以自身为主导的经济秩序，并且在这个经济秩序中扮演引领者的角色，需要大力发展科技和高附加值和高门槛的尖端产业以及更有利于资源配置的高端服务业——金融和法律都属于这种高端服务业。与之同时，还要建立内部的福利保障体系。这个阶段所需的资金需要从全世界筹集，通过自己构建的体系和秩序来获取。

每个时代都有自己的主题，每一代人都有自己的使命，当一个主题完成之后，一代人的使命也就完成了。下一个主题所衍生出来的机会和问题属于下一代人，完成使命的人和事都会逐渐谢幕。2013年取消了农业税，这并非源于上位者的仁慈，而是源于下位者的日益无足轻重！随着工业体系和城市化的进程，农村最有活力的群体逐渐迁徙到城市中成为新市民，农业税的收入逐渐变得越来越微不足道，代替它的是城市工商业的发展所带来的赋税收入逐渐占据绝对份额。所以，农业税被取消了。第一代彻底谢幕，第一个主题也彻底成为历史。

今天的我们是不是站在第二个主题逐渐进入倒计时的边缘地带呢？准确的说，在一个社会的城市化率才63%的时候，还没有进入到这个阶段，不管怎么说，城市化率到达83%的时候才会彻底进入这个阶段。今天的一切都只是整理和下蹲。下一波的上跳需要更高附加值的产业和更多的资金量来支撑，这种资金量显然不仅仅来自国内，而是来源于整个经济体系所能覆盖的区域。就如同今天纽约的资产不取决于纽约人的钱包厚度，而取决于它在整个美元体系中的比较价值。只要它比其它地方的东西更值钱，那么，大家就会去争抢它。

尤其是那些布局了未来高附加值产业的城市，未来的空间很大。

一个东西是不是有很大的成长空间，是不是会被政策扶持，取决于一个点：官方愿意接受什么筹码的质押！凡事官方愿意接受的筹码，官方都有意愿让其溢价。在城化进程中，土地就是这样的筹码。地方手上的土地高价出售给开发商，开发商把土地质押给银行，银行给予低利率的贷款，开发商做个样板间就开始预售，然后用回笼的预售款来盖房子，而把土地抵押获得的低利率贷款拿来购买新的土地，如此循环往复，越滚越大！在这个过程中，地方高价卖出筹码获得地方建设所需资金，银行向开发商和个人发放贷款获利，企业贷款循环滚动不断壮大，个人和家庭通过贷款抢筹获得增长溢价预期。整个就完成了闭环！

根据最新公布的官方数据，任何一个领域的官方收入都在下降，唯独房产税试点的税收收入增长了20%，还有一个与环保有关的收入也增长了20%，如果官方总部的财税收入都是如此，地方就更加不用说了。坦白讲，这个现实反馈比预想的要糟糕得多。不断发长期国债就是总部借钱让各地能继续运转，官方投资来实现布局和拉动就业。许多人会说，国债不是卖的很好么？不管是大资金还是个人都在抢购。

问题恰恰就在此，内部的资金去抢购自己的国债是左手倒右手，让外部的人买才是吸引新的资金。内部的资金应该投入到消费和生产环节，而不是去薅国家的羊毛。虽然官方喊话好几次，甚至处罚了两个机构，但避险情绪下的这种行为并没有收敛。

不仅如此，随着美元的降息，之前许多外贸企业一直拖着不结汇，把钱留在外面挣美元利息。还有一些势力利用内部的低利息大量举债，然后瞒天过海弄到外面去换成美元套利。当美元降息之后，这些资金都会卖掉美元回来，此举又进一步提高人民币的汇率形成一种循环叠加的效果。这些钱回来之后，面对当下的这种现实情况，你猜猜它们会去哪里？肯定会去抢长期国债！所以，必须要抛出一个绣球给它们，让它们不去咬骨头，而是去追绣球。

到目前为止，兔子的所有行为都是防御性质的，并没有采取大水漫灌的行为。如果总部收入越来越少，国债就必须越发越多，利息的负担就会越来越重，此时此刻，大家还一拥而上去薅羊毛加大利息负担，就会造成恶性循环和螺旋下降。为了避免这个局面，兔子做了两件事：
- 降低存量房贷利率；
- 允许企业质押股权低利率借钱回购。

第一项是为了刺激消费，因为这个社会唯一拥有消费能力的群体就是拥有房子的群体。大幅降低存量房贷利率可以让每个月的还款减少，从而能让这笔钱用于消费。为了避免大家把每月因还款减少而多出来的钱不用于消费而存起来，随后也会降低存款利率，甚至会持续降低。除此之外，还下调了二套的首付比例和利率，也是为了进一步刺激贷款意愿和住房及相关商品的消费。此举与第二个措施其实是相悖的！

第二项是允许企业把手上持有的筹码质押给银行，借到便宜的钱去回购自己的股份，而且只能用于回购自己的股份。上面说过，银行接受什么筹码的质押，就会扶持什么筹码溢价。质押之后获得便宜的钱去回购，回购之后再质押，循环往复。在这个过程中，其它人是不是会去抢筹并试图通过溢价获利？如果存款利率再下调，而企业的股权溢价预期上升，是不是更多的家庭和机构会把钱从低收益的地方搬迁到这种高收益的地方？至少政策的预期是这样设计的。如果大家热情很高，不排除银行退出低利率的贷款让大家去抢筹码。地方不仅拥有土地，地方还有一个叫国资委的机构持有本地企业的大部分股权！土地卖不动的时候，卖股权也是可以筹集资金解决自己的运转和投资问题的。

此时此刻，你是不是有种似曾相识的感觉？每一个参与者角色都没变，上一次的道具叫房子！而且这一次的门槛更低：房子的首付至少也是百万起步，这一次门槛差不多降低到十万的水平了。当然了，这一切还只是规划。第一步既是投石问路，也是为了应对严峻的现实所采取的防御措施。真正的后手，还需要等美元降息确定方向之后。

这里有一个问题凸显了决策者只是试图：不管是哪一边，最终都是末端人群扛下了一切，然而，这个群体不能同时投入到两个地方，如果已经下定最终决心，一定是把所有的末端投入到一个方向。二套降首付和利率，此举是为了吸引末端把自己有限的资金和负债能力投入到房子的消费。同时，又在另一边造势，试图让大家热情高涨的广泛参与。这不是发起战役正常的样子，而只是火力侦察行为。建议抓住情绪高涨的机会，见好就收。
