

# 央行多策齐发，咱们的切实利好有哪些？

昨天，我们共同见证了央行释放的史诗级大利好。

在美联储官宣降息 50 个基点之后，市场上一直有“降准或者直接降息”的传闻和预期。

大半个月前，外媒彭博社也发布了“中国政府考虑用现利率房贷替换历史高利率房贷”的新闻。

所以这些政策如果公布一两条，只能算符合预期，算不上“史诗级”。

可是昨天，央行却将一系列利好同一时间发布：降准、降息、降存量房贷利率、降二套房首付比例、出资 8000 亿专门用来投资股市……

这些政策同一天发布，力度已经空前，而央行行长潘功胜在发布多个利好的时候，原话都有“未来加码、不设上限、开放性”等表述，那就是这些利好政策是趋势，而不是一次性的。

这次央行在国务院举行的记者会，主题确定为“金融政策更好服务于经济高质量发展”，代表着“一切政策，为了经济”的基调。

过去两年，实际上许多人都感觉到经济大环境与自身收入减少导致的困境，那么这次多箭齐发，给经济最直接的影响是什么？对老百姓的直接利好是什么呢？

当前经济大环境，有自身结构的“通缩”问题，也有大量民众的“信心不足”问题，要彻底扭转这一势头，让经济重回正常增长空间，“一次鸡血”是不够的。

这次扭转经济大环境与全民信心，有轻重缓急。最先落地的抓手，就是股票市场。

很显然，过去两年，老百姓“信心不足”的一大源头，就是跌跌不休的股市。这次央行“首次创设”货币政策支持资本市场。

“首次”一词，就代表了过去几十年都没这么干过，具体政策是：出资 8000 亿元，其中 5000 亿用于“证券、基金、保险公司互换便利”，3000 亿用于“股票回购增持再贷款”。

简单解释，就是证券、基金、保险公司在资本市场的债券、股票 ETF、沪深 300 成份股等都是长期持有，便于流通的，都可以抵押给央行，低息或者免息换取资金。

当然，这些资金“都只能用于股市投资”。“股票回购增持”也是一样，回购增持的股票都能抵押换取资金，再次入市投资，等于放大了资金效能。

直白地讲，就是央行定向放水 8000 亿，给机构低息或者无息贷款，但这些钱只允许买股票。

过去，我们见过几轮调低房贷利率，鼓励老百姓买房；也见过专门针对小微企业的低息贷款，鼓励小微创业。

而一次性发 8000 亿，资金只能用于股市投资的贷款还是第一次见。

过去一段时间，老百姓信心不足的第二个原因，是通过每季度发布的 M1、M2 数据，一次次见到央行放水，但是效果不理想，数万亿资金都在银行空转，最后又回到央行。

这次央行 8000 亿贷款买股票，等于一次性解决了两个问题，第一是几亿股民对股票市场信心不足的问题，第二是过去天量放水资金在银行间空转的问题。

央行潘功胜行长在介绍这一政策时，原话是：“这 5000 亿不是一次性的，如果有效果，就会有第二次、第三次，我们是开放的”。

所以相对当前的股票市场，一个8000亿解决不了问题，但只要有作用，央行就愿意有后续的第二次、第三次的8000亿资金跟进，直到市场信心与基本面彻底扭转为止。

前文讲了，要全面扭转经济与全民信心，股市投入8000亿是第一步，那扭转经济大环境的第二步和第三步又着力在哪里?

两者是相关联的，那就是楼市需企稳，以及高房贷居民急需减负。

相比较彭博社大半个月之前报道的“按揭利率置换”，这次央行的政策更加直白，直接“降到位”，让过去的高利率房贷，降到现在的低利率房贷。

根据数据统计，当前全国背负房贷的家庭是5000万，这背后就是将近两亿的城市新居民或者所谓的“中产中坚”，这次降房贷利率，每年降低约1500亿的负担。

实际上，“降房贷利率”是“违背契约精神”的，当年做出购房决定时，是什么利率，都应该买定离手，国家不降，从法律上讲，购房人就得为“自己的选择负责”，所以国家最近两年屡次降房贷利率，是实实在在地削弱银行优质贷款收益，给老百姓“减负”。

这次将二套房首付比例降到15%，则是降低二套房的购房标准，过去三年楼市低迷的一个原因，是大家普遍认为“刚需群体的缩水”。毕竟在房地产市场最繁荣的二十年，婚房和“丈母娘刚需”是最刚需的群体。

最近几年结婚率和生育率的走低，也侧面印证了“首套房刚需”的减少不可逆。

这次降低二套房购房门槛，首付比例降到和首套房一样，未来二套房的贷款利率也很可能进一步降低，很显然是扩大“改善型需求”为刚需的政策，降房贷和降低二套房首付比例，直白点说就是增大购买人群，为“楼市托底”。

这次针对楼市下行和居民房贷压力大的政策同样是“一鱼两吃”，一方面给5000万房贷家庭减轻压力，另外则是为楼市托底，过去十年打压房价的呼声很高，鼓吹房地产“一鲸落，万物生”，“房价跌、消费自然暴涨”的观点成为主流，可是过去两年的现实告诉大家，几亿“城市中产”的体面，其实就是他们的房产。

一旦房产暴跌，整个资产缩水的他们，怎么可能成为消费主力？

这次金融政策多箭齐发，第四个目标，则是要解决“消费不振”的难题。

过去我们见到的货币政策，往往是降准降息隔一两个月，而且力度较小，这次央行宣布降准0.5个百分点，降息0.2%，而且还强调后续分别还有降准和降息的政策空间。

这次美国一次性降息50个基点，今年之内至少还要降50基点，而且经济学家普遍预测明年美国还有100基点以上的降息，所以美国的降息放水未来两年是持续的，这也就给了我们持续加码政策的空间。

但是政策给了，就相当于给一个病人“中西医结合多方用药”，最终的目标还是这个人自身免疫系统，自身健康循环要建立起来。

中国经济过去的长期短板，自身健康循环的最大堵点，就是“消费不足”。

这次降存量房贷，降息的同时，也官宣要保证“息差”，就代表过去已经极低的存款利率，要进一步降低。

这就是让过去“存钱为习惯”的这些人收益减少，用这部分收益，置换“敢于买房”这部分人的部分利息负担，这也是一种“财富再分配”。

过去一段时间，许多人给出的建议都是“捂紧钱包，多存钱”，与政策层面是不见兔子不撒鹰的博弈状态。

可如今国家多箭齐发，政策到位了，需要看到的就是新增存款减少，消费提振，散户资金入市，楼市企稳，改善型购买增加。

央行政策承诺的“政策有效，就持续加码”，这里的“有效”，就是以上这几项，至少有几个趋势扭转才行。过去老百姓和政策面进入了“信不信”的博弈，一方是你不给政策，我就不消费、不买房、不入股市；政策面是“你体现消费意愿、购房意愿、入市意愿我再出政策”，两方都不走出第一步，市场大势就不会有起色。

如今政策面已经走出了第一步，接下来就看市场与民众的反馈了，反馈的好，才是“政策有效”，才会有“继续加码”。

我们依旧需要警醒的是，昨天三大指数都大幅上涨，大 A 更是走出几年一见的大行情，“昙花一现”、“政策太迟”的声音依旧充斥网络。

这轮货币政策多箭齐发，扭转的是未来一个周期的市场大势，而不是确保每天大涨多少多少点，可是咱们的“网络舆论反馈”往往是以天和小时数为周期的，但凡未来一两天有回调或者涨势不够猛，都会有“昙花一现”、“猛药失效，彻底无救”的言论放出，这时就需要鉴别力和定力了。

央妈这次多箭齐发，能看出来为了经济已经拼了。而后续政策的“加码”，前提都是“政策有效的话”，

所以未来几个月市场和民众的反馈至关重要。

如果“任你千百策，我捂紧钱包”的声音继续鼓噪，最终造成“政策无效”的结果，那政策真可能昙花一现了!
