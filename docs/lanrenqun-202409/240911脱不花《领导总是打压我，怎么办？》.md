

# 领导总是打压我，怎么办？

240911 脱不花

你好，这里是全力以赴帮你解决职场问题的花姐信箱。我是脱不花。

今天收到的问题比较多，所以咱们闲话少叙，直奔主题。来看第一个问题：

“花姐你好，我在高校，前不久岗位调动，到学院担任书记。我感觉科级干部的能力参差不齐，所以想利用集中学习培训的时间，培训《干得漂亮》中的内容。请问花姐，对开展好部门内部中层人员综合能力的培训，您有哪些建议？另外，有《干得漂亮》一书的相关培训资料吗？谢谢花姐！”

首先，祝贺这位同学履新，第二要非常感谢你对于《干得漂亮》这本书的认同和重视。从私心来说，我当然巴不得全世界的内部培训都用《干得漂亮》当教材，但是既然是自己人，你问我关于综合能力培训有什么建议，那我觉得还是得老老实实的先说心里话：

我的建议就是，最好不要搞一种叫做“综合能力培训”的培训。

因为，高校学院里本就是大小知识分子扎堆、个性十足的地方，组织任何一刀切、大轰大嗡的活动都不容易，更何况是你刚到任，就针对“能力参差不齐”组织培训，这不一次暴露了两件事么：第一，你作为新领导，瞧不上原有团队，这一下就可能让你站在了大家的对立面；第二，你尚未确定焦点问题，也没有别的更有效的打法，姑且先抓一个半疼不痒的项目干着再说。新到任的领导，一怕操之过急，二怕主次不分，三怕被人预判。咱不能这么干。

当然，我这么一说，这批《干得漂亮》的销量就算是没了，没关系，你的事儿比我的事儿重要。

别着急，干部培训不是不干，接下来，我忍痛建议你，按照下面的步骤来开展干部能力提升的工作：

- 第一，你是新到任的书记，那肯定有搭档的院长，跟院长聊聊，看看他对团队最不满意、他要实现他的学术和管理目标，目前的卡点都在什么地方。然后，用你的能力，把这些卡点，一一对应到干部能力建设上。
- 第二，从这些卡点中，和院长共识选出一项关键举措，作为你们的速胜项目。

## 速胜项目对于新到任领导的重要性

- 第三，基于要打赢这个速胜项目，盘一下团队的能力短板，然后，针对性的、结合项目需要，组织专题培训。培训不允许只听课，而是要以分组提交项目方案框架为培训的验收标准。
- 第四，专题培训的人员范围一定要小。不允许人人都来参训，一定要分出先发阵容来，让最能打、最积极的人，先有能力提升，先出工作成绩。其他人，先看着，等有人从中获益了，你再组织任何培训，都具有天然的号召力和诱惑力。
- 第五，你到任的第一年，最好是一次一个速胜项目，按照和院长共识卡点——选定项目——以战带训——以训促战——经常庆功的方式，循环往复，来推动你的管理变革进度表。

这样的做法，是利用了人性中的两个特点：

- 第一，有选拔，就有稀缺性。人们的学习动机问题就在很大程度上被解决了。
- 第二，有比较，就有好胜心。人们就会互相较劲，而且较劲的重点就不会在你这个新领导身上了。

而经过这样的选拔和竞赛，一年时间，也足以帮助你在团队中精选出你的班底，谁最积极、谁最能打，清清楚楚。等你带队打赢了好几个速胜项目、又和院长形成了互补很强的配合关系，到时候，团队指哪打哪，彼此高度互信，届时再组织《干得漂亮》集体学习也不迟啊。说好了，我可等着呢啊。

借这个同学的问题，也分享下关于培训管理的底层逻辑。培训，最需要的是四个字：

绩效临近。

用大白话讲，就是用什么学什么、学什么打什么，才是转化率最高的。不打仗的时候组织的培训，用处都不大。所以，回单位都数数自己的培训费，看看平时浪费了多少钱吧。

好，再看第二个问题，这位银行业的朋友喊着“江湖救急”。他遇到了这么个事儿：

> “我 35 岁，是一名银行一线财富主管，近期被调到全市排名末位的支行，协助一位新上任支行行长开展工作，我们年龄相仿。相处两个多月我发现，领导是典型的一型自恋型人格，每天都沉浸在自我夸赞中，期待员工认可她。由于缺乏一线经验，对于财富这端，分行领导希望我能够辅助行长做好工作。日常探讨业务的时候，她喜欢说‘我最讨厌什么什么’之类的话，沟通比较难开展，我也起不到辅助作用。而且，她对于我们业务上的进步都归结于是她的功劳，不断打压我，喜欢在大会上对我指指点点，我的业务能力得不到肯定，每天面对她的无理取闹感到很抓狂。对于支行业务发展方向及工作重点的探讨，我们经常不欢而散，我感觉她自主、自傲，也经常因为她脱离业务轨道的奇思怪想，绩效分配的不公平，分配客户全凭心情等情况，和她发生冲突。近期我也做了心理咨询，咨询老师建议我去和上级反馈，但我也担心分行会觉得是我的问题，会不会觉得我是在告状，大概率是会让我去适应，以及调整沟通方式。花姐，我该怎么办呢？一天也呆不下去了。两年前也是一位自恋型人格领导，先去分行告状，让我换了支行，时隔一年，这次调动是分行安排，我担心，我在分行给人的印象是和领导相处不好，我很矛盾很焦虑怎么办。”

这位朋友，因为和上级的相处问题，你都要找心理咨询师了，说明这件事对你的困扰非常大。我不知道这件事对你的情绪影响到了什么程度，不敢妄言，只能建议你不要中断与心理咨询师的定期会面，让自己始终能处于被支撑、被帮助的状态比较好。

接下来我要冒着很大的得罪你或者伤害你的风险给出一点我的看法，请你兼听则明：

你说，两年前，也是和支行领导相处不好，所以调动的，而且你给那位领导也贴了一个标签，巧了，也是“自恋型人格”。这次的新领导，你都能给她进行临床分类了，不仅是自恋，而且还是一型自恋型人格。就是心理咨询师也不敢随便这么下结论吧。我不禁想提醒：咱是不是也太沉浸在给别人贴标签的世界里了？

你要说连续遇到俩领导，都处不好，这我信。但是你要说一个单位，连续碰到俩自恋型人格的领导，而且都换着法儿的、无理由的就欺负你打压你一个人，我存疑。

我不知道你的心理咨询师怎么说，但是，我反对任何一个人，学一脑子似是而非的心理学概念，就开始忙着解释周边世界。因为，这是在用简化的标签在偷换真实世界的复杂性，这是一种大脑的偷懒行为。请注意，这个世界不是被解释出来的，是被探索出来的。

我们给自己输入的所有认知，都是为了改变自己，而不是为了标注别人。

我相信，隔壁心理成长圈的三位顶尖心理咨询师，李松蔚、陈海贤和钱炜老师，也会认同这一点。

我担心你现在处于一种焦虑情绪之中，过度内耗，难以自我调节。所以，我建议如果你能休假，就申请一次时间稍微长一点的休假，比如两周到一个月。利用这段时间，给自己安排规律性的长时间户外运动、沉浸性的阅读或者其他心理咨询师建议的活动。争取让自己在某种程度上跳出这种过度焦虑和敏感的状态。

如果不能长时间休假，那么你建议你每天给自己安排正念冥想的专门时间，此前钱炜老师曾经专门为我们录制过一个正念冥想的引导词专辑，我放在文稿中，可以下载到手机里。我推荐你试试“探访心湖”和“与山同坐”这两组冥想词。刚好这也是我个人非常喜欢的，试试看，你会有很好的冥想体验。

顺便说一句，其他同学也可以试试。

给这位同学的最后一个建议，你抓紧该考证考证、该认证认证，争取尽快转岗到某个专业技能型部门工作吧。以你的风格，做一名专业人士，也许是比做业务管理者更舒适的职业选择。祝你一切顺利。

好，下面是第三个问题：

“花姐您好！我的垂直领导同时分管好几个部门，我的部门是最让她省心的，活干得好，员工状态也好。但最近业务受阻，各个部门都很吃力。上个月，她把我的一个骨干调到另一个部门去充实力量，我这边却没有补人。最近她发现我们部门的一个重点项目推进变慢了，说人手不够就把部门里的员工再抽一个出来填到这个项目里去。我就实话实说，部门少一个人，其余人就已经超负荷了，向她申请增加人手。结果，天就这么聊死了，她不予答复。花姐，您帮我分析一下，这是沉默表示否决吗？我应该怎么跟领导沟通，让她支持我的工作，尽快补齐部门的人员缺口呢？”

你看，你也说了最近业务受阻，那么，能控制成本费用的时候，领导肯定没有道理要增加人手。况且，你这次沟通也不是一个正式申请，是话赶话说到这里了，更像是你想起来顺嘴提了一句，领导不答复，但是也没否定，就说明领导还是给你面子，所以回避了正式冲突。你说天儿聊死了，我觉得没聊死啊，只是领导不愿意聊而已。那怕什么，又不是给个人谋私利、占便宜，由不得她愿意不愿意。你就好好准备、好好论证，再找机会，正式汇报一次。一次不行就两次，总之要让领导把你这个事儿当个事儿，经常被提醒，时时惦记着，不一定哪个时机，领导就把这个人手给你补上了。万一坚持不给你补人，但是领导清楚的知道你的难处，那么对你部门的项目交付情况也会多一点包容。总之，多说情况、多拉警报，对你的工作，没坏处。

第四个问题：

> “请问一下花姐，想跳槽回原公司，面试我的HR和领导都是之前认识的，有些尴尬怎么办？需要面试前和她们打招呼嘛？以及面试、入职后要注意些什么？”

这有啥尴尬的，一个老员工，在出去锻炼了一圈之后，想回原公司，只要硬条件达标，对公司对个人都是大好事啊，因为融入速度快、认同度和忠诚度也更高。不过，对个人来说，肯定不能装外宾啊。那不是面试前打招呼的问题，应该上门单独跟领导沟通一下，诚恳表达自己的想法，甚至请老领导帮你内推岗位，这才是最妥帖的做法。

这个同学还追问了一个问题：

领导结婚送什么礼物比较合适？

说实话，我结合前后两个问题，想跟这个同学说：咱要是不擅长，就完全不必非得跟领导搞关系。好好工作、努力学习、坦诚交流，已经是非常好的下属了。领导结婚，你要是收到请柬了，就跟去参加婚礼的其他同事一样，按照当地的标准送上红包。而且这种红包是对方以后也要找机会还礼给你的。是个互相抬轿子给面子的事儿。你要是没收到请柬，那你根本就不用送礼，嘴上恭喜几句就可以。或者，你们单位有这种文化的话，可能一个部门的同事会组织凑份子合伙送一份礼物，那就去凑个份子就行。

在这种事上，不必过于热心，更不必过于积极，不仅买不到什么好，还很有可能后患无穷。因为你这个领导要是单独收你的礼物，长期来看，他也混不了太好。

第五个问题是一个关于创业的问题的，可能不创业的人听起来觉得挺简单的，但是，只要有过创业经验的人都知道，这其实涉及到一个巨大的坑，几乎所有创业者都会遇到这个难题。同学问：

> “花姐，我想试着开始创业，可不知道合伙人和投资人的股权分配该怎么做，加入和退出机制该怎么做。哪里可以尽快学习到这些知识呀？”

我的建议是：

既然第一次创业，先调整自己周边的人际关系比较重要。

要让自己身边成功创业者的密度变高，因为你会从他们的举手投足间体会到创业者的思考方式和行事模式，这比知识和道理重要多了。所以，我推荐你可以先试试参加一些得到新商学的游学活动，多认识点儿其他创业者，你的问题，从这些真创业者身上，自然都有答案，而且都是浸满了血泪和汗水的真答案。
