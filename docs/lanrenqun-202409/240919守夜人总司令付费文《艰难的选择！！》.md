

# 艰难的选择

在经济增长最快的十年中，每年有8500亿美元顺差，10年就是8.5万亿。除去这些年3.3万亿美元的储备和2.2万亿的外部债。还有3万亿美元流入的钱花哪了？请注意，是流入，不是流出。这背后有个违背舆论主流观念的逻辑，本文用隐喻简明阐述并给出一个判断风险和选择的框架思路。没有阴谋论，只有两难的选择，最终决策者会选择长痛不如短痛的方案！

一个家庭里面只有这么多钱，是用来集中培养一个最有出息的孩子，苦一苦其他孩子；还是让每一个孩子都不掉队，让大家眼前不至于过得那么苦？作为家长，这是一个无比痛苦和艰难的抉择！

从现实中绝大多数家庭的行为来看：家里的情况就是这样。把钱平均地花在每个孩子身上，一家人其乐融融，也很温馨。虽然不至于有太大的出息，眼前也还过得去，大家也更容易满足，并在这种小确幸中感受到幸福。

只有极少数的家庭会反其道而行，会把所有的资源都集中在那个最有希望的孩子身上，其他孩子就过得非常艰难——不仅艰难还会与家里矛盾不断，都想往外跑，认为给别人当孩子会过得更幸福一些。面对这种棘手的局面，这种父母不惜代价集中力量去提升社会阶层的计划，就有可能因为其他孩子的不理解、不支持，甚至是连同外人一起指责和闹腾而半途而废！

外人为什么会唆使其他的孩子去跟他们的父母对着干？因为你们家真的升级成功了，他的孩子和家庭就会受到实实在在的威胁。所以，此时此刻，最好的策略就是挑拨这些在现实中感受到不公平的孩子反对父母继续执行这个升阶计划。让这个家庭回归到一种更温馨和平凡的生活状态。

这个父母为什么这么狠心？手心手背都是肉。为什么就要苦其他的孩子，把所有的资源都集中到那个唯一有希望的孩子身上？因为父母看到了危险！开越野车冲坡，一旦冲不上去就会滑下来的危险。

事就是这么一个事，已经讲清楚了，之所以要用比喻，是因为太敏感了，兔子和鹰酱这两个家庭都在借钱花，但他们把钱拿出来之后，花的方式不一样。鹰酱家是父母出去借钱，然后向大儿子和小儿子直接发钱。孩子们拿到钱以后把原来欠的债都还，孩子们变得轻松，花钱的意愿也就更强。父母身上背了大量的债怎么办？父母就时不时从家里面拿出一些祖传的宝贝，不断向邻居吹嘘它的价值连城，甚至叫他的孩子们相继出价抬价。结果这些宝贝的价格越抬越高，邻居们也相继掏钱购买。通过制造热点话题，吸引邻居们来回倒卖的方式挣了左邻右舍不少钱，鹰酱家的那些参与者趁机进一步减少了自己的债务杠杆，并且有更多的钱可以用于消费。但是做父母的背负了太多的债务，每一天的利息都非常高，以至于没有余钱去做别的事。

兔子家是个什么情况呢？15年前，兔子家的父母把自己家的祖屋和地契拿出来，让孩子们掏钱买。孩子们的钱不够，还贴心地给孩子们放贷。于是，孩子们存的压岁钱以及打临工挣的钱都投进去了，还打了欠条，欠了一屁股债。兔子家的父母通过这种方式聚集起了大量的资金，然后把这些钱全部投资在最有出息的那个孩子身上。各种培训班都给他报满，不惜代价请最好的老师，更高一个阶层的人所要具备的能力他都要会。而且，还往亲戚朋友那里借钱，交给这个孩子做启动资金。这种投入确实获得了不错的产出。这个孩子拥有了全产业链的完整能力，而且别人能做的事情他都能做，过得比别人又快又好。慢慢地，有高阶人士开始愿意跟他合作，并利用他的能力去做一些收入更高的事。照这个趋势发展下去，这个孩子就会真的实现阶层跃升。当他完成阶层跃升之后，如果父母能管得住他，他就会开始补偿弟弟妹妹这些年来的付出和牺牲。如果管不住他，兔子家就会逐渐变成另外一个鹰酱家。

当兔子家的阶层跃升正在进行得如火如荼的时候，许多矛盾也不可避免地产生了！弟弟妹妹们哀鸿遍野，觉得活不下去了。强烈要求大哥不要再去搞那些看不懂的东西了，让大哥把手上的钱匀一些给大家。大哥抢走的那些高端合作伙伴，也让别人非常生气。于是，就开始怂恿和挑拨兔子家的弟弟妹妹说：你父母无能又偏心，你这样不值得！你应该去跟他们闹，让他们停止偏爱你们的大哥。兔子家的兄弟姐妹一听很在理，就开始各种闹腾。然而，不管怎么闹腾，兔子家的父母就是不松口。不愿意把任何一分钱花在与战略方向无关的地方。就算不改弦更张，现在不是缓和了一点吗？放点水出来让大家喘口气总行吧？暂时不行！因为人民币比较坚挺，所以国际贸易人民币的使用额度超过了亿元。此时此刻继续保持坚挺，才能进一步抓住这个机遇期扩大它的适用范围。此时此刻放水会造成它的波动，而且它与主题无关呀！

现在兔子家是冰火两重天。一方面大家感觉到未来可期，另一方面又感觉到日子愁苦。两者都是真的！而且父母的意志不会因为闹腾就改变，这是一个冲坡的关键时候！正因为如此，暂时没有机会，还会有风险，对于热门的东西千万不要跟风。跟主航道有关的东西务必坚守，以待春风！

一个是父母负债，儿女轻松，父母无权办大事。不管是制造业回流，还是其他的什么什么议案，都只能喊口号。因为家长真的没钱！不仅没钱，利息越来越高，头皮发麻。一会儿人工智能，一会儿登陆火星，都是为了让左邻右舍掏钱来买自家的古董——但是那古董没啥价值！美国制造业的回报率是3%，但利率是5.5%，这还制造个啥，直接存利息还多挣2.5%！

一个是儿女负债，父母把所有的资源都投向大儿子，大儿子确实越来越出息了。各种之前自己这个阶层想都不用想的东西都会弄了！汽车出口第一，飞机也要开始出口了，甚至开始大量出口芯片，更别说光伏和港口机器。这些产业的投资都是砸钱砸出来的！

有一个有心的美国人问了一个这样的问题：兔子在增长最快的十年中每一年的顺差超过8500亿美元，10年就是8.5万亿美元顺差。中国外汇储备3.3万亿，再减去海外2.2万亿美元债务。还有3万亿美元去哪了？是啊，去哪了？这是流进来的钱，不是流出去的钱。这些钱花在哪了！不管是军工还是民用，这些成体系的产业是怎么来的？都是这些年砸钱砸出来的！

这十几年孩子的负债逐渐增高，父母的负债也不低，都花在大儿子身上了，因为他最有出息，最有可能把这个家带到更高的一个阶层。大儿子有没有乱花？有没有浪费？肯定有！但是瑕不掩瑜，如果不能上到更高的一个阶层，接下来维持温馨的小日子都很难。因为兔子家没有假古董能够忽悠所有的邻居们掏真金白银去买！当然了，等兔子上去之后，他们家也会有的。刚刚不就在法国发行了20亿欧元的长期国债。北欧的瑞典不是跑过来签署货币互换协议了吗？

兔子家的家长这样沉得住气，会让内部有些人抓狂，因为这样下去，他们的日子会越来越苦。他们以前与其他的兄弟姐妹比起来还相对轻松，他们更善于察言观色，打着为大哥服务的旗号相对轻松。其他兄弟姐妹一直苦惯了，这个时候虽然更苦但也就那样。当大哥快修炼成功的时候，他们以前的那种服务大哥的地位就没了，趁机偷懒捞好处的伎俩也没用了。这让他们非常恐慌。而且之前他们还与外人眉来眼去，两边的好处。所以他们强烈地要求大哥不要再修炼了，父母要匀出一些钱来给兄弟姐妹们缓一缓。其他兄弟姐妹可能会被蛊惑，但是兔子家的父母不会松口，绝对不会！此时此刻，绝对不会让钱流走，恰恰相反，会开放更多的政策吸引钱进来，进到哪里来？进到大哥那里！2024年11月份开始实施的一份新文件，开放了许多以前不开放的领域，这些领域除了私营医院之外，其他都集中在制造业。你猜什么类型的制造业会有高回报率，值得外面的钱去追逐？就是大哥所代表的领域！
