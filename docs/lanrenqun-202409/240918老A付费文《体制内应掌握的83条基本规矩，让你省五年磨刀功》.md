

# 老A讲体制内应掌握的83条基本规矩，让你省五年磨刀功

240918 老A
整理：公众号懒人搜索，懒人专属群分享
懒人微信：lazyhelper

虽然，已经写了几篇付费的文章，几乎将我关于体制内的感悟写完。很多同志，看完以后，点点头，恩恩恩，觉得写的不错。但是，却没胆量在工作日常中贯彻。

人最大的勇气，就是敢于改变自己。不能学其道，而不用其道，这样学到的一切都只能是像赵括一样，纸上谈兵，叭叭叭地教别人可以，轮到自己上了，却熊包一个了。

很多人建议我再写把这感悟写的详尽一点，恰逢大领导进京开两会，事情相对较少。借办公室电脑，敲下如下83条，让各位体制内的年轻人，更进一步的了解体制内的台面上的事和台面下应该注意和了解的事。很多道理可能你们都懂，但是没引起重视，或者没去做。希望你们若真觉得老A说的有道理，便落实在行动中。

承蒙各位看得起，大家都是付费看文章的，十几块钱，对于涨薪后的体制内人来说，不算多，但是看后没收获，没改变，也没什么意思，既浪费了金钱、且浪费了时间。
所以，我盼各位要做行动上的巨人！
开始：

- 1、每天上班，要把自己收拾干净，穿着整洁，不留指甲，尤其是男同志，更不要留长指甲。每天洗头，给人一个精神的印象。尤其是头发干净，千万不要临风飘雪花。打扮可以稍时尚，但不要大图案，尽量穿成熟一些，不要装扮成小可爱，单位不是家，没人看到你可爱便给予你父爱，除非干爹。
- 2、抽时间把单位的工作手册认真的看一遍，知道自己该做什么，不该做什么。免得以后被老资格忽悠过来忽悠过去的。
- 3、年轻人，如果家中没有特别重要的事，尽量早一点来，晚一点走，平日里八个小时，谁高谁低、谁敬业谁懒散，领导看不出来。但是如果你坚持在八小时之外找机会，一天不行两天，一个月两个月，一年不行两年，领导一准会发现一个敬业的你！
- 4、年轻人初进机关，待人接物说话做事，把握好一个度，既不用力过猛过度表现，处处都有你，哪哪就你能。也不要羞羞搭搭小家子气，或懒惰，或骄气，都不好。大大方方说话，面带微笑就够了。
- 5、记住，时刻要有正气，但是不是让你正气凛然，只要求我们把正气放在心间，能帮人则帮人，能渡人则渡人。
- 6、别人说话的时候，一定多倾听，少插话。这点很重要，尤其是上级说话，一定不要急着表现自己。即便你确有真知灼见，也不要急于一时。机关里，沉稳大气，对一个人是很高的评价。所以，就这点来说，学会倾听，并乐于倾听且真诚倾听，也是一件了不起的本事。
- 7、工作压力大，累，也不要牢骚满腹。真的，我想过这个问题，牢骚了活还是得干，牢骚只会让自己更累。牢骚太盛防肠断啊！所以，在单位，一句抱怨的话都不要说，除非你别有目的。
- 8、一般评价年轻人优秀与否，首要的标准是：靠谱。这个怎么说呢，大致就是说你有没有干工作的积极性，能不能主动去推动，愿不愿意吃苦去解决难题，肯不肯花脑力，去把问题想透琢磨穿，待人接物不讨人嫌，比较精通自己负责的业务流程，懂得工作中基本的规矩，就算是靠谱了。实话实说，现在的年轻人，都太过聪明太会取巧了。这不是不可以，但是领导都是聪明人，可以让领导看到你的小聪明，但是心术不正的小聪明千万不要让领导看到。
- 9、富二代到机关工作的，建议名下不要挂太多不动产。本来正正当当的来源，肯定没问题的，就是提拔的时候要上报，核实来源，程序复杂耽搁时间。并且人事处的同志知道你的家底，难免不泄露出去。会有别有用心的同事因为你家底丰厚而结交你，带你装逼带你飞，一不小心染上不该染的东西，抱憾终生。
- 10、找到单位电话本，尽可能录入手机里。大部分人的手机号都关联着微信，顺便把同事的微信也加上。点赞之交好过没有交集。花点时间把每个局室处走一遍，搞清楚人家到底在哪办公。
- 11、材料肯定是要写，安身立命之本。提笔能写，开口能讲，问策能对，遇事能办。机关讲文来文往，写是第一位的。能写好稿，说明你把事想通弄明白了，其他的自然也就不难了。所以，刚参加工作就被安排写材料的，是好事。年轻人把文字功底筑牢，不误砍柴工的。
- 12、在单位，每天都被表扬，绝对不是好事，这说明领导充分把你当外人。而批评你批评得多的，却不一定都是坏事，大部分批评你的领导往往是把你当自己人，准备教你。当然，那些心理变态故意折磨你的，除外。
- 13、关于喝酒这事，其实现在好多了，如果不喜欢喝酒，到了单位就说不会喝，过敏啊什么的，一般也不会真正强逼你，毕竟从上到下，都在倡导工作期间不饮酒。但是中国的无酒不成宴的国情，有时候难免要觥筹交错。当然你一会喝一会不喝的，别人自然认为你装逼，会觉得你是个势力眼，是一个看人下菜的厨子。
- 14、有事要讲或都有工作需汇报时，是面谈，还是打电话、发短信，里边也有讲究。有些事需当面谈，就别打电话。有些事需要给对方一点时间考虑或缓冲，最好就发短信。不要动不动为了缓解尴尬就给领导发短信，领导很忙的！
- 15、在机关内，走路要注意姿态，端庄大方，不疾不徐，我建议尽量走慢一点。我很清楚地知道,不少领导通过观察走路的姿态品评人物。我反复听过类似这样的话:某某某,看他走路那个样子,急索索的,快走到天上去了,一看就不是个踏实做事的。
- 16、和人家碰杯时,杯沿要低于人家的杯沿;敬酒时,一定要双手举杯;茶壶口不要冲着人;服务员给你续水,说声谢谢;这四点都是最基本的职场礼仪,但是我发现很体制内的职场人都不懂。
- 17、机关里边有一种人,帮不了你什么事儿,但能够坏事,千万别得罪。你自己看看周围谁是这样的人,把他圈出来,记住这个人,尽量不得罪他。
- 18、如果在人事和办公室干过，一般就能够了解整个单位的运行。业务处室是否重要，不是以忙不忙来判断，而是以它和本部门的核心任务的关系来判断，比如发改委的规划处。当然，任何岗位都能够锻炼人，但早年间在一些重要工作的岗位干过，对你的大局观，方法论和看问题的高度很有帮助。
- 19、有时，可以适当向上级透露一点关于同事中流传的关于他的不太好的说法。明智的上级知道，你是向他负责。但注意有些事别碰。
- 20、没有人会讨厌有上进心的人，但大家都不喜欢太有心机、有企图的人。要阳谋不要阴谋。
- 21、一定要注意说话的技巧，说什么都不能说死了，得有转圆的余地。凡事留一线，日后好相见。在机关单位不要动不动就把什么一分为二得看这一套面试托辞讲出来，可以有一分为二辩证的思维，但是不要动不动就这样讲，年轻而又太像世故的官场套话，让人觉得厌烦。
- 22、强者示之以弱，弱者示之以强。兵法如此，体制内也是这样。他人真的很强，强到你无法撼动、无法超越、无法攀比，只能去依附，那就示弱。而在弱者面前，就要逞强。因为弱者之所以是弱者，根源在于他的内心都是柔弱的，他的工作和生活中需要强者出现。
- 23、靠得住的，往往不是那些你帮过的人，而且帮过你的人。
- 24、老同志叫帮忙，弄电脑手机什么的，虽然麻烦，尽量帮一下，别显得不耐烦。话还说好听点，别说这个简单什么的，就说：我试试看行不。
- 25、请示领导工作或签文件，宁愿自己多为难点，把问题搞清楚，方案意见尽量完善，他能画圈就ok最好。不要想到把矛盾丢给上级，他让他为难，他也就会为难你。
- 26、有时间多看点书，看原著，看大部头。看这些书的时间段里，明显感觉思维能到达更深的程度。
- 27、不要在走廊里长时间通电话，尤其是该楼层有厅级以上领导办公室，尽量不要发生声音。当然，外人在走廊里这样做时，你也可以好意去提醒。这是对领导负责的人应该做的。
- 28、有些时候，领导也好，普通工作人员也好，可能有某些机会，为整个单位或者整个处室谋福利。本来这都是好事，但一定要注意了，好的想法，未必产生好的结果，总有些奇怪的人，会扭曲你的出发点或目的。所以，这种事我是不干的。并且，坚决不干、永远不干！举个栗子：你正好能免费搞到一些水果送大家，你心想很好啊，于是办了。可你不知道，原来有人拿了水果后恨你了，因为他的水果中可能有个别坏的，或者个头比谁谁谁的小一点。他不会记得你的好，只看放大自己的不满。所以我宁可送给看门大爷。
- 29、机关里有各种小圈子，也即所谓非正式组织，如老乡会、校友会等，酌情考虑是否加入。一般情况下，意义不大，吃吃喝喝，玩玩乐乐。
- 30、机关里也可以过得很轻松，尤其是对没太多想法的女生。你一进去就摆烂，什么都不会做，做不好，嘴甜点不得罪人，领导一批评就开哭，然后抓紧时间找个好老公，连着生俩孩子，专心经营好小家庭。几年下来，基础就打好了，以后30年就过上了衣食无忧开心快乐的美好生活。我可给你们指路了。女同志，别太有野心，小日子过起来最舒服。
- 31、举一反三。叫你拿个杯子你就拿个杯子，盖子也不知道拿一个，就没法教了。让你给客人倒杯水，你就只倒杯水，连茶叶都不知道放。气人不？
- 32、不要轻视权威，有些人你惹不起，有些事你碰不起。古人讲破家县令，灭门刺史。自古以来，都是乌云之上有乌云，今日虽不同往日，但务必小心谨慎。
- 33、工作没干好，领导骂怎么办？诚恳接受。年纪大点的骂你，有时不妨撒撒娇。男人也要会撒娇，很管用的一招。不会？学啊。我现在也在学习这门本领。
- 34、别他妈到处瞎打听。可以慢慢观察，细细琢磨，少打听事。机关里面小道消息最多的包打听，有几个地位高、风评好的？可以听、但别问。
- 35、有理且让三分，这就叫会做人。无理还要争三分，这就叫丢人。
- 36、任何一个组织，最核心的权力在于人事权。人事权其实只属于一把手，其他领导在人事上的权力本质上，都是一把手让渡的。这种让渡是必须的，既是为了工作的开展，管理的需要，也是为了平衡。所以，你懂得，过年送礼，别送错。
- 37、和领导说话，不要老想表现自己，很多时候，他们要的只是好的聆听者。
- 38、没必要跟每个人都好，没那精力，很多领导也未必喜欢你和谁都好。但一定要培养几个关系好的，才有人撑你，给你通气。
- 39、其实各种素质之中，相较而言最有实用价值的，可能要算敏锐性。政治上的敏锐性，工作上的敏锐性，文字上的敏锐性都很重要。有的人天生机巧，敏锐性很高。但后天也可以培养出较高的敏锐性。高敏锐性的年轻人，一定会得到更多机会。
- 40、给领导送文件，遇到重要的，主动收集好背景材料，来去文件，让他一下就能搞明白。送之前先想想他会问什么问题，自己怎么回答。别来个一问三不知。
- 41、换岗这事，其实不难。但有个过程和套路。如果上面有人，自然点名就办，这个就不说了。正常情况下，第一，你得表现出色，让上下知道你有你这号人，不入囊中，如何脱颖而出？第二，不能太急，让人觉得好高骛远，谁敢要啊，毕竟人流动太快影响工作；第三，不跨部门，处室之间交流，分管领导或双方处长同意一般就能成，并不难。所以要注意和自己处长的关系，他不放你是走不了的。一直瞒着，突然说要走，都会生气的；第四，业务上要培养下家，你能接上手；第五，说到底就是个人的活动能力，通过各种方式；和想去的地方的老大建立关系，适时提出。理论上大概是这样的。
- 42、处长之间有自己的圈子，会交流看法，如果觉得某个人不错，就全想去挖。当需要人的时候，会相互打听，也会推荐不错的年轻人，或者为与自己关系好的争取机会。所以即便不是自己的直接领导，也可以处好一点。反正就是你的情况，其实不止自己处室的人知道，可能不少人都知道一些。
- 43、新入职虽然收入有限，还是要时不时请下客，邀约同事们搞点有趣的小饮食。不论好坏，是个意思。
- 44、帮领导报账一定要及时，别拖久了。
- 45、机关里总有一些人，不大干工作，专事帮助社会上各色人等解决问题。这类人消息灵人面广，可以和他们关系搞好点，有需要可以打听下消息找点办法，最好不要请他们办事，他们的人情都是有价的。
- 46、可以带块表，随时掌握时间，好过掏手机来看，也好搭配衣服。两三千块就好，不要太高级了。
- 47、增加思维的维度。这个可能听着有点悬，也不好解说。你能区分平面的图形和立体的模型，一个是二维，一个是三维。人的思维也是如此。一般人的思维就是二维，可我见过三维思维，甚至四维五维的人。比如分析问题，一般人考虑，只考虑当前存在的问题和解决方法，这是二维维度。有的人就能考虑问题时加一个时间维度，想到这个问题未来会演化成何种态势等等，这就是三维维度。
- 48、有不止一个对我说过，最大的惩罚，就是把一个人冷藏，什么事儿也不分配。几年下来，这就人就废了。
- 49、年轻人一定记住，年轻时，岗位比级别重要。非常重要。
- 50、这么多年，一个切身感受就是，只要向着一个方向持续努力，总会有所成就。至于说到底能到什么高度，还要看时看运。放平心态，淡看云起云灭。大鹏一日同风起，扶摇直上九万里，也未可知。
- 51、半途接手的工作，做起来比较容易出错，要么不接，要么特别仔细。
- 52、开会准备会议材料，主要领导那份一定仔细翻翻，确保无误。这个程序，千万不能找任何理由省略掉。
- 53、开发多任务程序。就是你能同时处理多种不同性质的工作，这个要求很高，统筹协调能力。这个能力，太重要了。
- 54、别怕麻烦人，麻烦来麻烦去，关系也就建立起来了。但是也不要真的太麻烦别人。掌握好一个度。借钱就属于太麻烦别人的一种。
- 55、女领导一定要重视，她们不是能力过硬，就是背景过硬，并且心思细，不好得罪。得罪了，如果你不想办法化解，她会一直记得！
- 56、坚持走程序很重要，多请示多汇报。领导交待的事，一定要及时反馈。他们的记忆力很好。这条非常重要。
- 57、鞋一定要擦干净，裤子上一定不要挂串钥匙，裤兜别放太多东西。和领导同事吃饭，注意节制，不要像没吃过饱饭的，人家看不起。
- 58、可以在还年轻的时候，就着手了解孩子上幼儿园上小学的事，反正要不了几年就要面对，一般大机关都有指标和一定门路，也是机关的福利。不过这种福利也越来越少了。
- 59、一定要多看。看得到才想得到，想得到才做得到。看到什么，脑子一定要过一下，不要看到就过了，对看到的东西要有反应，思考。去领导办公室，看下领导看什么文件、杂志、书籍。到同事那，看下在办什么件。又不费事就掌握了信息，大数据啊。
- 60、看到领导要热情招呼，千万不要一低头当没看到，就算不是分管领导也一样。不止一位领导专门问过我，说某某是哪个处室的，看到领导也不打招呼，基本礼节都不懂。所以领导是很在乎这个的，会记住的！行礼打招呼未必加分，不打招呼肯定要减分哦。
- 61、初进机关，先搞清楚机关组成构架，班子组成，分管情况，各局室处主营业务。慢慢再了解以至掌握整个机关的运行机制。说实话，我发现在机关工作30年以上的人中，保守估计有一半以上，并不完全清楚机关运行的机制，当然是指大机关里。这点非常非常重要。为什么重要？太麻烦懒得写了。
- 62、头几年一定要下点功夫，把自己的工作钻深想透，有点专业性的味道。机关很大一个问题，就是不专业。但我又反对太专业，容易被人家定位死了，君子不器嘛。当然有些业务性部门，可能也不一定。
- 63、一般情况，尽量不要和人正面冲突。如果发生了，就要清醒认识，双方之间的关系，永远不会弥合了。直说就是以后防着点。
- 64、判断自己的位置，哪些人对自己成长影响最直接，哪些人是自己最直接的竞争对手。找到对手，不是让你去斗人家，是要去学习，并全面超越对方。而且机关的世界也很大，真有能力了，不用在一口小锅里抢饭吃。
- 65、勤快一点，多帮老同志办办事跑跑腿，但别参与他们的八卦。
- 66、不影响本职工作的情况下，主动多参加机关党委，团委，工会组织的活动，扩大交往面，多认识点人，各方面都有好处。
- 67、提高文字水平。机关工作无外乎办文办会办事，什么都好干，写稿最累。多学，学领导怎么改稿。光看成品看不出什么名堂，参加改的过程，清楚来龙去脉，成长就快。

- 69、嘴巴甜点，勤招呼人。与人打招呼，寒喧都是有技巧的，几句话让人家高兴一下，何乐不为？我小时候啊，上班看到女处长以至于女厅长做了新头发，穿了新衣服，都要夸一夸的：领导，这个发型真适合您，真有气质。真诚地说，不要油嘴滑舌，不要奉承谄媚。

- 72、打招呼很讲究。比如距离感，隔一两百米你就点头哈腰，那是傻。遇到大领导，我建议你站路边稍等，让人家先过，面带微笑点头问好。
- 73、多在单位食堂吃饭，可以交换很多信息。我从来都反对年轻人整天只和自己处室混，我基本上在食堂，从不和自己处室的坐一块。
- 74、不要卖自己的领导，尤其不要向大领导卖自己的顶头上司。可以为领导背锅，尽量不要背同级的锅。
