
# 即将登场(9不能仅仅学明朝)

回到明朝是一个现实的选择，但是也是一个需要克制的选择。这需要理论的眼光来看待，更需要哲学的眼光来看待。

## 国家的疆域

国家的疆域是什么？这其中有两层意思，一层意思是地理概念，这是一个国家之所以形成国家的地理合集，是地理空间的概念。另一层意思是政治概念，这是一个国家之所以为国家的人口、历史、文化、宗教的合集，意味着这个政治集合体对于国家的认同。

国家的疆域不是一成不变的。往往是，一个国家强盛的时候，疆域会大一些，衰落的时候，疆域会小一些。疆域的大小往往是一个国家强大还是衰落的标志。但二战后，这个局面有了很大的变化，由于反对帝国主义、殖民主义的兴起，大批量的民族国家出现，整个世界的疆域大小变化变得非常敏感。

像美国这样扩张领土的做法，一定会在整个世界留下阴影，这会带来不可逆转的反噬，一定会。疆域和领土不能等同。疆域往往大于领土，扩张领土只是其中的一个做法。由于民族国家的崛起，导致大家对于领土极其敏感，但是对于疆域，大家却没有太深刻的领会能力，因为这属于渗透的部分。

回到明朝，并不是我们的领土回到明朝，而是疆域。疆域往往分为两个部分，第一部分是领土性疆域，这部分包括领土、领海、领空，也是关注度最高的。第二部分是非领土性疆域，这部分非常复杂，但是可以大致分为几类，一类是没有领土但是有利益的区域，比如深海采集区、南极北极、太空等等，二类是国家利益覆盖的区域，特别是经济利益所在区域。

一个国家的疆域，往往决定了国家的形态，是一个陆权还是海权或者是陆海兼备的国家等等。历史的发展中，我们只能认为当前俄国作为陆权的极致，英国、美国作为海权的极致，我们作为陆海兼备的极致，基本上都是各自类型的范本。英国更典型一些，英国在扩张过程中疆域是扩大的，衰落的时候，疆域也是在收缩的。

## 维持疆域的因素

一个崛起的过程往往是300年左右，衰落的过程往往也是300年，比如英国，至今仍然在衰落的过程中。美国目前是将衰未衰，我们是将强未强，俄国则是衰落中出现了中兴。

但是维持疆域是很难的一件事，任何国家的兴衰都跟疆域的扩大、维持、缩小有关系。包括当前的美国，美国正在战略收缩，原因就是维持不了现在的疆域，只能选择核心利益予以维持，这跟一战之前的英国是一模一样的。

维持疆域往往有4个方面的因素：

第一就是战争。疆域特别是领土的扩大需要战争，不是平白无故扩大的，疆域的缩小，往往也是战争的失败造成的。敌对国对于疆域的觊觎，往往会造成战争，这是国家竞争的常态。

第二就是宗教。比较典型的就是南斯拉夫和非洲，尚未形成现代国家的概念，还是停留在宗教的概念里，往往会造成领土的不断分裂。

第三就是意识形态。比如东德和西德，虽然合并了，但是一个德国，仍然是两个制度，两个国家的体系，至今仍然没有完全统一，造成德国至今都非常沉默的根本原因，因为没有实现统一。

第四就是透支国力。对于边疆的过度开发和使用。比较典型就是唐朝，力量全部分散在边疆地区，导致中央无人保卫，过度透支国力，使得昙花一现。

疆域的核心在中央疆域，而问题的演变往往在边疆。边疆距离中央远，但是会让中央瞬间垮塌，这是因为一个信心和决心体现的过程。

美国人的边疆，东部在日本韩国一线、西部在中东一线，中央是空虚的，但是没有人能打到美国的中央，这让他有恃无恐。

我们的边疆，极其复杂，周边的所有国家跟我们都有关争关系，这是亚欧大陆的常态，所以也不奇怪。俄国人的边疆，也差不多，当前的俄国版图就是它力量的极限，苦苦支撑已经是非常不容易。

保卫中央疆域是每个国家必然需要的。首席问大家一个问题：如果我们跟WW或者日本、印度开战，我们首要防卫的是哪里？首都。只有首都是需要重点防御的。同样，反过来，我们要进攻或者日本、印度，首先进攻的是哪里？首都（或者是省会）。因为这里是它们的要地，不进攻要地，难道进攻森林公园？

同样，我们跟帝国作战，双方的重点是哪里？固然是重点工业城市、港口、基地等等，但是最核心的还是它们的首都。

这是核心疆域。当前大国之间的斗争，名义上是代理人战争，实质上都是远离自己国土的利益边疆战争，也就是所谓的发展权战争。

我们是一个以千年看待问题的民族。实际上，清朝将中国的疆域概念进行了划分，清朝之前，历朝历代的疆土徘徊在由400毫米等降雨量决定的中国北方农牧交错带，后期则跨过了400毫米降雨带，将疆土稳定地伸向西北干旱地区。主要是政治手法的不同，与前朝不同，清朝在北边采取了联蒙政策。彻底将疆域稳定的扩展到北方农牧交错带以西以北，因为汉唐时代，国力强盛时拓土西北，国力衰微后固守河西。

康熙时代后，清朝稳定将疆土延伸至中亚草原，至今350年，从没有丢失过这片土地。虽然有过多次反复，但还是跨过农牧交错带，将王朝的疆土实实在在锁定在伊犁河流域和喀什噶尔地区。

350年后，我们当前的观点要进一步扩张：要突破传统疆域的概念，进一步将我们的疆域拓展到海洋和利益毗陵区。

## 制衡为主，征服为辅

大家都在说澳大利亚，严肃的说，如果是作为领土予以征服，这是透支国力的情况；如果是作为制衡的产物，则是有可能。

我们国家的核心竞争力是制造业，制造业占全球制造业的30.3%左右，是美国、日本、德国和韩国的总和，我们占了30.3%，他们4家占29.9%。特别是疫情以来，我们充分意识到我们的核心竞争力其实是制造业。

目前，我们已经把中国的制造业做成了世界上最便宜的制造业。随后我们希望我们形成世界价格便宜、质量好的制造业。未来20年，中国的制造业变成又便宜又好又高科技的制造业，而且我们希望能够维持着30%的比例不变。

但是我们也有短板，最大的短板就是我们的资源都在海外，才能支撑这么大的制造业，资源稍微有一些变化，就容易影响我们的效率。

因此在这个过程中，拓展疆域的核心应该在经济疆域方面，采取的手法应该是制衡，而不是征服。

当前的世界，已经出于“过度饱和”的状态，国家已经非常多了，相互之间都处于竞争状态，这种状态是非常紧绷的，稍微有领土的差异就容易出现问题（当然，欧洲除外，欧洲的整体素质还不如经过了社会主义洗礼的非洲）。

在这种情况下，领土的变化往往牵一发而动全身，只有制衡才能解决问题。不战而屈人之兵是战争的最高境界。

比如说，澳大利亚跟印尼的天生矛盾，印尼跟我们的防备状态（印尼至今仍然防备社会主义，法律条文里面有），这种三角防备状态，让我们如何去惦记澳洲的领土呢？进群加微信pep458这不是将印尼推向澳大利亚，进而阻断我们的海上利益，让美国英国做梦都能笑醒的事吗？

同样的故事在缅甸也能发生，我们再去插手缅甸，分裂这个从明朝末年就已经比较完备的国家吗？这是典型的将缅甸向印度怀里推。

理论界一个案例是讲到“改土归流”之前，明朝进攻缅甸北部的麓川国，当时麓川国不服明朝廷，朝廷出兵重创它，结果打破了缅甸南北力量的平衡，南边的缅人向北进攻，灭掉了汉人，基本形成今天缅甸的版图，汉人再也没有办法插手缅甸。

不能因为一时的兴起，就破坏区域的利益平衡。

我们的核心一定是一些小国家：比如，海岛国家，非洲国家等，这些在所有大国的边缘之外，也是我们利益的核心区，这样的国家，有利于节省我们制衡的成本，也不容易打破平衡。

要知道，疆域扩展中，每逾越一类地理障碍，就意味着进入一个新的环境地带。

回顾我们国家的历史，疆域与所在土地上的社会组织、政权、意识形态、种族相对应，而将散处在各地的松散人群凝聚在一个政权之中、整合在共同疆域之下
的力量，不仅是一个政治和军事事件，文化认同具有更重要的意义。

跨越地理环境，将疆域的整合推向到经济生活方式完全不同的地区，应是政治、军事与文化的合力。

而同化非我族类的人群，需要500年乃至1000年的时间，除非它们因为我们的包容，进而消失在历史的长河中。

即便是美国人如此强大的洗脑能力，也无法完成对欧洲同类的同化；还有一个案例就是香港，2019年乱了半年后，到现在，其实仍然不服管教，整个治理难度很大，需要一代人的时间。

如果香港都让我们这么头疼，那更别提拥有200万日本后裔的台湾了。那个难度会更大。

“吃不到嘴里的都不是美味”。如果我们只是简单的领土征服，而不能消化掉这些利益，最后也会很快丢掉利益，进而会进入到战略收缩和最后的瓦解的状态。

中国历史上，在疆域特别是边疆治理上体现了德治教化的做法。大佬文集圈一种兼容并蓄的文化思想体现出来，也是文化先进的华夏对文化落后的夷狄进行“教化”。

历代王朝通过对边疆地区实行安抚怀柔、羁縻而治、移风易俗、兴学崇儒、科举取士等举措，体现有教无类这个特征。

最后是通过“恩威相济”“因俗而治”“广布文教”“互市通商”等策略，使我们国家的核心区与边缘区渐成政治统一体。

这是理想状态，即便是这样，也会消耗（而不是透支）我们的国力，因为这需要我们投入大量的教育资源、物质资源、金融资源。

可为了发展权，我们必须迈出这一步，必须有很强的利益边疆的概念。因为我们准备整合或者参与制衡的国家，基本上都是在边缘、半边缘地位的国家，它们的国内已经有不同的社会认知类型，彼此都有冲突，由此带来的结果就是看不到发展的希望，需求变革的心态非常迫切。

对此，我们有办法帮助它们发展，但是更需要稳固我们的利益，针对这些边疆地区，有几件事是一定会推行的。

第一，会推行“人类命运共同体”概念。实质上，就是针对各个国家、地区、个人现实存在的经济、政治、文化等种种差异性，倡导求同存异、互鉴互谅，强调不同国家要超越意识形态、历史传统、发展水平、社会制度等因素的差异。

第二，会促进政令畅通，推动它们走向现代化，但是在现代化的过程中，赋予它们的所有发展彰显跟我们是一体的意识。

第三，会推广普及我们国家的通用语言文字，在语言上传递汉语这个世界上最有信息量和历史容量的文字，从精神上维护统一的感受。2025-03-15
