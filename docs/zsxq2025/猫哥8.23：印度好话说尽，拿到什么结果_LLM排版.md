
# 猫哥8.23：印度好话说尽，拿到什么结果

最近，印度在被老美爆揍一顿后，果断向我们主动示好，中印关系也随之发生了一点积极变化，于是让大家感到义愤填膺，认为应该趁机踩上两脚，而不是选择缓和互动。本期内容我们就谈谈这个话题。

先来一个发生在美印之间的瓜给大家吃。

如果要评川普上台后，谁是地球上最悲情的人，我觉得莫迪当之无愧。

2019年，莫迪访美，川普在得州搞了一场“你好，莫迪”的5万人大型集会，两人手牵手共同亮相，并称莫迪为美国最伟大、最忠诚、最坚定的朋友。

2023年，莫迪再次访美，拜登同样进行超规格接待。不仅在白宫举办了400人的盛大宴会，还为莫迪精心准备了全素食菜单。

那时的印度可谓风头无两，谁看了这阵仗都觉得印度就应该是“世界第三”。

到今年3月中旬，莫迪还高调入驻川普的“真相社交”平台，并发帖和川普进行互动，大炫“兄弟情”。

一场印巴空战下来，友谊的小船说翻就翻，印度被美国一巴掌拍在墙上。

被抛弃的道理其实很简单，你战场上起不到牵制作用，连巴基斯坦都打不赢，还谈啥印太战略。经济上你兜里也没几个钱，不仅不停止购买俄罗斯石油，还不开放国内农产品市场。那么对于川普而言还搞个锤子，索性直接踢出群聊，对着印度一通输出，喜提50%关税。

这还没完，当印度退无可退选择硬刚老美并且转头东看时，老美不依不饶继续输出。

8月13号，在双普会晤当口，美国财长贝森特在接受彭博社采访中表示，如果美俄会谈最终失败，华盛顿将联合欧盟对印度实施二级关税制裁，最高税率可能达到200%。印度表示很懵逼，合着你们会晤把我当人质了。

8月17号有消息报道，原定8月25号至29号美国贸易谈判代表团队访问新德里的计划已被取消。这就意味着8月27号对印加征50%关税几无回旋余地。

同时在8月19号，贝森特在接受采访时，谈到进口俄罗斯石油时，再次把矛头对准印度，大致意思是印度是奸商，石油套利行为不可接受，但中国不是。

言归正传，谈谈这次王毅外长访印这件事。

很多人可能认为，中印关系破冰是在7月印度外长时隔5年访华作为主要时间点。其实不然，中印关系真正止跌企稳的时间要追溯到去年10月俄罗斯的喀山会晤。只能说莫迪不像欧洲一样执迷不悟一条道走到黑，在美国大选结果公布前，预判到未来美国政局会发生变化，还是懂得做人留一线，日后好相见的道理。

有了喀山会晤的基调，中印双方才开始逐渐进行各级别互动，包括：

2024年12月，在北京举行了停摆多年的中印边界问题特别代表第23次会晤，2025年1月，在北京举行的中印副外长级对话等。

但是好景不长，印度死性不改，在川普上台后，又重抄旧业当起搅屎棍来，在多个场合唱反调，选择向老美靠拢，于是中印关系又急转直下。只不过还是前面的观点，这次印度唯一聪明的地方就是没对直接寻衅滋事。

甚至就在王毅外长访问印度期间，多家印媒还单方面报道，我们已承诺向印度恢复供应三大关键需求，包括稀土、化肥和盾构机，并强调这是中印关系正常化的重要信号。

这些话听起来确实扎耳朵，搞得像我们主动让步，上门送温暖式的。

结合这次王毅外长访印的事，谈谈真相究竟是什么?

首先，这次王毅外长访印，分别在中印外长会晤、中印边界特别代表会晤达成十项共识，而且莫迪也 会见了王毅外长，当然通稿没提外长身份，而是外办主任，算是勉强把提格会见解释过去，毕竟7月印度外长来，没这个待遇。包括王毅外长访问完印度后，接着去阿富汗和巴基斯坦，都谈的很愉快，说明印度这次让的比较大。

所以，谁是主动方，谁在积极靠拢，答案不言而喻。

其次，需要搞清楚的是，印媒所报道的，迫切需要我们恢复供应的三大件如今是怎么一个状况。

其一是盾构机，这事主因还是印度自己拖欠尾款导致，去年2季度我们就暂停出口交付，后面又遇到我们稀土管制(盾构机部分设备含稀土永磁电机),所以这件事就摆起了。

我们倒不要紧，着急的是印度。做工程的都晓得，工程摆一天就是一天的钱，不可能上千号施工人员说聚就聚，说散就散。

其二是稀土，今年4月我们对7类中重稀土实施出口管制后，印度汽车、军工产业就受到停产影响。而且印度之前就有前科，将进口稀土转售老美军工企业。所以，虽然我们现在实行合规审批，民用优先的出口政策，但对于印度企业申请，如今大部分仍然处于待批状态。

其三是化肥，我国化肥一半以上出口到印度，而且印度特种肥料严重依赖我们，之前我们管控了化肥出口，就导致印度农业受到严重影响，比如尿素，2023年印度进口大致在200万吨，2024-2025财年已经下降到10万吨左右。6月虽然我们放宽了出口限制，但对印度仍然维持管控。

你看，上面三类物资，说大点，涉及印度粮食安全、基建能力和高端制造业，印度等米下锅能不急吗。之所以我们到目前对印度都是“特殊照顾”阶段，无非就是印度前几个月确实态度不端正，又想着当老美的马前卒。

实际上，就在7月份印度外长访华期间，就与王毅外长讨论了尿素、氮、磷和钾以及磷酸二铵、稀土金属和盾构机的供应问题。

而我们的态度是随着两国关系走向正常化，中国将恢复供应后者急需的肥料、稀土金属和盾构机。

这里个人认为有2个伏笔，其一是“正常化”,其二 是“急需”,何谓正常化和急需呢?说穿了，就是假如我评估认为你态度端正，那可以考虑解你燃眉之急，但不一定量大管饱。

那么,这次王毅外长访问印度，印度拿到想要的结 果了吗?我们聊聊这次访印，双方达成的20项共识。这里去掉一些虚话、客套话，直接捡重点干货总结一下：

先看中印外长会晤的十项共识。

1、欢迎莫迪出席上合天津峰会，双方支持对方办 峰会，包括印度2026年的金砖峰会和我们主办 的2027年金砖峰会。不出意外，我们明年要去印 度。

2、尽早恢复两国直航、修订民航运输协定、为两 国从事旅游、商务、媒体等活动人员双向往来提供 签证便利。两国商务往来后续会加强。

3、继续开展印度香客赴西藏神山圣湖朝圣，并扩 大规模。

4、维护以世贸组织为核心，以规则为基础的多变 贸易体制。

再看中印边界特别代表会晤达成十项的共识。

1、设立划界专家小组，探讨在条件成熟的地段推 进划界谈判。这点有非常大进展，划界不用整体 谈，先把能谈的落实了。

2、除边境西段既有的将军级会谈外，在边境东段 和中段设立将军级会谈机制，并尽快举行新一轮西 段将军级会谈。这点对推动整体局势降温、边境 和平安宁也有进展。

3、共享跨国河流的水文信息。这点我们是释放了 善意，打消雅鲁藏布江水电站对印度国内的顾虑。

4、同意重开三处传统边境贸易市场。

你看，从会晤安排和达成的共识上看，印度可谓态 度诚恳，好话说尽，但对于印度核心诉求，包括投 资、稀土管控、盾构机、化肥等事宜， 一个字都 没体 hiện 。

双方共识主要涉及边境问题、外交问题、经贸交流 等问题，这些可以说更多都是我们关切的内容。这 20项共识，绝大部分都是之前印度亲手建立的围 墙，现在又要自己拆了，我们仅仅只是顺水推舟而 已。

所以，大家放一百个心，印度是什么款式，我们太 清楚不过了，就算后面要恢复供应，也是有条件、 有限制的逐步放开，在未来一段时间，印度的7寸 会死死拿捏住，以观后效。

谈最后一个话题，为什么我们选择保持和印度互动 沟通呢?

其实开篇说的恨不得冲上去再踩上两脚，那是处于 个人情绪发泄的角度，但从务实的政治角度看，却 不一样。

其一，老美可以把印度一个大脚踢得远远的，但 我们做不到呀，这个恶心的邻居搬不走。他如果失心 疯在边境又搞动作，不是说我们怕他，而是我们有 更多的大事要办，不愿意在烂泥里面搅合。现在印 度主动贴上脸认错道歉，对我们缓解边界压力，就 是短期效应，也是一桩好事。

其二，上合和金砖之所以发挥不了能量，印度这根 搅屎棍是核心原因之一。这次好了，印度表态全力 支持就意味着8月底的上合峰会大概率会拿到重 大成果，这也是好事一桩。我们先不管未来落地会 是啥样，还有没有波动，至少在股东会层面大家先 把协议签了，以后拿协议说事，如果印度不遵守协 议，踢出群就是顺理成章。

其三，从生意的角度看，虽然中印贸易总量不到1 万亿人民币，还进不到我们的前十，但中国对印度的贸易顺差却仅次于美国，超1000亿美元顺差占 中国全球顺差总额的10.3%。按川普的逻辑，印度 妥妥是我们的甲方。现在甲方有难，伸把手，反正 赚钱不寒碜，无非就是全款包邮，概不拖欠而已。

至于企业要去印度投资还是技术转让拿门票，我相 信有了之前的深刻教训，大家都要掂量下，当然如 果有继续选择头铁的，那就自求多福。

所以，对于印度的主动示好，好话说尽，鉴于之前 印度的表现，我们也仅仅照顾了下面子，点到为 止，对于其核心诉求，采取模糊回应态度。主要的 考量还是处于战术层面，围绕是否符合周边外交战 略，是否有利于维护西部边境安宁并拓展新兴市场 作为立足点，这是基于务实角度出发。还是那句 话，以观后效。

至于印度未来还会不会两面三刀，蜜汁自信起来 (比如我们外长刚走，印度那边又对台湾问题的立场出现反复),我相信经过多年博弈，我们对付三 哥早已心中有数，手中有枪。
