
# 孩子大学专业如何选择？

达叔作为一个每年见上千人、自己本身是学财经的、做医疗、搞投资、特别爱赚钱的人，可以非常负责任的和你说：如果我是你，我一定会让儿子学医。即使儿子从打死不学医，我也会努力给他讲背后的逻辑，其中逻辑的一个环节，就是父母是孩子的明地图。这一点，你儿子十几岁，就已经开悟了，知道爸爸妈妈在医学领域，有人脉基础，说明你儿子非常有慧根，极其聪明，没有犯普通孩子想摆脱父母的幼稚病。

你纠结的点是：学医又苦，又累，又熬时间，政策环境对医生不太友好，未来心里没底。这个判断，是错的。错的原因，就是你和老公都是做医疗的，干医疗行业太久，太熟悉，和其他行业没有对比。

过去几年，哪个行业没被锤？教培、地产、互联网、金融、体制内，医疗，几乎所有赚钱的行业，都在被锤。也就是说，政策环境对医生不友好，不是单独针对你，而是目前的锤神，见谁锤谁，谁都被锤了，不是单独针对医生群体。有些行业，一锤就死，一锤就集体失业，比如教培、互联网、地产，都是连工作都没有了，这些行业的人，连工作都保不住。医生也被锤，有把工作都丢了么？没有。被锤的医生，只是一些人的收入降低，还有很多细分赛道的人，包括你在内，都实现了赚钱继续增长。所以，医疗从业者，从韧性的角度看，下限很高，上限也很高。如果对一线城市医生群体的待遇，都不满意，那你就是有点身在福中不知福了。

学医，一定要当医生么？不一定。学医，一定在大陆做医生么？也不一定。如果我是你，我会支持这个学霸的儿子学医，长大之后，能当医生就好好做医生，不能做医生，我就找机会，把他送进头部医疗公司，培养他做经销商老板。如果你觉得这条路，只赚钱，有风险，不够高级，就可以把他弄进投资公司，做金融，专门做医疗板块的投资经理。目前所有做金融、搞投资的人，不招纯学金融的，很多都是招学医、学药的，你家有钱、有人脉、把孩子送进投资公司的医疗组，还是很容易的。如果孩子，还是想做医生，你在珠三角，可以看看有没有机会，提前研究一下，送去香港做医生。在香港，一个家庭里，只要有一个人做医生，基本上就能保证是金领的家庭，至少是中产以上的生活。不愿意孩子去香港，你就运作一下，研究一下深圳，深圳有一些医院是和香港合办的项目，你老公的能力和人脉资源，你们一定能搞定。

你研究一下，全世界头部经济体，哪些领域，特别容易出有钱人？是有数据统计的。能买豪宅的群体，最容易出现三种人：搞金融的；搞法律的；搞医疗的。平时被捧上天的科技新贵，多不多？在买豪宅的人数里，搞科技的新贵们在舆论被捧上天，但真实的分布，没有前面三个多，科技迭代太快、风险太高、和医疗千万人群相比，占比太少。你去看一个昨天的新闻，湖北省某个县城，一个检验科主任，灰色收入就能近千万，达叔身边的医生，在上海，普遍都是豪宅、别墅、好几套房子。放在其他行业，比登天都难。

医生群体，不仅有机会赚钱，还有机会在婚姻上获得极大的突破，你儿子如果有心，能娶到非常牛逼的老婆。一线城市的医生，是人脉王，能链接到很多顶级资源，这一点你老公肯定深有体会。达叔身边上海的医生，每个人的微信里上市公司的董事长都有好几个，只看他们会不会运作。
