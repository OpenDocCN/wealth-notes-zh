
# 个人养老金，领钱时要缴税？

## 文/卢克文工作室嘉宾壹壹壹

原以为领个人养老金能“躺赚”到老，结果人社部这两天一锤定音：个人养老金领取时按3%税率缴纳个税，不区分本金和投资收益。

不少人直呼难以接受，自己辛苦存的钱，退休后取出来还要被“扣一笔”？那为啥不直接存银行？

实际上，这场争议背后存在不少误解。被征税的并非我们熟悉的“社保养老金”(即单位与个人每月缴纳的养老保险)，而是2022年试点后才开始推广的“个人养老金”——交不交纯属自愿，参与者自建专属账户，每年最多存1.2万元。

这些存入账户的钱，并非闲置，而是可以投资于符合规定的储蓄存款、理财产品、商业养老保险或公募基金等金融产品。不过，账户里的资金只有到退休后才能支取，这也体现了个人养老金作为长期养老储备的特性。

去年，税务局发布了《关于在全国范围实施个人养老金个人所得税优惠政策的公告》，对个人养老金的税收优惠政策进行了详细规定。

缴费时，个人向个人养老金资金账户的缴费，每年限额1.2万元，上班族可选择在工资薪金所得中扣除这笔缴费，以此降低综合所得的计税基数；个体工商户则可选择在经营所得中扣除。这意味着，只要你缴纳了个人养老金，存入账户的钱就能从应纳税所得额里“划出去”，从而减少当年需要缴纳的个税。

投资环节，计入个人养老金资金账户的投资收益，目前暂不征税。这为账户资金增值提供了空间，投资者不用担心收益被征税，进一步鼓励了大家将资金投入个人养老金账户进行长期投资。

领取时，个人养老金不并入综合所得，而是单独按照3%的税率计算缴纳个人所得税，缴纳的税款计入“工资、薪金所得”项目。

为了方便大家理解，我给出个案例。

假设某A年收入为50万元，全年缴纳社保公积金共计7万元，并且按照每年1.2万元的限额缴纳了个人养老金。

在未缴纳个人养老金之前，应交个税为：(50万元 - 6万元(免税额) - 7万元)×25% - 31920元(速算扣除数) = 60580元

而当缴纳了个人养老金后，应缴个税为：(50万元 - 6万元 - 7万元 - 1.2万元)×25% - 31920 = 57580元

两相对比，每年可以节省3000元的个人所得税。

每年进账户的1.2万元在存入和投资期间一直处于“应交但未交税”的状态，相当于是把眼下你要交的税，往后“延迟”，等你退休后再进行“补缴”即可。而且补缴时税率也不高，只需按最低档3%的比例来补缴。

因此准确来说，个人养老金不是“免税”而是“税延”，在工资、薪金所得项目中，税率最低为3%，最高可达45%。而在领取个人养老金时，仅按最低档收税，这意味着在领取阶段个人享受了极低的税率优惠。

要是你原本税率高，比如20%或者30%，那选交个人养老金就特别划算。

这正是政策设计的初衷，通过税收递延优惠(存钱减税、领取低税)，鼓励长期养老储备。官方已经明确，常规退休金(基本养老金)仍享受免税待遇，不受此政策影响。

个人养老金的税延政策设计本意是存钱越多，省税越多。

年收入10万的，每年最多省360元；年收入30万的，每年最多省3000元；年收入100万以上的，每年最多省5040元。

但这也暴露出了个人养老金体系更深层次的问题：政策究竟鼓励谁养老?

由于提款时需缴纳3%的个人所得税，收入未达个税起征点的人，参与进来反而会多承担3%的额外税负；即便对于适用3%最低档税率的纳税人来说，资金长期锁定，活钱变死钱，投资范围还受限，实际收益未必划算。

那么，究竟有多少人能真正从个人养老金政策中受益呢?

根据国家税务总局2023年个税汇算清缴数据，我国综合所得纳税人中，超过70%的人无需缴纳个税；在剩余不到30%的实际纳税人群中，60%以上仅适用3%的最低档税率，纳税金额较低。

这意味着，真正能享受个人养老金税收优惠的群体，仅占全国劳动者的10%左右。

即便在这个有限的受益群体中，考虑到资金长期锁定的时间成本，实际参与缴纳的人数也会更少。

虽然目前通过银行促销和国企推广，开户人数已突破1亿，但实际缴费人数仅占20%，约2000万人左右，人均年缴存金额约2000元。可以预见，未来真正会持续使用该制度的群体规模，大约就在2000万至3000万人之间。

从年龄维度来看，距离退休年龄越近，个人养老金的吸引力就越强。以年收入超过30万元(所得税适用20%及以上)的50-55岁人群为例，他们只需将资金锁定5-10年，提取时按3%的税率缴税，每年就能获得数千元的税收减免。这类人群主要集中在1970-1975年出生，除了经商的企业主和个体户外，能在这一年龄段获得如此高收入的，大多是收入稳定的体制内人员。

这也暴露了当前个人养老金制度面临的一个结构性矛盾：目标群体与实际参与者的错配。

最积极参与的群体，恰恰是养老保障已较为完善的高收入人群，然而理论上，这一制度最契合的应当是那些具备一定理财能力、高度重视养老保障，且对资金安全性有着较高追求的中产阶层。可现实情况是，这群人更多的是望而却步。

制度设计的初衷是通过税收优惠吸引长期资金入市，形成“参与人数增加→ 资本市场发展→投资收益提升→ 吸引更多参与者”的良性循环。只要能顺利开个好头，“个人养老金”哪怕作为理财产品也能长盛不衰。

但对当下还在资本积累的年轻人而言，强制储蓄的资金要等到退休(可能延迟至65岁甚至更晚)才能提取，这就相当于说资金会被锁定长达三四十年，但什么时候能领还得看国家政策，这种不确定性大大降低了参与意愿。

更关键的问题在于，“个人养老金”这款理财产品的推出时机太不理想——恰好赶上了楼市“深度调整”、居民家庭主要资产(60%-70%为不动产)价值缩水，中产群体更关注当下资产流动性，社会上讲究的是开源节流，而非长期储蓄。

再加上，各银行为完成业绩，将“个人养老金”当作“理财打包袋”被动推广，这让不少人觉得“个人养老金”是90年代强推社保的翻版，是以政策引导百姓存钱养老，本质上是为弥补当年社保宣传的不足。

这项政策确实带有一定的历史补偿性。改革开放后，全社会从“大锅饭”向市场经济转型，政府通过“只要交社保，国家来养老”的宣传建立了全民社保体系。但随着人口结构变化，单靠社保已无法满足养老需求。个人养老金制度的推出，实际上是在提醒大家养老不能全靠国家，要自己负责。

其实国家说的没错，养老原本就不应依赖单一支柱，国家兜底+企业补充+自己负责才是世界主流。只是这次政策推出时机不对，加上推广仓促，细节问题都没考虑周全就全面铺开了，这才引起诸多非议。

因为推出的时机不对，所以总有人批判中国的个人养老金是对美国401K的拙劣模仿。

但翻开历史就会发现，所有伟大的制度在起步时都饱受质疑——今天的个人养老金，像极了40年前刚出生的401K。

最初推出时根本无人问津，直到企业祭出“401K公司匹配缴费”的大招——员工存1块，公司就送1块。

这种“不存白不存”的羊毛诱惑，才让第一批美国人半推半就地上了车。

绝大多数美国上班族，其实也没有专业的投资理财能力。

针对这种情况，美国金融机构开发了两种“无脑”投资方案：

1. 终身定投指数基金，长期平均年化收益可达10%；
2. 生命周期基金：年轻时自动配置更多股票，临近退休自动转为债券等稳健资产。

以25岁年收入5万美元的年轻人为例，只需每年存入10%收入，按7%年化收益计算，到65岁就能轻松积累百万养老金。

讲到这里，很多人要说401K的成功完全依赖美股长牛，中国资本市场波动大，根本不适合复制。但历史告诉我们一个反常识的事实：在401K诞生前，美股同样被诟病是大起大落的赌场。但正是401K带来的万亿级“耐心资本”，才造就了美股的长牛奇迹。就像鸡生蛋还是蛋生鸡的问题——不是美股涨了人们才买401K，而是全民持续定投401K推高了美股。

纵观历史，无论是何种投资类型，个人投资者总是免不了追涨杀跌，总想买在低谷卖在高峰，但往往事与愿违，90%的投资者都高估了自己的择时能力。

反倒是“耐心资本”长期收益更高，如果把时间拉长，你就会发现自主投资账户的收益普遍跑不赢养老金账户。

所以，想尝试“个人养老金”的人，不如先忘掉“个人养老金”这个名称，就当是个长期理财账户，用“薅羊毛”心态看待税收优惠。

就像20年前没人相信社保能普及一样，今天的质疑终将被时间证明。给新事物一点时间。当我们的个人养老金规模突破万亿时，中国资本市场也会迎来自己的“耐心资本”红利。

记住巴菲特的话：“定期投资指数基金的业余投资者，往往能战胜专业投资者。”
