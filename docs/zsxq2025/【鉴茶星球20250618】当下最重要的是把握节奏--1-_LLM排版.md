
# 当下最重要的是把握节奏

当下无论是实体经济还是资本市场，
基本都是跷跷板走势，
它是由两个映射，
和若干个跷跷板走成。

一，映射，
## 映射一(宏观):
K型上沿，高科技产业一线和二线核心城市，资产和房产是美国映射，优质的东西越涨越高，上不言顶。
k型下沿：中西部、资源枯竭和人口流出城市，是日本映射，主要任务是化债，防止资产负债表衰退和留动陷阱。
上面的思路，是通过科技突破，最终让东部拉动中西部。

## 映射二：(微观)
供应链企业是美国龙头的映射， A 股是港股的映射。
东西缓和，利好外贸，出口，果链，
东西紧张，利好自主可控，华为。

二，跷跷板，
股市上因为资金量不足，此起彼伏。
1,科技和药呈跷跷板，
药强科技弱，科技强药弱。
药是创新药，目前看中期顶部，科技开始有回流。
## PCB、 光模块/MPO
科技目前大病初愈早期，主要集中在内，这几个内部也是此起彼伏小跷跷板，但互相形不成吸血。
具体是哪些讲过多遍不赘述。
如果扩散，会往Al眼镜，机器人。

2,新消费和老消费呈跷跷板。
## 老消费满足生存生活需求，地产白酒家电家居，
新消费是泡泡玛特和蜜雪冰城代表的情绪价值，还有珠宝黄金医美等。
港股新消费者有业绩，大A没业绩，目前真是泡沫且筑顶，新消费抱团瓦解，资金才会往老消费。

3,公募重仓股和私募重仓股跷跷板。
公募同时也是重仓，那些半导体、汽车，大蓝筹等。
私募重仓，包括量化和游资在内，在石油，军工(无人机、雷达),稳定币(蚂蚁系是核心)、可控核聚变(中洲，合锻，融发等)、稀土永磁等题材股。

稳定币完全跟美股和港股走，
这两天伊朗，可能又会把油运和航运短炒—波。

4,有业绩的和没业绩的跷跷板。
赛力斯之类的业绩非常好，你去买M8 订不到车，我们之前谈的牛磺酸铲子永安有什么业绩，涨的六亲不认。
五穷六绝，说的是跟基本面没关系。
随着年中报将临，有些资金会从没有业绩的转战有业绩的，比如游戏(巨人、冰川、华通等)、猪肉(小明和牧原)。
但注意，跷跷板4中，猪肉主力在缓慢建仓，不要上头。

上述是最近多篇的总结，一个好另一个就不好，
总之两个要点，
第一，看图很重要，在5日平台之上操作，5日上穿10日进，下穿10日走。
第二，不要追高，特别是当天涨高了你去追，电风扇你只能埋伏，不能跟着转，节奏不能踏错。
