
# 诺贝尔经济学奖：欧洲的新反思

欧盟的产业观念正在转变，从过去轻视制造、重视知识产权转向不惜损害市场秩序争夺制造业资产。这一转变在欧洲的多个政策中有所体现，例如欧盟考虑出台新规，强制进入欧洲市场中国企业要么向欧洲转让技术，要么与欧洲企业组建合资公司。

这一政策是《工业加速器法案》的一部分，预计11月出台，主要针对电动汽车、电池等关键科技与制造业领域。欧盟一位贸易专员公开表示，欢迎外国直接投资，但前提是真正的投资，这意味着要在欧洲创造就业、为欧洲增加价值，并向欧洲转让技术。这句话显然指的是中国80年代开始的“市场换技术”发展策略。

欧洲的技术困境源于过去五年来全球科技产业中出现的全方位技术落后，就连清洁能源和汽车产业也概莫能外。欧洲面临通胀与衰退并存的局面，工业发展被中美压制。德国经济停滞已经三年多，8月份工业产值环比暴跌4.3%，倒退回了2005年的水平。

北欧volt电池公司向瑞典法院申请破产保护，这家公司诞生时被视为欧洲的希望，但9年来烧光了约合1000多亿元人民币的投资，最终因产能拉不上去、质量拉不上去而遭遇失败。而中国的宁德时代、比亚迪等企业已经带着遥遥领先的电池技术进入欧洲市场，并响应欧洲要求在本地兴建电池与汽车的生产基地。

欧盟环保组织“交通与环境”的高级总监表示，如果欧洲不提出更多要求，将无法从中学习技术，他们只是一个组装厂。这表明本土产业竞争力的焦虑与政府产业部门达成了一致。

瑞典皇家科学院宣布将2025年诺贝尔经济学奖授予两个研究团队，表彰他们“对创新驱动型经济增长的阐释”。其中经济学家莫基尔主张知识要自由流动、互相验证，制度要打破知识的垄断阻隔、鼓励试错，才会让欧洲工业革命那种级别的辉煌发生。经济学家阿吉翁和豪伊特证明了“创造性破坏”的正当性，即真正的经济增长来自持续的创新竞争，而创新竞争必然导致旧产业的破坏。

诺贝尔经济学奖的评选带有欧洲政经界的世界观和反思倾向。获奖理论反映了欧洲对美国和中国在技术上领先的不满，以及欧洲需要醒来、不能让美国和中国主导技术领域的呼声。

欧洲的应对方式是用一切手段降低自己创新的门槛。欧洲的标准是一致的：本土利益至上。诺奖把经济学奖颁给了“创造性破坏”理论，并不是在批评欧洲的贸易保护政策，而是为了欧洲的“创造性”，不妨“破坏”点什么。安世半导体的控制权、在欧中企的核心技术、秉持了多年的自由市场规则，都在答案里。

对中国企业而言，欧洲市场正在重写一套非对称规则。中国企业未来去欧洲，需要在宏观层面，仰仗国家力量参与规则的博弈。今天的出海已经不再是单个企业的孤军远征，而是国家与企业的双向配合。
