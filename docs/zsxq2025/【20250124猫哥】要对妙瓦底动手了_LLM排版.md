
# 要对妙瓦底动手了

## 嘉宾供稿
1月21日，昆明举行了“湄公河-澜沧江执法中心”执法安全合作高官会，共有柬埔寨、中国、老挝、缅甸、泰国、越南六国参加。

与会期间，中国特地和缅甸、泰国进行了另一场会谈，就合作铲除缅甸妙瓦底电诈窝点交换意见，研商对策，达成积极共识。这个消息意味着，要对妙瓦底动手了。

我们知道，要打击妙瓦底，泰国和缅甸是关键，如果他们不同意，就很难达成。

## 问题来了，具体怎么合作打击呢?

先说缅甸。我早前写过文章，表示缅甸已经同意组建中缅联合安保公司，并修改了法律保障活动，这个安保公司，会是我们的有力抓手。

作为回报，缅方在公司里占股70%，虽然他们只有分红而无实际决策权（互联网AB股的玩法），但也是不小的糖，我们打击电诈越多，就越需要壮大安保公司。

这样一来，缅甸军政府获利就越多。其实这颗糖，敏昂莱不想吃都不行，目前的军政府，战场上节节败退，外交上空前孤立，唯一还愿意和他说几句话的，就是中国了。

尽管我们也不喜欢军政府，但外交的宗旨，从来不是喜欢与不喜欢，而是利益。就是没有永恒的朋友，也没有永恒的敌人，只有永恒的利益。

接着是泰国。和泰国的合作内容，主要是我们要求泰国，断掉妙瓦底各个电诈园区的供电。

泰国旅游业在“王星事件”后损失很大，已经有数万个订单被取消，旅游业是泰国的支柱之一，中国游客又是泰国旅游业的支柱。

如果再不积极打击电诈，中国游客大量减少，许多从业者就不好过了，反之，积极打击电诈，则可以带动大量中国游客进入，让泰国受益。

和缅甸类似，在施压的同时，也给一颗糖。

## 妙瓦底虽是缅甸国土范围，但供电、网络等基础设施，却长期是泰国负责，背后是泰国和克伦族武装的密切联系，妙瓦底的军阀苏奇督，就是克伦族人。

克伦族向来有独立倾向，不想成为缅甸的一部分，而泰国恰好和缅甸存在地缘冲突。历史上，两国战争不断，泰国有制衡缅甸的需求，便扶持克伦族。

克伦族两支主要武装--克伦民主佛教军和克伦边防军，武器来源大部分是泰国，不仅供应武器，泰国还向克伦族地区，大量供应水电等基础服务。缅甸政府就是想供应，克伦族也不愿意用。

这样一来，泰国就掐住了妙瓦底的脖子，只要在供电等基础设施方面做做文章（比如长时间检修），就能让妙瓦底很难受，我们的要求正是这个。理论上，妙瓦底可以自行购买发电机和燃料，然而成本会上升许多，而且私人发电机的稳定性，不可能和正儿八经的供电系统相提并论。

只不过，摆平妙瓦底还是存在一些难点的。

首先，缅甸军政府对妙瓦底的管控力，近乎于没有。

2010年，苏奇督被军政府收编后，他的私兵名义上改编为“克伦边防军”，实际上还是自成一派，不听号令。

缅军在妙瓦底的兵力，长期以来只有一个连左右，这点人数属于象征性存在，谈不上控制力，缅军用了许多手段，试图增加驻军，但都未成功。

2021年内战后，军政府压力山大，更不愿意惹苏奇督，苏奇督看到了机会，宣布脱离缅军编制，不再自称“克伦边防军”，而是改称“克伦国民军”。

双方连名义上的隶属关系，都不存在了，军政府对妙瓦底的影响力更弱，妙瓦底完全成了苏奇督、苏茂、苏丁温三人的“独立王国”。

所以，军政府给我们开开口子自然是好，但也只能帮到这了，别的真帮不上。单靠“中缅联合安保公司”，能不能完成任务是个挑战。

目前，“中缅联合安保公司”的规模有多大，还不好说，不过可以肯定，人数多了吧，有瓦格纳这种前车之鉴，总归是不太好的。

人数要是少了，又压不住敌人。要知道，克伦族搞电诈的可不止苏奇督，另一支武装--民主克伦佛教军，在电诈方面也大差不差。

真打起来，两家抱团的概率极大，他们加起来，可有两万人之众，当地很多民众也是受益于电诈产业的，所以民意支持不可忽视。

其次，泰国方面的配合度能有多高，也是个问题。

过去，泰国政府不是没有切断过妙瓦底的供电，但每次都是断几天又恢复了，2024年6月，泰国就曾切断过妙瓦底的供电，不到一个月又恢复了。

究其原因有两个，一是妙瓦底的电诈，本就与泰国方面有着千丝万缕的联系，许多泰国人尽管不直接参与电诈，却也受益于此。

比如绑架王星的车，轻易通过了湄索的边检，据后来的资料显示，泰国的边防人员明显放水，连象征性的检查都没有，就放行了。

不消说，平时估计没少收电诈集团的“红包”。

二是泰国西部，不是他信家族的传统领地。我们知道，当下的泰国总理佩通坦，来自他信家族，家族的传统领地在北边的清迈一带，对于泰国西部边境，他信家族没有太大的影响力。

皇室，还有各种地头蛇，才是这里的主宰。

他信家族本就和皇室关系不佳，之所以能重新上台，纯粹是新崛起的前进党过于激进。皇室在权衡利弊后，觉得他信家族虽和自己关系不好，好歹没那么激进。

双方的合作其实比较脆弱。而打击电诈，会实实在在损害皇室和地头蛇们的利益，没有人愿意轻易割自己的肉，脆弱的合作关系，很容易断裂。

只不过，尽管存在种种困难，但对于没有把握的仗，我们是不打的。对于没有把握的事情，我们向来也是不会把调门抬太高的。

既然开了三国合作会，既然把调门定得不低，还在通告里点出了妙瓦底的名字，就意味着我们会排除万难，使用种种手段达成目标。

毕竟妙瓦底不是小卡拉米，大规模的电诈、绑架活动，已经对我们以及周边国家，造成了很大的负面影响，这颗毒瘤是时候铲除了。
