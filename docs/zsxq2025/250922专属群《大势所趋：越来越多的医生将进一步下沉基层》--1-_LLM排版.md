
# 大势所趋：越来越多的医生将进一步下沉基层

## 三明医改的新思路

三明医改针对医疗资源不平衡问题，提出了解决方案：

1. **建设县域医共体**：整合县、乡、村三级医疗机构，打破管理壁垒。将县综合医院设为总医院，统筹管理下属乡镇卫生院和社区卫生服务中心的人、财、物等资源。

2. **推进分级诊疗改革**：通过建立转诊机制和利益共同体，引导患者就近就医。常见病、小病在乡镇卫生院首诊，疑难重症通过绿色通道转诊到县总医院，稳定后康复治疗再转回基层。

## 改革成效

三明医改取得显著成效：

- **医生薪酬提升**：公立医院在岗医务职工薪酬从2011年的4.22万元/年涨到2022年的16.34万元/年，医生平均薪酬从5.65万元/年涨到20.11万元/年。
  
- **医保基金结余**：2011年医保基金亏空2亿多元，2012年改革当年结余为正。截至2023年底，职工医保统筹基金历年滚存结余11.85亿元，城乡居民医保历年滚存结余3.62亿元。

- **药品耗材费用下降**：累计减少药品耗材费用支出近200亿元，采购药品耗材价格平均降幅50%。

- **居民医疗费用负担减轻**：城乡居民医保患者每次住院平均费用从4082元增加到5810元，但个人自付费用从2194元减少到1712元。实际报销比例从46.25%提高到70.53%。

## 医改核心

三明医改的核心是“利益的重新分配”：

- **淘汰中间商**：两票制减少中间环节。
  
- **压缩药企利润**：带量采购降低药企虚高利润。
  
- **压缩医院逐利性**：零加成政策和公益性定位。
  
- **消减医生灰色收入**：打破“以药养医”。

这些“省下来的钱”通过新制度重新分配给老百姓、医保基金、医务人员和医院，激活医疗正向循环。

## 未来医改趋势

中国医改正进入深水区，重点解决“怎么看好病”和“怎么看得起病”问题：

1. **优质医疗资源下沉**：推动优质资源向中西部和东北地区延伸，城市优质医疗资源向县级医院和城乡基层延伸。

2. **医疗巡回服务**：定期派遣医疗队到基层提供诊疗、宣教和培训服务，国家已组织82支巡回医疗队。

3. **医疗服务价格改革**：中央统一规范价格项目，不同等级医院职能分化，大型公立医院聚焦疑难重症治疗、医学人才培养和科研创新，基层医疗机构承担预防、保健、康复和常见病诊疗。

4. **医生执业流向变化**：专家下沉、资源调配成为常态，大学生乡村医生专项计划已吸引1万余名毕业生到乡村工作。

5. **医保基金转型**：医保基金成为强势的“超级支付方”与“价格制定者”，通过带量采购、DRG/DIP支付等掌握药品和医疗服务定价权。

6. **医药产业变革**：医药企业需通过成本控制或创新产品生存，行业将经历出清和更新，资源和市场份额向头部巨头和前沿创新药企集中。
