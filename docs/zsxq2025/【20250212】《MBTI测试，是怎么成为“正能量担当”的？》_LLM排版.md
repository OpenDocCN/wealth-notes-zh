
# MBTI测试，是怎么成为“正能量担当”的？

今天我们聊聊这两年特别火的MBTI。MBTI在年轻人里尤其流行，我们得到自己在接收应聘简历时，很多年轻人都会在简历中附上自己的MBTI测评。

MBTI，全称迈尔斯布里格斯类型指标。它的整个过程不复杂，是用4个字母来描述一个人的性格。

第一个字母，看的是你的注意力方向是外向还是内向，E代表外向，I代表内向。也就是咱们最常听说的，你是E人还是I人。

第二个字母，看你的感知维度来自感受现实的实感，还是依靠抽象的直觉，偏向实感的用S表示，偏向直觉的用N表示。

第三个字母，看的是你的判断维度，你是靠逻辑做判断，就用T表示。靠情感做判断，就用F表示。

第四个字母，是在生活实践维度上，你是倾向于严格遵循秩序，还是倾向于灵活安排事项。前者用J表示，后者用P表示。也就是我们常说的，J人，P人。

这4组特质彼此组合，就把人群分成16个阵营，也就是16个人格类型。

这个测试有多火呢？小红书上的MBTI话题，有44亿次浏览量和3500万条讨论。微博上的MBTI话题，有52.5亿阅读量，将近200万条讨论。每次MBTI更新题库，你就能看到有人呼朋唤友重新答题，确认自己的阵营。

MBTI还成了很多商业营销的抓手。比如2024年，天猫发布了一份“居家美学白皮书”，就从性格的角度分类，为不同性格的人设计了家居风格。再比如，海底捞在北京的一些门店，推出了I人和E人专属的等位区域。再比如，去年年底，很多时尚杂志在预测新一年时尚趋势的时候，也推出了针对不同性格人群的搭配模板。

但是，仔细想想，这个现象其实挺奇怪的。别看MBTI的诞生已经有将近一个世纪的历史，并且号称是源自心理学家荣格的心理理论，但实际上，它的科学性和有效性，一直遭受到学术界的诟病。专业人士的普遍态度是，这个测试，做着玩儿可以，但是真要当真，它还不合格。MBTI还有一个名字，叫“赛博玄学”。但是，反过来，它的流行却是实打实的。

那么，为什么偏偏是这个貌似有点不专业的测试，会成为全球性的爆款呢？

今年年初，天使投资机构青山资本发布了一份年度报告，报告的主题叫《MBTI流行原因及创业相关性》，里面有很多有趣的发现。从中你也能看到，一场流行到底是怎么从无到有一路爆发的？接下来，咱们就展开说说。

首先，咱们从MBTI的流行史说起。MBTI最初是作为职业性格测试出现的。在成为社交谈资之前，在很长一段时间里，MBTI都是用来给求职者做职业性格测试的。

1940年代，二战中男性大量参战，女性开始参与社会工作。但是，当时很多女性没有接受过系统教育，怎么选择合适的工作？有一对英国母女就想到，可以通过性格测试来做工作分配，她们基于荣格的心理学理论，设计了MBTI性格测试，帮助当时的妇女寻找匹配的工作。后来，经过反复实践和修订，这个测试在政府、学校和企业管理开来，广泛应用在各种职业测评中。

直至今日，MBTI都只是职业测评这个垂直方向的测试。当然，它的覆盖面很广，全球500强公司中超过80%都在用MBTI测试，十几年前，它的版权公司就能靠MBTI年入2000万美元。

2000年，MBTI测试被引入了中国，但是，在大众层面上的火爆，是直到2022年才出现的。当时在北京冬奥会上爆火的谷爱凌，在一次采访中提到了自己的MBTI，一下就在社交媒体上引发了非常大规模的讨论。假如你观察这个词在百度指数的趋势，会看到它在2022年上半年，有一轮非常陡峭的爬升。

当然，名人效应只能算是一个引爆点，不能支持它长久的火热。关键在于第二点，也就是，MBTI测试，天然具备流行的潜质。因为它足够简单，且足够积极。任何人都可以用成本极低的方式，从中完成一场正能量满满的自我确认。

首先，MBTI足够简单。简单到只需要回答一些问题，就能获得答案。这比多数心理测试要简单得多。比如今年年初，剧集《猎罪图鉴2》带火了一阵“房树人”心理测试。房树人，是专业心理咨询中最常见的测试之一，你在一张白纸上画出房子、树木和人物这三个意象，咨询师会根据你的画，来判断你潜意识中的心理状态和人格特征。显然，这个房树人测试的复杂程度要超过MBTI。至少它需要你专门抽时间画一幅画。

同时，MBTI的表达形式，是大脑最喜欢的分类式表达。这恰好能满足我们人类的大脑对有序信息的偏好。人天生就喜欢类别化的信息。而且，因为分类很笼统，所以对应描述也很模糊，你怎么解释都有道理，怎么解读都能说得通。这就进一步增加了这个测试的兼容性。

其次，MBTI是个只说好话的测试。没错，只说好话。不管你分到什么阵营，它都只对你做出积极的描述，只突出你的闪光点，几乎不透露负面信息。要知道，人大都是自我感觉良好的。据说心理学家曾经做过实验，问学生觉得自己的聪明程度能不能排班上前50%，几乎所有同学都认为自己可以。

说白了，简单的分类，加上正面的描述，就让MBTI成为年轻人里的正能量担当。
