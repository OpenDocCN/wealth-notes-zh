
# 加拿大移民是全世界近年来最大的坑之一

加拿大移民是全世界近年来最大的坑之一，至少对于国内的中产来说。加拿大的大城市房价，早已经被80年代香港移民，90、2010年代中国大陆移民轮番炒上天了。国内的中产过去，只会被沉重的房价和房租压得喘不过气来。北美房价最贵的10个城市里，加拿大居然占了4个。要知道加拿大的人口和平均经济水平，都远不及美国。

这个国家更恐怖的是名副其实的“万税之国”，你能够想到的任何方面，都给你征税，中产很容易被榨干。你在家里用水、排污，供水费用会被征收消费税。许多城市的水费账单还单独列出了“污水处理费用”或“排污税”(Sewage Tax)。在一些城市的市中心停车，你得额外支付“停车税”。冬天在家中开暖气或加油时，你同样需要缴纳“碳税”。

关于加拿大的“万税”故事，讲几天几夜都讲不完。

治安方面，恶化严重，枪击和抢劫越来越常见，谋杀率最近才创了近20年新高。社会秩序方面，倒退严重，印度人的大量涌入让加拿大各方面的秩序都不断倒退。国内中产过去，基本上都会被收割的一干二净。最后在寒冷的冬天里，留下悔恨的眼泪。
