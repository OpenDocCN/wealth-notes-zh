
# 水原星球5.16：恭喜了，学历低反而赚钱多！

最近微博官方约稿，除了房子，让我写几条关于高考和大学的内容，那就再写写。考上985的毕竟是少数，我来鼓励下那些考的没有那么好的同学。有人考370，有人考730，但在一个30多岁的“社会人”视角中，两者都没差别，毕竟今天很多能过得好或赚到钱的人，除了北京上海体制内或者互联网金融大厂，赚到百万以上这个级别的人，大多数是学历一般甚至不好的。

比如今年三个千亿市值公司老板，霸王茶姬初中，蜜雪冰城初中，泡泡马特三本，快手前十没一个大学生，抖音前十估计可能一两个。有人说福布斯U30上一堆高学历的年轻创始人，是的，但你不知道的是福布斯U30上，年净利润上百万的实际上没几个。

人生是人生，高考是高考，假设人生是一场大富翁游戏，高考考得好的人，只是在人生的某个环节上，摇出了666，拿到了几个好的道具，而不是成了游戏最终的赢家；相反如果取得了阶段性的成果就得意觉得人生这局游戏十拿九稳了，最后一般会因为没有及时扩大优势，因为高估了自己，高考后反而平平无奇，考上县城状元最后出来还不一定能找到一份月薪1万的工作，更不愿意拉下脸早点找对象生娃搞家庭发育路线。

考370的人，认为高考决定命运，他的人生就自暴自弃；考730的人，认为高考决定命运，孔乙己的他，还要经历社会十几年的暴揍，等揍完了三四十岁也来不及了。

即便你考上清华北大，老师也是照着5年前的PPT课件念书，人能不能学到东西，能力能不能提高，不是靠高考分数，而是靠自己。人生的每一步，都是起点，高考也是，考上985也好，考证成功也好，升职加薪也好，赚到100万也好，都不是成功，只算暂时领先，而不是最后的胜利。

所以，高考只是一个起点，只是人生中的一步，你的人生才刚刚出发，最后能拿下什么结果，全凭你自己。

但是，高考并非毫无意义，它能带给你两个好处：
1. 高考最重要的意义，是大部分人必须接受的战场，是改变人生的事件。
2. 如果能考上一个好大学，能给你提供很多好处。一个好的大学，最大的好处是两点：
1. 提供一个好的环境，包括校园环境和城市环境；
2. 给你更多自信。

但有意思的是，很少有同学明白高考的意义，很少有人知道，上大学真正的好处是什么。即便考上985的人，大部分也只是在校园里浪费了4-6年的时间。我为什么比一般人强一点，原因是我上大学第一周就“辍学”了，跑到清华北大去蹭课，发现最牛逼的学校也学不到东西，还得靠自学，但是学校的很多活动，人脉，资源，开阔的视野，都可以想办法变成自己的东西。

所以即便你考370，你和清华北大的学生的资源差距，只差一张火车票+一份勇气，相反，985的学生反而不一定知道自己的优势在哪，毕业后脑子里还是学生思维。越早去闯荡北上深杭广的人，越早在社会上实践的人，才能学到真东西(所以，建议你大一就开始实习或者做自媒体)。真正的大学，不在学校，而在社会。你是16年义务制的好学生，你就是社会大学的差生。到了社会后，别人看的不是你的分数，不是学历证书；而是你能办成什么事，能帮我赚多少钱? 只有你做到的事情，赚到的钱，人生过得怎么样，才是你的结果。

所以，高考考的好，别停下，继续扩大优势；高考考的不好，哭一次这件事到此为止，赶紧去北上深杭广，闯一闯。
