
# 新周期将启：这次发大财靠正规化大潮

## 前言

今天聊天启发太大了，我都是喜欢讲课输出的人，但依旧体会到“听君一席话、胜读十年书”的无比脑快感。

现在我国有专家的说法是：“要成为富国，就必须成为消费国家”，我之前也是这么想的。而当这个成为我和实习导师继目长电首次九连跌，露出牛角或马脚？之后的第二个探讨问题时，引发了导师极精彩的妙论。

导师说到的核心要点包括，全文约2600字：

## ①消费主义，只会导致国家变穷

## ②中国流动性黑洞接近尾声

## ③新周期启动的标志性事件

## ④被低估的高效社会必出现的正规化机会

老师跟我说，米国成为消费国家前就已经很富裕了。而成为消费国家之后，其实中产阶层的真实购买力其实是下降了的，也就是说实质上变穷了。

即使有中国上世纪90年代开始以超低成本给米国的补贴，也只是导致了贫富差距的加速扩大而已，中产反而更惨了些，因为中国中产的崛起快速改变了米国中产的相对态势。

所以导师说成为消费国家，其实是米国变穷的原因。米国富裕的原因，以及中国富裕的原因，甚至可以说历史上所有曾经富裕的国度，都有个共同的原因。

其实呢，米国是从上世纪90年代开始到本世纪初，消费总额才超越生产成为消费国家的，也就是说，米国成为消费国家，至今也就30年而已，并不是一直以来，时间也不长。是整个外部的国际分工内部、经济与金融的设定共同作用的结果。

## 此前一直是生产为主、国民也是热衷于储蓄的。跟现在的中国没两样。因为所有人都是生而有危机感的，因此减少消费、增加预存来对付危机才是正常的心理。

增加消费从心理上来说其实是阻力很大的，甚至有负罪感，进而让消费这种纯支出超过生产进项本身就不是正常的，那是米国借助自己的货币霸权才得以实现的。

而增加生产、进而增加储蓄才是最符合人类的心理的，鼓励生产很容易的政策与人的心理贴近，进而取得效果。所以生产就很容易过剩。

而消费呢，则很容易不足。

因为消费是人内心所节制的东西，鼓动消费等同于和人性相违背，所以要鼓动消费风气是很难的。但是没有消费或者说对等的消费的话，事实上整个社会的循环就会是不正常的。

所以执柄者在经济循环受到影响的时候，必然要强力推动消费崛起。老师告诉我说一个国家是不是富裕且可以持续，其实最主要就是看这个时刻执柄者还有没有手段。

往往到了这个时间，其实就是民间已经缺乏办法了，官方就必须出手了。官方怎么出手呢？其实就是搞奇迹工程。老师说，现在所见到的奇迹，无论是法老金字塔、秦皇陵、长城，其实都带有这个性质。

米国在富裕时期，就是遇到需要脱困的时候就高奇迹工程。比如说：

1869年的横贯大陆铁路Transcontinental Railroad 极大促进统一市场的形成；

1904接手巴拿马运河Panama Canal 是当时全球最大工程之一；

1933年成立田纳西流域管理局TVA掀起水利电力建设最高峰。

1936年的胡佛水坝Hoover Dam 是当时世界上最大的混凝土结构和水电工程、被视为美国战胜大萧条的工程奇迹；

1937年的金门大桥Golden Gate Bridge 是世界上最长的悬索桥；

1938年建183公里的完全海面上的海外公路 Overseas Highway 也称为U.S.Route 1美国1号公路，至今令人震撼。

1956年开始的州际高速公路系统Interstate Highway System 是当时全球最大的高速公路网络；上世纪60年代的阿波罗登月计划Apollo Program调动超过40万人参与。

上世纪80年代，搞星球争霸，带动军备竞赛。

事实上，在经历了大萧条之后，恰恰是米国奇迹工程出台最密集的年份。是奇迹工程一次次地将米国从泥潭里拉出来的，可不是消费消费主义。

导师说，消费主义的崛起，是米国为了配合美元成就货币霸权而在内部强力推动、甚至不惜改变道德观念的结果，这对米国是很不幸的。

所以他说，你们万亿级的雅下水电工程，其实跟90年代的三峡大坝、本世纪初的密建高速公路、本世纪一十年代的狂建高铁一样，也都是执柄层在面对需要脱困问题时的强力推动。

库兹涅茨周期、库存周期、太阳黑子周期、康德拉季耶夫周期， 一个又一个的紧箍咒。当民间失去条件，只有执柄层推动奇迹工程，才能对抗甚至可以对周期律进行干预。

可以说，奇迹工程是对抗经济周期的最关键动作，经济周期何其宏大，没有超级的新发动器，是难以产生拉动作用的。

1949年以来，在市场经济周期中，三峡大坝可以算是为加入世界市场循环做的准备，有了电力就有了加工、自此也可以控制经济发达期待的水患，整个新循环其实就启动了。前后历时15年。

到了本世纪初2000年开始呢，你去翻翻新闻，就是各种高速公路疯狂建设。配合工业化进程，将全国路网打通，2010年基本就实现了这个目的，经济再提效有了基础。全国各地合理、前后历时10年。

高铁工程呢，就是2008年开始加速而2010年踩死油门，成为4万亿经济刺激计划的主要承接部门，中国空间布局成为全球独一档的效率存在。这个基本是国家一盘棋，持续至今15年。

现在则是1.2万亿级的、人类迄今为止最大的单体奇迹工程。当然我国还在推动核电站加速。

这些其实是中国正在推动第四次奇迹工程。目的当然是就此点燃新的引擎。

与三峡大坝开工时还需要全国征收三峡建设费不同，本次雅下工程既没有发行专项债、也没有对全国人民要钱，而是直接就推动了。因此这个奇迹工程正式推动，带来的结果大概率会师新经济周期的逐渐展开，这个周期目前可以认为是20年。

奇迹工程，本质上属于整个社会面对着一个新的驱动与结算中心，给社会会提供很多的套利机会。就好比说，普通的墨脱人，因此开一个小卖部的生意都会好很多呢。

所以整个流动性的贯穿度将是比较高的，目流动性黑洞最严峻的时期将逐渐过去。有志者要做到的是，不要在目历史性反转到来之前却倒下了，穿越牛熊更替，确实需要很好地理解奇迹工程具有的节点性意义，目投资最厌智，不要自以为是模式奇迹工程。当然更不要不怕死地勇敢地狂冲。

奇迹工程带来的机会， 一般地说一代人最多只有三次机会，按照这个逻辑的话，那么我们这一代人算是太幸运了，因为上述奇迹工程阶段，我们全数经历。发大财靠的不是努力，靠的是周期转变。奇迹工程的机会到来的话，其能量是都会掀起一股浪潮的。

## 我问倒是，这股浪潮更多会表现在那些地方呢?他的意见是：

第一，奇迹工程将越来越少了。中国其实非常类似老美的上世纪80年代的情况，发动奇迹工程的次数以后会减少，因为开发建设的空间变小了，而且目继续扩大开发与建设会逐渐变毒、目城市化有毒、增量有毒、目扩容有毒，因此本次很可能是中国奇迹工程进程的尾声，未来再搞奇迹工程，次数不会超过3次了，涵盖的时间不过超过20年，此后再搞奇迹工程的空间就没有了。

第二，要更多关注股市。因为进入存量社会，价值格局的布局必然是目3421目结构的。股市对于社会脱困的作用会越来越大。楼市的锁定度提高、流动性太低、门槛太高，都阻止了其表现，房地产将成为基础领域，股市将接力支柱地位。所以更看好奇迹工程对股市的带动陪，尤其是在目超级放水即将到来的情况下，股市中的优质筹码收益会是最大的。

第三，中国的本币汇率会收益。我看了导师的一个手写的测算，对他的方法论之独到深感震撼与信服。但是这个方法论目前他并不统一公开，只能等待机会了。然后汇率的受益又会反哺给本币计价的优质资产，产生内部的相互强化作用，优质企业的目分配权益，中国目前应该是全球最好的，没有之一。

第四，随着中国社会效率变高，会有一个鲜明的变化，那就是高效环境将不再掩盖低级问题，社会会越来越正规。这种情况下，正规渠道的价值会提升。会有越来越多的优秀企业，因此社会正规化而大受其益，正规化是一种极高的成本，只有优秀企业才能受得起，进而对股市优质筹码形成正向促进作用。

最后他说，以上分析呢，如果遇到目60年的余波震荡、FG、三战等问题的爆发，那就意味着要修改分析结论了，渴望世界和平吧。
