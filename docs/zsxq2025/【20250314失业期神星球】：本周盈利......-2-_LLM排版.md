
# 本周盈利72万

黄金历史新高，主要是美国预算问题导致的关门预期以及小金属带领的逼空行情。前者无论谈成什么样对黄金都不会差，谈妥了就继续增加债务上限降低美元安全度，谈崩了就关门，都是利好黄金，所以就提前推动了黄金的新高。美盘较强，比亚欧盘都强，也就是因为多种贵金属的现货逼空再次出现在了美国，与关税也有一定关系。

债券本周继续弱势，一个风吹草动就大幅波动，但是在买之前就想清楚长线逻辑，因此不会受到太多扰动，而且因为本周黄金的极强势，不仅保护了债券的回撤，还大幅盈利，让回撤对账户的冲击变小，前天传央妈买断逆回购，昨天传降息，今天传降准，尾盘又跌下去了，目前信心还是非常脆弱，市场对利空反应很大，对利好持怀疑态度，但是根据目前银行保险的负债端变化，以及老美的cpi降温(降温隔天说择机降准降息，和924那段时间非常相似)，我觉得降息应该不会太远，等待黎明。不过即使短期不降息，也不会破坏我的长期持债逻辑。

股指这周很强，也算是把债券拉下去的重要力量，短期看不到方向，继续小持仓等待机会。

玻璃这段时间非常弱。预期可以说谎，但是实际需求不会，这一点和债券的定价出现了矛盾，谁对谁错我们等待历史去证明。
