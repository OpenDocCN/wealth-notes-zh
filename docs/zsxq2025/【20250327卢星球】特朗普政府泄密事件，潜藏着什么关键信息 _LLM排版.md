
# 特朗普政府泄密事件，潜藏着什么关键信息

## 文/卢克文工作室嘉宾星海舰长

2025年3月，美国上演了一出令人啼笑皆非的闹剧：一个记者，竟然被莫名其妙拉进了一个“美国内阁高官群”！群里大佬们肆无忌惮地讨论着国家机密！这件事曝光之后，美国政府瞬间成了全世界笑柄，大家都知道特朗普政府很离谱，但离谱到了如此地步，大家还真的没想到。不过，因为嘲笑的人实在是太多，反而无人关心这件事的背后潜藏着什么隐秘了。

## 1

首先，我们简要回顾一下到底发生了什么。

3月11日，《大西洋月刊》总编辑杰弗里·戈德堡在加密聊天软件Signal上，被一个自称是迈克尔·华尔兹（国家安全助理）的人加了好友。然后在3月13日，这个“迈克尔·华尔兹”把戈德堡加入了一个Signal聊天群，群的名字叫“胡塞政策协调小组”。然后戈德堡发现，副总统、国防部长、财政部长都在群里。戈德堡一开始觉得这只是个恶作剧，并没放在心上，但是到了3月15日上午11:44，“国防部长”突然在群里发了一份袭击也门胡塞武装的文件，然后下午大约1:45，也门真的被轰炸了！戈德堡这时才慌了，赶紧退了群，然后给华尔兹发了邮件告知此事，可能是为了防止自己被“人间消失”，戈德堡索性把这件事公开了，引发了轩然大波。

这件事果然引发了全世界的嘲笑，但是，我们并不能只满足于看乐子，更关键的在于，如何从这起难得的泄密事件中，观察出我们平时看不到的东西。

## 2

那么，这次拉错人，到底是啥原因呢？

目前最大的可能，是国家安全顾问华尔兹拉人的时候拉错了，因为杰弗里·戈德堡的名字缩写是JG，而美国贸易代表杰米森·格里尔昵称缩写也是JG，可能是华尔兹手一抖就点错了，把杰弗里·戈德堡这个记者给拉了进去。

看起来，这件事没那么复杂，就是个“低级错误”，但问题在于，后续的诸多细节，都难以用“失误”解释。

比如，华尔兹为什么要“主动添加”戈德堡？两个可能，第一，他把戈德堡当成了贸易代表杰米森·格里尔加上了。第二，他就是要加戈德堡。

第一种可能说不通，特朗普都上任俩月了，国家安全助理连贸易代表的好友都没加上吗？而且添加联系人需通过手机号或二维码，总不至于华尔兹连格里尔的手机也搞错了吧？那就只能是第二种可能，他就是要加戈德堡。

可是戈德堡一个杂志总编，何德何能让国家安全顾问主动添加？如果真是主动添加的话，肯定是有事找戈德堡，那么为什么在添加之后沉默了两天却不表露来意？
