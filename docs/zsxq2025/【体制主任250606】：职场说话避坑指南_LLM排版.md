
# 职场说话避坑指南

职场中，说话是一项重要的技能。无论在工作中还是生活中，说话都要注意分寸，避免说错话，造成不必要的麻烦。

## 说话的六大坏习惯

1. **显示自己聪明，抢话，接话茬，不听别人把话说完**  
   经常打断别人说话会显得唐突、没礼貌，显得不尊重人。在工作中，尤其是与领导互动时，更要避免打断对方，要耐心听对方说完，再以请教式、商量的口吻表达自己的想法。

2. **语速快、说话着急**  
   说话速度快往往表现出不稳重、不成熟。领导说话速度慢，是因为他们在职场上有多年经验，知道说话速度要慢，才能有充足的思考时间。

3. **总想反驳别人**  
   真正的聪明人不会轻易与人争论，尤其是在没有具体回报率的事情上。在职场中，要尽量理智、客观，以自身利益为出发点。

4. **喜欢打探隐私**  
   打探别人隐私是不礼貌的行为，会让对方感到不舒服。在职场中，要尊重他人隐私，不要随意打探别人的背景、家庭等信息。

5. **说话不看对象，没有考虑到听话人的现状、经历和利益，说话不注意对方的派系和人际关系背景**  
   在说话时，要充分考虑到听话人的背景和经历，避免触及对方的敏感点。在机关单位中，要注意不要谈论婚姻、学历、身高、长相、地域等敏感话题。

6. **不要动不动就提意见**  
   大多数人不喜欢听批评意见，尤其是领导。在提意见时，要把对人提意见变成对事提建议，以谦虚的态度提出具体的改进建议。

## 说话的原则

- **说外不说内，说远不说近，说己不说他，说好不说坏**  
  这是职场中说话的一个总原则，可以避免很多不必要的麻烦。

- **少讲一点真话，多讲一些空话，不讲假话**  
  君子讷于言而敏于行，多听少说，避免祸从口出。

- **沉默是金**  
  在体制内，规矩和秩序最重要。职位越高的人，说话越会云山雾罩，甚至说一半留一半，让你自己去揣度，去想象。这些话术都是职场前辈的血泪积累的结果。

奉劝各位：必须改掉别人一亲近，你就口无遮拦的毛病。奉劝年轻人：如果想在体制内行稳致远，谨记沉默是金。
