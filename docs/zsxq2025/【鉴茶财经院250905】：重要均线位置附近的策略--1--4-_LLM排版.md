
# 重要均线位置附近的策略

低位看逻辑，高低看图形，
我们讲的20日均线理论大家是知道的，
昨天，市场考验了20日线。

## 那么跌穿20日怎么办?不跌穿怎么办?

第一，如果不跌穿，
- 量能很大，强反弹，甚至反包，继续拿住，等待
新一轮新高。
- 量能不大，弱反抽，是减仓或者出掉杂毛的机会。
尤其是没业绩又前期涨幅巨大的二线国产算力芯
片，周一就讲要止盈部分。

之前反复说，低位看筹码，高位看图形，图形走坏了
跑，走稳了进，
要成为你的交易策略，不能脚踩西瓜皮滑哪算是
哪。

科技的震荡总要几天，抄底不是不能抄，你待站稳
5日平台后再进也不迟。

第二，如果跌穿，
1,过去6轮牛市中，阶段性跌破20日均线一共
99次，也就是不罕见。
2,随后，市场在T+5、T+20、T+60、T+180 胜
率分别为60%,67%,79%,92%。即短期消化后，
市场大多会重回涨势。
3,跌破20日均线后，平均回调天数在6.4天，平
均调整幅度为2.9%。
4,跌破20日均线后面临较大幅度回调有吗，有，
8%以上，仅有11次，平均回调11个交易日，平
均回调幅度为10%。概率不大，主要是政策收紧，
外部压力，ipo 开闸等。

因此策略显而易见，绝对高位要等待修复，当然没
走完，但可能也不是一两天立刻反转。

绝对低位是方向和风向所在，核心以大白马为主，
比如低pe 的电池，光伏设备，储能，饮料，同花
顺出海50等。

科技和他们是跷跷板，科技不强势，他们就会略强
一 点 。

当然也可能搞出一个新题材，比较绝对低位的大多
有瑕疵，所以今天能观察出很多东西。

对于绝对低位，这里不建议去搞该行业内的小票，
你别看它们涨的多，跌起来也是要命，
里面的绝大多数比科技杂毛还要杂毛，而且像电风
扇一样转来转去，都是一帮狗庄，低并不是你要涨
的原因，人家大白马涨，是因为估值修复。

为什么做事业搞投资，要多读传统典籍，因为那都
是让你心安静下来的东西。

让你走的更远，
《道德经》说，
飘风不终朝，骤雨不终日，
世界上事情，泰极否来，否极泰来，
无非循环而已。

淡定。
