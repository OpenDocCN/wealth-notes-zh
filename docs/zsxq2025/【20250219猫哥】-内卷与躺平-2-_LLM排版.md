
# 内卷与躺平
# 猫哥原创
现在国内应试教育内卷已经到了触目惊心的地步

昨天晚上与朋友吃饭，恰好一个朋友小孩在本地一所中学读初一，这个中学不算本地最好的，大概属于本地准一流这个层次。因为我的小孩明年也会上这所中学，所以就问了问学校情况。

这个朋友介绍了两个情况。
其一，小孩寒假结束之后立刻就要开学考试，考试内容初一上学期内容占80%,初一下学期内容占20%。我问，初一下学期不是还没开始吗?孩子都没学习过，怎么也要考试?答日，学校就是要考核孩子在寒假的自学能力!

我勒个草。
其二，孩子从大年初一就开始补课，这还不是找校外机构补课，而是到学校老师家里补课。
就这两个信息透露出这个学校学习氛围有多卷，孩子压力有多大。

这个孤例吗?
不是。
这是普遍现象。
我还了解本地某二流学校初中阶段学习任务有多重呢?反正孩子回家家庭作业至少要做5个小时，也就是说孩子不做到12点之后是不可能休息的。我朋友一个儿子上初中，每天做作业都是要到12点半之后才能休息，第二天7点要起床，每天睡眠时间只有6.5个小时左右。

这种高强度高压力的学习氛围导致一个结果，孩子上初中8个月，本来正是长身体的阶段，结果整整8个月，体重没有增加1斤，身高没有增加1厘米。
这TM还是二流学校。
最后朋友感慨，国内应试教育太卷了，他的计划是高中阶段将孩子送到国外去。
把孩子送出国我是绝对不愿意的，原因之前反复讲过，简单的说就是送出去大概率就回不来了，就算回来海外留学经历现在就业是减分项而不是加分项，但是孩子留在国内应试教育又这么卷，怎么办呢?

是选择适应还是选择躺平?
我个人的选择是躺平。
为什么?
首先，国内应试教育这么卷，归根结底就是希望让孩子未来读一个好大学，但是，现在社会上学历贬值趋势非常明显，好大学出来找不到工作或者找不到好工作比比皆是，现在的就业环境更看重个人技能而不是学历，当然好大学学历就业还是有一点优势的——而为了这点优势或者说为了日趋贬值的学历让孩子在青春期付出这么大的代价，我觉得是非常不划算的。

然后我觉得现在应试教育更倾向于将孩子培养成面面俱到的所谓综合性人才，打个比方，就是将孩子培养成一个基本没有短板的一个桶，但是真要做到这一点付出代价极大。而事实却是，真到了就业的时候，其实这个社会不需要每个人个方面能力都面面俱到，就业岗位更看重的个人某方面特长与优势。
比如宇树科技的创始人就自爆自己的英文很差，所以连考研都一直通不过，我就在想，假如一个人是没有语言天赋，或者说学习语言能力要差一些，真的硬要把英文成绩提上去付出的代价是非常惨重的，而最后得到的回报是非常小的。

所以，我现在对未来孩子规划是这样的。
学科校外培训一律不参加，作业如果太繁重也可以放弃一些不重要科目的作业，考试成绩差一点就差一点，老师有意见也无视，躺平最后能混个大学就成——在国内，或者说在大城市，如果不要求孩子一定要上985或者211,混个普通大学还是很容易的。
在这个过程中，我会让孩子强化特定的技能培训，根据我的孩子的兴趣与天赋，未来会往两个方面去努力。
一个是数学，奥数什么的一定要上，训练孩子数学与逻辑，尽量让数学与逻辑成为孩子的优势。至少也要形成缜密的思维习惯。
一个是绘画，这方面我的孩子还有一点天赋，所以可以培养一下，不求未来在绘画方面能有什么成就，至少在审美层面能有独到眼光。
中学阶段，我就打算给孩子培养这两方面技能，如果一定要说卷，就是卷这两方面技能，不用付出身心的代价，未来的结果大概率比拿到一个不断贬值的好大学学历可能更好一些。
