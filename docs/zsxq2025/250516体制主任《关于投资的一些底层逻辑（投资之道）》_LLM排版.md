
# 关于投资的一些底层逻辑（投资之道）

关注我公众号的大多是中产阶级，一个稳定安全的收入结构，需要包括：主业+副业+被动收入，越单一的收入结构，存在的风险性越大。真正的财富自由，不是赚得多，而是能从多个来源持续进账。主业、副业、投资+被动收入，构建你的“多重现金流系统”。

就拿我自己来说，我有主业收入，有副业收入，还有被动收入（房屋租金收入、投资理财收入）。副业和投资理财的收入早早的就超过主业好几倍了，很多读者都有做副业赚钱的需求。

其实万物世界的底层逻辑是相通的，不管是在体制内做到一定级别，还是做副业亦或者做投资，都在再跟人性做抗争。我一直跟读者说：要向内求，逆着人性做人，就是停止了与外界的对抗，开始向内求，开始了与自己人性的对抗。

不管是投资黄金还是纵横股市，你永远不是跟市场做斗争，更不是跟别人做斗争，而是跟自己的人性弱点做斗争，只有逆人性操作，才能绝处逢生！

现在我做投资每一次都要和大众反着来，当然我不是为了反而反，每一笔投资我都非常谨慎，不会轻易出手，我的原则是既要逆向行驶，还要逆向逻辑合理，在反向中不断的磨炼你的心性，和自己的贪嗔痴作斗争。

在知识大爆炸的今天，你用一年的时间可以学完金融投资领域的知识了，但你要想获得投资之道，得用十年、十五年甚至更久。普通人想通过投资赚钱，其实要学习和尚，当市场暴跌时，你以拯救苍生的大爱之心进场，当市场上涨时，你以无欲则刚的平常心离场。

## 一、普通人如何做好投资？

首先就是要过简单的生活，别被消费主义洗脑乱挥霍，多多积攒本金，刻意打造自己的被动收入网，比如：租金、知识产权、投资收益等等，然后把自己的现金流装进复利机器投资。因为人和人之间的智商差异不足50%，所以，拉长时间周期看：精明的人一直在负利；智慧的人一直在复利。

## 二、投资是人性驱动的市场

投资很简单，复杂的是人，人性使然，金钱永不眠，人性永不变，很多时候都是非理性的，因此投资市场一直循环演绎着贪婪和恐惧。当市场涨的时候，人的本能总是认为还能涨，所以贪婪追高，忽视了风险都是涨出来的，当市场暴跌的时候，人的本能总是认为还要跌，从而恐惧止损，不敢买，忽视了机会都是跌出来的客观事实！

投资中人性的荒唐：6100点：敢买的有99%，割肉的1%2500点：敢买的人有1%，割肉的99%绝望时卖东西，你会卖得便宜；冲动时买东西，你会买得昂贵！

投资市场永远存在着羊群效应，这也是绝大多数人不可能在投资市场赚钱的原因，人群的聚集不是智慧的叠加，而是愚蠢的放大。当人群乌拉乌拉疯狂的时候，你就反其道而行之，永远跟大多数人不一样，这样做的好处就是你能割大多数人的韭菜，因为资本市场永远是少数人赚多数人的钱，真理也永远掌握在少数人手里！

因此，比方说对于市场真正的悟道人眼里：涨停只是集体贪欲的发作，跌停不过是对人性恐惧的测试，而横盘震荡则是主力在看散户脱裤子的把戏。当你离人群越远的时候，你离自己越近，市场投资是一个人的英雄主义，无论是现实中还是市场中，人数的叠加通常不是智慧的增加，而是愚蠢的飙升，更何况一个终身投资者每天需要进行大量的脑力消耗，实在没精力社交，甚至都不想说话，这或许就是神交。

逆向思考：每次市场出现极端情绪，都可能预示着拐点的到来，别人贪婪我恐惧，别人恐惧我贪婪，你不但要反过来想，反过来操作，而且你的投资逻辑还必须符合天道！

其实，投资很简单，有时候就像买菜一样，比如蔬菜两三块一斤时候嗤之以鼻，但是涨到十几块，反而一哄而上去抢，这心理其实不适合投资。但是有的人就是反其道行之，买点低价格的菜，因为知道菜有价值，不会因为继续下跌而难过。

不要盲目，你盯着一个品种和它的价格，要持续关注，就像捕鱼，等鱼儿游过来就下抄网，而不是鱼游过来了，你犹豫了，反而鱼游远了，你开始追。

很多人犯了一个错误，那就是当价格跌出来性价比，反而不敢买入，我觉得还是担心继续下跌，这是有可能的，但是这种人即使继续下跌也不会买入，因为他还等着更低。说到底，就是心中没有一个价值的尺子，不知道它值多少钱，所以越高也不嫌高，越低也不嫌低。

投资，需要有一个安静的心态，不能靠情绪驱使，往往情绪驱使的，最终都会失败。不要后悔没有最高点跑，也不要后悔在最高点买，因为对于一波真正的大行情趋势来说，这两点都会被浪潮吞噬掉，对于一个投资者的漫长生涯来说，这也根本不值一提。

当然，作为普通业余投资者，我们还可以选择更务实的资产配置，比如用我的“3个三分之一投资策略”：三分之一：做固定资产和易变现的货币基金、国债等配置；三分之一：投稳健的蓝筹股和指数基金以及ETF基金。经济结构的调整，接下去一定就是通过货币的膨胀来把各种问题给稀释冲洗干净，而这片土地上真正的优质资产，一定是“比较出来”的。

尤其是那种出生根正苗红、靠近核心利益、有超强护城河的企业股份。你别指望这种股份会爆发增长，但是只要大环境缓和过来，它们一定是最先收到雨水灌溉的。

好的资产会发展的很好，只要能耐心长期持有，它就会滚雪球般，慢慢在五六年后，成为一台逐渐为你带来稳定财富的印钞机。大佬文集圈一定是根正苗红，并且在行业里是绝对毫无疑问的主导地位，同时市值巨大，几乎不可被撼动。
