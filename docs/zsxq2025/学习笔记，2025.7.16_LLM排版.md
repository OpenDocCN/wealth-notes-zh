
# 学习笔记，2025.7.16

1. 真正能引领中餐的是粤菜师傅，你看看人家粤菜师傅炒菜，真的是艺术品。盐，一点点，什么都是一点点，菜叶，几根。不过，粤菜在北方开不好，都嫌吃不饱……

2. 对于事业，我已经认了，就是我这一辈子，就现在的水平了，再高也高不了了，再低也低不到哪里去了，凡是偏离主业的业务，我都不再碰了，守护核心精力。

3. 我们总说，人是赚不到自己认知外的钱，其实真正的大钱，都是天上掉下来的，与认知无关。小钱，赚不到认知外的，大钱，全靠运气。时势造英雄，真正的认知，必须要敢于突破，站到比大家更高的一个维度去问自己一句，这么干真的是对的吗？

4. 投资是严谨的，两句话表达我的观点：
   A. 存款低于 500万，不理财。
   B. 通过投资赚到钱的难度远大于通过工作赚到钱的难度，因为投资是一个智力游戏，是一个人性游戏，要跟世界上智商最高的一群人博弈，同时又要对抗我们的惯性思维。倘若有个公式，就可以实现年化5%以上的稳定收益？你放心，银行跑步入场。

5. 40岁以后的课题就是与慢性病和谐共处，还有就是如何告别人生。去医院探望一次同龄人，难过一次。

6. 癌症的本质就是基因突变，说白了，就是生死有命富贵在天，人渐渐的终究信命。

7. 进化速度最快的人，是做交易的那群人，交易就如同MMA，要取百家之长，把自己训练出一流的进攻能力，同时又有一流的防守能力，学习能力是一流的，实践能力是一流的，性格没有明显的缺陷，不断的进化，现在市场上的那些顶级散户，大部分连高中都没念过，一方面，在读书这个赛道上，没什么成就。另一方面，在钻研、学习、复盘、训练方面，有着超人的天赋，我最近听了几次北京炒家的直播，他们对人性的理解，对市场的理解，对交易的理解，都是天才级的。

8. 五四运动，其实就两大流派，一派是鲁迅，鲁迅认为救国应该改变基层思想，现状是由人民思想腐化陈旧所致。一派是胡适，他认为体制才是问题之根源。但是不管哪一派，都是现代医学派，鲁迅对中医的定义就是有意无意的骗子。我还是那个观点，现代性是对一个人至高无上的评价，马斯克为什么那么多信徒？就是因为他是一个顶级现代派，朝未来，朝太空，高歌猛进，这一点也在不断的带动国内企业家，前些年，各类概念水很多，什么弱碱水，什么富锶水，现在基本没有了，唯一的概念就是凉白开了，凉白开是迎合了中国人喜欢烧水的习惯，再过几十年，这个概念也没人信了。不是中国的老百姓才迷信，全球的基层老百姓，都迷信，例如欧美人半数以上相信鬼魂的存在，只是迷信的内容不同而已，咱觉得韩国人挺理性的，实际上呢？传言中国普洱茶可以减肥，韩国女人也纷纷抢购。现代性、科学性，永远是精英群体的专属。

9. 关于油车与电车，听的最多的一句是，开过电车，油车再也回不去了，从便利性来讲，的确如此，例如开特斯拉，不需要带钥匙，不需要锁车，用车成本无限接近于0。但是，有些东西，不能一棍子打死。很多人喜欢把电车类比成苹果，把油车比喻成诺基亚。这是两个概念。很多人有类似的比较，是因为从来没体验过高级别的油车，你过去开桑塔纳现在开特斯拉，肯定有升级感，倘若你以前开宝马7系呢？

10. 对于很多美好，那句话是最准确的，斯人若彩虹，在没有遇到，没有感受过的前提下，无法想象，甚至认为所有人都跟我们想的一样，唯利是图，相互背后捅刀子。

11. 我们的人生追求，难道只有车子与房子吗？车子非要三五年换个新款吗？有没有一种可能，我们的消费越来越多样化。有钱人与没钱人在芬兰消费上有没有区别？她认为没有太大区别，吃的东西差不多，玩的东西也差不多，反正一到冬天就是去滑雪，你有月卡，我也有月卡。中国人的消费，其实就两样东西，车子，房子。未来，一定会多样化的。欧美走过的路，我们都会走一遍。
