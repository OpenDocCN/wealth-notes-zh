
# 学习笔记，2025.3.31

1，“人才市场”其实是“人力”市场，人才基本上不会流入市场，要么单干，要么被挖来挖去，出来找活就只有被人挑的份。类似的，好男人、好女人，流通率也很低，基本都早早过上了自己的安稳小日子，不会每天在网上发些好男/女人都死哪去了的抱怨，那些流通率高的人，要么是别人挑剩下的，一直在被选择，要么是目的本来就不是安稳生活的。

2，很多人已经习惯了找个人为自己负责，为自己做主，很难独立做出决定。让别人替自己决定，出了错，自己能有个怪罪的地方，情绪能宣泄出去。让自己替自己决定，前怕狼后怕虎，出错会让自己更憋屈，那感觉会让人习惯性逃避。大多数人为了不思考，可以做任何事，包括让别人做主他的人生。

3，人能否做出准确判断，很大程度上取决于他的脑子里能不能同时容纳多种不同的想法。很多人遇到和自己已有观念不同的想法，第一时间想的是拒绝，而不是验证、求真，这就导致他很容易基于固有认知做出错误的判断。

4，很多问题，并没有被彻底解决，因为它会随着时间的变化而被稀释，导致变得看起来不是那么重要了。就像，小时候的很多痛苦并不是不存在，也不是被自己真正理解、解决了，而是被更多的经历稀释了。

5，对别人不要有嫉妒心态，能嫉妒，说明他做到了一些你做不到的事，说明你能看的见、接触得到他，这种近距离接触学习的机会并不多，要多吸收他身上的优点，不要盯着缺点不放。

6，互联网时代，只要你想学，大多数东西都能学会。不要因为某一方面没有系统的知识就不去学习，如果发现自己对某个没有系统体系的领域感兴趣，应该高兴才对，因为能被整理成系统的东西，一定有很多东西都固化了，会越来越卷，那些没有体系的东西，才有机会去摸索出新的道路来，避开和别人内卷。

7，一个人如何看待你，不能说明你是什么样的人，但大概率能够说明他是什么样的人。因为一个人对外的观点，对别人的看法，都是基于个人认知、经验得出的，而非客观事实。

8，人不能活在过去，改变不了的，就随它去，多关注自己能改变的、自己想改变的。有一种心态是：我不是受害者，我是幸存者，我在一次次灾难里活了下来，我要珍惜这来之不易的新生活。

9，情绪稳定，是必修课，不是没有情绪，不是做个冷漠的机器，而是能够清晰感知到自己情绪的波动，不被情绪左右自己的决策，才能更好掌控自己的人生。你看那些让人后悔的话，基本都是在情绪不稳的情况下说出来的。

10，人和人之间，很难有感同身受，大多是以己度人。当你知道别人只能用他的认知水平来解读你的时候，你就不会有向他解释的想法了。就像，乞丐根据自己的经历，去说皇上下地得用金锄头，你怎么可能给他解释的明白。
