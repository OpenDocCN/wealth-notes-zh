
# 子明私享汇7.15：最后的一场大战

特朗普7月14日对俄罗斯发出威胁，要求50天内停火，否则加征100%关税，美国官员则表示将对购买俄油的国家加征二级关税。一方面，二级关税主要波及的是中国为首的金砖国家。另一方面，50天的时间，正好卡在中国的93阅兵之上。叠加最近美国紧急向乌克兰出售军火，欧洲紧急采购，以及本周欧洲很可能会通过对俄罗斯的新油价制裁，基本可以确认普京将在访华期间发动大规模的突袭，欧美也都在为这场突袭来做相应的时间准备。这场冲突有可能会是正式谈判前的最后一场大规模冲突，普京需要尽可能的将战线向前推，以获取谈判筹码。

为此他可以不惜一切代价，找东大签协议以获取支持。东大为了恶化俄欧、俄美关系，以防止战后关系缓和，并获得俄罗斯的感激，以防止战后与俄关系恶化，甚至为了在东美、东欧的谈判中获取主动，也会在最后关口极力支持俄罗斯以使其扩大战场上的优势。

特朗普为了平衡东大、普京、欧洲和军工集团的四方关系，需要一边默许俄东发动战争，一边支持欧洲防卫和制裁，需要一边给东大卖芯片，一边给欧洲卖军火，需要一边怒骂挚友普京，一边跟普京背后的男人谈笑风生。

普京为了确保这场军事行动的胜利，需要量大俄之物力，结与国之欢心，这些巨大的成本，就需要向乌克兰身上转嫁，通过更多的侵占领土，来麻痹民众，来向国内宣扬自己赢得这场胜利，可怜的则是乌克兰，资源给了美国，土地给了俄罗斯，自己则默默承担的战争带来的持续性伤害。

Thune hits brakes on Russia sanctions package

参议院多数党领袖图恩7月14日表示，他将暂缓推进针对俄罗斯贸易伙伴的备受关注的制裁方案。4月1日，美国参议院半数议员联合提出一项对俄制裁法案草案，将授权特朗普对仍与俄罗斯有贸易往来的中国、巴西、印度等国家的进口商品征收至少500%的二级关税。
