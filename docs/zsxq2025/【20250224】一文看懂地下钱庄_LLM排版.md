
# 一文看懂地下钱庄

随着国民经济的持续低迷，我国对非法金融的打击力度越来越大。毕竟任何一次意外都可能诱发系统性危机。这里所说的非法金融主要是影子银行或地下钱庄。

所谓的地下钱庄就是没有牌照的银行，经营的业务跟正规银行差不多，都是存贷、汇兑、投资等业务，只是没有政府发放的牌照，不受政府监管。所以杠杆可能更高，安全性也更差。若老板跑路，就很容易引发大规模的维权活动，增加政府维稳成本。所以越是政府财政困难的时候，对地下钱庄的打击力度也就越大。这些年打击地下钱庄有一个规律，就是经济越差，打击力度越大。尤其是各地接连暴雷后，P2P前些日子政府又打击了一批地下钱庄，数量高达100多件，涉案金额高达800多亿。

地下钱庄到底是怎么发展起来的？为什么越打越多呢？今天我们就来捋捋地下钱庄这个事。

文章内容主要分为三部分：一、地下钱庄是怎么发展起来的；二、地下钱庄经营的业务都有哪些；三、他是怎样规避监管的，他为何越打越多。

### 一、地下钱庄是怎么发展起来的

我们都知道地下钱庄是相对于国家银行而言的，国家颁发牌照的银行就是合法的，没有颁发牌照的就是非法的地下的。但比起钱庄，国家银行出现的非常晚，是最近100年的事，而在此之前的2000年里，民间的金融借贷都是由钱庄来完成的。只要有一定的资本金就可以经营，没有地下一说。在外资银行进入中国之前，中国的小钱庄遍地都是，规模小一点的1000~3000两的资本金，中等规模的3000~5000两，大点规模的5000到1万两。而像山西票号这样的资本金大都在1万到3万两，可以说是银行中的银行了。民间钱庄从汉朝一直延续到钟供建政，钟供搞了公私合营，民间钱庄被收归国有，彻底消失了25年。直到改革开放才再次出现。

### 二、地下钱庄经营的业务都有哪些

一是货币兑换。众所周知，深圳是最先开放的特区，最先开放的深圳是接待港澳市民和老外最多的地方，很多老外经过罗湖口岸来到深圳，想买点东西又懒得跑，银行就直接用外币支付。一些商贩们不愿意，后来发现其中有利可图，就接受了外币。一些商贩手上的外币积累多了，就会找有护照或者港澳通行证的熟人帮忙，到银行兑换外汇券，然后买紧俏商品，再拿到市面上高价卖出。

二是走私。我们都知道，四个经济特区开放后，最先进来的是从香港、澳门转移过来的三来一补企业。当时的政策是，四个经济特区各划定一块飞地，允许外来企业进驻。企业进来实行来料加工、来样加工、来件装配，双方互不作价，外商提供设备、原材料、样品、厂房、资金，政府不负责外商贷款，进出口实施免税或低税，但外商需要向中国工人支付外币工资。这样政府不仅能解决当地的就业问题，还能创汇。但你有你的打算。

三是汇兑业务。为了鼓励出口创汇，钟供政府又实施了一系列政策，导致外汇市场出现了官方价、调剂价和黑市价三种不同的汇率，这又形成了不同的价差，有价差就有利润空间。不仅外贸企业追求更高利润，国企也一样追求更高利润。比如深圳某财务公司有一笔100万美元进账，他需要将这笔美元换成人民币。80年代，美元兑人民币的官方价是1比2.9，而调剂价格可以达到一比四，黑市价可能是一比4.3。官方价调剂架黑市价之间价差如此之大，这就给了地下钱庄充分的发展空间，也算是变相鼓励外资通过地下钱庄进入中国市场。

四是骗税。我们都知道八九十年代，政府为了鼓励出口，实行了出口退税政策，外贸企业发货后，就可以拿着出货单据和报关单据到税务局去办理退税。地下钱庄在这里发现了商机，他们发现只要联合海关和银行内应，即使没有货物出口，也可以拿到相关单据，然后去税务局办理退税。

五是洗钱。地下钱庄还有丰富洗钱的业务，比如走私现金、信用卡消费、经常项目下的交易等方式。现金走私主要是小额现金，由真人携带出境。八九十年代，罗湖口岸附近有一群人叫做水客，以携带现金出入关口为生。每天好几趟，看起来像是去香港打工的农民工，其实都是现今的搬运工。这样蚂蚁搬家式的走私主要发生在八九十年代，额度也比较小，500万以下的可以人肉搬运，费用也比较低。但进入2000年以后，洗钱的规模越来越庞大，动不动就几千万甚至上亿，大额的洗钱业务就需要通过其他途径才能绕过监管。

六是高利贷。这是地下钱庄构建境内资金池最重要的目的，尤其是在浙江福建一带，地下钱庄主要以民间放贷为主业。80年代温州乐清有很多民间互助，会有一波小老板组织一个堂会，把大家的资金聚集起来，谁需要用钱就借给谁，再收取一定的利息，利息远高于银行同期。很多民企从银行贷不到钱，又想扩大生产，就不得不找民间高利贷。

随着中国加入世贸组织，经济快速发展，普通人的闲钱越来越多，网络也成为新的集资平台，地下钱庄逐渐以P2P的形式向全国网民高息揽储，再通过高利贷贷给网民。随着楼市降温，网络集资平台资金链断裂，引发了一系列维权事件，迫使政府在随后的几年里不得不大力打击以网络集资为主业的地下钱庄。
