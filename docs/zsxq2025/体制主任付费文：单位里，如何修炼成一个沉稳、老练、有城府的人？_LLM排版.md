## 一、有城府最好，没城府你也要学会装，可以少说话，甚至不说话，但一定不要乱说话，防套话

没有城府的人都是随时可以被其他同事拿捏住的，甚至是可以把玩的，那究竟该怎么办？我认为首先你在办公室里要成为一个起码是看上去有城府的人。

要保持一个70%的神秘感，这个神秘感会让人感到敬畏，也让人感到有点拘束。这就是宇宙的一个定律，如果你的底牌被人家知道了，超过了70%，他一定就不会再害怕你，甚至他就会开始拿捏你，因此你要注意保护自己的隐私，不要让超过30%的信息泄露出去。

很多人比较单纯，别人问什么就说什么，有的人刚迈入单位，不知道什么该说，什么不该说，不知道什么该做，什么不该做，生怕得罪别人，该说的不该说的都说了，该做的不该做的都做了，其实这是非常不成熟和稚嫩的表现，我们一定要渐渐的学会装，你要渐渐的善于把自己的这种真实想法都藏在心里。不要让那些办公室的那些同事一眼就能看穿你对某一件事情的态度，也不能一眼就看穿你此刻是高兴还是沮丧。

比如同事们坐在一起聊天了，你能不说话就不说话，自己多听。如果有同事问你，小王，咱们科长分配任务是不是太不公平？这样当领导，欺软怕硬，早晚都不愿意跟着他干了，简直就是混蛋。

这个时候你该怎么回答？如果你确实想不起来高明的回答，既不当着这么多人附和着说领导是坏蛋，也不想佛了同事的面子，那你就嘿嘿一笑，小幅度的点点头就是了，即便你内心深处也是这样想的，但是你别让他们知道你到底对这件事是什么态度。总之，和同事打交道，你什么都不表态，别人说什么你都用这一招笑呵呵的小幅度点头。既不表态是，也不表态不是，把自己内心深处的想法、所有的想法都隐藏起来，让他们摸不透你，猜不透你，他们就不敢再对你颐指气使的。

## 二、学会正确的使用微信和发朋友圈

很多人在现实中说不了几句话，但是在微信上上却交流的很多，动不动就你一条我一条的语音聊来聊去，动不动就朋友圈更新动态求点赞，要么就是在别人朋友圈里互动留言等等。我们可以利用互联网社交扩大我们的交际面，但是在办公室里我们就不要像那些年轻人那样肆无忌惮的去展示自己，不要再用那些只能显得你幼稚的头像和名字。

我们每个人在出生的时候，父母都会给我们取一个名字，都是经过斟酌的，所以取这个微信昵称的时候也不能太随便了，必须非常慎重。

很多时候领导和同事看到你用这样的微信名字，虽然嘴上不会说什么，不是公事的话，大家都不会明着指出来，因为明着指出来伤脸面，而且也没什么好处，你也不会感激他，如果你这个时候还自我感觉良好，其实非常糟糕。

什么蜡笔小新互联网漫画的热图就不要用了，你可以用你自己的照片做头像，你也可以用你家的孩子的照片，你也可以用这个风景画，这些都没问题，在我的观察中有的人的头像非常幼稚，有的男生二十七八岁了，30多岁了用小猪佩奇的头像，或者用什么灰太狼、米奇、老鼠、熊二这样的一些卡通人物的头像，会显得很不成熟、很不稳重。

这跟体制内对人的素质要求是相违背的。体制内非常要求一个人要稳重，尤其是你要到重要的部门去工作，比如说你要去到领导面身边当秘书，你要去组织部、去办公厅，如果你这个人不稳重就很麻烦，这个稳重其实可以从很多方面可以体现出来，包括你的衣着，包括你说话，包括你走路，包括我们现在说的微信头像等等。

你看看你们领导的那些微信头像，是不是清一色的，要么是大山大水，要么是书画作品。所以我建议你把你的工作微信头像换成熟一点，你的朋友圈里有太多的同事和领导，那你就不能再在朋友圈里发那些无病呻吟的动态，朋友圈也是我们的一张名片，从一个人的朋友圈中可以推断出他的性格、兴趣爱好、思想、生活状态等等的很多内容。

所有让你看起来不成熟的动态都不要再发了。体制内对人的要求非常重要的一点就是稳重。当然其实不光是体制内，体制外也是一样的，如果你要做到管理层要做到这个领导岗位，稳重这个素质是非常重要的。比方说有的人发朋友圈的频率太高了，整天刷屏，一点开朋友圈，几乎全是他的动态，甚至工作时间也在发朋友圈，这样就非常不好。

第一，说明你这个人非常幼稚，非常不成熟，自己有一点点什么想法，有一点点什么动态，有一点点什么情绪，心里就藏不住，老是想让所有人都知道，情绪太过外露，给人的印象就非常不好。第二，也说明这个人很闲，没有把心思用在工作上，工作时间老是玩手机刷朋友圈，工作态度有问题，所以从细节着手让自己更加稳重成熟一些。
