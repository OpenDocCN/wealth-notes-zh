
# 即将登场（6战场不一定是WW）

美国目前明显采取的“攘外必先安内”策略，从首席观察的视角来看，我们对它们在统一思想和行动方面的成效感到赞叹，但由此带来的民主党更大范围的反扑，也不可避免。

热战的概率在很大程度上取决于双方力量对比转换的那一瞬，现在距离那一瞬已经很近了，但是引发热战的方式一定是偶然的。回顾过去的历史，要么是一次偶然的伤亡事件，要么是一次突然宣布的决定，要么是一次演习。但一旦战争的大门打开，关闭起来就非常艰难。这也是首席反复说的：开始战争容易，但是想结束战争，一定由不得自己。

俄国当前也是这个局面：打开战争的方式很多，但是影响战争进程的因素更多。

## (一)日本的角色很尴尬

日本始终没有确定好自己的定位，这跟欧洲不一样。在亚洲大国看来，日本的定位应该是瑞士、瑞典这样的模式：保留皇室和封建体制，但是在地缘政治上，做一些辅助性的工业，不要掺和大国的兴衰。但是日本不愿意，那这就没有办法，只有在解决WW问题的同时，解决日本问题，否则WW问题仍然无法解决。

因为只要日本封建力量还存在，那么WW仍是日本再次起步东亚的第一站，也是迈开双脚站稳东亚的最核心的环节。控制WW，日本东可抗美，西可制华，南可威慑东南亚诸国及美国的包括马六甲海峡在内的海上生命线。同样，反过来如果我们完成解放战争，解决WW问题，日本的所有利益都会被切断，它们势必全身心投入到阻止我们解放的过程中。

所以，解放一方面是我们内战的延续，另一方面也是第二次世界大战中东线战场上，我们对日作战反攻的结尾。而这个结尾的前半段是由苏联解决的，那就是1945年老大哥摧毁关东军，但是剩下的势力没有被摧毁，这个过程同样需要我们跟俄国一起来解决。

## 首席需要提醒大家注意一点：

我们虽然是一个制造业很强的国家，但是我们同样躲不开一些规律，比如我们不能同时开启两场战争，也不能先后开启两场战争。这对于我国力的消耗、人口结构的消耗、社会意识的消耗将是巨大的。

回顾1840年以来的苦难，大概有这么几次影响稍微大一点的战争：
- 1840年开始的第一次鸦片战争
- 1851年开始的太平天国运动
- 1856年开始的第二次鸦片战争
- 1883年开始的中法战争(在越南开始)
- 1894年开启的甲午中日战争
- 1900年开始的八国联军侵华战争
- 1915年开始，日本以二十一条为代表开启了特别军事行动向东北渗透
- 1931年开始九一八，整个东北沦陷
- 1937年开始七七，正式开始大规模入侵
- 1945年开始解放战争
- 1950年开始抗美援朝战争
- 1962年开始中印边界战争
- 1965年开始抗美援越战争
- 1969年中苏珍宝岛战斗
- 1979年中越自卫反击战
- 1988年“3.14海战”

理论上说，我们基本上每隔10年就会一次战争，这对于国力的透支是巨大的。直到1990年世界发生了巨变，我们采取了超过一切国家的超强组织能力和动员能力，在极其艰难的情况下，利用30年时间，吸取了西方所有的制造业经验，形成了当前我们的制造业体系。这是非常非常难的。

我们不能透支我们的国力。虽然要打，但是一定要考虑未来。所以，如果同时摆在我们面前的是、美国、立本、澳洲、菲律宾、半岛，我们一定要有一个取舍。

这个确实很难，敌人也不会让我们利益最大化，我们也不能让自己的利益最小化，那就需要一个折中选择。在这些选择中，WW是我们的底线，也是牵动各方的核心，但是这并不是关键，关键是如果各方的妥协和目标都回归到日本这个点上，那么100年后，我们给自己的子孙留下一个怎样的地缘环境呢?

19世纪中叶，日本打开国门，由于战略纵深不够，英、美、俄迫使日本签订了不平等条约后，民族矛盾和社会矛盾激化，德川幕府统治结束，1868年，革新派实行“明治维新”,废除封建割据的幕藩体制，建立统一的中央集权国家，经济上虽然有很多的发展，但是仍然是一个四不像的体制：经济上模仿美国、陆军模仿德国、海军模仿英国。

150年来的日本这三个模仿从来没有变化过。可问题在于地缘政治也从来没有变化过，日本不是英、美、法、德、西班牙，英国的对面是松散的法国、德国、西班牙，俄国在非常遥远的位置；但是日本的对面，从来都是完整而庞大的中国和俄国，半岛上松散的两个朝鲜才能成为它的目标。

因此，退而求其次，日本的目标只能是松散的岛屿，比如、菲律宾，然后南下到东南亚一线。现在的问题是，我们14亿人要活的更好，需要东南亚5亿人的市场和非洲14亿人的市场，这跟日本是重合的。

并且在战略态势上，如果我们不能解决日本问题，100年后，我们的子孙仍然会面对这个问题，并且地缘形势会更加糟糕。但是这种选择一定是带有偶然性的，当前理论界普遍同意一个观点：日本在等，等待中美之间的冲突，进而寻求一个契机，让中美在日本以外扩大战争。

但回顾日本的历史，这同样带有日本式的冒险。因为理论界的这个观点再仔细看看，就会发现一个问题，那就是回到那句话：开启战争容易，结束战争可太难了。历史上，日本在1938年迅速扩大战果，但是，却陷入与位于太平洋东西两岸美国和中国同时作战的困境。

太平洋战争爆发前，日本国力已开始萎缩。1937年美国、苏联、德国、英国、法国、意大利和日本七大强国中，日本国民收入位居末位，而国防开支在国民收入中所占比例却居第一。1945年在美国、苏联和中国的三角合击下，日本从1870年改革以来的所有成果都灰飞烟灭。直到1950年朝鲜战争爆发，日本重新开始利用组织能力，解决了经济上的问题，但是政治上的问题和军事上的问题迄今都是零。

这里稍微解释一下，所谓是零，意思就是日本的地缘环境，跟200年前相比，没有任何的改变：中国仍然是统一的、俄国也是统一的，WW的利益目前还在手里但是非常的微薄了，半岛中，只能在38度线以南区域活动。

也就是说民治维新以后的所有努力，都是一场空。

在政治学家来看，任何没有改善自己政治版图的改革，都是非常肤浅的。我们的改革，最大的收益不仅仅是经济上的，更大的收益是我们稳定了北部领土线，这条线在30年来，从来没有成为我们的负担。作为普通读者可能没有感觉，但是作为首席的读者，要有这个觉悟。

而且随着我们政策的深入，我们经济上的成就正在反哺到政治上，那就是整个国家的统一和我们试图扩大领土的想法在不断的延伸，虽然这种延伸看起来是有害的，但是不能不说是一件好事。

但是大国的妥协往往也在一念之间，是精确的计算后的一念之间。什么意思呢， 就是如果从实力角度出发，能拿到手的利益才是唯一可靠的利益，按照这个角度，中美俄都能看上的筹码并不多，但是日本，是一个好筹码。

因为对于美国来说，驻军的主要目的是防止我们，但是次要目的是摁住日本，对俄国来说，北方四岛问题是一个涉及根本的问题，这意味着对日本需要采取积极进攻的态势才能解决，对我们来说，新仇旧恨都在一起，攻击日本没有任何道义上的阻碍。

而且还有一个很现实的问题，如果世界上少一个工业国，这对西方来说是一个妥协，这意味着竞争者少了很多，也是可以交易的。

## (二)做好美国人释放日本的准备

二战结束后，雅尔塔体系有了相当的调整。但总体说来，雅尔塔体系对于遏制德国和日本军国主义复活的制度根基及由此形成的地缘政治的基本格局没有改变。

在欧洲地区，苏联垮掉后，原属于苏联地缘政治势力范围的东欧转入西方阵营。俄国失去了苏联时期在欧洲地区的地缘利益。

但是在亚洲地区：俄国仍然继承着雅尔塔体系赋予苏联的占领日本北方四岛的权利；美国仍然占领着冲绳岛；中国在雅尔塔体系中被恢复并确认了的对台湾的主权的法律地位没有改变。

如果美国要动摇在亚洲地区的雅尔塔体系，那就三个事情要解决：
第一，俄国要撤出北方四岛
第二，美国要撤出冲绳岛
第三，我们不被成为对WW的主权地位

中美俄可能都不会认可，既然如此，那么这种体系就不能动摇。

那如果不动摇这个体系，那么日本就没有可能成为正常国家，那被攻击就很正常，在法理上，仍然是作为第二次世界大战的战败国来处理，并不作为普通国家的侵略来处理。

要知道，当前的德国被英法重重围困，根本原因就是为了防止德国东山再起，而德国选择在经济上向俄国要能源，也是试图重新东山再起的一种方式；

那么日本是怎么被围困的呢？还是雅尔塔体系，俄国占绝北方四岛、美国占据南方的冲绳岛，中国收回台湾岛，这基本上是减除日本的羽翼，让日本龟缩到本岛，断绝了日本重新大规模武装的可能。

可问题在于，150年过去了，日本当前的的局面跟1860年的时候没有区别。

明治维新后，日本的生产力发生了巨大变化，但是本土的政治空间太小，地缘空间也太小，没有办法必须出门侵略。

现在是一样的，工业上，中国强劲的势头，让日本财阀的很多行业都已经不赚钱了，即便是汽车行业，这个日本最后的堡垒，也基本上可以在5年内宣告不怎么赚钱，既然如此，日本究竟是愿意当瑞士瑞典还是愿意当英国呢?

现在看来，还是愿意当英国。不然日本为什么造那么多军舰呢?还在模仿055制造新一代军舰，包括最新的陆战装备。

美国的希望是，既要避免直接卷入跟中国的冲突，又要强力遏制中国崛起，那只有武装WW和菲律宾，并且武装日本。希望这三个马仔能打头阵。

首席说过，不能同时开启两场战争，也不能先后开启两场战争。擒贼先擒王。要么选择跟美国直接打正面，要么选择攻击日本。

这是唯二的可能性。因为日本的崛起只有一个历史条件可以造就，那就是：中国和俄国同时衰落，美国同时兴盛，予以最大程度支持。

这个历史条件在1860年-1890年间出现过，所以日本崛起了。但是现在是中国在崛起，俄国在缓慢复苏，美国在衰落同时给予了有限的支持。在这种情况下，日本的地位就显得非常非常的微妙。

首席的很多读者也是军事爱好者，首席请大家看看当前我们海军的部署和空军的装备，看上去都是在完成一件事，也就是在3000公里-5000公里这个区域内完成对美军的阻断和阻击，但是如果这种阻断和阻力并不是只针对WW的呢?

因为在WW这边打，打来打去，最后源源不断的支持还是来自北方的日本，这种持久战又有什么意义呢?

当前日本极度的沉默，一方面是它们意识到了危险，另一方面它们也在扎扎实实的备战，这种备战的程度远超20年前支援伊拉克战争时日本的备战。
