
# 即将登场(5不要害怕美国的技术)

帝国主义的核心本质不会变。同样，科技和人的核心也不会变。现在还是存在一种情况，就是虽然打破了美国对我们的科技讹诈，但是我们怎么认识科技上，还是模糊的。

## (一)经验值得我们吸取

历史上英国曾逼迫美国像印度一样，成为英国资本的附属国家。但是美国人并不害怕，充满了雄心，在抗击英国的政治压力和军事侵略中，走上了一条自主型的国家发展道路，这不仅使美国成为20世纪世界科技的中心，而且还使美国成为执领20世纪全球技术革命的先锋。苏联也是如此，十月革命以后的苏联，摆脱了对欧洲的依附，进而寻找到自己的发展道路。这两个国家都是典范，虽然意识形态完全不同，但是它们都成为世界上两个科技原创力和全球科技资源利用能力在世界范围内领先的国家。

## (二)当前美国人的科技走势已经进入歧途

英国人曾经准备大规模压榨美国人，让美国成为自己的附庸。但是美国人成功地避开了英国人设下的“世界主义”和“自由贸易”的圈套，在高关税保护下，美国依托其强大的民族工业而崛起。随后，用这个办法围困苏联人，现在是美国人重新用这套办法来对准中国人。剥夺敌对国家力量的最简洁的办法就是瓦解它的国家政治能力及支撑这种能力的战略产业。社会主义是以人民为本的主义，科学技术从属于人民，这使社会主义科技生长和转化有了最深厚的土壤。资本主义是以利润为本的主义，它的科技只能从属于科技，其发展空间必将日益萎缩且不可持续。美国人当前正在这个趋势之中，当前的所有创新都是为了利润，那就只能是为了科技而科技，并不是为了人民，也不是为了第三世界广大的工农阶级。

## 科技必须服务于国家

科技必须服务于国家，服务于国家战略，否则就是无根之木。很多科技从业者有一些迂腐的想法：科技没有国界，其实内心想的是，科技只姓欧美，这个国界只是中国的国界，而碰到了西方的国界，就自动跪下了。

今天的世界也是这样。我们为科学而科学，还是为救国?我们需要的是单纯的技术自由，还是自己的生存权和发展权?这需要明确。其实，当我们失去了民族独立和自尊的时候，科技对我们就没有意义了。

毛主席始终说： "武器是战争的重要的因素，但不是决定的因素，决定的因素是人不是物。力量对比不但是军力和经济力的对比，而且是人力和人心的对比。军力和经济力是要人去掌握的。如果中国人的大部分、日本人的大部分、世界各国人的大部分是站在抗日战争方面的话，那么，日本少数人强制地掌握着的军力和经济力，还能算是优势吗?它不是优势，那么，掌握比较劣势的军力和经济力的中国，不就成了优势吗?"

## 西方害怕我们科技进步的原因

西方这么害怕我们的科技进步。因为科技是摧毁它们经济壁垒的唯一方式，没有之一。首席提醒大家回忆一下历史，历史上并不是只有资本主义的全球化，世界上，在资本主义之前的农业社会也有全球化，那就是农业社会的全球化。农业社会的组织架构、动员能力、文化先进程度都比当时的游牧民族要强，所以虽然当时的游牧民族发动了大规模的进攻，特别是蒙古的全球扩张，但是最后都被融入到了农业社会的大潮中。

资本主义也是一样，资本主义在扩张中，产生了比农业社会更强的组织架构、动员能力和生产能力，因此农业社会无论如何抵抗，都无法逃脱资本主义的控制。为此，资本主义把贵族地区的西欧、美国作为核心地区，而把其他地方作为利润来源区。如果是利润来源区存在原创技术，那就是一定要摧毁，并且摧毁它们的组织能力；如果利润来源区没有原创技术，那就输出技术，让技术成为利润来源。这样，中心地区可以在利润来源区源源不断的获取大量高额利润而更加发展，而利润来源区因利润外流而穷人越来越多并失去了本土市场。

## 中国的科技发展

我们国家之所以要建立全国统一大市场，根本目的就是为了孕育和源源不断的创造更多的底层技术，否则就是低端技术的简单铺开。

当前在战略欺骗中，美国人最担心的实际上是地缘政治上不获益，与科技上同步不能获利，形成了双重暴击。美国的资本主义制度使其不能依赖人民，只能依赖技术，因此对技术叠加路径的依赖就成了美国科技发展的软肋。我们在学习美国、赶超美国科技优势时不能掉入美国陷阱。最典型的是当前Al和机器人的发展，让很多人形成了要被取代的悲观情绪。

这是一定会出现的，但是某种程度上会加速中国成为更有创新的国家。因为有了相同的工具，大家回到了拼天赋的时候，甚至是拼自律的时候。因为对于技术的依赖已经是人类的本能，长期使用手机，人类的情感也趋向机器化。通过技术手段来监控健康数据、工作效率，这已经是稀松平常的事，但这实际上也是人类逐渐趋向机器化的体现。所以不在于技术取代人，而在于人的惰性让自己主动靠近机器。

当然，回到宏观上，我们的社会主义制度仍然是最后的武器，使我们可以依赖人民，这有助于我们打赢人民战争，但为了取得全面的胜利，我们必须有强大的技术作战能力。因为战争是需要人民，而战役是需要技术。周末两天我们就讲到这里。
