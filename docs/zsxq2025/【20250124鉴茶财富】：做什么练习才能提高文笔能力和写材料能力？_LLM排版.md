
# 做什么练习才能提高文笔能力和写材料能力？

鉴茶财富进阶250123-100课程大法师

提问：大王好，我现在在一家国企搬砖，现在单位内卷严重，本人又容易焦虑。现在写材料的岗位才容易工作出成果，平常拜读大王的文章就很膜拜大王的写作水平。想请教下大王，平常可以有意做什么练习才能提高文笔能力和写材料能力。这个问题可能比较初级，还望大王不吝啬赐教。

南院大王回复：国企是需要笔杆子，而笔杆子的背后是视野，思维，然后才是文笔。可以先熟读教员选集1-4,尤其是第二本。平时《古文观止》多看甚至背诵，文章的结构，逻辑，措辞，耳濡目染你就提高了。日常写文章注意，多用排比句，增加气势，多用比拟句，把深奥的问题能解读的通俗，也能把通俗的东西拉出高度。

评论：
文以载道，材料不是写出来的，写只是把思考凝结成文字，所以不了解的事是写不透彻的，领导一眼就看出来外行了，这个最主要。具体到术的层面就是大王说的了，不过板凳需坐十年冷，万个小时定律。
