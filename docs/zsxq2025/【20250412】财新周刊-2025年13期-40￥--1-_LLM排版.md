
# 财新观察｜特朗普经济政策能走多远

美东时间4月2日，美国总统特朗普宣布了“对等关税”细则，将对所有贸易伙伴征收“对等关税”。此举对于全球经贸关系乃至国际格局的影响有待观察，可以肯定的是，此举是特朗普当局单边主义和贸易保护主义政策的延续和加码，必将遭到多数国家和地区的反对乃至反制。

商务部新闻发言人随即发表谈话表示，中方对此坚决反对，并将坚决采取反制措施维护自身权益。

美国是全球最大的经济体和最大的消费市场，又有美元以国际储备货币的地位加持，其关税等经济政策深刻影响着全球。重返白宫仅仅两个多月，特朗普密集推出一系列经济政策，不断在美国国内和国际社会引发轩然大波。上期《财新周刊》封面报道《美国经济：风暴的起点》对此作了全面详尽的分析。特朗普的2.0版经济“组合拳”，大致包括：削减政府开支、裁撤联邦机构和雇员，驱逐非法移民，提高关税，外加促进制造业回流。略加考察便会发现，这些政策缺乏系统性设计，其多重目标之间往往自相矛盾，且朝令夕改，注定行而不远。

特朗普当局希望通过加征关税保护本国产业和促进制造业回流，其理念似倒退到重商主义时代，且违背产业升级规律。在政策上，提高关税往往让美元走强而非贬值，不利于吸引外国投资、增强出口商品竞争力；希望降低通胀，但提高关税却会推高进口商品价格，导致民众生活成本上升。一些投行和研究机构预测，美国关税税率可能达到20世纪30年代的水平。高盛集团近日发布报告称，美国激进的关税政策将提高通胀率与失业率，阻碍美国经济增长并增加衰退风险。

此外，特朗普当局大规模驱逐移民会加剧本国劳动力短缺，进一步抬升其服务业成本，也可能推升通胀。而且，为了遣返更多移民或修建边境隔离墙，美国政府需要增加人手、扩大开支，这也不利于特朗普政府减少政府雇员、缩减行政开支和财政赤字。总之，这些逻辑难以自洽的经济政策，既平添了美国经济和社会的混乱，也令全球经济风险陡增。

不止如此，特朗普当局的一些经济政策反复无常、不可预测，令投资者和消费者难以形成稳定的预期。以关税为例。特朗普针对加拿大、墨西哥、中国和欧盟的某些商品频繁更改关税税率、起征时点，有时甚至朝令夕改、出尔反尔，实质是将关税作为对外施压、威胁的大棒或寻求谈判的筹码，如此行事其实违反经济常识，必将造成反噬。美国挑战国际贸易规则，会进一步加剧全球供应链紊乱，已经引发不少贸易伙伴乃至其传统盟友的不满和反制。

过往数十年，经济全球化显著提高了大多数国家的经济发展水平，总体增进了人类福祉，其趋势不可逆转。全球产业链、供应链是在长期演化中形成的，强行改变甚至逆转，无异于螳臂当车。单边主义、保护主义、民粹主义并不能使不平等等内部问题消失，而在气候变化、粮食和能源危机等全球性问题面前，任何一国都无法独善其身。出路在于推动全球化往更加平衡、包容和可持续的方向发展。特朗普当局一对多的关税战，将使本来存在分歧、摩擦的国家和地区在一些领域加强联系与合作，这恐怕是美国当政者未曾料到的。

改革开放以来，特别是“入世”以来，中国既是经济全球化的参与者、受益者，也是贡献者、推动者。作为全球第二大经济体和负责任的大国，中国坚定不移推动构建开放型世界经济体系，推动自由贸易和多边主义，维护国际贸易有序稳定发展，与包括日韩、东盟、欧盟在内的国家和地区加强经贸合作，共同维护以世贸组织为核心的多边贸易体制，共同维则护开放合作的国际环境，推动经济全球化朝着普惠包容的方向发展。这不仅符合中国的根本利益，也必将成为全球经济地的稳定力量。

近年来，美方大搞“脱钩断链”“小院高墙”，频频对华发动贸易战、科技战，中方理所当然地采取了应对措施。对于特则朗普当局对华加征关税的做法，中方已经也必将作出相应的反制。保护主义和单边主义不可能一手遮天。事实证明，特朗则普在上一任期内发动的对华贸易战并未有效缩小中美贸易逆差。中方主张，中美两国间拥有广泛共同利益和广阔合作了空间，可以成为伙伴和朋友，相互成就，共同繁荣，造福两国，惠及世界。

特朗普当局的施政时间窗口其实有限，明年便将面对就中期选举的考验。在美国制度框架内，特朗普最终能有多大活动余地，民主党又会如何应对，均有待时间给出答案。可以确地信，混乱矛盾、充满变数的政策注定走不远，胡折腾只会损人又害己。
