
# 消失的J20 总设计师

源自1月20日、1月22日群聊  
发布于2025年1月24日  

1月19日，中航工业网站突然变动，  
总工杨伟与总经理郝照平两人的简历双双被撤，  
而杨伟正是歼20的总设计师，  
30多年的军工生涯参与并主导了中国二代机到五代机的飞跃，  
也凭借着卓越的贡献跻身中科院院士和十九届中央候补委员。  

郝照平68年的，远未到退休年龄，  
在没有新任命而挪走简历，  
一般意味着出事儿，  
杨伟虽然63年的，已经超龄服役了，  
但这种国宝级专家若非身体原因，也不会轻易去职，  
而且即使退休，也会以荣耀备至的方式，  
断不可能跟一个出事儿的一起被从官网拿下，  
因此肯定也是有麻烦了，  
希望看在他的功绩，能够相对平稳的落地吧。  

航空领域，之前并非反腐的重灾区，  
之前仅有中航工业谭瑞松等寥寥几人落马，  
主要落马的都是隔壁的航天系统，  
阵亡率是俄乌前线炮灰的好几倍。  

考虑到上一轮航天系统落马潮的起点是流浪气球，  
这一轮沈飞成飞的新式战机起飞，  
很可能也敲响航空领域落马潮的丧钟。  

宋金和谈前夕斩岳飞，这个说了很多了，  
现在批量发生之后，已经不太愿意在提及了。  

过去十年，军民融合的角度一直都是军融民，  
未来，军民融合的角度可能会变成民融军，  
随着中国的外交战略转向，  
越来越多的民用大佬会接替军工大佬，  
主导中国的军民科技产业的融合与爆产能。
