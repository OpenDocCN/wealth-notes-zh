
# 文科生要被AI淘汰？当然不

## 人文、历史与国际政治的机会

这两年来，我和一些同龄的老朋友见面，经常聊到一个经典话题：AI时代，文科生的机会在哪里？大家关心这个问题，是因为各自的孩子都开始进入初高中，要面临很现实的专业和择业问题。

人文、历史、国际政治这些专业，跟一些比较熟的朋友，我会比较具体直接，建议孩子们考虑两个方向：

一个是人文、历史专业。这响应的是文化繁荣和重构中国故事的长期趋势。这种能力的买方很多。泡泡玛特们需要编剧班底，消费品牌们需要全案策划，影视综艺们需要内容策划，方特乐园们需要自己的“框架设计师”，地方政府和文旅集团们也需要各自的“宫崎骏工作室”。

《黑神话：钟馗》的预告片，在德国科隆游戏展开幕夜上一播，全球游戏玩家们马上为之疯狂，一堆TikTok博主、YouTube博主愣是凭着毫无水平的中文水平，连夜啃完了中国相关资料，出视频向全球网友介绍钟馗的故事。

未来十年，正是中国的传统文化在全球兑现红利的时代。这当中需要的趣味、立场、与民众的共同体感，AI很难抢先。

另一个方向是国际政治。这响应的是新全球化、新出海的长期趋势。这种能力的买方是谁？想成为新一代跨国公司的中国企业，以及想跟中国做深度生意的外国政府和外国企业。

在新的全球化时代，大量的中国公司、中国创业者要去往海外寻找机会，尤其需要重新认识中亚、中东、拉美这类非西方市场，而中国产业链里，此前这块的产能储备是严重不足的。

泡泡玛特当前在LinkedIn上有一个招聘岗位叫“Social Media Operations”，社交媒体运营，工作是运营TikTok美区账号，面向美国市场策划营销活动，检测趋势，分析绩效指标，给出优化建议，等等等等。这个岗位，要求的工作经验是零，接受应届生和2026届实习生。不要求工作经验，那要求什么呢？要求会玩美区的TikTok，有在美国生活、学习或工作的经验，熟悉美国文化，掌握英语和普通话，会用TikTok的分析工具。

类似的，京东也在招聘“海外安防岗”，不是在招保安，而是收集整理所在国家和地区，关于安全和损耗预防的法律法规，跟当地消防部门和警察局保持联络与合作，来为京东的海外业务提供安全保障和应急处理。

手机厂商vivo，也在2026年校园秋招里，列有海外市场与用户研究经理这样的岗位，不限专业，刚性要求就是外语。

你看，这类工作最核心的要求，其实都是能站在中国公司的立场，去当一个陌生市场的“本地通”。它可以依靠生活经验，但更值得依靠的是可更新、可沉淀的研究体系。

“区域国别研究院”

前两天，我读翟东升那篇聊美国资本力量轮换的采访，发现他也谈到了文科生的未来机会问题。从他给的信息看，区域国别学是一个非常有潜力的学科。

区域国别学的人才，可以帮中国的民众、中国的公司，重新以中国人的眼光和视角去解释当前的世界，而不是沿用美国教科书上的过时理论。

这种中国视角的用处，不是简单在情绪上反复撩拨“厉害了我的国”，而是去往其他国家，把中国当作案例和样本来解释。今天中国的生活对于许多国家而言是未来，所以这个时代的中国年轻人走向全球各地会有巨大优势，他们不再是去打工，而是去游历、去改造。

中国人民大学的区域国别研究院，是2023年从“国别研究院”升级更名而来的，是一个跨学科研究平台，教研人员有来自国际关系学院的、经济学院的、法学院的、外国语学院的、新闻传播学院的，也有来自外部智库与政府部门的。

大家不是用单一方法研究单一问题，而是把政治、经济、法律、文化、语言、科技等多种知识整合到同一研究框架里，为国家战略和对外合作提供系统性判断，比如评估高科技出口管制对某地区产业链安全的冲击、研究宗教文化对外交政策和社会稳定的影响等等等等。

区域国别研究，其实是把人文、历史和国际政治整合在一起，再加上各种必要的知识，打破学科框架揉进来，训练年轻人勾连全局和细节、历史和当下的能力，再用这种能力去帮中国和中国企业探路、开路。

哪个有进取心的出海企业、投资机构、财富基金、家族办公室，不渴望这样的战略智库班底呢？

更重要的是，这种交叉学科所秉持的“中国中心叙事”，也为学生们锚定了“意义”，明确为了谁的利益、什么目标而学习。这用来抗抑郁、反虚无和摆脱空心化风险，可太有用了。

区域国别研究的相关机构，是中国教育部2011年开始分批布局推动的，目前全国已经有大约200家高校有了相关院所，不同学校还各有侧重，比如西北大学聚焦西亚、北非、中亚和南亚研究，黑龙江大学侧重欧美、拉美、东北亚和中东研究。

如果你是有心规划自己持续学习之路的文科生，这类院所不妨关注一下。

## 总结

说回一个小故事。前两周，我跟一个自家晚辈聊天，问他：“你想不想成为新一代的马伯庸、张珂，或者给下一个LABUBU设计身世背景的人？”孩子马上眼睛亮了。

这孩子挺喜欢历史，但担心学历史不好就业。我跟他说：有个App叫HelloTalk，是一家深圳公司开发的跨语言社交App。它的产品定位很独特：鼓励世界各地的不同母语者进行双向的语言交流。怎么实现呢？所有用户在社区里都要注明自己的母语、目标学习语言和兴趣爱好。

可以跟任意在线的陌生人用任意语种聊天，可以发文字、语音、图片、涂鸦，也可以音频和视频通话，还可以把自己的生活故事、文化见闻发在“朋友圈”。平台在你聊天中还能提供100多种语言的互相翻译，帮你检测和纠正语法错误。

所以，当时我跟孩子说起，如果有时间，不妨去下载一个HelloTalk，试着认识几个外国朋友。聊什么呢？“每周找一个成语，或者你的历史课本的一段，讲给他们听。这个过程，你既是在巩固自己的语言和学习能力，也是在着手打磨和迭代自己的产品了。

这个产品积累三年，你会在历史这个大方向之下，逐渐细化自己的兴趣，发现自己心仪的课题和导师。如果坚持下来，只怕你还没进入大学，就已经攒出了一份让很多公司高看一眼的履历。”

如果你也有闲暇时间和兴趣，这个建议，也供你参考。

这也是我对“文科生会不会被AI淘汰”的整体看法：AI淘汰的是常规劳动者，而不是那些能为人类提供解释、安慰与方向的人。文科生的时代，从未远离，正在归来。
