
# 老美战略收缩给我们创造的巨大机遇

特朗普执政虽然还不到一个月，但是基本趋势还是很明显的，这个趋势就是主导老美执行战略收缩，这是美国登顶全球之巅近百年后首次出现的战略收缩。

## 为什么老美要主动进行战略收缩呢?

原因很简单，没钱了!或者说财政已经难以为继。拜登执政4年，美国国债增长了8万亿，现在美国国债已经突破36万亿，光是每年还利息就要上万亿，这怎么得了!按照这个趋势，一个总统任期就增加8亿的国债，最多两个任期，美元就有很大概率崩盘，所以执行战略收缩也是不得已的事情，这也是特朗普上任使劲折腾，除了民主党还在BB以外，共和党与美国精英阶层集体默认的根本原因。

比如，马斯克裁撤国际开发署，现在大家都像看笑话一样。其实这个国际开发署是美国输出影响力的重要工具，包括控制全球几百上千个NGO组织，包括控制全球主流媒体，可以说，这是过去老美掌握全球话语权的重要工具，这个道理连我们都懂，特朗普不可能不懂。

但是这个全球话语权的工具为啥现在就自己把它废掉了呢?因为这个全球话语权的工具几十年都是投入，找不到变现的渠道，或者找不到可以直接兑现利益的模式。

打个比方，过去几年很多人吭哧吭哧在短视频平台做电影切片解说，也积攒了几百万粉丝，影响力有了，但是找不到变现渠道，广告接不到，带货卖不出去了，说起来拥有几百万粉丝很风光，但是找不到变现渠道就都是净投入，所以现在这类账号现在很多都自己废掉了。特朗普现在干的就是类似的事情，国际开发署每年财政投入450亿美元，过去老美财大气粗的时候可能无所谓，现在老美自己都没有余粮都要靠借新债还旧债的时候，就不得不砍掉了。

当然这里还有党争打击民主党钱袋子的私心，但是明面上维持全球话语权却没有利益兑现的渠道却是美国精英阶层也无法否认的事实。

说实话，老美通过国际开发署掌握全球话语权却找不到利益变现的渠道根本原因还是老美制造业太拉胯，假如同样的事情放到东大，以东大当前如日中天的制造业，东大应该是很愿意每年花450亿美元来买下这个全球话语权的。

为什么?现在我们每年贸易顺差是1万亿美元，已经搞得全世界怨声载道，不光是是发达国家，即便是发展中国家也纷纷对我们加关税，发起贸易调查，如果我们掌握全球话语权，那么一方面动员各种NGO组织，一方面动员全球主流媒体，各种宣传全球化才是人类文明的希望，人权大于主权(各国人民都有享受东大廉价商品的权利)——一个主题，就是为东大商品畅通无阻进入全球各个国家制造舆论氛围。

有这个氛围，不但我们面临的贸易摩擦会小得多，更为重要的是，最多5年之内，我们贸易顺差还很有希望再增长50%以上，达到1.5万亿美元——当然代价就是全球很多国家制造业被我们摧枯拉朽的干掉，从这个角度来看，一年花450亿美元掌握全球话语权真的就是毛毛雨。

也就是说同样的话筒掌握在老美手里，就是空耗国库，掌握在东大手里，就能产生巨大的利益变现。现在老美主动把这个话筒交出来，这是好事，也是机遇，希望我们未来可以争一争这个话筒。

回到之前的主题。特朗普不仅仅是干掉国际开发署，FBI，CIA也被大规模裁员，情报部门也是老美传统输出影响力的抓手，过去几十年，老美在各国搞颜色革命光靠NGO可是不行的，必然有情报部门跟进，比如2019年的香港骚乱，CIA就活跃得很，但是维持这个抓手耗费的资金也是惊人的，每年至少大几百亿美元，既然，总体是战略收缩，所以这两个情报部门被裁员瘦身也是一种必然。

国际开发署废了，情报部门大裁员，这两项每年能给美国联邦政府省下上千亿美元，看起来是一笔不小的数字，但是与美国联邦财政的亏空相比还远远不够。拜登执政期间，每年财政赤字最小是1.3万亿，最高是2.7万亿，这里有疫情经济刺激的因素以及援助乌克兰战争的因素，撇开这些因素不论，现在老美每年财政赤字至少还有1.5万亿左右的亏空，特朗普还要想办法。

首先是联邦机构大裁员，这块特朗普与马斯克正正在干，估计还能剩下上千亿美元，还是远远不够，现在还能动手的还有两块费用。这两块一个是利息支出，每年上万亿美元， 一个是军费支出，每年超过9000亿美元。

利息支出要降低就要美联储降息，但是特朗普目前三板斧偏偏都是刺激通胀的，所以美联储目前是不敢降息的；然后就是军费。美军军费大家都知道腐败很严重，但是历届政府都毫无办法，美国国防部连审计都无法完成，又怎么可能去解决普遍存在的腐败问题呢?特朗普可能会选择另一个思路。

根据彭博社报道，白宫正在考虑撤出在第一岛链的大批美军，做出交换，华盛顿要求北京减少在该地区的海警船只数量。

这是什么意思呢?

美军每年军费近万亿美元，这笔钱大部分是花在驻外美军上了。美国在海外有800多个基地，这800多个基地就是800多个黑洞，趴在美国财政上不断吸血。美军在海外的基地有点类似我们过去的国营企业，什么都管，包括基建、幼儿园、学校、电影院，目的是为了维持海外美军的生活品质，但是造成结果就是财政负担极重。

老美每年花销近万亿美元，也无法维持海外800多个基地的开销，这几年美军成了欠款大户，什么基建费用、搬家费用等等欠了一屁股债，甚至屡屡出现海军拖欠水兵工资的事件。

所以，现在特朗普必须要考虑战略收缩，将海外基地驻军大幅度削减，财政也确实无法供养海外800多个基地的存在——虽然这些存在是维护美国全球军事霸权的支柱，但是这个支柱同样面临无法利益兑现的问题，所以也必须进行收缩。

在欧洲、在东亚、在中东、在非洲，老美都不得不进行收缩，这不以某个人的意志为转移，事实就是，一旦一个全球性帝国维持全球霸权的成本远远大于其收益的时候，收缩就是一种历史的必然，没有特朗普也会其他人来干。

看看现在特朗普在俄乌冲突表明的立场，美国在欧洲收缩的态势非常明确：
——美国停止对乌克兰的援助，未来援助乌克兰主要靠欧洲自己；
——为停火之后，美国不会派出军队去乌克兰，这个由欧洲自己派，而且这些军队如果与俄罗斯擦枪走火，不会触发北约集体防御的条款；
——北约国家要将国防费用提高到GDP的5%，潜台词就是美国会大幅度压缩在北约的投入。

同样的，我认为特朗普在东亚收缩也是大概率的，毕竟第一岛链解放军军事优势很明显，在这个区域内驻军与其说是对东大的威慑，还不如说是东大的人质，耗资巨大却达不到应有效果，反而成为财政的负担，所以在东亚收缩也是一种必然。

一旦，美剧内在东亚大规模收缩，那么我们就可以好好给美国谈一谈台湾统一的问题。

要解决财政的亏空，除了节流还要开源。特朗普开源的手段最初是打算加关税。美国每年进口3.2万亿美元，如果普遍加10%关税就是3200亿美元，如果加20%关税就是6400亿美元，这个来钱就是副作用大，主要是老美现在通胀有点控制不住。

所以现在特朗普换了一个思路，这个思路就是以加关税为威胁，一方面逼迫制造业向美国转移——台积电已经跪了，另一方面就是逼迫各国进口更多的美国产品，只要美国产品能向全世界卖的更多，也能给联邦财政带来更多的税收。

为了更多的捞钱，特朗普现在是彻底不要脸了，2月13日，特朗普签署行政令，暂停执行《反海外腐败法》。这个法令过去主要是约束美国企业在海外做生意的时候，不要使用腐败手段，比如给行政部门行贿以换取订单，这样干过去就会触发这个《反海外腐败法》——就会被美国行政机构惩处。

现在特朗普直接废掉这个法案，意思就是鼓励老美企业在海外可以不择手段去做生意——只要能给联邦政府多交税，特朗普现在是不管这个钱是怎么来的。

潜伏里谢若林有一句名言，这里有两根金条，你给我说那一根是高尚的，那一根是卑鄙的?特朗普就是这个态度。

特朗普这种要钱不要脸的态度对我们也是一种机遇，要不我们找个时间聊一聊，把东南那个岛开个价卖给我们如何?
