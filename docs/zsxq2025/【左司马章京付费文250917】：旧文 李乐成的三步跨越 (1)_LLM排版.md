
# 李乐成的三步跨越

原创：也无风雨也有情公众号：左司马章京25年3月11日

## 前言
希望在李乐成同志的带领下，工信部能够取得超越2008年的辉煌成就。

## 李乐成的三步跨越

### 第一步从管理工厂到管理地方

李乐成1984年毕业于华中工学院，该学校后来改名为华中理工大学，在2000年和同济医科大学、武汉城市建设学院合并为华中科技大学，成为“211”和“985”和双一流高校。与李乐成一样从华中工学院成长为正部级官员的就有：前教育部长周济(1970年毕业)、前河南省委书记谢伏瞻(1980年毕业)、前陕西省委书记、江苏省委书记、第十四届人大三次会议发言人娄勤俭(1982年毕业)、前湖北省政协主席、安徽省政协主席张昌尔(1982年毕业)、前重庆市市长、安徽省政协主席唐良智(1983年毕业)、甘肃省长任振鹤(1988年毕业)等六位。

19岁的李乐成的第一份工作，是进入湖北荆门市沙洋机械厂担任技术员。在接下来的十余年里，他凭借扎实的专业知识和卓越的管理才能，逐步晋升为厂长，并成功主导多项技术改造项目，使这家地方国企在市场化浪潮中站稳脚跟。

1996年12月，李乐成调任荆门市机械冶金工业总公司副总经理。从此时到2002年间，他先后出任荆门市机械冶金工业总公司总经理、湖北东光集团公司总经理和董事长等职务，成为荆门市支柱国企领导，为转型奠定了雄厚业绩。2002年1月，他出任荆门市委常委、市总工会主席，完成了身份和职业的第一次重要转换。在此期间，他还多次赴国内外参加高级管理培训，包括2003年9月在哈佛大学肯尼迪政府学院的公共管理课程，为其从技术专家向复合型官员转变奠定了坚实基础。

李乐成在荆门工作6年，历任组织部长、常委副市长等职务，多岗位历练，为日后肩负起更大责任打下经验基础。

### 第二步从管理一方到主政一方

2008年2月，43岁的李乐成接替升任宜昌市委书记的郭有明，升任宜昌市长，迈出了主政一方的关键一步。宜昌地处长江上游与中游的结合部，是湖北重要的航运枢纽城市，是三峡工程、葛洲坝水利枢纽工程所在地，被誉为“世界水电之都”。宜昌因此也备受国家领导重视，参与“三峡工程”(1994年-2012年)的宜昌市委书记罗清泉、孙志刚，其后分别升任湖北省委书记、贵州省委书记，其中罗清泉在宜昌工作时间长达11年。

李乐成调任宜昌市市长，面对三峡工程后续发展与产业升级的双重挑战，他提出了化工产业精细化转型的战略思路，积极引进新材料、生物医药等新兴产业。在他的领导下，宜昌市年均增速连续五年超过12%。

2013年3月，李乐成接替升任副省长的许克振，调任省发展改革委主任。2013年11月27日，据中央纪委监察部网站消息，湖北省副省长郭有明涉嫌严重违纪违法，接受组织调查。

### 第三步重回地方大展拳脚

在湖北省群众路线教育实践活动中，省发改委是省委书记李鸿忠的联系点。在李乐成转任湖北省发改委主任4个月后，省委书记李鸿忠就到联系点省发改委进行专题调研。他对省发改委的工作给予充分肯定，要求他们做带头学习的模范，把总书记对湖北的关怀支持、温暖厚爱，转化为推动湖北省深化改革、进一步做好经济社会发展各项工作的强大政治动力和精神动力。

在湖北省发改委主任任内，李乐成统筹全省“长江经济带”战略落地，推动武汉城市圈与“宜荆荆”城市群协同发展，其出色的规划能力得到了中央的高度认可。2017年2月，52岁的李乐成接替华中工学院“师弟”、升任浙江省组织部长任振鹤，升任湖北省委常委、襄阳市委书记。

襄阳市是湖北的副中心城市，襄阳市委书记在湖北官场具有重要地位。从1983年设立襄樊市(2010年改名为襄阳市)开始，到2017年李乐成出任襄阳市委书记，前后有10人出任襄阳市委书记，其中有5人升任正部级领导岗位。他们分别是阮成发(原云南省委书记)、唐良智(原重庆市长、安徽省政协主席)、王君正(原新疆建设兵团党委书记、西藏省委书记)、任振鹤(甘肃省长)、李乐成。

李乐成出任襄阳市委书记，提出“制造业立市”的战略目标。依托东风汽车等龙头企业，他大力打造新能源汽车产业链引入华为、比亚迪等巨头布局智能网联汽车生态。至2021年离任时，襄阳汽车产业产值突破3000亿元，跻身全国十大汽车工业城市。

2021年1月，他接替升任湖北政协主席的黄楚平，调任省常务副省长。2021年4月25日，李乐成和应勇、王晓东等省领导一起参加了老领导罗清泉的葬礼。

2021年10月，李乐成北上辽宁，担任省委副书记、代省长。面对东北振兴的深水区，他提出“以新型工业化重构产业生态”的发展战略，推动沈阳机床等老牌国企混改，布局人工智能、高端装备制造等战略性新兴产业。同时，他力推“数字辽宁”建设，三年内建成5G基站超10万座，数字经济占比提升至48%。2022年1月正式当选省长后，他主导的“山海关营商环境优化工程”吸引了超过千亿民营资本入辽，扭转了“投资不过山海关”的固有印象。

人民日报报道，辽宁以科技创新引领产业创新，攻克高端装备、新能源、集成电路装备等领域一批“卡脖子”难题，世界首款四座氢内燃飞机原型机在沈阳首飞，全球首制7500立方米液态二氧化碳运输船在大连交付，新一代隐身多用途歼击机歼-35A惊艳亮相……辽宁省目前拥有沈阳机器人及智能制造产业集群、沈阳航空集群、沈阳大连工业母机集群、大连盘锦绿色石化集群和等4个国家先进制造业集群。2024年大连GDP突破9500亿元，有望在2025年成为东北首座万亿元城市。

2025年2月的最后一天，李乐成履新工信部。工信部是推动数字经济和实体经济结合，实现智能化、绿色化、数字化转型的重要管理部门。在2008年中央4万亿投资计划中，工信部承担了振兴车市、推动电子信息产业发展(3上市)、装备制造等行业升级改造等多个领域的重点工作。

希望在李乐成的带领下，工信部能够取得超越2008年的辉煌成就。
