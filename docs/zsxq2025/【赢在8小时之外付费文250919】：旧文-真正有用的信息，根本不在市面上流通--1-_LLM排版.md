
# 真正有用的信息，根本不在市面上流通

原创：赞赏U 君公众号：赢在8小时之外25年5月17日

去年张雪锋老师推出了“高考志愿咨询服务”,售价近2万元。课程一推出，很快就有上万个家长蜂拥而至，纷纷报名。有人说，这明显就是割韭菜项目，那些家长是不是脑子进水了?还有人说，现在互联网和人工智能这么发达，自己的孩子适合报考什么专业，你上网搜一下不就行了吗?这就涉及到一个非常经典的问题了：真正有用的信息，会在市面上免费流通吗?

答案是否定的，如果你的父母是信息闭塞的普通上班族，那么你在面临报志愿的时候，还真得付费请教高人。而且，在这种关键性的问题上，Al 也帮不上什么忙，它只会给出一些冠冕堂皇的回答。

再比如，我想学习一门编程语言，我就需要知道现在市面上什么样的程序员最稀缺、收入最高。同时，我还要客观评估自己的智商，以及那个编程语言的难度如何，然后我才能全身心地去学习。这两条信息，我能免费从网上查询到吗?答案是否定的，我只能付费多咨询几个经验丰富的IT从业者，从他们那里获取最一手的信息。另外，我也会付费加入一些大神程序员组织的社群，每天要关注，他们平时在聊什么内容，我要从那些只言片语里获取有用信息。最后你会发现，真正有用、有价值的信息是不会在市面上流通的。

不要觉得现在互联网和AI很发达，穷人也能轻易地获取到有用信息。大错特错。穷人或者那些没有付费意识的人，每天只能吸收那些垃圾的、有毒的信息，会继续强化他们的愚昧。

下面我就跟大家分享5条、针对穷人的、关于获取信息的心态转变。

如果你不在认知和心态上发生这5个转变，那么你将会一直沉浸在有毒的信息粪坑里，不能自拔。

## 1.必须养成付费意识

首先我想请你反思一下：在一天的时间里，你观看的视频、收听的音频、阅读的文字、参与讨论的群有多少比例是付费的?如果上面提到的这几种信息类型，你都是获取的免费信息，那么你不可能做出什么英明决策。就像我刚才提到的那两个案例：真正有能力为别人指导“报考大学、选专业”的人，怎么可能会免费指导别人?真正知道“什么编程语言好上手、而且市场上奇缺这种程序员”的大佬，怎么可能会把自己的经验免费分享给别人?他们大概率会组织收费社群，或者有偿咨询，你需要做的就是花钱把他们的信息买过来，为自己所用。只要你有付费意识，你做任何事情都会事半功倍，少走很多弯路。

就拿U 君自己来说，我在“学习日语”这件事上，一共才花了几千块钱，然后就顺利通过了N2考试(最低等级是N5, 最高等级是N1)。因为我非常清楚，在学习一门新外语的过程中，起步阶段是最艰难的，靠自学很难突破那些瓶颈。但是，有个老师带着自己学习，那就轻松多了。很多人也知道，学习技能、学习语言要花钱找个老师，但是在“获取关键信息”方面，就舍不得花钱了。在我看来，信息比技能还要值钱，别看一条信息只有三言两语，甚至是简单的yes 或者no, 但是如果是某个大佬提供的信息，那它就价值连城。所以，如果你也是底层穷人，一定要大胆地花钱购买技能、购买信息。

## 2.关注大佬分享的只言片语

我在四年前就开始定投“纳指”和“中概”这两支鸡金了(大家可以在我公众号里搜索这两个关键词。是搜索，不是后台私信，也不会评论)。在过去这4年里，我至少写过10篇文章，来阐述我之所以会买这两支鸡金的逻辑。但是呢，每次我都会加上一句话：我不推荐任何人入场，因为任何一支鸡金和骨票，在未来都有暴跌的可能性。蠢货听了之后可能就害怕了，聪明人听了之后可能就跟投了。结果在2022-2025 这3年里，纳指和中概分别增长了77%和45%。当然，作为自媒体从业者，我为了让文章能够顺利通过婶核，我还是要加上一句：“我不推荐大家盲目买入，任何一支骨票或者鸡金在未来都有可能暴跌。”其实你去看那些大佬写的文章，基本也都是这种模棱两可的，他们不会明确告诉你：就买这支，就买那支。一方面，平台不允许推荐骨票；另一方面，未来是涨是跌，确实没人能预测。万一你买了之后，就一路暴跌呢?你肯定去找“推荐者”算账。所以，你真正要学习的是那些大佬思考的逻辑。

比 如，U 君自己的投资逻辑是这样的：如果一家公司是在“非发达国家”注册和上市的，那么我肯定不会买它的骨票。我之所以愿意持有中概鸡金(里面是京东、美团、阿里巴巴、腾讯、携程这些公司的骨票),因为这 些公司大部分都是在某个海岛注册(比如开曼群岛)、在美国上市的。如果某家公司在美国上市，而且用户高达几亿、十几亿，那么我还是很看好它的。而且，我自己受巴菲特先生的影响很大，我是坚定的“长线持有者”“价值投资者”“定投者”。我之所以愿意持有苹果、微软、丰田、三星这类公司的骨票，是因为他们在全世界都卖的很好，深受数十亿用户的喜爱。所以，我可以做到一个月不看自己的账户，我从来不会考虑什么K 线、止盈、止损这些技术性名词。当然，如果某个大神玩的是短线交易，人家就是研究 K 线的，快进快出，最后赚到钱了，那你也可以学习他的交易逻辑。

总之呢，每个人的交易逻辑和思考方式都不一样，你也要在模仿的过程中，逐渐找到属于自己的交易逻辑。不光是做交易，做自媒体、社群、网赚、变现时，你也会碰到很多“流派”,你必须在学习的过程中找到属于自己的流派和风格。风格确定下来之后，接下来你吸收的信息才会真的入脑入心。

## 3.高价值信息的生成成本极高

如果你也在某个行业里取得过一定的成果，你应该清晰地意识到：我自己是付出了巨大的成本和代价，才得出的这个信息，我肯定不会把它免费分享出来。因为在大多数情况下，把自己积累的宝贵经验免费分享出来，那么自己的利益会受损，运势也会变差。这听起来像是玄学，其实是天道。辛辛苦苦获得的信息和经验，一定要卖个好价钱，否则就辜负了自己的辛苦付出，天诛地灭。
