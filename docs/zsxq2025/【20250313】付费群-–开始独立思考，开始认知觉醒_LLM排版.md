
# 开始独立思考，开始认知觉醒

最近几年，网上有一句特别流行的话，叫你的认知配得上你的苦难。经常有网友用这句话来挖苦人。但是认知和苦难之间，其实并没有什么直接的关联，二者之间并没有一个直接的逻辑关系。

举个最简单的例子，前段时间因为 被禁而涌入 TIKTOK，中国小红书的大量美国难民，他们发布的内容，有些人的认知真的是只能用逆天来形容了，跟钟国韭菜比也不相上下了。老钟韭菜跟这些老美大聪明在一起，那简直就是卧龙凤雏。唉，那你说如果真的是认知决定苦难的话，那这些老美大聪明，为什么从小到大就没吃过老中韭菜吃过的苦呢？所以一个人苦不苦，跟认知高低其实没什么关系。

那么问题就来了，既然认知和苦难之间没有什么直接的关联，那为什么我们还要提升自己的认知呢？为什么独立思考，并不能改变我们自身的处境，我们还是应该学会独立思考呢？

对于我们这些根本就无力改变社会环境，甚至无力改变自身境况的人来说，到底有什么意义呢？有一本特别棒的小说就回答了这个问题，那就是来自德国诺贝尔文学奖得主赫尔曼·黑塞的《德米安· 彷徨少年时》。

赫尔曼·黑塞是一名德国作家，今天就让我们走进黑塞的作品，通过这本书提供的视角，再一次来审视我们自身，相信大家一定会有所感悟的。

德米安这本书的故事就发生在作者所生活的20世纪初的德国，故事的主人公是一个名叫辛克莱尔的德国男孩，辛克莱尔出生在一个物质充裕的家庭中，在贵族学校上学，同学全都是富二代和官二代，所以按道理说，这样的他只要老老实实听父母的话，听学校老师的话，按部就班的成长，就可以顺理成章地过上优渥的生活，永远都不用跟不属于他的另一个世界的，社会底层的那些人打交道。但是小辛克莱尔却偏偏在对面的公立学校中，交了一群社会底层的狐朋狗友，还为了融入他们而撒谎吹牛。

谎称自己曾经在一个农户那里偷了一袋苹果，然后社会就给他好好的上了一课，他自以为的朋友，一个名叫克罗莫的穷人家的孩子，竟然抓住他偷窃这一把柄，威胁他，敲诈他的钱财："我亲爱的朋友，我是穷人，不像你有个有钱的爹，有钱不赚那是傻子”。就这样，辛克莱尔的生活陷入了地狱，深深地感受到了来自另一个阶层的敌意，只能不断的从家里偷钱给克罗莫。

但是，随着一个转校生的到来，一切都变了。这个转校生就是德米安。辛克莱尔是这样描绘德米安的：我时不时的看向德米安，不知为何，他的脸对我有一种特别的吸引力，聪明、俊朗、坚毅，他在我们中间很有优越感，显得很酷，还有一种极具挑衅的自信，他穿着做派都像个王子，就好像自己正屈尊置身于一群乡下男孩中间，还得和他们打成一片一样。

德米安很快就注意到了内心不那么安分的辛克莱尔，并且透过他的观察和分析，立刻就猜到他肯定是被那个叫克罗默的男孩，抓住了什么把柄，正在被他勒索。辛克莱尔自然是非常的惊讶，我什么都没说，你就全猜到了，你是会读心术吗？但是德米安就告诉他，其实只要一个人自己动动脑子去思考，每件事背后的深层逻辑，就不难知道真相。

然后德米安也不知道对克洛莫做了什么，可能是揍了他一顿吧，反正从那之后，克罗莫就再也没有出现过。就这样，通过这件事，辛克莱尔和德米安成了无话不谈的好朋友，然后呢，通过跟德米安的交流，辛克莱尔的这个新世界的大门就被打开了，三观被彻底的颠覆。

很多人应该都听说过，圣经中该隐和亚伯的故事，该隐和亚伯是亚当和夏娃的儿子，是亲兄弟，结果呢，该隐却因为嫉妒亚伯而杀死了亚伯。但是呢，上帝对该隐的惩罚却不是让他杀人偿命，而是在他的额头上留下了一个记号，让所有看到这个记号的人都被吓跑，用这种方式来禁止任何人去为亚伯报仇杀害该隐。

过去辛克莱尔从来没觉得这个故事哪里不对，但是德米安就说了，该隐杀了人，但是呢，上帝却给了他一个记号来表彰他，给他提供庇护，这也太不合常理了。所以德米安自己思考之后，他认为事情的真相很可能是这样的，他觉得呀，该隐可能只是一个很有个性的人，人们总是孤立和排挤有个性的人，但是该隐呢，却毫不在意别人的孤立和排挤，因为他不仅是一个有个性的人，还是一个非常勇敢的人，于是该隐的这种卓尔不群，就让这些胆小鬼们感到非常的不舒服，于是他们就编出了这么一个故事，来弥补自己的恐惧。

辛克莱尔听后是大受震撼呀，但是德米安的爆言还远不止如此，有一天，老师在课堂上给他们讲起了耶稣和两个强盗一起被钉十字架的故事，其中一个囚犯，因为在生命尽头被耶稣感化了，真心忏悔了自己的罪恶，于是耶稣便许诺，他将和自己一起去往乐园之中，这个囚犯在基督教中就被称为好囚犯。但是德米安却不这么想，他说他是个囚犯，犯下了可耻之罪，天知道都是些什么罪，早不悔改，晚不悔改，偏偏在死到临头的时候痛哭流涕，要洗心革面，痛改前非了，他是真的被耶稣感化了，信了耶稣，还是死到临头因为自己是个胆小鬼，所以看见一根救命稻草，就不顾一切的抓住对吧，所以德米安就说了，这无非是牧师又一个劝善故事而已，再添加些感人的佐料和背景，甜美却不真实，如果让你选择其中一个强盗做朋友，或者问你哪一个更值得信赖，你肯定不会选这位哭哭啼啼回心转意的，而是选另外那个，他才是真男人，有性格，坚持走自己的路，不会在最后一刻背离帮助他的魔鬼。

通过跟德米安交流，辛克莱尔才发现，自己读圣经的时候，实在是太没有想象力了，完全就没有自己的想法，从来就没有独立思考过，只是一味地接受父母和老师灌输的东西。而在德米安看来，顺从社会的范式，顺从主旋律，可以让生活变得轻松，所以他就觉得这些疏于独立思考和自我反思的人，归根结底就是一种懒惰。

写到这，我们也就明白，德米安的自信与高傲是从哪里来的了，也正是从这时候开始，辛克莱尔开始独立思考，开始觉醒。

很快时光飞逝，辛克莱尔上了高中，上高中之后，辛克莱尔被父母送去了一所寄宿学校，就有点类似于现在那些衡水系学校一样，他清楚父母为什么送他来这种地方，也很清楚父母想把他变成怎样的人，他自然是可以按部就班的遵从父母的意愿，变成父母期待他成为的样子，但是他不想，于是，他就通过叛逆的方式进行抵抗，从过去那个聪明的好学生，变成了一个不好好学习，满嘴脏话，整天喝酒的坏孩子。

父亲得知后自然是非常的生气，于是就威胁他，你再不好好学习，再不听话，我就把你送到管教所去，就有点类似于现在的家长威胁小孩儿，你再不听话，我就把你送到杨永信那去，送到网戒中心去，但是辛克莱尔却只为父亲感到难过，因为父亲既没有达到自己的目的，也没有找到跟他沟通的方式，所以他觉得父亲这么痛苦也是活该，到最后，还得是德米安，在一个假期，辛克莱尔终于又遇到了日思夜想的德米安，而德米安只是陪他去酒馆喝了一杯，就解开了他的心结，他提醒辛克莱尔，你现在其实就像蹲监狱一样，你再怎么扑腾也还是出不去的，但是，即使我们身在牢笼里，灵魂也依旧可以是自由的，在现实中，我们或许什么都做不了，但是，我们内心始终有一种东西，它无所不知，无所不愿，做任何事都出色。

于是辛克莱尔就改邪归正了，不再做那些叛逆的行为了，因为他意识到自己其实就像一只囚鸟一样，再怎么扑腾也扑腾不出笼子，自己应该做的是，让自己的灵魂变成一只自由的鸟儿，冲破牢笼飞出去。也正是在那之后，辛克莱尔画了一幅画，画的就是一只鸟像破壳而出一样，从一个球里飞出来，他把那幅画寄给了德米安，德米安很快就给了他一封回信，信上写着，飞鸟欲奋力破壳而出，但是世界新生者必先摧毁世界，鸟儿飞向上帝，上帝之名是阿布拉克萨斯。

辛克莱尔自然是非常的激动，因为德米安不仅看懂了他画的是什么，甚至还替他做了诠释，此时的辛克莱尔，已经逐渐成长，为了一个精神独立的觉醒的个体，拥有自己探索精神世界的能力了，可以去自己追寻自己灵魂的自由了。

于是在那之后，为了找到一条属于自己的路，辛克莱尔做了大量的工作，一定能找到一条属于自己的路，不必跟随社会的范式，不必随大流，不必听从主旋律，他相信灵魂的自由是存在的，并且相信自己一定能获得灵魂的自由，而不是身在囚笼里面，灵魂也像个囚徒。

然后辛克莱尔就发现，此时的他已经可以像曾经的德米安一样，以一种叛逆的姿态去看待那些没有觉醒的，被社会规范所驯化的人了，在世人看来，我们有些怪异、疯狂，甚至危险，这也难怪他们，我们是已经觉醒或正在觉醒的一群人，他们也在付出，他们的努力也有意义和价值，但我们是以新的、个性化的、趋向未来的形态，展现自然的意志，而他们生活在固有的妄念之中，对他们而言，人类，他们和我们一样热爱的是一件成品，只需维护和保养，而我们认为人类是遥远的未来，我们所有人都在路上，未来的前景无人知晓，其法则也从未被书写。

故事的最后，第一次世界大战爆发了，一夜之间人们全都成了兄弟姐妹，都在谈论祖国和荣誉，然后，就为了这个事实上他们也不是很理解的，抽象的东西，去上战场当了炮灰。我们都知道，德国在一战和二战中，扮演了一个什么样的角色，所以这些被社会范式驯化出来的年轻人，满脑子里面的祖国和荣誉，在我们看来也就格外具有讽刺意味，就像之前被父母送去寄宿学校一样，身在这个国家的辛克莱尔，也没有办法摆脱既定的宿命，最后也被迫上了战场，但是，和那些炮灰是不一样的，他的灵魂是自由的。

故事的最后，负伤的辛克莱尔看到德米安在自己面前，温柔地对他说，你要仔细聆听自己的声音，你会发现我就在你的心里，之后，辛克莱尔就睡着了，而当他醒来的时候，德米安已经消失不见了，或许刚才的一切只是一场梦，甚至或许德米安这个人，从来就不曾真正的存在过，但无论如何，此时的辛克莱尔已经和德米安融为了一体，他就是德米安。

古希腊德尔菲神庙中刻有一句著名的箴言，认识你自己，古希腊著名的哲学家苏格拉底，也曾说过一句类似的格言，未经审视的人生不值得活，可即使是这个世界上最有智慧的一群人，都在不断地告诉我们，一定要认识自己，一定要审视自己，在现实中，真正能做到这一点的人也寥寥无几，而且随着时代的发展，要想做到这一点已经越来越难了，因为了解自己，认识自己最有效的方法不是科学的方法，不是查体质或者基因检测，或者不断记录自己身体的各项指标，而是要通过沉思，但是，现在的主旋律，主流观点都认为，不通过科学方法获得的认知，都不是真正的认知，通过沉思获得的认知是没有办法证伪的，所以在很多人看来，通过沉思获得的认知就不是真正的认知。

于是越来越多的人就摒弃沉思了，没有人再对自己感兴趣，这其实也是独立思考能力丧失的一种体现，即使是一些自以为觉醒的，自认为有独立思考能力的人，往往也只沉迷于一些实用性的认知，比如怎么赚大钱呀、学什么专业更有前途呀，怎么润发达国家呀之类的，他们可能已经觉得自己拥有顶级认知了，觉得自己现在的生活非常满意，但现实却是，一个人再怎么牛逼，也不可能逃离自己所处的时代，我们每个人其实都是肉体上的囚鸟，秦始皇再牛逼也没法看电视、上网、打游戏，智能手机被发明出来了，你再牛逼，你也没有办法不用智能手机，无论是科技革命的到来，还是第三次世界大战的到来，你都不可能去改变什么，也不可能不被波及，事实上每个人都是囚鸟，我们每个人都是不自由的，我们其实都在被时代和社会裹挟着前行，所以无论我们的处境如何，灵魂其实都应该像辛克莱尔所画的这只鸟一样，的世界，奋力地飞出去，摧毁原本，飞向只属于我们自己的那个上帝。
