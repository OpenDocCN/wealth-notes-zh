
# 把“小长假”变成“迷你退休”，人间值得

从后天起，我们将进入断断续续的小长假三连模式。先是清明，再是“五一”，然后是端午。而且因为今年年初的春节来得特别早，也就显得春节后的工作日持续得特别久。这自然也让这个小长假变得更受期待。除了表达思念之外，你也可以趁着这几天放松放松。为了迎接即将到来的小长假三连，我们今天就聊聊旅行的方法。

今年2月，著名的科技作家，《失控》《必然》的作者凯文·凯利写了一篇文章，讲的是他自己是怎么旅行的。

凯文·凯利堪称是作家里最会旅行的人，也是旅行者里最会写作的人。过去五十多年里，他睡过总统套房，也睡过集体宿舍；坐过私人飞机，也身无分文地搭过便车，还两次骑自行车穿越美国；他参加过盛大的狂欢聚会，也独自徒步去过地球上最偏远的地方。

那么，这么多年下来，凯文·凯利有什么心得？

首先，凯文·凯利认为，旅行分两种模式。第一种是RR模式，休息和恢复。这种旅行的目的，就是放松、休息，为之后的生活积攒能量。第二种模式，叫EE，参与和体验。这种模式下的旅行，不是为了放松，而是为了冒险。你的目的是发现新事物，获得新体验，投入一场结果不确定的冒险，遇见平常生活中不会遇到的事儿。

凯文·凯利更喜欢后面这种旅行，摆脱预期，把自己放进一个意想不到的境地，去体验，去冒险，你会重新认识自己。

当然，冒险也要讲方法，凯文·凯利的建议大概可以分成四类，接下来我们展开说说。

第一，出行前，怎么做准备？

首先，最重要的不是准备行李，而是选择旅伴。假如要结伴出行，一定要选不扫兴的旅伴，遇到问题，绝不抱怨，有问题也憋着，回家再说。

其次，行李，不是越多越好，而是越少越好。旅行时间越长，你带的行李应该越轻。旅途中会一直遇到上下楼，上下车，换酒店等等情况，这时候一个大行李箱反而是累赘。很多东西完全可以在目的地购买。但有些东西是必须带的。比如，多带一件额外的套，就算去热带，夜晚也会比你想象中更冷。再比如，带几个洗衣片，出门在外，临时弄脏衣服，洗衣片可以应急。

第二，行程怎么规划？

首先，凯文·凯利认为，要想了解你的目的地，应该两手抓。一边看最新版本的当地旅游指南，好处是全面且可靠。另一边看社媒平台和论坛，互联网负责提供速度，你可以找到最新鲜的好玩的东西。

其次，规划行程的时候，可以随性一点，给自己留一些调整的余地，到时候，假如你很喜欢某个地方，可以多待一天。对某个地方不满意，也可以提前一天离开。

最后，旅行周期，怎么安排最合适？

凯文·凯利认为，12天是最佳长度，头尾两天缓冲，中间10天深度游。而且，凯文·凯利建议你，用“直奔终点法”规划整个行程。下了飞机或者火车，别在城市里逗留，直接杀到最偏远的目的地，玩够了再往回走，直到最后一站，就是返程的城市。

这样做，能让你在旅途的一开始就最大化地感受文化的冲击，容易找到本地人才懂的好地方。当你在野外玩够了之后，再回城市，也能重新感到新鲜感。

第三，最重要的，怎么玩耍，才算得上冒险，才算得上有意思？

比如，蹭一场婚礼。出门在外，尤其是去了文化不同的地方，遇到婚礼，别怕打扰，可以大胆加入。你可以给新人带一个小礼物，这对你们双方来说，都是一场惊喜。

再比如，在旅途中的某一个环节，给自己一点小限制，这往往能激发不一样的体验。你可以选一天只骑自行车游玩，或者选几天，白天去不同的城市，晚上在火车上过夜。

再比如，你可以刻意挑选一些小众的目的地。比如药房、小作坊、五金店，假如你不怕，甚至可以去参观墓地。这些都是最能直接体验当地人文特色的地方。

再比如，跟随当地的人流。假如很多的人都往同一个方向走，大概率上是有什么集体活动，比如市集、游行、派对、舞会，跟着人流走，没准儿你就能发现这些好玩的地方。

再比如，多休息。休息不是冒险的反义词，相反，意外之喜往往发生在你停下来的时候。你可能在休息的时候遇到有趣的当地人，可能在休息的时候遇到好玩的事儿，也可能发现隐藏的城市彩蛋。总之，你可以慢下来，多休息。

第四，关于美食，这也是旅途中最重要的事之一。

比如，凯文·凯利说，你打车或者包车的时候，可以直接问司机，能不能请你带我去看望你的妈妈，咱们一起吃一顿饭。假如真这么做了，你、司机、司机的妈妈，三个人都会特别开心，而且你能吃到最地道的家常菜。

再比如，吃麦当劳，注意，这不是广告，是凯文·凯利的原话。凯文·凯利建议，即使你平时从来不吃麦当劳，出去旅游也可以去试试麦当劳，麦当劳的本地化做得特别好，去麦当劳是认识当地特色的一个好方法，比如意大利推出过巧克力榛子酱汉堡，法国有法棍汉堡，日本有樱花汉堡，印度有咖喱汉堡，中国有麦辣鸡腿堡。

再比如，想在景点附近吃到便宜又正宗的当地菜，怎么找？答案是，从中心区域往外走，至少走出五个街区，这里的小馆子，通常都是既便宜又正宗。

以上就是凯文·凯利的旅行建议。他曾经说过一句话，“你的二十多岁，是做一些不寻常、奇怪、大胆、冒险、无法解释、疯狂、无利可图的事情的最佳时期。在你的余生，这些经历将成为你的灵感源泉。”

除了凯文·凯利，还有很多名人，也有很特别的旅行方法。

比如，物理学家、诺奖得主理查德·费曼，主张深度体验的旅行。费曼年轻的时候去巴西教过书，喜欢桑巴音乐，因此在巴西的时候，他学会了两种巴西当地的乐器，潘得鲁小铃鼓和打击乐器弗利吉得拉，后来他熟练到可以参加巴西嘉年华游行中的乐队表演。

费曼度蜜月的时候去了南美洲的一个小国家危地马拉，他在一个博物馆里看到了一本玛雅人写的古书，上面全是点、线和奇怪符号。然后他就开始研究这本书，发现这些点和线代表数字，接着他解读出了一些玛雅人对数字和圆周率的研究。研究到最后，他甚至做了一场主题是“解开玛雅象形文字之谜”的演讲，你看，这个延伸有多厉害。

再比如，《每周工作4小时》的作者，蒂姆·费里斯，他认为旅行是寻找灵感的途径。他自己就是一个旅居各地、四海为家的人，他在爱尔兰打过曲棍球，在中国台湾开过健身房，在中国香港和内地参演过电视剧。

再比如，美国畅销书作家雅各布斯，他把自己叫做“生活实验者”。为了搞懂宗教，他花了一整年时间，完全按照《圣经》中的字面意思生活，包括不刮胡子、绝对诚实、常常祈祷，他去体验了美国各种基督教社区的生活，甚至还去以色列放过羊。

再比如，英国维珍集团的创始人布兰森。他是极限旅行的爱好者，曾经驾驶热气球飞越过大西洋和太平洋，还驾驶帆船环球航行。后来他还想上太空，因此创办了自己的太空旅游公司，维珍银河。

再比如，巴菲特最喜欢的休闲方式是打桥牌，一个星期至少打四次牌，每次要打满两小时。他认为，这是最方便且最能锻炼思维的方式。

日本作家村上春树，雷打不动坚持长跑，他的代表作之一就是《当我谈跑步时我谈些什么》。据说他认为，正是这种方式，让自己一直保持着少年的状态。

《智人之上》的作者尤瓦尔·赫拉利，作为未来学家，身处研究信息时代的最前沿，但他最推崇的生活方式，是每年固定花几周时间去冥想，以及每年60天的信息断食。

已故的传播学家，《娱乐至死》的作者尼尔·波兹曼，他在书里批判电视文化，但实际上，他自己也会看电视，并且据说尤其喜欢体育节目。

《每周工作4小时》里有一个重要的理念，叫做“迷你退休”，也就是工作一段时间后，你就要给自己一个假期，离开常规的生活，通过旅行去寻找另一种可能。而为了实现迷你退休，你必须提高自己的效率，精简自己工作模式，把重要的时间留给重要的事。

换句话说，思考的高手，大都也是玩的高手，反过来也一样。当然，你不必非得去参考谁，毕竟，假期的价值之一就在于完全自主的留白。就像蒂姆·费里斯说的，用25%的时间发现你自己就好，剩下75%的时间，去创造你自己吧。
