
# 司令4.23特朗普内外交困

特朗普内外交困，早晚是顶不住的，拖得越久越会众叛亲离。日本前恭后锯最能说明态势的变化，这就像一个大户人家开始出现破败的时候，外面的合作伙伴很难一下子看出来，因为宅院依然树大根深，金碧辉煌。但他们家的下人和狗，感受最灵敏：下人可以从主人对自己的态度，狗可以从狗盆中的食物质量判断出主人的状况。

没有根基就只能不断的赢，哪怕赢了九次，最后一次输了，也会很危险。一个社会的内部结构越是无法逆转，相互对立的人群就越缺乏妥协的余地。任何存量中的状况就是如此：一方多吃一口，另一方就少吃一个，彼此都必须在惯性的裹挟下往前走，没有办法凭借自己的力量改变生存处境，也没有办法凭借自己的力量扩展生存空间。

如果你观察社会上人群的诉求，你就会发现一个非常有意思的现象，一方面许多人控诉资本家剥削自己，压榨自己，让自己活得很压抑，喘不过气了。当资本家用无良工厂替代有人工厂，用运输机器人替代开叉车的工人，用机器臂替代人工流水线，用低空无人机替代配送员。许多人又会骂骂咧咧，怪资本家不再剥削自己! 这是一种充满悖论的诉求，却这种悖论却无处不在，之所以会这样是因为那些人无法改变自己的生存结构和生存状态，只能在惯性的作用下一路苟且，而环境的变化，又让自己既无力控制，也无力适应，于是就产生了充满悖论的诉求! 一群是如此，由一群一群的人组成的整个社会也是如此，什么叫生存博弈呢?这就是最直观的反应!
