
# 俄乌谈判陷入僵局

特朗普上任之初承诺百日内结束俄乌冲突，对于这个承诺我当时还是比较看好的，原因有两个，一个是特朗普亲俄，一个是俄罗斯经济因为几年战争的巨大消耗已经非常困难。但是现在100天已经过去了，俄乌谈判却陷入僵局，结束战争看起来还是遥遥无期。

现在回头盘点，我大概率是忽略了一股举足轻重力量的影响。这股举足轻重力量就是东大。对于俄乌冲突东大的策略现在盘点是非常清晰的。

特朗普上台，情况出现变化，一开始特朗普团队撇开欧盟与乌克兰与俄罗斯单独谈判，很明显，美国这个筹码已经开始向俄罗斯偏离，对乌克兰经济援助断绝，军事援助时断时续，俄罗斯慢慢取得战场的优势，现在俄罗斯成为天平强势的一方，欧盟与乌克兰反而成为弱势一方。在这个时候，东大这个砝码开始向欧盟与乌克兰加注。2025年东大与乌克兰外交关系开始积极互动，高层沟通频繁，特别是在推动乌克兰问题政治解决上达成更多共识。大家想一想，为什么过去达不成的共识到了2025年突然就能达成共识呢？另一方面，与俄罗斯双边贸易趋势也逆转，1-4月双边贸易同比2024年下降7.5%。虽然因为与老美贸易战，东大禁止了老美能源进口，是宝大宁愿跑到，犹又又早。
