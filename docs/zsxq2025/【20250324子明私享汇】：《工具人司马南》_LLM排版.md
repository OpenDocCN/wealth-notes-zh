
# 《工具人司马南》

中国历史上有一个不断循环的规律，每当有一批功臣集团做大之后，都会威胁国家的统治，并不断侵占普通民众的利益，成为国家统治上最大的威胁。可功臣集团平素的名声太好，皇帝想打击他们又怕影响太坏，便会突然变得宠信小人，任由这些跳梁小丑不择手段的打击功臣集团，待到功臣集团纷纷被打击和分化，势力弱到不再能对国家和民众造成威胁，朝堂之上重回平衡之后，皇帝就会英明神武的出手打击小人，让小人成为功臣集团的出气包来泄愤，历史的巨轮就是在这么不断的循环中前进。而前几年以打击民营企业而闻名的司马南，就是这个必要的工具人之一，没必要骂他，也没必要怀念他，就像替嘉靖胖揍群臣的陈洪那样，大家都只是扮演好了自己该扮演的角色而已。配图：陈洪 大明王朝 1566 司礼监秉笔太监
