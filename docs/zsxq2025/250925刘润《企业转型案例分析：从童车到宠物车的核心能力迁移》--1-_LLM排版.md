
# 企业转型案例分析：从童车到宠物车的核心能力迁移

今天分享我在甘肃为平安的客户分享了一些关于商业实际的看法，并进行讨论。在讨论过程中，有一位企业家提出了一个问题。他说：“你看，我这个行业和以前相比，完全不同了。现在行业下滑得非常厉害，我应该怎么办？这个行业已经彻底不如以前了。”

我回答：“那怎么办呢？你就要考虑转型。”

他又问：“怎么转呢？可是怎么都不好转。”

我说：“我给你举个例子。你觉得现在哪个行业最难做？” 

他说：“最难做的行业之一，就是母婴行业。”

为什么母婴行业那么难做？原因其实很简单。2016年，中国生育高峰时有1800多万人口，之后就开始下滑。因为2016年放开了二胎政策，大家本以为会出加微信top28688，进资料分享群现报复性生育，人口能涨到四千多万，结果只剩下1800多万，然后就开始持续下滑。2017年降，2018年继续降，一直到去年，2024年，降到了900万左右。也就是说，从1800万降到900万，减少了一半。

这意味着每年出生的孩子都在减少。比如，2016年中国能卖出100罐奶粉，到2024年就只能卖出50罐奶粉了。你说，这行业难不难做？大家觉得很有道理，确实很难做。

那怎么办呢？还是要转型。转型的核心是什么？就是要用自己真正的核心技术，去满足另一个完全不同的客户群体。

日本有一家叫做Airbuggy的公司，大家可以查一下。这家公司是做儿童童车的。你看，刚才那位客户提到童车，就能明白这个行业为什么会下滑。因为日本和中国一样，面临老龄化和少子化的问题，而且日本的老龄化、少子化比中国来得更早。

我经常建议大家有机会可以去日本看看，因为你能看到他们为了应对老龄化和少子化，提前尝试了很多年，有些方法有效，有些无效。你可以观察哪些是有效的。Airbuggy就是面临少子化带来的巨大挑战。孩子少了，童车的需求自然就减少了，那么怎么办呢？他们也在思考转型的问题，就像刚才那位企业家问的“怎么转”。

关键是要找到自己的核心能力。你要研究童车，童车的核心能力到底是什么？一定要把自己的能力从产品里抽象出来。记住这句话：一定要把自己的能力从产品里抽象出来。

他们在思考，自己的真正能力是什么？后来想明白了，他们说：“我们的核心能力，其实是能够照顾一个不会说话的人。”什么意思呢？比如你照硕一个大人，他会说哪里不舒服，哪里疼，你可以交流，能帮他解决。但孩子不会表达，你要用产品和服务去照顾好一个不会说话的人。

孩子无法准确表达自己的感受，比如问他哪里疼，他只会哭闹。面对这样一个无法交流的主体，要照顾好他是非常困难的。比如坐童车，孩子不会说舒不舒服，但你要让他坐得舒服，这就需要独特的洞察力和同理心。这才是他们的核心能力。不要只认为自己是做童车的，实际上，你是要照顾好一个不会表达的物种。

那么，世界上还有什么物种是不会表达、也需要照顾的呢？孩子少了，还有什么物种需要照顾呢？后来他们想到了宠物。宠物不会表达，也需要被照顾，而且现在很多家庭不生孩子，转而养宠物。宠物就是孩子的平替。

于是他们想，为什么不做一个宠物车呢？他们开始做宠物车，把童车改造为宠物车。以后你在公园看到有人推车，走近一看，发现不是孩子，而是一只吉娃娃或一条狗。因为有些狗年纪大了，走不动了，但跟家里人感情好，主人想带它出去，所以需要宠物车。他们用核心能力去研究，怎么让宠物坐得舒服。宠物舒服的方式可能跟孩子不一样，但他们用同样的能力去解决这个问题。

他们推出了宠物车，售价高达600美元，并成功打入日本和韩国市场。这两个国家的宠物现象尤为突出。2023年至2024年间，日本和韩国的宠物数量首次超过了0至15岁儿童的数量，即“毛孩子”比孩子更多。随着孩子数量减少，毛孩子数量持续增长，他们因此进行了重大转型。

这一转型的核心在于，他们不再局限于儿童群体，而是将目光转向了宠物市场。他们发现，自己的核心能力并非产品本身，而是将产品做出来的能力。通过抽象出这一能力，他们找到了新的客户群体——宠物主人，并将这一能力迁移到宠物车领域，取得了巨大成功。

转型的本质在于，找到自己的核心能力，并将其应用于新的客户群体。不要将自己局限于产品，而是要抽象出产品背后的能力，看看哪些其他群体也需要这一能力，然后进行迁移。这就是转型。

今天下午，我与平安进行了两小时的分享，回答了相关问题。晚上，我们还在星空夜话中，与主持人和大家一起讨论了许多话题。当前的企业家确实面临着前所未有的挑战和焦虑，但如果不转型，就是等死；转型则是寻找新的生机，就像角马必须迁徙一样。

就像角马迁徙时，必须跨过马拉河，一部分会死在途中，但这是必经之路。真正的优秀企业家可能会选择“找死”，因为原地等待必定是死，而出去还有生的机会。所以，我祝愿每一位企业家都能转型成功，找到属于自己的生机。
