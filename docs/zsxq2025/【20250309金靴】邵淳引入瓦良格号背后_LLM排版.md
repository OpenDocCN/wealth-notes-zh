
# 邵淳引入瓦良格号背后

关于邵淳为国家买来瓦良格号的故事，尤其是细枝末节，up主“温竣岩”的视频已经几乎全景呈现，推荐各位同志观看。而那个视频没有具体细说（公域平台也不可能细说）的部分，其实正是邵淳、贺鹏飞那帮心系祖国的大佬们始终难以克服的真正“内部阻力”之所在。

瓦良格号这个超级项目工程，从一开始就带有“爹不疼、娘不爱”的孤者色彩，否则那个投机商人徐增平也不至于在嗅到商机后、揣着这么一个香饽饽（全舰完工率67.3%、配套全部设计图纸、乌克兰工程师愿意来华辅佐续建）却竟然在各路领导面前屡屡碰壁、寻资无门，最终只能托赖拳拳爱国心的邵淳一行人。

这背后，有一浅一深两大历史原因在施加阻力……

首先说浅因：老猪的李鸿章式短见。

这个我昨天在微博上已经隐晦点出了：“一个证券公司，它买什么航空母舰，他肯定个人有好处!”、“胆大妄为，严肃查出，以敬国法，以敬效尤!”一声令下，搁置数年。

是什么最后把这两句话给“诈”掉的呢? 1999年5月的南联盟使馆惨案。然后，他插手了，直接要求引进瓦良格号。阻力，终于消失。所以可以明白为什么今天邵老的逝讯并没有得到其地位和贡献应有的网络热度了吧? 包括网友们提议“为他拍一部电影”、“要拍出当年的困难层层、阻力重重”……这压根是不可能的。瓦良格的传奇故事早已流传二十年，然而文艺界至今未获允许或支持拍摄出一部相关的主旋律影片，原因无需多言。无论如何，邵老千古。

除了微博上这两句，老猪当时还直言不掩：“中国不对外侵略，也不准备搞航空母舰…本届政府不考虑这个问题。今后不要再打报告要求造航母了。”

在那时，老猪的全部心思都在经济体制转轨和改革国有企业上面，这项工程同样需要中央财政耗费巨额资金。早在1992年稻哥南方讲话“重新为改革开放指明方向”之后，全国经济迅速发热——1992年中央印发货币1158.2亿元，同比增长一倍；次年更是达到1528.7亿元，创下历史新高，各地投机倒把兴起，诸国有银行都亲自下场创办信托和证券，导致储蓄迅速耗尽而贷款急剧增加。到老猪掌握实权时，赤字已然形成。

那个时候有“四热、四高、四紧、一乱”之说——四热：房地产热、开发区热、集资热、股票热；四高：搞投资膨胀、高工业增长、高货币发行和信贷投放、高物价上涨；四紧：交通运输紧张、能源紧张、重要原材料紧张、资金紧张；一乱：经济秩序特别是金融秩序混乱。

当时的财政部部长王丙乾迫于无奈，开始在财政会议上搞“摊派”，每个省被强行分配指标，从1000万到1亿不等。但由于前三次中央“借钱”都是有去无回，地方的积极性极低，甚至有个别富裕省份的财政厅长直接顶回去：“跟我要钱，我可没有”。

1993年在海南召开的一次座谈会上，接任财长的刘仲藜打比方说：“李XN同志任部长时有上衣和长裤穿；王丙乾同志任部长时，还有衬衫；到我这儿只剩下背心和裤衩了。”

会上有同志跟他开玩笑说：“您可能连背心加微信jntsg2，免费进资料分享群都没有，只剩下裤衩了。”那一年，消费者物价指数（CPI）狂飙到24.1%，为改革开放以来历史最高。也是那一年，中国外汇储备仅剩212亿美元，减去短期债务余额135.46亿美元，还剩76.53亿美元，连支付那年的贸易逆差（122亿美元）都不够。

财政、银行、外汇，三大赤字全部形成，这确实是老猪彼时面临的现实窘境。所谓“中国崩溃论”的西方智库和国际机构唱衰中国的论调也是从那时候诞生。

从数据看，短短十年时间，中央财政收入占比由1984年的40.5%迅速下降到1993年的22.02%，收支平衡压力急剧加大。1993年7月，全国人大任命怪物院常务副总理的老猪兼任央行行长，他随即在财政金融领域开始改革。

在全国财政工作会议上，老猪表示：“中央如果出现财政赤字，不能到银行透支。到银行透支就是发钞票，就会引发通货膨胀。这也是间接限制财政支出规模不能搞得太大。”在1995年后，财政赤字不再通过中央银行透支，整体上守住了3%的赤字率预警线等。这是老猪“抠门”的经济层面诱因。用他自己的话说：“问题不在于有没有赤字，而是这个赤字水平是否在承受能力以内，特别是赤字用在什么地方。”

显然，购买航空母舰这样回报周期相当之长、甚至从经济层面压根没有数字性和财报层面回报的事情，不可能得到他的支持甚至允许。

在老猪掌握最高经济大权的1998年-2002年，中央累计发行了5000多亿长期建设国债，用以重建被他推翻肢解的国有经济以及由此衍生的经济转轨和产业转移（工业重心从东北和西部全部向长三角、珠三角转移），这是他政绩规划的重中之重。所以看似“区区”数千万美金的瓦良格购买费，在老猪的思路中，自然是太昂贵太肉疼了——强军?关俺老猪何事!

更重要的，除了经济层面的狭窄视角，老猪考虑的还有一层：“国际观瞻”。即：购买航空母舰会否引起西方国家（以及日韩）对中国的观感。这会影响到他心心念念的入世大计。

早在1993年前后中国申办2000年奥运会失败时，就有帝国主义阴招作祟（八平方的蔓延影响，从1990年北京亚运会开始就在抵制中国举办国际盛会），这在当时就“吓住”了不少党内的投降派、开明派、与国际接轨派，其中就包括老猪。

这也就是为什么在连续发生银河号事件、台海危机等事端的背景下，1999年春天老猪访美竟然会一语惊人：“我这次来，要让国人消消气。”指望他支持购买航母?实在天方夜谭。

他是稻哥一手提上来用以“监军”的——监谁，当然无需多言。因而，稻哥的重要遗产：“修生养息”、“韬光养晦”、“和平与发展是时代主题”……也成为了老猪主政经济的核心律令。

以上是以老猪为代表的“制服组成员”从经济层面的阻力来源，也是形成购买瓦良格号困难的浅因。

除了浅因，还有一个深不可测的深因，它不来自于制服组，而是来自于“军装组”。

按道理，如果能拥有瓦良格号以及全部图纸，那么苏联用了二十六年（1962—1988）、历经了三个级别（“莫斯科”级、“基辅”级、“库兹涅佐夫”级）、建造了八艘航母才取得的所有经验和技术积累，就能一夜之间归我解放军所有，且这样的好事不过几千万美金便足以获得。

那么，为什么在批准阶段会如此困难重重呢?

这个，就要说到我军内部根深蒂固、甚至直到今天都依旧尾大不掉的“陆军与海军之争”，或又称“陆权与海权之争”，乃至直白一点：根除“大陆军主义”的改革斗争。

要看到，90年代后半叶购买瓦良格号这一“强大海军”的战略行为，从一开始就遭遇了军内高层的抵制。

而当时我军高层悉数为陆军出身，且非长者人马（长者直到世纪之交才得以插入郭徐二人）。

“大陆军主义”的最突出表现就是在行政等级上，过去二炮（今天的火箭军）、海军、空军三者的司令部均属于大军区级单位，而陆军的七个军区全部属于大军区级单位。

也就是说，整个海军或空军，在行政层级上其实仅仅和北京军区、南京军区、沈阳军区、兰州军区、成都军区、济南军区、广州军区这样的大军区相等，而海军所属的东海、北海、南海舰队等级要低于大军区。

所以说陆军这个兵种“凌驾于全军之上”绝不为过。

陆军在诸军兵种中地位显赫，以至于其领导职能直接由军委四总部代行，四总部的首长和七大军区司令也均由陆军将领出任，这种安排在世界上几乎是独一无二。

当然，陆军“享有”如此地位也不是天降福泽，这是陆军在漫长革命史上一枪一炮、以血肉之躯打下来的威严。

别的不说，建国后连续硬刚美苏两大帝国主义，没有陆军，谈何保家卫国?

蒙哥马利访华时就曾感叹：“战争有两条基本原则：一、不要进攻莫斯科；二、不要同中国的地面部队作战。”

而且，我军自建军以来仅有陆军部队，后来的空军、海军部队都是1949年以后才成军，其部队也多数是从解放军陆军部队中分离而来。

再考虑到建国后相当长一段时间内的现实防务需要，尤其是后期苏修在北方恐怖异常的压迫感，加上陆地邻国实在太多且多有冲突记忆……倚仗陆军、强大陆军、发展陆军成为我军建军核心，也确实是时代之需。

这就导致了不论是经费倾斜还是人事安排，亦或是话语权乃至军内与社会地位，陆军系统几乎一手遮天，而空军和海军往往是打辅助、唱配角戏的存在。

特别是海军，如果说空军还因为可以配合陆军地面作战而能够享受到一些政策和经费，那么海军实在是酸楚。

想想张召忠的眼泪，很多事情不难意会和理解。

1974年西沙海战，我海军和南越参战舰艇数量各为4艘，对面的4艘军舰是3艘驱逐舰和1艘护卫舰，吨位大的为1770吨，小的也有650吨，总吨位6000多吨；而我军首批参战的4艘舰艇最大的才570吨，比对方最小的还少80吨，而小的只有300吨，总吨位加起来也不过1760吨，还不足对方最大的一艘舰只的吨位。

1996年台海危机，局势紧张到极点，我军为打击台独而组织三军联演而战，但海军装备少得可怜，只得在商船甲板上部署陆军的自行火炮来弥补火力不足。面对对岸被美帝国主义武装起来的台伪军，我海军将士几乎是用手血肉之躯去顶部……

这就是我们海军昔日处境的缩影。

再想想2018年领袖海上大阅兵时，已“变身”为辽宁舰航母的瓦良格号率领一众舰船浩浩荡荡、铁流滚滚而来时，几代海军人纷纷发声“扬眉吐气”、“百感交集”、“老泪纵横”……

《解放军报》专职编辑张科进透露过一个事，2016年初他在编发某战区部队的来稿时，此单位的新闻干事专门打电话过来，强调要将他所写稿件中的“战区”去掉，代之以该单位所属的陆军军种……

要知道，那可是刚刚经历了领袖军改的阶段啊，“兵种本位主义”的积重之深可见一斑。

回望90年代，“大陆军主义”就是更坚如磐石。

要看到过我那篇《为什么我驻南联盟大使馆会被轰炸?》文章的同志应有印象，“造不如买，买不如租”思想的主要运用对象就是海军和空军。

尤其是海军的萎靡不振：
① 旅顺基地，陆上后勤供应的蔬菜，连根带泥，洗也不洗；
② 上海基地，东海舰队想在崇明岛建点，结果因为经费拮据被否决；
③ 舟山基地，有洞库泥沙回淤严重，不去修整，直接报废；潜艇支队官兵饮水问题无法解决；军民杂居，生活纠纷不断；
④ 广州基地，刘华清到上川岛时，按惯例列全体应列队欢迎司令员视察，结果非但没有，且是东一群、西一群、不立正、不敬礼，吊儿郎当如老乡看大戏。刘华清当时并没有发火，因为他知道这背后是干部风气问题，归根结底是大环境问题；
⑤ 下川岛，部队吃菜成了大问题，一艇一灶，每天早晨要乘船去岸上买食品。春节时全支队也吃不上鱼，肉也罕见。刘华清在南海军队会议上告诉后勤负责人：“你们的内心应当受到谴责”……

由于不给拨经费，榆林港的航道里，该炸的礁不炸、该建的码头不建，舰艇拥挤，这谈何机动作战?

2004年，“核潜艇之父”黄旭华接受《现代舰船》采访时，曾愤述：“20多年过去，美国走了多少型号，冷战结束，战争的危险并没有消失。美国包括日本的科研工作一点没放松，而我们的科研生产都断线了。团队解散，很多人才都流失了。如果我们也像美日那样，技术抓得很紧，我们的技术水平不会是现在这个样子，就算不超过美国，也至少不会和他差这么远。”

当年，军内曾流传戏语：“中国没有海军，只有穿着海军军装的陆军。”

这种局面下，指望军委的陆军头头们为海军出头、去请求拨款购买瓦良格号?痴人说梦……

瓦良格号项目从头到尾有一个最重要人士，就是贺老总之子贺鹏飞，他就是海军副司令。

其实按理讲，贺鹏飞在军中应该有一定地位和威望……

80年代后，我军内部一度有一个“杨廖萧贺”的说法，就是杨家将兄弟及其门丁势力、廖汉生家族（廖还是杨的妹夫）及其门丁势力、萧克家族及其门丁势力、贺老总家族及其门丁势力（其中萧克和贺老总还是连襟，他们俩的老婆是亲姐妹；同时贺老总还是廖的舅舅）。

但是读过我之前不厚的会员文章的同志都知道，90年代，二代们的声望与“晋升润滑度”，开始不如团π的做题家们……这其中，贺鹏飞也是典型之一，他的家族背景相当显赫，尤其在红二方面军出身的将领、干部、系统中还是有一定人脉能量的——只不过，涉及到更高一层的决策权，他red二代的身份加之海军将领的地位，在军中实在拿不出手。

这就是为什么在购买瓦良格号的过程中，贺鹏飞几乎动用了自己的全部力量与资源，仍然险些功亏一篑。

很长一段时间，“航空母舰”成了一个极其敏感的词汇，不再被提起。
