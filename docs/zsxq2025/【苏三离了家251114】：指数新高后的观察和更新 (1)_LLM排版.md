
# 指数新高后的观察和更新

指数新高后的观察和更新，2025年11月14日，苏三离了家。I1月合集第六篇。

创业板和科创板在1.5的低点处止跌获得支撑，在今天在方向选择中选择向上，技术性反弹。

根据这几天盘面的展现，简要陈述下自己的看法：

1. 今天指数(上证)的向上突破，趋势(新能源)和情绪(投机)共振，从头强到了尾，指数也收了光头阳。
2. 最强的趋势在以储能上下游供应链为核心的各种电新分支，狭义上的科技并不强，昨天抢尾的中际在创业板如此强势的情况下也未出水。
但有值得关注的点：今天弱势的AI都是近期调整幅度小，离新高近的，一些调整时间久，调整幅度大的硬件有比较明显的止跌(可能见底)迹象，比如今天的液冷。

为什么用思泉新材来做观察的锚点?因为它曾经在日K上是液冷分支里非常有辨识度的领头羊，它调整的时间/幅度可以用来作为客观判断硬件整体调整周期的尺子。

而在今天之前，在已经调整了40%的情况下，它仍未止跌，至少在我的框架里，如果同一个板块内，基本面没出现倾覆的情况下，跌幅最大的那些还没止跌，那对于同板块其他标的的持有型抄底都是无意义的。

底部通常都是一个区间，而非一个点，需要用连续的K线组合来判断，而非某一两天的反弹阳线，所以思泉I1.6-11.7的2连阳并未让11.5成为最低点，但是今天这根带有主动性(并非跟随AI核心反弹)的最高接近+15的实体阳线加进K线组合以后，大概率可以判断167附近差不多就是这一轮的底部了，后面可能又会再跌下来，甚至回到这个价格附近，只要不是直接一根巨阴反包今天阳线击穿新低，那这根阳线的15个点或再多一些就是留下来的震荡箱体空间。

这样的标的如果真实止跌见底是有指向意义的，我们也可以看到，在它止跌后，有一大批标的同样止跌，这种状态代表这些东西从基本面到技术面以及资金面有了三位一体的左侧交易价值(波段的起点区域)。

关于今天的反弹，昨天也能感受到一些端倪，昨天红利作为带有一定避险情绪的趋势集中地，昨天上午应该是被一些心态破防的科技里的止损资金买到高潮了，比如昨天盘中，你有没有梭哈中国移动的冲动…而红利显然是不能加速的，尤其沾染到情绪资金后，更是弊大于利，而绝望资金的离开，让科技短期已经无血可吸，看趋势做趋势的早走了，套牢的要割的也割了，杀跌动能衰减，技术性反弹一触即发。

硬件这里偏上的暂停，继续震荡等待，AI偏下的止跌兜住向下的势能，未尝不是某种意义上的高低切。

存储的高低切，开盘分时比较明显，存储的流动性切到了低位的兆易创新和佰维存储。这个内部路线的高切低也很合理，另外几个模组实在涨太多了。早上兆易分时走强对有准备的人来说是很标准的买点，可惜被不断走强的锂电/储能给吸炸了。

如果后续有考虑交易存储，可以重点看这两个。

真实共振指数的锂电/储能，创业板昨天探底回升后，有选择方向技术性反弹的需求，宁德200大单成了GWH最佳的导火索，宁王的巨阳线撑起了整个电新脊梁，上/中/下游共振，弹性和强度都拉满。

每一次指数的突破性阳线，最强势的永远是趋势之上的东西，所以于交易而言，如果想获得每个阶段的超额，就应该不懈的追逐阶段的主题与趋势，流动性与强度的交汇处。

这个月我自己做的也不算好，基本在趋势上踏空了电新尤其是锂电的行情，这个原本应该是在我能力圈内的。

储能的景气度衍生出的后续一系列供应链行情，其实肇始于9月，阳光电源/亿纬锂能/天际股份/多氟多I天赐材料，都是9.3号左右启动的，然后一路走到了今天，彼时我只是很主观的将它们归类到了固态的炒作里，并未深入了解它们上涨背后的基本面嬗变，而且因为存在一些偏见吧，我也很长时间没去看过跟踪过锂电的高频数据了。

话说，其实我原本打算今年或明年的某个时间开始左侧买锂矿的。归根结底，没做到主要还是自己菜吧，菜就算了还很傲慢。所以，大家还是要谦虚内敛，不能太膨胀。

投机情绪的高潮，高位/中位/低位齐齐爆发，队伍太整齐了。做纯粹投机的朋友，熬了一年终于吃了一顿饺子，善猎者善守，等待自己的舒适区再击球，多么重要。

这一波投机的高度很高，空间如此之大，后续低位会有此起彼伏的补涨/套利机会，慢慢来吧。

创新药大概率整体见底了，不管是调整的时间(7-I1月),调整的幅度(腰斩的不少),10.31的机构回补阳线算左底，在连续回踩后，这周走出了右底。

百济神州港美股都已经年内新高，算是先行的指标股，美股XBI也创了新高，算是中美共振。调整充分，关注度低，拥挤度极低，比起科技，我觉得创新药会更早更容易走出趋势，而且对量能和指数的要求也不高。

有兴趣的直接配置港股创新药 即可，ETF对普通人来说，创新药太难了。

最流畅的趋势是绝对高位的区域，拼的是交易的技巧。当然化工/钢铁内部有很多不高的东西，按绝对位置自己看图筛一下底部图形好的，或者不求快，直接ETF 也可以，两个今天都算突破了。但投机的部分却可以润泽到绝对低位的区域，只需要坚定在消费/医药内部投机即可。

周日列了一些消费的标的，整体还不错，今天再给一些供大家参考吧。

从消费各类别的走势的强度来看，服务类消费应该是最强的，主要就是旅游/零售。

从叠题材/玄学/区域/令名的角度考虑：祥源文旅(荒野求生)、新华都(福建+零售+阿里)、同庆楼(名字+餐饮)、家家悦(名字+零售)、喜临门(名字+家居)、报喜鸟(名字+服装+动物)、老百姓(药房+名字)、鹭燕医药(图形+医药+福建)。

然后明天周五，做投机的人会担心周末的监管不确定性，高标会大分歧，合富中国这种大概率会断板，而现在的中位股会有大面。这种情况下明天的低位首板就会非常好，当下的投机情绪下，一点就着，所以，低位萝卜蹲赔率很好。

美股今晚也在演绎预期兑现，政府刚开门，今天就大跌了。

我对美股科技有隐隐的担忧，甲骨文这一类高负债侧公司的走势/CRWV Al实在太惨烈了，而且根本见不到底的感觉。

尤其美股这段时间的走向，能看到很多资金的在从AI从科技流向其他传统行业，很多争论/质疑并不是空穴来风，它是会真切的影响到资金的信心的，特别是当所有东西都在高位的时候。

所以，如果你一定要做AI做科技且非套利交易，务必坚持买价格而非图形/脉冲，而当跌破价格时要及时止损，等下一个概率底价再介入。

英伟达将在Ⅱ月20号凌晨发布新一季的财报，理论上会有一根较大阳线的博弈价值，如同机器人的1 1.6,但是鉴于上一轮机器人博弈的难度 ，我觉得还是主动放弃的好。

如果非要参与，等一个大跌的日线后再说。

上证新高了，但我看不清后续能走到哪里，又或者我 内心里觉得它再往上冲一下可能今年就差不多 了；而两创只是反弹，能维持住强势的震荡就很好，没有大的动能再持续向上。

指数确实强势，但是缩量的环境，结构化 / 风格化也很明显，尽量 不要被来回割蹭。
