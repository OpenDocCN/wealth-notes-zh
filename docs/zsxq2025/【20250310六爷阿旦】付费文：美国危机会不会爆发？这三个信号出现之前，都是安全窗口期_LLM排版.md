
# 美国危机会不会爆发？这三个信号出现之前，都是安全窗口期

## 前言

当今世界最大的不确定性，其实就是来自于美国，而美国战略层面的宏观政策风险，其实是有迹可循的，尤其是美国当前的战略收缩，有明显的步骤和规划，而变数主要在执行层面的完成度上。所以要考察今年投资的主要宏观风险，我们可以在几个主要的节点上进行观察，再结合市场运行的位臵，就可以参考和判断。本文共分为三个部分：1、美元降息空间正在打开，2、财政紧缩与危机自循环，3、伊核协议与中美博弈。希望能对大家有所帮助。

很多人关注今年的投资机会怎么样？其实从宏观层面来讲，首先需要解决的问题，是是否有不可控的宏观风险，了解了风险之后，才能更放心地去寻找机会。而从宏观风险的角度来说，今年似乎有点柳暗花明的感觉，但是又有一点扑朔迷离。

柳暗花明的感觉，主要是特朗普上台之后带来的。因为他摒弃了拜登的金融战路线和联合盟友对我们进行围堵的战略。总体上美国所进行的操作，是在收缩和撤退，这相对于我们的上升期来说，提供了一个更从容的国际空间。

但是在另一个方面，也有很多人担心，美国在乌克兰战场撤退之后，有可能会集中更多的资源和精力来对付我们，而且美国与我们之间的矛盾冲突，从特朗普的这一套班子来看，似乎并没有改善的意愿，这就使我们对接下来宏观形势判断，仍然具有较大的不确定性。

你看美国的投资风向标是巴菲特，他依然保持了历史记录的现金储备，对美国市场的减持也一直在进行。而美国金融市场在不断创新高的背后，现实世界的经济形势却一直存在巨大的隐忧。所以美国金融市场的系统性风险，对很多人来说，犹如达摩克利斯之剑，是一个挥之不去的心理阴影。

从这几个角度来说，我觉得有一个综合的分析和判断，如果可以找到几个明确的时间和事件节点，在这些节点之前，总体上相对安全，而一旦这些节点出现，就要全面提高警惕。这样一来，就能给我们的投资，做一个宏观上的参考依据。

## 美元降息空间正在打开

首先我们从金融的角度来看美国的降息周期。美联储在去年进行了三次降息之后，到现在为止都是处于暂停降息的一个状态。而在美国暂停降息的这几个月内，欧洲以及五眼联盟国家，其实并没有停止降息。

在美国暂停降息之后，美国的盟友继续保持着降息的节奏，他们就跟美国拉开了彼此之间的货币利差水平。美国现在的利率在4.5%左右，而欧元利率已经降到了2.9%的水平，相对来说，欧元已经回到了3%的安全水平线内。

同时美元和欧元的名义利差也达到了1.6%，而同期美元和我们的利差水平也也从去年9月的1.7%，持续扩大到现在的2.5%。也就是说在美国降息周期内，美元对全球最主要的两大经济体，货币利差水平不降反升。而由于利差的扩大，遏制了资金从美国流向欧洲和我们的冲动。

所以至少到目前为止，美国的金融市场依然在较高的位置震荡，同时也吸纳了大量回流的美元流动性。那么从资金的角度来说，未来只有两个出路，要么就是美国给天量的美元在全球其他地方找到新的资产池子，要么就是在前者失败的情况下，消灭多余的货币。

所以在美元全面降息之前，美国的金融市场需要继续在高位震荡，留住资金，而美国的金融周期，暂时还没法进行大幅快速的降息。换句话说，寻找到新的资金池子之后，才能给美元的流出真正创造足够的价值空间。

从目前的情况来看，美国要寻找新的资金池子，做大自己的资产负债表，有两个方向已经是基本不太可能的。一个是俄罗斯所在的东欧方向，美国本身是在从这个地缘冲突带进行战略撤退。另一个是作为全球主要经济体的我们。

如果从市场化的角度来说，我们本身是最有潜力、最具有投资机会，也有足够价值空间的经济市场。但问题出在美国，现在要搞新冷战对抗，它不允许资本流入我们这边，来推动我们的产业发展。

这样一来，美国能够创造新的资产池子的地方，只能是在自己的盟友圈子里面。

一个是像加拿大这样，直接吞并之后，资产负债表中的资产就直接大幅增加了，这给美元不管是发行国债还是印刷货币，都增加了巨大的想象空间。

还有一个是欧洲。整个欧盟有巨大的存量资产，美国要做的是借助一次危机的机会，对欧洲的核心资产进行全面的抄底和控盘，这样不仅以后美国对欧洲资本的话语权会全面提升，而且在后全球化时代，美国只控制现在的美加墨地区，整个经济规模也是不足以与我们竞争的，美国还需要捆绑欧洲，形成欧美经济的内循环。

如果这两点中的任意一点，美国做到了，也就是为美国的资产负债表扩容做好了充分的准备，那么美国的降息空间，才算是真正的彻底打开，这个时候美国还需要做一件事情，就算是为全面降息铺平了道路。

这件事情就是让国际油价出现快速下跌。以现在美国四处出击，全面加征关税的操作来看，如果后面还要进行大幅快速的降息，美国国内的通胀水平一定会全面恶化。而影响美国通胀的核心因素里面，油价的即期权重占比非常高，所以美国如果既要进行降息，又要稳住通胀水平，那么打压油价，很可能就是一个必然的选择。

如果油价重回30美元以下的水平，那么美国国内的通胀压力会大幅减轻，这也将为美国开启全面的降息创造出空间。

## 财政紧缩与危机自循环

我们看美国现在正在做的事情，就是进行全面的开源节流。开源这一块大家都看到了，就是对全世界进行加征关税，这一点我们之前分析过，其实主要是美国税收口径的转移。因为全面加征关税的结果，必然是美国内部承担大部分的关税成本。

另一方面就是进行节流，这一块现在马斯克砍的比较狠。美国的节流，其实是迟早要进行的步骤，因为美国的国家战略，已经从全球化扩张转向了全面的收缩，过去美国的财政支出，通过各种各样的国际组织，在世界各国形成美国的影响力。但是在美国进行战略收缩的背景之下，美国已经不需要这些影响力，那么这些财政支出也就没必要了。

只要美国战略收缩这个背景和趋势是确定的，那么即便今天特朗普不做，以后别人也要做。这就像一个全球性的跨国集团，一旦决定收缩全球的业务线，那必定要砍掉非核心市场的营销推广支出。

现在美国就是要在几个战略缓冲带，也就是目前的主要地缘冲突区进行后撤。在这个过程中，一方面是将过去大量的财政支出砍掉，一方面还要将过去在全球化扩张时代，沉淀下来的存量资产变现，打包带走。

这两点来说，可能在今后很长的时间内，都将是一个无法改变的趋势。

事实上这也带来了一个比较大的风险。过去美国是全球最主要的财政支出方，美国通过发行国债，进行融资，然后在全世界撒钱，这样强化了美元的国际货币地位，但是也扩大了美国的债务规模。这可以说是美元问题的一体两面。

美国的财政支出，在这几十年里面是持续增长的。这是美元和美债在全世界进行金融活动，不断扩张的底层支撑。现在特朗普进行猛烈的财政支出收缩，很可能导致全球最大的资本方，进行剧烈的拆杠杆。而去杠杆操作往往风险很高，非常容易引发不可控的市场危机。

现在之所以还没有体现出这种拆杠杆导致的死亡螺旋循环，只是因为马斯克的削减财政支出，还没有全面落实到位。如果美国缩减财政支出的操作全面实施，预计在3~6个月之后，就会逐步的传导到依赖美国财政支出的各行各业。而在这些支出之上，很可能还有对外加杠杆的投资，到时候可能出现更大的崩塌效应。

在某种意义上来说，尤其是美国对国防开支上面的收缩，首当其冲可能会受到巨大影响的，就是欧洲。这也是特朗普一直在强调的一个重点，他认为美国为欧洲安全支出了太多。其实他的言下之意，美国的军费收缩和美国从欧洲抽身，其实是同一件事情。

这在基本支出和军事安全上面，对欧洲的影响是双重的。而这件事情最后对于欧洲的资产价格会带来什么样的影响，可能才是美国最关注的。欧洲国家如果将国防开支全面提高的占GDP5%的水平，又很可能会导致欧洲国家的财政状况进一步恶化，债务水平大幅上升。

在这样一个背景下，对美国后面所要进行的降息、危机、加抄底这个流程来说，可能也是必须要铺垫的一环。因为在危机这个环节，只有欧洲的债务危机全面爆发，才有可能给身处外部的美国，创造一个合理进入抄底的机会。

如果美国在对手一侧找不到足够的资产池，最终必然是在盟友内部做选择，从整个资产容量空间来说，欧洲无疑是最大的那一个，也是无可替代的那一个。

美国通过快速的支出收缩，可能会对国内的资产端有所修复，但是对于欧洲的债务端，则可能会严重恶化。所以对于欧洲来说，现在最大的威胁，其实并不是明面上的俄罗斯，而是背后虎视眈眈的美国。

美国在俄乌冲突上面，如果能够顺利实现媾和，那从战略层面来讲，就把降息抄底的目标锁定。后面只需要在中东地区依葫芦画瓢，整个中东到欧洲一线，美国完成战略收缩后的缓冲边界，就基本确定了。

## 伊核协议与中美博弈

目前在俄乌冲突这件事情上面，美国给乌克兰施加了巨大的压力，虽然停火方案和美乌矿产协议一波三折，但是从总的趋势和美国的既定目标来看，基本上并没有出现太大的悬念。就是总的发展方向，依然是在朝着美国预设的目标在走。

很显然特朗普对这件事情还是信心满满的，因为美国已经和俄罗斯在商谈另一件事情，就是让俄罗斯作为美伊谈判的中间人，来斡旋整个中东的停火事宜。其实这个事情本质上，就是美国在后全球化时代，在中东地区的势力范围划分问题。

我们过去讲，以色列是美国插在中东的一个钉子。那同时在以色列这个钉子旁边，国际社会安插了另一个钉子，就是巴勒斯坦。美国通过以色列来牵制整个阿拉伯世界，国际社会通过伊朗，再通过巴勒斯坦，主要是哈马斯组织，再来牵制以色列。

其实从这一轮巴以冲突爆发以来，以色列一直想做的一件事情，就是全面铲除哈马斯，然后吞并巴勒斯坦，进而消除身边的这个钉子。一旦巴勒斯坦在事实层面不复存在，那么所谓的巴勒斯坦建国，巴以1967年的边界线，这些也就全都失去了抓手，国际社会将找不到制衡以色列和美国的支点。

所以以色列和美国一直在筹划的一个方案，就是将加沙地区的100多万巴勒斯坦人，迁移到埃及、约旦和其他阿拉伯国家去。特朗普甚至扬言，加沙地区属于美国，并且对外宣称，想把加沙地区打造成一个海滨度假圣地。

但是目前来说，这个方案遭到了埃及和约旦的强烈反对，也遭到了其他阿拉伯国家的一致反对，同时还遭到了包括欧洲国家和我们在内的国际社会的共同反对。原因就是上面说到的，失去了加沙，就失去了制衡以色列的抓手，也就失去了在中东地区和美国博弈的支点。

以色列一旦吞并了加沙地区之后，后面必将是全面吞并整个巴勒斯坦，再后面就将是黎巴嫩、叙利亚，以及约旦和埃及。他们就像病毒一样，这种扩张的势头一旦开始，就不会停下来。

3月4日，阿拉伯国家紧急峰会通过了埃及提出的一项预计耗资530亿美元的加沙重建计划。根据峰会发表的声明，阿拉伯国家领导人还发出警告，任何试图将巴勒斯坦人赶出家园或吞并并占领巴勒斯坦领土的企图都将导致该地区陷入新的冲突阶段，并将冲突扩大到该地区的其他国家。

不出意外，以色列和美国，对该重建计划表示反对。而英国、法国、德国和意大利外交部长3月8日发表联合声明，支持重建加沙地带的"阿拉伯方案"。四国还明确表示支持巴勒斯坦民族权力机构在加沙地带重建中的核心作用。

所以就目前来看，美国和欧洲在中东的势力范围划分问题上，还没有达成一致。因为实际上现在美国和欧洲，在俄乌冲突如何终结这个层面，也还没有彻底解决。而先解决东欧的势力范围划分问题之后，才有可能轮到中东。

所以即便俄罗斯表态愿意作为中间人去斡旋伊朗，但是中东的情况明显要比俄乌冲突要复杂得多，俄罗斯自身从叙利亚进行了战略撤退，其在中东的话语权已经大幅削弱，这个中间人的含金量本来就不够，就中美俄欧四方来说，未来中东的势力划分，博弈将更加复杂和艰难。

中东的势力划分，谈判其实就是围绕两点，一个是伊朗，一个是巴勒斯坦。伊朗的问题就是看能不能达成新的伊核协议，巴勒斯坦问题就看能否达成全面可持续的停火协议。在这两个方面，我们目前还没有表态，在我们没有正式表态之前，不管美国和俄罗斯谈的怎么样，这两个问题都不可能有结果。

其实伊朗已经明确拒绝了美国的新伊核协议谈判要求，这对于俄罗斯的表态，某种程度上是一种否认。当然后面伊朗的态度也有可能会发生转变，但至少在目前来说，时机是不成熟的。

而这种时机的不成熟，对于我们从中观层面判断风险来说，就是一个最好的观察窗口。因为一旦中东的视力范围划分谈判完成，那最终最艰难的中美博弈与东亚势力范围的划分，必然走上前台。所以从这个角度来说，在中东还没有谈成之前，宏观风险释放的条件是不具备的。

## 最后

我在前面的文章中，其实有谈到过中美势力划分的问题。就是在于美国对于欧洲资产的觊觎和中欧投资协定未来重启的变数之间。我们有庞大的外汇储备，美国绝不会希望在制造危机后，可以抄底欧洲资产的时候，再出来一个截胡的对手。

要解决这个问题，美国要么对我们实施全面的金融制裁，并引发全面的金融脱钩和双边对抗。要么就得跟我们在势力划分的问题上达成一致。就这一点而言，目前处于薛定谔的猫状态，双边都还有很大的变数。

而就投资来说，在这些重大的不确定性风险之前，我们能观察到的事件风向标和时间窗口，大概也就是前面说到的这几个。希望这些梳理，对大家2025年的投资有所帮助。
