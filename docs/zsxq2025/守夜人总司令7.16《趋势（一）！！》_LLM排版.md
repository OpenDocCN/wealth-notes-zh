
# 趋势（一）！！
# 250716 守夜人总司令
设想一下，未来 10～ 20 年，中国整个社会的中等群体数量会越来越多，还是越来越少呢？答案是越来越多！产业的发展和社会的进步，一定会提供更多更高收入的岗位，当然了，这些岗位不会平均分配，有一些人会获得机会，而更多的人会被排除在外。这与特权没有关系，产业岗位不会平均分配是社会的自然选择。

这就好比 1978 年的时候，有些草根爬上来了，而有些原本还在国有企业里有一份稳定收入的职工，却因为国企职工自身的选择而错失了时代机遇，逐渐的沉沦下去。当你去与中年人交流的时候，看到的都是焦虑和哀叹，当你与年轻人交流的时候，往往会产生两极分化的反馈。一群人会感到绝望，另一群人已经春江水暖鸭先知了！

新一代中的优秀分子们的行动力比上一代人的优秀分子更强，思维更开阔，心态更积极，行为更敏捷，当然，道德也更低下，这个道德不是私德，而是指公德。所谓公德，就是在不在乎大多数人，会不会受大多数人道德观念和舆论氛围影响，从这个角度来说，年轻一代中的优秀分子，真的一点都不在乎！

我们 8090 后这一代人，即便再激进，我们 8090 后也会本能的认为自己与大多数人是一个整体，即便有点本事也没有那么自恋，但是新一代人中的优秀分子真的非常的自信，也非常的自恋。新一代优秀分子不认为自己与同龄的大多数人是一个整体，更多的优秀分子会本能的把那些人当做工具，也就是说，新一代优秀分子更加聪明，也更加的伪善！
