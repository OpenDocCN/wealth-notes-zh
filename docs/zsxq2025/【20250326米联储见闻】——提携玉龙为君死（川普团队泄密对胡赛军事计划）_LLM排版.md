# 你好，我是米糕

川普团队最近爆出机密军事会议摆乌龙事件，再次证明世界真是一个巨大的草台班子，有时候甚至还不如高中生对待月考的态度认真。战争是国家最高级别的行为，是最极端的国家暴力，所以在任何国家，对战争都极其严肃，要进行周密谋划，要严格保密，哪些人能参与谋划与讨论，哪些人不能，应该遵循什么流程和规则，都有严格要求，尤其是对一个掌握了全球最多核武器的国家来说，对战争的态度如此儿戏，真是令人细思恐极。以美国佬做事的风格，甚至让人不得不怀疑当年南斯拉夫大使馆被炸就真的可能是误炸。

美媒《大西洋月刊》总编辑Jeffery Goldberg被意外拉进了川普团队讨论对胡塞武装发起军事打击的讨论群，全程围观了白宫高层讨论战争决策的过程，并一直没有被发现，直到他把整个过程写成新闻稿件发在大西洋杂志上，对于一个媒体人而言，这是连做梦都不敢想的天降馅饼。要知道为了得到一些独家新闻，挖到一些政府决策的内幕信息，要付出非常大的代价，长期培养线人，有时候哪怕是只字片语都会让媒体人欣喜若狂，写一大篇文章捕风捉影，像这种被直接拉进群聊，核心敏感信息扑面而来，层出不穷的事情，真是百年难遇，估计对大西洋的主编来说，这简直像是走梦一样。以至于他的报道过程都没有对群聊内容详细解读，而是不断震惊于这个巨大的乌龙为啥会发生，确实，相比于高层讨论如何打击胡塞武装，如此重要的军事会议用这么儿戏的社交软件进行讨论，甚至混入了局外媒体人而不自知，显然是更大的新闻。

此事发生后，民主党开始猛追猛打，以证明川普团队在讨论战争行动时多么随意、多缺乏对战争的敬畏和系统性考虑。不仅毫无保密意识，在民用媒体平台组群讨论，还意外把记者拉进讨论群，而造成这一系列结果的直接原因，就是现在的国防部长赫格塞思在几个月前还是福克斯电视节目主持人，他的通讯录上媒体人的联系名单远远比军方人士要多，才会导致媒体人被拉进群聊来讨论美国大兵在前线开展军事行动。

川普开始并不觉得这件事有多严重，多次强调并没有泄密，行动也很成功，但是随着事件不断发酵和深度讨论，川普也开始意识到事件的严重性，会让各方质疑川普团队在军事指挥上的专业能力。当时在群里汇集了川普团队的核心人物，国防部长赫格塞思、副总统万斯、国务卿卢比奥、国家情报总监加伯德和白宫副幕僚长米勒等。会议内容如此敏感，参与人士如此高端，以至于大西洋月刊这个媒体人一度以为这是一个诈骗群，因为现在网上有很多电诈信息和假新闻都是这样操作。如果有人发短信通知你中了500万，一般稍有社会阅历的人都会怀疑是骗局，这个媒体人经历的这件事其实就相当于新闻人中了头彩，得到一条震惊全世界的独家头条。因为他是在无法相信美国国家安全领导层会透过Signal这个的民用软件来通报即将实施的战争计划。而且把媒体人拉进了讨论群。但随着美国官方宣布行动消息，他意识到该群组是真的。以至于全世界同时见证了世界真的是一个巨大的草台班子。世界头号军事强国的内阁成员是如何随意、糟糕、儿戏般决策一场严肃的战争。而且在讨论中，川普内阁的高级成员还把欧洲羞辱了一番，把欧洲当成了无能的战争提款机，商量着如何把战争成本转移到欧洲头上，从欧洲获得经济收益。真正是把桌底下最脏的部分拿到了台面上了，上次在美乌会面时惹祸的万斯甚至把欧洲比喻成需要美国擦屁股的无能之辈。国防部长在对胡塞武装发出攻击之前在群里发布了包含了即将对也门发动空袭的作战细节，包括攻击目标、所用武器和打击顺序，随后战争打响，在打击结束后，群里一片欢腾情绪，群成员之间互拍马屁。大西洋月刊全程记录了这些人彩虹屁，大西洋主编离开讨论群后，给群聊对象都发了邮件，随后事件曝光，美国国家安全委员会发言人证实这个Signal群聊是真实的。

事件曝光后，媒体查出来，媒体人之所以被拉进群聊，是被国家安全事务助理 Michael Waltz拉进去的，更诡异的是，此人与大西洋月刊的主编并非好友，是通过好友申请加了他之后，把他拉进一个名为胡塞小组的群聊。所谓即美国国家安全委员会的「主管官员委员会」（Principals Committee），是指NSC决策机制的一环，由总统国家安全事务助理和防长共同主持，其他参会人员包括国務卿、防长、中情局局长、美军参联会主席、白宫办公厅主任等高级官员，为美国总统提供多方位的建议。

白宫方面否认美国国家安全因此受到威胁，反而强调被披露的群聊信息展示了高级官员之间深入而周到的政策协调，“美国对胡塞行动的持续成功表明，我们的军人或国家安全没有受到威胁”。川普事后表示，他对美军打击胡塞武装情报泄露的事情一无所知，也对《大西洋》月刊并不感冒，《大西洋》月刊则援引律师分析说，白宫相关人员通过协调与国家安全相关的行动，可能已经违反了美国《间谍法》的多项规定。按道理，如果官员们要讨论军事行动，应当使用大多数内阁级国安官员家中专门安装的敏感信息隔离设施(SCIF)，或者仅通过批准的政府设备进行沟通。此外团队给部分群组消息设置了定时销毁，也可能违反联邦记录法。根据适用于白宫和联邦机构的记录法，所有政府员工都被禁止使用像Signal这样的电子消息应用程序进行官方事务，除非这些消息及时转发或复制到官方政府账号上。

事件曝光后，美国国会民主党人很快对川普政府发起抨击，英媒形容此事震动华盛顿。这是川普团队上任以来给民主党抓到的最硬的把柄，因为涉及到国家安全，而且被全程记录，民主党一定会穷追猛打，势必让川普付出代价。美国参议院民主党领袖舒默表示，将要求对此事发起调查。值得一提的是，美国国防部近段时间确实深陷泄密疑云，马斯克上周访问五角大楼前夕，被曝他访问期间将收听有关美国对华潜在作战计划，也引发轩然大波。

川普团队在群聊中恶心欧洲的对话被曝光后，也因为了已经对川普处理俄乌冲突方式不满的欧洲盟友们的进一步矛盾升级。表示现在普京都要失业了，间谍活动都毫无意义，美国人自己来泄密了。建立和加强美欧联盟是几代美欧人努力的结果，但川普政府在短短几周就削弱了它。从很多方面来看，欧盟是川普及其同僚所倡导原则的对立 面。自从万斯上个月在慕尼黑安全会议上发表讲话，质疑欧洲的价值观念及其民主制度，令欧洲领导人感到震惊以来，欧洲一直处于警惕状态。万斯后来还警告说，欧洲有“文明自杀”的风险。这次在群聊中，万斯更是毫不掩饰自己对欧洲的厌恶。从万斯在慕尼黑攻击欧洲民主的讲话、更不用说最近公开的这些交流中可以看出，他对欧洲的厌恶超越了交易性质。甚至否定了美欧有相同的价值观，这可是美欧联盟的基石。川普上台以后，一直对欧洲使用尖锐的言论和行动。

美国的态度倒逼欧洲领导人试图维持友谊关系的同时，争先恐后地增加国防开支，害怕在一夜之间失去美国的保护，沦为俄罗斯的炮灰，并努力与美国达成贸易协议，欧盟贸易专员周二已前往华盛顿与美国同行进行会谈。微信pep458与美国分手是一个价格高昂的前景。欧盟已公布了一项可能值8000亿欧元(约合8650亿美元)的计划，以帮助欧洲国家实现理想的军事开支水平。如此高昂的代价似乎也证明了万斯说欧洲是寄生虫也不为过。此次美国在安全军事事务上如此不专业的表现，更让欧洲盟友们对自身的安全感到恐慌，怀疑是否还能将欧洲的安全交给如此业余的团队。

民主党现在不但要调查泄密事件，还要求涉事高官辞职，其中防长赫格塞思与国安顾问沃尔兹应带头递交辞呈。川普团队则极力消解此事的负面影响，力图大事化小，小事化了，这件事其实也很考验美国社会目前对川普权力的制衡程度，如果这次事件最终不了了之，川普团队何止是权力滔天，简直是无法无天都无人可以制衡，这意味着美国内部权力失衡将进一步加剧，也势必引起更大程度的分裂和混乱。

世界原来真的这么儿戏，反倒是我们普通人太较真了。
