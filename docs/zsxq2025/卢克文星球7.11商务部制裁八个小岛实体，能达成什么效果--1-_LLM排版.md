
# 卢克文星球7.11商务部制裁八个小岛实体，能达成什么效果

## 文/卢克文工作室嘉宾特拉维斯
## (小岛=台湾，兔子=中国)

2025年7月9日，商务部公告，将汉翔航空工业股份有限公司等8家小岛地区实体列入出口管制管控名单，禁止向上述8家小岛地区实体出口两用物项，正在开展的相关出口活动应当立即停止。

那商务部为什么要把这8家小岛实体列入清单?这背后有什么战略考量?

我们来看看这8家实体都是什么来头，根据商务部名单这8家实体分别是汉翔航空工业股份有限公司、经纬航太科技公司、小岛中山科学研究院、仲硕科技股份有限公司、国际造船股份有限公司、中信造船股份有限公司、龙德造船工业股份有限公司、攻卫股份有限公司。

这里，首当其中的就是小岛中科院。小岛中科院是小岛军事装备领域集研发、试验、生产于一体的核心单位。相当于美国的“国防高级研究计划局”加洛马与雷神的复合体，具有极高的战略价值。我们熟知的雄风系列反舰导弹、天弓系列地对空导弹、云豹装甲车，还有各型舰载雷达、指挥系统、空空导弹都由小岛中科院研发生产。研究院董事长由小岛所谓的台“国防部长”兼任，院长均由台军现役军官担任。

其次是汉翔航空，原是小岛中科院下属航空工业发展中心，2014年又改制为民营公司，董事长和总经理均由台空军中将以上退役人员担任。上世纪90年代自主研发的经国号战斗机时期明星产品。进入新世纪后则主要以教练机、运输机及为主同时也承担F16、经国号战机的维护升级任务。

这两家主体可以说是未来台海军事冲突中，台军最具威胁武器的研发和生产单位。雄风2导弹可直接威胁我国沿海城市，而F16和F-CK-1经国号战机则是我们夺取制空权的阻碍。

导弹、雷达、航空零部件等对稀土依赖极大，且短期内无法替代，特别是钕铁硼磁材等材料的部件。此次被限制无疑将对这两家实体造成战略级打击。
