
# 重要信号：文科专业正在大量停招

今天我们来看一个重要信号和它体现出来的趋势。这个信号是：文科专业，开始大量停招了。众多高校开始裁撤文科相关的硕士专业。比如中国科学技术大学宣布，2025年起不再招收法律专业硕士研究生，注意，这里说的是法律，而不是法学；大连理工停招中国语言文学、科学管理学硕士；此外，北京外国语停招外交学国际经济方向的硕士；天津大学停招全日制会计学硕士；西南大学停招民族教育学硕士；华中师范停招管理学硕士；浙江大学停招非全日制物流管理硕士等等。目前粗略统计，已有超过30所高校停招200多个文科专业的硕士研究生，主要分布在管理学、教育学、法学、文学、新闻传媒、会计学、经济学等细分学科上。

要知道，研究生扩招是近几年的大趋势啊。在如此背景下，各大高校反而纷纷缓招、缩招甚至停招文科研究生，实在反常。

## 文科研究生“爆发式”增长

要分析这个异常现象背后的原因，还得从十几年前说起。此前十年，文科研究生的规模是“爆炸式”增长。从2013起的十年间，文科硕士在校生数量从57.3万翻倍到了116万，十年内总计向社会输送了超过800万的文科硕士毕业生。其中人数最多的前三位，是管理学硕士、教育学硕士、法学硕士，平均每年都分别向社会输送约33万、16万、14万的人才。文科硕士之所以会在这十年爆发式增长，有三个非常重要的原因：

一是学历内卷，刺激众多家庭开始进行高额的教育投资，从大学到硕士，再到博士。

二是社会需求充足。我们过去搞的生产一直 是“美欧提供技术，中国提供制造”的模式，技术往往来源于进口或者转让，所以中国企业对于研发、技术等领域的的人才需求较少，反而是对大规模协同下的管理类人才需求上涨，文科因此吃香。

三是投入产出比差异大。培养文科硕士的方式，多为传统课堂教学，靠的是文献、学术交流等。而培养理科硕士通常需要实验室、设备、技术支持等，很多项目还需要投入大量时间成本。相比于理工科的高难度毕业，文科专业硕士更加容易毕业。

基于这三個原因，尤其是第二个，转换一下视角，其实就是目前文科学生的现实难题和理科学生的优势。

## 文科生的现实难题

具体是什么难题呢？随着文科硕士数量快速增长，就业竞争压力逐渐成为一个现实难题。以中部某所高校，不点名了，2022年的硕士研究生就业数据为例。这所学校计算机、通信、电子、材料、生命科学等学院的硕士就业落实率均达到100%。而商学院、经管学院、文学院的就业率则只有60%-70%左右，旅游学院甚至只有40%。

另外，智联招聘的一份数据显示，2022年理科、工科签约率分别为29.5%和17.3%，而文科毕业生的就业签约率仅12.4%，不到理科生的一半，平均工资水平也比理科生低1000元到2000元。

所以有网友试图自嘲：

虚假的坑，是生化环材；真实的坑，是人文社科。

以上还只是硕士领域。大学本科文科专业的裁撤，这几年也一直在加速。

## 我们来看一组数据：

2022年，教育部撤销804个本科专业，撤销数量最多的前三位为信息管理、公共管理、服装设计；2023年，教育部撤销925个本科专业，前三名为信息管理、公共管理、产品设计；2024年，教育部撤销1670个本科专业，前三名为信息管理、公共管理、市场营销。

你发现了吗？这组数据体现了两点：第一是撤销的本科专业越来越多，第二是大部分都是文科专业。

此外，江西、福建、山西、辽宁、河北、河南等多个地方也推出相关政策，对各个大学中就业率低于60%的专业进行限招、停招或者干脆撤销。而在多省高校“红灯预警名单”中频繁亮相的，便是一众的人文社科专业。

## 文科缩招，理科扩招

值得关注的是，一边“裁”是趋势，另一边“增”也是趋势。近三年，教育部新增设将近3000个全新的本科专业，主要包括数字经济、人工智能、半导体相关、国防军工、通信电子、计算机网络等基础学科和交叉学科。目的是规模化、大力 度、高水平地培养一批贴合国家战略需求、符合社会企业需要的高技术型的理工农医人才，为国家突破技术“卡脖子”作贡献。

讲到这里，我们可以对未来的趋势作出几个基本的判断：

第一，在中美博弈的大背景下，教育资源会重点向理工科倾斜，“文科缩招，理科扩招”会是大势所趋。这个判断依据，除了我们上面提到的大学科目的新增和裁撤，还有两方面的原因：一是这近几年高考一直在改革，提高数理化的比重。而且一些数学好的学生通过专项计划，能直接上名校，偏科根本不是问题。另一方面是高校改革，国家近三年来的教育经费，大幅向类似哈工大这样的理工科名校倾斜，而政法类、财经类高校的经费则很难增长，甚至是在缩减。

第二，短期内，就业市场对高学历文科人才的需求比较饱和，未来或许会有更多的大学本科文科生加入到考公考编的队伍中来，竞争加剧，压力会继续传导。

第三，综合以上情况，我做出一个预判，当下是新一轮教育投资周期的起点。类似于二十年前学金融、土木、外贸等专业，当下投资子女进行前沿急需的理工科深造，能在未来相当长一段时间内获得红利。我们可以重点关注这几个理科专业——材料学、人工智能、半导体/芯片、装备制造、视觉传感等。

## 文科生的出路在哪里？

以上的趋势说起，都是对理科的好消息，当然，文科生听到这期参考，也不必过于焦虑。客观说，文科生遇冷，是时代周期的必然，但这并不代表，文科生就一定没有出路了：

就业竞争大，大多情况下不在于是文科科的问题，而是在于自身技能与岗位的契合程度。文科生有丰富的专业技能也能很好就业，只是总体而言比理科卷一些。如果你是文科生，在当下就业内卷、教育内卷的情况下，最重要的可能是锻炼两大能力——解决问题的能力和处理人际关系的能力。

因为文科是泛就业的群体，岗位首先要求你上手快，能解决问题，所以越是文科生，越要把实习看得更重要一些。

另外，因为是非严苛领域下的泛就业，必然会接触大量的人、事、物，如何正确地处理和解决这些问题，也很重要。这也是文科生在就业中的独特优势。

同时，我也建议文科生学习一定的理工科知识和财经知识，不要被基本的大门给拦住了。就好比互联网的产品经理，你一定不会写代码，但最好能看得懂代码。在这个时代，文科也许很难成为标准或者实际意义上的“专才”，那不妨让自己变成真正的“通才”。
