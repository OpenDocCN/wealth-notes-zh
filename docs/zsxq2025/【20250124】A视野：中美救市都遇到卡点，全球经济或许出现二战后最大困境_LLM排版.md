
# A视野：中美救市都遇到卡点，全球经济或许出现二战后最大困境

原创MEA森公众号：A视野2025年1月23日

自从924之后，村里说了很多要大规模刺激内需的举措，然而，至今真正落地的举措，并不多。面对全球“独一份”的偏通缩经济环境，村里为何至今没有大规模动作？真的不管经济了，还是到底哪儿遇到问题了？

军武体系和先进制造业不断曝出好消息，可是，房股市场和就业市场却一言难尽。到底怎么回事？是有什么我们不知道的卡点，还是一盘大棋？

可以这么说，上述问题早已困扰着整个社会的方方面面，无论个体身价几何。现在，川普2.0强势回归，明显是要办大事了。那么，后面我们承诺的大规模救市举措到底还落地吗？

本文，是近期森参加的一次金融高端峰会的应邀发言，属于内部闭门会的内容。在得到主办方允许的前提下，将里面的干货跟我们的粉做一个交流分享。法不轻传、道不贱卖，大家且看且珍惜。

前任美联储大神级主席，格林斯潘，曾经出过一本自传。在其中，他骄傲的宣布，美国经济之所以长虹，因为“弹性”。对此，他不怕浪费笔墨，浓墨重彩的做了大量的描绘。

那么，到底什么是美国经济的“弹性？从格林斯潘的叙事角度，就是无论美国经济遇到什么样的困境，最终都可以优先爬起来，并带领全球继续繁荣下去。如果真的是这样，那么，美国经济的密钥到底是什么？为何哪怕美国的盟友们都做不到呢？

二战后，美国主导的全球市场经济体系部分，实则是一个庞大的金字塔体系，以分工为抓手，以美元霸权为底座，逐步构筑完成的。

在这个体系内，美国初期的想法是，确保工业能力只在自己的盟友体系中保有，自己则负责先进制造业、军工、高阶科技和金融部分即可。

因此，大家可以看到，G7国家成立的时候，他们的GDP占比当时全球的始终比重在4%左右。本质上，这就是一种制度性的设计。

不过，随着一次次的技术变革，全球产业链链条越来越长，很多细分工业产业连美国的盟友们都不想做了。最好的例子，就是70年代日本将大量的落后产业淘汰给亚洲四小龙。

然而，这里面是有一个极为隐秘的全球霸权策略。尽管美国是以自己为中心逐步向全球分派更多利润率较低的细分产业，可是，它自己的创新也从来不是一帆风顺的。

美国的盟友们是追赶型的经济体，而美国自己是创新驱动的经济体。可想而知，追赶型经济体的劳动生产率上升速度大多数时候是快于美国的劳动生产率提高速度的。70年代的布雷顿森林体系爆雷，就是这种现象的反应。结果是，美国不得不重塑美元霸权体系，还要想办法打压自己的盟友。换句话说，这种情况是长期存在的，是属于这个全球分工体系的内生情况。

对此，实际上，美国人也是清楚的。

所以，美国精英们的策略是聪明的：养蛊

这就像阳澄湖大闸蟹相互比赛谁可以爬到湖中竖着的杆子更高，可是，真的爬的最高的那些大闸蟹往往也是被人类最早抓走当下酒菜的。因为，爬的这么高的大闸蟹说明是已经健壮且体力好，正好用来给人类补充营养。

其实，美国所主导的全球价值链分工体系，也是一个玩法。在一次次的PK中，日本证明了自己成为先进制造业的重要选手，很多领域可以打败天下无敌手。那么，80年代，日美产业链战争中，美国一口将日本吞掉，尤其是充分利用将二战以来日本全民积累的财富和股权红利吃掉，成了助燃美国在90年代成为互联网革命超级浪潮的关键要素。

客观而言，这就是格林斯潘口中的“弹性的真相！

现在的美国精英们，满口表达自己的后悔，当初不应该让中国加入WTO有了养虎为患的“痛彻心扉”。先不说整个过去20多年中美贸易的关系里面美国拿走利润大头的现实，美国精英们真正的失算到底是什么？

显而易见，2018年川普1.0是想要一口吃掉我们入世后大部分红利的努力，可是，失败了。

回顾历史，美国前后成功收割了英法德日韩，其它国家更是不遑多让。可是，唯独在中国这里，首次碰壁了。中国不仅拒绝美国的养蛊阳谋，而且，为了自保，敢于长期向美国亮剑。这个事情，实际上，对于全球经济和金融市场具有颠覆性的冲击。

入世后，美国可是带领所有盟友一起怒砸中国的生意，除了当时赚取每年两位数的利润，还想着日后可以再来个终极一割。问题是，这个阳谋完全落空，也就是格林斯潘口中的“弹性没有了。必须看到，这是二战后，第一次！

中国除了反抗，而完全独立自主的努力发展，甚至还获得了远比过去更加庞大的经济势力版图。本质上，这等于美国原本自己的很多地缘好处，还被中国拿走了，相当于第二笔“损失”。

因此，美国经济的弹性大幅减少，不敢经济着陆，无法经济着陆，只能不断拉行情，成了当前美国经济和金融市场的长期现象。

此外，这里还有一个怪圈。具体而言，美国越是打压，中国越是激烈反抗，美国的弹性越是差，国内亏空和高成本（高通胀）越是高企。

这里需要提醒，不要将高通胀与美股拉升、大国博弈等很多问题切割来看，它们实则是一个庞大的经济循环系统的有机组成部分。

由于始终无法拿走中国入世后的主要存量财富和重要资产的股权，美国只能保持对内的宏观杠杆率宽松，来维持整个国民经济的阶段性维稳。

但是，这种模式的后遗症很大。

首当其冲，就是极高的通胀和极高的债务飙升速度。尤其是当这两者一起来，美国经济的大量资源都只能用于各种内部安抚，根本没有多少资源可以用于下一代技术革命的研发投入。

明眼人马上就看得出，这就是经济学意义上的，资源错配。

现在，美国人已经不敢不加杠杆，没有债务驱动，就是死路一条，强心针每天一针不能少。问题是，长期严重的资源错配，实则就是为未来严重的金融危机埋下祸根。这个逻辑一旦看懂，也就明白，无论是川普还是拜登，都不过是裱糊匠，仅此而已。

但是，如果进一步分析，事态更加错综复杂。自从美联储加息以来，美元一直没能走出强美元行情，这个现象其实非常值得警惕。

正常情况，美元大循环是，一波弱美元行情促使美元流向全球来拉升全球经济增长，再一波强美元行情实现利润收割。这是货币端看到的“弹性”的表现。

然而，现在美国只能强美元行情，不敢轻易弱美元。究其原因是，如果马上实施大规模的弱美元，以今天中国在全球供应链体系的话语权，大量好处就全去了中国那里。可是，中国却未必拿这些美元去买美债来确保美元体系的闭环。相反，中国还会拿美元去买战略物资、贵金属、替代别国手里的美元债务等一系列操作。即大国博弈已经让美元循环体系无法做到闭环。

9月份美联储首次降息的启动，是以双方承诺配合让美元循环体系闭环运行为条件的。可是，如今川普回归，双方的信任又是大幅下降，美元无法马上进入闭环模式运行，又是突兀的问题。

这也就意味着，不是美联储不愿意降息，不是美联储很害怕关税战，而是如果美元循环的闭环问题没有跟中国达成共识，那近乎是驱使美国本土的流动性加速流向中国、建设中国、促发中国产业升级。

试想，满脑子都是修昔底德陷阱”的美国精英们，如何肯接手呢？

另一边，随着美债规模越来越庞大，一个新的挑战也诞生了。我们已经看到，10年期美债收益率持续飙升，这个事情说明，海外购买美债的动能不足。

你可以解释为大家没有这么多美元在手，也可以解释为没有这么多意愿。大佬文集圈不过，不管是主观意愿还是客观能力，现在的美债市场就是如此：如果美联储不启用QE，那么，就没有庞大的增量资金入场美债市场。中国还在渐进式的抛售美债，不断增持贵金属和核心资源持仓。

硅谷银行的前车之鉴就在那里摆着，这个事情才是今天美国联邦政府和美联储最为头痛的。

于是，一个困局就产生了。在没有美联储启动QE的前提下，其它国家也没有增持美债的动能，强美元就成了稳定美债市场的一个必然对冲举措。毕竟，强美元可以虹吸不少海外资金，多少还能够为美债融资提供支持的。美国不敢启动传统意义上的美元周期，即，不能启动弱美元，否则大量储蓄就会从美国本土流出，那美债市场更加棘手。

可是，没有弱美元驱使美元流向全球，美企利润怎么办，美国如何保证美债闭环，美国如何收割其它国家，美元加速与全球贸易脱钩，类似的问题就可以让美国国运面临严峻的挑战。

在这个剧本下，美联储如果继续熬着不实施QE，那么，“强美元+强美债”就成了一个必然的结果。可是，这个结果的代价是，美国需要长期向所有的利益相关方提供极高的融资成本，其它国家也可以利用这一点不断套利美元体系的红利。必须看到，布雷顿森林体系就是类似的逻辑下，直接玩崩的。

当年，法国总统戴高乐看出，“美元的超发+布雷顿森林体系=贵金属低估”。所以，他不断出口赚美元，又用美元去美国那里换黄金，最后把美国硬生生薅羊毛薅秃了。

那美国的上述困境，对于其它国家而言，无异于锥心之痛。

事实上，目前摆在全球其它国家面前的处境，分三类情况讨论。

第一类，美国的盟友

利用2023年4月重新启动的C6金融联盟机制，欧日韩等都扮演起美国金融市场的“奶妈”。他们积极配合强美元行情，默认本国储蓄和资源大规模流入美国金融市场，去为美国金融市场维稳提供支撑。这里面，日本是最好的代表。通过牺牲通胀和汇率，为美股科技股提供流动性支撑。这也是为何明明日本早就该加息抗通胀、稳汇率，日本央行妈就是不敢。毕竟，美联储可是监管着日本央行妈的一举一动。

当然，美国的盟友是不怕金融危机的，尽管大量资金外流。因为，这个C6机制的本意是，任何一个国家有流动性危机，其它国家可以无条件、甚至无抵押的紧急注入流动性，确保美国盟友之间形成攻守同盟，不会发生系统性金融危机。

第二类，就是广大的其它国家（不包括中国）

这些国家在面对强美元+强美债时，根本没有招架之力，任人宰割，是家常便饭。

由于没有类似C6金融联盟机制支撑，他们就是不断的经济下行、储蓄外流、资产承压，一副随波逐流的既视感。不过，由于中国给了不少这类国家的经贸和信贷合作支持，所以，本轮加息周期中，几乎没有一个国家被美国收割成功，这也是奇迹了。

第三类，中国

由于激烈的大国博弈，中国要求绝对的战略自主。所以，不仅不可能像美国盟友那样放任资金外流，甚至还希望其它国家当自己的“奶妈”。

我们看到，在资本项大规模外流的前提下，我们的贸易项却是直接行情拉满。2024年，我们的贸易顺差1万亿美元，约等于全球贸易顺差的1/3，前无古人。本质上，我们是用出口导向来实现向全球融资。

这里就遇到关键性的卡点了！如果美国无法利用传统养盅的招数收割我们来自救，那么，他们势必就只能通过金融把戏虹吸我们的储蓄，以便暂时稳定他们的金融市场。一旦如此，我们就面临可能被收割的命运也不敢随意大规模刺激救市。

原因也很简单，2023年~2024年，我们的资本项流出与贸易项流入规模，竟然大致相抵。换句话说，我们辛辛苦苦出口一年赚的钱，大部分又被美国人虹吸回去了。

于是，中国这里的救市运动就只能很谨慎，确保中美利差不能太大，否则就会出现资本项流出规模大于贸易项流入规模，那就麻烦大了。

不仅是保汇率难度加大，我们在全球的各种项目都是人民币计价的怎么办，我们可能面临很严重的输入型通胀。大佬文集圈要知道，如果输入型通胀很严重，是严重打击制造业的，也就是严重阻碍产业升级的。

于是，美国救市的卡点联动我们救市的卡点，我们救市的卡点又联动着美国救市的卡点，这才是真相。这才是为何我们924宣布这么多救市举措，可是类似降息降准等承诺仍然还在等落地的根本原因。两边还是没有谈妥，那么，上述的“剪不断理还乱就成了一团乱麻。

结果是，美国通胀控不住，我们偏通缩走不出。美国是，什么都涨，就是人贱；我们是，请来一把房股双大家久旱盼甘霖。

本质上，目前的态势，都已经不是一方单独微操可以管控的了。就像博弈论那样，对手的下子决定了我的下子，我的下子又决定了对手的下子。

川普想要上任后100天内来华访谈，真实的目的就在这里。在此之前，一定是双方各种谈判，各种讨价还价，这个逻辑就像之前所谓的第一阶段贸易协议那样。

放眼全球，2018年开始，全球主要经济体就已经步入债务驱动经济增长模式，无一例外。

这里我们需要明白，之所以会遇到这个问题，就是没有什么新产业做了，大家只能依靠烧存量储蓄的方式维稳了。

这种普遍性的做法，势必会导致一个必然的后果：全球下一轮金融危机可以被人类一直往后拖延，直到存量储蓄烧的差不多了，那就是一个史诗级的重大危机来了。表面看，好像繁荣周期持续很久，其实，经济的身子早就被严重透支了。

因此，要么危机不来，要来就是摧枯拉朽。那么，为何我们这里的救市举措一直是让人扭扭捏捏？

现在大家应该马上看懂了！2021年~2022年期间，我们这里前后10多次变着法子的明示暗示，后面美国会有很严重的危机，需要提防。

这个恰恰是与2021年开始进行楼市整顿、行业整顿等问题一起发生，目的也就清晰了。

现在，全球主要经济体，人口的总和生育率都早已击穿2.1，也就是击穿了代际传承保持基本人口稳定的警戒线。所以，后面大家都是高龄少子化趋势，无非步伐速度略有不同罢了。高龄少子化趋势成为全球主流趋势，意味着，用钱的人多了，赚钱的人少了。即，全球的储蓄率会越来越低，且不可扭转。

问题是，现在全球都呈现出人性怕痛的普遍现象。越是如此，反而各国越是要烧储蓄来救市，否则短期的困境就可能无法通关。这种近乎超越理性的做法，却又是全球的主要风景线。

因此，全球储蓄率后面几年继续保持快速下行，也是一定的，是不以人的主观意志为转移的。

学过经济学的朋友都知道，经济增长依赖投资，投资来源于储蓄。这也就暗示，后面全球的低增长是长期现象，赚钱难、投资难、选筹难、好项目少，反而是正常的。

不过，当我们把大国博弈的因素纳入一并考量，事情就更加棘手了。大国博弈的逻辑是，不惜代价，确保对手先倒下。而大国博弈要熬到最后，势必是需要海量储蓄支撑，否则如何可能？

既然如此，美国这里的做法就是对外收割，中国的做法则是储蓄领域“量入为出”。这也是为何我们感觉这几年救市举措总是打点滴的根本原因之一。

中国的核心策略是，利用出口对外融资，拿走更多人家的储蓄；如果实在外部融资很难，再里面烧储蓄，给自己的产能匹配订单即可。

所以，不是坊间说的，中国房股再也救不起来了。真的愿意，只要大规模的流动性注入，自然是拉起来的。关键是，结合上面的策略，村里现在更加强调救市节奏、力度、方向的管控，不能太少，也不能太多。争取熬到最后，让美国人先倒下，这才真相。用接地气的话描述，能够外面找奶妈的就极可能找，实则不行的再自己来奶自己的孩子。

当然，这里短线我们还要考虑到美国两党斗争的残酷现实。川普必然是要确保2026年中期选举的全面获胜，否则民主党就完成了反扑的一步。

川普完全可以选择今年上半年反复拉扯，同时优先解决美国国内的很多问题。然后，争取2025年下半年（最晚2026年年初）实现跟中国的大买卖敲定，这样他2026年的中期选举才更有把握。
