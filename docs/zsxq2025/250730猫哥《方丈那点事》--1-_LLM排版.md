
# 方丈那点事

少林寺方丈释永信被爆出种种劣迹，引发互联网上很多群众的愤慨，今天就给大家聊一聊方丈那点事。

首先，释永信腐败的土壤就是寺庙经济，可能很多人都不清楚，寺庙经济是这个世界最牛逼的商业模式，甚至可以说没有之一。为何？因为寺庙经济完美契合了“低成本、强刚需、高毛利、零税负”等普通商业模式想都不敢想的几个特征，并且拥有特别充沛的现金流。我们一个个来为什么寺庙现金流特别好？

首先是门票收入：如灵隐寺年接待1200万人次，门票45元，仅此一项年收入超6亿。少林寺每年门票收入也超过3个亿（少林寺门票收入70%归地方政府所有）。

然后就是香火经济：从几十元的香到几万元甚至几十万元的“头香”、牌位供奉、开光法物，利润空间极大。功德箱不设上限、无发票、无找零，现金直接入账，大部分寺庙甚至与时俱进支持扫码支付。

然后就是无限可能的“玄学消费”：寺庙文创如手串、佛像、奶茶、白酒等，经“开光”后溢价数十倍，这是普通人可以想到的，大众看不到的还有更赚钱的，比如在菩萨旁边供上一个个人牌位或者延生禄位，据说可以享受信众的香火，得到神仙的提携，就有延年益寿消灾避祸“功效”，这样一个牌位即便动辄数千至数万元，却有无数人愿意买单，成本却几乎可以忽略不计。

最后就是，历史积累+政策支持：土地、资产、免税三重红利，古代寺庙本身就拥有大量土地与佃户，现代虽不再收租，但多数寺庙地处景区核心，享有土地和资产升值红利。最近十几年，多地政府将寺庙作为文旅项目核心，引入资本开发周边商业，寺庙成为核心IP，只要这个IP立得住，就不愁没有源源不断的人流创造出巨大的商业价值。

实事求是的说，释永信对少林寺这个IP在商业版图的发展壮大也是做出巨大贡献的。很多人说，少林寺这个IP符号能做大做强主要来源于80年代的一部电影《少林寺》，这个描述就有点夸张了，少林寺这个IP最大的来源是金庸的系列武侠书，对于60、70、80后者而言，基本都是看金庸武侠长大的，全胜时期，号称有华人的地方就有金庸的武侠书。在金庸的武侠书中少林武当是标志性武学圣地，几乎每一本书都有对少林寺的描述，很多重要关键人物武功都来自少林，这是释永信能将少林寺商业化的基础，当然释永信的商业头脑也很出色，这么多年，能将少林寺这个IP做大做强并能到国外去跑马圈地就此一家而已，大家对比一下，在金庸武侠中一直能与少林并驾齐驱的武当商业化就差了不止一个档次。

寺庙经济最牛逼的特征就是免税。一般市场经济下一个经营主体，也要做个账本纳税吧，宗教场所却是特殊的例外，享有免税政策，无营业税、无所得税，税负成本基本为0。也就是说，寺庙无论收入多少，连个账本都不需要做，账上巨额现金怎么用，就是主持方丈一句话的事情。这种几乎没有任何监管，财务透明度极低的寺庙经济自然是黑幕重重。很多人为寺庙捐献以为是捐给菩萨，其实就是捐给方丈。

正因如此，寺庙被称为“完美的商业模式”：低成本、零监管、零纳税，现金流充沛，甚至还有地方政府和资本主动送钱上门。

虽然目前寺庙已经完全商业化，但是年轻人“为焦虑买单”，寺庙依然是成为年轻人最愿意去的“情绪避难所”，烧香、求签、打卡成为一种精神消费仪式。为寺庙消费大多数年轻人都很愿意慷慨解囊。

另一方面，玄学是互联网平台的流量密码，在大数据算法推动下，互联网平台也很愿意推广相关主题，如“雍和宫已读乱回”“灵隐寺斩孽缘”等话题破亿次浏览。

讲完了寺庙经济，我们再来说说方丈这个职务。宗教人士对国家属于有统战价值的人群，所以，寺庙的主持或者方丈一般都能其他的政治身份，比如政协委员，人大代表这些都是常态。更为重要的是，较大寺庙的方丈还可以轻而易举结交高级官员，高级官员也很愿意结交宗教界的名人，官场小说《国画》就描述了一个大城市市长与某个寺庙主持结交的过程，对于这些高级官员而言，在公务之余，到寺庙走走，与主持聊聊天是非常放松的，佛法包含哲理，讨论一下佛法，也能提升自己的认知格局。

所以宗教界著名人士在官场的交往天花板是很高的，很多有钱有势的大老板磕破脑袋也结识不了的大佬，这些方丈主持却能登堂入室，坐而论道。

好了，现在我们可以把自己代入一个方丈的角度，你面临的就是这样一个状态。寺庙经济很赚钱，现金流滚滚而来，没有任何监管，不需要做账，连税务检查都没有，巨额现金怎么用怎么花就是自己一句话的事情。然后自己在社会有政治光环（人大代表、政协委员这些），同时还与很多权贵有私交，有点小麻烦找这些有私交的权贵打个招呼就可以处理得干干净净。

在这样的情况下，你会不会腐败一下，放纵一下人性？反正横竖看上去都没啥风险！和尚又不是公务员说实话，别说是人，就算是佛祖估计也把持不住。所以释永信塌台只是冰山一角，国内这些寺庙主持，如果都较真查一查，真没有几个经得起查。

最后说说释永信为什么会倒台。官方通报罪名有几个。一个是与多名女性保持不正当关系；一个是涉嫌挪用侵占项目资金寺院资产；说实话，这些罪名十年前就有少林寺武僧实名举报，但是河南方面排除调查组调查最后的结论确实“查无实据”。

这说明了啥？说明了河南方面不愿意动他！当时这么大的舆情，即便释永信有某些高级官员有私交也不大可能这么容易过关的，所以河南方面不愿意动他大概率是地方政府的集体意志。

为啥河南方面集体不愿意动他呢？因为他对河南贡献太大了，一个过去连吃饭都困难的寺庙在他手里发展成为一个庞大的文化产业，给河南带来多少经济效益与社会效益？带动多少人就业？

某种层面释永信与少林寺这个大IP是一种紧密捆绑关系，一旦释永信倒下，少林寺这个IP影响力将受到大幅度负面影响，这是河南方面不愿意动他的原因。

另外，即便比河南更高层面也不愿意动他。为何？中国这20年，文化品牌出海真正站住脚发挥影响力的就少林寺这一个IP，即便官方花费大力气，包括人力物力的孔子学院最后都失败了，由此可见少林寺对于中国国家层面出海硕果仅存的文化IP是多么重要。如果释永信倒了，那么出海的这个少林寺品牌也基本是废了大半。

那么为什么释永信今天还是出事了呢？坊间传言很多，比较多的一个传言就是他未经批准去见了教宗，所以有人认为这是踩了红线，所以被动了云云。

但是我觉得释永信这个事虽然做得不对，但也不是太过（除非见面时候说了什么过头的话），但是释永信这把年龄与智商（我有朋友与他见过面非常推崇，认为他是一个有大智慧的人），基本分寸感应该是有的，如果没说过头话的，只是见了面，教宗本来与我们关系还可以，回来大致就是私下警告一下，顶天就是把释永信有关社会职务（人大代表、政协委员这些，方丈都不可能动）拿下就到顶了，不至于现在大动干戈。

将释永信过去新账老账一起算，这么干负面影响极大，不光是河南方面损失极大，好不容易出海站住脚少林寺这个IP也基本是废了。

所以导致释永信倒台的事情一定远远比坊间传言的各种小作文更大。

所以释永信出事就只有一种可能。ZZ站队出现了重大问题。
