
# 大英良心汉弗莱付费专栏：即将登场(2美国人的自救)(2)

当前的研究学风是很浮躁的。坐而论道，毫不掩饰对欧美的崇拜，只提出问题，不提供结论。为什么不提供结论呢？原因无非就是害怕担责或者是怕提出意见不准影响自己的羽毛。在面对大是大非问题的时候，理论必须是决策的先导和武器，而不能成为决策的阻碍和杂音。既然美国人提出了战略欺骗，那么在这个当口，应该采取的必然是清晰的理论支持。

## 正确看待美国的战略欺骗

从中立的角度来说，美国人的战略欺骗，核心是战略自救。这种自救是为了救帝国主义自身。作为首席来说，对于这种自救是不看好的，但是帝国主义占上风的态势是不会变的。

让我们用老大哥的案例举个例子吧。毫无争议，苏联在自己的存在时期在经济发展速度上并没有落后于发达资本主义国家。在经济和军事政治，老大哥是世界第二强国，在航天、核能方面成为第一大国。它发展了自己的能源经济潜能，建立了结构最发达的统一的能源体制。苏联的基础科学、教育体制和干部培养体制客观上被认为是世界上最好的。但是为什么老大哥会垮台呢。

戈尔巴乔夫2010年坦白的说，在他上台之际，国内并不存在特别尖锐的矛盾，国际上也没有任何一种力量能够消灭苏联。他的结论就是，苏联垮台完全是由于改革者自身的错误造成的。核心就是它这个叛徒。因为戈尔巴乔夫为首的苏共中央在改革的名义下推行一条自我否定、自我丑化的机会主义路线，放弃马克思主义一元化指导地位，放任意识形态多元化，理想信念缺失，是苏共覆亡至关重要的原因。

当前的美国是这样吗？有一样的地方，但是还不一样。当前的美国是准备集权，演变为一个大号的台湾：即，共和党试图铲除民ZHU党，如同当年民进党试图铲除寡民党一样。但是帝国主义的本质还是没有变化。

从历史的长河来看，美国一直在自救。

实际上从2008年危机以后，美国就在自救。当时，美国的反危机措施是左转回归“第三条道路”。所谓“第三条道路”，不是我国一些人曾经臆想的那样，是介于社会主义与资本主义之间的一条道路，而是20世纪90年代中期由美国民主党总统克林顿、英国工党首相布莱尔、德国社会民主党总理施罗德三人提出的带有一点改革、修正或调整意义的施政理念；具体说，是美欧发达资本主义国家的所谓左翼政党，面对国内阶级结构的变化随着现代化的推进，产业工人阶级人数下降，中间阶层人数上升为争取中间阶层为代表的中间选民而提出的一种策略性口号。

所谓走“第三条道路”，也即向欧洲社会民主主义+社会市场经济模式靠拢的调整、改革，走一条介乎美英和欧洲大陆模式之间的道路。

2012年以后，美国人同新自由主义拉开距离甚至抵制新自由主义的危机治理措施就是自救，这里面包括，四个方面的措施，再工业化、去金融自由化、增加社会建设投资、提高最低工资标准和减轻国内中小企业税收增加就业机会等。但是并没有什么用。这才导致了特朗普的两次上台。

那特朗普上台后为什么对民主党如此的仇恨，除了民主党已经异化为一个全世界的毒瘤党对美国毫无任何感情以外，并且对特朗普本人穷追猛打以外，最根本原因就是美国底层已经对民主党毫无任何情感可言，这为特朗普自救提供了支持。

准确的说，当前美国内部的自救已经到了最后最危险的时刻。进群加微信 如果美国人无法自救，那么内部出现社会主义思潮将是大概率事件，而这，将对整个欧洲尤其是北欧世界是巨大的打击。这意味着，长达600多年来，西方一直披着资本主义外衣的奴隶制社会架构，将在欧洲重新被社会主义打击。这对欧洲的农奴主和封建领主来说是不可接受的。

## 帝国主义美国当前并没有倒下，而是在准备战争

对时代性质的判断，是我们观察问题和进行决策的根本立足点。总有人认为帝国主义和无产阶级革命时代早已过去，现在是“和平、发展的时代”，甚至说从马克思到列宁再到毛主席对时代的判断都错了。甚至试图以此来颠覆我们看待问题的角度。这些伎俩都是不足为奇。

我们必须把主要矛盾和次要矛盾分开，把根本的立国方策与每个时期的外交活动、一时的曲折分开。当前众多事实表明，帝国主义时代远未结束，恐怕至少还要延续一个世纪。

为什么当前帝国主义比之前要消停呢？根本原因是他们处在被制衡的状态，即原来殖民地半殖民地国家的人民起来，社会主义国家兴起，尤其是社会主义国家拥有核武器、出现核威慑力量相对平衡，它们害怕原子弹，所以不敢轻举妄动。加上社会主义中国的灵活策略，将坚持斗争与有条条件妥协结合起来(列宁最早提出并实施)，可以创造、争取和平发展的环境，扩大经济合作，利用两种资源、两个市场。

但这并不意味着改变了世界的基本矛盾。也并非国际垄断资本主义所给的恩赐，而是反复博弈的结果。

美国当前接受不了衰落，一定会转移矛盾自救。它有三个选择：

第一个选择，在东侧，跟中国合作，牺牲台湾，从而换取交易筹码，这个筹码大概率是中国继续购买它的国债，并且在关键性的一带一路让步。

第二个选择，在西侧，跟俄国合作，牺牲乌克兰，从而换取交易筹码，这个筹码大概率是牺牲塞尔维亚或者是南欧的某些国家，从而继续在南部欧洲打开一个缺口，进而继续为南欧抽血。

第三个选择，在西侧，继续跟俄国合作，牺牲伊朗，从而换取交易筹码，这个筹码大概率是牺牲中国的利益，从而继续逼近一带一路，从而让中国投入的海量资源化为无形，进而收归美国人所有。

这三种选择都取决于美国人施压的力道。因为从特朗普的角度来看，实质上已经完成了内部动员。这可以成为美国版的腾笼换鸟。

什么意思呢？就是特朗普已经把旧有的利益体系，全部用新的利益体系取代了。在金融方面，使用加密货币替代原有的联储体系；在军事方面，使用加州的高科技企业替代原有的老牌军火商；在人事方面，用自己的舔狗取代原有的官僚体系。进群加微信pep458 一个几乎非常完整的皇家禁卫军已经形成。这也是马斯克为代表的高科技精英非常喜欢特朗普的原因，因为给了它们出路和上升空间。

如果这三种交易都不能达成，那么唯一的选择就是牺牲欧洲。为什么是牺牲欧洲呢？因为历史上，发生在欧洲的两次“世界大战”曾给美国军工集团带来了相当疯狂的机会。

当前美国的债务危机，表面上看，是乌克兰抵押矿产，欧洲还钱，但实质上，从根本上说，欧洲已经处于分裂的边缘了。

而英国正在为欧洲分裂做好准备。它们不介意再来一次。而欧洲的分裂，将为中国和俄国提供很好的发展机会和防御机会。

但是这是最后一步，在特朗普没有完成跟俄国的交易之前，是不可能跟我们交易的，在我们跟我们交易之前，是不可能分裂欧洲的。

这必须有一个先后顺序。这个先后顺序不能乱。如果乱了，那特朗普的所有计划都被打乱了，这也是美国自救的失败。

因为如果跟俄国的战略欺骗谈判失败，那就意味着在东欧会出现巨大的战略真空，这个真空的出现，将意味着当前工业能力极其强悍的俄国，会反推到波兰一线，这对于美国来说，又是不可接受的。

而跟中国的战略欺骗谈判失败，中国采取斩首方式对民进党进行强硬追杀，将意味着日本的位置出于真空，日本将迫不及待的成为正常国家，从而横亘在中国和美国之间，成为难以控制的烫手山芋，而美国必然放手让日本一搏对付中国，但是日本必然重新采取偷袭的方式，让中美先打一仗，从而进入到一种难以调和的热战状态。

这也是首席提出，最危险的则是中东一线，这里涉及到我们一带一路的利益，如果双方媾和，那么欧洲也能存活下来，但是却牺牲了我们的利益。

因此在这个区域，必须是我们反击的重点。

如果这三种欺骗方式都不能成功，那最后就只有一个可能性，战争再次在欧洲大陆爆发。只有那里才能让中国俄国美国同时感受到危险的远离。而美国的自救在某种程度上算是成功了。

当前美国在欧洲进行的合纵连横：超大力度扶持主要国家右翼的做法，看上去是一种西方整体的大和解，但是根本上还是为了进行战争动员，这种战争动员如果推进的好，就是针对中国和俄国的再次战争准备，如果动员的不好，就是各个国家内战的前夕准备。

美国人做好了两手打算：如果欧洲服务美国，就矛头全部对准中俄，但是如果中俄软硬不吃，那就只能牺牲欧洲。

毕竟，所有明眼人都看的清楚：如果美国衰落并且倒下了，想继承美国衣钵，变成世界老大的只有欧洲。
