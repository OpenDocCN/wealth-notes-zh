
# 飞瓜数据-2024年度抖音品类洞察

## 服饰内衣

### 电商趋势洞察

#### 服饰内衣品类电商总览
秋冬销售持续创新高，3月为换季爆发点。电商推广热度月趋势与环比表现显示，直播带货方式热度占比最高，达52.32%。品牌自营和商品卡也是重要的带货方式。

#### 服饰内衣细分品类分析
年度高增长品类TOP5包括新中式女装、大码女装、唐装/民族服、汉服和派克服。新中式女装增长达5006.5%，其中连衣、刺绣和盘扣等样式受欢迎。

#### 服饰内衣品类电商卖点分析
高热度、高增长卖点包括清凉、显瘦等。例如，“会呼吸的清凉内衣”女士洞洞透销售热度增幅达316%，主要面向31-40岁熟龄女性。

### 营销趋势洞察

#### 服饰内衣品类营销总览
品类曝光热度月趋势显示，抖音已成为服饰销售的重要阵地。引领色彩穿搭风格化，红人崛起自造秀场，卷出品质新高度。

#### 从跟随潮流到引领时尚
抖音已成服饰销售重要阵地，春风自抖音来，夏日精神度假，秋冬流行色彩等趋势引领时尚。品牌如CHENSMCHEN和MARIUS通过与抖音合作，举办秋冬发布会，联合GQ发布抖音in时装周等，提升品牌声量和销售热度。
