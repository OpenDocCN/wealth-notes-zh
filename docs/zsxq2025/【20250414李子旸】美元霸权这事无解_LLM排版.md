
# 美元霸权这事无解

美元霸权这事，还真难办。说起霸权，人人喊打，但要真把美元干翻了，霸权不再，国际贸易用什么货币结算呢？抢答的人会脱口而出：用人民币啊。可是欲戴王冠、必承其重。大家都用人民币结算，我们起码得让人民币境内外自由流动吧。这能做到么？

为了防电诈，本国居民账户稍有异动，都严格管制。这套办法很有必要，但显然不可能用于全世界。

那么美元为什么能在全世界自由流动呢？他们是怎么解决这个问题的呢？一方面当然是美国政府对电诈什么的不在乎，被骗是你自己活该。点儿背你不能赖社会，命苦你不能怨政府。但是还有一个更重要的原因。美元在全世界自由流动，背后倚仗的是一个没有明说出来的威胁：有敢对美元捣乱发难的，闹得小，华尔街剿灭；闹大了，美国军队武力剿灭。美元霸权的背后，其实是军事优势和军事霸权。这和国家内部的法币，原理是一样的，法币的力量，终究来自军队警察和监狱。

我国第一不干涉外国内政，第二没有遍布全球的军事基地和驻军等等，也就是没有世界军事霸权，我们也不谋求世界军事霸权。那人民币咋自由流动啊？自由流动了，岂不是任人玩弄宰割么？人民币不站出来接盘，美元霸权就只有肘腋之患而没有心腹之患。美国人还可以吃全球铸币税很久。此事目前无解。
