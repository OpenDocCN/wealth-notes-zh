
# 真金白银

“昨天的变盘，令人不禁想起了2022年的那个黑色幽默段子。如果核战，人都没了，那么亏钱也无所谓——所以闭着眼睛买；如果不核战，那么现在买的仓位就会暴涨——所以睁大眼睛卖。反正无论如何，干就是了。”这个段子勾勒出在极限的情景里，市场回报的不对称性。我们作为交易员，就是要不断地寻找市场结构中不对称的回报：向下损失有限，而向上回报无穷。而我在这里反复推荐的一个资产类别，就具有如此不对称回报的特点，并在昨日的俄乌局势中大放异彩。

昨夜，黄金跳升近3%，回到了3400附近。与此同时，另一个贵金属，历史上曾经作为中国货币系统的“双本位制”之一的白银，跳升了近5%。问题是，如果昨夜市场是为了避险而买黄金，那么为什么股票等风险资产反而上涨，而美债、美元这些传统的避险资产反而下跌。不仅仅如此，贵金属的技术图形也出现了非常有意义的结构。这些结构，对于贵金属价格下一步的价格走势，意义重大。

昨夜，美国ISM制造业进口指数暴跌到了2009年次贷危机以来最低，美国CEO相信指数暴跌到2000年有数据最低的水平之一。同时，全球的合并收购项目合约签订的数据也暴跌到有历史上最低水平之一。一般来说，制造业进口指数略微领先美国的制造业产出水平。不进口原材料，美国又拿什么来生产？因此，如此低的，类似千2009年次贷危机的进口水平，预示着美国制造业将出现大幅的收缩。

然而，这个数据在这个周期里明显受到了特朗普关税的影响。因此，美国今年一二季度的经济数据，在分析的时候需要考虑关税这个特殊的情况。比如，美国一季度的GDP萎缩了0.3%，修正后还是萎缩了0.2%。

这显然是因为一季度美国商家为了抢跑特朗普关税而大幅进口围货，导致历史性经常账户赤字，而进口拖累一季度GDP增长。

然而，二季度这个情况则会恰恰相反：由于渠道内存货充足，美国的经常账户赤字大幅收敛，导致美国经济增长大幅回暖。刚刚，亚特兰大联储的GDPNow指标，预测二季度美国GDP将增长4.6%。如是，这将是近五年来最高的经济增长。

当然，看完以上讨论的读者，至此已经知道这样的增长只是统计模型调试的结果。而实际上，人们很难去相信在特朗普关税的阴霾下，美国经济可以快速地增长。同时，模型没有考虑到在不确定性高企的背景中，美国消费的放缓和投资的减少。同时，美国还大量地进口黄金，仅此一项，就可以拖累GDP增长近2%。简言之，二季度美国GDP会高，但不是因为我们想看到的原因。三季度则将会继续放缓，甚至可能再次萎缩。如是，美联储当前“数据依赖”的货币政策很可能也将因为二季度经济数据表现强势而继续滞后。

本周，美方称很可能将进行对话。几天前，美国财长本森特称双方谈判进展不快。其实是因为严格控制稀土出口。如果没有稀土，美国的汽车工业将回到疫情时期无工可开的情况。曾记否，疫情时期美国二手车的价格弑升，因为新车无法生产？而美国的汽车工业是工会势力最强的行业。

按照东方的谈判习俗，一般是下属与对方沟通交流，谈好了之后请领导拍板签字，庆典仪式。如是，如果本周可以会晤，那么显示关税谈判还是取得了可喜进展的。毕竟，特朗普已经按耐不住了，而双方的“关税休战期”还有几十天的时间。

总结一下：美国的经济外强中干，二季度美国经济会因为技术原因而好转，但是经济增长放缓的势头在三季度将卷土重来。特朗普关税对美国经济运行造成了极大的不确定性。而从特朗普心焦火撩的表现来看，关税对美国经济的影响远大于经济数据所显示。因此，美国再次实施刺激政策、货币宽松的概率在上升而非减小。而股票已经嗅出了这个情景。

同时，不仅仅是市场的不确定性，俄乌的不确定性也到了剑拔弩张的地步。赌市中对千核武事件的概率判断一度痰升到近20%。如是，概率上这是一个大概率事件。

在不确定性如此之众的情况 下，黄金白银敝升的原因，就一目了然了。技术上，白银巨大的五十年年线“杯把子”结构（cup & Handle), 开始突破了。而我们之前一直强调的黄金“上升旗帜”(rising flag)也开始了其突破。

在基本面逻辑和技术面的加持之下，黄金将会继续上涨，白银将会突破。而交易数据显示：在美国日盘和夜盘里，黄金都是买盘；美股日盘是买盘，夜盘是卖盘；而美债，则是日夜都是卖盘。据此，全球资金的流向，一目了然。
