
# 特朗普的关税计划曝光

懂王的关税计划一直是外界关注的焦点，但具体到底如何实行，始终没个准信，直到1月29日，根据美国媒体的爆料，终于有了一些眉目。当天，美国新任财政部长贝森特提出先在2月份，对所有进口商品征收2.5%的基础关税，然后再每个月增加2.5%，经过十个月的时间，最终让针对进口商品的普遍关税达到20%-25%的区间。

贝森特这样做的目的，是既征了关税，又给外界一定的时间缓冲，毕竟关税会影响通胀、供应链等诸多问题，不能一下子拉满，但美国社会的大氛围又决定了贝森特不能后退，否则就会被视为软弱。同时，贝森特还试图把关税作为极限施压的武器，给外国政府一些紧张感，好逼他们来与美国谈判，想让对方拿出筹码交换。

懂王对这个计划不是很赞成，他在贝森特提出计划后几个小时，在空军一号上，就关税问题进行了讲话。大致意思是，普遍关税要征，但不给什么缓冲期，应该一上来就20%以上，你无法适应是你的事。对企业来说，你无法适应是你的事，懂王的风格向来如此，都这么多年，还不能适应，那就只能怪你自己；对其他国家来说，不能适应就更好了，逐月提高关税，容易让其他国家看穿你的底牌，觉得你就不是想加关税，而是更想借着关税，进行谈判。

这种心理一旦形成，关税的效果就会大大下降。不如一上来就极限施压，20%以上，让他们知道我们是认真的，这样更有利于获取利益。同时，懂王还有两个关键的反馈，一是要对重点国家征税，指的就是加拿大和墨西哥；二是对重点行业征税，主要是能源、钢铁、生物制药、半导体、汽车。

在美国政府内部，对于关税问题是分为两派的，包括以贝森特为首的温和派（事实上一点也不温和）和商务部长卢特尼克为首的激进派，懂王的立场，更偏向激进派。尽管存在分歧，但两者的共同点还是很明显的，首先，他们都认为要对进口商品实行普遍关税，其次，他们都认为要将普遍关税，提高到20%以上，分歧无非是具体的方式，和实现的手段。

这就意味着，后续美国对进口商品实行普遍关税，是大概率的。美国，要掀桌子了。很多人关心，这份计划对我们会造成什么影响呢？实话说，影响最大的不是我们，因为中国的对美出口以机电产品为主，不在懂王的重点关照范围内，反而是各个盟友和“半殖民地”，受影响最大。

前面说了，懂王的征税思路有两个“重点”：重点国家和重点行业。

重点国家方面，为啥要针对墨西哥和加拿大呢？很多人想不明白，其实原因很简单，美国从加拿大和墨西哥进口的金额，比从中国进口还多，每年占总进口额的三分之一，是最大的进口来源。如果能大幅减少从墨西哥和加拿大的进口，意义将很大（当然，能不能实现是另一回事）。而且，这两个国家还特别好拿捏，且不说经过美国数十年的渗透，两国早已是美国的经济殖民地，存在大量带路党。就拿贸易数据来说，两国对美国的依赖程度，都是极高的。加拿大每年75%的出口，是出口到美国，墨西哥则是80%，意味着关税大棒只要挥舞起来，他们甚至没有还手之力，短时间内，想寻找替代市场都不可能，柿子当然要捡软的捏。

重点行业方面，钢铁、药品、汽车、能源和半导体，除了药品多一些，其他四个行业，中国对美国都没多少出口，钢铁每年对美出口量不到一百万吨，以中国的钢铁产能而言，几乎可以忽略不计。半导体，我们自己都不够用，至今依赖进口，出口自然也不可能太多。汽车，拜登早就对我们加了100%关税，也没什么出口。能源，中国更不是能源出口国。

毫无疑问，这个关税计划就是奔着盟友和“半殖民地”去的，比如半导体，懂王在1月27日说，要对台湾省的半导体产品征收100%的关税，还暗示要取消对台积电的补贴，顿时把岛内吓得不轻。

我们在前面的文章里说过，懂王其实不太愿意出“对华加征关税”这张牌，果不其然，还真就预测对了，背后 reasons，主要有三点，一是经过多年的贸易战，美国上上下下都知道了，中国真的不好对付。懂王只有四年时间，是不太充裕的，想在有限的时间里取得好成绩，最优选择肯定不是和中国纠缠，性价比太低了，不如去捏软柿子，不仅肉多，防御力还低，站在懂王的角度，加拿大、墨西哥、中国台湾这些软柿子，简直是不捏白不捏；二是中国对美出口的商品，与民生关联程度太大，要是加太多关税，容易导致通胀卷土重来，而通胀正是懂王战胜哈里斯的关键，所以在涉及通胀的问题上，必须谨慎；三是懂王希望和中国做交易，还记得第一任期尾声，我们和懂王签署的协议吗？内容是采购大量美国的能源和农产品。到了第二任期，懂王肯定希望再来一次，毕竟农民和能源企业，是他的基本盘。全世界也只有庞大的中国市场，能吃下天量的美国传统能源产品和农产品，别的国家和地区都做不到。

而一旦关税对中国冲击太大，我们势必报复，让采购协议遥遥无期。基于以上三点，懂王便把关税的收割对象，转向了盟友和“半殖民地”。正应了那句话，做美国的敌人是危险的，做美国的盟友是致命的。

只不过，这不意味着我们高枕无忧了，毕竟后续还有可能发生变化，还是要密切关注。
