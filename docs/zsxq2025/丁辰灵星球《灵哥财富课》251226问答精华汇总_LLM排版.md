
# 丁辰灵星球《灵哥财富课》251226问答精华汇总

提问：灵哥您好！我是个投资小白。请问黄金实物现在买划算吗？还会不会跌啊？

提问：灵哥好，请教下：这两天 Rmb 汇率看这走势都快要破 7 了，感觉港股却没啥反应，钱都去哪了；地产作为国内的核心资产，这样的升值对于港股的地产公司而言会有利好吗？
丁辰灵回答：你觉得没信心的时候说来就来了。

提问：灵哥好，请教一下：贝森特最新表态抨击了美联储过去这些年的量化宽松政策，认为美联储买了太多美债、MBS，扩表太快，引发资产价格大涨，导致拥有资产的群体与普通工薪阶层之间的贫富差距扩大。特朗普未来更倾向于使用利率（降息）工具，控制使用数量（扩表）工具，财政政策也将进入紧缩期。这对于中国 a 股和港股有何影响吗？
丁辰灵回答：我公众号有写过啊，往前翻翻。

提问：灵哥，未来的铂金的中长期（3-5 年）趋势您怎么看，手头有 1 公斤现货铂金板料，考虑是留着还是不管它扔个 3-5 年。
丁辰灵回答：三五年不知道，这一两年肯定要涨的。

提问：请教灵哥：港股创新药最近几个月一直表现不大好，请问您觉得建议可以买入了还是再等调整幅度深一点？谢谢！
丁辰灵回答：还能调整到哪去？你也可以比较下，优必选也从高点往下一个大调整至今还未恢复。
