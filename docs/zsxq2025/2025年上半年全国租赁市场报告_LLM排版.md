
# 2025年上半年全国租赁市场报告

## 一、租赁市场景气度表现平稳，二线城市需求热度更强

### 1、上半年，租赁市场景气度表现平稳

根据安居客租赁市场景气度指数显示，2025年上半年重点40城租赁市场景气度整体保持在“温”的状态，市场表现相对平稳。春节后“小阳春”和“毕业季”期间，租赁市场呈现回暖迹象，市场景气度显著回升。

整体来看，今年上半年受到挂牌租金承压的影响，市场景气度表现弱于去年同期。究其原因，一方面，受到二手房市场承压的一定影响，叠加多城市保障性租赁住房的入市，上半年租赁市场整体呈现以价换量的特点。结合挂牌房源去化周期来看，进入二季度后租赁房源出租难度有所上升。另一方面，由于整体宏观环境下求职者对收入预期的下降，导致供需市场活跃度呈现转弱迹象。

### 2、全国40城挂牌价格表现弱于同期，西部城市价格表现较强

2025年上半年，全国40城租赁市场挂牌租金水平仍延续小幅下行趋势。结合城市表现分析，全国40城内部呈现显著分化。一线城市挂牌价格表现相对较弱，同环比均呈现下降态势。受到疫情后就业活跃度下降的影响较大。此外，中西部二线城市挂牌价格表现相对较强。一方面，由于二线城市普遍挂牌租金价格水平相对较低，市场内生性支撑较强。另一方面，二线城市产业结构的不断完善，吸引了年轻人口的逐步流入，对租赁市场价格形成一定支撑。整体来看，中西部省会城市及热点二线城市的挂牌价格表现要强于一线城市及华东城市。

### 3、整体供需较去年同期持平，二线城市需求热度表现更佳

2025年上半年，全国40城租赁市场供需表现基本较去年同期持平。一季度租赁“小阳春”行情供需活跃度表现强于去年同期，但持续性相对较弱。步入二季度后，租赁市场供需表现小幅转弱。临近“毕业季”窗口期，租赁市场呈现小幅回暖迹象，但热度表现低于去年同期。

根据安居客监测数据显示，上半年全国40城整体租赁新增挂牌量较去年同期下降约1.3%，较下半年环比下降2%，整体线上租赁挂牌量保持相对稳定。但受到各城市二手房挂牌情况的影响，租赁挂牌量分化严重。其中，25个城市新增挂牌房源量同比上涨，而15个城市新增挂牌房源量同比下跌。一线城市中，北京和上海新增挂牌房源量同环比均呈下降态势，而广州和深圳则呈现同环比小幅上涨趋势。此外，济南、天津、沈阳及乌鲁木齐等城市新增挂牌房源量同比涨幅超过30%，而宁波、三亚的新增挂牌房源量同环比跌幅均超过30%。

## 二、重点10城租赁市场韧性较强，二线城市表现更稳

### 1、热点二线城市挂牌租金收益率维持2.2%-3%

根据安居客监测数据显示，上半年，重点10城的挂牌租金收益率整体表现相对稳定，波动幅度相对较小。一线城市的挂牌租金收益率整体保持在2%以内，二线城市的挂牌租金收益率整体保持在1.6%-3%左右。

对比2024年各城市挂牌租金收益率变化，今年部分城市呈现出挂牌租金收益率小幅回升的趋势，如北京、广州、深圳及杭州等。今年春节后，北京、广州和深圳的挂牌租金收益率均有所上升。

### 2、上半年，重点10城租赁供需向中低价格转移

根据安居客监测数据显示，上半年重点10城租赁供需向中低价格转移。整体来看，新一线城市租赁需求热度表现最强。尤其，进入二季度后，新一线及二线城市需求热度呈现小幅回升。

结合城市热度情况来看，整体仍呈现较大分化。全国40城中，有18个城市租赁需求热度同比呈上涨，但过半数城安市租赁需求热度同比回落。其中，一线城市租赁需求热度同比均呈现小幅下跌，而二线城市需求热度表现更强。重庆、南京、福州、天津及东莞的租赁需求热度同比涨幅超过10%。此外，西安、武汉、厦门、成都、中山及无锡等城市租赁需求热度同比跌幅超过10%。
