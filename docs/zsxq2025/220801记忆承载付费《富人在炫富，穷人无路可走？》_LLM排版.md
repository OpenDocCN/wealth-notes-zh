
# 富人在炫富，穷人无路可走？

## 强势文化与弱势文化

《天道》中，丁元英提出了强势文化与弱势文化的概念。强势文化塑造强者，弱势文化塑造弱者。他只给出了结论，没有说明如何做到。

剧中关于强势文化的定义有两点：
1. 去掉等靠要的思想
2. 忍人所不能忍，能人所不能为

去掉等靠要的思想，不是让你孤军奋战，而是明白自己才是核心，围绕核心做资源整合。忍人所不能忍，能人所不能为，本质是次数多，需要不断练习和积累。

## 如何做到？

1. 控制自己的身体，绝对服从指挥
2. 进行魔鬼训练，提高身体的组织度

通过长期训练，使身体绝对服从指挥，像军队一样高效运作。大部分从事脑力劳动的人，无法让自己累晕，因为他们没有经过这样的训练。

## 逆袭的本质

逆袭的本质就是耍流氓，尤其对于没有天赋、背景和运气的人来说。他们需要像人形核电站一样，不断开牌，才能赢得机会。

## 总结

去掉等靠要的想法，能人所不能为，忍人所不能忍，这就是逆袭的三大要素。通过魔鬼训练，提高身体的组织度，才能实现逆袭。
