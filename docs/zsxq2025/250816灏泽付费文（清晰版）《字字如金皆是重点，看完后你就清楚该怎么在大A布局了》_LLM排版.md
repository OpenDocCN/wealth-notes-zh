
# 字字如金皆是重点，看完后你就清楚该怎么在大A布局了
# 250816 灏泽

今晚写了一篇非常简单粗暴的文章。
所有想通过投资，从而实现舒适人生的兄弟姐妹们，都应该花功夫读完。
因为字字句句，都是对于接下去整体形势的预判，以及灏泽个人最肺腑的意见。

1—在一切开始之前，灏泽希望你把下面这段总结，给反复读 10 遍以上：无论是做生意还是做投资，最容易亏钱甚至因此葬送人生的，统统都是一种人。
那就是明明自己完全没有足够的实战经验，也没有足够的胜利战绩作为支撑。
却依然盲目的认为，只要行情好，自己就能够浑水摸鱼在其中赚到一笔肥厚的时代红利。
这种人的本质，其实就是大脑空空的无知赌徒。
他完全不明白，在任何领域想要赚钱的根本基础，就是一定要懂、要钻、要熟。
只有掌握了这些基础，才有可能堪堪维持不亏，至于能不能赚到钱，那就是要取决于自己的运气、天赋，以及对整体局势的宏观理解了。

2—灏泽个人还有另外一个感悟，那就是要想在投资领域里赚到钱，你光光会看报表、K线、筹码分布都是毫无意义的。
就像一个只会炒菜的厨子，是永远不可能具备运营整个餐厅的能力。
别的区域我不敢说，但是江浙沪很多的金融大亨我都来往频繁，倒也不是摆谱，毕竟随便一个在金融领域混了一段时间的人，都认识不少大佬。
灏泽我的意思是，几乎所有能在投资、商业上做出好成绩的，统统都具备对历史、宏观、国际趋势，甚至军事、外贸等领域的深度了解。
很多人会以为，大A是一个很独立的投资市场，可事实上它和上述所有领域，都有着密不可分的关系，并且时刻互相关联。

3—贪婪之心是很恐怖的东西，我相信这段时间有很多兄弟姐妹会去关注大A，就是因为身边有人大呼小叫的号称自己赚到钱了。
人类就是一种很容易被身边环境所影响的动物，这是本能。
但灏泽还是希望你明白：其实这个社会分分钟钟都有人因为各种各样的方式原因发了财，但最终能够守住财富，并过上舒坦日子的一定一定一定是“能力值=财富值”两者匹配的人。
而通过运气搏到的财富，一定一定一定会最终形成巨大的反噬。
诚然，身边人赚到钱了但自己错过了机会，会因此心生懊悔和贪念。
可越是这个时候，你越是要静下心问自己“我目前具备了稳定盈利的能力么？我的经验、专业、性情是否能让我在风雨飘渺的大A里站稳脚跟？”
放心，你能压制住欲望，理性思考灏泽和你说的这番话，那就足以证明你是有慢慢成长，并最终创造大财富的天赋的。

4—永远不要迷信那些所谓的专业老师和内幕消息，灏泽我要想问到一些所谓内幕消息，那是分分钟钟的事，但是这也毫无意义。
因为“所有你能知道的内幕消息，早就已经被转手加工了无数次”。
而真正值得参考的内幕消息，只有两个人亲口面对面告诉你的，才算数一个是个企业的董事长一个是企业的市值管理负责人。
但就算是他们亲口告诉你，你也要思考一下“他们告诉我，是不是想要借我之口，去引诱其他人上钩？”至于其他所谓的你平时认识的啥好大哥好大姐，那些就真的别当回事儿了，都是吃饭喝酒吹水聊天的主。

5—这么些年以来，我非常反对身边的兄弟姐妹们用“炒股”这两个字，来形容自己的行为。
这不是我小题大做，而是用什么言语形容一件事，会潜移默化的影响你对一件事的理解。
“炒股”这两个字最大的问题，就在于太过于投机，也太过于让人追逐短期利益，这是非常危险的一种行为。
因为这会把你拖入“不停投机、不停炒股、不停在盈利/亏损后，再不停的找新股投机博弈”的恶行循环中。
恕我直言，我就没见过有谁是通过这样的方式赚到大钱的。
而灏泽在这里能说自己没见过，那就意味着真实的顶尖投资圈里，也几乎没有这样的人。

6—事实上，这么多年以来，真正能赚到大钱的人，最终还是遵循了一条亘古未变的核心：具备高耸护城河，并且占据了某个行业半壁江山甚至更多，同时核心管理团队依然在兢兢业业经营的企业。
像这种企业，永远都是非常非常好的存在。
且普遍都会在大盘低迷的时候，一边呈现出不错的股息回报另一边呈现出很快触底走稳的趋势。
而灏泽我在2024年底、2025年初就一直鼓励你要多关注这些企业了，至于你听没听，那就是你自己的缘分和悟性决定的了。

7—我个人对这一轮行情的理解，算得上比较小乐观。
之前也说过，所有大行情的出现，往往是伴随着其他领域不太顺畅为特征。
因为当越来越少的领域能留住水，那么水就会自然往有限的几个领域汇聚，比如银行储蓄、大A。
目前来看，前者已经达到了一个很大的数值，那么接下去就是慢慢溢向大A的过程，这就是很简单的逻辑。
那么什么时候，是行情的结束？
同样的，就是当你身边极高比例的人，开始表示自己在大A赚到了钱，同时高度关注某些名不经传的企业并号称自己有内幕消息时，这时候你就要谨慎了。

你要永远记住，这玩意有人赢就有人亏，而当很多人都小赢的时候，那...。

8—一定要坚定的认真研究你的票，不要随便别人一推荐，或者听身边朋友鼓动两句，你就按下筹码，这是最笨最懒也最容易闯祸的行为。
我个人的主力仓位，永远就是3—5支票极限了，而且我之所以会选择它们的前提是我对其了解的程度，已经超过了我对自己家楼下超市的熟悉程度。
换而言之，其报表、业绩、股东构成、高管背景这些公开的资料，我都烂熟于心，同时其整体的经营方案和状态，我也会从各方渠道去了解关注。
就很多人完全不明白“每一年就专注一两支票就行，超过了，不仅做不好而且只会浪费海量精力”这一事实。

9—最后，再推荐一些我认为必看的书籍吧。
不光光是有利于投资本身，同时对你的世界观形成也大裨益。
永远记住，你的投资盈利能力，就是你的综合内功的直接体现，而不是运气。
下面这些书，极佳：《全球通史》、《自私的基因》、《枪炮，钢铁与病菌》、《从0到1》、《彼得林奇的成功投资》、《巴菲特《致股东的信》》、《雪球特别版：段永平投资问答录》、《穷查理宝典》、《聪明的投资者》、《证券分析》、《乌合之众》。
你如果能把这些书，都认真的吃透，那么在常规的投资领域基本就能出山了。
