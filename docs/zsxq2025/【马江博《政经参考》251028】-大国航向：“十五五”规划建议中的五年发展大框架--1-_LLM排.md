
# 大国航向：“十五五”规划建议中的五年发展大框架

月 日到 日，二十届四中全会召开，审议通过了关于“十五五”规划的建议。“十五五”规划，是中国 年的国家发展规划，也是我们制定的第 个五年规划，它是未来五年，关于经济和社会发展最重要的文件，没有之。

我理解接下来的五年，有特别的节点意义：首先，它是中国 年基本实现社会主义现代化这个目标中承上启下的关键五年；其次，它是 在中美大国博弈进入战略相持阶段后的关键五年；最后，它更是中国经济实现新旧动能转换的关键五年。

而看完四中全会公报，我认为“十五五”的开启，将标志着中国进入 次加速的大转型、大升级。这背后，是未来五到十年发展和财富分配逻辑的巨 大重构，它将重新定义什么是 有前景 的行业，什么是 有价值 的资产。因此，理解“十五五”，才能理解中国未来五年的核心发展框架和动能。

这里需要先说明 个背景，目前公布的四中全会公报，只是“十五五”规划建议的缩略版，它给出的是方向、 原则和框架。按照惯例，《建议》的全文将在本周公布，里面包含各个领域详细的任务和举措，也蕴含着我们要发现的机会。

后续内容，我会在“十五五”规划建议稿全文公布后，再带你详细研读。好，今天的内容就到这里。
