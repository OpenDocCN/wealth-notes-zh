
# 美国航母军机南海坠毁，罪魁祸首是谁？

10月27日，很多人被美国CVN-68尼米兹号航母坠机的消息刷屏。这艘参与过海湾战争和阿富汗战争、刚完成中东任务的百战之师，为何坠机如此惨烈？短短半小时内，MH-60R“海鹰”直升机和F/A-18F“超级大黄蜂”战斗机接连坠毁。再联系到特朗普正在马来西亚参加东盟峰会，以及即将举行的中美元首会晤，此事颇引人深思。

美国军机坠毁的原因究竟是什么？

## 尼米兹号的任务背景

尼米兹号今年4月从美国布雷默顿母港启航，在中东地区执行打击也门胡塞武装的行动，10月初结束任务准备返回。这趟任务也是尼米兹号的谢幕之旅，计划返回基地后拆除关键设备，预计明年1月正式退役。然而，尼米兹号刚到南海就遭遇了坠机事故，实在令人尴尬。

## 坠机事件的疑点

坠机事件引发了很多猜测，主要原因在于坠机次序和原因都难以解释。

MH-60R直升机通常用于搜救，一般不会轻易坠机。而这次坠机事件中，MH-60R先坠毁，F/A-18F后坠毁，这让人难以理解。通常情况下，直升机坠毁后，不会立即起飞战斗机进行救援。

F/A-18F的坠机原因也值得怀疑。一般来说，航母舰载机在正常飞行阶段是安全的，除非发生特殊情况。这次坠机事件中，F/A-18F很可能是在起飞阶段出的事，要么是飞行员操作失误，要么是技术原因导致弹射器故障。

## 坠机原因分析

关于坠机原因，有多种猜测，但最可能的原因是美国海军整体老化的必然结果。

尼米兹号航母服役已有50年，早已老化严重。美国航母的维修分为两种：换料大修（RCOH）和维护、修理和大修（MRO）。尼米兹号虽然进行了MRO，但无法进行彻底的升级换代，也无法缓解舰体的金属疲劳。

美国能进行航母入坞维修的船坞只有四个，目前都排满了其他任务，导致尼米兹号无法及时进行维修。弹射器是航母的关键设备，如果维护保养不及时，很容易出现故障。

MH-60R直升机和F/A-18F战斗机也老化严重，近年来坠机事件频发。美军舰载机的事故率居高不下，这也反映了美军装备的老化和维护问题。

## 舰员的疲劳和压力

尼米兹号航母舰员已经出海超过7个月，超过了通常不超过6个月的规定，舰员早就人困马乏，事故发生的风险自然上升。此时，尼米兹号接到命令，不回家直接去南海，给特朗普总统壮声势。军事要为政治服务，但这也给舰员带来了巨大的压力。

美军工资停发，很多美军及其家属生活困难，这种情况下，美军舰员的思想和工作状态难免受到影响。

## 制造业空心化的恶果

美军各种奇葩事故层出不穷，潜艇撞山、战斗机相撞、直升机撞客机，这些事故都反映了美军装备的老化和维护问题。制造业空心化导致军工产业供应链断档，零部件断供，工匠退休，人才队伍断档，最终导致新装备难产，老装备带病工作。

美军频繁向中国炫耀武力，导致官兵服役强度和精神压力大幅度升高，操作失误碰上装备隐患，事故也就不奇怪了。

也许，这就是美军的新常态，也是一个日暮帝国的新的常态，时间长了，美军会习惯，我们也会习惯的。
