
# 底层破局

## 你没必要让太多人走进你的生命

你们都知道，我是很喜欢断舍离的，每过一段时间我就会清理掉那些我不需要的人事物，这件事儿给了我很多能量。因为我想一直保持自己的生活简单轻松，为什么年轻人不要太早谈恋爱，而是要好好搞钱，因为在你没有足够的资本和成熟度去面对这段复杂关系的时候，很可能带给你的痛苦和枷锁大过于快乐。很多傻狗之所以到30岁之后觉得人生无望很大概率都是因为年轻时候因为各种琐事和关系搞得焦头烂额，没有时间来让自己成长。所以我是不建议，年轻人太早就去享受生活，去操逼或者做一些太大的安排，尤其是穷鬼家庭，没有丰厚家底的人，不要太早在这些贱事儿上浪费太多时间。比如很多人一生不是被毁在自己没本事，没工作，而是是毁在年轻时候找了一个垃圾人，过了十几年垃圾日子，本来好好一个精神小伙，结婚之后，变得无精打采，能量完全被扰乱，再也没有时间精力去成长了。到了中年四五十岁再也没有心思和兴趣去成长了，一辈子的高度也就停下来了，再想突破，也是比登天还难。我想我之所以今天能够在自己生命的各个方面都有所建树，比同龄人要成熟一大截，跟我年轻时候没什么干扰，专心做自己的事情，专注于打造自己有关系。那句话咋说来着，你不要总是喜欢去追一匹马，你得用追马地时间来种草，遮阳等到来年芳草遍地时，会有一群骏马任你挑选。这个问题也是很多朋友都会遇到的，也是经常有人来问我的，人生在世，是应该现处理外面的各种人事物，各种琐事，还是先处理好自己的生活。举个例子，前天有个群友私信跟我说，最近去香港旅游了一趟，遇到了很多朋友，本来这是好事情，但是没想到，这么多年自己没有交新朋友，感觉遇到新的朋友还是有点不舒服，没有想象的那种轻松愉悦。这下好了，本来新朋友是好事儿，还可以谈合作，现在因为自己没有时间和心情去处理，朋友这种关系，现在变成了负担和沉重的包袱。人生就是如此，所有你没有能力和时间去处理的，都会成为你生命的包袱，所以最简单的处理方法就是，你只是去拥有自己需要的关系，安排自己能够应付的外在事物，如此，这些外在事物就是来支持你的，滋养你的，而不会过度消耗你。要知道，人的精力是很有限的，但这个物质世界是如此广阔，你不可能让每个人都成为你的朋友，你不可能把每件事儿都做好，你必须有所选择。你必须只做自己力所能及的事儿，而不是想当然地去做事儿。一件事儿是消耗你，还是加持你，就看是你真的需要，还是你只是假装需要。啥意思，你们看，很多人谈恋爱就不是因为自己真的需要这段关系，而只是因为看到别人谈恋爱好像挺开心的，自己也没有尝试过，于是乎，就匆匆忙忙开始了一段恋情，后来发现，根本不是自己想的那么回事。你以为恋爱的时候，对方会时时刻刻为你着想，不成想，恋爱之后，对方也有很多需求被满足，你平时自己的时间都不够用，更不要说去满足对方的需求了。你以为恋爱之后就只有甜蜜，什么事情都没有了，实际上你错了，谈恋爱如果要时刻保持甜蜜，需要一定程度的觉知和经营，你必须小心谨慎对待这段关系，好好经营，才能保持长久新鲜，否则很容易破裂。这些事情，都是你没有预料到的，最后你以为很甜的恋爱，让你焦头烂额。是因为你自己不好，或者对方不够好吗？其实都不是，只不过是因为，你并不是真的需要这段关系罢了。为什么你会觉得这是负担，我们来看看，负担这个词什么意思，就是说，你有一些，你压根不需要的东西，让你觉得自己白白付出了这么多，却一点儿收获都没有，如果你付出了一些东西，收获了更多东西，你不会觉得是负担。就是因为你付出了很多，但好像没什么回报，所以变成了负担。感情为什么变成了负担，是因为你其实压根不需要那么多感情和关系，但是你却以为自己需要，白白占用了自己大量时间。殊不知，并不是每个人都需要结婚，也不是每个人都需要恋爱，更没有哪条法律规定了人必须有很多人际关系。在这个世界上活着，如果想让你的生活不至于成为负担，你必须清晰地选择，去审视自己到底要什么，去拥有自己绝对需要的东西。有个原则是你可以去遵守的，如果一件东西，一个人，一段关系，是抛弃之后，你也可以活的很好的，那么就说明，这件事儿，完全没有必要。换句话说，如果你不是不谈恋爱就茶不思饭不香，那你你就不要随便恋爱。如果你不是不操逼就浑身难受，那你也没有必要找女朋友，花钱就可以解决的事儿，不必花这么多心力让自己搞得这么累。就像吃饭一样，你咋吃的舒服，肚子要舒服，其实也简单，如果后面的东西你不吃，也不至于很饿，那就不吃了。吃下多余的东西，肚子也不好受，肚子要消化食物，得花老大的力气。就是如此，总之，要想活的轻松，简单，自由，简化自己生活最简单的方式，就是不去安排自己不需要的事情，只做自己需要，自己能力能够处理的。其次，遇到任何事儿，先去处理自己的情绪以及提升自己的能力，再去看看怎么处理事情。我是从来不关心自己生活中的琐事儿的，我觉得我的人生里没有什么问题，琐事儿肯定有，但是不会浪费我太多时间和心力，因为不在我心里装着。我的眼睛永远都盯着怎么让自己快乐，开心，喜悦，幸福，怎么让自己的能力和天赋发挥到极致，我总是创造一种氛围，让一切问题都迎刃而解。蠢货才喜欢把时间花在处理外面事情上面，而智者总是从高纬度转化自己，从而让大问题变成小问题，最后变成没问题。在自己足够强大之前，如果你想保持自由的心态，那么就没有必要让太多人走进你的生命，进入你的生活。我们总会有个误解，认为朋友肯定越多越好，错了，这个世界没有人能够陪伴你一辈子，人生除了三五真心好友，陪着你傻乐呵，其他的就当个过客，走个过场，自己强大起来，比什么都重要。
