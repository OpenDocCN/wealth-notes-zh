
# 2025泡泡玛特POP+MART品牌手册

## 品牌文化

品牌秉持“创造潮流，传递美好”的品牌文化，致力于为消费者带来独特的潮流文化体验。

## 品牌发展历程

泡泡玛特成立于2010年，由创始人王宁在北京创立，最初以潮流百货杂货店起步。2020年12月于港交所上市，截至2025年6月，市值逼近3500亿港元，创始人王宁以208亿美元身家跻身福布斯中国富豪榜前十。

这五个节点分别代表了泡泡玛特从模式探索、渠道创新、资本运作、生态扩展到全球化的关键跨越，共同构建了其“IP工业化”的核心竞争力。

### Essential Nodal

- **2016年**：IP驱动战略的关键转折点，签约香港设计师王信明，获得MOLLY独家版权并推出“Molly星座盲盒”，标志着品牌从代理模式转向自主IP开发。
- **2017年**：推出自助机器人商店，以低成本、灵活性的方式快速铺开线下渠道，截至年底覆盖全国，成为其规模化扩张的核心引擎。
- **2020年**：港股上市与行业话语权的确立，泡泡玛特成为“潮玩第一股”，市值一度突破千亿港元。上市不仅为品牌注入资本，确立了其在行业内的引领地位。
- **2023年**：潮玩+主题乐园的全球化启程，北京泡泡玛特城市乐园开业，首推“潮玩IP+主题乐园”模式，推动全球化战略落地。
- **2024年**：全球化爆发与IP护城河强化，海外收入达50.7亿元，LABUBU等IP在海外市场实现破圈，例如泰国曼谷首店单日销售额破千万，北美市场LABUBU跻身销量前三。

## 品牌创始人

创始人王宁（1987年生）受家庭商业氛围影响，大学期间创业积累经验，2010年带20万元在北京创立泡泡玛特。初期遇挫，2015年敏锐捕捉潮玩机遇，转型后签下MOLLY独家IP，推出盲盒模式获成功。他坚持“极简”领导力，构建三层IP护城河，推动公司全产业链发展。2020年泡泡玛特港交所上市，市值破千亿港元。截至2024年，他以48.73%持股，成为河南新首富，引领中国潮玩文化走向全球。

## 品牌符号系统

品牌符号系统：标识

LOGO设计呈现水平矩形，字母间距均匀，整体对称，形成强烈的“符号感”——如同“潮玩行业的视觉印章”，使消费者能够快速记忆和识别。矩形布局具有高度适应性，无论是大型线下光箱、产品盒标签，还是小型在线图标，都能保持视觉完整性，强化品牌的统一形象。

## 结语

泡泡玛特通过盲盒经济打破圈层，利用非遗产文化的文化赋能，通过全球化和本地化战略，在48个市场落地，海外收入占比近50%。2024年市值超过1488亿港元，LABUBU等IP成为现象级文化符号，重塑了“中国设计”的全球话语。
