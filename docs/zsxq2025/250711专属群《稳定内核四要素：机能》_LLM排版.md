
# 稳定内核四要素：机能

机能，不只是能力，也包括身体健康、心理稳定、节奏适配等等一切让你能正常运作的条件。换成大俗话，就是我们的身体或者组织应当能够健康运行，跟得上我们的人生需求。

怎么让机能要素保持稳定？按照最小系统原理，我们仍然可以追问那些基本问题：
- 目标：什么水平的身体或组织机能，足够支撑我的正常生活、事业运转？
- 范式：为此我应该把身体或组织，拆成哪几个子系统，来制定机能维护方案？
- 耗散结构：我怎样去平衡自己的摄入与支出？

我们不打算共创一份养生方案，放进你的收藏夹里吃灰。我们讨论一个认识自己的陌生视角，希望重构你对机能管理这件事的认知，有动力把收藏夹里那些养生笔记用起来。

### 站在AI的视角看人类

如果人类可以把AGI看作一种有自我意识的硅基“人类”，那么站在AGI的角度，也可以把人类看作一种有自我意识的碳基硬件。

这个想法并不新鲜。如果我们真的拥有一台智能精密设备，即使它再木讷，我们也会精心养护它。因为站在客观视角，我们能够理性地认识到，这台设备价值重大、不可再生，是我们最重要的负熵来源。

### 把自己看作碳基硬件

把人看作碳基硬件，我们得到的第一个提醒是：应当把自己的意识和肉身区分成软件和硬件，在心态上，重视对肉身这台贵重硬件的大修和保养。

#### 工程化休息：杀程序

人类需要的种种休息，相当于硬件需要的关机或休眠。一台计算机在长期高负荷运行后容易变慢、死机。类似的，一个人长期疲劳工作的表现也会变差。它们的性能都下降了。今天人们焦虑和抑郁高发，我们也可以理解成，碳基硬件在环境要求下超负荷运转，导致能量不足或能量卡顿，切到了低能耗档位。

#### 感官系统=边缘算力

我们的感受和直觉，也是碳基硬件算出的高价值数据。只不过它不是来自头脑这个理性算力系统，于是往往被我们轻视。

把人看作碳基硬件，我们可以认为，大脑就是我们的CPU，而我们的感官系统就类似于边缘计算系统。我们的皮肤、舌头、鼻子、耳朵，也都是拥有算力的元器件。

#### 饮食质量=能源纯度

把人看作碳基硬件，我们的饮食的健康程度，等于我们摄入能源的纯度。人的呼吸系统和消化系统，就是这台硬件的动力转化系统。

这就相当于硅基硬件的电源管理模块，把外部能源转化为内部可用的能量。那么，锂电池电解液的杂质多少，是不是会影响电池的性能？天然气中的杂质多少，是不是会影响管道寿命？药品中的杂质多少，是不是会影响药效？

从这个角度看，我们进食就像给机器加油。重油重盐就像燃料掺多了杂质，会堵塞“动力模块”；暴饮暴食就是让发动机超载。

### 总结

人跟智能硬件的相似点还可以有很多，比如，生成式AI的预训练方式，跟人类的教育模式是相通的；人的神经系统，可以对应硬件的通信与控制系统；人的内分泌系统，可以对应信号与调节系统，等等等等。

总之，把自己看作唯一的碳基智能硬件，能帮我们完成一次小小的自我超越，把意识抽离出来，客观看待“身体”这个碳基硬件系统，理解并尊重它的运行规律。

在这个视角下，我们会更容易铁石心肠地对待自己，把收藏夹里那些养生、运动和休息教程用起来，给自己设定纪律，逼自己执行，来让我们的身体机能保持在某个稳态。
