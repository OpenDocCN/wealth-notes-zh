
# 卢克文星球《半年赚了12亿美元，特朗普家族是怎么捞钱的？》

## 最近，特朗普的儿子巴伦·特朗普陷入了一场质疑风波。

事情的起因，就是10月10日早上，特朗普放出来的对中国加征100%关税的威胁。

虽然这个关税没有真的加，但股市和币市的崩盘却是真真切切发生了的，特别是虚拟币，24小时内超过191亿美元的杠杆头寸被清算，总共有超过162万人被强制平仓，最大单笔爆仓单超2亿美元，无数人血本无归。

整个币圈都怒了，把矛头指向了特朗普的小儿子巴伦，是你串通老爸做空了虚拟币，还我们的钱来。

那么，这到底是咋回事？虚拟币爆仓又和巴伦有什么关系呢？

### 1 事情要从10月10日的早晨说起

就在特朗普宣布对华加关税前半小时，一个神秘的账户在虚拟币交易所Hyperliquid上，建立了巨额的比特币和以太坊空头头寸。

然后呢？特朗普关税威胁消息一出，市场应声崩盘，而这个账户在价格反弹前迅速平仓并提现1.92亿美元，完美离场。这次做空事件，对Hyperliquid平台上的其他用户来说，不亚于天塌了。

超过1000个账户被彻底清零，6300多个账户亏损，累计损失12.3亿美元。

有币圈内部人士透露说，“最后一笔空头订单在美东时间15:49下达，恰好在特朗普于15:50威胁征收新关税前一分钟。真是令人难以置信的运气。”

然而，事情还没完，就在10月12日，这个账户又投入1600万美元开设10倍杠杆的比特币空头头寸，引发了币圈的再度恐慌。

咋了？这位爷又得到内幕消息了？总是这么玩谁还受得了？

对这种内幕交易的猜测，著名的币圈大佬、币安创始人赵长鹏也站出来说，希望有人能够交叉核实。不核实不行啊，这种内幕交易简直是平台毒药，他要是去币安玩一圈，恐怕就再也没人敢玩币安了。

动了这么多人的蛋糕，大家肯定不会善罢甘休。

一个叫做Eye的虚拟币侦探，对这件事进行了持续追踪和研究。

Eye经过分析这个神秘玩家的交易账户，发现这个账户收到了特定钱包的手续费，而追踪这个特定钱包，发现资金流动和一个叫ereignis.eth地址有关系，而这个地址还有另一个以太坊域名：garrettjin.eth。

拆开garrettjin，不就是Garret Jin么？真相大白，这个人就是前加密币交易所BitForex创始人兼首席执行官加勒特·金！

从这个名字来看，加勒特·金大概率是个中国人，因为韩国人的“金”一般写作KIM，而只有中国人才会写作“Jin”。

从履历上看，似乎也能证实这一点，加勒特·金2008年从波士顿大学经济系毕业，然后到中国建设银行当实习生，不过最后没留下，2012年跑去香港开贸易公司，2015年进入李林和杜均创立的火币当运营总监，随后移居德国（他那个ereignis.eth地址就是德语），创立了医疗平台，但没过多久就不干了，学着赵长鹏创办了加密币交易所BitForex，自己当CEO。

这个交易所后来陷入了刷流量丑闻，又发生了私钥泄露事件，损失5700万美元，最后被香港证监会发出欺诈警告，被迫关闭。

看似加勒特·金一事无成，但因为他赶上了比特币的好时代，所以还是攒了不少家底，目前仍然仍持有46,295枚比特币，价值约51.9亿美元，这些钱足够他在币圈掀起巨浪了，所以被称为“巨鲸”。

虽然加勒特·金的身份确定了，但这并不能解释，他是怎么精准地踩中风口，以惊人的勇气和资金下达空单的？

经过虚拟币侦探Eye的进一步深扒，更有意思的发生了，这个加勒特·金只是一个白手套而已。

加勒特·金负责具体操作，而内幕消息，来自两个人：扎克·维特科夫和蔡斯·赫罗。

这俩是什么人？是WLFI的联合创始人！

还记得前几天跑到中东宣读特朗普的20点和平方案的美国中东特使史蒂夫·维特科夫吗？

这个扎克·维特科夫就是他的儿子。

而蔡斯·赫罗则是币圈资深人物，混迹币圈十几年，现在是特朗普家族的加密货币专家。

而WLFI是什么？是2024年由特朗普及其三个儿子（小唐纳德、埃里克、巴伦）共同成立的虚拟币公司，特朗普家族持有公司30%的股份，巴伦本人拥有其中约10%的股份。

现在真相大白了，做空虚拟币的，原来是特朗普家族。其中操盘手，就是特朗普的小儿子巴伦·特朗普。

很显然，巴伦负责从自己老爸那里得到消息，然后告诉扎克·维特科夫和蔡斯·赫罗，俩人研究做空方案，交给加勒特·金去执行，最后赚到的钱各自分账。

巧了，就在这次虚拟币灾难之后，巴伦订购了一艘价值数亿美元的超级游艇。

巴伦才19岁啊，虽然有个富豪老爹，但也没富到随便拿数亿美元出来买游艇的吧？这些钱是从哪来的？

不言而喻了。

### 2 说起来，巴伦的操作，已经算是正常了。

因为自己老爹、老妈、老哥、老姐都是这么干的，凭啥自己不能这么干呢？

相比他们，巴伦已经算是小巫见大巫了，事实上，不仅仅巴伦，整个特朗普家族，都是一个赚钱机器。

想当年，特朗普第一次赢得总统选举时，很多法律界人士都在争论，当富豪当了总统，还能不能保留其商业利益？

你总不能一边经商，一边当总统吧？

当时特朗普还比较收敛，不仅将相关商业活动交给儿子埃里克和小唐纳德负责，还承诺不开展新的海外交易，取消了一些尚在洽谈中的项目。

结果呢？国会山骚乱之后，特朗普的商业项目遭到了巨大打击，银行抽贷、社交媒体封号、很多合作项目也黄了。

对此，特朗普非常不满，抱怨说“我们主动给自己施加了这些限制，但结果呢？我们并没有因为这么做、因为做了正确的事得到任何认可。”

所以，这次特朗普上台后，再也不顾及那么多了，开启了捞钱之旅，而且每个家庭成员都没闲着。

而《福布斯》杂志的统计显示，6月份特朗普本人的净资产为51亿美元，创下了其历史最高纪录，比去年整整高出了12亿美元。

这些钱绝大部分都是他当了总统之后获得的，总额可能超过10亿美元。

要知道，总统挣10亿美元，其实不算稀奇，但问题在于，这是特朗普在上任三四个月赚取的，这就有点吓人了。

那么，特朗普是怎么赚到这么多钱的呢？

首先自然还是虚拟币。

咱们还记得，1月17日，就职典礼的前几天，特朗普推出加密货币$TRUMP，最初价值飙升，12小时内涨超15000%，在1月19日达到单价近75美元的峰值，总市值超过147.5亿美元。

然而没过几天，特朗普币就暴跌到了16美元元左右，特朗普狠狠隔了一把韭菜，初步估计特朗普至少获利3.5亿美元。

不仅特朗普这么玩，特朗普的妻子梅拉尼娅也不甘示弱，发行了加密货币$MELANIA，通过同样的手段在1月19日之前炒作市场，最后特朗普就职后的12个小时内大量交易，攫取红利，赚了超过1亿美元。

一同发币的还有特朗普的儿子们，反正都是狠狠赚了一波。

当然，这种吃相太难看的割韭菜行为不能总玩，总玩就没人信了，要想持续不断从虚拟币上挣钱，还是要老老实实控制上游，比如，获得独立的发币权。

为此，特朗普家族通过WLFI公司，推出了USD1稳定币和WLFI治理代币。

在大家都担心被割韭菜的情况下，怎么让大家都愿意去买特朗普家的币呢？

答案很简单，政府背书。
