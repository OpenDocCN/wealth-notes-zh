
# 特朗普上任在即，马斯克和红脖子为何内讧

## 文/卢克文工作室嘉宾 星海舰长

最近，美国陷入了一场关于“什么是检验MAGA唯一真理标准”的大论战，这场大论战爆发之突然，波及面之广，影响之大，都远超人们预想。

而一切的起因，还要从马斯克“政府效率部”的副手维韦克·加纳帕蒂·拉马斯瓦米说起。

听这名字，我们就知道这哥们是个印度裔。没错，他来自印度移民家庭，耶鲁法学院毕业后，拉马斯瓦米开了一家制药公司，主要是给印度老家的仿制药厂提供“贴牌”服务。后来拉马斯瓦米从政参与总统竞选，然后一看势头不对，马上转投懂王，莫名其妙地成了“从龙功臣”!开始和马斯克一起主掌成立的“政府效率部”(DOGE)。

可能是感觉春风得意，拉马斯瓦米也开始口无遮拦了起来。

2024年12月27日，拉马斯瓦米突然发了一条很长很长的推文，这条推文光在X上就有1.1亿次的阅读，在全美引发了剧烈的反响。

主要内容就一条：顶级科技公司经常聘用外国出生和第一代工程师，而不是“本土”美国人的原因，并不是因为美国人天生智商不足，而是美国文化太长时间以来崇尚平庸，而忽视了卓越。

你看看美国的学校里，崇尚的是舞会皇后、运动员，而非数学奥林匹克冠军!没有数理化，就没有科技进步啊!

美国人应该多看一些像《爆裂鼓手》这样的电影，少看《老友记》的重播。多看数学辅导，少过夜。

拉马斯瓦米想传达出来的意思就是，你们这么不思进取，怎么和那些崇尚天才、高考状元和科学家的中国人竞争啊!

拉马斯瓦米作为一个靠学习卷出来的移民后代（印度人卷学习一点也不亚于中国人），这么想是一点错也没有的。

但问题在于，美国人不这么想啊!

你一个喝牛尿玩牛粪的印度人，也配对我们美国人指手画脚?

企业为啥不雇佣白人?不是因为白人不行，是因为移民太多了!特别是像你这种印度人!你们印度人抢了美国人的工作，回过头来还说美国人不行?

虽然拉马斯瓦米是个土生土长的美国印度裔，但这不影响美国人把他当成印度人一样喷，美国都是被这些拿H-1B签证的印度人给搞坏了!

H1B签证其实就是美国的“英才引进计划”，从世界各国收割高学历英才为美国所用的，但现在，70%以上拿到H-1B签证的都是印度人，印度人有一条完整且成熟的F1（学生签证）→OPT（实习签证）→H1B（工作签证）→雇主提供PERM→申请移民(EB2/EB3)→等排期→拿绿卡的产业链。

所以，红脖子认为，H-1B签证有bug，太宽松了，需要收紧，比如把申请标准从6万美元，提高到50万美元!

这么搞基本上断了所有人拿H-1B签证之路。因为50万美元的薪水，放到美国也是高薪，能拿到这个级别薪水的，根本不用H-1B签证，直接走EB-1A就拿绿卡。

而真正需要H-1B签证的，大部分都是刚毕业的大学生，怎么可能有这么多工资?

这么不靠谱的事情，显然不行，所以马斯克也站出来说话了：

“美国之所以在最近一百五十年里非常成功，是因为它是一个精英统治的国家，没有H-1B工人，什么SpaceX火箭，什么特斯拉电动车，都是不可能做到那么成功的。和你们这帮红脖子有什么关系，你们就是阻碍美国再次伟大的绊脚石，应该滚蛋!”

假话不会伤人，真话才会。马斯克这么说，就相当于指着红脖子的鼻子骂了!

果然，红脖子觉得受到了奇耻大辱，开始疯狂网暴马斯克，还请了阵营内的大儒出来辩经。

红脖子大儒认为，你们说多元化、引进移民才有前途，可人家中国从来不是个“多元化”国家，人家也没开放移民，主体民族一直是汉族，结果人家又是搞六代机，又是航母，不也发展起来了?

还有，中国人也没让印度人当CEO，人家的公司不也搞得好好的?

说一千道一万，是你们这帮精英自己不行，反过来怪我们没文化?

马斯克你是不是想借着“政府效率部”裁员的机 会，把我们白人都裁掉，把工作岗位都让给牛尿玩牛粪的印度人?马斯克没安好心!

结果，马斯克还没说话，却把印度裔惹恼了。

你们吵归吵，我怎么躺着中枪了!我们觉得我们老乡拉马斯瓦米说的没错，你们美国人就行不行，就是垃圾!

结果，一大波印度人加入战场，在各个话题下与红脖子对喷。

就这样，美利坚打成了一锅粥。

注意，这还不是高潮，高潮的是，特朗普的表态。

12月28日，特朗普在接受《纽约时报》的采访时表示，“我一直喜欢这种签证，我一直支持这种签证，这就是我们拥有它们的原因。我的房产业务上用到很多H-1B签证，我一直是H-1B签证的信徒，我已经用过很多次了，这是一个伟大的计划。”

这一下子，红脖子更接受不了了!本来指望着特朗普给自己做主呢，万万没想到特朗普从背后给了一枪!

当年那个带领红脖子高喊“全世界的移民像毒蛇一样正在毒害我们国家人民的血液”的懂王去哪了?

怎么现在变得这么无情冷酷?

于是美国红脖子马上陷入了分裂，一派只骂马斯克蛊惑特朗普，另一派把特朗普和马斯克一起骂，还在网上狂刷特朗普跪舔马斯克的AI图。

巧了，前几天特朗普酒店门口的赛博皮卡爆炸案，就是个红脖子干的，很难说这是不是红脖子愤怒的一次集中爆发。

在红脖子和马斯克吵得不可开交的时候，刚刚失败的民主党支持者，却在一旁兴致勃勃地吃瓜，偶尔下场嘲讽一下红脖子们：你爱懂王，懂王爱你吗?

看着事情越闹越大，可能共和党方面也不愿意让敌人看笑话，最终还是让这次“大议礼”平息了。

怎么平息的呢?非常有马斯克的特色。

一方面，不管什么“言论自由”了，把参与论战的红脖子大儒们统统封禁。谁再撕扯这个事，谁就是不讲政治了!

另一方面，马斯克又发了个X，说“我非常清楚该计划有毛病，需要进行重大改革，我相信特朗普政府改变(H-1B制度）后，将使美国变得更加强大”。

简单来说，就是承认H-1B制度确有漏洞，政府应该修改标准。

马斯克这么说，无非也就是给红脖子一个台阶下，大家别闹了，懂王马上登基，大局为重啊!

那么，马斯克和红脖子，到底谁对呢?

这就要看站在谁的立场上了。

站在红脖子立场上，特朗普就是被蒙蔽的昏君，而马斯克就是那个蒙蔽皇帝的奸臣。

想理解红脖子，就要先了解红脖子。虽然红脖子这个称呼一般是贬义，但红脖子群体自有自己的骄傲。因为在某种意义上说，美国就是他们缔造的。

美国独立战争是怎么打赢的?不就是农村的红脖子自备干粮，自带枪支，拼命跟英国人死磕，才最终让美国独立的?

美国开国元勋杰斐逊曾说过：“土地的耕种者是最有价值的公民。他们最有活力、最独立、最正直；他们与国家一心同体，与国家的自由和利益之间有着最坚固长久的纽带。”

红脖子一直坚信，自己这个群体缔造了美国，也是最忠诚的爱国者。

假设有一天美国遭遇入侵，那些华尔街的富豪、精英、程序员、律师们，会拿起枪战斗吗?绝不会!拯救国家的重任，还是在红脖子身上!

红脖子有口头禅：我愿意为国家去死!

但是国家是怎么对待红脖子的呢?

全球化浪潮制造的产业转移，资本家不断把工厂关闭，然后搬到人工更便宜的亚洲，美国国内的岗位越来越少。而红脖子——这个国家最忠诚的国民没有工作，没有工资，小孩上不起私立学校，变成了被“落下”“遗忘”的人，牢牢钉死在了底层，毫无上升空间。

为什么特朗普能以一个政治素人的身份问鼎总统?就是他愿意给红脖子们说话，他要“把产业链带回来!限制移民!让美国再次伟大!”他重新燃起了红脖子们的希望，给红脖子们带来更多的工作机会和薪资选择!

所以红脖子才成为特朗普最忠实的部下。

但站在马斯克的立场上，情况却完全是另一回事。

马斯克是个理工男，他要搞火箭，让人类移民火星。但问题在于，如果把H1B签证都禁了，把岗位都给红脖子，靠红脖子这个四则运算都搞不明白的群体，火箭怎么能射上天?

现在的这一批红脖子，大部分都是在公立学校和州大学接受的“快乐教育”，人嘛，总是有惰性的，公立学校课业轻松，老师也不逼着你学，就这样日复一日地混日子，结果把人都混废了。这样的人，去造火箭，这不是开玩笑么?

为啥波音招了那么多印度裔来干活?不就是因为招不到合格的白人么?

合着你天天自诩勤劳忠诚勇敢，却啥也不想付出，不想努力不想读书，整天飞叶子喝大酒吹牛乱搞，就想拿到好工作，一栋房子一辆车，两个孩子一条狗?凭什么?就凭你这张白皮么?

但红脖子一看，我们红脖子打下的江山，凭什么你马斯克一个移民出来指手画脚?还想放进来更多移民?怼!

于是，一场史诗级的大论战，就这样爆发了。

说白了，这场大论战，本质上是“我祖祖辈辈都是美国人”群体，和“我为美国立过功”群体，为了争夺话语权而进行的PK罢了。

巧了，这样的PK，在中国也有一次。

清朝咸丰四年(1854年)，从广西发源的太平军已经彻底占领了江南、江北等地，势力一时无两。而与此同时，洋人那边也不消停，开始嚷嚷着修改旧约，否则就以武力相向。

偌大一个大清国，陷入了风雨飘摇。

如何才能救大清?

备受成丰帝信任的肃顺提了一个看法：重用汉臣!

为啥这么说呢?虽然清朝建立后一直在提倡满汉同治，但骨子里还是提防汉人的，比如北京城的九门提督、领侍卫内大臣都要是旗人，六部满汉双官制，但印章什么的都是旗人保管，各地的镇守将军，也以旗人为主。

但是到了清后期，旗人群体朽坏，不堪大用，要想救大清，就必须利用汉人精英!

于是，对这个问题，肃顺看得很清楚，他曾反复说过：“满人糊涂不通，不能为国家出力，惟只要钱耳。国家遇有大疑难事，非重用汉人不可。咱们旗人浑蛋多，懂得什么?”

肃顺绝对不是说说而已，曾国藩、左宗棠、胡林翼、郭嵩焘等汉臣，都是肃顺举荐的。为了解决清政府财政困难，肃顺甚至还建议革除八旗子弟俸饷例银!砍倒铁杆庄稼!

断人财路如杀人父母，肃顺这么玩，自然引起了八旗子弟对于肃顺的痛恨。

而慈禧和奕正利用了这种痛恨，在咸丰帝死后，成功发动了政变，结束了肃顺的时代。

1861年11月，大清第一权臣肃顺，被斩首于北京菜市口，享年45岁。肃顺成为大清历史上，第一个被斩首的顾命大臣，下场比鳌拜还惨。

对于肃顺的死，旗人人人拍手称快，哪怕斩首后，诸多旗人蜂拥而上，争食肃顺之肉。

而汉臣则捶胸顿足，曾国藩感叹：“此冤狱也，自坏长城矣!”

历史的滚滚车轮下，肃顺奋力挣扎，试图挽救大清，却终究沦为权力的牺牲品。
