
# 金融三剑客：中国资本市场-中信(三十)

## 一、审视

其实你去全世界任何国家，潜规则都是交朋友和搞小圈子。那怎么判断你目前积累的人脉的成色呢？很简单，你随手发条朋友圈就知道了。如果你对别人有人脉价值，不管你随手发点啥，都会涌出一群人来给你点个赞，互动一下什么的(俗称尬聊)。如果你对别人没价值，其实很多人已经把你的朋友圈设置为不看，如果你再发点广告啥的，可能直接就拉黑了，你发再多都白搭。

以上种种迹象都告诉你一个很冷峻的现实，由于你过往没有积累出有效人脉价值，你的朋友圈里已经几乎没有朋友，只有网友。这个景象，你换位思考，站在对方的角度想一想就明白。

如果各位还想了解那些权力掌控者的其它想法，方法很简单，你从权力的三大特性：唯一性，畅通性，扩张性，就能够推导出几乎所有的权力理论。说白了，世面上所有的权谋理论都是通过这样的基本原則给推理出来的各种定律和法则。

## 二、削权

好，现在你懂了权力的这个框架，咱们再说回绍伊古的这个安全会议秘书的职能，它主要就是召集安全会议，并为总统提供安全事务咨询和文件起草。说白了，秘书的权力来自上级领导，自身并没有多少决定权，而之前的国防部长在许多事务上都是有决定权的。你这么一对比，就知道两者的内涵差得远了。

至于另一个职务，军工委员会的职能则包括重大国防计划的决策、监督军工生产，另外每年还可以采购150亿美元左右的装备。很明显，这个机构的权力和油水都很大，但是，不会有多少落到绍伊古头上的，因为他只是副主席(副主席也有好几个),也不是第一副主席(现任第一副主席是梅德韦杰夫),下面的委员会成员也都是各部的部长、副总理之类的(都是拥有实权的主)。说白了，绍伊古实际上等于被上下实权人物夹在其中，并没有多大优势。

## 三、原因

讲到这里，咱们可以简单总结下，普京拿掉他国防部长的主要原因：

一是军改失败。绍伊古在2012年上任后便主持了军队改革，意图通过营级合成化改革，打造一支"少而精"的俄军。俄军军改后的营级合成化单位，名叫 BTG(营级战斗群)。一个BTG大约1200人，有步兵，有炮兵，有装甲部队，有通讯部队，有无人机分队，原先俄军在营一级的兵种都是单一的，现在将不同兵种合成到BTG 后，一个BTG 的战斗力可以达到过去两个团的水平。

理论上听着是没错，无论是我军还是美军，军改走的是这个路子。但理论和实战还是有重大区别的。俄乌战争后，BTG 就暴露了两个严重的问题：步兵数量少导致无法近战以及符合合成化要求的装备和人才相当匮乏，难以做到真正的合成(实际1+1<2)。

话说前者的问题(步兵少)还可以通过扩军解决，而后者就没那么好办了。合成化装备和人才需要"体系化的支撑",背后是国力问题(要持续的资源投入)，这就不是绍伊古一个人所能解决的了。

公正地讲，虽然存在各种客观因素，但绍伊古的责任肯定是不可推脱的。比如腐败问题，绍伊古捞了不少也是众所周知，本来资源就稀缺，他和亲信们还在其中大捞特捞，资源就更少了，这也严重拖累了普京的军改进程和最终实战的效果。

话说作为一线的操盘者，绍伊古对俄军的问题是很清楚的，那他有没有如实向普京汇报呢？并没有。于是这就牵出了普京要拿掉他的第二个原因：蒙蔽上级。

在绍伊古年复一年给普京的军改报告里，军改的效果那都是相当成功的，都是说俄军已经被他变革的非常强大了，收拾乌军压根不存在问题。普京并不在一线，很多信息是拿不到一手的，而绍伊古等人给普京汇报的信息，显然也会抹掉对自己不利的部分。只要不发生大规模战争，哪怕普京发现了问题也没法实际去验证。不过纸是包不住火的，现在全世界都看出来了，普自 nhiên也不傻，继续留着他过年吗？显然是不可能的。

第三个原因就是瓦格纳兵变。瓦格纳兵变的直接催化剂，就是2023年5月的时候，国防部要强行收编瓦格纳，夺取兵权，这件事导致直接逼反了普里戈津。

我们知道，普里戈津是普京的人，瓦格纳也是他的私兵，瓦格纳集团因为兵变被肢解，普里戈津也死得不明不白，等于普京的力量被削弱，绍伊古则成了背后的最大受益者。

## 四、人精

讲完这个现实案例的逻辑，咱们再回来看，现在刘副用朱可夫来比喻彭老总，这等于是把皮球踢给了教员，啥意思呢？逼迫教员要像赫鲁晓夫那样"杀伐果断。

说到这个朱可夫，读者可能对他不了解，那可是个人精中的人精。为何这样说呢？他向上管理的技术已经炉火纯青。在这里剑客可以给各位读者把他的特点稍微展开一下讲，你会理解更深。

通常来说，领导人对下属的欣赏和对儿子的欣赏，那是不一样的。你比如在东北老帅眼里，杨宇霆是不是青年才俊？当然是。但是老帅对杨宇霆的欣赏，和对他亲儿子六子的欣赏，是不一样的。

好比说你买了一个键盘，你对它抱有什么样的期望？你按N，它就弹N，你按M，它就弹M，这就是基本期望。如果你按N，它弹M，你按M，它弹N，键盘就是个废物，明白不？它在挑战你的底线，是可忍孰不可忍。

那么如果你按错了呢？你本来想要按的是M，结果按错了，按了N，5 它也弹起N，它的确符合了你的基本期望，但是，它在你心中，大约也就是个庸才了。即苦劳它有的(听话照做了)，但不会有功劳(没能输出正确的结果)

那一个好的键盘到底要怎么做呢？难道要你按N，它预判你本来想要按的是M，直接替你弹出M，给你个惊喜么？

是惊喜不假，但惊喜过后就是惊吓。卫青私自调兵救驾，武帝感动之后，就开始针对他削兵权。感动是感动于他的人才，他的变通，他的知音，他的忠诚，针对他削兵权又是最理智的决策。

为何这样做？因为我今天按了N，你可以说是预判我按错了，弹出M，明天呢？会不会打着我按错了的旗号，自行其是？

亮剑里孔捷多次说过，李云龙被罚得不冤，不听号令本身就是最大的威胁(自行其是)。如果不是战争年代(现在是什么年代?)，而且是情非得已，人才难得，他这样的人，是不会被重用的。即便迫不得已要用，临时用后也会迅速被雪藏。
