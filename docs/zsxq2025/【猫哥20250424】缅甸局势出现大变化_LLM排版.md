
# 缅甸局势出现大变化

近期，缅甸局势出现了一个重大变化，缅北重镇腊戌被正式交换给军政府。整件事是中国主持的，为此，中方向腊戌周边派出了数十辆停火监督车，迫使同盟军同意。同盟军好不容易打下来的地盘，为什么要还？其实，这是中缅交易的一部分。

年初我们的文章说过，敏昂莱在访问昆明后，和中方达成了一系列交易，其中最重要的，就是允许中方的安保人员进入缅甸。作为交换，中方会帮敏昂莱拿回腊戌。

你别说，敏昂莱这一次确实没耍小性子，回去就麻溜地修改了《私营安保法案》，允许外国安保人员在缅甸开展特殊业务。目前可以确认，中方安保人员已进入若开邦，主要在皎漂港附近，具体人数不详。

毕竟同盟军控制着若开邦的伪政府资金供应，还有电力、通信、供水等基础设施，严重依赖中国。面对极限施压，他们没有多犹豫。网上甚至有传言称，我们把彭德仁给扣押了，不过这个大概率是谣言。

当然，腊戌是同盟军数十年来，所梦寐以求的地方，好不容易打下来了，与其勉强占着腊戌，把弦绷得太紧，不如松一点先收缩战线，也就不用强制征那么多兵了。可以腾出手来，消化把新占领的土地。反正腊戌周边在手里，腊戌市区是被卡着脖子的。消化占领区，壮大实力，同盟军才能完成转型，从一支山沟沟里的游击队，转变为地方诸侯。

正是这一点，导致军政府内部，有一些人非常不满。对敏昂莱而言，能拿回腊戌市区，面子上不亏了。去年丢失腊戌后，军政府阵营内一片哗然，因为缅甸内战数十年来，从未丢过若开城市，缅军占据大城市，民地武占据乡村和山区，一直以来的规律。没想到，敏昂莱贡宁境一个丢失大城市的军方首脑，其威信受挫，本就不高的缅军士气，再遭沉重打击。

今年4月初的地震，更是令军政府的北方司令部被毁，向北方的物流几乎断绝，形势岌岌可危。军政府一旦倒下，民盟会重新上台。而当下的民盟，和美国关系太过密切，不是好事。所以，我们得拉军政府一把。腊戌市区的交还，能帮敏昂莱稳住阵脚。起码一时半会，他是不至于倒下了。

国内很多人对此感到不满，觉得失去了打通印度洋出海口的机会。其实，只要天下没有大乱，开疆拓土本就不太可能（大乱了另说）。当下，我们和军政府合作纯属不得已。军政府控制着缅甸南部，也就是下缅甸，中方规划的 中缅铁路、"中缅管道、皎漂港等重大工程，都需要经过下缅甸，别人有的是地方可以卡你。只要你没法灭了军政府，就得和他合作，完全不合作是不现实的。

同理，在美国没有真的吞并他国领土前，先当出头鸟，没准会引来围攻。等哪天美国吃了格陵兰岛，我们再跟进，遭到围攻的可能性会低得多。乱世之中，出头鸟的下场，通常和袁术差不多。我们更应参考老朱的策略：高筑墙、广积粮、缓称王。连西方列强都知道，对于大清，最好不要推翻，而是留着个架子，一点点剥夺其利益。留着缅甸军政府，也可以一点点迫使其让渡利益。它要不是陷入了困境，能同意我们派遣安保人员入境？

尽管让出了腊戌市区，但总的来说，同盟军还是赚大了。从2023年末发动攻势以来，不仅收复了整个果敢，还向西攻下了贵概、木邦等地，即使不计算腊戌市区。与其勉强占着腊戌，把弦绷得太紧，不如松一点先收缩战线，也就不用强制征那么多兵了。可以腾出手来，消化把新占领的土地。反正腊戌周边在手里，腊戌市区是被卡着脖子的。消化占领区，壮大实力，同盟军才能完成转型，从一支山沟沟里的游击队，转变为地方诸侯。
