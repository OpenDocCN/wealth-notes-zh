
# 钟国打出三张牌，博弈进入新阶段

## 时间线回顾

在9月底之前，中美贸易关系相对平缓。TikTok问题达成了协议，甚至有传闻中美领导人可能在10月底韩国举行的APEC会议期间会晤。但9月底开始，美国突然连续出手。

首先，9月29日美国商务部推出的“50%穿透规则”。这个规则规定，凡是被列入实体清单等限制名单的主体，其全球范围内持股超过50%的子公司，将自动适用同等管制。这个规则的真正威力，在于它把合规成本和法律风险，转嫁给了所有与中国企业有业务往来的全球合作伙伴。

其次，10月3日美国贸易代表办公室落地的对华海事、物流和造船业的301措施。从10月14日起，对中国籍、中国制造或中国运营的船舶分段征收港口服务费。

最后，10月9日美国财政部以“规避伊朗石油制裁”为由，将山东金诚石化等16个中国实体及个人，列入制裁清单，影响包括资产冻结、交易禁止等。

美国这三轮施压的节奏很密集，从9月29日到10月9日，不到两周时间。这是典型的特朗普风格，在新一轮谈判前搞极限施压，抬高筹码。

## 中国的反击

而这一回，中国的反应也非常快。10月9日，也就是美国宣布SDN制裁的当天，中国商务部、海关总署连发公告，开启了强硬反击。

中国这次打出了三张牌：

第一张牌，是稀土。中国将全面收紧稀土及相关物项的出口管制。公告明确规定，对最终用途为研发、生产14纳米及以下逻辑芯片，或者256层及以上存储芯片，以及制造上述制程半导体的生产设备、测试设备和材料，或者研发具有潜在军事用途的人工智能的出口申请，逐案审批。

第二张牌，是港口收费。10月10日，中国宣布自10月14日起对美国船舶也收取特别港务费，直接回应美国对中国船舶征收港口服务费的做法。

第三张牌，是市场准入。同样是10月10日，国家市场监督管理总局以“未依法申报经营者集中”为由，对美国高通公司开展反垄断立案调查。要知道，2024年高通46%的收入都来自中国市场。

## 稀土的战略价值

很多人可能会觉得，稀土就是一种矿产资源，因为中国储量多，所以能卡住美国。其实，事情不是这么简单。

中国稀土的真正优势，不在于储量，而在于产业链。2024年，中国稀土储量约4400万吨，占全球的34%。这个比例确实不低，但也不是压倒性的。越南有2200万吨，巴西有2100万吨，美国自己也有1900万吨。

但是，再看另一组数据，中国稀土产量占全球的69%，冶炼分离产能占全球的92%，稀土永磁材料产量占全球的90%。

美国的芒廷帕斯矿，是美国唯一运行的稀土矿山，2024年产量4.5万吨。但这些稀土挖出来之后，还得运到中国来冶炼分离。

那么，稀土到底有什么用？为什么美国离不开它？

稀土不是一种元素，而是17种化学元素的统称，包括钕、镧、钇、铈等等。这些元素有独特的物理化学性质，在高科技领域几乎不可替代。因此，稀土被称为“工业维生素”。

比如，在芯片制造领域，稀土无处不在。光刻机的精密运动控制，需要钕铁硼永磁体，一台EUV光刻机就需要几十公斤；化学机械抛光，需要氧化铈；刻蚀设备的耐蚀涂层，需要氧化钇；等等。

再比如，在新能源领域，每辆电动汽车需要2—3公斤钕铁硼磁体。每台风力发电机需要1吨以上。

再比如，在军事装备领域，每架F-35战斗机需要约400公斤稀土，用于雷达、电机、电控设备、陀螺仪。建造一艘核潜艇需要约4吨稀土。

简单说，没有稀土，F-35飞不起来，电动汽车跑不远，风力发电机转不动，芯片造不出来。

美国能不能找到替代方案？理论上可以，实际上很难。

首先是时间成本。有学者认为，美国重建完整的稀土产业链需要10到20年。而且这还是在一切顺利的前提下。

其次是技术壁垒。中国在稀土冶炼分离技术上，领先全球3到5年。比如，中国的“串级萃取”技术，能实现17种稀土元素的高效分离，纯度达到99.999%，成本只有国外的五分之一。

最后是经济成本。即使美国愿意投入巨资重建产业链，但是建设周期长、初期投入大，在经济上很难和中国的规模化生产竞争。

2025年4月，中国实施稀土管制后，福特汽车芝加哥工厂因稀土短缺停工一周。通用、丰田、大众等汽车巨头表示，没有稀土磁体，关键零部件生产即将瘫痪。这就是稀土的战略价值。

但有一点需要说明，中国商务部在新闻发布会上明确表态，“出口管制不是禁止出口”，“对符合规定的申请将予以许可”，并会考虑“通用许可、许可豁免等多种便利化措施”。换句话说，中国手握的是“水龙头”，可大可小，而不是彻底焊死。这个审批权本身，就是博弈筹码。

## 博弈态势的变化

看完前面这些，有人可能会问，今天的双方博弈，跟过去有什么不一样了吗？

最关键的地方在于，中国在经过上一轮极限贸易战后，有了更强的战略自信。这个自信来自很多方面。

首先，是经济的韧性。尽管面临外部压力，但中国通过“一带一路”等举措，与东盟、中东、拉美等地区的贸易额持续增长，有效对冲了美国市场的风险。内部庞大的市场和完整的工业体系，也提供了足够的回旋空间。

其次，是科技的突破。以华为910C芯片等为代表的芯片领域的进展，虽然与美国顶尖水平尚有差距，但已经证明了中国在“卡脖子”领域突围的决心和能力。这让中国在应对科技战时有了更多底气。

最后，是对对手的认知。经过几轮交锋，咱们已经熟悉了特朗普政府“极限施压、临阵多变”的套路。特朗普这种“关税讹诈”造成的冲击，正在变小，这是个非常重要的变化。

当然，面对当前的情况，我们必须保持清醒和客观。但博弈的“增量”和“变量”，确实正在向有利于中国的方向发展。

## 未来展望

从目前的进展看，双方回到谈判桌前的可能性，依然是明显存在的，而且双方都预留了回旋空间。美国关税生效日是11月1日，中国稀土规则生效日是12月1日，这中间还有谈判窗口。

而从长期来看，这次博弈最大的特点，是从“关税战”转向了“规则战”。美国的“50%穿透规则”与中国的“稀土0.1%规则”，都是通过规则来影响产业链的组织。也许，这种“规则战”比关税战影响更深远。
