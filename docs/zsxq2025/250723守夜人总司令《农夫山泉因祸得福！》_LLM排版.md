
# 农夫山泉因祸得福！

## 前言
从这个角度来看，貌似老钟（钟睽睽）的节操和底线比大家想象的都要高。

## 正文
宗馥莉宗大小姐的操作，不仅把官方试图给他父亲宗庆后的体面全部打碎了，还让普罗大众的心态以及舆论氛围，随着时间的推移和细节的呈现彻底反转。官方也不好使劲去包装和维护，毕竟撕开了就捂不住了！
当时官方是有期许的，消费者也是有期许的，只不过，这个期许都落空了。官方的期许落空在于娃哈哈集团的内斗造成公共利益受损，至今没有合理的解决方案，消费者期许的落空就更直接了，民众之所以会对娃哈哈产生强烈的情绪，源自于一种朴素的民族情感和内心诉求。现在民族情感被现实撕裂，内心诉求也变成了一种笑话，所以，民众的心态必然会引发强烈的反弹，这种反弹会作用在品牌上！

当我们重新回到事件现场，突然之间，大家会觉得农夫山泉的创始人钟睽睽并不像大家想象那样虚伪，钟睽睽大大方方的承认了自己的儿子是美国人，钟睽睽说他的儿子30多岁了，有自己的选择，但是，农夫山泉永远是一家中国企业，这个回应在当时的舆论狂潮中被大家无视了！然而，从股权架构的角度出发，农夫山泉比娃哈哈的股权结构更像100%的中国企业，因为农夫山泉股权结构没有那种EVA架构，就是海外资金持股内部生产企业的股权架构。
中国所有的互联网企业都是这种架构，除了华为这类没有上市的公司之外，绝大多数之前改制的公司都采用了类似的架构。一方面是为了在海外上市，另一方面也更方便资金的“国际化”。为什么包括恒大集团在内的一系列房企，在美元加息的周期中集中出现美元债务风险呢？就是因为为了方便搞出来的各种隐性通道，并且依托这些机制搞出了隐性的风险。
言归正传，农夫山泉的企业架构是中国化的。创始人钟睽睽的儿子确实自己选择成为了美国人，只要股权结构是中国化的股权结构，即便钟睽睽儿子是美国人，也得按照中国的法律法规来经营。就好比中国人在美国参加了一家企业，完全按照美国的股权结构和法律机制运行，这个美国企业的权限也只能在美国的法律体系和股权架构之内发挥作用，这就好比抖音海外版！

在农夫山泉被舆论按在地上反复摩擦的时候，回过头来看，农夫山泉的创始人钟睽睽，有一件事情做的还是很 有底线。钟睽睽肯定知道娃哈哈的内幕情况，不管是新生子问题，还是股权结构问题，都可以大做文章，因为普罗大众对事物的理解比较表面。大佬文集圈尤其是花边新闻，更能转移注意力，因为花边新闻更契合普罗大众的道德化判断方式，而且民族主义也是一个能够引领大众情绪的好抓手。然而，农夫山泉的创始人钟睽睽只是为自己辩解，并没有利用这些劲爆内幕进行反击，今天回过头来看，钟睽睽这个人还是有底线的！相比较而言，娃哈哈宗馥莉大小姐做起事来更加的不管不顾...

可能大众会把自己内心的美好期待投射到一个又一个符号身上，当这个符号瓦解之时，能不能会为自己曾经的的那种虔诚和狂热感到羞耻，这就是为什么英雄跌落神坛之后，更容易遭受普罗大众落井下石的原因。因为大家为自己曾经如此的崇敬这个英雄感到羞耻，为了摆脱这种羞耻感，就要加大力度表达对这个自己曾经无比推崇的人的憎恨和鄙夷，所以，之前反复说过：声望是最微不足道的力量！千万不要把乌合之众的认可当做自己的本钱，真正的本钱永远是占有资源并拥有不可替代的独特的能力。

虽然农夫山泉的竞争策略有点狠，甚至可以说是营销过渡或者剑走偏锋，但是，农夫山泉采取垄断渠道的方法，在零售上是非常成功的模式。以广告改变认知的手段，在营销上也是非常成功的。矿物质的水是不是比纯净水更好呢？所含的矿物质是不是真的如营销所言呢？这个确实是瑕疵，但不影响整体，因为消费者本来就是为广告付钱！就连茅台也未必是原厂，有可能是贴牌的，特别是市场销量大的产品尤其如此。然而，大家照样买不误，因为大家买的是广告给自己脑中植入的那种观念，至于产品与自己脑中的那个观念的匹配度能达到多少，并不会深究。

这些争吵让大家越来越清晰地认识到一个问题：不管是抬一个打一个也好，还是对别人寄予厚望，充满期待也罢，最终的现实都会回归人性和商业底色。这就好比外卖大战，最高调的那个京东外卖刘强东现在已经偃旗息鼓，之前以骑手代言人和良心企业家的人设刘强东吸引了许多善良且缺乏判断力的人，宣称要改变这个改变那个行业，到头来什么都没改变，京东外卖自己先退出补贴，高调宣布的福利也没有了。

这个社会上不乏一些天真的人，这些人的内心对外界充满着一些不切实际的期待，并且会把这种期待投射到一个又一个的个人或者企业身上，当这种期待落空之后，又破口大骂，把人家说的一无是处！这个群体每一次都会上当，而且每一当都不一样，这就是命运，活该！

恰恰是广泛存在着这样的群体，接下来这群天真的人会把对娃哈哈的失望，回过头以补偿的方式去支持农夫山泉。之前在自己心中无比无龌龊的农夫山泉，突然又变得慈眉善目起来..正是广泛存在着这样的群体，以及可以预见的行为和心理，所以，社会治理者才可以不把这群天真的人当回事，能够以无比简单的方式反复引导这群天真人的行为和情绪。
