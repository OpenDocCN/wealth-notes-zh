
# 卢克文青云读书会5.29环球军情第72期：一场改写南亚历史的暗战

## 巴基斯坦和印度的核武器研制

印度在1944年就开始着手为印度准备自己的核武器计划，当时印度著名的物理学家拉贾·拉马纳扩大和监督了印度当时的核武器项目。印度在尼赫鲁执政时期还决定不签署禁止进行核试验的国际公约，这为印度后来秘密发展核武器奠定了国内的合法性。1967年被称为印度铁娘子的英迪拉·甘地成为了新一任的印度总理，重新恢复了军用核武器研制的计划。

巴基斯坦在1976年7月秘密开展核武器计划，重要核物理专家卡迪尔·汗博士全面负责巴基斯坦的核武器计划。巴基斯坦先向欧洲、美国等国派遣大量的留学生和交换学者，以学习掌握最先进的核物理的技术以及核武器的研究，同时秘密地以各种方式向西方国家采购先进的离心机。

## 印巴冲突与核武器

1974年印度在塔尔沙漠引爆了第一颗原子弹，这次成功的核爆不仅让印度跻身为了拥有核武器的国家，也引起了巴基斯坦的恐惧和野心。巴基斯坦总理布托公开宣称，即便巴基斯坦人民吃草，也要造出原子弹。巴基斯坦一直对印度心怀不满，克什米尔地区也是印巴两国始终重兵对峙的地方。

## 印度情报机构的行动

印度的调查分析局RAW在1976年得到了一个关键情报，说参与706项目的巴基斯坦科学家经常会到镇上的一家理发店理发。印度的情报人员就假扮为巴基斯坦科学家，通过理发店获取了重要情报，最终成功阻止了巴基斯坦的核武器试验。

## 核武器对南亚地区的影响

印巴两国都拥有核武器，打破了原来长期被认为的拥有核武器的国家不会轻易发生军事冲突的准则。这场印巴空战对整个世界局势没有什么好处，对南亚地区的影响尤为显著。
