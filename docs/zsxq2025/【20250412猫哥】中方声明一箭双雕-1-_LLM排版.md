
# 中方声明一箭双雕

昨天，针对美方将中方关税莫名加到 145%——在加关税之前，特朗普再次表态很希望与中国达成协议，如果中方给他打电话，他将非常慷慨云云。很明显，特朗普加关税就是在与中方谈判前给自己制造筹码。但是中方的反制更为漂亮。直接将美国商品关税加到 125%。

这段描述堪称含而不露却一箭双雕的经典! 为什么? 我们一层一层来剖析。我们把这句话拆解一下。

## 第一层是，“鉴于在目前关税水平下，美国输华商品已无市场接受可能性”

这是啥意思? 意思就是在现有的关税水平下，未来中国采购原产地美国商品的可能性为 0。虽然美方出口中国商品主要是能源农产品，在现有关税水平下，确实农产品与能源是不可能采购了，但是我们每年还是要采购其他美国商品，甚至可以说有些美国商品是难以替代的，比如精密实验仪器与设备，比如高端芯片。这些产品即便加了 125%关税，如果国家层面不表态，我相信有些单位与企业还是会继续采购这些美国产品的。但是国家层面公开表态了——“美国输华商品已无市场接受可能性”。未来那个单位与企业还会采购? 国企与体制内如果采购，官帽子不想要了? 民企如果采购就是公开与国家唱反调对着干，都不用国家出面，舆论的口水都会将其淹没。所以，昨天下午资本市场半导体企业股价突然大涨，原因是，半导体协会出台了认定半导体产地的标准，就是按照流片所在地认定——也就是说，未来英伟达也好，高通也罢，如果要继续卖芯片给我们，只能将相关芯片拿到中国企业加工作生产，当然台湾省的台积电也算。否则，英伟达与高通等美国企业一块芯片也别想卖给我们，这就是国内芯片股大涨的逻辑。这是第一层含义，未来特朗普不与我们谈判降关税，美国产品将彻底丢掉大陆市场。

## 第二层含义就是对特朗普将军。“如果美方后续对中国商品继续加征关税，中方将不予理会。”

有了这个表态，特朗普即便再抽风也不可能继续对华产品加征关税了，毕竟特朗普多少还要点脸吧，那么这意味着什么? 我在之前的文章给过大家一组数据。单价在 50 美元以下的中国商品竞争力非常大。比如浙江工厂生产的咖啡杯，我们出厂价是 0.8 美元，在纽约超市标价是 48 美元；中国工厂生产的蓝牙耳机出批发厂价 3 美元，美国德州工厂出厂价是 22 美元；义乌生产的 LED 灯泡报价 0.5 美元，而密西根工厂生产的同类产品报价是 22 美元；深圳生产的空气炸锅 59 人民币质保 3 年，而芝加哥生产的同类产品报价是 499 美元质保半年；大家对比一下，即便美方对华商品加征 145%的关税，单价较低的中国商品依然有竞争力。所以理论上只要封杀特朗普继续加征关税的动作，不要将对华商品关税加到 400%甚至 500%，我们依然能保住单价较低的中国商品的美国市场。

现在中方一句话直接怼在特朗普脸上——“鉴于在目前关税水平下，美国输华商品已无市场接受可能性，如果美方后续对中国商品继续加征关税，中方将不予理会。”——特朗普但凡还要维系一点美国的国家颜面，都不好意思继续加征关税。否则就会成为全世界的笑柄。也就是假如维持当前的关税水平，中方用一句话不但让美方产品彻底丢掉了中国市场，同时也为部分单价较低的中国商品保住了美国市场。

现在中美处于僵持状态，谁先开口谁被动，但是中方这样一箭双雕的策略让僵持状态对中方相对是有利的。我们就是要逼着特朗普先低头。
