
# 普通人逆袭之路（二）

## 一、内在

如果你只有很小一笔钱，你想要翻倍，想要100%无风险的确定性的翻倍，是极其简单的。只要你自己过硬就行了。因为根本不需要依赖外部条件，无论外部条件有多么糟糕，这对你来讲都是小菜一碟。

就像那个浙大学生问段永平，请问大环境如此糟糕的情况下我该如何破局，段怎么说的?大环境有好过吗?一直都不怎么样啊，我们一直都在动荡中呢。我觉得我们一直都有危机啊，有人危，有人机，做好自己就好了。

## 二、出路

普通人的生态位是什么?是华府的低等下人。那可行的目标是什么?是努力成为高级下人(挪一个位置)。

绝大多数人改变自己的命运，都不是通过什么人云亦云的跟风。很多年前，我就见过无数人云亦云的跟风者。从君子兰，黑贝犬，到藏獒，到一路走来各种层出不穷的人云亦云的跟风者，我见太多了。

这些人，一个个的老去，我也没见谁改变了 自己的命运。最终改变了自己命运的是什么人?是改变自己位置的人。

你在学生阶段的时候，你慢慢的发现自己身边只剩顶级的学霸了(升学进好学校，分班进尖子班)。

等你工作的时候，慢慢发现自己身边再也没有普通员工了，而是纷纷变成各类高管，乃至董事长们的时候。你就明白，你在渐渐改变自己命运了。

你能比出来的，明白不?你小学同学啥样，中学啥样，大学啥样，毕业参加工作3年啥样，5年啥样，8年后啥样。你能比出来的呀，前提是什么?你一路在上升。

当你平生第一次去参加行业峰会的时候，走进大厅你已经清楚了。你们全行业有多少人，能进这个厅的，才有多少人?

## 三、散户

那咱们来看一个问题，散户的生态位是什么?实际上跟街头小摊贩是一样的。

有部剧《装台》,就讲这个的，第一集就有一 个街头卖胡辣汤肉夹馍的小摊主，被装台的那伙子人给按住了，让他收拾东西跟他们走。

这家伙他不知道人家是来包场的，他以为人家 是城管，就求饶，这么多人，咋就单抓我呢?话说这就是小摊贩的人生啊。

他要不要操心去哪儿买面粉 ?去哪儿买肉 ?当然要操心。买的不合适了，客人吃的不好，都是自己的事儿。他可不可能像大爷一样跟城管说，去，你给我去买面粉，买肉?可不可能?城管是来管你的，不是来伺候你的。你不仅要想办法做好自己的小生意，你还要看城管的脸色，今天卫生不好了，明天市容不好了，后天防火不好了。

这些对你来说，就是风雨雷电，就是自然不可抗力。而你，要拉着你的小板车，在风雨飘摇中出摊，0一家人的生计，还背在你肩上呢。

这就是你的生态位呀，这就是你要干的活儿呀，否则凭什么让你挣卖馍的钱呢?城管替你买面，城管替你卖馍?城管替你遮风挡雨?城管是你亲爹啊?

那我们回到散户，不也是同样的道理么?有读者和剑客说他也要去海外做个幸福的散户，马上随之而来的问题是什么?不熟啊，你怎么知道什么平台是正规的，什么是有问题的，你怎么知道入了金，就一定能出金呢?

同理，那个卖肉夹馍的小摊贩他自来熟?他是葫芦娃，一出生就知道哪里买面粉，哪里买肉丸?

那你即便现在国内做散户，每个品种都有它自身的问题。你通过银行的托管户，出入金相对来说是最安全的，手续费是最高的。正常不正常?很正常呀。

就像你去超市进面粉进肉就是比你去批发市场 更安全，但是也更贵。难道不对么?这就叫安全溢价。

你有时候发现国内的有些品种，0它会走“独立行情”,俗称该涨它不涨，该跌它不跌，它缩小了你的盈利空间(波动太小，你一味持有始终没有利润)。有问题么?

好比你开个卖早餐的小店，大部分几个月就该关门了(因为你始终在给房东打工，你就解决不了成本效率问题，周边竞争太激烈，供应链又没有议价权啊)。但是，偶尔有几个解决了效率问题以后，一旦生意变好了，你猜怎么着?房东要给你涨房租了，你是没见过么?

你有时候还会发现，你在某些交易所做的好好的，人家交易所忽然规定，暂停，不许个人户入场，只许商家，然后你的交易就迅速冷清了。

这就像你辛辛苦苦的在淘宝开了一家店，忽然有一天，规定，不许开店了。正常的很呀，就像一个卖胡辣汤肉夹馍的，他好容易站住了一个摊位，忽然遇到市场拆迁了，这太正常了。

各位看到了么?作为散户，你诉的所有苦，那个卖肉夹馍的，他都要经历呀。

你问他过的是什么日子?他就是先弄点面粉，先找个地方吆喝试试，“又没有人吃额滴馍，额滴馍燎扎咧。”
喊了一天，口干舌燥的，他望着卖不出去的馍，忧心忡忡的又蹬着三轮往家回，夕阳照着他落寞的身影，疲惫了一天，但明儿还得继续早起，得换个地方继续试试不是?不然呢?

话说这个生态位的人谁不是这样过活的?哪怕你爬到李子柒式的顶流，她也是这样一路走来的。

李子柒有没有委屈?肯定有。我们知不知道谁占理?也知道。那有没有办法?没有的。

因为谁敢强行替李子柒出头，那投资营商环境就被破坏了，以后谁还来投资?所以怎么办?李子柒就不得不继续又熬了三年，吃一堑长一智呗，谁叫你自己当初签合同的时候不长心眼，怨不得谁，顶流也得面对真实的生活啊。

## 四、定义

所以剑客之前在普通人系列讲得很多策略， 一直 在讲普通，普通，只适合普通人。很多人对普通的理解和我的理解，南辕北辙。

我说的普通是，虽然你没天赋、没背景、没资源、没特长，但前提是你像那个卖肉夹馍胡辣汤的小贩一样，早就被这个世界折磨的遍体鳞伤。你的心态普通了，你才叫普通人。

普通人=四无现状+认清现实

有读者上来就和剑客说，我是小白，有没有什么课程可以学习下?我都直接劝退。什么时候等你能够正视自己到底是个啥生态位了，你才叫普通人，否则你不是普通人。

你看罗永浩30岁的时候对自己的总结：一个没钱的少爷。

你没天赋没背景没资源你以为自己就够普通了?不，只要你还有一颗少爷的心，你永远都不会普通。

哪天你能够像中年的闰土一样，亲手取出自己的少爷心，踩在尘埃里，讷讷的喊出那声老爷，你就真普通了。

真普通了，你再来看剑客写的攻略，你才会真正的“读懂”它，用它去反复执行。

这些步骤，你站在学生娃的角度去看，是让你难以接受，但那一定是符合现实的。
