
# 八月必看：从指数行情到个股行情

## 前言
要看的尽早看，有些内容看晚了就没意义。七月份指数涨了不少，要是没有月末最后一天的那一跌，形势更是一片大好。但七月份不会有指数行情，如果有，必然会迎来主力的砸盘纠错。砸盘延续到了八月份的第一个交易日，按照惯例，接下来，你们还将迎来一波指数上的回调。

我们来讲讲其中的逻辑吧。这波指数上涨是从四月初开始的，懂王用关税战全球直接砸盘，东大率先反弹，逻辑是只有我们做好了跟美帝打关税战的准备，资金争先跑动东方避险，这个看港股的涨幅就明白了，港股走势远超大A。

港股的定价权一直都在国际资本手上，所谓国际资本很多都是出口转外销的，跟被炒上天的美股比起来，港股既有性价比优势，又能站在门外，绑定一把东升西落的时代红利。

后来懂王很快认怂，玩了一手TACO，算是给美股回了点血，更厉害的是哥们开了一手稳定币，绕过了美联储，踩着美元霸权玩自己的赛博印钞机，市场预期到了未来不差钱，那还有什么理由不涨呢？

转折点是7月底的斯德哥尔摩会谈，这一次，美帝感觉自己腰杆子赢了，毕竟连续对欧盟、日本大笔敲诈成功，内部又开了一轮赛博印钞机，不必太过担忧美联储卡着不给钱了，于是过来谈判的代表漫天要价，更是炮全开，要把关税加到100%以上，给点颜色看看。

当然，嘴巴上说得厉害，实际操作还不敢动真格的，最后的结果就是继续谈，再把之前的协议延长90天。懂王其实也要扛着内部压力的，好大喜功，各种鼓吹极端情绪的结果就是没办法在谈判里本着实事求是的态度，权衡之下，还是说最硬气的话，干软怂一点的事，能糊弄就糊弄，能拖就拖。

懂王其实忘了，到底谁才是自己的敌人，肯定不是东大，把火力瞄准东大，并没有半毛钱的好处，只会让自己真正的敌人美联储乘机搞事情。

美国7月份的非农就业数据崩了，远不及预期，最郁闷的是人家还搞了个回溯修正，不仅7月份数据不行，6份数据也很糟糕，言下之意是懂王根本不懂经济，上任就搞事情，把经济搞得一塌糊涂，虽然就业数据不行，但通胀压力也很大啊，那也是因为懂王瞎搞什么关税战，反正就是不降息。

懂王很生气，然后第一时间就开喷想解雇掉劳工统计局长。市场的反馈就很有意思了。按照以前的走势，每次非农数据不理想，美股反而会走得不错，因为大家预期的是未来降息可能性变大，所以股市涨，配套的是美元指数跌。

但8月第一个交易日美股是大跌的，这就有点不寻常了。有不懂装懂的说这是因为就业数据不理想，笑死了，以前一样的情况发生，为啥美股会涨？

我们说过很多次，股市并不是经济的晴雨表，而是流动性的晴雨表。美股之所以最近能涨那么多，不是因为最近美国经济有多好，大家可以去看看美国的制造业指数和小非农，6月份的小非农—3.3万人，这数据影响了美股上涨吗？

个人觉得，美股之所以跌的根本原因是中美这轮谈判全球看到了东大的强硬态度，对美帝的信心下滑，再加上美国内部搞内讧，美联储摆明了要跟懂王斗争到底，这一届白宫的草台班子，对外，没办法搞定东大，对内，没办法搞定美联储，内外交困偏偏懂王又不认怂。

数据不好看就要开除给数据的人，这已经是非常不尊重技术部门的独立性了，再加上不尊重印钞机的“独立性”，这就让全球投资者都担忧这位大爷随时会发疯，到时候就不是美股美债的投资价值问题了，而是敢不敢再持有美元的问题了。

要知道，美元霸权的重要支撑之一是东大提供的量大性价比超高的工业品。没有独立性的美联储有不计后果滥发美元的嫌疑，跟东大处不好关系的美国有可能会失去东大商品对美元的背书。

这个时候，投资者先走一步，落袋为安没有错吧。别忘了，美联储必须得在9月份降息，不是被懂王逼得，而是再不降息美国金融界就要绷不住了。

如果市场一片大好，美联储降息，那就是懂王战胜了美联储，颜面扫地，以后就要被懂王随便揉捏了。但如果刚好遇到美股、美债一起出问题了呢？金融市场再出点幺蛾子，整一下懂王，然后锅就扣给懂王的关税策略。

跟欧洲谈的关税协议，冯德莱恩回头就说了，6000亿美元投资没钱给，你就别当真了，反正15%的税率我们先执行着，大概率别的国家也会有样学样，这就很丢面子，其实关系不大，毕竟他们体量小，也不敢真的搞反制，但东大不一样啊。

实际如何其实不重要，重要的是，懂王的对手是可以控制资本市场的涨跌，并且他们想这么做，且又有了借口。那么我们得出结论就是，大概率，8月份美股会有一波下跌，跌幅还不小。

美股跌，港股必跟，港股跌，大A在指数上就有很大的回调压力。但指数涨跌跟你有什么关系呢？你又没买指数。我们要关心的是市场流动性，如果9月份注定美联储降息，8月份流动性会很差吗？

主力肯定会用关税谈判当借口砸砸指数，但市场活跃资金也不是傻子，指数走指数的，个股走个股的，看看过去几个月，多的是板块和个股没跑赢大盘，走得好的金融、基建、医药、新能源这些板块都很能影响指数。

资金在扎堆抱团炒作的时候，让一些质地不差的票惨遭市场抛弃，应涨未涨，那么刚好趁机高低切换下，有问题吗？所以八月份没有指数行情不代表没有行情，市场活跃资金现在根本不会走的，钱在，行情就在，无非是从一个方向转到了另一个方向罢了。

个人认为月初大A会有一个下杀动作，大概率是指数和个股一起被杀，但还是那个话，大家没必要对此过度担忧，明牌在那里了，9月份美联储降息，我们的80周年阅兵式，两大利好加持，大盘根本不会差到那里去。

记住，股市等同于故事，只要未来的故事够有吸引力，眼前现状再惨淡，也掀不起太大的风浪，无非就是大盘给你个好价格，接你上车。月初如果遭遇砸盘，不要无脑去追高杀低，仓位高的，你就不管不看了，仓位低的，抓住机会加仓就行了。

这里大家记住画个重点，八月份中美之间博弈会加剧，为了避免被误伤，大家最好远离港股，道理很简单，港股是外国人眼中的中国资产，定价权在老外手上，恒生指数从今年4月份低位起来到高位已经涨了超过30%。

还一部分人喜欢买港股里的恒生科技股，大家可以去看看7月底出台的《网络交易平台收费行为合规指南》，这是监管层要对赚钱赚到手抽筋的平台们下手了，利空科技股，所以，看着办吧。

总结一下，八月份尽可能的少碰港股、美股，当然了，如果赶上了单日暴跌，是可以酌情做个短线抄底的，记住只有暴跌才冲，没暴跌，不予理会。

说回到大A，个人认为八月份单以指数论，大A会强过港股和美股，并且大概率从中下旬开始，指数会迎来反弹。道理很简单，如果所有人都对9月份的行情有个比较积极的预期，那么8月份再怎么回调，它的杀伤力都会很有限，并且资金还会有动力去积极的修复指数。

所以八月份如果遇到大盘下杀，我们要优先买入券商板块。这里说说金融三巨头的逻辑。砸盘现在喜欢用银行，七月份指数上涨，工具人银行率先被祭出来杀指数，连带着保险一起被连累。

现在银行处在历史高位，东西两大佬都想砸一下指数，那么大概率大A会用银行砸盘。七月份保险被银行连累，走势比较颓废，但保险整体位置较低，所以八月份即便用银行砸盘，保险走势也会好过银行。

当然，对于长线投资者来说，这个位置继续定投保险依旧是个好的选择。但因为我们对九月行情有较高的预期，我A一向喜欢用券商来拉指数，所以八月份趁低买入券商中短期收益要好于保险。

大家也可以埋伏一波互联网金融，最近东大对外比较强势，不排除会在八月份在RMB结算上弄出点大动作，直接刺激下美联储，也算是金融上的亮剑吧，毕竟九月份要搞阅兵式，这也是在亮剑。

军工拿住了，随时能起飞。十年前也有大阅兵，当时军工板块跟着暴涨了一波，大家可以去复一下盘。关于军工的逻辑我说过好多次，这里真不想多说了，基本面已经变了，我们开始经常性亮剑了，最近公布了洲际导弹的视频，这是给某些人一点颜色看看。搁从前，东大是不会这么干的。

可以想到的，未来东大亮剑的时候会越来越多的，每回都会给军工带来一波冲的。九三大阅兵必会刺激军工上涨，手上没有的，赶紧低吸，确定性机会错过就太可惜了。

新能源板块本月估计会喜迎回调，道理很简单，本身新能源这个老赛道的套牢盘挺多的，板块被杀了几年，但还没杀掉那些人的精气神，最近涨了不少，里面的老资金，老庄就会忍不住搞事情。

最糟糕的是，新能源板块几乎是绑上了创业板指数，杀指数的时候，杀杀老赛道，需要那么多理由吗？最近贸易战打得正猛，我们现在在新能源行业里遥遥领先，市场占有率全球第一，只要贸易摩擦一起来，几乎就能预见，全球会有各种针对东大新能源的不利政策出台，这时候躲避风险很重要。

回避一下七月份曾大涨过一波的基建相关票。七月的水电站大涨了几天，我在文章里也写过，水电站概念是不具备持续性的，本身涨也是因为板块跌太多了了，属于超跌板块借一个利好快速反弹，但反弹不是反转，这里并没有基本面的好转，还真不同于新能源板块。

好歹新能源那也是有个反内卷，去产能来支撑估值，基建有什么？连带也回避一下地产吧，这玩意是政策导向的，对地产的政策基本出尽了，再搞大规模的刺激不太可能，所以近期还是要呵呵哒。

重点关注下通胀相关概念，比如说有色、猪肉之类的。啥都不说了，美元指数大跌，9月份美联储降息已成定局，美国通胀本来就没解决，这波一降息，更会雪上加霜。

当然了，有懂王这个大神在作妖，美联储跟白宫之间的矛盾会越发激化，无论谁压倒了谁，裂痕必然扩大。美元本来就是美国政府的欠条，政府各种不讲信用，跟全球金融玩火并，美元的信用它能好的了吗？全球必然会加速抛售掉手中的美元，这也就意味着大概率以美元计价的大宗商品会涨。

也别指望RMB汇率会大涨抵消掉这波大商品的涨幅，还是我之前说的，全球其实都暗地里希望有一波通胀，这锅还得由美国来背，既能化债，又能甩锅。

个人觉得创新药到现在，基本差不多了。这波创新药上涨有两个驱动，一是东大医药科技里被欧美追捧，工程师红利闪瞎了欧美的双眼，上半年中国药企光靠“卖药给老外”，就赚走了480亿美元，比去年全年还多160亿。二是国对创新药给了很多政策支持，资本纷纷往这个赛道里杀。

但问题的关键是板块跟个股都涨猛了，起码是短期内涨太猛了。创新药作为曾经的基金最爱，里面埋了一堆基民，还记得大明湖畔的噶兰吗？还是那个话，创新药极度依赖国外市场，这里有政策风险，自己去品吧。

国内市场因为医改，所以市场整体规模是萎缩的，这个趋势是不可逆的，大家可以去看看我对医改的一些看法的文章，这里就不多说了。最近关于创新药的利好新闻特别多，按照惯例，这个时候就是资金开始密集出货的时候了，如果你有票，可以考虑逐步减仓，如果你想入医药，特别是喜欢医药，建议你多观望一下吧，起码等看到了回调，出利空的时候再入，何必追高呢？
