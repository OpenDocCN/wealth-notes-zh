
# 2025年，哪些城市的造富能力最强？

最近，既是一年一度的高考志愿填报季，也是应届生的就业季。很多人都面临城市选择的问题。怎样判断一座城市的前景？通常的方式是看城市资源、看产业格局，或者观察企业分布。但是今天，我们要说一个观察城市的新视角。

这就是，观察一座城市的造富能力。

造富，就是城市能在多大程度上促进个体创造并积累财富。

就在今年5月，英国的投资移民咨询公司亨利及合伙人牵头发布了一份报告，叫《2025年世界富裕城市报告》。怎么判断一座城市的富裕程度？这份报告采取的方式是，统计百万美元富人的数量。个人净资产超过100万美元的人越多，就认为这座城市越富裕。

首先，按照这个标准，全世界最富裕的城市依然是纽约，拥有38.45万名百万富翁、818位千万富翁，以及66位亿万富豪，位列全球榜首。值得注意的是，旧金山湾区虽然以34.24万名百万富豪位列第二，但亿万富豪的数量超越纽约，达到82位。这是因为旧金山有硅谷，那里聚集了美国大量的科技资本。

统计显示，过去十年，硅谷的百万富翁增幅高达98%。之前有投资人透露，英伟达76%的员工是百万富翁，超过三分之一的人净资产达到2000万美元。报告中还做了个预测，“得益于硅谷科技飞轮的推动，尤其是近几年的AI创业潮，旧金山超越纽约只是时间问题”。

其次，亚洲正在成为全球财富增长的新中心。从迁徙路径看，越来越多的高净值个体正在迁往亚洲。其中，新加坡是最主要的目的地之一。凭借着低税率、优质教育、高安全指数等优势，仅在2023年一年，新加坡就涌入了来自全世界各地的3400多名百万富翁。

另外，香港凭借15.49万名百万富翁稳居全球第八。

同时，过去十年，百万富翁增长率最快的两个城市也来自咱们中国。没错，就是深圳和杭州。其中，深圳以142%的增速位列第一，杭州以108%的增速位列第二。报告还预测，未来10年，深圳和杭州这两座城市的百万富翁人口还将增长100%以上。

报告还预测，未来，科技将会成为推动财富增长的核心要素。比如，深圳的财富，主要就来自产业链与技术体系之间的长期磨合。在深圳，创业者往往能够带着技术积累与人脉资源，快速组建新公司，然后沿着硬件、算法或产业服务链持续扩大规模。而这些富人的财富未来还有巨大的升值空间，因为他们会凭借先前的实战经验，持续下注，赢得下一个风口。

再比如，杭州。作为移动互联网黄金时代的高地，过去杭州聚集了一大批懂流量、能试错的创业者。而他们的背后，还有大量的运营团队、增长咨询和创意组织等，这就意味着杭州形成了一个自主的创富生态。只要你还在这个生态内部，就有机会创造财富。

新世界财富的研究主管安德鲁·阿莫尔斯还总结了过去十年，创造财富的五类关键行业。

第一类是出口相关的行业。比如，科技和制造业。截止到2024年底，科技行业的富豪占全球千万富翁总数的20%以上。再比如，中国和印度是全球财富增长最快的两个国家，主要也是依靠科技产业和制造业。

第二类是外国投资，主要是股票和房地产市场。比如，2024年，迪拜的房地产交易量同比增长47%，其中外籍人士贡献了超过75%的交易额。

第三类是，离岸银行业务。像毛里求斯、新加坡和瑞士这样的国家拥有庞大的离岸银行业务，这是这些国家外汇收入的主要来源。新加坡作为亚洲最大的离岸财富管理中心，管理的资产规模已经达到3.8万亿美元。

第四类是，酒店和野生动物园旅馆。比如，豪华酒店是佛罗里达、法国、希腊和意大利等热门旅游地区重要的外汇来源，而豪华野生动物园行业是非洲外汇收入的关键来源之一。

第五类是，电影和电视。根据新世界财富公司的统计，截止到2024年12月，全球电影产业创造了4800位百万富翁和35位亿万富翁，其中包括制片人、演员以及主要制作公司和流媒体服务的执行人员和创始人等。

那么，前面说的这些趋势，跟咱们普通人有什么关系呢？这就要提到摩根士丹利公司的首席战略师，鲁奇尔·夏尔马的观点。他认为，对经济来说，富豪榜上不断出现新面孔是个好兆头，但前提是他们都属于“好富翁”。

按照鲁奇尔·夏尔马的标准，做大蛋糕，创造资源的，属于“好富翁”。而从不做大蛋糕，只专注于瓜分资源的，属于“坏富翁”。

首先，直观来看，富豪变多，对城市的发展来说是好消息。这背后的逻辑很简单，就是财富的分配和再分配。富豪的财富会通过税收、捐赠、公共服务的形式再分配给社会当中，然后用来改善公共基础设施、社会福利、教育医疗等。

但注意，长远来看，要确定富人对城市或者国家经济的最终贡献，需要先判断他们是“好富翁”，还是“坏富翁”。怎么区分这两种富翁呢？根据夏尔马的观点，所谓“好富翁”就是企业家型富翁，他们主要依靠技术创新、商业模式突破、产业组织升级等方式积累财富。而“坏富翁”是主要依靠寻租行业发迹的富翁。什么是寻租行业呢？主要指博彩、采矿，以及其他主要依赖开采自然资源的大宗商品行业。

夏尔马认为，这两种富人对一座城市的影响截然不同。

企业家型的富翁，往往会带动整个社会结构的升级。美国经济学家威廉·诺德豪斯曾经说，企业家的颠覆性创新，带来的社会收益远高于个人收益。根据他的计算，企业家本人往往只拿走不到2.2%的价值，其余都进入了消费者福利、社会创新与生产力提升。

比如，像深圳与杭州这种企业家型富翁密集的城市，他们创立的企业往往会带动上下游供应链发展，促进一套系统转型。说白了，他们代表的不不仅仅是一家公司，而是一个生态。

而寻租型富翁往往是在存量资源里获取收益，这些行业的财富往往来自对资源控制，而不是通过技术创新和市场竞争来实现价值。按照鲁奇尔·夏尔马的说法，寻租型富翁主要通过资源垄断、制度套利等方式获得财富。因此，寻租型亿万富翁比例较高的国家，经济增长往往较慢。

比如，2010年，印度寻租型亿万富翁控制的财富占整个国家GDP的12%，在此之后，印度的经济增长速度明显降低。而在2010年到2015年期间，印度的百万富翁增长了53%，其中大多数还是制药、教育、消费品等生产性行业。当时，印度整个国家的财富增长超过22%，被称为因“好富翁”而受益最大的国家之一。

再比如，巴西的富翁主要是依靠自然资源起家，企业型富翁的人数很少。根据夏尔马的说法，巴西企业家型富翁在亿万富翁中所占的比例只有36%，这也导致巴西在新兴市场一直处于落后水平。

巴西本土的经济学家艾德马·巴查曾经造了一个词，叫做“Belindia”。也就是，比利时和印度两个国家名字的合成词。他用这个词来比喻巴西整个国家的财富分布，富裕地区只有比利时的国土那么大，而其他更多地区像印度一样落后。

再比如，据说墨西哥是全世界仇富心理最严重的国家之一。按照夏尔马的说法，墨西哥的富豪大多发迹于垄断行业。他们几乎控制着墨西哥的主要产业，大到电信、混凝土制造等行业，小到电视娱乐和玉米饼的生产，而且他们在赚取垄断利润的同时，不断抬高商品价格。

据说，在墨西哥，很多矿业巨头平时都是深居简出，远离媒体的视线，而且当富豪的保镖是一个很吃香的职业。夏尔马在书中的原话是，这些超级富豪喜欢待在高墙林立、戒备森严的深宅大院，跟那些经常被视为国家象征的亚洲富豪有天壤之别。

根据夏尔马的观点，寻租型的富翁越多，城市的资源就会高度集中。这就导致中产阶层的机会减少，最终演变为社会阶层固化，企业的创新能力下降等趋势，最终会导致城市整体的经济活力下降。

换句话说，一座城市的真正价值，不是由富人的数量决定的，而是由富人的结构决定的。好消息是，未来趋势是，靠寻租致富的亿万富翁的比例将会越来越少。因为随着技术的发展和市场透明度的提高，寻租行为会变得越来越困难，未来的企业会越来越依赖创新来保持竞争力。

最后，回到个人选择，选城市是一次特别重要的决策。在报告的分析中，亨利及合伙人的CEO尤尔格·施泰芬博士说，城市和人才之间是一种共生关系，资本追随人才，人才追随机会，而机会就聚集在这种动态的城市系统中。换句话说，一座城市，可能在很大概率上决定了一个人财运的上限。
