

# 1.16 话题（来自同学）

人生应该是不断向上，通过战略、战术的层面不断解决一个又一个问题，但是对于曾经因眼界、能力不够而造成的不可弥补的遗憾，自己又如何做到自洽呢？

比如学业方面，因为曾经没有努力学习，导致没有考上好的大学，之后错过的很多机遇可能是永远也无法弥补的，对于这个又如何能做到自洽呢？

今天刚好看了一个视频，虽然题主用的是自洽这个字眼，但我觉得这也是一种创伤依恋，包括内容中所用到的“错过”这个词，其实影射了一种习惯的思维模式，都是不利于自身发展的。这个和我个人的情况非常像，一旦有这种思维之后，哪怕是在前进的过程中也会不断的陷入过去的比较和以及过去的经验中所遇到的更加优秀的个体的比较之中从而无形中给自己的现在增加阻力。

同时给题主推荐一本书，或者说是一个思想，那就是阿德勒的心理学，荣格和弗洛伊德之外的鲜为人知的另一位心理学三巨头，有一本通俗读物叫《被讨厌的勇气》。这些小视频只是点缀，真正让我走出之前的思维怪圈的是这本书，目的论而非原因论。

不断的归因无法对解决问题形成实质的帮助，有时候所谓的自洽，其实也是一种害怕承担责任和面对现实的逃避。

所谓机遇和遗憾，也是欲望的一部分，既然错过了，那说明在当时的情况，你不足以能够得到它，时候再去后悔只会徒增烦恼。既然只是自洽的问题，那自己是可以通过心态来调整的，其实是可以通过阿 Q 精神来自我调节的，如当初那个机遇也不足为过，你现在也过得很好，如果接受了当初那个机遇，你现在指不定会如何如何，过得还不如现在。人一定要对自己的所有选择都绝对认可，你要对你自己做的每个决策负责，即便是错了，也要学会接受，或者是尝试通过努力把当初错误的选择变成对的。

题主的痛苦是很典型的，具备当下这个“内卷”时代特色的，即所谓陷入“优绩主义”的陷阱的痛苦。在优绩主义规训下的孩子，未来也很大可能会成长为特别中国式的父母：“我的孩子一步落后就步步落后”了。现在你可能会怪我，将来你会谢我之类的。譬如杨笠说得“普信”，优绩主义的产物。即你只有足够优秀才配云云。当然对优绩主义也要辩证对待。

以前我和朋友会类似的说法，譬如我们比名校毕业的人，起步晚了，譬如同样想进大厂上班的话，人家大厂会去名校秋招，而不会来我们这种垃圾本科。这就是高考没考好的遗憾。但是我们回归到目的本身，就是学习的目的本身是为了自身全面而自由的发展。而现今网络教育的高度发展，这并非进入一个名校一个选项。曲线救国的方案，我可以通过自身优秀的项目经历，甚至做一个明星项目被大厂收购从而进入大厂（当然到那时候，我的眼光就不该仅仅是进入大厂了）。有学院派，自然也有“江湖派”。在互联网圈也有很多出身草根，没有接受过传统教育的创业者，如美图的蔡文胜。检验自身优秀的成分只有一个，那就是市场。

总结下，就是
1. 不要陷入优绩主义的陷阱。
2. 学习是为了自身全面自由的发展，而非外部定价。
3. 可以通过曲线救国的方式弥补。种一棵树最好的时间是十年前，其次就是现在。
4. 人生的容错率，其实并没有那么低。沉迷在已经浪费的时间里，是另一种时间成本。雷军在近40岁觉得一事无成。是因为他觉得自己聪明+勤奋都已经拉满了。但是却被江湖后辈的后来者如马云，马化腾，甚至丁磊超越了。其核心在于自己并没有顺势而为。在下一个风口里，如手机，他是站在风口上的那个人。包括电动车，他其实入局时是后来者，当时如果放眼的敢为天下后的哲学，其好处就是可以吸收先行者的经验。譬如我们摸着美帝和苏修过河。
5. 人生是场马拉松。譬如任弼时和设计师其实是同年人生，而且任弼时早年在党内排序比设计师高，开七大的时候，他是五大书记之一。但是英年早逝了。所以，对80，90世代的人来说，他的影响力就几乎为0了。过度内卷，也可能过度消耗了自己的生命力。

嗯，不要怕走弯路。某种程度上，走弯路是学会走路的一部分，乱花钱也是学会花钱的一部分。这其实也是富家子弟让我们羡慕的地方，因为他们的容错空间比普通人更大。所以普通人应该从小项目做起，小步快跑，迅速迭代地给自己不断积累做事经验和心法。从项目到操盘大项目。

学习有很多途径，大学只是其中一种，切不可因为没有上好的大学而气馁，我们现在来到了一个对爱学习的人来说最好的时代，b站，抖音上那么多大学的优质课程资源都搜得到，你完全可以去读b站博士，重要的是你得是一个真正爱学习的人，知识在于应用不在于学历，在人生的好年华里尽情的去用自己的脑子学习思索，获得内在的提升，才不枉来世间一趟，才能值回票价玩得尽兴。

接下来的话呢，是我的一种个人思考。其实从我们进入这种所谓的内心的内耗的过程当中，我们应该先给自己击个掌，因为你在今天发现昨天的自己是个傻逼，至少证明了一个道理，就是今天的你自己比昨天的你自己更聪明了一点。

而所谓要去做到我们自己的自洽，也就是说像那个道理里面讲的，我们不要怀念过去，因为过去已经成为既定事实，当然要做到没有那么简单，但是我们先庆祝一下我们认识到了过去的自己有多傻逼，那么在这个过程当中如果说我们把自己放在第三者的角度去考虑这件事情，你会发现你自己是在时间这个维度上不断的在进化的。那么每一个进化，我觉得都是值得庆祝的。

第2个，因为他现在已经造成了我们的一种后悔，我们拍大腿后悔，当初做了这个决定，做了那个决定，比如学习的时候不认真，比如遇到的那个男孩子或者女孩子没有好好把握住，那么这些事情如果说你现在从第三方的角度去考虑，其实你当时做的这个决定已经是你当时能够做出来的最好的决定了。那么他现在让你后悔。只是因为你。你进化速度还不够快。所以说在此时此刻更应该提醒自己。现在的自己要加强自己的学习，加深自己的实践。让自己的进化速度更快一点点，那么你就没有时间去后悔当初怎么没有把握住这个或者那个。

就像前两天李总在群里面讲他的侄子。清华博士毕业年薪百万。假设说你现在已经实现了年薪百万，那么你肯定不会拍大腿后悔当初没有认真学习对吧？即便你没有在清华毕业，但是当你实现了和这个清华毕业一样的收入的情况下。你是不会后悔的。

所以此时此刻我们能够做的一件事情，而且这是唯一一件事情就是不断的去加速我们当前的进化速度，因为可能再过个三五年你会拍大腿说今天怎么这么傻逼，没有好好的去做一些事情。

第3个，我觉得有助于实现自己的自洽的一种动作，就是把那些时不时会在自己脑海中浮现出来的，当时怎么就不咋地咋地这一件事情一件件的列在一张a4纸上面，当你把它画在纸上的时候，比他时时刻刻萦绕在自己的脑海中，会更加的让你清晰。当然这个清晰之后，因为你做了一个记忆的转移存储的动作，那么你的大脑在潜意识的角度会逐步的去放下，时时刻刻萦绕在你脑海当中的那些后悔，从而让你的此时此刻实现更多的自洽。

第4个点也相当于为你打一个鸡血。在毛主席《毛选》的第1篇文章当中，讲到了小资产阶级当中的第3类，他是这样子说的。

第三部分是生活下降的。这一部分人好些大概原先是指所谓诚实人家，渐渐变得仅仅可以保住，渐渐变得生活下降了。他们每逢年终结账一次，就吃惊一次，说：“咳，又亏了！”这种人因为他们过去过着好日子，后来逐年下降，负债渐多，渐次过着凄凉的日子，“瞻念前途，不寒而栗”。这种人在精神上感觉的痛苦很大，因为他们有一个从前和现在相反的比较。这种人在革命运动中颇要紧，是一个数量不小的群众，是小资产阶级的左翼。

那么其实这个不自洽的话呢，就和毛主席讲到这个“瞻念前途，不寒而栗”这件事情是差不多的了，那么从这个角度来讲的话呢，你看一下，其实毛主席是很欢迎这一部分小资产阶级的左翼的，那么在广义的社会革命已经完成的情况下，这一部分人其实是在小资产阶级里面最具有革命性的。那么对于自己的自我革命这件事情，你要相信毛主席对你的判断，你是具有革命性的，你是可以进化的，加油。

然后第5个点我觉得也是一个非常重要的点，而且是一个非观念，是物质上的一个点。那就是我们试讲，我们要不断的去实现我们阶层的突破，但是其实向上突围，它是一个循序渐进的过程，你没有办法一步跨三个台阶，我们讲的“步子跨的太大，容易扯着蛋”，其实就是这个道理。

那么这个时候你就需要理解自己的客观状况，到底是一个什么样情况，然后我们的实践不管你通过哪一种渠道去不断的实现自己的上升，它是有一个对比的参照物的。可以对比的是你平行的也可以对比是你过往的自己。如果说你一直对这个参照物选的不太准确，那么你就很容易实现你自己的不自洽。而要实现自洽，你就要把自己的参照物选相对符合你的客观条件。

因为其实向上实现突破的这种过程当中，以我们自己的生命来说，也可以是由多少个年头划分一个阶段的，如果说你的每一个阶段都是在实现一种向上的增长，那么其实你没有必要太过于去说自己是多么的痛苦。而如果说你从自己的家庭代际传承当中来讲，如果说你实现了你这一代人应该担负起来的责任，那么从这个角度来讲，你的向上突破就很值得鼓励，很值得被尊重，很值得被赞扬。所以你更加无需痛苦。

第6个点的话呢。是我自己的一种看法。我们都知道，培养自信最简单的方法就是你能够实现不断的胜利，能够从一个胜利走向另外一个胜利，那么在这个过程当中，你不需要做什么你都能积累起自己的自信，但是其实还有一种自信是来自于文化的自信。就像毛主席诗词当中讲到的。惜秦皇汉武，略输文采。唐宗宋祖，稍逊风骚。一代天骄成吉思汗，只识弯弓射大雕。

这种文化自信需要你有很深刻的积累，因为其实写这首诗的毛主席，那个时候只是刚刚把红军带到陕北，但是这首诗透露出来的一种意境是已经定鼎天下。

总结来讲，其实要做到自己的自洽，可能还会有很多很多的办法，但是我觉得以上的6个点做到任何一个点都会让你的自洽实现的更加快一点点。然后其实还有一个最直接可以实现自洽的办法就是。用重要的事情去填满你的时间，那样子你就没有空在那里伤春悲秋了。

当然最后，作为一个健康管理师以职业病的这种角度去谈，要实现这个自己的自洽让自己更加的快乐这件事情，其实除了这些理论的方法，还有一些可以从营养学的角度去进行一些补充，就像《消失的多巴胺》这本书里面讲到的一些一样，快感缺失或者说我们讲的逻辑难以自洽，内耗这些东西其实在营养上面也是可以去寻找一些方案进行解决的。

一切都是最好的安排

哈哈哈，普通人的生活哪有什么战略、战术可言，脚踏实地地走好每一步就是我们的现实生活。期望越高，失落越重。被所谓的心灵鸡汤洗脑了。醒醒吧，同学！

为啥要自洽？自洽了就封闭了，封闭了就熵增了，熵增就是反生命意志的，就是走向死亡。追求自洽和追求财务自由一样，都是资本洗脑的话术。不应该询问如何自洽，应该询问自己，明天要成为什么样的人，今天该做什么，然后去做就行了。
