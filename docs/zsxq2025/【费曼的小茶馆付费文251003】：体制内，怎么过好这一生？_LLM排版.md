
# 体制内，怎么过好这一生？

原创：湾区费曼 公众号：费曼的小茶馆 25年10月3日

## 前言
写系列文章，帮助你变得情商更高，在体制内活得更自如。
对于大部分人来说，进入体制，其实就是等于选择了在体制内过一生。
因为体制内辞职的人是很少的。
但现实中，想辞职的人却不少。
因为确实是，在体制内，大部分人都是过着一眼望到头的生活。
很可能你现在干的事情，到你快退休了，你依然在做。
只不过，你年轻的时候是做具体工作的，现在成了派活儿的小领导，就这两个区别罢了。

## 《肖申克的救赎》里有一句经典的台词：
人一旦体制化了，那一生都会体制化。

对于体制内的人来说，一生都是跟单位高度绑定了。
好像一眼能看到头的生活，也没有多大意思。
很多工作，干的时间长了，也失去了新鲜感。
所以，等你安稳下来，又有闲暇的时候，你可能会想：
我在单位里上班，到底是为了啥?
我的一生究竟应该怎么过?
怎么过好这一生，才是值得的?

今天，费曼就跟你说说，在体制内怎么过好这一生，才是值得的。
当然，在此之前，我要说说在体制内怎么过，才是不值得的：

第一，力工思维。
最近网上有一个词很火，就叫力工思维。
意思就是，你为了某个传统价值观，然后自己辛辛苦苦打工上班，付出一生。
其实，体制内也有力工思维的现象。
就是有的人，拼命争取提拔，就是为了家人孩子。
他自己天天吃苦加班，挖空心思争取提拔，想方设法攀关系。
目的就是让家庭家族更有依靠。
其实，这就是力工思维。
当时，我看到这个词的时候，我就觉得，用在单位的领导，实在太合适了。他们省吃俭用，最后拿着大把大把的钱去给孩子留学。
但其实，如果他们能换个思路，不热衷提拔，而是多关心孩子成长，我相信，这比耗费自己大半生的积蓄去供孩子留学要好。
其实，费力供养这种教育方式，不一定会让孩子感恩，反而会导致孩子嫌你给得不够。
甚至，有一些领导为了手头更宽松点，突破了红线，最后身陷囹圄。属于真正走错了体制内的一生。
所以，体制内想过好一生，不能有力工思维。

第二，攀比思维。
体制内的攀比思维，也是最无聊的。
一叶障目，不见泰山。
很多人因为攀比，比提拔，最后眼界越比越窄。
但很多人就是热衷于此而不自知。
因为你陷入了攀比，你的注意力就浪费在上面。
不是说提拔不好，而是说，你不能因为攀比焦虑，而丧失了关心学习生活成长的能量。
有些人就是喜欢盯着别人的生活。
别人又有什么事啦，又有什么八卦啦。
虽然喜欢八卦是人的本性，但不能被八卦占据了心智。
你的心智，应该留给你的生活。
如果你说你就是喜欢攀比，攀比就是你的生活，那我无话可说。

第三，价值单一。
说到攀比，就不得不说单位里人群的价值单一。
什么是价值单一。
就是唯提拔。
其实，因为在体制内，如果唯提拔，往往到最后你都不会开心。因为总有年龄比你小的，比你提拔快的。
你也总会遇到提拔的天花板。
昨天有一位读者跟我说，他多次提拔落选，最终想明白，没必要纠结不可能的任务。
现在他的想法就是，多渠道开源节流，做点投资、做点副业，尽可能在退休钱攒够一大笔钱。
其实，说这一点，不是说人一定要为权或为钱而活。
我认同，这也是体制内奋斗的一条路子。
如果你在体制内感到百无聊赖，你可以多留意，看看有什么事情，是让你好奇而又愉悦的。
以前我在读英文原版《万物简史》的时候，有一个故事印象很深刻。
澳大利亚有一位牧师，白天上班，晚上在家用天文望远镜找爆发的恒星，他长期钻研，有自己的方法，而且找到了很多，成果不输研究院。
他说，找这东西，是他一生最快乐的事情。

至于怎么过好体制内的一生，只要我们排除不应该怎么过一生就可以了。
我觉得，只要你不是力工思维，你是为自己而活，你就能找到你的热爱。
你不需要因为周围的攀比而焦虑，同时也要看到在上班之外，人生依然有很多可能。
在体制内的一生，没有标准答案。
如果非要有标准，那就是你选择让你开心愉悦的那条道路。
有这种感觉的一生，才是值得一过的人生。
