当然，中国的这一系列操作，是建立在实力的基础上，能拼命的前提，还是有足够的力量和肌肉能够跟对方打几个照面，场面上不能输，才能起到震慑作用。目前这一轮下来，川普有两个调整，首先是对75个国家暂时豁免了关税，其次是对包括电脑、服务器、半导体零部件、智能手机、GPU、路由器、液晶显示器等部件豁免关税，其中包括对美实施对等关税的中国也在豁免之列。这意味着中国对美关税不减的情况下，美国还是松开了对华产品的关税，原本是互殴，结果现在变成被打了。

而且在这次豁免之前，美国海关成为挡在川总宏图伟业上的巨大障碍，面对陡然上调，反复调整的关税，美国海关用死机的方式死给你看。4月11日，美国海关突然报告称，用于免除货运关税的系统出现了故障，受影响的范围包括所有处于90天对等关税暂停期内的国家的货物。美国海关称，他们发现用于申请货物免税的入境代码无法使用，而在这个问题得到解决前，美国政府将暂时无法征收关税。很难确定到底是真出了故障，还是相关部门以这种方式在抵制川普靠谱的关税政策，要知道突然暴增如此多的关税，势必引起清关系统的极大混乱，工作量会成倍的增长，对于川普这种反复无常的政令，包括美国执行部门都不得不以各种方式来应对，包括消极应对。对于海关人员来说，你发个社交推文，落实到下面就是成倍的工作量，所有的系统都要重新调整，最怕的是我调整完了，你特么的又搞赦免了，你这是在与中国对抗，还是在折腾自己人，根据公开数据，美国海关雇员一共有5万多人，今年因马斯克的效率部可能还裁撤了一些。他们每天要面对数百万甚至数千万进入美国的商品。

为什么可以利用小额免税的制度向美国蚂蚁搬家销售低价产品，实际上就是因为海关的执行成本太高，所以不得不放行。换句话说，美国海关不可能逐一检查每批进口商品都是否合规。前任拜登的策略，是将合规的压力转嫁给企业，要求美国企业做好供应链的合规检查，政府只在一些特殊情况下来出面监管。拜登当时特意留下了东南亚和墨西哥这两个漏洞，美国的经销商和客户会要求中国供应商将转移产能放到这些国家，来符合美国政府的合规要求。如今川普一刀切，美国海关的执法压力就会陡然猛增，执法本身是个技术活，如果不能执法到位，则法的约束力就会大幅降低，实际上很多时候，政府的公信力就体现在执法能力上，有些法规之所以没有足够严格，不是不想而是不能，在执法成本和法律的约束力之间做平衡。如果只对中国商品加关税，这个具体执行要如何才能有效实现，是需要机制和技术手段执行落地的，不是川总一句话马上就能做到的。实际上，4月2日川总宣布对等关税后，美国联邦海关就加班加点地工作，但估计还没有折腾完，又说不搞了，暂停90天。一来一回，能力本就有限的美国海关只能先死机，看看再说了。

更进一步看。美国海关要应对特朗普如此规模的关税，必须需要大幅加强产品分类识别和原产地验证能力才行。比如一箱零件，医疗设备零件和汽车零件税率差别很大，原产地是越南和原产地是巴西税率差别也很大。显然海关不能只看单据就放行，必须经过现场检查货物，核对原产地起运信息，请求货物出口国的有关机构对货物的原产地进行核查。如果发现有靠虚假信息进行避税的嫌疑，还要要求出口商提供一系列发票和付款信息进行核实。而美国如今对原产地的认定标准很模糊，一辆大幅采用中国零件在墨西哥组装的汽车，原产地到底算哪里?美国自己也说不清楚。这需要美国海关机构，尤其是负责以上事宜的海关边境保护局研发部署新的关税计算核对系统，通过AI等手段能快速识别确认货物分类，通过一系列对原产地认定的细化标准和指导操作，这是一个庞大的系统升级和专业操作，要升级这套系统需要提前准备才能确保关税令执行后能够有效执法，否则就漏成筛子一样，然则美国海关边境保护局本财年的预算早就确定了，主要花在边境安全和芬太尼上，而不是加强关税征收能力上，从财政预算上就压根没做大规模加关税的准备，一线根本不具备立刻执行的能力，如果在这种情况下强行推动广泛加税，后果只能是大量货物在海关排队，靠现有系统慢慢识别核对再放行，进而导致美国国内商品流通大幅延期，甚至出现物流的全面瘫痪，大量船只的货品不能卸载堵在港口。从这个角度看，川普政府本身也并没有做好跟中国全面打关税战的硬件准备，可以说是枪炮都还没有运抵现场，就开始吹集结冲锋号，最终败阵而归也是必然的。从这个角度来看，川普政府的执政手法不但粗糙甚至潦草，完全是业余级别，对于国家治理的一些基本操作和流程都没有基本常识，跟这样的对手，真的没法讲道理，只能用归谬法让错误急剧堆积产生立竿见影的效果，让对方知难而退。
