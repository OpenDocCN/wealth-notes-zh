
# 创新药大牛市

拉银行的目的是搭出指数台子，让题材好唱戏激活市场，作为港股的映射，大A的创新药是“补涨”，新消费也是如此。牛奶饮料是跟着蜜雪冰城补涨，各种化妆品谷子经济上是跟着泡泡玛特补，黄金饰品是跟着老铺黄金补，饰品之前的龙头是莱绅通灵，它起来后给大家讲了曼卡龙。

鉴于，各种药和新消费的题材都起的很猛了，创新药可以看看后排，比如之前讲的易方，或者干脆看中药， Ai 制药。新消费可以看看演唱会、游乐园(金马跌入MACD 绿轴了)。这几个都是后排，之前也不猛，能突突一下得撤，谁叫之前没有拿前排呢。请注意，这个当口，他们都不能跌破20日线，5日线跌破10日线一定要跑，甚至保守一点的5日平台都不能破。紧盯港股，如果港股消费崩科技兴就要及时转换风格。
