
# 卢助理播报+青云话题5.17

## 国际：

1. 美国总统特朗普周五接受美媒采访表示，他愿意访问中国，与中国国家主席面对面讨论外交政策和经济问题。
2. 当地时间16日，美联储主席杰鲍威尔向员工发送的一份备忘录显示，美联储计划在未来几年内裁减约10%的员工。
3. 美国史密森学会国立亚洲艺术博物馆16日正式向中国国家文物局返还战国时期楚帛书《五行令》《攻守占》。

4. 乌克兰国防部长乌梅罗夫16日宣布，俄罗斯与乌克兰当天在土耳其伊斯坦布尔的直接谈判已经结束，双方已同意进行一次战俘交换，将各自释放1000名被俘人员。
5. 日本内阁府16日发布的初步统计结果显示，经季节调整后，今年第一季度日本实际国内生产总值（GDP）环比下降0.2%，按年率计算下降0.7%。
6. 据韩联社报道，韩国前总统尹锡悦17日宣布退出国民力量党，并呼吁选民支持该党总统候选人金文洙。

8. 据中央纪委国家监委网站5月16日消息，广西壮族自治区党委副书记、自治区政府主席蓝天立涉嫌严重违纪违法，目前正接受中央纪委国家监委纪律审查和监察调查。
