
# 好好一个人，为什么一读书就困了？对学渣、学霸与大师的大脑分析

互联网上，总不缺有趣的东西。我曾经读到过一篇很有趣的文章，叫《好好的
人，怎么一读书就困了?》。文章用搞笑但并未深入解释的方式描述了一种在日常生活中屡见不鲜的现象：同样是读书，甚至是阅读同一本书，学渣和学霸的反应截然不同。

学渣翻开一本书的时候，也是下定了决心要好好
读书，认真读书，为中华之崛起而读书的。书中
自有颜如玉书中自有黄金屋，古人凿壁偷光程门
立雪，学渣可是给自己做了心理工作的，结果
读了几页书之后，读了还不到一个番茄时间，好
好一个人，不知怎么就困了，眼皮打架了，大脑
累了，脑子里昏昏沉沉像被塞了棉花，书本上的
文字很快就变了索然无味，甚至面目可憎，读不
下去啦。然后，手机上恰好推送了一条消息，点
开一看，哈哈哈，好有趣……不知不觉， 一刷手
机就是好几个番茄。

对比之下，学霸读书不但不会困了累了，反而越
读越精神，读的津津有味，乐不可支。学霸手里
的书越来越有趣….不知不觉中，学霸就把厚厚
一本书看了大半。而且，学霸读书之后，博闻强
识，读完还津津有味给别人分享，妙趣横生，发
人深思。学霸可能还会忍不住写篇文章来分享分
享，不小心又是洋洋洒洒一篇万字雄文。

不止是读书，其他学习活动，甚至广泛的认知活
动，凡是靠脑子的活动，都有这样的现象：都是
写作业，学渣一坐到书桌前就东摸摸西看看， 一
会肚子饿了要吃点零食， 一会嘴巴渴了要喝水，
一会又要上厕所，于是写个作业经常都要折腾到
半夜。所谓”懒驴上磨屎尿多“,学渣们总是不知
不觉就切入“懒驴模式”。但是，学霸写作业干脆
利落，生龙活虎，三下五除二，总是自然而然地
表现为“猛虎模式”。

这种现象，问题出在哪里?本质原因是什么?

问题出在脑子上，本质是脑子里有个部分的结构
在物质层面有差异。本书的这个部分，就是要解
开这个世纪谜题，给出这个怪病的终极治疗方
案。

## 学霸不靠思考，学渣才靠思考

随着脑科学的进步，科学家已经可以用fMRI来看到学渣学霸在读书学习时大脑的
差异了：

在读书的时候，学霸是“不动脑子”的。
这很“反常识”,但事实就是如此：相对于学渣，
学霸在读书时大脑神经活动更低，对应的就是读
书这个活动给学霸大脑带来的认知负荷
更低，更节能。所以，学霸
读书时就如同猛虎下山，有摧枯拉朽之势。学霸
的读书体验是愉悦的、轻松的、快乐的。

在学渣在读书时，大脑被迫“开足马力”。
翻开一页书，每个字都认识，但由于之前积累的
事实性知识和抽象概念量太少，频繁遇到不熟悉
的知识和概念，于是，这些都认识的字连在一起
就不“认识”了。因为理解是基于这些概念的，学
渣得不断去重头理解这些概念，甚至要大量查阅
来辅助这个理解过程，需要不断去编码这些新
知。所以，学渣读书特别“费脑子”。

而大脑堪称人体器官中的“电老虎”,只占身体
2%重量，却要消耗人体20%的能量，也就是
说，大脑的功率是人体平均功率的10倍!而动
用前额叶去主动的、深度的、有意义地加工处理
信息，这是大脑最耗费能量的活动。所以，学渣
在读书学习时很快就饿了累了困了。

对比之下，不涉及学习脑，只需让动物脑接受感
官信息的刺激和撩拨，那是几乎不耗能的。所
以，懒驴学渣一拿起手机就能秒变猛虎。

人与人的差别千万种，有一种是懒驴和猛虎之间
的差别。猛虎学霸的大脑，存储着自己的个人知
识体系。这个知识体系包含数以万计的知识砖
块，而且这些砖块之间建立了丰富的、有意义的
联结。这些知识砖块以及关联，构成了个人知识
体系，体现为一个有序的、高效的大脑神经网络。学霸在学习时，不需要“现挂”,而是从自己
的知识体系中“提取”已知。相对来说，“思考”的
比重更低，但是质量更高，效果更好。

对比之下，学渣因为长期欠债太多，缺乏知识体
系的积累，于是被迫“思考”,导致“思考”的比重
更高，但是质量更低，效果更差。学渣之所以是
学渣，不是因为他没有猛虎之心，不想细嗅蔷
薇，而是他的神经网络没有存货积累，稀疏的知
识砖块之间的联结非但不丰富而且杂乱，于是，
懒驴只能吃的下低质量的干草，要他吃高质量的
精神食粮，肠胃就不习惯。

由此，我们可以透过现象看本质。表面现象：都
是人，有人一看书就困，有人不这样。但本质原
因是：学霸不靠思考，学渣才靠思考!

学霸不靠思考，学霸靠脑子里的存货；学渣脑子
里没有存货，只能现挂，遇到什么概念都要现
学。而这个过程是非常耗费认知资源的，是天底
下最耗费能量的事情。于是学渣一学习就困了累
了。

## ”不思考”是为了更高质量的思考

现实世界里，人们往往不了解学习的本质，对大
脑原理和学习的认知过程近乎无知，于是往往落
入“重思维而轻知识”的陷阱。例如，网络上有很
多“认知吹鼓手”都表示要帮你升级认知，打开格
局，但是极少有人帮你构建个人知识体系。

事实上，到处都是相反的案例，提醒我们知识体
系才是重点，而非所谓认知升级。

举个下围棋的例子。人们觉得下围棋的关键是
“思考”,类似三国故事里那些动不动就“我有一
计”的谋士。一个小孩学习下围棋，父母的“指
导”往往就是“你得认真思考”。父母脑袋里的场
景往往是小孩子愁眉紧锁，冥思苦想，忽然灵光
闪现，用神之一手把对手打到一败涂地。

但是，棋类游戏都有一个经典活动：围棋大师以
一当十，甚至同时和100人对弈。100张棋盘在
体育场摆开，一人面前摆一盘，大师像散步一
样，边走边看边落子。走上几十圈之后，100个
对手或早或晚，陆陆续续全部败下阵来。

围棋大师对战百人，就像学霸读书，都是猛虎下
山的气势，原理也是一样：学霸不靠思考，大师
也不靠思考。

并不是说大师和学霸“不思考”,而是说他们不靠
“现场思考”,更多靠”提取“,从脑子里已经存储
的知识和经验中快速识别并提取模式，然后直接
应用，现场解决问题。具体来说，这个认知过程
是：输入信息，识别/再认模式，提取知识，决
策，解决问题。眨眼之间，如电光火石，走完一
个闭环。

在从新手成长为大师的学习之旅中，围棋新手通
过长期练习，将常见棋局和策略转化成知识组
块，存储在长时记忆里。研究表明，围棋大师可以
在大脑中积累数十万个知识组块。这些组块不仅
数量庞大，而且具有高度复杂性，涵盖了各种可
能出现的棋局和战术。在对弈过程中，大师能够
迅速从长时记忆中检索出适合当前棋局的信息，
快速做出决策。这种能力，是通过长期刻意练习
和对围棋的深入理解而形成的。

当然，学霸和大师们并非在思考上懒惰，而是因
为人类都是“认知吝啬鬼”(cognitive miser),在
本能上都会节约认知资源，用最高效的方式用脑
子。当学霸和大师面对难题、真正思考的时候，
因为他们脑子里存有海量的知识组块，而且有序
地组织为个人知识体系，他们的思考是高层次的
(high-level), 更高屋建瓴，更创新的。

大师们解决普通问题，只是猛虎模式。但面对真
正的难题与挑战，他们爆发出来的思考能力，那
简直是哥斯拉级别的，往往会得到普通人类颁发
的诺贝尔奖、图灵奖、菲尔茨奖章等以示崇拜之
心 。

## 学渣 学霸 大师

大师的“不思考”,是为了更高质量的思考，为了
更高层次的概念性思考。

## 问题不在于“认知升级”

各个领域都是如此：新手和大师在知识体系上存
在鸿沟级别的差距。所以，他们“看到的”不一
样，思考模式也不一样。例如，面对同一道物理
学题目，新手学生和物理学家看到的东西是不一
样的。

同一个斜面问题，新手关注到的是表面特征，而
专家脑子里建立的思维模型却能够把斜面的概念
与物理原理、原理的应用条件关联起来。

新手倾向于按照解决方法的相似性、表面雷同来
将物理问题分类，解决问题时往往停在相关领域
中事实性知识和公式的罗列，止步于what 和
how 层面的思考。

专家则根据可以应用来解决问题的主要原理来分
类。专家能看到底层的概念、关键思想，通过概
念和思想来引导自己去深入思考问题，能够回答
为什么要把这些定理和概念用在这个问题上，能
回答why 问题。

那么,新手是否可以通过认知升级、打开格局，
学会专家的思考方式?取得专家级别的表现?

很明显不能。人与人之间的根本差异不在于“认
知升级”,而在于个人知识体系。
