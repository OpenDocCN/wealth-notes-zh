
# 谈谈宗庆后财产分割的看法

## 非婚生子女权利

早在1950年《婚姻法》第十五条，就已经明文规定非婚生子女权利等同婚生子女；2021年《民法典》第1127条明确非婚生子女同为第一顺序继承人。在全世界，中国基本是最早明确非婚生子女权利等同婚生的国家。比如欧美是在社会主义压力下，才开始承认非婚生子女的普遍人权，比如美国1973年《统一父母身份法案》取消“非婚生子女”概念，德国1998年取消婚生与非婚生子女差异，日本则是到2013年才由最高法裁定“非婚生子女继承份额减半”违宪，宣告婚生与非婚生平权。

可以说，非婚生子女享有婚生子女一样权利，这是中国在人权领域的首创，究其原因，是出于社会主义人人平等的朴素理想，也是为了解决旧中国存在的一系列问题；之所以后来全球跟进，则是从西方称“自然法”，中国称“天理良心”，也就是朴素的道德观角度出发，但注意，道德并不是固定的，而是随着时代社会发展而变。

现在的问题在于，围绕非婚生平权方面，出现了封建时代道德观和现代社会道德观之间的冲突，近几年随着“嫡庶”文学的泛滥，出现了某些道德观的倒退，在人人平等方面，还不如20世纪50年代社会大改造时期。

需要强调，非婚生子女，犯错的是出轨的父母，而不是孩子，就算是强奸犯的孩子还得平等保障，更何况出轨，这是人生来就有的基本人权，现代法律要保护更具备普世意义的基本人权。

婚姻、出轨、继承权，这三者不是一回事，不能混为一谈。保障非婚生子女的继承权等基本人权，和鼓励婚姻忠诚反对出轨，没有任何矛盾，保护非婚生子女的天然继承权，并不等于鼓励背叛婚姻；这就像不能因为一个人犯罪，就杀了与此无关的孩子。

更不用说，非婚生子女不一定就是出轨导致的，现实中有非常多的原因会导致出现非婚生。比如未婚没领证就先生了孩子，还有的本来约好了要结婚，但有一方去世了或者分手了，孩子选择生了下来，之后又选择与其他人结婚，那么之前的孩子就要被歧视？没有这个道理。

关于有三个非婚生子女来看人性的本质，与我们当代追求的道德底线。人欲、事业、道德该如何把握平衡？

这个问题就把几个不同的问题混在了一起，出轨或许是人欲的一种，但绝非欲望的全部，固然成功人士确实有性欲望强烈且控制不住自己的，但多数成功者还是有很强的自控能力；至于道德，要看什么时代、什么环境的道德。

比如中国人习惯的一夫一妻制，实际上是人类文明史上极其特殊的制度安排，在中国只有70多年历史而已，而现在也开始与现实出现了一些摩擦。就不一概而论了。

## 将宗庆后树为先进模范，是否扭曲人性

过去老是强调先进模范如圣人一般不食人间烟火，这确实与现实偏差比较大，普通人可以仰望，但难以跟随。不过宗庆后这个先进，显然顺应人性。

要注意分阶段、分环境，不宜有圣人无瑕的想法。宗庆后当然是模范，他把一个已经濒临破产的校办小厂壮大到今天的水平，功劳很大。的确在这个过程中，宗家取得了丰厚的家族财富，但这就是时代的历史遗留问题，也是人性使然。

当年社会主要矛盾是1981年确立的“人民日益增长的物质文化需要同落后的社会生产之间的矛盾”，最重要的就是先把经济搞起来，而当时中国已经搞了几十年国有，根本没有成规模的私人经济，国有改制是必然。中国第一代民营企业家的原始积累，或多或少与国有资产有关系。那么过了当年的时代，难道就要以此为由把人都抓起来？这不就成了过河拆桥了？这么干，以后还如何取信于民，商鞅都尚且知道“徒木为信”，至少也得学学赵匡胤的“杯酒释兵权”吧？

当然，如果非得按照法律来讲，确实可以抓也应该抓，但事情不是这么算的，国家的发展和治理需要有所取舍，并非必须严格按照法律去执行。有些过线的要抓要惩，但有些该放过还是要放过，要承认历史贡献。

娃哈哈前身不过校办小工厂，那时候这种工厂多如牛毛，为什么偏偏宗庆后领导的娃哈哈发展起来了呢？还有1991年兼并杭州罐头厂，当时这个厂有2000多名员工，产能和产线基本完整，但产品没有市场竞争力，已经处于重亏损状态，是当地政府找到娃哈哈希望把这个厂子接过去，避免2000多员工下岗。

那么，娃哈哈兼并杭州罐头厂，是政府甩包袱，还是娃哈哈侵吞国有资产？这事说得清吗？

现在娃哈哈还在创造海量就业，而且可能是因为宗庆后很清楚自己的做法合情合理但是不合法，所以在劳动者权益保护方面做得很好，娃哈哈的员工待遇是真的不错，超越多数国企水平。社会保障等完全按照法律规定，多数员工准时下班，很少加班，更别说什么996、35岁优化之类的糟粕。

在娃哈哈干到退休不是啥稀奇事。在这些合法保障的基础上，不仅搞了福利分房，还因为起家于校办工厂，给学校所在的杭州上城区教师发补贴，林林总总太多了。

宗庆后从资产上讲，当然是大企业家甚至是大资本家，但他毕竟出生于1945年，经历过新旧两个社会，在行事做法上，更倾向于改开之前的国企大家长。至于出轨、非婚生子，说到底这是个人私德。这样的企业，难道非得因为法律说了，所以就要抓、要罚、要收归国有？过去是有这种教训的，比如健力宝、酒鬼酒，也都是从破破烂烂的国营厂开始的，当年都很辉煌，一个曾经打败过可口可乐，一个1997年在深交所上市了，次年超过茅台，位列全国第二。

之后如何呢？健力宝想着换总部更好发展，结果三水非要夺回来，结果因为不善于经营，被迫卖掉，新买家也不懂经营，最后破产了；酒鬼比健力宝好点，当地看上了酒鬼的效益，以关爱为名，大量安插官员，当然还有其他复杂原因，但政府的盲目干预，企业失去灵魂，是很关键的原因。

幸亏后来中皇收了酒鬼，抓住了救命稻草，却又出现塑化剂危机错失黄金发展期，直到中粮收购中皇母公司后才缓过来了。

那个时代有太多遗憾的例子，比如临朐秦池、三株口服液等等，火得一塌糊涂，又转瞬即逝。

因此，站在杭州国资的角度，虽然说从账面看，分红少到几乎没有，但国资最看重的不是分红，最起码国资保值增值了，提供了大量就业和税收，在宗庆后领导下也非常圆满地达成了包括商业、社会价值等在内的种种发展目标，兼顾国有和私有的优点，成为混合所有制改革的样板。

既然是混合所有制，就不能禁止企业的核心经营者不去获得相应的回报，否则那还搞什么改革？

仅仅因为这些，就足够成为一个时代的典范。对杭州乃至浙江而言，这也是一个促进投资的金字招牌，难道真的要翻旧账？还是要讲新人老人，讲既往不咎。

2012年中国社会主要矛盾已经转化为“人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾”，之后国家开始搞五年脱贫攻坚和共同富裕，就意味着，国家从上个时代的“允许一部分人先富起来”，进入“先富带动后富”的新时代。

从这个意义上讲，国家给宗庆后极高的死后美誉，也是为一个时代盖棺论定，安定一个时代的人心，但同时，也意味着要开启一个新的时代。也正因为此，可以看到近几年，有名的大企业、大老板都在强调社会责任，要搞新风尚，配合国家实现“共同富裕”，有些还不清醒的也受到了教训。

所以，把宗庆后树为先进模范，恰恰没有扭曲人性。不仅承认了宗庆后拿走的财产，也默许可以留给自己的后代，甚至包括非婚生子女等等，这是一种很现实的理性做法。

然而，宗大小姐把这一切撕破了，体面就没了，现在宗家还在争执，至于后面怎么做，就且看吧，不多讨论了。

如果有人因此就觉得宣传虚伪、脱离实际，小镇觉得这一方面没有看到国家一直以来的努力，也没有看到最近10年，全社会生活水平都在提高，后10%提升尤其快；另一方面也过于天真了，陷入非此即彼、非黑即白，但有多少人，不会希望自家有个当大官的亲戚，违法违规拉自己一把？这或许也是双标，恨自己没有而已。

## 什么算成功

成功按照最俗的说法就是成为人上人，有钱有权，不就是这样吗？成功大小，无非取决于爬到多少人头上的差异而已。

但成功真的就这么单薄？人生几十年、多则百年，最终不过黄土一杯。

有的爬得挺高，没多少年就死了，算成功吗？

小镇不妄谈妄论，就只说自己。小镇不想成为什么成功人士，只想尽可能实现令自己相对满意的人生经历，在获得觉得足够的物质财富之后，更多向内求。
