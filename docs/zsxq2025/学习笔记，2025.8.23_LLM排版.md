
# 学习笔记，2025.8.23

1. 从清朝起，中文词汇一部分来自于日语，一部分来自于英语，这是从外语角度的，从文献角度呢？一部分来自于佛教，一部分来自于基督教。佛教融入汉语比较早，有千年以上的历史，基督教对我们的影响也就是百年历史，例如我们说的礼拜几、元旦、圣诞，包括医院用的十字标，这都是基督教带来的。电商时代，也一直在创造新词，例如蛇果，听起来很恐怖不？其实蛇果也是音译，Delicious。

2. 台儿庄战役纪念碑，四川商会缅怀滕县保卫战的横幅，这个横幅的牛逼之处是每年都换一次新的，就在滕州最繁华的位置，你哪怕是路过都会肃然起敬。写作，需要日更，你知道为什么日更吗？人可以日复一日的抽烟，但是隔日抽可能吗？习惯是什么？是大宝天天见。

3. 有钱人的世界是彩色的，我们的世界是黑白的，有部电影特别好，但是尺度比较大，你有空可以看看《大佛普拉斯》，整个影片在讲穷人时都是黑白色调的，在讲富人时才变成彩色的。特别是没有在社会底层生活过的人，更应该看看，我是从地瓜堆里爬出来的，很懂这些，例如海边有垃圾村，就是很多人直接生活在垃圾场，以拣垃圾维生，他们也分帮派，也打斗，而且为了抢垃圾地盘女人会主动让男人睡……那是过去吧？不，就是现在。

4. 要学真本事，能在市场上换来钱的，而不是屠龙术，例如你会写PPT，会写公文，会端茶倒水，这些在市场上换不到钱，可以是具体的产品也可以是具体的服务，市场愿意为你买单。

5. 特斯拉，有时我开在路上，我都在想，感谢马斯克，让我们体验到了这么好的产品，原来车子还可以这样，原来智能化可以这么具体，从我开了特斯拉后，再也没有一辆车能让我觉得很上头。有一点，我们四人高度一致，就是看多中国，其实我不仅仅看多中国，还看多A股，中国制造业对全球的冲击可参考手机、电脑、电动汽车，我们有这样的制造业，还担心什么？股市早晚都会跟随经济的。

有些东西，要么早信，要么不信。信，就坚定。不信，也坚定。

6. 大家若是看网上评论，烟花爆竹，觉得禁放是逆行。实际上，禁放才是民心所向。现在，上坟，允许燃放。哪有几个放鞭炮的？大家对这些已经没有兴趣了，反而觉得很危险。我从来没放过，我小的时候，一到上坟，到处着火，现在很少有这种情况发生了，偶尔有人因为放鞭炮而点燃了周围的干草，还会被拘留。矫枉过正的标准是什么？农村，一切都是错的。上海，一切都是对的。按照这个标准来就行了。不要总是恋旧，时刻觉察自己身上的一些习惯，只要是农村带来的，就切割掉。

7. 如果有冲突，被挨揍了，我希望解恨，揍回去。钱，我反而不想要。错，这种事，只有让他赔了钱才是真正的折磨他，想起他就觉得自己不是个东西，以后回家他都要绕路走，生怕路过想起了伤心事。

8. 吃一堑长一智，小错误都是为了避免大错误，一定想着，祸从口出。当年有个很火的案子，成都车娅婷案，她被一个三轮车刮了一下，她骂了一句土话，从而被三轮车主打死了，这里面有个细节，无论是警方还是记者，最终都没写到她到底骂了一句什么？社会上，没人惯着你。一句话，就可以惹来杀身之祸。

9. 我刚看了一个数据，年过40岁，自然受孕率只有5%。

10. 山东人是什么特点？一件小事，反复的琢磨，然后生闷气。四川人呢？嗷嗷一顿，然后，就忘记了。我娘离我们家远，从来不掺和，从而没有婆媳问题。我旁观那些山东婆婆与云贵川媳妇呢？就觉得很有意思。婆婆气的一鼓一鼓的。儿媳妇呢？反而很好奇，咱吵架不是上午的事吗？咋下午还在生气？他们是天生的活在当下。过去的事，马上就翻篇了，前两年，我动不动出面帮我媳妇处理交通事故，要去擦屁股，然后我就非常生气，这么简单的事，为什么搞这么复杂？甚至反复拿出来说，教育她。她早忘了。
