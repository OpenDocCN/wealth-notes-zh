
# 德国，已经选择投中？

最近因为日本、俄乌问题的持续聚焦，导致很多信息被忽略了，其中就包括中德之间的合作事宜。本期内容我们就谈谈这个话题。

在最近1个多月的时间里，德国针对我们，确实蹦跶得很厉害。包括：
表示将在未来的6G网络建设中排除中国供应商，并考虑替换现有5G网络中的中国组件。
考虑对中国钢材加征关税。
讨论针对中国跨境电商小额包裹的税收措施。
还有德国外长错误的涉台言论。
给台独分子沈伯洋提供政治舞台，参加德国联邦议院听证会。

看到上述罗列的内容，大家会怎么想？是不是认为中德关系火药味十足，德国正在作妖的路上越走越远呢？但是，戏剧性的反转来了。

11月17号，中德在北京搞了一场经济金融领域的“碰头会”——第四次中德高级别财金对话。

实际上该对话机制已经存在了10年，前三次分别始于2015年、2019年、2023年，目的是为德国财政部长与我们负责财政的高层会面提供沟通平台。但蹊跷的是，前三次双方都约定俗成的间隔4年进行对话，而这次为啥却仅仅只隔了2年？

个人判断主要涉及3点：
其一，时机很好，现在处于中美休战期，又有俄乌战争调停作为遮挡，很容易排除外部干扰，低调的把事情办了。
其二，德国经济已经连续两年呈现负增长，国内压力很大。10月28日，德国16个联邦州中的13个州的州府市长曾警告总理默茨，称德国的城市、地区和乡镇正面临金融崩溃的威胁。埃森市市长库芬甚至公开表示：“现在几乎每一座德国城市都在破产的边缘”。
所以，默茨政府不可能墨守成规，再等2年，必须尽快搭上东方巨轮。

其三，原本计划德国外长访华，结果这货立场不对，是亲美保守派，来华前还妄议台海问题，吓得默茨紧急叫停访华行程（当然，就算不叫停，也不可能成行)。为了及时纠错，换人和换方式就是最佳选项。
于是才有了号称“中国通”的克林拜尔访华进行中德高级别财金对话。大佬文集圈当然，德国方面也释放出善意，取消对我们的贸易审核。

这次主持对话的，我们是何副总理，德国是副总理兼财政部长克林拜尔，他也成为德国新政府首位访华的高官。
而且，陪同克林拜尔访华的，还包括德国联邦金融监管局局长、德国央行行长、安联保险和Ergo保险公司代表等一票德国金融界大佬。
所以，你看这架势，就说明它不是一场单纯的例行公事对话。

从双方对外发布的联合声明也能佐证，其成果用8个字形容：“意义重大、远超预期”
以至于，克林拜尔在访问结束后，马不停蹄跑去新加坡炫耀成绩，同时顺带给新加坡来了一个“现身说法”（最近新加坡总理关于中日的言论，也让人哗然）。

中德双方一口气达成27项共识。这里挑重点，给大家谈谈所涉及的内容：
其一，金融市场互联互通：
欢迎两国公司到对方股市发行存托凭证（GDR/CDR），相当于开个第二上市渠道。
鼓励德国机构来中国发“熊猫债”，也就是借人民币。
支持在德国发展离岸人民币市场，共同用好中欧国际交易所这个平台。

其二，金融机构与监管合作：
欢迎德国投资者参与中国银行间市场。
欢迎符合条件的德国保险公司来华拓展业务。

其三、绿色与可持续金融：
继续在“央行与监管机构绿色金融网络”等多边框架下深化合作，支持绿色转型，鼓励两国金融机构在绿色债券、气候风险相关产品方面加强合作与联合研究。

其四，全球治理与多边主义：
明确反对单边主义和贸易保护主义，支持以世贸组织为核心的多边贸易体系。承诺在国际货币基金组织、世界银行、亚投行等框架下保持密切合作。

涉及金融方面的内容，不仅专业性强，而且啃起来也枯燥，这里我用更加通俗的归纳就是：
要让两国的钱和投资能更顺畅地流动起来。
要让两国的银行、保险公司等金融机构能在对方国家更公平、更方便地开展业务，同时协调好监管规则。
要一起为节能减排、保护环境这类“绿色事业”找钱、投钱，并制定大家都能理解的规则。
要反对“关起门来自己玩”的单边主义和贸易保护。
一句话描述，中德双方要一起做好生意，管好钱，共同应对全球性的经济挑战。

当然，这其中也少不了我们对稀土和关键原材料方面的松绑。

那么，达成这一系列的共识，背后又意味着什么？
要知道，双边合作发展主要涉及3个层面——“金融、产业、贸易”，其中涉及“钱”的金融投资领域，属于合作的最高等级。和企业合作、项目合作异曲同工。

更直白的说，德国在贸易合作（今年前三季度中国超越美国，成为德国第一大贸易伙伴）、产业合作的基础上，双方在金融领域的合作，已经把口子开得更大了。实际上也就意味着，德国的产业经济正在全方面加深和我们的绑定关系。

甚至个人还有一层理解，就是德国正在向我们进行一场工业“托孤”，把德国工业的未来交到我们手上。为啥这么说？
实际上德国工业向中国转移的事实，早就不是什么秘密。除了巴斯夫、宝马、大众等德国龙头企业主导的大规模投资外，你去江苏太仓的德国工业园区逛一逛，通过一个缩影，就能切身体会到这场产业转移的宏大叙事。它不仅涉及生产基地，而是包括研发中心、区域总部在内的等等。

比如在太仓，就聚集了数百家在细分领域做到极致的德国“隐形冠军”企业，特别是围绕汽车产业，实现了高度的本地化配套，有说法称“要造一辆汽车，七成零部件不出太仓就能配齐”。

大家都清楚，世界正在进入人工智能时代，算力的背后是电力。显然在这方面，我们的优势在全球是独步天下。

现在德国工业优势已经不多了，一旦选择对抗，未来大概率没戏。所以，这时选择和中国产业进行深度捆绑，彻底融入中国经济，是务实态度下的必然选择。

欧洲实际上已经烂透了，既输掉了当下，又输掉了未来，这方面德国企业早就看明白了。所以，不怕德国政客为了迎合国内政治正确怎么怎么样的，但企业的身体还是很诚实。

以前是德国制造卖全球，现在，包括未来是德企在中国制造卖全球。可以说，德国工业4.0计划，在德国失败了，但却用另一种方式在中国成功了。

而这次的27项共识，更像是顺应大势，促进中德金融投资领域融合发展，为产业赋能迈出更坚实的一步。

当然，作为欧洲经济引擎，德国做出了务实选择，彻底押注东方，那么接下来的中欧关系也会相应得到改善。11月27日，法国宣布，马克龙将在12月开启访华行程，虽然我们的态度和2023年已经今非昔比了，但还是让我们拭目以待，看看欧洲另一个大国-法国，会和我们达成怎样的成果。

不管未来还会出现什么地缘政治波动，中德关系能够进一步形成“你中有我，我中有你”的绑定关系，对于整个中欧关系而言都会起到压舱石的作用，这也是我们希望看到的。
