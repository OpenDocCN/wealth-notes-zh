
# 写作的瓶颈，就是赚钱的瓶颈

## 原创：U 君 公众号：U 君笔记 25年8月15日

在互联网上做自媒体、搞知识付费，“日更”是必须的。不论是写文章还是拍视频，你都需要写作素材，然后把它们转化成可以收钱的文案。甚至可以简单粗暴地讲：如果你能日更1000字，你就可以日入1000元，如果你能日更1万字，你就可以日入1万元。当然，这些文字必须是跟营销、成长、心智、思维相关的，不能是文学相关的。也就是说，如果你想突破赚钱的瓶颈，你就必须突破写作的瓶颈。

我总结，一共有5个瓶颈需要突破(内容创作、主观惰性、流量获取、平台规则、变现模式)。

## 1.内容创作

要知道，热门的内容是相通的，赚钱的项目也是相通的。

如果你想日更文章，但是苦于没有素材，你可以这样做：

在你刷短视频的时候，在首页碰到一个让你觉得非常有道理、让你产生共鸣的内容，你不要点赞或者收藏，更不要在评论区发表自己的观点。你需要做的，就是直接把它的文案整理下来，作为自己的写作素材。

想要获得它的文案，有两个思路：

①在微信上搜索“轻抖”这个小程序，然后点击“文案提取”,把抖音链接复制过去，然后就可以提取文案了。超多人声/各地方言/强大的附加功能尽在6配音3.Q版2智能图片 智能配音 视频检测 语音转文案 智能文案 去水印 去字幕 文案提取 提词器 图片转文字 查权重 账号估值 文案库 图文卡片 更多工具

注意，这个小程序每天需要你看30秒的广告，然后才可以进行提取，然后当天就可以一直不看广告提取了。今天是2025年8月15日，这个小程序还是有效的，我不敢保证当你看到这篇文章的时候，这个小程序是否仍然有效。提取完文案之后，你可以自己改写，也可以借助AI工具改写，比如把文案发给豆包，让它改写成你喜欢的风格，重合度不能超过50%。

②自己看完视频之后，直接根据自己的记忆和联想写一篇文章。我更推荐这种做法，因为我们最需要的是一个灵感，一个想法，当视频给你了一个灵感之后，你就可以围绕着这个灵感去写文章了。通过这种方式写出来的文章，会更加符合你一贯的风格和口吻，也会更加贴近你的主题。要知道，热门的内容是相通的，一个话题如果在抖音上非常火，那么你把它转化成文字、发到公众号上，大概率也会火。赚钱也是同理，你看别人做什么赚钱，也可以去模仿，这样的试错成本才是最低的，成功率也是最高的。

## 2.主观惰性

有些人获取素材的“客观思路”没有问题，就是在执行的过程中，始终无法突破自己的“主观惰性”。比如，当他在短视频平台或者公众平台上看到一个令自己产生共鸣的观点，就打算把它改写成一篇文章。

但是，当他写了100字之后，就觉得好像这个话题也没什么好写的，有点鸡肋，算了，还是放弃吧。而 U 君自己是怎么操作的？我其实在写文章的时候，也经常会遇到这种情况，刚写了两段文字，就觉得这个话题也没什么意思。但是，我会坚持写下去(我每天至少写4篇文章)。写到100字的时候我觉得很鸡肋，但是硬着头皮写下去，微信pep458到了500字的时候，我的大脑可能会发生奇妙的化学反应。很多有意思的观点和故事都被牵出来了。等我写完1000字的时候，这就是一篇非常吸引人的文章。在我公众号的文章里，至少有三分之一的文章是这样写出来的，而且效果都不错。很多人的问题是，始终跨不过那道“鸡肋”的瓶颈，每当碰到“可写可不写”情况，他都是选择“不写”。而对于我来说，每次碰到“可写可不写”情况，我都是选择“硬着头皮写下去”。这就是你和我的区别。

## 3.流量获取

到了2025年的今天，想要获取精准流量，主要有三个方向可以发力：平台推荐，自然增长，站外引流。

①平台推荐其实从2023年下半年开始，公众平台就开始搞起了“推荐机制”。只要你的文章吸引人，就有可能推荐给陌生的读者，大大提高涨粉速度。但是，如何获得系统推流，这是个玄学。一般来说，新号被推流的概率很大。还有就是，写一些有争议性的话题(比如孝道、情感、家国情怀)也容易获得推流。我的【U君笔记】那个小号，最近两个月才获得了系统推流，偶尔会有一篇阅读量过万的文章(其他文章的平均阅读量在2000左右)。

②自然增长我的【赢在8小时之外】这个公众号，已经写了1500多篇原创文章了(我还删除了几百篇),但是没有一篇文章获得过系统的推流。虽然粉丝增长慢了一些，但是我也觉得无所谓，毕竟自然增长的流量才是最精准、粘性最强、最有购买力的。这就好比，自然生长的走地鸡肯定比用激素催熟的鸡更好吃、更有营养。

③站外引流我的公众号粉丝里面，至少有三分之一的粉丝来自于“知乎”这个平台(我在知乎上的昵称是“U 君笔记”)。我在知乎上经常写回答(很多都是公众号上文章的改写、同步),结尾我偶尔也会留下自己的公众号名字，引导大家关注。有些知乎上的读者觉得我写的内容很有深度，他就会跑到微信上关注我的公众号。我觉得这种引流方式，放到2025年的今天，依然使用。只不过，你需要注意平台的规则。

## 4.平台规则

无论是公众号还是知乎，有很多规则是相通的，比如，不能写带颜色的、擦边的内容，不能违反公序良俗，尽量不要谈论正治和国家大事。但是，不同的平台也存在一些差异。最严格、最琢磨不透的平台可能就是小红薯和斗音了，一不注意就会被限流，你如果提到“微信”等字样也会被限流。甚至，在私信的时候都不能用“微信”这两个字，而是要用“+V” 等隐晦表达。知呼就相对宽松一些，它们玩的是“店大欺客、客大欺店”的规则。我在知呼上有十几万粉丝，是个有一点影响力的小V, 那么我在回答或者文章结尾留下自己的公众号进行引流，他们就睁一只眼闭一只眼。但是，如果你在知呼上只有几百个粉丝，没有什么影响力，不具有捅蘸价值，那么他们就会对你非常严格。如果你敢留下站外引流信息，他们就会给你禁言，甚至封号一段时间。另外，如果你写作时间长了，经历过几次违规，就会对平台规则有更加深刻的认知。就拿公众号来说，我在写作时，尽量会用傥、姓、儉官、骨票、鸡金、自刹这些错别字来代替原有的敏感词。还有，在提到大公司、名人的负面新闻时，尽量要用谐音或者错别字来代替，比如支附宝、惊东、马芸、王健琳等等。

## 5.变现模式

很多人在“变现”这个环节会面临两个瓶颈：不敢收钱，不敢卖高价产品/服务。

一方面是因为他们不自信，或者有完美主义：我有何德何能，能让大家给我交钱?实际上，只要你给大家提供价值了，不管是商业价值、成长价值还是情绪价值，都很宝贵。所以，你必须要收钱。只有你收钱了，你才会重视这件事，才会把它当成是事业来对待，才会创造出更优质的内容。

如果你接触过有钱人就会发现：他们在买东西时，会有一个最低的门槛。比如，如果去外面吃一顿饭价格低于100元，他们就会怀疑食材有问题。如果买一件T恤，如果价格低于200元，他们会认为这件衣服的材质、设计非常差劲。在知识付费领域也是如此，你如果一直免费为大家服务，微信pep458或者提供的都是低价服务，那么有钱人会在潜意识里认为你没什么本事。有人会问：如果我把价格提高，会有很多“忠实粉丝”声讨我，说我飘了。我的回答是：你早就应该把免费的、低价的粉丝筛选掉了，只为有钱人服务。
