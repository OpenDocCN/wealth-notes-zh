
# 子明7.30付费文：解读

原创顾子明子明解读 2025年07月30日22:23

辽宁 6听全文

之前，在《目7月政治局会议前瞻》中，预测了7月会议将低于市场预期，《目中美瑞典会谈展望》中，也预测了中美谈判的表面结果不及市场期盼，今天都已经兑现了。

按照传统与惯例，今天的文章将分析将决定下半年经济工作和资本市场的7月政治局会议：

一、中美2025贸易战取得胜利，中国解除“战时状态”

1.1中央对未来局面的预判，426是“外部冲击影响加大”，“外部环境急剧变化”，针对的是4月2日特朗普的全球关税战。730则调整为“当前我国经济运行依然面临不少风险挑战”，潜台词是对未来局势报以非常大的乐观，说白了，就是已经谈妥了特朗普访华。

1.2中央经济主要解决的矛盾，426说的是“统筹国内经济工作和国际经贸斗争”，也就是国内一切政策要为贸易战服务，指在为5月10的日内瓦会谈准备，而730回归了“统筹国内国际两个大局”，“统筹发展和安全”这种日常，不把特朗普当牌菜。

1.3426的“同国际社会一道，积极维护多边主义，反对单边霸凌行径”等类似的合纵话术，在730消失。欧洲错过跟跟中国达成协议的最佳时机，很多原本可以抢在美国身前的东西要落到身后。

二、为贸易战准备的兴奋剂退场，改为固本培元的长期调理

2.1 426的经济宏观政策，说的是“加紧实施更加积极有为的宏观政策，用好用足更加积极的财政政策和适度宽松的货币政策”，可以理解为林彪大决战时“我不要伤亡数字，我只要塔山”。730的说法变成了“宏观政策要持续发力、适时加力。要落实落细更加积极的财政政策和适度宽松的货币政策，充分释放政策效应”，突然变成了精打细算的李云龙独立团。

2.2 财政政策，变得节俭，730相较426少了“超长期特别国债”，还补充了一句，要“提高资金使用效率”，开始注重费效比了。

2.3 货币政策，变得迂回，426是很直白的动作“适时降准降息”，730则变成了“促进社会综合融资成本下行”。

2.4 政策工具箱，也从增量变成了存量，426说的是“创设新的结构性货币策工具，设立新型政策性金融工具”，730说的是“用好各项结构性货币政策工具”。

简而言之，不要琢磨短期刺激了。

三、因全面备战而推迟的“反内卷”和全国统一大市场建设，在下半年提速，大量白马企业涌现。

3.1 426的重点是保护企业，说的是“扎实开展规范涉企执法专项行动”，反内卷方面的措辞很轻，说的是“大力推进重点产业提质升级，坚持标准引领，规范竞争秩序”，随着外部危机结束，730的重点变成了淘汰劣质企业，说的是“依法依规治理企业无序竞争。推进重点行业产能治理。规范地方招商引资行为”，招招直逼要害。

3.2 全国统一大市场建设，从426的“加快”，变成了730的“纵深推进”，并提出了“推动市场竞争秩序持续优化”，此举会导致众多行业的市场集中度加速提升。

3.3 426的说法是“打造一批新兴支柱产业”，730的说法是“加快培育具有国际竞争力的新兴支柱产业”，这意味着有一大批新质生产力企业将要在下半年加速腾飞，建议重点关注福建板块。

四、因外部经济冲击大幅减缓，二季度加力的“两重”“两新”会在下半年减弱。

4.1 对于两新，426说的是扩围提质，730则消失了，意味着以旧换新和设备更新在下半年会减弱。

4.2 对于两重，426说的是加力实施，730变成了高质量推动，意味着国家重点项目将从量转向质。

预计下半年利空家电、消费电子、以及部分工程基建。

五、贸易战风险降低，并不意味着关税将被取消，外贸企业依然要承担巨大的压力。

730依旧说了，“稳住外贸外资基本盘。帮助受冲击较大的外贸企业，强化融资支持，促进内外贸一体化发展”。

六、随着中央城市工作会议召开，城市更新将成为主线，利好人口净流入的超大特大城市。

6.1 730提出“支持经济大省发挥挑大梁作用”，下半年加杠杆的就是这些城市。

6.2 因贸易战冲击，426未提及债务风险问题，730则言辞激烈“严禁新增隐性债务，有力有序有效推进地方融资平台出清”，意味着经济差债务重的地区下半年没啥大动静，更多聚焦于还债。

城市群之间的差异将在下半年开始迅速拉开。

七、资本市场，下半年主打一个稳字。

去年730的说法是“提升资本市场内在稳定性”，之后就是著名的924行情，新办举行新闻发布会，推出降准、降息，降低存量房贷利率、首付比例，创设新货币政策工具等一系列超预期政策组合拳。

今年426的说法是“持续稳定和活跃资本市场”，随着谈判顺利，叠加六月以来此起彼伏的大项目公布，直接把后续几个月的资本市场和大宗市场带飞了。

今年730的说法是“增强国内资本市场的吸引力和包容性，巩固资本市场回稳向好势头”，意味着在资本市场方面，国家队从攻势转向守势，并更多通过基本面和外资力量，而不是政策面来驱动资本市场。

可以更多关注外资建仓所青睐的标的。
