
# 特朗普最新1.7万字国会演讲全文

## 主要内容

* 提出通过减税和削减政府开支来缓解经济压力，进行永久的个人所得税减免；
* 废除《芯片法案》，并推动关键矿产和稀土的大幅增产，增强美国自主性；
* 赞扬马斯克领导的政府效率部，表示将继续精简政府机构并削减成本；
* 提出“每增加一项新规定，必须取消十项旧规定”的方针；
* 要求国会提供更多边境安全资金，以继续推进大规模驱逐非法移民的行动；
* 收到乌克兰总统泽连斯基的信，信中表示后者准备签署矿产协议与和平谈判。

## 演讲亮点

* 在43天内，特朗普政府所取得的成就超过了大多数政府四年或八年内取得的成就。
* 特朗普政府签署了近100项行政命令，并采取了超过400项行政行动。
* 在南部边界宣布了国家紧急状态，并调动了美国军队和边境巡逻队来抵御对美国的入侵。
* 终止了荒谬的绿色新政，退出了不公平的巴黎气候协议，退出了腐败的世界卫生组织。
* 命令所有联邦雇员返回办公室，结束了武器化政府。
* 签署了一项命令，使英语成为美国的官方语言。
* 结束了所谓多样性、公平性和包容性政策在整个联邦政府、甚至私营部门和军队的暴政。
* 从公立学校中去除了批判性种族理论的毒瘤，签署了一项命令，使美国政府的官方政策是只有两种性别，男性和女性。
* 签署了一项行政命令，禁止男性参加女性运动。
* 拯救我们的经济并为工薪家庭提供戏剧性和及时的救助。
* 打击通货膨胀的主要焦点是迅速降低能源成本。
* 创建了全新的政府效率部，由埃隆·马斯克领导。
* 通过削减我们能找到的所有欺诈、浪费和盗窃，我们将击败通货膨胀，降低抵押贷款利率，减少汽车付款和食品价格，保护我们的老年人，并将更多的钱放入美国家庭的口袋中。
* 平衡联邦预算，引入那些聪明、勤奋、创造就业机会的人。
* 推动历史上最伟大的经济的下一阶段是让国会通过给每个人减税的法案。
* 在各个方面进行永久的个人所得税减免，以便向受到通货膨胀特别影响的美国人提供迫切需要的救助。
* 呼吁对小费不征税，对加班不征税，对我们伟大的老年人的社会保障福利不征税。
* 将汽车贷款的利息支付设为可抵扣税款，但前提是汽车是在美国制造的。
* 减少国内生产和所有制造业的税收，提供100%的费用抵扣。
* 对那些对美施加关税的国家施加关税，平均而言，欧盟、中国、巴西、印度、墨西哥和加拿大等国家对美国的关税远高于美国对它们的关税。

## 结语

特朗普政府的政策旨在让美国再次伟大，通过减税、削减政府开支、增强美国自主性、精简政府机构、平衡联邦预算等措施，为美国家庭提供更好的生活。特朗普政府将继续努力，让美国成为世界上最强大的国家。
