
# 张佑君派驻境外延退，中信证券将迎来第四代领军者

张佑君在中信集团任职时就主导了中信集团境外整体上市，在中信证券更是亲自挂帅直管中信证券国际，锚定国际一流投资银行这一目标。

## 张佑君的职业生涯

张佑君是中信证券创始即加盟的老将，交易员出身，从一线做起，对券商自营、资管及财富管理等各业务链条都十分熟悉。

张佑君1965年7月生于江苏，1987年获中国人民大学货币银行学学士学位，1990年获中央财经大学货币银行学硕士学位。1992年，他加入中信实业银行深圳分行。1995年中信证券筹备成立，他从基层开始，踏足投资交易，四年间历任交易部总经理、襄理（中信证券早期管理岗位）、副总经理。

1999年长盛基金成立时，张佑君作为中信证券（长盛基金发起人股东）派驻代表，担任首任总经理。在他主导下，长盛基金稳步推进，基金同益、同盛、同智、同德陆续发行，奠定了公司规模基础。不过，2000年基金行业“对倒”、“倒仓”等坐庄行为被揭露，长盛基金也经历波折。

2001年，张佑君回到中信证券。2002年王东明升任董事长后，他接任总经理。2005年，北京市政府主导，中信证券与建银投资出资27亿元，筹建中信建投受让华夏证券全部证券类资产。中信证券出资16.2亿元持股60%，建银投资持股40%，前者算盘是择机合并，后者计划退出。北京市政府安排时任副秘书长黎晓宏任首任董事长，张佑君出任总经理。

两年后，黎晓宏离任，张佑君接任董事长兼党委書記。在其领导下，中信建投业务稳步推进。2010年，中信证券向北京国资转让中信建投股权后丧失大股东地位。2011年9月，张佑君卸任董事长，重返中信集团，历任董事会办公室主任、总经理助理，在集团境外整体上市过程中发挥主要作用。

2015年8月股灾后，中信证券多名执委及部门负责人被查，中信集团迅速调整班子，张佑君于当年11月任党委副书记，次年1月接任党委书记、董事长。当时原高管团队几乎被打散，大多转投他处。

张佑君任内曾担任中信里昂董事长，亲抓对中信证券境内外国际化。2013年，中信证券完成对里昂证券的收购，一度成为中资投行出海、打造“中国版高盛”的经典案例。然而好事多磨，中信证券早年曾希望借此收购，提升公司的国际化程度，可无论是员工个人价值观，还是企业文化，中信证券和中信里昂的婚姻一直存在间隙，二者内部矛盾不断。

在张佑君离开中信建投，重返中信证券并接替王东明成为新任董事长后，中信证券内部的“集体主义思想”就变得更加强烈，这一观念也在常年扎根香港、崇尚自由和银行家个人价值的中信里昂团队中，引起了较大反弹。尽管中信证券当年收购中信里昂后，承诺保留员工、薪酬和独立运作，但相关的“君子协定”在2018年中到期，资深外籍投行家常年拿着高额奖金，但公司业务并没有明显的突破，也没有跟上行业中的各类变革，这样的结果让中信证券很不满意。

面对高昂的人员成本，中信证券对中信里昂又进行了多方面调整，后者的年度奖金下调了60%，员工薪酬结构也有所调整。最终，中信证券决定更直接深入地管理中信里昂，使得这家香港精品投行高层彻底洗牌。2019年3月前后，中信里昂原高层接连出走，张佑君因此亲自挂帅，推动中信证券在全新的全球垂直一体化战略，锚定国内领先、国际一流投资银行这一目标。

一位中信证券“老人”称，王东明那一代的中高层极具个性，王东明豪爽直率、敢怒敢言。而多年未在中信证券任职、2015年风波后回归的张佑君，则以“技术流”、谦虚低调见长。“当时很多人不服，觉得他是来摘桃子的，想看看他有没有真本事。”事实证明，张佑君同样魄力十足，在管理上十分强势。他所带领的高管团队在2015年风波后稳住了中信证券的底盘，也逐渐恢复市场信誉。2020年，中信证券收购广州证券，加之前期一系列并购效应显现，又抓住机会接连股权融资，资产负债表不断扩张，用表能力显著提升，在国泰君安与海通合并前，已多年稳居行业第一。

## 邹迎光的崛起

早在4月，就有中信证券人士对财新表示，公司内部已流传“佑君总可能延迟退休，之后由邹迎光接任”。其后，接任人选仍存在多种讨论。中信集团副总曾琪、鲍建敏都曾被认为是热门候选。曾琪有工行背景，现主要分管中信证券、中信建投、中信保诚等金融子公司，并协管中信金控，近段时间曾去往中信证券调研。鲍建敏则来自中投公司，曾任执委，有多年投资经验。

几番权衡之下，深耕证券行业，辗转中信证券、中信建投任职的邹迎光被认为是更为合适的人选。邹迎光出生于1970年12月，尚未满55岁，五官轮廓清晰，一头整齐灰白短发，常佩戴边框眼镜，形象干练。邹迎光最早并非金融专业出身。1994年，他获首都医科大学临床医学专业学士学位，毕业后曾于首都医科大学宣武医院任外科医生。不过当医生的时间不长，邹迎光就将人生赛道转向了金融，去往中央财经大学求学；2000年，他在而立之年获得金融学硕士学位，毕业后进入证券行业，于当时的海南华银国际信托投资公司（下称“海南华银”）北京证券营业部任业务经理。

海南华银当时也是知名的金融机构，不过在2001年就被央行下令整顿，证券业务被广发证券托管，不久还爆发出联合多家金融机构伪造金融凭证的诈骗巨案。邹迎光在海南华银时间不长，就加入中信建投的前身华夏证券，逐渐在固收投资领域崭露头角，职业生涯的上升之路由此打开。

在华夏证券时期，邹迎光历任该公司海淀南路营业部机构客户部经理、债券业务部高级业务董事等职。2005年，华夏证券重组为中信建投，邹迎光历任债券业务部总经理助理、固定收益部行政负责人、执行委员会委员兼固定收益部行政负责人等。2017年9月，邹迎光调往中信证券，成为执行委员会委员。接近中信建投的人士称，当时邹迎光已是固收大拿，因此被挖角到中信证券任高管。2022年实际税前总收入超过900万元。到中信证券后，邹迎光曾任固定收益部行政负责人，将中信证券的固收业务推上新的体量。

多年来，固收业务一直是券商自营收益的主要来源。2025年中期，中信证券自营净收入（投资净收益+公允价值变动-对联营企业和合营企业的投资收益）达到190.52亿元，贡献了接近58%的营收。中信证券的自营投资赚钱能力远超同行，紧随其后的国泰海通、银行证券自营净收入分别为93.53亿元、73.99亿元，合在一起也不足其一。

2023年5月，时任中信建投总经理李格平被带走调查，市场哗然。五个月后，邹迎光被调往中信建投。彼时，接近中信证券的人士对财新表示，邹迎光将接任中信建投总经理，“邹迎光去中信建投之前不久刚刚成为中信证券党委委员，原计划提拔他为中信证券总经理，随后被安排去建投，在中信建投担任党委委员时间又不足一年，所以一直没正式任命”。

虽然一直未就任总经理，但邹迎光到任中信建投次月就出任执行董事，也在2024年8月成为中信建投财务负责人。“到2024年10月，他任党委委员已满一年。此时中信集团来人考察，大家才得知他要回中信证券任总经理。”接近中信建投的人士称。

11月6日，中信证券公告，当日上午，公司在中信证券大厦召开了董事会第二十八次会议，董事会8人全票通过，同意聘任邹迎光为公司总经理，任期自董事会审议通过之日起至第八届董事会届满之日止，公司执行董事、董事长张佑君不再代为履行公司总经理职责。

中信证券董事会每三年为一届，第八届董事会届满为2025年年末。加上邹迎光，中信证券经管会目前共有15名成员，绝大多数加入中信证券都超过20年，其中邹迎光是中信证券任职时间最短的一位。

## 中信证券的国际愿景

早在经历2015年的风浪前，中信证券已被打造成了证券业的领头羊，经历近几年发展，业务更趋全面，无论是自营还是其他业务，几乎都睥睨业内，素来享有“麦子店高盛”的坊间别称。

“当年的《高盛帝国》（中信出版社2010年出版）我们人手一本，当时中信的目标就很很明确，要成为高盛一样的国际化大投行，'麦子店高盛'的说法也是这么来的。”一位中信证券老人回忆当年的意气奋发。

不过他也表示，中信证券距离这一目标仍很远。早在2014年，中国的股市市值首次超过日本，成为仅次于美国的全球第二大资本市场。但一直以来，中国券商行业的资本实力和综合能力都与海外顶级投行存在较大差距。中国的银行、保险公司都入选过全球系统重要性金融机构，而证券公司从未入选。至今日，整个中国券商行业的总资产才与高盛（Goldman Sachs）（Morgan Stanley）资产规模相近；中信证券，净资产和净利润目前都只有高盛的五分之一，目前仅在亚太区域能与之竞争，恰是得益于当年收购了里昂证券。

在2023年9月的一场投资者活动中，张佑君曾谈到，与国际一流投行相比，中信证券在国际化程度、品牌影响力、资产负债表扩张及管理能力、金融科技水平、人才队伍等诸多方面仍有较大差距，距离建设国际一流投行的目标还有很长的路要走。

中信证券的境外业务主要由中信证券国际贡献，目前“中信证券国际”和“中信里昂”两块品牌并行运作，业务主要涵盖证券经纪、机构股票业务、企业融资及资本市场业务、资产管理等，已拓展至全球13个国家和地区，是国内券商国际子公司布局最广的一家。其业务重心集中在亚太市场，在该区域的市场份额超越高盛，并在印度、菲律宾市场稳居首位。近年来，公司正试图扩展在日本、东南亚及中东市场空间。

接近中信证券国际的人士称，中资券商在境外扩张面临多方面的限制，首先是“人民币并非自由可兑换的国际货币”，这意味着中资券商都不可使用境内母公司的资产负债表，评级受限，在境外融资成本也会偏高；作为控股的境外子公司还需要同时满足境外市场和境内市场的监管要求，在用人方面也激励不足，缺乏黏性。2024年，中信证券对中信证券国际完成了
一笔增资，规模约9.16亿美元，使得后者的总资产达到约409.7亿美元，折合人民币近3000亿元。财报显示，2024年中信证券的境外业务收入为109.48亿元，占比在17.16%。公司整体杠杆率则在4.72倍左右，虽一年提升了0.25倍，但仍不足高盛二分之一。

在前述投资者活动上，张佑君曾直言，中信证券的国际业务占比较低，国际化竞争力依然不足，在全球范围内开展业务、配置资源的能力相对较弱。公司境外业务收入占比与高盛等30%—40%的国际收入还有差距。资产负债经营能力有限，抗周期波动能力不够强，中信证券的杠杆率为与高盛、摩根士丹利差距较大。

这不是中信证券一家的烦恼，如何继续做大做强是行业整体的命题，尤其是宏观经济存在挑战，行业降薪、限薪的背景之下（参见《财新周刊》2025年第36期《金融业降薪来临》），解题思路似乎导向了合并。

“中信证券这些年虽然比我们强不少，但行业还没到'一超多强'的地步，在国际上也还谈不上有竞争力。”一位券商高管认为，在国君和海通合并之后，起码从体量上能够超过中信，在国际上的排名也是。“金融机构需要体量，然后是竞争能力，这里面体量是基础，是最开始的'1'，没有这个'1'后面其他都等于'0′。”他说。

自2023年高层提出打造“航母级”券商、一流投行的目标后，证券行业的整合明显加速国泰君安与海通证券在2024年9月开启世纪大联姻，国泰海通横空出世，并于今年上半年合并报表，1.8万亿的总资产与中信证券相当，3214亿元的净资产则超过了中信证券。

国泰海通之后，关于行业巨头间潜在合并的猜想再度升温。“中信+中信建投”组合的可能性，几乎每隔一段时间便会成为舆论场上的联想热点。不过，合并牵涉多方利益，北京是否愿意放手一直存疑。“之前是北京不同意，本来北京就没有自己的大券商，而且每年在建投还有20亿左右的分红。后来又说北京同意，但是要合并后公司注册地在北京。但这又动了深圳的利益，毕竟中信证券在深圳注册，目前依然无解。”一位知情人士称。

另一位资深的证券行业人士认为，北京也不会轻易松口让中信证券合并中信建投。“券商行业这些年马太效应越发明显，大券商牌照价值不同，能拿都会想要。而且北京一直未有控股一家实力强的券商，比起让给中信集团，北京国资为什么不自己做呢?”他说。

中信集团在十几年前就有意让中信证券牵手中信建投。2009年，受证监会“一参一控”限制，中信证券须转让持有的中信建投股权。此时，中信集团与中央汇金设计了“两步走”方案，即中信证券把持有的45%的中信建投股权转让给北京市国资经管中心，8%转让给世纪金源；其后，中信建投的另一股东中央汇金再将持有的40%中信建投股权转让给中信集团。若按此计划，中信集团和中信证券对中信建投的持股合起来，依然对中信建投具有相对胜出的话语权。不过，因为种种原因，“两步走”方案只完成了第一步，仅中信证券保留了7%股权（参见《财新周刊》2015年第24期封面报道《中信证券单飞》）。

2015年，中信集团再筹谋从中央汇金手中接盘中信建投股权，这一年市场牛走熊来，中信证券也陷入了意想不到的危局，高层遭遇大动荡，再无下文。

历经几轮调整后，中信建投目前不存在控股股东，也不存在实际控制人，第一大股东是持股35.81%的北京国资北京金控集团，第二大股东是持股30.76%的中央汇金，中信证券持及其一致行动人持股中信集团旗下的镜湖控股合计持股约9.47%。

据财新此前报道，此前在中信证券向北京国资转让股权过程中，北京金控与中信集团签署了股权代管协议，由中信集团代管中信建投，多年保持未变，因此中信集团一直对中信建投实际行使大股东权力。

“现在这个市场环境与十几年前已大为不同，如今将两家大券商合并不见得是好事，方方面面
的冲突可能反而折损效率，不如各自奋斗。”一家券商已退休的总裁称。■
