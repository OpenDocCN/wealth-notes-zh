
# 干他娘的特朗普

电影是来自英国，确切说是一个英剧，9号秘事，第6季第3集。 一个神神叨叨的失恋中年油腻男，由于怀疑前女友跟其服务的政客有一腿，落入其对手的陷阱。当政客被一枪毙命，中年男被移花接木塑造成了杀手，天衣无缝。

事后，其对手应该即是在任的英国首相对着镜头义正词严：严厉谴责暴徒，对家属表示哀悼，将全力以赴调动资源保护其他竞选者。

这个场景，让我立刻想起了拜登。去年7月，当那个疯疯癫癫的枪手把子弹射中特朗普耳朵之后，枪手立刻被毙，拜登发表电视讲话：严厉谴责，沉痛慰问，全力以赴对特朗普保护。

天下文章一大抄，从时间上来看，应该是2024年的美国大选，特朗普团队抄了这部英国BBC 在2021年的剧。

没错，我推测抄袭者是特朗普而不是拜登。 一 方面是因为子弹没打中，特朗普只受点皮外伤当众撒血拍照狂赚人气，另一方面是拜登就此完蛋。

我认为拜登干得不错，四年里领导美国走出疫情，复苏经济，股市连创新高， Al搅 动全球。尤其是俄乌战争到来，他携小泽司机顶住普大哥排山倒海之势，让美国再次成为西方老大哥。但一颗如打耳洞般精准的子弹，打掉了拜登的连任梦。特朗普要回来了。

斗争相辅相成。美国也是有法的，明着来已经无法阻止特朗普，拜登奥巴克莱克林顿这一派深层势力，也必然要从文艺作品里寻找灵感，给特朗普玩个阴的。

他们找来找去，翻遍了莎士比亚全集，摸烂了奥义书，用AI快速学习完永乐大典……无数个日日夜夜，无数次含辛茹苦，终于，他们搜到了鄙人的作品，在我的经典金瓶梅故事里，找到了制服特朗普的法宝。

这完全不是扯淡。因为美国人民也是可以用微信的，虽然美国版微信无论是whatapp、instagrm 抑 或x or redddit我们不好用，但即便是美国人来到我们这旅行，他们照样可以在我们的网络里尽情享用，仅仅需要他们的手机号开个漫游。他家的你不能用，你家的他可以用，他家的他在你家也可以用只你不能用。我的数理逻辑没学好，搞不明白这里面是不是藏着不对等。话说回来，拜登团队搜啊搜，搜到了我写的金瓶梅故事，里面详细讲了怎样让不可一世的西门庆一败涂地。方法很传统，怂恿他、鼓 动他、诱惑他，让他向清河县最深沉的boss 进攻。

特朗普的任期也就4年，按说熬一熬忍一忍也就过去了。就像是西门庆的诸拼头，都在熬。个中代表韩大嫂王六儿，其丈夫是西门庆铺子里的伙计， 一年的工资不到二十两银子。但王六儿被西门庆看中，西门庆手一甩就是三十两，又给买丫鬟又给买房，让这一家顺利步入小康。

这个过程，王六儿要熬。这话她对丈夫说过，伺候西门庆那行货子，霸王一般，熬忍得好不幸苦。关键是，她还不敢把西门庆掏空，任凭十八般武艺也得收着使用。把西 门庆一口气榨干掏空，来钱路子就断了。

都是聪明人，在竞争激烈的西门庆裙带圈里，大家都抱着同样的心思。好不容易赶上个出手大方的西门庆，既要陪他玩，又要陪着他慢慢玩，千万别把他玩脱了、玩急了、玩掀桌子了。那样一来，吝啬鬼西门太太当了家，大家一拍两散，来钱路子全断 了。留着西门庆，是清河县广大乡民共同的心声。所以在特朗普就任的头两个月里，尽管 他带着马斯克搞出一连串幺蛾子，但美国的深层势力并没有出手限制。

让他玩，特朗普玩的，其实对美国有好处。

比如俄乌战争，美国倾力支持了三年，钱花进去了，结果看不到边。想退出，但以拜登信誓旦旦喊过的那些口号，退出是啪啪啪太打脸。那就让特朗普上来，由他反悔。

再比如联邦大裁员，臃肿庞大的公务员体系，早就该裁撤了。但这些部门好多是过去几任设置的，好多人员是七拐八拐各类裙带，让一个像拜登这样的老政客来动手，是很难下手的。那就让商人特朗普来，再加个商人马斯克，他俩以办企业的效率大刀阔斧一通裁。

实事求是地说，如果是你们村里，村长宣布一不再给八竿子打不着的外村小伙子发奖 金，二要裁撤村食堂里那帮当闲差吃干饭的关系户，你拥不拥护，支不支持?

答案很明显，除非你就是关系户。

另外，全球加关税+卖500万美金一张的金卡，说白了就是搞钱，搞钱还债。美国政府负债已突破35万亿美元，这些债务是在最近20年里搞出来的。在克林顿时代，美国总负债还不到5万亿，而且从1998年开始每年财政都有盈余，债务每年都在缩减。只是从小布什开始，又要反恐，又要打仗，搞出金融危机后又要搞量化宽松印钱，导致债务越积越多。

怎么办?拜登没办法，奥巴马更没办法，他们都必须高举正确的大旗，烂事让不正确的特朗普来干。

特朗普的全球关税，实际上就是让全世界人民包括美国人民， 一起帮美国政府还债。

对此，全世界绝大多数地方均表示ok。

不报复，不反制，不仅不加税，甚至有多家趁机推出对美零关税，愿意谈判，寻求妥 协。按照中文互联网词汇，就是跪了。

跪了的真不少，既有近邻印度，也有兄弟越南，还有号称早就富了的小日子，其首相 周一一上班就找特朗普视频通话，跪求关税减免。

这些人的心思跟西门庆裙带圈诸拼头的想法是一样的，陪着特朗普慢慢玩。说白了，即便是姿势里带着点挣扎，那也是为了给大官人增添兴致，半推半就，纯属假干。

能跟西门庆实打实真干的，必须具备三个条件：

一是不差钱，家里有钱花不完。

二是在家说一不二。

第三，脾气，跟他要投。

相同脾气的两个人，好的时候可以把酒言欢同称好友，但顶起牛来，那可是如出一 辈，分毫不让。

敢同恶鬼争高下，不向霸王让寸分。

敢同恶鬼争高下的， 一定是神仙；不向霸王让寸分的，必须也吃惯了霸王餐。
