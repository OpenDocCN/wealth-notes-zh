
# 一赢各表的谈判
## 发布时间：2025年5月9日

巴基斯坦宣布击落的印度的先进飞机，印度宣布摧毁了巴勒斯坦的恐怖组织，中国宣布应美国邀请去瑞士谈判，特朗普宣布不会为了谈判而降低关税，并暗示谈判是中方主动提出的。

这几个事儿之间没有本质的区别。印巴中美对内宣布自己赢麻了的目的，都是为了切实缓和双方之间过于紧张的关系，为双方建立沟通机制来铺平道路。

因此，各方在宣传方面，都会允许自家媒体对胜利的成果进行适度夸大，激发爱国情绪，以提升执政集团的支持率，而不必担心对手扯破自家的虎皮。毕竟，舆论场上首先互不攻击，是双方降温局势和择机开启和谈的基础与诚意。

但这只是双方初步的释放善意，以及避免过程中都下不来台，跟是否能够和谈成功，并没有直接的关系。

这就跟英王特使马戛尔尼觐见乾隆时是否下跪，中英两国历史上记载的完全不同很类似。清英双方协商后，马戛尔尼执行的是单膝下跪，大清可以称之为英式下跪，英国可以称之为拒绝了按照清式下跪。

双方可以“一跪各表”，都能宣称自己赢麻了。
