
# 七星连珠，向内求的好日子

人一出生的操作系统，是心，后天装了很多的小程序，是脑。App 的体验，都是《金刚经》里“相”，回到清洁版本的操作系统，就是“明心见性”, "我心即宇宙”。世俗中的人们，总觉得，我现在不够有钱，不够漂亮，不够帅， 不够成功，所以不快乐，这都是大脑的 给你的，叫做分别心。

如果有钱，有很多钱就快乐，马老师应该没有烦恼，是世界上最快乐的人才对，分明不是，问题是他也苦大愁深的。所以人应该怎么样回到“心”，怎么过得持久的幸福和成功?

第一，寻找自己认为有意义的事情，人生本来没有意义，我们最大的价值，就是定义这个意义。你当好老师，你做好员工，你在群里大家快乐，这都是意义。哪怕你马上，没有找到人生的意义，但你感觉你在一些工作中，体现出了让自己感觉和崇高的东西，那就是它，深挖，持续。

第二，你真诚发掘和欣赏，你在这个追求你过程中，那些所体现出来的人性美好品质。比如慷慨，勇敢，坚韧，豁达，果断，等等。你发自内心的，欣赏这些，你这个过程中，所体现出的高贵品质，并因此感到骄傲和自豪。它和你的皮囊，地位，财富，没有一点点关系，这是你刚出生时，宇宙赋予你的。然后，你内心的光明和涓流就会放大，你会越来越充实，越来越自信。你活的像一个真正的贵族，不容侵犯，但又无比谦卑。然后，事业和财富自然而然就越来越好，这就是“向内求”的具体方法。

今晚9点，是七星连珠的具体时刻，这个时间，建议有条件的同学们，在房间可以静坐，冥想，读书，用念力沟通天地，从内心欣赏奋进和奔跑的自己。回到道法自然的状态，充电，补能。晚上休息后，其他房间的灯可以保持打开，通宵达旦。迎接一个全新能量场的开启。
