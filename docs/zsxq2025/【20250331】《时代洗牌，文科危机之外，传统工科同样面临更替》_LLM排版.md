
# 时代洗牌，文科危机之外，传统工科同样面临更替

我们在近期谈到国家科技发展、教育改革等问题时，都会频繁提到一个现象，就是中国包括全球文科的收缩和理工科的扩张，以及“新工科”和“交叉学科”的概念。这背后反映了什么趋势？对你有什么影响呢？今天，借着最近热度很高的复旦的学科改革，谈谈背后的学科重置趋势和影响，事关你或者孩子的专业选择，建议认真听。因为这个话题还在不断探讨和快速发展中，我的认知也有一个深化过程，今天的內容仅供你参考，后续我也会持续追踪最新动态。

正题开始。

## 文科缩招之外，新工科正在替代传统工科

先说下复旦大学正式启动的“教育教学3.0版”改革计划到底是什么。主要有两点变化：
一是，人文社科的招生比例，从原来的30%-40%，下调到20%。二是，本科招生中80%的名额，则是分给了理科、医科、新工科和交叉学科。这已经不是微调了，而是一次“大换血”。

复旦这一轮调整的背后原因之一，恐怕是危机感。根据全球三大高校排名机构泰晤士的最新排名显示，复旦的综合排名，从过去的中国前三，跌到第七，被多个理工科强校超越。而复旦这次改革，我认为，至少有四点值得关注：

第一，复旦人文社科的改革思路是“用数量换质量”，把教学目标调整为集中资源培养“国际视野的思想领袖”。这说明，不是要消灭文科，而是要控制规模。说白了，不是文科不重要了，而是以培养通用型管理人才为主的短平快的文科，不再是这个时代的需求了。

那文科人才未来还有超高的价值吗？我的回答是，绝对有，但集中在金字塔尖的少数文科精英上，包括新兴的交叉学科，大部分的普通文科人才，确实面临需求减少的问题。

第二，未来不是把文科资源，简单转给工科，而是花大力气建设新工科。

新工科和传统工科有什么区别？简单理解，就是传统机器和机器人的区别。大工业时代的传统机器，和代表战略性新兴产业的机器人，一听就是两个物种，直接看就是机器人比机器更智能，而发展到人形机器人，那就是国家定义的未来产业了，比战略性新兴产业更超前。

另外，传统工科，做的大部分是从10到100的制造生产，相当于“制造芯片”；而新工科，要做的是从0到10的全面创新，相当于“生产制造芯片的光刻机”。

在这一轮改革中，复旦宣布，将原来的工科学院，拆分成6个创新学院，包括集成电路与微纳电子创新学院、计算与智能创新学院、智能机器人与先进制造创新学院等，这些就是新工科。另外，集成电路、人工智能、智能机器人、先进制造，这些都是国家战略支持的重点产业。所以说，复旦这一轮改革，是进一步和国家战略看齐。

谈到这里我必须要提醒你，就是千万不要只盯着文科缩减，觉得自己学个理工科就顺应时代了，不对的！工科的调整力度才是最大的，一些旧工科专业相比新工科，可能还不如文科。比如四川大学裁撤了材料化学、生物技术等旧的理工科学科；北京航空航天大学取消了土木水利硕士专业；大连理工大学申请撤销材料物理、无机非金属材料工程等学科。这些可不是偶然，要知道，2024年，全国高等院校裁撤旧工科专业1670个，裁撤数量非常高。所以，被淘汰的不只是文科专业，而是所有跟不上时代、追不上潮流的旧式学科。

## 交叉学科和新兴学科正在崛起

第三，是人工智能正在成为专业基石。在这一轮改革中，复旦推行“X+AI”双学位项目，覆盖将近50个本科专业，同时试点“学科博士+AI硕士”的双学位培养体系。除了复旦之外，清华大学也成立了本科通识书院，开始推行“主修专业+AI辅修”。

我的判断是，AI的使用，会像办公软件一样，变成未来每个本科生都要掌握的基础技能。

第四，顶尖高校越来越重视交叉学科。复旦这次宣布，本科招生的增量，全部投放到交叉领域，招生比例要占到20%，和文科、理科、新工科一样。正在行动的，不只是复旦，而是几乎所有重点大学。2025年3月的《政府工作报告》强调“优质本科扩容”，推动高校优化专业布局。之后，国家发改委马上宣布，2025年“双一流”高校的本科招生要新增2万人。

优质高校的扩招当然是好事，但扩招的专业，大多数都是交叉学科。比如，清华大学扩招150人，重点面向人工智能与多学科交叉领域；武汉大学扩招85人，主要招收人工智能与多学科交叉的复合型人才；西安交通大学扩招200人，主要面向人工智能等交叉学科领域。要知道，交叉学科在2021年才正式设立，而现在名牌大学的增量机会，几乎都给了这个新兴学科。

所以，我判断，未来的学科重置，不是理工科打败文科，而是新理工科取代传统理工科，是新兴学科打败传统学科。

直白点说，现在传统的理科、工科、医科等，都要有危机感，今天文科所遭遇的，未来这些学科同样会遇到，而文科虽然社会需求在缩减，但至少普适性强，干点啥都行，而被取代的旧理工科尤其是旧工科，可能会跟过时的技术一样，处境比较尴尬。所以，不是看你是文科还是理工科，而是看谁能先完成专业重构，成为新兴学科。

那么问题来了，传统学科，怎么转变成新兴学科呢？

无外乎两条路：
第一，传统学科和AI等新技术融合，变成“AI+”的新学科；第二，多个传统学科融合，形成新的交叉学科、融合学科。这既是理工科的机会，其实也是人文社科的机会。

比如，理工科领域的材料科学、化学和物理学，相互交叉形成了“纳米科学与工程”专业；微电子、材料学和信息技术，相互融合形成了“集成电路工程”专业。再比如，人文社科领域的政治学、法学与信息技术整合，成为了新的交叉学科“国家安全学”；国际关系、区域经济和语言学、社会学等学科相互交融，形成了新的“区域国别学”；以及，艺术学和工程学结合形成了新的“设计学”。到2024年底，全国435所高校，已经设立了878个交叉学科。

说到这里，一直跟随我们课程的同学可能会想到，这和课程开头提到的国家大战略是一致的啊，要么是数智化改造带来的新型工业化，要么是未来产业和新兴产业主导的新质生产力。是啊，肯定会一致啊！而且必须要一致。因为产业变革、教育变革、人才变革等一系列全面改革的最终目的，都是为了实现科技领域的指数突破，而这背后是国家正在构建的“大科创体系”。之后的课程，我们还会继续解读这套体系。说到这里，你是不是觉得又打通了一部分经络？

## 总之，结论很明确，未来想找到好工作、拿到高工资，不是取决于你学的

是文科还是理科，而是取决于你学的是不是有前景、有潜力的新兴学科。

怎样挑选有潜力的新兴学科？我这里准备了三条建议，供你参考：

第一，就业选择的价值排序，在家庭投入、个人兴趣和学习时长因素之外，原则上是国家战略优于区域需求，优于短期就业导向。其中，国家战略优先级最高。最高层在人才工作会议上也强调，要“大力培养使用战略科学家”，“打造大批一流科技领军人才和创新团队”。信号很明确，新型理工科精英、尖端科技人才的溢价，还会不断提高。

如果国家战略学科这趟车不适合你，那就选择满足区域需求的学科专业。国家战略的重点在前沿科技，而区域需求的重点在产业落地，并且不同区域的对应产业也不同，所以匹配度更重要。比如，上海要重点发展三大主导产业，集成电路、生物医药和人工智能产业；江苏要重点投资新一代信息技术、高端装备、前沿新材料等领域等等。具体的产业政策，可以参考之前的内容。

但是越是国家战略需要的学科，可能需要的读书时间就越久，比如要读到博士，成本也越高，各个家庭需要根据自己的实际情况判断。所以，如果觉得不适合，可以直考虑短期就业导向，包括很多一二本学生，也可以多关注几个新兴的职业资格证书；而职业本科的学生，则可以考虑成为高技术蓝领、高技能人才。

关于这部分内容，课程59讲有更详细的解读，里面还附上了中高级蓝领的示例参考表，你可以对照着看。

第二，选择交叉学科，成为跨学科人才。未来成功人才的样板，大概率是“马斯克式狂人”，懂技术、懂商业、懂营销。可以理解成，理工科天才与商科精英结合的复合型人才。科技精英，将取代过去的贸易、房地产、金融、互联网等领域精英，成为新的财富群体，在这个过程中，新型工程师和新型高级蓝领也同样受益。

对应到产业领域，这就属于产业中的新质生产力。所以，对于个人来说，我认为，最优选的是直接选择交叉学科；其次是，抓住辅修学位、双学位的机会，不断增强竞争力。

前面也提了，越来越多专业，要么转型为“AI+”学科，要么融合成交叉学科，这些学科会逐渐替代传统学科。

第三，最重要的是落实到行动。说回这次复旦的学科改革，对于“新工科建设”，复旦其实认知是很早的，因为国家在进行这一轮“新工科建设”前，曾经有过三轮共识讨论，而第一轮讨论就是在2017年的复旦。这一次讨论的结果，被称为“复旦共识”，是新工科建设的发端。看起来，复旦的行动没那么快，但可喜的是，今年的改革力度很大。这告诉我们一个道理，在时代洪流面前，认知当然重要，但行动才是实现认知的关键。
