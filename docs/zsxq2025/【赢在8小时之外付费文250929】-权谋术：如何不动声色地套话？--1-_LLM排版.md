
# 权谋术：如何不动声色地套话
原创：U 君 公众号：赢在8小时之外 25年9月29日

世界上最值钱的东西是什么?是有价值的信息。而真正有价值的信息，一定不会出现在免费的网站上，而是在某些业内人士和前辈的嘴里。那么,当我们想要从他们嘴里获取这些信息(套话)时，该如何操作呢?今天我就跟大家分享5个核心技巧，不仅可以用在商业场合，在男女感情、同事相处等维度也可以使用。当然，如果你掌握了这些技巧之后，不仅可以轻松从别人嘴里获取你想要的信息，也能防止别人套你的话。

## 1. 引发对方的纠错欲望

人性有两大弱点：不喜欢被别人窥探隐私；好为人师。所以，我们在套话的时候，一定要围绕着这两点去进行。

举个例子，你乘坐一辆网约车，你想了解他们这个行业的收入情况，这时候最忌讳的是直接提问：师傅，你们一个月能赚多少钱啊?这时候，司机的大脑里就会拉起警报，他会想：这个家伙为什么想知道我的收入情况?他到底是同行还是记者?他有什么目的?总之呢，这时候司机大概率不想正面回答你，或者是敷衍了事。如果是在发达的、注重社交距离的国家，“询问对方收入”也是一种非常不礼貌的行为，因为没人喜欢被窥探隐私。

但是，如果你换一个思路，你故意说一个错误的、夸张的金额，那么会激发起司机的反驳欲和好为人师的欲望。比如，你想知道网约车司机的收入情况，你可以这样说：“我听说现在很多网约车司机的收入都能轻松过万了。”这时候，司机很可能会跟你说：“怎么可能?我们累死累活一个月才能跑6000元，如果哪个司机能月入过万，要么专跑机场线路，要么专跑夜间车，要么就是跑的专车。”

这时候你就可以问他：您为什么不跑机场线路呢?想要跑专车需要什么条件?这时候，他就会打开话匣子，把这里的门道都讲给你听，而且没有任何防御和抵制的情绪。看到没有? 你之所以能顺畅、全面地从他那里套话、获取信息，就是因为你的提问没有拉响他头脑里的警报，反而激发了他“好为人师”的冲动。在男女相处时，有谋略的人也会使用这种技巧，从而让自己时刻处于主动的地位。

举个例子，女人想要打听男人的收入和支出情况，她不会直接提问，否则就显得太功利了。她可能会旁敲侧击地跟男人说：现在大城市的生活成本太高了，我们女生即使省吃俭用的，一个月也要花掉5000元，太难了。这时候，男人很可能也会打开话匣子：我们男人每个月的开销也很大啊，比如 XX、XX 和 XX, 每个月也要花多少钱。在这方面，如果你不想被别人套话，一方面是对自己的隐私守口如瓶；另一方面是克制纠正别人的欲望。

## 2. 开放式提问/模糊表达

这种手段其实并不光彩，甚至是一种轻微的冷暴力，一般常见于上级对下级使用。如果是同级之间使用，那就有点不地道了。

举个例子，有个同事或者同学在微信上问你：明天有事吗?其实，你明天有没有事、有什么事，这是你的个人隐私，对方没有过问的权利。但是，既然对方问了，肯定就是想占据主导权，这时候，你有上中下三策可以应对。

①上策：假装没看到。如果有人问我“明天有事吗”,或者问你“在吗”,我都不会回答的，因为我没那个闲心跟对方去玩捉迷藏。如果过后对方问我：给你发消息你怎么不回啊?我就会说：我没看到，手机没提醒。我一点都不会尴尬的，因为我非常清楚，只要我不尴尬，那么尴尬的就是对方。

②中策：直接回答“有事”,然后不再向下聊。如果你能把这样一个开放式的回答扔给对方，那么主动权就又回归到了你的手里。很多人就是能量太弱，不敢说“有事”这种短小精悍的回答，而是要添油加醋地解释很多。当然，如果你不想回答这么生硬，也可以直接反问对方：什么事?这样的话，对方把事情和盘托出，你也有时间去权衡利弊、应该怎么回答，也有时间去找合理的借口去搪塞对方。

③下策：跟对方解释。如果别人问你“明天有事吗”,无论你说“明天没事”,还是“明天我要去做XX 这件事”,对方都会在此基础上进一步拿捏你。如果你说“没事”,他就会请你过去帮忙；如果你说“明天我要去做XX 这件事”,他就会让你先把手头上的事情放一放，先去做他的事。因为所以，你说得越多、解释得越多，你的主动权就越弱，能量场也就越弱。

这时候，无论你答应对方还是拒绝对方，你的能量场都会被进一步削弱。而且很可能会陷入后悔不迭的情绪当中。

## 3. 引发共鸣

我听过这样一个真实的案例：某女子小A 被渣男骗财骗色，然后她去网上讲述自己被骗的经历。一方面是提醒大家不要上当，另一方面是发泄自己的不满，希望网友能帮她找到那个骗子。此时，评论区就有一个小C 回复她说：姐妹，我跟你的经历几乎一模一样，我也是被渣男骗财骗色。然后，小A 仿佛找到了知音，两个人还加了好友。加上好友之后，小C 讲述了自己的经历，也引导小A 继续分享她的受骗经历。由于小A 这时候已经完全没有了戒备之心，于是一五一十地讲述了自己被骗的详细经过。但是，小C 其实是个骗子，他本身就是一个骗财骗色的渣男，他寻找猎物的方式，就是在网上找那些已经上当受骗的女人。

因为这些女人说以会上当受骗，就是因为思维存在着巨大的漏洞，被骗一次，肯定还会有第二次、第三次......于是，小C 就通过另一个小号，继续上演炫富把戏，从而让小A 继续被骗财骗色。

这件事给我们的教训有两个：
①让一个人敞开心扉、对自己建立信任感最快的的方式，就是引起对方的共鸣，这时候，对方会把自己的所有秘密都抖落出来。
②不要随便跟别人讲述自己的“悲惨经历”,一方面可能会引来骗子；另一方面可能会引来更加悲惨的人，你们同病相怜、互吐苦水，只会强化彼此的负能量。

## 4. 假装关心

有些底层的父母跟在外打工的孩子聊天时，很可能会借着关心的名义来控制你、拿捏你。比如，他们会问你：钱够不够花啊?如果你说“我的收入很高，根本就花不了”,这时候他们就会通过各种手段来支配你创造的财富。如果你说“不够花，我是个标准的月光族”,这时候他们就会无计可施了。

还有些办公室的老前辈，借着关心的名义来打探你的家境，从而调整对你的姿态。比如，他们会问，你住得远吗?房间冷不冷?父母有人照顾吗?他们实际上想问的是：你有没有车、有没有房?如果
有房的话，质量如何、面积如何?父母有没有退休金?

如果你如实回答，透露了自己的家底，那么他们从此以后就可以肆无忌惮地拿捏你了。反之，如果你给他们模棱两可的回答，让他们去捉摸不透，那么他们自然就不敢轻易得罪你。举个例子，即使你父亲就是个在XX 局看大门的，月薪3000元，你也要跟同事说：我的父亲在XX 局任职，具体的细节不方便透露。

## 5. 捧杀法

我之前说过我当年从高人那里获取信息的方法，那就是“捧杀”对方。比如，我在公司聚餐时， 一般会选择坐在公司老板/主管的边上，然后问他们：“您是如何努力打拼，才一步步取得今天的成绩?我真的很好奇，因为我很迷茫，需要你的成功经验中获得能量和启发。”这时候，他们都会一五一十地给我讲他们当初创业的经历，如何突破一个个瓶颈的故事。讲完之后，他的“好为人师”欲望也被满足了，我也得到了关键信息，也受到了启发和激励，一举两得。

而且，这种戴高帽的捧杀法屡试不爽，比请客送礼好用多了。因为几乎所有取得过重大成就的人，都喜欢跟别人讲自己的“辉煌历史”,在讲述的过程中，自然会透露出很多核心的信息。
