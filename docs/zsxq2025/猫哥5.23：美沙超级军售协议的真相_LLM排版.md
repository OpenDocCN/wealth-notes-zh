
# 猫哥5.23：美沙超级军售协议的真相

川普中东4日游，分别和沙特、卡塔尔、阿联酋签署了价值超3万亿美元的投资协议震惊全球。特别是和沙特签订1420亿美元的军售协议，给很多正在为印巴空战而兴奋的人浇了一盆冷水。

最近几天，不少人有一个共同的问题，那就是在全球目睹了中国武器的作战实力后，沙特为什么依然头铁选择和美国签订历史上双方规模最大的单笔军售协议，而不是选择我们呢？

先看看美沙军售协议的大致内容：

合作领域包括5个方面：

1. 空军升级与太空能力建设：包括采购F-15EX战斗机、无人机、卫星通信系统及太空预警能力。
2. 防空及导弹防御系统：包括“萨德”反导系统、升级版“爱国者”导弹系统，以及AIM-120C-8先进中程空对空导弹。
3. 海上与海岸安全体系：计划采购濒海战斗舰(LCS)衍生的新型护卫舰、MH-60R “海鹰”直升机，强化红海及波斯湾的军事存在。
4. 边境安全及陆军现代化改造：包括布雷德利战车改进型、榴弹炮、装甲车等装备，重点应对也门边境冲突和胡塞武装威胁。
5. 信息与通信系统升级：包括C4ISR指挥自动化系统、遥感卫星及Link16数据链整合，旨在提升多军种协同作战能力。

除了合同约定时间和F-35战斗机没明确外，基本上把能囊括的全部涵盖了，还包括培训及后勤保障等。

那么,1420亿美元是什么概念?

根据俄罗斯卫星通讯社发布的2024年世界军售贸易分析报告显示，去年全球常规武器进出口规模总金额是1116亿美元，而排名第一占全球38%份额，具有压倒性优势的美国，其2024年全年军售金额约420亿美元。

换句话说，美沙这单军售生意就超过2024年的全球军售总量。如果按去年的老美对外军售情况进行测算，未来老美谁的武器也不卖，只保沙特的武器交付，也要超过3年时间，正好在川普任期内完成。

这可能吗?

是沙特钱多人傻，还是我们在印巴空战后有点过度自嗨?

本期内容，我们从多个角度审视一下这桩超级军售背后的真相。

历史总是惊人的相似。

2017年5月，川普当上大总统后，首次出访的目的地也是沙特，并且也和沙特签订了令人咂舌的1100亿美元的军售协议。

结果呢?

直到8年后的今天，生个娃儿都上小学了，实际交付也仅仅只有100-145亿美元。过程中出现的问题有很多，比如军工产能不足、以色列的干预、沙特在也门战争中的行动、卡舒吉事件等等。

反正都是过去时了，我们只看一个结果，那就是协议完成率只有9%-13%。

所以有了前车之鉴，对于这次军售我们也要心头大抵有杆秤。如果你被1420亿的金额唬住了，那就意味着大概率认知和美国红脖子MAGA群体划等号了。

再看看买家沙特的情况。

在沙特的国防预算中，刨除人事、装备维护和训练等大头(占2/3),直接用于新装备采购的预算近3年大致在260亿美元上下，再扣除本地化的20%，对外采购金额大致是210亿。

也就是说，这单1420亿美元的军售大单，以沙特目前的国防预算，就算全部啃老美的装备，也要用7年时间去消化。

那么,沙特会不会通过大幅提高国防预算来实现这桩交易呢?

个人认为不会，为啥?

沙特2023-2025年，国防预算在750-780亿美元区间，大致占国内GDP的7%左右，占财政收入的20%左右。说实话这个占比已经有点高了，未来逐步温和提高预算有可能，但是用2-3年时间满足这笔军售交易，比如对外采购增加250-300亿美元，加上杂七杂八的费用，国防预算占财政收入占比将轻松突破30%。那么这不仅将严重挤占其他开支，还要拉高财政赤字。

尤其是现在中东虽然乱，但主要方向还是在巴以、以伊胡方向，沙特并不是需要现在立刻马上走穷兵黩武的路线。

因为沙特最头痛的对手是胡赛和伊朗，伊沙关系大家都知道，在我们的推动下已经和解了，虽然现在看和解状态很脆弱，但是毕竟还维持起的。最核心的是伊朗现在就是一只病猫，沙特的外部是没有强大的威胁促使大幅增加国防开支的。

而最重要的是，沙特的计划是到2030年国防采购50%的份额是由国内自己满足，小萨勒曼也是一个狠角色，在财政压力和外部威胁减小的双重影响下，不大可能放弃这个中期目标。

退一万步说，就算沙特豁出去，要吃下这单军火生意，美国也只有干瞪眼，根本没法保交付。

现在，单洛马公司积压的F35战机订单已经超过100架，包括F-15EX战机产能都在暴跌，各个已经下单的国家都只有眼巴巴的排队等交付。

甚至连川普心心念念的空军1号，都已经把交付时间推迟到了2029年。川普一直抱怨服役35年的空军一号让他很丢面子，要不是这次卡塔尔懂他，估计川普这辈子都坐不上新的总统专机。

这还只是美国军工产能现状的冰山一角。核心材料82%的钨、98%的镓、60%的锗依赖我们，现在我们又在实施出口管制，未来美国不管造飞机、造导弹，包括雷达、发动机等一系列武器，产能都将全面被制约。

所以，从历史的角度，从沙美双方的角度看，美沙签订1420亿超级军售大单，根本就是一张空头支票，实际能履行20%就阿弥陀佛了。

上述情况，除了吃瓜群众外，其实美沙双方都门清儿，那为啥还要签呢?

对于川普而言，上台100天，几乎一事无成，俄乌调停，贸易战几乎把国内支持率搞崩了(最新数据是约43%),现在最需要依然还是“赢”,赢麻了那种赢。不管三七二十一，沙特你必须给面子，回去好给红脖子一个交代，他们只看数字，不看逻辑。

对于沙特而言，川普在中东地缘政治方面做出了积极的动作，这点大家可以回顾前期星球文章“特朗普的背刺”。而且，毕竟现在老美还是沙特最大的保护伞，结果既然大家都清楚，就不需要在乎过程，这个面子必须给。不给，万一狗急跳墙再来搞个阿拉伯之春咋办。

于是，这份超级军售大单应运而生，政绩工程而已。

还是那个观点，现在川普只要出手都是赢麻了，动不动就是1000亿、5000亿，甚至万亿级投资，只要你敢信，他就敢吹，反正吹牛不上税。

比如阿联酋宣布投资1.4万亿美元这事，作为一个GDP和我们广州市差不多的国家，其财政收入大致在200亿美元，你敢信这事靠谱吗?

再说直白点，川普现在最大的压力是美债问题，川普中东4日游，除了拉来天量投资数字，有见中东土豪们承诺大量吃下美债吗?没有吧

料敌从宽的原则我们不是不应该遵循，但是当你面对特没谱的时候，这个原则反而会干扰你的判断。

看到川普现在的操作，让我想起一个真实例子。

多年前，跟一个房开老板到处去看地，这个房开老板除了资产和负债，兜里现金拿不出来500万，但就是这个情况，他就敢和当地政府谈几平方公里的项目开发，土地整治的业务，而且上上下下都还围绕这个思路做PPT方案，签招商意向性协议(不打保证金那种)等。回到董事会，还要反复推动这件事，最后结果可想而知。但也就是因为这样，活生生错过了去库存的黄金时间，最后落得个一地鸡毛。

文末，谈一谈为啥在印巴空战后，沙特等中东土豪为啥还不考虑我们的军事装备这个问题。

要知道，印巴空战，除了歼10-CE和霹雳15迎来高光时刻外，真正出彩的是我们的整个空战体系。

而要实现“共享数据链”和“体系化”,卫星定位、算力、电磁、红外、光电侦查探测技术等一个都不能少。所以这就不是印度、法国等把霹雳15研究透，能搞出来的事。

而沙特的国防军事，长期依赖美国供应，虽然前面也采购了一些我们的武器装备，但还是处于零敲碎打的层面，要论体系，现在沙特还是整体以美国武器为基础。

要切换体系，其1很难，多年投入有可能打水漂，其二也不敢，除非美国退出中东，或者我们在中东摆2艘航母时刻保持威慑力。

而且，沙特现在的军购诉求是高端武器+技术转让的模式。在这方面，我们除了对极少数国家开了一点口子，现在政策还是处于限制阶段，这就导致我们的出口主要还是涉及无人机、舰艇、防空导弹等常规装备为主，和沙特的诉求还有段距离。

2024年，根据俄罗斯的数据，我们对外军售金额是32亿美元，排全球第十，市场份额不到3%。虽然我个人认为这个数据我们藏了一些内容，但是不管怎么说，还是处于第二梯队这个层面。

我们的军工也就是最近7、8年开始发力的，要大幅出口还有一段很长的路要走，印巴空战给我们军工带来了机遇，但也不要想着一口气就吃饱，军售不是你便宜好用，我就一定买你的。

所以，大家稍安勿躁，相信在不久的将来，会看到我们军售在国际军火贸易攻城拔寨的时刻。
