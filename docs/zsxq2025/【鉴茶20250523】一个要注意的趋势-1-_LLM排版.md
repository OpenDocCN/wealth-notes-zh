
# 一个要注意的趋势

重申一下。目前微盘和小盘股的成交量，已经创下历史新高，抱团股的代表，宠物，医美，化工等，有筹码松动甚至瓦解的迹象。“炒小炒差”代表的北证50，昨天大跌6.15%，加上大盘收复关税战以来的低点，又处于强压力区间。所以风格切换随时可能发生，昨天其实就是小盘切换和退潮。昨天你会发现，大盘股明显比小盘抗跌。而小盘跌的模式，就是以前某位同学讲过的，“扶老太太过马路”。每天小跌几个点，过两天拉一下给你信心，然后又跌。历来，不怕急跌怕阴跌。切回哪里的，大盘一点的和科技。大盘一点的典型代表是银保券三傻，目前还有空间的是保险。科技只是逐步开始回流，昨天进入了HVDC。相对稳妥和能看的是创新药。倒不是因为环境改善，因为大A的走势，一直是复制和延后港股的走势和题材。第一阶段是复制港股科技大厂的走势，涨科技，第二阶段是复制港股消费ppmt等的走势，涨新消费，目前这个阶段，是跟随港股创新药的持续上行，滞后映射大A的创新药。创新药看前排，有技术的。所以，如果你是操作型手上又多小票，要注意几个跑的指标和图形，包括但不仅限于，一加速后高位放巨量大涨，出现/连续出现长上影线，高位放巨量下跌，跌破5日平台，平台图形走坏，5日线下穿10日线。当然，也有可能是切换不成，新消费反攻倒算，甚至大盘萎靡，但冰点不正是你看机会的时候。600亿元援军在路上。
