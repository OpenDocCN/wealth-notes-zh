
# 全球贸易战已成定局

美国今年经济危机就会爆发。因为懂王无差别全球加税。如果懂王只是对中国，和对中国的一些马甲国家加税，都不至于经济危机。典型的马甲国家就是越南，越南的进出口货物gdp，已经超过越南国家总gdp了，很明显，中国的货物转到越南然后卖给美国，规避美国对中国的重税，这个玩法在18年就开始了，越南马来西亚都是马甲国家，而这此懂王堵上了这个漏洞，连马甲国家都不放过。更别说还有日韩欧还有巴西，懂王这次是无差别加税，只是中国加的最重罢了。

这个做法很像二战前夕。二战前夕全球经济大萧条，但还没导致战争，是美国内部通过法案开启贸易战对其他国家加关税，然后英法德随后也开打贸易战保护本国产业，导致全球贸易关税相互报复，最后加到100% ，最终让那次经济危机彻底无解且蔓延到二战爆发。如果是民主党搞贸易战，民主党不会对日韩欧搞这套，只会针对中国，甚至连越南马来西亚马甲国家都不会搞。

沙利文做的那个讲述已经很清楚了，说中国就算让货物去越南马来西亚洗个澡转卖美国，也可以接受，因为这属于慢慢对其他国转移产业链，日子久了这些马甲国家一样会承接从中国转移出来的产业，一样能削弱中国。懂王连这个时间都不肯等。或者说，他故意不等。因为他要将经济危机的锅甩给拜登。

而中欧都没有能力组建自己的自贸区。即使中欧都在往这方向努力，比如中国和日韩谈和东盟谈，比如欧洲和伊朗谈，但因为各种地缘政治问题，没有战争开路的自贸区是难以成行的。

中国是生产大国，需要能源原材料，需要消费市场，日韩是消费市场东盟是能源原材料，但台湾没有收回，中国压不服日韩，也难主导东盟。欧洲是生产和消费都还行，需要的是能源和原材料，俄罗斯和伊朗是天然和欧洲经济互补的，但现在普京上头要吞欧洲，伊朗的伊核都要被懂王硬解，欧洲也难以和伊朗联手，欧洲是最惨的。全球贸易战已成定局，或者说，第三次世界大战彻底全面爆发，也是定局。一如二战前全球贸易战故事。有意思的，前几次世界大战的导火索之一是巴尔干半岛，这次巴尔干半岛也热闹起来了。

懂王能对中国造成打击的，或者说懂王联俄制华最大的杀招之一的，就是能源大宗涨价，这一招还没有打出来。能源涨价的逻辑很简单，懂王硬解伊核，和沙特普京碰头达成共识，原油猛涨，中国这个生产大国就难受了，而懂王已经全力开发美国石油并不会因此背负太高通胀，中国会迎接这波输入性通胀。

还有个大宗，举个例子，全球70%多的铜都是供给给中国的，中国很多产业都需要铜比如新能源产业，而铜主要是海外主要是南美，这个太远中国军事够不到那里，一旦大宗涨价中国这个世界工厂将无利可图或者干脆停工止损。懂王全球无差别加征关税，即使对中国关税最重，对中国打击也并不精准，是美国欧洲一起难受 的局面。但一旦原油材料涨价，加上李嘉诚布局在全球的港口被懂王拿下，来一手针对中国货船加手续费，这些才是精准的打击中国。

而懂王对伊朗已经磨刀霍霍，有一个魔鬼细节。伊朗对懂王最强硬的回应是，如果懂王硬解伊核，伊朗会把导弹对准美国在中东的所有军事基地。懂王一听，说好的，我这就安排中东加装各种反导设置。是的，最近美国五角大楼忙着送萨德反导等系统去中东，每个军事基地都加装反导。这就是懂王已经在预备着防卫硬解伊核后伊朗的报复了。当然，即使加装反导了，美国军事基地被伊朗打击受重大损失也是定局的，因为防不住伊朗导弹还是很多的。懂王作出的中东军事基地受损，是战术性受损，他达成的原油大涨和终结伊核腾出手来处理东亚，才是战略目的。

也主要是内塔逼懂王先中东后东亚，不是内塔给懂王上压力，懂王连胡赛都不想炸，懂王知道，硬解伊核是给我们一次神仙助攻。懂王押注的就是，我们高层动荡，我们军方混乱，我们在这种逆风局的时候难以指挥军队去梧桐。但我们专打逆风局。神隐7天后，军队稳住了，东部战区+南部战区+火箭军都指挥的动，这是近期最好的信号。甚至传出了侠做检讨的事。看来，侠拿的不会是叶剑英的剧本，而是绍伊古的剧本，缓缓图之慢慢落地的剧本。而且你别看军演只持续了两天，其中差点爆发冲突。

中国放炮了，美军战机悻悻而归。

看起来，我们似乎只用等着懂王在内塔的压力下硬解伊核，然后就可以趁机梧桐。但懂王预判了我们的预判。很可能，懂王不会拖，他会在4月硬解伊核。——台海的季风情况，全年度有4个时间段受季风影响较大，分别是东北季风期（10月下旬到次年3月中旬），海面经常出现4到6级的大风，浪高2-3公尺。其次则是5月上旬到9月中旬的“西南季风期”，但风力较弱，平均在3级以下。最后则是2个季风转换期，分别是3月下旬至4月下旬，9月下旬至10月下旬，这两个时间段风向多变且伴有大浪，5级大浪占17%。——如果只参考季风影响的话，最适合解放军登陆的时间段是5月至9月中旬这段时间，台海基本算是风平浪静。整个3月4月和9月10月，台湾海峡都是风急浪高，不利于我们大规模登陆，而5678这几个月就没多少大风了，所以如果懂王不得不硬解伊核，懂王肯定选4月台湾风急浪高来硬解，而不会拖到5月给我们完整的有利机会。也的确，懂王给的最后通牒时间窗口，也是截止到5月为止。——3月初，特朗普亲笔致信伊朗最高领袖哈梅内伊，措辞严厉地要求重启核谈判，并下达两个月最后通牒，否则将面临严重后果。包括卢比奥那边也表态，俄乌停火谈判的事拖不到5月，4月就得有结果，不然就你们两边继续打呗。——卢比奥称，乌克兰已经表现出终战准备，美国将在数周内知道俄罗斯是否做好准备，而非等待数月。若俄罗斯搞拖延战术，美国不感兴趣，特朗普也不会参加无休止的谈判。普京肯定继续上头不鸟懂王，也就是5月份俄乌战争正式告别谈判希望，开始扩大化，波兰可能要正式下场了。因为欧洲各国前不久放风说，如果有北约国家主动攻击俄罗斯，就不触发北约集体防御条款，不用全部北约国家都下场干，波兰说只要你们给我钱，这个活舍我其谁啊。——在2月23日，波兰总统杜达为说服特朗普改变俄乌谈判的立场，在会场外等特朗普等了等了一个半小时。原本计划为期一个小时的会谈，只持续了10分钟。——波兰总统："俄罗斯已以毫不含糊的方式表明，不幸的是，这种邪恶依然存在——俄罗斯帝国主义。我始终强调：无论是沙皇的、苏联的、斯大林主义的、勃列日涅夫主义的，还是最后的普京主义的帝国主义，本质上都是同一性质的帝国主义。它始终试图奴役其他民族、夺取他们的土地和自然资源，并对其进行剥削。波兰必须下场扛起欧洲的兴亡，因为就算俄乌停战了波兰也捞不到好处，普京明确表示过乌克兰的重建不 letting波兰插手，你波兰可能会先手进入乌克兰境内，——目前波罗的海三国和波兰提出从禁止使用杀伤性 地雷的《渥太华公约》退约然后看王毅访俄。王毅访俄貌似谈的很曲折。——看起来王毅和普京的会谈四平八稳，互相无事提倡中俄友好。但魔鬼细节是在王毅会见普京之前，王毅和拉夫罗夫举行了多次会谈。不是一次，是多次。王毅和拉夫罗夫是谈了好几次后，才见到的普京。如果顺利的话，按照之前惯例，是王毅和拉夫罗夫谈一次后就见普京的。王毅要和拉夫罗夫拉锯多次才见普京，加上普京在王毅到莫斯科当天普京搞了征兵16万的超预期动作，这些都是中俄互信关系不如之前的一个微妙信号。——当然还好王毅和普京会晤的时候，普京没有提俄美关系正常化和中俄美一起搞无核化这些我们忌讳的事，我们虽然劝不住普京靠拢美国和劝不住普京扩大俄乌战争的决心，但至少明面上中俄还是哥俩好的。懂王拿诺贝尔和平奖是不可能了。因为懂王叫停不了俄乌，且懂王无法抗住内塔压力还不得不硬解伊核，更别说开打全球贸易战将全球拉入深渊的骚操作了。1.0懂王其实还是挺爱好和平的。但2.0懂王啊，不是爱不爱好和平，而是全球历史进程走到这个份上，只有战争才能解决这一系列问题了。不打世界大战，美国就是慢性死亡，马斯克说三个月削减一万亿美元预算，现在只削减不到一半，美国自救难成。在乱世大爆发之前，岱岱先出去散散心充充电吧，这样好在后面有更充沛的精力帮大家直播乱世。
