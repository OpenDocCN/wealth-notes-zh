
# 子明6.16换人如换刀

## 万达和格力的时代剪影
商海沉浮两面镜，时代更迭几人留。2025年5月21日，国家市场监督管理总局无条件批准了由太盟、高和丰德、腾讯、京东潘达、阳光人寿等机构组成的联合体对大连万达商管旗下48家目标公司的收购案。此次交易涉及全国39个城市的48座万达广场，交易金额或达500亿元。自2023年万达商管对赌协议危机爆发以来，王健林多次通过出售资产回笼资金。此次交易延续了“以空间换时间”的策略——万达商管在失去资产所有权后，仍保留运营权，以换取短期资金输血集团。据知情人士透露，500亿元交易款将优先用于偿还即将到期的债务及利息。

## 黯然失色的万达
网友：昨天何立峰会见了美国摩根士丹利联席总裁辛克维兹，还说欢迎美资金融机构积极参与中国市场建设发展。我观察下来有个感觉，贝莱德代表共和党，摩根代表民主党。

子明：贝莱德之前更多地押注了民主党，现在为了讨好特朗普，所以献媚力度比较大，把巴拿马运河收购下来了。其实贝莱德买港口的意图可以参考王健林搞扶贫和投资足球的逻辑。

网友：说起王健林，想请教一下明公，最近看到王健林一直在出售自己的资产，那他手上还有还剩什么核心资产吗？是不是快卖完了？看样子主体已经保不住了，不过这样出售了也好，至少王健林自己能全身而退。他是把一线和新一线的资产卖掉了，其他的还留着吧？而且貌似三四线城市的万达生意还可以。不过凭借这些还是远远不够还债的，我看现在负债还是一两千亿。

没办法，只能一步一步来，他已经算处理得好的了。这轮出售应该是早就已经谈好的，只是现在才进入逐步落实罢了。其实只要有人愿意接，问题就不大。只是可惜这次老王面对的收购方是单伟健，他的眼光太好了，老王很难占到便宜。

子明：王健林最后的结果肯定是比贾跃亭和许家印体面，但是中国首富沦落至此，也只能说上层建筑对经济基础的影响实在是太大了。网友：等老王的核心资产被分食殆尽，“上层建筑”会不会给王家留点生活费？子明：王健林最后的结果肯定是某种程度的老赖了，部分负债很难偿还，但留给王思聪的资产，还是可以够他们几代人生活的，这一点王健林不可能想不到，他15年、16年左右在海外买资产就是为了给现在留条路。

网友：王思聪现在可以动笔写红楼梦了。子明：跟红楼梦的逻辑是一样的。贾府的衰败已经是不可避免的，要想办法留一些资产传承给后代。王熙凤最大的错误就是没有听劝，没去提前购置可以传承的祖产。

注①单伟健：著名的风险投资人，太盟投资集团（PAG）执行董事长。单伟健在美国读MBA时候的导师是前任美国财政部长珍妮特·耶伦。2024年3月30日，太盟投资集团联合其他投资者，与万达签订了重大投资协议，以600亿元人民币的资金换取万达商管集团60%的股权，此次投资解除了万达面临的对赌危机。

注②:2012年5月21日，万达集团宣布以26亿美元并购价格外加5亿美元营运资金拿下美国第二大院线LAMC100%股权，在此之后王健林开启了自己的海外“买买买”计划，据不完全统计，2012年至2017年初的5年间，万达在欧美、印度及澳洲等地进行了超20次大手笔投资，投资总额高达2451亿元人民币，涉及地产、酒店、影视、娱乐、体育等多个领域。

曾经风光无限的首富，如今却要变卖自己的核心资产来偿还债务。经营策略、市场环境、上层建筑..各种因素都在影响着这位商人的最终结局。不过王健林这次算是平安落地，也为后代留下了足够的资产。

看完王健林的境遇，我们再来看看另一位企业灵魂人物——董明珠的近况如何？

## 格力的武则天
网友：我还想问一下，王健林的下场已是如此了，那与他同一时代的董明珠的下场又会是什么呢？感觉她现在疲态尽显，丑相不断了。

董明珠虽然赚到了超额利益，但是一来是通过合理合法的途径，二来牵扯的层面也不高，从根本上来看，她的最终结局和这几个地产大佬就是天差地别。但她也快不行了吧。从和雷军打赌开始，感觉董明珠就开始走下坡路了。不过这些年来她只是在进行自己的主线——给格力“换房本”，没有掺杂那么多上层建筑的因素，所以几轮变动对她的影响没那么大。

子明：董明珠现在的状态和晚年的武则天非常相似。她们都是身体、精力等各方面能力已经不行了，但是必须要向外表现出自己依旧很坚挺。董明珠涉及的“换房本”层级太低了，就只到珠海国资委这个层面。而许家印、王健林、甚至是贾跃亭这些人都是整个中国“换房本”过程中的一个角。

网友：不管是政治人物还是企业家，都很难一直留在台面上啊。居安思危，功成身退反而是很睿智的选择。那她“换房本”算成功了吗？成功了吧，她搭建的那个组织架构，算是以小博大，通过小比例股权来实现对格力的控制。

子明：嗯，基本完成了，她现在就是格力的武则天。不过，即使董明珠把“换房本”做到了极致，还是很可能会出现一个颇具戏剧性的结局，被她自己钦定的接班人搞政变夺取权力。

注③:2013年12月12日，在央视财经频道主办的第十四届中国经济年度人物颁奖盛典上，雷军与董明珠就发展模式再次展开激辩，并打下10亿元的天价赌局。雷军称五年内小米营业额将超过格力。如果超过的话，雷军希望董明珠能赔偿自己一元钱。董明珠应称如果超过愿意赔10亿。2019年4月28日，格力电器披露2018年报，格力以总营收1981.2亿元击败小米1749亿元，董明珠正式赢得与雷军五年前定下的“十亿赌约”。

董明珠虽然表面上看起来依旧在商界叱诧风云，但由于格力所涉及的“层级”太低，所以本身产生的影響和被影响的体量都是有限的，她的落幕注定无法和王健林、许家印等地产大亨相提并论。

千古兴亡多少事，不尽长江滚滚流。不论是王健林，还是董明珠，这些企业家和他们夙夜操持的企业，俱不过是更大的权力斗争舞台上或大或小的角色，有人粉墨登场，有人勉力维持，也有人黯然失色。而现在的舞台上，要站着的更符合国家发展的生产力了。

## 换人如换刀
这两年来，很多位置出现了换人如换刀的变化，譬如央行行长潘功胜，贸易谈判代表李成钢，哥俩作为连中候补都不是的“双非”，连个小“山头”都算不上，可只要放到了“一把手”的位置，无论是经济刺激还是贸易谈判，在同行的衬托下都能取得立竿见影的成绩。最明显改善的是宣传系统，自从北大神童执掌意识形态以来，内宣就不再搞什么战狼了，以中央媒体为黑悟空、哪吒、DeepSeek等造势为代表，舆论宣传成为了先进生产力发展的贤内助，也从简单粗暴的搞敌对，转向内在提炼精气神，被虚耗多年的大国自信重新恢复，经济基本面虽不如前，但民众的怨气却少了很多。

权力的集中会带来较高的上限与较低的下限，近年来也出现了不少一把手“取上限”的可喜局面。对此，我的观点是该喷的要喷，该夸的要夸，毕竟我们不能确保未来选拔的都是如此优秀的人才。
