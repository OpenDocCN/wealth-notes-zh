
# 大佳7.22行情私货

关于下午焦煤期货的涨停。首先，关于煤炭的文件，是真的。但这个文件并非今天下
发。能源局的文件是7月10日。其次，文件最核心的部分，我截图出来了。年度产
能，不能超过公告产能。月度不能超过10%。这句话在传播的时候被刻意忽略掉了。
各大群传的话都是“对于超能力生产的煤矿一律整改“.所谓超能力，是有定语的，即超
过公告产能。这个文件有用吗?是有的。对于煤价下跌之后，有些地方通过以量补价
的方式来扩产是一个打击。但是，这并不是焦煤下跌的根本原因。根本原因是钢铁的
需求下降，所以对焦煤需求下降。而钢铁需求下降，是因为地产开工，以及基建的下
滑。煤炭和光伏很不一样，光伏是供给的问题，而煤炭是需求的问题。所以，并不认
为这个措施足以会让煤炭反转。最近期货市场暴动，还是因为5-6月看空预期太强，
以至于稍微有好的消息，或者稍微有多头就被打爆了。我们观点很明确，不看好黑色
系，尤其煤炭的长期反弹。至于交易上，我们不做期货，所以不给具体建议了。但是
在股票上，我们会把交建涨停和煤炭期货涨停，当作是毕业照。

根据煤炭产能管理规定，煤矿应当按照均衡生产原则，科学安
排年度、季度、月度生产计划，合理组织生产。年度原煤产量不得
超过公告产能，月度原煤产量不得超过公告产能的10%。企业集团
公司不得向所属煤矿下达超过公告产能的生产计划及相关经济指
标。煤矿要严格按公告产能组织生产，对超能力生产的煤矿，一律
贵令停产整改。

本次核查范围为山西、内蒙古、安徽、河南、贵州、陕西、宁
夏、新疆等8省(区)的生产煤矿，已进入联合试运转的煤矿参照
生产煤矿进行核查。核查内容包括：一是煤矿2024年全年原煤产量
是否超过公告产能(其中已进入联合试运转的煤矿以公告的建设规
模为准，下同),2025年1—6月单月原煤产量是否超过公告产能的
10%;二是企业集团公司在安排2025年计划时，是否向所属煤矿下
达超过公告产能的生产计划及相关经济指标；三是煤矿在安排2025
年季度、月度生产计划时，是否存在不均衡、不合理的情况。
