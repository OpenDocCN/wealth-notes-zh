
# 以中国为中心的地图

某种意义上讲，华北大演习是对七十年代北方防御作战做了一次“成果验收”。1985年，在对外部环境做彻底重新评估后，军委下令不再准备应对跟苏联的大规模战争.....
虽然有些时候不愿意承认，但无论官方还是民间，其实都对中国战略空间的扩展、海外影响力的增大充满兴趣。

因为地理意义上的“战略空间”与“国际政治权力”存在着本能联系，正如我们很难想象日本、韩国之类的国家能够成长为超级大国一样——除非对外扩张。
在美俄即将发起停战谈判、国际格局暗流涌动的当下，本篇参考美国国家亚洲研究局(NBR)首席研究员Nadège Rolland主持撰写的一份报告，谈谈中国对于自身战略空间的认识。

## “以中国为中心”的心理地图

作为“陆海复合型大国”，1949年之后的很长一段时间里，中国的地缘处境十分糟糕——同陆地强权苏联之间没有任何缓冲国与自然地理障碍，同海洋强权美国在东南沿海直接对垒。
在度过短暂的“中苏蜜月期”之后，这种战略空间的脆弱性迅速凸显出来。
有战略学者这样描述冷战中期我国的地缘环境：美国在海上努力实施对华封锁政策，试图建立新月形军事包围圈，与此同时，陆地上苏联的威胁像达摩克利斯之剑一样悬挂在中国人民的头顶.....
自1960年代末《全面战备》以来，中国领导层事实上将海洋利益置于次要地位，转而专注于应对来自北方边境的苏联大规模军事入侵。
尼克松和基辛格敏锐察觉到了中方的这一战略变化，经过七十年代的共同努力，最终形成了华盛顿梦寐以求的对苏“环形包围圈”，如下图所示。
美国、日本、中国、巴基斯坦、伊朗(1979年之后既不亲美也不亲苏)、土耳其、西欧。

1991年苏联解体从根本上改变了中国的战略空间格局，也间接影响到中方的“世界观”，并为中美潜在冲突埋下伏笔。
具体来说有以下四点变化：
1. 与中苏长达上万公里的接壤边境线不同，中俄的边界线只剩下4200公里，中间多出哈萨克斯坦、吉尔吉斯斯坦、塔吉克斯坦、蒙古等四个“地缘缓冲国”。
苏联同中国在西北、东北均存在边境纠纷，苏联解体后，新成立的中亚国家为了稳定局面，迅速与中方达成边界协议，中俄边境问题也在2001年随着《中俄睦邻友好合作条约》签署而告一段落。

2. 1970年代初苏联曾在中苏、中蒙边境的陈兵百万(100万至120万)，迫使中方也在北方部署了多达150万至160万各类武装部队，双方形成类似北约-华约在欧洲平原的大规模对峙。
1973年，苏联调整了中苏边境军事力量的作战等级，确保中苏边境附近的部队与苏联驻欧洲部队处于相同作战水平，并在远东地区新增了35个导弹基地——这些基地的大部分目标都对准了中国。
苏联解体之后，俄罗斯国力孱弱，远东军区和西伯利亚军区加起来只有15个陆军师，其中绝大多数都不是满编师，一个师只有三四千名官兵，武器装备还是上世纪七八十年代苏联生产的。
2022年乌克兰战争爆发后，缺兵少将的俄军被迫把远东军区仅有的有生力量调到乌东前线，对中国俨然处于“不设防”状态，我国北部陆地边境处于历史上最安全的时期。
上世纪六七十年代，为了对付来自北方的坦克集群，中国曾耗费无数人力物力在华北、内蒙等广袤的平原地带修筑了众多“人造山”。这些“人造山”以土石堆制而成，设置有导弹和火箭发射口，地下则有储备仓库等设施。
1981年我军在张家口地区举行十几万人参与的“华北大演习”，演习共分为四个科目：一是模拟蓝军集群坦克进攻；二是空降与反空降；三是陆军师坚固阵地防御；四是战役预备队反突击。某种意义上讲，此次演习是对七十年代北方防御作战做了一次“成果验收”。1985年，在对外部环境做彻底重新评估后，军委下令不再准备应对跟苏联的大规模战争。

3. 冷战后，西方趁机对俄罗斯战略空间进行挤压。
比如：格鲁吉亚(2003年11月)、乌克兰(2004年11月)和吉尔吉斯斯坦(2005年3月)连续发生有西方背景的激烈动荡，七个前华沙条约签署国于2004年3月集体加入北约，以及北约进一步扩大的承诺等。
这一时期中国同样面临战略空间遭挤压的问题，于是2005年夏天，两国举行了首次大规模联合军演——“和平使命-2005”，并在之后形成常态化机制。
话说双方上一次大型联合军演要追溯到1957年8月中国、苏联、朝鲜三国军队在三国交界沿海地区举行的演习，真是恍如隔世。

4. 2001年美国发布的《四年防务评估报告》明确表示，应积极挤压中国的战略空间，包括扩大对华海洋地缘战略包围和加强同“南北锚”“南锚”即澳大利亚，“北锚”即日韩的联盟等。
报告强调：“该地区(东亚)可能出现一个拥有强大资源基础和军事能力的竞争对手，美国将对付处于战略十字路口的新兴军事强国。”
同年，小布什政府以“911事件”为由大举进入中亚腹地，中国战略界认为美方此举有巩固对华包围圈、遏制中方战略空间的意图(事实上确实延缓了中国对中亚的影响力扩张)。
2005年至2010年间，美国将其绝大多数的驻外武装力量集中在一条横跨中东、北非和东欧的“弧线”上，收紧了从伊拉克、阿富汗到保加利亚和波罗的海的“钳形弧线”，并以关岛为中心构建起对华海上包围圈，如下图所示。

2013年中方换届后不久，《人民日报》一篇署名钟声的评论文章写道：“有些人的身体已经进入了21世纪，但头脑仍然停留在过去，停留在殖民扩张的旧时代，停留在冷战思维和零和博弈的旧框架中……”
这段表述日后被反复引用，用来含蓄地表示对美不满。
2001年阿富汗战争后，美国事实上打造了西起中亚腹地，经印度、马六甲、菲律宾、台湾地区到日本的对华包围圈，给中方造成战略空间被强烈挤压的感觉。

俄罗斯在欧洲和中东方向感受到比中国更加强烈的“战略空间挤压感”——相较于海洋，来自陆上的威胁更加直接，刺激性更强。
回顾历史上的“陆海复合型国家”，面临战略决策时往往都会遇到所谓的“威廉困境”。
俾斯麦主政时期，这位铁血宰相认为没有“绝对安全”这种东西，德国任何超越英国海上力量的企图都会“引发徒劳的军备竞赛，或者被英国和法国海军的综合优势完全抵消”，因此更多将资源分配至欧洲大陆。
然而到第一次世界大战爆发前的二十年里，帝国新掌门换成了威廉二世。
当时威廉的办公桌上堆满了各种研究报告，有的反复宣扬发展强大海军力量的必要性，有的则呼吁建造“巴格达-柏林铁路”以及争夺欧洲霸权的必要性。
威廉最终同意了所有人的意见，导致德国四处出击，四面树敌，且资源分散。
有观点认为，假如当时威廉砍掉大部分海军预算，把所有资源集中给德国陆军，那么陆军就有相当大的概率打赢一战，提前上演二战时横扫西欧的剧本。

考虑到1910年代沙俄的国势与1940年代初的苏联截然不同(一个处于封建帝国末期，一个处于新兴帝国上升期)，且1910年代美国的实力与1940年代也不可同日而语，德国很有可能凭借一战胜利实现其主导欧洲大陆的夙愿.....
事实上，与专注于海军的岛国不同，“威廉困境”是每一个陆海复合型国家决策者都会面临的问题，有学者这样总结道：
“虽然海洋和大陆国家特征是不变的，但陆海复合型国家所走的道路是选择的结果，双向扩张需要运气和高超的外交技巧，最终可能会分散有限的资源。”

1942年2月23日，罗斯福向记者解释美国对于二战的总体策略。
接下来把目光聚焦到中国。
如下图所示，美国国家亚洲研究局认为，理想情况下中国的战略空间应包括广阔的大陆和海洋——以欧亚大陆亚洲部分(东南亚、南亚、中亚和西亚)为中心，向外延伸到西欧和非洲，及其邻近水域(南太平洋、印度洋、北极和南极洋)。
这个战略空间不仅是一个军事责任区，而且中国将有能力在该范围内行使一定程度的国际影响力。
从“理想场景”向内收缩一层，第二等级的战略空间西起乌拉尔山和阿拉伯半岛，南至澳大利亚，北至西伯利亚，东至第二岛链。
该场景中，除俄罗斯外，印度、澳大利亚和日本是最重要的三个国家，因此美国正拉拢它们形成了一个“大三角”(即美日印澳四方安全对话)，对中国在西太平洋和印度洋的战略空间形成“双向压缩”。
再向内收缩一层，朝鲜半岛、台湾岛和中南半岛被中方视为核心战略空间——历史上中国曾在朝鲜半岛和中南半岛上打过境外战争。

过去十几年间，由于来自陆上的威胁基本消除，中国将许多资源投入了海军建设，在极短的时间内打造出一支足以匹敌远东地区美国舰队的军事力量。
《海权与陆权的对决》一篇中分析过，接下来中方将面临艰难的“威廉时刻”取舍——要么走扩大整合边缘地带和心脏地带的模式，继续管控跟霸权国家的分歧；要么选择突破岛链限制，进行海权对决，同美利坚摊牌。
前者的策略大致为三条：
1. 稳定东南沿海，管理与美国及其亚太盟友的关系；
2. 巩固与北方俄罗斯的伙伴关系；
3. 向西向南推进，整合中亚、西亚和中南半岛的各项资源，形成类似美加墨那种高度统一的北美自贸环境。

这种做法被称为“西进东卫”，遵循“阻力最小”的原则。
简而言之，充分利用中国与俄罗斯、中亚国家、巴基斯坦和伊朗的良好关系，同时给海军有足够的时间加强自身能力，最终择机转向东方以突破岛链限制。
它很大程度上跟美国的大战略相契合——2020年代华盛顿放弃了之前的“双扩张计划”(大中东和大中亚)，转而专注于东亚和西太平洋，中国战略界称之为“西撤东进”。
当然，如何做战略选择从根本上讲还是要基于自身与对手的力量对比，以及增长趋势。
尽管过去几十年里国际政治被披上了一层层“文明的外衣”，但“从实力地位出发”的本质其实从未改变。
站在全球地缘政治的视角，只有少数国家/区域有潜力成为真正成为世界的中心，放眼望去，无外乎美国、中国和欧洲这三大板块。
美国凭借近两百年的努力，已高度整合了北美大陆，实力最强。
欧洲在两次世界大战中未能决出胜负，之后从经济维度到政治维度的一体化进程(欧盟)又困难重重，更不要说资源层面还得依赖进口。
中国目前只巩固了大部分的核心战略空间，无论从战略深度还是自给自足的角度看，离北美大陆均有相当大的差距，至少要再整合一至两个地缘板块(如中亚、西伯利亚、东南亚)才能与之媲美。

实际上，美国在“问鼎中原”、挑战欧洲世界中
心地位之前，也曾经历了长达一百多年的整合发展时期，绝非一蹴而就。
1869年，美国“太平洋铁路”贯通，首次实现以铁路连接大西洋海岸与太平洋海岸。

有人将国家描绘成一个有机体，必须为各种“空间”而奋斗，才能在充满激烈竞争的世界中生
存下来。
在殖民扩张的年代，此处的“空间”单纯指地理
空间。
再往后到了经济全球化年代，地理空间仍是基
础因素，但在它之上又多了贸易空间、金融空
间、能源空间乃至数字空间、AI空间等新内
涵。
或许未来某一天，随着革命性科技架构的突
破，地理空间对于国家发展的意义会大 幅下
降，战略空间的含义则将被进一步丰富。
1812年冬天，拿破仑兵败莫斯科之际，俄军前线
最高指挥官库图佐夫曾感慨道：“取拿破仑而代之
的不会是俄国或者任何一个大陆国家，而将是已
经控制了海洋的大国。”
几百年后，或许人们也会这样讲——取代美国海
洋霸权的地位的不是另一个海权国家，而是某种
新形态地缘政治权力的崛起。
