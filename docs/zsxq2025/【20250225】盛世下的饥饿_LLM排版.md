
# 盛世下的饥饿

在钟国漫长的历史中，曾涌现出许多被后人广为传颂的盛世，例如文景之治、开元盛世、永乐盛世等。自古以来，无数文人墨客对这些辉煌的时代不吝溢美之词，将其描绘成宛如人间仙境般的存在。然而，若我们以冷静而深刻的视角，从史实和第三方证据出发，重新审视这些所谓的盛世，便会发现这不过是上层拳贵阶层的一场奢华盛宴，而身处社会底层的广大百姓依旧深陷生活的泥淖之中，艰难求生、苦不堪言。

以距离我们较近的康乾盛世为例，英国使节马加尔尼在拜谒乾隆后，曾对当时的盛世景象留下了这样的记载：无论是在舟山，还是沿着白河溯，北上京城的3日旅途中，他都未曾看到任何仁民丰衣足食和乡村富足繁荣的迹象，映入眼帘的皆是低矮简陋的泥墙平房，屋顶上盖着稻草，几乎没有什么像样的绅士宅邸，更遑论舒适宜居的农舍。

马加尔尼还注意到，钟国百姓极度缺乏肉食，以至于凡能充饥之物皆难以舍弃。他提到，狗肉是人们的常见食物，而那些生活在水上的民众长期处于半饥半饱的状态，任何能够果腹的食物，即便已经腐败变质，依然被视为珍馐。当地百姓每次接过他们的残羹剩饭，都会感激涕零、千恩万谢。由此可见，这被后世称颂的康乾盛世之下，百姓的生活竟是如此困顿不堪，连温饱都成为遥不可及的奢望。

正因如此，许多有识之士才会将这一时期讥讽为饥饿的盛世。这个称呼可谓精准地揭示了其背后所隐藏的民生凋敝的真实状况。

当然，或许有人会对此观点表示异议，认为这样的看法过于片面，是英国人对中国社会的恶意抹黑，是古早味的雾霾滤镜。那么，不妨让我们将目光回溯至钟国历史上首个备受推崇的太平盛世——文景之治，深入探究其中的真相。

后世许多人认为，真正称得上盛世的非文景之治莫属，尤其是那位赫赫有名的汉文帝。他曾免除了全国百姓长达13年的田租，这一举措在历史上堪称空前绝后。史书记载，当时的情景是：京师之钱累巨万贯，朽而不可校，太仓之粟陈陈相因充溢露积于外，至腐败不可食。这段话的意思是，京城的国库中钱财堆积如山，铜钱上的绳索都因长期存放而腐烂，以至于无法清点；皇家粮仓里的粮食，年复一年的堆积，旧粮之上又添新粮，甚至满溢至舱外，直至腐烂变质无法食用。

这样的描述无疑向世人展现了国家财富累积到令人惊叹的程度，让人对那个时代的繁荣与富足充满向往与敬仰。但是，在这些华丽辞藻之下，却会发现令人不寒而栗的现实：景帝去世仅3年，即公元前138年，这片昔日看似繁荣昌盛的土地便陷入了一场骇人的灾难，全国范围内爆发了严重的饥荒，饿瓢遍地，甚至惨烈到发生了人吃人的悲剧。

史书中记：大饥人相食。这不禁让人疑惑，曾经国家的财富积累堪称惊人，钱财多得无法计数，粮食满得溢出仓库，为何短短数年间便跌入如此深重的危机？更令人深思的是，此时的汉武帝刚刚即位，还未开启他日后颇具争议的奢华生活，也没有发动耗资巨大的对匈奴战争，国家依旧沿袭文景时期推崇的黄老之道，奉行无为而治的方针，尽量不干涉百姓的日常生活。那么，究竟是什么原因引发了这场突如其来的全国性大饥荒，甚至导致了人相食的悲惨局面呢？答案是通货膨胀。

自汉朝建立以来，历代皇帝都不遗余力地大量铸造铜钱，甚至进一步将铸币权下放给各地的诸侯王。如此一来，市场上的铜钱数量便像泛滥的洪水般迅速膨胀。表面上看似金钱充裕，实则隐患丛生。

汉朝统治为何如此执着于铸造如此庞大数额的铜钱呢？原因很简单，他们企图借此手段，大规模的攫取民间财富。当市场上充斥着海量铜钱时，官府便能够借助这些货币大肆收购百姓手中的粮食，使得皇家粮仓的储备日益充盈，甚至出现仓库堆积如山的景象。就这样，当时的统治者们将百姓财富抢掠一空，却在史书上留下了所谓的盛世之志。

至于其他被称为盛世的时期，百姓的生活同样苦不堪言。以开元盛世为例，杜甫曾感慨道：朱门酒肉臭，路有冻死骨。百姓生活困苦，难以维持生计，许多人沦为流民，最终逼得玄宗皇帝不得不下达禁逃亡诏，以遏制逃亡潮的蔓延。再看永乐盛世，表面上呈现出万国来朝的繁荣景象，实则是明朝用巨额财富换取附属国的臣服，每逢朝贡，这些国家都会获得明朝赏赐的丰厚财物，而最终为此买单的仍然是无辜的百姓，导致民间苦不堪言。

近几年，全国从上到下都在内卷，人们对此厌恶又彷徨。究竟是什么原因导致钟国成为一个内卷地狱？但实际上，翻看钟国几千年的历史，一个残酷的真相便是，内卷才是常态化的民间生活，像过去几十年的蓬勃生机，这样的时间，几千年来恐也屈指可数。由此可见，历史上的所谓盛世，虽然在国家综合实力、正治安定以及文化发展等方面达到了当时的巅峰，但从民生角度来看，广大百姓的生活依旧挣扎在贫困的深渊之中，难以摆脱困境，这其实是一个典型的国强民弱状态，远远称不上真正意义上的全面盛世。这样的繁荣，说到底只是上层社会的辉煌时刻，属于世家大族、门阀、权贵等统治阶层，他们尽情享受荣华富贵，肆意挥霍权力与财富，在这种所谓的盛世之中，社会贫富差距迅速拉大，形成一道难以逾越的鸿沟。

上层社会的豪门权贵们过着极尽奢华的生活，锦衣玉食流连于花天酒地之所，沉浸于纸醉金迷的享乐之中，对他们而言财富似乎源源不断、取之不尽。与这种奢靡生活形成鲜明对比的是，生活在他们统治之下的钟国普通百姓，百姓们穷困潦倒，衣不蔽体、食不果腹，艰难的挣扎在生死线上，而所有这些苦难和挣扎，都被盛世浮华的表象所掩盖，无人关心、无人过问，成千上万生命的挣扎和生死，最多只能在史书的角落里留下一句：大饥人相食。

在钟国人的历史记忆中，有一些词语像是条件反射般，带给后人美好的憧憬，轻徭薄赋必然是其中之一。每次历史中出现轻徭薄赋，通常与所谓盛世紧密相连。在大众的普遍认知中，这些阶段无疑是王朝最为繁荣的高光时刻，百姓理应安享太平，享受盛世的安乐。但是，如果我们从更长远的视角去审视，将时间线拉长，便会惊讶的发现，隐藏在轻徭薄赋背后的，竟是钟国社会版本的马太效应。

以汉朝为例，曾经辉煌一时的昭宣中兴，仅过了53年，整个王朝便走向了灭亡的深渊。而大唐的开元盛世，如同一场绚烂却短暂的烟火，仅14年后，安史之乱的爆发犹如晴天霹雳，毫不留情地宣告了大唐盛世的终结，从此，这个庞大的帝国开始了无法逆转的衰退之路。其他朝代也未能逃脱这样的命运轨迹。或许很多人会归因于皇帝的昏庸无能，比如李隆基，晚年他沉溺于享乐，重用李林甫、杨国忠等奸臣，也有人认为是信僚体系的腐败，官员们为了个人私利巧取豪夺，最终引发了民变，还有人可能会认为，这与军队的武备松懈和战斗力下滑有着密切关系。

研究钟国王朝周期律，有无数个切入点，每一个切面都有其理论依据和历史证据。诚然，这些原因都是导致王朝衰退的关键因素，但今天我们要讨论的切面——轻徭薄赋，却是一个不太常见的视角，许多人眼里轻徭薄赋是一项对国家和百姓都有利的理想政策，百姓的负担减轻了，生活压力似乎也随之消失，理应过上更加幸福的生活。但事实远比这更复杂。在古代，社会的生产力极为低下，这导致百姓的抗压能力非常脆弱，这既有人为的制度因素，也有钟国季风性气候带来的变数，这意味着，即使完全免除赋税，百姓的抗风险能力依然低下，一旦遭遇意外，他们往往不得不将土地变卖，沦为地主家的佃农，一旦失去土地，国家再怎么减免税赋，对他们也没有任何意义。

以汉朝为例，当时的税率非常低，实行的是30税1，汉文帝还曾一度免除田租长达13年之久。昭宣中兴时期也曾多次减免田租，这些政策为皇帝们留下了美名，但对那些失去土地的农民来说，这些惠民政策就像是空中楼阁，与他们的生活毫无关联，真正受益于这些轻徭薄赋政策的，是那些拥有大量土地的地主阶级，土地越多，他们缴纳的税额越少，财富便越积越多，如此一来，贫富差距迅速拉大，差距之大令人震惊。

更严重的是，地主阶级对佃农的剥削非常严重，很多情况下，他们征收的田赋甚至超过了农民的半数收成。原本旨在惠民的轻徭薄赋政策，在现实的扭曲下，反而加剧了贫富差距。古代中国的税收主要依赖于土地数量进行征收，然而，豪强大族凭借着权力与财富，通过行贿当地官员逃避税务责任，为了完成税收任务，这些馆员便将原本应由豪强承担的赋税转嫁给了无辜百姓。如此一来，便出现了皇帝假装减税，官僚假装收税，富人假装交税，只有钟国普通百姓最终承担了一切。

除此之外，古代王朝除了土地税外，还有人口税这一沉重负担，而那些沦为奴隶的人不需缴纳人口税，随着奴隶数量的增加，未缴税的群体逐渐庞大，随着卖身为奴的人数不断上升，权贵富人的需求也达到了饱和，毕竟他们并不需要如此多的劳动力，于是，那些失去一切生存希望的农民，最终成为了流民，为了填饱肚子，他们必然会向当地正府发起反抗，这也正是许多所谓的盛世时期，农民起义如星火燎原，蔓延不绝的根本原因。

与此同时，豪强大族随着财富的急剧积累，开始寻求更多的正治权利，朝廷中的官员多来自这些豪强大族，他们相互勾结扶持，形成一个庞大的利益网络，尤其在地方正府层面，彻底垄断了信员的任命权，甚至从数据的角度来看，县城婆罗门指数的高低，与王朝寿命呈高度相关。东汉末年，信员的任免已被世家大族牢牢掌控，我们熟知的袁绍，便常以四世三公的显赫家世自豪，这正是当时社会阶层固化，权力高度集中于豪强世家的典型写照。

这些问题相互交织、彼此影响，使得社会不可避免地陷入了严重固化的僵局，豪强大族以惊人的速度吞并农民的土地，导致富者田连阡陌，贫者无立锥之地的局面愈演愈烈。因此，从时间角度来看，轻徭薄赋的政策，往往未能如人们所期待的那样真正造福百姓，反而让他们陷入更深的贫困，进一步加剧了社会阶层的固化，和贫富的严重分化。

实际上，所有的统治者都清楚阶层固化的后果不堪设想，但是肉食者鄙，人性的贪婪与自私，使得他们死死抓住手中的既得利益，始终不愿主动放弃已经到手的权势与财富，哪怕社会危机四伏，正是这种短视的逐利心态，最终将王朝推入不可挽回的深渊，留下一片残破的废墟，让后人唏嘘叹息，又不可避免的开始下一次轮回。

防民之口，甚于防川，钟国历代帝王们，尤其是那些所谓的雄主，最害怕的是什么，是担忧想提前上位的太子，是惧怕权臣势力做大、结党营私，亦或是顾虑宫廷太监被外人收买，暗中对自己图谋不轨，这些因素固然都可能对皇权构成威胁，但他们并非历代皇帝最深层的恐惧所在，真正令帝王寝食难安的，恰恰是百姓过上安逸舒适的生活。

这一观点乍听之下似乎颇为悖论，与仁政爱民的传统理念背道而驰，但若仔细探究历史的深层逻辑，便能发现其中隐藏着极其深远的正治考量和权术谋略。众所周知，古代钟国社会以农业为立国之本，国家的经济命脉高度依赖于农业生产的兴衰，统治者对此了然于心，因此他们无不竭力促使农民持续劳作，确保土地源源不断的创造价值，他们深知若百姓过上富足的生活，人们便要享受安逸，不再有动力去垦荒拓土，或是兴修水利来防范自然灾害，一旦农业生产陷入停滞，国家的税收来源便会锐减。

桶治阶级从来不事生产，它们永远都是仁民身上的寄生虫，并且统治阶级的人数和欲望，总是如雪球般越滚越大，这就要求钟国百姓必须要不断创造更多的财富，才能满足统治阶级膨胀的需求，一旦民众因生活富足、安逸而躺平，这部庞大的国家机器便将面临瘫痪的危机。因此，为了防止这一局面的发生，统治者往往精心筹划，通过各种手段巧妙调控，让百姓的生活始终维持在恰到好处的状态，既不能太过困苦，以致民不聊生，引发动乱，但也绝不能过于舒适，以免削弱其生存斗志，这种状态被商鞅总结为弱民，被当代钟国人称作内卷。

故此，统治者们巧妙地利用税收这一经济杠杆，依据自身的统治需求与国家财政状况，灵活调整税率和税目，从百姓辛勤劳作的成果中抽取一定比例，使百姓始终承受生活的压力，不敢有丝毫懈怠，只能持续不断地投身于生产活动以维持生计。与此同时，徭役也是他们手中的重要工具，征调百姓参与国家的大型工程建设，如修筑长城、宫殿、陵墓等，既满足了国家基础设施建设，统治阶层奢华生活的需要，又使百姓无暇享受闲适时光，在繁重的体力劳动中耗尽精力，从而始终处于忙于生计的状态。

此外，古代社会犹如一座层级分明的金字塔，各阶层之间壁垒森严，而这种等级秩序，正是统治者维持权力的核心基石，孔子曾言：君君臣臣，父父子子，在这位被后世统治者祭奠的圣人看来，这种固化的社会秩序正是理想的治国之道，唯有各司其职，国家方能长治久安，这完全就是为统治者量身打造的道德监 狱。

对于居于金字塔顶端的统治者而言，他们深知一旦百姓在安逸的生活中获得足够的闲暇与思考空间，便可能开始审视周遭环境，进而察觉社会中的种种不公，对现有的等级制度心生疑虑，甚至深入思考其合理性与正当性，一旦这种思想在民间蔓延开来，便如星火燎原，引发对统治权威的挑战，进而动摇整个社会秩序的根基，为防范这样的潜在危机，统治者往往不惜采取严苛手段，以贫困与饥饿的阴影笼罩百姓的生活，他们通过垄断土地、加重赋税、限制商业等方式，挤压百姓的生存空间，使其在温饱线上苦苦挣扎，疲于应对基本生存问题，从而无力也无暇思考更深层次的社会矛盾，最终在无奈中接受既定的社会秩序，心生畏惧，不敢轻易反抗。

从统治者的角度来看，百姓的生活状况与统治成本息息相关，若百姓生活富足、衣食无忧，能够凭借自身能力妥善解决养育子女、购置住房、看病就医等问题，便会对统治者给予的小恩小惠视若无睹，统治成本也将大幅上升，这是统治阶层极不愿见到的局面，反之，当百姓长期处于贫困困境时，生存的艰难和不确定性，会极大地削弱他们的反抗意志，此时，哪怕统治者稍加恩赐，比如适量减税和赏赐，都会令百姓感激涕零，视其为救世主，将所有希望寄托于他们，从而更加依附权威，心甘情愿地维护统治秩序。

在钟国历史上，狂热的爱国主义是统治者经常拿来用的御民之术，因为统治者深知，若百姓的生活过于安逸，他们就不愿充当统治者的炮灰，为其征战扩张或平息叛乱，正所谓千金之子坐不垂堂，一旦百姓富足，便会变得谨慎，考虑得失，不愿冒险，但若百姓一贫如洗，便不会畏惧牺牲，正如俗话所说，光脚的不怕穿鞋的，统治者正是通过剥夺百姓的鞋，使它们毫无顾虑，唯有不断前行，试问，今天钟国最狂热的战争分子，一定不是权贵、富豪。
