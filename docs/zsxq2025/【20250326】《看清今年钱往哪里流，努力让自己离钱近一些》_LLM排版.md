
# 看清今年钱往哪里流，努力让自己离钱近一些

## 国家账本，要看四本账

政府财政一共有四本账，每本账都有自己的收入和支出。

第一本账是一般公共预算，收入来源主要是税收和非税收入，非税收入又包括了行政事业收费、罚没收入等。

第二本账是政府性基金预算，粗浅理解就是“土地财政”的账本。这本账中，大部分的收入都来源于土地出让金，也就是地方政府卖地挣的钱。除此之外，还有你坐飞机交的机场建设费，也就是民航发展基金，以及你买的彩票等。

第三本账是国有资本预算，也就是国企上缴的收益和用于国企发展的支出。这本账体量一直都非常小。

第四本账是社保基金预算，也就是我们交的社保的收入和支出。第四本账是单独核算的，只能调入不能调出，所以我们一般看前三本账目就行。

看账本有什么用呢？比如我们看下今年的第二本账，就是土地财政账本，里面说的很清楚，去年收入6.2万亿，支出10.15万亿，差额高达4万亿。什么意思？就是为了卖地，地方政府前期拆迁，七通一平，环境治理，配套设施等成本先付了，但是土地最后没有都卖出去，支出还多付了4万亿。

但地卖不出去，对于地方财政的影响很大，尤其是三线和以下城市，他们的的一般公共预算是动不了的，因为很多支出，像教育、环保、卫生、社保、工资等，都非常刚性，没办法腾挪，所以没有土地财政收入，这些地方政府就是“月光族”，收的钱收上来很快就得花出去，没有一点闲钱，所以也就无力支持项目建设、企业扶持等一系列事项，没钱付企业欠款。

我这么一说，你就能明白很多现象背后的本质原因了。

## 从2025年财政安排，看懂钱怎么分

了解完关于预算的基本知识后，接下来就是我们要重点分析的，关于2025年的财政资金的安排，就是找出国家真正重视的事情。怎样找呢？我们关注两点，一个是不同事项分类的“增速”，增速反映了中央重视这个事项的“绝对程度”；另一个则是各个事项的占比，占比反映了中央重视这个事项的“相对程度”。说白了，增速看的是投入有没有大幅增加，占比看的是份额到底有多大。

今年的总支出中，各个分类事项的加权平均增速是4.4%，这其中，科技是8.34%、教育是6.11%、外交国防是7.2%，这三项都大幅度超过平均增速，因此这些都是中央非常想做的事情。

而债务付息是7.73%，文旅体育传媒及其他支出是7.32%，这两项虽然也大幅超过了4.4%，但却各有情况。因为债务付息是被动为之，而文旅体育传媒及其他支出则不好界定，所以后续要继续看政策出台和具体文件，我帮你留意观察。

至于住房保障、社会保障和就业、卫生健康，这几类和平均增速差不多，都在4.4%上下，说明去年是怎么搞的，今年还是按照老方式继续。

最后，灾害防治及应急管理支出，今年砍了600个亿，增幅是-12.62%。另外，今年年初，中央纪委也重点提到，要在消防领域进行反腐。总之，今年跟灾害防治及应急管理相关的产业，你可能要有心理准备，预算下降，日子不会太好过。

## 2025年机会在哪里？怎么离钱近一些？

看完了国家对大类事项的资金安排，那么从具体项目来看，机会在哪里呢？这里我为你总结了四方面的机会，涉及内需、产业升级、科教兴国和地方政府投入四个环节。我们一个个说。

第一，是内需环节的机会。去年安排了7000亿元用于“两重”，就是国家重大战略项目建设，和重点领域安全能力建设”，以及3000亿元用于支持“两新”，就是大规模设备更新，和消费品以旧换新。效果多好呢？根据初步统计，汽车、家电以旧换新，分别超过680万辆、6200万台，带动产品销售额超1.3万亿元。今年的《预算报告》里则是提出，安排8000亿支持两重，5000亿支持两新，其中两新的资金扩容比较明显。具体来说，手机、平板、汽车、家具，包括刚刚官宣的电动车，都会从中受益。

除了两重两新领域的机会，报告里还有一句话很关键：“深入开展现代商贸流通体系试点城市建设，推进实施国家综合货运枢纽补链强链提升行动”。这意味着，国家打算通过打造节点城市，来降低全社会的物流成本。在这些城市大力建设物流园、物流基础设施、转运仓等，再通过节点城市向更低一级的县乡分发货物。2024年国家已经选取了首批20个城市，包括天津、石家庄、大连、哈尔滨、上海、杭州、宁波、合肥、福州、厦门、南昌、济南、青岛、郑州、武汉、广州、深圳、成都、西安、银川。你可以关注这个机会。

第二，是产业升级中的机会。今年，中央财政安排了118.78亿元，用于制造业领域专项资金。这笔钱主要要用于两个方面，一是新型技术改造，二是数字化升级。简单说，就是中央出钱，地方配套，给传统制造业升级，并且中央还安排了2000亿元的超长期特别国债，用于支持设备更新，比去年增加了500亿元。加快设备更新和技术改造，也是全面推进“国产替代”，通过大规模设备更新，换成国产的，对于国产替代的供应链来说，今年会有更多的机会。

另外，2025年还会启动新一轮专精特新中小企业财政奖补政策，去年已经支持了1039家小巨人企业和专精特新中小企业，这个政策今年还会加大力度。因此，企业要努力通过评审，获得这些资格，既有直接补贴，也会有更好的政策支持和信贷资金给到你。

第三，是科教兴国下的机会。报告提到“全面建立基于专业大类的职业教育差异化生均拨款制度”。这意味着以后教育会加大拨款给一些优质职业院校，包括它们的热门专业，比如智能制造相关的，会获得更大的红利。而一些没有特色专业的学校，生存空间可能会受到挤压。

报告还提到“中央本级科学技术支出安排3981.19亿元、增长10%，进一步向基础研究、应用基础研究、国家战略科技任务聚焦”。按照以往经验，加大基础研究投入，不管能不能出成果，首先受益的，肯定是科研设备制造厂商，他们是核心受益群体。而且目前中国科研设备大多依赖国外采购，国内替代品质量不行，其实也是个机会，高精度、高质量的国产科研设备赛道很有潜力。

第四，是地方政府投入中的机会。我在整理了去年各地政府的投资情况之后发现，地方还是以基建投资为主，但有两块值得关注：一个是城乡冷链等物流基础设施，也就是农村和城市间的基建，地方政府现在很重视这一块。另一个是新型基础设施、能源基础设施等，增速非常快，主要包括5G、大数据中心、人工智能、工业互联网、特高压、新能源汽车充电桩、城市轨道交通七大领域。你可以在这些领域找找机会。也就是说，城市里的数字基建、连接城市和农村的物流基建，会是地方持续发力的方向。

## 最后，我还有三个强烈提醒：

第一，做项目要远离高债务省份的非核心项目，远离还在搞传统基建的不发达的三、四线地区，而是要到东部地区，努力承接关乎国家战略的新项目。

第二，今年做企业，要注重核心技术，警惕技术壁垒低的项目。同时，在国产替代的大逻辑下，谨慎对待高度依赖进口技术的行业，比如高端设备代理。

第三，要有行动力。我无法叫醒一个装睡的人，面对国家的剧烈转型，你要跟着动起来。
