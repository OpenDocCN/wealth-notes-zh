
# 米联储见闻20250307——须臾扫尽数千张

川总又搞事情了，签署行政令将比特币作为战略储备货币，这是要给比特币进行主权信用加持，说白了，就是要给比特再点一把火，吸引更大规模的机构甚至包括美联储或其它国家的央行来持有比特币，把比特币拿来做美国的化债神器。而且川普今天已经公开说了，他以及经不在乎美股的涨跌了，因为他要为美国的长期发展打算。外界批评他主导的贸易战引发投资者对美国经济及通胀走势前景的忧虑，美股及美债价格均下滑，科技股走势的纳指已较高位跌逾10%，并进入技术调整区。较多反映传统企业的道指失守43000点。对于这种言论，川普强调在制订关税措施不会考虑股市表现，并相信长远而言关税措施可让美国变得更强大，又指出美股下跌是与全球主义者 (global i 有关。总体来说，川总现在就是在主动刺破泡沫去杠杆了。

今天沃尔玛方面已经在跟兔子的供货方提出要求，必须降低售价从而补贴增加关税造成的价格差，说白了，零售端不愿意通过加价刺激通胀，所以掉头将压力转向了供货方，你想要继续进入美国市场，就要自己承担关税费用，从某种意义上说，就是让兔子企业承担了关税负担，实现了川总当时所设想的预期，所以如果兔子企业不能尽快打开其它市场，依然要依赖美国市场的话，只能被人摁着放血，没有还手之力了。要想把自己的命运控制在自己手里，就需要积极去扩展其它市场。以美国持续收缩的外交策略，这会是一个长期的趋势，靠咬牙坚守是比较难的。

目前，美国对关乎非法移民和芬太尼关税的豁免将在4月2日到期。届时川普将宣布新一轮关税，包括面向世界各国普征的所谓对等税和针对汽车、药品及半导体等特定产业的进口税。从3月4日推迟到4月2日，川总就是这么一步步的去挤压对手的外加空间，我们反倒是完全不管了，随你折腾，不陪你玩。

今天，有外媒对中美领导同时举行的会议进行比较，川普新任期的首场国会联席会议讲话 兔子总理李强的政府工作报告。 一边的画风是“我们在43天内取得的成就比大多数政府在4年或8年内取得的还多。”老子天下第一，华盛顿只能排第二。而兔子方面的政府工作依然是一成不变的四平八稳、脚踏实地，给人感觉世界在大变，但兔子好像什么都没发生一样。这种不变在如此激变的时代凸显出巨大的价值。不过兔子的政府工作报告在谈及今年的困难和挑战部分对外部环境的描述篇幅大幅增加，字数上去年为40个字，今年为149个字，如果字数多少可以体现情势的话，那就可以简单理解为难度系数翻三倍。

这次川普的演讲不同于拜登以往做的国情咨文，美国总统正常是每年初会发表国情咨文，总结自己过去一年的施政报告。按理说川普还没刚够100天，按常理是不会安排这种国会演讲的，而川普这次是由众议长约翰逊邀请他进行联席会议演讲。本身的发起动机就可以说透着谄媚，结果也可想而之，所以川总这次演讲，主打一个自我吹捧美化，主要是川总的个人秀，是一个集体颂圣的聚会，川总在掌声中走上讲台，发表长达约100分钟的演讲，打破了前总统克林顿2000年创下的89分钟国情咨文演说的时长纪录，万斯起身鼓掌达到95次，几乎是每分钟会起身鼓掌一次，相当于进行了一场长达100分钟的有氧运动。但现场气氛从一开始便充满敌对，共和党频频站起来鼓掌，民主党不断有人骂骂咧咧，而且冷眼观看共和党鼓掌；一边站一边坐着，现场还是相当尴尬的。在讲话中，川总普怒喷拜登政府的经济、能源、产业、边境、性别政策等，称拜登是史上最糟糕总统。除了抨击拜登，就是在表扬自己。川普说，我签署了近100条行政命令，采取了400多项行政行动，以恢复我们这片美好的土地上的常识、安全、乐观和财富。只能说，川总自我感觉棒棒哒。其中最重要的两项成绩是移民执法和精简政府，总之各项工作都很棒，这6周工作简直帮美国重新镀了一次金。

虽然我们看热闹，但美国国内舆论围绕他的最新演讲反响不一。支持者认为，川总演讲感情充沛，再次展示自信、好斗的治理风格和领导方式，聚焦多项竞选承诺。批评者认为，这是川普的招牌表演：措辞浮夸，取悦选民，满嘴谎话。会场里共和、民主两党冰火两重天，会场外民意同样严重撕裂。共和党议员从头到尾热烈鼓掌、欢呼。民主党议员基本一言不发，举着“谎言” “虚假”等标语牌。唯一高声抗议的民主党议员格林在演讲开场仅三分钟便被请出场外，还有一位来自佛罗里达70岁民主党议员参加完会议回家就气死了。由于川总这场演讲传统上并非必须举行，明显是奔着跟兔子两会打擂台来的，所以在国际舆论上，欧洲也是冷眼观瞧，就在美国来回为难乌克兰之际，今天兔子跟乌克兰签署了一个农产品的贸易协定，这也充分说明兔子的想法，美国整谁，我们就帮谁，美国把敌人搞得多多的，我们就尽量把朋友搞得多多的。

昨天我们说了，美国现在是在回归杰克逊主义，所以内部的满意还是相当重要的，对美国民众来说，川普新任期刚起步，民众仍抱有期待。在川粉看来，川普做事有效率，在提升政府办事效率、放松管制，打击非法移民方面的表现较为满意，如果美国经销商可以压住进货价保证在关税战打开之后不会明显增加物价负担，相信还会进一步支持川普。

至于兔子的政府工作报告，虽然每年有调整，但整体文本结构保持持续性和稳定性，通常都是先回顾一年工作、展示工作难点、安排下一年的工作。所以每一年的政府工作报道，看上去平平无奇，但却非常符合全面深化改革的大趋势。兔子虽然也在经历巨变，但是一直还是以大稳定包裹小变量，即便是在疫情期间，在22年前后，在前年，去年经济最困难的时候，其实都蕴含着巨大的危机，但都在整体有序的组织体系下慢慢消化掉外力的冲击，让很多风险在不知不觉中被化解掉，所以每年工作报告，当领导说非常不容易的时候，一般的群众并没有太大的感觉，总觉得政府也没做什么，但是对比外部条件和环境的变化，才会知道有时候不变已经是持续努力换来的结果。兔子政府强调政策的稳定性和持续性，正如多年政府工作报告一直都会强调的稳中求进。

今天，著名的反华反共写手袁莉忍不住在纽约时报吐槽，川普上台让美国像是在经历一场文革，在此之前，龙应台们也应声落泪如丧考妣，他们把川普身上发生的一切等同于兔子政府，事实上，两国政府确实都很强调忠诚，但两份忠诚却截然不同。川普从不掩饰他喜欢别人拍他马屁，白宫网站现在全是各种吹捧川总最棒的文章，甚至还会罗列各方人士对川总的忠诚发言。兔子政府也强调忠诚，而且是忠诚，也是围绕领导的忠诚，但是这个忠诚背后还需要你能干事，你能为国家干事，你要扎扎实实做事情。不管是拿下秦刚还是摘掉李尚福，背后都有一个潜在的逻辑，你对国家不忠诚，对党不忠诚，就不容于权力层，不会因为你仅仅是忠于某个人就能够扶摇直上，实际上在过去这几年也不乏所谓江新军落马，有些也曾是嫡系，所以这个忠本身不仅仅是对个人的忠，更多的是共同事业的忠诚，包括有些企业，也许一直不是积极向党靠拢的，但如果 you利国利民，也还是愿意扶持鼓励的。这几年，不管是行业，还是文化产业，又或者是社会风气，难得出現了良币驱逐劣币的发展趋势，这背后就跟权力本身不徇私，国家利益高于一切紧密联系，只有在这样的 大环境之下，才能让所谓理想主义生根发芽，让好人好事不受委屈。

所以过去一段时间，兔子内地舆论场也在发生微妙的变化，之前人们对兔子自身的不满和抱怨正在减弱，催生这一变化发生的因素，一是兔子硬核科技的 大爆发，二是川普的冲击，当然也包括狗粮减少，但是让龙应台，袁莉这种死硬分子都哀嚎，确实是大环境发生了扭转，川总功不可没。我们以前说软实力的斗争有时候就是荷塘效应， 一切变化都在不知不觉中发生，星星点点会在一夜之间开满整个池塘。德意志银行研报发布《兔子吃掉全球：兔子的斯普特尼克时刻》，高盛研报《启航2025:在兔子股市中斩获超额收益》；摩根士丹利研报《DeepSeek 点燃市场，全球投资者重估兔子科技股价值》，更不要提英国大儒BBC和经济学人，最近铆足了火力在捧华踩美。各路美粉最近纷纷陷入静默，朋友圈更是集体消失，今早星舰再次爆炸，朋友圈尬吹的人几乎没有了。川总的上台或许会给兔子造成一点麻烦，但是也让很多人丢掉幻想，凝结共识，相比于川总制造的麻烦，也可算是有得有失。
