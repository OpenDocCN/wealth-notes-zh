
# 台湾的两难选择

近期岛内知名学者吴玉山受邀进行了多次演讲，从大陆压力下台湾的选择出发，谈到“政权自由度”“Trump 意识形态”等深层次问题，颇具启发意义。本篇在其核心观点的基础上拓展而成，供读者朋友们参考。

## 台湾的民主与自由

吴玉山是前国民党高官的儿子，属于蓝营色彩的学者，主张两岸对话协商，其演讲PPT背景用一只和平鸽来表达自己的期待与担忧。近现代历史上，公民自由和民选政治起源于英美，与之相较，法国、德国等欧洲大陆国家迈入现代民主则很晚。前者大约在十九世纪末，后者经历过魏玛共和国的短暂尝试，要直到1945年之后才真正步入正轨。

吴玉山认为，英美先于欧洲大陆诞生民权，是因为其作为“岛国”的地缘环境比较安全。民主制度重讨论、轻决断，重妥协、轻动员，不利于在危急情况下做出快速决策，只有在相对安全和平的环境下才能充分发扬公民参政议政权利。历史上，身处百战之地的国家或地区往往都采取了政治集中制，而当代意义的民主更多是在战后相对和平的环境下才逐渐孕育推广的。

简单来说，和平是公民权利之母，一旦爆发战争或政权遭遇重大威胁，常见的做法即用（部分）牺牲自由来保障安全。假如失去和平的大环境，社会及个人的自由也将被断送，古今中外概莫能外。

为了方便观众朋友们理解“和平”与“自由”的关系，吴玉山做了个很有趣的类比。在新冠流行期间，病人死亡的原因主要有三：其一是病毒的攻击，其二是免疫系统的过度反应，其三是原有病症并发。而社会面临压力时就像人体生病时一样，将产生“免疫”（即各方面限制性措施）。

平衡“免疫”与“过度免疫产生的副作用”是非常困难的，后者很容易将正常细胞一起消灭掉，造成超出疾病（外部压力）本身的负面影响，尤其是在长期不间断的外部压力之下。

## 台湾的未来走向

对台湾转向的预测（图中红色箭头）。吴玉山认为，“民主”和“自由”是两个维度，不能画等号。按照民进党当局的一系列限制性做法，未来台湾有可能从“自由民主”渐渐变成“非自由民主”，即空有一个民主框架，但社会自由度大大降低。

赖清德当局正通过各种方式不断限制岛内民众的自由，以防止渗透为名阻断两岸正常交流。在民进党的舆论叙事之下，“和平”“谈判”已经近乎等同于投降和背叛。演讲中，吴玉山回忆了民国在大陆和台湾的转型历程。中山先生创建民国之处，也曾想着“民族、民权、民生”，但很快就因为面对战争威胁而失败。

老蒋在大陆时几乎一直在训政+打仗，败退岛内后，除个别国民党权贵外，台湾社会整体上的自由度其实跟同期的大陆差不多。这里面老蒋个人因素固然占到了一定比例，然而无法否认的是，当时两岸乃至围绕第一岛链的冷战对抗确实也起到了很大外部影响。设想一下，在两岸高度军事对抗甚至经常爆发小规模冲突的情况下，作为弱势一方的台湾势必对内紧绷，战战兢兢，四处抓“渗透”。

同样的情况在大陆六七十年代“全面战备”时期也是类似的，为了战备，就要牺牲自由。

台湾自1949年5月开始施行长达三十八年的军事戒严，直到1987年7月才解除。吴玉山认为，台湾社会转型的八九十年代，恰好就是中国大陆推行改革开放、两岸趋向缓和、台海爆发军事危机可能性最小的时期，这并不是巧合。换句话说，中美关系和两岸间的良性互动，与台湾社会转型和岛内公民权利提升是紧密相关的，尽管绝大多数台湾人从未意识到这一点。

## 和平与自由的代价

这部分的最后，吴玉山以当下正在进行的俄乌停火谈判为例，谈到了一个至关重要的问题——你愿意为了和平牺牲什么，改变什么，或者说愿意为达成战争目标放弃什么。是领土，金钱，还是现行生活方式与社会制度？过去几年里，即便战火极少蔓延到俄罗斯境内，但俄民众也遭受了诸如军事动员和舆论、外汇、出入境管制等一系列限制措施。至于说乌克兰民众所遭受的代价，就更不必多言了。以此为鉴，台湾人是否愿意承受类似的代价呢，又是否愿意先行做出部分妥协呢——这些问题在岛内是缺乏理性、严肃讨论的。

疫情期间的检测和戴口罩是一种典型的为了安全而部分牺牲自由的场景。再往广义延伸，我们为了赚钱而上班，被工作各种限制，也是为了财富而牺牲部分自由。

1930年代的苏联是一个为了保障自身安全、为了准备世界大战而几乎牺牲一切公民自由的国家。

接下来把目光聚焦到Trump身上。吴玉山在接受采访时表示，Trump自出道以来其实从未输过，他在2016、2020、2024年的三次选举中，普选票得票率与得票总数均呈现上升趋势（输掉的2020是因为民主党得票暴涨），足见其获得越来越多选民的支持。

鉴于Trump惊人的吸票能力，迫使其党内同僚和竞争对手也学习他的政策模式，导致美国政治普遍“MAGA化”。换句话说，Trump不是一个人的现象，而是一个国家的现象，不是Trump改变了美国，而是美国社会本身发生的变化造就了这样的总统。吴玉山认为，Trump是当代美国的一面镜子，其言行举止在很大程度上反映了美国目前的状况，透过他我们可以更清晰地看见美国的全貌。

## MAGA主义的特征

吴玉山指出，今日世界局势变局，最根本的原因是中国经济改革以及之后的国力大幅提升。现阶段美中权力接近，中国相当于过去的苏联（军事）与日本（经济）的总和版，对美国而言已进入权力移转理论者所认为的危险区域，双方冲突势所难免。这便是Trump主义的第一个特征——争霸。

美国近些年内部收入差距不断扩大，在分配问题的归因心理上，民主党会提出传统的资本与劳动关系作为解释，MAGA则会提出外国与国内合谋者的责任，两者相比，后者更具吸引力。吴玉山认为，一旦美国经济发生明显衰退，如1930年代的大萧条，在所得分配持续恶化的情况下，右翼民粹主义将更加剧烈地兴起，刺向外部移民与少数族裔。此即Trump主义的第二个特征——民粹。

那么如何解决跟中国争霸以及国内收入不平衡的问题呢？Trump给出的药方是通过对外施加关税、对内给企业减税等方式促进产业回流。此即Trump主义的第三个特征——重商。

吴玉山表示，对Trump而言，霸权竞争涉及到美国国安，重商主义涉及经济，右翼民粹则攸关选票，他三个都想要做，但不同的政策目标之间会产生冲突，这让Trump的政策看起来十分矛盾，以至于朝令夕改。比如说，以争霸为出发点，北约对于美国十分重要，可如果从重商和民粹的视角出发，Trump又觉得盟友赚了美国的便宜，威胁要退出北约。

即在上述三个特征里，Trump最看重的是内政和民粹——他就是要让支持者“爽”，让美国不断“赢”，这是其一切言论和行为的根本出发点。更进一步说，Trump摒弃了之前历届美国政府平衡外部因素（如维持盟友体系以巩固霸权）和内部因素的做法，他选择“内部因素绝对优先”（此处包含在美洲的扩张和内部清洗），为了内部利益即使得罪盟友也无所谓。这种思维之下，Trump对盟友的主要诉求变成了“你应该如何帮助美国再次伟大”，“你能为美国创造什么价值”，而不是美国如何为你提供安全保障。

尽管很多朋友轻视民主党及西方左派秉持的各种思潮，如：进步主义、包容、多元化、圣母、开放移民等，但某种意义上讲，这套思潮对于美国维系全球化和实施海外干预是有利的。反过来，MAGA这套“让XX再次伟大”的理论将对全球化及国际间交往带来巨大冲击。

简而言之，温和左派更容易彼此相处——类似过去几年的民主党和欧洲，而极右其实很难彼此相处，因为每家都专注于自己优先/伟大。

美国钢铁已经没有竞争力了，而且尝试高价收购美钢的又是盟友日本，但仍被美国政府强势否决，其中一个重要原因就是是美国国内反对收购的民粹声音超过了理性判断。

吴玉山称，台湾需要尽快适应MAGA风格，少谈民主党那一套“价值观”话语体系——类似的表述只会徒增Trump反感。

作为岛内学者，吴表示自己对Trump的台海政策完全没有信心，“格陵兰”议题在白宫的优先级或许都要遥遥领先于台湾。展望未来，吴玉山判断MAGA很可能彻底取代共和党建制派的地位，跟民主党轮流执政，并将战略重心持续聚焦于美国国内及周边地区。

吴玉山认为，MAGA不是“孤立主义者”，而是十九世纪的地缘扩张主义者，倾向于抛弃力所不及之地，专注于周边影响力，这对中国大陆来说是有利的。说来有趣，早在1972年中美领导人第一次会晤时，中方就毫不掩饰对华盛顿右翼政治家的喜爱——“我喜欢右派，相对而言右派的人当权更让我高兴。”
