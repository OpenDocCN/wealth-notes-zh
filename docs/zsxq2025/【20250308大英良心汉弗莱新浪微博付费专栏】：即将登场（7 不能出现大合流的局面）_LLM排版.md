
# 大英良心汉弗莱新浪微博付费专栏：即将登场（7 不能出现大合流的局面）

目前一个普遍警觉的情况并不在于特朗普本身。而在于特朗普带动的这股潮流给我们的威胁和压力，这种威胁和压力会在5~8年后出现，那个时候才是最危险的。

## (一)右翼合流

右翼是当前白人世界渐渐在形成的共识。这一点是最危险的。如果长期读首席文章的，应该记得2022年，首席提出过，无论左翼桑德斯、涡轮还是右翼特朗普，最终都会走向合流，形成一个名字：法西斯。

战争不可避免是大概率事件，无非是发生在欧洲还是亚洲的问题。美国率先在右翼问题上走出一步，右翼民粹主义吸引选民的政党无需在政策制定上煞费苦心，而只需通过直白明确的话语和简单粗暴的政策将自身与右翼民粹主义挂钩。反过来，该行为也使白人群体在身份诉求不断被满足中变得更加极端。政党与目标群体在身份族裔议题上的相互强化使政党轻易实现了一呼百应的政治结果。

问题的根本都在于制造业的问题，而制造业的问题还不是西方放任金融业扩大化的后果。其实这些问题，西方学习中国就行了：对金融业进行强力监管，加强对制造业和科技的投入，开展强力研发，问题基本能解决。

但是西方这种二极管文化(称不上文明)不愿意这样，只能奉行鸵鸟政策：把问题外推。

制造业的问题由来已久。2012年，美国白人底层劳动者以62.6%的比例在制造业工人中占据主体地位，所以，美国喊制造业回归，因为制造业空心化必然导致大批白人失业。1970年以来，白人在美国劳工中的占比不断下降，到2022年仅剩55.4%，到2031年会跌至50%以下。即使是保住工作的白人劳工，也因传统制造业日益稀薄的利润而逃脱不了工资增长停滞甚至降薪的命运。对于大量白人劳工来说，曾经“一房一车一狗”的中产生活如今难以维系，财富缩水、社会地位降低已成为摆在面前的残酷现实。

近年来美国少数族裔的人口数量、法律、文化地位均快速上升。2020年美国人口普查数据，非拉美裔白人占比已经降至57.8%，而在18岁以下人口中已不占多数。预计到2045年白人比例将降至50%以下，届时美国将不存在主体族裔。但是民主社会主义这帮人，也就是美国左翼，在"肯定性行动”“伟大社会”倡议等政治举措也使少数族裔的基本权利受到法律保护，所以右翼对此是很不满的。

民主社会主义自身的问题，让白人对自身生存状况的焦虑与对其他少数族裔的怨怒，走投无路的绝望和不断升级的愤怒刺激他们走上右翼民粹主义之路。这个路线目前在美国走通了，接下来就是在欧洲走下去。

虽然这个路线看上去在欧洲会受到欧洲传统社会

主义左翼的制约，但是一个不争的事实是，潮流本身就是潮流。就是在不知不觉中，无论欧洲是否愿意讨论这件事，这件事本身已经成为了潮流本身，民众的思路不可能不被打开，无非是什么时候被打开的问题。

如果欧洲的右翼在逐步上台的过程中，完成了对欧洲尤其是西欧本身的思想统一，至少是大部分思想统一，那么一个史无前例的超级大号法西斯就会出现。因为社会主义是对抗法西斯的唯一有效武器。指望阿拉伯人这帮人是根本无效的。

但是社会主义阵营尤其是欧洲阵营的垮塌和部分向白人右翼妥协(比如，匈牙利)的现实，让亚洲社会主义阵营承担了绝大部分压力，这个压力自然而然就在我们身上。网络上细说的所谓1VS8的格局，就有可能出现。这是出于极端考虑的最极端现象。

为什么马斯克要频繁插手欧洲的选举，特朗普要直接搅动欧洲的安全神经，这一方面是战略欺骗，另一方面确实是存在美国右翼准备在欧洲内部推动欧洲的不安全感，制造事端，让欧洲右翼登台。主要采取几种策略。第一，帮助欧洲极右翼政党从主流政党那里争夺选民。这些极右翼政党包括奥地利的

人民党、西班牙的社会党等。由于很多欧洲国家的传统大党执政表现不佳，又被极右翼政党描绘成腐败、无能而又冷漠的“政治精英”。极右翼政党趁势捞取政治资本，极力争取选民。

第二，迫使主流政党吸收极右翼的政策主张，包括限制移民、对伊斯兰教等加以管控、减少欧盟对相关国家主权的钳制等。

第三，推动极右翼政党在欧洲议会增加影响力。开始在欧盟总部布鲁塞尔利用欧盟的机制、平台和欧洲议员津贴宣扬其反对欧洲一体化的政策主张。

最大的特点就是法西斯化，一个很典型的案例就是，上过战场的亚速营分子，对日本极右翼社团和美国极右翼分子进行过军事训练。

## (二)思想上的懈怠才是最大的危险

全球意识形态当前出于变革的前夜，要么向极右翼转向，要么社会主义的曙光会出现在美国本土。这两者是向左走还是想右走的问题。但无论怎么走，压力大小都取决于中国是否能抗住这一波。

这并不是危言耸听，在前面几篇里，首席说过这个观点：一旦战略欺骗不成功，那么美国内部就会出现革命的前兆。

当前的局面，简单来说就是美国人认为单靠自己的力量无法掌控世界，那就放下身段，拉着整个西方一起来整治南方世界，包括现有南方世界的左翼。

实际上，当前我们这个局面并不是很危险的。1949年建国以来，最危险的局面是1960年后，也就是苏联老大哥跟我们翻脸以后，在那个时候我们的地缘是最危险的，从20世纪60年代开始，苏联从社会主义国家日益蜕变为社会帝国主义国家。赫鲁晓夫试图联手美国并开始对中国施压，勃列日涅夫时期又在与美国展开全球争霸的同时，持续在中国北方边境进行军事高压。这迫使中国与日益力不从心的美国双方有了战略合作的意向。

而我们的处理方式就是，1962年打击印度入侵。在当时那种情况下，我们如果应对不力，美苏绝对会暂时放下彼此的对抗，多头出击，分裂甚至更迭中国政权。但是我们成功的顶住了美苏的压力，并成功利用了两个帝国的矛盾和猜忌，独立自主，连续爆炸原子弹、氢弹，并在第三世界获得了广泛的支持和认可。到七十年代，重回联合国。也就是利用了10年时间，从最危险的局面成就了我们的喘息期。

当我们进入喘息的时候，美国进入了最危险的局面。1972年，毛主席接见了尼克松，导致中国迅速向海洋边缘地带倾斜，80年代中国改革开放，加入世界市场经济使中国经济迅速发展的同时也拉动了边缘地带的整体性上升。

而我们跟美国的握手，让美国摆脱了最危险的局面，带来的结果是苏联的解体。

现在无非就是美国人想制造一次我们的最危险局面，但是谁来解体呢？俄国吗？还是美国自己？

毛主席曾经有一段很经典的话：中美关系改善，不是美帝国主义放下屠刀立地成佛了，也不是我们团结第三世界反对美苏两霸的方针改变了，这在上海公报中讲的清清楚楚嘛，你可以找来再读读。我们有些同志看不清美国帝国主义的本性，有必要向党内高级干部打个招呼。大佬文集圈由于苏联大兵压境，害怕他们对

我们发动突然袭击，这边中美关系又得到改善，就产生了崇美恐美，想依赖美国的思想。人家拿苏联屯兵远东来吓唬你，再讲点你爱听的话，抛点小恩小惠，就忘记了美国才是我们最大的危险。

我们内部有一些人，还不是少数人，虽然一直在接受美国帝国主义是中国人民最凶恶的敌人的教育。但一旦国际形势发生了某些变化，他们的头脑就开始马放南山了。

一是对美国和欧洲的历史不了解，对美国这种帝国主义本质的国家犯下的罪行也不太了解。二是理论水平不高，特别是哲学思维水平很低。

三是不愿意坚持自力更生的立场，喜欢偷机取巧，不想深入研究，总觉得拿来主义最好，最后就是倾向于投降算了。

不仅在某些干部当中有，在某些知识分子中、在某些搞金融的人里面最多的是最新的，自己没有脑子，脑子全在美国媒体的新闻通稿里，一旦有风吹草动，就立刻说，我们应该跪下，唯一的气节就是一句话：用出卖中国人民的利益来讨好美国。

从目前来看，双方已经走到了最后比拼制度之争的时候了。

在政治制度上，美国一直把自己标榜为自由民主的灯塔，而把中国的社会主义民主政治视为"威权政体”，并在国际上不断渲染“民主对抗威权”。

但随着金融资本对社会和国家权力支配能力的不断增强，美国的政治跟它们的贵族一样，越来越呈现出“返祖”的现象：具有强烈的封建主义和奴隶主色彩：极少数的金融垄断巨头和精英经理官僚集团在最上层，中下层管理人员和中等工商业资本家处于中间层，而小资产阶级和众多普通工人群众处于最底层。

而我们的最突出的特点是寻求人民利益的最大公约数，使选举民主和协商民主得到有效统一。

两国深层的制度对立，使中国顺利成章地成为美国危机转嫁的主要对象。这就是美国近年来对华战略转向的重要原因。

但是从美国目前来看，我们这一篇文章所说的极端情况暂时不可能出现，因为资源调动能力实在是要求太高了，美国人现在没有那个心气。

美国如果铁了心要从根本上破坏中美关系，我们也会奉陪到底，这也是驻美国大使馆发表帖子的原因。

这一个多月的特朗普身上已有了当年赫鲁晓夫和戈尔巴乔夫的影子：开始跟过去的盟国“亲兄弟明算账”了。

当一个国家开始把政治当买卖来做的时候，就是在战略收缩。

但是即便这样，我们仍然不宜开展扩张性外交，而要采取威慑外交和军事行动：因为我们不能把好不容易因为美国衰落造成的群雄并起的局面，演变为因为我们的扩张外交，造成大家都把矛头对准我们，从而瞬间为美国解围的局面。

1941年毛主席提出的“熬时间，不伤元气”的方针对于我们争取胜利仍有指导意义。
