
# 宠物经济三阶段

宠物经济的第一阶段是吃，通俗说是狗粮猫粮。那几个玩家我们都熟悉，中宠，乖宝，相对低位的是佩蒂，北交所有个路斯。中国的猫狗食品基本都是山东人搞的。给它们卖铲子的是牛磺酸。

第二阶段是给动物看病，很多人都没有保险，猫狗有，随便去宠物医院打个针，吃个益生菌，好几百没了。目前在搞的是瑞普生物，主要做各种狂犬疫苗，给狗子看心脏病的药，还开了连锁医院。

第三阶段是宠物玩具，尤其是宠物服饰。目前在搞的是依依股份，天元宠物，天元还有跨境和重组的概念。但国内做的宠物衣服都很便宜，充其量几百块，国际上有的是一件上万的猫狗衣服，爱马仕本来一开始就是做马具的。之前刷到过一个视频，一个女孩子给她的猫弄了个衣柜，衣服好几百万。

本文适合懂技术和高手根据逻辑去操作，市场成交量越小越低迷，反而是抱团和各路题材的天下，预期你的预期。水平一般的，定投去拉战线。总体来说，食品仍然是大头，但不少宠物食品已在走加速趋势，低位的相对安全，盯好高标。一旦瓦解，要随时记得跑。
