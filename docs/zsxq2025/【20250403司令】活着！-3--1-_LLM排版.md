
# 活着！

前几天日本爆出一个让人匪夷所思的新闻，一个80岁的出租车司机，在开车的时候出现恍惚造成车祸，让一个年轻的姑娘因此而丧命，这事引起日本社会的一片哗然，然而，争论的焦点居然是有没有对出租车司机做相应的健康检测。因为，悲剧发生之后，作为当事人的出租车司机，居然不知道发生了什么事，处于一种懵懂状态，日本的媒体报道出来说这位出租车司机平时长期服用某种药物来保持清醒。

这个出租车司机引发的车祸事件，日本社会和外面的人关注的焦点不一致，我们中国关注的是年龄，他们日本关注的是资质审核。在我们中国看来，不管你怎么审核资质，即便提交再多的检查结果和健康证明，好像与这个事情的核心理内容也不相关，这个事情的核心是什么呢？这个事故的核心是：80岁的人不应该开出租车！

想一想，一个80岁的中国老人通常是什么状况呢？别说开出租车，就算骑自行车都很容易摔倒，有的老人在这个年纪，甚至走路都成问题。80岁都差不多有曾孙了，四世同堂的老祖宗，不应该安享天年吗？

日本人也是东亚人，也是黄种人，他们日本就那么特别，那么老当益壮呢？我有一次在大阪，出了机场的时候，有一个老式英国产的出租车停在我面前，下来一个头发都白了的老人，我以为他是上一个乘客，结果他是司机。那时候下着雪，地面结着冰，这个出租车司机老头差一点摔倒，这个出租车司机老头说的话我听不懂，我只能听懂一些简单的日语，有多简单，你懂的！所以这个出租车司机老头讲什么我完全听不懂，这个出租车司机老头不停的手舞足蹈的示意我上他的车，然后还不断的重复：阿依配！阿姨陪！上车之后，看着后座上的屏幕，我才明白，他之前说的是：AliPay！意思是可以用支付宝支付，你不用担心没有现金。这个出租车司机老头可能看出来我是中国人，所以才不断重复他唯一会说的单词：AliPay！差一点误会了，我心里在想：不愧是樱花妹之国，就算有陪伴服务，阿依听起来还行，阿姨就算了！

为什么突然想起这件往事呢？我听到那个新闻之后，脑子里的第一画面就是那个白发苍苍的出租车司机，当时我不知道，您贵庚，这句话用英文怎么说显得尊重，就用小学课本上学的那句How old are you？问他，他可能也听不懂，拿了瓶水给我… 然后我就不说话了！

到了酒店，这个出租车司机老头飞快的下车，然后从车的后面绕过来，给我开车门，按道理说享受这种服务，应该理直气壮和理所当然，然而，当一个头发全白，佝偻着身子的老人这样做的时候，我一点都感觉不到被尊重，反而有一种说不出来的尴尬。

怎么说呢？这个出租车司机老头不应该出现在这样的场合，这个出租车司机老头可以提着鸟笼在公园里散步，也可以系着围巾，透过落地玻璃窗站在阳台上看雪。唯独不应该出现在这样的场合，扮演这样的角色，以这样极不相称卑微和力不从心的无奈去极力的扮演一个这个出租车司机老头并不胜任的角色。如果一个人是这样，那是命运的捉弄，如果随时随地都出现这样的错位，那可能出事了，出什么事呢？显而易见！

一个亿万富翁跑到工地上去体验生活，哪怕他把亿万富翁写在脸上，周围的所有人也只会把这个人当成一个苦力。因为亿万富翁不应该出现在那个场景中，就算出现，也应该是前呼后拥一大堆人。先有人来清场，这个亿万富翁站在中间，双手插兜，不置可否，不发一言，转身就走，这才是这个场景中大家公认的样子。如果不是这个样子，就不应该出现在这个场景之中，如果不是这个样子还出现了，并且经常出现，那这个亿万富翁可能真的出事了…

本文真正要说的，已经说了，之所以搞得这么绕，是因为过于直白就会看不到，因为直白的不应该出现。当这个绕的出现在这个场景，并且被你看到了，就说明…你懂得！
