
# 学习笔记，2025.3.27

1. 理性看待消费，是要看品牌，看传承，看贬值率，看市场占有率。买车不是买菜，要谨慎选择，你不要总是想着三两年后换车，咱要抱着开二十年的打算，理性再理性。只说一点。驾控一流就不可能同时拥有奔驰S 的舒适度。这在一个车上是不可能同时出现的体验。跑车，是驾控最好的车型，没有之一。但是，也是底盘最硬，最颠的。

2. 越是某方面优秀的车型，其短板，越明显。理想汽车，为什么卖的那么好？因为，它取了最大公约数。毕竟是做汽车媒体出身的，他知道中国消费者喜欢什么，大空间，大车身，舒适，冰箱，彩电，电车不能有续航焦虑，油车有电车的用车成本，还有一流的辅助驾驶功能，你想要的，都有……但是，你说开着理想去玩沙漠，去穿越无人区，去下赛道？那不合适。

3. 做事，获取 YES的两大核心技能。第一，习惯赞美别人。第二，习惯购买门票。

4. 裙带关系，到了一个不能直视的地步，每20年就是一个大的台阶，反腐一定会越来越深入，现在是抓人，抓事，抓监督，再未来是什么？不断朝上游去改变，裙带效应会越来越小。

5. 王楚钦人气不旺有两大原因。第一，名字起的不好，略拗口，想当明星，名字一定要简单，如林丹，王菲。第二，王楚钦带坏了乒乓球风，他比赛中阴招频出，遮挡、回抛这些争议行为，带坏了整个乒坛风气，这样的侥幸使他缺少底气，实力上，心理上，都是虚的，所以波动非常大。换副拍对他最大的影响，不是拍子手感，而是心理。

6. 大家为什么喜欢外企？人与人之间，松弛。近20年，特权肉眼可见的缩水，是谁的功劳？互联网。我在县城读高中时，那时动不动就有警车开道，喇叭里还会喊话：让开，让开……现如今，除了高考，几乎看不到警车开道了。管中窥豹，关键是，你不知道车里坐的是谁？

7. 对于我闺女而言，她没有太多天赋，不具备生机勃勃的生命力，没有一技之长，若是进体制，混个日子，还是蛮好的，至于勾心斗角，我闺女未必不是一把好手，陪领导睡觉，睡下属，她也会得心应手，这些都是底层选手的天赋。倘若，靠自己的才华，能养活自己。能主动远离各类漩涡。真是太完美了。如猫一般，自由，优雅，不低腰。

8. 你想做儿童舞蹈，想做成人舞蹈，想做减肥塑形，这东西就跟开饭店一样，你不能什么菜都有，突出一两个点就可以了，甚至只做一个点，在这个大环境下，一定要做迫切系列，例如减肥的，例如考学的，除了迫切系列外，人们是不愿意拿钱出来的。主要是觉得学舞蹈没什么用。把所有精力都投入到自身 IP的打造上，做低实体投入的生意。舞蹈，无论是培训，还是IP。其实，不在于推广，不在于运营，什么都不在于，只在于一件事，就跟饭店一样，你炒菜如何。若是我来运营？我只会做一件投入，不断的送她出去学习，让她不断的进化……

9. 想学习，只为自己学。为自己学，不需要迎合别人，什么样的内容都可以拿来读一读，但是分享出去，就要谨慎再谨慎，要考虑，会不会被和谐？会不会被训诫？还有一点，大形势不允许。经济越下行，民间越是暗流涌动，从而互联网监管越严，一是会设网拦截，二是会制裁信息传播者。这种变化，从我这几年做学习群的感觉来看，一天比一天形势严峻。我们不要成为出头鸟。

10. 很多人，上半辈子好好的，到中年陨落了。也许是离婚了，也许是破产了。倘若仔细梳理一下，其实就一个原因，没有建立拦河坝，没有形成一个越来越强大的后盾，随着年龄增长，拦河坝没有越来越大，阅历没有越来越丰富。反而背负上了债务。那，太难了。债务对于一个正常人而言，是癌症。房贷也是吗？是的！

历史发展总结一下就是四句话:
一、上帝欲使其灭亡，必先使其疯狂；
二、时间是筛子，最终会淘去一切沉渣；
三、蜜蜂盗花，结果花开茂盛；
四、暗透了，更能看得见星光。
