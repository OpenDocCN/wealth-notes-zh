
# 被抛弃的欧洲，还能自救吗
## 文/卢克文工作室嘉宾暴躁萌妹

最近，川普撇下欧盟和乌克兰，直接和普京勾兑俄乌停战方案，并在沙特举办美俄会谈，吓坏了欧公子。乌克兰作为战争当事方，欧洲作为受战争影响最大的地区，连上谈判桌的资格都没有？为避免被美俄混合双打，2月17日、19日，欧洲多国领导人举办两次巴黎会晤，主题很忧伤，讨论的是被美国撇下后欧洲怎么办。

那么问题来了，会开得怎样？欧洲能团结起来吗？

首先，此次欧洲峰会不是主动召开，而是被逼无奈、毫无准备，实锤欧洲战略自主就是一个笑话。

放在半年前，美国直接撂下欧洲和俄罗斯谈停战协议，不光是欧洲觉得不可想象，连中国网友都觉得不太可能发生。川普上台后，美俄关系迅速升温。先是和普京打了一个半小时电话，约定在沙特会面。两人通话后，白宫才拨打电话“通知”了泽连斯基，欧公子很破防。

川普和普京通话当天，万斯火速在欧洲开辟“二战线”。在慕尼黑安全会议上，万斯舌战群儒，说欧洲的威胁不是俄罗斯或中国，而是欧洲现在不“民主”，没有言论自由，给欧公子听得一愣一愣的。美俄官方的表态越直白，欧洲越害怕。

从西方媒体报道来看，欧洲国家反应是懵圈、傻眼、手足无措，好像真心付出多年却被渣男抛弃，不能接受眼前发生的事实。但其实，撇下欧洲、瓜分乌克兰，是美俄高层的共识。

这里说的“撇下”，是指果子由美国和俄罗斯来摘，怎么摘是美俄说了算，欧洲和乌克兰靠边站，将来在和会上签字就行，允许在细节上讨价还价。欧洲投入大量资源给乌克兰，付出恶性通胀的巨大代价，就是为了在战后谈判中摘果子。但美俄这么搞，乌克兰领土被拿去五分之一矿产，能源、港口什么也不留。欧洲就是个苦命买单人？

所以，欧洲政客、媒体纷纷“觉醒”，说不能指望美国，我们自己要支楞起来！2月17日巴黎会晤前，马克龙或许还心存一丝侥幸，与川普打了20分钟电话。但马克龙办公室不愿透露细节，很可能谈得一般。之后，马克龙紧急召集欧洲八国首脑到巴黎会晤。

但欧洲这些国家，要让它们立场一致，难度就和让中国人统一粽子咸甜差不多。第一次会议，仅有八个欧洲国家首脑出席，分别是法国、德国、英国、意大利、西班牙、荷兰、波兰、丹麦，外加欧洲理事会主席科斯塔、欧盟委员会主席冯德莱恩和北约秘书长吕特三个欧盟和北约官员。欧盟有27个成员国，说白了，这就是个排除了匈牙利等亲俄政权的小圈子会议。如果他们都达不成什么共识，更别说整个欧盟。

主要谈了三件事：第一，增加国防开支；第二，向基辅提供安全保障；第三，向乌克兰派遣地面维和部队。开了3个小时，除了同意提高预算加强防务，派遣部队这个关键问题，根本没谈拢，也没有发表任何联合声明。

这里提一嘴，派遣欧洲部队之所以关键，是因为川普是个社达，欧洲要上谈判桌，那就展示实力，这也是他和普京讨价还价的筹码。

2024年12月，马克龙在巴黎圣母院重新开放仪式的间歇见了川普和泽连斯基，川普要他俩为乌克兰停战提出“可信的解决方案”。几天后，泽连斯基提出部署欧洲维和部队，要求部署20万。问题在于，派遣地面部队，相当于欧洲对俄罗斯宣战，没有美国保护，欧洲只有挨打的份。英国表示愿意派遣部队，但前提是停战协议达成、美国同意；瑞典表示不排除派遣军队；最反俄的波兰表示不跟；依赖北约成瘾的德国不跟；意大利总理梅洛尼和川普是一伙的，不跟。关键是，一直积极游说各国组建欧洲军的马克龙，也放弃了。马克龙反对马克龙？

第二次会议，立陶宛、爱沙尼亚、拉脱维亚、捷克共和国等国加入，依旧没达成什么重要共识。

现在，摆在欧洲面前的现实，是川普要亲俄弃欧。欧洲想要在谈判桌上挣话语权，必须团结起来。但美俄这俩昔日死对头，都在沙特会谈中达成了四点共识，欧洲仍是一地鸡毛。

欧洲为什么会是这幅叼样？

第一，奢逸成性，血性丧失。苏联解体后，北约本不应再存在，但欧公子实在“由奢入俭难”，不想费钱费力建什么欧洲军，不要大国崛起，只要小民尊严。北约成员国国防开支占GDP比重（俗称给美国保护费）长期在1%以内，它们自己内心享受这种安排，你情我愿，不要老怪美国PUA。

2014年北约威尔士峰会期间，美国要求成员国至少拿出本国GDP的2%用于国防开支，欧公子支支吾吾，到2024年，仅有11个成员国做到。省下的钱，拿去补贴民众福利，这本来没什么。但没想到，德国带头在国防上躺平，从1990年东西德合并开始裁军，不到30年，曾经横扫欧洲的德国军队不见一丝踪影。现在欧洲能稍上台面的，只有英、法，德国非常拉胯。拉胯到什么程度？比如德国法律规定，不允许士兵加班，连士兵都抱怨说每天太闲没事干。2016年有一个新闻，由于加班限制，德国士兵被迫提前离开北约演习。

但德国的拉胯，不应归结于唯心因素，而是有其苦衷。二战后，因美苏争霸的战略考量，东西德被当“工具人”重新武装，国防实力在1980年左右再达顶峰。80年代末，两德统一提上议程，德国军队又成了眼中钉。1990年，美、苏、英、法签订《最终解决德国问题》条约，德国被迫抛售大量武器装备，大规模裁军，从此只想摆烂。现在让德国再武装起来，德国只会PTSD。

第二，各怀鬼胎，离心离德。欧洲各国，尤其是英、法、德、意等国家，都打着自己的小算盘，在组建军队上的想法南辕北辙。

首先看英国。俄乌冲突对英国是稳赚不赔的买卖。英国三年对乌援助134亿美元，英国军火商一年的收入就是425亿美元，这就是投资回报率！英国离俄罗斯最远，就算激怒俄罗斯，但俄也鞭长莫及，拱火成本低。如果战后俄德重修旧好，做大做强，英国将毫无存在感，拱火收益高。英国首相斯塔默发表文章称，英国已准备好在乌克兰的防务和安全方面“发挥主导作用”。这个主导作用，就是英国象征地派点兵，拿更多谈判红利，又不会引火焚身，死道友不死贫道。

再来看法国。法国内心深处不信任美国，一直想当欧洲军的老大，自戴高乐以来，法国就没有放弃过这个梦想。但谁都知道，欧洲军的老大，就是欧洲的老大。德国的国防实力就是渣渣，跟法国携手，大概率是法国主导。退一万步，要是德国认真起来，欧洲又会慌了。欧洲本来就是一盘散沙，德法不点头，建立欧洲军这件事根本不可能推动。

而波兰、立陶宛和波罗的海三国，离美国太远，离俄罗斯太近，伊斯坎德尔导弹系统就在家门口，你敢派一个兵试试？波兰一直站在反俄第一线，但对自己几斤几两掂量得很清楚。马克龙找波兰总理图斯克，谈判还没开始就结束了。图斯克向记者表示，不打算派兵，“将向未来可能希望提供此类担保的国家提供后勤和政治支持。”说白了就是，兄弟你上，我为你加油。立陶宛一开始跳得厉害，开会后，立陶宛议长说谈派兵“为时过早”。

第三，内患重重，有心无力。俄乌冲突打下来，德国连续两年经济衰退，是七国集团中唯一经济萎缩的国家。上届选民最关心的还是气候问题，一场战争打下来，现在连绿党都开始打经济牌了。德国工业高度依赖俄罗斯的天然气，恶性通胀引发连锁反应，大众等传统制造企业纷纷裁员倒闭，德国正在经历严重的去工业化危机。《经济学人》甚至嘲讽德国是“欧洲病夫”。

德国总理朔尔茨表现极差，即将下台，自己国内的事都摆不平，哪有心思谈什么派兵。第一次会谈中，朔尔茨最先离场，对媒体说，欧洲国家预算赤字问题阻碍了对乌克兰的援助。德国最不热心派兵，因为一旦派兵，就要在支付北约高涨的“会费”之外，额外付费，并且这笔开支费用昂贵。从训练到后勤保障，从装备到通信设施，哪个不烧钱？钱从哪儿来？借。但欧洲国家扩大预算赤字，经济一旦出问题，还是德国扛下所有。

从技术上看，欧洲国防体系高度依托于北约，这支部队搞到最后，还是执行美国标准，花钱搞这没用干啥。

总之，现在可能且愿意派兵的，只有英国和法国。但绝大多数国家都表示，美国不在，我们不跟。凯洛格要求欧洲尽快给出维和部队和资产的确切数量，但欧公子纷纷表示，只有在俄罗斯和乌克兰达成和平协议后，欧洲军才会派往乌克兰。欧洲现在面临一个无解的问题：美国参与，就是北约参与，俄罗斯不会同意；美国不参与，小弟们不敢上，怕被俄罗斯狂揍。欧盟国家GDP是俄罗斯9倍，人口是俄罗斯3倍，但完全是一盘散沙。

欧洲现在这个样子，只会让美俄看笑话。连这点流血牺牲的觉悟都没有，还说什么保卫欧洲？不抡家伙，就进菜单，世界还是那个弱肉强食的世界。
