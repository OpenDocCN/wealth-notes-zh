
# 从金字塔原理谈谈：普通人如何获取资源，实现阶层跃迁？

原创：主任 公众号：tz研究 25年4月25日

这是动物界的金字塔，其实可以说和人类社会的金字塔完全是一样的。注意这里说的资本主义社会，请不要对号入座。

人类社会与自然社会的同构性。自然界有食物链，从植物，到小动物，到中型食肉动物，再到塔尖的肉食者。人类社会也是如此，有的人是植物，有的人是小动物，有的人是中型动物，有的人是大型食肉动物，这个事实不仅不会有人教，反而会被刻意掩盖。我常说人首先是个动物，其次才是高级动物。书本上并没有很好地解释这句话，只说了人是“高级动物”,将高级放到了动物的前面，不论是在职场还是在男女关系中，这是后来你进入社会迷茫至少十年的主要原因。

毫无疑问，植物的数量永远是最庞大的，食物链越往上走，数量越少，因为上层所需要的能量是靠下层提供的，只有金字塔型的食物链结构才能稳定，这就是客观规律。食肉动物收割食草动物，食肉动物和食草动物从小到大，学的东西是完全相反的。由于植物的蛋白质含量非常低，所以食草动物得一天到晚低头啃草，一天8小时，365天全年无休。但是食肉动物不一样，它们每周抽出来两三个小时，直接去捕杀食草动物即可，它们积累两三年的能量，被自己在很短的时间内就掠夺过来了。人类社会也是如此，社会阶层的食物链，是上层对下层的掠夺；社会认知的食物链，是高维对低维的猎杀。社会就像一栋老房子：上层精英住在顶层修修补补，中层主力在中间粉刷墙面，底层工人支撑着地基不敢松手。最底层的劳动人民辛辛苦苦赚的钱，被厉害的家伙一下子就全部收了。这还不是最厉害的，最厉害的人，能把你全家6口半辈子的钱全部薅光，而且让你后半辈子还要一直偿还债务。不要误会，我说的不是诈骗局，而是很多合法的收割。所以，认清这个现象之后，到底是做食肉动物还是食草动物，要不要突破原来的阶层，进化成食肉动物，你自己选。

社会资源向权力、资本、知识集中，形成金字塔结构。底层通过体力劳动换取生存资源，中产依赖脑力劳动获取稳定收入，顶层通过资本/权力实现财富增值。社会上层人掌握了规则制定权、资源分配权。上层社会最清楚：规则是他们存在的保障。他们可以容忍底层人抱怨，可以默许中层人钻营，但绝不会允许任何人动摇他们“分蛋糕”的权力。

接下来的内容就更触及核心了，由于本文内容太过禁忌，也是为了保住这篇文章，特此以付费阅读的模式小规模放出。先说说改革开放以来，中国有过七次阶层飞跃的机会。抓住机会的人，实现了阶层的跃迁；错过机会的人，留在底层挣扎。

第一次机会：恢复高考。这次阶层飞跃，靠的是文化知识、个人才华。1977年恢复高考，77到79年三年间，高考录取人数大约有100万，被称为“百万雄师”，绝大多数人的命运都得到了彻底的改变。通过考试改变命运，在中国历史悠久，这是朝堂和民间之间长达千年的默契。自隋唐以来，科举制度就是中国社会底层向上流动的重要通道。“朝为田舍郎，暮登天子堂”,这种通道给了天下读书人以希望，从某种程度上保证了社会稳定。清末，运行千年的科举制度被废除，普通人再想要通过读书改变命运，摆在他们面前最重要的路变成了：考大学。恢复高考通过释放教育资源，打破了阶层和体制的固化，许多普通人，真正通过读书改了变命运。寒门出贵子，不少人在这里抓住机遇，很多农民、工人子弟鱼跃龙门，从底层走向中上层，如今政界、学界、商界、文艺界，都有他们的影子。后来的高考，在整个社会阶层流动中，依然起着非常重要的作用，但是随着大学扩招、社会财富分配规则的改变，社会精英阶层逐渐饱和后，高考对阶层改变的重要性就越来越低了。读大学的人，从最初大多数成为精英阶层，到现在大多成为中产阶层、底层，高等教育普及后学历红利减弱，高考对于阶层跃迁的作用越来越小，到今天已经微乎其微了。

第二次机会：乡镇企业。这次阶层飞跃，靠的是个人胆量、生意头脑。当时，中国还在实行计划经济，社会的主流声音对乡镇企业不看好。这也就是为什么在改革开放的初期，那些从事个体商业活动的人绝大多数来自社会底层，他们是失业者、返城人员、有刑事前科的人，他们是无可奈何，被迫走上了经商和创业的道路。乡镇企业的本质，是政策红利+本地资源的整合，这部分农民企业家，成了改革开放后的第一批民营资本阶层。在现在的人看来，那是个遍地黄金、遍地机会的年代。只要你有胆量、有头脑，就有发财的机会。这也是中国历史上，农民群体唯一的一次大规模“由农入商”,有相当一部分农民摆脱了土地的束缚，实现了阶层的飞跃。这个年代的很多农民企业家，到今天很多都是大佬，依然活跃，成为了市场中重要的资本力量。而随着城镇化接近尾声，这样的机遇也将成为绝响。

第三次机会：双轨制套利。这次阶层飞跃，靠的是制度漏洞、渠道关系。80年代，中国市场首次出现了套利的雏形。当时，79年后，国家对原材料实行价格管制，同时又允许企业生产自销产品，这就形成了"计划”和“市场”的两个价格。这让有生意头脑的人群开始打起自己的算盘，他们利用关系，用计划经济的低价买商品，再用市场价卖出，中间商专门赚差价，赚取暴利。一部分人通过低价获取商品、高价卖出，在这一阶段完成了资本的原始积累，实现阶层跃迁。在市场化改革过程中，计划内商品价格远低于市场价格，产生价格差，出现套利机会，这直接催生了一个叫“倒爷”的特殊群体。当时，所谓的“倒爷”们，利用这个漏洞，找到关系，获取计划经济内的商品供应批条，用计划内的低价买进来，再用计划外的市场高价卖出去，赚取官方价格和市场价格的价差利润，进行“空手套白狼”,使其赚得盆满钵满。这对于有内部渠道、有官方关系的部分人来说，无疑给了他们一种合法的套现通道，他们疯狂地复制这种倒卖手法，来赚取暴利。当然，这是制度改革过程中的漏洞，随着市场经济完善，这个机会也不会再出现。(但在特殊时期依然会存在)利用制度的缺陷和窗口进行套利，在90年代还会发生很多次，甚至直到今天，我们都能看到它们的影子。到了今天，双轨制漏洞虽然没有
