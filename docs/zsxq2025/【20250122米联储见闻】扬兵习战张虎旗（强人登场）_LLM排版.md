
# 扬兵习战张虎旗 米联储见闻20250122

在川普正式宣誓后，全球正式迎来了前所未有的强人政治时代，各国不管基于什么政治体制，只要是未来想要上餐桌的，都给自己安排了一个强人作为领导者。环视全球，对当今世界影响力最大的区域大国们都已处在政治强人领导下，强人纷纷登场，意味着各国都会在全球格局重塑领域主动作为，将会拉开二战之后全球格局最激烈的格局重整序幕，欢迎来到全球掰手腕擂台赛，如果现在哪一个国家还没有拥有一个强有力的领导者，几乎无法以大国待之，未来的全球博弈中，可能连餐桌都上不了，从这个角度看，英法德日在这一轮全球博弈中真是妥妥的配角，强人政治模式也日渐成为大国标配。

联合国秘书长古特雷斯在今年1月15日的联合国大会上说：我们不要心存幻想：这是一个动荡不安、极具不确定性的世界。他强调，人类已经打开了现代的潘多拉魔盒。

实际上，川普可以说是美国第一个真正正视接受了世界正进入百年未有之大变局的美国领导者，一直以来民主党都在试图挽留维护二战以来的旧有秩序，想要对旧的国际秩序修修补补，勉强维持美国的领先地位，而川普在第二任期的回归，一个很明确的信号就是，美国也要亲自下场，参与百年未有之大变局，不再是维持原有秩序和利益，而是要参与重建重构，并且继续保持美国在国际秩序中的领先地位，从格陵兰岛到巴拿马运河，再到加拿大，美国湾，民主党为什么以如此窝囊的姿态退出美国政坛，就在于民主党不能正视这种历史趋势，没有及时的跟上时代潮流，依然希望靠过往的经验和手段维持美国优势，所以遭到了难以避免的失败和挫折。

川普虽然不一定能带美国重回黄金时代，但显然川普在对未来发展的趋势上选择了正确的方向，时至今日，唯有变才能实现不变，唯有跟上百年未有之大变局的趋势，才能保持美国领先的地位不变，从这个角度看，美国确实通过川普再次当选按下了历史选择的正确键。一旦大国都选择走上餐桌，那么过去的游戏规则就彻底沦为大国博弈的工具，过往的一切历史文件，声明，协议，合约，都是可以随时更改废除的，美国退出世卫，退出巴黎协议，就是要告诉全世界，美国选择了要重塑游戏规则而不是维护游戏规则。所以，古特雷斯要大家不要自己骗自己，这就是大争之世，这就是强人掰手腕大赛，这就是百年未有之大变局。

正是因为世界格局的巨变才会催生世界政治强人的出现，就如同决定第二次世界大战及其掌握划分世界新格局的小罗斯福、斯大林、邱吉尔。当今世界能够决定全球未来格局的无疑是美国、中国、俄罗斯三个一线大国，以及印度、土耳其，沙特，巴西等区域性大国，这些国家有些是工业文明的最高发展阶段，有些占据了广阔的疆域，有些以文明生命力著称，有些地处核心战略要地，这些国家也都先后确定了政治强人在国家领导领域的地位，从普京到川普，从莫迪到埃尔多安，包括以色列恶贯满盈的内塔，都是当前国际社会公认的政治强人。世界巨变与政治强人的关系可谓是相辅相成，辩证统一。一方面，世界巨变让政治稳定性出现裂缝，政治传统被撬开缺口，强人才会脱颖而出，而政治强人在政治舞台上的非凡领导力和决断力，强大忍耐力和远见卓识又会反过来影响世界格局进一步变化。所以接下来的十年，将是全球政治外交非常精彩的十年，是真正的高手对决，其实过去十年已经是花样百出，精彩纷呈，作为一个研究时政新闻的人来说，我写了将近十年的时评，我只能说这事能干十年，真的是因为我们在一个百年未有之大变局，每天都有你想不到的事情在发生，可以天天写写十年。凑齐川普这枚全球第一帝国的政治强人后，确实有点角色都到位了，大戏要开场的感觉。

强人政作为一种政治模式，决策过程高度集中在一个强有力的领导者手中，国家的运行体现着领导者强大的个人意志和权威，外部局势的紧迫性导致内部国民对民主问题不再警惕和敏感，甚至主动让渡一部分权利谋求整体安全和稳定，所以复杂多变的国际博弈反过来也促使政治强人有生长和壮大的土壤。大变局中，主要国家必然在贸易、能源、科技和军事等方面产生摩擦，沟通与冲突，合作与竞争，威胁与对抗，会成为今后很长一段时间的主旋律，打一打，停一停，聊一聊，再接着打一打，双边或多边关系会处于长期不稳定的状态。从这几年的国际形势来看，中美俄依然是全球当之无愧的一线梯队，这三者强弱是不断变化转换的，虽然中美俄经济体量有明显差异，但它们在不同领域各有权重和优势，三国的三边关系决定着国际格局的最终演化，这一波全球大变局，实际上就是美国对俄罗斯和中国进行政治控制、经济遏制和军事打压以维持全球霸权与唯一超级大国地位的能力不足所产生的变化。这三个国家的博弈实质上是企图称霸世界和反对这一企图之争所构成的矛盾，不可调和。

对绝大多数国际观察家来讲，美国同时挤压和制衡两个头部大国是匪夷所思的，甚至有点莫名其妙，连川普这个前地产商都明白要合纵连横，要联俄抗中。但民主党和共和党的政治精英们却一意孤行，不断拱火、诱导并迫使中俄在被动中进行激烈回应。北约东扩到俄罗斯的腹地，不断台湾问题，俄乌冲突和台海问题都是美国亲手打造的地缘危机，最终刺激两个核武大国紧密合作，如同俄乌战争的发生，重新激发俄罗斯军工产业的全面复苏，倒逼原本只想赚钱的中国变成军工大佬。在这种强势对抗和外交的模式下，也让一众原本不用加入对抗的二线国家看到了落后就会挨打的危机感，伊朗朝鲜拼命搞核，印度巴西拼命搞经济，想要复制中国的强国之路，现在国际关系组织交错复杂，各国都在给自己找各种备胎，各种组织框架都在重合叠加，有些国家反复横跳，到处拿好处，都希望在新秩序重塑过程中实现国家利益最大化。印度想要取代中国成为亚洲最强，土耳其利用多重身份和咽喉要塞，在各方势力之间辗转腾挪搞新奥斯曼主义，沙特一直在寻找机会成为中东老大。巴西同样想做南美第一哥，强人心里都有梦，不是扩张就是干，谁对谁也不服气，最后还是比拳头。

不过有意思的是，这一轮，传统强国也在退出中心舞台，欧洲首当其冲，已经注定会缺席这一轮全球格局重构，国家众多且松散的欧洲在强人政治和全球变局中只会快速衰弱，默克尔曾经想要力挽狂澜守住欧洲的发展优势，但是分散的欧洲，政治权力碎片化的欧洲从制度到思想都已经无法选出强人，也无法团结欧洲，欧盟领导人也不具备维持欧洲利益的坚定信念与强硬手段，都是些吃拿卡要的职业政客。欧洲在经济、军事、外交上严重受制于人，加上俄乌一战大幅削弱了欧洲元气，川普上台之后估计连看都懒得看欧洲一眼，这次欧洲仅仅请了意大利总理出席就职典礼，对于川普要买下格陵兰岛的建议，丹麦稍微犹豫了一下，估计最后肯定也是妥协。未来的欧洲估计连做盟友的资格都没有了。缺少政治脊骨的欧洲如今只剩下恐惧与抱怨，恐惧俄罗斯和中国，抱怨并恐惧美国不稳定的情绪。欧洲的凌乱与实力都无法成为美国真正的盟友，只能在不同阶段被妥协，被牺牲。

卢比奥在就职后，做的第一件事是举行四方对话，美国国务卿卢比奥上任第一天就会见四方安全对话国家，印度、日本和澳大利亚才是美国现在比较看重的小伙伴，这四个国家人口近20亿，占全球GDP的三分之一以上，这个组织成立已有15年，川普看重这三个国家，说明美国真正的对手还是中国。四方安全对话2007年成立后不久便陷入沉寂，并持续了十年之久。直到川普在其第一任期内与日本、澳大利亚和印度领导人达成协议，重启对话。这次川普归来，被中国制裁的卢比奥第一时间与日印奥外长碰头，显然表达了川普外交领域的侧重点。卢比奥称中国是“美国有史以来面临的最强大、最危险的对手”。在与卢比奥会晤之前，澳大利亚外交部长黄英贤称与美国的双边关系是“我们最重要的战略关系”,并表示很高兴四方安全对话国家会议在就职典礼后不久举行。印度最近有三艘新的军舰开始服役，莫迪称显示印度在海上安全的领导力显著增强。至于日本，刚刚开始回暖的中日关系，石破茂在日本国内就遭了亲美派的强力狙击，这次卢比奥组织日澳印会面，再次给日本亲美派释放信号，日本对美国还是有用的，狗项圈还不能立刻摘。

新世界蠢蠢欲动，技术革命叠加政治变革，新的历史周期缓缓走来，世界是分是合都还未有定局，一超多强模式还是各立门户模式，是国家之争，制度之争，同时也是文明之争，历史河流浩浩汤汤，能容纳百川者才会天下归心，历史上成就的伟大帝国无一例外，包括美国在内，也是博取众家，谁是最终赢家不好说，但有一点毋庸置疑，历史没有终结时，世界向前推进谁也拦不住。
