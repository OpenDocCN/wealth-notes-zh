
# 卢克文星球《石破茂“最后一舞”——战后80周年见解》

石破茂，即将离任的日本首相，在首相官邸举办了一场以"个人见解"为名的发布会，内容围绕战后80周年的反思。这场发布会时机微妙，在8月15号日本战败纪念日和9月24号联合国大会演讲中，石破茂均未提及相关话题。临近下台，他选择以个人名义发表见解，这出戏颇具"欲说还休"的意味。

这份个人见解如同一把无形的利刃，直指日本极右翼的痛点。发布会前几小时，公明党突然宣布退出执政联盟，以"政治资金监管"为名，实则要求自民党基层支部断绝政治捐款。这一条件石破茂无法接受，公明党此举明显针对高市早苗。

值得注意的是，石破茂担任总裁期间，公明党对此未提一字，暴露出其看人下菜碟的本质。公明党退出后，立宪民主党和国民民主党迅速转向，维新会也撤回对高市早苗的支持，日本在野党在公明党和石破茂的双重夹击下重新集结。自民党在两院席位未过半，下一任首相宝座悬而未决。

石破茂的演讲暗藏五大灵魂拷问，其冲击力源于新颖的视角——从日本为何未能阻止战争爆发切入。他引用安倍晋三原话，首次拷问：国内政治体制为何未能有效刹车？接着揭露历史真相：开战前日本智库预测战败是必然结果，第二次拷问：明知打不赢为何还要打？第三次拷问直指核心：1935年"天皇机关说"事件中，立宪政友会攻击政府，最终导致军部卷入，暗示昭和天皇是战争责任人。

石破茂从宪法、政府、议会、媒体、情报机构五个层面展开反思，最犀利的问题在于"天皇机关说"事件，直指天皇主权说与战争责任。他强调制度虽重要，但运用更为关键，第四次拷问：政治是否应迎合舆论，损害国家利益？最后指出日本已解决统帅权问题，但需警惕狭隘民族主义和过度商业主义。

演讲结尾，石破茂指出战争记忆逐年减少，资料删减问题严重，并引用1940年斋藤隆夫反军演说被删减三分之二的事例。他暗示历史认知不完整的不只有日本，列强在帝国争霸时代的所作所为有何区别？做正确的事总是很难。

石破茂的演讲在雅虎日本评论区获得较多理性讨论，野党评价不一。立宪民主党肯定其总结历史，维新会质疑发表形式，国民民主党认为洞察体制问题值得肯定，而日本共产党则批评其缺乏对侵略战争和殖民统治的反省。石破茂在NHK采访中表态："必须正确认识日本曾经在中国和亚洲的所作所为"，但未在演讲稿中提及。

尽管存在局限性，石破茂的演讲仍作出五大贡献：阻止极右翼用安倍谈话盖棺定论，引出美日同盟打不赢中国的可能性，暗示天皇战争责任，讽刺极右翼迎合民粹，指出二战史料删减问题。中国媒体在指出局限性的同时给予正面评价。

石破茂的最后一舞看似孤独告别，实则投下深水炸弹。他以回归制度与理性的演说和一系列平衡外交的举措，撼动了日本"随美起舞"的外交惯性。尽管个人政治路线受挫，却为日本未来埋下独立与理性的种子，为务实派吹响集结号角。

执政末期，石破茂凭借对美关税谈判的良好表现赢得"石破别辞职"呼声。其深刻、精巧的演讲赢得国内外好评，为政治资本垫高。这场演说或许不是石破茂的最后一舞，当一个政治家勇敢擦拭历史镜面，挺直脊梁面对霸权时，他所照亮的不仅是个人前路，更是一个国家对独立自主与历史真相的艰难眺望。
