
# 三种趋势交易 (1)

## 一只鱼提问：洪总，您好。向您请教交易哲学方面的问题。

最近，我在看江恩相关的书籍，江恩说：“投机是一门有利可图的职业”。在交易市场，富有的人，他们的资产可以交给投资机构，获得超额回报。我是普通人，我的交易对手盘，可能是投资机构、专业交易员、专业分析师。这使得普通人，成为高胜率投机者，并不容易。

大多数时候，我买不到便宜的筹码，更多是，只能买入有价格势能、反弹的标的。但这种交易策略，带来的收益很小，改变不了我的生活现状。因为，我的本金只有10万元人民币。

我记得有句话：“人生财富源于择时与杠杆”。所以我想问，本金少的普通人，怎么获得更大收益?是否应该加点杠杆?

或者说，洪总，您在交易中，在胜率较高的趋势中，会加杠杆吗?多少倍杠杆是上限?我没记错的话，您在很多年前，回答过类似的问题，说这是诺贝尔经济学奖级别的問題。

## 洪灏答复251206

语音免费探讨一下这个问题，我经常说有三种趋势，一种就是趋势跟随的交易，一种是趋势逆转的交易。趋势跟随的交易并不是每天都有的，但是趋势一旦形成的时候，他就会走到底。不管发生什么事情，所以并不是每一天都有趋势跟随的机会，趋势反转的机会就更少了。

因为呢，随着你的时间，交易时间窗口的延长，那么趋势你也不断的扩大，但是趋势逆转发生的时间，那它的频率呢，就越来越小。

第三种交易呢，就是息差交易，息差交易呢，依靠这个规模效应，对吧?因为你的息差可能只是一个点，两个点，但是为了捕抓这一个点，两个点的收益，你需要一个非常大的规模以及非常庞大的交易系统，那么这个呢，一般都是机构做的事情，普通人是很难做的。

因此在你的问题里头，你说普通人对手盘是这个机构交易，其实也不是，因为呢，机构是很难像我们一样赌方向的，对吧?尤其是大型的机构，很多机构呢，最后呢。沦为一个息差交易的主体，同时机构并不比普通人胜率更高，因为呢，普通人操作的空間更大，那么机构呢，我举个简单的例子吧，比如说他希望每个月他都能够向这个投资者回报正收益，那么这个时候呢。
