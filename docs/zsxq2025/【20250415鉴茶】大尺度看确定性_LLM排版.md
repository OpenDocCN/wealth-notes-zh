
# 大尺度看确定性

大人，时代变了，以前的大盘涨，是喝酒吃药开车，拉主线，现在不一样了。

一
信创和顺周期，拉底部。

以前的消费代表是白酒，现在是乳业，商超，喝酒的改喝奶了。商超肩负拉动内需的作用，股东厉害和带免税的高看一眼。乳业有表现的是区域性品种，弹性比较大的，比如贝因美、熊猫、燕塘。新三板有个骑士。奶酪会成新兴亮点，如妙可蓝多。不要觉得消费萎靡，乳业的利润是在回升的。结合日本的经验，烈性酒从餐饮超市中大面积消失，取而代之的是功能性饮料，主打提神醒脑，够社畜喝很多年。日本是力保健，大A典型代表是东鹏饮料。

二
以前的自主创新，代表是信创，现在是MCU芯片，成熟制程，自主可控。这个市场已经从百亿美元，上涨到千亿美元级。德州仪器迟早被平替，无论是电池的还是汽车的。车规上纳芯微的头部地位明确，其余提过的那几家都不错。
