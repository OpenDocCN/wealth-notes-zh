### 战略的利弊

任何关于战略的讨论最终都会以“战略上一把双刃剑”的结论而结束

| 利弊 | 描述                                                         |
|------|--------------------------------------------------------------|
| 利   | 战略方向也会向眼罩那样遮住潜在的                             |
|      | 战略设定方向 为组织绘制航线图，以便组织齐心协力的行驶证市场中 |
|      | 战略聚合众人之力 提高了行动的协调性，聚合了大家               |
|      | 过于谨慎强调一致性，会导致“群体思维”，使组织丧失对周围的观察力 |
|      | 战略决定组织 战略向人们提供了理解自身组织并                 |
|      | 将其同其他组织区分开来的捷径                                 |
|      | 过分清晰的界定意味着过于简单化，有时导致模式化，从而丧失整个组织 |
|      | 的多样性                                                     |
