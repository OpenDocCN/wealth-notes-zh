
# 稳外资的政策优先级并不高

自改革开放尤其加入WTO以来，大多数时候，中国并不关注国际收支以及外商投资问题，因为中国市场太宝贵了。但2018之后，形势变了，于是稳外资成了是一项长期政策。

但不同时期的稳外资，重点方向不同，截止目前大概有三轮。第一轮是2018年之后，为了应对贸易战压力，需要在数量上稳定，所以重点瞄准金融外资，要把数据提起来，典型代表是“北向资金”,2017年开通，之后成为被关注的焦点；第二轮是2020年以后,为了应对疫情冲击，重点是稳定外贸，同时稳定外资；然后是2023年以来的最新一轮，虽然也在强调稳外资，但对外资的实际期待已经相对下降，更针对重大外资项目。

梳理上述三波特征，有利于准确理解当下和未来的稳外资政策。

官方层面，稳外资力度最大的其实是2023年上半年，2023年两会后，外资巨头纷纷访华，高层也频频会见，分管副总还在广东出席了“投资中国年”招商引资活动启动仪式，重视度超过过往。但外商投资并没有稳定，从2023年以来，外资增量呈下降趋势，需要注意的是增量姜缩，也就是增长放缓但总量仍在上涨，而且存量庞大。

在这种情况下，外资成为国内外关注的一个焦点,与对中国未来的信心相绑定。于是从2023年下半年开始，相关部门不断出面强调中国对外资的吸引力仍然很强，但注意说法发生了变化。

从过去强调增量，变成强调存量庞大、国际占比稳定、结构优化。

比如2024年1月介绍2023年前三季度国际收支情况，在边际增长数据恶化的情况下，强调存量数据的优势，认为“对外负债中，超50%为来华直接投资，规模3.3万亿美元，占比较2023年6月末有所上升”,以此论证“我国对境外长期资本保持吸引力”;

2024年2月召开外资工作会议暨重点外资项目工作专班会议，强调2023年全年保持引资规模及占全球跨境直接投资份额总体稳定，以国际占比稳定论证并未恶化；到了2024年6月，商务口介绍外资形势时则强调结构优化，指出2024年1-5月，我国制造业实际使用外资1171亿元，其中高技术制造业引资504.1亿元，占比较去年同期分别提高2.8%和2.7%,以引资结构优化论证未来信心。

当然这种说法的变化，在不同人眼里有不同理解。悲观的认为这是避重就虚，回避增量放缓、姜缩的现状；乐观的则认为这是中国产业升级、国产替代带来的必然结果，随着中国攻克一个又一个技术、产业难关，外资在高技术、先进制造领域的优势下降，导致对外资的挤出。

既然是解读政策，不同声音都需要理解，但更要搞清楚决策层怎么看。

悲观情绪达到最高是2024年初，当时中国外商投资出现了几十年以来的大幅下滑，外媒报道称国际资本对中国经济前景的信心出现大问题；在金融资本方面，北向资金也开始长期流出，这也说明金融资本在抛售中国资产，加剧了金融市场的悲观情绪。

但需要说明的是，中国国际收支平衡主要依赖货物贸易顺差，疫情三年实际是中国出口的黄金三年，再加上人员流动不便大幅降低了服务贸易逆差，所以2020-2022年，中国国际收支基本面大好，2023年全面放开后，由于出国恢复，服务逆差扩大，导致贸易顺差回落，但仍然高于2019年水平。

至于金融资本，早已不是国家关注的焦点，国家在意的只是外商直接投资，也就是中长期产业资本。而中产期外商投资确实出现了明显恶化，从2022年以来，外商直接投资增速转负，2023年增长同比下降8%,2024年同比下降更是高达27.1%,再次强调，这里说的的是增量，也就是增长的比过去的少了，但仍然在增加。

而决策者对外资增速萎缩的看法，显然是后者，也就是认为外资增速放缓，是中国产业升级带来的必然结果。

针对2024年初的悲观情绪，官方认为外商投资下滑，是因为外商投资工业企业利润累计下滑，原因是外商盈利能力下降，导致计提的“未分配利润、已分配未汇出利润”减少，导致看起来外资增速下降，但这并不代表中长期外资撤资或者利润汇出。

而另外的两个原因则是国际大环境， 一是受连续危机影响，全球跨国投资低迷；二是中美之间利差逆转，由于美国十年期国债收益率远高于中国，所以部分资金退场。

不过小镇认为拿中美利差借势中长期外资增长放缓，这是站不住的，因为中长期资本属于产业投资，并不会过度考虑几年内的利率波动，中美利差影响的是短期的金融资本，不能混为一谈。

小镇之所以认为，决策层实际把外商投资增速放缓视为中国产业升级的成果，是战略胜利带来的局部受挫，而非纯粹的负面变化，也跟后来的政策倾向有关。

实际从2024年之后，国家稳外资的实际力度是下降的，体现在稳外资政策明显更加低调，“投资中国”活动仍然举办，但期待下降，不再是具体负责外资的分管副总出席，而且高层与外资巨头会见的层级和频率也有所下调。

而具体的一些政策，也并没有采取倾斜外资的措施。比如去年下半年很受关注的外商独资医院扩大试点，核心卡点一个都没有放松，仍然视外资医院为公立医院的补充，必须采取同公立医院一致的标准，也没有开放一些药物、器械的许可，这对外资的吸引力很有限。

然后就是2025年2月17日，公布了《2025年稳外资行动方案》,这个方案确实比较实际，直面了外企面临的很多问题，比如药品领域提出“优化药品带量采购”,而在政府采购领域，也提出尽快出台政府采购本国产品标准。对“本国产品标准”,小镇在公众号《争取实现第二次入世，借力倒逼国内改革》和星球24年12月7日《政府采购政策发生重大调整，明暗两条线》已经谈了。

另一方面，由于制造业领域已经实现了外资投资负面清单清零，已经没有继续开放的空间了，所以下一步促进外资转向扩大开放服务业，比如方案提出“研究制定有序扩大教育、文化领域自主开放实施方案”。

但需要注意，教育、文化等服务业领域，不仅仅 是经济问题，还关系到社会、意识形态等深层次，不 仅仅是钱的事，开放难度很大。这些领域别说对外资开放了，对民企开放也比较谨慎，目前正在对前期过度市场化进行纠偏，前几年刚对校外教培进行了打击，未来更倾向于教育、医疗等基本公共服务回归公共属 性。

所以总得来说，国家确实强调重视“稳外资”,但要注意，重点已经变了。

随着中美博弈加剧导致高技术领域加速切割，掌握技术优势的外商受美国压力影响，对在华投资有顾虑；而对中国来说， 一般的外资没什么价值，还不如腾出宝贵的市场空间支持中国本土企业的发展，加速国产替代。比如今年，随着中国Al领域的突破，6AI相关产业的国产替代已经提上日程。

随着中国快速发展，还值得中国关注、引进的外资，越来越少，外商在华投资也越来越难与中国企业展开竞争。如果说对中长期产业外资兴趣下降那么对金融外资就是警惕了，未来更关注汇率和金融稳定，所以央行一季度暂缓降息降准，近期已经引起如彭博等外媒的强烈不满，批评央行“货币政策反复无常，降息时机不明”。但小镇早在一个多月前就说了，央行倾向于一季度暂不降息。

总之，从政策持续性和表达坚持对外开放的态度上，“稳外资”会成为长期坚持的政策，比如上周就召开了扩大高水平对外开放，做好2025年稳外资工作有关情况的吹风会。

但实际上，目前采取的稳外资措施，都是技术层面的，仅仅是相关部委在自身职权范围内推出了一些常规政策和原则性表态。但在上述趋势性变化面前，这些常规、技术性措施显然无法实现“稳外资”,要想彻底解决，需要采取更高层级的战略措施，要推动深层次的改革。比如对接高标准国际经贸规则，改革政府采购制度，还要涉及到教育、文化等非常敏感的领域改革，这需要上升到GWY以上的最高层级才行。

而目前还停留在部委层面，也就从事实上证明了。决策层对外资增量放缓，持乐观态度，稳外资的实际政策优先级并不高。
