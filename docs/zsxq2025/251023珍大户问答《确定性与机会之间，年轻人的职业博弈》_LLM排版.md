
# 确定性与机会之间，年轻人的职业博弈

## 251023 珍大户

### 提问：
进入圈子也一年多了。去年做的收益率最高的决定就是给自己和对象各买了两年的圈子门票，让她来耳濡目染一番。她从理财小白开始跟着买了破净股、储蓄分红型股，不到一年收益率超过20%。而我自己主要还是投资科技行业与汽车行业，我是从高三开始买股票投资的。上个月和同学小聚的时候了解到大家都在外面看新的机会，我也挂了一下，很快有一个量化私募联系了我，经过笔试面试等流程，不久拿到了雇佣offer。

基本情况：26岁，硕士，广东深圳。现职为某半导体设备大厂软件研发。

### 选项 A 留任（半导体设备）
- 薪酬：全年现金包约30万出头+年终8—12万+5万左右的加班费。
- 社保公积金：公积金双边合计约10%；周末加班按基本工资给双倍。

### 选项 B 跳槽（量化私募自营）
- 公司形态：深圳某量化私募，自营盘，全自动化交易，规模10亿+；人数不多，办公环境很好。
- 薪酬：全年现金包约40万出头+年终2+个月（区间化）。除此之外没有任何现金福利。社保公积金：按深圳最低标准缴纳。
- 核心算账：综合个税、社保公积金，B每月“确定性到手”较A提升约27%。
- 成长：转向低延迟交易系统开发，从0积累实战。
- 中长期：满三年可跟投（表现好可提前）。利好：通勤改善（电瓶车～30分钟；打车～20分钟），每天多～1.5小时可支配时间。

本来职业发展还蛮好的，现在多了一个选择。核心纠结（请珍大户和圈友指路）

### 回答：
<量化私募有什么作用> 不要跳，继续留在半导体行业。
从钱的角度，两边钱其实差不多的，加上年终公积金之后，现在的总收入跟跳槽后其实是差不多的。
你比较难以接受的点是上下班两个小时，导致的幸福体验急剧恶化。这个很好解决，买一个自动驾驶的车，上下班完全交给智驾就解决了。然后在车上的时间就是你的自由支配时间了。
但之所以让你留下不要跳，并不是从钱的角度或者时间的角度。
而是你现在的行业蒸蒸日上，未来还有很多年的好光景，你以后行业元老，加上公司还有上市配售的预期，这些都是极大的优势。在里面对科技前沿的信息与眼界，也能辅助你做投资决策。
相反，量化私募这里，国家对量化私募随时可以掐停。通道一停，或者账号一关，说你违规，就结束了。game over。到时候你只能离职回家自己炒股去。
关于不关，就取决于它的存在对社会是正面意义大的还是负面意义大。如果它阻碍国家发展规划，限制它就很必要了。
你可以想象一下量化私募对证券市场的作用是什么？它只能增加市场成交量，助推加速趋势，相反，不能充当阻尼器的作用。
也就是说，如果市场很火热的时候，量化私募的存在会让市场更加火热，市场下跌的时候量化私募会加速抛售，让市场跌得更凶，无论哪种情况都不是国家想看到的唯一不影响大势的时候，就是市场较为萎靡，武器低迷，成交量少，不活跃的时候。这个时候需要成交量来带动市场。因为此时还没有决定性的方向变化，所以量化私募也不可能助长暴涨暴跌，就没有太多负面性体现，纯粹是正面性。
但这种情况以后还会多吗？从历史的进程角度看，中国证券市场越活跃，量化私募越没正面作用，越有可能出现负面作用。相反，从历史的进程看，所在的半导体行业却是未来10～20年将要发挥巨大作用的行业。
那句话怎么说的——人的命运啊，个人的努力固然重要，但更多的还是要考虑历史的进程啊。
