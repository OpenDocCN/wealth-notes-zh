
# 卢克文青云读书会趣说挡史：资本主义（下）伊丽莎白与资本主义毒牙

大家好，我是杨文哲，欢迎收听卢克文工作室的党史栏目。上回我们说到，马可波罗来中国浪了一圈，回欧洲一通比划，勾起了西方一群破落国家对东方白富美的欲望。随后的文艺复兴催化了大航海时代的开启，开创者西班牙由于发现了新大陆一跃成为欧洲霸主，而英国却因为组建海盗打劫西班牙而和西班牙结了梁子，未曾想西班牙组建用来入侵英国的无敌舰队却只是此牙咧嘴地卖了个萌便灰飞烟灭了。这之后，世界便进入了17世纪。17世纪初成立的两家东印度公司对资本主义的发展起到了不可忽略的作用，也为两个世纪后中国的悲惨遭遇埋下了伏笔。英国与荷兰早就想避开西班牙的锋芒以及欧洲大陆内卷的环境，分别组建了自己的贸易公司，开拓海外贸易，就不跟西班牙玩了。于是16世纪的最后一天，1600年的12月31日，由英国女王伊丽莎白一世在去世前两年亲自授予了皇家特许状。不列颠东印度公司成立，1601年莎士比亚创作出了《哈姆雷特》，又过了一年多，1602年3月20日，荷兰东印度公司也成立了。17世纪到18世纪，西方许多国家也纷纷效仿英荷两国成立东印度公司，但影响力都很有限，就英荷两国成立的东印度公司影响最大。其中荷兰的东印度公司，还是世界上第一个发行股票的公司，并且它的股票连续近200年保持18%的收益率，巅峰时期市值将近8万亿美元，现在的苹果、微软、谷歌、BAT这些巨头，在它面前都是浮云。并且它能自己组建军队（雇佣兵），发行货币，与主权国家签订正式和约，几乎就是一个伪装成公司的国家。东印度公司成立后不久，1604年，英国与荷兰在欧洲本土的战争中败给西班牙，脱身后便分别经营各自的海外业务去了，仍处于欧洲霸主地位的西班牙却依然被欧洲的各种乱七八糟的纷争黏住脱不了身。东印度公司给荷兰带来了惊人的利润，因为当时的亚洲物产丰富且没有竞争对手，荷兰因此迅速崛起。后人称荷兰为海上马车夫，就是说荷兰是在海上搞运输的中间商。并且由于荷兰一直以来是个商业共和国，什么事都是资本家在一起讨论，商量好了开干，政府没什么存在感。这种模式可以充分放大商人的积极性，但缺点也很明显，就是一个松散的商业联盟在关键时刻无法发挥举国之力，难以制定长期的政策。所以当资本家得势时，荷兰这个国家就强大，当资本家遇到问题了，便会毫不犹豫地卖国。在资本家眼里，国家利益和其他利益一样，都是可以用来的交易的筹码。同时由于荷兰人擅长造船，加上这会又不差钱，于是17世纪时荷兰的海军非常强大，妥妥的海上霸主。也正是因为东印度公司的存在，荷兰开始把手伸向中国，东西方的交流与碰撞便因此频繁了起来。还是1604年，胆肥的荷兰偷袭澎湖，被明朝军队骂走了。1606年9月18日，一个叫张献忠的人出生，四天后的9月22日，另一个叫李自成的人出生。1607年，英国来到了大西洋彼岸的北美洲，建立了第一个殖民地，就是今天美国的弗吉尼亚州。由于当时北美洲的野生动物很多，具有巨大的毛皮贸易前景，荷兰人冲着这个生意前景于四年后的1611年，登陆了美洲。而荷兰人当时在美洲第一个据点就是哈德逊河入海口，当时荷兰人命名为新阿姆斯特丹。而此前的1609年，德国的开普勒创立了行星运动的第一、第二定律。意大利的伽利略则制成第一架天文望远镜。1610年，中国思想启蒙之父黄宗羲在浙江绍兴出生。1613年，俄罗斯历史上最强大的王朝罗曼诺夫王朝建立，俄罗斯由东欧一个闭塞的小国就此逐步拓展为欧洲乃至世界范围的强国之一。但他们当时的生产关系十分落后，以农奴制为主。同年，清代学术的开山之祖顾炎武在江苏昆山出生。又过了5年，1618年，始终没能从欧洲的乱七八糟的纷争中脱身的西班牙又卷入了三十年战争。三十年战争是第一次全欧洲大战，英、法、德、挪威、丹麦、瑞典等国都参战了。这场战争是欧洲各国争夺利益、树立霸权以及宗教纠纷激化的产物。所以对西方资本主义国家来说，世界大战早在四百多年前就已经开始了。1619年，湖湘文化的精神源头，中国朴素唯物主义思想的集大成者、启蒙主义思想的先导者，与黑格尔并称东西方哲学双子星的王夫之在湖南衡阳出生。其经世致用、实事求是的思想深刻影响了后来的另一个湖南人毛泽东。1621年，荷兰又成立了一个西印度公司，目的是与西班牙、葡萄牙、英法等国争夺海外殖民地。于是1622年，荷兰再次侵占澎湖，这回没人骂他了。1624年，荷兰打败西班牙殖民者占领台湾。此后，荷兰靠着东西两个印度公司在全世界范围内进行极其野蛮残酷的殖民和掠夺。动辄屠杀当地居民，搞种族灭绝，把活着的人变为奴隶做奴隶贸易。这种做法使得公司利润往往能高达百分之几百。马克思就说过，“荷兰——它是17世纪标准的资本主义国家——经营殖民地的历史展示出一幅背信弃义、贿赂、残杀和卑鄙行为的绝妙图画。”1629年，明朝崇祯二年，23岁的李自成在甘肃起义。一年后的1630年，24岁的张献忠在陕北起义。六年后的1636年，关外的满人皇太极改国号为清，清朝建立。1637年，法国的笛卡尔在创立了坐标系后，成功创立了解析几何学，同时还确立了西方科学研究的方法论，并提出了“我思故我在”的著名思想。1640年，英国爆发了资产阶级革命，英国从王国变成了共和国。资本主义革命开始，文艺复兴也就落幕了。同年的东方，蒲松龄出生，张献忠攻入四川。四年后的春节当天，李自成在长安建立大顺政权，是年春末，号称闯王的李自成攻破北京，崇祯皇帝自缢殉国，大明王朝嘎在了1644年。南明政权成立，占据半壁江山，定都南京。李自成的军队在北京城里杀人放火抢女人，其中就包括镇守山海关的吴三桂的爱妾陈圆圆。于是吴三桂冲冠一怒为红颜，投降清军，清军入关，清朝开始统治中国。而张献忠也于这一年冬天在四川成都称帝，国号大西。但两年后便因手下大将叛降引来清军中箭而亡。此前，李自成也在湖北被当地地主所杀，清军统一大陆指日可待。1648年，欧洲的三十年战争结束，荷兰获得独立，西班牙失利，随后本土爆发内战，西班牙彻底失去欧洲霸主地位，不再是一流强国。而荷兰靠着东西印度公司已达到了商业繁荣的顶点，继西班牙之后成为世界上最大的殖民国家。英国这时候已经开拓了北美殖民地，又有东印度公司与荷兰在东方竞争，西班牙没了之后，改朝换代的英国十分不爽荷兰成为大赢家，便把枪口对准了荷兰。1652年，第一次英荷战争。两年后荷兰失利，签订条约，退出巴西，巴西让给了葡萄牙人。四年后的1656年，荷兰使团到达北京，想与刚刚立足的清政府通商，却被顺治帝认为是来朝贡的，收下礼物后便把人打发走了。同年清军攻破四川，南明政权的永历皇帝逃往云南昆明。两年后，清军攻陷云贵，永历帝遂逃往缅甸。又过了两年，1660年，英国发生了斯图亚特王朝复辟，英国从共和制又回到了君主制。第二年，吴三桂攻入缅甸，缅甸王遂将永历帝献给了吴三桂。这时候的南明政权基本只剩下郑成功控制的泉州厦门、金门等地。为了能有要大的立足点，郑成功率军横渡海峡登陆台湾，一路所向披靡。这一年，顺治帝崩，康熙即位，这时候掌握实权的鳌拜便开始了迁界禁海，就是皇家背书的暴力拆迁，想穷死台湾。此举直接导致福建沿海一带百姓家破人亡、妻离子散。第二年，1662年，荷兰军队向郑成功投降，郑成功全面收复台湾。荷兰横行全球半个多世纪，结果被郑成功给干趴下了，而此时郑成功已经没有多少人和地盘了，还能干翻当时的世界霸主，可见其战斗力。但郑成功夺回台湾后没多久便得知父亲被清军所杀，祖坟也给清军刨了，儿子还和奶妈私通，他效忠的南明政权的永历皇帝也给吴三桂弄死了，自己手下的兵在台湾还水土不服（郑成功是河南人），在接连的打击之下郑成功很快便撒手人寰。次年，1663年，清廷勾结荷兰联手进攻厦门、金门，两岛陷落。经过一番内斗掌权的郑成功儿子郑经便携家人撤回台湾，形成割据政权。这时候对于清政府来说，台湾已然分裂了出去。这眼瞅着荷兰人在天朝没占到便宜，英国又来捅刀子。1664年，英军攻占北美的新阿姆斯特丹，改名纽约，荷兰人随即反击，第二次英荷战争爆发。但这次英国翻车了，开局不利的荷兰人修整过后在欧洲率舰队直接打入英国本土，趁着伦敦突发火灾直接进入泰晤士河歼灭了英国舰队，彻底封锁了伦敦。这是荷兰人的巅峰时刻，打进伦敦，全歼皇家海军，后无来者。就在战争期间的1665年，荷兰人列文虎克和英国人胡克发明了显微镜，两年后的1667年，毫无还手之力的英国人只得与荷兰签订合约，让出了大量的海外殖民地。这一年的东方，到1669年，清廷两次派人到台湾招抚，但地盘只剩台湾岛和澎湖列岛的郑家接连拒绝了清廷的招抚，坚持“照朝鲜事例”，就是想独立，我可以纳贡。只是迫于地盘太小，又主要是山，挨着大陆的金门岛也不在手里，清廷还在台湾对岸搞暴力拆迁，台湾与大陆地理上和经济上的联系都被切断了，便只好试图通过对外贸易来提升国力。这和几百年前破碎的欧洲不得已发展贸易而最终搞出资本主义的起始客观条件是十分相近的。当时台湾的两个重要的贸易伙伴便是德川幕府时期的日本和英国的东印度公司。荷兰人已经和郑家结下了梁子所以参与不进来。贸易的内容，有很多是战略物资和军备。因为当时的台湾郑家还是想着反攻大陆，至少也要以武谋独。所以台湾自古以来，要想保持独立地位，便会引入外部势力，便会挟洋自重。而穷台政策，只会加速把台湾推向外部势力的怀抱。只是当时英国和日本的实力都还不够强，对台湾的影响力有限。两岸关系就这样僵持住了。这几年间，康熙帝还是个少年，后人所称之盛世就要来到，但恰恰在这几年间，二十几岁的青年蒲松龄创作出了《聊斋志异》。1672年，33岁的蒲松龄回乡教书，此后屡试不第，直到71岁时才成为贡生。而在遥远的欧洲，野心勃勃的法国国王路易十四，就是那位曾给康熙送去浑天仪等科学仪器还私信过康熙同时又痴迷于芭蕾舞的那位长期是法国顶流的皇帝，看法国身边巴掌点大的荷兰却这么豪横，十分不爽，想灭之而后快。而十多年前差点被灭国的英国也一直对荷兰怀恨在心，这两个强盗一合计，便准备瓜分荷兰，于是英国在海上偷袭了荷兰的商船不宣而战，法国则在陆地上进攻荷兰。第三次英荷战争爆发。荷兰海军当时世界第一，但陆军就很一般了，海上继续胖揍英国，但陆地上招架不住法国，无奈只好掘开海堤淹没国土，这才逼退法军。杀敌八百，自损一千。但这期间荷兰国王比较给力，离间英法，同时结盟法国周围的西班牙、奥地利、普鲁士等国，远交近攻，迫使英法联盟分裂。最后英国买了荷兰部分殖民地，荷兰放水淹了自己国家，法国退兵。1674年，战争结束。此前的1673年，康熙十二年，吴三桂突然在西南起兵反叛，清朝的注意力被分散。郑经也顺势收回厦门、漳州、泉州与潮州，还攻下了广东惠州，反攻大陆似乎有望。1675年，当海峡两岸的政权都还在谋求更多的地盘时，英国建立了格林尼治天文台开始测量全世界，并孕育出了我们如今使用的世界时区划分制。1680年，康熙十九年，郑家的水军叛降了清朝，郑经见状心觉反攻无望，便放弃厦门班师回台，之后沉迷于酒色。1681年，郑经嘎了。郑家内部为了夺权再次内斗，而这时候康熙也平定了三藩之乱，灭了吴三桂，趁着郑家内讧，让施琅为水师提督准备收复台湾。1683年，施琅攻克澎湖，威逼台湾本岛，以武促统，即位的郑克降清，全国最后一支明朝抗清势力覆灭，清朝收复台湾，统一中国。而在此前一年，提出“天下兴亡匹夫有责”的顾炎武去世。与此同时，北方的俄国正在向东部凶猛扩张，征服了西伯利亚地区，并开始越过外兴安岭入侵我国黑龙江流域，这也与刚刚入主中原的清王朝发生了冲突。顺治在位的时候，中俄发生过两次冲突，但并没有就此消停。1685年，也就是康熙二十四年，康熙帝在统一中国后开始掉头对付北方的俄国人，并于次年打败俄军围困雅克萨城，迫使沙皇政府和谈来划定两国界限。沙俄在东边的扩张受阻后，又把重心转移到了西方，为了和土耳其争夺黑海的出海口，与奥斯曼帝国爆发了战争。还是1685年，英国的莱布尼茨在继两年前发表的微分学论文后又发表了积分学论文，微积分学创立。而刚刚统一全国的清朝政府指定了专营对外贸易的垄断机构“广州十三行”，洋行制度便始于此。次年，1687年7月5日，英国的牛顿出版了《自然哲学的数学原理》，书中阐释了著名的运动三定律，西方的科学技术也就此进入了一个快速发展时期。一年后，1688年11月初，英国爆发光荣革命，实际上是英国资产阶级的非暴力政变，英国的斯图亚特天主教复辟王朝存在了28年后寿终正寝。过了两三个月，1689年1月，英国出台《权利法案》，奠定了英国君主立宪制的基础。目光再回到东边，还是1689年4月，中俄商定将谈判地点改为尼布楚，结果才过了俩月6月，受到沙俄指使的准噶尔汗国便开始入侵喀尔喀蒙古，主要就是今天的外蒙古到贝加尔湖一带。一度让清军有些被动，但影响不大，只是噶尔丹的叛乱覆水难收。7月初，中俄开始在尼布楚谈判，之后一部分蒙古人不堪沙俄残暴统治而起义，沙俄已无法挽回局面。8月，彼得大帝亲政，但他的主要精力还是和土耳其打仗争出海口，同时向西方学习先进的科学技术，于是俄国人终于不再纠缠，于1689年9月7日和清政府签订了《尼布楚条约》，划分中俄东段边界。《尼布楚条约》是中国与外国划定边界的第一個近代主权国家间的条约。签订这个条约的中国政府是满清朝廷，但使用的国名是中国。这是以国际条约的形式第一次将“中国”作为主权国家的专称。东边基本消停了，就剩平定噶尔丹的叛乱了，那我们再把目光投向西边。1690年荷兰的惠更斯出版了光论，提出光的波动，说导出了光的直线传播和反射折射定律。两年后的1692年东方的王夫之去世，又过了两年，1694年康熙建造的雍和宫赐给四子，随后成为雍亲王府，而英国的莱布尼茨则支制作出了能进行加减乘除运算的机械计算机。1695年黄宗羲去世，他生前认为西方的几何学实际上是源于公元前一世纪，也就是汉武帝时期的周笔算经中的勾股之学。1698年俄国彼得大帝开始改革，同年英国人萨弗里制成的第一台蒸汽机，首次将蒸汽用作工业动力，世界也随即进入到了18世纪。1711年，英国东印度公司在中国广东建立了一个贸易点用白银换取茶叶，中英茶叶贸易开始。1729年，荷兰东印度公司也开始对华直航贸易。于是，欧洲人开始接触到一种此前从未接触过的神秘的东方树叶——茶。由于当时茶只在中国生长，绝对的中国特产就跟熊猫一样只此一家，而茶叶清香的口感颇受欧洲人喜爱，其中丰富的多种有益元素可以预防疾病，大大改善了欧洲人的健康状况。比如在18世纪中期喝茶，预防了城市中的水传播疾病，人口的健康有了保证，城市才得以扩张，工业才得以发展。可以说中国的茶改变了欧洲的历史。西方人发现神秘东方大国虽然没有马可波罗说的遍地黄金，但确实是遍地宝贝。地大物博这个词儿咱是当仁不让的。所以那个时期中国文化在西方声誉极高，西方世界对中国文化产生了某种
