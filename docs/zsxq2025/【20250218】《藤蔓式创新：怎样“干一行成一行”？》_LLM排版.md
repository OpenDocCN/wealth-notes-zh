
# 藤蔓式创新：怎样“干一行成一行”？

今天我们要讲一个商业案例，这是一个很奇葩的公司，日本的雅马哈。雅马哈的钢琴卖不动了，原因是钢琴市场这些年经历了一轮大衰退。

等等，听到这，有人可能会说，不对啊，雅马哈不是做摩托车的吗？还有人可能会说，雅马哈好像是做家具的。没准还有人会说，雅马哈是做路由器，做游艇的。

没错，这说的都是同一个雅马哈，绝对不是重名。这家公司奇葩的地方就在这，它是全世界唯一一个同时做钢琴、摩托、游艇、音箱、家具、房产、路由器、合成器的公司。本来一个公司的业务线丰富一点没问题，但丰富到雅马哈这个程度还是比较少见的。据说很多日本人的状态就是，开着雅马哈，弹着雅马哈，坐着雅马哈，玩着雅马哈，听着雅马哈，住着雅马哈。

注意，雅马哈可不是通过并购做这些业务的，而是每个业务都亲力亲为，并且都做出了名堂。

这也导致，雅马哈一旦遇到经营困难，总能东边不亮西边亮，一个业务衰退了，总有另外一个业务顶上。比如现在，日本的钢琴卖不动，他们就开始做吉他、萨克斯，还有特别受老年人欢迎的电吹管。电吹管也被很多人称为是银发社会最具商业潜质的乐器之一，因为上手快，几分钟就能学会，而且音色柔和，特别对中老年男性的胃口。

你看，这两年很多人说自己领域竞争激烈，想换个赛道。从这个角度看，雅马哈的故事也许对你有所启发。毕竟，谁也说不好黑天鹅什么时候出现。比如雅马哈，钢琴业务下滑还有一个原因，是中国中考从2018年开始取消了艺考生加分。没错，雅马哈原来最主要的客户群体之一，就是中国家长。现在艺考加分取消，这部分客户就迅速流失了。你看，这个

那么，回到行动，雅马哈为什么能经常干一行成一行呢？接下来，咱们就展开说说。

雅马哈的创始人山叶，是一名医疗器械修理工。1887年，在偶然的一次修理风琴的过程中，山叶对风琴的内部构造产生了兴趣。当时日本的所有的风琴都是美国产的，山叶根据看到的构造做零件、搞试验，造出了日本第一台本土产风琴。

但山叶毕竟是外行，这台风琴外形像模像样，却根本调不出音律，没法使用。不过这没有妨碍它成为雅马哈的起点。山叶在东京学习了调音原理，回来之后再次试验，造出了合格的风琴，并且成立了风琴制造所。后来雅马哈的第一个logo，就是一只凤凰，嘴里叼着一只调音用的音叉。

之后，山叶把业务扩大到维修和制作各种西洋乐器，其中就包括钢琴。钢琴的构造和风琴有相似性，但更复杂。修着修着，山叶就觉得，造钢琴比修钢琴要赚钱。因此，雅马哈又开始研究自己造钢琴。1900年，山叶造出了日本第一台立式钢琴，1902年，造出了第一台三角钢琴。

紧接着就是一连串连锁反应，钢琴都研究出来了，那就顺手研究一下其他的乐器吧。因此，雅马哈开始进军各种乐器，造出了长笛、单簧管、小号、长号、吉他、爵士鼓。上世纪七八十年代，雅马哈进入电子乐器领域，先后生产了电子琴、合成器、数字合成器。

这是雅马哈的主要业务之一，乐器领域的时间线。但雅马哈强就强在，过程中还会随时做新的尝试，开辟新的领域。

比如，雅马哈的摩托车业务是从木工开始的。因为造木管乐器，雅马哈掌握了大量木工技术，但这个技术光用来造琴好像有点可惜，于是它就开始进入木质家具领域。而日本又有很多木质房屋，因此，家具造得多了，它又开始建造木质房屋，后来甚至还造了飞机的木质螺旋桨。

据说雅马哈当时借了发动机来测试螺旋桨的性能，但借来的发动机一直出故障，雅马哈就找人自己研制出了发动机。到这儿，雅马哈掌握了引擎技术，在这个基础上，他们就把摩托车给造了出来。后来丰田汽车用的很多发动机都是雅马哈的。雅马哈想，有了发动机，那能不能造摩托艇呢？他们后来还真造出了游艇。造游艇掌握了玻璃钢技术，雅马哈还顺势开始造浴缸，造游乐场。

再比如，雅马哈不光是制造业大户，在电子业也有不少业务。雅马哈做电子乐器的时候，因为拿不下市场上昂贵的元器件，就开始自己研究信号处理，然后一路做出了家庭影院功放、数字信号处理器还有各种专业音响设备。90年代，凭借积累的电子技术，雅马哈还造出了路由器。

当雅马哈具备了足够多的技术、研发了足够多的产品之后，这些产品之间又产生了化学反应。就像钢琴和摩托，原本没什么关系，但是因为扎实的调音技术，很多人都说，雅马哈的引擎声是最好的。还有，因为音频技术方面的积累，雅马哈推出了最早的声音合成器软件VOCALOID，可以输入歌词和音符来让软件唱歌，第一个电子偶像初音未来，还有后面的洛天依，就是基于这个技术造出来的。

这就就是雅马哈错综复杂的 产品发家史。你看，过去我们提到的多元化的大企业，无非两大类，一类是通过收购和并购开辟新领域，比如巴菲特的伯克希尔，从可口可乐买到能源金融；还有印度第一大集团塔塔，横跨航空、化工、服务等等行业。另一类是通过核心技术，不断向新领域放射状迁移，比如三星用电子和半导体技术，扩展了芯片、手机、家电等等领域；三菱集团起步于造船业，通过技术积累逐步扩展到重工业、汽车业。

而雅马哈代表的，是另外的第三种路线，从当下有需要且有兴趣的事出发，走一步解决一个问题，遇到新问题就接着造新的产品。

吴伯凡老师给雅马哈的这种模式起了 一个名字，叫做“藤蔓式创新”。一根藤条在生长的过程中，会不断地蔓延。你可以想象一条爬山虎，它从地上长出来，就像一棵普通的小草。但在往上长的过程中，它会不断分叉、蔓延，从一个点，长成一条线，再扩展成一个面，直到底覆盖一栋楼。这种蔓延式的创新，也正是雅马哈的特点。

从结果这一端来看，它杂乱无章，好像什么都涉及一点儿。但如果从起点开始看，你会发现它的每一步都和前一步有关联，最终形成这样一个顺理成章的结果。

换句话说，创新往往不是孤立发生的，而是在现有的业务和产品基础上，通过微小的改进和变化，演化出新的产品或者技术。这些创新的结果，在结构或者功能上都和原来的业务有相似性，但同时，又有自己的独立性。而很多这样的结果联系起来，就可能有让一家企业发展出足够大的规模。

好，这是雅马哈作为一家企业带来的启发。那么，回到我们个人，明白了这个道理之后，可以做什么呢？其实，雅马哈创新的过程，有点像一类计算机的算法，叫，爬山算法。也就是，想象我们的 人生是一片浓雾密布的丘陵，能见度只有几米，你要怎么，才能到达最高的山峰？

最简单的算法是，无论怎样，都只朝着更高的方向迈一步。但这样做的风险是，如果你出发的地方就在一个比较矮的山上，你最终只能到一个比较矮的山顶，不能到最高的山顶。

更好一点 的算法，是在行走中加入一些随机性，也就是，一开始多尝试，乱走走，减少爬错山的概率，之后可以减少随机性，这样让你有更多机会找到最高的那座山。

而最好的算法其实是，反复把自己随机丢在地形不同的地方，在多次尝试之后，再回头去决定，哪座山是最高的。

放在我们的职业，或者生活选择中，也是这样。或许可以在早期阶段，“随机地”把自己放在不同的地方之中，多尝试、多探索，这可能会反而比一条道走到黑更好，它能让你避免在矮山上打转，有更广阔的可能性。当你认定已经找到属于自己的“最高的山峰”，就更不用浪费时间在其他的山上。
