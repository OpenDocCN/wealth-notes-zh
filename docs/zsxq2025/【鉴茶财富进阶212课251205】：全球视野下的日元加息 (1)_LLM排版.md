
# 全球视野下的日元加息

日本有独立的金融体系吗？没有。一直是美元的附庸式工具。

日元在过去拼命印钱，竟然还能够“避险”，主要是四个原因：

1. G7 工业国的体系和信心
2. 日本在海外拥有巨量资源，包括各种矿产，东南亚的工厂等，有底层资产
3. 日本汇率自走，日元可以便利的兑成美元，日企对美是贸易顺差
4. 日本有美国的“安全保障”，经济和安全形势比较稳定

俄乌和懂王上台以后，东大军力崛起以后，这些基础都逐步不存在了。

2022 年以来国际大资本的布局是：

1. 通过发动俄乌战争，掐断欧洲能源，唱空东大等方式，逼迫全球热钱回到美国，美联储加息加快这一进程。
2. 在美股高位和美国利率高位的时期，套现走人，寻找新的价值洼地，抄底便宜资产。
3. 这个过程不能一蹴而就，需要踏板和通道，日元就成了载体。
4. 美元指数高企时，逼迫日元走势弱，大量美元和美债抵押物（巴菲特就这种），换成更多日元，建立出发阵地。
5. 在阵地上顺手买入商社股，趁势把日本股市也抬高，高峰期套现，多赚一道，继续积攒大量可用现金。
6. 此时你再看 2022 年那些撤出地区，东欧，俄国，是不是需要重建，东大，是不是资产价格处于洼地？准备慢慢买入。

他们准备走了，日元你可以加息了。

一手限制对美贸易顺差，不能再沾美国的便宜，一手再吓唬一下全球市场，一手让自己手中的日元现金能增值。

全球大资本的运作方式是这样的：

先到美国吃一轮美股大涨和美债高利率，再到日本吃一轮日元贬值和日股上涨，最后到东欧东大吃一轮抄底和资产修复。

不知是不是有心人专门把咱们价格杀下来等对方…

懂王把 G2 都喊出来了，也禁止高市胡闹，这大局你不能破坏啊。

预期明年的东西大关系会急剧升温。

3 年一个信贷利率周期，时间差不多将将好。
