
# 双方彩礼谈不拢，男方一走了之，女生应该怎么办？

亲爱的学姐，您好。现有择偶方面的问题向您求助。

我的情况：98年县城私立高中教师，本科师范，家在县城，父母都是底层劳动者，母亲是工厂女工，父亲与他人合伙包室内装修工程，但收入很不稳定。有一弟弟，高中辍学，也是底层劳动者。弟弟曾吃过一段时间抗抑郁的药，我性格也偏悲观，自高中以来，一直在和抑郁焦虑自卑作斗争，因心理咨询价格昂贵，对医院精神科认知不够，因而一直未能接受专业的治疗，直到2024年下半年带高三，并试图在职复习考公，抑郁焦虑就再次复发，情绪很不稳定，精神脆弱，这才有意地找到了精神医院接受治疗，目前一直在接受药物治疗。容貌自评5—6分，在学校会被学生和同事夸赞好看，也被前男友们和相亲对象说好看，精致化妆后，也会吸引路人的目光。目前开始准备健身，微胖，胸臀都有，健身教练评价我的身材是属于普通人中很好的那一类，这是目前对自己的外形的看法，但在以前因为从小黑黑的胖胖的，从小学时就对自己的外貌和身材非常自卑，对外形耿耿于怀，十分在意，甚至幻想自己能突然变成小说中的绝世美女，变成流落在外的有钱人家的小孙女。现在想想有点好笑，但是直到现在外形的好看与否依然能影响到我的情绪。有慕强心理，高敏感，收入6K，存款3W+，内心一直对父母有成见和深深的不信任感，母亲情商低，心思单纯，父亲因收入困难，承担起家庭的内务和情绪价值的提供，过年时甚至会把饭做好端到我妈和我的面前，做饭洗碗都没让我妈我和俩干过，尤其对我极好。我有过一段情感，前男友是洛阳军区的军官。

男友情况：97年体制内铁路警察，身高170，长相小帅，白，有姐姐已婚丁克，哥哥未婚，父母在农村居住，做小生意，四五线城市有房，男友和哥哥从小在城市居住。脾气很好，相处近一年从未见他生气过。但总提及体制内生活痛苦，物欲低，非常节俭，存款30—40W，工资6K，二本土木，朋友圈简单，我知道的只有三个高中同学，男友带我和他们聚过两次，发现在他们谈话时，三个高中同学及他们的女朋友谈得很热烈，但男友总是很沉默，不怎么加入，我不想被冷落，总想加入他们谈话，但并不成功。整个聚餐过程内心很不舒服，告诉男友，男友只安慰了一两句就作罢。随后他们聚餐时男友就再没带我过。

学姐，我该如何自救？希望学姐犀利风，不破不立。

灰玻璃：
两个原因都有，中下阶层的婚姻主要考虑的是实用+家庭话语权，和爱不爱没有多大关系，因为消费不起。爱情本身不需要钱，但爱意的表达却需要钱。
你们能够因为彩礼谈崩，表面上看是你男友提出的“订婚三金1.1w+结婚6.6万”，和你姑姑提出的“订婚6.6w+结婚8.8万”的小差距，实际上是彼此的服从性测试。
彩礼的谈判，在很多农村家庭看来，是婚后家庭话语权格局固定的前戏。是一场谁更强势，谁的女儿/儿子更有市场的游戏。
所以你本人因为抑郁症，根本不想玩这个“谁是高位”的游戏，但你的姑姑，一个外人，比你玩得还起劲。甚至越俎代庖，预感到要玩砸了有不甘心，还反过来一直劝你不要和你男友妥协，让你最好直接断掉。
在学姐看来，你姑姑属实地是越界了。

现在双方僵持在这里，谁都不愿意妥协，还有你姑姑在这里跟个搅屎棍一样，双方都憋着一肚子火。
本来你和男友是相亲恋爱，感情就不深，你对他的描述中多有不满。你抑郁发作后情绪失控，想必对方对你的观感也是如此，双方都觉得对方勉强勉强，谈不上多喜欢。

学姐建议你是否结婚，你只和自己对话。如果你问了自己，确实就这么分手有不甘心的话，那么换一个中间人对接。
比如把你姑姑换成你舅舅，或其他话事人，双方彩礼取个中间值，和颜悦色带着礼物去给对方家一个台阶下。让对方也把彩礼提一提，意思意思给你们家也回敬一些脸面。
然后你曾经对结婚的犹豫、对生育的不确定，对男方本人的爱结不结的态度，需要你自己化解。
需要你自己亲自给他写个小作文，表达对他的情感和“坚定的选择”，对孩子的喜爱和生育规划。这个心结，需要你自己来让男方化解。
当然，如果学姐上述给你的建议，你做不到，也不想做。那就说明你确实不喜欢他，印证了男方说你“不想结婚”的评价，就不必再勉强自己了。

你和你的弟弟都有严重的抑郁症，这可能是因为家族遗传的原因，未来你有很多的路要走。婚姻没法解救你，独身主义、女权也没法解救你，唯有自己才是自己的救世主。
平静客观地接受自己，接纳自己的不优秀，多看一些心理学书籍可以帮助你从焦虑中解救出来。
