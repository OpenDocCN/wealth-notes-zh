
# 关乎钱袋子，金融三巨头再次集体政策“放风”

金融街论坛，是跟上海陆家嘴论坛并列，一北一南的重要金融峰会。三大金融“掌门人”同台亮相谈未来措施，是对“十五五”金融工作的关键发声。

## 央行：角色的深层拓展

央行行长潘功胜的演讲，最值得关注的是三个信号：

**第一个信号：数字人民币“提速”，虚拟货币保持“严打”。** 潘功胜透露了两个关键动作：一是在上海设立“国际运营中心”，专门管跨境合作；二是在北京设立“运营管理中心”，负责系统建设和维护。这意味着数字人民币不再只是“试点”，而是要“全国铺开，走向国际”。更值得关注的是他说，“研究优化数字人民币在货币层次之中的定位”。现在数字人民币主要替代现金（M0），未来会不会纳入更广义的货币统计？比如能不能像银行存款一样计息、能不能直接用于大额支付？这些“定位优化”，可能让数字人民币从“支付工具”变成“真正的货币”，甚至影响咱们的储蓄和理财方式。

**与数字人民币“热推”形成鲜明对比的，是对虚拟货币的“冷处理”。** 潘功胜直接提到，虚拟货币现阶段无法有效满足客户身份识别、反洗钱等方面的基本要求，放大了全球金融监管的漏洞，如洗钱、违规跨境转移资金、恐怖融资等，市场炒作投机的氛围浓厚，增加了全球金融系统的脆弱性，并对一些欠发达经济体的货币主权产生冲击。他明确表示要“继续打击境内虚拟货币的经营和炒作……同时密切跟踪、动态评估境外稳定币的发展。”

**我从潘功胜演讲中理解到的第二个信号：是个人信用修复“开绿灯”。** 潘功胜说，正在研究“一次性的个人信用救济政策”，对“疫情以来违约在一定金额以下且已归还贷款”的个人，征信系统不展示违约记录。这意味着比如2022年有人因为疫情影响还不上贷款，后来虽然补上了，但征信报告上留着“逾期”记录，导致后来买房贷款、办信用卡等被拒。这个政策就是给这类人修复信用：只要逾期金额不大，这个具体标准还没出，并且已经还清，央行就让征信机构把这条记录“隐藏起来”，不再影响个人的后续金融活动。

**我从潘功胜演讲中理解到的第三个信号：是央行的角色做了新拓展。** 潘功胜演讲最后谈的是宏观审慎的逻辑，其中有一个信号值得注意，就是央行角色的新拓展。过去，央行的核心职能是“最后贷款人”——也就是当银行、券商等金融机构没钱了，央行出手借给它们钱，防止金融系统更大的风险。比如2008年金融危机时，央行给商业银行“输血”；2023年一些地方小银行出风险，央行通过再贷款救急。这种操作，核心是“救机构”，确保金融体系的“血脉”不中断。

央行办法很多，如果市场出现“非理性暴跌”，央行可能会更直接地进行处置，提供资金，包括让“国家队”买资产，当那个“稳定器”。

为什么要这么干？因为现在的金融市场太复杂了。以前风险主要在银行体系，现在股市、债市、汇市联动性很强，一个市场崩盘可能传染到另一个市场，甚至冲击实体经济。比如前两年，债市一度快速波动，不少银行理财产品亏钱，引发老百姓“赎回潮”，威胁到了银行的流动性。这时候如果只救银行，不救债市，风险可能越滚越大。所以央行需要扩展职能边界，从“救机构”扩展到“救市场”，直接把金融市场的“火苗”，扑灭在萌芽状态。

## 证监会：核心是从“融资市”变成“投资市”

证监会主席吴清的演讲，核心就是一件事：怎么让资本市场，从“融资市”变成“投资市”。

**第一点，科创板“再升级”，创业板“改规则”，专啃“硬科技”这块硬骨头。** 吴清提到，10月28日“科创成长层”要迎来首批企业上市，还试点了“资深专业机构投资者、预先审阅等”。这句话我翻译一下，就是科创板要“分层”了，以后不是所有“科创企业”都去挤一个板，有技术、有潜力的企业会进“成长层”，享受更宽松的上市门槛、更专注的投资者关注。

另外是创业板改革，要“设置更加符合新兴领域和未来产业创新创业企业特征的上市标准”。现在创业板上市盈利门槛还是比较高，很多AI、生物医药企业，前期投入大、亏损久，确实够不上。新的确定趋势，显然是要进一步降低要求，只要这些企业技术过硬、市场前景够好。

为什么要这么做？吴清说得很清楚：类似人工智能、生物医药这些“新产业、新业态，投入高、风险大、回报周期长”，需要“更大规模的耐心资本投入和更灵活、更包容的融资环境”。简单说，国家要搞科技自立自强，得让资本市场当好“加油站”。

对投资者来说，这意味着未来A股会有更多的“硬科技”公司，但是也要注意，这些企业风险也更高一些，可能“翻十倍”，也可能“亏十年”，你根据自己的情况具体判断。

**第二点，上市公司“强筋骨”，做强实力，中长期资金成为“压舱石”，让A股减少过度波动。** A股为什么总被说“牛短熊长”？据我观察，一个重要原因是上市公司的质量参差不齐，中长期资金，比如养老金、保险资金，不敢长期持有。而吴清这次的思路很直接：

一方面，逼上市公司“做优做强”。他提到要“推出再融资储架发行制度”。什么意思？我解释下，现在上市公司再融资，得一次一次审批，流程长、不确定性大；储架发行是“一次审批、多次发行”，优质公司可以先拿到“融资额度”，需要钱的时候随时发，不用再等审批。这能让上市公司抓住发展机会，也能减少“突击融资”对市场的冲击。吴清还强调，要督促上市公司“加大分红回购增持力度”——说白了，上市公司得把利润分给股东，也要用钱给自己公司的股票投票站台，总之用真金白银证明公司价值。

另一方面，引“长钱”进场。吴清要“推动企业年金、保险资金等全面落实长周期考核”。现在很多保险资金、年金为什么不敢长期买A股？我分析一个重要原因，是考核周期短，比如一年一考核。而长周期考核，简单理解就是“不看短期波动，看三年五年收益”，这样资金就能安心当“股东”，而不是“过客”。吴清还提到“丰富适配长期投资的产品和风险管理工具”——比如更多指数基金、衍生品，让长期资金能更好地控制风险。

## 金融监管总局：服务国家指挥棒

金融监管总局局长李云泽的演讲，主题是“强化金融支撑、守牢风险底线”。李云泽提到，要“推动构建直接融资与间接融资协同、投资于物与投资于人并重、融资期限与产业发展匹配”的金融服务新模式。

我把它拆解成了对金融机构的“三道指令”：

**一是，直接融资和间接融资一起上。** 间接融资是银行贷款，直接融资是发股、发债。以前大部分企业融资主要靠银行，现在得让股市、债市发挥更大作用，给企业提供精准融资。

**二是，投资于物和投资于人并重。** “投资于物”是给工厂、设备贷款，“投资于人”是给教育、医疗、养老这些以“人”为核心的领域投钱。比如现在为了应对老龄化，得发展养老产业，金融机构就得开发养老理财、养老保险产品；比如职业教育需要钱，银行就要给职业院校放贷。

**三是，融资期限和产业发展匹配。** 比如制造业升级要长期投入，银行就得多给中长期贷款；比如小微企业周转快，就得有短期、随借随还的贷款。

李云泽特别提到，“锚定高水平科技自立自强，完善长期资本投早、投小、投长期、投硬科技的支持政策”。我理解，就是鼓励银行给刚起步的科技企业钱，以及别总想着赚快钱，要支持那些研发周期长、但能突破“卡脖子”技术的企业。背后的信号就是，金融不能再“嫌贫爱富”，得和国家战略，同频共振。
