
# 美联储的短期框架

## 发布专栏：知几共话

## 特朗普和鲍威尔，谁才是美国滞涨的救世主

2025年8月22日，鲍威尔最后一次以美联储主席身份在杰克逊霍尔发表演讲，他表示，“基本前景和不断变化的风险平衡，可能需要我们调整政策立场”。这一表态被市场解读为9月降息的强烈信号。

此前，市场普遍预期鲍威尔不会在此次会议上明确释放降息信号，因此这一表态引发了市场的剧烈反应，黄金价格瞬间暴涨逾30美元。

同时，美元指数闻讯下跌0.5%，表明市场对美元的信心有所减弱。美国三大股指也纷纷上涨，涨幅均超过1%，显示出市场对经济前景的乐观情绪。

鲍威尔与特朗普之间的意见分歧早已人尽皆知，此次最后一次以美联储主席的身份释放降息信号，鲍威尔究竟是鱼死网破的最后一搏，还是背后有更长远的考量？这其中的逻辑是怎样的呢？鲍威尔的想法最终能够实现吗？

## 短鸽长鹰。但问题是特朗普不会允许他长，所以就是鸽。

感觉短期也不是鸽。我认为美股有点反应过度了，9月，特别是9月的下半月，我会看空美股。

我也觉得市场反应过度，市场强行解释为鸽派利好，选择性忽视其他讲话内容。

## 看完讲话，我的理解是，鲍威尔认为特朗普搅乱了美国的就业市场秩序，所以美联储需要通过降息来解决就业市场的问题。

现在关注点就两个，一个是通胀，一个是就业。

通胀受关税影响，需要判断关税传导时间，是否是一次性的冲击。现在的数据有一些抬头，但并没有特别明显的反馈，得再观望下一步的数据，但已经好几个月了，老鲍也没办法说一直等下去，所以这块有点偏软；

就业方面，目前保持稳定，但本质是供给侧和需求侧都下降带来的短暂平衡。而鲍威尔担心的是需求侧的问题，一旦经济真的下滑，后续可能引发持续的需求收缩。

要在通胀与就业之间取得平衡，目前鲍威尔的选择是，让通胀降下来，而这就需要就业数据下滑作为充分理由。鲍威尔认为未来可能会出现大滞涨吧。

## 基于以上的逻辑来看，即使9月降息，也未必代表降息周期就开启了，可能只是一次满足市场需求的短期行为。

我认为鲍威尔是意识到，特朗普的搞法，可能会给美国经济埋下一个巨大的滞胀螺旋，所以未来要调整美联储的应对框架。

过去的就业率数据是基于不断引入外来移民，带来源源不断的实际性劳动增量来计算的。特朗普之后的就业率数据是基于停止了外来移民，增量不变、存量调整来计算的，也就是指标基础已经完全不一样了。现在的就业率对美联储来说已经没有意义了。

美国现在神奇的是新增就业差的同时，失业率也很低。

这还是因为特朗普上台后通过强制手段，把不缴社保税的非法移民从美国就业市场驱离，找低端工作的人大幅减少，就业率就上来了。

## 但企业雇员从原本不缴社保税的移民，换成必须缴社保税的红脖子，成本上升，企业也就缺乏了扩大生产招人的动力，导致新增就业的断崖下滑。

这种就业率与失业率的双下滑，很可能会造成生产力与GDP的螺旋下滑，如果美国的新质生产力再不能带来生产力大爆发，那么就很可能将陷入到螺旋滞胀的大危机当中。

美联储调整框架其实是去年就在提的，所以也不能完全算在特朗普头上，不过特朗普这玩法，确实有点过头，要限制。
