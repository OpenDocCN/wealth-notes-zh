
# 民企座谈会的背后安排

源自2月17日朋友沟通
发布于2025年2月18日

江苏省委不惜自损身价来捧杭州，是因为首辅一系来自于江沪浙，首辅的首席秘书就是从南京知府位置升上去的，需要帮助同一阵营的兄弟们造势，在媒体圈这叫做以腕养腕，影帝们自贬身价来帮助小鲜肉提咖位。

同时，这次座谈会首辅首次出席，企业名单自然也就由内阁和首辅首席秘书草拟，浙江的杭州温州自然就会成为大赢家。不仅江苏在宣传上竭力配合，进群加微信cjp168319 上海也把自家的名额让给了上海的浙商(韦尔)。更重要的是，把二马都请出来之后，内阁的经济发展也终于迈过了最大的历史屏障，毕竟，当年在上海投资马斯克的首辅大人，是这一波美国大选最大的中国赢家之一。而 北京这次的推广从24年的百度换成了25年的小米，是因为特朗普几个月前赢得了大选。

那么北京就不敢再继续打压二号首长马斯克来遏制上海，为了能够继续在京沪大战中扳回劣势，就需要选择扶持对标特斯拉的北京企业，于是，对标OpenAI的百度变成了对标特斯拉的小米，运气好的雷军，再一次被推上了风口。
