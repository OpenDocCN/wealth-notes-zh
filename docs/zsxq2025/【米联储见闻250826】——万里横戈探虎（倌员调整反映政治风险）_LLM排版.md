
# 米联储见闻250826——万里横戈探虎（倌员调整反映政治风险）

## 你好，我是米糕

昨天王莉霞的事说了头，因为时间关系没有展开，昨天也提示了A股可能存在的政治风险，今天继续分析一下从最近这些官员调整反映出来的政治风险。

王当天在会场被带走时，引起了很多人的不安，原因在于王的身份还是有些特殊的，作为少数民族自治区为数不多的女性正部级高官，同时担任中央委员，王背后的力量肯定是很强大的，王今年61，作为正部级干部，还是正当打的年纪，所以无论从哪个角度来看，这个时候把她拿下来都不是简单的反腐，背后必有政治原因。在王被调查前，中国共有12位女性正部级官员，其中三人担任省级政府一把手；王莉霞落马后，目前仍在任的省级女性正部级官员，仅剩56岁的四川省长施小琳，以及63岁的黑龙江省长梁惠玲。在四中全会处理中央委员，肯定不是一般的经济原因，近期处理的中央委员除了王莉霞还有本周未出席公开活动的西藏军区司令员王凯。中央委员是中央领导人行列，里面没有纯业务官员，都是各路嫡系。所以如此密集的处理中央委员，让人对目前高层的政治博弈的状态不安。

王是8月18日被中纪委带走，并将其定性为“压倒骆驼的最后一根稻草”，海外人士分析，王的落马不仅是内蒙的问题，而是与高层权力调整、北戴河会议以及高层派系博弈密切相关，其中可能涉及老王旧部围剿和老赵被边缘化有关。王莉霞被带走的过程也非常不同寻常，似乎就是要高调处理，达到震慑效果，之前有传言说王是在会场被带走的，但熟悉内情的人爆料，她是8月18日傍晚18时左右，从巴彦淖尔盟乌拉特后旗的抗灾一线返回呼和浩特，当时王正结束乌拉特后旗的现场会议，秘书接到电话，要求她立即返回自治区党委，参加由党委书记孙绍骋主持的紧急会议，学习中央领导关于救灾的指示。她匆匆结束会议，车队向呼和浩特进发。途经包头时，王指示秘书通知开车先回家换衣服洗澡，但很快，电话又来，称会议已开始，其他部门同志均到齐，只能直接前往内蒙古政府大楼。只能前往位于政府东侧四楼的主席楼，王莉霞下车准备上楼，却被三名身着白衬衫黑裤子的中纪委人员拦住，出示证件后，王试图上楼换衣服并处理会议事宜，但被拒绝。王当场震惊，腿软瘫倒，被旁人扶住后塞入商务车，迅速离去。当晚她可能被移送北京或山西接受调查。随后，她的亲信和相关人员如弘茂集团董事长樊晓平被抓捕。樊晓平长期与王保持政商互助关系。据爆料人称，王在途中曾多次核对会议细节，说明对于自己被捕还是有预感。王在体制内被称为“换装女王”，一年穿衣开销不低，衣服多为海外无标志定制，一套三四十万欧元；高跟鞋一双十几二十万欧元，饰品如胸花也价值不菲，专门有海外自媒体通过内蒙新闻看她每天换装的频率。路边社新闻称，王的亲侄子与浙江老板合作，在阿拉善盟买或承包煤矿，2023年阿拉善2.22重大煤矿事故，死亡53人，2024年5月，赤峰又在出现矿难4人遇难，王马仔内蒙应急管理厅厅长常志刚在2024年6月落马，据称与其在多次矿难事故中的责任问题有关，常被控后，全盘供出王。他俩在呼和浩特曾是正副级关系。王的落马实际上是这几次矿难叠加的结果，2.22矿难曝光的死亡人数是53，实际损失可能不止。

王现在落马，可能导致赵那条线的人在21大占位中会相对失势。这半年以来很多人事调整反映出一个问题，领导虽表面“一尊独大”，但并非没有挑战者，而且随着他身体和年龄的变化，这种挑战的力量愈发蠢蠢欲动，对苗的处理本质是对这种想法和动作的警告和敲打，但是在权力博弈中，这种变化就是生死一线，不管风险多高，都会做利益最大化的操作，隐形风险和挑战无处不在，21届董事会的人事安排，会是今明两年政治上的重点，可以说这段时间，外部内部都是稳字当头，外部内部的冲突都会围绕这个核心问题展开。

之前海外有传闻，说四后会后有权力分化，仅担任部分职务，有这种认识的人对权力的运作和逻辑完全没有基本认知，尤其是在春夏之交的事情发生之后，春夏之交为啥会发生，恰恰是董事会出现权力过度分散的结果，事实上，不管从领导的性格，还是现阶段政治生态和出境，还有外部环境来看，不存在分散权力的可能性，到王落马，五自治区主席——新疆努尔白克力、西藏齐扎拉、宁夏刘慧、广西蓝天立、内蒙古王莉霞全部落马，一把手亲自赴藏参加60周年大庆的操作，都表明，所谓分散权力不存在政治空间和实操的可能性。大家还记得2020年初那个亲自指挥，亲自部署吗？性格这个东西是刻进骨子里的，只要还有一口气在就不会妥协的。

最近朝鲜高调举行了表彰朝鲜人民军海外作战部队指战员的国家授奖仪式，小金亲自参加并讲话，他搂着牺牲士兵家遗孀孩子的照片更是传遍全网，小金还会见了返回平壤的朝鲜人民军海外作战部队主要指挥员，出席了祝贺演出，并单腿跪地为阵亡士兵授共和国英雄勋章。这次授奖仪式发生于美俄领导人会面之后，朝鲜此时公开授奖援俄士兵，说明俄乌冲突在国际领域的定性已经确定，所以援俄是正义之举，是合乎联合国宪法的行为，朝鲜才会公开纪念。

小金是在2024年8月28日，即乌克兰军队突然出兵占领库尔斯克三周后，亲自下令朝鲜人民军前往俄罗斯库尔斯克地区参战。之后，随着战事推进，朝鲜分别在2024年10月22日、12月12日和12月22日，先后三次以国务委员长名义批准作战计划，亲笔向特战部队下达作战命令。朝鲜方面还发布纪录片披露了许多朝鲜士兵参战的罕见画面，在国家授奖仪式的讲话中，金还提到了不少朝鲜士兵的英勇事迹。自俄朝签署盟友协议，朝鲜出兵俄罗斯以来，平壤共往俄乌前线派兵人数大约在1.2万至1.8万人之间。截至2025年8月，经公开确认的阵亡人数约为101人，从这次金出席仪式的现场士兵牌位看，也和这个数字大致相当。考虑到俄乌战争的惨烈程度，加上俄罗斯对朝鲜提供的回报，这个伤亡数字对小金和朝鲜来说都是非常值得的投入，战损比例远低于俄军和乌军，从另一个侧面也可看出，朝鲜士兵在高强度现代化战场的作战能力属于全球顶尖水平，朝鲜的军事装备与科技水平也没有大家想象的那么落后，朝鲜士兵的战场适应与生存能力，作战能力和作风要远超比俄乌士兵。可以说，这一战不但为朝鲜创造了经济价值，改善了国家的资源紧缺问题，也打出了风格和精神面貌，让世界无法小觑朝鲜的军事实力。

从军事及政治层面，这不仅是金正恩的胜利，也是朝鲜的胜利，所以朝鲜才会举行如此盛大的授勋仪式来表彰作战部队。对内，向朝鲜民众展现了金卓越的军事才能和对军队的掌控，以及对国际复杂事务的高超驾驭能力，作为娇生惯养的内定接班人，且从未经过战争历练的80后，小金在错复杂国际局势中，果断出手，且高效精准，这个能力相当了得，让人对小金的能力要再度重估。朝鲜人民军的作战实力强悍，可以对敌对国起到不战而屈人之兵的效果。以极小代价获得极大回报，能养军能带兵能用人，这其实也是小金能力的体现。金正恩在国家授勋仪式时说：当下朝鲜面临的外部安全威胁，我军海外作战部队官兵所发扬的英雄主义，令很多国家禁不住羡慕，同时感到畏惧。要知道，过万朝鲜士兵入俄作战，竟然没有一个俘虏，只有死士，在这个时代简直无法不令人动容。小金的说法并非是在国民面前自吹自擂。一直以来，朝鲜都面临着食品短缺问题，核武技术也存在投送能力短板，无法对其最大的想象敌美国形成有效威胁。通过参战，朝鲜获得了急需的粮食援助和导弹技术，包括核远程投送能力，这些赴俄参战的朝鲜士兵确实是为国家发展立下汗马功劳的英雄，解决了朝鲜的军事短板和燃眉之急。对朝鲜来说，俄乌战争是一场典型的代理人战争，另一个令人钦佩的地方在于小金对介入时间点的把握， 如果以基金经理的角度来看，小金几乎是在趋势扭转的前夜做出了精准投资，且投资对于整个盘面的走势发挥了巨大作用。通过参战，朝鲜成功重建朝俄盟友关系，并将两国关系上升到血盟高度，这份朝鲜人民军用性命换来的交情，俄罗斯人哪怕再势力再傲慢，也不会无动於衷。同时获得了俄罗斯的协防承诺，这意味着朝鲜从原来背后只有中国一个安全支柱，变成同时拥有中俄两个安全支柱，且增加了朝鲜在中俄关系中游走操作的空间，不仅极大增强朝鲜的军事安全能力，还增强了金在中俄之间的游走、要价、谈判能力，强化了朝鲜和美日韩进行军事对抗的底气和信心。全球三巨头，朝鲜通过战争手段将其中两个与自己牢牢捆绑，从而保障了朝鲜的地缘政治安全，这操盘能力不但远超金正日，甚至不逊于金日成，我相信很多国家和企业巨头都会羡慕朝鲜这种封闭体系居然能培养出这么优秀的接班人，实属不易。

对比李在明这次访美的丑态百出，韩国的奴颜卑族，朝鲜在与大国博弈保障国家利益方面，要比韩国有尊严得多，获得的实惠也更多，果然是家族企业和股份制不同，一个是真心希望国家好，一个是赶紧多捞一点安全落地。相比于自诩为交易大师的川普，小金可能才是当下国际政治舞台上，最牛的交易者之一，最强80后之名，依然无人可以取代。
