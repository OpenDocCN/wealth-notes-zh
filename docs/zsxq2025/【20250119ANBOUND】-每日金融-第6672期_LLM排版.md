
# 中国企业为何总陷低价竞争怪圈？

在全球化的浪潮下，中国依靠其比较优势积极参与全球市场。尤其是2001年中国加入WTO后，进一步加速这一进程。得益于中国庞大的劳动力供应、相对低廉的人工、环境、能源、土地成本，以及较低的社会保障水平，再加上劳动者的吃苦耐劳精神，中国迅速成为全球市场上的“世界工厂”。从那时起，低价似乎成了“中国制造”的标签和标志。

从企业角度看，中国企业最大的优势在于其规模化生产制造能力。与发达国家相比，中国企业在原始创新和盈利能力上或许并不突出，但它们在生产规模上的竞争力却异常强大。一旦中国企业掌握了某个领域的技术和生产能力，这个行业很快就会被中国企业迅速占领。庞大的生产能力会让过去曾由少数企业主导、拥有较高利润的市场迅速变得价格低廉，甚至趋向“白菜化”。

对消费者来说，低廉的商品价格无疑是好事，既能带来实惠，也有助于全球范围内拉低通胀水平。不过，从中国产业和企业发展的角度来看，单纯依赖产能规模的扩张也存在隐忧。以光伏、电动车、锂电池、智能终端等新兴产业为例，中国企业在争夺市场份额时，往往会将“高端产业”压缩为“低端产业”，甚至把有盈利潜力的企业推向无利可图的困境，造成产品质量的整体下滑，最终让消费者失去信心。从系统性来看，这种情况对国家和民众的经济效率并不高，投资回报率也显著降低。

有市场人士提出，造成这一现象的根本原因是什么？为何发达国家的企业很少出现激烈的低价竞争？中国企业应如何避免这种局面？这些问题值得深入思考。

中国企业在市场上迅速低价竞争并非新现象。早在全球化蓬勃发展的时期，国际市场上便流传着“中国买什么，什么就贵；中国卖什么，什么就便宜”的说法。简要分析中国制造企业为何倾向于通过低价迅速占领市场，背后有以下几个主要原因。

一是创新能力不足与创新文化匮乏。表面看，中国市场上的“创新”产品不少，特别是一些头部科技公司，研发投入也很高，但真正从零到一的原始创新少之又少。更多的时候，中国企业还是处于“跟随模式”，比如百度模仿谷歌，新能源车追随特斯拉，京东模仿亚马逊。这种创新路径的问题在于，虽然能快速赶上，但缺乏深度的创新文化作为支撑。国内很多企业更倾向于短平快的模仿，而不是投入时间、资源去做基础研究或者突破性的技术研发。如果企业扎根于长期创新而非短期利益，就不会让少数热门赛道被“挤爆”。说到底，创新文化的缺失不仅是企业的问题，更需要从教育、政策和社会氛围上全面培育，创造一个鼓励冒险、包容失败的环境。

二是政策引导的“过度精准”问题。政策支持本该是为市场保驾护航，但过度引导反而会让市场畸形发展。很多行业因为政策“点名”，资本和资源一窝蜂涌入，导致产能过剩、价格“白菜化”。这种“一条路走到黑”的现象在光伏、新能源汽车等领域屡见不鲜。更大的问题是，政策干预往往替代了市场的调节作用，资本流向由政策决定，而不是市场需求。比如某些地方政府为了吸引企业，倾斜资源扶持，结果是市场失衡、竞争环境恶化。政策应更多发挥指引作用，而不是成为市场的主导力量。要让资源真正按市场规律流动，政策就不能越位，而应聚焦于优化环境、引导长期发展。

三是商业文化的合作缺失。中国企业在商业竞争中，合作精神的缺失是一个长期存在的问题。很多企业习惯于各自为战，甚至为了争夺市场展开“同归于尽”的价格战。这样的恶性竞争看似在抢占市场份额，实际上对行业发展毫无益处。一些企业对短期利益的过度追求，导致市场被“做烂”，失去了可持续发展的动力。而健康的商业文化，核心在于创新与合作。像德国的工业体系，企业之间不仅竞争，还在技术标准、产业链整合上保持协作。相比之下，中国企业的“单打独斗”文化明显不利于形成一个良性发展的生态。打造合作文化，需要企业、行业协会甚至政策的共同努力，引导企业从价格战向价值战转变。

四是地方政府过度干预市场竞争。地方政府为吸引投资、抢占资源，不惜直接下场竞争，这种行为在近年来愈发明显。尤其是地方设立的产业基金，通过提供巨额资本支持，变相推动掠夺性竞争。这种模式表面看给企业提供资金保障，但实际上助长了市场内卷。更关键的是，这种大规模财政投入可能并没有真正考虑市场需求，只是为了短期政绩而快速推进。过去常见的土地补贴、税收优惠，现在被更加隐蔽的基金跟投取代，形式换了，但本质没变。问题在于，这种做法一方面，巨额资金跟投常常忽视市场规律，容易导致资源错配和产能过剩；另一方面，大规模的财政投入如果缺乏专业管理和风险把控，不仅会让市场失序，还可能为地方财政埋下隐患。因此，地方政府需要重新审视自己的角色，不应一味“下场抢项目”，而是要从规则制定、资源整合的角度，提供一个公平透明的竞争环境。通过科学决策、专业化运作，让产业基金真正发挥引导作用，而不是沦为市场内卷的助推器。

五是行政区域分割阻碍全国统一市场。中国的行政区域划分导致市场资源被区域化割裂，形成了一个个相对封闭的“小市场”。地方政府为保护本地利益，往往通过区域保护、补贴政策或设置市场准入门槛等方式构建隐性壁垒，看似扶持了本地企业，实则限制了跨区域竞争，妨碍资源的优化配置。这种割裂让企业在有限的区域内内卷式竞争，更倾向于价格战和同质化，而非通过创新或提升质量实现差异化竞争。虽然国家近年来出台了《全国统一大市场标准指引》，但地方政府在经济和行政上的职能交叉，加之利益驱动，执行过程中往往力不从心或存在阻碍。地方间更多依赖行政命令协调，而非依托市场机制，这进一步限制要素的自由流动。

最终分析结论：
我们并非否定中国企业在全球经济中的突出成就，也并非对中国企业未来发展的前景缺乏信心。在逆全球化和地缘政治影响下，虽然中国企业在全球博弈中发挥重要作用，但我们仍需要深入反思中国企业在发展市场经济过程中存在的一些结构性问题。特别是在创新能力的提升和产业发展路径的选择上，仍存在亟待解决的问题。只有当中国企业真正将发展动力建立在持续创新的基础上，并突破依赖低价竞争的旧有路径时，才能有效规避低价竞争的困境，实现从规模竞争向高质量发展的转型。

## 优选信息
【政策：证监会发布资本市场首部系统性行政处罚标准】
近日，证监会发布《中国证监会行政处罚裁量基本规则》（下称《裁量规则》），将于2025年3月1日起施行。《裁量规则》具有诸多看点：第一，鼓励“吹哨人”，对于案发前主动举报单位违法行为，并且积极配合查处的“吹哨人”，依法减轻处罚；第二，鼓励“坦白”，对违法事实没有异议，签署认错认罚具结书的，可以从轻处罚；第三，严惩贿赂、隐藏证据材料等行为，对于存在涉及贿赂，毁损、伪造、篡改证据材料等情形的，将被从重处罚；第四，建立主要负责人批准或者集体讨论制度。对情节复杂或者重大违法行为给予行政处罚的，应当经负责人集体讨论决定。值得一提的是，鼓励“坦白”的规定是证监会此番新设，此前并无类似规定，旨在激励相关违法违规当事人主动认错，以提高执法效率。证监会有关部门负责人表示，《裁量规则》具有三大突出特点。一是行政处罚裁量更加精准和透明，二是体现从严监管的政策导向，三是坚持处罚与教育相结合。对此，安邦智库（ANBOUND）的研究认为，《裁量规则》属于规范性文件。其出台一方面意味着证监会、各地证监局、交易所就资本市场违法违规乱象进行行政处罚拥有统一执法尺度；另一方面使得上市公司、中介机构等资本市场参与方拥有对照自查和争取权益的参考依据。

【形势要点：美联储退出央行与监管机构绿色金融网络】
近日，美联储宣布退出央行与监管机构绿色金融网络（NGFS）。据悉，NGFS是一家由全球央行等监管机构组成的组织，该组织致力于研究如何通过金融系统监管来应对气候变化。美联储表示，之所以作出退出的决定，是因为NGFS时至今日的工作范围已经超出了美联储的职责范畴。有分析人士表示，长期以来，特朗普对各国政府制定气候变化政策的努力持批评态度，此举或许是美联储试图采取措施避免受到特朗普批评的最新例子，尽管美联储长期以来旨在捍卫其独立性。随着特朗普即将重返白宫，业界担忧美联储未来的货币政策能否保持独立性。美国现任总统拜登的一名高级经济顾问警告称：“如果即将上任的特朗普政府真的去干预美联储的利率决策，可能会重燃通胀。”白宫经济顾问委员会主席贾里德·伯恩斯坦也提到特朗普过去表露的想法，即行政部门应该对央行的利率制定拥有更大的发言权。

【形势要点：香港2024年破产呈请企业数量创新高】
香港官方通报，香港去年破产呈请数量写下九年新高。据《明报》日前报道，香港破产管理署发布的最新数据显示，香港去年破产呈请数量录得9190起，按年增16.9%，是自2015年9875起后的最高水平。其中去年12月录得781起，按年增15.7%。数据也显示，香港法院去年全年发出接管或破产令的数量达8553起，按年增16.4%，是自2017年以来最高。强制公司清盘呈请数量录得740起，按年增30.7%，为2009年以来最高。法院发出清盘令录得443起，按年增25.1%，也创2009年以来最高。香港餐饮联业协会会长黄家和称，去年经营环境困难，不少中小企结业，使破产数字增加。不过，2024年12月起实行的“一签多行”，让深圳市民陆续到港消费，加上举办更多盛事，带动香港餐饮行业生意有所起色。虽然港人北上消费已无法逆转，但近一个月留港市民也多外出用餐，令餐厅延长营业时间，刺激夜市经济，从而吸引更多旅客到港。

【形势要点：大量资金外流致使人民币汇率承压】
近期，人民币汇率面临着巨大的下行压力，彭博社报道指出，除了中国经济疲软、美元飙升和美国提高关税前景的因素之外，大量资金寻求海外投资越来越成为人民币汇率面临额外下行压力的重要因素。日前彭博社称，随着投资者在海外寻求更高回报，追踪资本流入和流出的中国资本账户在去年出现创纪录的流出。根据它的计算，中国本土银行去年代表客户向海外投资的资金净额为1.33万亿元人民币(约合1820亿美元)，创下了纪录。中国的资本账户一直面临压力的主要原因，是因为外国直接投资流入中国的减少、当地企业向海外扩张的意愿以及资金从当地股市撤出，这些因素加在一起导致对美元的需求增加、资金外流加剧和人民币波动加剧。与此同时，中国多年来的贸易顺差(去年攀升至创纪录的9920亿美元)所提供的支持正在逐渐减弱。由于美国资产的收益率高于中国资产，出口商更愿意持有美元收益。彭博社称，这是前所未有的。如果这种失衡持续下去，可能会引发对资本严重流失的担忧，削弱中国政府管理人民币的能力，并加大监管部门出手打击的可能性。最近几周，中国央行多次就扰乱市场的行为发出警告，并试图通过设定每日人民币中间价，保持对人民币汇率的严格控制。中国央行还采取行动，承诺在香港发行创纪录规模的票据，以减少离岸市场的流动性。西班牙对外银行(BBVA)香港首席亚洲经济学家夏乐表示，资本账户流动对人民币的影响越来越大，可能会使中国央行支撑人民币的努力进一步复杂化。
