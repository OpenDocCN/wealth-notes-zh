
# 六月必读：警惕金银比陷阱
原创：失业期神 公众号：失业期神 25年6月22日

## 前言

## 金银比，是财富还是骗局?
美国“对等关税”政策落地后，白银单月暴跌12%,
而后因为关税政策的不及预期，预期的修复导致工业属性发力，加上黄金的新高带动，导致最近白银也创下了新高，那本文就基于白银的货币属性与工业属性，聊聊金银比的合理性。
白银工业需求对全球经济衰退预期高度敏感，关税加剧供应链断裂风险，直接压制电子、光伏新能源等核心用银领域，同期铜、铝等工业金属同步大跌，表明白银与顺周期大宗商品而非黄金的避险逻辑深度绑定；而后在川普的交易艺术下，关税危机开始逐渐降级，衰退受益(避险)品种黄金调整，白银也因此创下新高，这里面不仅蕴含着复苏预期，也带有黄金的平移投机资金利用传统的“金银币”做切换带来的冲击。
那金银比是合理的吗?我先说结论：往往是黄金先因为逻辑的顺畅先涨，而后白银会通过一些类似于“逼仓”的逻辑暴涨修复金银比，我认为是历史的产物，放在全球衰退期并不合理，因为白银在衰退期，是没有资格锚定黄金的。

黄金带有99%的货币属性，而白银只有40%，且其货币属性因为稀缺度的下降还在每年持续下降，金银比的修复往往发生在繁荣预期阶段，而金银比的拉大往往发生在衰退预期阶段。
恰是因为全球经历了太久的繁荣，而让金银比这个传统概念被资金承认且深度定义，导致每次金银币突破，都会引来巨额的套利资金做“空金多银”的套利切换，但是其实这几年在逆全球化体系下，衰退已经成为底色，货币属性才是黄金新高的基石，在这种概念下，我认为金银比处于失效的边缘。

一、货币属性分析：央行储备里有黄金，且黄金和美债两者成为外汇储备表的最重要的成分，这就给黄金赋予了货币属性，这是白银所没有的地位，即使大家把白银称作贵金属二弟，但两者的货币成分相去甚远，不能进入央行金库的金属不配称贵金属，这是它的身份死刑。

二、工业属性分析：白银在工业上的需求主要来自于光伏与新能源。关税政策下光伏企业大幅削减产能，电池银耗的下降、比亚迪的“汽车恒大”事件以及后续的“60天账期”事件，都在印证了光伏新能源处于收缩周期的节点，工业属性塌陷让碳酸锂、多晶硅等相关商品持续暴跌，同带有相关属性的白银，面临需求死刑。

三、美股相关性：历史上，金银比的冲高往往与恐慌指数VIX 正相关，意味着做多金银比就等于做多美股恐慌率，这也是为什么我认为白银的坚挺，也有部分依赖于美股的繁荣预期。

四、多空力量分析：白银的逼仓预期可以被资金推动，但是事实上发生的概率几乎为零，这是资本死刑。
1、白银供给太过稳定且多源，导致其显性库存历史新高：除了银矿本身的供给，白银的工业伴生主要依赖于重金属硫化物矿的湿法电解工艺，一是铜、铅、锌等金属的电解冶炼过程中，阳极泥会富集银、铂等贵金属，这是全球白银供应的主要渠道。二是部分金矿伴生银产量可达黄金的4倍以上，构成另一重要来源。
2、白银远月合约升水幅度高达年化8%，隐含持有成本高企，深度contango结构决定了其逼仓可能性极低。

五、流动性分析：白银极大受益于全球流动性宽松，但是黄金受益度的比较级是很低的，甚至相对受损，因为降低了硬着陆的可能性，也削弱了其避险属性的利好程度。在本周的议息会议里，鲍威尔在通胀的担忧下深度放鹰，本年度大幅降息的可能性在逐渐下降，甚至有7个委员投了今年不降息的一票，这种情况下我认为对黄金是好事，而对白银是压制。

六、规模分析：白银的盘子是黄金的十分之一，价格的大波动让其更加容易受到套利盘以及投机盘的青睐，特别是金银比突破历史极值时，这种趋势性资金大量扎堆，推高了风险，特别是当黄金不涨的时候，白银下行风险会更大。

所以我认为，金银比更多是一种资金经过漫长的历史经验而给大众强行植入的一种关联性，事实上在我眼里，衰退期的“金银比”如同“金房比”或者“金茅比”，是没有太多意义和关联逻辑的。就如我直播所说，黄金如同股票市场里的龙头大票，白银就像概念小票，在龙头大涨后，资金会因为投机需求去做多相关概念导致的轮动或者补涨，这在上行期可以找到合理解释，但是在下行期是矛盾的。

因此我上周开始把白银加入到商品空单组合里《2025年通缩投资白皮书》，用多金空银的方式，做多金银比的同时，也对冲了部分黄金的尾部风险。
