
# 聊一些形而上学的话题 (1)

市场就在那里，用的心去感受，而不是天天像个没头苍蝇一样，妄想找一副拐杖做定心丸。市场是在不断否定、不断修正、不断变化中前行的。人生也是如此。

这个世界，你唯一能靠的，其实只有自己。

第三个问题：情绪管理。我见过在二级市场长久的，做成功的，都是情绪管理做得非常好。向内看非常重要。内心要无比平静和坚定。一旦感觉到自己心浮气躁，就停止一段时间交易，更不要把那种心浮气躁、骂骂咧咧的行为传染给身边的人。保持定力非常重要。

第四个问题：放低你的预期。当下的无风险收益年化也就2％左右。顶级基金经理长期年化也就20％多。为什么有些人买进就会VY我要吃十个板（甚至不知道短期翻倍就进监管）我要涨几倍。定要放低预期。可以样，比如，分解成年能有个月以上正收益，每个月稳定2％就满足。这样可能更容易实现？

第五个问题：计划你的交易、减少出手次数、提高交易胜率。这里面涉及了每次交易都要测算风报比（盈亏比）、买多少仓位看错了如何应对，如何设定止盈策略、交易中的硬成本（佣金 印花税之类）冲击成本消耗。千万不能有完美主义的想法，什么买在最低卖在最高，什么要求100％胜率。千万不能有。

大盈小亏、螺旋形上升已经实属高阶了。

第六个问题：不影响生活的闲钱，给自己三次试错的机会。不影响生活的闲钱，一般是六个月到三年之间的薪资收入。然后用10年、20年时间把这笔钱如滚雪球般慢慢滚大。并且，找个机会，把本金取出。从生的角度，给自己三次这样的试错机会。如果三次都失败，说明你不适合这个市场。

放过自己，生活如此精彩，退步海阔天空。不太喜欢长篇大论废话，暂时想到的就这些。欢迎补充。
