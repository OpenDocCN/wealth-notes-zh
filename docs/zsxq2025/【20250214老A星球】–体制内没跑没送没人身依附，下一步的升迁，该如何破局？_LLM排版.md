
# 体制内没跑没送没人身依附，下一步的升迁，该如何破局？

匿名用户提问：您好A处，我快33岁了，
女，已参加工作10年半，目前是某乡镇党
群副书记，去年8月从另一个乡镇调整过来
的，也是目前我们县唯一的一个乡镇女副
书记。之前的乡镇是全县第二远的乡镇，
距离县城2小时车程，参加工作时在那里工
作了3年，任副职时又在那里工作了3年半，现在的乡镇距离县城1小时40分钟车
程。
问题一：我想考遴选，但现在为了乡镇明年
换届前的稳定，我们无法报名参加考试，
但眼看着我离35岁就只差两年多了。只有
我在乡镇目前的岗位保持不动，可能还有
一年考遴选的机会。但我不确定未来三年
内我的岗位不会变动，如果岗位发生变
动，我就完全没有考试的资格了。那如果
我考省州的事业编，从长远看是否可行?
问题二：现在这个乡镇情况比较复杂，到目
前还不太擅长处理这种特别复杂的情况。
新来的(去年12月)党委书记经常给我说
要我努力，以后还有大任，近期更是给我
说有一个机会，要我争取。我不确定这是
不是画饼?还有这个机会该不该争取，因
为据我所知，目前只有一个更远的乡镇镇
长位置空缺，明年可能有5个乡镇空出来。
我怕现在一争取就争取到这更远的乡镇去了。从内心来讲，我很怕在复杂的乡镇当
乡镇长，也不想离家超过1小时车程，因为
孩子还不到3岁，但母亲帮我带娃，身体又
不太好，想再请个人照顾，资金能力又有
限。所以觉得回县城也行，但短时间内不
知道回不回得了县城。
问题三：基于上面这些情况，你想问问老A
您还有没有其他建议，帮我破局。
老A回复：
很多同志没跑没送没人身依附，因为没钱跑
送，也不会人身依附，但是也在本辖区内稳
步的晋升，因为他们具备体制内人基本的素
养：工作能力强、工作态度好、能吃亏、能
忍让、虽不会巴结领导，但对领导绝对服从
作为等 寺 寺 。 这 种 状 念 ， 是 大 部 分 体 制 内 入 的 工
人状态。
所以，你也能10年熬到乡镇副书记的位置
上。下一步该怎么走呢?你心里其实并不清
楚，只觉得我们这个乡镇复杂，你多少有点
想逃离。真的复杂吗?一个距离县城两个小
时的小乡小镇，大概率不会太复杂的，比中
心城镇要省心省事得多!
所以，此刻，你得扪心自问：要不要走仕
途?如果走仕途，我觉得你遴选与否意义不
大，遴选上去当然有遴选上去的好，但就在
本地方发展，机遇也很不错!
我说一下我的看法：我若是你，我就不遴选
了。我会继续干好本职工作，平时多留心乡
长是怎么开展工作的，毕竟下一步大概率你
迟早都会去那个位置。
恕我直言，乡长那个位置我觉得比副乡长更
容易坐，当官不当副，是有一定道理的，因
为很多人压根就不听你的，很多副乡长压根
指挥不动乡镇的老中干。但是当了乡长后，
这个情况大概率不会存在。
所以，你不必畏难。乡长能力强，就多干
点，能力不强，书记自然就强势一点。反
正，在我看来，乡镇书记是要有点经验和水
平，但是乡长对这一块的要求比较少!
且，我若是你，我不会回县城，最起码拿到
级别后再回!我若是你，在上面也得找找关
系了，比如你作为副书记，你去给县里的副
书记汇报工作是天经地义
