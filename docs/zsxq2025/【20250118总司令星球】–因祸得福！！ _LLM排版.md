
# 因祸得福

沉浸在一种自我想象的悲情中是将一切事与愿违和求而不得合理化的最佳方式，无论是对个人还是对一个社会来说都是如此!委内瑞拉这个国家曾经也阔过，但沦落到今天的处境跟他自身的各种失误的关系非常大，兔子想拉他们一把，实在是烂的不行，根本带不动。

隔壁三仔家认为自己的苦难行军都是美帝国主义强加的，其实，苏联解体才是决定性因素，美帝国主义在这件事上的影响力反而无足轻重，因为三仔家从来就没有主动或被动纳入过美国的经济体系。

要说美国封锁和制裁，上世纪70年代也处于鹰酱的封锁和制裁之中，但当时他们的工农业生产水平和普通人的生活水平，不仅比兔子要好，也比韩国朴正熙创造汉江奇迹之前的南边要好，所以，鹰酱的制裁和封锁不是决定性因素，制度的选择也不是决定性因素，因为现在的制度和上世纪70年代的制度没有任何区别，甚至负责的那群人都没有换过，后来粮食为什么不够吃了呢?经济水平为什么直线下降了呢?

昔日三仔家他们在苏联廉价石油、粮食和工业援助的加持下，日子过得比当年的兔子要好得多，苏联援助三仔家他们搞成了机械化的农业生产，不管是工程机械，化肥，石油，甚至种子和口粮，三仔家他们都可以通过出口自己丰富的铜和煤炭来换取!

铜对于军工的需求非常大，任何工业大国都极度需要，兔子的子弹之所以是钢而不是铜，就是因为极度的缺铜，所以，现在的电器生产在大力的推铝电机，而不是采用铜电机。

当苏联轰然倒塌之后，三仔家他们不仅无法获得机械化农业所需的一切，甚至无法获所需的粮食缺口，随着1992年兔子与南边建交 ，北边与兔子的贸易也锐减，兔子也是煤炭多缺油，即便没有建交的原因，也没有办法为他们提供便宜的石油天然气，同时也不需要他们的煤炭，当时兔子裁军百万，全力发展经济，军工生产的需求也大幅降低，所以，他们就只能苦难了!

苏联不仅是个阵营的主导者，更是一套经济体系的构建者和维护者，当这套互补式的经济体系崩塌之时，那些严重依赖这套经济体系中的分工协作过上稳定生活的国家，在经济上瞬间就回到了脱钩断链的短缺状态，不是一个环节短缺，而是所有环节都乱套了!需要自己从零开始重构架构一个新的社会经济体系来保障自己原有的生活水平，一个生态的坍塌，到另一个生态长出来，就算能够成功，也需要很久的时间和各种天时地利人和的机遇，因为生存竞争的排他性，决定了那些潜在的竞争对手不让你正常的生长出来。

需要用灵活的博弈手段和高超的战略定力，以及务实的规划和执行能力，来完成这种生态的重构和漫长的生长，绝大多数社会没有这样的能力，能够从一个寄生体系切换到另一个寄生体系，就已经属于非常成功的壮举了!不管怎么样，朝鲜金氏政权总算挺过来了，叙利亚阿萨德自从苏联解体以后，试图切换到美国主导的经济体系没有成功，最后在反反复复的折腾中消耗殆尽…

兔子之所以在苏联轰然倒塌之时并没有受到太大影响，这要感谢二十世纪60年代兔子被美苏两大经济体系同时排除在外，兔子当时并没有为自己的处境编织出一种悲情，反而是以一种前所未有的豪情，从零开始自己构建了一个属于自己的完整体系，这个体系虽然存在低效和局部缺陷的问题，但它是一个能独立运行起来的体系，就如同一个最简陋的操作系统，然后，在改开的过程中，不断改进升级操作系统，并有选择性的引入并重构了一些应用程序，从而逐渐形成了属于自己的操作系统和应用生态，甚至有部分应用程序在体验和效率方面开始反超。

俄罗斯不是苏联，苏联曾经是个操作系统并提供各种系统服务，其他国家就如同应用程序，通过苏联提供的系统服务的相关接口，利用整个操作系统的资源和调度控制机制，来保证自己的应用程序能够向最终的用户提供各种功能体，当苏联这个操作系统崩溃之后，那些系统服务和调度机制就没有人提供了，依赖这些系统服务才能正常运行的应用程序就没办法继续向用户提供相应的功能。这些应用程序要能继续提供相应功能， 只有两个出路：1、自己在底层构建一个操作系统，并开发相应的系统服务。2、用另一个现成的操作系统所提供的系统服务和调度机制，然后把自己内部的调用接口全部改成它的。

许多国家没能力构建自己的操作系统，就只能改弦更张改接口，苏联这个系统崩溃之后，许多应用程序都没办法继续运行下去，只能改成鹰酱这个操作系统所提供的接口，才能继续为用户提供相关的功能和服务。兔子真的是一个例外，二十世纪50年代因为抗美援朝，被鹰酱体系排除在外，二十世纪60年代被苏联体系排除在外， 两方都没办法深度嵌入，只能自己从零构建起一个完整的体系，也算因祸得福。

佛家说：登临彼岸；基督说：进窄门；儒家说：虽千万人，吾俱往矣!合起来就成了存续的指引原则：要想登临彼岸，就必须进窄门，并坚持走那条更难的路!
