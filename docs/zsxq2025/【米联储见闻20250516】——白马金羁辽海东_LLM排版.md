
# 米联储见闻20250516——白马金羁辽海东

中美关税暂停，为期90天，外贸系统传来消息。订单激增，运费激增，川总真的是凭一己之力在刺激经济。通过这种反复无常的政策操作，让消费不为需求服务，而是服务于不确定性和风险。试想如果市场很稳定，消费必然理性，始终以需求为主导。而现在，经济矛盾恰恰是需求不足。如果一直都物质充裕，物价走低，消费必将进一步萎缩，则经济还要逐步走低。而按照川总这个玩法，把不确定性推到极致，让你不得不超预期的备货采购，甚至盲目跟风消费。90天的缓冲期，会让美国进口商买下远超过去计划的货品。加上人为造成运力紧张，运费增加，让每一个环节的成本都上升，最后让消费者多花钱买单，真是绝了。

中美暂停互加关税后，从兔子运至美目的货柜订单急增，每个货柜运费最高涨价达11500美元。5月5日从兔子到美国货柜订单的7天滚动平均值为5709个标准箱，但从5月11日开始，该数字已升至2.15万个标准箱，增幅达277%。美线5月底发货的船舱位已接近满舱。同时，美森、长荣COSCO等船公司也发布涨价通知，每个货柜的运价上涨500至1500美元不等。以上海港至纽约航线为例，目前40英尺长的货柜报价在4300美元左右。政策调整之下，企业都是羊群任人宰割，且已成常态，大家都默默接受了这种特殊状态，看天吃饭，默默忍受。现在就看川总的卖货艺术了。这一轮先跟兔子斗个法，然后服个软，然后就收获了订单保障。下一次90天后，不知递老金毛又会玩出什么花样了。我现在不但不担心，甚至有些期待，毕竟他所有一切的目的都是搞钱，而在现在这个情况下只要他搞钱，兔子就肯定能吃上一口，因为搞钱离不开产能，而兔子就是全球最大的产能来源地。只要川普的终极目的是搞钱，则不管他怎么折腾，兔子都是最终受益者。哪怕搞什么产能转移，最后还是兔子公司挂个洋名字换个马甲进入美国当特洛伊木马。

对于川普爱搞钱这个事，这次中东之旅真是看的很真切。纽约时报几乎是哭丧的口吻在报道，川总爱钱这件事让美国失去了用价值观领导全球的地位。毕竟这个世界最爱钱的国家和领导人，通常很难得到多的尊重。尤其是川普拿到中东大订单后，在沙特的黄金豪华宴会厅的讲台上宣布，美国将不再进行国家构建和干预，这个超级大国不再对你们的生活方式指手画脚。时台下观众爆发出热烈掌声，阿拉伯人买美回飞机送给美国总统，同时让美国人放弃价值输出，不再指手画脚。在民主党人和自由派看来，川普领导的美国跟乞丐有什么不同，拿了钱就闭嘴，大哥是这么当的吗？虽然全世界都很烦美国自以为是，指手画脚，但这不恰恰是美国彰显领导力和影响力的主要方式吗？如果美国变得像川总一样深谙人情世故，尊重他人文明和社会形态，只拿钱不管闲事，那美国跟看门狗有什么区别呢？一直以来，都是美国拿钱去砸别人，让别人必须听话跟美国走呀，如今变成美国拿钱然后闭嘴，民主党人只能暗自哭泣了。

川总在利雅得的演讲敦促中东地区人民以自己的方式决定自己的命运。沙特学者从川普嘴里听到这样的言论，仿佛看到了一位秉持马克思主义思想的政治家。川总甚至在此行会见了叙利亚新任领导人沙拉。川普与沙拉及沙特王储的合影震动了中东乃至世界。从会面金正恩到这次会见沙拉，川总让世界看到了一个对美国外交立场没有什么执念和坚持的美国总统。相比之下，拜登当面给沙特王储上眼药，川普不但刻意回避了这些负面信息，甚至盛赞沙特历史悠久，文明有序，王储了不起，让有钱能使鬼推磨这件事表现得淋漓尽致。很多美国总统不可能做的事情，在川总这里都处理的相当丝滑自然，没什么大不了。对美国的传统外交理念，川总还做出了深刻反思："近年来，太多美国总统深陷一种观念，认为我们的职责是审视外国领导人的灵魂，并利用美国的政策伸张正义，惩戒他们的罪孽。那些因为政治避难前往美国的沙特人，听完川普的言论瑟瑟发抖，毕竟，一旦失去了美国政府庇佑，他们可能随时会被清算。不过也有美国学者很诚实，川普跟其他美国总统没什么不同，他只是坦诚地说出了历任美国总统都真正关心的问题——美国利益，而且不再用人权民主作为筹码去要挟为难对方，对其他国家来说，川总不但务实，甚至更真诚可靠。

也盼望美国能继续推动这进程，尤其是该国与沙特关系正常化。不过，直到川总离开沙特转赴卡塔尔，该协议被提起的次数也寥寥可数。他甚至一度称两回关系正常化是他的梦想，但沙特会在自己的时间表内做到这一点，似乎暗示他无意大力推动以沙关系正常化。而相比于沙以关系，川普却在积极推进美国与叙利亚的关系，不去以色列看内塔，反而见了叙利亚的沙拉，并解除制裁。这让以色列真的有点坐不住了。毕竟内塔上月赴美时曾亲自向川总请求不要解除对叙利亚的制裁。然而，川总不只没听取其建议，反而先公开宣布解除对叙制裁，再会见该国过渡总统，被以色列普遍视为极端伊斯兰主义者的沙拉，希望两国关系正常化。这下彻底让以色列各界惜逼了。

毕竟在第一任期，川普是把耶路撒冷划给以色列的人，被认为是以色列的坚定支持者，他在上任前已数度高调发表亲以的言论，当选后也表现出对以的强力支持，包括要求美国停止参与及提供资金于经常与以色列作对的联合国人权理事会及联合国近东巴勒斯坦难民救济和工程处。可是，当内塔4月第二度访美后，情势却发生显著变化。先是以色列希望美国降低对以色列17％关税的主任务失败，随之而来的就是川总对以色列的情感转淡。川普4月底与内塔尼亚胡通话后，依然在社交媒体上宣称2人在所有问题上都站在同一立场，但事实显然非如此。毕竟，现在的川普只看实际利益，拿钱办事，对于这个德国后裔来说，犹太人的价值仅在于能不能有助于他参选、当选，以及提供足够的好处，在情感上没有任何依恋，必要时重拾二战期间德国人对犹太人的态度也不是不可以。

过去民主党跟中东搞价值观外交，要挟式外交，把以色列当成自己架在狙击枪上的一把钢刀，展现了一个超雄美国的姿态，以色列自然是狐假虎威。而如今，面对坐拥4万美元主权基金的中东狗大户们，每个都是川总的金主爸爸，要让对方听话固然是要架钢刀，但是要想让客户花钱就不是刀子可以解决的。这个时候，以色列的存在不但不能让美国获得好处，反而成了美国与中东狗大户苟合的障碍，这种情况下川总冷落一下以色列还是很好理解的。

以色列希望继续在中东称王称霸，美国只想让中东多多花钱。从这个角度看，现在的美国不但不希望中东保持混乱，反而希望中东好好发展，然后有更多的钱可以花在美国身上。这一下子就把美国的外交逻辑彻底扭转了。过去美国通过在中东制造痛点和混乱，左右油价，制造安全威胁，从而让美元可以为所欲为。而现在川总为了应付唯一的且非常强大的竞争对手兔子，不得不希望这些过去混乱的地方都尽快发展起来，参与到美国的建设当中来，以至于跟爹的立场和方向都出现了明确的分歧。还真是没有永远的朋友，只有永恒的利益关系。当然，川普的一己私见不足以彻底改变美以关系，但却让世界看到了美国的可塑性。对美国而言，没有什么是永垂不朽，只有金钱永不眠。
