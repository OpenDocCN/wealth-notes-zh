
# 4.23《新闻实验室问答》(1)

## 「这个时代」
Anonymous：方老师能聊聊对李嘉诚出售巴拿马运河两个港口给美国贝莱德财团事的看法吗？中国香港、大陆、美国，各方反应都很大。在这个政治神经极度敏感的时代，真的有任何行为可以脱离政治存在吗？
答：其实在任何时代，商业行为都无法脱离政治而存在。当商人决定在一个地方投资，他们多半会考虑当地的政治环境——可能因为稳定的政治局势和高度的法治保障而选择投资，也可能从腐败的政治当中看到牟利机会。正如徐贲老师曾经在一篇文章当中指出的，“商业”或“商务”（business）的最初意思是“公共事务”，而任何公共事务都不可避免是政治性的事务。
所以，“纯属商业行为”只是一种公司的公关话语而已。它的意思似乎是：我是商人，只管赚钱，不碰政治。可是，任何商业帝国的构建，从来就和政治息息相关——能否进入某些市场，能否在一些领域获得特许经营资格，能否获得补贴和政策优惠，能否进行收购并购，这些奠定商业帝国基础的关键因素，从来都是和政治互动之后得到的。成功的商人往往都不只胜在商业眼光上——或者说，“商业眼光”中的一个重要部分本身就是政治判断和身段。
当然，很多时候这些互动发生在幕后，所以在大众当中造成了一种“商业和政治无关”的假象。但这一次，幕后的互动没有做好（有论者认为，李嘉诚没有做好事先沟通，中国官方对出售消息的宣布感到措手不及），所以很多事情被摆到了幕前，官方宣传机器开动，引发诸多争议。让官商互动以这样的方式摊到台前，其实会让大家都尴尬，唯有民粹主义媒体和KOL从中得益。通过激烈的批判话语，政治力量似乎申明了自己的主导地位，但这也会损害招商引资的努力，并不利于经济信心的恢复。而商人的政治判断力和关系网络也会遭受怀疑。
总结来说，我的看法是：商业和政治之间从来都是不断互动而非互相脱离的，这次发生的事件是一次失败互动的结果。

## 「工作与学术」
Francis：方老师好！我毕业后即将在伦敦开始我的第一份正式工作，作为新人，想做的事很多，但因缺少经验而有些担忧，请问是否有或宏观或细节的建议？之前也有过半年的全职实习，但觉得毕竟是实习，和真实职场略有偏差。经常在听社交媒体上博主的执业经验分享，感觉每个人的观点都有正确之处。目前就在每天思考如何像海绵一样吸取能量，和不踩大坑。感谢！
答：首先恭喜你即将开始工作！因为不了解你的工作行业，所以很难给出具体的建议。你提到的社交媒体博主分享，让我想到之前曾经在《公司茶水间》播客录过一期相关节目，或可参考。
从你的提问来看，你应该是一个J人——希望有很充分的准备。这样当然很好，但是听人分享游泳的经验和 自己下水始终是两件完全不同的事情，所以我觉得，不如省下点看社交媒体博主分享的时间，赶紧去毕业旅行。另外，别人的建议始终难以完美地适合你的性格和特质。我在读博期间以及任教前两年，也听过不少建议，包括要选什么样的研究方向好出成果、要尽量少花时间在教课上以便多发论文等等。现在回头看，这些建议我基本都没采纳过——很简单，我不想因为听从这些“实用建议”，变成一个不是我自己的人。
还有一点就是，踩坑没什么不好，包括大坑，因为往往是不踩坑就难以真的学到东西。所有的经历都是财富，踩坑的经历更是。在踩坑和爬上来的过程中，你对自己也会有更多的了解。

## 「从工作中寻找意义」
Anonymous：方老师，请问您如何看待“从工作中寻找意义”这件事呢？人一定要做自己热爱且认同的工作吗？您应该对自己职业非常热爱，也发自内心认同工作的价值。我是新入职场的毕业生，在校期间也实习过几个不同的行业，但我好像始终无法从工作中找到任何热情和意义。周围很多人都劝我，不要在工作里找意义，这也是现在的主流论调。您认同这个观点吗？
答：我的确是很幸运的，从事过的两份工作（记者和大学老师）都是我热爱和认同的。但即便如此，这两份工作中也都有我不喜欢的部分，有时候我的热情也并非永远那样高涨。可能也是因为如此，所以我会“不务正业”地做一些其他事情，比如写这份newsletter。
我觉得无论如何，意义感是重要的。我希望这个世界上，所有人都能在工作中找到意义，但是很遗憾，这个世界很不完美，很多人的工作的确缺乏意义感，所以不得不妥协式地做一份没有意义感的工作，然后再去别的地方寻找意义。
所以，我的建议很简单：并不放弃在工作中寻找意义的期待，但同时积极在工作之外去寻找承载意义的地方。当然，这样做也需要你通过积极的尝试，去寻找到自己真正热爱的事情。

## 「写作入门与提升」
Anonymous：方老师好～我以后想成为一名英文记者，想问问您该如何入门英文报道写作呢？感谢！
答：我想不外乎几个方面：
阅读：平时大量阅读英文新闻，尤其是美联社和路透社的英文消息、《经济学人》的分析性文章；
找一本英文的新闻写作教科书看看，哪本都可以，因为基本原则都差不多，而教科书能给你的东西其实有限，因为只看书不练习是不可能入门的；
拥有一本《AP Stylebook》；
多多练习，最好有老师/编辑提供反馈，也可以尝试用AI工具提供反馈；
实习：寻找外媒实习机会，或者Sixth Tone这样的国内英文媒体。

Anonymous：方老师您好，之前我给您提问如何开始写作，您的建议是给自己、朋友写信。我实践了一段时间后觉得很有意思，日常的生活和想法落到笔头就变得不一样，写着写着，自己对很多事情的看法，不知不觉中都清楚了许多。请问您还有进一步的写作建议吗？还有普通人能够尝试的写作方法和渠道吗？
答：太好了！如果方便的话，你也可以更详细地分享自己写信的经历和体验，我很有兴趣了解更多（直接回复邮件就可以给我发信）。
如果说进一步的建议，那就是找一个有经验的编辑，给你提供反馈。绝大多数优秀的写作者都会告诉你，编辑的角色极其重要。当然，编辑是一种很传统媒体的职业，在这个自媒体年代，大部分内容都已经没有经过编辑这道门槛，这其实是一件遗憾的事情。
如果找不到编辑，可以用什么替代方案呢？一个是我在上个问题的回答中提到的AI，或许你可以写prompt“你是名经验丰富的编辑，你很在乎文章的逻辑和遣词造句，请你……”，看看效果如何。
另一个方案就是进入写作社群，互相为作品提供反馈建议。目前已经有不少这样的社群存在，比如Matters在做的“七日书”和三明治的“每日书”。
