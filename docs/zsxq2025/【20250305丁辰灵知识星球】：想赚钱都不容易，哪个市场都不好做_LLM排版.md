
# 想赚钱都不容易，哪个市场都不好做

灵哥：

总有人说类似这样的话，我是小白，我觉得港股好做，只要买那几个就可以了。或者还是美股好，买七姐妹就完了，买英伟达就完了，买特斯拉就完了，买QQQ 就完了。甚至以前还有人说我不看好币，你看币又涨了，灵哥你错了，A 股太难了。其实每个市场有每个市场的特点，觉得美股港股容易的，是没有遇到挨打的时候。人很多认知，其实不是自己的，大多数人懒于思考，懒于逆向思维，而且贪嗔痴慢疑一个不少。大多数人的思维跳不出他那个朋友圈的思维，这是大多数人无法突破的原因。几年前的时候，那是中概股还没有跌的时候，很多大厂员工死守着那些中概股，都觉得互联网纵横脾睨，压倒一切。如果你们还记得，大概七八年前流行降维打击这四个字。互联网人喜欢用三体的比喻，毁灭你，但与你无关。我那时候就说为什么买美股要买中概啊?大部分人是听不进去的，后来我在星球提醒了国家反垄断的风险，一共提醒了两次，还是不少人听不进去，一直到彻底埋了，星球也就不用续费了。

我们去年广州聚会，有个会员提问，他说觉得我对中国的科技信心，整个国家制度前途等他都认同，但是他去上某知名商学院，该商学院知名院长就是不看好中国，问我怎么看院长观点。我不知道那个院长到底看衰中国具体是什么，无法就观点回应，我就说他是不是真的不看好我不知道，但是他一定要说不看好。这是因为他代表的不是他一个人，他代表的是一个群体，一个阶级。金融圈都不看好中国，都觉得中国制度比美国也差，科技也差，前两年政策也差， blah blah，好多人都想润。人不会和自己的阶层作对，因为人活在他的社会关系中，所以这位知名的院长，他不能背叛他的阶层，他自己内心怎么想不重要，但在公开演讲他必然是这样的，所以他看衰中国对我决策并没有任何指标性意义。

前两天我不是和我券商首席的同学吃饭吗，我又想起来他当时说的一个细节。他说在最看衰的圈子，也就是债券圈，现在都开始乐观了，最有趣的是有人 say，本来觉得我们很差，现在特朗普上台以后一顿乱搞，觉得我们算很好的。你看加密货币圈子不就是如此，先是大骂特朗普，然后特朗普说要搞加密峰会，而大涨，立刻大赞特朗普，之后一天就割韭菜，币又跌了，又大骂。想做好投资是很难的，绝对没有哪个市场容易的说法。有时候就是运气，你恰好在某个上涨周期中，你就认为自己是股神了，这种想法非常危险。最终世界是公平的，你的认知和能力决定你的财富水平。今天的世界很复杂很动荡，能活的长的能笑到最后的才能说的上有能力。

擅长做交易的，美股即使这段时间在跌，但英伟达博两个反弹，做call 也能赚很多了。永远不要觉得哪个市场容易，想赚钱都不容易。还是回去想那五个字，贪嗔痴慢疑，自己若是亏钱了，不达预期多半的原因是这五个字。
