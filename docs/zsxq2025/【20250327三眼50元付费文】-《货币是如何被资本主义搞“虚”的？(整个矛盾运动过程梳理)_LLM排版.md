
# 货币是如何被资本主义搞“虚”的？(整个矛盾运动过程梳理)

货币是人类最伟大的发明。其实不应该叫发明，因为这属于本能的发现和发挥。争夺最喜爱的东西是本能，货币作为大家最爱就让人的这一本能更加充分和激烈了，只有这样才能刺激人们的生产分工和经济增长，因为生产分工的本质就是为了获取货币！市面上一大堆钱在那流动着，才能让人们有“足够机会”去获取，人们为了“抓机会”自然就脑子放光搞创意去抢货币了，分工就越来越精细化了。这就是市场经济不同于计划经济的功能，其实二者都属于市场经济，没有市场那还能叫经济吗？只不过计划是缺货币的市场经济。因为没有足够的钱来给大家伙“抓机会”啊，那大家伙就没有脑子放光的动力了呀，所以就要给计划驱动着分工前进。

当年秃头把能带走的值钱玩意都划拉走了，组织面临的情况是极度的货币短缺，这是没有办法的办法。就剩下一个值钱玩意带不走，土地！后来神龙靠此高速发展，这是后话了，先捋一下邪恶资本主义是如何把钱变虚的，又是如何产生货币霸权的，俄乌又为何是必然性？了解外部才能理解内部。

第一步矛盾运动：压榨和生产过剩。一开始矛盾圈圈还很小啊，主要是集中于资本主义内部小圈，就是其中一块小面积压榨另一块大面积的产生的矛盾，即资本家压榨工人阶级，不压榨怎么能“抓机会”呢？为啥大面积还能被小面积榨汁？因为小面积很鸡贼啊，就跟非洲草原上吃肉的占少数同理。这种小面积压榨大面积近乎于天性，就好比说是习性决定的，总之资本家压榨工人的劳动血汗了。

那搞榨汁得有个筐子把对象放进去了啊，就是产业资本了。其实资本家压根就不喜欢干这种事...因为资本家的最终目的就是为了获取更多货币，做人的目的性非常纯粹，搞产业这种活多累啊，抗事有风险还劳心，天天骂骂咧咧的。但一开始矛盾圈圈小这就没办法，必须得干这种“遭罪”的事了。

随着矛盾发展运动总有瓶颈期的时侯，即小圈圈里生产过剩了！因为那时候的资本家脑回路还是1.0版本比较直接，产业生产的目的是根本是利润，消费循环是啥能吃吗？利润能赚一天是一天，抢货币的机会可失不再来。榨汁vs浇水回流是天然相反的，所以往死压榨工人工资就会导致圆里的大面积逐渐提供不了消费需求了，制造鞋子的劳工买不起鞋子了。这时候矛盾就发展成供给vs需求的矛盾了，产生新的主要矛盾后就要转化解决啊，那就需要主动创造需求了。

第二步矛盾运动：殖民和反压迫。那么解决矛盾的主要方法就是把圈圈扩大，总不能小圈圈里让出面积吧？脑子突然放光了...虽然自己家内部需求不足，那不还有外部农场呢吗？矛盾出路就是资本+商品向外输出，此时脑回路2.0版还是比较直接啊！在这些原始发展过程中的货币叫做——实物货币。为啥叫这名？公式：用XX玩意把货币给搞多，就叫XX货币。在干产业的遭罪阶段，货币增值靠的是工人们的劳动价值凝结，靠的是扩大物质产品再生产规模来实现的，所以就叫实物货币。以前挖矿挖金子也一样，也得上山下山刨回去炼实物，里面都是带着劳动凝结的。

这时候资本主义邪路为了扩大需求抢市场，那是不是就要干架？一是和殖民地干架，二是资本主义之间抢地盘也干架。场景想象一下：干架是为了卖东西...跟菜市场抢地盘是不是很像？其实本来就是一回事。

干架打赢了的叫啥？一个叫菜霸，一个叫帝国主义嘛。名号一出看起来就不是一回事了，档次多多少少还是有所区别的，风险也不一样…原有的供需主要矛盾虽然得以缓和了，但是资本压榨工人这条线没变，随着矛盾运动发展又到了新的瓶颈转换期了。圈扩大了，矛盾范畴也随之扩大，激起无数个圆内大面积的联合反抗了！

这时候伟大的红色革命就来了啊，一波波的到了二战后，出现了十几个社会主义正义农场，主要矛盾圈圈达到最大：东西意识形态对立，即冷战！

第三步矛盾运动：造中产抢中产。矛盾都最大化了，下一步矛盾发展能咋办？单看资本主义这边，最大圈的主要矛盾谁也解决不了，那自然矛盾发展就要内化了啊，这下资本邪路农场的小动物要得好了...因为东西对抗导致资本主义建模大法受到威胁，威胁就会产生妥协，资本主义就需要对自身制度进行改造了。咋改造？为了缓和外部带动出内部的阶级矛盾，就要提高工人工资，工人玩不过资本家是因为性格上要求不多，吃草的能稳稳当当的无风险的生活即可，那就再加个福利保障制度吧，保证吃草子子孙孙无穷匮也，朝三暮四玩法加枣就变成了朝九晚五。

随着工资提高和社会转移支付就产生了中产阶级。资本主义中产阶级兴起后，在其内部就逐渐形成了纺锤体的社会结构，资本主义内部的需求就变大了。供需矛盾+阶级矛盾=逐渐缓和！当矛盾缓和腾出对抗空间后，就要为了下一次激烈对抗而奋力发展了..当然这得需要发展过程啊，这个过程填补就是二战后发达资本主义的高速发展。

原来2.0干架抢地盘时候，邪路农场对外的资本流向是从自家到殖民地，主要是干一些初级产业的商品，就是从殖民地掠夺自然资源回来，再给加工成初级工业品卖回去，制成品贸易也就是三成多说了。等到了二战后七十年代的时候，邪路农场家家 中产阶级肥力够用了呀，自身需求扩张就带来了互相的制成品贸易了，资本带着产业互相投资，大家走的都是一个队伍嘛，因为外面的主要矛盾还在啊。四小龙就是这时候飞速工业化的，属于跟着邪路农场搞跨太平洋分工了。总之西风队伍互相投资能占到全地球八成，资本农场就在邪路前面加了发达二字。

但发达归发达，矛盾发展运动是要继续的…因为此时资本家获取货币增值还是靠实物生产，发达邪路的内部产品需求逐渐又要饱和了，那赚利润的遭罪事就遭罪+1了，利润空间越来越小矛盾就来了，产业竞争不断加剧。过程中三德子和小本子这都起来了，但是甭管竞争如何加剧都得憋着，因为邪恶与正义的主要矛盾使得手工活再好也得簇拥在邪路大哥的保护下啊，你还敢真抢大哥生意？所以虽然竞争剧烈，但是整体上还是有个稳定规则的。

但是呢，突然苏大路线解散了…最大圈圈的主要矛盾解除后，次要矛盾就要登上台面了，发达邪路内部剧烈的竞争压力锅就要释放了，新的矛盾运动产生！

第四步矛盾运动：产业转移。那压力锅咋转化矛盾？次级圈肯定也不够啊，所以还是要在最大圈圈来转化，把另外半圈给画进来。这时候的矛盾就是画新圈的矛盾，咋画圈呢？原来欧最怕谁？最怕苏大啊，当然现在怕大毛子，总之武力上一直惧怕，这群老掉牙农场同而不和，力量始终捏不到一起，始皇笑而不语，全靠同行衬托。那老掉牙农场们是不是要捏一下才行？在苏大解散后的第二年，那边就开始捏欧梦了，寻求经济与政治的统一。除了大毛子骂骂咧咧的外，大漂亮也骂骂咧咧的啊..因为原有大圈的主要矛盾解散后，致使欧梦对大漂亮的军事保护需求消失了，邪路大哥的作用就变弱了，既然不是大哥了那么经济上就要重新算账了。这帮老掉牙农场脑子一放光，在不到10年间捏了十几个农场，而后一发不可收拾还要向东继续扩张，一旦捏成的话就是地球第一大经济体了，不仅对大毛子是个威胁，对大漂亮也是威胁！那大家伙都开始画新圈圈吧，地球上各种协定啊、贸易区啊就开始站队了。在此过程中跨国公司这股浪潮就开始了，所谓的经济全球化就开始了...

总之画圈圈很乱，即要纷纷捏造型vs又要经济全球化，看似矛盾其实并不矛盾！因为表面看起来好像是一通乱画圈分分合合的路线，实则按照资本邪恶性格来说还是一条主路线没变，就是资本追求垄断底色没变嘛，所有的矛盾发展运动都是因为抢货币多多这件事嘛。

在这场混乱的画圈中，东方就开始崛起了！因为发达农场产业竞争的矛盾激化与利润再扩大化需求，就要推生出新的资本运动啊。此时货币还是实物劳动凝结对吧？但是发达农场中产阶级的工资水平太高了，凝结的成本和效率这么一比，发展中国家的工人红利就凸显出来了，那要降低成本肯定琢磨着向发展中转移产业啊，不是什么善心啊。九十年代重头再来的大潮背后，要培养新工人和外部搞对接啊，不然人家咋进来一起画圈发展？怎么通过市场经济接收产业转移？当然这中间还要培养新工人素质、基础设施建设等等一系列铺硬件的过程。

总之到了02年的时候，神龙已经超越各个农场，成为全球第一大资本流向国…同时有条暗线的矛盾演化正在进行，就是发展中农场接收产业后，抢了原有发达农场工人集体的活，只不过这条矛盾线由于产业分工的水平落差太大还未能激化呈现。

第五步矛盾：经济体虚拟化。这个运动过程既有阴阳运动，又有矛盾运动，阴阳是对拉运动，矛盾是对抗运动。啥叫对拉运动呢？就是原来地球最大圈的主要矛盾解除后，两个半圈由于资本与劳动的天然对接需求，原有半圈之间的矛盾变成阴阳对拉了...

这个对拉场景就叫全球化，其实前面应该加上资本二字，因为是资本为主导的，总不能说劳动全球化吧？在混乱画新圈+不断合圈背后，是资本输出带动产业输出从资本主义之间转变到集中在发展中农场了。整个过程其实等于是用新的阴阳互合在消解原来冷战的主要矛盾..所以呢，其实在第四步里资本主义间“重新算账”的矛盾的本质并没变，只是因为后加入的半圈小伙伴，导致矛盾形式变化了！

变化成啥形式呢？就是产业资本的矛盾转变成了虚拟资本的矛盾。在上世纪70年代的时候，发达农场间的布雷顿体系就解套了，即发达经济体的货币挂钩绿纸，绿纸挂钩黄金这件事解套了。因为货币量需求变大了呀，与黄金挂钩就满足不了在争夺货币上的“足够机会”了，这个体系一解体后，货币量的增长就摆脱了实物生产的瓶颈束缚了。

但此时叫信用货币，和虚拟有啥区别？信用是指怎么造出货币而言，虚拟是指与实物劳动的组合而言。所以虚拟这个概念的本质并不是讲货币，而是讲组合关系，所以严格讲不能叫虚拟货币，应该叫虚拟经济，即虚拟经济体和实物经济体的共生关系。

先看神龙这边崛起发展，上面说计划经济发展慢并非刻意，而是被逼无奈因为缺货币量对吧？所以神龙的需求不仅是产业转移，还要进口货币。所以在苏大解散之后，94开始了挂钩绿纸模式，红纸汇率和外汇并轨，这样生产要素价格才能进行比较来凸显优势啊。那此时神龙不断进口的货币应该叫啥货币？信用货币？错...

回到上文逻辑，用啥换出来货币就叫啥货币，甭管人家货币是否靠信用造出来的，只要你拿劳动换那就是实物货币！所以进口的绿纸就当成纯金条即可，虽然人家内部已经脱离金条了，当然可供进口的金条不止这一家啊。随着资本与产业的转移，发达农场造信用货币即可换回外部产业的劳动制成品了，虽然一开始逆差的主要部分还是回销，但是随着转移发展回销的比重越来越小，这对于发达农场债券股票地产等资产信用的供应量自然就要求变大了。

随着发达农场的工业化完成，资本主义之间的竞争从物质生产逐渐转到非物质生产，产业资本家们都在向金融资本家转变，2.0向3.0升级了，发达资本农场进入到虚拟价值的争夺。矛盾依然是生产过剩导致的互殴矛盾，只不过生产过剩从物质生产这一维度转变到虚拟资产的生产上了，新的矛盾就是虚拟资产对劳动制造“采阴补阳”的争夺，劳动在哪？主要在东亚这块新增面积当中，东亚供应商商品，发达农场们供应信用货币。

为了获得“被使用”的需求，就开始了货币霸权的争夺，即谁能获得霸权谁就能拥有换取东亚劳动的优先权，这就要看谁够“硬”才能叫硬通货。制霸的当然是原来邪路大哥啊，根货币的起点嘛，自带地主身份的优先权。啥叫大地主？一点活不干那才叫大地主...大地主养成模式就是要不干活还能过好日子才行！

随着大漂亮那边产业转移过来，其生产制造就减少了，这种转移就导致发达资本主义从最开始的生产过剩型经济，逐渐变成了生产不足型，对于炒作信用资产即可换实物这件事就依賴上了。站在虚拟经济体角度讲，就是左手买你货右手造钱给你，此时货币虽然还保留着提供交换媒介这一作用，但增加了直接交换这一功能，这种快乐的松弛感就导致绿纸越来越虚，最后是纯虚了。

啥叫纯虚？就是如此虚实交换，大漂亮越来越生产不足，只剩下虚拟资本与市场消费，而各农场通过劳动所进口的绿纸，理论上已经无法从大漂亮那换回等值的物质产品了，实际也换不回来价值了...绿纸逐渐虚化到除了交易功能外已经没有实际信用了，或者换个角度去看，绿纸的信用就逐渐变成只剩下“被使用”本身了，一旦大家伙都不使用它了，那么它立马就崩溃了，这游戏只能中午玩了。

这种虚拟化过日子就好比啥呢？我在小区的小卖店里买东西，小卖店收到我印的钱后，在当天立马就又借给我了，因为他想吃利息多攒点钱，我给他打个欠条就行了，然后我拿着回流的钱再买他东西，就这样反反复复的倒腾，我们俩都很高兴..

小卖店为啥高兴呢？因为小区里都在用我印的钱，因为我够硬所以我的钱就叫“硬”通货，主打一个赞欠条。那么为了维持被流通的“硬”度本身，就要动不动的就把航母拉出来搞事，不是打压其盟友就是搜刮小弟，这就是霸权的表现。但绿纸的“硬”只维系在军事上，而在经济实力上来说，早在上世纪欧梦捏出的造型就已经超过大漂亮的经济实力了。

所以大漂亮的军事与经济其实是很不平衡的，这也是冷战解除矛盾后欧洲老掉牙们敢于重新算账的原因，也是大漂亮反反复复在欧梦制造地缘危机的原因。这是矛盾运动发展的新阶段，不同虚拟经济体之间必然围绕货币霸权展开攻守转化，俄乌搞事就是这种矛盾的必然体现。
