
# 米联储见闻250829——三军论事多引纳（小金来看阅兵，日本军工猛进）

## 你好，我是米糕

把小金和普京同时请来参加阅兵，果然是插了西方国家尤其是美国的肺管子了，今天美国各大媒体头条纷纷被小金获邀参加中国大阅兵所震惊。中俄朝现在都与美国及大部分西方国家不咬弦，美国媒体认为三国因各有所需同时高调同台亮相，示威的意思很明确，两个阵营的即视感愈发强烈了。上次中国举行九三大阅兵，朝鲜是派劳动党中央政治局委员、党中央书记崔龙海出席。这次小金罕见亮相创下多个第一次的记录，这些第一次意味着历史确实推进到前所未有的阶段。这将是2019年以来，金正恩首次访中国；首次出席大型多边外交活动；也是朝鲜最高领导人66年来第一次参加中国阅兵。上一次朝鲜领导人参加类似活动还是1959年时任朝鲜最高领导人金日成出席了中国国庆节阅兵式。按外交部的说法：中朝两国人民相互支持，共同抗击日本侵略，为世界反法西斯战争和人类正义事业胜利作出了重要贡献，维护好、巩固好、发展好中朝关系是中国党和政府的坚定立场；中国愿同朝鲜在促进地区和平稳定、维护国际公平正义的事业中密切协作。这也算是官宣了一把朝鲜在半岛和东亚秩序中的战略地位。

外媒也注意到，除塞尔维亚总统和斯洛伐克总理外，其他西方阵营和欧盟国家现任领导人均没有出席阅兵，而被西方视为邪恶轴心之一的伊朗，这次由总统佩泽希齐扬赴会，10年前，伊朗没有出现在中国九三阅兵的外国嘉宾政要名单中。除此之外，土库曼、阿塞拜疆和亚美尼亚也是新客人，按外交部的讲法是：全球南方群体性掘起，正从根本 上改变世界版图。东盟除泰国和菲律宾外，其他成员国均有代表入列贵宾名单，新加坡将由副总理颜金勇出席。微信pep458印尼总统普拉博沃和马来西亚首相安华将到场，而印马10年前仅派特使、部长赴会，这次均升级了参会规格。外媒认为，中国正在利用川普实施战略收缩的机遇，利用上合、金砖等多边组织发挥影响力，逐步填补美国留下的领导真空。

中朝关系一直是美国观察中国战略路径的窗口，朝鲜是最早跟中国建交的国家之一，1949年10月6日，朝鲜就跟中国建交了，之后朝鲜战争爆发，抗美援朝这场战争被形容歃血为盟的战争。1960至1980年代，中苏关系恶化，朝鲜在中苏之间来回摆动，两国关系至1960年代尾极度紧张，甚至发生边境冲突。1970年代，中美关系有所缓和，中方主动与朝修复关系。冷战结束后，苏联解体，中国成为朝鲜最重要的经济援助国和贸易伙伴。中韩1992年建交后，中朝关系再次转入冷淡期，直至2001年9月江访朝后，中朝关系才算作恢复正常。至2010年代，中朝关系又出现变化。小金2011年上台、领导2012年上任，最初几年，由于一些矛盾，是关系较冷淡的时期。两国元首未曾互访，甚至未通电话，中朝关系因朝鲜频繁的核试和导弹试射中陷入危机。2016年3月，中国一改以往对朝的兄弟态度，支持联合国安理会通过20年来最严厉的对朝鲜制裁措施，包括限制石油进口、禁止朝鲜纺织品出口、遣返朝鲜劳工等，对朝鲜经济造成了巨大压力，一度中朝双方还开始隔空打舆论论战，刊登批评对方的文章，两国关系陷入实质性低谷。直至2018年3月小金访华，两国关系才迎来回暖迹象，而2018年正是中美关系加剧对立的时刻，此消彼长，中朝关系转好背后是中美关系恶化，朝鲜对中国的战略意义不言而喻。之后，中朝关系从极度低谷转向极度蜜月期，2018年3月到2019年1月之间，短短10个月，金访问中国足足4次，这在朝鲜金氏三代政权前所未有 的。2024年更被两国领导人共同宣布为中朝友好年，双边关系再次巩固。自俄乌战争爆发以来，金与普京多次会晤，2024年更签署《全面战略伙伴关系条约》，外界猜想中朝关系发生微妙变化，从客观角度来看，站在中朝俄各自立场，这种变化是自然而然发生的，也是必然，不足为奇，决定问题的关键是大家确实也是站在各自地缘战略的考量形成合作关系，所以中方此次邀请小金出席，就是要打破外界认为中俄朝三方存在博弈，中俄各自对朝有小九九的印象，小金选择在中方的重大主场活动展示自己的国际外交首秀，这本身也是中朝关系紧密的一种体现，实际上，今年俄罗斯二战胜利纪念阅兵，领导去了莫斯科，彼时俄乌冲突胜负未定，小金也没有出席莫斯科的公开活动，而这次普京和小金选择在中国主场同时亮相，也宣告一件事情，这一轮阵营对立的核心主导国家还是中国。

2015年北京举办抗战胜利70周年大阅兵，观礼的外国领导人有30人，这里边既有总统、总理、也有一些议会的议长或者副总理，算起来统称为外国领导人共30人。而此次单就总统总理级别的领导人就有26个，国会议长和副总理级别的领导人一共有6个，派出特使和高级代表的国家有6个。从名单看，中国的影响力在中亚、南亚、东南亚、东北亚达到了高水平，中国已经站稳周边，是名副其实的亚洲大国。北方邻国俄罗斯总统普京、蒙古总统在2015年和2025年两次出席阅兵式，一直是捧场中国活动的稳定存在。十年前，派出总统出席的中亚国家有4个，分别是哈萨克斯坦、乌兹别克斯坦、吉尔吉斯、塔吉克斯坦。而此次除上述四国总统，土库曼斯坦总统也前往了北京，至此中亚五国领导人集体捧场。土库曼斯坦政治上中立，并不是上海合作组织成员国，此次愿意出席中国阅兵，是对中国重视和礼遇。2015年韩国时任总统朴槿惠前往出席，朝鲜派出了劳动党政治局常任委员崔龙海出席。此次韩国总统李在明没有出席，但向中国派出了总统特使团，并且第二号人物、韩国国会议长会禹元植出席。朝鲜则是小金将自己 的多个第一次留给中国。2015年，东盟十国除老挝、柬埔寨、越南、缅甸派出总统或者总理级别的人物前往北京，泰国派出泰国副总理兼国防部长巴威前往北京，其他国家并未有高级政要前往北京。此次越南、老挝、柬埔寨、缅甸一如既往高规格参加，印尼总统、马来西亚总理、新加坡副总理、汶莱皇家武装部队司令哈扎伊米均是第一次参加。东南亚国家中，除明目张胆在南海挑衅中国的菲律宾，总理被暂停职务的泰国，其他国家均派领导人或者高层政要前往北京观礼。2015年南亚国家只有中国的铁杆盟友巴基斯坦总统侯赛因。此次南亚国家中巴基斯坦前往北京观礼的是实权总理夏巴兹，尼泊尔总理奥利、马尔代夫总统穆伊兹、孟加拉国临时政府代表国家安全顾问拉赫曼均是第一次参加。中国在南亚的影响力明显在逐步提升。中印两国关系虽然没有完全恢复，但莫迪在阅兵前夕参加上合，实际也算是擦边朝会了。把朋友搞得多多的，家里办大事能来什么人，也是领导执政十年外交成绩的直接体现。

东方高度聚合的同时，西方集体缺席也是此次阅兵的特点之一，出席的大多数是亚洲、非洲、拉美、以及独联体成员国，没有西方大国的领导人。2015年出席阅兵式的欧洲国家领导人有波黑主席团轮值主席乔维奇、捷克总统泽曼、波兰众议长基达瓦-布翁斯卡、塞尔维亚总统尼科利奇。此次来自欧洲的领导人除了塞尔维亚总统武契奇，斯洛伐克总理菲佐，其他国家政要的名字并没有见诸媒体。中国官方只是提到了欧洲议会议员多斯塔尔，以及来自俄罗斯、美国、英国、法国、加拿大等14个国家的50位友人或其遗属代表将应邀出席。十年前，美日批评中国耀武扬威，欧洲国家虽然没有派出高层政要，但有不少官方人士出席。比如法国、意大利等西方国家虽未派最高领导人，但通过外长或部长级官员参与，体现对活动的支持。美国仅由驻华使节作为政府代表参与，日本也未派领导人出席。此次阅兵，曾经炒出邀请美国参加阅兵的消息，川普也没有公开批评中国，但因为欧洲与俄罗斯的关系，欧洲大多数国家自觉和中国保持了一定距离，欧洲对中国的态度相较于十年前明显对立很多。此外，伊朗总统首次出席，非洲刚果、津巴布韦，以及拉美的古巴，都是中国传统的反西方阵营的老朋友。这些国家的领导人集体前往北京观礼，是一种地缘政治的清晰信号，是明目张胆来战队，不是一团和气凑热闹。

这次阅兵看似中国自家的事情，实际上是勾勒出一个与美国主导的西方世界平行空间，以中俄为主要支撑的世界，伊朗和朝鲜的加入，正式确立了西方国家主导的世界秩序和非西方国家将要形成的国际秩序之间泾渭分明的队形。

对于这种队形的形成，美日是相当警惕的，尤其是日本，毕竟这个阵营中大多数在二战中与日本有血仇，这也是为什么此次阅兵，日本如此紧张敏感，先是用大型无人机海上卫士(MQ-9B) 在钓鱼岛附近对中国海警船只进行警戒监视，又通过外交渠道呼吁欧洲和亚洲国家审慎考虑是否参加在北京举行的九三活动。日本此举真是无异于此地无银三百两，但是要明白，日本敢于公开叫板阅兵活动，背后必然是美国撑腰，美国驻华大使馆在815日本投降日发了一个联合声明，作为二战战胜国领头羊的美国居然带着日本公然宣称美日在二战中共同维护太平洋的和平，想要重新改写二战史的想法昭然若揭，而作为被美国在二战中摁着摩擦的日本终于看到能够掀掉压在乌龟壳上二战战败国标签的机会。日本的军国野心从未根除，1967年，日本迫于国内外压力，宣布实施三项武器出口禁令。但在日美同盟框架下，日本通过军火采购、特许生产、联合开发等方式，从美国源源不断地获取先进军事技术和武器装备。多年苦心经营，日本防务工业持续发展壮大，被美智库认为凭借尖端技术与雄厚基础，跻身世界领先行列。波斯湾战争中，美军巡航导弹使用日制芯片，打击精度显著提升。美军隐身战机的引擎耐热材料、无人机的高解像度光电侦察设备、加工精密零部件的数控机床，也都来自日本企业，近年来，日本又不断加强在航空航天、导弹防御、网络与电子战、无人系统等方面的科研投入，自主设计或与盟友联合研发六代机、电磁炮、高超音速导弹等尖端武器。2011年，野田内阁首次放宽武器出口限制。 2014年，时任日本首相安倍晋三出台三项技术转让原则，2023年，日本再进一步，允许武器装备以特许生产的方式转让出口。去年11月，印度和日本签署协议，共同开发用于海军舰艇的综合通信桅杆。日方还向菲律宾交付一批军用直升机备件和海岸监视雷达，并答应赠送6艘护航舰。2025年8月 5日，澳洲宣布斥资100亿澳元，从日本购买11艘最上级护卫舰。日方还加紧向印尼推销该型战舰，预计合同金额达20亿美元。2023年底，日本同意对美提供爱国者防空系统。是的，你没听错，日本向美提供爱国者防空系统。

不知不觉中，日本早已是军工巨头，并且赚得盆满钵满，当我们嘲笑美国的军工企业不给力的时候，别忘了，有一个比东方大国更高效的日本军鬼悄悄隐身于美国身后，正在飞速提升自身的军事能力，当大家都在想美国如何在台海冲突中制衡中国时，实际上，中国拿下台湾最大的阻碍不是远在太平洋那头的美国，而是就在身边的日本，相比于美国，中日与台湾成犄角之势，中国有大陆，日本也有大陆，一旦日本快速完成核弹装备，实乃中国的心腹大患。中国军方发言人张晓刚在发布会上说，魔鬼被消灭了，但遗毒还未根除。中国一定要将日本牢牢钉死在战争罪人的耻辱柱上，这不但需要坚决的立场，还需要在军事上能够彻底摁住日本的能力。
