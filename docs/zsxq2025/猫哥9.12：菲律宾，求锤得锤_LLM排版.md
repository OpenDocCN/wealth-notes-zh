
# 猫哥9.12：菲律宾，求锤得锤

在9月4日的上期文章“9.3以后，我们将如何收拾菲律宾”中，提到接下来收拾菲猴的几种可能性，包括继续围困坐滩船、拖拆那搜破船或者启动经贸制裁。现在看来，格局还不够高，用大话西游紫霞仙子的话说就是“猜中了前头，可猜不着这结局”。

这里再顺便提一下，上述文章在发布后2天，有星友留言反馈，菲律宾对坐滩船已经完成了补给和换人。这个消息我查了一下，确实看到了有个别大V在转。但蹊跷的是，到目前为止我们官媒包括菲律宾官媒都没有谈这件事，甚至连视频或照片都没有，按说以菲律宾的尿性，一定会把“赢学”用到极致。那就只剩2个可能：

第一种，发布消息的博主要么吃了海外垃圾信息，没有过滤鉴别真伪就发了，甚至不排除别有用心。第二种，菲律宾接受我们检查，我们根据前面的约定同意恢复补给线，但这件事被双方冷处理下来。至于真相到底是什么？个人认为第一种可能更高，就算是第二种，那也没啥，毕竟菲猴只要乖乖听话，我们还是可以喂根香蕉续命。只不过现在讨论这个已经没有多大的意义了，因为我们放了一个大招出来，只是方向不在仁爱礁，而是黄岩岛。

9月10号，国务院突然发布消息，同意自然资源部关于新建黄岩岛国家级自然保护区的批复。

## 具体情况如下：
黄岩岛，东西长约15公里，南北宽约15公里，周长约55公里，大致呈等腰三角形，其总面积约150平方公里，包括礁盘和鴻湖。根据公布的自然保护区基本情况看，包括核心区12.4255平方公里、实验区22.8112平方公里。两者约34平方公里，合计约占整个黄岩岛面积的23%。具体位置大致在三角形的底边。

## 这是什么概念？
你可以理解为，剩余面积扣除鴻湖区，再搞个几平方公里的空间出来，一点问题都没有，留下的想象空间太大了。我们国内以前搞新区开发，一个3平方公里规划，就可以把学校、医院、长途客运站、大型商业广场、10来个住宅小区等全部装进去。所以，对于这个大动作，不仅我们的舆论，包括菲律宾、南海诸国甚至老美那边的舆论也都炸锅了。

菲律宾国内已经从11号开始，爆发了大范围的抗议。不过没啥卵用，按赵立坚个人微博的表述：“坚决捍卫主权和领土完整的又一重大举措”。其实我相信，大家在兴奋之余，都有一种比较模糊的感觉，你能get到那个点，但好像里面又有点说不清、道不明的内容。

新建国家级自然保护区，那么下一步动作是什么呢？是名义上打生态牌，实际上在黄岩岛吹沙填海，搞民转军那套吗？还是仅仅只是主张权力，限制开发，保护原生态环境作为第一出发点？其实看不明白就对了，之所以被形容成“高招”、“妙棋”，本身主打就是一个让人看不懂，大家都一眼看明白了，何谈高招妙棋呢？

不过这里，还是分享一些个人的观点。

## 先介绍一下背景：
我们在南海搞吹沙填海工程，其实从上世纪80年代就开始了，不过因为种种原因，包括装备问题，还处于蚂蚁搬家阶段，真正上强度是要从2013年左右算起。包括扩建永兴岛、永属岛、美济岛等。黄岩岛一直以来，除了有一些设想规划外，并没有规模化系统化的搞过。当然并不是说我们不想或者做不到。

比如，黄岩岛的地形特征，出了这片区域，海底马上从100米下降到3000米，导致我们要吹沙填海，只有2个选择，一个是漏湖区里面吹沙，一个是靠外运。这就导致前面破坏生态，后面成本高。黄岩岛如果能吹沙填海，那在南海就会和永兴、渚碧岛形成区域大铁三角，正好和渚碧岛、永暑岛和美济岛形成的小铁三角形成呼应。三角形大家都知道，最稳的。

所以，不存在不想这个说法，至于成本方面，那更不是个事儿，虽然会对决策有一点影响，但权重不高，我们历史上长城都修了，就这。核心原因，除了确实涉及生态方面的考虑，还是就是顾忌到睦邻友好，毕竟黄岩岛距离菲律宾直线距离大致220公里，距离马尼拉也就330公里左右，可以说如果通高铁，90分钟左右就到，更别提导弹了。

## 那么,为啥如今要放这个大招出来呢?
核心原因大家都清楚，就是菲律宾在不停的作死，清单之前我们前面文章都拉了，这里重点谈谈这招“妙”在哪里?谈3点：

第一，菲律宾在2016年炮制的南海仲裁案，其中有個表面文章，就是指责我们南海的动作破坏珊瑚礁生态。虽然站不住脚，但要知道当年美西方氛围是左翼主流，勉强在国际舆论上还有点市场。现在好了，我们直接搞自然保护区，无形中就把这个问题化解了。其实，现在全球是右翼抬头，谁TM管你这些。

第二，菲律宾最近几年，对黄岩岛的挑衅，大多数都是打着捕鱼的旗号，一来确实近。二来条件好，黄岩岛有80平方公里的鴻湖区，是天然大渔场。三来巧立名目，大家都懂。现在不允许，我们国家級自然保护区，禁止捕捞。以后自己去另外寻地儿，再来就不是驱赶，直接抓人。

你看，我们这招是不是有点杀人诛心的感觉，把菲律宾2个核心诉求全部堵了回去。

第三，就是涉及我们进一步确权的内容。早在1279年，也就是元朝时期，天文学家郭守敬的“四海测验”,现在的黄岩岛就是南海测量点，后经明清时期，我们持续管辖，并于1935年纳入版图，1947年划入断续国界线内，1983年正式命名。这是历史权力。

2012年后，我们实控，2024年宣布黄岩岛领海基线。你发现，从历史经纬、到实控、再到法理确权，我们都做了，但很多人不清楚的是，这还不完美，还差一环。

这一环就是行政管理，这也是国际上公认的判定主权归属的重要依据。现在我们这招，就是补上这一环，完全实现证据链的固定，而且我还是打生态派，是人类命运共同体的一另注解。

当然你如果按老美的剧本说，实控就行了，搞啥行政管理，多此一举。但是我们不是老美，也不会学他，我们是礼仪之邦，这面旗帜在未来很长一段時間不会变。

至于大家最关心的问题，会不会吹沙填海，搞民转军那套呢？其实这个问题我觉得这倒不急。因为今时不同往日，我们现在的军事能力已经不是10年前了，现在处于爆高端装备阶段。再大胆的说，现在我们几座岛屿加军事、海警力量，在南海已经到了产能过剩的临界点了，连老美的航母舰队都要绕着走，再搞个大堡垒出来，其实际意义并不那么大。

也就是说，我们经略南海的战略其实在悄悄发生变化，最近2-3年，有听过像以往那样大规模系统性搞吹沙填海吗？没有吧。甚至越南在偷偷摸摸的搞，我们都像睁一只眼闭一只眼似的，原因就是我们海空军、导弹等新质装备在下饺子，吹沙填岛已经不是优先级了。

但是既然我们提到“新建”,而不是其他措辞，我个人认为钢筋水泥、人员是要进去的，保护生态，总得有点配套设施吧，不然我们怎么搞行政管理，搞长期科研，搞治安巡逻，总不能一天到晚开着船环伺吧。

至于什么时候搞?规模搞多大?涉及哪些项目?会不会埋点你懂的内容?我觉得前期我们会克制一下，如果菲律宾反过来认为睦邻友好很重要，那我们可以照顾下情绪。当然，如果继续挑衅，没完没了，那就是我的地盘我做主了。

不要忘了，黄岩岛距离马尼拉也就300来公里，距离马哈陶镇准备启动的“前沿作战基地”也就400来公里。你在那边加强海域感知，灾害应对，我们搞生态保护，异曲同工。还记得永暑礁吗?当年我们就是定义的搞海洋观测站。

当然，大家要清楚，中菲问题的本质，还是中美博弈。

就在我们批复新建黄岩岛国家级自然保护区的同時，中美罕见的在同一天进行了防长对话和王毅与鲁比奥的对话，南海问题在防长对话中是有体现的。甚至我们的福建舰已经赴南海相关海域开展科研试验和训练任务。其中的道道不言而喻，对菲律宾哪会动用航母呢？

之前我们在家办大事期间，菲律宾名目张胆接待台湾地区外事负责人林佳龙的窜访，我们就说过：“菲律宾踩踏中方红线的行为必将付出代价，由此产生的一切后果必须由菲方承担”

现在反制措施如期而至，菲律宾求锤得锤。

所以，我们不妨让子弹继续多飞一会，一口气把剧追完也少了点乐趣。现在着急的又不是我们，就让菲律宾寝食难安，习惯一段时间再说。
