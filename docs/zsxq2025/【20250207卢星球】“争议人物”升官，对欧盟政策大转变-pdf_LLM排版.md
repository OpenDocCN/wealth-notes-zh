
# “争议人物”升官，中国对欧盟政策大转变

2月6日，中国外交部证实，原驻法国大使卢沙野已被任命为“欧洲事务特别代表”。此事引爆了欧洲的舆论，“欧洲事务特别代表”是外交部驻欧外交人员里级别最高的。从驻法大使到“欧洲事务特别代表”，无疑是升官。此前，卢沙野曾因一系列强硬言论，被称为“战狼”。2023年5月他被调回国时，外界普遍解读为，中国政府不希望外交官对欧盟太强硬，结果才短短一年多，卢沙野就升官并重回欧洲了，自然相当出乎意料。

背后原因，可能是中国的对欧政策，将出现大转变。而要了解来龙去脉，还得从2023年4月下旬，卢大使的“争议言论”说起。

2023年4月21日，卢大使接受了法国右翼媒体LCI的一次采访，LCI属于反华反到不装的那种，嘉宾多次公开做咪咪眼，投诉也没作用。可想而知，这次采访的火药味，将会有多浓。LCI的记者，一上来就问：“克里米亚是属于乌克兰的吗？请明确一下立场，不要模棱两可”。

卢大使的回应是：“这取决于我们如何看这个问题。”

记者马上说：“这不是取决于，请回答，克里米亚是不是乌克兰的？”

这明显是胡搅蛮缠。

卢大使回复：“这要看你如何看这个问题，要看历史。首先，克里米亚一开始是属于俄罗斯的，难道不是吗？是赫鲁晓夫在苏联时代将克里米亚送给了乌克兰...”

听到这里，记者强行打断卢大使的话：“但是你知道，按照国际法，克里米亚是乌克兰的，你可以提出异议，但它属于乌克兰的。”

卢沙野毫不客气：“怎么说呢，根据国际法，这些前苏联国家甚至没有实际地位，因为没有国际协议来具体确定他们的主权。”

这句话，其实并无问题，前苏联解体时，确实没有什么国际协议，来确定新独立国家的实际地位。但就这么一句陈述客观事实的话，让欧洲媒体炸了，纷纷报道称：“中国驻法大使表示，前苏联解体后成立的新的14个国家，不具有独立国家的主权”。

有没有实际地位是一回事，主权又是另一回事，卢大使的原话，并无否定他国主权的意思，但欧洲媒体在偷换概念方面，显然是专业的。

欧洲媒体铺天盖地的炒作后，欧盟通过官方渠道，向我们表达了不满。5月中旬，卢大使被召回国，担任“对外友好协会”会长。

我们当时的考量，确实有几分平息欧洲不满的意思，毕竟俄乌战局，那时对俄罗斯不利，俄罗斯战败的可能性，是客观存在的。一旦俄罗斯战败或遭到重创，欧盟对中国来说，就十分重要了，是避免遭到美国围剿的关键变量，是我们必须拉拢的对象，起码要争取其中立。

事实上，从贸易战开始，我们对欧盟的态度就以拉拢为主、斗争为辅，在2021年签署的《中欧投资协定》里，我们向欧盟开放了制造业、建筑业、金融业等多个领域。而欧盟向我们开放的，只有可再生能源领域，明显是不对等的，同时，哪怕欧盟在许多方面，做了不符合中国利益的事情，我们的反击也不算猛烈。

其考量，无非是彼时的欧盟值得花大价钱拉拢，因而在对欧外交上也不宜偏强硬，卢大使被调回国，实际降职，也就不足为奇。

只不过，当时间来到2025年2月，形势发生了大变化。

自扎波罗热反攻失败后，乌军虽偶有两点，比如“库尔斯克攻势”，然而总的战场形势，对乌军还是日渐不利，天平不断向俄罗斯倾斜。

2024年前（除了开战头一个月），乌军战斗力还在线，俄军哪怕想占领一个村都难，但进入2024年后，在各种不利因素的加持下，乌军战斗力下滑明显。俄军每天占领几个村，已问题不大，有时候一座几万人的市镇，也能两三天搞定。反倒是乌军，想夺回一个村十分艰难。赢，肯定是俄罗斯赢了，无非是赢多少的问题。

俄罗斯不会完蛋，意味着拜登“先击破俄罗斯、后围堵中国”的图谋破产，我们对于拉拢欧盟的需求，大大减少，没必要忍耐那么多了。

当然，最重要的还是懂王归来。

欧盟那边不少人以为，懂王虽然看欧盟不顺眼，但在“对华关系”上，却能成为欧盟的筹码，理由是六年前开始的第一轮贸易战，让中国措手不及。

那时我们心里没底，因而急需拉拢欧盟，便在不少问题上给了欧盟面子和里子。理论上，懂王二进白宫后，欧盟将能如法炮制，向中国要好处。

殊不知，欧盟只是想得美。

和2018年比，2025年的中国，大部分人对于贸易战已经有底了，我们确定以及肯定，美国打不动我们。中国产业链的韧性和优势，远比想象中要大。

一些人喜欢说，懂王要是加60%的普遍关税，咱们就完了。这就是搞笑。要是关税这么好使，还用得着懂王来加？莫非拜登不会加？

2月4日，美国邮政宣布，暂停接收来自中国内地和香港的入境国际包裹，主要是800美元以下的商品，结果就在短短一天后，就恢复接收了，光速打脸。

为啥？

因为在经历大通胀后，来自中国的高性价比商品，已深入美国大部分人的生活，尤其对于中下层而言，不买中国制造的后果，就是生活成本要大涨一波。谁能受得了？

美国在贸易战上打不动中国，是事实。既然如此，我们拉拢欧盟“共同抗美”的需求，便进一步减少。

在打不动中国的情况下，懂王极有可能把目标，对准那些容易拿下的盟友。你看加拿大不就是，特鲁多才抵抗了两三天，就跪了。换了我是懂干，我也倾向于从盟友身上抽血，毕竟和中国纠缠成本高，风险还高。收益还看不到，盟友就不同了，成本低、风险低，收益还相当可观。

除了收益，意识形态上，懂王早就看欧盟不顺眼了，欧盟可是白左遍地的，懂王最近刚收拾了作为颜革总部的“国际开发署”，可见其对白左是不客气的。

欧盟作为大号“国际开发署”，能逃得掉？懂王的就职典礼嘉宾名单上，就有泽穆尔和梅洛尼两个欧洲极右翼，还有博索纳罗、米莱、欧尔班等人，都是欧盟非常不喜欢的人。

前段时间，马斯克更是明牌支持德国选择党。毫无疑问，懂王和欧盟的意识形态大冲突，早晚会上演。说句难听的，欧盟现在更需要拉拢我们。

否则，有被中美“混合双打”的风险。

此种形势下，对欧盟的政策，没必要再像过去那样柔和，转向适度强硬才符合国家利益。而卢沙野的升职，就是转变的公开信号。

至于欧盟会不满，那就继续不满好了，磕磕碰碰是外交上的常事，不必过于在意，更用不着反思。外交政策的核心，一定要以“我”为主。
