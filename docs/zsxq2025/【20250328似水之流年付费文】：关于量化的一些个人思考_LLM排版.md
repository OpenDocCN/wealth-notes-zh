
# 关于量化的一些个人思考

原创 轻风吹断似水之流年 2025年03月28日18:51 浙江

背景照片是库拉岗日，上次喜马拉雅国境线的时候走过。这篇推送原本不在计划内。但最近量化确实成了市场里最大的一个干扰因素，网上各种怨声载道。这几年，陆续和几位国内顶级的量化管理人有过接触和深入交流，细节就不公开说了。自己也在实际操作中有些心得。所以有了这么一篇。

国内的量化和国际上那些其实还不是一回事，有点一群鬣狗围猎的画面感，同时开着作弊器和缺乏有效监管。这话题也没法深入展开，废号。但是丛林法则角度，既然我们没有能力去改变这个生态，抱怨、辱骂毫无用处，要么忍耐、适应、要么滚蛋；当然，更高一个级别，还可以利用其特质。

先说好的一方面，量化确实提供了流动性。举个简单的例子，你手上有个正常走势，非极端形态小市值标的万手单想快速兑现，以前可能要往下打二十个三十个单位才能卖掉，有了量化以后，可能只需要往下打不到十个单位，甚至三五个单位就给你迅速成交了。当然，往上想买，你可能真跑不过对方。

其次是提供了更大的波动区间。熟悉量化规则的人，真能在波动中获得超额收益。观察量化。

投喂了无数战法的量化，残酷而贪婪，没有人性中的弱点，所以确实非常具有杀戮性，收割了太多流动性。再加上量化与量化之间还开始内卷。

学习量化。

## 怎么学?

其实，就是反人性那套东西。情绪高点他是要收割的，跟着分几次卖，不要求卖在最高处；情绪低点，重要均线，分几次买，和量化抢血筹。

## 怎么对付量化?

不说歪门邪道投喂语料那些东西，正大光明的方法。

第一种是引导。单个量化其实没多少体量的，哪怕三五十个在一起也不怕，几百上干条鬣狗就很讨厌了。他们确实有冷酷、纪律性的优点，但也有蠢的一方面，机器毕竟是机器。其实关键时间、关键位置是可以引导的。做一个空头陷阱让他们组团割肉，也可以做一个多头陷阱让他们高位接盘。别以为不可能，但有些东西网上不适合说太细。

第二种是正面硬刚。

我们知道，这个市场有一个极其朴素的真理是：

价值决定价格，价格围绕着价值上下波动。就比如去年九月。很多人都买到了风报比超过1:5的血筹。然后迅速翻倍、翻几倍。

在这个过程中，我们找机会把成本做到0或者负 ( 这 不 需 要 教 了 吧 ? 翻 倍 卖 一 半 总 会 ? )

前提是： 市场主线的标，我们是要来回折腾做到整轮行情结束的。

有一个零成本或者负成本的前提，是根本不惧量化的。你敢乖离短期均线太高，动态仓分几次砸给你；你敢杀回日线、周线级别重要均线，动态仓接回，下次再乖离短期均线太高分几次卖给你。如此循环。

踏空?不存在的，零成本或者负成本的底仓一直在。

这样来回操作，加上利用量化的高波动，经过一个时间周期，你可能利用波动赚很多免费筹码，甚至超过最初开仓时候的筹码，成本还是零或者负的。这就是高波动给玩家的福利。

守着自己那一亩三分地， 一直守到行情结束，你可能不是量化的点心，反而量化大部分时间，在你这都讨不到好。前提是你战胜了自己的贪婪与恐惧。

就怕不熟悉的山头，去送人头。你又有几颗人头可以去送?

这也是我一轮大行情，坚守自己能力圈内有限标的原因。大部分时间，量化是收割不到我的。

就比如这个三月，外面鬼哭狼嚎。
