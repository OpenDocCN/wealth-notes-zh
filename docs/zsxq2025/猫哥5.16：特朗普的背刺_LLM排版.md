
# 猫哥5.16：特朗普的背刺

近日，特朗普的中东行取得空前的成功，至少在拉动中东土豪的投资层面获得惊人的成就。首先是卡塔尔宣布，未来将在美国投资1.2万亿美元，然后是沙特也宣布将在美投资6000亿美元，第三个是阿联酋，承诺未来10年内将在美国投资1.4万亿美元，三个国家投资累计达到3.2万亿美元。

那么问题来了，为什么中东土豪纷纷对特朗普慷慨解囊呢？天下没有免费的蛋糕。看看特朗普给海湾国家提供什么样的回报与承诺。

第一个是叙利亚，特朗普宣布将取消对叙利亚的所有制裁。

——“我很高兴地宣布，国务卿马尔科·卢比奥将于本周晚些时候在土耳其会见叙利亚新任外交部长。非常重要的是，在与叙利亚总统、你们的王储以及土耳其总统埃尔多安讨论了叙利亚局势之后，我将下令停止对叙利亚的制裁，以便让他们有机会重振雄风。埃尔多安前几天给我打了电话，提出了类似的要求。此外，我还与我的一些朋友以及我在中东地区非常尊敬的人士讨论了叙利亚局势。”

中东产油国当然是支持解除对叙利亚的制裁。第一，让他们境内的叙利亚难民赶紧回国；第二，减轻他们的财政负担——现在叙利亚政府的财政是卡塔尔出钱供养，一个月2900万美元。

其第三，解除制裁的叙利亚经济有望快速发展，作为以色列的邻国，叙利亚越强大，就越能遏制以色列。为了让特朗普取消对叙利亚制裁，卡塔尔功不可没，不但承诺未来投资美国1.2万亿美元，还给特朗普送了一架价值4亿美元的专机。

不仅仅是叙利亚，特朗普端上第二盘菜就是重提亚伯拉罕协议，这是一个推动中东和平进程，推动海湾国家与以色列关系正常化的协议。

但问题是，现在沙特找就明确开出条件，加沙不停火，沙特就不会与以色列建交。所以，特朗普现在要推动亚伯拉罕协议首先就是要施压以色列在加沙停火。

但是以色列只要停火，下半年就要重新选举，内塔尼亚胡就得下台被清算，所以内塔尼亚胡怎么可能答应停火？现在特朗普将这个承诺是放出去了，未来做不做得到就看特朗普的本事了。这是特朗普对以色列与内塔尼亚胡第二击。

特朗普对以色列的第三击是黎巴嫩。特朗普公开宣布将向黎巴嫩提供援助！谁会喜欢一个在美国援助下变得强大的黎巴嫩？海湾国家。因为强大的黎巴嫩和强大的叙利亚必然是遏制以色列的急先锋，所以过去以色列的国策都是让自己的邻居更加虚弱，现在特朗普竟然反其道而行之，不但要取消对叙利亚的制裁，还要援助黎巴嫩！

特朗普对以色列的三连击意味着什么？过去美国的传统国策都是偏袒以色列，按照以色列的意愿去塑造中东的政治格局，现在特朗普来了个180度大转弯，第一次是按照海湾国家的意愿去塑造中东政治格局，这就是海湾国家纷纷为特朗普慷慨解囊的原因所在。

那么问题来了，一贯亲以色列的特朗普为什么突然背刺以色列了呢？几个原因。

一个是现在特朗普与内塔关系已经彻底破裂。破裂的原因源于前白宫安全事务顾问被着特朗普与以色列讨论军事打击伊朗的事件。

白宫安全事务顾问理论上是总统的绝对心腹，这个位置的人不可能不知道总统对外政策的底线，特朗普对伊朗一贯政策是政治解决，但是华尔兹却公然与以色列撕下讨论对伊朗的军事打击计划，这不仅已经大大超出特朗普容忍的底线，更让特朗普有理由相信华尔兹已经被以色列收买，这让特朗普有一种强烈的被内塔尼亚胡操弄于手掌之间的感觉。所以不但华尔被解职，愤怒的特朗普甚至拉黑了内塔，切断了与内塔的一切联系。

一个是中东国家开出条件确实诱人，特朗普只要按照中东产油国的意愿去重塑海湾政治格局，光是沙特、卡塔尔、阿联酋三国就承诺对美投资高达3.2万亿美元！特朗普商人出身，只要条件开得足够高，没有什么事不可以交易的，所以特朗普就果断背弃了以色列，也背弃了美国数十年对中东的基本国策。

谈谈特朗普背刺后续的影响。

首先，特朗普目前所有的承诺都还停留在口头上——当然中东土豪的投资承诺目前也还停留在口头上，未来能否兑现主要还是看特朗普能否兑现承诺。但是犹太集团在美国势力根深蒂固，特朗普要背刺以色列按照海湾国家意愿重塑中东政治格局阻力极大，难度也极大，未来能否落地存在极大的变数。

其次我们也要看到，过去几十年兔子在中东地区一直是坚定站在海湾国家这个阵营，并且为了推动中东的和平进程做出巨大的努力。但是回报呢？沙特是比较明显骑墙的，但是沙特在兔子这几年累计投资也不过区区几百亿美元而已，特朗普仅仅口头承诺按照海湾国家意愿重塑中东政治格局就拿到3.2万亿美元的投资承诺。

从这一点可以看出，要对一个地区真正具备举足轻重的政治影响力，归根结底还是要有在该地区强大的军事投放能力，目前在中东地区有且仅有美国有这个能力，对中东国家的影响力就与兔子完全不是一个数量级。所以加强军事实力建设特别是域外跨地区军事投放能力建设，对于兔子而言还是很急迫的。
