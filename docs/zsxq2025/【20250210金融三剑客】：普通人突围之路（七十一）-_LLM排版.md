
# 普通人突围之路（七十一） 

今天是除夕，开头祝大家除夕快乐，阖家幸福，来年发财。下面继续上节内容。

## 一、趋势

以前宣传集采的时候，是把上万的心脏支架，价格打到700多一个。但医生是专业选手，他们手里，依然握着信息差和选择权。别的不说，做手术的时候，就问一句，进口的1万，国产的700多，请问你敢用吗?病人肯定吓得一哆嗦。

怎么解决呢?最终还得靠信息透明化和分层。80%的普通人，他们没得选，6也不要选，直接给他们性价比最高的，满足其需求就行，他来了以后跟着流程走就可以了。而另外20%的人和病，他们愿意花大钱，6做小的疗效提升(追求总疗效最优，价格不敏感),可以开放别的市场渠道满足他。

就像房地产市场，新加坡大部分人是住组屋的，保证其基本居住的实用性，少部分人钱有富余，去买贵的商品房改善居住体验。然后中间差距慢慢拉大，大到什么程度呢?普通人靠勤劳、靠省吃俭用，怎么也够不上为止。

简单讲，前者是价格稳中有降的游戏(通缩模型，类制造业),后者是价格持续上涨的游戏(通胀模型，类垄断型)。

对操盘手而言，这就可以同时起到刺激两边人奋斗的效果。前者看着自己财产缩水(赶紧奋斗换圈进资源高地),后者财产持续上涨(赶紧往圈上面爬获得更核心的可传承的资源，而不是过手财)。

工程领域也一样，同样一个工程，zf交给你的，和市场化建设的，拿到的钱、利润，不一样。那就把工程项目分为公益型的，和市场化型的。采取不同的考核指标，前者弱化利润，强调成本和质量。后者强调赚钱，以工期和利润为核心。

所以从趋势上来看，现在混沌的，掌握在医生手里的信息差，最后的趋势，就是往分层、透明的方向走。

## 二、举措

有人追求价格，那就带量集中采购，把价格打下去，俗称得让他看得起。如果你是在以集采、公立为主要属性的医院里，工资待遇可能提高，但药品回扣、医保外的路，逐渐就会被堵死了。

但总有人要追求更好的疗效，那就要放开高端医疗，俗称得让他看得好。如果你在允许使用进口、原研药的医院里从业，面对的多是那种不差钱的人群，原来那一套，就还可以继续玩下去。

之前有读者是某药企的从业者，曾经当面和被抓的那位高先生，有过面对面的对话：“如果价格这么低，我们进口企业都要退出了”。高先生说，“退出就达到我目的了，我就可以升职了，但你们退出前，要留几套产品给我。万一哪天我病了06我还是要用你们进口的”。

看到没?他不是不懂，而是懂的很。但他只能和业内的人这样说，这就是信息密度的差别(操盘手要信息分层，所谓的合规的原因)。

上面人都知一道好一坏，但,并不妨 -- 碍他的 - 霹雳手段。

革命不是请客吃饭，不是做文章，不是绘画绣花，不能那样雅致，那样从容不迫，文质彬彬，那样温良恭俭让。矫枉必须过正，不过正不能矫枉!——教员《湖南农民运动考察报告》

只要这个动作，暂时还影响不到他个人。甚至还能帮他个人，实现更多的升职和其他利益，他是不介意有其他负面影响的。

换言之，之后分层会有，高利润也会继续搞。但选择权，已经从医生手里，上移到了决策者手里(收权)。大多数医生的结构价值，被系统化给没收了。话说时代的车轮滚滚向前，作为个体，应该努力站在优势的那一边。

## 三、进阶

之前讲过，商业模式分两种：渠道和终端。一个游戏一定是渠道先展开，意思就是说，我要有，我要覆盖全民，要让大家能买到商品，买到服务，我得迅速铺开，让所有人都看得起。你先甭管好评差评，得有，这是第一步。

那么随着"有"这件事进入后半程，进入红海之后，新的市场增长点在哪里?就是“放心”两个字。有，不等于好。看得起不等于看得好。

买到服务里面的一部分客人，对当下的乱象并不满意，人家要的是放心服务。人家这部分人要的是不看价格，看疗效。即市场主导的环境下，所有增加成本的东西，一定是有客人愿意为之买单，才会产生的。

更重要的是，你要清楚，咱们国家当下的消费结构是商品消费占大头，服务消费占小头，而发达国家都是倒过来的。

而上峰给出来的方向，也就是未来我们要引导的消费方向，或者说我们未来的结构，也将是服务消费占大头(美国是医疗，教育，律师三大件)。

## 四、过渡

还是回到前面讲得那个工程管理的例子，高手选择的是什么?狠抓过程管理。高手他不可能一个人去干一百个人的活，但是这一百个人脑子里的东西，他都要知道，然后装进他脑子里，并整理清楚，随时供他的领导提调。

那他的领导就很爽，因为领导只需要对接他一个人，而他会把所有细节统统搞清楚，并且随口能回答。

话说领导有他这号的下属，起码随时知道目前战场上哪个局部需要添兵，哪个局部需要运输兵器，哪个局部缺乏粮草了。

这样领导就很安心，只要放权给他，就可以休假，就可以放松，因为领导知道，他办事靠谱。那他的权柄自然越来越大，说到底，是领导需要他权力大。你品，你细品!

翻开历史书，所有的权臣，干政的太监头子，本质上都是这么做到的。只有上面希望你权力大，你才真的能够权力大。

你去看，上到一个金融交易员，下到一个房产中介，只要是优秀的，这都是基本功。他们肯定是对历史数据了如指掌的同时，同时还对当下正在发生的事情十分敏感。

好比这个小区现在有一套房子，以很低的价格挂出来，他第一时间就能知道，业主前脚挂，后脚他就知道，当天就联系自己的客户(还没挂内网), 等第二天其他中介知道了 (挂内网排序会靠前), 人家都成交了。

如果你连这个都不知道，你就不要干这行了，交易机会就被人家抓走了。你的客户发现一次两次三次，永远从你这里拿不到最新的市场变化，人家就会改投别的中介怀抱了。话说给了你机会，你真得不中用呀。

那你的领导是什么角色?领导就是你在公司的客户。领导什么时候来找你?难道是他想要了解国际局势的时候来找你么?不，他只有在想要了解项目细节的时候，才会来找你。

你比如雍正王朝里面，康熙把胤初叫来，问他说，四阿哥的儿子病了，你知不知道?胤初赶忙回答，我这就去传太医，康熙不满的说，等这会儿再传，不就迟了么?他昨天夜里，已经派太医去看过了。

言外之意就是说，你这个太子当得不合格。你的下属四阿哥在外面办差，他家里发生的重大事情，你都不清楚，那你这个项目经理，是怎么当的呢?

你的泥工 (四阿哥), 水泥不够了 (儿子病了), 客户给送 过去的 (康熙解决的), 你这个项目经理(太子胤初)还不知道，你说你干得如何?

这个道理说白了就是过程的细节管理。胤初的过程管理等同虚设，那他的领导，怎么可能对他满意呢?

各位千万不要觉得领导这是为难谁(你把自己代入家庭装修的那个客户想想便知),一个人(那个设计师/项目经理)如果不能处理碰见的各种复杂事件，就说明他不适合升职(不能帮客户省心省钱省时，那还转介绍个P? 还会让他管理更多客户的装修 ?)。

## 五、比拼

而且，过程管理最重要的价值还不在于上面说的这些，你还要明确留痕待查。

剑客再打个比方，比如医生，你觉得作为一个医生，事关他职业生涯最重要的点是什么?是医术还是医德?答案都不是，最关键的点，0是认真写病历。

历史大数据会告诉你，那些平稳熬到老，成为有资历的主任医师，他们在当年同样的一批医学毕业生里面，未必是医术最好的，也未必是医德最高的，但基本上都是病历写最好的。

病历的本质是什么?就是所有医疗动作的过程管理。你看十个病人百个病人可以讲医术医德，你看一万个病人，十万病人，这里面靠什么?只能靠病历，只能靠过程管理。

因为在足够长的时间段内，在足够多的工作样本下，一定会发生什么?医疗事故，一定会有医闹，也一定会有说不清的时候，这时候，拼的就是病历的严谨性，完整性。

话说你的过程管理做到极致了，别人随便追溯工作中的任何一个环节，你都是OK的。
