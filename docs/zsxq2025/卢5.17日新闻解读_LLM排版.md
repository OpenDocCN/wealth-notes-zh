
# 卢5.17日新闻解读

## 2025年5月17日 新闻解读
1、印尼政府重新评估采购“阵风”战斗机合同。
据外媒16日报道：印尼政府已开始重新审视与法国签署的“阵风”订单。原本81亿美元军购大单，现在因舆论压力而出现转折。
在首批6架“阵风”即将交付之际，印巴爆发空战。在5月7日的冲突中，印度空军的“阵风”战斗机被巴基斯坦空军的J-10CE战斗机击落三架。成为“阵风”自服役以来，最大战损。
早前，印尼向法国达索订购了42架“阵风”战斗机，总额约为81亿美元，平价单价约为2亿美元。相比之下，巴基斯坦采购的J-10CE战斗机，单价约为6000万美元左右。单价差距如此之大的情况下，以3比0的战绩取得完胜。确实对全球战机市场是一次重大冲击。

印尼的“假想敌”长期以来是澳大利亚。受限于美国与澳大利亚的关系，印尼很难从华盛顿获得先进战斗机。一方面，澳大利亚已经开始接收F-35战斗机，并进入澳军服役。另一方面，出于战略利益和商业诉求，美国向印尼推销了F-15EX战斗机。原本印尼计划采购24架，但受到政治因素的影响，至今尚未获得美方“放行”。
随着印尼与澳大利亚之间战机落差的出现，印尼采取“多条腿走路”策略。除了积极评估土耳其的TF-X“可汗”战斗机之外，还与韩国共同研发KF-21战斗机项目。但该项目饱受一系列泄密事件的影响，前途未卜。甚至传出印尼将退出该项目研发，因为预算严重超支。选择“阵风”战斗机，一定程度也是印尼的无奈之举。
一方面，除了“阵风”之外，欧洲还有“台风”战斗机。然而后者是欧洲多国联合研发，对外销售需要获得所有参与国的一致同意。这也恰恰成为阻碍“台风”战斗机在国际市场的销售。倘若采购“台风”战斗机，印尼所要花费的外交成本过于高昂。
另一方面，“阵风”战斗机曾在与F-35的模拟空战中并不落下风。或多或少可以追平与澳大利亚空军的差距。同时，印尼是一个拥有超过三亿人口的新兴国家，而本国又没有系统性的航空工业。如此诱人的市场，也是法国航空业所神往的。
印尼当然有理由重新评估这份合同，将此作为讨价还价的筹码。要么让法国达索多打点折扣，要么让法国达索提升印尼空军的技术水平和能力。

2、美国允许阿联酋每年进口50万片英伟达芯片。
据美国商务部网站消息，当地时间15日，美国已与阿联酋达成协议，允许其从今年起每年进口50万片英伟达先进人工智能芯片。
美国总统特朗普和英伟达总裁兼首席执行官黄仁勋出席了协议签字仪式。
另外，根据该协议美国将与阿联酋合作建设一个拥有5吉瓦数据中心的人工智能园区，有望成为美国境外规模最大的此类园区。协议草案规定其中20%（即每年10万片）芯片将供应给阿联酋人工智能初创企业G42，剩余部分将分配给美国的大型高科技公司，包括且不限于微软、甲骨文等。这些企业有计划在阿联酋建立数据中心。
这也是特朗政府在取消拜登任内制定的所谓半导体全球“分级体系”。按照这一体系，阿联酋采购半导体芯片的数量和规格都有所限制。在阿联酋看来，这是美国“有意”限制其发展人工智能等高科技产业。
此番，特朗普出访中东三国，确实收获颇丰。至少从账面数据看，不但给美国吸引数千亿美元的中东投资，还销售了数千亿美元的美国产品。这点成绩足够特朗普继续“吹嘘”一段时间。
允许阿联酋每年进口50万片英伟达芯片，一方面是安抚黄仁勋为代表的美国半导体行业。美国政府的半导体政策，一直都在限制特定国家取得和降低行业损害之间拿捏。另一方面，阿联酋虽然国家不大，但在国际外交舞台上长袖善舞。无论是稀土资源还是高科技产品，阿联酋以“中间商”的身份，赚得盆满钵满。很难说，此番获得如此之多的英伟达芯片配额不会出现在国际市场上。
另外，美国的高科技企业可以在阿联酋建立研发中心。非美国以外的高科技企业也可以在阿联酋设立研发中心。人工智能的发展需要算力、电力、数据和算法四个部分，而美国只是在某个部分处于优势地位罢了。
