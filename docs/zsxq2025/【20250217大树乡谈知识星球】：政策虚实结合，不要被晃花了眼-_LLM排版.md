
# 政策虚实结合，不要被晃花了眼

这周国常会一反常态，研究了非常多的宏观大问题，包括提振消费、化解重点产业结构性矛盾也就是产能过剩、稳外资这三项大事。之所以说一反常态，是因为现在的国常会对这类宏观问题关注度不高，更关注具体工作，而且就算谈宏观，也往往只抓住一项，而不是全放在一起，这就太紧张了。

比如最近三个月的国常会。1月17日是促就业、基层医疗卫生机构布局建设、高等教育改革试点；1月3日是城市更新、农民工工资支付；去年12月24日是涉企行政检查、药改、食品监管、转移支付机制；12月16日是专项债、政府投资基金、安全生产、足球振兴；还有12月6日的高质量发展督查、黄河流域生态保护等，全都是相对具体的议题，像本周的这三大议题，应该是更高一层会议研究的。或许这种不同寻常的安排是为三月的政府报告做准备。

会后网络上也有很多解读，这些解读过于亢奋，还是一贯的老问题，就是看到开会讨论了什么就觉得都要落实，但注意，会议提到的并非都是实招，相当比例是虚招，也就是重复强调此前提出的原则性表述，表示这些关键问题高层始终关注。

这次会议虚实合在一起，是一个很好的分析案例。先说标本兼治化解重点产业结构性矛盾，会后网上有人解读为要进行新一轮供给侧结构性改革，甚至要像2016年那样以行政命令来一轮去产能。这种解读就错了。

虽然去年底经济工作会议部署了“综合整治内卷式竞争，规范地方政府和企业行为”，但需要注意，从2023年底到现在一年多，但高层对产能过剩的关注度并不高，光伏作为产能过剩化解的标杆，10月虽然行业企业联合搞了最低成本价，李总也专门去调研了光伏，但这不能说明就要搞供给侧改革。因为国家目前关注重点是科技创新、产业升级，相比2016年那一轮针对钢铁等传统的去产能，目前的特征是新兴产业和传统产业之间的博弈与竞争，新兴产业也出现了产能过剩，这背后牵扯到发改和工信两大部门，谁愿意牵头去砍自己主管领域的产能?谁愿意去跟地方博弈?

在这种纠结中，去产能的任务落在了更弱势的市监身上，2023年经济工作会议提出“要以提高技术、能耗、排放等标准为牵引”，以标准推动去产能，这是新的做法，但技术标准哪里是这么快就能出来的，任何一个标准都牵扯到复杂的博弈。就以两轮电动车为例，在2018年新国标之前，电动自行车就是一个灰色地带，各地监管不同、车型也五花八门，这是因为1999年的首个国标就很粗糙，之后一直想更新，但整整卡了20年，中间多次暂停又重启，可见有多难。而且让一个不管钱也不管帽子的市监去化解产能过剩，这不现实，产能过剩背后是地方政府、行业企业的切身利益，地方政府重复建设、内卷式招商很难遏制，想要规范企业竞争就更难了。

在这一背景下，化解产能过剩能走的多深，取决于李总关注度有多高，但显然李总更关心的还是科技和先进制造，就算是要化解产能过剩，也更多是从扶持新兴产业发展的角度出发，更注重为新兴产业的产能寻找出路。比如调研光伏时强调"用好建筑物外立面特别是屋顶空间，进一步释放光伏等新能源发展潜力”，之后调研中国建筑科技展，特别强调绿色化，就是要为光伏寻找出路。而且，去年上半年，我们在回应耶伦“产能过剩威胁论”时，也反复强调考虑到未来需求增长，产能不是过剩而是不足。

所以本周国常会强调的是“从供需两侧发力”，化解过剩，既要推动优胜劣汰，靠市场竞争加速出清，但也要为产业开拓新的需求点。需求增长，过剩产能同样可以化解。所以这一块，不要光看表述就想去当然，2025年不会搞新一轮供给侧改革，更不可能像2016年那样通过行政命令硬压去产能任务，而是把重点放在为新兴产业寻找新需求点。

类似的还有提振消费，这也是最近两年特别关注的，近期各路智囊的意见高度统一，都认为要提振消费。这次国常会，也大篇幅谈提振消费，再次重复强调提振消费工作的重要性，要求“转变观念，把提振消费摆到更加突出的位置”，还提出了很多措施，包括居民增收、深挖消费潜力、服务消费、大宗消费、新型消费等，还表示要实施优化消费环境三年行动。但大多数是虚招，并没有可行、效果可观的落实办法。比如促进居民增收，提到的“拓宽财产性收入渠道”，中国居民最主要的财产性收入渠道无非就是房产和股市，但现在房市、股市不是一时半会能恢复的，而在当前经济增速放缓下，工资性收入也在受到冲击。其他措施也大体类似，比如服务消费，虽然提到“一老一小”、便民、文体旅游、冰雪、入境等方向，但服务消费的增长是很慢的；新型消费提出的“人工智能+消费”新概念很美好，只是既然是新型，就意味着刚起步、还没想好怎么搞，国家不会贸然入场，而是会继续观察市场创新；至于消费环境，这个当然要做，但优化消费环境与促进消费并非直接相关。

所以真正能够落到实处的，只有大宗消费，也就是加大消费品以旧换新支持力度。这从本周发改。商务对外宣传就能看出来，两个部门都非常热烈的总结春节期间消费的火爆，尤其强调以旧换新政策的积极意义。比如发改说“以旧换新销售量达到860万台(套)销售额分别约66亿元、105亿元、141亿元”，还专门介绍新纳入以旧换新范围的手机等数码产品销售额占以旧换新总销售额的45%；而商务则介绍“有2009.2万名消费者申请了2541.4万件手机登数码产品购新补贴。”

以旧换新确实是非常好的政策，2024年推出后效果非常好，今年肯定会加力。2024年央地投入以旧换新资金约1700亿元，撬动了1.3万亿的消费，今年很可能要提高到5000亿元甚至更高，这对于化解产能过剩很有帮助。去年下半年开始也在加大力度宣传家电超期服役的危害，小镇单元门口就刚贴上了各种家电的不同寿命期。

而稳外资，这也是时不时就要强调的老话题，自2018年以来，稳外资方面面临的挑战很大，但能做的也有限。本周国常会提到的措施，包括有序扩大自主开放、相关领域开放试点、扩大鼓励投资产业范围等等都是过去反复说的，难度都很大，比如去年刚刚开展的扩大外资独资医院试点，实际跟前几年的没有本质变化，还是要求以本土医疗机构为主，外资采取统一标准并且作为本土医疗的补充，外资希望放松的并没有放松，对外资的吸引力并不大。

至于“政府采购做到内外资企业一视同仁”，落实起来也很难，先不说内资企业的成本压得更低，如何解释高价中标?何况选外资，还要承担安全审查等额外风险，跟国产替代的趋势也有矛盾，政府采购还承担扶持中国企业发展的任务，国外也没有向中国企业开放政府采购，我们又何必呢?

这次比较新鲜的是三点：加大外资企业境内再投资支持力度、鼓励外资在华开展股权投资、优化外资并购规则和并购交易程序。但外资企业境内再投资本就很支持，也很难再加大支持力度；并购方面我们有顾虑，而在贸易战加剧的情况下，外资也有顾虑；至于股权投资，内资搞股权投资都不容易，更别说外资了。因此这三点大概也就是推出一些具体的细则修改，不会大动。

总之，作为3月政府报告的前瞻，本周国常会一窥未来的重点，未来这三方面的工作，大概就如本篇分析的一样。
