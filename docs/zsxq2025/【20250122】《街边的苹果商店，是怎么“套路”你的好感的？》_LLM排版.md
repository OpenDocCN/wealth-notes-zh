
# 街边的苹果商店，是怎么“套路”你的好感的？

今天，将 从两个话题出发，为你提供知识服务。
首先，临近春节，第一个话题，咱们来关心关心健康方面。
这几天连续看到几个健身博主猝死的新闻。
就在今年1月初，韩国健身网红朴胜贤突然去世，年仅35岁。去年9月，来自白俄罗斯的健身网红伊利亚·耶菲姆奇克，在家中突发心脏病去世，年仅36岁。同样在去年9月，巴西的健美运动员马修斯·帕夫拉克，也被发现在家中去世，年龄仅19岁。去年6月，曾经拿过世界健美冠军的韩国运动员柳济亨，心脏骤停去世。
仔细想想，这些不幸的发生其实很奇怪。我们都知道健身有很多好处。比如说健身博主的身体素质应该比一般人过硬，怎么会频频发生意外呢？
前段时间，《南方风窗》上发了一篇文章，专门回答了这个问题。结论是，健身博主的意外身亡，除了自身原因之外，背后还有整个社会叙事的推动。
首先，作为一个职业，健身博主间的竞争更激烈，比其他类型的博主都要激烈。为什么？因为这是一个有标准答案的比赛。作为一个健身博主，你的体脂率、肌肉的围度、肌肉的形状等，都是可以量化的。
比如，前面提到的耶菲姆奇克，胸围大约是155厘米，而普通男性的胸围一般在80到100厘米之间。德国媒体《Tag24》说，耶菲姆奇克是当今世界上最骇人的健身者之一。
没错，一旦选择做健身博主，就意味着你要跟耶菲姆奇克这样的健身狂人竞争。很多健身博主为了快速达成目标，还会滥用药物。韩国健身网红朴胜贤生前，就承认自己会服用类固醇。
假如咱们再往下多想一步，就是很多健身博主是非常专业的，他们绝大多数人都知道过度健身和滥用类固醇的危害，为什么还会甘愿冒这个风险呢？
这就是要说到的，社会叙事层面的因素了。有些社会流行一种观念，认为男性应该强壮、强势，以及隐藏脆弱。注意，真正要人命的，就是最后这个隐藏脆弱。在很 多社会叙事里，都认为真男人应该打碎牙往肚子里咽。有痛忍着，才是纯爷们。
比如，来自美国加州大学南加州分校的社会学家迈克尔·梅斯纳，他研究的重点就是男性气质。之前他就有过一个疑问，他的父亲为什么会在56岁就突然离世？要知道迈克尔·梅斯纳的父亲，年轻时是高中橄榄球队的一员，二战时又在海军服役，平时基本不抽烟，也很少喝酒。按道理来说，他的父亲在身体上应该胜过很多同龄人。
但按照梅斯纳的观察，他的父亲特别排斥示弱。即便身体不适，他的父亲也会强撑，最终错过就医的最佳时机。
再比如，《健康心理学杂志》中的一项研究发现，男性在面对身体不适时，往往不愿寻求帮助，特别是当健康问题与“脆弱”相关时。
加拿大学者，也是畅销书《人生的12法则》的作者乔丹·皮特逊认为，假如缺乏积极的男性榜样和明确的价值观念，可能导致年轻男性会寻求极端或不健康的男性形象。说白了，假如缺少正确的榜样和积极的引导，男孩子可能会本能地去模仿那些极端的，甚至不健康的所谓硬汉形象。
比如，这两年国外有一个很火的健身网红，叫做做安德鲁·泰特。截止到2024年7月，安德鲁·泰特光是在X平台上就有950多万粉丝。但注意，泰特并不是一个积极正面的健身博主。2022年，泰特因涉嫌人口贩卖、组织犯罪等多项罪名被逮捕，2023年他从被拘留转为软禁，目前案件仍在持续调查当中。
但就是这么一个绝对算不上正面的形象，根据伦敦国王学院的调查，在18到29岁的男性受访者中，居然每5个人里有1个认同泰特的价值观念。因为泰特在社交平台上经常叫嚣，“男人的生活就是一场战争”“男人的成长就是要去征服”等。
英国“进步男性气质”项目负责人麦克·尼克逊就说，泰特的崛起，意味着传统刻板的大男子主义思想，在西方新一代年轻男性群体身上有复苏的迹象。
换句话说，健身博主突发意外这个现象背后，除了健身这个赛道本身带来的风险之外，很多人可能在无意中被卷入了一个比拼男性气概的漩涡里。
假如我们能从这件事里获得什么启发，我觉得可能是，人格模板的重要性。也就是，人的很多行为，不是经过深思熟虑的结果，而是他一开始就选择了一个人格模板，然后无意识模仿的结果。
尤瓦尔·赫拉利曾经在《未来简史》里说，读历史的意义不在了解过去，而是让此刻的我们更加自由。从这个角度看，读历史也好，看名人传记也罢，又或者多交几个朋友，本质上都是在搜集更多的人格模板。可供参考的价值观变多，才不至于被单一的价值判定裹挟，这或许就是赫拉利所说的自由。
再来看今天的第二个话题。眼看着快要过年了，假如你要出去逛街，那么接下来的内容建议你听听。我们要说的是，街边的商店，是怎么通过设计快速获得你的好感的？我们选取的案例，是苹果直营店。
苹果直营店是全球坪效最高的门店之一。根据eMarketer的报道，2017年，苹果零售店以每平方英尺5546美元的销售额，超过Tiffany和Lululemon，成为全球坪效最高的门店。
那么，苹果直营店的门店设计，有哪些要点呢？前段时间，我看到b站UP主“本子在隔壁”专门做了一期视频，内容很有意思。
第一个细节是，苹果店的笔记本电脑都是呈76度打开的。这样的开合角度并不是看屏幕的最佳视角。没错，苹果要的就是这个效果。因为当你看不清屏幕时，会很自然地用手去调节屏幕的角度。而这就是门店的最终目的，让你用手去触碰产品。
一旦你触碰产品，就能最直观地感觉到产品的材料、厚度、手感等细节，这样就会增加购买的可能性。而且据说，人一旦对某个事物投入时间、精力以及感官接触后，就会产生或多少的“拥有感”。而为了维护这种拥有的感觉，苹果的员工手册还明确规定，当顾客在试用某些产品时，员工不能主动去触碰这些产品。因为一旦触碰了，就会打碎这种拥有感。
苹果门店的第二个细节是，要求员工不能对顾客使用负面性的词汇。比如，不能说电脑用久了会发烫，而是要说可能有点暖和。再比如，系统有时候会崩溃，要换成系统偶尔会停止响应。再比如，当顾客说错苹果产品的名字时，员工不能纠正等等。
为什么要这么做呢？原因是，人对词语的敏感度是非常高的。比如，在20世纪70年代，美国认知心理学家伊丽莎白·洛夫特斯做了一次实验。实验的过程是，同时给两组人看同一段车祸视频，然后让参与者估算车辆的速度。唯一的差别是两组人被提问的话术。
比如，有一组问的是，你看到这两辆车“相撞”了吗？而另一组问的是，你看到两辆车“接触”了吗？结果发现，听到“相撞”这个词的参与者推测的车速明显高于听到“接触”这个词的参与者。说白了，词语本身会影响人的判断力。
另外，假如顾客对产品提出负面的评价，苹果也不提倡员工道歉，而是用3F工作法。“3F工作法”实际上就是feel、felt、found三个单词的缩写。
比如顾客说这个产品的价格有点贵。那么正确的说法是，我特别懂你的感受，我也觉得价格有点高，但是我在使用时发现某些功能特别好，最后发现挺值的。说白了，就是先接住顾客的想法，然后再把话题引向更积极的方向，最后让顾客感觉到舒适。
苹果直营店的第三个细节是，门店里不设置固定的收银台。没错，当你在苹果直营门店结账时，工作人员会拿着新包装的产品和刷卡机走向你，而不是你专门去收银台排队结账。背后的原理，其实就是“支付痛苦”。也就是，人在消费过程当中，会因为支付金额、支付方式等因素感到心理上的痛苦。
而去收银台排队时，你會最大限度地感受到支付痛苦。多数人在排队时，满脑子想的都会是钱的问题。这可能在一定程度上，降低顾客的购买意愿。
好，咱们刚才从苹果门店的设计，说到了好感公式。简单说，获得他人好感有两个方法，第一是多接触，多吸引别人投入。第二是多使用积极的词，多表达积极的看法。
最后，总结一下，今天说了两个话题。
第一，健身博主为什么会频频遭遇健康意外？首先，健身博主是最容易产生内卷的赛道之一。其次，当事人也可能受到社会叙事的裹挟，不自觉地忽略健康问题。
第二，苹果直营店是怎么获取用户好感的？我们说了几个设计要点。包括吸引用户接触产品、对产品产生拥有感，以及尽可能使用积极词汇，让用户保持好的情绪体验等。
