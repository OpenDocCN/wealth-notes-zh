
# “垃圾荒”为什么是好事（下）

## 碳捕集降本

2025年7月，中国科协开年会发布了2025年中国三组科技难题，分别是十大前沿科学问题、十大工程技术难题和十大产业技术问题。其中，“新一代低能耗低成本碳捕集与封存技术”被列在了十大工程技术难题里。

碳捕集技术，简称CCS或CCUS，就是把工业流程当中产生的二氧化碳封存在枯竭的油田里，让森林吸收，或者转制成甲醇，总之，用各种手段阻止它们进入大气。这种技术手段能让人类在推进碳中和的同时，又不耽误发展。

中国在2020年宣布碳达峰、碳中和目标，随后就开始密集地用补贴和政策来培育产业。到了2024年，中国已经成了全球碳捕集产业链里最有存在感的国家，碳捕集相关专利数量有1万多项，排在世界首位，占全世界总量的接近一半。

中国在当前的碳捕集降本什么概念呢？比如，有种成熟技术叫胺吸收法，2021年应用在国内一些碳捕集项目里时，连投资带运行的全流程成本是650元/吨。而到了2025年的中石化齐鲁项目中，成本就已经降到300元以内了。而借助其他一些技术路线，比如利用新疆、甘肃的枯竭油田来封存二氧化碳，成本甚至有望压到80块钱以内。

中国2023年推出了全国碳排放权交易市场，碳排放权限的价格从最初的每吨50块钱，逐渐涨到了2025年的70多块钱。所以，碳捕集行业还有个乐观的预计，认为如果在2027-2028年，国内碳价能突破150元/吨，而捕集成本降到200元/吨以内，整个行业就可以实现无补贴市场化运转了。

## 二氧化碳转能源

第二个新闻，7月初，央视三农频道说，中科院的两个研究所——大连化物所和天津工生所，合作干成了一件大事：用人工技术成功把二氧化碳转化成了白糖。

这个消息发出后，很多广西IP的老表在新闻下面留言说：“那我们以后种什么？”你肯定知道，广西盛产甘蔗，也是中国最大的蔗糖生产基地。

这个技术最惊艳的突破，是中科院的大连化物所和天津工生所联手模拟出人工光合作用，一家把二氧化碳变成甲醇，一家把甲醇变成白糖。这套技术，能量转化效率比植物自身还高。

而其中的前半段，也就是二氧化碳变甲醇的技术，成本已经降下来了，所以目前内蒙古、新疆、青海、甘肃等省份，都已经投产了10万吨量级的生
产项目。为什么在这些省份？因为这些都是光伏产业大省。把太阳能转化成甲醇，等于是把阳光变成了可存储、可运输的液体。

而甲醇变白糖的技术成本，当前还比较高，所以天津工生所下个阶段的攻坚任务也是降本。

## “碳再生”产业链

到了这里，你肯定猜出我想说什么了：

今天的垃圾焚烧产业，有机会从第一阶段的热能利用，转向第二阶段的碳资源提取，它跟我们这一讲提到的碳捕集和人造糖蛋白，正好可以串起一条“碳再生”产业链。

在这条产业链里，人消耗养料来维持生命运转，同时制造垃圾和二氧化碳；垃圾焚烧产业则解除了固废物对碳元素的“封印”；碳捕集产业重新归拢碳资源；碳转能源的技术把碳资源再次变成维持生命运转的养料。

只要产业成本能压到足够低，垃圾焚烧产业所排放的二氧化碳，就能被捕集、封存，再转化成蔗糖、淀粉、蛋白质，让碳元素的价值循环起来。

蔗糖、淀粉、蛋白质是什么？是人体最需要的养分，碳基生命运行所需要的基础能量。

我们再借用硅基视角审视这件事。你有没有发现，垃圾焚烧这个环节，不再是“消灭垃圾”而已。它作为碳再生产业链上的一环，是在释放、激活被封存的碳元素，把它们重新变回物质世界的“自由Token”，再交给碳再生产业链重新编程，用碳Token生成碳基养料，让它再度支撑我们这些碳基物种的生命循环。

这句话很重要，我再重复一遍：碳再生产业链，是在把碳元素定义为物质世界的Token，优化碳基物种的生命循环。

这是什么？难道不是我们碳基物种，在宇宙系统里，尝试修改自己的生命循环局限，掌握自己的命运吗？

这条产业链如果拉通，意味着人类在驾驭自己生命循环的工业进程上，迈进了一大步，从“碳耗散”迈向了“碳闭环”。

而在规模与工程中把成本降下来，恰好是我们所处这个产能大国的强项。

在这种循环里，假如1吨二氧化碳固定量，可以对应换取1份确定的糖与蛋白养分保障，这就相当于把二氧化碳，变成了类似黄金、电力的“价值通货”。

所以你看，“垃圾荒”不只是一个环保现象，不只该引发“盲目扩张”的指责。它也可能是一个信号，在追问所有人：你认不认同这是经济、甚至人类文明的新基本面？这条科技树枝干，值不值得你押个重注？

到了这里我的答案，仍然基于上一讲的那句话：如果一种浪费能换来大规模的创新与发展，那么浪费就值得承受。不过，又可以再补上一句：何况这次这个尺度，有碳基生命循环那么大。

## 碳再生的议题意义

当然，“碳再生”产业链真要形成落地闭环，远远没有我们描绘得这么简单，中间还有各种工程难点需要衔接与突破，说不定那个堵点就把整条路堵死了。

但即便如此，我还是想认为，我们花资源去搞垃圾焚烧与碳再生产业链，有重大的意义。

有个词叫“议题设置”，英文叫Agenda Setting，在这两年的认知战里经常出现。它指的是媒体通过关注特定议题，来引导公众的注意力。

这些年，议题设置又被扩展成了一种能力和权力，指的是权力者通过设置议题，来影响社会资源分配，与民众认知的优先级排序。

也就是说，议题设置之所以重要，是因为它决定了谁能获得关注、投资和人才。而在一个需要争夺资源的世界里，什么问题被认为值得解决，本身就是一种制度性选择。

垃圾焚烧和它背后的碳再生，是不是一个好议题？简直不能再是了。

在今天增速放缓的社会里，它给人才、资金、资源提供了一个有盼头的新的去向，给了社会一个配置资源的“新锚点”。无论它有多远、能不能抵达，都有助于我们跳出内卷。

垃圾焚烧、碳捕集、人工糖蛋白，这些议题表面土气，但都有现实刚需；它们都是真问题，能分发给工程、化学、生物、城市治理等各种专业和岗位；它们的技术还可检验，数据差就淘汰，没有刷榜空间。

研究稳定币当然也很重要。稳定币事关人类权力的再分配、全球金融架构的重新博弈。但垃圾焚烧和碳再生，事关物质世界的重构，是在寻求让文明运行更低耗、更可持续，也更贴近日常、更让人广泛受益。如果全球1.5%的电力可以用来支撑加密货币产业，也可以用来解决2亿人的蛋白质缺口，选哪个更接近“历史正确”？
