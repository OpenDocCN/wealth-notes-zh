## 社交校准

新手在聊天中容易忽略一个东西，就是所谓的平衡。很多时候没注意平衡，而你已经在跪

在你们两个人之间。聊你聊多了，妹子会觉得“关我屁事”。聊她聊多了，她会觉得你在跪舔。
把握住一个度是很重要的。
3. 有趣和正经的平衡：这个理解起来比较简单，就像吃饭一样。光吃白米饭会索然无味，光
吃菜， 味道好，肚子又吃不饱。聊天一直正经，妹子会觉得跟你聊天无趣，一直逗，又觉得你
是个逗逼。我们就根据妹子的反应，自己的感觉，来把握这个度。

这个也是需要校准的。
4. 可得性和不可得性的平衡：说白了就是推拉，不能只推不拉，也不能只拉不拉。不能一
