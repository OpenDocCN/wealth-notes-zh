
# 猫哥5.10石油的逻辑

4月2日特朗普对全球挑起关税战以来，国际石油价格连续暴跌，从每桶70多美元的价格跌到60多美元附近震荡。

那么国际油价暴跌的原因是什么呢？互联网上有两个说法。

说法一，这是沙特配合特朗普将国际油价打下来，帮助特朗普控制国内通胀，毕竟油价对美国通胀指数影响很大。

说法二，这是沙特与俄罗斯默契搞美国页岩油产业，国际油价打下来，美国页岩油产业将受重创。

这两个说法哪一个正确？

实事求是的说，哪一个都不正确。为什么？讲一讲国际油价波动的底层逻辑。首先我们可以确认一个基本观点，沙特主导的欧佩克对国际石油价格具有巨大的影响力。

那么问题来了。欧佩克成员只是涵盖了全球部分产油国，国际上还有相当一批产油国并不是欧佩克成员，为什么这个只是涵盖了部分产油国的欧佩克组织能主导国际油价？

第二个问题更关键，欧佩克究竟有没有能力对成员国行为进行约束？把这两个问题搞清楚了，大家就对国际油价波动的底层逻辑就搞清楚了。

欧佩克组织的老是谁？

当然是沙特。沙特为什么能够成为老大？原因无它，沙特开采石油成本最低!这是一个最最关键的因素。

因为沙特开采石油成本最低，所以导致沙特对欧佩克甚至全球其他产油国都具有直接的威慑力。我们以本次国际石油价格剧烈波动为例，看看沙特的能量。

本次国际石油价格波动根本原因既不是沙特配合特朗普控制通胀，也不是与俄罗斯一起去搞美国页岩油产业，而是惩罚欧佩克组织+某些成员国未能按照配额生产的违规行为。

欧佩克+组织内部是有协议的，这个协议就是会不定期开会，约定每个成员国的生产配额。这个配额的分配就是直接的利益分配，因为最近几年国际经济形势大环境不好，所以大家对于这个配额分配争夺很激烈，因为对配额分配不满，包括卡塔尔、厄瓜多尔、安哥拉就先后退出了欧佩克组织。

但是即便退出欧佩克组织并不意味着就可以彻底放飞自我，想生产多少就生产多少，这个大家看到后面就清楚了。

还有一些国家表面上不退出，口头上表示“服从组织安排”按照分配的配额生产，但是私底下却偷偷增产。比如最近的哈萨克斯坦，按照协议，3月份起生产石油的配额是146.8万桶每天，但是实际呢？产量达到了180万桶每天，这不仅严重违反了配额协议，同时也对欧佩克组织权威的挑衅。

这种挑衅沙特就必须出面惩罚，否则大家都不按协议生产，欧佩克组织就形同虚设。

欧佩克组织对于小弟这种乱来的行为通常有两种惩罚方式：

第一种是补偿性减产。具体到哈萨克斯坦就是你每天增产了33万桶每天，那么之后就要主动减产33万桶每天，来平衡你之前超出的份额。

如果哈萨克斯坦不愿主动减产怎么办？那么沙特将使出最后的杀手锏就是率领欧佩克组织成员国一起增产石油，让国际油价暴跌，一直暴跌到哈萨克斯坦开采石油成本价以下，让哈萨克斯坦增产石油不但无利可图，甚至还会大幅度亏损。

现在我们回顾一下从4月2日开始国际石油价格剧烈波动的过程。首先是哈萨克斯坦被发现大幅度增产石油，沙特为首的欧佩克组织与它协商无果（要求补偿性减产无果），于是沙特率领欧佩克成员国主动增产石油，将国际油价一下子打到60美元以下（这基本是哈萨克斯坦开采石油的成本线）。

然后哈萨克斯坦迅速认怂，其能源部长表示，将严格遵循2025年份额执行生产。并且还是执行补偿性减产协议，于是沙特等国停止增产，石油价格立刻反弹到68美元。

这个时候哈萨克斯坦又错误判断形势——主要原因是特朗普上台要大力发展美国页岩油产业，哈萨克斯坦认为沙特忌惮特朗普不敢长时间将油价打到60美元以下（美国页岩油开采成本也要超过60美元)哈萨克斯坦决定偷鸡，于是换了一个能源部长上台，新上任的能源部长宣布，国家利益高于欧佩克组织利益，意思就是还会继续增产。

沙特一看气得吐血，于是又拉着几个成员国宣布6月将继续增产，这个决定让国际石油价格继续下跌，5月5日再次跌到60美元以下，于是哈萨克斯坦第二次认怂，宣布将执行减产协议。于是国际油价再次反弹，目前在62美元左右徘徊。

从这个案例可以看出，国际石油价格归根结底还是开采成本最低的国家决定的，不要说欧佩克组织成员国国，就算是退出欧佩克组织的产油国也不得不看以沙特为首的欧佩克组织的脸色，你不听招呼，那么我们就一起增产，把石油价格打到你成本线以下——我还能赚钱，你就只能亏损了。

另外，沙特这个国家其实在地缘政治上没有多大的诉求，对于沙特而言，经济利益也就是石油利益才是放到第一位的，谁不听招呼动摇欧佩克组织的权威性，沙特就会毫不犹豫的出手，谁来说话都不好使。
