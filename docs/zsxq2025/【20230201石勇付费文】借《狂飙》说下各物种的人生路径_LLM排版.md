
# 借《狂飙》说下各物种的人生路径
原创 I MC石勇 石勇谈心理 2023-02-01 1 5:05 发表于广西

这段时间我一直在看电视剧《狂飙》。
有很多很多人也在追这部剧。
听说《孙子兵法》被带火了。
我看见网上有些人,当然是弱6、强7低扭群体,概括出了一条成功路径:当鱼贩,娶寡妇,
拜干爹。
这当然是半真半假。
这些低扭想和说的任何事情,你都不要当真,也不要全不当真。
比如他们嚷嚷说“躺平”,你就不要当真,他们真的可以做三和大神吗?深圳三和市场和广
州黄埔东区,还有很多大神的位置虚位以待,他们真的去吗?
不可能滴。
真正选择做三和大神的主体,弱7们,压根不可能嚷嚷。
但你也不要完全不当真。
因为他们确实在找各种理由尽力阻止自己“努力”。
又比如说“不恋爱不结婚”。这帮人一边这样说,一边身体又很诚实地相亲。
但也不能完全不当真,因为他们恋爱结婚的热情确实在衰减。
低扭,尤其是强7、弱6这个段位的低扭,心智和人格是分离的,心智-人格和身体是分离的。
他们处于“上不去,下不来”的状态。
基本心智模式是:以做梦、戏谑、踩人等方式来打掩护,说出自己内心的真实想法。
所以,真让他们行动起来,走高启强“当鱼贩,娶寡妇,拜干爹”的成功三步曲,他们知道
这是不可能的。

在社会食物链上,各物种在选择自己的人生路径时,我不作价值判断,而只作认知判断。
鱼有鱼路,虾有虾路。如果路能走,走上去,我尊重它的技术含量。
比如,一个人如果通过吹牛拍马,攀上某个领导,然后爬了上去,这也是本事 。原来我有篇
文章的标题直接就叫“那些嘴巴甜的人混得好,其实很公平”,够直白了。
正常羊有正常羊的人生路径,高狼有狼的人生路径,扭曲羊有扭曲羊的人生路径,低狼有低
狼的人生路径。
看到一群低扭,想走低狼的人生路径。
低狼的成功路径就是:以低狼的方式打出一个食物链,然后,努力洗白变成高狼。
如果打不出食物链,或无法洗白变成高狼,或洗白之前就被打掉了,都是失败的。
这条成功路径,技术含量其实非常高。只说两点。
1)得有强大的格商、体商,话多不多无所谓,但人必须狠
你看人家高启强,被徐江酒瓶子爆头时,都显得像是没事一样,直到走出门没人看见之后,
才摸着头嚎叫。
黑道大佬徐江也是如此。被高启强先用烟灰缸爆头时,显得什么事也没有,还潇洒地玩了脱
手表,拿酒瓶先把酒倒出来,再去爆高启强的头的漂亮动作。同样是没人看见之后,才显示
出痛苦。
这种超强的格商,这种hoId住痛苦和情绪的“内在实力”,哪个低扭具备?

一个读了个破大学,就觉得全世界欠他一堆钱的人,一个喜欢在网上嚷嚷,玩各种心理寄生
和投射的人,听得懂这种实力?
不是那块料,就走不了那条路。
2)看社会游戏规则
有的社会,低狼打出食物链成功后,存在一定的洗白机制,欧美日本甚至香港都有。因为在
总的社会游戏规则下,白道并没有完全占据所有的社会治理空间,低狼既可以有机会去提供
一个“秩序”,也可以通过基层的各种选举机制等来洗白。
所以我们可以长期地听到什么黑手党以及山口组。
但在中国大陆,你能长期听到有一个什么黑帮存在吗?
当然不排除会有低狼成功后洗白了,但他也明白,自己过去所做的很多事,随时都有可能爆
雷。
所有的人生路径,本质都是:获取生存优势。
问题只在于:能不能选择,能不能获取生存优势,以及,获取之后,拉长时间线,能否维持
而不自我挫败。
从这个角度看,在当下中国,人生路径的优劣一目了然:狼>高正>中正>高扭低扭>低正
>低认>低狼
低狼虽然可以让人怕,虽然也可以掠夺到很多食物,甚至还可以成功,但它既不具备持续性,又风险极大,在生存优势上其实是垫底的。
我承认狼的生存优势大于高正,社会食物链的运作决定了,这很痛苦。但认知就是认知。
高扭以下各物种的人生路径,其实都不属于具有生存优势的路径。
没办法,基本上由内在实力的性质和水平决定。
性质就是:有些人就是坏,而有些人真的坏不起来;有些人,突破真自我时障碍不大,而有些人,却突破不了。
水平是:有些人,以他的内在实力的水平和潜能,真的没办法从低正向中高正进化,或真的
没办法从低扭向高扭、狼进化。
所以,当我要问一个其物种具有多种进化或退化可能的低正选什么人生路径时,我会不作价
值判断地说:“如果你觉得自己变坏,玩各种忘恩负义两面三刀没有底线算计攻击,可以让
你阶层攀爬方面的优势,那是你的自由。但如果你做不到扼杀真自我,或没有在这条路径上
进行阶层攀爬的能耐,那还是老老实实地选择从低正进化到中高正的路径——这说明你的心
智—人格配置,其性质和水平,更适合正常羊的人生路径。你不过是还没有把正常羊潜藏的
那些能耐刺激出来,做出来而已。”
选定了人生路径之后,才是价值判断。
就像高启强绝对不允许他的下属,以及他弟高启盛碰毒品一样。
因为一旦他的下属和他弟,碰了这个极高利润而又极高风险的东西,不要说枪毙几次都不
够,首先,高启强的组织结构就无法维持,食物链就塌了。
价值判断决定了是在挖自己内在实力和社会实力的墙角,还是增长它们。
狼和高扭,一看到真自我的东西就怕,就恨,而千方百计要去找假自我的心法 ,就是价值判
断。
他们怕真自我的东西挖自己的墙壁,进而引发内在实力和社会实力体系的崩溃。
同样,中高正不管有没有找到真自我的心法,对假自我的东西都嗤之以鼻,也是怕它挖自己的
墙角。
能走中高正、狼-高扭这样的人生路径的人,明显的特征是:心智-人格实力的水平(且不管性
质)处于社会平均线之上。
社会平均线中的心智线是弱6,格商线是强6。
平均线内或之下,那就只能选择后面的生存路径了。
原来一个人的心智—人格处于什么水平,并不重要。因为心智—人格是可以提升 的。
但我看人,有一个粗暴的标准 :
头脑的水平提升的时候,心理、人格有水平有没有同步提升,至少不呆在原地?
如果头脑水平提升了,心理、人格没什么变化,一个人基本很难达到平均线之上。他这一生
的阶层,凭自己的话,弱6 到顶了。
像下面这个,即使在一个较痛苦的自我体验里,在一个有价值信息冲击的环境里,他头脑、
心理、人格,在实力层面,动都不动一下。

也是他的处境的原因。
