
# 国家发改委等两部门印发政务领域AI大模型部署应用指引，机构称大模型的赋能下软件业正加速演进

机构称大模型的赋能下软件业正加速演进。近日，国家发改委等两部门联合印发了《政务领域人工智能大模型部署应用指引（试行）》（以下简称《指引》），为政务领域人工智能大模型的健康有序发展提供了政策指导。

《指引》明确了政务领域人工智能大模型的应用范围、技术要求、安全规范等内容，旨在推动政务数据与人工智能技术的深度融合，提升政府治理能力和公共服务水平。根据《指引》，政务领域人工智能大模型的应用应遵循“积极稳妥、安全可控、注重实效”的原则，确保数据安全和个人隐私保护。

业内人士表示，随着《指引》的发布，政务领域对人工智能大模型的需求将进一步提升，相关软件企业将迎来新的发展机遇。大模型的应用将有助于优化政府决策流程，提高行政效率，推动政务服务智能化升级。

此次《指引》的出台，标志着我国在人工智能领域又迈出了重要一步。未来，随着技术的不断进步和应用场景的不断拓展，人工智能将在更多领域发挥重要作用，为经济社会发展注入新动能。
