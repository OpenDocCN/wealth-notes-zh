
# 卢克文251128：三体CEO毒杀董事长，林奇案始末

2006年，刘慈欣撰写的《三体》开始在《科幻世界》连载，一路蹿红。但红归红，在那个年代，科幻是小众圈子，变现的路径窄得可怜，刘慈欣也没指望有人会拿真金白银来买三体的改编权。一直到2009年，两个人找来了——宋春雨和张番番夫妇。宋春雨是个编剧，张番番是个名不见经传的三流导演。俩人业务水平很一般，眼光却是顶级的，他们读完《三体》后，意识到这是一个巨大的金矿。

此时，刘慈欣的单位娘子关火电厂因响应“节能减排”政策，彻底关停，刘慈欣对未来感到迷茫。卖掉三体的改编权，刘慈欣就能凑够在阳泉买房子的钱，算是求之不得了。宋春雨和张番番夫妇花了多少钱呢？之前有传言说是10万，后来被刘慈欣否认了，但最多也不超过30万。拿到版权后，张番番夫妇注册了一家公司，叫“百星社”影视工作室，并在国家电影局立项了三体电影，开始四处拉投资开拍。

拉投资就拉投资，但他们的条件很苛刻，要求导演必须是张番番自己，编剧必须是他老婆宋春雨。大佬文集圈问题在于，他俩之前压根没搞过什么大电影，就拍过几个小成本恐怖片，豆瓣最高分才5.7分。这样的水平，谁能相信？找不来投资，张番番也不急，而是把三体“捂”了起来，他相信三体的改编权会越来越值钱。这对张番番是好事，但对三体而言是个灾难，让三体被雪藏了整整6年。

时间证明了张番番夫妇的眼光，2014年，《三体》英文版通过美国托尔出版公司发行，火爆全球，2015年，《三体》获得雨果奖最佳长篇小说奖。至此，三体彻底火了。而这个时候，一个愿意接受张番番夫妇条件的人终于出现了——林奇。林奇1981年出生，是典型的温州商人，身上带着温州人那股敢想敢干，甚至有点赌徒性质的狠劲。他早年创业，靠着页游起家，创办了游族网络，并很快踩中风口，火爆一时，不到35岁的时候就已经身家几十亿了，还登上了福布斯富豪榜。

但林奇不仅仅想做个游戏老板，他是一个有野心的人，也是一个真正的《三体》迷。他的梦想，是把三体打造成中国的“漫威宇宙”，做一个跨越游戏、电影、动漫的庞大娱乐帝国。2014年，林奇拉上了之前在酒局上结识的作家孔二狗，成立了游族影业，当时刚从小马奔腾辞职的孔二狗也是个三体迷，二人一拍即合，林奇任命孔二狗担任游族影业CEO，决定投拍三体。

他们俩一起去找了张番番，但是说破嘴皮子，张番番夫妇仍然坚持可以投资，导演和编剧必须他俩干。林奇和孔二狗没办法，只能答应。2015年，三体电影正式官宣，按照游族影业的说法，《三体》电影投资超2亿，由张静初演叶文洁，冯绍峰演汪淼，之后还会围绕《三体》这一系列陆续推出6部作品。然而，张番番不具备驾驭这种科幻巨制的能力，开拍后，片场管理混乱，拍摄素材质量极差，据当时的工作人员回忆，导演连基本的绿幕拍摄逻辑都搞不清楚，特效团队拿到的素材根本没法用。但机都开了，只能硬着头皮拍。到了2015年7月底，实拍部分全部杀青，进入后期制作阶段，并计划2016年7月上映。

电影粗剪完成后，林奇满怀期待地组织了一次内部试映。结果是毁灭性的，大部分人看了后，对电影质量非常无语，一位非常喜欢《三体》的副国级领导，也专门跑去看了样片，看完之后只说了一句话：“要把《三体》制作成一部高质量的电影，要跻身世界顶尖水平。”领导说话就是艺术啊，什么都没说，但也什么都说 了：三体是中国的骄傲，但这样质量的电影，放出去了就是丢人。就这样，电影《三体》烂尾，孔二狗黯然离职。

对于林奇来说，三体电影的失败，不仅是几亿资金打水漂的问题，更是他“影游联动”战略的全面崩盘。那咋办？不能死磕电影了，要另想办法赚钱。这个时候，光线传媒也入局了，想和游族一起，把三体打造成一个集合剧集、动漫、电影和漫画的超级IP宇宙。但每次谈这个事，都会卡在张番番这里，不管什么都要求自己当导演。林奇最后忍无可忍，做了一个决定：买下三体版权！张番番一开始开价1个亿，林奇答应了，但过两天张番番又觉得要价太低了，又开价到2个亿。林奇火了，怎么也说不动张番番，他需要一个助手，帮他解开这个死结。

就在这个时候，许垚登场了。许垚，1981年出生，和林奇同龄。许垚履历光鲜，美国密歇根大学法学博士，回国后加入了复星集团，担任法律顾问，经手过无数跨国并购大案，专业能力极强。2017年，急于解决《三体》版权问题的林奇，经人介绍认识了许垚，两人一拍即合。林奇给许垚开出了高达2000万的年薪，主要任务就是啃下张番番夫妇这块硬骨头。

这是一场极其艰难的谈判。张番番夫妇虽然电影拍砸了，但他们知道手里的版权更值钱了，因为2017年，奥巴马专门会见刘慈欣，甚至好莱坞也在和他们接洽，谈购买三体版权的事情。这时候，许垚展现出了他作为顶级律师的手段。他花了整整大半年的时间，和张番番夫妇进行拉锯战。没有人知道具体的谈判细节，但结果是：2018年，游族网络以1.2亿元人民币的价格，彻底收购了“百星社”（张番番夫妇的公司，拥有《三体》改编权）。

游族终于拿到了《三体》改编权。有了改编权，可以放开手脚干了。2018年12月，林奇成立了“三体宇宙（上海）文化发展有限公司”，专门负责《三体》IP的开发。林奇是董事长，许垚是CEO。这是两人关系的蜜月期。在许垚的运作下，三体宇宙迅速走上正轨：B站的动画版、腾讯的电视剧版，甚至奈飞的海外版权合作，都是在这个时期敲定的。可以说，如果没有许垚，就没有后来《三体》IP的全球爆发。许垚自认为，他是游族的大功臣，是《三体》宇宙的救世主。

功臣应该得到封赏。许垚想要的，不仅仅是高薪，还有三体宇宙的股权，成为合伙人，而不是一个高级打工仔。但裂痕就在这个时候产生了。在林奇看来，我给你开2000万年薪，已经是对你工作的最大认可了。股权？那是公司的核心资产，怎么能轻易给人？更重要的是，林奇对许垚的能力产生了怀疑。许垚虽然搞定了版权，但在公司经营上，他没有展现出太强的天赋。三体宇宙成立后，前期一直在烧钱，并没有带来立竿见影的收益。

2019年年底，许垚曾向林奇汇报三体公司财务情况，林奇很不满意，连年终奖方案都没批。到了2020年，二人矛盾开始公开化。林奇开始对公司架构进行大调整，他要求三体公司所有事项，都要向外来户赵宇尧汇报，还收回了三体宇宙公司的公章和财务U盾。这就意味着，许垚这个CEO，已经被彻底架空。最让许垚无法忍受的羞辱是，林奇以公司经营不善为由，大幅削减了许垚的薪水。从2000万降到了500万。许垚崩溃了。

他曾经以为自己是诸葛亮，辅佐刘备打天下，将来是要封侯拜相的。结果现在天下刚打下来一半，林奇就要把他踢开，甚至还要羞辱他一番。最终，许垚决定报复。但许垚不是街头流氓，他不动刀，不拿枪。他是一个精通法律、智商超群的精英。他要用一种神不知鬼不觉的方式，让林奇消失，同时让自己全身而退。他想到了毒。

这绝对是中国商业史上最魔幻、最恐怖的情节之一。许垚通过伪造印章、租用库房等方式，冒用大学医学部及生物技术公司的身份，耗资数十万元，在暗网上购买了多种毒素（毒药），并在自己租用的库房里搭建了一个生化实验室，进行各种毒药的调配、试验。为啥要这么干？因为直接购买现成的毒药容易留下购买记录，并引火烧身。为了确保毒药的致死率和隐蔽性，许垚表现出令人发指的冷静和残忍。他先买了猫和狗，在实验室里进行活体实验，观察猫狗中毒后的反应、死亡时间，并详细记录数据，不断调整配方。他要调配出一种毒药：无色无味，服用后不会立即死亡，而是像生病一样慢慢恶化，让医生查不出原因，最后死于多器官衰竭。

经过无数次实验，他成功了。接下来，就是投毒。他先是对和自己有竞争关系的赵宇尧（林奇看中的继任者）、赵骥龙（三体公司副总裁），以及赵骥龙的妻子张女士，还有一位宋姓客户投放了“氯化甲基汞”，准备让他们慢慢毒发。但到了2020年12月，许垚突然得知了林奇准备安排他离职的消息，如果他离职了，那就没办法再投毒了。于是，他决定对林奇投放一种见效快的毒药。一次偶然的机会，他把一种见效快的剧毒封装进了是一款名为“培菲康”的益生菌药丸里，推荐给了林奇。林奇不疑有诈，吃了下去。

2020年12月16日，林奇毒发，短短两天内，就陷入了昏迷，医生经过检查觉得，这不是普通的病，这是中毒。这时候，一个关键的转折点出现了。陪同林奇就医的同事，想到了许垚。因为林奇和许垚的矛盾尽人皆知，也只有许垚送过那些奇怪的“益生菌”。这位同事赶紧报了警。12月18日，上海警方迅速行动，在许垚的住所和办公室内将他抓获。

在搜查过程中，警方被眼前的景象震惊了。在许垚的秘密实验室里，发现了大量的剧毒化学品、实验设备，以及那些还没来得及销毁的“氯化甲基汞”“马兜铃酸”“氟乙酰胺”“氯化镉”“四乙基铅”、河豚毒素等等毒药。然而，被捕后的许垚，展现出了极高的“反侦查能力”和法律素养。他坚持“零口供”，不管警方怎么审讯，他就是不承认杀人。他不交代毒药的具体成分，不交代投毒的过程。他知道，只要他不开口，医生就很难对症下药，林奇死亡的概率就越大。只要林奇死了，死无对证，他就有翻盘的机会。

2020年12月25日，圣诞节，林奇毒发身亡，享年39岁。林奇死后，许垚并没有表现出任何悔意。他不仅拒不认罪，还提前准备好了精神病鉴定的材料，试图把自己包装成一个有严重精神障碍的人，以此来逃避法律的制裁。毕竟，他是法律专业的嘛。所以，这起杀人案的审理过程，极其漫长且艰难，检方必须形成一个完整的、无懈可击的证据链。从毒药的购买记录、实验室的痕迹、猫狗尸体的检出成分，到林奇体内的毒素比对，每一个环节都不能出错。

最终，2024年3月，上海市第一中级人民法院终于作出一审判决：许垚犯故意杀人罪、投放危险物质罪，判处死刑，剥夺政治权利终身。许垚当然不服，他当庭表示上诉。他依然觉得自己能活，他觉得证据链还有漏洞，他觉得自己的精神鉴定还有文章可做。但是，法律不会放过每一个杀人犯。

时间终于来到了2025年11月，二审裁定下来了：驳回上诉，维持原判。这意味着，许垚的生命已经进入了倒计时，最高法的死刑复核一旦通过，这名曾经的精英律师，将面对法律的严惩。故事讲到这里，我们的心情是沉重的。《三体》中有个著名的“黑暗森林法则”。谁能想到，在《三体》这个IP的现实运作中，竟然上演了比小说还要黑暗、还要赤裸的“黑暗森林”故事。微信林奇和许垚，他们本都是这个时代的佼佼者。但最终，一个死于对他人的轻视和防备不足，一个死于自己的贪婪和狭隘。在巨大的利益面前，人性的贪婪、嫉妒、仇恨被无限放大。许垚懂法律，但他不懂敬畏。他以为凭借高智商可以策划一场完美犯罪，但他忘了，天网恢恢，疏而不漏。如今，斯人已逝。游族早已易主，《三体》的IP依然在全球火热，奈飞的剧集正在筹拍第二季，更多的衍生品正在路上。当全世界的观众在屏幕前为“水滴”摧毁人类舰队而震撼时，恐怕很少有人会想起，在那座繁华的大厦里，曾经发生过这样一起令人胆寒的投毒案。这起血案，终将尘埃落定。但它留给我们的思考，关于人性、关于资本、关于底线，或许永远不会结束。
