
# 关于美债到期的事

据说是6月有大几万亿的美债到期。但是细想想，这个可能不是什么大事，不会因为这大几万亿就导致美债崩盘，因为目前美债总规模超过36万亿，到期的可能只有不到1/4的规模，剩下3/4的规模不可能只在老美自己人手里，按照概率来说，肯定是均匀分布，就是目前持有大量到期美债的人，手上肯定还持有大量没到期的美债。

所以本质上，大家都是一条船上的人。最终无非就是当做谈判筹码，谈点好的条件出来。（甚至于说，如果有很好的条件的话，我们国家也不可能100％拒绝……）

所以想来想去，6月底那个美债集中到期的事，应该不是什么大事。

有可能就是债券利率更高点作为补偿，外加些谈判进展推进了。（但是普通人不要图这点利率，毕竟美元还要继续贬值）
