
# 付费文：我们和美国，有个交易？ (1)

西方发达国家的市场，他们的购买力本身来说，就是建立在当前的这个国际体系之上的，如果东西方达成共识，要维护这个体系，那么很明显，不可能只是我们参与维护，却不参与利益的共享。

从这个角度来说，美国的关税战，很可能就是一对后面的经济市场进行次新的利益划分。而这个利益划分的前提，应该说就是默认了现行的国际贸易体系不能被颠覆，这也是我们快速与美国达成新关税协议的基础。

另一个是金融话语权。现行的国际金融体系，都是以美元为核心，如果我们依然共同维护以美元为核心的国际金融体系，那么美国必须在这个体系对我们进行进金融话语权的让渡。这个事情相对来说更加直接和容易量化。

如果说这个新的交易确乎存在，那么个最简单的参考标准，就是看看后面美国是否会在国际货币基金组织和世界银行，对投票权和特别提款权改革中，将我们的话语权，也就是人民币的权重大幅提高。旦这种实质性的改变出现，那很有可能我所说的这个交易，应该就提前已经在暗中成为了现实。

除此之外，甚至还有可能对国际支付结算体系进行相应的改革，以提升我们在其中的话语权，应该说这种话语权的提升本身，就是对当前国际支付体系进行加固的举措。也是各方共同维护现行国际秩序的一个外在体现。

我所说的这些风向标，在后面如果逐渐成为现实，那么大概率的，这个交易可能就会实际的推进下去。反之，如果迟迟没有出现，则有可能还会出现很大的变数和反复。

最后对我们来说，在当前的这种国际形势之下，让自己立于不败之地，才有可能成为各方势力竞相争取的对象。而与此同时，我们还需要在战略层面保持极大的克制和定力。尤其是对美国和西方的一些承诺，要保持足够的清醒。

比如在当前的经济环境下，我们始终还保留若个重大的放水刺激的工具箱选择。这可以给全球经济提供重大的消费和需求刺激，将全球经济在短期内拉出泥潭。但是这也目前全球范围内，仅有的个最大的增量政策选择，旦我们做了之后，很可能就不再有其他的托底选项了。

但是在我们提供了这样的政策支持之后，美国如果缓过劲来了，会不会再次出尔反尔，这恐怕谁也没法保证，所以我们的政策选择必须慎之又慎。
