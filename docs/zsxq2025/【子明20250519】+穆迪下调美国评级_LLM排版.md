
# 穆迪下调美国评级

最大的政治变量，就是领导人的健康。领导人一旦展现出身体的问题，那么基于利益的本能驱动，内部各个山头都会谋求自立，外部各个敌人都会杀心自起。因此任何一个领导人，都会将自己的健康问题视为绝密，只有到最后时刻遮不住时，才会不得不宣布。反之，对一个领导人最好的攻击手段，也是针对他的健康问题做文章。

当年跑得快的西方记者最喜欢干的就是这个。拜登的癌症，是去年就已经查出来的，所以布林肯等拜登心腹一个个边缘化，对外，美国的政治压力骤然消失，让中国、俄罗斯等美国的敌人骤然缓过一口气。对内，民主党各个团伙开始了抢班夺权，内部相互攻伐乱成了一锅粥。也让特朗普能够以压倒性的优势横扫2024年大选。

穆迪下调美国评级，并不是一场突袭，而是一场人情世故。美国的信用持续下行，所有人都心知肚明的，只不过为了大局服务，美国政府不允许评级机构不打招呼的下调。随着特朗普阶段性完成了他百日新政中最大的两个项目，中东产油国的访问，以及中美关税战的停火，市场的短期的恐慌一度消散，股指重回。穆迪此时才会被允许公布这个重磅的炸弹，没有跟之前的坏消息形成毁灭性的合力。

随着特朗普第一阶段外战的告一段落，他的枪口将转向内部，解决美国新一年的财政预算、减税以及国债等问题。这个时候穆迪的下调评级，成为了特朗普对内部施压的筹码，从这能够看得出，穆迪不是在硬刚特朗普，而是在配合特朗普。

国际信用评级机构穆迪周五将美国的信用评级从“Aaa”下调至“A1”，理由是美国的债务和利息上升明显高于类似评级的主权国家。特朗普的全面税收法案周五未能突破关键程序障碍，要求进一步削减开支的强硬派共和党人阻止了该法案，这对这位共和党总统在国债说是一次罕见的政治挫折。
