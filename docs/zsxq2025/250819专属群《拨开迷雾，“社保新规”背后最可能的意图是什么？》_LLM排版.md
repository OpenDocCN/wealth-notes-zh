
# 拨开迷雾，“社保新规”背后最可能的意图是什么？

## 社保新规的宏观背景

社保基金的收支压力确实客观存在。根据财政部在2025年全国两会上提交的《关于2024年中央和地方预算执行情况与2025年中央和地方预算草案的报告》，2024年全国社会保险基金预算支出是10.6万亿，收入是11.89万亿，收入比支出多；不过保险费收入大概是8.7万亿，剩下的是财政补贴了大概2.68万亿。

未来，社保收支将面临越来越大的压力。中国的社保模式是“现收现付制”，即今天的年轻人养今天的老人，明天的年轻人养明天的老人。目前的大趋势是“老龄少子化”，交养老金的年轻人和拿养老金的老年人之间，比例会越来越不平衡，给社保收支带来压力。

根据中国社科院世界社保研究中心发布的《中国养老金精算报告2019-2050》，在2019年的预测，到2028年我国养老金当期收入和支出，出现逆转，也就是收不抵支。今年4月，央行主管的《金融时报》报道说，根据最新的《中国养老金精算报告2025-2060》，我国养老金收不抵支的年份，已经从2028年延后到2036年；累计结余耗尽的年份，也从以前预测的2035年，推迟到了2044年，都往后延迟了八九年。

《金融时报》引用了精算报告的负责人、社科院世界社保研究中心主任郑秉文的话说，新的测算加入了延迟退休的因素，也加入了2019年以来中央实施的各项改革措施，包括中央调剂制度、实现全国统筹等等。

## 政策意图分析

很多人认为，这次最高法关于社保的新规定，无论如何客观上也增加了社保执行的刚性，尤其是针对一些过去不缴社保的小微企业、个体户等低收入人群的社保刚性，从而增加社保收入。这种说法有一定道理，背后的影响肯定是有的。但是，如果从政策性价比和社会治理成本角度去考虑，假设政策制定者仅仅或者主要是着眼于“填补社保资金缺口”，那么强制小微企业和个体户参保，实际上效率低、性价比低且社会痛感度高，有点得不偿失。

比如最简单的，早在2019年，社保征收就已经统一划入税务部门，数据打通了，更严格的社保合规，其实早就可以实现了，但直到6年后的今天，基于种种考虑，执法力度仍然是偏宽松的。

## 当前社保合规的数据情况

《经济观察报》2024年8月的报道说，根据51社保在第十二届企业社保发展论坛上发布的《中国企业社保白皮书2024》的数据，2024年社保基数完全合规的企业，占比28.4%，此前的2023年是28.9%，也就是说，只有不到三成的企业，社保基数是完全合规的。2024年，28.2%的企业，是统一按缴费下限为员工缴纳社保的，而22%的企业，“按固定工资部分，不算奖金”为员工缴纳社保的。这些都是公开数据。

## 政策的核心目标

相比增加社保收入，未来很长时间政策的核心目标，是最大可能扩大社保覆盖范围，切断可能的风险传导链条，尤其是对过去游离在体系外的群体，要通过各种方式连推带拉的让他们进入现有保障网，哪怕是最低的保障也可以，但是先要有。

所以你能看到，这段时间无论是健全外卖员等灵活就业人员、农民工、新就业形态人员的社保制度，以及扩大失业、工伤、生育保险覆盖面，全面取消在就业地参保的户籍限制，完善社保关系转移接续政策，以及把毕业未就业困难大学生纳入救助范围等等，背后其实都是统一的政策取向。

## 政府让低收入群体参保的原因

很多人误以为，政府想救助总能救助，其实在现实政策执行中不是这样的。最简单的，政府想发钱，也要有个发钱的制度和发钱的体系，不然有钱也没法发，如果你不在制度和体系内，相关部门也不知道该怎么制度化救助你，最简单的，钱谁出？怎么出？中央对地方也不好考核。

这就跟过去一些新的违法犯罪，哪怕公检法知道，但是也没法惩罚一样，因为没有法律依据，没有工具处罚。

所以，还是我前面说的，我认为当前政策让低收入群体参保的主要目标，不是多增加的这些社保收入，而是想减少未来可能兜底的风险和成本，国家力图将未来可能出现的“隐性社会风险”，转化为当前的“显性保障压力”，不要留下未来可能失控的隐患。

## 未来是合规成本越来越大的时代

从更宏观的视角观察，我认为最近的系列政策释放了一个清晰的信号，就是未来我们在社会保障领域的合规要求“正常化”在所难免。这个进程，恰似过去十多年中环保标准的逐步升级。这种全社会合规成本的上升，是经济发展的必然产物。

而这种制度环境的变化，将产生一系列连锁反应。比如对一些小微企业来说，从员工没社保到有社保，或者全额缴纳社保，人力成本可能增加不少，这部分成本最终将传导到价格端。而从整体经济影响来看，成本推动型通胀，也就是由生产成本上升引起的物价水平上涨，可能将在部分行业缓慢显现，形成温和的价格上行，这对我们在第44讲提到的央行希望的“温和通胀”是有促进作用的。

同时社会保障的增加，从政策制定层面而言，有利于降低公众的“预防性储蓄”，在更大尺度上能让更多钱被拿出来消费，从而扩大内需，所以7月底最新的中央政治会议会强调，要“在保障改善民生中扩大消费需求”。你看，政策其实都是层层勾连的。

## 社会发展的角度

另外，面对老龄化社会的进一步发展，在社保合规的大背景下，老年人返聘再就业可能会越来越普遍，这就像隔壁的日本，因为老年人不用缴纳社保，这会让老年人在社区服务和低强度岗位等领域，获得更多机会，适应老龄化社会。

而更本质的，站在社会发展的角度，这套制度调整，实质上是应对人口老龄化不得不支付的“社会成本”，理解这一点，将有助于我们理解未来可能出现的更多政策。
