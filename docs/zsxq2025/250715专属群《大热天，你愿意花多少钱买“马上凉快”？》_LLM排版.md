
# 大热天，你愿意花多少钱买“马上凉快”？

最近，高温天气的纪录被再次刷新，有机构已经发出预测，今年有可能超过去年，成为有气象记录以来最热的一年。而今天我们要说的，是一个与高温有关的有趣的商业现象。这就是，卖冰块。没错，纯冰块。

假如你走进便利店，或者打开外卖软件，大概率都能看到单独售卖的冰块。有袋装的，有杯装的。一杯冰块从3块到5块不等，容量在三两到一斤之间。相当于花了5块钱，就买了一杯冰，比同等体积的水要贵上不少。

其实，卖冰块的生意很早之前就出现了，但是，这几年的热度特别高。以往只出现在连锁便利店，而现在很多商超和冷饮店都在卖。

2021年，盒马就推出了冰杯和袋装冰，去年，农夫山泉推出了自己的冰杯产品，美团买菜、叮咚等等线上渠道也上线了冰块和冰杯，蜜雪冰城推出了售价一元的单独冰杯。今年，古茗、书亦烧仙草等茶饮品牌也推出了冰杯单品。

这些冰块不仅单独卖，也被搭配成了套餐。比如，盒马推出了直接把咖啡、柠檬水、乳酸菌冻成冰块的冰杯，美团小象超市推出了生椰拿铁套装，包含一瓶黑咖啡、一瓶椰子水、一袋冰块。

《2025中国都市消费行为白皮书》显示，冰杯销量连续两年增速300%，一线城市每年的人均冰杯消费量，从22杯提高到了48杯。盒马的数据显示，今年6月以来，冰杯产品的销量在去年的基础上又增长了30%。

冰块凭什么卖得比矿泉水贵？原因很好理解，因为多了冷冻和运输的成本。

在生产环节，据说制作冰块前要先用滤膜去掉水里的杂质，然后用特殊的工艺，让冰块最大限度地排出空气。这也是为什么市面上卖的冰块看起来更透亮，融化得也更慢。据说冰块生产还有一个程序，把冰块的棱角都磨圆润，这是因为，锋利的边角在碰撞中容易破损，破损的冰块接触空气的表面积加大，也会更容易融化。

在运输环节，需要全程冷链，费用是常温运输的三到四倍。到了最终的配送环节，还需要保温袋和干冰。

注意，我们可不是要说，人家的价格合理或者不合理。咱们只是呈现过程，不对价格做评判。

那么，都是哪些人在生产冰块呢？目前市场上的参与者大致有四类。

第一是传统的制冰厂家，专门生产冰块，比如冰力达、晓德食品，他们拥有食用冰的全链条生产和销售能力，但过去往往只做B端的生意，C端消费者对他们不太熟悉。

第二类就是我们接触比较多的，盒马、罗森、711、小象超市等商超便利店。他们更多采用代工模式，优势在于离消费者足够近，销售渠道更多。

第三类是农夫山泉、伊利、蒙牛这样的饮料企业。这类企业处在冰块的“临近行业”。比如农夫山泉，本来就是生产饮用水的。再比如伊利和蒙牛，在冷链运输上有天然优势。

第四类是以蜜雪冰城为代表的新茶饮企业，他们售卖的冰杯和前面几种稍有不同，他们更多是在门店放一个制冰机，现场制作销售，产品直接面向C端用户，制冰的体量也比前面几类小一些。

回到消费者视角，国内消费者对它的接受程度正在走高。2024年全年，冰杯的销量超过5000万杯。过去很多年，中国人好像倾向于喝热水，喝冰水的习惯并不突出。但现在，年轻消费者入场，这个趋势正在发生变化。

根据咨询公司尼尔森的调查，有将近七成的消费者，习惯从便利店购买冰杯。

而且未来，冰块市场或许还有不小的增长空间。我们可以对比日本和韩国。

日本，在无处不在的便利店的加持下，每年一次性冰杯的消费量高达25.7亿杯，比瓶装水的销量还要高。再比如韩国，很多消费者一年四季喝什么都加冰，尤其是喝咖啡。韩国出现了一个缩略语叫“冻死冰”，也就是冻死也要喝冰美式。2005年开始，韩国饮料厂商就推出了“冰杯+袋装咖啡”的组合。

在这种喝什么都加冰的习惯下，日韩的食用冰，还针对不同的场景延伸出了更细分的产品。

从容量上细分，有160克的小冰杯、220克的大冰杯，韩国便利店还有650克的特大号冰杯，带两个吸管，可以作为分享装。还有更大号的针对家庭使用的袋装冰和针对户外的冰桶。

从形状上细分，有调制日常饮料的方块冰或者小冰球，还有整块的方形冰和大冰球。

从包装上细分，有适合便携的带盖冰杯，还有适合在家直接喝的无盖冰杯，还有扁平的盒装冰杯，适合那些家里冰箱小，没有太多冷冻空间的顾客。

而这两年国内市场上出现的各类冰块产品，也呈现出了类似的苗头。咱们国内的冰块市场，在总体规模变大的同时，也可能会出现产品细分的趋势。

同时，冰块只是表象，透过卖冰块这门生意，我们或许还能发现这么几个相对普遍的趋势。

首先，很多原本的“搭子”，正在变成独立的商品。类似的情况比较常见。比如，智能手机普及之后，手机壳、手机贴膜、手机配饰，都成了独立的生意。再比如，洞洞鞋火了之后，可以镶嵌到鞋子上的配饰也成了一门独立的生意。再比如，护肤品，过去水是水，霜是霜，精华是精华。后来市场细分出了不同的成分单品。

在一个信息足够透明、选择足够多的市场，消费者需求的颗粒度也在越来越细。

其次，便利性，正在成为商品本身。你看，很多时候，商家卖的不只是冰块本身，而是在卖“让你立马获得冰块”的便利性。毕竟传统上，获得冰块是需要时间的，少说也得在冰箱冻上几个小时。这时候，一杯两三块钱的冰块，解决的就不仅是冰块本身的问题，更是“即时满足”这个需求。

换句话说，消费者花5块钱买的不是冰块，而是等待水冻成冰的两个小时，以及这两个小时里要经历的烦躁、炎热和迫不及待。

而且随着消费能力的提升，人们对即时满足的需求也许会越来越大。今年夏天，淘宝、京东、美团，三家巨头罕见地正面硬碰硬，表面上看，竞争的主战场是外卖。但更多的分析认为，他们实际争夺的是外卖背后那个更大的市场，即时零售。

最初的购物状态是，人们看到广告，想买一个商品。然后专程去一趟店里，把商品买回来。后来有了网购，可以直接在网上挑选，等两天快递就会上门。而在即时零售中，人们从起心动念到拿到商品，只需要足不出户的30分钟。

消费者一旦习惯了即时配送，就不太能忍受回到线下的超市去排队。这就好比人们已经习惯了5G，就不可能再接受拨号上网。从这个角度看，即时零售对一部分传统业态的碾压，可能不亚于当年网购对实体店的打击。

在这个逻辑下，任何能缩短“需求产生”到“需求满足”这个时间差的商品，都可能有价值。

最后，假如说这件事对咱们普通人有什么启发，我觉得可能是，别小看任何一个“小众信号”。

你看，零售冰块原本可能只是爱喝酒、爱喝咖啡的用户才有的需求。但在极端高温、茶饮市场进化、生活习惯改变等等很多因素的推动下，它正在发展成一个大众市场。那些看起来很细分、很边缘的需求，可能会随时因为某个拐点的出现，突然爆发。所以，不要轻易忽视任何一个看似微不足道的消费信号。

反过来，有的时候，找小切口，走小路，可能也是成功的有效路径。美国一家环保袋公司的创始人莎伦·罗维写过一本书，叫做《小步向前》，她非常建议创业者去做一家“小公司”，不是规模上的小，而是，意图上的“小”，也就是，只瞄准一个小而重要的需求去行动。

在这里也要特别强调一句，小需求，不等于难度低。比如，卖冰块这个生意乍一看好像很小，但是至少目前看，这个生意的入行门槛不低。需要从业者有成熟的冷链，有足够强大的成本控制能力，最关键的是要有规模庞大的售卖渠道。这也是为什么目前的从业者，要么是面向B端的厂家，他们有稳定的下游需求；要么是冷饮连锁、便利店，他们有渗透力足够强，而且完全可控的销售渠道；要么是伊利这样的传统饮品大厂，他们的冷链和技术都是现成的，可以摊薄成本。

换句话说，你手里有什么样的武器和用这个武器瞄准什么样的靶子，这是两个问题。所谓的瞄准小众需求，绝不意味着可以拿着一把玩具枪，它依然要求你首先火力凶猛，然后才是把这些火力集中于一点，爆发出最强的冲击力。
