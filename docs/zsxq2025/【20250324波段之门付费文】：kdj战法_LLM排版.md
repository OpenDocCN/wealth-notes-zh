
# Kdj战法

这是我第一次详解一个指标。你会看到一个新的世界。

1978年，美国人威尔德发明了著名的强弱指数RSI、KDJ，还发明了其它分析工具如PAR、抛物线、MOM、摇摆指数、市价波幅等。这些分析工具在当时的时代大行其道，受到不少投资者的欢迎，即使在今天的证券投资市场中，RSI 仍然是非常有名的分析工具。可以说，威尔德是对交易员影响最大的人之一，你可能不用技术指标，但你绝对不会没有听过或了解过这些技术指标的用法，尤其是各种股评家靠这些工具出具每日预测。但有多少人知道kdj 的用法?有人认为kdj 指标不管用，kdj 金叉了，市场却不涨，死叉了竟然也不跌。于是，有人得出了结论，指标这东西是小学生用的玩意，真正的投资者是不用这些狗屁指标的。很荣幸，2000年在深圳，我有幸在一艘轮船上，听到了一个台湾人的讲解，我才知道了怎么去研究指标。指标是通过价格的变化反应人的情绪，所以指标的数值变化，反应了人的情绪变化。交易者的情绪亢奋以及低落，都会引起走势的变化，而指标就是间接描叙人的情绪的。
