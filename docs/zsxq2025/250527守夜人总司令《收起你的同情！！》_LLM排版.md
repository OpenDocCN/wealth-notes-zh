# 收起你的同情！！

## 许多人可能曾经这样憧憬过：
如果全国每个人捐给自己一块钱，自己瞬间就会变成拥有 14.5 亿的富豪！而且，一块钱对任何人来说都不是太大的负担。你还别说，真的有这样的机制——那就是慈善捐款！

每当天灾人祸的时候，大家都会踊跃捐款，那一刻神州大地血溶于水，一方有难，八方支援。无数的物资、资金，甚至人员从四面八方涌入灾区。此时此刻，这些物资是如何被调配和使用的，也没有人去关注资金的去向。大家的想法很单纯：只要我把钱财捐出去了，我的心意就尽到了！正因为大家都是这样想的，所以，海量的钱财和物资被源源不断的调拨到灾区之后，就会产生一种非常奇怪的现象：接收财务的当地机构突然之间变成巨富，他们一辈子都不曾见过这么多的钱财，并且掌握的分配这些钱财的决定权！

## 地方运行机制的困境
读过《波斯的选择！！》就会明白，任何一个地方都有一套与之相适应的运行机制。越是落后的地方就越官僚腐败且低效无力，对外虚伪，对内残忍。

为什么有些地方会应尽全力保住自己的贫困县的帽子？因为我弱我有理！因为我穷，所以国家不能不管；而你一旦拨款下来，我就有 1000 种办法让它消失于无形。这就是为什么越是投入资金去帮扶贫困地区，那个地方反而常年贫困的原因。只要贫困下去，就可以不劳而获！

如果是按照市场原则去投资，拿了人家的钱是不是要给人家一个交代？钱都用去干什么了？效果如何？这样按照市场原则公开透明的投资项目，经办人员就没有油水，自然也就没有动力。正因为如此，越是贫穷落后的地区，就越讨厌市场经济。他们利用当地民众受教育程度不高的特征，就把所有遵循市场原则追求利益的投资者都描绘成恶棍，把自己打扮成道德标杆。然后，要求所有的资金由自己来分配。结果他一分配钱就不见了。过几年他就成富人了！

这个社会发展到了今天，不仅街上没有流浪汉，连流浪狗流浪猫都会有人救治。所以你不用担心别人过得不好，更不应该越俎代庖。一个成熟的社会会有一套应对于各种问题的机制。如果没有，应该去建立机制，而不是呼唤良心！

## 同情心与机制的错位
从某种意义上来说，同情心是一种居高临下的优越感。为自己过得比别人好，才会试图通过这种施舍的方式来呈现自己的优越感。国家本来就有救助机制，而且随着经济的发展，国家的救助机制响应越来越快，物资的调配也越来越快，东西也越来越足。差你那一点么？！

华南有一个著名的寺庙，让那些信男信女向灾区人民捐赠大米。他们把一袋大米放到台子上，让烧香跪拜的男女扫码付钱，说这些大米凑满一车，就会发往灾区。一袋米可能买只要 50 块钱，但是他让人家扫码是 175！他们把台子上扫码过的大米抬下去之后并没有装车，而是转一圈又拿回来了！什么意思呢？就是永远都是这几十袋米在循环，根本就不会捐给什么灾区。那些扫码付钱的人以为米是从仓库里搬出来的，其实就是上一个付完钱后从台子上抬下去的米！

冷漠未必不是好事，正常的体系下，任何一个事情都有相关的责任人和相应的机制去处理。不需要局外人横插一杆子。那些动不动就煽动情绪，让局外人来解决问题的社会，最大的问题是职能缺位！而且会把职能缺位包装成道德感动。

## 职能缺位与道德绑架
一个山村里面没有教师，也没有教室。有一个没有编制的乡村教师，自己想办法搞了一间学校，自己教学生以避免当地孩子失学。这件事情本身是某些人的失职。结果，不去查漏补缺，追究失职的责任，反而会把它包装成一个道德感动的故事！你不觉得荒唐吗？

孙子说：善战者无赫赫之功，无煌煌之名。这个社会上的每个人只要做好自己该做的事情，查漏补缺完善系统，许多问题就会迎刃而解。华为的任总说，企业要运行正常，就是不断挖掘可能存在的问题并提前先解决掉，当那些可能导致失败的问题都被一一解决掉之后，剩下的不就是正常了！

当一个社会不再热衷于以道德绑架来判断对是非对错的时候，这个社会就会变得理性健壮，有章可循。而那些作奸犯科的现象才能逐渐绝迹…

## 腐败的根源与解决之道
当全国 70%的医药都必须走大医院这个销售渠道的时候。哪怕是恢复炮烙之刑，也挡不住某些科室主任和院长们的腐败：因为实在给的太多了，是个人都顶不住。真正要惩前毖后治病救人，只需要匀出 20%的份额到市场那个渠道里面去销售，医药代表就没兴趣想尽一切办法去拉他们下水！为啥呢？投入和产出不成比例的时候，你还投入吗？如果走大医院这个渠道的销售额降到 30%以下，医药代表压根就懒得上门了。无利可图谁去？

愚昧和特权如同硬币的两面。愚蠢的人群会拼命的捍卫特权，并在日常生活中动不动就要求权力介入，权力也特别喜欢愚昧者的道德化亢奋，同时热衷于以道德良心的仲裁者角色介入那些纷争。为什么越是贫穷的地区更容易出暴富的官僚？
