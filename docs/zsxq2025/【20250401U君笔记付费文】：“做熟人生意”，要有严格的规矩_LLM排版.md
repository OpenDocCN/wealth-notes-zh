
# “做熟人生意”要有严格的规矩

本文是2025年3月25日公众号 (U 君笔记) 的付费文章。

上次有个小伙伴咨询我，说他是做装修的，要面临给一个老同学装修房子。而且那个老同学一再暗示他要用真材实料，而且要少花钱（少加利润）。他自己不想做亏本生意，因为自己也需要挣钱养家糊口；但是又不想得罪老同学，所以左右为难。

我的建议是：找一些理由婉拒，让你的老同学另请高人，你千万不要伺候这种人。要知道，本来熟人之间做生意就有很多“不痛快”的地方。比如下面这3种小心思：

① 合作途中出现任何问题，双方都不好意思直接沟通，怕伤到感情。如果是陌生人之间合作，反而倒轻松了，有一说一，有二说二。不满意就要挑毛病，直到满意为止。这样双方心里都不会有怨气存在。

② 本来你用市场行情去报价，但是对方总觉得你报高了。他会想，我听别人说3万块就够了，你却给我报价5万。殊不知，他听说的那个3万元的报价，只是一种营销策略，是一种引人上钩的手段。他如果真的信了那个3万元的“鱼饵”,后面还会有8万元、10万元的坑在等着他。但是他可不管这些，他只会觉得你的报价比他们说的贵。

③ 人性中有两个很贱的弱点：枕边无伟人，外来的和尚会念经。比如，你如果一分钱都不赚，用成本价去做，对方也不一定会领情，他的内心仍然会嘀咕，你是不是在杀熟，在坑朋友。还有，你已经做到80分了，他始终觉得不如外人做得好，即使外人的水平只有60分。人就是这样，总是被“外来的和尚”所笼罩的神秘光环所俘获，进而忽略了身边真正优秀的人。

举个不恰当的例子，有一些蠢女人，她的老公是重点大学毕业，什么都懂，但是她从来不相信老公说的话。但是，外面那些小学毕业的阿猫阿狗，随便说两句话就可以让她信以为真，然后把钱拱手相让。

综上所述，最佳的策略就是不做熟人生意。如果非要做，就要按照市场价进行操作，利润一分钱都不能少。有人会问：U君，朋友之间不给点优惠吗?你怎么这么冷血?那我问你：既然是朋友，他为什么不多给我点，用真金白银来支持我的生意呢?两个观点到底哪个是正确的?答案也很简单，前者多是“占便宜”的穷人思维，后者多是“捧场”的富人思维。至少我自己是这样操作的：如果我找朋友帮我做事，要么我会要求他按照市场行情进行报价，一分钱都不要少算；要么我会多给他10%-20%的钱。因为我真心想让朋友多赚一些钱，然后过上更好的生活。我怎么可能贪图便宜让朋友割肉、让利呢?反过来也成立。如果我卖什么商品或者服务，哪个熟人想要跟我攀关系、套近乎、让我便宜点，我肯定让他滚蛋。要知道，“做生意”跟“送人情”有着清晰的界限。

假设我是卖针的，每根针1毛钱，朋友来买我也要收钱。晚上跟这个朋友一起喝酒的话，100元钱我请客我也愿意掏这个钱。因为1毛钱是生意，100元钱是交情，二者性质不一样，绝对不能混为一谈。如果你不明白我说的这一点，那么你的思维就出现了重大问题。如果不纠正的话，你的收入和境遇将长期在一个低水平区间徘徊。

那么问题来了，为什么1毛钱的针要收钱，但是100元的饭钱可以请客呢?背后主要有3个原因：

1. 认可度和成就感

无论别人是做什么生意的，无论别人是卖商品还是卖服务，如果你去买单，那么对方接收到的信号是：你认可对方。他除了能从你身上赚到钱之外，还能获得成就感和满足感。这些都是无法用金钱来衡量的。但是，如果他因为交情而白送的话，这两种感觉就消失了。这就是为什么我经常跟大家说：向上结交的核心就是买单。一个你平时接触不到的大佬，想要跟他发生交集、想要跟他聊天，最好的方式就是：他卖什么你就第一个买单。即使他身价过亿，即使跟你成交的这一单只有几百元的利润，他也会非常感激你的。这时候你付出的虽然是金钱，但是对方收获的是认可和成就感。反之，你把几百元直接给对方转账，要求对方跟你聊几句，对方可能并不会搭理你。因为这时候这几百元钱是冷冰冰的，是没有温度的。送礼的核心也在于此。下策是：直接给对方金钱或者好处，要求对方为你办事。对方很可能会直接拒绝你。因为这是一种冷冰冰的交易，甚至是对对方的一种侮辱。上策是：一定要先找一个由头，打着“打感情牌”的幌子，行“送真金白银”之实。比如，你可以这样跟他说：领导，我看您平时太节俭了，一部手机用了好几年，都卡成这样了，您还在用。我实在看不过去了，正好我一个经销商朋友送了我一部新的苹果手机，还没拆封，我把它送给您吧。或者，你也可以从他的家人入手：我看到大侄子学习这么好，我由衷地感到高兴，这孩子的前途肯定不可限量。下个月是他生日，这部手机就送给他当作生日礼物吧。如果对方收下了，或者不置可否，这时候他自然会问你：你找我是有事吧?接下来你就可以顺理成章地说出自己的诉求了。

2. 留下好评=留下希望

只要你在对方那里买单了，无论是在电商平台，还是在朋友圈，你都可能会留下成交记录，或者会给出好评。这对他未来的事业发展是一种促进作用，或者说，你这一单虽然没给他带去多少利润，但是你种下的是一份希望。希望总是让人心里暖暖的，这种感觉也是无法用金钱来形容的。反之，如果他直接赠送给你，那么就不会留下任何希望，他的心里甚至还会出现内耗和懊悔等负能量。当然，在很多情况下，“赠送希望”根本不需要花钱。比如，对方发朋友圈你给点个赞，对方写文章你给用心评论几句，对方都会非常激动的，因为你的这些动作也能给对方带去希望的感觉。

3. 养活自己的团队和公司

如果对方有自己的团队和公司，那么他就更在意营业额了。很多情况下，对方在出售的产品或者服务并不是他一个人获利，而是整个团队的命运。当营业额上升时，整个团队都是欢欣鼓舞的，都是充满希望的。而且，他们的团队成员也会非常尊重他们的领导。所以，如果你真心为他好，就多支持他团队，多为他们买单。反之，如果你白拿了他们团队生产出来的产品或者服务，哪怕你事后通过其他形式对团队成员进行补偿，他们的内心也不是滋味。另外，“团队”也是拒绝熟人客户杀价的挡箭牌。举个现实的例子，我之前搞了很多VIP群、知识星球等收费项目，也碰到过熟人的咨询，其中也有暗示我给优惠的。我碰到这种情况，都是统一把收人文案发给他们，让他们按照流程操作、去我助理那里交费，因为这个钱不是我一个人赚，以后还要走账、还要分润。所以，我不会给他们任何优惠，认可我就交钱，不认可就走人即可。这样操作的话，开始可能会感觉不近人情，但是我想说，如果你想维持10年以上的人际关系，诀窍就是在这里：彼此该买单就买单，一分钱都不优惠。至于一起吃饭时到底该谁买单，这些反倒是要次要的。因为在这个时候，钱就回归到它本身的角色，那就是钱。它跟成就感、满足感、温暖、希望……一点关系都没有。
