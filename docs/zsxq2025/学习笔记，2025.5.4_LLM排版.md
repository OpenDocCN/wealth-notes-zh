
# 学习笔记，2025.5.4

1，读书、看电影、交朋友，我们总是会去找那些“和自己三观相近”的，这其实也是在一定程度上给自己打造信息茧房，固化自己的认知，导致忽略很多事实和真相。
观点有立场，事实无立场，如果想要离真相更近一些，就不要总是试图用三观来概括什么，要透过观点去观察事实和逻辑。

2，我偶尔会拉黑评论区一些账号，这些账号都有一个共性，表达的内容除了情绪，几乎没有其他东西。
包括一些所谓KOL、大V，当他的感叹号使用频率高到一定程度的时候，说明情绪大于内容，不值得再在他身上浪费时间。
曾经有几个比较崇拜的 KOL，这几年眼看着他们慢慢从讲事实逻辑变到了只输出情绪。
时间是个好东西，能让很多东西都现出原形。

3，如果你在某个行业持续做上三五年，并且取得过一些不错的成绩，你就会发现：不一定是自己多优秀，而是同行太差了，衬托显得自己很优秀。
甚至很多时候，你都不需要做对什么，只要保持不出错，同行就能一个接一个暴雷自动退出游戏。
用雷军那句锐评就是，友商都是傻 X。

4，接受一种新思想，意味着对过去的自我的否定。
很多人的“美好”都在回忆里，否定过去，等于杀死过去的自己，等于否定自己的“美好”，这对他们来说是不可能的事。
所以，对大部分人来说，宁愿花费十倍百倍的精力去维护过去自己认为的那种“美好”，也不愿意睁开眼睛看一下那种“美好”到底是不是真实存在的东西，是不是别人为了控制自己的思想植入的某种概念。

5，经济学有个重要概念叫"只富一次"。统计显示，99%的人一生中只能抓住一次财富积累机会，多数人最终会返贫。
过去二十年观察发现，身边 80%的高收入者最终陷入财务困境。
主要问题有两个：一是把时代红利当个人能力，二是欲望超过实际控制能力。
某地产商高峰期管理50个项目，最终因资金链断裂负债30亿。
守住财富比创造财富更难。
常见的财富毁灭途径包括：赌博输掉积蓄、沾染毒品倾家荡产、盲目创业消耗本金、信用贷引发债务危机、all in股市血本无归、错误婚恋导致财产分割。
某上市公司高管因炒期货爆仓，不仅损失千万积蓄还背上债务。
真正的智慧是清醒认知能力边界。
某科技公司创始人在行业红利期套现离场，将资金分散投资，至今保持稳定收益。
记住：能控制欲望的人，才能掌控财富。

6，我所做的一切，都是我的隐私。
没有人知道我的任何事。
如果别人问起，我只说"我在工作"。
这是我一直以来的心态。
你会惊讶于有多少人想毁掉美好的事物，如果他们不知道，他们就毁不了，保持沉默。

7，能将你从谷底里拖出来的，从来不是时间，而是你心里的格局和你发自内心的释怀。
这个世界从来没有感同身受，你可以消沉，也可以抱怨，甚至可以崩溃，但一定要懂得自愈。
无人问津也好，技不如人也罢，你都要试着沉下心来去做自己的事，而不是让内心的烦躁、焦虑，毁掉你本就不多的热情和定力！你要明白，真正能给你撑腰的，是丰富的知识储备，足够的经济基础，持续的情绪稳定，可控的生活节奏，和那个打不败的自己。

8，爱自己最好的方式，不是买很贵的衣服，吃很贵的饭菜，而是不取悦别人，不否定自己，不委曲求全，不沉迷往事。
能接受的就接受，接受不了的就改变，改变不了的就离开，无能为力的事，当断。
命中无缘的人，当舍。心中的执着执念，当离。

9，穷人，对赚钱最大的误解，就是不停地学习。
一个粗糙的开始，就是最好的开始，不要等准备完美了再去做，因为完美的准备永远做不到。
一定要记住，先完成再完美，干的慢，总比什么都没有干好要好太多。只有行动起来，才会发现自己的潜力，随着时间的推移再突飞猛进。

10，当运气降临时，一个人的精气神会变得充足，气场会变得强大。
明明还是同一张脸，却能突然感受到强大的气场，思维会变得异常清晰，整个人
的精气神会变得超级饱满。
你再也不会继续消沉下去了，因为你有了内在的驱动力。
命运会推动着你不断向上，你会彻底改变之前的风格。
整个人的外貌、穿着打扮甚至身体状态都会自然而然地产生变化。
你会开始果断地舍弃那些有毒的人事物。
找到了这种手感之后，就继续顺着这种感觉前行，很快你就能顺势而起。
