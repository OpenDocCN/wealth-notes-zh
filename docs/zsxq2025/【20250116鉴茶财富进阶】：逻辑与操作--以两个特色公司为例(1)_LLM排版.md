
# 鉴茶财富进阶：逻辑与操作——以两个特色公司为例

投资创业需有大逻辑，该逻辑需经得起5-10年考验，不受短期波动影响。以下分析两个公司：

## 东鹏特饮

**逻辑**：消费降级，对标日本市场。随着白酒价格升高，提神饮料需求增加，类似日本力保健的年销量达2亿多瓶。该逻辑成立，业绩也支持。

**操作**：东鹏特饮在1008和1128之间形成箱体，跌破箱体即走（206.5）。若不跌破5日平台，可继续持有。1月13日放量16亿，换手率6倍，为高位放巨量杀跌信号，需及时离场。

## 海光

**逻辑**：国产替代芯片，卡脖子概念。业绩连续4年上涨，能兑现预期。

**操作**：1008-1219始终不跌破箱体，可持有。12.25大涨11%，形成新箱体。若12.31跌破5日平台，需离场。调整后，仍会根据长逻辑兑现预期。
