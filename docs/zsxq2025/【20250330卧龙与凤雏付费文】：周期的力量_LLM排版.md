
# 周期的力量

或许有不少人会认为，在股市当中赚钱的人，是因为技术有多高超，心态有多强大。其实很多时候，都是“运气”。说是“运气”，倒不如换个词，环境。大环境好的时候，牛市之下人人皆是股神，再垃圾的交易体系可能也能实现超过巴菲特老爷子的平均年化收益。而当周期更替，来到了熊市之后，再牛的技术，再强的心态，再高的认知，很多时候，也只能说一句话，“少亏就是赚”。

所谓的牛市熊市，一切都属于周期的产物。就像太阳东升西落，就像春去秋来，四季更替，就像天下大势，分久必合，合久必分。世间的一切，皆围绕着周期来运转。所谓的神话，所谓的成功人士，其实很多时候，都是周期的产物，时势造英雄罢了。

没有一个好的周期，没有一个好的“运气”，光靠努力，就是个屁，“时来天地皆同力，运去英雄不自由。”

而决定普通人“运气”的关键，正是对于周期的理解与把控。就像有幸刷到本篇文章的你，我们通过计算推演出指数的节奏图，跟大家梳理，什么时候是“好的周期”。就像年前和大家说的看空1月，但2-3月将是一波“好的周期”。

涨跌节奏客观来说，下周才是起点，虽然说上周周线已经收红，但是下周有可能会再次确认一下底部(确认起点)，也就是有两种选择，直接拉升/探底回升。我给大家提个醒，别过分期待有单边行情出现，过年红包行情，抱有平常心才可能会遇到。不追高，做短线的朋友可以逢低做多，没时间看盘的朋友可以准备过年了。大节奏来说，看空一月，看多2+3月，节后就是2月了，节前持仓与否在个人。

以上纯属个人观点仅供参考!股市有风险入市需谨慎!

一切皆有周期，就像股市中的，五一或者国庆节前都会炒作一小波旅游，就像冬季来临之前会炒一波煤炭，就像夏季来临之前(最近)有炒作一波电力。

理解周期，尊重周期，敬畏周期。好的周期，重拳出击，坏的周期，空仓休息，选择远远大于努力，这就是残酷的现实。

上周也有提醒大家，周线以及月线级别节奏有压制，所以日线级别难有表现，那接下来马上就四月了，新的节奏会如何演绎?

客观来讲，周线、月线存在压制，故此处的日线反弹大概率难以起到很强的效果，基于这一点，接下来选择向下是大概率的事情，哪怕选择往上，时间空间也是十分有限。

做股票就是在做指数。指数涨，大部分个股都能涨，指数跌，大部分个股都会跌。不以涨喜，不以跌悲，目空一切，方可大成。
