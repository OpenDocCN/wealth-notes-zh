
# 十五五的私有化(上)
# 发布时间：2025年11月21日
# 11月北京线下读者会系列内容
三中确立的是开放，四中确立的是改革，
改革就是对内的开放，
就是允许民营资本进入到过去受限的领域，
就是政府需要将一部分利益出让给民营企业。
十三五的提速降费，
是把电信行业的利润部分转让给移动互联网行业，
十三五的简政放权，
是把地方政府的预算与权力部分转移给平台企业，
十三五的科创板，
是把国企和地方政府的融资分给半导体行业。
展望十五五周期的未来，
可以回顾十三五周期的那些急速扩张代表。
海航代表的，是独立于国航、南航、东航之外的
民营航空公司，
华信代表的，是独立于中石油、中石化、中海油
之外的民营石油公司，
安邦代表的，是独立于国寿、平安、人保之外的
民营保险公司，
复星代表的，是独立于国药、上药、华润之外的
民营医药公司，


吉利代表的，是独立于一汽、二汽、上汽之外的
民营汽车公司。
本质上，15年之后的那一波民企高速扩张，
是一群妖精背后的神仙们借着十三五的东风，
安排他们去跑马圈地。
而即将迎来的十五五，
也会出现一大批这样高速扩张的企业。

《中华人民共和国民营经济促进法》是我国第一部
专门关于民营经济发展的基础性法律，在我国民营
经济发展历程中具有里程碑意义。这部法律2025
年5月20日起正式施行。
民营经济促进法第一次明确民营经济的法律地位…
如此多的"第一次"让民营企业吃下"定心丸"。
民营经济促进法明确，支持民营经济组织参与国家
重大战略和重大工程。

科技攻关项目，支持有能力的民营经济组织牵头承
担国家重大技术攻关任务，向民营经济组织开放国
家重大科研基础设施。
