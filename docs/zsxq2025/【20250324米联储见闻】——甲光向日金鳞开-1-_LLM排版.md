
# 米联储见闻20250324 甲光向日金鳞开

北京又开会了，这一次主要是把大资本家们聚在一起聊一聊，连最近风口浪尖上的李家都派出了代表参加。这场由国务院发展研究中心主办、兔子发展研究基金会承办的兔子发展高层论坛2025年年会把眼下炙手可热的大资本家们都汇集到了北京，苹果首席库克、三星电子会长李在镕，辉瑞、联邦快递、西门子、宝马、奔驰、高通、阿斯利康、沙特阿美、力拓、必和必拓等80多位跨国公司代表在列，其中来自美国的数量最多，达到近30家。所以，美国依然是这个世界经济的中流砥柱，也是不假的。兔子发展高层论坛一直是兔子官方层面与国际商界和学术界领袖就兔子经济社会发展重大议题开展高层交流合作重要渠道。过去来参加这个论坛的主要还是发达国家的大资本家，但是这几年东盟和南美的资本逐渐多起来，今年来的代表中，来自这些地区的企业家也占有相当比例。

兔子境内的譬亚喊喝喇守8宅宪天忒笢节气为－申报标准，触发经营者集中申报义务。至于不少人期待发挥威力的《外商投资安全审查办法》，此次适用则面临更大难度。《办法》所称的外商投资，是指外国投资者直接或者间接在中华人民共和国境内进行的投资活动。但此次交易双方和标的资产都位于兔子大陆境外，大陆及香港的港口资产已被精准切分。经此一役，也是宝贵的一次教训，需要对《外商投资安全审查办法》进一步补漏，或者出台一部兔子的《国际紧急经济权力法》(IEEPA)。此外，这笔交易还有很多交易变数。比如说交易尚需通过相关港口所在国家等司法辖区的反垄断审查。这就要看兔子的外交实力了，如何通过影响这些国家来影响这些交易。从更长远发展来说，兔子需要加快构建自主可控的全球物流网络，并通过国际组织争取航运规则制定权，以应对美国轻松挑战兔子权益的风险。对于想要走出正面对决美国的兔子来说是次大考，次综合能力的大考，能否拿下这次大考，也对于当下的士气尤为重要。

永远记住，胜利才是最好的团建，这事如今闹到这个地步，即便不能全胜，也定不能输得太容易。祝各位晚安
