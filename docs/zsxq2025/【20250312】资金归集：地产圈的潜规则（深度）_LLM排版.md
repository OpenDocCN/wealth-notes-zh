
# 资金归集：地产圈的潜规则（深度）

今天最高俭察院在两会上作工作报告，其中有一段提到了恒大。原文是这样的：指导广东、北京等地俭察机关对姮大系案件42人，钟直系案件49人审查起诉。这其中提到的钟直系案件，也就是纯粹的非法融资和金融诈骗类的案件，已经有过很多宗了。这一宗也没什么大不了，关键是姮大系案件。最高俭的这个公开表态，事实上意味着两件事：

第一，这3年以来，地产商群体密不发丧的潜规则被打破了，从此以后，暴雷的地产商可以公开发丧了。第二，暴雷地产商也都要承担刑事责任了。

话说姮大以及钟帼所有其他地产商，暴雷的唯一原因都可以归结到一个词上：资金归集。各项目公司必须将手里原本用于项目工程建设的钱，全都集中到集团总部，一般都是由公司设立的某个财富管理公司手上。恒大有恒大财富，阳光城有阳光城财富，万科有鹏金所，这些公司成立的目的，就是归集集团的所有流动资金。无论这些资金来源于哪里，即便是法律严厉禁止挪用的开发贷资金和预售监控账户资金，都要被强行挪用到这些财富管理公司，然后这些财富管理公司，再以这些资金作为本钱，与各大金融机构合作发行各种金融产品，实现资金的杠杆化，一般都能轻松实现三倍杠杆，也就是一块钱本金可以变成三块钱，地产商再拿着变回来的三块钱去投资，买更多的地，从而实现资产规模的迅速扩张，这就是钟帼地产圈的通用玩法。

除了极少数的几家央企，因为严格的财务纪律问题，无法玩弄这种资金游戏，其他每一家地产商都是这个玩法。而通过这种彻底的资金挪用，所构建出来的财务杠杆是非常脆弱的，它经不起市场的任何波动，房地产市场必须一直涨，而且还必须是持续性的大涨。所以到2021年，市场涨幅稍微缓了一下，姮大就第一个暴雷，接下来就是普遍性的暴雷，地铲百强企业一家接一家的暴雷，到现在为止，曾经辉煌的民赢地铲企业，已经几乎是全军覆没，全都处于暴雷状态。

所谓暴雷，也就是现金流枯竭了，资金杠杆断了，还不起债了。当然，在此过程中，其实还有很多地方上的小型地产商，他们的规模虽然不大，但是通过与地方上的小型农商行合作，事实上也全面复制了大型地产商的高杠杆玩法，其实也已经事实暴雷。

对这种普遍性的地产商暴雷现象，有司第一反应是盖上盖子，强行掩盖问题，也就是我们一直说的秘不发丧，这个盖子一盖就是3年半，在此期间，暴雷的地产商简直尴尬的要命，比如说毕桂圆这样的上市公司，到底要不要按照证券市场的要求发布年报，如果要发的话，到底要怎么发，直接说自己已经死了，现金流都断了，但是净府不许我死，所以我要假装活着，这种话公开说出来，估计全世界都要被吓死，所以毕桂圆也一直拖着不发年报，一直拖到今年1月份，净府的态度发生了改变，看起来已经允许地产商公开自己的死讯了，毕桂圆才将拖了两年没发的年报发出来，大家一看，卧槽。

毕桂圆2023年亏了1784亿，2024年上半年又亏了128亿，截至2024年6月底，已经有1473亿的债务违约，这些数据摊开来，这家公司已经死的不能再死了。而在公开地产商的死讯的同时，净府也在两会上借最高捡的工作报告定下了第二个基调：要抓人。许佳印于2023年9月被软禁，后来被转移到深圳看守所，因为对于如何处理他一直没有结论，坊间长期流传的说法是，许佳印家族向净府退还了500亿资金，买到了平安，现在最高捡的报告算是尘埃落定，一定要判，而且一定会重判。事实上，这也意味着一件事，对于其他那些已经事实暴雷的地产商，此前并没有抓人的，接下来也都会参照姮大的案例抓人回来重判。

这种抓人重判的处理方式，其实也非常好理解，葆交楼已经葆不下去了，要将没有完工的项目完工，是需要花钱的，哪怕是此前的债务可以拖着，收尾工程的工程款是必须要给的，不然连工人都不会入场，暴雷地产商的现金流已经枯竭，不可能掏出这笔钱。截止到现在这一刻，姮大那些稍微好一点的项目都完工了，剩下的全是硬骨头，必须要净府掏钱的项目。

这些项目合计达到778个，就按每个项目的收尾工程只需要花5000万来算，这里就要390亿，其他暴雷地产商的情况和姮大差不多，也就是说，每家地产商都要投上百亿，合起来就是上万亿。从现在的政策走势来看，净府已经准备掏这笔钱了，筹钱的办法其实也已经明确了，钟秧净府发行一笔万亿规模的长期特别国债，由秧行直接认购下来，然后秧行凭此印出这几万亿的钱，交给净府去完成葆交楼的工作。

不过按照财政的规矩，在净府出这笔钱之前，必须将前后的责任划定清楚，绝对不允许拖泥带水，也就是说，以净府出钱为节点，前面负了多少债，有哪些违规操作，哪些人要对此承担责任，承担什么责任，必须一一明确，将这些事情都明确了，净府才能掏钱出来填坑。这种财政惯例，就是将前后的责任进行彻底的隔离，后面的介入的人不能背前面的锅，前面的责任没有追究清楚，财政部门绝对不会对项目拨出一毛钱。

所以写到这里，大家应该可以很容易理解接下来的局势了，有了最高捡那句可以抓人的公开表态，地芳净府一定会展开大规模的抓捕，将暴雷地产商一网打尽，全部重判，能判多重就判多重，有多少罪名就给他们判多少罪名，唯有如此，才能把前面的责任都清干净，把所有的锅都背走，这一步走完，接下来才能谈财政资金进场葆交楼的事。
