
# 万物兴歇皆自然：米联储见闻20250120

今天，美国新总统正式宣誓就职，开启自己的新任期。川普的团队已经发出声明，将在今晚的轻功宴上发布上百条行政令，明天上班的川普将会是忙于签字的一天。而就在川普上台的前三天，真是各种魔性操作，先是TikTok关停一天立刻又重新恢复，川普来了一把英雄救美，顺便甩出了自己的解决方案，也没什么新鲜的，无非是全卖还是卖一半以及最后卖给谁的问题。接着是川普推出了以自己名字命名的加密货币，创造了一日增值300亿的造富神话，这笔操作让资本对金钱聚集的想象力又再次起飞上升到一个新高度，在国家信用都不再靠谱的当下，个人借助流量和技术平台享受到铸币的红利，短时间内制造暴富效应，相比之下什么宗教邪教国家组织，都弱爆了，这才是硅谷精英们看中川普的关键所在，他们知道美元迟早完蛋，他们要在美元完蛋之前变现美元信用背后的铸币权，某种程度上来看，美国的各路精英就是支撑美元信用的砖石，在美元大厦将倾之前，精英们正在加紧建设自己的资产诺亚方舟，避免在美元崩溃的大洪水中让自己的财富彻底崩溃。川普的上台，对于精英们践行这套暗渡陈仓的操作会大开方便之门，尤其是川普自己已经身先士卒。

川普去年7月在纳什维尔举行的比特币会议上表示，如果他重返白宫，他将要求政府建立大量比特币储备，以吸引加密货币爱好者并利用该行业在政治舞台上日益增长的影响力。这次帮川普发行川普币的代币网站称，已发行约2亿枚数位代币，未来三年还将发行8亿枚。"这个带着特朗普表情包的数字币颂扬的是一位无论遇到任何困难都不退缩的领导人。"该网站还刊登有一份免责声明，指出该代币"无意成为投资机会或证券的主题",并且"不具有政治性，与任何政治竞选活动、政治办公室或政府机构无关"。言下之意，我本无价值，但你也可以投资。实际上，川普拥有该代币80%的股份，这类代币完全遵循丛林规则，一般是投资者利用炒作抬高价值，然后在市场顶端抛售，后来者只能在价格暴跌时点算损失，完全是刺刀轮盘游戏，跑不出来就被碾成肉泥，而这种几乎等同于野鸡赌场一样的游戏竟然是美国总统在带头玩，川普想要给这个世界带来什么一目了然，那就是真正的丛林游戏，弱肉强食，在强盗和流氓的操作中让美国再次伟大。

拜登的监管机构曾经透过起诉有关交易所并打击加密货币公司时以防防范金融机构有组织的利用数字货币进行诈欺和洗钱。但川普现在的努力目标是将美国变成"地球上的加密货币之都",一场关于新铸币权的血雨腥风，会吸引全世界最贪婪和大胆的冒险家。美国这样每个毛孔都留着贪婪血液和基因的国家，必然会在所有选择中选择风险最高的模式实现帝国的扩张。

除了川普币，懂王在就职前还做了一件大事，跟TikTok联手导演了一场英雄救美的苦情戏，被最高法禁令逼停的TikTok应用仅仅停服一天就重新开始运营，配合懂王登记，TikTok上演了一出绝地逢生的苦情戏，让美国民众不得不三呼懂王万岁，在TikTok重见天日之时，将功绩算在懂王身上。TikTok也将这次停服作为一次大型的推广路演活动，显然在停服时字节方面已经和川普达成了一定的意向甚至是协议，tt的停运声明就称美国一项禁止TikTok的法律已生效，以及"川普总统表示，他将与我们合作，在他上任后找到恢复TikTok的解决方案"，"请保持关注!"可见tt方面是有备而来的。昨天中午，恢复使用后，tt的声明是"根据与服务提供商的协议，TikTok正在恢复服务。我们感谢特朗普总统向我们的服务提供商提供必要的澄清和保证，他们不会因提供TikTok而受到处罚。"

实际上，该法律规定将对分发或更新任何TikTok内容的应用商店以及甲骨文等其他互联网公司进行处罚。川普周日在社交媒体平台Truth Social上表示，他将于周一发布一项行政命令，"延长该法律禁令生效前的期限，以便我们达成协议，保护我们的国家安全。"他计划安排一项交易，让美国拥有与TikTok相关的新实体的50%所有权，但目前尚不清楚该交易将以何种形式进行。他写道："没有美国的批准，就没有TikTok。有了我们的批准，它的价值将达到数千亿美元。"显然这是川普给出的叫价，相比于拜登的全部吃掉，川普显然做出了让步，毕竟是数千亿的大生意，更何况现在即便关掉tt，好处也是归小红书，一定要在美国人发现B站，微信微博等更重要的社交平台前，稳住美国人民，说到底，这背后就是利益分配问题，显然马斯克非常馋，早先传出真要卖只卖马斯克的说法，这是什么公开分赃的操作，显然中方拒绝了这个提议，现在美国甩出50%的方案，中方的回应是要符合中国法律，这里面就涉及一些具体的技术如何分配，股权如何分配的问题，相对于不卖就禁，川普分一半的操作显然属于可以谈的范畴，对于中国来说，控制权显然是第一重要的，引入美国的战略投资原本就是可以接受的，关键是技术的主权问题，资产的控制权问题，所以，从tt的问题来看，中美高层已经有过深入的接触，因为这个问题现在的解决方式，显然已经是中美互相通气后的共同表演了。根据美国媒体报道，川普去年12月16日在佛罗里达州庄园马阿拉歌会见了TikTok高管。当天，该公司要求最高法院暂时阻止规定1月19日为最后期限的法律。据两位了解双方接触情况的匿名人士称，在那次会面前后，TikTok高管当时就表示，应该由川普来决定TikTok的命运。川普随后告诉记者，他对TikTok有"好感",并表示他将研究这个问题。

自从川普赢得大选以后，美国作为一个公知嘴里的法治国家典范开始一路狂飙，各种骚操作层出不穷，现在问题来了，在tt的问题上，如果换个总统就换个结果，这还算哪门子的法治，这连人治都不如，人治还要讲政策一贯性，还要装一下过渡一下，哪有一换领导所有政策立刻24小时反转的，但是按照美国的法律，似乎又不是不行，因为美国这个国家颁布一个法律跟我们发一个通知一样简单，严肃性还不如我们的通知，因为按照新法律中的一项规定，总统可以对禁令发布一次性的90天延期，如果他或她向国会证明正在进行"合规的资产剥离"并且在此期间完成。但目前尚不清楚在法律已经生效的情况 下他是否可以行使这一选择权，tt禁令在国会得到两党的广泛支持，由拜登总统签署，现在也得到最高法院的支持，所以从法治的角度来看，如果现在懂王一句话就可以让禁令失效，那么美国法治就是一个巨大的笑话。所以现在美国的法律专家正在绞尽脑汁帮川普圆这个法治的骚操作，如果用法律的话语系统编织成花言巧语让懂王的意愿名正言顺，如果这个事情最终按懂王的意愿操作成功，那么意味着美国的所有的现行法律可能都可以按照目前这个路径完成反转。毕竟连堕胎权这种东西都可以取消，还有什么不可能，未来公开击毙拘捕的非法移民可能都将合法化无罪化。实际上，美国已经发生多起在街头蓄意伤害华裔甚至致死的刑事案件，但是目前针对这一系列案件不管是侦办过程还是审判几乎都是龟速，美国本土媒体有意规避，华裔群体又担心国内的人看笑话，也选择静默，未来随着川普在法治领域的骚操作，民间伤害非法移民和有色人种的执法尺度必然会越来越松。只能当初拜登以安全理由封禁tt 本身就已经是对美国第一修正案的严重侵犯，如今在走完所有行政和司法程序之后颁布禁令再被新总统取消，只能法律在美国就是政客随意打扮的小姑娘，甚至是党争的重要工具，竞选的时候用法律来起诉川普，当选后判刑有赦免，真是跟闹着玩一样呀。

不过对中国来说，美国政客这笔烂账我们不关心，我们比较关心的是，川普上台以后，我们要做什么。这川普就职破天荒的邀请了不少国家领导人，我们派出了副主席韩正参加，算是例外，在就职礼前一天，韩主席安排了两场会面，跟副总统万斯谈的是中美贸易问题，跟马斯克谈的是美国企业与中国合作的问题，但实际上中美两国现在要解决的不是这些具体的问题，而是沟通渠道的问题，实际上川普团队点名要蔡奇参加就职典礼，就是希望仿效当时沙利文和王毅搭建沟通渠道来搭建一条懂王团队与领导的单线沟通渠道，之所以会有这个诉求，就在于美国政治力量的复杂性，川普需要国家层面之外的沟通渠道。但这次不但没派川普团队期待的蔡奇，甚至没派外交方面更实在的外长王毅，而是相对更虚的韩主席，说明中方对于中美建立新的沟通机制这件事并不着急，而是有听其言观其行的想法，毕竟川普在第一任期时，推翻了以往的中美沟通机制。2017年4月习川会在海湖庄园会晤时达成构建中美外交安全对话、全面经济对话、社会和人文学科对话、执法及网络安全对话四个高级别对话机制的共识。但拜登上台后，又否定了川普新建立的沟通机制。这种反复无常的操作直接导致当时的杨主任在2021年3月中阿拉斯加州安克雷奇举行战略对话时拍桌子骂娘，创造了中美自建交以来从未出现过的奇观。所以，对于四年一次又换人，中国领导人也累了，加上川普还提名了被中国制裁的卢比奥做国务卿，中国就更是不想随美起舞，颇有点爱咋样咋样 的意思。在台湾问题上立场明确的中国肯定不会主动解除卢比奥的制裁，如同美国至今没有解除董军的制裁一样，那么中美未来到底以什么机制进行沟通如今仍是未知数。这个沟通机制如何建立，才是中美关系接下来最重要的看点之一，此外，明天川普要公布100多个行政令，这中间有多少直接与中国相关，可能也是领导考虑到底要从哪个角度与川普进行交涉的基本盘。
