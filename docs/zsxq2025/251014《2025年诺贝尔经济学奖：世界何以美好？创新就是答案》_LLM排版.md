
# 2025年诺贝尔经济学奖：世界何以美好？创新就是答案

## 获奖学者

2025年诺贝尔经济学奖的得主有三位：美国西北大学的乔尔·莫基尔、法国巴黎法兰西学院的菲利普·阿吉翁，以及美国布朗大学的彼得·豪伊特。他们的研究成果解释了创新如何驱动经济增长。

## 核心成果

- **乔尔·莫基尔**：从历史角度研究技术进步的先决条件，特别是“有用知识”的积累和传播如何推动工业革命。他发现，命题性知识（解释“为什么有效”）和技术性知识（关于“如何做”）的结合是技术进步的关键。
  
- **菲利普·阿吉翁和彼得·豪伊特**：通过数学模型解释了“创造性破坏”如何让经济持续增长。他们论证了竞争和创新之间的“倒U形”关系，即适度的竞争能够激发创新。

## 创新与经济增长

- **莫基尔**强调“有用知识”的重要性，指出启蒙运动带来的科学知识传播是工业革命的重要推动力。
  
- **阿吉翁和豪伊特**提出“创造性破坏”的概念，认为新技术不断取代旧技术，新产业不断取代旧产业是经济增长的本质。他们通过数学模型论证了适度的竞争能够激发创新，而过度竞争或保护落后产能则会抑制创新。

## 社会启示

- **莫基尔**的“文人共和国”例子说明，知识自由流动、同行评价体系和非经济激励是推动创新的重要条件。
  
- **阿吉翁和豪伊特**指出，社会应鼓励创新，帮助转型，提供再就业培训，而不是保护落后产能。

## 总结

三位获奖者的研究共同揭示了创新驱动经济增长的机制，强调了制度、文化和知识积累的重要性。社会应鼓励创新，适应“创造性破坏”，实现从“积累型增长”到“创新型增长”的转变，从而实现持续进步。
