
# 中国造船业世界第一？

近年来，中国的造船业虽然崛起了，但这仅仅是从造船吨位上来说的。全球造船业的产能主要集中在亚洲，前三强分别是中国、韩国和日本。如果从造船三大指标来看，中国远超韩国与日本，连续15年全球第一。根据工业和信息化部的数据显示，2024年我国造船完工量、新接订单量、手持订单量三大指标分别占世界市场份额的55.7%、74.1%和63.1%。引人关注的是，2024年我国新接订单量和手持订单量均创中国造船史的最好水平。最新数据还显示，2024年，我国造船完工量同比增长13.8%；新接订单量同比增长58.8%；截至12月底，手持订单量同比增长49.7%。在全球18种主要船型中，中国有14种船型新接订单量位居全球首位。让时间倒转回2023年，中国就是全球唯一造船三大指标实现全面增长的国家。其中，最新数据显示，上海三大船企2024年累计交船69艘，新接订单128艘，同比增长19%和70%。其中，中高端船型达到98%以上，实现效率、效益双增长。

不过客观看中国造船业，“中国制造”的大部分实际还是一个拼劳力的船壳子。里面的设备尤其是主要设备，很多都是进口的外国品牌。有报告显示，中国船舶配套本土化率平均只有54%，相对日韩85%以上的水平仍有较大差距。除船舶动力系统及装置、甲板机械、船用舾装件等配套设备本土化率超过50%以外，其他船用设备本土化率水平普遍较低。电子电气设备的本土化率只有13.31%，舱室设备的本土化率只有18.8%左右，通讯导航与自动化系统本土化率更低，仅有1.66%；尾轴密封装置、船用电梯、装卸机械泵空压机、海水淡化装置、空调及冷藏设备、通讯导航控制系统及电子电气设备等配套设备主要依赖进口；船用柴油机二轮配套能力仍显不足，优异零部件仍然依赖进口。

以一艘大型集装箱船来说，数十万吨的体量，其中的主要设备，包括：锚链是德国产的，锚机是波兰产的，甲板设备是麦基高，也就是美国产的。船用柴油主机，是瑞士产的，WIN GD系列。船用发电机是德国产和日本产的，MAN品牌以及西芝。船舶的锅炉是韩国的。焚烧炉则是丹麦的，阿特拉斯品牌，最高焚烧温度可达1200度。热交换分流体设备是瑞典阿特拉瓦制造的，净油机是三菱重工生产的。船舵是德国产的，贝克牌。舵机系统是日本产的，川崎重工制造。螺旋桨也是日本制造的，日本品牌。侧向推进机，韩国现代重工制造。舵机操控系统，日本制造的。主机控制系统，日本制造的。侧推控制系统，还是日本制造，NAKASHIMA品牌。三部雷达，全部日本制造，JRC的品牌。一部电子海图系统，日本制造，JRC的品牌。船用自动监控系统，日本制造。GPS系统，日本制造，JRC品牌。主罗经和分罗经系统，都是日本制造。甚高频通话系统，日本制造，JRC的品牌。

看了这一系列的“外国制造”，可见造船业也是一个系统制造部门，“中国制造”的部分，还是主要体现在中国的强项部分，劳动力廉价，主要像盖房子一样，是在造船的壳子，然后再搞系统大集成。未来随着劳动力成本越来越高，中国造船业的全球竞争力优势就会下降，而劳动力更为廉价的国家，将会崛起。

由于核心技术不足以及配套设备的生产问题，中国在高端造船业方面还存在明显的短板。中国造船业有四种船型规模最大，分别是散货船、油船、集装箱船以及汽车运输船。其中散货船新接订单量占全球总量的79.6%，油船新接订单量占全球的72.1%，汽车运输船新接订单量占全球的82.7%，集装箱船新接订单量占全球总量的47.8%。不过，我国在大型邮轮、大型液化天然气（LNG）运输船等高端造船业方面仍有明显欠缺。比如，在高端船型设计方面，前期基本设计还依赖于欧美设计公司，核心技术创新能力和关键系统集成能力还不强。

以豪华邮轮为例。虽然中国第二艘国产大型邮轮已经完成了主体贯通，但是从总体来看，实现豪华邮轮的完全突破，还需尚待时日。一艘豪华邮轮的零件数量多达2500万个，其中供应链涉及几千家厂商。而中国在豪华邮轮的零部件制造和配套方面经验不足，相比欧美国家，核心零部件供应商的体系不够完善，合作关系也未成熟。最重要的是，豪华邮轮设计难度非常高，一艘大型邮轮的设计草图可能多达几万张，不仅要考虑舒适性，还要保证每个舱室都能满足安全标准，同时需满足美观和实用的双重需求。豪华邮轮市场早已被欧洲国家垄断，目前全球能够建造大型邮轮的国家仅限于芬兰、意大利、德国、法国和荷兰这五个欧洲国家。意大利的芬坎蒂尼船厂、德国的迈尔船厂是全球邮轮建造市场的主导者，大约97%的世界邮轮船队在欧洲建造，98%的在建订单也将在欧洲建造。

此外，中国造船行业生产效率大大低于发达国家，尤其是造船每修正总吨工时消耗、全员造船效率等效率效益指标与造船强国的差距仍然较大。有资料表明，中国造船行业生产效率约为发达国家的1/7-1/10，若按人均产值或人均产量对比，仅为发达国家的1/20或更低些。有研究资料指出，目前9家大型造船企业的造船水平与国外的差距约为14年。在人均年造船吨位、人均年产值和生产效率3项指标上，造船企业与国外先进造船企业相差5-7倍。据测算，如果日本生产效率为1，韩国为2/3，中国则只有1/7-1/5。中国在分段建造过程中尚未做到跟踪补涂，分段二次除锈比例达60%-80%，甚至达100%，涂料消耗系数达1.9-2.0。而日韩造船企业普遍采用机械化钢板预处理、分段制造过程中跟踪补涂方式，分段二次除锈面积在20%以下，涂料消耗系数仅为1.2-1.5。中国造船企业的大量浪费、重复施工和返工，使工时浪费达55%，实际工时约为日本造船企业的8-10倍。尽管部分数据资料较早，但是如果中国在这些核心技术方面发展比较缓慢，那么造船业生产效率的提升自然也会打折扣。

总体而言，中国造船业从三大指标看是连续多年处于世界第一位，但是在其他方面并非处于世界第一位，主要包括核心设备的本土化率不高、高端化明显不足以及生产效率偏低等。当世界在感叹中国造船业崛起，甚至开始设置障碍，抑制中国造船业的发展之际，我们自己更要全面的、系统的去了解中国造船业。否则，一叶障目，盲目乐观，那就更为别人制约中国造船业提供了借口。现实中，中国的造船业，实际只是全球航行船舶产业中的一个重要环节，存在着一荣俱荣，一损俱损的关系。

最终分析结论（Final Analysis Conclusion）：
中国造船业虽然崛起了，从吨位三大指标看是连续多年处于世界第一位，但是在其他方面并非处于世界第一位，主要包括核心设备的本土化率不高、高端化明显不足以及生产效率偏低等方面。因此，全面的了解中国造船业才能更加客观，才能理解全球航行船舶制造业实际是一个大系统。
