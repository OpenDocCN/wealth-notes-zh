### 超级个体/数字游民/一人公司/创业型公司的核心

在说明链路和实际操作前，先对商业有基本了解。创业项目/个体项目都有清晰模型，基于个人对商业的理解，从钱的流动拆解商业逻辑：

1. 这个赛道里有多少钱？
2. 用什么去赚这个钱？
3. 跟谁收这个钱？
4. 怎么样更高效的去收钱？
5. 怎么样收到更多的钱？
6. 怎么样赚得更稳定？

这六个步骤涵盖赚钱所有要素，按顺序思考和解决，用【价值体系】代入，得到简版商业模式路径——蜂房理论：

- 需求：这个赛道里有多少钱？
- 产品/解决方案：用什么去赚这个钱？
- 获客路径：跟谁收这个钱？
- 运营方法：怎么样更高效的去收钱？
- 增长路径：怎么样收到更多的钱？
- 壁垒构造：怎么样赚得更稳定？

价值创造=需求×产品，价值传递=获客，价值交付=运营，价值放大=增长，价值保护=壁垒。

### 如何开始？

各位可以代入目前阶段，跟着公式和思路拆解项目或业务，相信会有收获。

### 0-1阶段：目标→验证

0-1阶段核心目标是赚到钱=价值×交换=（需求×产品）×（获客×转化）。验证项目是否符合价值交换原则。

很多人赚不到钱，不是不够聪明或执行力问题，而是对赚钱理解出现误差。赚到钱是结果，需要具体完成过程，过分关注结果而忽略过程。

### 价值交换

创业初期常听到做好两点：

- 真需求和好产品

本质是价值创造，对项目和个体都关键。做项目要花时间和精力最多的地方也是这两点。公式内点以乘法结合，任一项为0或偏差，结果归零。

#### 需求

一句话概括：“某个场景下的某个问题”。拥有这个场景和问题的人是目标人群。

如何发现需求？强烈推荐梁宁老师的《真需求》，对需求和价值的阐述非常清晰。举例：

在筹备结婚时，我发现自己缺钱，开始在网上找副业。发现多数项目不透露概率问题，只讲成功案例。副业和赚钱项目是【概率游戏】，胜率受认知、资源、经验、能力影响。意识到需求后，我发现两个需求点：

- 如何快速分辨项目可行性？
- 如何快速分析成功率？

寻找对应答案，发现市面上缺少【教会一个人从零开始赚到钱】的产品。

#### 产品

产品不仅指实物商品，问题答案、复杂事件解决方案也可视为产品，是需求的响应载体，价值强度取决于产品和需求的契合深度。

一句话概括：“你能解决某个问题到什么程度”。产品设计最花功夫，但初期可寻找靠谱后端或初级产品。实物产品要考虑功能价值、情绪价值、资产价值。

以我为例，设计产品针对三个核心：

- 自主学习能力
- 自我秩序感建立
- 高效验证方法

#### 获客

获客本质是内容能力，要求内容能吸引和筛选目标用户。非营销专业出身，以所思所想去制作内容，存在偏差。理解本质后，获客动作就好理解：

根据获取用户场域不同，分为公域私域；根据获客内容不同，可分为文字/图片/图文/视频/直播。本质上需要在内容中植入吸引点，设置筛选逻辑，产出为选题。建议从微信中同龄段和同属性的朋友开始。

#### 运营1

运营是思维方式，代表效率。0-1阶段核心重点是转化。很多人认为朋友圈做营销转化不好，但获客阶段0-1核心重点是找到最容易感知产品价值、最愿意尝试、最能提供反馈的种子用户。这类用户一定可以在朋友圈私域中找到。

我在测试课程阶段做法：在微信中找了25个人，邀请试听3节课，给出反馈，并定价。根据反馈制作后续课程产品。找的对象情况如下：

- 5个在读大学生
- 5个工作3-4年的职场新人
- 5个工作10年以上的职场人
- 5个创业者
- 5个自由人

根据反馈和实践评估，判断课程吸引程度排序为：职场新人＞职场人＞大学生＞创业者＞收租人。23个人给出90分满意分值，2个人给出帮助不大评分，均为收租自由人。25个人共收1200元，平均价格48元。决定设置为9.9元，并设置千元社群作为高客单转化路径。

0-1阶段基本跑通，通过需求设计解决方案，私域快速获客，形成转化。

### 1-10阶段：目标→运营增长

拿到最初级结果后，围绕职场新人制作选题内容。选题内容逻辑简单，既然要教别人赚钱，案例最好就是自己。在公域破冰需要力度，参考富豪谷底求翻身节目，设置自己的破冰内容。
