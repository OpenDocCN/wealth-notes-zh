
# 楚国青蝇何太多 (1)

过去一个周末，可把懂王忙坏了，总结起来三件事，搞人，搞事，搞钱。先是哈佛禁收留学生一日游，禁令还没热乎，览砐加 筷院否了，接着是欧洲50％关税一日游，跟冯娘通完电话就推迟到7月了，然后是川普币持有者总统招待会，臭名昭著的孙晨宇借着川普币又春风得意起来，不断在社交网络直播聚会，还晒出了川总亲自送给 的价值10万美元的川普黄金陀飞轮腕表，这1850万美元真是没白花，买币送餐位还有实体礼物可以发朋友圈，作为美国总统，川总对于一切愿意为他花钱的金主都是超级n ce的。至于自己在外交内政方面各种埋雷惹祸，川总丝毫不在意，现在最明确的目标就是，能不能给自己搞钱。川总这种不顾一切的任性玩法倒逼纽约时报这样的民主党党报开始认真讨论如何推翻专政，并列举了美国过去在推翻全球其他专政政府和进行颜色革命时所采取的一些策略。

之前网上疯传6月份会有6.6万亿美债集中到期，制造市场的预期缺口。但其实美国财政部官网数据显示，美国6月到期的短债(B I ls)规模为1.27万亿美元，美国6月到期的中期国债(No es)规模为1757亿美元。6月美国没有到期的长期国债(Bonds)，最近一笔到期日是2025年8月15日。这也就是说，6月到期的美债规模仅为1.27万亿美元的短期美债＋0.18亿美元的中期美债，合计到期1.45万亿美元。不过这是基于4月美债报告统计的数据，过去两年美国借了大量短债，每个月都进行大量的短债滚动操作，都有上万亿美元的短债到期，然后每个月再新借上万亿美元短债来借新还旧。预计实际6月美债到期规模应该是2.3万亿美元左右，相比网传的6.6万亿美元相去甚远。

民主党可谓是用尽心力粉饰太平，让全世界看不到帝国的附骨之姐。奈何川总为打破旧世界而来，一登台就开始主动狂摇台柱子，全然不顾自己还在台上的现实，恨不得快点把台子摇散，自己趁火打劫捞点好处，否则真是无法解释川总上台之后的一系列蜜汁操作。现在的问题是，全世界都知递美债如果崩盘，全球必然地动山摇，因为有太多资产和金融产品是建立在美债基础之上的。川普现在就是以大而不能倒肆无忌惮的要挟各位，逼着大家出钱出力帮美国解决美债问题，所以加速在摇美债的台柱子，因为台子上不但有美国，几乎囊括了除俄罗斯朝鲜伊朗之外的大多数国家。

2024年美国政府债务占GDP的比率达到124.30%，接近史上最高水平，2020年疫情期间的126.3%。美国之所以去年开始大量借短债，这个主要是因祸联。昭郧气储局2022年开始激进加息。面对如此高的利率，美国财政部自然选择多，少发长债。当前美国财政部每年要支付的利息，也上升到1.1万亿美元，约占财政收入的1/4。这是川普天天喊鲍威尔降息的关键，因为不降息，美债成本大高，发债难度就大，但如果降息，现在超过5的国债都没人要，降息之后美元如果绷不住，美债更是瞬间要面临地动山摇了。现在长债几乎是扯着嗓子叫卖都不好使，短债数额越来越大，这两年几乎都是大量发行短债，不停借新还旧来维持，从而出现每个月都会到期天量规模美债的现象。2025年3月、4月美国国债到期额分别为2.36、2.8万亿美元。对美国财政部来说，每个月都背着要借2万亿美元的kpi，贝森特能不疯，关税不得往死里搞，川总现在还能跟大家说人话已经是极大的克制力，谁坐在他这个位置上都不可能淡定的，只能说，民主党在挖坑方面确实也是一流的高手。
