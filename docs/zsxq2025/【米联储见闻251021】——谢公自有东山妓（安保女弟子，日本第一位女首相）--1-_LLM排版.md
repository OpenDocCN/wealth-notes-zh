
# 【米联储见闻251021】——谢公自有东山妓（安保女弟子，日本第一位女首相）

日本选出了历史上第一位女首相，高市早苗在这个敏感的历史时刻成了日本沉没的替罪羊。高市这次能够当选，是自民党多位党内大佬撑腰的结果，选举当日，前首相麻生太郎、菅义伟及岸田文雄都站在她的背后。64岁的高市早苗在自民党内属鹰派，在政策上是安倍晋三的继承者。媒体给她 的头衔就是安倍女弟子，但安倍其实在外交策略上不但不算鹰，反而是柔性外交，其实往回看，过去十几年，安倍在任时反而是中日关系最好的时候，之后的菅义伟和岸田都是典型美奸，石破茂骨子里虽然更平和，奈何一个亲华平和的日本首相是坐不稳日本首相的位置的。

今年是抗战胜利80周年，自1945年战败，日本经济陷入崩溃，但战败后的日本作为美国摁在亚洲的一枚定龙针，并没有真正承受战败的痛苦，反倒是在美国的帮助下快速崛起。

日本战败之后，经过20多年的发展，1968年，国内生产总值(GDP) 以1568.97亿美元超过法国的1297.85亿美元；1987年，以25810亿美元超过英国的7452亿美元；来到1996年；以49230亿美元超过德国GDP的27000亿美元，成为仅次于美国80,730亿美元的世界第二大经济体。日本在制造业、汽车、电子产品等领域表现卓越，推动了其经济发展，日本早在上个世纪就进入发达国家行列。这个过程中，全世界都把日本经济崛起当成是日本人努力的结果，甚至淡化了日本在二战中的巨大罪恶，日本自身也因为经济的快速发展而淡化了二战战败对日本的影响，毕竟日本所以发起战争就是为了解决自身的经济困局，如今虽然战败了，结果却是殊途同归，德国也是如此，所以作为二战的两个重要战败国，其实日德都没有真正意义上遭受战败的惩罚，相反，作为美国全球二战秩序的一部分，日德不但获得了经济的高速发展，甚至栖身发达国家行列，成为分蛋糕者，反而是二战的胜利国苏联和中国，因为政治体制和意识形态的差异被美国排除在二战治理框架之外，所以二战的战场部分虽然结束了，但二战的秩序并未理顺，日本不但没有遭受到战争失败的惩罚，反而一直把服从于美国当成二战结果的一部分，所以不管是前苏联还是中国，日本对于这段历史从未接受，也未认错，更没有道歉，至今仍在叫嚣台湾有事日本有事，从这个表述就可看出，二战其实并未真正结束。中国想要实现真正的统一，绕不开日本这块礁石，毕竟要彻底夯实正确的二战史观，并且彻底解决日本对二战的错误认识，从这个角度来看，一个鹰派的日本首相此时上台未必是坏事，有些时候彻底解决问题是需要一些外力来推动的，让一个偏激的女人来做这个催化剂，在日本方面，这是一个最好的替罪羊，在外部来看，有时候疯子才是最符合逻辑的存在。

日本政坛一直是男人的世界，日本可以说是东亚国家中最为重男轻女的国家，所以让一个女人此时成为日本首相，肯定是非常之时的非常选择。日本自民党已有70年历史，在日本政坛可说是一党独大，历来由男性出掌党总裁。加之日本既有的文化，形成了日本所谓的大男人主义。女性参政，特别是当政更加不易。高市早苗是日本女性政治人物，共当选7届众议院议员，曾历任首相府特命担当大臣、经济产业副大臣、自民党总务大臣、政务调查会长、经济安保部长等职。她的态度强硬，主张修改日本宪法，把自卫队改名为国防军队。她是日本政坛上少见的女性代表人物。而且这次日本出现女首相，不排除是给日本出现女天皇趟雷，她如顺利上台执政，或许可以让日本国民观察女性领导能力，进而将延伸未来天皇也可以由女性扮演，一改日本传统由男性接任天皇的规定，毕竟现在的日本天皇只有一个独生女。作为女性执政者，在男性为主的日本政坛，高市必然以更加强硬，更加偏激，更加保守的姿态来表明自己堪称大任的资格，在对华问题上会更加强硬，她选前表明，若就任首相，将在适当时机前往靖国神社参拜，刚过的秋季大祭，高市就提供祭品给靖国神社，而且力求要对日本宪法作出修改，加快日本走向军国主义的速度。因为日本政界似乎形成了只有极端强硬的女性才有出头机会的结构。在胜选过后，高市关于舍弃工作生活平衡的发言传遍全网，甚至遭到了石破茂的公开质疑。这是一种军队的连长层级的思路，而且还是败军的思路。只能靠展示舍身的觉悟来维持队伍统率，由此也可看出，高市实际的认知仍是上世纪的产物，怀抱着对企业体国家的憧景，真正是军国主义亡魂附体的僵尸政客。战前的日本正是如此：全民共享一个目标，国家替个人决定活法；化作一亿火球，不分公私从早到晚为天皇效力。正因如此，日本才打着富国强兵的旗帜，成为所谓世界一等国，却又为天皇暴走，陷入一场必败无疑的战争，进而把国家逼至灭亡的边缘，战后现行日本宪法应运而生，就是将日本从企业体国家转向广场型国家，这对日本来说是一场革命性的蜕变。广场型国家并没有什么全民共同目标。国家只负责制定参加广场活动的人应遵守的规则并处罚违规者，仅此而已；至于在规则之内该如何生活，由每个公民自己决定。然而，随着日本美国逐渐内缩，全球格局和秩序出现变局，日本重拾企业体国家的冲动越来越强烈，这种焦虑与逞强也渗透在高市的字里行间和身体力行中。如同当年是因为台独倾向才能当选的民进党一样，如今的自民党如果不重拾军国主义也保不住自民党的政治地位。

自民党内主导倒石破的那批人，显然是在谋求所谓安倍政治的复辟。石破茂偏离了安倍晋三铺设的轨道，传统支持层便流向参政党。看NHK放送文化研究所每五年一次的日本人的意识调查，2013、2018两年(安倍政权期)九成以上的人表示对生活满意，自民党每次都能在选举中取胜。但现在，日元贬值带动物价高涨，国民生活受到重创。安倍之后，自民党本身也出现政治裂缝，安倍作为岸信介之孙、安倍晋太郎之子，在自民党内天生具备一种正统性。之后的首相都是傀儡，相比于菅义伟和岸田，高市也缺乏政治世家的背景，是一枚纯棋子，甚至不如石破茂的政治根基深厚，所以，她的上台是自民党内部大佬多方衡量的结果，但高市自己却常以安倍接班人自居，这就危险了。最糟的是，她对概念的界定极其狭隘，论域也极有限。比如当被问到参拜靖国神社时，她笑着说“靖国是战殁者慰灵的中心设施，是和平的神社。”为推动《日中联合声明》与《日中和平友好条约》，中日两国为拔除靖国这根刺倾注了多少外交心力，她对这些外交积累既缺乏敬意，也缺乏理解，她这番话的恶劣程度近乎扇了亚洲一记耳光。

今年10月10日，石破茂发表了战后80年的感想。他从国内政治体制失灵出发，探究未能避免战争的原因。他以国务大臣单独辅弼制、统帅权独立等制度缺口为前提，认为在日俄战争前由元老、至1930年代由政党来弥补这些缺口并承担政策整合之职——这一理解，是有扎实学术支撑的解读。然而，本可实现政策整合的政党，却在同英美的裁军与对华的止战方针上屡屡败给军部的统帅。石破茂的肺腑之言表面上语重心长，但并未触及日本的对外过失。石破茂虽然强调唯有国民持续从历史中真诚学习为何未能止战的原因，才能奠定和平国家的基石。但对于这个为什么到底是 什么，石破茂既没有点破也不敢点破。事实上，即便是如此讳莫如深的演讲依然遭到日本国内保守政客和极右翼政客甚至是民众的不屑一顾和反对，所以石破茂的辞职并非偶然，强硬鹰派且有军国梦的高市登场自然也不是历史的偶然，没有群众基础不会有川普，也不会有高市早苗。

相比于安倍战后70年谈话中那句“不能让与那场战争毫无关系的后代子孙背负继续谢罪的宿命”，日本国内有一种集体共识，就是日本可以不再道歉了，正是在这种心态催眠下，石破茂在80年谈话中继续要求日本人反省的说法自然不受欢迎。德国哲学家雅斯贝尔斯曾 把战争之罪分为四类：一法律罪责——由战犯承担，而与后代无涉；二政治罪责——由全民承担，并由后代继承；③道德罪责——由良心审判个人支持或旁观战争的行为；四形而上的罪责——人类作为会引发灾厄的存在而背负的罪。战后40年的1985年，西德总统魏茨泽克据此提出：虽然战争的罪责无法由后代承担，但无论有罪无罪、年幼年长，我们都必须为过去承担责任，从而宣示了对罪责的承继。

日本厚生劳动省8月29日公佈的人口动态统计资料显示，2025年上半年日本新生儿(含在日外国人)数量为33.93万人，同比下降3.1%，创1969年有可比资料以来新低。此外，2025年上半年死亡人数为83.68万人，人口自然减少规模达49.75万人，创历史新高。一边是想要振臂一呼重振军国魂的政客，一边是国民连生育意愿都很低下的现实，高市的上台本身撕裂扭曲的日本一次政治意外，而这个意外必然会成为推动整个亚洲政治秩序和国际格局变化的重要推手。
