
# 扒一扒“零认筹”和搞万科的背后
## 来自： 南半球聊财经2
Jason不跪 2023年11月11日 15:21

王石说：经过这一仗，万科连接了天和地。万科是地，那么什么是天？

这两天，有一则新闻推送，位于深圳龙华的卓越柏奕府项目出现“零认筹”。我对目前深圳楼市没有什么太多看法，网上有不少人吐槽“当年嘲笑环京，后来理解环京，现在成为环京”的段子。倒是对“卓越”很感兴趣。

在克而瑞的1-10月百强房企销售榜单上，它排在第24位。但之前多年，鲜有出彩（查了下2020年还是40名开外）。较为有意思的是，在这两年民营房企普遍融资难的状况下，它还能在去年发行了一笔美元债，它敢发，也得有人敢买啊，一看文件，原来是招商银行提供了信用背书。所以，买它的美元债，等于在买低风险。

我们知道一般只有地方平台有这个荣幸，能拿到银行提供信用背书。为什么一家普通的房企可以？

不仅如此，去年中债增信的首轮融资支持6家民营房企发行了8只债券，卓越集团就是6家民营房企之一。我之前说过，国内的房企，不能只看排名和叙事，关键要看“内情”。

然后我就去查了它的历史——1996年成立，2000年就开始参与旧改，2009年香港上市失败，2012年集团销售破百亿，2019年老板被带走调查......

老板，叫李华。

是不是有点耳熟？还记得2018年东哥大洋彼岸的性侵案吗？据财新的报道，东哥当时出事第一时间就向李华救助，因为——之前所有扒房企的故事中，都会出现金融保险，李华也不例外，在招商仁和人寿的股东名单中找到了它，前5位股东，不是招商局、中移动这样的大央企，就是深圳国资——看来，东哥没找错人。

我查了一下李老板的早年经历，发现他和兄弟96年创立这家公司之前，是体制内工作。即使这样，似乎也没什么新鲜的。但直到我查了它的离岸资料，一切，就相当合理了。（大佬文集圈这部分过于劲爆...）原来是和万科的故事一样。

我在上篇扒一扒文章中，提及过当年的万宝之争，宝能作为野蛮人，背后由安邦加持（安邦当时的能量不可小觑），而万科能引入深铁成功夺回控制权，背后有一个重要因素——万年系。当时深铁的董事长是林茂德。他亦担任深铁一家子公司的董事长，而那家子公司，深铁初期占60%股份，合作方占40%股份。合作方，就是万年系。

而如果穿透万年系的主要台前人物，会发现.....一切，又合理了。它让央企华润改变态度、宝能安邦认栽、恒大贱卖股份给深圳国资的所有解释，都合理了。再查一些投资细节，发现万科、万年、深铁在金融、地产等领域都有交集。

在前海的一个基金里面，除了上述几家，还出现了明天系的身影。之前我在想许老板是聪明的，因为当时它买入万科的本意，应该是见机行事。后来一看王石背后的能量更大，于是就亏个几十亿做顺水人情。但现在一想，或许祸根就出在当时。

除此之外，许老板的这张照片恐怕也拍得不好——因为传说李老板曾经和范小姐，也爱得很深。（坊间还有一种说法，是许老板知道大D会已靠不住，尤其是郑老大去世，所以半年后选择把总部从广州搬到深圳，自然要建立新的“关系”，但从后来的1300亿战投组成来看，李老板和背后的神仙似乎没有动心，并且万科18年喊出活下来，也与恒大的做法反方向）
