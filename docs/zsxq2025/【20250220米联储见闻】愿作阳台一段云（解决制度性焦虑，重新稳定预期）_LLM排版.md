
# 愿作阳台一段云
# 米联储见闻20250220

周一民营企业家开完会以后，大家可以看到这几天A股的表现，总体来说，方向的博弈已经大致结束了。客观的讲，当美俄在利雅得举行谈判，阶段性的世界大战已经告一段落。接下来的全球叙事主要看两个点：欧洲是否会与俄罗斯产生激烈冲突；中国是否会在南海或台海发生剧烈冲突。但从客观角度来看，如果没有孤狼暴走，这两个叙事的可能性都不大，全球可能会进入暂时的修复阶段。这也是为什么中国在这个时候时隔7年又举行了一次高规格的民营企业座谈会，因为能不能避免这两个叙事的发生，实际上与中国能否再次依靠新质生产力再次成为全球经济引擎有很大的关系。客观意义上来说，即便美国有很强的遏制中国的意愿，但是最起码这几年是不会有大问题。不管川普政府有多想弄死中国，想和做毕竟还是两回事。接下来，中美俄三方的戏码大抵是中俄在与民主党和共和党的分别苟合，共同制衡利益关系。中国牵着民主党，俄罗斯搂着共和党，中俄手拉手，一起上演一场伦常关系非常复杂的国际地缘大戏。

外部大方向不是中国可以决定，所以这个时候更要做好内部的小环境。这次民营企业家会议总体来说，不管是参会人员安排还是会后市场反应都还是符合预期，资本市场的表现也算给力，对应板块都做出了反应。这次出席座谈的企业家有很多大家很熟悉的，但也有很多公众不太熟悉的，比如共享集团彭凡、银河航天徐鸣、恒力陈建华、横磁科技董清飞、传化的徐冠巨。除了新晋出位的大热新贵宇树科技和ds，很多过去只有业界知道的制造业巨头也浮出水面。共享是内地铸造领域龙头，新质生产力体现在其3D打印技术，在银川的铸造3D打印全流程智能工厂是全球首个万吨级铸造3D打印智能工厂。银河航天是国内估值最高的民营商业航天公司，主要是卫星互联网解决方案提供商和卫星制造，是中国商业航天领域第一家独角兽公司。恒力是以炼油、石化、聚酯新材料和纺织全产业链发展的国际型企业。拥有全球单体产能最大的PTA工厂之一、全球最大的超亮光丝和工业丝生产基地之一、全球最大的织造企业之一。恒磁科技在稀土永磁行业中占显著地位，为国内唯一一家大型MRI整机供应商提供主要磁钢。传化集团自主研发的前纺油剂、无氟防水剂等，主要是解决卡脖子难题，实现进口替代。此外传化最早打造公路港物流模式，布局73个公路港，建成全国最大的公路港网。

这次的座上宾没有卖货的，没有卖房的，只有造货的，而且主要是有全球话语权的高端制造业代表。科技领域是座谈会主力军，覆盖人工智能、新能源、半导体芯片、新能源汽车、网络安全、机器人，互联网与信息技术，智能制造与高端装备；新能源与清洁技术；半导体与芯片关键技术等。说白了，这就是中国新质生产力整体核心梯队和领军人物，是中美博弈下半场的关键力量。

这里面有任正非这样的老将，马云马化腾这种上一代目互联网浪潮的代表人物，雷军王兴的次兴梯队，还有王兴兴，梁文峰这种90后高科技新锐，整体展现出民营企业老中青三结合，有巨头有尖兵的全方位多元化的矩阵规模。这也是领导想要的效果，要大的我有，要强的我有，要新的我也有，要奇兵也不是没有。手里有这些精兵强将，在跟别人博弈的时候才能有底气。尤其是外媒格外关注的马云参会，对外资释放的信号也很明确，即便不太关心中国经济的海外媒体，都不得不因为马云的参与关注一下这个会议，这也是马云的价值所在。对马云秋后不算账就是给外媒送头条，表明毫不动摇地支持民营经济的方针没有变、经济制度根基没有变、「两个健康」方向没有变，开会就是在关键的历史时刻解决特定群体的制度性焦虑，重新稳定预期。这个中国在外交战线的打法是一致的，当美国在外交政策上高度不确定时，中国的主要打法就是高度确定，不但对美政策确定，对任何国家的外交立场都是一旦确定就不轻易改变，让不变成为大变局时的杀手锏，现在对内同样是用这条金律，当安全稳定成为稀缺资源时，不变和稳定就是最稀缺的属性和资源，仅靠这一条就能保证外资的信心。

不得不说，中国的民营企业家们也是相当争气的，中国制造2025的完美收官，能够在这个时候打出这些关键的好牌，跟这一批优质企业家不无关系。因为有了这群人中国才敢从中国制造2025转向中国标准2035，宁德时代攻克固态电池、比亚迪弯道超车特斯拉、deepseek改写AI规则，宇树抢跑机器人赛道，几乎都是在截胡全球科技预期，并且让全球不得不按中国画出来的赛道去重新调整。这次马斯克憋出来的grok3，是在ds穷追猛打下不得不赶工出来的东西，为了证明比ds聪明一点，用了将近200倍以上的资源，想要保住美国在科技领域的领先优势，但实际上这个匆匆上线的模型反向论证了美国大力出算力的技术路线并非最优解，甚至误入歧途，这种科技领域的短兵相接，一来一回，中国早已从美国科技学习者的身份转变成同一个擂台上你来我往的竞争者，不但打得有来有回，而且似乎并未落下风。此次参会的企业，绝大部分都有较大的海外业务，说明关起门发展从来不是管理层所要的，不但大企业要走出去，连小企业、独角兽都要有全球视角，面向全球市场，宇树和ds包括游戏科学，从定位开始眼睛就一直盯着全球顶级水平和全球领先市场，民营企业立足广袤国内市场，深挖内需潜力，同时推动高水平走出去，利用中国产业链优势加强国际合作才是未来中国民营企业真正的发展之路。

除了大强子这种因为风化问题被拍图的企业家无法上台面，曾经被调查罚款的阿里马云、腾讯马化腾、美团王兴都出席座谈会，不但表明平台经济治理已结束，也说明有错就改，既往不咎，管理层依然认可他们的成绩和价值。前段时间著名扭腰时报黑华写手袁莉看到美国难民涌入小红书，竟然酸不溜秋的写了一篇文章，说什么如果不是管理层阻拦中国互联网企业做大，怎么会失去让中国互联网称霸世界的机会，现在搞到要靠小红书来自嗨，这个角度之精奇，确实让人叹为观止。但是不管袁莉怎么黑，现在美国最红的社交媒体是tiktok，全球最关注的ai应用是ds，都是铁一般的事实。而这次发言的六位代表，也没有一个来自互联网行业，几乎全部是实业代表，都是硬核科技派，平台经济的社会意义虽然重大，但已经不是新质生产力的核心旗手和战略支柱了。而且随着ds的普及，内容产品的价值将大幅贬值，服务业的ai替代会是大趋势，溢价也会大幅下调，只有实业，具备高科技含量的实业产品才是未来有市场的硬通货。中国必须要从商业模式创新转向技术底层突破，从而实现对全球产业链和工业架构的重新洗牌，这也是确保美国不敢也不能对华挑起军事冲突的关键。

第一次工业革命是蒸汽机时代，带来的是英国崛起，成为全球工业强国；第二次工业革命则是电气化与自动化，美国接棒英国，直接晋级超级大国地位。第三次工业革命则是信息技术与互联网，美国继续领跑全球，科技巨头主宰世界推动全球化进程；第四次工业革命是互联网+智能化，中国开始踏上征程，参与重塑全球互联网格局，第五次工业革命必然是人工智能革命，这次革命谁扛大旗，谁是赢家，谁就是未来的全球高地。中国从二十年前不敢跟美国竞争，到十年前不想跟美国竞争，再到2020年后不得不跟美国竞争，走到今天开始主动跟美国竞争，从不敢还手到全面防守再到现在转向全攻态势，这个过程有很多挣扎犹豫反复，但是最终还是走在正确的道路和方向上，而且走的还算稳健，孰为不易了。

会想六七年，参加民营企业家座谈会的发言阵营，分别是东软刘积仁、万向鲁伟鼎、时代王小兰、安徽艾可蓝、恒瑞医药、星箭特种玻璃、商汤科技、通威刘汉元、众人科技，和同讯息等。整体来说这次的企业硬核科技含量要高得多，而且相比上次是在修仙问题上给民营企业吃定心丸，这次则是吹响集结号。经过这七年，如果说当时的中国民营企业家还在观望，甚至准备随时跑路，现在基本上已经彻底把这批人清掉了，这七年很多事情水落石出，能坚守的都算是经过考验的，哪怕不是百分百的同志，也算是志同道合的同路人，所以这次民营企业家座谈会的氛围和与会人员的心态也是完全不一样，如果真要找个类比，这次会议可以算是民营企业阵地的遵义会议，明确了到底是谁是中国真正的领导者，明确了要跟随谁去谋求未来的发展，才最符合中国民营企业的利益。只有明确了大家方向一致，利益一致，这个大有可为、大显身手，才能真正落地。

负责任的说一句，未来可能还会有风高浪急，但最危险的暗礁险滩阶段已经过去了，如果现在还不敢乐观预期，那就真的会失去通向未来的入场券。
