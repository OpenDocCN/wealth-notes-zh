
# 卢克文青云读书会第73期 环球军情《世界主要国家情报机构--古巴》

古巴，对于大多数中国人来说是很遥远的，在地理位置上，它确实离中国很远，但离美国很近。我在美国留学的时候，曾经去过弗吉尼亚的Key West，就是离古巴最近的一个美国海边城市Key West。它那个地方的最南面，有一个类似于我们海南岛的天涯海角的标志，也是一个打卡点，它上面就写着距离古巴70海里，这个地方是美国本土离古巴直线距离最近的一个地方，我在那还拍了一张照片。

我为什么要去这个地方呢？一方面当然是景色很漂亮，Key West确实景色很漂亮，而这个地方也是当年施瓦辛格拍《真实的谎言》的取景地。如果稍微年长一点的朋友看过那部电影，都知道施瓦辛格有一段是在海里的大桥上，用鹞式战斗机飞越了大桥，取景地就在Key West的附近，是一个州立公园，它有一座桥，那个桥有一半是断掉，后来又新建了一座桥，那个沙滩是白色的，非常漂亮。

当然我去，一个是因为看美国的天涯海角，第二个是在天涯海角的旁边，有专门放美国海军飞机的一个基地，就在景点的旁边，我顺便近距离观察了一下。后来我就跟当地人聊天，当地人跟我说，从古巴以前卡斯特罗革命前后，他们可以用小的渔船，一个晚上就能划到佛罗里达。现在美国国务卿鲁比奥的爸爸和妈妈，就是卡斯特罗革命前后，跑到美国来的，而鲁比奥就是出生在美国的第二代古巴移民。

另外一个比较有名的古巴裔的政治人物，就是做过佛罗里达州州长的德桑蒂斯，也是当年卡斯罗革命的时候来的。后来为了能够吸引古巴人投奔美国，美国后来就专门有一个法律，只要你能够从古巴出来，不管你是游泳、划船、开飞机，不管你用什么办法，你只要能够摸到美国的这个沙滩，你就可以被美国政府庇护。所以在迈阿密城里面有一个区叫小哈瓦那，我还专门去了那个区参观，那个区里面有三样东西很有意思，一个东西叫猪湾博物馆。

当年美国人搞猪湾行动的时候，他们聚集了一大帮古巴人，在中央情报局的支持下要攻古巴，要打败卡斯特罗，后来失败了，他们就在迈阿密的哈瓦街区里面建了一个猪湾博物馆。第二，小哈瓦那里面的古巴人玩一种游戏，这个游戏很像我们的麻将，但他那个牌很大，这有点像中国的麻将。第三就是小哈瓦那的冰淇淋很好吃，这个是他们社区里面三个比较有意思的事情。

迈阿密也好，佛罗里达也好，确实能够看到很多古巴的二代、三代，甚至现在的第四代都能看到。当然到了第三代、第四代有的已经不会讲西班牙语了，都只会讲英文，但是他们是老一代的还是会讲的。猪湾博物馆里面当然是带有一定的政治倾向的，我只是去参观，就看了一下他们对于整个猪湾行动，用他们的视角去怎么描述的。

当然古巴对中国公民是免签的，但是有一个问题就是你如果要去古巴的话，可能要先到美国，再从美国去古巴，或者是从别的国家再到古巴，比较复杂。但是现在的古巴确实能够看到当年20年代到50年代，非常具有美国风情的感觉，因为古巴曾经是美国大量投资的一个国家，所以它还保留了非常多西班牙殖民时期的建筑，以及后来美国人的所谓的美式审美。所以古巴是一个神秘的国家，而且很有趣的一个国家。

对于美国来说，古巴是一个非常小的国家，它为什么能够屹立在这个地方？作为拉丁美洲乃至于全世界反美的一个前线，他怎么能够活到现在？而且卡斯特罗，据说一生历经了300多次暗杀都没有成功，而且最后能够安然离世，没有死于非命，他离不开什么？离不开就是他高效且精干的这种情报系统。古巴虽然是一个小国，但是它却有着非常完善的或者非常高效的情报体系。

其中，1961年美国的猪湾行动中到底是谁像卡斯特罗透露了美国当时的整个作战计划，而谁又是扮演了古巴007的这个角色。到目前为止，古巴和美国两边都对这个情报的泄露讳莫如深，谁都没有提。当然未来有可能随着美国档案的逐步解密，也许慢慢这个人会浮现出来。我想讲就是说虽然没有人知道古巴007，但是就从一个小事上，我们可以看出卡斯特罗对于美国情报的掌握程度。

1961年的4月21号，也就是猪湾行动失败后的第二天，有两名美国中央情报局的官员走进了设立在华盛顿特区的一家银行，他们以亚瑟阿维隆的名义开出了8张总额为10万美元的机票。当然在华盛顿特区的这家银行职员知道，这笔钱是中央情报局用来支持反卡斯特罗政权的流亡者的。然而一个星期以后，这名银行职员从华盛顿的一家媒体上看到了一则美联社记者发自哈瓦那的报道，他说，卡斯特罗发表公开声明，严厉谴责中央情报局策划颠覆古巴社会主义政权。

同时他说，中央情报局是以亚瑟阿维隆的名义向反古巴社会主义力量提供资金援助的。也就是说亚瑟阿维隆的名字在一周以后卡斯特罗就已经知道了，而他也非常清楚知道这笔钱是由中央情报局支付给反古巴的这帮流亡者的。短短一个星期，卡斯特罗就已经知道了整个事情的细节，他怎么做到的呢？某种意义上有点像公开的情报站，我已经知道了你想在干什么，某种意义上甚至是一种威胁美国，就是你想干的事情我都知道，连这样的细节我都知道。

古巴现在有两个主要的情报机构，一个叫古巴军事反间谍局，一个叫古巴情报局。古巴的军事反间谍局，创立于1962年，它主要是为了能够保护古巴社会主义革命成功的成果。他早年受到苏联克格勃的影响，苏联克格勃早期为古巴军事反间谍局进行了早期的人员培训。进入到80年代，尤其是到了后冷战时代，古巴的军事反间谍局就开始向经济情报领域，经济情报活动开始转变。

当然其中有两件比较著名的案件，足以看出古巴军事反间谍局的能力。其中最著名的是当时美国国防情报局的高级分析师门德斯，他为古巴从事间谍活动长达16年，前美国大使罗查后来被美国中央情报局调查出，他曾经过去为古巴从事情报活动长达40年，这也就凸显了哈瓦那针对美国乃至于美国情报机构进行了非常成功的反渗透作战的行动。

更有意思的是门德斯在被起诉以后，当法官问到他为什么要为古巴从事叛国行为的时候，门德斯说我没有拿过古巴人一分钱，我完全是出于意识形态，我无法接受美国政府长期且系统的制裁、歧视古巴政府和人民，在从事情报活动的行当里面来说是极其罕见的。他为古巴服务了16年，但是他没有拿过古巴人一分钱，而他本身自己是美国国防情报局的高级分析师，他不是为了钱的，因为他也知道古巴也不可能给他很多钱，他完全是看不惯美国长期对于古巴的制裁。

因为美国对古巴的制裁已经超过55年了，这可以说是这个世界对单一国家制裁时间最长的一个记录，就是美国对古巴的制裁。古巴的军事反间谍局是隶属于古巴武装力量司令部，它主要从事是收集关于军事情报的，尤其是大家要知道，美军在古巴的关塔那摩是有一个基地的，而这个基地是当年美国和当时的古巴政府签了一个条约。

有点像当时将香港的新界租给英国99年租期一样，当时的古巴政府签了一个这样的条约，将关塔那摩基地租给美国99年，所以在古巴境内还有一个著名的关塔那摩基地。而关塔那摩基地最有名的就曾经大量关押了所谓的基地组织和本拉登的恐怖分子，这个地方也成为古巴情报机构系统且长期进行监视和情报活动的一个主要针对对象。
