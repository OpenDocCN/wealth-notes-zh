
# 华瀚健康涉50亿港元洗钱案香港被宣判主脑仍在通缉中

经过六年调查被破获的香港最大洗钱案，背后是绵延近十年的贵州明星药企掏空在港上市公司资产案。这起案件被香港警方视为破获的规模最大的上市公司洗钱案，涉案金额高达50亿港元。

45岁的被告人名为王名俊，曾在联交所退市公司华瀚健康任职，案发时担任华瀚健康财务总监及董事会秘书。从案情细节看，2015年1月至6月，华瀚健康通过发行可转债、以“二股供一股”形式募集共计38亿港元，募资用途声称用于公司营运，包括兴建新医院、医疗中心及扩展物流网络。然而，募集资金在一个月后就开始被管理层非法挪用，随着资金转移，华瀚健康不久就走上了停牌退市之路。

2016年9月23日，这家贵州省的明星药企因审计报告出现异常而暂停交易，被联交所停牌。创始人邓杰、张岳为管理层的核心人物，2019年2月被香港证监会认定涉嫌披露虚假或具误导性的资料以诱使进行交易的罪行。不过，香港证监会至今未直接对上市公司及高管进行处罚。

华瀚健康在2019年遭香港法院下令清盘。同年，香港警方接获近百名投资者报案，直指公司账目有问题，怀疑管理层挪用资金。财新曾发布独家报道揭露，该公司管理层如何分批掏空上市公司资产，低价转至内地关联方，暗中实施资产大转移的内情。

香港警方商业罪案调查科诈骗案调查组的相关人士在新闻发布会上表示：“这宗案件的调查始于警方在2019年接获的报案，当时距离案发已相隔多年。案件牵涉范围极广，涉及投资者、公司高层、监管机构、金融机构及清盘人等。为了厘清每一笔财务交易的真实背景，尤其判断其是否具有合法目的，调查人员必须仔细检视大量财务文件与记录，并对公司多个账户的资金流向进行系统、深入地分析。”

王名俊被判入狱7年8个月，并被禁止在未来12年内出任公司董事。判刑后，警方表示将申请充公部分涉案资产，并透露案件其他主脑仍在通缉中。

## 财务总监当庭认罪

2015年1月，华瀚健康发行6.2亿港元可转债，由建银国际与华融国际各承销3.1亿港元。当年6月，华瀚健康又以“供股”形式再募资32亿港元。表面合规的资本运作背后，公司管理层却上演了“左手倒右手”的资本骗局。

王名俊作为该公司财务总监，在上述融资完成后，并未将资金按公告及发售文件所述的用于业务发展，而是转移至多个由高管控制的银行账户中，这些账户以个人或海外公司名义开设。警方核实相关公司既非上市公司子公司，也未见于财务记录中的合法业务往来对象。

8亿港元中的大部分资余被转入两个可疑账户，其中，一个账户属于被告担任唯一董事及股东的一家英属维京群岛注册公司在港开设的银行账户；另一个为被告本人在港的个人账户。

王名俊当庭承认，他在2015年2月至2018年12月期间，与华瀚健康前行政总裁邓杰共同参与洗钱活动，利用三个工商银行的账户，洗钱涉及约46.79亿港元、3460万美元；此外，王名俊个人还独自洗钱约5375万港元。

复杂的跨境资金流动，使得案件调查的难度和复杂性显著增加。警方因此花费了相当长的时间进行分析与取证。经过抽丝剥茧的调查工作，最终成功将被告人起诉并定罪。警方表示，华瀚健康的清盘人正通过民事程序追讨部分资产，剩余资产警方将申请高等法院发出充公令。

## 转移资产同步进行

华瀚健康通过上述38亿港元融资再加上历史滚存的资金，截至2015年6月底，账上有66亿港元现金可以投资。“事后看，这件事最可气的就是，华瀚健康管理层一边在实施资产转移，一边还在上市平台利用资本市场融资吸了一大口血。”华瀚健康的投资者表示。

也就是从2015年开始，华瀚健康将资金和核心资产有计划、分步骤地转移出华瀚体系，投向内地多个项目，包括分次剥离上市公司的优质药业、保健品、生物科技资产，将其切块出售给看似独立的老熟人，日后再重新接回自己手中。类似的操作在近三年内至少发生了七次，以德昌祥药业和汉方制药的交易最为典型。

剥离核心资产的同时，华瀚健康管理层处心积虑在A股寻找标的，希望将被转移的资产注入内地上市公司，股权分散且无实控人的嘉应制药被选中。从2018年2月开始，华瀚健康原高管团队有预谋地控制了A股上市公司嘉应制药的董事会，试图将被转移的资产在内地借壳上市。当盗窃行径被曝光后，该交易被迫终止。

被华瀚健康原管理层转移的较大体量的资产涉及七大中药、保健品和生物仿制品企业。此外，还包括六盘水市凉都人民医院、广东康慈医疗管理公司旗下的两家老年痴呆(阿尔茨海默病)康复医院，三块价值数亿元的医疗康养项目土地。前述资产或转入邓杰等人关联方名下，或暗中操作签署协议、具体金额外人无从知晓。

## 资产追索新机会

在王名俊宣判后，华瀚健康的资产追索有了新突破。“管理层的行为被定性为刑事犯罪，并且香港警方还在通缉涉案的主要人员，涉及邓杰、张岳的洗钱资金或超过10亿元，这是追索的新方向。此外，两人过往签字的各种资产腾挪的股权交易，也需要重新审视。”一位知情人士表示。

华瀚健康的创办人之一张岳担任董事会主席，曾任贵阳市云岩区第五届政协常委。邓杰作为行政总裁，负责集团的战略规划及营运管理。尤为亮眼的是，邓杰1985年毕业于北京大学法学院，曾于1998年任贵州省第八届及第九届政协委员。截至2015年6月30日，张岳持有华瀚健康26.29%的股份，邓杰持有0.06%。据财新此前多方了解，2014年以来，邓杰成为华瀚健康的核心人物；2018年后，因涉华融赖小民和白天辉案，邓杰被边控至今。

在香港清盘人进场五个月后，内地经侦部门对相关管理层的职务侵占行为正式立案。一份贵阳市观山湖公安分局的立案告知书显示，2019年12月26日，邓杰、张岳等人涉嫌职务侵占，符合立案标准，特此告知。此前，华瀚健康中小投资者于2019年11月中旬在贵州报案。

相关调查一直持续至2022年，投资者得到的是“邓杰、张岳等人无罪”的结论。在2022年10月24日由贵阳市观山湖区人民检察院出具立案监督审查通知书中，明确表示，现有证据尚不足以证明邓杰、张岳利用职务便利侵占公司资产。投资者则反复上诉，认为无论是募集资金还是资产转移的股权，都有太多疑点。

在追索的另一端，2019年8月，香港高级法院指派临时清盘人正式进驻华瀚，控制了除内地之外的公司账户。由于内地和香港两地的法系不同，香港法院指派的临时清盘人尚无具体操作以有效控制华瀚健康实际控制人对内地资产的腾挪和资本运作。

值得注意的是，2023年7月，华瀚健康清盘人之一陈浩然（Michael Chan）在内陆被协助调查、限制出境至今，具体原因不明。另一位清盘人徐丽雯在香港办公。在这样的纠葛之下，华瀚健康已资不抵债，整体的资产追索进度不佳。
