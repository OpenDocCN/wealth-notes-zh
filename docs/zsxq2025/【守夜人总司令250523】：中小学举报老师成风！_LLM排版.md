
# 中小学举报老师成风！

原创：觉悟者 公众号：守夜人总司令 25 年 5 月 23

## 前言
不用怀疑，就是为了让你们主动放弃。还记得上世纪八九十年代发生的事么？辈子能赶上两次，也是前无古人后无来者…

## 一
中小学老师陷入到了种战战兢兢的工作状态。他们有KPI压力，但是不敢管孩子。少数调皮捣蛋的孩子，可以绑架整个班级。当老师向学生提出学习要求的时候，往往就会被少数家长举报。这些特别热衷于举报的家长，孩子各方面的表现往往不咋地，成绩也不咋地。所以，这些父母在捏老师这个软柿子的时候，可以找到种变态的心理平衡。

不管有理没理，老师只要被投诉，上级部门肯定会优先处罚。这就造成了老师需要不断的自证清白。而且在把握整个班级的学习进度的时候，需要尽量照顾拖后腿的那个群体，而不是成为火车头的那个群体。因为节奏太快，后面的人跟不上，往往会被投诉区别对待或者说歧视。

如此来，整体进度和水平取决于最糟糕的那群人，而不是最努力的那群人。为了避免节外生枝，惹祸上身。许多老师会主动选择多事不如少事。
