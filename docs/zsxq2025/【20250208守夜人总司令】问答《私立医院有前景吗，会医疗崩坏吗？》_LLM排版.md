
# 问答《私立医院有前景吗，会医疗崩坏吗？》

## 星友提问：
司令您好。我是一名临床医学的学生，预计8年后进入重庆的医疗行业。想请教下：
1. 未来中国私立医院以及外资独资医院是否有发展前景？
2. 在三明医改集采和人口老龄化下，中国医疗行业的未来是否会类似于日本90年代，呈现所谓的“医疗崩坏”？如果可能，我未来希望在重庆从事更加市场化的医疗服务，能请您给点建议吗？

## 司令回复：
对于国家而言，接下来的医疗资源需求会非常大，因为老龄化非常集中，这种情况下最核心的问题就是谁出钱呢？即便公共医疗体系有这样的能力，也承担不起这样的成本。所以，会进行一系列的改革来压缩成本，每一项改革的最终目的都是为了将成本转移出去。

私人医院和外资医院肯定有比较大的发展空间，因为私人医院和外资医院能够有效地分担这些成本。如果官方提高收费标准，那将是一个非常严重的社会问题，但是通过第三方来提高收费标准，就能变成一个市场状况下的自由选择！无论是私人医院还是外资医院，出任何矛盾和冲突，官方政府也更容易以正面形象进行干预和监管，如果是自己的下属单位，这种干预和监管，就很容易变成大事化小小事化了的人情世故。

权力源于组织规模，把自己的下属组织拆散，就是在削弱自己的组织规模，除非这种行为能够聚集起更大的组织，否则没人愿意这么干。反腐的目的是为了让组织的凝聚力更大，聚集起更多的资源，进一步壮大组织，也就进一步的扩大和巩固权力。

言归正传，古往今来的所有治理问题，归根结底都是一个成本转移问题。面对一个社会结构性矛盾需要解决，作为治理者，核心手段无法是人群打散和把时间打乱。所谓把人群打散，就是把人的需求和满足需求的方式进行分流。所谓把时间打乱就是分批处理，不要一起处理。如果在某个时间窗口内满足人群需求的成本不够，那就用外力按住，就像车站进站一样，一批一批的放进去，这种模式主打一个托字！

现在社会的养老金和医疗方面，也只能这样：一是想办法尽量拖，二是想办法分流。拖可以把一些人拖死，死了就不需要了，分流可以让人在追求差异化服务的情况下自愿承担成本。
