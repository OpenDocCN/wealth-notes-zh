
# 鉴茶财富进阶250220: 强压力位

3360，好几天上不去，站不住脚，这就是强压力位。因为3360左右，是套牢盘密集区。

上去了，拉平1008中间就没什么阻力了。现在的情况，你可以理解为，孩子非得考上985高校，高考一次失败了，复读一下，再失败，再复读，但孩子一直挺努力的。努力的标准是什么？有量，今天1.7万亿。2万亿以上，牛市。1.5-1.9万亿，小牛。1.5万亿以下，弱市。如无外盘幺蛾子，预期下周二三左右，小调整告一段落，向上或向下的方向确定。
