
# 大树乡谈11.7：中国不再寻求新的特殊和差别待遇，有何影响？

一位读者提问：中国不寻求新的特殊待遇后，将对国内相关行业产生什么影响? 先说答案：影响可以忽略不计。别说“不寻求新的特殊和差别待遇”，就算现存的特殊和差别待遇，也没多少实际获利了。

下面展开谈。对这个问题，小镇在星球9月24日文《中国没有放弃发展中国家地位》已经谈过，中国宣布“中国作为负责任的发展中大国，在世贸组织当前和未来谈判中将不寻求新的特殊和差别待遇”,并不意味着中国放弃发展中国家地位，这个道理很简单，就不多展开了。

今天重点说“不在寻求新的特殊和差别待遇”对中国的影响。

这个待遇是关税总协定和世贸组织给予发展中国家成员的优惠待遇，按照世贸组织秘书处总结，截至目前一共六类157条，分别是： (1)旨在通过市场准入来提高发展中国家贸易机会的条款； (2)要求WTO 成员确保发展中国家利益的条款； (3)管理贸易措施的规则和纪律方面给予发展中国家灵活性的条款；(4)允许发展中国家享有更长过渡期的条款；(5)技术援助类条款； (6)专门适用于最不发达国家的规定。

这些规定的名称看起来有点绕，实际除了“最发达国家成员”根据联合国每三年调整一次的名单而外，全球并没有明确的发展中国家定义，基本采取各国自认的原则，而且每一个发展中国家在世贸组织获得何种特殊和差别待遇，也都是单独谈的。

所以除了第六部分“专门适用于最不发达国家的规定”仅适用于最不发达国家成员，具体规定可以来自前五部分；其他发展中国家享受的待遇总结起来无非四部分：可以获得更灵活的政策空间、享受更长的承诺实施过渡期、得到更多的技术援助支持、在贸易争端或谈判的时候能够获得更多便利。

实际谈判的时候，也是要讲策略的，并不是说一个国家说我发展不好，所以所有的特殊待遇我都要，其他国家不会同意的。比如中国当年入世的时候，就有舍有得，而且中国入世时开放水平就已经非常高了，体现在只要求了很有限的差别和特殊待遇，可以说已经接近底线标准，如果不坚持，会对中国经济经济社会稳定发展不利。其他但凡能够开放的，都开放了。

最有代表性的是关于农业补贴的微量许可标准，这是世贸组织《农业协定》规定的配套制度。按照《农业协定》,世贸组织成员国应该约束并削减对农业的价格干预和补贴，并且通过综合支持量来评估各国农业补贴的大小。但留下了一个口子，那就是就是综合支持量不超过某种农业商品或者农业生产总值的一定比例，那就视为“微量许可”,不计入综合支持量，通常而言，发达国家上限是5%，发展中国家是10%。

中国入世的时候，通过放弃部分特殊和差别待遇，换取了将微量许可上限提高到8.5%,这是为了保护中国农业。但中国的保护几乎是最低底线标准，因为在入世的时候，中国就已经承诺放弃使用农产品出口补贴的权利，而其他世贸组织成员国普遍到2015年甚至2018年才承诺取消。

对这一点，也可以理解为，决策层很清楚中国未来需要大量进口农产品，自然也没有对农产品出口进行补贴的道理。不过这在当时确实可以作为谈判的筹码，毕竟通常情况下，发展中国家出口粮食换取外汇是普遍情况，比如印度粮食出口在外汇收入中就占据重要地位，每年出口的大米占全球市场份额的三分之一左右。

所以，小镇才会说，中国在入世的时候，很务实，开放度也很高，这在关税约束和承诺开放领域上表现得尤为突出。

比如在关税约束方面，也就是承诺的关税上限，中国低于全球绝大多数发展中国家，甚至比挪威、以色列这类发达国家还要低。2001年签署入世议定书时，中国承诺2010年关税算术平均水平降至10%，还承诺2020年进一步降低到5%。

但2018年之后，美国发动贸易战，直接打断了继续下降的进程，但中国目前算术平均实施关税也已经降到7.3%，比不少发达国家还低，比美国强多了。而在开放的领域方面，中国也基本对标了发达国家标准。比如中国入世的时候，对100个服务贸易分部门做出了开放承诺，当时发达国家平均也就是108个分部门，现在中国几乎是全部开放。

至于反倾销、补贴、反补贴等等核心条款上，中国处于发展中国家最高档水平，基本对标发达国家。所以说中国在入世的时候，就没享受多少特殊和差别待遇，随着中国的发展，也不断提高对外开放水平，承担更多国际义务。比如中国在2024年底已经向所有已建交的最不发达国家提供100%零关税待遇，2025年6月还对53个非洲建交国实施100%税目产品零关税。

还要关注国家方向变化，2023年中国调整了对国际高标准经贸规则的看法，从认为这些新规则是针对、打压中国，转变为认为这些规则基本符合中国下一步内部改革的目标，可以借助国际高标准经贸规则加速国内相关领域改革，并且有利于在美国搞逆全球化的情况下，加速中国与世界的联系。之后中国对外就转变为进一步高水平对外开放，推动国内自贸区对接国际高标准经贸规则、申请加入CPTPP等。

比如2023年11月就印发了《全面对接国际高标准经贸规则推进中国(上海)自由贸易试验区高水平制度型开放总体方案》，经过一年多试点后，2025年7月推动所有自贸区全面对接国际高标准经贸规则，文件题目就很清楚了，《关于做好自由贸易试验区全面对接国际高标准经贸规则推进高水平制度型开放试点措施复制推广工作的通知》，开头第一句就强调“主动对接国际高标准经贸规则，实现规则、规制、管理、标准相通相容，是建设更高水平开放型经济新体制的重要举措。”

所以，中国并不是脑子一热，突然就说不再追求新的特殊和差别待遇，而是一步步稳扎稳打，到了2025年时机已经成熟了。

说得直接点，中国早就不再追求新的特殊和差别待遇，之前获得的不少也主动放弃了，现在不过是把已经做得，公开承诺出去。

可以这么说，到2025年，中国实际享受的差别和特殊待遇条款，已经寥寥无几，而在近些年与各国谈判的时候，早就不追求差别待遇了，而是给其他国家差别待遇，也是一定程度上弥补中国竞争力太强。就像中国早在2018年就举行了全球首个进博会，主题就是“新时代，共享未来”，展现中国推动建设开放型世界经济的决心。

现存特殊和差别待遇也很有限。参考2020年《中国外汇》期刊刊文《美国不承认中国发展中国家地位的经贸影响》，当时总结涉及发展中国家特殊和差别待遇条款共145条，比最新版本的157条少了12条，不过这不重要，因为新增的中国本来就没去追求。根据这篇论文的统计，在美国不承认中国发展中国家地位后，中国实际仅在“管理贸易措施的规则和纪律方面给予发展中国家灵活性的条款”下获得特殊和差别待遇，而中国有需要的只有12个具体条款，并且有六项是《农业协定》下的。

从2020年至今，中国开放度更高了，到了2025年，需要的特殊条款就更少了。

所以，就可以回答这位读者的问题。

中国不再谋求新的特殊和差别待遇，对中国经济社会以及各产业领域，可以说没有影响，因为中国一直按照自己的节奏，不断实质性放弃、削减这些差别待遇，也早就不再追求新的差别待遇。

现在的中国，哪怕是顶着某些国家的针对性政策，照样可以在全球取得竞争优势，反而是很多外企在中国竞争失败，不得不退出，这也体现在这些年外企在华营收萎缩的状况。这不是外企看空中国经济，而是竞争不过，导致利润下降，减少了例子利润的再投资。

从中国整体而言，接下来的重点是大出海、走出去，要主导全球经济，作为能够跟美国正面硬刚并且取得优势的超级大国，再跟其他发展中国家抢特殊待遇，实在太掉价了，也不利于赢得全球国家的尊敬。中国是建设、发展型国家，不是美国这种剥削、吸血型国家。

但是做出这一承诺仍然值得称道，因为除了韩国自我评定成为发达国家，目前中国是世贸组织所有发展中国家家中，第一个公开承诺不再寻求新的特殊和差别待遇的国家。
