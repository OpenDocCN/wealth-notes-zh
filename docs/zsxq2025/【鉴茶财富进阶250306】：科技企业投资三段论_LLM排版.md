
# 科技企业投资三段论

村长今天在发布会说，大家要冷静看待科技企业的风险。觉得贵说明你包容力不行。有些公司你的确看不懂，看不懂就对了，说明要砸钱，直到你看懂。科技企业的投资一般有三段：

第一段是概念阶段，要么属于讲故事，要么是对标美国搞大的，通常在烧钱，没盈利，没商业模式，甚至没成品。反正自己也不知道咋赚钱，但是有技术，这时候拼的是相信，是市梦率阶段，没法估值，你觉得行就行。具身智能，6合成生物，就属于这个，官方叫做“G，未来产业”。这时候就是炒概念，没人知道谁跑出来，就是看谁命好，寒王有芯片，沐曦和摩尔线程也不错，但人家先上市啊。

第二阶段是实现阶段，开始有了小范围的产品，某个领域的技术大面积成熟，能兑现了，这时候就看之前那些市梦里头，谁能跑出来，谁有大厂订单，谁的业绩能放大。这时候又能炒一波，有人会算你未来60年赚多少。

第三阶段是成熟阶段，产品大面积面世，各家的业绩趋于稳定，这时候就纯看资产回报率，你的股息率多少，现金流是不是充沛。

三个阶段怎么衡量呢？主要看渗透率。一般来说：

阶段1，渗透率1%以下，看谁故事好；
阶段2，渗透率1-30%，看谁增长快；
阶段3，渗透率30%以前，看谁赚钱。

算力属于阶段2，A芯片属于1-2之间，机器人，6G,人脑工程等，属于阶段1，起码还能炒5年。
