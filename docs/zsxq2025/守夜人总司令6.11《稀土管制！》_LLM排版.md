
# 守夜人总司令6.11《稀土管制！》

稀土管制！
250611 守夜人总司令
别的地方也有稀土，但是别的地方缺乏成熟的配套体系。像稀土这样的矿物质的充分利用，需要一套逐渐生长出来的生态体系，钱是一回事，时间是另外一回事！
我们中国的稀土管制会形成制度，这种管制的目的不是用来作为谈判的筹码，而是要形成产业链的长期依赖。
举一个例子，与稀土应用相关的原材料出口是禁止的，但是，用到这些原材料和相关技术的成品出口不禁止。抛开军工的应用，最大的应用份额就在新能源汽车上，如果我们保持现有的政策，并逐渐形成一套管制的制度，那么，海外的新能源汽车产业，为了应对这种不可控的非市场预期，不知道什么时候双方又掐起来了。
市场需求要求生产端具有稳定性，资本市场对于企业的估值也依赖企业的稳定性，如此一来，为了自身的利益，那些海外的相关汽车产业，就会把与之相关的技术迁移到中国来，然后在中国完成与稀土应用相关的生产，最后把成品出口到自己国家完成组装。
如果我没有判断错的话，这应该是我们中国制定相关政策的最终目的，因为对终端产品的限制不具有持续性。
当房地产熄火之后，我们中国急需要包括汽车在内的相关产业去拉动经济增长，如此一来，你不得不加大与别人的合作，甚至转移技术和生产线，在这个趋势不可避免的情况下，就需要在产业链的某些关键上游节点上形成绝对的控制优势。
这就好比我们中国今天能够设计出更好的芯片，但是我们中国造不出来，一方面是因为光刻机，更重要的是我们缺乏工业设计软件，这个脖子卡的比光刻机更厉害！光刻机限制的还是制造工艺，工业设计软件的限制会造成芯片设计图都画不出来，而且用别人的工业软件去画图的时候很容易造成安全隐患：曾经华为有一个芯片在设计完了之后，一切应用都正常，然而，当把华为芯片设计部署到核心服务器上去应用的时候，核心服务器会有一个暗藏的电路逻辑会自动向外传输数据。这个逻辑并不存在设计者的原稿中，是工业设计软件自己偷偷加的，只有在特殊条件满足的情况下才会被触发！
