
# 【20250207米联储见闻】——一呼三军皆披靡

从1月20日川普上台至今，仅仅18天，白宫记者们已经发出哀嚎，求求川总，能不能歇会，这工作强度谁顶得住，白宫记者都快累吐血了，新总统不但行程安排满，新闻份量重，关键还经常改来改去，记者们每天陪着川总连轴转，已经快挺不住了。不仅记者们挺不住了，美国国内、国外的观察家也几乎每日都处于震惊的友邦惊诧的状态中，内外政策各种离经叛道，匪夷所思，以至于记者们不得不当面怒怼川总，到底哪条法律允许你能这么做？急得民主党多等不等一天，已经开始着手要弹劾新总统。

前几天，川普说要让加沙人永久搬离加沙并由美国接管当地，被各方怒怼后，白宫发言人又说，川总只是想埃及和约旦暂时收容加沙人，国务卿卢比奥也将川普提议包装成一个重建计划，重建之后会让巴勒斯坦人回去。白宫行政管理和预算局上周突然发布指示冻结每年总额高达3万亿美元的联邦政府资助，其适用范围不明一时又引起大乱，结果迅速被法院叫停，两天后又收回指示。更离谱的是，为了打击中国电商，川总脑门一拍，不收中国包裹不就解决了吗？上午美国邮政就宣布即日起不收中国和中国香港的包裹，估计终于有稍微有常识的人告诉川普团队，电商包裹只是国际包裹中的一部分，全部绝收只怕要出大事，结果这条指示一天不到就废除了。紧接着，负责对外援助的美国国际开发署突然被关闭，网站也下架了，一大堆靠这个单位吃饭的NGO立刻断粮，多少公知在社交媒体哭爹喊娘，奔走呼吁，果不其然，开发署虽然暂停了，很快卢比奥又成了国际开发署的代理署长。

开发署雇员虽被勒令停工，但时间期限仅为90天，且国际开发署由国会立法创立，并不能由总统一声令下就废除，所以更大概率只是夺权，毕竟每年400亿美元的经费，花钱的权力必须掌握在川总手里，国务卿亲自代理署长，按照卢比奥今天最新的说法，这个部门的雇员将从14000变成249人，估计其中200人都是专门对付中国的，所以大家也别看热闹，这个部门以后只怕是更霸道更激进。

作为关税总统，川普从一上台就开始打关税牌，先威胁要向加拿大和墨西哥实施25%关税，关税的命令发布之后，眼见股市下挫，就同墨西哥总统和加拿大总理打电话，然后暂缓关税暂施30天，所谓的胜利只是加拿大早已通过的13亿加元边境安全拨款、任命一个芬太尼沙皇的虚职，以至墨西哥向美墨边境派出一万名国民警卫队阻止移民和毒品流出。马斯克和六小将在得到了财政部管控联邦政府近九成支出超过5万亿美元的系统存取权后，得到了数以百万计的美国人个人资料，还让马斯克取得了联邦政府采购名单的相关信息。白宫联邦人事管理局也由马斯克入主操控，上周该局通过电邮向200万联邦政府人员发出自愿提前离职电邮，承诺如果他们选择离职将发薪至本年9月，但国会并没有为此通过拨款。

川普上任不足三周发布的行政命令数目已超过拜登上任100日的总数，难怪白宫记者哭着求总统歇一歇，造成的混乱和惊吓程度比2017年首届任期有过之而无不及，新闻媒体几乎每天都是在选谁做头条新闻的焦虑中度过，毕竟要在关停国际开发署，对华加关税和针对加沙进行种族清洗中作出头条选择，还是相当困难的。也有分析人士认为，川普如此急就章的操作，就是让用疲劳战术拖死媒体，从而达到战术迷惑媒体，暗渡陈仓完成实际工作的目的，而提出这种打法的就是川普以前的智囊班农。班农一早就留意到，媒体和人们的注意力是有限的。要成功落实具争议性的政策，最好就是不断制造新的混乱和新的争议，使人們失去焦点，也就是说，真真假假搞一大堆，让你不知道他真正想做的是什么，从而减少舆论对改革造成的阻力，这就是班农所谓的“信息淹没战术”(flood the zone)。

每天搞出三件事给他们，他们只能咬住the zone其中一件，而我们就能趁机完成所有事情，一举搞定。这帮人永远——永远无法招架。攻势必须同时发动、持续重击、毫不间断，目前看来，川普正在完美践行这套战术，无论是民主党、自由派媒体，还是全球各国、国际组织和外交观察者，都被他绕得眼花缭乱，反而无法洞悉他的真实目的。这是典型的障眼法，所以党务之急是要看清楚到底什么政策是川普政府真正会认真实施的。

比如，这个国际开发署，一直是推动美国全球政策的桥头堡和重要执行机构，川普说国际开发署是在浪费资金，卢比奥直接说这个部门不服从命令，现在要全盘接手，包括马斯克的大规模裁员计划，其实都是夺权的一部分，要将不听话的人赶出系统。美国国际开发署于1960年代初成立，原本就诞生于冷战之处，表面上是负责代表美国政府管理人道救援项目，实际上就是介入全球各国政治的核心机构，根据美国国会研究处的资料，国际开发署约有10000名员工，其中三分之二在海外工作。该机构在60多个国家设有办事处，并在数十个国家开展工作，直接或间接管理和控制全球近千个各类NGO，是美国实现全球管控的重要触手之一，也是重要的情报收集和反馈机构。他们透过各种卫生文化教育环保弱势群体保护的机构与全球媒体也有着千丝万缕的联系。比如BBC旗下的所谓国际慈善机构“BBC媒体行动”(BBC Media Action)的部分资金就来自美国国际开发署，2024年国际开发署捐助了323万美元给这个组织，是该组织年度第二大捐助者。根据政府数据，2023年美国在国际援助方面的支出达到680亿美元，其中有400亿美元是拨给国家开发署，这可是个超级花钱的肥差部门，这400亿美元其实就是用于维系美国对全球实现控制和影响力的总经费。川普一直批评这个机构乱花钱，并列举了一些例子，比如向塞尔维亚一个“男女同性恋、双性恋和跨性别群体”(LGBTQ) 组织提供150万美元。实际上各种所谓社会群体背后，可能都有他们的影子，通过控制群体介入这个国家的内部管理。在必要的时候制造分裂甚至暴动。2019年香港反修例事件发生时，大量的NGO参与其中，不乏国际开发署的资金支持。

由于国际开发署的法律地位受美国对外援助法保护，所以川普大概率不会关闭这个机构，但会大幅精简，正如卢比奥所说，对川普有用的职能会保留，其中估计大部分都会于遏制中国有关。川普可能会效法英国首相约翰逊，把国际开发署作为国务院的一个分支机构，使其不再作为一个独立的政府机构来运作，约翰逊就曾将英国国际发展部与英国外交部合并。实际上也是大幅降低这个部门的权限。卢比奥表示，开发署的“每一分钱”都必须有证据证明能够让美国变得更安全、更强大、更繁荣，才能合理化支出。但卢比奥不会知道，一旦这个资金切断，很多国家内部设施甚至公务员工资都会停发，比如现在驻守叙利亚、负责看管数千名伊斯兰国囚犯的狱警就是靠开发署拿工资的，川普冻结资金90天，到底全球多少国家机构是吃狗粮办事的，就会纷纷浮出水面了。

最近和哪吒风头如此之胜，居然没有人跳出来阴阳怪气，说明确实是资金没到位了。川普上来以后，变得最温顺乖巧的就是台湾人，赖清德好久不说话了，这么听话也没得个好，最近川普再次提及，台湾在美国的芯片市场中占有98%的份额，将对台湾半导体厂征收最高可达100%的关税，加速推进台积电迁至美国，钱博士强调，美国原本期待藉由关税压力使台厂赴美设厂，不同于民主党花钱补贴芯片厂让他们搞生产，川普要直接很多，川总说，这些芯片厂很有钱了，为什么还要给他们补贴，老子就给他们加关税，我就不信他们不到美国来，此言一出，台湾股市随之大跌，大家都知道川总不是闹着玩，美国进口的逻辑芯片中，有44.2%来自台湾，记忆体芯片则有24.4%来自台湾。而台积电的前两大客户均为美国公司，第一大客户苹果约占台积电营收的25%，第二大客户则是Al芯片设计公司龙头英伟达，占台积电营收约11%。实际上，这两家公司都不是川普利益阵营的公司，如果川普对台实施高关税，最快遭殃的就是这两家公司，实际上，川普相当于敲打台湾芯片让这两家公司老老实实对他投诚。因为高关税本质上会抬高生产成本，与制造业回流美国是南辕北辙，所以其目的还是打击这些不在自己阵营中的科技巨头敲竹杠，而不是真的让制造业回流。

所以这十几天过去，川普出拳很多很杂，但真正有劲的并没有多少，而他到底要做什么还没有显山露水，跟这样的对手过招，就一定要稳住阵脚，不能随之起舞，我就静静地看你装逼，守好自己的门户，让自己先耍，你不动他就越发心里没底，反而越不敢动了。
