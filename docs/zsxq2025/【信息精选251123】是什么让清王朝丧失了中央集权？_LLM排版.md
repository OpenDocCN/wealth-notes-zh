
# 是什么让清王朝丧失了中央集权？

清朝不再是一个中央集权的政府。是从什么时候开始的？有两个标志：一个是湘军的崛起，一个是厘金的诞生。湘军崛起是地方督抚获得了兵权，厘金诞生是地方督抚获得了财权。兵权和财权都从中央转移到了地方。中央只剩下一项权力，就是任命官员职务的权力。从这一刻开始，清朝的格局就发生了改变。格局变成了不再是中央集权，而是中央权力轻，地方权力重的局面。

厘金是一项神奇的赋税。它帮助湘军，也是帮助清朝打败了太平天国。它也帮助李鸿章、左宗棠、张之洞等地方的实力派大臣开展了洋务运动。同时它也让地方督抚有了跟清廷叫板的底气。厘金到底是个什么样的赋税？它是怎么来的？今天我们就来聊聊厘金的故事。

要聊厘金，就要从咸丰开始说。咸丰自从登基开始，最让他焦虑，最让他发愁的事情就是找钱。咸丰是一个苦命的皇帝。他从登基到他去世，在他执政的11年里面，全国各地都在爆发动乱，其中还夹杂着英法联军的第二次鸦片战争。战争一直打到咸丰去世都没打完。战争是最消耗钱的，前线打仗资金消耗量大的惊人。所以从咸丰登基开始，他就一直在被缺钱所困扰。

清朝的主要税收是地丁钱粮，也就是农业税。每年能收上来的税基本上是稳定的。康熙还提出了一个「滋生人丁，永不加赋」的税收政策，这让税收更加固定。从康熙以来，清朝基本上每年的财政收入都是固定的是4,000万两左右。清朝每年支出也是固定的，也是将近4,000万两。支出的走向主要是官员的俸禄和军队的军饷。这样支出和收入都是固定的，好处是量入为出，问题就是缺乏弹性。一旦碰到一点天灾人祸，就容易引发财政危机。

在康熙、雍正年间，财务管理做的比较好，每年都能存下钱。户部的存银一直都在涨，最高峰时候存到了7,000多万两。到乾隆时期，乾隆好大喜功，大手笔开销，存银被花了一大半。到嘉庆、道光时期，有白莲教起义，有张格尔叛乱，还发生了鸦片战争，存银又花了不少，基本上快见底了。咸丰登基的时候，国库存银只剩下800万两了。800万两连维持朝廷的基本运转都困难，更别说镇压太平天国叛乱了。

咸丰登基的时候，国库存银只剩下800万两了。800万两连维持朝廷的基本运转都困难，更别说镇压太平天国叛乱了。咸丰先是想办法让户部筹钱，等把户部的钱都花完了，户部实在筹不到钱了。咸丰就把内务府的钱拿出来花，拿出来补充军费。内务府的钱实际上是皇室的私房钱。咸丰把私房钱都拿出来补充军费，能看出来他对平定叛乱的决心。但是内务府的钱对于战争的消耗来说，实在是九牛一毛，没撑几个月，内务府也没钱了。

咸丰的钱花光了，但是前线的战事偏偏是越打越严重。从1850年到1853年，咸丰用尽了办法，千方百计，一共弄到了3,000万两白银，供应前线打仗。换来的结果是什么呢？是清军节节失败，太平天国在南京定都。咸丰实在是没什么招数了，户部的存银只剩下29万两。29万两，他连京官和京兵的俸禄都发不出来了。

咸丰让大臣们出主意，大臣们就给他出了几个建议，这几个建议可以说是饮鸩止渴，但是为了救急，咸丰也只能批准了。第一招建议是官兵减少俸禄。从1853年起，咸丰下令，京内外文官武弁的俸禄都减少两成。朝廷明面上说的是减少两成，实际上剩下的八成也是经常拖欠。京城里面的京官还好点，京城外面的官员拖欠的更厉害。当官的当兵的都被减少和拖欠俸禄，他们的生活没有着落，他们唯一的出路，就是更加变本加厉的搜刮百姓。

第二招是卖官鬻爵。卖官鬻爵在清朝不是什么新鲜事，是自从康熙开始，朝廷就制定了一套明确的制度，什么级别的官，卖多少钱，都是有明码标价的。只不过卖官的名字不叫卖官叫「捐班」。所以在清朝，要想获得官员的资格，有两种途径，一种是捐班捐出来的官，另一种是科举考出来的官。科举考出来的官被视为是「正途」，捐班捐出来的官被视为是「异途」。所以捐班捐出来的官，一般不太受尊重，而且大多数都是虚衔，很少能有获得实职的。

尽管是这样，捐出来的官也是官的身份，至少可以跟官老爷平起平坐，见到官员不用下跪了。所以很多有钱人家，也会愿意捐班。从康熙、雍正一直到道光，捐班要花多少钱，都是明码标价的。到咸丰这他为了多收钱，居然减价大拍卖，把捐班的钱从1851年开始打九折，到1853年打八折，到1854年打七五折。打折卖官，确实是能多卖点钱，每年能多个几百万两。

但是还是远远不够。咸丰采用官员建议的第三招是铸大钱发银票。清朝的货币，是白银和铜钱双币种。为了搞钱，咸丰批准了大臣铸大钱的建议。从1853年开始，户部开始铸造「咸丰重宝」，也叫做「十铜大钱」。「十铜大钱」是一枚铜钱，可以抵原来的十文钱。「十铜大钱」虽然比原来的铜钱大很多，但是远远不值十倍的价钱，实际上啊，这就是官方通过制造通货膨胀来赚钱。

十铜大钱铸造了一段时间，朝廷还是认为成本太高，毕竟还是要用金属来制造。为了更进一步的减少成本，咸丰干脆批准了发行纸币。当时的纸币叫「户部官票」。户部官票是官方发行的纸币，面额有一两三两五两十两五十两不等。发行纸币几乎没有什么成本，可以让朝廷迅速的有钱。但是发行纸币的后遗症也大。我们知道，从金融的演变过程来看，从金属货币发展到纸币是历史的必然。清朝当时民间也有很多钱庄票号，发行自己的票据，这些票据能弥补金属货币的不足。但是纸币的发行必须要 有完善的金融理论来指导，必须得经过周密的设计，最简单的我们都知道，至少需要一定相当数量的保证金，还要严格的控制发行额度。咸丰发行纸币，他不是为了改善金融体系，只是为了应对财务不足，所以他们会滥用权力，发出的纸币没能力兑换，结果就是造成恶性的通货膨胀，直接掠夺民间的财富。

咸丰发出的纸币，民间根本就不认，但是官方的开支和工程，户部都是用银票来结算。户部一般都是用白银、银票、大钱搭配的方式来支付。比如说户部发军饷，发的是一半白银一半银票。官方发的银票民间不认，所以这实际上等于是只发了一半的军饷。再比如说，朝廷雇佣河工干活，他们发给河工的工钱，居然发的是两成白银八成银票，这等于是只发了两成的工钱。所以咸丰一朝，是清代历史上货币最混乱的时期。

咸丰这么一折腾，又是减少俸禄，又是卖官，又是搞通货膨胀，最后的效果也还是不好，他只是暂时解决了中央政府财政危机，对于提供镇压太平天国的军费，他还是毫无办法。在前线的将士等不到朝廷中央的拨款，只能依靠当地督抚解决。所以对于每个战区的督抚大员来说，头等大事就是筹集军饷。当时主要的战区是广西、广东、江西、湖南、湖北、安徽等南方省份，这些省份的官员各显神通，其中湖南、湖北的做法算是比较好的。

湖南是左宗棠开的头，我们说一下左宗棠的做法。左宗棠开源节流，他首先是整治贪官污吏。当时国家收的税其实并不高，但是老百姓实际交的很高，中间的部分，都被官员和胥吏给贪污了。左宗棠大刀阔斧的改革，他绕开传统的税收机关，重新组建一套由士绅组成的班子，用新班子来收税，这样就可以把贪官污吏贪污的那部分，用来充公。左宗棠这么做，是动了湖南所有官员的蛋糕，但是他也不怕，因为他有湘军撑腰，有刀在手，士绅和湘军的组合，是湖南的一股新势力，左宗棠是湖南士绅的领袖，他带着这批湖南的新势力，对传统官府衙门下手，把原来官吏拿走的蛋糕夺过来，变成了湘军的军饷。

左宗棠在湖南改革成功之后，胡林翼在湖北也进行改革。湖南、湖北为湘军筹集了大量的军饷。其他几个省的做法就不像左宗棠了他，没有进行大刀阔斧的改革，也没有去减掉贪官污吏的收入，他们的做法很简单，基本上就是以各种名目加税。真正受害的还是老百姓。所以贪官污吏的问题没解决，这就等于是税是多收了，但是多收的大头还是被官吏中饱私囊，只有一少部分用来支撑军费。地方官员就算再搜刮老百姓，老百姓每年的农业产出也是有限的，所以能被压榨的也是有限。

于是各省地方官，又把目光看向了富绅和商人。对于富绅和商人来说，他们用的办法是劝捐。劝捐就是劝他们捐班买官。卖官本来是朝廷中央干的事，原来的捐班流程是，各省把捐班的银子先上交给户部，户部收到钱之后，流程走到吏部，吏部来定给什么官职，然后请皇帝审批，皇帝审批之后再发回到各省，最后捐班的人才能得到官职任命的凭证，这种流程，走下来一遍需要很长时间，还经常有银子交上去了，然后没有下文。捐班的人呢，为了能尽快拿到结果，往往还得再花钱打点。

现在各省急需用钱，为了提高捐班的效率，各省督抚就请皇帝批准，让吏部下发空白凭照，地方督抚拿到空白凭照之后，就可以自己填写，然后定期汇总上报给朝廷，这样一来，绅商捐官，就变成了一手交钱，一手发凭照的生意。捐班的效率提高之后，地方督抚确实收入大幅增加，捐班收上来的银子，也成为了各地的军饷的主要来源之一。最早时期湘军的军饷，很大一部分，都是靠卖这些空白凭照来解决的。

对于绅商来说，捐来的官大多数都是虚衔，没有实职，他们就算再多捐一些，想补实缺，也得等到多年之后才有可能有机会。他们花上两三千两白银，捐一个七品知县的顶戴，最开始啊，感觉也确实还不错。但是等到捐官的人越来越多了，乡里面的知县满街走，甚至连知府、道员的顶戴都越来越多的时候，他们的快感也消失了，所以捐官的人也越来越少。捐官的人少了，但是打仗的军饷是一天都不能停，所以捐官就变成了，从他们自愿捐官，变成了官府带兵上门劝你捐官，实际上就是官兵勒索。

很多绅商怕官府，只能一捐再捐，用真金白银去换几张空头衔的黄纸。靠商人捐的钱还是不够，官府开始把目光，放在那些以前在外面做过真官的，退老还乡的官员身上。这些官员做过真官，根本就看不起捐班，对空头衔更不感兴趣，但是很无奈现在权势不再，现任官府的威逼勒索之下，他们也只能交出一部分财产。这里面有一个最典型的案例，是曾国藩的湘军劝捐勒索，居然勒索到了陶澍的儿子陶瑞的身上。

陶澍当过两江总督，他是近代湖南第一个做到督抚位置的人，连大名鼎鼎的林则徐，都是陶澍带起来的，陶澍为官清廉，对湖南的士子影响很大，陶澍去世之后，只剩下唯一的一个儿子是陶瑞，曾国藩的湘军居然到陶澍家，逼陶瑞捐款。陶瑞不捐，湘军就把陶瑞抓起来，诬陷他勾结土匪，最后陶家没办法，只能低头花钱消灾。因为这件事，左宗棠跟曾国藩开始闹翻，为什么左宗棠会跟曾国藩闹翻，因为陶瑞是左宗棠的女婿，当时陶澍和左宗棠结成儿女亲家，是轰动整个湖南的大事，现在陶澍去世，只剩下陶瑞一个孤子，居然被曾国藩欺负，陶瑞还不光是左宗棠的女婿，他还是胡林翼的妹夫。

有着这些关系，居然还能被曾国藩勒索捐款，从这就能看出来，湘军当时缺饷已经缺的非常严重，逼捐逼到这种程度，说明富绅已经被挖的差不多了，再逼再勒索也逼不出来多少钱了，所以从前线官兵想要军饷，必须开辟新财源，新财源就是厘金。

最早创办厘金的是一个叫雷以誠的人，1853年的时候，雷以誠的职务是刑部侍郎，他被朝廷派去扬州江北大营帮办军务，他的主要任务就是筹集军饷。筹集军饷是一件非常难办的差事，当时地方官府，已经把所有能用的招数都用过了，军饷还是远远不够。雷以誠的幕僚里面有一个叫钱江的人，这个钱江很有才华，也恃才傲物，他也是个传奇人物，是钱江给雷以誠出了个主意，他说咱们可以派官兵，到水路和陆路的要道去设卡，对通过的货物按照价值进行派捐，他说按照价值进行强行派捐，实际上就是商品过境税，当时的过境税被称作是「行厘」，行是行走的，行就是行走的厘捐，跟「行厘」相对的还有「坐厘」，坐厘就是对开店的商人进行收税，所以厘金最早叫厘捐，也是捐纳的一种，不过原来的捐纳都是自愿，即使是被威胁的，也是自愿捐的，这里面厘捐就不再是自愿的了，只要商品过境，只要是商人开店，就必须要捐，雷以誠听了钱江的建议，他开始尝试，1853年10月，雷以誠在里下河设立机构，开始收厘捐，实际上就是收商品税，最开始只是对米收税，每一石米捐钱50文，雷以誠本来是抱着试一试的想法去做的，没想到在半年之内，他就收到了2万串钱，他发现这个方法好，这个方法既不扰民，也不用逼捐，商人的怨言也不大，更重要的是这个方式稳定，以前逼捐的时候，有时候能弄到钱，有时候弄不到钱，收入很不稳定，收入不稳定，但是军饷的支出是稳定的，所以他们成天为钱发愁，雷以发现这个方式可行之后，他就给咸丰帝上奏折，他在奏折里面说，说这种方法细水长流，源源不竭，于军需实有裨益，他建议让江苏巡抚和南河总督，在各自的防区里面也试用，咸丰收到奏折之后，他并没有认识到厘金的特别意义，他把厘金当成是当时各种捐纳名目的一种，既然能弄到钱，咸丰就批准他可以继续使用，至于雷以誠建议的扩大范围使用，他也同意，他下旨让江苏巡抚和南河总督研究一下，如果可行，可以试用。

雷以誠收到咸丰肯定的回复之后，他开始在泰州设立分局，大张旗鼓的收厘金，他把收厘金的范围，从一开始的只收大米，扩展成各类的粮食、家禽，油、盐、茶、糖、丝、布、酒、药材等等等等所有的商品，只要是商品路过，只要是商家开店，他就要抽厘金，对于坐厘来说，包括一些银号钱庄，他都要按营业额抽厘金，江苏巡抚，南河总督收到咸丰的谕旨之后，也开始设卡抽厘，到1854年底，钦差大臣胜保发现了厘金的特异功能，他给皇帝上奏折，他说可否请旨，让各路统兵大臣，会同本省邻省督抚，仿照雷以誠办理厘金局，官为督劝，商为经理，不经胥吏之手，自无侵漏之虞，用兵省份就近随收随解，他省亦暂存藩库，为协拨各路军饷之需。

通过这份奏折，我们能看到，胜保很明显是看到了厘金的好处，他跟咸丰大力推荐，让所有统兵大臣都效仿，在胜保的奏折里面，有三点需要注意的，第一点是，他说收厘金让统兵大臣为主，各省督抚只是会同，说明他认为收厘金的主要用途，是为了供应军饷，这时候胜保正在前线统兵打仗，军饷不足是他最大的痛点，他想自己收厘金用来充当军饷，这点很能理解，第二点是，他想收厘金这个事是官督商办，不经过
