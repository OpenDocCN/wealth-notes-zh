

# 生命中不可强求的5件事

本文是2025年1月19日公众号（U君笔记）的付费文章。

从这一生中，有些事是可以努力的，而有些事是不可以强求的。

如果你非要强求，勉为其难地逼自己去做，那么以后你每次回想起这件事都会恶心一下。

总之呢，如果你不尊重自己的感受，那么老天一定会惩罚你。

我总结，人生中有5件事是不能强求的，否则会活得非常不爽，而且你的所有能量都会被压制，长期郁郁不得志。

## 1. 不要强迫自己向世俗低头

今天在知识星球有个小伙伴问了我这样一个问题：老家的人不允许她远嫁，因为还想让她给自己养老。

其实吧，父母不想让女儿远嫁，这个是非常好理解的，尤其是那种亲子关系比较好的家庭。

但是，如果父母给出的理由是“远嫁就没人给自己养老了”，那这个问题就很大了，因为这已经有强迫和绑架的味道了。

而且感受这个东西是不会说谎的，父母到底是关心女儿，还是害怕女儿将来不管自己，这完全是两种不同的感受。

还有10天就春节了，最近还有咨询我关于“过年不想回家”这个话题的。

其实上面这两类属于同一类话题，那就是要不要遵守传统的、世俗的约定。

比如，“过年就是要回老家”这件事，就是传统的约定俗成，你不回去很可能会被别人说三道四，父母也可能对你有意见。

但是，我的建议是：如果你不想回去，就不回去，不用找任何理由和借口。

因为你已经是个成年人了，如果不能按照自己的意愿行事，那你不配为人。

而且，如果你的经济已经独立了，不再依赖父母了，那么你就更不应该迫于他们的权威而做自己不想做的事。

如果他们不懂得尊重你的意愿，那你该闹翻就闹翻，不用惯着他们。

退一步讲，长辈如果想让孩子长大了跟自己很亲，那一定是靠爱和陪伴来滋养的，而不是靠道德绑架。

我觉得对于50％的年轻人来说，成长的第一步就是“推翻父母的权威”，否则你永远都立不起来，当个人都费劲。

什么样的人不需要跟父母闹翻呢？

比如，你的家庭不一定多富有，但是从小到大你都是在爱的滋养中长大，自己也发自内心地想陪伴父母，而不是迫于世俗的压力，或者父母的权威。

总之呢，如果一个成年人还总是迫于父母的压力而去做什么事情，那我会认为这个人还没长大，就是个巨婴。

甚至连做人的资格都没有。

## 2. 不要强迫自己跟不喜欢的人婚配

无论你选择结不结婚，生不生孩子，生几个孩子，这都是你自己的人生，即使是最亲的人，也不能左右你的选择。

U君自己就是从底层出来的，由于最近十来年，男女比例严重失调，农村的男光棍越来越多了，但是他们开车和繁衍后代的意愿超级强烈。

那怎么办？只能去购买巨丑无比、身体残疾、二婚带娃或者有精神疾病的车子。

就是说，很多底层男人买回来的车辆，都是无法唤起自己驾驶欲望的。

很多人都是在车子刚买回来那一阵子，会强迫自己开车，等到对方怀有小车之后，自己就再也不会开车了。

甚至后半辈子都对那事没有什么兴趣了。

我早就说过，如果一个男人在“开车”这方面失去了兴趣，甚至每次上车之前还要做心理建设，那么这个人已经不算人了，应该开除人籍。

女人也是同理，不要因为对方很老实（不值钱），或者你好（不值钱），或者很帅（不值钱），你就跟他草草结婚了。

后续的婚姻能否长久，主要看他的谋生能力、为社会创造价值的能力。

只要他为社会创造的价值多、赚的钱多，那么他就会长期吸引你。

总之呢，婚配这件事，不需要听从他人的安排和干预。

有人会说，父母也是为自己好。

我想说，如果你父母都是成功人士，而你又是个没有脊梁、没有思考能力的软体动物，那你可以听从他们的建议。

但是我还是建议大家先修炼自己，等到自己心智成熟了、财务牢固了、眼光毒辣了，再考虑进入婚姻。

## 3. 不要强迫告诉自己，什么东西对自己很重要

对于U君自己来说，只有我自己的生命健康是最重要的事情，其余的什么亲情、友情、爱情，以及什么道德规范，对我来说都不是很重要。

所以，我基本上不会被那些乱七八糟的事情所牵绊。

但是我见过有些蠢货，把父母看得特别重要，父母一不高兴，自己就吓得像小鸡一样。

还有的人，把婚姻和家庭看得很重要，自己的配偶整天殴打自己，她还是不想离婚，因为她不想让孩子没有完整的家，也不想被别人说三道四。

这些都是自作孽，只能常年在地狱里煎熬，没人能拯救你。

因为你的能量将长期被有毒的人吸食，你的灵魂会变得萎靡不振，你离被开除人籍也不远了。

实际上？

你只要敢硬气一回，跟父母闹翻一回，深深地刺痛他们一次，他们就再也不敢在你面前造次了。

然后你很快就会立起来了，原本属于你的宇宙能量也会回归到你身上，从此以后你的运势也会一顺百顺。

我就特别佩服那些把父母气出心脏病的 人，我也非常佩服那些打骂父母的人，我觉得这些人都是伟人，都是圣人，他们应该著书立说。

因为他们敢于把父母当年施加到自己身上的负能量反击回去，他们非常清楚，如果不让父母尝尝毒箭反噬的滋味，那么自己一辈子做不了人。

在家里你做不了人，到社会上也没人拿你当人看。

有毒的婚姻也是如此，如果你意识到婚姻是你所有痛苦的来源，那你就应该早点离婚，不要拿孩子当借口。

你自己已经做不了人了，孩子也不会把你当人看的。

## 4. 不要强求大富大贵、长命百岁。

俗话说，小富靠勤，中富靠智，大富靠命。

想要赚大钱，成为亿万富豪，不是靠努力和高智商就能获得的，祖先庇佑+优质基因+天时、地利、人和，一样都不能少。

如果你跟U君一样，也是出身底层，那我们的目标是，让自己过上小富生活就算功德无量了（不要想着拯救父母）。

如果还有余力，那就往“中富”上努努力，组建一个团队，打开自己的眼界和格局。

另外，“寿命”这个东西跟基因的关系最大，不必过分强求。

比如，市面上大部分保健品和忠药都会对肝肾造成损伤，所以，吃的营养健康一点就行了。

## 5. 永远不要强迫你身边的任何人觉醒。

即使你的境界已经很高，你的认知已经很高，也不要强迫任何人觉醒，包括亲人。

因为每个人都有自己的人生课题需要处理，只能靠他自己去经历、去试错、去受伤、去领悟，该经历的九九八十一难，一难都不能少。

天雨虽大，不润无根之草；佛法虽宽，不渡无缘之人。

如果一个人你无论如何都拉不住，怎么说都不听劝，那就说明，眼前的这条弯路是他必须要走的。

有些劫难，也是他必须要历经的。

不要以为你能改变谁，你连改变自己都很难，更不用说改变别人了。

其余的任何人你都改变不了。

能让一个人清醒的，是教训，而不是教育。人只有自己撞了南墙，才知道头破血流的痛，才能长记性。

比如，你的兄弟姐妹遇到买车买房、结婚离婚这类人生大事，即使你作为一个旁观者看得一清二楚，你也不要随便介入他们的人生。
