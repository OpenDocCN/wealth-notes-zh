
# 大佳10.12：股票市场（判断部分）

2025-10-12,股票市场。(判断部分)
1,单从上周五的回调来说，其实并不是很大的问题。上周五，只是一个结构化的再平衡。大盘成长风格向大盘价格风格的过渡。这个过渡在周四，创业板冲高回落，芯片股尾盘跳水，就已经初现端倪。而且，个股的波动也并不是很大。中位数甚至上涨。
2,最大的变量是周五晚上，川普针对我们稀土长臂管辖的突然发难。最终，美股科技股，中概，和A50 集体跳水。以目前状况来说，预计明天上证低开2%左右，恒生4%左右，恒科5%左右。大概率国家队会干预，但干预应该集中在大盘股。尤其是银行和央企。抄底的资金应该也会很多。且盘中一定会有冲高。
3,对于A 股来说，明天我们大概率会持股不动，或者只进行非常小的调仓。主要是因为A股只有轮动机会，而没有系统性的择时机会。我主要交易目标是在港股。最好的情况是，港股的外资对冲基金一键平仓，从而让港股出现系统性的择时机会。
4,对于科技股和互联网公司，并没有抄底计划。我们港股的抄底的方向是基本面稳定的消费股，以及一部分明显具有性价比的公共事业。
5,目前，对于抄底的讨论和议论非常多。这和今年4月7日完全不同。4月7日我们发文说我们不减仓，遭到的嘲讽更多，情绪更悲观。所以4月7日，4月8日等反而有利于抄底。明天不会给太好的位置。此外，国家队的态度需要认真观察。
6,总的来说，在A 股高杠杆且风格转化的关键时期，突然遭遇了这个对基本面影响不大，但对情绪面影响很大的利空，那么,就要关注情绪面对于杠杆和风格的影响。认为对基本面影响不大就不会产生太大的影响，这种观点往往会很危险。因为市场本身也不是因为基本面而上涨。
7,从中长期角度上来说，一次性跌到位，反而好一些。但明天预计很难，像是4月7日一样，一天就创造出了全年最低点。总的来说，即使有子弹可以抄底的情况下，也需要徐徐图之。
