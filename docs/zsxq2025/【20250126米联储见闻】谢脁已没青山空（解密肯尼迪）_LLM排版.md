
# 谢脁已没青山空：米联储见闻

## 肯尼迪刺杀事件解密

昨天，川普要爆出一个事，他将解密与前总统约翰肯尼迪和弟弟前参议员罗伯特肯尼迪以及民权领袖马丁路德金遇刺事件有关的文件。暗杀一直是美国政治的重要组成部分，美国成才是真正践行解决不了问题就干掉解决问题的人，而且不是比喻意义上的干掉，让人不说话，而是真正物理意义上的干掉，让这个人彻底从地球消失，上至总统政客，民权领袖，下至各路吹哨人，爆料人，被枪杀被撞死被心脏病暴毙，一直不是什么新鲜事。

1999年，美国鬼才导演库布里克拍下电影《大开眼界》，结果当年就因心脏病暴毙而亡，在此之前，他从未有过心脏病的记录，而20年前的这部电影当时看上去令人匪夷所思，直到萝莉岛事件浮出水面，大家才意识到从来都没有什么所谓阴谋论，只有还未被揭露解密的事实真相。只有真正天真的人才相信政治和权力是干净清白的，川普也长期因为阴谋论被民主党嘲笑是低智脑残，他为什么要揭秘肯尼迪被刺的文件，就是要通过这个举动告诉公众，他以前所说的所谓阴谋论都不是臆断的，美国的政治就是这么脏，这么没有底线就是这么流氓。

川普在白宫签署行政命令时说：很多人等待这一刻已有几十年之久，而相关案件的一切细节将会真相大白。该行政命令要求，国家情报总监和司法部长等官员要在15天内，向总统提交完全解密约翰肯尼迪遇刺文件的计划；以及在45天内审查并提交完全解密罗伯特肯尼迪遇刺及马丁路德金遇刺文件的计划。

肯尼迪总统于1963年在德州达拉斯被暗杀，他的弟弟罗伯特·肯尼迪在1968年于加州总统竞选民主党初选期间被暗杀，这距离美国最著名的民权运动领袖马丁·路德·金在田纳西州孟菲斯被谋杀仅隔两个月。与调查相关的许多文件在这些年中已经被公开，但有数千份文件进行了删减，特别是与庞大的肯尼迪调查有关的文件。根据已公布的文件，肯尼迪总统是被李·哈维·奥斯瓦尔德射杀的，奥斯瓦尔德是一名海军陆战队退伍军人，曾叛逃到苏联，后来又返回美国。当时美国政府的 一个委员会认定奥斯瓦尔德是单独行动。但这个结论一直遭受多方质疑，存在多个版本的阴谋论。数十年的民意调查显示，大多数美国人不相信奥斯瓦尔德是唯一的刺客。

1992年，美国国会通过了一项法律，要求在25年内公开所有与该调查相关的文件。在川普的第一任和拜登任内，均释放了大量与肯尼迪刺杀案相关的文件，但仍有数千份部分或完全保密。今天的美国要开始扩张，就要拉下大家的羞耻感和道德感，要从根子上认识到美国人到底是什么人，要知道选举政治从来都不是什么干干净净的民主自由，才会给川普增加新任期提供舆论和民意基础，你想要做做婊子，首先要让民众意识到自己本身就身处妓院，这样阻力就会小很多，主动扯下自己的遮羞布，就可以做很多羞羞的事情了。

根据官方说法，当时仅有一颗子弹击中了前总统以及时任德州州长长约翰·康纳利。康纳利当时也在车队中乘坐，并在枪击中幸存。新信息进一步质疑了奥斯瓦尔德单独行动的理论。全面公布所有编辑过的文件可能会显著增加公众的知识。但可能不会有“致命证据”，中情局和其他安全官员将推动保持某种程度的保密。

罗伯特肯尼迪是在洛杉矶的一个舞厅被一名巴勒斯坦人西尔哈刺杀。该 名男子称因美国对以色列的支持而感到愤怒。小罗伯特·肯尼迪曾在监狱中与西尔哈交谈，并表示不相信西尔哈杀了他的父亲，小肯尼迪在这次大选中临阵倒戈，以民主党之身支持共和党领导人，不排除是为了追寻父亲被杀的真相做出的背刺。

而马丁·路德·金则是被白人种族主义者詹姆斯·厄尔·雷射杀。金博士家族成员则声称，雷并非单独行动，而是更大阴谋的一部分。

在川普的第一任期，就曾经提及这件事，当时民主党报纸曾经根据相关信息做出报道，对肯尼迪被刺事件作出了各种阴谋论式的预测，给民众舆论做铺垫，不管是冷战阴谋，黑道出手还是联邦政变似乎都不能被完全证明，当时恰逢美苏冷战早期，时间节点倒是和最近有点像，而几年前日本首相安倍在街头被人当街枪杀，效果似乎与肯尼迪被杀异曲同工。与肯尼迪遇刺一样，安倍被枪杀的现场也是在数百人面前被当场射杀，然后出现大量新闻报道，刺杀肯尼迪的新闻出现在报纸头版，称“总统车队经过达拉斯市中心时他被步枪子弹击中，死于脑伤”。肯尼迪被刺后不到几小时，林登·约翰逊就在“空军一号”上，在肯尼迪的遗体旁宣誓就任总统。并带着总统头衔和肯尼迪的尸体一起飞往华盛顿。在新闻媒体的描述中：约翰逊宣誓就职时，约翰尼迪夫人就站在新总统的旁边，袜子上还溅有她丈夫的血。

枪击事件发生后，马上就有一名目击者对记者表示，枪响的时候，他看到得克萨斯州教材仓库大楼“五楼或六楼”的一扇窗户里伸出来一把步枪，然后又缩了回去。据刺杀者奥斯瓦尔德的妻子说，刺客曾是海军陆战队员，1962年从苏联回到美国时像变了个人似的，表现出的性格令人不解。就在刺杀事件发生两天后，刺杀肯尼迪的凶手奥斯瓦尔德在警方羁押下死于另一个凶手杰克·鲁比的枪口。至此肯尼迪被刺案就变成了死无对证的悬案。此后，当局成立了一个机构专门调查此事，调查了一年后，结果遭到各方质疑。最终，调查机构否定了奥斯瓦尔德得到了前苏联或古巴官方支持的观点，判定他是独自行动，但要求和特勤局进行改革。对比之下，刺杀安倍的那个FBI刺客最后也被证明是头孤狼，也没什么具体的刺杀原因，所谓邪教的说法，完全站不住脚。

和60年代，美国至少又成立了两个正式的联邦政府小组对这起枪击事件重新提起诉讼。1969年，司法部长拉姆西·克拉克任命四名医疗专家重新检视科学证据，克拉克的调查小组支持沃伦委员会关于杀害总统只有两颗子弹的论断。1979年，众议院遇刺案特别委员会发布了一份报告，援引新发现的证据和科技进步称，可能有未被提到的同谋参与这起谋杀案。在其存在的两年半时间里，该委员会内部冲突不断。据该委员会称，奥斯瓦尔德从得克萨斯州教材仓库大楼六楼的窗户开了三枪，一个身份不明确的人从总统专车前方的草丘处开了一枪。该委员会在其调查结果中引用了多名见过另一名枪手的目击者的话。但长期以来，调查结果都不足以平息人们对此事件的阴谋论想象，好莱坞在1991年还专门拍了一部电影《刺杀肯尼迪》。

但这部电影仅仅是聚焦这场历时34天、被很多人称作“马戏”的审判，和新奥尔良地方检察官加里森主导的那场大规模调查，并未针对刺杀事件作出结论。电视暗示存在一种阴谋论：彼时美国正处于越南战争时期，刺杀事件是中情局内部的反共和反卡斯特罗极端分子暗中策划，以阻挠缓和美国同苏联和古巴的紧张关系，并阻止从越南撤军。因为当时的肯尼迪希望停止冷战和越战。这部影片在一定程度上促成了《约翰·肯尼迪总统暗杀记录收集法案》在影片上映后的第二年获得通过。该法案要求在25年内，也就是2017年10月26日之前公开与该刺杀事件有关的所有政府档案。自那时以来，有数千份记录解密，其中很多都是揭秘美国的外交政策，一些档案详细记述了肯尼迪总统退出越战的计划，其他一些表明刺杀事件后，卡斯特罗非常担心美国会报复古巴。大量文件曝光了五角大楼提出的对付古巴的办法。

在这次川普当选的过程中，肯尼迪家族的突然倒戈绝对是 有特殊意义的事件，川普当选后，小罗伯特肯尼迪成了川普的卫生部长，与之相应的是，川普在当选第一天签署的行政令就包括退出世卫组织，今天更是传出消息，川普签署行政令暂停了美国疾控中心等一系列公共卫生机构的日常工作，就在上个月，当川普确定了小肯尼迪的卫生部长提名后，纽约时报专门写了一篇文章来揭露小肯尼迪的一些往事，包括他曾经携带毒品。时间是1983年9月，小肯尼迪在飞往南达科他州拉皮德城的航班上突发急症，当时他29岁，刚从法学院毕业，调查人员后来发现了他行李中有海洛因。肯尼迪后来承认犯下持有海洛因的重罪，肯尼迪承认自己在十多岁时因为父亲遇刺身亡而备受打击染上毒瘾。携毒被发现后，小肯尼迪就进了新泽西州的一家戒毒中心。在这次竞选中，这个故事被他描绘成一个关于救赎的故事。扭腰的文章除了提及小肯尼迪携毒的历史，还直言他是个花花公子，他的不忠导致了第二段婚姻破裂，并提及一位在此期间为他家工作的保姆指控肯尼迪对她有性暗示，并在未经同意的情况下触碰她。这些描述想要大 的目的很明确，小肯尼迪就是一个吃喝嫖赌五毒俱全的纨绔子弟，但如今他成了卫生部长，真是国将不国了。

小肯尼迪2023年4月以“肯尼迪派民主党人”的身份开始竞选总统，但后来他离开民主党，以独立候选人的身份参选。再后来，小肯尼迪推选，并公开支持共和党的川普，小肯尼迪显然已经公开背叛了民主党，美国媒体认为，小肯尼迪与一个肯尼迪家族和民主党同僚都恨之入骨的人结成联盟。但如果关于肯尼迪被刺事件最后被证明是民主党内部的清理门户，那么对于民主党而言将会是致命的打击，因为肯尼迪是颇受公众爱戴的总统，相比于攻击对手党，处于利益对党 内人士下手就是无底线的操作，一旦这个阴谋论被证实，民主党或许将彻底失去继续参与党争的机会，这对于川普谋划延长自己的任期甚至彻底改变美国总统选举的游戏规则都可能是必要的舆论铺垫。

在进入政坛之前，肯尼迪是美国知名的环境律师。1999年，他因与河流守护者组织的合作被《时代》杂志评为地球英雄。小肯尼迪还有一个主攻点是直接关于美国最大的利益群体，医药企业，他是疫苗的最大反对者，这常常被民主党媒体当成反智的标志，但事实上，小肯尼迪的这个结论源于他在做环境律师过程中接触过的一些案例和病例，2000年，肯尼迪一位心理学家朋友布里奇斯的儿子在接种了一种百日咳疫苗后遭受脑损伤，并被诊断为自闭症，这种疫苗目前已经不再使用。同时，肯尼迪一直致力于将汞从水道中清除，布里奇斯希望他调查硫柳汞，这是一种以汞为基础的防腐剂，这种防腐剂曾经用于美国的儿童疫苗，小肯尼迪的认为，美国国立卫生研究院只不过 是制药业的“孵化器”，是一种加强版的监管圈套。2005年，肯尼迪写了一篇文章，发表在《滚石》和《沙龙》杂志上，指责硫柳汞是儿童自闭症发病率上升的原因。这两家新闻媒体后来发现文章的一些说法是错误或可疑的，撤回了这篇文章，肯尼迪也受到了科学界的广泛批评。疫苗导致自闭症的理论被广泛推翻。但小肯尼迪认为，这就是美国制药业控制政府和主流媒体的证据。之后小肯尼迪出版大量书籍，创立非营利组织“儿童健康保护”，并推行反疫苗联盟。迪带来了他长期以来一直寻求的追随者。传达的信息核心是，制药企业与联邦政府联手从危机中获利，并对新冠疫苗持强烈怀疑态度。

如果关于肯尼迪刺杀事件能够重创民主党的信誉，那么与之紧密联系的很多相关利益企业，可能都会彻底失去民众信任。

而就在川普全面挑战美国公共卫生体系的同时，中国也掀起了集采药舆情事件，医疗卫生是全球最大的利益链条之一，在这个巨大的利益链条上，任何阴谋论可能都只是小菜一碟，因为在这条链条上，唯一没有悬念的就是命是靠钱买来的，能向谁花钱买命，就是最大的权力之一。
