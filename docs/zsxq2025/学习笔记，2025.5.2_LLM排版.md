
# 学习笔记，2025.5.2

1. 真正的离开，是默不作声的。还在指责、还在抱怨，说明还对对方抱有一丝希望。
2. 一个人如果有自己热爱的东西，就没功夫去关注别人身上的事，没工夫去关注那么多的新闻热搜，会把大量时间精力投入到自己热爱的东西上。
3. 家族积累可以传承，但是认知和逻辑无法传承，想要提升，必须去主动学习，主动训练。二代三代没有经历过一代的打拼过程，对很多东西的风险和认知就会不够，基础不稳，站得高也容易摔得惨。
4. 一个人如何对别人，不一定能看出来这个人的人品怎么样，因为短期接触，可以暂时伪装。一个人如何对自己，很大程度上决定了他的性格、人品，人没法一直伪装，尤其是在没人的时候还继续伪装。所以会说，慎独，一个人的时候，最能暴露自己的本性。和一个人相处时间长了以后，他会不自觉的用自我要求的方式去对待别人。
5. 思维方式，一部分取决于词汇量，你没法用自己不了解的概念去思考一个东西，脑子里积累的词汇量越多，能够跨学科思考的能力越强，人对事物的了解程度就越清晰、具体。词穷，是很多网民都存在的一个问题。很多事，好像自己能想明白，但是说不出来，看到别人说了一套，觉得表达出了自己想表达的东西。根本原因是，自己脑子里的概念太少，串不起来，需要刻意练习。想要练习，可以从每天写点什么开始，写作即思考。
6. 人的醒悟，不是靠读什么书、见什么人、上什么课得来的，任何轻易得到的道理，都只是一段信息，离内化成自己的一部分还有很远的路要走。真正让人醒悟的，是经历，是亲眼所见亲手触摸，是在事上练出来的。
7. 在人群中，很容易丧失独立思考的能力。一句谎言，一个人对你说，你知道是谎言，一万个、十万个对你说，你就会开始怀疑自己。事实没有变化，但是环境的影响，会让人的思维方式产生变化。
8. 在互联网上晒自己的生活，是挺危险的一件事，上到大网红，下到小路人，图片发到网站后，最终会发展成什么样子，解释权就不在自己手上了。被拿去编新闻、造黄谣、当杀猪盘个人资料、当训练资料……各种玩法都有。还有一点，以前靠照片辨别拍摄位置，需要有丰富的人文地理知识，现在，丢给 ChatGPT，它就能猜出来个大概，一个人被人肉的门槛更低了。
9. 很多人想改命，改命是一件逆天的事，你不能通过去做大多数人都在做的顺从天意、顺从人性的事来达到。改命第一步，先革自己的命，革自己观念的命，逆自己的人性，认清自己所处位置，才能知道下一步往哪走。
10. 爱好分两种，消费型爱好和创造型爱好。爱看电影、爱旅游，这是消费型爱好，在看电影、旅游的过程中不需要思考，不需要有什么输出，为的是开心，消费完就完了。爱写影评、爱写游记，这是创造型爱好，在玩的过程中要保持思考，有输出，可以在一定程度上提升自己的见识和能力。人要尽量培养一些创造型的爱好，输出、创造些什么，创造过程就是思考的过程。
