
# 卢克文7.15：李某被开除，是否合理

## 李欣某被大连工业大学开除，是否合理？
## 文/卢克文工作室嘉宾流离

7月13日，“女生出轨Zeus”事件时隔半年再登热搜，甚至以五条热搜霸榜，可见一时间舆论震动。起因在于7月8日，涉事女生李欣某就读的大连工业大学学生工作部(处)网站发布公告，称拟依照《普通高等学校学生管理规定》第三十条第六款和《大连工业大学学生违纪处分规定》第十九条第六款，对其作出开除学籍处分，并直接公开该生全名。按前者规定，对其退学属于“学校规定的不能完成学业、应予退学的其他情形”，而后者则是“拟开除”的直接依据——与外国人不正当交往，有损国格、校誉的，视情节给予记过及以上处分。

李欣某在有男友情况下，出轨乌克兰电竞选手并被其曝光不雅影像，很多网友对开除表示支持。

但也有人质疑，“有损国格、校誉”是否夸大了？开除学籍的处罚，是否太重了？

舆论争议主要集中在这几个问题上：
1. 李欣某被曝光隐私，是否同样是事件受害者？有鉴于此，不给机会直接开除学籍，惩罚是否过重？校方公告公开李欣某全名，又是否侵犯了隐私权？

先来看第一个问题，之前很多声音称李欣某与Zeus的不雅影像是Zeus个人偷拍，李欣某不知情，属于隐私权被侵犯。然而网上流传片段中，李欣某不但知道自己在被拍摄，还和Zeus一起对着屏幕打招呼，这显然与“偷拍”对不上号。

当然，Zeus将视频照片传到粉丝群，导致二人不雅影像在网上疯传，播放量达12亿次，并称她为easy girl，使该话题迅速登上外网12国热搜，确实做得也不地道。

更让网友愤愤不平的是，作为2011年女儿就出生的有妇之夫，敢做不敢当就算了，日前发文还将自己摘了个干净，称自己没有不尊重兔子和兔子女生的言论，自己没结婚恋爱，也不知道对方有男朋友，呼吁“停止仇恨，我们都会犯错”。

但互联网是有记忆的，当初的“easy girl”言论可不是别人栽赃的，有女儿却不敢提及，只敢打未婚的马虎眼，甚至连一句道歉都没有，Zeus实在毫无诚意。

而李欣某的作为也令人侧目，她不仅背着男友和外国人私会，飞往上海看比赛的机票钱还是男友出的。男友也没想到，自己没跟着去。李欣某竟然整出了这么个大活，看比赛看人家选手床上去了。

被曝光不雅影像，隐私权受侵害，当然是受害者。不过受害和做错事，本是两回事，不能因为她做错事就忽略受害者身份，也不能因为受害了就无视其错误。如果李欣某和Zeus的事，没被拍发出来，可能不会闹大，但拍了流传到网上，上了称千斤重，就难怪校方选择严格处理了。

所以开除学籍处罚重不重呢？从大连工业大学的校规出发，恋爱中出轨有孩子的外国人，确实属于“不正当交往”范畴，而Zeus所说的以及形容她们“像巴西牛排一样廉价”的言论，无疑是损害了兔子女性在世界上的整体形象，“巴西牛排”也成了风行半年多，用于嘲讽兔子媚外女性的流行词。

李欣某的媚外行为并非个例，荷兰人来华搭讪7000多女生、美国健身教练交往1242名兔子女友、外国人与上百女生当街接吻……Zeus不是第一个说“Chinese easy girl”的人，但李欣某的投怀送抱，却比真实地为本就被媚外女破坏多次的兔子女性形象，又增添了“巴西牛排”这刺眼的一笔。

你能说李欣某作为兔子人跑去老外床上，只是个人道德问题，不损害国格吗？

李欣某在校生的身份曝光后，学校的声誉自然也大受影响，相比当年该校学生翟凌爆出的“兽兽门”，此番涉及出轨、媚外，杀伤力更大。与外国人不正当交往、损害国格校誉，被李欣某犯了个遍，加上造成的社会影响非常恶劣，校方依据校规采取顶格处理，是有合理性的。

但也有部分法律界人士提出异议，他们提出，《普通高等学校学生管理规定》第三十条第六款，是关于退学的条款，与开除学籍的处分不能混为一谈，公告援引条款存在错误。且《普通高等学校学生管理规定》第五十二条里学校可予以开除学籍的八种情形中，李欣某未受治安管理处罚(第三款)也未影响学校教育教学秩序、生活秩序以及公共场所管理秩序(第六款)，更够不上第七款的“侵害其他个人、组织合法权益，造成严重后果”，与其余条款规定也无关联。因此，这些法律界人士认为学校的决定不符合相关规定。无权开除。

但事实果真如此吗？首先，公告的意思是“拟开除学籍”，并非最终决定，而且李欣某可在9月7日之前以书面或口头形式进行陈述或申辩，存在更改决定的可能，现在指责一个未落地的处分决定，为时过早，如果开除学籍确有不妥，还是可能更改的。

而且，公告也表明了学校早在4月就以直接送达、邮寄送达、留置送达等方式向李欣某送达了该校《学生违纪拟处分告知书》，公告只是在未得到及时回应的情况下，“为充分维护各方权利”的再次送达。李欣某仍有两个月时间陈述申辩，甚至可以在处分下达后，申请行政复议，提起行政诉讼，依法合规保障自身权利。

最后，《普通高等学校学生管理规定》第五十一条亦有规定：对有违反法律法规、本规定以及学校纪律行为的学生，学校应当给予批评教育，并可视情节轻重，给予如下纪律处分——其中就包括了开除学籍，因此除非李欣某申请复议或起诉成功，学校仍有权给予开除学籍处分。

再谈谈舆论争议的第三个问题，说学校公告两次提及李欣某全名，处分仍在拟定阶段，就暴露了当事人全名，显然有不当之处，涉嫌侵犯隐私权。李欣某可能也未料到，她不体面，学校这下让她更不体面了。

另外，在学校公布全名过后，部分媒体将李欣某的大名贴得满世界都是，搞得微博热搜一度出现其全名，学校有责，这些媒体也难辞其咎。

学校此举，确实侵犯了李欣某隐私权，不能全盘认为学校做法是对的。但像前面评价李欣某行为一样，也应该分开看待——侵犯隐私是错，但这和开除学籍是否正确，不是一回事，不能混为一谈。

所以网上一些将两个问题混淆界限，试图搅混水的言论，无论是站李欣某还是站学校，都可以说是别有用心，不能上当。

当然，仅仅停留在道德和法律层面争论，可能难以理解为何这件事半年之后再次发酵，又为何偏偏在这个时候登上热搜，我们不妨从李欣某和校方的利益立场出发，分别来看待双方的行为动机。

对高校来说，学校声誉极大地影响了招生成果，从而直接影响办学的各方面工作。如今正是高校陆续开始录取新生的阶段，大连工业大学出于招生考虑，在4月向李欣某送达拟处分告知书，7月以公告形式公开送达，试图赶在招生前尽力消除这场风波带来的负面舆论，是在情理之中的。

这倒不是说学校冷漠无情，优先考虑校誉而不保护学生，李欣某已在学校引发众怒，许多学生认为其行为影响的不仅是学校的声誉，更是全体学生的声誉，因此支持严惩李欣某，如果学校将此事带过或轻罚李欣某，反而会招致更大的反对。

比起保护一个犯错的学生，当然还是保护全体未犯错学生要紧，否则不成了按闹分配？

反过来，学校拟开除李欣某的决定，在网上同样博得了许多同情和支持的声音，大有一举扭转负面舆论之势，对反感媚外的广大家长学生来说，大连工业大学一下子成为了先锋典范，对招生自然也是有利的。

而对李欣某来说，当务之急不仅是要尽力挽回隐私权的受损，想办法减轻处罚，对她的未来还是非常关键的——退学和开除学籍大有不同，前者不属于处分，不记入档案，甚至可以重新考取其他学校学习，而后者是最严厉的处分，学籍被注销，记录被上报上级部门备案，将影响以后的学习、工作、考公等等方面。

当初李欣某对处分告知书迟迟没有回应，恐怕就有拖延的想法在里面，只是校方选择了公布出来。现在留给她的选择只有走程序陈述申辩，同时借助法律力量，尽力抓住校方程序上的纰漏，如果陈述申辩失败，就提起行政诉讼，争取将开除学籍的处分降低，或改为自己申请退学，尽可能降低损失。

网上现在质疑学校的法律界人士中，可能就有被李欣某求助的。对她来说，自然是想尽力挽回局面减少损失，而学校也想尽早彻底摆脱其影响，如若更改处分，减轻处罚或改为退学，反而可能令支持舆论失望，遭到反噬则得不偿失。

由于双方都有不愿退让的利益因素，再加上巨大的舆论声浪，因此最终对簿公堂的可能性，也将随着矛盾愈演愈烈，舆论越闹越火而变得越来越大。

另外，由于兔子和乌克兰存在民事司法协助条约，李欣某以侵犯隐私权为由，向Zeus起诉要求民事赔偿也是一条可选的务实法律途径。

法律界人士应当比我更懂这些门道，但号称维护李欣某合法权益者多，却鲜有人提出，如何更务实地帮她惩罚目前仍未付出代价的Zeus，挽回隐私权被侵害的损害，是他们想不到吗？

那显然不可能，不管是大也好，各路法律界人士也罢，站边李欣某，在乎的主要还是借此事发声博得的流量、粉丝以及背后的利益，怎么真正帮她维权，还真没几个人在意。

反过来讲，站边学校的大V、媒体，也没几个和大连工业大学有关系的，也未必很在乎大连工业大学如何如何，说白了，大部分声音都是为了站队，为了流量。

此事能一下子在微博搞出五个热搜，本就不见得是因为大家特别关心哪一方，而是这件事刚好涉及民族情感、性别对立的流量大热门，再加上一些社会上本就存在很大争议的隐私权维护、私德界限和高校处罚标准等话题。能炒起来，的确符合当今互联网流量规律。

因此关于此事的各处动态和评论区充满了各种立场先行的言论，也就不足为奇了。跟大部分人没直接利害关系的热点，基本都是大V媒体率先领队，各路粉丝纷纷站队，真正的对错是非，谁管呢？大V媒体有流量吃，粉丝们对线个爽，大家都有光明的未来。

于是我们看到，支持学校吧，会被一群人攻击为封建卫道士，漠视女性；支持李欣某吧，又会被另一群人斥为“巴西牛排”，崇洋媚外。

如今立场先行的舆论环境下，虽然说啥都有人骂，但也等于说啥都有人夸，也难怪互联网热衷于站立场了。

只是，在一次次的流量狂欢和对立大战中，网上的情绪和矛盾，却是越来越膨胀了。想让这种狂热慢下来，总需要一些相对客观的文字，在对立大潮中勇于发声，让更多人冷静冷静。
