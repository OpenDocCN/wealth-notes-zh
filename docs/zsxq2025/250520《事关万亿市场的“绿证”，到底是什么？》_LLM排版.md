
# 事关万亿市场的“绿证”，到底是什么？

## 绿证的定义

绿证全称是可再生能源绿色电力证书，可以把它当成量化之后的绿电身份证。例如，一个光伏电站每年能发很多清洁电力（绿电），以前只是默默发电，现在国家会发证书证明其绿电生产。每产生1000度绿电，就发一张绿证，以此类推。

绿证有两个关键功能：证明绿电产生和可以买卖。假设一个光伏电站产生10万度电，获得100张绿证，理论上可以赚两笔钱：电费和卖出绿证的钱。

## 绿证的购买者

购买绿证的主要是那些有减碳指标或需要证明公司/产品环保属性的企业。例如，一个想证明支持环保的公司可以购买绿证，这等于支持了绿色能源发展。

## 绿证的补偿机制

对于绿色发电，国家有两种补偿路径：绿证和机制电量（保障收入计划）。机制电量是新能源发电厂和电网之间的兜底协议，但2024年1月发布的《关于深化新能源上网电价市场化改革 促进新能源高质量发展的通知》提出，机制电量与绿证收益互斥规则，即电厂只能二选一。

## 绿证的价格走势

绿证并非今年才出现，2017年国内已建立绿证制度。过去3年，绿证交易价格持续走低。2022年交易量145万张，均价28.10元每张；2023年交易量2364万张，均价19.22元每张；2024年上半年交易量5700万张，均价9.6元每张。

## 激发绿证消费需求

为了提高绿证价格，需要激发需求。今年3月发布的《关于促进可再生能源绿色电力证书市场高质量发展的意见》提出，要激发绿证消费需求，采用“强制+自愿”模式。强制购买绿证的企业包括钢铁、有色、建材、石化等行业的部分企业，数据中心等。自愿购买绿证的企业包括有环保要求的外贸公司、供应链企业等。

## 绿证的政策措施

《意见》从五个方面提出政策措施：绿证供给、绿证消费、交易机制、应用场景、走出去。这些政策旨在推动绿色能源市场化，提高绿证价格，从而激励新能源电厂和碳排放大户进行技术升级。

## 绿证带来的机会

绿证的普及会激励新能源电厂做技术升级，倒逼碳排放大户做技术改良。此外，还会出现很多新的岗位需求，如新能源运维、碳资产管理、绿证交易员等。碳足迹核算相关证书的线上考证周期约一个月，可以为企业提供ESG报告中的绿电消费数据核验服务，收费每家企业3000到8000元不等，且可兼职。

## 绿证与社会叙事

绿证的普及可能会推动社会叙事，让人们更重视减碳。例如，雷军说小米汽车的工厂使用了光伏板，每年减少碳排放9905吨；伊利推出过零碳牛奶；还有碳中和火锅底料等。这或许说明消费者更愿意支持低碳，而企业也可以通过减碳获得环保溢价。
