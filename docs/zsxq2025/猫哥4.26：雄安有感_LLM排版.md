
# 猫哥4.26：雄安有感

这几天有幸去雄安考察，很是有点感慨，今天就给大家分享我的看法。雄安初步完成建设的时间点是2035年(很有意思也能印证我前期一个分析的时间点)。这个城市一旦建成，毫不夸张的说，它将远远超过这个时代任何一个城市至少20年以上!甚至可以说，雄安就不应该是这个时代的城市，因为某个特殊原因，它就提前出现了。

所以，基于过于超前的原因，建设成本也极高。讲几个点。

雄安光是规划就做了整整3年，之所以耗费这么长时间，原因就是雄安城市规划太复杂了。雄安不是一座普通的城市，严格意义上讲，它是三座城市集合而成的整体，包括，地面的城市、地下 的城市，云上的城市，我们一个一个说。

地面的城市强调建筑与自然环境的和谐共生，雄安90%建筑不超过45米，其中50%建筑不超过15米，绿化率50%，森林覆盖率超过30%。仅从这些指标来看，地面建筑是远远不够的，所以雄安大量的建筑与配套统统放到的地下。

看过《流浪地球》吗?雄安的地下城市完全能与《流浪地球》地下城媲美。雄安的地下城市包括以下功能：商业、城市物流通道、地铁、所有的城市管网、城市能源配套——印象深刻的就是一个城市变电站、能源中心地面上看就是一个公园，其实公园下面就是这些复杂的能源设施。

因为地下城要包含如此多的配套设施，所以地下工程开挖深度高达几十米，正是因为如此，雄安建设成本极高，之前规划雄安投入资金为8600亿，实际上雄安2021年开始启动建设，今年投资就要超过1万亿，4年时间不过建设了启动区一小部分，到2035年总体投资肯定要超过10万亿，至于到本世纪中叶投资要达到多少谁也说不清楚，但是几十万亿怎么也是要的。

云上的雄安规划科技感十足，你能想到的不能想到的现代科技信息化统统都有，一旦实现完全就是一座充满科技感智能化的未来之城。印象最深刻的参观雄安智慧农业科技园，类似农业科技园我也参观过不少，但是雄安这个是独一份的。我个人判断是这绝对是中国最先进最超前的农业科技园，全中国乃至全世界最先进的农业种植技术集大成的一个科技园。

在这个科技园农业应用了10大科技技术，包括，标准化无图栽培技术、激光补光技术、全天候气候控制技术、智能化水肥一体化技术、园区数字化管理技术、智能化玻璃温室设计与建设技术、潮汐式工厂化育苗技术、正压通风气候控制技术、综合防止技术、自动化设备制造与应用。这些技术一个一个详细说明可能大家都懒得看，我就挑一个给大家说。

全天候气候控制技术，这个就是通过各种设备让温室种植的环境达到五恒效果，包括恒温、恒湿、恒氧、恒音、恒尘等等。大家知道这是什么概念吗?毫不夸张的说，这里生长的植物比全中国99.999%老百姓居住环境都好。因为我们一般人家庭中最多也就是安装地暖空调，而地暖空调最多也就能做到恒温而已，国内也就是少数别墅家庭才会安装五恒空调系统，而这个五恒空调系统是传统的地暖+空调的成本3倍以上。

所以这个农业科技园不但投资巨大，大约2000亩土地总投资11亿，而且运行成本很高，我私下问了问管理人员，这个科技园大概要多久才能收回投资，管理人员回答大概要二十几年，当时我还在想，这么高端的农业项目二十几年能收回投资那还算好。

参观结束之后，我们都去品尝园区出产的小西红柿，这么好的环境以及这么多高科技，毫无疑问，这里农产品品质绝对是顶级的，但是价格也是顶级的，这个园区出产的西红柿售价要20元一斤。就看这个价格，我个人判断，这个园区要二十几年收回投资时不大可能的，原因很简单，高端高价格的农业产品一定是小众的，一旦大规模生产，不可能雄安乃至北京、河北每年能消化价格高达20元一斤的西红柿几万吨。

管中窥豹可见一斑。雄安的建设与规划将创新与高科技放到第一位，而经济性则是次要的——雄安项目很类似2008年奥运会开幕式。这是一个更注重科技应用展示的样板工程，全国乃至全世界真要大规模应用这些高科技估计要在几十年之后了，至少类似雄安农业科技园的项目，也要电力价格大规模下降才有可能——至少也要下降到1毛钱左右一度电才有可能。

但是因为有雄安这个样板摆在那里，对于科技应用的加速推广也是很有意义的。至于是否应该花费这么大的经济成本做这样的样板工程就只能留待后人评说了。毕竟雄安的定位是千年大计。不是吗?
