
# 马斯克改革的背后，潜藏着猫腻？

文/卢克文工作室嘉宾 小小罗

最近一段时间，马斯克率领他的政府效率部（DOGE）开展了一场史无前例的查账风暴，从财政部到五角大楼，从国际开发署到中央情报局，华盛顿的官僚们个个瑟瑟发抖。

这场由科技巨头主导的“反腐风暴”，究竟是制度革新的曙光，还是政治博弈的暗战呢？

事实上，这场改革的核心，可能与外界想象的有一点出入。

效率部以年轻的知识分子为主，核心团队由6名年轻的工程师组成，年龄介于19至25岁，基本都是名校毕业，拥有强大的AI技术能力，数据分析 和金融建模经验，而且都没有政府任职经历，属于是非传统建制派出身。

年轻人嘛，一是没什么根基和阅历好控制，一切唯马斯克和特朗普之命是从；二是初生牛犊不怕虎，不懂太多的人情世故，下起手来能够做到杀伐果断。

这对于深谙华盛顿官场潜规则的老油条们来说 是缺陷，对马斯克和特朗普来说却是优点。

比如25岁的加文·克里格，毕业于加州大学伯克利分校，曾在一家知名的人工智能数据分析公司担任高级工程师，这次在DOGE里负责起草关闭国际开发署的内部邮件，那封发给全体员工周一不用回总部上班的邮件，便出自他手。

还有21岁的印度裔阿卡什·博巴，在对冲基金公司担任过实习生，他在数据分析和金融建模方面水平出众，在DOGE内部主导开发“财政支出实时监控系统”，负责这次审查财政部 的支付数据。

马斯克的DOGE小组，采用“极简团队+技术赋能”的模式，打破了传统政府部门的层级与规则束缚，以近乎疯狂的手段创下有史以来最快速、最有效的查账记录。

比如针对国际开发署的查账，按照传统的审计方式，起码要请至少300位资深会计，审查一年半载才能完成如此庞大的工作量。但马斯克的DOGE小组在AI大数据技术的强大辅助下，3天就搞定了，真的就是降维打击。

那么，特朗普与马斯克这轮的“效率改革”能在多大程度上挽救美国呢？

1月28日，美国联邦人事管理局给大批联邦员工们发了电子邮件公布“买断”方案，根据这个方案，2月6日前主动辞职的员工将享受约8个月的薪酬补偿，薪酬与福利将持续至9月30日。

如果不接受“买断”，这些员工可能在后续的大裁员中丢掉饭碗且无法享受任何补偿。

首当其冲的国际开发署挨的刀最狠，上来就要裁掉超过9成的员工，还直接封门禁，关网站，全员被停工休假。

这个国际开发署啊，过去大家也仅知晓是美国负责对外输出搞颜革的，现在则是被查出还是民主党大佬们背后套钱的工具，奥巴马家族希拉里家族以各种名义圈了不少钱出来。

马斯克借着审查的名义爆出这些黑料一来可以精准打击这些政敌，收紧深层政府们的钱袋子，二来则能立竿见影的帮助特朗普收获民众的反腐信任，三来直接堵住这个乱花钱的缺口以节省开支。

接着是针对掌管钱袋子的财政部，财政部那是从美国建国以来都没有人敢去深入的查过账，这里面水到底有多深，天知道。

马斯克带领DOGE小组进财政部支付系统查数据的时候就遭遇到了代理部长勒布里克的激烈反抗，随后他被勒令停职。

杀鸡儆猴之后，财政部内部老实了许多，阻力迅速减少。

上百年没有人敢碰的数据如果被马斯克给弄来了，再配合AI模型和大数据分析，每年数万亿美元的款项花费流向何处一目了然，很多猛料都会被掀出来，哪些钱不该花，哪些钱纯属浪费，多少钱最终到了谁的手里？

这些资料分析成册之后握在马斯克和特朗普手里就是一本百官行述，威力无穷。

同样针对中情局联邦调查局和五角大楼的查账亦是如此。

实际上，特朗普借助马斯克针对强力部门的查账只是在剪除对手的基层党羽，下手果断、精准、稳妥，并不是像大家看到的那样简单粗暴。通过公布一些不合理的支出项目，但 不涉及具体人名，并且出台“买断”计划给这些人出路：

自己辞职走人，以前的事不再追究，还能继续拿8个月的薪酬。如果不安辞，硬抗下去就要真的开始清算了。

很明显的 是，特朗普这一次是先礼后兵。你体面就让 你体面，你不体面就帮你体面。

已经有一些特工同意了买断方案，因为他们知道自己屁股不干净，与其等着被清算，不如平安着陆。现在马斯克查账的金字塔顺序已经非常清晰了，底层就是国际开发署这类边缘机构，第二层是刀把子情报机构，第三层是钱袋子财政部，第四层是枪杆子国防部，而最顶层的则是美联储。

最难啃的是国防部和美国联邦储备，我们只能钦佩马斯克勇气可嘉并祝他福大命大。

面对特朗普-马斯克的重拳出击，民主党建制派们也不会坐以待毙等着被围剿。

民主党方面指责DOGE越俎代庖，你一个权力仅来源于总统行政命令的临时性组织，哪有权权力对国会正儿八经立法成立的联邦政府机构进行审查？哪有权访问核心部门的机密数据？人家国际开发署可是依照1961年的对外援助法由肯尼迪批准成立的，而且作为独立机构，本身就不受总统控制，更别说你DOGE是否有权去管人了。

2月8日凌晨，纽约联邦法官恩格尔梅耶（奥巴马任命）天不亮就紧急发出禁令，禁止DOGE访问财政部保存的敏感财务数据和个人资料信息。

此外，众议院民主党人杰弗里·斯提议发起名为《纳税人数据保护法》的新法案，用来阻止DOGE访问获取联邦政府机构的机密信息数据。

还有就是舆论攻击，民主党议员克里斯·墨菲指责马斯克针对国际开发署的封禁是为了讨好中国。墨菲的逻辑是，国际开发署是中国眼中的钉，马斯克废了这个部门，就能让中国高兴，换取马斯克在中国获取更多利益，这种言论就差说马斯克身在 美国 心在中国了。

而针对特朗普执政团队的攻击，民主党则是盯上了FBI的局长提名人，印度裔的卡什帕特尔。

帕特尔被曝去年从一名亲俄人士的电影公司收了一笔款项，用于给特朗普拍宣传片，并在宣传片 中声称，俄罗斯不是美国的真正敌人。民主党人一看 到这个 就来劲了，立刻表示，帕特尔这个人涉嫌通俄。

和民主党的反对不同，共和党内部，大部分人还是支持马斯克计划的。

比如针对国际开发署的裁撤行动，国务卿卢比奥功不可没，其同样力主大裁员，并将其由联邦政府的独立机构转变为由国务院领导的下属机关，铁了心要将国际开发署的控制权牢牢攥在手中，以方便发挥其在外交斡旋方面的巨大价值，为自己积累政治资本。

而 下个月中旬两党即将开展新一年度的预算方案磋商，共和党势必要与民主党合作推动预算案通过。

这种情况 下，如何限制马斯克极有可能成为两党讨价还价的筹码之一。

特朗普放手马斯克推动的改革，无非是两层意思。表面上缩减各种开支，节省成本确实可以开源节流，维持美国政府 的运转。因为当下高企的国债余额，庞大的贸易赤字和居高不下的通胀水平，已经压得美国人喘不过气了。

但是特朗普的任期只有四年，给DOGE设立的运行时限也才不到两年，改革事业嘛应当久久为功，就算马斯克运用AI大数据技术，提升效率节省查账的时间，但这也只能发现问题而已，根本不够“解决问题”的时间。

所以这轮的改革最终对美国来说也只是治标不治本。

深层来看，特朗普不过是借着改革的 外衣进行的报复清洗和内部权斗，对象就是民主党。

国际开发署的账单曝光之后，希拉里·克林顿奥巴马等政敌们纷纷中枪，成为众矢之的。如果马斯克将来真能从账本里，继续审查出背后所关联的民主党人的黑料，选择性的再发到社媒上通告天下，这一大帮人今后还选不选议员，选不选州长？

远的不说，两年后的中期选举，就能确保选上 来的人要么铁杆忠于特朗普，要么低头臣服妥协服软。到那时候的特朗普，才是真正的权势滔天功成名就，也能将MAGA的政治遗产顺利固化下去。

这 不，民主党籍的纽约市长埃里克·亚当斯，率先见风使舵向特朗普表忠，2月10号召开市政政府会议，要求属下不准批评特朗普，不干涉国安部在纽约的反移民执法活动。

特朗普第二天就指示司法部，撤销对亚当斯的腐败指控。看到没？只要你投靠特朗普，有罪也可以撤销指控。

至于马斯克，就算真的有为民请命的心思，也摆脱不了特朗普的控制，因为DOGE纯粹是特朗普用总统权力成立的临时部门，一句话能让马斯克开张，也能一句话让马斯克关门，不用担心马斯克跳反坐大。

古今中外，揣摩上意是很重要的学问，不分官场或是职场，如果马斯克能真正领悟特朗普的小心思，知道选择性查账，选择性执法，这样特朗普既满意马斯克的工作，马斯克自身也能善终，不会沦落到兔死狗烹的下场。

任何帝国崩溃前夕，最危险的往往不是外部强敌的重压，而是内部精英的疯狂自噬。

所谓改革，不过是重新划分蛋糕或者妄图延寿苟活罢了。
