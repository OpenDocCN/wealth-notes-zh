
# 如何抄底和复刻我们的组合，是否可以做出更多的超额收益？

这篇回答一个大家经常问我们的问题，如何复制，甚至超越我们的组合。这里，我们以我们的A股组合为例子。截至今天，24年以来，我们A股组合超额收益为33.14%。绝对收益为43.51%。通过观察我们的超额曲线。

我们可以很容易知道：
1. 我们的超额是曲折上升的。每隔一段时间，我们的超额收益都会刷新新高。
2. 排除掉去年10月8日，比较极端的曲线，我们的底部也同样是不断刷新新高。
3. 当我们在某个时间节点大幅度创新高之后，通常，接下来一段时间，超额收益会下降。但是，即使会下降，通常也会高过上一次的低点。
4. 通常情况下，在市场大跌的时候，我们的超额会更加明显。但在市场大涨之后，我们的超额会阶段性的不明显。
5. 如果市场持续震荡的话，我们的超额会温和上涨。

因此，如果大家要复刻我们的持仓，甚至想要比我们赚的更多的话，我们的建议是：
1. 如果我们超额持续回撤，并且接近市场上下两根线的中枢的话，则是比较好的复刻的时候。
2. 如果我们超额出现了明显的大回撤的时候，则是非常好的1：1复刻的机会。
3. 如果我们超额短期大幅度创行高，并且在曲线上呈现出非常明显的突出部的时候，则不是一个很好的机会。

当然，以上的规律只是一个参考。因为每一次超额的大规模上行，都伴随着市场本身的大的波动。也可能会出现比较大的意外。

最后，我们持仓的比例，都是严格按照我们的交易系统计算的。大家可以根据自己的喜好决定仓位。但是，根据我们这一年多的实际调研，按照自己喜好来决定仓位的用户，90%都会跑输我们。
