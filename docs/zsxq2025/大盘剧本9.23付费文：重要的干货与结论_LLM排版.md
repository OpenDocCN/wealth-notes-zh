
# 大盘剧本9.23付费文：重要的干货与结论

这两天的文章其实就是想从底层逻辑和心理上跟大家说清楚为什么要在上周四就说要进行20%-30%的减仓。我千言万语，其实都不如市场直接教大家，今天一下跌，提前减仓的朋友估计内心是毫无波澜的，甚至恨不得多跌一些才是。没有减的，就能更切身地体会到此前说的模糊操作的减仓思路了。

如果你非要纠结明天反弹要不要加，是不是减错的问题。我觉得不用纠结。要观察下面这点。首先是反弹是不是过了15分钟55线，如果这都没有过，就不触发你所谓减错或者要回补的信号。看着就是了。理论上来说，你至少等着15分钟回抽55线，再回落一段后，再评估你是不是减错，要不要再加回来的问题。

我上周四跟大家另外提出了一个要不要提前“加仓”的标准，那就是15分钟55线是不是被突破。白纸黑字如上图。这么多天的震荡，其实都没有突破15分钟55线。所以看着就行了。特别是今天早上的上冲，你看是不是正好到15分钟55线就下跌了，只有后面这波到达3774点的下跌实际走出来了，才能评估要不要加回来的问题。

如果你稍微看盘仔细点，不难发现：3774点正好对应了3分钟的MACD底部背离，你要在那个位置博个反弹，才具备操作的可行性。如果你这样做了，那就是尾盘的这个反弹。一切都很技术，而且一切都是提前写好的。

那么今天的这个调整有什么重要的技术意义呢？

1、今天的调整虽然尾盘有反弹，但是这是明显远离了日线布林线中轨。如果此前的轻微刺破大家还在纠结这种刺破程度对于主涨段的影响的话，那么今天这个有效跌破基本上已经坐实了。这是从3040点反弹以来第一次发生的事情。

2、今天的有效跌破意味着日线布林线中轨的锁定状态被彻底解除。也就是说我们不能再用这条均线的支撑作用来做简单的持仓操作了。同时，背离和结构分析的作用再次凸显出来。过去半年的时间，你看我什么时候说过什么背离什么结构，统统都是没有的，现在它们两兄弟的功能又回来了。

3、周线主涨段已经进入尾声。对于从3040点开始一直持仓的朋友，这会觉得有点悲伤。但事物一直是运动发展的，该来的总是会来。这个我在9月14日的周分析中已经跟大家预告过了。快乐的时光总是很短暂的。

4、上段中“进入尾声”的意思并不是说这个下跌就直接终结了主涨段在空间上的幅度。一般来说，中轨被有效跌破后，后面都应该有一段最后的反弹。但是为了防止没有这个反弹，所以必须安排大家提前在跌破的时候减仓。你仔细思考下这个操作的核心。
