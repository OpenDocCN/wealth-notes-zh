
# 卢克文星球8.21：达成共识，两国关系要大幅好转

## 中印外长达成共识，两国关系要大幅好转？
## 文/卢克文工作室嘉宾星海舰长

8月19日，广大网友被网上一则传言震惊了。

根据印媒的报道，中印外长在新德里会晤期间，兔子表示将终止贸易限制，并向印度提供化肥、稀土和盾构机。

本来吧，兔子网友对印媒的报道还半信半疑的，但是紧接着兔子媒体就报道，兔子外长与印度外长达成了10点共识，同时兔子外长还参加了中印边界问题特别代表第24次会晤，又达成10点共识。

两个消息联系起来看，似乎兔子真的向印度做出了巨大让步，提供关键物资了。

于是，很多网友都怒了，印度是什么样的国家不知道？给印度提供稀土，信不信印度马上就倒卖给美国？

那么，事情是不是真的？

一切，要从上个月苏杰生访华说起。

访华期间，苏杰生向兔子提出了一系列条件，反正就是为了中印友好大局考虑，希望兔子放开尿素、氮磷钾复合肥、磷酸二铵、稀土矿物和隧道掘进机供应。

印度求兔子办事，总要给点好处吧？并没有，印度连兔子最关心的边界和大和尚的问题都没提。所以，兔子没理印度的要求。但按照礼节，兔子外长也应该象征性回访一下，于是也就有了兔子外长的印度之行。

不过就在兔子外长访印第二天，印度媒体就引用消息人士透露，说兔子已经同意放开对印度稀土、化肥和盾构机的管控。结果兔子网友一听就炸了，纷纷觉得这是卖国。

但先别急，这个消息极大可能是个假消息。

为啥这么说？因为所谓“兔子限制对印度出口稀土、化肥和盾构机”本身就是个伪命题！这些东西兔子从来就没有限制过对印出口！

既然对印管制不存在，又谈何对印取消管制呢？

我们先来看看稀土。

这里要明确一点，兔子的稀土管制并不是针对印度的，而是针对全世界的，无论美国买也好，日本买也好，印度买也好，都要先申请许可证。

那为啥印度觉得自己被针对了呢？主要是印度人做倒卖生意做上瘾了，一个印度公司在申请了120吨兔子稀土的出口许可之后，转手就卖给了美国军工企业，结果就被兔子发现了，将其拉入了黑名单，而且对印度的倒卖行为严格审查。

结果到印度这边，就被媒体炒作成了兔子恶意卡印度脖子。

再看看化肥。

咱们都知道，印度农业技术落后，粮食亩产一直很低。

后来，莫迪的老乡、印度化学品和化肥部部长曼苏克·曼达维亚上任后，推动了一项粮食增产计划，以国家补贴的方式向农民提供化肥，以增加粮食产量。

于是，伴随着印度粮食产量的不断攀升，印度的化肥进口量也开始突飞猛进，2024年，兔子以412万吨的总量排名进口国第一，其中尿素186.5万吨，磷钾肥225.8万吨。

但是从2024年兔子夏粮减产后，兔子为了保证国内粮食产量，先是暂停了尿素出口，11月又停止了磷铵出口。印度为了弥补兔子化肥缺口，不得不寻求从更贵的中东国家和以色列进口，导致印政府化肥补贴支出增加了63%，背上了沉重的补贴包袱。

显然，这是印度自身化肥产能不足的锅，但印度偏要指责兔子，说兔子在故意打击印度农业。

最后就是盾构机了。

最近几年，印度一直在搞大基建，搞基建就离不开盾构机。当然，印度也知道和兔子关系不好，所以并没有从兔子买盾构机，而是找德国盾构机厂商海瑞克去买。

不过万万没想到，海瑞克生产盾构机的工厂都在兔子。

本来吧，这不是啥问题，不过盾构机生产好了，打包送到港口了，印度人又习惯性地使出了不付尾款的赖账招数，想用巨额港口存储费来逼德国公司发货，这样就能白嫖盾构机了。

但德国公司说的很明白，你不付尾款，我就不装船。

拖到了2024年11月，兔子发布了《两用物项出口管制清单》，其中将含稀土永磁材料的盾构机电机纳入敏感技术范畴。

也就是说，印度想要拿到盾构机，光付了尾款也不行了，还要提交最终用户证明和技术用途说明，并保证设备不会用于军事工程。

但印度人哪能保证这个？这些年印度一直在北部拉达克山区挖隧道，来增强自己往中印边境调兵的能力，而挖隧道也离不开盾构机，如果做了保证，那兔子以此为由锁机怎么办？

所以，印度一直拒绝提供有关材料，盾构机也一直在兔子港口扔着，哪怕印度人找德国总理朔尔茨告状，最后问题也没解决。

这件事明明责任在印度，但在印度媒体的宣传下，却变成了兔子恶意扣押印度的盾构机，兔子这是霸权逻辑等等。

看明白了没？稀土、化肥、盾构机这三样东西，前两种兔子压根就没针对印度，第三种纯粹是商业问题+印度拒绝提供材料，和兔子有啥关系？

既然兔子没有卡印度的脖子，那也就谈不上兔子放行三种产品了。

除此之外，就算兔子把这三种东西作为和印度交易的筹码，怎么可能外交部长去一趟，就把问题解决了？

这类具体政策的出台和修正，一般都需要直接管理部门拿出意见才行，外交部只有外交的权限，没有替主管部门拿主意的权利。所以，在兔子没有答应的时候提前放风，并不意味着兔子让步了，很可能就是印度的一种谈判策略。

你如果认了，印度就可以心安理得占便宜。如果你不认，就指责你背信弃义。

这一招，印度人已经玩了无数次了。比如1962年印度政府把兔子的忍让当作软弱可欺，说兔子接受了印度关于边界划分的方案，然后大搞“前进政策”，最后被兔子打了一通，印度人还觉得委屈，说兔子出尔反尔。

还有最近马来西亚空军战机招标，印度媒体又是大造声势说马来西亚将采购印度的光辉战机，最后马来西亚选了韩国的FA-50，印度网民又去网暴马来。

这次也是一样，印度媒体先放出消息说兔子让步了，印度人觉得兔子人怂了，印度大赢特赢。

如果回头兔子不认，那印度人就会再次指责兔子背叛印度，没有契约精神。

所以，印度媒体这种援引“消息人士”的三无新闻，我们第一眼就应该把它列为假新闻就好，为它生气更是没有必要了。

可能有人会说，既然兔子没有怂，那近期中印达成的共识是咋回事呢？

共识都有了，那肯定有利益交换，肯定就有让步了。

其实吧，大可不必这么敏感。

外交谈判，有共识太正常了，说双方观点分歧巨大也是共识，愿意继续谈判也是共识，怎么能说有了共识就是让步呢？你看看过去中印谈判，哪次没有共识？

说实话，这次兔子外长与印度外长达成了10点共识，中印边界问题特别代表第24次会晤又达成10点共识，从数量上来看，的确挺多的。但仔细看一下共识条文就会发现，基本没什么实质性的内容。

两国外长的10点共识中，要么是“一个稳定、合作、前瞻性的中印关系有助于充分释放双方发展潜力,符合双方共同利益”这种毫无营养的片汤话，要么是“中方支持印方主办2026年金砖国家领导一人会晤。印方支持中方主办2027年金砖国家领导一人会晤”这种顺水推舟的既定事项。

你说这些表述没价值吧？它的确从宏观上明确了中印关系发展的方向。

你说它有价值吧？从中真的看不出什么，更关键的 是，这些表述没有硬性的落实约束。

真正涉及具体问题的，也只有第6、第7、第10条。

其中，第6和第7明显是利益交换，印度用恢复中印直航航班和放松签证的让步。换取了兔子同意2026年继续开展印度香客赴西藏神山圣湖朝圣并扩大规模。

为啥印度人这么看重朝圣的事情呢？

从表面上看，是因为宗教。

兔子境内的冈仁波齐峰和玛旁雍错，在印度教中，被认为是三大主神之一的湿婆神苦行和升天之地，所以被称为“神山圣湖”。

对印度教徒来说， 一生必须要来这里朝圣一次，积累功德。其中莫迪在1980年就曾亲身朝圣过，就是因为这次朝圣，为莫迪“虔诚印度教徒”形象也加了不少分。

对莫迪来说，说服兔子同意恢复朝圣活动，那对以印度教徒为基本盘的支持者来说，可是实打实的政绩。

但实际上，印度人搞这种朝圣，包藏着领土野心。

其实一开始，印度教的（神山）“Kailash”并没有特指冈仁波齐峰，只不过英国殖民时期为了侵占兔子领土，故意宣扬冈仁波齐峰就是“Kailash”。

英国人的想法是，只要去的印度人多了，就能顺势对西藏发起主权声索。

现在印度虽然独立了，但印度人还是继承了这种图谋，印度媒体长期渲染“神山归属争议”，而朝圣活动也成为了传播此类叙事的载体，作为在国际上渲染西藏属于印度文化圈的一种例证。

既然明知印度包藏祸心，为啥兔子还同意印度人来朝圣呢？

说实话，兔子不在乎印度这种小伎俩，毕竟现在世界上还是讲实力的，冈仁波齐和玛旁雍错在兔子境内，印度想夺也夺不走，几个印度香客（一年限额750人）来朝圣，又能造成什么实质性伤害呢？

既然如此，不如大度一点，让他们来呗。

你看韩国人一直认为长白山是圣山，兔子不是也没限制韩国人去长白山么？

除此之外，第10条也算是兔子照顾了一下印度的愿望。

印度希望兔子和印度一起谴责美国，支持印度。

但兔子不愿意被印度当枪使，更不愿意让自己成为印度和美国之间博弈的筹码，所以兔子的表述是“维护以世界贸易组织为核心、以规则为基础的多边贸易体制”之类比较原则性的话，对印度来说，聊胜于无吧。

除此之外，中印边界问题特别代表第24次会晤10点共识里面，大部分也都是片汤话，唯一实质性的内容就是分享水文信息，但水文信息2022年之前一直在分享，这次只不过是把分享机制恢复了而已，所以也谈不上多大的让步。

有人可能不明白了，既然谈不成什么实质性成果，为啥还要搞外长互访呢？搞得跟中印关系真的转暖了一样。

其实吧，这里面藏着兔子的阳谋。

印度为啥走近兔子？无非是做给美国看，你要是不让步，我就投奔老中。

这个情况美国清楚，兔子也清楚，所以兔子并不排斥中印关系正常化，但也不会真的让渡什么关键利益，维持一个表面友好就行了。

如果我们仔细看一下中印外长共识第十条就会发现，给印度打气的味道非常浓，相当于说“爷们加油！可别丢份啊！我在道义上支持你！”

只有印度感受到了来自兔子的支持，才会有胆子和美国死磕，和兔子一起分担来自美国的压力。毕竟政治嘛，就是把朋友搞得多多的，把敌人搞得少少的。

兔子需要印度对抗美国，而不是把印度逼到美国那边去。所以做出一个中印友好的表演性姿态，无可厚非。

而这，才是最近中印关系策略性转暖的真正原因。
