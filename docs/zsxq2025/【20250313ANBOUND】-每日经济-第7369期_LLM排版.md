
# 为什么说家电制造也是新质生产力？

2023年9月，国家首提“新质生产力”。2024 年的《政府工作报告》则提到，要大力推进现代化产业体系建设，加快发展新质生产力。今年“两会”发布的《政府工作报告》再次提及新质生产力，并且强调要“因地制宜”地发展新质生产力。毋庸置疑，新质生产力和高科技不仅是今年各地的工作重点，也会是未来“十五五”规划的核心任务。而此次国家强调“因地制宜”，就是明确禁止那些“一哄而上”，违反产业发展规律，类似于“追星式创新”的行为。欲速则不达的做法对地方经济的发展是有百害而无一利！

安邦智库（ANBOUND）一直强调，新质生产力是创新起主导作用的、可持续的先进生产力，能够给经济带来真实的增长和价值。从生产制造的角度来看，既能体现技术创新，还要能达到或实现产品的更新换代与可持续技术升级的产业才能被称作新质生产力，否则谈不上是生产力。所谓新质生产力，关键是这个产品及其背后的产业，能不能经受住时间的检验，始终维持自己的产品定位，拥有持续的技术升级能力，不断的推陈出新，更新换代掌控市场的能力。

理解了这一点，就意味着地方政府需要将资源“智慧地”配置到那些真正拥有符合创新型先进生产力概念的产业。这种产业，不仅对当地经济发展、劳动就业有益，在全球市场上也能拿得出手，让外界折服的“中国制造”。而按照这种标准，可供选择的产业范围其实是缩小了。安邦智库的一位资深研究员指出，现在也就是汽车制造和消费电器，比如家电产业可以被称作是发展新质生产力的真正抓手。

单说家电制造，也许会有一种质疑：现在国内家电行业基本是供大于求，靠创新力拉动需求，举步艰难，何以能够与新质生产力的概念产生联系？还有一种担忧认为，我国的家电行业可能只有出海一条路了，而且这条路的未来极具不确定性。实际上，家电行业是中国最早“走出去”，最为成功的一个行业，在世界上已经形成了大气候。

从时间线的角度来看，我国的家电企业花了近半个世纪的时间从“一穷二白”发展到今日布局全球的态势。改革开放以前，我国基本要从前苏联和东欧国家进口家电。现在耳熟能详的美的集团在创立之初起初只有做塑料瓶盖、五金件和汽车配件的能力。改革开放初期，日本的三菱、东芝、松下、日立和三洋等品牌进入中国市场，发展十分迅速，形成了品牌效应。而此时的中国家电企业处于“萌芽状态”，海尔、格力、长虹等品牌都在缓慢起步的阶段。很长一段时间里，国内的家电行业是以贴牌代工的方式，将产品运往全球各地的。通过贴牌生产，中国家电业的经营规模得到了迅速的扩大，中国家电出口年平均增长幅度为47%，近百亿元的产值。不过，一些家电企业意识到这绝非长久之计，所以开始走自主研发的道路。90年代以后，国内家电市场与规模迅速扩张。全国几乎各个地方都有自以为豪的地方品牌。与此同时，一些走在行业领先地位的企业已经开始“走出去”了。1998年，长虹开始布局海外业务；1999年，TCL 在越南建立首个海外生产基地；2001年，格力在巴西建立工厂；2007年，美的在越南设立首个海外工厂……行业巨头的网点、工厂和基地遍布全球。比如早期走出去的企业之一海信目前在全球设有 36 个工业园区和生产基地、26个研发中心，初步形成欧洲、美洲、东盟、中东非、亚太和中国区全球“5+1”区域中心研产销一体化布局。需要注意的是，除了出海生产，一些巨头还收购海外名牌企业。2016年，青岛海尔斥资约 54 亿美元成功收购美国制造业巨头通用电气的家电业务板块就是一个经典例子。而像海信从 2015年开始则陆续收购了夏普的墨西哥工厂、东芝映像系统公司以及欧洲高端白色家电品牌古洛尼等。

如今，中国的家电产品已经“吃下”了海外市场，甚至胜过了以前奉为“老师”的日本品牌。《日本经济新闻》援引调查公司 BCN 的最新调查数据显示，2024年在日本销售的平板电视中有一半以上都是中国品牌。报道指出，这是自2004年以来，中国电视品牌在日本市场的占有率首次突破50%大关。《日本经济新闻》援引的社会反馈称：“现在越来越少的人坚持只购买日本本土品牌的产品。特别是在年轻消费者中，他们对中国品牌更为熟悉，通常会提前通过网络查询产品的性能和价格信息，进店后便直接指明要购买中国品牌的产品”。

从公开的数据可以看到，中国的家电制造，即使在经济波动的阶段，出口水平也是压得住阵脚的。以空调、洗衣机、冰箱等产品在 2023年的表现水平为例，2023年1-12月，空调累计出口量为4799万台，同比增长4.6%；冰箱累计出口量为6713 万台，同比增长22.4%；洗衣机累计出口量为 2879 万台，同比增长39.8%；液晶电视累计出口量为 9887万台，同比增长7.5%。海关总署最新公布的数据显示，在2024年全年家用电器出口数量达448144万台，比上年增长 20.8%。从产品类别来看，大型家电如空调、冰箱和洗衣机等依然是中国家电出口的主力。展望 2025年，出口市场排产增势仍然保持高涨，新兴市场的增长动力依然强劲。虽然美国的关税政策带来了不确定性，但也有观点认为这在无形之中驱动我国家电制造产能出海，尤其是在东南亚的制造体系建设。此外，海外通胀下降、货币政策压力趋小、耐用品补库存等因素将持续为海外需求增长带来支撑，减轻可能因为地缘博弈等因素带来的压力。

值得一提的是，我国的家电行业不仅有巨头，也有众多“小老大”。这些经营时间通常超过 10年的“专精”中小型企业，也有很多在海外打造着“中国制造”的名声。安邦智库的研究人员在南方某市调研时听闻了一家当地“小老大”的经验。这家小企业主要在欧美国家生产智能电热毛巾架。虽然这种产品在国内不吃香，只限于特定场所，比如酒店等，但在国外这家企业拥有巨大的市场。而且因为产品的稀缺性及性价比，这家“小老大”的地位很难被取代，地缘政治对它的影响也不大。其实，中国还有很多这样的例子。

安邦智库曾说过，相较于那些可能昙花一现的“独角兽”，“小老大”其实是支撑中国经济发展的主力军。成千上万亿的政策资源砸在“独角兽”身上，远不如砸在“小老大”身上，对中国产业的升级以及经济增长更为有益。这应该是日后地方政府制定政策时需要考虑的事情。

当然，家电出海目前面临的问题主要是短期效益和长期战略的平衡。市场环境的差异、文化障碍和人才匮乏、市场竞争日趋激烈、资金压力以及供应链和物流问题都可能对出口业务造成影响。不过，虽然家电产品出口面临诸多挑战，但家电产业的全球化进程是长期且不可逆转的。作为新质生产力的抓手，家电产业有实力逆风飞扬，在海内外市场上站得住脚，打造中国制造的品牌，为我国带来真实的经济增长和价值。

最终分析结论（Final Analysis Conclusion）：
家电制造是中国最成功，并且是最早出海的一个产业，在世界上已成大气候。这是新质生产力的一个绝佳例子，因为它是创新起主导作用的、可持续的先进生产力，并且对我国经济增长，对出口十分重要。（ZZJ）

【形势要点：国内拟通过新一轮“上山下乡”解决毕业生就业】
人社部部长王晓萍日前在全国两会民生主题记者会上表示，今年，应届高校毕业生 1222 万人，脱贫人口务工规模需要保持在 3000 万以上，还有大量的农村转移劳动力需要实现稳定就业。就业总量压力不减，结构性矛盾更加突出，群众就业增收面临压力。同时，外部环境更趋复杂严峻，国内经济回升向好的基础还不稳固，可能会对就业产生一定影响。针对中国高校毕业生就业问题，人社部也宣布将出台新一轮支持政策，其中包括稳定招聘规模，扩大“三支一扶”“西部计划”等基层项目人员数量，实施“百万就业”见习计划。所谓的“三支一扶”是指大学生毕业后到农村基层从事支教、支农、支医和扶贫工作。据人社部流动管理司去年发布的数据，2024 年中国中央财政支持招募 3.44 万名高校毕业生到基层服务。自2021年第四轮“三支一扶”计划实施以来，全国共选派 12.17万名“三支一扶”人员到基层服务。有分析认为，如果根本的结构性问题没有被解决，一味地鼓励年轻人下乡，或是去比较内陆的省份工作，这是很困难的。一方面，基层岗位吸引力受限于待遇、发展空间和公共服务资源。虽然“三支一扶”人员服务期满后留任率较高，但长期扎根仍面临职业晋升通道狭窄、区域发展不平衡等问题。另一方面，青年群体对职业适配性要求提升，新兴行业与传统基层岗位需求也存在错位。

【政策：工信部印发卫星网络国内协调管理办法】
工业和信息化部近日印发《卫星网络国内协调管理办法（暂行）》。一是明确了国内协调基本原则。就国内协调地位判定标准、各单位职责分工、干扰处置和监督措施等方面进行了规定。二是规范了国内协调程序。明确了建立国内协调关系的条件和流程，国内协调需求征集、汇总和公示，以及协调反馈等环节的时限要求。三是优化了国内协调机制。强化了国家无线电办公室统筹协调作用，提出了无需建立国内协调关系的情形，减少国内协调对象范围，引入五种完成国内协调的形式，降低国内协调复杂度。下一步，工业和信息化部将做好《办法》宣传和解读以及配套制度的制定，确保《办法》各项措施落到实处，进一步激发航天企业活力，推动航天产业高质量发展。据悉，去年 11月8日，工信部无线电管理局公开征求《卫星网络国内协调管理办法（暂行）（征求意见稿）》的意见。

3月10日中国对原产于美国的鸡肉、小麦、玉米、棉花加征 15%关税；对高粱、大豆、猪肉、牛肉、水产品、水果、蔬菜、乳制品加征 10%关税，开始正式生效。当前，新希望、大成、安佑等多家饲料企业已经宣布涨价。作为饲料主要原料，此次加征关税或进一步推高饲料成本。菜粕现货报价也出现了明显的上涨，并且上涨幅度明显。受上述影响，3月10日，南通散装粉状菜粕报价3700 元/吨，较3月7日涨330 元/吨；南通散装颗粒菜粕2720元/吨，较3月 7日涨340元/吨。此前，2月份，部分饲料企业已经宣布涨价50—300元/吨。开源证券分析师陈雪丽认为，关税反制助推国内农产品价格上行，包括养殖板块特别是生猪价格有望走出谷底。当前市场整体生猪供给充足，但到 3 月初基本已度过春节后供给压力最大需求最弱时段。展望后市，需求边际向好叠加中方的关税反制提振，猪价震荡磨底或渐至尾声。贸易加征关税对我国粮食进口和供给实质性影响有限，但对市场情绪有积极影响。食品价格下跌为 2月CPI 价格负增长的主要原因，其中猪肉影响CPI环比下降 0.03个百分点。
