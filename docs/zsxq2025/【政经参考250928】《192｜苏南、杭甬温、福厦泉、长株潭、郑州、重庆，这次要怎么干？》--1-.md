
# 政经参考250928：《192｜苏南、杭甬温、福厦泉、长株潭、郑州、重庆，这次要怎么干？》

## 苏南、杭甬温、福厦泉：东部经济强区的重点探索先锋型

### 苏南重点城市：先进制造业升级的样板

试点文件提出多项创新举措：

1. **技术协同创新**  
   支持牵头国际大科学计划和大科学工程，鼓励企业、高校、科研院所共建产学研创新联盟，引导行业领军企业发起设立并购基金，拓宽创业投资市场化退出渠道。

2. **土地管理改革**  
   合理划分土地管理事权，赋予试点地区更大土地配置自主权，建立产业用地长期租赁、先租后让、弹性年期供应、作价出资等市场供应体系。

3. **城市全域数字化转型**  
   支持无锡市构建以物联网为核心的数字产业体系，建设车联网先导区；依托南京、苏州建设基于人工智能和5G物联的智慧城市大脑，协同发展“互联网+先进制造业”。

4. **户籍制度改革**  
   探索户籍准入年限同城化累计互认，建立统一的人力资源服务平台和公共就业服务平台，推动城市一体化发展。

### 杭甬温：数字经济与民营经济的活力结合

杭甬温地区的独特优势是全球领先的数字经济生态和全国最活跃的民营经济。政策重点包括：

1. **创新科技资源配置方式**  
   聚焦“互联网+”、生命健康、新材料等领域，探索以市场需求为导向的关键核心技术攻关及迭代应用。

2. **土地配置改革**  
   改革产业用地“价高者得”的单一竞价方式，探索推行“限地价竞贡献”等模式，鼓励未来贡献大的企业获得用地。

3. **数据要素流通规则**  
   推动公共数据有序开放利用，探索公共数据授权运营，创建行业重点数据资源目录、标准及应用规范。

4. **数据融合创新应用**  
   以“产业大脑+未来工厂”为依托，拓展多元数据赋能场景，优化产业链、预测市场需求，提升效率。

5. **自贸试验区数据流通**  
   探索推动中国（浙江）自由贸易试验区港航物流、油品能源、商贸等行业数据的流通，打造数据要素赋能实体经济的“浙江模式”。

### 福厦泉：海洋特色与对台特色

福厦泉的试点方案具有海洋特色和对台特色两大特点：

1. **海洋特色**  
   提高经营性用海项目市场化配置比例，完善海域使用权的出让、转让、抵押、出租、作价出资等权能，支持探索分层设立使用权。

2. **对台特色**  
   推进台湾职业资格和职业技能等级直接采认，支持台资金融机构在大陆参股、控股各类金融机构，鼓励台资银行、证券、保险等机构在试点地区设立持牌经营机构。

## 长株潭、郑州、重庆：中西部内陆城市的综合探索型

### 郑州的综合探索

郑州的试点文件非常均衡全面，主要做法在前面各区域试点文件中均有体现。核心目标是为类似的中部省会城市做一次集合各种要素特点的全面且典型的探索。郑州的优势在于高铁物流枢纽、航空港实验区、商品期货交易所以及河南近亿人口的内陆市场，改革旨在形成内陆发展新模式。

### 长株潭：职业教育与数字经济融合发展

长株潭是典型的中部工业城市群，试点文件亮点包括：

1. **职业教育发展**  
   支持加快本科层次职业学校布局，支持开发区建设职业院校、技工院校，推进职业院校股份制、混合所有制改革，促进职普融通。

2. **数字经济融合**  
   推进国家超级计算长沙中心、湖南国家级车联网先导区建设，加快北斗产业发展和规模应用。

3. **绿色转型**  
   深化绿色产品政府采购，生态环境损害赔偿、环境权益抵质押融资等改革，探索重工业基地的绿色转型经验。

### 重庆：城乡要素双向自由流动

重庆的特点是大城市和大农村并存，试点方案的核心任务是探索城乡要素双向自由流动的制度性创新：

1. **土地资源市场化配置**  
   赋予重庆市更大的土地配置自主权，健全城乡统一的建设用地市场，优化产业用地供应方式。

2. **城乡建设用地增减挂钩**  
   依托重庆农村土地交易所开展城乡建设用地增减挂钩节余指标调剂，让农村沉睡的土地资源变现，满足城市需求。

3. **战略科技力量布局**  
   加快建设成渝区域科技创新中心，支持新能源领域全国重点实验室建设，参与创建一批国家技术创新中心、制造业创新中心、产业创新中心、工程研究中心等。

4. **数据治理**  
   依法打击黑市数据交易、大数据杀熟等数据滥用行为。

## 总结

各试点区域根据自身特点，探索要素市场化改革的路径，从苏南的先进制造业升级、杭甬温的数字经济与民营经济活力结合，到福厦泉的海洋特色与对台特色，再到长株潭的职业教育与数字经济融合、郑州的综合探索，以及重庆的城乡要素双向自由流动，各具特色，共同推动中国经济的转型升级。
