
# 中美瑞典谈判解析

从前期透露的信息来看，本次谈判双方的主题是很丰富的，包括：

- 美方希望中方进一步放开稀土出口限制；
- 中方希望美方取消芬太尼关税（20% ）；
- 针对中方购买俄罗斯能源问题，也是本次轮谈判的议题。

但是，理想很丰富，现实很骨感，双方谈判的议题很多，但是，却没有任何一项达成协议；唯一勉强能拿得出手的成果就是前期暂缓的34%对等关税，继续暂缓90天。

说实话，这个成果几乎就不能算是成果。为啥？因为特朗普已经宣布正在考虑访华事宜，并且声称得到了中方邀请，在这种情况下，前期暂缓的对等关税继续延期几乎是必然的，总不能再特朗普访华前，双方再次就关税问题大打出手，这样特朗普还怎么访华？

那么，为什么本次万众瞩目的中美会晤却没有任何成果呢？我们一个一个来说。

首先是中方诉求的取消芬太尼关税的问题。从目前美方对外达成的关税协议来看，美方基本是这样一个逻辑，属于盟友范畴的，大致就是加征15%关税，非盟友的一般是20%以上，如果美方答应中方诉求，取消芬太尼关税，那么美国对华加征关税就变成10%，比盟友还低，这对于美方是很难接受的。

特朗普执政半年，现在回头来看，特朗普执政的逻辑越来越清楚了，什么大战略特朗普是不大感兴趣的，作为商人，特朗普首先要做的就是尽量让美国实现资产负债表的平衡。拜登在卸任之前给特朗普挖了一个巨坑，24年10月到25年1月这个期间（也就是特朗普当选还没上任之前），美国财政赤字同比上升了57%！高达8402亿美元，而在2024年度同期，拜登任上的时候财政赤字是5360亿美元，也就是短短三个月时间，拜登在最后时刻拼命花钱，整整比正常情况下多花了3042亿美元。

所以特朗普上任的时候其实就是接了一个烂摊子，他要做的首先是想办法将赤字这个坑得填上。特朗普上任5个月（2—6月）财政赤字下降了32.6%，金额是4985亿美元，上年同期财政赤字是7401亿美元，同比削减了2416亿美元。特朗普填坑的方式，一个是让马斯克去裁员，但是效果不大，然后就是开源与节流，开源就是加关税，节流就是砍联邦费用，6月美国财政首次实现盈余，能做到这一点主要贡献就是特朗普砍掉了90亿美元教育经费。

所以，特朗普在对外谈判时，能多加一点关税是绝不放过的，除非对方在其他领域给美国输送利益。很明显，中方是不可能在其他领域给美国输送利益来换取取消芬太尼关税的，所以在这个问题上双方自然谈不拢。美方不取消芬太尼关税，那么中方同样不可能放宽对稀土出口的管制。现在中方发放稀土出口许可证都是短期申请，中长期申请一律否决，也就是说，目前欧美制造业能拿到的中方稀土都是仅仅只够3个月左右生产所用，中方在稀土上卡得很紧，已经做好了美方在3个月后翻脸，中方再次用稀土卡脖子的准备。

最后就是购买俄罗斯能源问题。在中美会晤之前，特朗普声称，要对购买俄罗斯能源国家发起二级制裁。所谓对购买俄罗斯能源二级制裁不过是特朗普一鱼二吃的手段，一方面拿着这个声明做筹码，力图在与欧盟谈判中讹诈更多利益，欧盟果然是跪了（最近接受了美方单方面加征15%关税的决定）。另一方面，也是拿着这个威胁，企图换取中方更多让步与利益。

但是特朗普可以讹诈欧洲，却很难讹诈中国，因为中国很清楚，出于战略收缩的美国在财政困难的背景下，绝不可能为了欧洲去制裁中国，那不仅会再度挑起与中国大规模关税战，让美国利益受损，另外，如果严厉制裁禁止俄罗斯能源出口，只会导致国际油价上涨，从而给美国带来更大通胀压力，让特朗普期盼的降息更加困难。

所以，关于购买俄罗斯能源的议题不过是中国美联合给欧盟演的一场戏，美方对欧盟有个交代，你看，这个议题我已经谈了，但是中方很强硬，所以谈不拢。

从中美瑞典谈判情况来看，中美关系目前处于一个相对平衡点：双方既无进一步改善关系的基础，同时也暂时不具备进一步恶化的条件——中美目前双方都有各自内部问题要解决，中国要解决产能过剩以及反内卷的问题，特朗普政府迫切要解决美加联储降息的问题，对于特朗普而言，要平衡美国资产负债表，降息才是目前的关
