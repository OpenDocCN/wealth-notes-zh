
# 红马甲，黄马褂，东哥为啥变了脸

上个月，春假，我开车在美帝闲逛，有十来天没吃中餐。这脾胃啊，说不上的空虚。在又一次腹中空空不知道吃什么的路上，突然间，一个熊猫的大脸蛋子闪过，我的第六感顿时敏锐起来，掉头回去，停下来细看：果然，PANDA EXPRESS,泪流满面。走进店里，三三两两的顾客，全是老外；无精打采的服务员，不冷不热；菜就那几样，一份主食+两个菜是10.5美元，量很大。倒是外卖挺多的，一顿饭不到半小时的功夫，好几拨外卖员进来取单。吃完我使尽喝口水压了压肚子，这菜不中不洋，不咸不淡，关键是，就这个水平的熊猫快餐，已经在美帝开了2458家店。

美帝傻白蠢，不流行收加盟费待收齐后卷款跑路的新商业模式。像这熊猫快餐，全都都是直营店，能开这么多，说明生意还行，利润可观。利润从哪里来?我吃完饭在出口处看到一个牌子，顿时想到马老宗师理论的精髓：剥削和压榨。

前两项的什么店长和副店长属于是画大饼，可以忽略不计。单看下面的大厨48K服务员40K的年薪，换算成每月是4千和3千3，根据著名公理”陈平不等式”,在美国月收入3千美元比不上在中国每个月2千人民币。我掏出计算器仔细算，大概而言，在熊猫快餐里上班的大厨和服务员等于是才拿2千多块人民币。

赤裸裸的压榨啊!根本用不着使用强大的算法，就可以对底层服务人员极尽压榨。更令人心寒的是，在美帝那疙瘩，根本没有人对这些底层服务员施以温暖的关怀。没有人查查消防，气改电，电再改气，全力以赴保障大厨的安危。没有人瞅瞅卫生，健康证办了没，拍黄瓜符不符合标准，只为让服务员工作得舒心。更没有人拎着一桶绿漆教他们把招牌刷一刷，第一张图大家也看到了，熊猫快餐的照片既不雅观，也不显眼，很影响生意。没人教，没人问，熊猫快餐只是个缩影，大批量的美帝餐饮界服务人员挣扎在温饱线。他们的川普，在打高尔夫；他们的州长，在争论厕所分类；他们的企业家，要么在白宫舔川普屁股，要么来京城正襟危坐认真学习……环顾地球村，他们连一个像东哥一样振臂一呼的好兄弟都没有。

东哥奉劝某平台：

你可以十几年不为任何一名骑手缴纳五险一金；你可以用强大的算法对骑手极尽压榨之能事；你可以长期对骑手事故率超过快递小哥三倍以上的安全问题置若罔闻；你可以在约60%以上门店都不赚钱的餐饮行业赚取千亿利润，丝毫不顾餐饮从业者的艰辛和挣扎；你还可以在超过40%毛利的幽灵外卖暴利驱使下，纵容和鼓励幽灵我认为，这里的“某平台”,首先指的就是美帝，然后才剑指某团。而这一段苦口婆心醒世良言之厉害，关键在于第一句：五险一金，一个都没交，多少人，截至2024,某团骑手总数745万。

有点常识的都明白，任何保险都是一场接力赛，新人交，老人拿，等到新人慢慢变老，库房里还有没有的拿。最新预算报告显示，2024社保基金收入11.89万亿，支出11.99万亿。缺口在哪?东哥给指了出来。

但这只是东哥厉害的一小步，然后他雄赳赳迈出了更厉害的一大步。东哥算互联网大佬，应该没啥疑问。

互联网大佬应该是什么样子，总的来说分为两类：

一类很高端很宏大，比如阿里，那谈的都是国际问题，马Jack退隐之前出席的都是国际会议。再比如携程的梁建章，富则富矣，仍十分钻研学问，是著名人口专家，常有长篇大论。

二类比较低沉，低沉的不使劲搜索都找不到这号人。像是已在电商里排老二的拼多多黄老板，就没听他说过话。还未上市但规模更大的字节跳动更是如此， TikTok 把美帝搞得鸡飞狗跳连骂带喊，却从未见张一鸣老板出来装个逼。

要么一类，要么二类，只有东哥，属于特殊的另一类。他很高调， 一直按耐不住的高调，所以他肯定不属于第二类。但高调的他，走的偏偏是底层路线：给农村老家的乡亲发年货，与快递小哥称兄弟，苏北口音从未刻意纠正，前两天又拉着一众外卖员吃起了火锅大餐。

其实东哥的日常生活，活得是非常高级的。镜头之外，他出没于美帝常春藤、迪拜顶级套房、地中海小岛大别墅……他的老婆怀着儿子在美帝出生，他的女儿在管家团队伺候下学着英伦风……但像很多老婆孩子在外流浪的人物一样，只要出现在镜头里，他就变得很朴素：兄弟们我来了，我来跟你们送外卖了，我穿上这身衣服是跟你们一样的，我不穿这身衣服的时候跟你们也是一样的，你我都是服务员，你在服务千家万户，我也在服务千家万户。

这一次吃火锅，座位里有一位兄弟代表，扎根基层送快递18年，勤勤恳恳任劳任怨，获得内赠股票，身家已超百万，人称“李百万”。这位李百万，富了之后不仅不退休，反而是发自肺腑爱上了快递这个职业，不仅自己继续送，还把儿子介绍来也当了快递员。他儿子小李兄弟，工作时开着红色小货车，下班开的是奥迪A6 黑色款。

也许是我敏感，这一套唠嗑，怎么听怎么像是那会场里，有代表对着镜头侃侃而谈：扎根农村养大牛，越养越上瘾，牛即是钱，钱即是牛，已数不清多少牛多少钱。火锅场里的小李兄弟，还把父亲珍藏了18年的初版快递服拿出来送给了东哥。东哥没有丝毫的嫌弃， 一把就披在身上，温暖的大手与小李兄弟开奥迪A6 的小手紧紧握在了一起。

开奥迪A6, 看看这品味，国情之下什么人士开这车最盛行?员工代表的向往，往往折射出他老板的理想。如果东哥换一身装束，在更多摄像机镜头的瞄准下，握着一双满是老茧的手深情地说：我来晚了。没有丝毫违和感。

我没见过其余互联网大佬这样干，所以东哥是最有理想的。可是在上个月那个万众瞩目的会场里，互联网大佬云集，连不给骑手交社保的某团光头王都坐到了第二排。偏偏他，理想丰满，现实骨感。

你说，他能不非常气愤，对着某平台狠干。
