
# 黄仁勋访华合作，美联储按兵不动

黄仁勋访华了。几个月前他就来过中国，打着视察中国公司的名义来的，其实是来北京想托门路见领导层，但那时候我们不太看重他，可能觉得黄仁勋精明算计吧，上次他来愣是没见。这次黄仁勋又访华了。背景很简单，懂王上台后重新打贸易战科技战，英伟达芯片有被限售的危机，黄仁勋几天前，花大钱找门路，终于和懂王在庄园里吃了一顿饭，对懂王表忠心。那顿饭吃的效果还是有的，懂王没下死手。

——两名消息人士告诉NPR，在人工智能芯片巨头英伟达首席执行官黄仁勋上周五（4日）前往佛罗里达州海湖庄园出席晚宴后，美国白宫改变了限制英伟达向中国出口H20的计划，并暂停进一步限制这种芯片出口。其实这个h20芯片已经是阉割版了，并不是最先进的芯片，属于特供中国的芯片。之前拜登限售的时候，搞了芯片国家配额，新加坡分到全球15%的比例好像，然后新加坡其实只用了很少一部分，其他芯片都转手卖给了大陆，新加坡赚这个中间商的差价。懂王上台后，肯定是知道这种转口洗澡套路的， 就打算禁止，黄仁勋那顿饭只是延缓了这个禁令一段时间。

吃完那顿饭后没几天，懂王打贸易战一直没效果，逼我们打电话都没逼成，就打算加大科技战力度，于是懂王对黄仁勋就反悔了。——4月15日，英伟达表示，公司日前收到美国政府通知，H20芯片和达到H20内存带宽、互连带宽等的芯片向中国等国家和地区出口需要获得许可证。黄仁勋没办法，就飞到中国谈了。这次黄仁勋访华是有效果的，上次他转了好几天一个大佬都没见到，这次他见到了两个。一个是副总理的峰哥。他没见到二号，只能见到峰哥，说明我们对黄仁勋还是比较保留的。峰哥不管科技但管经贸，也算他职责之内，就见了。一个是deepseek的梁文峰。这个是最重要的。因为英伟达不担心H20芯片这几十亿美元的小钱，英伟达担心的是中国彻底不用英伟达芯片转而扶持国内芯片，美国封禁芯片，中国只是短期内面临大模型算力短缺的现状，中长期靠国产是能解决的，因为有deepssk这个搞数据蒸馏的新路数，不必多依赖高精尖的芯片。

而AI芯片是有生态的，中国用久了自己的国产芯片，意味着中美ai芯片泾渭分明，久而久之英伟达将失去全球一半市场，而且中国培育出来的ai巨头企业，还将在全球出击打击英伟达的全球市场。所以对英伟达的禁令一出中国芯片相关企业的股票就涨了。这个在普通芯片领域就证明了，普通芯片国产化了后，中国企业将其价格做成了白菜价。

——2024年最后一个月，一个震撼业界的消息突然引起了所有人的关注。中芯国际，作为中国的芯片制造龙头，宣布将其28纳米工艺生产的芯片价格，从2500美元直接降至1500美元。——2025年刚开年，中国的外贸成绩单就公布了，你可能没想到，我们出口的芯片，平均每块才卖0.535美元，也就是不到4块钱人民币，比进口芯片的价格低了将近四分之一。所以上面不怎么待见英伟达也很正常。中国迟早要出几个芯片巨头企业，这些中国芯片企业也迟早要出海和美国芯片企业竞争。黄仁勋和他的英伟达，长远来看必然是我们的对手。

我们这次让黄仁勋来，主要是黄仁勋在懂王那里栽跟头了，黄仁勋态度比上次诚恳。而且，短期内，我们还是需要英伟达芯片的。所以，这次梁文锋和黄仁勋的交流，很重要。——据英国《金融时报》报道，英伟达CEO黄仁勋今日访问北京并会见了生成式人工智能初创公司DeepSeek的创始人梁文锋，讨论如何为中国设计下一代芯片，以满足客户需求以及中美双方的监管要求。这是最重要的，英伟达专门为deepseek设计能绕开监管的芯片。因为黄仁勋肯低头帮中国做这个事，所以他这次来才能见到峰哥。而且是单独会见，秘密会见。

不过可想而知，黄仁勋回美国后，懂王要整他。黄仁勋现在是想骑墙中美，但这条路很难走。黄仁勋放弃中国市场，那就是在未来制造一个强大到自己不可能战胜的对手，论成本控制美国企业怎么赢的了中国企业？黄仁勋不放弃中国市场想给中国绣红旗表忠心，懂王不会轻饶了他。而对于中国来说，扶持自己芯片产业才是王道，才是正路。

甚至之前有个段子，一群将大半身家投到芯片产业的基金经理和投资客，曾表达过这么一个意思，就是如果美国取消科技禁令重新卖芯片给中国，那他们就自己买个船去台湾开第一枪，挑起中美战争让中美重新恢复到脱钩对峙。原因很简单，他们的大半身家都投给了国产芯片，他们死也不愿意中美芯片问题缓和然后他们的钱血本无亏。是的，这几年从中央基金到地方财政再到民间投资，数以万亿计的钱都投到芯片产业了，现在别说懂王不肯取消芯片禁令，就是懂王肯取消，我们国内也一大堆力量不肯中美握手言和重归旧好。从这个看似搞笑实则严肃的例子，其实可以看出，亲美派为什么会一年比一年势微。

在2018年贸易战开打前，中国亲美派是很势大，中央很多权贵子弟在跨国企业任职做中美之间的掮客，沿海发达的地方政府的经济支柱是出口，民间企业也多有外贸主导的或者美元资本投资的。这些人的利益来源和美国有千丝万缕的关系，而屁股决定脑袋，这些人就算爱国，他们出于利益的考虑，也不会多么反美。比如现在还有权贵卖美国稀土。

而18年后，中国经济生态有几个大的方向调整。一个是中国外贸出口的重点从美国变成了东盟和欧盟。一个是国家扶持国产芯片，各方砸了万亿级别的钱去押注国产芯片。一个是国家扶持的国产电车行业，也是不依托美国市场和美国技术，他们反而厌恶美国动用长臂管辖阻碍中国电车出海，而这又是一个万亿级产业，背后的各方金主势力都不会怎么亲美。还有一个最重要，就是中国调整区域发展，大力扶持了成都杭州武汉合肥等新一线城市，和老牌的深圳上海广州相比，这些城市的产业并未主打出口，而是服务内循环。

可以看出，这些新一线城市对出口的依赖度只有20%以下。在经济层面，这些新一线城市的崛起，平衡了中国的区域经济发展。在政治层面，这些新一线城市的领导和企业，都会更自觉的维护中国国家利益，会更没有后顾之忧的支持反美。屁股决定脑袋，利益决定立场。加上这些年中国民众对美国越来越反感，对美国国运越来越看衰，因此中国这些年是越来越具有斗争性，而国内亲美派是一年不如一年。

岱岱再举个更清晰明了的历史，就是最近看的美剧《豺狼的日子》。这个美剧取材于一本同名的特工小说——1963年，因结束在阿尔及利亚的殖民统治，法国政治局势动荡不安；一名杀手，接受反对派的雇佣，着手刺杀法国总统戴高乐。一时间，欧洲上空阴云密布，全世界的情报机构都联合起来阻止这场谋杀，但除了「豺狼」这个代号，人们对这个杀手一无所知，连雇佣杀手的人，也对这个杀手一无所知。

这本小说是专业特工写的，介绍了很多秘而不宣的特工技术，自从出版42年来，一直是各国情报机构和恐怖组织的行动蓝本和培训教材——你可以靠这本小说自学成才当半个特工了。其实，看的时候我还很疑惑，戴高乐对法国有再造之恩啊，可以说是现代法国的国父，对法国作出了这么巨大的历史贡献，而戴高乐无非是让出了阿尔及利亚这个殖民地，在二战后殖民统治在全球瓦解的时代背景下，这个行为很正常很好理解啊，这群法国军人怎么会如此恨死戴高乐，不惜一次又一次的刺杀戴高乐？还是那句话，屁股决定脑袋，利益决定立场。

法国对阿尔及利亚有长达132年的殖民统治，和英国对印度那样的间接殖民不一样，法国对阿国是直接殖民，是让本土法国人去阿国落地生根的那种，而且法国用几代人时间，亲自下场从上到下建立法国制度，传播法国文化。这些要刺杀戴高乐的军人，很多父辈乃至祖辈，都是在阿尔及利亚出生的法国人，繁衍生息了几代人，他们从懂事时候开始，就把阿尔及利亚当法国的一个省，当自己的另一个故乡，而且他们在阿尔及利亚吃香的喝辣的，是拥有特权的。结果戴高乐让阿尔及利亚独立，让这帮驻阿军人回法国啃冷面包，于是这帮失去故乡失去特权的法国军人，就视戴高乐为十恶不赦的叛国之人，人人得而诛之，以悍不畏死的气魄发动了好几次刺杀行动，即使被抓捕被判死刑，这帮人都不怕牺牲前赴后继，贡献了特工激战的好几个名场面。戴高乐也贡献了几个历史名场面，他几次都要差点要被杀死，靠着和懂王那样逆天的运气才活了下来。

讲完这帮法国军人为什么会拼命刺杀戴高乐，就能更深刻理解屁股决定立场这句话了。现在中国各界的亲美反美，也一样的道理。我们要坚决继续搞国内大循环，培育新一线新二线城市，这些城市上去的干部，都不会是亲美派的后备军。我们要坚决搞国产芯片电车产业，搞这些不依赖美国市场技术的万亿级产业，这些产业的各方势力金主不会反对国家对美斗争的战略。

还有几个国际上的事。就是美联储迟迟不肯降息。美联储说，他们要评估懂王这轮关税战导致的通胀水平，然后再看是否降息。美联储说，这波关税传导的通胀，大概要到5月最后一星期才能看出来。而美债，在6月就要大规模到期了。真让美联储这么按部就班的来，那美债怕是要爆雷，懂王火烧眉毛了。所以，懂王很急，说要赶快下马美联储主席鲍威尔。

这还在博弈，民主党是撑鲍威尔的。看鲍威尔能否撑到5月末了。

日本的不配合，其实也正常，懂王开出的价码太离谱了。但日本如此不配合，也是意外之喜。这个石破茂还是挺滑头的，石破茂能撑久一点就好好了，只要撑到6月就好了。日本鬼子的骑墙心思其实很明显的。我们在柬埔寨搞了一个军事港口，美国说是中国海外军事基地要制裁封锁，柬埔寨说这个军事港口是国际通用的，不仅给中国用，任何国家都可以用。结果这个港口基地前不久刚建设好，日本军舰就不请自来，直接来这里停泊一会就走了，日本军方用这个行动来验证柬埔寨说的话是真的，不给美国找把柄。日本军舰去柬埔寨，看起来是示威我们，实际上理解了这个背景，就知道日本军方是帮我们打掩护，反而是讨好我们。所以说日本还是鬼精鬼精的。加上这次去越南，看联合声明也可以看出，越南不可能对懂王作出太大妥协了，至少会一定的偏向站队中国。那么日本越南谈不下来的懂王，更不好拿捏威胁中国让步了。中国这波真的很硬。

在贸易谈判这块，我们不用亲美派的人，用的是一个新人，李成钢。李成钢是贸易谈判专家，世纪之初我国入世谈判，他就作为谈判代表团成员参加了全部多边谈判。后来又担任商务部条约法律司副司长。这个部门处在世贸组织诉讼实战的最前线，负责处理各种吵架争端。后来到了2021年，开始出任常驻世界贸易组织代表，可以说对于世贸组织规则十分熟稔，也非常擅长多变谈判。这个李成钢是有名的鹰派干部，之前主攻欧盟东盟贸易谈判，美国私下把李成刚起了个外号叫Bulldog（斗牛犬），在英语语境里这不是侮辱性词汇，是形容一个人战斗力强，难缠之意。外国媒体分析李成刚的硕士学位是法学，这让他的逻辑思维更加缜密难缠，这次对美谈判我们也用了他。不要小看这个副部，他可能比正部的商务部部长王文涛，更重要。因为王文涛是长者妻族起家的，他在南昌的时候，每到周末必去上海走这个亲戚的门，后来在济南的时候积极靠拢核心，才上位商务部。王文涛不是谈判干部出身，也不够根正苗红，所以可能不能完全放心他去和美国谈判。
