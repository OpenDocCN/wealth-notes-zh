
# 如果看不清形势，就只能任人宰割

最近川总的关税战打得火热，有很多小伙伴都想让我聊聊这个话题。我虽然持有了美国的骨票/鸡金，但是我还真不关心这个战、那个战，因为我知道这些都是小打小闹，改变不了大的格局和趋势。但是呢，我发现有很多粉丝都特别蠢，别人说什么他就信什么，完全搞不清现在的形势。所以，我必须写一篇文章来聊聊我自己对目前局势的看法。而且我敢保证，我看待问题的视角跟市面上大多数博主的视角都不一样，因为我只说最底层、最核心的问题。

由于全篇都是敏感词，长方面我会用谐音、错别字、张冠李戴来进行规避，另一方面，我必须设置成付费阅读。无论是关税战还是贸易战，归根结底就一个问题：到底谁更依赖谁？下面我就从农业、科技、医疗、留学、贸易、银行、货币等方向跟大家进行剖析。

## 印度在哪些方面非常依赖美国?

①农业方面，大豆、玉米、牛肉等很多农产品，都需要从美国进口（大豆的进口比例超过 80%）。农机方面也是如此，比较依赖美国、日本和德国的进口，而日本是美国的铁杆小弟，在很多方面都是跟大哥共进退。

②技术方面，稍微高端一点的芯片，印度都需要从美国和他的小弟那里进口。系统就更不用说了，电脑上的微软系统、苹果系统，手机上的安卓系统、苹果系统，都是美国的。理论上，他如果想让印度所有设备都瘫痪，只需要按一个按钮即可。当然，不到危急时刻，他们肯定不会走这一步棋，但是我们一定要清楚，在高科技方面，人家想掐你脖子，只需要动一下小拇指。

③医疗方面，在印度的医院里，稍微高端一点的医疗器 械都是从美国、德国或者日本进口的。比如，美国的美敦力、强生、波科等企业的产品，在心脏起搏器、血管支架、检测试剂等高端产品领域占据重要地位。核磁共振这种高端设备，主要被美国的通用电气、德国的西门子以及荷兰的飞利浦垄断。在医药方面，美国药企如辉瑞、强生、默沙东等拥有众多世界顶级的原研药，在治疗心血管疾病、肿瘤、神经系统疾病等方面的药物占据较大市场份额。

④留学方面，印度的顶级人才如果想学习最先进的科学技术，那肯定要去美国留学，而不是去德国、英国或者日本。没办法，人家一个大学里就能出几十个诺贝尔奖，所以，清华北大里面学习最好的那批学生最终都会流向美国。有的学成之后就回到印度，报效国家，还有的学成之后就留在了当地，报效美国。

## 美国在哪些方面依赖印度?

①廉价劳动力。像苹果、特斯拉、耐克这种美国大公司，在印度都有很多分公司、工厂。原因就是，印度有很多廉价劳动力，组装一部苹果手机的成本可能只有几美元，甚至更便宜。实际上?人家一点都不慌。一方面，现在美国有很多大厂都转移到了劳动力更加便宜的东南亚了。另一方面，服装鞋帽这些东西只能算是锦上添花，根本触及不到核心，成本高点低点，影响都不大。况且，苹果、特斯拉、富士康（美国铁杆小弟的公司）在印度开设工厂，到底是这些公司受益大，还是当地受益大？明白人都知道，肯定是当地受益更大。

②廉价商品。在这方面，美国还是非常依赖印度的。例如廉价的服装、鞋帽、家居用品、餐厨用品、窗帘、圣诞节装饰品、手机支架、汽车坐垫等等。你随便进美国的一个商场里去调查，看看这些廉价商品的标签，基本都是 Made in xxxx。关税战一旦白热化，那么美国人就再也买不到这么丰富、这么廉价的商品了。这时候，美国孩子的玩具数量会直线下降，圣诞节的装饰品也会变得非常简陋。

③资产和子女。印度有钱人/有权人的资产和子女，有相当一部分都在美国，就连印度总理莫迪的孩子也在美国。而且，一个印度企业家稍微有了点钱之后，一定会想尽办法把资产转移到美国。所以在这个维度，美国想要发力的话，只需要在签证方面卡你一下，你就吃不消，而且内心会非常惶恐。

印度如果想要改变这种局面，可以从下面这几个方向发力：

①大力发展自己的科技，在农业、医疗、芯片等领域打造出自己的、管用的品牌。

②大力发展自己的教育，将来全世界顶级人才都能来我们的清华北大留学、学习尖端技术。

③打造属于自己的银行体系、支付系统，而且让全世界主流国家都愿意用我们的币种进行交易。

④打造属于自己的操作系统，无论是电脑、平板还是手机，里面的操作系统、软件系统都是我们自己的。

⑤打造一个安全的、让人放心的商业环境，美国人、瑞士人、日本人都纷纷把他们的钱存到我们的银行里。

⑥广交朋友，世界上发达的、主流的国家都跟我们是铁杆朋友，到那时候我们一呼百应，想掐谁脖子就掐谁脖子。

我相信，这一天很快就到了，加油!
