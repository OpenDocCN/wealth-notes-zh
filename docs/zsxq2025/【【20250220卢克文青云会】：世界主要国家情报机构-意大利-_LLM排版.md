
# 世界主要国家情报机构-意大利

意大利的情报工作历史悠久，最早可追溯至文艺复兴时期，当时主要为了支持意大利的统一。第一次世界大战期间，意大利的情报工作完成了重大演变，从军事情报工作转变为服务于国家利益的情报工作。1925年，意大利法西斯政权成立后，军事情报服务局进一步将情报工作制度化。二战期间，情报机构发挥了巨大作用，意大利在轴心国中的地位使其在情报工作上投入了较多资源。

二战后，意大利作为战败国对其情报机构进行了调整。1949年，意大利政府和美国政府成立了一个联合情报机构，这为后来意大利成为九眼联盟成员打下了基础。九眼联盟包括意大利、法国、德国等国家的情报机构，而五眼联盟则由英国、加拿大、澳大利亚、新西兰和美国组成。美国在构建盟友关系中，尤其在情报合作方面，形成了三个基本圈层：五眼联盟、九眼联盟和十四眼联盟。

1965年以后，意大利的情报机构转变为国防情报系统，旨在加强统一的军事国防情报。1977年，意大利政府将国防情报系统拆分为两个独立的实体，分别负责军事情报和民事情报。1997年，意大利成立了中央情报局（CII），部分业务并入意大利联合参谋部，而负责军事情报的部分继续独立运作。

2007年，意大利成立了共和国安全委员会，并设立了东部安全局、国际安全局和国家安全部。2021年，意大利推出了网络安全部门，以应对网络安全威胁。意大利的情报机构从传统安全领域向非传统安全领域扩张，参与打击网络犯罪等议题。

意大利的情报机构包括安全信息部、国防情报局、对外情报与安全局、国内情报与安全局等。安全信息部协调各个情报机构的工作，并做战略情报分析。国防情报局负责搜集、分析、传播国防和传统安全领域的情报。对外情报与安全局在全球范围内活动，监控损害或潜在损害意大利国家安全利益的情报。国内情报与安全局主要负责搜集意大利国内的情报，处理恐怖事件、反恐怖行为、有组织犯罪等。

意大利的情报工作投入较大，2023年全年的情报工作预算约40亿欧元，军事情报工作占国防预算的10%左右。意大利的情报机构在全球范围内活动，并与其他国家情报机构展开国际合作。意大利的情报工作在俄乌冲突中扮演了积极角色，提供了相当程度的情报资源支持乌克兰。
