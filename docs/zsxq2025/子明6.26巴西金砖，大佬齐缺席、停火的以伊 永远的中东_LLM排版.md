
# 巴西金砖，大佬齐缺席

发布时间：2025年6月26日

中俄两个安理会级大佬的缺席，让7月6日的巴西金砖峰会黯然失色。俄方的理由是巴西政府对逮捕令的立场，中方的理由是行程冲突，派二号代替，创下了自13年首次缺席的记录，也使得印度的莫迪成为了现场的最大咖。外媒对此的理由是巴西把莫迪作为主宾，导致兔子很生气，后果很严重。但这并不符合逻辑，中巴是全面战略伙伴，近期更是升级为命运共同体，高层互访频繁。印巴仅是战略伙伴，政治生态位与印度有竞争性，双方互动寥寥。而且，巴西对兔子有着极高的经济依赖，印巴贸易不足中巴贸易的一个零头，巴西脑子进水才会突然热脸去贴印度的屁股。此类重要活动选择不出席，一般都会找个冠冕堂皇的理由来掩盖真相，24年喀山金砖会议，巴西卢拉便以在家中摔倒脑出血为由缺席，以避免被仇视俄罗斯的拜登政府打击，此次中方的破例不出席，也必然有理由。

近年来，在中美对抗不断加剧的大背景下，中方长期积极出席金砖峰会，并积极为金砖扩容招揽会员，譬如伊朗，将金砖作为一个对抗西方的平台，借此向世界表达自己没有被美国孤立。推测此次中方不再积极的主要原因，倒不是不好意思见伊朗，而是跟俄罗斯非常相似，随着美国政权的更迭，调整对外的主要叙事逻辑，从反美统一战线转变成反法西斯统一战线。中俄美曾经是二战盟友。随着反法西斯同盟成为了中俄的叙事主线，几位棋手在雅尔塔框架内博弈更为舒服。类似于今年我们对气候环保类组织的参与度骤降，在其他更重要的事情面前，民主党时期主线剧情金砖也可以适度降格，毕竟，中俄两位安理会大佬真正的主场，是9月3日的大阅兵。

普京将不会出席金砖峰会，由于国际刑事法院发出了悬而未决的逮捕令，且巴西政府无法采取明确的立场。国际刑事法院于2023年对普京发出逮捕令，指控他犯有战争罪。巴西一位知情人士透露，东大将缺席在里约热内卢举行的金砖国家领导人会晤，这是他上任以来首次缺席此次峰会。据一位不愿透露姓名的政府官员透露，东大已告知巴西，L将代表出席下个月的峰会。L此前曾于2023年在印度举行的二十国集团峰会上代替其领导人出席峰会。他的缺席将与印度总理纳伦德拉，莫迪的出席形成鲜明对比。莫迪的责宾身份或许也在影响他的考量。
