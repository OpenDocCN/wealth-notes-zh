
# 怎样对抗“这要命的孤独感”？

今天我们聊一个很多人关注的问题，孤独。但我们不是要从心理层面出发，而是换个角度，看看一个社会是怎样面向孤独问题开战的。我们要说的案例是韩国。

2025年4月，韩国首尔将开通一个特别的热线电话，叫告别孤独120热线。假如市民觉得特别孤独，就可以拨打这个热线，然后就有心理咨询人员陪你聊天，假如有生活方面的实际困难，他们也会帮着联络相关的社会机构。

尽管韩国人没有像日本人那样发明出“孤独死”这样的专有名词，但并不代表这个问题在韩国就不严重。他们也一直在试图解决这个问题。而从他们的做法中，我们也能看到一个社会是怎么针对孤独问题作出反击的。毕竟，孤独是个普遍的社会现象，它的解决方案值得了解。

首先，韩国的孤独问题有多严重？从绝对数上看，它目前的程度确实远比不上日本。2024年上半年，日本有3.7万名独居者被发现死在家中，而韩国的数量大概是日本的5%。但是，它有两个特别值得留意的地方。一是，从孤独死的构成上看，韩国中老年男性占比更高。根据推测，这可能与失业裁员之类的经济压力，或者离婚有关。二是，韩国的自杀率这几年正在飙升。

根据韩国统计厅今年2月发布的数据，韩国2024年有14439人自杀，平均一天40人，这也是韩国自2011年以来的最高纪录。注意，这只是自杀死亡的人数，救活的没死成的还不算。而且韩国学者也越来越关注自杀问题，并且做了不少相关的研究。

比如，有人指出，韩国的自杀人数与明星自杀事件高度正相关。假如有明星去世，那么这段时间的自杀率会明显上升，而且会持续7到8周左右。心理学上管这个现象叫维特效应，这个词的来源是1744年歌德发表的小说《少年维特的烦恼》，里面的男主角维特因为失恋而自杀，后来很多读者居然模仿维特。维特效应在很多地区都存在，只不过韩国明星的自杀率相对较高，因此维特效应造成的破坏也就更严重。

当然，明星自杀只是诱因，而普通人自杀的根本原因，从前到后分别是孤独感、社会不平等加剧，以及经济衰退等。其中孤独感排在第一位。

其实，关于什么是孤独，你很难给它一个准确的界定。因为孤独不像愤怒、悲伤、兴奋那样，有特别明显的特征。你说不清孤独的人有什么特征，也说不清不孤独的人有什么特征。这也是为什么严格来说，孤独并不是一种单一的情感，而是一整套情感集群。其中包含怨恨、悲伤、羞耻、嫉妒、自怜等等很多情绪。假如非要概括，借用人民大学哲学院刘玮老师的话说，孤独是一种无意义感和脱节感。觉得做什么事儿都没劲，没意义，或者感觉自己跟周围脱节了。

当然，概念的问题可以先放一边。关键是，当孤独蔓延成一种普遍情绪，而且造成实际问题的时候，应该怎么办？

我们还是以韩国为例，这些年他们陆续使出了不少招数。而2024年10月推出的“无孤独的首尔”计划，堪称是这些招数的集大成者，几乎动用了所有能动用的工具。整个计划预计耗时五年，投资4513亿韩元，相当于23.3亿元人民币，是韩国迄今为止规模最大，投入最多的对抗孤独计划之一。

那么，这个计划都包括些什么呢？有三个要点，分别是，全覆盖、游戏化、个性化。

第一，全覆盖。也就是，调动尽可能多的资源，去覆盖首尔的孤独者。让每个人的生活状态都处在视线范围内。

比如，首尔政府跟外卖平台合作，毕竟，即使再孤独的人，也免不了点外卖。因此，外卖员算得上最理想的观察员，假如这期间发现某个人存在孤独风险，就可以上报给政府。后续政府就会想办法跟进，要么远程咨询，要么上门走访。

再比如，为了及时监测，韩国政府还制定了一套专门的评估标准。假如一个人半年以上不出门，且没有社交活动，就会被定性为隐居。这部分隐居者就是重点监测的对象。2021年，韩国19岁到34岁青年中，大约有5%的人处于隐居状态。

再比如，建立就近支援点。当地会利用水电等46种生活数据，寻找隐居家庭，然后把他们附近的便利店、洗衣店，作为隐居家庭的支援申请点。同时，首尔还专门成立了一类社区空间，叫做“心灵便利店”。任何人只要感到孤独，就可以去那里吃一碗很便宜的拉面，还能向店里的咨询人员求助。另外，他们还跟网络平台合作，通过发放堂食优惠券等方式，鼓励人们去餐厅就餐。

再比如，2024年，韩国政府就宣布，从2025年开始为20岁到34岁的人提供每半年一次的精神健康检查，后续还会增加心理健康中心数量，并加强自杀预防教育。

假如前面这些投入奏效，不仅可以救回很多生命，还能挽回巨大的经济损失。2023年，韩国有40.2万身体健康年轻人拒绝找工作。韩国因此造成的经济损失是巨大的。

换句话说，尽管韩国为了对抗孤独花了大笔经费，但这跟重度孤独造成的损失相比，依然是九牛一毛。

但是，孤独在很多时候，是人的主动选择。因此，只设立援助机构是不够的，关键是，得推动人们展开行动，主动走出家门。

怎么办？这就要说到“无孤独的首尔”计划的第二个设计要点，游戏化。也就是，把孤独援助设计得像升级打怪的游戏一样，一步步引导人参与。

针对中老年独居者，他们计划推出“中老年信息整合频道”。因为随着老龄化加剧，很多韩国的老年人会在晚年重新开始兼职，但他们的信息获取能力有限。因此，首尔市政府就准备推出一个专门的平台，帮老年人搜集各种兼职和临时岗位信息，以及帮助申领养老金、医疗补贴、住房支持等各种福利。

针对有孤独症的年轻人，他们会提供定制化的治疗方案。比如“园艺方”，也就是在花园或者森林举办疗愈活动。还有“自立处方”，是通过与社区合作，支持年轻人实现自立。

未来，首尔还准备推出“首尔青年伸展中心”，主要是给年轻人提供职业指导和实习机会，帮助年轻人提升就业能力。

好，刚才咱们说的是，在对抗孤独方面，首尔在社会层面做出的设计。这些设计的具体效果现在还不确定，有待后续观察。但至少目前看，这些举动已经把对抗孤独从一个人的战争，上升到了整个社会层面的议题，这或许是个积极的进展。

最后，回到我们自己。2024年底，中国老龄科学研究中心发布了《中国老龄发展报告2024——中国老年人心理健康状况》蓝皮书。好消息是，与2015年的数据相比，过去的将近十年里中国老年人的孤独感大幅下降，而且幸福感和获得感明显增强。

但挑战在于，依然有四分之一的老年人，有或轻或重的孤独感。同时，专家团队也给出了生活方面的应对方法。

第一，多使用短视频软件，而且最好能自己发一点短视频。目前，全国有36.6%的老年人使用智能手机，但因为统计方式因素，实际数据可能还要更高。中国老龄科学研究中心老龄社会和文化研究所所长李晶说，短视频使用与老年人心理健康各项指标，有积极关系。而且自己生成视频，要比被动收看视频更有益于心理健康。

第二，多运动。目前咱们国内经常锻炼身体的老人占38.3%，从不锻炼的是占30.6%，而经常锻炼身体的老人，要比从不锻炼的老人，孤独感低上11.3%。

第三，多参与家族活动或者邻里社交。这点不难理解，毕竟，孤独的根源是社交减少，而增强社交是缓解孤独最直接的方式。

换句话说，人总得把注意力投向一些“值得”的地方，要么获得乐趣、要么锻炼身体、要么增进关系，这或许是对抗孤独感最有效的方法。
