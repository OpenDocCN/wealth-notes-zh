
# 伊核开启谈判，关税告一段落

伊朗与美国的谈判于今日启动。此前，伊朗曾退让一步，同意由中间国家参与谈判，而非直接与美国对话，但美国并未接受中俄作为中间人的提议，而是选择阿曼作为中间人。在阿曼，美国与伊朗代表团分别处于两个房间，阿曼代表则穿梭其间传递信息。

俄罗斯也在场外参与，因为美国总统的中东特使当日会见了普京。此前，普京与美国总统进行了长达五个多小时的会谈，讨论俄美关系正常化。两国会谈的议题不涉及乌克兰问题，暗示着讨论的重点是中东问题、海外资产乃至中国。

普京之所以帮助美国施压伊朗谈判，是因为美国提供了重大让步，包括解封被冻结的俄罗斯海外资产，并计划重开大使馆。俄罗斯中央银行行长表示，截至2025年初，约半数2022年被冻结的俄罗斯境外资产已获解封。

一些自媒体分析认为，普京反对美国强硬解决伊核问题，因为这将导致原油价格上涨，而普京更希望美国页岩油产业受挫。这种观点认为，普京希望通过与沙特增产压低油价，将油价长期维持在低于美国页岩油成本的水平，从而让美国页岩油产业倒闭。

奥巴马时代曾推行类似的策略，通过环保政策限制页岩油发展，减少对中东原油的依赖。然而，特朗普上台后，承诺放宽环保限制，大力推动石油产业，计划将美国从石油进口国转变为石油出口国。这对美国国内石油金主有利，也使得美国与沙特、普京在原油价格上涨时具有共同利益。

特朗普现阶段的政策与奥巴马时代截然不同，讨论页岩油战略已无实际意义。普京在叙利亚的军事基地仍受制于美国，因此在中东问题上，大部分与特朗普合作。普京本质上欢迎美国解决伊朗问题，但主要原因是美国提供了重大让步，这是俄美关系正常化的交换之一。

俄美两国会谈长达五个多小时，未涉及乌克兰问题，而是聚焦中东问题、海外资产及中国。中国近期出访东南亚，包括越南、马来西亚和柬埔寨，这是从地缘和经贸角度考虑的。经贸方面，贸易战背景下，东南亚国家成为中国重要的出口转口国。地缘方面，则是为了"梧桐"战略。

中国此次出访东南亚最反常之处在于，出访前专门召开了周边外交工作大会。上次召开此类会议是在2013年，当时习近平总书记提出了亲诚惠容周边外交理念，强调"让命运共同体意识在周边国家落地生根"。2013年10月召开的会议是为了呼应"一带一路"战略，当时中国首访哈萨克斯坦，点题中亚。2016年5月，中国首访中东后出访塞尔维亚，点题中东欧洲。2025年4月，中国出访东南亚，点题东南亚。

中国此次出访东南亚有两个看点：一是柬埔寨的运河问题，二是越南对美国的依赖。越南领导人苏林对内强硬，对美软弱，中国已对越南涨电价，并卡住北部高铁谈判。越南军方在南海上与美军有暧昧关系，中国此次出访旨在解决南海和贸易问题，实质上是要让苏林认识到，得罪中国比得罪美国更严重。

马来西亚一直尊重中国，但近年来也有压制中国经济的迹象，去年曾反制中国纺织品和电器出口。中国此次出访马来西亚，也相当于与其他东盟国家会面。中国未出访缅甸和泰国，这本身也是一种态度。缅甸和泰国在电诈问题上表现不佳，且不太配合中国。

中国此次出访东南亚的共同点是台湾问题。特朗普二次变卦后，中美脱钩，甚至可能引发中东战火，东亚成为全球净土。中国重点耕耘东南亚，东盟已连续4年成为中国第一大贸易伙伴。关税方面，特朗普想等到中国电话是不可能的，除非他认输或强硬解决伊核问题。

特朗普暂缓关税90天，但这90天实际上只有60天，因为6月美国将面临大量国债到期问题。美国股市若在6月前与中国的谈判未果，将面临爆炸性危机。中国外贸企业可利用这段时间进行转口贸易。本次关税战打击最大的是金融领域，胜负手在于中国能否顶住外部冲击，稳住股市和汇率。

美国只有两招尚未使用：逼迫日元加息引发东亚金融危机，以及中概股退市。中概股退市策略五年前效果巨大，但如今效果有限，因为中国已有大量中概股在港美股上市，或做好了退市准备。特朗普的关税博弈前景悲观，因为只要中国内部稳定，特朗普就失去了最大的牌。

中国此次关税战的策略是"抻"，抻得越久，赚得越多。抻到4月29日，特朗普要么认输，要么解决伊核问题，要么丢脸。抻到6月，美国可能爆发经济危机，中国若能成功抗压并拉起牛市，将吸引全球资金避险，完成东升西落。这场博弈目前告一段落，但后续将高潮迭起。
