
# 长辈又买保健品，怎么让他们心甘情愿听你劝？

今天是国庆假期的第二天，我们从一个有点让人头疼的场景说起。假设，你长假回家，刚进门就看到茶几上摆着一堆大盒子。仔细一看，有什么灵芝孢子粉、人参胶囊、辅酶 Q10 等等。好家伙，爸妈又买保健品了，而且价格不低。

你想劝，又不知道怎么开口。你有你的道理，长辈有长辈的坚持，他们还会搬出一堆“科学研究”来反驳你。假如什么都不说，看着家里的钱就这么打水漂，心里又着急。

关于这个问题，有位老师特别有发言权：卓克老师。

今天，我们就梳理出了一套方法，帮助你用科学依据，来跟长辈聊保健品这个事。

先说卓克老师的基本观点，很多保健品的功效都被夸大了，我们花的钱，未必能取得相应的效果。

当然，要想彻底改变长辈的观念并不容易。假如能劝动长辈理性对待保健品，当然最好。假如劝不动，至少要选安全的。安全，这是底线。

那么，怎么做安全排查呢？很简单，找一个标志。

你可以把家里所有的保健品都拿出来，看看有没有一个蓝帽标志。这个标志长什么样呢？就是一个蓝色的帽子形状，里面写着“保健食品”四个字。

注意，所有合法的保健食品都必须有这个蓝帽认证。假如没有，要么属于未经批准的非法产品，要么属于普通食品冒充保健品。

这里涉及一个很多人不知道的事实，目前市场上存在大量普通食品冒充保健品销售的情况。很多你以为的保健品，可能只是普通食品。

比如花青素，听起来很像保健品吧？其实它只是“新食品原料”。

什么叫新食品原料？简单说，就是一些可以添加到普通食物里的东西。白糖、淀粉、食盐、食用油这些属于传统食品原料，不需要特别审批。但有些建议从来没有在传统饮食中使用过，现在非得要用，那就得审核一下它的安全性。等过关了以后，就可以作为食物原料使用了。

这些新食品原料只做安全性审核，最多吃了不会损坏健康，但并没有经过任何功效认证。《保健食品标注警示用语指南》中还专门规定，普通食品不得暗示保健功能。

但起码，食品还是安全的。更要命的是，还有一些未经批准的非法产品，看上去特别像天然保健品，但实际上里面可能擅自添加了激素或者药物，对人体可能会造成危害。

换句话说，资质认证，这是保健品的第一条红线。所有没有蓝帽标志的保健品，你都可以帮长辈清理出来，并且把危害讲给他们听。

好，这是第一个问题。怎么做安全排查？找蓝帽标志，没有的坚决清理。

排查完安全问题之后，接下来就可以看看，要怎么帮长辈理性地理解保健品的功效。

具体怎么做？假如仅仅是说“这些都无用”，恐怕作用不大。更好的方法是，通过剂量对比，帮长辈准确认识功效。没错，不能抛开剂量谈功效。

比如灵芝。按照宣传，灵芝对抗肿瘤功效突出，因为灵芝中含有的各种灵芝酸能抗癌、抑制肿瘤生长。这个结果是怎么来的呢？是根据相关研究，服用灵芝酸的患癌小鼠，肿瘤有明显缩小。你看，听着是不是挺有效的？

但是，注意了，重点来了。做实验的时候，灵芝酸的服用浓度是非常高的。

我们再来看日常保健品中的量。破壁灵芝孢子粉一次口服的量通常是 2 克左右，服用后人体内能检测到的灵芝酸血液浓度是极低的。这个浓度，和小鼠实验中用到的浓度，前后可能差了几十万倍。要想达到实验效果，可能得一口气吃掉几十万粒灵芝孢子粉胶囊。

而且，哪怕用实验剂量让肿瘤有所缩小，这也不是什么前所未有的大成果。很早之前，医院里用 X 光照射肿瘤就足以有类似的效果了。

再比如，人参。人参的功能主要集中在于人参皂苷上。人参皂苷虽然有 40 多种，但因为结构高度类似，所以具有统一的特性，高溶解性，低渗透性。

简单说，就是它们可以很好地溶于水，但跨膜能力差，无法被胃肠道吸收，无法进入血液。这就意味着人参皂苷的吸收效率并不高，用科学术语来说，就是生物利用度低，也就在 1%到 2%之间。

按照 2014 年卫计委发布的规定，每天食用干人参不得超过 3 克。那么 3 克脱水的干人参所含的皂苷大概有多少呢？一般也就 60 毫克。由于生物利用度最多 2%，所以每天进入血液的人参皂苷顶多 1.2 毫克。按成年人体重 60 公斤核算，剂量就是每公斤体重 0.02 毫克。

而那些宣称人参皂苷有疗效的动物实验里，皂苷剂量是远远超过这个数字的。比如，“记忆受损后的认知改善”实验，实验剂量是每公斤体重 25 到 100 毫克。“炎症因子降低”实验，实验剂量是每公斤体重 20 毫克。动物实验的剂量，已经达到了人服用保健品量的成百上千倍。在这么高的浓度下，也只是观察到了一些改善的迹象。

假如想把这些动物实验的效果放大到人体，那么每天我们至少要吃 3 公斤脱水的干参，持续几周时间，才能看到效果。要知道，那可是脱水的干参，吃完之后你再一喝水，可能马上就得急救去了。

你看，离开剂量谈功效，都是站不住脚的。

但有些长辈听到这里可能会想，哪怕有效剂量少，总比没有强啊。但这其实也不成立。为什么？因为很多元素，人体根本就不需要额外补充。

比如非常火的辅酶 Q10。人体几乎都是可以自产的。为什么？因为这个东西太重要了，稍稍缺乏就会带来致命后果。因此，人类的身体早就进化到，能够自己产生足够的辅酶 Q10。

一个成年人体内存储的辅酶 Q10 在 1 到 2 克之间，每天会消耗 0.25 到 0.5 克，消耗的部分 99%都会由身体自己合成回来。

再比如，虾青素，这是比较公认的具有抗氧化作用的东西。虾青素到底能不能抗氧化呢？能。但这个“能”是在多大程度上的“能”呢？这么说吧，假如人体自身制造的抗氧化物质能提供的防御值是 10 万的话，那么每天按时服用虾青素保健品，提供的防御值大约就是 5。

你可能会想，增加 5 个点的防御值也不错啊。但是要知道，有一个办法能增加 250 点防御值，就是每天保持 7—9 小时的充足睡眠。这个如此平常的动作，抗氧化功效已经是服用虾青素保健品的 50 倍了。

借用卓克老师的话说，生物进化是很伟大的，人体本身非常精密。从饮食、运动、睡眠、情绪入手，比很多保健品都有效得多。

把这些讲给长辈听，也许他们以后能少被所谓的“科学数据”唬住，能多一点理性的判断。

但是，假如了解了这些之后长辈还是想吃保健品，怎么办？那就只能吃了。毕竟转变观念并不是容易的事。

这时，我们就要注意，尽量帮长辈选安全合法的产品。

说白了，就是花小钱买个心理安慰，总比花大钱买个身体伤害要强。

关于保健品这个话题，咱们先说到这儿。

到这里，我们一共说了三个重点。第一，怎么做安全排查？找蓝帽标志，没有的坚决清理。第二，怎么理性对待功效？通过剂量对比来准确认识。第三，假如长辈非要买，怎么办？选择安全合法，且价格相对便宜的产品。
