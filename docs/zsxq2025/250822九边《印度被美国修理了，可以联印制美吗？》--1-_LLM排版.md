
# 印度被美国修理了，可以联印制美吗？

印度和美国的关系一直比较复杂。自从印度建国以来，其最主要的盟友一直是俄罗斯。印度为了制衡中国，引入了俄罗斯作为强大的外援。俄罗斯拥有丰富的油气资源，而印度极度缺油，因此两国关系越来越近。

在冷战期间，印度名义上不结盟，但实际上一直偏向苏联。苏联对印度提供了大量的工业项目和联合国否决权的支持。1971年第三次印巴战争时，美国支持巴基斯坦，而印度在苏联的支持下与美国硬刚，导致美印关系跌至冰点。

冷战结束后，印度开始转向市场经济，并尝试与美国改善关系。1991年，印度经济几乎破产，开始学习中国的沿海贸易、大基建和房地产模式。印美关系整体趋势越来越好，但冲突不断。例如，1998年印度进行核试验时，美国对其进行制裁，但后来印度通过妥协获得了美国的原谅。

现在，印度极度依赖美国，而美国为了制衡中国，也给了印度大量的技术和市场份额。两国关系被称为“全面全球战略伙伴关系”。然而，美印之间的共同点比中美之间的共同点少得多，印度在许多方面让美国感到烦恼。

印度的高关税、人权问题、宗教冲突以及政策制定等方面都让美国感到困扰。印度人爱占便宜的毛病也让美国感到厌烦。在俄乌战争和巴以冲突中，印度表现出与西方国家不同的立场，这让美国感到失望。

美国这次对印度征收25%的关税，不仅是因为印度购买俄罗斯石油，也是对其之前各种行为的惩罚。印度对美国的大部分主张都不配合，但美国又不得不依赖印度来制衡中国。印度虽然心眼多，但心里很清楚，自己离不开西方。

因此，印度和美国闹翻后，中国并不能与印度联合起来。印度会在所有事情上都唱反调，以彰显自己的非主流气质。中国与印度的关系应该是潜在的对手和假想敌。

从这件事可以看出，美国用关税大棒折腾所有人，但本质上还是因为美国手里有“消费能力”这张牌。中国的真正盟友应该是中国巨量人口催生的购买力，这将是中国强大的对外筹码。
