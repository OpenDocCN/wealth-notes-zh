
# 学习笔记，2025.3.4

1. 以后，在互联网上了解真实信息会越来越难，自媒体在为了流量造假，AI 利用自媒体的假内容做回答，创造更多假内容。是真实信息更少了吗？不是，是更多的假信息把真实信息稀释了。比如，自媒体平台上都有不少辟谣号，但是比起追流量的造假号，还是太少了。AI 时代，一定要培养自己去找到源头真实信息的能力。

2. 人们很多时候争论的都是观点，不是事实，因为底层观念不一样，所以都认为自己的观点是事实。不要把精力浪费在争论这种没有结果的事情上面，多去关注那些真实、对自己切实有帮助的东西。

3. 好多抱怨自己这一代没有上一代赚钱容易的人，都是马后炮，只看到了上一代成功的案例，没看到那些更多的失败案例。很多人身处时代当中的时候，也是看不到机会，只有十年后，一切尘埃落定，才知道当初是机会还是坑。

4. 那些总是喜欢否定你、打击你的人，最好早点远离。他就是看不得你好，通过打压你来获得优越感，和这种人在一起时间长了，会变得没自信，会错过很多可以改变自己的机会。

5. 如果想要获得成功，不管是哪一方面的成功，事业上的、精神上的、关系上的……你都得首先做到逆自己的人性做事。因为成功始终是少数，大多数人都在顺人性，都在被别人的概念和观点左右，你得首先从这种怪圈里脱离出来，才有可能成功。

6. 想要做好一件事，需要长期的心力支持，总想着今天做明天就看到结果，就会越来越焦躁，最终什么都做不好。不知道自己真正热爱什么、想要长期坚持什么的时候，可以先培养自己阅读和运动的习惯，让自己有一个能够长期做事的基础。

7. 没有实力，面子就像毒药，会让人变得盲目，分不清轻重缓急。但是，越是没有实力的人，越会花大量时间精力去维护自己的面子，因为那是他们觉得自己唯一有的东西、唯一可以掌握的东西。

8. 很多时候的争吵，都是内行和外行在吵，因为看问题的角度不一样。内行看，都是透明的，知道真正的问题出在哪。外行看，哪都是问题，但可能和真正的问题完全无关。

9. 大多数人都在说要逃离原生家庭，但其中大部分，都抵抗不过多年被影响出来的思维惯性，最终活成了父母的样子。只有少数人，能够从过去的影响里走出来，不断去观察、革新自己过去的观念，活出一种新的人生。

10. 很多人只是说要改变，但实际上没有那么想改变，或者说，他内心其实觉得，现在还可以，没有太大必要去改变。比如阅读，短时间内也看不出太大差距来，于是觉得，读也行，不读也行。但是看到别人通过阅读取得巨大成就的时候，又会在短时间内有一种迫切需要阅读的念头。躺也躺不平，冲也冲不动，反复在努力不努力之间摇摆，一辈子也就这么过去了，到最后来一个“平平淡淡才是真”。

当下的每一个果，都是多年前种下的因，改变不是一朝一夕的事，如果想要 10 年后有某个果，现在就该开始在因上努力了。
