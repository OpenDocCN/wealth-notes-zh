
# 虚空的魔法部
## 发布时间：2025年8月11日

自2022年以来的三年，魔法部的FM都是高危职业，原本要接FM的LYC和刚担任FM的QG都提前退休，都已经告知西方，准备接FM的LJC，在上个月底从北非考察后被带走问话，前几天原IDCPC的老部长去世，现任部长LJC未能出现，内部就都知道这次问题不小。鹰派的LYC不能接任，是因为其跟俄罗斯关系密切，鸽派的QG不能留任，是因为其跟美国关系密切，鸽派的LJC未能接任，是因为其跟拜登关系密切，2024年的东美破冰与利益交换，都是其一手操纵，但很遗憾的是，现在是特朗普上台，要搞新交易，所以今年3月会议之后，伴随着鸽派大规模的落马和鹰派的全面崛起，同样鸽派的LJC就开始了靠边站，之前美方被告知的FM接班人不需要再联系，由经贸负责人的H来负责跟特朗普团队谈判。

当然，LJC最终大概率不会被公开处刑，而是跟前面的LYC和QG类似，提前退二线，一方面是因为清水衙门，腐败不会太严重，挨收拾只是因为政治站队的问题，另一方面则是东大的传统，对于外交官员的站队错误并不会严厉打击。

你可以想象你就是一家房地产公司，你有老个很牛掰的政府公关部门经理，平时在单位耀武扬威的，突然有一天市委市政府大换血，原来老领导的敌对派系上台了，那么这个政府公关部门还可能像以往那样牛逼么。军队更类似于你们单位的施工单位，只要你们自己内部一二把手一换，施工单位肯定也得换，后面查到这个经理贪污了，不能容忍，把他辞了。政府公关部门，涉及到和政府的很多秘密利益交换，所以一般情况下不会往死里收拾，施工单位都是自己内部的利益交换，一般情况下都是往死里收拾。
