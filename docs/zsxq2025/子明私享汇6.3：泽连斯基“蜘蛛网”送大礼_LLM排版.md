
# 子明私享汇6.3：泽连斯基“蜘蛛网”送大礼

泽连斯基“蜘蛛网”行动的大背景，是特朗普与普京正在密谋推进元首会晤，试图绕过传统的利益集团来完成俄乌的停火。参考23年初的Wanderlust balloons，全球最具权势的两个人也在谋划当年的会晤，以实现全球最大的两个经济体之间的停火。就像白宫首次签署美乌矿产协议前，特朗普被泽连斯基偷袭搞得不欢而散，本次蜘蛛网行动，特朗普大概率也是不知情的，是在一小撮内部人的纵容和支持下，小泽对老同志不讲武德的搞偷袭，接下来，就看特朗普敢不敢像当年东大那样收拾这帮鹰派“大佬”了。

俄罗斯最重要的战略威慑武器被攻击，动了俄罗斯这个核大国的逆鳞，会导致普京短期内无法有任何实质性的让步，俄乌今年的停火基本就没戏了，即使特朗普与普京随后会晤，主要工作也只能是恢复两国之间的信任，解释这次偷袭美方真的不知情，以降低双方骤然升级的安全警戒度。

对于中国来说，是巨大的利好，原本随着特朗普上台，逐步对我们降温的俄罗斯，随着战火的升温将会更加依靠我们，经贸与投资领域更方面的合作会因此而加速。上一周还指责中国违反协议，准备制裁的特朗普，本周也会立刻变脸称呼我们为他的老朋友，并积极寻求拨通北京的电话，到时候决定接不接的主动权，握在我们手里。

乌方透露对俄实施特别行动摧毁41架俄战略轰炸机。总台记者6月1日获悉，乌克兰安全局内部人士消息称，乌克兰代号“蛛网”的特别行动历时18个月的策划和实施，成功摧毁41架俄军战略轰炸机。此次行动由泽连斯基亲自协调，安全局局长马柳克率队直接实施。

现场视频|俄乌第二轮直接谈判在土耳其举行。当地时间6月2日，俄罗斯与乌克兰第二轮直接谈判在土耳其伊斯坦布尔的契拉昂宫举行。

乌对俄发动大规模无人机特种作战美官员：白宫未提前获悉。当地时间6月1日，央视记者获悉，据一位未透露姓名的美国政府官员消息，特朗普政府并未提前获悉乌克兰会在本周末对俄罗斯发起大规模袭击行动。

五角大楼称中国间谍气球飞越美国上空。美国官员周四(2023年2月2日)表示，一颗中国“间谍”气球已在美国上空盘旋数日。在美国国务卿安东尼·布林肯计划访问北京前几天出现，这将是一次无耻的举动。
