
# 普通人职场危机应对手册

普通人，如何在思想上做好准备，应对潜在的职场危机？

我认为，第一要在思想上做充足的准备：非体制内行业，时刻做好被裁的心理预期。我每到年底，就会格外关心公司的裁员计划。

第二，对自己的职场能力有一个较为清晰的判断。这里分两步：

a. 假如你是一个普通211的毕业生，应该能意识到，即使考察你擅长的学科能力，你在全国人群中的位次，大概在10%～40%。这个位次，本身就意味着：在以学科为考察核心的资源分配中，你已然位于了牛马生态位。要想摆脱牛马生态位，要么家族有积累，要么自己去积累。

b. 学科能力之外，就是职场能力。本人在职场这几年的认识是——老板和领导，绝对不会主动干掉任何一个在他们当时的决策中，价值>成本的人。绝对不会。即：老板和领导，在职场运行中，从始至终，于当时的客观环境，都处在决策的理性态。没有任何例外。

所以，老板和领导对你的看法，百分之百就是你职场能力的客观评价。当然，这个能力，不能限于普通所谓搬砖的能力。

我之前认识一个做信息化的同行，这哥们有个独门绝活，是——回款。他有一次喝酒喝高了跟我讲，秘诀是——不要脸。我后来照着做，效果的确是很可以。

那再来看自己的职场能力：就要清晰地认识到——所谓的兢兢业业、任劳任怨，很可能只是公司整体事业中，占比非常非常小非常小的一环。而在你心中：以为很重要而已。

好了，现在你我都已经清楚的了：

咱们那点学科能力不够用。

咱们那点职场搬砖能力很细枝末节。

那就意味着：

咱们混得很差，再正常不过。

公司一裁员就裁咱，很科学。所以：

为了应对潜在的职场危机——咱们要

要么：

把自己的职场能力，变得对决定你职场状态的人，更有用一点。

记得，一定是决定你职场状态的人，要觉得有用。不是你觉得。

要么平时做点打算，量入为出，攒点积蓄。

就是渤海小吏经常讲的——准备好资源冗余。

要么：

具有快速适应社会地位、经济回报更差工种的能力。前段时间，带家人去我同学喝酒。

同学被裁约半年，遇到了几次体力活的工 作机会，没有去做。

回去路上，我老婆表扬我说：我遇到类似情况，肯定是第一时间，去上班了，不管是快递、外卖、保洁，还是其它。

原因是，她认为：

当我意识到，自己在思维上，先天能力不足后，就不必在什么“战略上”谋划了。

而是迅速的投入到具体的事情中，赚具体的一毛两毛。而不是，去思考，去规划，去学习提升。

我们这个智力水平的人，仅仅只适合，在干具体的事情中，基于经验，提高一点认识。

回想一下，六年前，开始硬着头皮干销售，各种痛苦，身边的同事，三个月换一波，六个月换一波......

慢慢做下去，还认识了行业里很多朋友，帮扶了我很多。

关键就是，我从那时认识大家后，意识到了——职场和之前想象的不太一样，要去适应它。

明白了，有一份公司给交社保的工作，属于我们这个段位的人，的职场高配鸟。

所以，要好好上班，变得对决定我们职场状态的人，更有用一点。

目前的职场，有社保的工作，并不多。

而我就有。
