
# 学习笔记，2025.8.25

1. 维也纳动不动就唱我们的民族歌曲，《好一朵美丽的茉莉花》，茉莉出自梵文，时间一长，就成了纯粹的中国文化。世上生命力旺盛的语言，从来都兼容并包；一成不变，就是死语言。为什么名著隔个三五年就重新翻译一遍，根源就在这里，一代更比一代强，根源就是词汇量更丰富了，文化解读更深邃了，最初的译本可能只纯粹的语言翻译。我签过天才翻译家李继宏的《傲慢与偏见》，你知道他怎么翻译这些名著吗？他会走进名著，去作者的生活地，去故事的发生地，去实地采风，去感受。

2. 我们自己都不喜欢的东西，别人咋可能喜欢呢？若是我们自己喜欢，就说明辐射力足够深，因为我们也代表了一定的消费水平，我们的收入，我们的眼界，若是我们都觉得很有营养，那么与我们实力相当的朋友试样也会有如此的感慨。写作的本质就是价值，你能提供给我们什么？例如你完全可以把李宗仁当李云龙去写，整个抖音曾经一度被那个“再电汤恩伯”刷屏了。
