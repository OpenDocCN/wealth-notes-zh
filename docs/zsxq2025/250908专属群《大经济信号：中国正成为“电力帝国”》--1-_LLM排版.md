
# 大经济信号：中国正成为“电力帝国”

中国全社会用电量在2025年7月达到1.02万亿千瓦时，首次突破单月万亿千瓦时大关。这一数据相当于东盟国家全年的用电量，英国《金融时报》称中国或将成为人类历史上第一个“电力帝国”。纵向对比，7月用电量较十年前翻番，这种指数级增长在世界能源史上前所未有。

2024年中国全年发电量突破10万亿千瓦时，占全球总发电量的1/3。由于发电过多，电网消纳能力不足，还出现了“弃电”现象。全国新能源消纳监测预警中心数据显示，2025年上半年，全国太阳能弃电率升至6.6%，风电弃电率达5.7%。中国已事实上成为全球最大的电力生产国和消费国，构建了覆盖发电、输电、用电全链条的“电力帝国”。

## 电力的背后是产业

用电量是衡量经济好坏的重要数据之一。经济学家认为，GDP等数据可以人为调整，但电表数据难以撒谎。美国近年来GDP增长但用电量下降，引发对其经济增长数据的质疑。

中国用电指标反映经济结构转型。2025年7月全社会用电量包括三大产业用电和城乡居民生活用电。第一产业用电量同比增长20.2%，增速居首，主要源于农业电气化、智能化转型加速。第二产业用电量增长4.7%，增速垫底，但结构变化明显。四大高载能行业（化工、建材、钢铁、有色金属）用电量同比增长0.9%，增速回落3.1%；新能源汽车、通用设备制造、仪器仪表等产业用电量增长迅猛，其中新能源汽车制造业同比飙升28.7%。第三产业用电量增长10.7%，反映新兴产业崛起。人工智能、大数据、云计算等产业高度依赖算力支持，数据中心耗电量占全国用电量的2.4%，且每年增长约20%。互联网和相关服务业用电量同比增长28.2%，远超第三产业平均增速。电动汽车高速发展带动充换电服务业用电量同比增长42.6%。城乡居民用电量同比增长18%，反映生活水平提升和电气化普及。

## 大国电力战略

国家政策继续支持电力产业扩容，核心意图之一是争夺未来全球产业竞争的主导权。电力不仅是传统工业运转的基础能源，更是决定新一代技术革命成败的关键要素。在人工智能、电动汽车、低空经济、脑机接口、具身智能等前沿领域，电力成为比数据、算法甚至人才更为底层的竞争壁垒。AI产业大模型训练所需算力背后是惊人的电力消耗。美国正在大笔投资电网，中国则在超前布局核电等清洁电力系统，未来大国博弈将取决于一个国家能提供多少稳定、廉价、清洁的电力。

中国目前拥有电力先发优势。当其他国家可能因电力短缺、电价高昂限制AI算力发展、迫使高耗能产业外迁时，中国凭借巨大电力冗余和成本优势，将成为全球高端制造业和数字产业聚集的“洼地”。中国在特高压输电、智能电网、大型储能、第四代核电等技术上处于全球领先地位，正在成为全球电力技术的标准制定者。中国在电动汽车产业链从动力电池生产到终端消费使用形成了全链路的电力成本优势。掌握更多更先进的电能和电力系统，就掌握了未来科技产业的命脉。

## 电力人民币的可能

能源与货币在现代社会深度绑定。工业社会，美元绑定石油；未来数字社会，电力将成为核心能源，货币权力也将向这一关键能源集中。中国有机会构建以电力为基础的新的人民币国际化路径——即“电力人民币”战略，通过电力输出和人民币结算结合，挑战石油美元的主导地位。

电力人民币的战略优势在于：
1. 电力是刚需商品，每个国家都需要，且消耗量持续增长，尤其随着数字经济、电动汽车、AI算力需求的爆发，全球电力需求将越来越旺盛。
2. 电力与石油有本质区别，石油是稀缺性资源，分布和定价受地缘政治影响；电力具有可再生性和本地化特征，生产和消费更容易脱离传统能源霸权体系，为“电力人民币”排除美国干扰提供了保障。
3. 中国在电力设备、电力技术和电力标准上具有全球领先优势，电力贸易往往伴随基础设施建设和长期运营合作，其他国家直接使用人民币结算能降低交易成本和汇率风险，尤其对美元储备不足的发展中国家具有吸引力。

中国已开展电力人民币探索，如中老铁路光伏项目采用人民币结算，哈萨克斯坦光伏项目用人民币计价，巴西水电项目通过熊猫债实现人民币融资。这些案例表明，电力人民币并非理论构想，而是已在实践中探索的路径。

从中短期看，电力人民币面临技术层面（跨境电网标准协同、调度安全）和政治层面（美国阻挠、美元惯性依赖）的挑战。但从长远看，未来能源金融秩序将发生大洗牌，电力人民币空间广阔。数字化、去中心化、碎片化的全球趋势正在削弱传统石油美元的基础，区块链技术、数字货币和区域支付系统的发展为绕过美元提供了可能。因此，电力人民币有可能以“边缘包围中心”的方式逐步渗透，最终可能部分形成与石油美元并行的双轨体制。
