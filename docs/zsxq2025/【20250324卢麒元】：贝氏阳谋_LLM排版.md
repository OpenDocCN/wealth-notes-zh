
# 贝氏阳谋

## 美国经济史回顾与现政府经济政策分析

大家好，今天是2025年3月22日，乙巳年二月二十三，今天聊天主题是“贝氏阳谋”。近期压力巨大，所以我们只谈外部事情，主要是聊一下美国经济史，分析美国现政府经济政策的经济学逻辑。这只是一个对历史的回望和对未来的揣测，大家权且听听。

### 美国历史上的三位重要总统

#### 本杰明·哈里森

本杰明·哈里森是美国第二十三任总统，是镀金时代的重要人物。在他的任上，美国超越大英帝国成为全球GDP第一，并完成了工业化。哈里森也是特朗普的偶像，他当总统的时间只有一任，是1889年到1893年，但这个人很神奇。

哈里森推动了《麦金莱关税法》等法案，涉及贸易战、海运战和汇率问题。1890年的美国像极了此刻的中国，应该在我们本届政府的领导下，中国可能就会完成对美国的经济上的超越。

#### 赫伯特·克拉克·胡佛

胡佛是美国的第三十一任总统，他执政的时间是1929年到1933年，那个时期被称为咆哮时代。胡佛推出了《斯姆特-霍利法案》，这个关税法案导致了全球的贸易战爆发。胡佛的经济政策看似一塌糊涂，导致了大萧条，但它的正面意义恰好是美国当代的思想家们选择贸易战的原因。

胡佛的经济政策导致了全球贸易总额暴跌，希特勒的法西斯主义抬头，日本军国主义扩张，苏联进入了黄金十年。这些事件都非偶然，所以必须了解经济史。

#### 理查德·米尔豪斯·尼克松

尼克松是美国第三十七任总统，他在1971年退出金本位，解除了布雷顿森林体系，结束了美越战争，完成了金美元向石油美元的历史性跨越。尼克松也推动了关税和物价管制，但带来了通胀问题。

### 贝氏阳谋的经济学逻辑

贝森特是美国现有的极其优秀的经济史专家，他非常清楚胡佛的法案所导致的后果，但为什么还要干呢？因为胡佛的法案短期之内是灾难，长期来看有意义。

胡佛的一系列政策导致了罗斯福新政，美国自此开启国家资本主义的道路。如果2025年贸易战全面爆发并且深度发展，我国也可能要采取重要资产和商品的供给制。

### 贝森特的阳谋

贝森特可能渴望一场由关税大战引发的全球经济的衰退和大萧条，他需要的是衰退和大萧条，他毫不介意恶性通货膨胀。鲍威尔的戏演给大家看，美联储表面上是要遏制通胀增加就业，但我不认为这是真的。

贝森特作为对冲基金经理，更了解经济剧烈波动过程中才有机会处理美元和美债的问题，特别是美债的危机。贝森特必然思考如何让资产负债表迅速完成平衡并有结余，正确的做法是处理衡量他们的尺子，那个就是美元。

美元要完成三件事情：贬值、充值、重置。美元贬的时候，理论上要求欧元、英镑、日元、人民币等主要货币跟它同步贬值，这个时候同步处理负债问题、资产负债表问题。

### 结语

经济史非常重要，每一件事情都非偶然，所以必须了解经济史。贝氏阳谋的背后，是美国现政府的经济政策和对全球经济的影响。
