
# 非税收入快速增长是怎么回事？

整理：公众号懒人搜索，懒人专属群独享

关于房地产税，这是另一个单独的话题，需要专门展开，小镇2021年曾经谈过，认为尚不具备房地产税出台的条件，2025年小镇的判断不变，虽然基础工作已经推进了不少，比如全国房产联网，但更多的问题尚未解决，还需要时间。至于法律上的限制，并非主要障碍，2017年首次内部征求意见时，法律和产权问题，也不是关注重点。单说时机，最起码要等到房地产完成转型，实现“止跌回稳”，重新进入上升周期，再说推出不迟。在当下房地产危机尚未度过的时候，不宜增加不确定性，这也是小镇在星球《房地产恢复与经济增长，一体两面》提到的决策选择，目前阶段最重要的是实现房地产的触底反弹。

重点回答另外两个问题：一是非税收的增长来自哪里？二是土地财政缺口如何弥补？

先说第一个问题，这两年网络上特别关注罚没问题，比如引起广泛关注的“远洋捕捞”问题、小商贩未经检验检疫被罚没数万等案例，不少企业被重罚也影响了企业运营。种种现象和舆论，就不赘述了。客观上，一些地区也确实把罚没作为财政缺口的短期补充，尤其疫情期间和疫情后。比如2024年长三角部分城市罚没收入出现大幅增长，这都是客观存在的，无需避讳。但也要注意，一些“跨省执法”也具有正当性，比如“远洋捕捞”这个案例，也有声音反映涉案的企业确实存在疑似诈骗或非法集资等问题，这背后存在地方保护。就不多展开了。总之在这种舆论关注下，2024年非税收入增长25.4%，很容易让人联想到是不是因为罚没收 入快速上升导致的。

但实际并非如此。需要注意，非税收 入来源很多，主要有12项：政府性基金收入、专项收入、行政事业性收费收入、国有资源（资产）有偿使用收入、国有资本经营收益、彩票公益金收入、特许经营收入、中央银行收入、主管部门集中非税收入、政府收入的利息收入以及罚没收入。其中罚没收入虽呈上升趋势，但占比在最高的2022年也仅为财政收入的2.1%左右，大概占非税收 入的10%左右，并非最主要部分，只不过罚没收入与民众距离更近，所以关注度很高。

比如2023年，一般公共预算收入216784亿元，非税收入是35655亿元，罚没收入是3936亿元，占非税收 入的11%。目前相关部门没有正式公开2024年罚没收入情况，但根据中新社年初报道，提到“罚没收入增长14.8%”，那么2024年全年罚没收入是4519亿元，占2024年非税收入的10%，也就是说，相比2023年，罚没收入占比反而下降了。

因此，2024年非税收入暴增25.8%，与罚没收入无关。同样来自中新社的报道，提到“罚没收入增幅逐季回落，其中上半年为25.2%、第三季度为13.8%、第四季度为—4.4%”，可见国家在2024年采取的整顿地方罚款的种种取得了很好的效果。

那么非税收入增长来自哪里呢？主要来自两块：一是部分中央单位上缴专项收益；二是各级政府盘活国有资源资产。

比如2024年11月蓝财长在新闻发布会上针对中央和地方财政收入不及预期、恐无法完成全年预算目标的问 题，就提到2024年后两个月，将安排有关中央单位上缴一部分专项收入，从而弥补中央财政收入。过去也有类似做法，比如2022年财政最紧张的时候，就安排央行一次性上缴超过1万亿元的结存利润，这笔钱用于留抵退税和加大对地方的转移支付；而2024年的上缴，主要来自中国烟草，2024年中国烟草实现工商税利16008亿元，增长5%，上缴财政总额也达到15446亿元，增长2.8%，都创了历史新高。

但就算创新高，增加的上缴财政总额也仅为几百亿元，对非税收入增长的贡献也不过1%多一点。所以最主要的非税收入增长还是来自盘活国有资源资产。小镇去年11月也谈了盘活存量国有资产，也就是公号发文《严厉监管新规的背后，是刻不容缓的改革》《中国的钱太多了，需要换个花法》两篇，举例了地方盘活闲置资产的做法，比如天津盘活科技广场项目。

中国国有资源资产存量已经非常庞大 了，但利用效率很低，大量闲置，在增量有限的情况下，盘活存。71.9万亿的庞大存量，那也是数万亿元。而从2024年非税收入暴增来看，国有资产盘活干的不错，很多地方在解释非税收入暴增的时候，都提到多渠道盘活国有资源资产形成的收入，是 非税收入增长的最主要原因，占比普遍在八成以上。

但这个数据肯定有水分。

所以第二个问题：如何弥补土地财政缺口也就有了新答案。

目前最主要的就是盘活存量，典型的如2016年提出的REITs，也就是不动产投资信托基金，2020年启动试点工作，经过几年试点后，2024年7月发 改印发《关于全面推动基础设施领 域不动产投资信托基金（REITs）项目 常态化发行的通知》，还专门召开了记者会，意味着REITs进入常态化阶段。

土地财政缺口猛地一看非常大，从2022年到2024年，已经连续三年下降，累计下降超过44%，到2024年已经降到4.87亿元，相比2021年的8.7万亿元，减少了近4万亿。这也导 致土地出让收入占全国财政收入的比重，从2021年的35.9%，下降到2024年的17.3%，基本接近2015年的低点。

关于土地财政的是非对错不多谈，各有看法。

但要注意，土地出让收入下降4万亿，并不代表需要寻找4万亿的增量 空间。因为土地出让金的相当部分要投入到对应地块的基础设施建设、公 共项 目等，地块卖出去的少了，土地出让 收入减少，相应的基建等支出也下降了。

这个比例各地不同，但土地出让金减少带来的财政缺口肯定大大低于 4万亿的表面数字，而且土地财政萎缩以 后，地方上乱花钱的冲动会得到很大 遏制。

所以从理论上，通过盘活闲置资产，能够很大程度弥补土地财政缺口。

但注意，目前还只是理论上。

2024年国有资产盘活带动非税收入暴 增25.4%，快得令人心虚。闲置国有 资源资产哪有这么容易盘活，随便一个项 目都不是一两年能完成的，否则早就干了，从2013年到2023年，只有2015年、2019年、2022年增速曾经超过20%，其他年份基本在5%左右，甚至2017、2020年还出现负增长。这种个别年份的猛增，很有意思。

非税收入由于来源广泛，而且征收很 灵活，所以现实中非常适合在账面上 弥补税收和其他收入下降，从而实现 财政平衡，反之当税收增长还不错的 时候，非税收入增长放缓甚至出现下 降。比如2023年税收增长8.7%，非税收入反而下降了 3.7%。

说白了，非税收入有时候就是用来拉 高财政收入数据的，对地方而言，是 季度、年度考核前突击的一大工具。

比如2024年非税收入增长，在3月、6月、9月、12月的变化就很大，很 可能是为了完成财政收入的考核目 标。

各地在这方面就做得更多了，就导致 了非税收入的暴增。

加上去年国家层面重视国有资源资产 的存量盘活，还要注意一个关键点，那 就是地方化债，目前地方负债率已 经成了很多国家增量政策划拨的关键 考核，比如2024年12个省份被要求 严控新建政府投资项目，核心考核指 标就是负债率和化债进度。

负债率是当地负债和一般公共预算收 入的比值，化债主要是降低分子，但 是在短期也可以通过做大分母来降低 债务率，税收很难操作，非税收入相 对就容易很多了，自然地方上就有很 大动力去做大非税收入。

2024年城投债到期规模很大，总计 5.4 万亿，年底累计剩余到期合计约 7.99 万亿，重点化债的地区迫切需要 想办法降低负债率，那么盘活国有资 产收入，就成了筹集化债资金的重要 来源。很现实地说，盘活里面的操作 空间还是很大的，否则国家何必强调 “强化覆盖全级次子企业的穿透式监 管”。

但这就带来了新的风险，意味着目前 的国有资源资产盘活，相当部分必然 含有或多或少的水分，可能不过是数 字游戏，意味着实际债务压力并没有 下降，甚至因为搞盘活存量，反而增 大了财政压力。

那么接下来在盘活的同时，就需要加 大监管。否则非税收入的增长可持续 性很成问题，而且存量盘活肯定是先 易后难，等容易的一批盘活，后面的 再想盘活难度就更大了。

因此接下来非税收入增长肯定要减 弱，而中央单位上缴也只能是临时性 措施，也就意味着表面数字的改善，对财政改善的实际影响有限。

总之，土地财政的改善，没有绝招， 是一个很长期的过程。几十年来，地 方财政宽裕也不过是土地财政这十来 年，其他时候都很紧张，也算是回归 常态了。
