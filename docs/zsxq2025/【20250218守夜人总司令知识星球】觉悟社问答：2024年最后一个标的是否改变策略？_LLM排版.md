
# 觉悟社问答：2024年最后一个标的是否改变策略？

## 星友提问：
司令您好，2024最后一个标的(力盛体育)今天直接跌破位了，是不是要调整好久了?

## 司令回复：
反正今天加了200多个，因为终于等到了。市场有上有下，他会随情绪波动，我们只需要关注自己在什么样的位置该做什么事就行。不用跟着情绪的起伏去频繁改变策略。不管是上还是下都需要利用外部情绪，外部情绪既能造成一致性预期，从而能形成自我强化的态势，对于企业的市值管理者而言，才更容易是四两拨千斤。

第2波阻力点在20元左右在准备之中，本来昨天是准备利用外部乐观情绪的，因为各种座谈会，支持民营企业发展，支持科技之类的官方表态，有利于形成这样的情绪。之所以虎头蛇尾，是因为普通人相信这种情绪，而掌握关键力量的市场群体，对于美国关税政策给接下来的现实带来的不确定存各种分歧，这些人更能搅动市场，而普通人是跟风，他们的看法比普通人的看法对市场的波动权重更高……

市场中存在各种博弈，而且非常脆弱，为每一场波动找一个说法，不过是一种心理安慰的行为。 一篇小作文发出来影响波动，为小作文辟谣就会引发剧烈的波动，小作文这些东西没有规律可言，没有章法。

这就好比八路军当年打伏击。自身实力强大，地势条件优越，就可以去阻击别人的主力。自身火力不足，或者地形并不那么有优，那就只能等敌人主力过去，阻击敌人的辐重部队。如果地形实在太差，火力也太差，人员不足，就只能偷袭别人的后勤人员，这里的重点是要在预设的有利阵地去狙击有把握的敌人，其他的敌人，你要放过他，要忍住。

通过计算之后，上上周五热情高涨的时候跑了，因为不可能一下子就到位，我觉得会有两三波才能到位。当时处于突出第1波阻力点16元预期的位置，所以就跑了。算了一下，第2波之前预判的阻力点20元的起步点应该在12.3左右。除非大环境巨变，第2波搁浅，否则12.3元这个点位就是二次进入的伏击阵地，今天等到了！我认为第2波之前预判的阻力点20元的突击前锋，差不多会在14.3左右止步。如果抵达并超出了那个位置，也会见好就收先撤一步。然后等待第3波的机会。

阻力点能不能突破心里面没谱，因为这个需要大环境的配合，但是在阻力点之前的一些波折，认为大概会呈现出这种状况。也准备按照这个判断去操作。

当然了，这是个人的看法。你们自己会有关于细节的把握，通常情况下都不谈细节，因为细节不可控。完全根据个人的判断和感知来操作。我不能确定对每波细节的判断都是正确的，但是我们任何人在执行一个作战任务的时候，都会有一个具体的策略，而不是完全的随机应变。
