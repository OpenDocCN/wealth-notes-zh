
# 非农造假后的资本市场
## 250805 子明解读

前几天，美国劳工部公布非农数据，就业大幅弱于市场预期，尤其是5、6月的大幅下调直接冲击市场，导致美股重创。很多朋友都在调侃美国数据作假，别人只是改小数点后面的数字，美帝是直接挪小数点，甚至几个朋友还开始唱空美股和美国。但是有一位伟大的教员说过，战略上要藐视对手，战术上要重视对手，美国短期内的风险并不大，这一轮冲击属于事件型冲击。

一周前我就在群里跟朋友们说了，美股要在七月底减仓，避免特朗普的关税达成冲击，A股月底前也要减仓，避免政治局会议低于预期的冲击。主要逻辑，在前几天的文章也进行了表述。

首先，要看中美这几个月来刺激经济的大背景。中美在4月全球关税战之后，都必须震慑他们的对手，先是彼此极限施压，之后又各种努力与作弊，让对手认为自己无限，以取得谈判桌上的优势。这也是为什么这几个月来，中美两国的股市先是一轮暴跌，之后又手拉手一起高歌猛进，越是比邻全球政要一起放假的八月，比邻最后的谈判时间点，涨得越凶。

在这个大背景下，特朗普可以忍受劳工局的虚假数据，以及美联储依照虚假数据坚持的不降息，中国同样也在放纵各种小作文，导致大宗与资本市场的兴风作浪，很多时候假牌也能当真牌来用。但中美允许一些人浑水摸鱼，是因为中美都需要从对手身上获取巨大的利益。只要前方传来了胜利的消息，解决了外部的威胁，中美马上就要把刀口转向国内，特朗普拿掉了劳工统计局的局长，逼着美联储副主席辞职，咱们这边各路妖精这几天也纷纷熄火，各种措施也开始动真格的。

所以，五月超市场预期的瑞士谈判以来，中美两国资本市场的政策饭，随着瑞典谈判结束了，大家要换思路了。但是，这并不是说资本市场没得玩，中美两国愿意搞Fake，是因为短期的Fake和忍耐，可以获取长期的巨大收益与回报，如今，中国政治局会议不搞强刺激，美国敢漏真数据，剑指美联储，是因为大家已经拿到了各自最重要的东西，索性不装了，摊牌了。

迫使美国政治让步的中国，可以专注于国内，全力发展科技创新，可以对内进行大刀阔斧的改革，一大批具有国际竞争力的企业将陆续脱颖而出，迫使欧中日韩经济让步的美国，也会调转枪口，放松监管加速科技，调整美联储人事安排，继续他的换房本，推动美联储大力度降息以支持实体经济。

所以，特朗普繁荣没变，短期不要担心股灾，中国也只是政策牛退去，长牛还是在的。8月后，尤其双王会面后，中美国内主要变化是，中国是新质生产力加速取代旧质，旧质纷纷被淘汰，美国是产业集团侵吞金融集团的利益，蛋糕重新分配。利益集团的此消彼长之下，全球两大霸主的内部集团之间的温差，将迅速拉大。
