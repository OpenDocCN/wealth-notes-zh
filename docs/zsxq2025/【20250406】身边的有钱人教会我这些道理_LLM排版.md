
# 身边的有钱人教会我这些道理

财富是对认知的补偿，而不是对勤奋的嘉奖。凡是与利益相关的事情，该争一定要去争。

身边的有钱人教会我这些道理：
解决麻烦的唯一方法，就是你要给对方制造一个更大的麻烦，麻烦与麻烦对冲，对方就会被迫跟你和解，而不是靠着讲道理或者委屈求全。委屈是求不了全的，只有实力才能获得别人的尊重。

接触过有钱人才发现，其实勤奋根本无法致富，整天工作的人，是发不了财的。财富是对认知的补偿，而不是对勤奋的嘉奖。凡是与利益相关的事情，该争一定要去争，甚至要敢于和人翻脸。反过来，凡是与利益无关的事情，就要克制自己的好胜心和反驳欲，不要轻易得罪人，不要轻易和人起冲突。

没有人会因为你的好而来对你好，他们只会因为你的实力而对你好。不要在乎任何的人情世故，当你有钱之后，仇家都是好朋友，钱才是最好的社交。忘掉人脉二字，社交的本质是价值互换，你没本事没资源跟人换，人脉就是个笑话。

人生的贵人，从来都是自己。当你的能力、资源、地位配不上你的社交野心，你所做的都是无效的社交。种下梧桐树，才有凤凰来，抱人大腿，不如锤炼自己大脑，看人脸色，不如提升自己成色。

永远不要怕别人的误解或者对你的不爽，没必要让所有人都了解真实的你，也没必要跟别人解释个不停。你是谁，懂你的人自然会懂你，不懂你的人，说再多也不会懂。

不恶语伤人，重视别人的感受。图一时的嘴快，讲话横冲直撞，夹枪带棒，伤了人，也是给自己埋下隐患。言语温和，懂分寸，说话中听，别人舒服，自己也舒服。

人都是会变的，可能现在你们的关系很好，穿一条裤子，有什么事情也能够无话不谈。但是你真的敢肯定，这辈子都会这样好下去吗？没有人能经受住利益的考验，一旦受到更大的诱惑，出卖别人也是分分钟的事。不要说自己不会，那是筹码不够而已。

成年人的交往，就不要以好坏来作为衡量准则，而是应该把利益放在第一位。在这个世上，不是所有的人，都值得你掏心掏肺的付出。也不是所有的人，都值得你低三下四当舔狗。对什么样的人，就用什么样的交往方式。就像现在流行的一个词，千人千面。和人交往的时候，从本质上来说，还是靠利益互换的。既然是交换利益，那么肯定就会有多和少的问题。这时候体现的，就是利益分配的能力。

接受一项工作，首先考虑责任的人，患得患失，这种人是守旧、无能的人。为掩饰自己的无能，往往常作正反两方面的行动。借钱容易借出感情，但是也容易伤感情。

生活中，大多数人往往是谈钱色变，平时称兄道弟的人，关键时刻一个都靠不住。能对你刀子嘴的人，本身就是刀子心。

没有人能抵挡足够大的诱惑，和足够多的筹码，红杏出墙，要么树太高，要么太矮。人在低谷的时候，不要去打扰任何人。天上下雨地上滑，自己跌倒自己爬。要想别人帮助你，你得酒换酒，茶换茶。

其实根本没有高冷的人，只是人家暖的不是你。在你面前沉默寡言的人，在另一个人面前说不定会变成话唠。每个人面对不同的人态度都是不同的，这是一种能力，不是虚伪。

做事聪明一点，做人傻一点。人们把这种介于聪明和傻之间的状态，叫做智慧。和任何人的关系都是利用和被利用的关系，没有其他，只是利益交换。

世上所有的美感，其实是一种陌生感和遥远感。一览无余只会徒生厌倦。所以人间最美的距离，就在将够着却又还没够着的地方。凡是存在的距离，都是合理的，我们不需要刻意走近或走开，只需做好自己，该走近的早晚都会走近。
