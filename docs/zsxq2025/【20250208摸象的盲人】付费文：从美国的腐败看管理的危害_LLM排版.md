管理就是腐败，保护就是残害。

人性都是相通的，不分种族肤色；规律都是普适的，不分东方西方。

是促进了还是危害了东大的足球事业？

管理就是腐败，保护就是残害。

这是行政学的基本原理，不存在任何例外，如果危害尚不显著，唯一的理由就是该部门还不够强大，只要该部门的权力和规模成长起来，就一定会造成巨大危害。

行政部门和行政人员，既没有意愿，也没有能力，管好任何事情。他们唯二的意愿就是寻租和扩权，而扩权也是为了更好地寻租。

为何行政管理一定会把事情搞砸？很简单，因为只有权力，没有责任。就算偶尔有一丁点责任，但与其巨大的权力相比也只是九牛一毛。

权力与责任不对等，一定会导致权力的滥用。

所以，政府重视什么，从长远来看，就一定会搞砸什么。

这听起来很悖论，但在聪明人看来，就和 1+1=2 一样自然乃至于必然。

但社会离不开管理，怎么办？

合理的管理，一定不是基于行政权力，而是基于责权对等。

父母对孩子具有管理权，是因为父母对未成年子女负全责，而学校对孩子
的管理权本质上是来自于父母的授权，如果父母不同意，那么学校可以拒
收，但不能违背父母意愿进行越权管理。

市场经济为什么有效？那是因为企业要对其产品负责。

对于较小的不太重要的商品和服务，事实上无需任何行政管理，只需要从
法律上明确质量问题的赔付责任即可。

对于关系重大的商品和服务，由于混账的有限责任公司制度，企业可能承
担不起赔付责任，那么就应当由有赔付能力的主体来代为管理。

通俗地讲，大资金（比如银行、保险公司或者专门的基金）为企业的商品
和服务进行全额担保，因此获得监管企业生产的权力，同时抽取一定费用
作为管理的收益。

担保人没有任何资格限制，缴纳并冻结保证金的任何主体都可以成为担保
人，而保证金除以法定保证金率就是其可以提供的担保总额。

不同的担保人之间是竞争关系，不存在垄断和黑箱。

一方面，担保人面向消费者竞争，通过承诺比法定最低赔付额更高的赔付
并简化理赔流程，可以帮助其监管的企业获得更多市场份额，从而从企业
获得更多的费用（管理费的全部或部分按量计费）。

另一方面，担保人面向企业竞争，管理费标准如果过高，或者管理严苛到
不合理的程度，企业就自然会转向其他的担保人，甚至企业也可以向亲友
和员工寻求集资从而进行自我担保，只要保证金到位就行。

担保人之间甚至可以进行更加激烈的竞争，那就是主动发掘竞争对手所担
保的企业的质量问题，帮助社会公众进行大量索赔，从而对竞争对手达成
毁灭性打击。这种玩法对资本很残酷，但是对人民很友好。

至此，行政管理作为一种野蛮落后的原始遗俗，就从人类社会消失掉了，
而狭义的政府也就不存在了。

但广义的政府任然存在，因为我们的目的不是消灭公权力，而是让公权力
只做它必须要做的事情，比如暴力机关，比如制定规则的立法机关，比如
裁定纠纷的司法机关。
