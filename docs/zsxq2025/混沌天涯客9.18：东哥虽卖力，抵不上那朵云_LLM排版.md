
# 混沌天涯客9.18：东哥虽卖力，抵不上那朵云

原创：混沌天涯客 公众号：混沌天涯客 25年9月18日

棋逢对手。如果一个高手对线的是一个臭棋篓子，那么纵然流量炒得天翻地覆，也就那样。就像罗永浩对上西贝贾国龙，罗永浩还没怎么动手，贾国龙就猛往自家店里捅刀。

热闹归热闹，但不精彩。精彩的必须是棋逢对手，像是罗对上梅西，德约对上纳达尔，你若让德约跟一位业余网球老炮儿对垒，即便这老炮儿地位够高引世界瞩目，只不过另一场热闹。

真正的精彩，是要棋逢对手。罗永浩的对手不能是贾国龙，老贾一个回合都没撑住。王自如仗着年轻撑住了，跟罗永浩对战几小时。时间是够长，战况完全是一边倒。至于其余的，西门子、星巴克、必胜客。。。包括老罗的前老板新东方，面对火辣辣的老罗，都纷纷避让，不对线。

在今天国内的互联网，老罗是没有对手的。这不禁让人想起昨天国内的互联网，他曾有过一个旗鼓相当的对手，两人大战三百回合，最后是罗永浩的英语学校被弄倒闭掉。

在今天世界的互联网，那位老方还在做节目。老实说他的节目我经常看，昨天他还以胜利者的姿态对罗永浩一番挖苦，把老罗的老底给揭了个干净。但是，这节目对老罗已经没了影响，两人分属完全不同的网。

罗有罗网，方有方舟。但不要以为没了对手，老罗就活得更痛快了。比一比他十几年前在说什么、在做什么，今天的老罗显然更加小心翼翼。他怼贾国龙的那场直播我看了，有一句话他多次重复：他老罗不反对预制菜。

这一句重复了好几遍的话，足以显露今天的老罗，在真正的资本面前，身段柔软如蠕虫。既然不反对预制菜，那么首先，上海家长正在呼吁的校园预制菜不可口的小问题，就别找老罗了。老罗自己没孩子，更不会反对别人的孩子吃预制菜。

其次，今年国内互联网最大的事件，外卖大战，京东、阿里各砸出百亿下场，与美团pk，争取让更广大的人们吃上更好更快的外卖。针对这一事件，罗永浩已经划清界线，他绝不反对预制菜，更不敢反对以预制菜为核心的外卖大战里赤膊上阵的资本大佬。

三英战外卖，率先赤膊上阵的是刘强东。继亲自送过外卖之后，他昨天又开启了人生中第一场直播，直播做他的拿手好菜：黄狗猪头肉。直播中，东哥兴致勃勃，他首先提到了贾国龙和罗永浩，但这两位的重量加一起也只能给东哥助助兴。在东哥的棋盘上，对手始终是一个。炒菜的时候没打着火，东哥立刻来了句：“这煤气灶不会是从马云那买回来的吧？”

东哥开火炒菜，不代表从京东点的外卖就是东哥炒的，也不代表外卖就是炒的，更不代表送到您手里的外卖曾被开过火。

今天流行电车，气改电，微波炉电磁炉电瓶车都是用电的。

在东哥忙炒菜的同一天，那边也爆出消息，马云亮相阿里园区某酒吧。

关于亮相，其实东哥更执著，除了在网络上抢热搜之外，他还有几次更有意义的线下亮相：亮相第七届全国青年企业家论坛并发表演讲；亮相夏季达沃斯论坛并担任联席主席；亮相吉林长春得到省长亲切接见。尤其是达沃斯那次，是上了7点钟新闻的，东哥在电视画面上做笔记的样子，很认真。

相比之下，马云的亮相，要么是在西湖边骑车，要么跑到内蒙古种树，现在又现身于某个酒吧，都像是未经安排，不小心流出。

仅从这一点，就可以管窥刘强东和马云段位的差距。东哥很卖力，马云很随意；东哥是赤膊上阵搏热搜，马云的热搜随随便便；东哥频繁亮相重要活动，马云不需要，即便是那个由东哥发表演讲的青年企业家论坛是在杭州举办，人就在杭州的马云收着邀请也懒参加。

今年以来，马云只亮相过一次重要活动，坐在第一排认真做笔记。但那一次就够了，抵得上东哥的千次万次。这一点，东哥和马云心里都清楚。

东哥是以马云为对手，有空就忍不住提他。但马云是没怎么想过东哥。棋逢对手，值得马云出来亲自对线的人，已是世界那一边的马斯克。

当在高峰的时候，马云被戏称是马主席，马斯克被调侃是马总统。已被白宫开除出门的马斯克，昨天接受了一个专访，内容很丰富，我认真看了。

马斯克说得最多的，是已经推进到第三版的机器人，他说那将是人类历史上最大的产品，所拥有的Al大脑和灵巧手部，足以让人类一边凉快去。

请注意，马斯克要人类靠边站，并不是号召人们躺平，躺在家里外卖就来了，这不是他关心的。马斯克矛头所指，仍然是令他耿耿于怀的华盛顿和西方精英社会。

他说，华盛顿烂透了，体系僵化，债务离谱，每年的利息支出甚至超过了国防预算，而坐在白宫里的特朗普仍然好大喜功。如果AI和机器人无法把这帮人替换掉，美国就麻烦大了。

他还说，西方社会烂透了，像是在自我毁灭，人们不愿意生孩子，犯罪问题被忽视，媒体装聋作哑对很多事情不报道。精英群体毫无责任心，只顾排着队到白宫拍马屁。

他今天用最大的精力鼓捣出人形机器人，就是要从根本上扭转这一趋势。

由此可见，虽然马云跟马斯克公开对过线，但今天的马斯克，连一句"Where is Jack Ma?"也懒得问。他心心念念的对手，是上炕之前甜蜜蜜，穿上鞋之后就将他抛弃，今天正端坐在美国白宫里不出意外还要坐个三年零4个月的特朗普。

三年零4个月，时间不多了。

特朗普已经快速把马斯克这一页翻过，一个暗度陈仓，把那困扰了美国人许久的短视频app拿下。

综合各方消息，为了拿下TikTok，特朗普付出了很大努力，做出了诸多让步，其用心程度，大大超过了他调停俄乌战争的漫不经心。

长期以来，世界两张网，一个网里有抖音，一个网里有TikTok。能用TikTok的网里也可以装抖音，只能装抖音的网里不能装TikTok。从特朗普第一个任期开始，他就试图从美国那个网里掉但没做到。继任的拜登嘴上说要做，实际很鸡贼，把禁令的生效期搁在自己卸任的那一天，既不影响当次的选举，又把难题交回给下一任。

特朗普上来了，继续把截止日一拖再拖。特朗普1.0+拜登+特朗普2.0都下不去手，原因很简单，在他们发觉之前，这款app如五湖的鲤鱼一般，迅速传播，用户破亿，对于总人口3.4亿的美国来说，相当于二分之一的人口，说明几乎所有年轻人全用上了。

硬件也许有后门，但软件里才是灵魂。

虽然像是微信、支付宝、美团、小红书、抖音、快手等等app美国人均可安装，但用这些的多是华人，只有TikTok深深俘获了美国年轻一代的灵魂。

里面的内容，当然是可以控制的。只要刷过抖音的都知道，那里有一面信息墙，越刷越难突破，越刷捆得越紧。人都是信息的载体，假以时日，人将活在抖音编织的墙里。

这种状况，一个网里的抖音可以，另一个网里的TikTok不行。其实换到用抖音的那TikTok个网络里也不行，说不行就不行。但对于烂透了的华盛顿，明知道不行，碍于年轻用户破亿，不行也得行。

不是他们真正关心下一代的身心健康，正如马斯克说的，自私透顶的西方精英，哪管迷失未来的年轻人。

他们有所忌惮，下不去手，只因为每隔四年年轻人手里可以投出的那张纸。冲动、幼稚、毫无大局观的美国年轻一代，谁让他们失去TikTok，他们就可以让谁失去华盛顿。

他们，就像现在的罗永浩，必须小心翼翼。
