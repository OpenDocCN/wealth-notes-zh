
# 2024年中国上市公司电影投资报告

## 中国传媒大学文化产业管理学院

本报告由中国传媒大学文化产业管理学院发布，文化产业管理学院彭健策划撰稿，汪润玺、盛振楠参与数据支撑。

### 报告内容概要

1. 中国投资电影产业的上市公司（含A股、港股、美股）的特征、业绩、市值。
2. 中国投资电影产业的上市公司对2024年内地上映规模以上票房影片出品情况。
3. 中国投资电影产业的上市公司2024年相关的重大投资并购活动及特点。

### 2024年内地上映国产影片情况

2024年1月1日至2024年12月31日在中国内地首映的国产影片共430部，电影票房为425.02亿元，同比减少22.7%，观影总人数为10.1亿人次，同比减少23.1%。其中，超过3/4影片的票房不及1000万元，市场影响力有限。

本研究统计票房超过1000万元的国产影片共104部，称为“2024年内地上映规模以上票房国产影片”。

### 2024年中国投资电影产业的上市公司界定

#### 1. 狭义的主营业务涉电影产业的A股上市公司

中国证监会上市公司行业统计分类与代码，电影产业归属“广播、电视、电影和录音制作业”，据中国上市公司协会发布的《2024年上半年上市公司行业分类结果（按行业排序），“广播、电视、电影和录音制作业”上市企业共21家。其中，东望时代（注：已经转型主营能源业务，严格意义上已不属于“广播、电视、电影和录音制作业”板块，但其子公司千鹿影业开展电影业务，为规模以上票房国产片2024年《骗骗喜欢你》联合出品方）。

#### 2. 广义的投资中国电影产业的上市公司

除中国证监会行业统计分类狭义的主营业务涉电影产业的A股上市公司外，其他中国A股、港股、美股上市公司中还有很多参与了2024年内地上映规模以上票房国产片的投资，是中国内地电影市场重要的资本力量。

#### 3. 2024年中国投资电影产业的A股上市公司特征分布

2024年中国投资电影产业的A股上市公司共有27家，呈现以下特点：

1. 影视企业A股IPO与非影视企业转型影视业的时间集中在2015年。
2. 注册地集中在北京与浙江。
3. 实控人类型民营国企比例接近。

#### 4. 2024年中国投资电影产业的A股上市公司市值

市值超过100亿元的为9家，市值排名前五的家为芒果超媒、光线传媒、万达电影、中国电影和完美世界。

### 总结

2024年中国电影市场面临诸多挑战，但仍有部分上市公司在电影投资领域持续发力，展现出较强的资本实力和市场影响力。未来，中国电影市场的发展仍需各方共同努力，推动行业持续健康发展。
