
# 做好迎接长期低利率的准备

有读者留言询问是否如之前说的把房贷重定价周期改为三个月，小镇回复仍然坚持。因为未来相当一段时间，中国将迎来低利率时代，利率还要继续下调，直到刺激经济的常规货币政策手段用尽，届时将可能真正采取货币宽松等非常规货币政策。国外机构也有类似结论。本周德意志银行认为中国当前更像是1980年的日本，而不是1989年的日本，还认为当前中国资产价值被严重低估，未来中国想要促进消费，必然会选择继续放宽金融体系，还可能会像日本一样采取宽松政策，这对于资本市场是巨大的利好，因此德意志银行认为海外资本需要尽快到中国抢筹。关于这份研报，小镇将于明天展开。

2018年，时任央行长还认为中美利差处于80-100个基点的“舒适区间”,将中国利率高于美国视为对抗贸易战的利好。当时央行也讨论低利率，但针对的是欧美低利率，虽然也判断中国未来利率一定会持续下降，但认为中国的利率水平仍然会高于发达国家，这种略高水平对中国经济是有利的，因此需要控制利率下降速度。于是，2019年9月易行说央行在进行货币政策操作时，将珍惜正常的货币政策空间，也就是尽可能延续正常货币政策发挥作用的时间。所谓的“正常”指的是利率在0以上，如果利率低于0，那么传统如降息等货币政策将失效，届时只能被迫采用量化宽松等非常规货币政策。所以易行时代，央行降息步子很小，基本是10基点、15基点的降。

正是基于以上认识，央行一直在批判“低利率”和“负利率”,反对量化宽松和现代货币理论。如2016年第一季度货币政策执行报告专栏报告就认为“长期低利率环境可能扭曲资产价格，降低风险溢价，造成风险累积，不利于金融稳定”,央行也频频出面，谈低利率的危害。重点批判的是2008年之后，主要发达经济体持续实施超低利率和量化宽松政策，导致全球长期处于“低增长、低通胀、低利率、高债务”的“三低一高”环境。然而后续发展令人惊讶，在贸易战和疫情冲击下，主要发达经济体迅速从“三低一高”转变为“低增长、高通胀、高利率、高债务”的“一低三高”，危机变得更为严重，不断拖延危机爆发的时间；而中国物价也迅速走低，并且持续低迷，GDP平减指数已经连续7个季度为负，在资产荒的背景下，国债被疯抢，于是出现了国债收益率被迅速压低。

最终，中国反而进入了低利率时代，而欧美日等主要发达经济体则进入了高利率时代。央行显然不希望这么快进入低利率时代，否则也不会从2016年开始就不断地阐述长期低利率的危害，但利率不是央行拍脑袋决定的，央行最多只能施加一定影响，让利率的波动变得更加平缓。利率的变化本质取决于一个经济体的整体资本回报率，资本回报率高，利率就高，反之就低。而资本回报率归根到底取决于经济增速。随着中国经济体量增大，潜在增长水平一定持续下降，“十四五”规划制定前，大多数专家、团队判断认为“十四五”期间中国潜在经济增长率在5%左右，之后不断下降，到2030年以后下降到4%左右。目前物价低迷不会一直持续，但就算回复正常通胀水平，考虑到全球贸易摩擦和中国巨大的产能，通胀率也就是在2%左右，比如今年全国各省报告，普遍下调了通胀目标，从去年的3%降低到2%。

而潜在增长率减去通胀，一般认为是十年期国债合理的利率区间，也就是在2.5%左右波动，这就是央行去年上半年曾试图守住的利率线。但市场机构对中国经济增速的前景普遍悲观，十年期国债收益率跌到1.6%，本质就是市场机构们认为中国未来经济增速要比预想的更低，他们认为也就是4%左右。去年某个首席经济学家就曾“得意洋洋”的公布自己的研究结果，认为中国GDP统计数据高报了。不过国外机构有不同意见，比如上面提到的德意志银行，就提醒道，被认为“日本经济奇迹”的上世纪80年代，平均增速也不过是4%左右，以中国现在GDP远大于日本的体量，竟然还能保持4%甚至5%的增速，这怎么可以批评为“缓慢”？德意志银行团队认为，在后世看来，对当前的中国经济增长不是“缓慢”而是“奇迹”。

但无论怎么说，还是要做好未来经济增速持续下降的准备。小镇有一个推断，国家从2023年开始实际已经在潜移默化的降低对经济增速的追求，从2023年增长目标5%左右，但更强调的是在5%以上，比如5.2%，到2024年变为5%，预计2025年会接受增长率降到5%以下，如4.8%或者4.9%。在这种趋势下，决定了未来长期处于低利率时代。那么这意味着什么呢？首先就如开头说的，尽管把房贷利率重定价周期改为三个月。

对于资本市场包括房地产的影响，上世纪80年代日本提供了一些经验：随着常规货币政策空间用尽，不得不开始使用超常规如货币宽松政策，催动利率不断下降，而低利率意味着政府付息成本下降，这是政府债务大幅扩张的良机，意味着未来会加大国债发行力度，目的也是为了刺激经济，于是日本政府债务大增。但低利率、低增长的情况下，意味着钱贬值速度下降甚至负利率情况下反而升值，于是无论居民还是企业更愿意把钱存起来，导致投资意愿下降，更不利于经济增长，于是日本在推行零利率和积极财政政策的情况下，增长反而陷入停滞。而危机最先爆发在银行业和保险业，出海就成为日本资本投资的主流，也就有了现在经常有人说的海外第二日本。

在缺乏投资标的情况下，银行更愿意提供低利率的房贷，低成本加上改善居住的刚性需求，其实是有利于房价上涨的，美国就是典型。而日本房价增速平稳，主要是因为日本国土狭小、人口众多，早在上世纪80年代就已经完成了城镇化，再加上日本人口老龄化，大量日本人随着出海潮离开日本，于是压低了房价。以上是日本的经验，但中国与日本差别巨大，决不能照搬。比如日本在美国压力下被迫迅速升值，虽然日本政府也希望升值，但速度太快了，而中国未来更倾向于保持目前汇率水平，甚至因为中国物价低迷，加上中美利差影响，人民币贬值压力很大，需要央行稳利率；还有货币管制，日本资金流动是自由的，所以日元也有美元老鼠仓之称，但中国未来反而会加强外汇管制，限制资金流动，人民币国际化也要让位于汇率稳定。

综合这些不同。与日本相同的是：未来国债等政府债会继续扩张，通过政府债融资将超过信贷，2024年已经发生了，信贷增速下降，但社融却是上升的，关键就是2024年政府债大举扩张。而银行业、保险业也必然要迎接挑战，具体在前文已经分析了。不同在于；由于中国更严格的资本管制，资金出海难度很大，不可能像日本一样大量出海获得收益。因此大量资金会拥挤在国内，导致国债收益率加速下行，于是能够稳定获得现金流的高股息资产会得到青睐。除了高股息资产，房地产市场也会在低利率影响下加速完成“止跌回稳”，目前全国重点城市租金收益率已经很接近房贷利率，而未来利率还会继续下降，这就意味着哪怕房价不涨，买房出租也是可行的投资方式了，但注意这跟过去的逻辑大不一样了，未来买房，需要更关注当地租房市场情况，租房情况反映了很多事情，比如人口流动、就业、收入等。
