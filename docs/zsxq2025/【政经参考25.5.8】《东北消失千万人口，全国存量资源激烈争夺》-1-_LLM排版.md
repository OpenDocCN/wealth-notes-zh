
# 政经参考25.5.8《东北消失千万人口，全国存量资源激烈争夺》

东、黑龙江、河南、湖南、辽宁、吉林、山西的人口减少最多，都减少了20万以上，山东更是减少了近43万人。

从人口自然增长，就是出生人口减去死亡人口的情况来看，31个省份里只有10个省份是正增长的，排第一的是"号称 最敢生省份 的广东，增加了47万人，其他的新疆、青海、广西、西藏这些地方则是增加了1到8.8万人，都是个位数，而且这些省份相对欠发达。

剩下的省份则是普遍地陷入到了老龄化中，就是老年人多，出生婴儿少，人口的自然增长率全是负的，其中四川、江苏、辽宁这些省份问题最严重，都减少了超过20万人。

这些数据背后的趋势是，我们已经进入了人口存量期。

首先，从总量上看：2024全年中国总人口减少了139万人，而2023年减少了208万，2022年减少了85万。

其次，从结构上看：中国已经进入了"中度老龄化 阶段，65岁以上人口占比达到了15.6%，60岁以上的则是占到了22％，老年人多、年轻人少、婴幼儿更少。

最后，从地区上看，各地的人口增长都十分乏力，即使是 最能生的广东也在变弱，广东在2010-2020的十年间平均每年新出生125万人，平均死亡45万人，自然增长80万人左右，但现如今，自然增长仅仅维持在40-50万人的水平。

所以中国跟人口相关的蛋糕已经很难继续做大，这对各个城市的发展会产生巨大影响。

简单说，以前是人多地少，所以土地和房子值钱，但未来10~20年是人越来越少，房子越来越多，各地自然增长人口都纷纷进入负数，各个城市为了发展，必然要大力争夺机械增长的人口，就是流入人口-流出人口的数量，其实就是优势城市抢弱势城市的人口。

而这背后的逻辑是，传统工业化已经遂声，人口作为生产者的功能落，作为消费者的功能则开对于城市而言，人口流入不再是一种负担，反而是巨大财富，是消费力和创造力，所以抢人会越来越激烈。

未来很多政策和新兴服务都会围绕人和人的消费来，而更偏向生产型的省份，比如江苏、山东，会面临内需不足的升级压力。这个判断，先放在这里。

接下来我就按照广东、浙江、北京、上海、河北、福建、江苏、山东的排

广东的人口增长主要得益于其数字产业的发达，这尤其能给年轻人创造很多高端就业机会，使得很多年轻一代愿意去浙江发展。今年以来，各大房企，不管是国企民企，拿地最多的城市，就是杭州，说明资本也看好杭州吸引年轻人带来的土地和房产增值。

北京、上海这两个直辖市情况比较相似，出生人口持续低于死亡人口，现有人口还不断流出。一方面是因为这两个大城市的老龄化太高了，60岁以上老人占比，北京是30%，上海则高达35%，说是老人之城一点不为过；另一方面，两个城市都是以金融业、服务业、大工业为主的产业结构，在当前环境下，普通年轻人很难获得跟京沪消费水平相比足够高的收入，所以很多到岁数就要返乡。另外京沪，尤其是北京，对常住人口的增长控制也是红线，他们想要的年轻人，真的是精英年轻人。

年轻人，收入和购买力都相对低一些。所以在当下的产业升级期，江苏依靠着工业优势自然能优先拿到红利，实现经济的高增长，但当未来，外循环一步步切换到大内需、大内循环上来的时候，江苏也需要有新的增长极，比如发展服务业，培育出更多能容易拿到高工资的年轻人，提升消费内需。

最后，山东的问题则比江苏还要严重一些，自然增长和机械增长都是负数，一年间少了近43万人，基本是全国垫底，没办法，重工业占比太高，省内人口则流向了青岛和济南。所以，大山东加油吧。
