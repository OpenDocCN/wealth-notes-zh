
# 普通人不花钱或少花钱，如何进行轻资产创业赚钱？没有高人指点怎么办？

## 前言

这个时代与过去最大的不同，是赚钱这件事情，对普通人来说，未必需要你大的投资。甚至你不花钱或少花钱，也可以日赚千元、几千、上万，这绝不是玩笑。说得直接点：这个时代，觉得必须得拿本钱投资才能赚钱的人，大多都不是聪明人。（当然，家里有矿、出生就含着金钥匙的人除外，我反倒赞成他们选好赛道，进行重投资，拉开与他人的区别）

这个时代，智慧、脑力和才华，才是一个人最大的价值所在，才是你在社会最有力的战斗武器。如果你不相信有可以日赚干元、几千、上万的事情，那只能证明，你从来就没有一天赚过这些钱，或者说你身边压根就没有这样的人。而不谦虚地说，我身边，这样的人比皆是。

所以，请不要一提赚钱、创业、做生意，你就想到租房装修开店、投资、雇佣员工这些字眼，永远不要用老眼光去否定你没见过的事物。。。这都是老掉牙的思维。只有被时代淘汰的人，才会有这样的思维。真正有头脑和智慧的人去创业和做生意，没本钱一样可以赚钱。因为，他们的头脑和智慧，本身就是资本。

说到这里，如果你觉得我说的有道理，这证明你还有救，证明我们还有缘分一路走下去。最可怕的，就是那些看到我的这些文字，就想上来抬杠、骂街的人。他们的口头禅是：“做生意不投钱，怎么可能赚到钱呢？”这些想证明自己永远正确，证明自己不是笨蛋和蠢货的人，他们永远不会意识到自己哪里有问题，永远不会意识到时代已经发生了变化，永远都是坐井观天，以自己的意志评判一切。

这样自我的人非常多，还比如一些原生家庭里，有些父母，他们永远不知道什么是对孩子的爱和温暖，他们只会在自己的世界里去考虑问题，考虑怎么让孩子永远都臣服于自己，怎么让孩子孝顺听话，至于孩子是痛苦还是快乐，他们根本不会考虑。。这样的父母，你是永远无法拯救的，唯一的做法，就是与他们只做有血缘关系中的物理尊重，而不必投入你的情绪和感情，说白了，父母有事给钱就行了。如果你不能调整好自己的思维，可能就会被困于这样的原生家庭，终生痛苦。

赚钱、创业、做生意，道理同样。如果你不脱离、不改变思维，就永远不会成功。所以，你不必拘泥于旧有的传统思维，要敢于突破，你才是崭新人生的开始，你才可以重新活一回。

最关键的是，你要有与众不同的思维，而且是必须要有，你才能脱离底层，脱离让你痛苦的原生家庭，实现真正的经济和精神上的独立。这也是我为什么在过往的文章中，总让大家提升思维的重要原因。

但可惜，我们身边大多数和人，只会听人安排做事，只会打工，对即使伤害他的父母也只会”愚孝”，只会在传统思维中行事不懂改变，而把“思维的提升“看做是鸡汤，自己一生困顿，还说他人的努力是无用之功。

所以，很多时候，我也特别烦一些人，张嘴闭嘴永远都是说：“我这辈子就这样了，改不了了。。。”而我希望，亲爱的各位读者，你不是这样的人。

如果遇到这样的人，无论是谁，你要给我彻底让对方从你的生活中消失。与什么人接触做生意赚钱、找什么人当老公、老婆，也从侧面大概率说明了你是什么样子的人。。。

此外，在这个时代，大家都在说经济形势下行，可是，还是如我之前多次说过的那句话：你看到的是经济危机，而我看到的，是很多人在日赚干元、万元。。。

为什么这样说？因为你看到的是，大多都是传统重投资的行业的不景气，和缺乏管理才能、懒惰、没有跟上时代形势的人；而我看到的，却是轻投资，以个人创业为主兼职为辅、每日废寝忘食、省掉很多中间环节的行业和人。或者那些虽然拥有重资产，却身价千万上亿的人，我清楚他们的思维方式。这就是你我看问题最大的不同。

而今天，我想和你主要聊的，就是跟你说说这不投资、少投资，绝不重投资赚钱的事儿。最关键的，是我要告诉你如何去做，如何实操？

## 普通人只会打工赚不到大钱根源在哪里

这么多年在商业布局、国学堪舆学、咨询师行业一路走来，我的每一步成长可以说中间走过了很多弯路，感觉耽误了太多的时间，真的是教训深刻。

我深刻体会到，若早年有幸得遇高人智者点拨，我此刻的人生、财富，或许早已跨越现今的边界。幸运的是，我凭借不懈的坚持与自我超越，终得以扭转命运轨迹。所以，虽然从事命理学工作，但我从来都只相信命由天定，但”运“在人为。

这些年人生的历程，无论权贵阶层还是中产精英，我见过了太多，这些都让我深刻领悟到：

对于出身平凡家庭的读者而言，缺乏高人引领，往往意味着要在摸索中历经无数曲折，才能窥见一丝成功的曙光。而可悲的是，自己身上存在的问题，自己并不知道。

比如，很多人觉得压力大、赚不到钱，就存在以下的情况：

一、机械重复打工上班，缺乏深度思考，也根本懒得去想，总说自己没时间进步和学习，实际上都是为懒惰找借口。

这些人的生活，似乎陷入了一种无意识的循环之中——年复一年地重复着同一份工作，很少停下来问问自己，“到底我现在走的路对不对？我怎么提升改变自己？”，更没对未来进行长远规划。

这种缺乏深度思考的状态，如同一辆没有目的地的列车，盲目前行，最终错失沿途的风景与机遇。

二、所做的工作，可替代性强，缺乏核心竞争力，也不能为自己今后的发展留下什么有积累的事。

他们所从事的工作，往往技术含量不高，容易被他人取代。

在快速变化的市场环境中，这样的工作稳定性堪忧，更谈不上积累有价值的事业经验和资源。

缺乏核心竞争力，使得他们在职场中难以立足，更别提实现个人价值的飞跃。

三、对今后人生的发展没有规划，也不懂得怎么去建立与他人相区别的壁垒。

这些人往往对人生或职场的发展路径一无所知，既缺乏纵向向上发展升阶的视野，也不懂得如何通过横向拓展来拓宽自己的职业边界。

他们不懂得如何在网络世界和职场中树立个人品牌，构建自己的护城河，从而在激烈的竞争中保持不败之地。

四、不懂商业思维，更不懂得商业变现都可以通过哪些方式。

此外，他们还普遍缺乏如何将自身的知识、技能转化为经济价值缺乏认识。

这种变现能力的缺失，不仅限制了他个人财富的增长，也让他们在职场或工厂之外，失去了更多可能的发展空间。
