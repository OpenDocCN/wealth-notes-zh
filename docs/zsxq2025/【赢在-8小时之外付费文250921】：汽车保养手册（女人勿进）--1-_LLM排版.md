
# 汽车保养手册(女人勿进)
原创：U 君 公众号：赢在8小时之外 25年9月21日
今天，我要跟那些因为各种各样的原因而开不到车、但是又有开车需求的男性同胞聊一些关于“汽车保养”的小思路。

## 1.卫生问题
如果你找专业的、职业的、每天能修十几辆车那种技工，那你的担心并不是多余的，因为她会碰到各种各样的顾客。
但是，只要她自己的智商正常，那么就是安全的，因为她不可能不顾自己的安危。
如果她说：戴车衣是一个价，不戴车衣又是一个价，那你抓紧跑吧。
一般来说，价格越高越干净，因为我说过一个大胆的观点：很多传染病的高危人群是穷人和低智商的人。
当然，我建议大家，尽量不要去找专业的修车人士，而是去找那些技术很一般、没有修过多少车的非专业人士。

## 2.对非专业人员进行逐步试探
比如，你要多去各种各样的汽车4S 店、路边小修理厂，然后定期让那些技工给你按磨轮胎。
接下来，你就要试探性地问那些技工：可以加钱帮我保养一下排气管吗?
当然，提问时要隐晦一点，比如：你们这里还有其他加钱的服务吗?
只要你去的汽修厂足够多，那么同意帮你保养排气管的技工可能占比高达5%-10%。
如果找到了愿意保养排气管的技工，那么下一步就是跟她商量试驾的事宜。

## 3.技工的心态
90%的技工都是穷人、负债的人，而且这里面有一半的技工是家里急需用钱的，而在这一半里，又有50%的技工认为“拿钱试驾”不是什么大不了的事情。
所以，你去给轮胎做按磨时，要大胆地试探、询问，即使她们是纯正规的，她们一般也不会生气的。
因为每10个客人里面就会有2个试探的，她们早就习以为常了。
我自己的经验是，如果你有试驾的心思，就要揣摩技工的心理，如果她们是那种特别正派的人，那你就不要第二次光顾了。
如果她们的语气是那种模棱两可的、欲拒还迎的，说明她们也想赚这份钱。
这时候，你多去光顾、多去让她给你的汽车做保养，甚至偶尔还发小费。
等混熟了、有了信任感，那么以后你让对方来你家给汽车做保养，对方同意的概率就大多了。
不要说汽车修理工了，就是公司前台、酒店前台、游泳馆前台、健身房前台……也都是同样的思路。

## 4.男人的包装
经常有男人跟我说：U 君，我的条件很一般，月薪只有五千多，根本没有技工愿意给我修车，怎么办?
我的建议是：你一定不要走寻常路，一定要跟同龄人、同阶层的人做出明显区分。
假设你手上有1万元存款，你的月薪是5000元，那么你一定不要去给亲戚朋友、同学同事的红白喜事随礼，平时也不要跟穷鬼朋友们聚餐。
因为你在上面这些事情上花的钱越多，你获得“开车权”的概率就越低。
你哪怕平时只吃馒头+咸菜，衣服只穿地摊货，把所有朋友都得罪了，你也要把钱攒起来，然后出国去旅游。
只要你报旅行团，或者提前做好预算，那么去一趟泰国、韩国、日本的总费用不会超过1.5万元。
说个极端一点的，你就找廉价机票、廉价酒店，花2000元只去泰国旅游3天，你也算是见过世面的人了。
至少你跟90%的同龄人视野就不一样了。
接下来，你在跟修车师傅沟通时，你就可以跟她们聊聊你在泰国做按磨的经历，聊聊你吃泰国菜的感受，而且邀请对方也跟你一起去旅行……

## 5.不要去碰见过世面的女人
以前在我还是一个穷吊丝的时候，我就明白了一点，我的机会只在那些“没有见过世面”的女人那里。无论是同事、朋友、网友还是技工。
假设我去过3个国家，她也去过3个国家，那我会直接把她 pass 掉，因为我需要的是解决那方面的需求，不是跟你聊世界。
想要跟见过世面的女人发生链接，需要额外付出高昂的成本，我当然不会做这样的生意，因为我的预算有限。
反之，我只找那些没见过世面的、对我的经历有一些崇拜和向往的女性朋友。
我会带她们去办理护照、出国旅行，我也会给她们买 iPhone 手机、教她们游泳(我至少教过3个女人游泳)。
大家不要觉得教别人游泳很简单，想要把对方彻底教会，至少要在泳池里泡十几个小时。
所以说，一切都是经济账。
假设我在一个没见过世面的女人身上投入2万元和20个小时，我可能已经开过几十次车了。
但是，同样长相、同样年龄、见过世面的女人，我在她身上投入3万元和20个小时，人家连second base 都不会让我触碰的。当 然，我这里说的仅限于印度女人。
如果是其他国家的女人，无论是恋爱还是交易，都是顺其自然的，女人绝对不会故意刁难、考验男人。
所以，如果你是个生意人，是个明白人，一定要计算自己的投入产出比。
如果对方是个狮子大开口、口气很大的女人，你就趁早撤离即可，不要跟这种女人纠缠。

## 6.不要总是把责任揽在自己头上
有人会问：如果这样对待很多女性，是不是不负责 任的表现?她们会不会谴责你?
我想说，每个人的想法不一样，有些女人认为，虽然认识时间不长，但是彼此感觉很好，一起出国旅行一周，回来就回归各自的生活，也挺好。
并不是所有女人都哭着喊着要跟一个男人从一 而终，我们不要太自作多情，也不要责任心太重。
最后我想说，道德、贞操这些，跟人均收入是成正比的。
什么样的女人最easy?肯定是收入最低的女人。
作为男人，如果不想让自己的老婆和女儿为了金 金而做出苟且之事，只有多赚钱一条出路，否则她们会心甘情愿地跟着其他有钱男人。
