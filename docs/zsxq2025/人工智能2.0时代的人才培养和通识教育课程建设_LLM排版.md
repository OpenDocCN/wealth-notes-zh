
# 人工智能2.0时代的人才培养和通识教育课程建设

## 人工智能的前世今生

人工智能0.0：古代（1956-2006，从规则到学习）
- 传统（知识+规则）：专家系统（知识库+推理机）
- 现代（数据+学习）：机器学习（模型、目标、策略），数据模型（IID，用数学模型模拟世界）
  - 常规统计学习方法：逻辑回归，决策森林，支持向量机，马尔科夫链，……
    - 小数据，人工特征，部分可解释，缺乏通用性和跨模态
  - 人工神经网络：与人脑最大的共同点是名字（原理、机制和架构并不一样），用神经网络表达数学模型
    - 传统神经网络：霍普菲尔德网络，玻尔兹曼机，……
    - 深度神经网络：深度学习（Hinton，2006）
      - 大数据，特征表示，基本不可解释，具备通用性和跨模态

## 大模型的工作原理: NTP（Next Token Prediction）

1. 收到提示词
   - 示例:“今天天气不错，我决定”
2. 将输入拆分为token
   - 准确地讲，这里不是“字”，是[“今天”, “天”, “气”, “不”, “错”, “,”, “我”, “决定”]“token”，可以进行语义计算。
3. 采用Transformer架构处理token
   - 理解token之间的关系
   - 识别提示词的整体含义
   - 概率预测+文字接龙
4. 基于上下文预测下一个token
   - Transformer：一种深度神经网络架构
   - 自回归（AR）：重复步骤4和步骤5直到形成完整的句子
5. 根据概率分数选择标记
   - 示例:{“去”:0.7, “停”:0.2,“站”:0.1}

## 大模型的工作过程：预训练-后训练-推理

- 预训练 强化学习
- 监督微调
- 接收输入 处理输入 进行推理 生成输出
  - （自监督） (RLHF等等)
  - （提示词） （上下文） （测试时计算）
- 阶段1：模型训练（预训练+后训练）
- 阶段2：推理

      - 显著提升推理能力。
