
# 丁辰灵星球 250904 问答精华汇总

提问：你好灵哥，美国降息前后有色板块是否有可能因为预期落地有个短期的回调？要不这几天看情况减点仓？然后中长期依然看涨

提问：灵哥今天军工大跌是不是因为靴子落地就大跌？那 17 号如果公布降息，会不会也是靴子落地？相关的有色等板块也可能大跌呢？
丁辰灵回答：降息和阅兵这种纯事件还是不一样的，大 A 每次都是这样，去年航展也是这样，结束就跌了。买预期卖现实，但美国降息恰好是要落到现实才会让全球资金发生真正的改变。原因是上次我直播说的全球大部分人其实不关心财经。
