
# 下一个低点时间预期(福利版)

## 前言

今天很惨烈，可以说是跌得稀里哗啦的。小指数当中国证2000跌幅就近4%, 大指数显然算是比较强的。

## 强势行情的特征

强势行情最大的特征是什么?我说出五点，你记在心里：

1. 暴涨暴跌经常会遇到，偶尔突然一根长阴线跌下来，极容易一步到位，顶多后面再出现缓跌再到止跌见底，整体的调整周期并不是特别长；
2. 暴跌后，通常未来都会对应一根标志性的反包长阳线，眼光不是要你看眼下，而是放眼看未来。因为波段行情并未结束，故此调整下来依旧是倒车接人的机会；
3. 比较适合调仓换股，比如今天暴跌的人形机器人，我曾多次强调不要追涨追高就是这样的逻辑。也就是说未来找到低点，你完全可以调仓到低位补涨的品种当中；
4. 有现金仓位的面对这种大跌无疑是千载难逢的机会，用两种方式来操作： a、越跌越买，摊平成本价格； b、还是找到调整的低点买入；
5. 多去观察下周一反包的品种，这类股极可能是未来比较有潜力的对象，要是能够符合低位补涨的主线，那么跌下来就是给大家送钱的机会。

## 下一个低点时间预期

很多朋友今天大跌后，追问最多的问题是这次调整低点在哪里?我本文就提前解密，但提前说清楚，这只是初步推算并未确认，未来还是要以我的实时发文提醒为准。

就拿国证2000来说，我认为产生低点的时间最晚在3月12-13号位置，前后偏移不会太大，预计调整8个交易日左右。

## 上证指数的时间预期

上证指数从时间上来看；调整结束的时间在3月5号前后。预计它的调整时间和空间不会有国证2000那么大，但国证2000未来上涨空间和时间却仍然比上证指数要更大更长一些。

## 操作建议

国证2000时间看3月12-13号，上证指数看3月5号，这倒不是说国证2000一定会跌到12-13号。我开篇就说过，强势行情容易出现单日大跌，可也很可能一步到位，故此进入到下周初大家应该逐步买回来，四个字："逢跌低吸"准没有错的。

代码经历了单日大跌的风险释放，相对低点区域已经相对安全了，搞清楚这一点就足够了。至于3月5号和3月12-13号，那个时间会成为指数的低点，那个指数会在哪个时间构造低点，这是不确定的。你只需要记住一点，到了3月5号前后，盯紧我的文章提醒，否则就等3月12-13号。起码看完这篇文章心里面有个预期低点构造的时间，不至于心中没底。
