
# 石建军：3494点波段调整的空间和时间

石建军，笔名石天方。中国资本市场第一代证券投资分析家，九三学社上海金融委专家委员会成员，华东师大国际商务硕士学位专业兼职硕士生导师，上海交大金融学专业经济学学位。1990年进入证券市场，1994年为上海证券报专栏作者，中央台财经评论嘉宾，香港文汇报财经版特约采访嘉宾，水晶球财经网2018、2019名博评选第一名。

擅长用经济周期理论结合波浪理论预判大势，擅长结合国家产业政策导向和行业发展前景捕捉投资方向。创立和实践投资操作三格言：1、明大势者赚大钱；2、主流品种决定投资收益；3、价值投资选股、趋势操作选时。

具有极为丰富的证券投资管理经验，对国际主要股市和国际商品市场的运行规律有着深刻的研究，极为准确地预测了中国股市多个历史性的重要顶部和底部。

本周大盘的短线反弹仅仅维持了四天。周五的向下阴线，技术上蕴含着较为强烈的再下台阶信号。上周五的文章中，公开了在《3494点的回调是短线的还是波段的性质》这篇极为重要的专栏文章中提示的两个调整目标位置：“其中一个结构的预案是，大盘向下调整的目标首先指向3200点，但暂时不会去考验3152点。另一个备案是，3494点开始的调整目标将直接指向3087点缺口方向。”

从目前A股的走势看，上证大盘直接运行第二个预案的可能性已经大大增加。即：大盘极有可能将在封闭3087点周线缺口之后才能真正有机会去酝酿下一个向上回升的波段。否则，多方很可能将难以组织起没有后顾之忧的像样反攻。

阅读我所有文章的粉丝投资者在我年前撰写的年底特刊《对2025年A股大盘的前瞻性预判》一文一定注意到了我的一个重要观点：“从历史的角度来观察，每次波段行情中第一次跃上年均线，其后几乎都会有一个回归年均线的动作。”目前，这根年均线的位置是在3062点。理论上，大盘调整到这根年均线的附近，是有技术支撑意义的。但请注意，这并不等于说，调整最多只能在3062点。历史上，弱势市场中，向上突破年均线后的回落，多次是跌破年均线后再拉起来的。

我注意到，有投资者留言，担心本次调整是否会直接指向2900点方向。首先，我在之前的年度展望中已经谈到，2025年上证大盘的极限低位是可以指向2900点的。但我认为目前的这个调整，不太可能直接指向2900点。理由是：由2021年12月的高点3708点与2023年5月的高点3418点的连线构成的中期下降通道上轨压力线，目前的位置是在3030点附近。如果本次大盘的调整直接破位3000点回到原来的中期下降通道里，那就意味着9.24行情花大力气打出的下降通道的努力完全失败。那其后的调整目标就不仅仅是2900点那样简单了。换言之，即便以后出现重大黑天鹅事件导致上证指数再度考验2900点，但至少，目前这个波段调整，是不太可能直接破位到3000点下方的。【参见下图】

由此可见，本次由3494点开始的这个调整波段，最有可能的止跌位置，是在3030-3087点的范围内。停顿在3100点附近或者破位到3000点下方的概率都不是很大。因为，前者，未跌够有后顾之忧。后者，下跌超预期有大问题。目前看来，都是小概率。

之前，还有投资者留言，问究竟还有没有春季行情。我可以明确地回复：行情还是会有的，时间就在春节后。但是，有一点不得不提醒：不要对春季行情的空间抱太大的期待!

为什么？因为季线！投资者是否注意到，继上季度的那个放量高位大墓碑阴线之后，今年第一季度的季K线目前为止还是阴线。这根2025年的第一根季阴线的高点在3351点。这就意味着，若没有极其重大的利好消息刺激，今年的春季行情，基本是收不了季阳线的。换言之，节后的春季行情，3300点就已经构成重大阻力。

如果我们假定本次调整的止跌位在3050点附近。那么，整个春季行情的指数反弹空间就不到250点。这就意味着，春季行情中的上证大盘最多回升8%。所以，所谓的春季行情，大概率只是一个次级反弹行情。（其后是否还有更大级别的行情，以后再讨论）

可以想见，在预期只有8个点的春季指数反弹行情中，要想收获超过10%以上的整体性收益，唯一的办法就是找到优势板块。

我们注意到，在本周只有四天的这个短线反弹中，如果没有介入到人形机器人板块或芯片领头股，那基本就只有亏损的份。但是，就在这短短几天不到2%的指数反弹中，人形机器人的多只领涨股就都收获了20-30%的涨幅。

所以，尽管节后的行情预期只有8%，但领涨板块上涨30-50%以上是完全可以预见的。

那么，节后春季行情有赚钱效应的领涨板块会是哪个板块呢？毫无疑问，必定还是在新质生产力中。

机器人节前表现不错，节后机会就相对有限了。活跃资金会不会以芯片半导体为主攻方向？不得而知。但从国家的政策导向来看，只要是科技股的行情机会，芯片半导体，是不会缺席的！

另外，国际黄金价格始终在上升通道中，但近半年以来，A股的黄金股表现却并没有明显同步。这其中，是否也酝酿着机会？

实际上，方向性目标还是比较明确的，就是行业的几个领军企业。但不急，春节前以及春节期间，还有近一个月的时间。目前要做的，只是继续观察和跟踪。一旦时机成熟，果断出手就是了。
