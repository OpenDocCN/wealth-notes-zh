
# 李嘉诚闪卖全球43个港口的背后

## 交易主体和核心标的

贝莱德是全球最大的资管公司，掌控着超10万亿美元全球资产。其创始人芬克与民主党关系密切，但公司能穿越近37年美国政治周期并持续壮大，足见其身段的柔软。历年来，美国政府制裁国家时，贝莱德等犹太财团常会抄底捞钱。

长江和记实业注册地在开曼群岛，掌舵人李泽钜为加拿大国籍。从历史轨迹看，其英资背景更大，一直秉承在商言商的态度。我国外交部对此交易也未做评论。

本次交易核心标的为巴拿马运河区域的5个核心港口：巴尔博亚港、克里斯托瓦尔港，由长河实业、新加坡国际港务集团、台湾长荣集团和曼萨尼略国际码头公司运营，贸易量占7成。

## 交易达成过程

自川普去年底以"国家安全"为由对巴拿马运河施压后，各方开始谋划方案。今年1月，贝莱德创始人芬克发起交易，亲自向川普及财长、国务卿做简报，并拉拢花旗CEO考伯特和地中海航运参与。全程采用"王对王"的沟通方式，保密等级极高。

3月4日深夜，长河发布公告，宣布与贝莱德财团达成原则性意见，出售除兔子大陆和香港外的涉及23个国家的43个港口，总价值228亿美元。

## 交易出人意料之处

1. 交易达成的速度之快：从川普威胁算起仅2个多月。
2. 交易标的范围之大：涉及199个泊位及配套系统。

## 长河急于清盘销售的原因

驳斥川普与李嘉诚存在私人恩怨的说法：上世纪90年代初，川普曾向新世界集团化缘，而非李嘉诚。理由有三：
1. 李嘉诚低调务实，与川普作风不同。
2. 李嘉诚偏好成熟现金流资产，曼哈顿滨河南地块属于高风险标的。
3. 当时李嘉诚主要精力在欧洲基建、港口投资和大陆生意。

## 交易背后的三个原因

1. 川普的极限施压：巴拿马运河具有交通咽喉价值，且曾是美国的"私产"。川普多次威胁要收回运河主权，并给李嘉诚划道："要么卖给美国公司，要么等着资产被没收。"
2. 巴拿马政府的变脸：突然审计指控长和"违法经营"，要求重新审查经营权，并配合川普炒作"兔子控制运河威胁论"。当地政府一边查账，一边暗示"只要卖给美国公司，所有问题都不是问题"。
3. 李嘉诚的"金算盘"风格：
   - 全球经济衰退、贸易保护主义加剧，港口业务已不算优质资产。
   - 长江和记是李嘉诚核心资产，港口业务一年贡献8亿美元利润。
   - 通过交易套现190亿美元现金，相当于把未来20多年的利润全部揣兜里。
   - 卖掉部分港口后，长和市值约200亿美元，在手现金可抄底其他资产。

## 交易疑虑

为何不只涉及巴拿马的2个港口，而要"清盘销售"呢？

1. 财务角度看，这些被交易的港口利润率非常低。
2. 政治看，贝莱德通过这桩交易，将开启整合覆盖全球航运资源之路，甚至拿到国际航运定价权，未来可能对兔子形成骚扰。

## 后续观察

这桩交易目前尚未板上钉钉，后续变数及影响仍需观察。长河的操作站在企业角度或资本角度看虽无问题，但却给国际航运带来巨大隐患。
