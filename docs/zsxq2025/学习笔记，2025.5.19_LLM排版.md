
# 学习笔记，2025.5.19

1，所谓的悲观，所谓的乐观，取决于一个人内心的强大与否，你强大你就是乐观的，你弱小你就是悲观的。如果你是小白的话，进军一个项目，你其实要做到两点：第一、屏蔽弱者的声音。第二、关注强者的声音。因为弱者不管做任何项目，声音永远是抱怨，这个不行那个不行，这个麻烦那个麻烦。而强者哪怕他是小白的时候，他也不会发出抱怨，他只会默默、扎扎实实的一步步把它做好。

2，其实人最好的状态就是销声匿迹，半隐半藏，除了赚钱以外，平常是不需要有太多的社交和喧哗，我们需要的是安静和独处，不要被外界打扰，享受生活，低调清醒。

3，如果你的磁场足够强，福报足够大，那么谁欺负你谁倒霉，这一点不夸张，因为他欺负的不是你，而是你的守护神，你可能会觉得自己没有什么本事，但是你要记住，你的守护神有很大的力量。

4，人活着就要做一个有锋芒的善良人。遇强则强，遇弱则弱，遇善则善，遇恶则恶，该翻脸时就翻脸，才能让人看清你的底线。

5，贵人是相对的。比你高一两个职级，比你多赚个四五倍，还愿意带你玩，愿意指点你的，就是你

的贵人。多思考一个问题：贵人凭什么要帮你？很多人说是价值交换，其实我不太认同。能形成价值交换的那不叫贵人，那叫生意伙伴。愿意指点你的贵人，应该是从你身上看到了一些自己的样子，比如你的勤奋，你对某些事的坚持，闪光点等等，同时这种指点对他来说也不需要付出很大成本的。君子之交淡如水，小人之交甘若醴。贵人指点，反而是非价值交换。

6，人大附中崛起的唯一因素是生源。另一个证明，就是我亲眼目睹二十年内，回龙观的昌平二中从垃圾变成名校。这事说明了什么？名校=基因×努力。这个努力，还不是家长的努力，还只是这个孩子自己的内心驱动力。学校的作用，最多是把北航苗子催肥成清华学生。但到了清华，又一定会现原形的。所以，不要把高估任何鸡娃的效果。要坚信基因决定90%的结果。这样，才能以正确的方式鸡娃。鸡娃也要守正出奇，你是打工人，但绝不能以培养一个优秀的打工人的套路继续鸡娃。打工是可以失业的。失业后能继续可持续发展才是教育的本质。培养想法+有办法的人，才是整个教育的核心。

7，潜力股其实是有苗头的。潜力股从来都是绩优股，一般很早就体现出了与众不同，很早就开始自己各种折腾，或者直接创业，或者各种搞钱。即便是大器晚成的人，也是很早就在不断折腾了，只不过到了某个阶段终于厚积薄发了。

8，鉴定一个人够不够成熟，有一个很重要的标志：在事情进展得不那么顺利，自己表现得不那么好，或者情绪不那么稳定的情况下，不该做的、不该说的能忍住，该做的事情依然能继续做下去。

9，年纪越大越喜欢那些直截了当的人。行就行，不行就不行，彼此坦坦荡荡，爽爽快快。我是真没时间和心思，去胡乱猜想一句别人没讲清楚的话了。谁要是敢跟我磨磨唧唧、阴阳怪气，我直接就拉黑了。

10，情绪波动大的时候，最应该管住的是自己的嘴巴。这不仅会保留你的神秘感，增加别人的敬畏心和探索欲，又能保护自己不会陷入到是非中。很多人际关系崩塌、项目失利，都是因为祸从口出。所以，当你累的时候、烦躁的时候，把嘴闭上。出去溜达一圈，缓一缓再说，有奇效。
