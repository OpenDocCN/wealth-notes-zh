
# 两大尚书对调

今天聊两个事吧。
一个是吏部尚书和统战尚书对调。
干结和泰丰对调。
这个毫无疑问，是削弱干结的。
甚至听到一个说法，干结可能有问题。
我们本来是想河南楼接吏部的，但他倒霉没能上去，
就退而求其次让干结上。
干结在山东和夫人搭上线，属于可以信的那类人。
但其实，也不是完全信任。
因为他并不是完全体的吏部尚书。
也就是说，这个干结其实是一个不完整体的吏部尚书，权力是被制衡的，关键重要的人事决策，还是交给前任陈拍板。
这就其实已经埋下了伏笔，部分信任，未完全信任。
现在看，我们对泰峰的信任度还更高。
因为泰丰交过投名状。
我们对内蒙倒查三十年剑指花花的时候，泰丰就是内蒙一号。
泰丰是和我们并肩战斗过，跟着我们往死里得罪过人。
所以后来因为内外大环境变化，我们提高统战部地位职权后，我们想到的就是让他管统战。
现在对干结进一步不信任后，又把这个位置交给了泰丰。
其实泰丰是不太愿意对调的。
因为他知道，吏部的前任部长才是实质性的掌控者，
虽说吏部大于统战部，但他去吏部当这个名不副实的
一号，真实权力是小于统战部的一号的。
吏部尚书是名不副实的，唯一有用的，是能够入局的
特性。
是的，很多吏部尚书都曾入常，成就大长老金身。
而统战部是没有入常的管理的。
但泰丰不在乎，因为他69岁了，已经来不及了，干结才会追求这个位置。
因为干结61岁。
61岁的他如果还留在吏部这个位置上，他有可能在下
个五年入常。
69岁的泰丰调换到吏部一号这个名不副实的位置上，
对他个人来说权力变小了，而且他年纪多大没有入常
的可能，他是为了服务大局而牺牲自己的。
所以，这次对调新闻，看起来两个人都没有亏。
但本质上，是泰丰小亏，干结大亏，干结直接出局。

干结已经不可能入常了，甚至可能过个几年就退二
线，64年啊可惜。
你看看，干结会不会落马？
很难落马。
你看看枪杆子管用人的苗落马后，牵出多大的风波。
管用人选人的人落马，倒霉的不是他一个，而是所有
经过他手里审批通过的人事任命，都要重新审查一
遍。
那将牵扯上百号省部级干部，会人心惶惶局势动荡。
所以，干结就算应该落马，也不能落马，先对调抽离
出吏部，入常竞赛先出局，后面再缓缓降落，他不能
落马也不会落马，安心，不会在政界复制军界的大动
荡了。

至于干结这个和夫人有关的人，怎么会出局，就说来
话长了，后面有机会再聊吧。
然后是今天的植树新闻。
之前每年这个时候，两个枪杆子的左膀右臂都会出来
的。
但是，今年卫东还是没有出来。
只有一个侠。
这不是好消息，所以可能，卫东真的不妙。
而侠貌似依旧，岿然不动。
