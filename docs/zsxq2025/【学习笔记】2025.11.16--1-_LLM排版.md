
# 学习笔记 2025.11.16 (1)

1. 想要彻底改变自己，只需要一年。一年的时间不算长，却足以改变一个人的
一生。你可以开创一项事业，并让它走向成功；也可以坚持运动，锻炼出好身
材。只要你有实质性的付出和坚持，没有什么事情是一年时间完成不了的。
2. 你每天应该做的事，是让自己变得更加有价值，自己的产品或者服务能卖个
更高的价钱，自己的受众更加精准一些，粘性强一些。
3. 学校不会教你搞钱的学问。搞钱的学问，要么有贵人带，要么自己开悟。这
两个的大前提都是要多接触外面的世界，多跟别人交流，不要沉浸在自己的小圈
子里。
4. 能花钱解决的问题，绝对不花时间。那些从来不想花钱解决问题的人，他的
问题一定会越来越多，永远生活在社会的底层。
5. 成年人之间如果感情出现了嫌隙，发生了冲突，大概率是因为钱没有谈妥，
分赃不均。一个成年人对你最大的认可，就是愿意为你掏钱。
6. 做自媒体，必须要以一个输出者的角度，去看热点新闻，去研究，去喂食那
些喜欢在网上发表意见的屌丝。如果你悟透了这一点，你的收入会直线上升，当
然朋友也会越来越少了，因为你已经看不起那些浪费时间的人了。
7. 许多时候，幸福是需要付出代价的，美也是需要付出代价的。有些人以为生
活就是波澜不惊的样子，其实人生的底层常常是真刀实枪的对决。
比如钱和性，这种涉及生存繁衍等根本利益的事情，都需要你变得优秀、聪明或
强悍。
8. 你不可能在所有的事情上都出类拔萃，那样只会浪费时间和精力。如果你想
成功，就必须在欲望上有所取舍，必须学会专注。
9. 灵感是易逝的，必须立刻行动。所以当你有灵感去做某件事，就马上去做。
如果想发一条推文，就立刻发；如果想解决一个问题，就立刻动手；如果想读书，
就立马去读。在产生好奇心的那一刻去学习是最有效的，那才是真正的学习，
真正留下的东西是你在真正渴望去学的时候学到的。
10. 是费尽心力集中意志还是使用自由意志改变生活。
一定要相信自由意志，不要相信使用意志力“自律”和努力，人的注意力是有
限资源，用自律只会短时间耗尽一天的精神储备让这一天报废，通过循环游戏
引导自己的爱好到自己需要做的事上是最有效的。
