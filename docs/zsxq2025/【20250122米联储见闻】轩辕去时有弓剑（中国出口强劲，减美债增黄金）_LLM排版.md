
# 轩辕去时有弓剑（中国出口强劲，减美债增黄金）

今天中国发布了今年的经济数据，赶在川普下周一上台之前，5%的目标，不多不少，完成任务，当然这个数据如此准确，肯定会有人有想法，但现在市场心里都明白，数字已经不是经济的状态的表述，而是管理层的态度与预期，经济冰火两重天，内部各种吐槽声音，但是出口量却是破纪录的大幅攀升，顺差更是创历史纪录，中国经济到底怎么样，其实看美国的态度就知道，即将上任的美国国务卿卢比奥这几天忧心忡忡的表示，美国如果再不改变，10年后，什么都要看中国脸色了。美国媒体对于中国的出口数据和顺差记录也是忧心忡忡。

中国海关总署统计分析司司长吕大良在周一的新闻发布会上表示，如果把中国去年出口的电动汽车和插电式混合动力汽车首尾相连，可以横跨亚欧大陆、从北京排到罗马。

顺着这个逻辑我都有点迷糊了。大家要注意一点，不管是去年的经济会议还是政治局会议，促进内需都是今年的重点工作，中国这几年经济下行减速的过程并不是为了慢慢停下来，而是为了换动力车头，换轨道，从低端制造业房地产支撑的国民经济转向高端制造业消费服务支撑的方向，所以经济上会有刹车减速腾笼换鸟的动作，这是一个过程，今年是中国制造2025的收官年，虽然在美国开启打击围剿中国制造2025之后，中国官方已经不再提这个事，但官方不提这个名字，不意味着当年制定的这个计划没有按部就班的推进实施，大家最近看到陆续发布的各种成果，包括小红书逆转天罡的操作，都不是从天而降的，而是一步一个脚印积累出来的，所以从高层制定计划定位今年为扩张内需年，到扩大免签国家范围，再到一系列成果的发布，今年会是确认车头转换完成，新轨道建设完成，开始住燃料加速推进经济朝新方向新层次发展的起点年，也就是十四五的收官战，十五五的开张年，这也是为什么政府会在财政上松开预算，加满燃料，大力推动的原因所在，我们从管理层的动作可以反推出有些布局已经完成，有些调整已经到位，今年中...

下周一，川普将正式上任，开启他时隔四年后的第二任期。川普从第一任期到这次竞选，思路都很清晰， 一是希望通过增加关税来促进制造业回流，并减少美国中产和民企的税收；二是保守的移民政策。但是从美国媒体最近释放的各种信息来看，制造业难以回流的根本原因是美国整体产业生态被破坏了，现在要重新扶植生态，而中国的低成本大战让这个生态很难健康的长起来，而且相比于川普的第一个任期，经过8年的努力，中国在全球制造业的占比不但没有下降，反而进一步提升，虽然转移出去很多产业，但最后发现生产什么最终都还要从中国进口零配件，美国一家自动车生产企业在过去几年迁回了美国生产，但是老板接受美国媒体采访时坦承，除了组装在美国完成，几乎所有零配件都需要从中国进口，而相比于以前直接把单车在中国生产好整车进口，现在除了成本高，工期长，交货不稳定，没有任何市场意义的竞争优势，他认为这种方式的竞争不能让美国制造业真正获得生机，是完全不符合市场规律的操作，另一方面中国原本是全球继美国之后最有潜力的消费市场，却因为美国方面的强势阻断，导致中国消费力下降，进入通缩状态，要知道日本通缩对全球经济的影响非常有限，而中国进入通缩，实际上会导致全球经济发展后劲不足，中国市场在进口方面的数据持续下降，也是中国顺差不断创纪录的原因所在，所以对于川普上台后依然靠关税来强行回流制造业的操作，很多美国企业并不看好。

最让美国人担心的是，目前中国已经对全球超过100多个国家实施免签，小红书的中外对账已经把中国的低物价传到四海皆知，中国还在以打击电诈为契机减少国民前往周边国家消费，接下来，如果美国开始挥舞关税大棒，人民币一定程度的贬值会在高层能够承受的范围内，一个全球价值洼地就这么华丽丽的闪亮登场，因为俄乌冲突所占据的空域优势，已经让中国航司在国际航线方面吃到红利，小红书的加速推广宣传，免签政策的鼓励，外国人到中国花钱会成为今年政府主导的工作方向，且汇率一定程度的下跌也会抵消关税对中国商品竞争力的影响，甚至出现我昨天所说的，刺激中国代购，洋倒爷的大量出现，因为征收高额关税，以及禁止进口，必然在中国和这些国家之间形成价格差，只要中国免签的大门常打开，这个价格差就会刺...

在金融方面，中国目前的做法是一增一减。减持美国国债，大量增持黄金。增持黄金的目的其实就是为人民币一定程度的贬值提供支撑。中国长期以来是全球美债的最大持有者，最高点在2013年11月，持有量达13167亿美元，距今已经是12年前的事情了，到去年11月，中国持有的美债已降至7600亿美元，创下15年来最低，比最高点减少了40%,要让手里巨额美债不贬值，还能卖出好价钱，中国也是费劲了心思。这些年为了阻止中国一以贯之的抛售美债，耶伦老奶奶跑了多少趟，最终还是拦不住的。在过去两年里，央行增加了16.6%的黄金储备，这意味着抛售出去的美债都大量的换成了黄金，央行用实际行动重新定义了全球资产价格，除了明面上的增持，中国还通过一系列隐秘交易购买黄金，实际增加的储备可能远不止这个数字。

在川普当选之后，中国还快速的做了外交安排，迅速的解决了和印度的边界争端，并且利用川普当选快速修复和日本的关系，以至于中日关系出现了短期内180度的掉头操作。去年日本外相访华是为了深圳孩童被刺案兴师问罪，而这个月日本派遣外相访华，已经开始为2020年就没有成行的两国元首互访进行外交铺垫。除了印度和日本，中国还与印尼进行了外交互动，印尼名不见经传，但却是世界人口第四大国，是东盟第一大国。除了继续敲打菲律宾，中国已经尽可能把周边的朋友搞得多多的，麻烦减到最低。中国在川普当选后在所能准备的领域都做了各种准备，围堵就是没有接触川普团队，没有对川普当选释放任何一点信息和信号，保持着你不动我不动的状态，即便川普主动邀请领导参加就职典礼，甚至点名蔡奇参加，但是中国最终还是按惯例派出了韩正，甚至没有排出在外交上更能深入对话的王毅，显然，中国不希望在川普没有做出任何政策表态之前，与川普有任何正式的外交领域的接触，不给川普团队任何提前摸底试探的机会，保持着高度戒备和底线思维状态。这会让川普当选在政策的实施上更加谨慎，中国这种随你怎么样，我就这个样的操作，加上先进武器眼花缭乱的登场，确实让川普在当选后的三个月对中国的言谈和态度越来越谨慎。
