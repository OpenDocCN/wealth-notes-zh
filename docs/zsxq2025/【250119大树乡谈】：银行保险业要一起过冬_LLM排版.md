
# 银行保险业要一起过冬

中国金融体系核心是银行，大多数信贷来自于银行体系。正因此过去对银行非常维护，总是优先确保银行利益，这也是延续了上世纪90年代以来的经验，尤其体现在照顾银行维持足够的净息差，总是担心如果银行利益受损就会采取消极对抗，反而影响稳定和发展经济。

所以过去十几年，在2023年以前，银行系统最承受压力的也不过是2020年面对冲击，要求银行为主的金融系统全年向企业合理让利1.5万亿元，具体办法包括延期还本付息、发放普惠小微信用贷款、减少银行收费等措施，最终也实现了。但说到底，对银行来说，合理让利，是救人更是救己，如果在危机冲击下，银行仍然选择不让利，那么实体经济受损最终一定会危及金融本身。服从国家安排，在充分考虑基础上合理让利，金融系统的利益也能得到最大保护。

因此，实际上2023年以前，银行系统压力并不大，也就是面临第三方支付等小小的压力，但后来也基本解决了。但时代不同了，2023年金融会议要求金融系统必须讲政治性、人民性，这是金融工作的基本原则，原话为“坚持以人民为中心的价值取向”“深刻把握金融工作的政治性、人民性”,如果丢了这两大基本原则，那么金融系统本身的强大并不是好事，金融强国并非金融自己强，而是要求金融服务强国目标，服务实体经济，而不是片面维护金融系统自身利益。

于是掀起了对金融系统的全面整顿，通过改革监管体制等约束金融系统真正让利。于是从2023年开始，商业银行净息差纷纷突破原本1.8%的警戒线，目前净息差已经压到1.4%-1.5%,而未来还要继续降。

银行业缩小净息差，也冲击到了保险业，出现了利差损，也就是保险资金投资收益率低于保险合同约定的预定利率而导致的亏损，主要出现在寿险公司中，过去寿险投资的主要标的就是银行这类稳健理财。这就让银行和保险很难受，很难继续躺着吃红利。

这一改革难度其实很大，因为长期以来，金融领域都是安置关系户的地方，能够合法的获得丰厚的回报，工作稳定、社会地位高、收入高，几乎完美。但这种模式是低质量的吃租金，并不符合未来高质量发展战略方向。

所以本周金融总局召开2025年监管工作会议，确定了2025年的重点任务，相比去年，最大的突出变化就是提出“切实提高银行业保险业高质量发展能力”。这句话很委婉，看起来的像是“金融强国”勉励金融业一样，但实际上是对银行业保险业提出了更高的要求，“高质量发展”意味着必须告别过去躺着吃利差的做法，金融业不能继续作为旱涝保收的避风港，也要出去接受风雨的洗礼。

报道中也提到“督促银行保险机构强化资产负债联动管理，苦练内功、降本增效，着力夯实可持续发展基础”2025年提出这一要求压力很大。因为这几年全社会投资收益率都在下行，所以大量资本才会不顾风险疯抢国债，这已经对银行业和保险业带来巨大冲击，而去年银行净息差、保险利差损问题已经压力大增。

目前看，短期内全社会投资收益率低以及继续推动金融业让利这两大核心不会变，那么银行业保险业也只能向内求，去“苦练内功”，自己强化“资产负债联动管理”，但提高效率何其容易，尤其很多中小机构能力非常有限。

所以当前的重点转到了“降本增效”，这很可能意味着银行业要加速整合、出清，现在网络非常发达，继续保留大量线下网点已无必要，而过去安置在两大行业的关系户们也到了该走的时候，继续留着也没太大意义，意味着裁员降薪。毕竟最上级金融监管部门已经正式落实了大降薪，全面与部委公务员对齐，根据财新报道，降薪不会停留在监管部门，而是要继续延伸，包括下属单位、协会、事业单位以及金融央企及其下属机构。

那么商业机构呢?如果想继续赚高薪，那就靠本事赚钱，如果能够开拓更多金融市场业务、中间业务，那大可以继续赚，如果不能，那就一起过冬。

这次是官方首次提出“切实提高银行业保险业高质量发展能力”,但重视度很高。而且会议明确“加快推动重要立法修法工作”,大概率指向《银行业监督管理法》修订，草案在2022年11月已经发布，大大加强了监管力度，提高违法成本，银行保险业需要提早准备了。

就在本周1月17日，证监会也加大惩罚力度。公布的2025年2号罚单，对某内资会计师事务所在提供审计服务时未勤勉尽责开出罚单，罚没总计2000万元。

三大领域合并，2025年将继续对金融行业的整顿。
