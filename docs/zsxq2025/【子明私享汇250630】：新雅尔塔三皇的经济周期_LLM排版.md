
# 新雅尔塔三皇的经济周期
## 发布时间：2025年6月30日

自六月中旬特朗普的生日后，全球将进入到一个短暂的和平阶段，中美俄三个超级大国会变得非常和谐，中美的关税、芬太尼、Tiktok谈判会加速，美俄的停战与取消制裁谈判会加速，中俄的能源与中亚开发会加速，中国资本市场也会在六月开启一片欣欣向荣，只是尚不确定这个窗口期会持续多久，是否会有什么意外发生。经济方面的影响，可以参考特朗普胜选到上台，中俄两国股市都迎来了一波强势反弹，人民币与卢布资产也止跌企稳。外交方面的影响会更明显，中俄与欧洲的关系都在迅速恢复。逻辑很简单，特朗普赢得大选后的第一个时间点，是希望在6月生日举行阅兵，把另外两个超级巨头都邀请过来，所以他需要努力实现24小时内停止俄乌冲突，以解决普京的“护照”问题，也要跟中国迅速重建沟通渠道，以尽快达成“世纪交易”。在特朗普的最高意志诉求下，白宫上下都会对推动美俄、美中关系服务，中方与俄方乃至全球资本都能感受到浓郁的暖意，大家都能跟着一起赚钱。当然，这种暖意也会被一些事情打断，譬如在以泽连斯基为首的反对力量阻挠下，特朗普的俄乌24小时停火成为笑话，特朗普的6月生日大阅兵自然也成为泡影，尤其是三月，将首次出访确定为中东，随着特朗普对中俄兴致的匮乏，白宫上下对美俄，美中关系突然变得不上心，中美关税谈判停滞，美俄解除制裁谈判停滞，白宫团队转变为重点推进与以色列与海湾国家，以推动伊核协议与亚伯拉罕协议，最终以突袭伊朗核设施作为故事的阶段结局。随着六月周期的结束，世界将进入到九月周期，与半年前华盛顿官员主动联络中俄给胡萝卜类似，五月开始，北京的官员也会主动联络美俄，提出一系列令“双普”垂涎三尺的筹码，从瑞士到伦敦，都会主动合作，中亚和远东的投资也会加速落地，中国的对外开放与经济政策的落地也会骤然加速，三国的资本层面也会因此而蠢蠢欲动。在没有外力打破惯性的基础上，中国的外交与经济都会围绕着9月进行，极力构建反法西斯的“三皇同盟”(注：参考《42| 巴西金砖，大佬齐缺席》)，一些外交上的妥协(注：参考《40 | 中俄齐变脸》)、经济上的让步也都可以接受(注：参考《43|乐观的中美谈判推进》)。毕竟，对于这种世界的棋手来说，一旦签署了盟约，就标志着可以开始瓜分更大的利益。
