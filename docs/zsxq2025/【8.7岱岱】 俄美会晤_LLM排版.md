
# 俄美会晤

今天生病了，感冒咽喉发炎，一篇短文吧。普京和懂王准备会晤了。这几天懂王的特使在莫斯科，谈了几个小时和普京，然后放出这个消息，特使谈成了，谈成了会晤。懂王前不久还和普京不欢而散各种嘴炮，现在主动派人上门请求会晤，很迷之操作。而且，一直说9月中国会邀请世界各国领导人访问，一直在传懂王会在9月访问中国，而普京是必然来的，所以9月懂王只要来中国，就能和普京会晤。

懂王也大概率9月会来。7月22日，特朗普在与菲律宾总统马科斯会晤时首次公开回应访华计划，表示“可能在不远的将来访华”，并强调中美关系“非常好”。他未明确是否参加阅兵，但提到访华期间可能与普京会面，暗示行程可能与9月时间点相关。

新时代是给足了懂王面子啊。所以，懂王现在求见普京是很反常的。因为9月他就能见到普京了。而且目前懂王和普京的会晤，肉眼可见的不会有什么结果。

普京再度重申两个停火条件，乌东四州归俄和乌克兰永不加入北约，这两个条件是懂王一直都接受，但乌克兰欧洲一直不肯接受的。普京既然没有让步，欧洲乌克兰也不可能答应这两个条件，那8月份俄美会晤就大概率谈不出停火结果。那懂王要这个8月会晤有何用呢？而且懂王还是前段时间大怒然后主动上门请求会晤的卑微。因为懂王在乎的不是俄乌，而是黑料。普京方面对懂王的怒气值是逐渐升高的。梅德韦杰夫公开说懂王有黑料，而且在俄罗斯手里。

是的，黄金浴。懂王急着8月见普京的最大原因，是黑料，而不是俄乌。能不能促成俄乌停火懂王不关心，懂王关心的是普京要真的没耐心放出自己的黑料那就玩完了。懂王不为美国考虑，也得为自己考虑，于是选择卑微的派人上门求见普京。至于说俄美8月会晤，达成背刺中国计划，这个就太扯淡了。

年初普京和懂王如胶似漆的时候，两家会晤还能有这个可能。后面普京太狂而懂王太莽，把俄乌谈判玩砸了，普京说你懂王不是说好上台就让我吃乌克兰吗怎么一拖再拖你是个渣男，懂王说欧乌太难搞了我已经让步很多了你普京还不体谅我的难处我不爱你了，在中国没有挑拨离间的情况下，两位就从蜜月变成了冷战，8月会晤最多就是修复俄美关系而已，无法跨级到背刺中国的地步。

然后是国内几件事。很有意思。一直以来，我们认为危险的人，现在还是安全的。比如鹤真人。鹤真人和易纲都出面见美国公司了，亲美派给美国公司积极信号。而一直以来，我们认为安全的人，现在却是危险的。最近在传一个说法，xj马进京后，并没有如期的调换到统战部的岗位，反而陷入了漩涡。

甚至说，他的老家都被抄了。这个消息有点大，因为很多人都知道，马和夫人关系密切，比干结还密切，属于其妻子和夫人都有亲戚关系的那种。如果让我做二选一的选择题，马和干结里头必须舍弃一个弃车保帅的话，我都会倾向于保马而不是保干结。干结就算能死命保住，但也很难上下一届的7人了，他从吏部调去统战部，就是失去了下届上位的机会。考虑到夫人，马是更重要的。60后的干结已经出局了，而现在局里头，60后只有这6个人。

这6位60后，在下届会上位3到4人，上位竞争还不是多激烈。竞争激烈的是这6位中是否有人能上位东宫，以及下面一大帮人上位25人局委。

然后是10月份四中的主题。四中主题本来是经济的，谈下个十五五规划，甚至有消息说四中另一大主题可能是公共富裕和全国统一大市场。比如这个地方无序竞争，就是给全国统一大市场准备的。这个是很合理的，去年下半年开始就倒查了一批用土地税收政策不合规招商引资的案子，查的人仰马翻。

现在很多地方财政压力到公务员工资都发不下来，地方经济发展全靠中央给的专项债，全国只有少数的地方才能搞这个地方资金的招商。这个事要在10月定下来的话，那基本上中国区域经济的梯队已经定型了，一二线就是那些一二线，三线城市很难再有升级为二线城市的机会了，时代列车关上了对落后地区跨越发展的车门，车门焊死了。

然后是共同富裕也说会提，说共同富裕迈出坚实步伐，但经济上行不搞经济下行时搞这个是很难的，10月大会上怕是很难取得实质性的进展。就目前一条，你以为上面会落实劳动法，你以为上面会提高基本工资，结果目前都没有，目前是让强制缴纳社保。共同富裕期望值不要太高，十五五规划和全国统一大市场倒是可以期待下。

然后10月大会除了经济建设外，当然就是d内建设的主题了，也就是最重要的人事。这个人事，就要看这几个月神仙打架的结果如何了。
