
# 全球金融资产的大贬值

## 发布时间：2025年5月13日
## 【特朗普与产业集团系列4】

产业集团挑战金融集团最经典的案例，是2020年10月马云在外滩炮轰金融巴塞尔老妇人。但是马云赌错了，一周之后，拜登带领着金融集团战胜了特朗普的产业集团，之后，马云和他的蚂蚁就被收拾了整整拜登四年。2024年11月，特朗普赢得了美国大选，三个月后，神隐4年的马云现身北京民营企业座谈会，这也代表着，伴随着全球产业集团的再次得势，中国产业集团的反攻倒算全面开启，前两年国内对金融集团的肃整，也会变得更加严厉。

亲不亲阶级分，中美的收益受损群体是高度一致的，很多需要大量资金突破的领域，都会迎来政策性复苏。譬如中国被打压了四年的金融科技，先进半导体，火箭飞机，都在摆脱束缚与近年来的污名化，竭力的破土而出。

而通过金融与资本控制获得稳定性金融“地租”的产业，都在遭受各式各样的不同挑战和打击。货币资产(汇率)要贬值，联邦利率要下降，国债利息要下降，实业税费要降低，隐形门槛要拆除，医保开支要压缩，名校经费要压缩，油气矿的资本收益率要压缩，稀土锂矿的资本收益率要压缩。

理论上，所有地租性的金融资产和类金融类资产，都在产业集团的统治下被反垄断和打击，要以更为廉价的方式向生产性产业集团输血。毕竟，这一轮是产业集团赢了。
