
# 学习笔记，2025.4.23(1)

1，我公众号发的文字，你可以随便拿去分享，标你的名字都行，只要别让粉丝反过来说我抄你就行，资源类的，自用或分享都可以，别拿出去卖。去年还说过，你可以在评论区分享自己的账号或者自己的业务，评论都是自动审核通过的，只要不是拉人头、厚黑、收割的事，我也不会干预。我也能跟着长长见识有什么新奇的业务。后台经常有人问能不能用，能。但是我一看问的账号，0互动，就不太想回，如果经常看公众号，应该知道这条。

2，健身、读书、赚钱，不是为了比谁强的，是为了让自己的生活变得更好的。总是和别人比较，永远没有尽头，永远活在焦虑里。和自己比较，每天都有进步，每天都有成就感，人生会顺遂许多。

3，现在网上一些鼓励大家积极、努力的话题下，经常有人给评论，哪着火了，哪出人命了，哪坑人钱了……这些负面新闻，告诉你努力没用，世界越来越烂了。世界是平行发展的，有人过得很好，有人过得很不好，如果把注意力始终放在那些不好的事上，就会慢慢变得不好。而且，只有小概率事件才能算得上是新闻，媒体们追热点，算法助力，会给人一种铺天盖地都是一类事的错觉，实际上就是那一件事，极小概率的一件事。注意力回归线下，回归具体的生活，接触真实具体的人，晒晒太阳遛遛弯，世界还是很美好。

4，对大多数人来说，影响他们观念的，不是逻辑，而是故事。故事可以让他们有代入感，能够和故事中的某个角色共情，能够让他们潜移默化接受其中的一些观点。就像很多普通人仇富的观念，就是在故事中学来的，他们并没有用逻辑去判断观念的对错，只是在故事中普通人被富人欺负，就和故事中的角色有了同样的观念。

5，如果想要得到一个东西，首先要做到的是，提高自己的价值，让自己配得上它，配不上的东西，自己根本就拿不住。就像，月薪三千背个三十万的包，别人也会觉得那是假货。

6，很多人都说向往自由的生活，但从没想过，什么才是自由的生活，于是只能接受博主们兜售的一套套概念，一会是到处旅游，一会是随意消费，一会是勇敢裸辞，一会是千人斩……虽然不理解，但是都想要。自由应该是注意力自由、时间自由，你想把时间用在哪就用在哪，想做什么就做什么想不做什么就不做什么，可以不用拿时间去换钱，可以不用为了得到谁的回报去做自己不舒服的事。每个人都有自己真正想做的事，不一定非得和别人一样。

7，不要陷入自证，不要总是想着如何向别人解释自己，每个人都在以自己的认知看待别人。想要让别人理解你，就得重塑他的认知，投入巨大，父母儿女都不一定能扭转，更别说没那么熟的人了，不值当。

8，一件“坏事”，是让自己损失了什么，还是让自己学到了、得到了什么，全在一念之间。你觉得他对自己有启发、避免了更大的损失，是在警示自己，那就是好事，你觉得这是别人针对自己，天要塌了没法玩下去了，那就是坏事。以“我能在这件事上学到些什么”的心态去看待周围的一切，每一件事都可以是在助力自己成长。

9，好多大事都是闷声干出来的，大钱也是闷声赚的，在外面吵吵嚷嚷到处说的，吵吵嚷嚷吸引注意力才是他们赚钱的方式，因为一旦真正赚钱的方式被公布，一定冒出来很多竞争对手来分自己的蛋糕。就像现在的自媒体，变现能力很大程度上取决于流量多少，不用吸人眼球的方式让自己流量大起来，就赚不到什么钱。很多闷声赚钱的自媒体是做 SEO引流的，也不标题党，就埋好关键词等着精准用户来搜，不用涨粉也能赚钱。

10，大器晚成，前提得是个器，不要觉得自己20岁做不出的事到了 30岁就自然而然能做成，拿姜子牙熬到成老头才被重用当成自己怀才不遇的挡箭牌。一个人能一鸣惊人，是因为之前都在蓄势、在等待，而不是在躺平、摆烂，积累到了，自然而然会爆发出来。积累经验，积累生产资料，持续积累，总会有量变达成质变的时候，但是空等不行。
