
# 如何减少世界的撕裂

如何减少世界的撕裂，是过去十多年学术研究的热点之一，也是公共讨论中时常涉及的话题。新闻实验室会员通讯也长期关注这一问题。

简单来说，极化表现为一种“我们- 他们（us -them）”的撕裂，而且是一种你死我活的无法协商、无法妥协。我们在身边、在新闻里，都越来越多看到这样的例子。

这个撕裂的世界还有救吗？过去几年，香港浸会大学教授 Cherian George 一直在研究这个问题。他走访了澳大利亚、英国、加拿大、斐济、德国、印度、印度尼西亚、爱尔兰、新西兰、南非、土耳其、美国、香港和新加坡等国家和地区，将来自全球各地对抗极化的努力总结下来，写成了一本书，书名是《 Fighting Polarisation: Shared Communicative Spaces in Divided Democracies（对抗极化：分裂的民主社会里，共享的交流空间）》。

这本书将于下个月正式上市，目前还在预售中。不过，Cherian George 教授最近在《Journal of Democracy 》上撰文介绍了这本书，让我们可以对其中的精华内容一睹为快。

本期会员通讯，我们就基于这篇文章，介绍这本书中的一些案例和主要观点。

## 导致极化的三种中介力量

极化并不是一种从天而降的现象，它的发生是有原因的。要对抗极化，就需要了解极化产生的原因。Cherian George 认为，我们对彼此的看法大部分时候都不是源自直接的接触，而是由他人过滤和建构的。我们很少有机会直接跟意见相反的人面对面沟通，而是要通过三种中介力量来互相了解，它们分别是：政客、主流新闻媒体、社交媒体。

而这三种力量，即便不是故意想要煽动仇恨，也会有动力想要去分裂社会，而不是弥合裂痕。

为什么？对于政客来说，分而治之、“群众斗群众”是一种有效的统治术。通过制造和维持社会分歧，政客可以更容易地动员自己的基本盘，将复杂的社会问题简化为非黑即白的选边站队。当选民被情绪化的身份认同所驱动时，他们往往不会深入审视政策细节或政府表现，而是基于“我们 vs 他们”的逻辑进行投票。这种策略让政客能够回避真正解决民众实际问题的责任。

对于主流新闻媒体而言，冲突和争议能够吸引更多关注，带来更高的收视率。绝大多数以广告为基础的商业媒体依赖于攫取受众的注意力，而恐惧、愤怒等强烈情绪比平静的理性讨论更能牢牢抓住人们的注意力。因此，商业媒体有着天然的动力去放大分歧，突出冲突。

社交媒体平台的算法设计则进一步加剧了这种分化。这些平台的盈利模式建立在用户参与度之上——用户在平台上停留的时间越长，看到的广告就越多，平台就越赚钱。而能够引发强烈情绪反应的内容——无论是愤怒、恐惧还是仇恨——往往比温和理性的内容更能让用户持续参与。算法会自动推送那些能够激发情绪、引发争议的内容，同时将用户困在信息回音室中，让他们主要看到与自己观点一致的信息，进一步强化偏见和刻板印象。
