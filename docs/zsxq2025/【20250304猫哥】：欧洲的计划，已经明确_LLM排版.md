
# 欧洲的计划，已经明确

欧洲还会支持乌克兰多久，这是许多人心中的疑问。近期，德国外交部长贝尔伯克指出，应该再支持乌克兰一年，逼迫普京和谈。

贝尔伯克虽是个疯批，这番话却说出了欧洲的心声。尽管艰难，但许多欧洲国家的确想再支持乌克兰至少一年，为什么是一年呢？

第一次世界大战的时间，是1914年7月28日-1918年1月1日；第二次世界大战的时间，明面上是从1939年9月1日，德国闪击波兰起，但严格来说，却是从1941年6月的“巴巴罗萨”行动起，到1945年结束。毕竟美苏不参战，构成不了真正的世界大战。

从中我们可以总结出一条规律：两次世界大战的真正持续时间，都是四年多。这是因为，到了战争的第四年，双方都会消耗严重，通常会有一方撑不住。第一次世界大战的第四年，法国军队出现了大规模兵变。而德国境内，工人到处造反。二战同样如此，1945年的德国已经物资短缺，连找一桶汽油都困难，希特勒自杀后，尸体要用汽油烧掉，但半天也找不到足够的汽油，最后费好大劲才弄来。1945年的日本，也好不到哪里去。

当时的侵华日军，出现了大量的“国军部队”，也就是使用缴获的国民党军队的装备，放在以前，这些破铜烂铁日军压根看不上，都是扔给伪军用。这就是物资短缺造成的。

按照两次世界大战的历史经验，欧洲有理由相信，如果乌克兰再撑一年，没准俄罗斯会先撑不住，毕竟已经出现过“瓦格纳兵变”，是内部矛盾激化的体现。再出现一次，没什么不可能。

虽然俄罗斯的经济看起来健康，增长率尤其不错，2024年达到4.1%，比2023年增加了，但只是看起来而已，问题是明显存在的。其经济增长，很大程度上依赖军事开支，也就是政府大量采购国防相关产品，以及因战争而发出的各种工资、奖励与补贴，国防开支占政府总开支的比重超过30%，正常情况是10%左右。

这种状况不可能一直保持，也缺乏继续提升的空间，再增加比重就要严重挤压民生了。又比如通胀率依旧高企，2024年为9.5%，已经连续几年处于高位，显然不是好事。和2023年相比，2024年的通胀率还涨了不少。

高通胀迫使俄罗斯央行保持高利率，长期维持在20%以上。绝大部分行业不可能有20%的利润率。考虑到企业的经营是很依赖贷款的，如此高的利率下，许多企业要么涨价，转嫁成本，反过来加剧通胀，要么转向更高利润的产业，也就是和军工相关的产业。

当资金、人才涌向军工相关产业，将对一般制造业形成“挤出效应”，经济结构更加扭曲，不可持续。别看表面风光，背地里，俄罗斯是咬着牙在坚持。

既然俄罗斯能咬着牙坚持，欧洲也得咬牙坚持，而且欧洲有一定的底气。首先，懂王停止援乌只是停一部分。比如“星链”，对乌军作战有重大意义，许多乌军的网络连接就靠“星链”。前段时间曾有媒体报道，马斯克有暂停“星链”的想法，很快被马斯克否认。说白了，乌军用“星链”这事，对马斯克而言是赚的。

早期“星链”是免费的，但后来已经变成了付费，人家付费的东西，你贸然停了，会对商业前景造成负面影响，以后谁敢买呀？毕竟谁也不知道你哪天就把“星链”给掐了，说不定特斯拉的销量都要受到影响。

2025年1月，特斯拉的欧洲销量下滑明显，同比暴跌超过45%，注册量不到一万，原因就和抵制有关，很多白左对“特马组合”不满，偏偏白左又是特斯拉的最大潜在客户。所以，马斯克还真轻易不敢掐断“星链”。

其次，“援乌”和美国许多地区的利益息息相关，尤其是摇摆州。拜登曾说过，“援乌资金”中大部分留在了美国国内，重振了美国制造业。有没有重振制造业，不好说，但大部分银子留在了美国国内是事实。

军火商正是依据这些资金进行扩产，增加就业。现在懂王刚上台，没有选举压力，自然敢停，但一旦临近中期选举，压力就来了，那些摇摆州的军工产业从业者，是能得罪的？

以上两点，决定了懂王所谓的“停止援乌”不是全停，欧洲不需要补上全部空缺，只需补上一部分，咬咬牙，撑上一年半载，还是可以的。

从领导人的态度上看，欧洲各国也有底气撑下去。英国首相斯塔默是坚定挺乌派；法国总统马克龙也是挺乌派，且马克龙没有连任压力；德国准新总理默茨，是坚定挺乌派；意大利总理梅洛尼虽然是极右翼，但也是挺乌派，不少人感到疑惑，极右翼不是和俄罗斯一伙的吗，怎么挺乌？

其实吧，欧洲极右翼和俄罗斯是互相利用的关系，他们在野时，由于意识形态和俄罗斯相同，就会挺俄，等上台了，需要考虑国家利益，就会转变。以意大利为例，它不希望俄罗斯在巴尔干的影响力太强，让俄乌两败俱伤再好不过。

别看勒庞、魏德尔等人和普京关系密切，她们一旦上台，大概率也会援乌；欧洲核心国家的领导人，在援乌这一点上并无分歧。

当然，欧洲对乌克兰的支持力度不会是卖肾式的援助，顶多是维持战线，让乌军不要崩溃。

对我们而言，这是有利的。懂王最近卖乌克兰的动作实在太大，搞得俄罗斯有点飘，比如，近期缅甸的敏昂莱受邀访问莫斯科。以前我们说过，敏昂莱的合法性很成问题，各国都不愿意承认他，更别说邀请访问了，我们邀请敏昂莱来访，也只是让他去昆明和深圳，可没让他进北京。

俄罗斯让敏昂莱去莫斯科，表达支持的意思十分明显。“俄乌战争”都没结束，就惦记上东南亚了，时刻不忘往东边伸手。要是让“俄乌战争”停下来，那还得了？

所以，还是要支持欧洲的。战争一天不停，“中吉乌铁路”就一天不用担心被搅黄。从这个角度看，还是要支持欧洲的。
