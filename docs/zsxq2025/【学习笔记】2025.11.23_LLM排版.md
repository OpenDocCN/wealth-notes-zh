
# 学习笔记 2025.11.23

1. 任何公司不能创造现金流和利润时，投资风险就会极速上升。投资如果不关注公司潜在的创造利润和现金流的情况时，就跟赌博没什么区别。任何科技革命宏大叙事都不是你该投资的理由，你的理由有且只有一个，这家企业未来能创造巨大的利润和现金流。

2. 生活里总有人，揣着私心逼你大度，装着糊涂敷衍你的委屈，只站自己立场指责你不肯迁就。善良得有分寸，退让也得有底线。那些不顾你感受、损害你利益的关系，早断早清净。要是善良换不来温暖，只换来得寸进尺，那不如让它长出护己的刺。从现在起，我不想刻意当老好人，也不做恶人，就想做一面镜子：你对我狠，我又何必对你心软？

3. 谁让你堵心，你就离谁远一点。世界上最傻的事就是生气，生气是替别人惩罚自己。身体和心情都是自己的财富，这世上没有任何人和事值得你愁眉不展，人生在世，除了生死攸关的大事，其他都是过眼云烟。学会释怀，懂得看淡，才是处世之道。

4. 生命是活着的根本，健康是身体的靠山。没了生命，人就没了存在的意义；没了健康，再美的生活也无从享受。总有一天你会懂：钱财永远比不上生命金贵，健康更胜过世间万千。所以一定要好好疼自己，健康是头等大事，必须攥紧。人这一辈子，不用追大富大贵，不用求身价百倍，家人和睦、日子安稳、身体硬朗，心里没愁、身上没痛，就是最踏实的幸福。

5. 别让情绪主宰你的生活，与其钻牛角尖，不如学会与生活和解。

6. 世间万物皆有定数，与其抗拒不如接纳。人生不可能尽善尽美，但也不会一无所有。思虑过度伤神，计较太多累心，唯有放宽心胸、看淡得失，才能拨云见日。

7. 别总想着让所有人都喜欢你，别人怎么看你，不重要！重要的是你是什么样，就是什么样，不用去拼命地解释自己的好坏。有人懂，你就好好珍惜；不懂，也没关系。毕竟，讨好别人，你还不如多花点时间来提升自己。

8. 终有一天你会静下心来，像个局外人一样回顾自己的故事，然后笑着摇摇头，浮生不过梦一场。所以能够治愈你的，从来都不是时间，而是内心的那份释怀和明白。顺其自然，一切随缘。过去事，过去人，笑笑就好；现在事，现在人，尽心就好；未来事，未来人，随缘就好。

9. 人的放纵是本能，自律才是修行，短时间能够让你感到快乐的东西，一定能够让你感到痛苦。反之，那些让你感到痛苦的东西，最终都能让你功成名就。记住，低级的欲望，放纵即可获得，而高级的欲望需要克制才能达到。

10. 健康，是所有人的“底牌”：穷人丢了健康，是雪上加霜，日子彻底塌了；富人没了健康，是一辈子白忙，财富成了空；男人失了健康，爱人可能会成别人新娘；女人没了健康，丈夫或许会重新装点洞房；老人没了健康，天伦之乐成了遥不可及的奢望；孩子没了健康，父母会痛得肝肠寸断。人这一辈子，什么都能丢，唯独健康不能，没了它，一切都是空谈。

11. 永远不要自我贬低。不要说自己身体不好，不要说自己能力不行，不要说自己智商不行，不要说自己长得不好看。因为你说的多了，你的潜意识真的会相信。你怎么看你自己，别人就会怎么看你。
