
# 张兰请暂停

有时候不得不佩服这样的人--圣基茨和尼维斯联邦女企业家张兰，面对抖音的封禁，立刻做出反击。不仅自己在视频号重整旗鼓，还让旗下数十家麻六记分店全部开直播，迅速补齐了流量损失。

老实说，我没吃过麻六记，既没去店里吃过，也没泡过她家麻辣粉。但十年前我去吃过俏江南，印象里是个走高档路线的馆子。十年来技术升级产业升级消费更是节节升级，我想当然以为麻六记是比俏江南更高端的存在。没想到点开她家直播间，29.9元6桶，比绿皮火车上的“康师傅”还便宜。

再查了一下，麻六记实体店也走平价路线，菜品很便宜。怪不得张兰在直播间又蹦又跳，一会怼儿媳妇，一会骂韩光头，什么话题low扯什么话题，原来张兰的客户群体大反转了。十年前的俏江南，对准的是官商精英，必须要豪华、气派、有档次，配合着法国设计师打造的“兰会所”,吃喝玩乐一条龙。十年后，能边看张兰直播边下一堆麻辣粉和千层肚的，不用想，就知道不是精英。

这就是张兰的本事。上，可以让山珍海味吃腻的高大上群体吃得舒心放心安心；下，可以让菜市场里讨价还价的大姐大妈一股脑儿下单狂购。上下跨度之大，我目测至少隔了100亿。因为，那时候来俏江南的高大上人群中，有一位在四川待过爱吃川菜但又吃不了太辣的家族资产超百亿的无锡人，最爱张兰调配的新川菜口。

吃饭，砸锅，是坏人；好人吃了张兰的饭，就得帮她罩着锅。所以那几年里，裤兜里没几个钢镚的张兰摆出了顶级富豪的谱，其宝贝儿子汪小菲名列京城四少之首，搭过章子怡，泡过张雨绮，再跟安以轩眉来眼去，最后锁定了大S。

要知道那时候的章子怡可不是撒贝宁和汪峰能触及的存在，举个实例，曾经在港交所上市的万达商业，里面就有章妈妈1800万股，价值16亿。

妈妈是个股神，难怪章子怡越来越懒得拍戏。对于此种交易的背后原因，台湾省的一个杂志《壹周刊》做了点探寻，结果被章子怡告到香港高等法院。由于该杂志拿不出真凭实据，被判败诉，要支付给章子怡39万港币。

无独有偶，该杂志在将目标对准章子怡之前，也曾追踪过大小S。其中，报道小S老公嫌她不生儿子，将其揍得鼻青脸肿的故事，一度畅销于两岸三地。

小S也把该杂志给告了。官司正在进行的时候，大S来北京为好姐妹安以轩庆生，邂逅汪小菲，一见钟情，见到第5面订婚。其实那时候汪小菲的火爆脾气已经很著名了，他与张雨绮情动时分互扇耳光的故事广为流传。

于是有人说，个性刚硬的大S火速选个暴脾气+富二代的汪小菲，就是给个性软弱的小S做个样，看老娘怎样收拾男人。

中华传统，死者为大，所以我不能说大S死得好。但实事求是，她死的很是时候。

在她去日本之前，一场关于医保集采的大讨论正热火朝天，一位上海瑞金医院的觉悟不高的郑医生弄了一句“麻药不s、血压不降、泻药不x”的顺口溜，一时间流传，弄得流感高发季挤在医院里的广大群众心情焦灼。

即便是像对付那位跟踪油罐车的记者那样，立刻对郑医生严肃处理，但顺口溜已经传开了。又赶上各地两开会，春节大团聚，七嘴八舌，不利于大局。

这下好了，大S因流感猝逝于日本，号称世界上医疗水平最高的日本，得流感都要了命。顿时，台湾省内同胞纷纷去接种流感疫苗，内地同胞优越感更是猛增，论效率论速度论水平，风景还是这里好。与此同时，一则美国流感大爆发，两千万人感染，1.1万人死亡的消息，跟我国疾控中心给出的只有7人因流感死亡的数据，更是天堂地狱级别的对照。

大家应该发现了，过完年，那场大讨论就没影了，这其中，大S以死明志，贡献突出。实际上，在吃药方面大S早有功绩，2021年，YQ正酣，大S携母亲行为表演，合体写下“我要疫苗”，并在网上大骂那位姓蔡的省长不作。

这件事，当初也曾传遍网络，让我等美滋滋接种科兴者顿觉我很幸福。不少来自宝岛的艺人如萧敬腾、孟庭苇、欧阳娜娜等也现身说法，说已经在内地接种了灭活疫苗，免费的。大S想要的话，就快点过来。

这就是大局。俏江南的兴盛与衰亡是符合大局，麻六记的崛起也是符合大局。张兰是个能掐中大局的女能人，这大概源于她自称的属于爱新觉罗的后代。在大S因闹离婚没有再飞过来，并与妹妹一同掐断了在内地的演艺之后，张兰落井下石，将前儿媳妇一家弄成了吃饭砸锅的典型。

我相信，过去三年里张兰的无数拥趸，无不抱着此类愤恨：小的方面，是吃汪家的饭，砸小菲的锅，好处给了韩光头；大的方面，两姐妹多少年来在内地赚钱无数，关键时刻不仅不表态，反而模棱两可，甚至点赞台D。

顺着这个大局，过去三年里，无论张兰汪小菲如何表演如何造作怎样无下限，都没事。流量支持，再来一遍。

按理说，这一次也不该例外，尤其是大S死在日本，死得其所，非常贴合大局。过去三年里习惯成自然的张兰汪小菲，便不假思索的玩起流量，铺天盖地，占满热搜。

事情总是这样，成也习惯，败也惯性。一则包机的假消息，引燃了网络，传得韩光头和小S都出来反骂，岛内群众在忙着打疫苗之余，也纷纷被这对母子的无底线所震惊。

关键时间在变，这是2025，岛内正热闹一件大事，另一位韩光头领衔的立法院搞了个预算大删减案，将蔡赖逆流苦心经营的潜艇、无人机等项目尽数冻结，金额高达6.35亿美金。

成功与否，两派正对垒，民意最可用。此种情景，母子俩异常欢快的蹦跳，有可能会挑动厌恶情绪，影响胜负天平。

所以，“战兰”请暂停。
