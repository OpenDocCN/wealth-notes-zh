
# 学习笔记，2025.3.24

1，泰山，五岳独尊，听着，很牛逼。其实，没多少游客，近两年因为大学生特种兵的缘故，一年才突破600 万游客，我再说个数据，江西的三清山，一年 2000 万游客。除了五一、十一以及寒暑假外。泰山，人并不多。

2，事物发展有属于自己的规律，这就是天道，你看，信教的人，尤其是信基督的，经常说一句话，科学的尽头是宗教，例如牛顿也信上帝，其实呢，欧洲这些科学家信教，多是童子功，是一种家族习俗，就如同我们过年就回家上坟是一个道理，牛顿本身就出生在天主教家庭。相反，科学家对上帝的理解跟老百姓信的那个上帝，不是一个上帝，科学家理解的上帝其实就是天道，是一种自然运转的规律，不是一个具体的人，也不是一个具体的符号或形式。

3，现在，一聚餐，两类信息最密集。一，谁资金链断裂了，谁出事了。二，身边借钱的朋友越来越多，三角债催收越来越难。

4，我们日常收入是小溪，人与人的差距，表面看，是小溪的宽度，我月薪5千，你月薪1 万，你比我牛逼，但是长线来看，差距在于有没有拦河坝，例如我月薪 5千，但是我贷款买了房子，等于强制储蓄，我定投黄金了，也是强制储蓄，而你呢？都花了。

5，入口的东西，杂牌越来越没有竞争力，把核心业务做好，把不赚钱的，都砍掉。

6，教育会不会改革？不会，我上学时，喊了很多年素质教育，后来突然不喊了，有领导告诉我，是因为提出这个口号的人不在了。中外教育的差距，简单的归纳就是底层系统性设置的差异，是把学生当人去教化还是当工具去规训，当人，核心是个性体验+部分工具性，当工具，核心是纯工具性+极少个体体验，前者可以理解为现代教育，后者理解为传统教育，既然现代教育更高级，为什么说不会改革呢？因为经过千年验证，对于老百姓而言，考试赛道依然是最简单最公平的龙门，若是真的搞不用对结果负责的教育？工作实行推荐制或面试制？那，老百姓家孩子别想端上铁饭碗了。

7，高能量场的男人带着你，你是拥有无限可塑性的，让你干什么你干什么，甚至赴汤蹈火，命都可以送，还有什么不能答应的？在绝对崇拜的前提下，就是绝对的百依百顺。

8，阶层之差的根本是什么？是时间与意识。也就是说，跟我们生活在同一时期的有些中国人，他们的思想高度，领先了我们几代人，他们不可能表达，我们也不可能GET 到，就如同清朝那些前卫的学者，他们学英语，学微积分，你想，跟普通的中国人讲，与对牛弹琴有什么区别？不用说远了，还活着的杨振宁，你看看他是什么时候学的高等数学？

9，吃点喝点，哪怕买点奢侈品，都不会使我们返贫，但是，错误的投资，会！而且，很难翻身！人，春风得意时，一天赚个几十万都很容易。但是，当你指望上班来还债时？一年，攒5万元，很难！

10，到我这个年龄，初中同学不会聚了，高中同学也不会聚了，大学同学也不会聚了，我们大学同学 20周年都聚过去了。但是，局部都有联系。即便是聚会，大家在一起也没有话题可聊。我们上次去爬山，有朋友喊了她前同事一起，她以为挺有话题，爬山过程中才发现，彼此没有话题可聊，当年关系之所以能建立，是因为当时有共同的环境，一旦失去了共同的环境，他们俩的共通点以及可交流的点太少了。这就是常说的，一期一会。
