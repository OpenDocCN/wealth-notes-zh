
# 卢克文251024：答读者关于成都城建问题

我估计这篇文章阅读量较低，因为只答了成都城建一个细节问题，但是我真的忍不住写这篇文章，因为我太有感触了。

有读者在星球向我提问：成都的东进南拓西控北改中优政策，已经施实多年，效果究竟怎么样呢?我一看这题目就乐了，我太想回答了，我天天在这生活，那可太有倾诉欲了。

也不说废话，直接说重点。

我以前没搬来成都之前，在网络上老听说成都的格局是“东穷西贵、南富北乱”,其实这八个字，已经完全不符合成都现在的城建格局，可以说错得离谱。

现在成都真实的格局，是整座城市都进行翻新，变化非常大。

东边其实不穷了，东门二环内有攀成钢核心居住区、东郊记忆商业与旅游区两大重点，尤其攀成钢居住区，富裕居民人数众多，得以在这个区域内，形成了成都最好的生活配套服务，无论夜宵摊、中医按摩、社区医院、课外辅导、特长教学、篮球馆、三甲医院、面包店、精致中餐、特色日料、中小学校，都是成都一流水平，可以说是居家最合适的区域。

那有人说西门不是有浣花溪这种顶尖居住区吗?难道攀成钢比得过他们吗?不是这么回事，攀成钢可能没有浣花溪顶尖，但富裕居民的总数多，因为人多，才可以供养大片优质的生活配套服务。

我曾经在《分享一些我做小生意的心得》这篇文章里面，提到有一个卖拉面的小摊和一家咖啡馆大家记得吗?这个拉面差不多25元一碗，每天晚上八点出摊，十二点到两点卖完拉面，就是依靠攀成钢周边的优质人群才得以维持。而那家咖啡店更离谱，平均一杯咖啡卖到50-60元左右，从中午十二点到晚上七点，只上七小时的班，每天客流量不断，也是依赖攀成钢的高消费人群才能生存下来。

是先有“大量集中的高消费人群”,有了这份土壤，才能“养育出高消费精致服务配套服务”。

西门浣花溪再强，但它地小人少，是不能跟攀成钢的总规模相抗衡的，难以孕育出高质量的生活服务配套，最多只有极少数为顶尖人士而设的隐蔽会所和餐饮，所以西门也有很多顶级黑珍珠和米其林林店。

成都的朋友也带我去吃过这些很有名的黑珍珠和米其林，我劝大家一句，能不去就别去了，真的一般般，死贵死贵。成都好吃的根本不是黑珍珠和米其林，绝对不要迷信那个榜单，好多都是样子货，成都最好吃的，是街边小店，是小摊小贩，是某个不起眼的火锅店，是看起来一脸沧桑默默无闻的某个大叔。

我其实打算列一张清单，告诉大家来成都哪些店最好吃的，以免大家被米其林和黑珍珠误导，我以我在这生活多年的真实体验告诉大家，哪些店是真的好吃又不贵。

大家等我的清单。

继续说东门，东门的东郊记忆是一处非常不错的旅游商业区，很适合带着全家人或者朋友到这散步游玩，我们常常在各种成都宣传照里，看到一面墙上写着巨大的“成都”两个字，游人们在这排队合影，就是东郊记忆这儿。

东郊记忆还诞生了一大批优质的，具有网红性质的餐馆店、小摊小贩，全世界各种风味都有，另有各种特色酒吧、二三流歌手的小型演唱会。到了晚上过来，还能见到很多很多小孩在这跳街舞，大抵都是四岁到七岁的样子，在地上噌噌噌刮灰，整个东郊记忆从早到晚都是一个很有生命力的地方。

除掉攀成钢和东郊记忆这两核心区，你再往东，就慢慢靠近龙泉山了，又形成了休闲旅游区，出三环后，这边的房价要便宜许多，主要是青龙湖、龙泉山为主，马路建得极宽，小区的距离较大，已经无法形成攀成钢这样的高密集型居住区，所以商业氛围要淡了许多，这里更适合跑步、野营、健身、烧烤。

东门大概是这个样子，东穷这两个字完全不适合现在的东门，以前说东穷，是因为这里聚集了大量收入不高的普通工人，但这是老黄历了，跟现在的东门没有任何关系。

然后成都的北门呢，其实替代了以前的东门。北门是成都普通平民聚集区，这里也有一些工厂和写字楼，但主要人群是大量月收入4 -7千为主
的普通蓝领和白领。

北门也是四川其他各城市移民，到达成都后安家落户的重要聚集区，因为这里房子建得极密集，房价又便宜，一家人如果大家都在工作，父母加上
子女四个人都能赚钱，差不多一个月也有两万左右
的收入，扣掉生活费后，全家把钱凑一凑，是可以
在北门搞个首付，然后全家一起慢慢还房贷，在成
都落户生根。

我上次跟一个司机大哥聊天，他说最近这些
年，好多人坐车时跟他聊天，以前在南门买得起房，
因为经济下行期，收入下降，这些人把南门的房子
卖掉，重新在北门买套房，生活质量就保住了。

成都北门以前是批发市场、火车站、工厂的重
要汇集点，是来成都艰难求生的落脚点，但是经过
不断改建后，就慢慢成了平民聚集区，以前批发市
场老板、工厂老板也会落脚在成都，成为北门的中
高产，但总体来说，北门从北二环开始，就是大量
普通劳动人民定居的地方。

一直到凤凰山体育公园。这座体育公园得单独拿出
来说，因为这是中超成都队的主场，每次比赛的时
候人山人海，气氛极热烈。

成都的西门是我较不孰悉的地方，因为我不喜欢去
西门。

大家一定觉得很奇怪，为啥我不喜欢去西门。因为
西门太旧了，老旧的房屋、街道太多，我一去就心
情不太好，每次都是速进速出，没有在西门停留的
兴 趣 。

我记得我刚刚到成都时，房产中介把我带去西
门看一栋公寓，那公寓其实建得不错的，但我站在
高处往下一看，周边全是低矮陈旧的青砖瓦房，呈
现一片衰颓之气，然后我掉头就走了。

我作为一个80后的年轻人，肯定是要住在有
朝气、有活力、有大量现代建筑、贴近现代生活的
地方，西门好多地方像一个老人，暮气太重，所以
我不爱去。

当然西门也有西门的好，一是这里确实有一小
片顶尖的富贵区，不过区域太小。二是这里汇集了
大量老成都，以致沉淀形成了极佳的一些餐饮店
铺。

但跟东门的店铺明显不同的是，西门的优质餐
饮更有成都味道，东门的优质餐饮综合了全球各地
的风格。

西门总给我一种老派的感觉，你听听那边的地
名：浣花溪公园、青羊宫、武侯祠，听着就一个比
一个老，甚至连那边的商场都给我一种老派感，我
是真不喜欢在暮气较重的地方生活，所以平时不去
西门。

成都的南门是现在成都最重要的地方，是成都建设
的重点。其实成都整体大开发、大建设快差不多了
，城市的高速生长期已经结束，后面是一些边边角
角的补充，已经可以确定南门是成都的重中之重。

我之所以选择来成都，就是因为在南门逛了一 圈，
我就决定在这定居了。

这边的硬件水平，除了缺少深圳人才公园那种
地标性建筑，其他各方面跟深圳差不多，南门的平
均水平，可能比深圳除去地标外，还要高一点点的。

以火车南站为分界线， 一直到兴隆湖，成都南
门又新又靓，写字楼、公园、小区建得干净又漂亮，
商业区较为繁华(但我觉得比深圳有差距),写字
楼崭新透亮，公园则极好，比深圳广州的公园要好
许多许多，还有大量漂亮的人工湖，给人很不错的
生活享受。

南门最核心地段是金融城，年轻人很适合住在
这里，攀成钢更适合中年人一些。因为金融城有最
好的写字楼、最好的公寓、最好的商场、最好的现
代酒店，楼下就有演唱会场所，还有成都最漂亮的
双塔，甚至连这里的公寓进出的女孩子，也明显比
其他区域更年轻、更好看。

但是，我还是要说，成都这种地段的繁华是不
如深圳的，金融城这么核心的区域，我晚上九点以
后随处都可以停车，马路边来来往往的人流量也不
大。这要是在深圳，这个点还是人山人海，马路边
黑压压全是人。

金融城有一个华商商场，我强烈推荐来成都旅
游的人过来看一看，因为华商，是全国唯一的，整
栋楼全是中高级餐饮的商场，里面你能吃到全国一
流的各风格、各地区、各品牌餐饮，体验感极佳。

成都真正的繁华，其实从火车南站到天府五街
就结束了，从天府五街再往南，华阳片区比较接近
北门的平民区，价格不贵、市井热闹，但不是那种
现代化高质量的繁华，开始有点力竭的意思。

比如那个南湖国际社区，周边其实挺好看的，
以这个建筑质量，这要是放在上海，价格起码八万
一平起步，但在成都，就一万出头好像(懒得查了),
每天上下班时间，大门口黑压压全是电动车自行车
啥的，气质特别平民化。

南边还有麓山国际这个顶级别墅小区，还有麓
湖顶湖住宅区，但人气都不够，我去麓湖看过几次，
就不想住在这里，因为太冷清了，干啥都不方便，
像旅游区，不像住宅区。

再再往南，就到怡心湖、秦皇湖、西部展会、
兴隆湖、科学城了，这一带建得超级超级漂亮，有
些地方的花海堪比省级植物园，现代建筑也规划得
一流，开车从这路过，让人心旷神怡，但就是搬来
的公司还不够多、产业还不够密，商业还较为薄弱，
人气还要等等。

总之，南门这样的，从火车南站到天府五街，
是商业核心，尤其金融城最重要，从天府五街到兴
隆湖，人气就越来越淡，但现代建筑建得极佳，也
适合年轻人住。

我个人的看法是，成都最适合定居的是两块，
一块是攀成钢、一块是金融城，如果收入不高，适
合定居在南门华阳，不太建议定居在西门温江、北
门大部分，这两块地便宜归便宜，但离工作区域太
远，每天交通不方便。

回到问题，问的是成都搞东进南拓西控北改中优，
结果怎么样呢?这么说吧，现在的成都，东南西北
中全部升级了，东穷升级成了高端住宅区和商旅区
，北乱升级成了平民住宅区，南拓把农田升级成了
繁华商业区，就是西控没咋做好，西门变成了一小
部分现代商业加一小部分古迹加一小部分住宅，大
部分都是低矮的七八十年代，甚至更久远的老旧建
筑，但也没办法，要拆掉成本太高了。

对了，还有个中优，其实没啥好说的，因为二
环内不管咋搞，它都有钱，因为春熙路、天府广场、
人民公园、锦里都在这里嘛。

我觉得现在的成都，不是以前的“东穷西贵、
南富北乱”,而是“东居西旧、南华北平”了。
