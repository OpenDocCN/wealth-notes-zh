
# 《感冒与缺席》

发布于2025年3月12日

本周一，副委员长接主持全国人大全体会议，并在开会前表示，今天下午Z委员长因呼吸道感染请假，自己被委托主持会议。也成为了自2006年以来，首个因健康问题缺席的人大会的正国级领导人。

虽然海外媒体一路狂欢，但目前来看，较大概率是真的健康问题。一般来说，领导人的健康情况是高度机密，按惯例不会公开病因，能够公开说是呼吸道感染，也就有可能真的是呼吸道感染。而且，两会开幕时Z已经亮相，如果真的有问题也不会在这种全球瞩目的时候搞突袭，泱泱大国也要注意门面。

目前来看最大的可能，应该是真的感冒影响了声带，为了避免在全球直播的过程中影响形象，就只能请假了。
