
# 石天方付费文：节后大盘会怎么走？

石建军，笔名石天方。中国资本市场第一代证券投资分析家，九三学社上海金融委专家委员会成员，华东师大国际商务硕士学位专业兼职硕士生导师，上海交大金融学专业经济学学位。1990年进入证券市场，1994年即为上海证券报专栏作者。作为全国最早的中央行财经评论嘉宾，曾在中央台、中央电视台、上海《第一财经》、湖北卫视等财经媒体长期担任嘉宾。是香港文汇报财经版特约采访嘉宾。水晶球财经网2018、2019名博评选第一名。在全国投资者中有着广泛的声誉和影响力。

擅长用经济周期理论结合波浪理论预判大势。擅长结合国家产业政策导向和行业发展前景捕捉投资方向。创立和实践投资操作三格言：1、明大势者赚大钱；2、主流品种决定投资收益；3、价值投资选股、趋势操作选时。

具有极为丰富的证券投资管理经验。对国际主要股市和国际商品市场的运行规律有着深刻的研究。极为准确地预测了中国股市多个历史性的重要顶部和底部。

本周大盘是一根十字星，收盘指数在5周均线附近。目前的周均线情况是，5周和10周均线向下，20周均线向上。且5周均线即将与20周均线缠绕。这样的周均线排列，很可能意味着短期大盘将继续以小幅震荡为主。向上和向下暂时都还不具备明确的突破性的动能。

自去年四季度出现高点3674点之后，我对整个A股行情一直维持着谨慎的看法。实际结果也显示，四季度的放量季阴线着实"很难看"。而今年一季度的季线呢，虽然还只是开了一个月的头，但一月份还剩下一个交易日，这根开头的月阴线也是铁定的了。

再看成交量。去年最后三个月的月量爆棚，但今年一月份的月量只有之前月量的一半多一些。严重的缩量再加上一根下影线的月阴线，明确提示了近期大盘是很难向上运行的。这就是为什么我一直在说“大盘的调整并不充分”的原因之一。

虽然大盘调整仍然很不充分，但为什么市场经常有跌不了多少的看法和感觉呢？很明显，一个重要原因，就是，政策密集期似有无形之手在稳定大盘的运行。其基本策略，显然是通过时间来换取空间。应该讲，一段时间以来，这个策略还是明显影响了大盘的近期走势的。

就拿这次3494点的下跌波段来看，一跌破前低3152点，就被一股力量托起。本周三眼看大盘要再下台阶，周四又被利好政策托起。按照这样的模式，3140点反弹的性质是否也会出现一些变异呢？

如果没有周四的维护，3494点的下调波段极有可能直接去运行旨在封闭下方3087点缺口的下跌第5浪，从而为节后的春季行情奠定基础。但周四和周五的维稳，虽然还没有完全改变其后仍会继续运行下跌第5浪的结构，但也出现了3140点的反弹可能变异成延续反弹的小X浪的可能性。如此，节后开盘第一周，也就是二月第一周的5、6、7号这三个交易日就比较关键了。

如果节后第一周大盘阴线向下，那么，大盘就仍然按照原来的3494点调整波段的下跌第5浪的结构去运行。调整目标还是封闭缺口并寻求3050点附近的止跌回稳。进群加微信

如果节后第一周大盘继续向上反弹并站稳在3270点的上方，那么，大盘其后就有可能去运行小X浪的反弹结构。这个小X浪的反弹结构，理论上将在3350点附近受阻回落，并再次进入另一个向下调整的小波段。而调整的目标，仍将是封闭下方3087点的缺口。

从以上分析可见，我是强烈认为大盘只有在封闭了下方3087点周线缺口之后，才有可能去启动今年像样的波段向上行情的。而在我的分析系统中，直接向下运行下跌第5浪后的春季行情，要比先小X浪反弹而后调整到位再启动波段回升的后一种模式要来得更早更好。

顺便提一下，有没有可能3140点开始的直接是一轮向上运行的可以再试3500点前高的波段向上行情?

遗憾的是，我认为可能性很小。理由还是那句老话：向上，需要成交量的进一步放大。去年四季度三个月份的那么大的月量压在上面，能否再度出现进一步的成交量去消化它们，至少目前来看，似乎还是只能望洋兴叹!

都说今年的行情要好过去年，平心而论，至少在一季度内，我认为勉为其难。操作层面上，无论节后大盘走出哪种预期中的走势，投资者的主要选股方向仍然应放在半导体、算力和人工智能上。前者，是国家自主创新政策的导向要求。后者，是全世界都极其关注的话题。你可以追寻活跃资金的脚步走，但我更主张在上述每个板块中选一个行业的领军企业，适当兼顾市场情绪，反复低吸高抛做波段。

至此，无论节后大盘怎么走，看准心仪股票，持半仓以内的底仓，涨跌皆有应对。坐看云卷云舒，不亦乐乎!
