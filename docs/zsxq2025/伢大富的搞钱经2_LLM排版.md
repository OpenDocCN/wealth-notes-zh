
# 伢大富的搞钱经2

## 自述：我的第一桶金(上)
每个人的原生家庭多少都有些缺憾，我也一样。我妈是那种典型的被重男轻女思想洗脑的女性，虽然受过高等教育，收入也不错，但她是初代“扶弟魔”，几乎包揽舅舅的一切，平生最大遗憾的就是没生个儿子，并觉得女人最大的幸福就是找个好男人依靠，如果是靠自己，那就是劳碌命。我在叛逆期的时候几乎不想理她，觉得这妈怎么这样啊，等长大了发现，很多妈妈都是这款，属于时代烙印洗不掉的。但她也不是没优点，比如对我学业非常上心，会想尽办法送我进最好的学校。记得有次我俩散步，说到学校里有留学项目，妈妈说：“如果你想去也能选得上，我砸锅卖铁都会供你。”可是去美国读完大学和研究生很贵的哎，但她就是属于兜里只有100块也愿意为子女花99块交学费的人。听起来是不是很矛盾，要知道前一晚她还嫌弃我是个女孩呢。仔细一想也能理解，我外公那么重男轻女，口头禅就是闺女我就养到18岁，妈妈如果没有靠读书走出来，可能就跟村妇差不多，想做“扶弟魔”也没条件了，没工作没钱啊，只能做干电池，榨干自己、成全男人。但问题是她总是叮嘱好好读书，却从来不说为什么。家里的事情从来不让我过问，理由是小孩子管好学习就行了，知道那么多干啥。——学习很重要，重要到父母都砸血本了，但为什么重要，我不知道，这就是那些年我的纠结。感觉入社会前，整个人都是活在父母营造的真空世界里，包括高考填志愿。他们让我做老师，说稳定体面适合女生，可这个理由说服不了人，我那么辛苦的学，难道就是为了教比我还笨的孩子么,不甘心啊。相信不止我一个人曾这样困惑，长期的辛苦的做一件事，却没有目标，或者说有但那个目标不是自己定的，而是父母亲友告诉你的，你活得像个提线木偶。我爸妈还有个硬伤，就是不太乐意跟孩子提钱，好像钱是个很坏的东西，会把孩子污染了一样，他们愿意倾其所有，却只是希望我读出来找个工作，然后结婚相夫教子，至于活成这之外的模样，不好意思他们没想过。以至于我在大学里，看到学长开奶茶连锁店月入6位数，看到班里女生拿学费去四季青进货开桃宝店，月赚5位数时惊愕不已，觉得人生还能这样啊，还能自己挣好多好多钱，想买啥就买啥。就这样，以前的我狗带了，钮祜禄大富萌芽了，我开始给自己找活干了。
