
# 二月必读：黄金大事件与未来展望

## 前言
寻找市场短暂错判中的机会

12月8日，精准地在黄金起爆前发布了《十—二月必读：黄金的投资风险与机会》，期间黄金在体系内与体系外定价都出现了边际变化，促成了这波10%的单边上涨。周五黄金发生了三件大事：央妈连续三月购金、险资放开购金权限、老美割裂非农公布，是利好落地还是长牛开端?我来做详细点评。

### 央妈连续第三个月购金
1月净增16万盎司，我在《十二月必读：黄金的投资风险与机会》已经强调过，央妈的购金分隐性和显性两种，隐性动作是一直都在做的，而且经过测算，规模是显性购金的10倍以上；而显性购金则取决于与西大的明面关系，关系好则停止购金增持美债，关系差则继续显性购金，去年4月的耶伦来访奠定了这个论断。

显性购金的规模本身并不是最重要的，增持与否所代表的背后态度才是关键。而且1月是这轮逼空行情的起爆点，能顶着金价的拉升去明面增持也坐实了我之前强调的，现价并不是央妈最看重的，黄金给人民币资产背书以及对抗贬值才是。

### 险资放开购金权限
人保财险、太平保险、平安保险、太平洋保险、泰康人寿等十家保险公司作为试点被总局批准可以买黄金了，保险全行业35.91万亿的总资产规模，试点的前10家总资产差不多是20万亿的规模，按照不超过1%的要求，上限有2000亿，经过与星球内大型保险买家的交流，我估算短期落地的规模会在1000亿左右。

根据世界黄金协会的统计，养老金投资组合中增加10%的黄金配置，可以大幅提高风险调整后的回报率，既能减少整个投资组合的波动，又能提高整个投资组合的收益。我们现在试水1%，而且只让直投不能受托投，按照星球里保险买家的说法：战略层面，配置型的资金会先进，战术层面交易的钱还被摁着。说明路径走通后大概率资金类别和口径都会继续放大。所以重要的不是现在投多少，而是开始投了。

### 美国非农数据
1、一月非农14.3万，低于预期17.5万；
2、十一月十二月非农合计上修10万；
3、失业率4%，低于预期4.1%；
4、工人每小时薪资年率和月率都比预期高；
5、截止2024年3月的12个月，就业新增人数下修58.9万，低于预期的82万。

除了第一点是差于预期以外，二三四五点都好于预期，意味着数据虽然割裂，但是整体的方向还是代表美国就业数据较为强劲，降息短期无望。美联储古尔斯比的发言也预示着到2025年底都不一定能达到降息要求。

### 通胀调查
昨晚的密歇根大学一年期通胀调查直接飙升到了4.3%，创下一年新高，暗示特朗普的关税政策下，美国通胀飙升很难阻止。在逆全球化导致的全球衰退大环境下，老美顶着36万亿美债独自不降息甚至加息?

本周刚好在读书会做了《见证逆潮》的解读，黄金价格 = 1 / 美元安全性这个说法其实也恰好吻合了我一直以来所说的，美元主动降息其实对黄金并不一定利好，但是为了防止硬着陆出现的被动降息，才是黄金的最优持有环境，而在这个硬着陆出现之前的博弈，甚至加息预期的酝酿，都是最美好的温床。所以我重申，千万不要听信别的自媒体的片面定价表示美元不降息就是利空黄金，其实利空的只是美债。

看完这些，你大概就能深刻理解，为什么美国非农的反弹，通胀的反弹所带来降息预期的压制，都无法把黄金价格打下去，黄金更像是一种信用主权的赖账风险补偿，因为不信任，所以才涨得坚定。

目前就看老美这个独立高增长的债务泡沫是被deepseek还是非农零售数据去刺破了。
