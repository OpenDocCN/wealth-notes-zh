
# 《体制内星球问答》

## 匿名用户提问：
您好A处，我快33岁了，女，已参加工作10年半，目前是某乡镇党群副书记，去年8月从另一个乡镇调整过来的，也是目前我们县唯一的一个乡镇女副书记。之前的乡镇是全县第二远的乡镇，距离县城2小时车程，参加工作时在那里工作了3年，任副职时又在那里工作了3年半，现在的乡镇距离县城1小时40分钟车程。

问题一：我想考遴选，但现在为了乡镇明年换届前的稳定，我们无法报名参加考试，但眼看着着我离35岁就只差两年多了。只有我在乡镇目前的岗位保持不动，可能还有一年考遴选的机会。但我不确定未来三年内我的岗位不会变动，如果岗位发生变动，我就完全没有考试的资格了。那么如果考省州的事业编，从长远看是否可行？

问题二：现在这个乡镇情况比较复杂，到现在还不太擅长处理这种特别复杂的情况。新来的（去年12月）委书记经常给我说要我努力，以后还有大任，近期更是给我说有一个机会，要我争取。我不确定这是不是画饼？还有这个机会该不该争取，因为据我所知，目前只有一个更远的乡镇镇长位置空缺，明年可能有5个乡镇空出来。我怕现在一争取就争取到这个更远的乡镇去了。从内心来讲，我很怕在复杂的乡镇当乡镇长，也不想离家超过1小时车程，因为孩子还不到3岁，母亲帮我带娃，身体又不太好，想再请个人照顾，资金能力又有限。所以觉得回县城也行，但短时间内不知道回不回得了县城。

问题三：基于上面这些情况，你想问问老A您还有没有其他建议，帮我破局。

## 老A回复：
很多同志没跑没送没人身依附，因为没钱跑送，也不会人身依附，但也在本辖区内稳步的晋升，因为他们具备体制内人基本的基本素养：工作能力强、工作态度好、能吃亏、能忍让、虽不会巴结领导，但对领导绝对服从等等等等。这种状态，是大部分体制内人的工作为人状态。

所以，你也能10年熬到乡镇副书记的位置上。下一步该怎么走呢？你心里其实并不清楚，只觉得我们这个乡镇复杂，你多少有点想逃离。真的复杂吗？一个距离县城两个小时的乡镇，概率不会太复杂的，比中心镇要省心省事得多！

所以，此刻，你得扪心自问：要不要走仕途？如果走仕途，我觉得你遴选与否意义不大，遴选上去了当然有遴选上的好，但就在本地方发展，机遇也很不错！

我说一下我的看法：我若是你，我就不遴选了。我会继续干好本职工作，平时多留心乡长是怎么开展工作的，毕竟下一步大概率你迟早都会去那个位置。

恕我直言，乡长那个位置我觉得比副乡长更容易坐，当官不当副，是有一定道理的，因为很多人压根就不听你的，很多副乡长压根指挥不动乡镇的老中干。但是当了乡长后，这个情况大概率不会存在。

所以，你不必畏难。乡长能力强，就多干点，能力不强，书记自然就强势一点。反正，在我看来，乡镇书记是要有点经验和水平的，但是乡长对这一块的要求比较少！

且，我若是你，我不会回县城，起码拿到级别后再回！我若是你，在上面也得找找关系了，比如你作为副书记，你去给县里的副书记汇报工作是天经地义的，你汇报过吗？给组织部长汇报工作，也理所当然，你汇报过吗？

你是女干部，且如你所说是全县唯一的女副书记，你不必像其他男同志那样非得和常委们打成一片，当他们的肚子里的蛔虫，你只需要稍微近一些就好，因为他们推你更上一层楼，是水到渠成的事，不是那么太费力！毕竟大家都知道：女干部有优势！有力的女干部更有优势！

至于孩子，如果老公事业处于常态化停滞期，那让他多费费心。又或者，三岁的孩子，还没上幼儿园，可以接到乡里来。
