
# 中专学生的真实状态

以前的同事卡在35岁之前去一个中专上班了，也算个铁饭碗，这两天聊了下，挺有心得：

1. 他去之前以为那里的人整天混，各种为非作歹，怼老师，打架什么的，特别难管。去了发现不是，大家更像一种 啥也不干 ，很多人上课就睡“ ”觉，下课继续睡，或者拿出来手机玩会儿，晚上熬夜玩手机或者打游戏。跟老师的互动，更类似一种非暴力不合作，你说啥他们也不理你，也不闹。

2. 这群人最大的问题是啥也不干，不知道该干啥，进而啥也不想干，连对象也不搞（绝大部分不搞，极一小部分有对象），反正每天就是浑浑噩噩啥也不干。想象中的黄毛、打架、派出所、搞大肚子什么的，也有，但极少。

3. 学校要求跟学生们谈心，跟他们讲努力讲奋斗讲发展，一个个都迷迷瞪瞪，好像在听你话，又好像没再听，后来听他们说是 你说的那些我们都懂，然而并没有什么卵用，懒得回应 。

4. 整体的感觉就是这群人超级累，干啥都无精打采，正经事和不正经事都不太想干，而且他待了四年，每一批都差不多。不过好在非常好管，只要你不给他们提要求，他们一般都听你的。
