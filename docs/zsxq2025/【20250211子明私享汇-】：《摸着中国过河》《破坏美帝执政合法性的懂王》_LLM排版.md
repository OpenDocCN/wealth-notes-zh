
# 《摸着中国过河》《破坏美帝执政合法性的懂王》

## 《摸着中国过河》
源自2月10日群聊
发布于2025年2月11日

懂王现在做的，是河伯的生意，
是在破坏美帝的执政合法性，
而马斯克他们选择资助扶持懂王，
也是为了破坏美帝的执政合法性，
打碎一个旧世界建立一个新世界。

马斯克跟懂王并不是一路人，
双方只是有共同的敌人共同的目标才走到一起，
18时候，红党联盟团结在老板周围打蓝党也是如此，
在破坏旧秩序的问题上，大家的利益一致，
只是在建立一个什么样的新秩序上，诉求各不相同，
并在击溃蓝党后开始了新的路线之争。

现阶段，在双方有共同敌人的情况下，
马斯克代笔的利益集团，只会不留退路的支持懂王。

## 所以几年前查账坠机的事情
阻挡不了马斯克的新式查账，
现在的查账更多是基于政治集团的换房本，
而可以参考18、20的两轮反腐。

## 高举着政治正确的大旗，以便于获取更多民众的支持，
正如汉东的反腐，是沙家帮对赵家帮的换房本。

至于马部长的Paypal帮，
跟十多年前的马省长帮没啥本质区别，
AliPay甚至万斯之类的年轻人，
也跟十多年前的最年轻部级干部差不多，
都是在之前进行了较多的天使投资而已，
IPO骤然提拔马小将们也很类似于当年的之江新军，
其实中美一对照，
就会发现懂王和马部长也是在摸着中国过河。

## 《破坏美帝执政合法性的懂王》
源自2月10日群聊
发布于2025年2月11日

天庭(玉虚宫)的执政合法性来源，
就是造个美梦，让底层的小妖们有奋斗的目标，
这样勤奋修炼的小妖们才有资格成为天庭的丹药，
天庭童子们才能够过懒洋洋的幸福好日子，
其实也是太极仙翁在为天庭负重前行，
否则他们的丹药从哪里来，还不是靠全球剥削妖族么。

## 而懂王现在做的，类似于中国传统神话里的河伯，
不再像玉虚宫那样故弄玄虚的洗脑，
也不要求你多劳多得的奋斗修炼，
而是要求你每年固定的上交足够数量的童男童女，
直接扯下了天庭的伪装，
这么做对很多人很多国家都是利好，
但也会引发很多人信仰崩溃，
发现了之前被欺骗的群妖自然会奋起围攻玉虚宫。
