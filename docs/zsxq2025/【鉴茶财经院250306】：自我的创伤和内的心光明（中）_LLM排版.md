
# 自我的创伤和内的心光明(中)

这个世界有剑法和心法，通俗的说，是操作技巧和人生模式。但外面剑法都不好好教，何况心法呢？只有剑法，你的内力是不浑厚的，人活得不是特别快乐，成就也不会特别大。我们上篇知道，你的心，是操作系统，是先天的，你的脑，各种是后面预装的。

大脑的本质，是一个算力工具，情绪是它的副产品。你真正的心，是光明的，宁静的，不垢不净。问题在哪?99.9%的人，是被大脑支配的，错把工具当手段。算力工具的特质，就是喜欢竞争，喜欢对抗，喜欢压倒别人。操作系统的特质，是干净，流畅，和外界没有关系。由此出现了一个很大的问题，很多人的人生模式是对抗模式，而不是发现模式。

对抗是什么，你永远不知足，你永远都很累，你永远在追求，你永远都被外部的东西束缚。竞争≈外部的认可。马斯洛需求理论模型，就是5层一路打怪，打到最高一层，所有的怪都服你这个赞美你认同BOSS，你。竞争模式，是一路在确定，一旦别人不认可你甚至打击你，否定你外在的东西，你的需求得不到满足，或者创伤被激活，就很難压住情绪。

而且，你端着立一个人设，每天要装成一个懂王，做自己不喜欢的事，本身就很累很内耗，甚至胃酸，心里堵的慌，想必大家都有过感受。

而发现模式是什么呢？世界是一个地图，我在探索地图的过程中，不断发现我是一个什么人，真正的我长什么样子。跳出别人期待的自己，找回生命本来的轻松和完整，活出自己的原厂版本。我是条鱼，我就是不会跑步，我是株草，我也不羡慕大树。不是说我非得成为谁，我非得挣多少钱，我对事事努力，但对结果不抱期待，因为成不成，都是一种发现”。

对待世界要有游客心态，你是来玩和体验一番的，景区的饭菜不好吃，景色不好看，你会很生气很上心吗？不会。你会关心周围的人，坐飞机来的，坐迈巴赫来的，拎个还是吗？不会。你根本不在意他们。

这就是《金刚经》告诉我们的，应无所住而生其心。人生不就是80年的景区旅行吗？在生活中，你不断发现，哦，原来我喜欢安静，哦，原来我不喜欢冲突，哦，原来我感情很丰富，哦，原来这是我的创伤。然后，就承认那个真实的自我，我就是恋爱脑，我就是爱冲动，我就是做不到唯利是图。

再然后呢，1,接纳那个真实的自我，有些实在搞不定的，你就躲远别去搞了。你这人心思很细，别人不回信息你受不了，想东想西，内耗的不行。你也别学什么课程了，谈恋爱的时候，碰到谁故意忽冷忽热，4小时不回你，直接拉黑删除。

2,以发现的心态去体验一切。炒股亏了，割肉了，不要相信大脑的懊悔，痛苦，气愤，甚至嗔心一起，随便挑一个股重仓，结果亏的更惨。而是，你要从心的角度去发现，啊，原来我是一个优柔寡断的人呢，呵呵，下回慢慢调整。赚了，发现自己，哎呀是个有耐心的人呢，定力还可以。所有的事情，没有对错，只有不断发现的真我，这才是凡事发生都有利于我。

有利于你什么？有利你发现自己啊，这个过程，是明心见性。碰到问题不要被大脑的电子信号蒙蔽，我很心酸，我很委屈，我特气愤，而问自己，哎，这个事你发现你自己什么了？对抗的角度，你是消耗的，你是疲惫的，得到的快感越来越少。发现的角度，你是开心的，你是积累的，获得喜悦越来越多。对抗，是一种被动的应激反应，发现，是扭回人生的主导权，主动的和生命赋予自己的东西连接与和解，你才真正找到了自己。

人，越真实，越快乐，越真实，越有成就。那么,如何避免把真实搞成人性，甚至肆无忌惮，如何医治自己，已经有了的创伤呢？我们下篇再说。
