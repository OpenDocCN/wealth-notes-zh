
# 房产税可能很快就要来了

这几天专门研究了一下房产税，给大家分享一下我的思考。

我的判断是房产税快要来了，最快明年就会在部分城市试点，并且试点之后如果没有出现太大的问题，在2028年左右就会在全国落地。

那么为什么房产税会这么快到来？原因很简单，地方财政快要撑不住了。

土地财政过去一直是地方财政的重要支柱，因为房地产市场的下行，导致这个财政收入出现一个巨大的缺口。

给大家算一下账。土地财政最辉煌的顶峰是2021年，当年卖地收入8万亿，然后是房地产开发带来的关联税收——大致是卖地收入的15%左右，这样算起来，2021年地方财政从土地中直接与间接收入是近10万亿。

因为地方政府在卖地收入赚得盆满钵满，所以地方就一直在大举举债扩张，现在公布的的地方政府显性负债是47.5万亿，隐形负债是10.5万亿，合计是58万亿左右，但是这个隐形负债明显偏低，所以地方政府实际负债应该远远高于58万亿，这个沉重的负债在房地产下行，卖地收入锐减的情况下压得地方财政喘不过气来。

2024年，地方卖地收入大致是4.8万亿，加上房地产开发关联的税收一共收入是5.5万亿左右，这个收入对应的全国大致有30%的省份财政非常困难，财政收入相比支出低于35%，所以去年中央行明确提出，要给地方政府支持，要保运转。这里给大家讲一个要点，在中国，地方政府运转是必须要兜底的，不兜底不但基层会失控，而且养老金、医保这些支付不出来，也会出事。

所以，面对地方财政的困境，国家一定会想办法解决的。所以，我大致推算了一下，地方财政每年必须要从房地产收取7万亿左右的收入才能保证大部分地方财政能维持运转，在过去的模式下，这7万亿的构成是卖地收入6万亿，房地产关联税收1万亿。与这个数据相比，2024年的缺口就是1.5万亿。

到了2025年，情况变得更严峻，今年1-10月，全国卖地收入大致是2.5万亿，预计全年卖地收入不到3万亿，加上房地产关联税收也只有3.5万亿不到，与7万亿相比，就有3.5万亿的缺口。

在房地产一直无法止跌企稳的情况下，明年卖地收入很有可能继续下降，那么带来直接影响就是地方财政缺口会继续扩大，就算今年，很多地方公务员工资都是靠公务员贷款来发的，这种情况还能维持多久？所以明年房产税大概率会推出试点，截止到2025年，全国城市商品房价值大致520万亿，加上商铺与写字楼，接近600万亿，这600万亿的房产哪怕扣除200万亿的免税幅度，也有近400万亿的税基，400万亿的税基假如平均房产税率是0.8%，也能给地方财政带来3.2万亿的收入。这3.2万亿对于地方财政就是维持运转的救命钱。

未来中国房地产比较理想的发展模式大致是这样：
500万亿的商品房，每年折旧率按照2%计算，每年大致就能产生10万亿的商品房销售，对应的卖地收入以及关联房地产开发税收，大致是3万亿——3.5万亿左右，如果再加上3.2万亿的房产税收入，恰好就能满足地方财政房地产7万亿左右的生命线。

以上就是我的预测2026年试点，并且在2028年落地的逻辑。

当然，还有另一种可能性，在房产税试点的时候出现较大的问题，比如导致二手房市场雪崩，并且产生不计成本的抛售与踩踏效应，那么房产税要扩展到全国也会暂缓，但是从目前情况来看，出现这种可能性不大。

接下来就是未来房产税会怎么征收，以及对房地产市场的影响。

房产税怎么征收目前基本都是明牌，我给大家归纳整理一下：
免税范围：首套自住房、人均60平米免税、农村自建房。
税率区间，大致在0.5%——1.2%，家庭人均居住面积超过50平，估计税率就是5%，超过100平，估计税率就是1.2%。
计税方式：估计各地会不一样，但是一般会按照房产评估价值打一个折扣。
怎么征收：考虑到征税成本，估计还是以居民自行申报交税为主，政府部门会给房产税设定一个交税时间点，超过这个时间点会有滞纳金，这样倒逼居民主动纳税。

对房地产市场的 影响。
房产税的推出肯定会对房市有较大影响，包括不限于二手房挂盘量剧增，房价下跌幅度加大，越是品质一般或者较差的刚需房跌幅越大。

在房地产目前还没有企稳的情况下推出房产税，会让房市继续深度探底，具体参考样本就是日本。
日本房价在1990年到底高点后，经历了长达近20年下跌，一直到2009年才基本企稳，在此期间，全国房价平均下跌了近50%，六大核心城市高房价地区，房价跌了62%.
国内从2021年高点以来，全国房价平均跌幅大致是30%左右，一线城市北京上海跌幅是30%，广州是40%，深圳是35%左右。
我觉得中国经济基本面比日本好一些，而且房产税推出时机也与日本有区别，所以即便明后年房产税推出之后，预计全国房价跌10%左右，一线城市大致还跌15%左右就可以企稳。

所以，最后就给大家两个建议：
其1，趁着房产税还没出来之前，赶紧将自己手里多余的房产卖掉。
其2，准备上车的同学，可以再等等，在房产税出来之后再去抄底。
