
# 李嘉诚

今晚本来想等普京和懂王通电的。但时间可能拖很久，晚上9点才开始通电，他们哥俩好不知道要聊多久，怕太晚不好写文章。今天我们聊聊李嘉诚吧。之前写过，我们和李嘉诚的关系其实是不好的。早在我们主政福州的时候，李嘉诚在福州开发房地产，我们觉得是港商，就把最好的地给了他，结果他损坏了福州当地的历史建筑去开放，福州本地人举报上去惊动了中央。李嘉诚就是这样的商人，只求利。这样的德性，李嘉诚还得罪过深圳。

2004年政府出资9亿和李嘉诚谈判。李嘉诚毫不犹豫地拒绝，狮子大开口把价格提到13.5亿元，谈判多次李嘉诚都未松口。深圳市一怒之下斥资27.48亿元投资在隧道旁新建了“深盐二通道”，到2008年全线通车，大部分车辆走这条免费通道，李嘉诚为了止损，主动找到相关部门以2.5亿元的价格“甩卖”给了深圳市。因为那件事，我们不仅不得不花更多的钱修复历史建筑，而且还挨中央的通报批评，这是我们仕途上第一次通报批评。可想而知，老李和我们没有啥情分，反而有很大的恩怨。

瓜友可能想问了，这是我们唯一一次被通报批评吗？不是，我们仕途上有两次通报批评。第一次是李嘉诚，第二次是一个女贪官。2003年4月20日，杨秀珠作为浙江省建设厅的副厅长，带着女儿、女婿和孙子从上海机场逃离。我们在浙江的时候，就发生了这件事，这个女贪官外逃。因为这件事，我们又被通报批评了。这就是我们两次挨批的情况。

对这两次挨批，我们可想而知是十分愤怒的。后来我们搞了一份红色通缉名单，这个女贪官位列第一位，其他比她级别高贪污比她更多的贪官，都没有她这么重要能位列第一。所以啊，李嘉诚很明白，我们对他的态度如何。因此，2013年我们上位后，李嘉诚就开始大面积抛售大陆资产，全面撤出大陆。李嘉诚知道，我们迟早会找到他现有的过失收拾他。

包括后面的香港暴乱，香港几大家族除了霍家外，都是反对我们的。特别是李家。这个梁子没有缓解，反而越结越大。最终，有了最近的新闻。与美国贝莱德牵头的财团达成原则性协议，拟向该财团出售旗下和记港口集团80%的资产，转让其持有和运营的分布于23个国家的43个港口及配套物流网络，包括巴拿马运河两端的巴尔博亚和克里斯托瓦尔港口。李嘉诚把港口卖给美国，有很多层次的背景。

国际博弈的背景，就是英国为了力求美国恢复对乌克兰的援助，威逼李嘉诚卖资产给美国。中国政商的背景，就是李嘉诚已经不可能回中国了，他只有海外发展全球发展这条路，美国英国是他决不能得罪的，所以李嘉诚跪的很快。因为李嘉诚知道，他不是华为。我们会力撑华为不让华为跪下，但我们不会力撑李家。李嘉诚面对英美的威逼，撑再久最终结果也是卖资产。迟早得卖，那就晚买不如早买，因为晚卖肯定卖的钱更少。

李嘉诚光速下跪，我们觉得很耻辱。但在他看来，他光速下跪是卖了一个尽可能的最高价，他赚了。所以，李嘉诚卖巴拿马运河港口给美国，这件事，我们无法怎么指责他。换你在那个位置，你也得卖。福州往事+香港黑暴乱，注定了李嘉诚无法重新投诚我们，我们不会接纳李嘉诚，李嘉诚已经不是未来也不会是中国商人。但是，李嘉诚这件事还是很让我们恼火。因为他影响了我们对美博弈。

我们和美国的谈判博弈中，美国的巴拿马运河格陵兰岛加拿大这些，都是懂王想要的，我们对这三个地方的影响力很有限，制约不了多少懂王。李嘉诚的巴拿马运河港口，算是我们仅有的几张牌，结果李嘉诚举手投降，我们牌也没法打了。而且，李嘉诚不是只卖巴拿马，他是把他手上的全球港口都卖给了美国。这就过分了。你李嘉诚如果只卖巴拿马，我们还不会怎么怪你。知道你扛不住，知道是你迫不得已。但英国美国根本没想要你全球港口啊，要的只是巴拿马运河港口的，你李嘉诚怎么卖全球港口？

——“与美国贝莱德牵头的财团达成原则性协议，拟向该财团出售旗下和记港口集团80%的资产，转让其持有和运营的分布于23个国家的43个港口及配套物流网络，包括巴拿马运河两端的巴尔博亚和克里斯托瓦尔港口”。这不是英美的威逼，而是李嘉诚的险恶用心。很简单，李嘉诚的港口分布全球，可以说我们一带一路的很多港口，都是李嘉诚早年买下的资产。现在，这些港口都是美国人的了。中美一旦撕破脸，美国会更容易全方位封锁我们，因为李嘉诚助攻懂王反华了。所以，岱岱一言以蔽之。

你李嘉诚只卖巴拿马港口，你是被逼无奈，我们不会怪你。你李嘉诚把所有港口都卖了，你李嘉诚是资敌反华，我们肯定要搞你。所以我们最近骂死李嘉诚了。

中央港澳工作办公室与国务院港澳事务办公室的网站先后在13日和15日转载了香港《大公报》的锐评文章“莫天真 勿糊涂”及大公时评文章“伟大的企业家都是铮铮爱国者”。相关文章说，这笔交易“引发国人高度关注和强烈质疑”：长和为何轻易将这么多重要港口转让给“不怀好意的美国势力”，所谓“精明”的交易有没有考虑到国家民族利益？香港大公报五问李嘉诚，并告知“交易非普通商业行为。面对如此大事大义大节，有关企业当三思，应好好想想有关问题的性质和要害是什么，好好想想自己要站在什么立场、站在哪一边”。

当然，李嘉诚也有笔杆子。联合早报就帮李嘉诚说话。举大旗，说官方笔杆子攻击李嘉诚，是违背一国两制。你看，李嘉诚装都不装了，直接和我们对着干。如果说，香港黑暴时期，李嘉诚还只是暗地里和我们对着干，还公开对抗中国。那么今天故意把全球港口卖给懂王，就是李嘉诚公开和我们对着干了。

李嘉诚已经是公开的反贼公开的商贼。然后继续说笔杆子。其实李嘉诚早早就布局了笔杆子，香港时期，有一个记者写文章讽刺李嘉诚的香艳往事，说他带李嘉欣回家偷换，被他发妻看到了，两人吵架，他发妻直接当场身体不好被气死了。说那个小儿子目睹了这一切，看到母亲死在自己眼前，受到了巨大心理创伤，一度不认他这个爸爸，去娘家生活了很多年，多年后才原谅他这个老爸。李嘉诚两个儿子都有心理创伤，大儿子被绑架过，听说至今怕黑不敢独处狭小黑暗的空间，小儿子则是母亲的巨大刺激。所以李嘉诚对两个儿子有很大的愧疚心理。

反正这件事被那个记者捅出来后，李嘉诚直接砸钱叫人，砍断了那个记者的手。当时写文章是靠手写字的，砍断人家手，不仅是血的教训，也是砍断那个人吃饭的饭碗。买凶砍人后，李嘉诚也开始资本布局笔杆子。当然，李嘉诚他自己看来，砍个手算比较温和的报复了，毕竟按照李嘉诚这个级别的财富，他的确可以直接让那个人消失。比如张子强就消失了。张子强抢劫他儿子后，其实李嘉诚没有报复张子强，他还很聪明，知道张子强爱赌，多少钱都会赌光，到时可能会再找他，他很麻烦，于是他劝张子强把钱买李嘉诚公司的股票，说保他三代富贵，然后帮他移民外国让他过上富豪生活。

张子强也是傻，或者说过于自信过于留恋江湖生活，没有答应。后来过来赌博输光了，于是张子强筹划其他富豪绑架案，继续赚大钱。但香港富豪在几次被绑架后，都重金聘请保镖，还改造升级了防弹车，那种总统安保级别的防弹车。张子强的ak47都打不破那些车的车玻璃。这碗饭没法再吃了。于是张子强转头找到李嘉诚，他竟然觉得李嘉诚是这些被绑富豪里头最好说话的那个，于是多次找李嘉诚打秋风，说是借钱，其实就是敲诈嘛。

李嘉诚借了一次又一次，实在忍不了，这是个无底洞，只有千日做贼哪有千日防贼的话。所以李嘉诚动用了大陆和霍家的关系，因为香港治不了张子强，判不了张子强死刑。于是我们看到，张子强有此跑到内地后被内鬼叛徒出卖位置信息，霍家有黑道背景帮忙的，然后张子强立马被大陆抓到，而且光速宣判光速枪毙。李嘉诚才长舒一口气，安稳了。李嘉诚也是有手段的，毕竟上个世纪的香港是草莽江湖时代，没手段当不首富。只不过，他这次是他人生以来最大的危机。香港黑暴之前，我们就算看不爽李，我们也不好动手。毕竟他那个时候举白旗投降了，没有反对我们没有反华。我们搞他这个级别的人，会被很多人说我们公报私仇，说我们寒了香港商界乃至华人商界的心。所以我们还真的没动他，让李嘉诚安然甩卖资产退出大陆。他只要不反华，我就不会公报私仇的搞他。香港黑暴爆发后，我们才看清李嘉诚反动的本质。我们在香港黑暴后是真的想搞李嘉诚了。但李嘉诚早早跑没影了，资产几乎都在欧美，大陆没有，拿捏不了李嘉诚。

他在香港还有资产，但一国两制我们也不好。现在，从暗地里反华到公开反华，李嘉诚竟然公开助攻懂王高我们，我们绝对是搞死李嘉诚的心都有了。其实李嘉诚和懂王没有多少交情。甚至还有点小交恶。因为李嘉诚手里有笔天量财富，就是facebook股票。4%左右的股票。而facebook的小扎和懂王是交恶的，当然现在小扎很跪舔懂王。这笔投资是facebook早年危机寻求融资的时候，李嘉诚那个女的红颜知己，劝说李嘉诚投资的。现在fb市值已经1.47万亿美元，李嘉诚等于有500亿美元的财富在里面。当然，李嘉诚早早把这个股份捐给了他的基金会慈善会，用来避税了。所以名义上财富统计的时候，没把他这笔捐出去的钱统计进去。周润发也是这么做的，他说钱都捐出去了，实际上是左手倒右手，避税罢了。周润发其实是暗独，他也是暗地里支持港毒的，他不太喜欢我们大陆政府，觉得我们管制的太多太不自由了。记的刘銮雄说过，他的财富被低估了，香港前几家族财富都被低估了。

业界也有个说法，说这么多年，中国首富其实一直还是李嘉诚家族。李嘉诚太会藏钱了，中国钟睒睒全部身家才500多亿美元，他一个facebook股票就顶钟睒睒全部身家了。李嘉诚是刻意低调的，他又分家给两个儿子，李嘉诚家族全部财富可能有一万多亿人民币。可惜啊，我们无法打土豪分土地补充国库。手伸不远，要是有懂王的全球执法权，李嘉诚这万亿财富就等着被收归国有补贴国民吧。
