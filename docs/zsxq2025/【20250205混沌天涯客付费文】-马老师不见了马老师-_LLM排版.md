
# 马老师不见了马老师 

我是不太喜欢“卢”这个姓的，这源于《水浒传》里的卢俊义。卢员外，空有满门财富一身武艺，被拐上梁山给宋江当傀儡。关键是，历史上的傀儡多是一号位，如汉献帝，吉祥物一坐吃喝不愁。卢俊义这傀儡当的还是个二号。空待在梁山喝酒吃肉是不行的，他要负责下山打家劫舍，在兄弟们目光如炬的监视下，为赚点吃喝打得汗流浃背。窝囊。

与之相对，“鲁”这个姓我就天然喜欢，因为有一种故乡感。孔子登东山而小鲁，有位二代爷生在内战时期的山东，就取名作小鲁。还有常青树鲁豫姐姐，姥姥家是山东人，名字就带了鲁。再就是民族的良心鲁迅先生，我一度认为他姥姥家也在山东，后来才知道他妈姓鲁。

我喜欢鲁，连带着它的近义词撸，撸什么干什么,朗朗上口。

上面我查准字典用了一句成语，成语里带出一位钢琴家，郎朗。

郎朗弹过的曲子很多，没怎么听过。但有一首，一首郎朗自认为是中国人心目中最美歌曲的那首，旋律一开，朗朗上口：“一条大河波浪宽，风吹稻花香两岸。我家就在岸上住，听惯了艄公的号子，看惯了船上的白帆……”

2011年1月21日，郎朗施展他鬼魅般的手指，疾风骤雨般敲打着琴键，让这首曲子响彻在白宫的夜空。

那时候白宫的主人是傻大个奥巴马，呲着白牙笑得高兴。奥巴马笑，白宫里那些长着五颜六色头发的政客也跟着笑，边笑边鼓掌，为这天籁般的曲子喝彩。

只有同坐的黑头发黄皮肤的宾客，脸上浮起一种神秘的笑意。这笑意的内涵在曲子的第三段揭晓：

“朋友来了有好酒，若是那豺狼来了，迎接它的有猎枪。”

朋友是谁，豺狼是谁，在这首曲子里是有专指的。因为这首曲子来自一部电影《上甘岭》，脍炙人口，家喻户晓，几代人不用郎朗提醒也能朗朗上口。朋友指General Jin Generation 1；豺狼，则专指上甘岭另一边的美国佬。

总统奥巴马傻笑，国务卿希拉里傻笑，白宫里一众美国政客傻笑得不行，他们压根儿不知道，对他们最犀利的怒喝，已经端上在白宫晚宴的高潮中。

天底下最痛快的事儿，就是当面骂了他他还不知道。

所以，当从卢比奥改名作鲁比奥的美国新国务卿被告诫要"耗子尾汁"的时候，我真怀疑他听懂了没有。

他听没听懂谁管他，反正我们都懂，哈哈。

鲁比奥姥姥家不在山东，而是在古巴。他1971年出生在美国迈阿密。

迈阿密那个城市漂亮极了，蓝天、阳光、沙滩、美女，而且非常摩登，全市有超过300栋摩天大楼，汇集着诸多金融、商业、媒体、娱乐、艺术公司，被评为美国最干净的城市和最富裕的城市。

但每一座漂亮城市背后，都有不堪入目的血泪史。知道建设这座城市的主力军是谁吗？是古巴移民，更准确说是古巴流民。从1959年开始，我们最著名的老朋友之一卡斯特罗开始执政，这位有着崇高理想勇敢心灵的斗士，看不惯治下那些被美式生活方式腐蚀的同胞。教育他们?难。改造他们?费钱。怎么将这些不可救药的人清掉，打扫干净屋子重建一片新天地?幸好古巴有个邻居一一迈阿密。

风向顺的时候，弄一艘小船一晚上就过去了，古巴那些跟不上新形势的臭鱼烂虾，扶老携幼，拖家带口，趁着夜色开着船就走。

幸好古巴是个岛国，家家有船人人会游泳；卡斯特罗也足够仁慈，与其浪费子弹，不如让这些烂货走。从1960到1980,持续了三十年，每年都有数十万古巴人偷偷渡到了迈阿密的码头。在1980年的马列尔事件中，有150000名古巴人一次性渡海到达迈阿密，创了吉尼斯世界纪录。

鲁比奥的父母，就混杂在这些臭鱼烂虾中。

这些人到达迈阿密，肯吃苦，能付出，脏活累活抢着干，摆地摊、卖煎饼、住地下室，捅下水道……终于，1966年，铁石心肠的美国人松了口，出了一个针对古巴移民的法案，给所有来美国居住满一年的古巴人统统上了户口。

由此可见，美国人是真坏啊。他们知道这些古巴人通过辛勤劳动赚到了钱，他们生怕这些古巴人把钱带回去建设家乡，索性，一纸文件，收买人心。

本就是臭鱼烂虾坏种子的鲁比奥，就这样被美国政客收买，长大后他也加入进去，成了美国政客里的臭鱼烂虾。

还没上任，他就放言，要把古巴列入支持Terrorism的黑名单。

数典忘祖。

在我们山东人眼里，古巴可以说是世界上第三好的国家，80%的工作从属于公共部门，为国家雇佣，换种说法就是带编制。

鲁比奥不第一时间回乡祭祖，到爷爷坟头哭一哭，反而是主动带头恶狠狠对家乡围堵。其心可诛。

但不可否认的是，奥巴马傻大个的时代一去不复返，鲁比奥这东西，内外通透。

世间所有的道理，都藏在过去的故事中。这些故事我们从小就朗朗上口，无它，按情景——一八小填空就行。

疏和堵，说得最清楚的是大禹和他老爹。大禹治水，用的是疏，将洪水疏导进河沟再汇流到海；他老爹，愚蠢的用堵，堵是堵不住的，没用，越堵越成洪水猛兽。

老朋友卡斯特罗，深知大禹治水的妙处，古巴那地方以前怎么样，看过《教父2》的都清楚，花天酒地，歌舞升平，不把习惯于这套东西的那一代人赶出去，他的理想天地不可能长久。

这可以叫过滤，淘汰，物竞天择，择良木。双向选择良木。

就像是卢俊义，被宋江一伙儿选中做了良好的木材，不只因为他多金会武功，更因他内心孱弱，安于服从。

同理，鲁迅作为民族良心，晚年住在上海，租界。租界里除了洋人就是买办，能有几点文化氛围。但鲁迅知道，保持良心在线，还得住上海。彼时的上海，即是鲁迅选择的良木。

卢和鲁，这是没有办法的事儿。

就像现在，当马保国老师的金句传递到鲁比奥耳朵中，卢比奥听没听懂不知道。但地球人都知道的是，马老师的武功，真要是走到擂台上，好像不太行。

但这并不重要。只要大家高兴，只要高兴的人多，不行也行。马老师算山东人，大半辈子在河南居住，但他现在已经搬到上海了。

上海黄金地段，毗邻鲁迅故居，流量大，粉丝多，手舞足蹈一番比划，语重心长说几段金句。对内，这是属于73岁马老师的时代；对外，刚满60岁的那位马老师已溜走。
