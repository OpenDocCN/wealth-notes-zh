
# 2025年“反内卷”是最重要的事情--回归正常、回归有序，这才是正常有序的市场竞争

继续内卷下去，所有人都将会是输家。2024年，互联网讨论度最高的词语中，“内卷”绝对能排前三。小到街头的小摊贩，奶茶店的店主，大到互联网企业、光伏、新能源汽车等产业，全都卷入了内卷的漩涡。这本来是一个经济学概念，指在资源有限情况下，投入人力越多，人均收入越低。但在当下，它却成为席卷各行各业的“紧箍咒”——企业打恶性价格战换销量、员工加班拼时长拼效率，但最终收益却不断走低。

一个最能说明国内“内卷”的典型案例是：欧洲人拿中国的光伏面板当围墙当栅栏。由于中国的太阳能面板价格太便宜了，价格最低跌到0.8元/瓦，比水泥板还便宜，很多欧洲人直接买来当作围墙使用。好好的科技成果却当成了废物利用，真的是暴殄天物，而且还引发了国际上对于中国光伏产业的警惕。国内光伏业龙头，隆基绿能董事长钟宝申说，他有一次去能源局开会时，一位领导总结当前光伏行业是“内卷外化，低价挨骂”。

制造业“皇冠上的明珠”，汽车行业也不例外。数据显示，2023年国内上市的18家车企中，只有6家盈利，另外12家亏损之和还赶不上丰田一家。背靠吉利和百度的极越汽车，以及哪吒汽车逐渐要被淘汰出局，小鹏、蔚来这些头部的车企基本上都是亏损的状态，“卖一辆亏一辆”。

席卷各行各业的“内卷”之风，是如何刮起来的？用低价竞争，换市场增长，形成规模的绝对优势后，再精细化运营换利润，这是消费市场高速增长期的典型打法。但需要看到的是，居民消费需求已经发生巨大变化，人口红利带来的增长奇迹也很难持续，当外部需求放缓时，还在追求“规模优势”“极致效率”，这套模式就变成了“卷生卷死”。内卷这场游戏，没有人是赢家。

就在去年，电商平台的仅退款更是引发众多店主的抗议。我们和很多做电商的小老板聊过天，大家都表示，别看他们营业额一年几百万，上千万元，最后算下来还亏本，要不是想到还有那么多员工要养，他们早就不干了。这个竞争压力，也在向供应链上游传导。还有江浙一带的老板，看到如此内卷的环境，利润从10%压缩到现在不到3%，而且账期动辄就是大半年，还给的是商业票据，如果这个公司后面出了点啥问题，要账什么的，那就彻底完蛋，一毛钱也要不回来了，都是纯赔的。于是有些老板就打算收手，安抚好老员工，做好赔偿后，就彻底把工厂关掉了，反正以前靠着时代红利也攒了不少钱，这辈子够花就行了。

市场经济讲究的就是赚钱，盈利。任何一家企业赚不到钱，不断内卷下去，始终都是走不远的。而这一切，最终导致的就是各行各业就业容量萎缩，可以说是全民都在为“内卷”买单。正式因为看到内卷竞争的害处，国家终于出手了。2024年7月的中央会议中指出要强化行业自律，防止“内卷式”恶性竞争。同年12月的中央经济工作会议中，更是明确指出，综合整治“内卷式”竞争，规范地方政府和企业行为。

一场“反内卷”风暴正式打响。很多人可能还没有留意，对于反内卷的政策表态已经升级了。最开始提到反内卷的时候，用词还只是“防止”，如今则是“综合整治”。从这个表态的转变来看，“反内卷”势必要全面铺开，覆盖各行各业。早在去年，国内便有多个行业公开呼吁，要加强行业自律、抵制恶性价格竞争，让行业重新健康发展。2024年8月，中国水泥协会发布《关于征求加强水泥行业自律政策建议的函》，要求强化行业自律，防止“内卷式”恶性竞争。12月，光伏行业中国光伏行业协会联合16家光伏头部企业召开专题座谈会，就防止“内卷式”恶性竞争，强化市场优胜劣汰机制等进行交流。同月，12家风电公司签署了《中国风电行业维护市场公平竞争环境自律公约》，抵制低价恶性竞争。

行业在呼吁，企业家也在站出来明确表态。2024年底，吉利控股集团董事长李书福就明确表示，必须旗帜鲜明地反对“内卷式”恶性竞争，不搞大跃进式的恶性发展，不追求短期销量目标，放眼长远，坚持长期主义。不过，与其他行业大张旗鼓的高调表态不同，2025年开年外卖就悄然打响全方位“反内卷”第一枪。上周，“美团逐步取消骑手超时扣款”的词条冲上了热搜，并且在元旦后在全国全面上线“跑单满12小时下线休息”的防疲劳机制。

超时对于骑手来说就意味着收入减少，为了准时送达，骑手是一刻钟都不能放松，时刻紧绷着穿梭在各大街头。在生活、经济等多方压力下，骑手小哥都希望每天能“多跑几单”，工作强度愈发紧张，收入天花板也始终无法突破。如今美团主动修订规则，释放了一个明确的信号：那就是平台不再鼓励“单王”式的内卷竞争。从收入分布来看，月入过万的单王毕竟是少数，大量骑手还处于中位数收入期间。但人人又都想当“单王”，引发了拼时长、拼速度的“内卷”。现在有了“上限”，收入分配也更加公平，对于骑手而言也解除了心头的一大焦虑点，终于能喘口气了。

叫停的同时，还有疏导。美团通过“取消超时扣款”就是个典型样本，从负向激励转向正向激励，骑手送慢了不再扣钱，送的好还有激励。这个信号再明确不过，从拼效率变成拼服务，让骑手之间拼单量的互相竞争，变成服务顾客为导向的竞争。在为产业链下游的骑手松绑的同时，美团还在为产业链上游的商家减负。新年刚开始，不少餐饮老板在社交平台上晒出美团的“新年礼包”——后台直接收到现金奖励，少的数千，多的上万，有零有整。这是美团近期刚刚面向商户推出的10亿助力金补贴计划，里面既有老字号，也有社区小店，只要是认认真真搞产品创新、升级服务的，都有补贴。

这其实是，平台通过主动让渡利益，向全行业释放风向标：不要再低价内卷，要回到产品和服务 的良性竞争上来。现在餐饮行业产能过剩，已经成了一片红海，不少商家将经营当成了淘汰赛。为了更多的订单，商家拼命砸钱做营销，价格不断下调，当一家商家这么做的时候也许还有点成效，可当所有商家都这么玩，最终把获客成本抬高到无法承受，利润全被“内卷”卷跑了，这就是一场餐饮版的“囚徒困境”。

现在平台主动出手，打破了这种竞争僵局。这笔补贴金额虽然不多不少，但对于拼现金流的餐饮商家来说是“及时雨”，能够有更多的信心，放在产品、服务创新上来。在这之前，美团还大刀阔斧改进了流量分配机制——弱化营销因素在流量分配中的权重，鼓励餐饮良性竞争。要知道，对于平台型企业而言，流量就意味着直接的现金收入。如果流量分配机制不再倾斜愿意买量做营销的商家，那么势必会影响到平台的收益。但美团依旧还是迈出了这一步，通过牺牲自我利益的革新，给全行业带来表率：外卖关乎万千实体商户和外
