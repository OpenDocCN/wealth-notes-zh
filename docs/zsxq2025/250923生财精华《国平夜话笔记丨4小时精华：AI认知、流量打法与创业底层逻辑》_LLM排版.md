
# 国平夜话笔记丨4小时精华：AI认知、流量打法与创业底层逻辑

## 一、内容与爆款的打法

「让内容在渠道中流动。」不应该追逐爆款，而是应该让爆款持续爆。内容是流动的，不是孤立存在，而是要在不同渠道之间产生迁移和放大。

爆款不是偶然，要靠“策略为王”，策略优先于战术。持续爆款 = 策略框架 + 情绪价值 + 流量杠杆。

爆款需要很强的数据挖掘能力。不要去抄爆款，而是要去挖掘还没有被满足的隐藏的需求。

## 二、AI是什么，究竟怎么用？

做私域的朋友，给粉丝们群发消息，发完就掉粉。解决方案有两个：第一，不群发，而是让客户主动来找你；第二，发了，就需要让ta感动到哭，需要对ta十分了解，能提供无与伦比的情绪价值。

谁来提供情绪价值？用AI。

AI做客服和获客，是因为AI可以读懂人类语言的字面意思和隐藏信息。但国内的AI，都是来劝退我们用AI的。有一些厉害的程序员排斥AI编程，主要是用了不好的模型，让他们用AI编程出错很多。目前做编程最好的模型还是Claude code。

现在的人用AI，思路都是「把铁锄头做成了金锄头。但是这个锄头，真的需要存在吗？」

试着所有的场景，都用AI去颠覆一下。

AI不是目的，AI的应用场景才是关键。打个比方，现在的AI大模型就像汽车的引擎，而一辆车是必须有好的轮子、车架、玻璃等才能跑起来。

AI就是一个电子大脑。用数学和硅基组建了一个神经网络，而人是用蛋白质组成了神经网络。

「未来应该是用AI来接管物理世界，进而来影响物理世界。」

再打一个比方，现在的AI就是当年的蒸汽机。把蒸汽机用在纺织厂后，产生了纺织巨头；把蒸汽机用在火车上，进而需要修铁路和钢铁，就产生了后来的钢铁大王。

AI也要找到合适的场景与基础设施。在这些不同的场景中，也会诞生很多巨头，目前来看，普通人还有机会。

工作流将被淘汰。未来工作流不是人设定，而是AI自动拉出来。

所有的软件都值得在旁边再加一个智能体。让软件能操控物理世界。

例如炒菜机，现在是只有四肢，能用AI来做它的大脑。类似的场景有很多很多。

比如，在找工作和找对象这件事上，也可以用AI来做。让AI去筛选分析20个目标岗位，去投4000个机会；告诉AI你的理想型究竟是什么，然后让AI自动登陆你手机的相亲软件，去加2000个人，然后筛选50人，然后再从中选择。

用AI创业，业务需要逻辑闭环。要计算好成本结构，做到先胜而后战。同时创业成功也很讲究正确的时机，也需要计算成本结构。

在国平老师看来，很多人还在用“关键词/知识库/相似度”这类落后的方式。

## 三、什么人能用好AI？

未来的AI赢家，是脚踏两条船的人。一边很懂业务逻辑，一边非常懂AI。对AI的真正认知来自于对AI的实践。多用，疯狂用AI的算力。

很多人不知道自己是用AI的高手。其实一个人，如果平常很擅长解决问题，也很能理解AI，用好AI。

懂工作流就是懂代码。工程师就是有解决问题的思维。这个就是算法思维。

如果你不是代码高手，那是用AI编程的时候，有一个办法：让AI写代码的时候，每一行都写上注释，并且用最详尽小白也能看懂的形式展示。

用好AI，会让一些人的迭代速度提升100倍。

用Claude code需要配合很多厉害的插件、mcp、浏览器来做，这样它就是一个生态。现在几百人的开发团队，用好Claude code可以砍掉两个零，留下几个人。

## 四、信息产业的本质和内容平台消亡

国平老师一个大胆的预测：3年之内，AIGC会毁掉所有的内容平台，还在疯狂拉低内容的下限。

因为蚂蚁雄兵在往里面灌垃圾内容，比如用n个账号发布水货内容。

那我们可以做的就是，在平台死掉之前，趁这个时间差，疯狂地屯用户，疯狂拉私域。

我们从事的是信息产业。信息产业的产品质量，就是信息质量。

互联网公司需要提升信息质量。

互联网公司不生产钢铁，产出的就是信息。谁能提供更高质量的信息，谁就能崛起。新的平台起来，旧的平台就会开始死去。

「百度、知乎已经死掉了」，因为内容质量不够好。

平台要做的就是提供信息，谁能提供更好的信息谁就能起来。就算是大厂去做内容，只要内容信息质量不够好，这个业务也是做不起来的，比如微信问一问。

平台与用户的关系，商家和用户在平台眼里都是“驴”，拉磨换胡萝卜。

平台扶持一个流量的标杆（如罗永浩），只是制造“可复制的希望”，让更多内容创作者看着希望去勤勤恳恳地给平台更新内容。

这就是平台给商家和用户设的局，劝大家不要轻易入各种局。而是应该有掀桌子的勇气。

卖希望，是世界上最好的生意。

## 五、搞流量的底层逻辑

在洞察需求、找到流量洼地这件事情上，国平老师做得非常极致。

如何挖提示词？

比如劳力士的手表，有3w个词。比如汽车行业，找到150w个词，把汽车之家等汽车垂类网站的所有的信息，整站下载。然后在进行数据分析。

比如安可创新，一个做充电宝充电线的品牌，在他的指导下做起来成为行业第一。其中一个就是，他们穷尽投了50万个词。例如母亲节礼物也可以投，只要roi合适，就是好词。

能想到这个，需要的是思路的转换。

杠杆思维，没有杠杆的流量操作不要做。一个项目2两个月，花1.5个月只为找到杠杆，也是值得的。

在搞流量上，很多人战略失误：去做流量的生产者，不是流量的整合者。

打个比方，其实不需要为了吃一个鸡蛋或者一只鸡，开了一个养殖场。

世界上永远有在角落里无人问津的流量，只要把它们找出来就可以了。

做整合的人，会比做生产的人，赚更多。这里举例shein的案例。

整合胜于生产。绝大多数人只是在“生产”流量，真正赚钱的是“整合”流量的人。

价值洼地思维，举个例子：下雨天，普通人接水，高手挖坑，赢家把坑挖到最深。先找到结构性低估，再深挖到极致。

## 六、创业就是把对手的钱，为己所用

美团是消耗了几千亿资产的结晶。

从之前的百团大战到现在的外卖大战，短暂的1-2年之内，美团先由着饿了么、京东花钱来投入市场，等市场成熟，就来收割。

美团之所以敢这么卷，主要是因为他们内部的运营效率更高，至少高20%。那么在采买流量上，就可以用高对手10%的成本出手。并且保持这个20%的效率优先优势，这个游戏继续玩下去，一定会胜利。
