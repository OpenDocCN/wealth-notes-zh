
# 高低切，低位在哪里

大盘冲击60日线的关键时刻，一蹴而就比较难。现在的问题是，低位在哪？一条是高位快涨不动了，芯片，模型很高。AI驾驶，AI办公，AI金融，Al驾驶之前专门谈过智驾ETF，具体厂商看谁？最好是既有机器人，又有汽车视觉的。智驾芯片也可。另一条是大科技线的相对低位，即24年的“未来产业”。

2会报告，在战略新兴产业领域，新增的未来就两个，一个是合成生物，一个是低空经济。生物出效果太慢，另一个就是低空。低空，刚刚有专门的通用航空ETF，龙头就是大家熟悉那几个，中信海直，万丰奥威等，当然，如果你还能跟机器人挂钩，更好，很多电机电控，是通用的。低空也是25年准备的主线(3)。这几天，新华日报连续发三篇文章问。机器人ETF仍值调整期，逢低慢慢吸。琢磨透逻辑后，做自己熟悉的，不要胡子眉毛一把抓。科技上的微操，直接看恒生科技的风向标即可。
