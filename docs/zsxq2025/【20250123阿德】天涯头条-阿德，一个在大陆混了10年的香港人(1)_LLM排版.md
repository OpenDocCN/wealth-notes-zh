
# 阿德，一个在大陆混了10年的香港人

2017年是香港回归祖国20周年，也是我个人在大陆混的第10个年头，所以我想用文字跟大家分享一下自己在无亲无故的情况下在大陆生活十年的心路历程，从我24岁到马上35岁的这段岁月。

我知道现在大家都不像以前那样那么喜欢香港了，因为近年发生了很多冲突，闹得很不愉快，但正因为这样，我才想把自己的故事写出来，希望能让大陆人多了解一下香港，同时也让香港人多了解一下大陆。

随着大陆和香港的对立激化，我也难免躺枪，刚来大陆那阵子（2007年），当我说自己是香港人的时候，人家会表现得对我格外友好。

但我现在一说自己自己是香港人，对方好像会不自觉地把我当成是那些殴打大陆游客的香港人，甚至会问我“你是中国人吗？”

其实说真的，最好不要问香港人这个问题，因为这个问题本身就很奇怪，就像你是男人，我却故意问你，你是男人吗？你听到会有什么感觉，会不会觉得我在找茬？

已经回归祖国二十年的香港人不是中国人，还能是哪个国家的人？

你可以试试问香港人这些问题，比如“如果给你穿越，你会去哪个朝代？”“你对敦煌莫高窟感兴趣吗？”“你想在冬天的时候去北京故宫看雪吗？”然后你会发现香港人骨子里其实是很中国的。

你有没有听过香港群星大合唱的《滔滔千里心》？

这是一首专门为华东水灾筹款而唱的歌，我之所以印象特别深刻是因为当时整个香港——从大街小巷到电视机前——都在循环播放这首歌曲，为了给受灾的大陆同胞筹款。

1991年的华东水灾是中华人民共和国历史上第一次大规模、直接呼吁国际社会援助的自然灾害。据当时初步统计，安徽全省受灾人口达4800多万人，江苏全省受灾人口达4200多万人。200万无家可归的灾民在淮河大堤上搭起了一眼望不到头的临时帐棚，并已有灾民患肠道疾病和疟疾等传染病。

当时香港演艺界人士为了赈灾筹款，在跑马地举行了一场规模空前的大型音乐会叫“忘我大汇演”（其中beyond唱《大地》那段视频特别动人），在短短十天时间，香港的赈灾筹款总额已达到4.7亿多港元，远超其他国家的捐款援助。

建议你有机会真的要去听听这首《滔滔千里心》，唱得特别有感染力，当年九岁的我听着听着也忍不住流眼泪了。

这是当年这首歌的视频，很旧很不清晰了，毕竟那是25年前。

所以真的还有必要明知故问香港人一句“你是中国人吗？”

有一次我在青旅遇到一个香港人，他是第一次来大陆旅游，他一开口就跟我吐糟大陆人不排队、说话大声粗鲁、随地扔垃圾这些鸡毛蒜皮的事。看他对大陆的误解那么深，我心里挺不舒服的，为了让他变得包容些，我尝试跟他说说自己对大陆的理解。

“你在大陆坐过巴士吗？”我问他。

“坐过，顶呀，那些大陆人都不排队的，巴士一来就一窝蜂涌上去，超没素质！”

“大佬呀，怎么排呀？这里又不是香港，巴士站会有指示告诉我们该在哪里排队，大陆的巴士站根本就没有设计成可以让人排队，所以没法排呀！那你有坐过大陆的地铁吗？”

“有！”

“大陆人有排队吧。”

“有……”他不太甘心地承认。

“对呀，那是因为地铁月台上清楚写着该在哪里排队，所以排不排队是环境造成的，不是大陆人天生就不排队，这点你能理解吧。”

“嗯。”他点头。

“还有你说的大陆人乱扔垃圾的问题，唉，拜托，大家都是香港人，就别装了，要不是因为乱扔垃圾会罚1500，你觉得香港人会有现在那么有公德心吗？说白了，这也不过是环境逼出来的素质而已！其实我觉得随着国力的提升，大陆人的素质肯定会越来越高，这是毫无疑问的，但这需要时间呀！毕竟有那么多亿人！反正我可以很肯定的告诉你，你现在所在意的这些大陆人排队不排队的问题，随着时间的推移，最后都不会是问题！”

那个香港人和我算是他乡遇故知，所以我们吧啦吧啦聊了好久，最后他还问我，“如果有一天大陆和香港发生很大很大的冲突，你会站在哪一边？”

说实话，我有想过局势会变差，但从来没想过会差到只能二选一的地步。这是我对他的回答：

“我不会因为自己是香港人而盲目地站在香港那边，但我也不会因为现在人在大陆混就识时务地站在大陆这边，我想我会尝试化解这个冲突吧，因为这个让人痛心疾首的冲突本身就不应该存在！”

要说自己怎么突然就来到大陆混，而且一混就是十年，那就得从2006年说起。

2006年7月某个下午，我在家楼下突然被两个男人拦截，他们一左一右地把我夹在中间，顿时让我动弹不得，然后左边那男的从口袋里亮出证件：“我们是警察，你应该知道我们为什么找你吧？”

我没有太惊讶警察的出现，我反而奇怪他们怎么这么晚才来逮捕我，我都等了大半个月了，我明明已经留下了足够的线索，傻瓜都能抓到我了。

我很冷静地回答警察说，知道。

警察看我没有反抗而且又那么合作，便透露说：“我们已经留意你好几天了，本来还以为你是被人陷害。”

“没有陷害，是我做的。”

没想到警察听到之后却有点激动：“我顶呀！真的是你做的？你怎么会傻到连回邮地址都写上去！”说完，他掏出手铐，咔擦一声拷在我的手腕。

警察在我家搜集完证物之后，就把我押上警车，送到警局。在落口供之前，警察说了一句tvb警匪剧里经常会出现的对白：“而家唔系事必要你讲,但你所讲既野会成为呈堂证供。”

然后就开始对我进行正式的盘问：“刘燿德，你是否承认在2006年7月3日寄了一封恐吓信给日本内阁官房长官安倍晋三？”

关于给安倍晋三寄恐吓信这件事，说来话长。首先我想说一下，香港不像大陆，一打开电视就是勿忘历史的抗日剧，香港社会基本上不关心这段已经过去几十年的历史。相反，大家爱日货爱到不得了，日本动漫、日本av、日本零食、日本料理以及日本电子产品等等，通通大受欢迎。

日语更是最多香港人学、排行第一的外语。（英语在香港不是外语，是官方语之一。）而我因为从小喜欢看动漫，所以在2004年也开始自学日语，两年后还考过了日本语文能力试的2级。当时为了学好日语，我每天都会上网浏览日本当地的报纸，所以比一般人知道更多日本的时事，比如2006年日本首相选举，大热首相候选人安倍晋三竟然高调声称：当选首相之后会继续参拜靖国神社！

为了表达抗议，我用日文写了一封信，然后寄到安倍晋三的日本官邸。我这封信，后来被香港媒体称为“死亡恐吓信”，除了因为信上写着“不许参拜靖国神社！若参拜会遭天谴！天诛！”之外，我还在信内放入了刀片。

事后有很多人问我这样做的动机是什么？他们无法理解竟然会有香港人做出这样激进的反日举动。其实我也不理解自己为什么会那么冲动。我不想把自己这种犯法的行为说成是爱国，爱国应该是很高尚的情操。我觉得自己就是热血过头，头脑一热就去做了。

不过，在决定寄出这封恐吓信之前，我也有过一番思想挣扎，一方面我觉得必须得进行激烈表达，另一方面我的法律意识又不停地告诫我犯法是不对的，最后我总算想出了一个折中的方法，一个特别能体现愤青思维逻辑的方法：
