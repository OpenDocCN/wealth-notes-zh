
# 牛市与熊市形成的原因

很多时候，牛市是由政治力量挑起的，同样，熊市也是由政治力量操控的。不同千2007-2008的股灾，是由美国传到过来的。2015年股灾的背后，是双创进行了大晕的融资。2021年股灾的背后，是科创进行了大量的融资。经济基础反作用千了上层建筑，产生了引发系统性风险的安全问题。原秩序不允许汉东有这么牛逼的存在，所以前有贾跃亭，后有马云，再加上个房地产的许家印。随着政策的调整，几位牛市先锋便纷纷折戟。从实操层面，反而非常的纯粹与简单，牛市是中央为新经济提供了—种新的融资工具，熊市是中央把这种带来牛市的融资工具紧急喊停。停掉双创牛市，只要喊停P2P和场外配资。停掉房地产牛市，只需要三道红线。停掉科创牛市，只要喊停互联网配资和严查大基金。可以说完全都是神仙斗法的最终结果。
