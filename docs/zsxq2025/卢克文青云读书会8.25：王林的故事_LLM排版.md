
# 卢克文青云读书会8.25：王林的故事

## 一八十年代“气功热”的由来

气功源自于中国。最早的古典气功的理论，就建立在中医的养身理论之中。最早可以追溯到东晋许逊撰的《灵剑子》，在之后的各个朝代也都有所记载。

新中国成立以后，气功一词逐渐广泛且频繁地出现于汉语当中，甚至逐渐成为一个全球性的概念。而气功的崛起与二十世纪下半叶中国社会思潮的变迁密不可分。

八十年代以来，世界变化之快，几乎三年就是一个时代，人们在社会剧烈的动荡中，总要找到一点精神依托。这种精神寄托在西方体现为各种宗教思想，但西方的宗教思想当时在中国不成气候，中国的很多传统文化也在文革中慢慢变得不再被人们接受。

在这样一个人们缺少精神支柱的八十年代，可以修身养性和充满玄学的气功，渐渐成为了人们新的精神支柱。它不是迷信，也不是新的宗教，它是人们获取精神需求的一种方式。

在这个过程中，气功以及气功大师也受到了官方和科学界的支持和重视。官方主要是站在探索科学的角度上，比如能让血肉之躯和钢铁做对抗的硬气功，如果能搞清楚原理，也许就能够应用到更多的地

方。气功这个词开始频繁出现在各大媒体的报道当中。甚至在1987年还成立了中国人体科学研究会，由张震寰将军担任理事长，钱学森教授担任名誉理事长。在钱老的观念里，气功和特异功能都可以统称为人体科学。

就这样，气功在人们的精神需求和官方牵头的双重推动之下，在社会上流行起来。

然而，有人的地方，就会有利益，就会有人不择手段地去谋取利益。那个时候谁要是会一点和气功沾边的东西，就马上会成为受人景仰的大师。

于是就有人把对气功的痴迷，当成了一门生意，谎称自己会气功，以此来博取名利。所以在八九十年代出现了很多会特异功能的人，比如开头所说，用气功灭火的严新，还有拥有透视眼的张宝胜，能用意念杀死动物的张宏堡。总之，一些特异功能大师全部都冒了出来。

但随着改革开放的推进，全社会遍地是机会，人们开始有了新的精神追求，一心一意图发展。

但是，仅是发展两个字，是无法支撑起一整个意识形态体系的，这也给西方文化的入侵提供了机会，当然这都是后话了。意识形态的真空，如果我们不占据，就会有别人占据。

这些所谓的气功大师们如果真的能站在传统文化的基

础上，紧跟时代发展需求，建构出属于我们自己的意识形态叙事体系，也许还能在一定程度上与西方的文化分庭抗争，也能继续延续“气功热”。

但是这些假大师们只是在忙着圈钱赚名声，甚至是阻挡了社会的发展，气功的存在，对我们来说已经是成本大于收益了。

1994年12月5日，中共中央、国务院颁布了“关于加强科学技术普及工作的若干意见”，其中第12条明确“要充分认识破除反科学、伪科学的重要性”，才将气功彻底从社会上剔除。

而那些借着气功圈钱赚名声的假大师们，也一个个在质疑和考验中，逐渐被时代所抛弃，落了个身败名裂。

而“大师”王林，却是一个例外。

## 二中国最后一个“大师”

在2000年以后的新时代里，其他大师都销声匿迹，但王林却不一样，曾经他和其他的假大师们一样名声很大，也和其他大师一样在中央颁布反对伪科学的文件后遭到质疑，在表演气功时被人戳穿过。可即使这样，还是有很多好多人前赴后继想要结识他，成为他的信徒。

他不光名声响亮，和之前的很多大师不同的是，那些人大多骗取的都是普通百姓的钱财，而王林，崇拜他的人，都是社会中的精英，从各大明星到企业家，甚至有些地方的

一把手，党政干部，都对他唯命是从。这些社会中的中流砥柱们争相拜访他的家，只为能和这位大师搭上一点关系。

他就是中国最后一个“大师”，王林。

## 三大师王林的前半生

根据王林自己说的版本，1952年，他生于江西省萍乡市芦溪县。

被亲生父母抛弃的他，被一对穷苦夫妇收养，父亲是银匠，母亲是普通农村妇女。

作为大师，王林的同年就经历了很多和别人不一样的事情。

在他小时候得过一场病，十三天不吃不喝，家人无奈把他放进棺材，准备葬礼。

就在丧葬举行时，王林突然又活了过来，令当时在场的所有人震惊。

七岁时，王林被一个老道士看中，成为了这名道士的徒弟，离家跑到峨眉山学习法术，“太极玄功武术”，并跟着师傅四处游历学功夫，于是练就了一身“茅山术”。

后来在下乡劳动的时候，因为破坏农业学大寨运动，王林被关进了监狱。

作为大师，王林似乎特别乐于助人，在监狱里他不忍心看着狱友们吃苦，于是他经常用法术变出大鱼大肉，还有美酒，分给狱友享用，这让他在狱友心中地位很高。

后来王林认识了两个被冤枉判死刑的人，王林替他们感到不公，于是便用法术帮他们越狱，自己也一起跑了出去，但不久后便被抓回。

以上是王林对自己前半生的自述，就像是人们印象中的大人物那样，从小便充满了传奇的经历，然而真实情况和王林所说差别很大。

王林应该生于1947年或1948年，王林之所以说自己生于1952年，是因为根据当地说法，1952年是龙年，更符合他大师的身份，而且王林在1965年时作为知青下乡，按照王林自己的说法，1952年出生的他在1965年时只有13岁，这显然不可能。

王林自小都没有离开过芦溪县，所以也没去过峨眉山。他自小就不是个老实孩子，小学都没读完就整天在大街上晃悠，王林的妈妈，会一些简单的戏法，王林便学了一些，后来还遇到一个杂耍老人，他对此非常感兴趣，便学了一些。

再后来，王林赶上了上山下乡，在江西省宜丰县务农，萍乡的杂技团去他劳动的地方表演，看他有一些杂耍基础，便又教了他一些。

王林就是在这个时候，学到了那些日后让他叱诧风云的“法术”，并且在他下乡劳动的时候，也已经初见端倪。

在务农的休息期间，王林经常和其他人显摆他的魔术，他经常是拿草帽一盖，便变出香烟。吃饭的时候，拿出一个杯，就变出白酒。但他从不告诉别人是魔术，而吹嘘自己这是法术。

谎言和吹嘘，贯穿了王林的一生。

在1979年劳动的时候，王林被关进了监狱，真实罪名是猥亵妇女罪。这一点王林之后倒是直接承认了，说自己确实扒过女青年的裤子。

但进入监狱这件事，对于普通人来说，一定

是人生的重大挫折，但对于王林而言，当他进入到监狱的那一刻，他人生的轨道发生了变化，监狱并没有成为囚禁他的牢笼，反而成了他大展拳脚的舞台。

一个因猥亵妇女而进入监狱的人，在正大光明地从监狱出来后，犹如猛龙过江一般，成为了受人敬仰的大师。

现在我们打开八十年代那座监狱的大门，一起来看看，“大师是如何练成的”。

## 四大师的崛起

监狱里的生活是非常枯燥无聊的，有一天，王林突然对狱友说起，自己曾经和道士学过法术这件事情，狱友听了一脸蔑视。

据日后看管过王林的狱警说，王林早期在监狱里并不像他自己所说的那样，受人尊敬，反而经常因为吹牛而被人欺负。

有一次，狱友决定要彻底治一治这个天天吹牛的家伙，于是就围在王林身边叫嚣着，“来，你变，我们看看你能变出什么来。”

他们没想到的是，这一次王林准备充分。

王林先拿出一个空杯子，倒扣放在桌子上，然后王林围着这个杯子手舞足蹈，嘴里不断念叨。

接着突然大喝一声，一掌拍在杯底上，然后拿起杯子，反转过来，拿掉盖在上面的纸，再把杯口冲向众人。

在场所有人都被惊得说不出话，离杯子最近

的人凑上前去闻了闻，扭头对其他人说，“白酒味”。

场面顿时像炸了锅一样，众犯人一拥而上，高喊着“给我尝一口!给我尝一口!”王林笑着大声说道，“都别抢，都别抢，人人都有份!”

接着王林拿着酒杯，给在场的狱友们一人喂

一口，这时王林仿佛是一个神父，为下面每一位虔诚的信徒泼洒着圣水。

之后王林就开始频繁地在监狱里表演他的

“法术”，监狱里出了一位奇人，这件事算是传开了。

王林的招牌法术，“空杯变酒”其实就是一种

特别简单的魔术。

原理就是在杯里提前放好酒，用类似塑料薄

膜的材料将杯口封住，然后将报纸或者布料

盖在上面，在拿走报纸的时候，同时把塑料

薄膜拿走，这样，酒就可以倒出来了。

现在大家觉得很简单，但在电视还没普及、

社会上到处都是有特异功能的气功大师的八

十年代，足以给人不小的震撼。

但按照正常人的想法，能变出酒，也应该能

变出别的东西。于是很多人起哄，让王林变

一些红酒出来。

王林却对大家说，自己学的“茅山术”,是古

代的一种法术，古代只有白酒，没有红酒。

这样的说法，有理有据，让人们信以为真。

然而真实原因是，监狱里弄点白酒可以，但

是红酒，是肯定弄不到的。也正是因为这个

原因，王林在监狱里只表演过空杯变酒，他

日后的另一个招牌表演，隔空取蛇，就没

有在监狱里表演过。

也正是在监狱的表演，让王林尝到了吹牛的

甜头，并成为了监狱里的特殊人物。

也不是所有人都相信王林本人的说辞，很多

人觉得王林这都是一些小把戏，但有一些

人，确实被这些戏法唬到了。

人一旦开始无脑地相信一些玄学，那么做出

什么离谱的事情都有可能。陈远东是监狱的

一名狱警，因为他对王林的迷信，把自己推

上了一条不归之路。

陈远东一直希望能从王林这儿学个一招半式

的，有一次，王林以教陈远东法术为由，让

他在监狱放电影的时候，把电闸拉了，并走

到一个地方，烧掉一张纸。

陈远东照做了，可等了一会，什么都没有发

生，有点耐不住性子的他一回头，只见自

己的身后一个人都没有了，王林已经趁着他玩

忽职守的时候，弄断了窗户上的两根铁栅

栏，带着其他两个人跑了。

因为这个事，陈远东从看管犯人的狱警， 变

成了阶下囚，被判了六年。

王林这次被抓回来后，从南昌北郊的江西

省第一监狱，换到了当时的江西省第四监狱，

还加了刑。

王林来到新的监狱， 一开始都是干最脏最累

的活，但王林依旧孜孜不倦地在监狱中展示

他的“法术”。因为此时的王林已经发现所有

人都对有特异功能的人有一层特殊的滤镜，

不论对方是何身份。而且因为他之前越过

狱，这让他会法术这个事情传得神乎其神。

慢慢地，王林会法术这件事情再一次在监狱

里传开了。

前文说过，在八十年代，不光是民间痴迷于

气功和特异功能，当时很多官员也对此颇感

兴趣。而王林的本事，也渐渐传到了一些当

地领导的耳朵里。

有一天，正在劳动的王林，被一个狱警叫

住。

“王林，别干了，和我走一趟。”

王林跟着狱警走到一间办公室，进门后他看

到监狱很多领导都在，还有一些他不认识的

面孔。

但从监狱领导和他们说话的态度上看，这些

新面孔绝不简单。其中一个领导对王林说，

“听说你以前去过峨眉山学过法术，今天给

我们演一个看看。”

敏

锐的王林立刻意识到，这是他人生中可遇

不可求的一次机会。

王林顿时认真起来，小心翼翼并一脸笑脸地

和领导说， “当然没问题，但您看我回去稍微

准备一下，主要是想休息一下，一小会就

好，刚刚还在劳动，比较累，怕出什么岔

子，您看……”

监狱领导回复道，“可以，你去吧，休息好

了，一会再过来。”

之后王林赶快回到自己的房间，把上次表演

剩的一点白酒和经常用的杯子拿出来，为他

接下来的表演做准备。

王林拿着自己的道具，回到领导的办公室，

直接把杯子扣在桌子上，然后像往常一样手

舞足蹈，再把杯子反过来，扣在上面的报纸

一拿掉，一杯白酒就在里面了。

至此之后，王林在监狱里的日子越过越舒

服，很多重活累活也不用他干了。偶尔还给

监狱的领导们表演一下他的法术。

领导对他的青睐，让王林有了一些意外的发

现，这也是王林和其他大师的不同之处，王

林开始执着于结交级别高的大人物，在他看

来一个崇拜他的大人物，远胜过1000个崇拜

他的小人物。

他开始向往权力的力量。

为了能够和更多的领导拉近关系，王林开始

在监狱里声称，自己不光会法术，还会用气

功给人治病。

气功治病这件事在当时人们的认知里是确实

存在

的，并且能做到者少之又少，于是监狱里有一些小领导，尝试让王林给自己的家人看病。结果，王林还真把病给人家看好了。

这里要解释一下王林用所谓的“气功”能给

一些人看好病的原理，这个是有科学依据

的，其实就是一种安慰剂效应。

只要让病患自认为受到了治疗，随便吃一点

什么,然后告诉病人这是效果很好的药，病

人就会感觉病痛消失了，或者缓解了好多。

比如抑郁症、偏头痛、十二指肠溃烂，安慰

剂效应都能达到将近三分之一的治愈率。还

有很多病本来依靠人体就能自愈。

安慰剂效益还有一个特点，那就是价格越贵

越权威的安慰剂，效果就越好。比如以前很

多人相信喝童子尿就能治病，其实就是一种

安慰剂效应。

而王林作为一个会法术的奇人，监狱里又有

那么多人相信且见过他的法术，再加上那个

时代人们对气功和特异功能的肯定，王林的

权威性简直爆炸。很多人在去之前就认定，

他一定能治好自己的病，安慰剂效应直接拉

满。

其实到这里大家已经明白，王林所谓的特异

功能、气功，都是假的。他能变出什么来，

完全受制于他所处的环境能弄来什么东西。

会用气功看病也是假的，他和那个年代很多

所谓的特异功能大师一样，都是依靠欺骗生

存。但王林和其他大师最大的不同是，王林

走的是“信徒高端化”的特殊路线，这也是

很多假大师没过多久就塌房了，但王林在20

00年后依旧活跃的根本原因。

在监狱里已经名声大噪的王林，通过监狱的

口口相传，他的本事已经越过了铁丝网，传

到了监狱的外面。每个人的人生转折点，都

是经过了无数次的铺垫后，再迎来一个较大

的拐点，才能彻底让自己命运的车轮彻底改

变方向。而王林的拐点，就在1987年。

在一个雾蒙蒙的晚上， 一列政府的车队驶入

了江西省第四监狱，这列车队里的人来自三

个单位，江西省司法厅、公安厅，还有南昌

市的气功协会，听说监狱里有一位会特异功

能的奇人，于是他们代表着各自的单位，来

对王林进行测试。

如果这个王林货真价实，那么这绝对是给江

西省长脸的一件事。所以监狱方面非常重视

此次测试。

监狱的大门一打开，监狱的领导们齐刷刷地

站在里面准备迎接。而王林，也在监狱里等

着迎接这场改变他人生的表演。

当时有一篇报道，是这样描写王林给领导们

的表演内容的。江西省公安厅的同志把一个

保险箱放在一个房间里，并有人在里面看

守，保险箱里放了一叠现金，然后把保险柜

锁住，保险柜的钥匙由一个民警带出监狱。

然后公安厅的同志对王林说，“你要把另一个

房间保险箱里的东西变到这里来，能做到

吗。”
