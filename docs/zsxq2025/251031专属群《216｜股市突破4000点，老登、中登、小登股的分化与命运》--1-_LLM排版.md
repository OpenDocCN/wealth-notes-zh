
# 股市突破4000点，老登、中登、小登股的分化与命运

10月28日，上证指数突破了4000点，这是近十年来首次突破这一关口。船舶、铁路、光电、航天、电池等板块涨幅明显。同时，中美两国领导人最终确定在韩国会晤，新一轮关税战缓和，这对股市也是利好。

今天我们就借着4000点这个背景，来谈一个与股市相关的话题——老登股、中登股、小登股的分化与命运。

## 事件背景

这场争论的导火索是一位券商分析师给某高科技公司做全年净利润预测，预测数字很高，引发了一位投资领域资深人士的质疑。这位分析师随后将质疑文章转发到朋友圈并配文说：“买你的白酒去吧，老登”。事件的起因大致就是这样。

随后，投资了“老登股”的投资者和投资了“小登股”的投资者，在各个投资社群、雪球、微博等平台进行了大量的争论和交锋。

## 老登股、中登股、小登股

老登股的核心板块是白酒、地产、煤炭、电力、银行、保险、传统中字头央国企、石油化工等。这些产业处于成熟期，甚至有的已经到了衰退期。它们的增长逻辑从“成长驱动”转向“价值防御”，核心看点是高股息、低估值，以及行业出清后的龙头优势。

中登股则是医药、军工、新能源、光伏、锂电、电商、平台经济、有色金属等行业的股票。这些产业大多处于从成长期向成熟期过渡的关键节点，部分领域已经出现“内卷”。

小登股的核心板块则是AI、算力、半导体、机器人、量子计算等新兴产业。

## 背后的大分化

这场争论背后反映了中国经济转型期的深层分化，至少包含三层分化：

首先，是不同代际群体在当前环境下的认知分化。“老登股”的投资者们，一般是45岁以上，信奉格雷厄姆、巴菲特的“老钱”，投资偏好是追求确定性、现金流和低风险。“中登股”的投资者们，年龄一般在35-45岁之间，经历过移动互联网和新能源的浪潮，信奉“赛道为王”。“小登股”的投资者们，年龄往往在35岁以下，不拘泥于传统估值模型，信奉“想象力比确定性重要”。

其次，不同群体认知分化的背后，其实是不同经济叙事的分化。“老登”叙事是“稳健”的叙事，根植于过去四十年“水涨船高”的时代。“中登”叙事是“突围”的叙事，它们曾经是新动能的代表，但技术扩散和资本涌入，迅速将蓝海变为红海。“小登”叙事则是“颠覆”的叙事，讲述的是未来的故事。

最后，这种分化也反映出中国经济的两种发展模式。“老登股”所代表的产业，代表着一种自上而下的，甚至行政性的结构。“小登股”所代表的产业，高度集中在中国科技创新走廊，是一个群雄逐鹿的结构。

## 未来判断

总体判断，分化将是常态。“小登股”是未来的大趋势，但也将面临残酷的“去伪存真”。中登股的机会在于“二次成长”，特别是新能源、创新药等。老登股的价值在于“稳定器”和“分红源”，但指望它们回归昔日的荣光，也不切实际。

未来国家意志将强力引导资本，持续流向“小登股”所代表的“新质生产力”领域。这意味着“小登股”的波动虽大，但它们的长期趋势与国家战略高度绑定，可能将获得持续的制度性红利。

对于个人投资者来说，交易既要仰望星空，更要敬畏市场，核心是管理好自己的仓位和心态。
