
# 要做的事太多了，没时间配合表演

特朗普的动作已经比较清楚了，他可能就没想真正打关税战、贸易战，因为他就没考虑过如何执行，相比关税战、贸易战，他对炒股以及借此巩固小圈子的凝聚力更感兴趣。

看似对华加征关税从34%、84%再到125%、145%一直在加码，但这就是朝令夕改，因为不同程度的加征关税，对应的工作是截然不同的。关税的剧烈变动，会深刻影响到国内、国际，必须对其影响进行周密的测算，针对性地开展工作，否则必然搬起石头砸了自己的脚。

对特朗普而言，无论有再多的理由，也是在毫无准备的情况下主动发起了一场必败的挑衅。一个基本常识是，在贸易战领域，掌握更多顺差的国家优势更大，尤其当这个国家的顺差来自于制造能力而非自然资源的时候，自人类进入工业时代以来，这一点已经被反复验证。

当贸易战打响的时候，对通过制造业获得大量顺差的国家而言，损失的是财富标志的货币，可能因此导致收入减少、储蓄下降、投资萎缩、产能进一步过剩、局部失业等等问题，但只要做好政府调控，终究是能扛住的。

但对以消费为主的逆差国，除了留下更多不能吃不穿的货币，损失的是维持社会运行和生活必须的制成品，尤其当这个国家还需要维持一个繁荣的资本市场以回购本国货币时，发动贸易战带来的不确定性，会造成资本市场的动荡，引发全球投资者的不安。正如最近一周多，美元指数已经跌破100，要知道年初还高达110。

回顾特朗普过去的10天里，已经成了社交媒体治国，随心所欲的发一段话甚至一句话，就搅动全球资本市场。很显然，特朗普就是在以美国总统的身份炒股，吃相极为难看。

这也就是小镇在《特朗普到底想干什么?"对等关税"背后的深层逻辑》一文分析的，特朗普已经成为美国制度的不仅体现在政治领域也体现在资本领域，当法定权力堪比国王又掌握全球最不受限制特赦权的美国总统，还能够将声音直达每一个人直接影响选票，一旦选择任意妄为，又该如何阻止呢?显然美国流传两百多年的祖宗之法，没有解。

马斯克曾经感慨地说，美国议员年薪只有20万美元，为什么资产有2000万美元，钱从哪里来的?其实很简单，利用内幕消息炒股早就是美国议员等上层官员的标配，更不要说更复杂些的旋转门和利益输送。

美国"Unusual Whales"是一家专门聚焦分析国会山投资活动的金融分析机构，今年公布的一月份报告显示，2024年美国民主党议员持有的股票增值31%，共和党议员增值26%，反观巴菲特管理的股票增值不过27%。

需要注意以上仅仅是平均值，有"国会山股神"之称的佩洛西，她丈夫管理的股票增值71%，但在国会山众"股神"中仅位列第十，排名第一的共和党众议员戴维·劳泽持有股票增值149%，第二的民主党众议员黛比·沃瑟曼·舒尔茨增值142.3%。

但这些"股神"们，也不过是利用内幕消息进行操作。比如佩洛西在美国政府宣布支持新能源补贴政策前，安排丈夫购买百万美元的特斯拉股票，还在微软获得美国国防部220亿美元订单前，同样由丈夫提前在低点买入微软股票。于是2021年全年投资回报率56%，2024年提高到70%。

但上述"国会山股神"相比美国历史上最强的"白宫股神",甚至可以说赚的是"辛苦钱"。

在特朗普最近"Unusual Whales"发布关税相关消息前，都有一股神秘力量在股市提前精准布局。最典型的就是美国时间4月9日晚9时37分，特朗普在自己的社交平台公开发文称"现在是买入的绝佳时机"，还附带了自家上市公司"特朗普媒体集团"的股票代码"DJT"。

在特朗普发布这条买入建议之前，已经有交易者大量开设看涨期权，甚至有人开设了当日到期的看涨期权。不到4个小时，美国时间4月10日凌晨1时18分，特朗普还是在社交平台上宣布对中国加征125%的关税，而对其他没有提出反制的国家授权暂停90天，并将对等关税大幅降低至10%。

随即，美股三大指数大逆转，全线暴涨，这些提前布局的看涨期权，在一个小时内暴涨超过20倍。

特朗普以总统身份操纵股市，很可能已经是老手了。第一任期的2019年6月28日，就有人提前投入大量资金做多美股，第二天，特朗普在日本宣布中美恢复贸易谈判，提前布局的这个账户获利近20亿美元。

现在的特朗普更加露骨，直接通过社交媒体频繁发文控制股市，操作越来越急切，仅对部分国家暂停90天加征关税这一件事，特朗普竟然可以吃三次。

先是4月7日白宫国家经济委员会主任凯文·哈塞特在接受采访时，释放这一消息，引起股市暴涨，提前布局吃一波上涨；随即，白宫又发布消息辟谣，引发8日股市暴跌，这又是做空和低位吸筹的机会；到了4月9日公然号召买入自家公司股票，10日凌晨发布这个刚刚被辟谣的消息，引发一轮暴涨。

于是这几天的美国股指，坐上了过山车。

也就只有手握完全不受限特赦权的美国总统能这么干，不由得想起拜登卸任前告别演讲所说的"我想警告这个国家，一些让我非常担忧的事情。这是一个危险的国家，权力掌握在极少数富豪手中，这是一种危险的权力集中，如果他们滥用权力不受控制会带来危险的后果。今天，一个拥有极度财富、权力和影响力的寡头正在美国形成，实际上威胁到了整个民主以及基本权利和自由"。

特朗普根本不遮掩这些操作，他甚至在自己的社交平台上放出来一段视频，视频内容是特朗普在白宫椭圆形办公室炫耀，指着自己小圈子的朋友，说这个人今天在股市赚了25亿美元，还说这个朋友的投资公司实际就他一个人。

目前美国网络上估计，仅这几天，特朗普的小圈子就大赚1600亿美元，而特朗普家族获利高达几百亿美元。

不过在支持特朗普的看来，这或许是一种MAGA成功：特朗普以自己的方式成功暴打了华尔街的资本巨头们，还要庆祝特朗普成为美国历史上干掉亿万富翁最多的人。

讽刺的是，正因为带着小圈子一起赚大钱、赚快钱，特朗普的团队反而更加稳固了，毕竟哪怕当几年高官，哪有几亿美元早早落袋为安来的香甜。特朗普还提前对MAGA们发出买入号召，这甚至可以解读为，特朗普对MAGA成员的洗脑，只要相信特朗普，就有钱赚。

但是，这仍然是党争的政治能力，对美国整体、未来和治理有害无益。

加征关税并非决策之后就自动落实，要把决策变为现实，需要开展大量扎实的工作。比如海关方面就需要很长时间的筹备，目前美国海关根本不具备执行能力，小镇在4月10日《在实干者眼中，特朗普的关税战不过是纸上谈兵》已经展开谈了，这篇文章小镇是4月10日写的，就在美国时间4月11日，美国海关报告关税系统出现故障，无法征收关税。

无论是真是假都不稀奇。因为美国海关系统很落后，过时的语言架构导致系统效率很低，庞大的数据又导致更新困难，美国、日本很多年代久远的系统都面临类似问题，所以现在出现故障很正常，或许按照特朗普新政令调整系统，结果系统崩了；也可能是体面的谎言，毕竟都清楚特朗普没有认真准备打关税战，如果海关冲的太往前，很容易两头堵，反不如等一等、看一看。

需要为加征关税做准备的不仅仅是海关，海关只是一环，甚至说的直接点，就算海关仅仅只是做一些威慑性动作也可以。因为在跨境贸易方面，守法仍然多于违法，只需要做出威慑动作、抓几个典型，就已经可以明显提高征收的关税。

而且，特朗普加征关税，目的是为了筹钱，以此证明可以获得支持减税法案延期的资金支持。这钱可以通过海关获得，也可以直接对美国消费者征收，美国一些消费账单上已经出现了"关税附加费"，也就是费率不一，目前还不清楚"Tariff Surcharge"，这是美国统一行为，还是少数美国商家的决定，小镇更倾向于认为是某些商家以此作为涨价的借口。

关税的增加，最终一定会由美国消费者承担，这是很基本的常识判断。只不过这一切来的太快，毕竟现在还没有正式加征。

特朗普喊关税，目的不是关税，更没有想着真正执行，所以可以毫无顾忌的不断加码，甚至如果中国继续跟进，特朗普喊出200%都很有可能；但当加征关税突破70%，再增加已经没有多少实际意义了，如果中国继续跟进特朗普，真要加到200%，那就太掉价了。

更重要的是，中国从不拿政策开玩笑，是真的在认真准备执行加征关税的决策，这要进行极为复杂的筹备，涉及国内外方方面面。所以中国决定以后特朗普再升高关税税率，我们也不跟了。

以下简单列举需要做的工作，实际工作复杂度远高于此。
海关：任务最重、最急、最繁琐，需要调整系统、规则，确保关税数据库与最新政策同步，还要进行人员培训、增加检测设备以应对稽查工作量激增，协调全国范围的海关稽查，并与地方、相关部门协同，打击逃税、走私等违法犯罪，还要准备更多的仓库，用于存放积压的商品等等，事无巨细。

举一个特别细节但很重要的工作，那就是各类产品的"原产地"认证规则。我们要坚决打下去，但要尽可能伤敌最大、自损最小。那么比如中国需要的半导体产品，就需要进行优化。就在4月11日，中国半导体行业协会发布关于半导体产品"原产地"认定规则的紧急通知，确定"流片地认定为原产地"，这非常重要，这就体现了我们在认真准备。

展开谈下，芯片生产大致分设计、晶圆制造也就是流片、封装三个环节，由于半导体产业全球分工协作，所以2018年后关于原产地的认证非常复杂，需要考虑产值占比等等维度。但这是常规状态，现在猛然加速，有了更大的战略目标，因此原产地的定义一下子变得简单了，只看在哪里流片，如美国英伟达、高通、苹果等等都只是设计并不生产，流片基本在台积电，这样一来，在中国的定义中，英伟达、高通等美国芯片只要在美国以外流片，那就不属于美国货，不必被征收高额关税，但美国那边不这么看。进群加微信

类似关键细节的处理还有很多，每一项调整都需要进行认真细致的评估，绝不能拍脑袋，毕竟朝令夕改会造成混乱。

商务：进行对外交涉，筹备可能出现的美国在等机构的诉讼，还要研究细化下一步的反制WTO措施，关注中国企业的诉求，组织召开行业座谈和调研，尽可能降低关税战对中国企业带来的冲击；不仅如此，还要关注中美关税战对其他国家带来的冲击，关注潜在的国际合作，判断中国与其他国家的各项协议是否需要调整，最直接的，欧盟对中国电动车的限制会不会松动?中国企业在欧洲的布局是否可以加速?

举一个具体的例子。

为了应对外贸萎缩，需要协调内外贸一体化，促进外贸转内销，但难题可太多了。最直接的就是商品标准问题，同一种商品，不同国家的标准不同，数以万种工作量极大；就算标准问题解决了或者临时开个口子，但之所以很多企业更愿意做外贸，恰恰是因为内贸太卷了，尤其拖欠账款问题，所以有些商品，实际国内售价、周转率都更高，但仅账期这一项，就不如做外贸省心。

目前商务已组织座谈会研究帮助外贸企业扩大内销渠道，如京东也宣布设立2000亿元专项采购资金，帮助外贸企业转内销。但小镇对这件事的看法并不乐观，虽然现实压力很大，但一项工作提出几年进展缓慢，已经说明难度，只能希望能够借外部压力，各方达成更大限度的共识，把内外贸一体化向前推进一步。

除了以上工作，还要抽空编写一份2.8万字的《关于中美经贸关系若干问题的中方立场》的白皮书，要占据国际道义制高点。

外交：非常忙，要跟各国沟通，阐述中国的态度，避免各国误判和连带制裁，还要争取国际舆论支持，筹划高层互访，抓住机遇推动共建开放的区域主义，以此不断扩大中国的生态圈，建立更加平等互惠的新全球化。

近期尤其忙，大家从新闻中应该能看到，毕竟这周在这个领域召开了首个最高层级的工作会议，上次类似会议还是2013年，但当时仅仅是一个座谈会，而且也从当年关注经贸合作，首次拓展到安全维度，是一次全面升级。这一块小镇判断应该是不适合目前谈的，所以只提一下，具体的内容就删掉了，等下周完事再谈。

以上是直接的对外部门，对内部门也非常忙。

美国的输华商品，很重要的一部分是农产品和能源产品，涉及到中国粮食安全和能源安全，中国需要考虑如何完成替代，还要评估加征关税对中国两大安全的影响，判断由此对全球大宗商品价格带来的影响，平衡国内生产和国际进口。

虽然我们之前就已经准备，2024年已经开始减少从美国的进口，更多从拉美进口，2025年本要降低国际购买、更多支持本国生产，比如在牛肉领域，刚刚对进口对国内牛肉价格影响进行了评估，要更侧重保障本国肉牛养殖户的利益。

但变化来的很快，都需要尽快推动，以维持粮食和能源安全。

农口：任务很重，需要在新情况下，统筹农业扶持政策，包括财政补贴、农产品收储等等，还要配合开展海关稽查，关注价格调控，还要与其他部门、地方沟通，要注意今年已经明确要落实粮食产销区省省际横向利益补偿，那么国内国际需要把握两个平衡。

这么多不同种类的农产品大规模采购，都需要进行评估、调整，工作量很大。

类似的还有能源部门以及遍布海外的农业、能源企业。

财政：要测算关税战带来的收入影响，调整财政预算，还要筹划新一轮刺激政策和税收减免政策，对冲受影响企业的成本压力。

央行：需要关注人民币汇率稳定，警惕资本异常流动，还要为受影响的外贸企业提供紧急避险工具支持。

发改工信：要评估对宏观经济和相关行业的影响，还要对相关产业产能继续拧调整，两部门协同确定分行业、分地区的扶持政策，推动进口替代和产业链国产化进程，持续降低对美依赖。

国资：要推动央企改革，还要协同证监等金融部门稳定中国股市，等等。

太多太多部门的协同，难以简单概述。比如税务、公安、海警等等部门都要进行调整，这是体系作战。

不仅中央部门，地方也要进行配合；政府以外，行业、协会、企业乃至海外华人组织等等，都需要根据新政策进行准备。

中国要忙的正事太多了。

34%、84%、125%、145%的关税加征，对闹着玩的特朗普来说，仅仅是个数字；但对认真做事的中国来说，这意味着截然不同的应对策略，每一次调增，太多部门都需要重新进行测算和规划，如果继续跟着特朗普追加，那工作真没法干了。

要抓住特朗普谋私利误国运的战略机遇，能推进多快就推进多快。接下来不管特朗普说什么，中国决定了的事，就要一直做下去，哪怕特朗普突然有一天签署行政令，说取消对中国加征的关税，我们要不要停，也取决于我们。
