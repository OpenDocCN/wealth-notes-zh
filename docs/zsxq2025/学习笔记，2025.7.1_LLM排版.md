
# 学习笔记，2025.7.1

1. 在过去，包括农村，盖房子是不需要借钱的，就是老百姓虽然穷，但是不至于负债累累。后来，出现了商品房，房子越来越贵，结果就是全民负债，表面是负债，其实是提前透支了人生，等于把未来的时间借给了今天。这种透支会导致一个人视线越来越狭窄，等于脖子上栓了一根缰绳，最终的结果就是思维的管道化，不敢乱想，不敢试错，生怕还债中断，人生所有的可能性都被扼杀掉了。

2. 找男人就一个标准，有一技之长的，这一技之长必须是能换饭吃的，你说他篮球打的很好，这个不行，生产力才是男人最核心的竞争力。一个男人，再帅，再幽默，但是赚不来钱，得不到社会的认可，最终就是吃吃喝喝，混一辈子，关键是越是这样的男人，恶习越多，因为他的事业没有精进的可能，整个人没有进化的机会，从而恶习会加固，相反，他若是有点事业干着，整个人在不断的进化，人是越来越好的，越来越有大爱。

3. 头婚，其实很少深思熟虑，更多是机缘巧合，或同学，或网友，或朋友介绍，觉得不讨厌，就走到一起了。二婚呢？彼此都有经验了，知道自己想要什么了，理性了，那么可以把这个当选择标准。前段时间有朋友到我这里玩耍，我跟她也是这么讲的，你这个年龄了，找二婚的男人，就一个原则，选择有结果的，若是他岁了，还一事无成，说明他各方面都很普通，包括性格，因为你真是个不错的人，是个对社会有价值的人，肯定从收入上就反馈出来了，若是没有反馈，就说明，市场或社会不认可他，至于帅或暖，那都是假的，不当饭吃。

4. 企业的成败，取决于领头羊是否能带对路，但是从长远来讲，人的思维总存在盲区，不可能次次都判断准确，只要判断错了一次，就输了，就如同我们怎么也没想到阿里巴巴会被拼多多打败。而城市呢？城市是多元的，开放的，相互碰撞的，是一个可以不断自我进化的生态系统。

5. 前段时间有个悲剧，父亲带着12岁儿子骑行，儿子摔车后遭汽车碾压致死，这场悲剧其实就是父亲一手酿造的，骑车不仅仅是一个体能业务，更是一个安全业务，父亲对规则缺少敬畏心。上一次类似的悲剧是八达岭野生动物园，女司机中途下车跟老公换位置，老虎以为是投喂，老虎追咬女司机，女司机的妈妈为了女儿与老虎搏斗，结果就是女司机重伤，其母死亡。都是对规则缺少敬畏心。年前，河南大雪，高速封了，我从郑州坐高铁回来的，一等座二等座都没票，我买了张商务座，商务座其实很吵，因为前面就是驾驶舱，高铁司机全程都在朗读一些指令，而且会反复确认。我推测，飞行员也是如此的繁琐+严谨。为什么？因为，每一个疏忽，都有概率是致命的。这不是一个人的致命，是一车人。为什么我们开车不这么严谨？我们总觉得，小概率离自己很遥远。君子不利于危墙之下。

6. 恒大，若是降价断臂求生，死不了，是各地不允许它降价，县城呢，一定程度上是允许自由定价的，这个时候，谁若是能做到别继续装逼了，不搞什么变相降价，又是送装修又是送车位，什么都别搞，就是直接明面上降，周围六千咱就直接打五千，先甩出去再说。理论上可行，实际上，越小的地方越不能这么做，因为人情社会，你抵出去的房子怎么弄？提前预售的房子怎么弄？前面开盘的一二期怎么弄？你在北京上海这么搞，真可以，纯市场行为，消费者也理性，愿赌服输，在小地方？你敢降价，都去你家门口上吊！所以，只能掩耳盗铃，价格还是那个价格，只是送了装修送了地下室送了车位。有个本地朋友问我，她家房子挂了一年了，最近有人要买，比她的持有成本只高一点点，在犹豫中。我说了，我的原则是不参与到任何人的重要人生决策当中去，就医、婚姻、投资，三大禁区。她的意思是，不会怪我的，但说无妨。我说，你孩子毕业后，你会让他回到县城吗？人口流向代表着地产未来，县城地产最终就是三四千一平。我的房子，多在自用中，要么我住着，要么父母住着，要么书店，要么仓库，我已经不考虑成本问题了，若是属性是投资，我早卖了，五折我也卖。

7. 当下，有没有比较好的理财渠道？存银行就行了。前段时间，我想去青岛买套房子，机缘巧合是我攒了点私房钱，有二选一的开支，要么买辆特斯拉，35万左右，要么就是当首付在青岛买个小房子，偶尔去度假的，特斯拉一直在犹豫，可能是我关注电车比较多的缘故，最近抖音一直给我推送自燃的，几乎每天都有几辆，另外我总担心买了特斯拉以后，也不开，就扔那里，每天贬值，主要是我每天用车的频率太低了，一天不到5公里的驾驶里程，我车又太多，关键是我最近总想起《尘埃落定》里的那段话：这个世界上任何东西都是这样，你不要它，它就好好地在那里，保持着它的完整、它的纯粹，一旦到了手中，你就会发现，自己没有全部得到。

8. 中国消费，就看两个数据，一是房子，二是车子，因为中国人就关心这两个消费，其它消费都要靠后。中国经济晴雨表，一是看股市，二是看球市，中国足球市场最好的时候，也是经济最好的时候，一个标志性大事件，恒大夺取了亚冠，现在中超众多俱乐部，生态正常的都没几家，可能再过两年，连联赛都组织不起来了。足球不仅仅是世界第一运动，在中国也是第一运动，足球俱乐部需要持续投入，需要信念支撑，同时又是省级名片，若是足球生态不行了，说明头部企业普遍不行了，没有闲钱搞情怀了。

9. 真正的顶级享受是什么？事前，有说不完的话。事后，也有说不完的话。进行中，那种灵魂的契合要远高于肉体本身的快感。遇到了一次，你就对这些彻底戒了。是真的没意思。

10. 炒股这件事，就是高智商对低智商的降维猎杀。这就是我反复提到的，万存款以下，不理财。否则？要么，理少了。要么，理没了。万是韭菜与镰刀的心态分界线。你有了，是一个心态。你没有，是另一心态。尤其是年轻人，蒸蒸日上的，更不要炒股，你赚不到钱还好，就怕你赚到了钱，从此就不务正业了，总想钱生钱。不论是出于什么意图，给别人推荐股票这件事，结果大概率都是害人。因为这是在诱使别人去碰一件他不懂的事。本质上就已经错了。就算运气好，短期赚了钱，但这笔认知之外的钱，最后会让他以后犯更大的错。还有一点，一旦阈值被打开了，就再也闭不上了。
