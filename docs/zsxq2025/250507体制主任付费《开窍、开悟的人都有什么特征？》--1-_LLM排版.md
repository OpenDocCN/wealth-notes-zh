
# 开窍、开悟的人都有什么特征？

我们总说开窍开悟，到底什么样的人才算真正开窍开悟了？人需要两次诞生，一次是出生，一次是觉醒。世上最不公平的事，莫过于人与人之间的“觉醒速度”，有人浑浑噩噩几十年，依然在底层打转；而有人一旦开悟，短短一两年就能彻底翻身改命，这不是玄学，而是认知层面的降维打击。普通人唯一的出路就是认知觉醒，越早开窍开悟，人生逆袭的可能性就越大，时间不会等人，生命只有一次。80岁的悔恨，已经没有任何意义。

那么如何让自己开窍开悟？（全文共4000余字，只渡有缘人）

## 一、开悟的人已看透社会的运行规则

大多数人之所以平庸，不是因为能力不足，而是被错误的认知困住了。他们以为努力就有回报，以为善良必得善果，以为遵守规则就能成功，这些观念在现实生活中往往失效。社会的真相是：资源有限，竞争无限，赢家通吃。

开窍的人往往懂规则、识规律、总能快速洞悉事物的本质，且戒掉了情绪。比如说要想混好职场，首先你要充分了解职场，特别是职场的运转规律和法则。这就好比玩一个游戏一样，你只有充分了解了“游戏规则”，才能有机会玩好这个游戏，事实上，不了解“游戏规则”的人，不知道什么时候就“触雷”了，别说提拔进步了。

那么，什么是“游戏规则”呢？我想主要有两个方面：一方面是明面上的制度，黑纸白字写出来的规则；另一方面是潜在的、约定俗成的（“潜规则”），这个需要有人点拨或者开悟。很多人在35岁以前，基本上还是学生思维，只知道埋头苦干，不懂人情世故；总是试图追求公平，而别人暗地里却在送礼，拉关系；总是被动等待领导提拔他，他们不知道好东西是需要自己争取的。

很多人认为只要努力工作、认真负责，就能被领导看到，就能获得领导赏识，提拔晋升也是顺水推舟的事情；开窍以后才明白，你不去抢占领导的注意力，领导就是看不到，你不去争取资源和机会，那你就永远没有机会。

记住，熟悉规则的人，特别是熟悉单位运转“潜规则”的人，才是更容易谋事成事的人。

## 二、开悟的人非常珍惜时间

时间是普通人唯一可利用的资源。我常说时间面前，人人平等，普通人没有显赫的背景、丰富的资源和殷厚的家底，缺乏足够的试错成本。在这种情况下，时间成为了我们唯一可以利用的宝贵资源。每一个人都是平等地拥有时间，钱，可以再生，但是时间不能再生，每个人就那么几十年，过完了就永远不会有了，人的生命只有一次。时间才是一个人最最宝贵的财富，不可再生的财富，因此一个开窍了的人，他一定非常珍惜自己的时间，然而，很多普通人却最不珍惜时间。他们将时间消耗在家长里短、垃圾资讯中，白白浪费了提升自己的机会。

要是有人过来跟你闲聊扯淡，聊各种跟赚钱提升无关的话题，立马拒绝，别人就是把你当成情绪垃圾桶。总而言之，把自己的时间交给不相干的人，交给不能加分的人，都属于浪费。

时间，浪费生命。时间就是你的命。一旦你对此坚信不疑，那么在花时间这件事上，你就会变得非常慎重。

要想更好地升级，你必须成为一名合格的“时间投资人”，你需要认真思考，认真选择，想尽办法将大部分时间投资到高价值的人和事上。如果长期坚持这样的原则，你的时间投资回报会非常丰厚。如果你是一个对时间花费很不敏感、很不明智的人，你会浪费大量的时间，很难做到持续升级。从今天开始，你不仅要理财，更要理时间。认真做好时间投资，而不是随随便便就把它花出去。

最近几年，我一直在给自己的时间定价。从最初的几块钱一小时，到现在的几百、几千元一小时。这样做，一方面，可以让我更加重视自己的时间；另一方面，可以帮我“拦人”，避免一些无效社交和一些乱七八糟的人来消耗我。

所有的精力，所有的注意力都应该只花在自己的身上，花在跟自己相关的事情上面，普通人的最大资产，就是时间，就是每天的24小时，这就是开悟的人第一个特点。

## 三、执行力，是开悟者的终极武器

执行力是所有能力的祖师爷。人与人之间的差距，没那么复杂，说到底最根本的差距就是执行力的差距，执行力是离钱最近的底层能力。很多人经常高谈阔论的谈理想，懂得很多的道理，但是如果你仔细去关注他们的生活，其实仍然是一地鸡毛，当你的想法创意，好点子在你执行力不足的时候都会归零，讲白点，执行力是所有能力的祖师爷，有的人就算没有天赋，执行力拉满依旧可以取得很大的成绩。

许多人总爱把事情复杂化，认为成功需要诸多条件，但世界的底层逻辑其实异常简单：只要你有勇气去行动，并且持续执行，就掌握了成功的核心密码。

看透规则、选对赛道、突破认知，这些还不够。真正能让一个人在一两年内翻身的，是恐怖的执行力。大多数不是没有想法，而是想法太多，行动太少。他们总是在等“完美时机”，等“准备充分”，等“有人带路”——结果等来等去，机会早就被别人抢光了。

开悟的人不一样，他明白自己的时间很宝贵，他们信奉“先干再说”。不会？学！不懂？问！做错了？改！他们不会让情绪干扰行动，不会让拖延消耗机会。这个世界奖励的不是聪明人，而是敢行动的人。你去看那些真正逆袭的人，没有一个是靠空想成功的，全都是在一两年内疯狂试错、快速迭代，最终找到自己的爆发点。因此，那些开窍的人，他不会一直坐在那里空想，一直坐在那里纠结，一直的思维层面权衡利弊，他不会，他一定是在实践中找答案，在行动中找答案。

## 四、凡事向内求

真正开窍的人，凡事都会向内求，不管发生什么事情，他第一时间是从自己的内在找答案，而不是到外在找答案，他不会期待去改变别人，去改变环境。他知道外面没有答案，答案就在自己身上，如果自己接受不了对方是什么样的人，接受不了环境是什么样的环境，那他可以直接选择离开。如果他离开不了，他就会选择接受，他不会去企图改变自己不能改变的。他只专注于改变自己能改变的。

什么是自己能够改变的？那只有自己了。发生任何事情，他第一时间他都问问自己，自己能够做些什么，自己当下能够控制的部分是什么，不能控制的部分又是什么？如果说自己能控制的部分，他肯定尽自己最大的努力去掌控，但是不能控制的部分，他能做到立刻马上把它放下，不去纠结。

举个例子：某个人在单位里不得志，或是晋升无望，或是收入不高，或是其他问题。弱势文化群体面对此种问题或局面时，多半是抱怨。要么抱怨领导不给机会、要么是说晋升的都是领导的人，要么就是埋怨单位的用人制度不合理，（也许这都是客观事实，但这个人的抱怨是无法改变这些事实的，我们周围的环境从来没有因为我们抱怨很多而改变）。

真正的强势文化群体面对同样问题或局面时，不是这样想问题的，他们真的会从自己身上找原因，改变能改变的，接受能接受的，如果单位里的机制还算公平公正，自己愿意继续干下去，他们多半会去努力提升自己，让自己更厉害点，以便获得更多机会，这些人通常都是那种很努力向上的人，一般来说，这种人不会混得很差。

如果单位里的机制极不公平，乌烟瘴气，那这帮人也会沉潜自己，努力提升自己，他们深知这里不属于他，在体制内的话争取考走、参加遴选或者调动，在体制外就会蓄势待发跳槽到更好的单位。

强者从不抱怨，不再抱怨挫折，不再到处诉苦，生活从不缺少苦难，但开窍之人明白，过多的倾诉只会消耗自己的能量，让他人徒增烦恼。

向内求→找自身原因→改变自己→超越自己  向外求→找外界原因→期望外界改变→自己不改变

从不抱怨，只改变自己能改变的部分，当人不再“向外求”，而是“向内求”，这是一个人开悟的标志，也是从“弱势文化”开始向“强势文化”转变的一个重要标志，当一个人的心智逐渐被强势文化的精神内核填满，“等、靠、要”的思想就会消失的无影无踪。

## 五、情绪异常稳定

开窍之后，你会活得非常的通透，身心都很愉悦，恰恰那个时候你的能量层次才是最高的。你试着练习成为一个情感淡人，一切以利益为度量衡，就会发现生活中很多困扰都悉数消失了。

脾气是本能，谁都能一天生个八百回，但控制心性，不被本能操控，保持理性，耐心分析，才算得上是真本事。想成为强者，就必须要锻炼自己的心性。如果总是因为一些小事就惊慌失措、乱发脾气，那你的命运就会永远掌控在别人手里。

大家可以去看看电视剧《天道》，丁元英就是一个情绪异常稳定的人，这是他超脱世俗的一个主要表现。吃面时，餐馆老板说他是好吃懒做的闲杂人等，他没有回击。吃早餐时，摊主误认为他没有结账，让他再付一次钱，他也不恼。在公园的座椅上，他的座位被陌生情侣蛮横的霸占，他默默地起身离开。这是软弱吗？一个随手布局就能搅动市场风云的人怎么能软弱呢？因为他已然超脱情绪之外了。他能清楚看透每个冒犯他的人的出发点，能够理解他们无理行为背后的动机。

他人的行为如同狼吃羊，羊吃草一样是必然的，又有什么好生气的？他人的一切言行不过是自身利益支配下的必然，必然发生的事，有什么值得大动肝火的呢？当你能像丁元英一样，看透每个人言行背后的底层逻辑，你就再也不会被他人撩动你的情绪了。
