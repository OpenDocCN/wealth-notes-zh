
# 哈佛毕业演讲

中国留学生蒋雨融在哈佛大学毕业典礼上的发言，在互联网上引发了热烈的讨论，其中不乏争议和批评。这个事件本身及其激起的网络舆论，折射出了时代的情绪，以及情绪之下的社会弊病。

虽然有不少人试图从蒋雨融的简历和资料中寻找“黑幕”的蛛丝马迹，但是，与前段时间的协和“董小姐”事件不同，蒋雨融的求学经历中应该挖不出什么真正的“黑幕”。她在微博上回应说，“小蒋一步一步走出来的路坦坦荡荡，随便他们去扒”，应该是拥有这份自信。

但是，在她个体经历的“坦坦荡荡”背后，是她拥有的系统性 privilege（特权、优势）。拥有这些 privilege，是她的幸运，不是她的错；但是在公开发言中，避而不谈这些 privilege，一味强调个人奋斗，就会令人观感不适。考虑到阶层固化的社会背景，这种不适就会升级为一种刺痛感。

例如，在演讲后第一次回应网络舆论时，她强调了自己小时候面临的霸凌、转学等遭遇。但在她的叙事中，命运的转折似乎全部依赖于自己努力读书，“读很多很多书，读我能找到的所有书”。换句话说，今天的成绩全都归功于个人的奋斗。

问题是，其他喜欢读书且同等资质的学生，就算再努力，也并没有机会和她一样高中就去英国读书，然后本科去美国留学。自然，随后申请进入哈佛读硕士的几率也要小得多。蒋的家庭背景和她的个人奋斗一样重要。

留学是一门全球化的产业。我一直觉得，发达国家和地区的留学产业显著加深了发展中国家的不平等。在香港教书，我时常会听到同事、同行感叹：内地的学生真有钱，学费就算再涨几万块钱他们也还是会来的。对于这样的说法，我的感受是五味杂陈的：一方面，确实是有一些家境富裕的学生，虽然到香港读书的花费比到欧美要便宜，但显然也不是底层家庭可以负担的；另一方面，内地人均 GDP 只是香港的 1/4 而已，平均来看，香港的学生拥有的教育资源和机会实际上是远远超过内地学生的，“有钱”只是内地人中的极少数。一些到香港读书的内地学生，家境其实只是普通中产，在港生活颇为拮据。

当然，美国人在发展留学产业的时候，不必考虑对中国社会的影响。不平等的加剧是产业发展过程中一种不自觉产生的副产品。

但是，也有个别项目，在设计的时候就是冲着“网罗各国权贵精英”的路线去的，他们实际上很明确地在拿学校的名头去交换各类政商关系。这样做得最大张旗鼓的，就是蒋雨融所在 的哈佛大学肯尼迪政府学院。该学院以培训各国高级公务员（包括中国，它曾被称为“中国高官训练营”）、吸纳各国权贵子弟（包括中国，比如红三代“兔主席”）就读为特色，其显赫的声名与地位也与这种路线有分不开的关系。

再次强调：这些留学都是正当的途径，哈佛肯尼迪的做法也是制度化的设计，很难说是什么“黑幕”。但是，这种设计会让大众产生一种“精英相互勾结”的观感，在实际上也可能真的会带来精英抱团自利的结果。

至于蒋雨融的家庭，应该称不上什么权贵（真正的权贵家庭根本无法在墙内被讨论），虽然多半是属于前 1%。她站上风口浪尖，主要还是要怪自己不自知，似乎也没有读懂舆论背后的情绪，只是反复强调自己的经历经得起放大镜的观察，却并未就自己的 privilege 以谦卑的态度予以承认，并放低姿态，强调自己只是幸运儿。

艰难的论述题，单薄的生命经验接不住

蒋雨融在解释自己为什么要开通社交账号回应争议时说，她不想把网络上的话语空间拱手让人。这个说法很合理。但是，她希望占领话语空间，是为了什么呢？她自己可能会说是为了人类的福祉，但是民众看到的是精致利己。实际上，演讲刚结束不久，她自己就在各个社交媒体平台注册并发布了视频。在视频封面上，她自己写上的几行字是“哈佛校史第一位中国女生毕业代表演讲，全场泪崩”——如果是自己这么写，就颇为奇怪了。我只能理解为：她沉浸在自己的良好感觉里，沉浸在周边人的夸赞里，对可能的问题和舆论的水温浑然不知。

这种浑然不知，也正符合了人们对她以及对哈佛“离地”的印象。

这次离地的演讲，可以说是离地的哈佛校方和离地的蒋雨融共同促成的。先说哈佛校方。公众号“谷雨星球”曾采访9年前第一位登上哈佛毕业典礼讲台的中国学生何江，了解演讲人的选择流程。何江透露：“哈佛的毕业典礼演讲向所有学生开放报名，只要你敢，就有被选中的可能。报名时，需要提交自己的故事和主题大纲，组委会再选出契合当下也符合哈佛价值观的人选，经过重重关卡后，最后选出3名毕业生。一旦入选，就会有专人一起来过演讲稿，找角度，并上专门的演讲课程，训练仪态，整个过程会持续一两个月之久。”

“契合当下也符合哈佛价值观”，这是理解哈佛选择蒋雨融的关键词。川普的第二任期开始以来，哈佛遭受了严厉的打压，但同时也以最硬的腰杆在挺着。显然，一个来自中国的学生重申“世界大同”的方向，会被认为是对川普的最直接回击。

问题是，蒋的演讲徒有价值观，并无实质内容：

与其声称“如果世界上有哪个女人买不起卫生巾，我也会因此变得更贫穷”，不如讲一个具体的买不起卫生巾的女人的事，或是介绍一个解决月经贫困问题的项目；

与其高喊“如果一个小男孩死于一场他没有发起、也从未理解的战争，我的一部分也会随之消亡”，不如分享一个具体的战争劫难故事；

与其表示自己在哈佛“学会了如何面对不适，如何深入倾听，如何在困境中保持温柔”，不如讲一个自己深入倾听和保持温柔的故事。“Show, not tell（展示，而非说教）”本是演讲101基础内容，但蒋的演讲里充斥的是 tell，很少有 show。这样的演讲非旦没有对川普形成打击力，反而坐实了川普对美国精英大学“离地”、“woke”的批评。

这可能也怨不得蒋——她只是真的没有贴地的故事可讲。开头那个洗衣机的故事，也被人指出是在坦桑尼亚的特权阶层才能享有的烦恼。其实，即便是富豪，也会有自己的生命经验，也会有刻骨铭心的艰难时刻，在蒋的身上却一点都看不到这些。她选择了申请上台，却并没有这样的生命经验在支撑，结果就是让人只看得到口号，看不见具体的人。

蒋雨融可能很不解：现场的哈佛师生不是热烈鼓掌吗？是的，但他们的鼓掌多半是由于上述与川普的政治斗争。他们是在为己方立场加油鼓劲，并不一定是在赞赏演讲本身的内容质量。

如果是10年前、15年前，蒋的演讲可能不会引发太多的反感，甚至会受到追捧。但是在全球化退潮、越来越多民众意识到自己在全球化的过程中被精英剥夺的今天，这套“地球村”的话语要再具备说服力和动员力是极为艰难的。蒋的演讲中体现出来的非常单薄的生命经验，是完全不足以接住这个主题的。

为理想主义寻找新的出路

我虽然不喜欢蒋雨融的演讲本身，但同意她演讲中的那些论述的大方向。也即，我认为我们不能放弃理想主义的追求，我们依然应该相信鲁迅所说的，无穷的远方，无数的人们，都和我有关。如果我们因为看到了许多精英的虚伪，就堕入怀疑一切理想的虚无，那迎接我们的将是真正的沉沉暗夜。

问题是，理想主义需要新的出路，一味地唱高调已经不再有用。蒋雨融面对的批评，反映的正是理想主义、进步主义面临的尴尬境地。

新出路的探索一定是极为艰难的。但我想，它离不开这样一些基本的原则——

第一，新的出路一定是真正照顾到所有人尊严的，此前的全球化数十年，创造了一小撮“达沃斯人”，同时积累了大量不满的、愤怒的普通人。已经拥有 privilege 的人，必须尽量多地看到别人的努力，和自己的幸运，在这个基础上再去寻找提升所有人福祉的方法。

第二，新的出路一定是言行合一的。任何表态都需要有实际的行动对应，需要真心去实行自己信奉的理念。反过来，不必假装自己支持一切进步理念，应该如实地表达自己的疑虑和能力限度。

第三，新的出路多半会自下而上、从具体的社群里生长出来。我同意一篇公众号文章的判断：“理想主义的行动必须是一种长时间的、缓慢的、不炫耀目的实践。不是造势，而是布线。不是一次演讲，而是一百次对话。不是赢得舆论，而是赢得信任。在这个时代，说理想，不如做好一件具体的事情：一个社区活动、一家独立书店、一种新的教育尝试、一场耐心的协商、一段能说真话的关系。这些事情没有影响力，没有传播性，没有 KPI，但它们是地基，是种子。”

在坚持学术自由、抵抗政治干预的这场防守战役中，哈佛的努力值得尊敬；但是，对于如何更主动地创造出新的进步主义叙事、提供新的社会革新解法，哈佛在这场进攻战役中可谓交了白卷。

我自己也越来越不相信这种新的出路会在哈佛诞生，或者在任何一所光鲜亮丽的大学诞生，包括我自己供职的这一所。但相信，出路一定会被一些人走出来。他们不会穿着华丽的服装，享受众人的欢呼——恰恰相反，他们可能正穿着破旧的衣服，在人们看不见的角落里默默耕耘。
