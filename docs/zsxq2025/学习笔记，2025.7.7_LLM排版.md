
# 学习笔记，2025.7.7

1，前些年赚钱容易的时候，感觉买个车买个房，就跟买个白菜似的，现在也回不去了，不由的感叹，当初轻易可以拥有的东西，如今再也难以抓住，于是不由自主的给过去的日子镀一层金，称其为黄金时代，其实历史上的黄金时代，都是这么来的。我一直都觉得，不管对国家也好，对个人也罢，当下才是最好的时代，我比过去任何一个年龄点都成熟，都有爱，我不是变老了，而是长大了，国家也是如此。我学历史时感慨万千，这么说吧，我小时候，也就是上世纪年代，其农耕模式跟上下年没什么区别，你想想，这是多恐怖的事？等于一直在不断的转圈，结果，三四十年过去了，家家户户竟然有了小汽车。

2，高盛预测房价要跌到2027年，若是真想买，可以慢慢等待，中国的房子结构决定了，不适合高质量的居住，空气流通、噪音、低素质的邻居。好房子的第一原则是任何一面墙都属于自己，包括上下楼板，另外入户必须隐私。

3，别人看病喜欢找熟人，我喜欢找陌生的，熟人多尴尬。而且，越是熟人越是敷衍。朋友的岳母，肠胃不好，找的主任给看的，老太太怕肠镜，主任的意思是不想做就不做，一年后就确诊了，肠癌。

4，其实县城里，有很多隐形的钢饭碗，例如医院的停车场，永远排队，再贵也有人停，刨除租金、人工成本，不就是钢饭碗吗？只要地球不爆炸，这个饭碗就一直存在。类似的钢饭碗，很多很多。都掌握在少数人手里。就如同我之前写的，一把手掌握加油站的审批大权，这玩意最终就跟发福利一样，你们几个小兄弟这么多年一直跟着我，表现很好，这样吧，一人一个。前些年的加油站，那是金饭碗。再如一家医院，也趴着一群银饭碗，医疗废物谁来回收？氧气谁来供应？这都不是一般人能拿到的业务，需要一方诸侯才可以，小一点的业务，那么就可以角逐了，例如休息大厅的凳子谁来提供，再如医院的电梯谁来维保……再如学校食堂。给咱，咱都不愿意干。结果，你知道有多离谱吗？找关系能找到天上去。上次，我的偶像级朋友，跟我谈到他在山东的业务，其中的一个业务是某学校的食堂，我当时还在想，这也值得拿出来说？后来才明白，是我幼稚了。我怎么明白的？管中窥豹，鼠头鸭脖。但是，我不羡慕，从我的角度来讲，我觉得做生意就做纯市场的，不送礼，不内幕，要足够清爽。我敢说，我赚的每一分钱都是干净的，都是良心的。安全第一，赚钱第二。若是哪天，孩子们也要做生意，我一定也会这么劝他们，一定要阳光的、纯市场的、清爽的生意，做一个能经受起推敲的人。

5，北京二环，看看老北京人的生活，拥挤，狼狈，对了，还有说不出来的自信，皇城根下的。到我们这个年龄，肯定对所有都祛魅了，深刻领悟了那句话，全世界有钱人过着相似的生活，全世界没钱人也过着相似的生活。

6，再过几十年，各大一线城市都会出现贫民窟，为什么过去没有？因为过去可以遣返回老家，但是倘若他本身就是一线城市户口的人，往哪遣返？即便是1%的坠落率，也是个天文数字，这些人慢慢的就会汇集到一起，苟且生存，久而久之，形成了一个片区，这是必然趋势，这是城市发展模型推演出来的，全球特大城市都有的通病，上海弄堂，北京胡同，广州城中村，其实都算得上类似的区域，只是还不够撕裂……

7，咱就是打工的，领导让咱干什么，咱就干什么，干对了得不到表扬，干错了就挨骂，明明是他让干的，一样挨骂，还不能讲理。中国的社会结构里，就没有“讲理”这个场景，中国人讲的理往往只跟身份有关，跟逻辑和对错无关。你能跟你爹讲理吗？能跟县长讲理吗？相反，跟高地位的人讲理本身就是你不讲理的表现。这是塔型结构的典型特点，层层PUA，谁也不能反抗。老外的社会是平面的，你是总统，但是你也是个人，你做事也要讲理，就如同马克龙在大街上跟市民辩论，对方没觉得他是总统就高人一等……

8，通缩的典型特点就是必需品涨价，奢侈品降价，通俗一点来讲，就是买的起的拼命涨，买不起的拼命降。比方说，日产天籁，新车才10万元。过去天籁也算豪车，现在价格跌的多狠呀。

9，现在各个地方都大力发展旅游产业，想捣鼓成旅游城市，旅游大跃进。其实，这些小景区，最终都会成为地方政府的负担，门票平衡不了运营成本。

10，你知道为什么比我们优秀的人拒绝加我们微信吗？因为，被低层次的朋友关注到自己的生活，就是一种危险。稍微有点钱，有点地位的女人，都不会爱上外卖小哥，滴滴司机的。这些底层的人，一个月剩不了三五千，抽烟、喝酒、甚至找女人。对于有钱的女人来说，钱对于男人的吸引力，远远大于女人的本身。所以，有钱的女人和没钱的男人，几乎是物理隔绝的。

11，毕业20年，看同学差距，男生，多来自于杠杆差别。拥抱互联网的，一个层级。没有拥抱的，一个层级。女生，差距多来自于，嫁给了谁。
