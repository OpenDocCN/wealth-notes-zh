
# 大树乡谈星球8.8提问：《外国代理人法案》会出台吗？

## 提 问：
高校微博统一纪念波伏瓦，背后什么情况?西方对兔子高校的渗透很严重了吧?有什么应对政策吗?
## 类似的《外国代理人法案》会出台吗?
## 镇长回答：
可以理解对这件事的忧虑，不过小镇认为，渗透当然存在，但上不得台面，更多是脱产小资产阶级自以为独立思考的闹剧罢了。
你说的“微博纪念波伏娃”，小镇也看到了。的确存在某些豆瓣组之类的秘密小圈子，这些圈子，喜欢搞一些很无聊的行动，比如在高校官微发一些含糊糊的东西，觉得自己很了不起，就像是默默反抗的地下工作者似的，实际这种行为，跟阴沟里老鼠没有任何区别，甚至都不敢公开提出自己的主张，这还不如已经即将被连续判刑的“乱港”分子，对这种人人和事没有必要太在意，也远谈不上渗透。
## 应对办法很简单，等这些人毕业工作几个月，从“脱产阶级”回归真实的阶层，想清楚自己的立场到底在哪，体会真实的社会生活，很快就矫正过来
现在高校很多问题根源在于，几乎所有的普通家庭的大学生，在进入高校之后，几年内脱离了自身阶层，变成了“脱产阶级”,而因为高考前管理严格，高考后过于松散，很容易被小资产阶级思想吸引，总觉得跟主流对着来，才叫独立思考。却忘了，自己的利益其实恰是人民群众，总觉得自己是特殊的，其实很可怜。
那些真正出身富裕家庭的孩子，反而不会这么幼稚，去搞这种无聊的东西，因为他们更清楚社会是怎样的，清楚大学期间应该做什么。
波伏娃这个人，从性别角度是女权主义先锋，但从个人品行作为来看就是“法奸”,在兔子的文化中,属于遗臭万年的，在法国也被认为是背叛，争议性很大。比如波伏娃在二战期间，组织法国女大学生去“慰问”德国侵略者，还举报法国抵抗组织成员这已经不是个人道德问题，就是个人渣而已。
但是呢，毕竟伏波娃是向德国投降，而不是向日本投降，所以在兔子，波伏娃这个人就成了一个简单的符号，那些吹捧波伏娃的大学生们，很可能都没看过波伏娃的作品，可能只是偶尔听说有这个名字，然后临时去找了几句似是而非的话而已。
波伏娃代表极端女权主义、绝对利己精英主义。前者如波伏娃认为一切矛盾都来自于“父权压迫”，而后者则支持她为了自己的利益，毫无底线地迎合侵略者、出卖同胞。这种极端女权和绝对利己，恰恰容易被还没有进入社会、思想比较简单的人误以为是“独立思考”的偶像，实际啥都不是。
比如关于女性目前面临的困难，能说都来自“父权压迫”吗?那又如何解释兔子性别平等在全球都是领先的，这本质是大学生脱离生产劳动和社会实践，搞不清楚现实社会的复杂性，盲目地把看到的某些离经叛道的小道理论，误以为是现实，这就陷入了对现实的抽象理解。
还有一个原因在于高校学生活动中的女性主导性。正如小镇之前谈到的，女性在校园里实际是性别强者，因为从生理上，男性的优势是暴力和体能，然而这是被严格约束的，很多男性被教育的阴柔化，失去了阳刚气，因此女性在校园尤其高校拥有超越实际力量的强势，这种强势在面对现实社会仍然以男性为主导的规则体系时，会产生剧烈的冲突，也就导致校园内，往往以女性为主导进行校园生活、学生活动的组织。而且进入大学后，女性觉醒社会活动的时间，要比男性更早、更普遍，很多男性还在想着去网吧玩游戏的时候，女性已经在琢磨校园活动参与了，这是男女天生的差异。
这就意味着，在目前的高校，包括大学微博账号等，多数由女生运营，而高校管理水平很低，对学生活动严重缺乏监管，这些账号基本上没有任何审核，就是学生自己随便用，出过的问题和笑话太多了。虽然高校也组织一些红色纪念活动，但是形式单板，远远无法适应大学生的需要，而大学生面临就业、婚恋等各方面的压力，也导致极端话语更容易乘虚而入。
当然，在这个过程中，也确实有一些以NGO组织为代表的渗透活动，这个不否认，但也就那样，小打小闹，都不敢在现实中打出旗号，这算什么呢?还不如花几万块钱买水军搞搞舆论呢。而且高校数量很多，其实仔细查下，最近半年在微博发伏波娃的高校，数量很有限，基本就是长三角、江浙沪一带的高校，以及一些相对高层次的高校，占比很低，而更多学校不跟随，可能也是应为组织者的精英情绪，看不上，或者学校管理比较严格，学生更加畏惧，这也恰恰说明，这件事在高校实际影响很小。
而且发这种东西，就那个掌握账号的人就能做到,非常简单，在水军活动中，属于性价比特别高的。
至于《外国代理人法案》,兔子是不会出台的，因为没有必要。美国二战期间为了规范外国影响力搞了《外国代理人登记法》，1965年和1995年都重修过,近年来针对大选“通俄门”，重启了这部法律。但是要注意，这是因为美国搞“票选政治”，又是移民国家，美国法律允许游说团队敢预政治，确实存在合法的外国代理人。
而兔子决策体系本身具备高度的防外部干扰能力,也就是小镇总结的“血脑屏障”,在兔子不允许存在合法的、可以干预政治外国代理人，又何必多此一举去制定这样一部法律。在兔子活动的NGO等境外组织,有《境外非政府组织境内活动管理法》已经够了。
