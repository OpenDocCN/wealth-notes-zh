
# 最损耗男性能量的5件事

原创：U 君 公众号：赢在8小时之外 25年10月2日

无论是男性还是女性，体内都蕴含着巨大的性别能量，而且差异很大。如果你盲目追求男女平等，那么你的男性能量很可能会被压制、被阉割。今天就跟大家聊5个“女人可以做、男人坚决不能做”的事。

如果你能戒掉这些事情，那么你的男性能量会被重新激活，从此以后神挡杀神、佛挡杀佛。如果你认为我讲的东西太虚、太玄，仍然一意孤行，那么你的男性能量会进一步被削弱。

## 1.永远都不要庆祝自己的生日

老祖宗有句话是这样说的：母在不庆生，父不留须。为什么母亲健在的情况下不能庆生？有两种解释：

①古代医疗条件非常差，在女人在生育时很容易出现意外情况，要么保不住大人，要么保不住婴儿。孩子生日这一天有点像母亲的受难日，是非常凶险、非常痛苦的一天，所以不能庆祝。

②严格来说，古人并不像我们这样“每年过生日”，他们只有“过寿”，而且一般是老年人过整数大寿，比如60岁大寿、80岁大寿。但是，“尊亲在不敢言老”，可以想象一下：你的父母今年都80多岁，然后你大摆宴席给自己过60岁大寿，合适吗？

当然，我今天要讲的理论跟上面这两个原因无关。我之所以不建议男人庆祝生日，是因为，生日并不是你通过努力、通过奋斗获得的成就，没有庆祝的理由。

如果你赚到了人生中的第一个100万，值得庆祝；如果你考上了公务员，值得庆祝，如果你娶到了一个美若天仙的老婆，也值得庆祝……因为这些事情都是你费劲千辛万苦才实现的“成功”，必须要庆祝一下，理由有三：

①当你习惯于追逐并庆祝自己的成功时，你的大脑就会自然而然地开始寻找更多的成功。

②当你庆祝自己的成功时，就会向外界传递出一种积极向上的信号，吸引更多的资源、人脉和机会。

③庆祝成功会加深成功的感受，而这种感受会大大提升我们的自信心，从而拥有追求更多成功的动力。

但是，生日根本不算成功，因为我们没有费任何力气。男人庆祝生日，会给自己的大脑传达一个错误的信号：未经努力而获得的事情也值得庆祝。

如果男人大脑里充斥的全是错误的、混乱的信号，那么他最终会沦为一事无成的废物。

类似的，凡是因为自己所在城市有高铁、有机场、有繁华的高楼大厦、出过哪些企业家而感觉自豪的人，大概率都是废物。

因为那些东西跟你一毛钱关系都没有，跟你关系最大的只有两件事：你的时薪购买力如何；你每个月能休息几天。

## 2.永远都不要给别人做陪跑、做嫁衣、做炮灰

我从小到大都喜欢打乒乓球，但是也仅仅是在业余的圈子里玩玩。碰到稍微练过两年的选手，我就玩不转了。

所以，无论是学校里还是单位里组织乒乓球比赛，我都从来不参加。因为我非常清楚，假设20个人比赛的话，里面至少有5个练过的，那么我肯定进不了前5名。

这时候，如果我本着“友谊第一、比赛第二”的心态去参加比赛，只能被别人虐，做他们的炮灰。

所以，我从来不去自取其辱。

我的老师、领导、前辈们也经常劝导我：别把输赢看得那么重，重在参与、交流。

我却不这么认为，我坚信，比赛就是为了赢，如果明知道赢的概率是0，那么就没有参加的意义。

因为赢能增强男人的自信心，输能削弱男人的自信心，成功是成功之母，失败是失败之母，就这么简单。

反之，之前为什么我很喜欢玩网游《穿越火线》，而且每年都要往里面充值？因为在这款游戏里，付费玩家就是大爷，免费玩家就是炮灰。

一个房间里有30个玩家，我有80%的概率能打进前三名，20%的概率能成为第一名。

赢的感觉很好，因为这种信心和成就感是能迁移的，特别擅长玩游戏的男人，赚钱不可能差劲。

## 3.永远都不要用道德和感情去绑架有钱人

一般来说，越穷的男人感情越丰富，但是他的感情都很廉价、很分散；越是富有的男人越无情，他的真情很稀缺、很集中。

之前就有一个读者咨询过我这样一个问题：U 君，我有一个堂哥，他的身价至少有半个小目标，但是有一次我向他借5万元他都不借，太冷血了。

我说，不能怪人家冷血无情，怪就怪你没有本事。他之所以不愿意把钱借给你，就是因为你在他眼里没有任何利用价值，借给你的钱大概率是要不回来了。

说得再难听点，你的父母在他眼里也没有任何利用价值，你们全家的现在和未来对他来说都属于“负资产”，沾上就甩不掉了。

所以，他才敢直接拒绝你，丝毫不怕得罪你，因为得罪你没有任何后果。

再说了，你的堂哥为什么能积累几千万的身家？就是因为他非常拎得清，知道什么是资产、什么是负债。

如果他是个“重感情”的人，今天来个阿猫就借出去5万，明天来个阿狗就借出去8万……他这种性格，就不可能成为有钱人。

因为外面的那些豺狼虎豹会把他生吞活剥的。

反之，如果你真的想借钱，就去找那些比你们身价高1-2个阶层的亲戚，反而能借到钱。

一方面是因为，你们家跟他们家的差距还没那么大，他可能会想：今天我帮了你，哪天我落难了你也可能帮我。

另一方面是因为，他的财富值还不够多，智慧也不够高，不明白“把钱借出去就大概率收不回来”这一事实。

所以，聪明的男人把核心精力都放在“赚钱”和“提升自己的利用价值”上面，只有蠢货才整天去研究人情世故、道德绑架。

## 4.永远不要指望男女平等

男人和女人初次见面、在外面约会吃饭，如果女人主动提出来要AA制，那么男人会觉得这个女人的家教太好了，会肃然起敬。

但是，性别反转就不成立了，如果男人敢开口说“我们AA制吧”，女人会瞬间觉得这个男人太猥琐了、情商太低了，真下头。

类似的，无论是什么行业（服务员、快递员、文员），在同事之间，女性大概率瞧不起同级别的男性，男性大概率会仰视同级别的女性。

举个例子，同样是月薪4000元的餐厅服务员，女服务生大概率瞧不起男服务生，但是会仰视月薪8000元的男厨师。

同样是月薪5000元的办公室科员，女科员大概率瞧不起男科员。因为女人可以把自己的工资全花掉，做一个月光族，甚至负债几十万，也依然不愁嫁。

但是，男人即使不吃不喝、把工资全攒起来，依然买不起车、买不起房，不会有女人看得起他。

所以，如果你是男人，我建议你忘记“男女平等”这件事。

如果你想要跟女性平等对话，要么你有一个非常殷实的家庭背景，要么你的收入至少是对方的3倍以上。

虽然在发达国家，男女相对会更平等一些，但是女性依然会青睐那些收入远高于自己的男性。

所以，每当有男性粉丝咨询我，能不能考那种月薪很低的公务员，我都会问他：你父母的收入情况怎么样？他们能否给你买车买房？

如果他们也都是穷人，那我不建议你考公，否则你在择偶市场仍然处于生态链的最底层，饭店里最底层的女服务员都看不上你。

但是，同样家庭背景的女性如果考上了公务员，即使是月薪2000元，她也能嫁得非常好，甚至能嫁给年入百万的老板。

当然，男性同胞们也不要觉得这种现象不公平，我觉得这是一把双刃剑，一方面男性会被双标对待，另一方面，男性的能量和潜力会被激发出来。

比如，创业者、老板群体里面90%都是男性，福布斯排行榜里面也差不多是这个比例。

## 5.绝对不能没见过世面

我自己就喜欢跟没见过世面的女生做朋友（比如刚毕业的大学生），因为性价比最高。

带她们出国旅行一次、坐一次头等舱，或者买一个奢侈品包包、一部苹果手机，她们都会非常兴奋。

然后，她们就会给我提供巨大的情绪价值和那方面价值。

但是，如果一个女人见过世面、见过世间繁华，那我就会离她远远的，因为我非常清楚，想要打动这种女人，我需要付出的成本就高多了。

但是，性别一反转，这个法则就不成立了。

如果两个人一起出去旅游，女生发现男生没坐过飞机、没吃过日料，走到哪里都觉得新奇，那么女生就会觉得这个男人真没出息。

所以，我一直在公众号里跟广大男性同胞强调，在你年轻的时候，不要攒钱，而是要花钱出去见世面。

举个例子，现在正是国庆假期，我建议你不要在国内旅游，一次都不要去。把钱攒起来，直接去国外旅游。

毫不夸张地说，你在国内花了2万元、去了10个城市，也不如花5000元去一次泰国或者韩国旅行。

大家不要觉得去这种国家很low，你不要忘了，我们14亿人里面，至少有10亿人没坐过飞机、13亿人没有出过国。

也就是说，男人越早成为“国际旅客”越好，至少在“见世面”这个维度，能给你加大分。

在餐饮方面也是如此，一般来说，女性比男性更懂吃、更会吃。

反正我自己是这样，平时吃什么都行，我没感觉日料、泰餐、法餐有什么新奇的、有什么好吃的。

但是，我还是尽可能地多了解世界美食，因为女人就对这些感兴趣啊，我环游世界、品尝世界美食，最重要的目的之一就是女人。

至于开拓眼界这些，都属于次要的。

当然，“见过世界之后就对国内景区祛魅了”属于副产品。
