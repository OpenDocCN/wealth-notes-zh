
# 太平邪国

太平天国建立政权之后，洪秀全就躲进了南京王宫里，老老实实的当他的吉祥物，一直到死都没有再走出王宫一步，也不参与任何政治事务。太平天国的政权则由东王杨秀清一手把控，所有大小事务都由杨秀清一言而决。

就这样的局面，在当时满清朝廷的各种奏折文献中，对于太平天国是否存在洪秀全这个人物，一直都有着巨大的争议。从广西一路追击太平军追到南京的清廷大将向荣甚至在奏折中斩钉截铁地宣称：“洪秀全实无其人”、“或云系刻木偶为之，实无其人”。

而对于太平天国要建立什么样的治理秩序，大家不要报以任何幻想。太平魔军本身就是一支整天装神弄鬼，以屠城和杀戮为凝聚力的军队。建立在这种军队之上的政权，根本也谈不上建立真正的社会秩序，它所有的治策一定是奔着掠夺和杀戮而去，这是毫无疑问的事情。

太平天国最重要的政治纲领性文件就是《天朝田亩制度》，在刚刚定都南京之后，第一时间就颁行天下。这份文件的开篇喊出了“有田同耕、有饭同食、有衣同穿、有钱同使”的共产主义口号。不过对这四句口号，太平天国从来没有采取过任何实际的措施。再强调一次，从头到尾，太平天国就从没实施过任何均分田地财产的措施。

关于这份制度，有三个地方值得重视：
第一，他规定太平天国之下的每一个人都必须信奉拜“上帝教”，不信教的一律杀掉，这种做法就裹挟了他治下所有人。在当时的满清朝廷看来，太平天国的国民全都是教匪，全都应该被诛灭，没有一个人无辜。
第二，他规定太平天国政权将会直接管理到每一户人，在这个政权里面，最低的官吏名为“司马”，管25户农民，凡农民的生老病死、婚丧嫁娶、民事纠纷全都由这位司马大人一言而决，太平政权由此直接具备了最底层的动员能力。
第三，对所有农业产出，农户除了保留最基本的口粮之所需外全部上缴，太平政权因此掌控了最充沛的社会资源，它的政权组织裹挟了所有人，权力能够直达最基层并且拥有最为极致的财富搜刮能力。

如果各位看起来觉得很眼熟的话，不要惊讶，教员就全面复制了这一套制度，也就是极限的底层动员制度。在当时来说，这一套制度赋予了太平天国极强的战争潜力，也因此具备了与满清朝廷长期对抗的能力。

要知道，当时的满清朝廷内忧外患。在内部，总数只不过100万出头的满清八旗子弟，已经多年未曾经历真正的战争历练，早已变成了只会喂鸟唱曲的废物，根本无从再动员总人口超过3亿的汉族人口。没有动员能力，就无从集中资源激发士气应对战争，所以，太平天国从广西杀到湖南，从一团散沙打成一支精兵，再沿长江而下占领南京。从头到尾，满清朝廷都没有调集过所谓八旗或绿营精兵予以镇压，只能依靠地方团练部队打点不痛不痒的追击战。进群加微信 不是人家不想调兵，实实在在是在鸦片战争中跟英国人拼过一场后，手里头剩余的那点强军都死光了，真的是没兵。

在外部，欧美列强虎视眈眈，炮舰就停在广东沿海，随时准备攻入这个孱弱的大国。满清朝廷面对人家的炮舰战战兢兢，也没法把全部精力放在应对太平魔军的肆虐之上，迫于无奈，满清朝廷只能在扬州建立江北大营，在南京城郊建立江南大营，以图遏制太平魔国的扩张。但是，在两者之间完全不对等的社会动员能力之下，满清朝廷根本就不是太平魔国的对手。

1856年4月，太平军攻破江北大营，6月攻破江南大营，此后其势力就席卷整个江南。除了在湖南地区，由于悍勇的湖南人坚决抵抗，太平军无法立足之外，整个江西、安徽、浙江以及湖北和江苏的大部分地区，都纳入了太平天国的势力范围之内，开始实施“天朝田亩制度”，强迫所有人信教，建立起直达农户的治理体制并毫无保留地搜刮民间的每一分钱余财。这种极致的搜刮当然会激发占领区民众的反抗，于是，太平天国之下的民乱不断，从一开始的小规模抗争发展到后期经常就是一州一府的全面反抗。比如公元1861年5月，浙江嘉兴全面罢市；11月江苏无锡农民集体抗租。

面对这些抗争，太平天国政权的处理方式非常简单粗暴——大面积的血腥屠杀。太平魔国对每一次的反抗都设定一个屠杀区，在屠杀区内的人，无论有没有实际参与反抗，全都要被杀光。太平魔国辖下的人口规模因此发生了猛烈减少，因为满清和太平魔国都没有严谨的人口登记，所以这个恶魔政权到底在江南地区杀了多少人，也没有精准的数据统计，史学界的估算是8000万到1.6亿之间，原本一片繁荣的江南大地几乎被太平魔国杀成了荒漠。
