
# 金融三剑客：家有烂尾娃怎么救

## 一、差异

### 1. 目标
人首先要想清楚，自己到底要啥，这个大目标，不要弄错。生活中的目标并不来自于老师，来自于你自己。你从小到大接受的那些教育，你有没有发现都是被动的？你不是说自己做了一个人生规划，自己拆分成小学，中学，大学等阶段，然后请老师来给你施教，不是的。当你在还懵懂的阶段就被领进了幼儿园大门，从此之后，都是老师告诉你，你该做什么。然后不停的给你月考期中期末考，来矫正你，让你习惯。话说这个过程持续多年之后，你就会形成一个潜意识，就是得有人告诉你：你该干嘛。实际上，从你步入社会的那一刻起，就没有人告诉你，你该干嘛了。你会发现，老师们都集体退场了，接下来是你的自由活动时间，你的余生爱干嘛干嘛，并没有什么生活上的裁判给你打分。

### 2. 思维
学校是线性的，生活里不是。线性的意思就是说B（中学）是C（大学）的前置条件，A（小学）是B（中学）的前置条件，如果你要抵达C，必须先经过A，然后B，最后才能C。如果你想上来就直接干C，不是你能不能的问题，而是不允许你这么干的问题（其实学的那堆东西，后面基本上都用不上，除了正儿八经的科研）。你要知道，你从小接受的那个教育的目的并不是为了教育，而是为了选拔。什么意思呢？既然是选拔，它就必须做成线性的，否则就没有公平性了。但是你进入社会之后，这个限制就取消了。

## 二、路径

### 1. 选择
其实你孩子上小学第一天，你就可以跟他讲，他可以考虑三个选择：
1)不读书。连基础教育都不受，我们爱干嘛干嘛，好处是咱一点苦都没吃，坏处是你现在玩的开心，玩着玩着，身边就只剩二流子了，正经人都读书去了，到时你还觉得有趣么。
2)读书但不参加高考。你要重点加强外语的学习，低年级就出锅留学，需要长期在锅外读书，生活。甚至某天你发现回来都不习惯。
3)读书参加内地高考。这里面又分一本，211,985，…，你的目标不同，战略战术不同，最后吃的苦也是不同的。

### 2. 沉没成本
也就是说，他选好目标，你再根据他的选择，给他分配适合的资源以及方法论。然后每隔一年，你就问他，你确定还是去年的选择么？是，咱们就继续，不是，咱们就调整方案和资源。如果他连续十次（高一了）选择都是第三项，那还能调整么？不能。为什么？因为你们家的沉没成本已经砸出去太多了。烧一锅水，你要么烧到100度，要么干脆一根柴火别浪费。你刚开始烧，温度低的时候我就会提示你要不要止损。但你都烧到90度了，我只会劝你坚持，务必坚持到100度，否则你前面的苦，全白吃，这个90度就叫沉没成本。

## 三、海外
咱们先来说说出去的选择，前提还是回来，不回来就不是单纯划算的问题了，而是一个认可度的问题。你哪怕在锅内读的是清北，都不一定有当地那些锅际排名比清北略微靠后的学校的招牌好使。因为强龙不压地头蛇，人家郭嘉的人，未必知道清北是什么。

如果你在锅内读A校，去海外读B校，B校几年的学费生活费比A校多了几百万，但是B校回锅之后，企业的起薪高，升迁快，B和A在企业那里的差值，可以弥补你那几百万多出来的教育费用。如果可以，那就是划算的。各位可以自己去外面大企业HR打听下，一般来讲，别说弥补50%，甚至30%都非常艰难。也就是说，它是净亏的，无非亏多少而已，明白吧。

但是，你注意，我要转折了。对于有些家庭的孩子，账不是这样算的。昔日乾隆要平定准噶尔（康熙留下来的两个命题之一，雍正完成了其一锅库银的事，打战的留给了乾隆）的时候，大臣们都反对，理由是不划算，这地方拿回来，也是我们给它转移支付。然而乾隆力排众议，他说，我们不能只算经济账，也要算正值账。他能说这句话的前提是什么？是雍正给他打了基础，就是家里有余粮嘛。所以小到一个家庭，也是一样的，你到底应不应该只算经济账，要看你家的情况。当你们家一年的收入，可以覆盖孩子全部的留学费用时，剑客认为，留学是值得的。

这件事的作用不在于学历，而在于给到你更多比较的对象。各位知道不知道这世上所有事物或者商品溢价产生的原理是什么？溢价实际上是因为什么产生的？定价体系的真相

因为人的见识少产生的。我们来看一个现象，单位里面，领导总是喜欢安排大家做一些其实不打粮食的事情，就是说这些事其实和企业盈利无关。越是非民企，越是普遍。你看着明明不是很重要的事情，还搞的很重要的样子，又是评比，又是排名。然后还要分个优良中差，搞得好像谁比谁高了半级，年底就能多分条带鱼似的（在公立学校当老师的读者应该非常有感受）。其实民企里面也搞这套的。能升管理层的就升你个主管，如果你是需要出门见客户的，还会允许办公室给你你名片上印个总监的头衔。不能升管理层的同志，也会给你各种所谓的提升路径。什么高工，资深高工，甚至到后来，都有企业内部的科学家，首席科学家，就跟金融机构里面的首席分析师一个调调（体质内一些部门也在搞，什么首席会计师，首席啥的）。这套东西的目的是什么？就是增加你的沉没成本。剑客讲过，人的决策系统绝大部分情况下，都是受到预期左右的。

假设你想跳槽（受委屈了或者钱少了），但如果你还留恋自己在原单位首席的头衔，就会产生一种损失厌恶的情绪，你会觉得自己好像损失了什么似的。这个损失厌恶的预期，就会导致新单位给你的加薪，都不香了。因为在原单位，你有很多确定性，你知道工作体验是什么，新单位过去了会有很多不确定性，你唯一能确定的是加薪了，但是你不知道去了之后，自己高薪能拿多久，是否还是首席，是否还能得到内部认可的那种“地位感”。
