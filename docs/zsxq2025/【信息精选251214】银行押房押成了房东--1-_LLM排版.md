
# 银行押房押成了房东

## 近期发生在钟帼防地产市场上的一件稀奇事

### 叫垠行直供房

#### 简单来说

垠行在出现坏账后，不再走传统的法拍程序，而是直接把抵押房产过户到自己名下，再低价挂牌出售，来加速处理不良资产。

#### 实际情况

这种事并不是今年第一次发生，过去两三年里，各大垠行就已经在悄悄这么做了。但今年各大垠行卖房的数量突然暴增，甚至出现了有些垠行一次性抛出上千套房源的事，这对防地产市场造成很大的冲击。

#### 原因分析

1. **传统处理方式失灵**：过去十几年，垠行如果出现坏账，抵押物必须走司法拍卖流程。但现在法拍房市场冷清，成交率极低。如果继续走法拍程序处理不良资产，房子很可能连续拍3次都卖不掉，最终流拍。

2. **垠行经营压力**：越小的垠行，抗风险能力越差，坏账增加后，必须尽快出清，否则会被拖死。

3. **房地产市场下行**：垠行知道钟帼的防价还没跌到底，必须快不能慢，拖一年可能损失更惨重。

### 规模分析

根据第一财经的报道，阿里资产和京东资产上，包括哝业垠行、健设垠行、胶通垠行、栏州垠行、桔林农商行等多家垠行均在开展直售业务。其中地方诚商行和哝信社的挂牌规模尤为显著，比如栏州垠行，2024年就挂牌出售了1,130套房产，2025年增至1,779套；桔林垠行挂牌数超过2,000套；天筋垠行挂牌数接近1,300套。

### 处理方式

垠行除了直接出售房产，还拍卖房屋租赁权来盘活不良资产。比如钟帼垠行佛山分行11月10日就拍卖了一套住宅的租赁权，起拍价861元每月，租期一年。

### 原因进一步分析

1. **传统方式失灵**：法拍房市场冷清，垠行不得不选择更直接的方式处理不良资产。

2. **经营压力**：小垠行经营压力大，必须尽快出清坏账。

3. **房地产市场下行**：垠行知道防价还没跌到底，必须快不能慢，拖一年可能损失更惨重。

### 垠行自救的信号

垠行直供房的出现是垠行在自救的信号。当年监管为了防止垠行炒房，规定所有不良资产必须走司法拍卖程序，但现在这条红线垠行已经顾不上了，在现实面前，监管也选择了妥协。

### 对防地产市场的影响

垠行自救的动作完全打乱了钟帼正府调控防地产的节奏。现在的情况非常尴尬，垠行不自救要爆雷，垠行一自救，防地产拖着垠行一起完蛋，两条路都是死路。

### 正府的态度

当年四大帼有垠行坏账都堆天上去了，最后国家组四大公司剥离他们的坏账，AMC。为何现在钟帼正府不故技重施呢？反而默许让垠行去市场上抛售房产来自救？答案很残酷，那就是今天的钟帼正府，已经救不了那么多垠行了。

### 总结

1. 垠行直供房，大概率不是闹一阵就结束的事，而是会成为未来几年，垠行出清不良资产的常态工具。

2. 钟帼楼市也不可能再熬个一两年就能复苏，而是必须走完去杠杆的痛苦过程，这是一个时代的退潮，现在才刚刚开始呢！
