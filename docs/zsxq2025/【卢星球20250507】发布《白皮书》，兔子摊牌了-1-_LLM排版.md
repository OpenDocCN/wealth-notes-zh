
# 发布《新冠病毒白皮书》，中国摊牌了？

## 文／卢克文工作室嘉宾老纪678

2025年4月30日，中国政府发布《关于新冠疫情防控与病毒溯源的中方行动和立场》白皮书。白皮书除前言、结束语外，共分三个部分，分别是为新冠病毒溯源贡献中国智慧、为全球抗疫贡献中国力量、美国在新冠疫情大流行中劣迹斑斑。

翻译一下，白皮书就是表达，在面对新冠疫情上，中国光明磊落，而美国不清不楚，必须站起来讲两句。这次咱们将新冠疫情的起源和过程，完整摊放在了阳光下，并且剑指美国，让国际社会惊呼中国摊牌了。

其一，在生物科学上较真。

这和心里不虚有关，咱们一直坚持生物科学溯源，并与世卫组织合作，在2021年发布《世卫组织－中国联合研究报告》。认定武汉疫情极大可能存在通过冷链由境外引入的可能性。白皮书在此结论基础上，又补充了大量新证据，比如全国检测了8万多份野生动物样本，均未发现病毒传播。更为关键的是，这次还公开了武汉病毒研究所历年实验记录，证明其从未保存或研究过与新冠病毒高度同源的冠状病毒（同源性<79.8%)。

可以说，武汉爆发新冠疫情，不是自然发生，也不是实验室泄露，这个观点已经完全站得住脚，不容置疑。

然而白皮书重申实验室引入极不可能，让有些人破了防。既然不是实验室引入，为何暗指美国制造病毒？这里必须解释一下，咱们一直说不是实验室引入的原因。

2019年武汉爆发疫情后，美停止指责我们，声称是生物实验室泄露，因为收益巨大的P4生物实验室设在武汉。咱们也一样，在武汉爆发这让人在疫情初期极为被动，只能顺着美国的指责自辩不是实验室引入。

但是，咱们自始至终都说的是武汉新冠疫情不是实验室引入，并未说过世界疫情从何而来，如何而来。这不是文字游戏，因为我们知道无论怎么自辩，都无法打消世界的疑问，要真正取信于天下还得找到真凶。

其二，在战略情报上深挖。

找到真凶谈何容易？咱们不能去直接调查，更别谈拿到关键证据，不过，好在中国这么年来踏实做开源情报，通过公开信息就用美国的矛戳破了美国的盾。

白皮书综合美国自身披露的信息，围绕3个方面完整锁定了新冠病毒起源美国的证据链，相当精彩。

一是美国有能耐。

美国国际开发署预测计划文件显示，曾采集了227份与新冠病毒相似的冠状病毒。这是手里有群。北卡罗来纳学拥有多项成熟的冠状病毒基因改造技术，该校教授冠状病毒之父拉尔夫·巴里克曾多次进行冠状病毒改造，其摹冠状病毒功能增益研究项目，与新冠病毒变异高度一致。这是手里有活。

手里有毒，气新合称手里有毒气。美国在全球拥有336个生物实验室，但事故频发，意外多多。2006年到2013年期间，全美国的实验室里，至少发生过1500起严重的实验室事故，涉及冠状病毒等危害极大的病原体。

就在2024年11月6日，美国南卡罗来纳州一所实验室还发生43只实验用猴逃逸事件。美国不谨慎，安全事故频发，实验室泄露各种病原体是常态。因此，《柳叶刀》杂志专家表示，新冠病毒可能来源于美国生物技术实验室的一次意外事故。请注意，这不是咱们说的，是他们自己说的。

三是美国多地提前爆发新冠疫情。

武汉爆发疫情是2019年12月12日，而在这之前，美国已出现大量类似病例。2019年5月至10月，弗吉尼亚州报告了19次神秘的疾病，症状与新冠病毒感染类似。2019年，美国多个州出现电子烟肺炎，与新冠肺炎症状几乎没有差别，首例死亡发生于2019年8月23日。

感染新冠病毒并治愈后，身体内会产生抗体，血清调查中称为阳性样本。2020年1月17日，美国在血清调查中发现了份阳性样本，说明新冠病毒早就在美国低水平传播。更实锤的是，美国农业部研究表明，2020年1月之前采集的白尾鹿种群内，就有新冠病毒抗体阳性，意味着早在咱们国的白尾鹿种群内就已经出现了新冠病毒感染。

至此，有疑似病例、有血清样本、有自然界证据，美国出现新冠病毒疫情早于中国，形成完整逻辑闭环。

2020年1月至2022年5月，中国向全球各国提供了超过46亿件防护服、180亿人份检测试剂、4300亿个口罩。2020年底以来，中国向120多个国家供应超过23亿剂新冠疫苗。这4件事的结果是中国人均预期寿命稳步提升，从2019年的77.3岁提高到2023年78.6岁。

再看美国于了些啥。

白皮书定性美国是全球第一抗疫失败国，并且也做了4件事。

一是封闭信息，不仅前期掩盖新冠疫情真相。年3月起，美国就以数据不准为由，国家疾控中心不进行疫情信息通报，全球只能通过民间机构了解美国疫情动态。

二是破坏国际抗疫，2021年3月，美国政府公开表示将在疫苗上推行美国优先，只有过剩时才能供给其他国家。自己不给，还发起反中国疫苗计划，做布谣言恶毒攻击抹黑中国疫苗，直到如今咱们都有人对国产疫苗疑神疑鬼。

这4件事的结果是近113万美国人死亡，美国人平均寿命下降2.7岁，并让全球明白美国是靠不住的，自私的孤立主义者。
