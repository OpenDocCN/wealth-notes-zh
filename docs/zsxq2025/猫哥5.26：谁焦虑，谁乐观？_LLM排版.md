
# 猫哥5.26：谁焦虑，谁乐观？

最近看到一份麦卡锡做的《中国消费趋势调研报告》，这份报告是2024年年底麦卡锡调研了11930名中国消费者。有一些内容我觉得很有意思，这里给大家分享一下。

首先就是，现在中国消费者，谁对未来最悲观？最悲观的人群是农村42—65周岁的。农村中老年人群最悲观，这个结论可能出乎很多人预料，但是其核心逻辑是解释得通的。

首先，因为性别的错配，农村人讨媳妇是很困难的，城市有3000万剩女，农村就有4000万光棍。没有老婆，年龄也大了，农村医保相对城市要差得多，加上经济大环境不好，中老年这个人群在外打工赚钱也越来越困难，在这种情况下怎么不可能悲观？

也就是说在经济下行阶段受到冲击最大的还是农村中老年这个群体。

次悲观的人群是1、2线城市26—41周岁这个人群。也就是说大城市目前就业主力人群是仅次于农村中老年人的次悲观人群。

为什么这个人群对未来悲观？两个原因。

首先是30—41岁这个人群结婚主要在2016年—2022年左右，按照中国结婚必须匹配婚房的传统，2016年—2022年这个阶段恰好是匹配中国房价最高的时期，大城市的年轻人在结婚的时候掏空6个钱包买下了婚房，现在面临3个压力：一个是婚房价值缩水；一个是育儿压力；一个是35岁之后因为经济下行的裁员焦虑。

而26—30岁这个人群主要是大学毕业之后刚刚开始就业的人群，现在就业市场确实不大好，一方面工作不好找，一方面薪资下降幅度很大。

最让我没有想到的是，现在大城市的年轻人(30岁以下的人群)居然与中年人(30—41岁)对未来悲观指数居然是一样的，年轻人与中年人对未来同频感受，这就是经济普遍的寒意传导下的结果。

有悲观就有乐观，那么现在国内那些人群对未来比较乐观呢？

最乐观的人群是大城市58—65周岁的人群。

为什么？这个人群基本是出于刚刚退休或者即将退休的人群，大城市目前退休金与医疗保障是非常给力的，让这个人群没有后顾之忧。

更为重要的是，58—65周岁的大城市人群完整吃到了改革开放四十年的高速发展的红利，最差也有了一套价值不菲的房产，也有一定的积蓄，加上良好的营养条件，让60岁足有的人群精力与体力远非过去能相比。

身体好，有积蓄与房产，未来还有良好的退休金与医疗保障，所以这块人群基本没有受到经济下行的冲击，对于未来最为乐观。

次乐观的人群是三线城市家庭收入2万以上的人群。

三线城市家庭收入能达到2万以上已经不是普通的群体，或者是体制内实权干部，或者是在城市做生意有稳定的现金流。

经济下行冲击最大的大城市(主要是房产贬值幅度较大，加上外贸出口的不确定性)，三线以下城市冲击反而要小一些，三线城市经济生态是最稳固的，上限没有大城市高，但是下限也不低。

所以在三线城市如果有稳定的生意，或者说体制内掌握实权，基本就不会受到经济下行的冲击，对未来的乐观程度仅次于大城市60周岁左右的人群。

以上就是我分享的麦肯锡调研报告内容，这里简单总结一下。

现在经济下行，大部分劳动力整体是对未来比较悲观的，而对未来乐观的只是小众群体——或者大城市临近退休，或者在小城市收入有护城河。

最后谈谈我的观点。

从去年924高层会议以来，整体的救市政策以及最为关键的财政政策，基本就是雷声大雨点小，所以924到现在大半年过去了，当时提出的主要目标都没有实现，包括房地产的止跌企稳，以及经济走出通缩的状态。

造成雷声大雨点小的原因我分析可能有两个。

其一,决策层最为关注的经济指标是GDP, 看起来还可以，基本都在4%—5%左右徘徊(实际上现在经济两级分化很明显，大部分大企业还过得去，但是中小企业很惨淡，而中小企业关联的恰恰是最大的就业市场),因为GDP看起来还过得去，所以并没有触发决策层要加大财政力度救市的底线。

其二,外部的巨大不确定性让决策层在大规模救市问题上比较谨慎，感觉是要留足弹药应对外部不确定性的冲击。

最近官方媒体发表了一篇文章《理性看待中美日内瓦经贸谈判成果》,大致意思就是虽然日内瓦中美达成互降关税的协议，但是这很可能是美方给自己零售商一个囤货期，一旦这个囤货期结束，美方很可能再次挑起关税战。

所以决策层的财政子弹可能是在特朗普未来再次挑起关税战再使用。

所以目前的经济状况是比较磨人的，大家都在熬。

5月12日公布的联合声明显示，美方承诺尽快取消4月8日和9日对华加征的91%非理性高额关税，同时将34%所谓“对等关税”中24%的部分暂缓90天，为下一步中美经贸谈判留出一个缓冲期。

舆论普遍认为，中国对美关税战取得了积极效果。面对美方霸凌行径和非理性操作，中国以负责任大国的态度，对美进行有力反制，对此应予以积极的肯定。但是我们也应该意识到，中美贸易冲突并没有结束，随后的经贸谈判必将是复杂而艰难的，对此我们要有清醒的认识。

美对全球加征所谓“对等关税”后，美国国内政治经济态势发生了较大变化，首先是美国的消费者难以承受物价上涨和货物短缺的影响，美方关税政策也遭到了美国商业巨头的反对和抵制，甚至在美国政府团队内部也出现了巨大分歧。其次是美国的经济运行指标出现了明显的负面变化，美经济衰退和可能陷入滞胀的预警信号已经说明加征关税问题的严重性。

更令美方头疼的是，如果不及时止损，美国经济指标二季度一旦出现大幅下滑，三季度再作出止损的决定恐为时晚。美国经济如果在新政府上台第一年出现衰退，将影响其中期选举。从以上角度分析，不排除美方为了止损经济而采取缓兵之计的可能。一旦美国内政治经济矛盾缓解，美方关税威胁也许将卷土重来。
