
# 美国也要过苦日子(1)

出来混，欠的债都是要还的，
新冠期间，很多人都躺在家里，但依旧歌舞升平，
不是老板给我们发钱，
而是老板从未来的我们借钱，发给当时的我们。
美国这几年也是如此，马照跑舞照跳，
也不是拜登政府给他们发钱，
而是拜登从未来的美国人借钱，发给了他们。
我们如今在还疫情时期的账，

勒紧裤腰带，所以感受很痛苦，
同样，美国政府也要还他们大放水的账，
终有一天，也要勒紧裤腰带的，
只不过还账的负责人从拜登变成了特朗普。
兔子之前依仗的是金融集团，
因此是用金融放水的方式来处理债务，
让金融集团在过程中作为中间人赚的盆满钵满，
如今换成了财政集团上台，
处理债务的方式就变成了财政作为主导。
同样，拜登依仗的是金融集团，
更多是依仗美联储和华尔街来处理债务，让全球
金融资本集团在过程中赚的盆满钵满，如今MAGA集团
上 台 ，
处理债务的变成了亲近特朗普的各类MAGA资本。
这几年兔子经历的局面，全球也都会遭遇，
无节制的放水会被喊停，全球经济会骤然U转冷，
特朗普也会大搞全球版的非税收入和远洋捕捞，各国
也会像地方政府一样融资难度激增，过苦日子，只有
MAGA 的美式高质量发展能够一枝独秀，
