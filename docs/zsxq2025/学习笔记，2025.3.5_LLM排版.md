
# 学习笔记，2025.3.5

1. 一段关系中，不管是长期关系还是短期关系，都要尽量摆脱“说服他人”的念头。能被说服的人，看到事实之后自己会改，不能被说服的人，撞到南墙也只会觉得是南墙碍事。
2. 现在玩无人机的门槛说高，很高，得考证，说低，很低，谁都能上手玩，管不过来。如果可能的话，尽量远离有无人机的地方，因为这种东西玩不好，就是高空抛物，砸到人身上容易出人命。
3. 如果有一个三体人那样全方位监控一个人每天都做了什么的东西，你会发现，大多数人不成功是很正常的。
4. 劣质基因就是好吃懒做，逐渐堕落，而优质基因是什么样的？不断的精进，对身体，对学业，对事业，都是如此。
5. 希望你是健康的、安全的、永远不挑战概率的，你慢慢就会发现，人活在这个世界上，其实很不容易，能活着本身就是奇迹了，我们要珍惜这个奇迹。
6. 看我们的养老模式有什么BUG，很简单，参考发达国家即可。
   - 我们比发达国家退休早，发达国家多是 65 岁退休。
   - 我们比发达国家领的多，我们退休后，退休金只是略少于工资，但是在发达国家里，譬如日本，养老金也就是工资的四分之一左右，这就是为什么日本老人还在拼命工作的缘故。
   那么，这个缺口怎么解决？
   - 要么，延迟退休。
   - 要么，降低退休金领取比例，这个不现实，会出大问题。
   - 要么，让社保基金赚钱
