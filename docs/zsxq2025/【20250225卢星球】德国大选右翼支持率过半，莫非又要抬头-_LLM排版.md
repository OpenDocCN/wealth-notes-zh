
# 德国大选右翼支持率过半，莫非又要抬头

## 基本盘分析

联盟党（黑党）由基督教民主联盟和基督教社会联盟组成，两者都奉行基督教民主主义。联盟党的基本盘是当地基督徒、中产阶级和企业主、农场主。联盟党自西德首任总理康拉德·阿登纳确立社会市场经济模式以来，逐渐形成了“中右翼”保守主义政治倾向。经济发达的西部南部仍是联盟党票仓，这里大部分民众保守而温和，对多元化也有一定接受度。

选择党成立于2013年，2017年崛起为反难民的极右政党。选择党在东部六个州取得第二大党地位，成为议会第三大党。选择党的基本盘是东部的乡下人、底层群众，走平民路线。

## 新票仓分析

德国经济连续两年负增长，难民问题也日益严重。大批选民由于对朔尔茨和执政联盟的失望而离开，但真正吸引他们的是联盟党和选择党的反非法移民主张。这两党在指责朔尔茨政府对非法移民打击不够，力主驱逐非法移民，加强边境管控，收紧移民政策等方面达成了一致。

## 默茨新伙伴

联盟党党首默茨准备组阁上台。鉴于上届三党政府扯皮垮台的教训，联盟党多半倾向于两党组阁。符合这个条件的只有选择党和社民党。选择党基本可以排除，尽管党魁魏德尔率先伸出橄榄枝，提出和联盟党合作，但默茨强调，由于对外政策立场的不同，两党无法走到一起。联盟党只能硬着头皮找社民党。朔尔茨作为总理，把德国祸祸成现在这样，作为社民党候选人，又以16%得票率的成绩，交出了1887年以来社民党最差竞选答卷。于国于党，他都是千古罪人。

## 未来展望

联盟党和社民党组成“黑红联盟”共同执政，是可能的结果。而它们的执政能否扭转德国目前的颓势，能否在世界大变局中，让德国和欧洲能占有一席之地，还是取决于两党能否克服分歧，能否像默茨留给选民的印象那般严谨务实。德国和欧洲的命运已经到了关键节点，默茨手中握着的，可能就是避免德国再次滑向极右翼深渊的最后一次机会。
