
# 刑检还是民行？如何抉择

匿名用户 提问：
公周兄好，近有调岗及下一步努力方向问题，
恳请指点。 一、背景： 法硕男 ，有工作
经验，现省会基层试用期检助。个人风格稳
重细致、动静适中、踏实肯干。进单位后一
直在刑检，部门相处融洽，领导亦比较认
可。近期定岗，党组将安排我到公益民行部门。
此前，分管、中层员额曾多次与一把手沟通，
想留下我，他也一度应允，但最终仍安排我
到新部门。个人分析原因：
人员平衡。新部门新老不接，仅有一个员
额（近几年退休）加几个老资历编外，剩下
俩年轻助理都是女生。
出成绩。一把是刑检出身且新到院里，可
能考虑新部门近年更易出亮点，有个年轻男
助理更好。 二、个人打算： 虽然很喜欢刑
检想留，但未再去找一把谈。一是还不熟，
也本未征求个人意见。二是分管他们已提过
多次未获准，我再去说怕更引起逆反、反感，
像是不服从组织安排。所以下一步准备在新
部门先学先看。办案比刑检少很多，就尝试
多写调研、案例、论文等；遇到可打造的案
例就及时跟领导报，学着写，看能否有亮点；
参加些竞赛，争取一定外在荣誉，培养院内
口碑等。三、规划及疑问：入额要 年才
到门槛，但本院员额数基本已满。个人初心
本想在系统内、刑检口，往业务专家方向努
力。但听员额讲，在那边干好了就可能面临
丢不掉、回不来的情况。新部门好处是，在
本市系统内同一条线上，总体都比较缺年轻
人，而刑检人才济济，所以后面竞赛、入额
压力可能小点？ 请问公周兄如何看待这次
调岗。我该以兴趣为主，在半年或一年后找
时机申请回刑检；还是争取潜在的更高入额
可能性 ，错位竞争更好？恳请指点开导。

公周解析： 对组织安排与个人意愿之间存
在的矛盾，你能够正确看待、冷静分析，处
理思路比较清楚，对在民行部门刷存在、树
人设、出亮点也有具体打算，我表示赞成。
检察院专业性比较强，选择某个专业领域以后，再调换的难度确实比较大。如果有机会留在刑检，或者尽早调回刑检，我完全赞成。现在主要问题是，民行部门比较缺人，一把手综合权衡才把你放过来。
即便半年或一年后，情况可能仍然一样，即便提出申请也未必能如愿。对此还是应该做好两手准备，防止期望落空影响心态。 为
增加后续寻找时机调回刑检的成功概率，可以
从以下几个方面尝试：
一是按部就班做好工作，注意态度和工作效
率，防止部门领导和一把手猜疑你故意不好
好干。但既然你对这个领域不是很感兴趣，
也不打算长期从事，就没必要表现出过多深
入钻研的努力。
二是学习积累刑检业务，表现出兴趣和培养
潜力。比如一些征文、调研、竞赛等，民行
相关课题为主，偶尔也可以掺杂刑检内容，
释放一些信号。
三是与刑检部门的领导同事保持良好互动，
巩固他们对你的认可度，遇到合适机会请他
们出面帮你争取。四是对民行部门领导也
要保持良好关系，照顾他的面子和自尊，不
要显得瞧不上民行业务、总想攀刑检的高枝。
将来要放人，还得他同意才行。
