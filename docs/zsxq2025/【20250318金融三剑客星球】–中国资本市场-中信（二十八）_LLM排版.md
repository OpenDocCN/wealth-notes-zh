
# 中国资本市场-中信(二十八)

## 一、敏锐

彭老总给教员递上的《意见书》中，一些用词如“小资产阶级狂热”、“政治性”等在当时异常敏感的政治背景下显得过于激烈。他针对大跃进中的人祸，狠批猛抓，直指祸端的始作俑者，这引起了教员的反感。因为那个始作俑者，教员不可能不保。

1959年4月的二届全国人大一次会议后，刘副同志刚刚当选最高领导人，在庐山会议上不可能立即批臭他。彼时的外部环境恶劣，社会主义阵营内持续动荡，波匈事件余温未散，赫鲁晓夫集团开始谋求与美国亲近。更不要提1958年的长波电台事件、联合舰队事件、金门炮击战等一系列事件。此时中国内部绝对不能自乱阵脚。

1956年3月17日，教员主持召开中央书记处会议，做了主调发言：赫鲁晓夫的秘密报告值得认真研究。至少可以指出两点：一是他揭了盖子，一是他捅了娄子。说他揭了盖子，就是讲，他的秘密报告表明，苏联、苏共、斯大林并不是一切都是正确的，这就破除了迷信。说他捅了娄子，就是讲，他作的这个秘密报告，无论在内容上或方法上，都有严重错误。

大半年后，1956年11月15日，教员在八大二次会议上再次指出：我看有两把刀子，一把是列宁，一把是斯大林。现在斯大林这把刀子，俄国人已经给丢了。

上述讲得这些，彭老总显然都没听进去，也就有了后面的被利用。在后来的温格中，刘副同志还在秘密指示红卫兵从三线建设工作岗位上揪回彭老总，反复进行批斗。

这就是政治敏锐性，其实早在延安整风时，刘副同志那时也奈何不了彭，抗美援朝的巨大功绩更是让刘副同志比肩不了。但是委身官场时，这种性格则极易四面楚歌，给人以落把柄。比如高饶事件时就开罪了刘副同志，“军事党”-理论更是严重的政治之过。

## 二、对照

庐山上，刘副同志直接开说彭老总：“魏延的骨头、朱可夫的D性、冯玉祥的作风”、“与其你c党，还不如我c党”。这让教员在面对彭老总的万字《意见书》时，陷入了两难。

特别是谈及朱可夫，那可是那一时期社会主义阵营中极度敏感的事件：1957年秋天时，朱可夫被赫鲁晓夫打倒，并被污蔑为所谓的“反D分子”，他的国防部长位置也被换成赫鲁晓夫的心腹，马林诺夫斯基。

说到换国防部长这个敏感话题，历史没有新鲜事，去年俄罗斯同样干了此事，即国防部长绍伊古也被拿下，咱们在这里可以与历史简单对照下，你会理解的更深刻。

今天执政的统一俄罗斯党是由几股支持普京的势力合并而成，这其中最大的那一股就是绍伊古的势力。所以普大帝让他去主管国防，本意是让他作为自己的代表把手伸到军方体系中。各位可能有所不知的是，虽然普京名义上当了二十年的老大，但要不是这场战争，他在军方体系中一直就插不上手。

除了末日将军苏洛维金是他提拔的外，其他军方关键岗位上的实权人物都是原有的军方体系自己按照内部惯性产生的。这一点也是有传统的，苏联时期的传统一直延续至今。

其实这也是历史的客观规律，任何一个领域如果长期优先占有社会资源，它就会逐渐形成一个自己的体系（圈个小山头，彼此拜师称大佬，互相提携对方的子女，关起门来玩封闭游戏）。

不管是基于传统惯性也好，还是基于利益的捆绑，它都会自成体系，成为一股独立的势力，外部是插不上的手的，这种情况在任何国家都出现过。

所以你现在知道了，川普派出去查美军账目的人员为什么总是出意外，而且也只能不了了之了？

话说在绍的管理下，虽然他不懂得打战，但做工程预算等工作还是很厉害的。俄罗斯每年的军费也就是600-700亿美元，在过去绍大将主导国防的十年中，这些钱加起来还没有美军一年花得多（美军一年就8000多亿美元的花销）。

绍大将就是用这点钱搞了军改，装备了几十个营级的战斗群，生产了四架图160战略飞机，生产了300多架其他飞机，弄出几千辆坦克，还建造几艘军舰。同时还支持了一场打了两年多的战争，还给大家都加薪了，各种补贴也到位了，甚至还让整个体系中的成员都能分点钱。

各位是不是有点佩服他（绍伊古）的统筹能力和长袖善舞的能力？

不过人越是春风得意的时候，往往就越容易马失前蹄，因为你无形之中就会自我膨胀，而你的这种膨胀又会给别人带来很大的压迫性，从而逼着别人作出反应。

那个因为腐败被抓的心腹和绍大将在国防部中的另外两个心腹，就因为得意而忘形，他们竟然敢公开议论“建储”的问题（吃了豹子胆了）。

话说有些事清即便是关起门私下聊，一旦泄露出去，都会引发剧烈反弹。何况是公开讨论。

这难道不会让人产生一种司马昭之心、路人皆知的危机感么？

于是我们看到，先是4月24日，副国防部长伊万诺夫（绍伊古的钱袋子）被逮捕。然后是5月13日，国防部第一副部长察里科夫和另一个副部长克里沃鲁奇双双辞职，国防部人事局局长库兹涅佐夫被拘留（三人都是绍伊古的亲信）。

普京不仅快速拿掉了绍伊古的防长职位，还把他的亲信一锅端，这也为别洛乌索夫的上任铺平道路。5月普京突然提名别洛乌索夫为国防部长，原国防部长绍伊古，则改任安全会议秘书和军工委员会副主席（明升暗降）。

其实按照俄罗斯一贯简单粗暴的形式风格，当下的这种明升暗降的手法已经是非常非常克制的手法了。

## 三、特性

说白了，在老大的眼里，他是在扩充自己的势力，即当你这个势力还听话的时候，你就是我代表，一旦不太听话，那你就会变成更危险的存在（拿掉没商量）。

不理解的，这里剑客把权力的三大特性简单带大家串一串，你就懂：

1. 唯一性（定于一尊）。一个组织内，权力只有一个，这个权力是不喜欢被分割的。

所以，历朝历代的政治发展，就是一个权力逐步集中的过程，到了清朝的军机处，就完成了集权的最高峰，所有人都是权力的奴才，都是临时性的授权，只有参与不能决策。

换言之，一个组织内，除了一把手，其余的人都是权力的服务员（彭怼三面红旗是大忌）。

2. 畅通性（如臂使指）。权力是不喜欢有任何的阻碍的，必须有极强的畅通性，就是它必须是不能受到任何阻力的。

上位者为了保持权力的畅通性，就要采用诸如奖励、惩罚、资源掌控、信息垄断等手段。

这也是为什么权力命令后，不管你是不是能够完成任务，态度上不能讨价还价。必须先行接受，保证畅通（彭不理解不执行，刘副不理解先执行）。

3. 扩张性（厌恶真空）。这就是权力的触角会越来越大。如果权力不扩张，就会形成一潭死水。如果你谁都不信任，那么你的资源也无法利益最大化（信任彭，让其去打朝鲜）。

当然，这个扩张必然得依靠信任和心腹来扩张，不信任的人，你也无法给他倾斜资源，如果把资源给你不信任的人，这不是扩张，这是自取灭亡（某些下位者开始如绍伊古一般膨胀，权力要均衡）。

所以，各位如果35岁以上，还想求比较有价值职位的话，基本上留给你的道路就是内推。

内推的意思很简单，就三个关键要素：

1. 你得行（对权力有价值）
2. 得有人说你行（权力信任你）
3. 说你行的人，他得行（他掌握权力）

话说这三件事一定要聚合起来，你才能上位，才能成事。也许你真行，问题是，你已经超过35了，面基层岗位，那个小主管（比你还年轻很多），他不肯收你。他一看你的年龄就直觉上认为你不好指挥，就不愿意收你了。

而更上面的权力（能给你一个比较高的职位的人），他评判一个人的方式是非常简单粗暴的，就看有没有人推荐你（毕竟你来了，他是要授权给你的，话说一个不值得信任的人，他怎么分权给你？）。
