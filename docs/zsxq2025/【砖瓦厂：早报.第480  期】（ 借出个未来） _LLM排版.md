利率的数字，整体反应的是经济增长率。但低利率时代，虽然借钱成本降低了，但资金的成本，与恢复经济活力的难易程度，关联度很低。

我在478期早报里提到过：

印钱、借钱来花，是用政府信用担保，把个地区富人的借钱，借给政府，政府再去分配。这个过程中：

- 富人失去现金，拥有债权
- 政府得到现金，拥有债务

然后拿这些现金去做二次、三次分配。大多数时候，长期利率的趋势，应该稍稍滞后于实际经济增长率。

为了应对经济低迷印钱是很正常的，但不能忽略：没有无成本的现金流。

我们想象中的无抵押印钞，如无国债做抵押，来印钱。实际上，也是拿铸币权拥有者的信用做抵押。

在这个过程中：

- 印钱方（如政府）
  - 失去的是：铸币权的信用的等级
  - 得到的是：现金
要是是有抵押，如通过长期国债来抵押，而印钱

在这个过程中：

- 印钱方付出的代价是：成为债务方。
- 购买债券的人，成为了债权方。
债务方短期内，会得到现金，需要长期支付利息。
债权方短期内失去现金，可能长期收获利息。

无论是哪个过程，全都是：政府用自身都信用，向市场上有钱的大户（也可以是外国政府）借贷。

资金的留向是：

- 短期 从 大户到政府
- 长期 从政府到大户

与普通人的关系是：

政府拿到现金后，现金向蜂糖样流动，粘到哪里，哪里收益（可能会惠及部分普通人）。而没有粘到的地方财富对应着比例被稀释（短期会对大部分普通人产生负面影响）。

藉此，要是能做大蛋糕，就是另个话题了。但目前这个领域关于这个话题的讨论，全都建立在做大蛋糕的比例，是＞债务的成本的。

九十年代，朱相执政时，国债利率多次突破过10%。意味着：中夹政府像市场里的大户借钱时，成本＞10%。同时意味着：市场里的穷人的财富，是以x%缩水的。x取决于债务的规模。

当初朱相借的钱于什么去了呢？用来购买长三角、珠三角的产能，投资中西部的基建。以及，给四大国有行等注资，改善其资本结构，在高盛大摩的保荐下，用于其后续的赴港上市。坏账则留在了长城、东方、信达、华荣身上（其实就是财政部身上，也就是每个人身上）。

这个过程中

利好的在：四大行们、长三角、珠三角的工厂。后来，他们前后接住了：上市的机会，中国加入WTO的机会。做出了远超当时成本的蛋糕增量。

当然，核心还是加入WTO。没有加入WTO，则当初通过国债保下来的产能，很可能会变成全国人民身上的蚂蝗。

所以，我们回过头看，所谓的印钱，就是以政府信用、普通人的为代价，来向拥有着绝大部分存款的top5%大群借钱（想想，当年能买国债的，哪个不是富户）。

然后去搏，去挺。去赌财政增速（不是经济增速）与资金成本，谁大谁小。赌赢了，就是汉江奇迹。赌输了……可能是社会动荡，如80年代的洋跃进。继而，通货膨胀（或紧缩），政府信用降低，社会财富更加集中，或向海外流失。

所以，运气很重要，外界环境会高度影响内部政策的执行力度。

显然1980年代后的三十年，是我们民族少有的——既能够师夷长技，还能够相对独立自主的阶段。

时光去不复返，新的时代，原有的秩序解体。如何应对，是全新的话题了。而国家的运气，又叫国运。
