
# 《中庸》第19讲：《致曲章》、《前知章》和《自成章》

## 第23章《致曲章》

“其次致曲，曲能有诚。诚则形，形则著,著则明，明则动，动则变，变则化。唯天下至诚为能化。”

这段话阐述了在完成诚明之后，人应当如何实践和验证中庸之道。子思所说的“致曲”，并非指人无法直接接触天道，而是强调在具体的现实中去实践和体验中庸之道。通过“致曲”，在复杂的现实中去躬身入局，才能证明自己的诚明和成就。

“曲能有诚”，意味着在伟大的实践中才能证明自己的诚明。诚明之后，必须回到现实中去，通过实践来验证和体现中庸之道。“诚则形”，在伟大的实践中，掌握的中庸之道才能显现出效果。“形则著”，效果显现之后，会形成一种样式、一种规范、一种模式。“明则动”，当效果显著时，会形成对社会广谱性的号召，大家跟随行动，朝着一个方向努力。“动则变”，整个社会发生变革，形成新的潮流。“变则化”，集合天地之力，化育共生，创造新的生命、新的国家、新的时代。

## 第24章《前知章》

“至诚之道，可以前知。国家将兴，必有祯祥；国家将亡，必有妖孽。见乎蓍龟，动乎四体。祸福将至，善，必先知之；不善，必先知之。故至诚如神。”

掌握中庸之道、至诚之道的人，可以前知未来。国家将兴，必有忠贞之士显现，国家将亡，必有邪恶之士得势。“至诚之道，可以前知”，掌握中庸之道的人，可以提前预见未来。“见乎蓍龟，动乎四体”，指的是在文书、立法、行政、财经和军队中能看到忠贞之士或邪恶之士的行为。

“唯天下至诚为能化”，至诚的人可以化育共生，创造新的生命、新的国家、新的时代。

## 第25章《自成章》

“诚者，自成也；而道，自道也。诚者，物之终始，不诚无物，是故君子诚之为贵。诚者，非自成己而已也，所以成物也。成己，仁也；成物，知也。性之德也，合外内之道也。故时措之宜也。”

“诚者，自成也”，掌握天道、中庸之道的人，需要通过实践来证明自己。“而道，自道也”，中庸之道必须通过实践来完成。实践是检验真理的唯一标准，是为“自成，自道也”。

通过以上三章的学习，我们可以看到中庸之道不仅是一种理论，更是一种实践哲学。掌握中庸之道的人，需要在实践中不断锤炼自己，同时也要服务于人民，化育共生，创造新的时代。
