
# 米联储见闻20250116——长吁莫错还闭关

尹锡悦终于被抓起来，这是韩国警方第二次发起逮捕，也是韩国宪政历史上首位被逮捕的总统。曾经作为最高检查长呼风唤雨的尹锡悦，最终还是逃不开青瓦台魔咒。这次抓捕是1月3日韩国高级公职人员犯罪调查处对尹锡悦实施逮捕行动失败后的第二次尝试，当时双方对峙5个多小时之后以失败告终。为什么布林肯亲自来给尹锡悦撑腰以后，反而逮捕尹锡悦的行动成功了呢？

从行动细节上来看，上次逮捕发生在1月3日，而签发逮捕令的时间是2024年12月31日，签发逮捕令两天后实施逮捕给了尹锡悦一方充分的时间进行人员动员和准备。逮捕时间是上午七、八点开始，在白天进行，给了尹锡悦支持者阻碍逮捕的便利时机。而此次是1月15日在清晨实施逮捕。从时间上来说，实施逮捕令出其不意攻其不备，且凌晨实施逮捕，尹锡悦一方没有进行更充分的群众动员，现场组成人墙护卫尹锡悦的主要是尹锡悦律师团和国民力量党国会议员。现场阻挡执法人员进入官邸的尹锡悦支持者并不多，等天亮之后支持尹锡悦的示威者达到六千多人，但仅限于在总统官邸表达立场，对逮捕行动不构成影响。此外，公调处1月3日首次实施逮捕令派出的公职人员150人，包括公调处相关人士30人和警方特别侦查团120人。而总统警卫处共有700多名工作人员，负责韩国总统官邸警卫的是陆军首都防卫司令部第55警卫队，当时组成人墙阻挡逮捕的人数是实施逮捕人员的两倍以上，当时的情况下，公调人员不是警卫处的对手。经过第一次逮捕失败之后，公调处吸取教训，联合警察厅国家侦查本部、国防部调查本部三方出动。将逮捕任务交给了韩国警察厅，此次执行逮捕任务时，警方出动上千人。从警方人员规模来说比总统警卫处的人员多。这种实力上的压倒性对比是可以成功实施逮捕的根本原因。由于有国防部调查本部出动，14日晚些时候在收到执行尹锡悦拘留令的公文时，第55警卫队回覆了批准进入，相当于打开了官邸外围阻截线。第一次实施逮捕时，总统警卫处精心部署，态度强硬。而此次实施逮捕令之前警方已经对总统警卫处处长、副处长进行了传唤。1月3日首次逮捕尹锡悦失败当天，韩国共同调查本部就对总统警卫处长朴钟俊、警卫次长以妨碍执行特殊公务的罪名立案调查。1月15日当天在实施逮捕尹锡悦之前，警方率先逮捕了代行总统警卫处处长职权的警卫处次长金成勋。也就是说，为了确保第二次逮捕行动成功，调查部采取了剪裙边的方式，把能够拱卫尹锡悦的各方力量全部都压制住，对其重用的得力警卫官员进行有针对性的打击，降低了警卫处誓死保护总统进行抵抗的决心。估计是考虑到大势所趋，尹锡悦非常清楚，如果坚决抵抗，将会发生流血事件，也不一定能够成功，抗争到最后一刻已没有意义，最后选择主动走出总统官邸保全最后的体面。

虽然尹锡悦被逮捕了，但是事情并没有就此结束了，也不能说美国方面的影响就彻底消失了，目前尹锡悦是以涉嫌内乱等罪名被拘，按程序，他抵达公调处后，将接受当局有关短暂实施戒严一事的讯问。有官员透露，此次审讯由首席资讯长副局长李在承主持，尹锡悦的法律代表也在场。根据韩国法律规定，公调处在逮捕尹锡悦后，只能拘留他48小时进行侦讯，之后必须决定是否申请拘留令，延长拘留期限最长20天，否则就要放人。宪法法院表示，正在审查尹锡悦律师所提诉求。这次尹锡悦被拘留与其宣布戒严直接相关，但刑事叛乱调查与弹劾程序是分开的。在尹锡悦执政期间，韩国的社会面貌也是肉眼可见的在衰退，2024年，韩国失业率升至三年多来的最高水平，韩国12月经季节性调整后的失业率从上个月的2.7%升至3.7%，创下2021年6月以来最高水平。此外，韩国就业人数比往年同期减少5.2万人，为2021年2月以来首次萎缩，就业市场恶化之际，韩国消费者和企业信心出现自新冠疫情暴发以来的最大跌幅。戒严令事件使韩国陷入动荡，导致股市暴跌，去年在全亚洲的货币中表现最差的韩元也再次遭受打击。从目前的局势来看，川普上台后对韩态度可能直接决定了尹锡悦的下场，以民主党退场的方式来看，这次布林肯在日韩可以做出的承诺非常有限，以至于日韩国内政治都出现了一定程度的转体，尤其是韩国，连基本的体面都保不住。

美国方面，虽然洛杉矶大火和TikTok难民席卷了美国的舆论圈，但主流官方媒体最近关注的焦点是，为什么在美国连续围追堵截之下，中国去年的贸易顺差依然高达7万亿人民币，美国主流媒体几乎发出天问，包括扎克伯格再次呼吁美国政府应该继续加码对中国的限制，被美国如此打压还收获高达7万亿人民币的顺差，这让美国确实有点内心崩溃。

根据美国媒体统计，即使按通货膨胀的情况做出调整，中国去年的贸易顺差远超包括德国、日本或美国等出口大国在内的任何国家在过去一个世纪里达到的水平。自从美国在第二次世界大战后一度主导全球制造业以来，中国的工厂正在以任何国家未曾有过的规模占据全球制造业的主导地位。来自中国工厂的商品大量涌入全球市场，逼得欧盟提出要求，任何加班生产的中国产品都不许进入欧洲市场。美国呼吁全世界工业化国家和发展中国家都加征关税，减慢中国商品涌入他们国家的速度。显然，美国已经意识到如果仅仅是美国或发达国家跟中国打关税战，显然是不够的，不但会增加发达国家发展的成本，可能会让发展中国家利用廉价的中国产品实现弯道超车，甚至在生活质量上逐渐赶超发达国家，所以当务之急，不仅仅是美国对华实施高关税，还应该动员全世界一起对中国实施高关税。

中国海关公布的数据显示，中国去年的出口货物达到25.45万亿元，进口18.39万亿元。由此产生的贸易顺差为7.06万亿元，打破了之前2022年5.86万亿的记录。尤其是12月的强劲出口，包括一些可能是赶在川普上台开始提高关税前匆忙向美国出口的商品，导致中国的单月贸易顺差创下历史新高，达到7529.1亿元。制成品贸易顺差已占中国经济总产值的10%。与之相比，美国对制成品贸易顺差的依赖在第一次世界大战初期达到峰值，也仅占美国生产总值的6%，当时，欧洲的工厂大都转向了战时生产，不再制造商品。中国出口从汽车到太阳能电池板等各种商品，为中国带来巨大经济利益，美国媒体认为，高规模的出口不仅为中国创造了数百万个就业岗位，也为高收入的工程师、设计师，以及从事研究的科学家创造了就业机会，中国工人的工资在过去十年增长了近一倍。
