
# 亚马逊逆向工作法：驱动增长飞轮

亚马逊的增长飞轮由两个互相嵌套的飞轮组成，分别包含四个要素：

第一个飞轮的四个要素是：客户体验、平台流量、卖家、选品。它们的关系是：客户体验越好，平台流量就越大；平台流量越高，就越能吸引卖家来开店；卖家越多，平台的选品丰富度就越高；选品越丰富，客户的体验就越好。这四个要素往复循环，驱动飞轮越滚越大，象征亚马逊零售业务的增长。

第二个飞轮的四个要素是：业务增长、降低成本结构、降低商品价格、客户体验。它们的关系是：业务越增长，越能降低亚马逊的成本结构；成本越低，商品价格也能越低；价格越低，客户体验越好；客户体验越好，业务增长越多。

亚马逊在20年前就意识到，降低成本和提升客户体验之间可以正相关，这是很先锋的。在当时，电商和传统零售业普遍认为好体验等于高成本，两者只能二选一。亚马逊总结出这个飞轮后，高管们都非常得意，想把它公之于众。但贝索斯却制止了，说这是亚马逊的“秘密武器”。

基于这个增长飞轮而进行的亚马逊业务回顾会，有一套严谨的会议流程：定义、测量、分析、改进和控制（DMAIC），也就是六西格玛精益管理流程的核心。

找准关键指标是驱动飞轮快速运转的关键。亚马逊早期就尝过粗放指标的苦头。例如，最初给选品团队选定的KPI指标是“新商品的页面创建数量”，但发现平台上的商品品类虽然增加了，但销量并没有增长。于是，亚马逊开始修改选品指标，逐步改进用户体验：

1. 首先关注新增商品的数量（新商品的页面创建数量）。
2. 然后关注新增商品是否切中用户需求（商品页面的用户访问量）。
3. 接着关注能满足有效需求的“有效供给”（库存商品跟商品页访问量的比值）。
4. 最后关注高效满足有效需求的“高效供给”（两天内可配送的库存商品跟商品页访问量的比值）。

最后这个指标，亚马逊规定的KPI是95%。这几次改指标，每一次都在逼着团队更贴近用户的真实需求，这正是亚马逊“客户至上”的具体体现。

总结来说，亚马逊用关键投入指标来驱动业务增长飞轮的运转。我们的人生也是一个系统，工作、健康、关系、成长互相影响、互相驱动。我们也可以找到自己的关键投入指标，比如阅读的深度、休息的质量、思考的频率、对他人的善意。当这些输入不断积累，我们生活飞轮的产出指标，比如成就、健康、收入、幸福，也能得到增长。

亚马逊的管理哲学不仅解释了它一家的成功，还证明了“理性”这种东西可以被制度化，增长不必依赖天才的灵感，也可以转向可复制、可优化的机制。随着互联网增速见顶，这套管理智慧会被更多企业借鉴过去，优化自身效率。
