
# 老是被正职打压，副职该怎么做才能破局拔升？

匿名用户 老A
A师您好，我是省国企总部中层副职，正职表面对我很支持，偶尔让我替他去找领导汇报工作，私下却打压，饭局、出差、一些利益处理带员工去都不带我，目前遇到的问题是，上会的议题是我带员工做的，思路上加了很多分，正职让员工上会汇报，不让我汇报，让我指导员工，如果汇报的不好，说我指导的不好，汇报的好，是员工能力好，我应该尽心指导员工把工作做好，还是表面指导不做提升呢？核心问题是副职被隐藏，帮正职干苦活累活辅导员工，但是没有表现的机会，副职应该尽心辅佐，还是干一半留一半，等机会上位呢？

首先，在我看来，饭局、出差不带你去，算不上打压的范畴，我若是他，我也不带副职去，带下属去，我想怎么使唤就怎么使唤，带你去，让你定个票，你估计都不太情愿。

就你问的这个问题，如果你觉得这个会规格比较高，比如单位领导们都参加，你也很想在这个会上露脸，你不妨直接给正职说：这个会我来发言吧，领导，我也得锻炼锻炼啊。如果他同意，皆大欢喜，如果他不同意，他也不好意思让你再辅导员工，你都拒了我了，你又让我培训员工，他怎么好意思！

想上位，不要把精力浪费在和中干正职的鸡毛蒜皮身上，多干点少干点，对你晋升一级都没太大帮助，干一半还是留一半，这些也都是无关痛痒的。在这个位置上如果你想尽快晋升正职，请务必一定必须把精力多多得放在单位一把手那里，围绕着 him 做工作！

老A，《职场晋升指南》作者。
