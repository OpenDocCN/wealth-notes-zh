
# 跟朋友聊美墨边境墙

突然觉得特朗普在墨西哥边境地区的墙，和长城的作用很类似。长城主要的用途就是经济封锁与贸易，文明6对此研究的很清楚。我家孩子玩习惯了黑神话，想玩文明六和三国志不会操作。明公是文明6的大师级玩家，这种情况有没有其他办法帮孩子进入这种游戏状态？孩子是小学二年级的男孩子。

其实美搞关税壁垒、大院高墙是打基础，长期坚持才能脱离绿纸依赖。通过某种方式，激起他的欲望，尤其是针对叛逆期的孩子，就要让他叛逆着来按照你的意图去做某些事儿。譬如美国对中国pua，很多时候，就是故意让中国觉得自己这么做是对的，是叛逆美国，结果做的却是符合了美国的利益。

有一种观点，长城倒逼松散的草原部落整合成为统一王国。所以科技壁垒，会不会也倒逼兔子科技突破了？等兔子破了鹰酱的科技“长城”，是不是兔子也得“美利坚化”了？英国扶持德国、大清，遏制法、俄；美国人扶持德国、国民政府，遏制法、俄。长城，让中央政府有了筹码，可以来制衡，跟谁开关做贸易，谁就能崛起，看谁不可控了，就掐断他的贸易。

海权国家与陆权国家天然的矛盾，大陆均势政策，离岸平衡。搁在上个世纪，就是军事禁运。你阿根廷想要打大英，发达国家直接对你搞军事禁运，你就趴窝了。李成梁在辽东也是这么玩了几十年。努尔哈赤就是他养的一条狗。明朝后期内部崩坏，外部的平衡也被打破，就缔造了女真的大团结。列强在中国也是，孙中山先后被日本、苏俄扶持，苏俄也扶持过冯玉祥。西方列强在中日不能团结上，有高度的共识，坚决不能让国民党和日本穿一条裤子。

到今天还是如此，各种离间和压制。他三亿多人要平衡其他大陆劳累命要防过劳死，离岸国家都是这个策略，日本肯定也不愿意中，韩完成统一。那就把日本统一成省。内化成省的成本很高，产业倾销地，原材料供应地，又不承担治理成本，是最优选择。是的，要不然美国几乎每年都在打仗，但是州却没增加。希望我们这代人能看到自己当列强的盛世。想想英国巅峰期统治世界100年甚至北爱尔兰都没同化，苏格兰离心离德。这套玩法西方他们念的经不行。这么推演下去，美国如果不能真赢下去，他们的国族认同恐怕也会有问题。
