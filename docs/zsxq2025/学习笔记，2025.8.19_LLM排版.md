
# 学习笔记，2025.8.19

1. 我经常给自己一种暗示，不管这个世界是不是越来越好，反正我们一定要越来越好，时刻寻找机会，不悲观。永远乐观。
2. 很多地产公司暴雷之前，做给内部人员，一定要尽快全身而退的离职，因为这里面烂账特别多，乱七八糟的事也多，不要让这个事炸到自己，自己经手的钱或物，一定要记账，时刻保护好自己。说白了，有很多违规操作。例如，一房多卖。一房多次抵押。还有股东之间扯皮，貌似贷款刚批下来，直接让股东把钱划走了。远离这个漩涡，因为这些违规操作会造成大舆情，大舆情一定会有人进去，你不能成为这个替罪羊。在爆雷之前，尽可能的辞职。
3. 我觉得，一定要随时止损。错了，就要回头。不内耗。
4. 县城人口是净流出的，现在的农村就是未来的县城，一说人口流出流入，我就想起了欧神，他总说重庆未来是中国第一城，因为人口最多……投资必须要有地理确定性，就是不能超出你的实控范围。为什么我选了xx市？因为我对xx市最熟悉，出海投资的前提是去发达国家，韩日、欧美，东南亚除了新加坡外，都是关门打狗的惯犯，绝对不能去。
5. 象棋天才王天一不是被抓了，也不影响他是中国象棋有史以来最牛逼的天才，他为什么能吊打所有人？本质就是算力，别人能看三步五步，他能看十步八步，所以大家怀疑他是软件作弊，因为他的下法像软件，有着人类无法达到的前瞻性，我经常在抖音上看几大棋王对掐，其他人的粉丝就骂王天一的粉丝，看吧，作弊坐牢去了，结果最近一公布名单，凡是能叫上名字的大师，一网打尽了。当一个运动，所有顶尖运动员都在违规时，就要思考，是不是模式出了问题？中国足球与中国象棋是不是只是缩影？到底根源是什么？是不是监管有问题？当时我做图书工程，参与竞标时，我跟我哥说，我就是想证明一个点，我以最阳光的姿态进入，不行贿，不内幕，我就看看能不能做成，事实证明，绝对没有任何可能。
6. 我也是这么觉得，我与他们接触的越多，越心疼，就是原本他们有个平稳的生活，铁饭碗，结果给了他们太多潜规则的机会，反而使他们进入了圈套。农民工相关的诉求，优先级永远是最高的，因为很容易形成大的舆情，进而点燃火药桶。
7. 我曾经问过高中老师一个问题，若是你做高考试卷，能考多少分？他说，150 分的题，能考120 分。剩余 30分呢？属于学霸专属。老师的极限就是教到 120分，120 分往上，都是学生自己的天赋，与老师无关了……
8. 做自媒体，写公众号，开始涨分慢，很难坚持，非常枯燥。其实，枯燥就对了，若是轻轻松松，那能出头的人不会是我们。什么是我们的天赋或 ENJOY？我们愿意主动去承受对应的痛苦！
9. 朋友和我联系，说是涨工资了，一个月涨了300 块钱，要请我喝羊汤，他很开心。我说，你那点工资，不要吃了喝了，要好好攒着，百多块钱，对我来说，没有任何感觉。对你老师，就是将近一天的工资。一顿饭就吃没了，没有必要，我替你心疼，对与大部分人而言，赚钱太难了。再说，跟我在一起，不需要礼尚往来。我出去吃饭，要么我买单，要么能不去就不去。
10. 对于写作，什么能写，什么不能写？凡是你所想，都可以写，你要训练自己的大脑，敢去想过去不敢想的事，无所禁忌，不要总是前怕狼后怕虎，你跟我聊天可以写，人们喜欢真实的东西，真实自有万钧之力，大部分人所谓的心里话，其实并不是心里话，他甚至都不知道真实的自己是什么样子，因为他从来没朝下探索过，你有机会可以关注一下木子美与李银河的文字，排除性的偏见后，你会发现，她们都是人间清醒系列……
