
# 对台政策的变化
## 发布时间：2025年10月24日

四中的十五五，本质是决定利益分配的，
我们可以用十五五和十四五的文字对比，
从港澳台与海外统战的利益盘再分配角度看：
十五五：促进香港、澳门长期繁荣稳定，推动两岸关系和平发展、推进祖国统一大业，推动构建人类命运共同体。
十四五：要保持香港、澳门长期繁荣稳定，推进两岸关系和平发展和祖国统一。要高举和平、发展、合作、共赢旗帜，积极营造良好外部环境，推动构建新型国际关系和人类命运共同体。

## 1. 港澳的长期繁荣稳定，从中性的保持，正向改为“促进”。
## 2. 两岸关系的和平发展，从中性的推进，正向改为“推动”。
## 3. 祖国统一，中性的“推进”不变，负向改动取消“大业”。
## 4. 人类命运共同体，维持不变。
## 5. 新型国际关系，大篇幅消失。

我们把这一段统筹来看，就是把整个盘子的资源和预算，从十四五重心用于“新型国际关系”的统战资源，十五五转向重点用于港澳台，尤其要加大对台湾的统战力度。
因此，新的吴石将军和馆长，乃至郑丽文都会得到更多的支持。
这跟特朗普的美国优先与战略收缩异曲同工，港澳台都会因为这一轮的利益调整而受益。
