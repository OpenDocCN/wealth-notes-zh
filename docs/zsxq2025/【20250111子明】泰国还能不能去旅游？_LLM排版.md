
# 泰国还能不能去旅游？

依靠外资或者旅游的国家，一定会竭力搞超国民待遇，这方面是不用怀疑的，但国家和地区经济的发展水平，也决定了治安水平，选择去物价低的地方，自然就要接受平均安全度降低。参考九十年代的中国，治安的确也不好，但是老外丢了个骑行车，公安局长帮你找。在东南亚体验和享受廉价的超国民待遇，自然就要承担少许的风险，这跟当年外资投资中国差不多。

对于统治集团来说，民众就是他们的资产，也是他们跟他国统治集团博弈和交换的筹码，用以赚取他国民众身上利益的新增利益。当美国的统治集团想从中国民众身上赚更多利益时，就会驱动美国民众反华情绪高涨，迫使在美国人民身上赚Dollar的中国统治集团让步，当中国的统治集团想从日本民众身上赚更多利益时，也会驱动中国民众反日情绪高涨，迫使在中国人民身上赚RMB的日本统治集团让步。

当美国的施压取得了胜利，美国的利益集团可以从中国民众兜里掏钱时，他们就会淡化反华的舆论，让中国的统治集团恢复从美国民众兜里掏钱的资格。当中国的施压取得了胜利，中国的利益集团可以从日本民众兜里掏钱时，他们就会淡化反日的舆论，让日本的统治集团恢复从中国民众兜里掏钱的资格。这样就会很好的理解，带着巨大利益去东南亚的中国游客时不时出安全消 息，是因为部分统治着他们的部分中国利益集团，正在扩大从东南亚的民众的兜里赚钱的规模，用威胁既得利益缩水的方式，迫使东南亚的统治集团出让东南亚民众身上的利益。
