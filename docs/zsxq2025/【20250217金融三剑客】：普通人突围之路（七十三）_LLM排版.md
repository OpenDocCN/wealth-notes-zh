
# 普通人突围之路（七十三）

## 一、心理

常凯申之所以被叫做运输大队长，不完全是他手下无能，很多时候，也是因为他自己非要靠前瞎指挥。你想啊，如果连前方一个阵地上的机枪怎么架，他都要亲自指挥，只能说明什么呢？说明他内心惶恐，他不做点什么就无法压抑自己此刻的不安情绪，那他就不是想打赢嘛，他只是想要让自己尽快好受点。

话说他好受了，他每次都成功的输出了自己的不安，对下属发泄了一通吼叫，结果呢？仗又打败了。如果一个人对生活的感受很细腻，他就会立即发现生活里没有傻子。只有先知先觉、后知后觉，以及知而装傻。一个人一旦发现自己被使坏了，他就一定会以某种方式给予反馈。

生活里，我们会觉得有些人莫名其妙地对我们好，有些人莫名其妙地对我们坏。其实，哪有那么多莫名其妙。只要你显微镜足够强，颗粒度足够细，你一定可以发现原因，社会就没有傻子。

很多人陷入困境，是因为他们解决不了某些在他们心中的障碍。是那些障碍让他们看起来弱势，并不是别人天然地比他们强。说白了，上述这种死要面子活受罪的心理其实是有深层次根源的，来哪呢？你耐心看完下面这个故事，就知道了。

## 二、剧情

白鹿原这部剧很多读者看过，上世代有两个大户，白家和鹿家，这一代的族长依然是白家人，白家的白嘉轩，与他同辈的是鹿家的鹿子霖。

白嘉轩的长子叫白孝文，已被老爹内定为下一任的族长，当下在祠堂里教娃娃们读书，读圣贤书。

白嘉轩作为白鹿原的族长，为什么安排儿子白孝文饱读圣贤书，在祠堂里教孩子们读书？这就是给他铺路，这就是最经典的继承人教育。

你以为的继承是什么？是跑男里面的继承人游戏？大家抢夺一份遗嘱？谁抢到了就像西虹市首富一样坐拥300亿？接下来只需要思考坐着花还是躺着花？

那东西没用的，白嘉轩的爹也是族长，你看过小说就知道那是不可能靠一句话就能完成的继承的。你想要真的说了算，除非大家愿意听你的，明白吧（很多老臣或者下属不服你二代，你能接个P）。

白嘉轩怎么干的？他先建祠堂，就是建立村里物理上的圣地；而后立碑，免费教村民读圣贤书，就是建立村民精神上的圣地。

话说物理上的，精神上的圣地都建立了之后，再安排自己的儿子白孝文做校长，那么下一代的村民，也就自然而然成了白孝文的学生（你们都是我教出来的，培养出来的）。天地君亲师，看懂孝文他爹的苦心了么？

然后白家开始给白孝文娶妻，是为了让他传宗接代（继续当族长）。但是白孝文迷上了那事儿，就是打扑克。

于是家人就教育他，要节制欲望啊。白孝文年轻人么，按捺不住的，和媳妇在他教书的地方，也就是祠堂里偷偷地打扑克，结果差点被别人撞破，受了惊吓，从此ED。然后他就迷上了住在村外的，黑娃的女人田小娥。

田小娥原本是外村的女人，被卖给武举人做妾。黑娃原是白嘉轩家里长工鹿三的儿子，跑去人家村里做麦客，与别人的妾偷情被发现，游街示众无处容身，遂逃回本村。

族长白嘉轩也不接纳他们俩口子，不许他们进村，于是他俩就只能住在村外的破窑洞里。后面黑娃就上山当了土匪。田小娥呢，一个女人家没吃没喝的，就做了鹿子霖的拼头。然而鹿子霖为了恶心白嘉轩一家，就指使田小娥去勾引白孝文。

神奇的是，白孝文在与田小娥偷情的过程中，发现自己居然又行了，从此一发而不可收拾，又迷上了这项运动。

鹿子霖后面就把这桩事情公开了，他想要看白嘉轩会如何徇私枉法，结果没想到，白嘉轩是个狠人，用带刺的荆条把亲儿子抽得皮开肉绽、遍体鳞伤、奄奄一息，逐出家门。从此，白家没这号人。

白孝文走投无路，只能选择和田小娥继续厮混在一起，打扑克，抽鸦片，直到大饥荒发生。

大饥荒下，没有吃的，白孝文问亲爹讨口吃的，白嘉轩也不给，宁愿看着儿子饿死。最后还是鹿子霖给了白孝文一口饭吃，他才有了力气逃荒去了西安城。

自从白孝文走后，前面上山当土匪的黑娃已经混成了扛把子，重新杀回了村里报仇，把白嘉轩的腰杆子都给打断了。

“你不是公正无私么？你不是要挺直了腰杆子做人么？你不是不让偷情的我和田小娥进祠堂么？那就打断你引以为傲的腰，让你从此佝偻着。”

后来黑娃人被抓了，要被枪毙，白嘉轩居然出来为他求情，救下他的命，并且告诫他要改头换面，重新做人。

再后来，白嘉轩的儿子白孝文在城里当了营长，混得风生水起的，白嘉轩又重新接纳了他，允许他认祖归宗，回归祠堂。

## 三、问题

故事很长，问题来了，族长白嘉轩嘴里所谓的公平正义的标准到底是什么？

如果说偷情的白孝文可以被看着饿死，可以被逐出祠堂，永远不许他回归。那么请问，做了土匪杀人越货的黑娃，为什么就可以被宽恕，甚至被救赎呢？何况，为什么黑化之后，作恶多端当了营长的白孝文，又可以重新回归祠堂了呢？

其实白嘉轩的原则至始至终都是一以贯之的，就俩字：面子。

黑娃不是自家的孩子，而且打伤过自己，他做得坏事越多，白嘉轩此时选择饶恕他，就越显得自己以德报怨，村民们就越是敬佩自己。

而白孝文是亲儿子，他处罚得越是绝情，村民们同样越是敬佩自己。至于后来孝文混出息了，那自然是可以回归祠堂的，因为给自己脸上贴金了嘛。

杀一人而三军震者，杀之；赏一人而万人说者，赏之。

你可以理解为，都是一种预期管理。

再一个问题就是白孝文先前因何ED？又为什么沉迷于田小娥而不能自拔呢？

很简单，他先前被迫饱读的那些儒学的冠冕堂皇的大道理和现实就始终对不上。他不是一个存天理、灭人欲的机器，他是个正常的男人。

那他爹白嘉轩为什么可以对上呢？因为他演的那套假道学，克己复礼摆出来的正直的样子是有回报的，是能够换来村民的尊敬，是能够坐稳族长位置的。

话说白嘉轩处在正反馈里面，他演得越多，越像影帝，就能得到越多的权力。

可是白孝文呢，他还不是族长啊，他处在负反馈里，他学着他爹的那个样子亦步亦趋，只有损失，没有奖励，于是他坚持不下去的。

他ED是因为他精神已经崩溃了，但他身体是很诚实的。简言之，心里全是假道学，自己的精神与身体成天打架，无法自洽（装逼没好处，受不了）。

其实这里面的逻辑很简单，责任，意味着欲望和压力。快乐，意味着他可以自洽。如果一个人在面临一个个选择时可以自洽不内耗，说明他把这些事都想清楚了。

把某一个具体的事想通，并不厉害。把很多事想通，意味着他在处理事情的指导方向上有了连续性和一致性。这意味着他有了完整的价值体系和思维工具。

未经世事的快乐，是珍贵的；历经世事的快乐，是智慧的。

白孝文跟田小娥偷情的时候是堕落，可是越堕落，越快乐，于是身心合一了，心里的束缚放下了，于是就又行了。

so，回到咱们前面的话题。作父母的，望子成龙没毛病，我也是当爹的，我也望子成龙。包括我们国人都非常认同的所谓人要奋斗，要上进，要出息，这个大道理对不对？对。

但是这个大道理背后的底层逻辑是什么？有人思考过么？

这个大道理，其实是站在集体视角下的决策逻辑，是集体视角下的最优解（你去看咱们的语文教材，里面非常清楚的写着杜绝宣扬个人自由主义思想）。

而且这方面的舆论管控也是很严厉的，走相关路线的大V如今也非常窘迫。比如最近爆出的一个瓜就很明显。两个在国内被屏蔽的自由派大V（实在混不上饭吃），组局在泰国清迈卖房，有15%的抽佣，结果项目踩雷了，价格贵、质量差、最要命的是房本下不来。

面对众多买家的维权，两个原先称兄道弟的大v先后甩锅，不愿意扛责任，最终闹到油管上节目撕逼。其中一个大v说自己经济情况紧张，连买房的100万都挤不出来，原因是自己老婆（后来离婚）炒股亏了890万，全家深陷债务泥潭。
