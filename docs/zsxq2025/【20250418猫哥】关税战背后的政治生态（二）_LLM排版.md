
# 关税战背后的政治生态（二）
# 猫哥原创

我们继续来谈谈美国的政治生态。美国总统大选是这样，我们假如某个州有1000万选民，有5张选举人票，只要一个总统候选人在这个州获得选民票的简单多数，那么就能获得这个州的全部选举人票。这里有一个容易被忽略的关键，这就是美国每个州的选举人票是对人口密切相关的。这个选举人票的数量不仅能影响总统大选，而且还决定了这个州拥有的众议院席位。

美国人民没有户口限制，是可以自由迁移的，那么决定人口迁移的因素有那些呢？一个是政治因素。

比如我是一个女性，我希望实现打胎自由，那么我就应该迁移到红州区居住，因为红州支持打胎，而蓝州一般是禁止打胎的。比如我是一个男性，我喜欢狩猎，那么我就可以迁移到狩猎合法化的州，而不是禁止狩猎的州等等；以上就是政治因素，当然真正决定人员迁移主流的还是经济因素——那个地方就业机会多，收入高，就必然吸引更多的人员向这个地方涌动。

从这个角度看，美国设计的选举人票数与人口数量挂钩的制度还是很科学的，一个地方经济搞得好，吸引人口多，那么选举人票与众议院议员的席位就更多，对美国政治的影响力就越大。

我们了解了以上基本信息之后，就可以发现，以上政治制度设计对于美国两党哪一个党更有利？毫无疑问当然是民主党！

原因很简单，民主党的基本盘在城市，在沿海，在经济发达地区，正常情况下，中西部地区、农村是没法与沿海地区与大城市拼经济的。但是民主党基本盘与共和党偏偏就拼个旗鼓相当！2024年美国大选，民主党在掌握行政机构的情况下甚至还惨败给特朗普。

为什么？原因无它，民主党实在是太拉胯。明明优势在我，民主党就可以将一副好牌打得稀烂。拜登执政四年，仅仅因为一个港口淤堵居然束手无策，眼睁睁看着港口淤堵演化成席卷全国的大通胀。其无能可见一斑。

美联储开始暴力加息去控制通胀的时候，拜登政府毫无作为，本来手里还捏着一副好牌——拜登完全可以将特朗普对华产品加征的关税降下来——当时高盛就做过测算，对华产品关税降下来，可以立即将通胀降低1个百分点，配合美联储加息，效果应该更好。但是拜登偏偏瞻前顾后，顾忌这个顾忌那个，与中方谈判了几轮因为开价太高，最后无疾而终，眼睁睁看着国内通胀飙升，最后民怨沸腾，支持率一路下降。

不仅如此，民主党对自己基本盘也不友好。先是巴以冲突后，组织警察去学校抓捕反以学生，得罪了最有行动能力的青年激进左翼。再是在经济上一堆首鼠两端的政策，导致自己最大的金主，硅谷的“人民富豪”们纷纷跳船，马斯克不用说，扎克伯格、贝佐斯也都投机了共和党更不要说奥巴马、佩洛西背刺拜登把高层的矛盾公开化，严重动摇了本来就不稳固的支持层。选举事后哈里斯就是除了no trump外，什么政策都拿不出来。

而特朗普与拜登相比就强多了，一个是特朗普很善于表演，表达也很接地气，更为重要的是，特朗普很有点光棍精神，想明白的事情就敢于下决心去干，管他以后有什么后果，比如这次冒天下之大不韪对全世界挑起关税战，就颇有点干了再说——我不管未来是不是洪水滔天，其执行力绝不是拜登与民主党能相比的。

说实话，遇到特朗普这样的流氓当美国总统任谁也觉得头大。拜登任上多少还要讲一点规则，达成协议也能遵守，而特朗普呢？就是只要你让步他就觉得还有更大的油水可以砸出来。

2018年因为中兴滑跪，然后特朗普后来对中方开出什么条件？要求兔子所有国企如同中兴一样由美方派出监管层；华为要拆分为6个公司，由美国公司收购；兔子高科技公司要在美国上市，美国要控股；兔子放弃兔子制造2025，放弃产业升级。说实话，这样的条件就算是李鸿章在也是不敢签的。

这一次特朗普搞了一个海湖庄园协议，其核心就是要全世界分担美国36万亿的债务，其中给兔子就分了15万亿美元，分担的方式就是购买美国百年国债，这个国债没有利息每年还要给美方交纳2%管理费。特朗普现在天天表演，或者表态很愿意与中方达成关税协议，或者表态在等中方电话，对此中方开出谈判的条件是：

其1，美方必须约束内阁成员发布诋毁性言论，比如你不能一边骂兔子人是乡巴佬一边要我们来谈判。

其2，美方必须任命具备权威性的谈判负责人，意思就是你不能今天达成协议了，明天又反悔。

其3，谈判需基于平等与互惠原则，意思就是你如果提出不合理的条款就别谈了（具体参考上面特朗普提出的条件）

其4，美方需率先降低关税表达诚意，关税战是你先挑起的，你得表现有解决问题的诚意，没有诚意也别谈了。

中方能提出这样的谈判条件，确实是过去与特朗普打交道倒尽了胃口，与拜登打交道就不大可能提出上述谈判前提条件。

说实话，现在中美两个全球最大的经济体互相加100%以上的关税是非常罕见的，也是很不正常的。如果这种高关税长期维持下去，战争的可能性会越来越大，当然，现阶段还问题不大。

从这一点来看，特朗普是极其不负责任的，而美国对此居然毫无约束力，从这一点来看，美国的政治生态确实出现了重大问题。

当然，特朗普这样折腾不仅仅是中方难受，美方自己的经济也绝不会好过，我们拭目以待。
