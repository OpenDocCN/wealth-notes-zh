
# 聊几个事

## 重庆胖猫、春风法官和珠海杀人的对比

重庆胖猫事件引发男女对立，官方下场拉偏架，但315时还有人翻旧账，藐视群众智商。春风法官事件中，群众已分析出女法官不公，官方却给予死后哀荣，未整顿司法系统。珠海杀人事件中，凶手是坏人，因司法不公报复杀人，导致几十人死亡，北京成立专项小组清查珠海公检法，逐件审查卷宗。珠海司法口人人自危。

这些事件对比强烈。胖猫自杀，老汉杀春风法官，官方未整顿，珠海事件后整个司法口被整顿。个人不支持极端行为，但官方做法似乎鼓励报复社会。扁鹊说“上医治未病”，应在自杀事件时就整顿，而非等更多人效仿珠海做法。

对胖猫事件，不应一年后飞龙骑脸，而应拨乱反正，清查重庆相关方面。对春风法官事件，不应快速给予高级哀荣，而应趁机清查当地司法口历史卷宗，修改撞车赔偿法律法规，减少法官自由裁量权，从根源上杜绝类似事件。

## 方星海被抓

方星海因金融内鬼传闻被抓，他设计了许多金融做空工具，给股市埋下制度性漏洞，负责对接美股美方，谈判审核公司底稿。他对内混乱，对外有功，但美国开启审查后，谈判进展顺利。方星海是金融亲美派，若安然落地，能证明其非内鬼，现在出事，则性质改变，对股市短期震荡，中长期利好。

鹤真人可能出面保平安。鹤真人目前有两个麻烦，儿子和方星海可能攀咬他，若连鹤都保不住，情况严重。

## 李嘉诚的事

传闻李嘉诚暂缓出售美国港口，可能是彻底取消或暂缓。中国多管齐下，至少能暂缓出售。李嘉诚若不做叛徒，就得一条路走到黑，彻底取消出售意味着得罪英美，断送财富大头。李嘉诚若死，两个儿子可分头押注中外，大儿子继续押注国外，二儿子投资中国换取政治背书。

## 伊朗可能逃过一劫

情报显示，懂王派B2轰炸机去中东，干胡赛或伊朗。胡赛被干，但B2轰炸机未回美国，伊朗松了半口气。内塔急切，懂王不想干，内塔可能给懂王上压力，逼懂王硬解伊核。
