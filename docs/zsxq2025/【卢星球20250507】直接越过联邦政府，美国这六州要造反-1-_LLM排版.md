
# 直接越过联邦政府，美国这六州要造反？

昨天，社交媒体上出现了一张声明，大意是美国马萨诸塞州、缅因州、纽约州、康涅狄格州、罗得岛州、佛蒙特州六个州的州长，向加拿大六个省的省长发出邀请，拟在波士顿举行会谈，讨论美国关税政策的影响，以及美国和加拿大如何继续合作维持强劲的贸易关系。这张声明带了6个州政府徽章，且有6位州长签字，看起来像我们的红头文件那么正式。有网友说，如果是真的，这简直可以看作是美国版的东南互保协议，影响很大。

东南互保是清朝末年(1900年）在义和团运动期间，中国东南沿海和长江流域的各省督抚与西方列强达成的一项地方性协议。

看看地图就知道了，这六个州，位于美国东北角，属于美国新英格兰地区（缅因州、马萨诸塞州、康涅狄格州、罗德岛州、佛蒙特州）和纽约州，地理位置上和加拿大经济最发达、人口最稠密的地区毗邻。历史上，新英格兰和加拿大东部（如新斯科舍省、魁北克省）在殖民时期同属英国殖民地，共享相似的文化根源。即便在1776年美国独立后，许多家庭仍保持跨境联系。至今缅因州约20％的人口有法裔背景，大多数都是从加拿大魁北克移民来的。

经济上，两者联系就更紧密了。新英格兰依赖加拿大的能源供应，尤其是魁北克省的水电。例如，马萨诸塞州和康涅狄格州通过输电线路从魁北克进口电力，2023年魁北克水电占新英格兰电力供应的10%-15%。缅因州的林业和造纸工业与加拿大的木材贸易密切，缅因州和加拿大大西洋省份共享渔业资源（如龙虾、大西洋鳕鱼），两地常合作管理渔业资源。新英格兰的滑雪胜地（如佛蒙特的斯托滑雪场）和加拿大游客市场关系密切，魁北克居民常前往新英格兰旅游，反之亦然。新英格兰的小型制造业（如马萨诸塞州的医疗设备、康涅狄格州的航空部件）大有供应链联系。2023年，马萨诸塞州对加拿大出口额约为60亿美元。

有人会问，五大湖地区的密歇根州、印第安纳州和威斯康星州，以及西雅图所在的华盛顿州，同样和加拿大接壤，为什么没有加入此次的东南互保呢？

恭喜你发现了盲点。因为密歇根州、印第安纳州和威斯康星州这几个州虽然也和加拿大接壤，虽然也依赖加拿大的能源和供应链，但是这几个州的产业主要客户都在美国内地，所以对加拿大的依赖没有新英格兰地区那么严重。更重要的是，这几个州不像新英格兰地区一样有共同的历史和文化背景，不太容易共同发起行动。

不过华盛顿州的情况有些不同，西雅图(Seattle)所在的华盛顿州大西雅图地区（包括金县、贝尔维尤、雷德蒙德等地）拥有多元化的经济，以科技、航空航天、零售和医疗等领域为主。西雅图被认为是全球第四大创业中心（领先于纽约和波士顿），科技公司数量超过6000家。另外，西雅图有全球最大的波音飞机生产基地。亚马逊、微软、星巴克、好市多等知名企业的总部都在西雅图。

所以华盛顿州在经济上和加拿大联系不大，对这事没兴趣也就顺理成章了。

那么，美国东北部的六州发起的东南互保能成功吗？从法理上来说，《美国宪法》(1789年生效）明确规定，联邦政府拥有独家管理对外贸易和征收关税的权力（《宪法》第1条第8款）。禁止独立设置关税或干涉国际贸易。历史上看，美国地方各州确实发生过几次税收反面造反，但都时间不长，且被联邦政府解决了。

第一次，是邦联时期(1781-1789),美国尚未形成强有力的联邦政府，各州拥有较大自主权。部分州（如纽约、宾夕法尼亚）曾对进口商品征收自己的港口费用或州级进口税，以增加财政收入。这些措施有时被视为准关税，但严格来说，它们更像是地方税收，且主要针对州际贸易或港口使用。例如，纽约在1780年代对从邻州（如新泽西、康涅狄格）进口的商品征收费用，引发争议。这种行为导致州际贸易摩擦，是促成1787年制宪会议的重要原因之一。这些州级税收并非真正意义上的国际关税，因为它们不直接针对外国进口商品，且在宪法生效后被禁止。

第二次，南北战争前的废除法危机(Nulification Crisis)(1832-1833), 1828年和1832年的联邦关税法案（被称为可憎关税,Tariff of Abominations）被南方州认为偏袒北方工业州，损害南方农业利益。南卡罗来纳州通过废除法(Nulification Ordinance)，宣布联邦关税在其州内无效，并威胁脱离联邦。

政府会做什么反应呢？我想，美国联邦政府大概率有这几步棋可以走。1、法律诉讼。联邦政府（通过司法部）或受影响的州、企业可向联邦法院提起诉讼，请求宣布州级关税违宪并颁布禁止令。法院几乎肯定会裁定州级关税无效，贵令州政府停止实施。2、行政干预。比如联邦政府可能通过扣减对该州的联邦拨款（如交通、医疗或教育资金）施压。联邦拨款是州预算的重要来源，扣减会造成显著经济压力；联邦海关与边境保护局(CBP)负责执行关税政策，可能直接拒绝执行州的关税要求，确保联邦法律优先。如果州级关税影响国际贸易，国务院可能介入，防止外国政府报复或提起国际诉讼。3、极端措施。如果州拒绝遵守法院裁决或继续对抗，联邦政府可能采取更强硬措施。比如动用联邦机构（如FBI或国土安全部）确保州级关税不被执行。在极端情况下总统可能根据《叛乱法》(Insurrection Act)将该州国民警卫队置于联邦控制之下，以维持联邦法律的执行（类似1957年阿肯色州种族隔离危机）。联邦政府也可能限制该州参与联邦贸易项目或港口运营，迫使其屈服。
