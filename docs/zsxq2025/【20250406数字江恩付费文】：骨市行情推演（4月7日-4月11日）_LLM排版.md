
# 骨市行情推演（4月7日-4月11日）

受对等关税的影响，全世界的股市都在跌，美股更是大跌。周末人心惶惶，担心A股周一大跌。从常理上来讲，如果周一开盘前没有利好对冲的情况下，周一大跌难免，但A股存在一种神秘力量，往往会出来护盘。股市有其自身的规律，利空只会影响短期走势，中期走势并不会改变运行趋势。

## 中期走势：
年预测里提到4月是容易发生大事件的月份，一般情况下，大事件往往伴随着大机会。所以，在年预测里，4月是逆转向上的月份，是今年股市里一个好的介入点，然后走一波行情。时间周期里，4月上半月有3个重要的时间点，分别是3、7、11。3号已经过去，发生了对等关税这件大事，剩下的2个时间点都在下周。下周一是7号，注定走势不简单。

在月度预测里，4月7日变盘向下走一波回调走势，但现在发生了对等关税这个大事件，在护盘力量参与的情况下，短期走势就变的略微复杂一点。

## 先分析4月7日大盘走势：
周末发生了大事件(反制)，低开难免，如果开盘前没有利好对冲，估计这个低开还不小。从的走势结构看，7号呈先跌后涨态势。预判大盘7日上午低开回落触底，下午反弹震荡。

7日注意事项：如果上午大盘表现强势，就要注意下午冲高回落。

## 下周股市涨跌节奏：
周一低开下探后反弹到周二，周三周四回落调整，周五触底后弹。注意事项：上面预判周一低开后的反弹可看做是主力强力护盘，如果发生主力护盘不力的情况下，大盘会在周三止跌，周四五回出现反弹，毕竟11号前后是个重要的时间点。
