
# 美股“黑色星期一”再现

周一盘前，美股指数期货暴跌，飞流直下的K线宛若一把锋利的尖刀切入了黄油，丝毫没有反手之力。执笔时，纳指期货跌了近6%,标普期货逼近4800,道指期货再次跌去逾千点。几日功夫，美股回到了2022年初的高点。那一年，美国股债双熊，其惨烈程度四十年之最，股债之间逆向相关性的消失，杀得所有的对冲策略片甲不留。

然而，三月以来的这次史诗级暴跌，已经可以载入史册。如果现在美股期货的走势在今晚美股开盘延续，那么我们现在目睹的美国股灾将是1929年大萧条以来最惨烈的连续三天暴跌，只有1987年的“黑色星期一”可以与现在堪比。1987年后，美股出现类似盘面的情况，是1998年长期资本轰然暴跌、2008年全球金融危机、以及2020年3月新冠疫情期间。

我们在去年底展望今年全球市场的时候，是全球市场里唯几强烈看空美股今年走势的声音。在去年十二月的一次峰会中，我与美国早负盛名的老行尊Ed Yardeni同台对于美股今年的走势做主题演讲。 Yardeni在80年代初对于美债利率见顶、美股开启长期牛市的惊人判断令其一战成名。他对于标普的目标价在7000点，为华尔街最高的目标价之一。与这样的的超级多头老行尊同台，并持有完全相反的、并与市场共识完全相悖的预测，我点压力可想而知。 Yaedeni 最近大幅下调了他的目标价，这就是一个优秀分析师的特征，绝不一边倒死多死空。而凯恩斯也曾说，“当事实改变了，我的意见也会随之而动”。

有时候，几十年都风平浪静；有时候，酝酿了几十年的风浪几周里就席卷了全球。

现在我们需要知道的是，市场价格是否已经计入了最坏的情景?以及美股史诗级的暴跌对于全球其它市场的影响。

特朗普在讨论关税实施的时候，只考虑到美国的贸易赤字平衡，却忽略了一些国际账户的基本概念。首先，美国并非对于所有国家都有贸易赤字。比如，澳洲、新加坡、英国对于美国都有贸易盈余。因此，这些国家惨遭额外关税令人费解。贸易平衡整体国家经常账户的一揽子平衡，而不是对于每一个国家的平衡。但特朗普的关税计算方法，却把与每一个国家的贸易独立核算，企图达到与每一个国家的平衡。这就是为什么,特朗普关税里如果公式计算结果不足10%,关税就按照10来计算。

在国家账户里，有一个复式记账法：经常账户和资本账户之和等于零。因此，经常账户的逆差会被资本账户的顺差平衡掉。简言之，美国人出口消费，但进口资本。这个主题，我早已在不同的场合与读者反复分享过。外国资本的流入，直接导致美元成为了全球的储备货币，商品定价的基础。外国人的买入，让美国股票市场享受了相对于其它市场极端的溢价。外国资本的流入也压低了美国资本市场的融资成本，让美国成为了全球资本市场的中心，并且能够以极低的成本为美国的财政缺口再融资，同时美国消费者也可以用远低于其它国家地区的价格买到物廉价美的消费品。在美国买过车的朋友想必都身有体会。这些，都是美元体系形成的基础。

如果现在根据特朗普得设计，美国的贸易逆差被削减到零，同时建立起各种贸易保护措施，那么资本流向也将发生变化。按照前述的经常账户和资本账户的平衡等式，美国很可能会因此失去全球资本市场中心这个别的国家梦寐以求的国际地位。美元体系也将由此土崩瓦解。在资本流向从美国转向其他国家之后，美国的资本市场必然大幅回落并跑输其它市场，美元将走贬，而美国经济也将由于如此严重的系统性冲击而进入衰退。这些，都是我们之前与专属读者分享的关于今年“东升西降”格局的逻辑。进群加微信pep458

当然，短期内，市场价格的波动反映的是市场内部结构的迅速恶化。在连续几天的史诗级暴跌之后，市场必然出现补好展要求。而现在的盘面显示的更多的是多头快速、但是还算是有序的撤离减仓，而不是交易商强平那些无法补好展的头寸。黄金由于今年以来的强势表现，也成为了短期补子展的工具。如果情况恶化，也不排除黄金可以直接做为好展抵押品的可能，而不是先要卖掉筹现金来补好展。

简言之，现在看到的行情仍然不能算是“强平行情”。这也表现在美国信用利差虽然已经开始飙升，但是也只是从历史低点回到了历史均值。距离前几次市场危机中出现的高点还是相差甚远。

中国果敢坚决的反制，将对于特朗普的游戏产生极限施压。我们在上篇讨论了这次博弈的尾部风险。这次博弈中，拖的时间越久，博弈的共同利益越小，但是美国能占的利益比例，也将迅速的减少。美股的暴跌已经产生了巨大的负面效应，并对于特朗普的博弈策略产生重大影响。刚才，媒体问特朗普如何看待美股的暴跌，他说“我并没有有意的制造股灾”。言语之间，带着一丝怯意。如果今晚盘前没有消息，那么市场对于美股的“黑色星期一”的担忧也可能自我实现。

从周五中概股盘面看，今天亚洲盘开盘将面临考验。因为在市场危机中，唯一上涨的，只有波动性。
