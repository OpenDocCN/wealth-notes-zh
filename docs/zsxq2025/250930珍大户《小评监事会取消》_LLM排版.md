
# 小评监事会取消

## 250930 珍大户
## 整理：公众号懒人搜索，懒人专属群独享

看到圈友转发了一条新闻，说工农中建交+邮储六大行都发公告，说已经收到核准，可以不设监事会，用外部监管和审计来替代。

这事有什么影响？

表层的影响，其实就是更加务实了。

我随手查了一下，工商银行有三名监事，一个是职工监事，另两个外部监事分别是人大和南开的经济类教授，意味着选人上比较重名誉。其薪酬没有披露，但绝对不可能是不花钱的。

农行，有三个职工代表监事，额外发5万块薪酬，以及三个外部监事，每个人30多万的薪酬，人物背景没披露就不好推测了。

中行，一职工5万+3外部每个近30万，这三个外部监事选择也挺有门道，我刚才一查才发现门道，里面两个是退休返聘的其他大行的领导，分别来自审计口，纪检建行也跟中行一样，三个教授，分别来自人大、央财、浙大，每人25—29万。

大家可以自己想象，如果你是教授，每年拿人30万左右的钱，需要出席4次会议，你能匀出额外多少时间去真的监管这个公司什么账目、财务情况吗？

大家内心都很清楚，人家公司要的是你身上光环，是不是你不重要，重要的是得需要你是某某大学的教授，基本上属于光环交换，互相撑一撑名誉，大行说某教授是我行监事，你在学校里说，自己是某大行监事，大概跟那个笑话，比尔盖茨女婿和世界银行副行长的笑话类似。

另外除了监事之外，各行还聘了很多独立董事，功能也类似，给钱，找各种有名誉的人，参加会议，名义上是一种额外监督作用。但很多担任独立董事的人，一个人身兼全球N家上市公司的独立董事的位置，每家每年飞过来开2—6次会议，同时兼任5家的话，意味着一年开10—30次会议，飞来飞去，真的对每家公司有多熟吗，就未必。反正全程有人安排，有人照顾的很好。

所以最基本的影响，砍掉监事会，反正每年也有监管机关管着，有大量的审计要审，监事基本起不到什么作用，一砍，各家公司都省了100万左右的支出。

但问题来源是，为什么这么务虚的一个东西，但当初要设？对小公司来说，没有全面审计，为了平衡股东之间利益，有权的股东日常当管理，没权的只出钱的股东就可以占个监事的名额，有个法定的审核和质疑的权利。

但对上市大公司来说，每年审计、监管，肯定比监事能做的事严多了，为什么还要引入外部董事和外部监视呢？这个我就比较印象深刻了，2000—2005年，是我从高中到刚进大学的时期，公司法修订也是2005年修的。那個年代最重点的讨论，就是反复质询国企不行，中资不行，各种论战，论证国企落后、制度不行、水平不行，中国企业眼光落后、水平没有国际牛，当年可以说是“公知横行的年代”，但反正回顾起来看，你也不能说人家错，毕竟骂一骂真把很多企业给骂的努力起来了，确实也起到了正面激励的作用。

当年企业超级喜欢引入国际大咨询公司来估咨询，付费改革公司管理制度。独立董事和外部监事，当年也没少聘公知，一般越敢喷的越有名，有名的，聘来当第三人就显得比较厉害了。

比如厉以宁，在我学生时代是很出名的人，被称为“厉股份”，观点就是各种国有制度都落后了，一定要搞股份制改革，股份制之后就灵了。当年他就担任过工行的独立董事还是监事我快记不清了。

所以现在再看监事都可以直接取消这件事，不止是省了几个人的人员开支的问题，更多的是现在开始有一种制度自信的感觉，不太担心外界的批评了。

这个算好事还是坏事我也不知道，正反两方面吧，（因为前几天，上海学生校园午餐的事，我看新闻看的就有一种感慨，如果不是在上海，其他地区都很怕闹出各种舆情，真有类似事件，会很快开始有内部整改，“自我批评”。而上海几乎算是中国最前沿的地方，其他省份都要学习的目标，所以明显感觉到，他们是不怎么怕有舆情的，也不怎么怕有外界批评，类似于过去二三十年的美国那种感觉，根本不怕批评，我也不改。太过自信之后就是开始的傲慢。这也算是真到制度自信之后的一种弊端，就怕自我革新能力会减弱。）

好的方面，就是制度自信，可以更加务实、不用太过务虚了。
