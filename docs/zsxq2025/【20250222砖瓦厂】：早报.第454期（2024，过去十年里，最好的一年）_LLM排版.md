
# 2024年，过去十年里最好的一年

2024年，我小舅一个人种了四五亩鱼腥草，还管理十几亩鱼塘。舅妈在县城工厂做流水线，舅妈还做农村水利建设的劳务，去年家庭毛收入有十多万。他说，要是再年轻十几二十岁就好了，很有信心，做一个包工头。

我姨爹讲，他前年鱼腥草收入最高，毛入是10万以上，加上稻谷的收入是2万。每年五一前，去做劳务，工资月结。十一后，继续做劳务，地点在家附近。高的时候，全年毛入在15万左右，低的时候，也有10万。他也发出同样的感叹，要是再年轻十岁就好了。

我爸讲，他2024年的收入，是他这辈子最高的。他打零工+种蔬菜，除开开销，年底去存了四万的定期。

我们小一辈的里面：我小堂哥换了X5，堂弟换了XC90，大堂哥把两辆大几十万的吊机贷款还完了。一个姐夫把高新企业的资质拿到了。一个妹夫（也是我战友），去新平台（地方国企）三年后，现在做了管理层，负责公司的专精特新业务。一个嫂子，进了某副厅级单位的领导班子。一个姐夫，去年年底拿到了被拖欠很久的运输款。一个朋友，去新疆种辣椒，价格没有2023年好，盈利不行，但是积累了很多种植经验。他告诉我，很有信心，在2025年打出漂亮的一仗。

这些是我这几天，见亲戚时遇到的、听说的。他们都是八辈子的贫农出身的人。我两个堂哥，一个堂弟，都是初中毕业（辍学）后去工地做劳务，慢慢拿高新技术企业资质的姐夫，是二十年前的国企下岗工人。现在已是处级领导的嫂子，是我们本地二流高中毕业。初中毕业时，家里穷，她妈不想让她去上高中，是他爸坚决支持她，她后来考上了一所211。现在想想，要是还有农村户口的话，能有一些地，平时种蔬菜+务工，纯收入和生活水平，比在城市做快递强多了。但是，有农村户口的80后90后，越来越少了。

我去吃了十几顿饭，整体特点是：1.在农村生活的60后70后，普遍脸上洋溢着幸福的笑容。2早年进入体制的大学毕业生，家庭最幸福。3.赶上了地产、基建大潮的一代包工头们，经济实力最强。4.80后90后在民企工作且失去农村户口的人，最沉默寡言。

我说一个极端乐观的事情大家听：我学长，农民，种红菜苔。三亩多地，去年年底物流快停时，红菜苔价格大跳水——低至1-2毛/斤，他硬是采摘后低价销售掉了。私下里说：没有浪费（用旋耕机打碎在地里），心里舒坦多了。

总之，春节我遇到的绝大部分人，都是开开心心喜气洋洋的。2024年，是过去十年中最好的一年。
