
# 猫哥5.4：公平与效率

最近协和4+4事件很火，今天给大家分享一段深度思考。鉴于这段深度思考可能引发评论区的不当言论，所以评论区暂时关闭几天。

董小姐的4+4引发公众的集体愤怒，但是如果年龄长的一点朋友就应该知道，类似的事情如果放到30年或者20年前都不算个事，也就是放在这几年才会引发公众的集体情绪，大家知道这是为什么嘛?

为什么公众情绪在10来年时间发生如此大的变化?

任何执政者在做选择的时候，都要面临一个重大选择：要公平还是要效率?

公平与效率如同一个硬币的两面，你翻开一面就一定要盖住另一面，期望兼顾是不可能的。

打个比方。

年轻人在选择工作单位的时候常常会有这样苦恼：国企还是民企? 国企注重公平，但是效率却低，进国企比较稳定但是却要忍受长时间的低工资。而民企注重效率，工资高但是年龄过了35岁就有可能被裁员。假如有人给你讲，有拿高薪并且稳定的国企或者永远不裁员的民企——那多半是杀猪盘。

建国以来，我国的主要路线在公平与效率之间也是多次摇摆。

计划经济是追求公平，但是效率确实太低。

小平同志提出白猫黑猫抓到老鼠就是好猫，这是一种典型追求效率并且以结果为导向的思路，是对过去计划经济的矫枉过正。

因为在计划经济时代，在抓老鼠前，光是该派黑猫还是白猫都要争论半天。

小平之后的江胡基本思路与小平都是一脉相承，不过说法变成以经济建设为中心，核心内涵还是以效率为主，其结果同样是硬币的两面：

一面是经济获得高速增长，三十年翻天覆地变化。

一面是行政机构腐败渐渐生态化常态化，老百姓办事走后门成为常态。

这就是二三十年前，董小姐这个事根本就不算李事的社会基础，细究起来，在那个时代，董小姐家庭甚至还可以说是清廉的，祖辈都是院士，父母这一辈居然也就混个国企校领导，孙女走个后门安排个医院工作算个啥?比这个过分的比比皆是。

要知道公务员逢进必考还是2006年公务员法颁布之后的事情了，在2006年之前，公务员都是人情安排。

转折点是2012年新时代之后。

新时代明显是左转开始注重公平。

实事求是的说，在新时代之后，在公平层面取得巨大的成就。

行政机构的腐败得到极大的抑制，现在的官员绝对是改开之后最清廉的。通过互联网技术，基层政府大部分权力还政于民，过去很多要求爹爹告奶奶才能从政府机构办下来的事情现在在手机上操作就可以完成。

城乡差距、地区差距肉眼可见的缩小，至少在基础设施方面已经极大的缩小。

几亿贫苦人群脱贫实现了两不愁三保障。

社会风气明显好转，整个社会道德水平大幅度提高。

当然，获得这些成就也不是没有代价。

这个代价，用权威人士的描述就是这十几年我们的经济基本是走型，之前十来年是走一竖，现在是走一横。现在可以回答为什么董小姐事情会引发如此大的公众情绪的原因了。

因为这十几年我们是以牺牲效率为代价在追求公平，所以稍微有点不公平的社会都难以容忍，同样也因为这十几年整个社会道德水平大幅度提高，公众眼里就容不得沙子。

写到这里可能会有人有疑问，有没有办法在追求效率的同时兼顾公平呢?这个很难。

历史上倒是真有一个政党给民众许诺可以兼顾公平与效率。

大家猜猜这个政党是谁?

这个政党就是纳粹党，党魁就是希特勒。

那么希特勒怎么兑现给民众的承诺呢?

希特勒的办法就是让德国年轻人穿上军装，用坦克与斯卡图武装起来，到国外去抢，用国外掠夺的财富在国内再次分配。

二战前几年的德国民众的生活水平是整个欧洲最高的，这也是苏德战争打了两年，德国居然还没有进行总动员的根本原因，一直到1943年1月，斯大林格勒战役失败后，东线出现一个巨大的豁口实在没办法才进行总动员的原因。

现在的特朗普就有点希特勒的意思。本来老美在过去全球化中受益是最大的，中国以制造业立国就赚点苦哈哈的加工费，设计、品牌、营销都掌握在老美手里，在产业链利益分配中拿走了绝对的大头，只不过老美内部分配机制不公平，导致红脖子没有分配到利益而已。

特朗普的思路就是通过关税对全世界讹诈一笔财富，然后给国内减税来分配。这种对外掠夺在实现国内的效率与公平注定是难以成功的，过去希特勒是如此，现在特朗普也是如此。

说说董小姐的事件的后续。

现在是卫健委介入，而不是纪委以及教育部介入，说明上面的意思就是董小姐事件就是董小姐为止。

没有扩大化的意思。

这个事要扩大化太难了，要涉及对整个医疗、教育体制甚至院士体制进行翻天覆地的变革，这么大的动作不亚于一场革命，治大国如烹小鲜特别是现在外部正在激烈博弈的时候，国内实在不宜搞太大的动静。
