
# 与朋友们聊马斯克

他在国外形象早崩塌了——这不就是当年的马总吗？没出事前都喊爸爸，出事成范小勤了。还有点不同在于，国内马云更多是被裹挟被动的，马斯克纯自找的。

马斯克为什么会从一个科技狂人走向深度参与美国政治？这里面到底是什么原因？马斯克是觉得自己不可能通过常规途径成为巴菲特比尔盖茨这样的教父级别大佬吗？实在想不明白，因为信仰。美国董事会成员没他位置了，就得想办法另辟蹊径，成为美国黄轩。共和党建制派也不能接受他吗？他的公司方向本来应该是民主党的基本盘，或者民主党圣位已经满了心，结果只能争一线天机了。

共和党建制派不接受他。子明：民主党是渐进式改革，马斯克等不及了，要颠覆性改革。彭斯是真正有理想和道德的标准共和党建制派人物。子明：民主党要的是性别种族平权，马斯克推动的是碳硅平权。所以他只能选川普这个非主流搏一搏了，把川皇博成主流。子明：中国革命不就是这样子的吗？工人阶级并没有选择和自己更接近的地主阶级和资本家阶级一起革命，而是选择了更保守的农民一起革命。但是川皇成了主流我怕第一个要宰的也是马斯克。老美的历史还是太短了。工人阶级不是跟农民一样底层+无产+被剥削吗？为什么会和地主资本阶级更接近？可能工人和厂长企业家离得更近？虽然地位差得远。

这个接近指的是相处的物理距离吗？我以为是社会属性就没懂。那时候的工人阶级跟现在的中产一样。Trump floats $5 million'gold card' as a route to US citizenship，明码标价。提问：工人阶级不是跟农民一样底层+无产+被剥削吗？为什么会和地主资……子明答复：革命的本质是生产力革命，是要代表先进生产力。子明：中国这种拉着落后生产力搞革命还能成功，在历史上也是一朵奇葩。是不是因为中国那时候就没有啥生产力，更谈不上先进？子明：还是当年太祖的阶级分析nb。有道理，太祖真的还是太牛逼了。太祖确实太牛逼。

懂王不也是拉着先进生产力和红脖子？中国这种拉着落后生产力搞革命，还能成功，在历史上也是一朵奇葩。马斯克“政府效率部”21名工作人员辞职！联合辞职信中指责"政治空想家"。先进生产力也是一个阶级，落后也是一个阶级，先进定个小目标，包装成先进阶级然后革命成功以后慢慢改造。子明：譬如，英国和德国的革命，地主阶级都是主力，尤其是德三，可以说就是个工人+地主容克的组合。
