
# 大佳笔记：2025-10-27，股票市场。（判断部分）

2025-10-27，股票市场。（判断部分）
1，受到中美会谈，以及亚太股市集体新高影响，今天情绪高涨。很多避险的资金重新进入市场，成交活跃，放量上涨。4000点临门一脚。
2，因为今天市场放量的。观望资金入场强烈。所以没有形成风格挤压。属于普涨。偏向大盘成长。此外，个股虽然普涨，但中位数上涨0.42%，不算太多。个股连续跑输指数。个股的高点在9月18日盘中。现在还有一定的距离。实际上，上证虽然新高了。但科创创业都没有新高。所以和上个月相比，实际上是大盘价值贡献的更多。
3，这一轮4000点更像是一个数字游戏，和以前的4000点并不一样。而且，这个4000是调控的结果。如果没有调控，上周五就已经突破了。因此，不要太在意点位。
4，除了中美关系利好之外，ai的映射继续。这一轮的映射，主要集中在硬件之上。此外，今天核电大涨，带动电力上涨。电力是表现最好的公共事业板块。其实美股电力走的也非常好。因为算力到最后，都是电力。美国缺电的现象很严重。
5，今天金融街论坛，信息量很大。我们晚上会继续细看。目前最大的消息就是央行重启买债，给国债市场注入流动性。国债苦尽甘来。看一下这波央行的调控目标在多少。我们自己的目标点位是10债1.6%-1.65%。
6，总之，就是不要太在意点位的问题。现在的4000，和10年前的4000已经完全不同。现在的4000只是一个象征意义。这是在高度控盘之下的一个必然。同时，也有大把的股票现在的位置在4000以下。4000点代表的是某队的控盘能力。在这个基础之上，最重要的信息就是上证牛本身。这意味着，市场不会出现流动性风险。所以可以经常保持很重的仓位。在交易上，不建议大家因为上了4000就减仓。自己的持仓本身的估值，要比指数的估值更加重要。
7，q3之后，我们继续加仓权益的战略不会改变。而且央行重启买债这个事情本身，让我们感觉到了q4政策边际的变化。
