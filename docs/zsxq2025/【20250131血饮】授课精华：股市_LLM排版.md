
# 授课精华：股市

目前美联储停止降息概率剧增，一旦1月28号美联储宣布停止降息，必然导致股市受到强烈情绪冲击。就像2023年10月美联储暂停暂停加息必然导致降息预期大涨，带动全球股市上行一样。到9月17号美联储降息落地，股市在情绪主导下进一步上涨一样。如果1月28号美联储停止降息，整个预期也会反转。从炒作加息预期到落地也需要时间。情绪的反转就始于2025年1月28号，也就是大后天的美联储停止降息预期。预期必然导致加息情绪再次抬头，这样股市上涨情绪就会逐步回落；加息预期抬头的同时，美股也即将开始调整，大概率会与跟特朗普上次执政时期一样，熔断暴跌约30%左右。港股大概率会跟随美股下跌，这样一来，外部冲击+港股通外资收缩，导致A股增量不足，年后大概率开门黑。

市场已经预计今年10月份美联储加息，如果继续降息，会导致通胀上行和就业增加，而就业增加，美联储就没理由降息，通胀抬头也会让美联储转为加息，目前的转向符合我之前的预判，也就是美联储降息是战术下蹲，加息将重返，有鉴于此，股市对应的交易策略也要全部改变了。如果1月28号的会议上，美联储强化加息预期，那么市场会随之暴跌，美股和港股下跌就会传入内地，一旦股市开起下跌模式，盈利下降预期下，资金会倾向于离场，割韭菜也会加速，这样股市就会面临全线下跌风险。

这种背景下，股市操作建议改如下：
1、节前降低仓位，或者清仓离场观望，超强势个股除外。新手和承受能力差的参考这个执行。
2、要买就集中买热点。因为，暴跌以后资金必然扎堆，要买就买强势热门股，不买热门概念后排甚至排不上的垃圾股。
3、暴跌预期下，炒股技术不行的就休息，避过加息预期带来的暴跌行情，待到连续暴跌以后再行入场。
4、全面降低交易频率，新手一年只交易一个月就已经足够。

近期股市的热点分别是算力、智能体、新能源汽车，文化传媒、华为、抖音。其中前两个会持续发酵。后面的课程我们讲一些股市最基础的问题和方法，到时候大家也能总结出符合自己性格的交易模式。今天讲课就先到这里。当然，面对外部加息预期冲击，央妈与财政部也在出台各种利好刺激股市，看证监会能否作出根本性改变，打赢这场金融世纪大决战了。大家离场后，可以密切审慎关注。面对外部冲击国家会发动逆周期调节，这个时候可以介入，具体我们看政策什么时候出。
