
# 卢麒元10.12：会员群聊

我于10月25号在上海安排一个见面会。这是向心坍缩提出后，应上海金融机构的朋友，还有一些企业家的要求，前去讨论一下操作细节。因为，按照规定，我不能在线上讨论具体内容。朋友们不用担心，宏观判断已经在此平台上说清楚了。并无遗漏，也非小灶，只是讨论一些细节和技术问题。

关于向心坍缩，时间竟然提前了整整十天。还好，大家仍然有足够的操作时间和空间。同时，此次引爆点，是州普的过度反应。川的性格，反复无常，仍然会有戏剧性的情况发生。不过，既然大势所趋，还是要做出系统性的安排。

我先说几句宏观形势：

美制造业回归，貌似轰轰烈烈，其实根本没有可能。百万工程师千万熟练工人，才是再工业化的前提条件。美十年内，根本无法再造。更扎心的是，美仍然在工业坍塌，飞机汽车等传统工重仍然在衰落之中。

美意图重构供应链，但川的团队完全不懂供应链重构策略。川的关税战，是供应链粉碎机。现在，仍然在脱钩断链，商业生态环境被破坏了。此次中方反击，再次击穿美供应链。

川团队，无真正的财政专家，金融杀手贝森特无法理解财政再平衡。大漂亮法案减收增支，再次重创联邦资产负债表。美元信用，岌岌可危了。

川团队，充满投机客(包括川本人和家庭)。他们在金融领域胡作非为，短视且颛预。短短数月，金融稳定被破坏殆尽。资本市场陷入危机的前夜。

四月份的“狼来了”，只是预演。时隔六个月，这次狼真的来了。并且，川服软，也没用了。唯，可以给没转身的明友一次机会。

朋友们要注意：此次危机与1929年不完全相同。此次危机，两件事同时发生。一是资产价格坍塌，二是货币重置。要注意，两件事的次序和节奏。次序和节奏，意味着机遇。转身的时间和方向很重要。
