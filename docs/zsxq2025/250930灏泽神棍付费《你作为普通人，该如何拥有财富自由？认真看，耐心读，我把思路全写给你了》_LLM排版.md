
# 你作为普通人，该如何拥有财富自由？认真看，耐心读，我把思路全写给你了

灏泽的这篇文章，篇幅非常精简，含金量很高。只要你还没有超过 55 岁，那么文章内的参考思路，都是能让你更加清晰知道自己该怎么努力，并一步步更加靠近财富自由的结果。因此，如果你对现在平庸的生活不满意，也不知如何发力解决，那么，耐心把本文读完，能让你和你的家人，都收获巨大。

很多人并没有意识到，其实这几年，就是绝对的时代、环境、财富的新周期，接下去会有不少的淘汰产业被世界所放下，但也有很多新兴产业尤其是科技类产业，会成为崭新的强力经济发动机。所以，哪怕你现在只是一个很普通的小白领，但只要认清这个趋势，并用耐心和毅力，去布局那些优质的企业，时间一定会以很好的前途回报给你。

灏泽一直笃信一条逻辑：单靠自己的单打独斗，所能创造的财富是很有限的，但只要坚定的看准大势，跟好国运并朝着顺应的领域去安排，那么自然就能享受到撼天动地的财富推背力，而这就是所谓的大财之源。

你一定要明白，其实那些比你富裕的人，并不比你聪明也并不比你有能力。你们之所以存在身价上的差异，就是因为他在你纠结内耗的时候，比你更勇敢的迈出了那么一步去进行了试错和总结。结果，他一步步的靠失败和损失积累经验，终于在无数次鼻青脸肿后，摸索出了适合自己的长期耕耘方向，之后，就是一路的坦途和顺利。

灏泽我还要很明确的告诫你：一定要警惕那些“一次性让你赚大钱发大财”的事儿，因为这种事儿的本质，都是赌博。赌博从来就和实力无关，而是纯粹凭运气的。你应该做的，是日复一日的坚持思考，并找到那些好企业、好方向，然后坚定的守望。这才是真正的长期主义。

对于上班这件事，一定要有清醒的认知。上班可以给你相对稳定的生活，但是终究是为别人的生意做贡献，而且你的收获永远就是工资而已，是很难突破天花板的。当然，上班也有上班的好处，那就是大多数上班的岗位，都没有太多的风险和压力，按部就班地做好手头事儿就可以了。

如果你对目前上班这件事不满意，你更应该做的，就是把自己的财富进行踏踏实实的布局。去找好企业，然后有稳定营收、经营状况良好，并且有着专业负责的管理层的企业。去持有他们的一部分，再持有一部分，周而复始逐步积累。当你拥有的良好企业的股份越多，你的身价自然也会越来越多。这才是适合普通人的最好财富布局。当然，切忌孤注一掷的豪赌，永远只用财富的一部分，去做积累，任何领域都是如此。

这一轮的科技产业，其实都是非常实在，绝大多数报得出名的科技企业，画饼的成分虽然有，但却也前所未有的务实。不是说它们思想觉悟高，而是如今这个时代，大家都相对清醒了很多，知道企业的首要任务就是能盈利，先盈利才有资格谈别的。这同时导致了一件很不错的结果，就是这些企业都会随着规模的增长而实现盈利的增长，不太会出现规模上去了，盈利却不上去的情形，这就是很好的趋势。

另外一方面，就是事实上如今全世界能朝着新科技推动的，也就我们和白发鹰了，所以这些企业也必然会在未来出现非常非常高的估值。一句话总结就是“祖国强大，科技才推得动，然后才能有好的收益”。

你一定要记住，大财富从来就是长期的“自我否定、自我复盘、自我坚持”得来的结果。至于所谓的小聪明，在上述这些行为面前，是不值一提的。灏泽我曾认真思考过这些年结交的所有富人朋友，几乎都开始变得越来越谦卑理性了，不能说全部吧，但是至少 99%都在完成原始积累后，开始明白自己的第一桶金是靠运气得来的。

下面这段内容，并非建议仅仅是纯粹的了：灏泽真心的希望，你能明白，人生中其他的诸多杂项事物都是不值得浪费你的宝贵精力的。你每天能运用的大脑精力，也是有限的，所以与其和那些狗屁倒灶耗费，不如正儿八经把自己的财富好好的安排到位。

我不会给你任何建议，关于如何去选择所谓的好企。因为这是一项你自己应该去花功夫做的事，更重要的是，当你开始花费精力和时间，开始搜寻好的企业，开始一年一年的翻看其报表，一条一条的了解其业务细节和高管构成，然后在各种社群去参考其他那些或靠谱或不靠谱的讨论时，你本身的内功，也是会慢慢增长的。这个过程会很痛苦，因为你会经历漫长的煎熬，但却依然颗粒无收。进群加这种时候，就是正儿八经考验你韧性的时候了。

灏泽我也不愿意提及具体的企业，我只再说一条非常核心的观点：今年会是科技产业的元年，这并不是说我们曾经就没有科技产业，而是接下去的时代，科技产业会以一骑绝尘的强度和高度继续进行发展。这中间当然会有结结实实的几次凶猛调整，也会因此造成很多麻烦和损失。但是，当这个世界上，有一门产业是只有两股力量能玩转，并且我们还同时拥有强大的基建优势和工业配套时，那么最终的胜利天平，终究会是我们的。而这所带来的财富回馈，自然也会在时间的推力下，成为一笔恢弘的数字，绵延不绝。
