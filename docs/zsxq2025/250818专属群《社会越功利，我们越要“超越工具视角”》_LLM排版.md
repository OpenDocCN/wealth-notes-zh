
# 社会越功利，我们越要“超越工具视角”

前段时间正是全国高校的录取季，很多新生在晒录取通知书，场面很热闹。比如中国海洋大学，在通知书里放了一小袋来自南极的南极海水。北京工业大学的通知书，用的是大飞机、运载火箭同款的碳纤维复合材料，可以直接用来切西瓜。

而热闹背后，今年还有一些变化也许值得关注。

首先，有些第一年招生的新学校，分数不低，可以说是巅峰开局。比如，宁波东方理工大学、大湾区大学等，还有一名674分的河南考生，因为选择了福耀科技大学而登上了新闻。我们列举的这些大学有个统一的分类，叫新型大学。也就是，直接对接产业，所学即所用的大学。

其次，有些专科院校分数线超过了一本。比如，山东省专科公费医学生的投档线，超过本科一段线62分。浙江有一名考了604分的考生，报考了武昌职业学院的定向培养军士。

这些新闻之所以有热度，关键原因之一，是它们松动了很多人习以为常的“唯名校论”。一些高分考生似乎不再执着于名校光环，他们把实打实的就业放到了更高的优先级。前面提到的一些院校都在就业层面有一定的独特优势。有些大学是在做新产业的定向培养，有的院校是在学习阶段就锁定了特定职业。

而且换个角度，大学的变动已经很久没像今年这么热闹了。今年自从高考结束以来，关于大学的新闻就特别多。有关于新高校、新专业的，关于新办学模式的。

当然，大学的改革，肯定不是为了改而改。它是基于新的挑战，来提出新的方案。那么，现代大学遇到的挑战到底是什么？它们的改革方向又是什么呢？

关于这些问题，著名的管理学者克里斯滕森曾经写过一本书，叫《大学的窘境与革新》，这本书的中文版今年5月刚刚上市。译者也是一位教育方面的资深专家，清华大学经济学教授、前副校长杨斌。最近杨斌老师还作为解读人，在得到名家讲书解读了这本书，其间还补充了很多近年的新信息，以及中国高校的视角。

正式开始之前，咱们先做个小小的铺垫。提起大学改革，好多人觉得这是一个特别神圣的事。首先你要面对未来的学子。其次你还得对过去的传统负责，毕竟，这都是前辈们的心血啊。

但事实上，要面向未来不假，但所谓的对传统负责，也需要看情况。很多所谓传统并不是前辈呕心沥血的结晶，而是偶然的产物。

大学改革的第一步，也是最重要的环节之一，是要先扔掉那些不必要的思想包袱。

比如，问你个特别容易得罪大学生的一个问题，大学的暑假能不能改？估计多数人的第一反应是，这怎么能改呢？哪个大学没有暑假呢？没有暑假还能叫大学吗？

但是，假如追问一句，大学为什么会要有暑假？这个传统是怎么来的？估计很多人未必能答得出来。现代很多大学的暑假，是参考了美国大学的设置。

长暑假最早出现在19世纪的北美地区，为什么要放暑假呢？当时不是为了让学生们做社会实践，而是因为夏天天气炎热，在男生为主的大学校园里，特别容易发生冲突。学校一琢磨，干脆给大家放假吧。考虑到当时交通不便，学生回家需要花大量时间在路上，于是，三个月的长暑假就出现了。

换句话说，最初的暑假，是学校在管理层面上的一个解决方案，解决的是夏天校园冲突高发的问题。从这个角度看，暑假能不能改？当然有可能改。只要它更能针对现代大学的独特挑战，更能满足现代学生的需求，它就可以改。

再比如，美国精英大学通行的很多模式，比如，高额学费、通识教育，还有针对老师的终身教职制度，等等，都不是历史上的传统，而是1933年到1953年之间担任哈佛大学校长的科南特，在他任职期间明确的。后来又被很多学校效仿。

换句话说，在大学改革这件事上，要尊重传统是真的，但也需要明确那些传统是怎么来的，别被一些不必要的光环绑架。

说到这，你可能已经发现了。大学改革在某种层面上和公司变革有点像。

你看，一般公司变革最大的阻力是什么？不是设计不出好方案，而是大家心理上不情愿。你要问同事们某个东西能不能改，大家第一反应就是不能，过去一直都这么做的，怎么能说改就改呢？

克里斯滕森认为，这是大学改革要解决的最重要的问题之一。这就是， 要把人们拽离一个误区，这个误区叫，“历来如此”就必须“长此以往”。

更何况，今天的大学正在面临一些挑战。克里斯滕森认为，其中最明显的挑战是，投入产出比的失衡。

一方面，一些学校付出的成本越来越高，硬件建设、教师聘任、科研花销等等，都在持续增加。我们时不时还能听到涨学费的新闻。前不久就有新闻说，国内超几十所大学学费上涨。

但另一方面，这些投入在一定程度上，并没有直接反映到学生的竞争力上。很多学生上大学，为的是增加竞争力，但四年学成，他们面对的竞争反而更激烈。

那么，面对这些挑战，大学应该怎样变革呢？

根据杨斌老师的洞察，有一个可能的探索方向，叫，超越工具视角。注意，不是忽略工具视角，而是超越工具视角。

你看，现在很多人不是希望通过大学获得一门谋生的手艺吗？杨斌老师说，人们越是有这个期待，大学越应该超越工具视角。什么意思？谋生本领肯定要教，但不能只教这些，否则一个个大好青年，岂不坍缩成了只会赚钱的“工具人”？

尤其是现在，AI进展飞快，大学就更得赋予学生那些AI永远替代不了的东西。

比如，元学习能力，也就是学习的学习。在一个知识快速更新的时代，这比掌握具体知识更重要。

再比如，人际网络的建构，大学有一个重要价值，是帮年轻人形成社群，建立人际网络。你学习四年期间结交的身边人，是无法被代替的。

再比如，身份认同，包括文化认同、职业认同等。说白了，大学得让学生获得更高水平的自我觉知。按照我的理解，就是得让人们重新对“大学生”这个身份感到光荣。

再比如，价值观塑造，尤其是在信息爆炸的年代，大学的工作之一就是帮年轻人形成好的价值观。

当然，这些都是抽象的方向。回到具体的大学，可以灵活调整的部分其实不少。

比如，克里斯滕森举了一个例子，美国的杨百翰大学爱达荷分校。它原本是一所两年学制的社区学院，从2001年开始，正式变为四年制大学，并且开始实施一系列创新改革。包括，把传统的两学期制改为三学期制，学生轮流在春、夏、秋三个学期上课，可以更快完成学位。再比如，鼓励学生选择在线课程。再比如，与产业界紧密合作，从大一开始帮助学生梳理职业方向，准备简历、模拟面试、培养沟通技能。

学校的改革推进了20年后，学生人数从8000人增加到了25000人，并且明显提升了就业率。

这是国外的例子。再看咱们国内，很多大学改革也在陆续落地。

比如，更灵活的学制。像福耀科技大学，用的是“3+2+3”分段模式，本科3年、硕士2年、博士3年。学生可以一口气读完，也可以根据个人规划选择中途退出，并获得相应的学位。学生可以根据自己的情况灵活规划。

再比如，大学教育与产业深度结合。很多高校和地方政府、创新区或者企业合作，成立了科创中心。比如，安徽师范大学、安徽信息工程学院和科大讯飞联合成立了“AI+高等教育研究院”，成都的电子科技大学与华为合作成立了“计算机-人工智能致远班”。

今年还出现了高校“订单班”，也就是，学校与企业达成合作，按照企业的“订单”来调整教学计划和课程设置，有针对性地培养学生。比如上海应用技术大学的化妆品技术与工程专业、郴州智能科技职业学院的智能机器人技术，都用了这种模式。

关于高校的改革还有很多，我们在这里就不展开说了。

最后，作为清华大学前副校长，杨斌老师说，今年是2025年，而100年前，1925年，清华经历过一次重要的转型，从一所培养出国人才的预备学校，转变成了为国内培养人才的大学。当时推动变革的清华教务长、前辈学者张彭春说过这样一句话：“在新旧交替的时候，正是研究的最好时候，最好的机会。”这句话也许同样适用于今天。
