
# 新闻解读

西三河分别是印度河（主要干流）、杰赫勒姆河、杰纳布河这几条河归巴基斯坦控制，印度虽不能大规模截流这些河流，但可以在上游建设有限的水电项目。根据条约，西三河总水量占印度河流域的80%(1665亿立方米)。杰赫勒姆河和杰纳布河是巴基斯坦西三河的主要支流，对灌溉系统至关重要，尤其在夏季冰川融水高峰期贡献显著。

东三河包括拉维河、萨特累普河、比亚斯河这几条河流最终汇入巴基斯坦内的印度河，但是印度有能力完全控制其水生命周期。这几十年印度利用1960年《印度河用水条约》的漏洞（未明确禁止上游国家完全截断水源），在东三河修建不少高坝水库，兼具灌溉、防洪与发电功能。其中最重要的是巴克拉大坝，是印度最高的混凝土重力坝之一，也是该国重要的水利工程之一。其他的还有比阿斯-萨特累引水工程，将比阿斯河的水调入萨特累普。
