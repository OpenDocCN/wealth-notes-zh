
# 学习笔记，2025.1.20

1. 打磨个人IP，做时间的朋友，通过时间杠杆放大个人价值，以此形成复利。个人IP的选择可以从自己的专业相关入手，选择一个IP，从一个渠道做深。上述选择不难，坚持最难。坚持下去，资源会主动来找你。对了，若真是想做个人IP，请记得“真实”。
2. 多关注身边的朋友，维护好那几个愿意为你付费、愿意帮助你的人。在赚钱的路上，愿意帮你的人会一直帮你，同样要向他们及时反馈成果，多交流，留住真正愿意帮你的人，并常怀感恩之心。
3. 以“利他”思维做事、待人，做到超预期交付。没有永远的竞争对手，所有人都是潜在的合作伙伴，“利他”最终利己。
4. 任何好东西，坏东西，抛来剂量谈效果，都是耍流氓。任何食疗，效果都约等于0，所以，不存在什么食物相克，什么跟什么都可以混着吃，其原因参考第一条。
## 矿泉水好还是纯净水好？
哪个口感好，选哪个。矿物质概念，酸碱概念，都是智商税。原先，很多科普作家写这些。现在，没人写了。因为，能写的人，都被枪毙了。所以，能流行的民间科普帖，你都反着信就行了！
5. 东北文化跟山东文化是一回事，从接站到送站，从早上一睁眼到半夜送回酒店，必须是无缝隙的安排项目，说的好听一点，好客，说的难听一点，都累。在山东最累的工作就是接待。
6. 建筑物只要跟佛教挂钩，就很难荒废了，因为，不断有大和尚发愿，要重修XX寺或XX塔，他一发愿，信徒们就掏钱。
## 信徒们为什么掏钱？
这也是一种征税。信仰税。青藏高原，地大物博，可是，寺院却密密麻麻，钱是哪来的？牧民的。别看他们家有几千只牛羊。卖了钱，他们也不舍得花，要捐掉……买来世的幸福。佛教里的轮回、因果等概念，最直接的作用就是寺院的兴盛，不断的新建，不断的重修，反正信徒们有钱，信徒们永远不缺，一代又一代。
这里面最巧妙的一个激励机制，是留名。英雄盖世，万古流芳。这不是普通老百姓可奢望的。
## 怎么才能留个名呢？
修庙立塔留功德。即便是牛逼如武则天，在这个机制面前，也不能免俗。到龙门石窟捐佛像。
7. 为什么古代很多名玉今天都不出名了？本质是，颜色不行。其实，泰山也产玉，泰山墨玉。试图把其它颜色的玉石卖出高价，这不现实，因为审美是很难改变的，去深圳水贝看看就行了，高端玉石手镯、吊坠，没别的颜色。产品销售过程中，科普成本是最高的。
8. 国内，凡是你需要买票看的长城。基本，都是近20年修缮的。真正的野长城是什么样？凡是路好走的地方，砖都被村民偷回家盖房子了。剩下的，你自己想象。基本，就剩个墙根了。
9. 不知道是从谁开始的，一搞旅游开发就是复古风，为什么经济难发展？因为大家总朝后看，不朝前看。现代性才是对一个人，对一件事，至高无上的评价。动不动就是古风。
10. 家庭不能给他助力，所以必须要他全身心的攻学业，先达到一个足够高的学历以及能力，有一技之长，能养活自己，能在一线城市站稳脚，这是唯一的出路。家庭经济条件好的，就比较占优势了，因为他有足够多的容错成本，可以随意试错，走歪了也无所谓，随时可以从头再来，甚至有没有文凭都无所谓，所以家庭经济条件好的反而更容易演绎多姿多彩。我现在看的话，除了985，211外，其它大学，都是草台班子。
