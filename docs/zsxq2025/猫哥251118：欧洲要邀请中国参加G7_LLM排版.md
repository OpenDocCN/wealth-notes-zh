
# 欧洲要邀请中国参加G7

近期，彭博社披露，马克龙试图邀请中国参加明年在法国依云举行的G7峰会，届时，不排除法国邀请中国加入G7集团，让G7变成G8。德国高层已经同意了该想法。

显然，马克龙的举动，不是个人的心血来潮，而是欧盟的一次集体共识。

1973年，加拿大、法国、德国、意大利、日本、英国和美国七国的财政部长，就“石油危机”一事召开会议，商讨应对策略，这是G7的起源。后来，逐渐发展为七国首脑定期碰面的固定模式。对于重大问题，七国的通常会先在G7平台上商量好，再和其他国家进行协调。

比如印度，在2019年前，G7从未邀请印度参加峰会，印度对西方来说，谈不上十分重要，但贸易战后，印度在产业链上替代中国的潜力，被西方所看重。2019年，莫迪首次受邀，前往法国参加G7峰会，此后每年都会参加，形成了固定机制，G7的举动，就是西方阵营的政策风向标。

此前，G7和中国的互动很少，不仅没有正式邀请我们参加过峰会，连部长级会议，都没有。这也正常，G7对中国的敌视不是秘密。

## 问题来了，欧洲这回，难道转变思想了？

马克龙邀请中国参加，乃至加入G7集团，葫芦里到底卖的什么药？

首先，是G7本身的问题。过去G7的话语权大，建立在“七国GDP占世界总量超过一半”这个基础上。随着非西方国家崛起，七国的GDP在世界中的占比大幅下降。以2024年为例，按名义GDP计算，G7的经济总量，占全世界的四成出头，按购买力平价(PPP)计算，则占世界总量的30%不到。无论哪种口径，都比巅峰期下降不少，导致G7说话越来越不好使了。

典型案例，就是G20的建立。G20的建立，让更多发展中国家获得了话语权，稀释了G7的份量。从近些年的趋势来看，各国对G20的重视程度，明显超过G7。

G20涵盖了全球80%的经济总量和60%的人口，每年的参会规模、议题数量和媒体报道热度，都远超G7，越来越有“小联合国”的意思了。而G7本身，则逐渐沦为“地缘政治平台”，除了骂中俄，就没啥实质性内容。更尴尬的是，特朗普的上台，让G7连骂中俄都成了奢望。

2025年6月，在加拿大举行的G7峰会，出现了历史性的尴尬一幕：未能发表联合公报。要知道，G7峰会上一次未能发表联合公报，还是在1975年。特朗普本身对盟友就不重视，甚至反感。G7作为美国盟友体系的重要支柱，在特朗普眼中，自然不是什么好东西。

为期两天的峰会议程，他只参加了一天，就提前离开，借口是“中东局势升温，需要紧急回国”。其实就是不重视。G7公报上对俄罗斯的态度，令特朗普十分不满，为了不惹怒美国，只能不发表联合公报。以特朗普如今的态度，在G7上骂中国，他也是兴趣不大的。

想想看，连骂中俄都不行了，G7的存在意义是什么？

美国可以不需要G7，但欧洲不行呀。失去G7这个平台，对欧洲的话语权打击很大，在G20和联合国，欧洲的话语权都远不如G7。如果中国参加G7峰会，那么，G7本身的重要性，是不是就重新凸显了？G7经济总量占世界经济的比重，是不是大幅增加了？欧洲的话语权，是不是能保住？

很多观察者觉得，如果中国参加峰会，很容易引发G7集团的内部分裂。其实，这是多余的担心。特朗普早在6月就表态过，对邀请中国参加G7峰会，持开放态度。

为啥？想想看，特朗普和中方会面一下，并不容易，一年也未必能见一次，但如果咱们参加G7峰会，会面次数多了，对他也没坏处。

以前G7不邀请中国，是意识形态问题，而特朗普又不在乎意识形态。马克龙能办成这事，特朗普肯定开心，给中美做中介，何乐而不为？

客观地说，中国加入G7集团基本没可能，咱们连奥巴马提议的G2都不感兴趣（特朗普前段时间又提了），低一个等级的G7，门票就更不稀罕了。只不过，不加入G7，不等于不参加G7会议。G7的会议，有多个等级，最高等级是峰会，由各国首脑率领高级官员参加。次一级的是部长级会议，还有扩大会议等等。参加一些低级别会议，比如财长会议，聊聊经贸、产业等非敏感话题。

其实，这是一个很好的、改善中欧关系的机会。当前的中欧关系，说白了，就是两大障碍：汽车和稀土。欧洲国家，尤其是法国，不希望中国汽车在欧洲卖得太多太好，冲击自身利益，法国希望从中国手里，得到一些汽车行业的技术和投资。

同时，欧洲又希望，中国不要在稀土问题上卡脖子。中欧汽车谈判，从2023年开始，到现在，已经谈了两年多的时间，对中国来说，给法国一些东西，不是不行，毕竟完全不投资当地，完全不输出技术，不现实。当年西方企业来国内搞合资模式，也是输出了一定技术的。关键是，给多少的问题。给多了，咱们不接受，给少了，欧洲人嫌少，这就是谈判艰难的原因，不过，根据《大公报》的报道，谈判基本完成，只差临门一脚。双方达成交易的可能，非常大。

如果汽车问题上，欧盟让步，那么在稀土领域，我们自然也不卡欧洲的脖子。这种大交易，通常会在重大场合完成。今明两年，有两个重大场合，一是12月，马克龙可能要来华访问，二是明年6月的G7峰会（如果邀请中国参加的话）。但12月大概率达不成太大的交易，主要是时间紧迫。

那么，明年的G7峰会，恐怕是达成交易的理想时间，借着“历史上首次参加”这个噱头，实现破冰，让不断下滑的中欧关系，止住颓势。

最后说一句，真正要看到的，是邀请背后的无奈。但凡有一点办法，欧盟都不可能邀请中国参加G7峰会。特朗普今年上台时，欧盟拼了命游说特朗普对付中国，试图“祸水东引”，还不断出台对华不友好的政策，然而，特朗普根本不吃这一套。在与中国的对抗里，欧盟也占不到半点便宜，这才无奈地想要缓和。
