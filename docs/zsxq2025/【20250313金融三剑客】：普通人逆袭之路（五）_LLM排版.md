
# 普通人逆袭之路（五）

## 一、奥妙

那螺丝钉是什么？一个工具人（棋子）而已，那被取代（成弃子），不是早晚的事儿么？即便没有比你更廉价的替代品（棋子），难道你自己不会老，不会累么？是不是这个理？So，剑客这里讲的普通人的“学习”，永远都是指隐学，如果你尚未开始过？又何来晚之说呢？

隐学的奥妙就在于，让你学会听懂别人的话外音，即你听中国人说话，不要听他说了什么，更要听懂他没说什么，那个才是重点。用什么去听懂？用你的阅历去听懂。

## 二、构建

那这个阅历（翻译器）如何构建呢？语言+背景+行为+结果。怎么理解？

中国人常讲，深入了解一个人不仅要听其言还要怎么样？观其行。

观其行之后就结束了吗？非也，还要察其果。即你听他当初是怎么说的，更要注意他说话的时间、场合等背景条件，这些都要搞清楚（避免张冠李戴，断章取义），这些都知道以后就算完了吗？没有。

你还要看人家到底干了什么（说的那些东西干了没有，如果干了的话，看他实施了多少比例）。

这就完了吗？还没有，你最 终还要看最 后实施的结果是什么。结果里面又分短期的，中期的，长期的。即你的追踪时间还要根据你的需求设置长短。

顶级高手，或者说专业选手的评估标准是什么？

知行果 合一！你可以认为人 天生是知行 合一的（做 不 到 就不是真 知，即知 到极 处便 是行），而那个果，则可以用来验证他的“真知”的含 金量。

所以，如果你单纯看一段字面意思，要理解到位靠什么？靠你的个人经验，靠你丰富的阅历。他是个老狐狸，你也是个老狐狸，你说了一句，你听懂他说了什么，而且听懂了他没说什么。

那他没说的那一堆是他告诉你的么？不是的，是你自己本来就知道啊。明白了？否则你是怎么听懂的（为什么两个人在一起久了有默契？就是共同的经历长期渲染的）？

你看，剑客在这里并有没有告诉各位那些爬上 去的人（非普通人），他们平常到底在忙什 么？没 有。但是有 些 读 者（本 就 在那 个 位 置的），本 来就知道嘛，于是他会心一笑。

## 三、障碍

你懂了隐学的这个奥妙，那人类要锻炼Al，让它更进一步，脱离那些漂浮在表面的范畴，进入更深层次的领域（拥有真正的智能），除非怎么样？

除非每个人都脑后插管，包括那些已经在上面的人（大财阀，大政客等），你也得强制给他们脑机接口，可以直接读取他大脑里的内容。

换言之，你得让AI有途径拿到人们台面下的想法（脑机接口直接读取上传服务器），而仅仅只是台面上的语言（一些可以公开表述的，政治正确的话）。

如果脑机接口技术不成熟，至少你也得想办法让Al把所有人都监视起来（数据搜集全方位无死角），否则它永远都只是在一个台面上的语 言大集合里面学习。

说白了，它是个学霸，而且是一个不知疲倦没有情绪的顶级学霸，但它始终学的都是一个表象层，它只是一个政治正确层里的顶级学霸而已。

就好比你问它1+1=几，它不会算错的，但是你问一个人1+1=几，人类会给你很多种答案的（比如人会指鹿为马，睁眼说瞎话，就是皇帝的新装的故事）。

这也是为啥中国那些真正顶级的会计师，并不在哪所大学里（一流监狱/二流私募/三流机构/四流大学），而在上海提篮桥监狱里呆着。

刘慈欣说得很正确，人类最大的优势恰恰是会撒谎（这也是智人统治地球的优势所在，即善于通过想象的故事把人团结在了一起）。

so，我们所有人从小到大好好念书，念的是什么？“真理”，是所有“政治正确”的集 合。

各位还记得买椟还珠的故事吗？珠，就是那个知晓隐学的老板；椟，就是那个跟进流程的助 理，是那张投递标书的包装纸。

所以那个斯坦福李飞飞团队只用50美金训练出 来的Al模型，本质只是个可以低成本的做出来 的高档包装纸。

问题是，包装纸它包裹了什么呢？包裹的那个有价值的珠，在哪里呢？在人身上啊，在那 个常伴关键人物的和珅身上啊，在那 个能探听到关键态度的首席身上啊。

你以为结果真的是他分析出来的？不，政治最 大的套路就是没有套路，套路可以随时定，为 了需要可以随时给你弄个框框出来的。

对，你现在知道了，那个“关键态度”的结果是打探出来的。那洋洋洒洒的大篇幅分析是干 什么用的？是用来包装那个结果的，把它包装成能够公布出来用的。

然后他也假装那么说，你也假装那么听，实际上你 们愉快的完成了一笔交易，一笔关于关键信息传递的交易。

乃至外行人还都没话说，因为形式上看起来一切都是基于专业理论框架分析出来的。谁也 不好讲你们什么，因为不能空口无凭，得有证据呀。

所以，你经常看到有什么群众举报，真得是群 众举报吗？群众能看得懂里面的数据在讲什 么？
