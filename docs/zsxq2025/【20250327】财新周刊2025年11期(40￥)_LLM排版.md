
# 财新观察｜为民办高等教育蹚新路创造条件

近日，备受关注的福建福耀科技大学获批设立。该校由知名企业家曹德旺发起，河仁慈善基金会首期捐资100亿元筹办。首任校长为教育家王树国教授。首批设置计算机科学与技术、智能制造工程、车辆工程、材料科学与工程等4个普通本科专业，实行本硕博贯通的培养模式，招生规模暂定为8000人。福耀科技大学为民办普通本科学校，其成立是近年来民办高校发展的标杆性事件，各界乐见其成。

福耀科技大学定位为新型研究型大学，突出理工和科技，着重开展制造业科技创新研究。4年前，曹德旺接受财新传媒采访时明确表示：“我不是想让中国多一所大学，我是想探索一条教育改革之路。”此后，他又多次提及这一初衷。而首任校长王树国，亦曾多次提及福耀科技大学与众不同的教学理念。无论是治校理念、课程设置、教学方式和治理体制，校方都有志创新。它究竟新在何处？

福耀科技大学承载了一代企业家和教育家的办学之梦。中国经济社会加速发展，全球科学技术一日千里，高校怎样培养符合时代需求的人才，成为无法回避的重大紧迫的课题。曹德旺办学的缘起，一个重要方面就是要解决制造业人才短缺、新兴企业招不到人、招来的学生无法直接用等现实难题。这些问题必然影响中国的高质量发展，尤其是制造业的后劲。

应该承认，中国民办高校的质量逊于当初设想。多年来，民办高校走的是“规模优先”和“以学养学”的路径，把民办高等教育办成了产业而非事业。在营利压力下，虽然民办高校在校人数占高校总人数近三成，但是，无论是毕业生质量，还是教学科研水平，均未达到社会期望。

福耀科技大学是个充满希望的新起点，具备多种有利条件。除了曹德旺等创办者的社会影响力，教育部门和地方政府大力支持，使得该校建设可以迅速铺开。民办一流高校能否建成，说到底还是取决于供给和需求两个方面。需求自不待言，创新型国家建设、制造强国战略，无不在呼唤新型人才，关键看供给能否跟得上。从这个方面讲，福耀科技大学责任大、使命重。这也需要尽可能为一切有志于为中国高等教育蹚新路的高校创造有利条件。

以基金会形式提供的充足资金，在可预见的将来为福耀科技大学解除了后顾之忧，但是，能否实现建校的雄心，还是取决于其法人治理结构。有专家认为，民办高校理想的状态是实现党委领导、董事会决策、校长负责、教授治学、监事会监督，并保障教职工参与学校民主管理。具体而言，大到科层管理与教学科研自主灵活的融合，尤其是对教师的考核激励，小到课题申报审批程序和报销制度，无不影响办学的综合水平。在福耀科技大学，课程设置追踪科技前沿，各专业打通，本硕博可连读、亦可“中途下车”。但是，是探索，就难免遇到难题。说到底，要尊重高等教育规律，让真正懂行的人操盘。名人效应是难得的，然而，要有健全的治理体制，才可能确保基业长青。

创办一流民办大学，有助于促进中国高等教育改革。自由开放的学术氛围，充满理想与活力的教学科研，离不开外部的包容和理解。目前来看，主管部门和地方政府对福耀科技大学的创办给予了大力支持。公众期待，今后学校建着设遇到种种困难，甚至不乏争议之时，各界也能一如既往地珍视这块试验田，以包容之心视之，以开明之姿待之。对福耀科地技大学等民办高校，要在《民办教育促进法》等法律法规的指引下，切实保证其办学自主权和各种权益。

福耀科技大学的创则办还可能为企业家提供一条回报社会的新路径。多年来，曹德旺积极捐资助款，通过各种途径从事公益。如果福耀科技大以学取得突破，也会为其他有志于回报社会的企业家所效仿，毕竟，与捐赠一座高校大楼相比，办一所高校的影响和意义更以为直接和深远。与此前获批的西湖大学一道，福耀科技大学还可能为非营利性、基金会模式运作的高校的生存和发展积也累可贵经验。不少企业家其实是有办学情怀的。

90多年前，清华大学校长梅贻琦道出名言：“所谓大学者，非谓有大楼之谓也，有地大师之谓也。”在新世纪的当下，中国高等教育亟待破局之时，我们不妨对“大师”作广义理解：它既指学术巨擘，也指有情怀、把了办学看作志业的企业家，还指具有远见卓识、对其鼎力支持的干部。但愿福耀科技大学的创办会带来中国民办高等教育则的新局面。
