
# 怎么求人办事？事情办的成，与办不成，原因是什么？

## 网友李五的问题：

有网上的大师张四说，求人办事，只能有三个人，中间人，你，有权的人。交往过程中，最好留下证据，作为制约，这说法正确吗？

## 陈旭回复：

这种说法，是错误的。

实际上，求人办事，吃饭的时候，引荐一下，可以是三个人。但具体办事的时候，只能是你和对方，不能有中间人。更不能留证据。具体办事的时候，还找中间人，那就是找中间人当证人。

## 找证人，留证据，这实际上，是对当权者的威胁。

求人办事，还要求第三人在场，这实际上，是穷人靠威胁，胁迫办事的思维，在起作用。

当权者也不是沙子，你弄个证人，还弄个收据，日后作证，当权者自己倒霉啊?

当权者知道你这个动机，因为你 不靠谱，就不给你办事了。

陈旭以前在私企打工的时候，老板曾经对我谈过，说他第一次送礼的感受。

那时候大家都很穷，他送完礼，就担心上面收礼不办事。 一晚上没睡着，后来事情办了，睡觉才踏实了。

再后来，他就明白了，别人不给 你办事，就不会收你的礼物，收了你的礼物，就会给你办事。

## 那么,有没有收礼不给办事的人呢?这样的人当然是有的，而且陈旭估计，还不在少数。

所以你看人一定要准确，不能看走眼。你不能因为担心对方，收礼不 给办事，你就找第3人作证，或者让对方打下收据。

你这样留证据的行为，就是暗示对方不要给你办事。能给你办事的人，他也不会给你办事。

因为你的行为告诉对方，你是一个 很麻烦的，危险的，不好打交道的 人。

万一稍微有点不如你的意，你在外面造谣诽谤，那上面的人也不划算呀。上面的人也不是沙子。

那么,万一对方收钱不给办事，这个风险再来承担?

这个风险当然是你来承担，你看人走眼了，你就得承担这个后果。你要是不愿意承担这个风险，那就不要求人办事，就不会有任何风险了。

所以求人办事，引荐的时候可以有 第3人，但是具体比较机密的话的时候，尽量还是两个人在一起。

这倒不是说，信不过中间人。而是说，你说话办事要给当权者安全感。知道的人越少越安全。

对方没有安全感，就不会给你办事。对方感觉越安全，给你办事的动机和意愿就越强烈。

当然并不是说，有中间人在场，就 不给办事了，不是这个意思。

实际上，很多事情，就是中间人在场办成的。

陈旭的意思是说，办事，要尽量让对方感到安全，靠谱。

## 我国并不存在贵族，大多数上位者，都是从底层上来的。

当权者，当年也经历过，你此刻正在经历的事情。这意味着，你经历过的一切，你内心的想法，当权者都知道。

虽然当权者知道你的想法，但他不会说出来。他只是看你的行动，然后决定自己的行动。

所以很多人说，自己遇不到贵人。

其实不是遇不到贵人，而是贵人把他们筛选淘汰了。

大家都知道，陈旭在私企上班，从底层保安升值到私企老总级别，只花了5年半的时间。

请问，如果没有贵人提携，我怎么 可能这么快升职?

现在问题来了，为什么陈旭，总是 能遇到贵人呢?

## 怎么办成事?怎么遇到贵人?

陈旭回复：其实很简单，你让贵人 感到安全靠谱，又有能力，你肯定可以遇到贵人。

办事找证人，留下证据，这就是企 图靠胁迫办事……

社会底层的穷人，他们总是企图靠 胁迫，逼迫别人办事，所以穷人遇不 微信到pep4贵58人，办不成事情。

陈旭常说，人生就是一连串的交 易。

在陈旭看来，事情办成的原因： 做人做事靠谱，值得对方信任，开 价又合适，事情就会办成。成功 者都是通过自己的行为，让别人自愿给你办事。

【2】利益不够，开出来的价格， 不合适。
