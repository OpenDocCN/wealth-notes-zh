
# 天机不可泄露：不同社会阶层的角色定位与认知信仰（附阶层结构图）

由于工作业务的关系，我接触过很多形形色色的人，虽然存在少数的个例，但基本上，同一个阶层的人，都会呈现出一些共性；而不同阶层的人，差异化非常大。我说的个性和差异，不仅包括了人的外在形象、语言表达，还包括了思维方式、认知水平、价值观念、行事逻辑等等。在所有这些方面，认知信仰的分化是最大的：同阶层群体的认知与信仰无比趋近，不同阶层群体的认知与信仰差异巨大。

虽然大家都生活在同一个社会中，但是不同社会阶层所处的微观生活环境是天差地别的，每个人面临的成长环境、接受到的教育，以及个人对自己的未来定位，都完全不一样。阶层不同，社会给个人潜意识中赋予的角色定位就不同；而社会角色定位的不同，决定了不同阶层群体所产生的认知信仰各不相同。

人的社会角色塑造了认知信仰，同样地，人的认知信仰能够重新定位自己的社会角色。理解了这句话，就掌握了洞察社会和人心的密码。

在讨论社会阶层的问题之前，必须先对社会阶层结构有所了解。阶层结构图见文末。

## 社会底层的角色定位与认知信仰

在整个社会阶层结构中，底层群体规模最大，社会底层是社会阶层结构中的基石，支撑着整个社会的运转和发展。基石不稳则大厦倾覆，所以这个群体最重要的就是稳定。

如何让这个群体保持稳定？无他，一是要让他们能吃饱饭，二是要让他们有事干。吃饱饭了，就不会有生存之忧惧，就能够安心生产、生活；有事干了，就没有时间胡思乱想，就不会有人去大街上捣乱。

吃饱饭的问题，在古代的农耕社会比较难，风调雨顺的年份还行，一到了灾年，就各种饿殍遍地。所以，古代的战乱发生，几乎都伴随着自然灾害，因为粮食收成不好，底层老百姓吃不饱，只能揭竿而起。

但在生产力大发展的今天，吃饱饭已经不是问题。全民脱贫，解决的是吃饱饭的问题，这是几千年来最伟大的成就之一。现在的人，只有吃得好不好的问题，没有吃得饱不饱的问题。

但有事干就相对难办一些。有事干分为两个层次，一种是身体有事干，对于个人而言就是得有工作，对于社会而言就是要提高就业率。就业率是任何社会都非常关注的宏观经济指标。无业者有工作了，社会就业率到达一定水平了，有事儿干的目标就算达成了。但这个目标没那么容易实现。

另一种是精神有事干。人的思想是闲不住的，只有一空下来，无所事事的时候，就会胡思乱想，胡思乱想的人多了就容易出社会问题。精神世界虚无缥缈，看不见摸不着，怎么控制？有两个绝招。

一是给思想造笼子，给每个人从小就造一个思想的牢笼，把思想关进笼子里，只允许一个唯一的标准思想出现。换句话说，就是建立一个思想的框架，把人的思想框在里面，一旦有人跑出了这个框架，就打为“异教徒”或“反派”。这个统一的标准思想，在古代东方叫做“儒学”，在古代西方叫做“上帝”；在现代东方叫做xxx，在现代西方叫“皿煮、自游”。这些在各个时代、社会，都是主流价值、zz正确。不同时代，不同的地方，叫法不一样，本质都差不多。只要人的思想统一了，群体的精神就稳定一大半了。群体精神稳定了，社会也就稳定了。

不过，人的思想总是躁动的，关在笼子里时间长了，就一定会有破笼而出的冲动。所以还有第二招：精神消费。

所谓精神消费，目的就是给精神找点事做，让人的精神忙起来而不能空下来。如何做？大多数人的大脑，其实就是一个跑马场，你每天接收到的信息，如同一群群野马，从你大脑的跑马场中呼啸而过。只要制造足够多的精神消费品，比如各种影视、综艺、小说、明星八卦、新闻热点、鸡汤等等，将它们一股脑塞进人的脑子里，大脑连消化都消化不过来了，哪还有空闲去思考别的呢？

换个说法，这叫精神文化需求。人脑子空下来就会胡思乱想，但短视频一刷就是一整天停不下来，一部接一部的电视剧、电影就能让人们天天讨论不停，逼真唯美的游戏玩起来更是无法自拔。这些都没兴趣？没关系，还有社会上各种奇葩新闻、突发事件、争议话题、明星网红、体育赛事等等，有热点话题就扩散热点话题，没热点话题就制造热点话题。

总有一些是你感兴趣，能吸引你的注意力的，这些东西让你跟着喜怒哀乐、嬉笑怒骂，将你拉入与自己无关却让自己感同身受的虚幻假象之中。总之，一定不会让你的精神空闲下来。

于是，精神待在思想的笼子里，享受着充满刺激的消费品的投喂，有数不玩的事情要操心，再也不会“胡思乱想”了。

在社会底层人群中，人们普遍将道德与情感作为主要的认知和信仰，道德与情感也是底层群体的行为逻辑核心。这对于社会来说不是问题，问题在于，对我们的个体而言，道德、情感是低层次的认知，容易受到操控和攻击而不自知。精神消费品就是不断地攻击这个低层次认知的弱点、漏洞。某种程度上，“道德标准”就是思想的牢笼，“情绪感知”是精神的快速消费品。所以大部分底层人，即便过着最困窘的生活，也依然能够穷开心，只要精神消费品的投喂管够，手里拿着一部手机，再苦再累都没关系。

## 社会中层的角色定位是什么?

对于整个社会而言，中间阶层是社会的中流砥柱，是一个社会发展进步的动力，因此，这个群体必须要“专精”，就是要有自己的“独门绝学”。只有这样，整个社会才能不断向前发展。

社会中层的认知信仰，是“独门绝学”，他们行为逻辑的核心，也是自己的“独门绝学”。

所谓的独门绝学，是一个人的立身立命之本。它们可以是一项高端的技术，也可以是一种商业模式，或者是一个产品的核心配方。这个“独门绝学”，是资源本身，也可以给你带来资源的东
西。总之，它是某一部分人立身立命的根基。这是底层人不具备的，也是中层能优于底层的关键。

中层如何获取“独门绝学”，上层的认知与信仰是什么，这些认知信仰如何影响社会和我们个人的发展，鉴于下面这部分内容敏感的东西较多，不能公开发表，只能付费阅读。

“独门绝学”，好比是武侠小说里风清扬的独孤九剑，达摩的易筋经，往往是在一个宗门、家族、组织里的机密，师傅传给徒弟，长辈传给晚辈，只有少数人有机会学习。而放在商业市场里，就像宁德时代的电池技术，字节的短视频推荐算法。

要从底层爬到中层，最好的办法就是练成一种独门绝学。社会就是一个优胜劣汰的筛选体系，每个人都在修炼自己的武功，只有练成独门绝学的人，才能在市场上获得一席之地。

有的人天才卓绝，自己就能创出一套独门绝学；有的人家学渊源，从小就在家里关着门修炼独门绝学；有的人运气不错，能够拜师学艺习得一种独门绝学。自己练还是找师傅学，能不能学到、学会，要付出什么代价，就得各凭本事了。

但大多数人底层人呢，都是懵懵懂懂，浑浑噩噩，不知江湖是为何物，不知人情为何冷暖，不知世态何时炎凉。

为什么有人为了获得一种“独门绝学”，愿意付出巨大的代价？比如林平之，为了辟邪剑谱可以入赘，还可以自宫。因为有了“独门绝学”，就可以开宗立派，意味着江湖地位的快速提升，意味着能够让自己和家族跃升几个社会层级。而平平无奇之辈，只能被淘汰掉，成为底层中的一员，在道德与情感的笼子里过完虚幻的一生。

## 社会上层的角色定位是什么?

上层是社会资源的分配者、权力的使用者、社会的管理者。这一阶层能够决定社会的走向、一国的兴衰、历史的起伏。因此，这个阶层的认知必须具有相当的高度。

上层的认知信仰是规则，规则是道，是普通人看不见的真
理、规律。底层人喜欢道德与情感，比如电影里常有的桥段，灾难来临的时候，为了救一个人，让一群人面临危险。观众喜欢看这些，看完了很感动，感动到落泪。

但是社会现实不是这样的。社会现实是，当洪水发生时，为了救一座大城市，必须牺牲几个小村落；让小部分人先富起来，就有必要牺牲一部分人的利益。现实里有牺牲、有利益的取舍，为了生存，必须进行残酷的斗争。做一个决策，为了利益的实现，不得不压制情感与道德。

这是上层人群的行为逻辑，上层决定了整个社会的发展方向，他们能够决定所有人的利益得失。利益与利益相互结合、斗争，由此产生了政治、d派，产生了各种组织、团体，各种帝王术、权谋计。

相对于道德与情感，权力、利益、规则、斗争这些才是上层的核心问题。

有人会问，难道越到上层，人就越没有道德了吗？当然不是。所谓夏虫不可以语冰，曲士不可以语道，这样的想法，其实是一种阶层思维的局限性。普通人所认为的道德，其实是一种弱势文化、底层文化，是弱者的自我保护。

比如孝道，汉代以孝治国，于是有很多荒唐的故事，什么卧冰求鲤、为母埋儿。这些都是在底层才会出现的故事情节，是底层百姓的自我保护机制。试想，一个社会中层的富裕家庭，需要这样的孝道吗？完全不需要。当一家人在海滩度假、吃大龙虾时，你跟他们说要卧冰求鲤、为母埋儿，他们会觉得你是神经病。

恰恰相反，在大家族里，不但不需要所谓的孔融让梨式的“美德”，反而要在家族内部形成竞争，优胜劣汰，让最优秀的后代脱颖而出，才能保证整个家族的延续。

底层的筛选，是将能够驾驭情感道德的人，从低认知的“羊群”中筛选出来；上层的筛选，是将最擅长争夺利益的人，从精英的“狼群”中筛选出来。这在任何利益集团、组织内部都是适用的。如果一个组织内部，各个都是温良恭俭让的谦谦君子，那这个组织迟早要完蛋。

为什么民国最早成立的国民d必然会败？这帮人七十多年前，干不过穷乡僻壤里发展出来的弱小组织；七十多年后的今天，依然干不过一个半路出来的民进d，混得要多惨有多惨。

其实不难理解，看看老蒋当年搞的d内思想建设就明白了，一个大组织里推崇的都是封建儒家的“四维”、“八德”，什么“礼、义、廉、耻”、“忠、孝、仁、爱、信、义、和、平”，全都是农业社会那一套道德标准。这些道德标准用来忽悠底层群众还行，但在组织内搞这种精神建设，那就太低级、太小儿科了。

任何物种、组织，想要发展壮大，都是要经过竞争淘汰的。背后的本质，是利益的争夺、博弈，而非道德上的谦让。在一个组织里面搞这种低层次的思想标准，那大家只能表面一套、背后一套，最后自己把组织玩废了。

再看看这边的组织内部，隔三差五就是斗争，刀口向内，哪有什么温良恭俭让的含情脉脉？这才是一个组织发展壮大的内生力。

## 总的来说，下面的两句话一定要记住：

一个强大的人，使用的却是低层次的认知信仰，那注定会沦落为弱者。一个暂时的弱者，开始使用高层次的认知信仰的时候，那么他注定会变得越来越强。

当人们把道德和情感当成认知信仰和行为依据的时候，一定是他处于弱势阶层的时候。什么时候你开始摒弃这个体系，你就开始走向另一个层级了。
