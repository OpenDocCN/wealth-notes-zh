
# 屠龙手段-老实人需要长的几个心眼

## 1.要用非常高的格局和宏大的叙事，把自己的人生经历串起来

有一个粉丝向我提问：“杨老师，我现在想寻找自己的优点。我发现我自己抛开以前的经历不说，我觉得我只是英语阅读还行。听说都不行，但是看文章还不错。您不是说要多看大部头吗?我把巴菲特的滚雪球，还有芒格的书都看了。”

我感觉她年龄得有35岁以上了，还带着两个孩子，我就想怎么能让一个两个孩子的妈妈全职在家炒股呢?这后面都是深渊和隐患。但是当时人很多，我没法教育她这个。我就问她：“你大学原本的专业是什么?”

她说：“我大学原本的专业是图书管理系统，所以等于没有什么专业。”我又问：“那你后来工作做什么呢?”

她说：“我做的是手机供应链采购。但是我因为学文科，其实后面并没有特别了解手机的整条链路。”我又问：“那你现在做什么?”

她说：“我现在发现特别喜欢看理财的书，所以把很多理财的大部头都写下来了。我其实英文阅读不错，但是听说写是不行的。我感觉我就这两个能力了。”

我说：“首先，你推销自己的时候，不能抛开个人以往的经历。你需要寻找你现在做的事情，与你以往的事情的相似性。你以前做的是实体产品的采购：在产业链上找性价比最高的材料和零件。你现在做的是金融产品的采购：在整个金融市场上找性价比最高的产品。”

就像巴菲特“捡烟头”理论一样：我们不论购买什么东西，都需要寻找“价值洼地”。买卖股票、期货、房产也是如此，像我们做品牌买流量也是如此：我也要天天寻找价值洼地的流量—到底哪个KOL实际流量高但是定价低，到底哪个播客哪个APP 的开屏广告实际定价是便宜的，到底哪个明星具有成长性，现在购买他当代言是值得的?

采购是一个底层能力：寻找价值洼地的能力。而你的英语能力是赋能这个底层能力的，能够让你具有全球采购的能力。

所以总结起来，我告诉你怎么介绍自己：
我具有寻找价值洼地的能力，这个能力体现在实体产品上，也体现在金融产品上。我干过供应链采购，和理财投资，都很成功。而且我英语很好，因此我的能力可以延展到在全世界范围内寻找价值洼地，从而进行国际投资，和全球采购。”

我还告诉她：“然后你把这个故事不断打磨，不断按这个方向去做，把寻找价值洼地形成一套方法论，从多少个维度能看出来价值洼地，这才是更强的核心竞争力。”

## 2. “管理”并不是真的管人，而是要因材施教，真无药可救的，则要把他们的影响“圈住”

第二个故事是来自另一个粉丝。她在某医院当行政，因为工作不错，领导要提拔她，但是在提拔的路上，让她先管两个关系户。她跟我说，她非常痛苦，这两个关系户小孩什么都不要干，把工作都堆给她，而且还不能说，不能碰， 一说就各种理由。

我说：“你知道管理是啥意思吗?管理并不是真的让你管她们。他们已经长大了，但凡有点出息，也不可能去当关系户。领导要提拔你，你当了中层之后，会遇到很多这样的关系户：又懒又不要脸。现在让你管两个，就是锻炼你管理这种人的能力。否则直接提拔你，你会崩溃的。”

所以她们已经没救了，你就哄着就完了。天天夸他们俩，工作做得不错。她说：“可是我的工作做不完咋办?他们两个什么都不干。”

我说：“做不完，就跟领导说，两个小朋友已经很努力了，但是成长需要时间嘛，再给他们一些时间和空间。领导就明白你的意思了。医院也不是你家开的，尽力就行，实在干不完就冲领导眨眨眼睛，说我刚才那段话：他们需要时间和空间—暗示就行。”

我后来跟她说，你不要以为“管理”这个词就是简单的管人的意思：它其实是因材施教的意思。有能力的提拔，能用的糊弄过去，关系户给他们圈起来，哄住，伤害降到最低就行。

我还跟她说：“不要跟他们较真，他们只是你升职路上的垫脚石。”我没跟她说的是，作为普通人，发展才是硬道理—自己发展了，才能收拾这些人。她就应该把这些人默默记在心里，但是不动声色，哄住他们。

先顾着自己升职，升到了副院长，比这些人高三个级别以上，再慢慢收拾关系户：那个时候才是杀鸡儆猴，敲山震虎的时候。

对于关系户，比他们高一级，是不足以收拾他们的。但是我们也不要放弃希望，放弃自己的发展，记住这些人，时间长得很，只要自己成长了，有的是时间收拾这些蛀虫。

既不能以牺牲自己的前途为代价与这些人缠斗，也不能丧失信仰变成和他们一样的人。

所谓“管理”, 就是先把这些有害的人圈住。专注自己的发展。

最后我说：姑娘，你可以看看张居正的故事。他是怎么与各路人马缠斗的，怎么一路升职打怪的。你看看他经历了什么妖魔鬼怪，你就会发现，你周围那两个小关系户，都不够看。

要想当大领导，就要明白，做任何事儿，收拾任何人都是有其时机的，但是最重要的，就是要把自己的根基打稳。

## 3. 如果有资源，一定要学会撬动资源，不要遇见事情就想出去单干

第三个粉丝是个非常年轻的姑娘。她是学美术的。毕业后去了一个游戏公司，一开始是做前面的海报等投流产品的策划的。

后来她说，发现真正的前端部门的“投放部门”:也就是购买流量，看ROI(投资回报率)的部门。她遇见一个特别好的女领导，给她捞到了投放部门，手把手教她投放。但是干了半年之后，她又想转行，因为她有两个困惑：
a).投放部门有另一个跟她一起进来的女孩，专业搞投放的，能力特别强。虽然她跟领导关系好，但是耐不住对方实在能力极强，所以好的资源的项目都给另一个女孩去做。
b)、她觉得光干投放，就把她的美术能力丢了，她觉得是不是去别的地方找工作能清闲一些，再把美术这件事当副业捡起来。

我问她：“你是美术哪方面比较好?”

她说：“手绘。”
我说，那你出去干什么呢?你好不容易找了一个好领导，关系也不错，你最难的半年已经熬过去了，现在转行再来多不值?
你不想放下美术，你还想多赚钱，你还想搞副业?那你就按我说的三部曲来：

第一步，找你现在的投放部门的女领导，你跟她谈，你决心要跟那个能力比你强的女孩挑战一把。她安排一个公平的投放比赛，如果你 的投放数据比她好，以后也给你一个好项目做做。反正领导已经认为她能力比你强了，你即便输了，她也会认为你这个孩子很有挑战的勇气，你万一赢了，就一把翻盘，你怎么都不亏。

第二步，你去跟领导说，你会手绘，要利用自己的长处。你给你的投放广告下面留下一行字：入群可以领游戏手绘的周边。你去手绘几个游戏的周边：书签、信笺等等。做一些周边，让公司出点打印费。这样你很有可能因为这个特殊的周边，比另一个女孩投放成绩好。你可以靠手绘和投放的综合能力打赢。

第三步，一旦你用手绘打赢了第二步，拿到了比较好的数据，你再去找其他的游戏或者二次元的公司，说你用手绘周边和投放的综合策略得到了多么好的数据。如果他们需要手绘周边和投放的综合策略，可以把活儿外包给你做，你帮他们订制一套周边+数据投放的全包服务——这不就是你想要的副业吗?

这个故事告诉我们，我们不能想一出是一出，一定要“审时度势”能撬动，动公司资源证明自己能力，千万不要盲目地“打一枪换一个地方。”
