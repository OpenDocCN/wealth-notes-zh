
# 解读川普《交易的艺术》

## 全文概要
美国总统特朗普的自传《交易的艺术》，由记者托尼·施瓦茨代笔，深入剖析了特朗普家族三代的创业历程。书中将失败案例描绘为个人成功，夸大成就，但更像一本自传体小说，描绘了特朗普心目中理想自我。尽管经验真假参半，它提供了职场与商场启示，曾长期占据《纽约时报》畅销书榜单。特朗普家族三代接力赛式的发家史，从祖父移民美国，到父亲积累财富，再到特朗普本人成为地产大亨和美国总统，强调"think big"理念为成功关键。对话鼓励听众学习在资源匮乏时破局的方法论，勇敢追求更高目标，不松懈，勇往直前。

## 特朗普家族创业历程
特朗普家族三代的发家史，表面上看是三代传承厚积薄发，实际上都是在各自的时代抓住主线不放。懂王的爷爷搞擦边，因为那时候美国到处都是草莽英雄，你要敢于在灰色地带寻找商机。而懂王的爸爸空手套白狼是胆子大，赶在危机的时候抄底，抓住百年一遇的机会。懂王自己则擅长媒体操纵和摆弄乌合之众。三代人玩的都是最时髦、最主线、最风口浪尖的东西。所以短短100年就从寒门登顶权力巅峰。

## 懂王生意经
1. **志向远大**：敢于想象，格局要大，目标要高，用中国人的话说叫做志存高远。懂王在书里说，我喜欢好大喜功，因为我不觉得这是件困难的事情。既然人们总要思考点什么，那为什么不往大的地方想呢?

2. **留有余地**：风险不可避免，但你可以把风险转嫁给别人，永远不要让自己成为最后一个承担损失的人。如果自己很可能承担风险，那你就要做好预案，从plan b到plan z你都要想好应对的方法。

3. **保持灵活**：不要把自己困死，留出转身的空间。交易中最大的错误就是你过早做出不可撤销的承诺。因为你过于着急，太想得到，往往容易被拿捏。

4. **洞察市场**：市场是战场，你要知道你的敌人是谁，他们的弱点在哪里，你必须了解你的客户想要什么，他们害怕什么，如果你不能让他们相信需要你的产品，那你就不是在做生意。

5. **杠杆施压**：制造竞争假象，迫使对手让步。有时候你并不需要真正的有其他买家，只需要让对方觉得你有。

## 启示
这些经验不仅适用于大型商业交易，对普通人日常生意和决策同样具有启发意义。做生意你必须了解市场，就是你只要能打动消费者就行了，最核心的是你要了解你是销给谁，你如何给对方制造恐惧和传递希望。
