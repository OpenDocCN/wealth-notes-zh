
# 卢克文青云读书会趣说挡史：学生毛ZD（下）舍我其谁之文明其精神-第7期

## 学生毛泽东(下):舍我其谁之文明其精神

大家好，我是杨文哲，欢迎收听卢克文工作室的党史栏目。上回咱们说的学生毛泽东是如何注重锻炼身体的，并且由一个人发挥主观能动性去自律去锻炼身体，使得自己的身体从弱变强。引申到一个国家，也可以通过发动全民同心协力来使国家从弱变强。

很多人肯定会觉得自律好难，经常是决定早睡早起锻炼身体，结果坚持不到两三天就不行了。其实自律的驱动力一样是源于人的内心，具体点就是你的目标，你的志向。这些年网络上有一种观点说自律的驱动力是野心，我认为野心这个词不是很恰当，野心更多的像是一种欲望，而志向则不同，它属于自我实现的需求。你想要干一件事和你笃定自己的余生，就只干这一件事的心态是大不一样的，这便是立志的重要性。

我们前面几期说到毛主席在少年时期就已经想要救国救民，但那个时期还不算真正励志，只是有一个模糊的想法在前指引，具体这条路该怎么走，他当时还小也是不清晰的，所以凭感觉认为应该去当老师，教育更多人，这才报考了师范学校。但是到1917年下半年可能由量变发展到质变了，也有可能是暑假的一个月的穷游起了最后的催化作用。我们身体倍儿棒的润之的心性发生了重要的变化。这个变化得从一封信说起。

1917年8月16日，毛泽东和萧子升二人结束行程返回湖南，8月20日返回学校，8月22日开学，8月23日正式上课，也就是在这一天距离打秋风结束仅仅一周，毛泽东写了一封长信给当时远在北京的黎锦熙老师，这封信恐怕便是毛泽东成为毛泽东的起点性标志，一个从平凡到伟大的分水岭，一个从优秀到卓越的分水岭。

学生毛泽东通过这封信向自己的老师倾诉了许多的心里话，首先就是吐槽当时社会的各种问题，简而言之就是烂透的，还被一群不学无术，胸无点墨之人把持政权，天下必须要有大的变革才行。提出欲动天下者，当动天下之心，而不是照猫画虎模仿其他国家的一些政权的形式，比如什么议会、宪法、总统、内阁这些表面上的东西，那么如何动天下之心，动其心者当具有大本大源，什么是大本大源夫本源者宇宙之真理，也就是世间万物的本质规律。

而这个本质规律实际上一直存在于天下人的心中，因为天下人莫不是宇宙的一份子，这种宇宙真理与法则多少会体现在每个人的心中。所以当掌握了这样的宇宙真理，便可以号召天下统一思想，统一行动，颠覆现有的社会，创造全新的社会。

学生毛泽东进而提出，必须要出现伟人来改造哲学，转变天下的思想，接着批判了当时社会上的那些精英阶层，所谓的励志无非是模仿中外前辈，而不是从本质规律出发，所以认为那种志向基本是盲从之志，而有真志真立志的人要具备两个特点：

一是认识并领悟宇宙之真理，也就是大本源。

二是能根据大本源确立自己中终生不渝的奋斗目标，而这个目标要符合真理指引的方向，而要成为这样的人，且得经历一番求索，要从少年时期就不断的求索才行。

他就是在说自己，因为他紧接着就反思自己，也曾一度重复这些人立盲从之志，今后要改正，并正式立志只将全副功夫向大本大源处探讨。

随即便说，圣人既得大本者也，贤人略得大本者也，愚人不得大本者也，圣人通达天地，名贯过去，现在未来洞悉三界现象。这话说完，其实我们就知道学生毛泽东已经决定要成为圣贤，但正如我们上期节目所说的，学生毛泽东的格局早就不局限在腐朽的精英阶层，而是放眼普天之下的黎民百姓芸芸众生，哪怕他们的大多数现在还是愚人。

但他们占天下人的大多数，他们才是天下的主体。所以要动天下之心，以动天下就得先团结大多数人。于是信中中学生毛泽东继续表示，吾人欲使此于人而归于治非，普及哲学不可。

所以后来毛主席的领导方法最根本的特点就是群众路线，就是通过思想教育和理论武装，使广大群众变成自觉的战士，依靠他们完成天翻地覆的伟业。看看毛主席时代我国创造的一个又一个奇迹，就知道毛主席早在学生时代的认识是多么的超前和正确。掌握的哲学武器的人民大众便是6亿神州尽尧舜，而离开了哲学思想的武装，群众又会退化为一盘散沙。

那么如何去向这些人普及哲学，小人类君子当存慈悲之心以救小人。这里的小人是文言里的小人，不是现代汉语里的人格道德层面上的小人，而就是指的芸芸众生，群众百姓。既然要存慈悲之心，就小人就不能像那些自以为是的精英阶层那样自命不凡和清高，而应该视众生为平等。

学生毛泽东信中继续说道，若以慈悲为心则此小人者吾同胞也，吾宇宙之一体也。君子小人都是骨肉，同胞都是宇宙的一部分。

如果君子都只自顾自的奋斗，那些小人便会愈发沉沦受难，那么整个天下包括君子自己都会遭到反噬，所以君子要尽力的拯救他们，帮助他们开发他们的智慧和才能提升道德境界。

然后君子和小人便可以一起共建一个理想社会，就如同东方的佛祖所说的极乐世界。尽管当时学生毛泽东还没有接触到马克思主义的理论，但他的思想许多地方已经同马克思提出的共产主义不谋而合了。

总结一下就是心若菩提，救苦救难，普度众生，天下大同。

远大志向确立了以后从哪里起步呢?学生毛泽东信中说到读书为上，教书办事为下，也就是他想把主要精力放在学习上，如果学习不好，教书办事也不会好到哪去。

因为毛泽东当时是师范生以后的就业方向主要是当老师，所以才说教书办事，但随即陷入了迷茫。因为如果没有经济基础支撑，如何把主要精力放在学习上，所以他在信中末尾直言自己最迷茫的便是赚钱好难。这封信写完就开学上课了，他的同学好友也都回来了。

1917年的9月16日星期天刚开学不久，毛泽东、蔡和森、彭泽厚、张昆弟四个人跑到昭山寺露营，毛泽东此时还在迷茫于赚钱好难的问题，但此时他还没有接触到马克思主义经济学，自己也想不出什么好办法，只是表示吾辈必想以最容易之方法解决经济问题，而后求遂吾人理想之世界主义。

用现在的话说就是先财务自由，然后做自己想做的事，只是他想做的事是革命。然后在这次录音期间，他就首次提出了心力的说法，就是心的力量，我们现在叫主观能动性。

由于几个人在寺院里露营，就自然聊到客了佛教，彭泽厚说自己想出家为僧，结果不知道当时毛泽东想要普度众生的志向，是不是恰好与佛教的思想不谋而合，还是感慨于佛法无边，恐怕恰是他要求索的宇宙之大本源。

居然也表示出想出家为僧的意愿，并且比同行的张昆弟还要强烈。你看这毛泽东学生时期还差点，一念之差就出家当学和尚了。好在毛泽东的性格对猛烈的革命运动更感兴趣，不然这历史就真改写了。

一周后1917年的9月23日，毛泽东、蔡和森、张昆弟三人又去了岳麓山下的蔡和森家，再次深谈理想与志向。这一期间应该是受到正在如火如荼进行的新文化运动的影响，毛泽东提出了文化革命的想法，说要把唐宋以后的文集诗集全都用炉子烧掉，要进行家族革命，师生革命。

但这种革命不是兵戎相见，而是要除旧布新，要用新思想新文化来教育人民，才有可能实现国家和民族复兴，不再受列强的欺辱和宰割。这恐怕就是后来毛主席发动文革的初衷，以及文革时期毛主席反复强调的要文斗不要武斗的思想源头。

毛主席终其一生都在为实现自己年少时期的理想而奋斗。
