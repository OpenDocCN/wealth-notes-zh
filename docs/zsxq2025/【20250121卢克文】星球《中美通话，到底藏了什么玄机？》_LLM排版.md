
# 中美通话，到底藏了什么玄机？

文/卢克文工作室嘉宾 流离

1 月 17 日晚间，应美国当选总统特朗普之约，中美双方进行了通话。问题来了，双方到底谈了什么。

先看懂王说了啥。毕竟还未上任履职，他只说了“希望继续保持对话沟通，期待尽快见面”以及“美中是当今世界最重要的国家，应保持长久友好”的片汤话。

而我们的态度，基本都在通稿里。首先，我们指出“高度重视彼此互动”。其次，我们强调“中美两国间拥有广泛共同利益和广阔合作空间，可以成为伙伴和朋友”。最后，我们指出“中美之间难免会有一些分歧”，并且在台湾问题上“希望美方务必慎重处理”。当然，双方也就乌克兰危机、巴以冲突等重大国际问题交换了意见（特朗普称讨论的是贸易平衡、芬太尼和TikTok等问题），并同意“建立战略沟通渠道”。

整个通稿看下来，似乎看不出什么，但是如果和历史进行对比，就能发现大有文章了。

2017年懂王上任之初，两国很快举行了通话，整体氛围可谓融洽无间。我们当时的用词，是相当友好的，比如我们指出“中美两国发展完全可以相辅相成、相互促进，双方完全可以成为很好的合作伙伴。”2017年初的懂王，也显得非常热情，他表示：“我相信，美中作为合作伙伴，可以通过共同努力，推动我们双边关系达到历史新高度。”并强调“美国政府坚持奉行一个中国政策。”

2017年的通话，是“完全可以”“很好的合作伙伴”，可以说都是溢美之词了。而2025年的这次通话，却变成了“可以成为伙伴和朋友”，听着就不温不火，冷淡了很多。

从“我们是朋友”到“我们应该处好关系”，但凡有个朋友的人，都能品出意味：关系不行了，别说当朋友，表面友好不打架都不错了。

再说，虽然特朗普现在没资格代表美国政府表示“坚持一个中国政策”，但就冲他同时给大陆政府和台湾当局发邀请函来20日就职典礼的行为，你敢说你敢信吗？

而且，17年新年电话中，两国元首“同意保持密切联系”，表示要“加强各领域交流合作，并期待着早日会晤”。这表明电话中多少谈出了一些成果，而且也都同意加强交流合作，都期待会晤。2025年的电话就不同了，不管讨论的是啥，只说交换了意见，没下文。多半就是没达成共识。

而“密切联系”跟“战略沟通渠道”也差远了，前者还能算友好措辞，后者完全没有什么友好意味，只是为了防止中美战略误判，才必须搞这个渠道。

不仅如此，这次电话只有特朗普单方面“期待尽快见面”，没有了双方希望尽早会晤的字眼，很显然，我们方面的态度冷了许多。

两份通稿看下来，可以得出结论——和川普的这次通话，双方都显得冷淡了许多，从前的融洽氛围不在了，尤其是我们这边，表现相当谨慎。

为什么呢？川普卷土重来，不禁让中国人想起他第一个任期的所作所为——先大打中美友好牌。

2016年11月的首次电话和2017年初特朗普给我们打贺年电话，双方都特别强调“合作”“共赢”；17年4月，我们还应邀前往海湖庄园与特朗普会晤；这一年11月8日至10日，特朗普也对华进行了为期三天的国事访问。

这一年，中美之间一派和睦气氛：两国元首达成互访、多方面达成重要共识、各级别交流加强、开展各领域广泛合作、经贸领域也签订了2535亿美元超级大单。结果呢？不到一年，懂王就挑起了贸易战，对中国商品加税，更是2018年12月在加拿大绑架孟晚舟后，要求将其引渡至美国，还有科技战等等。

在特朗普的种种挑衅行为下，中美关系迅速跌至低谷。

对我们来说，历史教训不可忘。特朗普第一个任期，一开始对华也很热情，但还不是说变脸就变脸、说背刺就背刺？虽然对他的“善意”，仍要礼尚往来，但他的“劣迹”，我们也没有忘记，必须谨慎处之。

所以我们对懂王的态度，目前很明确，就是他递过来的“善意”多少接一些，但是很谨慎，不能全部接。不仅通话内容是这样说的，实际也是这样做的。

似乎是为了延续这种“善意”，1月17日新华社报道，我们将派韩正赴美参加20日特朗普总统就职典礼。

要知道，美国总统就职典礼，从1874年有记录以来，150年间从未有外国领导人参加，几乎是个不成文规矩。特朗普打破这个成规，向中国在内的多国发出邀请，一方面是要实现他将就职典礼办成国王加冕，另一方面就是借机抛出橄榄枝，对一些国家示好。

而特朗普早在1月初就表示已通过代表，与中方高层进行过交流，并邀请了中方最高层参加就职典礼。

我们不可能应邀的，否则就跟应邀的米莱坐一桌了，绝对不合适，但会派“特别代表”，在此之前各路海内外媒体也是绞尽脑汁地猜想“特别代表”是谁，有说政治局常委蔡奇的，有说外长王毅的，众说纷纭。

特朗普在就职典礼上，表面是给了中方“特别礼遇”，实际上也给了一定的外交压力。若派常委“五把手”蔡奇，级别太高，有损外交利益。派王外长，似乎又不够高，显得气度不够。

而韩副主席一是正国级人物，地位够高，能对标美副总统万斯，且不是常委，做特使出席不算敏感，二是他外交经验丰富，2023年曾代表我们出席英国王加冕仪式。总的来说，是最稳妥的人选。

所以派出韩副主席作为特使出席典礼的结果，既是对美方的“投桃报李”，也不损害我外交利益，互相给面子。而今日韩正与万斯会见，对接下来中美关系的布局也很关键。

前文提到，懂王同时给大陆政府和台湾当局发邀请函，似乎本有恶心大陆的意思，但最后还是“保大陆而弃台湾”。

懂王不久宣布，“因天气寒冷”典礼转移到室内进行，只邀请部分贵宾入内观礼，室内空间不够，有些人不能进去，台湾由韩国瑜率领的“庆贺团”，就吃了闭门羹。

尽管把十几万手握门票的观众拒之门外，最大原因仍在于驴象两党斗争，但还是有维护中美关系考虑在里面里面的。

不过，特朗普对中方高层“盛邀”，以及将台湾当局拒之门外“友好行为”跟最近鼓吹重启对华贸易战、主张对华强硬的态度，似乎又大相径庭了。

又是对抗又是示好，他这是唱的哪出？

如今的美国，已经不是2017年的美国，两党内斗不休、美元通货膨胀、工业空心严重、民生艰难，陷入了特朗普第一任期未见的困境。

一时半会，特朗普很难腾出手对付中国。

特朗普第二任期接手的美国，是个烂摊子，有太多事情需要他上任后处理。再加上拜登临走前给他埋的财政、人事等“雷”，更让他头痛不已。

而且，特朗普承诺要尽早结束俄乌冲突，光靠美国是不太可能达成的，必须要有中国参与。

这么一看，对内，特朗普希望缓和对华关系，希望中国在“安内”过程中别来搅局；对外，在解决俄乌冲突、和他所说的贸易平衡等国际问题，他也希望能拉上中国帮忙。

至于喊着对华重启贸易战等狠话，估计还是憋着要兑现，只不过先打“中美友好”牌，装个样子，但狐狸尾巴终究是藏不住的。

他打算在就职当天与日、印、澳召开四方外长会议，摆明了是想继续拉拢亚太盟友，继续对抗中国，“假友好，真对抗”的意图不要太明显。

这也是本次通话，我们方面态度相对谨慎的另一原因。

很多人对懂王是有期待的，期待着中美关系重回蜜月期，特别是在这几天的小红书对账后，鼓吹“人类命运共同体”声音更是多了起来。

仿佛中美马上就能握手而和了。然而，高层显然记得懂王第一任期的变脸，既不喜也不悲。这也正是我们对美国应有的态度，你越上赶着热情，它越不拿你当回事。

热脸，多半只会贴了冷屁股，不如冷淡一点。没有过高的期待，就没有过多的伤害。
