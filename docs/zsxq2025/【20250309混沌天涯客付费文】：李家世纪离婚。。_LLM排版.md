
# 李家世纪离婚

打鸡血是一个词，也是一件事。作为事儿，发端于上海。大约上个世纪六七十年代，上海出了一位叫俞昌时的神医，发明出了鸡血疗法，将公鸡的血抽出来，注射到人的静脉里，有奇效。具体可以治疗高血压、偏瘫、不孕不育、牛皮癣、脚气、痔疮、感冒、腹泻、失眠等等，效果随着病毒的进化自动更新。上海领风气之先，很快风靡全国，随着参与的医生和患者越来越多，又演化出新的法门：要大公鸡的血，要没跟母鸡睡过的处公鸡，纯白色的“白衣天鸡”最好，但保持新鲜更重要。很多医院里都涌现出患者拎着活蹦乱跳的公鸡排长队的场景。

在上世纪很长一段时间里，上海是亚洲第一大城市，贸易中心、金融中心、文化中心。上海的文化是引领全亚洲的，尤其是电影，出品过许多家喻户晓的电影明星，其中有一位二流女演员，许多年后弄出8部样板戏，照样风靡。但注意，那些戏的风靡，只是局限于大陆地区了。不仅放在亚洲无人问津，即便是同源同种的港澳台地区，也只笑嘻嘻当戏看。那时候接棒上海，以文化尤其是电影引领亚洲走向世界的，换成了香港。即便是搞样板戏的那位演员，闲下来也爱跟家人一起看香港电影。巧合的是，那时香港电影圈的一股潮流，跟上海医学圈的那股如出一辙，也在打鸡血。

鸡血的主题是，霍元甲系列。霍元甲是一个人，也是一系列电影。作为人，他的武功比马保国老师高，但也高不了太多。他跟西洋大力士的比武，在真实记载里只有一次，还因为比赛规则无法取得一致被取消了。作为一部电影，《霍元甲》就太成功了，被翻拍了不知多少次。围绕着霍元甲还有许多衍生，在这一系列的电影和电视剧中间，扛鼎之作是1972年的电影《精武门》。由李小龙扮演的陈真，将印有“东亚病夫”的牌匾一脚踢飞的场景，比同时期拎着公鸡到医院打鸡血更管用。不同于霍元甲确有其人，陈真完全是虚构的。由谁虚构？虽然后来也多次演过陈真的甄子丹委员将陈真与李小龙划等号，表示陈真是李小龙开创。但其实创造陈真的另有其人，此人乃作家倪匡。

倪匡这个人，生平是很有意思的。作为出生于上海的宁波人，他在上海长大一直读到中学，中学期间被打了鸡血，于1951年辍学参军，随后成为一名军官参与社会大建设。在苏北修过水库，到安徽挖过淮河，最后志愿报名到最艰苦的地方，去了内蒙古呼伦贝尔大草原，要开辟农场种水稻。没有不能干，只有不敢干，敢叫日月换新天，倪匡的做法符合那个时代。只是在内蒙古期间，种水稻失败的他熬不住零下几十度的严寒，拆下农场的木桥生火取暖，被定为重罪。关押期间，他暗通关节逃了出来，从内蒙古一路南下直到香港。

在香港，他开启了写作生涯，产量之高堪称华语作家之最，除了小说，他还写了四百多部电影剧本，其中最出彩的就是《精武门》里的陈真，卖点即是打鸡血。从“打鸡血"到“打鸡血",可以看出倪匡的变化。十年前在上海读中学，是他被打了鸡血去改天换日；十年后逃到香港写小说，是他给笔下人物打鸡血从而带动着观影人群也跟着打鸡血。他的角色，跟同时期上海的神医俞昌时差不多。

由此可以得出规律，被打鸡血者，容易得肝病；擅打鸡血者，可以爆大名。而频频在打鸡血时期卖出精心饲养大公鸡的，则是超人李嘉诚。

离婚需要冷静期，这是一个道理，也是一项法律。作为法律，是从2021年1月1日开始实施的，自婚姻登记机关收到离婚申请之日起必须要冷静30天。这不长不短的30天，可以挽救许多因一时冲动要破裂的婚姻。剩下的离婚，都会是预谋已久，深思熟虑。

想与世界离婚，就是川普经过深思熟虑，上台后要抓紧做的事。打着美国优先的旗号，不停找茬、使绊、加关税，扰动世界各国纷纷跟进。墨西哥、加拿大惶惶不安；欧盟密集开会，酝酿反击川普之余，还推出了8600亿美元的重整军备计划；东边的日本也挨上了，川普对着"美日安保协议“一通挑刺，日本感觉到了美国不靠谱，正加紧增加防卫预算。

这是一股逆全球化的离婚潮，引领这股潮流的，不仅只有川普。全球化分为多个层面，贸易的全球化，信息的全球化，文化的全球化，科技的全球化……不可否认的是，除了贸易全球化是我方想要的，其余的几项，我们都在自成一派。信息全球化，不用多说了，打开手机，我们用的app 和世界其它地方用的app 截然不同，信息离婚已经大功告成。文化全球化，我方的票房神作，在其它地方溅不起几点水花，而在其它地方拿了大奖的，哪怕是金马奖，也已经越来越难登陆我方的院线，电影也离婚了。科技的全球化，从汽车到手机到Al大模型，我们用我们的，世界用世界的，互相借鉴，各自安好。唯有贸易，中国制造，仍是需要出口创汇的。

也就是说，离婚是我想要的，但需要留个出口。可是川普来了，对准的就是贸易全球化。除了已经两次加关税，川普还使出一记狠招，废除800美元以下小额进口商品免关税制度。这一招直接将擅长化整为零打游击的贸易商重拳击倒。我看到的消息，广州番禺区的希音 (Shein) 村，今年订单大减甚至归零，车间空荡荡一片。

川普才上台40天，接下来还有4年。多长的离婚冷静期也难拖过去。此情此景，李嘉诚的卖出正逢其时。涉及23个国家的43个港口，一口气卖给了贝莱德财团，包括在巴拿马运河两端的港口。消息出来后，属于李家的港股00001号长和股价连涨两天，涨幅达到35%。港口，是服务于贸易的，如果全球贸易面临大退步，则这些港口将入不敷出。

很多人在骂李嘉诚是跪了。川普说要夺回巴拿马运河，持有运河经营权的李嘉诚就要赶紧卖出给美国财团。这种跪舔的方式，跟台积电要加码在美国投资1000亿美元如出一辙。但实际上，李嘉诚只是退了。面对这一场世纪大离婚，无论是香港第一股还是台湾第一股，要么跪，要么退，半点鸡血也不敢有。港资台资的历史使命都过去了。所有的鸡血，都要留给“三驾马车”里的消费，今年的大会报告，消费一词出现了32次，为最高。大力提振消费，被列为工作任务之首。

大力提振消费，全方位扩内需。别无它法，一管又一管的鸡血，就要不停地打进去。
