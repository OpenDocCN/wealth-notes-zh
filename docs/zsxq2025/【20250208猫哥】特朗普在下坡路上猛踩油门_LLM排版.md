
# 特朗普在下坡路上猛踩油门

这段时间一直在观察特朗普雷厉风行的动作，发现了一些有趣的事情。我们首先从一件小事说起。特朗普之前宣布对华商品加征10%的关税，从2月4日生效，其中就包括跨境电商的快递包裹，在此之前，只要货值低于800美元的跨境电商快递包裹是免税的，但这一次特朗普要对这个也要加税了。

但这个事出现了反转。在2月4日加征关税生效24小时之后，白宫突然宣布暂停对跨境电商的快递包裹征税。

那么究竟发生了什么，让特朗普出尔反尔？是中方施加了压力导致美方暂停对跨境电商包裹征税吗？不是。是特朗普这个征税决定让自己遇到了大麻烦。

现在美国每年要接收14亿个跨境电商的包裹，其中来自中国的就占比高达80%，也就是说来自中国的包裹超过10亿个，换算下来，每天就要收接近300万个包裹。过去因为对跨境电商包裹是免税，所以这些包裹到了美国海关是走“无申报通道”直接秒过，现在要征税就比较麻烦了，海关与边境局要核对包裹货值，然后给收件人寄账单——包括加征税额多少，以及人工处理这些事务所产生的人工费多少（据说每单包裹的人工费就高达17美元）——相比过去无申报通过工作量激增何止十倍。

现在美国海关与边境管理局完全没有能力处理每天高达300万个包裹的这些加税事宜，更何况特朗普对联邦机构还在大裁员，搞得人心惶惶，在这种情况下如果还要坚持对跨境电商包裹征税只会导致一个结果，就是美国海关口岸全部瘫痪！这就是特朗普在对华包裹征税生效24小时不得不紧急叫停的根本原因。

从这个案例，大家看出什么名堂没有？特朗普出台政策很多都是拍脑袋行为，根本就没有考虑具体的执行与可行性。甚至可以这么说，特朗普搭建的团队就是一个草台班子。

不仅对跨境电商征税是如此，对加拿大、墨西哥征税同样如此。为什么特朗普对墨西哥征收关税突然暂停？表面上原因是墨西哥做出一定妥协，根本原因与上述案例一样，根本无法执行。美墨边境与美加边境都是不设防的，征收关税的前提是有足够的力量防止走私，而这两国边境连非法移民都无法控制，更别说控制走私了。

所以，特朗普执政现在颇有魔幻三拍风格：一拍脑袋：有了！二拍胸脯：成了！三拍大腿：遭了！

现在问题来了，是什么导致特朗普执政朝令夕改？原因无它，特朗普搭建的是一个草台班子。搭建草台班子的原因就是特朗普与建制派有不可调和的矛盾，特朗普执政首先就是要对付建制派，要捣毁深层政府，所以无法任用（或者是无法信任）相对专业的官僚，只能选择政治素人，用外行去领导内行，因为不熟悉政务，所以才会出现朝令夕改屡屡犯错的情况。

比如，没有任何政治、军事、外交经验的卢比奥担任国务卿；比如，只有很短服役经历，最高军衔不过少校的担任国防部长；比如，没有任何公共卫生管理经验的担任卫生部长。与此同时，特朗普与马斯克还联手对联邦机构大裁员，大批有专业经验的公务员被勒令下岗，未来换上来的是大批没有任何经验的生手去处理联邦各种事务。

特朗普这样大刀阔斧的干不但会导致政府政令混乱、效率低下，而且早晚一定会出大问题。原因很简单，政府机构特别是专业事务的政府机构是为整个社会安全与稳定兜底的，一旦这个兜底的机构出现大面积问题，那么整个社会一定会出大问题，前段时间美国出现罕见的直升机与客机撞机事件就是最好的证明。

治大国如烹小鲜，这是我们老祖宗几千年就总结的经验与教训，现在特朗普治国完全是做江湖菜的路数，老美血条再厚，也经不起这么折腾。几年前，张召忠将军曾经有一个预言，大致是如果特朗普未来如果再次当选，很有可能特朗普在美国走下坡路的时候猛踩油门。现在我们看到的就是特朗普冲着一个错误的方向猛踩油门。
