
# DeepSeek的意义，是划时代的嘉宾供稿

DeepSeek（深度求索）的R1大模型，实现了爆红，登顶美国app下载排行榜。在《拜登的自杀式攻击》一文中，我们分析了一部分，今天再来分析一下它的划时代意义。

DeepSeek的第一大意义，是性能强悍。我试用了一下，和GPT的最新大模型o1相比，在中文回答上，比如写文言文，DeepSeek的R1是完全碾压o1的。

比如，让R1仿照《过秦论》，写《过美利坚论》、《过隋论》等等，虽然还不算完美，但已经能模仿得像模像样了，绝大多数学生都写不出。这个网上很多人发现了，且玩得不亦乐乎。而这一点，GPT的o1是完全做不到的。当然，中文语境有优势，其实是正常的，毕竟GPT的主阵地在英文。

于是我又切换到英文，让两个大模型，分别模仿特朗普的语气，写一篇演讲稿。可以明显看到，R1模仿得更像，会使用大量特朗普风格的词汇，看起来更有情绪，而o1尽管也有一些类似，但明显只是皮毛，干涩得多。

因此可以说，即使在一些英文回答上，R1也强于o1，最保守的说法，也可以认定，两者的性能是相当，对于GPT来说，就非常尴尬了。

因为GPT的大模型，动辄花费上亿美元，有的甚至花了10亿美元，而深度求索的几个大模型，单价都不超过600万美元，其中，最新的R1是557万美元。

在运营成本上，R1也比GPT低得多。以输出成本为例，R1每百万Token的输出，仅收费2.19美元，而GPT的o1要收60美元。

成本低这么多，关键是通过优化算法，而不是堆硬件。

过去的大模型输出成本高，就在于参数需要全部工作，分秒不停，打个比方，假设一个大模型有5000亿个参数，每一个参数，我们看作是一个灯泡。

传统的大模型，每一次回答，需要检索所有灯泡，才会做出回答，这就要求所有的灯泡始终处于工作状态，不能停下来，耗电量就高。

光电费就低不了。

而深度求索的思路，是只需要一部分灯泡处于工作状态(5%左右)，剩下95%的灯泡休眠，如果这5%的灯泡里，没有检索到合适的资料，就自动跳到下一个5%，前面5%关掉。

这样一来，始终有95%的灯泡处于休眠状态，电费就能省不少，最终成本也就低得多。性能强悍，价格还便宜，于是不少人干脆不续费GPT了。可以预见的是，如果美帝不加以阻止，不再续费的人会越来越多。

对Open Ai来说，这一手简直是要命。

其CEO山姆·奥特曼，明显破防了，直接在X上开骂，指责深度求索是“复制”的，笑死，看来他的心胸，也没有多么开阔。

敌人越是气急败坏，就越说明我们做对了。

R1本身是适配华为昇腾的，意味着大量的国产中低性能显卡，也能达成过去只有高端显卡，才能达成的效果。毫无疑问，国内显卡产业，将从此进入正循环。

英伟达是否向我们出售显卡，已没有那么重要。

我们知道，ai和显卡，是美国这几年的最大亮点，也是其国家战略的支撑点，美国针对中国搞的“小院高墙”战略，就是以Ai和显卡为核心的。

上世纪70-80年代，美国通过封锁苏联，严禁技术流入，并启动半导体产业革命，彻底拉开与苏联的技术差距，为冷战的胜利奠定了基础；

现在，借着Ai和显卡，美国试图启动新一轮产业革命，彻底拉开与中国的技术差距，奠定胜利基础。

而中国由于得不到高性能显卡，Ai技术无法进步，将如同当年的苏联，被困死于“小院高墙”中。

不仅美国，很多中国这边的人，也相信这一天叙事。只是深度求索的大模型，彻底击碎了这一套叙事，既然国产显卡也行，还怎么玩“小院高墙”？还有，我在《拜登的自杀式攻击》一文中说过，美国搞算力“种姓制度”，等于让出160多个国家的市场，给我们，当时还只是一种预测。

现在，随着深度求索R1的爆红，已经可以确定了。美国会陷入一种越制裁越尴尬的境地，原先靠Ai+显卡支撑的美股，也就开始暴跌了。

比如英伟达，昨天一天就跌了17%，但这不会是终点，后续还有很大的下降空间，堰塞湖的决堤，要开始了。问题来了，美国有没有办法阻止或者延缓呢？还是有的，比如炒作新的热点，前几天宣布启动的“星际之门”计划，就是典型。这个计划，将由Open Ai、甲骨文和软银共同投资5000亿美元。其中Open Ai负责运营，甲骨文提供技术支持，孙正义负责出钱（果然，美国人宰起日本人来，是不手软的)。

本质上，这就是个数据中心罢了，和中国的大量数据中心项目，没有区别。

但美国政府为了炒作，愣是叫它“星际之门”，这个项目有两大硬伤：一是5000亿美元太吓人，所以马斯克说，他得到准确的消息，项目只有不足100亿美元的资金；二是巨型数据中心，需要大量的供电，而美国的电网，尤其是项目所在地德州的电网，一向比较薄弱，能否承载得起还是个未知数。

预言一下，如果项目能成，在某些人眼里，可能又是一次工业革命。

除了“星际之门”，后续美国还会有更多类似的炒作，在无法拉开生产力层面差距的情况下，美国想维持堰塞湖不决堤，就必须持续炒作。

每年一次工业革命。

比如最近沙特就说，将向美国投资6000亿美元，这个数字相当于沙特全国GDP的60%。显然，小萨勒曼摸透了懂王的心理，也摸透了美国目前的叙事结构。

在深度求索取得重大突破后，我们已看到全新的局面，不可否认，在高性能显卡等硬件层面，我们距离美国尚有差距。

这就使得，如果美国全力按照DeepSeek的路线，进行发展的话，理论上可以重新建立优势，因为DeepSeek的参数极限，理论上不到7000亿个。

而美国是能做到万亿参数的，所以，我们还是不能太骄傲，只不过，这次突破意义仍然巨大。中国的工业基础，要比美国完整得多，意味着相关产业一旦进入正循环，追赶速度是很快的。和美国竞争的底气，我们是越来越足了。能不能赢不好说，起码是不会输了。
