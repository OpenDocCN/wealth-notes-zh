
# 抖音大健康直播获客实操讲解

## 主播案例介绍

以一位大健康轻运动主播为例，该主播2个月从小白到月收益10w+，之前是做线下养生门店的，0直播经验。

### 轻运动赛道优势

1. **对新手友好**：直播内容相对安全，风险性低，管控宽松，不易违规。
2. **动作留人**：对语言组织能力要求不高，画面和贴片即可吸引用户。
3. **简单易上手**：学会15-20组动作，会数拍子，能讲解动作锻炼部位即可开播。
4. **主播身型要求**：建议身材匀称，能通过画面修饰修长。

### 踩过的坑

1. **违规**：懂医学知识反而容易因使用专业术语导致违规。
2. **数据制作**：不会做评论灯牌等数据，需观察并记录互动点，参考同行话术。
3. **日收益提升**：从每天赚两三百提升到一天赚八九千，回本周期快。

## 直播账号设置

### 基础设置

1. **名称**：与行业相关，如“轻运动”。
2. **头像**：个人头像、卡通或风景。
3. **简介**：简洁明了，如“古法健身操践行者！关注我，跟我一起练起来！”。
4. **背景**：前期可不设置。

### 隐私设置

- 关闭同城展示
- 关闭推荐可能认识的人
- 关闭私密账号

## 养号流程

1. **模拟真实用户行为**：用不同wifi刷抖音，用手机套餐流量刷，用注册手机号通话。
2. **打上喜好标签**：评论、点赞、收藏同行短视频，停留2分钟以上刷同行直播间，点击小黄车/小风车。
3. **通过充值消费缩短养号周期**：充值10-30元抖币，购买便宜产品。

## 平台推流规则

### 抖音算法底层逻辑

1. **浅层数据**：点击、停留、互动、关注、灯牌、点赞、转发。
2. **深层数据**：成交密度、GMV、千次成交。
3. **不挂车直播间考核数据**：点击、停留、互动、关注、灯牌、点赞。

### 直播间八大流量入口

- 直播推荐、视频推荐、关注、同城、个人主页、商业化、搜索、其他。

### 重点打开直播推荐流量池

- 只有feed推荐流量打开，才能持续获取新流量，快速破人气节点。
- 起号阶段feed流量占比应达90%以上。

### 抖音推流逻辑

1. **初始流量**：开播5-10分钟推进来，小白主播难以接住。
2. **叠加流量**：考核直播间数据是否达标，做好点击、停留、互动、关注、灯牌、点赞数据。
3. **奖励流量**：数据好的账号优先获得下一波流量。
4. **在线阈值（人气节点）**：30、60、80、120、160、180、230、260、500、1000。
   - 突破260阈值意味着起号成功。

## 轻运动直播起号五大阶段

### 风控期

- **特点**：30分钟曝光>300，直播间有很多人头人。
- **目标动作**：破风控，让抖音认为你是正常直播间。
- **数据参考**：30分钟曝光>300则账号正常。

### 冷启动期

- **特点**：1小时曝光500～1000，1小时场观150～300。
- **目标动作**：打开feed流量池，破30在线人气节点。
- **数据参考**：直播时长60分钟，人均观看时长>45秒，互动率>2%，关注率>2%，灯牌率>0.6%。

### 增长期

- **特点**：1小时曝光2000～10000，1小时场观600～3000。
- **目标动作**：破60～120在线人气节点，建立精准人群画像。
- **数据参考**：直播75分钟，人均观看时长>90秒，互动率>3%，关注率>3%，灯牌率>0.8%。

### 稳定期

- **特点**：1小时曝光10000～30000，1小时场观3000～9000。
- **目标动作**：破160-260在线人气节点，巩固精准人群画像。
- **数据参考**：直播1小时以上，人均观看时长>120秒，互动率>3%，关注率>3%，灯牌率>=1%。

### 成熟期

- **特点**：1小时曝光30000～100000，1小时场观9000～30000。
- **目标动作**：破500-1000在线人气节点，优化人群模型，进行引流。
- **数据参考**：直播1小时以上，人均观看时长>120秒，互动率>4%，关注率>4%，灯牌率>=1%。

## 账号推流少的原因

1. **账号设置成私密账号**。
2. **账号有问题**：基础曝光低，违规记录。
3. **人、内容、场景有问题**：
   - 主播状态差，内容不吸引人。
   - 内容选择有问题，抄袭或先抄袭再超越。
   - 场景搭建有问题，贴片不够吸引人，场景不符合内容。

## 破解推流少，实现层级跃升

### 解决账号问题导致的流速低

1. **不要频繁切换地区登录**。
2. **不要一证多实名**。
3. **账号连开**：测试高曝光时间。
4. **更换账号**：无法解决则更换新账号。

### 工具的正确利用

1. **福袋的正确使用**：在接近流量节点时发放，节日大促时少用。
2. **红包的正确使用**：与福袋用法相同，可叠加使用。
3. **鱼塘的正确使用**：辅助达到数据考核标准，实现羊群效应。

### 调整人、内容、场景

1. **调整主播状态**：多看、多听、多练，保持好的状态。
2. **调整直播内容**：选择市场认可和同行使用的爆款内容，内容要有共鸣、痛点、钩子。
3. **调整场景**：贴片吸引人，场景搭建符合内容。
   - 轻运动场景可选择宽阔适合运动的地方（如客厅）。

## 注意事项

1. **开播时间要固定**。
2. **起号期间不要频繁换内容**。
3. **保持每场直播都在1小时以上**。
