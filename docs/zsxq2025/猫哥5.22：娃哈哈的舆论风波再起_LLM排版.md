
# 猫哥5.22：娃哈哈的舆论风波再起



近期，娃哈哈风波再起。
事情的缘起是，有人发现，娃哈哈的纯净水是由
今麦郎代工的。随后，娃哈哈发文承认了，但表示，
双方的合作已经在4月份终止。
原因是“抽检不合格”。
这很快引发了一大波质疑，毕竟代工有弊端，很
容易导致质量不稳定，你自己都说了“抽检不合格”
。
还有一部分人觉得，既然都是今麦郎生产的，我
干嘛不直接买今麦郎。
代工很常见，是小事，但其背后体现的问题，却
不小。

一朝天子一朝臣，每个老板，都有自己的班底，
宗女士接班后，开始不断更换娃哈哈的老臣。
吴建林、余强兵、潘家杰、张晖等老臣，退出了
娃哈哈集团董事会，替代者，是叶雅琼、费军伟、洪
婵婵、王国祥等多人。
监事名单也有变化，孔沁铭、许思敏等新人，接
替了蒋丽洁、郭虹等老臣。
此外，比较重要的新人，还有严学峰、祝丽丹、
丁秀娟等。
宗女士长期执掌“宏胜饮料”，自然而然，她的
新班底，主要来自“宏胜系”，典型的如，洪婵婵，
是宏胜饮料的财务部负责人，钱袋子。
严学峰、祝丽丹、丁秀娟三人，都是原“宏胜饮
料”的，其中严、丁二人，都关联着多家公司。

叶雅琼，虽然不是“宏胜系”的，但2021年后突
然被宗女士火箭提拔，短短几年，就从一个分公司党
委书记变成了娃哈哈政务中心总监。
我们知道，老臣们肯定不愿意退出，所以对于宗
女士的一系列举措，反对意见很大。
而宗女士的办法，就是削弱他们，怎么削弱?
还是以本次代工风波为例，娃哈哈不是没有工厂
的，娃哈哈一直以来，走的都是重资产模式，工厂遍
布各地，理论上就没必要找人代工。
官方回应表示，是2024年后需求飙升，娃哈哈营
收突破了700亿，自己做不过来，才找的代工。
但这个理由，明显说不通，因为娃哈哈的营收，
早在2013年就突破了700亿，当时也没说做不过来呀，
## 怎么今天就做不过来了?

更诡异的是，娃哈哈一边找今麦郎代工，还一边
大量关停原有的工厂，总数达30家。
看起来，宗女士要走轻资产模式，彻底改变娃哈
哈的重资产模式。
但事情，显然没那么简单。
被关停的工厂里，大部分都与杜建英存在关联（
在其中有股份或担任职务），杜建英是老臣中的老臣
，1991年加入娃哈哈，宗庆后在世时，杜是其左膀右
臂。
关于她的事情，很多人都知道，她与宗女士存在
不和也并非秘密，自然成了开刀的典型。
至此，一条清晰的脉络出来了。
宗女士需要构建自己的班底，需要赶走老臣，老
臣们不愿意走，那就只好借着走轻资产路线的名义，
不断关停娃哈哈的原有工厂，打击“杜建英们”。

当你们的资产、职位都没了，还有能耐和我对抗?
当然，老臣只是原因的第一层，第二层，是世界
派和本土派的碰撞。
宗女士的认同，更接近“世界人”，这一点不是
秘密，比如曾经半开玩笑地说，要效仿李嘉诚把公司
搬到国外，也就是走“世界化”路线。
她说这番话的时候，是2013年。当时正值李嘉诚
逃离风波，官媒还发了一篇文章《别让李嘉诚跑了》
，宗女士却偏偏要在这个时候，说效仿李嘉诚把公司
搬到国外。
原视频那语气，也不像开玩笑。

但我们知道，娃哈哈的主体向来是在国内，哪怕
宗老爷子活着的时候，已经在境外注册了一堆皮包公
司，国内的主体也是搬不走的。

究其原因，在于娃哈哈的模式，就是需要在地化的，你跑到境外，很多事情就没法运作。
重资产模式，就是会带来大量的本土派，再加上
老爷子活着的时候，是很强调公司要有人情味的。说
白了，就是不能太苛刻，该尽的责任要尽。
这样的弊端，就是效率不如996，卷不起来，但好
处是环境相对宽松，有利于保持团队稳定和凝聚力。
同时，论待遇，很长一段时间，娃哈哈在行业内
还是相对不错的。
所以，娃哈哈有许多工作了几十年的人，他们确
实是愿意一辈子跟着老爷子的。公司搬走了，他们怎
么办?你宗女士有大把钱，随时可以到世界上任何一个
地方生活，但多数人没那个能力。

现在大量裁撤原有的工厂，走轻资产模式，顺便
把资产转移出去，势必损害他们的利益。他们便和老
臣们联合起来，抵制新路线。

一些人说，你怎么抵制改革呢?改革不好?
而我想说，是人都会有老的一天，今天你35岁的
上司被降本增效，你不是兔死狐悲而是幸灾乐祸，觉
得自己有机会上位了，那么明天呢?
等你也被降本增效的时候，就笑不出来了。所以
，不是不能改革，是不能这么乱来。别的不说，大量
裁撤工厂，员工补偿给了多少?
如果宗女士想继续“世界化”，大概率是一条死
路，且不说会被内部抵制，从市场角度，也是行不通
的。
首先，许多人支持娃哈哈，是看在情怀的份上，
是看在老爷子的份上。比如2024年，娃哈哈的营收能
在多年缓慢下跌后，重回700亿人民币大关，就是情怀
在起作用。
强行效仿李嘉诚，想做“世界人”，势必砸了基
本盘，这里面的损失可就海了去了。
其次，饮料行业属于资本密集型，没有太多技术
性壁垒，一旦“世界化”，没有了本土保护，以娃哈
哈的体量（年销量堪堪100亿美元），很难是外企的对
手。
当年老爷子活着的时候，为什么要打爱国牌?
原因之一便是，纯市场化竞争，是竞争不过外企
的，人家的资本远比你雄厚，可以砸钱换规模，卷死
你。
这也是许多国家的企业，在成长期的要求。
你打了爱国牌，获得了保护，那么作为交换，你
得在就业、投资等方面，给个底线，这就是娃哈哈多
年来，赖以生存的模式。
如果你当了李嘉诚，把公司搬到了境外，可就没
有保护你的必要了。真以为你是西方企业的对手?就算
是境内的农夫山泉等对手，也能趁机分一杯羹。
届时，倾覆并非不可能。
今天，许多“精英”仍未意识到，我们熟悉的那种
全球化已经终结，哪怕他们常年生活在国外，也依
然固执地认为，特朗普现象只是偶然事件。
世界很快就会“回到正轨”。自己可以继续当“
世界人”，此处不留爷，自有留爷处。
对此我只能说，不撞南墙不回头，是人性。
过去数十年的全球化红利，让他们日子太舒服了
，躺在温柔乡里不愿出来。
