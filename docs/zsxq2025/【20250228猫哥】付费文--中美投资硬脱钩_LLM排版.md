
# 中美投资硬脱钩

最近，在 DeepSeek 的带动下，港股恒生科技指数走出了一波小牛市行情，原因有很多，其中之一就是海外资金正在跑步进场抢购中国科技股。因为港股是以美元计价，香港是我们的离岸金融中心，外资看好中国创新，要投资中国科技公司，港股就是海外资金的到岸码头。但是，这又让川普政府坐不住了，打算在中美投资领域搞点事儿。2月21日，白宫网站发布一份名为“美国优先”投资政策的总统备忘录，宣布将调整美国投资政策，进一步限制与中国的双向投资。这份备忘录一共提出了12条具体的政策举措，逐一啃起来比较繁琐，这里我们将其中的几个核心内容向大家分享。

首先，这份备忘录明确表示，美国欢迎盟友和伙伴国的投资，尤其是那些能帮助美国发展高科技产业的投资。比如人工智能、量子计算、生物技术等领域，并且打算通过外国投资委员会来加速审查盟友的投资，让他们更容易进入美国的高科技行业。但是，这个盟友和伙伴国不包括中国，甚至明确指出“中国的钱不能碰美国的关键技术和资产”。而且对于盟友和伙伴国的投资还有个前提，那就是盟友的投资不能跟对手国家合作，这个对手国家不说大家都懂，主要就指中国。也就是说，如果你是美国的的朋友，欢迎投资，但如果你跟中国走得近，搞中转、搞合作什么滴，那就不行。不仅如此，在川普1.0时代或者拜登时代，我们企业投资还可以通过签署“国家安全协议”来缓解审查压力，但现在这条路也被堵死了。因为这份备忘录打算执行“推定拒绝”策略，也就是说，除非有特别充分的理由，否则中国企业的投资申请很可能会被直接拒绝，以后不会再搞那些复杂又费时的“缓解协议”，而是直接采取更严格的审查措施。当然，这其中也包括加强在美上市的中国企业的财务监管。在2020年12月，美国国会通过了《外国公司问责法》，规定如果在美上市的外国公司连续三年不让美国上市公司会计监督委员会检查其审计报告，就会被强制退市。现在，川普政府显然打算重新操弄这个事，并对中国公司赴美上市进行严格审查。目的就是把中国企业赶出美国资本市场。

其次，备忘录里明确划出了中国的钱不能碰美国的哪些“命脉”，包括以下几个关键领域：其一是核心技术，涉及人工智能、量子计算、半导体等。其二是关键基础设施，涉及能源、交通、通信等。其三是个人数据，涉及美国公民隐私数据。其四是农业和食品供应，涉及美国的农田和食品供应链等。当然，最重要的是涉及军工产业方面。在这份备忘录中提到，中国正在通过各种方式，系统性地引导和促进对美投资，目的是获取美国的尖端技术、知识产权以及在战略产业中的影响力。包括通过“军民融合”政策，让企业和研究机构为军方服务，扩大军事影响力。通过在美股上市、游说基金公司、吸引投资等方式，从美国市场拿到大量资金，间接支持中国的军事发展。换句话说，美国的钱正在帮中国军队变强，对美国本土和全球美军构成了威胁。除了限制中国对美投资外，川普政府还打算阻止美国资金流向中国军工产业，这里不仅仅指的是传统军工产业，还涵盖了包括半导体、人工智能、量子技术、生物技术、高超音速、航空航天等。这些领域都被认为与中国的军民融合战略密切相关。具体手段可能包括资产冻结、经济制裁等。甚至连美国大学也要同步，包括大学资金和开放大学资源。

从上述3个核心内容看，其实总结起来就是三句话：盟友和伙伴可以来投资，带动美国发展，美国欢迎。中国的钱不能碰美国的关键技术和资产。保护美国投资者的钱，不能让它支持中国发展军事以及相关联的上下游产业。

相比于拜登时期对中国设立的高科技壁垒和投资禁令，川普政府这份投资备忘录有雷同点，也有区别。雷同点就是摒弃了对华单打独斗策略，有点拉帮结伙的味道。区别点就是：拜登搞的是集中在高科技领域的精准限制，而川普范围更大。拜登留了中美企业合作的口子（比如通过第三国市场投资，成立合资企业等，勉强绕开限制），但川普简单粗暴的提高审核标准，基本相当于一刀切。总体而言，这份总统备忘录对中国相当强硬，完全是冲着中美投资硬脱钩方向去的，也把很多时政分析砖家搞得精神分裂了。为啥？因为就在白宫发布这份总统备忘录前2天，川普还信誓旦旦的说与中国达成贸易协议的可能性很大，估计很多博主文章还没写完，这下又来剧情大反转。当然，这很特朗普!个人认为，虽然它来势汹汹，但是我们也不用过度恐慌。首先，从拜登再到川普2.0，类似的的东西太多了，并没有新意。只是双方路径不一样而已，拜登更依赖“巧实力”，特朗普更信奉“硬实力”，拜登试图用技术壁垒拖慢中国发展，特朗普则想用经济脱钩重塑全球产业链。两者殊途同归，这份备忘录更像是一盘多加了点佐料进去的“回锅肉”。其次，这类备忘录并不是什么法案，也不是行政命令，只是一个川普施政的思路和指南，像川普这类充满了不确定性施政，我们目前更应该听其言，观其行，让子弹再飞一会，做好斗争的准备。第三，不管川普政府最近和我们在博弈过程中吃了什么憋，还是用老一套凭空捏牌的手段制造筹码，我们的态度就应该如同王外长那句话一样“他强任他强，清风拂山岗，他横任他横，明月照大江”。最后，我们也要充分认识到，虽然最近中美之间有一些“缓和”的信号，但川普的“美国优先”政策不会轻易改变，无论是盟友还是对手，都得在这个框架下行事。对于中国来说，如何在这样的环境下继续推动经济发展，将是一个巨大的挑战。同时也预示着，这场中美博弈还在继续，大家不要因为一些过程中的积极因素而放松警惕变得乐观，中美关系未来的走向仍然充满巨大变数。
