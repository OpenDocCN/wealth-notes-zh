
# 三增加两减少：就业市场会这么变

今天，我们说说2025年的就业趋势。伴随着开学季，2025年的春招已经拉开帷幕。经常有文章为了吸引流量，动不动就说今年是史上最难就业季。但是，你千万别被这些消息带偏了。实际就业变化，可能跟很多人想象中不太一样。

比如，以前总有人说，大不了卖保险去。这话现在不能说了。因为过去几年，离职率最高的行业之一，就是保险业。2019年，中国有912万保险从业者。而2023年底，这个数字大概是281万，减少了69.2%。每10个保险从业者中，有7个人离职。你还觉得这份工作那么好干吗？

再比如，有人说大不了送外卖，也能月入过万。这话现在也不能说了。现在一个骑手要想月入过万，首先必须身处北上广深其中的一个城市。其次他必须是平台的专送骑手，不能是众包骑手。而专送骑手是不能拒单的，必须风雨无阻，24楼没电梯也得准时送达。这个门槛可不低。

保险行业和外卖行业的 变化都 很值得留意，以后我们会专门讨论。

再比如，关于薪资，网上有句调侃，说小红书上人均月入10万元，人均库里南。但实际情况呢？前几天，《经济参考》的主理人马江博老师专门做了计算，计算的依据是2024年国家税务总部的数据。

2023年，我国无需缴纳个税的人员比例超过70%。也就是说，在所有拿工资的人里，有70%的人每月工资收入不到5000元。而剩余不到30%的实际缴税人员中，又有60%以上，适用3%的最低档税率，也就是工资不超过8000元。而能够达到10%以上个税税率 的，也就是年收入9.6万以上的，占12%。说白了，假如一个人年收入达到10万元，那么他差不多跑赢了全国90%的劳动者。

同时，过去20年间，我国GDP的绝对增长率达到829%，而人均工资性收入的增长率是629%。而经济增长的主要蓄水池之一，是房地产。也就是说，这些经济增长中有不少体现在房地产价格的上涨上。

这就意味着，即使一个人收入较高，假如他生活在北上广深这样房价高的 大城市，他的消费压力可能反而更大。

关于就业的机会与收益，咱们不能只看个人体感，也需要参考客观数据。

而从数据上看，就业市场的情况可以用一句话概括。短期存在挑战，长期依然乐观。

我们先看短期的挑战。今年1月中旬，国际劳工组织发布了《世界就业与社会展望：2025年趋势报告》。这份报告的核心结论是，全球经济增长正在放缓，导致劳动力市场难以完全复苏。2024年，全球就业缺口达到4.02亿，失业率保持在5%左右。根据相关机构的预测，这个趋势可能还会延续到2025年。

但是，这并不意味着就业市场会持续低迷。我们再看长期的乐观。就在今年1月底，世界经济论坛在瑞士日内瓦发布了《2025年未来就业报告》。结论是，到2030年，全球就业市场将迎来巨大变革。

一来，随着技术进步、人口结构变化、经济压力等因素，全球将新增1.7亿个工作岗位。二来，全球22%的就业机会将面临调整，9200万个工作岗位被替代。

注意，这两个数字的差值是，1.7亿减去9200万，是正的7800万。也就是说，未来5年，全球将净增7800万个工作岗位。

就像前面说的，短期看，就业确实存在挑战。但长期看，也有乐观的一面。

那么，回到个体身上，我们应该怎么选择，又应该培养哪些能力呢？

在世界经济论坛发布的《2025年未来就业报告》中，专门预测了2030年之前，就业人数增长最快和减少最快的工作。

其中，增长最快的工作可以分为这么三类。

第一类，是由技术进步带来的工作岗位，主要跟人工智能、机器人以及可再生能源等领域相关。包括软件和应用开发人员、智能制造技师、新能源设备工程师等。这也是增长空间最大的一类职业。

第二类，是由人口结构变化带来的职业机会，主要包括护理类职位和教育类职位。一方面，老龄化加剧，专业护理人员的需求会大幅增加。另一方面，随着全球范围内对教育的进一步重视，教师等职业也会迎来一轮增长。

第三类，是一线基础行业的蓝领工人。包括农业相关的农场工人、物流相关的轻型卡车司机或送货司机、建筑业相关的建筑架构工和技术工人等。世界经济论坛的报告认为，这些是维持现代文明基石的工种，而伴随着上一代蓝领工人的退休，再加上年轻一代倾向于追求更高的教育，这就导致蓝领劳动力市场出现缺口。当然，随着新技术普及，这些工作的实际内容可能会发生变化。比如货车司机，可能会从自己开车，转变成为远程技术驾驶。
