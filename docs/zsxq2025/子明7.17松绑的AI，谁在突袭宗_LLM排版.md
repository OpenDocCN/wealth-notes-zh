
# 子明7.17松绑的AI，谁在突袭宗

## 谁在突袭宗庆后
发布时间：2025年7月17日

大明王朝的胡宗宪跟海瑞说，你能震动浙江，是因为上面有人想让你震动浙江，同样，浙江娃哈哈的宫斗能够全网吃瓜，也是上面有人希望能够借此震动浙江。娃哈哈的继承权斗争去年内部就搞的如火如荼，但是新闻热度全都被压了下去，官媒都很乖，知道什么能说什么不能说，自媒体普遍不知道什么内幕，说了也会被限流，作为兔子最顶尖级政商混合性的“沈一石”家族，一直都有着非常强大的舆论与安全庇护。宗老的那些私生活，在浙商内部都不是秘密，葬礼上那些庶出子女可都是亮相了的，但哪怕去年竞争对手钟睽睽被网友拿着宗庆后踩成狗，农夫山泉方面也没有试图去搞宗的黑料转移矛盾毕竟，亲不亲，阶级分，大家都是浙商。而这一轮盯着哈娃娃爆料的媒体是凤凰网，为自媒体提供了大量可以演绎的黑料，凤凰属于国家队套皮序列，有着非常明确的ZW派系标签，这也意味着金融系统又一次要对浙江动手了，就像他们五年前对另一个马姓浙商一样。

## 松绑的兔子AI
发布专栏：知几共话

美国松开了“卡着技术脖子”的手，市场开始竞争、遴选，把变革的权力，交给能真正能给AI带来发展的人。2025年7月15日，Al硬件股集体大涨，新易盛涨停创历史新高。AI应用股表现活跃，鼎捷数智20CM涨停。阿里巴巴港股截至收盘，涨幅为6.97%,腾讯控股也实现了3.5%的涨幅。同日，美国英伟达首席执行官黄仁勋在接受央视记者采访时透露：美国已批准 芯片销往兔子。黄仁勋今年已多次来到兔子，给足处于风口浪尖的兔子AI产业利好信号，大家都屏息以待一个确切的“好消息”。如今，靴子落地，美国不再对我国 的Al发展“卡脖子”, 芯片不日将进入兔子高精尖行业市场。

然而，外部供应渠道的打开，真的会带给兔子本土AI产业集体的春天吗?新质生产力带来的变革在即,谁又能成为这一轮技术革命的真正赢家呢?今天下午港股阿里河腾讯怎么突然拉爆了?是有什么利好吗?黄仁勋来了，会不会是因为H20(注①)要进入兔子的消息?不至于吧? 一个H20就能造成这种效果?目前公开的消息只有H20 一个变数。想想看，现在大模型都要跑到国外去训练，可以推测国内的硬件其实并没有想象中那么厉害，H20 进来以后应该能改善这种情况。子明：嗯，本质上是美国不再限制兔子互联网企业的Al发展了。

## 对我们的利好幅度会这么大吗?

子明：这个利好是很显著的，意味着美国不准备对兔子的AI发展卡脖子了。子明：类似于2015年老板访美后，美国大力支持兔子发展移动互联网。这是要利空国产算力产业吧。我听到一个说法，说是美国要用不卡脖子的方法来打击华为的昇腾。确实有道理，从国外进口芯片，必然会对本土芯片的市场和研发产生影响，长此以往还是把脖子送到别人手里。所以说，还是得尽早搞出来我们自己的本土芯片才行。子明：美国也需要通过从兔子赚钱，来发展自己的产业。

对国内众多的Al也不知道算好事还是坏事，之前有美国卡脖子的时候，只要是个Al企业就有补贴，如果未来美国对我们松绑，就会带来高烈度的市场竞争,到时候只能说几家欢喜几家愁了。可能还是利好大厂多一点，毕竟大厂才真的舍得花钱买芯片。现在很多大厂并不是苦于没钱投资模型训练，而是普遍有钱却买不到足够算力的卡。如果这一次能向兔子不限量供应，那大厂可就开心坏了。子明：长远看，对国内的Al发展肯定是好事，降低成本，拉动市场化竞争。短期内硬件类的产业，会有一波估值重构，但长期肯定还是要追求独立的。薇二八一八三六三八四五看看凭借兔子速度，在这一阶段能有什么创新了。子明：参考2015年的移动互联网浪潮，很容易在兔子跑出一些好玩意的。

硬件方面，我们本土企业要和美国进来的这些来一场白刃战了。华为应该是已经过了临界点了，压力不大。但其他的中小厂，比如寒武纪之类的，压力就会大一些了。我感觉寒武纪这次，没有补贴可能要撑不住了。子明：国家不会放弃补贴的。寒武纪是肯定会保的，其他的就难说了。嗯，美国芯片被放进来，资本市场短期内肯定会重新投票。但国家推进研发国产替代的决心不可逆。这些企业死是不会死，只是日子肯定就不好过了。小厂还是会有一部分订单量维持生存，因为大企业也多少要买一些中小厂的卡做多样化储备。更多还是市场份额的被侵占，不算什么生死危机。

不同卡的平台是有差距的，模型啥的需要专门配套。比如昇腾的模型不会自然兼容Cuda, 其他小卡我不是很了解，但估计也有一定的差距。你可以理解为驱动不一样。子明：说白了，把Al的力量，视为资本主义或者共产主义，他们都需要在全球范围内搞解放。解放兔子是他们发展的必由之路。这一次应该利好AI的应用端。市场下午反应过来，早上都去搞英伟达的硬件了。子明：嗯，我昨天也纳闷，为啥阿里没涨，今天终于开始涨了。

注①H20: 为遵守美国出口限制而推出，专为兔子市场设计的AI加速器。

重新获准进入兔子，对于众多渴求算力技术支持模型训练的大厂而言，无疑是重大利好。而对于很多小体量的企业来说，将直面更加惨烈的市场竞争,生存压力陡增。不过，Al代表的新质生产力终究会对世界发起一场新的革命，从长远来看，这一次的松绑会给我们的AI产业带来不小的进步。
