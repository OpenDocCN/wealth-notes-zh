
# 牛市的逻辑

牛市是怎样养成的。

隔夜，中美的在斯德哥尔摩的经贸会谈圆满结束，双方就中美经贸关系、宏观经济政策等共同关心的经贸议题开展了坦诚、深入、富有建设性的交流，回顾并肯定了中美日内瓦经贸会谈共识和伦敦框架落实情况。根据会谈共识，双方将继续推动已暂停的美方对等关税24%部分以及中方反制措施如期展期90天。

九月初的纪念抗战和二战胜利的庆典，似乎也开始捷报频传。美股三大指数在盘中第七次创出了新高，但收盘时终究回落。市场还在继续地等待美联储今晚的议息结果。

特朗普的关税谈判进行得如火如荼，但是关税对于美国通胀的影响却并不明显，与之前市场的共识完全相悖。如果看中国的进出口数据，我们可以看到中国的出口遇强则更强，但是中国的进口增速自22年以来却一直放缓。除了2022年疫情时期的最低点之外，现在进口的增速相当于2015年泡沫破灭前以及2019年疫情爆发前的水平。简言之，内需依然力有不逮。

截至六月，美国经济已经连续第63个月扩张了，继续避免了衰退之命。美国经济长期连续扩张的平均时间大约是42个月，也就是850个交易日。这个长期平均的经济周期波长，与我的850天市场交易周期理论不谋而合，再次验证了我的量化模型对于经济和市场的周期的观察。最长的一次连续扩张，是2008年全球次贷危机之后，到2020年疫情暴发之前，时长持续了125个月。

尽管如此，美国的经济增速的台阶还是拾级而下的——从60年代的4字头，到90年代的平均三字头，再到如今的2字头。换言之，尽管经济周期的波幅因为美联储的积极干预而降低，增长的时长似乎在延长，但是经济增长的潜力却是在不断下降的。跟踪我的研究报告的读者，一定会想起那张中美劳动生产率周期的比较图。如今，美国经济的劳动生产率周期似乎进入谷底然后开始攀升。当下美国轰轰烈烈的人工智能革命，似乎将引领美国的劳动生产率周期走出谷底。而在这样新旧周期交替的阶段，经济对于贵金属的需求就增大，而贵金属的价格也将应运而涨。这是我们长期看多金银加密资产的基本面原因。

与此同时，美国市场的投机迹象愈发明显：仙股涨上了天，看涨期权的成交量飙升，美股市场的融资量也达到了历史的最高峰。当然，现在的货币发行量与2000年泡沫时期不可同日而语。但是如此高的融资量和如此狂热的散户的参与程度，导致美股三大指数的短期交易波浪阶段，都逐渐出现了见顶的迹象。

如果我们用同样的技术分析方法来观察中国离岸和在岸市场，我们可以看到类似的超买的现象。同时，我们却看到了市场的其它一些指标的表现一反常态。对于这些指标行为的解读，将决定我们下一个阶段对于全球各个市场的布局。

在前不久的一篇专属报告里，我讨论了市场对于美元极度悲观的情况：做空美元是大部分机构投资者的共识；做空美元的仓位非常集中，是各类主要资产类别里最集中的仓位；美元的技术上出现了极度超卖的现象。一般来说，当市场的情绪和仓位出现如此极度悲观的的时候，就是逆向交易的时机。我们当时认为，在96.xx这个位置上，美元应该至少有一波反弹，并对于风险资金上行的势能造成阻力。

过去六天，美元连续走强。不仅仅如此，美元和美国长期通胀的关系出现了一些微妙的变化。在图一里，我们展示了美元和美国长期通胀预期（图中逆序显示）的关系。在2021年四季度之前，我们的数据分析显示，当美国的长期通胀预期上升的时候，美元走弱，反之则反。这是美元和通胀预期的正常相关性。

这是由于通胀预期高企，市场对于美元购买力在通胀环境下被消磨的担忧导致美元汇率（比较价格）的下降。然而，在2021年美联储开始以历史最快的速度上调基准利率之后，美元的走势和通胀预期开始背离。这时候，美元的走势由市场对于美联储利率水平的预期一致——通胀越高，利率越高，则美元作为一个高息的资产类别的需求就越大，美元汇率就越强。这时候，美元变成了一个可投资配置的资产类别，一种可以获得投资回报的工具，而不仅仅是一种货币支付的手段。这个时候，美元和通胀预期的关系，是反常的。

直到2025年伊始，我们的数据分析再次观察到这两个变量相关性的正常化。也就是说，现在如果美联储的货币政策执行开始有效地控制通胀的预期，同时最近的数据发现，即便是今年以来的贸易战，也没有对美国的通胀水平有明显的冲击、美元的购买力并没有被大幅度地蚕食，那么美元和通胀预期的相关性再次回归正常化。

这时，通胀预期维稳，美元反弹。这就是最近这几天市场的交易情况。这个观察，也完美地解释了今年以来美元的弱势——因为市场担心贸易战带来的通胀将消磨美元的购买力，美元不再是一个无风险高息资产。如果美联储已经成功地控制了通胀预期，那么将有助于全球风险资产的表现。

中国的经济开启了下半年的旅程。然而，中国经济由于上半年出口的强势和消费的补贴，上半年的经济表现是超过了市场的预期的。然而进入下半年，如果真的如市场共识认知的那样，因为上半年形势很好因此有关部门暂时不需要大张旗鼓，那么下半年增长的压力可想而知。相信这也是最近大型工程快速上马，育儿补贴也正式铺开，学前免费教育应该也将很快地实施。

在工程消息的带动之下，上游的大宗商品价格开始上涨。虽然上周的下半周，商品的价格出现明显的回调，这很可能是因为交易收到窗口指导以避免上游价格出现过快过激的上涨而导致市场的潜在不稳定性。其实，尽管期货价格在飙升，它们的实际绝对水平还是远在历史高点之下。同时，现货的价格并没有出现大幅的上涨。现在的政策，就是要非常规和超预期，才有可能扭转经济里日益根深蒂固的通缩预期。

历史上，上游的通胀前景改变之后，就会逐渐传导到下游影响消费品的通胀预期。在下图，我们展示了猪周期和工业金属的关系。工业金属价格与上游制造业密切相关，而猪周期则可以作为下游消费的一个代理指标。在图中我们可以清晰地看到，上游的通胀是如何传导到下游的。有的市场人士认为，期货价格的上涨并不能影响下游的价格。然而，我们的数据分析显示这个上下游传导的关系是一直有效的。原因很简单——我们要改变的是预期，而不是价格。在预期发生改变之后，人们的需求自然会发生变化，从而改变实际的价格水平。

如果通缩的预期得以遏制，那么对于股票市场的表现也是洞若观火的。当下，市场对于去年国庆后的一日游行情依然记忆犹新。市场共识认为，到了3500以上，国家队就要减持。然而逻辑上，这种理论显然是站不住脚的。如果市场可以继续上行，那么为什么要在这个位置上减持？同时，如果国家队不减持将导致市场预期改变而继续上涨的话，那么国家队的策略就不应该如此路人皆知了。不是吗？

当然，如前所述，我们看到很多技术指标一如美股一样出现了阶段波浪交易见顶和短期超买的现象。但是超买可以用持续的超买来化解超买的情况——尤其是在市场预期发生转变的时候。我在前两篇专属文章里讨论了这种情况的可能性。这时，市场表现出来的特征，是市场的绝对高度不出现大幅的上升，但是市场内部的轮动加速，一些板块跌，另一些板块涨。香港市场最近就是这样的盘面。即便是昨天市场早盘出现了较大幅度的抛盘，创新药板块依然表现强势，并在下午交易中笑傲了绝大部分的跌幅。而我们的流动性指标，依然显示出中国市场内充沛的流动性。
