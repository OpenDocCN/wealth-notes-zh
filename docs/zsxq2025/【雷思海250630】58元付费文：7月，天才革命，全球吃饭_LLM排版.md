
# 7月，天才革命，全球吃饭

7月到来。一场稳定币的革命，从两朵小花，开始绽放，并将打开下半年全球市场的吃饭行情。

(稳定币革命，也可以叫天才革命，“天才”是美国的稳定币法案的英文缩写的中文翻译，Guiding and Establishing National Innovation for US Stablecoins Act》，简称GENIUS ACT。)

在6月的文字《史诗级的货币事件与未来》中，谈到稳定币，将给未来全球资本市场，带来史诗级的影响。这影响在6月开了两朵花，用真金白银，告诉了世界：极度震撼，史无前例! 这两朵花分别开在了美国与中国。

6月5日，美国第二大稳定币公司，Circle 以每股31美元的价格在纽交所上市。首日涨幅168%。两周后，股价最高达到320美元左右，12个交易日，股价最高翻了10倍，后来有所回落。

另外一朵，开在了香港。6月25日，国泰君安国际 正式获得香港证监会批准，牌照升级，可以提供虚拟资产交易，成为首家获此全方位虚拟资产交易服务资格的香港中资券商。受消息刺激，港股国泰君安国际，当天直接翻倍，大涨198%。

市场如此暴力，是认为国泰君安国际，未来将可以直接交易稳定币等虚拟资产。因为此前不久，香港刚刚公布了稳定币条例，8月生效。与关税战的那点利益相比，稳定币才是新未来所在，是大国金融未来的真正战场。这将是一个百万亿美元级别的市场，并将构筑未来10年，全球金融生态的新底座。能够参与稳定币竞争的国家，目前看来，只有中美欧3家。但未来的格局，将必然是一家独大。

稳定币挂钩稳定的资产，比如国债，现金等。举个例子，假设特斯拉发行1千亿美元的稳定币，那它就得始终在账户上，为此1千亿稳定币，留存出1000亿美元的资金，且这个资金要接受监管，定期核查。为的是，保证你手里的特斯拉稳定币，随时可以对付成等值的现金。这就叫100%的资产储备。

这个限制很重要，如果没有这个限制，那么稳定币公司，就与银行没什么不同了。1美元的现金，理论上就可以创造出无限多的稳定币来。那就意味着稳定币的大泛滥，也就不会有人要了。百分百等现金储备，是从货币端，保证足额兑付。

这还不够，稳定币必须上区块链。这是从资产端，保证稳定币的唯一性，不滥发。也就是说必须有强大的算力，作为保证，不然你的稳定币，依然没有什么约束，结果会没有人要。

算力是稳定币的核心，没有算力，没有资格发稳定币。有了强大的算力，你就可以发稳定币，投资者才认可，但这只是一面，还需要监管认可。

美国的《天才法案》，给稳定币增加了另外强大的背书：美联储直接入场监管。能让美联储监管的，只有银行。可见，稳定币公司的地位，至少等同于银行。

也就是说，美国通过两大手段，给稳定币以信用：算力+美联储背书。任何有信用的东西，都必然会创造信用。获得算力+美联储监管双重背书的美元稳定币，一旦流行，很大概率会成为一个巨大的信用货币创造核武器。就像当年次贷危机前的美国资产泡沫的推手，美元信用创造的初代核武器CDS 一样。

或许正是出于对未来灾难重演的预防，《天才法案》的监管条款，才明确规定，稳定币必须100%的现金等价物储备，就是拔掉了稳定币的信用创造的核能力。

但是，稳定币的信用创造，并非只有一面，它还有另外一面，是以反银行的方式，创造出来。

什么叫反银行的方式，我们知道，银行是通过借贷，来创造新的信用货币的，银行借给你1亿，这1亿，不管你给了谁，最终定会再存入其他银行，不管谁存，都等于创造出来了1亿的信用货币。

而稳定币则相反，它通过压缩市场的资金需求，节约出来巨大的信用货币。打个比方，以前20万亿美元生意，需要5.1万亿的信用货币才能周转。如今，20万亿美元的生意，只需要1000亿美元，就能周转。

那么，省出来的5万亿美元，就会从这个20万亿美元的生意需求中，摆脱了出来，那么，这5万亿美元，必然会流向其他资产。

这个比方，不是空想的。20万亿美元，正是开头所提到的美元稳定币CIRCLE 公司，2024年所发行的约1000亿美元的稳定币，在过去的交易额。

虽然CIRCLE 并没有直接创造，哪怕1分钱的信用美元，但是，它节约了本该被占用的信用美元，用反银行的方式，给市场增加了巨量的空闲信用货币。

这是一场巨大的革命。全球市场从未经历如此的场景。

6月份，纳斯达克创造历史新高，意味着稳定币这只信用货币的隐形猛兽，已经悄然出笼。

美国的稳定币如此，香港的稳定币也一样不例外。

6月29日，香港财政司司长表示：稳定币的法例将在今年8月1日生效，将推动发行人把稳定币应用推广至不同场景，助力解决企业经商及市民生活中的实质痛点。

如果香港的稳定币挂钩港币的话，等于间接挂钩美元。稳定币的展开，很快就是全球性的，而且初始可能主要从挂钩美元或者美债开始。

这意味着，未来一段时间，美联储即使不降息，美元的需求，也会逐渐被美元稳定币，所逐出。

注意，不是其他货币，是美元稳定币。因此，这一点也不会危及美元的霸权利益，反而会加固美元的霸权利益，增加全球对美债资产的需求。

当其他池子的水，被挤出来之后，并涌进更少的池子里时候，即使水的总量不变，也会造成某些池子里的洪水泛滥。

这是个常识。

所以，稳定币大发展的爆发期，美元大概率是要走弱的。

注意啊，走弱，不意味着失去霸权，美元强弱，是美元潮汐收割战略的一体两面。

当然，一项新认知的展开，是需要时间的。现在，全球还没有看到一家机构，谈到稳定币的信用反向创造功能。本人是第一个。

不过，稳定币的这个本质上与当年CDS 无异的超级能力，迟早会被市场普遍接受。它不仅影响资本市场，翻过来会推动全球地缘的演变。因为，美元突然变得隐形的泛滥了，那么，华尔街就要全球去买资产了，地缘关系就会走向缓和，只有和了，才能做生意嘛。

所以，美元贵的时候，世界地缘就不好，美元便宜的时候，美国与世界的关系就会好。想想奥巴马小布什那会，美国与东大几乎称兄道弟，要做全球的G2 伙伴。

那么，7月的市场判断，也就可以有一个大概了：

1、 美元。 美元中长期走弱的趋势，已经基本确立，7月，美元会继续弱势走低，如果稳定币的反向信用创造被市场逐步认知的话，不排除美元指数会试探95的位置。

2、 美股： 随着稳定币推进。美股流动性将会重新聚集，稳定币排挤出来的，其他资产池子的货币，将增援美股。如此，美股可能会继续反弹，试探新高，纳斯达克是主攻方向。

重点还是算力所主导的方向，因为稳定币的内核其实是算力，监管算是如虎添翼吧。

3、 中国资产。 美元走弱，实际上是双重共振的结果：稳定币反向创造出来的巨大信用货币余留，也就是节省出来的巨大信用货币，除了买美国资产之外，也必然会涌向全球。中国资产，必然是追逐的标的之一。华尔街很想在中国资产上吃一口。

因此，东西两大国的关税战，7月可能会有更好的消息。双方将比预期更快的走向缓和与妥协。

上周，懂王发了一个推文，称与东大已经达成了关税协议，看来并非虚言，老美应该会逐渐放弃，不合理的要价，台阶备足。

如此，则中国资产的外部空间，7月会走向更大的积极一面。

那么，7月份：人民币，大概率是继续缓慢的强势。此前所说的，6个月内见到7.1兑1美元的位置，还是继续维持。大A:7 月看多。上证估计会冲击3500一线，未来可能进一步上攻。

大金融的行情，大概率不会就此终结，但是要注意新方向：稳定币相关产业，是新炒作与潜伏的主要标的。

鉴于稳定币，最终依靠的是算力，因此，算力与芯片方向，以及由此衍生的人工智能、AI等等，7月依然可能是比较值得逢低关注的板块。

总之，7月是下半年，大A 吃饭行情的重点关注月份，可以积极一点。

在往期的文字中，多次谈到，如果大A的行情好的话，则神秘力量的托举行为，就会收敛。因此，神秘力量主攻的银行板块，可能就不是入场的好时候，而是逢高兑现的时候。

依然还要关注量能变化，如果两市总成交，维持在1.5万亿左右，则可以积极一些：稳定币，人民币国际化，跨境电商，大消费、大内需，人工智能与AI仍然可能会不时活跃。

如果不足1.5万亿，则确定性机会比较少，防守为主，可以考虑做神秘资金的主攻板块。与神秘资金共进退。

4、黄金。 黄金依然维持此前的判断。 4月23日，国际金价从3509的高点下杀。这宣告了关税核武冲击波的确定性交易机会，暂时走完。 这个结局，提前两天，在星球里做了判断。

黄金阶段性高点或许在3500一线出现，也就是说3500左右，进入分歧较大阶段，确定性大幅度下降。此后，有2种走法，1是区间震荡时日(具体时间也需要看到时的时空，数月或月余，长短都有可能),再冲新高；2是跌入暂时的熊途。具体如何，需要到时再看地缘与人心时空的方向。。不执着。。

目前来看，黄金依然处在走法的第一种，也就是震荡区间(数月的震荡)

7月来看，黄金突破新高的概率依然不大。因为现在市场可能重新关注稳定币，所可能给市场带来的新变化，不仅仅是虚拟货币领域，也包括可能给企业带来的新机会。

在新交易方向吸引资金注意的时空，黄金很难得道资金的过多注意力，因此，7月大概率，箱体震荡还在继续。

不过，需要注意的一点是，由于稳定币的逐渐扩展，未来黄金可能会成为其他国家的稳定币抵押资产，同时，美元流动性的溢出，也会增加对黄金的需求，因此黄金的长期向上，有了更多的支撑。
