
# 利益之争：K字签证，支持还是反对？

## K字签证概述

中国国务院在2023年8月修改了《外国人入境出境管理条例》，新增了一种K字入境签证，从10月1日开始生效。这种入境签证面向全球科学、技术、工程和数学领域的外国青年科技才，要求本科以上学历，要么毕业于境内外知名高校或科研机构，要么在这类机构从事相关专业教育、科研工作的，就可以在没有国内单位邀请和担保的前提下，申请K签，入境中国。

拿着K签来中国能干嘛呢？能在教育、科技、文化等领域，进行交流、创业、商务等活动。虽然不能直接就业找工作，但“创业”“商务”这俩关键词，还是激活了大众的争议和担忧：

1. 国内就业压力已经很大了，还要引进外部科技人才，这不是跟老百姓抢工作吗？
2. 境外高校的学士学位就能申请K签，对外国科技人才的认定门槛也太低了吧？
3. 欧洲北美不少国家都深陷移民潮困扰，为什么要主动让人打扰我们的平稳生活？

## K字签证争议的由来

这种担忧，还真不是毫无由来。《纽约时报》采访到了一个深圳猎头就言之凿凿地说，今后外国留学生留在中国的难度将大大降低。阿里和华为这样的科技巨头，也更容易引进阶段性的海外顾问。

而9月底这一轮中国社会对K签的关注，是被印度媒体带起来的。9月20日，美国总统特朗普把美国H-1B工作签证的申请费拉高到了10万美元，来实现“把工作还给美国人”的效果。当时消息一出，亚马逊、微软、Meta、谷歌等科技巨头，连夜召回海外在出差和探亲的员工。这带来的结果是，各地飞到美国本土的机票价格纷纷翻番。

## 中国K签的背景

我猜，无论你支持还是反对K签，你对它的出现都不会太意外，因为我们以前的讨论里已经对这事有过铺垫。中国从2023年开始正式提出“高水平对外开放”，此后金融、电信、医疗、文旅服务等维度都在打开国门。我们在88讲（《7月政治局会议：四个风向》）关注的7月政治局会议明确的后半年7项重点工作之一，也是继续推进高水平对外开放。这些，你都曾在专栏里听到过。

在更早的《商业参考》第三季里，我们聊入境旅游业时，还大胆预测，中国增加免签国家数量、力推入境旅游，攒下来的跟外籍游客相处的经验教训，未来是要用在外籍人才来华工作管理服务问题上。2024年上海就出台了文件，来探讨事业单位招聘外国人的制度机制。

现在，K签来了。

## K字签证的意义

就我个人而言，我也担心这之后的生活环境变化。但无论我们喜不喜欢它，它所蕴含的意味和趋势都很明显，就是中国要加大跟全球人才资源的互动剂量了。中国政府8月出台K签的时候就解释说：中国发展需要世界人才的参与，中国发展也为世界人才提供机遇。

《环球时报》更直接地说：“我国国内部分科技产业一定程度上受制于专业人才……以更加开放的姿态拥抱全球人才是双向赋能的必然选择。”

那么，为什么是这个时间点呢？我的理解是，当前中国恰好等来了两个时间窗机会。

### 美国移民政策收紧

第一个是，美国正在系统性地放弃甚至排斥外来技术人才，影响了高端科技人才这种全球资源的流向。中国想要接盘的话，宜早不宜晚。美国当前想要从自己就业市场里挤走的竞争者，还真不只是印度人和华人。在特朗普的MAGA治国思路下，从他的第一任期就有了排斥移民的趋势。到了第二任期更是变本加厉，他在给H-1B签证大幅涨价之前，已经在要求联邦合同和项目优先录用美国公民、缩减“家庭团聚类”移民份额，还在考虑给外籍员工过多的美国企业加税了。

在这个趋势下，外籍科学家离开美国开始成为趋势。斯坦福大学有一份报告说，2018年中美贸易战以来，离开美国的华裔科学家增加了75%，其中2/3前往了中国。

中国这边，2024年以来也有不少重磅案例：

- 陈婧，计算机科学家、区块链领域专家，全职回到清华大学；
- 齐国君，多模态人工智能方向的知名学者，加入了西湖大学；
- 符天凡，AI制药领域的青年学者，加入南京大学；
- 孙少聪，癌症研究专家，加盟首都医科大学；
- 陈沪东，美国国家工程院院士，也回到中国大学任职。

这些名字串联起来，就是一个清晰的信号：被美国释放的学者型顶尖科研力量，正加速回补到中国的科研和产业生态中。

### 世界对中国好感度上升

到了2025年，H-1B越收越紧，我们有理由相信，原本栖息在Meta、谷歌、微软等美国企业的一批“学子”型外籍科技人才，也在琢磨离开美国。离开美国去哪里？哪里友好去哪里。这个“友好”既包括了薪酬待遇，当然也要包括创业和生活便利。迪拜和新加坡，在过去5年就已经吃到了这股人才红利。这期间，加拿大的“数字游民签证”、德国的“蓝卡计划”、日本的“特定技能签证”，也被数字游民们翻了出来，重新评估和审视。

这又说到第二个时间窗机会了：2025年以来，世界对中国的好感度正在急剧上升。

从年初的小红书大对账，到DeepSeek和千问大模型的风靡，再到最近93阅兵亮出的军工科技实力。9月底，你还可能见过一个TikTok的爆款短视频，一位百万粉丝的大V激动地展示，他在张家界见识到的爬山外骨骼。

这些中国文化、医疗、科技与社会生态引发了海外惊叹，也创造了时间窗口。为什么说是时间窗口呢？因为你不知道什么时候这种仰视和羡慕，会转成敌意和忌惮。

## 中国的开放与融合

要是我们把观察尺度再拉大，还可以发现，二战后，美国从“世界工厂”变成“世界人才磁场”，也是受益于几个关键动作：

1. 1945年前后，随着二战的结束，美国发起了一项“回形针行动”，以军事行动的方式，从德国、意大利等战败国抢夺了几千名纳粹科学家回美国本土，直接奠定美国航天、核能、导弹的技术优势。
2. 1950年代，美国为了应对跟苏联的冷战，通过了《国防教育法案》，开始鼓励STEM教育、给外国留学生开奖学金。这部法案的通过，让美国得以持续吸纳东欧、苏联体制外流出的人才。
3. 1965年，美国进一步废除了按国别配额的种族歧视移民制度，开始按技能、家庭团聚等因素来接收移民，尤其为亚洲高技能人才打开了大门。

这三步走下来，今天的美国逐渐成为了全世界科学家和技术人才的事业灯塔，硅谷强势崛起，“科研—产业—资本”的正循环，也成了全球科技精英的美国梦。

当然，今天的美国受益于过往几十年这种大规模欢迎移民的决策，但同时也没少从中吃苦，移民引发的身份与社会问题，我们也聊过不少了。特朗普驱赶移民的动作，其实就表达了美国社会MAGA人群对当初决策的后悔。怎么能吃到甜头，同时把害处降到最低，这是要考验效仿者智慧的，也是今天K签引发巨大社会讨论的原因。

## 中国的开放选择

那不开放行不行？中国靠自己不是也能搞出DeepSeek、搞出电磁弹射技术吗？还真是不行。直白点说，想在一个不幸福的世界里独美，那就是在引战。中国今天的贸易体量占全球份额的1/7，掌握着其他国家的产业链和供应链命脉，中国人在全世界都建起了唐人街、中国城，如今在倡导人类命运共同体的同时，会把本土的人才开放尺度做到多大？这也是其他国家会问中国的，也是101讲（《“提振消费”为什么必要且急迫？”》）金刻羽的那句话：让其他国家也参与到供应链里来，关乎和谐的国际关系。

## 总结

所以啊，在历史级的巨变之下，我们作为普通人，是支持还是反对K签？应当怎么看待和应对K签所代表的趋势变化？这是我前面说了这么多之后，真正想要请教你的。趁着假期，你不妨把这篇文章转到某个群里，讨论讨论。这不是一个丢给AI就能得到好答案的问题，但它是一个事关我们每个人的超级问题。在这个国庆假期里，也抛给你。

我粗浅的想法是，在未来，除了血缘和身份证，我们对“中国人”的定义，可能会往两条脉络上延展：一条是文化认同，一条是利益绑定。也就是说，你是不是熟悉中国故事和文化、核心信念是否跟中国对齐；你的核心资产在不在中国、核心利益是不是跟中国绑定，这些要素可能会超越肤色与发色，来影响别人认为“你是谁”。

但我确实没有结论。期待听听你的想法。
