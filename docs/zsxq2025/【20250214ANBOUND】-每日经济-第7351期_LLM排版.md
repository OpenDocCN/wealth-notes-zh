## 【周边国家理应成为中国外交的重点】
长期以来，就中国外交而言，对美外交一直是中国外交工作的重中之重。相关业内专家多次指出，维持中美关系稳定发展是中国外交战略布局中最重要的环节之一。此前，中国主动提出构建新型大国关系，就充分体现出中国对发展中美关系的期待。根据新型大国关系的内涵——不冲突不对抗、相互尊重、合作共赢，中国在发展对美关系上有三个着力点：一是加强战略互信，有效管控分歧，确保不冲突不对抗；二是增进各层面交流，促进两国相互尊重；三是推进各领域务实合作，继续互利共赢。中国始终强调，对美政策的重点应当放在加强战略互信、有效管控分歧上。
毋庸置疑的一点就是，对美关系缓和的实现和中美正式外交关系的建立是中国改革开放事业得以顺利推进的重要外部因素。有了美国一定限度之内的支持和认可，美西方掌控的全球经济-技术-金融网络最终接纳了中国的融入，而美西方大量的资本投入和技术转移也是中国经济-科技崛起的重要助推力。
同样毋庸置疑的一点就是，近年来，中美关系消极面的不断深化是中国和美西方关系恶化的重要原因，而美国的一系列举动使得中国在诸多第三世界国家的利益布局都受到了显著的挑战，巴拿马就在近期正式宣布完全退出“一带一路”合作项目。看起来，似乎只有继续全力尝试改善对美外交关系才能有效解决中国当下所面临的一系列问题。但是，安邦智库（ANBOUND）资深研究员表示，为有效维护中国的海外利益和全球地位，在未来（至少3-5年内），周边国家应当成为中国外交的关键发力点。
第一，全力深化和优化对周边国家外交关系，是逆全球化时代的客观要求。至当下为止，过去由资本驱动，过分聚焦于经济层面的全球化已经呈现出一系列问题和疲态。就经济上而言，过去全球化的全面拓展并未能实现其此前承诺的普遍富裕，国家和国家之间、国家内部的贫富的差距依然呈现出持续的扩大态势，这已经引发了全球性的不满；社会-文化层面，过去全球化的全面推进并未能够有效弥合文化-意识形态方面的差距，不同文化和意识形态之间的对立程度甚至还有所强化，而过去全球化造成的世界移民潮则引发了包括发达国家在内的诸国主体民族的强烈不满，右翼民粹主义势力越发煊赫。并且，全球产业链-价值链的拉长，使得全球经济体系在面对重大疫情-自然灾害以及地区矛盾冲突时，显得十分脆弱。安邦智库（ANBOUND）此前就强调，区域化将会是未来的主导性趋势，主要大国的发展重心都会逐步转向自身的地理邻近地区。特朗普上台之后，一再要求丹麦割让格陵兰，还要求加拿大加入美国，这些都不是单纯的戏言。
