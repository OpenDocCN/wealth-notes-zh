
# 瓦格纳在尼日尔的政商关系对中国的启示

据外媒报道，西非国家尼日尔军政府近期已经强制在石油领域工作的三名中国高管离境。这些高管分别是中国企业以及合资公司在当地的负责人。尼日尔方面表示，原因是中国员工与当地工人之间的工资待遇差距引发了争端。据了解，尼日尔与中国的合作始于 2008 年，中国主要在该国从事石油资源以及铀矿的相关业务。

尼日尔所在的萨赫勒地区长期处于动荡不安的冲突态势中，内战频发、恐怖分子四处制造袭击。对于分布在尼日尔和萨赫勒地区的中国企业与机构而言，有一个问题需要认真思考，那就是中国在当地以及整个非洲大陆到底应该如何制定部署相关战略？特别是在美国秩序退出全球，法国秩序退出非洲的背景下，中国应该建立一个什么样的灵活机制，最大化维护自己的利益？安邦智库（ANBOUND）在《商业化的战争》报告中已经明确预测，随着世界地缘政治空间的碎裂化，今后的世界将处于一种低强烈度冲突和小规模战争的阶段，这种低强烈度冲突和战争，可能遍布世界各地，尤其是非洲。而在这种不适合用正规军事力量的地方，中国应该要建设一个私人武装安保集团，使用“雇佣兵”就是一个适当的解决方法。这种做法，国家几乎不用给钱，只要给一些资源，对安保集团就足够了。

那么，对于中国而言，有没有什么值得借鉴的例证呢？瓦格纳集团（Wagner Group）2023 年7月以来在尼日尔的布局就是一个值得参考的例子。这个组织是一个较中国而言进军非洲的后来者，它在尼日尔的政商关系和军方关系，也体现了俄罗斯在非洲通过代理势力扩展影响力的战略。瓦格纳原本是一家国际化的安保公司，依靠俄罗斯政府的支持，在世界独立行动进行商业化的战争，拥有自己训练有素的军队和网络。瓦格纳经过一段时间的经营，依托俄罗斯情报机构以及特种部队的背景，在非洲和中东获得了规模很大的根据地，形成了国家资源-军事能力-矿产权益“三位一体”的运作模式，在 “商业化的战争”中运作得很成功，规模越来越大，直到在乌克兰战场“吃瘪”，创始人普里戈任丧命，但它在非洲依旧保持着存在和发展。
