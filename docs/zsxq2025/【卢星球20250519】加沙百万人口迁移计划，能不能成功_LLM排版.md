
# 加沙百万人口迁移计划，能不能成功？

## 文／卢克文工作室嘉宾星海舰长

5月16日，美国全国广播公司(NBC)曝出了一个大新闻，说美国政府正制定一项计划，将多达100万巴勒斯坦人从加沙地带永久迁往利比亚！

那么，这个迁移巴勒斯坦人的计划，到底是什么呢？

其实吧，所谓迁移巴勒斯坦人，根本不是什么新提法，而是早已有之的计划和方案了。

早在2023年阿克萨洪水行动开始一个月的时候，以色列情报部门就曾提出过一个方案，提议在占领加沙后，将230万住在加沙的巴勒斯坦人驱逐至埃及西奈半岛，搭建若干帐篷城，最终变成巴勒斯坦和其他阿拉伯国家。

此外，美国不会向加沙派遣美国军队，美国也不会为加沙地带的重建支付费用。

按照特朗普的说法，长期战争导致加沙地区遍布未爆炸弹和地雷，目前确实不适合人类居住，因此需要在重建期间，让居民暂时迁移。总而言之，这个方案是为巴勒斯坦人好。

特朗普还真不是说说而已，真的找了约旦国王、埃及总统塞西去谈这件事，不出意外，遭遇了强烈反对。

看约旦和埃及不行，内塔尼亚胡又提了个新方案，建议巴勒斯坦人迁往沙特阿拉伯。他认为沙特拥有广阔的土地，足以让巴勒斯坦人建立独立国家。

然后，就被沙特喷了一通。

显然，这个方案起码在中东国家里面，是行不通了。

不过，卡扎菲的被杀并没有让利比亚迎来和平，反而让过去被卡扎菲压制的各路武装和部族势力趁机夺权，开启军阀混战阶段。

一直到2016年前后，利比亚的局势才慢慢明朗起来，形成了两股主要势力。

一股是军事强人哈夫塔尔领导的国民军，他在东部城市图卜鲁格建立了利比亚国民代表大会HOR，自立政府，控制了东部、中部、南部主要城市，势力庞大。

另一股呢？是一些反对派把部分部落头领、前任政府官员拉拢了过来，以利比亚首都的黎波里为核心，形成了一个新政府一民族团结政府GNU，控制了东部地区。

因为政府成立的时候走了选举程序，所以西部民族团结政府获得国际社会广泛承认。

有个统一能建国的势力，另一方面也是因为经过第一次中东战争，巴勒斯坦被自己的阿拉伯兄弟给瓜分了。

埃及拿走了加沙，约旦占据了约旦河西岸，叙利亚则分到了加利利一小块。他们没有一个把巴勒斯坦人建国梦放在心上的，都想把巴勒斯坦吞并占为己有。

后来呢？经过几次中东战争，阿拉伯国家被以色列打得很惨，又把占据的土地吐了出来，改为扶持巴解组织和着干。

发展到今天，阿拉伯国家也是对巴勒斯坦人其实也只有利用，而没有多少同族之情。

比如，他们不愿意接收巴勒斯坦难民，不仅现在不接收，原来也不接收。虽然嘴上支持巴勒斯坦人，但实际上避之如蛇蝎，根本不愿意接纳巴勒斯坦人。
