
# 美元美股双重风险(1)

原本应该等到已日的，现在看已经提前了。特朗普不是假摔，而是真的跌倒在地上了。全球资本不接受特朗普要挟，开始四散。下周三，日本若反刺，则美国资本市场会翻天覆地。

美元资本夺路而逃，也会利好黄金等大宗商品。此次危机，不同以往，请牢牢记住，这次是美元重置的危机。不仅仅是美股严重高估，美元本身就是严重高估。巴菲特持有美元现金并不明智(持有美元债券一样愚蠢)。必须对冲美股与美元双重风险。

我在读《美元陷阱》一书。中文版推荐的著名专家学者，自露怯于扉页之上，未必都是十恶不赦，也是愚不可及(江涌除外)。他们护盘美元，恶意摧毁本币，诱导国家与美共赴美元危机。将来，请大家梳理线索，给历史留下一个交代。

此次美元危机，会否利益中国呢?总体趋势上看，确实是有利于中国的。但是，外资涌入确实利弊参半。一则，我国门槛太低了，资产价格低到大宋程度。二则，我国专家学者缺乏通胀前置的政策预期，他们压根就不相信会有一轮大宗商品的牛市，这会导致我国下半年也出现滞涨的局面。真急人，却没办法。

说到这里，诸位的操盘逻辑也就呼之欲出了。我仍然坚持，请大家按既定方针办。至于短股，请自由发挥，唯注意外资的取向。至于A人工智能，适可而止。泡沫破灭的时候，好坏皆会被波及。

我与诸位一样，肉体凡胎，会犯错。虽诚恳，无遮掩，却会有疏漏。所述尽心而已，不对的地方请见谅。
