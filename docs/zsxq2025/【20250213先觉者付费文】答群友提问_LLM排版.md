
# 答群友提问

群友：我想问个问题。你说我们要多练习输出，而不是输入。要怎么样把握别人对我们进行"输入"这件事，你是如何做的呢?
回答：付费换来的信息，可以尽情输入，不会吃亏的。人只有收了钱才会说真话。
想免费获取信息的人，他获取的都是假信息。原因是这样的，没有为价值买单思维的人，你跟他说真话，他会生气的。因为他们每天沉浸在假信息、假新闻里面，被洗脑太深。
我自己是这样看待新闻的：所有新闻都是假的。几乎所有的出版物，电视节目，互联网免费的自媒体，都不会说真话。因为大众不喜欢听真话。
很多人有贪图便宜的习惯，只为看得见摸得着的东西买单。贪图便宜的穷人，结局就是落入富人为他们编织好的陷阱。
他们眼睛里看到的、耳朵里听到的所有信息都是让他们更穷的信息。还有让他们情绪更加负面的信息。
让人悲伤、愤恨、自豪的新闻，只有穷人才会去看。他们的情绪时刻都被假新闻牵着走。早就丧失了独立思考能力。

群友：请问你是怎么看奢侈品和消费主义的呢? 因为我经常跟风去买包 ….什么新包出来就很想买，虽然知道是被割韭菜但还是经常忍不住。
消费能给自己带来快乐和满足感就行，而且不是成瘾性的。我觉得女人换包、男人换车换表都是好事，说明有追求，实力也在提升。
快乐是比较难获得的，自己支付得起的快乐就是适合自己的。这一点来说，不论钱多钱少的人都是公平的。阈值是不断提高的，所以能让自己快乐的事越来越贵、越来越有挑战性，所以不论多有钱的人都在努力向上攀登。
