
# 要贷款吗？

提 问：
司令您好，我现在遇到一个抉择，希望您能指点一二。我自身企业走传统银行线上融资有瑕疵过不了，走线下融资每家银行都能给到我两百多万的额度，但是每家银行审核条件，放款条件都十分严苛，以至于我做了近三个月时间都未能有实质落地(有中间人操作),目前工行卡在最后一个环节，需要有实力的人为我担保(我00年生，银行认为我太年轻，质疑是职业经理人),当我找到了同行有实力的人为我担保，又提出担保人配偶也需要签字，就此僵住。
目前很为难，我的公司前两年依托于父亲窑炉工厂运作着，承接大量业务，但是迫于环境，资金等压力父亲工厂于今年年前停掉。因我公司有承接父亲公司部分业务，债务压力，所以压力骤增。(去年父亲经营不善，现金流，应收账款等尽数耗光)。
于年后我司筹划融资事宜，但一事不顺，万事皆难。年后即遇官司，经过长期博弈都摁了下去，将最优质业务剥离给其他公司深度绑定(不愁生活来源)。因没有资金任何业务都开展不了，上下游每个环节都缺钱，微信pep458有些业务不是没有人不愿意帮忙，而是大家都真没钱，这数月以来每个月什么事儿都不干，基本盘的维护成本都要二三十万!没有任何来源，真是抵不住了!

这数月以来与金融机构也打了无数交道了，很累很疲惫，也很失望，没有搭上宽松贷款的末班车。目前还有一个选择，线上银行，我已与微众银行经理深度沟通，他能通过特殊渠道为我争取200万额度包过，但年化12%-15%。
但是其中也有很多问题点：
1.他的承诺是否真的靠谱
2.我的公司未曾贷过一次款，线下四大行只要过了一家，后面银行放款都会宽松不少，(中介公司最开始承诺几家银行同时做给我争取1000万)但是目前一家都过不A 了，第一家如果过的是微众这种银行，对后续融资有不好影响
3.线下银行实在没办法的情况下也只能选择微众银行死马当活马医，能否做下来两说，但如果做下来了，微信pep458仅有200万的额度很难有所作为，一半的资金都需要还债，100万能撬动项目，但是目前的行情都是些能勉强维持的业务，没有暴利，那么如何在同时抵住遗留债务、贷款高额费用的前提下经营开展业务，如何通过这100万抵住将近500万的遗留债务问题，还不算父亲公司必须打发的资金，太难了
4.如果现在拉爆以后再无翻身可能，所以想到最后都只能被动拖拽着往前走，在牢笼中寻找透过墙隙

唠叨太多，若有幸，希望能聆听司令。

司令回答：
你还是把这个生意关了吧!这种腾挪的意义不大，你的问题好像之前说过。你说有个人找你父亲商量投资的事情，但不愿意全力以赴，我之前跟你说过那是投石问路。他投入一点，让你父亲用自己的所有资金和资源去尝试一下这个项目的前景，如果有起色，他就会全面接手，没有他就撤了。
以你今天反馈的情况来看，这个事情应该没有什么起色，而且，包括金融机构在内的所有市场参与者都在采取谨慎的措施应对未来的颓势。供应链指标数据显示：所有的生产链条中个环节都在对未来采取比较悲观的防御措施!在这样的趋势之下，那些高负债的挣扎，前景是堪忧的。什么情况下才值得高负债去赌一把呢 ?当然是在整体趋 势向上的阶段，越是向下的阶段，越不值得赌 。因 为螺旋叠加的向下趋势下，各环节的配合 ，更容易 出现意想不到的意外，而你在做这件事情的时候， 所依赖的那些确定性因素，往往会变 成更多的不确定性因素，同时又没有足够高的利润去对冲相应的风险。

有的时候不瞎折腾，反而是稳妥的策略。因为当真正的机会来的时候，你手上是有资源的，而不是什么资源都没有，还有一堆负债需要偿还。
