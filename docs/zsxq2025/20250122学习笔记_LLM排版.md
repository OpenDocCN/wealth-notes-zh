
# 20250122学习笔记

## 学习笔记，2025.1.21
1. 人要学会过磅。你总觉得自己怀才不遇，那很简单，去人才市场一碰，别人就会给你出价。你总觉得自己遇不到好男人，去相亲市场一看，看看别人给你匹配什么样的男人就行了。
2. 这个世界对剩女普遍有误解，以为剩下的全是高学历的优质女，曾经有篇社会调查就讲这个，那些其实都是个例，是她们不愿意走进婚姻，主动选择的单身模式，她们想走入婚姻太容易了，实际上，剩下的，多是真正的边角料……知道为什么杀猪盘多发生在这些女人身上吗？因为，她们渴望的。只有骗子能给！
3. 公猩猩跟母猩猩在一起半年就阳痿了，咋办？换个母猩猩接着就完了。这是动物本性。人也如此。小时候老师要求写日记，我们能坚持得很好。可是你见谁坚持到老？博客刚出来的时候，人人都去开博客，仿佛人人都是作家，可是坚持写的又有几个呢？后来又玩微博，天天刷，等红绿灯也刷，做饭也刷，你现在还刷吗？微信朋友圈，也是如此。不是内容不行了，而是人的惰性决定的，对什么都会审美疲劳的，就是把范冰冰嫁给我们，我们能保证天天不厌烦吗？我们睡着范冰冰可能还想着柳岩。从这个角度可以得出一点：无论什么模式，都不会常青，包括一个王朝。
4. 例如《中国好声音》第一季大家觉得很新鲜，我们甚至能说出所有参赛选手的名字：平安、金池、吉克隽逸、李代沫、金志文、梁博……最新的一季，你能说出几个？再过上十季呢？也许节目都没了，就跟当年的《正大综艺》一般。所以，对于读者的离去，我觉得是必然的，你今天关注我，明年未必关注我，也许明天就不再关注了，因为你也会审美疲劳，再牛B的人也不过就那么几把刷子。所以，读者换了一拨又一拨。是人性决定的，当然与我自身成长太慢也有关系。
5. 有钱人的故事你讲出来就是流量，你的职业摆出来就是流量！你说说你学法律的，你做金融的，你说你不懂，你谦虚了很多人有个误区，以为自媒体要讲专业，错了，自媒体关键是要讲故事，讲认知，你们看我讲过专业吗，没有的，我从来不讲专业，最多科普流程。
6. 买东西，跟朋友买东西，大件的东西，基本不合适，所有的商家都会坑好说话的大客户，尤其是人傻钱多的类型，你跟他走得近，好说话，他一定会把你的优先级降低，会在服务和品质上怠慢你。熟人里的人情世故无非就是这样，要算账就找能算账的买，不算账就偶尔跟对方买一点，让他多赚一点，维持一个良好的熟人关系。
7. 没事别老看什么报告，生意就是业务，业务就是买卖，我们从来不提商业和市场这类大词的，讲的就是订单和客户这类具体活动。赚钱人的状态，没有社交的，也没时间弄其他，唯一见的，就是老婆孩子和客户，所有的时间，都指向直接赚钱的活动了。
8. 跟人学，起码你有个师傅来问，你去问同行，不太愿意跟你说的，同行是冤家。我毕业十年，我也经常警醒自己的，要积极乐观，要力所能及的勤快，不要陷入消极主义，哪怕你不追求事业，也要积极勤快，身体和心理是联动的，你一消极，连荷尔蒙都懒得分泌了，身体还会变差，恶性循环下去，很难再调理回来。
9. 小心你所处的环境、所交的人，你看的书、课程、电视剧、电影、图片、文字，你的注意力不能放错。要是放错在垃圾、低质量信息上，你的脑子就被毒害了。这个毒害，是有壁垒的，把后面正确的、实践有效的信息，都挡在外面。摄入错误的、劣质的信息，你的思维就低劣，变得愚昧无知，顽固不化。合法毁掉一个人的一生的方法，就是给他灌入错误的、低劣的信息思想。我何时发现了自己的愚昧？因为我努力很多年，没有现实结果，我理解了自己的愚蠢；因为牛人有现实结果，我认可了他们的智慧。从此，我只愿跟牛人玩、跟高手玩、跟专家玩、跟有结果的人玩，只学他们释放的信息，对他们的思想和语言，我深信不疑。我理解了他们的智慧，要听话；不理解他们的智慧，我也听话；在听话、实践中理解高深的思想。
