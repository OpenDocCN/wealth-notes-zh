
# 【米联储见闻251114】——自馀嚣嚣直可轻（中国稀土的功臣）

面对中国的稀土逼宫，美国正在到处寻找解决办法，满世界找矿已经是不争的事实。10月26日，川普在东盟峰会期间，与马来西亚、泰国、柬埔寨签署贸易协议，涉及稀土出口。10月28日，美日签署稀土战略合作协议，通过技术共享与供应链整合降低对中国依赖。11月6日，川普在华盛顿主持 C5+1 年度峰会，与中亚五国领导人依然是讨论能源矿产合作。中亚地区拥有丰富的稀土、铀等资源，哈萨克的稀土储量可能成为全球第三大稀土生产国，仅次于中国和巴西。

美国并不是第一天知道中国手握全球稀土供应产业链的命脉，但确实是第一次感受到中国居然敢用稀土管控这张牌，且自己居然也没有什么还击的办法，这让美国第一次体会到了切实的掣肘。为了避免被中国卡脖子，美国政府早就开始在关键矿产上下功夫。今年7月1日，美国、日本、印度和澳洲外长在华盛顿举行四方安全对话会，宣布启动四方关键矿产倡议。10月20日，川普与赴美访问的澳洲现任总理阿尔巴尼斯达成一致，共同签署了一项稀土矿物协议。双方计划投入85亿美元，其中美国政府投资25亿，澳洲政府15亿，企业承担45亿，重点用于开采、加工和精炼环节。

可以说，最近川普参与的大部分外事活动都有稀土合作的内容，中国的稀土管制确实是让美国很头疼。迫使美国将关键矿产安全提升至国家战略核心层面，美国政府开始直接参与大型矿产交易。2025年7月，美国国防部以4亿美元收购美国最大稀土生产商 MP Materials 的15%优先股成为公司最大股东，与公司签署10年期保底价采购协议，并提供1.5亿美元贷款用于提升重稀土分离能力。9月，美国能源部通过谈判将美洲锂业旗下位于内华达州锂矿项目23亿美元贷款，转换Thacker Pass为公司5%-10%股权，成为其潜在最大股东。美国内政部10月6日宣布以3560万美元收购加拿大大公司10%股份，并获得额外Trilogy Metals7.5%股权的认股权证，同时推翻拜登政府否决，批准连接阿拉斯加矿产区的安布勒公路项目，释放铜、钴、镓等国防关键矿产潜力。

11月11日，财长贝森特扬言，中国对稀土的控制不会超过2年，美国将终止其垄断地位。虽然专家认为，一个典型的稀土矿产项目，从勘探到建设需要9年时间，最短都要6年3个月，最长要18年。但美国确实正在动用国家力量多管齐下。美国还在组建规模达50亿美元的关键矿产投资基金展开，重点投资全球范围内的战略矿产项目。美国国际开发金融公司(DFC) 正与总部位于纽约的投资公司猎户座资源伙伴(ORP) 洽谈，拟成立一家合资企业来运营该基金。中国控制着全球60%以上的稀土开采量，85%以上的精炼产能。中国在稀土等关键矿产上的绝对影响力，让美国不得不全力以赴构建新的供应链以确保安全，和中国不希望在芯片等领域被美国威胁一样。这是一场时间和空间的较量。

和美国一样积极行动的还有欧盟，2023年，欧盟委员会就曾推出首份旨在打破对华过度依赖的宏大计划——《关键原材料法案》(CRMA)。该法案在2024年3月18日正式通过。欧盟委员会主席冯德莱恩10月末透露，欧委会将提出一项名为的RESourceEU计划。该计划将重点关注关键矿产领域的回收、联合采购、储备和投资。日本自2010年前后与中国关系紧张开始就启动了减少对中国稀土依赖的产业链布局。2012年日本和越南成立稀土研究技术合作中心，日本提供技术资金支持越南开发东堡矿等资源。日本综合商社双日公司最近开始从澳洲进口部分稀土元素，这是日本首次从中国以外的渠道获取这一关键资源。欧盟日本等美国的盟友在稀土等关键矿产上产生焦虑，很大程度上来源于这些国家担忧中国会对他们采取报复措施。这些参与美国在芯片高科技领域对华围堵的盟友们，非常明白自己做过什么。

令人意外的是，还有一个国家也开始布局稀土，就是中国的战略协作伙伴俄罗斯，近来在稀土问题上的动作大得让人意外，原本凭借和中国的良好关系，根本不用担心稀土产业链安全的俄罗斯，也开始要构建独立于中国的供应链。俄罗斯的动作也说明，大国永远是大国思维，永远不会将自己的国家利益系于任何人之手，哪怕是朋友。绍伊古10月24日在西伯利亚挥铲奠基，现场的红色横幅上就写着“全球最完整稀土全产业链”的字样。俄罗斯砸下7000亿卢布重金，目标是打造西伯利亚稀土加工集群，2030年前实现“开采-冶炼-加工-应用”全链条国产化，绍伊古直言这关乎国家安全与战略自主，邵伊古直言要打破技术依赖，其实早在2023年5月，俄总统普京就签署《关键矿产安全法》，将稀土明确为战略资源。2024年3月，俄罗斯政府批准《2030年前稀土产业发展规划》，提出全链条国产化目标。普京近期还亲自下令，要求政府在2025年12月底前拿出详细的稀土产业发展路线图。核心目标是建立自主可控的稀土开采和加工产业，减少对中国和美国的依赖。

俄罗斯探明稀土储量约为2800万吨位列全球前五，绍伊古此次出席的是托木托尔矿床的开采奠基仪式。以往俄罗斯和中国在稀土上的合作模式是俄罗斯出矿产中国出技术。2023年俄罗斯和中国讨论过稀土合作，当时中国甚至愿意赴俄罗斯建精炼厂，但是这一方案最后却被俄罗斯搁置了。2021年阿拉斯加会谈时，俄罗斯曾主动提议，跟美国共同开发北极稀土资源。结果不了了之。2024年11月与印度签署稀土合作协议，印度提供选矿设备和技术培训，俄罗斯长期供应精矿。可见，稀土管控不仅让美日欧心口痛，也让大国都看到了这个命门的重要性。凡是不想缺席未来高科技盛宴的国家都不会希望在原材料方面被卡脖子。绝对安全的前提是绝对自主。即便是中国的友好邻邦，在看到中国拥有垄断优势时也会不寒而栗，开始畅谈供应链安全。由此可见，国与国之间的信任关系，在绝对的国家安全面前，脆弱得不值得一提。

自从中国打出稀土牌，很多人也纳闷，为什么如此重要的筹码最后独独握在中国手里，老邓1992年曾讲：中东有石油，中国有稀土。很多人认为老邓是今日中国稀土称霸全球的主因，但事实上，前副总理方毅才是真正的幕后功臣。中国稀土储量位居世界首位，其中80%的稀土分布在内蒙古自治区包头市的白云鄂博地区，白云鄂博矿是世界罕见的多金属共生矿。从1978到1986年时任国务院副总理的方毅，曾七次来到包头进行调研，方毅是技术专家型官员，同时兼任中国科学院院长。

1978年7月，方毅第一次来到包头，包头的铁矿石矿床中蕴含大量的轻稀土。除了可用于球墨铸铁和玻璃制造的路，还包括石油精炼所需的。矿床中还蕴藏像衫等中稀土。美国在70年代已将其用于制造超音速战机和导弹电机必需的耐高温磁体。方毅在1978年视察包头时就断言，稀土在钢、球墨铸铁、玻璃陶瓷、军工、电子和新材料等方面有重要的应用价值。1979年，方毅在包头结束考察后不久，就率领中国的顶尖工程师团队拜访美国最先进的工厂，其中包括洛歇马丁和麦道飞机在洛杉矶附近的工厂，老邓1979年与卡特总统共同签署中美科技合作协定和文化协定。《中美科技合作协定》为期5年，它规定中美将在平等、互惠和互利的基础上，在农业、能源、空间、卫生、环境、地学、机械等科技领域以及管理、教育和人员交流等方面进行合作。

方毅亦与美国能源部部长施莱辛格签署两国在高能物理方面的合作协议。以“三大共生矿的资源综合利用为试点，大力推动以科学技术促进工业的振兴和发展，是方毅根据加快国民经济发展迫切需要提出的创造性工作思路。这三大共生矿指的是四川攀枝花钒钛铁矿、内蒙包头白云鄂博矿和甘肃金川镍矿。攀枝花铁的储量近百亿吨，居全国第二位，共生的钒、钛储量分别占全国的58%和90%，钛储量居世界前列；白云鄂博矿富含铁、铌、稀土，稀土储量居世界第一；金川矿的镍储量约占全国储量的70%，还有钴、金、银等贵金属。三大矿是名副其实的聚宝盆。稀土、钒、钛、镍等都是国家建设所急需的重要物资。但是因为分离技术不过关，这三个矿从建设开始就年年亏损，每年国家补贴大量资金，而且环境污染严重。方毅亲自抓三大共生矿资源的综合开发利用，为此曾七次到包头调研。直到上世纪90年代至本世纪初，中国的稀土提炼工程师成功攻克重稀土的分离技术，才让中国在重稀土生产上傲视全球。

所谓前人栽树后人乘凉，能够把中国带到今天这个样子，是无数能人志士不断前仆后继的结果，尊重人才，重用贤才，始终是中国能够发展进步的关键。这几天市场又在传吴主席有病的消息，虽然吴主席立刻发了出访信息辟谣，大家也可以看出，人的因素在事物发展过程中的影响力。少一些出师未捷身先死，多一些不拘一格降人才，国家才有未来，人民才有希望。
