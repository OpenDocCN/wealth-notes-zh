
# 跟朋友们聊WG

近代史你会很明显发现，没有使用古代史的很多分析工具。你看特朗普的动作就知道了，不是削藩，而是放权。哪里像削藩了，更像削钟秧。wg本质是在争夺钟秧权力，类似魏晋之交的局面。统治集团，需要在钟秧与地方之间寻求平衡，wg与三年特别困难时期，本质上都是毛选择拉拢地方对抗钟秧。现在老板跟懂王，也都是在走这条路。只要你想拉拢地方，就需要对内神化自己，控制舆论权力和军权，对外放纵，允许各地瞎折腾。

然后地方就会献祥瑞。地方献祥瑞本身就是地方向一把手的一个投名状，取得了一把手的信任，一把手才会支持他们来干钟秧，甚至来取代钟秧。是不是还有争夺社会主义建设的路线之争，毕竟刘的建设思路和苏联背景已经让毛感到死后天下大变。带头喊核心算不算献祥瑞？就拿太祖来说，别看大家都一直骂他集权，其实如果你读透了历史，就会发现，他所有的动作一直都是放权。苏联自己都变了，只有当放权导致自己受到威胁或者出现系统性风险的时候，他才会转变为集权。

太祖魅力无敌，群众基础太好了。站在无产阶级的视角来看，太祖50-53是第一轮放权，57-59是第二轮放权，66-69是第三轮放权。每一次放权，结果都是乱的一塌糊涂，然后又不得不收权。然后每制衡周一轮，周就赢一轮？第一轮干掉高，第二轮干掉彭，第三轮干掉林。我们现在的历史观，很容易把太祖的放权，解读为集权。那削弱钟秧的根本目的是什么？是路线不同？还是理念不同？

削弱钟秧是为了另立钟秧吧？第一轮放权的高潮是五马进京，是让地方大员来夺钟秧的权力。再上井冈山。站在“钟秧”的角度，自然就要骂太祖是“集权”了。客观上来看，太祖某种程度上，的确是在集权，但如果站在钟秧的角度，太祖是在放权。

你只要放权，地方为了取得执政的合法性，就必然要放卫星，一个个跟贾跃亭一样。wg对于钟秧来说肯定是放权，毕竟太祖认为：“只有用'wg'这种方式，公开地、全面地、由下而上地来揭发我们执政党、政府的黑暗面′，才能真正解决问题。红党是要集权，蓝党是要放权，新党是一边集权一边放权。是不要钟秧，但是方法是钟秧自己说不要自己。这就别扭的要死，相当于对所有人喊一声 “集合”就为了“解散”一样。钟秧是不是就是以周为代表的这群人，技术官僚。毛想把权力从他们手上分散到地方。

你们大概难以想象，建国初期的官僚很多都用的中华民国的人。这是必然的吧，打得那么快，干部梯队不够啊。红党在政治经济上集权，蓝党在政治经济上放权，新党在政治集权，经济放权。你们看什么电视剧，里面很多城市被解放了，许多官僚从抽屉里拿出镰刀锤头旗，然后说"哦，原来你也是地下革命者”。红党内部派系林立，但是在集权的问题上，是能够达成高度统一的。毛虽然在国家治理的各个方面在放权给其他人，但是宣传口从来都是自己的铁杆儿把控住的。他们就是墙本身。老板不也一样么,十八大之后啥也没要，就要了文宣。一个国家可以没有执政党，但是不会没有这些官僚。特朗普对x的马斯克，tt的周受资。红党在干掉放卫星的问题上是利益高度一致的，无论是贾跃亭各个领域的吹牛逼，还是王健林遍地的卫星城。

就是要把权牢牢掌控。大家可以带入到一个公司里面，老板想要提拔一批年轻人，执掌着中枢的老头子们，肯定会想办法让这帮年轻人吃大瘪。哪怕明知道这样会损害公司的利益，也会想办法放大这批年轻人的错误。胡，赵……屁股决定脑袋啊。然后倒逼着一把手把这批草台班子给废掉。从五马进京，到上海疫情，我们经历太多了。赵是自己作死。胡的确可惜，他太理想派了。赵那个样子就像是不自杀的高，疯狂把火烧旺。再往下说就危险了。行于所当行，止于不可不止。
