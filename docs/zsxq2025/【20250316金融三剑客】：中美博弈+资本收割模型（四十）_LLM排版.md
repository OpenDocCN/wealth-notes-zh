
# 中美博弈+资本收割模型（四十）

话说马部长的敏锐思维和战略洞见，看到了这个百年一遇的机会，他要创造历史，他也说服了川普，并且果断开始采取行动了。继FBI、IRS, 美军之后，又要开查美联储了。就是你不敢查我查，你不敢管我管，这就是马斯克主导的效率部DOGE最近掀起的一股风。

谁也没想到他能搞出这么大的动静。其实查到FBI的时候，马斯克主导的DOGE已经出乎全世界的意料了。查到IRS, 也就是美国的国税局，这个已经匪夷所思了。美国的国税局可不是一个单纯的收税部门。

国内的一些自媒体都在开DOGE的玩笑，用上了那个西厂的段子，说DOGE (也就是效率部)是西厂，说马大人是厂公。实际上美国的国税局 (IRS), 才是真正的西厂。他们拥有非常大的权力，而且拥有武装力量，收税收不上75来的时候真的会武装上门，就地踏平的。

所以在马斯克带着人去查美国的西厂 IRS (国税局) 的时候，熟悉西方这套运作体系的人，都惊呆了。那么当他开始要调查美军的时候，其实整个西方世界已经不是吃惊，而是，“默然”。这就像周星驰在喜剧之王里面讲的：一个人短时间内受了过多的刺激，他是没有表情的。

那其实大家当下就这个感觉，你马斯克到底要干嘛？美军到底有没有问题，是和尚头上的虱子，全世界都知道。美国国防部已经连续7年未通过审计，那些1200美元一个的咖啡杯、1万美元1个的马桶盖、70万美元一只的扶贫山羊都是毛毛雨，五角大楼曾经丢失过数百万个F35战斗机零部件，在没有海的阿富汗修过海军码头，给泽连斯拨款2000亿美元，实际到账700亿。美国国防部每年有8000亿美元的预算去向不明，合计有1.9万亿美元的资产是笔糊涂账，2012年，联邦特别特派员史蒂文斯去利比亚查账，在美国大使馆“被恐怖分子杀害”，2019年，懂王专门派了一个班子去阿富汗搞审计，飞机失事，全员摔死。种种惨状，不一而足...

历史没有新鲜事，当年明英宗决定“让大明再次伟大”。他同样有一名九千岁助手，司礼监大太监，王振，被称为“王先生”。两人一合计，决定对兵部，查账。然而，阻力是非常大的，“群臣多有异议”。

《稗史汇编》提到，朝臣上书言：‘军饷改革非一时之计，恐扰民甚矣。’上怒曰：‘汝等皆为贪腐之徒，岂敢妄言!’恰在这时，也就是正统十四年，1449年的七月，边关忽然报告，蒙古的瓦剌部大军在太师也先的带领下，南下攻掠明朝边境。朱祁镇于是决定亲自上阵，御驾亲征，诡异的是，这场战争，他要求英国公、成国公等勋贵，大明三大营的高级将领全部出征，他还强令包括六部尚书、内阁学士在内的50多名高级文官必须随行，几乎是把大半个朝廷打包带走。

后面的故事我们都知道了，《明史》上记载说，朱祁镇的大军在河北怀来附近的土木堡遭受也先的蒙古骑兵围攻，全军覆没。明英宗被俘，半个朝廷的高官和军事勋贵们死伤殆尽，王振被军方将领锤死，史称“土木堡之变”。

虽然我们的科技发生剧变了，几千年来的人性，有过丝毫的变化吗？五角大楼在阿富汗的揩油，和对付毛文龙的招数一模一样，马斯克力撑懂王，你说和吕不韦投出秦始皇有何区别？现在的 问题在于，谁也不想做安徒生童话里那个说话的小孩子，穿没穿衣服，大家都不讲，现在马斯克出来讲。你讲了，然后呢？你查出美军有问题，接下来你准备干嘛？

那么再接下去这件事就更加无厘头，继美军之后又加码了，DOGE居然去查美联储了。如果说美军还只是美国浮在水面上的冰山，那美联储就是水面下的那部分，冰山的基座。它是一个公私混合的机构，其复杂性我们之前详细解析过。

历史上，想搞大变革的强者、勇者很多，每个英雄都想搞把大的。等真的开始操作，能有好下场的，就沒有几个。能做成的，普遍都是人头滚滚、血流成河，抓一大批，杀一大批，才能往前推动一步。得持续几十年，才能镇得住场子，才能不被反噬。现在的问题是，对方的体系，能给他这么长的操作时间么？显然很难。

当然，我们现在的看到的都是现象，比如今天比尔盖茨被爆料了，明天FBI, IRS, 美军，甚至美联储都要被爆料了。你说有没有问题？肯定有问题。但那些个问题，最后到底是是不是“问题”，就要两说了。

换言之，你有问题，这是一回事，你的问题被定义成问题，则是另一回事。就像在企业里面，员工有问题，那就是个问题，但如果是股东有问题，那是不是问题呢？就不是董事会以外的场合，能决定的了。

美国也一样，美国只是一个伪装成国家的公司。它从建立之初，就是按照运作一家公司的方式去设计的。来自十三个州的资本们坐在一起入股。说好了我开创什么产业，将来的权力就算我的，所以才有积极性去研发。

他们是面对这么广袤的美洲大陆，大多数时候，都是依靠市场自发的力量，依靠资本的运作，一步步走到今天。美国本身就是一群资本在一片新大陆上，跑马圈地，跑出来的结果。又不是说一开始就有个美国，它就不是历史形成的。是先有的公司，而后才成立的美利坚。是1776年，托马斯·杰斐逊那帮人，关在一个没有窗户的小黑屋里讨论了很久很久，讨论出来的美国。

大众们由于接受流水线教育多年，那个脑子是按照工人标准塑造的，已经僵化了，凡事都喜欢求个答案。为啥呢？因为他们以为世界是围绕着答案转悠的。真相呢？事物运行的轨道其实是围绕着权责利转悠的，而非什么大道理。

权力是什么？本单位可以做什么，有哪些自主权，可以扶植什么经济，可以暂缓什么执行。责任是什么？有多少关于本区域居民福利的部分，要支出。

利益是什么？说穿了就是我能问存量收，还是问增量收，多少要上交，多少可以留存，以及别的方面，比如，能否卖块地，或者成立个公司，以公司名义举债。

话说当你近距离接触过一个时代的底层逻辑之后，你去看文学作品，你的视角和作者是不一样的。他看到的是当事人的身份，你看到的是身份背后运作的逻辑。他看到的是当事人的感情，你看到的是那个小说背后的背景墙下的人性。如果有一天，你能够近距离接触到很多有权的人，有钱的人，你会发现，我今天告诉你的这种看世界的角度，在这个群体里，是广泛存在的。

so，在这个群体，到最后能争得出结果么？争不出。因为只有在工人这个群体里才存在所谓正确答案，董事会层面是没有正确答案的。

真相就是：这个世界所有的门槛，都是看人下菜碟的。你非常平庸，那在导师眼里，最大的价值就是给他打打杂，跑跑数据。你善于写论文，那在导师眼里，就多写几篇，好让他作为指导老师多出现几次。如果你干活不会，发论文也不会，那他当然讨厌你，刁难你，让你延毕，因为在他眼里，你就没价值呀。

但是，你注意，我要转折了，如果你超出了这一切，他甚至都不来剥削你，不拿你当劳力，反而给你背书，推你一把，就像当年的欧阳修对待苏轼一样，老夫当避路，放你出头，推你出头。

为什么？因为我觉得你有前途呀，你小子早晚能成事儿，你就像口袋里的锥子，早晚会出人头的，那他不如做个顺水人情，为什么不卖你个人情，让你记住他呢？卖你个人情，将来的回报是巨大的，让你当免费劳动力，才值几个钱？因为这个世界本就是锦上添花，雪中送炭。这个世界本就是你有，就会给你更多，你没有，连你身上最后一件单衣也要拿走。

看到了么？这才是现实里真实的人啊。那个心胸开阔，举贤苏轼的欧阳修，与刁难无用弟子的欧阳修，很可能是同一个老师。倚天屠龙记里面，赵敏郡主带着人来犯武当山，如果作为小书童，重大事件上有立功表现，你觉得张三丰会不会较劲你哪天的作业没交全？对，这个世界上所有的门槛，都是看人下菜碟的。
