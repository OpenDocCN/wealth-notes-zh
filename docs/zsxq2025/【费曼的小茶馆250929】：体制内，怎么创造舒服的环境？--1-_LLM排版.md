
# 体制内，怎么创造舒服的环境？

## 前言

写一系列文章，让你在体制内变得情商更高、力量更强。现在这个时代，提拔不是最有性价比的。最有性价比的，其实是舒服的职场环境。你想想，你的大半辈子，都是要上班的，是舒舒服服一辈子好呢？还是辛苦劳累一辈子好？如果你能想明白这个问题，你就对提拔祛魅了。

现在为什么很多年轻人喜欢遴选到上级，其实是为了摆脱基层辛苦繁重的工 作状态。恰好是因为有些上级单位存在事少闲多的状态。如果知道上级给出的岗位，都是非常辛苦的，那年轻人又未必愿意选择了。所以，体制内的舒服环境，在当下是很稀缺的。平时很多读者咨询我，怎么才能在单位轻松点，因为他们遇到了各种问题，有的是压力大任务重，有的是人际关系紧张，有的是生活不方便。其实，这些问题都可以归结为怎么才能创造舒服的环境。今天，就说说，怎么在体制内为自己创造舒服的职场环境：

## 第一，是预判

这一点是最重要的。很多人在体制内过得辛苦，是因为不会预判。那预判什么呢？就是预判哪些单位好，哪些位置好。如果不会预判，那就会踩坑。我遇到有的读者，原本在单位里好好的，但他非要调动，调到了新的单位，发现氛围跟自己想的 不一样，太难受了，于是，又想回原来的单位。但原来的单位领导，又变了，帮不了他回去，所以，他就非常痛苦了。又有的读者，原来在自己部门好好的，但是，他觉得自己的舒服很稳了，在跟领导交往中不注意，他的有些言行可能得罪了领导，于是，被领导调到一线部门一线岗位。现在过得就比较艰难。之所以说预判，就是想提醒，你要清醒的知道自己所处的位置，如果现在的环境很好，你就 要珍惜，不要折腾；要低调，维护好 来之不易的局面。如果你想要调整岗位，或者调动，那你也要提前了解到新的单位、新的岗位是什么样的。尽量避免去到了新位置很快又后悔。

## 第二，合理地控制工作量

如果你在体制内，你一定要合理地控制工作量。因为工作量的多少，直接决定了你的幸福程度。那怎么控制呢？

首先，是守护好自己的职责边界，不属于自己的工作，尽量找理由不接。因为你接了这一次，下一次就是你干了。你拒绝的时候，你就说，你手头上刚好有几个急事在忙着，跟不过来。如果处长、科长非要你接，那你接下以后，也不要急着干。尽量拖到最后的时间节点。因为拖到后面，处长、科长可能看到你太忙，他就会找别人干了。如果你接了，干了额外的任务，那你要找机会跟领导说，不能再接这些任务了，还是给回某某接回来合适。当然，如果你是部门负责人，是处长、科长，如果你没有什么上进心了，你感觉到提拔的前景也很一般，也是同样的操作。就是尽量不接任务。其实，现在很多中层正职，都喜欢拒绝任务，不是自己部门的工作，坚决不接。所以，不要觉得有什么不好意思的。我观察过了，其实整个形势就是这样，别人这样干，你也可以这样干。

其次，就是学会装傻。 为什么有的人工作任务越干越多，有的人工作就是轻轻松松。就是因为有些人会装傻。比如，很多工作交给一些人干，领导就是不放心，总是担心收尾。其实，都是这些人故意装傻的多。所以，如果你遇到这种情况，你也要学会装傻。领导找到一些新的任务给你，你就装着懵懵懂懂，似懂非懂。如果你接手工作，你可以马马虎虎地干，干得差不多，就可以把工作交差，让领导来收尾。领导要验收、要修改你的工作，他必然会很辛苦。说不定，下次他就 不把这个任务给你了。另外，装傻的好处，就是单位遇到一些安排人员下基层的任务，又或者有重要任务的专班，领导不会安排你去。因为担心你千不好，坏了单位的口碑。

此外，你还可以把工作搞砸。现在很多人总是担心工作搞砸，其实，搞砸工作的风险是可控的。你前期评判好工作的重要性，如果你觉得搞砸了，也没啥问题，那就是可以适当搞砸。工作搞砸了，领导怕了你了，你的工作任务就会少了，又或者领导给你增加人手了。我平时接触一些读者，就是领导一不给增加人手，二又不同意他换岗位，但是工作量又巨大。这些读者，又不敢撂挑子，有多少任务就接多少任务，全都干完。长年累月高强度上班，身体和精神都承受了巨大的压力。但就是不敢搞砸工作，结果是领导吃定了他。这种痛苦一直在消耗身体的底子，日子过得很艰难。其实，任务并不是一定需要完成的。如果千不完，你就把一些工作搞砸了，把部分压力转嫁给领导。

## 第三，就是守护好来之不易的舒服环境

前面说了要做好预判，要适当减少工作量。如果你处在一个舒服的环境，一个钱多事少离家近的环境，那你就要好好珍惜了。

首先是耐得住寂寞。在舒服的环境，往往有一定的边缘化。有的人呢，又觉得自己被边缘化了，还是想展示一下自己的风格和想法。于是发发日常幸福的朋友圈，发发岁月静好的读书学习健身生活。建议这种状态还是尽量少发。因为确实容易被人感受到你 很幸福，很舒服。不排除有领导或同事眼红你，然后，想方设法给你加任务，甚至给你调岗。那你就麻烦了。

其次是跟领导和同事适当保持距离。因为有了距离，别人就不知道你的忙闲；另外又因为有距离感，这种距离感会产生严肃性，于是，又会让领导和同事尊重你。所以，我经常说，社恐也有社恐的好处。你默不作声，跟别人保持距离，其实也会让别人猜不透你的想法，不知道你的状态，反而会用一种一般心理来尊重你。在体制内，为自己创造舒适的工作环境比什么 都重要。这也是我一直说的，你要尊重自己的感觉，要有松弛感，这样的人生才是赚到的。只有这样，才能吃到体制内最大的红利。
