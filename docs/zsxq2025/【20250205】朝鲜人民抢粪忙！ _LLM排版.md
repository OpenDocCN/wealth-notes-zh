
# 朝鲜人民抢粪忙！

朝鲜长期缺乏化肥，政府为了准备春耕，每年1月要求民众提供粪便当作肥料。而今年政令要求上缴的粪便量实在太高，远远超出人均每年排便量，不仅传出黑市交易怪像，民众害怕惩罚竟沦落到被迫偷窃、争抢粪便，甚至为此打架，以求在期限到来之前凑足额度。

综合《每日邮报》报道，朝鲜长期缺乏肥料，领导人金正恩为了提高农产量，要求民众每年1月提供粪便供农耕施肥之用。一般每年粪便人均142至145公斤，然而今年朝鲜政府开出的“堆肥捐献”额度高得吓人，成人500公斤、小学生200公斤，在20日前必须完成，否则处罚，引民心恐慌。消息人士称，未达标者可能必须当众接受批斗，甚至送进恶名昭彰的劳改营。

对此，民众把脑筋动到公共厕所甚至邻居粪坑。朝鲜平安南道匿名居民透露，民众常为争抢粪便动粗斗殴，例如殷山郡有两名男子竟以斧头、铁钮打架重伤，有个工人跑到处民宅想铲走茅坑里的排泄物，屋主发现之后拿铁钮对着他猛敲，双双送医。

平安北道另一名匿名居民透露，大家必须把上缴的堆肥送到附近农场，并向各自工作单位或就读学校缴交完整填写的堆肥捐献单，这些机关学校把所有捐献凭证额度加总呈报上级。而朝鲜妇女通常未获公家分配工作，只好经由朝鲜最大的妇女组织朝鲜社会主义妇女联合会达成捐献任务，然而家庭主妇的捐献压力更大，平常必须到市场做小生意帮助收入微薄的老公养家活口还得为捐屎任务揪心，全国妇女收集粪便目标多达1公吨。

一名妇女烦恼不已，小区公厕全上锁了，我上哪儿去收集足够的粪便，而且最近生意不好，收入不够，没法买黑市配额，我们不得不偷别人猪圈旁边的猪粪堆肥，或者偷挖别人家的粪坑。她愤怒说道，国家给我们粮食不够吃，却强迫我们提供堆肥，世界上还有哪个国家的人民会为了争夺人的粪便而打架？政府要负责任。
