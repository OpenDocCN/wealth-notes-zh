
# 《孟子》第13讲：滕文公上之其命惟新

大家好，今天是2025年12月6日，乙巳年十月十七。今天是正式课，《孟子》第13讲：滕文公上之其命惟新。我试一下麦，三点钟我们准时开始。明天的时间排满了，我今天讲课也没时间讲市场，如今天晚上有时间，我就今天晚上聊几句市场，明天真的没空了。好的，我们一会儿见。

大家好，今天是2025年12月6日，乙巳年十月十七日。今天是正式课，《孟子》第13讲：滕文公上之其命惟新。这堂课其实在我过往参阅的资料里边，多有对孟子的负面评价，主要是这堂课似乎印证了司马迁那句话，司马迁对孟子一直说他是一个比较迂阔的人，就是有些脱离现实，脱离实际吧。

司马迁说孟子的学说“迂远而阔于事情”，就是迂腐、过于远大而，阔就是辽阔的阔，阔于事情，阔于眼前的事情。很多人在这篇文章里边寻找证据，而我对此事的看法截然不同。因为你读古书，特别是读经典的时候，如果你脱离了当时的环境，脱离了眼下、当下的现实，你会觉得可能他思考的问题未接地气，但你回到当时的历史和当下，你就看明白了。

好，我们一边的讲，一边的介绍一些历史的现实。滕国是一个鲁国的附属国，滕国很小，才方圆五十里地，人口也非常少，十万之众，但它仍然是一个，国虽小亦是国呀，所以这里边涉及到孟子与滕文公的交往，孟子对滕国的一些建议。滕国是所有的接纳孟子，并且严格按照孟子思想去治理的一个经典案例，当然最后滕亡了。

也就是说孟子的仁政的这块实验田最后在战国的时期，因为它太小了，它终于还是没有成功。这个实验室，怎么好说呢，就是说它不是一个成功的案例，但这一段对话对当下却有非常重要的意义。孟子与滕文公交往的时候是战国的后半段了，后半期、后半截，那个时候儒家的思想被遮蔽了，因为孔子去世的时间已经比较久了，这个时候是世上流行的思想实际上是非杨即朱。

就是当时道家的思想和墨家的思想盛极一时，不是道家，就是墨家。如果今天的人理解不了道家，那么你就设想一下子自由主义。自由主义就是这种比较无政府的这种自由主义，就更强调市场的作用，希望政府不要干预，小国寡民老死不相往来，让市场发挥作用就可以了。墨家是个什么状况呢?其实墨家也是，如果说道家是右的无政府主义，墨家就是左的无政府主义，他更强调人民自己结成公社来自理。

是在当时这样的一个环境下，孟子提出他的主张和想法。所以这里边每一句话它都有针对性的，如果你没有这样一个背景的解读，有的时候你会出现严重的误读。我先将第一段给大家讲一下子，因为量太大，所以不能作为古文来讲了。“滕文公为世子将之楚，过宋而见孟子。孟子道性善，言必称尧舜。”这个时候滕文公还是太子，他将出访楚国，路过宋国的时候专程来拜见孟子，孟子给他论述了性善论。

“言必称尧舜”，好多人说孟子道性善，并且言必称尧舜，这对滕文公有什么帮助吗?我刚才讲了，因为那个时候大家迷恋的是道家的自由主义，或者是墨家的集体主义，就是他要求就是底层的公社，他对政府是一种非常反抗的抗拒，这两个都是无政府主义，一个是极右的，一个是极左的。这个时候孟子是提出大一统的主张，他还是主张大一统、中央集权，主张走社会主义道路。为什么要“道性善”?因为他很了解小国的太子滕文公没有这种自信。

“道性善，言必称尧舜”，用我的话概述就想去告给他，只要你有善良的本性，你就可以成为尧舜。后来是教员写诗：六亿神州尽舜尧，说的也的这个意思，只要你心存善念，皆可为尧舜，就是鼓励滕文公，说你可以成为尧舜那样伟大的人物，你是可以有一番作为的。这在当时的历史时期，面对对小国的太子滕文公缺乏足够的自信，他又将会成为滕国的国王，所以他必须给予他一个基本的鼓励。

世子自楚反，复见孟子。孟子曰：“世子疑吾言乎?夫道一而已矣。”孟子发现世子不是很相信孟子的这套理论，就是滕文公并不是特别是认同孟子的理论，就是有怀疑，这里边他用了三个比喻。一个是：“成谓齐景公曰： ‘彼丈夫也，我丈夫也，吾何畏彼哉?’”就是最有名的这个武士成覷,他曾经跟齐景公说过，他是大丈夫，我也是大丈夫，我为什么要害怕呢?我怎么会畏惧他呢?他就是勇气，在强调勇气。

颜渊曰：“舜何?人也。予何?人也。有为者亦若是。”颜渊大家都知道了。这里边实际上是在鼓励世子：“彼丈夫，我丈夫”,大家都是大丈夫，他能行，我也行。颜渊说：“舜何?人也。予何?人也。有为者亦若是。”只要你想有所作为，都可以做得到。

公明仪是曾子学生，也有人说公明仪也曾是孟子的老师，但我无法考证，时间也不是很容易对得上。公明仪说：“文王我师也，周公岂欺我哉?”公明仪的话是他以文王为老师，他认为周公是没有欺骗他的，就是周公说的就是我要寻求的那个榜样啊。

“今滕，绝长补短将五十里也，犹可以为善国。”滕县，把这个边边角角切一下子合到一起，“将五十里”,就是五十里方圆。“犹可以为善国”，你是可以把国家治理得很好的。《书》曰：“若药不瞑眩，厥疾不瘳。”厥疾，厥就是昏过去了，就是病重得很厉害，这个药要虎狼之药，瞑眩就是吃完以后天旋地转这种药才能治好你这个重的病。

“《书》曰”,《书经》是这么说的。那么孟子是打算给滕文公开一剂治国良方了，而且这个方、这个药还是重药，“药不瞑眩，厥疾不瘳。”

第二个部分就比较有意思了，这个是争议最大。就是这是第二部分整个说的都是什么呢?都在说定公死了以后，文公给他父亲办丧礼的事情。好多人对这一段就意见很大，就是“三年之丧”这个事情，说有这么大的意义吗?一会儿我来讲。

“滕定公薨，世子谓然友曰：‘昔者孟子尝与我言于宋，于心终不忘。今也不幸至于大故，吾欲使子问于孟子，然后行事。’”显然文公对孟子高度信任，所以他父亲死了以后，他让然友去问问孟子，我该怎么处理这个后事，怎么处理这个葬礼的事情。

然友之邹问于孟子”,然友专门去邹这个地方来向孟子请教。“孟子曰： ‘不亦善乎!′”太好了，他对滕文公的学习态度非常肯定。“亲丧固所自尽也”，就是家里边的这个父亲走了，必须要尽孝尽礼，“自尽”要亲自尽孝尽礼。

曾子曰：‘生，事之以礼；死，葬之以礼、祭之以礼，可谓孝矣。’”引用曾子的话(曾参的话)就是活着的时候要事之以礼，死要葬之以礼、祭之以礼，这个才算是尽孝，就是尽礼才能算尽孝，或者是以礼尽孝，这是中国人在孟子之后形成儒家的一个非常重要的事情。

“诸侯之礼吾未之学也，虽然，吾尝闻之矣：三年之丧，齐疏之服，旰粥之食，自天子达于庶人，三代共之。”意思非常明确，就是诸侯的礼，我没有学过，我没有研究。虽然我听说过，应该是三年之丧穿粗布衣，喝稀粥，从天子到庶人，夏、商、周三代都是这样做的。所以我觉得，就是孟子说我认为你应该是服丧三年，“齐疏之服，干粥之食，服丧三年”。

“然友反命”，然友回去告诉滕文公定为三年之丧。

“父兄百官皆不欲”，他的长辈们都不同意，百官也不同意。“曰：‘吾宗国鲁先君莫之行。’”我们的宗主国鲁先君没有这样做。“吾先君亦莫之行也”，我们的国王的父亲也不是这样的。“至于子之身而反之，不可”，到你这儿来，你这不是就和他们不一样了吗，这样不好。

“且志曰”，就是我们这个国家的历史记载说“丧祭从先祖”，这个办丧祭，后人要学习你的长辈，长辈怎么做你就怎么做。滕文公对百官和他们的回答说：“吾有所受之也。”什么意思呢?就是我这样做是有原因的，我是有依据的，他没有讲是依据孟子的看法。

然后滕文公对然友说：“吾他日未尝学问”，我呢，平时也没怎么好好做学问，喜欢驰马试剑。“今也父兄百官不我足也”，他们不太满意；“恐其不能尽于大事，子为我问孟子。”恐怕这件事情就是丧礼这个事情我怕办不好，你再去问问孟子。

“然友复之邹问孟子”，然友又去邹问孟子。

“孟子曰：'然，不可以他求者也。’”孟子说，这件事情必须君王自决，你不要去问其他人，也不用参考任何人的意见，什么志，什么鲁先君，什么吾先君全都不要参考，就是你来定——君王自决。
