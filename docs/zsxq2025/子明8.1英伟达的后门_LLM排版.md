
# 子明8.1英伟达的后门

## 发布时间：2025年8月1日
昨夜网信办约谈英伟达这事儿， 一句话解释：
“金杯同汝饮，白刃不相饶”,
在经过数轮中美谈判，
解决了英伟达芯片入华后，
旋即给黄仁勋一击杀威棒，让他好自为之。
美国今年在推动新的芯片立法，要求在出口的芯
片中植入后门，也让兔子陷入到两难的抉择之中，
现如今的办法就是养肥一个美国利益集团，
让这个利益集团为了自身的利益去国会山游说，
阻挠芯片立法的相关后门内容，
否则将面对利润巨大的市场拱手让人。

属于标准的“美国反对美国”。
这个事儿的做法是非常聪明的，
黄仁勋也愿意拿出利润的百分之几来游说，
但我方的实操层面有点糙，
英伟达的公关也没啥经验，
一个不小心搞成了吃瓜的大新闻，
把战场从国会山搞到了兔子的键政圈，
估计近期会想办法降舆论热度。
