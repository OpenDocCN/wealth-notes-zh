
# 你就是不懂得牛市的基本原理

正如剑导剧本所透露的那样，行情进入了“主动调整”。有些观众却不淡定了，在此“故意调整”的当下，纷纷私信表示自己的股不涨，已经受不了了。这一方面说明了观众入戏太深，另一方面说明了智力的欠缺。

金主、制片人、导演合谋做一部大戏的目的到底是什么呢？难道真的为了所谓的“艺术”吗？去他妈的狗屁艺术吧，那都是忽悠你上钩的。真正的目的无非是如下几条：

1. 骗观众的钱；
2. 把金主的钱洗白；
3. 趁机潜规则……

作为演员，你有必要着急导演还没有潜规则你吗？那种着急咋还不涨的，就是不懂牛市这场大戏的基本原理。只可着一个演员捧，那怎么能骗到不喜欢这个演员的观众的钱呢？所有演员都要至少捧一遍，所有板块都要至少有一轮大行情。这样才能做到“应割尽割”。

捧你，你得先付出点什么吧？既然是牛市这场大戏，都会至少干一遍的。现在没干你，那是更大更粗的暴击在后面。那些年轻的刚进剧组的演员就喜欢乱换房间，巴望着住的离导演近一些。一个是想着早点儿被潜后捧红，一个是想着能多被临幸几次。而历次牛市大戏中，这样想的基本都失败了。

最可能的情况是，导演已经玩遍了附近的房间，你刚好搬过去等着早点被大用。而导演感觉附近的房间早就玩腻了，这样你刚好错过了。而老演员就在那趴着，一动都不动，因为她们深知早晚会被导演敲门。总之，牛市是一定会艹到你的。

但我知道，告诉你也没用。你很年轻，又痒得不行，会乱动的。

这轮调整是上面故意的降温之举，假的，不必当真。
