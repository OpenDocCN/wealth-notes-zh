
# 【20250121米联储见闻】——历天又复入西海 (1)

懂王归来，宣誓后的第一件事，就是开始翻拜登的烧饼，拜登任期内80%的行政令被懂王彻底推翻，美利坚，迎懂王，一夜之间就换了天。作为失而复得的美国总统，川普的宣誓典礼更像是一场战前动员，他承诺将立即采取一系列行动彻底改变国家进程，开启新的“美国黄金时代”。而经济学人对川普即将开始的新任期扣上了“新美帝国主义”，看来尘封在六七十年报纸上关于美国的描述次将再次重出江湖。

在四年前被赶下台后，川普之所以可以回归，一个很重要的原因在于，按照过去的资本主义路线，哪怕是玩到极致也不能解决美国现有的问题，所以不管川普是否如何承诺，是否愿意，他都必须要对美国现行政策做出大幅调整，川普的愿景是重塑美国，重建共和国，清除国家的敌人和他自己的敌人。可惜这么伟大的愿景和工作，美国人民只给川普留了四年时间，当然，如果川普干得好，可能他的副总统会接替他成为下一任总统，又或者美国还有没有下一任总统。

川普说：“我最近的当选是是人民授权我全面、彻底地扭转可怕的背叛以及所有这些已经发生的背叛，并将人民的信仰、财富、民主以及他们的自由还给他们”，在时长29分钟的就职演说中，川普宣布“从这一刻起，美国的衰落结束了。”在演讲中，川普用去年七月被刺杀事件再次证明了自己是天命所归。

川普来势汹汹的复仇意愿，让拜登在任期的最后时刻还在安排赦免，包括拜登两个兄弟在内的五名家庭成员，以及前众议员利兹·切尼、退役将军马克·米利，安东尼·福奇博士。拜登是真的怕，怕川普不顾一切的打击报复。

懂王在就职典礼结束后迅速采取行动，通过多达100项命令和行动重新施加他对政府的影响。他誓言立即宣布南部边境进入国家紧急状态，并派遣军队守卫边境。他说，他将终止促进多样性、公平和包容性的政府项目，将把墨西哥湾改名为美国湾，并承诺夺取巴拿马运河。现年78岁的川普成为就职总统年龄最大的人，超过了四年前宣誓就职时比他现在的年龄小五个月的拜登。40岁的万斯则成为历史上第三年轻的副总统。川普的就职口号是，要开启美国的黄金时代，重启“美国优先”的政策，让美国比任何时候都更加伟大。

100多项命令中，最主要的8项主要行政命令。第一，任命政府官员，任命数十名内阁级官员及部分政府部门的官员。另外，将数千名联邦政府公务员重新归类为政治雇员，使他们更容易被解雇。第二，废除拜登政府行政命令，一次性废除前总统拜登实施的78项行政指施，其中有至少十几项支持种族平等和打击歧视的措施被废除。第三，退出《巴黎气候协定》、世界卫生组织，再次退出2015年落实《巴黎气候协定》，并指会致函联合国通报这项决定。同时退出世界卫生组织(WTO)。第四，赦免国会骚乱涉案人士，赦免1500多名因2021年1月国会骚乱而被定罪的人。第五，宣布国家紧急状态，声称美墨边境的非法移民问题以及能源问题是国家紧急状态，将派遣军队前往边境，把毒贩集团列为外国恐怖主义组织。将支持美国化石能源产业，还包括取消对阿拉斯加石油钻探的限制以及取消暂停天然气出口。第六，推迟TikTok禁令75天，下令司法部长在75日内不要对TikTok采取行动。第七，废除跨性别权利，只承认男女两种性别，并停止联邦政府内部的多元化、公平性和包容性计划(DEI)，并要求政府使用生理性别sex，取代社会性别gender字眼。第八，终止美国出生公民权，由即日起30天后，若孩子不是父母至少其中一方为美国公民或绿卡持有人，当局将拒对该名在美国出生的儿童提供美国公民身份，包括护照，结束美国的出生公民权。

一言以蔽之，彻底回归美国的保守主义操作，而且刀口向内，懂王上线的第一波政策主要还是针对美国内政。他当选后高调宣布要在上任第一天向加拿大和墨西哥所有商品实施25%关税，向兔子加10%关税，并称欧盟如果不多买美国石油、天然气的话将以关税待之，甚至警告丹麦如果不在格陵兰拥有权问题上让步将对其收高关税，但昨天的行政令内没有关于关税的部分，只签署行政命令责成商务部、财政部等部门对加、墨、中等国的各类贸易议题进行调查，并研究如何成立对外收入部，对美(External Revenue Bureau)国国税局征收关税。

海外机构分析，懂王没有一上来就挥舞关税大棒，可能跟关税鸽派财长人选贝森特有关，而能够说服懂王谨慎使用关税武器的唯一原因可能是美股，川普一直以美股股价作为他管理经济的成功指标，关税明显会对不少企业经营造成打击，加重市场对通胀重临和联储局减慢减息步伐的担忧，最近已多次导致美股普遍下挫，川普也不希望自己刚一上台就迎来美股大跌。

但是这并不意味着川普会放弃关税这个工具，相反，川普在就职演说中表明他将立即开始彻底改革美国的贸易体系，以保护美国工人和家庭，不会向美国公民征税来让其他国家富裕起来，而是会向其他国家征收关税和税款来让美国公民富有起来，将关税的收入以立法方式确立，从而让其新增收入可以用来抵销减税，可能是川普的考虑之一。而且在白宫的行政命令签字场合上，川普声言他考虑将会在2月1日落实针对加拿大和墨西哥的25%关税。至于针对兔子的关税，川普似乎有意将它同TikTok的问题扯上关系。在白宫行政命令签字场合上，川普称如果他与TikTok达成了协议而兔子不批准的话，他会向兔子加关税。显然关税现在是川普用来勒索兔子的一个武器，所以川普不打算一下子把子弹打完。对于一心想搞钱的川普而言，越是没有交易价值的盟友可能越早成为关税打击的对象，比如墨加，反而是兔子因为还有别的价值，关税的棒子反而是举而不落。

在川普就任之时，老财长耶伦在卸任前签署了最后一项命令：美国财政部将从1月21日至3月14日启动暂停发行债券期。暂停为公务员退休和残疾基金以及邮政退休人员健康福利基金注入新资金，以确保政府在现有资金框架下继续运转。也就是美国联邦政府为筹集资金而发行的债券总额又接近限额了，财政部只能暂停债券发行。如果不暂停的话，就会突破债务上限而出现债务违约。暂停发行债券期是在债务上限约束下的一种紧急应对措施，为国会讨论和通过提高债务上限争取时间。现在的美国政府是靠借钱度日的，在不能发债融资的情况下，财政资金会越来越紧张。

钱从哪里来呢？为了维持政府的正常运转，耶伦这项措施的第二条就是暂停公务员退休和残疾基金，以及邮政退休人员健康福利基金注入新资金，美联储也没有余粮了。

耶伦这四年财长，全世界的推广美债，各种苦口婆心，但是仍挡不住美债抛售潮一波接一波，10年期美国国债收益率不断向上试探正逼近5%的重要关口。2024年12月20日，美国财政部发布2024年10月国际资本流动报告(TIC)，数据显示，美债的海外持仓总额在连续五个月的上涨后突然锐减，直接下降了772亿美元，跌至8.6万亿美元。这场美债抛售的主力军，正是兔子。数据显示，兔子在2024年10月单月就抛售了119亿美元美债，2024年前十个月，兔子累计减持额已经达到562亿美元。除了兔子，扛不住货币贬值压力的日本也选择减持美债，去年10月，日本的美国国债持仓按月减少206亿美元，降至1.1027万亿美元，这是日本在过去七个月中的第六次减持美债。此外，英国、开曼群岛、加拿大、比利时和瑞士等国家和地区也在当月减持美债。这一减持潮显然不仅是个别现象，而是一个全球范围内的趋势。这种变化不仅反映在具体持仓数据中，国际货币基金组织(IMF)的最新报告也进一步揭示：美元在全球外汇储备中的份额已从曾经的85%以上下降至目前的58%，创下30年来的新低。

大家要记得有一个荷塘效应，兔子的美债持有量距离最高峰期已经下降了一半，美元的份额也几乎掉了40%，某些时刻的到来，可能会以难以想象的方式和速度发生，就像tt难民一夜之间冲进小红书一样，我们可能会在有生之年见证一个又一个难以想象的奇观。

长期以来，10年期美债被视为全球无风险资产的基准，因美国国债具有极高的流动性和安全性，几乎没有什么违约风险。但长期的财政赤字让美国的债务越堆越高，美国政府通常通过发行新的债券来偿还到期的旧债，而这些新债又带来更高的利息支出，美国实际已经陷入债务的恶性循环。靠着帝国军事余威在强撑美债，但是这个游戏还能玩多久，大家不知道，但这个游戏肯定完不下，大家是确定的。1990年时，美国政府债务总额约3.2万亿美元，截至2025年1月10日，美国未偿债务总额已达36.17万亿美元。2024年一年，债务就增加了2.22万亿美元，其中后四个月更是激增1.12万亿美元，美国未偿债务规模已大幅超出33.1万亿美元的法定债务上限，更高的利率让美国政府的借贷成本如滚雪球般不断攀升，36万亿美元的超级债务，川普的疯狂操作，即便大家嘴上不说，心里都明白，美国要暴力赖账是迟早的事情，现在关键是谁会成为美国的刀下鬼。目前看来，欺软怕硬是大概率事件，中俄，川普估计都不会硬碰，但是盟友犹如脆片点心，能吃一口算一口，懂王是不会客气的，如果台湾可以卖个好价钱，我相信也不是不可能的，毕竟债多压身，卖点祖产还债，都是常规操作，到了必要的时候，压根不是美国想不想卖，甚至还要看兔子想不想买了。
