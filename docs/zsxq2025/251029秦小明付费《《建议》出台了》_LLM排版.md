
# 《建议》出台了
# 251029 秦小明
《建议》出台了。
《建议》是在之前《公报》的基础上对十五五进一步细化。《公报》里一笔带过的内容这里展开来讲了，鹅少整理了几个重点：

1) 现代产业体系
1、优化传统产业。特意提到矿业、冶金、化工、轻工、纺织、机械、船舶、建筑等+质量。
2、新兴产业和未来产业。蓝海产业有：新能源、新材料、航空航天、低空经济。量子科技、生物制造、氢能核能、脑机接口、具身智能、6G。
3、服务业。生产性服务业向专业性服务业转化。意思是发展高端服务业，高附加值服务业。
4、基础设施。算力列入新型基础设施。

2) 科技
1、原始创新和核心技术。集成电路、工业母机、高端仪器、基础软件、先进材料、生物制造
2、科技+产业。
3、数字中国。人工智能+

3) 消费
通过就业、收入、预期来促消费以新供给促消费。提到直达消费者的普惠政策力度。暗示消费补贴

4) 反内卷
在坚决破除阻碍全国统一大市场堵点卡点中提到。综合整治内卷式竞争

5) 财政
注意提到税收：完善地方税、直接税体系，健全经营所得、资本所得、财产所得税。适当加强照耀事权。地方财权事权未来持续削弱

6) 提到金融强国
补全了公告中没提的金融，但是在市场经济体制这一大框架下提到的。而且是从银行、资本市场制度、金融机构、监管这些角度提到的。不要过于yy。

7) 民生
就业和社保是下一阶段的重点难点。房地产在此段提及。优化保障性住房供给，满足城镇工薪群体和各类困难家庭基本住房需求。增加改善性住房供给。提升房屋质量。房地产重点是住，是保障，是质不是量，可能涉及收储。不应有过多yy。

8) 环保/能源
风光水核多能并举。

9) 安全
建军一百年目标
推进祖国统一大业。
