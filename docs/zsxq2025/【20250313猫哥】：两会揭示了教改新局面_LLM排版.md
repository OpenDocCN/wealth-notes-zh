
# 两会揭示了教改新局面

## 两会教育板块分析

今年两会的政府工作报告中，教育板块虽然被放在2025年政府工作任务的第三项，但其社会关注度依然很高。本期将结合教育板块的重点内容，探讨其背后的逻辑和变局。

### "免费"与"扩容"问题

"逐步推行免费学前教育"这一政策多年来的呼声终于正式落地。这项政策未来财政投入巨大，以2022-2024年新生儿约2900万计算，若全面铺开，按平均1200元/人/月计算，新增开支一年就超3000亿，几乎相当于2024年军费开支的20%。

推进这项改革主要有三个目的：
1. 作为催生促生的配套措施
2. 通过直接降低育儿成本，提高家庭其他支出占比，服务扩大消费这一头等工作任务
3. 更深层的逻辑是通过延长普惠教育链条，重构"教育公平"的起点。根据德国经验，免费教育覆盖3岁以上儿童后，弱势家庭儿童认知能力差距缩小了40%。

此外，政策还包括"扩大高中阶段教育学位供给"和"扎实推进优质本科扩容"。

过去几年，"让50%初中生上不了高中"的中考分流政策成为悬在家长头顶的达摩克利斯之剑，导致小学卷、初中更卷。其核心原因在于我国职高虽然办了几十年，但一直未能扭转"差生收容所"的印象。

在这样的背景下，但凡手头有"余粮"的家庭都会陷入"中考焦虑"的军备竞赛，导致现在小学5-6年级就搞初中早培的趋势愈演愈烈。

每一朵花，盛开的季节不一样。中考、高考的目的是选拔人才，但不能矫枉过正，变成过早的区别人才。比如有部分学生，青少年时代成绩不咋滴，但到了大学阶段突然开窍的鲜活例子比比皆是。

扩大供给并不代表没门槛，只是在出生人口下降带来的教育资源冗余的大背景下，这个政策既是对前期分流政策的纠偏，又是给焦虑吃降压药。

既然高中扩容，自然而然大学本科也要适当做到同步匹配，但本科扩容有个前置条件，就是优质。何为优质，就是本科专业必须适配社会发展，学以致用，否则就会面对生源枯竭，不改就淘汰。

### "融合"问题

如果说"免费"和"扩容"这两个未来教改方向通俗易懂，内容明确，那么关于"推进职普融通、产教融合，增强职业教育适应性。分类推进高校改革"就让大家云里雾里。但这才是本次教改的重点。

在两会期间，教育部部长怀进鹏谈到教改问题时提到DeepSeek和机器人在最近一段时间引起国内外广泛关注，说明了中国科技创新和人才培养的效果，但也提出了面对重大科技变化和产业变革，教育如何应对的问题。

个人理解现在卷小学择校、卷中高考，最终目的还是就业。以前我们知道职教以及大学教育已经和社会发展脱节，知道要改，但在人工智能时代飞速发展下，改革从知道要改变成了不得不立即启动。

### 政策方向

第一，职教突围。最近几年有些职校办得有声有色，但在经济下行的大背景下被淹没。今年政策强调"职普融通"，本质上是将职校从"学历链底端"推向"技能金字塔顶端"。

在德国，技工收入可达教授水平的70%，而我国仅30%。突出职教地位，包括推动"职业技能等级证书全国互认"，正是判断到未来将出现"高级蓝领"收入碾压普通白领的趋势。

现在江苏、山东等地试点的"中职-高职-本科"贯通培养模式，让学生用5-7年获得"技能本科文凭"。这种"学历+证书"双轨制，不仅对于很多普通家庭很有吸引力，而且正在得到企业认可，逐步正在让职教从"教育备胎"变身到"就业王牌"。

第二，产教融合，增强职业教育适应性。现在已经有了一些积极的化学反应。比如特斯拉在上海的"超级工厂班"，学生在大二大三阶段就能接触Model Y的生产线。这种企业反向"定制"人才的案例，让学生在毕业时已经掌握了相对成熟的生产技术，起薪超8000元。

同时，职教专业设置还高度动态化。比如新能源汽车进入爆发期后，有职校在3个月内就能开设电池维护、智能驾驶等课程。当大学还在纠结毕业论文时，职教早已学会用"就业率×起薪"的公式计算专业存亡。

第三，分类推进高校改革。去年高考，大量非双一流理工类大学的录取分数远超某些985大学的冷门专业。同时，2024年全国高校撤销专业榜单上，公共事业管理、市场营销、广播电视学位列前三，而人工智能、集成电路等专业的增设数量暴涨300%。

高校不改革不行，残酷的现实摆在面前。一些985大学社会学专业毕业生转行率高达67%，而深圳职业技术学院芯片封装专业校招年薪已突破15万。市场正在用真金白银重构"体面工作"的定义。可以说，从今年起高校真正即将迎来一场震荡，很多专业要接受来自企业、社会的多重考验。

### 教改挑战

2025年的教改政策还存在不小挑战。比如，要破除"蓝领歧视"，需要收入分配、社会保障等系统改革配合。

比如，我们还有很多地区的职校还面临"三无"困局。无设备(实训基地空置率超40%)、无师资(60%教师从普高转岗)、无企业合作。

比如，家长和学生的观念转变还需要时间。有调查显示，65%的家庭支持职教改革，但仅12%愿意让孩子首选职教。至少目前大家还处于一种"精神分裂"状态，一边承认"职教本科"，一边挤破头买学区房。

但不管怎么说，虽然今年教改表面是应对生育率、就业率的应急之举，是一场针对教育焦虑和社会生育压力的精准手术，实则是中国教育从"学历社会"向"能力社会"转型的里程碑。

政府工作报告里的每一个政策术语，最终都会化作千万人命运转折的推手。个人认为这场变革早已超出教育本身的范畴，它与人口结构、技术革命、产业转型同频共振。

学生不再盲目刷题，学校不再追求大而全，而是深耕"产业需要的专业"。这场变革没有浪漫主义，只有适者生存的冷酷逻辑。但或许，这正是教育回归本质的开始，一个"技能定价"的新秩序正在形成。

未来的幸存者可能是那些提前嗅到风向的人，家长不要再去为了上985、211而使劲卷，而是应该把关注力放在"孩子适合什么赛道"。
