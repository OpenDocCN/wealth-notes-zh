
# 世界经济论坛首席经济学家展望：2025年1月

## 执行摘要
2025年1月出版的首席经济学家展望伴随着通胀预期的显著上升。这标志着全球可能充满事件的一年。多数首席经济学家（56%）预计全球经济将在未来一年走弱，而17%的人预计会改善。多数人（61%）认为这种影响将是长期性的，而不是短期干扰。在即将到来的新政府领导下，首席经济学家预计贸易、移民、放松管制、财政政策、产业政策和外交政策将发生重大变化。大多数人预计通胀和公共债务水平将上升，以及股市上涨。全球经济格局日益碎片化，这在商品贸易中尤其明显，94%的首席经济学家预计未来三年将进一步碎片化。重大多数还预计劳动力流动和技术及数据的转移障碍将增加。地缘政治竞争和国内政策选择被视为当前碎片化趋势的关键驱动因素，突出了政治和经济因素之间互动日益重要。

## 1. 下行风险上升
全球经济前景依然低迷，下行风险加剧，这主要是因为围绕11月美国总统大选经济影响的 uncertainty 加剧。根据世界经济论坛最新对首席经济学家的调查，多数人（56%）预计全球经济将在未来一年走弱，而17%的人预计会改善。与2024年8月的上次调查相比，对明年的预期有所缓和。国际货币基金组织最新预测显示，今年全球经济增长率为3.2%，与2024年持平，未来五年将小幅放缓至3.1%。这反映了44%的首席经济学家对2025年美国经济增长的预期——三倍于预期短期增长的人数。

## 2. 美国成为焦点
首席经济学家展望建立在最新的政策发展研究以及世界经济论坛新经济与社会中心组织的公共和私营部门领先首席经济学家咨询和调查的基础上。本简报旨在总结当前经济环境的形成轮廓，并确定政策制定者和商界领袖为应对地缘经济和地缘政治事件对全球经济造成的复合冲击而采取进一步行动的优先事项。本简报中介绍的调查于2024年11月下旬进行。

## 3. 全球一体化面临压力
全球一体化正在承受日益增大的压力。跨国公司预计将采取一系列适应措施，包括改变供应链和组织结构调整。一般来说，发达经济体从服务贸易中受益更多，但大多数首席经济学家指出，服务业作为经济发展驱动力的作用日益重要。

## 4. 贸易动荡的一年
首席经济学家预计贸易战动态将在未来几年加剧，这既包括美国和中国之间，也包括更广泛的领域。然而，预期也是，发达国家倾向于比发展中国家从服务贸易中受益更多。
