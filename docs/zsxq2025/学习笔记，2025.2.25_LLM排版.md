
# 学习笔记，2025.2.25

1. 现在信息爆炸，看上去每个人都是无所不知，上到怎么上火星，下到怎么社工人的隐私，只要想知道，就能得到足够的信息。但对大多数人来说，这样的信息意义不大，甚至知道的越多就会越焦虑。真正能让自己变好的信息，是那些和自己生活切实相关的基本常识，这恰好是大多人不愿意去了解、学习的，因为“不够酷”“不够震撼”“不够刺激”。

2. 和共识不一样的人发生冲突，你越想说服他，他就越不愿意接受你的说法，任你有再大的劲头也发不出力，越用力自己越难受。不要太过在意别人对自己的认同，只筛选不改变，多去和那些和自己同频的人交流。

3. 我们可以严格要求自己，对他人一视同仁，但是不能以同样的标准要求别人，要允许别人对自己区别对待，允许别人有各种人格上的缺陷，你可以选择的是，是否继续和这种人交往。己所不欲，勿施于人，己所欲，也勿施于人，每个人都有自己的人生课题，允许自己做自己，也允许别人做别人。

4. 很多普通人对自己没做成大事常找的一个理由就是：怀才不遇。觉得没能给自己展示才华的机会和平台。以前或许有怀才不遇，因为信息交流很闭塞，多数人一辈子都走不出多远，缺乏被人看到的机会。互联网时代，每个人都可以去面向全世界展示自己的才华，这么开放的环境下还做不成事，只能是才还不够，得多练。

5. 短视频刷多了，很容易让人有急功近利的心态，觉得自己七天、半月做不成事，就是不正常，那些博主说的没有基础可以每天赚多少万才是正常状态。越想速成，越容易钻进博主给自己设的套，只要不把“速成”的想法丢掉，就会有钻不玩的套。十年的事十年做，这是常识，告诉你一年能把十年事做成的，都是骗子。

6. 一个人伤害你，大概率是他权衡利弊后的选择，什么“冲动了”“喝多了”都是借口。比如，你可以看一个人喝醉酒会不会揍他老板，会不会揍比他壮比他狠的人，如果不会，那他就没醉，是在找自己可以得罪的人去欺负。

7. 能量低的时候，不要让自己闲下来。低能量时做什么事效率都不高，与其纠结内耗白白浪费时间，不如去做能让你心情好转的事情。天气不好，就追会儿剧，看会儿小说，点个香薰泡个澡，放首音乐收拾房间。天气好，就出门闲逛，喝咖啡吃蛋糕，找个公园晒晒太阳。懒得走路，就去看看电影，做做美甲，甚至是找一个可以观景的餐厅，喝茶发呆。改变状态最快的方式，就是换环境，高能量场所能迅速拉回状态。记住，人要学会求助。但关键时刻，只有你能救自己。

8. 我希望今年我的关键词是“稳定”。1.收入稳定。2.情绪稳定。3.健康稳定。4.一切稳定。

9. 有个心理学叫空杯效应。把我们比成一个杯子，如果装的太满，就很难再装进去水。我们如果想的太多，或者以“我已经知道很多”的心态去学习新的东西，就会不由自主的懒惰，或是力气用错了方向。在学习新的东西之前，不管是在原有知识的积累还是全新领域的学习，都要以我现在是个空着的杯子的心态去装水。

10. 聪明人不会跟任何人走得太近，因为他明白，近则不恭。
