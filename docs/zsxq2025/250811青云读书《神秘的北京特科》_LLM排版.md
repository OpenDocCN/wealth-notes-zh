
# 神秘的北京特科

大家都知道在我党的历史上，尤其是在大革命时期，有过中央特科，就是在上海设立的，我上个月还和两个朋友去了一趟上海特科的纪念馆，因为中央特科遗址当时也是爱棠新村案的案发现场，所以这个地方还是比较值得去看一看的。虽然介绍的内容是比较少的，但还是可以去看一下，因为它是免费的，如果你们来上海可以去看看，在静安寺后面。大家对中央特科都比较熟，周恩来、顾顺章、陈赓、李克农，包括钱三杰、胡底、钱壮飞都比较熟。

但是在中共的历史上其实一共有过三个特科，一个是大家都知道的中央特科，另一个叫北京特科，还有一个更神秘的叫国际特科。今天我们来讲的就是北京特科，原来叫北平特科，因为那时候北京不叫北京，叫北平，所以也叫北平特科。北平特科是上个世纪30年代，尤其是顾顺章叛变革命，在上海的党组织遭到了严重的破坏以后，由陈赓在北方建立的一个情报组织。

当时周恩来对北京特科有过一个这样的指示，就是不与党的地方组织发生关系，单独进行情报、兵运、保卫、锄奸等运动，受到中共中央和中共北方局的双重领导。这里面涌现了两个比较有代表性的人物，一个叫潘东周，也叫潘文郁，另外一个就是小说《红岩》里面华子良的原型韩子栋，这两个人都是北京特科里面活生生的人物。

大家都知道1931年4月份，已经是顾顺章叛变革命以后，在钱壮飞的掩护之下，在上海的中共中央并没有被一网打尽，但是也遭到了严重的破坏。当时陈赓是中央特科二科的科长，也就是情报科的科长，他和顾顺章同时在苏联契卡，也就是后来的克格勃，接受了完整的情报和保卫工作的训练。我在前几次节目里面讲过，顾顺章在契卡的时候主要学习的是锄奸和保卫，也就是所谓的暗杀行动。陈赓脑子比较好使，而且读过书，所以陈赓主要学习的是情报搜集、整理分析，所以这两个人都是由契卡，当时还不叫克格勃叫契卡，就是所谓的肃反委员会专业培训过的。

当时他们两个人完成受训以后就回到了国内，参加国内的中共的活动，顾顺章由于其艺高人胆大，当时在安全保卫和锄奸行动中，顾顺章确实发挥了很大的作用，后来也跻身了中共中央的高层。

由于上海当时是中共中央所在地，也是中共中央华东局的所在地，也是中共江苏省委的所在地，所以上海当时非常重要，因为有三个机关都在上海，一个是中共中央的机关，一个是中共中央华东局机关，一个是中共中央江苏省委。虽然上海当时是称为中华民国特别市，但是它在行政关系上还是归江苏管的，所以当时把江苏省委也设立在了上海。顾顺章叛变以后，周恩来就跟陈赓说到北方去，当时他就到北方去开辟新的革命的根据地，陈赓带着当时他的妻子王根英和三岁的儿子陈知非，一起坐船从上海到天津，同行的里面还有两个人，一个叫陈仰山，一个叫张麻子。

陈仰山是我党隐蔽战线上非常有名的一位情报人员，解放以后他也担任了高层的领导职务，一直来说陈仰山是属于特科系统的，因为当时中共的情报系统就是谁领导由谁的名字来命名。比如说像潘汉年系统，就是潘系统，像西安有王克坚系统，陈赓有陈赓系统，当时陈仰山就属于中央特科领导下的，但是他后来也到了北京特科从事前期的筹备工作。

陈仰山是周恩来特意安排，一路上一方面是保护陈赓和他的全家，第二方面是要求陈仰山跟陈赓到北方去展开情报工作。陈赓的回忆录里面曾经有过这样的记载，就是以天津北平为中心的北方党组织也遭到了严重的破坏，很多同志被捕，当时中共中央给陈赓和陈仰山三个任务，第一是营救被捕的同志，第二看看能不能在天津建立类似于中央特科的组织，第三就是清查叛徒，看看哪些人在这次的这个叛变中。

因为顾顺章叛变革命以后，因为他在党内是担任重要领导职务的，对整个党的情报和保卫系统十分了解，所以当时他就供出了很多人。当时中统方面就根据他的口供，在天津、北京、上海、武汉展开一系列的大搜捕，抓获了很多
