
# 四月份四个时间窗口

## 前言

时间是衡量万物从重生到毁灭的标准。3月份时间窗口只有1个，就是21号，故此没有专门发文。现在3月底了，4月份即将到来，不少朋友催更4月份时间窗口，昨晚耗时3个小时终于推算出来了。4月份时间窗口比以往都要多一些，以前不是2个就是3个，这个月居然有4个，这说明4月份行情估计会比较折腾。由于时间窗口比较多，估计4月份时间窗口有利于短线交易选手尽情发挥，无非就是时间窗口出低点就买入，出高点就卖出，乐此不疲，方能截取短线上涨果实。

## 4月份时间窗口

4月份时间窗口共有4个：分别是4月1号，4月8号、4月14号和4月25号。

### 4月1号时间窗口

4月1号时间窗口共有三组窗口共振，比如3509-3388点运动时长72个交易日；3227-3439点运动时长73个交易日，从3494点运动到4月1号也是73个交易日。其他两组时间：2689-3346(61T)、3152-3140(60T)、3418点运动到4月1号也是60T；3152-3418(50T)、3494-3388(50T)、3140点运动到4月1号也是50T。故此4月1号是第一个时间窗口。

### 4月8号时间窗口

4月8号时间窗口共有三组时间共振，比如3284-3388点运动时长65个交易日；3494-3439点运动时长65个交易日，从3418点运动到4月8号也是65个交易日；其他两组时间：3152-3227(28T)、3284-3418(28T)、3388点运动到4月8号也是28T；3388-3439(14T)、3297-3340(14T)、3419点运动到4月8号也是14T。故此4月8号是第二个时间窗口。其中，丘比特之箭交叉时间指向4月7号，与8号误差1个交易日也产生了共振。

### 4月14号时间窗口

4月14号时间窗口共有三组时间共振，比如2933-3152点运动时长75个交易日；3509-3297点运动时长75个交易日，从3346点运动到4月14号也是75个交易日。其他两组时间：3152-3140(60T)、3227-3388(59T)、3140点运动到4月14号也是59T；2689-3509(32T)、3227-3140(32T)、3388点运动到4月14号也是32T。因4月4号清明节，故此后移1个交易日，实际上4月份第三个时间窗口在4月15号。

### 4月25号时间窗口

4月25号时间窗口共有三组时间共振，比如3674-3140点运动时长68个交易日；3284-3297点运动时长68个交易日，从3140点运动到4月25号也是68个交易日，故此4月25号是第四个时间窗口。其他两组时间：3152-3494(37T)、3284-3140(38T)、3297点运动到4月14号也是38T；3284-3418(28T)、3140-3388(27T)、3439点运动到4月14号也是32T。因4月4号清明节，故此后移1个交易日，实际上4月份第四个时间窗口在4月26号。

## 对4月份时间窗口的系统性描述

因为3月31号是下周星期一，4月1号是星期二，和两个时间相邻，建议大家注意1号时间窗口会不会提前1天出现变盘形成短线重要转折点。8号时间窗口与1号是反向的，比如1号是低点，则8号是高点。至于4月15号，这个时间窗口有点难以提前判定；当8号时间窗口成为高点，4月15号要是指数仍然在8号高点附近则形成小双高，要是15号指数是基于8号高点产生的调整以跌的方式抵达的，则是小低点；4月26号看不清楚，太远了，只能临近时再给出明确的判定。

总之，4月份时间窗口比较密集，几乎是每周都有一个，这就说明了一个问题，4月份市场恐怕还会保持震荡格局为主，不具备爆发波段主升的月份，其交易策略更加偏向于短线低买高卖为主。有了这样的认知就可以对4月份行情交易进行预期管控，即涨跌都不会持续，涨了要舍得卖，跌了要敢于买，不要涨了就说后面会创新高进入主升，跌了又认为熊市来肯后面会创出新低。
