
# 阿里巴巴将token转化为抽成率：阿里巴巴飞轮

过去三个月，阿里巴巴股价跑赢以中概互联网指数基金衡量的行业平均水平364个百分点，这是因为2025年2季度云业务收入增速好于预期，以及管理层对外卖和闪购业务投资战略做出充满信心的表述。上周在杭州参加阿里巴巴云栖大会后，我们越发看好阿里云，这源于未来的外部客户创收机遇（即云业务收入）及其与国内电商的协同效应。我们认为，随着阿里巴巴的叙事从“国内电商市场份额流失方”转向“中国互联网一线资产”，云业务和中国国内电商两方面的积极发展意味着估值倍数应上升。我们将2027/2028财年云业务收入预测值上调2%/6%，原因是我们更看好生成式AI的应用和阿里云的变现。我们将2027/2028财年中国电商集团调整后EBITA上调2%/3%，原因是阿里云的AI服务和应用创造出价值。基于更高的财务预期和更高的估值倍数，我们将目标价从170美元/165港元上调至245美元/240港元。

云业务收入前景更为强劲。2025年2季度阿里云收入增长率连续第八个季度提高，同比增幅达26%，主要受生成式AI需求推动，这些需求只来自三个行业：互联网、自动驾驶和具身智能。这是因为生成式AI的应用基于高水平数字化和实时数据的生成和采集。不过，其他行业正在快速采用生成式AI。我们预计中国的生成式AI扩散速度将超过此前的SaaS浪潮，原因是可实现的效率提升范围更广（基于语言、视觉和智能体的工作流差不多涉及所有职能）而且部署摩擦也更小（API/智能体而非全流程重新设计）。我们预计未来12-36个月内营销、服务、编码、财务部门和供应链应用将从工具测试全面转向智能体自动化，服务成本将稳步下降，大多数消费者面向型漏斗的转化率/吞吐量也将提高。

预计生成式AI和电商将产生协同效应。理论和历史表明消费者通常会通过更低的价格、更丰富的品类和更好的匹配获得技术驱动的效率中最大的份额，即消费者剩余，而生产商/平台通过规模、抽成率/定价权和新市场来保持价值。就阿里巴巴的商业生态系统而言，我们预计：(i) 商家会看到运营支出得到节省和更高的转化率；(ii) 消费者将受益于更好的推送、内容和价格；(iii) 阿里巴巴获得为服务（广告、工具、智能体平台）重新定价的空间，这是因为其对业绩做出更大的贡献-特别是在基于价值的定价可得到保护的线索生成/流量质量方面。综上，我们预计消费者仍将是最大受益者，阿里巴巴则可借助更高的效率/广告投资回报率和智能体工作流订阅将部分增量剩余变现。

对投资的影响：我们认为国内生成式AI曲线的弧度/延伸速度将超过SaaS，而且阿里巴巴有望参与每一个阶段-计算、平台和应用-同时直接改善商家经济效益。随着生成式AI的发展推动云业务变现、商家节约成本和国内电商增量变现，我们认为阿里巴巴的股票叙事将回归“中国互联网一线资产”，这对应15-20倍的市盈率倍数。由于外卖和闪购投资扭曲了阿里巴巴的2027财年财务前景，我们鼓励投资者基于2028财年为阿里巴巴估值。按照我们的盈利预测，该股当前股价对应12倍2028财年预期市盈率，在我们看来这提供了显著的估值上调空间。我们按15倍2028财年预期市盈率为阿里巴巴估值，按当前股价对应36%的上行空间。

在2025云栖大会上，阿里巴巴展示了一整套强大的AI模型和应用，其中许多可直接用于其庞大的电商商家生态系统。通过利用这些工具，商家可以实现其业务各个方面的自动化和优化，从内容创作到客户服务，从而提高效率，改善用户体验并可能提高增长率。

在此我们将重点介绍几种为电商商家带来特定益处的AI工具：
- 造点：造点是AI图像和视频生成平台，与阿里巴巴的通义万相视觉生成模型相融合。它可以让商家直接用该平台生成优质营销内容，从而简化内容创作过程。
- 通义千问3图像编辑模型：该模型提供了更强的多图像编辑功能，具有更好的视觉一致性。这使商家能够高效并划算地制作专业产品照片。
- 通义千问3-Omni：新的全模态大模型，可处理多种语言的文本、图像、音频和视频，并以实时流媒体文本和自然语音进行响应。这可为用于免提多模态支持的高级客服智能体提供助力。
- 羚羊AgentOne：旨在服务于企业AI应用的平台，为营销、分析、客服和运营等领域提供基于场景的解决方案。商家可将该平台与数据分析、个性化营销和工作流优化相融合。
