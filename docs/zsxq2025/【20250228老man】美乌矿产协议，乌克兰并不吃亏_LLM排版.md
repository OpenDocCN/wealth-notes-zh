
# 美乌矿产协议，乌克兰并不吃亏

关于川普出卖了乌克兰的说法，最近这几天甚嚣尘上，甚至有些更扯淡的说辞，宣称川普就是俄罗斯的间谍，他有涉及情色的把柄，被俄罗斯情报机构抓住了，并策反了川普。这种种扯淡的谣言到四处传播，似乎川普一定会投靠俄罗斯，出卖乌克兰，成为史上最大的奸贼。

说回美乌矿产协议的事情，这件事现在已经算是尘埃落定，双方终于达成了协议，并且协议全文也已经发布了。具体来说，这是一份非常公平公正的协议，美国并没有侵犯乌克兰的利益，没有强行掠夺乌克兰的资源，并且通过这份协议，美国与乌克兰实现了利益上的强绑定，双方达成了真正的国家层面的战略绑定。协议签订之后，乌克兰的利益就是美国的利益，美国也有了充分的军事支持乌克兰的理由。

该协议的主要内容为：

第一，双方合作成立复兴投资基金。乌克兰以全国所有矿产及配套基础设施的评估价值的50%出资，美国以现金方式对等出资。双方在该基金的具体份额和权责留待下一步讨论。

第二，该基金负责投资于乌克兰境内所有的矿产，勘探、开采以及基础设施建设。这两条就是传统意义上的招商引资，国人对此也是非常熟悉了，就不多说了。下面两条就很有意思了，后面进行详细说明。

第三，也是最关键的条款。基金产生的收益，需全部用于在乌克兰境内的再投资。任何一方要转让其在基金中的部分或全部权益，均需得到另一方的书面同意。这也就意味着，美国并不打算掠夺乌克兰的资产，并不打算侵占乌克兰的利益。从乌克兰的矿产开发中挣到的钱，依然留在乌克兰。美国政府事实上只获得了一个数字上的收益，并不会拿走，并不会变成美国政府口袋里的钱。

第四，美国对乌克兰的稳定和经济繁荣的发展提供长期的资金承诺。这是一个非常有意思的承诺。乌克兰一直想要获得美国的军事保护承诺和安全承诺，不过最终川普没有提供这样的承诺，而是换成了资金承诺。然而事实上，这样的资金承诺，远比虚伪的安全承诺来得更加有效。

各位，请你们仔细想一想，美国加入这一份复兴投资基金挣到的钱都留在乌克兰并因此在乌克兰形成名义上的庞大资产，同时，美国还因为资金承诺的缘故会深度参与乌克兰的战后重建，并留下更多的资产。到那个时候，乌克兰一大半的基础设施名义上都由乌克兰与美国共同持有，这就意味着，俄罗斯继续侵犯乌克兰，就等于侵犯美国，美国可以名正言顺的出兵与俄罗斯进行战场上的直接对抗。甚至到双方的利益关系进一步深度绑定的话，乌克兰直接并入美国也不是不可以。相信饱经战乱的乌克兰人民，对于并入美国这件事也不会投反对票。

川普是个很有意思的人，他在乌克兰问题上的谈判方式是完完全全的商人作风，真实的利益绑定才是最好的安全承诺，比一切虚伪的政治承诺都来得有效。

而在乌克兰那边，他的利益并没有受到丝毫的损害，这是一个非常正常的招商引资动作，乌克兰政府根本就没有卖国。对泽连斯基来说，达成这份复兴投资基金协议，算是他向川普交出的第一份投名状，相信接下来他也一定会开始清算民主党在乌克兰的贪腐案件，这将是他交出的第二份投名状。有了这两份投名状，乌克兰从此就真正的安全了。
