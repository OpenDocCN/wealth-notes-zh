
# 委内瑞拉会不会成为第二个叙利亚？

2024年11月底，短短十来天时间，统治叙利亚20多年的巴沙尔·阿萨德政权轰然倒台，让全球大跌眼镜。1年后的今天，委内瑞拉马杜罗政权会不会重蹈覆辙?本期内容我们结合当下情况，聊聊这个话题。

估计有很多人认为，马杜罗会成为第二个叙利亚阿萨德，脑洞是不是开得有点大?毕竟马杜罗一直以来都被誉为拉美地区的反美斗士，手上还掌控着十几万正规军和几百万民兵组织，还有委内瑞拉的热带雨林、山地作为屏障，虽然手上那点可怜的装备在老美面前不够看，但凭借武装人员的规模和复杂地形，怎么也够老美喝一壶，不至于步阿萨德后尘，选择跑路吧。

理论上，持有上述观点的并没错，也是目前的主流观点，而且老美在委内瑞拉家门口舞刀弄剑已经3个多月了，迟迟还不敢下狠手也是事实。

但实际上呢?个人认为，这个观点的底层逻辑是有问题的。

为啥这么说?

我们先看老美方面：

首先，大家真认为川普会选择派兵实际占领委内瑞拉，进行一场近身肉搏战吗?如果川普就是这个目标，那没啥话说。

但如果不是呢?至少我认为，川普围困马杜罗，目标很简单：

其一，觊觎委内瑞拉的石油资源。

其二，重新启用“门罗主义”,掌控拉美。现在美国大的趋势是收缩，但收缩也有个底线，就是对自己的后花园，还是要有影响力和控制力。

至于占领委内瑞拉，想都不要想，因为后面还要治理，老美不是圣母，它不会趟这浑水。

也就是说，虽然占领委内瑞拉对我们而言，是最美的剧本。但答案很明显——不会。老美目前的国内经济环境和中美博弈的态势，它根本不敢贸然深陷委内瑞拉这个泥潭。不要认为五角大楼和白宫哪些顾问高参是吃干饭长大的，会让阿富汗的剧本重演。

所以，老美现在的策略很清晰：

派重兵封锁委内瑞拉近海，单方面宣布关闭委内瑞拉领空，摆出一副“关门打狗”的架势，让委内瑞拉自乱阵脚。然后，留2个后手时刻威胁马杜罗：

其一，考虑用导弹执行斩首任务。

其二，派遣小股力量，渗入委内瑞拉境内执行抓捕或击毙马杜罗的行动。目前有消息称，美国中央情报局和特种部队已经部署到位，随时准备启动。

所以，如果按老美的策略发展下去，实际上马杜罗手上的牌(军队和地形),并没有多大的用武之地。我能打你，但你打不到我。老美可以用非常小的代价，颠覆委内瑞拉政权。

这里很多人可能还有个疑问?

这2个后手，都必须有个前提，那就是需要取得情报和当地势力支持，否则怎么搞空袭斩首?怎么地面渗透呢?

这里就说说马杜罗方面：

虽然马杜罗对外，一直树立的是强人形象，但实际上外强中干，内部早就千疮百孔了。

有民调显示，约7成民众对马杜罗政府是不认同的。虽然我们可以不信民调数据，但委内瑞拉国内经济烂得不能再烂的现实，是摆在那里。

在这种情况下，国内民众支持率又会有多高呢?它就是一个独裁政府，不要把凝聚力拔得太高。

之所以马杜罗能够维持住目前国内政局，其核心抓手是将石油等经济命脉与军队进行捆绑，形成利益共同体，这其中拿大头利益的还是被高层拿走了。

也就是说，马杜罗政府的国内权力结构，所呈现的特点是“上层稳固、中底层松散”的局面。

缺乏民众支持，又有反对派从中搅局，老美通过围困制造压力，再大把大把洒美金策反中低阶层的部分军队，要想获得情报，搞小股部队渗透，那不是手拿把攥的事吗?

所以，我们回头再看老美的策略和马杜罗困境就清楚，所谓的手上抵抗老美的资本，对委内瑞拉的控制牢固，实际上相当脆弱，其个人也是慌得一批。

12月4号，《纽约时报》援引消息人士就报道，马杜罗现在每晚睡在不同的地方，并且不断更换手机。

这个消息是否准确属实，虽然要打一个问号，但是从客观环境和逻辑来审视，应该八九不离十。

至于外援方面，虽然会有，但我认为更多的也是象征意义。大鹅现在自顾不暇，距离我们又太远了，而且目前来说，中美也处于休战阶段，只要老美不是采用实际占领为目标，外援就很难对马杜罗给与足以影响战局的支持。

其它不说，叙利亚阿萨德与中俄的关系如何，想必应该比马杜罗好得多吧，后面怎么样?

实际上很多时候，如果自己都不能自救，那寄希望于外援，就属于妄想。也正因为如此，委内瑞拉目前的局势对马杜罗来说是相当危险，并不是主流观点那么的乐观。

想象着美军下场，马杜罗拼死抵抗，把战争拖它个三年五载，然后我们坐收渔翁之利。

可能吗?

实际上，在这方面，美委双方都清楚，并且目前马杜罗已经在着手准备后事了。

聊聊马杜罗和川普的那通电话。

大致时间是在11月21日左右，据说这通电話是由卡塔尔、土耳其等斡旋下启动的，川普在电话中表示，1周内只要马杜罗立即辞职，就能保证他、他的妻子和儿子，安全离开委内瑞拉，否则将面临军事打击。

而马杜罗方面给出的条件主要涉及3点：

1、要求对他和核心成员进行全球赦免，包括解除制裁、终止国际刑事法院指控。

2、保留军队控制权。

3、成立临时过度政府，18个月内进行自由选举。

虽然，马杜罗开出的条件看似拖延战术。为自己争取最大的利益，同时留个未来反扑的后手，但本质上就是立场出现松动。川普是一个精明的商人，深谙交易之道。马杜罗只要立场不坚定，内心出现波动，那就会往死里搓这个痛点，直到大赢特赢。

所以，马杜罗提出的这些条件被川普当场一口否决，然后宣布单方面关闭领空，部署力量深入委内瑞拉境内执行任务，甚至马杜罗政府试图再次致电华盛顿，都被拒绝。

熟悉的极限施压，是不是又来了。

马杜罗近日在谈及与川普通话时，还说道：这次对话在语气上是相互尊重的、这是一次诚挚的交流，双方正在朝着相互尊重的对话迈进，等等。

说实话，当我看了马杜罗的表态后，都忍不住想笑。人家老美挂完电话，准时准点关闭了你家领空，这算是贴脸输出吧。怎么还在言之凿凿谈什么尊重，诚挚?

马杜罗不是反美斗士吗?不是对美强硬派吗?表态的内容不应该是对抗性很强的话术吗，怎么更像打算服软，当添狗了呢。

以至于这通电话，现在看完全成了一道送分题，川普兵不血刃就把马杜罗苦苦经营的人设拉跨了，导致反美斗士的形象，摇摇欲坠。甚至会起到连锁反应。

试想，老板都打算收拾细软跑路了，无非就是条件还没谈拢而已。那让国内拥护者怎么想?让紧紧围绕在身边的利益共同体怎么想?

会不会之后，下面人估计跑得更快，甚至直接把他出卖了领赏去呢?

所以，我认为，马杜罗成为第二个叙利亚阿萨德，不是没可能，而是可能性正在增加。说穿了，马杜罗是独裁份子，不是极端份子，在面临大敌，国内又一片乱麻的时候，他不傻，首先看中的也仅仅是安全和钱，最终还是惜命的。

所以，未来局势的发展我判断大概率会围绕2条主线进行：

其一，马杜罗深挖洞，到处藏匿，躲避老美的导弹斩首和人员渗透，遥控指挥对抗老美，先拖一段时间，才能给自己卖个好价钱。

但这条路子收益虽然能放大，但风险也很高，怕就怕冷不丁的情报泄漏被精准锁定，或者几十枚导弹落地，国内瞬间做鸟兽散，成为叙利亚的翻版。

其二，继续找和川普沟通的机会，进一步妥协，寻找一个双方都能接受的结果，然后远遁它国。

以上仅是我结合目前掌握的信息做个判断，也不排除美军彻底下场，或者委内瑞拉同仇敌忾的可能，只是概率非常小而已。

当然，很多人会问，如果马杜罗真成了第二个叙利亚阿萨德，那对我们有什么影响?

我认为，要说影响，那肯定有点，但要说影响有多大，根本谈不上多大。

因为我们能源战略已经实现了多元化，在委内瑞拉的投资布局也少得可怜，未偿还的债务也不多。对于我们而言，委内瑞拉甚至连暗桩都谈不上，充其量起到一个在拉美牵制的作用。

未来不管谁上台，发展经济都是首要任务，否则也坐不稳，到时候就留给老美自己去收拾。阿根廷就是一个典型的例子，之前米莱对我们多刺头，现在还不是敞开了拥抱我们。

而且，美国的强硬干预可能加剧拉美国家对“门罗主义”回归的担忧，促使它们更加积极地寻求与我们合作，以实现战略平衡，这未尝不是机遇。

要知道，老美已经不是90年代中东海湾战争时如日中天的老美了，那时的老美只要龇个牙，中东一群王爷都要跪，你看现在呢?

所以，老美要在南美拔钉子，过程一定不会轻松。

中美博弈，东升西落是这个时代的主旋律，不会因为一个委内瑞拉马杜罗就影响历史的进程，我们需要做的，就是干好自己的事，静观事态发展即可。
