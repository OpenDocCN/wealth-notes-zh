
# 资本市场进入一个关键的节点

讲一讲资本市场，这是当下经济通缩的环境下唯一的亮点，但是这个亮点也到了一个关键的节点。

为什么这么说呢？前面我在知识星球的文章也做过分析，这一轮的牛市是很明显的结构性牛市，其特点就是，与经济基本面挂钩的比如地产、建筑、旅游、白酒等等基本不涨，甚至还有一定的下跌，涨的都是科技赛道，比如AI、芯片、机器人这些。

但是科技赛道这些明星股票目前已经出现难以为继的疲态。整个9月份，以易中天为代表的科技龙头股票基本出现是一个震荡的走势，股价没怎么跌，也没怎么涨，但是值得注意的是，其融资净买入额却大幅度增加。

也就是说，现在这些龙头明星股票是融资盘换手前期的获利盘，融资盘有一个基本特征就是促涨也促跌，也就是说融资盘其实是投机性质很高的资金，它们其实没有什么基本面信仰，一旦趋势逆转，杀跌将是非常惨烈的。

那么为什么科技赛道这些龙头股票基本涨不动了呢？两个原因。

其一，估值已经非常高了。

其二，三季度报马上就要出来了（其实大资金已经知道了相应企业三季度的业绩表现）。

牛市有泡沫大家都能接受，估值高一般情况下问题也不大，怕就怕估值被证伪，被业绩证伪，这才是最麻烦的，偏偏很多企业三季度业绩马上就要公布，一旦业绩被证伪，很多股票将面临很大的杀跌调整。

所以，这一轮牛市就进入到一个关键的节点：该涨的已经涨不动了，而且面临着三季度报的考验；基本没怎么涨的——也就是与经济基本面挂钩的会不会有一轮补涨呢？

我觉得大概率不会。原因很简单，因为经济基本面还没有改善，特别是房地产还没有企稳止跌。

所以，如果仅仅从纯粹市场观点来看的话，好像可以做出一个基本的判断：这轮牛市基本结束了，原因就是该涨的涨不动了，而且面临较大的调整压力，不该涨的因为经济基本面没有改善，所以大概率也不会补涨。

但是这个认识与判断是有偏颇的。

因为目前的资本市场决不能仅仅从市场观点来认识，因为还有一个非常关键的因素：国家队。

或者说国家的意志。国家队目前是资本市场最大的权重，其持仓占比之高前所未有。

给大家一组数据。跟踪上证50指数的最大的ETF里，国家队持有份额超过了82%；跟踪沪深300指数最大的ETF里，国家队持有份额超过了69%；跟踪创业板最大的ETF里，国家队持有份额超过了42%；跟踪中证500指数最大的ETF里，国家队持有份额超过了61%。

凭借这些筹码，国家队在短期内是能够主导市场涨跌的，具体到某一天甚至基本能控盘。

现在国家队操盘风格与过去也有了本质区别，过去国家队一般就是防风险，基本是在市场大幅度下跌的时候国家队才出来干活，但是现在不一样，即便指数涨到3800点，市场出现大幅度波动的时候，一样可以看到国家队活跃的身影。

那么国家队操盘风格出现如此大的变化呢？两个原因。

其一，落实前期高层会议对资本市场的有关精神。"持续稳定活跃资本市场"、"增强国内资本市场的吸引力和包容性，巩固资本市场的回稳向好势头"，按照高层会议这个表态，国家队的意志绝对不可能让这轮牛市就这么结束。

其二，宏观层面目前有学习日本经验的倾向。2010年日本央行在全面货币宽松的政策指引下开始买入股票ETF，2013年安培经济学推出后，央行买入股票ETF被纳入量化宽松的政策框架，日本央行买入股票ETF力度加大。

央行买入股票ETF被纳入量化宽松的政策框架背后的逻辑是这样的：在经济通缩低迷的情况下，央行直接货币宽松效果不大，因为即便将利率降为0，市场也缺乏贷款的兴趣，在这种情况下，央行买入股票ETF就是曲线救国给市场释放流动性，一方面给资本市场投资者创造财富效应，提振消费，另一方面可以提升企业估值，改善企业资产负债表，让企业获得更低更便宜的融资。

那么央行买入股票ETF最后达到什么效果呢？

短期内效果很明显：2013年QQE启动之后，当年日经指数暴涨57%，对当时安培经济学形成正向反馈，光是日本央行买入股票ETF当年就盈利16万亿日元，对市场确实造成一定的财富效应，提振了消费。

同时大型企业融资利率10年间下降了120个基点，降低企业成本。

副作用就是目前日本央行持有股票市值已经占资本市场的7%，还成为72家公司10%以上的股东，价格被扭曲，被动持有多个公司股权，却不行使投票与管理权，削弱了公司治理。最后就是退出路径缺失，找不到接盘侠。

简单的总结一下，央行大规模买入股票ETF短期内对于改善通缩释放流动性效果确实很明显，长期这么干副作用确实很大。

所以，到了中国，现在国家队虽然大举买入各种股票ETF，但是估计不会达到日本央行那个地步，国家队买入股票ETF最主要是控制市场走向。

所以，对于未来资本市场走势大概率会按照这样的路径演绎：三季度报出来之后，前期涨幅较大的明星科技股会面临较大的调整，叠加前期不怎么涨的股票也会同期调整，在这个时候国家队会出来干活了，通过大规模买入股票ETF，一方面稳定指数，一方面就是拼命给市场释放流动性，同时引导市场去挖掘一些科技赛道前期涨幅不大的赛道同时也属于新质生产力的范畴，接力市场的涨势。

所以，给大家建议就是，前期涨幅较大的AI、芯片、通讯基金这些，统统可以止盈出来观望，等三季度报出来之后，看看市场选择的新的方向与赛道。
