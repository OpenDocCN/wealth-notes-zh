
# 卢麒元《中庸》第23讲：第二十九章《三重章》

## 2025年3月15日

大家好，今天是2025年3月15日，乙巳年二月十六日，正式课。今天是《中庸》第二十三讲：第二十九章《三重章》。三重很重，今天聊得可能稍微穿越了一些，结论也有一点儿震撼。我试一下麦，三点钟我们准时开始。

《中庸》的最后这一段，是子思最重要的部分，也是学以致用最重要的部分。我想了想，把最后的这几个章节做得略微的细致一些，然后休息一段时间，我们再考虑开启《孟子》。因为我想了想，既然要把古典社会主义整理出来，不如我就干脆一边整理，一边就给大家交流一下，留给朋友们。

读古文，它有三样东西是最重要的。第一个是回到那段历史去，如果你不能设身处地，你很难理解。比如说，很难理解子思是怎样思考的？就是：为什么他要讲中庸之道？他的受众是谁？这三件事情你想清楚了，你才能理解子思他文中的本意，否则，就变成了后世人借子思、借《中庸》、借四书五经贩卖自己的东西。

我们回到今天，我先念一遍《三重章》，然后我再开启。“王天下有三重焉，其寡过矣乎！上焉者虽善无征，无征不信，不信民弗从；下焉者，虽善不尊，不信民弗从。故君子之道，本诸身，征诸庶民，考诸三王而不缪，建诸天地而不悖，质诸鬼神而无疑，百世以俟圣人而不惑。质诸鬼神而无疑，知天也；百世以俟圣人而不惑，知人也。是故君子动而世为天下道，行而世为天下法，言而世为天下则。远之则有望，近之则不厌。《诗》曰：‘在彼无恶，在此无射。庶几夙夜，以永终誉!’君子未有不如此而蚤有誉于天下者。”

此篇也是《中庸》之中讲方法论非常重要的。因为《中庸》大部分在讲道、在讲德、在讲理，而方法论的部分，这一章就非常重、非常重，还有，我们后边还有。他为什么要讲三重呢？三重是指议礼、制度和考文。他认为这个是你辅佐君王进行改革的最重要的三件事。

什么叫议礼？就是伦理；制度就是法理和治理；考文讲的是学理，实际上是舆论准备。这三件事情都是进行改革的必备的条件。而且，如果你脱离了伦理、法理、治理、学理来直接进行基于利益的改革，注定是要失败的。

子思所处的时代，“春秋”已经结束，“战国”已经徐徐拉开大幕。春秋时期周天子已经变得边缘化了。虽春秋五霸仍然尊崇、尊奉周天子，仍然主要的还是在行周礼，但是各个国家的改革已经开始了。因为你要争霸，你就必须对自己的内部进行改革，到了战国时期，将出现战国七雄，最后是秦整合天下。子思这个课堂是当时天下最重要的一个课堂，是中央党校高级领导干部培训班，他得教他们。所以，这里边讲的君子之道，不是你怎么做一个谦谦君子之道，我将其翻译成一句话：君子辅佐君王，王天下之道。

因为春秋五霸、战国七雄，必将有一替代周天子。虽孔子在世时强调“克己复礼”，他希望恢复周天子的威仪、恢复周礼、恢复大一统，但子思非常明白：回不去。天下是随着时间的推移，变成另外一个的天下。在某种意义上讲，中庸之道不是主观唯心主义，它是一种非常的实事求是的客观唯物主义。

《三重章》实际上是一个极高维度上来讨论王天下方法论的，这么一个非常重要的一堂课。本文开篇就是“王天下有三重焉，其寡过矣乎!”开篇就是王天下，王天下是动词，这个王天下是夺取天下治理天下的意思。他说有三个重中之重的事情要做好，你要如果能把这事办好，你就可以少犯错误。如果这三件事没办好，像川普这样的，那你可能结局就会非常麻烦了。所以他开篇王天下，王天下是讲的“伏羲氏王天下”的这个典。

“三重焉”指的是议礼、制度、考文。其实翻译成今天的白话文，就是伦理、法理、治理、学理，这四件事情构成了社会制度。也就是说，构成了一个国家或者是王天下那个天下的一整套秩序的基本规范。如果这些事情理不清，你王天下使用的是粗暴的武力，你根本就无法持续下去。

问题是：怎样做到“三重”呢？故君子之道，本诸身，征诸庶民，考诸三王而不缪。“本诸身”就是基于良知。“征诸庶民”，就是让你遵循民意。“征”就是从庶民那儿得到意见、征询，或者是证实。“考诸三王”，三王是夏禹、商汤和周文王。因为这三王有一个共同的特点，他们就是颠覆了前任，重新建立了天下。

子思要求你一定要依据这三个：故君子之道：本诸身，征诸庶民，考诸三王而不缪。做什么事儿都要出于良知，要关注民意，要符合传统。

“上焉者”指的是在世之周天子，“上焉者虽善无征”，现在的周天子人是好的，但是没有办法由他来证明周礼的合理性了，因为周礼已经破碎了。“无征不信”，就是一个已经在春秋五霸捣得稀碎的周礼，你如何让老百姓信呢？你不能证明了嘛，不能证明老百姓就不信，“不信民弗从”，老百姓怎么跟着周礼再走呢？怎么还支持你呢？

“下焉者”指的是诸侯，“下焉者虽善不尊”，不尊则不信，你地位不够，老百姓也不相信你，不信你民弗从。这里边提出了两个“民弗从”，这是一个双重悖论，从而得出一个君子之道的结论。

子思认为，完成议礼、制度和考文这样的伟大责任。虽然大家主张天子议礼、天子制度、天子考文，你懂得天子议礼、制度、考文，不是天子本人能做的，要有人辅佐，所以这堂课开始讲君子之道。你们那，如果有一天回到岗位上去，辅佐君王、王天下之道，那么你们怎么做？原则告诉你，“本诸身，征诸庶民，考诸三王而不缪”，这三条是最根本的重中之重，这三条没做到，改革必然走向反面，必然失败。

请问，川普“本诸身”了吗？他做事是出于自身的良知吗？不是，巨大的利益导向。他“征诸庶民”了吗？选票表达了他对红脖子的认同，但美国不仅仅是红脖子的美国，美国是世界的美国，“庶民”，既包含了红脖子，也包含了难民，包含了一切在美生活的居民，广义的讲也包括了全世界各国的人民。“征诸庶民”，就是你懂天下的民意吗？第三是“考诸三王而不缪”。特朗普是道统、体统、法统的破坏者，他重视传统吗？

他是否在传统上进行了考文呢？学理的叙述呢？理论的准备和舆论的准备呢？没有。是乱拳，是王八拳，一通乱打，这样的改革怎么能成呢？虽然他身边有一个近似吕不韦的商鞅——马斯克，但依旧是走不远，因为子思一再强调“三重焉”，又一再强调“君子之道，本诸身，征诸庶民，考诸三王而不缪”。请你观察所有的改革者，如果没有“三重焉”，不能“本诸身，征诸庶民，考诸三王而不缪”，必然是基于利益的改革，可以短暂见效，终将失败。
