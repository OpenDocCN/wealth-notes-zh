
# 金融三剑客：中国资本市场-中信（二十四）

## 一、纠偏

然而，在经过一段时间运作之后，发现基层问题一大堆，教员从大局考虑（坚持路线+维持团结），直接在1959年夏天的庐山会议上做了自我检讨。其实早在上庐山之前，从1958年10月到1959年4月，教员就主持召开了足足五次会议（一次郑州会议、武昌会议、中共八届六中全会、二次郑州会议、上海会议）强力纠左，遏制浮夸风，情况已经得到一定的扭转。

### 举几个例子：

1) 1958年11月10日，教员审阅修改了《十五年社会主义建设纲要四十条（一九五八~一九七二年）》初稿，将第四十条中"一切浮夸和虚构，都必须反对"这句话改为"浮而不实，爱好排场，谎报成绩，表里不一，这一切，都是要不得的。

2) 1958年11月21日，教员在政治局扩大会议上作了讲话，共讲了八个问题，第一条就是共产主义过渡问题，他说：“我们现在是一穷二白，还有一个一穷二弱。现在吹得太大了，我看是不合事实，没有反映客观实际。建设社会主义，我们没有经验，现在吹得那么厉害。我担心我们的建设。有一种树，叫钻天杨，钻得非常快，就是不结实，建设搞得太快了，可能天下大乱。这个问题，我总是担心得很。”

3) 在针对某些领导干部要在水利工程中搞"一亿九千万土石方"的问题，教员说：“一千九百亿土石方，总是多了，请你们议一议。你们如果一定要搞，那也没有办法，不能杀我的头就是了。我看，明年水利工程照五百亿土石方，一点也不翻。今年是五百亿，明年是五百亿，后年是五百亿，你搞他十年，不就五千亿了吗?我说留一点给我们的儿子去搞也可以，何必我们统统搞光?"

4) 教员还找来胡乔木、吴冷西、田家英这些笔杆子们谈话，说："虚报不好，比瞒产有危险性。报多了，拿不出来，如果根据多报的数字作生产计划，有危险性，作供应计划，更危险。记者到下面去，不能人家说什么,你就反映什么,要有冷静的头脑，要作比较。

5) 1959年4月15日，教员主持召开最高国务会议，在会议上说："去年的大跃进，伟大胜利也确是真的。报纸上吹的那些东西，不能全信，我是不信的。什么几万斤一亩，哪有那个事?那是并拢来的，禾秧要熟的时候，把许多亩并到一亩。这些浮夸现象要破除，要搞老实的。"

## 二、决定

不久之后，即1959年8月2日（农历1959年6月28日），召开了全国瞩目的庐山会议，会议全体通过了批判彭黄张周为“反党集团”的决定。庐山会议即八届八中全会，全会指责彭老总的那封著名的《意见书》信件，是“右倾机会主义向党进攻，妄图篡党夺权的纲领”，宣称彭老总、克诚同志、闻天同志、小舟同志结成了所谓“军事俱乐部”性质的集团。教员也在全会开幕时讲了话，8月11日在全会上再次讲话：庐山出现的这一场斗争，是一场阶级斗争，是过去十年社会主义革命过程中资产阶级与无产阶级两大对抗阶级生死斗争的继续。

五天后，全会通过了《关于彭德怀同志为首的反党集团的错误的决定》等决议，决定将彭黄张周等撤销国防、外交、省委等领导职务。随后，林副被任命为国防部长，主持军委日常工作。

## 三、派系

其实林副一开始是站队高岗的，但高岗犯了教员的忌讳，什么忌讳?高岗飘了，以为教员会换他当接班人，就酝酿扳倒刘副。高就各种喊话说什么教员是红区代表，刘是白区代表，借教员的大旗来反刘，还想拉拢林。去了杭州密会林说："现在党内有'二元论',即所谓教员代表红区，刘代表白区。教员就不能代表白区啦?白区就不是执行的教员的正确路线啦?这不是公然贬低教员、抬高刘吗? … 白区算什么?不就是在大城市搞搞工运，撒撒传单，贴贴标语，组织组织上街游行吗?靠这个能打败蒋介石?能取得革命的胜利?你我都是搞军队、搞根据地出身，你说根据地和白区，拿枪杆子打仗和搞地下工作，哪个分量重?哪个对革命的贡献大?还不是军队吗?还不是打仗吗?

林表示继续说："我看我们的党现在是一分为二，有红区的，有白区的，干部也是一分为二，有红区的干部，有白区的干部。教员是代表红区的，刘是代表白区的。现在的权力就掌握在白区干部的手里，再这样下去，我很担心啊!

然后，高还拿出了一份名单，是中组部副部长安子文拟定的政治局委员名单，这份名单本来有两份，一份有林的名字，另一份没有，高就特地拿了这份没有林的名单，来刺激一下他。果然，林看后非常生气，问这份名单经过政治局同意了吗?高说没有，这只是安子文私自拟定的名单，但安子文只是一个组织部副部长，他还没有这个胆子私自拟定政治局的名单，背后肯定有人指使。谁呢?高没有明说，但林猜是刘副。

到了这个时候，高已经确信自己对林的预期牵引达成了。林铁定站在自己这一边了，就安慰了他几句后，起身离开了。事后，林果然写了一封信，让叶群送给高的妻子李力群，并吩咐李力群一定要亲自交给高岗，对其他任何人都不要说。至于林当时给高写的什么，并没有其他人知道，当然，世上没有不透风的墙，高去杭州密会林的事，后面还是被上面知道了。不过，教员还是想保住林的，他也不想牵扯太多的人，就派陈云去杭州让林与高划清界限，不要惹火烧身。

林是个聪明人，就主动向组织交代了高来见自己的经过，取得了组织的谅解。当然，他也隐瞒了一些对他不利的事。直到"九一三事件"后，这件事又被提了起来，林也被重新定性为"高饶反党联盟的漏网分子",当然这是后话了。

除了林，高之前还拉拢过陈云, 陈不接受，还把这件事捅到了教员那里。这件事的败露再加上之后高的后台（斯大林）也去世了，教员就打算打压一下高岗，刘那派人逮着机会也狠批高岗。结果几方面因素一叠加（拉拢林、陈败露+后台去世+被刘派批斗+教员打压），高岗承受不住压力，就出事了（自杀了）。

其实高把问题想严重了，教员那边始终都是在平衡各方利益，或者叫保持党内团结，看他太飘了，起起落落一下，并没打算一棒子打死（都打死了谁来制衡后面的政工干部?）。后面温格时期教员还后悔说如果高在就好了，但他自杀了，就没法了，也就不得不重用了林副。

显然，那一轮斗争，以刘副那派胜利告终。不过，很快第二轮斗争又来了，就是这次彭的万言书事件。
