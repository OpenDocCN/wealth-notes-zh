
# 泡泡玛特品牌档案全收录

## 品牌文化

泡泡玛特秉持“创造潮流，传递美好”的品牌文化，致力于为消费者带来独特的潮流文化体验。其核心概念是“创造潮流，传递美好”，构建了一个涵盖IP孵化、产品设计、全渠道零售和沉浸式体验的潮流文化生态。

## 品牌发展历程

泡泡玛特成立于2010年，由创始人王宁在北京创立，最初以潮流百货杂货店起步。2020年12月，泡泡玛特在港交所上市，市值一度突破千亿港元，创始人王宁以208亿美元身家跻身福布斯中国富豪榜前十。总部位于北京望京，是中国领先的潮流文化娱乐公司，致力于通过IP孵化、产品设计、零售渠道及文化推广构建全产业链生态。

### 关键节点

- **2016年**：泡泡玛特签约香港设计师王信明，获得MOLLY独家版权并推出“Molly星座盲盒”，标志着品牌从代理模式转向自主IP开发。
- **2017年**：泡泡玛特推出自助机器人商店，以低成本、灵活性的方式快速铺开线下渠道，成为规模化扩张的核心引擎。
- **2020年**：泡泡玛特在港交所上市，市值一度突破千亿港元，确立了其在行业内的引领地位。
- **2023年**：北京泡泡玛特城市乐园开业，首推“潮玩IP+主题乐园”模式，推动全球化战略落地。
- **2024年**：泡泡玛特海外收入达50.7亿元，LABUBU等IP在海外市场实现破圈，例如泰国曼谷首店单日销售额破千万，北美市场LABUBU跻身销量前三。

## 品牌创始人

泡泡玛特创始人王宁（1987年生）受家庭商业氛围影响，大学期间创业积累经验。2010年，他带20万元在北京创立泡泡玛特。初期遇挫后，2015年敏锐捕捉潮玩机遇，转型后签下Molly独家IP，推出盲盒模式获成功。他坚持“极简”领导力，构建三层IP护城河，推动公司全产业链发展。2020年，泡泡玛特港交所上市，市值破千亿港元。截至2024年，他以48.73%持股成为河南新首富，引领中国潮玩文化走向全球。

## 品牌符号系统

### LOGO

## 结语

泡泡玛特通过其独特的品牌文化和创新模式，成功地将自己从一个潮流杂货店转变为全球领先的潮流文化娱乐公司。其IP孵化、产品设计、全渠道零售和沉浸式体验的潮流文化生态，不仅为消费者带来了独特的体验，也为中国潮玩文化走向世界树立了标杆。
