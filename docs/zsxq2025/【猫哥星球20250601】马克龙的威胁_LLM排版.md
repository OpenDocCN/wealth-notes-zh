
# 马克龙的威胁

现在我们想一想，假如北约军队进入泰国、越南、甚至是韩国、日本，东大会认为这是对自己的威胁吗？肯定不可能啊，毕竟北约军队进入亚洲某个国家这个当地国自己的主权问题，东大对亚洲任何一个国家也没有领土要求，唯一有一点边境领土纠纷就是印度，但是以印度傲娇的特性，印度能允许外国军队进入本国领土吗？用脚指头想也不可能的。

所以，盘点下来，东大绝对不能容忍外国军队特别是北约这样的组织的军队进入的地区只有一个台湾。所以，马克龙的威胁其潜台词就是这样一个意思——如果东大不阻止朝鲜军队到俄乌参战，北约军队将进驻台湾！

最近东大对俄乌谈判的表态是这样的：俄乌谈判应该达成一个公平持久、有约束力，并为个当事方所接受的和平协议。这个表态不偏不倚，非常中立，任何当事方都挑不出毛病。

但是我们如果仔细分析，现在俄乌谈判要达成一个和平协议，要做到“一个公平持久、有约束力”，并为个当事方所接受的和平协议可能性基本为0。为什么？因为至少俄乌双方诉求差距太大了！俄罗斯诉求是要承认俄罗斯吞并乌东四州的现实；乌克兰诉求是俄罗斯要从乌克兰领土上无条件撤军。

双方诉求差距这么大，怎么可能谈成一个并为个当事方所接受的和平协议？谈不成怎么办？那就只能继续打下去，直到双方都接受现实。继续打下去俄乌战场就是整个欧洲的溃疡，消耗的是欧洲、俄罗斯的实力，美国被咬被绑着消耗一点，只有东大是最大的赢家，这就是过去我在知识星球文章阐述的东大策略——水多了加面，面多了加水的逻辑。

很显然，欧洲对东大的立场很恼火，但是明面上又挑不出东大的毛病，毕竟东大的这个表态可谓没有偏袒任何一方。欧盟更希望东大的立场倾向于欧洲，至少也要施压俄罗斯做出让步。既然在劝和立场上挑不出毛病，就拿着朝鲜军队做文章——如果东大发挥自己影响力阻止朝鲜军队进入俄乌战场，那么就意味着东大迈出施压俄罗斯让步的第一步。

现在欧盟在俄乌问题上非常急迫，所以马克龙甚至拿着台湾问题来威胁东大。但走马克龙的这个威胁注定是没有效果的。原因无它，一场印巴空战让全世界都看清了一个现实，东大出口的二流空战体系可以轻易碾压全世界非体系化最先进的战机，那么意味着现在东大自己的一流空战体系只有老美能抗衡。

问题是在东大国境线1500公里范围内，现在连老美都无法抗衡东大，在印巴空战后老美在整个东亚都在做战略收缩，区区北约一点象征性军队就算就是进入台湾又能怎么样？反而如果北约军队进入台湾有很大可能激怒东大，如果东大因此反制北约，开始为俄罗斯提供实质性的军事援助，那么欧盟就是搬起石头砸自己的脚。

所以马克龙的威胁注定是一个嘴炮而已，既不能实质性真正威胁到东大，而且如果实施还有可能产生严重的政治后果。看来欧盟对于俄乌冲突现在确实是有点急跳脚了，什么招数都有枣无枣先打两杆子再说。
