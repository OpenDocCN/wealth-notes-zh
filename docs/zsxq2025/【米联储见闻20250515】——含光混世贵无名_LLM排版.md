
# 米联储见闻20250515——含光混世贵无名

川总实在是大爱中东狗大户了，而中东狗大户也真的是爱川总。作为一名真正的富二代，川总为了进入政坛，不得不将穷鬼的事情当成自己的工作内容。这对于川总来说真是太乏味了，估计只有访问中东，看到和自己一样喜欢金碧辉煌的狗大户们，川总才会真正找到回家的感觉。更何况大户们，消费起来公关起来那叫一个痛快，你办事我给钱，要多少开空白支票行不行？这才是川总想象中权力与金钱交相辉映的样子。什么送个飞机还要质疑为什么要收？你是不是有病呀，有人塞钱给我，难道我不要吗？中东狗大户们也是真爱川总，毕竟拿了钱真办事呀，不会呻呻歪歪扯什么意识形态，这头拿钱，那头伊朗就松口可以放弃核反应堆了。掏中东腰包的美国政客何其多，像川总这么办事爽快不含糊的美国总统有多。

与此同时，美国还与卡塔尔签署一份意向声明，包括超过380亿美元的潜在投资；美国将为在卡塔尔的乌代德空军基地使用设施承担费用，以及在未来协助卡塔尔提升防空和海上防御能力。到了这一步，川总必须为自己这次及时的中东之行抚掌称赞，毕竟过去这100多天到处橙壁于最启还是中东狗大户们让自己获得拿得出手的成绩单，这都是真金白银的进账呀。中东大户们正在将川总的外交影响力转化为超能力，变相支持川总当选，为川总助力，这才是榜一大哥该做的事情。所以这三个国家的领导好不容易懂到这么爱钱的美国总统，都希望与川总建立更紧密的关系。

沙特是核要求，阿联酋则想要寻求人工智能方面的资源。作为全球人均最富裕的国家之一，阿联酋承诺向美国投资数万亿美元。今年3月，阿联酋宣布一项为期10年，总额11.4万亿美元的投资计划，重点关注半导体制造业和能源领域。阿联酋驻华盛顿大使馆称，阿联酋目前在美国的投资总额已达1万亿美元。而卡塔尔一贯的作风是有枣没枣都打一杆子，对中回都特别大方，更别说时隔20多年才临幸它一次的全 cầu霸主，卡塔尔是与美国建立最正式安全关系的波斯湾国家，拥有美国在中东最大的军事基地。去年美国悄然与卡塔尔达成一项协议，将美国在卡塔尔乌代德空军基地的使用期限延长10年。卡塔尔也修订1992年与美国签署的国防合作协议，旨在进一步加强两国的安全伙伴关系。2022年，拜登政府还将卡塔尔指定为主要非北约盟友，这一称号通常授予与美国军方保持战略合作关系的亲密盟友。这次为了感谢川总临幸，还赠予川总一架波音747-8型飞机。

这老么子的清高，左手用各种消息收割资本市场，右手自己发行数字货币，还有人排队上门送礼，一律不拒，全部收下。这有多少赚多少的架势，让民主党人真是后悔自己白当家这几十年，被人们是为腐败的操作，却在川总那里成了为纳税人省钱。当他们可以免费（得到飞机）时，为什么我们的军队以及我们的纳税人应该被迫支付数亿美元去买……只有傻子才会不接受这份礼物。言下之意，我收到这个飞机是帮国家省钱呀，你们为什么还要响呻呻歪歪的。

当然这些逆天的操作之外，川总作为一家地产商的大佬，还有很多传统产业在权力的加持下原地起飞。川总家族在一众阿拉伯国家有一系列的地产项目，沙特吉达川普大楼红海海岸豪宅项目、卡塔尔多哈的川普国际哥尔夫球会以及同样在4月公布的阿联酋杜拜川普国际酒店大楼，所有项目都几乎是川普家族授权挂名而已，兴建营运全部是当地公司负责。送楼送飞机，中东狗大户搞这些实在是太轻车驾熟了。川总这次访问就特别访问沙特、卡塔尔、阿联酋三个国家，一视同仁的在这三个国家各留一晚，偏偏没有去以色列，将美国总统的外交之旅彻底打造成川总的商务出行。谁出的钱多，谁给的好处多，就去睡哪里，川总的身价尽显同时满载而归。毕竟谁能说签大把投资交易订单的总统不是好总统呢？懂粉们估计还要感动的流泪，毕竟快80的人了，坐这么久飞机去落后地区给美国人搞钱，含泪赚了，也是应该的。

脱下裤子做流氓，丝滑转变。所以也别怨狗大户们，当皮里阳秋，移情别恋，毕竟都是欢场，逢场作戏，然是谁能提供价值就投资谁。不过，做流氓这个事情也是需要实力的，如果大户们发现，在川总身上大把花钱，换不来自己想要的东西，下一回可就没那么容易捞钱了。
