
# 学习笔记，2025.2.20

1. 很多人想要的“权力”，不是去承担对应的责任，而是享受那种趋势他人的快感。拿不到实际权力，就去幻想更大的权力，去指点江山，实际上，连自己的人生掌控权都拿不到。
2. 阻拦一个人的不是无知，而是误以为自己知道便从此止步。无知，还会让人有求知的念头，觉得自己什么都知道，此后很难再有进步。对世界认识越多的人，越会意识到自己认知的局限性，越会变得谦卑。
3. 这个大环境下，做任何项目，都需要谨慎再谨慎，不是我阻挡你们成为亿万富翁，而是试错成本太高，倘若你有 1000 万，扔个二三十万试错，那无所谓。倘若只有二三十万，ALL IN 呢？那不合适。关键是，ALLIN 的时候，人们很容易不舍得认输，原本三个月不盈利就砍掉了，结果硬是支撑了三年，什么都赔上了。
4. 每个人都是无价的，但只限于对自己，对别人，有很多可以衡量价值的标准，你美若天仙、才富五车、富可敌国，但不能给对方实际帮助，那就是对别人没用。
5. 人本性里就是会有很多弱点，自私、好色、贪婪。但是自私不代表一个人就是恶人，他依旧可以很善良，如果用道德完美来要求人，世界上找不到几个完美的人。遇到那些喜欢用高尚道德要求他人的人，要看他自己有没有首先做到。
6. 人的很多改变看上去好像就是一夜之间的事，昨天还在孩子气，今天就成了一个成熟的大人。这些改变，并不是突然发生，大多是厚积薄发，是积累到了某个程度的自然呈现。
7. 当一个人负债累累的话，以后，我大概率不会见她了，因为她太穷了，太负能量了，一个朋友不能给我们加分，一定会给我们减分。现在，先屏蔽她，不让她看朋友圈。过段时间，我就会把她删除。删除她有个重要的原因，我觉得她不是一个合格的朋友，她跟我讲了很多学生家长或老师的隐私，她在X中代过课，X中里我们有不少共同熟悉的校友，她讲了一箩筐他们的八卦，关键是我作为更亲密的接触者，这里面很多八卦是胡扯……这是当老师的大忌！有句话，应该打印了贴她脑门上。当面说真话，背后说好话。
8. 在这片土地上，所有优秀的女人，没有一个没有绯闻的，而且版本不一。
9. 其实很多人都对健身教练这个行当有误解，以为练的好就有人买课，健身教练这个职业，决定能否赚到钱，第一是人好不好，这个包括成长环境，受教育的程度以及基本礼貌。第二是沟通能力如何，能否把自己的一些想法或见解深入浅出的表达出去。第三有没有专业知识以及持续学习的能力。
10. 做形体管理这个领域，有两点需要注意。第一点，若是真的想在这个领域有所建树，要不断的接触有钱人，接受她们的熏陶，看看她们为人处世的方式是什么？一个圈层有一个圈层的基本氛围，你不能跟这个氛围格格不入。第二点，深耕健身。一切形体管理的最终归宿，就是健身。
