
# 社会大局怎么看？政治看分配和社会结构，经济看货币和产业结构

社会大局，最重要的有两个，一是政治，二是经济。

## 一 政治的局

政治的核心在于利益分配。利益各方很复杂，但总得来说，包括几方面，即中央与地方、地方与地方；政府、企业与居民；社会上层、中层与下层群体；产业与产业之间。当然要具体再分，还有很多更细致的分法，我们大概了解就好。会不会看这个局，决定我们能否在分配中占到更优的位置。比如城市化运动，其实就是将更多资源分配到城市的局，那么必然的，越早成为城市人口，越早在城市落户买房，就越能在这个局中占到更多的资源分配。

不同城市的资源分配差异也很大，这是地方与地方的资源分配局。二十年前、十年前，在北上广深落户，肯定比在二三线城市落户，能占到更多的资源分配。同样的，每年大把的削尖了脑袋往体制内钻的人，是认准了体制内的资源分配一定比体制外更多。理解了这点，就可以更好地理解阶层、圈层之间的流动。

任何一个政策，分析其中的利益相关方，也就是局中的不同玩家，就能明白这个政策所谓何来。比如养老政策、教育医疗改革，其中的利益方都是谁？利益分配变动是什么样的？看清了谁分得更多，谁分得更少，也就明白了其中的导向。

社会是金字塔结构，搞清楚有哪些阶层、不同阶层都是什么样的玩法、从一个阶层跨越到另一个阶层需要什么条件，也就知道这辈子最可能混到哪个层次，人生就都通透了。

## 二 经济 的局

经济与政治密不可分，经济是政治利益分配的延伸，也是政治利益分配的具体体现。要看懂经济的局，关键要搞清楚以下几个东西。

### 货币

货币就是钱，但不仅仅是钱。对于普通人来说，货币就是每天辛辛苦苦赚的东西，是钞票、银行卡余额。但这只是最浅显的一层。如果只认识到这里，就一定是被收割的命。货币的价值是变化的，二十年前的100块钱，和二十年后的100块钱，价值可能相差数十倍。所以货币本质上也是一种商品，也有价格，只不过它的价格是用其他普通商品来衡量的，而衡量它的价格，主要是看它的信用。

你有了多少钱，也就有了多少钱的信用，这个信用的基础，有时候是粮食，有时候是黄金，有时候是美元，有时候是土地。了解这一点，就能通过看清货币信用的基础逻辑，而预测未来重要资产价格走向。信用跟什么有关，决定了什么东西最赚钱。

### 产业结构

资源是分层的，核心资源都是国字头，核心产业完全被控制住，剩下的不那么重要的资源，再丢给市场来分配。在产业结构中，离核心产业越近的位置，自然分配到的财富越多。油气、矿产、交通、水电、金融等等，你能想到的所有重要资源，都被牢牢掌控，普通人少有参与的机会。

从国有到市场化，从地产到衣食行消各行各业，再到互联网、高科技，哪些控制多，哪些控制少，以什么样的方式控制，都直接决定了财富的分配规则。弄清楚这些分配规则，就能选好行业、项目，找对位置。

### 资产

资产有两个作用，一个作用是作为货币的蓄水池，另一个作用是消灭流动性过剩。

楼市的局。众所众知，楼市是最大的蓄水池，如果把楼市里的水全放出来了，整个经济大池子的水平面恐怕都要涨三层楼高。但是这楼市的蓄水池，更不能往里放水，再放水池子装不下了，还得往外崩。所以呢，楼市的水，既不能涨，也不能跌，更不能跑，安安静静当个蓄水池就行了。

再比如股市的局。这个局里的玩家，包括政府（证监、银监会）、企业、机构、散户。股市在创立之初，是为了给国企融资，找接盘侠的，后来随着民营经济发展，民企也有了入场资格。过去20年，随着经济高速增长，货币大量超发，这些超发的钱如果流通在商品市场，那老百姓家里大米蔬菜得飞上天去。怎么办？通过股市来消灭掉这些钱。

直白点说，就是在通膨较高的时候，把这些热钱都吸引到股市上来，造成股市的价格高涨，紧接着收缩货币，于是股市一泻千里。这些过剩的流动性，在一涨一跌中，消失于无形。在这个过程中，各个玩家都扮演什么角色？这取决于他们的利益。

### 消费

消费的局，和普通人的关系最大。

消费的核心是要人们肯花钱，人们花钱多了，企业生产就提高了，就业就多了，就也多了员工的收入高了，能用来消费的钱就多了。这是一个良性循环，但前提是人们肯花钱才行。为了让人们花更多的钱，怎么办？消费升级，也就是让你买更贵更好的东西。

所以你如果是做生意的老板，就要看消费政策往哪走，怎么升级？升级后市场在哪里？哪些人会受益？这都是可以把握的方向。

社会大局是波涛，小局是浪花。能识局的人，就能避免很大大坑，能破局的人，就有了打破阶层壁垒的潜力。
