
# 警惕2026的美润四方

源自0224武汉读者线下沟通  
发布于2025年2月27日

说的好听一点，特朗普现在就是要搞美国版的“共同富裕”，要通过从富人的兜里搞钱，来收买底层的选票。譬如特朗普之所以重用马斯克，就是因为他擅长搞钱，马斯克只要能搞到钱，支撑特朗普搞美帝换房本，就能在特朗普新政权里面如日中天。而特朗普的敌对力量也很清楚的知道，想要阻止特朗普换房本的唯一方式，就是在换届年，断绝特朗普的粮草。

所以2022年兔子换届年遭遇的各类事情，在2026年的美国换届年也会大规模的重演，别看马部长如日中天，他2025年也同样有马省长2021年的风险，虽然特朗普会募集到大量的资本在2026年发钱，但他的反对派一定会竭尽所能，发动各种力量，试图在2026年也搞出一波美润四方。
