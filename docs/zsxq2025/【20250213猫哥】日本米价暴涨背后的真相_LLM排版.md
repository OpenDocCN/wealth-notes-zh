
# 日本米价暴涨背后的真相

## 嘉宾供稿
刚刚过去的1月，日本大米类价格同比上涨70.7%，再次刷新去年12月同比上涨的64.5%的历史最大涨幅。

现在日本国内超市终端零售米价已经从之前的1斤普通大米价格折合人民币9元左右上涨到了22元左右，而且不好买，很多零售网点处于上架即抢空的局面。

到底是什么原因导致这类大通货价格出现如此惊人的快速上涨局面呢？让我们先看看日本官方给出的理由。

## 日本官方给出的理由
原因主要就两点，其一是23年夏天的热浪导致大米供应量减少，其二是外国游客增加导致大米需求增加，所带来了严重的供需矛盾。

## 两个主要原因的分析
### 23年酷暑导致的粮食减产问题
虽然网上有很多关于天灾导致日本局部地区颗粒无收的报道，但是根据日本官方给出的数据，2022年日本全国食用大米产量为673万吨，而遭遇热浪袭击的23年呢？总产量为661万吨。确实减产了，但也只相较于22年减产了区区12万吨，减产幅度不到2%。

24年的整体数据还没出来，但是根据日本农林水产省在去年11月召开的粮食部门会议上发表了大米供需预测，24年产量将比上一年增加22万吨，达到683万吨。

注意，这是2024年年末的预测而不是年初。也就是说，最后实际数据应该是八九不离十的。那么既然24年的大米产量还要高于22年，供应端又何来问题呢？

### 游客增加导致需求增加的问题
整个2024年，由于日元贬值，外国游客确实疯狂涌入日本，全年游客数量大致在3700万人左右，刷新了历史记录。

但是，这影响有多大呢？我们做个简单的测算。用极限算法，假如每位游客每天吃1斤大米，游客连吃7天，3700万游客合计消费大米也就是13万吨。日本每年大米消费总体在700万吨左右，游客占总大米消耗量也不到2%。

而且，这是“游客经济”总量极限测算，有几个游客1天能吃1斤大米？往年日本就没有游客了吗？如果只计算游客增量，比如24年比23年增加1200万游客，那影响更是微乎其微。

上面说了，24年大米的产量较23年是增加的22万吨。别说3700万游客，就算6000万游客，充其量也就把24年增收的22万吨大米给消化掉。

因此，日本官方给出的米价大幅上涨甚至闹“米荒”的2个主要原因是根本站不住脚的。就算是客观存在的事实，这么微小的变量，难道就会造成大米这类大通货同比上涨70.7%的局面吗？

更加诡异的是，23年遭遇酷暑导致减产后，2024年日本大米1-7月出口量同比还增长了23%，将近2.5万吨。

当然，也有分析人士把原因推给日本政府一直回避的日本农协长期引导日本大米减产政策有关(关于日本农协的情况，大家可详见去年9月7日公众号文章“日本米荒的背后”，里面有详细的描述，这里就不多提)。但这个因素也站不住脚，为啥？因为日本农协的政策干预是一个长期渐进式动作，它并不是短期诱发主因。这里从22至24年3年的大米产量就能看出。

那么，导致当前日本大米价格持续飙涨背后深层次原因是什么呢？个人判断这与日本农林中央金库去年6月美债暴雷事件息息相关，是一起金融暴雷事件引起的连锁反应。

## 日本农林中央金库美债暴雷事件
日本农林中央金库又称“日本农协银行”，不仅是日本第五大银行，同时也是日本3300个农业、林业和渔业合作社的中央金融机构，而且地位超然，归口管理不是日本金融厅而是归农林水产省。

所以，这个金融机构本质上就是日本农协的“钱袋子”，日本农民的家当几乎全在里面。那么它为啥暴雷了呢？

因为日本一直实行的是低利率政策，不仅国内贷款业务收益少得可怜，而且作为帮扶农业的政策性银行，很多时候对于体系内的贷款甚至是亏起做。所以，久而久之这个农林中央金库就走上了对冲基金的路线，吃下了很多美债、欧债。

多到什么程度呢？我们都知道日本现在是美债第一持有国，日本金融机构持有的海外债券规模约120万亿日元，这其中，农林中央金库就占20%份额。如果在那个岁月静好的时代，这些债券还属于安全资产。

但错就错在，农林中央金库误判了形势。

第一，农林中央金库没想到美联储加息来的如此猛烈。因为美债价格和到期收益是负相关，美债收益率低的时候，美债价格就高，但拜登上任后，为打压通胀一口气把利率干到5.25%，这就导致美债收益率大幅上涨而美债价格大跌。农林中央金库因为前期买入了很多美债，这就导致账面上出现巨大浮亏。

第二，农林中央金库没想到美联储降息会来的如此之缓慢。从2022年加息开始，到2023年底，市场虽然多次传出停止加息或者降息的声音，但只听雷声响不见雨水落，甚至连硅谷银行破产都没见利率政策松动。这下就尴尬了，一边需要买入更低的美债来摊薄成本，一边新买入的美债继续被套，亏损越来越大。

我们目前无法得知农林中央金库对冲基金在投资美债的时候加了多少杠杆，但从后续的事态发展看应该不低。

因此对于账面的浮亏，就2条路可以选择。一条是割肉止损，另一条就是调动资金死扛到底。割肉止损就把浮亏做成实亏，而死扛到底理论上就不存在亏损一说。但死扛到底钱从哪里来？在美联储暴力加息的阶段，谁家还有余粮？

当然，美联储倒是可以伸把手借钱给农林中央金库(实际上也是有这个打算，比如美联储纽约银行就把它纳入常备回购便利池子中)，但问题是5点几的利率这类政策性银行能抗得住吗？这不等于持续放血吗？

因此在去年6月，农林中央金库就突然宣布将在截至2025年3月的一年内出售规模超过10万亿日元(约合人民币4600亿元)的美国国债和欧洲债券，以弥补巨额损失。这就相当于割肉一半仓位，血亏700亿人民币，把浮亏变成实亏。

考虑到日本农业占日本GDP 大致在1%左右，也就是330亿美元，而农业的低利润和日元贬值带来得进口成本增加，这700亿人民币也许把日本农民1-2年收成全部亏没了。这可是日本农民的血汗钱，什么储蓄、医保、社保等都关联其中。

## 日本政府的应对措施
怎么办？最好的办法也许就是通过农产品涨价来填这个巨坑，让全体小本子买单。

作为日本政府，也只能捏着鼻子配合造势，推高米价。没办法，农协虽然对日本经济创收贡献小，但它不仅掌握着日本粮食安全的命脉，同时手里还握有大量农民选票，这尊大神谁也得罪不起。

因此我们才看到，从24年上半年日本米价温和上涨到下半年加速上涨的行情，而在这个过程中，日本政府除了吹风维稳并没有实际拿出干预动作的干瞪眼现象。任由米价飞涨，一直拖到今年1月才宣布首次动用储备米，并计划向低收入家庭发放“大米券”来缓解米价上涨趋势。

以上就是日本米价暴涨背后的真相。

虽然日本政府已经宣布干预，但个人认为日本政府在其中能起到的作用相当有限，日本政府和日本农协之间还将展开一场持续时间很长的博弈，米价这类大通货价格飙涨所带来的连锁反应，比如其它农产品的涨价、推高其它成本的趋势将进一步吞噬日本的竞争力，让我们拭耳以待。
