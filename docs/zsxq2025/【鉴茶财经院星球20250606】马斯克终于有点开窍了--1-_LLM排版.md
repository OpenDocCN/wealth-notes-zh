
# 马斯克终于有点开窍了

今天谈点底层规则类的东西。

马斯克和懂王结合，为什么我们一开始就不看好？

因为马斯克没有催收能力，他无法保障自己的利益，以为天天讨好，待在一起，努力做事，就能保持影响力，有点幼稚了。它只有付出的权力，是否兑付，兑付多少，完全掌握在懂王手里。不平衡的关系迟早翻车。

你那点黑料，根本形不成催收。

啥叫催收能力？刘邦你是皇帝，大舅哥掌握着长安北军，樊哈也听吕后的，她说话就算。罗马首富克拉苏能催收凯撒，人家自己有八个军团。

现实里最简单的，村里的宅基地，村长若你分配不公平，七八个亲兄弟随时找上门。

姚广孝知道自己催收不了朱棣，帮夺天下后继续当他的和尚。

范蠡知道自己催收不了越王勾践，灭吴后直接润了。
