
# 武汉大学事件背后，让人不寒而栗

近期，武汉大学的事情，牵出了福特基金会。福特基金会与CIA有着密切联系，这不是秘密。有人在“福特基金会”的官网发现，2009年到2012年，“福特基金会”曾资助武汉大学开设一项名为“性别与法律”的课程，每年3.67万美元。该课程的教科书也由“福特基金会”提供。好家伙，还真有境外势力。

不过，这只是事情的冰山一角，因为“福特基金会”不可能仅资助一个项目。很快，事情有了更令人触目惊心的发现。“福特基金会”的资助项目许多是公开的，有网民进行了整理，发现接受资助的国内高校远不止武大，下面随便举几个例子。

北京大学，880万美元。人民大学，450万美元。清华大学，389万美元。还有中传、浙大、农大、浸会、社科院、川大、政法大学、成都大学、天津大学等等，受资助的金额各不相同，有的多有的少。资助的时间大部分在最近十年，平均每年几十万美元。资助的项目，只有一小部分是正常科研，大部分项目，属于“性别研究、LGBT推广、少数民族文化”等看着就居心叵测的议题。

梳理账目，我们会发现几个有意思的点。首先，受资助对象以名校居多，越是清北这种，资助金额越高，项目越多。原因也很简单，名校学子在毕业后，往往会进入精英圈子。资助这些人，能起到事半功倍的效果。其次，近年来，“福特基金会”在尝试下沉。受资助对象里，出现了吉首大学、贵州大学、广西民族大学、陕西师范大学等非名牌高校。翻开“福特基金会的资助记录”，2010年前，基本只资助名校，但2010年后，非名校开始受到青睐。甚至有一些大专，也拿到了资助，比如苏州工业园区职业技术学校。这所学校，名气不大，却十分重要。它是1997年由新加坡总理吴作栋提议成立的，中新两国共建，作为中国大学教育改革的样板之一，以新加坡的南洋理工为模板，先后有多位国家领导人前来视察。另外，陕西师范大学在2014年—2016年接受了三个项目的资助，全是性别研究的议题，合计150万美元，平均每个项目50万美元。其他各个大学，项目资助金额少有能上30万美元的，而陕西师范大学的单个项目金额，明显超过其他大学，是不是有文章尚不清楚。但可以肯定，“福特基金会”很重视陕西师范大学。

受资助者里最令人感到不寒而栗的是中央党校，你绝对没看错，中央党校，2008年曾接受“福特基金会”43.2万美元的资助。一共有五个项目，其中一个便是性别议题。中央党校的重要性无需多言，竟然能让人家把手伸进来，何等荒唐。难怪2018年贸易战时，某位中央党校教授跳出来，要求我们赶紧投降。注意，以上还只是公布的账目，没有公布的更多。即便2012年后，中央党校不再直接获得资助，但你敢保证没有其他方式？恐怕没人有信心。

当我们的高校乃至党校，都接受美国人资助，课题教科书由美国人提供时，怎能不出事？在我看来，高校大规模接受美国资助，有两个主要原因。

第一个，是穷。教育是吞金兽，大学教育尤其如此，而中国的教育投入长期不足，哪怕到今天，教育投入已达GDP的4%，许多方面也还是做得不够。比如，大学的硬件设施，不少大学宿舍至今没有空调，北方的大学，空调覆盖比例甚至只有30%，这个数字已经是近年来努力提升的结果。这一切都表明，大学很缺钱，南方和长三角、京津冀相对还好点，当地政府有钱，北方其他地区就要差不少，青岛大学的空调话题，今年夏天就成为了热点。想当年，我们大学是毕业那年才装的空调。前面三年的夏天只能靠风扇，相当难受。就算清北等经费充足的院校，银子总觉得不够用。这种情况下，如果有愿意资助，学校往往来者不拒。哪怕明知道对面不怀好意，但看在银子的份上，也会睁一只眼闭一只眼，这就给了对面乘虚而入的空间。

第二个原因，是客观要求。改革开放能成功，一个关键，是我们融入了西方主导的经济体系，占据产业分工地位。问题在于，不是你想融入，人家就会允许你融入的。像俄罗斯，它倒是想融入，但西方允许么？而融入的前提就是接受西方的渗透，这句话很残酷，却也是事实。举个例子，中国想吸引外资，但外资凭什么相信你？仅仅是一些招商优惠，以及基础设施，就行了？当然不够，还得有一条，得有数量足够多的、认同西方价值观的人，也就是香蕉人。西方企业在华招聘，往往是这些人优先。由他们充当代理人，西方企业才放心。想让人家来投资，前提就是咱们得培养出一批香蕉人。不然，人家就不来投资了。所以，你得教育体系是不是得西方化？是不是得接受西方的各种资助？这么做的好处，是经济发展起来了。坏处嘛，就是被渗透，整个社会，充斥着西方的代理人。为了发展经济，过去数十年，我们对意识形态工作的重视度不够。我们不够重视，敌人可是十分重视的，这就造成了如今的现状。虽然经济取得了巨大成功，但副作用也实在吓人。

未来必须纠正一些做法，不健康的发展坚决不要。有人说，你这样会抑制经济发展。我想说，有命挣钱，你也得有命花。对个人而言，天天996、007，虽然挣钱了，但最后全交给了医院，有什么意义？国家同样如此，一切向“钱”看，拼了命发展，虽然能取得了一些成果，但如果社会因此炸了，取得的成果再多，也得失去，意义何在？只不过，想改变不是那么容易的，对方布局了数十年，暗桩多得去了，以后类似武大的事情，不会少，大家应该有心理准备。冰冻三尺，非一日之寒。
