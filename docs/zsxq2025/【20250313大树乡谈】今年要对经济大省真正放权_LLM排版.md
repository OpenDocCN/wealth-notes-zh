
# 今年要对经济大省真正放权

每年两会非常关键的就是领导人下团组会议的表态，自2023年起，一号已经连续三年参加江苏代表团审议，每年主题不同。

2023年主题是高质量发展，提醒新班子不要大干快上。当时刚刚放开，各方都认为放开后经济会迅速好转，当时最担心的就是经济过火，所以专门提醒要稳住步伐。

2024年主题是新质生产力，整个2024年国家对科技创新、先进制造的投入极为坚定，直到去年七月大会新的理念被制度化，到9月才开始回应市场诉求，推出了一系列宽松政策。当然这次也提到“经济大省”，但并非核心重点。

而今年的主题是“经济大省挑大梁”，这背后涉及到央地关系调整和对地方扩大授权的关键难题。

自2012年以来，虽然为了搞活市场，上任李总一度推动以简政放权为核心的改革，将部分中央权限取消或下放到地方，但总的基调是权力向中央集中。包括财税方面的营改增、国地税合并，人事上的大规模跨省、央地调动，纪律上推进监察体制改革等等，地方上的自主权、财权、人事权持续缩小，也令地方发展经济的积极性有所降低。

2020年是一个转折点，高层意识到随着中国经济社会发展，不同地区的差异正在拉大，不可能都按照中央统一规划去执行，不同地区必须有不同的发展规划和政策。也就是在2020年8月，主张“大国大城”的陆铭作为最年轻的发言者参加了经济社会领域专家座谈会，他提出“在城乡和区域发展方面，如果一些体制性结构性问题得以解决，将能产生巨大的‘制度红利’”。他说的“制度红利”主要是强调强化都市圈的增长极作用。

2021年，在推出保障性租赁住房政策时，一改过去全国相对平均分配的做法，开始强调“人口净流入”，将保障性租赁住房建设集中到人口净流入地区，这是一次标志性的决策转变。之后越来越多的政策和资源分配开始强调地区差异。

典型如2022年5月以县城为重要载体的城镇化，就把全国各县分为五类：大城市周边、专业功能、农产品主产区、重点生态功能区、人口流失等，每类有不同的发展定位，其中人口流失县城要求严控城镇建设增量，引导人口向临近经济发展优势区域转移。

2022年3月工作报告正式提出了“经济大省”概念，原话为“经济大省要充分发挥优势，增强对全国发展的带动作用”，但当时只是理念上先提出来了，实际措施很少，就类似今年首次提出“投资于人”，要想落地还要等到2024年下半年。

随后因为二季度上海经济暂停，稳经济的压力迅速增大，于是经济大省就变得更重要了，当年8月前任李总首次在深圳召开经济大省主要负责人座谈会，广东、江苏、浙江、山东、河南、四川六省参会。

会后前任李总还到深圳考察，在考察时表示“既然要求广东担起重任，国家政策也要给力。政策性开发性金融工具对广东成熟项目要予以倾斜性支持，抓紧形成实物工作量”。

这个表态就是之后对“经济大省挑大梁”的政策基调。

2023年换届后，新班子仍然沿用“经济大省”的路线，把稳经济的重点放在经济大省，同时进一步加大了地区间政策差异。比如2024年2月，对债务高风险的12个省区，既允许发行更多特殊再融债置换隐性债务，也要求严控新增政府投资项目。

这也就意味着近一半省区的经济增长很难指望了，稳定5%增速目标的重任，自然就落在了经济大省；当然如重庆、辽宁等省区也在努力摆脱债务高风险的标签，希望重新放松政府投资项目的管控，内蒙古这两年产业转型做得很好，都完成了预定增速目标。

到2023年底的经济工作会议，首次把“经济大省要真正挑起大梁，为稳定全国经济做出更大贡献”列入次年重点工作。

但注意直到2024年9月之前，“经济大省挑大梁”基本停留在理念和口号上，虽然2024年2月说放松高质量项目的土地指标，向经济优势地区倾斜，3月说“更好发挥经济大省对区域乃至全国发展的辐射带动力”，6月说把新增地方专项债额度重点向经济大省特别是制造业集中地区倾斜，7月继续强调发挥好经济大省挑大梁作用。

但迟迟看不到真正有力度的支持性措施。直到2024年9月24日之后，才终于进入实质支持阶段。不再是过去单单强调经济大省的责任，也开始给经济大省承担责任对应的权利和资源。比如2024年10月常务副总在河南和江苏调研，要求“有关部门强化要素保障，支持经济大省挑大梁”；12月则授权北京、上海、江苏、浙江、安徽、福建、山东、湖南、广东、四川以及雄安这“十省一区”在专项债发行时，开展“自审自发”试点，把专项债的审批权下放到这些地方。

这十省区市，就是目前阶段认定的“经济大省”，可以获得更多的授权和支持。而在2024年底的经济工作会议上，原本跟城乡融合并列的“区域协调发展”被单列了，“加大区域战略实施力度，增强区域发展活力”成为单独一节，既然要增强活力，就必须授予更多自主权。

简单梳理了下过程，在小镇看来，2024年9月24日之后，“经济大省挑大梁”才进入实质阶段，经过2021年到2024年三年的酝酿，2025年会迎来经济大省扶持政策的突破和实质落地。

毕竟2025年面临的挑战很严峻。小镇之前在《2025年实际增速大概率在5%偏左》汇总了各省最近两年的增长情况，经济大省的情况也不太好。上面提到的十个省区市，在2023年、2024年都完成了GDP增速目标的，只有北京、江苏、浙江、山东4个，而安徽、湖南、广东已经连续两年没有完成预定目标，尤其是广东作为中国第一经济大省，2024年实际增速比预定低了1.5个百分点。

经济大省也面临各自不同的挑战，有的还是特别艰巨的转型难题。比如广东面临的主要问题是房地产下行危机，广东一直是房地产大省，两大房企巨头都在广东，房地产危机严重冲击到关联行业，还带来了债务重组和压力，目前危机还没有完成清理，就意味着2025年的广东大概率还是完不成5%的年度目标。

其他如湖南已经连续两年没有完成目标，虽然偏差值从2023年的1.9个百分点缩小到2024年的1.2个百分点，今年目标虽然下调到5.5%，但完成的难度很大。

复杂情况下，各省底蕴也大为不同，内部区域发展均衡、产业结构合理、科技创新旺盛、市场竞争力强的江苏、山东、浙江就很稳定，作为10万亿级别的经济大省，最近两年三省实际经济增速都保持在5.5%以上。

而在经济大省压力增大的情况下，对经济大省的要求进一步提高。今年新提出了四大要求：科技和产业创新、深化改革开放、落实国家重大发展战略、促进共同富裕。

那么唯一的解决之道，就是对经济大省进一步放权，提供更多支持性政策，在总体继续强化中央集权的大趋势下，要对地方放权，难度很大，大大超出各部门的常规决策权限，需要一号的直接拍板决策。
