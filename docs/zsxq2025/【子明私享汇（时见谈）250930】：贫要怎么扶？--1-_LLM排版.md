
# 贫要怎么扶？

## 发布专栏：知几共话

如何解决“相对贫困”,背后仍是是“鱼”和“渔”的思考。2025年7月30日，民政部发布《2024年民政事业发展统计公报》。数据显示，截至2024年年底，全国共有经常性社会捐赠工作站点和慈善超市1.3万个(其中：慈善超市3594个);全国社会组织捐赠收入1297.9亿元。

2020年，我国正式宣布实现了“全面脱贫”。但在日常生活中，我们偶尔仍会听到一些民众生活困难的案例。随着慈善事业的发展，我们也增加了许多帮助他们的渠道和工具，然而，什么样的资助才是行之有效的帮扶方式呢?这其中有着怎样 的经济逻辑呢?

我想请教下，有什么比较靠谱的方式，能让捐款给到真正困难的人手中?韩红慈善基金会?还是别捐给私人的慈善基金，韩红的基金会之前就曝出挪用资金的黑料，李连杰的壹基金还支持藏独。我一般用腾讯公益，每年捐一点，还能抵税。

我喜欢捐给学校。就像国家队救市一样，我更倾向于如何用最小的资金达成最大的成效。直接捐给困难的人其实不难，难的是你的捐助能否真正帮他们摆脱贫困。解决贫困的核心是提升生产力、匹配劳动生产关系。我大学同学有不少都是农村出来的，现在都能实现底薪30万。我觉得帮助农村培养更多的人才，这样的扶贫才是真的有效的。美国年年给非洲扶贫，有用吗?没用的。中国把产业链转移过去，培养出来的技术工人，这才是真的扶贫。

我的扶贫思路是帮助底层的优秀人才打通向上进阶的通道。这个思路肯定是没错的，但是我们要去想，这事应该让谁来执行?后续由谁来监督?什么时候能干完?如果说十年后才能出结果，那以我们村老人为例，他们可能都等不到那一天，还谈什么制度和宏大叙事啊，帮不到他一点。我是赞成明公的思路，但也确实存在这样的个案。针对个案要用特殊性制定特殊的帮扶政策，而打通上升渠道才是在长时间内应该逐步实施的，要不然后这种个案会一代接一代，这种上升策略就是尽可能减少他的后代还处于贫困的概率。很多时候，瞎扶贫、瞎资助，还不如不资助。

嗯，是啊，我们现在身边确实还有人生活非常艰苦，有的甚至每个月生活费不足10块钱，家里连电灯都不敢用，为了省钱偶尔晚上点一下蜡烛，一根蜡烛能用几个月，真的很难想象这种生活。其实现在的政策已经相对健全了，比如五保户是有国家补贴的。先富带动后富，始终是我们的政策方针之一，这是历史发展规律。面对尚且困苦的民众，我们愿意伸出援助之手，而正确的资助方式能事半功倍，打通他们向上的通道，使他们融入社会大生产，才能让社会需要他们，而非常需要社会。
