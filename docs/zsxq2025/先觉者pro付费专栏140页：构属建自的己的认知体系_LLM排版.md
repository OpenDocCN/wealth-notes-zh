
# 先觉者pro付费专栏140页：构建自己的认知体系

## 边界之外：构建认知的开始

很多人问我问题。提出问题的过程，也是解决问题的过程，一个善于提出问题的人，他的问题多半能在提问中就得出答案。所以一个人是不是善于提问题，本身就是认知能力的体现。但是现实中大多数人都是不会提问的，从他们的发问就显露出了认知上的混乱，而出现这种混乱的原因，在于他的认知缺乏体系与结构。

## 什么是认知的体系与结构?

举个例子，一个机器人出了故障无法工作，有人就会问维修人员：这个机器人坏了，怎么办?这样的提问，维修人员肯定也解决不了问题。正确的、有利于解决问题的提问方式应该是这样的：

这个机器人无法行动了，是不是它的机械腿坏掉了?

更好的进一步的提问方式则是：

这个机器人其他功能都正常，只是不能移动了，是否是它的传动环节出了问题?或者是动力系统出了问题?

当然，如果能更好地提问，你就能找到问题的答案：

这个机器人有哪些功能?

每个功能的关键部位在哪里?

损坏造成的故障是什么?

排除故障的方法是什么?

认知的一个关键作用，就是解决问题。很多人觉得自己认知很高，这也知道，那也知道，但实际上却什么问题都解决不了，真叫他干一件事，这也不行，那也不行。他不是真的不行，只是缺乏认知体系和结构，只要稍一点拨，立马就能改头换面，做什么样样行了。

因为很多的问题，不是靠破碎的信息就可以解决的。破碎的信息能够轻易获得，但是认知的体系和结构却很难建立。只要稍微复杂一些的问题，稍微超出生活经验的问题，就需要超出经验之外的解决方法。

## 换言之

靠日常的生活经验就可以解决的，你可以解决，别人也可以解决，谁都可以解决，这个东西就不值钱。而真正值钱的，能让你的认知产生价值的，就是你能超越日常生活经验的认知，能够超越他人思维边界的认知。这就需要你构建自己的认知体系和结构，而且你的认知体系和结构，要比大多数人的维度更高。

遗憾的是，这个世界上大部分人，都是被社会打磨出来的相似型号的机器人。不仅外形相似，从思想观念到言行举止，再到生活方式、家族传续都出奇地相似，他们的时代风格、潮流时尚、观念认知，都像是一个模子里印出来的。

## 站在更高的视角

如果你的视野足够高的话，就更容易发现这一点。就像站在几十层楼的楼顶俯瞰城市，你会发现城市街道上涌动的人群，每一个阶层的人，都有每一个阶层的一致性。

为什么会这样?因为本质上，社会还是由人在治理，而在社会治理上，天然有同一性的需求。人们越是趋同，社会治理越简单，治理成本也就越低。人们越是趋异，社会治理难度就越大，治理成本也就越高。
