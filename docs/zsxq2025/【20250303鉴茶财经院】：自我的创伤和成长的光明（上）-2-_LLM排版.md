
# 自我的创伤和成长的光明（上）

今天的文章很重要。白宫会谈当中，特朗普，万斯为什么竟能置大国体面于不顾，和泽连斯基当场吵了起来? 大风大浪见得多了，他们为何在世界媒体面前就能吵了起来? 因为真的压不住，因为泽连斯基激活了他们心灵的创伤。

一、吵架的时候，人们展露的才是自己内在的真实人格。

## 1,万斯一直讲什么?

他讲你小泽不尊重总统，不尊重美国。万斯的心理创伤，是得到认可与重视。万斯小时候，家庭破碎，母亲改嫁5次，他是一个极度被忽视，被歧视的人。不停转学，换环境，他融入不了小朋友的圈子，孤独，缺爱，得不到他人的鼓励和认可。“被重视，被认可”，是他的执念，他非常恐惧被人忽视，所以他非常努力，上进。他必须用“常春藤精英”、“硅谷投资人”“副总统能”等标签武装自己，以对抗潜意识里的“低价值感。因此，他会在字里行间和细节中，来寻找自己被重视、“被认可、“很优秀的肯定，才能获得安全感。他通过指责别人、碾压别人，要人感恩，来获得优越感。

## 2,懂王也不是个正常人

懂王小时候父母忙着做生意，没精力管他，他学习又差，经常通过各种叛逆急于证明自己。是个自卑又自负的人。这种人都通过崇拜强大来慰藉内心，所以很粉人设是硬汉的大京子。谈判中，懂王一直讲什么? 你没有牌，你陷入大麻烦。只有我能帮你。我强大。没懂王你2礼拜就完蛋。懂王最担心的事情，是虚弱，是自己实力不够，是被看穿底牌。他恐惧的是真实的自己被人看见”，怕别人发现那个掩盖在强者外套下的自卑小孩。极度害怕的是被戳穿”。因此他反复炫耀自己什么都懂，他实力很强， 一切都在掌握，听他的就行。

但泽连斯基讲什么? 大京子也说2礼拜要搞定我，一样没搞定，大京子出尔反尔 你也信? 你才给我多少东西? 欧洲给的更多。没你支持我一样行。言外之意是，懂王你并不像吹嘘都那么强大，你外加强中干。你不敢给安全承诺，你不行。你弱。

瞬间也把懂王给干破防了。懂王和万斯，都是没有能够真正接纳自我的人。一个恐惧被人忽视，天天要尊重，一个焦虑自己不行，天天讲交易艺术。内心充盈的人，是非常谦卑的，根本不在乎你是否尊重他，他尊重自己就行。自信强大的人，是不讲什么交易艺术的，三瓜俩枣有啥意思，平推就是了。这两个人，都活在层层包装之下，一旦被人撕开伪装，立刻痛彻心扉，产生应激反应。

我们的 一些国人，从50后到90后，基本都生活在奋进、生产、动荡、焦虑的氛围里，很多人童年和少年时，得到的爱和环境都是不够的。像万斯一样成长历程的人，很多。其实都正常。那怎么办呢? 成大事者要过情关，我们下期再讲。
