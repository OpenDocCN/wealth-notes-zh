
# 重启“钱袋子引擎”:中国经济突围战的最后一公里

原创：森 公众号：视野 25年3月10日

中国经济能否跨越中等收入陷阱，就看未来5年内人均能不能破2万美元。然而，我们现在依旧是偏通缩的环境。核心原因在于，市场的预期很强，可是，收入的现实依旧低迷。没有民众收入重新回归增长的通道，再好的愿景也只是愿景。

这次村里的不少新的举措，确实是直接锚定“收入正常增长机制改革”。如何从中获得有利于自己收入增长的机会？如何从这些机会中挖掘到好的投资方向？就成为了当务之急。今天的文章，非常关键。森不仅要回答上面2个问题，还要分析这一切背后的底层逻辑，以及村里全新战略布局中的“机会”、“风险”、“套利空间”与"真实战略意图"。

为什么加入WTO后，我们的全民收入可以如此暴增？我们不能忘记，这个世界上，有这么多发展中国家加入WTO并不只有我们。可是，只有我们，真正从中获得巨大的利益。

苏联崩了后，克林顿要求全美顶级大脑门，拿出一个方案，强化全球化的推进。通过这个方式，确保美国可以从全球获得最大化的利益的同时，也彻底垄断全球所有的财富关键节点。但是，这个也会有代价，那就是会把大量的低端就业岗位从美国本土，转移去东亚，尤其像中国这样的市场。

1995年的全球化旧金山会议上，与会的大佬们都很清楚这个代价，却根本没有去在乎这个代价，他们都只关心全球化的巨大财富前景。为了解决这个代价可能带来的社会风险，整个西方都同意，向自己的中低收入家庭的民众，尤其是年轻人，逐步输出“奶头乐”、提高社会福利保障和确保金融市场长期繁荣，三手抓，三手都要硬，以对冲大规模的普通家庭民众的失业。正是这个战略性的决策，才有了几十年后川普回归，以及他身后巨大的复兴力量。

在中国入世后，中国确实获得了巨大的全民收入增长，甚至大家觉得这是理所应当的。另一边，欧美日韩精英们则从全球化浪潮中，每年获取最最起码10%+的收入倍增。而这些人，同时还可以利用资产的涨幅，又从金融市场上，再捞一大笔。

因此，即便有识之士都觉得，日子是好过，可是后患无穷。然而，既然日子好过，那所有的隐患，就交给将来有智慧的当家人去处理吧。所以，次贷危机中的占领华尔街运动，还是中产家庭的运动；可到了现在，美国白人普通家庭进一步返贫，自然就变成了MAGA。

可以想见，对于MAGA而言，恰恰是克林顿等人掀起的全球化浪潮，让自己变成了返贫族的一员，民主党是坏蛋，其次就是抢了他们饭碗的中国人了。因此，在这个局中，中国天然就站在MAGA的直接对立面，不用怀疑。

另一边，民主党人之所以拼命推进全球化，且一开始对中国很包容，也有他们的算计。只要中国进入中等收入水平(人均1万美元~2万美元)，就启动全面收割计划。这个微操，是民主党历史上，屡试不爽的招数，太多前车之鉴的倒霉国家经历过。可是，如果收割不
了，那就会导致美国全球霸主地位不保，且美国财政的债务扩张会面临严重的瓶颈。

要知道，这种瓶颈是可以要了他们的命，甚至强化他们内部分赃不均引发的内斗。这就是奥巴马、拜登、佩洛西等人积极布局遏制中国的真实目的！

此外，就是我们说过的，眼下处于全球创新周期尾部，大家也都没有太多新的产业可以做，这天然使得国与国的关系就是偏紧张的，都要相互抢饭碗、抢储蓄、抢人才、抢资源。于是，现在的中国，同时面对三股史诗级的冲击：MAGA、反全球化的崛起；2、民主党的收割之战；3、创新周期尾部收入增速边际递减。这三股力量相互交织，对于我们过去几年的收入增长，产生了极强的负面冲击。

问题是，财富是要保持长期自由交换才能产生的。一旦你大规模囤积(不管主动还是被动)，甚至是超大尺度金融空倒，那就没有去自由交换。这时，手里的资产就会不断内在价值耗损，相当于持有者自己消费了。在这种背景下，除了从外部找现金流回补，根本没有其它办法维稳。这就是森独创归纳的“资产耗损回补理论”。

好了，关键点了。如果民众收入增长趋势瓦解，甚至还出现了严重的收入倒退，那么，市面上的自由交换就会大幅减少。而且，在预期恶化下，人们会逐步减少自由交换，选择囤积现金。 在这种局面下，大家手里的资产就会因为无法自由交换，而出现加速度的价值耗损体现为资产价格暴跌，或者是有价无市。

在这种背景下，一方面，民众的情绪会更加悲观，而使得上面的恶性循环加速；一方面，连村里的资产也会大幅贬值和价值耗损，他们也没有办法做到国有资产不流失。各位，这不就是过去几年，我们亲身经历的事情吗？所谓的偏通缩的经济基本面，恰恰是上述双重恶性循环的结果而已。因此，要走出困境，一切的一切，就是强化收入增长，这就成了起点，却也是终点。

大家会发现，问题到了这里，更加复杂了。三股力量的打压，使得我们的收入增长很难；可是，如果收入不增长，那么，我们的全局性经济压力就会持续放大。事实上，这就是眼下中国经济的重大难关。这个难关度过，才会有我们晋升为新的发达国家的窗口期。既然现在全球都没有太多新的产业做，如何发力？

所以，这次大会，村里用的组合拳，就要看得懂。首先，新质生产力加速投建和驱动，尽可能挖掘新的产业就业机会。其次，利用财政刺激的杠杆效应，比如以旧换新，让大家手里的钱可以买到更多东西，形成"财政刺激的乘数效应”，进而带动更多就业机会。再次，积极推进金融财富效应，除了明确提出“稳楼市股市”、大量创投基金支持股市等金融造富手段外，还第一次如此清晰的要求给大家的消费贷更长周期、更低利率、更高额度、更多宽限。这就是过去100多年欧美的宝贵经验。

还有，进行工资改革，形成收入再分配。也就是说，限制高收入群体的过高收入，扩大中等收入群体规模，提高低收入群体收入。另外，后续要降息降准，让全民在更低利率负担下，去消费和自主创业。最后，继续保出口，能保多少就保多少。

这里，我们做一个总结：
消费贷+房股拉行情+新产业
收入再分配+提高退休金+稳出口
=收入增长改革方案

有能力实业赚钱的，去实业赚。实业中赚不到多少钱的，可以去金融市场找机会。有特殊理工技能的，新质生产力给你们机会。收入再分配，实现普惠性。对于能力实在很有限的城市年轻人，还可以啃老。对于有正经工作、但收入未必很高的年轻人，可以用消费贷来解决收入不足引发的消费能力欠缺问题。这一套组合拳，基本上，就把这个社会上大部分人的不同情况，一网打尽了。

不过，整个传导，并不是所有人马上可以体会到的。上述适合不同人群的政策，有些人会更加体会到，比如新质生产力、股市、部分收入再分配的获益者。可是，也有不少人，或许要等到下半年，甚至是明年才会有体会。从中我们不难看出，人群中收入趋势的分化，一定也会带来他们各自手里有的资产价格的分化。大佬文集圈更早获得收入增长政策红利的人群，他们的资产价格也就会越早拉升。反之，越晚享受到收入增长政策红利的人群，他们的资产价格有可能要等待整个经济都修复了，才有可能起来一点。

因此，投资不仅是套利，也是充满风险。你必须知道，不同资产背后的持有者人群到底是谁，你在跟哪些人群站在同一个战壕内，这一点是选筹的重中之重。

接下去，我们来看一下，后续几个值得特别关注的投资逻辑和主线。
1、MAGA的崛起，意味着，全球很多资金重新加速分配。因此，瑞士、欧洲的军工、黄金、人民币核心资产(美国的对手盘)都会从中获益。 而且，香港会重新夺回前几年被新加披拿去的亚洲金融中心的地位。此外，川普也是普惠性的要回报，所以，最终他要启动降息，最起码是弱美元和弱美债，否则没法玩下去。

这个对于中国核心资产的重估，意义重大，属于重大利好。

2、面对前期拜登的金融战，中国流动性一度失血。现在要确保2030年左右实现中等收入陷阱的成功突破，中国国内的刺激力度只会越来越激烈。这里我们要明白，不仅仅是关税战，而且拜登的烂摊子，都倒逼我们需要优先搞活内需。

现在川普之所以这么疯狂，就是因为美国阶段自己加杠杆也是乏力。那么，中国加杠杆正当时的过
程中，货币流动性持续充裕，甚至还要虹吸全球流动性，就是必然的。

基于此，本轮救市行情，我们需要注意：
第一阶段，科技先行，先拉起高度，炒热预期
第二阶段，传统内需板块，逐步启动
第三阶段，确认通胀回归

这是年内我们投资思路的主线中的主线，更是把握节奏的底层逻辑。面对现在还处于第一阶段，非常多的人，至今还没有意识到上述逻辑。

3、创新周期尾部的困境
除了上面提到的新质生产力，国内大规模基建和其他重大项目的落地，也是重要的应对手法。具体落实为，统一大市场。要实现这个，那么，在物流、通讯、电力、机械、平台等诸多节点上，今年村里的动作会非常大，项目会非常多。因此，我们不要一提“创新周期尾部”就觉得这东西是负面的。

其实，正确的思路是，越是外面干扰项多，越是“创新周期尾部”冲击大，越是要深挖内需，越是要大规模上项目。可以说，这一块的投资机会和创业机会，也就逐步打开了。

不过，这里我们仍要明确一点，任何投资都要先行尊重上面的三个阶段理论，这是政策红利传导机制，绕不过的。如果不能尊重现实，强行蛮干，那就是机会变成了风险。

其实，要提高收入，还有一门绝招。收入，实则是区分为"实际收入”和“名义收入"。名义收入，就是你实得多少钱。实际收入，就是把你实得的钱，扣除通胀后，还剩多少钱。所以，不要觉得提高收入的办法就没有了。

真的形势紧迫，考虑到我们供应链上游近乎全部是在村里手中，还可以人为创造通胀。通过这个方式，倒逼名义收入增长，即，形成“收入-通胀”螺旋强化的循环。对比欧美日韩印，我们要做这个事情的空间，可以说是巨大的。

而且，由于我们一手控制全球最牛的供给侧，真的要压低通胀，也有的是办法，毕竟产能无限大。

一旦名义收入增长，哪怕一部分是被通胀重新顶起来的，那么，现有的各类资产的自由流通就又活了。这种盘活，会极大的刺激不少资产的保值增值，进而驱动内需加速回暖，带动更多人的收入增长，形成“乘数效应”。其实，近期严惩“内卷式竞争”，就已经包含了这个战略图谋了。

下一个关键观测点，就是上游很多重要的产成品，会不会集体涨价。如果这个节点来了，那么，整个“名义收入—通胀螺旋循环就会提速，进而带动更多资产也加速修复价格。从这个角度，现在布局优质资产，就是跟通胀“抢跑”，比谁先到。

由于我们是郡县制市场经济，而不是美国式的私有财阀联盟，真的要实行这些举措，会非常快。后面几个月，CPI和PPI的走势，至关重要，将直接影响市场的情绪、决策和热情。市场是逐步走强，还是加速走强，就看这组指标。

以上是我个人的情况，仅仅作为心路历程给大家分享，不作为直接投资建议，还请知悉。
