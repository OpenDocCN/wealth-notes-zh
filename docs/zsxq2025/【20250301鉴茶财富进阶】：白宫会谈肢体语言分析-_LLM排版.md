
# 白宫会谈肢体语言分析

小泽去了美国，就俄乌停战和矿产协议和懂王会谈，在一大堆记者和媒体面前，他们谈的很崩，我们从现场姿态，分析一下这三人的肢体语言。

## 小泽

1. 双手环抱在胸，这是一种极强的物理防御姿态，说明他缺乏安全感，无意识的自我保护。
2. 屁股坐半边椅子，说明焦虑，不安，缺乏自信，坐不住。
3. 双腿是学生式正坐，略悬空，说明拘谨，不踏实，未来迷茫。
4. 身体向懂王一切较大倾斜，说明有求于人，又比较急迫。
5. 说话却眼神不盯着懂王，说明抵触、被动、有怨气、不够敞开心扉。

## 懂王

1. 双腿分的很开，脚在地上，屁股坐大半个椅子，说明占主导地位，把握局面。
2. 双手掌型摊开式比划，动作很大，是开放，自信，咄咄逼人，并不在乎对方的看法。
3. 身体向小泽一方靠拢，是无意识施加压力，试图以力服人。
4. 身体也略前倾，说明他内心也略有急迫，并不十分从容。

## 万斯

1. 双掌交叉，表现出积极，自信，踌躇满志。
2. 双臂支撑在膝盖上，说明警觉，控制，但又充满准备和期待，随时跃跃欲试。
3. 身体前倾，眼神盯着发言者，说明专注，认真，随时在分析和权衡。

人的行为习惯不是一朝一夕所能改变的。这是一场从肢体语言就看出谈不出结果的东西，既不公平，也没诚意，双方都很急躁，在大庭广众之下，他们的细节被无限放大。

在沟通场合中，有一个信息传递模型：

- 词语传递7%的信息
- 语音语调传递38%
- 肢体语言传递55%

调整自己的姿态行为，有时比你讲话要有用的多。
