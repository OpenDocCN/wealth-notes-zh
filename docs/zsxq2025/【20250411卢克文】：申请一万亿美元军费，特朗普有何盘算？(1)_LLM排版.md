
# 【20250411卢克文】：申请一万亿美元军费，特朗普有何盘算？

2025年4月7日，就在全世界的注意力集中于关税大战时，特朗普悄悄做了一件大事--申请一万亿美元国防预算。

不对啊，特朗普不是要降本增效，要削减国防预算，怎么军费还暴涨了呢，左手省开支、右手就扩军，岂不是左右互搏。那么,特朗普这种自我打脸的操作，有何盘算?

美国的国防预算向来一骑绝尘，占全球军费总开支的近40%。过去数年，美国一直维持每年300亿-500亿美元的军费增幅。照这个进度，原本超1万亿美元，也就再等个三四年。特朗普不过是加快了这一进程，没有他，达到1万亿也只是时间问题。

但钱也不是想加就加，每次加钱，依旧得面对那个灵魂拷问，钱从哪来? 现在美国的债务状况和财政赤字不必多说，马斯克率领的效率部风风火火干了快两月，在几乎得罪整个华府官僚集团的价格下，才削减了1300多亿的开支，远不及预期。好家伙，现在特朗普脑子一热，将去年底拜登签批的8950亿预算再提升至1万亿，相当于马斯克累死累活干出的成果，转手又贴给了五角大楼。

这恐怕是人家准备提桶跑路的原因之一。

那么,这么一大笔钱，都准备花到哪呢? 美军未来烧钱的几个大项目，包括福特级航母后续舰、哥伦比亚级战略核潜艇等，这些都是单价130亿美元/艘左右的吞金巨兽。空军方面，有哨兵洲际导弹，六代机，F47B21 轰炸机以及有新版“星球大战”计划之称的金穹导弹防御系统等等。除了装备更新以外，空袭胡塞武装，调兵威慑伊朗，军援以色列乌克兰，都是花钱大头。

特朗普突然提出万亿级军费，有以下四个方面的盘算。

一是为了明年中期选举的选票。3月30日，特朗普声称自己不排除寻求第三任期的可能性，而且有方法可以做到。但是想要实现第三任期，必须修宪。这就得掌握国会两院三分之二绝对多数，明年中期选举，共和党必须大胜。想要争夺选票，创造就业岗位无疑是最好的噱头。而通过军备采购拉动就业，则能实现特朗普“以军带民”的政治诉求。比如前不久宣布的波音六代机项目。从创造就业上看，项目理论上可以创造2.3万个就业岗位，辐射链覆盖了12个关键州(大多为铁锈带摇摆州)，其中最大的密苏里州圣路易斯工厂直接创造1.2万个岗位，并通过上游供应链，影响摇摆州俄亥俄州(航空铝材)，预计可以影响包括员工家属在内的大约40万张选票。项目的200亿美元首期投入中，40%定向流入特朗普重点扶持的铁锈带军工企业，包括俄亥俄州莱特-帕特森空军基地的数字化改造项目等。

其他的例子，包括诺格公司在得州新建的生产线，直接关联着2.5万个就业岗位，洛马公司的供应链，更是覆盖美国一大半的州。这些个以军养民的大项目，特朗普必须加大投入，好成为自己笼络选民的金字招牌。

二是为了挖民主党墙角，分化诱导军工复合体们“投共”。3月23日，特朗普宣布了前文提到的军工项目一“金穹”国家导弹防御系统。特朗普定下的指标是：能够100%拦截敌国发射的导弹，拥有极快的反应时间，够在敌国的弹道导弹发射后第一时间予以摧毁。稍有军事常识的人都知道，这想法完全是天方夜谭。但项目能不能成，特朗普其实不在乎，“金穹”计划有三个相对独立的部分，分为早期预警系统、中段拦截系统和末端拦截系统，三个系统，都是波音、洛马、诺格等传统军火商的势力范围。就连马斯克也能跟着分到蛋糕，卫星发射可是他的拿手好戏。特朗普还专门强调，要将金穹项目列为2026至2030财年的核心预算，什么意思呢? 就是未来研发费用会贯穿第二任期，金额庞大到无上限，各家公司能捞多少钱，全看特朗普的心思。这块千亿美元级别的大蛋糕，足以把军工复合体们撬过来，成为特朗普的支持者。

三是为了拉拢手握枪杆子的美国大兵们，花钱买忠诚。特朗普第一任期时，还愣头青般跟美军对着干，派人去阿富汗查账，最后弄了个机毁人亡草草收场，任期最后时刻，还被佩洛西联合参联会主席米利，架空了军权。特朗普也知道，美国军方腐败至极，但不提高军费，军方就不会鸟他，对于军头而言，谁当总统都一样，只要不妨碍搞钱就行。特朗普的对等关税操作，震动了美国朝野上下，上到华尔街的资本大鳄，下到从事工农业的普通人，几十万人走上街头反对他。可想而知，万一发生政治动荡，特朗普能够依靠的，只有军方。而现在把军费提高到一万亿，开价这么豪横，买的就是忠诚，什么几千美元的咖啡杯，几万美元的螺母，几百万美元的山羊统统忘得烟消云散，得到军方的拥护以后，特朗普再怎么胡作非为，总统位子都能稳如泰山。

四是为了维系美国的军事霸权，为外交博弈增加威慑力。不同于拜登时期的保持现状，特朗普的军事威慑更加激进。一方面是自己亲自动手空袭胡塞武装，又调派进驻迪戈加西亚基地威慑伊朗，B2对以色列的军援更是只多不减。另一方面则是继续鼓动自己的马仔出力。菲律宾在防长赫格塞思访问后，立即将苏比克湾两岛，划为了军事保留区，此区域距中国黄岩岛仅200海里，日本则在部署更多的无人机，配合美国收紧对华监视网。而最近，中国在军工装备领域的持续爆发，更是极大刺激了特朗普军备竞赛的决心，这些都需要白花花的银子投入。尤其是近年来随着美国执行战略收缩，很多国家或者武装势力，不再把美国放在眼里，这一点让特朗普很不爽。所以他才说，要针对那些坏势力重整军队，要让世界再次害怕美国。

综上所述可以看出，特朗普这次画出的万亿级军费大饼，绝非单纯的国防需求，而是政治算计、利益输送与霸权野心的混合产物，最终目的还是为了自保。

这一万亿美元，也许能为特朗普家族带来安全，但对美国而言，却带不来安全。中国用占美军三成比例的军费开支，成功在多个领域建立起对美军的优势。在军费投入不对等的情况下，军事建设上还能取得好成绩，证明我们的军费利用率极高，哪怕美国再维持高额的军费开支和增幅率，也只能减缓中国缩短差距的速度，未必能跑赢解放军的实力增长曲线。

此外，特朗普的关税乱拳，也在无形中给美国的军事建设工作带来超高的额外成本。中国对于多种稀有矿产的出口管制，会让F35、B21等高科技武器的生产成本直线飙升，甚至影响正常交付速度。说句难听的，美军最该考虑的事，不是军费涨到1万亿美元后，能不能和中国不到3000亿美元的军事建设成果一较高下，而是应该考虑追加的这1000多亿美元，能不能抵消掉关税政策引发的通胀。

美国国债如今已经高达36万亿美元，每天要还大约22亿美元国债利息，相当于每分钟烧掉150万美元。按照美债的增长速度来估算，到2028年美国国债利息将超过军费开支，到时候，军费可能将难以为继。历史证明，军事膨胀从未真正带来安全，只会滋生更多不稳定。1914年的欧洲，新兴工业强国，与老牌帝国的博弈，最终引爆世界大战。1930年代，经济大萧条催生的民粹主义，将世界拖入更黑暗的深渊。如今美国的战略，与当年何其相似：将头号工业强国视为威胁，在关税战遇挫后转向军事冒险，这种路径依赖，正在重蹈历史覆辙。当美国民众为这份天价账单买单时，或许该问一句：究竟谁才是特朗普口中的坏势力?
