
# 大鹏一日同风起（中国正式确立自己为世界头部强国）

有人说，最近有三件大事，堪比中国在外交领域的三大战役，是确立中国作为世界头部大国的重要标志性事件。一件是九三大阅兵，另一件是福建舰实现电磁弹射和回收能力，第三件是中国在联合国宣布中国在当前与未来的WTO谈判中，不会寻求任何新的特殊和差别待遇（SDT）。这三件事密集发声在今年9月，是中国正式确立自己作为世界头部强国的标志性事件。

大阅兵和宣布取消特殊待遇可能相对好理解，很多人对福建舰实现电磁弹射的重要性似乎认识不足，认为这只是一个技术环节的小进步，为何会成为标志性事件。但是从官媒报道的力度和选择的报道时机来看，这件事不仅仅是海军领域的一个技术突破。这次官方通稿的内容是说：歼-15T、歼-35和空警-600三型舰载机，已于此前完成在福建舰上首次弹射起飞和着舰训练，标志著福建舰具备了电磁弹射和回收能力。这个通稿释放了两个消息：第一，按照中方一贯风格，能这样公开实机电磁弹射的长影片，说明已经训练得很成熟了，这意味着福建舰航母已经形成初步战斗力，距离完整战斗力，也只是时间问题，剩下的只是舰载机配置、人员配置积累。第二，歼-15T、歼-35和空警-600，这三款完全不同类型的舰载机，均通过电磁弹射完成舰载起飞，充分说明中方自主研发的电磁弹射的可靠性。

目前世界上的航母起飞方式主要有4种：滑跃式、蒸汽弹射式、电磁弹射式、垂直短距起降。其中滑跃式起飞是最早的航母起飞方式，受限场景比较多。滑跃式起飞的舰载机，重量不能太重，否则就起飞不了，所以滑跃式起飞的舰载机就没办法重载起飞，携带的弹药会比较有限，滑跃式的舰载机起飞效率也比较慢，没办法搭载固定翼预警机，于是就研发出弹射式起飞方式，根据弹射方式不同，又分蒸汽弹射和电磁弹射。

蒸汽弹射使用的是蒸汽驱动，电磁弹射则由电力驱动，相比蒸汽弹射，电磁弹射的效率更高，而且弹射过程是比较均匀加速的，会更温和，对舰载机和飞行员的负荷更低。所以，电磁弹射可以适用所有类型的舰载机，并且效率更高。在此之前，世界上只有美国福特级航母是用电磁弹射，美军其他航母也基本是采用蒸汽弹射。但福特级航母的电磁弹射出了名的可靠性差，故障率高。电磁弹射器最早的设计指标是每弹射4166次允许出现一次故障，但据美国媒体的报道，福特号在实际测试中，平均181次就会出现故障，实际故障率是理论故障率的23倍。因為故障频发，川普在2019年还曾经说，他计划为美军航母订购传统的蒸汽弹射装置，替换在战时可能发生故障的新型电磁弹射系统。川普认为：美国把大量的钱花在了电磁弹射系统上，而没人知道在恶劣条件下它的性能如何。所以，美国的电磁弹射技术并不成熟。

相比之下，中方的电磁弹射技术使用的是直流电，不同于美国使用的交流电，有中国电磁弹射之父称号的马伟明，在2017年接受采访时就表示，美国目前是中压交流，还处于第一代水平，差我们一代。美国才刚开始做预研中压直流，要赶上我们是十几年以后的事情。现在马伟明设计的 中压直流综合电力系统已经广泛运用于中国各类新式舰艇上。而目前海外包括美国在内都还停留在中压交流综合电力系统，而中国则已经率先在各类新式舰艇上使用了中压直流系统，中国电磁弹射的储能系统是采用超级电容的方式，具体参数不详。而这还是8年前的采访内容，以tg (土贡)的效率，这八年要做多少事，但美国的福特级航母的电磁弹射系统，至今都还没有公开的影片证明能完成对F-35C 的弹射。这次福建舰一上来就公布了对歼-15T 这类重型舰载机，以及歼35这类隐形战斗机的电磁弹射，说明中国在相关技术方面不说遥遥领先，也已经做到人无我有，人有我强了。

通稿里还有一句；航母在电磁弹射技术上的成功对推进海军转型建设具有里程碑意义。这句话非常关键，技术能力决定了空间大小，这意味着，中国海军将从近海防御，开始朝远洋作战的方向迈进。拥有完整战斗力的航母战斗群，对中国在对岸外围，拒敌於国门之外，有重要作用。登岛本身是 用不上航母，但在第一岛链外围，将美军拒之门外，航母能发挥重要作用，这次基本是明牌了，福建舰的打击范围可以覆盖第二岛链。所以，某种程度上，福建舰形成完整战斗力，已经将美国在近海跟中国进行海战的想法彻底掐死在摇篮里，同时也将大大推进台海统一的步伐。所以，这件事的意义不仅仅局限于军事领域，而是接下来很多重大战略决策的物质基础。

而这次李总在联合国会议上强调，中国在当前与未来的WTO谈判中，不会寻求任何新的特殊和差别待遇 (SDT)。也是中国公开承担 大国身份的标志性事件，说白了，中国已经是真正的巨人，不需要发达国家的特殊照顾了，商务部的官方描述是，中方此举彰显了以实际行动支持多边贸易体制的坚定立场和大国担当，但同时强调中国 仍是发展中国家。实际上，中国现在实际享受的SDT有限，反而因此屡遭发达国家批评，如今做出政策宣示，不仅有利谈判推进，也能在美国转向保护主义的背景下发挥引领作用。过去在WTO 谈判中，美国与其他西方国家经常批评中国假装发展中国家、拒绝承担更多责任与义务。如今中国做出政策宣示，就是表态中国不会再以发展中国家的身分和其他发展中国家共同寻求特殊的差别待遇。这相当于将自己区别于一般的发展中国家，并且表达出自己可以做大哥的意愿。

2001年，中国加入世贸组织，成为第143个成员。加入 WTO, 中国即取得多边、稳定、无条件的最惠国待遇，并以发展中国家身份获得普惠制等特殊优惠待遇，使中国出口贸易有较大的增加。但中国实际上享受的SDT 极有限。根据WTO 现有155条特殊和差别待遇条款，中国仅享受其中15条，其中真正具有价值的是农产品补贴的微量允许条款，允许发展中国家补贴产值10%,而中国的标准是8.5%。此外，中国在COVID-19 疫苗知识产权豁免，以及贸易便捷化协定上也放弃部分SDT, 所以看似优惠，对中国已经影响不大了。反倒是常常落人口实。不过，中 国虽不再追求SDT, 但仍强调保有发展中国家身 分。因为发展中国家是一种政治定位，而非单纯经 济概念，中国从过 去的第三世界国家到如今的全球 南方国家领头羊，发展中国家成员的定位更符合中 国的政治、经 济与历史条件。商务部国际贸易谈判 代表兼副部长李成钢昨天也在发布上会上强调，中 国仍然是世界上最大的发展中国家，中国发展中国 家的地位和身份没有改变。中国始终是全球南方的 一员，永远和发展中国家站在一起。虽然已经上身 穿西装了，但是双脚插在泥土里穿草鞋的传统不能 丢。

而刚结束的九三阅兵更是通过亮相一批全球 领先的新武器新装备，让世人看到中国军工和科技 的发展。这之后不但有沙特跟巴基斯坦缔结战略安 全协议，更有中美关系融冰，国会议员时隔6年访 华。道理说不通的时候，略通拳脚还是管用的。去 年12月，《纽约时报》专栏作家托马斯 ·弗里德曼 在访问北京和上海后写道：在我们睡觉的时候，中 国已在高技术制造业的所有方面都取得了巨大的 飞跃。今年5月，《纽约时报》又发了一篇文章，在《中国远远落后于美国的时代已经终结》,文中 表示：十来年的功夫，中国已完成从山寨之国到世 界级产品巨擘的蜕变，有些甚至超过了西方的产 品。曾经以模仿 iPhone 闻名的小米去年交付了 13.5万辆电动汽车，而在十年间投入100亿美元 后，苹果公司放弃了生产电动汽车的计划。中国正 加速推进机器人规模化部署，并制定了大规模生产 人形机器人的计划；2023年，中国安装的工业机 器人数量超过了其他国家之和。

对于中国依然是最大的发展中国家，面临创新 能力不适应高质量发展要求，产业体系整体大而不 强、全而不精，关键核心技术受制于人状况没有根 本改变的问题，但问题就是发展的契机，正是因为 我们有这样那样的问题，所以今天的中国依然是充 满希望，值得期待的。如果没有问题，那么发展的 动力和方向在哪里，解决一个又一个具体的问题恰 是发展的底层动力。所以人不要怕自己有短板，反而要担心自己看不到自己的短板。

现在回头看中国这几十年的发展，不管是谁，都会感触良多。最近网上有个帖子，说了件往事，1992年，也就是30多年前，钱学森给时任副总 部长邹家华写了一封信，钱学森是当代中国十分重要 的科学家，被誉为中国航天之父和火箭之王。1992 年8月，已经81岁高龄的钱学森写信给邹家华，从产能、环保、国际形势、技术可行性与现有技术 基础等方面进行分析，建议中国汽车工业应跳过用 汽油柴油阶段，直接进入减少环境污染的新能源阶 段。钱学森在信中表示：我们决不应再等特，要立 即制订蓄电池能源的汽车计划，迎头赶上，力争后 来居上!大约10来天后，邹家华回信钱学森，不 仅认为钱学森的建议有远见，十分重要，已经责成 国家计委科技司具体研究，而且提出中国开始展 开电动汽车的研制开发计划。要知道，这 可是30多 年前呀，当时我们连手机大哥大都没见 过，电 力驱动的汽车，对于一般人来说简直是天方 夜谭，然而已经有国士以超乎常人想象的眼力为国 家 未来发展献言献策，更重要的是，这些话被听进 去之后，还被坚定的实施执行哪怕领导换了，事情 没有被耽误。

自此之后，中国开始投入人力物力攻关电动汽 车的研制开发。在八五、九五、十五、十一五期间，中国持续投入资源培育电动汽车产业，在2008年 前后初步形成相当规模的电动汽车整车和关键零 件生产能力。2012年6月，国务院发布《节能 与新能源汽车产业发展规划(2012—2020年)》，提出加快培育和发展新能源汽车产业，实现由汽车 工业大国向汽车工业强国转变。伴随国家政策的有 力扶持，新能源车开始快速发展，速度远远超出预 期。如今的中国是全球第一的新能源车产销国。有 数据显示，2024年中国新能源乘用车全球占比高 达70.4%。可以说，曾经在 汽车工业领域严重落 后于西方的中国，通过电动车 真正实现了弯道超车，中国在新能源车领域乃至背 后整个产业链已 经具备全球领先优势。恰恰是中国 在新能源方面 的底气，才能让一个发展中国敢于公 开承诺未来 十年减排7%—10%的计划。

今年是十四五的收官之年，也是中国制造 2025的成果展示年，只能说，这个国家认认真真 做事的人太多，默默埋头苦干的人太多，关注国家 前途命运的人太多，不管舆论场上多少风波，从 来没有真正影响这个国家前进的步伐，纵观历史，可 能从来没有哪个时期可以像现在的中国一样极大 释放人民的积极性和潜力，如果这都不能算制度优 势 ，那就不知道什么才算制度优势了。
