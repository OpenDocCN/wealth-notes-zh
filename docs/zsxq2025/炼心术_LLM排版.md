
# 炼心术

## 决裂：剪掉头上的辫子，也要剪掉心中的辫子

在清朝末期，许多人习惯蓄长发、扎辫子。当时，从国外留学归来的中国人如果剪了短发，会被认为是“假洋鬼子”。这种行为背后是深层的意识形态，认为留头发是天经地义的，不留头发是大逆不道。

1911年，主席18岁时接受了新思想，决定剪断自己的辫子。这不仅是剪掉头发，更是对抗满清腐败黑暗社会的一种行为。身边的同学起初犹豫，但主席带头剪辫后，十几个人跟着一起行动。

剪辫子象征着与过去的决裂，重新活出一个自我。大司马曾说：“踏着自己的尸体前行”，不断打败自己，树立全新的自我，才能告别过去，拥有未来。

### 心中的“辫子”

每个人心中都有许多“辫子”，有形的和无形的。

**1. 感情**：许多人沉浸在过去的感情中，无法进入新的感情，活在过去的思念中，不利于拥抱未来。

**2. 婚姻**：一些夫妻关系紧张，生活在对立和煎熬中，对孩子的成长也没有积极影响。当断则断，才能拥有更好的未来。

**3. 圈子**：无效社交消耗光阴，浪费时间，思想毫无禁忌。成年人应该认识到这一点，与过去的圈子决裂。

**4. 客户**：一些销售人员被劣质客户拖累，浪费时间和精力。应该果断舍弃，与优质客户合作，才能取得进步。

### 无形的“辫子”

**1. 认知**：如果不能屏弃过去的错误认知，就无法拥抱新的认知，认知不迭代，就会原地踏步。

**2. 虚荣心**：好大喜功、虚荣心过重会导致失败。追求名牌和奢侈品如果不能带来实际价值，只会增加烦恼和压力。

**3. 贪婪**：学无所成、学无所精，是因为贪婪，没有一门学问是学深的。应该专注一门学问，深入钻研。

**4. 面子**：把面子看得比天还高，往往无法成事。真正开窍的人不会把面子放在第一位，而是务实。

### 总结

与过去的自己决裂，才能拥有未来。无论是剪掉头发，还是剪掉心中的“辫子”，都是重新活出一个自我的过程。只有不断反思、总结，才能不断进步，拥有更好的未来。
