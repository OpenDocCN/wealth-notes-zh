
# 会议解读

统战论文章再度拖后，回答瓜友问的比较多的问题，也插播今日会议解读。

1、不是说开河会吗了？怎么今天还在北京开局会？我文章说的是七月初开河会，请看清楚。北戴河地区开始全面禁止无人机，也是从7月1日禁止到8月31日。没能去成的金砖峰会也是七月初。我昨天文章说的是准备提前开河会，在七月初，而不是正在开，请看仔细再问。

## 2、河会是提前开的吗？

河会历史上，基本是7月8月开，大都是7月下半和8月上半。七月初开河会看起来也没突破惯例，还属于七月，但只是看起来属于，实际并不是。一个是在我们角度，肯定希望河会拖到后面开，最好能拖到8月初开，拖的越晚越好，而不是提前到7月初。一个是预期的7月初金砖峰会我们是前几天才告知外国不去出席，也就是7月初我们的行程安排是出访的，并不是河会，是前些天才改变行程，这说明有突发情况修改了7月初的安排。这两点就注定7月初的河会是提前开的，而不是正常开。

既然我们不想提前开，也没准备提前开，那么事实是提前开了，所以说这是不利于我们的信号。而这个不利于我们的信号，则是6月开了很多天局扩大会议后产生的，所以的确不乐观。今天在北京开完这个会，明天后天就得去河北闭门开会了。

3、今天的会议怎么解读？出的《决策议事协调机构工作条例》是啥？有人说，这个决策议事协调机构是八王议政，或者是中顾委的翻版。说是老人们都能通过这个机构重新施加对国家政策的影响。

其实是错的。首先，这个决策议事协调机构不是新鲜事。在18年机构改革就出来过， 是既定的工作安排。

我用大白话和你讲吧。我们党政是分开的。时政界一直有府院之争的说法，这个说法在老二号时期达到高潮。一开始，用经济小组和鹤真人的几个金融委员会，分了老二号的经济大权。然后，再搞外交小组分了国务院的外交大权。这个做法其实汉武帝就搞过。汉武帝时期分外廷和内廷。外廷就是功勋大臣为主的丞相府。内廷就是汉武帝自己任用的一些秘书郎搞的小组。当下的小组治国和各种委员会分国务院的权，就类似汉武帝用内廷取代外廷的做法。我们的内廷，就是各类小组。而这些小组加在一起，有个统一的名字，叫决策议事协调机构。所以，这些年是一号权力不断扩大二号权力不断缩小的一个过程。标志性就是各类决策议事协调机构的诞生和扩大。而现在，要给这些小组委员会上一个紧箍咒了。

就是今天这个条例。《决策议事协调机构工作条例》——要准确把握职责定位，坚持谋大事、议大事、抓大事，对重大工作实施更为有效的统领和协调，做到统筹不代替、到位不越位。这些小组委员会，依然是抓大事。但他们的权力受限了。——做到统筹不代替、到位不越位。是的，这不是元老们借尸还魂搞什么八王议政中顾委，而是我们小组治国的buff被削弱了。府院之争以后会变成国务院的权力不断触底反弹，而 我们，也就是中央对各类经济工作的控制力下降。

好消息，不是八王议政。坏消息，集权程度还是被削了。当然，其实坏消息也并不多坏。加大国务院各部委的权力，党政分离嘛，让职业专业搞经济工作的部委获得更大话语权决策权，是合理的。也是合情的。毕竟我们抓了外交大权后，搞出了小刚子的事。我们抓了教育大权后，搞出了一刀切教培的事。现在党政关系进一步理清，加强国务院也是为了以后不再发生这类事。

当然，我们抓了政法大权后搞出了扫黑除恶这样青史留名的伟大功绩。有功有过的，不能一概否定集权行为。但集权趋势是会减缓，乃至慢慢放权的。只要放权不是放给老头子们，而是放给各部委各路大神仙，就行。只能说，在河会前夕，我们给集权的小组治国模式上一个紧箍咒，主动让出一部分权。你也应该嗅出不太乐观的信号了吧。不过不至于那么不乐观。日本媒体说，我们依然让懂王9月来。而不是7月8月。懂王当然急着访华，急着看这次河会亲美派给我们上强度上压力，逼得我们提前邀请他访华。亲美派的确给力，连tiktok都不声不响的达成交易了。

估计是亲美派告诉懂王，让懂王不要再疯疯癫癫的反华了，这关键时刻懂王还反华是拆亲美派的台。懂王还真的听劝，以至于有美国记者提了反华的问题，懂王还帮中国说话。

大概是亲美派劝懂王改变斗争方式。上次苗案干结案，反对派给我们巨大压力的时候，是懂王救了我们。懂王以为我们内部危机一推就会倒台，于是来了不讲理的超高关税。结果懂王的外部危机倒逼了内部的团结，特别是民意的反美支持，反而是间接帮我们渡过危机。亲美派明显急哭了，明显在骂懂王是猪队友，太急不可耐没有战略定力了。懂王一看我们没有一推就倒的程度，他自己犯了战略盲动的错误，亲美派肯定让懂王乖乖的，不要再反华再挑衅了，安心等河会的消息。但目前我们的嘴巴还是没松，还是说是9月来。看河会开完后的结果了。
