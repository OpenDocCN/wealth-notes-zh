
# 【20250318米联储见闻】——岁华销尽客心惊（李嘉诚卖港再起波澜）

李嘉诚卖港口的事情，还在进一步发酵，极有可能成为川普2.0任期与中国的第一场正面博弈。这个事情走到今天这一步，中方并非没有责任。在川普已经公开喊话要拿回巴拿马运河运营权并盯上李嘉诚的港口后，中方没有第一时间做出反应，等到双方都达成交易意向公开后，才后知后觉。只能说大家忙着开两会，结果忽略了外面发生的大事。到今天，外联部的人才率团前往巴拿马做协调，相关部门才开始重视李嘉诚的这笔所谓商业交易。

随着舆情热度升温，今天彭博社引述知情人士信息，内地有关部门已开始评估长和出售海外港口业务的交易是否存在安全漏洞和垄断风险。同日，香港行政长官李家超首度就事件开腔，表达了港府的三个态度：

第一，是社会对事件有很多议论，反映社会对事件的关切，相关关切是值得重视的。言下之意，此事引起公众强烈关注，因其涉及公众利益，不能仅仅视之为一般的商业交易，就不能再用在商言商做挡箭牌。

第二，是特区政府要求外国政府为企业提供公平公正的环境，我们反对在国际经贸关系中使用胁迫或施压手段。言下之意，如果长和是在美方胁迫下不得不交易，港府会对企业提供支持和帮助，港府撑不住，还有中央政府，企业应该对这种胁迫和施压行为说不，不能助长霸权横行。

第三，任何交易必须符合法律和规则的要求，特区会依法依规处理。既然此事涉及公众安全，又有胁迫因素，就不能视之为简单的商业行为，特区政府会依据相关法律对交易做出审核与判断，依法处理，如果不能找到相关合适的法条，因其威胁国家安全，那么在国安法的相关要求下也会对交易做出干涉，有法要处理，没有法立法也要干预处理。

港府的这个表态很明确，将会以港府为主体单位对这笔交易进行干预，港府背后当然是中央政府，这意味着长和这笔交易可能并不会那么顺利地达成，有可能成为中美在川普2.0阶段第一场正式的battle。此消息一出，长和股票应声下跌3%，股价从发布交易后上涨到52.95到今天已经下跌至45.1。

按彭博社发布的消息，北京方面对这家香港企业集团迫于美国压力剥离巴拿马业务不满。包括国家市场监督管理总局在内，多个政府机构已按高层指示，研究这笔交易是否可能危及国家安全或违反反垄断法规定。鉴于老李要出售的港口均不在中国内地和香港境内，目前尚不清楚可以用哪些方法来阻止交易以及中国如果干预交易是否会激化中美关系的矛盾，直接引发冲突。知情人士透露，开展评估并不一定意味之后会采取任何行动。但从港府的表态来看，似乎不可能什么都不做。

在商言商，这是一笔好买卖，但政治经济不分家，尤其是在这个当下从来就没有简单的在商言商。当初老李想卖欧洲电信，英国竞争和市场管理局也对这笔交易进行了调查，认为有可能对英国电信市场的公平竞争构成负面影响，因为这个交易受阻，老李在这笔业务上损失超过1300亿港币。那么今天老李想要全盘交易旗下海外港口，势必也会对中国的贸易安全造成风险，中方介入干预合理合法，但要从依循什么程序去操作这件事，考验国家在国际事务中的软实力。

长和目前只有12%的收入来自中国内地和香港的业务，其余大部分来自欧洲、北美和澳洲。这或许也是李氏家族有持无恐的原因。有外电报道长和及贝莱德的最高管理层均有参与谈判，当中包括长和创办人李嘉诚本人，以及其长子李泽钜。贝莱德则由主席 Larry Fink 参与谈判。财团成员之一、贝莱德旗下的 GIP 行政总裁 Adebayo Ogunles，以及买家之一的 Til 则由身兼 MSC 总裁的大股东家族成员 Diego Aponte 参与。Larry Fink 在进行交易前曾致电予白宫，也说明这根本不是一单简单的港口交易。

但长和发布消息时，联席董事总经理陆法兰强调，交易纯粹是商业考量，无关政治，颇有点此地无银300两的意思。这次交易选在中国举行两会时进行，也颇显出老李对中国政治活动的熟悉程度和心机之深，确实是有心瞒天过海达成这笔交易。实际上有信息显示，中方早先曾多次接触长和提出购买或转让这些港口资产，长和方面一直未有松口，所以这次突然全盘转给美国才会引起中方震怒，这已经不是简单的在商言商，甚至不是钱的问题，而是直接挟洋捅刀，有点报仇的意思了。

两会结束当天，3月13日，大公报立刻发表题为《莫天真勿糊涂》的评论文章，指长和与贝莱德这笔交易是美国运用国家力量，通过胁迫、施压等卑鄙手段，侵吞他国正当权益的霸权行径，是包装成商业行为的强权政治。美国将以此交易为范本，控制世界更多关键港口，并用长臂管辖实施打压手段，让中国船舶无处可靠。面对如此大事大义大节，有关企业当三思，应好好想想有关问题的性质和要害是什么，好好想想自己要站在什么立场、站在哪一边。话说到这个份上，几乎是怼脸开大了。

3月15日，大公报再次发文，质问长和为何如此轻易将这么多重要港口转让给不怀好意的美国势力？表面宣称的商业行为暗中藏着怎样的政治算计？所谓精明的交易有没有考量国家民族利益？在大是大非面前，当事的企业家应怎么取舍，带领企业驶向何方？到这里已经不是不爱国的问题，而是害国卖国的问题了。

李之所以做出这种选择，显然是两害权其轻，在他看来，得罪美国的代价比得罪中国代价高，不可预测性更大，所以宁愿顶着压力和骂名都要来一次长痛不如短痛。中国要想彻底解决这个问题，必须要让李嘉诚们明白，损害中国利益也是要付出代价的，避免这些人在做出相关决策时不考虑中方利益。中国在与美国正面对决的阶段，未来更多的国家和企业都会面临在中美之间站队的问题，如果不能妥善处理这件事，会让越来越多的国家公开倒向美国损害中国利益，所以必须要让李嘉诚为这个交易付出代价，这一拳必须打出去，否则后患无穷，只会成为别国和企业效法的对象，美国尤其是川普倘若这么轻松就吃到甜头，此后只会没完没了，所以这不仅仅是眼下的问题，而是关乎到长期斗争的形势。

买方为首的贝莱德基金是国际上最大的投资基金，旗下管理资产高达11万亿美元，这次与贝莱德一同参与的另两家投资公司，分别是 Global Infrastructure Partners(GIP) 和 Terminal Investment Limited(TiL)。GIP是贝莱德在2024年收购的一家投资公司，在其基金之外独立运营。GIP的领导层都是来自美国和英国的前投资银行资深高管，比如瑞信、高盛和英国的麦格里银行，基本都拥有多年的经验和广泛的人脉关系网络。而TiL则是由几家来自意大利和中东的大船运公司组建的港口投资公司，其领导层和国际航运及港口行业联系广泛，与长期经营港口的和记黄埔更是做了几十年的生意。所以看上去，这更像是一次资本老钱对于新崛起工业巨头的围剿。

从美国的政治光谱来看，贝莱德是过去多年在金融界和商界推动多元共融(DEI)的主要力量，领导层的政治倾向其实更接近民主党。但世界第一的规模让贝莱德这家公司在美国两党内部都拥有极大影响力，在川普第二次当选总统后，贝莱德迅速和 DEI切割，向川普示好。

川普一上来就跟中国在港口业务上直接进入擂台战，也表明了川普要重新与中国争夺贸易主动权的决心，实现美国再工业化的野心。很多人会觉得美国重新工业化是一个不切实际的目标，但在机器人时代，限制美国重工业化的人力问题根本不是障碍，掌握了资源、技术和市场，美国要恢复工业并没有想象中那么复杂，但实现这个目标的第一步必须要让中国的工业大树先枯竭，美国才能获得推动自己工业化的机会和资源优势，如果没有不对等的竞争优势，美国不可能跟中国争夺工业制造的领先地位，所以抢占港口这样核心的工业战略资源本质上就是对中国工业安全发起了攻击。

看清这个事实，就会明白为什么各方的评论越来越严厉，港府也跟着介入，之后可能还会有更高规模更激烈的博弈。关税政策+抢占关键贸易枢纽，实际上就是从贸易终端去砍中国工业这棵大树的根。在川普的视角里，如果只给中国加关税，就会导致中国的商品通过第三国继续流入美国，如果中国的商船没有港口可以靠岸，谁还会买中国的产品，中国的产品卖不出去，工业化还如何支撑？

川普经济政策的终极目的就是美国的重新工业化，但美国重新工业化的一个关键障碍是美元汇率过高，在维护美元霸权的前提下，川普必须另辟蹊径来让美元汇率下降，目前，川普的想法是，美国和世界主要经济体签订协议，要求这些经济体将手中的美国国债兑换成超长期(100年)的不可交易不付利息的特殊债券，然后通过联储局与其他央行之间的货币互换协议来保障国际上的美元流动性。锁定各国央行手里的美元储备货币的地位，同时减少金融市场对美国国债的需求。各国央行不再需要购买并持有普通美国国债，美元的需求下降，美元汇率下降，美国产品竞争力上升，再加上关税保护，美国再工业化就会顺利达成。而这背后最关键的一环就是如果说服各国接受超长期美国国债，实际上超长期的意思就意味着赖账不还了，别人找你借钱，然后告诉你100年以后再还，实际就是在赖账了，这需要美国有非常强势的国家手段，也就是你不同意你立刻死，你才会答应这样的买卖。考虑到川普要用这种方式实现他的政治目标，那么他今天所有一切的荒诞都不算荒诞，只是刚刚开始。

对疯子，没有别的办法，不要考虑讲道理，武装到牙齿，然后让所有牺牲中国利益的群体和个人付出惨重代价，这样的反击，越快越越好，越重越好。
