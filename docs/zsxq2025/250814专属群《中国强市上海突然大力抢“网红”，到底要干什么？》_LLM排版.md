
# 中国强市上海突然大力抢“网红”，到底要干什么？

最近，上海这个中国经济最强城市，同时也是科技、工业和外贸之城，突然发了一个关于内容的新文件，开始强力抢网红。具体的文件名，我先卖个关子，稍后再说。

上海作为经济强市、科技之城，推出内容产业文件抢网红，本身就有点令人惊讶了；而且，从文件来看，对网红的吸引政策，不仅是给补贴，甚至要给网红快速落户和探索职称。根据我的观察，这是中国的核心大城市，首次给网红与科学家、产业人才等通常意义上的“高端人才”以同等对待，网红也开始“高端人才化”。

不仅如此，文件通篇都是类似“金融要素支撑”、“产业配套服务”、“行业高端交流交易平台”、“产业链畅通”这样的词汇。你会发现，这不像是在谈网红经济，而是谈一个战略性新兴产业。所以文件一出台，就引发了巨大关注。

在我看来，这个信号非常有意思，它不是网红经济的简单升级，而是城市竞争上升到新维度之后，要把内容当作核心生产力，用来赋能产业和城市。这个新趋势，很重要。

## 网红“高端人才化”

开头提到的文件，就是7月15日，上海市委宣传部、市委网信办推出的《上海市关于支持互联网优质内容创作的若干举措》，下面简称“沪九条”。

接下来，我带你仔细梳理一下文件的主要内容：

首先最直接的，就是真金白银支持优质内容和内容企业，给奖励、给补贴。比如对于优质的单项内容，市级层面最高给10万元奖励；而且鼓励各方多用技术赋能内容，对开展人工智能等技术，赋能内容创作相关应用场景建设的主体，给予项目总投资最高30%、不超过2000万元的支持。

但直接发钱，杠杆效应还是不够广泛，“沪九条”干脆把对科技产业常用的金融政策，用到了内容产业。比如“推进设立相关专项基金，重点支持互联网优质内容产业链企业发展壮大，服务创作者创新创业”；还要“为创作者和产业链企业设立贷款绿色通道，提供政策性优惠担保”，并提供“贴息支持”，还要开展“版权质押等融资方式创新”。你看，这就对内容产业跟科技产业一样，赋予了金融优先的地位。

但是到这里还不够，上海还要对“优秀创作者予以落户支持”，加大人才额度分配支持力度，把优秀的互联网内容创作者纳入人才计划，甚至探索要把他们“纳入职称评审范围”。你看，这就跟鼓励科技人才一样。

此外，上海还要“推动全市地标景区、文博场馆、特色街区、时尚空间以及重大节展赛会等场景资源向创作者开放，提供创作便利”。而且要“鼓励国有企业围绕产业发展、金融服务、民生保障、城市更新等领域，提供拍摄场景和资源支持”。在我看来，这背后是把大量的公共资源和国企资源，优先导向互联网内容产业。

而所有的这一切，背后就是“产业”两个字。“沪九条”开头就明确提到，要“支持黄浦区、杨浦区建设在全球范围具有影响力、引领力的互联网优质内容创作集聚区”。接下来重点来了，要“推动形成创作主体云集、产业链条畅通、政策体系健全、配套服务完善、优质内容涌现的友好型生态”。“鼓励各区结合实际打造互联网优质内容创作产业链优势环节，优化全市产业布局”。核心大城市对互联网内容产业的这种表态，这种定位，确实非常罕见。

我仔细梳理后你会发现，“沪九条”之所以特别，不是因为激励力度大，而是因为这背后是一套极其标准的扶持战略产业的逻辑，像打造集成电路、人工智能一样，把网红经济打造成全新的“内容生产力”，我认为这是要让上海在科技之城、工业之城之外，变成“内容之城”。这也是我想请你关注到的重点。

## 背后的产业竞争意图

说到“内容生产力”，它到底是什么意思呢？为什么现在它的重要性大幅度突显出来了？

要理解“内容生产力”，就需要回顾一下基于内容生产的网红经济发展史：初代网红，主要靠礼物打赏、流量分成来变现，只做到自己火；之后，发展出广告和直播带货，开始和实体经济结合，但主要集中在低单价产品中，比如日用品、平价美妆等，相对一个城市来说，依然规模有限；但到了“内容生产力”阶段，内容被寄予厚望，要发挥自己的杠杆效应，要能带火高溢价产品，比如新能源汽车、高科技产品，甚至要能带火整座城市，成为真正的“生产力”。

那么，内容真的可以承担起作为生产力的重任吗？

接下来，我就带你做两层推导，分别看内容对于产业和城市的赋能潜力。我们先来看看内容带火高溢价产品，或者说赋能产业的能力，我们就拿新能源汽车为例，这背后靠的是两条链路：

第一条链路，从共情到共识。比如小米创始人雷军，在十周年演讲中分享自己艰苦的奋斗史，这就是共情。而对企业家的共情，最终转化为对小米汽车的认可，也就是转化为对品牌的共识。小米SU7的火爆，就是例证。

第二条链路，从共识再到信任。小米今年6月开新车发布会时，不少人激情下单。没试车、不看参数，就能做大宗消费的决策，靠的就是前期积累的品牌共识，转化成了产品信任。所以，作为“后来者”的小米汽车，硬是能挤上原本就拥挤的牌桌，我认为背后就是内容杠杆带来的高赋能。

所以在我看来，“沪九条”真正的政策意图，不是吸引极少数的头部网红，而是把“内容生产力”当作杠杆，来赋能整个城市的产业竞争，这种赋能包括两个方面：

一方面，把原本分散的内容产业，集成成高附加值的内容产业集群。比如作为上海重点打造的互联网优质内容创作集聚区，杨浦区已经成立了“大创智国际生态创新联盟”。央视新闻的报道说，目前这个区域已经形成了以“文化+科技”为特色的产业集群，通过两者融合，既催生了新的文化业态、延伸了文化产业链，又集聚了大量创新人才。下一步将重点培育泛娱乐、大设计、新消费三大文化科技产业集群。

除了内容产业的自我发展，另一方面，“内容生产力”显然还可以赋能其他产业。比如，成为其它产业竞争的新突破口。当前企业内卷的关键，是产品高度同质化，更深层原因是生产能力同质化，但是现在产品想要实现大幅度差异化，是越来越难了。而除了产品之外，更能差异化的，显然是大家对品牌的不同认知。雷军和小米汽车，让市场看到了内容的巨大价值。所以，你能看到，现在各行各业都在纷纷做创始人IP。创始人IP背后，其实是整个企业内容生产力的较量，这是企业在产品较量之外的另一种破局之道。

## 升级城市话语权

但还不仅如此，我们继续推导，企业能靠着“内容生产力”战胜对手，那城市可以吗？我认为是可以的。一个城市的内容生产力，显然可以帮助升级城市话语权。接下来，我就和你讲讲我的理解：

之前，很多在科技和工业等传统领域没有优势的二三线甚至四五线城市，就在靠内容生产力，实现弯道超车，获得了远超自身经济量级的城市声誉度，比如淄博和哈尔滨。

但一般的城市，没有强劲的产业支撑，想留住人还是很难的，所以这些网红城市，也只能是拿流量红利。但一线、新一线大城市不一样。它们能让网红经济，或者说内容对城市的杠杆，极度放大，产生切实的产业甚至财政效果。

这里面最典型的就是杭州和成都，这两座网红之城，成功让自己在实打实的产业能量之外，加上了内容能量，让“创新”“年轻”“活力”“幸福”“有未来”等词语，成为自己的城市标签，进而可能成为大家尤其是年轻人的共识。

在这些城市，“内容生产力”不只是赋能文旅等少数产业，而是赋能整个城市的品牌。换句话说，这些城市的“好感度”，会变成城市的“总体加分项”。而有了这个“加分项”，就有机会成为更多年轻人的“优先选择”。

这对一个城市的红利太明显了，正因为这种共识，外地企业也更愿意在这里投资发展，城市的房价也可能更高，这个城市的土地财政也能更持续，财政收入也更健康，这个城市就越来越正循环，因为大家都相信这些城市是有未来、有潜力的。其实，杭州这几年的新增人口，和成都在逆势中稳健的房价，就说明了问题。

从这次“沪九条”的表述来看，我认为，上海显然意识到了这一点。我刚才是提到了，上海要“推动全市地标景区、文博场馆、特色街区、时尚空间以及重大节展赛会等场景资源向创作者开放”。这既是对创作者的支持，更是通过这个方式，把上海的吸引力、时尚力，传播出去，提升城市的整体溢价。

而且上海还要“鼓励国有企业围绕产业发展、金融服务、民生保障、城市更新等领域，提供拍摄场景和资源支持”。这也是要把城市的创新力、服务力、保障力，传递出去，这同样是要提升城市的整体溢价。

你看，这次上海，想得真的是明明白白。总的来说，我认为他们最终的政策目标，是把“内容生产力”，和产业、城市做深度融合，把内容这个“杠杆”发挥到极致。

所以你听到这里，趋势就比较明确了，未来的城市和企业的竞争力，不仅在于能造出优质的产品，同样重要的是，怎么能让更多人了解你的产品，了解你的企业，了解你的城市，形成共情、产生共识，并最终建立起信任和对你未来的向往。

说白了，不管是“科技之城”还是“创新之城”，都得喊出去，只有让更多人相信这座城市有未来，这样城市才真的有更大潜力的未来。

未来城市之间，在内容生产力上的竞争将越来越激烈，而对那些缺少产业吸引力支撑，也没有特色资源和内容生产力的城市来说，未来冲击可能会更大。总之，这真的是一个大时代，作为一个观察和研究者，我喜欢这样的大时代，也希望带你持续追踪。
