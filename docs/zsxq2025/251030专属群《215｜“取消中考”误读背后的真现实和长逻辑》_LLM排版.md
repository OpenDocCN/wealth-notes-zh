
# “取消中考”误读背后的真现实和长逻辑

## 贯通式培养的推进方式

贯通式培养是未来教育的大趋势，但试点周期会很长，短期内难以快速推广。以上海发布的《普通高中高质量发展实施方案》为例，除了提到“试点长学制贯通培养模式”，后面还提到“支持优质高中向下延伸、优质初中向上延伸办学，推动建设一批高质量的完中和含高中的一贯制学校”。这里的“完中”指的是同时包含初中和高中的学校，贯通式培养与优质学校绑定在一起。参与成都试点改革的学校大多是基础较好的学校，包括一些名校，因此成都的试点也可以说是从优质学校开始的。贯通式培养从来不是单独存在的，而是优质教育扩大化的新模板，与之前的“集团化办学”有着类似的政策意图。继集团化办学和背后的教育资源横向一体化后，以“贯通式培养”为核心的教育资源纵向一体化开始成为重点。“横向”是指资源的空间整合，把不同区域、不同学校的优质资源连接起来；“纵向”是指学段的时间贯通，把小学、初中和高中培养体系打通，这两者放在一起，我总结成“教育资源的一体化”。这些信号背后的大趋势是，教育资源正在迎来新一轮的再分配。而掌握主动权的，是在上一轮发展中脱颖而出的优质学校，这一点在集团化办学、贯通制试点上，都得到了验证。

## 贯通式培养主要解决的问题

贯通式培养主要解决过去选拔模式带来的弊端，推动教育的模式，从以“标准化、频繁选拔”为主，转向以“长周期培养”为主。以“标准化、频繁选拔为主”的旧模式，就是通过频繁的考试、竞赛、分层录取等方式，比如中考、高考，甚至学校内部的各种考试等，实现快速分层分级，同时把更多资源向“胜出者”倾斜。这套模式的核心是快速淘汰机制，最终形成全社会参与、高强度投入的“教育锦标赛”。这套模式在短期内确实有效，具体表现为两个好处：一来，国家能快速匹配资源投入，向优胜的学生和学校投入更多资源，大概率不会错；二来，对学生和他的家庭来说，目标明确、任务清晰，能激发出强大的竞争动力。不过缺点也很明确，因为综合性考试、标准化考试，通常只能选拔出综合型人才、标准化人才，所以对理工科的偏才、专才并不友好；而考试的优胜者，也不等同于国家急需的人才，尤其是在当前科技发展、大国博弈加剧，以及对高水平科技人才需求加速的大背景下。

## 应对思路

对家长们来说，面对这个短期变化不快、但长期必然到来的大趋势，要做好长期打算。我知道这里面平衡不好取舍，特别对于低年级的孩子，因为长时间的转型期，其实是最难受的。目前贯通式培养还只是很小范围内初步试点，还有待观察，因此我先给出三点方向上的建议，供你参考：

第一，尊重现实。孩子的学校教育，跟着学校和国家改革节奏走，该应对短期选拔的，就应对短期选拔。

第二，内心要重新定位培养目标。家长们要意识到，不是“一切为了应试”的长周期培养，是个必然到来的过程，未来一个家庭的教育目标，不再是培养标准的工程师或者白领，而是培养适应未来社会挑战的创新型或者高精度人才。

第三，具体到教育资源的配置方面，要打提前量。不要把资源单纯都投入到“课外辅导+应试培训”上，也不要只追求“分数”和“学历”方面的暂时取胜，因为学历在贬值、阶段性考试的实际重要性也在降低。所以家长们要拨出一部分家庭资源，投入到“高阶素养”上，比如跨学科整合知识的能力，与AI协同的能力、不断解决问题的创新能力等等，支持孩子参与更多实践性学习、项目式学习、创新性训练，并鼓励他们在兴趣领域内，进行深入探索。

## 总结

贯通式培养的本质，是更侧重能力提升的长周期培养模式，用来减少过度的教育内卷、适应新一轮的产业转型和人口新周期。贯通培养不只出现在从小学到高中的基础教育领域，在高等教育领域，“一贯制”的培养趋势更快、更明显。这里的“一贯制培养”，你简单理解就是本硕博连读。新型研究型大学的“3+2+3”的分段培养模式，就是本科3年、硕士2年、博士3年，中间不考研。这几年，大学硕博连读，甚至本硕博连读的贯通培养大趋势越来越确定。相比基础教育的贯通培养，高等教育的贯通培养会加速到来，因为它更直接服务于国家需求。而基础教育的贯通培养，我认为这是个日益清晰的趋势，但是短期内一定不会快，因为改变需要的配套条件比较高；但是中长期的 影响，一定超过大多数人的预想，尤其在AI对教育的冲击之下，标准化选拔，变得越来越没有意义，因为你再标准也标准不过AI。
