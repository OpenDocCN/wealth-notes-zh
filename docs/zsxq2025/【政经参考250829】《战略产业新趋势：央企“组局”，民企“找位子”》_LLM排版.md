
# 战略产业新趋势：央企“组局”，民企“找位子”

## 央企产业新布局

中国的央企并非都归国务院国资委管，财政部也管辖一些特殊央企，如中国国家铁路集团、中国邮政集团、中国烟草总公司等，以及五大国有银行（中国工商银行、中国农业银行、中国银行、中国建设银行、交通银行）和中国人寿、中国光大、中国中信等金融机构。而直接归国务院国资委管的一类，通常是更偏市场化运作的企业。

国务院国资委在2024年明确定调，央企要向“关系国家安全、国民经济命脉的重要行业和关键领域”“关系国计民生的公共服务、应急能力、公益性领域等”和“前瞻性战略性新兴产业”这三个方向集中。2025年，目标更具具体性，要“在产业焕新、科技创新、设备更新三大领域加大投资力度”。

央企抢滩战略性新兴产业的信号明确且持续加强。根据观察，央企的产业新布局大致可以分为两种形式：

**第一种**：直接组建新央企，或对央企下属公司进行重组。例如，中国资源循环集团是为了抢滩资源循环利用产业；中国数联物流信息有限公司是首家“数据科技央企”，抢滩新一代信息技术产业；中国长安汽车集团则抢滩新能源汽车产业和抢占新一代信息技术风口。

**第二种**：更多央企自己出手，直接布局战略性新兴产业，抢占制高点。例如，中广核集团、中核集团和国家电投集团的第三代核电项目，总投资规模超过2000亿元；招商局集团推进“科创产业板块”重构，投资118亿入股医药领域龙头企业人福医药，与合肥市政府共建生物基复合材料生产研发基地等。

央企投资战略性新兴产业成为主流，实质是把更多资源集中投入到最可能形成国家竞争力的赛道。央企的产业身份正在改变，从过去的托底“压舱石”转变为战略产业的样板，带动更多产业链的发展。国企负责战略攻关，而民企则聚焦更市场化的一线创新。

## 央企负责“组局”，民企自己“找位子”

央企产业身份改变的原因至少有两个：

**首先**，在新一轮大国博弈下，国家安全的边界被放大。科技安全成为“重头戏”，甚至可能影响传统领域的国土安全，如人工智能对国防军工的影响。这也意味着，保障科技安全的重要性提高，战略产业成为新的安全底线，央企只能选择提前布局。

**其次**，经济环境出现周期性变化，整体投资意愿和预期下降，需要央企进行逆周期投资。在民营企业“不愿意”多投资、地方政府“没能力”多投资，但国家战略需要多投资的领域中，央企起到先行和示范作用，反向加大投资力度。

央企改变产业身份的趋势非常确定，因为这是时局造就的趋势和国家对产业突围的态度的体现。国家和央企主动承担高风险的行动，向民企和地方政府传递出更确定的信号，影响它们的产业站位和投资决策。

对于想要跟上这个趋势的民企来说，要尽早考虑如何成为合作者和下游企业。央企主导的创新联合体是核心机制，由央企牵头，联合上下游企业、高校、科研院所等，共同打造“产学研用一体化”的协同平台。全国已有24个央企主导的创新联合体，涉及1000多个项目、超过800家科研院所。

民企可以以三种方式参与创新联合体：提供技术、提供应用场景、成为资本合作方。在“央企+民企”共生的新型产业集群中，央企负责“组局”，民企自己“找位子”，看是切入技术链、市场链还是资金链。

## 央企布局对城市的影响

央企积极布局战略性新兴产业，不仅会密切与民企的协同，在城市崛起中，央企的加入同样能产生推动作用。以中国长安汽车集团落户重庆为例，其他项目合作也类似。

中国长安汽车集团落户重庆，自然会带来产业集聚和人才集聚。因为长安未来要布局固态电池、飞行汽车、人形机器人，这些新项目就会带来新投资、新企业，重庆可以完成从产业链到产业生态的升级。而只要产业升级，就能带来新的人才。

但央企和民企最大的差别是产业投资的高确定信号，带来的城市自信的提升。因为央企布局背后，是来自国家意志的巨大确定性。这也是为什么重庆媒体要反复强调，中国长安集团是“首家总部落户重庆的一级央企”。央企总部落户在北京以外的城市，本来就是“新鲜事”。
