
# 骑虎难下

四年前有篇文章叫《A119: 伊朗的破局：霸中东而救自身》，这篇里面有，外面也还挂着。当时做了一个判断，伊朗没有筹码与域外大国进行交换。它必须创造筹码，这个创造筹码的过程就是以进为退，四两拨千斤。但是，创造筹码的核心是为了进行价值交换!因为他们非常清楚，自从一战之后，世界性的大国不会允许这个地方被整合起来。任何人当老大都不会允许!这个判断在4年后的今天基本上验证了，它的那些筹码已经换取了加入金砖、获得苏式战机技术、获得超音速导弹、获得兔子的4000亿美元的投资等一系列他们急需的东西。最重要的是加入金砖——等金砖结算体系上线之后，全面封锁的经济就等于彻底打开了一个缺口。与之同时，它现在试图跟鹰酱也做一个价值交换。这个交换很隐蔽并且不在军事方面，最后再详细阐述。

在这个区域存在五个潜在的区域力量，这些力量具有整合整个地区的潜在能力。他们分别是：埃及、伊朗、沙特、伊拉克、土耳其。这些国家除了沙特之外都拥有过建立和维系帝国统治的经验，而且这些帝国都曾经覆盖整个区域的大部分地区。唯一的一个例外就是沙特!沙特是一个崭新的国家。正因为如此，它是所有域外大国优先选择的合作伙伴。我在南区的训练基地见过沙特的士兵，但从未见过伊朗的士兵。兔子愿意扶持沙特并非因为它有钱，而是因为它最无能——鹰酱也是这样考虑的!

纵观这些年的历史，你会发现，海湾战争打垮了正在崛起的傻大木治下的伊拉克，那时候的伊拉克真的有点吸引人。随后的两伊战争消耗了伊朗，全面封锁让伊朗变得更加的虚弱。阿拉伯之春不仅搞乱了埃及，也极大的削弱了埃及，再过十年都喘不过气来。土耳其深陷美元债务危急，要不是俄乌冲突让它变得举足轻重两头吃，它早就崩溃了，它的汇率在鹰酱第一轮加息的时候就暴跌90%!这些年为了发展借了太多美元债务，我们是通过土地集资向内部借债的方式筹集发展资金，土耳其则是向外面筹集资金。其结果就是被人家捏着经济命脉。你别看土耳其咋咋呼呼的，内在的虚弱让它也只能咋咋呼呼吸引眼球。

沙特是唯一两边都争取的对象。沙特可以通过资助的形式到处培养武装团伙配合鹰酱的战略把整个地区搞得乌烟瘴气。但它自己的实力摆在那里，没有整合的能力：它虽然占据两座宗教圣地，但约旦是正儿八经的圣裔家族。沙特本身占据的是约旦国家家族的领地，以前这个地方叫汉志。所以，它就更加没有号召力了。抛开宗教和钱的影响力，沙特还有什么能让整个区域引以为傲和感到服气的东西?似乎没有!埃及和伊拉克都曾经是历史上阿拉伯的主要王朝，伊朗的波斯帝国和土耳其的奥斯曼帝国都有着现成的号召力。唯独沙特什么都没有，而且自己的战斗力还非常拉跨。所以，它就只能抱大腿，不抱这个就抱那个。

为什么不能让这个拥有10亿人的区域整合成一个整体呢?因为它地处亚欧非的关键地带，不管你是依靠陆权当霸主，还是依赖海权当霸主，它都卡在关键的咽喉位置。如果它的体量够大，号召力和影响力足够，它会造成什么样的效果，上一次已经演示过一会了!切断了亚欧大陆的商贸通道，逼迫欧洲去汪洋大海上找生路。自从日不落帝国这种全球视野的老大诞生之后，后续不管是谁当家，都不会再允许再来一次了!因为谁控制亚欧大陆的中心地带谁就能控制整个亚欧大陆，继而辐射整个世界。你猜那地方为什么叫中东?因为它就处于中央地带。当二战后苏联的势力不可避免要渗入并控制那个区域时，土耳其被拉进了北约，以色列被扶持了起来，沙特成了重点投资对象。

总而言之，拥有全球视野的玩家，不管彼此如何争斗，都存在一个心照不宣的共识：不能让这个地方存在一个能把大家整合起来的潜在势力。伊拉克侵略科威特之所以要被揍趴下是因为一旦默认就会开一个坏头。利比亚和埃及以及叙利亚试图组建阿拉伯联邦也一定要拆散。让他们相互纷争不断，彼此消耗，动荡不安符合所有真正玩家的利益。为什么以色列能够长期存在?并不是因为以色列自己多么厉害。我们看到了以色列的国防军多么拉跨，在没有外力的持续支援下，它也打不赢那些中东战争。但是，以色列非常清楚：不管谁当老大都需要自己这根搅屎棍，只要自己愿意干脏活，为真正的玩家办那些说不出口的事，它就能被允许一直存在。这恰恰是它的求存之道，它要让全世界都恨犹太人，这样犹太人还会因为害怕全世界的仇视而主动向以色列聚集，继而让全世界的犹太人为了获得安全，不得不保住以色列作为一个犹太人国家的存在。它的存在又满足了域外各位大哥的诉求。

谁都知道以色列必须存在，只有让以色列存在，这个棋局才能持续下下去。它符合所有大哥的根本利益。知道越南人为什么这么恨兔子么?因为当年他们本来是可以整合整个中南半岛继而整合整个南亚的。然而，当他们迈出第一步的时候，就被兔子强行打断了。于是，它就再也没有机会了。在兔子的各种加持下，老挝和柬埔寨都有了与之分庭抗礼的底气。越到后面它越缺乏整合的机会，甚至保持自己的独立性都会越来越难!

我们为什么恨日本?因为它们的残忍么?普通老百姓可能是因为这个，但社会精英不是因为这个。假如我们能从1927年开始建设，完成工业化的时候，再利用二战时苏联的虚弱，未必不能整合外东北，外伊利河谷、外蒙古，以及琉球和缅甸江心坡等地，继而在中南半岛和南海诸国还未立国时完成整合。当然了，这种历史机遇期一旦被阻断就再也不会有机会了。一旦没有了机会，命运就会像着往前不同的轨迹前进….不管是一个国家、一座城市，还是一个人都是这样!山西大同有个著名的耿市长，他借了 许多债按照自己的环保和旅游理念对大同进行了彻底改造。当时许多人骂他到处拆，他在接受采访的时候痛心疾首的说：大同只有这一次机会，如果赶不上就彻底没机会了!今天大家反应过来 了，明白了他说的只有这一次机会，错过了就再也没机会了具体是指什么。所以，今天的大同人都非常感激他。然而，在当时大家是不理解的。

之前伊朗通过创建什叶之弧这样的武装力量来创造筹码，通过这些筹码来与域外大哥做交易拯救自身的处境。事实证明这个策略不仅是对的，而且经济实惠。但是，它的后遗症也 是显而易见的。今天的伊朗必须要妥善处理这个后遗症问题。这与阿富汗的阿塔的处境一样：我可以进步，也可以反恐。但你们要给我一些援助，让我能够稳定发展。否则，我不会全力以赴与他们切割并协助清理它们。没有换到筹码还把自己的依仗搞没了。

波斯的困境就在于：域外的大哥只肯给这么多，目前能给的无法解决自己的核心问题。然而，因为它太强了，一旦给的太多，就有了整合整个区域的能力。这是哪个大哥都无法接受的!大家都喜欢那个弱鸡沙特，因为给它再多也掀不起浪花来。这就如同以前大毛之所以把最先进的武器卖给印度不卖给我们，因为我们的仿造能力太强，工业配套太完善，自己研究的能力也太强。大毛知道三哥菜，可以放心大胆的把最先进的东西高价卖给它。波斯知道如果再进一步，兔子也会选沙特而不是选自己。波斯也知道就算再进一步，大毛也不会把核技术给自己。而且，随着俄乌战争大毛开始占据优势，波斯也没办法为其创造更大的价值——没有筹码就交换不到东西。大毛倒是急切的想拉波斯全面下水，问题就在于：不管不顾的下水，出事了谁负责?没有一个大哥愿意做出明确承诺!所以，波斯内心最大的渴望其实是鹰酱。只不过这个弯一下转不过来!波斯需要隐蔽的纳一个投名状。这个投名状就是在鹰酱不可避免要降息的时候，把油价和大宗商品的价格搞起来。目前这些东西还是用美元计价的。它们起来了，美元就会坚挺。美元坚挺兔币的汇率就不会一直走高，能放出来的水就有限。兔子现在需要放水创造预期，也需要更便宜的生产资料和油气成本。因为兔子是最大的生产者，生产成本的大幅提高不仅可以挤压利润预期，继而影响社会经济增长预期。又反过来决定外部资金抢购其国债的热情程度——因为经济增长预期与长期国债的收益是同步的。如果接下来的两个月鹰酱的就业数据和相关经济数据显示好转，鹰酱甚至可以在大家充满预期等待继续降息的时候突然取消。

波斯并不想扩大冲突规模，它只是想做一个交易。大毛是希望它深陷其中的，油气和大宗商品的上涨不仅有利于鹰酱，也有利于大毛。昨天兔子家的老王跑去莫斯科的时候，普京以前都是提前在等，这一次迟到了，不仅摆谱，还让媒体把老王在那无聊等待他的画面视频发了出来。兔子此时此刻并不希望冲突扩大规模，这三个月对我们很重要!
