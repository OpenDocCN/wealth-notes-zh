
# 这份重磅文件，直面民营企业家的“安全感”问题

## 对民营企业自身的要求更规范

这次最高法文件的第三部分，是“依法引导守法规范经营，促推民营经济健康发展”。这部分主要面向两方面的问题展开。

一方面，是在过去的发展中，民营企业自身也出现了不少问题，比如一些民营企业，特别是一些中小微企业和家族企业，普遍存在内部监督机制不健全、控股股东或实际控制人权力滥用，以及中小股东权益易受侵害等痛点。

另一方面，市场中存在的串通投标、逃废债务、欺诈失信、侵犯知识产权、无序竞争，以及一些消费领域的乱象等问题，造成“劣币驱逐良币”的负面影响。

所以，这次的条款对于民营企业的内部治理，也提出了针对性意见：

第一，依法惩治民营企业内部腐败等犯罪行为。最近你可能关注到了，很多互联网大厂的内部反腐，越来越严格了，这是个大趋势。文件要求，要适时发布指导性案例、典型案例，这方面会继续加强。另外，文件特别提出要“依法惩治、有力遏制串通投标犯罪”，这是个重点，你要注意。

第二，要推动民营企业完善内部治理。文件提出，对民营企业股东之间的争议，要积极引导通过股权转让、减资分立等途径，实质性化解经营僵局。简单理解就是法院不能简单就案办案，要考虑企业实际经营，不能让企业内部就这么僵持着打下去。

另外，要依法追究控股股东、实际控制人实施关联交易“掏空”企业、非经营性占用企业资金、违规担保向企业转嫁风险等行为的法律责任，切实维护中小股东权益。这意味着，以后对民营企业大股东和实控人的司法尺度将越来越严格，我分析这一条对上市公司的约束性将会比较大，避免大股东对小股东利益的漠视，这也是繁荣股市的一个关键基础。

第三，文件提出，要完善“非正常经营企业清算、注销等市场退出机制”，打击这里面企业逃废债务等违法失信行为。另外，要惩治企业的金融诈骗、危害税收征管、侵犯知识产权、扰乱市场秩序等犯罪，更要“严惩破坏生态环境行为，防止以污染为代价发展”。这几个部分，很多民营企业要特别注意，这都是最高法关注打击的重点。

第四，针对一些消费领域的乱象，文件提出“依法审理涉消费纠纷案件”。这个部分很多要求很有意思，比如要“落实食品药品惩罚性赔偿制度”，如果售卖有问题的食品药品，那么将面临惩罚性赔偿。

但同时，又要依法规制“知假买假”，惩治违法索赔行为，这其实是 对一些不良打假人的规范。此外，还要求打击经营者“卷款跑路”“霸王条款”、收款不退等常见的日常消费陷阱。以后这一部分的司法尺度，也会越来越严格。

## 重点是解决民营企业的安全感

在谈完了民营企业企业的内部治理后，《指导意见》的第四部分，聚焦司法体系对民营经济的保护升级。这一部分非常关键，它的核心在于通过刑事司法、财产处置、冤错纠正和人格保护四大维度的制度设计，来破解民营经济中长期存在的“安全感缺失”“产权焦虑”“维权困境”等法治痛点。

第一，是要加强产权司法保护。要“依法严厉惩处敲诈勒索、强迫交易、聚众扰乱企业生产经营秩序等犯罪”。这些犯罪，往往具有黑社会性质。而且文件提出要“坚持全链条打击”，就是不仅要抓前面的手下，还要抓后面的主谋和受益人，要打掉整个犯罪链条。

另外，要“研究出台审理民刑交叉案件司法解释”，民营企业的很多案子，可能既涉及民事纠纷比如“合同违约”，又嫌刑事犯罪比如“合同诈骗”，这次最高法强调，要“严格区分经济纠纷和经济犯罪界限，坚决防止将经济纠纷作为经济犯罪处理”。

比如不能因为一些民营企业合同没履行好暂时还不上钱，就简单化给相关企业家认定为“诈骗”，进而限制人身自由。现实中，尤其跟一些地方政府的纠纷，更容易出现类似的不规范现象，这就要求地方上判罚规则要清晰明确，不能成为民营企业家头上的炸弹。

我分析下来，这既是在给地方警告，未来也要通过新的司法解释，统一像“合同诈骗罪”这种界限比较模糊的罪名的认定和裁判标准，遏制地方司法滥用情况，为民营企业和企业家划出安全区域。

第二点，是规范处置涉案财产。这一条特别重要，要解决的，是“查扣乱象”与“牵连无辜”的实践难题。之前在一些案件中，涉案财产处置不规范是一个突出问题，非常影响民营企业家的信心。一些地方办案机关，存在超范围、超标的查封、冻结企业财产，混淆企业财产与股东个人财产、违法所得与合法财产等不规范问题。

有时候，一家企业可能因为一个比较小的办案金额，使得整个资金账户都被查封，这是典型的“办一个案子，垮一个企业”的负面效应。

具体文件怎么规范对涉案财产的处置呢？

首先，要求要严格按照最高法定的《关于规范刑事裁判涉财产部分执行工作的意见》，不能乱来。

其次，要尽力运用比如刑事手段，把被骗被侵占的财产追回来，还给受害的民营企业和企业家，保护他们的合法财产权。但同时，也必须严格区分，哪些是犯罪得来的非法财产，哪些只是和案子有关需要查证的财产，哪些是当事人的合法财产。包括要区分开，民营企业的公司财产和企业家 的个人财产。就是前面说的，不能为老板个人涉嫌犯罪，就把他公司的合法财产和账户，都给冻结或查封了，影响公司正常经营。

此外，要把涉案当事人的财产，和无关的第三方案外人的财产分开。我举个例子，比如不能把犯罪嫌疑人 与案件无关的家人名下的财产，甚至他们合法卖给不知情人的车子、房子，简单地当作嫌疑人的财产给处理了，特别是第三方是在不知情下善意合法取得的。总之，要避免“一人涉罪，全家牵连”“一企涉诉，全部查封”的粗暴做法。在这一方面，要让民营企业家的信心务必恢复。

依法纠正涉企冤错案件

《指导意见》第四部分要解决的第三大问题是冤错案。文件提出，要“依法纠正涉企冤错案件”。

冤错案对企业和企业家来说是巨大打击，本应及时纠正。但在过去的实践中，涉企冤错案往往涉及复杂的历史因素和地方利益，传统再审程序，存在启动难、审理慢、效果弱等问题，并且纠正已经生效的涉企冤错案，更是会导致部分办案人员受到问责，所以一些地方对“澄清冤错案”的动力不强，一些案件长期在程序中“空转”，得不到实质性解决，严重损害司法公正和企业权益。

所以这次文件要构建一套更高效的纠错工作机制：

文件提出，要“对限制人身自由强制措施是否符合法定权限、条件和程序实施严格审查”。就是要严格查一下，当初抓人、逮捕的时候，手续是不是齐全、理由是不是充分、过程是不是合法，不能随便限制企业家人身自由。

同时要“及时发布再审纠错典型案例，坚决做到有错必纠、依法纠错。健全甄别纠正涉企冤错案工作机制，建立健全工作台账，积极开展专项排查。”就是要把一些已经纠正过来的典型案子，社会关注度高影响大的案子，及时公布出来，这既是警示，也是参考。还要把冤错案件的情况登记好，并且积极进行专项评判和检查，督促地方加快速度。

另外，文件要求，“加大上级法院提审和直接审理力度，完善指令异地法院再审制度，防止程序空转”。而对一些上级指令后地方反应很慢、甚至石沉大海的情况，要求“加强对指令再审、发回重审案件的跟踪指导监督”，这是要避免地方保护主义对司法公正的干扰。

《指导意见》第四部分要解决的第四个，也是最后一个问题，是“企业家人格权益易受侵害”问题。

文件提出，要“依法保护民营经济组织及其经营者人格权益。依法严惩网暴伤企违法行为，依法打击侵犯民营经济组织名誉权和经营者人格尊严、隐私权等犯罪。”这里用了“犯罪”这个词，就是强调用《刑法》作为武器了。

另外，还要充分发挥“人格权侵害禁令”的作用。我简单解释下，对于侵害企业家人格权益的网络谣言官司，打起来时间通常比较长，但谣言传播快，等打完官司，相关企业和企业家可能受到的损害非常大了，所以法律提供了一个类似临时“紧急刹车”的机制，相关企业和企业家可以赶紧向法院申请一道“禁令”，要求造谣者立刻停止传播谣言，防止伤害扩大。

今天咱们谈的这两部分内容，我觉得很重要，因为这标志着中国民营经济治理，进入了一个全新的历史阶段：它既不是简单的政策松绑，也不是单向的利益让渡，而是一场深层次的法治环境的优化，本质上是在降低市场运行的制度成本 ——正如好的道路能提高交通效率，明晰的法治，则能显著提升资源配置效率。这对于正处于转型升级关键期的中国民营经济来说，价值远比减税降费等政策更为深远。
