
# 子明私享汇7.4：TikTok的潜在买家

几经波折，TikTok即将迎来自己的新买主。2025年6月29日，特朗普在接受福克斯新闻采访时称，他已经为字节跳动旗下的TikTok美国业务找到收购者，并形容其为“一群非常富有的人”。特朗普称，他会在约两周内披露买家身份和交易细节，并承诺有关交易仍需中国政府的批准，也相信中方最终会同意。

30日晚间，甲骨文盘前股价拉升涨超7%，报225美元。甲骨文首席执行官称，公司在2026财年开局强劲：“我们已签署多份大型云服务协议，其中一份预计从2028财年起每年将贡献超过300亿美元的收入。”

懂王关于TikTok交易的预告，与甲骨文股价的大幅上涨，其中是否存在某种关联?

懂王口中“非常富有”的神秘买家又究竟是何来头?要说是甲骨文，倒也说得通。他们的董事长拉里·埃里森一直是懂王的坚定支持者，之前还把自己的私人高尔夫球场提供给懂王举办竞选活动，募集竞选资金，实实在在地支持了他八年。真的把TikTok给甲骨文的话，懂王也算是没忘了老朋友。

我记得上次的TikTok收购方案，甲骨文就已经和黑石集团、创投基金Andreessen Horowitz组成财团，几乎就能完成收购了。这一次估计还是和这两个公司合作完成收购，也算是完成自己上回没完成了的工作。只是不知道，在懂王就剩一个任期的情况下，这个交易能不能顺利兑现。

话说回来，甲骨文嘴是真的严。不仅是TikTok收购，关于那个“将贡献超过300亿美元年收入”的云服务协议的具体买家，他们也是缄口不言。我猜可能和“星际之门”项目有关。

在经过多轮戏剧化的波折后，TikTok终于要迎来自己在北美业务的新买家了。甲骨文领导的财团预计会凭借自身强大的实力，以及和特朗普集团的亲密关系，极有可能拿下此次机会，然而，特朗普的任期不如预期的那样长，这笔交易最终能顺利落地吗?
