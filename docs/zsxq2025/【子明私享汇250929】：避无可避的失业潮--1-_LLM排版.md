
# 避无可避的失业潮

"毕业即失业",在今年从调侃变成现实。2025年8月，国家统计局公布了本年度的就业相关数据；全国城镇不包含在校生的16-24岁劳动力失业率升至18.9%,创下2023年12月统计调整以来的新高。而这组数据的背后，是创纪录的1222万高校毕业生与同比缩减22%的岗位需求之间的系统性失衡。

换句话说，几乎每五名非在校青年中，就有一个人处于失业状态。这究竟是经济新周期中的"阵痛",还是大时代转折的"预告"?严峻的就业形势究竟因何而起?未来，我们还能看到转机吗?

编者注：此处说的"电磁弹射"指的是上图所示新闻防部的发言——福建舰具备了电磁弹射和回收能力。这应该是"赢学"的一种。今年就业数据又在下滑，这种形式下,"赢学"是不能长久的。年轻人找不到工作之后，这些都会成为反噬。

关于失业的数据，预测未来能到多少百分点? 目前看还在恶化当中，而且是结构性的恶化，除了少数大国重器高精尖的真人才要赚麻了，其他大量本科生研究生还要再往下挤压，下面的就更惨了。失业率有可能会一度飙升到30%以上。

差不多。目前青年还有他们的长辈能啃，暂且还能压制住情绪。也多亏了张一鸣，他走到哪里，哪里的人口就要崩塌，无论是新疆还是非洲，一旦刷上瘾了，就不生孩子了。要不然经历了恐怖内卷却面临巨大落差与失望的青年们，要是胡思乱想起来，可就要命了。

嗯，主要是现在的年轻人也没那么好骗了。年轻人失业率怕是会更高。

你看特朗普所有的政策，核心点都是要把工作留在美国国内。那我们呢?我们有没有什么相应的举措?我们可以鼓励企业出海，卷死他们。

现在低端制造业被打击的，减少了好多蓄水池。我觉得根本问题还是分配不公。如果没人认真去解决，就只能不断"给药"延缓问题爆发。跟分配没关系，就是产业升级。

低端产业出海被堵了，参考拼多多的Temu本来如火如荼的，现在美国取消免税政策、加征关税，基本就熄火了。劳动力密集型的低端产业虽然一直在向内陆转移，但市场越来越小，内卷、去产能的阶段，不知道还要持续多久。

美国打的就是我们的三寸，让我们瘫倒。特朗普没那么针对中国，他是对全球统一打击。过去美国是通过全球的转移支付，向不发达国家输出美元和债权，让这些国家有钱来消费美国的服务业。现在特朗普的美国放弃转移支付了让大家自力更生，自生自灭。特朗普限制中印的工作签证，就是中国限制菲佣的工作签证，都是为了保障底层民众有就业。印度的美元主要就是靠这帮打工人汇回本国，被特朗普这么一搞，账户平衡都要崩的。

就业形式的严峻，或许源于全球产业升级的浪潮已不可逆，低端产业及其所承载的就业人口正面临被淘汰的风险。同时美国也在全球范围内"回撤"就业岗位，进一步减少落后产业的生存空间。时代在进步，可它留给人们的机会却在减少，如何缓解这种结构性失衡，将成为每一个国家新时代的第一个考题。
