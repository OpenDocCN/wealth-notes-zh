
# 女人对情感的6个重大误判

人类是一种喜欢说谎的动物，男女都一样。男人说谎是为了骗别人，女人说谎很可能是为了骗自己。

今天分享6个女性喜欢对自己说的谎言，或者说是重大的误判。

## 1. 择偶方面的误判

如果你随便问一个女人：你喜欢什么样的男人？你将来想要跟什么样的男人婚配？她很可能回答：性格好的，有责任心的，愿意给我花钱的。

实际上，她描述的根本不是“好男人”，而是“容易被她控制和操纵的男人”。如果她真的找到了这样一个男人，她可能会觉得对方很无聊，很没有男人味。最后，她还是去找了一个能掌控自己的人。

甚至，有的还会找一个经常殴打自己的男人。所以，如果你是个男人，你一定要找一个你可以主导、可以掌控的女人，这样的话，这段关系才能长久。

如果你们势均力敌，或者你的势能比对方还弱，那么对方肯定会飘上天，最后这段关系很难长久。

那么问题来了，如果是男强女弱的组合，男人不会飘上天吗？大概率不会，因为男人不会跟女人比收入、比能力。换句话说，男人的收入是女人的10倍都无所谓，丝毫不影响这段关系的发展；但是，如果男人的收入跟女人持平的话，家里肯定会闹得鸡犬不宁。

## 2. 事业维度的误判

很多女人都这样标榜自己：赚钱比男人重要多了，也靠谱多了。但是，能创业成功、赚到大钱的女人寥寥无几，大概只占男性的十分之一。

所以，大部分给自己贴上“搞钱”标签的女人，最后不仅钱没赚到，家庭也没有建立成功。

我知道，女人并不一定要结婚、生子，但是如果你事业维度很一般，都30多岁了还只能拿几千块钱，而且父母无法给你提供任何助力……那我还是建议你找一个靠谱的男人嫁了。

其实，大部分女性的价值主要取决于自己的人生目标：低价值的女性追求成功的事业，高价值的女性追求成功的家庭。

有没有那种追求事业成功的高价值女性呢？也有，只不过很稀少。比如，那些在科学、艺术、商业等领域有很高天赋的女性，她们如果把重心放在“组建家庭”这件事上，那就太可惜了。

从某种程度上来说，男人也是同理，如果自己没什么本事，那就早点组建家庭，否则等自己老了之后，可能更悲惨。

只不过，男人的适婚年龄比女人要长一些，可以多挥霍几年。

## 3. 对好闺蜜的误判

女人的嫉妒心天然就比男人强，这是刻在基因里的。所以，如果你找到一个还不错的男人，千万不要征询“好闺蜜”的意见，否则她可能会撺掇你离开这个男人。

一方面，女人天生就喜欢挑毛病，看见男人的缺点，你把自己的男人带到好闺蜜身边，她第一眼就能看到这个男人的缺点。她给你指出来之后，无论你们分还是不分，你的心里都会有个疙瘩，属于给自己找不痛快。

另外，如果你的好闺蜜是单身，她也会劝你保持单身；如果你的好闺蜜是离婚状态，她也希望你早点离婚，这样你们就“同病相怜”了。

但是男人的“好哥们”无论再怎么不靠谱，一般也不会说对方女朋友的坏话，只会说：你小子能找到女朋友，就烧高香吧，还要什么自行车。

因为男人和女人面临的择偶压力根本就不是一个量级的。女人非常清楚，只要自己不太丑，只要自己能过道德观，那么可以一个月换一个男朋友，何必在一棵树上吊死？但是男人不一样，他们会想：好不容易把一个女人忽悠到手了，可不能轻易让她跑了。

所以，能够逆着人性操作的人，才最有魅力。比如，用情专一的女人，以及有放松心态的男人，才能让异性欲罢不能。

## 4. 对经验丰富的误判

无论一个男人有多开放、多开明，无论他怎么诱导你，你都不要把过去的经历和盘托出。哪怕你以前有过20个男朋友，你也要说：“我之前只谈过两个，而且那时候根本不懂爱情，我没有任何幸福和兴奋的感觉。只有碰到你之后，我才知道什么是真爱，什么是性福。”

男人可能喜欢跟经验丰富的女人玩，但是她们想要娶回家的，一定“经验少”的女人。

我记得老舍先生说过这样一句话：世上真话本就不多，一位女子的脸红，胜过一大段对白。换句话说，女人的害羞、脸红、经验少，永远是加分项，永远会让男人心动。

但是反过来就不成立了，男性一定是越成熟、越有经验，才会显得越有魅力。男人如果呈现“扭捏之态”，只会让人作呕，瞬间没了性趣。

另外，在真心话大冒险的时候，无论是男是女，千万不要透露自己的风流往事（比如嫖娼、出轨这类事情）。即使你是带着真心和诚意去忏悔，你也是在摧毁这段关系。

男女双方之间隐藏的秘密越多，这段关系越会长久。

有人会说，以后万一露馅了，不就更糟糕了吗？我的回答是，生命很短暂，在心智非常成熟、身体非常有活力、对异性非常有感觉的阶段也不过二三十年，一眨眼就过去了，谁还纠结那么多？

在这片土地上，重男轻女的家庭比例是多少？总之，比例不低。

所以，如果你是个女人，你一定要通过各种蛛丝马迹来判断，自己的父母到底是真的爱自己，还是把自己当成一个赚钱工具。

比如，如果你的父母让你给你的弟弟买车买房，那么他们大概率是把你当成一个赚钱工具。

同理，如果你已经有孩子了，也不要把手里的房子和钱都交到子女手里，你要把这些优质资产牢牢地把握在自己手里。

这样的话，即使你老了，你也依然有谈判的筹码，这玩意比亲情靠谱多了。
