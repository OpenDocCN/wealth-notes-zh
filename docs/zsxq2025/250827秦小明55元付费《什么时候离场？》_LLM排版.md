
# 什么时候离场？

## 四川投资交易市场里，有句古老的谚语，叫：
会买的是徒弟，会卖的师父。

## 那么一般在什么情况下，我会考虑离场？

## 纯个人经验分享，无关行情判断，请谨慎付费。离场可以翻译成几个别的常听到的词汇：止盈、止损、平仓。

### 1）逻辑发生明显的边际变化

这里有三个关键词：
- 逻辑，你需要知道你为什么持有这个仓位，驱动是什么。
- 边际变化，不是静态的“好”与“不好”，而是相较于当下定价逻辑的新的情况。
- 明显，越明显，离场应该越彻底。举例：
  - 如果一个大宗商品处于过剩状态，且市场长时间对这种状态进行了交易与定价，表现为价格明显且持久地下跌，甚至已经跌破完全成本，进而现加微信号jntsg2，免费进资料群

### 2）达到开仓或者持仓过程中设立的离场点

在建立或者持有一笔仓位时，交易/投资者往往无法准确预判未来价格的“终点”。但通常应该设立一个“目标价”。这个目标价有时会随着交易过程中的新增信息不断动态调整，则以最新的目标价为准。

例如，一个简单的“边打边撤”的方法：以股票市场的交易点位为例，如果指数每上涨 10%，则减仓 20%。当股指上涨达到 50%时，则清仓离场。

### 3）达到风控标准

此条以止损为主。风控的维度又分为三个：
- 单笔止损，总量控制，强制平仓。
单笔止损很好理解，字面意思即是。
- 总量控制在于将资产配置视为一个整体，一旦整体资产跌破某个位置，仓位则一定要先行离场。
- 强制平仓在杠杆交易中常见，此时必须离场，杜绝“硬扛”。

例如，当你持有一系列仓位面临不利局面，某一个盈利的仓位已经快速回撤，你设定的单笔止损标准为保住本金，则应在盈利回撤完毕之前果断离场。

或是，当你的持仓变动，使得账户总权益跌到一百万附近，如果你的总量标准为不允许账户跌破百万权益，则此时也应果断离场。杠杆交易当中，一旦触发强制平仓，则不应再坚持原有仓位，即便事后拉长时间来看判断并未失误。

以上是我在投资交易中常用的三大类离场思路和方法。三种方法相互并不独立，互相影响，也互为补充。

例如，当交易者对逻辑变化并不自信或者看不清楚时，则应该利用“目标点位”作为技术补充。又例如，当交易者面临市场反向运动又坚定自己的逻辑判断时，则应以“不超过风控强平线”为下限。

由于大家投资与交易，面临的 risk-return profile 千差万别，因此不可能有一个统一的标准的“离场点位”。有人做短线，有人做长线，有人做定投，有人做单边，有人做对冲，有人做趋势...

因此，但凡碰到“具体指导”的建议和策略，都需首先弄清是否与自己的交易投资体系匹配与兼容。

投资交易永远充满“主观”判断与偏见，回头看，遍地是黄金。往前走，处处是陷阱。因此，主观的判断与行动（开仓），必须配合客观的无条件执行的离场条件，这样才能活得长久。

活得久还是赚得快，又是一个非常深刻的投资哲学问题，今天就不展开了。如果还没有自己的投资和交易体系，第一件事恐怕不是赚钱，而是学习。
