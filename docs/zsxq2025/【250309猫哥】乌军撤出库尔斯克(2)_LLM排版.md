
# 乌军撤出库尔斯克

3月9日最新消息，乌军基本从俄罗斯的库尔斯克州（简称：库州）撤出，包括去年占领的最大城镇苏贾，确认俄军已进入苏贾市中心，没遇到太多抵抗。其实严格来说，乌军还控制着库州部分区域（不到100平方公里），但无关紧要了，考虑到目前的恶劣态势，坚守剩下的区域毫无意义。撤军的原因，是撑不住了。

去年8月份的“库尔斯克反击”，一度取得不错的战果，最高峰时，乌军在库州的占领面积达到1200平方公里，比香港略大。拿下的最大城镇，是苏贾，人口体量约5000（算上周边的村庄为一万），北边还有个较大的村庄小洛克尼亚，两者共同组成了乌军在库州的防御中坚。为守住这块地，作为谈判筹码，乌军投入了第80空突旅、第95空突旅、第47机械化旅等大量精锐部队。俄军知道不好打，所以不硬刚，慢慢蚕食。通过半年多的蚕食，俄军把苏贾和小洛克尼亚的外围，蚕食得差不多了，乌军的控制区面积，被压缩到不足400平方公里，处于一个狭长的区域内。最窄的地方，还不到六公里。偏偏最窄处，是乌军通往库州的唯一补给线。六公里对无人机来说，并不远，于是补给线头上，无人机满天飞，炮弹不时砸下，导致补给困难。

乌军几次尝试反击，想把俄军推离，都没有成功。补给不足，是很难作战的，不撤不行了，总不能把这么多主力部队，葬送在库州吧？撤军的传言，早在1月份就出现了，第一步是把几个精锐突击团撤走。乌军现在最宝贝的几支部队，就是第33突击团、第425突击团和第225突击团。这些部队装备最好，兵员也最好，战斗力最强，撤军时的“待遇”自然也最好，第一个走。所以在3月初，库州的俄军报告，表示乌军的无人机活动大幅减少。第二个步骤，是从北边的小洛克尼亚撤出，最后一步，是从苏贾市撤离。但计划赶不上变化，3月初，一支俄军突击队，顺着废弃的天然气管道爬行12公里后，进入苏贾市内，出现在乌军防线后方，引发了乌军的慌乱。俄军趁机大局进攻，乌军局部防线崩溃，导致了无序撤离的现象，有些单位甚至被歼灭，仅从流出的各路视频统计，损失就不低于千人。只不过，乌军在库州总体有1.5万兵力，损失不到10%，尚在可承受范围内，主力还是保存下来了。当然，俄军本来也没指望围歼他们。现代战场透明度高，想要成建制歼灭敌军一个大单位，是很难的，整个俄乌战争期间，只有两支部队被成建制歼灭，就是乌军第36海步旅和亚速团。俄军的策略就是两翼压缩，再不断骚扰补给线，等你受不了了，自己就会走。从弗勒达、大诺沃西尔卡，再到库州，都是如此。

撤出库尔斯克，对乌克兰来说有好有坏。先说好的方面。受制于兵力不足，乌军进攻库尔斯克时，不得不抽调部分顿巴斯方向的主力部队，导致顿巴斯防线兵力空虚，被俄军抓住机会。比如“纽约镇--捷尔任斯克”堡垒群，这么重要的地方被顿军两个旅几乎拿下，实在是不应该。库尔斯克攻势开始后，乌军为达成最大战果，把部分恰索夫亚尔的部队，调往库州，偏偏恰索夫亚尔又挺重要，是区域内的制高点，不得不再从“纽约镇- -捷尔任斯克”方向调来第24机械化旅。这就是拆东墙补西墙。第24机械化旅离开后，“纽约镇--捷尔任斯克”方向的乌军兵力不足，仅有一个质量较差的国土旅，和一个编制残缺的空突团。被俄军抓住偷袭，几乎丢掉了整个堡垒群。库尔斯克攻势开始后，类似的案例还有不少，从去年下半年以来，俄军在顿巴斯几乎顺风顺水。但从库州撤出后，乌军几个精锐突击团回援捷尔任斯克，抓住俄军换防、注意力不集中的空当，夺回部分城区，稍稍稳定了战线。更多的乌军撤出库尔斯克，意味着回援更多，俄军的攻势将被遏制。

坏的方面，则有两个，一是失去了一个谈判筹码。拿库尔斯克换顿巴斯，一直以来就是泽连斯基的主张，要达成这个目的，起码你得守住苏贾，有那么几百平方公里的占领面积。只有50-100平方公里难以构成筹码，几条村（主要是苏贾西部城区和邻近村庄）在国际谈判中没意义。剩下的那么点地方，属于一个极其狭长的地带，想守也守不住。拿库尔斯克换顿巴斯的可能性，被硬生生打掉了。二是俄罗斯的条件，可能更苛刻。俄罗斯的和谈条件，数年来经历过多次变更，“哈尔科夫大败”后，普京底气不足，只要“实控线停火”。2024年形势好转后，停火条件变成了承认五州入俄。当然，赫尔松右岸和扎波罗热剩下的三分之一，俄罗斯无力控制，就只能是嘴炮。此番“库尔斯克大胜”后，条件肯定会提得更高。过去两年，梅德韦杰夫多次声称，俄罗斯的新领土范围包括敖德萨。作为“边缘人”，梅德韦杰夫早已不在俄罗斯权力核心圈子。他的表态，理论上不具备官方意义。但理论和实际，从来就有互相转化的可能。毕竟，俄罗斯人对敖德萨的情结，是明摆着的。如果能控制敖德萨，乌克兰经济将彻底完蛋（失去出口，很多廉价商品难以出口，比如农产品，走陆路不划算，且和东欧构成竞争），会成为欧盟的纯纯包袱。

对乌克兰来说，目前已没有更多奢望，最好的结果是“按实际控制线停火”，如同曾经的《明斯克协议》那样，而要达成这一点，必须守住现有战线。咱们还是希望乌军守住，京子开路虎不是一件好事。
