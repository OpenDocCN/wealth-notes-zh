
# 卢克文青云读书会8.20：下半年宏观经济与资产配置展望

## 出口：关税导致下行压力较大

关税是当前经济中的重要因素。自2018年以来，美国对中国加征了多轮关税，包括芬太尼关税、对等关税以及232调查的行业关税。目前，美国的关税水平已达到1940年以来的最高水平，约为18%~23%。

美国的关税政策可以分为三个部分：10%的基础对等关税、232调查的行业关税以及10%以上的对等关税。其中，基础对等关税和232调查的行业关税难以谈判，而10%以上的对等关税则可以根据不同国家进行调整。例如，英国目前对这部分关税为0，而中国则推迟了90天。

中国在中美关税谈判中的态度强硬，但自4月7号以来，特朗普将重心转向其他国家，对中国的态度有所软化。稀土成为重要的谈判筹码，中国是全球唯一的拥有稀土全产业链的国家。此外，2026年的中期选举也为特朗普提供了与中国交易的动机。

综合以上因素，中美之间的关税谈判最艰难的时期已经过去，预计最终关税税率将在50%左右。关税到期后，中美关系将大概率处于阶段性的稳态。

## 出口压力分析

尽管关税紧张时期已过，但出口压力尚未完全解除。首先，出口订单的激增主要来自存量订单的释放，而非新增订单。其次，转口贸易在上半年出口表现中超预期，但美国与越南、柬埔寨等国的贸易协议可能对转口贸易造成不利影响。最后，上半年明显的抢出口效应可能在未来导致出口下降。

出口下滑将引发一系列问题，如企业利润下降、失业率上升以及制造业投资减少。据测算，出口下半年的下降对全年名义GDP的影响约为0.5%。

## 地产：房价下跌趋势明显

北京地区的房价在5月份开始出现明显下降，预计下跌幅度至少为15%。上海的情况类似。地产市场的下滑对企业利润和投资产生影响，进而影响宏观经济。

## 资产配置展望

在当前经济形势下，企业需要采取自救措施，如出口转内销、拓展海外市场以及出海建厂。然而，这些措施都存在一定的局限性和挑战。因此，企业需要根据自身情况制定合理的策略，以应对未来的经济变化。

## 结论

下半年宏观经济与资产配置展望充满挑战。企业需要密切关注政策变化和市场动态，采取灵活的策略以应对风险和机遇。同时，政府也需要出台相关政策，支持企业发展和经济增长。
