
# 卢克文青云读书会趣说挡史：资本主义（上）汉武大帝与资本主义萌芽

大家好，我是杨闻哲，欢迎收听卢克文工作室的党史栏目。我们前面几篇文章介绍了陈独秀、李大钊以及青少年时期的毛泽东这三位中国共产党草创阶段的核心人物。但光是几个核心人物的铺垫还是很单薄，由于共产主义革命是相较资本主义乃至整个私有制而言的，因此还是很有必要梳理一下人类社会生产力的发展是怎样的脉络，尤其是资本主义是如何席卷全球的。

资本主义的爆发式增长是人类进入工业化时代的一个重要推动力，由此激化了阶级矛盾从而产生共产主义的意识形态。当我们较为清晰地了解资本主义的发展脉络后，我们才更能理解为什么近代中国会经历一个觉醒年代，为什么只有社会主义才能救中国，以及为什么没有共产党就没有新中国。接下来的几篇文章将从人类社会生产力的起点一直讲到中国共产党的成立，中间可能会穿插做一些阶段总结和议论。

## 1、生存条件造就不同文明

生存是一切生物的本能。基于这个本能，人类作为一种生物、一种动物，就得不断地发展生产力，不然吃不饱、穿不暖，就会饿死、冻死。所以也可以说，发展生产力也是人类社会的本能。

如今我们发现大量出现在原始社会时期的生殖器崇拜内容的绘画作品、雕塑作品。原始人为什么要崇拜生殖器呢？因为原始社会没有多少生产工具，无论是狩猎还是采集，其生产力几乎全由人力决定，人力又几乎全由人口数量决定，哪个部落的人口多，力量就大，就更有机会生存下去。人多力量大这个道理原始人就已经认识到了，原始社会，生殖力就是生产力。所以原始人崇拜生殖器，不是因为他们都是老色批，而是因为他们肉眼可见那玩意能造人。

现代人崇拜高科技，倒不是因为高科技也能造人，而是因为高科技基本上决定了如今生产力的基础，哪个国家的科技水平更高，哪个国家的生产力水平就相对更高，哪个国家的生存和发展主动权就会更多。当然制度也很重要，但科技水平是个基础。所以原始人崇拜生殖器和现代人崇拜高科技，本质上都是崇拜生产力。

当人类掌握了种植术之后，农业水平就成了衡量生产力水平的重要指标。因为农业能让大家吃饱穿暖，能给人更多生存发展的机会。但那时候的农业技术也不发达，农业生产水平主要取决于先天条件，就是土地和气候。全世界最适宜农作物生长的纬度主要集中在南北纬30度~40度左右。

在南半球，这个区域就是南非、澳大利亚南部和阿根廷，从地图上就能看出，所经之地缺少连续大片的土地；在北半球，这个区域自西向东依次为地中海地区（西班牙，意大利，北非，希腊，土耳其），中东地区（阿拉伯地区和伊朗高原），印度（新德里附近的一片平原），青藏高原，中国的黄河与长江流域，以及美国的大部分地区。

可见，只有中国的黄河长江流域和美国有着大片连续像样的土地，并且还有足够的水源（河流多），所以方便耕种的平原地貌也很多。由于美国那块地被大洋包围，远古时期没什么人，所以我国这块地方便成了天选之地，自古人丁兴旺，物产丰富，富强是个必然。

这也使得我们的先民很早就发现，只要勤劳勇敢，大家想要的一切几乎都能从土地里得来，所以我们国家的先民最主要的生产方式便是农耕，并且我们有先天条件摁着一块地去精耕细作，久而久之也就形成了一种思维习惯或者说是民族精神：自力更生，自食其力，遇到困难便开动脑筋团结一致去战胜困难，这样一来，生产力便随之提升了。

大禹治水就是这种精神的体现。同时也形成了安土重迁的观念，因为在那个靠天吃饭的日子里，迁徙是个大冒险，没人敢保证所迁往地区的土地能有足够的生产力保证自己的生存。

但是欧亚大陆西边的欧洲就不一样了。古代欧洲适合农耕的土地实在是不多，主要集中在地中海沿岸，还尽是山。欧洲大部分土地都在相对寒冷的地区，那里植被种类有限，生长周期也更长，不是你想种啥就能很快长出啥来的。

中学的地理课本里也说过，欧洲，以及同纬度的我国北方地区，多数土地更适合放牧，因为从土地里最容易长出来的是牛羊等牲畜的口粮，也就是各种草。于是欧洲的先民最早的生产方式主要是游牧。这个牧前面之所以有个游字，也是因为自然条件所限，一块地方的草吃完了不会及时长出来，那就只能赶着牛羊去游荡到另一块有草的地方去，没有条件也没有动力去摁着一块地精耕细作。直接占有并获取另一块土地上的财富便成为了游牧民族普遍的思维习惯。

只是游荡的过程自然是不确定的，必然经常遇到危险，所以西方的先民每换一块地方放牧，就是一次冒险，免不了与其他人发生冲突，发生冲突免不了诉诸武力，于是久而久之，游牧民族便拥有了普遍的冒险精神，并且崇尚武力。这几种植性延续至今，就使得当两种民族以文明的规模碰撞到一块时，游牧民族在农耕民族眼里，就是一伙野蛮的强盗。

## 2、汉武帝的作用

我们现在常讲一带一路，其实我国早在春秋战国时期就已经形成了“海上丝绸之路”的雏形，但由于那时候的航海技术十分落后，加之陆地面积足够大，又尽是万顷良田，世界那么大，双腿就能到达，又何必出海呢？所以我国自古以来就没有多少出海扩张的动力，也就不会形成什么海权秩序。守着土地，辛勤劳作，繁衍生息。

当需要更多土地去容纳更多人口时，我们才会以最初的起点为中心向外辐射扩张土地，久而久之，也就自然而然地形成了陆权秩序。我国开始建立农耕文明的陆权秩序后，便自然产生了地主阶级。总有人会率先通过暴力占有某块土地，并迫使其他人在自己的土地上创造财富，然后把其中大部分都交给自己，这就是地主。农民臣服地主收租这种获取财富的方式放大到宏观层面上，就是早在商周时期便已现雏形的朝贡体系。

而在欧洲，由于纬度适宜的地区主要集中在地中海沿岸，但就意大利、西班牙和北非那几块地实在是少得可怜，尽管古代欧洲的先贤，比如亚里士多德这些人，也很重视农业，但巧妇难为无米之炊，地理条件实在不给力。

不过这种环地中海展开的地缘格局使得他们在做稍微大一点的空间跨越时，会更加频繁地通过海路到达，并且本来出海与游牧这两种行为方式就有颇多相似之处（主要是冒险精神和对武力的依赖），于是当北方的游牧民族入侵到地中海沿岸后，这种地缘格局便将游牧与航海这两种行为乃至思维很好地结合到了一起。我想，欧洲便是基于此最终建立起海权秩序乃至海洋文明的。

我们国家自西汉武帝时打通丝绸之路，便有了一个重要的对外交流的渠道，加之自身的强大，我国的文明也就开始成为一个外向型文明。这种状态持续了一千多年，到唐朝中期达到鼎盛，也随着唐朝灭亡而基本结束。外向型文明的特点，一个是武德充沛，能打，不一定穷兵黩武，但至少没人敢轻易进犯我们。为什么这么说呢？因为土地是农耕民族赖以生存的饭碗，有人要砸我饭碗，我肯定以命相搏。所以以我们有个成语叫“守土有责”。外向型文明另一个特点是文明程度高，文化成果丰富所以又有足够的势能不断向外输出；随之而来的就是民族自豪感和民族自信心都很强。直到现在，中国的主体民族名叫汉族，而海外华人的聚居地名叫唐人街，也是这种民族自豪感和民族自信心的延续。

那汉武帝是如何打通丝绸之路的呢？公元前139年，张骞不辱使命，凿空西域，促进了东西方文明的交流。但丝路贸易并没有随之而起，因为匈奴的威胁还没有消除。于是汉武帝决定先击败匈奴。匈奴为什么要侵犯西汉呢？主要还是前面说的地理条件因素，那时候游牧民族生活的地方种不出多少像样的作物，生存资料始终紧缺，这种客观因素使得游牧民族会周期性地对农耕民族实施抢劫来贴补家用。

匈奴也不例外，匈奴当时抢劫的对象就是中原地区的汉朝，这就必然威胁到了西汉政权的利益。如果北方这个压力来了，汉武帝不顶住的话，那么如今中国的地缘政治版图恐怕就会跟现在的欧洲一样，整个被匈奴铁骑给压碎了。汉武帝当时是坚决顶住。公元前127年，也就是西方的耶稣诞生的这一年，卫青河南之战胜利，使长安免除了匈奴的直接威胁；公元前121年，河西战役，霍去病把匈奴摁在地上摩擦，歼敌数万，并占领了全部的河西走廊；公元前119年，张骞第二次出使西域，打算加强与西域各族的友好往来。同年，霍去病封狼居胥，就是今天外蒙古首都乌兰巴托一带的肯特山。“是后匈奴远遁，而幕南无王廷。”此后真正意义上的丝绸之路随即出现，朝贡体系得到强化。历史上所有的商道都是战争打出来的，打完了，稳定了，才经商。这一年，汉武帝还宣布盐铁专卖政策，西汉随之走向富强。

在汉武帝的坚决抵抗下，匈奴元气大伤，一时不敢向南进犯大汉，向北退又太冷，只能往西去往欧洲。当然匈奴也不是全都跑了，不然后面也就没王昭君和班固什么事了。一部分匈奴人西迁后在欧洲建立了匈奴汗国，以阿提拉帝国——也就是匈人帝国——为代表，涉及现在东欧的广大地区，包括如今的俄罗斯伏尔加河流域，高加索山诸国，乌克兰，白俄罗斯，波兰，波罗的海三国，以及德国的一部分等等。匈牙利这个国名就是源于匈奴这个词。匈奴人来欧洲了，挤压了西方学者口中的北方蛮族日耳曼人的生存空间，便使日耳曼人被迫南迁，原本主要在欧洲北部活动的日耳曼人就这样开始侵犯地中海一带的西罗马帝国的势力范围。

而匈奴人也一样垂涎相对富饶的西罗马帝国。终于，公元476年，主要在匈奴人和日耳曼人的入侵下，西罗马帝国分崩离析，欧洲的奴隶制度也就此终结，中世纪开始，欧洲这才进入封建社会。封建建国，像中国的周朝尊周天子为最靓的仔一样，欧洲分裂的诸国尊教皇为最靓的仔，然后大家各玩各的。所以表面上看中世纪统一于教皇，实际上仍是分裂的状态。只是这种统一的宗教意识形态，为以后组团出去打劫提供了理由上的便利，也使得那些翅膀先硬的人脑袋还在宗教的控制下，没法彻底放飞自我。

拜汉武帝所赐，欧洲从此永无宁日。反过来想，如果当年汉武帝没有顶住匈奴南下，匈奴就不会去欧洲，如此，欧洲的地缘政治格局可能就类似今天的中国，统一的格局可能会周期性地延续至今，而中国则有可能就是严重破碎化的欧洲，那么东方领先于西方上千年的历史就不可能出现，甚至东西方的历史发展进程都会调个个。

所以说，汉武帝抗击匈奴在世界历史中的意义是巨大的，它不仅守卫住了农耕民族的土地，巩固了从秦始皇开始的统一的地缘政治格局，还奠定了华夏大地农耕文明崛起的基础，但也给欧洲最终产生资本主义埋下了伏笔。

## 3、中华文明的转型

欧洲的中世纪刚开始一百多年，在欧亚大陆东边，公元581年，杨坚结束了东汉灭亡后300多年的三国两晋南北朝的分裂局面，建立了隋朝，也恢复了因分裂而崩溃的朝贡体系。但仅仅37年后的公元618年，隋朝就被唐朝取代，朝贡体系也被唐朝继承。

与唐朝几乎同时出现的便是伊斯兰教。伊斯兰教在今天欧亚大陆中间迅速团结了阿拉伯人并于公元632年建立了阿拉伯帝国，也就是史书中的大食。到唐朝灭亡的时候，阿拉伯帝国的势力范围已经扩张到和今天阿拉伯国家所占地方差不多的情况，那时候还包括地中海的几个主要岛屿，比如塞浦路斯，克里特岛，西西里岛，撒丁岛和科西嘉岛等，还包括西班牙。于是从这时候开始，欧亚大陆上的三个主要文明都已登场，从西向东依次是基督教文明，伊斯兰教文明和中华文明。

其中伊斯兰教文明，或者说阿拉伯文明，一直扮演着连接欧亚大陆东西两端中间人的角色。中学历史书就经常提到，中国的许多先进生产力，比如四大发明，都是通过阿拉伯人传到西方的。

公元907年唐朝灭亡后，经历了53年短暂的分裂时期，也就是五代十国时期，很快，中原大部分地区又重新统一于北宋。只是这短短53年时间，足够外族重新侵占唐王朝曾经控制的西域、河西走廊以及北方地区。于是北宋政权从公元960年建立之初便处在强敌环伺的状态下，加上早在安史之乱之后，唐王朝便失去了对丝绸之路的控制力，唐朝后半部分那一百多年的时间里，丝绸之路名存实亡，丝路贸易也逐渐中断。公元894年，离唐朝灭亡还有13年，日本也不再派遣唐使了，可见我国的外向型文明模式已然走到了尽头。

中华文明基本上从宋朝开始，就逐渐转变为内向型文明。华夏文明开始震荡下行。

但由于宋朝的地盘大大缩小，只能说基本统一了中原地区，汉唐时期那种恢弘大气没了施展空间，我国的文化特征也就从豪放转型为内敛，越来越精致也越来越内卷。也正是因为宋朝失去了汉唐时期北方辽阔的土地，政治经济重心被迫使南移，我国南方地区才迎来发展机遇。到了南宋，地盘进一步被压缩在南
