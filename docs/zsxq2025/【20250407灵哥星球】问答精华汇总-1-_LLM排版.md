
# 问答精华汇总(1)

提问：灵哥，我们对美限制稀土等金属出口，虽然打击了美国，但也限制相关企业的销量，降低了利润，这算利好还是利空?
灵哥回答：每个公司情况不同，要具体问题具体分析。如果过度依赖初级出口的公司可能是利空因为出口额面临下降。出口管制带来国际价格上行，对于不那么依赖出口的在产业链有独特地位的企业形成利好。

提问：灵哥好，半导体大单流入，具体关注哪些，材料设备还是制造
灵哥回答：都可以关注，贸易战往后打，中美脱钩叙事半导体板块利好。比较利空的是消费电子出海，果链，达链。当然如果特朗普愿意在4月9日延长90天，什么都会反弹。看明天资金动向吧，不仅A股，还有美股，如果要延迟的话，明晚美股应该有抄底资金涌入。

提问：灵哥，华生这两天的文章提到特朗普的做法和巴菲特2003一篇文章中关于美国贸易赤字的思路如出一辙，所以必须重视美国的做法，按这个逻辑这样，这次中国的麻烦是不是还挺大的，不少国家也选择站美，短期A股和港股市都没那么容易恢复回去。想听听灵哥的想法。
灵哥回答：他文章我看了，特朗普肯定要跟中国谈的，打就是为了谈。他喊的越凶，代表越想谈。他也有各种时间限制，比如明年的中期选举。他把tiktok禁令推后了75天，隐含的当然是这期间要找中国谈。我们现在主要是看上面的想法，究竟会在怎样的条件下谈，还是暂时不鸟他。后者会给股市造成更大的不确定性。

提问：灵哥，大跌大买逻辑，适用于如今这种局面吗?美股和全球股市暴跌，港股也暴跌。
灵哥回答：很多公司已经非常便宜了，无法保证你明天是否会涨，但中长期当前位置很低了。另外美股短期也超跌了，正常标普5000点附近反弹，5000守不住，下面看4600。再跌各种杠杆都要爆了。

提问：不理解为何没有任何措施出台，放任股市下跌? No.2不是说了准备充足了超预期的手段，临了又悄无声息，这种难以捉摸的预期管理实在费解。
灵哥回答：彭博社报道：清明假期官员都在开会讨论各类政策。再给大家讲个真事，去年1月大跌的时候，之后的春节证监会几乎全员加班。慢慢来，上面着眼的是整个经济层面复苏应对，牌是要一张张打的，不能跟着美国节奏。我今天看到很多半导体公司特大单都在流入，但挡不住小单流出，很多散户怕，散户要跑，那就让他们先跑吧。

提问：灵哥您好，最近出现的统一大市场概念，您怎么看。
灵哥回答：如果只是概念，来的快去得快。

提问：灵哥好，小白提问有关估值的问题，我一直很困惑，以长亮科技为例，这家公司我理解是成长类的公司，是不是应该用PEG去看这家估值呢，它的peg非常之高，但是又坚信灵哥对这个股票未来的判断，所以就很困惑，是不是这样看企业估值是不对的，这种股票应该怎么去分析呢
灵哥回答：不是的，估值对于很多企业的市值并没有绝对意义，大部分A股公司的涨跌都和估值无关，因为变动太大，大部分和事件驱动的情绪有关，和流动性有关。情绪在表现就好，情绪不在就表现不好。

美国知名投资人Bil Ackman发了一个观点，他认为特朗普会以对等关税需要谈判的名义，推迟关税实施。如果是这个剧本，美股会反弹，但是中长期仍然不看好。今天人民日报发文表示会有很多政策要出，下周如果A股有恐慌，你们别恐慌就好，国家队随时就来了。特朗普当总统的世界，你们要习惯各类极端的情况，所以心态很重要。
