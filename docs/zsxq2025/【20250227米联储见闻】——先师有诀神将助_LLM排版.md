
# 20250227米联储见闻——先师有诀神将助

川总召开首次内阁会议，总结新领导班子工作成绩，过程融洽，发人奋进。川总领导才华折服众人，对任何挑战马斯克参与内阁工作合法性的人，川总只会送三个字：滚出去。

会议高度肯定川总上任以来对国家面貌及美国优先战略的推进。首先谈乌克兰问题，川总称这是上任以来的第一个重要成果，泽连斯基将献上矿产协议，让美国收回对乌克兰的军事援助支出。川总强调协议对美国无附加要求，无需安全承诺，只要他出马，美国就能一本万利。

马斯克虽非内阁成员，但以政府效率部人员身份出席会议，川总公开消除与马斯克关系不和的谣言。川总询问官员是否对马斯克满意，并表示会把他扔出去。川总称政府正致力于削减规模，支持马斯克处理脏活，库吏就是一把刀，在达成目标前不会折损马斯克。

川总对给联邦雇员发述职邮件一事表示支持，称不回邮件者为素未果餐，浪费联邦资源，将被裁。新备忘录要求各机构在3月13日前提交大幅裁员计划，裁员主要集中在试用期员工，下一轮将针对资深公务员。

川总暂停了美国国际开发署90%的对外援助合约，约有一万个项目被取消。华尔街日报称，被砍经费影响对中国研究工作进度。川总拒绝评价是否会阻止中国对台动武，表示不会对此事评价，不给台巴子任何幻想空间。

川总展示商人天赋，对黄金签证作出说明。川总称夜总会有黄金VIP卡，进入美利坚也可弄金卡，赶走非法移民，留给有钱有实力的移民，使美国越来越有钱。川总称美国身份价值500万美元，相当于4000万元人民币，优越感拉满。

川总计算，美国有近500万非法移民，哪怕空出十分之一发放黄金签证，就有2.5亿美元，这些人来了还要置业消费，妥妥的印钞机。川总强调不会强制金卡移民为美国创造就业岗位，也不会按原始国籍实施配额制度。

川总计划4月2日开始对欧盟征收25%的关税，适用于汽车和其他商品。欧盟利用美国，成立欧盟就是为了对付美国。美国和欧盟之间有约3000亿美元的贸易逆差，欧洲对美国一点毛病都没有。

国会通过川总的预算决议，未来10年减税4.5万亿美元，削减政府开支2万亿美元，但未来10年将至少增加2.8万亿政府赤字，每年平均赤字将占GDP 6.8%，美国国债占GDP比率到2034年将由117%升至125%。

预算决议提出增拨3000亿美元给移民及海关执法局、海关及边境保卫局等机构，以及美军来落实驱逐过千万非法移民出境的政策。同时，预算将美国国债上限提升4万亿。

预算责成众议院能源和商务委员会削减8800亿美元开支，教育和劳动力委员会削减3300亿，农业委员会削减2300亿，以及其他多个委员会削减额外5000多亿。这些钱原本是用于老年、残疾或低收入医疗保险开支、学生贷款、学校午餐之类的福利。

川总提出向所有中国制造的船进入美国港口收关税，每艘船的关税费用高达150万美元。这个决策会让日韩笑开花，势必影响船主的订单，但问题是，就算要找韩国佬造船那也是一年以后才能收获，如果实施这条，美国物价会推高成什么样子。

对中国来说，估计也麻了，看到欧洲都被折腾成这样，中国估计心态好多了，既然是神经病，那就没啥好说的，没碰到我，随他疯，碰到了，往死里打。
