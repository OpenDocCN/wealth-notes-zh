
# 关税的本质是消费税
# 发布时间：2025年5月27日
# 【特朗普与产业集团系列5】
产业集团和金融集团，
他们之间征税的逻辑是不一样的，
产业集团希望由民众承担税负，
金融集团希望由产业集团承担税负。
而这也是特朗普关税的底层逻辑，
特朗普代表的是美国产业集团的利益，
所以他会推动对美国企业减税，
同时用关税让美国民众承担消费税，
将企业原本承担的税负转移给老百姓，
满满的都是资本主义企业家的算计。
中国自90年代以来是产业集团长期执政，
主思路是用房地产让中国民众承担消费税，
来推动给中国企业进行免税、补贴和退税，
论做法，比特朗普这个地产商的资本主义还资本主义。
十八大后中国金融集团迅速崛起，
他们通过三道红线，竭力遏制房地产这个消费税，
在海南搞自贸港玩零关税，
严厉打击石油、电商等产业集团的偷逃税款行为，
要求创投补税，对平台经济搞反垄断罚款。
这跟民主党和欧洲推动绿色低碳的思路高度一致，
他们一边努力降低全球关税给民众的消费减税，
一边通过加征碳税让产业集团承担高额的税负。
没有无缘无故的爱，也没有无缘无故的政策，
执政集团的阶级属性决定了他们所实施的政策。

特朗普建议自6月1日起对欧盟征收50%的关税
美国总统特朗普在社交媒体上表示，建议自6月1日起对欧盟征收50%的关税。如果产品在美国制造或生产，则无需缴纳关税。特朗普表示，欧盟成立的主要目的就是“在贸易上占美国的便宜”。

美国与欧盟的关税谈判期限延长至7月9日
美国总统特朗普表示，欧盟请求将关税谈判期限延长至7月9日，他已同意这一请求。特朗普称，本次与欧盟就关税问题的谈话“非常愉快”。
