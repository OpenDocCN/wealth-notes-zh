
# 旋转门夹头

温沌小站2025年02月24日15:29

## 01

本照片拍摄于2018年，刚登上首富宝座的许教授意气风发，回到武汉这个他读了四年书的城市。母校热烈欢迎了他，博导、教授。大楼冠名，换来的是首期两千万捐赠。

一个图名，一个图利，很平等。更让许教授感受到平等的，是湖北蒋书记的高规格接待，两张沙发并排坐，中间隔着一小桌。跟着蒋书记一起参加会见的王省，只能随秘书们坐侧边位置。这种待遇，是2017、2018、2019这三年里许教授游览各地所得到的标准待遇。相对比，那些更加如雷贯耳的，无论是BAT还是遥遥领先和BYD，他们的boss出行地方，通常得不到许教授的待遇。

为什么？现在地球人都已经知道了，许教授坏得很，于国于民百害而无一利，跟遥遥领先的国之栋梁差了不知道多少数量级。有一点我们要实事求是，许教授纵横江湖直到60多岁，在2020年之前的二十多年里，心底很坏的他，是很能帮人赚钱的。

恒大所至，拿地、建房、造势；无论是卖他地的城市，还是买他房的居民，都是赚的。买的越早，赚得越多；买的又早又多，那就赚嗨了。我就知道一位熟人，在2008年那次探底的时候，趁着恒大大降价。抄底购入多套房，很快就赚翻，从此财务自由专心钓鱼。

不仅买房的赚到了，买恒大股票的、债券的，理财产品的，在2020之前的许多年里，都赚得很嗨。

赚得更嗨的，是各地方。许家印是擅长造概念的，如天女散花，一个个所上去洋气看起来气派的大项目，什么烟大养生谷、海上威尼斯、海花岛、美丽之元。每到一地，概念吹出，内告和销售铺天盖地，让雨管是有房的or没房的厂大居民心痒难耐，完全忘记“房子是用来住的”金玉良言，疯狂地涌进了售楼处。

我自己是去看过那个海上威尼斯的，位置在一个若不是恒大我这辈子不会踏足的小城启东。说实话，对于长三角地区想有个海景房夏天来度假的广大居民，这项目挺诱人的。住不住是一回事，买两套同在这，过几年怎么也说也得翻倍。不用过几年、顿时就翻倍的是周边的地皮。再加上项目开工建设带动出的一长串gdp，当地躺赚。

这就是概念的力量。现代世界，最高阶的段位是造概念。就像这两年，AI横空出世，说白了也是美国资本造的概念。概念造好了，干亿万亿就有了。对于广大居民来讲，有AI不见得更幸福，没有AI的时候也没觉得少了啥。浑浑整塑，芸芸众生，只不过跟着弄潮儿造的概念在走。

所以，当我读到最新的会议精神，2025的工作重点是"大力提振消费，充分挖掘巨大消费潜力"，第一时间，我想到了许教授。

这个混蛋，是真能够把广大居民的消费欲望和潜力榨得干干净净。不仅因为他会起名字、创新词，更在于他的历史纪录，2020之前，跟着他买东买西的都赚到了。相反，如今的国之栋梁，无论是遥遥领先的汽车手机大屏幕还是赶日超美的哪吒2大电影，只是需要广大居民贡献，都没办法让广大居民赚到。上半年买的车，下半年新款又出来了，明年一过。价值剩多少？

俗话说，杀头的生意有人做，亏本的买卖无人干。

翻译成人话是，有钱赚的事儿怎样限购限货都有人离婚结婚见缝插针，只花钱的事情需要反反复复下文件。

但现在，许教授跟七年前坐一张桌子的蒋书记一道，坐进了牢狱。
