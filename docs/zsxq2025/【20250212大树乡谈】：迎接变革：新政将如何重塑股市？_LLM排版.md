
# 迎接变革：新政将如何重塑股市？

核心：稳字当头、腾笼换鸟。

本文谈股市改革方向，不涉及具体股票和荐股。本文分析主要依据两个公开信息源，对比二者之间的变化，并结合过去一年多来的演变。一是1月13日证监年度工作会议，二是2月1日吴主席在《求是》杂志发表的署名文章《奋力开创资本市场高质量发展新局面》。小镇对比了这两个信息源，基本内容和结构体例高度一致，新的署名文章并没有提出新的政策，而是重复年度工作会议的要求，但提供了年度会议通稿没提到的内容，还对涉及国家队的关键表述进行了调整，可以感觉到微妙的政策变化。

全文结论有五：
1、2025年核心是稳字当头，要避免股市波动过大，做好抗冲击准备；
2、国家队要从常态化干预转变为突发应急的战备力量，将减少出手；
3、未来股市重点不是继续增加上市公司数量，IPO不会放开，仅针对科技型优质企业放开一个口子；
4、基于以上判断，2025年上市公司监管还将继续加强，要加速劣质上市公司退市；
5、证监能够掌控的公募基金，也将成为监管、改革的重点，要在考核机制上动刀子，把基金管理费与投资者利益捆绑在一起，但小镇认为改革大概率是浅层的，作用并不大，不过是缓和矛盾；更值得期待的还是丰富公募基金的产品谱系，2025年公募基金产品创新将加速，推出适配国家政策导向的新产品，指数型产品还将继续加大发展。

以下为分析正文：
从2024年股市来看，观察2025 年股市走向，最关键的有三点；一是增量资金从哪里来，二是国家队如何出手，三是股市结构调整。

增量资金来源有二：
一是国家开闸放水。小镇在星球1月15日《央行倾向于一季度暂不降息》,2月8日《做好迎接长期的准备》已经谈了，国家仍然尽可能珍惜常规货币政策空间，延迟开启量化宽松等放水措施，还要把更多资源投入到科技创新和产业升级，因此不可能放任金融等虚体经济过度繁荣，否则谁还愿意搞科研和产业?
二是中长期资金入市。这也是目前市场机构非常关注的，网上也有很多夸张的说法，但小镇在1月26日《中长期资金入市，道阻且长》已经分析了，综合考虑2025年能够入市的中长期资金大概也就1万亿左右，有作用，但很有限。

这也可以从证监对2025年工作的总体部署看出，2025年总要求是“稳字当头”,上次提出“稳字当头”还是2022年初，重点是稳定市场，而不是让市场波动太大。这也是2025年整个中国经济战略的总要求，所有部门必须贯彻执行，要做好防冲击准备。去年12月14日，证监传达学习经济工作会议精神，第一位的要求也是“突出维护市场稳定这个关键”。所以增量资金就不用过度期待了。

然后是国家队。在1月的年度会议上，提到了"加强战略性力量储备"这句话，指向的显然是国家队。过去国家队一直被寄予厚望，这一轮国家队从2023年三季度入市，一度发挥了作用，但是后来“聪明”的市场机构很快看破了国家队的操作逻辑，于是借助国家队入场退场疯狂套利，国家队已难以为继，到2024年8月，国家队已经很难阻止市场急速下跌，于是就加速催生了9月24日的一揽子政策。

政策发挥的效力、持续时间都在迅速萎缩，但又不 能不出手，战略性力量仍然是稳定市场的核心，在遭遇突然冲击的时候，也只能靠国家队顶上去。所以既然国家队效力在下降，那就需要加强国家队的力量，也就体现在“加强战略性力量储备”。但注意是“储备”,参考过去一年半的国家队干预，事实已经宣告国家队常态化干预的失败，常态化只会导致市场机构把国家队当成肥羊，甚至加剧了股市的动荡。因此，从去年三季度开始，国家队出手变得更加隐蔽、次数也有所减少，这一趋势也将持续，国家队未来将作为战略力量，只在有可能危及市场稳定的极端状态下才会出手。

2月吴主席的撰文也证明了这一点。1月会议通稿还提到了“战略性力量”,2月的文章就调整为了“丰富稳市政策工具，做好增量政策储备”，国家队这么关键的表述直接删除了，显然是弱化了对国家队的期待。这种字词的修改不能单独作为政策趋势的证据，但结合2023年三季度以来国家队干预效果迅速下降，甚至出现干预失败，而近半年国家队下场次数明显减少，两相对照，就可以得出论断：2025年除非发生重大问题、重大冲击，有可能危及资本市场稳定，国家队将会大幅减少出手干预。因此今年股市，常态情况下，除非发生重大变化，哪怕跌一阵子、突破某个点位，也不必过于期待国家队会干预了。

第三点是股市结构调整，这是未来资本市场改革的重点。中国资本市场规模已经很大了，30多年下来，中国股票市场、债券市场规模都已跃居世界前列，商品期货交易额也跃居世界前列，总量增长已经接近极限，按照吴主席的话“已经到了从高速增长向高质量发展加快转变的阶段”，当前资本市场健康稳定发展的主要矛盾也已经变为“发展质量问题”。

去年初吴主席上任，一开始把保护投资者权益放在工作首位，暂缓、整顿IPO，但中国股市的核心任务是支持实体经济、科技创新和产业升级，要为优质企业融资，所以2024年中，证监的工作重点已经从保护投资者权益重新转向支持企业融资，要打通优质企业上市的通道，如果不能保证企业融资，证监这个部门的价值就大打折扣。而且暂停IPO 也带来很多负面问题，投资者对整顿 IPO 的态度也从一年前的支持到毁誉参半再到负面居多，必须重启。

但未来绝不是过去只顾做大规模的粗糙做法，IPO这一做法已经证明失败了，既没有搞好资本市场，也没能让证监系统提高地位。未来要“做优做强”，存量已经基本到顶，未来重点是盘活存量，也就是说股市需要“腾笼换鸟”，保持总量基本不变的情况下，加速退出，换一批真正代表中国未来的优质企业上市，这也是很多中小投资机构期待的。

因此2025年乃至未来的IPO的规模会缩小，基本保持在当前水平上。事实上从2024年1月至今，上证、深证上市公司的数量基本稳定。于是2月撰文出现了一句话“引导标杆性高科技企业登陆A股市场”，从吴主席行文来看，大概率证监已经确定了“标杆性高科技企业”标准，甚至拉了潜在目标企业清单。

但问题在于，中国现在资金太多、投资机构太多，好的企业、好的项目根本不缺钱，要么自家前途非常光明，要么现金储备多的愁怎么花。比如当红的DeepSeek，还有华为、字节等优质企业，会愿意上市吗?对这些公司来说，根本不缺钱，如果有融资需要，也大可以在一级市场获得充足的资金，又何必上市自找麻烦?

所以证监要顺应决策层对科技创新、产业升级的偏好，未来方向也已经确定为“做优做强”，要推进资本市场的“腾笼换鸟”，把那些劣质股票赶出去，但上市企业数量也是证监的核心政绩之一，也关系到证监与各地、其他系统之间的合作，在大多数地方，辖区企业上市仍然是一大政绩，想要清理难度很大。所以“腾笼换鸟”也得先找好新鸟，才可以增1减1。

这又回到上面的问题，在当今中国，优质企业根本不缺融资，全国各地也都在抢夺科技型企业，比如最近杭州六小龙爆火，南京一把手就带头发问"同样是长三角中心城市，南京为什么没有产生人工智能‘六小龙’?"。

最终要看证监能拿出多少资源吸引优质企业，支持优质科技型企业也成为接下来证监的工作核心。2月7日证监发布《关于资本市场做好金融“五篇大文章”的实施意见》，几乎可以视为一份支持科技型企业发展的专门文件，对科技企业爱得非常深。该意见提出了一系列措施，具体就不在这里赘述了，毕竟意见虽然写了，关键还是要看如何落实，证监掌握的政策、资金资源也有限。不过有一个点很值得关注，那就是“支持境外上市优质科技型企业回归A股”。

目前中美之间的摩擦加剧，而美股也出现不稳迹象，客观上存在吸引优质科技型企业回归的条件，而从证监的角度，国内优质科技型企业不怎么缺钱和政策支持，想要吸引这些企业上市难度比较大，而今年又需要拿出支持科技发展的政绩。那么吸引境外上市企业回归，就更为现实了。但注意这次不同于2018年，2018年证监曾推出CDR政策推动互联网巨头回归A股，这一政策已经失败了，而未来要推动的“境外上市的优质科技型企业”，跟2018年想要吸引的目标大为不同。可以关注2025年将有哪些境外企业回归国内上市，借此可以体会，在证监的心目中，到底谁是“标杆性高科技企业”，由此也可推测，国家未来看重的领域。

总而言之，小镇在《2025:激荡的变数之年》已经分析了，2025年变数很多，面对这些变数，稳定是第一位的，先防守再进攻，因此资本市场根本改革已经提上日程，但不要指望一两年内就见到大牛市的到来。在目前情况下，牛市必然是短牛、疯牛，这不符合大方向。
