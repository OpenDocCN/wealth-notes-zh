
# 与朋友聊医药集采

我跟高中同学聊，三甲的副主任医师，集采后在手术台上断了两把集采手术刀了。坦白说，手术刀用断应该是不规范操作导致的，特别是装卸刀片。

子明：不就是拼多多模式嘛，现行阶段的唯一选择，只能靠市场慢慢清退各种雷碧。不要吹拼多多的仿品质量有多好，也不要一闷棍就把拼多多打死。

拼多多模式没问题我们也支持也是现阶段唯一的解决方案。但你不能睁着眼睛说拼多多都是好货，质量没问题。我身边就没有。

以现在的kpi模式，只能到出了大问题或者问题多到一定程度才能整改了。

不是我牛逼，是真不治病，我自己吃的降压药换回原研就好了。类似于当年啃树皮，这是当时为了发展工业的唯一选择，但是你不能昧心说啃树皮比大白米饭好。当然，也不能说大家都要吃大白米饭，因为国家的确已经养不起了。

对普通民众，他们除了吹好做安抚，也没别的招。对于自己，可以选择思考，可以认命相信，也可以选择堵住耳朵。

子明：把经济发展起来，蛋糕做大，才是唯一的解决之路。不会搞经济呀，只会修桥铺路，会盖房子，搞大蛋糕的本事没有，切蛋糕的本事很大。宁要社会主义的草，就是那时候的宣传语吧。医保不支撑都吃原研药……能牺牲一部分人当代价，国产药也不是不治病，主要还是质量不稳定。

子明：只要持续下去，有批评的声音，迟早也会跟拼多多一样，白牌拥有品牌的质量。

问题就是有人批评，他把批评的人解决掉了。这事吧别太信他们，时不时有点声音，他们才不至于太过分。

子明：拼多多不就是靠无理由退货这种极端的批评，做起来的么。这个时候体现中药的优势了，吃下去没啥效果你不能说我草药质量不行。

避免不了一部分人当代价，但要因势利导，完善监管和让国产药进步起来，出清差的，别搞到最后劣币驱逐良币，还是拼关系的老路。他们选择了这条路，所以我们只能自强赚钱去买原研药。

就像他们要包庇油罐车，我们就只能去赚钱，找能吃健康油的路子。这么大的政策这么大的利益不打个好几个来回能好好谈么。看完哪吒,还学申公豹他爹放下屠刀跟着去的人，那就只能认命当丹啰。

任由效果不好的中成药去薅医保羊毛，那就不是市场化出清，市场化出清是乡镇医疗，便宜的抗生素+激素。现代西药最讨厌了，吃下去就管用，效果立竿见影，所以容易被人抓住把柄，实在是太讨厌了。

集采是国策。
