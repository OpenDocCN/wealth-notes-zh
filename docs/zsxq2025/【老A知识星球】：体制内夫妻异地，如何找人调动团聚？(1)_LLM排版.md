
# 体制内夫妻异地，如何找人调动团聚？

提问：匿名用户 老A

老A, 您好。我是垂管单位的市级部门科员，想咨询下夫妻异地调动的事。今年年初单位有3个夫妻异地的调动到省会城市。我知道的时候是2024年底，过来考察他们3个，我也是夫妻异地，当时我就给人事上打了电话问是不是有什么条件不符合，回复说没有什么条件要求，夫妻异地调动政策一直都有，这次是省局下的名单，他们按名单通知准备材料，名单没有我，说是要找人。后面听说是省局老大要走了，走之前解决了这一批，全省差不多200多人。所以我想请教下A师，像这次机会我是错过了，以后还有机会吗?像今年刚调动完，还会启动吗?再启动是不是新领导来了才启动?我梳理下我的关系，有如下：1.省局对口触室的触长，去年给了我去外省培训的机会，知道我，对我也认可，但是我跟他没有进一步关系，只是工作上。2.本局的分管领导(副触级),去年两次出差，回来都给她带了礼物，对我也认可。这个夫妻异地调动的事我之前咨询过人事科科长，结果人事科科长就给我的分管说了，有一天正好和分管一起出去办事，分管领导问我夫妻异地几年了，我说6年了，她说说短也不短，还有机会，说要找人，至少在省局人事触触长这个级别。3.姐姐的好朋友老公是我们系统的，从我们市出去的，现在另一个市任一把(正触级),这人很有前途，提拔很快。以上至个是我目前仅有的关系，想请教下A师，哪个关系我可以用?该怎么提这个事?

当后续还会有机会。是否启动、啥时候启动，有时就是领导一句话，有人关系硬点，领导可能会安排特事特办，大家关系都不硬，那就等一等，等到有一批了，再统一弄。

二、这个关系，他大概率不会帮你用力。
