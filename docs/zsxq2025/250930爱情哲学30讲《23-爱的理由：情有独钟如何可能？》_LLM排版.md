
# 23 爱的理由：情有独钟如何可能？

你好，欢迎来到《爱情哲学 30 讲》，我是刘擎。

假如你听说一位好友恋爱了，可能会问“你为什么会爱上这个人？”或者“你爱 ta 什么？”常见的回答是，因为这个人具有一些我喜欢的品质。这种回答，在关于爱的哲学讨论中，被称为“品质论”（Qualities View ）。

但对于品质论，当代哲学仍然存在着分歧和争论。

这一讲，我们将更深入地讨论上一讲提出的问题：如果爱一个人是基于他的品质，那么心爱之人的“不可替代性”又如何解释呢？

## 品质论

探讨这个问题，我们先要清楚“品质论”是什么。

品质论的核心命题是：我爱上某个人，是因为 ta 具备某些值得爱的品质。这些品质可能是外貌动人、性格宜人、聪明智慧，或者是道德品格、艺术天赋，乃至一种整体的人格气质。

品质论的吸引力在于，它赋予爱情一种理性上的可理解性。但有人会说，“爱是一种感情，甚至可能是一种感觉，根本不是理性的”。在我看来，这种说法陷入了神秘论或不可知论，在理论和实践上都不值得认真对待。

在理论上，这种观点错误地将感情与理性完全对立起来。但心理学和其他社会科学的研究都发现，感情并不是无缘无故的，而总是包含着对直觉感受的认知、分析和判断。在这个意义上，感情的反义词并不是理性，而是麻木。

在实践中，这种极端神秘论的主张，会对爱情最低限度的稳定性造成威胁。

想象一下，恋人和你满怀深情地晚安道别，第二天醒来，你发了一句“早安”，收到的回复却是“对不起，我对你已经没感觉了”。你惊讶地问“为什么呀？”得到的回答是“不要问为什么好吗？爱根本没有理由，爱完全就是非理性的”。

面对此情此景，你会作何感想呢？

而品质论的前提信念是，我们坠入爱河时，是有根据的，并非随意或盲目的。感情的生成、发展和消逝，都在原因，虽然原因可能是复杂、多样和隐秘的。有时候，当事人确实不清楚自己为什么爱上这个人，或者误会了自己爱的真正理由，比如，你可能以为自己爱上一个人的财富，但其实你爱的是这个人出众的才华。

品质论相信，无论理由是不是被准确表述和认识，它都是存在的。因此，品质论就是对“爱的理由”做出解释。

## 品质论的“可替换性”困境

如果爱的理由完全依赖于品质，那么，一个具有相同或更好品质的人，似乎就可以替代当前的恋人，但这一推论，与我们对爱人“不可替代”的信念格格不入。这就是所谓的“可替代性难题”（trade-up problem）。

让我们做一个思想实验：有一个女生与一位海员相恋。这位海员不仅英俊潇洒，而且通情达理，懂得关怀、体贴和温柔，善于聆听，拥有诸多优点。他们相恋三个月后，海员接受了是一项远洋航行任务，需要六个月才能返回。在那个通讯不便利的年代，书信难以送达，电话也不普及，只能偶尔收到一些电报信息。
