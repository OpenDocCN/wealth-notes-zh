
# 逃离大城市，去小城市养老是一个伪命题

## 猫哥原创

这几年网上很流行一个观点，大致是将大城市房子卖掉，然后去小城市养老，小城市房价低，物价低，环境也好，是一个比大城市养老更优的选择。这个观点看起来很有道理，其实是一个伪命题。

### 为什么？

原因很简单，我们的生活品质其实非常依赖政府供给的公共产品的水平。小城市每年财政收入大致就1-2个亿左右，而大城市财政收入一般能达到1000亿以上。小城市与大城市相比财政收入相差几百上千倍，其提供的公共产品的差距极大。

首先是医疗资源。这几年随着经济的发展，国内县一级医院硬件水平是上去了，但是软实力（诊断与治疗水平）依然与大城市有很大很大的差距。

举几个例子。几年前，我的一个亲戚因为脚肿送到县医院治疗，当地医院判断是肾病引发的脚肿，然后就按照肾病治疗，结果治了十几天病情没有任何好转，果断送到重庆一家三甲医院，最初是送到肾病科，但是医院会诊之后，判断是肝脏才是主要矛盾，然后转到肝胆科。治疗一段时间后，肝脏指标正常，几天后脚肿就消失了。

说实话，对于中老年人而言，一般大病都是综合性疾病，心血管、肝肾等器官多少都有些毛病，真到发病时，那个器官的病症才是主要矛盾，这就很考验医生的诊断经验与水平，这方面小城市医院与大城市医院的差距不是一般大。

再来一个。今年4月，我的一个老师59岁因为轻度肝硬化引发的静脉曲张吐血，被救护车就近送到一家区级医院，还是个2甲医院。晚上9点送到医院，凌晨4点人就没了。事后我找相关领域的专家咨询了一下，如果我的老师是送到三甲医院有经验的医生来治疗，这个病症虽然凶险，但是大概率命是能保住的。这还是大城市区一级的2甲医院，与3甲医院就有这么大的差距，大家可以想一想大城市3甲医院与县城医院差距有多大。

然后就是食品安全的问题。按照普通人的思维，小城市距离农村更近，食品应该更新鲜更可靠。真的是这样吗？

事实是食品安全与供应链距离关系不大，真正决定食品安全的是政府的监管。大城市因为财力充足，所以对食品安全监管非常给力，甚至能直接追溯到食品供应链的每一个环节，也就是产供销生产的每一个环节都有可靠的监管。在这样的背景下，大城市居民对市场供应的食品不管是超市农贸市场还是餐馆的食品完全可以放心买放心吃。基本不会出现问题。

而小城市虽然国家也很重视食品安全问题，也投入巨资完善对供应链的管理，但是限于当地财力薄弱，监管还是有漏洞的。几年前我到一个小城市出差，早上起来在县城最繁华的街道看到一家卖杭州小笼包的，价格很便宜，基本是大城市价格的1/3，然后买了一笼包子，拿着一个包子吃了一口，立马就吐出来，原因就是包子里面肉的味道有明显的异味，大概率是死猪肉或者病猪肉。

几天后路过当地一个乡镇的农贸市场，其中有一个区域是专门卖腊肉香肠的，还没走进这个区域就能闻到一股很强烈的臭味，很难想象这些腊肉香肠使用的是什么肉，反正是我吃了一辈子都没有闻到的臭味。

第三个就是环境问题。小城市因为基础条件差，在招商引资方面与大城市有天然的劣势，所以小城市引进的产业一般都是小型企业，包括不限于小型水泥厂、小型化肥厂、小型制药厂等等，这些小型企业同样存在工业污染的问题——某种程度甚至可以认为，这些小型企业之所以能在市场上与大企业竞争，主要优势就是成本低，比如工业污染的治理投入成本低。

而小城市因为财力薄弱，对工业污染的监管也很有限，这就导致小城市的产业环境污染远远大于大城市。所以与普通人认知完全相反的事实就是很多地方小城市环境质量还远远不如大城市。

我自己曾经到某个小城市出差，因为当地引进一批化工企业，整个城市都弥漫着一股刺鼻的味道，我自己觉得很难受，而当地人居然习以为常。

还有一个教育资源问题，小城市与大城市差别也极大，当然今天我们谈的是养老问题，就不展开谈教育了。

除了医疗、教育、食品环境等公共产品的供给之外，小城市商业与娱乐配套与大城市差别极大。现代人越来越追求生活品质，除了吃喝拉撒之外，总有点娱乐性追求吧，但是在小城市这方面的供给十分匮乏。

小城市没有像样的健身房，没有像样的康养设施，没有像样的娱乐配套，当大城市餐馆都开始配置精致的歌舞表演的时候，小城市甚至没有像样的文化活动。

我们养老图个啥？无非是两个方面。一个是拓展生命的宽度，让我们生活过得更有质量，一个是力争延长我们生命的长度，让我们更长寿，这两方面大城市都碾压小城市。

中国人平均寿命目前大概是78岁左右，大家猜猜超过这个平均数是什么地方？是的，超过我们人均寿命的毫无例外都是大城市，包括北京、上海、广州、深圳、成都、武汉、南京、杭州等等大城市人均寿命都在82——84岁左右。

而低于平均寿命的是那些地区呢？是小城市或者就是农村。

个人估计大城市居民与小城市居民寿命可能有8-10年左右的差距。

所以，大家不要被小城市房价低，物价低所迷惑，认为这些地方最适合养老，其实生活的质量归根结底还是城市公共产品供给水平决定的。

最适合养老的还是大城市。越发达的地区生活质量越高，越落后的地区生活质量越低，不管是年轻人还是老年人都是如此。
