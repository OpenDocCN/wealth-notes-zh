
# 体制内乡镇提拔副科，要找强关系还是弱关系帮忙？

近期县里有提拔副科的机会，本人是乡镇的党建办主任，近期新任的县委书记对我还算比较认可，因为之前走动少，属于弱关系。本次提拔的主要竞争对手可以通过中间人找到县卫生局下属机构的一个领导，竞争压力很大。目前有三条资源可以努力，1是老领导、贵人，目前是县政府的副县长A。2远房亲戚、县组织部所属机构的一个领导B。3是老领导C。AB都是强关系，C是弱关系。A会尽力帮忙，会找中间人给党建办主任推荐。B答应有机会直接给组织部推荐。C和县委书记私交好，但与A有过节。C也曾表示愿意帮忙。

但考虑以后长远，目前仅与A进一步联系，没有与C联系。

想请教A叔，在竞争如此大的情况下。
问题1:主攻方向是以A为主？还是AC为主？B还有必要吗?
问题2:新县委书记在本次提拔中话语权可能较少，仅汇报想法、踏实工作即可?

我个人建议找A就够了。在我看来，乡镇的党建工作办主任，提拔成班子成员是大概率事件，早晚得事。

你不要盯着所谓的竞争对手，县里动用，提副科，是全县一盘棋，并不是在你们乡只提一个。如果两个都优秀或两个关系都很铁，动议两个，也是很有可能的。建议以A为主，B可以问他寻个建议。毕竟都是亲戚，这段时间多走动一下，就此事多让他们想想办法能出主意的出主意，能打招呼的打招呼。新一把，尽量贴近一些，但这件事指望不上他，贴近，是为了不让他说坏话、背后作梗。

老A，《职场晋升指南》作者。
