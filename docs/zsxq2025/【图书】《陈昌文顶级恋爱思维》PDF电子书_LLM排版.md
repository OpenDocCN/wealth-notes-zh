
# 陈昌文顶级恋爱思维

一般的男女关系只有博弈，非常恶心。只有顶级的男女不会进入到博弈状态，进入到爱，与情感！普通人的恋爱，情感，婚恋都是博弈与斗争，非常恶心。男女之间的长期关系，本质上是博弈关系，也就是会付出的越多，谁越珍惜谁，付出少的人有优势。

富豪男人，牛逼的人，优质的人，要注意，您对这个穷的，不行的人越好，您越爱他，他越可以背叛您，因为他背叛您的成本越低，所以他能拿捏您，在这件事上，我建议大家对等谈明白。

**我付出什么，您付出什么？**

这种爱情，感情，是真感情，否则，就是假的，爱情这件事很有意思，谈的越明白，就越是好感情，谈不明白，就是背叛，诈骗。也就说，您越是不愿意谈明白，您就会被对方诈骗。对方会说，爱情就是无条件的，不断的诈骗您的财富。而谈明白了，我给多少，您给多少，我们必须对等，反而是最好的。当然，如果您真想跟对方一辈子，您就可以让对方付出一切，彻底爱上自己。

**不过您相信您自己的人性吗？**

如果您不相信自己的人性，对于有良心，善良的富豪，富婆来讲，就是谈明白，对等付出。

记住，是在客观事实上对等，而不是心里对等，因为，富人的财富远远多于穷人。心里对等，会让穷人占便宜，最终抛弃富人。

谈恋爱，情感是个狠心的智慧，本质上，您不爱任何人，您对谁也没有情感，只是想跟更优秀的人在一起，只是想跟能匹配自己的人在一起，如果上头，就证明您的思维错误。所以，遇到不合适，不合理的人，直接抛弃就行了。不要有什么负罪感，更不要认为有什么爱情，情感，只有当一个顶级优质的人理解了，爱情，情感的博弈，还很善良的时候，然后，又非常狠心，能对等付出的时候，这个爱情，才是门当户对，大家都不亏。

不要别不优质的人，低劣的人框架，套路，恋爱，情感，就是玩火，一不注意，引火烧身，这是天下间最难的事情，谈不好，正常，没有几个人能谈好这个东西。
