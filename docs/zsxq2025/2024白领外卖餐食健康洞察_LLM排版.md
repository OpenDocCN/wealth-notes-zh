
# 2024白领外卖餐食健康洞察

食物是健康的基石。在当前党中央和国务院提出的“以健康为中心”的大政方针下，如何吃得健康，已成为人人都要关心的重大问题。《健康中国2030规划纲要》《国民营养计划（2017-2030年）》《健康中国行动（2019-2030年）》等重大政策文件都把营养和合理膳食放在重要地位。作为自己健康的第一责任人，每个人都需要懂得如何吃得合理、吃得营养，且落实到行动，并持之以恒。而餐饮行业，作为餐食的供给侧，不仅需要提供品种丰富多样、富含营养的餐品，而且有责任通过餐食服务给广大消费者提供如何合理选择和搭配餐食的知识。

自新冠疫情以来，餐饮外卖的消费者人数以及频率持续、快速上升，已经成为大多數家庭一日三餐不可或缺的组成部分。与在外就餐不同的是，不是偶尔为之，而是经常性的，有些人甚至是天天都点外卖。因此，外卖对营养和健康的影响就成为了一个值得研究和探讨的课题。

研究必须要有数据，而外卖平台所积累的数据则是开展这方面真实世界研究的宝贵源泉。摆在读者面前的这份《2024白领外卖餐食健康洞察》，是有史以来第一份研究外卖行为对消费者营养和健康影响报告。选择白领人群作为研究对象是因为他们点外卖的频率高，从而对营养和健康的影响比较明显。然而，在白领人群中发现的问题对于总体外卖人群也具有相当的代表性和参考价值。

从《2024白领外卖餐食健康洞察》中可以了解，在白领外卖人群中，有相当一部分人选择的品种多、营养比较均衡；另一部分人则喜好肉类和油脂；还有一部分人则以吃饱为目标。总体而言，外卖场景下比较普遍的问题是人们对粗粮和蔬菜品类点选少，与营养均衡、合理膳食的目标相比，尚有不少差距。可喜的是从数据分析中可以看出白领外卖人群中有相当大一部分人有讲究餐食多样化和营养健康的认识和需求。从而提示，餐饮行业应该满足这部分消费者的需要，同时引导这方面认识不足的消费者提高认识，改变行为。除此以外，这份报告中还有很多有意义的发现，需要进一步收集信息和数据开展更深入的研究，包括能量、碳水化合物、脂肪、蛋白质以及微量营养素的摄入情况等。洞察的进一步深入，可以对外卖餐食在营养和健康方面存在的问题作出更精准的分析，为进一步开展消费者科普教育和指导、餐饮行业健康化提供更好的技术支撑。

感谢为《2024白领外卖餐食健康洞察》提供和分析数据以及编写人员的开创性工作，并期待看到第二份更深入的有关外卖餐食健康洞察的报告。
