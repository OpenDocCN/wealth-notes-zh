
# 再关注一只消费股

我们近期玩情绪股的主线是两条，一是医药，主要是创新药；二是消费，特别是新消费。

上个月重点提到的绿叶制药近期遇到了主升浪，此时持股即可，不要做波段，否则容易踏空。另一只恒鑫生活马上要除权，看看除权之后能不能继续涨，好在我们买的位置比较好，是长期通道的下轨，目前还有不少利润垫。

今天我要提另一只新消费股，这是我在研究AH股比价的时候发现的，我们知道，很多公司既有A股又有H股，两地的价格绝大多数情况下是不同的， 一般来说A股贵，H股便宜，那么差价最大的是哪一只呢？我们来看列表：

排名第一的叫安德利，溢价-74%的意思是该公司的H股比A股便宜74%，目前安德利的A股是61.93元人民币，而H股是17.12元港币。为什么两地价格会差这么多？这引发了我的好奇心。

安德利在港股的名字叫安德利果汁，这家是中国最大的鲜榨果汁生产商，所以你对安德利的品牌认知不深，原因是他主要做出口业务，国内业务不多，且主要做B端市场，就是卖给企业，比如可口可乐的果汁供应商就是他家。本来公司股价一直在10港币上下波动，但由于今年一季报非常好，所以公布财报的第二天股价就出现了爆发式增长，见下图：

深入研究之后发现，这家公司非常适合做养老股，上市至今没有一年亏钱，且年年有分红，当然近期不仅是养老概念，还叠加了新消费情绪，以下是该股的一些亮点：

1. 国内果汁龙头一共是四家，分别是海升、恒通、安德利和国投中鲁。其中海升+恒通已经接近破产倒闭了，空出来的市场被安德利和国投中鲁瓜分，而安德利吃掉的份额最大，比如海升的厂子都给了安德利；
2. 安德利的H股持续回购，称得上是回购王，你要看金额数肯定不能和腾讯、阿里比，但人家回购的比例高啊，他家的H股从1.4亿股持续回购到现在，就只剩下7000万股了，整整减少了一半。你说哪家上市公司能够把股本买回来一半呢？这家公司只搞果汁，从来不搞其他业务，一旦手头有余钱就回购股票。这么多年下来，H股的流通股越来越少。本来就是小盘股，现在盘子更迷你了；

3. 年轻人喜欢新消费，他们不喝酒了，改喝果汁，特别是NFC果汁，NFC是英文Not From Concentrate的缩写，中文称为“非浓缩还原汁”，是将新鲜原果清洗后压榨出果汁，经瞬间杀菌后直接灌装（不经过浓缩及复原），完全保留了水果原有的新鲜风味。原本公司主要做出口，现在国内的销量也越来越多，你看现在超市里出现越来越多的NFC果汁了，同时，越来越多的奶茶店也开始卖NFC果汁。他们的供应商很可能就是安德利。你只要喝一下就知道NFC果汁不同于传统的果汁；
4. 今年第一季度公司业绩出现了爆发式增长，预计这种增长会维持全年，这也是公司股价在5月6日暴涨的原因；
5. 公司在5月6日大涨之后，持续调整到昨天，今天的一根大阳线预示着调整很可能已经结束，这个位置也是我今天入手该股的位置。

由于有以上利好，我认为目前的位置非常适合建立一个观察仓，所谓观察仓就是当你发现一支好股出现了买点的时候，无论如何先买一个底仓，然后只要股价涨了，那么回调就继续加仓，即当长期通道向上的时候，每次触碰短期通道的下轨就可以加仓。
