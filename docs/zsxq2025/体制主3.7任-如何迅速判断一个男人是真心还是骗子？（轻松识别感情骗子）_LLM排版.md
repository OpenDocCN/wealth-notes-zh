
# 如何迅速判断一个男人是真心还是骗子？（轻松识别感情骗子）

有的男人是所谓语言的巨人、行动的矮子。光说不练假把式。女人如果遇到这种人，最好的办法就是选择无视。久而久之，他便会乖乖退下。毕竟，情感骗子的耐心也是有限的。当他在你这里得不到回报，他会毫不犹豫地撤退，将欺骗的目标迅速转移。

甜言蜜语多，实际行动少。这个绝对是感情骗子的试金石。看一个人是不是真的爱你，你不要听Ta说什么,你得看Ta做了什么。

## 二、警惕特别完美的人

如果说一个人的价值是恒定的，那能匹配到的伴侣也差不多是一个恒定的数值。就像1+1=2, 它不太可能出现1+1=10的情况。世界上真的有这么完美的人吗？没有。你遇到的白马王子可能是个感情骗子。

也就是说，可以供给一个女性选择的伴侣，但条件都是差不多的。他们会有各种各样的缺点，比如：A长得帅，但是没钱；B有钱，但是长得丑；C有钱帅，但是很花心。如果此时出现一个长得帅、有钱、专一，并且非常着急的想要和你发生性关系，那这个人极大可能是骗子。

在体制内的人员非常好判断。但是体制外的通常比较难判断一些，一个男人工作自由，既不是朝九晚五的上班族，也没有固定的上班时间，多数号称跟朋友合伙开了公司、做工程，但资产不详、身家不露。你每日没看到他忙忙碌碌，却只看到他清闲的与你暧昧调情、约会聊骚，那多半他可能是个失业N年，混吃混喝的骗子。

如果你和他发生了关系，那么,他的生意就突然忙起来了，或者告诉你资金出现问题，急需钱，各种理由问你拿钱。

## 四、若即若离、经常撒谎

若即若离，是玩弄感情的人经常使用的手段。一方面，他们会让对方非常没有安全感，从而让对方在他们身上投入比较多的精力；另一方面，就算他们是时间管理大师，也很难分神到多个对象中、完美地回应每个人。

像这种经常找不到人的情况，要么有固定的伴侣，要么有很多个不固定的伴侣。总而言之，经常消失的异性一定有鬼。

## 五、感情骗子都有一个特征：急于求成

骗子会在一星期乃至1个月之内迅速冷淡下来。骗子都是讲究效率的，用最短的时间割做多的韭菜。有种男人在你们刚认识没多久，就会急着要和你发生关系，这时你必须提高警惕，不要轻易上当。除非你也贪图他的肉体，你们只是纯粹的保护关系。但很多女人往往会通过发生关系而慢慢动了真情，这才是真正要命的事情。

当你发现一个男人和你认识很长时间，该做的也做了，却总是不肯主动愿意带你回家见他的父母和家人，这说明他只是把你当成他的备胎，他没有真正要和你在一起结婚的打算。这样的感情骗子最常使用的招数就是不断给你洗脑，借口事业忙或是挣钱少，不能给你更好的生活，所以一再逃避婚姻。当你有意无意提及结婚的事情，他便会想方设法转移话题。当你对他还有利用价值的时候，他会好言好语地哄着你，只有当他找到了比你更适合的对象时才会将你一脚踹开。
