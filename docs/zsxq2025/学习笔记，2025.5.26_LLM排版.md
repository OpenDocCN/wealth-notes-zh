
# 学习笔记，2025.5.26

1. 好习惯的威力大得惊人。养成一个好习惯的优势在于，以后行动时，就再也不必进行内心的权衡。不再消耗能量去思考是否要做这件事，你只管去做就行了。

2. 敢于谈钱才能赚到钱。穷人之所以很难翻身，就是因为他们总是无视“利益交换”的客观规律和人性本质。他们喜欢跟你谈善良、谈情商、谈感情、谈道德，就是不跟你谈钱谈价值交换。

3. 好朋友是会渐渐疏远的。每段人生我们都会碰到不同的人，会出现几次“相见恨晚”。同样的，也会莫名其妙就不联系了，渐渐疏远。从前我会怀疑自己是不是做错了什么，是不是有什么隔阂。后来才知道，其实这就是人际关系的更迭。大多数朋友都会逐渐疏远，或早或晚，慢慢接受就好。

4. 人只能被筛选，不能被改变。人的性格，跟基因、经历、教育、成长环境息息相关，谁都不可能回炉重造。不管是谁，一旦陷入到“要改造另一个人”的境地，生活里就会出现个巨大的黑洞，吸时、吸力、吸钱。所以，在与人交往时，在选择朋友、爱人、合作伙伴时，最重要的是筛选，而不是妄想着对方“能被自己改造得更好”。

5. 永远不要让自己的情绪失控。一旦发现自己情绪失衡，请务必自救脱困。不要让那些烂人烂事牵绊你太久，深陷其中。过分被动，过分执着，都是心魔。该放手的始终是要放下的。调整自身能量场，坚定前行，好运自会相随。

6. 癌症本质是老年病。浙江全省癌症发病率增长9.68%，为什么会出现这个结果？在话题评论里，各种各样的猜测都有，比如有些人说空气污染，有人说核污染，有些人说食品不安全，有些人说是新冠摧毁了人的免疫系统，这些都是不正确的！最大的原因，就是民众富足，医疗保健完善，导致民众寿命增长所引起。

7. 人情世故不是必修课。如果你太单纯，就不要花太多精力去搞人情世故那一套。你不适应，也没那天赋，不要老想着看清别人，你能认清自己就够了。安安静静地搞钱就好，其它的都不重要。

8. 内行都是拿时间堆出来的。所谓内行，就是做的次数足够多，犯的错误足够多。各种可能性，都尝试过了。每一个细节，都打磨过了。该走的弯路，都走过了。该交的学费，也都交过了。

9. 祛魅才是一个人变强的开始。很多人唯唯诺诺，一直自我否定，压抑自己的内心，就是因为他把别人想象得太强大，太完美。其实，任何一个人、一个组织，当你深入了解之后，你就会发现，其实只是表面风光，背后都是漏洞百出。

10. 该离开时绝不迟疑。当你离开错误的环境，离开有毒的人和事之后，你就能体会到能量回归的快感。心情也会变得更加安宁、沉稳。从此以后，能干扰你的事情越来越少，良性磁场开始回归，周围的一切都开始变好。
