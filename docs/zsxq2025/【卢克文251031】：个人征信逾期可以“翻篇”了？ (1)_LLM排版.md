
# 个人征信逾期可以“翻篇”了？

央行传来好消息，特定逾期可以申请“信用修复”

这不是空穴来风。10月27日，金融街论坛年会上，央行行长潘功胜透露了一项正在研究中的一次性信用救济政策，引发广泛关注。

到底什么样的逾期可以“翻篇"?谁又能真正按下“信用修复”的快捷键?

首先要说明，不是所有在疫情期间逾期的人都能得到这次征信修复的机会。央行行长潘功胜指出，政策主要是帮助那些受疫情影响发生逾期、但后来已还清欠款的人。

具体来说，必须同时满足三个条件才有修复资格：
- 逾期发生在新冠疫情期间；
- 违约金额在规定标准以下(具体标准待公布);
- 欠款已全部还清。

信用修复≠违约记录“一笔勾销”，而是将符合条件的记录在征信报告中隐藏起来。这项政策预计明年初开始实施，很多曾经受不可抗力“暂时失守信用防线”的朋友，有望“上岸”了。

政策初衷是好的，可为什么很多人不买账?

仔细想想，有几个现实问题：
首先，它帮的是“已经上岸的人”：按照现行规定，逾期记录在还清后满5年本来就会自动删除。疫情封控从2020年开始，不少还清欠款的人，逾期记录长的已经快五年，短的也有两三年，这个政策无非是让“污点”提前几年消失而已。

而这个政策救不了那些“还在溺水的人”。对于至今仍无力偿还债务的群体，这个政策起不到任何作用。

目前全国约有7亿人有信贷记录，其中接近3亿人存在不良征信，占比高达42.8%。(数据来源于2025年《中国居民负债情况调查报告》)

这个占比还是很吓人的，差不多每两个借钱的人里，就有一个信用有问题。

这也解释了为什么现在各种刺激消费的政策效果都不太明显：这么多人逾期，说明生产生活都不能保障，还怎么消费?

所以，出现了一种尖锐的批评：这个政策没法帮大家解决困难，它只是想让更多人恢复“贷款资格”,好让银行有更多业务可做。

很多人一提到征信报告就一肚子火，觉得这就是银行用来拿捏人的工具。

想贷款先查征信。银行打开报告一看，一句“不合格”就能把你拒之门外。

就算勉强批了，如果征信有瑕疵，利率立马就得上浮10%到30%。算算看，房贷三十年还下来，多花出去好几十万，一辆车就这么白白没了。

原来，银行就是把你的征信翻来覆去地挑毛病，好理直气壮给你的利息“加价”。

但真相正好相反：这套征信系统，其实就是在逼着银行“区别对待”,防止它无脑放贷。

大众印象里，银行是靠低息吸储、高息放贷，吃息差挣钱的。但银行玩的是现金流游戏，如果你办一笔30年的房贷，难道银行要慢慢熬30年才把钱赚到手吗?

假如，你从银行贷出300万，它会马上把你的贷款合同作价250万，连同其他人的“欠条”一起打包成一个理财产品，转手卖给其他金融机构。

这就是大名鼎鼎的ABS制度。通过它，你这边月供刚还了没几期，银行那头早已通过打包转卖，把本金和利润收回囊中，根本不用等你30年，继续投入下一轮“金融流水席”了。

这套玩法看似花哨，但实际还在监管范围内。尤其是征信系统管住了银行的手脚，让银行不至于肆无忌惮的放贷。

回想网贷最疯的那几年，恰恰是征信约束对放贷机构严重缺位的时期。

当时，不少消费金融公司用低成本资金(比如4%的同业借款，不正规的小贷会非法吸储),转头就以24%甚至更高的利息往外放，有的还玩“砍头息”。

很多公司放款用毫秒计数，你还没来得及反悔，钱就到账了。它们将资金无差别地输送给根本没有还款能力的人，包括在校大学生。

为什么它们敢这么肆无忌惮?不怕别人不还钱吗?

还记得前面提到的 ABS制度吗?小额贷款公司同样会把你的债务合同层层转卖，区别是，银行的转卖有监管，接手的都是可查询的正规机构。

但这种小额贷款公司可就不那么讲武德了。一张3000块的欠条，转几次手，最后300块就卖给了催收公司。

对于最终接手的公司来说，十笔里只要成功催收一笔，它就回本了，两笔就是血赚。

所以，即使你不还钱，它们也稳赚不赔。

正是这种暴利模式，催生了络绎不绝、手段激烈的暴力催收，将无数借款人拖入了“以贷养贷-债务转卖-无限催收”的恶性循环，最终酿成了诸多社会悲剧。

所以，征信报告远不只是悬在借款人头上的剑，它更是在逼着银行用一套统一的标准，捆住它们为了利润不惜盲目放贷、制造系统风险的手脚。

回头看，消费金融最疯的那几年，正好是疫情前后。

天灾没威胁到很多人的生命，却威胁到了他们的生计。

不少人铤而走险，借了“有毒”的钱渡过难关。

那时候逾期的，多数是小店主、要还房贷的打工族——他们是经济的毛细血管，他们的困难，更多是“诚实的不幸”，而不是成心赖账。

征信系统本来是为了建立一套统一的“信用语言”，让金融机构理性评估风险，而不是被短期暴利带偏。

但在特殊时期，它却成了锁住很多人的脚镣。

银行批贷款，主要看两点：一是还款能力(你赚的钱够不够还)，二是还款意愿(你过去有没有按时还)。

就算你现在收入不错，但只要征信上有连三累六的逾期记录，你就被打入了黑名单。(连续三个月逾期，或者两年内累计逾期六次)

你只能去找地方小银行或者民营银行。它们为了做生意，愿意承担更高风险。

但金融业的规矩是：风险高，价格就得高。

所以，你只能接受更贵的钱，这样一来，国家给的各种低息优惠，你都享受不到。一个过去的信用污点，会让你在未来很多年里，一直承担更高的借钱成本。

也正是因为这个，现在推行“征信修复”机制，才显得特别有意义。

它不是纵容失信，而是给那些因临时困境而跌倒的诚实者，一个宝贵的“重启键”。

通过必要的程序和审查，帮助他们修复信用，重回经济正轨，这不只是救个人，也是在给整个经济“活血”，疏通那些被困难堵住的毛细血管。

征信修复机制为“已经上岸”的人清除了历史障碍。

但那些还在债务深水里挣扎，爬不上岸的人怎么办?

正是这个悬而未决的问题，将公众的期待引向了更深度的解决方案——个人破产法。它和征信修复是一套组合拳：帮“好起来的人”修复过去，一个帮“病太重”的人终结噩梦。

很多人觉得国外个人破产法特好，法院一盖章欠债就一笔勾销，其实，这有个普遍的误会。

就说美国吧，个人破产法压根不是“免单”，更像强制还款计划：它虽然能最终帮你减免部分债务，但前提是你必须拿出诚意，在未来三五年里，用稳定收入尽力偿还债务。

法院只保留你的基本生活保障(比如自住房、代步车)，大额消费全部要受限。

而且申请门槛不低。你得拿出切实可行的还款方案，还得让债主点头同意，不是你单方面说破产就能破的。

就算申请成功了，代价也不小。破产记录会跟着你七年，这期间贷款买房买车基本没戏。连信用卡都可能被换成“担保卡”——得先往里面存钱才能刷，想想看，在人均存款才3000多美元的美国，这样的日子得多紧巴。

所以你看，微信pep458根本不是“免费的午餐”，而是一套严格的债务重组机制。

其实这套思路，在我们的司法实践中也有体现。

只是咱们的《破产法》一直被称为“半部法”——2007年起，企业破产法出台，可普通人的债务纠纷，还得走民事官司，靠法院强制执行。

虽然法院处理这类案件的逻辑上，也是债务重组，但因为缺少一部全国统一的《个人破产法》，各地都是“一个案子一个办法”。

深圳、浙江等地虽然在试点“个人债务清理”，但整体还是摸着石头过河。

为什么推行一部个人破产法这么难?

第一，技术上确实有难关。我们现在还很难把一个人的家底彻底查清楚。他到底有多少张银行卡、外面有没有投资、亲戚名下有没有代持的资产……这些信息散落在不同部门，没完全打通。

这就导致一个核心问题：怎么区分谁是“真还不起”的老实人，谁是“装可怜”的老赖?

深圳试点就遇到了这种情况。从2021年到现在，法院收到一千多起申请，最后真正免除债务的不到10%。为啥这么少?就是因为查不清财产，有些人就想钻空子，一边申请破产，一边拖延还债甚至转移财产。

另外，全国范围的财产查询系统也尚未打通。

现实中就出现过这样的情况：某人在A省欠下债务，但他的财产都在B省，A省的法院很难去执行。

第二关更棘手：道德风险。立法者本意是想救那些因为创业失败、生病等客观原因实在还不上的“诚实而不幸”的人，但万一法律被老赖利用，变成“合法逃债工具”怎么办?

这又绕回了第一个问题：如果连一个人有多少财产都搞不清楚，谁敢轻易打开“债务豁免”这个口子?

不过，希望也在出现。这几年，国家在财产联网上下足了狠功夫：

房产全国联网，名下有几套房一查就知道；
金融资产这张网也越织越密：央行盯着大额资金流动，中国结算掌握所有股票账户，银保信汇集了银行理财和保险信息。

最厉害的是法院的“总对总”查控系统，一旦被列为失信人，法官在电脑前就能直接冻结你全国几乎所有账户里的存款、理财和股票。

有了这些技术做后盾，一个重要信号终于出现了。

今年9月，《企业破产法》修订草案公布，这部已经实施了18年的法律，第一次明确提出要解决因企业破产被牵连的个人债务问题。

这意味着什么?

想象一下，一个勤勤恳恳开餐馆的小老板，当初只是帮朋友做个担保，在贷款合同上签了个字。

结果朋友的生意黄了，人也不见了，这笔债“哐当”一下就全砸在了他自己身上。

后果呢?很可能是一家人省吃俭用买的房子被拍卖，银行卡被冻结，多年的积蓄和安稳生活瞬间归零。

一次讲“义气”的帮忙，最后却让整个家庭背锅，甚至妻离子散。

而现在，法律终于看到了这个问题。未来，这样的担保人有望通过合法途径获得重生机会。

这关键的一步说明，立法思路正从“只救企业”转向“也救企业背后那个活生生的人”。

我们呼吁个人破产法，不是为了给老赖行方便，恰恰是为了更精准地识别和惩罚老赖。

它的核心，微信pep458是在冰冷的债务数字之外，保护“诚实而不幸”的人，同时牢牢守住公平底线。

这和“个人征信可修复”政策的方向是一致的：不是给失信开后门，而是给诚实的奋斗者一个宝贵的“重启键”，让努力生活的人在经历风雨后，还能看到重新开始的希望。
