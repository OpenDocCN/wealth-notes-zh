
# 泪痕春雨：人格的塑造(2)

《人格的塑造》

世界上最畅销的书，是什么书？肯定是《九阴真经》、《降龙十八掌》、《如来神掌》、《独孤九剑》……我是说，假如世界上有这种书！想让自己的书畅销，最常见的套路是什么？那就是把自己书，包装成《九阴真经》。

你经常为自己武功低微而烦恼吗？你经常为自己受人欺负而烦恼吗？你想让自己武功突飞猛进吗？你想让自己有朝一日称霸江湖吗？请赶紧把握这个可遇不可求的机会！我的《九阴真经》含泪大甩卖了，九块九，只要九块九。九块九，你买不了吃亏；九块九，你买不了上当，绝对物超所值。

这听着，就有点象江湖骗子在卖包治百病的药。但是，万一它是真的呢？当然了，如果你觉得《九阴真经》是假的，还有《葵花宝典》、《辟邪剑谱》等着你。换而言之，没有人能随便成功！拿上《九阴真经》随便一练，就能成功，这不是扯呢嘛。所以，你想成功，就需要付出代价！所以，欲练神功，必须得先挥刀自宫！总而言之，你只有人格黑化了，才可以成功！所以，老实人，你不能再老实了！这个声音，听起来，是不是更加真实，是不是更加诱人。

社会的博弈，永远如同经济的博弈，它有一个定律是不会变的。竞争门槛低、或是风险低的地方，挤满了人，最后最大突破的，就是如何在内卷中取得优势；关键是，最后的利润，一定会被内卷到底。竞争门槛高、或是风险高的地方，肯定有巨大的利润，因为竞争的人有限，内卷的就非常有限。真正的问题是，你如何跨过这个门槛呢？你愿意承担这种风险吗？寻找一条门槛低、风险小，但是利润大的地方。这是所有人梦寐以求的。当一个人告诉你，他找到了这种地方，只要给他九块九，就可以带你过去，大概率是上不可能的。甚至给九万九，大概率上也是不可能的。

高门槛、高风险，才会把绝大多数的竞争者淘汰掉；从而让竞争变得不再激烈，从而留出高利润的空间。低门槛、低风险，永远意味着竞争者一片汪洋，从而伴随着各种毫无底线的内卷，最后把利润拉到最低。在这种背景下，你寻找一条低门槛、低风险，却利润巨大的地方，那就是不可能的。有没有例外呢？这个，我不敢说没有。比如，有人花了两块钱，买了一张彩票，然后中了五百万。这门槛低不？这风险低不？这利润大不？真心说，类似的例外，都好象抓奖券一样。所以，我们买一本书时，千万不要抱太多不切实际的幻想。知识的价值，就在于提升你底层的认知，绝不是帮你突然打通任督二脉，进而成为绝世高手。

因为不论你如何了解人性，也无论你如何了解社会，有一些底层的逻辑，是不可能改变的。那就是，这个世界，从来不缺象你我一样的聪明的人，也从来不缺象你我一样的愿意吃苦的人；却有很多家庭条件、社会资源都比你我要优越的人。

当你明白这个道理，你就会知道，成功永远不是件容易的事；实现阶层跨越，永远不是件容易的事。底层认知的突破，只是让你有机会，看清前面的门径……但是剩下的路，它还很漫长，而且充满曲折与艰难。

我写的这个《人格的塑造》，只是试图解释，我们为什么会成为这样的人？或是我们想改变自己，为什么会这样难。看完各种成功学的秘籍，在现实生活中，你大概率上，不会有什么收获，只是会保留一些希望所谓希望。

看完我写的《人格的塑造》，在现实生活中，你大概率上，也不会有什么收获；只是会明白一些事的底层逻辑。

人格是如何形成的

所有的，在生命最开始，都没有什么本质的不同，都是动物性的。人与人的不同，主要 是后来选择的人格不同，塑造出来的人格不同。

人在刚出生时，都没有所谓的的人格。因为刚刚出生的人，啥也不懂。所有的行为都是自然的流露；他一点也不会顾及别人感受，一切只要自己舒服就可以了。什么叫廉耻，他根本不懂；光着身子，只要不冷，他才不管有多少人看着他。想尿了，直接就尿，他才不管在什么地方，什么场合；不用说想尿屎了，就是想拉屎，他也不会管什么地方、什么场合。此时的他，只知道饿了要吃、渴了要喝；不舒适了，他就哭；高兴了，他就笑；没事了，他就躺在那里睡乎乎。稍长大一点，他也是任着自己的性子做事；想干什么，就干什么。决定他行为的，只有外界的利害强化；只要自己舒服了，他才不会管别人是什么感受；此时的他，只懂地索取；绝不懂得什么叫回报；更不懂得什么叫感恩。在很长一段时间里，人就是这种状态。圣人也好，滥人也好，并没有什么分别。

但是，随着人越来越大，这种完全自我为中心的时代，就会渐渐远去。因为人需要拥有自己的的人格了，他需要塑造自己的的人格了。一切是显然的，人如果一直保留最初的那种形象，肯定是无法在社会上生存的。在你无知无识的时候，别人可以原谅他所有的行为；也可以理解他的所有行为。随着他渐渐长大，他再这样，肯定是行的！所以，你必须得有一个人格。

在 人无知无识的时候，人展示在外的形象，就是内心自然的流露，没有什么修饰，更没有什么伪装。但是，随着渐渐懂事，展示在外面的形象，就全是经过修饰的、甚至是伪装出来的。所以，人格的本意，就是面具。

从孩子懂事起，父母会告诉孩子。什么事能做，什么事不能做；什么事，有的时候能做，有的场合能做；什么事，有的时候不能做，有的场合不能做。当孩子学会说话时，父母还得教会孩子。什么话能说，什么话不能说。什么话，有时候能说；有的场合能说；什么话，有时候不能说。有的场合不能说。类似的教育开始后，父母还会教孩子，什么场合应该说什么话，干什么事。换句话说，许多不论你愿意说，不愿意说，都得学会说；有的事，不论你愿意做，还是不愿意做，你都得学会做。

类似的教育，目的只有一个，那就是让孩子渐渐融入社会、适应社会。在此过程中，孩子也会渐渐会知道，外界对自己的评价的意义。换句话说，你的言行比较得体，人们就会表扬你、称赞你，而类似的反馈背后，都会伴随着一定的奖励。再换而言之，你的言行如果不得你根本，人们就会讨厌你、讥讽你、批评你；而类似反馈背后，常常会伴随着一定的惩罚。

在这种背景下，人将获得一个简单的、粗糙的、幼稚的人格面具。为什么说这是 一张面具呢？因为从此以后，人所有的言行，都不再是自然的流露，而是经过各种修饰、甚至伪装，才开始的。为什么说，现在的人格面具，是非常简单的、粗糙的，幼稚的呢？因为整个人格的塑造过程，它是一个长期的、反复修整的过程；关键是，人需要考虑的，绝不是简单的什么事不能做，什么事不能说；什么时候该说什么话应，什么时候应该做什么事……所以，这只是人格塑造的开始。

真正的 人格塑造，还需要漫长的时间进行，更痛苦的心理挣扎，才能渐渐塑造完成。因为在人格塑造的方程，人需要面对无数个两难的、矛盾的、充满利弊的选项中，寻找一个最优的选项。类似的过程，充满了患得患失的思考；更充满了无数试错的尝试，最终渐渐确立一个比较稳定的人格自我。最经典的，最常见的矛盾选项，有如下几种。脸厚点好，还是脸薄点好？外向点好，还是内向点好？奸滑点好，还是老实点好？胆大点好，还是谨慎点好？虚伪点好，还是真诚点好？爱慕虚荣好，还是注重实利好？淡泊名利好，还是有野心好？放纵自己的欲望好，还是压抑自己的欲望好？遇事时，只想着自己好，还是愿意为别人着想好？与人相处时，友善一些好，还是凶恶一些好？为人处事，低调一些好；还是张扬一些好？

在这些矛盾的选项之中，我们应该选择哪一个？毫不夸张地说，我们所有的选择，都是基于利害考量之后的结果。在这些矛盾的选项中，任何一种选项，都有可能让你受到惩罚；任何一种选项，也都有可能让你受到奖励。我们很难简单的说，这些矛盾的选项中，哪一个选项是最优的；因为类似的原因，明显矛盾的两种选项摆在面前，总有人会选择A，也总有人会选择B；关键是，就是单纯的基于功利考虑，我们也很难说，他们谁的选择更正确。

最经典的例子就是。遇事的时候，一个人只想着自己好？还是愿意为别人着想好？单纯的基于功利角度，我们也很难说，哪种选择更好。

再经典的一些的例子就是。遇事的时候，谨慎点好？还是胆大点大好？单纯的基于功利角度，我们就更难说，哪种选择更好了。

每个人，都会基于自己的成长环境、生存环境；并且基于自己的分析判断，选择自己认可最有利的 行为模式。而类似的行为模式渐渐趋于稳定，就形成了我们的人格特征。

人格是什么

人格，就是人表现在外的行为模式。不同的，总会表现出不同的行为模式。

什么是人的行为模式呢？或是，什么是人格呢？说得通俗点，说得简单点，甚至说得幼稚点，大约是这样的。有的人，给人感觉是好人；有的人，给人感觉是坏人。有的人，给人感觉是君子；有的人，给人感觉是小人。有的人，给人感觉是英雄（有魄力、有担当，视死如归）；有的人，给人感觉是狗熊（胆小怕事、贪生怕死）。有的人，给人感觉慷慨大方；有的人，给人感觉吝啬小器。

我们现在的 问题 是，人与人之间，为什么会出现这种差别呢？我在上一节，给出的答案非常简单。那就是，人为了更好的适应社会，需要在众多矛盾的选项中，做出一些自己认为最有利的 选择。类似的解读，在许多人看来，就是以小人之心，度君子之腹。这就好象，看着我对历史的解读，许多人会感觉深恶痛绝；因为在我的历史解读中，好人 与坏人的界限非常模糊，无论好人，还是坏人，他们的行为，都 是受制于现实的利益关系；谁也无 法真正超越现实的利益关系。

如果一个人选择的是反面内容（坏人、小人、狗熊、小器）。你说他是基于利益关系，这个谁也会认为没有毛病。问题是，如果一个人选择的是正面内容 (好人、君子、英雄、大方)，你也说他是基于利益关系，这个多少是让人难以接受的。

我们现在的 问题 是，在现实生活中，选择做一个好人，对自己更有利；还是选择做一个坏人，对自己更有利？
