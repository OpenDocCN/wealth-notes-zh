
# 普通人突围之路（六十七） 

## 一、劣势

当年有个视频很火，京东刘强东手下有个女高管，怀孕了，饭桌上跟东哥讲，我还能干。东哥讲，也不能什么都自己干，要把机会留给年轻人。

然后就是网上有一堆人在视频下面评论女的说话不注意场合，导致东哥在全国人面前很尴尬，感觉堂堂一大公司老板像虐待女孕妇一样。

其实呢？这就是女高管的窘境：鸡肋鸡肋，食之无味，弃之可惜。你好不容易坐上的位置，放弃，不甘心；坚守，那就没时间生孩子，有的甚至没时间谈恋爱结婚。有些人熬着熬着，就熬成了热搜上那个86岁终身未婚无儿无女的北大老太太。

话说你熬到最后，你也不可能和公司过日子，你最后的最后，还是会被老板请出去。因为机会，总得留给年轻人，不是么？当你看明白女性在职场上天生不利的处境，你就该知道，作为男性，自己已经很划算了。

作为男性，你不要觉得娶媳妇生孩子是拖累，不是这样的。你买房子不是给老婆孩子买的，你追求赚更多钱，也不是给老婆孩子赚的。那只是你作为一个人本身，你想要成为更好的自己的副产品而已。

换言之，那本就是你自己要做的事情，现在有人替你生孩子，有什么不好么？对于男性来讲，能早点结婚早点当爹肯定是好事。

## 二、成本

抖音的张一鸣一毕业就当爹了，意味着他今后不需要再花时间去谈恋爱（且是成本越来越高的），去建立亲密的信任关系（之前讲过中产缔结信任关系的最佳路径，那个时候张还不是大佬逻辑，换个苹果手机还得思考再三）。

因为一个人，你在不同时间段里面你的单位时间价值是不一样的。好比你刚参加工作时一个小时值100块，那你去陪女朋友聊天，一个小时刚性支出也就100块。随着你后来职级的渐渐升高，你一个小时的价值变成了1000块，那么对不起，你陪女朋友聊天，刚性支出也升高到了1000块。站在经济学的角度，你告诉我，用100块去泡女生划算呢？还是1000块？那么再往前追溯到大学时代呢？你那一个小时，一毛钱都不值，你完全是零成本在泡女生啊。所以，张一鸣就选了一个最经济的方式完成了这件事。

很多事儿，赶早不赶晚的，你早点出发，成本极低，因为越早的时候，你自己越不值钱，你的时间越不值钱。等你的时间一旦开始值钱了，你就开始定型了，你能回旋的余地是很小的，你做每件事的机会成本，刚性成本，都会迅速攀升。于是乎，路就越来越窄了。对，你没看错。一个人再优秀，他也不可能经历完所有的东西，他可能某个领域里出类拔萃，于是他是大佬。所以，这里面就蕴藏着普通人在当下这个时点的一个重大商机，后面再详述。

就是那句话：天下武功，唯快不破。你做什么都比别人快，那你永远是在自己的时间还不值钱的时候就已经出发了，而不是把事情都往后拖。事情永远是越积越多的，就像那个老太太，难道你也要积到86岁再做么？

再举个例子：一个人总爱发脾气，说明什么？这个策略在TA以前的人生里是有效的，也就是说，要么是从小父母宠坏了TA,TA 一直都是靠发脾气解决问题的，要么是TA以前的异性朋友也曾经很包容TA, 这进一步纵容了TA。这会导致什么？Ta很难再找到能容忍Ta脾气的另一半了，相当于什么？被自己的伴侣锁定了，选择权被极大降低了。

所以，你别看某些人闷闷的，脾气好得很，其实他心里很清楚，这样就等于锁定了另一半的出口（和用钱把另一半养废了的策略同理，你想想家庭主妇就懂了）。

一言以蔽之，人的行为，是有路径依赖的，当你不理解一个人的行为倾向时，你应该立马意识到：TA很可能曾经大大受益于这个行为倾向。

## 三、提拔

你了解了这个逻辑，那身为企业的管理层，也一样有他的行为倾向和路径依赖。是什么？他们从来不会提拔任何一个勤勤恳恳，老黄牛式的员工。原因主要有两个方面：

第一个方面就是上面讲得，关于成本的考虑。只不过比较隐晦，很多人不做管理者，他就看不明白。

道理很简单，你如果长期能兼任很多人的工作，请问提拔了你，谁来干你的事情？你但凡当过一天部门经理就会知道，人事部门，有一项核心考核指标叫做人力成本核算（人效/人产）。这是每一个经理都要向公司高层上交的述职报告。

结果在这个部门，你一个人就兼了五个人的活，拿着1.5倍的薪水，请问，提拔你，要用多少钱去填补你留下的坑呢？要用3.5个人的薪水。

换你做部门的头，你把这种人力报表交上去，你让公司高层怎么批？人力成本上升3.5个人头，工作绩效不变。老板批你这个报表，他是脑子进水了吗？

任何一个主管都是职业经理人，他的首要关注点是什么？对上都是邀功的（提交本年度自己的工作亮点永远是他的重中之重），他要详细阐述的是经过他的决策，替公司省了多少薪水，增加了多少绩效（降本增效）。

而不是说，他要向上提拔这个老黄牛，结果公司的“好处”呢，是要多花3.5个人的薪水，绩效呢，还不变或者因为五个人内耗还下降了。

那他这个经理还有什么存在的价值？他如果曾经受益于这个行为倾向，请问他是怎么被提拔到这个位置上来的？

类似的问题之前也有读者问过剑客，他想问的很简单，既然现在产能过剩，失业率高，那为什么企业不多招人，让五个人去干原本一半的活（工资也减半）。

怎么说呢？其实他能思考到这一步已经很好了。因为有些东西，管理层是不会告诉你的。

雇人，除了工资考量，还有很大一部分开销是税。而这个税，就是五险一金。它的存在，就没办法让老板走多雇人的模式。

举个例子你就懂，我们就按照2024年上海的社保最低缴纳算好了。我们能看到的是最低企业要缴纳2975.4元，但与之同时，上海的最低工资是2690，至于为什么会出现社保最低缴纳基数和最低工资存在差异，这个咱暂且按下不表，就账面数字给到的情况来说，很明显，最低工资是赶不上最低社保缴纳基数的。

这意味着什么？只要是正常给员工交五险一金的单位招人的时候，他就不可能按照原本一个人能干的活，分成两个人干这个模式。因为很简单，假设这个人的工资是6000，那么对企业来说，用人成本就走到9000块了。而如果企业雇佣两个工资3000的人，那么对企业来说，他的用人成本就直接1W2 块了。换言之，多了一个人的社保3000块。

在打工人的视角里，一个人6000块和两个人3000块是一个概念，但对企业来说，前者的成本就是要比后者省。而且这种省是什么吧，越是老老实实给员工缴纳五险一金的企业，出于企业合规经营的考虑（特别是后面要去北交所和各地股权中心挂牌的企业），越需要这么省。

其实你代入这个逻辑。再去看为什么互联网大厂愿意超高薪给一个人，也不愿意分开给两个
人，其实也是这个道理。

因为这里面，还有一个五险一金缴纳的最高标准限制。比如说对互联网大厂的hr来说，招一个月薪7万的人，最高，就是每个月缴近1W9，算下来每个月成本就是9万不到一点。但如果他选择招两个月薪3W5 左右的人，那么这个人，每个月也都要多缴近1W9，算下来，成本直奔11万了。

讲到这，你看明白了吗？可能在这位读者的视角里，两个人干一个人的活和一个人干两个人的活，如果总工资差距不大，前者貌似更好（体力得到了很大的节约，电池电量足）。但五险一金制度就决定了，如果企业想要选择两个人干一个人的活，那么他需要付出的成本，是大幅增加的。

换言之，每个人干原本一半的活，对经济是好的（有更多的消费人头）。但在现有的制度下，对企业来说，这个做法，是纯亏的。而且老板用一个人加班，比用两个人还能规避掉加班制度可能带来的问题，而且甚至很多时候，用一个人加班，他还不需要付加班费（又降本增效了）。

那你告诉我，你要是管理层，你会怎么选？说穿了，这个税的问题，它是影响各个行业的底层指挥棒。你但凡会算一点账，通一点人情，你就知道你的部门经理绝对不会提拔你这种老黄牛。他可以为了自己的KPI选择继续忽悠你，画饼你，可以安抚你，给你提供情绪价值，但唯独不会提拔你。
