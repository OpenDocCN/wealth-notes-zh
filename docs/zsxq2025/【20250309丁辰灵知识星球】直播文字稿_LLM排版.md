#### AiAgent

网上已经有很多批评炒作Manus的负面了，具体我就不多说了。即这个原理并不复杂，是用虚拟机形式模拟用户进行任务拆解、整合等。但是我对其实效性、准确度，和数据合规等都不置可否。

这类的机会还是属于有数据的成熟公司，大厂的机会在公域。各类垂直应用的机会是类似用友、金山办公等这样的有数据有客户的公司。

因为真正要起到能帮人类有效解决问题是很困难的事情，如果没有对人类行为的深刻理解，没有足够的数据训练，是很难做这类似的复杂任务的。

就好像也不是万能的，未必能完全替代deepseek搜索引擎，因为很多时候AI其实并不知道自己在说什么，仍然是概率的游戏。

#### RISC-V

是一款开源且免费的指令集架构。与RISC-V CPU X86 和 ARM 等传统指令集架构相比， RISC-V 不仅完全免费开放，而且支持自定义指令扩展，因此被视为更适合AI时代的算力架构。

国家现在突然宣布支持RISC-V，是和贸易战中美脱钩有很强的关系。大家记住，一个关键词脱钩；第二个关键词，制造中国的新垄断。中国新垄断只有靠庞大的市场规模和创新来获得。

所以打破西方垄断的唯一办法就是开源，一出来，甚至连英伟达的故事deepseek openAI都没法讲了，RISC-V 代表的是硬件开源的机会。

所以仍然是很早期，这个会持续相当长的一段中国发展和替代的时间，现在才刚开始。RISC-V 本来适合物联网这样的芯片，现在基于DeepSeek对算力的低需求，以及RISC-V的低成本优势，即使一套RISC-V解决方案可能只有人家30%左右的水平，但购买三套就可以弥补这个不足，整体成本可能仍然更低。

特别适合聚焦边缘计算、车规芯片、定制化场景。ARM向RISC-V切换技术难度所需时间与人力成本可控，但核心挑战是生态，包括操作系统、EDA工具等。

### 除科技外，今年港股 A 股还有哪些好的投资机会?

港股创新药，另外消费地产等都有机会，不过我会建议大家谨慎追高，特别是科技板块。涨太快太高容易回落，无论是多好的概念，因为市场总是会因为零到一的爆发而给出溢价，但之后再好的概念，不会天天有爆炸性的能范式转换的大新闻出来。

无论是deepseek,Aiagent,RISC-V 主义的东西，因此如果你追高，一定要严格的止盈止损的纪律来做短线。如果要想稳妥，可以买同一板块中，龙头涨了，第二名第三名还没怎么涨的公司，因为资金炒龙头炒高了，肯定有个雨露均沾往下滴灌的需求。

还有就是找那些没怎么涨的板块，一定要逆向思维。很多人看到一些板块或者公司不涨，甚至略跌，看到别的板块涨的欢，就反复易手，其实这样赢的概率会缩小。

因为股市就是这样，很多公司的大涨一年也就那么几天，错过了就白干了，甚至可能会亏钱。基本上一定要有资产的概念，你买的一定要是一块优质资产，你的配置也是围绕着优质资产去配的，不要只是看涨跌，似乎不涨的就不是好资产。

一块地放在那边永远是一块地，这个庄家在政府那边，如果货币政策大水漫灌，这个地就是会升值。如果一个地方有源源不断赚钱的产业，比如杭州之前有互联网，现在有六小龙，财富效应总是要覆盖到地产的。

很多优质的垄断高股息资产，如果你去配置，就要相信现在不涨，甚至跌(比如由于科技股太火，资金出逃),后面还是会涨回去的。

地产其实要有信心，一方面地产是基础资产，另一方面，由于房价大跌，一线城市的房租回报率已经可以赶上银行利息了，这就是底。
