
# 【米联储见闻251212】——谢朓已没青山空（妖风起，人心变）

最近国内的舆论氛围有点热闹，不少魑魅魍魉粉墨登场，大家最近眼睛要擦亮一点，多看少说，让子弹飞，一般来说，舆论场出现妖风，通常根源都在上面，所以，这股子妖风背后的味道不寻常。不管是什么吃瓜盟主还是芳华新解，背后都是对人心的调拨，要知道现在内外局势都如干柴烈火，不管极左还是极右，都会将中国从正确的轨道上拉偏甚至走上歧途，中国将持续开放，中国将坚持以发展为抓手解决自身问题，这要求中国人在历史问题上要有清晰的认知和基本的共识，任何会撕裂立场和认知的舆情，背后只有一个目的，分裂大家的集体共识。引发新一波所谓反清复明和西方伪史论浪潮的红楼梦悼明说，以及为招魂的电影《芳华》解说，其实本质是一套东西，就是历史虚无主义，把中国从偏右的社会舆论场推向极左的道路，当下中国要想在环境复杂的内外局势中行稳致远，坚持常识，保持冷静，在认知不要偏激，才是符合大多数人利益，更容易凝聚共识的。

我也观察到，最近群里有人讨论这个吃瓜蒙主的《红楼梦》解读，让围绕这部清代小说的索隐派观点翻红。其实这并不是什么新东西，当年刘心武在百家讲坛的解读红楼梦也算是其中一支，索隐派在红学研究中不是什么新流派新玩意，只是现在借着互联网的传播产生了新的舆情推动力，从过去红迷和红学研究的小圈子突围而出，索隐派主张通过分析小说细节寻找隐藏的历史事件或寓意，这股索隐派风潮中，林黛玉被指隐喻明朝末代皇帝崇祯、贾宝玉是传国玉玺、《红楼梦》讲的是反清复明的暗史，清朝在汉人政权明朝之后主宰中原，是中国历史上最后一个封建王朝，由满族人1644年入山海关创立，1911年辛亥革命爆发后灭亡，前后266年。对于这股子索隐派的解读，正规学院派人士不以为然，认为只是年轻人玩梗，但也有自媒体人敏感的察觉到，这些言论似乎是有人故意在挑动民族矛盾。这波舆情还有吊诡之处，就在于平台大佬们入局，前几天抖音集团副总裁李亮公开发文说，最近关于红楼梦的讨论很热烈，但出现不少史实错误或没有依据的猜测，呼吁“不管何种观点都要有事实为前提，理性讨论，拒绝盲目吃瓜”。李亮还透露，抖音辟谣团队这两天一直在收集资料，核查这波由吃瓜蒙主引起的历史讨论。说实在，平台热点这么多，什么时候看到平台高管跳出来主动挑起公众关注，甚至主动收集资料来核查推进历史讨论。在这个热点中，关注度最高的两大讨论方向，分别是《红楼梦》乃悼明之作，以及满汉全席是满人吃汉人。抖音官方核查称，前者缺乏可靠证据，未成为学术界共识，后者则缺乏权威信源支持，且与历史事实严重不符。抖音上出现的其他缺乏依据的延伸讨论还包括，英国工业革命窃取明朝蒸汽机技术，郑和下西洋乘坐蒸汽动力船，夫妻肺片和老婆饼都是清朝满人发明，清代严禁民间教育导致识字率不到1%,清代强迫汉女断足式裹脚，清宫剧涌现是因为娱乐圈清朝八旗后裔操控、企图美化清朝，华伦天奴品牌创始人祖上为满清贵族、品牌中文名辱华，以及元清非中国论。

所谓西方伪史论，即主张古希腊、古埃及、古罗马等西方文明，以及近现代西方人在自然科学和人文领域的突破，都是伪造或剽窃自中国。其中一个典型案例，便是宣称西方现代科学是抄袭明朝编纂的《永乐大典》。在最新的红楼梦悼明的热潮中，便有网民发文称：发明，它不叫发东，不叫发西，不叫发秦，不叫发唐，不叫发宋，也不叫发元，唯独他叫发明，因为他都起源于永乐大典，中华文明的知识宝库。说实在的，但凡读完义务教育都知道这个解释只有四个字——牵强附会，且不说英文的发明一词与明没有任何关系，即便是现代汉语中的发明二字也是近现代才有创新之意，过去这两字甚少合并使用，更不要说有创造新事物之意。这种拗谐音的梗真是过于牵强。这种言论确实会让人觉得中国民智未开，甚至让永乐大典蒙羞。

至于1644史观，就是把1644年清军入关当成中国近代史起点，把清朝定性为异族殖民统治，还把近代中国的落后挨打全怪罪满清，把满、蒙等少数民族，与占中国人口逾九成的汉族对立，用征服者与被征服者的叙事制造矛盾，要知道五十六个民族凝成一股绳，抱成石榴籽，这是中华民族共同体的根本，也是今天中国之所以如此强大的基石，没有藏疆蒙等少数民族的团结，中国不是今日之中国，中国历史是一部完整的中华文明传承史，切断任何一部分，中华文明延绵不绝的脉络就会被拦腰砍断。这些言论看似玩笑无害，但是破锄头发力之处都是挖向中华民族共同体坚牢墙体，千里之堤溃于蚁穴，一些言论一时一刻看似乎无毒无害，不伤大雅，却会在人心里种下种子，日本长期以来都有分裂中国之心，这种事到真的像是日本人的手笔，借重新解释历史，利用虚假史观煽动民族分裂的情绪，对中华民族的团结和社会稳定和谐制造隐患。1644史观的部分逻辑就曾被分裂势力利用，日本就曾用“满蒙非中国论”为分裂中国寻找藉口，而这种论调的核心，就是把清朝和中国割裂开来，因为台湾归属中国本质上与清王朝是中国一脉相承密不可分，甚至新疆西藏的历史溯源，都绕不开清王朝的历史，清王朝当然是中华文明的一部分，过去是，现在是，将来也必须是。有些别有用心的重拾类似逻辑，就是要动摇现代中国疆域的历史法理基础，给分裂活动留下可乘之机。

早在1990年代，美国汉学界兴起新清史研究风潮。西方学者研究清朝历史重视汉语以外的档案，强调清朝的满洲元素有别于中原王朝的朝代特征，认为清朝不是过去中国的改朝换代，甚至认为清朝不等于中国，中国只是清帝国的一部分。这种新清史的观点就是对大一统的中国认同提出挑战，这股风潮曾经导致清史在中国媒体层面成为敏感议题，甚至愈演愈烈，以至于官方在2002年启动编撰的大型史书《清史》迟迟未出版，担心这种讨论进一步甚嚣尘上。这也足以说明，外部势力想要借助于重构历史观来分裂中国的野心从未根绝，而且一直伺机而动。

至于前一段时间对《芳华》的解说也一时爆红，这条视频长期挂在B站头条的位置，不管是因为算法还是B站本身有所图，本身就不是正常现象，我是每天都要看B站的人，但是我至今未点开这个视频，因为光是看标题，我就知道他想说什么，而这个想说什么背后的心思也很明确，就是要利用历史敏感事件的分歧和讨论来撕裂民众的认知和共识，形成对立。这个片子我虽然没看，但是也从其他评论这个片子的文章中一窥视频本身的观点，无非是否定wg是内乱的官方论述，且最终是走资派彻底胜利，窃取成功果实，占尽资源、政策、市场、时代红利，并看似情深款款的表白大哥，来迷惑人群中对老毛心存感激敬意的群体，要知道老毛穷极一生将中国人民组织起来，团结起来，动员起来，就是要让曾经一盘散沙的中国人拧成一股绳，让内斗天王老子，外斗一塌糊涂的中国人能在东方屹立。更何况，冯小刚这个人本身就是一个投机分子，我是不相信冯小刚拍这个片子能安什么好心思的。老毛固然有伟大之处，但非要硬说他刀枪不入，那不是尊敬他爱护他，这就是低级红高级黑。

至于说什么黄轩长得像王洪文，更是心怀叵测，至于最后那句“历史从来都是胜利者的战歌……那些夺嫡中落败的王爷，自然就成了心怀叵测的乱臣。那些力主改革却触犯了权贵利益的臣子，便被描写成祸国殃民的奸佞。”这种极具煽动性的鼓吹，无非是希望借助当下经济转型中，利益受损群体积聚对社会现状的不满，成为不稳定因素甚至是随时可燃的干柴，让中国在面对外敌的时候无法团结，而内部稍有风吹草动立刻乱成一团，要非常清醒的意识到，当下最利于中国的局面，就是安全与稳定，之所以在大国竞争中，时间在中国这边，就在于中国能保持安全和稳定，乱利于外地，稳则遇事不慌，对于这些不利于团结的舆情，还是希望大家理性分析，冷静看待，不要入戏太深。

社会转型期，内忧外患，尤其是台湾间谍和日本鬼子都闻风而动，趁机搅浑中国舆情这个池子，搞乱中国的人心。妖言之所以可以惑众，在于它踩中人心的痛点，或是印证了你的偏见，兼听则明，多看少信。
