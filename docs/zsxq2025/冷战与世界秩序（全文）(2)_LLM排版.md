
# 冷战与世界秩序（一）铁幕降临

本文为2017年2月18日冬川豆讲座的录音整理稿，整理者xqmxqm，校对者三马兄，冬川豆配图、注释。世界秩序的概念是非常古老的，也就是说，有成型的文明体系，就会产生相应的世界秩序的概念。但第一次世界大战以前，特别是十九世纪末期，世界秩序的概念实际上是已经降到最近五百年以来的最低点。俄罗斯虽然继承了拜占廷的世界秩序，但是自从加入欧洲式的世俗君主制以后，就把这个概念搁置和压缩了。拿破仑战争和维也纳会议以后，俄罗斯尽量冒充一个普通的欧洲君主国，这是俄罗斯内部灵魂分裂的基本原因。当然俄罗斯这方面也是有它的原因的。从东正教本身来说的话，我们可以把它看成是两个部分的组合。一部分是秩序的生产者，就是东正教的教会组织、村社组织诸如此类。它们的秩序生产能力是不强的。在没有跟外界接触的情况下，大体上来讲，只能够产生一些原始性的领主和亲兵。蒙古人给俄罗斯输入了上层建筑，大大加强了汲取能力。但是就是因为汲取能力的加强，反而妨碍了东正教社会基层的进一步发育。结果彼得大帝跟欧洲人接触的时候，他为了跟欧洲列强竞争，发现东正教传统社会能够支持的那个王朝体系跟欧洲的绝对君主无法相比，于是就采取了输入政策。输入政策具体就体现于引进德国新教徒、引进瑞典人和欧洲的客卿诸如此类的做法。整个十九世纪，引进最多的是德国技术人员。没有这批为数极少的、在俄国人口中连1%都不到的客卿的话，俄罗斯帝国没有办法维持一个欧洲式的门面。

同时，十九世纪也是欧洲人传统的神圣罗马帝国那个世界体系的最低点。从三十年战争和威斯特伐利亚体系开始，基本的趋势就是：为世界体系牺牲最大的投资者付出最大的牺牲，像黎塞留和法兰西这样尽可能地牺牲世界体系为民族国家服务的actor占到了最大的便宜。因此十九世纪末期是欧洲本身的民族国家的最高峰，民族国家积攒了大量的红利，刺激了众小民族的进一步发育，而整体上的世界秩序的概念已经处于最低点了。世界秩序概念的消失是两次世界大战得以爆发的最重要的原因之一，所以威尔逊主义和罗斯福提出的四大警察、集体安全措施，等于是从世俗意义上来讲弥补民族国家体系、威斯特伐利亚体系——准确地说，威斯特伐利亚体系只是民族国家体系的母体，并不是民族国家体系的本身。威尔逊主义的实质，是要在民族国家体系之上，把最近几百年逐步削弱、近乎消失的世界体系补起来；罗斯福的四大警察理论是威尔逊主义的进一步落实。

但是它们产生的时候，苏联已经产生了。苏联是二十世纪世界体系最大的刺激者，也是最大的秩序消费者。我们要注意，苏联和以前的俄罗斯东正教国家不一样，因为它的基本目标是企图建立世界帝国，因此它在俄罗斯本身采取的是纯粹消费的政策，把支持俄罗斯专制皇朝的核心——东正教的教会和村社组织都当作首要敌人，必欲除之而后快。二十年代世界革命在欧洲和西亚全面受挫以后，统战的重要性开始上升，“一国建设社会主义”派在莫斯科战胜了世界革命派。用通俗的话来讲，就等于是燃烧燃料的人发现获得燃料的速度太慢，没有像我们原先设想的那样可以很快地把全世界都当作自己的燃料来燃烧，现在我们还是只有俄罗斯这个基本盘，因此我们必须精打细算。这种政策的结果就是三十年代共产国际世界革命政策的调整，对外就是体现于统一战线策略，欧洲的人民阵线政策是这个统一战线政策的主要体现。

第二次世界大战爆发以后，苏联得到了最大的一次引进外来秩序的机会，因此直接导致了共产国际本身的解散。共产国际解散的主要动机就是，要在统战对象——以英美为核心的资本主义世界的眼前向他们表示，俄罗斯以后要做一个正常国家，停止它燃烧既成秩序的政策，尽可能地模仿以前的沙皇政体。这样做的原因就是，最初几十年的苏维埃政权跟彼得大帝的绝对君主制不一样：彼得大帝的绝对君主制虽然燃烧秩序的能力超出了东正教社会的生产能力，但是超出的程度不多，在欧洲的秩序输入和东正教社会本身的秩序生产两者之间的平衡之下，大体上还能够维持局面；但苏联的体制在不断破坏东正教原有的社会基础的同时，从欧洲输入的东西并不太多，对德国开战以后，二战以前主要的输入者——也就是拉巴洛条约①的德国停止了输入，这种情况下苏联是支持不久的，必须依靠美国这个新输入者来取代。

斯大林时代的苏联就体现出了托洛茨基派或者西方左派所谓的复辟特征。复辟特征体现于很多细节，比如说像是军阶制度的恢复，国企一长制的恢复，社会上允许东正教会重新活动，取消“一杯水主义”、恢复婚姻和家庭，等等。按照真正的Leninism者和真正的左派的看法来讲，这就是说，苏联对资本主义社会或者说对整个剥削阶级社会的进攻严重地后退了，甚至采取了妥协式的、热月式的做法。从斯大林的角度来看，他采取的是实验式的方式。他作为布尔什维克党内最土鳖的人，已经跟那些老近卫军不同，老近卫军是不用负实际管理责任的，而他必须负培养干部队伍和管理的责任。他不可避免地感到，革命以后，尤其是在集体农庄摧毁了东正教社会的最后残余以后，苏联的社会秩序生产能力已经完全跟不上了，而干部队伍的膨胀等于是秩序的消费者大大增加，生产者大大下降和消费者大大增加的情况下，他如果不采取一定的让步政策，内部的维持是很难的。但是他处在内外交困、战争和国内的重工业建设压力都很大的情况下，只能采取试探性的、摸索性的方法，也就是说哪一方面输入得最多，他就尽可能地先把哪一方面利用一下，把空缺填补起来。与其说他有什么整体性的、设计一个不同于Leninism体制的斯大林体制的倾向，倒不如说斯大林体制就是在Leninism已经快要把燃料烧光的情况下，四面八方尽可能地摸索，抓到一捆柴就把这一捆柴弄出来、抓到一筐煤就把这一筐煤弄出来拿来用的一种乱碰式的政策。

我们要知道，在communism存在的整个几十年之内，所有的communism者都是没有办法从马克思本人和早期socialism者那里拿到什么真正管用的思想资源的。因为不要说是马克思本人，就是整个socialism的思想家，他们都是秩序的消费者，他们研究的都是怎样对资本主义社会或者前资本主义传统社会进行消费。他们都是物质主义者，物质主义者的实质就是把社会看成一筐煤或者一捆柴，看成是可以利用的物资，对秩序生产的问题完全忽略不计。因此可以笼统地说，只要是“唯物主义者”，基本上就是消费者。对于有生产能力的、像教会这样的组织，一般来说他们都是“唯心主义者”。“唯心主义者”为什么要讲“唯心”？就是因
为生产秩序、输出秩序是非讲“唯心”不可的。把社会看成物质本身的结果就是，你只能分配现有的资源和消费现有的资源，无法对存量资本进行进一步的增加了。

从斯大林自身的角度来看，第二次世界大战时期是他的一个重大的实验期。没有他这方面的实验的话，苏联是没有办法在后期续命这几十年的。从他的角度来看，恢复东正教会意味着对斯拉夫社会本身进行让步，希望从斯拉夫社会本身能够接受新的社会资源。但这个让步是不完整的，效果不大。因为东正教社会主要是两个层次，第一是家庭组织，第二是村社组织，更高级的东西，像西欧的那些自治城市和封建主义，在东正教社会中基本上没有产生出来，这两层社会可以制造出像莫斯科公国或者说各领主亲兵这样的组织，但是斯大林实际上只恢复了家庭组织。村社组织如果要恢复的话，那就要把他自己亲手建立起
来的集体农庄解散。这样做，在一个以农民为主的国家要严重地影响政令通畅，甚至很可能会引起苏联本身的重装问题，引起斯大林本人及其集团在政治上会不会遭到清算的问题，所以
他不敢冒这个险。

恢复家庭产生的正面作用还是有的。照苏联在列宁革命以后的那种情况，一个很明显的现象就是：苏联的孤儿之多、流浪儿之多变成了严重的社会问题，以至于尽管苏联专政机构如此之强大，又按照十九世纪那些家庭社会化的理论搞了很多大食堂、孤儿院诸如此类的组织，试图假定自己在能够取消家庭的情况下尽可能维持，但是成本仍然是高得无法承受。斯大林恢复家庭以后，这方面的成本一下子就下去了。我们可以设想，如果他肯进一步恢复村社的话，说不定还能够更多续命一段时间。但是这种更多续命肯定是要以斯大林本人及其核心集团的政治生命为代价，所以他不愿意走得这么远。

另一方面的续命方式就是从罗斯福那里大量输入盟国的资源，这个资源才是后来苏联能够进行几十年冷战的根本。我们可以简单粗暴地说，列宁破坏俄罗斯军队、破坏俄罗斯社会造成
的那个国家，如果仅仅依靠自己的话，是不会比巴黎公社长命多少的。它之所以能够维持，是依靠第一次世界大战的失败者，主要是德国，出于报复协约国和给自己寻找训练基地的理由，把广大的苏维埃俄罗斯当成了魏玛共和国的外府来利用，因此渗漏出来的组织资源和技
术资源帮助苏联度过了最初二十年的最艰难时刻。新经济政策时期的美国资本也起了一定的作用，但是美国资本起的主要是经济方面的作用，在政治和组织方面来讲，是德国人起的作
用比较大。

德国人在二、三十年代希特勒上台以前，为了维持自己的军事传统，把大量的德国军官送到
俄罗斯，红军军官主要是德国人培养出来的。二十年代那些构成苏联经济骨干的莫斯科缝纫
业托拉斯或者是其他各个托拉斯之类的大企业集团，主要也是引进的德国的组织技术。二战
爆发以前，苏联学校里面的主要外语是德语。组织方式，不要说是俄国社会主义者的所有组
织机构都是从德国社会民主党人那里学来的，就连苏联成立以后重建国家垄断经济，也就是
说除了集体农庄以外的所有经济结构，都是从德国那里学来的。美国企业给他们的投资主要
是在资金和技术这两方面，但是组织方面主要还是依靠德国。

《悄悄的自杀》从调查苏联工业生产的主要工艺流程和设备型号入手，引用大量美国政府的
解密文件、国会证词和苏联资料，论证了“今天苏联百分之九十的工艺技术来自西方”，而
如果第二次世界大战不爆发的话，德国给他们留下的这些东西，即使不说能够维持几十年吧，
至少多维持十几年应该是没有问题的。乌拉尔地区的工业基地是新经济政策以后苏联经济建
设的典型，就可以看成是德国管理方法-组织技术-美国资本-自然科学结合的一个典范。没
有这样的东西，整个计划经济都是无法运转的。东正教社会原先自己的技术生产能力就很差
劲，非常依赖于德国的工程技术人员。比较低级的技术，农业技术、轻工业技术以及相应的
组织，自己还是能够产生得出来，但是像是后来我们当作socialism主要特征的那个军工生
产体系和作为社会轴心的国企管理体系的话，没有德国人的输入和美国人的资金根本不可能
建立起来。

也就是说，我们要把Leninism、Stalinism和Marxism本身区别开来：Marxism完全是解构者，
只依靠Marxism建立起来的社会不会比巴黎公社更长命；Leninism是在Marxism的基础上长了
牙齿，加大了它的破坏能力；而Leninism则是在Marxism-Leninism的外表之下实行德国垄断
大资本主义的组织技术和引进美国的技术，等于是一个挂羊头卖狗肉的体系。它本身也没有
独立的秩序生产能力，如果外来的输入终断以后，它本身是无法维持的；但是它挂着这个牌
子同时引进垄断资本的这一套，本身还有维持几十年的能力。

罗斯福签署租借法案。后来朱可夫元帅同作家К ·М·西蒙诺夫谈话时说：“如果没有美国
人的火药，我们就会处于很艰难的境地，我们就不可能生产出战争所需的那么多的弹药。没
有美国人的斯蒂倍克牌卡车，我们就没办法去运输我们的大炮。战争各个方面都很需要的特
种钢也是由美国人提供的。……现在大家都说盟国从来没有帮助过我们……但是要知道，不
能否认的是美国人为我们迅速大量生产并供应了许多物资，没有这些物资我们就不可能组建
我们的后备部队，也不可能把战争继续下去。”

四十年代的时候，斯大林搞这一套已经有相当的经验了。失去德国这个输入者以后，他通过
《租借法案》从美国得到了更多更丰富的输入，从而填补了这样的真空。美国《租借法案》
当中，75%是给英国和俄国的，俄国稍微少一点，但是少得不多。可以说，早在联合国成立
以前，如果忽略掉没有工业化的整个世界的话，美国已经是世界经济体系的真正输出者了，
英国和俄国从经济上来讲都是美国的分支机构。所谓俄国socialism建设的主要成就，包括
第二次世界大战以后的那些技术成就，全都依靠美国在第二次世界大战期间、《租借法案》
期间给它提供的大量援助。轮胎工厂、橡胶工业、坦克工业、后来的军工技术开发之类的，
全都是美国给它打下的基础。

所以说，冷战时期的美国从技术上讲，等于是在跟自己的影子作战——苏联与其说是美国的
敌对势力，不如说是美国的影子。苏联70%的技术是直接来自于美国的，剩下的那30%，表面
上看来是自己研发出来的，但是实际上，它研发所依赖的那些基地也是美国在第二次世界大
战中间给它搞出来的。等于说，苏联在二战以后一直是在靠吃老本过日子。冷战爆发以后，
美国的输入逐步中断，在52年以后就基本不复存在了。老本还可以吃一段时间，但是不会吃
得太久。

可以说，美国给它打下的这个老本可以供应苏联的时间，比苏联给贵妃打下的老本可以供贵
妃维持的时间要长一些，因为这个老本是要大得多。我们都知道，贵妃五十年代的整个工业
基地、一百多个项目都是苏联援建的产物，包括核武器项目也是苏联专家打下的基础。现在
拿出来吹嘘的所谓两蛋工程之类的，其实是一个广告性的东西，跟曹县战争时期所谓的王海
大队、空军英雄之类的，都是文宣部门的产物，实际上在真实的历史上要么是完全不存在，
要么虽然存在，但是起的作用是非常边缘性的。之所以要把王海这些人拿出来，就是因为要
掩饰这个事实，就是在曹县战争时期，真正打仗的是米格，是苏联飞行员，但是他们不能公
开露面，一公开露面就暴露了贵妃只是共产国际的一个分支机构，是入侵种花冥国的一个机
构，这是他们最忌讳的事情，所以处处都要强调自主。但是自主的东西只是一层皮，骨子里
面是苏联飞行员在那里干。

同样的道理，即使在苏联专家撤退以后，哪怕是直到现在，贵妃的重工业基地还是在吃苏联
当年的剩饭，但是因为自己的社会没有秩序生产能力的缘故，吃剩饭自然是吃得一代不如一
代。八十年代以后引进的那些美国技术作用的区间，跟五十年代引进的苏联那些东西作用的
区间不一样，也就是说，它并不能够真正完全填补苏联输入最重要的那些地方，尤其是重工
业那一块，所以像瓦房店和哈尔滨那些军事工业、重工业的基地到现在还必须用。到九十年
代以后还要从俄罗斯和乌克兰输入一些补丁，因为原来的那个基地已经破烂得差不多了，不
加这些补丁根本运作不动。现在的贵妃虽然钱比原来要多得多，但是从组织和技术两方面来
讲，比五十年代后期反而还不如，因为原先那个底盘已经破烂得差不多了，而现在打的那些
补丁只是零零碎碎的，并不能完全弥补基本盘的破烂。

我说贵妃的这些事情，就是为了让大家理解苏联的事情。苏联的基本盘，也是在第二次世界
大战刚刚结束的时候被美国人打造得最完整，之后就是一个不断破烂的过程，到八十年代已
经破烂得差不多了。即使是在现在，在冷战结束以后又从西方重新输入了一些补丁的情况下，
都还没有恢复到二战后期那个比较完善的水平。自己的社会没有秩序生产能力，靠外来输入
的底盘，那你就只能是吃老本了。

这种做法很像是移植器官。如果是你自己有一个完整的肾的话，那没有什么关系，哪里缺了
一点，血液会给你运资源来补充，骨头断了，也可以从其他地方输钙质来把断的骨头长好。
因为你自己的身体是一个整体，可以说你的身体有秩序生产能力，哪个器官差了一点，别的
器官可以来补充。同时你还不断在吃东西，还在不断呼吸空气，不断产生出新的细胞，不断
产生出新的组织。这样的损失是无关紧要的，有了伤口也很容易长好的。但是如果你自己的
肾脏坏了，从别人身上引进了一个肾脏，那么这个肾脏跟你自己的机体一直是有排异反应的，
你不但不能依靠你自己的机体去补充这个肾脏所需要的资源，还必须不断地吃药，把自己身
体正常的免疫反应压下去，免得这个器官过早地用坏了，尽可能延长这个移植肾的生命。但
是即使在用尽了各种保护措施的情况下，这个移植的肾脏也还是用一天少一天，因为它毕竟
不是你自己的东西，在你身体里面是一个异质体。它可以给你续命，续五年或者是更多时间，
但是归根结底它早晚是要用坏的，它不可能像你自己原先的那个肾脏那么好用，可以自己补
充自己。

如果你们能够理解这个比喻，就可以理解socialism制度的实质了。首先，原版的、彻底原教旨主义的socialism是纯粹的破坏者，它唯一精通的技术就是Leninism的技术，它擅长于把原有的社会搞到解体，把原有的社会组织资源和技术资源都拿出来燃烧得干干净净，这样
做的结果当然是自己活不了了，活几十年都活不了。它能活，就是以各种挂羊头卖狗肉的方
式从资本主义社会那里拿来移植肾脏，一个移植肾脏用坏了以后再移植第二个肾脏，然后再
移植第三个肾脏。就是这样的话，你也要应该清楚，移植肾脏是不可能永远用下去的，你早
晚会死的，续命的时间总是有限的。苏联就是这样的，经过了两阶段的续命：二战以前依靠
的是德国的移植肾脏续命
