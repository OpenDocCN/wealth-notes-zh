
# 记忆承载付费《男怕入错行，女怕嫁错郎，人怕选错城，更怕买错房》

最近好些读者来请教我一件事，就是他们几年前，从某些网红公司购买了咨询服务，有关于孩子填报志愿的，也有关于如何在自己城市选购房产的。花费很大，几年过后看，效果很差。

当时很热门的，人家当年建议她娃读土木，结果她娃刚入学，三道红线出来，她娃一毕业，别说薪水，工地都找不着。还有读者，花高昂的代价去咨询网红专家如何买房，几年后，且不说跌不跌的问题。即便在都跌的情况下，他买的那个属于跌最狠的。太糟心了。

深表同情，但这话咋说呢，也不见得是那帮人故意使坏，非得坑你。他们没能力选出最好的，同样不可能有能力选出最差的。

原因其实只有一个，就是你要解决这些事儿，不属于头疼医头，脚疼医脚能解决的。治病要去根，你不能光盯着患者的脚，脚疼是症状，不是病根，病根也许是痛风呢，你截肢有什么用呢。

所以我今天要教大家一整套的方法论，针对咱们人生当中的这些大事儿。

人生有什么大事儿？男怕入错行，女怕嫁错郎，人怕选错城，更怕买错房。是不是这些事儿？今天给安排上，一揽子解决方案，给你把病根去了。

## 第一部分：引言

我先写一段引言。一开始就讲的很明确，不是别人做的不好，不是那种收费你几万块的咨询类网红公司做的不好，而是很多事情，他做不到那个程度。

举个例子，当年每年开工面积18亿平米的时候，那些咨询类公司看到的状况是什么？是市面上土木的需求很大，苦是苦了点，虽然有些远离城市，但工作终究是能找到的。所以人家基于那个现象建议你娃读了土木，这是他们的判断依据。

那他们不明白的是什么呢？是当人均居住面积40平米，接近国际水平之后，每年的开工面积本来就会大幅度下降。而当这个开工面积从18亿平米跌落到10亿平米时，很多工地关了，学土木的自然不好找工作了，即便不怕打灰吃苦。

你看懂这个问题了么？做投资的，玩的是什么？是预期。买预期，卖事实。他们站在一手消息源头。做实体的那帮人，就是接活儿，雇土木的学生的那些公司老总，他们站在二手消息源。他们要等事实落地，传导到市场，才能感受到。

而那种网红咨询类公司，身处三手消息源，他们是靠着和搞建筑的那类公司的老总搭关系，交流，喝酒，来通过人家的嘴判断一个行业的就业情况。

再把他们探测到的消息，以信息服务的方式提供给填报志愿的家长们。所以你说他们骗了你么？也没骗，因为连他们自己也不知道未来是什么。你非说他们骗了你，只能说，连他们一起被骗了。

难道怪那些开建筑公司的？人家接到活儿，讨不到尾款，他们也是受害者，人家也不是故意骗谁，人家当年缺员工的时候，当然拍着胸脯说，你来嘛，你来嘛，因为他们招不到人，他们咋知道日后呢。

谁站在一手信息高地，在大海的深处像鲨鱼一样盯着变化的苗头？只有投资人。所以我教你的第一件事情就是通过金融视角来看世界。

当你站在金融视角来看世界，这个问题就清晰了。这个世界实际上就是几个集团，你在国内待，你就是选择A集团，你要移居某个国家，那就是要去B集团，以此类推。那么如果我们在A集团内部选择分公司，就是什么？就是选择城市。你去北京就是分公司1，去上海就是分公司2，去杭州就是分公司5，去什么什么，以此类推。

那么分公司里面又有什么？又有业务，这就是行业。

你比如分公司3是深圳，深圳有深圳的强势行业，电子类，它的这个类，在集团内部，就是一哥。换句话说，如果你读了个西电的信电系，你去深圳，会发现你的校友，也许比你在西安时都多。所以你的小孩要报志愿，要选学校，要选专业，要选择读书的城市，你自己要选择行业，你要选择买房的城市，这一连串选择源头在哪儿？在集团上么？

你要这么去想问题就太宽泛了，而且这么讨论毫无意义，只会引起大家吵来吵去。我的观点是，这种话题根本不会有答案，每个家庭有每个家庭自己的想法，各有各的考量与道理。我今天只讨论集团内部，假设集团定了，请问，一切问题的核心是什么？就是分公司，就是城市。

你这辈子最重要的选择并不是选老婆，而是选城市，记住我这句话。你把分公司给我选好了，你接下来很多东西都可以围绕它展开，围绕它进行二次选择。如果你分公司选得一坨，那我告诉你，你这辈子就是拣了芝麻丢西瓜。看着你也很努力，没有努力到相上。

那么请问，选择城市这件事有没有公共答案？没有。全国人民不可能都去一座城市，你有很多天然的限制。比如你出生在乌鲁木齐，从风土人情，从气候习俗，从父母亲人，从各方面的角度考虑，你最远愿意去的地方是西安。那么实际上你的选择范围就只在西北五省内部，不涉及其他地方。

所以我今天会分析讨论很多城市，但是你不需要较真，咱不是来比牛魔王和孙悟空到底谁牛的。咱都不是孩子了，咱都是大人，咱们是根据自己的情况来做一个适合自己的选择。A再好，如果你的选项范围是XYZ，那么对你而言，最适合的就是X。明白我这句话么？

明白了，咱们就开始进入主体内容。

## 第二部分：我们一定要接受一个观念，那就是财富的本质是流通

我问大家一个问题，什么是钱？有人可能会说，银钱银钱，钱就是银子嘛，古人以银子为货币。货币就是钱。

那我问你，穿越到银子还没有被开采出来的年代，你拿着银子能买到东西么？银子难道减损了么？银子难道是假的么？为什么买不到？因为它就没有参与流通。

项少龙拿着港币穿越到秦代，买不了东西的，别人不知道那是啥玩意儿，回到现代，就有价值了。没啥原因，因为在古代，它没有参与流通，回到现代，它参与了。

你去看日本战国时代非常有意思。织田信长家是非常有钱的，可是他们家里并没有金矿。有金矿的是谁家？是武田家，甲斐产金。织田家的钱是哪儿来的呢？俩字：流通。

织田信长的领地，是不设卡的，你从A地贩卖东西到B地，只要在织田家境内，无需缴纳过路费。于是商贸非常繁荣，织田家从此就非常有钱。有人觉得很奇怪，为什么？不收钱的反而最有钱，收钱的大名反倒挣不到钱，咄咄怪事。

道理很简单，我就问你个问题，如果你们家产柿子，像西游记里面驼罗庄那样，漫山遍野都是柿子树，一到收的季节，烂的遍地都是，都沤肥了，它有价值么？没价值。

驼罗庄那几十里的柿子树几十年来沤的肥，奇臭无比，最后还是靠猪八戒变身一只大猪，给拱干净的。村民们非但卖不了钱，还得请八戒吃大餐，才能给当作垃圾一样收拾干净。可是，我们把这些柿子卖到长安去，在长安十二时辰里面，它是什么？它就是张小敬最爱吃的甜品，火晶柿子，吃完水盆羊肉，吸一个火晶柿子，太美了。这就是价值的产生。价值是依赖流通的。

驼罗庄的人天天对着柿子，愁死了，长安人想吃柿子吃不着，不发生交换，谁都没有价值。那个财富就不是财富，那个柿子就只是负担，垃圾清运的负担。

如果西安永远只有西安人，就是那个网络段子，烂怂大雁塔有啥好看的。但如果有了外地人，那就是朝圣去的，美啊，壮观啊，心灵得到升华啦。

我讲这个东西是让你明白什么？是让你去理解城市。

当它是所有华人的香港的时候，它一定发展的很迅速。当它只是香港人的香港的时候，那它的发展速度一定会慢下来的呀。财富是怎么产生的？买出来的呀。
