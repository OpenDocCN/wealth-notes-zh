
# 米联储见闻20250611——闻道龙标过五溪

这个世界，所有的道理都需要实力支持，我一直都说，美国是一个非常务实的国家，川普更是一个非常务实的总统，当中美是两国对立时，很多核心矛盾都无解，但是当美国分裂成两股势力后，很多核心矛盾就变成了博弈的筹码，美国党争正在成为侵蚀美国国家利益的毒液，但即便每个人都能轻易看出问题，身在其中却也身不由己。当年兔子内部神仙斗法时，香港街头人潮涌动，美国隔岸观火，今天回旋镖精准命中靶心，正可谓人在做天在看，苍天绕过谁。

这边厢，伦敦传出积极信息，中美经贸磋商机制首次会议有框架性结果，所谓框架性，就是大家把自己的边界和红线交底，达成了彼此的共识，在这个区域内，可以实施可执行的合作和交易，大家都知道对方要什么，对方不能放弃什么，互相都能接受对方的底线，才能谈上真正意义的合作。这里面一个最大的变化，就是中美谈判不再是美方单方面瞎逼逼，而是真正互相解决关切的对话。

中方会后的表态很常规，还是过去那套，可见中方从一开始就有很明确的诉求，反而是美方的表态一直在变化，这次会谈后，美方的表态是：本次会议取得积极成果，进一步稳定了双边经贸关系，将按照两国元首通话要求，与中方相向而行，共同落实好本次会议共识。这中间很关键的一句话就是与中方相向而行，这是中方一直以来所强调的事情，现在美国终于接受了。

这边厢会谈传出积极消息，那边厢，路透社透露，兔子商务部已向美国三大车企的稀土供应商发放临时的出口许可证，通用汽车、福特汽车和斯泰兰蒂斯的供应商已获得部分稀土出口许可。商务部同时表态，愿意为符合条件的稀土出口申请提供绿色通道，以加快对欧盟企业的批核流程。上述车企获得的稀土出口许可至少部分有效期为六个月，但目前并不清楚具体获批的出口量、具体产品类别。这个结果表明，兔子确实是拿稀土拿捏了美欧。

在美方释放缓解稀土的消息同时，兔子这边也发布了一个消息，就是长和出售海外港口资产的问题。说明长和卖资产这个事情确实不是李家能说了算的，当年他能买以及他现在能不能卖，都不是他能输了算的。而且这次不是李家出面表态卖或者不卖，而是巴拿马运河管理局跳出来说，目前交易结构或将导致港口所有权过于集中，损害巴拿马市场竞争力，也不符合中立原则。并称不会允许美政府船只免费通航。你看到没有，当大哥们在台面底下把利益谈妥了，小弟们就都好办了，各归其位，在商言商，通情达理。所以解决问题，就要直面根本矛盾，本身是中美关系中的暗礁，骂老李再狠也没用，而中美的矛盾最终是双方实力决定的，实力不够，矛盾就是矛盾，根本无解，实力到了，矛盾就是形势逼人强，不行也行了。

实际上，中美这轮沟通效率非常高，5月初次接触，6月领导通话，三天后再次接触谈两天，就达成了“框架协议”，延长世界上两个最大经济体上个月达成的贸易休战。远比川普1.0时代的谈判节奏快很多，说明美国有自己的难处和紧迫性，确实也是等不起。谈判小组成员之一、商务部长卢特尼克表示，美国对兔子限制稀土矿物和磁体出口的担忧已经得到解决。他还说，在协议实施之前，川普和领导将听取有关协议的简报。更有意思的是，他说了一句：“他们专注于努力实现习主席告诉川普总统的内容，我认为双方都有额外的动力来完成工作。”这说明中方在落实领导意图方面不遗余力，而美国也对此表现出了配合态度。

回顾这一段时间以来，中美的斗争过程，有一件事很明确，要想在斗争中获得话语权，卡住对方的七寸才有话语权，所以对中美来说，这几年比的就是谁的短板少，谁能拿捏对方的短板，谁能死死卡住对方的供应链。兔子出手晚，但出手时已经占据绝对的胜算，不但茶能垄断，而且时机精准，把王牌留到了最后，不出手则已，一出手卡死，而且是完全不做利益考量，相比之下，美国在卡兔子芯片这件事上，利益考量太多，绳子一寸寸收紧，又担心一下子卡死了经济利益受损，结果给了兔子极大的时间和空间去解决问题，争取活路，结果不但没有卡住，反而让对方长出了义肢，脖子粗到卡不住了，优势就不再是优势，而造成这个局面的根本，还是社会制度决定的，权力分散，利益分散，就会导致在斗争中，力有不逮，落于下风。过去美国之所以一直打顺风局，实在是因为没有碰到过组织能力如此强大的国家。美国在打压兔子这件事上，触及的利益方实在太多，决策太难，很难形成真正的致命打击，所以美国的压力虽然大，但耐力，打击力度，并没有造成毁灭性的影响。都到今天，一个是慢慢打光所有的牌，一个是不出手则已，一出手直接掏裆，虽然不要命，但也是真疼呀。

但是也不要指望这次会谈就能让美国从此老老实实，川总是一个反复无常的人，你想要他不搞事情，最好的办法就是不断的给他找麻烦，只有不断的给他找麻烦，他才会没时间找你麻烦，所以目前川总跟民主党，跟纽森的斗争是值得期待的，但这里面很微妙的东西在于，如果把川总搞没了，等于白谈了，如果川总快速解决问题，他歇下来必须又要给兔子找麻烦，所以让川总保持高频深陷党争的状态，最符合兔子利益。这个时候与民主党多年以来形成的各种明的暗的关系就有用武之地了。

对于这次中美谈判，纽约时报也作出深刻反省，感叹美国明知道稀土短板是美国的致命伤，但是过去十余年间，美国始终未能建立替代供应链，获取制造导弹、战斗机、智能炸弹等各类军事装备所需的关键稀土资源。以钐金属为例，兔子基本是独此一家，而这种鲜为人知的稀土金属几乎全部用于军事领域，钐沽磁体能在足以熔化铅块的高温下保持磁力不衰减，对导弹整流罩等狭小空间内高速运转的电机耐热性能至关重要。虽然这次谈判兔子放开了制造汽车需要的含镝、铽两种受限稀土的磁铁出口许可，但这两类磁铁无法满足军用级别的高温需求，而关于钐的出口，兔子至今没有松口。钐在美国的主要用户是航空航天和军事承包商洛克希德·马丁公司，每架F-35战斗机上大约要使用23公斤磅钐沽磁体。而洛克希德·马丁公司因为向台军售还在兔子的制裁名单上，这意味着兔子几乎不可能对这家公司放开供应。拜登曾经努力解决这个问题，签订了建造两个钐生产设施的大额合同，却因商业可行性问题搁浅，以至于明明知道这会是致命短板，美国却没有办法解决。结果在俄乌冲突巴以冲突打光美欧库存之际，兔子一道禁令就让钐供应中断了两个月，而此时正值美国及其欧洲盟友急欲补充先进武器库存。这里面兔子耍了一个心眼，虽然兔子很早就对涉台军售的美国军火承包商实施了制裁，但这些制裁措施是禁止兔子企业和个人与美国军事承包商有任何财务关联。但是没有锁死这些企业获得钐，因为兔子将钐出口给化工企业，这些公司将钐与沽混合后出售给磁体制造商，后者再出售给军事承包商。所以，这些军工企业一直觉得自己抄了兔子的后门，躲在化工企业背后洋洋得意，感觉兔子制裁是个花架子，也就未对钐的供应做任何战略储备，依然遵循商业原则，用多少买多少，甚至连增加库存都不愿意，但是4月之后，新出台的稀土出口管制措施明确规定，许可证的发放将严格依据供应链终端用户的性质进行审批。就钐的出口许可而言，这就意味着终端用户就包含军火承包商。兔子限制出口的七种稀土金属中，有六种主要需求来自民用领域。但钐几乎完全用于军事目的，美国国防部的规定虽要求军用磁铁的铸造或冶炼必须在美国或盟国完成，但允许原料从任何国家进口，因此多年来低成本的钐一直来自兔子。其结果是，跟兔子骂骂咧咧搞了十几年，枪把子居然拿捏在别人手里。真是对美国佬的心大服气，这典型就是童年期没挨过揍的体质，美国现在的中坚阶层确实就是一帮没挨过揍，也没挨过饿的霸王龙，自然也就没有所谓战备这根弦了。
