
# 问答精华汇总(1)

灵哥认为大家要多关注上下游整合的公司，这是国家鼓励的方向。例如神华的垂直整合模式，拥有铁路等资源，可以抗风险。小米也是从软件到硬件一体化整合。过去的平台模式将逐渐被淘汰，消费者会更有选择。

垂直整合的过程实际上是抢夺了中间商的利润，减少了寻租，符合国家统一大市场方针。在北京，很多餐厅也是符合这个原则的，比如灵哥经常去的干岛湖鱼味馆，就是干岛湖集团到北京开的，鱼都是干岛湖运来的，这种餐厅往往能提供价廉物美的菜品。

大众点评、小红书等平台上的高端餐厅，做得并不一定好吃。京东现在搞外卖，很多人不以为然，觉得市场竞争激烈，京东参与进来吃力不讨好，对业绩也未必有帮助。但京东的物流都是自营的，这是重模式，可以抗风险。京东搞外卖，现在给商家零佣金，可以暂时不赚这个钱，但他可以赚即时物流的钱，可以带来用户运营上的便利等。比如灵哥买京东的积分，也可以用来抵扣外卖。

当前很多企业的打法都是垂直整合，泡泡玛特和设计师都是深度签约和绑定，除了盲盒机，还有泡泡玛特乐园去扩大垂直品类用户的消费场。归根到底就是用户价值！能给用户价值的企业最后必然优胜！

港股跌，有些资本借李黄瓜事件煽动一下，顺便获利了解，是正常的。对冲基金往往就是这个德性，哪有套利机会全球来回跑，美国停止紧缩美股要反弹一下，那也是要拥抱的机会，亚洲跌了又可以再加回来。对今年港股，灵哥乐观预期不变，现在外国投资很多长线资金要回归港股，不少还在路上。

提问：对于一个初进股市的新手，如何系统地学习，逐步进阶更高的阶段？目前纯小白一个，希望指点得越详细越好。

灵哥：从K线来说可以学布林线、均线、神奇九转等指标，记住均线EMA、MACD、SAR、RSI、CCI。多关注下周线、月线。这都是基础知识，但是光学K线很容易刻舟求剑，K线只能辅助，最关键是研究公司业务基本面，还有就是宏观基本面。公司业务基本面其实就是培养自己做生意、商业底层逻辑的能力，即什么生意在一段时间内会好？投资就是自己不做，看别人做参与别人的成功。比如现在和以前相比，高端消费更难做还是更容易做？还有小米造车，是不是为公司开启第二增长曲线？等等，这些自己要有认知和判断，财富就是认知变现，说的就是这个意思。没有什么捷径。宏观政策往往和货币政策相关，和国际博弈也相关。

提问：金山云这是怎么了？金山办公也一蹶不振了。

灵哥：最近就是科技股调整，只允许涨不允许跌？美联储昨天不降息预期之中的事情，灵哥之前已经写过了，特朗普上台时间太短，和美联储很多事还没有勾兑好。现在美联储很尴尬，想把股市的锅推给特朗普的关税，特朗普写了个帖子，意思是美联储应该随着加关税降息就不会造成衰退的问题。现在是双方甩锅，那么美联储背后毕竟是那么多银行大佬，所以只能宣布停止缩表，做较为鸽派的预期，美股因此而反弹。

美国各种雷太多了，资本也在流出美国。灵哥的一个朋友在欧洲做移民中介，前一天来咨询的美国人有1、2个，现在有十几个。
