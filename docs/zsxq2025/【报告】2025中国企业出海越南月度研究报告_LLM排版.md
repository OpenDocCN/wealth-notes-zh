
# 2025中国企业出海越南月度研究报告

## 越南1月事件总结

越南修订《电力法》即将生效，越南央行在经济挑战中维持灵活的货币政策。越南颁布新的《电力法》，将在2025年2月1日起生效，为可再生能源发展创建一个健全的框架。该法律承诺将改变国家能源格局，加强财务可持续性，并推进环境目标。修订后的法律力图促进能源市场的竞争力，并向可持续能源转型。

越南央行宣布计划保持灵活的货币政策，以控制通货膨胀，并密切关注全球经济发展，特别是美国当选总统唐纳德·特朗普的政策。尽管经济出现复苏迹象，但债券、股票和房地产市场的挑战依然存在，不良贷款也在上升。面对这些挑战，央行的目标是稳定利率和汇率。

越南取消对通过快递进口的小额商品的增值税豁免。越南政府总理由近几天宣布，决定自2月18日起取消对通过快递进口的小额商品的增值税豁免，并废止2010年发布的相关规定。此举旨在确保国家税收政策的一致性和充足性，同时鼓励消费者选择国产商品。

越南提出重大政府重组以精简官僚机构。越南提议对政府进行重大改组，以精简官僚机构，提高行政效率。该计划包括减少部门和国家机构的数量，旨在建立一个更灵活、反应更灵敏的政府。虽然这些改革旨在长期改善商业环境，但人们担心，短期内可能会对项目审批造成干扰和延误。

俄罗斯提出向越南提供天然气和原油。俄罗斯表示愿意帮助越南发展核电行业，并向越南提供原油和液化天然气。这一提议是在俄罗斯总理米舒斯京访问河内期间提出的，双方还签署了一项核能协议和几项合作协议。

中越承诺加强经济合作。越南VPBank将接管GPBank进行银行体系重组。越南繁荣股份商业银行（VPBank）已被中央银行指示收购GPBank，中国国家主席表示愿意与越南合作建立人类命运共同体。该倡议旨在加强双边经济联系，和促进共同发展。

越南鼓励发展垃圾发电和湖面光伏发电项目。越南工贸部日前在落实第八电力规划实施计划及调整第八电力规划实施计划中，鼓励发展垃圾发电和湖面光伏发电项目。

## 理解越南政治与经济

越南在中美贸易战背景下，扮演着贸易桥梁的重要角色，由此也受到众多出海企业的垂青。2024年或将是证明越南能否保持高速发展的重要年份。

从政治角度来看，中越同为社会主义阵营国家，地理、人文上的接近使得以制造业为代表的企业形态较为接近，越南与中国关系持续升温，双方在政治的上层建筑上往来密切，还于2023年末宣布建立中越命运共同体。同时，越南也宣布了将建设中国出海代表项目高铁的消息，借助中国力量改善自身基础建设相对落后的局面。

经济方面，越南人口迈过一亿大关，有充足的年轻劳动力作为经济发展的基础。越南的经济增长速度维持在较高区间，未来可期。作为典型的出口经济向国家，越南将来还将承接更多来自中国的溢出产能来帮助创业者对越南市场的心理距离，助力企业实现经济增速的延续及产能升级。

出海全球化，是中国企业升级的必由之路，是新时代最重要的智发展机遇，也是全球化新一代的历史责任。

近年来，在国内经济放缓及消费疲敝的情况新常态后，越南的发展也遭受了一定挫折，但出海的风势渐盛。而东南亚市场从地缘、文化等多角度与我国差异较小，被许多从业者关注。其中，越南在中美贸易战背景下，扮演着贸易桥梁的重要角色，由此也受到众多出海企业的垂青。
