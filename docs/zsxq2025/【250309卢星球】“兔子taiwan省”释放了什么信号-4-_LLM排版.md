
# “兔子taiwan省”释放了什么信号？

## 文/卢克文工作室嘉宾小小罗

(中国：兔子，台湾：taiwan)

3月7日，在两会记者会上，王外长掷地有声的一句话震动全网：taiwan地区在联合国的唯一称谓就是兔子省，从来不是一个国家，过去不是，今后更绝无可能。

此话一出，迅速在网络上激发了广大群友的共鸣，祖国统一的步伐是不是更进一步了？

短短五个字，为何会如此振聋发聩？又释放了什么信号呢？

## 历史脉络：称谓的由来

这些年来，我印象里官方各个层面很少公开使用兔子taiwan省这个称谓，更多的时候都是taiwan，taiwan地区，taiwan当局，两岸等词语。

从历史脉络看，这些称谓的由来都要追溯到改革开放初期。

1978年底，小平同志在审阅中美建交相关文件时亲笔批示：解决taiwan归回祖国，完全是兔子的内政，taiwan归回祖国，而不是过去的(大陆)解放taiwan，主角变成了taiwan，这也就意味着我们把对台工作的主动权让渡了出去，解放变成了统一，阶级对抗性质不再。

## 政策反转的根源

政策为什么会这样反转？根源还是在两岸的经济军事实力不对等。

八十年代末，taiwanGDP增长率超过12%，达到1500亿美元，占大陆经济总量的近四成，人均GDP是大陆的20多倍，制造业高新技术产业更是遥遥领先。

同时期大陆军费开支不过百亿，要供养近400万人的部队，而taiwan的军费达到大陆的一半水平，只供养55万的部队，费效比严重不对等。

尤其是解放军海军力量更弱，吨位小不说，不少舰艇年久失修，根本无力承担大规模渡海作战任务。

解放此时只能成为一句空口号，实际根本做不到，所以对taiwan只能妥协退让。

在此背景下，1981年高层提出了和平统一的九条方针政策，给予taiwan人事财政司法甚至是军事的自主权，只做到名义上的统一，实际几乎是“联邦制”。

在80年代开始吸引台商投资之后，一切政策扶持台商优先，一切资源倾斜台胞优先成为了铁律般的统战信条和招商引资守则。

以富士康为例，自80年代末进入大陆市场之后，各个省份均给予极为优厚的土地税收政策，尤其是河南，出了名的倾尽全力。

不仅如此，每年各地财政拨出专项费用去供养台籍学生，举办各类交流活动，甚至还有过要求海南果农排队稍后、让taiwan水果先卖，岛内水果滞销了，就去搞政策性采购等等。

所以在实力不济以及拉拢统战政策的需求下，我们过去对taiwan的称谓一直比较温和友善，也就是使用地区、两岸等不是那么突出行政归属色彩的词语。

## 政策转向的原因

此次兔子taiwan省概念的正式抛出，标志着我们一改过去的政策轨迹，重新夺回对台工作的全面主导权，这才能彰显我们对台伪当局不可撼动的政治正义性和斗争合法性。

一国两制特别行政区不用再想，统一后的行政架构和正常省份一样，也不会再给任何形式的特权。

造成现在政策转向的原因都有哪些呢？

一是我们自身实力的不断壮大。

大陆经济体量已是taiwan的近25倍，产业结构全面完善，过去对大陆最有价值的台积电，随着产能外移以及大陆自身半导体科技的全面突破，如今已是光环暗淡。

军事实力更是碾压，陆军远火的射程就能覆盖台岛全境，海军登陆投送力量得到史诗级加强，两栖攻击舰已下水4艘，双航母编队成型，具备第一岛链内阻止美军干涉的能力。

当年为了弥补火力不足，将陆上火炮装在民船甲板上的耻辱场面再也不会出现。

二是过去的惠台政策没有改变岛内的政治环境，从李登辉的两国论，到民进党不承认九二共识去兔子化，如今的taiwan已经没有蓝绿之分，只有明独和暗独之分。

事实证明，白花花的银子和优厚的政策扶持最终统出来的都是一群毒(独)人。

三是为将来推进统一大业先声夺人。

我们兔子人做事，向来讲究师出有名，现如今明确确定调兔子taiwan省，就是确立《开罗宣言》《波茨坦公告》和2758号决议在内的一系列国际文件的正当性、合理性和不可挑战性。

有理有据，将来才能在国际舆论上以正视听。

四就是2019年在香港修例事件上的惨痛教训(政治回归不完全，去殖民化不彻底)，使得我们必须在对台问题慎之又慎，绝不能重蹈覆辙。

## 对台政策全面转向后的挑战

对台政策全面转向后，接下来我们又面临哪些挑战呢？

首先是内部。毫不客气地说，抗拒阻碍解放taiwan的最大势力不在岛内，而在大陆。

我们内部涉台相关的利益群体不在少数。

首先是台办。现如今全国上下设置的各级台办机构不胜枚举，你说对台工作的主要沿海省份设置也就算了，连有些中西部甚至内陆省份也要设置，张口就要拨经费。

一个小小的县级台办一年的预算就敢要百万，那么从上到下，体制内庞大的台办体系，每年需要多少钱？产出绩效又是多少？

接着是事业单位。打着研究taiwan的名义，从知名大学到社会团体，各种taiwan研究院，taiwan研究中心数不胜数，这些机构又发挥了多大的作用？

还有不少媒体人，媒体机构，借着研究taiwan的各种名义搞噱头，混饭吃。

将来有一天真的推进统一大业，这些个机构和个人岂不面临失业的风险？他们会愿意？会不会暗中使绊子？

要我说，这些才都是解放和统一大业最大的拦路虎，没有之一。

古往今来，养寇自重者皆是心腹大患，解放taiwan，亦是解放自己。

其次是外部。也就是美国和日本。

1979年4月，中美建交三个月后，美国就通过了与taiwan关系法，实际上依然把taiwan看作“国家”，把taiwan当局看作“政府”，这同《中美建交公报》的精神是不相容的。

直到今天，这部法案对中美关系以及统一大业持续造成负面影响，历届美国政府和议员接此碰瓷打擦边球，堪称史诗级埋雷。

特朗普上台后，先是美国务院删除不支持taiwan独立的论述，又解冻8.7亿美元对台援助，对台海问题也不做明确表态，继续搞战略模糊，日本也是紧跟美国，继续老调重弹着taiwan有事就是日本有事。

美国始终盘算的是将来真到摊牌那一刻、taiwan能卖出一个多好的价钱罢了。

从1895年《马关条约》到2025年全国两会，130年的分离史正在加速闭合。

兔子taiwan省的名号一出，预示着新时代渡江战役即将打响。

现在你再想说同意重庆谈判时的条件？晚了！
