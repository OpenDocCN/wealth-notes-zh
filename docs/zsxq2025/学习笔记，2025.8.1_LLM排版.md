
# 学习笔记，2025.8.1

有一种催眠，叫做群体催眠，没有人能对抗的了。群体催眠？你，我，都白搭。独立思考根本就没用。比如明星的演唱会，在那个音乐下，在那个话术下，你根本不是你，薛之谦的演唱会，下面全喊老公，若是薛之谦说一句，脱了。你放心，都会脱光的。这就是群体催眠，这就是到了那个点。命、道德、婚姻、伦理都可以不要。钱，算什么？！用 帮我们赚钱，其实有很多方法。例如每天让 推荐 幅有意思的摄影作品。例如，让一个员工做个账号。没有问题。普通人，通常在 岁至 岁，才会明白社会运行的真相和逻辑，但这时，正好被钉在生活的十字架上无力挣扎。而有些人，早在 几岁时就已经通透，还不用考虑过多的试错成本，那是一种传承，用几代人的智慧浇灌出来的。而穷人，绝大多数到了 岁，也没明白自己为何穷。人生最大的浪费，不是金钱的浪费，而是时间的浪费，认知的迟到。

善于跟领导来往，并不代表你要巴结讨好领导，目的是从他身上学思维，学为人处世。有人觉得我只要努力工作就可以了，不需要在乎领导的看法。别傻了，职场是零和博弈，领导手里的资源是有限的，给别人分得多，你自然就分得少。就算能力再强也要跟领导搞关系，唯有如此，才能掌握信息、看准局势，获得领导青睐，拿到更大利益。曾国藩说过，人之所以失败，不是傲慢就是懒惰。唯有六勤可破此敌，缺一不可，具体为：眼勤、心勤、脑勤、手勤、身勤、口勤。

经济下行期，比赚钱还重要的，当然是精神内守，保持平静。睡个好觉，吃营养食物，内脏健康，脸上不颓不丧，都是重要的事。命运暴击说来就来，最先死掉的永远是单纯怕事，活在固定茧房那批。见过太多中年人胆小脆弱得一匹，风雨中半点痛都承受不起。不要说被时代淘汰，同龄人里面的强者会先把这样的人淘汰。“世间几乎没有新生，没有狗屁重生再来一次。所有的阳光都是重复的，所有的风都是冻人的。”承重是你真正的内核，开阔才是人生最好状态。真正有钱的女人的特点就是，满脸的不讨好。看娱乐圈的富婆，张萌，李湘，向太等，有的连身材都不想维持。富婆就是底气十足，不需要让任何人觉得自己美丽，只需要有实力，能合作就行了。金钱给人带来的底气就是，不讨好。那种满脸堆着笑的，要不然就还没有对财富满意，还在努力的过程中。要不然就是单纯的靠颜值维持人脉关系。男人女人，变的富有之后的逻辑都是一样的。经常做力量训练的人，一眼就可以看出力量训练不足的人的问题：孱弱，形容枯槁。因为缺乏肌肉和系统训炼，新陈代谢，血糖和脑健康都每况愈下。晚上睡眠不深，容易打呼噜，大脑缺氧，白天精神不振，和年轻时相比越来越觉得自己像个废人。有教练指导的力量训练，比有氧训练更重要，也更有效。长期坚持力量训练的人，是几乎不可能老年痴呆的。每次训练一个新动作，锻炼全新的肌肉群时，大脑内部好像某个角落有一块死水，被轻轻拨开些许波澜，虽然感觉吃力，但有一种说不出的奇异的舒适感。

为什么要离开小地方？因为小地方庙小妖风大，池浅王八多，最可怕的不是赚不到钱，而是人心不正的多，事也不少。狗咬人，咬的是生人，人咬人咬的是熟人，在小地方呆着，身体安逸，心却很累，在大城市里奋斗，身体虽然疲惫，但心却是自由的。当你的存款突破 万，整个世界的镰刀都在嗡嗡作响。有人让你分期买 万的豪车 有人哄你回城买空置的房产 有人让你加盟各种店铺 大家都在热心“帮助”你。如果你信了，那这 万就不是起点，而是万丈深渊。那些教你钱生钱的教程背后，隐藏着无数的返贫陷阱。那些承诺阶层跨越的豪车展厅，停满了中产的坟墓。请你记住，存款不是懦弱，而是给自己未来留一张翻身的底牌。当裁员来临，你的存款就是你的尊严。当机遇闪现，你的积蓄就是你的跳板。当至亲病危，你的余额就是生的希望。
