
# 学习笔记，2025.7.22

1. 越南摩托车超级多，没有秩序感，台湾摩托车也多，但是井然有序，越南那边有点咱小时候的感觉，坐在河粉店里吃饭，能听到此起彼伏的喇叭声，有摩托车的，有汽车的，有大货车的。
2. 短视频时代，很多火具有偶然性以及羊群效应，不需要理由，大家都说一个地方好，接着就都往那跑，什么人间烟火气，这些其实都是刻意拔高的，淄博也有看守所，也有监狱，好人比例与坏人比例，跟全国各大城市都差不多。
3. 《中国好声音》里第一期有个丁丁，唱《爱要坦荡荡》的那个，她获得YES的时候说，她从小就喜欢唱歌，感觉整个生命就是为唱歌而生的。千里马常有而伯乐不常有。不过呢，也不用担心，这是一个有才藏不住的时代，她可以开抖音，是真热爱，真有才，一定会脱颖而出的。
4. 电车充电一定越来越贵，因为充电桩想赢利就必须提高电价，最终的结果就是电车行车成本接近于油车，现在已经有这个趋势了，高速上我充的电，基本都在1块5 以上，平均起来3毛钱一公里。电车，在城区转转，家里有充电桩，平时充电两三毛一度，那么，的确省钱。若是跑长途？不怎么省钱。当然，有些司机会研究，哪里充电便宜。
5. 大部分岗位在AI面前，都已经没有意义了，过去我觉得AI离我们很遥远，现在觉得是如此的牛逼，它比搜索引擎牛逼的地方在于，它能听懂你要什么？你让画图它就画图，你让做设计它就做设计，还可以写剧本，现在写一些不带感情的文章，我觉得比人写的好多了，带感情的，也用不了多久就超越人类了，前段时间，我逢人就科普，买特斯拉，现在我逢人就科普，抓紧下载豆包，我连我儿子都给科普，必须马上下载。
6. 我做事，有两个阶段，第一阶段，是没有脑子的时候，只要我遇到一个高能量场的朋友，他的话我就当圣旨，他怎么安排我怎么做，哪怕我有自己的思想，有自己的想法，我也全部先摁下去，认真执行，我就是相信他。第二阶段，我自己独立做主后，我依然会咨询高能量场的朋友，我肯定会有自己的的一些想法，我不会再压抑了，而是会提出来，我这个想法对不对？为什么不对？得到一次辩论的机会，从而我会优化一些想法。人穷的时候，一切独立想法都是错的，是对的，你咋可能没赚到钱呢？所以放弃自己的想法才是关键，但是越穷的时候，人越固执，就如同我跟一个读者讲，当你迷信一些想法时，就多问问AI，他问了AI一个问题，有没有一款中药通过了双盲测试？结果，AI回答有，他截图给我，我一看，里面明确写着，试验组和安慰剂组无差异……
7. 下一次收入差距，我认为就来自于AI应用的熟练程度，你有没有拥抱AI，有，是一个结果，没有，是一个结果。现在有一点是肯定的，未来我们不能被取代的技能越来越少了，余华在谈到AI的时候，说AI缺少灵魂，缺少温度。但是，按照我现在的感觉，AI有温度用不了多久，因为喂养的数据足够大的时候，它可以模拟任何一种情感，比人还懂人，甚至未来很多人会跟AI谈恋爱，一个AI可以跟无数人同时恋爱，根据你的性格，它会给出不同的方式，让你觉得爱的恰到好处。
8. 对AI的应用，我认为提问的第一前提是，问具体的问题，而不是一个面，要问一个具体的点。人与人的交流，其实一坐下，从两三个问题，就能判断对方深浅。我现在的理解是，AI约等于千军万马，过去，我们是一个士兵，今天，我们是 一个将军，就是我们的能力，被放大了千倍，从而我们会比别人收入更高，会给我们带来指数级的收入。
9. 永远不要发有争议的内容，什么是有争议的？现代医学与中医，俄乌战争。你哪怕是调侃，是反话，也不行。因为，有50%的人是不认可的，会默认为，你是异教徒。你可以在文章里，在日记里，表达自己的信仰。但是，在朋友圈里，不能。
10. 没有冲突就没有热度，很多人就是因为怕冲突从而不敢表达，总希望把话说的很圆滑，谁也不得罪，这样的人成不了大V，成为大V的人，都是爱憎分明，谁爱咋着咋着，我就表达我的观点。能脱颖而出的特斯拉车主或蔚来车主，要么死忠粉，要么超级黑，要想成为大V，必须要了解什么是榴莲型账号，就是喜欢的人，喜欢的不得了，不喜欢的人，闻着就恶心。
