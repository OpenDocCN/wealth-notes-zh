
# 关税与媒体

过去一周，你最频繁听到的一个词，很可能是：关税。川普发起的“关税风暴”不仅把全球市场搅动得上蹿下跳（主要是下跳），而且让“美国经济即将陷入衰退”、“国际秩序就此改写”等说法被热议。你可能已经不想再听到“关税”这个词了（其实我也是）。不过，鉴于这件事情的重要性，我们还是来谈谈它，尤其是谈谈它和媒体行业之间的关系。

## 关税会影响媒体行业的发展吗？

关税是施加在进口的商品之上。乍看上，只有跨越国界的实体商品才会受影响，服务业并不在受影响的范围内。但其实，生产无形的精神产品的媒体行业同样会遭到高关税的打击。

一个最直接的原因就是：媒体行业高度依赖广告收入，而如果大部分的广告商都受到了关税的负面影响（产品成本增加、消费者支出减少），它们势必会减少广告投入，也就会显著降低媒体行业的营收。比如，汽车行业一直是广告大户，但是因为汽车零部件往往要从多个国家进口，一辆新车在美国的售价预计将会因为川普的关税而增加3000美元，而这势必会影响销量，从而降低车企的利润，进而让车企的媒体广告投放变少。

根据花旗银行研究部门的预测，以广告收入为营收大头的媒体公司今年的收入可能会下降4%，而不那么依赖广告收入的媒体公司今年的收入也可能下降1-2%。

尽管最新的消息是，川普已经对绝大多数国家的关税提升按下了90天的暂停键，但是一家国际新闻媒体的高管表示：“这不太可能不会对营销预算产生不利影响，而且我们也没有读到任何表明关税只是短期政策的信号。”

另一位行业高管则表示，他们非常担心川普的关税政策对那些进口到美国或供应链较长的奢侈品牌的直接影响，因为它们提价的空间已经非常有有限。这将不可避免地影响它们的营销预算以及与媒体的合作协议。

时尚和美妆行业的确因为关税风暴而高度紧张，因为它们非常依赖全球采购的香料原料和包装材料。而相应的，高度依赖时尚品牌广告投放的媒体就会受到连带影响。

此外，美国媒体的生产材料也会受到关税的影响，比如从加拿大进口的新闻纸（占到美国媒体用纸量的80%），以及从多个国家进口的科技产品（比如相机、硬盘）。这些产品不是那么容易就找到本土替代的——美国本土只有三家造纸厂生产新闻纸，它们的产能远远满足不了媒体的需求。

在陡然上升的经济压力面前，一些小型的地方媒体已经开始考虑裁员、降薪等自救措施。例如，有大约150年历史的纽约州地方报纸《The Cortland Standard》在3月13日宣布关门，核心原因之一就是新闻纸可能随关税增长而涨价25%的预期。

## 站在投放广告的商家这边看

受到关税影响，大家的投放可能会加剧两种已有趋势。第一，商家绕开中介代理机构，直接和媒体达成交易，节省佣金支出。第二，商家更加在意投入的每一分钱都要有产出，因此可能会倾向于选择那些通过技术来提升效率的投放方式。

《福布斯》的一篇报道则指出了一个很有意思的现象：来自中国的不少跨境电商从业者，并没有被高达104%的关税吓到，他们觉得自己在亚马逊等平台上依然有优势，因此在广告投放上会更加激进。比如，他们会大力投入品牌广告（而不仅仅是产品广告），甚至会对美国本土品牌构成威胁。

受到关税影响的不仅是媒体机构，也包括个体内容创作者。一方面，他们同样深受广告市场下行的影响；另一方面，不少头部创作者都推出了自己的周边产品，乃至个人商品品牌，这些实体的物品都会受到关税的影响。例如，美国直播网红QTCinderella就说，关税将影响到她的饰品品牌，因为该品牌的大部分材料都是进口的。她透露，“我的一些供应商已经撤出了美国，还有一些供应商将价格加到了三倍或四倍。”

YouTube头部网红MrBeast则表示：川普的关税本意是要让制造业回流美国，但讽刺的是，由于新的关税，现在MrBeast旗下品牌反而会选择到美国之外去生产，因为其他国家不会对原材料征收20%以上的关税。

而在TikTok上活跃的一些读书类博主们（被称为BookTok社群）则不断发布短视频强调，关税会让书价变得更贵。还有书迷会担心之后能不能用正常的价格购买到海外版的图书（有的大热作品会在英国或其他国家首先发布）。

## “赶热点”成功的媒体和内容创作者

虽然关税可能重创广告市场，但至少在过去一周，媒体也有因为川普的关税政策而得益的地方，那就是：被每天变化的新形势搞得一头雾水的大众，涌向了媒体去阅读相关报道，试图理解究竟发生了什么。换句话说，川普又给媒体带来了流量。

《纽约时报》报道说，据尼尔森公司统计，从4月2日川普宣布在全球范围内征收关税到4月7日，福克斯商业频道的收视率与前一周相比增长了25%，与去年平均收视率相比增长了35%。

同样，另一家财经类的电视新闻频道CNBC的收视率也出现了飙升。尼尔森公司称，该频道在4月7日上午9点到下午6点的平均收视率比11月份以来的工作日平均收视率增长了75%。

当然，在这些狂飙的数字背后，是各家媒体的工作人员被不断出现的新消息弄得团团转。福克斯商业频道的主播Liz Claman就说，过去的五个交易日是“疯狂的”。“我们一直在调动嘉宾、取消嘉宾、增加嘉宾——半夜里，我醒来给我的制片人、执行制片人和嘉宾预订人发短信。我们进行三方聊天，我们三个人晚上每个小时都要起来一次。我们会说：‘他怎么办？’‘这个人怎么办？’‘让这个CEO来吧！’‘暂缓！’‘取消他！’……”

在社交媒体平台上，很多内容创作者都在赶热点，产出关税相关内容。一个例子是，数学内容创作者Stand-up Maths发布了一段解释白宫提供的关税方程式的视频，在YouTube的热榜上获得了排名第19位的好成绩。

擅长对公共政策做解释性报道的Vox也是火力全开，推出了解读川普关税政策的系列专题，内容丰富且不断更新，其中包括：

- 川普因对一项统计数据的离奇误读而导致经济陷入混乱；
- 上一次美国全面征收关税时发生了什么？；
- 川普的关税真是一团糟。但关税总是坏事吗？；
- 任天堂Switch 2的延期如何解释川普的关税政策；
- 两张图表解读川普股灾的真正输家；
- 目前最好的理财建议是最反直觉的建议；
- 川普的关税是中国的机遇。北京能抓住它吗？；
- ……

在中文世界里，晚点团队对20多位全球贸易从业者的采访，展现他们的工作细节和应对打算，为信息的拼图提供了重要的补充。

## 虚假头条与极端渴望好消息的市场

在疯狂的新闻周期里，真真假假的信息都触动着风声鹤唳的市场。4月7日，一则乌龙式的消息就在短时间内让市场暴涨暴跌——可能不少人都曾在微信群里看到过那则消息。

事情的原委是这样的。当天开市之前，福克斯新闻播出了对国家经济委员会主任Kevin Hassett的采访。记者在采访中询问：白宫是否会考虑暂缓关税计划。而他的回答是：“我认为总统将决定总统要决定的事情。有50多个国家正在与总统进行谈判。”

也就是说，他并没有针对“暂缓”给出任何确定的回答。然而，当天上午10:13，一个在X上有约百万粉丝的账号Walter Bloomberg发布消息，称川普正在考虑对中国之外的国家暂缓90天增加关税（HASSETT: TRUMP IS CONSIDERING A 90-DAY PAUSE IN TARIFFS FOR ALL COUNTRIES EXCEPT CHINA）。

这个账号的名字里虽然有Bloomberg这个词，但是它和彭博社并无关系。账号多年来一直在交易日发布与交易相关的新闻，通常采用从彭博终端机复制粘贴的全大写通讯社标题的形式。这个账号深受那些用不起彭博昂贵的终端机的市场观察者的欢迎。

账号的运营者表示，他是从另一个叫T3 Live的账号看到消息的。而X上最早的相关消息，是在10:11由一个叫做Hammer Capital的、仅有692个粉丝的账号发布的。

也就是说，Walter Bloomberg这个账号错误发布了没有可靠来源的信息。

如果事情就到这里，那么影响力还会比较有限，因为Walter Bloomberg尽管有百万粉丝，但毕竟，认真的金融从业者都不会认为这个账号是权威的信息来源。

更大的问题出在：正规的、有公信力的媒体，在没搞清楚的情况下就急忙发布了消息。

10:15分，也就是在相关消息在X上出现仅仅几分钟之后，CNBC的主播就在直播当中播报了这则消息。不过，主播也提醒：他们还在努力寻找和查证这则消息的来源。

在查证的过程中，CNBC的一名员工错误地认为该消息已经确认。10:18，CNBC就在屏幕上打出了字幕，称川普正在考虑90天的暂缓期。

两分钟后，路透社发出消息称，据CNBC报道，川普正在考虑90天的暂缓期。路透的消息传给了全球媒体。

受这个消息的刺激，标普500指数迅速从-4.7%拉升到了3.4%，市值增加2.4万亿。当然，很快这则消息就被辟谣，市场又迅速下跌。

从反弹到下跌，这个堪称经典案例的事件背后，当然是媒体操作错误带来的后果（CNBC是最大责任人），但也凸显了许多投资者对任何好信息的渴求。

几天之后，川普真的宣布了90天的暂缓期，市场也应声大涨。不过，这场关税风暴并不会就此结束，目前的市场依然是风声鹤唳，人们也依然处在既渴求信息、又害怕新消息的状态。
