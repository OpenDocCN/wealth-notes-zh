
# 尼日尔抢劫中资企业，是怎么回事？

## 嘉宾：梦回安西

近期，尼日尔军政府驱逐了三名中资企业高管，并吊销一家中资经营的酒店，还有传言称，尼日尔要求中资企业补交800亿美元的税款。

消息一出，舆论哗然，到底是怎么回事呢？

话说2023年7月，尼日尔政变后，军政府建立，先后驱逐了法国和美国的势力，可想而知，尼日尔遭到了西方的严厉制裁。

不仅西方，周边的“西非共同体”国家，也就是尼日利亚等西方有较大影响力的国家，也对尼日尔采取了一定程度的限制措施。

这就导致了一个问题--尼日尔的经济陷入困境，财政艰难。

2024年，尼日尔为缓解燃眉之急，曾向中石油借了4亿美元周转（以预付石油款的形式），规定一年后偿还，贷款利率7%。

现在期限到了，显然尼日尔没钱还，便想方设法搞钱。
