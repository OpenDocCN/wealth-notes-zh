
# 越南大改革的背后有什么考量

## 文/卢克文工作室嘉宾小小罗

特朗普在美国的效率改革工作如火如荼进行的同时，隔壁的越南也悄无声息的开展了一场大改革，力度之大堪称自1986年革新开放之后最大手笔。

这场由苏林主导的大改革具体有哪些内容？改革背后的考量又有哪些呢？

## 改革内容

2月3日，越共成立95周年纪念日上，越共中央正式公布了党政机构改革方案。

具体包括：
- 撤销原有的7个党委机构，新设立中央党机关党委、政府党委、国会党委、祖国阵线和中央团体党委。
- 对越共中央职能部门进行整合，中央宣传部与民运部合并为中央宣传民运部，中央经济部更名为中央政策战略部，撤销对外联络部。

机构改革了，人员更替随之而来：
- 原对外联络部长黎怀忠调往越共中央办公厅坐镇主任，协助苏林处理外交工作。
- 原办公厅主任阮维玉出任越共中央纪委主任，晋升为政治局委员。
- 越共中央战略政策部部长陈流光出任中央经济部长，中央经济部更名为中央战略政策部。

越南政府层面的改革力度更大：
- 原建设部和交通运输部合并成立新的建设部。
- 原农业与农村发展部和资源环境部合并成立新的农业环境部，并从劳动荣军社会部接管扶贫管理职能。
- 原内务部和劳动荣军社会部合并成立新的内务部，同步接管劳动、工资、就业、优抚对象、劳动安全与卫生、社会保险等职能。

精简之后的越南政府组织机构减少了5个部委和4个直属机构，从30个精简为21个。

各部委还精简了61个司、264个局等内部机关，公共事业单位削减15%-20%，人员规模缩减约20%，预计有15万名公务员要下岗。

国会领域，裁撤合并三分之一的内设委员会，例如司法委员会和法律委员会、经济委员会和财政预算委员会完成合并，并增加国会的监督职能，将人民诉求委员会升级为监督与人民诉求委员会等。

公安系统基层领域撤销县一级公安，垂直管理体系由四级调整为部-省-乡三级，694个县级公安单位及旗下6000支队伍关停解散。

法院、检察院、军队系统等其他强力机关相应也要撤销县级组织机构，不过时间推迟至二三季度实行。

教育领域重组两大科学院和两所国家大学，并允许外资在越独立办学，推行国际通行的学分制。

媒体领域解散越南通讯社、越南之声、越南通信公司三家下属的电视台，全部整合至越南国家电视台一家当中，统一喉舌。

地方领域，海防市、义安省、河静省、岘港市等都已经提出省至县级多个部门和机构合并的方案，同时越共层面研究合并部分省级行政单位的方案。

在经济机制与产权制度方面，苏林的手笔也是足够大。
- 成立仅6年的越南国资委被裁撤，解散后所管理的19家国企其中18家划归财政部一家归公安部，借鉴新加坡淡马锡模式重构政企关系。
- 针对房地产市场的发展，则出台了新的《土地法》《房产管理法》等一套组合拳。
- 同时将在年内推出多部其他领域新法规，包括《首都法》《证券法》《增值税法》《数据法》《公共投资法》《电力法》等等。

## 改革考量

光看内容就知道，改革力度确实很大，为啥这次越南要折腾这么大动静呢？

首先是要全面对接《全面与进步跨太平洋伙伴关系协定》(CPTPP)的高标准要求。
作为第一个加入CPTPP的社会主义国家，越南十分清楚，要么突破自身的体制束缚、全面对标国际规则，要么永远停留在全球产业链分工的底端。

比如这次首次允许成立独立工会，完善劳动法保护工人权益，以迎合CPTPP对劳工权益保障的要求。
整合农业部与环境部，大力整治环境体系迎合CPTPP对绿色经济的要求，整合建设部与交通部，推进以交通为主的基础设施建设方便更好的对外开放。
新推出的《增值税法》中，越南政府提高征税门槛，减轻小企业的财务负担，也取消了部分低价值进口商品的增值税豁免，确保当地企业的利益同时又保留了2%的减税政策，促进国内消费。
《公共投资法》则提升了各类项目上的国家资本投入上限，解决各类关键项目资金不足的问题。

其次是针对政府机构臃肿和效率低下的问题。
按苏林自己的话来说，越南财政预算70%的资金用来发工资、维持政府运转和还债，用于经济发展的资金被严重挤压，已经到了难以为继的局面。
公务员工资占多了，其他的公共事业支出就会受到影响，所以这次就是要平衡财政开支，把钱省出来，同时解决人不行的问题。

比如解散运转不理想的国资委。
当初越南成立国资委，就是参照中国的做法，力图实现政企分开，提升经营效果，从军方产业的手里夺回经济主导权。
结果却因为政府内部不团结，互相推诿扯皮，导致管理效率低下，很多行政审批还需要原来的单位来办，各部委利益不一致，都有自己的小心思，政策人才资金都很难像以前一样那么支持了。

比如越南电力集团，越南石油集团两大能源巨头关于电力和石油价格定价机制的审批权仍然在原来工贸部手里，国资委协调不动，两部委扯皮也是越南近些年来缺电的原因之一。
还有越南铁路集团，从交通部转到国资委之后，预算下拨的流程变了导致很长时间资金不到位，万名铁路职工一时连工资都发不出来。

外界看来裁撤国资委表面上好像是在进步，实际上则是无奈的回归，回归到既当裁判员又当运动员的地步，这和市场经济的原则是相违背的。

## 政治考量

而苏林在自己上台刚半年多就迫不及待开展这场改革，还有很现实的政治考量。
当前摆在他面前的有三个问题：如何巩固自身权利，处理与军方的关系，南北势力平衡的问题。

透过此次越共中央的改革，苏林的核心班底已露出水面，除了开头提到的三位，其他提拔的包括阮仲义继任中央宣传民运部部长，还有组织部长黎明兴（苏林公安系统老领导黎明香次子），以及中央内政部长潘庭镯（同样公安出身）等。
这六位心腹将是协助苏林掌权的关键所在。

还有苏林基本盘公安部的职能则得到了扩充，收获了原司法部、交通部、信息通信部转移的网络安全管理、戒毒、驾照考核等权利之外，国资委解散后旗下的越南移动通信公司也转移到了公安部名下管理。
充实公安部自己的钱袋子。

同时，借着精简的名义，苏林还顺势调整梁强办公室人员和范明政的政府系统，削弱梁强和范明政的影响力，在四驾马车中突出自己的地位。

在处理与军方关系的问题，苏林这次没有触碰军方的核心利益。
军队经商的顽疾不是一朝一夕能解决的，况且这轮改革力度之大，速度之快前所未有，必然遭到官僚集团内部巨大的阻力，手握刀把子的苏林只有暂时向军队妥协才能换得支持，整合两大强力部门的力量以推进这轮改革见效落地。

但是妥协的同时还得要想办法压制军方势力继续做大，仅依靠公安系统的力量难以做到，所以这次趁着国资委解散，苏林将18家国企全部交给新组建的财政部，目的就是为了提升文官系统的经济实力，抗衡军方。

至于南北两派势力平衡的问题，苏林是两头同时出手。
前文提到的《首都法》，将给予首都河内许多经济发展上的特殊政策，进一步提升河内的经济地位。
由原财政部和计划投资部合并组建的新财政部，部长人选也由北方派出身的河内人阮文胜担任（原交通部长），这些都是苏林在提升北方派的经济话语权，平衡两方的经济差距。

与此同时，苏林还趁着撤销建设部的名义，将原来的部长阮青毅（南方派大佬阮晋勇长子）调回到胡志明市任常务副书记，看似丢了部长职务的他，其实是苏林对南方派的安抚，在下一届党代会时可能接替阮文年成为胡市一把手，为进入政治局做铺垫。

苏林今年已经68岁了，远超规定的退休年龄，但他是个很有野心的人，不会像前前任黎可漂那样意难平，屁股还没坐热就铩羽而归。
而要想在明年的十四大上作为“特殊情况”继续获得连任提名，苏林就必须有拿得出手的成绩取信于民，树立威信，平衡好各派势力，这也是他现在铁腕推行改革的政治考量。

## 改革对比

越南的历次改革举措一直是摸着我们过河来完成的，这轮改革举措同我们历届的机构改革相比有相同和不同的地方。

相同的地方都在实现政府机构数量与规模精简，职能整合和人员缩编，减少运行成本。

不同的地方在于我们的改革采取的是渐进式的策略，从80年代到现在，每次改革都是针对当时政府面临的突出问题所采取的，有的力度大，有的力度小。
而越南这次涉及全社会几乎所有领域的改革走的是很激进的路线，典型的不破不立策略，有可能造成局势短期的动荡。

这场涉及政治权力斗争和经济效率提升的内部博弈，既关乎越南未来国运的天花板，也可能改变东南亚地区的地缘经济格局。
但不论最终是否成功，这起码展现出越南在面对困局时敢于寻求自我突破的勇气。
这不仅是越南的挑战，也是所有社会主义国家转型的终极命题。
