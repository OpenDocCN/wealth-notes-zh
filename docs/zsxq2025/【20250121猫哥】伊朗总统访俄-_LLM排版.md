
# 伊朗总统访俄

1月17日，伊朗总统佩泽希奇扬访问了莫斯科，双方签署了《全面战略伙伴关系条约》。在这个敏感时刻，两国加强抱团，关系升级，相当值得关注。在此之前，伊朗和俄罗斯的关系以2001年签订的《相互关系基础及合作原则》为基础，合作并不怎么深入，是一个很普通的关系。但俄乌战争后，形势发生变化，伊朗向俄罗斯提供了无人机等一系列武器，双方的关系出现了升级的需求，于是本次的《条约》应运而生。

和原有的关系相比，《条约》最大的突破，是出现了军事合作的内容，包括加强人员互访、增加联合军演数量、以及不与第三方合作，危害对方安全利益等等。这算不上同盟条约，却是一次质的突破，尤其是“不与第三方合作危害，对方安全利益”，意思就是伊朗承诺不和乌克兰合作，损害俄罗斯的利益。如果伊朗和某个国家处于战争状态，俄罗斯也承诺不和该国家合作，危害伊朗的利益。这是本次《条约》最大的亮点，已经属于安全合作的范畴。两个国家的关系密切与否，关键指标之一，就是有没有安全范畴上的合作。如果没有安全合作，即使经贸往来多，大概率也是塑料兄弟情，谈不上紧密；如果有安全合作，哪怕经贸往来不多，也可能关系紧密。

除了安全合作，《条约》还有两个大的合作内容，一是俄罗斯向伊朗出口天然气，按理说，伊朗是不缺天然气的，不需要从俄罗斯进口。但实际上，则是另一回事。俄罗斯在被极限制裁的情况下，能源出口受阻，急需寻找新的出口方式--能源出口国。比如沙特，就发现了大量进口俄罗斯石油，再转卖给欧洲。沙特的出发点很简单，不赚白不赚，赚了不白赚。伊朗同样是能源出口国，所以俄罗斯的能源先输送到伊朗再出口，也就是玩一手洗澡蟹，完全可以。对伊朗来说，多赚点钱不是坏事。对俄罗斯来说，多一个阳澄湖何乐而不为。

当然有人会感到疑惑，伊朗不也被重重制裁了吗?其实伊朗主要是被美国制裁，欧盟虽然也有制裁，但属于不情不愿的态度，程度并不高，伊朗2024年对欧盟的石油出口还有45亿美元左右。最近这些年，都稳定在一年50亿美元左右的水准，俄罗斯可以通过伊朗这个阳澄湖，向欧盟增加一些油气出口，尽管金额可能不会太多，但对战争状态下的俄罗斯来说，蚊子腿也是肉。

第二个合作内容，“国际南北运输走廊”。这是一条印度提出的经贸路线，货物从印度西部出发，先到达伊朗，再途径阿塞拜疆，最终到达俄罗斯，把印度和俄罗斯连接起来。这条线路的关键是伊朗。此前，伊朗一直对“国际南北运输走廊”犹豫犹豫，主要是害怕美国的态度，毕竟印度加强与俄罗斯的联系，不符合美国利益，尤其是在俄乌战争发生后。直到佩泽希奇扬访俄，《条约》签订，伊朗明确支持，双方才算是正式敲定了。

总的来说，俄伊双方的关系前进了一大步，从曾经的只有经贸关系，迈进了安全合作的范畴，并且经贸的联系出现了深化的趋势。因此，说《条约》加强了俄伊两国的抱团程度，是没毛病的。

对俄罗斯来说，加强与伊朗的抱团，是应对俄乌战争的重要一环，毕竟俄乌局势和中东局势是联动的。如果懂王真的能够24小时结束俄乌战争，俄罗斯可能未必有意愿和伊朗抱团，因为这样容易让懂王误会，但经过这段时间的试探，懂王无力快速结束战争，已是现实。

比如一开始，懂王的说法是24小时结束，结果到了近期，就变成了100天结束，背后的原因，就是俄乌战争牵涉了太多利益集团的利益。军工、能源、建制派等等，无论哪个都不是好惹的。在这种情况下，能够确定的是，就算懂王要推动俄乌停火，能接受的底线也是“按实际控制线停火”，不可能让俄罗斯占太多的便宜。这对俄罗斯来说，就不太划算了。毕竟付出了巨大的伤亡，战场上也是俄军占优，如果“按实际控制线停火”，等于连顿巴斯都没有拿下，远远达不到预期的价位。在短期内难以停战的情况下，加强与伊朗的抱团，让中东局势和俄乌联动，就是俄罗斯的刚需了。

而对伊朗来说，抱团的需求更大。懂王对伊朗的态度有多恶劣，是明摆着的，上一个任期不仅撕毁了《伊核协议》，还击杀了苏莱曼尼。伊朗倒是想认怂，无奈美国不收。很多人不理解，懂王为啥不同意伊朗认怂，其实很简单，美国的中东战略，必须制造分裂。美国作为域外国家，想要介入中东得有个抓手，就是制造地区紧张局势，哪些因素，最容易导致地区变得紧张呢？当然是什叶派和逊尼派的冲突。海湾国家作为逊尼派代表，战斗力不足，他们和伊朗关系紧张，就会邀请美国介入，这是美国立足中东的基础。即使是奥巴马当年对伊朗缓和，也只是出于集中力量对付中国的考量，属于权宜之计，并不是真的想放过伊朗，所以伊朗就没有跪的基础。

懂王新任命的中东问题特使威特科夫，属于狂热的以色列支持者，对伊朗可不怎么友善。特使这种职位，通常反映的就是总统本人的态度。下一个四年里，懂王有可能以“伊朗威胁”来逼迫沙特，重新推进沙特和以色列的建交。虽然经过“阿克萨洪水”，小萨勒曼建交的胆子小了，但不代表完全消失。在沙特内部，虽然反犹依然是主流，但亲犹人也不是一个两个。不少沙特人是亲犹太的。前段时间德国发生恐怖袭击，凶手就是一个亲犹太的沙特润人，离奇的背后，是沙特在美国体系下的时间久了。内部亲西方的相关利益者，数量在不断增加。不少沙特人，逐渐变得和高华一样，把和西方的交往，视作融入所谓的文明世界，越是沙特上层越是如此，这是懂王的机会所在。而这，对伊朗极其不利。

不管伊朗愿意与否，加强与中俄的交往都是必须的。未来四年的中东，可能会走向更加动荡的局面，懂王势必加大对伊朗的施压力度。无论懂王还是内塔，都有心思在伊朗推动颜革。一边加大制裁，一边资助颜革，会让伊朗压力山大，上一个任期懂王正是这么干的。在空前的压力下，伊朗的内部矛盾，不排除加速爆发。总之未来四年，伊朗将面临严峻的局面，和社会面临动荡的风险，我们必须密切关注。
