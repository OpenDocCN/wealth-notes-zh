
# 经济又到重大拐点处，人民币汇率会不会再次贬值？

进入 5 月，随着中美关税休战 90 天，我们的微观经济体感加速“寒气满满”。就在此时，川普急不可耐，又提出要跟我们进行第二回合的贸易谈判。秉持胡萝卜加大棒的策略，川普又断供了我们的飞机发动机、核电站设备，还在南海地区反复挑衅。另一头，美国国内日益紊乱，川普还想强推《大美丽法案》，使得美债收益率居高不下的同时，美联储还不愿意降息辅助。在欧洲战场上，俄罗斯现在完全按照团灭乌克兰的新战略去打，未来会不会引发欧美直接下场，为未可知。外有灰犀牛和黑天鹅，内有刺激剂量不足导致的偏通缩环境，一场看得到的承压即将向我们袭来。如果处置不当，还会引发人民币汇率再次走向贬值，从而让这股寒气持续飙升。

经济会走向哪里？人民币汇率会不会再启贬值通道？站在重大十字路口，我们需要知道下一站通向何方。仿佛就在一瞬间，这个草台班子组成的世界，就完全混乱起来。今天的文章很干很烧脑，大家慢慢品。

美国国内的老钱们，充分利用川普跟马斯克不和的契机，从非法移民、减税法案、美联储货币政策等多个维度向川普发起进攻。按照目前的态势，只要美联储不出手，美债收益率短期很难被压制下来。换句话说，美债规模后续还要因为融资成本太高而持续失控，美国的内乱还会强化大家对美元不再那么信任。问题是，美元指数中最大的块头是欧元。在比烂的时代，真的把美国人逼急了，他们可以选择痛扁欧元，来确保全球资金还是更信赖一点美元霸权。所以，此时此刻，乌克兰去对付俄罗斯的战略轰炸机，看似很勇，也确实迎来了俄罗斯的报复，对于欧元区经济后续走势，是一次灾难。被逼入绝境的普京，绝对是往死里干，否则他在国内就要被自己人干。

那么，欧洲到底下场吗？如果欧洲下场，美元这把就又稳了。其实，我们回顾过去 40 年的全球经济史，就会明白，这一切都是被设计好的。80～90 年代，亚洲如火如荼，美国前后收割了日本、东南亚、韩国等重要经济体，也造就了克林顿时期的辉煌。进入本世纪初，欧元强势的很，美国前后用了接近 20 年的时间，持续打击欧元。甚至，充分利用次贷危机的机会，转嫁压力给欧洲，形成欧债危机。即便是最近的俄乌冲突，也是这一手法。这就是为何美国可以快速从次贷危机中走出阴霾。到 2018 年开始，美国发现欧洲也榨不出太多的钱，亚洲大部分经济体已经经过了 N 遍。兜兜转转，就只有中国还没有被捧上餐桌。这才是中美连续这么多年进行大国博弈的真正原因！但是，从最近的这场关税战来看，美国暂时还真的割不到我们。尤其是印巴空战后，美国多少感触到我们的军工底气，也就更加不敢强行收割了。所以，接下去，美国强化对欧日韩印的收割，就是即将发生的事情了。

于美国而言，多少可以补上收割不到我们的“缺口”。于是，问题就来了，这么玩下去，短期要马上启动“弱美元周期”的策略，恐怕还要再等一等。也就是说，人民币兑美元，也就是面临着不小的压力。可是，我们的国内经济所需的增量资金，就两个口子。一个是央妈主动放水；一个是人民币汇率升值带来的。第一个口子，现在被美联储不肯降息而死死的掣肘着，我们的央妈还不敢动。第二个口子，又恰好被高企的美债收益率也压制，我们自己国内的资金也觉得当前的美债收益率很香啊。这就构成了当前国内内需不足的核心逻辑，却又是被美国人牵着鼻子走的。

如果是美国单方面战略设计出来的，或许我们还有办法对付。很明显，这个局，有太多环节，美国也是赶鸭子上架，也是被形势所逼，就像高企的美债收益率。这里，我们需要特别提到美债收益率。这个就是美国建制派老钱们组团搞川普的核心手法之一，威力比废掉马斯克还大。毕竟，哪任总统任内出现美债危机，那就是他的问题了。老钱们，也是用极限施压的办法，在威逼利诱川普就范。而我们这些外围国家，实则是被美国内政的博弈所裹挟了。其直接的结果是，今年下半年，尤其是这个三季度，我们国内经济增速的压力，还会陡增。预计，7 月份村里要出一些维稳举措，但是，也只是维稳”。

逻辑也很简单，就是上面的两个释放增量资金的口子，还被人家捏住。其实，俄乌冲突如果再升级，后面的不确定性是很吓人的。比如，即将失败的乌克兰，有可能引爆类似脏弹，或者去袭击核电站之类。只要这个事情一出，全球金融市场必然会有一波血洗，不是开玩笑的事情。相信村里也是对此有预期的，更不敢马上把救市的子弹全部撒出去了。当然，所有这一切的直接结果是，局势变成了死亡螺旋。即，要么大家一起放水刺激，要么就相互掣肘都拖死。请注意，这个逻辑，才是今年下半年最核心的地缘、金融、宏观政策等方方面面的底层逻辑。

大家应该已经看得出，所有下场的大佬们，竟然都处于险境，都只能孤注一掷，都没有太多的腾挪空间。这是我们理解当前地缘政治经济和金融市场的时代底色。也因此，注定会有更多大佬会选择“不按常理出牌”，会主动打破默契、约定、规则。本次中美谈判，科技领域肯定也是绕不过去的。明眼人都看得出，中国在供给侧的成就这么大，人民币汇率是被人为低估的。但是，这也使得美国目前暂时只能这么做。因为，当中国科技追赶的程度已经威胁到美国科技企业的全球利益，美元霸权就已经出问题了，美元潮涨潮汐也出问题了。也就是说，美国无法验证自己是全球具有碾压性的科技超级强国，那么，它就注定只能依靠金融手段保持相对强势的美元和美债收益率。

以前大家是“为信仰充值”，现在大家为短期的收益充值。但是，美元霸权体系的信仰已经不再那么坚定，无法构成真正意义上的闭环。那么，还有什么招数呢？或许，现在只有一场全球经济衰退，才可以让美元潮涨潮汐功能恢复。既可以让川普跟鲍威尔达成共识，也可以让中美达成共识。这场旱灾来了，穷人家先饿个半死，富人家还有点余粮，所以，富人家的话语权依旧有效。不过，请注意，这种微操也有很大的 bug。美国的大资本们，现在在全球没有很廉价的资产可以收，使得美元无法从海外获得利益，美元无法从美国本土流向其它国家，弱美元如何开启？无法开启，那全球经济势力版图的垄断租金如何创收呢？

这也提示我们，当前人民币汇率看似有压力，但是，中长期依旧是被严重低估的。毕竟，美元体系现在的强势，不可持续，这才是根本原因。本质上，这一切都是美元体系内生的困境，随着时间的推移会慢慢浮出水面。就像美国的芯片战略困境：如果现在出口芯片，美国可以赚更多贸易顺差；可是，如果出口芯片，中国就有了科技战略崛起契机，美国霸权长期就没有了。进一步而言，这个困境意味着，现在美国对付贸易逆差的困境和高债务困境，是“两难全”。

所以，我们要做的，就是继续等待，哪怕当中微观经济体感还是很不济，哪怕人民币汇率还有一次承压的风险。只有等美国人自己玩不下去了，后面才轮到我们出手。顺序一错，方向全错。必须看到，美国是一个平台、一个系统，并不是一个传统意义上的国家。我们跟他直接斗，那近乎等于跟全球唯一的这个平台和系统去斗，我们吃亏的。最佳的策略，只能是让这个系统和平台自己出问题，甚至很多其它国家也不愿意积极拥抱它，我们的生机也就来了。这就是当前村里的核心战略。马上，出口的形势又要严峻了。这也意味着，我们赚取美元流入的能力也会受到影响，自然人民币汇率后续的承压不小。

当前，央妈是优先保汇率，甚至是阶段性的放弃全面拯救地产（只托不举）。不过，有一个关键性的中间地带，就是在香港金融市场上，村里是要发力的。因为，这里如果可以构筑起比较强大和繁荣的金融护城河，则内地人民币资产的压力会相对再小一点。第二道护城河，就是积极引导大A慢牛，这个很关键。在微观经济体感日益承压的背景下，大A是目前最容易启动的经济少数亮点领域之一。而且，只要大A有一定的赚钱效应，则资金外流的压力也会小。通过这两道护城河，人民币贬值的压力就会变得相对可控，而不会陷入无序。

其实，村里最终极的战略预判是：如果不能收割中国成功，那么，放弃通胀管控，将是美国人最终的结局。当初美国人同意我们加入 WTO，想要的是发展我们成为“依附性经济体”。可是，事与愿违，这一切在 2018 年就被证伪了。而经过前面这么多年的相互博弈，原本美国依赖隐性殖民与暗线渗透的策略对付我们，也玩不下去了。所以，现在只能通过蛮力搞乱我们经济的外部环境，再做尝试。然而，诚如我们上面提到的，这原本就是一场“伤敌一千，自损八百”的无奈之举。因此，只要我们能够熬到美国最终放弃通胀目标管理，那么，后续我们这里的困境也就会瞬间消除。

如果说有什么可能再次打乱上述节奏的，A森并不担心我们周边的地缘冲突，也不担心关税战，更不担心人民币汇率的阶段性压力。我真正担心的是，欧洲如果下场俄乌冲突了，美联储的态度就会再次暧昧，因为他们又找到了多收割一点欧洲的好机会。那么，他们用高息保持美元资产的吸引力的套路，就要再多用一段时间了。世事如棋局，日日新。只要美国内部乱哄哄，欧洲也不敢下场俄乌冲突，那么，就是对我们经济重大的外部利好消息。比烂的时代，在国际丛林社会中，别人家倒霉，那就是你的机遇。这个逻辑，永远不会过时。

考虑到外部形势如此错综复杂，接下来几个月内，国内经济微观体感，许还有一次往下俯冲，能够将这个阶段熬过去的，未来市场会有不菲的回馈。加油～
