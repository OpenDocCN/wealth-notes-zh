
# 石破茂访美

当地时间2月7日，日本新首相石破茂正式开启首次访美行程，并和懂王进行了会谈。对于访美，石破茂是期待甚高的，从去年11月5日也就是懂王获胜起，石破茂就在沟通访美了，只是懂王一直没答应，以致于拖到了现在。

问题来了，本次访问，到底有什么成果呢?从双方发表的《联合声明》来看，内容主要有：

1、两国共同研发AI和半导体
2、日本将国防预算占GDP比例提高的2%
3、坚持半岛无核化
4、《日美安保条约》适用于钓鱼岛和琉球
5、反对所谓的单方面改变海峡现状
6、在“美日印澳”合作机制下，推进自由太平洋政策

六条里，只有两条具有实质性意义，分别是第一条“日本提高国防预算占比”，和第四条“《日美安保条约》适用于钓鱼岛和琉球”。

第四条其实不是懂王的首创，奥巴马时期就已经存在，懂王第一个任期也是这么说的，基于强烈的孤立主义倾向，日本国内在懂王二次获胜后，普遍有担忧。担忧他真就不管日本了，那样的话，日本在东北亚就是彻底的孤家寡人。现在通过第四条，强化《日美安保条约》，无异于给日本吃一颗定心丸，作为交换，日本得提高国防预算开支，也就是购买更多美国武器。

剩下的四条，都是虚的。只能说，成果着实不多，距离石破茂和日本国内的期望，有很大距离，尤其是最受关注的“美国钢铁收购案”一事，懂王没有松口的意思。

“美国钢铁收购案”，就是2023年末，日本制铁公司试图收购美国钢铁公司（简称：美钢），美钢是美国最大的钢铁企业，年产量1600万吨。如果吃下美钢，日铁的总产能将达到8600万吨，成为世界第三大钢铁公司，仅次于宝武和安赛，而规模在钢铁行业里是十分重要的。为体现诚意，日铁给出了149亿美元的收购价格，溢价高达30%，美钢的股东们个个乐开了花，最后董事会投票，以98%的高票通过，同意收购。然而，美国政府却无法接受。

一来，美钢的前身就是大名鼎鼎的卡内基钢铁，熟知美国历史的都知道，卡内基钢铁是在二次工业革命中崛起的，是美国崛起的缩影。对美国人来说，美钢有着重要的象征意义，落到日本人手里实在难以接受。二来，美钢的员工和工会也普遍反对，毕竟换一个老板，待遇等各方面指不定有多大变动，万一新老板要对公司降本增效呢?美钢的大本营恰好在宾州，几万张美钢员工的选票，对大选有着决定性意义，哪个候选人都不敢轻易得罪。于是乎，一桩商业收购案，和政治紧密关联起来。拜登始终不敢做决定，直到去年12月，在犹豫了很长时间后，拜登以“国家安全”为由，否决了收购案。令日本方面大失所望。

当然，这不意味着收购没希望了，毕竟换总统了，万一懂王同意呢?对日本来说，收购美钢极其重要，它不是一桩普通的商业收购那么简单，更是个风向标，关乎着一个关键问题：美国是否愿意为日本开绿灯。日本这些年，许多产业逐渐凋零，哪怕是曾经强势的汽车产业，都受到了严重冲击，为了减少冲击，日本越来越多的企业，选择转向美国市场。

以汽车为例，日系车这些年在华销量，是不断下滑的。但在美国，日系的销量依旧稳定，不说丰田这些巨头，即使像斯巴鲁这种，在其他市场早已消失的品牌，在美国依旧活得不错，一年能卖出60多万辆。这种情况下，日本就很需要美国“开绿灯”了。

简单来说，你可以对其他国家搞贸易保护主义，但别对我搞就行，否则，如果日本品牌在别的市场下滑，在美国还要挨铁拳，混合双打之下，说不定要完蛋。毕竟日本本土的市场规模不够，在目前的世界大形势下，难以自立，必须和某一个大市场绑定。只有这样，才能抵御即将到来的寒冬。

而日本制铁对美钢的收购案，就被视为美国是否愿意“开绿灯”的风向标。因为钢铁行业，是许多行业的上游，尤其是汽车，如果美国不反对日本掌控上游，则“开绿灯”的概率很大，反之就是不愿意“开绿灯”。

所以，日本上下对收购案极为关心。石破茂访美的最核心诉求，也是希望懂王转变态度，只不过懂王的表态是“日本制铁公司已同意将进行巨额投资而非收购”。

既然不开绿灯，那么后续四年，懂王在贸易逆差、汽车等问题上对日本重拳出击，就是板上钉钉。日本受到的冲击将进一步加大，未来严重不乐观。前两天，日本央行的前行长黑田东彦表示，“失去的几十年已经结束，经济已经复苏”。事实证明，他高兴得太早了，苦日子才开始呢。

从去年12月以来，石破茂多次放风，表示想在访美前，先访华。说明他懂王的态度有一定的准备，一旦懂王的态度不利，就转向中国。现在懂王不松口，能够预见的是，改善对华关系将是石破茂的外交重点。而我们对此，也是欢迎的。

不过，大家不用有太高的期待，石破茂属于弱势首相，对华能够释放的善意，势必十分有限，比如，石破茂曾希望和中国互相免签。但他上台后，却发现难以推动，最后只是略微放宽了中国的游客签证，把团队旅行签证的可停留天数，从15天延长至30天。还设立了一个10年旅游签，也就是在10年内，可以多次往返日本，不用每一次都提交申请。只不过，这种签证的申请和审批都很麻烦，一般人拿不到，也用不上。

要知道，中国早就对日本大幅免签了，而日本政坛却连对等都不愿意，别的方面更无法期待太多，背后是中日在历史问题、产业结构、地缘冲突上的一系列矛盾。

在一系列矛盾前，中日关系的改善空间是有限的。
