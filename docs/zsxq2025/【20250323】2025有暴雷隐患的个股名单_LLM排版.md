
# 2025有暴雷隐患的个股名单

投资者需谨慎评估持股风险，并关注行业动态和政策变化，以做出更为理性的投资决策。A股投资爆雷是指投资者所投资的A股上市公司出现重大负面事件，导致股价大幅下跌，投资者遭受重大损失的情况，以下是一些常见的爆雷原因：

## 信息披露违法违规
上市公司未按规定真实、准确、完整地披露公司重要信息，如财务数据造假、隐瞒重大事项等，例如*ST龙宇就因2023年被会计师事务所出具“无法表示意见”的审计报告，而被证监会立案调查。

## 违规减持与内幕交易
公司的大股东、高管等在限制期内违规转让股票，或利用内幕信息进行交易获取不正当利益。例如新易盛的董事长高光荣因涉嫌违反限制性规定转让股票被立案调查，其2022年的减持量较计划超出近17%。

## 经营不善与业绩变脸
公司的经营状况不佳，业绩出现大幅下滑甚至亏损，导致股价下跌。例如部分ST公司，像ST富迪三季度的扣非净利润亏损1.61亿元，拖着整个公司在泥潭里挣扎。

## 行业风险与政策变化
所处行业面临市场饱和、技术变革、政策调整等不利因素，影响公司的盈利能力和发展前景。例如房地产行业受持续的调控政策影响，相关上市公司股价普遍受挫。

## 公司治理问题
公司内部管理混乱，存在大股东侵占上市公司资金、内部利益输送等问题，损害了中小股东的利益。例如祥源文旅的实控人及其关联方存在非经营性资金占用，公司被责令改正并罚款。
