
# 农村婚恋市场观察

说说，我看到的农村婚恋市场。我的一位大哥，70年生人，儿子90末生，无固定职业。在他的撮合下，他的儿子和他们镇一位01年的女生开始了交往。前几天，女方第一次到男方家，他请我们去做客，一起热闹。然后，我亲眼看到了，和他认识三十多年的时间里——最热情、积极、主动、活泼、健谈、嘴角上扬的他。

原因在于，大家观察到的绝大多数农村生活为主的男性样本，过了30岁，还能结婚的概率，会急剧下降。与之形成鲜明对比的，是乡村女性婚恋市场的巨大优势。

邻居给我讲了一个极端案例：我们村我一位小学学长，数年来，拥有了第一次去相亲的机会——对方是北方某省三个孩子的妈妈。三个孩子不是最大的困扰，最大的困扰是——里面有两个男娃。

在农村婚恋市场上，已婚女性的谈判筹码优势上：未婚>带一个女娃>带两个女娃>带一个男娃...

我那位学长拿到的机会，是媒人们在综合衡量了他的条件后，给出的资源匹配。对了，他农村自建两层楼房一间，十万元车辆一辆，存款(据他的母亲讲)可以覆盖一套县城的房产。一个弟弟小他两岁，我学弟，也是未婚。身高175,偏胖，极为勤劳那种。昨天路过我的一个弟弟家门口时，我老婆和我聊起了他：

91年出生，168,独生子，市里住房一套，十万元车辆一辆，城里务工为主

而这么多年，几乎，是几乎没有获得过什么相亲的资源。和他类似条件的另一位弟弟的母亲，在听闻邻居这边，有远方亲戚的适龄女性资源后，带着礼物连夜拜访。

插播下，农村(居住为主的)男性婚恋市场的排序是：
独生子女≈有姐妹出嫁的多姐妹家庭>两兄弟家庭>离婚带娃男性。

离异带娃男性的婚恋市场，若经济条件普通，基本上是永久关闭了。

所以，男性普遍在离婚这件事上，更加谨慎。可能不是因为爱，是市场行情太差了。我不完全统计，老家方圆三百米内，有四位媳妇走掉后再婚成功，而男性从此孑然的例子。

是不是一个人生活、一个人养娃，就一定比普通的家庭组合生活，更差。这个话题，我们讨论过：关键的问题在，家庭是多数普通农村家庭最小的社交端元。一旦家庭破裂，他就非常容易失去社交的资源，精神上出现问题的概率会大幅度攀升。

大家可能会观察到：四川、湖北、重庆、湖南，繁育出了非常有特点的男女相处模式——大量的男性主内，女性当家做主。我从小就亲历着：非常明显的男性花围裙群里之社会。

大年初二，一个高中同学回来，约我去他家喝茶。我去见到的场景，是他的父亲在手洗衣服。家里的社交，如来客人了的礼节事务，全程是他母亲在做。

类似的场景，可能在四川、湖北、湖南、重庆，比例非常高。而这个情况，目前应该没有任何缓解趋势。

刚才聊到，农村离婚男性市场行情之差。再举一个身边的例子，我邻居闺蜜的儿子：2020年离开本地后，一次没有回家后。男方是我学弟(小我一届),女方是我小学同学。

男方家里，为了维持这个名存实亡的家庭，每月约支出3000元，给女方。女方住在县城，带娃。男方按月打房贷给女方。男方母亲去年提议：给女方6～7万块钱，买辆车，方便从县城回老家。也就是说：男方在维持这段婚姻的过程中，老一辈的年均成本是4～5万块。

邻居这位闺蜜，最主要的劳动收入来源是：和邻居一起，给去年讲过的电商老板做蔬菜采摘工作。时薪是12-13元。另外，男方父亲经营一家小杂货铺。

我邻居闺蜜讲，手上有存款，现在为什么每天去工作，长时间的工作。是因为，忙起来，可以忘掉很多烦恼，可以真正的践行——儿孙自有儿孙福~
