
# 中俄各自都拿到了最想要的成果

## 中俄在两场活动中的核心成果

部分星友对9.3阅兵中出现的大批新质装备非常感兴趣，希望详细聊聊。这里想说的是：

其一，现在网上相关信息已经非常丰富了，就不需要再去重复赘述。我们很牛逼，从追赶到并行，再到现在全面超越，着实振奋人心。

其二，我们主要还是把角度放在时政方面，至于新装备的解读，大家不妨搜一搜网上军事博主的详细拆解。

本期内容，我们聊聊这次北京天津的两场活动，中俄各自都拿到了什么最想要的成果。

其实成果很多，比如我们再次向全球彰显了大国风采、军工能力、号召能力、组织能力等。

再比如普京4天行程，累计工作48小时，和17国展开接触，实现了自俄乌战争以来的外交重大突围等等。

但要说当前双方最想要的现实成果是什么呢？先让我们把目光聚焦到上合天津峰会。

这次上合天津峰会尘埃落幕，意料之中取得了丰硕成果，用我们王部长的介绍，峰会取得了八项成果，这里我们大致过一下，包括：

1. 制定未来10年发展战略。
2. 落地了应对安全威胁与挑战综合中心、信息安全中心、打击跨国有组织犯罪中心、禁毒中心这“四个安全中心”。
3. 做出成立上合组织开发银行的政治决定。
4. 设立能源、绿色产业、数字经济等6个务实合作平台。

之所以这次峰会成绩斐然，个人思考的核心原因，按我们传统说法其实就是天时、地利、人和的聚拢：

其一是天时：川普对全球发动关税战，搞战略收缩，在中美G2时代的主旋律下，很多发展兔子家没得选，只能积极向我们靠拢。

其二是地利：这次我们是主办国，实力、影响力包括组织控场能力那都是全球独一份，这个大家应该深有体会。

其三是人和：包括俄罗斯和印度，这两个上合刺头，也因为俄乌战争和印巴冲突，必须选择和我们同频。

特别是印度这根搅屎棍，之前8月23号星球文章就预判了，在被老美暴揍一顿，一脚踢开后，只能向东看，这次天津上合峰会一定会有大成果。

事实证明莫迪还是人间清醒，虽然会后有传言，印度否决了阿塞拜疆从对话伙伴国升级为正式成员的申请，但从大面看，阿三这次态度基本端正，该签的字都签了。

回到主题，为啥说这次北京天津的2场活动，中俄各自都拿到自己想要的成果了呢？这里就涉及2个核心成果，我们一个一个来说。

首先是俄罗斯方面。

最大的成果是终于和中蒙敲定了途径蒙古国的西伯利亚力量2号天然气管道线(蒙古国境内又叫东方联盟管道)。这是普京最最想要的结果。

这条设计年输气能力高达500亿立方米的天然气管道线路(2019年投产的“西伯利亚力量1号”年输气量380亿立方米),从2014年俄罗斯方面正式提出，到现在已经足足拉扯了超过10年，完全成了一场马拉松。

最初分歧的焦点主要涉及2个，一个是蒙古国对过路费要价太高，期望每年能从中捞它几十亿美元。另一个是我们和俄罗斯的结算价格一直在博弈，俄罗斯诉求是参照欧洲价格，这个我们肯定不同意，我们要的是更低的价格。

结果一场俄乌战争打到现在，中俄双方价格谈的七上八下，蒙古国又出幺蛾子，拜登用一张大饼把蒙古国的魂勾走了，于是这条管道又增添了地缘政治方面的可靠性、安全性问题。

于是，去年底俄罗斯心一横，打算启动途径哈萨克斯坦的替代方案。这对于哈萨克斯坦而言，完全是天降一笔横财。

那么，为啥途径哈萨克斯坦的线路方案迟迟停滞不前呢？核心原因就是钱。

一来，经哈的方案比原方案输气距离足足长了2000多公里，其中还要穿越天山与塔克拉玛干沙漠，增加的建设成本起步150亿美元以上。

二来，如果按30年运营周期测算，期间运输成本比经蒙古国，估算增加450亿美元。

这还不算运营期间的维护成本，毕竟线路长了2000多公里，条件又艰苦。

三来，我们也不太感冒这个替代方案，因为不符合我们能源多元化布局的整体战略。我们和中亚国家已有线路，这不是自己和自己较劲吗？

所以，虽然普京着急，但急也没用，经哈哈萨克斯坦的方案毛算从投资、运输到维护，估计额外费用动辄就奔1000亿美元去了，这是运天然气又不是运黄金，本来就是薄利，账算不过来就只有摆起，大家再琢磨下。

转折点是川普上台。

川普是实在人， 一看世界地图，就知道蒙古国的地缘态势被中俄包圆了。但凡砸一分钱进去都是傻逼。还飞机运稀土，什么鬼!于是就把蒙古彻底凉在一边。

这才有了今年蒙古国的悬崖落马，回头是岸。

这份具有一定法律约束力的备忘录，虽然诸多细节还待后续谈判固定，但仍然标志着项目在拉扯十多年后取得了决定性进展，大家达成了原则性共识。

虽然敲定方案是一个三方共赢的局面，但同时大家也要清楚，之所以我们能积极配合快速达成一致，另一个重要因素就是来自俄罗斯的配合。这里就涉及到上合峰会取得的重大成果之一：做出成立上合组织开发银行的政治决定。

是滴，筹建上合组织银行就是我们最想拿到的结果。

聊聊成立上合组织银行这件事。成立上合组织银行是我们早在2010年就提出的构想，并且一直以来都作为上合组织的重点课题在进行攻关。但奈何前有俄罗斯挡道，后有印度搅屎，上合成员国有一票否决权，这是上合组织成立的原则和精神，轻易又不可能把这条废了，所以生生把这件事搁置十几年。

俄罗斯之所以前期阻碍上合组织银行的筹建，最重要原因就是俄罗斯在2006年就与哈萨克斯坦共同发起了欧亚开发银行，并在其中持有67%的控股权，掌握着绝对主导。

俄罗斯一直担心上合组织开发银行会削弱欧亚开发银行的影响力，动摇其在区域金融领域的地位，甚至顾虑中亚地缘政治被我们排挤。

说直白点，那时我们还没像现在这么强，那时俄罗斯也没得俄乌战争，大鹅最关心的还是自己的一亩三分地利益。

后面的事大家都清楚，俄乌战争爆发后，俄罗斯被剔除SWIFT系统，自己资金都转不动了，欧亚开放银行已经成了摆设，俄罗斯的态度才逐渐发生软化。

但软化和认同推动不是一回事儿。当然，还有印度这根搅屎棍在，莫迪又不想得罪老美，又想薅上合组织银行的羊毛，眼睛死死盯投资方向和自身的份额，所以筹建上合组织银行这件事一直这样膈应着拖起。

但拖起对我们影响最大，上合组织谁是大富翁？谁是基建狂魔？当然是我们呀，未必还有谁？

中亚甚至包括伊朗这些成员国以及其他对话国，国内基建烂的一塌糊涂，如果仅仅只靠点对点进行双边合作，推进速度慢不说，还有政治波动的带来安全风险，我们要顾及到资金进去了，建设启动了，对方反水出幺蛾子这些事，毕竟吃亏也不少。

而上合组织银行不一样，说白了这是上合组织大家的金库，你敢违约试试。

我们的一带一路、人民币国际化、产能输出，如果有上合组织银行开路，那就事半功倍。

虽然我们过去已经组织成立了金砖银行和亚投行这两条腿，但毕竟那是十年前的布局，从性质、业务范围、治理结构都不可同日而语。

比如，金砖银行和亚投行运作了10年，累计发放贷款大致也就1000亿美元上下，平均1年也就100亿美元，显然还是不多。但你又不可能贸然去改变约定的出资金额、股权结构、投票权等。

再比如，金砖银行主要服务于金砖成员国家，如果上合成员不是金砖成员，那就只有找亚投行，但亚投行有110个成员国，大家近些年都是愁眉苦脸的，要找资金，啥时候轮得到你。

所以，需要再加一条腿，给区域市场注入新的活水，顺应时代潮流变化。所以，既然西伯利亚力量2号天然气管道线，普京最关切的能源生意我们配合了，那么反过来，对我们的重大诉求俄罗斯不设卡才行，这叫礼尚往来。况且俄罗斯在这方面也有现实需求，毕竟还是大家受益。

至于印度嘛，在经历了印巴空战、老美翻脸后，如今也只能从善如流了。

当然，成立上合组织银行除了是我们的最大成果外，也是上合组织其他成员国的迫切需求。

前面说了，如果你的贷款需求金砖银行、亚投行不能及时满足或者匹配，那就只有找世界银行或者国际货币基金组织，但这两个主体背后的主子又是美国，你要贷款那就得美国点头。

但老美你懂的，多半会要求你改革政治体制、改革金融体制，对于大部分中小发展国家而言，基本没得抵抗力。要么国内经济困难加剧，矛盾激化，要么被老美随意拿捏，甚至吃干抹净。左右为难，你说怎么办。

但上合组织银行的成立，就会一定程度解决这个难题，至少能给大家多一个选择。

给你提供中长期低息贷款，给你提供全球最有实力，效率和性价比最高的基建资源，还不要求你让渡部分主权，难道不香吗？

所以，当俄罗斯和印度搞定后，成立上合组织开发银行的节奏明显提速：6月，在上合组织成员国财长和央行行长会议上，各方在推进成立上合组织开发银行方面取得了实质性进展。7月，上合组织成员国外长理事会会议，各方围绕建设上合组织开发银行等融资保障机制达成原则共识。最终，在天津上合峰会上一锤定音。

当然，对于中俄的这两件拖了10来年的大事，能够同步解决，虽然可喜可贺，但我们还是要清醒的认识到，这背后与我们自身实力不断发展壮大和国际环境剧烈变化分不开，现在也仅仅只是开篇，后面还有很长的路要走，说不定过程中还会有颠簸，但我坚信，我们现在是能够应对任何外部的波涛，走好走稳中华民族的伟大复兴这条路。
