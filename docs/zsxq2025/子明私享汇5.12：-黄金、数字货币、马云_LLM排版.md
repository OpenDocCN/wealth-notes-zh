
# 黄金、数字货币、马云

## 发布时间：2025年5月12日
## 【特朗普与产业集团系列3】

德国军事家克劳塞维茨说过，
战争是政治的延续，
德国哲学家马克思说过，
政治是经济的集中体现。

2024年的美国大选，
是产业集团与金融集团的战争，
2025年的美国政治，
则是产业集团改革上层建筑的动作。

统治集团变化了，政府的工作重心也会改变，
不同于特朗普1.0时代，
白宫常常为金融集团牟利的股市揪心，
获得产业集团全力资助的特朗普2.0，
更为关注为产业集团融资的债市和联邦利率。

中日等国的威胁方式和妥协方向，
也都从美股转向了美债。

敌人的敌人，某些时候可以成为伙伴，
产业集团的最大诉求，
是摆脱金融集团对产业集团的盘剥。

为了实现最后的胜利，
跟产业资本家对立的工农 MAGA，
也成为了可以团结的对象。

同样，受金融集团盘剥的中国、俄罗斯、伊朗，
这些曾经的敌人对于特朗普集团来说，
如今都可以短期变得可以合作，
甚至布雷顿体系瓦解后被金融集团抛弃的黄金，
也是这个失意者联盟的成员之一。

而那些跟金融集团坚决不会妥协的死敌，
都是实业集团天然的最坚定的盟友，
其中既有金融集团视之为洪水猛兽的数字货币，
也包括被金融集团打压了拜登四年的马云和他的蚂蚁。
