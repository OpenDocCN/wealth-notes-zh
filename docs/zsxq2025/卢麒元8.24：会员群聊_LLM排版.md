
# 卢麒元8.24：会员群聊

就国内而言，经济整体尚未进入增长轨道(一些行业和一些地区仍然处于通缩过程)。也就是说，经济基本面并不支持一个唐突的牛市。然而，资金量确实正在迅速增加(央妈放水6000亿+存款搬家20000亿)。资金量可以支撑一个技术性的牛市。

就国外而言，美欧日牛市已近尾声。加息周期开启，或许就构成获利了结。另外，MIT 报告也算是警告，AI无法形成生产力，也无法形成连续性现金流。意思是，深调周期就要开始了。讲外部，要说的重点，就是时间节点。内部和外部节奏错位了。

川的解放日是四月二日，六个月后是十月二日。关税战的影响开始显现，供应链正在加速重构(非美化)，所有在美的终极商品价格都将上升(通胀加剧)。贝森特也应在十月前后，大规模扩张美债发行。美国年底资管的最优选，是全部资产速冻。一则压抑通胀，二则逼迫资金进美债。

我仍然坚持向心坍缩的看法。唯，时间节点难以精确测定。我甚至担心，此次大A唐突牛市，再次为特定人群走资创造条件。我也一再提醒，AI泡沫的问题，不希望2000年科网股破灭的故事重演。

此次关税战，是否会造成类似1929年的大条呢?我想，应是1929年罾加1971年。先是资产价格崩溃，然后才是美元重置。2008年的危机被熨平，而非解决。这次，算是一次性清算。这次，能否拉上垫背的，以解决美国的历史遗留问题呢?大概率是可以的，欧日韩台全跪了。美国只要七伤拳不死，就能重伤欧日韩台，顺利完成收割。

至于俄罗斯，要看这次石油操作的幅度和力度了。大萧条的油汽价格崩溃，会重创俄罗斯经济。如战争不能告一段落，问题也会相当严重。至于我国，就看年底的转身了。将人民币与工业用电挂钩是重点。如能再启动水循环系统性建设，足以挨过这次危机了。

长金的部分，凡是境外纸金，请务必尽快完成转身。金银周期并未结束，但各国政府会在危机中采取强制性手段。纸金等金融产品极其不安全，务必小心。

至于短股的部分，各自把握好。我已经提示了AI风险，也不排除富贵险中求。不过，仍然希望，在速冻之前，全部转化为现金。机会，比什么都重要。

重要时点，会沟通的。
