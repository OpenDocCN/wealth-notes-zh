
# 公众号细分垂直领域的专家IP打造与变现经验分享

## 经验分享

### 个人背景

Peron 佩龙，2011 年中国人民大学本科毕业，进入市场调研行业，从事市场与用户研究工作。先后在博睿胜伦、Kantar（凯度）、尼尔森（Nielsen）工作超过 6 年，随后在 2017 年转到甲方互联网、智能硬件公司同程艺龙、vivo 开展用户研究工作 6 年。2023 年，离开职场，开始探索人生的第二曲线。

### 转型经历

一开始，Peron 想从自己的兴趣爱好着手，转行去做男士素人/形象改造，但半年后一无所获。随后，接到了一个朋友的公司（一个大公司）做消费者/用户调研项目的研究外包，费用 6w。随着需求变多，收入也还行，Peron 注册了公司，相当于创业，重新回到了市场调研行业。

### IP 的 1.0 阶段-探路

接了一段时间外包项目后，Peron 面临需求不稳定、利润低的问题。为了拓展客源和业务，他开始寻找更多的客户，接更多的一手或二手的调研项目单子，并考虑知识付费。

2020 年左右，Peron 开始在互联网上输出专业领域的内容，在人人都 是产品经理平台开设了账号，积累了一些粉丝。2022 年 1 月注册了一个自己的公众号“用研那些事儿”，积累了大约 2000+垂直粉丝（基本都是同行）。2023 年 8 月，Peron 重新捡起了断更半年的公众号，发布了回归后的一系列文章，并改名为“佩龙 Peron”。

### IP 的 2.0 阶段-聚焦 TO C

2023 年 12 月，Peron 把公众号“佩龙 Peron”改为“Peron 用户研究”，加上了行业属性。他意识到，当下这个阶段，他的资源、能力只能支持他在垂直领域深耕，走精准获客的路。

### 研究专家型IP

Peron 研究了刘润和小马宋等专家型 IP 公众号，发现他们都通过分享个人相关的内容、商业观察、职场技能提升干货、个人成长&鸡汤类内容来构建个人 IP，并通过这些内容进行精准获客。

### 启发与总结

1. B 端和 C 端业务都要做，C 端知识付费业务，一定程度上可以帮助 B 端业务获客。
2. B 端和 C 端业务要有侧重和取舍，就算刘润、小马宋，也没法同时做好 B 端和 C 端的业务，而只能根据自己的优势选择其一。
3. 做好内容的 1+3 法则。“1”是 1 个 IP，不要做只分享干货内容的工具人账号，而是做有血有肉的个人 IP。“3”是内容框架，包括专业领域的知识、干货；基于人群破圈内容的构建；围绕个人 IP 内容的构建。

### 2024 年重心转向 TO C 知识付费业务

2024 年 3 月，Peron 做出决定，重心转向 TO C 知识付费业务，建设了体系化的知识付费产品：

1. “用户研究成长圈”知识星球，定位入门级产品。
2. 大厂用户研究训练营，定位中端产品。
3. 1V1 年度陪跑咨询，定位高端产品。

此外，还上线了“和君商学·战略用户研究班”和“出海调研 VIP 会员俱乐部”，瞄准行业内的用研一号位和出海调研人群。
