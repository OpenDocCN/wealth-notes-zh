
# 被分管领导故意刁难，应该怎么处理？

我单位是地市正处级单位，我是事业编正科，在这个单位七年了，分管是个没有结婚的老妇女，副处级，官僚主义严重，派头十足，经常压榨欺负下属，说话尖酸刻薄，做事对下属异常苛刻。我才领这么点工资，被她精神PUA，我有几次忍不住怼她，她就去一把手那里告我的刁状，自己在我面前充好人，让一把手一次又一次找我谈话，话说得很难听，挑拨一把手和我的关系。我现在和分管关系已经非常恶劣了，僵到极致，分管到处抓我小辫子，想要整我。我时时提防着。现在应该怎么办?我不想和她和好了。

## 问题

## 分析

谁都不愿意和别人闹矛盾，不近人情的领导不多，否则也当不了领导。下属需要找到上级的G点，实现良性互动。群众的眼睛是雪亮的。分管尖酸刻薄等等，大多数人也会有意见。一个人的性格是藏不住的。包括一把，心里应该也对她有所了解甚至有过冲突。

领导一般会用自己的话语权打击别人，别人根本就没辩明的机会。你是有的。一把找你谈话，不知道你咋说的，怎么扭转一把的思想认识。要在这方面下功夫。要客观冷静分析一下事情发展到现在，是什么原因造成的。然后才能对症下药。

## 选择

被动应付。按照自己的心情来，爱咋地咋地，不吃官僚主义那一套，把该干的活干好，谁也不能开除你。这样的下属，领导也头疼。

硬刚到底。发现利用她与别人的矛盾，团结更多的人。抓住她工作的纰漏，留足证据，告她的状。找各种借口不执行或打折扣执行她的任务，她又抓不住把柄。积极和一把汇报工作，争取一把的理解。

修复关系。找机会谈一谈，委曲求全，接受她的领导风格，服从领导。你看她一方面官僚主义，尖酸刻薄；另一方面，也讲究策略，会在你面前充好人，背地里告状。真真假假，面子要维护，该硬的时候还要硬。就是个度的问题。我感觉越是分管这样的人越容易满足，她要是自己有办法也不会去找一把。

调整岗位。经常打架，工作无法正常开展，请求一把调整岗位。
