
# 学习笔记，2025.8.15

AI，会不会把我们带出泥潭？让我们拥有国际化的视野？很难。你这么想，若是隔壁邻居开个AI平台，AI一定是阉割版的，它不能政治不正确。

国产的AI，其实也存在这个问题。例如，你问一个最简单的问题，国内机器狗是否使用的国外开源？答案是，NO。

其实，你看看王兴兴的创业历程就知道了。机器狗、机器人这些东西，小米曾经就是第一梯队。还记得雷军的铁蛋不？小米用的就是开源，小米也不避讳，在介绍里也写的很明确：小米“铁蛋”借鉴了麻省理工学院仿生机器人实验室开源的MitMinicheetah项目成果，包括电机驱动器、中心板代码和硬件等，在此基础上进行优化和重构，开发出第一代CyberDog……

所有智能化的业务，包括武器装备，最难的其实是0到1，从1到2反而容易了。0到1，是需要非常漫长的投入，因为要构建底层逻辑，例如国产操作系统，你一搜一箩筐，其实？都是基于Linux内核的二次开发。通俗一点理解，国内手机操作系统，也多是基于安卓的二次开发。国内车机，也是如此。基本都是基于安卓的二次开发。就是你看到的车机，其实是一台手机操作系统。

特斯拉，除外！

什么样的团队或投入才可能开发出独立系统呢？类似华为的鸿蒙。我还是那个观点，在科技前沿方面，尤其是研发领域，我们并不在头部，但是在具体应用方面，我们就是头部，参考中概互联以及大疆无人机。

要买全球车型，审美在线，品牌在线，哪怕过上十年八年，你回头看看，依然经典。

现在看看2000年，2005年，2010年，2015年的陆地巡洋舰。哪一版不帅的要死？

有个朋友靠AI帮人写论文，赚了1000多万。

写应该不是多大的门槛，核心是业务。这个业务我大约在20年前就接触过，我当时谈过一个英语系的女朋友，她就是帮人写论文，在谷歌上投放竞价广告，一年30多万的收入，那个年代的30万，堪比今天的300万。

前两天，我咨询了大学老师这个问题，问她如何看待这个现象，她跟我讲，正经大学的正经老师，基本都是自己写，因为AI对于相关领域的研究和认识不可能超过他们，野鸡大学以及下面的职业中专，老师多是代写或AI。

通篇，你写了上岸，但是一点价值都没有，我读完以后，只觉得浪费了几分钟。

她说，那我删除。我说，不要，这是你来时的路，另外字数太多了，你要少写，写两三百个字就可以的了，当个随笔，当个日记，另外你把线拉的稍微长一点，你计划写20年，你站在20年后看今天，20年后，你就是县长，为什么？因为，你在本地拥有10万粉丝，他们关注你的一举一动，关注你的分享，你就是本地王，这种渗透力是无敌的。

她问，那你觉得自己像县长吗？我说，不像，我就是。她说，要放在过去，这一句话，就足够砍你头了。我说，在我自己的世界里，是。

大多数人说，对于写作，怕坚持不下来。

其实，写作，这是美好的事，咋可能需要坚持呢？前几天我分享的段永平浙大分享，他反复提到了一点，就是要有长线思维，五年或十年或者二十年的规划，对于写作这个行业更是如此，你能站在未来看今天，就无敌了，这个世界上值得玩的游戏有很多，有什么游戏比把自己的想法装进别人的脑子里更刺激？所有的大人物，本质都是洗脑高手，他们让你认可了他的说法，甚至使用了他的操作系统，他让你爱谁你就爱谁，他让你恨谁你就恨谁，甚至你为了捍卫他，愿意为他去死。

只要有写这个意识，并且愿意写，那么就有东西写，因为你会启动摄入模式，会主动去观察，去采访，去思考，说白了，你一直在进化，可怕的不是写这个行为，而是你的进化速度，一个人持续的进化，20年之后，你对这座小城就是俯视状态，那个时候其实你不可能是本地王，因为本地人读不了你，他们甚至觉得你胡说八道。

农村人对有出息的人，就一个评价标准，有没有对家乡出力，一是基建，二是政策，三是安排工作。

那真是一人得道鸡犬升天。

现在微信信息，我基本不回，不管是关于定投的还是写作的，为什么不回呢？因为我坚信，若是一个人真的要找我，他会通过朋友介绍联系上我，你这么想，你都想学定投了，难道一个做定投的王者不值得亲自见一面？你想学写作了，一个写了20年的自媒体人，不值得见一面吗？这就是最好的门槛。

你看，警察为什么多抽烟？因为，两个原因。

A，盯梢时，不抽烟，人熬不住。

B，案情研讨会，都是一根接一根，而且是不限量供应，你看《重案六组》里就有类似的镜头，新拍的刑警题材的多没有这个画面，因为抽烟不允许出现在影视剧里。

做互联网的人有个特点，什么都喜欢自己研究，喜欢微信交流，不愿意见面。而传统生意人呢？喜欢见面。你看看雷军，每天都在见不同的人。

有个自媒体，专门晒这些企业家专机飞行线路的，例如网上晒了马云在长沙考察盒马，当时阿里巴巴突然大涨，因为有国外媒体透露，马云可能会出席北京的会议，我在想，他要是去北京，怎么不提前一天去？

所以，我总觉得，他大概率不去。次日，我去这个账号看了看马云的飞行轨迹，他从长沙下午不是直飞北京，而是先去了重庆，从重庆晚上飞的北京。

一天，都要周转几个城市，见N个朋友。

这种信息交换密度与我们宅在家里搜索来的所谓的知识，差了十万八千里。

就如同飞书CEO谢欣说的：AI给生活带来的实际变化很小，大家还是该上班上班，该还房贷还房贷……

真正的成长，还是要见人，路费是最好的学费。
