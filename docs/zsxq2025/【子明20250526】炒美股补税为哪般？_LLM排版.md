
# 炒美股补税为哪般？

发布时间：2025年5月26日

为了解决全球后疫情时代的天量赤字，全球主要经济体都在琢磨着加税解决，也都不约而同的搞起了全球税务信息互换，以查清楚池塘里面都有哪些大鱼。中国作为过去四十年全球经济增速最快的地区，秉承着“能力越大责任越大”的原则，自然也就成为了本轮查税和征税的第一重灾区。各国为了能够从中国这边征收到更多的税收，各股力量也都开始八仙过海各显神通。

只是有的利益集团更喜欢搞罚没，有的利益集团更喜欢征收关税，有的利益集团更喜欢征收个税。作为西方最主要税收来源的个税，为了能够从中国身上征到，剑拔弩张的黄老爷和县长们也可以坐起来聊分Dollar。为了表示诚意，也都纷纷跟中国分享了一部分的涉税信息。为了避免我们一家独吃蛋糕，黄老爷们自然懂得留一手，给我们的税务信息都是不完整的，使得我们眼睁睁看着巨大的税收蛋糕，但都无法形成直接征税的证据链。

于是，税总只能把各类信息按籍贯分给各地税务局，让大家自己想办法。而各地税务局手头只有些支离破碎的信息，所以征税时也没办法按照全年来计算和扣除，只能针对一笔笔盈利的信息来“诈”各位老爷，用反复计税不扣除诞生的巨额税款与罚款，让大家为了自证清白，主动提供全年的盈亏。因此，只要不是地方特别黑的情况，基本都可以跟征收科谈出一个比较低的缴税额度，甚至私下给点钱就能对付过去。但是，在全球补亏空的大背景之下，进一步的补税扩大化，终有一天还是要来的。
