## 赖德在这个节骨眼敢于冒天下之大不韪，公然叫嚣“敌对势力”论，推动实质性法理台独并踩中红线，其实也是迫不得已。他眼下再不拼一把，恐怕就永远都没有机会了。

从054A抵近苏澳军港可以明显看出来台非法武装已经不堪一战，台非法政权已经丧失了一切底牌。一艘海军护卫舰都抵近海岸线不足30海里是什么概念？这等于是贴着鼻子刷存在感了，两岸军事实力对比已经被拉大到了无法接受的程度。

从2000年之前敌强我弱，到2010年之前互有千秋，再到2020年之前频繁空海巡逻“遛狗”，现在很明显狗子已经被“遛”死了……

网上大多数观察家还没有注意到这个问题，作为严密关注台海两岸军事信息的人，我是第一时间注意到了这一点。

517号“衢州”舰抵近苏澳军港，引发台非法武装内部一阵骚乱，各种无线电信号传递乱中出错被我方严密侦察也是在情理之中。在“敌”舰突然冒出来的时候应对失措，泄露重要无线电频率和信号特征也就水到渠成。

这里必须要解释一下，517舰抵近苏澳军港可不是仅仅为了示威，更多是为了进行无线电侦察。台非法武装仓促之间很难完全执行无线电防护和静默规定，可能会紧急启用保密频率或者慌不择路之下明码发报，这些在战时都是极其重要的事情。517舰这一次突袭想必是收获满满，也不知道台非法武装慌乱之下干了什么,个人推测有可能有：

1、仓促之下明码发报，为我破译其密码系统提供了重要参考；
2、动用紧急频率，为我侦获其隐蔽电台频率、位置提供了重要信息。有些无线电侧向设备能够在很短的时间内定位电台位置，这些平时不动用的保密电台有可能就此暴露；
3、呼号、代号没来得及正确使用，慌乱之间忘了暗语、代号，以舰名、单位名直接呼叫，为我破译其暗语系统提供了参考。

总之，无线电频率和内容泄露可不是个小事情，517舰捅这一杆子，你猜后面有没有专业的电子对抗装备跟着呢？海里的侦查船，天上的侦察机？我猜是有的。

我们从2000年以前被人压着，到2010年代勉强形成均势对抗局面，再到2020年代好不容易拥有了“区域拒止”能力，开始了年复一年的“遛狗”行动。既以我方掌握了制造能力的、几乎无限的装备生产能力为凭借，反反复复不断出动舰艇和飞机到台海海域组织巡航行 动，让台非法武装主战装备急剧消耗寿命。

把中美两国的"由海向陆"摆在一起对比一下，你才能发现差距。

美国版本的“由海向陆”是浅近纵深下的非接触打击模式为主，以舰载航空兵为主要火力支援，在海岸线上摆出“战斧”巡航导弹-舰载机投弹-舰炮形成的三层式火力网，配合少量的海军陆战队特种行动，对陆地国家发起干涉。

这里面需要注意的是，美国陆军实际上缺乏攻坚作战能力。自美国成立以来，美国陆军一直乏善可陈，缺少与强大对手正面对抗的经验，包括二战中对德作战，德军主力也是在苏联那一边。

尤其是以弱胜强的战斗，美国陆军几乎一场都没有，因此造成了美国的“由海向陆”策略一直到破产都只能做到“浅近纵深”、“非接触”，说白了，欺负欺负小国还可以，真的打，连个二流国家都不一定打得过。伊拉克这种体量的对手就是他这个“由海向陆”的天花板了，再大一点例如伊朗，就超出了他军事干涉的上限。

中国版本的“由海向陆”尽管现在还没有形成完全体，不过很显然浅近纵深是不能满足的，一次性投送一个陆军集团军规模的突击集团恐怕才是最终形态。

中国陆军可是从来都不怕“以弱胜强”的，反而是打惯了以弱胜强的硬仗。这次曝光的巨型登陆驳船很明显是纵深突击准备的，不打纵深突击，用不上这么庞大的登陆神器。

这是什么概念呢？

一个集团军，是一个最基本的突击集群，同时也是信息作战的一个基本单位。

一个集团军有两三个合成旅，这是基本的突击单位，同时也是信息作战中的一个“信息节点”，是负责获取打击目标的信息节点，在接敌作战中摸索到敌军的要害所在；
有一个炮兵旅，这是基本的火力单位，同时也是信息作战中的“火力节点”，是在合成旅获取打击目标之后进行火力输出的主要单位；
有一个防空旅，为作战部队提供区域防空能力；
有一个陆航旅，为作战部队提供空中突击力量和垂直运输能力；
有一个情报侦察旅、一个特种作战旅、一个信息旅、一个电抗旅，分别提供各自专业领域内的作战保障支援能力；
有一个工程防化旅，提供工事构筑和防化学武器能力。

以上作战和支援、保障部队，就能够形成一个基本的突击集团，拥有完整的纵深突击能力，能够短时间离开完善的保障体系独自完成纵深突击任务。

说白了就是，能够送上去一个集团军，就无论如何都能打开至少一条通道、形成十几公里的正面战线，为后续部队大规模登陆提供充裕条件。一个集团军规模的登陆突击力量，跟美军那种小打小闹治安战为主的“由海向陆”不是一个概念。

大型登陆驳船的曝光充分说明我军现在的登陆作战思路，绝不仅仅是"由海向陆"，而是登陆以后能够持续作战。

第一，这东西就是个巨大无比的靶子。

是的，巨型登陆驳船是个非常显眼的靶子，也没有任何自卫能力，在没有绝对制空权/形成绝对压制的前提下是根本就没办法用的。反过来说，敢于用这玩意儿，就说明绝对制空权、制海权和对地面火力的绝对压制，我们有充分信心。

第二，海军陆战队根本用不到这东西。

海军陆战队的装备自带泛水登陆能力，即使没有码头也能抢滩登陆，不需要搞什么驳船。只有老陆需要这东西，而老陆不以集团军为单位行动，没有意义。

第三，只要动用集团军，那就绝不是小打小闹。以前大家想象中的“火箭弹洗地"很明显是个笑话，根本不可能，我们没有对对岸抱有任何幻想，我们是扎扎实实按照登陆打巷战来准备的。
