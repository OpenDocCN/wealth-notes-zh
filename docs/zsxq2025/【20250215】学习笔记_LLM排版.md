
# 学习笔记

成武是菏泽的，穷地方，县委书记大刀阔斧，直接规划了一座新城，而且是水城，当地老百姓都说好，一到晚上，仿佛是大上海，不知道现在房子烂尾了多少。没有上海的命，但是得了上海的病。有兴趣的可以去抖音搜一下成武夜景。在地产大跃进时代，谁敢大刀阔斧，谁就能扶摇直上。但是，不能是最后一棒。

举债越多，GDP增速越快，但是后患也无穷。山东这些年发展最快的区域是潍坊，但是潍坊也是城投爆雷最多的地级市，增速快到什么程度？官方都一度怀疑数据造假，去核实后感叹，真牛逼。

开始做自媒体，一定要有流量思维，比方说追热点，标题党。后来，应该把精力放在内容思维上。不要想着多少人看，而是想着我今天的分享，能get到自己吗？能给大家的生活和工作加分吗？

很多男人之所以穷，赚不到钱，是因为他将时间，都花在了没有意义的事情上。每天花大量的时间，看军事信息，历史趣闻，国际关系……看起来好像会让自己，显得博学多才的内容……饭桌上，最常讨论的话题就是，中美关系如何，国家经济如何……一个个口若悬河，指点江山。但是你问他们，怎么把一个小店经营好？或者怎么赚到钱？他们立马哑口无言。一个男人来说，真正值得骄傲的是，你有本事通过自己的努力赚到钱，给自己的家人，打造一个好的生活条件。让自己的父母，可以安心养老。让自己的太太，可以免于操劳。让自己的孩子，可以安心学习。能够做到这三点，比你懂再多历史、军事、国际关系都了不起……

段永平反复讲的那句话是什么？做对的事。要说财富神话，澳门不是更多吗？有人问了段永平，如何看待比特币？他回了四个字：不懂不碰。在你没有熟悉基本规则前，进入就是韭菜。我们做自己最擅长的投资，把目标降的低一些，不要想着一年翻倍。一年10%，你在中国就是顶级选手。这十年，我接触过炒股的人里，只有两类人活了下来。一类是基金投资者，例如张坤的信徒们。若是放眼整个股票投资圈，基民的收益是第一梯队。一类是定投派，拥抱时间的。凡是炒个股的，抓涨停板的，都是喜乐无常。近10年，整个涨停板战法里，除了徐翔，你能数出20个大户吗？过亿的。

都说远离消耗你的人，但，如何远离呀？1，敬而远之。2，撕破脸。这个远离，是需要你有一颗坚如磐石的心，才行呀。说起来容易，做起来，其实很难。在远离的过程中，你会经历撕心裂肺的痛苦，经历过后，你就成长了。

平行世界，我以前办公室在万德，万德最初定位是5A写字楼，后来成了公寓，日租房，我是开了眼界，鸡窝、吸毒的，应有尽有，电梯里永远都是黄毛+大纹身，平时我们正上着班，走廊里此起彼伏的叫床声，我当时就在想，日常生活中，从来没遇到过这些人，为什么突然冒出了这么多？其实他们一直都在，只是一直都生活在城市的褶皱里，一是褶皱空间，一是褶皱时间，过去在城乡结合部，后来在日房租，昼伏夜出。

XX被拘留的时候跟我讲，一个拘留所有3000多人。什么意思呢？若是把他们原先的生活轨迹点在这座城市的地图上，就是星星之火，密度非常高。我不知道大家有没有刷到过刑满释放的。下面全是喊哥的。这就是一座城市的社会网。混社会的。

脚气不会传染妇科病的。这完全是两种不同的细菌，混合洗没有任何问题，我说的是，脚气，不会导致妇科病。但是，尖锐湿疣，会到处传染，口腔，菊花……小姐多是皮肤科医生，为什么要提出帮你洗澡？其实就是望闻问切的过程，通过颜色、形状、味道，就能判断出是否有皮肤病，从而是YES OR NO。

农村人喜欢串门，城里人就很少有串门这个概念了，过去谈事，都要到家里去，现在谈事都在外面，彼此之间也不知道对方住哪里。

普通院校，非体育专业，全体师生体能比武的话，北大清华依然是独一档的存在，自律不仅仅是学习自律，跑步也自律，人家操场上的标语是什么？为祖国健康工作50年。
