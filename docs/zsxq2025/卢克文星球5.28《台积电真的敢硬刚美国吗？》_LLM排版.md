
# 台积电真的敢硬刚美国吗？

250528 文/卢克文工作室嘉宾 西门丑鬼

近日，多家主流媒体报道，台积电向美国商务部下属工业与安全局（BIS）致函，措辞强硬地警告称，若美方执意对芯片征收进口关税，将影响公司在美国亚利桑那州的投资计划，台积电甚至有取消投资计划的可能——此举被理解为台积电硬刚特朗普政府。报道还指出，台湾省内舆论质疑赖清德当局不给台积电撑腰，致使“护岛神山”孤军作战。那实际情况如何？台积电想要什么？是否能达成所愿？

这事传得神乎其神，但似乎不太符合台积电的性格。为了还原事情的真貌，我们去扒了一下函件原文，果然和想象中不太一样。

这封电子函件是台积电亚利桑那公司，应美国商务部的邀请，对半导体232调查的征求意见稿进行反馈，标题为《台积电亚利桑那公司关于半导体及半导体制造设备进口232调查的评论》，卷宗号 BIS-2025-0021（XRIN 0694-XC121）。

函件全文符合跨国企业在政策游说中的常见策略，理性、礼貌、谦卑，并且表达意见时使用的是请求的措辞，比如“respectfully requests”（恭敬地请求）。

传闻中台积电对美国的威胁，原文比较完整的表达是：“简而言之，促进已经在进行的半导体制造项目快速推进，符合美国的安全利益，而拖延、阻碍或损害此类项目，则会削弱美国的安全利益。例如，对进口半导体制造设备征收关税，将导致项目成本上升，并可能延误进展，在某些情况下还可能危及许多已宣布的项目和仍在考虑中的项目的商业可行性。”

“台积电甚至有取消投资计划的可能”大概就来自于里面的这句“在某些情况下还可能危及许多已宣布的项目和仍在考虑中的项目的商业可行性”——注意原文甚至不敢直接提到台积电，也没有说取消。

还有一个表述倒是直接提到了台积电：“新的进口限制可能危及美国在竞争激烈的技术行业中的领导地位，并给许多已承诺在美投资的半导体项目带来不确定性，包括台积电亚利桑那公司在凤凰城的重大投资计划。”

函件全文三千多字，绝大部分篇幅是在表忠心、叹功劳，比如：

“台积电亚利桑那公司和台积电华盛顿公司提供了安全可靠的供应，为维护美国国家安全做出贡献，特别是在自然灾害或疫情等紧急情况下。”

“我们通过首次先进封装投资，将3DFabric®本土化的战略，进一步确保了最关键和先进的制造工艺在美国本土落地。这增强了（美国）国内技术实力，并显著提高了供应链的弹性。”

“在美国的领导下，美国半导体制造业正在经历历史性的扩张，台积电在亚利桑那州雄心勃勃的投资计划就是推动因素之一。”

为了说出那两三句稍微硬气点的话，用大量的表忠心、数功劳来铺垫，谨小慎微渗出纸面。

恕我直言，看不出来硬刚在哪里。充其量也就是相对坚定一点的“雅蠛蝶”，是一个悲催的小弟抓住时机喊出的：大哥别打，自己人啊！

用直白的语言把中心思想提炼出来就是：大哥您打人是对的，但是我恭敬地请求您不要打自己人。您看家里的碗是我洗的，厕所也是我刷的，亚利桑那州的三亩地也是我耕的，能给家里带来不少收益呢。您把自己人打坏了，很多活可不就没人干了吗？亚利桑那州的三亩地，也可能会受到影响啊。

事实上，很多半导体企业都收到邀请对232调查提供意见，比如镁光、海力士、英特尔等等。回应的结构大同小异，都是陈述自己在行业中的重要性、关税政策的后果，然后提出意见，但措辞显然没有那么委婉。

以英特尔为例，在提出意见时使用的“urge”（敦促）一词，比如：“英特尔敦促政府通过考虑以下调整措施，来加强美国制造业的领导地位”，“我们敦促政府在评估半导体、半导体制造设备及衍生产品进口对美国国家安全的影响时，考虑这些因素”。

和台积电“respectfully requests”（恭敬地请求）相比，谁软谁硬一目了然。

从逻辑上分析，台积电也不具备硬刚美国的条件。

台积电拥有最先进的尖端制程工艺不假，它包揽了全球90%以上高端芯片代工，尤其是人工智能芯片，美国短期内无法培育出同等技术水平的本土代工企业——但它的万丈高楼是建立在美国的地基上的。

比如，台积电7nm以下制程的EUV光刻机，含美国Cymer光源技术，受EAR（美国出口管制条例）管控；设计流程严重依赖Synopsys、Cadence等美国EDA软件供应商；3nm/5nm工艺关键材料（如高纯度光刻胶）和设备（如极紫外光刻机）依赖美国及其盟友供应。

如果美国切断技术授权，台积电先进制程产能基本就瘫痪。除非台积电有“我活不活无所谓，我只要你死”的勇气，否则它无法真正跟美国硬刚。

至于台媒指责赖清德当局没有给台积电撑腰，比如台湾《联合报》流传最广的那篇《台积电不想跪，赖政府呢？》，那就更是无稽之谈。

征求意见这一环节，通常是一个国家政策或法规制定过程中的内部行政流程，其他国家如果插手那就属于干涉内政了。等它最终发布，你怎么抗议和反制是一回事，但中途干预尤其是公开干预，那就违反国际规范和礼仪了。

所以赖清德当局跪舔华盛顿是事实，但说句公道话，这一次却实属“人在家中坐锅从天上来”。

虽然谈不上硬刚，但从函件看台积电确实有点急了。它“恭敬地请求”美国政府豁免已承诺投资企业免受关税及进口管制冲击——这一立场还是非常明确和坚定的。

就是以在美国投资建厂为筹码，要一张免死金牌。

我们分析一下它在美国的产能和营收，就知道它为什么急了。

截至2025年第一季度，台积电的主要产能仍然在台湾地区，占比85%；而美国、欧洲、日本等其他地区加起来一共才15%，就是说在美国的产能最多不超过15%。根据台积电2025年第一季度财报，来自北美的营收占比却高达77%。

这样一计算，需要通过外贸进入美国的订单至少占总营收的62%。

对比之下，而英特尔在美国本土的产能约占40%，在美国本土的营收却只占20%-25%，无需海外产能的输入——特朗普关税政策对英特尔这样的本土公司直接影响不大。

所以你看，台积电不急就怪了。如果美国按原计划对台湾半导体加征25%-100%的关税，台积电不死也得脱层皮，友商却安然无恙。

那台积电能如愿拿到免死金牌吗？这得从232调查说起。232调查指美国商务部根据1962年出台的《贸易扩展法》第232项条款，对特定产品进口是否威胁美国国家安全，进行立案调查。这项调查给了美国总统一个权限，就是根据调查结果，决定是否对被调查产品进行进口限制。

大概是这么个流程：

第一阶段，由商业主体（企业或工会）、政府部门申请，或商务部自行发起调查；

第二阶段，商务部通过问卷、公开听证会等形式收集信息，进行评估，在270天内向总统提交调查报告；（半导体232调查目前就处于这一阶段）

第三阶段，总统收到商务部的报告后，于90天内作出是否采取行动和如何采取行动的决定。

这一次半导体232调查，是4月14日美国商务部发起，显然是配合特朗普关税战的动作，为特朗普任性的操作提供合法性。所以它是先有的结论，再有的调查。台积电等相关企业提出什么意见，其实都不重要。特朗普啥都知道，调查只是走个流程，最后怎么做完全在他一念之间。

既然在特朗普的一念之间，那台积电表现的很着急，就不是什么好事情了。

特朗普何许人？平生不爱吃斋念佛，专爱敲诈勒索。你不急还好，你一急，他还不乘你病要你命？台积电这一次三千多字的“陈情表”，起到的反作用只怕更大一些。

鉴于特朗普对制造业回流的执念，和半导体产业的重要性，可以合理预测，他可能会给台积电一定程度的豁免，但台积电需要给出更大的投名状。

2020年5月，台积电宣布在美国亚利桑那州投资120亿美元建设5nm晶圆厂。

2025年3月，台积电在特朗普见证下宣布对美投资总额追加至1650亿美元，含3座晶圆厂、2座封装厂及1个研发中心，成为美国历史上最大的外来直接投资之一。

这一次不出意料的话，投资额度不上大幅度拔一拔，台积电只怕吃不兜着走。

那台积电不能转投我们吗？恐怕很难。

就如前文所说，台积电的核心竞争力高度依赖美日欧的半导体设备和材料，它的高端产能无法脱离以美国为中心的现有国际体系。

台积电掌门人张忠谋多次公开表示，“芯片产业链很长，没有一个国家能够完全控制整个产业链”，实际上相当于承认台积电没有独立自主性，它必须依赖全球产业链。而美国就是这个产业链的链主，台积电转投大陆，首先就要自废武功。

所以，摆在台积电面前的选择并不多！
