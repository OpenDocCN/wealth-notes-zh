
# 世界主要国家情报机构-哈萨克斯坦

哈萨克斯坦是我们国家重要的“一带一路”上的一个节点，虽然它地处中亚，但是它的战略位置还是比较重要的。因为它连接着南亚、俄罗斯以及中国的西北地区。这个国家本身来说又是一个游牧民族国家，而且油气资源很丰富，国家的主要财政收入就来自于它的油气资源，它也是中国很重要的一个油气资源，尤其是天然气的主要的来源国。

1992年苏联解体以后，哈萨克斯坦就获得了独立，自哈萨克斯坦从独立至今，中国和哈萨克斯坦之间关系总体还是比较良好。当然中间也曾经出现过一些比如说中国西北地区
的哈萨克斯坦这个族的人，响应哈萨克斯坦前总统纳扎尔巴耶夫的号召，回流到了哈萨克斯坦。这些人现在很多从事中哈贸易，因为他们会说中文，也会说传统的哈萨克斯坦语，他们就从事两国贸易。

当时90年代，作为少数民族的哈萨克斯坦，能够
拥有一个以自己为主体民族的国家，对于整个民
族来说是有一个高度认同的问题。这个其实也反映出一个现实，我们在一些少数民族地区如何处理这些少数民族的中国公民和他所
谓的主体民族国家之间的关系。

我们先从哈萨克斯坦在独立以后的情报机构的组
织结构来看，哈萨克斯坦的最重要的国家情报机
构叫国家安全委员会，也就是所谓的前苏联的克
格勃的那一套的制度。因为克格勃以前就叫国家安全部长会议，缩写叫
KGB。1992年7月13号，当时的总统纳扎尔巴耶夫将哈萨
克斯坦的克格勃，改组为国家安全委员会，并且
将这一天定为国家安全委员会纪念日。

现在据我了解，整个前苏联的15个加盟共和国里
面现在只有白俄罗斯，还保留着克格勃这个名
称。也就是说白俄罗斯现在的国家情报机构的名字仍
旧叫克格勃，也就是国家安全部长会议。

那么哈萨克斯坦就将克格勃改组为了国家安全委
员会，国家安全委员会这个名字更接近于全球的
一个惯例。一般国家都是把情报机构作为国家安全委员会的一部分，或者就把它作为国家安全委员会来处
理。国家安全委员会设有主席，这个主席就是相当于
中国的国家安全部部长，那么由总统任命，哈萨
克斯坦国会来批准，直接对总统负责。

下设八名副主席，这个国家安全委员会下设包括
对外情报局、反间谍局、国家反恐中心、边防总
局、航空局、经济情报局、警卫局、政府通讯
局、网络安全局、边防学院和国家安全委员会、
军事学院。

虽然它这个名字从克格勃改过来了，但是从它下
设的部门结构来看，其实还是有着非常深的克格
勃的前苏联时期的这种所谓的大安全的概念。比如说它包括了航空局，包括了经济情报局、警卫局、政府通讯
局、网络安全局，包括边防学院，其实这个还是沿用了前苏联克格勃的那个架构了。

那么1993年纳扎尔巴耶夫，在国家安全委员会和
国防部另外又设立了情报局，1997年就是设立了
对外情报局，2009年纳扎尔巴耶夫将对外情报局
直接纳入到总统管理。因为原来对外情报局就是由国家安全委员会来管理，后来2009年，当时的总统纳扎尔巴耶夫就直接
掌握了对外情报局。
