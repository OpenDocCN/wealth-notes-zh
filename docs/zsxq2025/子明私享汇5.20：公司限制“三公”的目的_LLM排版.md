
# 公司限制“三公”的目的

发布时间：2025年5月20日
源自05192024群聊

站在员工的角度，
一个公司如果严格办公开支，
往往都是公司不行了，
媒体也会习惯于如此表述。
但站在公司的角度并非如此，
如果公司真的是快不行了，
往往都要打肿脸充胖子和撒币装有钱，
譬如许家印请天价网红造势的时候，
恒大的窟窿就早已遮不住了。

同样，我们作为一个“大厂”,
开始严查“三公”并不是为了节约那点钱，
而是为了清洗旧的交易秩序，
并清洗依附于旧交易秩序上的团团伙伙，
打击和遏制一批分公司的山头，
这跟前段时间精准打击南直隶地区崛起的“掼蛋”
的思路是相似的，
只不过这一轮清理旧秩序的规模，
扩大了很多很多，
再次回到了公司十年前的那一幕。
