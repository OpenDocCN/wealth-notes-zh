
# 学习笔记，2025.4.3(3)

1. 想要脑子清醒，脑子里可以经常循环这些词：科学、常识、概率、交易。很多骗局，都属于明显有违常识的东西，但贪念能让人忘记常识，不去计较概率和付出成本，会认为自己能是以小博大的赢家。《中国少了一味药》写的就是少的常识。

2. 成长的捷径，不在什么秘籍和外挂，在看见和相信。你看见优秀的人，你相信自己也可以成为优秀的人，你就会往那个方向发展。如果你长期泡在烂环境里，会觉得烂才是常态，很难想起来自己要去成长。

3. 自信，不一定是我当下很厉害，也可以是，我能清晰看到，我未来一定会很厉害。人得首先相信自己能做到，才能慢慢真正做到。比如，相信自己每天至少读一页书，未来就一定能思考更清晰，就更倾向于去长期阅读，从而变得越来越自信。对自己了解越多，规划越清晰，就越不容易陷入迷茫。

4. 对别人不要有太高的期待，或者说，自己的情绪、得失，不要过多寄托在别人身上。依赖别人，就是把自己的安全感放在别人身上，这样会患得患失，无法用正常方式和别人相处。性格弱会表现为讨好，性格强会表现为掌控，本质上都是希望对方能够牢牢握在手上，依赖对方。

5. 人如果40岁还不能准确认识自己，没有想过自己能做什么，该做什么，不能做什么，不该做什么，对自己，对身边人来说，都不是什么好事。自己活得糊糊涂涂，却想着去控制别人的人生。很多亲戚、家长就是这样，明明自己几十年没有什么亮眼的经历，却觉得能够指点孩子的职业路径，强行干预。

6. 很多事情，用道理讲不清，不是非黑即白，不是是非对错。同样一件事，不同的人看会有不同的角度，有人夸也有人骂，不要试图跟所有人讲道理，要去感受自己的经历，自己理解自己最重要。

7. 在不确定一个人愿不愿意帮自己大忙的时候，可以先找对方帮个小忙，如果对方愿意帮忙，及时给出回报，再考虑让对方帮自己大忙。谁帮谁都不是天经地义的，一上来就让别人替自己解决人生大事，本质上是对自己不负责，不能为自己的人生做主。

8. 相信很多东西的前提是，它得有逻辑。一件事如果没有逻辑，那就是胡扯，你不该在上面耽误时间。一个人如果没有逻辑，那就是糊涂蛋，你应该早点远离他。科学或许不能完美解释这个世界，但现阶段它是最好的解释模型，是人和人之间能够正常交流的基础。而且，科学很重要的一点是，它允许被证实，也允许被证伪，任何人都可以去质疑、去学习科学的东西，所以科学一直在进步。有些东西，是不允许你质疑的，不允许你讲逻辑，只要求你跟随，这样的东西，离得越远越好。

9. 人可以有大志向，但不能只有大志向。你可以有改变世界的心，但必须得先做到改变自己，改变自己周围的环境，一步一步向外拓展区改变世界。如果只有改变世界的心，而没有任何实际行动，就成了眼高手低。或者说，这种大志向，根本就不是想要改变世界，而是想要等着世界改变，让周围环境变得更好，让自己能在上面占到便宜。

10. 年轻人迷茫很正常，尤其是生活在互联网时代的年轻人，见过了太多答案，眼花缭乱，容易有既要又要的心态，既要博主们的精致生活，又要忽略个人实力去赚大钱。多经历，多尝试，多碰壁，跌跌撞撞，慢慢就找到自己的路了。人吃六个包子才能吃饱，不是说只吃第六个就行了，前面的五个，也得一口一口吃。
