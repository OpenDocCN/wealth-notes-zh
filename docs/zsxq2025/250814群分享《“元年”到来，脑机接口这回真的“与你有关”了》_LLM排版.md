
# 脑机接口这回真的“与你有关”了

2025年，被称为脑机接口元年。这一年被许多人称为脑机接口元年，因为在这一年，该技术同时在研发、应用、政策三个维度实现了跨越。

## 研发层面

根据工业和信息化部公布的数据，过去一年全球脑机接口的专利新增数量持续增加，截止到今年年初，全球专利数量已经达到4.7万个，其中中国专利数量超过1.54万件，位列全球第一。

## 落地应用

仅仅过去这半年，国内就有不少关键进展。例如，今年6月，中国科学院和复旦大学附属华山医院的团队实施了国内首例侵入式脑机接口的临床试验，为一名四肢截肢的患者植入了脑机接口设备，两三周后，他就能通过脑机接口设备下象棋、玩赛车游戏，控制电脑触摸板的水平跟普通人差不多。

## 政策层面

今年3月，湖北省出台了脑机接口医疗服务价格标准，这是国内首个价格标准。出台价格标准，关键作用之一是为医保的报销额度提供参照。同时，就在8月7日，工业和信息化部和科技部、财政部等七部门联合印发了《推动脑机接口产业创新发展的实施意见》，这是脑机接口行业的顶层设计文件。

## 脑机接口的核心原理

人脑的本质就像一台“从输入到输出”的机器。脑机接口的核心其实也很简单，就是替大脑完成两件事，一是接收信号，二是输出指令。在接收信号这个环节，大脑的工作原理是，先把一切感官信息，都转化成神经电信号，然后输入给大脑。而脑机接口的基本功能之一，就是替代人的天然感官，把外界的声音画面等信息，转化成大脑能直接理解的电信号。

## 脑机接口的想象空间

未来的人们可能会面临一个问题，寿命很长，但老年后的大脑无法清晰思考。除了保护好我们的“原装大脑”，还有一个可能的角度就是，用脑机接口设备来增强人脑的功能。例如，通过脑机接口，让大脑直接指挥机械臂。再比如，有些人的身体硬件有行动能力，只是大脑无法好好控制它们，这也可以通过脑机接口修复。

## 脑机接口与其他技术的交叉

脑机接口需要对人脑的电信号解码，而这件事恰好是AI擅长的。柔性电极能让脑机接口与人脑的对接更加顺畅，更充分。芯片的功能足够强大，体积足够小，脑机接口才能更顺畅地落地，它的穿戴才能更便捷。6G之类的技术，能让脑机接口的信号传输更快，更稳定。

## 新技术的演化逻辑

未来学家雷·库兹韦尔曾经提出过一个“加速回报定律”，说的是，很多技术之间是相互关联的，一个技术爆发，也会带动周边的技术发展。这就形成了彼此促进的加速度，因此新技术的发展往往比我们想象中更快。同样，有时对新技术的需求，也往往会来得比想象中更迫切。

关注新技术，不仅仅是在凑热闹，开脑洞，更是在为它在某一天的突然靠近，提前做好准备。
