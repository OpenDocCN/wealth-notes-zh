
# 2025我国集约化猪场种猪蹄病发生率与解决方案

## 我国集约化猪场蹄病发生率

2024年我国生猪存栏4.27亿头，主要集中在南方和中部地区，14个养猪大省存栏量占全国75%。集约化猪场定义：年出栏500头以上。

前20集团存栏母猪1049万头，占全国数量的25%。跛足发生率高低多以因跛足淘汰母猪数占总淘汰母猪数的比例来衡量。

## 蹄病相关概念

"猪肢蹄病"是对猪四肢和四蹄疾病的总称，也称为跛足病，主要病征是行走异常、站立不稳，四肢支撑身体困难等。金宝公司最早建立了“行走移动评分”评定标准以判断肢蹄病的严重程度。

## 肢蹄病对猪行为健康的影响

躺卧时间增长，活动量显著减少。跛行母猪仅占6.3%，非跛行占14.5%。采食量减少，影响营养摄入与生长。喝水减少，排尿频率降低，后躯变脏，导致泌尿生殖系统感染。母猪弓背比例增加，行动不灵活，容易压死仔猪，造成经济损失。

## 肢蹄病对母猪卵巢的影响

闭锁卵泡数与母猪后腿病发生呈正相关（P < 0.01）。

## 肢蹄病对母猪生产性能的影响

跛足母猪的PSY（每窝产仔数）显著低于健康母猪。23个养殖场，2014年至2020年间214,254头母猪淘汰状况分析显示，因跛足淘汰母猪的平均胎次为2.6±1.3，其他原因淘汰母猪的平均胎次为4.0±1.8。

木乃伊胎发生率在跛足母猪中是健康母猪的2倍（P = 0.014）。有表皮损伤母猪死胎几率增加（P = 0.021）。有白线病变母猪死胎几率增加（P = 0.036）。有蹄跟病变母猪压死仔猪几率增加（P = 0.017）。
