
# 学习笔记，2025.2.22

1. 对不如自己的人，客气、尊重，是最优解，不管是从人品角度还是从利益角度都一样。他们不一定能帮上自己什么忙，但如果你让他不满意了，他有很大可能会坏你的事。那些大佬待人都是客客气气的，不一定是他们真就那么礼貌，而是他们知道不礼貌的代价是自己不愿意接受的。当着韭菜的面，可以说要长期主义，要坚持，要努力，让韭菜感觉到自己的亲民。小圈子私底下交流，可能就是“傻叉的共识也是共识，你不割别人也会来割”。
2. 人大多都有一个“背叛成本”，不背叛，不是因为他足够忠诚，而是别人给他开出的价格不够，不足以让他冒险去背叛，利益大到一定程度，父母、兄弟、合伙人，都可能反目成仇。
3. 近些年在交流上的一个感受是，要多用陈述句，少用反问句。反问，很多时候对别人来说是一种质疑、不认同，容易引起别人的抵抗。陈述，尽量只描述事实，说自己知道的、看到的，不要轻易否定谁。
4. 情绪稳定，不是没有情绪，而是让情绪在一个合理的范围内，不让情绪干预自己的行为。控制不了情绪的人，通常也掌控不了自己的人生，极易被别人的看法左右。
5. 做生意，不是报课学出来的，是在市场上真刀真枪拼出来的。那些报课学习做生意的人，只是想有钱，想享受当老板的权力，被讲师们一人公司的理论洗脑了，并不想亲自下场去赚钱，去感受赚钱这个过程。课程，只适合学习一些已经成定论的死知识，学编程理论、学乐理基础，不适合学习那些变化无穷的东西，赚钱、艺术、创意，这种东西教不出来的，只能自己去练，去悟。
6. 比起拿假话骗人，拿真话骗人更不容易被识别。你说创业很难，普通人适合打工，他拿出三五个普通人创业成功的例子，告诉你这才是普通人该有的生活，丝毫不提这是多大普通人基数才出现的成功案例。普通人能创业成功是真话，但是他把核心的概率和成功因素给隐藏了，让你误认为普通人都可以这么成功，误以为跟着他学习就能成功。
7. 好多人是不知道自己没有边界感的，他们认为自己可以随意打探别人的一切，认为这才叫相互了解，就没有想过，别人有没有打算被他了解。一个小技巧，别人没有主动说的，大概率就是不想说，别非得刨根问底。
8. 若是想做形体管理这一块业务，应该专注于健身，而不是舞蹈或跳操。健身是抗衰老的终极武器，说白了，就是抗阻，常规抗阻肯定没有器械抗阻更有效果，这就是为什么瑜伽馆也纷纷上了健身器械的缘故。
9. 当经济下行时，集体进入了轻度抑郁状态，轻度抑郁的表现特点就是对外面的事表现的很漠然，觉得都与我无关，谁爱咋着咋着。若是继续下行，那么，底层老百姓越来越难。治安也会是个大问题。
10. 不想参加一些没有价值的聚会，但是，人家给你摆上筷子与碗，是最好的胁迫。为什么后来我参加活动少了？我那些酒肉朋友就这么对我，晚上聚餐，我不去，他们就把筷子与碗给摆上，拍视频发给我，说你不来，我们没法开始，我就不得不去。
