
# 1027洪灏星球问答（苍位和黄金白银）

请问洪总：
我准备将退出的一部分成长股配置到商品，请问现在是否是一个建仓的好时机?在现在的宏观环境下，请问洪总认为总仓位应该给到多少给大宗商品?在具体的标的选择上，除了之前提过的贵金属，铜，铝，是否还有其它的推荐?印象里洪总似乎比较少系统性的讲商品所以期待能够回答一下。谢谢!

洪灏回答：
我假设你是想把一部分成长股获利了结，因为成长股今年大幅跑赢其他板块，如果你布局早的话应该有比较丰厚的盈利，不如把一部分利盈利撤出来转到其他方面。大宗商品比如铜和铝，我在下半年展望时特别预测过的品种，主要是是因为流动性的条件上升。本周美联储降息是大概率事件，流动性的条件会更好。这些都有助于工业金属铜、铝跑赢其他的资产类别。特别是铜，在周末提示突破之后今天开盘跳升了。贵金属也是我们6月推荐过的机会，但金银回调还没有结束，我在最近的报告里有提到，我们已经获利了结，尽量避免参与短期反弹。所以，我认为工业金属可以给到20%以上的仓位。

提问:
洪灏总今年对黄金白银高低点全部正确了，上周黄金白银也大幅回撤了，不知道老师后面怎么看这两个贵金属？

回答：
黄金和白银从高点回调之后，下行的趋势或者说技术上的回调趋势是还没有走完的。很多朋友很着急，白银跌了挺多的从53跌到47，一些朋友想急急忙忙的冲进去。其实我觉得不要这么着急的，因为这么大的上行趋势或者说泡沫，没办法在短短的一两周的时间消化干净。当然，短期会有一些技术反弹，这些反弹是用来出货的不是补货的。另外，今天早上我们看到风险偏好回暖之后，金和银都是下行的，工业金属在上涨。在接下来一段时间，随着风险偏好继续回暖，地缘政治有所缓解。黄金和白银已经是今年跑的最好的资产类别了，不排除很多基金经理会获利了结的。最高时候获利超过80%了，所以获利了结是很正常的。换句话说不一定要死磕黄金白银，可以关注其他的机会，比如铜和铝，技术上已经开始突破了。所以关注别的品种会有更好的机会。
