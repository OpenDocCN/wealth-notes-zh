
# 米联储见闻250930——后来继之有殷公（中国已成长为巨人，不再寻求 WTO 差别地位）

今天是新中国建国76周年纪念日前夜，今晚去看了志愿军第三部——浴血和平，客观的说，从艺术角度来说，这部电影很一般，槽点很多，但是这部电影比较聪明的一点是密集加入了大量的历史事实，虽然是个流水账，艺术处理也很青涩，但因为考虑到真实的历史，尤其是最后加入了志愿军遗骸回国的情节，还是让人感触良多，我们对抗美援朝战争的了解还是太少，对我们今天的一切到底是怎么来的代价换来的没有清晰的认识，对于自己从哪里来，该到哪里去没有足够的认知，结合昨天提到的十五五规划提到的抓住历史的主动性，是需要对自身使命有清醒认识的才会明白其意义，而这些清醒的认识是建立在对因果的了解，所以历史虚无主义为何危害大，它让我们忘掉责任，忘掉民族精神，忘掉我们作为一种文明的继承者该有的血性。把9月30日定为国家烈士日是意义重大的，在国庆期间看看相关的历史电影，纪录片，书籍，也是很有必要的，从来就没有什么理所当然的幸福生活，一切都有因果，不能尊重因果，必然会有报应。

这次在中国国庆前夕，中国总理在纽约联合国总部宣布，中国作为负责任的发展中大国，在世贸组织当前和未来的谈判中，将不再寻求新的特殊和差别待遇。其实是中国经过76年的发展，长大成人的成年礼。但随后李成钢强调，中国仍然是世界上最大的“发展中国家”，中国作为“发展中国家”的地位和身份没有改变。换句话说，中国并没有因为发展将自己当成发达国家的一份子，所谓苟富贵，勿相忘，今天的中国没有因为自己发展了，就忘记对全世界无产阶级的承诺，俄罗斯当初舔着脸想要加入西方阵营被反复玩弄的现实让中国非常清醒，新中国的建立是为了建立真正践行社会主义体制，人民英雄纪念碑背后的碑文一直在强调，新中国的建立是什么，为什么，为了谁！所以，中国还是第三世界的成员，中国依然是发展中国家的一份子，穷朋友们依然是朋友，中国只是放弃了在世贸组织中按照发展中国家身份享受的特殊和差别待遇。

中国在2001年加入世贸组织时，人均GDP不到1000美元，因此享受“发展中国家”待遇。2020年初，美国贸易代表办公室宣布将中国排除在“发展中国家”之外。2023年美国国会两院更是分别通过法案结束中国的“发展中国家”地位。美国这么做，无非是认为中国占着发展中国家的身份，在全球竞争中处于不公平的位置，因为按照联合国的一些规定，发展中国家身份确实在各种游戏规则中有优惠地位和更宽容的条件，不得不说，当初美国建立起联合国的体制框架时还是有救世之心，也就难怪当时大家把美国当灯塔，在世贸组织的规定中，“发展中国家”可享受特殊差别化待遇，在消除关税壁垒的速度和幅度，以及在判定倾销的标准上，要求都比“发达国家”低。其中有超过150项条款都明确发展中国家享有“特殊和差别待遇”(Special and Differential treatment)，发展中国家成员可在关税减免、延长磋商期限、非关税壁垒等方面享有更优惠的待遇。这等于是给不同重量的拳手在擂台上实施不同的打击规则，比如，中国等“发展中国家”进口关税平均为14%，高于“发达国家”的7%；过渡期在4至8年，长于“发达国家”的2年；享受比“发达国家”更低的判定倾销的标准等。

早在川普的第一任期时就批评过中国的“发展中国家”的地位。在2020年2月18日美国贸易代表办公室宣布，更新其认定为“发展中国家”的名单，将中国、香港、印度等25个经济体排除在外。理由是，相关法律在1998年制定，已经“过时”，而当前这些国家可借由“发展中国家”的认定进行不公平的出口补贴，从而伤害美国的产业。2023年，美国参议院和众议院，分别通过一项名为“结束中国的‘发展中国家′地位法”(Ending China's Developing Nation Status Act)的法案，核心条款均要求国务院在所有美国参与的国际组织与国际协定中反对将中国作为“发展中国家”对待，并推动将中国归类为上中等收入、高收入或“发达国家”;若组织缺乏更改机制，则应推动建立调整机制。该法案并不直接“改判”中国在世贸组织等机构的正式地位，而是指指示美国作为成员国在谈判与日常治理中持续反对中国主张“发展中国家”待遇，施压国际组织限制中国在新协议中的“特殊与差别待遇”。

根据美国贸易代表办公室的公开声明，华盛顿的态度是，除非中国和其他主要经济体放弃给予“发展中国家”的特别差别待遇，否则世贸组织不可能进行有意义的改革，美国认为这种待遇给予了“发展中国家”不公平的优势。所以，这次中国在联合国的表态，也可以看作是中美谈判中，中国作出让步的一部分，对美国来说，既然要跟中国正面竞争，一定会在规则上保证对美国更有力，那些让中国占便宜的规则必须要通通干掉，中国也意识到，如果不去掉这些优惠政策，中美竞争就无法服众，美国不会服气，全世界也不会服气，因为这些优惠政策，让同台对打的双方，一个带着手套一个赤手空拳。中国需要让美国在这方面找不到任何借口。

所以中国虽然在 WTO 内不再寻求新的特殊和差别待遇，但并未放弃“发展中国家”地位。当然，站在西方阵营的视角来看，中国抱着发展中国家身份不放，是因为“发展中国家”在国际舞台上有很多好处。比如1972年的《人类环境宣言》、1987年的《蒙特利尔议定书》、1992年的《联合国气候变化框架公约》、1997年的《京都议定书》，以及2015年的《巴黎协定》，都在不同程度上承认“发达国家”应该照顾“发展中国家”，并允许后者暂缓执行减排措施，以及更宽松的减排义务，还要求“发达国家”向“发展中国家”提供资金和技术。背后的逻辑是，“发达国家”在过去几百年间是温室气体的主要排放国，所以现在应该承担比“发展中国家”更多的减排义务。

此外，西方认为，“发展中国家”的定位还有政治利益。中国长期将自身定义为“最大的发展中国家”，从老毛提出“第三世界”的理念，到美苏选边站，再过渡到团结“亚非拉兄弟”，发展到今天的南方国家概念，老毛70年前的布局，现在西方智库的学者们才看懂，当年抱着这帮穷亲戚原来有这么多政治红利，但是只有中国才明白，解放全人类就是tg的初心，而要在西方阵营的包围之下带领发展中国家杀出一条血路，杀到今天，甚至让西方国家开始垂涎发展中国家的发展动力，这背后有多少血泪苦难，多有前仆后继，只有中国人自己知道，也只有发展中国家明白。

大西洋理事会的一篇文章表示，中国通过金砖扩容、融资工具与话语输出，力图在全球南方内塑造主导性影响，用于重塑全球治理、削弱美国权力，并期待由此获取更大国际领导力与制度塑形能力。

但这一切从来不是别人赐予的，更不是从天而降的，西方人只看到今天的果，却从未真正去关注中国为果埋下的因。

实际上，从国际上各种标准来看，中国虽然还是发展中国家，但是从各项硬指标来看，中国距离发达国家已经是一步之遥，比如世界银行人均国民总收入(人均国民生产总值，GNP)超过某一标准就被认定为高收入经济体。该标准每年都在更新。2025年，该标准为13935美元，而2024年中国的人均GNI为13660美元，已经非常接近这个水平。按照中国上半年的增长态势，实际增速在4.5%到5%之间。综合测算，2025年中国的人均GNI将大概率超过高收入国家的门槛。

综合性更高的划分标准来自联合国开发计划署(UNDP)，该机构综合考量健康、教育、经济及不平等维度，为全球不同发展程度的国家评估一个人类发展指数(HDI)，并分为极高、高、中、低四档，只有第一档国家属于“发达国家”，其他三档为“发展中国家”。依据UNDP《人类发展报告2025》(以2023年数据计算)，“极高人类发展指数”(Very High HDI)国家共有74个，这部分通常被认为是“发达国家”，中国HDI为0.797，排名第78位，距离极高人类发展指数也只有一步之遥。尤其是考虑到中国是一个拥有10亿体量规模人口的国家，中国的总体实力已经远远超出一般发达国家，甚至比某些发达国家遥遥领先了。

不知不觉间，我们都见证了一个国家默默成长为一个巨人，这里面既有文明的密码，也有民族性的奥秘，千千万万的中国人前仆后继，在200年民族苦难史里涅槃重生，我们作为能够经历，参与，分享这个重要历史阶段的中国人，是非常幸运的。

国泰民安，从来不仅仅是一句吉利话，要做到这四个字，难如登天，难得的是，我们现在确实做到了。
