
# 世纪大谈判：雅尔塔2.0

2月7日，Trump在接受媒体专访时表示，他已与普京通过电话商讨以谈判方式结束俄乌战争。他对外界留下了充分想象空间，表示"我还是不说为好"。考虑到这是自2022年战争爆发以来俄美总统首次直接谈话，引发各方高度关注。2月9日，俄总统新闻秘书佩斯科夫回应塔斯社询问时说，他既无法证实也无法否认关于两国总统通话的消息。佩斯科夫指出，随着美国新政府开展工作，美俄之间正进行许多沟通，这些沟通通过不同渠道进行，他个人有可能不知道某些事情。

克里姆林宫于2月12日以文字声明的方式证实了这一消息，表示"通话持续了近一个半小时"，宣称莫斯科和华盛顿"是时候展开合作了"，紧接着便爆出美俄元首有可能近期在沙特会晤的消息。根据美媒说法，Trump希望泽连斯基和普京在2月底或3月份会面，4月份宣布停火，5月9日（乌克兰最近一次动员截止日期）前发布正式停火协议声明，之后基辅将被要求不再延长战时动员令，并于下半年举行总统大选。

## 雅尔塔会议80周年："三巨头"如何塑造当今世界秩序

过去几天，俄罗斯官媒放出许多关于纪念"雅尔塔会议"的消息，俄副外长亚布科夫公开表示，不排除召开新"雅尔塔会议"的想法，以便巩固在特别军事行动结束后形成的地缘政治格局。为纪念雅尔塔会议八十周年，克里米亚的一家画廊还举办了一场名为"雅尔塔2.0"的艺术展，其中最惹人注目的是一幅美、俄、中三国首脑（想象）画作——普京坐在中间，中美元首分坐两侧。

## 雅尔塔会议回顾

雅尔塔会议是一场"三赢"的会议。对于苏联来说，它获得了罕见的地缘政治利益，不光在广袤的中东欧地区收获了直接影响力，还即将在远东地区扫荡日本的势力范围。直到1950年《中苏友好同盟互助条约》签约前，斯大林还念念不忘雅尔塔体系，在思想上费了好大劲才突破其框架。苏联作出的重要让步主要是希腊问题（希腊爆发内战时莫斯科坐视英国消灭希共）和德国战争赔款问题（苏联提出的赔偿方案被英美否决）。

对于美国来说，雅尔塔会议最关键的意义在于苏联和英国认可了罗斯福对战后秩序的总体安排，这是其最在意的事情——框架大于一切。虽然雅尔塔会议是在克里米亚召开，但某种意义上美国才是真正"坐庄"的那一个，负责提供筹码、让渡利益给苏联和英国（美国战略界将雅尔塔-波茨坦协议视为"战术性退让"）。

## 当下的俄乌战争与"新雅尔塔框架"

在俄罗斯人眼中，乌克兰战争是俄国与北约之战，是"第二次卫国战争"，终极目标是要算冷战结束后三十年的总账，为今后几十年的战略环境奠定基础。普京不想要第二份"明斯克协议"，而是想要"新雅尔塔框架"。这一点某种意义上跟欧盟、乌克兰的诉求类似，即大家都不希望再搞个和稀泥的临时性协议，都希望拿个一劳永逸的答案。

然而，俄乌停火最大的可能仍是"欧洲版三八线"，原因有四：
1. 今天美欧的力量明显强于俄罗斯，他们不会答应莫斯科重划势力范围的要求，甚至连眼前的乌东四州都不会允许，最多睁只眼闭只眼，默许俄军对现有占领区的控制。
2. Trump无意让渡美国的利益或额外支付资源来协调俄欧双方，对他来说，“调停俄乌”更像是竞选期间许下的一张口头支票，如果现在欧盟能单独支持乌克兰打下去，Trump其实不在意停不停火。
3. 雅尔塔会议之所以让三方都感到满意，是因为二次大战打掉了德国和日本两大强权，海量的利益等待收割，而现阶段没有哪个重要玩家因此倒下（乌克兰摆不上桌面，利益不够分），真要倒也只能是力量最弱的俄国先倒下；简单点说，这场战争并没有打出来"新利益"，仍处于零和博弈状态。
4. "外卡选手"中国尚未下场；中方介入乌克兰战争的力度远低于美欧，并非当事国，理论上可以不参与它们的谈判（前面提到，其实没什么利益可分，参与谈判唯一的好处是可以避免美俄无限制靠近）；反过来说，只要中国置身事外，美国和俄罗斯就谈不出"新雅尔塔框架”。

总而言之，俄乌战争并没有达到普京重塑欧洲安全架构的意图，我们只能说这场战争"冲击"了现有秩序，把现有秩序摇晃了几下，距离重构相去甚远。新雅尔塔协定更像是俄方的美好想象，希望充分利用Trump的孤立主义思维，在国力下降通道中巩固自己对前苏联加盟共和国的控制（按照俄方设想，其势力范围还包括塞尔维亚、中东部分地区和西非、北非的部分地区）。

## 后普京时代的俄罗斯与中美关系

西方有一种观点认为，应将对俄政策聚焦于"后普京时代"，包括可能发生的重大外交转向以及二次解体。在他们看来，普京时期俄罗斯与西方的关系已无法挽回，而且当下普京的强力地位尚能压制俄国内部各种矛盾分歧，所以当下不必急于同俄罗斯达成协议，完全可以以拖待变。

中方学者代表、上海环太国际战略研究中心副理事长兼主任王南森以古训"退一步海阔天空"来劝诫俄方友人。未来俄罗斯可能的走向的确是一个重大"X因素"，需保持警惕性、开放思维与底线思维。

## Trump是否可能请求中方帮忙斡旋？

简而言之，如果停火条件过于偏向俄罗斯，欧洲和乌克兰就会引发激烈反弹，甚至故意破坏停火；反之，如果条件有偏离俄罗斯诉求的倾向，则普京未必答应。在后一种情况下，美国将需要有影响力的第三方来对俄"配平筹码"。

上任前夕，Trump曾多次发言称"中国可以帮忙实现俄乌停火"，有美方学者表示，请中国帮助的代价可能是一份新的联合公报，美国需明确"反对台独"立场，改变其"不支持台独"的长期政策。

## 结语

世界每隔一段时间就需要（局部）格式化。近现代历史上，每段国际秩序都是有期限的，往长了说最多百年（如维也纳体系），往短了说也就十几二十年（如凡尔赛体系）。其中固然有体系设计优劣的原因，但更重要的还是各国实力消长以及旧体系矛盾不断累积所致。设想一下，假如一个被现行秩序压制的国家突然变强（如二战前的德国），它肯定是要掀桌子的，即使暂时沉得住气那也是在等待时机，而非真正顺从。从这个意义上讲，世界秩序每隔一段时间就不得不进行更新，而更新的方式有很多种，如：战争、国际规则修改、领土变更等。
