
# 王朝的三股力量

本篇接《一步死棋》和《马宁远之死》。前文是从地方官的微观角度入手，用各种蛛丝马迹去揣测“天意高难问”，本篇试着切换到整个大明中枢的视角。

嘉靖朝末年朝中共存在三股力量，分别是以严嵩父子为首的“严党”，以裕王和徐阶、高拱、张居正为首的“清流”，以及吕方为首的司礼监—前两派势同水火，第三派保持战略模糊。

在大明官场的设置中，虽然没有内朝和外朝之分，但司礼监实际上充当了内朝的角色。

所谓“内朝”，是指直接服务于嘉靖的私人权力系统，包括宦官机构、机要秘书、情报机构等，它们的运作相对机密，权力大且不公开，与文官组成的“外朝”互相制约。

无论是剧中设置还是真实历史上，明朝中后期司礼监掌印太监的权力可与内阁首辅匹敌，史称“权过元辅”。再考虑到司礼监掌控着东厂、锦衣卫等特务机构，吕方本人又长期服侍在嘉靖身边，用“春江水暖鸭先知”来形容他并不为过。因此故事就从吕方切入。

明史上的王振、刘瑾、冯保等知名人物均为司礼监掌印太监，比较特殊的是魏忠贤，他是司礼监第一秉笔太监兼东厂提督，但时任掌印太监王体乾是其举荐的。

《大明王朝 1566》这部剧的开头是一场讨论“开源节流”的御前会议。王朝存在的时间越久，花钱的地方就越多，来钱的地方则越来越少；因此一个王朝最根本的问题就是“钱从哪里来，打算花到哪里去”。

在嘉靖看来，严党有严党的事情要办（里子），清流有清流的事情要办（面子），清流跟严党势同水火、斗来斗去，里面其实少不了仲裁者的故意为之。至于清流心心念念的“倒严”大计，对于嘉靖和吕方来说存在很大风险，即使执行也是极其机密的事情，绝不可能在公开会议上流露出情绪。政治家是最厌恶风险的一个群体，“在做决策时，优先考虑的往往不是‘利益’，而是‘后果’，以及‘后果’自己能否兜得住。”

从严世蕃的喊话能够体会到，吕方最初跟严党合作得比较多，是在嘉靖迈后才渐渐靠向裕王。吕方对冯保说“思危、思危、思变”，本质上是他自己的体会。

吕方第一次暗中押宝清流，是在裕王府的李妃（即万历生母李太后）生下世子后。世子的诞生极大拉高了裕王在嘉靖心目中的地位，由此使得吕方的中立天平开始发生倾斜，这期间的标志性举动是派冯保去裕王府当差。两个单位、两股力量要合作，第一步就是互派人员。

冯保是吕方的下属，他为了讨好嘉靖擅自打死钦天监监正周云逸，而周云逸冒死进谏是裕王一派默许的。在吕方看来，如果不把冯保交到裕王府，自然就无法化解司礼监与裕王之间的隔阂，因此冯保是带着认罪道歉的政治任务去的。

为了在道歉的同时保护冯保人身安全，吕方以“皇帝赏赐”的名义把冯保等若干名太监、宫女派去裕王府。经过一番努力，冯保最终成功获得了裕王和李妃的信任，并事实上成为裕王府与吕方沟通的秘密管道。

当然，这种秘密沟通尚达不到结盟的程度，因为嘉靖对于司礼监的红线是绝对不能与裕王党和严党其中之一走得太近，这一点吕方十分清楚。

冯保在犯错被贬之前，是大名鼎鼎的东厂提督太监。他的这次犯错客观上为吕方提供了一个将其安插到另一方阵营里的机会。

因浙江审案审出了严世蕃指示毁堤淹田一事，吕方担心闹得天下大乱，便私自联系严嵩和徐阶斡旋。尽管是出于对嘉靖的“忠”，但实际上人家并不领情，反而有被蒙骗的感觉，事后吕方被发配去“修吉壤”，暂时离开漩涡之地。

在《大明王朝 1566》的剧本设置里，锦衣卫和东厂都归司礼监管。严格来说，锦衣卫和司礼监是两个不同系统的机构，锦衣卫名义上受嘉靖直接指挥，可嘉靖是不会直接喊锦衣卫来办差的，他大多数情况下通过吕方去指挥锦衣卫，因此渐渐地锦衣卫就成了归吕方管。

比如果副秘书长协助常务副市长分管某几个局，管到最后副秘书长本人也拥有了很大影响力，成为半个分管领导。剧中有过这样一段情节，锦衣卫去浙江拿人期间，首先拜会了织造局太监杨金水，并征求其意见。因为杨金水和织造局也归司礼监管，吕方是他们共同的领导，有共同的管理领导那就是“亲兄弟单位”，锦衣卫自然把杨金水当成自己人，甚至会因为杨金水和吕方的关系而刻意保护他。

回到正题。既然吕方是嘉靖的情报总监，冯保又是吕方的人，那么冯保实际上也充当了嘉靖在裕王府的眼线。很多时候，上级在你身边有个眼线并不是坏事，他可以发挥意想不到的作用。

比如：在毁堤淹田事件之后，嘉靖召身在浙江的谭纶（浙直总督胡宗宪的参军）进京，谭纶本是裕王府詹事，按例应先到府中拜见裕王。然而在这一问题上，高拱和张居正之间出现分歧，后者担心谭纶若是先拜见裕王再进宫，容易被生性多疑的嘉靖猜忌。高张两人争执不下，最后是李妃一锤定音：“该怎么回答就怎么回答，王爺今天晚上应该见谭纶，最好叫冯保去叫他来，父子一体，没有什么应该隐瞒的。”

这里面“叫冯保去叫他来”就是关键一笔。裕王最初是不喜欢冯保的，但李妃却发挥了他的最大价值，如何统战对手是一门很大的学问。

杨金水有句名言：我就是皇上和老祖宗派到浙江的一条狗。毁堤淹田事件后，杨金水回到宫里接受嘉靖问话，在自己也参与毁堤淹田的情况下，他毫无隐瞒地说出了来龙去脉，并报告严世蕃是幕后黑手。

接下来切换到嘉靖视角。在三股力量的“权力跷跷板”背后，是嘉靖的“帝王心术”——宦官为手，清流为面，严党为骨，皇权为魂。

裕王是嘉靖唯一的儿子，也是帝国唯一的储君，嘉靖为培养裕王倾斜了不少政治资源，比如派给他的徐高张三位师傅，都是清流领袖。在改稻为桑大计实施期间，裕王一派举荐的谭伦、海瑞和王用汲也均得到重用。“把清流党安排在裕王身边，本身就

是嘉靖的一种态度，即希望儿子上任后能改变当前的政治生态。”那么为什么他自己不改变呢？因为严嵩太好用了，帮嘉靖干了太多脏活累活、遮风挡雨，他让严嵩做了近二十年内阁首辅，政治理念和政治生态上已经形成某种“绑定”，绝不

仅仅是一句“被严党迷惑”能下得来台的。

这跟《雍正王朝》里康熙选择雍正做接班人有异曲同工之处。康熙晚年采取了一种仁慈、无为的治国方式，对各种贪墨行为睁只眼闭只眼，考虑到自己年事已高，没有雄心壮志去大刀阔斧整顿了，因此便把振衰起敝的希望寄托给四爷。四爷以“孤臣”自居，性格果断狠辣；与之相较，八爷党跟当前利益集团高度绑定，很难起到革故鼎新的效果。

从这个角度讲，组织上在考虑一个单位主要负责人的继任者时，不仅会从政策延续性角度出发，还会重点思考如何“左右平衡”，防止在某一条轨道上渐行渐远、积重难返。现实政治中，钟摆式前进才是常态。

目光转向严党。在剧中情节发生的时期，严党整体上处于节节败退的状态，严嵩思考的挽救措施有四点：1、力推“改稻为桑”，向嘉靖展现自己仍有搞钱价值。2、把自身行为与嘉靖进行绑定，让清流投鼠忌器。3、拉拢自己的学生浙直总督胡宗宪—胡负责东南沿海抗倭战事，倒严势必牵连胡宗宪，进而影响东南战局。4、向清流示好，暂时缓解矛盾。

如果按照严嵩的计策来，或许能够慢一点崩盘，可严世蕃做的几件事完全打乱了严嵩部署：1、用“毁堤淹田”去推“改稻为桑”，主动为清流提供突破口。

2、公开场合抱怨严党为嘉靖办事背锅——在领导眼中，一件事情只要你获得了利益，就决不能再抱怨。3、因胡宗宪不支持“改稻为桑”而予以打压，分不清轻重缓急。4、策动“通倭案”向清流发起攻击。

整个过程中，严党的应对措施比较混乱，令出多门，而且心存幻想，全然忘记了

了大明官场的那句祖训：金杯共汝饮，白刃不相饶。剧中嘉靖的一句名言。嘉靖对裕王存在矛盾心理，既希望其成才，又担心他提前做大。即便有戒心，但严世蕃用通倭议题去攻击清流及背后的裕王显然不是一步好棋。

汝贞是胡宗宪的字，严嵩希望养寇自重来保护严党，但实际上胡宗宪已经跟严党产生隔阂。严嵩问他徐阶阵营有无插手干预浙江，胡宗宪回答说“没有根据的事情，弟子不敢妄加揣测”。严嵩意识到胡宗宪的圆滑，评价他“会当媳妇两头瞒”，然后说自己也是个“媳妇”，只不过比他“年长一辈”。

在官场和国际政治中，权力结构主要存在三种模式：绝对掌控、阵营对抗和分化仲裁。“绝对掌控”，是指掌权者或霸权国家拥有几乎所有权力，可以摆平一切事情，比如朱元璋以工作狂的姿态亲力亲为，废除丞相制；再比如世纪之交时的美国。

“绝对掌控”的要求非常高，往往只适用于个别几个历史阶段或极少数历史人物，由于精力、实力有限，权力终究是要被分享的。

既然要分享，那就存在两条路径可以选择，第一种是阵营对抗。简单来说，主要掌权者或霸权国家直接加入一方阵营——往往是初期较弱的一方，把部分权力分享给阵营内的人/国家，联合起来打压另一派力量。

以国际政治为例，冷战期间的美国选择联合西欧对抗苏联；再以嘉靖朝官场为例，严嵩最初是朝堂上弱势的一方，嘉靖曾全力支持他上位来取代之前的文官集团。

阵营对抗模式下，因担心己方阵营内有挑战者出现，霸权国家会谨慎行事，以削弱而不是彻底击败敌对力量为目标。比如有一种观点就认为，苏联/俄罗斯是美国刻意塑造的威胁，必要时用来敲打欧洲，防止欧洲实现终极一体化。

除此之外，还有一种更省力的策略是分化仲裁。该模式下，主要掌权者或霸权国家会刻意塑造两股或多股力量的对抗态势，自己超然世外，并保留随时打破平衡的能力。

放到古代历史，这叫“帝王心术”，往往出现在晚年精力不济之时。放到地缘政治里，叫“离岸平衡”——当下美国有向这个趋势发展。

在美国的离岸平衡策略里，其力量将集中在亚欧大陆周边三个关键地区—西欧、波斯湾和东北亚，同时放弃所有对内陆事务的干涉，并对重大地缘政治议题保持战略模糊立场。

电影《大决战》中有一个名场面，叫“刘亚楼，你记一下”。这个场面之所以爆红于网络，是因为领导下的指示清晰明了，毫不拖泥带水，下属可执行性强。

军事作战中，指挥官下达命令必须如此，成功了拥有无限荣耀，失败了则要承担责任。不同于军事指挥命令，官场之上许多话是模棱两可的。一句话说出来解读的角度有无数种，解释权在上级，责任主要在下属，表

述越是模糊，权力运作的空间就越大。剧中嘉靖便是“战略模糊”的高手，他经常写诗让下面人猜，把倒严的悬念保留到了最后一刻。

小说里有句旁白：“任何一句话，你不说出来便是那句话的主人，你说了出来，便是那句话的奴隶。波谲云诡的事在没有铁定之前，后发则制人，先发则很可能受制于人。”
