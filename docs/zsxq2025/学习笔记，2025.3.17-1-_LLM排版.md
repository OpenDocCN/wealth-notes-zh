
# 学习笔记，2025.3.17

1. 一个人能被嫉妒，说明和嫉妒他的人差距还是不够大，是因为自己做到了他想做但没做到的事。等你持续进步，真正彻底和他拉开距离，两个人就不是一个世界的了，根本就看不到他的嫉妒。

2. 不用什么人都去理解，什么事都去搞懂，关注好自己能力范围内的一亩三分地，才能过好自己的人生，精力太分散，最后什么都落不下。网上的内容，大多都是情绪化的，目的是为了吸引你的注意力来变现，自然要用尽夸张的方法。大多数人的目的，不可能是为了让自己情绪不稳定才去上网，但却总是被那些情绪化的内容吸引注意力，浪费很多不必要的感情和时间，可能还要被忽悠花更多的钱着买一堆不需要的东西。比如，你在小红书、抖音、视频号上看到的很多商品，去拼多多和淘宝搜同款，大多能便宜20%左右。但是给你来一个“姐妹们/家人们，买它”，你可能就上头为情绪付费了~

3. 人不要总想着为别人、为集体做什么，先想想可以为自己做些什么。一个人如果连自己都照顾不好，不可能照顾的好别人。

4. 普通人很容易陷入的一个时间黑洞就是和人辩论，聊一些乱七八糟和自己不相关的事。什么国际战争，明星绯闻，男女对立，社会公平……总觉得自己在正义的一方。就是没想过，人生这场游戏，你是想赢，还是想要辩个输赢？如果只是想要辩个输赢，你已经赢了，赢了很多次，然后呢？自己的生活还是一塌糊涂，那这个赢，又有什么用。

5. 一个人不能给其他人提供价值，别人永远不可能成为他的人脉，价值不是靠自己说出来的，得别人认，愿意为你的价值支付价格。空气很有价值，人都离不开，但是很少有人愿意为它支付价格，甚至有一部分人还会以给空气造成不可逆伤害的方式来获取利益。如果只能给别人充人头，最后怎么被忽悠死的可能都不知道。

6. 好多人在网上看多了这个赚十万、那个赚百万的案例，就觉得上班耽误了自己赚钱，要搞副业，要裸辞，要创业，要早日财富自由。实际上，一个上班这种只负责少部分环节都搞不明白的人，很难有离开职场独立赚钱的能力。大多数赚钱的事，都在闷头搞，公布出来的，要么卷的要死，要么请君入瓮准备收割。

7. 好多人说是想赚钱，实际上只是想花钱，想变得有钱，而不是去想自己能创造什么生产资料出售来赚钱。表面上是投资者，实际上只是金融消费者，追涨杀跌，没有自己的投资逻辑，根本拿不住钱。

8. 如果经常有人当面夸你，但是会贬低不在现场的其他人，就要警惕，很可能他在别人面前也是当面夸别人背后骂你。反过来也一样，不要在背后说谁的坏话，尽量当面说真话，背后说好话。

9. 一个人是否经历丰富、学识渊博，都不妨碍他被骗，因为很多人都不愿意接受现实，不愿意接受真实的自我，谁能帮他不去面对现实，他就愿意放弃思考接受别人替自己做主。一些年轻人总是拿老年人买保健品说事，说老年人认知跟不上容易被骗。年轻人也没少被骗，只不过保健品换成了知识付费、短视频的注意力消费、高溢价的情绪商品消费。保健品那些淀粉团子起码吃不坏身子，短视频刷多了人真的容易变傻。不愿意思考的人占是大多数，被骗才是他们的刚需。

10. 一些成功的人会把自己的成功归结为运气。运气也是实力的一种，他得有足够的实力能扛到运气的到来，才能凭借运气取得更好的成果。贾跃亭要是有无限子弹抗到现在，电动车的故事可能就得改写，但是他实力没能抗住。人生是一场无限游戏，有不下牌桌的能力，才能在机会来的时候抓住。
