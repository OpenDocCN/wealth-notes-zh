
# 学习笔记，2025.3.1

1，很多人想要的东西，都属于类似彩票中奖的小概率事件，天上掉个开挂老爷爷、隔壁来个小富婆、霸道总裁爱上我、掉落山崖学秘籍……渴望拥有那些被塑造出来的独一无二的经历。
这些都属于不切实际的想法，小概率事件，可遇不可求，每天想着这些事，只会钻进骗子的怀抱。
生活是在经历一些大概率事件，生来大概率健康、付出大概率有回报，所以才会相对稳定。
如果到处都是小概率事件，那就进入乱纪元了，什么都是不稳定的，数学不存在了，拥有再多，下一秒都有极大可能被收走。

2，焦虑分两种，一种是我们可以改变的，一种是我们不能改变的。
比如减肥，只要足够想改变，少吃多动，就能瘦下来。
比如战争，无论自己多想，都不是个人能力可以改变的。
适当焦虑是未雨绸缪，过度焦虑是杞人忧天。
要把注意力放在那些自己能够切实改变的小事上。

3，通常，说一个人“爷们儿”，不是说这个人粗鲁、好斗，而是说，他能承担责任、能成事。
一些人为了有“爷们儿”的面子，最后发展成了家暴男。
不过，语义也可能随时间而变化，就像现在小红书上很多女性用户会用“娘们儿”“太蒂了”来表示对女性的称赞。

4，一个人，不缺爱，才能好好去爱别人。
不缺，才能很好的往外给。
足够自信，才能发自内心去称赞别人。
“自我”是基础，要先满足自己，先好好爱自己，再考虑和别人的关系。

5，一件事，对自己有价值，不代表对别人有价值，汝之蜜糖彼之砒霜。
如果真想给，多思考对方真正需要的是什么，而不是打着“为了你好”的旗号去要求别人做不喜欢的事、接受不喜欢的东西。

6，越是没有做成过事的人，越容易低估做事的难度，越容易看轻他人的结果，也喜欢慷他人之慨，比如，会经常用到“不就是”来简单化某个复杂的概念。
一个真正在成长的人，会深刻理解“人外有人”，变得谦卑。

7，一个生意人，逐利，是正常思维，但是不代表凡事只看利益，人眼里如果眼里只有利益，就会变得唯利是图，没有信任可言，靠近这种人会有很大风险，随时可能会被出卖。

8，如果某件事已经有足够多的利益相关者，就会出现，阻拦新人加入，防止外人损害这件事的利益。
想要打破这种僵局，最好的方法就是掀桌子，重新定义利益，重开别的局。
比如某些软件、公司靠闭源形成几乎垄断地位，其他人想要改变这种局面，那就用开源团结起来所有人，重开一把新游戏，重新定义规则。

9，故事茶，其实就是三无茶。要是东西真好，咋没成品牌？
相信品牌的力量，消费者要相信，做生意更要相信。
过去大家开一家酒店，总是想叫自己的品牌，自己去做营销，不愿意加盟，总觉得是给人家分钱，但是就是不反过来想，人家是给你导流，现在再看看呢？酒店不加盟，基本就是死路一条，而且这玩意会有信息不对称，做七天连锁赚钱的，也会开如家，也会开汉庭，他们的工作就是全省四处找房子，天天研究。
品牌最大的好处是，降低科普成本。
价格透明。

10，体制内的人，没有护照了，过去，为什么他们能有意识送孩子出国留学？
看县城就行了，局长家，县长家，甚至派出所所长家的孩子，都在国外读书，根源是什么？他们先出去过，去看过世界。
现在，为什么他们不送孩子出去了？
一是他们没有机会出去了，二是考虑到仕途影响，也不敢了。三是也没钱了。
过去我以为他们恨日本，恨美国，只是调侃或是表面上做做文章，深入交流才发现，他们竟然真是这么想的，我当时就在想，村里老百姓这么想就罢了，你们竟然也这么想。
他们看过世界，就会带着我们拥抱世界。
相反，他们看不到世界。
就会带着我们闭门造车！
