
# 分享两个实用有效的知识点

虽然现在是互联网时代，但是信息差还是存在的，这里分享两个实用有效的知识点。

## 知识点 1，异地捞人

比如有亲朋好友被异地警察抓捕，怎么异地捞人？最简单最实用的办法就是，也不要乱找关系，直接去当地，找到当地专门做刑案最大的律所，给他签捞人协议，只要当地律所敢收这个钱，多半就有靠谱的渠道捞人。这是最简单，最靠谱的渠道。对比一个失败案例。

我有一个亲戚因为经济纠纷被异地警方抓捕，出了事之后，这个亲戚的妹妹从美国赶回来营救，妹妹很有钱，说不惜一切代价也要把哥哥救出来，于是听人介绍花重金（上百万）从首都请了某大律师捞人。首都名气很大的律师到了小地方完全不好使，反而逼迫当地要办成铁案，本来一个不大的经济纠纷最后的结果是被判了 4 年半。

事后亲戚告诉我，如果是在当地找律师，可能花个几十万人就出来了，而且根本就不会有牢狱之灾。再来一个。

缅甸捞人。

现在境外做黑产的其实与国内有千丝万缕的联系，比较著名的就是福建安溪（出铁观音茶很出名的）就有很多在境外做黑产。如果真有亲戚不幸被卖到缅甸这类地区。

提供一个靠谱的渠道，跑到安溪一个律所一个律所去问，就是要缅甸捞人，大概就是几十万左右，只要当地律所敢收这个钱，只要不是太复杂，2 周左右人就能回来。

## 知识点 2，癌症治疗

目前国内很多癌症患者因为求生欲望，往往会不惜一切代价去国外治疗，这个去国外花费极高，几十万是基本价，上百万也很正常，如果人真治好了还好说，人没治好就是鸡飞蛋打。推荐一个很经济靠谱的渠道。

全世界最顶级治疗癌症的医院是梅奥。梅奥在全球也与多家医院签署了联盟关系，中国与梅奥有联盟关系的医院有 3 家。

分别是杭州邵逸夫医院、西安高新医院、西安国际医学中心医院。病人去这几家医院，住院之后就可以向主管医生提出需求——要求外国专家（就是梅奥的专家）对病情进行联合会诊，这个专家的会诊费可以自己出，因为有联盟关系，梅奥专家会诊的费用大概就几千元。

这是成本最小的方式，如果梅奥专家说没办法了，就别出国折腾了，概率是鸡飞蛋打。如果梅奥的专家说有办法，就要梅奥专家拿出治疗方案，就在国内联盟关系的医院治疗，效果是一样的。

唯一可能有点区别的就是药品，国外有些癌症药品国内没有批准上市，这个也有办法，跑到海南，博鳌有一个医疗先行试验区，这个医疗先行试验区可以搞到国内没有批准上市的大部分国外的药物。

这个模式性价比远远高于花大价钱去国外治疗。
