
# 米联储见闻250923——旄头已落胡天空（俄罗斯侵入爱沙尼亚领空，试图激怒北约？）

普京自从跟中美领导人面对面交底之后，彻底放飞自己，在欧洲空域数次擦边，北约成员国爱沙尼亚向北约告状，9月19日，三架俄罗斯军机同日入侵其领空长达12分钟，这是一次史无前例及明目张胆的入侵。但俄罗斯国防部否认指控，表示俄军战机仅飞越波罗的海中立海域。这3架战机经独立检查证实，此次飞行严格遵守国际空域规则，没有侵犯其他国家边界。并强调战机从俄罗斯西北部卡累利阿地区起飞，经过波罗的海中立海域，距离爱沙尼亚海岸外的韦恩德卢岛超过3公里。爱沙尼亚告状说，俄罗斯今年已四次侵犯爱沙尼亚领空，今天的侵犯更是史无前例的明目张胆。北约则表示当时已即时派战机拦截，形容者是俄罗斯另一
次鲁莽行为及北约应对能力的证明；川总则称，假如消息属实，将会是一大麻烦。上周，波兰告状，两架俄罗斯战斗机侵犯了波罗的海Petrobaltic钻井平台的安全区。之前是罗马尼亚召见俄罗斯驻当地大使利帕耶夫，就无人机闯入领空一事提出抗议。俄罗斯则称，没有能证明飞机国籍的客观证据，罗马尼亚方面的抗议被驳回，理由是其纯属臆想且毫无根据。

一次是偶然，频频发生必有鬼，爱沙尼亚已正式请求北约就俄罗斯侵犯其领空一事启动北约第四条，进行集体磋商。北约第4条为当一个或多个成员认为其领土完整、政治独立或安全受到威胁，各缔约国将共同协商，此条也被视为援引第5条集体防御原则的第一步。第5条规定若北约成员国遭受武装袭击，其他成员都将视此为针对所有成员国的袭击，并将采取必要行动协助。自1949年北约成立以来，第4条仅被引用8次，但这是过去两周内第2次被引用。俄罗斯似乎在频频刺激北约，想要拉北约下场的意思，现在的俄罗斯在乌克兰战场已经完全没有压力，加上跟中美元首都达成了平衡和默契俄，欧洲几乎没有任何可以跟俄罗斯博弈的本钱，川普上任之初，各方原本预期俄乌战争很快能结束，没想到经历数次美俄通话、美乌白宫骂战、美国一度中断对乌军援、美国持续对俄祭出最后通牒、美俄与美欧两场峰会后，不只俄乌战火没有停止，欧洲安全还升起新狼烟。

事实上，川普似乎有意缓和欧俄紧张，总是以事件可能是误会为由，淡化俄罗斯有意为之的氛围，希望息事宁人；但俄罗斯似乎没有给川总打圆场的意思，波兰更是不依不饶，要求华盛顿采取行动，以展示对华沙的支持；北约也在同日宣布启动东部哨兵行动，目标是加强北约东翼的防御，范围从北部的波罗的海国家一直延伸到南部的罗马尼亚和保加利亚，丹麦、法国、英国、德国等盟国也已承诺参与行动。本来就因俄乌谈判陷入紧张的欧俄关系，现在又是一片风声鹤唳，东欧情况也恰如2022年战争爆发前的俄乌边界，充满俄罗斯准备动手的猜疑与对峙。事实上，从更宏观 的视角来看，打从俄乌战争爆发起，后冷战欧洲的 安全秩序就已经崩溃，不论是围绕乌克兰战后安全保障的欧俄分歧，又或是俄罗斯无人机入侵乌克兰，欧洲人对俄罗斯的恐惧是深入骨髓的血脉基因，所以现在的欧洲各国在美国不能给予足够安全保障的情况下都是草木皆兵，而俄罗斯也乐于用这种方式不停的去刺激欧洲人尤其是波罗的海附近的东欧国家，毕竟这些国家曾经都是俄罗斯的一部分，三十年前倒反天罡，弃主而去，背靠美国耀武扬威，如今美国对欧洲的影响力日落西山，俄罗斯有王者归来的意图，欧猪们怎么不可能瑟瑟发抖。

冷战结束后，欧安组织是维护秩序的重要框架，其成员并不仅限于西欧或欧盟，还包括巴尔干国家、俄罗斯、南高加索、中亚国家，以及美国、加拿大两个北美国家。欧洲各国也普遍将冷战结束
时签署的《新欧洲巴黎宪章》视为安全秩序基础，乐观地预判欧洲对抗时代已经结束，未来将是民主、和平和统一的新时期，部分国家曾经基于地缘政治中的泛欧洲逻辑，冷战后的合作型欧洲安全秩序应该纳入俄罗斯。

以德国为例，从早 期柏林的视角出发，欧洲安全秩序的组成，应该包括欧俄共同参与的区域安全机制；与此同时，德国 也可以利用经济与社会政策工具，来促使俄罗斯向 西方价值观靠拢。

冷战时期，联邦德国就是美苏阵
营对抗的前沿，并以自己的特殊位置与接触苏联的新东方政策，成为东西之间的调解者，一直延续到德国统一。冷战后欧洲的多次冲突，德国也以欧盟大国的身份，发挥调解作用，尽量避免欧俄的全面性冲突，包括2014年克里米亚危机、以及持续长
达8年的顿巴斯内战。尤其是在默克尔时代，东德出生的默克尔在俄罗斯与德国之间搭建了非常稳固的关系和桥梁，甚至开通了两条北溪管道，为德国的工业化提供了充分的能源保障，但是败家莫过
于政客，当年默克尔卸任，普京和领导都是依依不舍，这是历史上欧洲大国与中俄关系最好的阶段，除了美国不高兴，中俄德都很满意，默克尔之后，中俄与欧洲的关系迅速恶化，最后直接导致兵戎相
见，爆发俄乌冲突。

之所以会变成这样，源于欧洲不只德国一个大
国，还存在美国这个域外霸权。冷战结束以来。欧洲内部就存在两种安全秩序愿景：一是延续过去的两极结构，由美国继续主导欧洲安全秩序；二是加强欧洲防务建设，打造欧洲成为与美平等的欧洲安全事务协商伙伴。早期，德国就是后一观点的强力
拥护者，主张在欧安组织内加强泛欧洲安全合作，希望俄罗斯平等参与欧安秩序塑造；但这种观点，受到有离岸干预传统的美英反对，并主张北约机制仍应是欧洲安全秩序的主导。正因如此，放眼冷战
后数十年的欧洲，欧安组织在区域安全的功能被不
断弱化，再加上俄罗斯经济一度衰弱，毕竟苏联解体的冲击过大，导致欧洲仍处于一种修正后的两极
秩序中。美国主导的北约仍在欧洲安全秩序发挥核
心作用，即便德国等欧洲国家试图通过相互交织机
制来构建错综复杂的欧洲安全秩序，北约却依旧牢
牢占据主导地位，导致诸如欧安组织等欧洲本土机
制，最终沦为辅助性平台，或只剩论坛作用。可以说，从一开始将俄罗斯至于欧洲安全架构之外，就
埋下了隐患，如此巨物在欧洲之畔却不能建立牢固
关系，爆发战争和冲突，只是时间问题。

德国之前主张建立泛欧洲安全秩序，并支持俄
罗斯参与，同时德国也是北约东扩的首轮推动者，
这也主要出自东扩本身可为德国争取更多联盟资
源、重塑前华约国家之间政经关系的考量；但之后
的北约继续向东南欧、波罗的海与巴尔干国家扩
张，就不再完全契合德国的地缘政治与安全利益。在德国看来，北约首轮东扩已经改善了欧洲安全，
继续东扩势必无法兼顾俄罗斯的地缘政治空间与
安全利益，只会导致欧洲安全边际效益递减。换句
话说，德国知道只有处理好欧俄关系，才能通过跨
大西洋安全机制稳住俄罗斯，维持安全稳定的欧洲
安全秩序。德国明显在北约第 二轮与第三轮东扩
中，持续发出质疑声，从科尔到 默克尔，德国明
显承继了新东方政策战略思路，试 图利用全球化
经济机遇，促使俄罗斯在政治上欧洲 化，2014年
克里米亚危机后，波罗的海国家与东 欧盟友不断
批评德俄特殊关系，一直到俄乌战争爆 发前的
2021年，俄罗斯仍是德国最重要的煤炭、 石油
与天然气供应国，占据德国35%的石油进口、
50%的煤炭进口与55%的天然气进口。有别于美
英坚持的大西洋主义、法国主张欧洲独立防务的欧
洲主义，德国更倾向在两种主义间取得平衡，并也
因此成为克里米亚危机前的欧洲安全秩序支柱。但
随着克里米亚危机爆发，德国平衡欧洲的信誉就已经
已经破产，在俄罗斯看来，德俄关系已经不足以抵销
北约东扩的压力，在其他欧洲国家看来，德国为了
自己经济利益出卖了欧洲，经历了8年顿巴斯内战
后，俄罗斯在2022年悍然开打全面战争，就是要
以一己之力重塑欧洲安全格局。因为德国曾经是俄
欧之间的协调者，所以当全面战争爆发，受影响最
大的除了俄乌就是德国，可以说德国在俄乌冲突中
的损失不亚于两个参战国。结果就是，俄罗斯如今
在欧洲内部已经没有沟通渠道，需要借助美国来传
话 ，简直是天下最大的笑话。

德国虽然具有沟通欧俄的传统，但对俄交好的
原始动机，并不视友俄本身、而是要藉此维持欧洲
安全，现在的德国必然转向以实力求和平的抗俄姿
态。在朔尔茨时代，德国对乌克兰的支持远超欧洲
其他国家，说明德国比谁都希望俄乌冲突尽快结
束，朔尔茨曾发表时代转折(Zeitenwende) 演讲，
要求德国推进安全与外交政策转型，并提出五项行
动任务：第一，向乌克兰提供武器；第二，德国与
欧盟一起加强对俄制裁；第三，德国加强对北约联
盟防务贡献并增加在北约东翼驻军；第四，设立
1000亿欧元特别基金提升联邦国防军防御力，并
将德国国防预算提升至北约要求标准；第五，保留
通过外交手段解决冲突的空间。2023年，德国发
表二战后首份《国家安全战略》,斥责俄罗斯对乌
军事行动是对欧洲安全秩序的背叛，德国视俄罗斯
为在可预见未来内，欧洲与大西洋地区的最大安全
威胁。不仅如此，德国还突破战后70 多年来不向
冲突地区运送武器的原则，成为继美国后对乌军援
最多的国家。

2025年新政府上台，德国仍视俄罗斯为对欧
安全最直接与最大威胁；美国有意减少对欧安全保
障的承诺后，德国反转过去角色，开始致力于说服
美国支持欧盟援乌抗俄，并承诺提升国防支出、增
强欧洲防御能力。上次川总接见欧洲驻门生，只有
德国总理默茨试图想要给川总一点压力对抗俄罗
斯，其他国家大气都不敢出。现在的德国不仅对援
助乌克兰金牛座远程导弹持开放态度，更承诺继续
向乌克兰提供数十亿美元军事援助和远端武器采
购支持，更在近日的俄罗斯无人机入侵波兰事件
上，扮演要求美国正视威胁的强悍角色。

历史就是一场轮回，地缘决定了很多时候，人
的想法会变，但国家的关系不会变，不管是欧洲的
德国，还是亚洲的日本，都不得不面对一个难以改
变的宿命，在庞然大物旁的艰难生存，日德在大国
眼皮底下讨生活的宿命也造就了日德两个国家特
有的品性，以至于让两个国家总有点挥之不去的悲
剧色彩。
