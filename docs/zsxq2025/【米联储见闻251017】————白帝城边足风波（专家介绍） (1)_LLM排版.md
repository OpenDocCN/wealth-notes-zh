
# 米联储见闻251017——白帝城边足风波（专家介绍）

## 专家介绍

你好，我是米糕。昨天的会议写了一半，把专家部分写了，剩下四个企业家部分今天继续写完。企业家分别有四位，央国企代表李洪凤，江鑫，民企代表周宇翔和叶国富，分别来自电气，产业投资，工业软件和文创零售。

李洪凤是中国电气装备集团党委书记、董事长，中国电气是国务院国资委监管的国有重要骨干企业，中国西电、许继电气、平高电气、宝光股份、西高院、宏盛华源、保变电气等7家上市公司，拥有输变电装备技术全国重点实验室等国家级创新平台20个、国家级质量检测中心8个、院士工作站1个和博士后科研工作站6个。承担着国内输配电行业标准归口管理工作，以及国际电工委员会(IEC)等多个技术委员会(分技术委员会)的国内归口工作，是电气装备领域的国家队，电力俨然已经成为这一轮新质生产力发展的重要基础设施和战略资源，也是对美博弈的重要核心基建底气之一。

中国电气装备与哈工大形成了战略性的合作协议，在新型电力系统装备建设和能源安全方面还有很大的想象空间。中国未来必然也必须要出口电力，从而让中国成为能源出口大国，这是中国走向星辰大海的关键领域。实现全方位自主和技术领先是定动作。

江鑫是现任合肥产投集团党委书记、董事长，这家公司成立于2015年3月，是合肥市国资委所属三大平台公司之一，集定位于业投融资和创新推进的国有资本投资公司，注册资本175.7亿元，国内主体信用评级AAA级，国际信用评级BBB级。拥有全资企业21家，控股企业2家，参股企业20余家。截至2024年底，集团资产总额突破1200亿元。这家来自内陆城市合肥的产投公司之所以今天坐上庙堂，只有一个原因，就是它是专门成立来推动中科大的成果转化的，目前核聚变的best工程就在合肥，这也是国家重大的战略科研项目，是这个公司的成立只有一个目的，就是搞钱帮中科大有价值的科研落地转化，是国家队版本的红杉资本。从这个两个发言可以看出，央国企现在都是跟大学研究机构贴身抱团发展，科研与应用两条腿走路，这也是近十年，中国科研成果快速转化的重要原因所在。

民营企业方面，这次参会的两家企业到底颇有些意外，其中周宇翔是黑湖科技创始人兼首席执行官，这家公司名气并不大，周宇翔是90后，是美国达特茅斯学院最高荣誉毕业生，2016年在上海创办黑湖科技，专注于制造业数智化系统开发。他2023年入选《财富》中国40位40岁以下商界精英榜，2025年获评2024福布斯中国科创人物。作为一家上海科创公司，参加李总的会议就不奇怪了，毕竟是在李总执政期间成长起来的科创企业。所以周总的发言主要是表扬上海的创业环境，对科创企业的服务态度。10月11日，周刚刚参加由上海市发展改革委员会指导、解放日报社主办的——新征程新作 为：上海民企的信心与动力——“民企发展正当时”座谈会。估计两场会议的发言稿都没怎么调整。这个企业主要服务于中小企业的管理系统，在上海服务近1000家客户，全国覆盖32000家工厂，涵盖食品、汽车、家电、航空等领域，如为蜜雪冰城打造柔性生产管理系统，实现消费、制造与科技融合，属于技术服务型企业。选他来参会，还有一个原因，就是这家公司经历了赛道调整，从2022年前依赖于美资投资到美资撤资后引入国内资本，再到现在转战越南、印尼，在当地工厂推广黑湖系统，引入中国柔性供应链模式。这个过程是国家希望让民企可以看到学到的，毕竟中美剧烈博弈是大概率事件，而以过往中美经济关系的紧密程度，必然会大量中小企业在经营中遭遇这个过程，虽然并非每个企业都能闯出来活下去，但是对国家来说，这也是必然经历之痛，只能让杀出血路的人给大家作榜样了。

叶国富的出现确实有些意外，因为在经营领域叶国富说实在的名声一般般，企业也都是以山寨思路起价，但是随着消费降级，shein这样的低价品牌在欧美大杀四方，对于名创优品这种掏用户零花钱买无用之物的企业，国家也是非常鼓励的，作为零售业的代表，名创优品虽然未必口碑好，但生命力是很顽强的，几乎是全国遍地开花。更难得是，名创优品是土生土长的广州本地品牌，对广州来说，这些年能拿得出的本土企业越来越少了，这次名创优品成总理座上宾，算是给广州一点面子，十年前的名创优品就诞生在广州一个地下室车库里。2015年，名创优品启动全球化战略正式进军海外市场，截至2025年6月30日，名创优品已累计进入全球112个国家和地区，集团全球门店数超过7900家，遍布纽约、洛杉矶、巴黎、伦敦、杜拜、悉尼、新加坡等全球知名城市核心商圈。2020年正式登陆纽交所，2022年正式在香港交易所上市，今年9月，名创优品旗下潮玩品牌TOP TOY在港交所递交招股书，摩根大通、瑞银、中信证券为其联席保荐人。虽然名创优品不如popmart名气大，但说到产业规模，其实远超popmart。

每次一二把手开经济形势专家和企业家座谈会，什么人参加，讨论什么话题都会引起各方关注，毕竟这些会都不是随便聊聊，都是针对当下的问题来破题，显然，中美博弈，控制风险，刺激内需，勇闯海外，依然是国家层面经济生活的主要方向。

中国马上要开四中全会，接下来就是韩国apec时间，这段时间很敏感，中美都把攻击力拉满了，10月14日中国开始向美国船舶收取特别港务费作为美方相同政策的回应，在此政策下，一艘由美国美森轮船(Matson)营运的货柜船进入宁波港后需要缴付627,943美元的特别港务费。据航运业资讯媒体《TradeWinds》报道，一艘由美国美森公司营运的货柜船将中美港口费之争的首位受害者。对于这些在海上漂了半个月的货船来说，这真是无妄之灾了。这艘悬挂美国国旗、载运量达2378标准箱的Manukai号10月13日停靠宁波港，14日卸货，被收取了627,943美元的特别港务费。据航海追踪数据提供商显示，该货船已离开宁波前往上海。目前该事件尚未获得宁波港或政府部门的正式确认，但由于船舶停泊时间、吨位及营运背景均与政策执行细则相符，业内普遍认为这一案例可能是船舶特别港务费政策生效后，中国港口对美国船舶收港务费的首例。

按照交通运输部发布的文件，从今年10月14日起，停靠中国港口的美国船舶，按每净吨收取400元。费用逐年上扬，从明年4月17日起，每净吨收费加至640元。2027年再升至880元，以及2028年的1120元，费用由船只挂靠港口所在地
的海事管理机构负责收取。中国这个操作目的很简单，美国对中国船只收取港务费的目的就是让船务公司不敢购买中国制造的船舶，从而打击中国造船业，而中国对等打击，就是让船务公司也不敢贸然改订单买美国船，因为中国才是头等制造业大国，一艘船造出来不是用几年就报废了，而是几十年，按这个趋势，在海上航行，装中国产品的概率更高，做中国生意的概率更高，所以，只要这个对等机制在，就让美国的港务费白收了，除了抬高美国进口产品的成本，继续推高通胀，对美国没有任何好处，也让美国希望通过港务费打击中国造船业的操作破产。据路透社报道，克拉克森研究公司在报告中表示，中国新的港口费用可能影响占全球运力15%的油轮。杰富瑞分析师Omar Nokta则估计，全球货轮中13%的原油油轮和11%的货柜船将受到影响。
