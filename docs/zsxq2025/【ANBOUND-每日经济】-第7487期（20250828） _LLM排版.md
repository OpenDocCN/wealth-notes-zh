
# 黄河三角洲几大急迫问题需重视

黄河是我国的母亲河，发源于青藏高原巴颜喀拉山北麓的约古宗列盆地，全长约为5464千米，是我国第二大长河，也是世界第五长河。自西向东，黄河流经青海、四川、甘肃、宁夏、内蒙古、陕西、山西、河南、山东九个省（自治区），呈一个巨大的“几”字型，最后在山东东营垦利汇入渤海，并形成了巨大的黄河三角洲。

现今黄河的入海通道，乃是1855年兰考铜瓦厢决口之后，夺大清河河道而注入渤海的，至今已历经170年，在自然和人为因素的共同作用下，三角洲范围内发生决口、改道50余次，造成海岸段淤积与侵蚀变化不断。

作为世界上含沙量最大的河流之一，黄河每年大量的泥沙进入渤海，在入海口沉积形成了黄河三角洲。以蒲城为顶点，古代三角洲面积约为7200平方公里；以宁海为顶点，近代三角洲总面积超过5000平方公里。

有数据显示，黄河入海口年径流量约300亿立方米，一般情况下，黄河每年能够携沙造陆3万亩左右。2022年，黄河入海口利津站实测输沙量为1.25亿吨。另有资料显示，黄河三角洲平均每年以2-3公里的速度向渤海推进，令其平均海岸线每年推进390米，新造大陆面积达30多平方千米，不断挤兑着本就不深的渤海。20世纪40年代初到本世纪10年代以来，渤海海域面积萎缩近5700平方公里。

值得一提的是，人为的围填海工程的建设，对渤海的面积产生影响。环渤海地区是我国围垦强度最大的区域之一，占全国总围垦面积的46%。在2000年之前，围填用地以养殖地、盐田和农用地的缓慢无序扩张为主。而2000年之后，环渤海区域主要填海增长的土地类型转变为建筑用地，这也让渤海的围填海规模逐渐走向了高强度的有序扩增，平均年萎缩速率达到了141平方公里，自然岸线长度也连年缩减。据研究人员统计，在1980年到2017年的38年间，环渤海区域围填海面积增加1988.5平方公里，占渤海总面积的近3%。若按照这个速度下去，一千多年后，渤海或将被填平。

不仅如此，黄河三角洲湿地结构也发生了较大的变化，自然湿地面积减少严重。2000年以前，自然湿地面积占总湿地面积比例均在90%以上，2000年以后，比例逐年下降。

## 优选信息

### 中国民营企业500强进一步集中在制造业

8月28日，全国工商联在辽宁沈阳发布“2025中国民营企业500强”榜单，京东、阿里巴巴、恒力位列前三。榜单显示，制造业企业占比达66.40%，较上年持续提升，凸显制造业在民营经济中的核心地位。制造业企业营业收入总额达29.63万亿元，占500强总营收的68.84%，同比增长7.66%；资产总额24.67万亿元，占比48.23%，增长7.92%；净利润0.96万亿元，占比53.21%。这一趋势印证了民营企业深耕实体经济、强化制造业根基的战略方向。创新驱动成为制造业企业高质量发展的关键引擎。500强制造业企业研发费用总额0.80万亿元，占500强研发总投入的71.31%，同比增长7.54%；研发经费投入强度达2.77%，研发人员总数115.17万人。

### 东北经济第一城被“卡”在哪？

2024年，作为东北经济第一城，大连GDP达9516.9亿元，与万亿目标仅差一步之遥；2025年上半年，大连GDP达到4647亿元，同比增长6.0%。然而，数据显示，2012年大连GDP就达到7000亿元，在此量级上，大连徘徊了十年之久。大连冲刺万亿大关的最大隐忧，正是产业结构本身。目前，大连仍以石化、装备制造、造船和电子信息为四大支柱产业。其中，石化作为第一大产业，占工业总产值比重超过40%。与此同时，大连新兴产业发展滞后。《大连市数字经济发展“十四五”规划》就曾指出，大连在基础软件、工业软件、人工智能、智能终端、信息通信设备等领域，缺乏自主创新能力与核心竞争力，尚未涌现具有广泛影响力与产业引领地位的企业。与此同时，软件产业与传统产业结合不紧密，没有抓住传统产业数字化转型的发展机遇，在工业互联网平台、智能制造等领域缺乏全国知名产品。除了产业，营商环境也是大连亟待补上的短板之一。尽管大连坐拥天然良港，地处环渤海与东北亚经济圈交汇处，大连仍受“投资难过山海关”印象的影响。

### 多地推进收储存量房并拓宽用途

今年以来，各地存量房收储工作已从试点转向深化。广西壮族自治区住建厅8月22日公布的数据显示，截至6月底，广西全区收购存量商品房2万套，拉动全区商品房销售面积增速9.35个百分点。广东省住建厅7月31日公布的数据显示，今年以来，广州、深圳、佛山、珠海等4个城市已完成4个项目共1377套的存量商品房用作保障性住房项目收购工作，收购金额14.43亿元。广东省住房政策研究中心首席研究员李宇嘉表示，“目前全国有60多个城市明确支持收购存量商品房，郑州、广州等城市已取得较好的成果。”58安居客研究院院长表示，“在价格方面，目前来看，多数城市以‘重置成本’为定价的基准，一般来说收储价会较市场价低30%至50%。”此外，针对市场结构性错配问题，大部分城市会优先选择配套成熟、户型合理的房源。例如广州明确优先收购90平方米以下整栋房源；深圳要求房源单套面积小于65平方米，以匹配保障性住房标准。不过，考虑到本息偿还、已收购房源未来配租配售的价格等因素，目前收购存量商品房仍面临资金平衡难度大等问题。对此，各地在收储的主体、用途、价格等方面也有所放宽。比如，在收储后用途从保障性住房拓展至安置房、学生或企业宿舍等。广西提出，支持各地在收购主体和用途等方面充分发挥自主权。李宇嘉建议，“在拓宽用途方面，已收购存量房源可以结合个人创业创新、文旅、夜经济、社区用房、养老托幼设施等新经济、新业态、新产业领域进行盘活。”

### 新能源汽车反内卷启动容易“刹车”难

彭博社称，中国试图遏制电动汽车行业激烈价格战的努力目前收效甚微。尽管6月官方呼吁避免“内卷式竞争”，但CAM汽车咨询数据显示，7月全国前20大汽车品牌中，有七家反而加大了折扣，其他品牌则维持原有力度或仅小幅收缩，整体促销水平仍高于去年同期。比亚迪作为国内销量最高的车企，其折扣幅度仅从6月的7.9%降至7.5%，变化不大，平均售价也从11.62万元降至11.48万元。吉利的平均售价也从10.57万元降至10.43万元。分析师指出，车企即使不直接下调标价，仍可通过免息贷款、赠送充电桩或车载服务等方式刺激消费。比亚迪即将在8月29日公布业绩，外界关注价格战对利润和交付目标的冲击，该公司计划到2025年交付550万辆，而今年前七个月已交付约249万辆。相比之下，奔驰、宝马和奥迪等欧洲高端品牌7月平均售价反而上升，部分原因是监管机构要求银行停止向经销商支付高额贷款佣金，遏制其通过变相让利加入价格战。国内厂商方面，零跑汽车称政策不会改变其定价策略，强调产品功能与性价比优势，并依托垂直整合模式持续降本；小鹏则认为政策主要针对以低价取量的竞争者，并未改变政府鼓励创新和质量导向的长期方向。业内人士普遍认为，政府要扭转市场格局仍需长期努力，短期内价格进一步下跌或明显回升都较困难。还有观点称，这和此前的去产能还是不一样的，没有哪家车企想成为第一个不参与价格战的那一方，所以价格战开启容易结束难。
