
# 特朗普会成为下一个希特勒吗？

提起希特勒，基本都是要单开一页的。自二战结束以来，有人说他坏，没人说他菜，人类个体能够达到的破坏力极限，基本就是希特勒了。所谓反派中派，坏蛋坏中坏，哪怕在小说中，都很难构写希特勒这么强大的反派，或者说，后世的反派或多或少都参考了希特勒的特征，属于不削没法玩的类型。

希特勒诞生在20世纪，是他个人的幸运，他没有诞生在21世纪，则是21世纪的幸运。咱们且不讨论希特勒的政治观点，仅以个人能力而言，希特勒的能力实际上是还没有完全释放出来的，因为他生在一个依靠广播和报纸传播信息的年代，他的大多数传播手段仅限于德国本土，还没有使用过互联网的放大器。

希特勒的演讲能力，基本是有史以来第一人，演讲渲染能力之强如同魔音灌耳，效果犹如女巫咒语一样。有人对希特勒的演讲进行过音频分析，其声音走向居然完美的符合了贝多芬的乐理，绝大多数听过希特勒演讲的人对其评价都是精神魔咒，其效果也很明显，他让一个奥地利屌丝在四十四岁成为了德国元首，可以让几百上千万人心甘情愿的为他去死。注意，是为了希特勒的目标，去死。

你可以说二战德军是受国家体制动员才上的前线，但哪怕党卫军，其人数也有百万之多，二战中愿意为了希特勒去死的人，人数绝对不下于百万。45年德国战败，从上至下多少人陪着希特勒死，这种忠诚可不是表面功夫吧？

而特朗普，他连几个愿意为他开枪的红脖子都找不到。无论是国会山事件，还是20年特朗普败选，以及接下来四年民主党对特朗普的追杀，这群看着呜呜渣渣，每次游行示威都要带枪的红脖子们，也没有一个敢站出来为特朗普开几枪。因为这群人跟特朗普的本质是一样的——志大智小，色厉胆薄，干大事而惜身，见小利则亡命。

当然，时代背景不同，我们也没法要求今日的人和20世纪的人在行为上一一对应。但有些事，还是可以比较一下的。

希特勒这辈子，说将自己的一切都献给了德国是没有问题的。他没有任何不良嗜好（在希特勒那个年代，还没有现代意义上“毒品”的概念，他注射毒品也是因为精神压力过大而不是为了享乐），青年从军冲锋陷阵，中年当了元首也从不享乐，每天沉迷于工作到几乎不睡觉（二战后期更加严重），当上元首后，他的地位和权力已经如日中天，他有很多种手段继续维护他的权力并保证自己不受报复，但他并没有到此为止，而是带领着德国争取更大的“生存空间”，直到最后的毁灭。

到了柏林战役，希特勒直接自杀，愿赌服输，没有用任何手段尝试着延长自己的生命，你可能觉得希特勒知道自己必死无疑，但以柏林战役的时刻，他想要延长几个月的生命还是做得到的，硬要赖着活着也不是不可能，不然看看萨达姆呢？

所以，历史对希特勒的评价，是疯子，是暴君，是坏人，但唯独不能说希特勒是小丑。他发自内心的相信他所说的一切，并且用尽他的全力去实现它，哪怕最后到死前也依然不忏悔，他邪，但他不假。

而特朗普，是21世纪传媒手段与直选制民主结合到一起所产生的奇妙怪胎。在美国历史上，我们看到过很多总统，不能说每个都优秀，但至少每个都“有人君相”，是体面人，像特朗普这样的疯子，确实是开天辟地头一回。

但是，如果把特朗普的出现归结于偶然，那就大错特错了，因为我们可以很明显的发现，21世纪的“特朗普”越来越多了。巴西的伯索纳罗，萨尔瓦多的奥特斯，阿根廷的米莱，越来越多的语出惊人，政策极端的表演型政客占据了高位，这显然是一种现象，而不是偶然的因素。

在前信息时代，信息爆炸没有达到今天这样的冗余程度，人们更多的还是关注自己的生活。人们可以通过感知自己的生活，来判断政治人物的好坏，通过工资高低，职位多少，物价变化来决定是否要换一个总统，所以，那个年代的政治选择相对而言理智很多，因为人们看重的是政绩和政治主张。

电视的到来，对直选制民主产生了巨大影响，典型案例就是1960年肯尼迪赢尼克松，在这场竞选中，一场电视辩论居然成为了两人的胜负手，尼克松因为在电视辩论中表现出了疲态，最终让更多的选民选择了肯尼迪，这其实是一个标志性事件，标志着现代传媒开始逐渐颠覆竞选逻辑了。

从那之后，我们可以看到美国总统的形象越来越好，如果没有彩色电视的话，里根选上总统的难度可能要高出几个数量级。竞选逐渐变成了政治人物的形象竞争，这种趋势一直持续到21世纪，直到智能手机发明，社交网络诞生了。

2010年，iPhone4发布，2016年，特朗普就选上了总统。特朗普，从政履历是空白，从军履历是空白，不是律师，不是议员，没有领导过社会组织，但就是这样一个人，居然当上了美国总统。

原因是，特朗普捕捉了互联网时代竞选总统的最重要秘诀——有活儿。特朗普以一个传播专家的洞察力，意识到了21世纪选举游戏中最重要的技巧——有活，要吸引流量。

有人分析过特朗普的传播策略，他之所以敢说出“注射消毒水”之类的智障言论，其实根本不怕别人批判，他根本不在乎别人批评他，因为黑红也是红，批评的流量也是流量，只要流量笼罩在他身上，就会挤压别人的流量，这就是21世纪的竞选技巧。

那么，什么人能够一直吸引流量呢？只有小丑可以永远吸引流量，因为他们可以毫无心理负担的整出一个又一个新活来，而任何体面人都做不到这一点。

所以，特朗普再次上台的短短百日里，全世界就已经被他的活整得眼花缭乱。分析特朗普的政治动机，将他行为合理化为“大棋”的一部分是极其愚蠢的事情，因为那就像分析小丑的表演背后有什么深意一样。

历史会证明。特朗普的一切行为，都只不过不过是政治小丑的表演，他所追求的，也不过是一个小丑所追求的一切——名利，财富，掌声，鲜花，唯独没有政治抱负。

希特勒统治德国的八年后，德国统治了从大西洋到伏尔加河的几乎整个欧洲，特朗普也将统治美国八年，到时候我们拭目以待。

其实，说了这么多，做个最简单的假设，就可以比较出希特勒和特朗普的根本区别了。

如果，现在上帝给了一个选择题——让德国成为世界第一强国，让日耳曼人永远统治地球，但条件是希特勒必须立刻饮弹自杀，你猜他会怎么选？这个答案应该没什么争议，希特勒会毫不犹豫的打死自己。

同样，如果上帝也给了特朗普一个选择——让美国great again，让美国的霸权主义江山永固，但条件也是特朗普要立刻自杀，你猜他会怎么选？我认为他会立刻宣布这是Fake news并对上帝加收125%的关税。

所以，哪怕再过一千年，希特勒都会是人类历史上无法遗忘的恐怖邪神，而特朗普，他能做到的，只不过是作为政治小丑时代的开篇人物被历史记下，仅此而已。
