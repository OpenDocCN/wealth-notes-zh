
# 学习笔记，2025.8.22

做投资就是准备打仗，打仗的核心有二，一是准备，要提前七八年去挖战壕、备粮草，等牛市，资金要在风平浪静的日子提前铺设进去。二是预期，打仗就有输有赢，胜败乃兵家常事，那么就要做预期管理，赢了怎么办？输了怎么办？赢了要考虑大赢怎么办，小赢怎么办？输了要考虑大输怎么办，小输怎么办？还有一种情况是僵持，那要再考虑僵持怎么办？还有一种可能，行军路上遇到了洪水，全军覆没了，那么也要考虑好，黑天鹅或黑犀牛怎么办？量化交易牛逼的地方就在这里。我们人类呢？只是凭感觉进或出。而量化交易呢？是一套完整的交易逻辑，无论出现什么棋局它都有对应的预案，赢的时候吃个痛快，输的时候呢？及时止损。

人一退休，真是秒变老头，一方面是过去上班时会染发，退休后不染了，一方面是精气神没了，我认识的那几个，在位置上的时候，算是气宇轩昂系列，腰板很直，如今？全蔫了。看来，人家说的真对，权力是最好的春药。不在那个位置上了，药劲就下去了。

我很勤奋，天全勤，谁能做到？有人会问，你累不？我一点都不累，因为我很享受这个过程，其实意识到这一点，多大年龄都不晚。抖音上的那个兰姐，走在她后面就是岁的小姑娘，其实她退休了，她退休后学习营养学、健身学，风雨无阻自己练，现在 多人主动找她当教练，没办法，她在家开了个私人健身房，那真是口碑介绍，其实她什么都不用说，身材一展示就可以了，我们所有人见了她都会喊老师，她跟所有的健身教练都不同，因为她本身是高级别退下来的，能量有、素质有、知识有……

我认为核心有两点，一是继续开放，让外国人不断走进来，也会试点开放国际互联网，同时也鼓励国人走出去。二是让有结果的人安心，相信这里是安全的，无论生活水平还是社会治安，我们都是遥遥领先，为什么这些有结果的人还跑呢？因为他们觉得对他们这个群体不友好，他们怕结果归零。所以，未来肯定会让规则越来越明晰，什么是红线，你只要不过红线就是安全的。

一定要去大城市，因为地方越小，周围人群对你的消耗越大，他们对你的关注，对你的热情，对你的非议，都会左右你的情绪，例如生活在村里，情绪几乎全被外人掌握。而在上海呢？没人关注你，没人议论你。别人也允许你做你自己，你爱咋着咋着。不被关注就是自由。

要给自己种心锚，节，是穷人过的。有个投票，你被选中了春晚的演出，但是腊月二十九，你得到了一个消息，父亲病危，要求你回去看最后一眼，请问，你是选择继续演出还是回家？的都选择了回家。那些年，我经常在思考一个问题，大年三十，这是最重要的日子，这些演员以及现场的观众，他们为什么不在家里呢？为什么要出门？这两年有些变化，十年前，大年三十那天，县城的马路上，一个人也没有，都在团圆。慢慢的才明白，原来，大年三十不一定非必须都在家里，可以有 ，也可以有 。

我觉得人一定要不断的自我疗愈，不要总是反复去想一些错事，然后折磨自己。我同桌前年或大前年借了我 万元，差不多就是腊月二十七八，很着急，他生意一直都做的不错，过去还有私人司机，同学聚会的 位，这两年也没联系我，当时说的用个十天八天。我都慢慢的放下了，不断的提醒自己，下次，咱不了，赚钱这么难。

之前，我跟孩子们讲过一个很重要陋习，一定要不要学，就是视金钱如粪土，编制第一，金钱第二。咱不要。咱要反过来，谁有钱，咱认为谁优秀。钱是什么？是一个维度。两维空间是平面世界，很多问题是无解的，例如如何在不穿过其他物体的情况下到达封闭空间的内部。若是升一个维度呢？那很简单，空降进去就是了。两维世界里很多无解的问题，若是站在三维空间看，就觉得 ，增加一个维度就是增加了一个外挂。

换操作系统是残忍的，但是是有必要的，为什么等于我们从小到大学的都是错误的。人的操作系统具有无限可能，例如，咱爹咱娘没了，若是切切喂鸟，咱愿意不？肯定不愿意。但是，倘若有一套逻辑是这么一个闭环。人死以后，需要神鸟把人带去天堂。那么，你愿意不？肯定愿意。这就是天葬嘛？！县城里有些领导干部，甚至大老板，身上是臭的，为什么臭？一周或两周才洗一次澡。这是他们从小养成的习惯，觉得澡不能洗的太频繁。他们级别够了，眼界够了，财富也够了，但是操作系统没有升级，从而身上很臭，昨天我还看了一个热门话题，中年老公臭死人。口臭，脚臭，身上臭。这类老公，多是凤凰男，他们使用的是农村操作系统……
