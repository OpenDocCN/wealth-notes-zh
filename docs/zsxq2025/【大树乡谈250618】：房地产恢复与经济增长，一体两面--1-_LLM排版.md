
# 房地产恢复与经济增长，一体两面

今天数据很多，但结论很简单。

13 日国常会研究房地产，主题是听取构建房地产发展新模式和推进好 房子建设有关情况汇报。提到三点：立足长远的房地产转型，以“好房子”作为未来核心方向，以及摸底调查优化政策。

虽然表述上更加积极，相比去年经济工作会议和今年工作报告“持续用力推动房地产市场止跌回稳” 的表态，加深为“更大力度推动房地产市场止跌回稳” ,但不能因此理解为国家要推出更大力度的增量政策支持，实际上，国家对房地产的支持性政策，基本已经应出尽出，接下来进入政策的存量优化阶段。

在小镇看来，这次会议的态度就很明确，房地产问题不是短期能够解决的，属于中长期的底层调整，当前阶段对房地产的政策支持已经基本到顶，除非发生远超预期的严重冲击，导致房地产出现加速下行。否则，短期内能够拿出来支持房地产回稳的实质政策只剩一项，那就是摸底强调的“对全国房地产已供土地和在建项目进行摸底，进一步优化现有政策”。

这就意味着，未来一段时间，房地产的“止跌”还要持续，就算出现短期的好转，也并不意味着开始“回稳”。

近期房地产一系列数据就证明了这一点。

售价方面：原本4月统计数据显示，70个大中城市的商品住宅销售价格出现环比持平或略降的现象，同比降幅也出现持续收窄，但注意仅仅是下行速度变换，并未扭转。而到了5月，70城新房价格环比跌幅创7个月以来最大、二手房跌幅创8个月以来最大。前五个月法拍房住宅成交均价也降到每平方米9083元，同比下降7.1%。

投资和贷款方面：5月房地产投资同比下降12%，创今年最大跌幅，下降速度也在加深，带动前五个月投资同比下降10.7%；今年1至5月，居民部门新增贷款累计5724亿元，占全部信贷比重仅为5.4%，创2009年以来的最低水平，还不到2021年峰值的两成。

居民更倾向于保守，呈现非常强烈的多储蓄、少负债的倾向，这也体现为住户存款定存率持续上升。2025年前五个月人民币存款总共增加14.73万亿元，而住户存款增加8.3万亿元，占比达56%，更需要注意的是住户存款定存率已经突破74%，近年来保持近2个百分点的年均增速，如此增长迅猛的居民存款，出现在定存利率持续下行的背景下，5月一年期定存利率已经跌破1%。

今年以及今后还会持续压低存款利率，但很可能无法缓解居民存款的定期化，因为存款不仅仅是出于获利，更取决于对未来宏观经济的期望，越是定存利率下行，企业尤其居民会更倾向于延长定存期限从而获取更高定存利率。关键还是取决于宏观经济形势。

还有一些数据证明，房地产“止跌”还需要时间。

尤其是库存，房地产库存有多重指标，小镇目前主要看最影响短期房价走势的狭义库存，也就是现房库存，即已竣工的待售新建房。今年初，一些地方出现了狭义库存下降，尤其是郑州，2024年底狭义库存减少130万平方米，这得益于郑州严控土地供给增量的管控政策，全国多地也出现了狭义消化周期的下降，比如徐州、深圳、苏州、昆明等地消化周期下降到18个月以内。

但需要注意，随着保交房等政策推进，已完工的住宅面积开始增长，于是今年前五个月，再次出现狭义库存缓慢增长，而房价持续下降，潜在购房者倾向于持币观望，这也导致全国范围狭义消化周期的上升。

房地产市场的分化也更加明显。在全国范围消化周期增长的同时，根据克而瑞数据，今年初重点50城的狭义库存同比下降11%，较2024年9月最高点下降约2305万平方米。重点城市内部也在分化。比如北京这类城市，二手房已经占据多数；而多数城市，去化艰难的主要是郊区项目，比如成都，年初狭义库存2240.67万平方米，主城区仅为425万平方米，反而是2023年以来的最低值，去化周期不足5个月。

待售角度也不乐观。虽然自2025年以来，待售面积出现一定下降，但是要看到，这种下降是在创纪录的高水平基础上的相对下降，从年初超过8亿平方米，下降到5月末的7.74亿平方米。

不过也有些好消息。根据中国最大的房地产信息综合服务商克而瑞的评估，5月典型二手房平均议价空间为15.4%，仍然呈持续下降趋势，较4月的15.5%环比下降0.1个百分点，这从侧面证明，房价继续下行的空间正在缩小。

其中苏州、南京、西安、武汉等持续7个月低于10%，证明挂牌价中的“水分”已经相对很小，而且持续收缩，房市交易变得更加理性。

施工面积也在快速下降。前五个月，住宅施工面积43.5亿平方米，下降9.6%；住宅开工面积1.7亿平方米，下降22.8%；住宅竣工面积1.3亿平方米，下降17.6%。

所以现在的情况，就是要想办法控制增量，完成存量的消化。这也导致未来房地产显而易见的分化，按照健康消化周期6至12个月，那么广大经济发展相对滞后、人口增长缓慢甚至流失的城市、地区，周期要长得多。

这有利有弊。弊主要是广大持有这类消化周期长地区房产的业主，而有利的一面，则是从全国经济增长来看，目前地区发展差异巨大，经济增长主要还是依赖经济发达地区，也就是近几年国家强调的“经济大省”,大城布的效率更高，新兴未来产业也基本集中在发达地区，在房价、经济下行的情况下，包括人口在内各种资源，会进一步向发达地区聚集，大城市的房地产更快回稳，有利于经济的恢复。

归根到底，房地产什么时候“止跌”,还是要看宏观经济恢复情况，而房地产本身就是宏观经济恢复的关键难点，也是经济恢复的动力。

这一轮房地产下行，和价格下行同步。

价格指数方面，PPI连续25个月为负，CPI连续26个月在0%左右徘徊。而GDP平减指数，已经连续8个季度同比为负，创历史最高纪录，已经超过了受亚洲金融危机影响的1998年至1999年的七个季度。与过往不同的是，这次主要是受房地产供需变化、下行的冲击，由于房子在中国居民财富和社会经济中的地位，也导致居民消费倾向有所下降。

当然，还有一个不同在于，这次消费数据的下降，背后关键原因是饱和供给带来的。尤其是准备替代房地产带动经济增长的新兴产业，陷入价格战、内卷之中，代表先进生产力的新兴产业，价格下跌明显超过平均。典型是汽车，5月的汽车数据很有意思，乘联会和统计局口径偏离很大，乘联会广义乘用车销量同比增长13.9%，而统计局社零销售数据同比增长仅1.1%，主要是因为乘用车价格普遍下降。

分析判断的基础数据就是上述内容，不过多着墨分析过程了，直接给小镇的分析结论：

目前国家在促进经济增长和未来发展方向上，态度很坚定。虽然已经提出“投资于人”，但政策惯性仍然是倾向于投资，而且投资倾向更重了。

相比短期的经济下行压力，国家更在意的是决定未来的科技创新和产业升级，只要短期经济压力不突破一定底线，那就不急于增大刺激。

而要稳定底线，就要避免经济的过度波动，就要管控好风险，这就体现在危机出清节奏放缓。最典型、最重要的就是房地产危机的处理，按照一般规律，如果纯粹按照市场经济规律，房地产危机出清，一般3年内足以完成，2008年次贷引发的美国房地产危机，如此猛烈也不过就是2年多，所谓出清就是快速下跌，跌到位了，自然就重新建立平衡，然后随着经济的恢复，进入新的上升周期。

但如果这样的话，对经济的冲击太大，尤其会严重影响到科技创新和产业升级的战略方向，会出现严重的不确定性，对于正处于关键节点的中国，显然不利。

所以国家选择是放缓房地产出清的节奏，表现在对房地产的政策刺激上比较保守。跌得太猛了，就出台政策刺激一下，房地产缓慢波动了，那就慢下来继续观察，让房地产保持在一定的波动范围内，从而争取时间。国家就可以把更多资源投入到认为更重要、更紧迫的领域。

带来的负面影响就是，由于房地产迟迟没有跌到底，而是变成缓慢的阴跌，就导致各方面投资更加保守、居民倾向于多储蓄少消费，这也就导致了GDP平减指数等迟迟无法恢复，也就出现了更长的通缩周期。

这本身就是两害相权取其轻。

所以在各方面的政策，也就呈现观望的特征，更强调稳和慢。就是要把不确定性压到最低。

在这样的逻辑下，就不要指望国家会针对房地产、价格萎缩等投入更大的增量刺激政策，反而在看到宏观数据满足底线要求的情况下，会更加珍惜政策资源，选择暂缓扩大刺激力度。

所以，要想理解国家政策，就需要重新理解国家对宏观数据和重点行业的态度变化。比如在GDP上，已经不是过去追求越高越好，而是只要看着能够完成全年目标，那就足够了，而前5个月的数据显示，上半年增速已经可以实现5%以上，今年可预见范围内也没有新的重大冲击，全年增长已经基本可以稳定在4.9%以上，也符合“5%左右”的目标。

什么时候房地产真正“止跌回稳”,呈现持续向好趋势，届时经济增长也就正式进入新阶段了。
