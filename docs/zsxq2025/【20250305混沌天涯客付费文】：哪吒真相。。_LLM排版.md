
# 哪吒真相

一个时代有一个时代的语言。我对文字比较敏感，比如下面这些话语，其实传递的是一种精神。

"中国人不吃这一套"，这句话大家应该没忘记，中文很凝练，说全了是"中国人只吃自己这一套"。更进一步，是中国人只吃自己的粮食。

中国人口众多，记忆里没吃饱的年月并不久远，只吃自己的粮食能不能吃饱，需要付出巨大努力。于是就有下面的第二句话，这只是前半句，后半句更进一步："饭碗里要装自己的粮食"。

这是个伟大目标，实际尚未做到。虽然三大主粮自给率在90%以上，但这个时代，人们已经不怎么吃主粮了。鸡鸭鱼肉蛋奶，其饲料来源，大豆占主要，而大豆的自给率低于20%。也就是说，吃饱没问题，但要吃好，同志仍需努力。

努力是一个漫长的过程，一年一年，一步一步。漫漫长路要不懈怠、不动摇，尤其是对那种认为美国的粮食又饱满又便宜干脆全进口不再自己种的错误想法，要时时刻刻予以清除。

所以，在这个过程中，信心很重要。

中国人，你要自信。这既是一句话，也是一本书，这本书畅销十余年势头不减，是因为它卡中了时代的语言。唯有自信，方能挺起胸脯，直起腰杆，甩开膀子，大干一场。

这就引出了第四句话——干就完了。说这句话的是一位山东姑娘，女舵手，在接受采访时，记者问她面对敌舰掀起的惊涛骇浪怕不怕？她闪着坚定的目光回答：我怕，我怕打仗不用我……干就完了。

"干就完了"，多么豪迈，多么热血。敢同恶鬼争高下，不向霸王让寸分。后面也是一句话，十余年前说这句话的那个人，说完后就完了。所以这变成了一句不太吉利的话。

幸好中华语言博大精深，可以换成另一句："我命由我不由天"。

明白哪吒为什么火了吗？知道中国电影票房前三名是怎么炼成的了？

## 因为它们贯彻的是一种时代语言：

第一名，《哪吒之魔童闹海》，我命由我不由天。

第二名，《长津湖》，敢同恶鬼争高下，不向霸王让寸分。

第三名，《战狼2》，干就完了。

但在"魔童闹海"出世之前，排在"战狼"后面当第三名的，是一部《你好，李焕英》，关于母爱的，稀里哗啦感动。

这就构成了一个因果链：干是行动，敢行动是因为自信，支撑自信的是饭碗在手吃饱吃好。而作为高级动物之人类，最初的吃饱吃好是源自母亲。

于是我终于找到了我一直不自信的原因，小时候没怎么喝过母亲的奶。

同理，我也发现了如哪吒一类不信命不退出支棱着脑袋始终闹腾无休止的人物到底是什么因为什么？小时候吃的奶水很足（哪吒娘胎里待了三年），足够自信；奶水断掉之后却被母亲抛弃，愤怒扎心。

## 别人怀胎十个月，他在里面待三年半，享受的是多于三倍的呵护。但出生后的哪吒赶上了坏运气，需要割肉还母，剔骨还父，跟父母恩断义绝。这个过程，是人都会无比痛苦，何况是曾被母亲三倍呵护过的哪吒。

他不服，要闹，他命由他不由天，根源在此。

哪吒是神话，怀胎三年是传说。但神都是从人中间想象出来的，神性与人性相通。

从哪吒的故事可以了解到，如果小时候备受宠爱，住着大house，好几个服务员关照，是父亲的掌中宝，母亲的心头肉，处处高人一等，悠哉乐哉漫步人生路。突然间，幼小的心灵被重重一击，父亲有事，母亲绝情，所有的温暖变成了冷冰冰。这样的遭遇，会塑造出哪吒闹海的脾性。

一个典型的例子，是前国民老公王思聪。作为家中独子，1988年出生的他，生下来就是万千宠爱。父亲在外忙，或许交道不多，但母亲林宁以及姥爷姥姥，对他是宝贝极了。童年悠哉乐哉的思聪哥，十岁的时候遭遇当头一棒。那一年王健林摊上了事，得罪了大连的那个霸王，不仅俱乐部被卖走，生意也受到挤压。家中独苗王思聪，必须要背井离乡踏上留学路。先去新加坡，又去英国，都是没有人情味的地方。他作为一个东北土老板的儿子，那感觉像是从热炕一下子进了冰窖。

所以，长大成人的王思聪，一是速速回国，再也不去新加坡，更不留恋度过了中学和大学时光的英国。二是养成了爱闹腾的性格。

一直闹一直闹，直到去年，父子俩双重锤击之下，老爸将万达控制权让出，儿子得到了定居日本体验东京热的机会。

王思聪已经是过去式了，今后他出现在热搜上的可能性微乎其微。但另一位如王思聪般的闹腾男，却仍在进行。

没错，大家肯定猜到了，他就是太平洋另一岸的川普。

川普是个富二代，出生的时候父亲已是纽约房地产大亨。他上面有两个姐姐一个哥哥，诞生之初，宠爱之极。但几年后他有了个弟弟，宠爱开始向弟弟转移。而且已经有三个儿子的老川普，决定分头培养，送小川普从军。

13岁，川普被父母送入了纽约军事学院，在那里他受到了残酷的训练。那场景，我们在《阿甘正传》里见识过，却并没有《阿甘正传》里刻画的温馨。从娇滴滴的富二代，一下落入美式军校的丛林，身形高大的川普，又被选作去打橄榄球。哇，美式橄榄球我是站在场边亲眼见识过的，那种暴力对抗程度，当妈妈的真忍心。

这就是川普的童年和少年。童年优渥，少年残酷；童年乖宝贝，少年扛着铁枪喊号子，抱着球在泥地里滚。根据我们刚才总结的规律，这种环境塑造了他爱闹腾的性格，生命不息，闹腾不止。

上任才一个月，川普就做了远超拜登一年能做的事。闹腾出的惊涛骇浪，已经在重构这个世界的秩序。

关键是，这才一个月，接下来还有四年，四年之后，没准还会有……
