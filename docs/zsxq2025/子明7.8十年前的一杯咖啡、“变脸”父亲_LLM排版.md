
# 子明7.8十年前的一杯咖啡、“变脸”父亲

命运给予的馈赠，早已在暗中标好了价格，亲情也不例外。
2025年6月15日，名校毕业的高材生李新野在父亲节当天发表了题为《我的父亲李松坚》的文章，公开控诉其父亲明园集团董事长李松坚攀附权贵、感情不断等问题，引起了社会的广泛讨论。

类似李新野这般含着“金钥匙”出生的公子哥，站出来公然指责为其提供优渥人生的父辈，案例并不多见。
即便是马斯克和王健林这样的首富也未能幸免。
豪门显贵，父子成仇，这究竟是亲情的缺位？伦理的悖逆？亦或是不可逃避的财富“诅咒”。

最近马部长火气不小啊，看什么都不顺眼。可能和他家闺女……嗷不，应该是儿子有点关系吧。
他那些照片已经完全看不出男性特征了，马首富的人生确实内容丰富。
首富的孩子们，心理疾病往往都很重，参考王思聪。
这是长期活在父辈的阴影下导致的吗？
首富们一般创业早期没空管孩子，而在孩子的眼中，父亲的变化太大了，甚至言行变得非常不一致。小时候父辈就是个处长，长大了变成了地产商，这个过程中，小朋友是很难接受老爹言行的变化的。

一代目的骤然崛起孩子们都是懵的，根本不知道自己还要往什么方向努力，以及自己面对的到底是一个怎样的父亲。
我觉得还有一种负面压力——妈妈这个孩子的另一个至亲，在丈夫事业骤升的过程中承受的不安和不适，也会投射到孩子身上。
我查了一下，马斯克这个儿子是04年出生，而马斯克刚好04年投资特斯拉，08年孩子的父母就离婚了，同年马斯克刚好成为了特斯拉CEO。
这一时期事业的巨大成功，侧面反映了马斯克在孩子童年时期大概率是缺位的。

创业一代的大儿子往往都是看着老爹被资本异化为一个资本，内心能不扭曲吗？
小儿子就无所谓了，他们看到老爹的第一眼，他就已经是一个资本家了，自然也不需要再去适应和选择。比如很多国民党高级将领的孩子们的反而信仰共产主义，是同样的道理。
他们小时候看老爹，是一个有理想的三民主义战士，可等老爹们成为了统治阶级后，孩子们就会觉得：“这还是个人吗？”
现在有点理解为什么有的二代是自由派了。
“投共”的现象，只存在于国民党一代的孩子们，同样的，“投资”行为只常见于red二代，三代就基本不这样了。
有点好奇red三代们会是什么意识形态。

就是正常心态了，不会有集中的扭曲，更不会出现red二代那种逢×必反。
哈哈，感觉很多red二代有一种“脱粉回踩”般的深深恨意。或许red三代们也不信他们爷爷那一套，顶多就是无感，不会那么恨。
对于孩子们来说信不信那些宏观的东西不重要，造成他们反叛的根本原因是心理学上，父辈形象的形象的轰塌。
比如你的父亲是站长，小时候你觉得他是你的偶像，为理想奋斗的三民主义战士，结果去了天津几年，那玉面金佛，那斯蒂庞克..你就会觉得KMT简直就是烂透了。
而如果你是站长的二儿子，你小时候就坐着斯蒂庞克上学，你就会觉得KMT是正确的，而且你发现你老爹相比于其他政客，人品素质那是相当不错的。

道理就是这么简单……跟感情一样，需要在正确的时间遇到正确的人。
我想到了方孟敖、方孟韦。
对，方孟敖很难接受正直的老爹给四大家族搞钱，而方孟韦能看清楚，老爹是这群烂人里最不烂的那个，参考的坐标系不一样。

《我的父亲李松坚》上热搜了，和明公昨天说的几乎一样。
嗯，李新野就是经历了创一代上升期的，看老爹跟川剧变脸似的。
经历社会毒打之后就会了解父辈。

你觉得这种人会遭遇多少社会毒打吗？方孟敖就是把天都捅漏了，有中统局长替他摆平。所以，你猜国内的顶级开发商背后是不是也站着顶级家族神通广大的妖怪背后咋可能没有神仙主人。
王思聪能到处怼人，是因为他有个能替老板们干脏活的爹。他老子王健林，人前一副精英做派，而在大佬面前，只能跟龟孙子一样。
回家卸下防备后，又跟各位在家宅的时候差不多。
那些逆子九成都会走过“鄙视他爹—理解他爹—剩下一成拧巴着废了。成为他爹”的历程，
从概率的角度来看，大部分人的基因质量都不如他首富级别的老子，有同样的手牌也玩不过他老子。
当他们发现这一点的时候，往往为时已晚。首富是十亿分之一的概率淘汰出来的，而他们的儿子，不过只是一个基因重组的产物。
在公子们成长的过程中，父辈们正经历着商海的混战、政界的沉浮，那些终登绝顶的人必然也变了一副面孔。而目睹这一切的公子们尚处年幼，他们不明白其中的缘由，只知道父亲面目全非，他们埋藏着这份不安、压抑，甚至仇恨。
终于，当他们利用父辈的资源走进社会时，便毫不犹豫地向着来时的路“开枪”了。

十年前的一杯咖啡，如今又重新飘香。
2025年7月7日，淘宝闪购与饿了么联合宣布战绩：淘宝闪购日订单数超8000万，日活跃用户超过2亿。
而就在前一天，美团餐饮外卖的日订单量更是突破了1亿单。
受外卖商战补贴影响，港股茶饮持续拉升，7月8日，茶百道、奈雪、古茗、蜜雪冰城等茶饮品牌均有较大涨幅。
电商巨头的商战再升级，奶茶咖啡价格受补贴影响大幅优惠，茶饮股也随之全面高开。
如此大规模的补贴是否预示着新一轮经济周期的启动？这一轮和过往又有哪些共通之处？大洋彼岸，美国的“五万亿”补贴又会走向何方？

这个周末淘宝闪购直接补贴500亿，冲击美团。
嗯，外卖商战已经开打了。我很好奇最后结果是利好美团阿里这种平台，还是利好蜜雪冰城这类品牌。
目前来看，美团、阿里和京东三家对茶饮行业的补贴力度都不小，补贴占比分别是20%、50%和50%。
仗打完才能谈利好，正在打，就难免会有回调，只有打完了才知道边界在哪。
眼下应该是利好我，早上点了一杯咖啡才两块钱二毛钱。
这也太便宜了。
回想上一个十年经济周期，也是开始于各类补贴，当时“稳定币”的玩法，就是支付宝搞的。
新周期要开始了吗？那这一轮的风口在哪？机器人有可能吗？

上一个十年的契机是移动网络爆发，所以补贴的是滴滴这些。那现在再次搞外卖是什么意思啊？
上一次的补贴，本质是新兴资本集团寻求铸币权，旧集团为了解决经济危机而做出的妥协。
货币本质上要跟货物产生某种关联。譬如，我们最常见的石油美元，上一个时代也可以称为“奶茶马币”，再上一个时代可以称之为“游戏马币”。
现在的外卖补贴只不过是大家还没想明白用什么作为货币的新锚，暂时延续了旧时代搞奶茶咖啡店罢了。

这就是十年前的今天。
我用AI数据模型推演了一下平台外卖补贴奶茶业之后的业绩影响。初步来看本轮商战中茶百道的利润表有望收益最大，古茗自身实力稳健，我个人觉得这两家股票值得关注。
商家也要让利补贴，未必能得到多大好处，而且现在有点透支消费，消费者奶茶喝多了就会腻了……
这点可以参考当年的摩拜0F0大战，短期内肯定能够刺激商家估值的。

美国现在也在做自己的“五万亿”，是不是都差不多？
本质上都是向未来借钱，钟镁走了两条不同的路子。兔子走的是有特色的资本主义道路，通过资本市场向未来借钱，美国走的是有社会主义特色的道路，通过国家大规模的举债。
有时候想想也挺有意思的，我们相互走上了对方的道路。
但本质上差不多，都是向未来借钱。
以时间换空间。
向资本家借钱就需要给资本家对应的政治地位，所以兔子马在UP；向市场举债，是向金融大佬借钱，就需要给他们对应的地位，所以美国马在DOWN，美国这样做前两年会很舒服，但是后面搞不好会很困难，参考兔子的21-24年。

我们当年搞四万亿的时候，澳大利亚的铁矿石跟着也能舒服，现在是美国要搞五万亿，兔子肯定会跟着爽。
而马斯克愤怒的点是，这五万亿没有给新兴产业。
外卖奶茶的补贴大战，拉开了新周期的帷幕，这是我们通过市场向未来借钱以刺激当下的经济的调节手段；美国则是通过国家举债向未来借钱，以为政令背书。
最大的资本主义国家和最大的社会主义国家，在这一轮又走上了彼此的道路。
