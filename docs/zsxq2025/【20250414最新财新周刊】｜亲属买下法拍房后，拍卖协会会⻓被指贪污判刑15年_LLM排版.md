
# 亲属买下法拍房后，拍卖协会会长被指贪污判刑15年

文|财新周刊孙良滋

七年前的2018年3月，53岁的马驰在天津市拍卖行业协会(下称“天津拍协”)换届选举中当选会长。次年他开始接触法院，参与司法拍卖资产处置。这个业务拓展，为他卷入刑事案件埋下伏笔。

2018年3月，马驰在天津市拍卖行业协会换届选举中当选会长。2020年，在协助天津市滨海新区法院处置非法集资案房产时，马驰的亲属参与拍卖，先后购入一套别墅和四套公寓。2023年5月马驰被监委调查，五个月后遭刑事立案。2024年2月，天津市河西区检察院指控马驰涉嫌贪污罪和挪用资金罪，当年12月31日，河西区法院作出一审判决，认定马驰非法占有国有财物共计1597万余元，构成贪污罪，且系数额特别巨大，判处有期徒刑12年，并处罚金200万元；挪用公司资金503万余元归个人使用，构成挪用资金罪，且系数额巨大，判处有期徒刑4年，决定执行有期徒刑15年，并处罚金200万元。

## 与处非办和法院合作

马驰是天津本地人，1965年出生，17岁参加工作，曾经曾在天津迎宾馆、中国国际旅行社天津分社工作。1992年，27岁的马驰下海经商，主要做进出口贸易。据其子马子扬介绍，2007年，马驰在家人建议下进入拍卖行业，同年成立天津市汇绅拍卖有限公司 (下称“汇绅拍卖”)。

2018年3月，天津拍协换届，马驰当选新一届会长。天津拍协官方网站介绍，其成立于2002年11月，是经天津市社团局批准登记设立，在业务主管部门指导下结成的行业性、非营利性社会组织。当上会长之后，马驰和司法机关的联系多了起来。2019年起，他参与天津市和平区法院的非法集资涉案财产处置。马子扬也是在2019年进入汇绅拍卖工作，据他回忆，当时拍卖行业已经不太景气，为扩大协会业务量，在得知滨海新区法院有较多非法集资案件的涉案财产需要处置以后，马驰和天津拍协工作人员一同接触了滨海新区法院。

2020年1月14日，滨海新区防范和处置非法集资领导小组办公室(下称“处非办”)召开了一次会议，参会人员来自滨海新区区委政法委、区法院、天津拍协等单位。会议明确，为加快涉案资产处置速度，加强涉案资产拍卖工作效率，由区处非办与天津拍协签订框架协议，具体项目由区法院执行局开展拍卖变现工作。

六天后，滨海新区处非办与天津拍协签订了一份协议书。双方约定，滨海新区处非办委托天津拍协为非法集资案件被扣押物品处置提供咨询，资产处置责任单位向天津拍协提供相关案件扣押物品的清单。协议同时明确，天津拍协向处非办推荐符合资质的拍卖公司拍卖相关资产，具体拍卖合同由资产处置责任单位签署，天津拍协则对拍卖公司实施监督等。协议有效期一年。

2022年5月，滨海新区处非办与天津拍协签署了补充协议书，明确天津拍协提供服务的非法集资案件为盛世富邦案、泰诺润发案和权释案等15起案件，双方合作至工作完成。两份协议签署时，天津拍协的联系人都是马驰。
