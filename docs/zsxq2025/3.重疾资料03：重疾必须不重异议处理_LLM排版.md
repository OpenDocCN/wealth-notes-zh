
# 重疾必须不重异议处理

## 为什么要买重疾险？

重疾带来的危害大，导致收入中断，支出不断。人这一生患重疾的几率达到72.18%，癌症从闻所未闻发展到见怪不怪。重疾没有替代品，需要一笔“专款专用”的钱，重疾险具备这一优势。医生只能挽救一个人的生理生命，却不能挽救一家人的经济生命（重疾险可以）。

以前在谈重疾险时，往往只关注医疗费用，而忽视康复。古人说：“三分治七分养”今天依然适用。养的钱没算在保额里。手术后变得虚弱了，心情不好，经济条件不如以前，更容易再次的癌症。收入损失的钱没算在保额里。

所以：重疾险至少需要100万才够。Ⅰ.良好的治疗费用约40万 Ⅱ.良好的康复费用约20-40万 Ⅲ.由于患病未来收入的损失约20-40万。

没有重疾险，人类就会缺乏面对未来的底气。每售出一份重疾险，就会少一个下跪的人。一个国家的文明程度，往往以弱者活得是否有尊严为标志。

大病面前你有两种选择：打开存折看余额、打开保单看保额。

保险是你必不可少的“保本”护照，谁对生老病死感兴趣？今天健康不一定等于以后一定健康。因为有病才有医院，有风险才有保险，风险不会以是否相信而转移。

风险发生有4种选择：家人、朋友、医保、保险。保险才是你明智的选择！

胃癌是因为我们的食物。肝癌是因为我们的水。肺癌是因为我们的空气。重疾险，不连累生你的人，不拖累你生的人。

现代医学那么发达，癌症都能治愈，很多人并不是因为病而死的，是因为没钱，看着心爱的家人倾家荡产的救他，他的自尊心难以忍受，而绝望的死。你无所谓，但你的亲人呢？你考虑过他们的感受吗？他们砸锅卖铁也要为你治病，你为什么不早投保呢？有保险做后盾，病人安心与病魔作斗争，家属也不会因高额药费穷困潦倒。

香港著名演员黄日华，比刘德华红，比梁朝伟帅，妻子血癌花掉750万。他说：“妻子的医药费全由保险公司支付，根本没经济上的压力，有一个好心态才是关键。”

我有房子，不需要保险。保险是一张安全网，是预防财务负债的一种有效手段。不要把所有的鸡蛋放在一个篮子里，多一个篮子少一份风险。房子变现能力差，要有人接盘才能出售，保险是“急用的现金”。你的房子在上海，请问在汶川怎么办？保险是：风险的转移——家庭保障，责任的体现——养老、教育，生命的价值——有计划的储蓄，对未来的准备——应急现金。

遗传因素只占10%，环境因素占60%。空气，水质，环境污染与遗传无关。人只要有呼吸就会有风险，意外与残疾与遗传没关系。今天你坐着笑着跟我说“对不起”，我认识是礼貌，明天你躺着哭着跟家人说“对不起”，那是无奈和绝望。万一碰到了怎么办？现在吃蔬菜有毒素，喝饮料有色素，吃荤菜有激素，吃什么心理都没数。这与遗传有关吗？

银行是晴天拿伞的地方，给你的是本利和；雨天拿伞的地方，给你的是保额。请问：你最需要伞？家庭的财务计划应分为两部分：树干：消费以追求利润为目的；树根：保障性投资，用保险规避风险。可能一阵风把树干刮断了，一把火把树干烧了，但只要树根保险解决“走得太早，活得太长，中途波折”。如此题，只有保险能解决。

人们总希望自己的财产放到安全的地方，但有没有想过，把自己身上的癌细胞也存在保险公司，那一定很安全。你说你爱孩子，要保护好孩子，但你没有足够的保障时，拿什么去保护你的孩子？孩子买保险是买家具，大人买保险是买房子，家具时一定要放在房子里的。没有房子，家具会坏。疾病和意外，无论你多么讨厌，它每天都在发生。保险，你买不买，它都会慢慢走进每个老百姓的家里。

买保险要具备三个条件：公司的实力，适合自己的产品，用心的代理人。现在连看个热闹都会有风险，我们有什么理由不买保险？请珍惜你可以买保险的机会，保险代理人看中的是你的健康，年轻和健康是你投保的最大资本。

现阶段重疾治疗重治疗，轻康复请问：康复费用，收入损失，社保保吗？社保与商保结合，才能完善。问自己三个问题：①如果生病了，谁可以给你30万？②如果发生意外，谁帮我还贷？帮我照顾家人？③谁能承担我生病时的收入损失？

市场不景气，将来付不起保费怎么办？如果公司宣布减薪20%，你要不要接受？买保险就像穿新鞋子，开始痛，过段时间就习惯了。（少买一件多余的衣服，少跑一次超市）风险与明天不知道哪个先到，如果发生在你还能付保费之前怎么办？开门七件事，第八件事是保险，当你发生风险的时候，保险后会转，帮助你付前面的七件事，真正做到了“急用的现金”。李总理说：“常说病来如山倒，我们就是要运用大病保险等多种制度，不让一个人患大病，全家都倒下。”

买保险是爱的承诺，付保费是对爱的分期付款。你爱人不需要爱吗？事实证明，死亡理赔款没有一个爱人是拒收的。当风险发生时，你的爱人抱着枕头哭，还是抱着保单哭。投保时一群人告诉你：“买保险没用。”出事时一群人质问你：“你咋不买保险！”

路虎很贵，开的人越来越多；夏利很便宜，已经被淘汰。诺基亚很实惠，已经倒闭；苹果手机很贵，用的人越来越多。股票、P2P理财、信托收益都比保险高，为什么国家只出台文件鼓励买保险？

总之：保险再贵也贵不过上不封顶的医药费，更贵不过人的生命。未来癌症不是病，没钱医治才是病，没有保险会要命。拒绝保险，就是拒绝平安、拒绝家人的幸福、拒绝财富、拒绝责任。
