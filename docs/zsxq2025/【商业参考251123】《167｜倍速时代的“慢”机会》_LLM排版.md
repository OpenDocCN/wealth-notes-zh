
# 倍速时代的“慢”机会

欢迎打开《蔡钰 ·商业参考 4》，我是蔡钰。

你知不知道前段时间有部爆款影视剧叫《许我耀眼》？它被称为“短剧类型的长剧”，不少人对它批评不断，但又欲罢不能。

为什么批评不断？原因是情节不合理，毫无常规故事该有的起承转合，就是把各种高能爽点密集拼接在一起。那又为什么欲罢不能呢？也是因为高能爽点一个接着一个，像辣条一样让人上头。

这让我想起，近几年有两个概念形成了一组“社会困境因果链”，非常反映现实，也指向了某种消费服务的创新机会——它们一个叫“倍速社会”，一个叫“时间贫困”：社会运转越加速，人就越没有时间；越没时间就越加速；人越加速，又越让别人的生活倍速转动，把别人也卷入了这个螺旋。

内卷的原意，本来的是无法突破上限的低水平竞争，但不少行业哪怕有新的技术和市场突破，也都觉得自己很卷，主要是因为同行们把彼此都拖入了倍速螺旋。

## 长剧的短剧化

这个变化，在消费市场体现最明显，其中最标志性的变化出现在影视消费行业：用户的耐心从前些年的看5分钟解说一部电影，快速缩减到了刷1分钟短剧。

2024年以来，影视剧市场上，“长剧短剧化”的新趋势就开始出现了。意思是，长篇影视剧主动吸纳短剧的情绪性、强节奏、脸谱化人物等特点，来适应观众们被短剧驯化了的观看习惯。

这种变化出现的标志点，是在2024年6月出现的一部古装偶像剧，名叫《墨雨云间》。这部特意请来短剧导演执导，主打长剧短拍。它开场短短十分钟，就让女主经历了若干反转，从被陷害、被解救、被迷晕、被活埋、再到逃生。

这部剧一上线就成了爆款，被年轻观众誉为“国产剧辣条”，形容它又重口味又上头。它不仅在国内各个社交平台和影视数据榜单拿下了十多个第一，还在中国香港、中国台湾、越南等地也登上了热搜榜首位。它上线5天广告收入就翻了3.5倍，找来谈合作的品牌超过40个。

长剧短拍，在当时的影视行业里被视为自降身份，《墨雨云间》也一度引发了行业群嘲。但它的制作人于正，夹枪带棒地回应说：“可笑的是还有很多人看不起短剧，其实这是时代的趋势——短剧精致化和长剧快节奏，浓缩叙事刻不容缓，真不知道那些吐槽的人在高贵些什么。”

后来发生了什么？到了2025年，越来越多的长影视剧，借走了这种长剧短拍的策略。今年那部《许我耀眼》，就是其中之一。

## 倍速消费的三种解释

好，回到需求端，今天观众们，为什么越来越需要倍速观看影视剧？

日本社会前些年也有类似现象。日本作家稻田丰史2022年写了本书就叫《倍速社会》，里面提出了三种解释，我觉得挺有道理：

第一种，观众获得影视平台的包月会员资格后，消费心态从“为某个作品付费”变成了“倍速才能看更多”。

第二种，观众的看剧心态从欣赏作品变成了消费内容。“欣赏”强调的是行为本身，通过接触、玩味，让自己沉浸在作品中。而“消费”带有功利性，图的的是获取谈资、直奔爽点。

第三种，影视剧的剧本也出现了说明书化的趋势。

什么意思呢？影视剧们越来越倾向于用台词表达一切。比如，主人公在雪地奔跑，明明已经在表情痛苦地大喘气了，但还是要亲自说出“呼吸好困难，冰冷的空气让我肺疼”这种台词。

为什么要这么做事无巨细？为了降低观众的理解门槛，不用他们再亲自理解意境了。
