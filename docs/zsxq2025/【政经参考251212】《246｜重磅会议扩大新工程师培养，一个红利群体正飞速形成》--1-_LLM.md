
# 重磅会议扩大新工程师培养，一个红利群体正飞速形成

欢迎来到《政经参考》，我是马江博。今天这个话题我很喜欢、也非常重要，涉及到我之前就提的，一个关键的就业红利新贵链条的飞速形成，而且这个链条我认为未来二十年，都将是最重要的红利之一。

我们先从一个特殊的高层会议说起。12月4日，召开了一场“卓越工程师培养改革座谈会”。一般认为，工程师培养听起来像是高校的教学安排，按常理，最多教育部部长出面就差不多多了。但让人意外的是，这个座谈会的规格很高，中央政治局常委、国务院副总理丁薛祥，亲自出席并讲话。而且部署节奏特别快，当天教育部还召开了“卓越工程师产教融合培养工作推进会”，教育部部长亲自布置任务。

这意味着什么？说明在决策层眼里，这件事早就不是“教育圈的小事”，而是一场关乎国家发展的战略工程。

## 什么是卓越工程师培养改革？

那么，到底什么是“卓越工程师培养改革”？我们先捋一下进程。2010年，教育部就发布了“卓越工程师教育培养计划”；后来在2017年，开启了“卓越计划2.0”，提出要布局建设现代产业学院和未来技术学院；后来的2021年，教育部公布了首批未来技术学院所在的12家高校名单。

这里提到的“未来技术学院”，我理解它瞄准的是10—15年的前沿性、革命性、颠覆性技术，目的是培养能够引领未来发展的技术创新领军人才。这是布局未来技术的，突出科研。

而工程师教育的关键加速，也发生在2021年。当年9月，中央召开了中央人才工作会议，最高层提出将卓越工程师作为四支战略人才力量之一，这个定位非常之高。

一年后的2022年8月，教育部和国务院国资委印发《关于支持部分高校和中央企业试点共建国家卓越工程师学院的通知道》，在10所工科特色高水平大学和8家央企，布局建设18个国家卓越工程师学院。大幕从此拉开。

不同于前面“未来技术学院”布局的是未来技术，这里提到的“卓越工程师学院”，则聚焦“国家急需”的当下，解决国家当前发展面临的重大工程技术难题。它的培养模式带有强烈的“任务导向”和“结果导向”，特别强调工程实践，要求培养的人才能加快在重点企业、重大项目中发挥作用。

“卓越工程师学院”背后说白了，就是三个字：新工科。

以重庆大学国家卓越工程师学院为例，这所学院说要，“围绕智能网联电动汽车、机器人创新应用、工业母机、半导体、新材料、智慧能源、生物医药及高端医疗设备等领域，以主动服务国家重大战略和区域经济发展需求为导向……探索具有中国特色的新工科教育和工程硕博士培养新范式。”

注意，除了“新工科”，还有一个关键词是“工程硕博士培养”，也就是国家卓越工程师学院，主要是招收硕博研究生。目前也有少量学校，比如重庆大学，在面向当年的理科本科新生，进行校内二次选拔。

为什么要强调“工程硕博”的培养呢？以前工科主要是学术型硕士和博士，培养模式往往跟企业需求严重脱节，而国家卓越工程师学院的“工程硕博”，则是一次重构。

我总结一下，所谓“卓越工程师培养改革”，就是从2022年开始，国家通过建设“国家卓越工程师学院”，推动工程人才培养从学术导向，转向实战导向的一次系统性变革。核心是聚焦“新工科”领域，通过校企共建，培养“工程硕博士”，让工程人才更好解决国家急需解决的问题。

## 改革到底新在哪里？

刚刚谈了“卓越工程师培养改革”的进程和内涵，接下来我们看看，改革到底新在哪里？

我觉得，主要有三大关键变化：

第一，打破学科围墙，综合交叉培养。教育部的数据显示，通过打破高校“学科化、院系制”的传统工科人才培养模式，采用校企理事会负责制，建设了50家国家卓越工程师学院。这些机构不是传统意义上的院系，不按机械、电子、材料这样的学科细分，而是围绕重大工程问题，把不同专业的老师、设备、课程全打通。

比如一个新能源汽车项目，可能同时需要电池材料、控制系统、人工智能算法的人才——学院就直接组队培养，让学生从入学起，就习惯跨学科协作。《中国教育报》的报道提到，浙江大学卓越工程师培养的“项目制”例子，面向航空制造、集成电路、智慧能源等关键领域，与企业一体化组建联合培养基地和中试研发基地，打破了以往单院系、单学科、单导师零散化培养的模式。

第二，校企深度绑定，解决实际问题。卓越工程师学院的学生，往往会有两个导师：一个是学校的教授，一个是企业的高级工程师。更重要的是，学生在校期间要尽量参与企业的真实工程项目。教育部12月10日的新闻发布会上提到，“实行工学交替培养，把学生入企实践作为必备环节，工程硕士入企不少于1年、工程博士不少于2年”。《中国教育报》2025年的报道说，各个国家卓越工程师学院，聘任校企导师2.3万名，其中企业骨干1.3万名、一线总师2000多名；而企业向国家卓越工程师学院提供了1万多个来自产业一线的研究课题；另外，校企协同建设200多门“高阶性、前沿性、交叉性、挑战度”的核心课程，联合建成了300多个工程师技术中心。

第三，学生毕业申请学位，可以不写论文。取而代之的，是工程硕博士可以以产品设计、重大装备等实践成果申请学位，说白了，让工程教育回归它的本质，就是培养能动手的工程技术人员。

这次座谈会上，国务院副总理丁薛祥强调，“要完善卓越工程师培养体系，推动机制化常态化培养。”什么意思？我理解就是这项工作不再只是在几十所学校试点，而是未来要在全国范围内铺开，变成工科教育的“标准配置”。未来会有更多“双一流”高校和龙头企业加入进来，向更多急需领域拓展。

## 不可忽视的背后影响

那么，更进一步，这背后到底意味着什么呢？

我们需要从更本质、更宏大的视角看这个问题。在国家卓越工程师学院启动的2022年，召开了党的二十大。其中有一个关键变化值得注意：之前的党代会报告里，教育通常放在“民生保障”部分讲，科技则归入“经济发展”板块。而在二十大报告里，教育、科技和人才被放在同一段来表述，而且被定义为“全面建设社会主义现代化国家的基础性、战略性支撑”。这一调整意义重大。

而另一个信号是，这次召开座谈会的国务院副总理丁薛祥，同时还兼任中央科技委员会主任，我理解这次国家是要把教育和科技的资源全面打通。

而这些改变背后的主要原因，是中美博弈加剧，科技成了国力竞争的核心战场，而中国更需要大量的科技和工程人才，这就是“国家战略人才力量”。

今年的“十五五”规划建议里，明确提到的“国家战略人才力量”，具体包括五类人：战略科学家、科技领军人才、卓越工程师、大国工匠、高技能人才。

我之前课程里提过，其中战略科学家是指定方向做规划，科技领军人才是带领团队做重大攻关，卓越工程师负责解决从设计到产品的工程难题，大国工匠负责关键制造，而高技能人才则负责最后的生产规模化。

这五类人才是从高到低、从顶尖科研到高级蓝领的一个完整的科技产业人才链条，也是未来一段时间内最大的就业红利链条。未来十年甚至二十年，想投身国家战略产业领域的同学，就是在这五类里找自己的位置了。

其中，卓越工程师将成为“最抢手”的群体，特别是在战略性新兴产业领域，工程技术人才将非常抢手。而这次的座谈会，其实也印证了我们的预判。座谈会上再次强调了最高领导人提出，要“加快建设规模宏大的卓越工程师队伍。”注意，“规模宏大”这个词很关键，说明国家需要一大批这样的人才，这是新时代真正的结构性机会。“卓越工程师”这条路，就是你通往新阶层的阶梯。

## 对普通人的影响和建议

那么，这事跟普通人到底有什么关系？我至少有三点要分享给你。

第一，未来的就业红利，正在向卓越工程师这样的“新工科”集中。

一方面，就业对口率高。目前的模式是，企业采取提前签约、免除试用期、折算工龄等方式，支持学生留企就业发展，首届2000多名工程硕士留在本领域重点企业就业的比例达到81%。

另一方面，能获得国家的更多资源倾斜。教育部提到，在人社部门支持下，校企共同把工程师素质能力标准纳入卓越工程师培养全过程，浙江、广东、黑龙江、江苏率先实现了人才培养与工程师职称评定的无缝衔接，1100多名工程硕士，毕业直接获得工程师职称证书。

不过我前面也说了，目前各个高水平大学的“国家卓越工程师学院”，主要面向硕博，只有少量校内本科新生的二次选拔。但一些特色的地方院校，也在直接的高考招生上尝试创新，比如2025年北京化工大学，新增卓越工程师学院，在6个工科领域，实施小班化、导师制、本硕贯通培养，总招生计划有210人，这也是一个新的选项。

另外，未来的研究生教育，会越来越强调解决问题的能力和交叉复合型人才，学生要让自己具备“T型能力结构”——既有专业深度，又有跨界广度。

第二，未来的应用型人才红利，将越来越区域化、城市化、产业集群化。

像卓越工程师学院这样的新型工科学院，校企联合培养越来越深，学院和当地产业、企业的“绑定”程度也越来越深。这意味着，一个学生读的学校有没有产业资源、所在城市有没有产业集群，比如合肥的新能源、深圳的电子信息、西安的航空航天，将直接决定他的实习机会和就业出口。

而未来“跨专业、跨区域找工作”的难度会越来越高——随着培养的精细化，企业显然更愿意招自己参与培养的“亲儿子”。所以，高考填志愿、考研择校，都要更前置地考虑目标学校的城市产业生态和就业链条。

第三，国际化能力，会成为新型工程师的“关键加分项”。

这次座谈会，丁薛祥特别提到，要提升工程教育国际化水平，推进工程师资格和工程教育标准多双边互认，更好服务企业“走出去”。

我认为显然决策层的目标很明确，不仅要培养自己的工程师，还要让全世界更多国家认可，甚至采用中国的培养标准。未来中国要主导全球产业链，既需要技术、产品这些“硬实力”，也需要教育标准、人才认证这些“软实力”来支撑。

中国企业出海已经是大势所趋，从高铁、光伏到电动车、5G，中国技术和标准正走向全球。一个有国际化能力的新型工程师，将是稀缺中的稀缺。
