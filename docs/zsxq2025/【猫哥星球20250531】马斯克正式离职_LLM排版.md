
# 马斯克正式离职

除了马斯克，其他内阁成员，多是不得罪人的好“好先生”，比如司法部长邦迪，说着要解密爱泼斯坦案，实则解密了个寂寞。国土安全部长诺姆，说着要驱逐大量非法移民，实则抓得还没进来的快。卢比奥更厉害，马斯克好不容易收拾掉了国家民主基金会和国际开发署，他却把前者给救了回来，恢复了运作，后者也想通过改组，恢复运作。得亏特朗普还没答应。商务部长鲁特尼克、交通部长达菲之流，则更不用说，只会拍马屁。马斯克的离去，意味着内阁里的实干派不多了。剩下一帮溜须拍马之徒，能成什么事？

上台前，马斯克以为联邦政府和他的公司一样，想裁谁就裁谁，就一句话的事。等上台才发现，根本裁不动，全世界的公务员都有编制身份保护，美国也不例外，美国有《彭德尔顿法案》，绝大部分公务员是裁不了的。少数能裁的，是国际开发署这样的二级乃至三级单位，但这些单位的上级，不会眼睁睁看着下属部门被裁。毕竟编制少了，明年的拨款就要减少，换谁都不乐意。所以，马斯克的裁员遭到了内阁的巨大反对，根本不用民主党出手。马斯克的离职，也说明了节流道路行不通，想解决财政危机只能靠“开源”。

短期内，“开源”手段有且仅有关税，正常年份，美国的关税不到一于亿。想要缓解财政赤字，关税收入得增加到一万亿美元左右。

客观地说，“降本增效”失败，不代表马斯克去白宫一趟完全没有收获。马斯克起码有两个收获，一是塞了一批人进入政府，包括但不限于人工智能顾问克里斯南、加密货币事务负贵人萨克斯等等。马斯克走了，这些人可没走，只要他们呆在政府里，马斯克就能保持影响力。能把一批人塞进政府，这是多少资本家梦寐以求的。二是多项合同。这几个月，从NASA获得的合同总金额，就不下10亿美元，特斯拉还从能源部和国务院，获得了两份总价8.65美元的合同。

这些加起来，已经比马斯克的大选总投入要高了。最最重要的是，能平安落地。前两个月，民主党闹得凶时，马斯克是真有生命危险，谁也不知道，哪天他就被打一枪。离开了政府，民主党大概率不会再打击他了。留得青山在，不怕没柴烧。那么问题来了，往后的马斯克会不会继续参与政治？在我看来，是必然的。原因无他，世界首富不参与政治，就是个笑话，世界首富根本没有中立的空间，你非要中立，两边都收拾你，只能选一边战队。只是参与的方式，会有所改变，简单来说，不再自己撸袖子上阵，改为资助候选人的方式，逐步培育势力。

类似的案例，在美国历史上发生过无数次，比如共和党的头号金主科赫家族，70年代初，也是亲自上阵，结构发现效果很差，毕竟政治不是谁都能玩的。而且金主直接上阵，往往显得扎眼。失败后的科赫家族，转而当起了金主，通过隐蔽资助的方式实现自身利益，效果出奇地好。未来的马斯克，极有可能采取类似的方式。唯一的问题是，培育势力需要时间，起码五年起步，形势会给马斯克这么多时间吗？我们拭目以待。
