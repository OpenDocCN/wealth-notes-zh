
# 始祖鸟的烟花教训

## 价值观高于人设

这场烟花活动的是非，可以用局部正确与全局正确来考量。我认为，这场烟花还"炸"出了两个硬道理，对中国公司，尤其是消费品公司来说，是非常及时的一课。

第一个教训：价值观这东西，比人设硬多了。品牌的人格魅力，得建立在真实稳定的价值观基础上。

大众的愤怒逻辑很简单：市场给到始祖鸟、给到蔡国强的高认可和高溢价，不是仅仅用来买他们遵纪守法的。始祖鸟背叛了自己的品牌人格，让受众感受到了错付和背刺。

人们对品牌要形成持续的情感认同，首先得对品牌奉行的价值观形成向往。你得在价值观上，展现自己高于目标人群的执着与纯粹。管理学上有句格言叫，“只有偏执狂才能生存”，意思也是，你得对观念和目标更执着，才能点燃和引领其他人。

所有致力于打造人格型品牌的公司，都是在跟市场缔结一种默契：为目标人群塑造值得向往的高我，对价值共识更坚定、更执着、更纯粹。

品牌们动不动喜欢提社群、社区，指的就是这类向往高我的目标人群。我们在《情绪价值 30 讲》里专门讲过耐克的品牌之道。它就是通过展现自己对体育精神的执着狂热，来团结运动爱好者们的。

类似的，在山姆超市眼中，椰子水只有泰国原装的，西梅只有智利进口的，所有包装袋不到万不得已不出现中文。于是会员们才相信，它对优渥中产人生的追求比自己较真、比自己狂热，“跟着它没错”。

始祖鸟，全球性的高端户外品牌，它的高溢价，也建立在对户外文化的专业和执着上。

户外圈有个著名的共识叫“无痕山林”，Leave No Trace，意思是，人们去探索自然时，要做到不留垃圾，不破坏环境，离开时就像没来过一样。

户外品牌对“无痕山林”原则的极致奉行，往往也会收到极致的正反馈。而在这种默契下，品牌在奉行价值观的行为上稍有滑坡，动摇的不光是自己的品牌形象，还有受众们的信念内核，错付和背刺感就是这么来的。这种感受的伤害力有多强？反正嬛嬛最后灭了四郎。

别说始祖鸟放烟花了，7 月份，山姆超市也经历了一次塌房事件。原因是，它上架了一批家常菜级别的大众品牌产品——好丽友、卫龙、溜溜梅、徐福记等。这让中产会员们觉得：你个浓眉大眼的鄙视链缔造者，竟然也背叛了革命。

你可能还记得当时互联网的热闹。不少网友说：“我花钱办卡，不是为了买楼下小卖部，就能买到的好丽友的。”还有人说：“我开会员就是为了买进口货，怎么国产货越来越多了？”这抱怨的都是，山姆没有守住中产与下沉生活之间的壁垒，背叛了它牵头的价值观。

所以啊，品牌要做人设，得先确定真实、稳定的价值观，老板跟公司得真把把它当信条，才能让组织用作行事标准时不变形，经受住市场越来越严苛的“生活参与”式考验。从这样的价值观里长出来的，就不再是凭空而立的人设，而是扎实的品牌人格。

## 全球一致性

再来看第二个教训。第二个教训也很要命：在互联网时代的全球化之路上，“一个品牌，两套说辞”的玩法失灵了，出海公司们必须把最底层的价值观在全球各个市场拉齐。

中国这些年就在做这件事：一边推“一带一路”，一边推核心价值观。核心之内坚守原则，核心之外客随主。

说回始祖鸟。这次烟花事件发生后，始祖鸟的加拿大总部跟中国团队的措辞有明显的差异。

在英文世界的道歉信里，始祖鸟说：“最近在青藏高原举办的烟花活动跟始祖鸟的价值观不符。我们听到了你的担忧，我们也感同身受。此次活动完全违背了我们对户外空间的承诺、我们的定位，以及我们希望员工和社群成为的样子。我们对此事深感失望，我们道歉，没得说。我们正在跟涉事的当地艺术家和中国团队直接沟通，并将改变我们的工作方式，以确保此类事件不再发生。我们期望所做的一切都体现我们的环保理念，对不符合环保理念的行为零容忍。我们已经开始努力减轻此次活动对环境和社会的影响。感谢社群对我们的监督。”

这番话，让海内外不少网友感觉傲慢，始祖鸟全球团队像是一个裁判，把责任甩给了中国团队。

而相比之下，始祖鸟的中文道歉信就显得诚恳很多，不但谈到了项目的初心与偏差，认可了公众的批评，给出了具体的补救行动方案，还留了邮箱。

两份道歉信为什么会有这么明显的态度差异？原因其实很简单：始祖鸟，是“全球品牌、分头运营”。

2019年，中国运动服饰集团安踏，收购了芬兰运动公司亚玛芬，间接把亚玛芬持有的始祖鸟收到了旗下。但从公开资料看，在全球市场上，安踏给亚玛芬保留了比较大的自治权，始祖鸟的设计研发与品牌战略职能，仍然放在加拿大总部。

而在中国市场上，安踏体系对始祖鸟业务链条的介入更深。比如这次烟花项目，是属于始祖鸟发起的一个名叫“向上致美”的系列营销活动，过去三年，这个系列都只在中国落地。

户外人群虽然也是泛运动人群，但坚持的价值观跟运动人群仍然有微妙不同。运动人群的价值观源自竞技体育，推崇强大、征服、赢；而户外人群的价值观却源自探索自然，强调的是敬畏与尊重。

擅长服务运动人群出身的中国团队，在拿捏“无痕山林”这类户外专属价值观时，未必能拿捏到位。

从这个角度，我们更容易解释始祖鸟英文道歉信的姿态。它的全球团队想说的是，“我们也不认同，但我们也没办法。”那句“改变工作方式”的承诺，大概率指的是，在未来，它的全球与中国团队要建立新的沟通机制，来实现品牌价值观的真正对齐，和对品牌行为的真正约束。

所以啊，这是始祖鸟的烟花教训留给中国企业的另一个警示：全球化不仅是资本和渠道的扩张，更是价值观的拉齐与文化势能的守护。

对正在布局品牌全球化的中国公司来说，非常有必要把不同市场的底层价值观拉通，并建立跨区域的沟通和约束机制，在无数市场和渠道当中，守住品牌的稳定灵魂。

## 总结

始祖鸟这场危机还没结束。短期看，品牌修复需要时间，中国市场的信任重建更需要真金白银的投入。但长期看，这可能会成为一个转折点——不只是对始祖鸟，对整个行业都是。

因为这场烟花教训反映的趋势是：在互联网时代，品牌的每一个动作都在全球同步直播，消费者对品牌价值观的要求越来越严苛，容错率越来越低。

所以我猜，接下来，会有更多品牌开始重新审视自己的价值观体系，特别是那些正在全球化的中国品牌。

所以请教你：如果你的公司正在全球化，你会为品牌选择什么样的底层价值观？如果你是消费者，你会原谅始祖鸟这次的“失误”吗？如果你来操盘始祖鸟，你认为，怎样设计危机处理方案，能最大限度地挽回事态和人心？期待你的思考。
