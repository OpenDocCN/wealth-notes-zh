
# 电影《南京照相馆》的开创性意义，在哪里？

250804 文/卢克文工作室嘉宾 云梦竹

今年暑期档，最大的黑马，莫过于申奥导演拍摄的《南京照相馆》，其在短短七天内取得了9亿票房的好成绩。对于一部故事片来讲，这个成绩可谓优秀。

相比于商业化的成功，该片的艺术造诣更令人叫绝。

该片的故事原型是衣廊附近的华东照相馆，照相馆学徒罗瑾在1938年收到了日本军官送来冲洗的200张相片胶卷，有大量关于南京大屠杀的内容。罗瑾认识到这是日本侵华兽行的铁证，随即冲洗了三十张以作保存。

随后罗瑾认为，其中一部分妇女被强奸的内容是对受害者的再次伤害，遂将其去掉，只留下了16张装订成册。

导演申奥也致敬了这一行为，以“不愿让历史上受害的女性再次受伤”的名义，删掉了首版中日军对中国女性的强奸镜头，转而采用了以更加隐晦的手法间接呈现的方式，比如片中空洞无神的妇女，疯癫绝望的妇女。

后来，罗瑾因生活所迫加入了汪伪政府的直属通讯队，将相册藏在驻地毗卢寺厕所内，突然有一天，相册不见了。

惊恐之下，罗瑾逃离了此地，相册也不翼而飞。本来，罗瑾以为该相册已经遗失，然而在1946年，审判南京大屠杀主犯之一谷寿夫的庭审中，该相册做为“京字第一号证据”重现天日。

原来，汪伪通信队的另外一名队员吴连凯，在茅厕中看到了这本相册，立刻认识到了这本相册的重要性，随即将其藏到佛像下，直到抗战胜利。吴连凯拿到相册当天，队长就开始进行搜查，明确表示有人私藏皇军相册。

我的推论是，汪伪通讯队的一名成员，在厕所无意间找到了该相册，看到内容后，惊悚之下扔到厕所旁边，这才被随后上厕所的吴连凯拿到。此人没有第一时间上缴或者销毁，但又害怕自己被追责，遂向汪伪政府上报。

至于这个人是谁，历史并没有给出答案，就这样，罗瑾与吴连凯，两个中国青年，在日伪高压统治下的南京，冒着生命的危险，一前一后完成了保存南京大屠杀血证的接力，为日后的正义审判提供了有力证据。

历史原型是小人物的故事，而导演也非常细节地从小人物的视角出发，去描述这场惊人的罪行，将南京大屠杀这一宏大历史事件，以一种具象化的视角展现在人们眼前，成功规避了过去南京大屠杀电影存在的种种问题，实现了艺术性和故事性的完美结合。

要想理解《南京照相馆》好在哪里，就需要知道，日本为了洗白南京大屠杀，都做过哪些“努力”。目前，日本学术界洗白南京大屠杀主要是三个方向：

否认大屠杀的种族灭绝性质

他们认为，南京大屠杀不过是像英国征服印度，美国征服印第安一样的战争灾难，属于古代社会的平常事件，而非一个民族对另一个民族的系统性种族灭绝，进而降低事情的严重性。

为此，日本延伸出了几种说法，一种是南京有士兵藏在平民中，所以日军才被迫杀人。

另外一种说法是，这只是日军底层军人的个人举动，与整个日军无关。但如果你去看日军底层军人的回忆录，他们又会说是上层的命令，自己只是被迫执行，就这么，南京大屠杀的历史责任，在各方争论中不断飘散。

之前的一些南京作品，总会安排一些人性未泯灭的日军在暴行后觉醒，这种作品设计是非常愚蠢的。他会让人误以为日军也是有良知的，他们仅仅是因为身处日本军队这种军国主义系统中，才会做出如此罪行。

这种安排，变相洗白了日军自身的罪恶。

而南京照相馆则没有，给我们揭示了 一种纯粹的恶。不需要去展现日军在看见屠杀惨剧后幡然醒悟，重拾人性。因为所有日军在南京大屠杀中都是有人性的，只不过，他们没有把中国人当成人来看待。

大屠杀的出现，不是对方人性泯灭，而是加害者将被害者非人化了。

这一点，在南京照相馆中，就得到了充分体现。这种非人化的出现，反过来证明了，南京大屠杀就是日本系统性的种族灭绝。

细节探究

否认种族灭绝之后，日本对细节伤亡数字进行深究，质疑是否有三十万人。我必须在这里提醒大家，只要和日本人争论这个细节，我们就输了。

在南京大屠杀中有这么一个故事，一名中国少女被日军轮奸，随后被西方大使馆解救，当时的西方认为，日本依旧有法律，就向日本驻南京的高层反馈，日本驻南京的高层假模假样的过来调查，询问那名中国妇女，是否记得桌子上台灯开没有，各位想想看，一名十几岁的少女被轮奸到语无伦次，如何能够分清台灯开没有，最后她说开了。

日军高层哈哈大笑，那个地方没有台灯，你在撒谎。

质疑南京屠杀人数，本质上就是日本重复这个把戏，通过细节的探究来推翻整个事实。

在过去的南京作品中，经常出现一些集中性的处决，乃至血海上漂浮的尸体，这些画面依旧在通过数量去强调日军的罪恶。这些导演希望用这种方式来展现日军的残暴，却无意间正中日本的下怀。

他们刻意的将中国拖入数字争论中，通过数字的争论，将一个个个体磨灭，成为历史悲剧的一个注脚。

《南京照相馆》却以一个个挣扎求生的个体，将人从数字中拖了出来，让我们知道三十万不是数字，如果日军只杀了29万9999，难道就不是罪行了？

这部作品最大的优点在于，他展现了活生生的人，在历史悲剧中的挣扎。而这些人，每一个人的死亡都是日军不可磨灭的罪行。

政治宣传说

在日本右翼和学术界的语境中，”南京大屠杀“被称为针对日本的政治宣传，是一种仇日教育。这个说法和国内某些鼓吹中日友好的人高度契合。

在过去拍摄的南京大屠杀电影中，他们最大的问题，就在于太像纪录片了，因为南京大屠杀是一个非常核心的历史事件，这个历史事件的重要性，改编就变得尤为困难，很容易拍成纪录片。

而一旦这些片子的纪录片属性太重，他们就滑入日本右翼的语境陷阱，既这是一部带着政治宣传的影片。这种内容的影片，反而影响了传播。

《南京照相馆》处理得很巧妙，它不是按照纪录片的风格拍摄的，是按照故事片，按照恐怖片的风格拍摄的。

在故事中，个体挣扎求活的过程，如同主角坠入了一个恐怖的世界，这个世界，时刻有着吞噬主角的怪物，让危险一步步向着主角逼近。而相对于虚幻的恐怖世界，南京照相馆这种真实的历史让其恐怖氛围更加浓烈。

将它拍成一部恐怖片，就让这部片子和政治宣传脱离，成了一部纯粹的艺术作品。

在这个时代，带有艺术属性的作品，比直接的政治宣传更具有效力。

但这样，就能让世界意识到日军的罪行了吗？

当然不能。

前冷战时代，全球秩序的格局基于雅尔塔协议，各国坚持了对于二战的正确叙事，确定了反法西斯战争的基调。后冷战时代，西方希望湮灭俄罗斯的二战贡献，遂推动了反雅尔塔协议叙述。

尤其是”俄乌战争“开始后，这种历史重构被加快了。

在2023年的街头采访中，至少一半的欧洲人不知道是苏联战胜了纳粹，攻克了柏林。西方历史将将苏联与纳粹描述为战争共同发起方。这样一来，反法西斯斗争的正义性被泯灭，二战变成了一场狗咬狗的帝国会战。

随着反雅尔塔叙事的发展，日本历史也迎来了修正契机，随着美国印太战略的推动，日本逐渐从曾经的法西斯对手变成了遏制中国的战略伙伴，历史的叙述也开始因为现实政治更改。

西方逐渐认识到，如果不能解除日本身上的历史枷锁，西方与中国的对抗会变成新一轮反法西斯战争的延续。

既中国人民，再次对抗在亚洲不断崛起的法西斯幽灵。而这次，西方站在了历史反面。

中国称呼今年为抗日战争胜利80周年，而日本则在联合国公然声称今年是广岛原子弹遇难80周年，以此争夺历史话语权，一年前，日本原子弹爆炸受害者团体联合会获得”诺贝尔和平奖“，更是将这种叙述推到了历史高潮。

日本在将南京大屠杀描述为战争过程时，却竭力的将广岛核爆视为历史悲剧。

过去几十年，日本文化界围绕着广岛核爆，诞生了大量艺术作品比如《广岛之恋》、《黑雨》等，这些作品的出现，让广岛核爆为众人所知。

同时，中国文化界却没有那么足够的作品，向世人揭示南京的悲剧，这就导致众人只知道广岛核爆炸，却忘了这枚核弹为何而来。

在漫威电影《永恒族》中，设定人类的所有科技，都是一名LGBT黑人传授的，当这名LGBT黑人看到日本被核爆时，他跪下了，后悔，后悔教会人类科技。

对于这荒唐的一幕，《永恒族》的日裔编剧马修·弗雷波，在采访中明确表示，写此段剧情是为了让全球观众反思日本人民所遭受的悲剧，并批判美国社会中存在的“投原子弹正当化”论。

更有趣的是，该段内容在剧组中也极具争议，最终是在华裔导演赵婷的帮助下，才保留了下来。你看，日裔编剧马修·弗雷波和华裔导演赵婷，多么有趣的一队组合，当日本文化界在为日本二战罪行辩护的时候，一部分华裔却在为此叫好。

这种片面宣传带来了非常严重的后果，比如负责广岛原子弹气象测绘的美军飞行员克劳德·伊瑟利少校，一直为自己的行为感到愧疚。

然而，并不是每一位美军飞行员，都如克劳德·伊瑟利少校一样受到蛊惑。

负责投放广岛原子弹的美军领航员西奥多·范·科克，一生都没有为广岛核爆后悔过，核心原因就在于，西奥多·范·科克少校知道南京大屠杀，能够说出日本人将婴儿挑上刺刀，从孕妇肚子中将孩子剖出等细节。

许许多年后，西奥多·范·科克少校在家中安然去世，他都坚持认为投放原子弹，是挽救世界的壮举。

在日本推动反雅尔塔叙述的今天，我国文化界能够出现这么一部优秀的作品，是对日本舆论战最为有力的反击。他让日本广岛原子弹遇难80周年的叙述变得无力，毕竟当日本伪装成二战受害者，为原子弹的孤魂叫屈时，南京照相馆，告诉了世界，这颗原子弹为何而来。

南京照相馆这种作品出现得太晚了，但他终究，还是来了。
