
# 中西部不对称流动加剧，强者恒强，难免虹吸

## 中部人口缩水百万

2024年中部六省的常住人口总计减少了94万人。安徽、湖北和湖南主要是因为老龄少子化导致人口减少，而江西、河南和山西则更多是因为外流导致人口减少。

### 安徽的抢人策略

安徽是中部六省中唯一一个人口正增长的省份，总人口只增加了2万人，但机械增长人口（外来流入人口减去流出人口的数量）达到了15.7万人。合肥增长了14.9万人，芜湖增加了4.1万人。安徽能"抢人成功"的原因有两个：一是风投式招商，合肥国资通过股权投资，把大量产业链招揽到本地；二是梯度转移，招商来的产业并没有全部集中在合肥，而是在合理的梯度安排下转移到了马鞍山、亳州、滁州、安庆等城市。

### 河南与山西的情况

河南一年少了30万人，机械增长人口占到了28.9万人，大部分人都流向了江浙沪和北京。山西属于"人口小省"，本地人口只有3000多万，随着产业不断流失，2024年一年山西少了20万人，主要是外迁到了京津地区或湖北地界。

## 西部省份不对称流动

### 西南地区

四川、重庆的自然增长人口减少较多，老龄少子化情况比较突出；而贵州、云南、广西、西藏的自然增长人口则是正增长，特别是贵州，自然增长人口一年增长了11.6万人。贵州人口增长的原因可能是少数民族地区比汉族生更多孩子，以及养育成本低。

从机械增长人口来看，情况则相反，四川、重庆的机械增长人口非常多，分别达到了21.3万人和8.2万人，而贵州、云南、广西三省则是净流出，机械增长人口都超过了-16万人。

### 西北地区

陕西的人口维持微增长，西安市增加了8.9万人。新疆是人口的"双份赢家"，自然增长人口有8.8万，机械增长人口更是达到了16万，这背后和国家战略密不可分。新疆要成为西部粮仓，要成为向西开放的桥头堡，还要保证新疆的安全稳定，所以大量的资源流向了新疆地区。

内蒙古则是"双减"，自然增长人口减少了6.8万，机械增长人口还流出了1.2万。蒙东和蒙西地区的人口占比下降，赤峰、呼伦贝尔、通辽等传统农业依赖、产业单一的地区人口则持续外流。而区域协同联动发展的呼包鄂乌地区，占比在持续提高，呼和浩特、包头、鄂尔多斯等经济强市，凭借产业升级、高收入及优质的公共服务，推动人口集聚。

## 人口变动，强者恒强

### 全国人口变化的特点和趋势

1. 1962-1976年的婴儿潮人口加速步入老龄化，退出劳动力市场，传统意义上的人口红利在远去。不过未来，这批要大量退休的富裕老年人，正在成为我们的一个新人口红利，未来银发经济很可能是我们最大的内需，至少是之一。
2. 中国的出生人口在2016年见顶，人口已经成为存量竞争的要素，各省的老龄少子化现象正在普遍化，未来的城市谁能抢到更多的人口，就有持续发展的基础。楼市大分化的本质，就是人口，尤其是年轻刚需人口流向的分化。
3. 人口的分化未来会越来越集中，中国目前是四个经济增长极，长三角、珠三角、京津冀、成渝，位于这些大都市圈周边的省份，人口会被持续虹吸到都市圈中去。
4. 经济发达地区内部，随着产业的外溢，人口有向低能级城市转移的趋势。
5. 经济欠发达地区，人口不仅向外省转移，还会不断向省内的大城市转移。
6. 未来抢人大战的力度和烈度会更加空前，人口趋势和城市优势的固化，虽然缓慢但很难逆转。
