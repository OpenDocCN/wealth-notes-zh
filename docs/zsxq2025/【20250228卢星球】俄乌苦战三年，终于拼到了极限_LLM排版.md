
# 俄乌苦战三年，终于拼到了极限
## 文/卢克文工作室嘉宾 风雨如歌

俄乌战争爆发三年了，这场迄今为止21世纪欧洲最大规模的地缘冲突，已经让参与出牌的人个个气喘吁吁。在这些参与者的背后，留下了一道道惨不忍睹的深深裂痕，这些裂痕包括但不限于：

1. 军事同盟极化。芬兰与瑞典入约，让北约与俄罗斯的边界延长了1320公里；俄朝绑定，朝鲜向俄提供了超100万枚炮弹，以及无法准确计算数量的士兵；核战略外扩，白俄罗斯允许俄罗斯战术核武器部署。
2. 能源秩序崩坏。欧盟对俄实施了九轮制裁，遭受能源危机反噬，被迫加速能源转型；美国趁机补位，天然气出口一度激增154%，成为欧洲能源主要供应国。
3. 被击穿的经济链。美国对乌军援约为1750亿美元，大部分没出国门被军工利益链消化，但却推动了北约国防开支集体突破GDP2%门槛；欧盟为替代俄气多支出逾3000亿欧元，导致巴斯夫等巨头将产能转移至中美；《通胀削减法案》引发欧洲产业外流，法德工业竞争力遭双重挤压。
4. 经济集团化。美欧推动“友岸外包”，中国深化“双循环”，“一带一路”在中亚投资增长41%，填补俄欧贸易萎缩的真空。欧盟建立“乌克兰重建基金”，G7推行俄油限价，上合组织吸纳伊朗并启动本币结算体系，土耳其掌控黑海运粮协议，印度成为俄油贸易枢纽，巴西推动南美洲防务一体化。
5. 南方觉醒。金砖国家扩员至10国，非盟加入G20，77国集团推动联合国改革，已不满足自己的安理会席位数量。
6. 全球治理体系削弱。联合国安理会关于俄乌的12项决议均因否决权搁浅。

接下来，我们重点回忆一下，俄乌战场到底发生了什么？

## 1

对乌克兰而言，好消息是，和2023年相比，俄军占领的领土也就增加了3000多平方公里，不算多。坏消息是，俄军攻城掠地的速度，比2023年明显加快。2023年，俄军常常一个月占不到两条村，能有几十平方公里就不错了。但从2024年4月以来，俄军每月拿下几百平方公里已是常规操作，还打出了一系列精彩的战役，如弗勒达战役、库拉霍沃战役、大诺沃西尔卡战役、纽约镇战役等等，都是酣畅淋漓的大胜，一扫2023年和2022年的阴霾。而乌军几乎全年被动，就连“库尔斯克反击”这个唯一的亮点，也是虎头蛇尾，远不如两年前的哈尔科夫反击。

前两年俄军打得不好，和“没有准备好”有很大关系。2022年9月，俄军在哈尔科夫被乌军偷袭得手，兵力严重不足是关键。7月的“双子城战役”结束后，前线大批俄军放假回国，哈尔科夫地区就不剩几个俄军。这个操作看似抽象，其实是不得已而为之。从2月到7月，俄军伤亡数万，仅马里乌波尔战役就伤亡两万以上，要知道，俄军这些年打的都是小规模战争，伤亡顶多几百人。现在突然伤亡几万，军队难以接受，为安抚前线，有必要放假一段时间，平复心情。

俄罗斯政府也没有准备好。普京最初的设想是突袭基辅，推翻泽连斯基，扶持亲俄政府。这个设想的依据是2014年乌军的战斗力。那时乌军没啥战斗力，俄军稍一介入，便被打得大败，既然是破房子，我再踹上一脚不就行了？但乌军2014年后完成重建，战斗力提高，普京对乌军的印象没有及时更新，更没做多少准备。预备役没有动员，仅靠不到十万的合同兵，过于勉强。当伤亡超出预期，前线兵力不足时，自然处处是漏洞。

最重要的是，俄罗斯社会没有准备好。“俄乌战争”后，俄罗斯遭到的制裁力度远超2014年，真正的全方位、无死角制裁，不仅融入欧洲无望，还要与整个西方为敌，被“文明世界”彻底开除。于是在2022年，有20-40万人逃离俄罗斯，大部分是中产和富豪。即使支持战争的人也乐观不起来，毕竟乌克兰有西方的鼎力援助。社会思潮的摇摆，影响了俄军战斗力。

有这三个“没有准备好”，俄罗斯的国力优势难以发挥，内斗不断，甚至整出了瓦格纳兵变。只是当时间来到2024，形势变了。富豪想润西方？对不起，你的资产早已被冻结，解冻遥遥无期，跑出去就是送死。中产和普通人想润西方？不好意思，人家不要。现实就是，即使俄罗斯想投降，西方都不接受，大部分俄罗斯人意识到，除了获胜别无选择。有共识，就好办事。于是俄军军改完成，经济转向战时轨道，一切为了前线。当俄罗斯调整好姿势，准备死磕后，乌克兰的劣势就会暴露。

西方可以援助装备、银子，却援助不了人员。刚一开战，乌军就动员了上百万人，其中许多是死硬反俄的退伍老兵。他们经验丰富、技能出色、意志坚定，给俄军制造了很大的麻烦。然而随着伤亡的增加，乌军兵力吃紧，战斗意志下降明显。巴赫穆特战役时（2022.9-2023.5），乌军和瓦格纳展开逐屋争夺，双方打到最后一栋楼。但从2024年起，只要俄军攻入城内，乌军很快会撤退，仅做象征性抵抗，无论弗勒达、谢利多沃还是库拉霍沃，都是如此。

刚刚结束的大诺沃西尔卡战役，乌军第110机步旅的公开抱怨称，他们有足够的炮弹，有足够的无人机，却没有足够的步兵守住阵地。即使无人机满天飞，步兵的作用依旧无可替代。顺便说一下，第110机步旅这支部队，在2024年2月被打残后，理应后撤整补，然而乌军指挥部没有这样做。该旅仅进行了简单的补充，便重返战场。因士气低迷，以致于多次出现跑路事件，有两次甚至带崩了防线，让俄军白捡大便宜。就算这样，该旅也没有撤下去，而是不断被安排到有需要的地方“救火”。只能说，乌军的兵力是真紧缺。2025年1月，乌军启动军队结构调整计划，准备组建18-20个军团。

## 为什么要调整结构呢？

因为原有的结构，绷不住了。2014年军改后的乌军，以“旅”为基层单位，架构为“旅--区域司令部--全国司令部”，没有师也没有军，相当高效，能充分发挥战斗力。而发挥战斗力的前提，“旅”的兵力充实。这在战争初期不是问题，可到今天，兵力不足的“旅”越来越多，一些旅甚至只剩个架子，哪来的战斗力呢。理论上，应该把残破的旅合并，实现资源整合，但乌军内部山头林立，极右翼的“亚速”系部队，甚至和泽连斯基存在较深的矛盾，合并难度太大。只能退而求次，重建“军”这个层级，通过军长协调不同的旅变相整合资源，代价就是效率低得多。

不仅缺兵力，乌军还缺装备。开战之初，乌军的装备是充足的，来源主要有苏联库存、西方援助和自行生产。经过三年消耗，苏联库存见底，只能靠后面两者。对于自行生产，乌克兰不可谓不努力，战争期间还搞出了无人机产业链，能达到60%-70%的国产化率，还能生产一些装甲车和卡车炮。为降低乌克兰的生产能力，从2022年9月开始，俄军不断用导弹和无人机打击乌克兰的电力系统，经过两年多的不间断“关照”，电力系统毁坏严重。目前，乌克兰大部分地区，一天停电20小时以上。电力匮乏，工厂难以运转，许多大件无法生产，不得不寄希望于西方援助，但西方“去工业化”严重，连炮弹都供应不上，美国说要增加炮弹产量，一直到今天，也没增加多少。

以155毫米炮弹为例，开战前，美国的月产量是1.5万枚，到2024年中旬，也不超过4万枚，根本无法满足前线乌军的需求，其他北约国家也没好哪里去。今天乌克兰炮弹供应的中坚，是韩国。韩国的155毫米炮弹产能一年20万发，还有300多万发库存，够用一阵子。从2023年起，韩国大量向乌克兰间接提供炮弹。所谓间接供应，就是美国先向韩国采购炮弹，再转交给乌克兰，就不算韩国直接援助乌克兰，俄罗斯生气归生气，也无话可说。截止2024年末，韩国合计供应炮弹上百万发，巧了，朝鲜也在向俄罗斯提供炮弹，难怪有人调侃，俄乌战争是朝韩之间的代理人战争。当然，即使有韩国，依然不能满足需求。

乌军和俄军的炮弹发射比例在1：10左右，许多乌军都是这样反映，也就是乌军发射一发炮弹，俄军会发射10发。炮弹都不够，可想而知，坦克等大件更勉强，“豹”式、“艾布拉姆斯”等西方坦克的援助数量，合计不足500，完全跟不上消耗。乌军在2023年巅峰期曾有过7个坦克旅，但两年的2025年，仅剩三个，剩下4个都被改编成了“重型机械化旅”。坦克旅有90多辆坦克，而“重型机械化旅”只有50-60辆坦克，反映的问题，就是坦克数量不足，只能降级。

人员不足，装备也不足，乌军逐渐陷入劣势，历史一再告诉我们，当前线打不赢时，内部分歧就会增加。2024年2月8日，泽连斯基签署法令，将乌军总司令扎卢日内解职，并将其除掉军籍，理由是“体检不合格”，这个理由侮辱性极强。在2023年12月的一次采访中，扎卢日内表示，要再动员50万新兵。要知道，乌军在开战后动员了一百多万人，而官方的阵亡数字才3万，算上受伤，总伤亡10万出头，理论上还剩80多万人，没必要再动员50万新兵。同时，乌克兰社会对动员的热情减少，抵触情绪日增。动员难度很大。各方普遍估计，乌克兰政府的实控人口，不超过2500万。50万就是2%，比例相当大，容易引发不可控事件。所以，泽连斯基反对再动员50万。

而扎卢日内显然清楚真实的伤亡数字，知道兵力不足，不动员没法打。战略上，扎卢日内认为，短时间战胜俄罗斯不现实，得慢慢熬，别老是想着进攻。基于这种考量，从2023年11月开始，乌军投入大量资源，用于修筑防线。但泽连斯基认为，慢慢熬？熬得过吗？防守没出路，应该将有限的资源投入攻势，再打出一次“哈尔科夫反攻”那样的大胜利，从而一举击败俄国。双方都有各自的道理，都无法说服对方。

2024年2月，阿瓦迪夫卡局势恶化。借着这个由头，泽连斯基将扎卢日内解职，由陆军司令西尔斯基接任。西尔斯基的策略不是一味地防守，他要和俄军对攻，“库尔斯克反击”就是其手笔。8月6日，乌军出人意料地攻入俄国库尔斯克州。俄国毫无准备，丢失大片领土，最高峰时，乌军控制的库尔斯克州面积达1200平方公里。如果俄军主力不回援，仅靠二三流部队难以抵挡，乌军可能横扫库尔斯克，如果俄军调走顿巴斯的主力部队，回援库尔斯克，那么顿巴斯乌军的压力，将大大减轻。理论上，无论俄军怎么选，乌军都占便宜。但理论只是理论，不等于现实。

俄军的选择，是尽量不调顿巴斯的主力部队，而是调赫尔松州的部队，以空降军和海军步兵为主。这些部队赶到后，很快遏制了乌军的攻势，时至今天，乌军占着库尔斯克400平方公里左右，属于食之无味，弃之可惜的鸡肋。“库尔斯克反击”不仅未能达成目的，反而导致乌军面临的局势进一步恶化。“捷尔任斯克--纽约镇”城市群，是乌军在顿巴斯最坚固的堡垒之一，经营十年之久，防御体系完善，正常情况下，没有5到10万兵力很难攻克。俄军知道不好打，开战后就没怎么碰。为准备“库尔斯克反击”，当地乌军陆续调走，仅剩一些质量较差的部队，俄军抓住机会，仅用一支不足万人的偏师，就拿下了这里。放在以前，这种事想都不敢想。

乌军兵力不足，装备也不足，俄罗斯内部又比较团结，胜利的天平正加速向俄罗斯倾斜。泽连斯基的战略不能说错，乌克兰体量不够，消耗战很难玩得过俄罗斯，必须梭哈一把。但如果像“库尔斯克”这样，梭哈失败了呢？除非北约亲自下场，否则单靠援助，乌克兰没有获胜希望。

对川宝来说，与其耗下去，不如快刀斩乱麻，及时止损，能止一点是一点。万斯说得很清楚，俄军有人力和装备优势，无法被战胜，总统只是在面对现实。当然，从战线上看，乌军并未出现大崩盘，看起来有逆转的希望。正因此，很多民主党人以及离岸民主党人，对川宝的180度转向感到愤怒。

随着美俄谈判的进行，乌克兰获得援助会减少。一个缺兵、缺钱、缺装备的乌克兰，还能战斗多久呢？没人知道。但可以肯定的是，前所未有的严峻局势，即将来临。
