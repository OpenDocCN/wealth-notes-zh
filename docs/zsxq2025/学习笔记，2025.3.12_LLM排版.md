
# 学习笔记，2025.3.12

1. 不要在意别人的目光，放弃给别人留下好印象，这个世界上除了自己，没有人值得被讨好。一个真正的高手，不需要任何人的认可，也不会在乎别人的眼光和评价。另外，如果你想做成一件事，就不要在意别人的嘲讽。因为真正做事的人不会嘲讽你，他们深知其中的不易，嘲讽你就相当于嘲讽过去的自己。

2. 及时止损，当别人不需要你的时候，要学会收回你的热情，并礼貌退场。你可以躲在被窝里哭，也可以喝酒喝到吐，但是你不可以拿起手机发不该发的消息，打不该打的电话。"及时止损"是成年人的必修课。一扇不愿意开的门，你还一直敲，那就是你不礼貌了。

3. 搞互联网得有恒心，得坚持，前期肯定比较惨，毕竟流量有个积蓄过程。但是，一旦开到奖，人生就彻底不一样，一年抵十年甚至是1000年。关键是先动手，不断去试，不论成败。

4. 其实人最好的状态就是销声匿迹，半隐半藏，除了赚钱以外，平常是不需要太多的社交和喧哗，我们需要的是安静和独处，不要被外界打扰，享受生活，低调清醒。

5. 普通人也可以接触到一些"高于自己阶级"的人，比如老板。想要提升自己的阶级，你可以先去试试，能不能把老板琢磨透，能不能把老板的言行举止分析明白，能不能成为老板那样的人。如果这第一道坎都不过去，却想着比老板更有钱、更成功，恐怕不太现实。

6. 一个人的说教或者道理管用，别人肯听，很大程度上是因为他很强，人们信的不是他的话，不是事实，而是他身上的权威、能量、身份那些过往的标签。比如，同样的一句话，马云说和马大爷说，人们的信任程度就是不一样，大多数人都慕强。

7. 学习，是俩字，学是输入，是让知识进入脑子，习是输出，是让知识出来。只学不习，是知识囤积者，只习不学，那是没记忆的鱼。

8. 岁数不能定义一个人的精神状态，一个人20岁，可能暮气沉沉，一个人80岁，可能朝气蓬勃。人有盼头、有希望，就会心态年轻、有动力。人要是整日抱怨，觉得什么都不美好，人的状态就会越来越差。这也是一种自证预言，越说什么，越信什么，越容易成为什么。

9. 公开场合能谈的，基本都是正确的话，第一目标是保证讲话人的安全，而不是让你去思辨事实，这种话，听听就行，听多了你也能谈，模版就那么几套，能够保证挑不出错来。有那个精力，可以去多读些一些老家伙们临死前写的书，那些经过实践验证的书，敢说的、有用的东西会更多一些。

10. 在你一无所长时，不要琢磨着结识贵人、拜访贵人、求贵人、混圈子，没人会搭理你，更不会帮助你。他们跟你又不熟，也没有什么利益关联，你也不配与他们为伍。就算帮你，你也干不好什么事，你没能力、认知低，总不能把钱直接打到你的账上吧？所以，要得到贵人的关注，让他们指点你，你必须至少有一个绝活，有一个拿手的本事，或者你有什么独特的资源、信息，先去利他，在某方面给他提供价值。
