
# 央行为什么公开说让物价上涨？对你影响多大？

最近，央行发布了《2024年第四季度中国货币政策执行报告》，其中明确提出，"把促进物价合理回升作为把握货币政策的重要考量"。意思是央行接下来的货币政策，要重点考虑怎么能促进物价上涨。很奇怪是吧，你看央行也不藏着掖着，因为央行觉得这不是坏事。

这件事的幕后其实指向一个关键问题：就是中国正在进行一轮新的"温和再通胀"。

什么是温和再通胀？为什么要这么做？对你会有多大影响？搞清楚这些问题，能帮你更好地理解央行推行物价上涨政策的背后逻辑，避免过度恐慌，同时帮你理性规划自己的消费和投资决策，提高应对经济变化的能力。

好，正题开始。

## 解释几个基本经济概念

咱们先来学习几个基础的经济概念，为理解后面的内容疏通障碍。

首先是通胀和通缩。

通胀其实就是物价全面、持续的上张。而我们要讲的温和通胀，它的逻辑是，在涨价的预期下，人们会担心"未来更贵"从而提前消费，商家那时候有利可图，就会扩大生产、增加招聘，这又进一步增加了就业、收入和居民消费。说到这你就明白了，温和通胀，其实不是一件坏事。当然，恶性通胀就是另一个逻辑了，不是今天的重点，就不展开说了。

而通缩，则是截然相反，物价全面、持续地下降。这会使得居民不敢花钱，进而导致有效需求不足，这时候企业为了争夺客户，打起价格战，然后企业挣不到钱，又会裁员，加大失业，最后大家挣不到钱，有效需求就更低，形成一个恶性循环。

换个更形象的说法，通胀和通缩的本质，其实就是"钱在哪里"的问题：经济通缩，是钱躺在银行里；而经济通胀，则是钱流通在市场中。

理解了通胀和通缩的含义，那么怎么衡量这两者的程度呢？经济学家创造出了CPI、PPI两个指标，CPI，也就是消费者物价指数，它是衡量生活物价的，比如吃喝住行，全部统计在里面。

而PPI，是生产者物价指数，它则是衡量生产物价的，反映企业的原材料等各项生产成本。按照我们刚才说的温和通胀的逻辑，适度的CPI和PPI涨幅，是有利于经济发展的。比如CPI每年涨2%～3%，PPI每年涨1%，都是比较合适的。

好，概念懂了，继续说。

## 激活经济的举措

我们先从当下的经济环境说起：从2023年上半年以来，中国的CPI长期在0附近波动，PPI连续28个月为负，汽车、光伏、电池等领域，价格战不断，各行各业都在降价求生，企业投资、居民消费都比较谨慎。

最终的结果是，经济体中的货币高达300万亿，但超过180万亿的资金，都沉淀在了银行中，没有流通进市场。简单说，就是经济体并不是没钱，而是钱被"冻"住了。这就像人体的血液一样，血液要在血管里正常流动，才能更好地运送营养和氧气；要是全充血了，在器官里不动，长期下来人肯定会生病。

面对这样的情况，中国央行就得想办法，把这些沉淀的资金"逼"出银行体系，进入到市场流通起来。

所以这段时间央行想了好多办法，比如降低利率，把利率从5%的高位，一路降低到现在的1.5%。背后的想法是，利息少了，居民就不存钱了；再比如，增加市场上的货币供应量，从2021年到今天，广义货币供应量，简单理解就是市场上的钱，也就是M2，从200万亿突破到了300万亿，释放出来的钱超过了100万亿。

但这些措施效果并不理想，利率低了，大家存钱的热情并没有下降。同时，虽然放了更多的钱出来，但钱在市场上转了一圈，大部分被存回到银行吃利息了，体现在数据上，就是居民存款历史上首次超过了150万亿。

因此在各种办法都不太给力的情况下，推动物价回升，就进入了央行的视线。央行开始在货币政策报告中强调"涨价"，我列一下政策原文啊：

2024年第一季度：维护价格稳定；
2024年第二季度：维护价格稳定、推动价格温和回升。注意，光稳定不行，开始进一步强调推动价格回升了。
2024年第三季度：把促进物价合理回升作为把握货币政策的重要考量，推动物价保持在合理水平；
2024年第四季度：实施好适度宽松的货币政策，把促进物价合理回升作为把握货币政策的重要考量，推动物价保持在合理水平。

从中你可以看出来，央行对于物价的表述越来越"重"，从最初的"维护"已经变成了"重要考量"。

道理很简单，只有物价有温和上涨的预期，大家感觉能挣到钱了，商家才会扩大投资、增加招聘，居民也会有更多的就业机会和收入，才会增加消费，整个经济的资金也就慢慢流通起来了。

所以"涨价促消费"，正在从经济设想，成为政策现实。

## 什么是温和再通胀？

但中央想要的可能还要更深一步，这就是被很多经济学家，称为当前中国经济走出困境的核心办法：温和再通胀。之所以说是"再"，是因为在2000年到2018年这一轮经济周期中，货币已经"通胀"过一轮了，现在则是要在前一轮的基础上"再度温和通胀"。

那么，为什么温和再通胀被看作是走出困境的关键呢？

首先，我们开头解释过，温和再通胀不是一件坏事，它能让人们基于涨价的预期，担心未来会更贵，从而提前消费，带来投资和消费的提振；同时，国家也能更好地应对外部的冲击，加快推动中国建设"内循环"体系；而对企业来说，也能从激烈的价格战中脱身，喘一口气，商业生态会更友好。

其次我们要知道，目前地方政府面临的一大困境，就是政府债务问题。这个债务的具体数据，不同机构有不同的看法，但普遍认为是超过100万亿，这么多的钱要还，压力确实很大对吧。只要维持每年3%的CPI通胀率，那么只需15～20年的时间，这么多的地方政府债务，就能被有效化解。这样做的本质，是通过提高物价，让钱变得"不值钱"，钱都不值钱了，债务不就被稀释了吗？这么解释，你是不是突然明白了很多。

所以，不管是从经济、债务还是从产业的角度来看，你千万要低估央行以及政府推动物价上涨的决心，这会是一个长期性、持续性的政策导向。这个判断很重要，你一定要记住。

或许你还记得，去年的时候掀起了一波水、电、燃气的涨价潮，这是国家试图通过推动基础物价上涨，来带动整体物价上涨，但效果其实并不太好，舆论声音也有点大。

因此，从去年下半年开始，国家调整了"涨价策略"，从单一的价格调整，转向财政、货币、基建、预期管理等多个领域的"组合拳"，用更系统、更协同的方式，推动物价稳定和温和再通胀。

具体来说，国家的协同措施包括以下四个方面：

第一是稳定消费品物价：比如政府启动了"以旧换新"计划，对家电、汽车消费，提供10%～15%的补贴，这不仅刺激了消费，还能避免汽车、家电等行业的恶性价格战，稳定了大件消费品的物价。再比如发放定向消费券，面向文旅、餐饮等领域，撬动居民的消费倾向，进一步稳定生活消费品。

相信你也能感受到这一点。

第二是稳定投资品的的价格：货币政策方面，继续降低存款准备金率、贷款利率和存款利率。2024年央行已经释放了2万亿元的长期流动性资金，并推出了一系列降首付、降契税、取消限购的措施，来稳定房地产价格止跌回稳，房贷利率也降到了历史的低位。而对于股市，央行也推出了"股票回购再贷款"等政策工具，总之就是让更多的货币能直达资本市场，更好推动股市"慢牛"，长期稳步发展，提升居民财产性收入。

第三是稳定生产品的物价：包括新增1万亿元特别国债，支持三大工程，就是建设保障性住房、城中村改造和"平急两用"公共基础设施建设，定向支持新能源、半导体等战略性产业，对生产农业和民生必需品的企业，提供生产补贴，稳定产业链价格，缓解成本压力。同时，各地纷纷加大政府的基建投资、制造业投资，用持续投资，来带动生产产品物价的稳定。

第四是大力进行预期管理：比如延长春节、五一假期，释放假日经济潜力，这不仅促进了消费，还带动了相关产业的发展。同时，中央直接强调要"反内卷"，让企业别这么比着降价了。

听到这里，你应该明白了，推动物价上涨，绝不是简单的价格调整，而是为了重塑经济系统的"造血功能"，让淤堵在银行中的资金以良性的方式，重新进入市场流通。

## 普通人的应对策略

我们作为普通人，面对国家这一长期性的温和通胀战略，既不过于恐慌，也要避免"躺平"心态，这里给你两个建议：

第一，在消费方面，可以把握"早鸟红利"。

现在正是政府和企业联合补贴的窗口期，比如新能源汽车置换补贴，已经有多地宣布沿用2024年的政策，延长到了2025年。当然，2025年以后的补贴情况，还要等新的政策出台。如果你现在有更新老化汽车、家电等必需品的需要，不妨趁机提前升级。当然，前提一定要是必需品。

第二，在资产方面，选择配置“通胀友好型资产”。

如果你有投资的的需求，建议考虑一些能对抗通胀的资产，比如核心城市的房产，特别是那些人口还在流入区域的品质住宅，还有一线城市核心区的小户型房产，仍然具有保值功能。另外，一些公用事业等必需品类股票，就是我们日常生活中密不可分的行业股票，像是自来水、燃气等等，以及黄金，都可以作为保值的工具。

当然，这些只是参考，都不作为任何的投资建议。

最后，更本质的，还是要看清时代趋势，找到面向未来的城市和职业，让自己收入，跑赢温和通胀。

同时也欢迎你把这一节分享给你的家人和朋友。
