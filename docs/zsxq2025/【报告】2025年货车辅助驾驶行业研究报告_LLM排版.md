
# 2025年货车辅助驾驶行业研究报告

## 行业概况

货车ADAS：指安装在货车和/或牵引挂车上的各类驾驶辅助系统，含0-2级自动化

### 基本定义 汽车驾驶自动化分级

GB/T 3730.1-2022 汽车、挂车及汽车列车的术语和定义 第1部

| 分级 | 名称           | 动态驾驶任务后援 | 设计运行范围         |
|------|----------------|------------------|----------------------|
| 0级  | 应急辅助       | 驾驶员           | 有限制               |
| 1级  | 部分驾驶辅助   | 驾驶员           | 有限制               |
| 2级  | 组合驾驶辅助   | 驾驶员           | 有限制               |
| 3级  | 有条件自动驾驶 | 有限制           | （接管后成为驾驶员） |
| 4级  | 高度自动驾驶   | 系统             | 有限制               |
| 5级  | 完全自动驾驶   | 系统             | 无限制               |

资料来源：GB/T 40429-2021 汽车驾驶自动化分级。

### 货车分类

一般按总质量分为重、中、轻、微型货车，按性质分为营运、非营运货车

| 类别   | 最大设计总质量范围（kg） | 车辆类型             |
|--------|--------------------------|----------------------|
| L类    | >3500                    | 栏板货车、多用途货车 |
| M类    | >3500且车长≥6000mm      | 厢式货车、仓栅式货车 |
| G类    | ≤12000                   | 自卸货车             |
| N1类   | ≤3500                    | 载货汽车             |
| N2类   | >3500且≤12000           | 载货汽车             |
| O1类   | ≤750                    | 挂车                 |
| O2类   | >750且≤3500             | 载货汽车             |
| O3类   | >3500且车长<6000mm      | 特殊结构货车         |
| O4类   | >10000                   | M3类车辆             |

资料来源：通渠有道整理研究。

包括21个信息辅助类和16个控制辅助类功能

| 信息辅助类（21个） | 控制辅助类（16个） |
|-------------------|-------------------|
| N Navigate on Autopilot | AEB advanced/automatic emergency braking |
| ISLI intelligent speed limit information | AES automatic emergency steering |
| AEB advanced/automatic emergency braking | RCTA rear crossing traffic alert |
| FDM forward distance monitoring | EBA emergency braking assist |
| AES automatic emergency steering | FCTA front crossing traffic alert |
| ...                | ...                |

资料来源：通渠有道整理研究。

## 货车市场特征

货车体积大盲区大、载重大刹车难，不同生产场景需要不同规格的货车，因此货车的定制化程度极高。货车市场受经济调控、环保排放、道路交通安全政策等直接影响，绝大多数散户和物流企业利润薄，导致对货车购置、使用和维护成本极为敏感。

## ADAS市场痛点

安全政策驱动、投入产出不高、功能比较鸡肋、技术积累偏弱

## 发展历程

头部企业引领与安全法规驱动并进，2024年是ADAS上规模的关键节点

## 产业链构成

由零部件厂商、系统供应商、货车制造商、运营服务商、配套商构成

## 国家政策

明确2025年重卡AEBS装配率达20%，达标车辆核查与科技降险是主驱动
