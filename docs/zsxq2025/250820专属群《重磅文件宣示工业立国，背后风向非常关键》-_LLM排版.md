
# 重磅文件宣示工业立国，背后风向非常关键

## 重磅文件宣示工业立国，背后风向非常关键

最近，大家都在关注社保的事，我在前两讲也做了观察和分享，但其实还有一个非常重要的文件，被忽视了。这就是7月29日，央行、工信部、国家发改委、财政部、国家金融监管总局、证监会、国家外汇局七部门联合印发的《关于金融支持新型工业化的指导意见》（以下简称《意见》）。听文件名字你就知道，这是中央专门为大力推动新型工业化而推出的一个政策大合集。

这件事为什么重要呢？在“十五五”规划即将出台的当下，这份文件其实就是一次关键的政策发令枪，简单理解，就是国家是在下决心甚至是下狠心，要把巨大的金融资源，深度导入新型工业化，尤其是制造业中。这个信号和影响，非常之大。

比如文件的最开头，就直接指出：“工业是立国之本，是经济社会高质量发展的基石，实现新型工业化是以中国式现代化全面推进强国建设、民族复兴伟业的关键任务。”

“立国之本”“发展基石”“关键任务”，这个定调已经没法再高了。类似说法不是第一次提，但我认为这个时候着重提，背后是以顶层设计的高度，把新型工业化置于国家战略的关键位置，这意味着中国未来的发展路径，将确定无疑地聚焦于以科技创新为驱动，以高端制造为支撑的工业体系升级。文件背后的国家意志，尤其明显。

接下来，我会用两节课，详细谈谈我对新文件的观察和理解，带你看懂国家如何以国家意志，让金融政策和产业政策高度结合，以及对企业的具体利好。这个话题会比较枯燥，我会努力把它讲清楚、讲得离生活更近一些，也建议你好好听一听。

## 大风向，很重要

先说下什么是“新型工业化”，新型工业化是相对于传统工业化来说的，新华社的文章说，数字化、智能化、绿色化，是新型工业化的鲜明特征。发展新型工业化，金融作为资源配置的核心工具，非常关键。但之前，存在大量资源错配的问题：

第一，是期限错配，就是工业技术攻关和产业升级，往往需要长期资金支持，但传统银行信贷以短期为主。

第二，风险收益不匹配，科技创新具有高风险、高收益的特点，而传统金融机构的风险偏好很低，更倾向于重资产行业的抵押模式。

第三，政策协同不足，现在很多产业政策和金融政策没有很好结合，导致资金难以精准导入重点实体经济领域。

而《意见》的出台，正是为了破解这些难题，要构建出一套以新型工业化为核心的新金融模式。我认为它本质上，是要金融业从过去的“金融配套房地产”，转向“金融配套新工业”。

所以文件提出：“到2027年，支持制造业高端化智能化绿色化发展的金融体系基本成熟……服务适配性有效增强”。

我认为，这个目标包含两层意思：第一，要在未来两年中积累经验，形成成熟体系；第二，在风险可控的前提下，要增强服务的精准化，金融资源要与新型工业化需求动态匹配。背后的含义，也是要求金融要“甘当绿叶”。

包括之前中央也反复强调，金融要把功能性而非盈利性放在第一位，说白了，我理解这个功能性，就是金融业要按照国家要求支持实体经济发展，而不是自己挣太多钱。因为金融业的利润高了，实体经济的利润就少了。

中央这个大风向，非常重要，过去两年也对金融业产生了巨大影响，尤其是银行应该深有体会，但是目前看，我判断决策层觉得还是不够，所以新文件要求继续变。

## 三个关键政策

搞清楚了大风向，接下来我们谈谈《意见》的具体内容。具体举措是从文件的第二部分开始，首先是针对新型工业化的核心痛点——产业科技创新能力不足与产业链供应链韧性薄弱，给出了三个关键政策。

第一，“优化金融政策工具，支持关键技术和产品攻关”。

《意见》提出，要“发挥结构性货币政策工具激励作用，引导银行为集成电路、工业母机、医疗装备、服务器、仪器仪表、基础软件、工业软件、先进材料等制造业重点产业链技术和产品攻关提供中长期融资”。

我刚才说的这些领域，就是这次文件要发展的重点领域。它们的很多关键攻关周期，往往需要很久，我认为实质上是在试图建立“技术攻关专项信贷”机制。同时，文件说，对突破关键核心技术的科技企业，要给予在上市融资、并购重组、债券发行方面的“绿色通道”，这是个很重要的具有特殊偏向性的红利政策，我相信未来股市上会有更多这类企业涌现。

另外，文件也说，要加大对各种创新，比如首台重大技术装备、首批次新材料、首版次软件等新产品，在推广应用上的支持力度，我判断，一个可能是，这会跟对国产替代的各种支持联系在一起。

第二，除了金融工具，还要“引入长期资金和发展耐心资本，加快科技成果转化”。

《意见》关于这部分，有两点非常值得关注，比如提出“加强上市预期引导和政策激励，推动社会资本投早、投小、投长期、投硬科技”。翻译成大白话，就是国家要通过对相关领域的企业的企业在上市和政策激励上释放更多信号，让资本看到投资的确定性预期，从而愿意投。这个话说得很直白了，是在通过指挥棒做预期管理。

第三，这次《意见》也强调，要“强化重点企业金融服务，支持产业链自主可控”。你看到“自主可控”这个词，就知道主要针对的是“卡脖子”行业和背后的大国博弈了，文件提出要为重点产业链链主企业和重要配套企业，提供综合金融服务。还要完善并购和贷款政策，让这些链主企业聚焦产业链上下游开展投资。总之，这个文件，背后的国家意志很强烈。

## 有控有扶，先立后破

刚刚我们讲了《意见》第二部分，针对产业科技创新能力不足，和产业链韧性薄弱问题，给出的关键政策。接下来我们讲《意见》的第三部分，聚焦的是金融服务如何深度嵌入现代化产业体系的问题，核心方向是金融不能游离于产业之外，而是必须进入到实体中来。文件主要给出了四方面的针对性政策。

第一，对于传统制造业的转型升级，《意见》的核心原则，就是“引导银行按照有扶有控原则优化授信政策”。这个说法很关键，针对“有控”，我认为自然是指要管控金融资源流向落后无效产能，这些“僵尸企业”创新活力低下，不仅加剧了行业内卷，还挤占了大量本应该流向创新领域的信贷资源，所以必须严格管控。

而针对“有扶”的场景，一个现实是，传统制造业的升级往往面临“不转型等死、转型找死”的两难境地，大规模设备更新与数字化改造需要巨额资金，但企业原有资产往往已经抵押殆尽，银行因风险考量不愿意追加授信。所以现在要建立一套新机制，对符合高端化、智能化、绿色化方向的，要加大支持力度。比如通过“以租代购”，就是用租赁代替购买的模式，减少企业一次性投入压力。这样一来，当银行、租赁公司、投资者共同分担风险时，单个主体的压力得以稀释，整个系统的容错空间会更大。

第二，对于新兴产业与未来产业，政策着力破解“估值不清”与“耐心缺失”的双重难题。科技企业的价值评估，向来是金融业的难题——传统财务指标，无法捕捉技术壁垒与市场潜力，导致金融机构要么盲目追捧“明星项目”，要么集体回避早期投资。所以，针对科技型企业“轻资产、高波动”的特征，文件明确要“扩大科技贷款投放”，这个简单直接但是有用。

同时，对中小企业，要加大对它们“专精特新”发展评价结果的运用，作为融资的“加分项”，以及推广企业创新积分，我理解是可以用积分为依据，代替传统的抵押物，来进行贷款。

而对在新一代信息技术、基础软件和工业软件、智能（网联）汽车、新材料、高端装备、商业航天、生物医药等新兴产业中更成熟的企业，支持它们到股市等资本市场融资。同时，推动政府基金、国企资金、保险资金等“长钱”，投向未来制造、未来信息、未来材料、未来能源、未来空间和未来健康六大未来产业。

第三，文件也反复强调了产业的绿色低碳循环发展。当前，传统产业的绿色低碳转型，面临“船大难掉头”的困境——既需要巨额资金更换新型的设备工艺，又因为环保约束面临着营收下滑的现实风险。所以这次文件强调，要“先立后破”，我的理解是，先让这些传统企业有资金去改造，有了新设备，然后再把旧设备淘汰掉，所以银行要用贷款等方式，支持这些企业的技术改进和产能置换。这对很多传统产业企业，是个好消息，避免了“一刀切”式抽贷引发的行业震荡，形成正向循环，这比行政命令更能激发企业的内生动力。

第四点，主要面向企业在使用金融服务时的具体疑难。很多中小企业融资难的核心症结，还是在于信息不对称——银行难以验证企业订单的真实性、履约能力以及产业链地位，而中小企业本身能用来抵押的资产又没多少。所以《意见》给出的不少政策，就是看怎么能让制造业的中小企业，更简单、更容易地享受贷款等金融服务。

比如，文件明确提出，要让“金融机构运用大数据、区块链、人工智能等科技手段，简化业务手续流程，提高服务制造业特别是中小企业的效率”。还比如，要“支持企业征信机构整合打通大型制造业企业、仓储物流企业等产业链数据，向银行提供链上企业地位、供销交易、账款确权等信息”。这样也是能让更多中小企业，有更多可信的信息来提供信用证明，方便它们贷款。这再次说明了我们开头提到的，银行要有按照国家要求支持制造业的功能性作用。
