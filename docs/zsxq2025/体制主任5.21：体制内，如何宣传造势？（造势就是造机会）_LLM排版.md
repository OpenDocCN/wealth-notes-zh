
# 体制内，如何宣传造势？（造势就是造机会）

原创：主任 公众号：通透的职场人生 25年5月21日

一个善于宣传的人，不论做什么都会成长很快，关于宣传的作用，我觉得毛主席有一句话说的特别精辟，长征是宣言书，长征是宣传队，长征是播种机，在华夏大地播撒下星星之火的革命火种，向全国百姓展现了红军“官兵一致同甘苦”的队伍本色，更向全世界彰显了中华民族百折不挠。

宣传和造势是在体制内制胜的两大法宝。我们说酒香也怕巷子深，是金子不一定发光，在体制内，有的时候事情是你做的，但是领导不知道，你就等于白做。很多人只知道低头干活，不知道抬头造势，这样在职场没什么发展的，两条腿走路的人才能走得更高更远，既要吃苦耐劳能干活，又要会造势、会宣传，打造一个有利于你发展的舆论和口碑，这样你才能在同龄人脱颖而出，因为造势就是造机会。

当然说这个造势不是说你干一点工作就到处宣传，也不是说你在朋友圈去晒加班的图片，也不是口头和领导汇报说我昨天加班到几点，这都不是造势，弄不好还会引起领导的反感，造势我们要善于用好工具，用好各种资源。

一、摒弃学生思维。

我们知道体制内有一种人，做一分说九分，做了一点工作生怕别人不知道，生怕领导不知道，事情还没有什么进展，工作还没有什么思路，就频频跑到领导那里去请示汇报，这种人我们说不提倡，我们批评这种人，他们太浮躁了，不实事求是，好大喜功。但是还有另外一种人走向了另外一个极端，就是只知道埋头工作，从来不向领导汇报，有了工作成绩，别人也不知道是他做的，他们在单位里面默默无闻。而且这样的同志往往有一个称号叫做老黄牛，平时他做的工作最多，但是年底评先评优，跟干部提拔的时候，大家都不记得有他这个人。

为什么会出现这样的情况?我们还是从学生时代开始说起，学生时代我们最重要的任务只有一个，就是学习成绩好，学生时代你是不需要宣传自己的，而且有些学习成绩好的同学还喜欢掩饰自己，就是不承认自己刻苦努力这样一个事实。平时在学校里面的时候，都装作不怎么听课，也没有怎么认真看书做作业，实际上在家里默默看书做题，做到晚上12点，所以在学校里面你越低调越好，你学习成绩好，你也不需要刻意去宣传自己，老师会到处表扬你，亲戚朋友会拿你做榜样教育他们的小孩，你看人家学习多刻苦，学习成绩多好，所以我们经过十多年的学校教育以后，脑子里面就形成了这样一种思维，就是默默的努力工作就好了，不需要刻意的去宣传自己做的工作，这就导致了成为彻底的老黄牛。

我举个例子，比如说单位有一项工作任务，就是负责单位的微信公众号的运营。一开始这个公众号的运营是交给单位里面一个刚进来的小姑娘去负责运营，她非常努力，也非常认真，单位里面有什么亮点工作、重点工作，她第一时间就编辑成微信，她的文字功底、编辑技术应该说都非常不错。公众号说实话质量也比较高，但是运营了半年，公众号的阅读量非常低，单位里面也没有几个人清楚到底是谁在运营这个公众号，上级单位就更不知道有这么一个公众号。

后来这个小姑娘就被调整到了别的工作部门了，从基层借调了一个小伙子来负责公众号的运营。这个小伙子文字功底和编辑能力都比较一般，编辑出来的公众号文章没有原来那个小姑娘好，但是他每一次发一篇公众号文章，他就把这篇文章的链接也发到单位内部的微信群里面了，微信群里面就一片这种点赞的声音，都夸他做的好了。而且小伙子还想了办法，就是他加了局长的微信，每次局长下去基层视察工作，做出什么重要指示，他都会编辑一条图文并茂的这种微信，把局长的这种光辉形象放上去，然后把这篇微信文章也同时发到局长的手机里面。局长看了以后就很满意，对他也有比较深的印象，觉得这个小伙子能力很强。

同时这个小伙子还跟上级单位负责微信运营的负责人建立了联系，每次单位里面有什么微信文章发布的时候，他就把这条微信也推送给上级单位。有的时候上级单位也会发一些系统内单位的工作动态、工作亮点之类的这种微信文章。所以小伙子他发的微信文章有好几次都被上级单位的微信转载了。

在年底组织的这种微信影响力评比上，单位还被评为了系统的新媒体运营先进单位，这样的话，小伙子在单位一下子就出名了。大家都知道单位里面有
一个姓李的小伙子，宣传工作干得非常好，而且局
长也觉得很有面子。写到这里，读者你想一想，同
样是两个人在运营单位的微信，一个人写文章的能
力很强，但是做的工作却没有几个人知道，也得不到认可。另一个人因为懂得在
适当的场合把工作成绩展现出来，得到了领导的肯
定。这个小伙子比小姑娘到底强在哪里。他知道自己做
的工作要使劲的吆喝，知道把工作成绩发到单位的
微信群里面去，广而告之。更重要的是，他懂得体
制内搞宣传工作，领导是第一位的，宣传工作也是
为了宣传领导，另外他还懂得把工作成绩让上级单
位知道，进一步提高了工作的影响力，所以两个人
做同一件事情，就会有了不同的结果。

二、口碑宣传，用舆论去造势。

有的人会自己说自己好，这个方法有没有用?有一
点用，但是用处不大，最好是别人说你好，但是我
们怎么样能让别人说你好?

比方说和领导相处的时候，你不仅要让领导看到你的优良品质，更要想办法让领导听到更多有利于你的声音。因为你和单位主要领导接触面、接触时间
非常有限，他对你的判断很局部，你要让领导全方
位的了解你，那你就要想办法让领导听到有利于你
的声音，你必须找一个传声筒，这个传声筒是谁?
这个传声筒可以是领导身边的人，可以是你的直属
领导，可以是外部的合作伙伴，只要这些人为你发
声，有利于你的声音就会越来越多。

舆论造势它的本质是什么? 舆论造势的本质实际
上是在管理你在领导心目中的人设。领导没有时
间，也没有那么多精力去研究你、了解你，他对你
的初步印象就来自于舆论的声音，那你通过传递声
音来树立一个好形象，来管理领导对你的信任度，
你的口碑、你的风评，大家都说你好，领导也会
在做判断时候受到影响，这是一种间接的造势，你没
有和领导直接接触，但是群众的呼声影响了领导对
你的判断，通过其他人的评价和传播进行造势，进
而影响舆论，这是一个很好很好用的方法。

三、重要的荣誉要争取。

我以前说过，我们当我们还是一个新人的时候，所
谓新人就是进入体制内3到5年内的这么一个概念
了，你要尽量的去争取这些体制内的荣誉，但是如
果你已经工作5年以上了，你已经成为了业务骨干
了，一些不是很重要的荣誉可要可不要。

比如说什么优秀共产党员，什么优秀工作者，什么
业务骨干，这种荣誉你可以不要，当然了具体的荣
誉你是要还是不要，这个也不是绝对的，我的建议
就是说一些含金量比较高的荣誉，其实你还是要
去争取的。比如说你在市一级工作，那么有省一级的荣誉，国
家级的荣誉，就最起码比你所在的地区，比你所在
的单位要高一级的荣誉，你是可以去争取的，你
得到了荣誉，在本地区就会发文表彰，这也是造势的
一种方式。

四、抓住关键的机会。

这个事我在以往的付费文章说过好多次了，如果你
是新人，要学会适度的刷存在感，因为新人缺少的
就是曝光度，缺少的就是存在感了。大家都不太认
识你，也不知道你这个人的性格怎么样，能力怎么
样，所以在一些关键的场合，关键的机会一定要把握住，把
你的才能展现出来，让大家知道你。

体制内机会很少的，要善于抓住机会，你必须把握
任何一个能够显现自己的机会，让领导能够认识
你，让组织能够看到你。因为每一个机会对你而言
都是很难得的机会。

人生的重大机会是不多的，当一个机会出现的时
候，哪怕只有1%的成功概率，你可能也需要付出
100%的努力去抓住它。对于没关系没背景的人来说，你必须去把握每一次
的机会，你把每一次机会都要当做自己脱颖而出，
改变命运的机会去对待，你没有关系，也没有背景，
你除了比别人更拼，做事情更加努力，更加渴望，
更加珍惜这些机会，没有别的选择。

一定要抓住每一次机会，比如向大领导汇报工作的
机会、在上级借调帮忙的机会、单位业务竞赛的机
会，甚至是主持单位晚会、表演节目的机会，都
有可能成为改变职场轨迹的转折点，有的人不屑于这
样的机会，也不想付出努力，看到别人得到回报又
喜欢说风凉话，只等着“天上掉馅饼”是不可能的。
体制内就经常会有各种各样的比赛，比如说演讲比
赛、知识竞赛、什么技能大比武、征文比赛、辩论
赛，还有当然还有一些是体育类的比赛，比如说乒
球、篮球、羽毛球等等，这些比赛就基本上每一
年都会有各种各样的比赛，有一些是自己单位内部
举办的，就自己单位内部组织，还有一些就是区里
面、市里面、省里面，包括系统内举办的，比如说
你是公安系统的，可能全省的公安系统会进行比
赛，你是海关的，那么可能海关总署会举办一些比
赛，然后各个地方的海关进行内部的选拔，把这
个种子选手选拔出来以后，再去海关总署去参加决
赛。

其实普通人想到演讲比赛，什么技能比武这样的机
会，他想到的就是好麻烦，都是形式主义，但是聪
明的人遇到有这样的机会，他想到的是这是露脸的
机会，他想到的是这是展示自己的机会，他想到的
是这是脱颖而出的机会。而且这些比赛 的机会都
是代表单位去的，甚至是代表你所在的 区，你
所在的县，你所在的市，去比赛的，在这些 比赛
当中露脸的机会就不一样了，你得了第一名， 你
得了一等奖，整个单位都知道你了，甚至整个系
统都知道你了。

比如说你在组织系统、公安系统、纪检系统，可能
有好几百人、上千人都因为这个比赛知道了你，了
解了你，有的单位还会搞公众号，把这个比赛选手
比赛的视频、照片、信息发上去，多威风，是不是
你一下子就出名了，你的存在感就不一样了，之前
可能很多人都不知道你的名字，更不用说知道你的
能力了，当然了，我必须跟大家说清楚，不是说你演讲比赛拿了第一名，业务竞
赛拿了第一名，领导就一定会提拔你，没有这么必
然的联系，对于普通人看到机会要像饿狼一样，要
有这种嗜血的敏感性对待机会，主任认为凡是这种
代表单位去市里面，去省里面，去整个系统比赛的机会，单位的一把手都会非常重视，因为领导都是要面子的，他都是要政绩的，比赛的选手代表的就是你所在单位的水
平，也就是说如果你不行，就代表你所在的单位不
行。

作为年轻人，尤其是作为有仕途追求，想进步的年
轻人，体制内的各种比赛，绝对是你刷存在感露脸
的非常好的机会，所以对于这种机会一定要认真的
去准备，认真的去争取，认真的去表现。

五、抓住重要机会做好汇报、适当刷存在感。

关于汇报工作的重要性，其实我在以前的付费文章
中里面也说过很多次我可以这么说，不管什么级别
的领导，其实都非常重视在他的领导面前汇报工作
的机会。在体制内也好，在体制外也好，在重要场
合说话的机会对于一个普通人来讲是非常非常难
得的，你抓住一次重要场合说话的机会，胜过你辛
辛苦苦干三五年的时间，有很多人包括一些大领导
都非常重视在更高一级别的领导面前汇报工作的
的机会，非常重要的一点就是认真准备。

其中一个案例是这样子的，就是有一个厅级领导，
他要去向省级一把手汇报工作，而且不是面对面的
汇报，是在视频会议上汇报，实际上就是做一个全
省的经验交流。因为单位里面某项工作做得比较
好，被省领导看中了，说是要在视频会议上进行经
验交流，然后这个稿子都是秘书提前写好了，领导
也反复的修改过，但这个厅领导还是非常重视。早
上7点钟左右，我就去办公室换衣服，然后就看到
这个厅领导的办公室他的门是打开着，他拿着稿子
在念，因为我的办公室跟他的办公室离得非常近，
就是对门了，他就在这里念稿子，而且是用自己的
语言来表达，并不是照着稿子念的，说实话，我看
到以后还是被这种态度激励到了，有点像什么?有
点像我们当初高考的时候一样，本来你说这么大的
领导，这种场合的发言也经历了很多了，驾轻就熟，
轻松就可以应对了，你说有必要这么有必要这么
拼?但是人家就是这么拼，就是这么重视，后来证
明这次汇报也确实很成功，还把单位的案例和和先
进经验编印了，发给各个市县去学习。不管多大的
领导，其实他都非常重视，抓住机会去展示自己，
去做好重要场合的汇报，千方百计的去做好这个汇
报。

再举个例子我们机关有一位处长，每次开会我听他
讲话，总觉得他讲的是所有中层干部里面说的最出
彩的。同时我也注意到了一
个细节，就是每次开会，别人都是拿个笔记本的，
他通常是抱着一摞材料进会议室，各种数据总结
材料、法律文件，依据什么制度规定，领导问到哪
个数据他信手拈来，如果他一时忘记了，他一查手
头的材料就能够回答上来，所以我就从他身上学到
了一点，就是在一把手面前汇报， 一定要做足12
分的准备，抓住公共场合汇报发言的机会，这是造势
最好的机会。

六、用媒体、平台的宣传造势，笔杆子是最好的宣
传手段。

为什么我们说老黄牛的人没出路，埋头苦看的人不
受待见?就因为你只知道干活，你不知道吆喝，不
仅我们工作要出工出力，你还要让自己大放异彩，
让别人看到你的光芒。我们做任何一件事情要追求
转化率，让它转化成领导看到你的机会。

比如说我们传统的媒体，电视、广播、报纸，还有
新媒体，比如说微信公众号、微博，视频号，还有
一个就是体制内的专门的渠道，比如说信息专报，
办公厅有自己的信息内刊，比如说政务信息等等这
些渠道，除了这些还有一个就是内部的渠道，比如
说这个宣传片、宣传栏，宣传手册。

我们打天下要靠笔枪杆子和笔杆子，其中这个笔杆
子就是宣传的阵地，这个阵地如果你不去争取，就
被别人争取了，胜利的果实就被别人窃取了， 一定
要抓住一些宣传的机会，上报纸、上电视、上新闻
广播、上这个信息专报，上宣传栏、上杂志内刊、
上内网。

如果你在县里，或者你在乡镇工作，你发表的文章
在全国性的杂志上，其实相当于是给你们单位长脸
了，你代表的不仅仅是你个人的，而是你这个单位，
这种事情很快就会被人知道的，很快就会传遍整个
单位 ，你不用担心说一把手不知道。

有人咨询我，自己没关系没背景，怎么才能获得提
拔?我说主动要求去写材料，因为不管哪个单位都
缺写材料的人。单位的大领导很忙的，没时间听你
会汇报，有时候你也没机会去向大领导汇报，但是
你写的东西他会看到，想要跨越多个层级给单位一
把手沟通，让他认识你、更进一步了解你，只有靠
写材料。

重要的材料一定要非常重视，因为这是打造你自己
口碑的机会，你辛辛苦苦干了大半年，当你埋头苦
干，当你苦于领导不了解你，正发愁怎么更好的刷
存在感的时候，你不妨用这个方式造势。 总之，至少要6分做，4分说，通过各种方式和途径把自己
做的工作宣传出去。
