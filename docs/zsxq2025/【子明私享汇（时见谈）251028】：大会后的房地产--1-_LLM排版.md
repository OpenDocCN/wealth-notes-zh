
# 大会后的房地产

从“造富神话”到“家的本质”,高歌猛进的楼市站在了拐点上。2025年10月23日，二十届四中全会公报发布，其中将房地产相关表述置于“加大保障和改善民生力度”部分，明确提出要“推动房地产高质量发展”。这也意味着，房地产行业将彻底告别过去的高增长、高耗能发展模式，走向高质量发展的新轨道。

不少的读者都在关心，大会后的房地产行业是否会迎来转变。今天，我们就借着大家的讨论，一起来揭开未来房市发展的神秘面纱。

目前跟我上海线下说的是一样的吧，见特朗普之前，必须要先完成内部集权(主席负责制),开完会马上开谈(何总已经去了)。可以顺便谈谈明年展望吗?明年按理说是最好的一年。虽然是“最好”,但还是会有明显的“温差”吧?

对，可以参考2016年，当时只有杭州市房价复苏了。那还会有16年之后的普涨吗?不会了。看来这次如果没跟上节奏，后面会变得更难。这次是要大幅拉开，所以国家会非常大力度的作保障。高速电梯变慢速电梯了，嫌弃慢还要丢人下去。我们客户的判断是2026、2027会比较稳定，他们比较担心2028年的形势。

这跟我的判断是一样的。如果我没有混上梯云纵的机会，就准备清盘大部分生意了。如果你是非常传统的生意，那么就应该学习十年前的老登，利用这个化债的机会抛售资产。2015-2018是老登们跑路的最后时间。十年前的2015就是国家把房价吵起来，给老登跑路。到了2018-2019房地产就进入含科量时代了。上涨的资产很多就不是老登的了。老登跑路，就那么两三年的光景。好，这次可以好好准备一下，从2027开始，做一些资产清盘。房子先别急着卖吧?说不定跌个10年之后可能又是加0的上升。

房地产的时代已经过去了，房地产背后的利益集团已经被消灭了。人口是最确定的宏观变量，Al机器人时代来了，大规模刺激人口生育已无必要，人口的供给与需求两端都已经确定。以东京为例，东京的房价能起来是因为财阀永存，你有钱你可以和小泉家一个小区，所以房地产“含权量”大的还能起来。而我们“含权量”大的房子都是配给的，和普通人财富无关。中国极少地区的房价，还有上涨空间，但是把房地产作为融资杠杆和化债方式已经结束了。

杭州的房子是不是多少能保值。我感觉即使回到三十年前，不说贬值，涨的还是三十年前的房子吗?没有地折价三十年的房子算啥。至疫情以来，全球在疯狂地发钱，房地产哪怕保持微涨，其实就是在贬值。可是这一轮的房地产基本上都已经是最好的地段了，微信pep458下一轮房子不太可能有更好的地段了，这个地段溢价在?未来的房地产如果涨价，是消费品涨价的逻辑。有钱人喝茅台一斤几千块钱，没钱人喝散酒一斤几块钱。有钱人吃日本和牛，没钱人吃合成牛肉。要是说茅台与和牛涨价，逻辑是通的，但是如果你说酒和肉要涨价逻辑是错的。

那也就是先进生产力的城市价值就是茅台和牛，其他就是散酒合成肉。未来本质上除了绝版地块外，只有独享土地的别墅才有价值。棚改的房价是城乡二元结构中城市对乡镇的剪刀差，也是权力对非权力的二元结构中的收割效应体现。现在已经割不动了，没有新的收割对象了。暴涨的房价过程中，最大受益者仍是权力阶层，无论是个人还是政府。在大会后，政府明确了房地产行业的定位：强化民生属性，推动高质量转型。住房的保障作用将代替投资作用，作为一项“民生商品”,走在未来的发展舞台。房价飞涨的时代一去不返了，我们也将积攒能量冲击新的发展机遇。
