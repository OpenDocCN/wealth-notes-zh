
# 卢5.10日新闻解读

## 2025年5月10日 新闻解读
1、美国的普雷沃斯特当选天主教新任教皇——“利奥十四世”。

综合多家媒体：梵蒂冈西斯廷教堂的烟囱在当地时间8日傍晚冒出白烟，众枢机主教已在秘密会议中选出美国枢机主教罗伯·方济各·普雷沃斯特为第267任教皇，名号为“利奥十四世”。69岁的美国枢机主教普雷沃斯特(Robert Francis Prevost)当选新任教皇，这是美国枢机主教首次出任教皇。普雷沃斯特出生于芝加哥，于2023年晋升为枢机主教。来自世界各地的上万名信徒聚集在圣彼得广场等候，白烟升起时，人群挥舞着自己国家的国旗欢呼雀跃、掌声雷动。

此次参与选举新教皇的枢机主教有133人，创下历史新高。上次选举时，共有115位枢机主教参与。这也反应了前任教皇方济为扩大天主教在全球影响力做出的努力。新教皇虽然出生在芝加哥，但父亲是法国人，母亲是西班牙人。而他在成为新教皇后向广场聚集的信徒致辞时，使用了意大利语和西班牙语，而非英语。某种程度上也说明了他的倾向性。与此同时，他虽然是美国人，但没有在美国的天主教教会晋铎高级教职务。反而有很长时间在秘鲁教会工作的经验，他的晋铎也是在秘鲁完成的。成为秘鲁奥古斯汀修会的高级神职人员。2023年后被前任教皇调往梵蒂冈，担任主管全球主教事务部门的枢机主教。由于常年在秘鲁传教，他也持有秘鲁国籍。换言之，他是美国、秘鲁双重国籍的教皇。

他早年毕业于美国宾夕法尼亚州的一所大学，主修数学。理性与感性同时在他身上汇聚。另外，这位教皇对美国政治有一定的参与。他曾是注册的共和党人，这也是当白宫得知教皇来自美国后，特朗普第一次时间表示祝贺并认为这是美国的伟大。不过，不知道特朗普是否知道新教皇在2019年曾公开表态对前者的不满。就在两周前，当美国副总统万斯访问梵蒂冈之际，新教皇也曾对万斯所秉持的移民政策大为不满。更深一层看，此番新教皇来自美国，据说是拜登向方济各推荐过此人。一方面，民主党在宗教议题上具有更大的影响力。相较于特朗普所拥抱的基督教福音派更多元、灵活。另一方面，梵蒂冈从来都与政治权力保持良好的合作关系，两者互为利用。否则天主教也不可能存活到今天。新教皇来自美国、活跃于南美、登基于欧洲，作为拥有14亿信徒的教皇，此等经历着实具有全球性、多元化。

2、戴维·珀杜宣誓就任美国驻华大使。

当地时间5月7日，戴维·珀杜宣誓就任美国驻华大使。美国总统特朗普出席就职典礼。戴维·珀杜1949年12月出生在美国佐治亚州，是一名共和党人，曾于2015年至2021年担任佐治亚州联邦参议员。现年75岁的戴维·珀杜在美国政商界有着丰富的资源，也被认为是最富有的国会议员之一。他曾在世界知名运动品牌“锐步”担任高管，也曾担任零售公司莎莉公司(Sara Lee)的高管，从事公司管理和国际商贸的时间长达40年。他在新加坡和中国香港居住多年。据《华盛顿时报》报道，在达乐公司任职期间，他积极推动该连锁商店进军中国市场。也为特朗普的两次大选出力甚多，更是参议院里最支持特朗普的参议员。相比于拜登任内，为了确定驻华大使人选花费了9个月时间。特朗普的驻华大使人选在上任100天左右就搞定。一方面是前任大使伯恩斯是典型的民主党人，与特朗普完全不在一个频道。另一方面，特朗普本人对于中国的了解，恐怕只能用是否有在东亚工作、生活的时间长短作为标准，进行挑选。戴维·珀杜的中文名字叫庞德伟。他之所以用“庞”作为自己的姓氏，据说是专门找风水先生测算过，只有这个姓可以“压住中国龙”。从一个侧面也说明，美国这些所谓政商精英对于中国的傲慢态度，宁可听信江湖术士的话，也不愿意真正了解当今的中国。虽然外界对驻华大使有颇多期待，但是以美国现有外交体系运作的方式，驻外大使基本上只是礼节性代表。不但无法进入决策层，甚至在美国国务院内部地位都不高。一方面，许多驻外大使，都是美国总统大选的政治回报。根据政治捐款数额的高低，决定出使国家的级别。另一方面，美国的驻外大使不少都是商人出身，并不能真正掌握外交使团的权力。多少有点吉祥物的意味。反而是大使馆馆长、CIA在大使馆内的情报主管、公使等掌握实权。上一任驻华大使伯恩斯，最喜欢坐着中国的高铁、飞机，边读书边游览中国，顺手在社交媒体上各种diss中国。不知道新来的这位大使会不会有什么新花样。
