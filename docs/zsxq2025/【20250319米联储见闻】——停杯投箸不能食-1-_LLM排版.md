
# 20250319米联储见闻——停杯投箸不能食

华尔街日报今天发了一篇文章，称兔子leader对长和计划出售巴拿马运河港口的交易感到愤怒，其中部分原因是长和此举事先没有征得北京的批准，甚至没有提前知会中央政府，知情人士还透露，兔子政府原计划用巴拿马运河港口问题作为与川普政府谈判的筹码，结果却因长和此举而失败。领导感到愤怒的是因为觉得李JC的举动违背中方利益。美国智库战略与国际研究中心 (CSIS) 学者指出，长和退出巴拿马港口业务，是美国在与兔子全球竞争中取得的重大胜利。这种解读无疑会加剧美国内部的鹰派声音，并鼓励其进一步在其他战略要地对中资企业施压。从美国地缘战略角度看，通过贝莱德这样的美国资产管理公司控制巴拿马战略港口，无疑有助美国增强对巴拿马运河这一关键国际航道的影响力。

外交部方面的态度是不评论具体商业交易，但反对在国际经贸关系中滥用胁迫、施压的手段。中方还重申从未参与运河管理运营，也不插手运河事务，暗示美方的兔子威胁论缺乏事实依据。对兔子来说，巴拿马运河是重要的国际航道，兔子是运河第二大用户，占运河货运量的约21%。长和出售巴拿马港口资产给美国财团，可能增加兔子船只未来通过运河的不确定性和成本。

华尔街日报一直以来会有一些从兔子内部高层释放出来的所谓内部消息，也有一些所谓深喉信源，华尔街日报甚至尝尝被用来被内部人当成某种舆论工具反过来敲打高层，这篇报道意思也很明确，交易令领导难堪和挫败，更令领导沮丧的是，北京方面似乎也并没有什么有效的手段可以阻止交易，至于所谓拿巴拿马运河跟川普谈判，我觉得兔子方面不会有这个想法，但是最起码不希望巴拿马运河成为美国拿捏兔子的筹码，虽然领导和李JC的关系一直不怎么样。

没想到老李在这件事上会做得这么绝，毕竟不仅仅是巴拿马两个港口，而是全球40多个港口。现在的问题是，兔子似乎缺乏足够的手段阻止交易，因为资产位于内地及香港以外，所以交易各方均有信心完成交易。长和为了避免应对媒体，甚至取消了新闻发布会，大有一条道要走到黑的感觉。

对于此时出这个事情，尤其是贝莱德参与其中，我总觉得跟2月21日蒋超良被处理隐隐有些联系，这次李JC卖港口不但卖得方式非常吊诡，隐约还有给某人一点颜色看看的意思，早不卖晚不卖，蒋刚被处理，这事就快速成了，有没有可能这背后需要某些妥协才会让这笔交易被暂时停下来，任何摆在台面上的矛盾，必然是有台面下的利益要谈，老李和贝莱德都不是真正在乎钱的主，他们不过是更大权力阶层的白手套，这笔交易背后要博弈的一定是更大的政治利益和权力，钱在这场交易里恰恰是最不重要的。

2月18日，央行开民企座谈会，马云成座上宾，领导显然有点胜利者姿态，感觉跟资本已经打成一片，三天后蒋超良下马，蒋指向谁不言而喻，很快3月初，贝莱德和长和就开始做事情，并快速给领导制造了一个巨大的难题，前脚刚跟民营企业家其乐融融，后脚就要来干预企业的交易，眼下正值兔子稳定外资和民企信心之时，长和这一次不仅给兔子制造了现实难题和未来的风险，也跟当下的舆论战和信心战重重一击，这样精准的打击如果没有内部的一些呼应，不可能操作的如此精准。要知道清华经管委员会里可是长期为贝莱德的高层留了位置，2018年10月，清华经管顾问委员会召开会议，新增了两位委员，其中一个是当时兔子的新任央行行长易纲，另一个就是贝莱德的主席larry fink，也就是这次跟老李谈交易的贝莱德老大。而当时参加会议的就有老朱和老王。七八年前，大家可都是亲密到在小圈子里称兄道弟的，七八年后，前脚大管家被扣了，后脚贝莱德就来买资产了，你要说这里面只是老李的问题，我就只能呵呵了。

回溯到2013年，2019年，数次香港的街头运动，背后的资金链条都可以追溯到李和某个团体身上，这次老李做得这么绝，恐怕也不仅仅是英美的压力。老李，不过是更老一点的白手套，没什么特别的。

10年前，老李大量抛售兔子资产时，其实也是国内要给领导上眼药的时候，那个时候可不仅仅是长和在跑资产，海航，安邦，明天系，那个不是在大量转移资产到海外，股市汇率一顿猛杀，至今都有人说是金融ZB，只是因为老李有钱有的比较早，大家似乎觉得他可以自己说了算，但是从这次巴拿马运河港口的交易来看，实际上，老李之所以可以买下全球40多个港口，就说明他不过是帮权力洗钱的高级包衣奴才罢了。1999年长和收购巴拿马港口的时候，美国佬和现在的兔子人一样一片哗然，甚至叫嚣必须终止交易，但最终克林顿政府的审查结果是，长和与兔子政府毫无关系，收购指令来自英国方面，变相表明老李是英国政府的白手套，香港版东印度公司，所以才能顺利收购巴拿马运河，这些年，老李买下全球这么多港口恰恰也说明这个事实，他不过是权力的资产代持人罢了。这场交易涉及的不是纯粹商业逻辑，美国的强势施压、兔子对战略性的港口控制权，后续可能带来的航运、贸易甚至安全影响。

长和风波发生后，局面与当年滴滴赴美上市被叫停有很多相似之处，滴滴当年涉嫌抢跑赴美上市，这一回长和则是在没有得到北京支持下和美国财团谈了交易。最后滴滴以退市收场，滴滴那次退市也导致美国资本与兔子权力出现巨大且难以修复的裂痕，好多积怨是这些年在博弈过程中逐步撕裂和积累下来的，但是相比于上次兔子管理层有大量手段对滴滴下手，这次内地想要干预交易台面上的办法相当有限，要的是台面下的妥协和博弈，对于今年开局打得很顺风顺水的领导来说，这一把似乎也是在提醒领导，没那么容易就能搞得定这内内外外。

就在全网追讨李JC之际，领导跑到贵州去视察了，似乎每次中美出现重大博弈的时候，领导都会以在地方视察的方式露个脸，似乎要告诉外界，这些事都不是事，我压根不在乎，我只在乎国内的事，最近欧洲邀请领导访欧也拒绝了，二把手去。而每次处理这些看似外部的矛盾，最后都以内部斗争为终局。

到目前为止，除了港澳办转发的大公报时评，北京官方没有明确表态，内地主流官媒也没有加入讨伐，显然高层不但清楚发难的是谁，目的是什么，以及解决问题要付出什么代价，可能也还在谨慎评估后续处理。现在要看的是，官方会以怎样的方式介入？会不会直接叫停，基于什么理由和原则，能不能让外界信服？如果这件事真的跟内部斗争有关，是内外勾连共同发力的结果，大概率此事会暂缓推进，值得注意的是，川普方面最近提出了要跟兔子和加拿大开启贸易谈判，是不是也准备拿这单悬而未决的交易作为谈判筹码逼兔子就范。

长和此次出售港口资产，除了可以预见的贸易风险和隐患，还将引发一系列的涟漪效应：首先，它给其他在海外有重大资产布局的兔子企业敲响警钟，迫使其重新评估海外资产的安全性与政治风险。未来一段时间，可能出现更多兔子企业调整海外投资布局，尤其是在美国及其盟友影响较大的地区。其次，此次交易也可能鼓励美国及其盟友进一步对兔子企业施加政治压力。如果此次美国通过市场机制成功逼退长和，今后美方可能会在其他敏感资产上如法炮制，没完没了，对兔子企业形成更大的压力与潜在威胁。

从更广义的层面来看，长和的资产出售进一步暴露出兔子企业在海外投资中的制度性脆弱性，面对这样重大的危局，兔子竟然找不到合适的制度性工具去阻止风险发生，规避损失，这对于想要走向星辰大海的兔子来说，是类似当年GPS被封时刻一样的困局，你想要走向蓝海，就要准备好一切乘风破浪的工具，否则迎接你的不是星辰大海，而是狂风暴雨。这对于想要在全球布局资产和发展贸易的兔子来说，是一个重大的制度性课题，要如何建立能维护自己贸易利益制度性安全措施，并确保这些政策工具的可执行性和权威性。英美在这方面深耕了近两百年，不吃亏就想占便宜是不可能的。长和的决定也可能影响其他兔子企业的海外布局策略。在地缘政治风险上升的背景下，更多兔子企业可能会重新评估在政治敏感地区的投资，转向风险较低或与兔子关系更稳定的地区。此次事件给兔子的警示意义是明显的：在当前国际环境下，商业与政治的界限日益模糊。无论是国家还是企业，都需要深化对地缘政治与经济关系的认识，建立更全面的风险评估和应对机制。兔子要加强对全球关键基础设施和战略通道的研究，建立预警机制，及时发现潜在风险，为企业提供必要指导和支持。政府应加强对海外兔子企业的保护和支持，尤其是在政治敏感地区或战略要地的企业。这包括完善外交保障机制、提供法律支持、建立风险补偿基金等。同时，鼓励兔子企业提升风险管理能力，建立专业的地缘政治风险分析团队，优化投资决策流程，灵活调整海外布局。

从更长远的角度看，人类命运共同体并不是一句空话，而是依靠对秩序的强势支撑，对善的有效鼓励，对恶的严厉惩罚，对利益分配的主导权来推动认同的。如果这一次，兔子真的吃了亏，一定要记住，没有白吃的亏，下一次要百倍的拿回来，让李JC这三个字与这个大坑永远的钉在错题本上。
