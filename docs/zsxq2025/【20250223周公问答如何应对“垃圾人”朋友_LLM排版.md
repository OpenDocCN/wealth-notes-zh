
# 如何应对“垃圾人”朋友

匿名用户提问：
老师，先抱歉，就一个很无聊的问题提问，但是我真的觉得好烦好烦。我一个女性朋友，十多年的朋友，但这几年变得非常自恋，每天聊天都在说自己很优秀，自己很棒，很为自己自豪。她以前外貌形象不佳，经历过大动后也是比较自强自立，工作也渐有起色，年前挣了一二十万。
我有自己的生活和世界，但是她几乎每天都给我发消息，要么说她在网上社交软件对异性们的评头论足，要么说她又感悟了什么，然后就是“我真为自己自豪”，我真的完全不想看她的“小作文”。她真的超级自恋，有时候我努力聊下我自己的事，因为我也有我的烦恼和困扰，但凡聊起来，她就敷衍两句，说我的情况很特殊没法解决，又开始说她如何如何。我觉得我的能量都被吸取了。
时不时的约我，但是出来从来都是在巴拉巴拉她如何如何。我是一个不太爱讲自己的烦恼和困扰的，因为我知道说了也没用，大事没有解决办法只能慢慢来，小事基本能解决。她天天说，要爱自己，不要陷入别人的框架，我承认她说的这些道理没问题，可是这种无聊的废话我觉得很无聊。偶尔敷衍还行，每一次都是这些东西，我的工作本来就是繁杂的，我真的受够了。我有时一天不回她消息，她还会问我怎么不回消息。我又不好跟她直说我现在很讨厌她，也不好提出她的问题去改变她，我只想让她闭嘴。
前年因为一次关于她前男友的问题是没有按照她的心意来回答，她直接拿我的一些伤疤来攻击我，然后大半年我和她互相没联系。后来又联系上了，她反而变本加厉的自恋。虽然我知道这个提问要过很久才能回应，但我相信，等老师回应我的时候，她应该还是这个样子。我在一些地方搜索相关情况，有很多相同经历的人，但是都没有解决办法，只能等待老师回复。真的是被烦的想拉黑她了。

公周解析：
以前我们也讲过垃圾人的概念，这位女生就在其列。思想认知如何偏颇，人格如何带有缺陷，是她自己的问题，这里暂且不论。在交往中毫无相互尊重、换位思考的意识，始终自我为中心，一味宣泄个人情绪，把你当做呼来挥去的免费出气包和情绪垃圾桶，显然是违背了朋友关系的基本属性。对这样的人，难道你要一直忍气吞声、任凭自己的能量被她吸干？这也太卑微、太懦弱了。
别说是十年的朋友，就算是血缘相通的兄弟姊妹，也不可能长此维持下去。所以逻辑是很清楚的，问问自己，她是个合格的朋友吗，还愿意跟她做朋友吗？如果愿意，那就坦诚交流，表达自己的不爽和不满，让对方回到恰当的边界，继续友好交往下去；如果不愿意，那就像前年那样果断静音、拉黑、忽略，且不再给她恢复的机会。为防止她找茬、作祟，也可以先采取冷处理态度，少回复、不回复，等她质问时亮底牌，每天真的太忙太累了，不想再听她说这些事，请不要再联系了。
然后随便她怎么说，过两天拉黑就是了。任何一段人际关系都是有保鲜期的，每个人都是彼此的过客，没有必要片面追求婚姻、爱情、友谊的长度，让自己忍受那些不必要的痛苦。友好分手，各自别过，对双方都是解脱。
