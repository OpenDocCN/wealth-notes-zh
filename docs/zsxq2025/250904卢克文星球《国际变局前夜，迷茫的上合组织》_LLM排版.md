
# 国际变局前夜，迷茫的上合组织
250904 文/卢克文工作室嘉宾 云梦竹

当前国际局势呈现出一种纷繁复杂的状况，俄罗斯的特别军事行动、伊朗与以色列的持续冲突、印巴空战，这些改变地缘的危机背后，是原本的世界秩序在各个角落悄然瓦解。礼乐征伐自天子（美国）出的时代已经离我们远去，礼乐征伐自诸侯（多极）出的时代正在到来。

就在这个时代，9 月 1 日，各国代表齐聚天津，参加上合组织峰会。本轮会议可谓众星云集，但令我感到意外的是两人，一人是土耳其总理埃尔多安，土耳其连上合组织成员国都不是，也来到了此地。印度总理莫迪也在会场，要知道，巴基斯坦总理也受邀前来。也许除了上合之外，再也没有国际会议，能让这对老冤家一同到场。其实吧，在大家热烈捧场的背后，却是上合组织难以诉说的尴尬。

在旧秩序逐渐崩塌，新秩序尚未建立的混乱时期，上合组织的意义到底在哪里？

## 上合的变化
上合组织本质上是打着反恐旗号的反颜革组织。其组织的核心成员就是中俄与中亚五国，意义就在于确保中亚地区不变色，防止西方对中俄腹地的渗透。最近几年，随着美国在中东地区军事部署的减弱，美国对中亚的渗透逐渐减少。在失去了中东以及阿富汗据点之后，中亚已没有颜色革命风险了，理论上，上合组织的历史使命已经完成。

但各国得益于上合这么一个良好的沟通平台，不愿意就此放弃，都希望上合能够在国际社会中起到更大的作用。在此期间，上合进行多次扩张，新增了印度、巴基斯坦、伊朗、白俄罗斯等国，甚至连土耳其都成了对话伙伴。随着越来越多的国家加入，上合组织也变得混乱，这种混乱，在本轮会谈中体现得淋漓尽致。

首先是印巴。印度在上合发言中，指责某些国家支持虔诚军，导致了克什米尔地区的恐怖袭击。巴基斯坦则批评某个南亚国家支持俾路支解放势力，发起了针对巴基斯坦的袭击。最终上合的处理模式是各打五十大板，既谴责了恐怖分子对印度的袭击，也抨击了针对巴基斯坦的袭击。

在双方新闻媒体中，印巴都在宣扬自己的胜利，认为自己在上合得到了支持。但实际上，上合组织的表态，不过是例行公事罢了。

伊朗与土耳其关注点又有不同。伊朗最为激进，2023 年甚至希望上合各国签署互不侵犯条约，将上合升级为一个军事同盟，但这个观点没有得到中俄的支持，随后伊朗对于上合的热情就消退许多。土耳其则是希望借助上合这个平台，与中亚等突厥国家加深联络。某种程度上，现在土耳其是上合最不欢迎的国家，但也不能直接将其赶走。在本轮会议中，土耳其与伊朗都把加沙问题作为重点问题抛出，让上合给以色列施加压力。

俄罗斯则在本轮会议中进一步阐述了自身对于俄乌冲突的看法，希望得到来自上合的支持。可以说，各方的需求在此处汇聚，其诉求已经远远超出了中方解决问题的边界。

## 上合似乎越来越没用，变成了各方势力的许愿池，但上合真的没用吗？

在本轮会谈中，印巴双方都得到了自己想要的，上合组织看似在印巴恐怖袭击问题上各打了两大板，却给了南亚一个清晰的信号：中国对于南亚的需求是稳定。

对于巴基斯坦而言，其最惧怕的是因为美巴双方的关系紧密，使得中印关系好转。印度则惧怕中国进一步投资巴基斯坦，导致南亚局势走向失衡。中方各打五十大板的态度看起来是在和稀泥，实际上表达了一个关键态度，中国不准备在南亚有所作为，既不会支持印度，也不会往死里扶持巴基斯坦，这个局面恰恰是印巴双方都能接受的底线。

在中东问题上，中方支持巴以两国论，并对以色列施加了压力。虽然这个声明只是口头上的，但给伊朗和土耳其一个准确的信号，中方不会为了与美国缓和关系，在中东问题上出卖伊朗和土耳其，虽然以色列竭力想与中方交好，但中方依旧从国际舆论角度，支持巴勒斯坦。

这个信号将会成为伊朗与土耳其中东政策的重要参考。

同样地，在俄乌问题上，中方表达了推动俄乌和平的态度。这意味着，中方短期内不能接受乌克兰从国家层面的消亡，但和平依旧是有代价的，如果俄罗斯与乌克兰的和平协议包括领土割让，中方不一定反对（看各方的出价和具体情况）。

从这个角度来看，虽然中国在上合组织中没有对各方诉求表达支持或赞同，但中方给这些问题定了性，并清晰地指出了自己的底线。在当前国际社会，这种准确且清晰的态度已经很罕见了。

美国针对全球的政策一天一个样，导致各国在政策制定时时刻需要考虑背叛的风险，而这些不确定性，在上合是没有的。这个行为符合中方的一贯观点，即在这个不确定性的世界，提供确定性。

哪怕中方的表达不能如各方心意，但确定的态度本身，就是各方后续政策制定的重要基石。简单来讲，无论我们对于一件事支持还是反对，大家都能接受；但美国那种先支持后反对，或者先反对再支持，就会害死人。

上合虽然不是一个国际同盟，但它又确实接纳了所有反西方国家，所以在客观上，上合已经具备了成为反西方同盟的潜质。各方愿意在这个平台说出问题，本质上也是在寻求解决办法，而中方则在这种场合处于超然地位。

一方面，中方规避了盟友所需要承担的义务，防止出现被拖下水的连锁反应。另外一方面，让中方具备了与西方持续博弈的资本。

打个比方，如果未来欧美持续对台军售，中方完全可以以上合为框架，向伊朗和俄罗斯提供军援与军售，通过迂回手段达到制裁西方的目的，以此迫使西方在关键问题上让步。

今天伊朗提出加沙问题，也许中方只能糊弄过去，但如果在台海危机中，伊朗再次提出类似问题，就可以成为中方介入中东、进行大范围博弈的缘由。乌克兰问题也是同理，大家把问题说出来，中方才愿意介入。中方愿意不愿意介入是我们的事情，各方说不说，就是他们的事情了，现在看来，情况还是很好的，至少各方愿意说出来。

伊朗和俄罗斯等国家除了聊到自身地缘焦虑，也努力在上合推动自己的经济议程，其中普京提出希望上合发行联合债券，并推动统一结算。这个论点也得了伊朗的支持，在上合会议中，伊朗总统佩泽希齐扬再次兜售去美元战略，希望上合建立非美元的国际货币体系，开发包括中央银行数字货币在内的共享数字系统，并创建多边互换基金以支持面临制裁或流动性危机的成员国。这里，面临制裁或流动性危机的成员国就是指伊朗和俄罗斯。

从这里我们可以看到，伊朗和俄罗斯在面临美国的制裁之后，希望与中国一道推动去美元化战略，哪怕伊朗和俄罗斯非常清楚，上合内部“去美元战略体系”建立后，人民币会成为最大赢家，他们也在所不惜。

可以说，上合组织本轮会议，有点黄袍加身的味道，各国都急切希望中国站出来主持公道。俄罗斯和伊朗甚至愿意支持以人民币为核心的贸易金融霸权。毕竟当他们被西方体系排斥后，他们甚至比我们更反西方。

但中方非常清楚，这个战略时机还未到来。

而这也是现在上合组织松散的原因，我们不希望上合成为北约那种军事同盟，去负担远超自己实力的军事义务。但我们又希望上合存在，成为未来博弈中的一枚棋子。

所以让上合保持今天的状态，对我们而言，反而自由度最高，也最为有利。
