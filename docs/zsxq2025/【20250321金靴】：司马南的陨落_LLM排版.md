
# 司马南的陨落

税务问题那就不说啥了，属于一锤子下去，绝对连哼都哼不出来。认命吧，司马大爷，自己作的。不过从宏观来看，依旧是一桩明确的信号。经济罪也好，道德罪也罢，在上层社会都 不是罪，因为那里只有一种罪：政治罪。无论如何，这一轮看来是要誓死提振民营经济了。去年秋天民营经济促进法征求 时，他的全平台矩阵账号被禁言，妥妥的被祭旗。

对于民营企业有两大象征性人物：外向的标志性角色：马云；内向的标志性角色：司马南。这两位棋子如今双双得到与2021年时截然相反的行政待遇，被调转棋盘，颠覆了身 位。这桩信号发出之后，各地方政府会更松开手脚、大干快上“当好店小二”。
