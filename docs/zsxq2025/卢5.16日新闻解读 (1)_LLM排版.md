
# 卢5.16日新闻解读 (1)

## 2025年5月16日 新闻解读
1、美国与卡塔尔签署3160亿元经济与军事合作协议。
美国与卡塔尔当地时间14日签署总值超过2435亿美元的多项协议，包括卡塔尔从美国购买波音客机和武装无人机等项目。美国总统特朗普星期三续程到卡塔尔访问，这次的中东之行主要是要争取波斯湾国家的投资。白宫声明说，卡塔尔航空与美国签署总值960亿美元的合同，购买多达210架波音787“梦想客机”和777X客机、价值近20亿美元的MQ-9B武装无人机、价值约10亿美元的反无人机武器系统等。两国企业还签署多项涉及能源基础设施和高科技发展的合作协议。

其中卡塔尔此番采购210架波音客机，对处境艰难的波音公司而言，无疑于天降甘露。一方面，可以维持其在全球客机市场的份额。虽然全球市场以波音和空客为主，但波音在过去几年，发生的一系列飞机质量问题，让外界不得不有所调整。另一方面，卡塔尔航司是世界一流航空公司。一下子采购如此之多的大型客机，除了继续巩固卡塔尔航空公司在全球航空市场的份额之外，进一步将多哈打造为全球航运枢纽是该国的战略目标。

不过，这次特朗普访问中东三国，主要是为了争取三国对美投资。否则他也不会把马斯克、黄仁勋、奥特曼等一众高科技大佬带上。目前看沙特承诺投资1万亿美元，阿联酋宣布参加美国国家AI项目，投资额至少3000亿美元。单纯看金额确实相当可观，加上三国还采购了数千亿美元的军事装备。着实可以让特朗普在支持者面前“赢麻了”。

然而，从特朗普首个任期内，富士康老板郭台铭也曾宣布在美国投资设厂，还邀请一众美国政府高管出席厂房奠基仪式。数年过去了，根据当地媒体报道，至今厂区仍旧是一片空地。类似状况，会不会出现在三星、台积电等在美投资项目呢?至于，沙特、阿联酋等国的投资承诺，又有多少可信度呢?毕竟特朗普只有四年任期，而中东三国领导人的任期远比前者要长得多。

这些投资与采购项目，足够让特朗普在社交媒体上吹嘘好几天，足够继续维持自己在支持者群体中的人设。至于项目何时启动、执行等一系列“复杂”工作，没有人会在意。毕竟情绪上头，一切都是“正确”的。

2、土耳其正在敲定台风战斗机的采购计划。
根据近期国际防务媒体报道，土耳其正在敲定从英国采购台风战斗机的计划。采购案分三期，首批次20架台风T4和20架T5，交付时间分别是2027年和2028年。第二批次，从英国购买2架二手台风T1和12架C-130J军用运输机。第三批次：英国将无偿赠送28架原本计划销毁的T1，用于拆卸零件。其中T4将配备相控阵雷达，T5将配置全彩色数字化座舱显示屏。

这也是土耳其退出美国的F-35项目后，为了维持本国空军主力战机规模和水平的备选方案。一方面，土耳其自行研发的第五代战斗机项目--TF-XKaanx，2024年2月完成首飞，预计服役时间为2031年。然而眼下土耳其空军的主力战机仍旧为F-16系列，面对F-35逐步在其他北约国家服役的现状，安卡拉已经出现代差。倘若再不采购更先进战斗机的话，土耳其将难以应付日益复杂的地区局势。

另外，作为土耳其空军宿敌的希腊空军，美国于2024年批准向雅典出售F-35战斗机。另外，希腊为了维持与土耳其空军的力量均衡，采购了法国“阵风”战斗机。虽然，后者在刚刚结束的印巴空战中，出现战损。但不可否认，其依旧是一款出色的现代化战斗机。

随着“阵风”战机在希腊空军服役，以及TF-X项目仍需时间的双重压力下，土耳其只能寻求外购。只是土耳其所能选择的机型并不多。欧洲联合研发的“台风”战斗机被纳入视野。

台风战斗机也是一款鸭式三角翼设计的双发超音速战斗机。于1994年3月首飞，2003年8月开始服役。这是由欧洲多家航空公司联合研发的战机。目前在意大利、西班牙、英国、德国和奥地利等欧洲国家服役。沙特、卡塔尔、阿曼和科威特等国也曾相继采购。该飞机最高速度为2.0马赫，飞行高度约为19000米。最大航程3790公里。装备一台相控阵雷达、激光瞄准器、多普勒雷达等。外挂点位13个，总载弹量为9吨。可以搭载响尾蛇空对空导弹、AIM-120/132等空对空导弹、流行空对空导弹。还可以执行对海、对陆打击任务。

阵风vs台风，均为三角翼的4.5代战斗机。至少从技术参数方面打成平手。这也让土耳其不至于与北约盟国拉开太大的差距。
