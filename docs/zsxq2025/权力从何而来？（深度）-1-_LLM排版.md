
# 权力从何而来？（深度）

乔治·马丁的小说《冰与火之歌》里有这样一段描写：
瓦里斯问提利昂：
“提利昂大人，三个地位显赫之人坐在一个房间里
一位是国王，一位是僧侣，最后一位则是富翁，有个佣兵站在他们中间，此人出身寒微，也无甚才具
每位显赫之人都命令他杀死另外两人，请告诉我，
究竟谁会死，谁会活呢?”

提利昂说：
"国王、僧侣和富翁……谁死?谁活?
佣兵听谁的?这是个没有答案的谜语，或者说
有太多的答案，一切端视于手握利剑的那个人”

“然而他却什么也不是”瓦里斯说道：
“他没有王冠，没有金银珠宝
更没有诸神的眷顾，只有手里那把剑”
“但那把剑具有决定生死的力量”
“是啊……但既然真正决定我们生死的是手握刀剑
之人”

我们为什么还要假装承认国王拥有力量呢?”
我们讨厌症治但离不开症治
我们避免讨论它可又时刻关心它
症治到底是什么呢?
施密特认为，症治是划分敌我
马基雅维利说，症治是权力的争夺
毫无疑问，症治的核心要素是权力
权力是非常神奇的东西
任何人只要拥有一点点权力就能感觉到它的甜蜜
当我们能够将自己的意志强加给别人
让别人按照“我”的意愿来行事时
那种予取予夺的感觉能让人产生巨大的幻象和满足

如果一个症治学老师来讲课
有关于症治和权力可以滔滔不绝讲上好几年

权力是暴力的延伸
在我们开篇时所讲的谜语里
国王、僧侣、富翁……他们都想获得武士手中利剑的支持
造航母、建海军，研制先进武器，都是想掌握利剑
谁的拳头越大谁就越占上风
但权力并不仅仅来源于暴力
古今中外的无数事例告诉我们
仅依靠暴力是无法维持权力运转的

那么富翁呢?

一个富可敌国的人有没有权力呢?
马斯克、扎克伯克、比尔·盖茨、贝索斯……这个名单还能无限拉长
他们毫无疑问拥有巨大的权力
甚至于他们的一句话
一个行动就足以改变某些症治进程

国王想要掌握暴力机器
拥有军队和武器
也需要金钱的供养
但金钱在权力面前也是软弱的
就像《纸牌屋》里的经典台词说的那样：
“金钱是萨拉索塔的巨大豪宅
保质期就只有十年
权力是古老的石砌建筑
能屹立数百年”

僧侣有没有权力呢?
《大宪章》是教宗与国王及封建贵族签订的权力契约
神权与王权的斗争就是半部人类史
就算在今天
也没人敢小觑宗教领袖的影响力
在古代中国，儒学就是柔性的法律
能够协助政府维持地方秩序
皇帝和国王呢?
为什么他们仅凭出身和王冠就天然拥有了权力?
皇帝的权力来源于祖辈通过暴力与人民定下的约

朱祁镇、朱厚照的权力来源于朱元璋
那我们可不可以联合武士、富翁与僧侣(士大夫)
推翻朱祁镇的统治呢?
当然可以但未必成功
因为朱元璋与当时的“中国人”建立了一种规则的一致
这种共识经过血与火的洗礼，是极难打破的

实际上，和大多数人想的不一样
弱者更需要强者而不是相反
在一个纯粹的丛林社会
强者能获得更多权力和资源
弱者只能接受最残酷的掠夺
也就是说
权力实际上是弱者们为了获得庇护而献给强者的武器

为什么很难打败朱祁镇?
因为他的祖辈建立起的秩序已经让人们形成了共识——在朱氏王朝的庇护下，我们能够获得足够的保护

《冰与火之歌》中
这个有关国王、僧侣、富翁和佣兵的答案
乔治·马丁是这样写的：
瓦里斯最后说道：
“力量存在于人心
人相信什么是力量
什么就是力量，不多也不少”
提利昂：
“这么说来
力量不过是骗人的把戏?”

“力量就像墙上的影子”瓦里斯说：
“但影子却能杀人，即便是矮小的人物
也能投射出硕大的影子”
因此，本质上
权力是在别人脑海中建立的观念、打下的思想钢印

三金哥为什么可以得到权力，当上曹县一把手?
他通过症治清洗掌握了军队(武士)
他掌握的军队能够获得资源的供养(富翁)
他提出的“先军症治”是他的意识形态(僧侣)
更重要的是，
他是红三代出身(权力的思想钢印)
多数人或者人们普遍认为
他所掌握的秩序可以为他们提供庇护

看到这里一定有人说
今天全世界的形势有些不同
大多数国家已经是珉煮帼家
再也没有皇帝、富翁和僧侣
人民正在当家做主
症治和权力的解释应该有所改变

但事实是换汤不换药
形式的改变不会影响本质
所谓的珉煮社会是怎么形成的?
是随着科技发展，工厂需要大量工人
社会需要发展经济
以及两次世界大战需要平民参与战争
双管齐下形成的新的社会制度

珉煮是什么?
就是人民大众当家做主
人民大众本质上是弱者的联盟
弱者总是愿意让渡更多的权力来获得更多的保障
这也是为什么所有的珉煮国家
最后都会走上福利主义路线的原因
“我”愿意用我手中的权力换取安全、养老、医疗……

对于个体来说，福利当然是好的
这不正是中国古人所追求的理想社会吗?
“大道之行也，天下为公……
故人不独亲其亲，不独子其子
使老有所终，壮有所用，幼有所长
鳏、寡、孤、独、废疾者皆有所养”

然而我们必须警惕的是
这是一把双刃剑
希特勒的核心政策就来自于用社会福利绑架大众

权力像弹簧，你弱它就强
你的权力一旦交出是很难收回的
既然权力的本质是观念
那么症治的本质呢?
大卫·伊斯顿的说法最被广为接受——
症治是社会价值的权威性分配
也就是俗称的分蛋糕

用美国大选做例子
本质上
红蓝双方的分歧是利益分配的分歧而不是意识形态的分歧

我们都知道
今天的世界秩序是雅尔塔体系的延续
1945年
二战的战胜国美、英、苏在克里米亚的雅尔塔
制定了利益分配的条约并形成了新的世界秩序
1991 年苏联解体后
世界进入美国主导的秩序并在此基础上进行了深度全球化

在这一波全球化过程中
一部分美国人的利益受到了损害
自上世纪八十年代开始一直到中国加入世贸组织
再到今天
传统美国工业区也就是媒体上常说的锈带工业区
流失了大量工作岗位

我们常常可以看到这样宣传：
过去一个普通工人、推销员可以养活一个四口、五口之家
他们除了日常用度外还可以买得起大房子
开上不错的汽车
但这几十年来
他们的工作岗位和收入日益减少
日子越来越难过

与之相对的是，在这波全球化中
东西海岸的金融、科技等行业都发了大财
相关从业者收入高得令人咋舌

然而
能让金融家、高科技产业的大佬们安心发大财的坚强后盾，是红区子弟组成的、强大的美国军队
是他们维持了帝国的运转和“霸权”

既然双方付出了同等的努力
何以分蛋糕时有大有小呢?
利益分配不均是所谓“撕裂”的根本原因
也是孤立主义路线与全球化路线的总冲突

在瓦里斯的谜语中
症治由国王、武士、僧侣和富翁组成
如果四者的关系不能保持平衡
症治就一定会出现问题
你总能在“症治理念”的背后看到利益的影子
无论它被进行了怎样的包装

假如你看到一个狂热的保守主义者忽然改变了立场或者相反
那一定是他所坐的椅子发生了改变

一些媒体喜欢过度渲染双方的分岐
其实很多政策背后各有深意
2023年美国最高法院裁定
宪法禁止大学在招生过程中考虑申请人的种族因素
这是一个争议已久的话题
政策向某个或几个特定种族倾斜是好的吗?
是公平、正义的吗?
如果从眼前来看
这似乎更符合公平竞争的规则
但从长远来看
如果我们不帮助某个或几个大的族群发展自己的精英
从而带动整个族群的进步
那么几十年后
这个族群可能会越来越边缘化
可能会成为一个很大的社会问题
从而影响整个国家的发展甚至安定

孤立主义者的背后也有深意
托克维尔认为
美国珉主活力的关键在于结社
他观察到
症治结社不过是美国五花八门结社中的一种
美国人无时不刻都在结社
宗教、政治、经济、道德……方方面面

他这样说：
“我最初听到美国有10万人公开宣誓不饮烈性酒
时
还以为这是在开玩笑……
后来我终于了解到
这10万美国人是惊于他们周围的人酒鬼越来越多
才决心戒酒的
他们的行动宛如一个大人物穿上一件朴素的衣服
以引导一般公民戒除奢华”

托克维尔认为：
“当你想到美国有不可胜数的实业在被人们共同经营
看到美国人到处都在孜孜不倦地推行某些宏伟的计划
而这些计划遇到一场小革命也会前功尽弃的时候
你又会不难理解如此多忙碌的人们
为什么没有给国家制造麻烦和没有破坏他们都受益的社会安定”

“美国人正是由于享有一种带有危险性的自由
才学会了可以尽量减轻自由所带来的危害的方法”

托克维尔的观察无疑是正确的
但在今天
深度全球化正在腐蚀这种结社的传统
让美国社会逐渐散沙化
根据美国企业协会的报告
近几十年来
美国人结社、参加集体活动的频率越来越低
这种改变对美国社会产生的影响是非常深远的

也许每个人都感觉到了
今天的世界正在发生着剧烈的变化
我们有幸或者不幸成为了历史的见证人
旧体系行将就木，新的秩序
一种多数人都认可的权力共识正在重新建立
这个变化我们也许在几年、十几年间就能看到
那么再远一些呢?
更远的未来会是怎么样的?
也许
能够改变现有症治形式的唯一途径是科技的发展
科技将再一次深刻的改变人类社会
想到到这里还不禁有些小雀跃呢
让我们拭目以待吧!
