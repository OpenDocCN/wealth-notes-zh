
# 普京只会跟特朗普谈

发布时间： 2025年5月20日

从2022年普京挥军乌克兰开始，全球的各方力量就都在进行停战谈判，并为战后的利益分配打得头破血流，各方也一度草签了数个版本的分配合约。

这些谈判都是民主党 转的，充分满足了民主党大金主们的利益，才会驱动拜登政府不断拿纳税人 暴总帕 克兰来硬抗普京，逼迫俄罗斯的息 1

去年开始，特朗普对介入俄乌谈判极度感兴趣，是为了从民主党的身上摘桃子，驱动贝莱德、西方石油公司等准备分蛋糕的跨国集团，把 缴税的对象从拜登团队换成特朗普团队。（贝莱德刚通过买港口纳投名状了）

这跟布林肯刚上任就约见阿拉斯加很相似，拜登新政府的诉求，是对已经签署的第一阶段协议以及后续的新协议，把利益输送带对象换成民主党阵营。（譬如，将农产品，石油，换成医疗，服务）

随着美国跟乌克兰签署了矿产协议，谈判的焦点变成了俄罗斯。

现在普京的最佳博弈策略是撕毁旧协议体系，把之前达成的默契和利益输送都赖掉，向特朗普提供他亟需的稀土与石油，抛开中间商，直接来向特朗普输送利益，踢掉贝莱德、西方石油公司等集团，节约天价的通道费。（东大也是这个思路）

因此，普京不出席会谈，把泽连斯基当小丑耍，是他只需要跟 亲爱的 特朗普同志进行亲切友好的单独谈判。

而特朗普如今看似风光，但其实也是焦头烂额，原本只想摘个桃子，但却被普京拽入了旋涡当中，进退不得。

不支持普京，之前夸下的海口就没办法兑现，更无法从东欧抽身，支持普京，就意味着抛弃欧洲与乌克兰，让朋友们的利益严重受损，这也会极大影响特朗普在关税贸易、美债的利率等重点工作推进。
