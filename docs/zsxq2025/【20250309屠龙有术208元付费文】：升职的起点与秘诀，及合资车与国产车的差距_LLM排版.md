
# 升职的起点与秘诀，及合资车与国产车的差距

## 前言
仅对升职加薪感兴趣的人有效，不想升职加薪的人，就不用了看了。本文披露的是从员工怎么当领班主管等低级职务的方法……不需要的人，可以不用付费阅读，请勿误解。

有人问：合资车仅仅只是套用了一个外国品牌吗? 实际技术水平和国产车一样么?

关于这个问题，是这样的。合资车之所以质量比国产车好，不仅仅是因为他有外国的品牌，而是因为合资车有远远超过国产车的品控。也就是，合资车的产品质量控制远远超过国产车。

其实，你只要打开合资车的引擎盖子，你就会发现，合资车很多零件，都是国内汽车零件厂商生产的。外国人并不生产所有的汽车零部件。外国人只是按照他的标准，选择符合他们要求的汽车零部件罢了 (其中很多汽车零部件是中国国内生产的)。

## 有人说，就这么简单?

实际情况当然复杂的多。然而，基本的原理，就是这样。就像国人都买外国牛奶，并不是我国的奶牛生产不出合格的牛奶。

废话，动物都是一样的，全世界各地的奶牛，都是牛，差距不会太大。我郭牛奶比外国的差，不是我郭奶牛不行，是我郭生产牛奶的人不行。

实际上，人们愿意多花钱买合资车，多花钱买进口车，是为了购买高质量的服务。合资车、国产车的产品质量和服务，远远超过国产车，原因并不是外国人产品质量好，而是外国人相对认真负责一点点而已。毕竟没人愿意修车排队。

可能外国车，合资车的技术比国产车好一些，但我相信差距不会太大。国产车和外国车，合资车差距最大的地方，应该是…… 国产车厂商的管理人员，工人，责任心远远低于外国厂商和合资汽车厂商。

外国汽车厂商，能做到全球顶级品牌，仅仅是他们愿意为自己的行为负责，他们懂得，不负责的结果就是被淘汰。然而我郭人不太懂这个简单的道理。这才是最大的差距。

## 怎么从员工升职成领班主管?

仅仅是，当保安上夜班不睡觉，比其他同事更责任心，做人做事更靠谱。舍得花钱给同事买夜班饭，愿意付出代价赢得良好的人际关系，获得大家的拥护支持和好评。舍得花钱请经理吃几顿饭，平时说话好听一点，有眼色，会来事，善于讨得上司的欢心。不过如此罢了。

就像消费者都抢着购买外国品牌汽车，抢着购买合资品牌汽车一样，原因是外国人生产的汽车(进口车 ), 或 者 外 国 人 监 督 生 产 的 汽 车 ( 合 资 车 ), 品质有保证……

外国汽车品质有保证的原因是，外国人比较负责 任……外国人负责任的原因是，他们知道，如果不负责任，生产出来的商品不靠谱，就会砸了自己的牌子，导致汽车卖不出去，最终倒闭，自己失业……

一个道理……外国汽车具备高品质的原因是，因为外国人知道，如果做人做事不负责任，生产的汽车品质不靠谱，就会失去消费者….…

购买劳动力的消费者就是私企老板(或者官府和事 业单位的领导)。 就像有工人给我干活，那些认真负责的，努力做好工作的人，他们的工作就会长久，因为他们工作靠谱，我放心。那些做事不靠谱，糊弄人的工人，就会被解雇。因为他整天想着偷奸耍滑，工作品质没有保证。

所以啊，升官发财的秘诀之一，就是工作靠谱，让别人放心，让别人对你产生信任。在任何地方，都做一个做人做事靠谱的人。

在一个地方，如此这般，混几个月没前途，就赶紧跳槽，换个公司继续这么做。陈旭换了三家公司，第三家公司，我就从保安员升职做了领班。

就凭这一点点优势，陈旭就淘汰了其他保安员，让自己成为了领班。升职加薪的起点，就是愿意为自己的行为负责，愿意多付出，愿意多尝试，不怕失败，不怕损失，仅此而已。

这么说吧，假设你是一头猪，只要你有 “认真工作 + 讨好领导”的意识，你多换几家公司，你也会遇到喜欢猪的领导，于是你就升职加薪了 ….

有时候，成功的起点，说出来都没人相信，大家会以为是笑话。然而这是真的，是我的亲身经历，不是笑话。
