
# 赫拉利访谈：“人之为人”的优势到底是什么？

## 大历史的概念

大历史的研究可以追溯到贾雷德·戴蒙德的大历史三部曲，包括《第三种黑猩猩》《枪炮、病菌与钢铁》《崩溃》，以及麦克尼尔的《瘟疫与人》。这些书都堪称它们那个年代最厉害的作品。

大历史的特点并不是它大，而是研究格局大，它已经延伸到了人类的能力之外。大历史研究的是那些左右人类文明，却又不受人类左右的力量。

《第三种黑猩猩》讲的是基因对人类的影响，《枪炮、病菌与钢铁》讲的是自然资源对人类历史的塑造，《崩溃》讲的是人这个物种的感受力存在巨大的局限，而这种局限让人们对自然的缓慢崩塌视而不见，最终导致文明的崩溃，《瘟疫与人》主要讲的是细菌对人类文明的影响。

## 赫拉利的独特角度

赫拉利研究的不是那些人类之外的因素，而是那些存在于人类之间，但又没有被人类意识到的，塑造人类文明的因素。他研究的是我们生而为人的真正禀赋，这关系到每一个人。

赫拉利所说的，那些人之为人独特的禀赋到底是什么呢？这个禀赋在今天受到了哪些挑战？人类又有没有应对的预案呢？

## 人类最重要的禀赋

人类之所以能有今天的文明，是因为人类能展开大规模的协作。之所以能展开大规模协作，是因为我们有讲故事和相信故事的能力。讲故事与相信故事的能力，是人类最大的优势。

尼安德特人比智人更强壮，更有劲儿，甚至连咬合力都更强，还有研究说他们也比智人更聪明。但为什么最后是智人在进化中胜出了？是因为我们会讲故事和相信故事。
