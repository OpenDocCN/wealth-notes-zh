
# 决战下半场1：中美“后关税战时代”

最近，A森分别对国内多个城市进行调研，收获颇丰，产生了不少全新的研究成果。目前，市场普遍担心的是，中美关税战所谓休战90天，差不多是到8月12日左右结束。那么，接下去，两边又会如何相互折腾？川普会打什么牌？我们会如何应对？我们的应对，对于国内的资产、就业、物价、财政、就业等各方面又将产生什么样的系统性影响？

由于这件事实在太过重要，接下去，【A视野】会在公号发布一组付费合集《决战下半场》，本组合集包含3篇文章，分别从3个维度剖析底层逻辑。

*   第一篇《中美“后关税战时代”》，会在2025年7月12日发布
*   第二篇《美元信用崩塌下的终极避险》，会在2025年7月14日发布
*   第三篇《中国经济进入新范式》，会在2025年7月16日发布

请注意，三篇逻辑一脉相承，层层递减，缺一不可。

以下内容为第一篇《中美“后关税战时代”》的正文内容

从4月2日的所谓“解放日“开始，川普执行全球关税战，已经有数月之久。然而，真正跟美国达成新的贸易框架的国家，非常少。即便是传统盟友如欧洲和日本，也都是一拖再拖，期间，不乏对美发出极为强硬的论调。本质上，就是川普已经陷入了严重的“新不可能三角”：制造业回流VS严控债务风险VS防止通胀再次高企。

我们可以看到，无论是川普逼越南签署防转口贸易条款，还是要自筹关键产业链，经过本轮中美关税战交手后，川普对整个事情的理解已经发生改变。如果说，此前仅仅只是要打压我们，顺便抢走我们更多的财富，但是，目前这些都已经放在其次的位置。中国以稀土作为卡脖子的武器，已经实质性的让美国主动解除了不少卡中国脖子的重要科技领域。

因此，川普如今要的制造业回流，更多优先是避免未来美国国家安全受制于人。这个改变，意义重大，也直接导致所谓的海湖庄园协议等已经毫无全面执行的必要。

在川普2.0初期，制造业回流是对自己的支持者的“承诺反馈”。即，你们选我，我上去后，给你解决就业问题，也帮你们打击建制派的贵族利益。所以海湖庄园协议的内容，有着一种极为浓郁的味道。希望用对外收割，缓解国债压力，进而为举债对自己铁粉发福利、增加新的制造业就业机会，提供可能

然而，现如今，川普已经充分感知到，如果未来美国连卡中国脖子的办法都没有，甚至还要被中国卡脖子，那么，又如何完成第一步“对外收割”呢？这岂不是让后续整个政策链条都土崩瓦解了吗？！理解这点也就明白，为何90天的休战期，川普联手贝森特的化债策略早已发生系统性的改变，即，所谓的宾夕法尼亚计划。

事实上，这等于是巨大的回撤，相当于承认对外收割已经出现系统性的问题。然而，如果要这么干，那美国的通胀未来将会是难以压制的。一方面，自己人背债，等于自己人背走通胀压力。一方面，以锚定安全为优先级的制造业回流策略，还要打击转口，势必加速中美经济脱钩的进度，但是，美国短中期根本看不到解决供给瓶颈的任何迹象。（这点也是近期华尔街大佬埃利安亲口承认的）。

因此，川普的回撤一步，直接引发“制造业回流VS严控债务风险VS防止通胀再次高企”三者之间无法同时满足的尴尬境地。不过，我们也要明白，这也意味着，后续川普会想其他办法，争取再次找到收割我们的机会。毕竟，整个策略的破口就是“无法对外大规模收割”。那如果这个破口能够暂时堵上，起码短期游戏又能够玩起来了。所以，我们可以看到，川普威胁对金砖国家中要去美元的国家进行加征关税。我们也看到，川普疯狂囤积大量铜矿，然后，在近期突然宣布要对铜进行高关税加征。

有关这一点，是因为他看到中国是用铜大国，每年进口量庞大，国内供给缺口根本无法靠自己缓解。他就是想要用铜来卡我们的脖子，以便置换我们手里的稀土供应。基于此，我们也不难明白，川普从来没有放弃对外收割，尤其是对我们收割。那么，90天关税战休战结束后，川普势必是两手抓，两手都要硬：既要想办法先国内化债做好防守，又要重新找我们的供给和金融端的痛点来谋求再次对我们发动进攻。

很明显，游戏不仅没有结束，游戏只是变得更加暗流涌动罢了。当前，中美都在各种领域疯狂试探对手的软肋，这个才是表面看起来暂时斗争没有如此激烈的根本原因。即便试探出来了，那你要布局，也是需要一定的时间的。就像美国人布局铜矿囤积的事情，最早是3月份就已经开始，4月份则是全面提速。类似的逻辑，其实中国也是这么干的。

这里需要注意的是，近乎所有经济强国都已经将中国在2024年对外贸易顺差突破1万亿美元，作为重大事件。毕竟，在创新周期尾部，大家都赚钱很难，都想要对外多出口多赚钱。今天，川普要大幅减少美国对外贸易逆差，何尝不是基于这样的逻辑呢？！因此，有关这个方向，后续会成为美国攻击我们的一个所谓的“软肋”。

此外，美国精英普遍认为，由于我们的地产风暴，只要打爆我们的外需，就可以打爆我们的经济。先不管这个论调是否正确，可是，这确实是美国精英们的“共识”。本次7月底重磅会议之前，有形之手高调宣布大量“反内卷”措施，甚至给企业直接框定产能上限，实则就是要提前应对这个看得到的需求侧风险。

这一策略的核心，就是要像2015年“供给侧改革”一样，控制产能，提高企业的利润率，拔高物价效应，逼大规模囤积在银行内的资金流到社会方方面面，进而带动全社会的货币乘数效应起来。唯有如此，中国内需的正循环，才能走起来。那么，为何要在8月12日之前马上大规模执行？

除了是我们的国内市场也确实扛不住更多的内卷外，更重要的是：村里已经按照相对较坏的剧本假设，去关注8月12日以后的“中美后关税战时代”的博弈惨烈程度。

中美将不会有任何真正意义上的长期协议。目前所有的协商、谈判，不过是休战。休战的目的，不过是为了给自己喘息期，让自己有足够的时间和空间，继续往死里干。甚至，我们还可以再去往深里思考。当美国企业利用90天休战完成了大规模的库存囤积，川普还会担心很多吗？暂时，除了稀土等少数供给，其他方面，川普绝对会充满底气，重新谋求对我们的收割机会。

须知，美国是封建制市场经济体，其内在的成本是极高的。如果不能从外获得收割收益，它的整个社会的运作都会出现危机。既然这是明牌，那么，我们对于8月12日之后的中美关系，或许只能适度悲观，而不能乐观。其实，近期市场也是这么来看的。

本次反内卷，有不少人认为，是新一轮供给侧改革。这个说法，从经济学角度，貌似没错，但是，很不全面。本次反内卷，是与同期再次强化的“全国统一大市场”，混合在一起说的。也就是说，村里已经做出了重大的策略改变：

产能在需求侧的锚定，逐步从过去几年锚定外需更多，加速切换为锚定内需更多。如果继续锚定外需（美元），将来我们就会完全受制于人。而且，锚定美元外需，等同于我们源源不断的给美国人输送利益，已经不符合新时代的斗争形势。此外，我们已经连续8个月增持黄金，对美债仓位的减持不断推进，都意味着，我们手里的美元已经是烫手山芋。

考虑到美国人即将大规模的疯狂举债，继续锚定外需，就是继续支持美债扩张，那么，后续中国经济势必会面临需求“从外切换为内”的阵痛期。如果这个阵痛期一定要来，早来比晚来更好。当然，我们也可以认为，目前是在川普关税战、美联储降息、川普科技战等诸多关键事务还高度不确定性的前提下，算是一种提前布局。先打一个提前量，确保能够在川普突袭时，我们企业在需求侧不至于过度狼狈。本质上，就是先给我们的经济打一个需求侧的提前量。如果后续外需承压更多，或者美联储极端情况下年内真的不降息，则我们不至于因为总需求不足而完全受困。

因此，本次反内卷，只不过是整个刺激内需的一个“起手势”。核心，就是要更好的应对即将到来的“中美后关税战时期”。目前来看，川普将有可能采用这些组合拳，在后关税战时代对我们发力。锁定新的卡中国脖子的赛道 + 打击中国转口贸易 + 抬高中国制造业成本 + 开辟新的赛道虹吸中国储蓄+ 制造地缘不确定性 +围猎一带一路

总体而言，就是抬高成本，打击收入，截留利润留底，增加运营风险，精准锁喉，阻挠突围管道。请注意，这个组合拳，是目前美国正在干的事情。但是，在8月12日之后，以上策略，将会是川普2.0整个班底的长期策略。因为，如果这个策略还是不行，那就真的很难快速制定新的、可行的、有效的方案了。大家也看得出，尽管欧日韩印还是偏向美国，也对我们各种负面影响，可是，由于我们始终是虚与委蛇，导致美国再次组建联盟抱团对付我们，很难。

核心点在于，中美进入后关税战时代，是与美国跟其它国家仍处于关税战模式，齐头并进A。在无法收割中国的前提下，美国想要对外收割的目标资金总量并没有减少，那么，平摊到所有其它国家头上的“收割额度”也就变相增加。请注意，这一点格外重要！它既分化了美国跟这些国家的利益，却也会让它们跟中国之间生嫌隙，而且，反而会促使这些国家加速所谓的独立性。所以，这并不是完全有利于我们的事情，尽管我们也没有办法。

后关税战时代，意味着，美国在全球的统治模式，将进入“双轨制”。对中国，是一个模式。对别的国家，是另一个模式。初期，这种“双轨制”并不是那么的明显和重要。毕竟，很多发达国家长期跟美国建立了大量利益共同体的机制。然而，如果这个“双轨制”的态势进入这么一个节点，即，中国模式的获益并不比美国盟友模式的获益少，那么，这些盟友国家就有可能反水。事实上，这个才是美国全球统治的“双轨制”模式无法长期执行的根本。而且，这等于让其它国家有办法在两个不同的轨道之间，开始套利。

可是，“双轨制”所带来的全球统治成本却是极为高昂的，甚至是越来越夸张。当川普宣布跟中国关税战休战90天的时候，就是宣告”双轨制”的时代到来。这个时代，对于美国而言，是非常危险的。名义上，它好像依旧是全球共主；可是，实际上，它可以自主管控的地盘越来越少，甚至难以覆盖它的支出。如果美国无法解决这个问题，那么，美国的债务压力会长期加速膨胀，最终成为压垮国运的关键力量。从这个维度来看，8月12日后，无论川普折腾什么，只要中国确保“守正出奇”的战略不变，做好战略定力工作，那么，美国全球统治“双轨制”模式将在未来几年成为所有国家都看见的新趋势。

需要明白，现在并不是高举”义旗”马上寻求推翻美国霸权的好时机。因为，当我们进入“双轨制”模式，我们已经逐步事实性的独立于美国的封闭体系，他们却还拿我们一点办法没有。战略套利，我们来；全球统治成本，美国人付。不得不说，这是现阶段最有利于我们的事情。其实，现如今，川普还要找办法对付我们，手里的筹码和工具，已经不多了。

中美PK的焦点，并不在于一战定乾坤。真正的要点是，谁先被拉爆，谁先无法覆盖自己的成本。按照调研的情况，所谓的战略定力，实则也是锚定这一点。

总结一下，“后关税战时代”，不过是实践中论证了美国又一次经济混合战的溃败，并没有多么了不起，也没有特殊性。只是，经过此役，中国经济未来只能加速锚定内需的压舱石，中央财政加杠杆，其它社会部门都很困难了；美国则因为成本失控，未来只能靠超发货币稳住金融系统，靠地缘卖战争创可以说，这个是重大的转折点！它将彻底改变我们自己国民经济运行的方式和逻辑，也将彻底改变美元霸权体系的未来走向。
