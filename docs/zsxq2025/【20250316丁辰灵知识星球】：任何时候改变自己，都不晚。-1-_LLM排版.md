
# 任何时候改变自己，都不晚

今天下午和一位资深会员喝茶，非常有能力有资历。现实生活中大家都会感觉很成功的人，也是在各方面工作感情股票都有自己的烦恼。聊到后面我就问你的烦恼压力究竟是什么，他说是对未来的不确定性，但和我聊了以后感觉好多了。我就开玩笑说，其实我觉得你比较幸运，在现在有压力的时候能找到我帮你梳理。我说我在创业的时候遇到那种人生的不确定性，晚上翻来覆去睡不着觉，只能靠自己。

我29岁时做游戏公司，投资人没钱了，我自己拿我自己积蓄继续去给员工发工资，然后我每天出去见投资人融资。结果遇到一个所谓的大佬，把我公司的技术负责人挖走，那个技术负责人就告诉所有的员工说公司早没钱了，是拿自己的积蓄给你们发工资，他也快没钱了，你们跟我去新公司吧。有的人愿意去，有的人不愿意去但很震惊。我记得有个女技术经理跟我说，Bob 我对你失望，公司遇到这样的困境你为什么没有和我们说呢？你没有把我们真正当兄弟，造成被这个技术负责人煽动得手，我们这个团队组起来是多么不容易。现在人散了，我是肯定不会跟过去的，但是我也没法待了，你招人吧，我做交接。

我没办法，在2007年的1月，我又招了一个刚毕业大学生，女生给她交接了一两个星期，春节后就离开了。我就不得不自己下场编程了，因为是真的没钱了。我那时候有七年没自己编过程了，后来又招了几个人，把游戏Java 服务端的代码一点点重构出来，熬了八个月游戏重新上线。

肯定是有很大的难受，冲击，不接纳，自我攻击。那个技术负责人是我师弟，已经很长一段时间阴阳了，动辄以抑郁症为名请假，后来女技术经理跟我说，你怎么会相信他？我也是自责了很久，我不知道为什么会变成这样。因为在我的信念系统里面，你辜负我的信任自己离开就罢了，为什么会要恶意挖人呢，在我最困难的时候还要踢我一脚呢。

我们会员今天听到我讲这个故事的时候，他就觉得好多了，他说相比较而言他的苦恼压力似乎就不算什么了。

记住，地图不是疆域。我们所有的选择看起来是自由选择，其实是按照自己的一个地图在走。地图之外，我们是不敢去触碰的。但实际上我们可以有更多选择，地图不是疆域，在我们每个人地图之外，有非常广阔的天地和可能性。这就需要我们愿意去尝试各类可能性，会需要我们的生活我们的选择可以更有弹性，而不是固守在一种信念系统中。

我们的地图，我们的信念系统，行为模式，主要都是在童年完成的，成年之后如果不是遇到巨大痛苦逼自己改变，大部分时候我们是不会愿意改变的。所谓谁痛苦谁改变，大多数人不愿意触碰到创伤，更不愿意发现创伤后去真正愿意花时间去疗愈创伤，实现和解，然后才可能产生新的信念，让自己得到成长。

我们的很多信念其实不是我们自己的，而是我们的父辈家族社会身边的人灌输给我们的，这些信念价值观一定适合我们吗？不一定。我们18岁之前形成的主要的地图，一定能适应人生未来几十年的挑战吗？毫无疑问是很难的。

但是我们的社会晋升筛选体系，他其实并不鼓励灵活性，官僚体系一干可以干几十年，读书可以从小学到博士，按照自己已经形成的一个行之有效的刷题的方法最终可以在30岁之前拿到博士学位。我们要觉察这种社会体系对我们个人幸福成长的危害之处。

还有我们的本性问题，即人类社会几百万年形成的集体潜意识，比如男人就是天生要去冒险的，去外面打猎。但在现代社会，会被各类的社会体系所压抑和束缚，让男人变得唯唯诺诺，甚至男人不像男人。

因此当我们说面对不确定性感受到担忧，到底是什么意思？男人的使命，基因里的使命就是拥抱不确定性，我们的使命在部落时代开始就是保护女人，保护孩子，保护部落。当你去丛林打猎时，当你去战场杀敌时，随时会丢掉生命，你除了勇敢的往前走，哪里有其他选择？

现代社会评价体系，信念价值观，真正符合自己潜意识里感受到幸福完满的那个进化了几百万年的本心吗？所以今天的人很痛苦，往往都是这个原因造成的，你世俗追求的未必是你内心想要的。即使你意识到自己应该成长，那应该如何去成长。我说，就是把贪嗔痴慢疑展开，对照自己，到底哪些部分是可以提升的，可以改变的。

会员说到了金山云的问题，他是去年七八月我们在北京聚会结束之后，专门后来又私下问我。我给他很确定的答案，最后他还是没有买，因为3元的时候，他想等2元，涨到4元，他想等3元。结果金山云涨了十倍，他没抓到这个机会，他很懊恼。而他有另外一笔重仓的投资做错了巨亏，这样一来一回，相当于几个月时间差了一两千万的收益，他就很。这就是疑吗，迟而不决是疑。这就是他的人生课题，是需要修炼的。

很多人都觉得人生中年了，好像什么事情都定型了，什么事情改变都太晚了。这样的信念价值观哪里来的呢？为什么要这么想呢？男人至死是少年，男人至死都应该为我们身边人去打拼去奋斗去保护我们的部落，在今天就是我们的国家。为了这个崇高性，任何时候都不晚，改变自己，什么时候都可以。
