
# 中国台湾经济强劲增长的深层逻辑

2025年，中国台湾经济交出了一份亮眼的成绩单。台湾有关部门最新公布的数据显示，2025年第三季度，GDP年增率修正值高达8.21%，显著高于初步估计的7.64%，不仅连续三个季度实现增速攀升，更是直接带动全年经济增长预测至7.37%，较初始预估上修2.92个百分点，将创下自2010年以来的15年新高。

安邦智库首席教授陈功先生指出，中国在实施地缘战略并将其落实在产业层面的时候，要特别关注“杠杆效应”的存在。很多战略性产业就不是一个简单的“必须有”的问题，然后不计代价去搞、去做，就能成事的。这种“杠杆效应”的存在，必须要求战略性政策，要有精密的设计，才能具有实效，才会将经济的副作用降低到最小程度。

中国台湾在全球人工智能（AI）产业链中拥有的核心制造地位、完整的半导体产业生态以及政策红利的精准加持，使之成功抓住了AI产业爆发的历史机遇，实现了出口、投资、产业协同的全方位增长。

台湾经济向来以出口为导向，2025年的出口增长呈现出结构性爆发特征，其核心动力完全来自AI相关科技产品的强势表现。数据显示，2025年上半年台湾出口增长25.92%，进口增长20.49%，其中资通与视听产品、电子零组件的出口年增率分别高达63%和22%，这两类产品占整体出口比重分别达到36%和34.8%，合计贡献了出口增长的七成以上。

进入下半年，增长势头更为迅猛，9月台湾出口订单同比大增30.5%，达到约702.2亿美元，连续第8个月呈现增长态势，其中资讯通信产品接单金额176亿美元、电子产品190.1亿美元，同比增幅分别为7.0%和10.5%。台湾有关部门预测，2025年第四季度商品出口将达1723亿美元，年增率高达36.84%，为全年经济增长画上圆满句号。

这一出口增长态势的背后，是全球AI产业爆发引发的算力基础设施建设的热潮。在全球算力军备竞赛下，微软、谷歌、亚马逊等大型云端服务者加速布建AI数据中心，对AI服务器、高端芯片及相关零组件的需求呈指数级增长。而台湾地区正是这些核心产品的主要供应地，凭借成熟的制造能力和快速响应能力，成为全球AI产业链的“供应枢纽”。

半导体产业作为AI产业链的核心环节，更是此次经济上涨的“压舱石”。2025年台湾半导体产业总产值预计突破2098亿美元，同比增长19.1%，其中集成电路制造产值达1338亿美元，增速高达23.1%。台积电作为全球半导体制造的领军企业，其表现尤为突出：2025年三季度营收331亿美元，同比大涨40.8%，3nm和5nm工艺收入占比合计达到60%，仅该公司的产值增长就贡献了台湾近4个百分点的GDP增幅。

值得注意的是，美国推迟半导体相关产品关税的政策红利，进一步放大了台湾科技产品的出口优势。2025年初，美国曾计划扩大对华贸易关税范围，引发市场对台湾半导体产品出口的担忧。但最终，美国将半导体产品排除在关税清单之外，主要原因在于台湾半导体产业对全球科技供应链的重要性，以及美国本土科技企业对台湾半导体产品的高度依赖。这一政策调整使得台湾对美科技出口逆势增长，2025年前三季度，台湾半导体对美出口额达到320亿美元，同比增长12%，占台湾地区对美出口总额的45%。

如果说AI需求是经济上涨的“导火索”，那么民间投资的大幅增长与产业链的协同效应则为增长提供了坚实支撑。数据显示，2025年第一季度台湾制造业国内固定资产增购年增率高达82.3%，带动民间投资表现优于预期；第二季度，半导体与资本设备进口动能延续，支撑了上半年整体民间投资的稳健表现。这种投资热潮主要集中在科技产业，尤其是半导体先进制程、AI相关设备等领域。

台积电的产能扩张是投资增长的核心代表。为应对AI芯片的爆发式需求，台积电不仅加速推进既有2纳米工厂建设，还计划在台湾南部科学园区周边增建三座2纳米晶圆厂，使其在台湾的2纳米制程产能规划总量达到10座。目前，高雄楠梓科学园区的F22工厂已进入量产阶段，月产能达1万片；P2厂已于8月1日启动设备装机，预计年底可启动试产，年底前月产能将提升至3万至3.5万片。这种大规模投资不仅直接拉动了资本形成，还带动了上下游产业链的协同增长，从设备采购、材料供应到工程建设，形成了庞大的产业涟漪效应。

台湾半导体产业链的完整性进一步放大了增长效应。除了台积电在制造环节的主导地位，日月光在封装测试领域、联电在成熟制程领域均占据全球领先地位，再加上印刷电路板、精密机械等配套产业的协同支撑，形成了“设计-制造-封装-测试-配套”的完整产业生态。这种完整的生态优势使得台湾能够快速响应全球AI产业的需求，从芯片制造到终端设备组装的全流程高效衔接，不仅提升了交付效率，还形成了难以替代的竞争壁垒。

总的来说，2025年中国台湾经济高增长无疑是中美竞争下全球AI产业爆发与自身产业优势结合的成果，全球地缘政治等于刺激台湾出现了一轮高科技投资高潮，进而拉动了台湾的经济增长。在这些方面，AI需求驱动的出口增长、半导体产业的技术垄断、产业链的协同效应，共同构成了经济上涨的核心逻辑，反映了台湾在全球科技供应链中的重要地位。而中美竞争让高科技成为战略竞争的主要舞台，大幅度提升了对台湾高科技的需求，为其扩大了市场。

最终分析结论：
在逆全球化思潮蔓延、国际供应链重构的大背景下，高科技成为战略竞争的主要舞台。台湾地区凭借在全球AI产业链中的核心制造地位，成功抓住了AI产业爆发的历史机遇。从数据解构来看，出口的爆发式增长是核心驱动力，而AI需求的指数级攀升则是背后最关键的推手，叠加产业投资加码以及产业协同，共同促成了这场15年来最强劲的经济增长。

## 形势要点：“十五五”时期存量改善或有较多投资需求

国家发改委投资研究所研究员吴亚平表示，“十五五”时期扩大有效投资既要保持投资合理增长，也要提高投资效益，需要统筹处理好六大关系，包括政府和市场、质量和总量、投资和消费、存量和增量、传统和新型，以及“投资于物”和“投资于人”的关系等。吴亚平表示，以基础设施为例，当前我国有规模约400万亿元的存量基础设施。但这些资产并非都符合高质量发展的新要求，产业领域、城市建设领域有大量的存量资产并不满足高质量发展和高水平安全的要求。这使存量基础设施更新改造具有很大的投资需求。对于“投资于物”和“投资于人”的关系，吴亚平表示，受人口总量以及人口老龄化等因素的制约，很多行业对于物的消费需求已基本见顶。然而，如果从人的全面发展和人口高质量发展的视角审视，“投资于人”的相关重点领域仍存在很多供给侧短板弱项亟待补齐或加强，这也意味着在惠民生领域蕴含着较多的投资机会。他提到，“十五五”时期扩大有效投资要聚焦新质生产力、新型基础设施、新型城镇化和重大工程等“三新一重”领域。其中，电力、水利、新基建、生态环境等重大工程，有的未来需求比较大，有的则短板弱项比较明显，总体来看潜在投资增长空间比较大。

## 形势要点：长三角一体化发展空间格局落定

国务院正式批复《长三角国土空间规划（2023—2035年）》，这是国家层面批准的首个覆盖长三角全域的国家级国土空间专项规划。从战略角度看，规划的核心价值在于将“一体化”和“高质量”两大关键词空间化、具体化。它通过设定明确的空间底线，如到2035年耕地保有量不低于16148万亩、生态保护红线面积不低于7.7万平方千米，为高速城镇化区域锚固了生态与粮食安全基底。同时，规划着力构建以都市圈为引领的空间格局，支持上海发挥龙头作用，并推动上海大都市圈与南京、杭州、合肥、宁波等都市圈的协同融合，旨在提升世界级城市群的整体效能。此外，规划对G60科创走廊、沿沪宁产业创新带等重点创新载体的空间保障，以及对传统产业向苏北、皖北等地有序转移的布局安排，体现了优化生产力布局、促进区域协调发展的战略考量。不过，规划的成功关键在于未来执行的协同性与适应性挑战。虽然规划强调“一张图”系统的权威性和严格监管，但如何在跨省级行政单元的实际操作中，真正实现标准统一、数据共享、监测同步和执法协同，仍需突破体制机制障碍。例如，先行示范区在元荡湖治理、数据互通等方面的探索为全域规划提供了宝贵经验，但也揭示了深层协调的复杂性。此外，规划需平衡刚性约束与发展弹性，在坚守底线的同时为未来技术创新和不确定性风险预留弹性空间，避免过度划定的“蓝图”束缚地方发展的活力与特色。当前各地发展意愿强烈与指标紧约束的现实矛盾，也考验着规划在实施中的动态调整智慧。

## 政策：国家标准《外卖平台服务管理基本要求》发布

市场监管总局近日发布实施推荐性国家标准《外卖平台服务管理基本要求》。具体内容包括，一是强化主体责任落实。通过规范商户入驻审核、日常运营管理等环节，引导外卖平台切实履行对商户的服务与管理责任。二是聚焦平台收费与促销行为，强化竞争秩序规范。明确平台收费规则及促销活动管理要求，督促引导外卖平台简化收费项目、规范促销行为，推动各类收费信息透明化、促销规则合理化。三是聚焦平台用工管理，强化配送员权益保障。围绕劳动强度、职业关怀和社会保障等方面，提出合理确定劳动时间、完善职业培训体系、探索多元保障方式等内容。四是聚焦平台内争议处理，强化协商机制建设。督促平台建立完善消费者权益保护体系，完善消费者、商户、配送员等多元主体投诉、申诉及处置机制建设，确保平台管理程序透明，畅通投诉举报与异议反馈渠道，主动接受社会监督。分析认为，《基本要求》旨在对现有法律法规和监管要求进行细化和补充，将有助于帮助外卖平台优化管理方式、提升服务质量，进一步推动外卖行业在良性竞争中创新发展。

## 形势要点：中国绿证实现国际化还需走一段长路

今年，中国绿证在国际化道路上取得重要突破。国际最具影响力的可再生能源倡议RE100对中国绿证使用从“有条件认可”变为“无条件认可”。在本次COP30上，中国绿证也是作为能源成果亮相大会。然而，绿证要真正实现国际化还需要走一段长路。罗兰贝格副合伙人屈思啸表示，一方面，国际规则普遍要求绿证具备小时级溯源、时空匹配及使用后强制注销等功能，我国绿证当前缺乏完善的全流程追溯和强制注销机制。在发行、交易等环节透明度有待进一步提升，且存在环境效益重复计算的风险，这导致了目前除RE100以外，中国绿证与国外绿证体系互认程度较低。另一方面，当前我国政策未明确绿证与碳市场的抵扣机制，这使得绿证的环境价值需要通过较为复杂的认证手段才能与碳足迹挂钩，企业购买的绿证无法直接、简便地用于抵消碳足迹，进而降低参与国内碳交易的成本或降低海外“碳税”水平。

## 形势要点：新形势下自贸区还有很多需提升之处

自贸试验区十年改革探索，硕果累累。站在十年的历史节点回望，自贸试验区这片曾经的“试验田”已变成“高产田”。但同新形势下新使命新要求相比，自贸片区还有很多可以提升的地方，比如创造新增长点乏力；产业规模较小，承接产业创新发展空间不足；现有空间均为建成区，且主要是港口、海关特殊监管区域，缺少承接科研机构、产业项目落地的空间。商务部国际贸易经济合作研究院副研究员朱福林也指出，中国已进入国内改革深水区和攻坚期，自贸试验区推进高质量发展也面临诸多挑战。如存在顶层设计不足导致管理机制不畅、高价值制度创新较为缺失、重大开放风险压力测试不足、对接国际经贸规则发展前沿存在一定的差距等问题。中国社会科学院城市发展与环境研究所发布的《城市蓝皮书》也提出，中国自贸试验区建设存在包括开放力度整体不高、资本流动方面并未取得突破性进展、税率偏高、法律制度有待完善、试点经验缺少系统性等问题。
