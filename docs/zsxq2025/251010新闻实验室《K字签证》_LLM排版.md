
# K字签证

K字签证旨在吸引全球STEM领域的年轻专才，但在国内引发担忧和质疑。这些质疑反映出政府治理与民意舆论的复杂关系。

## K字签证的争议

K字签证允许符合条件的年轻人无工作邀请申请入境，给予更大灵活性。然而，在中国青年失业率逼近19%的背景下，这一政策被视为对本国年轻人的“背刺”。社交媒体上批评声浪此起彼伏，认为政策标准过低，并担忧催生欺诈产业链。

## 中国移民政策的演变

从建国初期的严格管控，到改革开放后的“摸着石头过河”，再到21世纪的加速融合，中国移民政策始终在服务国家战略、应对外部变化和回应内部压力之间进行平衡。

### 建国初期：严格管控

建国初期，中国对外国人实行严格管控，外国人几乎绝迹。这一时期被称为“静态的几十年”，国际迁移被高度政治化。

### 改革开放：引进来

改革开放后，中国将跨国流动视为服务经济现代化建设的工具。1985年，《中华人民共和国外国人入境出境管理法》出台，标志着移民管理开始走向规范化。

### 21世纪：加速融合

21世纪，尤其是中国加入WTO后，作为移民目的国的吸引力快速增长。2020年，居住在中国大陆的境外人员已达140万。

## 中国移民政策的矛盾

中国移民政策长期行走于一架严重失衡的天平之上。一方面，国家不惜重金吸引“高端人才”；另一方面，对普通移民日益收紧管控，并对国内民众排外情绪谨慎回应。

### 高端人才的引进

以“千人计划”为代表的一系列国家级人才项目，通过提供丰厚科研经费、优越薪酬待遇等，吸引全球顶尖人才。

### 对普通移民的管控

广州非洲商人社群的命运变迁最具代表性。随着他们的存在日益可见，地方政府开始频繁进行签证和居住登记检查，许多非洲商人发现续签签证变得异常困难。

## 公众舆论的影响

近年来，“超国民待遇”成为极具煽动性的话题，深深刺痛了公众神经。民族主义情绪高涨，使得任何试图推进移民政策自由化的举措，都必须直面国内日益增长的保守情绪。

## 中国移民治理的困境

国家发展的客观需求和全球竞争的外部压力，要求中国更加开放，吸引更多元的移民。然而，碎片化的官僚体系、以安全为导向的治理惯性，以及汹涌的国内民族主义和民粹情绪，共同构成了一股强大的保守力量，使得任何实质性的开放举措都变得举步维艰。

中国作为一个新兴移民国家的未来之路，将注定充满挑战。
