
# 屠龙手段-“所见即所得，所见才是现实”的世界

在讲今天的观点之前，我想先给大家讲两个故事。

## 故事I:
1854年，伦敦闹霍乱，死了很多很多人，大家都束手无策。这个时候来了个医生叫John Snow, 他用了一个非常简单的方法，把霍乱止住了：就用了一张地图和一根铅笔。

他做的事情也非常简单：就是把每一个死亡的人的居住地，用一个黑点，点在伦敦街区的地图上。渐渐地，他发现有一个地区的黑点特别密集，几乎所有的死亡，都围着一个固定地点。他就跑去了这个黑点的集中地。然后他发现，黑点中心的中心是一口井。原来周围的人都是因为这口井传染了霍乱。他组织人把这口井封了。霍乱就渐渐消除了。

## 故事2:
伽利略刚开始用望远镜观测木星的的时候，发现有两颗星星绕着木星旋转：这就是我们现在所知的“木卫一”和“木卫二”。当时罗马天主教会，就要找他的麻烦：即便他这个发现，还没到哥白尼的那个“地球绕着太阳转”那么离经叛道。他只是说“有两颗星星绕着木星转”，就违反了罗马天主教会的教义。

那个时候，罗马天主教会认为：所有天体都是围着地球转的。所以说任何天体不围着地球转，都是离经叛道的。如此一来，伽利略也差点要倒大霉。但是他比较幸运的是，望远镜这个时候普及了，很多贵族都开始观测天体。而木卫一和木卫二绕着木星转这件事，是非常稳定的：很多人都能观测到。于是这件事情，形成了一个“共识”。

大家长眼睛的都看见了，所以罗马天主教会也没得说。

上面两个故事告诉我们，到底什么是“现实”?很多东西存在于客观世界，但是并不存在于大多数人的视野中，或者说不能戳破到他们的眼前，这些人就不能理解。

在第一个故事中，John Snow是突破了人眼和人脑不能准确的记忆时空的局限，所以也就无法进行疾病的“溯源”。而Snow的做法，就是把人类不能在脑子里看见的东西，摆到了人的眼前：所以大家同意他封了那口井。

同样地，在过去，没有人知道细菌、病毒，或者某些天体的存在是现实，有很多人很难以接受细菌、病毒或者天体的概念：直到显微镜和望远镜的普及。当很多人都得以看见它们的存在的时候，这些东西在大众的脑海中就形成了共识。

虽然有很多思想先进，或者受过很多科学或哲学训练的人，可以一直看懂抽象的事物，但绝大多数人，甚至是日常的领导和精英，也很难看懂抽象的东西。

我遇见过很多次，我跟领导汇报，做了无数页PPT, 算了多少数字，讲了多少道理，都不行的情况：但是只要我把demo给他看一遍，马上就懂的。

任何东西，只要停留在抽象层面，即便是现实，很多人也看不懂，或者不敢拿资源跟你一起赌：只有跟John Snow一样，把它变成一个“眼见为实”的东西，很多人才开始有安全感，相信你，来跟你一起做事业，赌未来。

这件事给我的人生指导是很大的，我把所有的现实，由上到下分成几种：

## 实物现实
## 互动现实
## 动态现实
## 图形现实

实物现实：现实中看得见，摸得着的东西。

互动现实：虽然摸不着，但是可以虚拟互动。

动态现实：视频或者直播能展示。

图形现实：能用一张图解释。

我们来解释一下：

如果你做一个产品，最好做成“实物”——实物是绝大多数人能理解的东西。实物解释起来也容易。

如果做不成，可以做成“互动”的，很多游戏、VR和AR也可以让大家理解。

要么就动态现实：这就是为什么视频或者直播的博主比文字的好做。理解的门槛低一些。

图形现实：一图胜千言，文字说清楚的时候，可以像John Snow那样展示一张图。

这些年我做很多事情，也开始学习用这些不同层次的现实来帮助别人理解，或者获得资源了。

比如，我们后续要推出一个动画片叫《驾驭粗心小神兽》。粗心这件事，很多孩子的理解都很抽象：我们把粗心可能的脑过程，都拆解出来了，让孩子跟这些“神兽”互动—这样就创造了一个现实。

我们在售卖化妆品的过程中，也创造了很多“现实”：比如胶原蛋白的三螺旋结构，比如荧光蛋白染色的渗透结果——我们给大家一一讲解。

这些照片，能够帮助大家一下子理解很多材料界的现实。同时，我们也第一次把功效的照片放在直播间，给所有人去看去皱的实际功效。

这些举措，都帮助我们说服了消费者。

大家可能说：“这些看上去是非常标准的操作啊，没啥神秘的啊!”

是没什么神秘的：但是日常总有人犯这样的错误。

面对没受过科学和哲学训练的人，试图用非常抽象的理念说服对方。很少有人，去跟自己的老板、老师、爹妈去隔空论理呢?

很少有人用实物现实、互动现实或者图像来进行说明。

我们多少年前的禁毒教育，是直接给大家看吸毒的人的样子：按我们的说法，是恐怖但有效。

我们并没有对未成年人进行长篇累牍的海洛因机理的学习，直接给大家看吸毒的人鬼样子——这样的教育震撼人心。

同样地，对于很多说理的工作，可以借助宣传片、纪录片、竞争对手的产品、竞争对手的视频来辅助，保障事半功倍。

实际上，人脑认识到什么是现实：是需要很多价值观的证据的，这就是真理的背负着经常要承担的代价。
