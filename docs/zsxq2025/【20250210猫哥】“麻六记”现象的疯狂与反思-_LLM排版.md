
# “麻六记”现象的疯狂与反思

2月8日，抖音安全中心发布公告，因账号发布不实信息（汪小菲包机送回大S骨灰），违背公序良俗和道德底线，不尊重逝者及家属，伤害公众感情，恶意炒作营销等行为。经综合研判对“我是夏小健”、“张兰·俏生活”、“张兰之箖玥甄选”、“汪小菲”的抖音账号予以无限期封禁处理。

至此，围绕在张兰、汪小菲和徐家之间的一场全民吃瓜“狗血连续剧”终于也将划上句号。但回顾整个过程，不仅让我们看到了中国新消费经济的魔幻，同时也带来了深刻的反思。

让我们把时间拨回到那个岁月激荡的年代。

1991年，34岁的张兰靠在加拿大唐人街端盘子、扛猪肉，每天工作16小时攒下的2万美元，回国创立了“阿兰酒家”。

这个四川女人不得不说在商场还是很能打，回国后将市井川菜包装成“高端艺术”，并在2000年创立了“要做餐饮界LV”的俏江南。

2008年北京奥运会，俏江南中标成为奥运餐饮服务商，张兰身价飙升至25亿元，登上胡润餐饮富豪榜第三名。

从洗碗工到“餐饮女王”，精准押注政商宴请市场，用“水墨画包间”的天价设计打造身份符号，将吃饭变成“阶层通行证”。这一路草根逆袭，张兰用了17年时间。

但一场资本游戏的“血腥教育”，又将张兰还没成型的商业帝国打得体无完肤。

为了冲刺上市，张兰在2008年事业达到顶峰的时候引入鼎晖投资2亿资金，并签下“对赌协议”——若2012年底未上市，需以4亿元回购股份。巧了，也正是就在俏江南筹划上市前夕，张兰的儿子被称为“京城四少”的汪小菲结识了大S，并快速完成了闪婚。

2010年9月，汪小菲在台湾明星安以轩生日宴上结识大S，双方相识20天、见面4次后就闪婚。

也许，对于张兰和汪小菲而言，需要“两岸夫妻”这个形象来提升品牌形象和政治安全性从而助攻上市。而对于大S而言：借“内地豪门”标签也能巩固内地商业价值。至少从婚后来看，她在广告代言费方面也是赚的盆满钵满。

虽然这场婚姻从一开始就被质疑为“利益联姻”，甚至在婚礼时张兰宣称“万达赞助场地”还被王思聪的无情嘲讽（结婚当天三亚狂风暴雨被嘲“老天都看不下去”），但确实是为双方都带来了短暂的双赢。

真正导致俏江南坍塌的导火索是2012年我们出台的“八项规定”。自那以后，高端餐饮重创，俏江南A股、港股上市之路接连折戟，最后才导致2014年私募基金CVC以3亿美元收购俏江南82.7%股权，就此张兰彻底出局。

虽然张兰曾痛斥CVC“空手套白狼”，但她对杠杆收购、股权质押等金融工具的无知才是悲剧根源，她以为资本是燃料，没想到成了引爆炸药的雷管。

当创始人沦为“局外人”，失去张兰的俏江南迅速衰败，门店从80家缩水至20家。这印证了一个残酷规律：当品牌与创始人人格深度绑定，资本驱逐创始人的那一刻，就是品牌灵魂死亡的开端。

其实，假如事情就定格在这里，那么我相信对张兰一家也许还存有一丝敬意，但后来的剧本却朝另一个方向快速飞奔。

2020年麻六记成立之初，汪小菲仍想复制俏江南模式，但门店生意惨淡。直到2021年11月汪小菲与大S官宣离婚。张兰发现“骂声比菜香”的流量密码，于是离婚大战正式爆发。

2022年3月，大S闪婚韩国歌手具俊晔，汪小菲微博暴怒：“连床垫都不换!”

2022年11月：大S起诉汪小菲未支付赡养费，但汪小菲却晒出一年4000万台币（约900万人民币）的账单，包括电费、司机费甚至再婚头纱费用，由此引爆热搜。

而张兰更是神操作不断，直播间上架“绿茶+卤蛋+软饭套餐”，模仿具俊晔打碟，用“嗡嗡嗡”暗讽大S现任丈夫。

也许是尝到了甜头，掌握了网暴、诋毁这个流量生意的黑暗兵法，于是在之后便一发不可收拾。张兰母子将“流量”玩到极致。比如：

塑造受害者联盟：汪小菲微博发疯文学“我不想再给这个家付电费了”，张兰哭诉“被徐家欺负到吐血”，激发网友“守护我方菲兰”的集体情绪。

制造性别对立：将大S刻画成捞女+心机女，强调汪小菲“付钱养前妻现任”，挑动“国男vs台女”争议。

司法舆论化：大S起诉汪小菲侮辱诽谤，张兰却在直播间高举证据一“我有她刷你信用卡买再婚首饰的记录”等等。

大S从那时起，就不断被黑化了。但张兰和麻六记呢?

麻六记酸辣粉销量暴涨，张兰抖音粉丝从200万飙至1000万，而大S的微博评论区遍地能看到辱骂甚至诅咒。

汪小菲从“伪豪门公子”变身“疯批前夫”，张兰从“失败企业家”变身“为儿征战的老母亲”。网友涌入直播间不为购物，只为听“床垫后续”，张兰边讲故事边喊“3号链接酸辣粉没了”。网友打赏“兰姐挺住”的礼物，被即时转化成“送10盒酸辣粉给灾区”，彻底完成了“愤怒一共情一消费”的流量变现、情绪货币化的闭环。

但更加戏剧性的事情还在后面。

当麻六记的爆款产品酸辣粉被扒出是代工厂贴牌，6元成本卖39元后。面对“割韭菜”质疑，张兰居然理直气壮的回复“我的故事不值30块吗?”。当产品被检出山梨酸钾超标后，麻六记却用“友商抹黑”来甩锅。

不仅如此，张兰还在直播间利用民族情绪，将大S再婚炒作成“韩国软饭男拐跑中国媳妇”，背景音乐长期循环《雄赳赳气昂昂》。

一边是产品质量存疑、法律纠纷缠身，一边是网友边骂边下单为“情绪价值”付费，商业的基本逻辑彻底被粉碎。有人坦言：“这就是新时代的街头卖艺，只不过以前要胸口碎大石，现在要直播撕前妻。”

这就是当下流量经济野蛮疯狂的一个时代缩影。

那么，当这场闹剧落下帷幕后，有哪些值得我们反思呢?

张兰母子的麻六记成功，本质是审丑经济的胜利：他们证明了只要够疯、够狗血、够会编故事，没有底线，只有流量也能收割销量。

而在这个过程中，网友既是消费者也是共谋者。

网友边骂“汪小菲巨婴”，而却主动送上门被收割。

愿意为各种乱七八糟的“梗”掏钱，获取社交谈资。

还有忽略其欠债、炒作的事实，把张兰视为“独立女性标杆”。

这些种种乱象的背后，也许是互联网时代的身份焦虑引起。当现实生活越发平庸，网友需要“连续剧式商业大战”来填补空虚。

但这终究不是社会发展的主流。截至去年，麻六记已在全国开80家门店，但大众点评评分普遍低于4分，标签多为“价高量少”“服务拉胯”。

当失去了流量红利，消费者逐渐走出闹剧后，张兰的麻六记还剩下什么呢?

借用一位餐饮人士预测：

——靠仇恨和八卦撑起的品牌，注定死于新的仇恨和八卦。

在中国商业史上，张兰与汪小菲母子的名字早已超越了企业家的范畴，成为一场融合资本博弈、豪门恩怨、网络暴力与流量狂欢的现象级大戏。这对母子用二十年时间，书写了一部充满争议的生存指南。

观众们一边唾骂，一边忍不住打赏催更。在流量至上的时代，企业家已不需要产品经理，而是需要编剧、演员和水军。当商业沦为真人秀，我们都成了付费观看的群演。

当“张兰模式”被无数中小商家效仿，当网暴和诋毁成为财富密码的时候，我们确实已经到了该正本清源的时候了，希望这次互联网平台出手能够给大家提个醒，未来商业一定属于类似哪吒这样的高质量产品才能赢得掌声和财富。
