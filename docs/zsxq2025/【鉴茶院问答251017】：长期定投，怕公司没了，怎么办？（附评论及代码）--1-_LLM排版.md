
# 长期定投，怕公司没了，怎么办？

长期定投是一种投资策略，通过定期定额投资来分散风险，平滑市场波动。然而，如果投资的公司出现问题，比如破产或倒闭，投资者可能会担心自己的投资会损失殆尽。那么，面对这种情况，投资者应该怎么办呢？

首先，投资者应该评估公司的基本面。了解公司的财务状况、经营情况、行业前景等，判断公司是否真的存在严重问题。有时候，暂时的困难并不代表公司会倒闭，可能只是暂时的困境。

其次，投资者可以考虑逐步减少投资。如果对公司的前景没有信心，可以逐步卖出持有的股票，降低风险。但需要注意的是，频繁交易可能会增加交易成本，影响投资收益。

再次，投资者可以分散投资。不要把所有的资金都投入到一家公司，而是应该分散投资到不同的行业和公司，降低单一公司风险的影响。

最后，投资者应该保持冷静，不要恐慌。市场波动是正常的，投资者应该根据公司的基本面和市场情况做出理性的投资决策。

附：评论及代码

评论：
1. "长期定投是一种很好的投资方式，但也要注意风险。"
2. "如果公司真的倒闭了，投资可能会损失，但分散投资可以降低风险。"

代码示例：
```python
def long_term_investment(company_list, investment_amount, frequency):
    """
    长期定投计算函数
    :param company_list: 投资公司列表
    :param investment_amount: 每次投资金额
    :param frequency: 投资频率（每月、每周等）
    :return: 投资结果
    """
    investment_result = {}
    for company in company_list:
        investment_result[company] = 0
        for _ in range(frequency):
            investment_result[company] += investment_amount
    return investment_result
```

使用示例：
```python
companies = ["公司A", "公司B", "公司C"]
amount = 1000
freq = 12
result = long_term_investment(companies, amount, freq)
print(result)
```

以上代码展示了如何计算长期定投的结果，可以根据实际情况调整公司列表、投资金额和投资频率。
