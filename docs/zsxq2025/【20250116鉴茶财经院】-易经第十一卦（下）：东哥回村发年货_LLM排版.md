
# 易经第十一卦（下）：东哥回村发年货

## 六四：翩翩，不富以其邻，不戒以孚。
《象》曰：翩翩不富，皆失实也。不戒以孚，中心愿也。

翩翩，这里是轻飘飘的样子。一个人富了，却没有让邻居和周围的朋友们得到好处，这个怎么会得到信任，持续进步呢？要做一个体系的核心，就是要不说大话，以诚待人，共同富裕，大家自然会放下戒备，全力助你。就像这次东哥回到光明村，给村里60岁以上的老人每人发放了1万元的现金红包，给当年小学的每位老师送上了10万元的现金红包，老师说多了多了，哎。给全村村民，家家户户准备了各个方面的年货大礼包，包括电动车、电冰箱、剃须刀、零食、衣物、食用油、白酒、坚果礼盒、词典笔、插座、摄像头、炒菜锅等30多种，林林总总，细致周到。村民扶我青云志，我送村民大礼包。东哥在村里说话，心平气和，慈悲雅量，村民们自发的在他门口放鞭炮，这就是泰，这就是财富给你带来了快乐。相比之下，另一个王姓大佬，也首富了，天天想的是砍掉供应商10%的价格，拖一年再给你结账。在海外的工棚，连巴西都看不下去了，说不能厨房和卫生间离这么近。这些在商场上当然没错，在境界上就差点意思了。钱是让你高兴的，你背个骂名和非议，不泰啊。

## 六五：帝乙归妹，以祉，元吉。
《象》曰：六五以祉元吉”，中以行愿也。

“帝乙归妹是历史上很有名的一个故事，“帝乙是商朝最后一个王，纣王的父亲，“归妹”是嫁女儿的意思。帝乙把他的女儿嫁给了他下面一个诸侯国的首领西伯侯，获得了福祉，大吉大利。西伯侯是周国的首领，就是后来的周文王。帝乙为什么愿意把女儿下嫁给周文王?就是因为看到他品德突出，心甘情愿。很多大佬和领导愿意帮助你，不是你的技巧和套近乎有用，而是人家觉得你这小伙子人品不错，诚恳踏实。在上位者和大佬的眼中，你的一举一动几乎都是透亮的，没必要玩什么心眼技巧，浮躁的世界真诚最可贵。你的修养和品德到了一定境界，自然会得到关照。

## 上六：城复于隍，勿用师，自邑告命，贞吝。
《象》曰：上六“城复于隍”,其命乱也。

城墙坍塌在外面的壕沟里，已经不需要军队的攻打，城内命令大乱，若还是顽固坚守，只会悔恨。万物是周而复始的，否极泰来，泰极也会否来，当经济的周期走了，市场的趋势没了，该放手就要放手，该调整就要调整，不要死扛着不放。现在中国的经济为何这么卷?明明已经过剩了，外部的环境变了，很多企业和老板还在穿新鞋走老路，用过去的方法解决现在的问题，能有利润和出路么？在什么样的周期下，做什么样的事情。顺天应人，就是真正的泰。
