
# 砖瓦厂早报：当富人含房率低了以后

我猜，对人群做区分的话：中国越往上的富人，房产所占总资产的比例越低，其他金融资产，如现金、股票、保险占比高。而中间层的人，特别是线、二线城市中间层的人，房产占家庭资产总比的数值最高，也是目前社会中，状态最不稳定的阶层。相反，三四线、农村人，若是不需要二线城市生活的子女，家庭现金流用于储蓄或消费，生活状态会优于二线城市的中间层。

换旬话讲：当个家庭的主要资产，是持续贬值的房产时，他们未来资产的变化，一定是先收缩资产负债表，再把资产向现金、存款上转移。直到房产占家庭总资产的比例，大概来到30—40%的位置。这个阶段，必然伴随着低婚育率，所谓消费升级的转向。

这个阶段，想到了戴老板的篇旧文，里面提到了万科的项研究找中国房产见顶的主要信号。

所谓「居民部门利息保障倍数」，指的是用居民总储蓄除以每年房贷的利息支出，代表了供房者们的还款能力，其临界值是1.5，即老百姓的储蓄总额如果只能还一年半的房贷，那房价就会见顶。美国2007年泡沫破灭前是1.46，日本1989年则是1.49。而「新房名义市值对居民最大购买力的占用比例」的公式相对复杂一些，即（“城镇住房新开工面积X销售均价）/（居民部门总储蓄＋信用净增长－本年利息支出），数字越低越好，房价下跌的临界点是60%，房价暴跌的临界点是80%。日本泡沫破裂前是81%，香港1997年房价暴跌前则到了88%。所以报告的核心结论是：只要第一个指标不接近1.5，第二个不超过60%，中国的房价就不可能见顶。

明明根据指标，2018年前后，中国房产的信息好的很。甚至到了2020年，照样好的很，离国际警戒线，还是非常非常远。为什么，突然就进入了下行通道呢。万科报告里，从国际国内历史中提取出的核心指标为上述两个。这两个指标都很好理解：一个代表月供能力，一个代表首付能力。根据宏观数据：2018年左右中国居民的月供能力，中国居民的首付能力，都顶呱呱的好。

中国这两个指标是多少呢？先看「居民部门利息保障倍数」，2013年的保障倍数是10.6，2015年是10.4，而且在万科的预测里（假设未来十年的居民房贷复合增速20%），到2020年这个数字是6.6，到2025年是5.3——都离房价见顶的临近值1.5很遥远。而新房市值对居民最大购买力的占用比例」这个指标，在2013年到2015年一直在40%~50%之间波动，而在万科的预测中（假设中国城镇住房新开工面积X销售均价在五年内复合增速不超过11.8%、未来十年不超过5%），中国在2020年和2025年都不会触及60%的警戒线，离80%则更远。简单说，万科报告的结论就是：中国房价距离泡沫化和见顶很遥远，房价到2025年仍是安全的。

可为什么突然就不行了呢？

赵阿Q之前搬运过组数据：为什么这么多人，关注财政部的发布会？因为，这似乎是一场分蛋糕的盛宴。而根据央行官网报告，来看目前存量蛋糕之一存贷款数据：已知2023年人民币存款余额为284万亿元，人民币贷款余额为237万亿元。再参考招行2023年金葵花存款数据：2.5%的人群，包揽了存款总额的80%。大致推测：中国97.5%的人群占拥有存款约为57万亿，人均为5万元。而前2.5%的富裕人群，人均存款约为101万元。

问：基于存量情况，要刺激消费，恢复增速，应该怎样做。也就是说：单单只看排除了中国顶层都2.5%（也就是大家）的群体，剩下的97.5%,人均存款是5万元。而到了2018年，中国的top2.5%的群体（也就是大家），房产套接着套。该置换和改善的，差不多搞定了。随着投资地产收益率开始下降，即掌握着我国最大部分存款的大家，慢慢开始把资产往金融资产上搬运了。

《中国家庭财富调查报告》：2020年后，一线城市高净值家庭的投资性房产占比从45%降至38%，金融资产（股票、基金、保险等）配置比例从35%升至42%（2022年数据）。招商银行《中国私人财富报告》：2021年报告显示，可投资资产超1000万元的高净值人群中，金融资产占比首次超过房产（54% vs. 46%），主因是资本市场活跃和房地产政策收紧。胡润百富榜：2022年榜单中，企业家财富增长更多依赖IPO和股权增值（如新能源、科技行业），而非房地产资产。

一线城市：北上广深等城市受调控政策影响最深，投资性房产交易量下降，金融资产成为主流配置。例如，深圳2021年二手房交易量同比腰斩，同期公募基金规模增长35%。这样就相当于：把煮沸地产的柴火撤掉，去烧别的炕了。

虽然，从万科谭杰的公式看：把中国居民当整体看，首付能力，月供能力都还OK。当这些能力中，真正拿着真金白银的人，用脚去投别的票了。留下群：像我这样的，吃完上顿没下顿的家伙，掏空六个钱包后，在风中凌乱。所以啊，很多宏观的问题，旦做人群分层处理，就好理解多了。而我们这群人的能力，是无论如何，也支撑不起持续走高的面粉价格的。

从2015年到2025年这场旷古烁今的财富大转移中，达到的效果是：1.中国的top富人更富；2.中国的旧钱更稳——以70前后的公务员、国企、教师、医生为主；3.中国的失去农村户口的农二代进城者的净资产收缩，以至于归零，甚至变成负数；4.中国的底层，在转移支付的影响下，稳定向好。其中，人群3还有个显著的特点：现金＋存款/总资产，比值极低。若养育子女＋赡养老人，几乎是手停口停，离婚率极高。但是，那当年，用全社会的力量，来维持房价上涨，使得人群3的净资产向上。
