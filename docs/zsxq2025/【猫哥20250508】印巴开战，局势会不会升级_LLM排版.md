
# 印巴开战，局势会不会升级？

## 嘉宾：梦回安西

2025年5月7日，印巴正式开打，互相出动战机，朝对方境内发射导弹。巴基斯坦方面称，击落了6架印度战机，这个消息一开始让许多人感到夸张，印度这么拉垮？后续经过验证，该消息部分准确。印度空军被证实击落的战机有：一架幻影2000、一架米格29、1-3架阵风，另有架战友无人机（无人机也算广义的战机），是印度从波兰进口的。最令人兴奋的战果，是阵风。阵风是法国装备的最先进的战斗机，技术水平在4.5代，号称有能力和五代机较量。

数量上，有说一架的，有说三架的，尚存争议，但可以确认至少有一架阵风被击落。阵风的残骸视频，是坠落点附近的印度村民发的。起初，村民们以为，击落的是巴基斯坦战斗机。估计视频一出，最坐不住的是法国人。谁都知道，阵风不是歼-20的对手，只不过，吹牛是一回事，被捅破又是另一回事。就在3月，印度才和达索集团签订了采购26架阵风-M（阵风舰载型）的合同，价值75亿美元。现在，印度人重则取消合同，轻则砍价。毕竟在印度人看来，我的飞行员不会有问题，能被击落就是你飞机的问题。相比之下，巴基斯坦方面的战绩，就逊色得多了，只击中了一些民用目标。其实，印度选择开战是有隐情的。

经济上，这几年的巴基斯坦深陷泥潭，经济接近0增长，还连续遭遇了大洪水、大停电、大通胀等各种天灾人祸，损失惨重，不得不靠IMF续命。巴政府已经多次向IMF借款，哪怕对面条件苛刻。政治上，巴基斯坦的局势也很糟糕。2022年，总理伊姆兰·汗被罢免后，巴基斯坦便出现了不小的政治动荡，各种暴乱、袭击、游行，可谓数不胜数，尤其是西北地区。究其原因，就在于伊姆兰·汗是普什图人。熟悉巴基斯坦政治的都知，巴总理的宝座，长期被两大家族垄断旁遮普省的谢里夫家族，和信德省的布托家族。

印度存在升级局势的积极性。话又说回来，兔子不可能允许印度生事。在4·22恐袭发生后，兔子驻巴大使姜再冬表态，“理解巴方合理的安全关切，支持巴方维护自身主权和安全利益，支持尽快对有关事件开展公正调查”，意思是印度对4·22恐袭的调查，不公正。主要是印度那边太急了，事情刚一发生，媒体马上发布了大量资料，跟事先准备好似的。袭击的对象，也足够蹊跷。以前恐怖组织都是袭击印度军警，很少袭击游客，偏偏这次改为袭击游客，还是清一色的外国游客，且正值万斯来访期间，怎么看，都像是有预谋的。

尽管一般情况下，印度会息事宁人，但当前的 世界局势，旱巳不是一般情况，大趋势就是更加动荡，所以，谁也不敢保证莫迪不会头脑发热。印度是投鼠忌器，还是艇而走险，让我们拭目以待。
