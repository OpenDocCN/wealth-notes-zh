
# 提振消费有新招，但效果有待继续观察

有读者提问：“能否谈谈刚出的提振消费专项行动方案?我看着有些奇怪，里面提到开展增收促进行动，居民增收问题是能通过专项行动解决的吗?”专项行动肯定解决不了居民增收问题，增收是因。消费是果，而不是反过来。只是增收是消费的基础，一个提振消费的汇总方案，当然要把增收放在第一条，这是符合逻辑的，而且去年底经济工作会议也是把“增收减负”作为提振消费的重点。

《提振消费专项行动方案》八部分三十条，目前市场已经开始炒作消费概念，但在小镇看来，这个行动方案目前真正算得上有力度的只有3000亿的以旧换新，其他的举措到底能不能落地、力度大小、效果如何，还需要继续观察。要观察的不是这个行动方案，而是行动提到的措施具体推进情况，看对应部门的动作。

不同部门出台的文件特点不一，这份方案就是典型的发改风。也就是通过发函、座谈、调研等搜集各部门与消费有关的举措，汇总在一起，再进行分类，所以条数会比较多，近几年类似方案基本在20条以上，但不要看到条数多，就认为提到的都是重点，不可能的，没有这么充足的资源，尤其具体到今年，更要有主次先后。

这份方案，应该视为各部门认为可以促进消费的政策总目录，具体怎么做、能做多少，还是要看后续的进展。有一些属于长期正确的方向，但不代表是目前的重点，因此也不必过多联想。

针对提问，重点说下第一部分的“城乡居民增收促进行动”。

收入是消费的基础，去年底的经济工作会议把提高城乡居民基础养老金放在了提振消费部分，但等到3月政府报告，还是放回了民生部分，而且只涨了每月20元，仍然是一个常规的涨幅，还是有些失望的，目前国内基本养老金各省差别比较大多的400多元，少的100多元，实在太低了。从这件事就能看出政策的真正倾向。

第一部分，小镇觉得今年有必要关注的有三个：
一是健全最低工资标准调整机制。这个在公号上周《授人以渔：谈谈如何看懂政府工作报告》谈过了，多年来最低工资标准没再调整，今年需要恢复上调，而且要建立长期机制，动态调整，而不是每年临时决定怎么调。

二是战略性力量储备和打通中长期资金入市堵点。“战略性力量储备”主要指国家队和策略调整，具体在星球2月12日《迎接变革新政将如何重塑股市》已经谈了，未来国家队不会像过去一样，股市稍有变动就顶上去，只会在必要的时刻才投入，避免被摸清运作逻辑导致被收割；而中长期资金入市，小镇仍然保持星球1月26日《中长期资金入市，道阻且长》的判断，预期今年能够进入股市的中长期资金是有限的，目前中长期资金方顾虑的点，证监并没有解决。

三是解决拖欠账款。今年由于加大发债，地方财政情况有所改善，应该能清偿一批，这一块会有具体措施。

年内值得关注的还有几个：
一是以旧换新。3000亿的以旧换新是提振消费的顶梁柱，也是目前最大的支持性政策，不排除年内增加投入的可能，这也是发改最擅长的刺激方式。

二是完善全口径消费统计制度。目前消费统计口径比较小，只包括商品和餐饮消费，国家现在对服务消费的重视度很高，有必要扩大统计口径，因此这件工作预期今年会重点推进，调整后可以迅速提高消费的总额度。

三是央行的结构性货币工具创设。只要中央定了，各部门就要围绕一个目标发力，所以央行也要为了提振消费推出新的工具，上周央行还专门强调“结构性降息也是降息”。

促进消费的结构性货币工具，估计就是三种。第一种是促销费的再贷款，注意不是贷给个人，而是贷给消费设施，也就是商场、购物中心、物流配送中心等市场主体，具体可参照“第23条完善城乡消费设施”。

第二种是配合重新鼓励个人消费贷，可能针对个人透支消费面临的还款难，提供一个纾困工具，允许延期还款。

第三种是普惠小微企业贷款延期还本付息支持工具延期和扩大。

四是财政的两个贴息。也就是方案提到的个人消费贷款和消费服务业经营主体贷款给予财政贴息。但这两个贴息的效果不宜高估。

个人消费贷重新被鼓励了，但要注意财政的贴息是给到银行，而个人消费贷主要是通过互联网等平台获得，财政的贴息发挥的效果要大打折扣；而消费领域的经营性贷款贴息，这也不好说，因为前几年房贷利息高的时候，有一批人用经营贷腾挪房贷，所以经营性贷款才出现暴增，后来存量房贷利率大幅下调，这批人非常吃亏，所以经营贷规模也降下来了，就算贴息，吸引力也有限。

五是扩大外国人入境消费。去年扩大了入境免签的范围，今年还要继续，这个也是确定的。

其他的就需要观察了。

有的是示范性的。主要是花钱，效果不好说，关键规模上不去。比如国际消费中心城市建设，还有全社会适老化改造、多层建筑加装电梯、发展老年助餐服务等，目前只能作为经验探索。

很多是表态性的。比如休息休假，这不是监管能解决的问题；新型消费也是刚刚起步，到底怎么促进还没有想法，需要等地方实践；而冰雪消费、文体旅游、消费品牌、放心消费环境、消费设施、营商环境等，这都是常规工作，这次方案并没有提出新的措施或概念，基本上之前怎么做，那就继续做，只不过工作成果可以算入提振消费的成绩单。

有的属于正确但是难度很大，需要很长时间才能看到明显改善的。提高居民收入是一个，另外就是服务业扩大开放和内外贸一体化。

前者提出很多年了，目前制造业方面已经全面开放了，而服务业有很多限制，国家也希望能够继续借助外部力量推动内部改革，这也有利于提高服务供给水平，从而促进服务消费提高，但是从实际落地来看，进展很小很小。

因为如医疗教育文化养老等服务，不仅仅是简单市场活动，还涉及到深层的体制改革，尤其这几年正在对过度市场化进行纠偏，扩大开放就需要打断这一纠偏过程，或者需要高水平、高难度的协调，目前没有看到哪个部门愿意牵头进行这项复杂的改革，于是都认可要扩大服务业开放，但目前还没有具体的改革方案，未来一段时间很难看到实质变化。

而内外贸一体化，这是近两年的重点，国家对等标准的态度发生变化，发现这些标准与国内要推进的改革方向基本一致，再加上随着中国企业走出去，面临内外贸标准不一致的问题，而考虑到全球贸易环境恶化，需要为外贸企业寻找兜底，支持外贸商品转内销，种种原因使得内外贸一致化受到重视，但这件事也急不得，工作量很大。

还有的需要观察效果。如育儿补贴的关注也很高，一些地方已经开始实践了，主要两种模式，一是生育后一次性发放，二是分几年逐年发放，发放范围也有差别，有的是不分几孩，还有的是二孩及以上。目前国家层面的标准还没有公布，但大概率是给一个原则方案，具体怎么补贴还是由地方决定，补贴的水平也不会高，能够达到当地人均年收入的10%就不错了，理论上也可以略微提高中小城市和乡镇的生育率。

总之，类似这种几十条的方案，就是一个汇总，有新招但是并不多，还是要降低期待，观察后续的效果。
