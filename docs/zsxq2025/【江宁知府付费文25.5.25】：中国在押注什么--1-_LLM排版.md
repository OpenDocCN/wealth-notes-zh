
# 中国在押注什么

原创：印闲生 公众号：江宁知府 25年5月20日

《中国的下一个“中心任务”》里提到，制造业、科技、金融、军事是大国竞争的四个核心领域。目前中国在制造业领域已经领先美国——这通常是历史上崛起大国最先实现突破的赛道，如果非要在剩余科技、金融、军事三条战线中挑一个，科技的确是最优赛道。

因为军事竞赛意味着撕破脸，金融赛道则更像是其他三条战线获胜后的“战利品”。

所以原文给出了这样的总结：中国未来十年的基本策略是在力保制造业优势的前提下，科技和 GDP 并重前行，淡化军事对抗与全球治理博弈，同时弥补金融短板。

进一步延伸，可以表述为以下三点：

1、中国的首要目标是实现更多文章加薇：pep458伟大复兴，这一目标只能通过工业革命级别的技术领先与转型来实现；
2、二战结束以来，国家发展的可能性事实上被限制在一个狭窄的框架内——包括固有领土边界与各种国际规则，军事工具并非重塑全球秩序的首选；
3、台湾是个例外，对台湾来说，经济工具已经被证明是无效的，发生不可控事件的可能性非常真实。

在此基础上，本篇谈一个更宏大的话题——中国在押注什么,以及这种押注的利弊。

迄今为止，人类历史上共发生过三次工业革命。

第一次工业革命诞生在英国，在瓦特改良蒸汽机之前，生产所需动力依靠人力、畜力、水力和风力。伴随蒸汽机的发明和改进，工厂不再依溪河而建，很多以前依赖人力与手工完成的工作开始被机械化生产取代。

从1815年的维也纳会议到1870年的普法战争，英国是全球唯一的工业化强权，是全球30%工业产品的生产地(1870年)。

1815年至1914年也被称为“帝国世纪”,或“不列颠治世”。

某种意义上讲，第一次工业革命和第二次工业革命其实是连在一起的——第一次工业革命的主要意义在于它为第二次工业革命提供了研究方向与部分技术基础。

发生在1860年代至1914年间的第二次工业革命，是真正奠定当前人类生活方更多文章加薇：pep458式与一战、二战国际格局的大事。

人类旅行的方式(火车、飞机、轮船、汽车)、交流的媒介(电话、电报)以及我们熟悉的内燃机、电动机、交流发电机、变压器和整流器、白炽灯、电磁波、录音机、油印机、各种化学化工产品等，都是这一时期诞生的。

1858年的跨大西洋电缆路线图，北美和欧洲之间的通信时间从十天(即通过船传递消息所需的时间)减少到几分钟。

一位1800年的工程师如果穿越到一个世纪后，面对1900年的电力系统时，他会感到震惊和完全无法理解。

与之相较，如果让1900年的爱迪生(1847-1931)穿越到2000年，他会非常熟悉几乎所有的工业系统主要组件，因为它们跟一百年前的设计原理基本保持不变。

实际上，今天我们日常使用的汽车发动机(在概念上仍然是四冲程奥托循环发动机或更高效的柴油发动机)、摩天大楼(仍然用结构钢和钢筋混凝土建造)、无线电(仍然由赫兹波携带)或印刷图像(仍然由半色调技术产生)等，大都保持着第二次工业革命的模样。

而一些更简单的发明，如回形针、啤酒瓶上的卷曲瓶盖、带刺铁丝网等，则保持着跟一个多世纪前几乎完全相同的样子。

大多数二十世纪的伟大创新都不是“0到1”的发明，而是对镀金时代技术的定性改进。

可以这样讲，在1945年原子弹爆炸之前，人类的科技进步完全局限在第二次工业革命范畴内。

这一时期，英国凭借工业化先发优势主导世界，但随着技术扩散，拥有更多人口和组织能力的德国、美国和苏联崛起，很快在体量上压倒了英国。

此即前文提到的“更大规模的高度组织化国家”将取得胜利。

为什么苏联能够在斯大林时期迅速完成工业化、成为两极霸权之一呢?

答案也在于此。

所谓“斯大林模式”,本质上就是以国家组织推动的方式抄第二次工业革命的作业。

二战期间，在广岛原子弹爆炸前的那一刻，美国、苏联、德国、英国和日本的武器装备水平基本处于同一档，美国略领先于日本，德国略领先于苏联，但主要制胜法宝都是靠“数量压死质量”。

假如当时德国的核心人口与工业体量跟苏联一样，谁来了也回天乏术。

斯大林在1931年的讲话：“我们落后发达国家50-100年，必须在十年内弥补这个距离，要么我们这样做，要么我们就会被压垮。”

1939年的苏联是一个人口1.7亿的庞然大物，比今天的俄罗斯还多3000万人。

俄罗斯军工企业很大程度上仍在苏联时期的技术框架内。俄乌战争已经向世人表明，即使在当下“趋乱”的环境中，依靠武力夺取利益仍将面临巨大反噬。

第三次工业革命有两个大的分支：第一是原子能技术，第二是信息技术。

我们先谈第一个。

1939年4月2日，德国物理学家格奥尔格·约斯和威廉·汉勒最先提出将核能用在军事领域的可能性，但由于当时主持核研究计划的海森堡错误判断了实验方向，让希特勒以为开发核武器的费用将会过于庞大，最终没有在这个方向上下重注。

与此同时，英国、苏联、美国和日本也均启动了核武器研发计划，其中以美国的“曼哈顿计划”最为著名。

至于原子弹在1945年美国主导战后国际格局时发挥的重要作用，此处就不再详述了。

1945年11月，盟军最高司令部拆除日本一座用于核武研发的加速器设施，该设施由日本物理学家荒胜文策主持建造。荒胜文策在1943年时已设计出超速离心机来分离铀235，但一直没有投产。他也设计出可行的原子弹制造蓝图，但因为铀不够而在投降时尚未完成。

二战后，原子能技术进行了分化：一个方向是威力更大或更受控的核武器，另一个方向是民用核能。

在“武器”这条路线上，随着氢弹试爆成功，已基本走到了尽头。

而在“核能”这条路线上，以核裂变为技术基础的核电厂并没有从根本上重塑人类能源模式，它更像是一种补充，真正的颠覆性影响要等可控核聚变落地。

当下，可控核聚变已经是一个“微微开口的宝盒”，正等待着人类摘取，只不过这条道路会走多久，尚不得而知。

值得注意的是，每当“科技革命宝盒”近在眼前时，各国的统一做法都是政府投资主导，由政府协调企业和科研院所力量进行集中攻关。

只有不清楚哪个方向有“宝盒”时，才会放任市场进行低成本尝试。

《人民日报海外版》(2024年11月12日第05版)

可控核聚变研究的成本极高，需要巨额资金投入以支持先进的实验设施和技术开发。根据中国聚变工程实验堆(CFETR)总体工作计划，2050年将建设聚变商业示范堆，完成人类终极能源。

第三次工业革命的另一个分支也就是大家熟悉的信息技术革命，包括计算机、互联网等，也被称为“数字革命”。

信息技术革命是从1970年代开始的，由于美国完全主导了这一进程，使得华盛顿最终兵不血刃就赢得了冷战。

作为信息技术革命的亲历者，我们能够大致梳理出它带来的几方面影响。

1、军事层面，信息化战争成为主导形态，第二次工业革命时期注重的“性能指标”开始变得没有那样重要。

以刚刚结束的印巴空战为例，或许阵风的某些账面性能高过歼-10C，但并没有什么作用，理论上体系占优的一方可以实现对另一方的绝对碾压。

2、经济层面，信息技术革命诞生了海量财富与科技巨头，美股“七姐妹”——苹果、微软、英伟达、Google、亚马逊，Meta 和特斯拉中，只有特斯拉稍微跟第二次工业革命沾点边。

根据世界银行的数据，2024年全球数字经济规模已超过50万亿美元，传统制造业衰落的美国正是因为吃到了信息技术革命这杯羹，才得以保持世界第一强国的地位。

3、人类生活的方式改变。

爱迪生穿越到今天或许对飞机、火车、汽车、大楼不感到陌生，但他肯定没见过iPhone、微信、抖音和电子支付，相较于原子能技术对“国力”的提升，信息技术具有更大的普惠性。

随着量子计算机迈向实用化的开发在全球取得进展，中美的脱钩正在加剧

据《日经新闻》引述麦肯锡的报告，中国在量子技术领域的投资远高于欧盟和美国。目前中国与欧美已在大量尖端技术领域实施脱钩，互相封锁了量子技术的信息，对彼此进展也不甚了解。

值得一提的是，当前美国政府及科技界似乎更倾向于认为“信息革命2.0”(生成式人工智能)是最触手可及的那个“宝盒”,并为此下了重注。

但也有一种观点认为，下一次工业革命不会再来自信息领域——与其他信息时代的技术一样，AI 所做的只是协调、组织和优化现有的生产模式。

真正的颠覆性技术或许来自新能源、新材料、新储能方式、新生物技术等方向，比如出现一种新的建筑材料，完全颠覆我们今天的基础设施建设形态。

总而言之，这种对未来革命性技术发展方向的判断，很大程度上会决定一个企业乃至国家的前途命运。

介绍完三次科技革命的故事后，下面把目光聚焦到当下的中国。

大佬文集圈

接下来这段话是美国学者引用自国内的讲话稿(越来越多的海外学者开始关注中国的官方讲话与政策文件),我们把它复制过来——

“历史经验表明，科技革命总是能够深刻改变世界发展格局。

16、17世纪的科学革命标志着人类知识增长的重大转折。18世纪出现了蒸汽机等重大发明，成就了第一次工业革命，开启了人类社会现代化历程。

19世纪，科学技术突飞猛进，催生了由机械化转向电气化的第二次工业革命。

20世纪前期，量子论、相对论的诞生形成了第二次科学革命，继而发生了信息科学、生命科学变革，基于新科学知识的重大技术突破层出不穷，引发了以航空、电子技术、核能、航天、计算机、互联网等为里程碑的技术革命，极大提高了人类认识自然、利用自然的能力和社会生产力水平。

为建设世界科技强国而奋斗——在全国科技创新大会、两院院士大会、中国科协第九次全国代表大会上的讲话(2016年5月30日)

简单来说，中国在押注一个前景，这个前景就是我们现在正处于下一轮科技革命的开始阶段。

人类科技的进步不是线性的，突破往往发生在短短几年或十几年间，并导致国际政治权力和财富不成比例地流向首先跳跃成功的国家和个人。

在中国决策者看来，新兴技术可以推翻现有的政治经济秩序，巨大的变化意味着巨大的机遇——大英帝国和美国之所以崛起成为全球霸权，是因为各自开创了一场全球性科技革命，而中国在十九世纪二三十世纪积贫积弱，则是因为一次又一次地错过了工业革命。

现在，历史正在重演，人类再次处于科学剧变的悬崖边。

直言不讳地说，当下中国的中心任务之一就是建立一个能够将人类推向新技术前沿的工业和科学体系。

近些年中国以国家主导的形式投资了大量科研基础设施

文章最后，做几点总结。

1、中国的科技战略基于两个押注：第一，世界确实处于规模可与工业革命相媲美的经济转型的风口浪尖；第二，如果这场新技术革命发生，中国有能力引领它。

虽然这两个押注都存在一定的不确定性，但很大程度上也是基于当前形势的一种理性与乐观选择——乐观主义者更习惯于捕捉“可能性”。

2、回顾人类科技发展史，新技术的探索往往伴随着大量争议，在取得真正的突破之前很难界定成果。

尤其当实施政府主导的投入时，不排除科研人员烧了大量来自国家的科研经费，最终却竹篮打水一场空。

正因如此，只有当面向“宝盒”冲刺的时候，才更需要由政府协调资源集中突破，而在其他大多数时期，仍将以市场主导的探索为主。

3、科技竞赛本身就是大国竞争的常态，它与地缘政治竞争从来都是一对孪生兄弟。

套用知名著作《美国反对美国》里的一句话：

“技术优势已成为美国人民族优越感的来源，想压倒美国人，你必须做一件事：在科学和技术上超越他们。”
