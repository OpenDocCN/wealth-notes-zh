
# 中国实际面临的困难和挑战，要低于纸面统计

正在候机，就不拿出电脑了，手机简单谈谈。自去年决策层关注提振消费、投资于人两大转型，国内外密切关注中国的消费问题，论战不停。

一方是以余永定为代表的官方智囊，认为中美消费数据存在很大差异，美国消费以服务消费为主，中国存在规模优势等，总之认为“中国消费不足”是伪命题。

另一方则主要是金融机构、部分经济学家，比如大摩中国团队的邢自强等，认为中国消费不足是目前亟待解决的问题，而之所以消费不足，是因为国家没有真正转型，在“投资于人”上雷声大雨点小，用于民生保障的资金投入不足。邢自强就列举数据，认为中国民生保障支出仅占GDP的8%左右，而发达国家一般在20%，发展中国家大致在15%到20%。

总之认为决策层存在认知惯性，不敢加大赤字、加大投入，呼吁国家应该推出大规模消费刺激政策。加大刺激，也是大摩中国团队一贯的主张，只是不同时期理由不同，去年及以前是为了避免如日本的衰退，今年一季度是为了对冲美国贸易关税冲击，而现在理由调整为提振消费。

不过需要说明，邢自强忽视了中美民生保障投入的巨大差异，对只有中国有的投入直接忽略，仅按照美国标准统计。比如美国至今没有全国性社保体系，而中国不仅全国覆盖，还有计划补充社保不足的社保基金，如果把这一块纳入，也就是统计财政支出和社保基金支出，那么民生投入占比就提高到16%以上，这还没有计算脱贫攻坚等投入。

作为专业金融机构尤其外资机构的分析师肯定不会提中国特殊性，这一点需要自行注意。

在“中国消费”上，官方智囊与金融机构的分歧巨大。也有试图弥合的，如粤开证券的首席经济学家罗志恒26日就发了一篇文章，从统计核算和实物量角度分四个层次对比中美消费，不过这篇文章他已经删除了。

罗志恒的分析结果是：中美人均消费支出差距大，实物消费差距小；中国人均消费、结构有较大提升空间，消费水平区域差异大。

这篇文章没啥敏感的，但作为证券行业的首席，发这种文章不合时宜，显得“不专业”。金融、经济学以及需要同美西方进行国际对话的领域，分析一定要注意采取西方能够认可的专业方法。比如邢自强的大摩团队的分析，就是标准的西方逻辑，虽然对中国宏观政策的判断偏差很大，但更能被接受，而罗志恒的文章，想要兼顾两方，明显有余永定等官方智囊强调中国特殊性的特征，这就不符合西方标准了，很容易遭到业内“不专业”的质疑。

或许这就是删文的原因。

采取何种分析方式取决于自身定位。如果是金融机构、强调国际认可的学术界，还是要用后者，弱化中国特殊性，采用被西方认可的标准分析；但如果是要真金白银投资某个领域，或者从事中国宏观政策决策，那就必须重点关注中国特殊性。比如在消费上，是“充足”还是“不足”，非常关键。如果认为在目前经济发展水平下，中国消费充足，那么投资的时候，就不能妄想短期内能够大幅提高消费，要把重心聚焦在存量竞争；反之，则应该关注增量。

从分析政策趋势的角度，小镇更倾向于关注中国的特殊性，毕竟金融机构等依据的统计数据直接对比，实践指导价值很低，主要用于吸引投资者。

比如中美消费，目前中国GDP以美元计算，是美国65%，所以近年来一些机构和学者认为中国经济表现差，中国较美国GDP差距拉开；但一个诡异的现象是，在实物商品和餐饮服务消费方面，中国大概是美国的九成，这是直接以美元计价，并未考虑物价因素。那么，中国以较美国更低的GDP，实现了更高的消费，到底是谁相对消费不足呢？

还有最近被关注的重点商品消费，各方论战经常提到的就是肉类和蛋白质消费，罗志恒也把这一项作为分析的重点实物量之一。

这个指标差异很大，最近经常被拿来认定中国消费不足的，就是“中国人均肉类消费39.8公斤”，认为远低于欧美水平。

但了解中国与欧美各国统计不同的就知道，这是一个笑话。中国统计的肉类消费的确是39.8公斤，但仅限哺乳类也就是猪牛羊，而且是净肉不包括内脏杂碎血液等，也不包括加工后的火腿肠等肉制品，仅仅统计居民购买消费的部分，如果一个人去食堂餐厅吃肉或者自家养着吃，就算吃1吨肉，在统计上也是零肉类消费。

而肉类实际还要包括鸡鸭鹅等禽类，以及水产品，统计蛋白质摄入还要加上植物蛋白。根据联合国粮农组织数据，2021年中国蛋白质摄入124.9克，超过美国的124.3克和欧盟的113.7克，其中六成来自植物蛋白。而美国蛋白摄入75%来自动物蛋白。

这一蛋白质摄入数据也被拿来论证中国消费不足，吃不起肉。但真的这样吗？

各国统计差异很大，就如韩国，海带也算肉。在小镇看来，进行跨国比较的时候，人均肉类占有量要比消费量，更有价值。

根据2024年官方统计，中国人均肉类、禽类、水产品等占有量，按照最保守的估计也得120公斤。而考虑到肉类占有量，也就是生产量的统计大大低于实际，因此实际人均占有量还要更高。

那么问题就来了，中国人有可能实际肉类消费只有占有量的三分之一吗？也就是吃一斤、扔两斤？如果从人均占有量来看，美国人均优势并不高，再加上植物蛋白摄入量，还要有新的变化。

很多统计指标都存在类似问题，这也是同样的数据，强调中国特殊性的官方智囊，与强调普适性的金融学术机构，判断差异巨大。

不仅具体指标如此，实际GDP核算也有巨大差异。这几年，网络上经常质疑中国GDP增速下降，这确实是事实，也跟中国GDP核算历史性改革有关，不过这一点很少被提起。

金融机构和学术界不提很正常，这不利于论证经济不行；而官方智囊也不提，就很有意思了。

2020年GDP核算从分级核算彻底改为统一核算，最关键的变化就是从“上管一级”变为“下算一级”。

过去是地方统计本区域然后上报上级审核，也就是GDP，是“上管一级”，就导致地方汇总后，往往大于上级GDP，比如中央核算全国要明显低于各省核算GDP加总。

这就推动了改革，改革为“下算一级”为核心的全国统一核算，就是由上级部门核算本级和下一级GDP。国家统计核算全国和各省，把各省下发，然后各省据此核算本省和下属各市GDP数据。中央核算的全国数据与各省汇总误差下降到1%以内，很接近美国联邦与州汇总0.6%的偏差值；而省和地市汇总，采取严格对应。

这就在机制上天然压低核算，地方上近十几年也开始弱化GDP增量，发达地区想要低报以减少缴税和转移支付，落后地区想要多获得国家转移支付和财政补贴，中间的也有比增长更重要的考核指标。

这就导致，2020年之后的增速，仅因为核算方法的变化，一定低于过往，这个值不好估算，但1个百分点很可能有，毕竟十几年前，GDP才四五十万亿的时候，各省汇总的GDP能比国家汇总的高几万亿，虽然国家会挤水分，但水分必然存在，比如辽宁。

核算方式的重大改革，与两大冲击重叠，反而不好评估。小镇个人觉得，压低GDP数据，是一个更优先的选择，没有必要把数据搞得太高，尽可能挤干净水分，现实面临的挑战实际低于账面数据，这更安全。

而分析决策倾向，也需要把这一点纳入考虑。
