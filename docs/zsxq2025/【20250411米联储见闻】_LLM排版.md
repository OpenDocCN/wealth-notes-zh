
# 20250411米联储见闻

惊涛骇浪的一周，中国的资本市场展现了超强的凝聚力和稳定性，表现之好足以让过去不看好中国的人改变信仰。胜利是最好的团建，这次面对川普的关税攻击，自上而下，各个行业和领域都展现出了正向的精气神，难以想象这样的事情如果发生在十年前会是什么样子。从这个角度看，过去十年，中国各方面的进步和成长都是经得起检验的。

过去这一周，中美之间发生了猛烈的碰撞，所产生的结果和后续影响还会持续，这将会是出于历史转折点的关键时刻。庆幸的是，我们有靠谱的管理团队，高素质的国民，较好的经济基础和基本面，而此时的美国可能真的处于历史上最虚弱的阶段，这一点很重要。当然，美国毕竟是美国，斗争也才刚刚开始，开好头，但对未来的预期不宜太乐观，而是要有理有据，一步一个脚印的把事情做好，把优势压实了。

在对美国关税做出三次对等反制后，中国宣布不再跟美国进行数字游戏式的斗气，因为任何人都知道关税超过100%对贸易而言就是天花板了，再往上加纯属幼稚的表现。中国虽然可以跟美国斗，但能稳住更符合中国的利益，所以在继续强调奉陪到底的同时，也强调对谈判大门敞开。领导立刻走访越南大马和柬埔寨三国，其实主要是去越南敲打，但单独去不好看，顺便走走其他国家。从这个操作来看，中国既表现了自己能够反制的实力，又展示了作为成熟大国的理性。中国不是要做跟美国缠斗的武疯子，而是要多极世界的定海神针，一个有肌肉但同时还有脑子的大国才是大家都需要的。

川普在中国没有做出第三次反制后，也很识时务的松口，愿意跟中国达成协议。可以说过去一周的金融市场，就是双方的主战场，五个交易日的反复博弈，最后各自窥觊了对方的底牌，没有这个交手回合，中美是不可能坐下来好好谈的，尤其是美国，所以中国一直没有接触川普，就是等待这个可以互相摸底的机会。只有实质性的过招，才会让谈判更有效率和诚意。不过，中美目前仍缺少沟通契机和管道，有可能转向非公开的隐秘渠道。

央行3月以来四次明确提及择机降息降准后，《证券日报》10日发文称，进一步实施好适度宽松货币政策。但是在过去一周，大家也看到了，相比于这些政策，国家队、主力、央国企扎扎实实的拿钱入市才是稳定信心更有效的工具。这些凝聚共识之下的行动力，远比货币政策更能激励人心。本周A股的表现给管理层后续继续跟美国博弈提供了很大的底气，目前中国就是要积极的开展微观工作，具体工作，为当下的经济注入稳定性、确定性。其实美国这种战争模式的冲击反而能让行政部门高度重视起来，在应对问题的过程中解决一些过去长期以来懒政倦怠的毛病，对政务服务水平有积极促进作用。比如商务部就提出将帮助出口遇阻的外贸企业开拓国内市场，深入开展内外贸一体化工作。

很多人说，川普在中国反制后，立刻团结其他国家把枪口对准中国，所以中国依然是处于极度被动的局面。但其实，川普在一周之内将如此严肃专业的关税问题视如儿戏，朝令夕改，试问哪个国家敢真的相信川普的承诺，跟这样的人谈协议，真的能保障自己的利益吗？尤其是乌克兰被背刺的遭遇就在眼前，谁敢把国家和民族的命运压在川普这种不靠谱的人身上。所以表面上看，川普似乎玩了一手很漂亮的联众包围，但实际上，没有坚实的信任基础，这种包围圈纸糊的都不如。

今天网上有帖子说，越南访美回来后，立刻就会被中国棒击，一些一带一路的基础建设，包括越南向云南广西买电的合作都会立刻产生紧箍咒效应，虽然网文未必真，但这个态度是必须的。这就是所谓的实力底气，没有那个大国会相信自己说几句话就能让小弟们乖乖听话，恩威并施之余，能让你立刻吃苦头才是真正的杀手锏。领导这个时候立刻访问东盟三国，也足见高层的重视程度，胡萝卜和大棒永远都是最佳组合。越南如此，李嘉诚又何尝不是如此，如果中国连港口买卖都叫不停，拿什么跟美国斗。

实际上，关税确实是一张很好的牌，美国政府要是能打好这张牌，可以慢慢用不确定的预期耗死中。精挑喜欢行业和产品，结合经济节点加码，要挟谈判，节奏掌握的好，可以耗死中国，加上打击面小，温水煮青蛙，管理层的重视程度，应对烈度都达不到现在的水平。由于波及面被分而化之，反抗的决心也不团结，反而容易对中国经济造成慢慢腐蚀拉胯的作用。但是这样的操作需要政府有极高的专业水平和耐性，需要做精细专业的准备，需要大量的技术官僚和人才，已经系统性的支撑和机制保障。过去的美国是具备这个能力的，这也是全世界害怕美国的原因所在，那些金融机制的建立，那些长臂管辖的机制，都是高手为美国量身定制的兵器，但是如此一张好牌最后被川普打成了笑话，这不但暴露出川普团队内部缺乏真正的精英人才，对贸易、外交政治的理解非常肤浅，甚至没有能力做出长期系统的战略布局和应对策略。布林肯和沙利文确实已经不如前辈了，但是川普的团队跟他们相比，简直是蛮夷水平。如此一张好牌这么乱打出来，简直是给中国帮了大忙，减少了博弈的时间成本和陷阱，连舆论压力都降低了。

原本美国进口商还打算借着关税去跟中国出口商压价钱，但是川普一下子把关税提高到145%，直接的结果是中国商品或者涨价，爱买不买，或选择下架甚至退出美国市场。不少中国卖家表示，现在的情况并不是加关税那么简单，而是更个结构性的变化，关税不仅会导致海关清关延迟，还会导致物流成本大幅上升。亚马逊上约一半的卖家来自中国，仅在深圳，就有超过10万家亚马逊商家注册，年营收高达353亿美元。中国海关总署的数据显示，2024年中国跨境电商进出口2.63万亿元，按年增长10.8%。对川普来说，这确实是张巨大的筹码，如此多人的生计，如此大量的贸易额，如果一点点挤压，会有很多叫牌和让利的空间，但是如今一次把牌桌砸了，即便当下很痛苦，当年对这些卖家来说，少了犹豫不决就是立刻另谋生路，反而节省了时间和试错成本，不会积重难返。而现在，或者提高卖价，或者退出市场，最后都是美国消费者买单。

也有中国商家表示，会第一时间减少在亚马逊的广告投入，过去这部分要占利润的40%，如今利润减少，先砍掉这个预算。美国约85%的钢笔和铅笔产自中国。2022年，美国人就从中国供应商那里购买了价值4.3亿美元的钢笔，中国在全球文具市场中占80%的份额。在美国家庭中，仍有三分之一的蒜、一半的低价家具，以及近80%的圣诞玩具是中国制造的，70%的相框来自中国。南华早报说，白宫办公室里川普的照片和他签署声明用的钢笔，可能都来自中国，这些看上去不起眼的东西，很快都会以高额的价格侵蚀美国居民的生活成本。美国的零售商预计，在现有库存耗尽后，从6、7月开始，美国的电子产品价格将大幅飙升，实际上也在暗示，中美关税谈判的时间窗口也就是一两月而已。苹果在这个时候空运了600吨，接近150万台手机，显然对于可能出现的货品荒的非常担忧。

从来不用为物资担心，享受着全世界低价供血的美国人也开始大量囤积商品，有美国居民告诉美国媒体，他现在每天都会去买两瓶橄榄油，家里的橄榄油已经多达20瓶，这是55岁的他过去从未经历过的事情。美国的机构认为，川普现在的操作就是在没有救生筏的情况下，把美国人民赶下汪洋大海。

按照白宫透露的信息，川普团队最初的设想是34%的对华关税会直接逼中方谈判，而美方已经做好了以中方发表全面禁止芬太尼及前体的生产以换取美方撤回20%加征关税的协议。也就是说，美方最终的底线是对中方加14%的关税，比其他国家的10%略高，但也能让中方接受。不仅美政府收到了钱，也进一步解决了芬太尼问题。川普是打算用这种钝刀子割肉的方式占便宜，但是没想到，中国提前预判了他的预判，以意想不到的强势反制让川普团队接下来的技术动作彻底变形了。也让美国拿着关税筹码跟中国上谈判桌的想法彻底破产了，因为现在中国手里也有关税筹码，中国或许愿意帮你管一管芬太尼，但是条件肯定是对等取消关税，那14%你也别想了。

因为美国的政治制度是需要向选民作秀才能得到支持的制度，所以原本可以好好打的关税牌，被打得高调、强势，但是最终变成了荒腔走板，红脖子们看不懂专业的关税策略，只能看到关税数字和打击的国家数量，川普显然也愿意彰显自己横扫一切的威慑力，上下一心，最终就把美国的一张好牌，打得稀烂。所以教育还是重要的，子孙不长进，祖宗们留多少家底都会被败光的。
