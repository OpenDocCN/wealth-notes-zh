
# 学习笔记，2025.2.2

1. 提问越具体，答案越明确，甚至对一些人来说，问题清晰描述出来的时候，答案就已经跟着出来了。很多人糊涂的原因也在这里，他不会提问，找不到问题所在，只会关注一些笼统、宏大的概念，导致自己陷入虚无。

2. 人们常常陷入这样一种认知误区：认为自己自身的情况独一无二，就好像被幸运之神特别眷顾一样，从而觉得自己能够突破常规。但实际上并非如此。从宏观角度来看，个人的经历并没有什么特殊之处。回顾历史就会发现，新出现的事件往往是旧有模式的重复，根本不存在真正意义上全新的事物。历史虽然蕴含着无尽的经验与智慧，然而人类却始终未能真正领会其中的教训。

3. 一个人，一旦习惯把自己代入“弱者”视角，就很难再有有什么成就了，在他身边也会很危险。因为他会认为，自己的弱，是外部环境造成的，外部环境有责任对自己进行扶持，慢慢就会成为吸血鬼，吸食所有自己可以接触到的人。人如果觉得环境对自己造成不好影响，最该反思的是，自己为什么会进入到这样一个环境，为什么没有去选一个自己更喜欢的环境。自己身上的事，优先在自己身上找问题，改进之后，受益最大的也是自己。

4. 普通人最大的底气不是善良，不是品格好，而是财富。有了财富，才有行善的资本，才有拒绝的底气，才能有更多的试错机会，没有财富，善良和品格就像一个空头支票，很多时候都无法对他人产生实际价值。

5. 很多人认为自己和成功的人最大的差距就在那些他们展现出来的东西上面。混不进好圈子觉得是自己穿着打扮太低廉，要用奢侈品才能混进圈子，觉得要学习这些圈子的餐桌礼仪才能坐在一起交流。这些都是结果，不是原因，想成为富一代，就该多去看富一代的打拼过程，而不是只看最终成就和二代们为了维护地位制造出来打游戏规则。

6. 一个人愿意给别人多少信任，很大程度上取决于自己预估的对方背叛成本是多少。比如借钱，老同学一张嘴就是三五万，你借了，说明你认为对方的信誉值这个钱，或者说，你认为即便他不还，三五万买断这段关系，自己也不亏，可以避免以后更大的坑。反过来想也一样，你想要取得一个人的信任，就要看自己能拿得出手的东西在对方心里有多大分量，信誉、抵押物，总得有点什么，不可能陌生人上来就卖惨求这求那。

7. 一段关系能够持久存在，一定是有什么东西能够在这段关系里持续流动，利益、情感、物质、思想，双方都能在上面有所得，才会倾向于去维护这段关系。如果只有一方在持续给予，而另一方只懂得接受，这种不对等的关系就无法持续下去，甚至，给予方会因为投入太多无法割舍，用尽各种手段留下接受方。

8. 很多人焦虑，是因为看不清自己，如果知道了自己的喜好，找到了愿意奋斗终生的事业，就没工夫焦虑了。如果你不去找到自己焦虑的根源，就会被别人塑造出来的其他焦虑左右，今天报课学技能，明天买药谈养生，增加越来越多的焦虑。多向内求，能看透你的，只有你自己，你不愿意拿回大脑的掌控权，就会被别人灌输各种杂七杂八的概念。

9. 人，知道的越多，不知道的就越多，见识过更大的世界，就会知道自己的渺小。只有无知和停止学习的人，才会认为自己无所不能。持续学习不该只是一句口号，而是想要持续成长必须要去做的行动。

10. 不管什么时候，只要开口要钱或借钱了，会使对方立刻下头。这种下头，也超出了对方的预期。对方是真的想给。但是，你开口了，她就觉得失望。她就是不懂男人，若是她主动出去找份工作，哪怕一个月收入三千元，孩子交给保姆去带，她会生活的非常好，因为男人会格外的尊重她，主动给她钱，她越是不工作，越是主动要，男人越瞧不起她。
