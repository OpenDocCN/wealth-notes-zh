

# 2025:忠告与建议

原创 王福重  
2024年12月31日 07:20 北京  

## 前言

全部年终秀加起来，不如我这篇。

所谓命运，就是生在哪个时代，开元盛世和天宝后期天壤之别，其实间隔不过几十年而已。投胎真是技术活。

有些变化是周期性的，花谢了会再开，有些却是结构性的，黄鹤一去不复返。而世事变迁并非匀速运动，某些时点，特别剧烈，更需当心。

首先要当心产品特别是食品安全，竞争激烈或者太卷了，生存边际的厂家商家大增，必然趋向败德，消费者则处于危险之中。

更不能愚昧消费，比如绝大多数保健品，比如非转基因食品（转基因更安全更健康）都是哄愚昧消费者的。奢侈品也少买，都变穷了，面子不要太要紧了，省钱过冬为上。

祸不单行，经济不好人更易患病，但此时医院药房里的有效药因为医保局处长们艰苦卓绝的努力大为变少了，医生的仁心也少了，你须要头脑清醒。我的建议是，所有的中药都不能用，尤其注射液。西药尽量用原研药，比如阿奇霉素选进口不选国产的，多花点钱值得。对了，只有三甲医院的部分西医是真医生，余者都是滥竽充数，根本不会治病，别找错了人。你要不信我，冥顽不灵，我和你没话说。

投资环境已经恶化，陷阱层出不穷。过去的成功经验，不适用变化了的时间和空间。以前在某些领域非常成功的大佬的话，最不能听。比如还在大肆鼓吹房地产的。什么时候都得面对现实，尊重市场。

投资只有一个秘诀或者技术，那就是什么都买一点。黄金（这是标配），股票（我是不碰A股的，但如果跌狠了，你也可以大胆投机一把，因为此时就会维稳，以保持指数在三千点左右。但不要恋战），债券等等。这是因为凡投资都存在不确定性，分散才能降低风险，提高预期收益。比特币价格已经较高，一个买不起，0.01个也可以，这对后代是一个交代。另外，在支付宝上就能买卖美股ETF，长期持有它们起码是安全的。注意，任何投资都应该在可承受范围内，以及在能力范围内，杠杆不能太高，始终保有足够的流动性。

对了，可以换些美元存着，吃点利差和升水。这是仅有的可以薅的羊毛，最简单的套利。

创业对大部分年轻人是自找苦吃，除非你有足够资源。不要好奇，不要冲动。

交社保的唯一用处是可获得医保，医保保障水平正在迅速下降，跟自费差不多。再这么下去，没必要交。

个人养老金，因为它的投资并不保本，就是鸡肋。一般说，大张旗鼓的事，不是要对你好，而是有人急需你的支援，你就是那不惜的代价。

如果大城市生存艰难，可以去二三线，切不可回县城和农村。最深的套路都在那里，它们是大酱缸，进去就废了。

看看电影刷刷剧，可以缓解压力，避免抑郁。趁着影视剧网站还能打开。

要学习新知识，最重要的是AI，先从了解它的发展史开始，技术细节可以忽略，然后是运用，比如炒股，比自己琢磨可靠多了，用过的都知道。以后我们离不开AI。

年轻人除了把英语学好（日语，法语、西班牙语，德语亦然），掌握一门生存技能比如电工修车什么的，也属必要。以后劳动力少了，人工会很贵，体力脑力报酬趋同，要有动手能力。

整个社会流氓意识正在抬头，这世道已经没理可讲，少跟人争执。要时刻眼观六路，耳听八方，必须加强锻炼，遇到突发情况要跑得快。

热爱家庭，这是唯一的温暖所在。多多陪伴家人，温柔以待，接纳他们所有的毛病。

看护好孩子，小孩子的世界已被污染，霸凌无处不在，上下欺瞒，不可能解决，邯郸初中生杀人案的判决看到了吧。家长要保持战兢兢的心态，每天叮嘱好自家宝贝。

今年高中毕业考大学的，首选好学校，能去985，211绝不能去别的学校。记住：学校最重要，专业无所谓。大学毕业的，必须考公，千方百计挤进体制内，做个精致的利己主义者，虽然被人厌恶，但好处是太大的。读研究生就争取拿到博士，硕士没什么用。留学依然是人生的高光。

恋爱是必需品，但婚姻非必选。自己能力不够，更不要急着生孩子，孩子是无限责任。这才是真正的爱心。

要从更大范围关照人生，人挪活树挪死，这是古训。你看富豪们往哪跑，你跟着跑就可以。最适合中国人生活的外国，依次是日本，新加坡，加拿大，美国，欧洲部分国家。这个事情你不做，你的后代可能没有机会做。

你也许有未来，也许没有，完全取决于你的选择。眼光和选择最重要，努力倒在其次。

最后，祝大家新的一年平安顺利。
