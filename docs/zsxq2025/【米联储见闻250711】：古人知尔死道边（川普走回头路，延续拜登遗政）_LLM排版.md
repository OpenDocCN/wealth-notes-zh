
# 米联储见闻250711：古人知尔死道边（川普走回头路，延续拜登遗政）

局势又在默默发生变化，在川普上任接近半年之后，很多事情从开始的180度大掉头又开始慢慢摆回到原来的位置，这种变化说明，有些外交政策和处境并不是人力可以决定的，存在必有其逻辑，而不会因为人为因素就出现翻天覆地的变化，因为底层逻辑的限制，天马行空的想当然必然遭遇此路不通要走回头路，结果一顿操作走下来才发现，现实情况所限，很多事的选择并不多，不在其位，觉得是在位上的人蠢笨，等自己来做，才发现也没有比前任聪明多少。所以现在的川总也很少说拜登是老糊涂了，因为轮到他来做，其实也并没有做得多好，相比之下，布林肯和沙利文组合，虽然年轻傲气，实算是美国经济政治最后的硕果仅存了。

在早期疯狂热情的拥抱俄罗斯，疏远乌克兰之后，各种改革推动不了的川普又不得不重新审视拜登的政治遗产，昨天，乌克兰总统泽连斯基在罗马举行的记者会上表示，暂时不会与俄罗斯谈判，称必须完成此前达成的战俘协议，才有可能进一步对话。当日他曾与川普进行建设性对谈，基辅已收到美国将恢复军事援助的政治讯号。此外，乌克兰已分别与德国、挪威达成协议，对方将为乌克兰采购爱国者导弹系统。乌克兰需要额外10套爱国者导弹系统，他的团队正在努力寻找融资。俄罗斯在昨日的轰炸中，向基辅和其他几个地区发射约400架无人机和18枚导弹。言下之意，战争还将继续，且必须继续。这就是人性，打到这个阶段，谁也输不起，但凡还有一点还手之力都要继续打下去。

川普也表示，美国将向乌克兰运送更多武器，以帮助该国抵御俄罗斯日益加剧的军事扩张。该批武器可能包括防御型爱国者导弹和进攻型中程火箭，但具体尚未确定，对这一说法，五角大厦和白宫尚未回应。不过，川普表示，他将于7月14日就俄罗斯问题发表重要声明，但未透露声明内容，这意味着美国在俄乌冲突的问题上将有重大转变。川普表示对俄罗斯感到失望，并补充北约将全额资助乌克兰购买美国武器，包括爱国者防空导弹系统。已批准向乌克兰运送更多防御式武器，以帮助抵御俄罗斯进攻。还称正考虑对俄罗斯实施进一步制裁，并在会议期间表达对普京的不满，指其不愿停战、杀死很多人。

对川总而言，对俄态度当然不是普京的行为所决定的，一定是美国的利益所决定的，只能说欧洲说服了川普，达成了共同利益诉求，这也是为什么最近欧洲又在作死，跟中国对着干的核心原因。对欧洲来说，中俄相比于美国，威胁远大于利益，远交近攻的道理，欧洲是刻入骨髓的，加上英国这只看不见的搅屎棍，让欧洲彻底下决定跟中国合作是不现实的，既不符合欧洲的传统价值观，也不符合欧洲政客们的现实利益。所以别看川总闹得凶，但是在真实的利益导向驱动下，美欧关系依然是牢不可破的，也不是某个政客几句话就能撕裂的，相比之下，美西方国家和亚洲、非洲的关系才是因势利导。

美国务卿鲁比奥今天抵达马来西亚吉隆坡，参加亚洲国家高级外交官的聚会，这是新一届美国政府高级官员的第一次亚洲出访，鲁比奥计划在抵达马来西亚首都吉隆坡后不久出席与东盟国家外长举行的会议。还将在吉隆坡会见俄罗斯外长拉夫罗夫。川普的对俄声明等到下周一再发，估计要等这场会面的谈话结果。鲁比奥这次访问前对媒体的说法是，亚洲之行旨在加强美国在亚洲各地的联盟和伙伴关系，并通过建立这些关系来对抗中国。丝毫不掩饰美国始终把中国当成心腹大患的执念。而且在川普政府身上越来也多的看到拜登团队的操
作，这次亚洲之行的宣传策略又是重申美国对推进一个自由、开放和安全的印太地区的承诺。这是美国官员用来强调中国在该地区咄咄逼人的海上和领土行动的常用说法。所以，熟悉的味道又回来了，台湾这次搞超长汉光军演肯定也有美方在背后操刀。

在川总发布的关税14国名单中，至少有八个国家将派外交官出席在吉隆坡举行的会议，而且东道主马来西亚就是14个国家之一。马来西亚官员对川普提议对该国出口到美国的商品征收25%的关税感到惊讶，这个数字与4月份提出的24%几乎相同，自那以后，马来西亚一直
在努力与美国谈贸易协定，两国政府官员已经举行了25轮会谈。马来西亚总理安瓦尔·易卜拉欣在东南亚外长会议的开幕致辞中表示：“这不是转瞬即逝的风暴。这是我们这个时代的新天气。”这句话精准了表达了小国在面对大国博弈搅起外交风暴的无力感。

由于，鲁比奥现在是中方制裁人员，虽然王外长也在当地参加活动，目前还没有迹象表明双方会安排会面。相比于美国的任性霸道和喜怒无常，中国这几年在外交策略上更喜欢把中国比作一个可靠的贸易伙伴，与美国形成鲜明对比。相比于中国的利诱，美国的打法是酷刑，通过关税政策向各国施压，要求它们减少与中国的贸易和投资关系，所以看似美国跟这些国家谈判，实际上都是谈这些国家该怎么对付中国，使第三国与中国的关系复杂化。这里面很多国家与中国的贸易往来超过与美国的贸易往来，所以让这些国家非常为难，更可怕的是美国自身政策缺乏稳定性，如果美国可以坚定坚持自己的承诺，这些国家也是可以咬咬牙的，但可惜美国也没把他们当个人，又怎么会在乎对它们的承诺，也就是因为这个原因，才让这些国家与中国的关系产生了气孔，能够维持下来。

在亚洲问题上，民主党和共和党一直都在努力向亚洲国家展示对该地区的战略愿景，并致力于在经济、外交和军事方面制衡中国。但做大哥是需要花钱的，相比于把钱花在欧洲更让美国人觉得值，在黄猴子们身上撒钱让白老爷们发自内心的不爽，即便日韩与美国如此亲密了，说翻脸就翻脸，这好比欧洲和中国一样，外交上的最大鸿沟有时候还是种族。鲁比奥在访问到之前访问了拉丁美洲、加勒比地区、欧洲和中东，可见对亚洲的重视程度有限，而且他本可以安排更长时间的亚洲行程，在更多国家停留，但最后只选择在吉隆坡参加会议，并未去往其他国家，相比于美国政府在中东冲突上花费的时间和资源，在亚洲投入的时间和资源远远不足。亚洲拥有世界上人口最多的国家和除美国之外最大的经济体。鲁比奥的亚洲首访如此蜻蜓点水，更是让东盟国家感受到了川普团队对亚洲战略的态度。

一套。既然皇帝把面子给的足足的，欧洲自然要拿出对低等人种该有的态度，这次中欧会谈既不顺利，跟欧洲再次表现出无礼的傲慢有很大的关系，除了对等制裁，国内几乎没有宣传王外长的欧洲之旅，原来计划领导要访欧的事情估计也不会安排了。而这当然也是美国想看到的。欧盟的领导人原定于7月底前往北京参加峰会，但有关计划也一直处于变动中。欧盟最近采取行动限制政府采购中国医疗器械，中国立刻宣布反制措施。至于稀土，继续死死卡住，一点都不能松。欧盟现在自己要搞小圈子，过去欧盟一直跟着美国大哥混，觉得可以吸尾流，省心省力，但是川普的
