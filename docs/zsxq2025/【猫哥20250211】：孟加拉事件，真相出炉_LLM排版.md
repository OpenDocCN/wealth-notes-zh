
# 孟加拉事件，真相出炉

## 嘉宾：梦回安西

2024年，孟加拉国发生的政权更迭，一直迷雾重重，虽然孟国内部矛盾不可忽视，但有没有外部势力参与，始终是个巨大的谜团。现在，真相出炉了。

近期，美国国务院前外交官迈克·本兹，在塔克·卡尔森的直播间，承认哈西娜政权的倒台，是由拜登政府策划。按照迈克·本兹的讲述，哈西娜和美国的矛盾始于奥巴马时期，那时希拉里正担任国务卿，试图让孟加拉同意允许美国在孟加拉湾驻军，理由是对抗中国。遭到哈西娜拒绝后，美国便盘算着下手。

美国在调查后发现，孟加拉年轻人喜欢听嘻哈，于是以资助嘻哈为切入点，大力培养街头意见领袖，而嘻哈歌手们不断制造各种“反哈西娜”歌曲。大批年轻人受到吸引，逐渐站到了嘻哈一边（所以说，娱乐圈的很多事情不单纯，得多一个心眼）。资助活动的直接领导机构是“国际共和学会”，这是“国加民主基金会”旗下的组织，而民主基金会的上级，就是美国国务院。到2019年，“国际共和学会”已构建好一个地下网络，包括170个活动家以及超过300名关键人员，也就是说，那时候就准备好了。

只等一声令下，时机合适，一举拿下哈西娜。中间五年都没有动手，很大程度上是顾虑印度的感受，毕竟哈西娜政权和印度关系密切。推翻哈西娜政权，势必恶化印美关系，在“联印抗中”的大环境里可能得不偿失。实际上，哈西娜拒绝美国驻军大概率也是印度的要求。

只是到了2024年，形势出现变化。俄乌战争爆发后，美国急需印度站到自己一边，毫无保留地对抗俄罗斯和中国。然而，印度明显是多头捞好处，一方面继续买俄罗斯的廉价原油，加工后出售给欧洲赚钱；另一方面，在对抗中国上，显得出工不出力。美国对印政策最大的问题就是它既希望印度配合，又拿不出足够的好处，尤其是在制造业转移上，迄今为止给出的好处寥寥。其实这也正常，毕竟美国自己都希望制造业回流，能有多少留给印度？胡萝卜不够，那就大棒来凑，在地缘政治上对印度施压。

哈西娜在位多年，孟加拉国的经济发展虽有进步，但幅度不算大，总体依然是靠纺织品出口，在脆弱的国际环境下很容易被击垮。加之家族成员的腐败问题，随着时间的推移，内部不满日渐增加，时机成熟了。

2024年7月，“公务员配额”问题爆发，哈西娜应对失当，“国际共和学会”立刻动员整个网络，在短时间内推翻了哈西娜政权。而新上台的尤努斯，和美国的互动密切了许多。特别是前段时间，尤努斯会见了索罗斯的次子兼接班人亚力山大引发很大争议。对于哈西娜的倒台，莫迪严重不满，至今还保着哈西娜。印度许多媒体都认为，这就是美国干的。只不过我们知道，推翻哈西娜这件事，做得很周密，始终没露出什么证据，证明是拜登干的，并且孟加拉内部确实矛盾不小，因而主流观点依然认为，和美国关系不大。直到这次，塔克·卡尔森的采访，美国国务院前外交官迈克·本兹的讲述，才终于真相大白。问题来了，为什么会在这个时间节点，把真相爆出来呢？

我们知道，塔克·卡尔森属于懂王的御用媒体人，深度介入政治，这不可能就是一次普通的采访。比较可能的原因是，出于“联印抗中”的大业考虑，懂王需要改善与莫迪的关系。莫迪马上要启程访美了（2月12日出发），有必要释放一些善意，懂王这个人外交作风从来是胡萝卜加大棒，对莫迪也不会例外。大棒，就是打击印度非法移民。美国近期在打击非法移民的行动中，对印度非法移民，进行了区别对待，也就是拿手铐拷上，头套套上，像囚犯一样送上飞机。其他国家的非法移民，尽管也有许多被抓的，但遭到如此待遇的却唯独印度一家，此事在印度引发轩然大波，很多印度议员抗议莫迪过于软弱。胡萝卜，则是揭露2024年孟加拉政权更迭的真相，释放一些善意。等于承认是拜登的错误，我后续不搞你了，咱们一起抗中吧。

另外，“国际开发署”被砍掉对印度也是好消息，美国对孟的各种援助，“国际开发署”是重要参与者，砍掉“国际开发署”后，尤努斯能要到的援助必然减少。

不过，塔克·卡尔森这时候把真相捅出来，不仅是因为莫迪的来访，也和美国内部形势，息息相关。第一任期时，懂王就公开说过，“伊拉克没有大规模杀伤性武器”，目的就是抨击小布什这些建制派。这个事，当时在美国政坛引发了很大争议，因为过去两党是斗而不破的，通常不会有揭老底的事情出现，就算全世界都知道伊拉克没有大规模杀伤性武器，你身为总统，也是不能亲自说的。到了第二任期，两党已经分道扬镳，懂王与深层政府的斗争进入了深水区，互相揭对方的老底就更没顾忌了，甚至是一种必然。

在对外援助这个问题上，懂王想减少对外援助重回孤立主义，就有必要证明，民主党的对外援助不是真正的援助，而是在输出动乱，制造问题。而孟加拉国的事情，就是挺好的证据。你们瞧瞧，又是民主党干的，又控制了一个地方，果然是深层政府，把美国人民的钱拿去干不正当的事情。

塔克·卡尔森在油管的评论区，有许多类似的评论：“他们（民主党）声称没有钱帮助北卡州的飓风灾民，却有钱援助孟加拉”，“我对于我们对其他国家的干涉已经厌倦至极”。

可以说，随着懂王和民主党斗争的深入，更多的黑幕将被揭开，到时候我们会发现，很多原先以为是客观原因所导致的事情，背后都有一只看不见的推手。
