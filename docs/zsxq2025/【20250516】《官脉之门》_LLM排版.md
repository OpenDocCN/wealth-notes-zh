
# 《官脉之门》

## 前言
历史要不要读，历史要不要研究，对职场人士来讲是个值得讨论的话题。历史案例可以作为职场人闲聊或研究的共同谈资，却不适合职场同志拿来就能用。本作品的目的就是为使所有职场同志能懂、能用，而且更为深刻帮助职场同志了解领悟职场。

最需要注意的是，不少讲课的所谓专家之流动辄就只讲头部大佬的话题，比方说过去的马老师，如今马斯克等，这些过去的、现在的大佬们有没有值得我们学习研究的地方呢？当然有，但这个不具有复制性、模仿性，也就是说这些大佬放在绝大多数的职场同志手里，不具备应用性，或应用性不足。

如果说这些专家在课程上玩追求节目效果，或者使这个课程更能卖，无疑通过抓取话题人物，或者抓取头部大佬的话题来讲，确实可以得到他们想要的结果。但作为职场同志，一个虔诚的学习者来讲，却是有问题的。职场同志要学习的，应该是可以复制，可以效法，可以直接实践的对象，这就要求职场同志学习科学、专注科学。

对此，鄙人要讲的是，科学也分自然科学与社会科学，重要的事要多说几遍，是科学，咱记住，而不是文学。讲课的很多专家之流，为求演讲节目效果，往往抓的是典型性，也就是文学，并没有抓取普遍性的科学形象。这里面的道道，如果叶老师不讲，可能很多职场同志便完全忽视了这个问题，被各大所谓的专家给PUA了一遍又一遍，而今天看到此文的职场同志则可能在准备骂娘了。

对此，鄙人只能讲非淡泊无以明志，非宁静无以致远，放过这些专家，就是在放过自己。本作品《官脉之门》，领悟其中的内涵，就能领悟这个社会，领悟这个社会，你就能以上帝视角看透身边的人和事，人脉、钱脉、官脉看似各异，实为一体，从社会角度，人脉才是这三者的中心，从经济利益角度讲，钱脉又成为了中心，但如果从中国特色的社会来看，官脉才是所有的人和事的最中心，也是最核心的部分。

中国的社会就是以权力为核心的社会，权力掌控一切，这与社会性质无关，为什么这么讲，从与咱的历史、生存环境、文化思维所形成的大一统格局密切相关，只要大一统格局还在，或者人人都坚持大一统格局的现状，权力就会一直是社会的最核心部分。
