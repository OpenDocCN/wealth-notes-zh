
# 谈谈向上管理

## 一、先说管理领导的预期

在工作上是优等生，就像在校的学生一样，你每次考试90分，偶尔考了60分，老师认为你偷懒了。而差等生，平时考试不及格，偶尔考了60分，等来的是老师鼓励和夸赞。

工作和学习一个样，给人太多的希望，一开始把对方的预期调得太高，一旦达不到要求，立马就会失望和不满，偷一点点懒就会被人发现。

1、比方说对于新人，我一直强调不要用力过猛在机关内，新人表现要适度，我见过单位新人刚进来用力过猛，刚开始工作时就像打了鸡血，让人眼前一亮，但很快他们又像泄了气的皮球，往往不仅没有表现好自己，还导致后续劲头跟不上，最后反倒落下一个不积极、态度变差的口碑。

体制内最显著的特征就是规矩很多，其中也包括上班时间，大多数单位是朝九晚五，这里建议如无特殊情况提前10分钟到足矣，上班第一天就提前半个多小时，那以后也只能维持这个标准。

万一哪天有事或身体不舒服怎么办，坚持不住了怎么办，不想早起了怎么办，冷不丁那天准点上班那大家必然觉得你变“懒”了，态度不认真了。

还有新人入职时，每个单位都会有这样一个现象：为了留个好印象，复印资料、打水、跑腿样样都抢着干，用力过猛的人，他们的热情持续不了多长时间，半年过去，自己也熟悉单位规则了、惰性逐渐有了，以前跑腿帮忙之类的琐碎活不想干了，大家就会觉得你是态度有问题，不如以前了。

工作千千万，永远干不完。曾经我在年轻的时候一接到工作任务，就恨不得立刻做完，哪怕通知里的时间要求是一周或者2—3天后做完，但我都会不顾一切的玩命干，顾不上喝水、有时候都顾不上吃饭，结果最后越干越多，有时候经常做无用功。

大家知道体制内的很多工作都是领导拍脑袋想出来的，有的时候你拖个一两天，可能领导就把这个事给忘了，或者说突然上面的部署和要求就变了，你之前加班加点干，可能最后发现白干了，一切又要推倒重来，做无用功，所以有的时候拖一拖，可能就不用做了。
