
# 外卖大战的热闹背后，你要看清新逻辑和新机会

前段时间，外卖大战打得如火如荼，在京东、饿了么、美团之间，一场围绕中国人餐桌的“外卖三国杀”硝烟四起。但是今天，我想带你抛开具体的商业竞争，从人口视角来看这次的外卖大战。各方之所以拼尽全力一定要打这场外卖大战，是因为各个平台需要在巨大的人口周期变动下，争夺新一轮的结构性人口红利，这背后是单身经济的崛起，正在重塑消费市场。

简单说，就是平台们发现了外卖行业和即时零售行业的新增量，关键突破口就是单身人口和独居人口。这也是我一直说的，看一件事，要努力发现本质。

## 外卖大战背后是争夺新增量

我们先从人口变化的数据说起。我在文稿中放了一张来自“中银证券”的图表，它显示了过去二十中国家庭人口结构的变化，你可以点开看看。

我先解释下图中的两组概念，一组用来描述一个家庭有多少代人，有一代户、二代户、三代户。比如三代户就是有三代人，包括爷爷奶奶、爸爸妈妈和子女，二代户、一代户也同理。另一组用来描述家庭人口的规模，有一人户、两人户，一直到六人户，就是这个家庭中实际有多少个人，不考虑代际辈分。一人户，就是整个家庭中只有一个人，比如户口迁出的独居人士、离异或者丧偶后的单身人士等，其他的以此类推。

根据图表显示，从2000年到2020年的这20年之间，人口结构有两个非常明显的变动趋势：

第一，一代户，也就是只有一代人的家庭的占比在持续暴涨，一代户占比，从2000年的21.7%，到2020年的49.5%，二十年的时间内增长了2.2倍，而且2020年已经接近50%，也就是说，到2020年，中国一半的家庭都是一代户。

第二，一人户和两人户的占比显著增加，合计比例从2000年的25%，涨到了2020年的55%。特别是一人户，增长速度远超其他规模的家庭。

也就是说，中国的家庭结构有了明显的“小型化”“微型化”趋势。

那么，这些快速增长的一人户家庭，年龄结构具体是什么样的呢？我在文稿中还放了另一张“中银证券”的图。

图片显示，相比前十年，2020年各个年龄段的一人户比例，都是在上涨的，尤其是25-34岁以及45-59岁年龄段的群体，增长幅度明显比较快，他们会是外卖消费的新增量人群。至于65岁及以上人口一人户的大幅增长，大概率是老龄化和寿命延长加剧下，丧偶增多的原因，但他们不会是外卖的重点消费人群。

好的，关键点来了，就是不论是翻倍的一代户，还是快速增长的一人户、两人户，这些微型家庭对应的相对年轻的单身人口、无孩夫妻等，恰恰是对外卖需求和即时零售需求最大的群体。

那么，是什么原因导致了这种变化呢？搞懂人口结构变化的成因，能帮你加深对当前现象的理解。

一方面，城市化进程加速了人口流动，年轻人离开家乡前往大城市工作，独自租房生活但迟迟不结婚，成为常态。另一方面，现代社会的价值观更加多元化，婚姻和生育不再是人生的必选项，许多年轻人更倾向于追求个人自由和事业发展。此外，高昂的房价、育儿成本以及职场压力，也让不少年轻人推迟甚至放弃组建家庭，干脆选择一个人生活。

这些都是一代户、一人户攀升的原因。而且从目前的数据来看，未来“家庭微型化”“人口独居化”以及“低生育化”的趋势，还将加大，独居人口、单身人口、无孩夫妻的规模和基数，还将不断增长。

而对于我们开头提到的外卖市场来说，这些人口的持续增加，意味着外卖、即时零售的渗透率还有上升空间，尤其是在大城市和夜间经济场景丰富的三四线城市。

所以结论就是，外卖市场各大平台的激烈竞争，是基于未来增量市场的争夺，他们争夺的是未来单身人口和微型家庭的消费能力，尤其是年轻的单身人群。因此，外卖平台必须抢占这些新的年轻用户，因为他们的消费习惯一旦形成，将具有极高的粘性。

## 新增量背后是新消费模式

刚刚我们讲了当前家庭结构变化的成因，以及对外卖市场的影响，其实外卖市场只是一个缩影，这种变化还会影响更大的消费模式，这也是今天我想带你看到的深层逻辑。

中国之前的传统消费模式，是以大家庭为单位的，所以一方面会注重刚性需求，比如房子、车子等结婚大件；另一方面也注重大家庭多人口下的性价比和实用性，比如自己做饭，选择大型超市集中采购等。

而随着家庭微型化和独居现象更加普遍，这部分群体由于独自生活，所以他们的消费逻辑完全不同——他们不会基于大家庭多人口的角度考虑消费模式，而是更注重便捷性、个性化和弱社交属性。我展开讲讲这几点。

首先是便捷性优先。独居者，尤其是年轻的独居者，对综合成本更在意。一个人采买性价比不高，所以不愿在做饭、家务上耗费精力，因此对他们来说，外卖、即时零售、便利店、社区团购、家务机器人、多功能家电、家政服务等，是比较大的刚需。

其次是个性化消费。年轻的单身人群，更愿意为兴趣买单，比如潮玩、游戏、电竞、短视频等；无孩夫妻也更愿意为旅游或者电影付费，特别是在经济下行期，这种“心灵慰藉型”消费，最契合这部分群体的需求。

最后是社交属性弱化。传统家庭消费往往涉及聚餐等社交场景，而独居者更倾向于“一人食”“一人游”，甚至“云社交”。而且相比于人际交往的复杂和繁琐，他们更愿意养宠物，所以宠物经济的爆火也是必然。

而这些特征反映在外卖市场上，就是未来竞争肯定将持续，但最终可能演变成为“即时服务生态”的竞争。未来的外卖平台，不仅送餐、送药、送日用品，也不仅会提供上门服务，比如家政、保洁、维修等等，甚至会把你绝大部分跟生活、消费相关的事项，比如各种大件购买，都涵盖了，变成你的生活入口，甚至有一天你可能在上面找工作。

相比之下，传统依赖家庭消费的行业则面临挑战。比如大型商超的家庭采购减少，甚至周边的便利店，都可能被配送中心取代。再比如电影院，因为缺少社交需求的独居者更倾向于线上观影，线下影院将面临寒冬。

## 整个社会消费模式的更新

好，刚刚我们讲了消费模式正在从以家庭为单位，逐渐转向以个人为单位。最后我们再讲讲这背后所代表的整体社会消费模式的更新。

第一，是消费决策权的转移，从“家庭中心”向“个体分散”转移。传统消费模式下，家庭的决策者，通常是父母或长辈掌握主要消费权，购买行为往往兼顾全家需求，比如买房考虑学区、买车考虑空间、家电追求耐用性等等。

而独居者和单身群体的消费决策，完全由个人主导，更注重“我想要”，而不仅仅是“我需要”。这种分散化的决策权，让市场从“大单品时代”，进入“碎片化需求时代”——小众品牌、个性化服务、垂直赛道的机遇将大幅增加。比如精准面向特定人群的定制产品，就是细分的商机。

第二，消费价值从“长期耐用”，向“瞬时体验”迁移。家庭消费注重“长期持有”，比如房产、大家电；而独居者、单身者、无孩夫妻等群体，更看重“瞬时体验”——外卖吃的是便捷，短视频刷的是即时快乐，盲盒开的是瞬间惊喜。这种变化让“耐用性”不再是消费的核心指标，反而“新鲜感”“便捷度”“情绪价值”成为关键。商业逻辑也随之调整，产品的生命周期缩短，服务响应速度加快，包括出现了“订阅制消费”，比如会员制、租赁经济，就是让用户不断获得新体验，而不是长期占有。

第三，消费场景从“固定物理空间”，向“流动数字生态”扩展。传统消费依赖固定场景，比如超市购物、餐厅聚餐、商场逛街。而独居者的消费场景则高度流动化——外卖送到家，购物靠直播，社交在线上，娱乐在云端。这种趋势让空间的价值被重构，线下空间从“交易场所”变成“体验场所”，而线上空间从“信息渠道”升级成“消费生态”，所以未来的消费潜力一定是在线上。这样你也就明白，为什么各个平台要这么激烈竞争了。

第四，消费伦理从“实用理性”，向“自我定义”演变。家庭消费往往受社会规范约束，要做“正确的选择”；而独居者的消费更倾向于“自我定义”——可以花5000元买潮玩，却不愿花500元吃一顿“社交饭”；可以养猫每月花费上千，但自己点外卖只选最便宜的套餐。

这种“自我定义”让消费行为变得难以预测，但也催生了新的市场需求：一是反套路消费，拒绝被传统价值观绑架，比如“不买房主义”“零彩礼婚姻”；二是悦己经济，愿意为取悦自我付费，比如小众爱好、虚拟偶像；三是懒人正义，理直气壮地为“省时间”买单，比如代跑腿、智能家居。这些新需求的背后，对应的都是有待挖掘的新机会。

而这些新趋势，用一个词来形容，就是方兴未艾，未来的增量，还很大得很。总之，希望今天的内容给你打开了一个新的视角，能让你看到新的机会。
