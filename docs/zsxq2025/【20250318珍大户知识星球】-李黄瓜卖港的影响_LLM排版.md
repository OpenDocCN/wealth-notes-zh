
# 李黄瓜卖港的影响

今天3月14号，港澳办用及其严厉的用词批评了李黄瓜卖港口这件事，已经直接开口说，请想清楚你是要站在哪一边。

## 原文链接

https://www.hmo.gov.cn/sgazx/gayw/202503/t20250313_40505.html

之所以这么严厉，是因为里面已经开始预判有卡脖子的风险，认为后面老美很可能会把港口政治化，作为排挤中国国际贸易的手段。

港口是一种有限的稀缺资源，不卡脖子时候不稀罕，拿来卡脖子的时候才知道紧张，同理与很多的“资源”。

## 影响分析

影响无非两个方面：1是替代，2是反制。

### 替代

替代是指对方卡脖子的手段使出后，我们需要转移路径去替代对方卡脖子的路线，所以需要查找目前有哪些公司拥有海外的港口资源。这一条的影响出现的不会太快，可能要等政治新闻出现后才会开始反应，但反应的幅度很可能比较剧烈。所以需要提前做功课挖坑等待。

### 反制

反制是指使用我国拥有的稀有资源去卡对方。因为涉及到政策的缓慢草蛇灰线的布局，所以影响应该是偷偷淡淡的，我能想到的包括我们前面提到的矿业托拉斯，不要太快跑。

### 替代案例分析

李嘉诚卖掉的港口，我搜了一下新闻，说是包括23个国家的43个港口。卖掉的港口，你会发现基本处在欧美控制区，如巴拿马运河两侧的港口，还有墨西哥和哥伦比亚的港口，这都在老美眼皮底下。

欧洲地区的港口打包出售，英国菲利克斯托港、荷兰鹿特丹港、德国汉堡港、比利时西班牙波兰这可能是怕欧洲集体出点啥政策。

### 可能性分析

我感觉有两种可能性：

1. 李黄瓜其实就是欧美资本的亚裔白手套，现在摘手套了，还给人家。
2. 李黄瓜聪明的看到后续的政治跌宕风

险，如果在政治强压下，即便他有股权，也保不齐后续有可能失去控制权，所以就选择先卖了。

但即使第2种可能性下，他依然有不同的选择。只要把这些港口卖给国企，一样风险甩走了，还控制权不外流，而且还能在国内捞到政治资本。但为什么他不愿意这么做呢？很可能他内心是认为欧美最终会赢得竞争，所以要国内的政治资本没用，还是铁心保留自己在英系下的爵士政治资本更有前途。

所以才有开头哪一篇，声色严厉的问了一句，想清楚你到底是站哪边。

国内经营海外港口的主要就是两个系，一个是中远系，涉及中远海控和中远海运港口（母子公司分开上市），一个是招商系，涉及招商港口/招商局港口（同一公司分别上市）。

### 直接受益方

上面两个港口，是属于直接受益的。而中远海控受的影响比较两面，因为它主要是经营集装箱业务的，如果出现卡脖子之后，其实是要受影响的。主要是子公司的港口受益，集装箱业务受损。但要考虑目前它只有4倍市盈率，估值实在是低的离谱，虽然估值有24年三季度红海危机导致运费暴涨利润大增的不可持续原因存在，但红海危机这件事至少证明，如有危机，运费暴涨的概率很高，在国际贸易中，航运公司更可能是强势方，所以我会觉得有利因素的力量大于不利因素。

而那两家港口，则铁定是卡脖子的受益方，或者说，是长远布局未来不被卡脖子的发展受益方。但考虑到，政治风险可能不会立马出现，线的走势上也不像是近期要涨的样子，所以我会觉得，这种目前低估值的不被看好的公司，但又代表坐地起价的有可能受益公司，其实是比较好的定投选项，可以持续缓慢定投进去等涨，靠时间来把概率推高。
