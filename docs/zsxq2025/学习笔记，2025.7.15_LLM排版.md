
# 学习笔记，2025.7.15

1，世界都属于精英群体的，新人一定在不断的取代老人，各个领域都是如此，优秀的房子，优秀的岗位，都属于新人。不管什么年代，财富都属于精英群体的，而且是年轻的精英，你想想，是不是这么回事？所以，优秀的房子也一定属于精英，就是永远不要担心我们的子孙后代买不上房子，若是他们足够精英都买不上房子，你放心，这片土地不值得他们留恋，要么他们重新洗牌，要么就换块土地。我小的时候，村里再穷的人家，也能给儿子盖上婚房，我们总以为土地紧张，实际上，把全上海开发了，把全中国人搬进去都住不满，地有的是。

2，我们做投资的三大原则是什么？第一，本金够多，其实决定成本的核心是本金。第二，持仓足够长，以10年或20年为单位。第三，时刻在线，确保闪电来临时能劈中咱。

3，股市、房地产，发展不起来一个重要的原因，外贸抽血严重，外贸的GDP占比越来越高，但是，大家不结汇，不结汇的原因是美元理财或买股票，最差收益也有年化5%，而在国内呢？3%都很难，他们没有结汇的欲望，从而等于钱一直在往外走。

4，人老了以后，就不值得学习了，身上一股老人味，除非这个人的状态不是变老了，而是长大了，一直在不停的蜕变。王石就是，杨振宁也是。他们能吸引年轻女孩的根源是有着男孩一般的生机勃勃。我现在都特别佩服冯唐，怎么发明的油腻这俩字，真的太准确了。李亚鹏被王菲淘汰，一个很重要的原因就是他一直在变老，而不是长大，李亚鹏年轻时是一个非常有生命力的男生。李亚鹏的衰老就是从研究传统文化开始的，各类收藏，下一个李亚鹏就是杨子。

5，我们的阶层很难固定，因为优质阶层在不断的寻找确定性，哪里有确定性他们就去哪里，而普通人实现越级难度越来越大，最简单的，攒1000万越来越难。其实，不是越来越难，是一直很难，之前我算过，年薪80万的家庭，在省会城市属于很体面的了，无论是郑州还是济南，就这样的家庭，30年攒不到1000万，一是税，二是消费，这个收入级别的人，可能玩玩滑雪、潜水，但是玩不了更高级的，更高级的游戏主要是来自企业主，凡是拿工资的，不管年薪50万还是100万，其消费的天花板其实并不高。

6，跟企业家闲聊，他们普遍反映，现在的年轻领导是有理想有抱负的，济南做药店的小姐姐，她跟我讲，过去医保结算周期很漫长，新领导来了以后，主动找他们谈话，总是问他们，有没有需要改进的地方？大家反馈这个问题以后，他接着就给解决了，缩短周期、不拖欠。而且，现在干部高学历化，我看了一下浙江那边，从省里到地方，基本都是985了。985是一代人里的精英，思想、正义、抱负，全面的精英。

7，有时我觉察我自己，我突然觉得，原来从小到大，我们从来没接受一个求真教育，反而认为造假是一种潜在的规则，就该如此，大家对此都心照不宣，我们知道是假的，他们也知道是假的，但是谁都不会去求证。疫情时，很多时候0确诊是为了保证当地正常运转。不过，我是满怀期待的，感觉一代更比一代强，叛逆其实就是改革力量，现在总有人说，感觉00后是颓废的一代，例如考公热，躺平热，其实这都是表象，社会不是由大众推进的，是由少数精英推进的，我看好我们的下一代。

8，酗酒是全世界的课题，在农村，每个村都有酒鬼。其实健身房、球馆也是酒鬼聚集地，很多人都有酒精依赖，戒酒比戒毒难，明星能戒毒戒不了酒，酗酒最厉害的是白人或蒙古人，从这一点来讲，特朗普能做到滴酒不沾，真的是超级牛逼。国内的话，基层公务员也是酗酒高发区，前段时间，乡镇工作人员一车四人被大车撞死了，其实隐掉了两个关键细节，一是司机酒驾，二是逆行。

9，中日情绪未来会不会缓解？取决于舆论风向，说变，一两年就变过来了。前几天我看儿子的历史试卷，里面提到甲午战争后带来了什么改变，我看了一下答案，其实这道题都是错的，当年甲午战争后，国人对日本并没过多的仇恨，相反是仰慕钦佩的理性，看一个细节就行了，大量留学生赴日，还有一个大背景，师夷长技以制夷。现在，其实大部分人都是理性的，看日本密密麻麻的中国人就行了，只是大家只干不说，至于那些恨的咬牙切齿的呢？他们一辈子也不会拿到日本签证的。现在去日本办理签证，需要30万的保证金，因为国内这边跑的太多了，从而非常严格。我认为，仇恨教育马上就成为过去式了，这不是由某个人决定的，而是时代进步决定的，现在虽然喊的山响，但是已经比过去强多了，我们上学时，那是动不动上街，甚至打砸，现在哪有了？！声音越来越弱了。

10，你觉得自己变好的根源是什么？金钱打磨，想驾驭金钱又被金钱反驾驭，来回纠缠，慢慢占了上风，我一直都信仰那句话，赚钱是最好的修行。其实不管欧洲还是日本还是美国，经济基础决定上层建筑，这就是一句真理，这些国家的工薪阶层的确生活的很恬静，很有爱，很有幸福感。为什么经济基础很重要？因为做事业的过程是一个不断进化的过程，短板在慢慢的补掉同时长板越来越长，这就是为什么我写了那句，全世界的有钱人过着相似的生活，没钱人也过着相似的生活，若是深入一点写，那就是全世界有钱人的思想也差不多，对人性的理解，对世界的理解，对运行规则的理解，都是接近的，用我们词汇来讲，就是通透系列。
