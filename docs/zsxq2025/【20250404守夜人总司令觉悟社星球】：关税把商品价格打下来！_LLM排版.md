
# 关税把商品价格打下来！

用关税的恐慌情绪把价格打下来是意料之中的事，出乎意料的是居然能够打这么多，这个是好事，能够让成本更低！关税是由进口商支付，不是由出口商支付。任何一个中国企业都不能在美国直接售卖中国的商品，必须是由某个美国本土的进口商来做这件事情。我们到4店买国外产的奔驰也是如此，大众的途锐，在德国的出口价是人民币44万，而在中国的零售价超过100万，进口这些商品的经销商自然是要赚钱，但加价的绝大部分是关税！这100万至少有40万是关税，这个关税是进口商交给本国政府的，最终由本国的消费者承担，同样的道理也适应于其他地方。

美国人如果进口的是必需品，那么，本土经销商只能加价，如果有替代品，那么经销商的进口渠道就会转移。当然也存在另外一种可能，用本土生产来替代进口，比如说俄罗斯被整个西方制裁之后，也强调要用本土生产替代进口。然而，事实是怎么样呢？俄罗斯可以把麦当劳肯德基改个名字继续经营，但是，也只能在这个层面完成本土生产替代进口。汽车和其它轻工业品，不过是用中国的产品替代了欧洲的产品而已，中国企业

在俄罗斯那里生产和组装，打上俄罗斯的牌子！比如说中国的长城奇瑞就在俄罗斯那里建厂，比如说莫斯科人和拉达品牌的汽车，除了那个外壳是俄罗斯的，其他都是中国直接拉过去的零件，甚至那个外壳都是中国在当地的工厂生产的！

这是什么意思呢？工业是个体系，最终的产品质量和产品成本，是整个体系运行效率的反应，即便在某几个零件和环节上可以完成替代，整个体系的运行，依然延续过去的惯性，内在的相互制衡和消耗，会把局部的优势摊平！

现任美国总统特朗普此举与其说是对外，不如说是对内，特朗普目的在于迫使内部的各个掣肘的势力向特朗普自己低头，因为马斯克的美国政府效率部门那一系列操作得罪了好多人，结果造成了群起而攻之的局面，特朗普需要化解这个局面，最好的化解方式就是创造筹码！关税受影响最大的可能是跨国集团，而反对特朗普最严重的也恰恰是这个势力…只要跨国集团势力愿意低头，作出形式上的妥协，特朗普就会区别对待，就像拿捏脸书的老大马斯克一样。

关税对我们中国的影响集中在非必需品层面，就是那种没有太多门槛的东西的出口会受影响，而这种东西又是底层劳动力生计的依赖，这就需要国家去与美国做交易，反而对于中高端的产品出口影响不大。

总的来说，特朗普无法改变太多，动手展示强硬也只能找胡塞武装这种做给美国自己内部人看，甚至连伊朗都不敢动，甚至有可能私下里与对方达成一个核扩散方面的协议来自吹自擂。你会发现特朗普所做的所有事情，首先要满足内部那群MAGA群体抽象的人赢的感觉，特朗普要迫使俄罗斯在形式上配合自己“赢”，但是俄罗斯也会要让美国特朗普做出实际上的利益让步，在这一点上要达成一致，需要找到一个冤大头。

总的来说，关税对市场情绪的影响远大于事实上的影响，尤其是对于那些具有核心地位和规模优势的市场主体，至于情绪的释放，必然是越充分越好，正因为如此，这是机会。另外，对于2024的最后一个标的（指的是002858力盛体育)，之前说过，30天之内即便出现情绪波动，也不要被吓尿了，等待着！
