
# 学习笔记，2025.2.27

1. 信息太多，对于筛选能力差的人来说，不是好事。对信息没有判断能力，今天觉得张三说得对，明天觉得李四说得对，但张三和李四的观点完全矛盾，自己就会陷入痛苦，而张三和李四都在自己的观点上赚到了流量和钱。人得先让自己有一个稳定的内核，才能更好接受外部信息，而不是反过来，总想着先用什么外部信息来塑造自己，那样容易捏成四不像。

2. 对大多数普通人来说，时间花在哪，自己的成就就在哪。财富自由，不是为了获得金钱上的自由，金钱是工具不是目的，财富自由目的是为了获得时间上的自由。你可以不用再拿时间去换钱，想把时间用在哪就用在哪，这才是真正的自由。

3. 终生学习不是一个口号，而是生活在不断变化的世界中的一种很实际的策略，停止学习，就是停止了探索未知，停止了自我成长，此后只能算是过往的重复，像个行尸走肉，没有了生也没有了活。

4. 大多数自媒体的收入依靠的就是流量，为了获取流量，他们会用尽各种吸引人眼球的办法，吸走你的注意力，但实际上并不能给你提供太多价值。想要吸人眼球，意味着内容就要情绪化，让你放弃思考，跟着他的逻辑走。你不能指望靠情绪化的内容来培养自己的逻辑，情绪化的内容只会让你陷入注意力的黑洞，不停的刷刷刷，让你情绪上有一种获得感，但是刷完就忘了自己刷过什么，隔天再刷一遍。

5. 肯德基无论是从食品安全还是卫生标准来讲，都是县城餐饮的天花板……县城里很多年龄大的60后，70后，大部分人没吃过肯德基，他们不仅仅不吃，还自我催眠，说垃圾食品，我以前天天在朋友圈发肯德基，他们要么提醒我是垃圾食品，要么背后说我炫富。实际上，你怎么才能正确的了解肯德基？早上六七点，可以进去看看。很多家长，带着学生在那吃早餐，你觉得是奢侈品，对于很多人而言，只是家常便饭。

6. 县城四大现象：
一是核心商业区被金店占领，例如中心街，过去是服装品牌的天下，现在是手机品牌+金店品牌的天下，金店装修的一家比一家奢华，越穷的地方越如此，例如菏泽的单县、曹县，那才是真正的金碧辉煌，金店对应的是婚嫁市场。
二是蜜雪冰城霸占了每个县城的王牌拐角处。
三是八马茶业开在了每个县城的C位，例如滨州的阳信县，公园里有个电视塔，电视塔下面就是八马茶业，C位中的C位。
四是牙科诊所逐渐医院化，这一点德州各地最明显，仿佛一家家三甲医院，广告牌也都在C位。

7. 一个东西，没有成为品牌，只有一个原因。产品不行。我们总是迷信天然的、古法的、零添加的。其实呢，添加未必有害。真正有害的是天然存在的，杂质、毒素、细菌。例如古法榨油。零添加吧？但是，杂质在。黄曲霉素在。为什么自己榨的油一热锅就冒烟？杂质。为什么要吃品牌油？剔除了杂质与黄曲霉素。酒，也是如此。自酿的葡萄酒、白酒，都不能喝……云南的蘑菇，厉害不？北方有一样东西，也很厉害，那就是玉米糊糊发酵，叫酸汤子，东北吃死人的案例多如牛毛，一句话总结，任何与发酵工艺有关的传统食品，都要谨慎再谨慎。包括，腌制。想吃。吃品牌的。别自己捣鼓。

8. 什么找对象尽量一定要找城里孩子？因为农村孩子与城里孩子，在卫生以及公共秩序上，差了一代人。当时，选业主委员会时，我跟着走访了那么多邻居。一开门，我就知道他是城一代还是城二代。我们与邻居家，其实也差了一代人。我说一个点，大家未必信，过去农村是没有垃圾桶的，东西都是随手扔在地上的，每天早上要扫地，一扫都是一大簸箕。吃饭，骨头、餐巾纸，都是直接扔在地上。现在，县城里的小饭店，依然是如此。

9. 说着言不由衷的话，在我们这里吃饭，哪怕是再小的饭局，骑友聚在一起人均10块钱的饭局，也要说祝酒辞，大家非常熟悉，但是这个套路依然要进行，全在说着虚假的客套话，谁都没觉得有啥不妥。都觉得累，但是谁也逃离不了这个游戏模式。90后，00后，可以。所以，他们是我们的希望。

10. 我们怎么窥探到更高级别的游戏规则？若是懂英语，看世界。若是懂汉语，看上海。我们有机会窥探到、学习到很多我们永远无法接触到的东西，从而我们可以站在更高的维度去思考我们自己的人生。
