
# 特朗普战略调整，贸易战或迎转机，中美势力重新划分？

## 特朗普战略调整

特朗普上任后，美国整体战略目标发生转变，开始集中于内部变革。与第一任期相比，特朗普发起的贸易战、科技战等更多指向外部，希望通过解决外部对手来转移内部矛盾。然而，这种策略最终以失败告终，促使美国转向内部变革。

特朗普的第二任期需要重新调整美国的内外部战略。保持高度警惕的我们，似乎能从中看到一丝转机，这可能是一个宝贵的窗口期。

### 美国的战略调整

特朗普的第二任期将战略重心转为内部调整，尤其是在初期阶段，可能是必要的。用我们的话来理解，就是攘外必先安内。特朗普的战略调整，如果不先进行内部的清洗，必将受到多方的掣肘，是没法转型成功的。

特朗普与拜登政府最大的区别在于他们代表的资本利益不同。特朗普背后是本土产业和军工资本，拜登政府背后主要是金融与跨国集团资本利益。他们最主要的矛盾在于对全球化的态度和取舍上。

特朗普的第二任期还未开始时，就已经对未来的美国战略有一个明确的定位，那就是拿下加拿大与墨西哥，以及巴拿马。用北美的区域扩张，来对冲全球的战略收缩。从本质上讲，这是将美国的全球化战略导向区域化。

然而，美国国家战略的区域化与美元的全球化在战略上是必然冲突的。如果美国走向区域化，那么美元就不可能继续维持全球货币的主导地位。这对国际金融资本而言，利益将受到严重的损害。

特朗普现在正借马斯克之手，对跨国金融资本集团控制的情报机构，和对外进行宣传舆论战的组织与新闻机构进行大清洗。这还只是对外围的一个清除动作。在外围清理完毕之后，接下来可能就是针对华尔街的各个金融集团进行削藩。在这之后，才有可能真正触及到关于美元的重新定位与战略调整问题。

特朗普的孤立主义，重回区域化战略，应该是一个长期的战略目标。而围绕着吞并加拿大和墨西哥，格陵兰岛，巴拿马，建立一个纵跨整个北美大陆的新美国，应该是特朗普孤立主义的一个远景目标，或者说是一个托底的蓝图。

## 美国的刀口转向

特朗普的关税政策调整，跟他的第一任期相比，出现了比较大的变化。在特朗普的第一任期，可以简单的理解为，特朗普的主要战略目标就是针对我们，所以其关税政策，包括后面的科技战，都是以我们为主要的目标。

但是到了第二任期，特朗普的关税政策调整，正在发生一些新的变化。最主要的变化就是其针对的目标，开始变得更加广泛，甚至从国家开始转为不同的产品。特朗普的关税政策调整，在背后可能体现了内外两个不同的调整目标。

就美国内部而言，美国在对外增加关税的同时，在内部推行减税。在这一增一减之间，特朗普似乎有意在将美国的主要税种，统一到联邦政府层面。即以关税取代其他税种，作为主要的联邦税收来源。

这样做的好处是将联邦的财政收入进一步集中，同时这个操作也是对于美国作为金融帝国的一个确认。金融帝国的特征，就是对外输出货币，然后向全世界购买商品。那么在这个过程中，通过统一的关税来对整个美国市场的消费进行统一征税，其实是对输出美元的一个认可和加强。

就美国对外政策而言，推行统一的关税有利于美国进行对外政策的调整。通过针对不同国家，不同产品的关税，来调整美国与其他国家的贸易关系，甚至是协助美国国内的产业政策调整，以及美国与盟友国家之间的产业链政策协调。

综合内外部的情况来看，这种用关税来进行统一征税的操作，其实在某种意义上已经变相的承认了此前美国主导的制造业回流和供应链重组战略的失败。也就是默认了这种调整不可行，至少是短期内不太现实。

所以后面我们还会看到美国对欧盟征税，对日韩征税，对所有美国现在国内有部分工业基础的产品都会征税。这应该是美国从全球化转向区域化的一个财税政策上的主要转变。但是这里面还有一个核心的矛盾没有解决，就是美国的经济与社会两极分化的结构性矛盾并没有化解。

美国在拜登政府任内，通过激烈的加息，在全范围内引发了一轮疯狂的美元回流潮。但比较尴尬的是，美元的回流，并没有在其他的主要经济体引发资产抛售和资产泡沫崩盘的情况出现。同时，美元的集中回流，反而还推高了美国的通胀水平。
