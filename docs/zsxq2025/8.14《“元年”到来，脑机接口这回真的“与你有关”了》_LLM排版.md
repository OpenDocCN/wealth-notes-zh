
# “元年”到来，脑机接口这回真的“与你有关”了

2025年，被称为脑机接口元年。这一年被赋予此称号，是因为在研发、应用、政策三个维度上均实现了重要突破。

### 研发层面
根据工业和信息化部公布的数据，全球脑机接口专利数量持续增加。截至2025年初，全球专利数量已达4.7万个，其中中国专利数量超过1.54万件，位列全球第一。

### 应用落地
过去半年内，国内脑机接口应用取得多项关键进展。例如，今年6月，中国科学院和复旦大学附属华山医院的团队实施了国内首例侵入式脑机接口临床试验，为一名四肢截肢患者植入了脑机接口设备，使其能够通过设备下象棋、玩赛车游戏，并控制电脑触摸板。

北京脑科学与类脑研究所的“北脑一号”系统已应用于渐冻症患者的治疗，而“北脑二号”系统也在研发中，两者各有侧重。

### 政策支持
今年3月，湖北省出台了脑机接口医疗服务价格标准，这是国内首个相关价格标准，为医保报销额度提供了参照，标志着脑机接口进医保的第一步。

8月7日，工业和信息化部与科技部、财政部等七部门联合印发《推动脑机接口产业创新发展的实施意见》，明确了2025年到2030年的发展目标：实现关键技术突破，培育2到3家全球影响力龙头企业，构建完善的生态体系。

### 脑机接口的核心原理
人脑的本质就像一台“从输入到输出”的机器，通过感觉器官接收外界信号，经大脑加工后输出指令。脑机接口的核心功能是替代大脑完成信号接收和指令输出。

在接收信号方面，脑机接口将外界的声音、画面等信息转化成大脑能直接理解的电信号，如人工耳蜗的应用。在输出方面，脑机接口将大脑的电信号翻译成机器或肢体能理解的信号，如通过脑机接口指挥机械臂，或帮助瘫痪患者用意念打字。

### 脑机接口的未来想象空间
随着人类寿命的延长，脑机接口技术可能成为增强人脑功能的重要手段。例如，强化思维能力、增强情绪敏感度等。目前已有研究显示，脑机接口可以改善情绪，缓解痛苦。

### 脑机接口与其他前沿技术的交叉
脑机接口的发展得益于多种前沿技术的支持，包括AI、材料技术、摩尔定律、6G和神经科学等。这些技术的进步共同推动了脑机接口技术的演进。

### 新技术的演化逻辑
未来学家雷·库兹韦尔提出的“加速回报定律”指出，技术之间相互关联，一个技术的爆发会带动周边技术的发展。脑机接口正是这一逻辑的体现，它不仅服务于失能人士，未来可能成为许多人不可或缺的助手。

关注新技术，不仅是为凑热闹或开脑洞，更是为它在某一天的突然靠近提前做好准备。
