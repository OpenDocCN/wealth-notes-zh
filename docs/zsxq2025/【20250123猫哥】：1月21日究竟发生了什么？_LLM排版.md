
# 1月21日究竟发生了什么？

中美通话之后总体还是比较顺利的，特朗普自己也发推表示满意。在通话之后，特朗普还对媒体表示，已经收到中方邀请，将在未来访华。特朗普都要来中国了，是不是中美关系开始向好的方向发展的？最差也是一定程度缓和吧。所以，在1月20日特朗普就职仪式上，特朗普的施政宣言只有一次提到中国——中国在运营巴拿马运河。所以有理由相信，中美关系应该暂时缓和下来。

但是仅仅一天之后，风云突变！1月21日下午，特朗普突然对外宣布，将对华产品加征10%关税，这个说法很快再次加码，在几个小时之后，特朗普再次宣布，最快将在2月1日对华产品加征25%关税！

那么，1月21日究竟发生了什么，让特朗普对华态度出现如此巨大的变化？

首先，要讲一讲基本的背景。之前特朗普在竞选时承诺他将上任后24小时结束俄乌战争，所以1月20日一过去，舆论就开始要特朗普兑现承诺，这个给特朗普施加了巨大的压力。导致特朗普对外界解释是正在与泽连斯基联系，并且将很快与普京对话，而卢比奥也帮忙解释，总统会立即着手推动这个事。

但是如何结束俄乌战争？之前特朗普承诺的是24小时结束俄乌战争，手段无非是单方面向乌克兰施压，强迫乌克兰接受既成事实。但是如果这样干政治代价极大，国内阻力也极大。政治上的代价是会全面动摇与欧洲的盟友关系，后果简直难以预料，所以连冯德莱恩这个著名的“美奸”最近都在公开表示，欧洲要加强与中国的经贸关系——这世纪就是给美国传话，特朗普不要在俄乌战争上肆无忌惮，否则欧洲就有可能倒向中国。

其次，美国国内的精英阶层包括军火资本，能源资本也反对特朗普单方面抛弃乌克兰方式寻求和平。所以，特朗普要排除上述压力与阻力解决俄乌问题，还必须想办法对俄罗斯施压，要俄罗斯做出一定的妥协与让步，这样才能给方方面面做出交代。

现在俄罗斯在战场上占据优势，特朗普要怎样施压才能让俄罗斯做出妥协呢？在特朗普正在为这个难题苦恼的时候，1月21日发生了一件事情让特朗普破防。这件事就是中俄通话。

这次通话表面上的理由是中俄传统的新春祝福，但是时间节点却不大对。2024年中俄的新春祝福是2月8日，也就是在中国传统春节除夕的前一天，而今年中国除夕是1月28日，正常情况下这个新春祝福通话应该是1月27日，但是偏偏就在1月21日下午提前搞了。

很明显，这个通话时间节点是精心选择，有意为之，特意选择在特朗普就职上任的第二天，其针对性不言而喻。

那么中俄通话谈了什么？除了惯例的新春祝福之外，中俄通话还谈了几点内容。

共同维护国际秩序。两国强调要共同捍卫以联合国为核心的国际体系，推动各国捍卫联合国宪章宗旨与原则，践行多边主义。这些话平时说说没有问题，但是近日特朗普宣称要吞并格陵兰岛，加拿大，要摄取巴拿马运河。对比一下，这就是中俄联合宣布，反对特朗普上述主张，公开打脸特朗普。

加强多边合作。普京总统强调，中俄关系的独立性与稳定性，不受国际局势的影响等等。这些话平时说说也没有问题，现在特朗普为如何结束俄乌战争争焦头烂额，为了应付各方压力，特朗普不得不思考怎样给俄罗斯方面施压，让俄罗斯做出一定的妥协与退让——现在中俄通话的时候，普京特意强调中俄关系的独立性与稳定性，不受国际形势的影响——潜台词就是，不管美国怎么施压，中俄关系都会保持“稳定”——中方不可能向俄罗斯施压，同时还会一如既往在经贸关系上支持俄罗斯。

这种挟中自重，拉着老中背书暗示俄罗斯不可能在俄乌战争中做出任何妥协，这就让特朗普彻底破防了！所以，中俄通话之后，恼羞成怒的特朗普彻底翻脸，立刻宣布最快将在2月1日对华产品加征25%关税，摆出一副要掀桌子的姿态。

特朗普的宣布让资本市场立刻跳水，我们不得不立刻让六部门印发《关于推动长期资金入市工作的实践方案》来对冲，本来春节前只有几个交易日，这个利好最好的发布时间是春节后开盘前，但是因为这个意外的变数不得不提前发布。

这就是1月21日意外事件的来龙去脉。今天评论区暂时关闭。
