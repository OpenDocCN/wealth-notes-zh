
# 易经16中“优势在我”

初六。鸣豫，凶。
象曰：初六鸣豫，志穷凶也。
刚做出一点成绩就自鸣得意，必然招致凶险。
雷声催发的周期是星辰大海，到处吹牛就代表你没干正事，男人的志向坍塌萎缩了，还能有好的吗?
改开刚开始的1991 年前后，牟其中了解到苏联急需大量生活物资，而国内一些航空公司需要飞机。
于是抓住这个机会，用500车皮的日用品罐头、服装、暖瓶等，从前苏联换回了4架图-154飞机，
再将飞机转卖给四川航空，从中赚取大约1.6亿人民币，巨额利润。
“罐头换飞机的传奇让牟其中洋洋得意，到处演讲吹牛，要把喜马拉雅山炸一个口子，要把满洲里打造成北方香港，要8个月造出超过 IBM 的芯片。
他的结局可想而知，
而他带出的冯仑，潘石屹，王功权等人，只是稍微踏实了一点，却做出了很大的成绩。

六二。介于石，不终日，贞吉。
象曰：不终日，贞吉，以中正也。
蒋介石，原名是瑞元，又叫蒋志清。
当年他非常喜欢易经，读到这一卦，把名字改了，
名介石，字中正。
这一句，市面的通俗解释是，保持清醒，坚守中正，
像石头般坚定不移，这样就长期是好的。

这种称斤卖的大道理有啥意思?
真正的含义是什么?
介，在甲骨文中，是披上盔甲之意。有甲打无甲是碾压。
这句的意思是，
披上盔甲，坚如磐石，抓住有利时机突击，不要一整天都在打，这才是中正之道。
市面上有很多机会，但不是每个机会都属于你，你只能找自己熟悉的领域，
抓住几个重点概念股，仔细研究，深入了解，打一
把主升浪，拿完走人，轻仓休息
不要每天买来卖去， 一整天都在交易，到最后啥也赚不着。
大王给大家讲过孙子兵法的核心意图，
就是用积累巨大势能，狮子搏兔干一锤子买卖。
常凯申说80万打60万，优势在我，
真是笑了，800万打60万才叫优势好吗?那才叫
介于石”。
他的易经根本没读明白啊。
常凯申打仗和炒股，喜欢追涨杀跌，永远满仓，什
么钱都要赚，懂不懂的都要参与，买的股票跟开超
市的似的，有800万大军都赔的精光。
教员呢?他看不懂看不清的时候，从来都是轻仓，
游击战麻雀战跟你玩玩，
真看清形势了，跟着主线， 一把重仓，三大战役直接赚的盆满钵满，后面的事情，他已经不操心了。
教员才是真正择时和交易大师，润之才是真介石"。

六三：盱豫，悔；迟有悔。
象曰：盱豫有悔，位不当也。
盱，是眼神往上看。
媚眼取悦别人，招致悔恨，要是悔悟太迟，又生悔
恨。为啥?位置不当啊。
豫，是生机勃勃，大机会来临，
很多人抓住机会的方法，是依附别人，讨好别人，
抱住大佬的大腿，自己不想出力，就希望人家提携
一把，给你资源你致富。
结果后悔了，再后来发现时机错过了，又后悔了。
原因很简单，凭啥呢?人家为何带你?
当年的中山先生也失误过，自己搞革命，靠俄国，
靠日本，那些人没一个真帮他的。
自己的不接地气，被人称为“大炮”。
到后来才明白枪杆子里出政权”,要打造自己的基
本盘，等建立黄埔军校时，为时已晚，没几年就走
了。
常凯申也一样，每天想的是靠英靠美，天天指望别
人援助，骨子里就不相信自己能行。
易经告诉我们，要抓住大机会，成就事业人生，你
终究谁也靠不住，只能靠自己。
不是说你不能找大佬和大腿，而是说你要清楚的知
道，一切都归根到底是因为你有能力，有位置。
你有自己的志向，价值，人品，坚持，才能获得真
正的支持的帮助，获得平等的尊重和欣赏
独立自主，才能丰衣足食
从今天开始，打造和强化自己的基本盘。
