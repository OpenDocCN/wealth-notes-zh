
# 2025年房地产政策的三大变化

2025年是房地产“止跌回稳”调控的关键一年，也是“十五五”规划制定年。目前能够给出来的扶持政策已经基本到顶了，去年10月住建概括为“四个取消、四个降低、两个增加”，之后再想继续加力的空间已经很小了，想要推动房地产行业恢复健康发展进而复苏，更多需要时间，接下来主要还是继续执行住建的“442”政策。

但这也不意味着2025年就没有变化了。梳理了三个可能的新变化：

## 第一个变化：尽力而为，量力而行

主要是针对很受关注的“三大工程”和“城市更新”概念，最近留言也有读者咨询这方面会不会在2025年加力。认为情况已经发生了变化，“三大工程”已经大幅弱化，而城市更新则开始取代“三大工程”，但力度会比较有限，正如分管副总在全国城市更新工作部署视频会议上说的“要讲究策略方法，坚持尽力而为、量力而行”。

早在2023年经济增长面临困难的时候，国家推出了一系列稳增长政策，但很有意思的是表现最显眼的并不是主导项目建设的发改系统，而是住建系统。2023年4月最高层会议首次提出城中村改造、平急两用公共基础设施建设、保障性住房建设等“三大工程”，7月最高层会议再次重申，紧接着分管副总分别在7月、9月部署这三大工程，到了12月底的经济会议，再次提出“加快推进”。这才是2023年稳增长政策中最亮眼、最被期待的，也最适合作为传统房地产和土地财政经济的过度，因此从2023年二季度到2024年，一直盛传国家要在三大工程以及收购存量商品房方面投入重金。

但注意，虽然2023年12月经济会议提出“加快推进”，但2024年所有的高层会议再也没提“三大工程”，这项工作似乎瞬间消失了。与此同时自2024年3月开始，最高层正式关注房地产问题，3月国常会突然研究房地产问题，突出强调房地产的重要性，提出“房地产产业链条长、涉及面广，事关人民群众切身利益，事关经济社会发展大局”，到5月国常会后推出517新政，8月开始推出一系列政策，最终9月明确“止跌回稳”的长期目标，这意味着国家最高层意识到房地产问题非常严峻，已经不是战术层面能够解决的了，“三大工程”加力已经不合时宜了。

这也体现在分管副总的调研上，2024年4月分管副总调研房地产，去的不是三大工程突出的城市而是严控增量的郑州，关注的焦点也变为了保交房，紧接着5月召开了全国切实做好保交房工作视频会议。工作重心的转变，代表着国家对房地产问题的空前重视。在这种情况下，国家已经不可能在“三大工程”上投入大量资金，市场机构还有房地产相关行业不过是想象力过于旺盛罢了。

“三大工程”已经事实上搁置了，起码相比2023年推出时弱化了很多，这是“量力而为”的一方面。而如前面说的，当红的城市更新也要“量力而为”，注意在官方用语中，凡是出现“尽力而为、量力而为”的，一般就是倾向于量入为出，不要大搞了，如果有钱那就多做，没钱就慢点来，把钱集中到更紧急的领域。

目前能够确定的就是在2025年底前完成三项工作：全面完成2000年底前建成的城镇老旧小区改造任务、基本完成已排查出老化燃气管道的更新改造任务、基本消除县级城市建成区黑臭水体。其他的就不强求了。

“三大工程”中唯一得以强化的，只有城中村改造，这也是去年9月宣布的“新增100万套城中村改造”，而在年度住建工作会议上则扩充为“在新增100万套的基础上继续扩大城中村改造规模”，改造范围已拓展到300多个城市，涉及面很广，如果自家属于城中村，可以关注下当地公告。

## 第二个变化：严格控制增量，加速止跌回稳

2023年7月明确提出“我国房地产市场供求关系发生重大变化”，供需决定价格，房地产也不例外。而从2022年5月开始，烂尾就成为普遍关注的焦点，而当一对小夫妻不断在网上发布房地产相关视频，郑州成为被关注的焦点，于是2024年4月分管副总赴郑州调研保交房。

2022年郑州被爆全国烂尾楼第一，但仅仅两年多，到2024年郑州就基本完成了保交房，而且还成了全国去库存第一的重点城市，全年狭义库存规模减少了214万平方米，这还是在网上负面舆论发酵导致购房意愿大为下降、狭义消化周期长达25个月的情况下实现的。

正因此，自2024年下半年以来，郑州经验已经成为国家房地产政策调控的重要参考。一个核心经验就是减少城市土地供应量，在面临财政压力的情况下保持克制，从2022年开始郑州土拍成交宗数迅速下降，从2021年的98宗，骤降到2022年的49宗、2023年的35宗以及2024年的20宗，比2018年巅峰的151宗少了131宗。不要觉得是因为土地卖不出去，要知道作为省会城市，有很办法可以把土地变现，城投平台、国企就是干这个的这就充分体现了郑州的决心。

正因此，从2022年开始，郑州连续三年商品房成交大于供应，2024年成交556万平方米，是新增供应量291万平方米的近两倍，这才实现了全国重点城市第一的去库存成绩。

所以2024年下半年这个经验也将被推广到全国。2024年9月最高层会议和12月经济会议，都强调“商品房建设要严控增量”，注意这个政策绝非短期应急，也必将长期坚持，也就是“人房地钱要素联动新机制”强调的“以人定房、以房定地、以房定钱”。

而2025年新的变化是不仅商品房要严控增量，保障房也要开始严控增量。最近网上有一些提出加大保障房建设的呼声，出现公开呼声往往就是面临政策障碍的信号之一。年度住建工作会议上，在“增加保障性住房供给”前面也加上了“以需定购、以需定建”，相比2023年刚提出保障房建设时候的迅猛劲头，现在弱化了很多，明显是在压过于激进的保障房建设，实际上目前一些地方已经出现了以保障房为名继续搞土地财政，在房地产供给严重过剩，房价不断下降的情况下，大规模搞保障房建设很不合时宜。

在保障性住房方面，也出现了新变化，从过去强调租赁为主，开始向配售转变。配租、配售这个北京部委人员会比较熟悉，都是保障房，前者是以相对廉价的租金长期租赁，后者以核定成本价购买但不能交易。但注意绝无可能盲目、大规模收购，收购的前提一定是目标商品房价格足够低。

## 第三个变化：要探索、寻找未来房地产建设领域稳增长的新项目

正如前面所说，2025年是多个房地产或者说城多建设领域项目的完结年，比如老旧小区改造、燃气管道更新、黑臭水体处理等，城中村改造虽然加码，但仍然是循序渐进开展，“棚改”已经不会再出现了。而乡村的农村危房和抗震改造工作，也基本在2025年阶段性完结。

传统商品房建设和新提出的保障房建设都要严控增量，接下来怎么办?考虑到2025年是“十五五”规划制定年，参考过去经验，每当经济增长不理想的时候，基建和城乡建设都会被重点关注。

老旧小区改造就是2020年提出的，7月下发的指导意见提出了“十四五末期力争基本完成2000年底前建成的需改造城镇老旧小区改造任务”的目标。而到了2022年二季度经济再次遇到难题的时候，5月提出了城市燃气管道等老化更新改造；2023年则推出“三大工程”，只不过2024年发现形势不对，就大幅削弱了。

2025年房地产政策和建设项目已经基本确定，以后怎么办?还能搞什么?

目前高层明确提出的是绿色建筑，但当前只能说未来值得期待，短期内量级不够；而住建提出“中国建造”升级版和好房子概念，去年也非常隆重的举行为了中国建筑科技展，但概念终究要转变为具体的项目，这也是接下来几个月需要重点关注的。
