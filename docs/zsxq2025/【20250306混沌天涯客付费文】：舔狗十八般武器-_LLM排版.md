
# 舔狗十八般武器

原创混沌天涯客 2025年02月21日15:57 加拿大

## 01

美国是没有企业家的，比如没有人称呼马斯克是企业家，更具体点，根本没有人称呼他为XX家。以前叫他火星人、狂人、巨头，现在喊他是川普的亲儿子或干爹。

在川普1月20日的就职典礼上，一帮巨头挤在一起，聆听川普演讲。那演讲我也听了，一个多小时，没有稿子，又臭又长。时间就是金钱，一分钟比一斤黄金更贵重的巨头们，表现出了明显的不耐烦。他们站没站像，交头接耳，有的插兜，有的查手机，有的开着直播，有的盯着别人的老婆。

是不耐烦，是形式主义，但他们必须要站在那里，尽由摄像机捕捉。因为事情的发展具有两面性，站在那里或许得不到什么好处，但不站在那里肯定会有害处。

果然，就像前几天挨个数人头的时候在BAT三巨头里发现缺了李彦宏的后脑勺，百度的股价立刻大跌一样。没能出现在川普座谈会上的美国巨头黄仁勋，其英伟达也很快迎来了大跌。

黄仁勋不参加川普就职礼，是因为那时候他在中国。新春佳节，忙着与英伟达中国员工开年会、发红包、扭秧歌。

我查了一下，资料显示黄仁勋仍然是拿着台湾省身份证的，所以他也算是中国人。与另一位也拿台湾身份证的他的表侄女AMD的首席执行官苏姿丰一样。

## 02

中国人的聪明，在芯片领域展现得淋漓尽致，最强的芯片设计商，最大的芯片制造商，都在中国人的领导之下一骑绝尘。所以，能击败他们，或者说能让他们感受到压力的，肯定也是中国人。

1月27日，川普上台满一周，DeepSeek横空出世，将华尔街的科技股锤掉了一万亿美金。其中，英伟达受伤最重，当日下跌17%。

马上，刚返回美国的黄仁勋就被川普叫进了白宫，两个人商谈半日。具体谈了什么？大嘴巴的川普闭口不谈，爱说话的黄仁勋什么也没说。

请注意，在美国当企业家，必须是要会演讲的。面对数万人的现场大会不怯阵，套个T恤披个黑袄就上场；面对记者的长枪短炮更是谈笑风生，像在直播间里找到家人一样。

但这一次，面对DeepSeek的冲击，黄仁勋保持沉默，什么都没讲。

同样的，冲击英伟达的DeepSeek创始人梁文锋，这一个月里火遍地球了，也什么都没讲。他没参加年会，没接受采访，没借着喧嚣声势坐进任何直播间，即便是前几天他在座谈会上露了脸，也只是认真听使劲记，什么都没讲。

现如今，不仅是美国的巨头们要会演讲，中国的企业家玩社媒，搞曝光，同样一套一套，花样新潮。开会时跟梁文锋坐一张桌子的老前辈，无论是任老爷子还是雷军大哥，均是个中高手。

但很奇怪是不是，大红人梁文锋，一点点曝光度都没有，唯一的个人专访还是在DeepSeek没火之前。一旦火了，他是一句话都没透出过。
