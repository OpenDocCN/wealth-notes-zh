
# 3 亿建仓：超牛股再有高手入局
# 251016 尹香武
# 前言
# 反复有好结果的人，就是该跟随的榜样。
# 其实在推送《我国此股，会涨两倍》、《此股市值将飙，会创证券新高》、《“三年五倍”超牛股》之后，与我的《超级客户》一样，来自深圳的某投资高手，也《转调重仓》了这只股票，耗资大约 3 亿。
# 今天和他通了个电话，专门聊了这件事。他三个说法对大家应该是有帮助的：
# ①此股是迄今他在中国科技股上的最大持仓。
# ②他为何看中这只股票？将其作最大“好仓”？
# ③对其股价预期 16 个月是翻倍才会考虑了结。
# 下面我将精要部分推送给有志者参阅。
# 心满意足、有新期待？
# 第一，他为何看中这只股票呢？他说的是他所看到的信息监测表明，在中国 AI 领域真正的领头羊其实是华为与阿里巴巴，DEEPSEEK 只是个工具模型，但是在平台上，算不上大。
# 其中，阿里的 AI 最值得看好，因为阿里在 AI 技术上、模型上、数据中心上都取得了进展，这是独家的，更难能可贵的是阿里巴巴具有 AI 落地的最好条件，各项业务之间，具有交互加强的作用，很可能因此带来加速度。
# 目前阿里的全球数据中心的亚太布局是最强的，开源模型也是最强的。内外部条件都处于最佳状态。这是华为所没有的。大佬文集圈而且阿里巴巴与美资关系很好，能够获得更多的外部资源。美资持仓也非常大，因此会有更多的外部支持可能给到阿里巴巴。
# 从 AI 产业的纵向条件、横向条件来看，阿里巴巴是全球第一、没有之一。这是他重仓的关键原因。
# 第二，此股是迄今他在中国科技股上的最大持仓。
# 他其实持有很多美国公司的科技股，之前对中国科技股都是敬而远之的，在中国股市更多是偏向传统产业的《基石资产》。这只股票，是他的第一次中国科技股产生了浓厚兴趣。
# 目前他持仓的金额折算成人民币大约 3 亿，在他持仓结构中，一出手就到了 3.1%，好生给了，是很是断的。而且他这次判断是做了综合考虑的。
# 未来，他有考虑在出一些美股，然后将资金转入阿里巴巴。也就是说意图继续加仓。他说这是他对中国科技股的最大好仓。“好仓”是港台地区的专用词语，意思就是看好的仓位的意思。
# 第三，对此股股价，他预期是翻倍才会考虑了结。
# 我直接把他给我的一段文字给大家看吧：自 2021 年“控制资本无需扩展”开始，互联网板块整体下行，阿里经历了漫长低谷，股价从高点腰斩再腰斩。直到 2024 年 4 月，股价才企稳回升，并在 2025 年真正企稳、本年累计上涨逾两倍。
# 阿里 2025 财年二季营收 2476.5 亿，剔除出售高鑫零售、银泰的影响，同比增 10%。净利润同比大增 76%，呈现良好势头。阿里将核心业务重组为四大板块：阿里中国电商集团、阿里国际数字商业集团、云智能集团及其他业务。
# 最大亮点来自淘天集团与即时零售的突破。阿里整合饿了么与飞猪，推出“淘宝闪购”，成功切入外卖与即时零售赛道。闪购推出短短几月，日订单达 1.2 亿单、月交易买家超 3 亿，骑手数量增长 3 倍。这不仅让阿里在近场零售上缩小与京东的差距，还重构了“线上+线下”供应链模式。预计未来三年，闪购及即时零售可为平台贡献 1 万亿新增成交额。这场看似“500 亿外卖大战”，实际上是阿里一次战略性的突围，将原本滞后的 B2C 远场模式升级为近场即时零售，显著增强流量与消费黏性。
# 第二个亮点是云智能集团。该业务收入达 334 亿，同比增长 26%，AI 相关收入连续八个季度环比增长。阿里计划未来三年投入 3800 亿发展云与 AI 基础设施，聚焦算力、大模型与 AI 芯片平头哥 PPU。阿里已经将“AI+云”将定为长期战略重心。
# 其他业务如菜鸟、盒马、阿里健康、钉钉、高德等，协同效应正逐步显现，为消费体系补上物流与近场履约短板。
# 总体来看，阿里正从此前的“防守调整期”转向“结构重建期”，双主线明确：一是消费板块回归效率导向，二是 AI+云布局打开长期空间。零售与科技双轮驱动，使阿里重新具备成长性。虽然股价自低点翻倍，但不到 20 倍 PE 仍处合理区间下沿。阿里从危机中完成业务重塑，开始进入新增长周期，市盈率看高 40 也就是从目前的 18.7 倍上 涨 114%，是可以预期在 16 个月内实现的。
