
# 有本事的人最后都因道德问题身败名裂！

任何一个有本事的人最后以道德问题身败名裂，都是因为犯了天条，或者把所有人都搞到了自己的对立面。大伯一辈子都在经济领域的岗位上工作，快退休的时候，突然把大伯高升调入去干纪律检察的工作。之前的工 作性质是实操，调动资源，安排人事，后面的工作性质是监督。为什么要把一个实操一辈子的人调去监督呢？我也问过大伯这个问题，大伯说：如果要查问题，没有谁比实际参与的人更清楚哪些事和哪些人在哪个环节有哪些问题，让这样的人去查问题不用拐弯抹角，可以直奔现场。

当年，康熙王朝的索额图倒台，半天审不出什么来，然后康熙让明珠去审，一下子全都审出来了，因为索额图的很多事明珠一清二楚，明珠的问题，索额图也一清二楚。做事的人总是会有纰漏，就算再小心，也只能对外面的人滴水不漏，对于内部的人就是漏的跟筛子一样！

雍正让岳钟琪去拿下年羹尧，为啥不是从京师派钦差，而是让年羹尧的副手去办呢？因为只有年羹尧的副手才可能有类似的威望稳住那些人，如果是外人去抓人，只会让内部形成一个整体。年羹尧手上可是有十几万大军，这是一道圣旨能搞定的事么？

每一个时代都有一个时代需要解决的问题，要解决这些问题就需要敢打敢拼的人，所谓非常之时，须待非常之人，非常之人，方可建非常之功。解决了当下问题的有功之人，如果不愿意隐退，那就只能兴大狱。因为每一个时代都会有每一个时代的新问题，而新问题不可能依赖老的一群人来解决，如果老的那群人占着茅坑不拉屎，那就只能...

每一个时代都会有这个时代的新问题，每一个时代都会有一群新人来解决这些新问题，这群新人通过解决新的问题来获得地位、荣誉和价值。上一批人如果能及时隐退，就能保全名节，获得尊荣，否则就只能身败名裂。

以利导行，以罪收口，是更古以来的规矩。古今中外，古往今来，只要是人类社会，这样的规矩就更古不变！

大伯当年荣升都察院的时候，接任者是一个敢打敢拼，很会来事的小伙子，在当年那个改革的关键时刻，这样 的新锐力量，更受赏识。大伯一生清廉，原则性强，大佬文集圈所以，采取的是荣升的方式，组织上没有用大伯推荐和栽培的人来接大伯的位置，而是从另一个部门调了那个小伙子来接任大伯位置。那个小伙子姓仇名声，因为工作出色，后来调去云南昆明当一把手，再后来荣升云南巡抚，走得比大伯更远，然而，仇和最后还是栽了跟头，现在在秦城监狱反省。

只要做事就会得罪人，只要得罪人就会有把柄，只要有把柄就祸事不免，能不能平安落地，保全名节就看有没有人愿意保你。在《投名状》电影中，庞青云在只有 800 个人走投无路的时候庞青云没有怕过什么，当庞青云荣升两江总督功成名就的时候，反而站在河对岸喃喃自语道：我这一生如履薄冰，我能走到对岸么？
