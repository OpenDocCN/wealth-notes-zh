
# 为什么那些不懂行的人也能在互联网上赚钱？

## 全文概要

讲述者深入探讨了个人品牌建设、行业圈子特性及互联网营销策略。强调在任何行业，卓越技能与作品是赢得尊重的基础，而进入固化行业圈子需要持续分享有价值内容，尤其在抖音、快手、小红书等平台。互联网营销中，自我定位与自然沟通策略对建立信任至关重要。此外，即便缺乏专业背景，也能通过特定策略在某些领域实现经济收益，突显突破传统认知的重要性。最后，互联网的混沌与虚拟性要求营销与创业需灵活调整策略，理解并运用其独特规则。

## 第一条原则：以作品出发

无论什么行业，都是以作品出发的。无论是跑步、越野或者是摄影，无论你从事什么行业，你能够有拿得出手的硬本领，硬本事，别人就尊重你。你有一技之长，有一个拿得出手的本领，比如写字，写毛笔字很厉害很牛逼，那么他就会仰慕你，敬佩你。

## 第二条原则：事物相关性

我们所讲到的事物都有相关性。比如拍了很多马拉松，发现拍马拉松的这个人他为什么又出现在我镜头了？虽然这次不是马拉松比赛，而是越野比赛。么我又发现我上一次拍了预约比赛的这一个选手，我去少门拍，我去莆田拍越野赛，怎么他又在了，他又去商家了。所以我就知道了。

中国现阶段很多领域都是相通的，很多层级都是很固定的，很固化的。你看起来有几千上万、几十万人规模，其实来来往往复复就是那几百几千几万人就是那一个群体，就是那几十万人100万人就在那个地方在玩掉一圈子。

## 第三条原则：圈层特性

当你沉浸在一个比赛一个逻辑当中，比如说月野赛这一个细分类目当中，只要你干的时间足够久，坚持的足够久，你就会发现举办这个越野赛的另外一场越野赛的，北方某个地方的某场越野赛的，南方某个城市哪场越野赛的他们的负责人都认识的，承包商运营商都认识的，这个圈子就这么大。

## 第四条原则：公开平台分布

那么你要进入到这个圈子里面，首先我刚刚所讲到提高自己能力，用作品说话。第二就是这些圈层在互联网上分布在哪里？对吧？公开平台分布在哪里？很明显那就是抖音、快手、小红书、必样公众号、视频号，他们这一些高手大佬也会去看你在你这个领域不但分享什么价值，他刷的时候就刷到你各位。

## 第五条原则：自我定位与沟通策略

很多人一直说怎么样找同行，或者是说怎么样让别人关注我，加我，其实你就是在这个平台上不断分享价值。你只要分享时间久了，突然有一个上热门了让更多这个领域赛道能看到你，你就他就知道你了，你只要足够的深入，你在这个领域只是说认识你人多少的问题而已。

## 第六条原则：营销推广逻辑

在互联网做营销推广，你要理解一个核心的一点，家里的人并不是100%都是客户，有些人是麻烦，有些单会来问你，各种非常莫名其妙的问你，有些人家里你和他不认识，他说你好大佬。那么你想问你借个100块、500块、1000块，你都和他不认识，他莫名其妙来加你，加完你之后莫名其妙来寄钱。

## 第七条原则：突破传统认知

中国现阶段有很多行业门槛很低，所以有很多人完全没有掌握这个技能，他就能够分饭吃了。兄弟们，包括说刚刚结束的上门越野赛，那个巡线员，他自己两步路都不会看，两步录这个户外的导航轨迹软件他自己都不会看，他还当那个巡线员，还是说我第一次来当这种机就是光绪户对吧？他是赞助商明额光绪户，然后来这个地方体验一下，反正也不要钱，免费对吧？很多行业都是如此。

## 第八条原则：互联网社会的混沌与虚拟性

这个社会是混沌的，这个互联网社会是没有规则的，是虚幻的。你在互联网上想赚钱，想营销，想做什么项目，你一定要理解突破自己原有的这种固态的认知，就是因为你你不会你不懂这个社候就是不懂人赚那些不懂行的人钱。

## 结语

互联网营销与创业需要灵活调整策略，理解并运用其独特规则，突破传统认知，才能在互联网上获得成功。
