
# 卢克文7.26星球：海南自贸港封关，红利在哪风险在哪

## 海南自贸港封关，红利在哪?风险在哪?
## 文/卢克文工作室嘉宾星海舰长

7月23日，国务院新闻发布会上，国家发展改革委正式官宣，定于2025年12月18日正式启动海南自由贸易港封关。那么，封关到底指什么，具体有哪些好处，我们作为普通人，又能有什么机会呢？

“封关”是一个海关术语，封的是“海关”，让海南全岛成为一个“境内关外”区域。封关之后，海南执行的是一线放开，二线管住，岛内自由的政策。

所谓“一线”，是指国境线，也就是从境外到海南自贸港，全部放开，只要在免税税目上的货物，都可以自由入境，享受零关税政策。目前，一共有21%品类的税目商品享受这一政策，年底封关后，这一比例将提高到74%。也就是说，除了一些危险商品，可能造成倾销的商品，以及国家命令禁止进口或者加征特别关税的商品，其余的，运到海南都可以零关税。

那二线管住呢？“二线”主要是指海南自贸港与境内其他区域之间的分割线，也就是从海南自贸港进入内地的货物，要征收关税了，这就是“管住”，主要是为了防止免税商品无序流入内地造成冲击。还留了个口子，在海南加工增值达到30%的进口商品，也可以零关税进入内地。比如，乌克兰小麦进入中国市场是要征收关税的，但封关后进入海南是免费的，你如果在海南办个厂子，免税买100万的乌克兰小麦，磨成面粉，然后以130万的价格把这些面粉卖到广东，那就不用交关税了。

岛内自由呢？主要是在海南自贸港岛内，货物、资金、人员等可以较为自由地流动和运作。所以，所谓封关主要是针对商品的，封关也并不是封岛，而对于我们内地人来说，去海南出差、旅游，和现在一样，不需要办理额外的证件，相关流程也没什么变化。相反，封关后，对内地人享受进口商品，反而是一种利好。

为啥？因为我们买进口商品，其实大部分都要承担13%~17%的进口关税。但是如果我们不在内地买，而是去海南走一趟，回的时候就能直接买到八五折的商品，而且享受高达10万元的免税额度(单件商品8000元的限额也取消了)。相比而言，香港的条件就苛刻多了，居民旅客免税额度只有12000元，超出就要交税。

这意味着什么？意味着我们未来只需要跑一趟海南，就能带回价值10万的进口商品，自用还是转让，都可以享受到价格的红利。例如索尼70-200mm G2镜头在香港折合人民币约8660元，而内地行货价格可能高达10500元左右，在香港买要便宜接近2000元。但免税额度有上限，你只能买一个。如果你到海南，一次就能把从相机到镜头全给配齐了，而且因为这些都属于国行，还能享受国行保修。除了相机，还有iphone、化妆品、奢侈品等等，都可以在海南免税买到，毫不意外，海南会成为属于高端用户的“购物天堂”。

不过呢？如果把海南封关的意义，仅仅理解为买东西便宜了，那只能说小了，格局小了。

海南封关的重大意义，是用封关的政策，来给海南提供一整套全新的发展动能。说实话，海南此前的经济发展，并不尽如人意。因为地形原因以及和内地没有公路铁路相连通，也缺乏制造业产业链，海南的经济只能依靠旅游业和农业，除了三亚和椰汁，你还听过什么海南品牌？但这两点，是拉动不了整个海南的。所以，这次的自贸港，国家层面给足了海南政策。

其中，最诱人的就是税收政策了。比如，在内地，企业所得税一般是25%，而个人所得税，只要月工资超过1万就要交10%，月薪8万5以上的，个税高达45%。虽然这一税率在世界上相比不算太高，但已经让很多企业和个人肉疼了，特别是一些高收入群体。这次海南自贸港内部的税率是多少呢？

企业所得税，只要经营范围在“鼓励类企业”名录上，只交15%的企业所得税。如果是核心四大产业(旅游业、现代服务业、高新技术产业)企业，直接免征所得税。而个人呢？最低3%，最高15%。

显然，海南封关后，对一些高盈利企业和高收入人群，有巨大的诱惑力。在资本的趋利性引导下，会吸引大量公司和人群搬到海南去。我们想一下，香港是怎么发展起来的？就是香港低至8.25%~16.5%的企业利税，才引得7000多家跨国公司在香港落地。而海南15%的企业所得税再加上30%工业增加值免税政策，简直是原材料依赖进口的加工业企业的巨大红利，没准已经有很多内地老板在研究了。

对于一些做国际贸易和跨境电商的企业，海南的吸引力也很大。毕竟海南已经开展了国际互联网交互试点，那么海南的跨境企业，在与外国客户沟通的时候会更加顺畅，沟通效率也会更高。所以，毫不夸张地说，低税政策一旦落地，会有大量大陆企业搬迁到海南。企业有了，人有了，产业有了，企业要经营，人要消费，产业要发展，海南经济不就起来了么？

当然，海南搞自贸港的目的，不仅仅是吸引内地企业，也在着力吸引外资。比如，海南封关后，采取“非禁即入”的政策，也就是说，外商想投资，只要不在投资准入负面清单之上，几乎所有的都可以投资！比如原来受限的外国医院，是不是可以直接在海南落地？连带着一些外国已经审批但中国还未审批的药物，也可以在海南使用了？就算这一条落地不了，普通人去海南的医院看病，吃进口的抗癌原研药或者靶向药，也会比内地便宜的多。

此外，海南自贸港带来的不仅是人、企业、资源的自由流动，还有钱的自由流动。你在海南做国际生意，需要资金往来吧？而中国是有外汇管制的，这个问题怎么解决？海南建设了一种FT账户，海南封关后，将全面执行这一本外币一体化、账户内可自由兑换的银行账户体系。这一点，恐怕所有头疼于中国的银行收汇难、结汇难、转账难的的国际贸易企业、个人或跨境电商，都知道含金量有多大。

如果你是外国企业，也可以按准入前国民待遇原则，获得境内银行的结算、存款、融资、结售汇等金融服务。本质来说，账户独立于传统账户体系，实行分FT账核算，既方便兑换，也可以防范风险。

更关键的在于，海南封关政策红利外溢之后，也让我们普通人有了更多可能，无论是去海南找工作，还是去海南创业勇闯国际市场，甚至去海南当代购，都要比已经内卷到极致的内地市场要强许多。也许，这就是海南封关最大的意义。

海南封关，是国家层面通过政策来给海南释放红利，拉动海南经济。但如果我们跳出海南，从更大的视角来看呢？毫无疑问，海南可能成为中国高质量开放的又一个重要窗口。

可以回过头来看看2020年6月印发全国的《海南自由贸易港建设总体方案》，开篇是怎么写的？"当今世界正在经历新一轮大发展大变革大调整，保护主义、单边主义抬头，经济全球化遭遇更大的逆风和回头浪。建立自由贸易港政策和制度体系，着眼国内国际两个大局，深入研究、统筹考虑、科学谋划作出的战略决策。"看到没？这是战略决策，高度一下子就上去了，高度甚至直追1980年改革开放伊始，建立的经济特区。而且单从给的政策来看，海南比当年的深圳的政策要好多了，深圳都能从一个小渔村发展到当今的超一线城市，那拥有更宽松政策的海南会如何？

深圳的意义，不仅仅在于其成为了中国的电子之都，也不仅仅在于它孕育了以华为为代表的一系列科创企业，更在于深圳是中国的试验田，无数制度、政策创新、产业模式和人才使用方式，都是在深圳试验之后，才推向全国的，最终造就了中国如今的发展成就。

问题来了，海南会不会也走这条路？

在全球化退潮、世界右翼崛起、贸易壁垒森严、原有国际贸易体系濒临崩溃的大形势下，如何继续赋予中国发展动能？答案，可能就在海南这个中国顶层设计、主动布局的自贸港身上，世界越封闭，中国越要开放，而且是高质量开放。一旦海南的自贸港模式成功，其探索出的新经济政策和制度体系，很快就会向内陆推广，并成为中国新时代高质量开放的一部分。这时候，受益的就不只是海南的普通老百姓了，而是我们每一个人了。

当然，对于海南前景，我们在保持乐观的同时，也不能脑子过热。不信可以看看汕头，当年和深圳一起建立经济特区，可是现在呢？还有什么声音？海南也差不多，改革开放以来国家给的政策不能说少吧？海南毗邻香港，面向东南亚，地理禀赋不能说差吧？但当下的海南，发展得如何呢？显然不如人意。所以，有了政策固然好，但具体还要看产业扶持、人才使用、区域协作以及国际大势等方面的努力和机缘。

而且，海南自贸港的设计和论证，始于2010年代，那个年代的国际环境，和现在能是一样么？那么按照当时国际环境设计出来的政策和制度，放到今天，可能在执行的时候就很难达到预期目标了。不是你开放了，就一定有钱进来。

很多炒房中介都在炒海南是下一个香港，但别忘了，香港的支柱产业是金融业，而海南呢？有让金融产业变成支柱的人才和体系优势吗？

别说海南了，我们可以看看香港的金融业如今如何？随着中国与西方国家关系恶化，西方国家逐渐取消了香港的特殊地位，很多制裁政策都与大陆看齐了，这样一来，外国公司去香港经营或者在香港设立地区总部，都受到了很大限制，香港资金流入流出也遭遇了一定管制。这就导致了，香港的亚洲金融中心地位逐步被新加坡夺走，连带着金融人才也在流失，2019年至2023年间，选择永久离港并提取强积金(MPF)的人数，累计达到了15万5千人。

连整个司法、社会制度、金融体系都与西方国家接轨的香港都如此，又怎么能指望完全执行大陆制度的海南，成为金融中心呢？哪怕你在制度上已经做了很多修订和完善，但就凭你在中国大陆一条，大部分西方国家资金就不会来。

再加上海南的外汇管制并没有彻底放开，很多外来资金仍然要受到比较严格的监管，吸引外资的政策，很可能达不到预期。搞到最后，海南可能的确会吸引到一批企业，但以内地过来避税的企业为主。问题来了，内地企业去海南，那在内地交的税就不就少了？

本质上来说，海南还是想吸引外资，但如果受限于种种原因外资不来，那海南还和没封关一样，处于自己的内循环之中，封关意义大大降低。

而且，就算是吸引内地人来旅游购物，说实话，以海南的整体服务业水平，相比新马泰也有一定差距(这些年海南的负面新闻不少)，海南消费又高，如果不下功夫重整一下旅游业的话，最后可能只火了一批代购，仅此而已。

以上这些，都是海南面临的困难和不足。所以，海南自贸港的前景是有的，但怎么把好棋下好，就看海南从上到下，怎么努力了。
