
# 理解人生“为什么”和“怎么办”

## 人生脉络“为什么”

我们对自己的人生已经无限熟悉，但仍然有必要梳理人生脉络。直接抓住超级机会信号固然重要，但理解个人命运与时代环境和历史进程的交汇更为关键。

中科院心理所的钱炜老师分享了一组好问题，帮助我们意识到梳理人生脉络的必要性。他向世界卫生组织的欧洲干事提出了三个问题：
1. 支持家族命运轮回的最大推动力量是什么？
2. 考虑到世界其他地区生活吗？是哪里？为什么？
3. 如果要给孩子留下重要的家训，会告诉他什么？

这些问题引导我们将个人命运置于更大的时间尺度和社会背景下理解。类似地，我们普通人也应思考这些问题，将个人命运与时代环境相结合。

## 人生脉络“怎么办”

每个人的生活处境、性格兴趣、出身背景不同，需要不同的方法来梳理人生动线。我们可以向精英和身边人学习，寻找适合不同场景的超级问题。

马斯克以“第一性原理”贯穿人生，彼得·蒂尔用“你知道但是很少有人赞同的真理是什么？”来筛选创业者。费曼建议每个人准备一组12个左右的超级问题，不断测试新知识或结果。巴菲特和芒格分别问“人生的投资机会是一张只能打20个孔位的卡片，你会怎么选？”和“到哪儿去可能会死？别去。”这些问题帮助我们抓住重点、避开障碍。

## 超级问题的受益者

2025年，一些懂得用超级问题将个人命运与时代脉络勾连起来的人已经受益。

1. 中国和美国的网民通过“小红书对账”，绕过媒体茧房，看到了世界的另一面。一些年轻人基于信息差赚钱，如博主Yami通过接俄罗斯旅行团实现“反向”留学。
2. DeepSeek发布，展示了强大的推理和答题能力，其低成本让人意识到AI对人类是范式革命。提问能力可能是普通人实现命运跃迁的机会。
3. 国产动画电影公司可可豆调动138家动画制作公司资源，打造出全球票房冠军《哪吒2》。饺子导演善于提问和求助顶级资源，成功串起顶级团队。

这些事例表明，掌握提问能力，才能在答案泛滥的世界中生存并过得更好。

## 总结

通过超级问题，我们可以梳理人生动线，将个人命运与时代环境相结合。在2025年，提问能力是普通人实现命运跃迁的关键。
