
# 未来属于稳定币

稳定币是骗局还是未来？这是一场事关我们的钱包和生活的货币革命。

美国通过了天才法案，正式迈出了官方推动稳定币发展的第一步。主要的银行比如JP摩根、美国银行、花旗银行，还有电商巨头亚马逊、零售巨头沃尔玛等等，都说正在研究推出自己的稳定币。

稳定币就是一种一比一挂钩美元的加密货币。加密货币如比特币，实际上就是数量无法随意增加的数字黄金。它本质上不是货币，因为它有一套加密算法，使得在网络上人们没有办法随意增加这个东西。所以说它就跟实物黄金类似，只不过它是数字化的，不占空间，传输速度特别快。在地球上任何空间，只要有网络，它都可以快速传输。它的价格波动非常大，为什么呢？因为它值多少钱，完全取决于社会上人们对它的预期，对它的追捧程度。

但是稳定币就不一样，它本质上是美元或者欧元以及其他货币的数字化身，它本身就是货币。当今稳定币发行的两大巨头：Tether 和 Circle。比如，你用美元从 Circle 买了稳定币，这个币跟美元是一比一挂钩的，价值不会波动。你随时可以用你的币来换回美元，也可以用美元去买这个币。收到你买币的美元之后，会把这个美元存在一个大银行里，这个钱只能用来买短期国债，非常安全。它能随时满足你换币或者换美元的需求，它的价值不会波动。而且因为它是加密货币，所以没法仿制，没法造假。它又在网上能够快速的传输，具备了加密货币的这些优势。

你买了这个稳定币，你就可以购物转账。现在很多国家，尤其是对美元特别开放的那些国家如阿根廷，他有些交易，有些店铺、咖啡店就开始接受稳定币。

那么你用稳定币跟你用银行卡去买卖差异在哪呢？

第一，稳定币的结算速度超快，尤其是跨境支付。为什么呢？你原来用银行卡，实际上你的钱是存在银行账户里。你下了个单，那银行就收到你的交易指令，就把你账上的钱汇给卖家的账户。如果是跨行，那银行得通过中央结算系统把钱划到另一家银行去。那家银行再把它划到卖家的账户上。那要是跨国跨境汇款，这个钱还得经过国际的swift系统在两个国家之间的银行结算体系之间划转。表面上看，你下个单很简单，但实际上这背后的银行账户的结算是一系列的指令传输、对账，最终才能实现你的交易支付，非常复杂。我们知道做这个事儿的主要的国际巨头，有万事达VISA，还有中国的银联等等。

但是稳定币就不一样，稳定币就在你手机的专属数字钱包里。你说把它转给谁，他就立刻通过网络直达对方的数字钱包，微信pep458啪一下就到账了。尤其是国际间的转账支付，那速度甩开传统银行100条街。

第二、手续费几乎为零。你用银行卡支付，比如刚才所说的VISA、万事达、银联，他们都要收取一笔费用，你可能感觉不到，但是商家要支付这笔费用，世界支付结算巨头的生意简直太赚钱了，VISA去年他的收入是360亿美元，利润就有200亿美元。但如果你是用稳定币支付，你就再也不需要支付VISA卡这种雁过拔毛的费用了。

亚马逊每年因为买家刷银行卡，要付给VISA、万事达这几家公司几十亿美元。所以现在要是有不用花钱就能够支付得更快的手段，那亚马逊能不喜笑颜开，赶紧抢着去推出稳定币吗？那你要是以为稳定币就这么点事儿，那就太低估稳定币了。

你用1万美元从 Circle 那买了一万稳定币，拿到这一万美元之后，他去做什么呢？现在监管要求他只能把这个钱委托在指定的大银行，只能买点超短期的国债。但是能不能拿着1万美元去发放贷款呢？你说现在监管不允许，那没关系，那将来监管可能会允许。银行可以放贷款，Circle 为什么不能加放微信贷款理8，论免费上进它资料是分享群全可以去做。

那手里这1万美元，他能不停地发贷款，Circle理论上能创造出来好几万甚至几十万美元的贷款。你说还能这么干吗？那岂不是变戏法一样在发贷款吗？现在你已经触及到货币银行的本质了。下面讲个轻松点的故事，看完这个故事，保证你对稳定币有个非常清晰的认识。

北宋年间，四川这个地方金银铜十分匮乏。人们买东西的时候，金银铜币就不够用，那怎么办呢？人们就铸造了铁钱来用，可是铁的价值量很低，用来买卖东西非常不方便。你买去两头牛，得拎着几十斤铁，那东西太重还容易被抢，而且四川的空气又潮湿，铁钱特别容易腐烂。这个时候，人们就被逼的想出了一个妙招。

一个有钱的富商叫张三，他去李四那买几头牛，需要付100斤铁。可是双方都怕这100斤铁太重太惹眼，你推这个独轮车，推着这100斤铁很容易被抢，所以张三就给李四写个欠条，欠条写明张三欠李四100斤铁钱，见票付铁。

李四因为信得过张三，因为张三很富裕，大富商，有专门的钱库卫兵保管，钱放那里安全还不容易生锈，于是他很乐意的接受了。

李四去找王五买东西的时候，他说，我手上有张三这张欠条，100斤铁。你是要这张欠条，还是要我去张三那把100斤铁提过来给你？王五一听，这100斤铁我拿着也挺难受，他也知道张三很有信誉，就说，你给我欠条就行了，你在背面写上几个字儿，此欠条转让给王五，这就叫背书。

后来由于张三声名远扬，大家都很相信他，很多人干脆就把自己家里的铁钱拿到张三那，说：我这100斤铁钱存在你钱库里，你给我开一张100斤铁对应的欠条。张三很开心。这个储户就拿着这100斤的欠条，买东西去了。

你有没有发现张三现在开出来的欠条，替代了铁钱变成了流通货币。人们开始用它买卖东西，不再用铁钱了。这个欠条就叫交子，这就是世界上最早的纸币，在北宋年间的四川诞生了。张三当时叫钱铺，跟今天的银行本质上是一回事。

交子跟今天的稳定币本质上是一回事。交子跟铁钱是一比一挂钩的，交子实际上就是铁钱的稳定币，现在你开始触及稳定币的核心了。

到这里，张三就发现这个事完全不一样了。张三现在手里有1000斤铁钱，那都是人们拿来存在他这里，然后换他开的交子票，人们拿着交子去买东西去了，所以他这里有1000斤铁钱，同时他在外面开出了1000斤铁钱的交子票。这1000斤铁钱安安静静的趴在库房里一动不动，为什么呢？因为没人来提现，那张三就打主意了，我要是拿这个铁钱出去放贷款，不是能赚利息吗？于是张三就拿出100斤铁去给赵六放贷。赵六一听这太好了。但是你别给我100斤铁钱，那东西太重太不安全。你也给我开一个100斤铁对应的交子票，我拿着这个交子就去买东西。那贷款到期了，我给你还回来100斤铁加利息不就行了吗？张三一拍大腿，太聪明了。于是张三就又凭空开出来100斤的交子票给了赵六。

现在张三的库房里有1000斤铁钱，但是市面上流通的交子是多少？1100斤。这多出来的100斤哪来的？他发贷款凭空造出来的。张三可开心了，我这1000斤铁钱还趴在这没动，我能不能再继续开交子？当然可以，张三就继续开交子票发贷款，库房里的铁钱还是那1000斤，结果他凭空发的贷款就开出去9000斤的交子票，再加上一开始人家存在这里他开出去的1000斤交子票，合计1万斤。理论上他还可以开更多的交子票出来，发更多的贷款。

天呐，这不是变戏法吗？现实中能这么干吗？现实中银行可不是只吸收存款、发放贷款那么简单，它能凭空造钱来赚钱。现在你知道银行的生意为什么这么赚钱了吧。

张三这个游戏什么时候会玩不下去？那就是赵六他们还不上钱的时候，赵六们把交子都花出去了，可是生意失败没赚到钱，到期了没有铁钱还回来怎么办？那麻烦就大了。

老百姓要是慢慢听到风声说张三发贷款，都收不回来来了，那你说你手里拿着张三印的交子，你慌不慌？你会立即跑到张三家里，让他给你兑铁钱。那张三怎么办？说不定脑子一转，赶紧去找那些不知情的人说，把你的铁钱存到我这里来，我给你付高利息。那不知情的人就把铁钱拿来了。张三就用这些新收进来的铁钱去应付那些挤兑要提走的钱，这就变成庞氏骗局。所以说从银行到庞氏骗局其实就只是一步之遥。

但纸终究包不住火，有一天老百姓都知道了，张三里面的铁钱很少，但是发了这么多交子票，大家全都跑到他这里来挤兑铁钱，这就叫银行挤兑。那张三怎么办？他只能关门跑路。

这个小故事揭示了银行和货币的全部奥秘。你只要掌握了这一点，稳定币、美元、银行这些你全都清楚了。

今天的稳定币跟交子本质上是完全一样的。稳定币就相当于交子，美元就相当于那个铁钱，就是张三的钱铺。交子比铁钱好用，所以交子替代了铁钱。稳定币又比美元好用，所以今天稳定币正在替代美元成为流通货币。但你不要忘了，交子也就是纸币诞生以后，人类的银行体系发生了巨变。那么同样的稳定币诞生以后，同样会对人类的银行体系产生巨大影响。所以说这是一场革命一点都不为过。

我们再讲另一个故事，你就更加明白为什么稳定币势不可挡了。

19世纪的时候，美国是金本位货币体系，就类似于中国北宋时期四川的铁本位体系。当时全国有一万多家小银行，他们就跟富商张三一样，人们来到他们的银行存黄金，然后银行就对应开出了一些标准化的银行券，就跟交子一样，实际上就是钞票。人们就拿这些钞票去买东西，大家也可以拿着钞票再去银行兑换黄金。当时各家银行都发行各自的五花八门的银行券。因为有些银行很小，这些银行发行的银行券钞票就只能在方圆几十公里范围内流通。因为出了这个范围，大家就不知道这家银行了，没人认了。

当时的美国存在不同的钞票，实际上是存在着一个非常复杂的汇率体系，各种不同的货币之间都要兑换。这些银行就把老百姓存进来的黄金对外发贷款，发的贷款也是银行券，通常发的贷款都是手里黄金储备的好几倍。那个时候没有中央银行，没有美联储，很多小银行贷款控制不好，结果就造成挤兑、倒闭、破产，一次次的金融危机。后来美国就组建了中央银行，也就是美联储来统一发行钞票，就是今天的美元，美国的货币才统一起来。这美元就是黄金的稳定币。

今天的稳定币会对现有的银行体系以及它的经营产生什么冲击？

冲击太大了。首先银行汇兑结算服务的费用全部归零，这个业务很可能就没了。因为大家现在都用稳定币支付，不需要那么复杂的账户结算体系了。数字钱包直接点对点就实现结算，比你快，比你成本还低，还安全。什么银联、VISA、通通就不用了。swift其次就是存款大搬家，传统银行管理的资产会大幅减少。假如Circle发行的稳定币规模越来越大，就会自然的吸引越来越多的存款跑到那里去。迟早有一天 的监管会被放松， 可以发Circle放贷款。Circle 立即摇身一变成一家数字银行，那么传统的银行形态就将被完全取代。

第三就是现在的传统银行，将会进行大规模的瘦身。要么物理网点大规模裁撤，人员大规模削减，凡是从事支付结算业务的人全部被砍掉……现在看，银行似乎还没有受到多少冲击，Circle被严格限制，存它那里的钱只能去买点短期国债，不能干别的。但其实从长远看是不可避免地要放松管制。到时是能够干别的，甚至发贷款的。Circle所以说现在的银行对稳定币是又爱又恨，就跟当年诺基亚对智能手机又爱又恨一样。你说是干还是不干？想发稳定币又担心存款流失，而且反洗钱监管不允许，因为钱转移到私人数字钱包里，转账的轨迹就看不见了，至少目前的反洗钱系统就抓瞎了。当然不排除未来出现新的技术创新，能够跟踪货币支付路径来实现反洗钱。

银行要是不发稳定币，就只能眼睁睁的看着像Tether、Circle这样的公司日渐做大，存款也哗哗的往他们那跑。所以说现在银行很焦虑。摩根银行早就开始试水，开始推出自己的稳定币了。但它
的稳定币是非常受限的，是被阉割的稳定币。它只能在自己的系统里运转，不能到社会上流通。他瞻前顾后，不知道该怎么办。

现在Tether、Circle 发展有多快？ Tether 的一年
的交易额达到五十多万亿，而Circle的一年交易额达到了二十多万亿。全世界每年的美元结算一共也就一千多万亿，这俩就这么几年已经占到了总结算额的4%左右了，未来肯定还会加速，为什么呢？现在监管合规了，大家以前担心稳定币的安全性，现在不用顾虑了，那它的使用和接受程度肯定会大大提高。

有人说稳定币大行其道，会不会冲击中央银行的货币政策传导，这个说法似是而非。实际上如果你把 Circle 完全看成是数字银行来管理，你 不要这样去限制它，那么它对货币政策的传导是没有任何障碍的。

有人还说美国积极推动美元稳定币，是为了维护巩固美元的霸权，这说法就太表面了。美元稳定币确实让美元支付更加便利高效，但这跟霸权不霸权八竿子打不着的事儿。

随着这样的私人稳定币发行公司越来越日益发展壮大，甚至有一天转变为加密货币时代的数字银行。那传统的大银行是会华丽转身，还是会像恐龙或者诺基亚那样倒下，谁也说不清，反正新势力正在不可阻挡的崛起。

让我们想象一下未来，有一天 Circle 会变成一家巨大的数字银行。他吸收人们的美元存款，然后发给你稳定币。你可以用来做交易，做支付，不论是跨境还是境内，跟微信聊天一样秒到账，手续费就接近于0。银行卡慢慢消失了，手机上的数字钱包成为你手上的银行。传统银行的分支机构大面积消失，只剩下一些财富管理职能或者贷款管理职能。

有一些开放的小国欢迎稳定币，它的本币有可能会大大减少甚至消失。有一些大国，比方说阿根廷，它实现美元化的进程就会更加轻松便捷……
