
# 2025年经济展望
# 猫哥原创

讲一讲我对2025年的预判。怎么说呢？对于2025年我的预判是下半年比上半年好，上半年可能比较难熬，特别是一季度。

## 为什么会有这个预判呢？

我们首先从最近的央妈与财爸的拉扯说起。12月下旬，中央开了两个高级别会议，一个ZZJ会议，一个中央经济工作会议，前一个会议明确提出未来的货币政策是适度宽松，中央经济工作会议也延续了这个说法，央行也拍着胸脯表示会“择机降准降息”，正常情况下，央行很快就会有具体的行动，但是这一次却很反常。

中央经济工作会议都过去半个多月了，现在都是1月11日了，央行的降准降息还只是干打雷不下雨，昨天的表态还是“择机降准降息”。

现在问题来了，央行为什么一次在等待“择机”而迟迟不采取具体的行动呢？更奇怪的事情还在后面。

1月4日—1月10日这一周，央行连续开展了逆回购操作，全周通过逆回购回笼资金高达2600亿，如果我们继续深挖竟然发现一个很不可思议的现象——这是央行连续4周开展逆回购的操作了，也就是说在两个中央高层会议之前，央行就开始逆回购回笼资金的操作，在两个高层会议之后依然持续在开展逆回购操作，连续四周央行回笼资金超过1万亿。

不仅是逆回购，而且昨天央行还明确宣布，暂停在公开市场买入国债。这里给大家解释一下，央行在市场上买入国债也是释放流动性的一个重要渠道，比如12月央行大致在市场上买入了超过2000亿的国债，相当于央行给债券市场释放了2000亿流动性，现在央行暂停购买国债意味着就是关掉了通过债券市场释放流动性的渠道。

现在问题来了，不是说货币政策要“适度宽松”吗？为什么央行却反其道而行之，一直在收紧银根呢？

央行在“适度宽松”的主题下反而不停收缩流动性是迫不得已。

现在的经济状况是，不管央行怎么宽松，大量资金实际是留在金融系统出不去，大量资金留在金融系统空转导致一个结果——资金蜂拥去买相对比较安全的债券（债券市场的逻辑参考我之前的文章）。

资金疯狂买入债券导致国债收益率节节下降，现在10年期国债收益率已经跌倒1.6%，30年期国债收益率已经跌破2%，就这样还有大量资金源源不断涌入继续购买。

国债收益率下降，导致中美国债收益率利差继续扩大，然后就有资金想尽一切办法出境去买美国国债，或者外贸出口结汇干脆就留在境外，这就给我们人民币汇率造成巨大的压力，现在人民币兑美元汇率不断贬值就是这个原因。

人民币汇率控不住不仅会给我们金融系统造成巨大的风险，同时也会对我们外贸进出口造成巨大的负面影响。

在这种情况下保汇率就是央行头等大事。

那么怎么保汇率？

首先就是央行要想办法将金融系统空转的资金回笼，免得它们无处可去跑到债券市场兴风作浪——这就是连续4周央行被迫通过逆回购回笼资金的原因。

其次，央行停止购买国债，免得对节节攀升的债券市场火上浇油。

以上就是中央定调货币政策要“适度宽松”，而央行却不得不反其道而行之去收紧银根的根本原因。

更耐人寻味的是，央行在宣布停止购买国债的时候，是这样解释原因的；“因为政府债券（国债）供不应求”。

这是啥意思？

国债是财政发行的，国债供不应求潜台词就是国债发行量太小了！这是央行公开向财政表达不满！

如果不是国债发行太少了，导致供不应求，我央行何苦在“适度宽松”大背景下做恶人，去搞银根收紧呢？

如果国债发行量足够大，那么不但可以将金融系统空转的资金调动出来，然后财政拿着国债募集的资金去搞建设也好，去补贴消费也罢，实际都可以让金融系统空转的资金释放到市场上，从而形成良性循环。

也就是说，现在情况是这样，如果财政层面不加大扩张的力度，央行不但不能执行“适度宽松”的真格，反而不得不搞银根收紧——否则汇率就保不住了。

所以，现在就是财政拖累了央行，同样也可以理解为拖累了经济复苏。

那么财政是怎么考虑的呢？

周五下午，财政部开了新闻发布会，在会上首先廖副部长特别强调，2025年财政是非常积极的，已经充分考虑到加大逆周期调节力度。

但是遇到具体问题廖副部长是这样回答记者提问的：

记者：2025年财政举债空间有多大?

答：较大;

记者：具体赤字率多少?

答：在走流程，不便告知。

这个发布会给市场是啥感觉？

网上有个段子是这样描述的：

你问他什么时候结婚，他说我是爱你的；

你问他彩礼多少，他说准备较大规模；

你问他见不见家长，他说在走流程。

简单总结一下，画饼很大，但是实质性信息一点不给。

假如你是女方，遇到这样的男方，是啥感觉？

所以财政的态度就是这样，或者是犹豫，或者是还在观望，反正兜里那点子弹抠抠索索就是不肯痛痛快快拿出来。

不要给我说什么财政赤字率需要人大批准，财政真要扩张赤字率人大那一次没批准？如果真下定决心，财政扩张的额度，赤字率也可以用我们计划提出XX指标来回答市场。

财政只画饼不给具体信息，市场怎么办？

所谓一年之计在于春，所有企业在做今年计划时都要根据大环境而定。

现在大环境就是，居民部门与企业部门中长期贷款都在剧烈收缩，大家都希望财政扩张填补这个居民与企业部门收缩的缺口，只有财政给市场一个定心丸，才有可能逆转居民与企业部门需求收缩的状况，但是财政偏偏就不给这个定心丸。

市场现在不缺钱，但是极度缺乏信心。

现在这个信心央行给不了，只有财政才能给。

财政偏偏现在就只画饼。

财政如果认为，靠着一句话“2025年财政是非常积极的，已经充分考虑到加大逆周期调节力度”——就能给市场定心丸是不可能的——毕竟画饼是不能充饥的。

现在大家理解央行为什么这么着急，连暂停购买债券的原因都毫不犹豫点明了是财政国债发行太少了！

所以，财政的定心丸的时间我们现在只能去等，等3月的两会，两会即便财政给出具体的扩张额度，要发挥效用也是几个月之后的事情了，这就是我不看好上半年，对下半年相对还能看高一线的根本原因。

房地产，上半年还要跌，如果两会财政给出有力度的措施，那么也要下半年最快四季度才有可能企稳。
