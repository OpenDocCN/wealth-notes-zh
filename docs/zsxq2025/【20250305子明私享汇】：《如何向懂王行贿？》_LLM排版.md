
# 如何向懂王行贿？

发布于 2025 年 3 月 5 日

老大，往往都是空架子，眼睛一睁，几百人的吃喝拉撒。任何一个利益集团的老大，都需要搞到足够多的钱和私账，才能够养起一支强大而精锐的亲军，成为政治斗争中最重要的底牌，所以，他们也都要经营自己的“江南制造局”，让很多利益集团通过建立共同的资产，形成紧密的利益绑定。类似于电视剧青瓷，很多买虚拟资产的行为，就是一张进入利益集团的门票，你哪怕明知道这些玩意不值钱，但是为了交换权力也需要装作他很值钱来买入，甚至公开的吹捧其真的值钱。很多大佬们捧明星，玩字画古董，甚至造假币，本质上都是在制造这种金钱与权力之间的旋转门，很多被锁定片约的顶流明星，签约买断作品的顶流画家，本质上都是在为权贵们洗钱和做交易。

入股蚂蚁的各大家族，就是买了之江系的门票，这是马云能够逆风走通蚂蚁所有上市手续的根本，同样，也是他遭受致命打击的根源。如今，特朗普无论是搞金卡还是虚拟币，本质上就是制造一个懂王完全可控的“青瓷会所”，搞一个美国版本的蚂蚁金服，上到普京、小萨勒曼等有权有钱的真大佬，下到马云、张一鸣、孙正义、 CZ 等有钱无权的企业家，都可以通过购买这类资产来向懂王行贿，把之前的秘密交易，甚至国会山的交易，搬到海湖庄园的官邸。就像之前把蚂蚁的场子砸了一样，懂王去砸国家开发署等部门，就是去砸别人的青瓷会所，只有把别人的场子都砸了，大家才会一起回到懂王的场子里面做交易，当各利益集团都是用懂王币来做交易，那么形成的利益共同体就会坚定的站在他这一边，为他提供安全保障，这种牌对于大佬们来说是非常明的牌，所以他的敌对力量也会在他搞起来之前不惜代价的进行狙击。
