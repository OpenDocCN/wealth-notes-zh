
# 女人必须学会隐藏性别优势
原创：U 君 公众号：赢在8小时之外 25年8月16日

男人和女人分别有自己的性别优势，但是，真正厉害的人一定懂得隐藏自己的性别优势。

男人的性别优势是什么?力气大，胆子大，敢于冒险，忧国忧民，有家国情怀。

但是，男人一旦显露出自己的性别优势，那么他一定还停留在底层。

有的男人喜欢把“百善孝为先”“若有战召必回”这种口号作为自己的签名，是不是感觉非常有情有义、英勇神武?

但真相是，这样的“热血男儿”基本都是社会最底层，月薪大概率不会超过5000元。

他们到了40岁之后，大概率会穷困潦倒，因为他们始终看不透事物的本质。

而上层的男人是怎么做的?

他们会给孩子弄一个海外身份，让他们以后有更多、更丰富的选择。

他们自己也可能会把一部分资产放到海外。

还有的男人喜欢转发“正能量”的内容，比如“某地的环卫工人月薪2000元，38度的天气下还要穿着长袖扫大街”，然后他就呼吁：

“请提高环卫工人的待遇，他们是这个城市的净化者 !”

但是，如果你让他给环卫工人捐几百元钱、提高这些人的生活质量，他立刻就不做声了

而上层的男人面对同样的“正能量新闻”，会作何感想?

他会想办法搬离文明洼地，然后确保自己和自己的后代不会沦落到扫大街的地步。

最后你会发现，那些特别有“男子汉气概”的男人，大概率都是底层的牛马，每天高谈阔论、胸怀天下。

但是，到了需要“自己献身”或者“拿出真金白银”的关键时刻，他们就都蔫了，比太监都萎。

这样的男人，将会长期停留在社会底层，他们吸引到的也都是底层的女人。

因为这些底层女人也没什么智慧，她们会被底层男人的“英雄气概”和“家国情怀”所吸引、所哄骗，最终一起穷下去。

说完男人，下面我再说说女人，由于会涉及到“性”相关的内容，所以我必须设置成付费阅读。

“隐藏性别优势”的功效在女人身上比男人强100倍。

主要有下面5个原因：

1. 资源不会向弱者聚拢

如果你是个女人，而且很享受“好男不跟女斗”“男人要让着女人”“家里要女人管钱”这些特权……

那么恭喜你，你将长期停留在社会底层，任何好的资源都不会向你靠近。

因为除了“权力特权”之外，其他的大部分特权都是弱者的专属。

比如老、弱、病、残、孕……这些人走到哪里都会享受到一定的特权。

你愿意把自己划分到“弱者”那一栏，确实不会有人跟你竞争，但是也没人愿意跟你合作，因为你在强者眼里就是个累赘，是个拖油瓶。

为什么在底层是女权社会，在顶层却是男权社会?

因为在底层，女性完全可以通过“我弱我有理”来控制和打压底层男性。

但是到了顶层就不一样了，竞争非常激烈、非常残酷，没有人会给你特权，大家只看结果。

你强，就有人愿意跟你玩、愿意跟你合作，你弱，大家就全部弃你如敝履。

所以，如果你是一个女人，你不想停留在底层，就把自己的女性特权隐藏起来，修炼自己的真本事。

2. 性吸引力很廉价

弗洛伊德说过：一切欲望的本质都是性欲。

所以，这个社会上存在很多跟女人相关的交易，本质上都有“性交易”的影子。

举个例子，有些长得漂亮的女销售员业绩非常好，本质上也是男人在跟她沟通的时候，满足了自己一部分性欲。

(其实性别互换也是同理，长相帅气的男销售员也适合去推销针对女性群体的商品)。

但是，这种类型的交易不会太长久，而且也很难走到一个很高的层级。

一方面是因为，你的颜值和身材会随着年龄的增长而衰退;

另一方面是因为，靠“性吸引力”你也只能卖点廉价商品。

之前就有这样一条新闻：保时捷女销售一年卖出去170台保时捷汽车。

然后很多人在下面调侃说：这个女人肯定是靠陪客户睡觉才取得的这种业绩。

我想说，这些都是穷人的意淫罢了。

如果真的靠陪客户睡觉才能成交，那么不用说170台车，就连17台她都卖不出去。

一方面是因为，能买得起保时捷汽车(均价150万元)的男人，压根就不缺性资源，早就过了“来者不拒”这个阶段。

另一方面是因为，在购买这种大额商品时，客户更看重你的专业性和用心程度。

你如果真暗示客户“下单我就陪你睡觉”，那么客户反而会起疑心和戒心，怀疑你要破坏自己的家庭，甚至是对手派来搞自己的。

3. 骚的门槛是最低的

如果你看过时装秀就会发现，真正高级的女模特，绝对不是那种前凸后翘、涂脂抹粉、穿着很“女性化”的女人。

因为她们非常清楚，“骚”只是一种非常低级的特征，不值钱。

或者说，骚的门槛是最低的。

如果发骚有用的话，那么这个世界上最富有的群体应该是妓女，但矛盾的是，大部分妓女都生活在底层。

结论就是：骚只对底层的穷男人起作用，因为底层男人性资源匮乏，普遍处于性饥渴状态。

但是到了上层，骚不仅不会起作用，还会起反作用，因为上层的强者会认为你是一个没有真本事的人，靠不住。

所以，如果你想征服优质男人，一定要记住：可以有风情，但是绝对不能骚，可以纯情，但是绝对不能妩媚。

在“那件事”上也是如此，你可以说一些让男人血脉喷张的话，也可以穿一些让男人欲罢不能的内衣。

但是，你一定要把握好尺度，如果你完全“放开了”，那么男人当时可能会很兴奋，但是事后会觉得你很廉价。

4. 卖惨是下下策

为什么做直销、保险、微商的女人特别不受待见?

原因就是：她们过度使用了自己的性别优势，里面还夹杂着一些“骄傲”和“卖惨”的成分在里面。

比如，很多做这些行业的女性，特别喜欢立“独立女性”或者“单亲妈妈带着孩子创业”这些人设。

但是呢，你所立的这些人设，跟专业性没有半毛钱关系。

顾客只想买到物美价廉、专业的产品或者服务，他们不会在意你到底是已婚还是离异，男人还是女人。

而且，就像我在第一条中说的那样：你过度使用自己的性别优势(特权)，只会让大家鄙视你，而不是尊重你。

所以，无论在销售领域，还是在职场上，女人想要获得尊重，最有效的方式就是隐藏自己的性别优势，专注于自己的能力和专业。

讽刺的是，你越隐藏自己的性别优势，男人就越信任你、心疼你。

换句话说，“隐藏性别优势”反而为你最大的“性别优势”。

举个例子，如果一个年轻的女人在马路边上卖炒饭，路过的男人都会心疼她，认为她很不容易，进而支持她、支援她。

但是，如果她在摊位前面拉了一个条幅，上面写着“单身宝妈独立创业”，那么路过的男人大概率会非常反感这种卖惨和立人设的行为。

在微信上做生意也是如此，只要你足够专业、足够努力，你不需要刻意宣传自己是个追求独立的女人，男人自然全都看在眼里。

当然，我说的是常规生意。

如果你卖的产品或者服务专门是针对女性群体的，那你可以立“独立女性”的人设。

5. 就事论事

在赚钱这件事上，我们经常能看到几个男人坐在一起吃饭、喝酒、谈事情，但是，几个女人坐在一起筹划事情的现象就很少见。

为什么会这样?

原因就是，男人的合作意识比较强，即使彼此之间不熟悉，甚至有矛盾，但是未来合作，可以求同存异。

但是几个女人坐在一起聊天时，很可能会把重点放到事情之外。

比如对方的穿衣打扮，感情状态等等，而且女人之间的竞争意识强过合作意识。

所以，如果女人想赚到实质性的钱，必须要有合作意识，甚至淡化自己的性别，就事论事。

比如，在跟男性合作、谈判时，不要穿非常女性化衣服和妆容，在对话的时候，不要以弱者自居，也不要聊八卦内容。

总之，重点放在事情的推进上面。

另外，如果你是个创业者，不论男女，一定要多读《孙子兵法》和《三十六计》，把格局打开。

很多时候需要牺牲自己的短期利益，这样才能有长远的未来。

举个例子，我之前说过一个分钱模式：如果有人给我介绍客户，我会把销售额的5%-10%分给介绍人，每个月都给。

对于这种分钱模式，男性创业者的接受程度就远远高于女性创业者。

因为他们知道，一旦介绍人可以源源不断地拿到好处，那么他就会把这份收入看成是自己被动现金流的一部分。

为了获得更多的被动现金流，他会玩命地为我介绍更多、更优质的客户。
