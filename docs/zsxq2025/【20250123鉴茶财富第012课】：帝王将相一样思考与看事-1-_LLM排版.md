
# 帝王将相一样思考与看事

在古代中国乃至全世界，只有帝王将相的思考方式是全面的，多维的，相对客观的。为什么？他们是统治阶级，他们得独立思考、全面兼顾、是一个正常人。所以有人说《二十四史》就是帝王将相的家事。不然说什么呢？说普通人吗？你知道治国之道最重要的是什么吗？

商鞅说，治国之道，务在弱民。你觉得普通人，他让你有独立思考能力吗？他会让你知道游戏规则怎么玩吗？都是一堆聪明人，还怎么管理呢？这个就不展开了。“百姓”和“平民”其实是两个不同的概念，从秦代以后，都是黔首。不完整的思维方式，研究起来有意义吗？

我们并不是要学帝王将相的骄奢淫逸，而且学习他们的思考模式，关于这门学问我们搞了2000多年，其实它可以代称：全面和独立的思考方式。

鲁迅说，中国旧社会吃人，虽然夸张，但是不搞会这套思维模式，分分钟被人搞成渣。皇帝是什么？利益的集合体，多少人盯着，除了皇宫哪里也不能去，光靠看奏折就得决定天下大事，那一定得是最顶尖的思考方式。

那么这套思维方式是什么呢？

第一，归集同类事物，得出统一结论。为什么说“读史使人明智”,因为太阳底下没有新鲜事，历史总在不断的重复，一个皇帝面临的新问题，前朝或者更前朝应该是面对或者解决过的，两相一套，以前这个类似的事情，他背后的真正原因是什么？解决它有什么成败得失呢？把皇帝代入这个环境，自己应该怎么解决也？

这其实就是今天MBA课程的案例教学法、情景教学法。

第二，从常识和常理的角度入手，仔细分析每事一情的个真人正所原处因 -- 的或环境意、、图背。景，一得个出帝王一将相那，个看不出别人的真正意图，是要用生命做代价的。

我们很多人小时候都学过的那个砸缸的司马光，其实是北宋的大政治家和史学家，他写了本《资治通鉴》,从春秋写到当时，1362年，意思是“鉴于往事，资以治道”,是一本皇帝的标准教科书。

300多万字，把每一个历史人物当时讲什么话，为什么讲那句话，写的淋漓尽致。理所当然，这本书，很快成了禁书，只有皇帝和太子能看，而且有专门的御批传世。

里面有一个案例，说齐王考察干部，问即墨大夫，类似今天的青岛市长怎么样，问自己身边的秘书、随从、宫人。大家都说，青岛不行，被他搞得乱糟糟的。齐王把青岛市长叫上来问，市长说，你身边的秘书，本来就跟青岛没什么关系，人民的死活他关心个毛啊，无非就是我没给他们好处。我再干三年你看，三年后，齐王又问身边的人，青岛市长怎么样，大家都说，好得不得了。市长说，三年前我关注民生，大家都说我不好，三年后我不管民生，净讨好你身边的人，大家都说我好，你要哪种呢？

司马光通过无数这种案例，告诉皇帝，任何人，他说什么,做什么,都是屁股决定脑袋的，你不能光听他们嘴上讲的好听，越好听越大的道理你越得警惕。你要详细的分析，那个人做那个事，希望得到什么,大家伙那么评价他，又是出于什么样的诉求和占位呢？

司马光一再在书里通过各种案例告诉皇帝，他的反对真是反对吗？一个人做了反常的事情，真的因为傻吗？

不找出别人做那个事情做大的合理性，那你就是最大的傻瓜，迟早被玩死。

里面有我们都熟悉的案例，楚汉争霸，项羽失败，很多人都认为两大原因，一个是不重用军事天才韩信，最后四面楚歌被韩信打败。但实际呢？司马光给皇帝分析说，项羽是靠江苏起家，别人跟他打仗都是图富贵的，他跑去西安，既得利益集团怎么办？老家的房产和土地不升值，谁还跟呢？项羽要衣锦还乡，难道只是炫耀和胸无大志这么简单吗？

而且韩信草根出身，贸然弄一个高位，项羽原来的军事班底龙且、项庄弄不到一起，到底是增强战斗力呢，还是削弱战斗力呢？

项羽的做法傻吗？当时那个环境，他只能做出这种决定。而项羽为什么失败呢？

司马光分析说，一个原因是项羽是妇人之仁，轻信了刘邦的话，但凡他有作乱的巨大能力，他就一定会那么干，刘邦从来就不是一个人，而是一个利益集团，你相信他一个人的话，管用吗？所以后来朱元璋大杀开国军功贵族，甚至硬套帽子去杀，你觉得洪武皇帝有病吗？说远了，拉回来。

另一个原因，大家都说项羽是一个失败的管理者，是因为不会笼络人心，司马光认为，项羽的分封制，不可能公平，再公平会有人认为不公平。

司马光认为，项羽不是不懂郡县制，不想搞天下一统，而是想以力服人，但其实力量根本不够，力量不够就得玩手腕，或者说搞阴的，你偏偏妇人之仁，难道这不是死于教条，死于纠结吗？

所以你看看，人家都是这么去理解世界，这么去思考，这么去教皇帝的，人家的游戏规则第一条是：世界上没有无聊的人，也没有傻子。

这才是合理的思维方式，如果别人做了一件事情，你以为很可笑，很哗众取宠，很不合逻辑，而你迟迟找不出他做这个事的合理性，找不到背后真正的利益动机所在，甚至被带了节奏，那才真正危险。

西方有句谚语，大意是，如果大家在一个牌桌上玩钱，你迟迟找不出谁是傻瓜，那么你就是那个最大的傻瓜。

这套东西从来都是不传之秘和禁书啊，别人凭什么教你呢？在中国，你卖个炸油条都得搞定城管，真认为全世界有纯粹的就经济论经济吗？
