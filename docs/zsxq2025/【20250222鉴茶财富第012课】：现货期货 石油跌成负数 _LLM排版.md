
# 现货期货 石油跌成负数

市场上流通的货物分两种，一种是现货，一种是期货。现货通常是指马上能交易买卖的实物商品，买卖方在极短的期限能完成付款交货，所以也称实物。另外一种，马上生产不出来，但是又特别大宗的，就是现货的对称：期货。

期货与现货完全不同，现货是看得见摸得着的商品，期货主要不是货，而是以某种大宗产品，如棉花、铁矿石、大豆、石油等及金融资产，如股票、债券、国债等为标的标准化可交易合约。

期货既然是期，大家会约定一个期，比如三个月，六个月，一年等，到期如数交割。那么是不是所有未来较长一段时间交付的都

是期货呢？不是，必须是大规模、大宗、非定制化的商品才能做期货，比如说，你订架波音飞机，3年后交货，那就不是期货。飞机不是大豆小麦等庄稼，一段时间之内，他加班加点也就造那么几架。期货的服务对象，都是有大规模弹性产能的企业。比如农场主，他既可以种10万亩小麦，也可以种1万亩小麦。

1万亩-10万亩这么大的产能弹性空间，就需要期货来指导了。特别大型的农产品、矿产、原油企业等，如果他不知道未来的需求和预期，胡乱生产多了，烂在地里亏钱，生产少了，有商机赚不到钱。那么以销定产就是最好的选择，未来你愿意买，我现在才愿意种，不然买卖双方两眼都是一抹黑，都是亏的。

比如未来12个月之内，有个榨油厂在期货市场愿意以3000元/吨买入10万吨大豆的合约，大豆农场主一看可以，那我多雇人手种吧，如果有人只愿意买1万吨，那么我就少种点。

等到大豆成熟，期货合约到期，榨油厂准备好仓库，到农场的仓库提大豆，或者大家在约定的集散地交割实物，双方平仓，这是最单纯的循环，市场上其实发生的不多。

更多的是通过期货是为了发现远期价格和保值。比如说，铁矿石4月的价格是5000元/吨，钢厂9月份要用到10万吨铁矿石，担心9月份的价格要涨，于是，钢厂4月份时，在期货市场买进10万吨的铁矿石期货，价格当然是5000/吨。到了9月，铁矿石果然涨了，变成了5500/吨，一吨涨了500块。这时，钢厂要炼铁，从现货市场5500元/吨买进10万铁矿石，同时，手头原来持有的10万吨期货合约也平仓卖掉。钢厂买现货，一吨多付了500块，但期货市场卖了合约，一吨也赚了500,维持了自己不受大的波动，这就是套期保值。

像航空公司套期燃油，食品企业套期小麦等等都是套期。

当然，期货的杠杆也是远超其他金融产品的，不在本文的探讨范围之内。虽然炒期货的很多，但金融是必须服务实体经济的，到了最后，实物是要交割的，合约到了最后一天，钱货要两讫的。

于是就发生了今天的原油期货暴跌，-37.6美元/桶，也就是说，你要我把原油拉走，你得付我37美元。什么原因呢？这是5月份要实物交割的原油，油轮停在港口，你得有储油罐接收啊。不然船能白等你吗，违约也是要赔钱的。

而现实呢？因为疫情影响，美国飞机趴窝，都不开车，成品油消耗太少，炼油厂不开工，原油油库都是满的，你怎么办？你只能花钱求人家把油拉走了。所以37.6美元，体现的是仓储成本。你不花钱求人家拉走行吗？不行，石油这种东西，你往哪里倒呢？这可不是一般垃圾，你送人也没办法送，原趟再拉回去那更是开玩笑，运费比油都贵，而且拉回去干啥呢？一样没油库。

因此，-37.6的价格只是期货，不是现货，体现的是5月份要交割的近期原油合约，而6月份要交割的原油合约，又涨成20多一桶了。

而期货指导现货，既然期货这样了，现货肯定是跌的。期货的价格可能跌成负数，但大家放心，商品的价格最低只能跌成0，别人花钱求人你把油加满再走，这种好事我也想啊。

以前大家在初中课本中学到的，资本家宁肯把牛奶倒了也不卖，也是这个道理。一般的消费商品，价格最低只能跌成0，你问他为什么不干脆白送给广大老百姓？他有能力通知到广大消费者而且能让你来提货，这不得花吗，你以为广泛通知客户的成本比造牛奶低吗？有这能力就是做生意了，就不会倒奶了。

所以没有什么大惊小怪的，-37.6美元的期货合约，这就是一个货到地头死的案例罢了。
