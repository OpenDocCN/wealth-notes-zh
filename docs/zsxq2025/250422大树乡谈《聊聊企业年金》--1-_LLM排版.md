
# 聊聊企业年金

## 养老三大支柱

养老有三大支柱：

1. **第一支柱：社保**
   - 城镇职工社保（2024年完成体制内外并轨）
   - 城乡居民社保（2016年完成城乡合并）
   - 国家兜底的城乡居民基本养老保险
   - 统称为“五险一金”中的社保部分

2. **第二支柱：年金**
   - 职业年金（针对机关事业单位）
   - 企业年金（针对各类公司，包括央国企）
   - 缴纳方式：企业和员工按比例缴纳，用于补充养老

3. **第三支柱：个人养老金**
   - 建议选择稳健的R1风险投资项目，如保本保息或存款项目

目前最全面的社会保障是“六险两金”，即在传统“五险一金”基础上增加补充医疗保险和年金。

## 职业年金的由来和问题

职业年金是为弥补2014年双轨制并轨后机关事业单位人员养老金替代率下降而推出的补充养老保障。

- **缴费比例**：单位8%、个人4%，统一缴纳到省级账户
- **现存问题**：
  - 财政全额供款的单位采用记账方式（类似“视同缴纳”），非财政全额拨款的采用实账，形成虚实账并存
  - 记账模式下资金无法参与投资积累，未来财务负担大
  - 离开机关单位后，虚账模式下资金难以划转至个人账户

## 企业年金

企业年金始于2004年，发展缓慢：

- **规模**：截至2024年底，参与职工3241.84万人，规模超过3万亿元
- **参与情况**：央国企和外企为主，民企罕见，互联网大厂如华为、联想等仅有部分企业设有年金
- **制度问题**：
  - 暗箱操作：企业缴费分配不透明，高管可能获得大部分
  - 门槛高、流程繁琐：跨地区经营的企业需与多地人力资源部门打交道

## 未来改革方向

### 职业年金改革

- 解决虚账问题，否则未来财务压力巨大
- 调整编制限制，解决内部不公平问题
- 缩小职业年金与企业年金差距，避免形成新的双轨制

### 企业年金改革

- 降低门槛，简化流程，提高透明度
- 考虑将城镇职工养老保险、企业年金、职业年金、住房公积金整合

## 改革建议

- **整合改革**：将三大资金合并运营，降低缴费率
- **调整比例**：职工养老保险降费率，企业年金、职业年金、住房公积金合并后整体降低缴费率
- **取消住房公积金**：在商业房贷利率与公积金利率相近、人均住房面积已较高的情况下，住房公积金必要性下降
- **调整缴费比例**：将职工养老金单位16%、个人8%降至各8%，或将三金整合后单位、个人各5%
