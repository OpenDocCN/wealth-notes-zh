
# 钇—90量产破垄断，中国核医疗的里程碑时刻

在中东局势扣人心弦期间，中国核医疗产业悄悄的迎来了里程碑时刻。6月中旬，中核集团传出捷报：经过3年技术攻关，经秦山核电“和福一号”辐照过后的钇—90玻璃微球顺利出堆并通过相关检测，标志着我国成功掌握商用堆规模化生产医用钇—90的技术，彻底终结了该领域长期被欧美垄断的历史。

## 钇—90玻璃微球是什么？

简单说，它主要是以钇—89为原料，在核反应堆中吸收中子之后变成一种能释放β射线的放射性同位素药物，然后通过血管介入手段，将它输送到肝脏动脉，对肝部的肿瘤病灶进行精确治疗的微创技术。

## 中国肝癌的挑战

在我国，肝癌主要呈现3个特征：首先，高发；其次，不易被发现；第三，药少。因此，长期以来，我国肝癌患者整体5年生存率不高。

## 钇—90的优势

相比传统治疗方式，钇—90的优势主要有3点：

1. 不需要进行开腹手术，钇—90玻璃微球通过股动脉穿刺局麻即可，通常在门诊1—2个小时即可完成，不需要住院。
2. 释放的β射线穿透距离仅2.5毫米，不仅在肿瘤内部形成高剂量辐射区，同时尽可能减少辐射对肿瘤外正常组织的影响。钇—90半衰期相对较短，治疗效果更快，也不会长时间影响患者生活质量。
3. 治疗效率高。在患者接受钇—90玻璃微球治疗的12天内，释放的辐射95%会被吸收到包含肿瘤区域。术后肿瘤客观缓解率达40%—70%，中位生存期延长至16个月。

## 钇—90的突破

钇—90也并不是什么新鲜东西，这项技术在国外已经有20多年的临床经验了，2016年海南开始尝试引进这项技术，2021年最终落地成功实施第一例树脂微球肝癌手术。

## 钇—90国产化的意义

钇—90的国产量产，将引发万亿级全球核医疗产业链的重构。最直接的受益者就是国内肝癌患者，单次治疗费用预计逐步将从几十万降至10万以内。

## 秦山核电站的贡献

秦山第三核电厂作为中国仅有的一座商用重水堆核电站，拥有国内其他核电站难以复制的优势，是目前我国唯一的医用同位素生产基地。

## 未来展望

在2027年，8种同位素将全面实现国产替代，这不仅仅只是一个技术突破，是中国抢夺世界高端医用同位素量产的里程碑。

## 结语

钇—90的量产突破，我们除了感谢协同奋战的十几家科研单位和技术人员外，不得不提的就是位于浙江省嘉兴市海盐县秦山核电基地的秦山第三核电厂。这个之前籍籍无名的核电厂，现在堪称大国重器。

最后，大家还是理性看待这项技术，也不要去过度神话其功效，认为钇—90就能“一针治愈”，毕竟，降费缓解、提高生存时间和质量与彻底治愈是两回事。对于患者而言，低费用、高质量的多活一天，就是福音，具体治疗措施还是需要遵循医嘱进行。
