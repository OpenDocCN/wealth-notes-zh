
# 汹涌深沉的黄河水——李大钊

大家好，我是杨文哲，欢迎收听卢克文工作室的党史栏目。

“铁肩担道义，妙手著文章。”我党历史上另一位开篇人物李大钊，字守常，河北人。河北，古时的燕赵之地。

《觉醒年代》第一集中，陈独秀在日本初识李大钊时就说：“燕赵多慷慨悲歌之士，认死理。但现在中国，就缺认死理的真汉子。” 不知道真实历史中的陈独秀有没有说过这句话，但在剧中的时空里，算是一语成谶。

因为守常先生在38岁那年便慷慨就义，化作悲歌一曲。而守常先生这位被后人称作“革命完人”的人的确是一条响当当的真汉子。

李大钊一生没有污点，没有绯闻，甚至同样是包办婚姻，他和比他大六岁的几乎是以童养媳身份嫁过来的妻子赵纫兰也是恩爱有加，幸福美满。

这在当时的封建包办婚姻中是十分罕见的。这点在《觉醒年代》里也有着细腻的演绎。若论受教育程度，李大钊与赵纫兰和陈独秀与其原配妻子高晓岚倒是很像，一方是满腹经纶的大学教授，另一方是目不识丁的家庭主妇。

但结局实在是相差太大。陈独秀原配是文盲与自己没话说于是后来便劈腿小姨子高君曼，但到最后和小姨子也还是劳燕分飞。

李大钊却从来没嫌弃过自己的老婆是个文盲，相反，他非常爱他的家，和妻子一生中一共生育了9个子女，其中4个夭折，只有5个孩子保了下来。

他的妻子赵纫兰也非常信任他的丈夫，虽然自己没有文化，不理解丈夫工作的具体内容，但是她明白她的丈夫一直在做正确的事，而且是正确的大事，是能改天换地的大好事。

她也一直这样教育自己的孩子。所以即便李大钊从日本回到老家大黑坨村时，自己的孩子一时间都认不出自己，也丝毫没有影响李大钊与子女之间的感情。

或许是因为赵纫兰和李大钊从小青梅竹马吧，因为李大钊从小父母双亡，由祖父拉扯大，到他10岁的时候，祖父年事已高，就给李大钊娶了赵纫兰来照顾他。

婚后，赵纫兰既是李大钊的妻子，也是李大钊的长姐。打理家务，想尽各种办法攒钱供李大钊上学。她为李大钊付出的一切，李大钊都记在心里，自己也争气，书读得好，回国后在北大任教，生活也有了保障。

这个时候的李大钊气度不凡又多金，在北大也拥有一堆迷弟迷妹，但李大钊从未有过任何别的想法。而赵纫兰常年操劳家务面容憔悴，文盲又小脚，经常被误认为保姆。

家里来客人她怕给丈夫丢人，就偷偷躲起来。但李大钊会帮她整理衣服，扣好扣子后大方介绍给客人。为了让妻子住得舒服，他在家里盘了炕。这个细节在《觉醒年代》里也有体现。

有客人来，李大钊他们就在书房开会，赵纫兰就坐在炕上缝补，时不时给他们添茶水。闲时，李大钊就教妻子读书认字，能写简单信件。这简直是绝世好男人啊!仅从李大钊对他糟糠之妻的这种感恩和关怀，就可见守常先生的品德和人格。

我想可能正是因为李大钊能有这么一个稳定踏实的大后方，他才可以潜心地研究马克思主义，并将革命的火种传播出去。

也正是因为有这么一个幸福美满有爱有温度的大后方，守常先生才会让人感觉是个可以依靠的深沉暖男。

以至于北京市民司马南先生都形容他有“静穆”之气。党史专家胡澄也形容李大钊是个朴实雄健的革命完人。

李大钊不像陈独秀干什么事都特别秀，特耀眼，特吸睛。仲甫先生去了北大分分钟就成了学生们的心中偶像，成了保守派的众矢之的；

李大钊很沉稳， 一方面投身新文化运动，协助仲甫先生办报办杂志，写文章论述新文化的思想，另一方面潜心研究马克思主义。

其实之前他在日本早稻田大学留学期间，就已经接触到了马克思主义的著作了。1917年俄国十月革命之后，他发现马克思主义居然率先在俄国成功地建立了社会主义国家政权。

而不是书上说的率先在发达的资本主义国家建立政权，于是他开始深入思考俄国革命给中国革命可能的启发，他开始有种感觉，如果社会主义革命在俄国都能取得成功，那么在中国会不会取得成功呢?

紧接着一战胜利，守常先生在次年1918便发表了著名的《庶民的胜利》和《布尔什维主义的胜利》，算是我国最早的马列主义文献。

1919年五四运动后，几乎全中国的有识之士都意识到当时的中国虽然名义上共和了，但实际上没有改变半殖民地封建社会的根本面貌。

北洋政府并不能完成反帝反封建的历史使命，西方的民主共和制似乎并不能解决中国的问题。就是在此时，李大钊在《新青年》上发表了著名的《我的马克思主义观》，详细阐述了自己对马克思主义的哲学、经济学的理解。

实际上也是一篇高度凝练的对马克思主义哲学、唯物史观、政治经济学的总结，马克思主义得到了更为广泛的传播。而此时期，李大钊还未满30周岁。

1920年9月，李大钊成立了北京共产主义小组，之后积极发展工人运动，理论联系实际，传播马克思主义，还在长辛店开办劳动补习学校，亲自给工人讲学，宣传革命思想。

这恐怕是最早的深入基层的群众工作，就是要面对面地交谈，让工人们意识到这所谓的命运是不可以改变的，要让工人意识到自己不应该受到压迫，也就是意识到阶级矛盾，具备阶级斗争的基本意识。

可以看到大钊先生的路子一直很稳，先潜心做理论研究，然后开始实践，每一步都走得很踏实，而且他尽管当时在北大做教授，一个月几百块大洋，放现在妥妥的高薪。

但是他的家庭生活一直过得很简朴，甚是有些拮据。因为他把大部分钱都拿出来支持革命运动以及帮助工人等受压迫的穷苦群众了。

我们在《觉醒年代》里也能看到，大钊先生可以和那些衣衫褴褛的穷苦工人很自然地打成一片，一起做饭，一起吃饭，一起有说有笑。这点是陈独秀做不到的。

陈独秀自始至终也未能摆脱自己的资产阶级精英身份的局限性，他正直、善良、爱国、同情劳苦大众，但是同情并不意味着就能和劳苦大众同甘共苦。

但是李大钊就是农民出身，他身上没有那种精英气息，所以才能很自然地和基层的人民群众打成一片。他又很幸运地接受过良好的教育，两相结合，便共同养成了他朴实、沉静的人格特征。

再之后就是著名的南陈北李相约建党，公开地搞革命政治运动。中国共产党正式成立以后，李大钊主要代表党中央指导北方的工作，在北方广大地区宣传马克思主义，开展工人运动，建立党的组织。

1922年至1924年，他受党的委托，奔走于北京、上海、广州之间，帮助孙中山改组国民党，为建立第一次国共合作的统一战线做出重大贡献。

李大钊还领导北方党组织配合五卅运动，配合北伐胜利进军，开展反帝反军阀斗争，为大革命胜利推进做出卓越贡献。可以说，在党组织还很弱小的年代，守常先生以一己之力，撑起了北方的半壁江山。

1924年国民党一大确立了国共合作的总方针，以反帝反封建为目标的大革命开始。此时期，李大钊还充分展示了其韬略，这点经常被后人忽视，李大钊实际上也是个纵横捭阖的猛人。

具体的过程十分复杂，简言之，他洞悉了当时中国北方各派系军阀的利益诉求，以及刚刚成立的在世界上并不讨喜的苏俄对中国的态度。

充分利用自己在北京大学的职位，在错综复杂的关系网中周旋，差一点就能联合冯玉祥等势力，干掉一心想当皇帝的张作霖，为中国共产党获取起码东北两省的地盘作为根据地，直接从苏联取得援助。

如果真做成了，可能就没后面南京政府和蒋光头什么事了。只可惜，在计划正势如破竹地进行时，日本人为了自己在东北的利益，联合张作霖联合绞杀了北方革命。

1927年4月6日，张作霖派兵悍然闯入苏联大使馆逮捕了李大钊，同时被捕的还有李大钊的妻子和儿女。

李大钊毕竟社会影响力巨大，连当时最反动的报纸，也不得不对守常先生的学问和人格表示钦佩。张作霖考虑舆论影响，也不敢轻易下手。后来经过多方营救，李大钊的妻子和子女平安获释。

我看了一些考据，张作霖当时发电报给其他军阀问处理意见，基本回复是处死，但并没有足够的证据说明张作霖曾发电报给蒋介石问如何处理李大钊，而蒋介石的回电是立即处决。

主要是因为，当时北伐已经开始，北伐的目标正是要推翻帝国主义利益的代理人北洋政府，而蒋介石当时正是北伐军的总司令，实际上是张作霖的敌人。

而李大钊又是国共合作的代表人物和北方地区领导人，张作霖完全可以把李大钊作为政治筹码，牵制北伐，既然如此，他为什么要问自己的敌人如何处理能牵制敌人的筹码?

说不通。但是李大钊被捕仅仅6天，蒋介石便在上海发动了四一二反革命政变，大肆屠杀共产党人和国民党内进步的左派人士，充分暴露了蒋介石反动的本质。

于是在反动这一点上，瞬间和和张作霖一致了。也许正是因为如此，张作霖发现他的敌人居然秒变成和自己一样的坏蛋，心里一阵暗爽，一哆嗦，就发个电报试探他，“光头你好，原来你和我一样是个坏蛋，我抓了个好人，你觉得怎么处理呀?”

两个坏蛋还能怎么处置好人?肯定是把好人弄死。所以，无论张作霖有没有发电报问蒋介石，以这些人的反动立场，也一定会处死李大钊这个北方地区共产党的骨干人物。

所以最终，张作霖还是决定处决李大钊等一众革命者。1927年4月28日，李大钊在京师看守所被处以绞刑，慷慨就义，时年38岁，化作悲歌一曲。

和日本人勾结的张作霖，也很快遭到了报应。一年后，1928年6月4日，皇姑屯事件，张作霖被日本人炸死。

纵观中国共产主义运动的先驱，伟大的马克思主义者守常先生的一生，我们会发现，他并不是一个很情绪化的人，他所做的贡献也没有那么符号化，像陈独秀那样，新文化运动、五四运动、建党，接连的高光时刻。

李大钊的贡献更多地是踏实的、稳健的、润物细无声的一种深刻的影响。要非要有什么标志性的东西，恐怕就是五四运动时期，他所做的那个著名的预言：“试看将来的环球，必是赤旗的世界!”

可以说，是李大钊首先指明了历史发展的新方向，即社会主义方向，也率先揭示出，真正要解决中国的救亡与复兴问题，必须学习俄国的十月革命。

李大钊如同中国的普罗米修斯，从俄国取来革命的火种，播撒在中国的土地上。最后这星星之火，终成燎原之势。

他是一个世纪以来中国先进青年的精神导师，一代革命青年的指路人。毛主席曾在延安深情地对斯诺说过：“我在李大钊手下担任国立北京大学图书馆助理员的时候，曾经迅速地朝着马克思主义的方向发展。”并说李大钊是他真正的老师。

守常先生的文章文采斐然。如果说他关于马克思主义的文章读起来太理论、太学术，那他当年27岁的时候，在早稻田大学留学时写的那篇《青春》则令人热血沸腾：

“以青春之我，创建青春之家庭，青春之国家，青春之民族，青春之人类，青春之地球，青春之宇宙”。这篇文章看得我很感动。守常先生真真儿对的起那句“铁肩担道义，妙手著文章”。

知乎上看到过一个比喻，我觉得特别好。中国共产党在孕育之初，是有父母的。仲甫先生是父亲，守常先生是母亲。

有仲甫在，你就会觉得有力量，有信心，要战斗。有守常在，你就会觉得很安心，很温暖，特坚定。一个汪洋恣肆，一个理性克制。

跟陈独秀在一起会感觉这个人挺危险，但是贼刺激。和李大钊在一起就会感到很踏实，很放心，很有安全感。

即便知道这人也是个头铁的猛人，但是他不会动辄就任性出个幺蛾子啥的。这俩人真的是绝妙的搭档。而他们所孕育出的中国共产党的最具代表性的杰出结晶，就是毛主席。

如果说陈独秀是呼啸狂放的龙卷风，李大钊就是汹涌深沉的黄河水，在风的激扬与水的克制下，毛主席将那星星之火炼成了革命的熊熊烈火，把旧世界烧了个一干二净。
