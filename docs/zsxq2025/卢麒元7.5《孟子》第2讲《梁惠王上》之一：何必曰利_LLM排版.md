
# 卢麒元《孟子》第2讲《梁惠王上》之一：何必曰利

今天是2025年7月5号，乙巳年六月十一。今天是《孟子》第二讲《梁惠王上》之一：何必曰利。上一堂课我们简单地介绍了一下孟子，这一堂课就开始《孟子》的全部课程，将来在课程结束之后，还会有一次综合的论述与评价。今天的课约略分成三个部分， 一个是介绍一下子整个的背景；第二个是我们讲一讲这篇文章；最后讲一讲我对义利的思考，就是对义利之辩的我的看法。

我先给大家念一遍今天我们讲的这个课文。孟子见梁惠王，王曰：“叟不远千里而来，亦将有以利吾国乎。”孟子对曰：“王何必曰利，亦有仁义而已矣。”王曰“何以立吾国”,大夫曰“何以利吾家”,士、庶人曰“何以利吾身”。上下交征利而国危矣。万乘之国，弑其君者必千乘之家；千乘之国，弑其君者必百乘之家。万取千矣，千取百焉，不为不多矣。苟为后义而先利，不夺不餍。

这是一篇流传千古的义利之辩，这对中国的政治伦理产生了极为深刻的影响。以此文开篇其实是很有意思的，但我先不讲这篇文章，因为孟子的文都比较浅，就是容易读、比较浅白，不是不深刻，是比较容易读懂，更贴近于日常语言，容易读懂。

我先介绍一下子梁惠王和梁惠王所处的战国时期。你知道，三家分晋之后出现了魏国，赵魏韩。赵在靠最北边，魏居其中，韩居其南，赵魏韩。三家分晋之后，魏国的创国者魏文侯、魏武侯励精图治、礼贤下士，任用李悝、吴起等完成了初步的改革，并且建立了强大的魏武卒，成为了战国初年的第一强国，最厉害的。

惠王则有点意思，如可以做一个比对的话，惠王更像今天美国的川普，惠王有点像懂王。为什么他的爷爷叫魏文侯，他的爸爸叫魏武侯，而他变成了惠王?因为他称王了。同此时期，秦国的国君他还叫秦孝公。公侯伯子男，他本应是侯，但他称王了，你能理解此人内心的深处。魏惠王所处的时间是公元前400年到公元前319年，他活了81岁。因为他迁都至大梁，就是今天的开封，所以他又被称为梁惠王。

梁惠王本姓是姬，跟周文王、周武王是同一个姓，他的名字叫莹，姬萄同学。姬萄同学在位五十年，他是公元前369年到公元前319年，在位整整五十年，然而他的治理却出现了大的问题。他不像他的爷爷魏文侯，也不像他的父亲魏武侯，确实有点儿不似人君。心很大，他称了王，然而内政外交可以说是一塌糊涂。

惠王二十八年，也是公元前342年马陵之战，他的太子被俘；惠王三十一年，也就是公元前339年，岸门之战大败；后元五年，也就是公元前330年，败于商鞅，献河西之地，最重要的一块土地，河西之地。商鞅变法之后，秦国强大，夺取了河西之地。后元十二年，公元前323年，楚国伐魏，破襄陵、取八邑。你看，先失河西之地，然后再丢了八邑，楚国又抢走了八个县城，二十年连败四战，盛极而衰。

也就是在这个时候，公元前320年，就是楚伐魏之后，大约三年左右的时间，梁惠王迎来了孟子。这个时候惠王已经年近七旬，可能超过七十岁了，还没到，差一岁，孟子这一年五十三岁。就是梁惠王糟蹋他爷爷和他爸爸留下的家底，糟蹋差不多了，这时候到处碰壁，这回孟子周游列国来到魏国。孟子已名扬天下，已经成为一代大儒，所以梁惠王想请孟子给出出主意。

战国时期，其实各个国家的情形大同小异。这里边我们要讨论一下，战国时期变法图强的根本性问题是什么。这一点有一点像当代：到底如何让国家富强起来?或者是说今天我们讨论的问题：到底如何完成工业化?或者说我们今天替美国人思考的问题：到底如何完成再工业化?在战国时期答案约略有两个， 一个是搞帝国主义，就是通过暴力扩张来把国家搞得很大。一个是走社会主义道路，通过改革，让国家富裕起来，老百姓愿意为国家而奋斗而献身，就是两条思路。第一条思路基本上是法家的思路，第二条的思路是儒家的思路，帝国主义和社会主义。我为什么不用资本主义呢?其实，战国时候的帝国主义思路和资本主义仍然有一些不同，但战国时期的社会主义反而跟资本主义有一点相似或者是类似的地方，所以法家搞的帝国主义里边也包含了社会主义的一些元素，特别是商鞅变法里边。

非常遗憾，非常遗憾，梁惠王、年轻的时候的梁惠王，在魏文侯、魏武侯卓越奋斗之下，魏国强大起来了，梁惠王心里的答案竟然是帝国主义。因为魏国的确具有搞帝国主义的必要条件，就是魏文侯和魏武侯确实使得国家富强起来了；第二条就是，魏国确实拥有一支强悍的军队一一魏武卒。所以他在他的有生之年，或者是他执政的整个的前四十年时间，在搞帝国主义、搞帝国主义，没想到他的帝国主义失败了。

而孟子呢，他的方案是社会主义，我一再强调，孟子的解决方案就是中国的古典社会主义方案。孟子有没有问题?当然有问题。孟子是否真的理解或者清楚，社会主义改革不亚于一场革命?你看到商鞅变法最后的惨烈结局就知道了。要触动既得利益集团，靠君子之道是无法落地的。但孟子确实是不具备两样东西， 一个是法家的权谋与诈术，他是没有的；另外一个，他也不具备孙子的非常卓越的军事能力。所以我们看到了，依然眷恋着帝国主义逻辑的梁惠王遇到了决绝的古典社会主义孟子，产生了激烈的碰撞，这是道与术的剧烈冲突与碰撞，这个碰撞延续了整整两千……差不多……我看了一下应该整整延续了两千三百年吧、两千三百年，非常有趣，非常有意思。梁惠王做错了什么呢?这是一个非常大的问题，我认为中国古代史研究一直存在着严重的缺陷，就是对古代史的研究没有着眼于基本的经济分析。

梁惠王没有坚守魏文侯当时的改革，在孟子眼里，梁惠王对内不仁，他没有办法完成对魏国旧贵族的整合，没有完成像商鞅变法中废井田、开阡陌、奖励军功，就是古典的均田制没有进行，他没有给予自耕农、给予新生的地主阶级以足够的发展空间，维护了旧有贵族的利益。他的不仁，导致他的民——他的士、庶阶级与统治阶级并无亲近感，不仁。

梁惠王对外不义。他打了无数的仗，自他当王之后一直在进行战争，他就是欺凌弱小，以夺占别人的土地、财产和利益为目的，并没有一整套完整的外交或者是战争的逻辑和思路。非常像懂王，就是对内不仁，削贫——夺取贫困人口的东西给富人，反向罗宾汉；对外不义，就是能占的便宜都要占到，从来不承担他应承担的一个大国的责任、使命、担当；所以，惠王这个人不似人君。

孟子又对魏国建立本身——三家分晋本身，他就存在伦理上、道德上的质疑，尤其严重的是梁惠王在与赵、韩处理晋国王族之后，他整个是一次非常残酷的屠戮和灭绝，感到深深的不耻。因为他们灭绝了晋王之后，纷纷称王，就是把周礼踩的粉碎，就是无论是从仁和从义，还是从礼的角度，确实是梁惠王都是个垃圾。

然而，周游列国的孟子也不能免其俗，所以他依旧去拜访了梁惠王，希望能够改变一些事情。孔子和孟子有个共同特点，他们的特点都是止于魏，他们就是没有再向前走，就是没有去到大秦帝国，没有去到秦，这是一个历史的遗憾。所以不能验证他们的思想，或者在不同的土地上遇到不同的君王，是否有一次碰撞，或者是产生一些完全不一样的历史结果，这个不敢去深想。

好，我先过一遍《何必曰利》。“孟子见梁惠王”，这个不用翻译。“王曰：‘叟不远千里而来，亦将有以利吾国乎’”“叟”，就是老同志，你不远千里而来，是不是将有一些非常牛的策略，以帮助我国能够获得稳定与发展呢?我想，梁惠王原话问的是这个。

当然现在翻译的乱七八糟，这个“有以利吾国乎”，不是让我国赚钱，是你的主意亦将有谋以利我国乎，是帮助的意思。

“孟子对曰：‘王，何必曰利，亦有仁义而已矣。’”惠王啊，何必一定要讨论利益呢?其实这里边有一个转换主题的意思，其实惠王说的并非利益，而是帮助的意思。他没有说帮助，他这儿偷换概念、偷换主题了，孟子。

“何必日利，亦有仁义而已矣”。这个翻译大家的争论比较大，“亦”呢，他们把“亦”字当成只有的意思，只有仁义罢了，我不这样看，这个“亦”字我还是当它翻译成“也”，也有仁义而可以……

下面的话呢，则是孟子接连、连问，三连问，三连问是将这件事情捅到最後。“王日‘何以利吾国’，大夫日‘何以利吾家’，士、庶人日‘何以利吾身’”。就是大王问怎样利吾国，怎么帮助我们的国家?当然了，现在的大部分翻译是，我们国家怎么会有好处?大夫说何以利吾家，其实这个利是帮助，但是他说，这个翻译都是我们家怎么才有好处?士、庶人何以利吾身，我自己能有什么好处，所有人都在问好处，我国的好处，我家的好处，我的好处。三个都是连问。
