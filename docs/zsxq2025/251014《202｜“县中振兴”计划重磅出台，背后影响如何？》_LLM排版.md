
# “县中振兴”计划重磅出台，背后影响如何？

## 县中的困境

县中数量占到了全国高中总数的 49.76%，近一半；而学生总数，则占到了全国高中生的 57.27%。县中占据着高中教育的半壁江山。预计到 2032 年，全国普通高中在校生将达到 4032.6 万人的峰值。如果县中办不好，中国高中教育的半壁江山就谈不上高质量；如果县跟不上下接下来的高中入学潮，未来几年可能会是个大问题。

县中振兴，是个立体的解法。县中自身更具体的困境，也必须要解决。而这些困境，正是这次《行动计划》要针对性去解决的。

### 资源的困境

全国还有 920 所县中，学生的平均校舍面积没有达标，缺口总面积达到 660 多万平方米；全国县中学生的平均理化生实验室面积，也没有达到国家规定的 3 平方米标准。农村县中的“空心化”与县城县中的“拥挤化”并存。当前有不少县中，面临教育的基础设施资源缺乏和不均衡的现实。

### 教育质量困境

县中课程供给同质化严重，提供给学生的选科组合少，甚至只提供传统文理组合。县中课堂教学讲授时长占比超 80%的“满堂灌”，与机械刷题主导的应试训练普遍存在。

### 师资的困境

部分弱势县中的优秀教师“引不进、留不住”，而一些省会和地市中学，通过跨区域挖走县中骨干教师的现象也屡禁不止；另一方面，县中教师学历偏低、学科结构失衡，培训机会少的现像广泛存在。

### 生态的困境

部分强势的省市中学和民办中学，通过跨区域掐尖和高额奖学金，使得不少县中“尖子生”被虹吸，县中难出人才，在这种现实下，家长们也倾向把孩子送到教育资源集中的地方，形成县中学生生源“流失—下滑—再流失”的恶性循环。

### 保障的困境

地方财政对县中的保障力度有限。2023 年，和城区高中经费相比，县中经费投入的平均总量，要低 2800 多万元，县中经费增速比城区高中低 12%。

## 教育资源和教育质量

《行动计划》着手解决县中当前的具体困境。

### 资源问题

推动现有普通高中挖潜扩容，确保新增学位供给；优化重组现有教育资源，比如允许“利用义务教育学校和中等职业学校闲置资源举办普通高中”。新建一批普通高中，重点补齐教室、实验室、图书馆、运动场地、学生宿舍等基础设施短板。升级教学仪器设备，加强学科教室、实验设备、体育器械、照明灯具等教学条件配备。

### 提高县中教育的质量

落实研究性学习和跨学科教学要求，关注学生学习过程，创设与生活关联的任务导向的真实情境，促进学生自主合作探究学习。强化学科实践，开好理化生等实验课。让学生从“被动接受知识”，向“能力自主生成”转型，培养更多的创新能力。

## 解决老师保障问题

### 配齐配强师资队伍

深化“县管校聘”管理改革，及时补充县中紧缺薄弱学科教师，依法依规设置正高级教师岗位，县中可以通过特设岗位引进急需的高层次教师。对未经人力资源社会保障、教育等部门同意，恶意从县中抢挖优秀师资的学校，取消其国家和省级评优评先、改革试点等资格。

### 加强师资培养培训

实施“头雁教师岗位计划”，培养县域领军教学人才；建立“名师工作室”，打造教师发展共同体。实施“基础教育教师学历提升计划”，将县中教师作为在职提升学历层次的重点。

### 优化教师待遇保障

保障工资，“按照国家规定落实工资待遇政策”；保障房子，“将符合条件的县中教师纳入当地住房保障范围”；保障荣誉，“各级各类教育评优评先项目适当向县中学校和教师倾斜”；保障职称，“在职称评聘、待遇保障方面给予县中教师更大支持”。

## 教育生态的新变化

### 严格规范招生行为

全面落实属地招生和“公民同招”政策，严禁省属、省会城市高中违规面向县域掐尖招生，严禁空挂学籍、跨区域抢挖生源。

### 推进招生改革

鼓励各地在具备条件的县域，开展中考改革试点，探索登记入学、均衡派位、划片招生等多元化录取改革。优化高校招生专项计划政策措施，将符合条件的县中纳入实施范围，加大对教育基础薄弱地区的倾斜力度。

## 财政投入与管理体制变革

健全经费投入机制，调整经费投入结构，加大省、市两级对教育基础薄弱地区县中的投入支持力度，优化教师工资待遇保障，按标准足额拨付生均公用经费。探索市县结合管理体制改革，鼓励有条件的地方加大市级在资源配置、经费投入、教育评价、规范办学等方面的统筹力度。
