
# 关于这轮以旧换新补贴的疑问

镇长好！关于这轮以旧换新补贴有一些疑问想要向镇长请教。目前的政策解读多数认为政策本身是为了扩大消费，但根据我个人观察至少在手机等电子产物品目国家补贴实际上被厂商全盘笑纳。主要手段包括老款机型以国补代替正常降价、新款机型提高起售价等，而又以前者更为魔幻。很难想象多家国产厂商已经上市几个月的旗舰机型在国补后售价仍比某多多一家企业的百亿补贴贵几百元，且国补需要登记电话、身份证号、住址等敏感个人信息而百补不需要。这导致很多网友对政策有效性产生了深深的怀疑。

对此有网友认为国补本身就是为了补贴厂商，避免厂商资金链出现问题。但根据公开信息，仅小米一家厂商就有一千七百多亿现金储备；OV荣耀等厂商没有上市不过百亿级别应该也是有的。作为普通人实在难以想象坐拥如此规模现金的厂商仍需要国家输血维持。

并且即便是补贴厂商似乎也并不需要如此大费周章。自己百思不得其解，故想请教镇长关于国补的目的和执行怎么看？

## 镇长回复

你好，这是一篇关于经济形势分析的月度章《啼准降息、增量刺激，到底在等什么？》共四页，也就是四月经济形势分析会，提出支持的结构性货币政策工具，如服务消费与养老再贷款，不是给消费者的，而是给服务、养老的供给者，比如商场、养老院、相关商品服务的生产商等等，以提高供给质量，这在服务领域特别突出，未来财政政策也会采取类似的结构性刺激。

## 补贴政策的思考

在全球各国都在相近办法搞产业补贴，而中国遭受比较大的补贴质疑的时候，以旧换新这种方式进行变相补贴，也没啥不好的，反正企业赚的钱大多数还是要通过各种渠道流入到于家万户，这不就保证了就业。这也是国家促进消费的传统思维，促进生产、增加就业，然后带动消费，而非直接刺激消费，目前政策有所调整，但仍然主要延续过去的做法。

最后，还是要再说一点，对于补贴，千万不要想着没有任何问题，水至清都无鱼，骗补的多了，才说明这项政策有活力，未来才更有期待。比如新能源汽车领域，当年骗补的多余牛毛，资金有效利用率并不高，但是最终一定是大浪淘沙，而且随着监管加强，骗补的终究会被淘汰，而这一轮补贴也就支撑了宁德时代、比亚迪等巨头。
