
# 领导提拔人时，他脑子里在想什么？

一个老大哥分享的故事：省纪委的小伙子平时谨言慎行，一次开车送分管副职检查基层时，因保安不让他过而破口大骂，结果次年晋升时被副职以人品待考察为由否决，副职还描述了当时小伙子对保安蛮横无理、飙国骂的情景。

## 领导重视的三方面

### 1. 品质：绝对的忠诚

领导用人最看重忠诚，需要绝对的执行力和成为自己的“自己人”。认个大哥，大哥说啥就是啥，不能没有忠心干活的人。自己人用得踏实听话，交代任务不打折扣，行动上马首是瞻，干工作心往一处想，劲往一处使劲。

之前《走向共和》里李鸿章也说过：“我不任人唯亲，难道让我任人唯疏不成？非让我的部下不听话，处处和我作对，就不叫培植私人势力了。”

### 2. 能力：绝对的靠谱

能力强能确保常规工作开展，是提拔的必要条件，但绝对忠诚、听话才是充分条件。利人有的是，忠人才是第一选拔条件。

### 3. 发展：绝对的资源

领导也需要借力扩充资源。例如，一个学长在非典封校期间说要去北京参加阅兵，教导员才同意放行。当然，普通人能努力达到自己的天花板就不错了。领导知道哪个岗位有退休空缺就往上走，哪个组织要对哪个岗位考察提拔，他本来也正苦于搭关系，借平台跑官呢，得知你爸你爷爷能安排个酒局，随便都是副部级，他深知自上而下的打招呼比自下而上的蚂蚁拱粪要轻松得多。

## 站位不同，用人方式不同

### 提拔前夕

新换了一把手，按住人事不让动，谁能招商就动谁，那就重能力，不管品质多坏的猫，只要能抓耗子、搞招商、提指标、做好数据就行。

### 工作日常维护

那就看品质，工作上有态度就行，能力可以后天培养，但品质不行，绝不能给自己留个油桶，影响下面的政治风气和干事氛围，增加管理难度和成本。

### 别人打招呼

想塞两个人，那就必须重品质，能力提都不提，品质是可左可右的筹码。

## 总结

以上是关于领导提拔人的思考，供大家参考。
