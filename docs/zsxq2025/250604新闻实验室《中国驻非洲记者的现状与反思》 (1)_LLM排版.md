
# 中国驻非洲记者的现状与反思

## 软实力战略为何受阻

19名中国驻非洲记者的访谈揭示了软实力战略在非洲的困境。这些记者来自党媒、市场化媒体和非政府组织媒体部门，驻扎在西非、南非等重点区域，驻非经历从8个月到12年不等。他们普遍对中国媒体在非洲的影响力持怀疑态度，认为软实力战略存在三大难题：

1. **本地化人才稀缺**：缺乏懂当地语言、文化和政治的专业记者和专家。
2. **人力物力有限**：外派记者身兼数职，无力深挖真实故事。
3. **与非洲社会联系浅表**：官方交流活动流于形式，无法建立实质联系。

更大的难题是，中国媒体制度限制了记者的创造力。许多记者坦言，自己的报道与国内"内宣"无异，难以打动本地受众。他们羡慕西方媒体的语言优势、本地化策略和更高的效率，但认为西方媒体对非洲的报道存在误解和偏见。

## 理想与现实的博弈：记者的三种应对策略

面对现状，驻非中国记者发展出三种独特的"反思性"策略：

1. **融合**：将日常工作中的不满消融于服务软实力的目标中，内化对"服务国家软实力"的认同。记者们普遍认为，遵守上级指令是为了回击负面描绘，推广更准确的中国形象。市场化媒体的记者则强调职业自主性，认为文化竞争需要更多本地化报道。

2. **设定边界**：在"正面报道"与真实性之间找到平衡。记者们会主动寻找展现中非真实、多元面貌的题材，并通过向上级提交"内参"传递真实信息。

3. **泛化与非进攻性**：主动降低媒体的角色和重要性。记者们将"软实力"泛化为宽泛概念，认为媒体只是"汽车喇叭"，主要任务是"风险控制"，即反击西方抹黑。他们否认中国新闻模式对非洲的输出企图，认为非洲新闻生态千差万别，中国模式不具可复制性。

## 记者"反思性"背后的深层逻辑

这些"反思性"策略并非偶然，而是多重因素共同作用的结果：

- **个人层面**：中国新闻业存在"舆论倡导者"和"上班族"两种职业取向，驻非记者是两者的混合体，既要服从体制要求，又要追求理想。
- **组织层面**：中国主流媒体受国家对外传播任务和经济压力的双重影响，官方对"高质量外宣"的呼吁为记者提供了更多反思空间。
- **社会结构层面**：中国媒体在非洲的报道受制于体制管理，远离当地受众。过度正面的报道导致中国媒体与非洲受众产生距离感，这种矛盾使得记者不得不自我调节，在有限空间内实现"个人—组织—国家"的动态平衡。

这篇论文将"反思性"引入对中国驻非记者的个体分析中，突破了以往以国家、媒体机构为中心的研究框架。研究发现，这些记者并非被动执行者，而是拥有相当自主性和能动性，会根据实际情况灵活调整报道策略，努力在"国家叙事"与"个人专业性"之间寻找空间。这种"反思性"不仅是个体心理的自我调适，也揭示了中国外宣实践的真实困境，为理解中国国际传播的复杂性提供了新的视角。
