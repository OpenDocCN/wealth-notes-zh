
# 破美元者，黄金：推动黄金上涨，会远比“人造股票牛市”好得多

今晚我们来聊一聊一个我之前一直很想聊的话题，这个话题甚至是4年前我就很想讲，但是由于当时课程结束导致一直没有讲，拖到了现在，今晚终于有机会跟大家分享了。这个话题就是黄金。

在4年前讲股票的时候，我就在那篇文章里面就非常明确地说了，从我个人的角度并不看好股票市场能够成为下一个类似于房地产市场这样的池子。要是房价下跌导致居民的财富出现一定程度的缩水，我觉得大部分人还是能够承受的，哪怕出现下跌，很多人还是可以安慰自己，我好歹还是有一套房子在这里，水泥钢筋看得见摸得着，再怎么跌住着也挺好的。但是股票不一样，股票跌了就真的是跌没了，不管之前账户上有再多的钱，当你跌没的时候，没了就是真没了，别说混凝土钢筋，连个空气都不存在，只是一个虚幻的数字增减过程。

如果我们承载居民财富的主要载体是股票市场，而股票市场一旦出现那种30%,50%这种幅度的下跌，意味着什么?意味着很多人的财富是要出现大幅度缩水，这样的话整个社会是要乱的。不要以为这种30%,50%的下跌很罕见，实际上每一轮牛市过后的股灾，都出现了这么一种幅度的下跌，甚至70%都不出奇。

如果说我们居民的主要财富载体都是股票，那每隔个十几二十年就来这么一次财富的大幅度缩水，我觉得很多人真的是要疯掉的，这就是为什么站在我的角度来看股票市场过去不曾是中国居民财富的主要承载市场，它在未来也绝对不可能替代今天房地产市场的一个地位成为居民主要财富的载体。

股票市场天然的属性就是多的会越多，少的会越少，它天然地就会倾向于更有资金优势和信息优势的资金，而不是普通人，普通人进去这样的市场里面，是很难获得超额收益的。

在过去的几十年里，无数次已经证明了这种全民追逐泡沫的结果，最后一定是财富被极少数人的拿走，最后绝大部分人落得一地鸡毛。

而在今年的讲股票的两节课里面，我也详细地给大家讲了，这个世界最暴利的两个东西，一个就是印钞票，另外一个比印钞票更暴利的是印股票。印钱都需要先掏钱去买一堆纸，去买几桶变色油墨，而股票说增发就说增发，说上市就说上市，连印东西的成本都省了。

进入到这样的市场，你如果没有办法去成为“印”的那方，那么你天然的就是别人剥削的对象。

所以我非常非常不建议大家将自己甚至是家庭的主要资产构成配置成股票，如果真的要配置的话，保守起见只能去买一些价格和市盈率都处在相对低位的大型国企，还稍微安全一点，但是一旦碰上大的系统性事件，哪怕是这种也未必安全。

那什么资产在我看来相对安全且有增值空间呢?自然是今天的主题了。

我们今晚就可以围绕两个主要的角度来探讨。

第一，主要的“池子”其实就那么几个，如果房地产市场已经处于一个下行周期，而且肉眼可见之的时间范围之内我觉得也很难会再去复制像过去那20年那样的高速的增长，而股票市场由于它的根基和自身的特性都不足以成为一个承载绝大部分国民财富的市场，那么这种情况下，为什么黄金会比较合适。

我们可以好好比较一下，相对于股票或者数字货币这些东西来讲，黄金在我看来有一些什么样独特的价值，这些独特的价值给我们带来什么不一样的思考。

第二就是站在政策的制定者或者管理层的角度来讲，为什么去推动黄金价格的温和上涨，甚至是成为一个远比今天承载的功能和任务都要更多更大的池子，从长远的时间来讲对整个国家的金融系统和国民财富构成是有利的。

我们先围绕第一个角度，黄金自身的独特性，表现在哪里?

在我的投资哲学里面，黄金的本质是什么?我觉得，它是衡量世界混乱程度的一个极其重要的先行指标。

如果在你的认知系统给出的判断里，你觉得往后接下来的这几十年世界仍然会沿着之前的轨道运作，全世界的主题依日是和平和发展，依舊是歌舞升平，马照跑，舞照跳，那么我觉得你没必要去碰黄金。

但是，如果你对接下来的世界局势的判断是和平与发展这个阶段在大体上已经结束了，在接下来的日子，全世界很有可能面临的是此起彼伏的冲突、战乱，小规模、中等规模甚至大规模的战争都有可能爆发，那么在这种情况下，黄金就是一个必备的资产配置。

你不一定要把所有的资产置换成黄金，但是整个资产组合里不能没有。

国内鼓吹用钱去买股票的人很多了，而那些反贼公知鼓吹把人民币换成外汇的也很多了，但是正儿八经这些年一直在说，一直在说要在自己的资产配置里面增加黄金比重的，我应该算是比较坚定的一个了。

最近在知乎包括一些平台上，其实很多人在问第三次世界大战是不是开始了，地球online S3新赛季是不是开打了?其实我觉得这个问题从问的那一刻起就错了。从2018年贸易战开始，甚至往更早的时间往前推，可能从2016年特朗普上台开始，地球online S3赛季就已经开始了，关税战、贸易战包括对整个世界造成了极大影响的生物战，都是表现形式。

新冠已经在全球范围内造成了多少人的伤亡这已经是一个无法统计的数字了，到这个时候，居然还有人在问，地球online S3赛季是不是已经开打了，也未免太后知后觉了。

都说盛世古董，乱世黄金，我觉得中华民族数千年的历史所积累下的生存哲学是非常靠谱的。

这些结论在数千年延绵不断的历史以及一次又一次的王朝更替、权力更迭之中已经得到了无数次的验证，没有理由觉得下一次会有什么样不一样。

至少在被证伪之前，我倾向于认为大概率这个结论依旧是适用的。

这一点在跟数字货币拿来对比的话，我觉得会非常的明显。

之前我在讲阿根廷的那一节课的时候已经提到了数字货币很有可能就会暴涨。但是从家庭配置的角度，我一直都极其反对大家去参与进所谓的数字货币里面，因为我从一开始就说的非常清楚了，所谓的数字货币，在我看来它就是一场高科技的骗局。

按照我个人的观点，这只是在人类社会的某一个特定的时间点出现的一类特定的标的，在它之前人类社会不需要这些东西，在它之后当人类社会的整个科技和文明到更高层次的时候，也不需要这套东西。

如果把数十年甚至上百年看作一个长的时间节点的话，数字货币很有可能就只是昙花一现的一样东西。

先不去探讨它真正的开发者是谁，以及这些人背后的目的是什么,我觉得光是从家庭配置的角度来讲，把钱扔到数字货币上就是极其的不负责任的。

从人类的历史来讲，一样东西越高级，越复杂，大概率意味着，这个东西会越脆弱。
