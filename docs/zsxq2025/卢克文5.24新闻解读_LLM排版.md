
# 卢克文5.24新闻解读

## 2025年5月24日 新闻解读
1、卢武铉去世16周年追悼仪式在韩国举行。
据韩媒报道：当地时间23日，韩国已故前总统卢武铉追悼仪式在庆尚南道金海市烽下村举行。卢武铉遗孀权良淑、儿子卢建昊，前总统文在寅及夫人金正淑、国会议长禹元植等出席追悼仪式。
抛开卢武铉在任期间政绩不谈，不可否认他一直都是韩国民主力量的象征。从悬崖纵身一跃则成为他留给韩国最鲜明的印记。一切是非曲直，皆因自杀而停止。
卸任总统深陷腐败丑闻，以自杀“谢罪”的桥段，确实符合东亚文化的特质。一向将儒学上升为儒教的朝鲜半岛更甚。

再度将卢武铉搬到前台来，除了纪念这位已故总统之外，更像是一场凝聚民主力量的政治动员。一方面，卢武铉是在李明博任内自杀身故。后者是保守政治力量的代表人物之一。倘若真的是悼念活动，主办方应该将所有健在的卸任总统都邀请到现场。事实上，只有文在寅前总统夫妇到场，如此泾渭分明的阵营对立，对于化解当今韩国社会的撕裂并无益处。
另一方面，卢武铉家族涉嫌贪腐，并非空穴来风。利用总统名义在海外收取不义之财，则是成为压垮卢本人的“最后一根稻草”。类似这种围猎权力、权力变现是西方政治制度的一种结构性弊端。对此，韩国社会并没有足够且清晰的认知。

很容易陷入似乎以死“谢罪”或“明志”的举动一发生，就足以证明卢武铉的“清白”的认知误区。1948年，韩国建国后，急于摆脱国家积贫积弱状态，在政治制度上全盘西化。不但接纳美国人李承晚回国摘“果子”，而且在国家制度、管理模式直接copy美国。美国有美联社，韩国就有韩联社。美国有白宫，韩国有灰宫（青瓦台的英文名）。美国有参谋长联席会议以主席，韩国也有参联会以主席....凡此种种。一片被儒教文化浸润超过千年的土壤，强行在短时间内被植入“外来物种”。甚至都没有来得及提供适应期，便强行上路。最终的结果，韩国总统成为全球最高危的职业。李承晚最终以贪腐被赶下台，在夏威夷聊度残生。朴正熙、全斗焕、卢泰愚到金泳三、李明博、朴槿惠等，无一逃脱牢狱之灾（朴正熙除外）。
刚刚被弹劾下台的尹锡悦更是集大成者，在任不到三年，集齐背叛、投机、贪腐、滥权乃至戒严等政治行为。让原本已经微妙的韩国社会，因为意识形态走向撕裂乃至暴力。

2、特朗普禁哈佛招收国际生 中国外交部：反对将教育合作政治化。
针对美国特朗普政府以哈佛大学与中国政府合作等理由，禁止该校继续招收国际学生，并强制现有国际生转学，中国外交部回应时称，反对美国将教育合作政治化。
中国外交部发言人毛宁23日在例行记者会上表示，中美教育合作是互利的，中方一贯反对将教育合作政治化，美方有关做法只会损害美国的形象和国际信誉。中方将坚定维护海外中国学生和学者的正当合法权益。

表面看特朗普政府以国家安全、反犹和意识形态禁止哈佛大学的国际项目。事实上，有针对性的打击美国知名大学并非新闻。早在特朗普首个任期，以美国联邦调查局为主的一系列执法部门，针对知名大学的中国学者和亲中的美国学者展开了“中国计划”，并且还对其中若干知名学者进行了司法调查、起诉和审判。这实际上造成了中美学术交流障碍。甚至迫使一批在美中国学者回流中国。

特朗普重返白宫后，意识到只有将自己的意识形态在美国大学，特别是知名大学能扩散出去。才有可能让自己成为更具影响力的美国政客。换而言之，他要与民主党在校园内展开争夺战。长期以来，美国高等教育界是民主党的支持群体。大学越是云集，民主党的势力越强。相反之下，共和党或保守阵营鲜有获得这一群体支持。
教育从来都是塑造人心、改造人心的最佳途径。这一点无论何种意识形态都如此。哈佛大学作为全美最知名的大学，自然也成为第一个对象。以目前状况看，这一趋势只会不断扩大，甚至不排除针对美国高中。
在特朗普看来，“闭关锁国”是最符合美国利益的方式。因为，外面的世界太复杂。
