
# 米联储见闻20250117——有手莫辫猛虎须

随着美国网络难民涌入中国app，美国最高法院快速做出回应，推迟tiktok禁令270天，川普也表示将考虑延缓tt禁令，更有意思的是，tt的ceo将会参加川普的就职典礼，字节跳动原本计划在1月19日关停tiktok，目前又获得的一线转机。这次真正导致tt被暂缓执行禁令的原因还是大量美国用户涌向国内小红书平台，如同A股大涨会导致美联储开始犹豫要不要继续降息一样，美国用户涌向中国app让美国执政者不得不收起任性，中国的存在确实提供了一个B计划，非此即彼的操作让美国的政策选择变得不能那么任性甚至要小心翼翼，因为一不小心就会让中国成为收益方，莫名其妙的赢麻了，这种局面会让美国的政策回转空间日益收缩，做事情被制肘，美国长期以来霸道惯了，早已不习惯谨小慎微多方考量的制定政策和计划。

看到美国在政策方面出现反复，不是美国没有诚信，而是因为长期任性的制定政策，早已让美国政客失去了严谨的决策机制，习惯于一拍脑袋就立法，结果实施不下去，最后不得不一而再再而三的延期，赦免，导致整体政策的实施效果大打折扣，政策震慑力也是逐步消减。

这次美国网民涌进小红书事件已经引起国际舆论的高度关注，美国网民不但自己涌进来，还带着全球网民涌进小红书，不得不承认美国在舆论阵地上真是绝对高低，美国网民去哪里，流量热度和关注力就在那里，小红书目前在全球多个国家和地区都冲上了下载量榜首，在Google Play上，小红书的累计下载量超过一亿次，应用数据研究公司Sensor Tower估计，本周小红书在美国的下载量同比增长超过200%，比前一周增长194%。字节跳动旗下另一款社群媒体应用Lemon8在上个月也经历类似激增，12月的下载量跃升190%，达到约340万次。让美国政客沮丧的是，不管他们怎么打压tt，其它美国社群媒体极少吃到1.7亿TikTok用户搬家的红利。

面对美国网民用脚投票，主动前往只有一个中国总部、没有区分国际版和中国版的小红书，并且公开表态要向中国贡献自己的数据，政客们终于绷不住了。气得自由亚洲这种台娃媒体公然嘲笑这个美国网民从一个中国公司搞得海外监狱翻墙到了一个直接在监狱里的中国app。看到台蛙们这么生气，看来美国人主动靠近中国，确实让一大堆殖人浑身不自在。尤其是中美网民无障碍沟通，更是让这些过去横在中美之间制造信息差的蛙民异常焦虑。尤其是美国网民为了讨中国网民高兴拼命秀中文，让这些只想在中国人面前秀英文的中间商感到生理不适公开说很沮丧。

从数据上看，其实涌进小红书的网民数量整体没有超过100万，但是舆论效应是过亿级的，名为“TikTok”的话题目前在小红书有15.6亿浏览量，976万讨论量，“TikTok难民”也已获得3.4亿浏览，743.8万讨论。有意思的是，由于新鲜感，尽管在美国最高法院表示推迟tt禁令后，小红书的下载量和在线活跃用户并没有立即下降，有不少新注册的海外用户接受媒体采访时表示，他们最近花在小红书上的时间远远超过TikTok。不少表示，即使 TikTok 保留下来，也会继续使用在小红书上创建的帐号。言下之意，美国国会和政府用了大量的人力物力资源帮小红书做了一次免费的超级路演和推流，其效果是数百亿宣发费都换不来的，在过去这几年，美国出钱出力最后让中国赢麻了的事情还真不少。

由于现在进驻小红书对于这些外国用户来说，是一个超级红利期，可以快速涨粉和打开新的市场，也打开新的机会，这意味着很多外国博主将来需要和中国电商平台和商家对接进行商业互动。我个人觉得有些过滤，要相信中国在处理大数据方面的能力，这次小红书在接住泼天流量的情况下目前没有出现太大的问题，背后不可能没有政府部门的介入，包括上海网信办在内的数据指导，如果这次能够成功实现对接，一个很可怕但是惊人的事实就是，中国在即时数据和信息的监管方面已经拥有非常完善的技术能力，要知道中国本身就是全世界网民人数最多的国家，每天需要处理天量数据，如果这次对接成功，意味着中国向管理跨境数据技术方面又迈进一大步，算力即国力，数据即财富，这对于中国未来进入全数据管理模式还是有很大的帮助的，政府不会轻易放过这种练兵的机会，现在已经不是中国要关门的时代，能够对100多个国家实施免签入境政策，说明对线下管理的能力非常自信，自然也会对线上管理能力有自己的信息，这种方向输出的机会，高层不会轻易放过的，所以很多人的担心是过虑了，不但不会关闭，我估计很多中国app都会开始意识到这波app出海是重大商机，要快速跟进。

实际上，已经有app开始跟着吃红利，小红书用户激增和汉语学习需求的激增，语言学习软件“多邻国”近期也在社交帐号上专注于推广其普通话业务。上海市政府新闻办也透过其小红书官方帐号表达对“TikTok难民”的欢迎，虽然这篇贴文发布后仅几个小时即遭删除。但多家官媒已经下场发文，对小红书的此番走红表示庆祝。其实国内各种社交平台很多，为什么是小红书接住了这波流量，我觉得背后不排除有大策划的可能性，如果没有准备，这一波冲击之下肯定会有很多问题出现，比如服务器宕机，审查问题，传播事故等，但实际上这些都没有发生，只能说，小红书对于这波流量并非一点准备都没有，虽然我们现在还没有渠道了解到是否背后有官方操盘，但是app选的如此精准，对接如此丝滑，话题引导如此到位，要说没有人策划，我是不相信的。从平台选择来看，小红书素来避开政治议题、营造生活氛围做差异化竞争。年轻女性用户多，偏激政治言论少，狂热的政经用户少也是3亿人的生活指南，主要看点都在消费生活娱乐方面，在这个领域率先打开口子，不能说完全没有政策引导。考虑到，小红书对外海用户开通了直播功能，这也是其他平台无法想象的，如果没有政策支持，几乎不可能做到这一点，因为在境外使用大部分app都无法进行直播的，毕竟直播内容是不可控的，但是在小红书不但能直播，甚至能同时多人连线互通直播，这意味着监管的风险非常高，目前虽然没有翻车，但这个直播模式还是高风险状态，能否保留要观察。英国媒体就注意到，一个“中文课”的对话直播间吸引了超过30万人观看，数百位美国网友在直播间里分析、痛斥美国税收等政策，身着警服的美国人也加入发帖，在抱怨加班降薪的小红书贴文下也能看到位址显示在美国的用户直接回复“你应该罢工”，这对先前的小红书营运环境来说是难以想像的。对于这种中国网友教坏美国网民的操作能持续多久，要观察一下，而且各种反华组织和机构这次还没有回过神来，一旦回过神来一定会利用各种方式渗透进小红书制造负面影响，微博微信都不缺这种人，更何况小红书，所以这个事件最终走成什么样子还是要看舆论双方角力的结果，但是现在和以前不同的是，这不仅仅仅是讲道理分输赢，真正决定谁在这场舆论博弈中获胜的因素是技术，是数据分析的能力，舆论战的斗争形势也在不断变化。

随着懂王归来日子渐近，加沙停火也正式进入落地阶段，在经历了超过15个月的毁灭性战争之后，以色列和哈马斯的谈判代表已同意在加沙停火。根据协议将释放人质和囚犯，并恢复“可持续的平静”。停火将于1月19日星期日开始。停火的第一阶段将持续六周，以色列军队将撤至加沙东部，远离人口稠密地区，人质将在这一阶段开始释放，在协议的第一阶段，巴勒斯坦人将能够返回加沙所有地区的居民区，并将获得大量人道主义物资。约100名人质仍在加沙，但以色列当局认为其中约35人已经死亡。在为期42天的协议期间，将释放约33名以色列人质，该协议需要得到以色列政府的正式批准。以色列将于周四上午进行批准投票，但目前至少有两个强硬派已经宣布反对与哈马斯达成协议。双方最大的分歧可能是加沙的管理者问题，以色列拒绝让哈马斯继续管理加沙，也拒绝让目前管理西岸部分被以色列占领地区的巴勒斯坦自治政府接管加沙，希望在冲突结束后保留对加沙的安全控制权，但这个结果很可能不会被各方接受，目前这场冲突已导致超过46640人丧生。

大国博弈，这些区域冲动都是博弈之下的副产品，川普上台会修复美俄关系，加沙冲突自然也会随之缓解，这些死难者不过都是博弈的牺牲品罢了。中国人还是要努力把命运掌握在自己手里，多收割博弈红利，少替他人做嫁衣。
