
# 学习笔记，2025.3.2

1. 一个心智健全的成年人，有自己的主见，脑子里也应该能够同时容纳多种不同观点存在，关注事实的复杂性，而不是去纠结一些非此即彼的是非判断。
2. 人大多数时候，都是活的一种“感觉”，而非一种“事实”。你感觉房子重要，你奋斗半生拥有了一套房子，就会觉得值，这半辈子没白过。他觉得房子不重要，赚钱就花，从不考虑明天的事，每天逍遥自在，也觉得很舒服。房子重不重要不是事实，而是每个人的感觉，或者说，是别人给这些人脑子里植入到底层逻辑里的一些概念。别人觉得不重要，事实不重要，很多时候都是靠“我觉得”来做出决策。
3. 人可以很坚强，骨头硬、心硬、意志硬，总是能做出一些超越生物极限的事情。人也可以很脆弱，听到某句话就控制不住眼泪，遇到某些事会无法决定生死。人都是复杂的，不能用单一角度去看某个人。
4. 说一个人是“女强人”，是说这个人事业心强、能力强、有韧性，而不是性格强势、得理不饶人。说一个人是“硬汉”，是说这个人果断、有责任心、不轻易示弱，而不是暴力、爱打架。有些人总是误以为给自己贴上某个标签，就能成为大众理解的标签下的样子，实际上都是自己的偏见。
5. 一个人所处的环境、所接触的人，都是自己选择的结果。对应的，别人伤害他、环境干扰他，也是他允许的。因为他完全可以选择远离，即便短时间内不能在空间上远离，也能立刻在心理上远离，不把自己的精力和思绪浪费在不喜欢的人和事上。
6. 主动付出、对他人有基本信任，是一个低成本筛选交往对象的方法。你付出，他只进不出，你就知道这个人没法长久来往。你信他，他辜负信任，你就知道以后没必要再给他信任。好的关系是相互奔赴，主动的一方往往有更多选择的自主权。
7. 对他人，多“从善揣测”，少阴阳怪气、少抱怨，不要给自己树立太多意识上的敌人。你觉得世界是美好的，世界就是美好的，有这种心态，对自己的心境也是一种滋养。
8. 延迟退休，什么人最高兴？管人的人。我刚参加工作时，单位领导45岁就要开始退居二线了。现在？60岁，他觉得自己是小青年。让他让位？他写申请，说想为人民鞠躬尽瘁，死而后已。多大的官就盼着不退休了？正科以上。医院的话，所有的科室主任。学校的话，教授、书记、院长、校长，无论正副，都不愿意退休，一是在岗收入高，二是身份加成。权力不仅仅是男人最好的春药，也是女人最好的春药。在职，一个状态；退休，一个状态。
9. 我大学同学，多从政，真正能脱颖而出的，很少，现在能混上副科的，都算不错，根源有两个：一是前面的人，不想退出。二是过去论资排辈的游戏规则，被打破了。论资排辈，你慢慢熬，慢慢等。所有人都有个美好的未来……现在不。80后没混上副科，90后已经当镇长了。因为，刮了新风。启用年轻干部！
10. 这个大环境下，能吸来存款，都只有一个原因，超级IP坐镇。例如，我去吸存。那，肯定无数人飞蛾扑火。但是，超级IP一般都会建立几层防火墙。至于，会不会揪出来。要看，案件影响有多大，超级IP是否具备灭火的能力……
