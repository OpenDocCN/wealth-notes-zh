
# 收拢人马，准备埋伏（含标的提问）

台风桦加沙过去了，本来预计17级，结果登陆只有13级，而且登陆点一路西偏，这源于来自北方的冷空气形成的副高压中和，并冲淡了台风的超低压气团，使台风无法继续吸收更多的热量持续膨胀。

在此要说的不是台风桦加沙，而是市场。本来中国官方政府打算赌美元降息，让中国官方政府手中因以为傲的科技明星不停的膨胀，等美元大幅降息之后，让溢出的海量资金冲过来接住，同时，也可以冲淡中国政府国债这个超低压。顺便拉抬大量业绩持续下滑的国有企业的估值，从而让银行能够把大量闲置的资金以合法合规的方式放给国有企业，同时减轻银行的存款利息的压力。可谓是一举多得的好想法。

然而，中国官方政府想法是好的，执行大打折扣，不仅外面国际环境不配合，内部国内环境也不配合。不管内外环境，但凡有一方配合，第一阶段完成之后就可以启动第二阶段：不管是内部还是外部的资金，只要有人接住了第一阶段的筹码，中国官方资金就能抽身出来，然后去把那些压到地板上，又与大规模工程项目相关的便宜周期性企业的股权收拾拢起来，完成布局之后，再放出2026年的大规模工程建设计划的刺激消息，然后就可以同时时在现实世界和虚拟的资金市场为这些企业提供巨大的拉动力。中国官方政府破局点其实都已经选好了，明年增长最快的地方应该是西藏，西藏的增速将超过其他所有省份!

但是，天天有不测风云，计划的落地需要形势的配合。如果没有配合强行推进，效果就会大打折扣。所以，在形势明朗之前，包括决策者在内的所有人都会处于一种两难的犹豫状态，这种决策者的犹豫会通过市场的摇摆不定和政策的模棱两可呈现出来。

当然了，这种决策者犹豫的摇摆状态是一种最痛苦最难熬的状态，任何人都处于煎熬之中，包括决策者本身。俗话说，三军之灾起于狐疑，决策者的摇摆不定，会让所有的执行者东一榔头西一锄头，没有方向徒增消耗。在此之前中国决策者很乐观，赌美元必定在2025年秋末大幅降息，结果赌输了，美元只是象征性的做了个样子，并没有如预期那样大幅降息。虽然美方的数据也差得一塌糊涂，但人家算强行做假也顶住了，而且，当华尔街处于利益的考虑达成一致性的预期之后，华尔街明知白宫的数据是假的也当做真的去交易，过一段时间又大幅修正。此举虽然损害信誉，但是只要所有的参与者都基于利益需要，而将错就错的认可金融市场的波动，数据的真假就不重要了。这是一个很可怕的事，但是在你们无法与国旗鼓相当之前，全世界都无可奈何。如果是直接的军事冲突，我们中国有信心做到旗鼓相当，而且，战斗的结果会改变人的被观念裹挟的惯性，然而，在没有战火撕开这个迷雾之前。所有人即便愤愤不平，也只能基于惯性被裹挟着往前走，华尔街恰恰就是利用了这种被裹挟惯性，才可以肆无忌惮的基于假数据进行交易。

如果没有大量的外部资金入场，中国内部的资金热量能不能让台风持续输入能量，完成二次升级呢？不能! 即便决策者下了决心，主要的执行者手里拿了一堆无法脱手的筹码，怎么去执行决策者的决心呢？没有主要执行者连环使劲，那么，再大的计划，再劲爆的消息，最后就只能是一个计划和 一个消息，舆论上就算再热烈也无济于事。

中国官方决策者不是不想制造资本台风，做梦都想，主要是现实的超高压和和外部出乎意料的低压让整个局面卡住了!上不能上，下也不能下。对于中国官方决策者而言，当务之急，最核心的事情是防御，如果对方美国没有因为被迫无奈而大幅降息，造成资金的大量外溢，那么对方美国的节奏就没有乱。一旦对方的节奏没有乱，资金可能进来，但不是惊慌失措，落荒而逃的进入，是有计划有目的的进入。这个目的是什么呢？

在回答这个问题之前，问一句题外话：什么人群最虚弱呢?当然是这20年新增的中国中等收入群体，这个中产群体没有退路，高负债高支出缺乏家庭支撑，甚至还要背负很多的负担，这个中产群体的体面生活其实非常的脆弱。脆弱的群体最容易交出手上唯一值钱的筹码，只要有一种意外让这个中产群体陷入绝境，不管这个筹码付出多少的心血，也不得不交出来。

基于同样的道理，此时此刻最危险的恰恰是过气的科技股热点，这些曾经躺在风口上被吹起来的猪，因为惯性和成本的原因暂时还没有回到年初的状况。如果有一股外力强行的推一波，这种因成本的原因而形成的惯性是撑不住的，又因为已经过气了，难以吸引存量资金去支撑，那么在外力的打击下就很容易连环踩踏，最终回到年初的水平。

当然了，真正要被暴击的时候，肯定不仅仅是这些过气的科技股热点，只不过过气的科技股热点更容易成为突破口而已!突破口造成的溃疡是为了形成泥沙俱下的裹挟力，所以，中国官方决策者接下来的重点是防御这种形成泥沙俱下的裹挟力的可能性。正因为如此，2025年9月底的时候中国官方决策者一改过去的鼓舞态度，突然变得谨慎起来。作为对方美国并没有像我们中国预期的那样惊慌失措，饥不择食，美元降息的幅度象征意义更大，这不是降息多少的问题，这反映的是对方美国对整个局势恶化的控制力。当对方美国控制力还很强的时候，即便是败退也不可全力出击，只有美国落荒而逃，完全失去控制力的时候，才能全部力出击。

战场上是如此，其他领域也是如此。

在防御的阶段该做什么呢?诚如上个月所说：收拢人马，且战且退，不要追热点，留足筹码，等待时机。什么时机呢?为明年2026年潜伏的时机!
