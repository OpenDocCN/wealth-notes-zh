
# 怎么跟领导谈提拔

## 跟领导谈提拔前，要做好这些准备。

### 一是跟领导谈提拔前要刷存在感。

说白了，就是要跟领导混个脸熟，让领导知道有你这么一个人。如果你平时有给领导汇报工作，那么，这一步可以省略。如果你平时没给领导汇报过工作，那你一定要找机会汇报工作。

有的读者会问，我连汇报工作的机会都没有，怎么办？确实，一些大单位的一把手不好接触。既然如此，你就创造一些机会接触。比如，自己主动参加单位的集体活动，征文比赛、演讲比赛、文艺表演等等。在这些活动上，你是能跟领导打个照面。又或者，主动承担领导看得见的重要工作。比如，参加一些工作专班。我有一位朋友，就是参加市里的重要专班，认识了分管副市长。朋友提拔前，这位分管副市长专门给朋友单位的一把手打了招呼。

又或者，寻找机会偶遇，比如，一起乘坐电梯、食堂一块儿吃饭，主动报名参加领导出席的活动。这些活动，就是跟领导混个脸熟。在混个脸熟过程中，如果有机会主动介绍自己，一定要大大方方介绍自己。

当然，以上跟领导混熟的方法中，最重要的是汇报工作。

### 二是平等的心态对待领导。

有很多跟领导只是工作上的熟悉，但私人的亲密度是不够的。亲密度不够，就是跟领导边界感太强，导致无法探讨一些个人问题。这个问题的原因，就是自身的心理建设不够，说白了就是胆子不够。很多人害怕找领导，害怕跟领导聊天，是因为心里有明显的上下级障碍，一是不知道自己怎么聊；二是怕说错话得罪领导。这种担心其实是多余的。

你要明白，大家在体制内上班，除了领导和下属的关系，也是同事关系，某种意义上来说，是平等的，只是分工不同而已。事实是领导并不排斥下属找自己，甚至喜欢下属主动找自己；同时，领导喜欢真实的下属。跟领导聊天，你越放松，领导就越放松，领导对你的印象就越深刻。

### 三是想好提拔的具体诉求。

有些人找领导谈提拔，只是简简单单地跟领导说，“我想提拔，我想升职加薪，我想得到更大的锻炼。”这种诉求，很笼统。领导就算想帮你，也不知道从何下手。所以，提拔前要想清楚你的诉求，至少要想清楚以下几个点：

你想提拔晋升到哪个岗位？你和目标岗位的匹配度是怎么样的？你和其他竞争者相比，你的优势在哪里？能给领导或单位带来什么好处？

想清楚这些，你就能说清楚，领导为什么提拔你的理由。因为，等你真的跟领导谈的时候，领导会问你这些问题，如果你做了准备，你就能应对自如。

### 四是要研究领导的个人特点。

在跟领导谈提拔之前，还要研究领导的性格特点、兴趣爱好、行事风格、个人习惯等等。研究这个有什么用呢？主要还是为了准备你的谈话聊天方式和内容。比如，有的领导性格比较急，那么，你谈提拔诉求的时候，就应当开门见山。千万不要说话绕老绕去，一旦你绕来绕去，导致他不耐烦的话，他一句话打断你，问你想干啥，会让你很尴尬。如果这位领导有耐心，那你就可以适当提及自己的工作情况和工作表现；又或者，聊一些领导感兴趣的话题。比如，他喜欢打球，你就聊打球；喜欢喝酒，你就聊喝酒。这会让谈话氛围变得更轻松。

### 五是确定找领导谈提拔的场合与时机。

找领导谈提拔，通常来说有三个合适的场合和时机：

第一是领导独自在办公室的时候。通过领导的旁人打听，了解领导的工作日程。如果领导这段时间很忙，那这段时间，你尽量不要找领导聊。记住，只有等到领导空闲的时候，他才有心思听你说话。同时也有一个轻松的氛围，来自由表达自己的想法。

第二是约领导吃饭的时候。如果你能约领导出来吃饭喝酒，那你跟领导谈提拔的事儿，就成功了一半。

第三是在领导家里。如果领导同意你去他家里，那你提拔的事，成功率应该接近七成。

## 见到领导，应该怎么谈提拔？

有了谈提拔前的充分准备，接下来，到了合适的场合和时机，就要跟领导敞开来谈提拔这件事儿。

### 一是跟领导谈提拔的套路。

在办公室，可以用“开门见山型”。领导都是职场上的老狐狸了。你主动找上门，领导第一眼就能看透你的想法。如果领导问你有啥想法，说明领导是直接的性情中人。你就可以直接说，跟随领导工作多年，在领导的指导下感觉自己成长迅速，自己想趁着年轻，做多点事，看看有没有什么机会锻炼和进步。如果领导听了你的诉求，可能会指出你的问题。这些问题，有的是别人反映给他的，有的是他在班子会上了解到的，有些是他观察你的工作了解到的。

等领导指出完你的问题后，你要进行表态，首先要表达自己会努力补足短板，改进工作，提高能力；其次要表达，感谢领导的提携之恩。还有一种情况就是，你直接说完自己的想法以后，领导没有说话，没有表态，只是简单说，“知道了”。那你遇到的是那种喜欢阴晴不定、喜欢拿捏人的领导。你更得客气放低姿态，对领导表示感谢，然后迅速结束交流。比如，感谢领导的聆听，我汇报的情况大概就这么多，往后我将会更努力工作。那我先回去工作。

跟领导谈提拔，做到清晰明确，就足够了。如果不擅长表达，把自己诉求说清楚，其实就可以离场了。

在饭局上可以用“过渡铺垫型”。由于饭局上时间充分。那么就不要一上来就说提拔晋升的事儿。因为这样会显得饭局很功利。有一次，我请领导吃饭，领导一上来就问我，为啥请他吃饭。我的真实目的，确实是希望他能在提拔上推荐我。但我没有明确说出来，我只是说，领导在工作上给予了太多的指导，为了感谢领导，所以想请领导吃饭。在饭局上，可以展开来聊天，聊聊最近大家的工作，聊聊自己的工作感受。同时，要多感谢领导，要具体说出领导做了哪些工作，给大家或者你自己的启发。领导听到别人认可他的工作，领导的兴致就会上来。随后，你就可以聊到自己的提拔。比如，以虚心请教的口气和姿态，说说自己的职场困惑，问问领导下一步自己该做哪些事，才能得到进步和成长。如果领导展开来谈，说明他也重视你的个人发展。那你也可以顺势试探性地问一问，最近留意到一个提拔机会，个人分析以后感觉自己条件也是基本符合的，想争取一下。领导如果让你积极争取，说明他会推荐你或支持你；领导如果让你再耐心等等，说明这一次提拔，你时机还没到。不管领导如何表态，都要表达感谢领导的关照。

最后还记得谈谈，自己如果得到提拔，会给领导带来哪些好处或利益。尽可能多地展示自己的价值。比如，你的业务能力、你的领导能力和团队协作能力、你对部门的贡献等等。

### 二是要注意一些细节。

注意仪容仪态，减少肢体动作。眼神要笃定，不要躲躲闪闪、飘忽不定、左顾右盼。交流时，要跟领导有眼神接触，不回避目光。说话的时候，要想清楚再说，要缓缓地说，不要匆匆忙忙地说，要让自己的诉求显得庄重。如果紧张，思路有点乱，可以稍作停顿和思考。

### 三、谈完提拔，还要做一些什么？

跟领导表达完提拔的想法后，后续要做的就是耐心等待。有些人心里不踏实，找完领导后，不到一个月，又想找领导表达提拔诉求。这样你的诉求就太密集了，会让领导感到烦，甚至反感。所以下一次提诉求，至少要等三个月以上。当然，在这个时间段里，你得积极表现自己，争取工作上有成绩，并且争取机会给领导汇报工作。如果有节假日，还要记得问候或拜访领导，给领导送点礼物。这些做法，就是不让领导觉得你虎头蛇尾，提完诉求后就没有表示了。而是让领导知道，你一直在努力争取。如此，领导就会惦记着你的事，有提拔机会的时候，也会关照你。

以上就是怎么跟领导谈提拔。说一千、道一万，都不如实践。如果你平时表现不错，一定要大大方方找领导谈提拔！
