
# 懂王至关重要的两笔交易



特朗普近期的大部分套路的短期目的只有一个，那就是搞钱。用中国老话来说，就是倒左先倒胡，在干掉蓝党之前，先把共青团和贾王财团的生意都废了，理直气壮的做大做强国企经济，在干掉红党之前，把地产金融和互联网平台都废了，理直气壮的做大做好防疫经济。



特朗普要换房本，也需要对民主党搞一轮经济换房本，通过虚拟货币收割取代传统的犹太国际潮汐收割。用懂版俄乌可以取代之前的睡版俄乌协议，用懂版中美贸易贸易协定取代之前的睡版中美协定，用懂版交易取代之前的睡版交易。用懂版金卡取代之前的传统版绿卡，跟拜登跟日韩签署的协议也都统统换成自己的，用 Doge 扫平民主党的各类影子、金融、安全与情报机构，再建立 一系列只忠诚于懂王的汪汪队。



明知道要被换房本，民主党不可能不反击，所以从好莱坞的奥斯卡，到华尔街看衰的金融大佬，从泽连斯基，到美国驻华大使馆，再到各大机构，越来越多的反懂力量开始反击发力了。特朗普想要完成胜利召开美 共 2 0 ,吸取东大经 验，就必须要在内循环的过程中，还搞到足够多的钱，民主党为了避免美共20D 的三个统一，两个确立，就需要在一片赢麻了的声音中，搞出来美版的上海乱局，并狙击每一个懂王能够达成的搞钱协议。



现阶段，乌克兰的生意的阻力是非常大的，民主党联合欧盟，是很难短期内达成交易的，懂王外部确定性最高的两笔生意，都来自于东亚，一个是老友安倍残存的自民党安倍派，日本的投资与汇率，一个是东大老友麾下的新党，贸易与Tiktok 交易，在民主党与欧洲的主导下，懂王在乌克兰吃瘪吃得越猛烈，他对中日的诉求就越高，达成的协议就会对东亚更为有利。
