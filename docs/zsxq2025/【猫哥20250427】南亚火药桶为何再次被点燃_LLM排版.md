
# 南亚火药桶为何再次被点燃

## 事件经过

最近几天，南亚印巴方面的局势陡然升级，剑拔弩张，可以说一出场就是高潮，开战氛围直接拉满。才缓和了几年的印巴关系，突然就亮红灯了。先梳理一下事件的经过：

4月22日，在印控克什米尔地区的帕哈尔加姆镇，发生了一场针对平民的恶性恐袭事件，恐怖分子向游客进行无差别开火，造成至少26人死亡，20余人受伤。导致正在沙特访问的莫迪提前结束行程紧急回国。袭击发生后，主要活跃在巴基斯坦境内的恐怖组织虔诚军的分支——“克什米尔抵抗组织”通过社交媒体宣称对此事负责，指责印度政府允许8.5万名“外来者”定居当地导致“人口结构变化”。而虔诚军因为和巴基斯坦存在千丝万缕的联系，所以印度政府立即将矛头指向巴基斯坦。随即于4月23日，印度宣布采取一系列针对巴基斯坦的反制措施，包括：

1、关闭印巴边境通道，暂停所有跨境贸易的人员往来。
2、外交降级，限制巴基斯坦驻印外交使团人员规模，并且驱逐军事顾问和武官。
3、吊销所有巴基斯坦公民签证，暂停签证服务，要求已经持有签证人员72小时离境，医疗签证可延长至4月底。

最具有杀伤力的是，宣布暂停执行印度河河水条约。要知道，从1960年条约签署以来，哪怕中途双方爆发战争，印度除了威胁也没有暂停此条约。巴基斯坦约80%的农业灌溉和60%的电力依赖印度上游放水，现在又是春耕灌溉季，印度现在“断水”，将直接导致巴基斯坦大部分农业“颗粒无收”，这不就等于逼着巴基斯坦和印度拼命吗?

4月24日，巴基斯坦宣布对印度反制，包括：

关闭领空禁止印度航班通行，暂停与印度的所有贸易及第三方过境贸易，取消印度公民的南盟面签签证，并要求48小时离境。

对于印度暂停执行印度河河水条约，巴基斯坦宣布“视为战争行为”。

4月25日凌晨，印巴双方在克什米尔两国实控线附近多个哨所发生小规模交火。

## 恐袭事件分析

在了解整件事的发展轨迹后，我们首当其冲的需要思考的一个问题，那就是这场针对平民的恶性恐袭事件，真如印度指控那样，是巴基斯坦在背后扮演重要推手，还是另有原因呢?

首先，我们捋一捋巴基斯坦和这场恐袭凶手虔诚军分支“克什米尔抵抗组织”之间的关系。

虔诚军创立于上世纪80年代末90年代初，早期得到过巴基斯坦三军情报局的资助，用于反对印度在克什米尔的统治，推动该地区脱离印度。但到了2002年，巴基斯坦迫于美国反恐压力取缔了虔诚军，该组织便转入地下，通过分支机构继续行动。

其中，规模最大的一次发生在2008年对孟买的恐袭，造成177人死亡，10名枪手后经证实，接受过虔诚军训练。

所以，印巴之间围绕虔诚军以及其分支的问题展开了长达多年的撕逼，印度指责巴基斯坦的纵容，巴基斯坦则否认还有关联。要说巴基斯坦和虔诚军的关系已经撇得一干二净，那确实是在哄鬼，谁不想手里多张牌?就像印度长期暗搓搓支持巴基斯坦国内的极端势力搞破坏一样。

但仅凭这层历史关系，就把帽子扣给巴基斯坦，个人认为比较草率，逻辑上也讲不通。

其一，策划这样一场天怒人怨的恐袭，巴基斯坦的动机是什么?难道动机是制造印巴战端再起吗?要知道从2021年以来，巴基斯坦经济衰退非常厉害，通胀率超30%，年轻人失业率高达25%，贫困率超40%。这种情况下，国内连安保投入都举步维艰了，哪还有资源去主动挑事印度。

其二，如果真要报复一下印度，为何又会选择一个和自己撇不清干系的“克什米尔抵抗组织”动手，还大摇大摆的第一时间摊牌，这不就等于脸上直接写“凶手”两个字给全世界看吗?除非巴基斯坦脑子进水了。

其三，巴基斯坦国内现在陷入一个怪圈，就是“越反越恐”，2024年巴基斯坦开展了6万次反恐行动，但今年国内恐袭数量居然同比还增加70%。这种现象说到底还是贫困问题导致的，连瓜达尔港如此重要的区域，周边的恐袭都搞不定，就更别谈对其它区域有多大的约束力。

因此，我认为更大的可能性巴基斯坦这次是被躺枪了，完全是“克什米尔抵抗组织”的独走。

那么为啥这个组织会选择独走呢?我们从前文提到的恐袭组织对印度的指责说起，谈谈这次恐袭发生地的情况。

## 恐袭发生地情况

大家可能有个错觉，就是认为克什米尔地区就是一个火药桶，乱七八糟的，为啥居然还会有游客呢?事实上，不仅有，而且还很多。

因为在印控区域内，藏着一个旅游胜地，就是绰号南亚“小瑞士”的度假小镇帕哈尔加姆，对，就是这次恐袭的发生地。

这个地方因为风景优美、气候清凉，相比普遍炎热的印度而言，绝对算得上是一个避暑胜地，所以非常受印度人欢迎。在2024年，就有超过300万游客到此一游。

很多人奇怪，旅游胜地和恐袭能扯上什么关系?旅游产业发展不是造福一方水土吗?

要分析其中的因果，我们就需要对历史进行回顾，这里简单粗线条说说。

其实，整个克什米尔问题的导火索就是1947年英国殖民者草率划分的“蒙巴顿方案”引起。当时规定克什米尔的土邦可以自主选择加入印巴或独立，而印度教土邦大君选择加入印度，但77%的穆斯林居民倾向于巴基斯坦，直接引发了第一次印巴战争。

后来，划分印控区和巴控区，印度占据原克什米尔土邦约一半以上的领土，包括查谟、克什米尔谷地等。其中查谟地区以印度教人口为主，是印度在这个区域的基本盘，而克什米尔谷地穆斯林占比95%。

恐袭事发地，帕哈尔加姆镇就是克什米尔谷地这片区域。

简单说这里一直存在虽然是印控克什米尔，但是大多数土著却不认同印度，甚至排外情绪浓厚，穆斯林和印度教相互看不惯，长期存在反印武装活动。这点和俄乌的克里米亚很像。

前期印度为了摆平矛盾，出台了印度宪法370条，就是说印控克什米尔地区具有特殊地位，拥有独立宪法、旗帜、限制非本地居民购房和获取政府职务，默认“国中之国”形式的存在。

但这种平衡被莫迪领导的印度人民党打破了。他们认为这种“特权”就阻碍统一，并为分离主义提供温床，所以从2014年上台后，就着手废除这条宪法。2019年，印度人民党以压倒性优势完全控制两院，最终成功修宪。当然那时，欧美因拉拢印度制衡我们，所以对莫迪的这番操作睁一只眼闭一只眼，也为莫迪改变区域现状提供了很好的外部环境。

从19年开始，这片区域就放开了投资，允许外来移民定居、购买土地以及开展投资活动。

前面说了，因为这里度假条件非常好，所以近几年，印度在这里已经砸下了上100亿美元投资，在途资金还有200亿美元。并且，莫迪政府在这里大力发展旅游业，还有个小心思，就是向这个穆斯林集中区“掺沙子”，通过产业带动进行人口稀释。

这就直接导致一个什么情况呢?印控克什米尔地区的旅游区那是混的风生水起，年均GDP增长超过10%，基建、旅游一拥而上，热火朝天。而巴控这边呢，基本上原地不动。再加之最近几年巴基斯坦整体经济低迷，所以这个地区现在是冰火两重天，贫富差距越来越大。

更关键的是，在发展的过程中，印控区的穆斯林也不断遭到打压，当地人并没有分到多少利益，反而被新居民严重冲击生活，这就激化了区域内穆斯林群体强烈不满。

莫迪在印控区域单方面改变现状，通过投资带动人口迁移，稀释穆斯林人口比例，对不同宗教信仰进行打压，加之区域贫富差距逐渐放大，导致各种矛盾、愤怒越来越大，最终酿成了这次惨烈的局面。

## 莫迪政府的意图

个人认为这才是事件背后的真相。

那么第二个问题来了。上面说的恐袭真相，莫迪政府清楚吗?我认为他非常清楚。而且按道理说，出现恐袭事件，在巴以最近2年关系大差不差的前提下，理论应该是联合追凶为主，共同打击极端组织势力。

虽然印度政府近年来重点开发的“和平旅游示范区”，也是莫迪宣扬治理成效的“政绩工程”。恐怖分子选择旅游旺季在开放景区发动无差别袭击，既重创印度经济信心，更直接挑战莫迪政府的国家安全叙事，但真犯不着一上来就急着针对邻国进行掀桌子。

## 印巴局势走向

个人判断，是莫迪在借题发挥。

其一，是转移国内矛盾。2025年初印度农民抗议运动再起，反对党质疑GDP增速掺水，此时渲染“跨境恐袭”威胁，既能凝聚民意支持，又可借机推进《国家安全法》修订，强化威权统治。

其二，是给美国递上见面礼。要知道最近美印的关税贸易谈判的结果对印度而言是比较有利的。比如：

1、印度通过削减美国商品关税，换取美国对印度商品征收关税从26%暂时降成10%。
2、美国承诺在关键技术、半导体制造等领域加强和印度合作，助力印度工业制造计划。比如4月26日，苹果就宣布将供应美国市场的组装工厂搬去印度。

另外还包括深化美印同盟关系等等，这个大家都清楚，莫迪最好这口儿——在“印太战略”中扮演核心角色。

这几年，美国推进的“印太战略”，说到底就是通过激化中印边境摩擦、挑动印巴对抗，迫使我们分散战略资源。美国现在巴不得我们周边乱，现在日韩被暂时按住了，翻不起浪，菲律宾不够看，只能骚扰。这次关税谈判川普给莫迪这么大的面子，自然莫迪要做出一些成绩出来。

巧了，恐袭发生的时候，万斯正在印度访问，于是就把矛头直击对准巴基斯坦，准备大干一场，也好在川普哪里捞到更多的好处。

说实话，莫迪搞的这出戏，已经演过多次了。

## 结论

最后，对于印巴局势的走向，个人认为是：既然巴基斯坦是躺着中枪，印度是借题发挥，那么这场冲突就不会失控，演变成第四次印巴大规模战争。毕竟今时不同往日，现在印巴各自拥有约160枚核弹头，沙欣-III导弹与烈火-V的射程均覆盖对方全境，谁敢真正掀桌子呢?，一旦开战，莫迪的工业制造计划还搞个锤子，并且以双方目前的国内经济状况并不满足大战的条件，各有各的经要念，最终还是谈判解决。
