
# DeepState与野蛮人

名义上的统治阶级都很清楚，这个世界有一个Deep State。尤其是对于中国来说，Deep State发现国家面临崩溃后，都会引入外部的野蛮人对内部进行洗牌，让王朝得以在狱火中重生。

用一个相对奇葩的角度来看，无论是张角黄巾叛乱、引董卓入京，还是引五胡乱华到安史之乱，甚至元金清时期的少数民族入关，以及东南互保和孙文接受美日苏的注资，背后都是群体意志的选择与运作。他们抛弃了旧的宿主与秩序，去尝试一种新的可能，而流血牺牲来改变世界的动作，都交给了这些“野蛮部落”。

因此，很多斗法都是这种天庭博弈的延伸，神话故事中天庭引蛮夷的西岐消灭商纣，现实世界中美国的资本家支持懂王上台，这些台面上的动作和下注背后，都是新的集团在推翻旧的Deep State，来换一种新的社会运行秩序。

正如所谓的百年未有之大变局，便是随着生产力的发展，一大批新崛起的Deep State对于旧秩序的不满，他们投资并引“野蛮人”入关来打碎旧秩序，来建立符合“美丽新世界”的新秩序，也使得了很多原本不具备资格的人上了牌桌，普京如此，懂王如此，老板亦如此。

懂王对美国军事与情报机构一定会血洗的，大批的军事、安全与情报官员会以各种古怪理由被拿下，这是换房本必要的条件。我们可以回顾过去十年国内的两次换房本，18换房本的时候，干掉的是军副的郭和徐，政法的周，20换房本的时候，干掉的是国防部的魏和李，政法的孟。

卧榻之侧岂容他人酣睡，要换美国房本的特朗普和马斯克，必然也要把枪杆子和刀把子换成自己人，所以这段时间美国也一定是军事上看似很鹰，但实际上并不会真的对外动粗。
