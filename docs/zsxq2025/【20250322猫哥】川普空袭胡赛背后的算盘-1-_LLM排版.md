
# 川普空袭胡赛背后的算盘

最近几天，才消停不久的红海，突然又翻起浪来了。从3月15日，川普亲自下令对也门胡塞武装采取“果断而有力的军事行动”开始，美军已经持续多日空袭也门。同样是炸胡赛，相比拜登的有限克制，这次川普来势汹汹，打算挽起袖子大干一场。当然，来而不往不是“拖鞋军”的风格。3月17日，胡塞武装用18枚弹道导弹和巡航导弹及一架无人机袭击了位于红海北部海域的美军“杜鲁门”号航母及其舰队，算是对美军空袭也门的报复。

其实，这次空袭胡塞并不是川普突然发难，而是早有预谋，只是根据时局的变化将这张牌打出来而已。为啥？因为就在川普正式入驻白宫后的第三天，就否定了之前拜登对胡赛的认定，重启将胡塞武装认定为外国恐怖组织的程序。并且在3月4日由国务卿鲁比奥宣布程序走完。下一步想干什么，彼此都门清儿，只是需要一个开炸的噱头和时机而已。这个噱头就是巴以停战问题。

对于以色列而言，巴以冲突持续了这么久，加沙大半地方都炸成废墟了，但还是挡不住加沙新生儿的出生热情，一边搞生育，一边源源不断参加哈马斯，导致哈马斯在迅速回血。所以，摆在美以面前的就有两个选择，要么赶走加沙民众，让其它国家收容，要么继续开干。目前因为包括联系的苏丹、索马里，甚至索马里的分裂势力索马里兰，均无人响应收容加沙难民，所以只能故意破坏停火协议，继续封锁、继续空袭加沙。

对于哈马斯而言，根据临时停战方案，第一阶段的人质释放工作已经完成，现在是不是该进入第二阶段了呢？但事实确是以色列一边保持空袭强度，一边加大封锁，还要求全部释放完人质再谈。啥？人质全部释放再谈？那不等于以色列不认敲定的停战方案吗？手里筹码没有，还拿什么谈？哈马斯心想除非你比我傻半截。

作为哈马斯的兄弟伙，胡塞自然要站出来力挺，唇亡齿寒道理谁都懂，不可能坐视哈马斯陷入绝境。于是在3月9日以色列故意切断加沙供电后，胡塞随即就是向以色列发出通牒，要求解除对加沙地区的物资封锁，并在3月11日宣布，恢复对红海海域以色列船只的军事打击行动。这下就正中川普下怀。

上期文章说了，川普目前除了在收回巴拿马运河港口这点有眉目外，其余内政外交包括俄乌调停都是处于一阵乱麻的状态，需要另外在中东方向再开一局，转移矛盾，接着奏乐接着舞。这就是时机。对于川普而言，通胀压不住，股市跌成狗吗？没事，对着胡塞一阵疯狂输出，电视上循环播放F-35投弹画面，支持率立马回血。更绝的是，这次他特意穿着高尔夫球服、戴“让美国再次伟大”的帽子听战报，活脱脱把战争搞成“硬汉真人秀”，给选民喂“强国药丸”。反正胡塞武装就像打不死的小强，过段时间再炸一轮，又能上头条。

不仅如此，川普通过空袭胡赛，也要告诉那些不听话的小兄弟，除了谈判，除了极限施压，不要忘了美国还有导弹。这是拿胡赛来立威。当然，除此之外，川普挑动红海冲突，背后其实还藏着美国中东战略的一盘大棋。

先说航运问题。苏伊士运河（含红海航线）作为连接亚欧的黄金水道，占全球贸易份额的12%至15%左右，虽然胡赛这帮“拖鞋军”专挑红海的以色列商船和美国军舰下手，但只要刷导弹无人机，就会吓尿马士基、地中海航运等全球国际航运巨头公司，谁也怕被误伤。于是就选择绕道好望角，这样一来就导致整个航运周期增加8-15天，运费暴涨三成。

对于川普而言，借挑动胡赛，收拾胡赛这件事，远不止要维护自由航行，当和平卫士这么简单，还要达到几层目的：苏伊士运河一堵，就是动华尔街的资本家钱包，自然有求与川普。炮弹一响，黄金万两，国内军火商的订单又来了。第三，国际油价应声上涨，德州页岩油的金主们又高兴了。第四，也是最重要的，就是通过收拾胡赛，彻底掌控苏伊士运河，那么几乎就把全球三大航线（巴拿马、马六甲、苏伊士)尽归其下，为后续中美贸易博弈，美国的船运振兴计划打下坚实的基础。

再说伊朗问题。川普搅动红海，另一个目的就是逼迫伊朗让步，从而彻底掌控中东。伊朗现在日子非常不好过，可以说已经到了崩溃的边缘。第一是通胀问题，伊朗官方的通胀数据目前显示是徘徊在40%左右，但有伊朗专家表示实际通胀率已超过50%。比如大米，相比2024年，价格已经飙升200%，还有住房、公用事业成本、其他食品价格都在大幅上涨。仅1月份，超过三分之一的必需品价格上涨了40%，比去年同期上涨了一倍多。第二是贫困问题，目前有22%至27%的伊朗人处于贫困线以下。整个国家的贫困率约为50%。第三是社会动荡问题，整个2月份伊朗全国至少发生了216起示威活动，参与者包括退休人员、工人、医护人员、学生和商人，更大规模的民众抗议活动也正在酝酿中。一手加薇pep458。

再进一步恶化，再和美国硬刚，不仅佩泽什基安的总统位置坐不稳、甚至连最高领袖哈梅内伊的宝座说不定都要被掀翻。所以，和川普的接触，捏着鼻子也要搞下去，只不过看什么条件而已。目前，我们虽然无法得知川普写给哈梅内伊的劝降信内容，但从哈梅内伊拒绝态度，并表示华盛顿的要求过高，德黑兰不会迫于压力而进行谈判的反馈看，川普提的条件显然让伊朗捏着鼻子都觉得吃不下去。

那么既然如此，双方也就有了先干一场再说的“默契”。胡塞武装背后站着谁？当然是伊朗。伊朗现在是硬着头皮给胡塞打钱送武器，嘴上说着“和胡塞没关系”，心里巴不得靠这帮也门狠人教育一下美以。只不过伊朗让胡塞闹腾，无非是想告诉美国：“老子手里还有牌，谈判不是你说了算？”

至于川普，前面说了，对于胡赛现在有一种截然不同的态度，和拜登相比所谋甚大，所以根本不吃这套，咬死胡塞的每一发导弹都是德黑兰点头才发射的，空袭的目的是警告伊朗，下一个也许就是你。胡赛是张好牌，既不用直接和伊朗开战，又能逼它让步，持续放血。至于地面部队？门都没有！阿富汗的泥潭还没晾干，炸完就跑才是王道。

再说以色列这根美国中东的“铁杆盟友”。内塔尼亚胡正愁彻底解决哈马斯的同时，被胡赛偷袭的问题。这下川普立马送来神助攻，炸了胡塞的防空系统，等于替以色列清场。简直像在说：“老弟别慌，大哥帮你”。毕竟以色列的“铁杆盟友”这个人设不能崩。

不过以色列也别高兴太早。胡塞武装的“巴勒斯坦2型”导弹射程2150公里，覆盖以色列全境不在话下。美军空袭再猛，也拦不住胡塞用土法炼钢的导弹搞偷袭。这场博弈里，以色列既是美国的盾牌，也是胡塞的靶子，稍不留神就得替大哥扛伤害。

因此，特朗普下令空袭也门胡塞武装这事儿，表面上看是反击恐怖主义，维护航行自由，但实际上川普想达到的效果是既要逼伊朗让步，又想攥紧红海的航线，顺带帮以色列撑腰，同时给自己捞点强人形象。

而伊朗呢？经济被美国制裁得半死不活，手里捏着胡塞这张“穷人的王牌”，既想搅局又怕引火烧身，既不敢松手又怕烫着自己。活脱脱一场中东版的“猫捉老鼠游戏”。

这场冲突才刚开始，后续中俄是否介入，我们拭目以待，现在能确定的是，既然川普出牌了，那么大概率红海的风浪短期内消停不了（除非俄乌有新的变化），因为这里面不仅联动俄乌问题，同时又把石油、国际航运、巴以、地缘霸权全部搅在一起。

对于我们而言，现在最迫切的就是尽快解决全球军力投放问题，现在川普在我们家门口不敢惹事，但专挑离我们很远的要道下手，也让我们很难受。
