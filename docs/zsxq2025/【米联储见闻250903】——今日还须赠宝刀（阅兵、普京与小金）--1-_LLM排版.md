
# 【米联储见闻250903】——今日还须赠宝刀（阅兵、普京与小金）

十年磨一剑，今年的九三阅兵实在是信息量太大，我们慢慢来捋吧，从外交角度，政治角度，军事角度，国际战略角度，今年的九三阅兵都是干货满满，光是看今天纽约时报给大阅兵的版面，就知道这次大阅兵的含金量，因为除了阅兵，还要写普京，除了普京还要写小金，除了小金还是写小金公主，对于新闻人来说，这次中国大阅兵简直是新闻宝藏，从头到脚都是新闻热点，话题，创举，随便写点啥都是头条，是欧美媒体想要低调都无法低调，拼命忽略也忽略不了的中国主场外交活动。这次阅兵不仅是一场军事实力的展示，在观礼台上，谁出席谁缺席，领导人之间的排位，每一处细节都充满了国际外交的政治密码。十年前的70周年阅兵，与这次阅兵的景象已经截然不同。阅兵式之外，由于汇聚了各方关键人物，花絮彩蛋都数不胜数。

小金绝对是这场大阅兵除中国元素外的国际顶流，小金把自己的第一次献给中国，对中国是真爱，要知道在欧美媒体上，小金因为常年雄霸负面人物头条知名度远胜于其他国家领导人，小金赴华参加多边外交活动，成为全球媒体最关注的新闻热点之一，从小金的专列到小公主的随行，甚至到小金的车牌，小金在华专车车牌号码“7.27.1953”，这尼玛真是给老美上眼药，1953年7月27日是朝鲜战争停战日，朝鲜和中国都把7月27日视为抗美援朝胜利纪念日。说好了庆祝二战胜利，抗美援朝战争从哪个角度来看都不算二战一部分，毕竟老美也是二战胜利国，对于中国庆祝二战胜利，老美不好说啥，川普也只能酸两句，但是小金直接把抗美胜利纪念日放上专车车牌，这尼玛让老美情何以堪。金正恩以车牌彰显抗美援朝历史，堪称“车牌外交”。在大哥的地盘给老美上眼药，未来朝鲜的战略使命可见一斑。

此次阅兵的另一个插曲就是印尼总统，在访华前，印尼多地爆发示威，总统普拉博沃一度宣告取消访华行程，不知道是不是看到大家都去了，生怕印尼缺席历史时刻，错过啥福利，最终赶在9月3日凌晨抵达中国，要知道第二天阅兵，就算是领导人都要四五点起床排位进场的，为了参加这次阅兵，印尼领导是真拼了，这也算是排面了。今天下午，领导会晤普拉博沃时，称对方克服困难来华出席纪念活动，体现对两国关系的重视，还是相当领情的。印尼总统这个去而复返的操作确实是给领导面子了。领导也表态，中方支持总统先生施政，支持印尼尽快恢复秩序和稳定，支持印尼发展壮大。这三句话，可不是随便说的，这都是要具体做事情的事。

官宣小金出席好，大家一直在想是不是会出现中俄朝领导人同框的画面，事实上，中国对这件事，不但不避忌，甚至是主动为之，美国所定义的邪恶轴心国家这次全来了，不但小金来了，伊朗也来了，小金不但来了，还坐在仅次于普京的核心位置，中俄朝从爬天安门城楼开始就是并肩同框，是的，你们看错，他们一个是被联合国主导制裁的国家，一个是被国际法庭称之为战争罪犯的国家，如今他们都跟中国并肩而立，同排而行，这是什么意思，这就是公开告诉美国，世界的规则不是你说了算的，善恶黑白也不是你说了算的，正义与邪恶更不是你来定义的，中国要完全按自己的意志和主张交朋友办事情搞发展。这次大阅兵反复强调一个正确的二战史观，不但要明确中俄作为战胜国应该获得的国际地位，更主打一个美国不能垄断二战叙事，扭曲历史。

相比于十年前，小金的排面远胜于朴槿惠，2015年举行抗战胜利70周年大阅兵时，韩国时任总统朴槿惠虽然被安排在领导右侧第二个位置，紧挨着普京，但和此次金正恩能近距离站在领导旁边有根本不同。另外一个值得注意的细节是，此次参加阅兵的外国领导人和中国国内领导人是穿插安排的，每隔两三个国际领导人就有国内领导人的坐席安排。十年前，当时出现在天安门城楼的外宾站在领导右手边，国内老领导站在领导左手边。如果说十年前搞阅兵还有对内安抚博弈的政治功能，这次大阅兵几乎就是全面对外，以往，领导左手边第一个位置一般安排卸任总书记。但486并没有现身，小金此次被安排在这里可以说是中国给予的重磅礼遇。所以，小金抱上俄罗斯大腿，中国不是一点想法都没有的。而且未来台海战事爆发，朝鲜对于遏制日韩还是有切实作用的。从阅兵的观礼位次来看，当今中国的最大的挑战在外不在内。如此庞大规模的外宾出席观礼，说明阅兵已经不只是中国的国内议程，而是重大国际外交事件。穿插安排外宾和国内领导人，有招待客人的意味。而对小金的格外礼遇，更表明解决台海问题已经在议事日程上，中俄朝领导人同框现身具有重要历史和现实意义。

朝鲜领导人之前两次阅兵还是金日成曾在1954年建国5周年阅兵时登上天安门城楼观礼。当时金日成站在老毛的右手边，金日成的右边是周恩来。1959年10周年时金日成再次登上天安门城楼观礼，彼时毛左右站着的是苏共领导人赫鲁晓夫和越南国家主席胡志明，金日成是在周恩来的陪同下观礼。从今天小金的位置来看，历史高度已经超过爷爷了，加上小金如今可以拍着普京肩膀承诺永远支持俄罗斯，甚至跟普京称兄道弟，这个格局相比于金日成当年要看两个大哥脸色强到哪里去了。他爸爸更是拍马都追不上了。

这次大阅兵，东南亚国家中，除了明目张胆在南海挑衅中国的菲律宾，总理被解除职务的泰国，其他国家均派领导人或者高层政要前往北京观礼，印尼总统更是连夜赶来，中亚五国外加白俄罗斯、阿塞拜疆、亚美尼亚等独联体成员国领导人几乎全员到齐，南亚多国首次派高层政要前往北京观礼，伊朗总统首次出席中国阅兵活动。其他非洲刚果共和国、津巴布韦，以及拉美的古巴，都是中国传统 的反西方阵营的老朋友。这些国家的领导人集体前往北京就是明目张胆的抱团站队，而不是一团和气凑热闹。出席此次阅兵的领导人名单勾勒出了一个和西方平行的、以中俄为主要支撑的世界，西方国家主导的世界秩序和非西方国家形成了清晰明确的分野。而且这次遍邀国际友人出席，向世界还发出一个清晰信号，中国过去是一个成功打败日本帝国主义的大国，是战后秩序的开拓者，今天同样是拥有众多同道支持者的大国。中国过去不屈服于日本的侵略，今天敢于不屈服于美西方的压迫，而对于任何想要通过武力手段挑战中国的国家，要从经济，外交，军事等多个层面考虑一下自己的胜算。

日本外长今天就对记者说，关注中国阅兵，强调日本是个爱好和平的国家。嗯，是不是爱好和平，关键看你的肌肉能让对方冷静到什么地步。纽约时报撰文说中国煽动民族情绪，宣传反日仇日，领导想藉阅兵传递一个讯息：中国不会再受霸凌。纽约时报阴阳怪气不奇怪，如果说凝聚共识就是煽动民族主义，不忘历史就是宣传反日，那么中国还应该加大煽动和反日的力度。华盛顿邮报表示，中国试图利用川普今年上任后造成的动荡，来团结各国领导人，反对美国主导的秩序；路透社指，中俄朝巨头聚集意在挑战西方。卫报则指，中国将在反西方领导人出席的阅兵典礼上展现军事实力。法国France24撰文分析指，这次大型阅兵展现了中国 的军事和领土野心，并似乎将中国定位为西方国际秩序竞争对手阵营中理所当然的领导者，同时试图推进一种历史叙事，以证明中国对台湾的领土主张是合理的。新加坡海峡时报则指，中国举行了史上规模最大的阅兵仪式，展现了其日益增强的军事实力和地缘政治影响力，同时中国正努力塑造成后美国时代国际秩序的守护者。甭管这些海外媒体是阴阳怪气还是明贬暗骂，反正中国这一把高调的让他们想看不见都不行。

尤其是俄朝领导人由出席阅兵的26个国家领导人聚集一刻开始，到一起登上天安门城楼、接待二战老兵到最后坐下观礼，他们2人就像左右护法一样，长期处于领导身边。酸得川总发推特说，叫领导代他慰问“正密谋反美的普京和金正恩”,从气死老美这件事来看，中国是做的相当成功的。

对普京和小金来说，他俩都急需一个国际舞台，普京从上个月的阿拉斯加外交到这个月大阅兵，用两张红地毯向世界宣告，俄罗斯依然是全球最有影响力的大国之一，普京依然是国际政治领域的外交巨头，俄罗斯面对种种难题下仍然是世界一流的大国。在美、日、韩三国军事合作日益紧密的情况下，中俄朝巨头同框是一个明确的战略回应，等于向世界宣布，中俄朝不仅是友好伙伴，更是同西方世界对抗、促成多极世界体系的重要战略盟友。中国这次是搭了一个舞台，让普京和金正恩都体面地重返国际外交视野。愿意搭台给别人唱戏，这是好大哥才有的品质呀。

这次大阅兵将会是一次历史的分水岭，是中国彻底挣脱美国对二战秩序主导的成人礼，是宣告解决台湾问题已经在路上的宣言。
