
# 【米联储见闻20250519】——太白入月敌可摧

大总统带着中东大订单满载而归，但美联储主席并没有给总统好脸色。穆迪下调了美国信用评级，从3A变2A，这等于是给大总统的光辉业绩上眼药。毕竟中东土豪提着钱等着投资美国，而华尔街的专业人士却挂出了红牌警告，小心投资美国的风险。真是啪啪打脸大总统，一点都不想让总统有丝毫痛快。

上周联储局主席鲍威尔在第二届Thomas. Laubach研究会议上发表重磅讲话，首次系统性地阐释了联储局对未来经济环境的预判及货币政策框架调整方向。此次讲话不仅是对过去五年全球供应链动荡、通胀失控等挑战的反思，更为后疫情时代联储局的职能转型奠定基调。

鲍威尔为什么要在这个时候来做这个演讲？本身就是要对川普一系列操作所带来的不确定信号做正视听的解答。毕竟白宫是白宫，华尔街是华尔街，川普已经给华尔街带来不少麻烦，现在华尔街需要亮亮自己的牙齿了。这次演讲有三大要点：其一，未来通货膨胀可能更加波动，美国可能正遭遇一个供应冲击更频繁、持续时间也更持久的时期；其二，鉴于2020年以来通胀和利率的重大变化，联储局正在调整政策框架；其三，由于随后几年的意外高通胀，2020年的框架被证明是不理想的，但2%的目标保持不变。

怎么看这个演讲？一定程度上，鲍威尔承认了美联储过去通过加息降息来调节美元供应控制全球金融市场的模式正在逐渐失效。美联储需要更复杂的货币工具和货币政策框架来解决美元美债面临的困局。过去那种仅靠一点利息引导就能收割全球的好日子已经一去不复返了，美联储需要更精妙的游戏规则设计，让聪明人可以继续来收割大众。不愿意轻易放弃货币权力的美联储必须全力自救。

所以鲍威尔演讲开篇就做出了一个判断：美国正进入一个供应冲击更频繁且可能更持久的时期。这背后一个重要的认知变化在于，美联储不再能够利用过去的货币体系轻松解决美国的问题。外部冲击更频繁、更持久意味着美联储解决问题所需要的时间可能更长，因为现在的问题更复杂，难度更大。这一判断标志着联储局对全球经济结构性矛盾的认知正在发生变化，而且是一种被迫升级。背后另一个重要的信息在于，如果不升级认知，现在的美联储已经无法解决美国面临的问题和困难。

相较于2020年框架评估时对通缩风险的担忧，当前的核心矛盾巳转向供应冲击加剧、地缘政治冲突及气候危机等多重因素叠加导致的供给端脆弱性。这种转变直接体现在通缩加剧。即使长期通缩风险冲突和全球产业重新布局的大趋势下，话语权正在削弱。美国现在的一切政策选择都可能面临着杀敌自损的局面。美国也不再提哪个吹着口哨就能把问题解决的绝对强者。所以鲍威尔认为，联储局在处理通胀问题上可能将重新回到2019年之前的货币政策框架，放弃平均通胀是否达到2%的框架，转为只关注当下通胀是否达到2%。说白了，苦一苦老百姓，或许还有转机，允许经济状况在短期内承受更大的痛感，从而给美国经济修复问题换取时间窗口。

自2012年以来，伯南克领导的FOMC首次将联储局的货币政策框架编纂成文，题为《长期目标和货币政策策略声明》，被称为共识声明，设定了2%的通胀目标。这是为了解决2007-2008年次贷危机带来的三低问题。2008年全球金融危机至2020年新冠疫情之间，10多年全球陷入低增长、低通胀、低利率的三低时代。发达经济体政策患上利率病，政策利率普遍在0%附近。这被称作零利率下限。经年日久，利率病的隐患逐步显现。一旦经济出现轻微的下行，央行便放水。待到经济再度衰退，央行降无可降。在经济疲弱时。

川普上台之后，多次要求美联储赶紧降息，为美国经济发展降低成本，但是这次美联储换由关注平均通胀转变成只关注当下通胀，意味着美国经济已经彻底走出三低，而进入三高时代，所谓三高，就是高利率、高增长、高通胀。2020年5月，由强劲需求和混乱的供应链导致的美国通胀率达到6%，联储局不得不在5个月内累计加息75个基点，迅速灭火。现在美联储复盘过去的货币政策，认为是平均通胀目标的制定影响了美联储的货币政策，在加息速度上不够敏感灵活，常常滞后。而现在把目标调整为只关注当下的通胀率，则意味着美联储的货币政策会更灵活，甚至更不确定，极端情况下，一个相对于不那么确定的浮动的美元利息时代可能会随之而来。这就好比，当玩家们已经熟悉了固定靶的游戏规则，百发百中时，美联储现在丧失了嘉霍忖器嘉钳赞赁见漳所以把固定靶改成移动靶，让资本和机构在与美联储做货币博弈的过程中的难度加大，从而为美联储制定更符合美元利益的游戏规则创造可能性。这意味着未来到底是加息还是减息，判断的标准已经发生改变。

如威尔此次还明确表示等斩审视就业和通胀缺口的衡量标准，避免将率简单等同于通胀风险信号，弱化对劳动力叶协对热的敏感性，转而构建更具容错性的政以应对供给冲击导致的经芍经济数据失真。务鲍威尔指出实际利率上升不仅反映通胀波动性，更指向中性利率水平的永久性抬升，这一判断与5月会议维持4.25%-4.5%利率区间不变的决策翌辑一致，暗示联储局正为更高更久的利率环境构建理论支撑。未来降息周期或较市场预期更谨慎。说白了，市场要接受零利率时代已经结束的结论，并将长期忍受高利率货币政策带来的痛感。美元必须依赖于高利率维持自己的货币地位。市场也好，川总也罢，不要妄图在此刻拉开美元堰塞湖的大坝。

此外，鲍威尔这次还特意说明了一下到底该如何看待美联储的话术。在冲击规模更大、更分裂的时期，美联储需要更清晰地传达政策不确定性。这就是我上面所提到的固定靶变移动靶的概念。你们不要猜我的想法，我就是很随机，我就是不好猜。你们也不要押注单一的方向，我的每一次决策就是此时此刻根据当下做出的判断，所以可以是加也可以是减，而不是过去加多少的问题。留给市场的不确定性不再程度上的多少，而是方向上的不同，形同买大小一般。所以现在美联储决心抛弃自己的确定性，把面纱带上，通过不确定性来度过这个对美元、美债、美国经济来说都非常艰难的窗口期。

换句话说，美联储正从需求管理者转向系统风险缓冲器。如果说过去汝争目经济的强势状态，美联储是完全肯定的，那么现在美联储已经开始弱咚鲸总霄色的忠诚性。相比于美国经济，美元才是最后堡垒。美国经济可以暂时衰退、萧条甚至崩溃，但是美元不行。美联储现在的重要任务之一就是要在美国经济下行的过程当中依然保住美元的全球货币属性和地位。在全球化退潮与产业链重构的背景下，重塑美元的价值体系。所以美联储强调货币政策框架的灵活性与沟通效能将成为维护经济韧性的核心。通胀预期决定了美元的地位，这是联储局一切行动的基石。如何在全球秩序重构中守住这一基石，是未来五年美联储面临的最大考验。

美元是美元，美国是美国，这才是世界的真相。米糕祝各位晚安。
