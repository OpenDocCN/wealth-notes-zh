
# 罕见红利大规模到来，城市更新创造巨量财富

城市更新行动是未来几年少有的增量红利，意味着城市从“增量扩张”到“存量运营”，将取代拆迁重建，成为城市发展的主旋律。它会大幅度改变现有的房地产格局，影响巨大。

## 城市更新的背景

城市更新是对城市中的各种楼房、厂房、基础设施等进行更新改造，特别是那些使用年限长的老旧房屋、老旧厂房、老化的水管电网等。早在 2021 年，中央首次提出了要实施城市更新行动，当年批复了北京、重庆等 21 个城市作为试点；之后又有上海、天津、宁波等城市也发布了相关城市更新方案。而到今年为止，第一批试点城市的三年行动都基本结束了。

中央发布《关于持续推进城市更新行动的意见》，就是中央认为这些试点搞得还不错，所以要开始在全国范围内进行大规模推广了。这被视为政策发令枪，红利要扩大。

## 城市更新的具体内容

### 房子

针对房子的更新，重点提到了两个方向：

1. **危房拆除**：D 级危房要拆除，C 级危房要改造。建筑承包、建材供应、抗震加固、房屋体检、墙体修复、废料收回等公司，在这其中都会有机会。
2. **闲置利用**：老旧厂房、低效楼宇、传统商业设施等存量房屋要改造利用，土地性质也要跟着转变。比如北京把 798 厂等工业老厂房改成了文化创意园，深圳把旧写字楼改成产业孵化基地。

### 小区

针对小区的更新改造，提出了具体的政策方向：

1. **基础设施更新**：燃气、供水、排水、供电等老化管线都要重换，管材、线缆、管道配件、施工安装、电气设备等行业的生意会很受惠。
2. **配套设施配齐**：未来大量小区都要增设电梯、停车位、充电桩、消防设施、通信基站等，还要新增助餐、家政等公共服务设施。

住建部在今年 3 月的时候提出：2000 年以前建成的城市老旧小区都要纳入改造范围。数据显示，中国城市中，2000 年前建成的老旧小区约有 22 万个，涉及居民近 3900 万户，面积约 87.6 亿平方米。

### 城区

针对城区的更新，主要有四点：

1. **建设城市里的“完整社区”**：这是城市更新的基本单元。要完善公共服务，建设便民商业、公共活动场地等设施；构建“一刻钟生活圈”，方便居民日常生活。
2. **扩展城市功能**：建设更多的养老设施、托育设施、医疗应急服务、健身锻炼场所、公共文化空间、物资储备中心等。
3. **城市基础设施改造**：城市燃气、供水、排水、污水、供热等地下管线管网和地下综合管廊要改造，污水处理、排水防涝、防灾工程等方面要重点推进，还要推进交通线路分级。
4. **城市生态修复**：黑臭水治理、湿地保护、土壤污染修复、修复受损山体和采煤沉陷区、保护乡土植物等，全部都要齐头并进。

## 未来趋势

大规模城市更新之下，未来最重要的三个趋势：

1. **城市从“增量扩张”到“存量运营”**：城市更新将取代拆迁重建，成为城市发展的主旋律。
2. **社区经济崛起**：本地生活服务成新蓝海。“完整社区”和“一刻钟生活圈”的推进，将彻底改变社区商业生态。
3. **城市竞争转向“软实力”比拼**：未来城市之间的竞争，不再是拼 GDP 或高楼大厦，而是比公共服务、生态环境、文化魅力。

通过城市更新，地方政府将加速从“土地财政”转向“人口财政”，用更好的教育、医疗、养老配套，吸引人才定居。像成都、苏州等宜居城市的吸引力，将进一步增强。
