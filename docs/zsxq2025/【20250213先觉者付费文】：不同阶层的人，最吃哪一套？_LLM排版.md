
# 不同阶层的人，最吃哪一套？

上篇文章里，对于如何与不同阶层的人打交道，我讲了
一个策略：
跟底层谈感情，对中层讲认知，向上层摆利益。
这个策略，当然不是绝对的。
策略这个东西，是术，术有千千万万，因时因地，你觉得有用，就可以去实践；觉得无用，也很正常。

很多人看了文章后有个疑问，说利益这个东西，谁都喜欢，底层人也喜欢，只要是人都喜欢，那为什么跟底层
人不谈利益，只谈感情呢?
有这个疑问，说明看我的文章不多，不了解我所讲的社会认知五层次的系统观点。
在我的社会认知五层次理论里，包括了感官、情绪、逻辑、利益、规则。

其实，在情绪之下，还有一个东西是也有用的，那就是
感官 ( 身 体 )。
有些人什么都不吃，就吃拳头，但是呢，现在是法治社会，这个就不提倡了。
但是武力威慑，物理消灭，不管对于一个人还是一国，依然是最根本的一种力量。
在利益之上，也还有一个东西，叫做规则 ( 权 力 ) , 它比利 益更好用，但大部分人都没有，所以我也就不在这里多讲了。
这五个认知层次，对任何阶层的人都有用，只不过对不同的阶层的人，效果不一样罢了。

跟底层可以谈利益吗?
当然可以，任何阶层都有自己的利益，不管底层、中层、上层，都可以谈利益。
那为什么不跟底层人谈利益呢?
答案很简单，因为底层的利益少的可怜，几乎无利益可谈。

比如你跟客户谈生意，谈的是业务合作、利益交换，讲一个互惠互利。
底层人唯一的利益来源，就是打工挣的死工资，用以维持生存。
老板要辞退你的时候，多给你几百块钱遣散费，你觉得
这就是叫谈利益了吗?
蝇头小利不叫利益，喜欢要点蝇头小利，也不叫谈利益。
很多底层人，都是为了蝇头小利，而丢掉了自己真正的
利益。

为什么大多数时候，你的老板、领导，只会跟你谈感情，
谈做人，谈奉献?
因为跟你谈利益，你也没什么可以拿得出手的东西来交换。
你跟老板说， “老板，我不跟你谈感情，我要跟你谈利
益，你把公司的利润分我一点可以吗?"
老板估计会马上让你滚蛋。

除非，你掌握了一个可以创造巨大价值的独家技术，或者手里有一个可以带来利润的大客户，这个时候，老板
才会坐下来，跟你谈利益的问题。
因为你手里有了筹码。
所以说，跟底层谈感情，而不是谈利益，不是因为底层
不喜欢利益，而是底层没有可以用来做利益交换的筹码。
底层是一个资源稀薄、情绪浓郁的阶层，在这里利益换
不来利益，只能换来一大堆无用的情绪。
想要谈利益，前提是必须双方有对等筹码。
底层谈不了利益，因为底层缺少筹码；底层也不喜欢谈
认 知 。
所以只能谈情绪、谈感情了。
老话讲养儿防老，底层人生养子女，就是为了养老。
这是利益，但不能赤裸裸地说出来，要用感情包装一下，
称之为“孝道”。
不管父母如何，子女必须得养老，否则就是“不孝”,
这就是用情感来交换利益。

朋友讲义气，就要两肋插刀， "插刀"付出的是利益，
讲义气则是情绪感受(情感),这也是用情感来 交换利
益。
所以你看，底层充满了情绪，以及被情绪、情感裹挟的
人和事 。

再来说说上层。
跟上层谈利益，有三点需要阐释清楚，我这里具体说说，
可能比较敏感，不能完全公开：
其一，不是所有生活在上层的人，都是社会上层，他们
很多其实是社会上层的附庸。
上层人指的是哪些人?
有的人谈到社会上层，就喜欢举出一些愚蠢的富二代的
例子来抬杠，说这个富二代也是上层人，他的认知高
吗?
问这个问题的人，其实就是对社会阶层一知半解。
社会上层，是指掌握社会分配权力和管理权力的这部分
人，比如一些厉害的政界大佬、商界大佬。

出入上层，或者生活在上层的很大一部分人，并不是真正意义上的社会上层，而是社会上层这个阶层的附庸。
这个附庸群体，包括了社会上层的裙带成员、依附而来
的利益体。
这部分人，在生活上、资源上，有了社会上层或者接近
上层的水平，但认知上、权力上远远达不到。
如果他们的认知能够打通，有足够能力，能很快转化成
真正的上层。
与之相比，普通人即便认知和能力俱佳，也很难从底层
走到上层。

其二，上层人更加注重利益和规则认知，并不是没有感
情、情绪，只不过一般人提供的感情、情绪，对他们没
有价值。
有的人情绪阈值低，随便你跟他聊几句，他就很开心；
有的人情绪阈值高，就算你跪舔，也没有用。
以社会上层占有的资源数量，愿意为他们提供情绪价值、
感情价值的人太多，质量也更加上乘。
所以当你面对他们时，你所能提供的情绪价值，非常廉
价。
刘姥姥进大观园，嬉笑怒骂、丑态尽出，也只是博得王
府众人一笑。
人作为一种“东西”,既是生命体，也是资源，同样也
有价值高低之分。
擅长提供情绪价值、感情价值的人，实在是太多。
吃惯了精粮和特供的人，哪能在乎一个普通人的口粮
呢?
试想一下，如果你在路上遇到一个乞丐，这个乞丐一直
对你说好话，呲着大门牙讨好你，你是什么感觉?
你不但不会感到情绪上的愉悦，相反，你只会觉得厌烦、
受侮辱，认为这个乞丐就是为了几个钢蹦才如此作贱自
己。

所以说，人和人之间谈感情、谈情绪，前提是，你们的
价值地位是平等的，至少是接近的。
否则，感情、情绪就毫无意义。

其三，社会上层最注重的不是认知，并不是说社会上层
没有认知，而是他们可以用利益和权力来借用认知，无
需亲力亲为。
对一个人来说，认知如果是通往利益和权力的通道，那
么认知就是最重要的，这一点中层人深有体会。
但是，如果利益和权力都已经达成了，那么认知就退居
其次，如何维持和使用利益、权力，成了主要的目标。
你给皇帝讲如何成为皇帝，是没有意义的。同样的，你
给上层人讲如何成为上层人，这样的认知也是没有意义
的，因为他已经在那个位置了。
其次，当你拥有利益和权力的时候，就可以调用别人的
认知，而不必亲力亲为。
比如你给老板汇报一件事情，告诉他你的方法论、操作
细节、理论与实践的正确性，这些老板也许会听，但这
些一定不是他最想听的。
老板最想听的，是你能用你的这些认知、方法论，给他
带来什么样的结果，带来多少技术、市占率、利润。
认知只是手段、方法，最终是为了达成利益和权力，虽
然看起来重要，但其实到了一定高度的层次，也没那么
重要了。
