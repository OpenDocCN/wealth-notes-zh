
# 一个小人物之巅

前两天我分享了一首歌词，留言区里，好几位朋友问我是不是疯了？一开始我没弄清楚什么意思，众所周知，我目前旅居温村，白天洗碗，晚上滴滴，从白到黑昏天暗地，可能好多天都见不着新闻。直到手痒了，拂去键盘上的灰尘，把几个热门网站打开浏览一番，才明白那几位朋友问我疯了是在关心我，因为最近，出了两位疯人。

一位姓李，是位姑娘，事发在江西南昌。话说这两年江西的奇葩事儿挺多的，上次回北京，我还特意去吃了一家江西菜。说实话，油大辣多，很不对胃。李姑娘的事儿我就不评论了，刚刚南昌方面发出了第二份通报，全文很长，我完全相信。

另一位传说疯了的，才跟我多少有点关系。其实关系还不小，此人年龄与我相仿，经历与我相似，都能靠码字获得些名气赚得些银两，人生的1/3是相似的。不相似的是，我的名气比人家小得多，我靠码字赚的碎银子比人家少得多得多；更截然不同的，同是二十来岁进体制，我二十年前就飘然离开了，而当年明月老兄，竟一直在体制内如鱼得水。

这是我完全可以理解并早就已经料到的。我没怎么读过他的书，《明朝那些事儿》，我应该是在书摊前翻过，只是没买过也没通读过。但其中的意思，我大概能了解，因为我见过当年明月，还跟他聊过一会儿天，聊的当然是历史。同是历史非专业研究者，水平一个是半瓶醋一个是二两酒，只需聊一会，几段对话，我就能明确知道他跟我不是一路货色。

他沿续的是一种雄主、明君、能臣、志士的思维模式，这种模式不好也不坏， 一堆人挤在故宫办公室里斗心眼，有意思也没意思。当年明月觉得有意思，就一直钻了进去；我觉得没意思，早早抽身出来。

我大体记得那次聊天，我只请教了一个问题，明朝亡于谁之手？这个问题有很多层次，多数说李自成，少数说皇太极，一般的说亡于党争，不一般的认为责责任全在崇祯……当年明月是专家了，他的回答自然比不一般更不一般，他说，明朝亡于万历。万历执政长达48年，懈怠懒惰不上朝，把大明的底子都给掏空了。明亡于万历，在史学界也算是一种共识。当年明月沿续这一共识，说万历本有个很好的开始，在他即位的前十年， 一代能臣张居正辅政，锐意进取，开拓创新，搞出了著名的“一条鞭法”,大大提升了行政效率，从中央到地方政令畅通，税收大幅增加，国库逐年充盈。只要万历按照张居正的既定方针走下去，明朝不仅不会亡，还会比朱元璋时代更强大。

听到这个答案，我明白当年明月老兄的历史研究没入门。

明朝之亡，亡于张居正。

在做进一步解释之前，我要先抬出一位史学大家，吕思勉先生。

我是读吕先生的书入门的，他写西汉的章节里，评价了另一位著名能臣霍光。

在历史上，霍光是比张居正更牛逼的辅政大臣，他开创了昭宣之治，后世将他与远古名臣伊尹并称，谓之“伊霍”。相比之下，张居正是没开创什么之治或什么中兴的。

吕先生评价霍光，只用了一句话，就道出了其奥秘：
“霍光秉政，鉴于武帝时天下疲敝，颇能安静不扰，与民休息。”

我来狗尾翻译一下：那霍光没什么了不起，他就是个富二代，吃吃喝喝，不爱折腾事。只要不瞎折腾事，大汉的百姓是地球上最好的百姓，勤劳、老实、安分守己，不出几年，天下就能恢复。

要知道，在霍光秉政之前，是汉武大帝时代，那可是个雄主，干的事情有多么伟大我就不赘述了，请参见58集电视连续剧《汉武大帝》。

只是有一点，他干到后来，为了从百姓口袋里多捞钱以继续他的千秋伟业，连“算缗法"和“告缗”法这么缺大德的玩意儿都弄出来了，鼓励百姓互相揭发偷税行为，举报者可以得到违法商人一半财产做奖励。

你想想，用这一套搞一遍，天底下还存在有钱人吗?

所以汉武帝挂了后，霍光及时与民休息，向雷军同志学习，能不管就不管，能少管就少管，让天下人自己发挥聪明才智吧，霍光乐得清闲。

OK, 这是一脉智慧，是传承自老子的中华传统智慧。只可惜的是，这一脉智慧不怎么常用，两百年也难得遇见一个。而当年明月所称颂的张居正，不仅不属于这一脉，反而是站到了这一脉的对立面。

张居正搞的新政，一条鞭法，到底是什么?很简单，两手抓。一手抓考核，整顿官吏，严肃纪律，月有稽，岁有考，大大提高了各衙门的办事效率，实现中央到地方政令畅通。另一手，改变税收制度，用银子代替实物征税，方便税收完成。同时丈量天下土地，将纳税土地面积从四百多万顷增加到七百多万顷，财源几乎翻了一倍。

这些办法，都不是与民休息，都是在为明朝之亡埋雷。打通行政效率，重构税收体系，简单征税办法，这一条鞭下来，在张居正的任期内尚可以节制。但他一死，接下来的那几位，可就不那么自觉了。

以前的皇帝不自觉，想收税，但行政效率低，令不出紫禁城，想收不一定收上来。所以哪怕在张居正之前出过几任胡闹的，比如明武宗之流，任他在紫禁城闹闹，对整个大明影响不大。

但配备了高效行政和便捷税收的“一条鞭法”，就如一根长鞭握在了皇帝手中，指哪儿打哪儿，想怎么收税就能怎么收。各色税种就纷至沓来了，万历搞的“矿税”，就把大明折腾个半死，然后是“辽饷”、“剿饷”、“练饷”，只要有事，先征税。税征得越多，就越出事。

明朝由此陷入一个死循环，你说，明之亡，肇始者是不是张居正?

这一点，吕思勉先生早就指明了，但他属于旧学者，一点儿都不受重用。
