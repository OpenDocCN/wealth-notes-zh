
# 1967真假麻匪火拼：再晚五分钟，张麻子就成了俘虏

## 前言

这篇文章是关于1967年真假麻匪火拼的历史事件，重点讲述武汉的“百万雄师”事件。这一事件非常重要，能够帮助我们理清一些关键问题，且不影响主线剧情，因此先进行介绍。

那一年是公元一九六六年，麻子宣布了他伟大的设想——人人都能参与麻匪、人人都能成为麻匪。但实践起来出现了各种问题，比如真麻匪和假麻匪的混战，假麻匪还破坏运动。麻子原本认为群众的事由群众解决，真理不辨不明，群众也会在大鸣大放大辩论里成长。结果发现越来越不对劲：

1. 麻子强调要文斗不要武斗，但武斗屡禁不止。
2. 麻子发现有人制造了一批“假麻匪”——扛着红旗反红旗。
3. 麻子三令五申政府军队不能干涉群众运动，结果发现有人在拉偏架。
4. 有人明明想当真麻匪，却莫名其妙地加入了假麻匪；或真麻匪组织被官僚诱导成了假麻匪组织。

## 事件背景

在不可遏制的武斗中，假麻匪有人暗中帮忙，他们使用军队武器，甚至现役军人直接加入假麻匪参与。真麻匪反制，抄了假麻匪的军械库，缴获了一批武器。结果有人鼓噪，说真麻匪严重威胁到鹅城秩序，必须让军队介入剿灭。

这个现象在重庆和武汉最为严重。那一年的七月十日前后，麻子和工作组先后前往武汉考察假麻匪问题，但这一去就险象环生。

武汉的假麻匪自称“百万雄师”，在麻子与武汉调查期间，兴风作浪为所欲为，不断攻占真麻匪的据点。最严重的一天是七月二十日，百万雄师包围了张麻子的行营——武汉东湖宾馆。这种行为在历史上有一个专有名词——兵谏。

## 兵谏事件

当时麻子身边的人都非常紧张，人潮已经包围了宾馆，别说有意伤害，就是搞一点踩踏事故都要了命。夫人劝麻子赶紧走，说局势已经失控，你的安全完全没有保障。麻子很生气，说我不走，我不信人民会伤害我。他倒要看看是谁在背后鼓动人民、误导人民。

麻子的脾气倔强，有时有点小孩子气。电影《平津战役》里有个桥段，因为华野某部不听命令擅自进攻，导致傅作义王牌军如惊弓之鸟跑回张家口，麻子气坏了，但他生气的方式是惩罚自己，穿着单衣站在雪里，说你们都不听我指挥，那我就把自己冻坏给你们看！又倔又萌。后来翔宇亲自出马，温情脉脉地把大衣披在麻子身上，好劝歹劝，才把麻子劝回指挥部。

所以麻子真发倔了，九头牛也拉不回来，但形势又万分危急，夫人只好找到老二，让老二去劝麻子。结果老二真劝动了，麻子从东湖宾馆走到东湖码头坐船离开，刚刚走到码头，“百万雄师”的人潮就冲进宾馆了，第二批撤离的工作人员不同程度地受到了伤害。

这是一次极其严重的政治事件，甚至都不需要对政治有多了解都明白这性质有多恶劣。麻子为什么开始不走呢，因为他相信群众是被煽动的，群众不可能有心伤害他，那么他倒是要看看是谁在背后煽动的，所以倔脾气上来了。为什么老二一来劝麻子就走了？大佬文集圈因为他冷静下来了，知道这事不是老二策划的了——所以夫人也聪明，知道找老二来劝得动(也是有当面表态的意味)。

## 事件调查与结果

为什么麻子会怀疑老二呢？因为武汉是四野的地盘。这个是有说法的，解放军解放了一座大城市，就要在大城市留下干部，最典型的三野之于上海、二野之于南京，就都是属于他们的“地盘”，陈和刘也分别是上海和南京的第一任市长。这还有另一个更通俗易懂的说法叫“山头”。不过中央肯定非常注意削平地方各种山头，彼时武汉军队一把手也不是四野的人，但武汉终究是四野的“地盘”，麻子第一时间怀疑老二也是情理之中。

还有一个很大的背景是，麻子和老二已经产生了嫌隙了。麻子请老二出来掌军，是想制衡官僚，是想让老二打果，但老二虽然是军神，打仗无敌，搞政治却没那么应手，至少搞不过那些摸爬滚打多年的老油条。老二想打鬼，就必须搬出麻子这尊神来，所以把麻子越吹越大，有时候搞得麻子很被动。此外老二也确实有些膨胀，有些事情做的不地道。比如任人唯亲——尤其是他老婆和儿子；任人唯亲不可怕，关键是任的人普遍水平比较低，再加上老二身体不好，经常卧病在床，于是那些“亲”就胡搞乱搞，成事不足败事有余。你说老二有没有责任，当然有责任啦。再加上老二也并不是心眼特别大的人，膨胀了之后挟私报复的事也有，所以很多地方麻子就很被动，因为为老二最终背书的那个人就是他。

最著名的就是《滴水洞家书》麻子直言“我是钟馗，有人借我打了打鬼”还有那句最著名的“我是要跌的粉身碎骨”这句名言也是出自此处。这篇家书就是批评老二的，让夫人呈给了老二；老二表示悔改之后，麻子就销毁了这封信，因此现在只有只言片语流传。

就是到了武汉，麻子也发作了一次，在的东湖宾馆当着老二的面，摘下了搞个人崇拜的标语，并且骂了一句“这些王八蛋东西”。大佬委屈麻子自己心里门清，那他当神打鬼，只会让自己越来越被动，“捧得越高，摔得越惨”。不过他就算批评老二，还是把他当亲儿子来看的，不是敌人的批评，而是恨铁不成钢式的批评，跟在井冈山时一样。

这事就调查，确实跟老二没关系。最后查到武汉军区 leader 为止了——我们也给他取一个花名就叫陈道道吧。也确实有充分的证据证明百万雄师事件就是陈道道一手策划、煽动的，意在围攻麻子行营展示肌肉、制造政治压力。那么陈道道背后有没有人指使呢？大家普遍对此持怀疑态度，因为他就是一大老粗，怎么想的出真假麻匪火并的套路来？此外百万雄师事件有一个人是既得利益者——即对麻子示威，又敲打了老二，可谓一石二鸟。但是这种猜测在历史上没有充分的证据，没有证据的事我们就不会写。

## 事件结尾

最后再说一下这件事情的结尾，看一看麻子的菩萨心肠。按理说这种兵谏围天子行营的事件，在古代是要诛九族的；别说古代了，你就是搁斯大林遇上这种事，你有几个脑袋我给你配几梭子子弹。但是对陈道道的处理仅仅是撤职而已，麻子自己生怕弄得扩大化，特别强调“陈道道我是要保他的”。这就是麻子的处事特点，雷霆手段、菩萨心肠，他相信自己同志本质是好的。

然而故事还没有结束。麻子走后兄弟们去了上海去了浦东，开始过起了吃着火锅唱着歌的生活。老干部嘛，毕竟在嗡嗡嗡期间受了那么多苦，所以用宝贵外汇来买进口小轿车是可以理解的，所以盖几座小洋楼享受一下生活也是可以理解的，所以要睡几个女学生女服务员也是可以理解的。

而陈道道则是“可以理解”中的典范，这位仁兄为了给自己盖小洋楼，下令推平了一座幼儿园，并消耗了几十万宝贵的外汇小洋楼装备现代化设施。这件事情在当时引发了巨大的反响，一位诗人专门创作了一首诗《将军，你不能这样做》，很快风靡全国：

在这新长征的路上
且听前进的后人
和前进的法律一道
大喝一声：将军
不能
这样做!

只能说啊，麻子菩萨心肠，但是兄弟们真的不是很给力啊。
