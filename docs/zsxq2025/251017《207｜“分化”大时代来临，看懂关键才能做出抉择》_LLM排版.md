
# “分化”大时代来临，看懂关键才能做出抉择

## 从“空间套利”到“时间套利”的本质转变

过去几十年的大增量时代，就像一座大厦里所有或者至少大部分电梯都在上升，不管你个子高矮，只要坐上电梯，大部分人都能随着电梯往上走。而现在还是同一座大厦，里面的电梯状态则是发生了改变：有的电梯继续快速上行甚至飞快上行；有的人没坐上电梯要走楼梯，虽然有点喘，但是还能往上走；有的则是被困在电梯里不动；甚至有的人发现，自己处在一个下行的电梯。这时候，你努力做出正确判断，选上或者挤上一个上行的电梯或楼梯，就更加重要。

要理解这种外界环境的变化是怎么发生的呢？首先要理解过去几十年我们增长的底层逻辑是什么。我称之为“空间套利”，所谓“空间套利”，就是利用在进程、成本、城乡资源上的巨大“势能差”，来获取增长。具体表现为：

第一，进程势能差：就是打开国门，将处于更优势发展阶段的国外的技术、资本、管理模式引入中国，与国内庞大的劳动力、土地等要素相结合，实现了“后发优势”。

第二，成本势能差：主要是指沿海地区率先发展，当成本上升后，产业再向内陆、向西部转移，不断寻找新的成本洼地。

第三，城乡势能差：城市化进程，本身就是一场巨大的空间套利。大量农民从低生产率的农村，进入高生产率的城市，释放出巨大的消费红利和增长红利。

现在看，在这几种势能的叠加下，增长是相对容易的。因为“势能差”客观存在，你只需要去“连接”和“搬运”，就能创造财富。这就像水往低处流，是自然规律。换句话说，过去是“水涨船高”，大家都能分一杯羹。

然而，当下一些地方和领域呈现出了“收缩显性化”现象，这背后的趋势之一，就是“空间套利”的红利正在减少——包括国内外发展的进程势能差在缩小，国外优势技术的扩散红利在逐渐耗尽，我们必须走到技术前沿，开展“从0到1”的原创。沿海和内地之间的成本和环境优势，也在缩小，甚至很多低端产业，不得不外迁国外；而城市化进程，也开始放缓。

当“空间”上套利的可能性日益减少，持续增长的关键动力从何而来？我的答案是，转向“时间套利”。所谓“时间套利”，就是投资未来，用当下的投入，换取未来的高回报。它不再是主要依赖于外部的“势能差”，而是更多依赖于内部的“创造力”。

具体来说包括下面三个方面：

第一是投资科技——研发今天不太赚钱、但明天可能占据市场的技术。

第二是投资人才——培养和吸引能够创造未来的顶尖大脑，和系统的科技和产业人才链。

第三是投资效率——通过数字化、智能化，提升现有资源的利用效率。

根据我的观察，“空间套利”现在越来越像“存量转移”，而“时间套利”则是“增量创造”。我认为，这场从“空间”到“时间”的转移，是分化的关键原因。那些能够成功转向“时间套利”的城市、企业和个人，将进入“进化”通道。他们不再挤在争抢存量的池子里，而是在创造新的增量，这群人是未来的定义者。
