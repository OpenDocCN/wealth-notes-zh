
# 未来的“美洲世纪”

毋庸置疑的一点就是，到今年为止，逆全球化趋势的基本进程已经持续了大致十年之久。未来，全球经济发展格局的进一步分化已经是不容置疑的事实。就未来在全球经济发展格局中能够居于领先地位的国家和地区而言，评论界已经形成许多不同的看法。有观点指出，凭借着年轻的人口结构和日益增强的经济实力，印度未来在全球经济格局中的地位会进一步强化；也有观点指出，凭借着类似的优势，以越南为代表的东南亚地区有望成为未来全球经济格局中的关键一极；甚至也有观点看好整个中亚五国未来的前景。对此，美洲地区似乎在一定程度上被评论界给忽略了。

不过，安邦智库（ANBOUND）资深研究员多年来一直表示，就未来全球经济的基本格局而言，“美洲世纪”的可能性不容低估，以阿根廷、巴西和墨西哥为代表的拉美国家和以美国、加拿大所组成的北美国家将会在全球经济格局中占据更为重要的地位，拥有更为强大的影响力。

安邦智库在之前的多篇分析中都指出，由于过去偏重于经济层面的全球化的片面性和肤浅性，在面临自然灾害和地区冲突等突发情况时，过长和过于复杂的全球产业-供应链显得无比脆弱，而全球贫富差距的日益深化也使得各国内部对这一片面且肤浅的全球化进程日渐不满。因此，区域化将会是未来长期的发展潮流，美国的主要注意力和资源投入都会逐步向地理位置更为靠近的美洲大陆倾斜。

近期，在俄乌冲突等事务上，美国实质上采取了将欧洲抛在一边的单边主义做法，在贸易、投资和防务开支等方面，美国和欧洲之间的矛盾在一定时期内恐会持续激化，加上美国保守主义势力兴起之后与欧盟建制派之间意识形态对立的加深，美欧关系的冲突面正不断强化。并且，千禧年之后，受制于自身结构的臃肿和法律规制的绵密纷杂，欧盟的经济发展活力日渐衰减，在尖端科技研发等方面逐步落后于中美两国。美欧矛盾的强化和欧盟自身经济活力的减弱恐怕会弱化欧盟这一美国资本长期以来的首要海外投资地的重要性。

就整个美洲而言，美国最主要的任务就是从意识形态和经济事务的角度对整个美洲大陆进行有效的管控和整合。在北美地区，美国和加拿大的经济实力自不必说，虽然在贸易纠纷等具体问题上，加拿大已经和美国之间产生了十分明确的对立，但是在意识形态上，加拿大与美国分歧甚小，在经济事务上又存在着对美高度依赖，其自身最终全面纳入美国的地区战略规划只是一个时间问题。因此，美国最主要的任务是整合整个拉美地区。

就发展经济和进行战略整合而言，拉美地区对于美国有着十分重要的意义。

拉美地区的矿产资源十分丰富，其拥有世界上至少 65%的锂储备（集中在智利、阿根廷和巴西）；拥有 49%的银储备（集中在秘鲁、智利、玻利维亚和墨西哥）；拥有 44%的铜储备（主要集中在智利和秘鲁，墨西哥次之）；拥有33%的锡储备（秘鲁、巴西和玻利维亚）；拥有26%的铝土矿储备（集中在巴西、圭亚那、苏里南、委内瑞拉和牙买加）；拥有 23%的镍储备（集中在巴西、哥伦比亚、委内瑞拉、古巴、多米尼加）；拥有22%的铁储备（集中在巴西、委内瑞拉和墨西哥）。

此外，拉美地区还拥有十分丰富的油气资源。充分掌握并有效开发这些资源，对于美国的战略资源自主和摆脱对华依赖而言，有着十分重要的意义。

并且，拉美地区还构成了非常庞大的消费市场。拉美地区总人口约 6.65 亿，占世界总人口的 8.3%。该地区人口数量明显集中于几个头部国家，巴西、墨西哥分列世界第七和第十人口大国。同时，拉美地区的人口结构较为年轻。相关数据显示，巴西、墨西哥、哥伦比亚、阿根廷、秘鲁、智利六国中，15 岁以下人口占比约为 24%，15-65 岁人口占比约为 65%，人口结构呈年轻化状态，拉美人口性别比例也较为均衡。这几年，拉美地区整体的经济形势为波动型增长，消费潜力相对理想，巴西和墨西哥则被视为最具增长潜力的市场。这些都能够对美国未来可能的资本投入提供相对理想的回报。

还需要看到的一点是，虽然拉美地区部分国家有内部政治不稳定等一系列问题，但是，就全球范围而言，拉美地区的整体形势依然是相对理想的。拉美地区并不存在某些东南亚国家内部严重的军阀割据问题，其犯罪集团的活动-破坏能力也远不及中亚-中东地区的伊斯兰极端势力。拉美国家的内部问题主要是政府基本治理能力欠佳造成的，这些问题的可补救性远高于中亚、中东和东南亚的部分主要国家。
