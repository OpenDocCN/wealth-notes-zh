
# 学习笔记，2025.7.17

1，那群北大清华的孩子宴请老师，让我给当主陪，让我把老师还给喝醉了，席间我问了老师一个问题，150分的题，你能做多少分？他说 120 分，他教了20多年数学。我接着很诧异，这么少？他跟我讲，大部分孩子是考不了 100分的，而尖子生的140 或150 是怎么考出来的？天赋！好学生一定是比老师强的。我又问了他一个问题，我儿子学习中等偏上，若是强行塞进实验班是否可以？他认为不可以，因为实验班的老师，焦点只在10个孩子身上，只讲他们能听懂 的，剩余的孩子，完全靠内驱力以及同学氛围，至于说指望老师照顾？因为你不 是天赋型选手，老师看不见你，除非你主动去普通班，去当鸡头。在普通班里，这个老师还谈了一个点很有意思，每个班里差不多有 10个摸鱼的，10个努力追赶但是完全跟不上节奏的，他的观点是应该完全按照成绩来分班，让差孩子跟差孩子一个班，这样的结果就是差孩子里也有 10%的拔尖生，形成一个新的正态分布，一群原本不被重视的差生，成了新的好学生。

2，初代网红都比较通透？譬如木子美，芙蓉姐姐，凤姐。木子美，她股票持仓1000多万，她懂男人，懂女人，懂心理，懂写作，她自己都调侃，自己怎么成名的？因为滥交+随手记录，我觉得如此豁达的女人真的少见，你知道网红为什么能成为网红吗？是因为他们足够真实，真性情。今天的头部网红，其实也都是通透系列，有些时候，大家是为他们的表象所迷惑，例如快手的那几个头部网红，二驴、散打，你以为他们真的如直播时的那么愣头青？其实一个个都是高情商选手，大浪淘沙出来的，亿里挑一，这么说吧，普通人干县长，一天都干不了，普通人干网红，也是一天都干不了，山上，风大，一般人压根站不住。网红的脑回路跟正常人是不一样的，网红是天生的美特斯邦威，不走寻常路，他们更真实，看待问题的角度也不同。

4，工作的缘故，经常能接触到实习的00 后，在他们身上，能感受到一丝家庭的厚度，例如咱，都是地瓜堆里爬出来的，没有任何底蕴，父母一穷二白,。但是，他们多是富裕家庭出来的，要么就是书香门第，能感受到家庭厚度在他们身上的印记。

5，他说，真金白银是最考验人的，也是最训练人的，我觉得一定要区分学习与学问，学习对应的是什么？习惯，就是我们学习的目的是为了培养或改变我们的习惯，若是没有达到这个目的，说明我们的学习是没意义的。学问对应的是什么？是课题，是题目，是解决问题，也可以说是提出问题。日常，我说跟着一个人学到了很多，其实就是学到了一些具体的习惯，例如你今天分享给我的饮食结构，我就觉得特别好，我就会加以引用，前段时间几个学霸来玩，一个在常春藤读书的跟我分享了40分钟闹钟法，我现在就是严格执行这个，早上起来第一件事，先定闹钟，响了一次，立刻起身，先提肛30次，然后去跳绳，再俯卧撑，再农夫行走，然后洗手，再坐下工作，就是整个训练不需要单独拿出来时间来，不知不觉中就完成了10组综合训练。我现在的观点是，若是学习没有变成具体的习惯，就等于什么都没学到，人与人 的差距就是习惯，我就是在不断的分享一些优秀的生活习惯、思考习惯，例如如何安全行车，如何处理交通事故，如何分餐，如何长头发，是生活里的点点滴滴，包括不拿不占。健身房里有位老大哥，他说很喜欢看我文章之类的，我就问了一句，您网名叫什么？他说了以后，我看偶尔还留言，但是后台显示金额为0，我就在想，这就是标准的山东人，他该学的没学到，不该学的学到了，什么是该学的？文章是有偿阅读的，这就是基本的契约，你看，就要付，山东人一般怎么想？咱俩这关系？看看不是给你面子吗？所以，偶尔有人发信息找我办个什么事，我一看他名字后赞赏金额为0，直接就PASS掉了，不合格，要是我？要么我不看，只要看，我肯定尊重游戏规则……
