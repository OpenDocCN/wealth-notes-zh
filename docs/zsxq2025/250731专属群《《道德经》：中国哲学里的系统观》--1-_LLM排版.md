
# 《道德经》：中国哲学里的系统观

《道德经》是春秋时期老子的作品，是全球第一个轴心时代的智慧结晶之一。它不仅影响中国，也启发了黑格尔、叔本华、海德格尔、托尔斯泰、罗素、维特根斯坦等西方思想家。

近年来，《道德经》成为最热的传统哲学文本，出现在课程、书籍、播客和语录卡片上。重读《道德经》后，我发现它是一本基于系统论讨论世界观的哲学著作。老子将宇宙视为一个系统，探讨其运行范式以及个体如何与之相处。

## 刍狗的觉悟

《道德经》第五章有句话：“天地不仁，以万物为刍狗；圣人不仁，以百姓为刍狗。”这里的“刍狗”是古代祭祀用的草扎小狗。“不仁”在《道德经》的语境里并非恶意，而是无意，不是刻意去仁爱偏私。万物和百姓在宇宙系统中只是服务于整体目标的工具。

宇宙系统的基本目标是长久存续，周而复始地运行。在这个目标面前，天地不偏袒王富贵，也不特别为难他，只是把他当成系统稳定运行的支持元素。大自然下雪可能对农户有利、对牧民有害，但大自然是无意识的。

“圣人不仁”同理。圣人作为系统内的觉知者、执行者，做什么也不是情感用事，而是用行动去维持系统的稳定运行。天地和圣人一边不仁爱万物和百姓，一边也不仁爱自己。第七章马上就说：“天地所以能长且久者，以其不自生，故能长生。是以圣人后其身而身先，外其身而身存。”天地能长久存在，是因为他们不把自身利益放在前面。圣人把自己置之度外，反而保证了自身的存活。

这像不像我们说的系统和主导者的关系？系统存，主导者才能存；系统崩，主导者也崩。所以保护系统，就是在优化自己的生存环境。

老子非常提防和警惕“仁”这件事。他在第十八章说，“大道废，有仁义”，意思是当世间开始推崇仁义的时候，说明“大道”已经没了，变成了乱世，仁义变成了值得强调的稀缺品。基于这个逻辑，我们就能理解，为什么于东来做生意会被攻击、大冰开直播间帮民众抒怀解困也会被攻击，因为“仁义出”的背面是“大道废”。

在我们今天的社会，搀扶和救助摔倒的老人，变成了值得敬佩的事；胖东来用透明的利润率卖质量合格的产品，就能被民众点赞。这不是反过来说明，在整体上，这类义举已经不再是社会的普遍现象吗？

## 命名结构

《道德经》开篇的第一章第一句是：“道可道，非常道；名可名，非常名。无名，天地之始；有名，万物之母。”

“道可道，非常道”意思是“道”可以被总结，但能被总结出来的道，不是恒常的道。老子认为的恒常之道，是宇宙系统的终极运行规律。他说，道其实有四种：人道、地道、天道，最后才是大道或者叫常道。人搞不定最后这种。人为什么总结不出宇宙终极运行规律？举个例子，一个人、一个人类组织，再有智慧，能够参透生命的所有秘密，也无法了解生命对面的“死亡”是怎么回事。死亡所对应的宇宙规律，人就是总结不出来。

柏拉图有个著名的“洞穴假说”，也是在表达类似的理念：一群囚犯要是一生都被锁在洞穴里，只能面对一堵墙壁，只能看到身后各种事物在墙壁上的影子，他们就会以为，这些投影就是“现实”，但实际上，这却仅仅只是真实世界的降维投射。

放在系统观里，就是普通人无法理解时代的全景，也不能跳出时代的局限。

“名可名，非常名”又有什么玄机呢？我的理解是，事物可以被定义、被赋予角色，但角色也会变化，事物不存在永恒绝对的系统身份。

举几个例子，“福报”曾经是褒义词，但前些年它跟 996 联系起来之后，就有了阴阳怪气的色彩。相反，“发疯”曾经是负面词汇，但今天却被年轻人用来表达不内耗、做自己。今天西方的左翼自由派也在保守化，他们在全球政治光谱里也变成右翼了，政治上的左右之别也是相对的，也不是“常名”。

把事物的名字、身份给定义出来，这个动作有个更重要的意义，就是把一件事物跟其他事物区分开来，让它独立存在。

“无名，万物之母”，是说万物的母体是混沌状态，不过万物还没有被区分、定义出来，里面没有名字、没有结构，也就没有分别心，整个系统绝对对齐。

而“有名，万物之始”，有名字，就开始有了区分，万事万物就独立于混沌存在了。你肯定很熟悉那句“道生一，一生二，二生三，三生万物”。道就是无，道生一就是无中生有，是从 0 到 1，比如生成式人工智能。AI能写小说、能画画，无中生有。

《圣经》写创世纪，也是起初一片混沌，神创造世界的动作是说要有光，要有昼夜，要有陆地和海洋，要有草菜树果。你看，也是在起名字。

全球近代从手工业到 AI 时代，共发生了四次工业革命，为什么叫工业“革命”？因为一代一代的技术突破都改变了人类社会的运行范式，重构了系统。任正非重视基础科学，华为每年要投入 30 亿到 50 亿美元，去跟高校院所共同研究那些看似无用的基础理论，背后的逻辑也是重新回到“大道”里，寻找能够建构新范式的全新技术思路，也是在求解“从 0 到 1”。

新名词不断出现的过程，也就是系统不断出现新角色、新结构的过程。那系统为什么要做“命名”这个动作呢？我们也可以认为是为了反熵增。为了方便进行价值排序，好分配资源。分类，分工，排序，就是在形成范式。“一生二，二生三，三生万物”，也就是系统范式的细化过程。“灵活就业”“直播电商”“新能源汽车”“三折叠手机”被定义出来，都是为了区别于原来的整体，来争夺生产资源和市场资源。

新的分类出现，让系统内有了分别心，有了结构，有了利益和价值排序，也就有了博弈和冲突。

## 对齐主系统

《道德经》这句话：“域中有四大，而王居其一焉。人法地，地法天，天法道，道法自然。”它说，宇宙中有四种“大”：道、天、地、人。而人之“大”，就是王的治国之道，是通过对其他“三大”的学习形成的。人学地，地学天，天学道，而“道法自然”，道不学任何人，它效法它自己。“自然”就是“它自己的样子”。所以我们说的大自然，也可以翻译成“宇宙运行的样貌”。

这种从人到地、从地到天、从天到道的超级对齐，能实现宇宙的熵增最小化，让“天长地久”、大道永存。

## 系统的调节回路

“天之道，损有余而补不足；人之道则不然，损不足以奉有余。”这句话出自第七十七章。你肯定也很熟悉了：天之道，是大系统的调节回路；人之道，是子系统的增强回路。

大系统为维持平衡，倾向于“削峰填谷”；而小系统为了自身扩张，往往“强者恒强”。“人之道”正好对应西方的马太效应。举个例子，一个人越爱思考，脑子就越好用，越容易享受到思考的快乐，变得更爱思考。反过来，一个人不爱思考，就难以感受思考的快乐，也就更不爱思考。

那大系统怎么来建构调节回路呢？要靠“反者”和“弱者”。

## 系统的反作用力

“反者，道之动；弱者，道之用。”这句话体现了系统论里的反作用力与调节回路。反作用力、反效果，是“大道”用来维持循环运行的动力。系统通过调节回路，制造波动和反转，让整体进入新的平衡周期。弱者、弱势方，是大道的作用工具，承担着启动系统下一轮更替的使命。

在人之道里，人越爱思考就越沉迷思考，但代价可能是久坐导致肩颈腰椎出问题。来自思考之外的副作用和反效果，会逼得读书人不得不少读点书，出门跑跑步、打打球，这是天之道的调节回路。

再看阿里巴巴。阿里电商业务强到一定程度，试图全面升级天猫来提高利润率，结果引发商家与消费者的怨气和反对。这时候，拼多多悄然崛起，以弱小的“边缘创新者”的身份挑战了阿里的电商版图。过程中，阿里一度轻敌，以为拼多多是在替自己培养用户，没想到拼多多却借助需求方的力量完成了自我增强，打破了垄断，开启了电商行业的新周期。这就是“弱者，道之用”。

## 总结

我对《道德经》的重新理解，就分享到这里。你会发现，我们聊的五句话背后，藏着一个完整的系统模型：从混沌生成，到命名结构；从对齐主系统，到接受反作用力；从分化博弈，到自我循环。

也许，我们除了把《道德经》当作哲学经典，还可以像阅读一份宇宙操作指南那样，从系统角度重新理解它。

今天的人们想在《道德经》里找到怎样的指引？我猜，人们多少会认为，一个能流传两千多年、能跨文化广泛传播的文本，一定有穿透情境的解释力。它或许能在今天这个变局时代帮我们重新理解自己与世界的关系，重新建构信念和意义。

人们在书里找到的答案也许各不相同。统治者读它学治国，修行者读它修身养性，世俗人读它看处世智慧。但我们多少可以揣测说，《道德经》所描绘的宇宙运行法则，提出的人与系统关系主张，击中了当代年轻人的心。
