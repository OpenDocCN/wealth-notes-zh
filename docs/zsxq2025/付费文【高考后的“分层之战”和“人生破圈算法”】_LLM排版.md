
# 社会的层级

1. 上层：高护城河生产资料和权力所有者。
2. 中层：小型生产资料所有者和有产打工人。
3. 基层：无产者。

上层对基层，最在乎的是稳定，是局面，是给予希望。上层对中层，最在乎的是安全距离。护城河与保护层的设立本质靠的是“生产资料”，方法是通过“制定规则”。你要按照规则去玩我设定好的游戏。这是上层最优的算法。

中层对基层远没有上层对基层那么友好，因为他同样希望自己的阶层有一个牢固的护城河，下面的人不要威胁他的生态位。中层最希望这个社会普通的学校每天上半天，孩子们天天玩，好学校和坏学校拿钱隔开。中层对上层，是讨好的。因为规则是人家定的，因为海量的生产资料是掌握在人家手上的，所以要争取好感让自己成为广大阶层人群中的幸运者。

基层这个群体，由于基本没有掌握生产资料和基础物质的竞争力，所以眼界和个人能力上是相对来讲最弱的。有出息的前置因素是接纳现有的，看明白长期的，一步步去实践它。抱怨和愤怒一丁点意义没有。基层不玩了就坏事了，基层如果无序，那金字塔就该塌了！一定要有一个机会是公平的，是能让人看到希望的，是让整个家庭为之努力好多年的。

# 高考的本质与重要性

当时开高考推动会的时候，最开始的16字方针是“自愿报考，领导批准，严格考试，择优录取”。小平同志把“领导批准”给划了，变成了“自愿报考，严格考试，择优录取”。

国家要你学习那些知识是为了啥呢？

1. 通过学习这些知识的过程，培养你独自思考和解决问题的思维能力。
2. 筛选，把孩子们分层筛选出来。比如说，孩子们走入社会后就能发现，很多岗位明明是个人就能干，但凭什么你去干呢？人家是博士生，你是高中生，你还有啥话说呢？

前面我们说了，基层一定要看到希望，希望的产生来源于公平。所以我国应试教育阶段最本质的核心，是公平。在国家能够提供的有限高等教育资源下，将三六九等所有家庭当中“不差的孩子“筛选出来，给子一个更高的平台，防止未来社会分层。国家需要给基层一个途径，能够最大程度上的接近中层的门槛。

如果说人生分十层，考上985和211意味着从1层坐电梯到了4层，考上普通一二本意味着坐到了三层半。这是国家给你的红利，别的你啥都不用考虑，生而平等的一次红利。总有鼓吹上学无用论的，亲爱的们，千万别信。看明白这层逻辑推导，你就明白了“高考"的红利有多大。

# 有限博弈 无限博弈

孩子18岁前的教育，叫做“有限博弈”。是国家设定好的一个公平性的筛选机制，本质上是为了维护社会的长久运转，防止社会固化分层，全是拿分数说话。这套规则是专门设定出来的，给予基层一次机会，尽可能缩短进入中层的门槛。

18岁以后的人生，叫做“无限博弈”。

1. 终身持续学习的兴趣和能力。
2. 妥善的处理各种各样人际关系的能力。
3. 抵抗压力和失败，勇敢再站起来的能力。

人生从此开始变成了一场概率游戏，你只能利用自己的努力，调集多方的资源去将这个概率提高，然后尽人力，知天命。

基本上所有人生的成功，全都是上面三条的综合体。所有考完试的孩子，都要接受一个心态的转变，这人生从此再也没有规定的解题动作和规定答案了。

# 高考后的人生模式一 成熟赛道

高考后的人生，通常有两种模式，一种是根据已经成熟的，规则明确的赛道往前走。因为赛道已经成熟了，规则已经定好了，玩这种游戏就要允许游戏的开发商发币、删号开外挂。这条赛道上，理论来讲阶层跃迁这事概率极低，需要看命的。

因为上层建筑人家有足够深的的护城河，中层本身也有相当优势的领先身位。尤其在一个已经非常成熟的赛道上，像公务员、银行这类岗位中，下一层想要真的实现破圈太难了。这种规则已经设定的相当明确的赛道，更多的是为优势者设定的，是为了秩序和稳定。

这种规则和赛道明确的行业，通常不会对人有太大的赋能。更多的是把人变成平台或组织的零件，时间越长你越离不开它，到时候就越会被这个平台所控制。

# 高考后的人生模式二 不成熟赛道

高考后的另一种模式，是在那些规则不成熟，不明确的赛道上切换；或者此时脚上踩着成熟赛道，但眼里看着别的赛道准备随时跳车。只有规则和秩序还没有明确形成的时候，或者说机会还没有被看懂的时候，布局还没形成的时候，才是后来者的机会所在。

这条规则适用于炒股、创业等等太多领域。

为啥报志愿选大学的时候一定要去北京、上海、深圳、广州、杭州这样的城市去呢？一个北京的211和985的兰州大学比起来要去哪个呢？只有最大的城市，最多的人口，才会产生最多的碰撞，从而创造出最多的不对等，然后才有那些新的规则不成熟，不明确的赛道出现。

越是小的城镇，越是固化的，越是熟人社会，里面全是按部就班的人，你如果是个既得利益群体还好，不是的话那就真一眼望到头了。越大的城市，就算他活的有千般辛苦，即便会成为干电池，但最起码不确定性和碰运气的成分更大了，能给你提供最大
的新赛道可能，见得市面也比小城镇要大的多。

在兰州，也许十年出现一个新赛道。在北京，一年也许出十个新赛道。这相差就是百倍的差距，这个百倍的概率，都会算在你未来是否能闯出来破圈的概率里的。

马云和刘强东，这俩一个中层出身，一个基层出身，最终实现逆袭的都是靠着互联网这个当时还混沌初开的行业杀了进来的。这些年互联网的码农们收入相当高，产生了最多的跃层群体，本质上也是因为写代码这事别的家族和既得利益群体帮不上忙。

所谓的人与人距离的迅速拉开，本质上是因为变了赛道。

# 总结

1. 开篇那三张图，建议多看，知己知彼才能百战不殆，每个阶层都有各自的想法和诉求以及最优解题思路，多思考。
2. 既得利益阶层的最大优势是定规则，然后拼优势。
3. 破圈向上阶层最大的出路是新赛道和多机会。
4. 运气很重要，尽人力，知天命。
5. 想好了自己要啥，然后承担相应的风险，在成熟的赛道上想暴富本身就是痛苦的事，想改变就得换方向，或者改变自己的执念，做个平淡是真的游戏人生者。
