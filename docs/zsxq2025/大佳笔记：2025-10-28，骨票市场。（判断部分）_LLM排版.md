
# 大佳笔记：2025-10-28，骨票市场。（判断部分）

2025-10-28，股票市场。（判断部分）
1，打一个喷嚏就能上4000，今天是打完了喷嚏之后的缩量震荡。又是突破象征意义点位之后的过渡行情。
2，盘面上，可以写的并不太多。最主要的在债市，和一些q3报告披露之后的股票走势。债市央行重启买债，利好大涨。自从9月之后，债和股的相关性就极弱。这是因为利率到了高点之后，不再和股市形成跷跷板，而是走自己的独立逻辑。尤其是9月之后的下跌，更多的是基金新规带来的影响，以及持续弱势下，长债买盘的枯竭。未来，国债的走势，将会更大程度的独立于股市。
3，今天讨论比较多的是十五五的规划建议。我们认真看过。认为，相关措施非常宏观，不足以找到和市场直接联系。建议作为宏观视角的一个看法，而不要作为直接的交易依据。
4，今天分享了10年前创业板牛市的一篇经典文献。分享这个文献的意义在于告诉大家，10年的时间很多东西改变了，比如科技。但很多东西没有变，比如股票投资者的行为。比如经典的线性外推逻辑。
5，今天我们持仓新公布了三季报。苏泊尔符合预期，索菲亚甚至略微超出预期。三季报甚至环比略有改善。让我们更加相信，q3一些下行行业中企业的韧性。
6，今天是4000点10周年。目前，市场中估值处于最低水平的是食品饮料，今年涨幅最惨烈的也是食品饮料，但是，十年以来，市场涨幅最好的也是食品饮料。这也一点，非常值得我们自己思考。
7，后续，我们的计划不变。等到q3继续披露业绩之后，会在10月底和11月初，继续提高权益的仓位。4000点并不代表所有的股票都很贵，相反，本次的4000点，板块之间估值差异极大，机会甚至比3000的时候还多。
