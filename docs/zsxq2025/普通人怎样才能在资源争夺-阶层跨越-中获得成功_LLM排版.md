
# 普通人怎样才能在资源争夺(阶层跨越)中获得成功

普通人怎样才能在资源争夺(阶层跨越)中获得成功？我在《网红碰瓷明星，本质是低阶层对高阶层的资源抢夺》一文中分析过，网红与明星的矛盾碰撞，本质是低阶层玩家对高阶层玩家的资源争夺事件，并且这样的事件，今后还会层出无穷。

这不刚刚没几天，又有一个霍姓的明星中招了，不一样的“演员”,同样的“剧情”。要知道，低阶层对高阶层进行资源争夺是很难的，至少需要满足两个条件。首先，处于低阶层的玩家，社会认知要脱离低层次阶段，达到一定水平。其次，处于高阶层的玩家，社会认知水平一定要低于本阶层。

上次吴凡事件是这样，这次的霍事件又是这样：一个星二代，自我感觉极好，在半公开的微信群里谈致命的个人私事，社会认知水平极低……被人曝光其实是意料之中，到现在才被曝光反而是意料之外。如果不是受到吴凡事的启发，估计那姑娘还想不出这招来。霍与吴凡，都属于出身比较好的小白玩家，没什么好说的。但从某种意义上，这类事件提供了一种资源争夺(阶层跨越)的模板。也就是说，今后之类事情只会多，不会少。

在资源总量不变的情况下，跨越阶层就意味着资源的重新分配，也就是低阶层要蚕食高阶层的资源。

普通人怎样才能在资源争夺(阶层跨越)中获得成功？下面就跟大家分享一下，资源争夺的两种模型，以及成功的方法。

资源争夺，有直接和间接两种。

直接的资源争夺是赤裸裸的，可以分为上层对下层的争夺，和下层对上层的争夺。一种是上对下的，比如富人收割穷人，地主收割农民，这种都是强势阶层直接抢夺弱势阶层，来稳固提升自己的地位。这种也是最轻松最容易做到的。二是下对上的抢夺，这种比较难，比如吴凡事件、霍事件都属于下层去向上抢夺的案例，剧本也都雷同。还有我之前文章里提到的《天道》,里面的丁元英帮助村民脱贫致富、击垮乐圣集团，就是一个非常好的资源抢夺成功的例子。这种要想抢夺成功，不仅取决于双方的认知差距，还要学会借势。

社会认知差距的概念，我在之前文章里讲过了，吴凡是典型的流量明星，走到现在的位置，靠的是资本和背后团队运作，他的成就跟他个人的社会认知关系不大。霍也一样，社会认知水平低，本身也没什么亮点，能走到现在也跟他父亲有关。这种明星都是包装产品，他们自己对社会认知水平有限。而另一方，都是下层打滚的人，对娱乐圈、网红圈的炒作都很熟悉，再加上背后还有高手帮忙。一开始她也只是想抢流量而已，基于这个目标，她已经成功了，只是没想到吴凡送了个大礼包。

后面的结果，跟都竹其实没关系了，跟国家意志有关了，这就是借势：一是站在舆论与道德优势一方，二是国家要整治娱乐圈乱象的大背景，三是中国和加拿大的恶劣政治关系有关。包括吴更多的犯罪行为，这些是超出都竹预期的。

但是像这种下对上的争夺，风险很大，成功也不容易，搞不好自己也要被关进去，所以除了自己必须要有较高的社会认知水平外，要选择阶层高、认知低的争夺对象，要斗智斗勇斗狠，还要学会借势。

以上是直接的资源争夺。

第二种是间接的资源掠夺，其实是一种资源的再分配。比如互联网对实体行业的资源争夺，新经济对旧经济的资源争夺，new money 对 old money 的资源争夺。这也是为什么，这些年做电商的，做新媒体的，做短视频的，都能赚的盆满钵满，而干实体经济得很艰难的原因。对我们而言，一定要站在新经济的一边，因为传统行业里，资源已经被行业老人吃进去了，你想再去分一杯羹，让行业里的老人吐出来很难。

但是在新行业新经济里，还没有这种垄断，群魔乱舞中，你是可以去分一杯羹的。并且，像互联网行业的资源，一定也是抢夺、挤占的传统行业的资源和利润。

好了，总结一下，资源抢夺(阶层跨越)怎样才能成功？

答案很简单：

跟着强势阶层，收割弱势阶层；
跟着强社会认知的人，收割弱社会认知的人；
跟着强势行业，收割弱势行业。

以上，大家可以举一反三。
