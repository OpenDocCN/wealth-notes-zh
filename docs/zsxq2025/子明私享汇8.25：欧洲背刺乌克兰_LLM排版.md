
# 欧洲背刺乌克兰
## 发布时间： 2025 年8月25日

德国21日发布通告，应德国签发的逮捕令，意大利警方逮捕了涉嫌北溪爆炸的乌克兰男子。之前俄罗斯一直指责北溪爆炸是英国与北约操纵的。作为管道的起始过，芬兰是涉事国家中唯一没有公开北溪调查报告的。

巧合的是，意大利、德国、英国、北约、芬兰的领导人，还有一直坚持欧洲能源自主的旗手马克龙，8月18日拉着“肇事国”乌克兰一起访问美国。目前，全球的主流媒体和自媒体都在说，俄乌的停战谈判很崩，特朗普也在taco打退堂鼓。

但看国际分析不要看政客们说什么，而要看政客们做什么。俄乌领导人谈成什么不重要，两个人的见面才重要。有了领导人的见面，下面的一系列勾兑才有可能推进。北溪与北溪爆炸的相关领导人集体来访问美国，也就意味着普京的天然气出口限制被摆上了俄乌停战的谈判桌。

这也是为什么把能源转卖到欧洲的印度突然遭遇了特别关税。特朗普是把印度转手的俄罗斯天然气直接卖给欧洲，而乌克兰，则会成为这一轮交易桌上的盛宴。

中国社科院俄罗斯东欧中亚研究所所长孙壮志：这件事在欧洲国家和乌克兰之间会制造一些裂痕。特别是德国跟乌克兰之间，以及意大利跟乌克兰之间，因为这些国家的社会层面在援助乌克兰方面就比较分裂，会使欧洲国家的民众对支持乌克兰产生一些疑问。

## “北溪” 天然气管道爆炸事件背景补充：
2022年2月乌克兰危机全面升级后，德国对俄罗斯实施经济制裁，包括停止从俄罗斯进口煤炭和石油，同时俄罗斯也停止通过“北溪”向德国“供气”。同年9月26日，“北溪-1”和“北溪-2”管道在丹麦、瑞典附近海域发生剧烈爆炸，疑遭人为破坏。事发后，丹麦、瑞典和德国分别启动调查，但拒绝俄罗斯参与。2024年2月，瑞典、丹麦相继宣布结束调查，但对调查结果讳莫如深。德国表示仍将继续调查。
