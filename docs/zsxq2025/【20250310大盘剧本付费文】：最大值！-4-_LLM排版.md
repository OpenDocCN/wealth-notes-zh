
# 最大值!

原创：肥猪仔 公众号：大盘剧本 25年3月10日

今天直接把关于“最大值”的结论跟大家写出来：

1. 不能只看一个背离。背离始终是一个过程量，不能作为主要的判定依据。只能是辅助。相反，应该把当前的走势放到更大的结构尺度来讨论。这个结构尺度预示着市场近期内就是会出现向上的变盘。所以，我们要以更大的耐心去忍受一些波动和调整；
2. 此前跟大家分析“调整”的可能性的时候，有一种情况是再次逐步震荡回到55周均线附近。这种情况，随着指数在55天均线上震荡的时间越长，就越不可能。这种情况目前来说概率都已经很小了。如果你潜意识里总是担心这个情况的话，就应该暂时把这个石头放下；
3. 排除掉上面这个最差情况后，目前只有两种上涨方式是可以满足第一种大周期向上变盘的特征的：

一种就是从1月13日最低点3140点开始，已经是一个周线的主涨段了。目前还在持续。这种情况目前的技术信息是无法排除的。

比如今天的布林线中轨仍然出现了精确的支撑。在这种模式下，市场随时向上快速上涨都是有可能和合理的。这种随时上涨的结果自然就冲破了前述的“背离”的影响；

第二种就是当前的震荡，特别是最近8个交易日的震荡是形成日线的第二段调整的一部分。这个调整的特征是以55天均线为托底的。随后展开日线的大三段上涨，继续周线主涨段。这其实也是技术上完全合理的。

在这种模式下，指数所有在55天均线之上的震荡都是可以的。也就是说，就算是再用上个三天左右的时间再回抽一次55天均线，大概3320点附近，也是可以的。这样正好在日线上形成一个宽幅震荡的平台。这种就是当前所谓调整在时间和空间上的最大值了。

美其名曰最大值，你在日线上，其实也不是不可以接受。

你接受不接受的关键点其实在于你牛熊思维是否转变。没有转变的话，就再怎么都接受不了。
