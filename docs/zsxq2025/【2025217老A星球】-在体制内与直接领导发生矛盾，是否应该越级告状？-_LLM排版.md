
# 在体制内与直接领导发生矛盾，是否应该越级告状？

匿名用户 老A 向老A老师提问：之前拜读了您的回忆录，其中提到如果与直属领导发生矛盾，第一时间向上级领导告状。想请教您一下：告状的技巧有哪些，会不会留下坏印象？上级领导知道彼此有矛盾后，会不会不想掺和或者偏袒另一方呢？感谢。

老A老师回答：

有句老话：尽信书，不如无书。虽然我在回忆录里那样讲过，但有个大前提是：你和小的直接领导已经彻底拉爆撕破脸。但，一般情况下，正常下属不该这样做，你这样做了，即便你和直接领导发生矛盾，你去告状了，上级领导也不会偏向你，甚至觉得你是个刺头，即便你和上级领导有点互动和渊源。

如果和上级领导没有过私下里的互动，如果你去告状，上级领导大概率会说：年轻人要低调，要学会服从领导安排。如果直接领导不依不饶，上级领导甚至会支持你给直接领导道歉，毕竟，在单位里，下级服从上级，这是纲目！不然，任人都可以以下犯上，那科长们是不是也可以动不动就和局长产生矛盾？
