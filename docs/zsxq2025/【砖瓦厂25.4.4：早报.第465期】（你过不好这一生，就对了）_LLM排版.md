
# 砖瓦厂25.4.4：早报.第465期（你过不好这一生，就对了）

短视频最近有种表述，非常火：搞懂了这个世界运行的规律，明白了那么多道理，却过不好这一生。真正搞懂世界运行的规律，小到一个细分行业的规律：如怎样把销售这份工作做好，都是超级超级难的。真正搞懂了，去执行了的人，大概率过得不会差。换句话说：我这样的穷人，大概率是没有搞懂或者是，自以为搞懂了，泛泛的懂了一点点，忍不住去卖弄。

我说过很多次：我讨厌读书人，有钱的读书人除外。原因是：我们这些在学校里待了很长时间，且日子过得一塌糊涂的，所谓的读书人，也就背点考试内容，真正懂得东西，太贫乏了。拿两点来衡量：1.是否能够把生活过好。2.能否赚到超出社会中位数的钱，或有社会地位。要是这两点都是否定的，且我还坚信，自己懂什么世界运行的规律，什么思考纬度高，什么看透了什么的本质那我就是自以为是的傻比嘛，而搞懂这个世界运行的规律——牛逼的方式，是成为某个领域的牛人。比如：外科专家，高级工程师，高阶程序员，高级产品经理……社会为他们薪水所买单的多少，很客观的反应出了：他们对这个世界某一领域运行规律的掌握水准。而非，似是而非的知道一点：宏观上的观点。

前几天睡前看知乎，记得有个大概这样的提问：你的人生被什么习惯给耽误了？有个高赞回答是：长期沉浸在，什么都懂一点的快感中。问什么，都知道那么一点。可真正做事情的时候，用现在流行的话说：没有一件事，能够干闭环。在整个学生时代，算算数快一点，知道的省会城市多几个，朝代会多背几个，生僻字躲认识几个会得到赞扬。乃至于长大后，聚会的时候，天南海北，天文地理，体育经济，都能接住话题，自己的声音贝听见，继而内心充满喜悦。后来我们知道，这种技能，差不多只说这应试阶段有用：是小规模记忆力的优势体现。但具体在做事方面，却不灵了。把事情做好所需要的品质：坚韧、靠谱、专注等等，与它关系不大。那句：闭环的能力。强调的，就是把一件事情完成的能力。

我们很多人在公司上班，所做的是偏螺丝钉的工种。通常不清楚事情的全貌。也不清楚，赚钱——从头到尾、从一个idea到回款完成，是怎样产生的。比如我自己，从小励志做的事业，是成为一名工程师。后来当上了工程师，骨子里不屑那些做行政工作的干部—认为他们的存在，没有实际价值。后来进入社会，慢慢才明白：行政干部的重要价值，是把整件事盘下来——从找资源，到项目结题时的公关。而至于中坚的执行步骤：从勘查到完成报告。实则是当时并不重要的环节。同理，也发生在许多信息化公司中：这些公司里，拿最多报酬的，并不是CTO，而是销冠。原因也是：把整件事搞定，最难的环节是—商务。技术人员，工程师们参与其中，解决的只是不重要的一—交付。不影响结果。而发现商机，搞定关键人，解决预算，搞定验收，顺利回款，是链路中，核心的部分。做这部分工作的人，不需要技术队伍的帮忙，也大概率能找到团队，把事情闭环。但反过来，就不成立了。但要完成上述，靠什么都懂一点的才能，是万万不够的。它需要与人打交道，与自己打交道，与社会的既定规则打交道，与各种资源打交道。还要能算明白账。

很显然，我这样的，在青少年时代长期沉浸在什么都懂一点点的快乐中的人，及其所之既倦缺乏上面的能力。甚至，还鄙夷过上面的能力。所以，混得差。
