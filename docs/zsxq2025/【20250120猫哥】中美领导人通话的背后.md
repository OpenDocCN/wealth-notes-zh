

# 中美领导人通话的背后
# 猫哥原创

讲一讲中美领导人的通话。

这次通话双方都取得不小的成果。

首先是特朗普发布的信息，提到了三点，芬太尼、TIKTOK、中美贸易问题。

这三个问题大致焦点是这样，芬太尼美方需要中方帮忙，TIKTOK是中美在角力，双方要找一个平衡点妥协，中美贸易是老生常谈，也就是中方要多采购美国产品减少中方贸易顺差。

特朗普热情洋溢主动发推提到这三个问题，说明这三个问题对于特朗普而言都取得不错的成果。

芬太尼中方会进一步管控出口；

TIKTOK双方初步达成共识，今天得到消息是特朗普希望美方能占股50%，注意不是51%，而是50%，大致是中美共管；

中美贸易问题估计也是达成初步共识，中方会加大对美国产品采购，而特朗普可能会象征性对华产品加征一点关税；

然后我们再来看看中方新闻稿。

在中方新闻稿中，中方取得的成果有两个，一个是明线，一个是暗线。

明线是，双方都同意建立战略沟通渠道。

这句话表述的意思就是，双方都有意愿解决卢比奥被制裁情况下，怎么重建中美沟通渠道的问题。当然，双方通话只能是定大原则大基调，接下来的细节才是下一步双方工作人员磋商的问题。

谁去磋商？

当然是这一次被派出去参加特朗普就职仪式的中方特使。

然后就是暗线的成果。

成果是这一句，（特朗普）期待尽快与中方领导人见面。

尽快不可能时间得太远，但是半年之内国际上都没有重大活动，也就是说，中美领导人见面不可能是国际场合的见面，只能是双方单独见面。

双方怎么单独见面？

特朗普就职仪式邀请中方领导人，中方领导人都没有去，那么就只剩下一种可能——山不来就水，水就来就山，特朗普就只能是启动访华事宜。

所以，中方新闻通稿的这一句话——（特朗普）期待尽快与中方领导人见面——表达的意思就是——特朗普希望尽快访华。

这就有点意思了。

本来之前市场的预测就是，特朗普上台之后，中美关系会进一步恶化，但是从这段时间各种迹象来看，特朗普上台之后，中美关系不但有很大概率不会进一步恶化，反而很可能有所改善。

为什么？

出现这种大逆转我觉得拜登有很大功劳。

拜登各种坏事干净，各种给特朗普挖坑，把特朗普能打的牌打得干干净净，逼着特朗普就只能在对华关系寻求缓和了——当然中方前期一系列强硬反制也起了关键性作用。

那么特朗普访华想干什么？

一个是公事，一个是私事。

公事大致是这样：特朗普2.0基调是搞盟友，同时在寻求对华关系的缓和基础上摄取最大的利益，比如加大对美产品采购，比如在地缘政治冲突中寻求中方的合作等等。

那么私事又是什么呢？

是重建与中方领导人的私人友谊！

特朗普是商人，商人都无利不起早的，与其它美国总统相比，特朗普特别看重与大国领导人的私人友谊。

为何？

总统是有任期的，但是家族企业利益却是没有任期的，如果能与大国领导人建立私人友谊，对于家族事业的发展壮大是有巨大利益的。

但是，在特朗普1.0时代，这事就搞砸了，我印象最深的事情是，在特朗普1.0中后期一次中美领导人通话时，特朗普强调与中方领导人有良好的友谊，而中方领导人直接打脸回应——我们是工作关系。

在特朗普1.0最后任期，因为新冠疫情输掉大选，导致特朗普很疯狂，各种抹黑甩锅中国，逼着中国在特朗普下台之后立刻制裁了特朗普内阁几个刺头，没有直接制裁特朗普已经是中方最大克制了。

所以，在特朗普2.0在总基调是战略收缩，是搞盟友情况下，重建与中方领导人的私人友谊对于特朗普就格外重要了。

怎么能重建友谊呢？

当然是面对面坐下来吃吃饭聊聊天。

这才是特朗普急不可待访华最大的动力。

特朗普这个动机对于稳定中美关系也是有益的。

经过特朗普1.0的磨合，我觉得特朗普在2.0执政期间应该更成熟，假如在特朗普访华期间，中方也能配合释放一些友好信息，那么在特朗普2.0时期，中美关系就不至于失控。只要不失控，就符合我们斗而不破的基本原则。

所以这一次参加特朗普就职仪式的中方特使肩负着重大使命，一个是落实重建战略沟通渠道问题，一个是进一步落实与推动特朗普的访华事宜。

资本市场对于中美领导人通话也做出积极反应，周五境外中概股大涨，A50大涨。外部环境缓和了，对于财政部尽快出招尽快推动经济复苏也是有益的。
