
# 金融三剑客 –中国资本市场-中信（二十一） 

## 一、波动

20年前，社会上曾经有过一股藏獒热，话说一条纯种的藏獒能卖到上百万，如今价格崩盘，已经没人玩了，这就是典型的在短期内快速积聚共识和流动性，然后又快速崩溃掉的案例。可能有读者觉得这就是个泡沫，就是个骗局，其实只是因为藏獒的周期太短了，显得好像比较拙劣。

其实类似的事情在各个领域都有发生，“蒜你狠”各位还记得吗？就是2010年中国民间流行的一句时髦用语，源于大蒜疯涨超过百倍，价格超过肉和鸡蛋的现象，那年还相继出现了糖高宗、盐王爷、苹什么、豆你玩、姜你军、玉米疯、油你涨、辣翻天、鸽你肉、棉花掌等等一些列的涨价风潮，背后都是海量美元(奥巴马时期的QE2)短期涌入后快速引发的通胀潮。

由此中国市场上的流动资金越来越多，这都是美国佬准备炒中国股市楼市的钱。之所以都在楼市和股市里，是因为这两个市场泡沫产生快，泡沫也大。可是，老美忽略了一个重要问题，或者说他们即便不忽略也无法回避一个问题：不管他们在经济学上有如何花里胡哨的理论，但这里是政治主导的社会经济行为。

然后就是2012年5月，郭树清突然出来说了一句：“穷人还是不要炒股的好。”(后面他又说了收益超过8%，你就要打个大大的问号，又是一波非标清洗)，当时是引来网民的一片痛批的：难道上峰就不能给穷人一点翻身的机会了吗?

其实证监会言外之意是，股市很快就要要跌了，你们这些赔不起的小户，不要给我的工作添麻烦了，赶快提前逃跑吧，逃晚了，咱就要给美国佬挖坑了，你们也会被陪葬的。

话说美国佬的钱随着QE2大举冲进中国，中国为了保证物价稳定，就用楼市和股市招待他们，中国隐忍了一个秋天，终于熬到了冬天，奥巴马那边率先先扛不住了，美国各大银行包括高盛纷纷割肉离场，各家在账面上都赔了上百亿美元(熬不住中国跌跌不休的股市)。

so，如果我们把周期拉长一点，假设一个人30年前持有1万元人民币，瞬间穿越到2024年，那在他的观感里，会觉得人民币也蛮泡沫的，中间的波动并不比狗子小。

## 二、对策

关于这个波动的问题，有读者曾私下和剑客说，他有个经验，他的所有投资都不允许自己的亏损过30%，因为一旦超过30%，根据历史经验，他就失去了止损能力(不愿意割肉)。

即既然卖了也没剩多少钱了，就索性放着了，最后有很多标的就越跌越多，有的甚至接近归零了(惨痛的回忆)。

他这个说法有没有道理？有点道理，但如果你再想深一层，这个做法有个小问题，即这个30%是怎么量化出来的？它不科学，它就无法成为一种普适性的方法论。

道理很简单，他这个数字设置的是很主观的，即你把它改成20%，也有道理，改成35%，也有道理(基于自己和样本的历史数据调整)。既然如此，它就不是高置信度的。

比如某些牛逼的品种(各类资产都有类似的品种)，它会先跌破你的30%，然后再涨100倍，这时候你可能就会把自己的止损线再调整成35%，但如果它们下一次的回调都变成了先跌破35%，然后再涨100倍呢?

反正不管你怎么调整，你都会陷入“感觉主义”和“经验主义”。换言之，你无法遍历所有数据是其一，再一个就是你无法根据过去的涨跌幅去预测未来的涨跌幅和震荡幅度，这是其二，因此这个区间不管你怎么设置，你都无法在胜率上稳稳超过那些设置其他止损百分比的人，所以就算你一段时间用得好，也只是运气，而不是一个值得大多数人跟随的更底层的方法论。

比方说比特币这种十几年间一直在上涨的标的，你就算在早期介入，也可能在任何时候被深套，比如那个美图公司，它是2021年3月、4月入场的，btc和eth各买了5000万美元，加起来1个亿。eth的成本是1600+，btc的成本是56000+。之后冲高到69000后开始回调，一度跌到16000，美图浮亏严重。当时很多美图股东都在骂蔡文胜拿公司的钱瞎搞，不过在去年又变成了“真香”。去年底借着这波行情蔡文胜获利了结，最终盈利接近80%下车，整体盈利7900万美元。

换言之，中间即便你设置了任何的止损百分比，都必然会提前下车，因为它的回调实在是太深了，惨烈的时候能一下子让你的总资产缩水80%-90%。

而频繁下车的人，最终无一例外，也都吃不到比特币真正的涨幅，因为祂根本不知道什么时候该重新上车，正因为投资的体系不稳定，就总有一次由于他的犹豫，导致彻底上不了车。而这种错过，往往是一辈子的，也无法靠其他收入弥补得上。

## 三、逻辑

为什么要这样做，更底层的原因还是在于财富它无法固化，它像水一样在社会各个载体之间流动，此起彼伏，哪里有新的共识汇聚，财富它就会如潮水般流过去。

从这个角度看，你再去理解虚拟币这十年来的扩张就清楚了：它是快速汇聚了大量的共识和流动性，从而成为了新型财富。

但有读者始终觉得数字币始终是虚的，没有应用价值，也没有国家背书，等到哪一天大家的共识像藏獒一样突然崩塌了，币价也会崩的一文不值。

这个说法目前有理论上的可能，但实际上并没有那么简单。你看，藏獒是看热闹的人多，真正手里有狗的人还是少，所以维护共识和流动性的基本盘还是太单薄了。

相比之下，虚拟币的群众基础的扩张是挺快的，尤其是这十年的暴富效应，新上船的人越多，决定脑袋的屁股就越多，共识和流动性的底座就愈发稳固(特朗普/马斯克这些大佬也在给它增加共识和流动性以方便自己转移财富用)。

在美国市场上，比特币也逐渐由"限制"叙事转为"收编"叙事，比特币从美元体系的"边缘人"成为了数字美元体系中的"主流资产"，美国对加密资产的定位也变更为美元体系中的数字黄金。

换言之，在历经了16年，在内部和外部无数次打击和接近归零中，比特币跌跌撞撞地逐渐受到了大漂亮某些大佬的认可，这个标志点，就是比特币ETF的通过，因为自那时开始，比特币正式以主流资产常见的形式，成为了普通老百姓资产配置的一部分。

不过，很多人还是觉得财富一定要有咱们自己的政权背书才安全，其实看你从哪个角度看。

举个例子，之前美国最大的燃油管道系统被黑客攻陷，勒索500万美金等值的比特币，最后运营商Colonial被迫支付了巨额的赎金。

由于支付的是数字货币，警方无法追索犯罪分子和黑金的去向。话说黑客攻击重要数据库，然后索取比特币赎金，这几年在国外已经是一个很成熟的产业。全球灰色交易的需求并不小，所以它脱离主权背书和监管的套路(去中心化)，反而在很多场景下成为了一种独特的竞争优势。

话说人的职位高低、成就大小、财富多寡，和TA已经解决的问题的大小，往往成正比。所有的社会分工，都是为了解决问题。所有的商品和服务，归根结底也是为了解决问题。虚拟币它能解决很多灰色交易的需求，这就是它的生态位所在。

道理很简单，你想想，在发达社会，律师为什么这么受待见，还能进大佬私宴，凭什么?富人之所以成为富人，所有的，皆只是非法的或者合法的取得了巨大的不合理的财富。而律师代表的就是这个不合理而使之合理的层面。

这个虚拟币站在政权的角度，现在只是缺一环，缺律师(合适的时候也会有，有了它就合法化了)。明白吗?

背后你穿透了现象看本质，都是共识和流动性定义了财富，不要拘泥于旧思维下的表象。

其实最初的时候，几乎所有国家对比特币的政策都是负面的，以郎咸平为首的经济学家和主流媒体们，对比特币也是极尽嘲讽。

但是你要明白，这帮人的立场并不总是要反应客观真实，因为他们不认可比特币才是对他们性价比最高的观点。这个道理也很简单，无非说对了，那果然牛逼，经济学家就是厉害；错了，反正大家都错，连国家都搞错了，所以错了不是很正常?

但问题是啥吧，如果他们看好比特币，最终还真成了，那他们就成了吹哨人(你想想之前疫情和波音的吹哨人现在在哪儿?)。
