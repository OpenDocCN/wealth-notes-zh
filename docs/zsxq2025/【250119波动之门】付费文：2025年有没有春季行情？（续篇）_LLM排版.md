
# 2025年有没有春季行情？

1月14日阳线起立，到底有没有春季行情呢？有没有春季行情，要看这波调整是不是结束了。我们先看那个最严格的3106点怎么来的。下面是一个技术活。这是上证的60分钟线。

自3494下跌，第一浪跌了148.40点，按照一般的下跌规律，整个下跌浪是第一浪的2.618倍。全程一共下跌148.40点。

## x2.618=388.5

所以，见底的理论点位是：3494-388=3106点。我们还有一个利用30分钟线的算法，是3125点。

1月13日见到了3140点，见到这个点位，基本可以确认，底部到了，或者说，十分接近了。

## KD

上升行情，月线死叉之月，就是买入之月。请回溯6124牛市，5178牛市，2016年1月到2018年1月的上升市。你在死叉的时候买入，是不是就等于买了后面一个月的阳线。

当2635点牛市底部确立的时候，我们抄底了。可是5月见顶以后，我们为什么一再强调，大底在九月，你去看月线在7月死叉了没有？在8月死叉，说明9月是阳线。再利用周线的结构，就可以确定大底在九月。

你知道了这个规律，最起码不会在6月和7月伸手建立中线仓位。也就是你用技术最少保护了自己两个月。

牛市中的月线死叉，除了最后一次，都是买点。有时，它不是最精确的买点，但却是稳赢的买点。

1月14日和1月15日的买点，既是为1月余下的时间买的，也是为二月买的。我说的已经非常明确，现在可以肯定，二月十阳线。三月是不是阳线，我们还要根据二月的情况再判断，距离太远，容易出现细节上的误差。

这篇文章，全文只有九百多个字，我认为107元相当便宜。为什么是107呢？春季行情整个结论，我给它定的价格是7元，但是月线死叉这个技术，值一万元或十万元，我把它的价格定为100元。

因为它不仅告诉了你现在及以后一个月的走势，也为整个牛市的抄底，有了一个月线的指引。如果在2月有什么大的偏离预期，我将在这里进行后续的补充。

请大家注意保密，技术知道的人越多，对你将来的操作越是不利。
