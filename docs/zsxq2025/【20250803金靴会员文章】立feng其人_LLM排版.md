
# 立峰其人
## 文/欧洲金靴

这次的金融会议，除了“党取代国务院”这个组织层面的特大变革，还有一个人事信号就是立峰的出头。会上，立峰做会议总结；会后，金融监管总局、人民银行、国开行、证监会、各省委在内部开会传达落实会议精神时，除了表示要落实领袖和强子的讲话精神，还提到了要落实立峰会议总结的讲话精神。

加上这场规格极高的会议前一天官宣的接任鹤出任中财办主任(以中央财办主任的身份会见法国总统外事顾问博纳)，可以说此次会议后，立峰同志已然我国金融第一重臣(他目前分管的工作包括收拾中国深陷债务危机的房地产行业以及监管价值六十万亿美元之巨的中国金融行业)，各金融机构与诸地方应已了然。

有人甚至给他起了个外号：新陈云。立峰的进步迹象，尤其是取代鹤，大约自前年就已开始造势。2021年4月举行领袖和法国总统马克龙、德国总理默克尔共同出席的中法德领导人视频峰会中，中方在视频中列阵的副国级随侍，只有立峰和两位外事主管杨主任和王外长。之后5月，领袖在北京视频连线俄罗斯大帝，见证两国核能合作项目(田湾核电站和徐大堡核电站)开工，又是由立峰和俄副总理诺瓦克主持仪式，我们副国以上陪同人员也是只有杨主任和王外长。直至去年大会入局。

简单聊聊立峰。立峰1973年参加工作，当过农场知青和水电站工人，后参加高考中榜。四十年前，他在厦门大学拿到硕士学位仅仅几个月后，遇到了在河北郁郁不得志的领袖——高杨在河北不待见他，认为其“高调”，实际不过是因为领袖政绩出色罢了。无奈，领袖南下前往福建(思想开放的项南主政)进行改革探索，在厦门与立峰相识。

80年代，两人在厦门这个经济特区一起搞“新工业园”，厦门的发展搞得红红火火。后来领袖离开厦门乃至离开福建、一步步去接棒历史，而立峰则一直坚守福建。

上世纪80至90年代，也正是我国对外开放的关键时期，立峰一路履职一把手，从泉州到福州又重回厦门——这三座城市在福建的经济总量加起来占全省70%。终于，立峰于世纪之交成功晋升副部(坐稳省委常委)。

2009年春天，领袖基本确认坐上大位，立峰随即宣告离开福建，被上调天津。攒资历的同时，他也捧获一项历史性考验：天津滨海新区。2009年5月12日，天津市加快滨海新区开发开放领导小组第九次会议在天津港保税区召开。是次会议上，天津市委常委、组织部长史莲喜正式公布了人事变更的消息：立峰以天津市委副书
记之职，兼任“滨海新区工委书记、管委会主任、塘沽区委书记”。

这是继皮黔生、苟利军之后滨海新区第三任管委会主任，同时，立峰还被任命为“天津市加快滨海新区开发开放领导小组”副组长。这是天津推动滨海新区建设的核心机构，级别相当高：组长为天津一把手的丽，第一副组长为天津市长的黄。

滨海新区是天津市下辖的副省级区、国家级新区和国家综合配套改革试验区，是国务院批准的第一个国家综合改革创新区。此时，距离立峰由福建东南沿海北上津门不过二
百余天。被点为滨海首任一把手，足见对时年54岁的立峰的重视及培养意图，这背后自然有领袖嘱托。这里面其实还有一层更深的考虑：天津，是当时二号人物的老巢，调立峰去天津，也是尚未坐上大位的领袖的一种布局。

而作为回应，次年(2010年5月)，那位二号人物便前往天津考察，并重点关注了滨海新区……

毫无疑问，这是立峰人生中极为重要的转折一站，在天庭换届的历史性当口，滨海之旅这将决定他
的命运。那个阶段，有天津媒体公开报道：“立峰履新滨
海后，就住在管委会大楼里，常常在每晚9点左
右召开办公会，一开就到深夜。”

初来乍到的立峰，上任伊始就展开了大规模的密集调研，滨海新区所有重要的功能区、行政区以及大项目都跑了个遍——这也是其一贯作风，在
福建省内诸市做主政官时就爱下去调研，或许是
跟领袖在厦门学的(众所周知领袖是个“基层控”、“调研控”)。

据随他调研的人士介绍，立峰在滨海与其他新任领导初次调研时“多听不说”甚至“只听不说”
的风格有些不同，他往往会当即给出一些评点，这些评点往往能切中一些问题的要害，“可见其
搞经济开发的经验还是比较足的”。

立峰自己也在公开场合表示：“一个人一辈子能
遇到像开发滨海这样的机遇，是很难得的，没有
理由不干好。”

当时的滨海新区其实有点“死局”的味道，本身
滨海经济属于外向型经济，占比达到60%以上，
而由于国际市场的萎缩(国际经济危机),很多经
营出口加工的中小企业出现经营上的困境，一些
投资项目也出现后续投资放缓的情况。

所以前文说了，这对立峰不仅是机遇更是考验，
干得好往上走直接进京，干不好很可能从此靠边
站。发展终归靠人，用人则赖于体制，而滨海(或者
说天津)当时最大的问题就是体制僵化、人事根
基深冗。

如时任发改委宏观研究院国土所副所长肖金成评
价天津所言：“历史上所形成的‘诸侯经济’
必须到了被打破的时候了!”

“动奶酪”的大动作随即开始：撤销天津市塘沽
区、汉沽区、大港区，设立天津市滨海新区，以
原塘沽区、汉沽区、大港区的行政区域为滨海
新区行政区域。此前的归属于滨海新区的东丽
区和津南区的部分区域，不划入滨海新区行政区
范围，但仍为滨海新区产业规划区域。

改革必然要遭遇一定的阻力，尤其是管理体制改
革，牵扯面广，且直接涉及人事变动等问题，作
为一线“操盘者”的立峰其肩头压力可想而知，
但这块骨头他最终是啃了下来。

体制改革完成后，雄心勃勃的立峰推出十项重大
建设——包括滨海新区核心区建设、响螺湾和于
家堡中心商务区建设、南港工业区、轻纺工业园
和生活区建设、东疆保税港区及其生活配套区建
设、临港工业区建设，以及中新生态城建设等，
并推动滨海新区的做出“十大改革”,打响高端制
造园区的声势(尤其是建设滨海云计算基地，推
动“天河一号”和”曙光”超级计算机落地),
也直接缔造滨海高达25%左右的经济年增速。

除了发展经济，立峰当时还大力实施文化惠民
工程，仅2011年一年，即在滨海新区建成百座
居民书屋和文化室，放映公益电影3560场次，
还举办了首届滨海国际艺术节等大型展演活动，
总人数120多万群众参与。

台湾工业总会大陆组的组长，为台商提供赴中国
投资等谘询服务的黄建群，在天津的滨海新区曾
和立峰有深度接触，他后来对立峰曾有评价：“
他(立峰)对待台商等外商外企很热情，不论是在
厦门还是天津，在台商中少有负面评价。不过他
又很拘谨，向来只谈公事，几乎不会与商人把酒
言欢等工作之外的交流。”

这个说法也符合《每日经济新闻》之前对立峰的
走访调查，据有关人士向《每日经济新闻》透露，
立峰在厦门工作时，给人印象最深的就是他一不
喜欢串门，希望大家不要因为他不登门造访而误
解；二不喜欢别人到家里串门，欢迎大家业余时
间到办公室谈事。

根据公开报道，立峰在厦门一次讲话中曾说：“
我的亲属亲戚没有一人在厦门从事房地产、工程
建设和经商等，今后如有任何人打着我的招牌参
与任何竞争性的工程、土地招投标或者其他经济
活动，务请同志们坚决抵制，并且立即报告市
委。”

2013年1月，天津市政协换届，58岁(临近60
大关)的立峰被临时安排在天津市政协主席的位
置上——为一年之后到发改委担任副主任奠基
前提。

2014年，立峰离开天津，正式进京履新发改委，
分管工作包括地区经济司、西部开发司、东北
等老工业基地振兴司，直至三年后成为发改委一
把手(后逐渐取代宁吉吉吉，一个典型的没有基
层工作经验的官员，这种久居深宫型的官员在新
时代十年很不受赏识)。

正是在立峰的任内，发改委改变了注重外国投资
的做法，减少了与外国高管的会面，转向强调与
国有企业的密切协调。

对于立峰本人而言，他也是在经历着跟随国际国
内形势的变化而变化的过程。

回想2008年在厦门大学发表讲话时，他还曾吐
露他早年对资本主义的信仰，并在演讲颂扬了
小平1978年做出的自由市场改革决定。

然而后来他就任发改委一把手后，《经济日报》
上发表的署名文章中语气已截然不同：“加强党
对经济工作的全面领导是我国经济发展的根本保
证。实践证明，办好中国的事，关键在党。”

包括对共同富裕的呼吁，也一改年轻时从政曾迷
信“先富带动后富—一在发改委一把手任上，立
峰为《人民日报》撰写的专栏署名文章中曾大力
支持领袖的共同富裕目标，并呼吁“应当提高劳
动报酬比重”。

这个变化实际并不在于进京期间，从福建到天津，
立峰的变化就已经出现。

天津的国有经济占比较重，而滨海新区则突出外
向型，立峰主政滨海时不但扩大对外招商，等于
更引导国有企业对滨海的投资经营。

当然了，这里有一个“污点”恐怕不得不提：急于求成也好，经验不足也罢，滨海乃至天津市确
实在立峰任内进入了债务高积累阶段。

前文曾述立峰在滨海推进文化惠民工程，他其实
有点儿学习当时薄的重庆模式。但问题在于，薄
在重庆并没有大量举债，但天津市的投资主要依
赖债务驱动。

2011年，滨海新区宣布要撬动70个文化产业项
目快速推进，“打造完整文化产业体系，正在
力争使文化产业跻身于支柱性产业”,投资高达
450亿……

后来到2019年时，天津市政府、企业和家庭的
债务总额已经超过7600亿美元，为历史之最。

根据荣鼎咨询当时援引的数据称，天津所有借款
人的年利息总额是其年名义经济增长率的十二
倍…

直到今年，惠誉评级的一项分析也发现，天津的
“麻烦”比所有其他城市的都大。尽管天津长期
以来一直是重工业中心，有广泛的税收基础，但
由于附属于天津的借贷机构的负债如此之高，以
至于支付的利率位居中国最高之列。而且天津近
五分之三的债务今年到期……

这些问题立峰自己当然也不是不知道。

2015年，也就是离开天津一年多，立峰重返天
津滨海新区，率发改委在滨海召开“第一次国
家级新区工作经验交流会暨新区工作推动会”。

会上，立峰直截了当的表示：“目前一些已获
批的的国家级新区和正在报批的新区，在规划
建设中或多或少存在追求‘大手笔'、贪大求全
的现象，这些新区多以上海浦东、天津滨海新区
甚至国际大都市为标杆，脱离了当地的发展实际
……每一个阶段都有不同的情况，因此，要立足
各地实际和比较优势，合理确定新区建设目标
和功能布局，科学编制新区建设规划体系，充分
体现并最大限度发挥新区自身优势条件，将其转
化为新区的核心竞争力。一些基础相对薄弱的新
区借建设新区之名向国家有关部门‘要地',不
少农田变成城市建设用地，土地节约集约利用
水平有待提高。当前，各地申报新区的热情很高。
除已经批复的12个新区外，不少地方也提出申
请设立新区，一定要切实加强对新区的规范管理
和统筹指导，加强国家对新区建设的后续指导。”

这等于是拿自己主政过的“政绩之所”作为反面
教材，点醒当时一帮大搞快干的新区。

那个时候的滨海在立峰离去后其实又是进入了
新一轮的“死局”,债台高筑，放眼望去尽是“
鬼城”“空楼”“无人街”,也引发了立峰离开
后诸多天津市政府与滨海新区政府之间关于资源、
投资等的明争暗斗，这个我在此前写权健集团
束昱辉投资足球(泰达就是滨海龙头)的文章中提
过。

一年后，2016年9月，曾任“天津市加快滨海
新区开发开放领导小组”副组长，当时已是天
津(代理)一把手的黄，宣告落马。

这也算是给了一份交代(当然黄的问题很复杂，
有天津爆炸的责任，也有弟弟腐败的牵连),客观
上保全了立峰。

立峰十年前进入发改委后，其工作在表面上不涉
及对外开放，但是正如接手滨海新区，领袖又交
给他一个更大的历史重任——推进“一带一路”
建设工作领导小组办公室，彼时正是设在国家发
改委，立峰正是此项工作负责人。

这是一个比建设滨海新区要艰苦十倍不止的史
诗级重任，事关民族复兴乃至国际格局的重塑。

由此也再一次得见领袖对其的看重。

去年10月，立峰顺利入局；今年3月，坐稳副
总理(排名第二，仅次于肠胃的丁)。

作为闽系干将，立峰在领袖的支持下也已经搭
建起自己的金融财政队伍。

刚刚上任财政部的蓝，之前他卸任山西一把手
时我在朋友圈发过，蓝来山西就不是当政的，
而是刷履历的——背后自然有高人指点。

蓝本来就是个财人，从广东韶关开始突变地方
大员，都是为了铺路。两年前他卡着59岁的门
槛副省升正省，之后又完成地方大员的履历填
补，这是典型的培养。现在万事俱备了，财人
回财口，继续老本行(甚至他日后可能会更进一步，因为财口的位置
历来并不需要封疆大吏，蓝被强行拎到封疆大
吏，终点站绝不可能仅是财口或中委级别，后
续观察)而说到蓝“背后有高人”,高人是谁?首
先正是他取代的昆。

昆当年由广东省委调去省财政厅当厅长时，其
手下得力干将正是副厅长的蓝。

之后昆升副省长，随即推举蓝去出任审计厅长。

蓝此后去往韶关做一把手后又升副省长，背后
一路都是昆。

而昆，正是立峰的厦大老同学。

由此，蓝的晋升当然不足为奇。

昆十年前进京出任财政部副部长，一年后，老同
学立峰也由天津调进发改委任副主任。

因立峰的身份(闽系),昆和蓝连续扶当财长，同
时也是立峰自已金融财政领域队伍的搭建。

除了昆，还有一位立峰的干将同样得到擢升一
王江，作为已经受历练的“金融副省长”,今年
春天刚刚上任光大银行一把手，不到半年后担
任年初新成立的中央金融委员会办公室的二把
手(分管日常工作)。

这次金融会议官方通告中，也在文末提及“中央
金融委员会委员……等，参加会议”,算是让这
个新部门完成了亮相。

王江，与立峰和昆一样，均毕业于厦门大学经
济系(财经系),师从“社会主义财政理论家”邓
子基。

过去二十多年，一大批受教于清华大学经管学
院(1984年时任国家经贸委副主任的猪创立的)
与北京大学经济学家厉以宁的官员们，把持着
财政、金融、国税、货币等诸多要津，个别厉以
宁的高徒甚至平步青云、直达天庭……

而现在，随着团派人马的离开历史舞台(此前文
章《落幕注脚》已详述), 清华(猪系)、北大(厉
系)的经济学也将随着导师与其高徒们的相继辞
世而挥别共和国历史。

作为领袖干将的立峰，及其金融队伍的核心骨干
则基本实现落位，国家经济学界有望吹来新风。

后续其工作成色，我们拭目以待。

期望不要再辜负领袖重托，短短半年时间内已有
