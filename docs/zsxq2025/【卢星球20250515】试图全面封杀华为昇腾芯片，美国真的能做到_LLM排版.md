
# 试图全面封杀华为昇腾芯片，美国真的能做到

## 文／卢克文工作室嘉宾风雨如歌

当地时间5月13日，美国商务部下属的工业与安全局，发布了一份《指导意见》，其措辞，可谓空前严厉。

《指导意见》表示，”在全球任何地方使用华为昇腾芯片均违反美国出口管制。”此话一出，被很多人理解为，美国要彻底禁止全世界使用华为昇腾芯片。昇腾芯片，就是华为推出的系列人工智能处理器，用于服务器、小站、集群等等场景，如果简单粗暴一点，也可以理解为我们日常所说的显卡。

那么，这到底是怎么回事？如此荒唐的举动，真能成功？

事情，还得从今年一月说起。

当时，在拜登的要求下，为阻止先进GPU流入中国，美国工业与安全局(BIS)制定了一份《人工智能扩散暂行最终规则》（后面简称：《规则》），拟于5月16日生效。

《规则》的主要内容，是将金球国家和地区分为三等。

第一等，是18个主要造成只包括日韩、五眼成员、德法意等等，购丢笑茛啪GPU完全不受限制。

第二等，感梦鍔个国家，主要是亚非拉，每年只能从美国购买最多5万片GPU,如果要多买，得向美国申请豁免，流程非常麻烦。

第三等，是中俄等22个国家，几乎不能买。

这种情况下，你还出台《规则》，限制一百多个国家从美国购买先进GPU，甚至完全买不到，那不是把大量市场，拱手让给中国厂商？

甲骨文、微软、亚马逊和Meta等多家龙头企业认为，”《规则》施加了过多的监管负担，扼杀了创新”，并将市场领导地位拱手让给了中国。

《规则》还引来了许多县梦不满。

令、以色列就被列为了悠色息究以犹太人的暴脾气，必然不能忍受。第云等要〉｀还包括了所有中东海湾王爷国，这些国鸯贵暴韬，尤其是与特朗普的关系，可得很密切的。

人家乐意被列为第二等？他们强烈要求，被划入第一等。

如果美国答应，《规则》就起不到遏制中国的作用。

拜登之所以把海湾王爷国、新加坡等列为第二等，就是因为这些国家进口了大量美国先进GPU,再转手卖给中国。

比如新加坡，贡献了英伟达营收的15％，谁都知道，新加坡本土没那么多市场需求，就是当转口贸易商。《规则》一旦施行，中间商就当不成了。

”美国工业与安全局承认，《规则》会损害美国”与数十个国家的外交关系。

在国内外的强烈反对下，特朗普政府只得用最新出台的《指导意见》，替代了《规则》。

相比针对全世界的《规则》，《指导意见》只针对昇腾，范围缩小了很多。

昇腾在售GPU有两款，性能虽然不及英伟达，但已是市场上的老二，仅次于英伟达。拿GPU市场份额来说，英伟达在中国大陆占据了90％左右。

理论上，美国废除《规则》后，有利于市场多买英伟达的GPU，但这只是理论。经过美国多年的限制，相信纯自由市场的人越来越少了。

就算你今天不限制，谁知道明天会不会？谁也不想吊死在一棵树上，万一今天华为昇腾没了，让你达成了垄断，明天你又开始限制，怎么办？没人敢赌。

其次，最重要的是，昇腾已经有了市场竞争力。

在DeepSee戏匡工之前，大模型的路子是堆硬件，谁的硬件性能强大，显卡数量多，谁就占优势。

简单来说，就是力大砖飞。

但在DeepSee戏匡［之后，全世界已经意识到，不需要那么强、那么昂贵的硬件，也能开发出性能不错的大模型，算法优化是一条可行的路径。

既然如此，对许多买家来说，就没必要再买太多昂贵的英伟达GPU了。

买高性价比的中国GPU，比如昇腾，也不错。昇腾910C巳能满足部分需求，下半年，昇腾920可能推出，性能接近英伟达的H20。

而价格上，则要便宜不少。对任何买家来说，高性价比的产品都是充满诱惑力的。国产用50％的价格，实现英伟达80％的性能，这就是性价比。

正是以上两点，使得昇腾这几年的发展速度很快。

然而仅仅两天后，美国便针对昇腾，出台了措辞严厉的《指导意见》，显然，他们不会放弃遏制中国，特别是在Al等高精尖产业上。当然也不会放弃脱钩的意图，美国财政部长贝森特在瑞士谈判后表示，我们不想和中国全面脱钩，而是希望实现战略性的断裂。

明眼人都知道，全面脱钩和战略性断裂，毫无区别。

说到底，特朗普政府还是想全面脱钩。

而《指导意见》的出台告诉我们，瑞士谈判取得的成果，充其量算中场休息。

残酷的下半场比赛，即将开始。
