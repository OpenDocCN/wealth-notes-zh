
# 普通人突围之路（六十八）

## 一、关 系
道理很简单啊，他犯不着为了你，做一个对他自己不利且有害的决策，或者说仅仅是为了你向往的什么职场公平。公道从来不在人心，是非关乎利益，人的身体永远是诚实的，都是根据利益做决策的。
你想啊，你入职一家公司，人家上来就让你签合同是不是?这是在干嘛?在给你谈未来你们之间的利益分配啊。包括你娶媳妇，你嫁老公，婚姻法里面有没有一个字提爱情?没有。有没有一个字提公平?也没有。从头到尾提的 全是什么?全是财产分配，全是利益分配话题。
话说同事关系是这么产生的，夫妻关系也是这么建立的。你所有的脱离了血缘之外的，新的
人际关系，都是通过利益合作，利益分配产生的。
你去商店买东西，或者你去卖东西， 一买一卖之间，发生了什么?发生了一次合作。有一次交易记录意味着你们有过一次合作，有过一次合作意味着你们有过一次信任。没有这东西意味着什么?意味着你们是路人，意味着你们没关系 。
包括你们的问题，你们的留言，助理为什么会转给我?我不要工资，难道小助理也不要吗?没有你们的付费，小助理都养不活，哪来的这个圈子。这依然遵循着最基础的人际关系的底层逻辑。

## 二、案 例
说到这里，之前有个读者还是个学生，也是类似的问题，他是硕博连读，心态最近被导师搞 的有点抑郁，如何走出困境?
防止被导师白嫖的方法，就是要"及时兑付"。这就像借钱一样，钱一旦出了你的手，名义上是你的，实际上已经不是你的了。你要想把这钱要回来，你付出的时间和精力成本已经远远不止你借出的钱了。
他当学生，他暂时还没挣钱，但他的时间、劳动、智力成果，也是"钱"。而他，就是被导师白嫖了。被白嫖，然后他再想把钱要回来，却被人说心眼小。
正确的做法是什么?当你每支付一笔钱时，就要对方把他的代价押上来。为了表示诚意，你可以多付20%。但是对方一定要抵押。生意要一笔一笔地做，不要对方啥都没给你，你就把一笔巨款借出去。
你在课题组，做出一个成果，就要学会要权，要钱，要管人。老板给权，给钱，让你参与分配研究生的文章如何发表时，再进一步提供价值。你老板还要人干活么?他需要你，就要"付费"。每一笔结清后，再谈下一笔。长此以往，你老板就在心里瞧得起你了，不会把你当傻子了。

## 三、方 式
其实无论是做生意，还是上班与同事们处关系，本质都是和人打交道。那一味的积德行善就能保证你一直往上走吗?非也。能否改变命运要看你的方式正确与否，即是否走在正确的道路上。
其实和人打交道，正确的方式说难有点难，说简单也很简单，是什么呢?我先给你一点好处，然后再问你要点好处，你给，我就再给你一点好处，从此进入良性循环的周期；如果你不给呢，那咱的好处就中断。
就这点事，简单吧。我会提前示好，主动帮你 个小忙的，然后我就会要求你也来帮我个小忙，如果你不帮，那咱俩合作就中止了。
其实万事背后都是投资，都要算投产。这在投资里叫啥?叫仓位管理。这么点小事，很多人一辈子都不懂，非要瞎搞。他要么就做个老好人，同事请他帮忙，他马上就倾尽全力，就像个满仓梭哈的韭菜，自己的事情也不顾了。等到经理叫去训话，批评他的工作没有完成，他急了，怕了，请同事来帮忙，同事不来帮，还看他笑话。他一下子遭遇打击，整个人都灰心丧气。从此对所有人都抱有敌视的态度，自我封闭，再也不肯对外接触，很快，这种人就会沦为窗边族。
其实很多这种人的，他们到了中年还混得一塌糊涂，还老是想不通。觉得我好心，也没有好报，那我自我保护，怎么也还是坐冷板凳呢?对呀。这在投资上又叫盈亏同源，因为你不懂盈亏同源嘛。
你完全把自己封闭起来了，你的确不会再被那 些小人同事晃点了，但是你也失去了所有获取人脉的机会，你也就再也无法进步。做生意的 人是一样的，投资的人也是一样的，第一次亏了，第二次怕了，再也不敢了，于是一辈子做不下去 。
你指望说完全不遇到拖欠货款的客户，完全 不遇到小人同事，根本不可能(水至清则无鱼)。你要拒绝一切，你等于堵死了自己的生意，堵死了自己的职场上进之路。你真正要做的是仓位管理。要做到始终在盈，吃小亏赚大便宜的这么一个状态。这才是解决问题的唯一办法，这才是不把自己的路走进死胡同的唯一办 法 。
那在职场里，具体怎么个仓位管理呢?简单讲就是，你是否习惯于经常和上司对齐颗粒度。
首先看接收端。不管外界给你来多少需求，无论 是直接上司的，还是来自于产品经理的，还是来自于项目经理的，还是来自于本部门或者其他部门同事的，总之，你先来者不拒。即别人要发需求过来，那是他们的权力。问题是，做不做，什么时候做，你是要排序的。
什么意思呢?哪怕你名下挂了7件事，你当下也 只做1件，做哪一件?做能够决定你职业生涯的那一件。通常来说，就是你把7件事拿给你的顶头上司看，让他来决定哪件优先级最高，就是哪件。道理很简单，因为是他决定了你的考评，并不是那堆产品经理项目经理以及隔壁同事。
那什么情况下才会有例外?除非有一件事，是 大老板直接安排给你的，你要直接向老板汇报的，那这个可以压倒一切(毕竟他站得最高 ) 。

## 四、误 区
很多人他是一种什么工作方式呢?是谁来都可 以打断他。谁的语气急迫，他就先做谁的事。好比周一，他的顶头上司交代给他一件重要的事，要他周三务必完成。他也答应了。但是到了周二，忽然某个项目经理一通很焦急的跟他 说要救场，他就救场去了。救完场，到了周三，他交不出顶头上司要的东西。他就这么号人。
搁你手下，你也不想要这种人，因为他拎不 清。而你这个当领导的，永远都不知道你这个手下到底在干嘛，有没有按照你的命令执行任务，还是执行着执行着，就出了偏差。
除了以上说的，你的工作是要有排序的(优先 级排排好),更重要的是，你还要让你的上司 可以做到根本不需要联系你，就能随时随地的 掌握你的动态。他只要随时访问公司内网或者 你的工作日志(行事历),打开你的状态栏，
就能够知道你昨天做了什么,今天要做什么。
他的这个感受你能理解吗?就跟你玩游戏一样，你在操控那个游戏人物，你随时随地点开 那个人物，就知道他现在在干嘛，干到什么进度，未来还要做什么任务。
你可以想象一下，作为一个管理者，遇到这 两种不同的下属时的心情。一种是你完全可以像 控制游戏人物一样方便；一种是你都弄不清那 货在做什么。换你，你选谁?

## 五 、视野
除了这些工作以外，就能提拔了?非也，你还 要知道项目的全貌，比如部门与部门的关系，比如主管们之间的关系，比如公司的方向，行业的 变化，都要关心。
比如说你上面的俩主管，在更高级的领导面前，是竞争关系。明白这里面的关系吧?就好比旅长让李云龙守住阵地，阎长官通过楚云飞一通忽悠，李云龙就被人家当枪使了，损失了 不少自己的弹药士兵不说，一回头，还发现阵地没了。
回来被旅长往死了骂，李云龙还很委屈，我一 个当兵的，我咋知道你们上头有矛盾呢?换你 是旅长，你还能留着这种李云龙么?
他已经闯下弥天大祸了，他自己倒委屈上了，觉得他只是一个干技术的，他怎么知道上司和 上司之间的各种竞争关系，各种内部斗争....这种人你觉得他会前途么?绝对不会有。他的 问题不在于某一天因为昏庸而犯错，无意识拆 了上司的台。即便他不犯错，这辈子也绝无 可能升上去。你想一想，你的上司通常会带什么 人去见更大的老板?就是你能够跳出自己工作 的局限，平常就对项目，公司，行业的变化，事事关心，时刻留心的人。
