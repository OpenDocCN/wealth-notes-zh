
# 货币、空气币、瑞幸收割

第一课讲了黄金，黄金天然不是货币，但货币天然是黄金，因为贵金属本身有价值，值钱，不像今天的纸。货币的发行人都是国家，国家铸货币和维护社会运转要不要成本。要，那么多官员，水利设施等等，成本很高。国家主要收入来源只有两个，一个是收税，一个是铸币。

国家铸币本身就是不公平交易，比较隐蔽，不好看的出来，变相的称为：铸币税。古代的货币，基本都是贵金属，货币的重量材质基本就是价值。如果那样流通起来，国家一分钱好处没有。怎么办呢?掺和东西。

以汉朝的铜钱五铢钱为例，重3.25克左右，铜70%，铅20%,锡10%。铜之外的30%不值钱，但他价值跟铜一样，这就国家的便宜，那30%就是铸币税。五铢钱用了700多年，好几个朝代，算是良心货币，只收30%的铸币税。

当时有很多民间和诸侯私下偷偷铸铜钱，比如铜60%,其他40%,他不也就收了40%铸币税吗?重量大差不差，一般人摸不出来。私铸货币是和国家抢钱，历朝历代都动不动诛族。

到了王朝末期，很多皇帝胡来，铸造“当十大钱”,用一个薄薄的大铁钱，基本不含什么铜，“当十”，就是值10个小钱的意思，强行规定这个值10个五铢钱，甚至还有当百的，老百姓又不傻，发行这个，大家把好钱都藏着，不花，或者以物易物，市场上流通的全是垃圾货币。这个叫做“劣币逐良币”。

在公司，单位和职场里，没能耐受重用，能干活的
人才就不说话了，这也是劣币逐良币，社会现象，说远了。

所以，之前的货币都是有抵押货币，抵押就是那个贵金属。纸币，一开始是作为铜钱提取券存在的，最早的纸币是中国四川发行的“交子”,四川都是山，带铜钱重，翻山越岭太麻烦。

001课讲过，1974年以后，纸币就变成纯信用货币了。纯信用的意思是，它没有抵押，不能逆向换回贵金属，这个信用是什么呢?刷国家的脸，国家背后实力，持有矿山，铁路等资产。

100英镑钱纸币，印刷成本1元，99块钱都是铸币税，但不能单纯这么看，国家各项开支和设施建设，都是有成本的。
