
# 基础教育加大科技权重，普通家庭怎么应对？

基础教育正在加速“科技化”，这背后是国家对未来科技人才的提前识别和选拔。以下是对这一趋势的分析和建议：

## 基础教育加速“科技化”

2025年1月，中央发布的《教育强国建设规划纲要（2024－2035年）》提出要“面向数字经济和未来产业发展，加强课程体系改革，优化学科专业设置”。这标志着基础教育正在不断增加科技或者说科学的权重。

### 三大趋势

1. **成体系地增加人工智能等科技类课程**

   国家早在2017年就提出在中小学阶段设置人工智能相关课程。2024年，教育部进一步明确要“进一步完善相关课程中人工智能教育要求”。课程体系从小学低年级的感知和体验，到高年级和初中阶段的理解和应用，再到高中的项目创作和前沿应用，形成了一个体系化设计。

   地方层面也在快速跟进。例如，广东省发布《人工智能赋能基础教育行动方案（2024– 2027）》，上海市和北京市也发布了类似方案，旨在构建多层次的人工智能教育课程体系。

2. **选拔机制的改革**

   “科技特长生”正在成为新趋势。这类学生通常在科技创新实践活动方面有一定特长，如机器人科技特长生、信息技术科技特长生等。地方还在构建全新的培养模式，例如上海市教委、同济大学、黄浦区人民政府合作共建同济大学科技中学，北京市朝阳区与北京化工大学等高校联合打造北京中学科技分校。

3. **兴建科技中学和科学中学**

   除了新建一批科技中学、科学高中，一些老牌科技中学也开始推动“联盟化”。例如，深圳科学高中与北京、上海等地的18所科学特色高中成立全国首个“科学高中联盟”，旨在探索科学教育的大型试验，为培养顶尖科技人才做准备。

## 科技人才的识别和选拔越来越前置化

加大基础教育的科技权重，既是国家对科技人才的提前识别选拔，也是应对技术冲击的政策托底。其背后的政策意图至少可以分为两个方面：

1. **修复应试教育的弊病**

   大规模应试教育虽然能够标准化地衡量能力水平，但也导致对科技人才识别不足，且资源过度投入到标准化应试中。因此，通过“强基计划”等选拔机制，以及科技中学等新型高中，将更多天赋型科学苗子放在一起培养。

2. **应对技术冲击的托底行动**

   让大多数孩子能感知和适应未来的科技社会，掌握一些基本的科技技能，避免成为时代浪潮的弱势一方。

## 三条建议

1. **避免过度投机心态**

   虽然科技特长生很重要，但建议家长理性评估自家孩子的能力和兴趣，不要抱着太多的投机心态，去赌“科技特长”这条路。

2. **让孩子有更多的基础科技感知和科技思维**

   未来没有纯粹意义上的文科生甚至艺术生、体育生了，类似人工智能这样的技能，未来可能是所有学科和工作的基础。

3. **补强“非科技基础能力”**

   跨学科思维和团队协作等基础能力在整个职业生涯中更可能获得更高薪资、晋升到更高职位、更快掌握专业技能，并且对行业变革具有更强的适应能力。

## 结语

基础教育加大科技权重是大势所趋，家长应理性应对，既要重视科技教育，也要补强其他基础能力，帮助孩子更好地适应未来的科技社会。
