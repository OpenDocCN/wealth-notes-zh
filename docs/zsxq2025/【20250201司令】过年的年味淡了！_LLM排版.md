
# 过年的年味淡了！

所有人都在回忆小时候过年的那种浓浓的年味，都在感叹岁月流逝下时间冲淡了一切，包括记忆中的年味。大家越来越不太注重过年的仪式感，许多禁忌和习俗也在时代的变革中以不合时宜的理由逐渐被有意无意的忽略掉了。人们都认为这是一种与时俱进的移风易俗，同时又感叹年味不像以前那样浓烈。

女生都喜欢仪式感，认为生活的幸福却决于各自仪式感是否到位，然而，没有哪个女人愿意在过年的时候去追求那些已经传承了千百年的仪式感。为什么呢?因为那些仪式感真的需要自己动手，越是精细到位就越需要付出太多的时间和精力去忙碌。

人性的追求总是充斥着天然的悖论，你渴望有年味，但你不想像父辈那样遵循着古老的传统习俗所规定的繁琐仪式。你更不想别出心裁的精心准备食物去招待前来拜访的亲友，你更不太愿意在大年的前几天密集的穿梭拜访各路认识和不认识的亲戚，也讨厌各自客套的寒暄，甚至不太愿意跟‘陌生人'说话，更愿意独自玩手机。

记忆中的年味之所以很浓，是因为你以充满期待的心情参与了许多给你带来快乐的事情：走亲戚能收红包，看舞龙舞狮能够凑热闹，贴春联和穿新衣服，以及大年初一精心准备的早餐，这些充满仪式感的美好，都是不需要你参与其中的，你只需要等待它自然而然的发生，就能得偿所愿，然而，你想过没有，是谁在你醒来之前就把这一切都准备好了！

你所喜欢的仪式感是不需要自己参与其中，不需要为此付出代价就能轻松愉悦的享受现成美好的期待。当你不再是你那个收红包的人，而成了那个发红包的人之后，你就对红包也不那么期待了。当你成了为一批又一批到来的客人整天在厨房忙碌的人，而不是那个拼命吃零食，被亲戚追着塞红包的人之后，似乎也不再那么期待亲戚们的到来，那份热闹和温馨对你而言变成了无法回避的繁重和辛劳！

世界在一夜之间就长大了，但许多人还一直活在十几二十年前的心态中，还认为自己是个宝宝，而不是为别人准备惊喜，创造仪式感的当家人。昨天听一个朋友说起：他妹妹今年都三十了，当亲戚去家拜年的时候，她既不起身招呼，也不去做任何准备，而是抱着手机头都不抬的说了句：大人不在家，刚出去了，你们下次再来!其实他和他父母就是下楼去送上一批客人了，只要她稍微招呼一下，他们就回来了。我不觉得她妹妹是因为太有个性才如此的拽，她的反应更多的说明她确实不知道该如何应对这个局面。在她自己这30年的人生中，可能一切都被父母照顾的太好了，以至于她真的认为自己还没长大，人情世故也好，生活日常也罢，都与自己无关。自己是那个享受结果的人，不是那个需要去应对过程的人，所以，她才会脱口而出说了句：大人不在家!她没有意识到自己已经是大人了，在她的意识中，她会本能的认为支撑起一个家的正常运转所需要解决的各种问题都是‘大人'的事，与自己这个‘小孩’没啥关系。

其实，他妹妹这种心态和这种处理问题的方式，并不是独此一人，而是在某个年龄段的群体中普遍存在的状况。在时代进步的浪潮中，会自然而然降低对普通人的要求，从而让许多人可以一直保持幼态。你可以说这是一份纯粹和率真，也可以说它是一种角色和年龄的错位，这两种看法取决于你从什么角度来判断。如果你试图以一个收红包的人的角色来看待过年的氛围，确实年味淡了，因为到了这个年纪，外界对你的期待变了，外部世界不再会主动去满足你的期待，反而期待你去满足它们的各自期待，同时，也不再对你像孩童时期那样包容。

我总认为人在外部环境的裹挟下被迫到了某个年龄就去干某件事是不对的，许多人并没有准备好去承担那些角色的准备，如果贸然去扮演那个角色，必然会无法胜任，而无法胜任的自身状况也必然担负不起那个角色的一系列责任，这会让没准备好的人变得更加痛苦。就好比上文提到的那位兄弟的妹妹，虽然已经到了结婚的年龄，但她的心态和自我角色的认定显然还没准备好，不是没有准备好扮演妻子和母亲的角色，而是还没准备好扮演大人的角色!如果周围人强行让一个‘小孩’和另一个‘小孩’组成一个新的家庭，这和闹着玩的过家家有什么区别呢?两方的大人都会不放心的，往往会主动和被动的卷入其中，并且过度干预新家庭的事物，甚至越俎代庖的去处理和制造新家庭中的各自矛盾——最后肯定会一拍两散…因为这个‘家'其实自始至终都与此连个‘小朋友’毫无关系，是在外力的作用下维系的，不具有生命的特征，无论是外力使劲不够还是过于使劲都会让其彻底碎裂，而两个当事人，不仅没有能让其自动运转的能力，甚至连这样的想法都没有。

不要强迫那些缺乏能力又没有准备好的人去做其无法胜任的事，那是一种对所有相关人员的残忍。只有自己会主动为仪式感做准备并付出代价的人才有资格说仪式感对自己的生活很重要!
