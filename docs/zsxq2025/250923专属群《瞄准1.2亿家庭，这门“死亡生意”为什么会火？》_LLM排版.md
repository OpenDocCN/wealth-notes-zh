
# 瞄准1.2亿家庭，这门“死亡生意”为什么会火？

今天我们要说一个你可能从来没关注过的领域，宠物殡葬。你可能觉得奇怪，这么小众的领域，有什么可说的？根据企查查的数据，截止到2025年6月，全国企业里提供宠物殡葬服务的，也不过8000家。但是，假如仔细观察，你会发现，这8000家公司里，绝大多数都是在三年内注册的。其中光是2024年下半年，就有1000家公司成立。什么概念？平均一天成立将近6家公司。宠物殡葬的市场规模也一直在持续增长，而同步上涨的，还有服务的价格以及人们的付费意愿。

前几年说到创业，很多人动不动就把猫粮挂在嘴边，好像猫粮就是普通人创业的捷径。而现在，宠物殡葬呈现出的势头，虽然不如猫粮，但也相当值得留意。

为什么？《人类简史》的作者赫拉利有一个观点，过去的商业竞争，争的是用户的钱包份额。而未来的竞争，争的是用户的关系份额。也就是你能多大程度上和你的客户建立亲密关系，以及你能在多大程度上，服务于你客户现有的亲密关系。而宠物殡葬业，就是建立在宠物与主人的亲密关系之上。从这个角度看，你或许可以把它当成一个“关系生意”的样本，从中也许能看到一些新的商业可能性。

接下来，我们就从几个问题出发，来了解一下宠物殡葬行业。

先强调一句，宠物殡葬目前并没有被纳入《国民经济行业分类》，严格来说，它还不算一个行业，只能算是一类服务。今天为了讲述方便，有时我们会称之为行业，这只是个口语表达，你知道就好。同时，这个领域处在发展早期，很多规范还有待建立，后续它也需要在满足行业规范的前提下发展。

好，咱们正式开始。

先交代个背景，宠物殡葬的增长，来自宠物本身的变化。简单说，中国的宠物正在发生三个变化，变多、变老、变重要。

首先，变多。根据《2025年中国宠物行业白皮书》，现在我国城镇养宠家庭1.2亿户，占全国家庭总数的23%。就在去年，我国宠物的数量还首次超过了4岁以下的婴儿。

其次，宠物不仅在变多，它们的寿命也在延长。这就导致出现了很多老龄宠物。目前，国内7岁以上的老年犬已经占到整体的23%，老年猫的比例是11%。预计到2026年，我国老年犬猫的数量将超过5300万只。

最后，为什么宠物寿命变长？很明显，是人们越来越重视宠物。要定期体检，有病要及时治疗。现在很多人都管宠物叫“毛孩子”，从这个称呼就能看出对它的重视。

之前，消费产业投资人黄海老师去日本考察，他就发现，日本宠物店的装修几乎跟母婴用品店完全一样，而且店铺提供的品类也差不多。而回到我们国内，很多人也开始给自己的宠物购买小衣服、推车、婴儿床等。黄海老师就说，母婴店里卖的婴幼用品，宠物行业也可以照着做一遍。

而在经历了宠物老龄化之后，宠物服务的下一站，就进入了殡葬阶段。根据《2024年中国宠物管理白皮书》，在我国的养宠人里，62%的人愿意为宠物殡葬支付500元以上，超过60%的人认为，宠物殡葬仪式相当于“与家人告别的重要仪式”。

在这里要提醒一句，在宠物殡葬领域，有两个背景信息需要特别留意，这是了解整个行业的前提。

第一，根据相关的规定，宠物去世之后，主人是不能私自找个公园之类的空地随便埋掉的。第二，假如想买一块专门的宠物墓地，价格不低。之前市面上也有些提供宠物殡葬的，收费虽然低，但他们提供的准确说法不叫殡葬，而是叫“动物无害化处理”，充其量只是替主人处理掉动物遗体。

而在“草草处理”和“花大价钱买一块墓地”，这两个极端之间，就留下了一个巨大的空白区间。既要给养宠物的人提供足够的情绪安抚，让他们觉得宠物被郑重对待，同时费用又在人们的可承受范围内。现在很多提供宠物殡葬服务的公司，瞄准的就是这个区间。

但问题是，宠物殡葬具体可以做什么呢？这主要得看，养宠物的人都遇到了什么真实问题。毕竟，有真问题，才有真需求。

从这个角度看，宠物殡葬业的机会，也许隐藏在这么几个问题背后。

第一，透明问题。宠物殡葬业一度有个乱象，一些公司收走宠物遗体后，直接就转手交给了其他机构，用很低的成本去做“动物无害化处理”。至于宠物主人拿回的骨灰，可能根本就不是自己宠物的，有可能都不是骨灰，而是草木灰之类的。

怎么解决养宠人的担忧？很明显，最直接的方式就是把殡葬环节透明化。

比如，有的公司会花很大的成本，专门建一座纪念堂用来完成宠物火化。宠物主人可以透过玻璃，全程观看自己宠物的理容、告别、火化过程。而且还有服务人员提供一对一讲解，告诉宠物主人每一步的细节。还有公司推出了“一宠一炉+全程录像”的服务。就是每只宠物单独火化，整个过程全程录像，宠物主人可以通过手机App实时观看。

宠物主人一般愿意花高点的价格，来买这份安心。

换句话说，这些公司赚取的，是客户的“安心溢价”。这个逻辑放在其他行业或许也成立。假如维修人员能让客户看到自己的维修过程，家政行业能让雇主看到保洁的打扫过程，没准都能获取一份额外的“安心溢价”。

回到宠物殡葬，透明化的火化场所，一般需要一笔不低的建造成本，入行门槛比较高。因此，多数公司都会瞄准另外一个门槛相对低的环节，叫，仪式化的告别。

这也是宠物主人想要解决的第二个问题，怎么让告别更有仪式感？

在这方面，从业公司已经做出过不少相关的尝试。

比如，给宠物制作详细的“生平档案”，包括宠物的性格特点、生活习惯、与主人互动故事等。在最后的告别仪式上，还会有工作人员朗读这份档案。

再比如，一些机构还推出了“宠物生前规划”服务。像人类的遗嘱一样，宠物主人可以提前为宠物规划身后事，包括火化方式、纪念品制作、告别仪式具体安排等。

再比如，还有些公司会为宠物主人制作纪念品，或者提供定制服务。比如，骨灰DNA保存技术，也就是从宠物骨灰中提取DNA信息，永久保存。据说这项服务单价接近一万元。再比如，在宠物火化后，把骨灰制作成钻石、吊坠等纪念品。

再比如，很多机构会定期举办“宠物追思会”，专门邀请一些曾经失去过宠物的人分享回忆、交流情感等。这些机构的服务不仅停留在殡葬本身，还会运营宠物主人的社群。

除了这些专门做宠物殡葬的公司，还有一些公司是利用原本的能力，切入到宠物殡葬的某个环节。

比如，景德镇有一家陶瓷工厂，之前一度濒临倒闭。后来他们转型做了宠物殡葬用品，包括陶瓷骨灰盒、雕像、墓碑等等。据说产品出口到美国能卖到几百美元，是国内价格的10倍以上。宠物殡葬，成了这家公司的第二曲线。

再比如，山东曹县的宠物棺材。曹县棺材厂的老板们，把原来的棺材产业链扩展到了宠物领域。而且这些产品还很适合出口，比如在美国，有些宠物遗体是直接埋掉的，宠物主人就特别看重棺材的防潮密封。因此，曹县的厂家就基于这些需求，对产品做了改造，再加上价格优势，很快就在当地打开了市场。

再比如，绿色殡葬。很多人希望宠物的身后事不仅体面，还要环保。这就催生了对生物降解骨灰盒、树葬、海葬等产品服务的需求。还有“数字化纪念”，这两年也渐渐成了趋势。一些年轻的宠物主人想给宠物建立专属网站、制作纪念视频、创建虚拟墓园等。

这些需求都催生了各式各样的新机会。很多企业并没有直接做殡葬服务，但都在产业链的某个环节找到了自己的位置。

换句话说，当一个需求快速增长时，很多配套还来不及出现。这就导致整个产业链出现了很多空白环节，而对于后来者来说，机会可能就隐藏在这些空白里。

今天，关于宠物殡葬业，我们说了不少相关的尝试。有些内容听起来可能有点零散。但是，回到本质，我们想说的重点不外乎三句话。

第一，很多服务型产品，或许都存在“安心溢价”。就像宠物殡葬一样，让用户看到服务的过程，这本身也是在提供服务。

第二，仪式是情绪服务的具象体现。在宠物殡葬领域，很多情绪层面的服务价值，都需要通过仪式来交付。这放在其他行业或许也适用。

第三，在一个高增长的行业里，产业链上的每个环节都可能有机会，关键是针对自己的能力，找到适合的位置。
