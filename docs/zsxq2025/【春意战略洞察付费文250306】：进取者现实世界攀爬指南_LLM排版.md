
# 进取者现实世界攀爬指南

我喜欢爬山，工作之余爬了很多山，不仅锻炼了身体，还增强了自己各方面的见识。爬山之余，我经常会思考一个问题，社会这座山是怎么样的？普通人又该如何用自己的时间和精力爬这座现实世界的高山？

我们将这个话题分为三个部分：认识这个世界，明白这个世界，打造你的世界。

## 一、认识这个世界

从贫穷的农村起步，经过漫长的奋斗，看看现在的生活，和起步时已经跨越了好几个层次。可我抬头向上，所谓【九重天】并非戏言，上面的层级，甚至比下边的层数还要多。

很多年前，互联网上流传过一个非常热的文章【中国社会的九个阶层】。我们今天只谈财富，只按照财富来划分层级。文章的作者他自己的财富水平很有限，所以他的很多描述是错的。因为咨询的关系，我这方面的阅历比他深厚多了。因此，我描述的客观世界，应该比他更符合世界的现实。

我们先来看一张表。

这里第一列净资产也就A后边的数字是资产的量级而已。

### A5层

看完表格，很多人第一个反应是A后边的数字是什么，怎么还有A呀？难道真有谁的资产只有几万元？是的，这一层是中国底层中的底层，是社会上的残障群体。据统计，中国有1300万盲人，8300万残疾人，约占总人口的6%，虽然这些人不一定人人贫穷，但大部分人还是占据了中国社会的底层。

除此之外，还有老少边穷地区，大凉山、墨脱、阿里等不通公路的地区，贫困人口多的是。具体我就不多介绍了，大家知道存在这个阶层的就行。

### A6层

我们来看这个层级，财富总量十万到几十万的量级。中国国际金融公司曾经发表过一份报告。2023年的数据，中国的社会财富总量790万亿人民币，其中国有财富总额为360万亿元，占45.5%。私人财富430万亿元，占比54.5%。平摊下来，每个人平均56.4万，只计算私人财富的话，平均约为30万元。

我们知道，统计数据中，一个社会的基石是很难有大的变动的。占比最大的这一级，足足有13亿，其平均财富是63万，这个数字应该不会差太多。

当然，这个级别的人群，还可以继续细分：工人、农民、教师、营业员、中小店主.....这些人互相之间，也存在鄙视链。

### A7层

比A6更高的是A7。这一级是京城人士，或现代中产阶级。在中国，虽然大家表面上不说，但心里都心知肚明。京沪人士看外省的吃穿用度时，内心也往往生出【乡下来的】眼光。

除此之外的其它城市，像成都、杭州虽然号称锦官城、人间天堂，但我去过几次，在细节上跟京沪还有很大的差距。真正的差距在产业。中国的产业分布是很不均匀的。各种Bat、vc/pe，只有在这里才有大量的高薪职位。

### A8层

之上是通俗的讲，他们是各领域的精英。你的公司的老板，大点公司的CEO、CFO、还有知名媒体的主持人，武磊、窦文涛都算。按照中国目前的标准，他们的收入大概每年150万左右，当然好点的也有400-500万，甚至更多。

### A9层

A5是残疾人，A6是普通人，A7白领，A8是天才。天才之上是什么？上面还有五层。难道还有人，读书比天才还好？难道是博士、院士？

不是，读书读到A8这个程度，已经到顶了。而要成为A9这个阶层，需要其它的东西。前面的表格中，A7、A8的成长历程是一样的，都是读书。只不过一个是庸碌平凡的学生，一个是顶尖变异的学霸。

### A10层

再往上一阶，是A10，因为你想要升阶，需要换新的模式。举个例子，目前双色球头奖是人民币500万元，扣除20%的税，实际到手大约400万元。有这么一笔几百万的横财，无论你是A5、A6，还是A7，衣食住行各方面都能得到改善，甚至临死之前还可以拍一部几十集的财产争夺电视连续剧。

但是愚蠢的人类呀，你知道A10是个什么概念吗？10亿元意味着从今天开始，你每个月都要中一张500万的头奖，一年才能勉强凑满5000万，接下来连续每个月不断中满20年，才能勉强凑满10亿元。

### A11层

A5是残疾人，A6是普通人，A7白领，A8是天才，A9、A10鬼才，那再往上是什么？我觉得是没有。抛开时代，做到鬼才已经到顶了，已经穷尽所有智慧、勤奋和运气。所以，与前者相伴，更多的是时代赋予他的东西。

### A12层

A11再往上，这是能够写出来的最后一阶了。上一个台阶，往往是做出来一个成功影响时代的 产品，比如老干妈、福耀玻璃、海天酱油、携程网，但是单个产品的能量毕竟是有限的。

再往上，你需要有更多这样的产品或公司在自己名下。你需要掌握的不是一个产品，而是一个产业，至于说这些东西允不允许你掌握，这就不是我能写了的。

## 二、明白这个世界

认识不等于明白。看到山头，不代表你能明白山头，更不代表你能找到上山的路。想知道更多，你必须能够看明白这个世界。

社会好比是一台复杂的机器，精力供给、消费循环、债务周期、阶层流动……就像一个个零件，互相咬合，层层嵌套，让世界成为高耸的样子。

文章的第二部分，让我一起看明白这个世界。
