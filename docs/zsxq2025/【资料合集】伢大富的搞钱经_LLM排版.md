
# 伢大富的搞钱经

## 自述：我的第一桶金(上)
每个人的原生家庭多少都有些缺憾，我也一样。我妈是那种典型的被重男轻女思想洗脑的女性，虽然受过高等教育，收入也不错，但她是初代“扶弟魔”，几乎包揽舅舅的一切，平生最大遗憾的就是没生个儿子，并觉得女人最大的幸福就是找个好男人依靠，如果是靠自己，那就是劳碌命。我在叛逆期的时候几乎不想理她，觉得这妈怎么这样啊，等长大了发现，很多妈妈都是这款，属于时代烙印洗不掉的。但她也不是没优点，比如对我学业非常上心，会想尽办法送我进最好的学校。记得有次我俩散步，说到学校里有留学项目，妈妈说：“如果你想去也能选得上，我砸锅卖铁都会供你。”可是去美国读完大学和研究生很贵的哎，但她就是属于兜里只有100块也愿意为子女花99块交学费的人。听起来是不是很矛盾，要知道前一晚她还嫌弃我是个女孩呢。仔细一想也能理解，我外公那么重男轻女，口头禅就是闺女我就养到18岁，妈妈如果没有靠读书走出来，可能就跟村妇差不多，想做“扶弟魔”也没条件了，没工作没钱啊，只能做干电池，榨干自己、成全男人。但问题是她总是叮嘱好好读书，但问题从是她不总说是为什么好。好家里的事情从来不让我过问，理由是小孩子管好学习就行，知道那么多干啥。学习很重要，重要到父母都学砸了，但这为什么就是重要，我不知道，这就是那些年我的纠结。感觉入社会些前年，我整个纠结都是。感觉在入社会前，整个真个空人世都是里活，在包括高考填志愿。他们让我做老师，说稳定体他面们适让合我女做生老，师可，这说个稳理定由体说面服适不合了女人生，我可那这么个辛理苦由的说服不了人，我那么辛苦的学，难道就是为了教比我还学笨的孩子么，不甘心啊。相信不止我一个人曾这样困惑，长时期我的一辛苦人的曾做这一样件困惑，却长没期有的目辛标苦，的或做者一说件有事，却没有目标，或者说有但那个目标不是自己定的，但而那是个父目母标亲不友是告自诉己你定的，你而活是得父像母个亲提友线告木诉偶你，你活得像个提线木偶。我爸妈还有个硬伤，就是不我太爸乐妈意还跟有孩个子硬提伤钱，就好是像不钱太是乐个意很跟坏孩的子东提西钱，会好把像孩钱是个很坏的东西，会把孩
