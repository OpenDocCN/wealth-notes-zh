
# 一场外卖大战，让物价回到20年前

250716
先问个问题，你上周末点外卖了吗？
就在上周末，国内的单日外卖订单，
史无前例地超过了2亿单。什么概念？假如粗暴地按人头平摊，平均七个人里面就有一个点外卖。

外卖订单为什么破纪录？很明显，是因为美团、淘宝、京东三家的外卖大战。至少目前看，这场竞争已经催生了几个“之最”。

首先，订单数量之最。美团7月12日23点后发布数据，即时零售日订单量超过1.5亿单。紧接着，淘宝闪购饿了么宣布，7月12日当日订单超过8000万。京东虽然没有公布当日数据，但之前京东公开表示，日订单量已经超过2500万。假如综合计算，7月12日国内外卖订单的总量，至少超过2亿，实际可能更多。

其次，骑手收入之最。7月12日这一天，有可能是国内外卖骑手单日收入最高的一天。根据美团发布的数据，他们在全国有40万众包骑手，日收入一度超过500元。什么概念？相当于这一天赚到了之前三天的钱。据说还有骑手的日收入超过了1700元。

最后，补贴数额之最。截止到这期节目录音，只有淘宝公布了预计投入的外卖补贴总额，是500亿元。尽管美团单和京东没有公布数据，但从购买端看，红包的额度都相当大，动不动就是满48减28，满30减20，还有很多小额订单出现了零元购。这也直接让线上订餐的价格大幅下降。一份快餐只要几块钱，假如选择美团拼好饭的到店自取，一份快餐的价格甚至能低到3块钱以下。一场外卖大战，让一顿饭的价格回到了20多年前。

现在竞争依然在继续。据说美团的即时零售部门，已经启动了轮流调休，确保全天24小时都有业务骨干在场。美团内部还打出了一句口号，叫“战一夏，定乾坤”。而早在6月，淘宝闪购已经和饿了么正式合并，有点兵合一处的意味。

而戏剧性的地方在于，据说上周末美团本来没打算冲订单，他们本来想维持上上周的数量，只要比淘宝多就行。但就在7月12日傍晚，美团监测到淘宝闪购派发了一批大额红包。于是美团当即决定，启动饱和反击，派发了大量的满减券，而且数量上不封顶。据说当天美团有三分之一的订单，是在启动“反击”之后出现的。

你看，商场如战场，在这一刻是不是突然具象了？对手的任何一个风吹草动，都可能突然搅动整个战役的局面。而且真实的商战哪有那么多的投机取巧？主要比的就是谁火力更猛，谁的补贴额度大。

就像《孙子兵法》，开篇就是《始计篇》。这个计不是计谋，而是计算。算算对手有多少兵力，自己有多少兵力，然后基于刚性的实力去正面迎接。

那么，这场外卖大战是怎么打起来的？各方为什么这么重视？你要是去搜相关的分析，会看到一堆专业名词，什么打通场景、形成闭环、业务反哺等等。今天，为了方便理解，我想讲个你相对熟悉的故事。理解了这个故事，我们就能理解这场外卖大战的逻辑。

这个故事是关于漫威和DC的。漫威，就是复仇者联盟那家公司，有钢铁侠、绿巨人、美国队长等等。DC，就是超人那家公司，他们有闪电侠、神奇女侠、正义联盟等等。

这两家公司已经竞争了很久，有80多年，四舍五入差不多就是100年。按说这两家都是娱乐业的巨头，商战的技巧应该很高明。但是假如仔细看，你会发现他们之间的对抗特别直接。说个不太恰当的比喻，甚至有点像小孩子怄气。你创造一个角色，我也必须须创造一个差不多的。就好比小孩子之间，你买了新橡皮，我也必须买块新的，你买了新衣服，我也不能落后。

比如，论跑得快的，DC有闪电侠，漫威有快银。要身体上有机甲的，漫威有钢铁侠，DC有钢骨。要在海里游的，漫威有海神纳摩，DC有海王。会射箭的，漫威有鹰眼，DC有绿箭侠。要大反派，漫威有灭霸，DC有达克赛德，两个人长得都差不多。你要是想要组团出道，漫威有复仇者联盟，DC有正义联盟。

为什么两家的角色都这么相似呢？其实，他们并不仅仅是在创造角色，他们的本质是在造“门”。每一个角色，都是在某个特定的生态位上，打开了 一扇门。而每个角色所代表的门，最后都通往同一个叫“IP宇宙”的地方。

比如，最近魔法元素很受欢迎，而DC在这个生态位上有个角色，叫命运博士。而漫威假如没有会魔法的角色，就等于失去了喜欢魔法的观众，因此必须创造一个奇异博士来对标。

而不管是奇异博士也好，命运博士也罢，目的之一都是把观众带进他们各自的电影宇宙。你可能会因为看了奇异博士，从而踏进漫威宇宙的大门，进而一口气把漫威宇宙的几十部电影全看了，还可能会因此去买各式各样的周边。这背后的收益远远不止一张电影票钱。

因此，漫威和DC都会紧紧盯着对方，你在闪电侠这个类型上有角色，我也必须有。这不仅仅是为了卖东西，更是在“拉客进门”。

这是个非常朴素且古老的逻辑，但是，偏偏多数商业竞争都绕不过这层逻辑。

比如，今天的外卖大战。每一笔派发的外卖优惠券，都是在试图拉你进门。用互联网行业的话说，这叫“打开率”。你打开一个App的频率越高，你消费的概率就越高。

比如，京东。以前京东的主力产品是3C数码，但问题是，人一年能买几个手机？但相比之下，外卖的使用频率要高得多。据说京东自从上线外卖以来，已经唤醒了4100万180天以上未活跃用户。假如你点京东外卖，它还会给你派发3C家电的优惠券，据说转化率已经达到了32%。也就是10个点京东外卖的人里，有3个还可能会再买点别的。

因此，做生意的关键，是先拉用户进门。

那么，进门之后呢？这些门其实都通往同一个地方，叫“即时零售”。现在电商零售的竞争，在价格和质量层面已经形成了相对稳定的格局。有的主打品牌质量，有的主打白牌性价比。那么，下一个竞争维度是什么呢？很明显，是时间。过去大家比的是让用户用更低的价格买到更好的商品。而现在比的是，让用户在更短的时间用更低的成本获得更大的满足。

现在在电商领域甚至流传一句话，叫“无即时，不零售”。而即时零售的“最后一公里”，也是最关键的一公里，就是从配送员到用户的这一段。谁掌握了这一段，谁就掌握了即时零售的入口。这也是为什么几个平台都不惜下血本给补贴。

听到这，有人可能会觉得奇怪，难道现在的商业竞争都这么没有技术含量了吗？仅仅是发补贴，比谁钱多就行吗？

当然不是，前面说过，补贴是最重要的弹药，但具体的作战策略还是需要仔细思考的。

好多年前，营销大师杰克·特劳特曾经提出过一套商战模型。虽然是很多年前的观点，但是对今天的商战也很有解释力。

特劳特说，商战里一般有两个角色，现任冠军和后来的挑战者。尽管实际上商战的参与者可能很多，但本质上都可以归到这两类角色里。

对现任冠军来说，最重要的任务就是防守，你不能让挑战者在任何一个维度超过你，哪怕超过一丁点都不行。为什么？因为在用户的心智里，你是第一名。而只要被别人超越那么一点点，你就会失去这个用户认知。就像奥运会短跑一样，所有人都只会记得谁是冠军，谁是亚军，没人会记得亚军比冠军到底慢多少，哪怕是只慢0.01秒。

据说美团面对淘宝的优惠券策略，一上来就制定了上不封顶的反击计划。从这个角度看，这也许就是为了捍卫用户心智。

而淘宝闪购饿了么也要保证自己不下牌桌。根据特劳特的定位理论，用户在一个品类里一般只记得两个产品，最多三个。比如可乐，只记得可口和百事。因此对第二名来说，防守也至关重要。

而京东的角色更偏向后来的挑战者。而根据特劳特的观察，挑战者最常用的策略，是瞄准冠军的长处里的短处。注意，是长处里的短处。比如，别人的外卖很强，这是长处。而其中的短处在于，有一部分外卖商家没有堂食。那么你就可以瞄准这个地方发力。为什么不直接瞄准短处？因为对冠军来说，一个明显的短处大概率上无关紧要，否则它自己早就补上了。当然，这里只是举个虚拟的例子，不指代任何具体的平台。

这就好比，我做不了外卖里规模最大的，但我至少得让用户记住，我是外卖里对商家审核最严的。就像营销领域里说的，用户的心智，才是商业竞争的终极战场。

目前，这场外卖大战还在持续中，结果不好说。但至少2025年的夏天，因为这场外卖大战变得更加热闹了。我们也能从中感受到商业的海洋依然汹涌澎湃，依然充满生命力。
