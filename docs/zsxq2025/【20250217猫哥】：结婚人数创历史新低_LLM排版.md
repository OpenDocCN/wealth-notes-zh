
# 结婚人数创历史新低

近日，民政部公布了2024年结婚数据，去年登记结婚人数只有610万对，创下1979年以来历史新低，同比2023年结婚人数下降了20%。说实话，结婚人数下降这是大家都能预料的，但是2024年相比2023年下降这么快还是出乎预料。具体原因网上分析有很多，比如城市农村男女性别结构性失衡，城市化水平提高，妇女受教育程度等等。

但是我觉得可能大家忽略了 一个比较重要的催化剂作用 — — 这就是2024年经济相比2023年差很多，年轻人生活压力遽然加大，包括就业难，薪资低，对未来比较悲观，这都会极大影响年轻人的结婚意愿。

中国总体还是一个比较传统国家，非婚生子是不被主流社会接受的，所以结婚率指标一定是生育率的前瞻指标，所以2025年生育率一定很难看。

从这个角度来看，为什么我一直强调效市场救经济是迫在眉睫，原因就是时间拖得越长，不但救市场救经济的代价会越大，同时还会深刻影响民众的行为，比如结婚率、比如生育率。结婚率与生育率断崖式下跌对未来影响是非常大的。

我个人比较看好今年能实现经济复苏，所以今年结婚率与2024年相比不会太难看，甚至2026年还有可能反弹一下，但是从长远来看，低结婚率与低生育率可能是一个长期趋势，当然，如果地方政府给力，在最近两三年在引进东南亚年轻妇女领域力度更大一些，那么结婚率与生育率才有可能出现较大的反弹。

但是即便如此，国内整体的结婚率与生育率出现较大的反弹，那也主要集中在农村与小城市，大城市整体趋势就是单身群体将不断增长。

目前全国单身人群大致是2.4亿，未来10-20年时间，国内单身人群可能增长到3.5—4亿。这可能是未来国内经济中最大的结构性机遇。

在中国传统文化中，夫妻是个体最为主要的陪伴。所以我们现在几乎所有的商业模式以及商品与服务的供给都足按照传统夫妻模式设计的。

一旦单身人群不断扩张，甚至未来10-20年左右成为城市的主流，那么未来大城市商品与服务的模式将出现巨大的变化。

举个例子
现在我们的餐馆的桌子偏大，原因就是餐馆默认来吃饭的客人或者是以家庭为单位，或者是朋友杜交，基本就没考虑单身个体的就餐需求 — — 只有少数快餐才有这方面的考虑。

那么未来随着单身人群的不断扩大甚至成为主流，那么会不会有相当一部分的餐馆的桌子变小，然后菜品分量变小，价格下降呢？也就是说餐馆定位就是为单身人群提供更有针对性，体验更好的就餐服务。我觉得这个可能性是很大的。

同样的，单身最大的需求是陪伴，如果没有夫妻陪伴，那么单身人群一定会找替代品来满足自己的精神需求，这个陪伴的对象可以是宠物，可以是玩偶，也可以是虚拟世界的电子产品，随着科技的发展甚至可以是AI人形机器人。

对于单身人群陪伴需求的替代经济，我个人觉得可能是未来最大的增长点。

我觉得未来中国最好的状态是这样，农村大力引进东南亚妇女解决光棍问题，把出生率提一提，城市通过供给侧改革大力发展陪伴经济，让单身群体消费能最大化的释放出来。
