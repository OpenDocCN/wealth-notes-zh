
# 《特朗普的军事换房本》

源自2月11日顾子明说《特朗普换房本的三步走》
延伸，发布于2025年2月17日

历史上任何一次换房本过程中，最重要的一步，就是军事换房本，用某一种方式或者手段，把军队的高级将领换成自己人，这一套换房本的工作才算是完成。而前三代新中国核心，分别给全球打了一个换房本的模板，186的军事换房本模式是军事对抗，286的军事换房本模式是百万大裁军，386的军事换房本模式是现代化换装。三个方式，都是找了一个理由，将只听上级的军队关系网和利益网打散，用一个契机，建立起新的效忠机制。186的方式，副作用大，国家风险高，但效果彻底；286的方式，副作用小，个人风险高，但经济性高；386的方式，副作用小，各项风险都低，但耗时长、成本高，财政压力大。

我们的老板，则是三种方式的集大成者，18的时候，学386搞现代化换装，19的时候，学186搞军事对抗握住军权，20的时候，学286对旧军事系统搞新裁军。而现在留给特朗普的时间很少，他只能同时将三个方式一起进行，学186搞对格陵兰和巴拿马的威胁吞并，介入俄乌谈判，学286查国防部的烂账，搞百万大裁军，学386推动美军搞无人机和火箭技术，也把这一轮的美国换房本风险推到了极致，当然，正如高启强所说，风浪越大，鱼越贵。
