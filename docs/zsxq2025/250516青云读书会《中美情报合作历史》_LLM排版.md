
# 中美情报合作历史

中美情报合作向来都是国际关系中的一个很重要的衡量标准，两个国家的关系好不好？一个可以从军火贸易上可以看出，另一个方面就是可以从情报合作上看出来。英国、美国这样的国家，虽然有各种各样的问题，但是他们始终在情报上有着比较密切的合作，军售当然也是一样，美国和英国几乎可以说在情报共享的程度是非常高的，甚至有一些战略情报都是可以共享，包括当年美国向英国提供原子弹的技术和全套的资料。

从这个意义上来说，也可以看出英美两国确实是一种比较特殊的国与国关系，上个世纪80年代，我们也曾经跟美国有过这样一段历史，那么今天我就跟大家来讲一讲。

如果把时间维度拉长，应该说中美 的情报合作其实是分两个阶段的，第一个阶段就是在1949年之前，当时的美国政府和中华民国政府就已经有了情报合作，其中最有名的就是后来在重庆建立的中美特种技术合作所，当然也是在小说红岩里面出现过的。最早建立中美特种技术合作所是由戴笠和当时的海军美国的情报军官梅勒斯两个人撺掇的，出资方是当时的军统局和美国海军情报局，当时主要是针对日本的情报的搜集，它主要做信号收集，无线电、密码等等这些，包括有线无线的通讯，短波长波电台的监听搜索以及捕获。

后来随着国共内战的结束，国民党政府败退，1949年以后，中美之间、北京和华盛顿之间就没有了，像当时民国政府那样密切的情报合作，一直到1979中华人民共和国和美国正式建立外交关系以后，中美之间再次打开了情报合作的大门。

在1979年之前，1972年的2月份，也就是尼克松访问北京的第二天，当时陪同他来访问的美国国家安全助理基辛格与当时中共中央军委负责工作的叶剑英，在北京军区的西山指挥所进行了一次秘密会谈。当时基辛格就向叶剑英出示了一份资料，将当时美国所掌握的苏联在中蒙边境、中苏边境的部署情况，以及中苏核弹头的数量做了一份清单，清单中数据的精确和详细，当时让叶剑英颇为意外。

后来1975年基辛格再一次访华的时候，和当时的外交部部长乔冠华又举行了一次会晤，乔冠华后来向当时负责中共中央军委工作的邓小平进行了工作汇报。基辛格通过乔冠华希望能够向中方传达了一个消息，中美情报机构能否进行某种程度上的合作，譬如说展开对苏的电子信号侦察，诸如此类活动。

1975年中国的“文革”已经到了第九个年头了，国家经济，整个国家的国力已经损耗得很厉害。当时邓小平虽然在负责中央工作，但是当时的邓小平还是面临着比较大的政治压力。所以到底后来乔冠华向邓小平传达了这个信息没有，现在没有办法考证，事后中国方面向美方提出的拒绝的理由是中美尚未正式建交，不便于在如此敏感的领域展开合作。

但是美方当时却不断地向中方，包括情报机构，向中国的军方、中国政府提出这样类似的请求。当到了1978年5月，卡特上台以后，当时美国国家安全顾问布尔津斯基访问北京，当时美国国防部长负责国际事务的副部长埃布拉姆斯也跟布尔津斯基一起到了北京。

这也是30多年以来，美国国防部高级官员首次到中国大陆来访问。同年，伊朗爆发了伊斯兰革命，而在伊斯兰革命之前的伊朗巴列维王朝是美国在中东地区的紧密的盟友，由于伊朗的伊斯兰革命以及全面反美的意识形态，到了1979年1月份，伊朗单方面的宣布将收回两个监听站，并将当时驻扎在伊朗的美国中央情报局人员全部驱逐出境。

这个事情当时就已经震动了整个美国情报界。因为在1978年之前，巴列维王朝作为美国在中东地区最紧密的盟友之一，双方在伊朗的北部，当时苏联和伊朗接壤的地区，建立了两个地面监听站，其中一号站是监听无线电信号的，位于伊朗德黑兰的贝赫沙赫。一号站主要是接收当时苏联拜克努尔航天基地周围的通讯雷达和遥测信号，二号站在伊朗东北部的卡布坎，主要是截获当时的苏联，也就是现在的哈萨克斯坦境内的萨雷沙甘反弹道导弹实验基地的各种信号。

霍梅尼上台以后，美伊关系急转直下，所以美国无法继续在伊朗进行对苏的情报监听工作，对美国搜集苏联情报造成了巨大的障碍。

1979年1月1号中美正式建交以后，1月28号邓小平就飞往了美国访问，成为30年来首个访问美国的中国大陆领导人。邓小平1979年的访美其实有几个目的，第一个目的就是定向对美友好的基本外交基调，中国的外交政策要从一边倒向苏联，转而调整到积极开展与美国的关系。

第二个就是见识一下美国当时的综合国力以及美国人民的生活水平，走入美国，更好地了解美国，实地地去考察美国。第三个就是告诉美国，中国可能要对越采取军事行动，希望美国能够在国际政治和情报上给予一定的支持。

这个是后来公开出来的，邓小平访美的行程，但是没有公开出来的部分有13个议定书，议定书里面有两个议定书是关于武器技术和情报合作的，但是由于中美双方长期的默契，始终没有对外披露这13个议定书的具体细节。

在这一次邓小平访美的过程中，邓小平还利用了一个深夜的机会，到访了位于弗吉尼亚州兰利的中央情报局总部，并且在中央情报局总部的7楼会议室，听取了当时美国中央情报局局长特纳的会谈。会谈结束以后，中美双方就敲定在新疆进行对苏情报监听的合作。

这次会谈结束以后，美国方面还特别带领邓小平参观了现代化的中央情报局的技术设施和控制中心，而特纳将卫星地图放大到大屏幕上，并且指出了美国认为比较合适的建立中美联合监听站的位置。

这也是开天辟地第一次，世界上第一个访问美国中央情报局总部的共产党领导人邓小平，邓小平也是为数不多能够亲自到中央情报局参观访问的外国领导人之一。

邓小平访美结束以后，美国参议院代表团就访问了北京，其中有一个成员就是后来的美国总统拜登，当时拜登是参议院情报委员会的成员，拜登和参议院代表团访问北京的目的之一，就是确定双方在情报领域的合作。

大家都知道中央情报局作为美国最有名的对外情报机构，除了人力情报之外，它对于技术情报这一块是比较重视的，尤其是信号情报、电子情报这一部分。

信号情报按照美军的定义，分通讯情报、电子情报和仪器情报，搜索通讯雷达信号，监测各种无线电信号等。比如海面上有一支美国海军舰队，各舰艇之间通过无线电的通讯被搜集到以后，就可以根据无线电交互的流量，就可以发现和或者说判断其中哪一条舰是指挥舰，或者哪一条舰是补给舰。

1979年，中国开始对越进行自卫反击战以后，中央情报局就不断向中国提供苏联在中蒙边境和中苏边境调动部署的卫星照片和信号，确保中国不会腹背受敌。
