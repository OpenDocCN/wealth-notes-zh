
# A视野月度研报——只说精华

## 近期市场分析

在政策方面的消息极为有限，中美博弈转为暗流涌动，直接碰撞减少。自5月起，内需市场加速走弱，有形之手却极少做直接沟通，让市场感到迷茫，对后续经济和市场产生极强的不确定性。如何理解后续国内经济走势中的机会、风险与套利空间？如何把握国际热点与国内市场的联动逻辑？如何在困难的市场环境中减少损失、放大收益？这些都是当前必须正视的重大问题。

近期，A森亲自调研，获得了大量一手信息，整理成这份【A视野独家月度研报】，希望能解答上述疑惑，并帮助把握接下去的重大机会。

### 上半年市场回顾

2025年上半年的市场可谓一波三折、先扬后抑、暗流涌动，并非不重要，反而是重大的历史时刻。因此，有必要对此进行总结性复盘。

#### 国内经济

1. **经济体感变化**：1-2月逐步上行，3月达到上半年顶点；4月川普发动关税战后略有下行，5月加速进入观望状态，6月则是一个不容忽视的低估。主要原因包括：
   - 1-2月川普上台前后，大量重要政策未落地，中国出现“Deepseek时刻”，打破美国Al炒作浪潮下的“美国例外论”。
   - 1-3月市场进入抢出口热潮，预期川普将发动大规模关税战，出口对经济的托举力度不小。
   - 4月川普发布史诗级全面关税战，但传导给国际贸易仍需时间，对市场信心（金融端）有冲击。
   - 5-6月川普突发同意中美关税战休战90天，前期中断的国际贸易产生直接影响（时滞效应），市场注意到有形之手不再公布新的救市举措。

2. **市场预期变化**：关税战休战后，市场更加迷茫。市场理解有形之手的逻辑后，认定只要出口还能延续（90天休战），不会公布新的救市举措，所谓的关税战利多反而变成市场维度的“利空”。

3. **国际形势**：
   - 川普几乎没有一个政策是施政达成预期的，包括关税战。
   - 海湖庄园协议的核心内容美债化债置换无法推进，倒逼他跟美联储交恶，美联储更不愿意配合降息减轻债务负担。
   - 川普成功利用关税战收获年化达到6000亿美元以上的财政增收，但以侵害美国、乃至全球经济增长为代价，具有极强的紧缩属性。
   - 美国债务增长速度极快，严重影响《大美丽方案》未来的经济成效展望。由于无法成功化债，明年美国到期需要化债的规模极为庞大，构成2026年美联储不得不执行大规模降息/QE的根本动力。

### 调研发现

#### 宏观政策

尽管7月又有北京重磅会议，但市场整体预期非常低，认定依旧是“托而不举”。对比“美国例外论”，国内对自身的观点是“新版迷失30年”、“长期通缩论”、“经济结构转型论”，确信不会再有任何大规模刺激。短线而言，支持这种观点，7月份的会议确实可能采用“托而不举”。但长期来看，不支持这种观点，不能因为偏通缩的相似性就做直接的划等号。日本迷失30年是以放弃追赶换来美国的兼容；而我们是不放弃追赶，更不可能跟美国的霸权兼容。任何忽略这一点的，都意味着后续宏观政策突变所可能带来的风险。

#### 突破点

在大环境的压制下，有形之手能够马上发力的，应该就是A股。A股修复所需的资金体量不多、敏感度高、惠及人口数量庞大、且有直观拉升情绪的价值。从历史来看，在城市化、出口、基建等几个主要的稳增长工具都发力空间有限的情况下，A股就成了唯一留下来的可用工具。

#### 可关注的主线

- 科技+ 国产替代+ 稀土：关税战已经从关注税率逐步切换到科技战，双方都不肯妥协，都有有形之手的扶持，护城河明显。
- 军工：中美彼此都将军工作为压舱石。美国已要求盟友大幅提高国防开支，全球已进入类似于50年代、70年代的“全球军工超级周期”。军工板块远不止只是传统逻辑，而是对标下一代技术革命开启的重要意义。
- 稳增长：以旧换新、特别国债、地方化债等大量举措仍在不断发布。top多次对内对外宣誓要转型为“消费大国”，消费购买力是撬动海外庞大资源和战略利益的关键工具之一。
- 大金融：如果要确保散户大量入场，大金融是最重要的信号灯。如果这一块迟迟没有动静，则散户很难被说服入场。
- 香港市场：香港金融市场正在发生翻天覆地的改变，从稳定币到港股扩容，这个迹象非常明显。不少从美国本土回流的资金更愿意呆在港股，享受外汇自由进出便利。近期，香港金融市场正在替代日本，成为美元carrytrade的套利对象。

## 我们的担心

1. **地产**：核心城市新房市场与二手房市场近乎冰火两重天。按照瑞银的推测，过去5年内买房的业主，有接近一半出现了当前房价击穿买入房价。在“负财富效应”的前提下，居民家庭的消费欲望无法有效刺激。这一情况已超过91年日本泡沫破裂的程度，回撤速度之快、力度之猛，也算是创纪录。
   
2. **产能**：目前各地仍以奖励生产为主，对于需求的托举非常有有限，使得产能对比需求出现严重不匹配的过剩，汽车就是很好的例子。尽管有形之手一直努力打补丁来处理此事，但效果仍需观察。

3. **川普的不确定性**：川普的不确定性，是其对外谈判的内生组成部分。无论当前川普如何淡化对华的博弈，其做的每一步，不是针对民主党，就是为将来针对我们做前置条件的创造。预计，今年年内中美达成实质性的条款，可能性极低。这个预期中的结果，也势必带来两边激烈的博弈，即，川普放大我们经济的不确定性。

## 总结

在中美博弈前景极为不明朗的当前，7月北京重磅会议的预期需要放低。作为经济为数不多的亮点，进一步推动A股的赚钱效应，以慢牛、长牛、局部赚钱效应等风格为主。后续需要关注的是：
- 美联储何时重启降息？
- 7月北京重磅会议，有无新的提法？
- 8月12日，90天休战后的新态势？
- 《大美丽法案》一旦落地，对于10年期美债的影响？
- 9月初的纪念活动？
