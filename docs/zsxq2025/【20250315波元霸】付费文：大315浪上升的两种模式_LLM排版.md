
# 15浪上升的两种模式

2025-03-15 14:52 91508次学习 商品介绍

在上篇付费文章中，我预判了两种走势：

*   **一下一上**，跟踪关键点在于时间决定浪型
*   **浪型升级**，不是A4浪上升了

这种浪型比较难划分，主要就是去年国庆节的那10天涨了近千点到底算是什么浪，很难受。

## 两种走势的判断

那么，怎么判断到底走哪种走势呢？主要看时间，如果3140点以来的上升时间超过29天，即3月初任何一天再创2月份新高，则第一种走势就大概率失效了，那么预行调整。然3月7日刚好略微突破创新高，就确定第2种浪型升级走势无疑。

## 大315浪上升的两种模式

大315浪上升，估计有两种模式：

1.  **小趋势延伸浪后的正常模式**
2.  **大趋势延伸浪的特殊模式**

这与上篇文章分析一下一上不同，这次都是上，只是表现形式不同。

### ① 小趋势延伸浪后的正常模式

则其15浪上升判断见顶于3月25日（包括前后一天），主要看时间，不看空间。这种模式，则一般情况下15浪上升速度是缓的，在日线和60、30分钟顶部钝化全部消失后，重新构筑30分钟顶背离共振就足够了。

这种模式的理论依据是：重要下降趋势线（3674与3494连线）突破后，一般要经历2-3次回抽确认的过程，这个符合率几乎百分之百。除非这条下降趋势线在国家队认为不是重要的。事实上，这条重要下降趋势线突破后，仅只有一次回抽确认，我认为是不够的。

正常情况下，如果3月25日之前依然上升速度慢，纠结上行，基本上就先当成这种模式判断了。那么后面大概率会有大3152浪调整出现，估计ABC结构。调整空间一般要跌破3297.53点的，除非大3152浪调整时间足够充分（如何充分4月份再讲）。

再之后开启迷你型上攻的轰轰烈烈的大3153浪主升浪，3674.4点肯定要突破的嘛。

### ② 大趋势延伸浪

即大3153浪上升已经开启。

这种模式，152浪调整时间那么短，我个人至少目前认为可能性不大，但得有此防备再说。

这种模式主要看3点：

1.  **上升速度和空间**。如果3月25日前突破3509.82点，下周上升速度依然象周五一样比较快，可以考虑。重点观察大金融板块（证券保险银行）和四大金刚（有色金属煤炭钢铁有色）的持续性，尤其是证券板块，我认为先搭“炮架”的概率大些，不太可能炮架没搭好就急吼吼往上冲，中信证券去年国庆前就是例子，看图示。
2.  **3月27日及之后几天再创新高，时间密码失效**。这个46天的时间密码，我在被腾讯和谐的那篇公众号文章中已经说得很详细了。
3.  **看日线顶部钝化和60、30分钟顶部钝化消失后，等30分钟顶背离重新构筑的情况**。

总之，大315浪上升是基本上明确的，往上突破3674点创新高是确定的，只是以小趋势延伸浪后的正常模式，还是大趋势延伸浪直接上攻，哪种模式值得跟踪与商榷。

至于大315浪上升到什么点位什么时间，估计得下篇付费文章再阐述。我都已经谋划并测算好了，两种模式和点位。但现在公开实在没有必要，我得防着有人搞破坏。

## 后市展望

能做的就是3月25日之前，估计需要下周初调整到3380~3390点区间整固后再上升。3月底和4月份，估计行情依然不乐观。所以仓位控制方面，建议大家控制在半仓左右，只要行情确定，以后有的是你们满仓的机会，不要着急，行情来了永远不会踏空的。

余下的七八天行情，估计权重股蓝筹股与小盘股是夹杂着前行，说不上以谁为主持续攻击。但总体上，我认为权重蓝筹股在搭炮架，修复均线，为后面大3152浪调整后的大3153浪迷你型的轰轰烈烈作准备。
