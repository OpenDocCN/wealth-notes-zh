
# “富足时代”终将不可阻挡地到来

我们今天的预测来自万维钢老师。先说结论，万维钢老师说，我们这一代人将在有生之年，见证人类从短缺时代迈向富足时代。

什么是富足时代？之前，奇点大学的执行主席彼得·戴曼迪斯和畅销书作家史蒂芬·科特勒合著了一本书，书名叫《富足》。他们参照马斯洛的需求层次理论，提出了富足金字塔模型。

富足金字塔的最底层，是水、食物、住所以及其他维持人类基本生存的东西。第二层是能够进一步提高人类生活质量的各种“催化剂”，包括充足的能源、教育机会、信息通信技术等等。而第三层是健康和自由，这些形而上的高层次追求。富足时代的标准就是，能够同时满足这三个层次的需求。

那么，这样的“富足时代”是不是离我们很遥远？并不是。其实，回顾我们过去的历史，已经有意象表明我们正在加速走向富足时代。注意，是加速。

比如，在19世纪，全世界90%左右的人都处于极端贫困之中，而到了1981年，这个数字已经下降到了42%。到今天，全世界处于极端贫困状态的人不到10%。再比如，目前世界粮食的总产量，足够喂饱地球上的每一个人。做个最极端的假设，即使从今天起科技进步完全停止，人类也有足够的资源和能力让所有人都能过上好日子。

换句话说，回顾历史，人类一直往富足的路上一路加速狂奔。这个进展是怎么发生的？我们看两个角度，资源和技术。

第一，资源。世界上的资源，本质上是无尽的。注意，我们说的不是资源本身无穷无尽，而是相对于人类的 使用需求，地球上的资源仍然绰绰有余。我们来说一个具体的例子。根据历史记载，人类第一次发现金属铝是在古罗马时期。当时，一个金匠向古罗马皇帝提比略进贡了一个餐盘。这个盘子的材料跟其他盘子明显不同，颜色像银，但质地又轻。当时这个金匠声称，这种金属的提炼技术只有他一个人知道。结果提比略担心这种金属会取代黄金和白银，不仅没给金匠任何赏赐，还下令将他斩首。没错，这个神奇的材料就是铝。

我们暂且不论这个故事本身的真实性，但铝确实一度是世界最珍贵的金属。在19世纪，只有皇室和贵宾才有资格使用铝制餐具，而其他的宾客只能使用黄金制作的餐具。没错，就在一两百年前，铝比黄金还值钱。但这是因为地球上的铝金属很匮乏吗？当然不是。铝元素的总重量，达到了整个地球重量的8.3%，仅次于氧元素和硅元素，是地球上蕴藏量最丰富的第三大元素。而过去之所以匮乏，是因为人类的技术无法充分利用地球上的铝。而今天，我们每个人都可以轻松获得大量廉价的金属，这就是技术进步带来的贡献。说白了，很多资源并不是匮乏，只是我们的技术还不足以让这些资源为我们所用。

那你可能会说，有没有资源彻底被消耗完的那一天呢？答案是，只要技术进步得足够快，就很难出现这种情况。根据万维钢老师的洞察，当我们说“使用”什么东西的时候，我们其实只是暂时借用而已。

比如，你喝一杯水、洗一次澡，水资源本身并不会减少，而是重新进入了自然循环，等待着下一次被使用。再比如，食物，实际上是把太阳能转化为化学能的一个载体。你吃掉一个馒头，你只是利用了其中的化学能，组成馒头的原子本身并不会消失。换句话说，我们用过的物质并不会消失，只是换了另一种形式继续存在。

而回到现实，值的也不是那堆原子，而是它们存在的形式。比如，信息。当你在购买一张游戏光盘时，光盘本身很便宜，你本质上是在为光盘中的内容付费。再比如，餐厅。餐厅的食材价格并不高，但它为你提供了用餐环境、服务、体验。说白了，值钱的是物质本身，而是它们存在的形式。而技术的价值就在于，它可以改变物质存在的形式，为我们所用。

那接下来，我们需要面对的问题是，技术真的会一直进步吗？没错，万维钢老师提到的第二个角度就是，科技进步是加速展开的。它不是线性增长，而是指数级增长。

比如，最简单的例子是计算机的算力。摩尔定律指出，每隔18个月，一个集成电路芯片上的晶体管数目就会增加一倍。这意味着什么呢？也就是，每过18个月，你可以用同样的价格买到运行速度是原来两倍的电脑。尽管在1975年，摩尔本人对摩尔定律做了修正，把时间改为每2年增加一倍，但这个增长速度仍然是指数级的。

再比如，人类登月这件事。早在20世纪50年代，美国空军所就做了相关研究，他们追踪了自莱特兄弟以来的多飞行器，发现飞行器的加速过程是飞快的。而且经过进一步推算，他们得出结论，人类很快就会去月球旅行。之前，凯文·凯利还在《科技想要什么》中就提到，1953年的时候，世界上还根本不存在任何一种能够实现这种未来旅行的技术。当时，即使是最大胆、最梦想家也没有预测到，人类能够在2000年到来之前就登上月球。但技术发展的结果是，1957年苏联就率先发射了人类第一颗人造卫星，1969年人类就首次登月成功。

那么，技术的发展为什么是指数级增长的呢？首先，某种意义上，技术进步是边学边算的结果。也就是，一个东西刚刚被发明出来的时候，生产者还不够熟练，所以成本高，卖得也贵。但随着用户越来越多，产量越来越大，生产者会越来越有经验，就会发现其中的各种窍门。因此，生产者就可以改进它的性能，降低它的成本。

其次，技术本身还有一个增长机制是，量变引发质变。比如，电子管改进到极致后出现了晶体管，然后晶体管又进化成了集成电路，接着发展出了光刻技术这样更精密的制造工艺。再比如，人类对能量的汲取能力也在指数级增长。过去几十年间，光伏发电的成本已经降低到了原来的几百分之一，目前已经低于传统化石能源的成本。而像我们国家，有全世界80%的光伏产能，甚至只凭借光伏，我们就能获得几乎无限的能量。

最后，技术还存在叠加效应，也就是两个技术组合在一起，会出现一加一大于是的效果。

假如资源本身是无限的，就意味着富足时代总有一天会到来。再加上，过去我们的科技水平是指数级进步的，这就更加明确了富足时代的加速到来。比如，人类将有可能在几年内实现AGI，也就是通用人工智能。有了AGI的帮助，我们就有可能在有生之年，见证富足时代。因为AGI会帮助人类从繁杂、无聊的工作中解放出来，甚至会帮助人类解决一系列的科技难题。

那么，假如真到了富足时代，人类的价值到底是什么呢？答案是消费。按照万维钢老师的说法，富足时代，其实就是消费的时代。没错，未来大部分工作都可能会交给机器人去做，而人对经济活动最主要的贡献就是消费。为什么消费也算是贡献呢？因为，消费其实就是你为产品投票，你在告诉生产者应该往哪个方向走。

去年青山资本还发布过一份年度主题报告，题目就叫《消费，新时代的劳动》。其中提到，今天的消费行为，涉及的选择、决策、思考越来越多，越来越消耗精力，因此，从表象特征上看，消费属于劳动。同时，今天的消费行为，为整个社会贡献的价值也越来越多。它能促进经济循环，实现资源的再分配，并且为商品生产指明方向。因此从本质上看，消费也像劳动一样在创造价值。

换句话说，你可以把未来社会想象成一条大船，每个人都是舵手，而掌控航向的方式，就是去消费，去做出选择，社会这条大船将在无数人的选择中，驶向人们期待的方向。

最后，借用《富足》这本书里的一个观点，说的是，在地球这个小小的星球上，人类所面临的各种各样的巨大挑战并不是一个个独立的点。恰恰相反，它们像一排排多米诺骨牌一样聚集在一起，只要推倒其中一张，所有的骨牌都会倒下来。换个角度看，如果我们战胜了某个具体的挑战，那么就很有可能会连带着战胜其他更多的挑战。说白了，从更乐观的角度看，这个世界是一个巨大的正反馈环，每一次的成功，都会带来更多的正收益。
