
# 美俄元首通话，停战要来临了

## 文/卢克文工作室嘉宾星海舰长

2月12日，特朗普给普京打了一个“漫长而富有成效”的电话，随后宣布了一个震惊世界的新闻：美俄元首已经就俄乌停战的问题，达成了共识。紧接着，乌克兰方面证实，泽连斯基与特朗普通了电话，同意和俄罗斯“进行进一步的接触和会谈”。很多人都惊讶了，普京和泽连斯基死磕了三年，一直是拒绝与对方接触的，现在竟然被特朗普两个电话就搞定了？

## 电话里到底说了啥？

这个电话非同一般。要知道，美俄元首的热线电话，已经4年没有响起过了，上一次美俄元首通电话还是2021年的12月，拜登和普京谈了4分钟就匆匆挂了。可是现在呢？特朗普和普京一聊，就是90分钟。

90分钟，可是能谈很多事情的。俩人说了什么呢？可归纳为以下三部分：首先就是俄乌停战，双方经过讨论，同意“立即启动谈判”以结束俄乌冲突。有意思的是，特朗普和普京的态度是“商讨”，但在随后几个小时后和泽连斯基的通话，却变成了“通报进展”，而不是商量，态度可以说是一个天上一个地下。

其次就是美俄关系的“全面重启”，特朗普提议在沙特会晤，而普京则邀特朗普访莫斯科，这差不多是5年来，最显著的美俄关系破冰标志。

值得注意的是，特朗普特别提到“两国的伟大历史，以及我们在第二次世界大战中如此成功并肩作战的事实”。说实话，在美国历史观的叙事中，一直淡化苏联在二战中的作用，导致大部分欧洲人现在都以为什么是美军率先占领柏林。现在却突然重谈当年“并肩作战”，不排除是在暗示战略关系重塑的可能性。

还有中东局势、能源合作、人工智能及美元影响力等议题，甚至包括换囚，俄罗斯释放了涉嫌走私毒品的美国驻莫斯科大使馆的工作人员福格尔，而美国则释放了比特币交易所BTC-e 的联合创始人、俄罗斯人亚历山大 · 维尼克。

说实话，光靠一通电话就谈成了这么多事情；是不可能。这里的每一条交易，都涉及复杂的利益交换，不可能靠俩人打个电话就拍板定下来了，哪怕总统也不行。只有一个可能，双方的团队已经早就秘密达成了协议，双方领导人打电话，只是最终确定而已。

其实，俄美的这种秘密接触，早就有端倪了。2024年12月25日，一架俄罗斯特别飞行中队(保障俄罗斯总统专机的机构)的伊尔-96专机悄悄飞到了华盛顿，并在那里停留了两天。专机上是谁？去美国干什么？谈什么？没人知道。紧接着，普京在邀请各国领导人参加5月9日胜利日阅兵时，专门透露会有“神秘美国客人”，外界普遍猜测会是特朗普。

就一个邀请的事，本没必要这么神秘兮兮，显然邀请参加阅兵这件事，只是个添头，俄罗斯特使坐专机前往美国的重头戏，还是围绕俄乌停战和重启美俄关系进行提前试探。

但是没想到，美俄秘密谈判的事情，被英国《每日邮报》给泄露了！根据《每日邮报》泄露的特朗普的和平计划，在2月上半月，普京会和特朗普通电话；紧接着，普京和泽连斯基通话。然后在2月底或3月初，俄乌双方开始面对面会晤，特朗普有可能亲自参与，敲定谈判条款。4月20日(复活节)，签订停火协议。5月9日，公布协议条款。

有意思的是，5月9日，正好是俄罗斯胜利日阅兵的日子。要是真按这个计划来，那就是给普京送了一个大礼包，让普京不仅能在战场上拿到好处，还能在胜利日宣布胜利，对普京的个人形象非常有利。

此外，根据这一计划，乌克兰将被禁止加入北约，乌克兰将承认俄罗斯所吞并土地，并从库尔斯克撤军。两国战线成为非军事区，由欧洲(也可能是英国)的军队负责监管，美国不参与。同时，为了安抚乌克兰，美国将继续向乌克兰提供军事援助，同时乌克兰可以寻求欧盟援助，十年内向乌克兰的援助金额估计为4860亿美元，并同意乌克兰在2030年之前加入欧盟。

从现在美国各方透露出的消息来看，这份泄密的和平计划真实性非常高。

如果真按这个计划来，乌克兰妥妥成了最大输家，割地、不能加入北约、国土还要进驻外国军队，可以说输麻了。但是谁也没想到，第一个跳出来对这份泄露的和平计划表示不满的，竟然是俄罗斯。

明明这份计划对俄罗斯有利，俄罗斯为啥不承认呢？俄罗斯主要纠结的问题在于：欧洲军队，大部分是北约军队，北约军队进入乌克兰，和北约东扩到乌克兰有啥区别？那俄罗斯这场仗不就白打了？万一俄罗斯和这些欧洲军队起了冲突，是不是会引发俄罗斯和北约的全面冲突？所以俄罗斯常驻联合国代表瓦西里 · 涅边贾表示，任何未经联合国安理会授权而进入乌克兰领土的外国军事力量，都将被俄罗斯军队视为合法目标，即使他们被冠以“维和”的名义。

显然，这一核心诉求不满足，俄罗斯是不会松口的。

结果呢？美国还真让步了。美国新任国防部长皮特 · 海格塞斯站出来说，驻乌克兰的国际维和部队不应在北约主持下部署，因此不受北约集体防御条约第5条的约束。也就是说，进入乌克兰的北约军队不算是“北约军队”，所以就算被俄罗斯打了，也不会引发北约与俄罗斯的战争。

说白了，特朗普在给俄罗斯吃定心丸呢。最大的心病解决了，俄罗斯这才同意公开通电话，并在随后的新闻通稿中，确认了双方元首同意停战的消息。

现在无论是特朗普，还是普京，都得到了自己想要的东西，唯二破防的，就是乌克兰人和民主党了，现在正在网上疯狂骂特朗普，民主党的报纸甚至声称“这是美国历史上最可耻的时刻之一”。

## 从目前的情况来看

持续了三年的俄乌战争结果，大概率是领土属于俄罗斯，市场属于美利坚，通胀属于欧罗巴，废墟属于乌克兰。说实话，面对这样一个结果，中国是不太愿意看到的。一方面，俄罗斯处境变化会对中国有深刻影响。最近三年，无论俄罗斯和中国的关系如何好，我们的认知始终非常清楚，那就是俄罗斯和中国亲近并不是对中国有好感，而是迫于现实的压力。但凡有选择，俄罗斯都不会选择靠近中国。

我们可以看看，三年来，虽然和中国的经贸往来已经突破了2000亿美元，但中俄之间愣是连一项说得过去的经贸协定都没签，甚至在2024年中俄贸易的增速还接近停滞了，俄罗斯还出台禁令，禁止某些中国产品通过俄罗斯过境运输。俄罗斯和伊朗一样，对西方都有一种蜜汁向往，这是刻在骨子里的，只不过西方一次次把俄罗斯怼得满头包，只能回过头来找中国。这就跟一个恋爱脑一样，只有被高富帅伤害得遍体鳞伤，才最终会回过头找老实人。

但只要高富帅再勾勾手指，他们还是会扑向高富帅的怀抱。现在中国和俄罗斯，还有很多项目没有完全敲定，比如中俄管线问题，比如中吉乌铁路向西延伸问题，比如朝鲜问题，比如金砖国家合作和金砖国家货币问题等等，这些问题，都需要俄罗斯的配合。俄罗斯和美国关系回暖后，这些既定计划会不会生变？我们看特朗普上任第一件事，就是通过声索巴拿马运河，逼巴拿马退出了一带一路。如果这是既定战略的话，那么特朗普对俄罗斯做出这么大让步，会不会要求俄罗斯用疏远中国，作为让步的条件？细思极恐。

另一方面，是俄乌停战的红利，中国可能吃不到了。很多人想不到，调停战争，也是有好处的。不仅可以提高国家威信和影响力，更可以在调停之中获取很多利益。比如，停战之后，要重建吧？重建可能要花上千亿美元，是一个大蛋糕。谁来重建呢？现在世界第一基建大国是谁呢？中国！如果中国参与了调停，在分这块蛋糕的时候，毫无疑问就能占到先机。

所以，中国对调停俄乌战争，一直是非常积极的，任命了经验丰富的外交官李辉作为中国政府欧亚事务特别代表，前后开展了四次穿梭外交，还在俄乌冲突爆发一周年之际，发布了一项包含12点的和平计划，希望能促成俄乌停战。可惜当时双方立场差异太大，中国似乎也高估了对自己的影响力，最终没能实现中国主导下的停战。而现在呢？停战是美国主导，很多情况可就不好说了，不排除会彻底把中国排除在外，让中国只能眼睁睁看着俄美借着这次停战的契机，通过一系列勾兑，重返蜜月期，实现特朗普“联俄抗中”的战略目标。

面对如此严峻的形势，唯一可以宽慰的，就是虽然双方首脑都同意停战，但怎么停战，能不能真的停战，还是未知数，谈判还面临很多障碍。

比如，领土问题。现在是俄罗斯在乌克兰占了五州，而乌克兰则占了一部分库尔斯克。按照正常的思维，双方各退一步，俄罗斯退出扎波罗热和赫尔松，乌克兰退出库尔斯克，是最完美的方案。但是呢？作为对土地有特殊执念的斯拉夫人，怎么可以把嘴里的土地轻易吐出去？

考虑到现在五州的土地，俄罗斯还没有完全占据，下一步不排除俄军会趁着停战之前发动一次大规模攻势，无论是彻底占据五州，还是扩大在哈尔科夫一带的占领区，然后等停战时用哈尔科夫，来换五州未占领领土，都是稳赚不赔的买卖。

比如，和谁谈的问题。作为和俄罗斯打了三年仗还没怂的总统，泽连斯基是最有资格代表乌克兰谈判的。但问题在于，因为2024年乌克兰没举行大选，泽连斯基的总统位置其实是“不合法”的，所以俄罗斯多次强调，“只会与乌克兰合法领导人谈判”。但问题在于，现在距离5月不远，搞大选，也来不及啊。前不久，特朗普的特使基思 · 凯洛格倒是说要求乌克兰大选，但根据大选的时间表，是在年底，那停战的时间总不能也拖到年底吧？现在要么泽连斯基主动辞职，要么乌克兰发动一场政变把泽连斯基赶下台，这样才能勉强绕开这个难题。

再比如，俄罗斯的诉求不仅仅是停战，还要求取消制裁和解冻3000亿美元资金。但这个问题，特朗普的兴趣就不太大了。特朗普的和平计划，从底层逻辑上来说，是一种“止损战略”，在特朗普看来，这场战争是深层政府挑起的一场不必要的战争，花了那么多钱，死了那么多人，不仅分散了美国对中国的注意力，属于亏本买卖，哪怕强压乌克兰让步，也要及时止损。但对俄罗斯制裁和冻结俄罗斯资金，并不算美国的损失啊。相反，制裁俄罗斯，让美国获得了欧洲接近一半的天然气和石油订单，属于妥妥的红利啊。特朗普不会轻易地解除对俄制裁，而不解除制裁，俄罗斯会不会同意停战，还在两可之间。

虽然双方元首已经就停战达成共识，但要实现停战，还需要非常艰难的谈判，不是一个电话就搞定的。而且，就算真的停战了，中国也没必要特别揪心。在中美俄大三角之中，两家联合对抗一家完全正常，我们可以回顾一下，50年代是中苏联合抗美，80年代是中美联合抗苏。2014年到现在也基本上是中俄联合抗美，本质上是弱势两方联合对抗强的一方。

现在中国势力增长迅速，美国自觉不敌，选择联俄抗中，也只会是一个历史的必然。对于必然要发生的事情，也就没什么可担心的，正面面对即可。更何况，特朗普这样只占便宜不吃亏的主，就算联合俄罗斯，也不会给俄罗斯太多好处，现在美国已经占据了欧洲能源市场，特朗普又要大力振兴国内油气产业难道还真的能把欧洲市场吐出来让给俄罗斯不成？对俄罗斯这样一个伪装成国家的加油站来说，只要还想靠卖油过日子，就必须依靠中国，也不可能真的倒向美国。

所以，我们大可不必如此介怀，发展好自己，才是王道。
