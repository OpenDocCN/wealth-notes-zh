
# 学学亮哥

指数好看， 个股就难看了， 胡椒面式的维个稳， 上证还有点余力， 其他的不忍睹。

主要原因是， 鼓作气， 再而衰， 三而竭。

这 轮大涨主要是信心修复和预期的涨， 并没有基本面和业绩的支撑， 通俗的说人们不信。

而且小盘微盘涨太多了， 几倍的比比皆是。

咱们之前多次谈中证2000代表的小微盘， 日月周线全是历史高点。

我们举过例子， 和女朋友吵架， 最好是 次吵完， 情绪发泄干净。

放量下跌才洗盘充分， 然后企稳。

但这几天 直没有放量大跌， 阴跌就还要跌。

操作说我们讲过， 搞短线， 跌破5日平台就得走， 容忍度大 点， 跌破箱体也得走。

再回头看这 轮， 你拿的品种， 是不是从跌破5日平台就开始萎靡呢。

那我们需要悲观吗， 不需要， 个叙事和趋势开始后， 是不可逆的。

跌多了后面会给反抽， 反抽就是窗口， 然后等下 次机会的到来。

主力和机构不是不会亏， 而是他们亏了和跌了会检讨原因， 下次再来。

这个5日平台跌破了， 看好下 个5日平台再战， 这个箱体没了， 介入下 个箱体。

小时候， 你以为没带红领巾天就塌了， 长大后觉得没考上好大学天也塌了， 再然后觉得工作找不到天也塌了。
