
# 银发经济（二）：搞清底层逻辑和竞争方向

## 积极老龄化与 ERG 模型

这节课所有的讨论，先从一个概念开始，就是“积极老龄化”。这个概念是世界卫生组织在 2002 年提出的，它要回答的问题是，什么才是正理想的老年人生活？我总结一下，大概分为三点：安全保障、身心健康和社会参与。但这些都属于“大道理”，说大道理往往是简单的，难在“怎么做”？所以，我们还需要引入一个新的需求模型 ERG 模型。ERG 模型是美国心理学家、耶鲁大学教授阿尔德弗，对马斯洛的需求理论做的一次修正和升级，可以理解为马斯洛需求模型的 pro 版本。

先来说，修正了什么。我们先复习一下马斯洛需求理论，马斯洛认为，人类需求主要分为五类：生理需求、安全需求、社交需求、尊重需求和自我实现需求。阿尔德弗则把人类需求，重新划分成了三类，创立了 ERG 需求模型：其中 E，代表生存需求（Existence Needs），比如食物、安全、基本收入、健康保障等等；R，代表关系需求（Relatedness Needs），比如社会交往、赢得尊重等等；而 G，则代表个人潜能发展与自我价值实现的成长需求（Growth Needs），比如技能提升、职位晋升等等。

我认为这个分类，和积极老龄化的概念正好能呼应。这么划分最直接的好处，是更简洁、也更接近需求本质。举例来说，面向老年人的衣食住行、医疗保健，包括大多数适老化改造，既满足生理需求、也满足安全需求，是彼此交织在一起的。但其实这些需求所对应的，都是刚性需求，背后的逻辑是同质的，不应该彼此区分开。而在 ERG 模型中，生理、安全这些需求，都被统一为“生存需求”，简洁明了。

## ERG 模型下的银发族需求分析

讲完 ERG 需求模型对马斯洛需求模型的修正，再来说，它升级了什么？这个更关键，也是我认为 ERG 模型解释力更强的原因，主要分成三个部分：

第一，ERG 模型中的生存需求、关系需求和成长需求，同时出现、相互影响。也就是说，每个人的理想状态，几乎都是在生存保障的前提下，同时想要赢得尊重和实现自我价值，而不是像马斯洛所说的，一层一层向上递进，满足了生理需求后，才考虑要满足安全需求。

在之前的课程里，我看到有同学评论：中国进入老龄化社会，对各种向老年人卖虚假保健品的骗子来说是所谓的“好消息”。为什么很多老年人特别容易被所谓的“保健品骗局”吸引呢？我认为 ERG 理论刚好能解释这个现象，这些骗局往往会同时击中老年人的多层需求：先用包治百病、延年益寿等说辞来夸大疗效，满足老年人的生存需求；再通过线下会销和一对一电话以及社区网点嘘寒问暖问候的形式，满足老年人的关系需求；最后的有的还会附带分销属性，让老年人从客户变为销售，帮着一起推广产品，触发老年人发展个人事业的成长需求。

对很多老人来说，这并不是单纯买个产品那么简单，而是一种多层次需求被照顾的体验，所以他们才会更容易卷入其中。

## 跨界融合与终身消费市场

好，刚刚我花了很大的篇幅去聊 ERG 模型，有两个原因：一是帮你对银发经济的风口和趋势，形成更全局、更基础的判断。二是帮你为自己的赛道，找到参照、确定所处的层级，从而决定之后的行动策略。

这里我从 ERG 模型出发，继续推导出两条底层建议，供你参考：

第一条建议，是在商业想象力下，强化跨界融合的能力，提供满足多种需求的系统解决方案。什么是系统解决方案？现在一些高端养老机构，除了提供最基础的医疗保障、健康餐食、社交活动之外，还提供老年课堂，让养老院里做过教师的老师们、教授们，继续发挥余热，给养老机构其他的老人讲讲课，贡献着自己的智慧。除了满足银发族的生存需求、关系需求外，同时也满足了成长需求。相比传统养老院，这些养老机构的解决方案更系统。

再比如，老年旅游和医疗保健结合的康养旅游模式，有的旅行社还会增加非遗工艺的学习，提高文化属性。相比一般的旅游项目，这也算是一种更好的系统解决方案。

## 中国的红利

最后，我还想简单谈谈银发经济的风口期。之前有同学在评论区提问，说老年人的消费力不是应该降低么？这么看，银发经济不就是“夕阳经济”么？这个问题很有水平，背后涉及到一个消费理论，就是美国经济学家莫迪利安尼提出的生命周期消费理论。这个理论认为，人到老年，收入水平下降，只能消耗储蓄或者依靠养老金维持生活，所以消费能力自然也就下降了。

这个理论有合理性，但它背后有两个限定条件，我提醒你关注一下：一来，人老之后消费力下降，这是微观机制，而不是宏观趋势，我们不仅要看一个人的消费水平，还要看整个群体的消费规模。而现在的趋势是，老龄化逐渐深化，银发族的规模在大幅度增长。

二来，中国的市场实在太特殊了，我们面临着可能是史上最大规模、也是最迅速的一批享受了中国经济高速发展最大红利的富裕老人的到来，他们有很大的消费潜力待发掘；另一方面，中国的养老或者说银发经济的市场，之前发育很一般，我们不是像西方那样的缺乏增量的成熟市场，而是巨大的待开发、待升级的新兴市场，正在经历从 1 到 10 的过程，增量巨大。所以我才会反复说，这是中国大规模人口下的新周期红利，也是普通人能搭上的为数不多的班车。
