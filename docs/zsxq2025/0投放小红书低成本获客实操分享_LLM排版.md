## 小红书引流千万别踩的坑
100种死法，一定避开！
1. 私信、薯群、评论区直接发二维码、手机号、QQ号、邮箱、小程序、外部链接、微信号
等联系方式或引导用户发其联系方式；
2. 私信、薯群、评论区回复内容触发加微、微信等敏感词、违禁词，或者通过常用的谐音、
图片、表情包的形式，分享你其他平台的联系方式、链接等；
3. 集中一个时间段批量回复私信或评论进行引流，频繁发同一个引流素材，如相同的文字话
术、图片、视频、表情包等；
4. 持续发营销类笔记导流，收到官方提醒后没及时调整（一定要结合干货笔记和互动笔记）；
5.被举报后平台引入人工审查。因为平台的审查机制是先算法，算法查不出来，但是你被举
报了，再人工来看，人工看你只要涉及到引流，一定会被惩罚、限流的。

## 小红书账号被提醒导流违规要怎么办
被判站外导流注意以下三点
1. 倘若账号收到导流违规的提醒但没有被禁言，那就要小心了，近期不能再有导流行为，哪
怕是引导客户发也不行；
2. 若账号已经被判定为导流行为而被禁言限流了，就没有办法解除了，只能等后期投流做规
避，若是再次导流违规，会按照7天--30天-永久禁言的阶段增加禁言限流的天数；
3. 倘若被关“小黑屋”了就基本上没有办法捞出来解决了，所以，要提前做好规避。如果是
已经被禁言的，可在释放之前做好投流工作，避免解除账号后再次禁言。
