
# 2025年实际增速大概率在5%偏左

连续三年增速目标设置为“5%左右”,2023、2024年都完成了，但今年要做好增速在5%以下的准备，既然是左右，那么偏左也属于完成目标。

回顾新世纪以来的历年GDP增速情况，根据目标增速与实际增速的偏差情况，大致可分为四个阶段。见文后第一张图。

**第一阶段：2011年以前，实际增速明显高于目标增速。**

**第二阶段：自2012年到2019年，实际增长在上方紧贴目标。**

但这一阶段已经是非常用力才稳住的，2008年底到2010年底推出了“四万亿”刺激计划，带动信贷突破10万亿，但刺激计划结束，由于全球金融危机影响仍然持续，当时面临很大的下行压力。上任李总非常重视数据，他曾说“我每天早上上班第一件事，就是先看两张汇总了上百个经济数据的表格，从粮价涨了多少到电用在哪里，还用红黑两种颜色标明了数据升降。”因此，在上任李总任期内，非常重视用宏观调控稳住经济增长，对市场需求有求必应，这才稳住了经济增速，表现出来就是实际增长紧紧贴着目标增速。

**第三阶段：2020年到2022年，经济增长上下波动极大。**

这一段属于极特殊情况，比如2020年直接取消了增速目标，而2021年因为2020年基数低的原因，所以实际增速非常猛。

**第四阶段：2023年至今，实际增长紧贴目标但开始重合。**

从这一阶段开始，GDP增速目标就设置为“5%左右”，表面上似乎回到了第二阶段，实际增速紧贴着目标增速，而且2023、2024年都完成了。但要注意，相比第二阶段过于看重短期经济数据，第四阶段更强调高质量发展、先进制造等长期方向，可以接受短期数据的波动，但为了保证增长也下了很大力气，能够维持住非常不容易。

如果只看国家整体情况，还不足以掌握实际情况。小镇于是整理了全国各省区市2023年、2024年的目标增速、实际增速情况，并根据目标与实际的偏差进行了分类。绿色代表实际增速超过目标，橙色代表实际增速低于目标，颜色越深代表偏差越大。如文末第二张图所示。

可以看到，2023年、2024年能够完成目标的省份并不多。2023年有15个，2024年只剩下10个，能完成目标的省份更少了，而且超出的幅度也在下降。如果把超出标准定在0.5%以上，那么2023年有6个，到2024年只剩下2个。

2023年、2024年都完成了目标的省份，只有6个，既江苏、山东、浙江、北京、天津、内蒙古，而中国经济第一的广东，已经连续两年没有完成增长目标，偏差从2023年的-0.2%扩大到2024年-1.5%。

而在没有完成目标的省份中，偏差值远大于完成目标的省份。偏差值高于1%的省份，2023年有10个，2024年有9个。

这或许跟2023年GDP复核上调有关。但现在要考虑2025年的增速，那么统计方法的修改带来的影响可以忽略。

虽然2024年以来中国科技和产业突破喜人，但相比房地产的巨大体量，局部的突破要想对GDP增速产生明显影响，还需要时间；而虽然2025年对经济大省进一步授权，比如将专项债的许可权下发，但能够带来的动力也是有限的。

因此，考虑到2023年、2024年增速目标完成情况，很难期待2025年各省能够普遍完成设定目标。从目标设定也可以看出来，三年来，各省普遍下调增速目标，越来越靠近国家整体目标。2023年，还有20个省区市将增速目标设置在6%左右或以上，2024年下降到12个，而2025年只剩下6个。

当然也有一些省份设置了更具雄心的增长目标，表现为2025年设置的增速目标大于2024年实际增速。比如山西、海南偏差值分别达到2.7%、2.3%，也有部分省份设定的目标高于实际增速，但这并不能代表对应省份有足够的信心完成，而是国家目标卡在这里，能不能完成是能力问题，而目标设置是一个态度问题。

还有一个变化，在全国CPI目标公布前，各省已经将居民消费价格涨幅从去年的3%降到2%，只有西藏仍然维持3%的目标。而最终国家设定的目标也确实降到了“2%左右”，这成为1998年亚洲金融危机以来最低的通胀目标。

一般我们说是否实现增速目标，说的都是名义GDP，通胀目标设置降低，再考虑到2024年CPI只涨了0.2%，以目前情况看，2025年也很难实现2%的温和通胀目标，这必然会影响到名义GDP的增长。

当然上述分析并没有考虑各省经济体量不同，需要注意江苏、山东、浙江等经济大省仍然有很强的增长活力，在更优惠政策支持下，2025年值得期待，但能不能抵消其他省的差额，很难说。

所以，需要做好2025年实际增速低于5%的准备，很可能是4.8%。

国家承认增速下行，也认可未来增速会继续下降，并不会为了实现目标而强行采取大放水等应急政策，只要经济整体面向好，可以承受短期压力。但国家希望增速下行的尽量慢一点，也就体现为2023年到2025年增速目标一直不变，但实际增速从超出目标0.2%到持平，而结合上述分析，2025年实际增速低于目标0.2%。

那么2026年到2030年就尽可能保持实际增速4.8%左右，偶尔上到5%，等到了2030年前后，再把增速目标下调到4.5%左右，如此尽可能保持平稳。这也是适应未来长期低利率时代的必要准备。
