
# 今天的日本消费业，能给我们哪些启示？

今天我们要关注的，是日本的消费市场。这几年，对很多国内消费品品牌来说，日本公司是一个很难忽视的参照系。为什么？一来，日本的零售业穿越了所谓“消失的30年”，凡能活到今天的都有两手。二来，日本社会的老龄化来得更早，很多公司在应对老龄化方面已经摸索出了成型的答案。

有一些日本品牌，圈外人可能了解不多，但在行业内，已经是很多人研究的焦点。比如号称地表最强杂货店的堂吉诃德、被称为意大利沙县的山里红，还有被称为生鲜效率之王的lopia，等等。

但是，市场上关于这些品牌的资料，不少还停留在几年前。那么，这些品牌今天发展得怎么样？深入细节后还有哪些新发现？

这就是咱们今天要回答的问题。2024年12月，黄碧云老师带队得到新商学，在日本做了深度游学调研，带回了好多一手资料。

好，咱们正式开始。

要了解日本的消费行业，需要先了解一个大背景。这就是日本的就业率。目前，日本的大学生就业率已经高达98.1%，创下了1997年以来的最高记录。年轻人的高就业率，是消费行业繁荣的基础。

那么，日本的就业率是怎么提上来的呢？有这么两个关键原因。

首先，短期看，日本的高就业率得益于旅游业的复苏。过去两年，日本每年大概接待3000万游客。什么概念？要知道，整个日本的人口总数是1.25亿左右。即使按照最暴力的算法，这笔旅游收入平摊，每四个日本居民也能分到一名游客的旅游消费。游客多，对服务业的需求就多，而日本服务业的训练体系又很成熟。这就导致大量的年轻人可以快速上岗。而在工资方面，日本高中毕业的蓝领工人，每月收入大概是20万日元左右，折合成人民币大概一万块。年轻人有钱赚，自然就会去消费。

其次，长期看，消费和劳动的基础都是人口，因此日本也一直在想办法解决少子化的问题。比如，目前东京的一些公司规定，有孩子的员工，尤其是孩子在三年级以下的，可以上四休三。而且尽量不安排这些员工加班，晚上的应酬也一概取消。要知道，日本职场的人际规则是出了名的复杂。你看日剧里面，员工下班后和前辈一起喝酒不是常事吗？但现在这些都在发生变化。当然，这些方案都有滞后性，至于会不会奏效，还需要长期观察。

好，基于日本目前的就业状况，我们再来看日本的消费行业。其中有哪些地方可以给我们一些启发呢？

接下来，我们从黄碧云老师的调研中，挑选三个案例展开说说。

第一，向堂吉诃德学放权。没错，很多公司在强调放权，让听见炮火的员工做决策。而堂吉诃德在这方面的做法或许值得借鉴。

堂吉诃德这两年在国内热度很高，这是一家零售店，但多数人都喜欢叫它杂货铺。因为它的商品确实太杂了。即使是一间小店，也能有十万个SKU。从怀旧玩具到二手奢侈品，都能买到。

但问题是，从零售的角度思考，他们如何处理这么多的库存？如何确保周转效率呢？

最关键的方法之一，就是放权。堂吉诃德的店面商品管理完全放权。一线员工有40%的商品自主权，可以自由安排门店的商品。因为一线员工更了解顾客，因此放权的好处是，在不同的区域的山店，商品都会有很多强的适销性。比如在东京池袋区的店里，因为华人游客比较多，因此卫龙辣条等中国食品的专柜就做得比较大。同时，堂吉诃德的山员手里，都握着几个类似用户私域的聊天群。门店里占比最大的商品类型之一是尾货，而尾货的销售主要就靠山员通过聊天群引导。

换句话说，堂吉诃德的销售，很大一部分是靠山员来驱动的。那么，怎么确保每个山员都能诚心诚意地为自己操心呢？都能把店里的事当成自己的事呢？

这就说到，堂吉诃德的员工制度。首先，一线员工的定位是自负盈亏的小老板。前面说了，堂吉诃德对一线山员全面放权。但光放权还不够，关键是责权利等边，你掌握权力的同时还要享受利益，承担责任。假设这名员工负责这个区域，那么他就要想办法去盈利。其间员工可以自主定价、进货、开发新产品。同时，对员工每半年评价一次，业绩超额，就涨薪，反过来就降薪。

其次，在堂吉诃德的考核期，有20%的人会面临内部失业。注意，只是内部失业。公司并不会直接辞退员工，而是让他先转到其他部门，并且给他半年的调整期。这期间他的原岗位会被新人替代。总之，谁有本事谁上，能者多劳，多劳多得。

最后，给员工放权之后，管理层做什么呢？在堂吉诃德，管理者的主要作用是培训员工以及责任共担。一方面他们要负责教授员工必备技能，包括如何陈列、如何订货等等。另一方面员工出现问题，管理层要共同担责。对于只管事不担责的情况，堂吉诃德是零容忍。

换句话说，对堂吉诃德这种SKU极多的杂货店，它的所有决策都必须依赖一线员工的体感。而调动一线员工的基本方式是放权，并且设计责权利一体的管理制度。

第二，向萨莉亚学低价策略。这几年萨莉亚的热度很高，关于它的传闻也有很多。比如，萨莉亚为了提高效率，规定店员用拖把拖地时，必须沿着一个方向，不走回头路。再比如，萨莉亚建立了自己的种植园，用来控制成本。再比如，门店还规定店员每次走动必须产生有效动作，没事在店里走来走去浪费时间，等等。
