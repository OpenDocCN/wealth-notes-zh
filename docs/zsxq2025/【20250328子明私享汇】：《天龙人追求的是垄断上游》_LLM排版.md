
# 《天龙人追求的是垄断上游》
# 源自3月28 日电话沟通
# 发布于 2025 年 3 月 28 日

很多国际上的事情，
把美国视为北京，把其他国家视为各省，
就会变得通俗易懂，
譬如特朗普的教育双减、川明医改，美共中央的八
项规定。
美国收购打通美国东西两岸的巴拿马运河，
就是北京出台《有效降低全社会物流成本行动方
案》,
美国逼迫加拿大、乌克兰、格陵兰等资源地区重新
选举，就是北京对包头、赣州、贵州等资源地区进
行反腐，
中美都需要搞新质生产力，
双方的天龙人也都需要提前的控制新质生产力所
需要的资源与原材料，
并牢牢的锁住对手原材料的交通咽喉。

天龙人们的生意逻辑跟大众是不一样的，
人家追求的是稳定无风险的利润，
参考当年新冠疫情期间，
天龙人家族掌控的都是疫苗和试剂的原料，
想要买都需要现款现货，甚至提前打钱。

他们操纵着舆论，可以轻易赚的盆满钵满，
而大众熟知的张核子这种台前的妖怪，
既要垫资承担风险，也要承担老百姓的骂名，
甚至科兴疫苗之父杨晓明，最后也得进去。

航天投资董事长韩树旺、 中原银行原董事长徐诺
金、南充市委原书记古正举、国药集团原总工程师
杨晓明，均涉嫌严重违纪违法。

当然，美国政局的变化，让最终的方式也出现了变
化，
之前北京是拜登模式，目的是彻底颠覆原有的旧秩
序，
数据资源的马云马化腾，实体资源的俄罗斯乌克
兰，都没有好日子，
现在北京是特朗普模式，手段变成了入股合作开
发。

国内的马云马化腾，国际的张一鸣李嘉诚，都能拿
到一个好价格。

随着东西方的天龙人先后对上游资源完成了整合
控制(换房本),
那么打压经济遏制发展，靠“赢麻”度日的阶段也
就会过去，
中美都会转为权力推动经济建设，并撒钱刺激消
费，
而他们掌控的舆论机器，也会转变为呼吁奋斗(996)
和消费(消费贷),跟前几年完全不是一个样子。
