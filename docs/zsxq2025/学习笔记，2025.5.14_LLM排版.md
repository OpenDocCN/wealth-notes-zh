
# 学习笔记，2025.5.14

1，跟气场相投的人在一起，你会感觉内心特别平静、放松。人与人之间是有磁场的，当你身边有善良的、契合磁场的人的时候，你的潜意识就会感觉到非常平静，无需太多顾忌，可以做回最真实的自己。反之，如果跟某些人共处一室，你会觉得焦虑、烦躁，无论如何都静不下来，那么说明这个人的磁场跟你相克，你需要尽快远离。

2，如果家里的顶梁柱不能解决经济问题，那么儿女的人生将会多走 10～20年的弯路。经济问题是一个家庭的根本问题，不解决的话，人生将永无出头之日。混日子的父母相当于把儿女推向一个更加艰难的沼泽地。那些完全不靠父母而逆袭翻盘的人，都是凤毛麟角，小概率事件。没有父母的托举，大部分孩子都要在这残酷而复杂的社会竞争中饱尝人间的辛酸与艰辛，受尽他人的白眼与刁难，可能一辈子都无法脱离贫穷与痛苦。

3，小白的成功的需要先质疑自己，因为脑袋里装了太多错误的观念。人是观念的产物，之所以是小白，之所以不成功的根本原因就是观念有问题。小白有两条路要走：第一、清除自己认为正确的错误观点。第二、践行正确但不理解的观点。现在想想，听话+执行实际上是很高的门槛，因为这意味着他否定了过去的观点。

4，许多事情不需要一鼓作气。比如你想锻炼身体，不需要一下就每天一小时。你就是每天三分钟，甚至经常断断续续，也没有关系。比如中间断了半个月，继续再锻炼三分钟就可以。韧性才是真正的王者。过于追求一鼓作气或过程完美，很容易虎头蛇尾自暴自弃。

5，我们大多数人都是不聪明的，那赚钱跟聪不聪明，其实没有太大的关系，它跟执行力有关系，聪明的意义在于哪里？无非就是学得快一点，悟的快一点。普通人尽可能把所有脑力上的竞争变成执行力的较量~

6，大多数人不具备出丑的承受力，出丑的承受力是什么？就是永远毫无波澜。自己表达观点的时候，表达不好的时候也大胆的表达，在初期写不好文案的时候，也敢展示。他的内心是王者模式，哪怕前期是小丑，也只是暂时的~

7，说个残酷的真相，只有社会底层男人才会将工资收入“上交”。在家里把工资交给老婆管的，都是社会最底层的男人，地位低，收入低。而略微有些成就的男人，没有一个是把钱交给老婆管的。因为他平庸没有地位，收入还低。他只能以牺牲个人尊严和家庭地位的方式来讨老婆。不然的话他可能根本找不到老婆。做生意的，稍微有些头脑的，除非他老婆有经商或者财务管理的才能，一般也不可能把钱交到他老婆手里。说到底，钱是一种能量，管钱需要才能。让钱生钱更是一种本事。一般人根本不具备这个才能。真正的做法是谁有理财的能力，谁来管钱。

8，“人必须出门，必须社交，必须去体验不同的事物，见各种各样的人，脑子才会思考，才能感觉到自己是在活着，不管你出门干啥，哪怕是去咖啡厅坐着，长期宅家的后果就是失去灵性，变成彻头彻尾的呆子。总之，别让自己烂在家里，只有出去，你的美好灵魂才能呼吸到新鲜空气，只有跟人接触，你的美好肉体才能得以释放它的魅力。

9，要在前路荆棘的时候依然前行，要在迷茫的时候依然前行，要在绝望的时候依然前行。一句话：要在任何时候前行。因为只要前行了，荆棘也只是一条走得慢一点的路，迷茫也只是内心假想的焦虑，绝望也不过是内心假想的恐惧~

10，有个说法是，人无为而富。听上去很妙对不对，毫不费力但把钱挣了。金钱不是努力追逐的目标，我们越陷入“赚钱焦虑”，越违背了赚钱的本质。财富法则在于：成为与金钱同频的人。培养丰盛意识，消除匮乏感，提升自我价值，保持对财富的信念，顺势而为做自己擅长的事。让内在能量与财富能量匹配，这时金钱就会像水流般自然涌入。洛克菲勒也说过，“财富是对认知的补偿，而不是对勤奋的奖赏。”财富积累的本质，是内在能量的显化。
