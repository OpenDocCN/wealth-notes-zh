
# 学习笔记，2025.7.10

1，战争跟地震其实是一回事，都是定期释放压力，让关系更加平衡，有邪恶，有正义，邪恶可能短期内会席卷全球，但是最终还是会被打回去的。譬如成吉思汗，譬如希特勒。世界一定是越来越文明的，越来越好的，这个主旋律是不会变的。

2，重复性的劳动会占用大部分精力，使人没有精力去学习，去思考，去进化，而同龄人在日新月异的进化中，你是在服务人民的过程中，不断的原地踏步。等于，你在退步。人，一定要在一个高进化的岗位上。要么，搞知识。要么，搞人。例如，你在大局办公室主任的角色上。那绝对，每天都在进化。若是非让我给成年人一个建议，我的建议就是必须要有一个独立的空间，家人也不能去的地方，这里完全属于你，你不儿子，不是父亲，不是丈夫，什么都不是，只是你自己，你在这里可以认真的学习与思考。一个人腾飞的关键是复盘能力 思考能力，而独立的空间就是这一切的前提，那在车里不可以吗？感觉不一样！

3，对于一些机会，为什么基层老百姓最疯狂？因为，他们渴了好久了。工作，不顺利。收入，不顺利。人，一旦渴了，就容易乱步伐。不是有个成语嘛，病急乱投医。

4，要想在股市中赚到钱，必须要明白一个最基本的道理，你不是股神，你战胜不了国家队，国家队什么水平？能看到底牌，能左右走势，资金量越大，其正收益无限接近于5%。年，社保基金公布了这两年炒股收益，一个年化7%，一个年化5%。这个收益若是放在雪球上，能被嘲笑死，其实，这已经是顶级收益了。于这一点，我调整了自己的预期收益，预期收益年化3%，然后拥抱超长线。

5，北京上海的同学，看其他地方的高中生活，没出过国？没看过演唱会？没参加过体育比赛？没爬过山？没去过西藏？他们觉得不可思议，你们高中三年都干什么了？县市的孩子甚有同感，就是坐牢了，除了学习，还是学习。

6，这两年，工作方针有没有大的变化？有，防守更多了一些，一切以别出错为前提。大刀阔斧往往会成为商鞅，结局惨淡收场。

7，我们要接受父母是普通人，我们自己是普通人，我们的子女也是普通人，在这个基础之上，要接受我们的国家是一个普通的国家，未必真的遥遥领先。这是我们普通人成熟的标志之一。

8，若是马保国没有登上擂台？他依然是一代宗师。普京若是不打乌克兰，俄罗斯也是超级大国。有时，打一架是有积极意义的。

9，男人的中年危机来源于，以前的无限可能渐渐变成了一条狭窄且一眼望到头的路，于是开始绝望、恐慌。女人则正好相反，人到中年，突然发现，别人贩卖的焦虑或规范，都是虚张声势，于是渐渐自由。所以，在不影响他人以及尊重内心的前提下，人是可以尽情的愉悦自己，要平衡好家庭、社会、自己三者的关系。

10，餐饮、穿衣，目前中外差距还是蛮大的，例如我们现在还没有形成不同场合着不同服装的习惯，上层群体已经具备了，老百姓还没有。穿衣，还没有进入得体阶段。还有一点差距非常大，那就是烟草文化。中国的烟盒是全球最漂亮的，如艺术品一般。老外的烟盒，全是烂嘴或烂腚的照片，让你看了就恶心。那天，我看了一个有意思的观点，说上海、北京人均寿命长，除了医疗条件好外，有个更重要的原因，禁烟环境最好，而其它地方呢？越是密闭的环境，越是烟雾缭绕……这就是我鼓励大家的地方，时刻的去看一下更前卫的生活方式，然后去照镜子，一一改进，让我们越来越现代，不要做一个守旧的人，要做新时代的人！
