
# 美股史诗级反弹预示什么？

市场反弹能否继续？昨夜，特朗普宣布全球关税暂缓90天，并把“对等关税”大幅下降到10%。与此同时，特朗普认为中国“不尊重全球贸易”，因此把中国关税上调到125%。美股松了口气，标普继续从我周末预测的“一步到位”的4800点关键位置开始反弹。纳指一夜之间飙升了逾12%，是美股历史上第二大单天上涨。而市场隐含波动率VIX暴跌了35%，单天幅度历史上最大。对于全球市场来说这是历史性的一夜。

回溯历史，昨夜美股这样的盘面出现的时间点全部都在美股极端的熊市中。美股最大的单天涨幅出现在2001年1月3日，也就是互联网泡沫破灭之后，美国经济即将进入深度衰退之时。其它显著的单天涨幅，都出现在2001年4月美国衰退迹象逐步显现、2008年雷曼倒闭之后以及2020年新冠疫情。简言之，熊市多暴涨，期间出现的史诗级单天涨幅更多的是熊市的确认，而并非牛市的开始。昨夜美股成交量再创历史新高，这是过去几个交易日第三次创出的历史新高。这也和熊市底部特征不太相符。

这已经不是第一次特朗普用自己的“迷之自信”影响市场。在昨夜宣布关税延迟之前，特朗普在社交媒体上宣称“现在是买入的好时机”。历史上，2018年12月25日，在那年美股由于美联储决策失误而导致一个季度暴跌了25%的行情时，特朗普也曾发过一个类似的推文，美股随即触底反弹。

其实，在市场运行到这样的关键位置的时候，比如我们之前预测的标普4800点的下跌点位，任何消息都可以引发反弹，甚至连坏消息也可以被解读为利空出尽。然而，在特朗普对于自己对于市场的影响力屡试不爽的时候，市场出现如此历史性的波动反而将增加特朗普日后进行关税谈判肆意妄为的可能性。也正是因为对于自己的能力的误判，将很可能导致特朗普误判了美国经济的韧性，把美国经济乃至全球经济都带到沟里。

昨晚特朗普折腾之后，国际投行刚刚大幅上调的美国经济衰退的概率又被大幅下调。然而，如今美国的经济和市场的走势都取决于一个“疯王”的一念之差，因此压力其实是在上升，而非下降。即便是昨晚史诗级的反弹，主要股指依然还在200天均线的下方。当然，毋庸置疑的是，短期内市场的技术反弹应该能够得以延续。

之前，我们还认为恒生科技指数在4400点左右是一个重要的点位。昨天早盘，尽管海外市场还是风雨飘摇，恒科低开近6%但却强势收涨近3%。这个单天近10%的反转，也是具有历史意义的。那么，昨夜美股的历史性行情以及特朗普的新的125%关税，对我们中国的影响又是几何？

显然，特朗普关税是冲着中国来的。昨夜，特朗普故意把中国孤立出来，并以“不尊重”为借口加码关税。但试问谁会尊重一个赖皮耍泼的老头？但怀璧其罪。我们在之前讨论过这种极限博弈中中国的选择：博弈拖的时间越久，对于美国则越不利，因为美国需要付出更多的成本来达到一个更低的目标。当然，整个博弈能够产生的利益总量也将随着时间的推移而下降，中国也难免需要承受谈判的成本，但中国的谈判成本远比美国的成本低。这是因为，在一个高通胀预期的环境里，美国对于价格波动的敏感性远远大于当下处在没有任何通胀压力的中国。同时，美国在博弈利益中的占比也将随着时间的推移而减少。这就是为什么特朗普急于迫使中国来谈判、急于达成一个协议的原因。我们可以简单地算一下，美国从中国进口大约1.5%GDP。如果现在对于这部分进口增收125%的关税，那么对于美国经济增长的影响将远超1%。而今年美国GDP的增长预期还不到2%。特朗普这样作下去，美国很难不衰退。

此刻，股市作为一个最显性的指标，也将成为谈判的筹码和输赢的评判标准。昨夜特朗普的让步，显然是受到了最近市场史诗级波动的干扰，马上证实了特朗普谈判的软肋。而中国股市如果能够继续保持它的韧性，那么在谈判过程中中国将获得更多的主动权。顺带说一下，年初至今，中国市场表现依然大幅优于美股。

如是，稳定市场应该是近期的工作重点之一。外媒报道，一系列的会议和讨论似乎在密锣紧鼓地进行。如无意外，我们应该会看到更多、更坚决的措施来支持市场和经济。在上一篇专属文章《洪灝：关键时刻》的结语中，我们写到：“短期，无论如何，在这个位置上都应该有技术反弹：国家队将继续出手稳定市场，利率存准将很快下调，促消费政策也将很快出台。当然，这一切都将在特朗普关税的背景噪音和负面的新闻头条中进行。市场风高浪急，交易逆水行舟。”
