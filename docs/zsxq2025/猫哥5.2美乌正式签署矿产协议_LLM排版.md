
# 猫哥5.2美乌正式签署矿产协议



当地时间4月30日，美国和乌克兰，正式签署了矿产协议。不得不说，这个消息还是比较令人意外的。而更令人意外的，是协议内容。按照一般理解，特朗普在白宫吵架后，丢了面子，必须在矿产协议上把场子找回来，但我们仔细看看协议的条款，就会知道，他没有找回场子。乌方的诉求，一直是获得安全保障和援助，用资源换美国的安全保障。而特朗普的诉求，是在不给安全保障的前提下，获得乌克兰的基础设施、港口和资源，理由也很充分，你们拿了美国那么多援助，欠美国钱，必须还。但这份协议里，这些都没有。有的只是，双方同意设立一个基金，各持股一半，天然气和石油，还有稀土资源的日后投资收益，将汇入本基金。内容就这么点。如果细究，其实美国是吃亏的，第一，特朗普坚持乌克兰应该还钱，也就得在矿产协议里注明，欠美国多少钱，其实，特朗普这个说法，不是纯粹的信口开河，因为确实有一部分美国对乌援助，是以贷款形式发放的。但现在协议没有涉及，等于这部分钱，是收回不来了。乌克兰连你会名义上的债务，都不愿意承认。第二，收益没有保证。虽然协议中注明，天然气、石油和稀土等多种矿产资源的收益，注入基金，但我们以前的文章就说过，乌克兰没有多少油气和稀土，那点储量只是聊胜于无。而且，协议还有补充条款，已经在开采的项目，收益不会注入基金，只有未来新开发的项目的收益，才会注入基金。这简直是个大坑。因为开新项目就得投钱，乌克兰肯定没钱，要开采，只能美国单方面出钱，美国出钱雇佣乌克兰人，美国买设备，美国负责销售，拉动乌克兰经济。乌克兰啥也不出，最后收益还分一半，不知道的，还以为特朗普是大慈善家。这个补充条款，还有政治意义。如果有不少美国公司，在乌克兰投钱开项目，那不就等于自身利益，和乌克兰捆绑了?如果乌克兰战败，他们的投资就要遭殃。到时候，油气利益集团，还会支持“联俄”?都说联俄是没有希望的，特朗普不信，总觉得和普京关系好，就能联俄，现在知道厉害了吧?第三，就算有收益，也带不走，因为协议规定，基金收益必须用于投资乌克兰重建，如果要有特殊做法，必须取得双方同意。问题是，乌克兰不可能同意，美国又没有控制权，等于赚了钱也带不走。说实话，哪个美国企业会这么傻去投资?所以，这份协议很难落实，特朗普想靠开采乌克兰资源回血，更是天方夜谭。既然这份协议缺乏实际意义，为什么还要签呢?因为特朗普需要“赢”。我们知道，上台三个多月了，东一榔头西一棒槌地敲了半天，看似整活很多，实则收获寥寥。外国领导人，一个妥协的都没有，连一份重要协议都没签，特朗普太需要政绩了。尤其是对全世界开展贸易战以来，美国国内反对声浪很大。本来吧，民主党在大选失利后，一度陷入了怀疑人生的地步。对特朗普的各种刀劈斧砍，未能做出有限的抵制，连国际开发署被端，都没能阻止，还出了 一些叛徒。然而贸易战后，民主党看到了机会，开始重新集结。贝索斯旗下的亚马逊，甚至还想在商品中标明，受到贸易战的影响程度，也就是这个商品的价格里，有多少是贸易战加的税。反对的不只是民主党，还有许多党内诸侯，对特朗普的质疑声，正在四起，他面临上台以来的最大危机。这时候，只有政绩才能堵住悠悠众口。恰好4月30日，是特朗普的执政百日。西方政客是很看重执政百日的，特朗普也不可能例外，在密歇根州举行了盛大的庆祝仪式。为了给庆典活动增光添彩，必须拿出点政绩。而乌克兰愿意签矿产协议，也算是政绩，一时半会，也找不到其他愿意妥协的人（哪怕只是名义上妥协），只能找泽连斯基。只不过，就这份协议的内容来看，还不如不签。毕竟你嚷嚷了这么久，说要让乌克兰还钱，要吞并乌克兰的矿产、基础设施和港口，一副气势汹汹、要把乌克兰生吞活剥的样子。为此，还和泽连斯基在白宫吵了一架。结果到了最后，竟然是这么个内容。请问，别的国家会怎么想?就算和你吵架，不给面子，也不会有什么后果，那我干嘛还要给面子?你斗不过中国可以理解，但连乌克兰也拿不下，算怎么回事?乌克兰可是被美国拿捏着的，别的不说，光是减少援助，就足以让乌军吃尽苦头，3月初，特朗普暂停情报援助后，没几天，乌军便在库尔斯克大败。一个命脉完全在你手上的国家，泽连斯基手上也没什么牌，没有能对特朗普造成压力。正常情况下，就算泽连斯基再硬，也应该谈出一个美国占优的协议，无非是美国给乌克兰一点面子，别赚那么多。结果呢?最后谈出来个空洞协议，内容还是明显对乌克兰有利的，简直让人大跌眼镜。再次证明了，所谓的极限施压，所谓的交易的艺术，不过如此。只要你不怂，敢于抵抗，怂的就会是特朗普，他会在急于出政绩的心理下，大幅让步。至此，急于出政绩这个弱点，算是充分暴露了。第一任期，他能靠极限施压讹到好处，主要是大家不熟悉他的风格，生怕他来真的，把桌子给掀了。谁也不愿意看到这一幕，宁可出一点利益，换取他的消停。但随着时间的推移，大家已经日益熟悉他的三板斧，知道了他没有长期计划，更没有掀桌子的决心，应对起来，就从容得多了。斧头一旦钝化，再想讹到利益，就难了。
