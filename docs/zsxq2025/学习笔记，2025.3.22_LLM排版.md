
# 学习笔记，2025.3.22

1，孩子一生是什么曲线，都是他的命。X副局长，当年她被警校录取了，是第三志愿，她进了学校才知道，警校是包分配的，同学的父母多本身就是系统内的，遗憾的是，她毕业时，政策正好改了，不接收了，于是她重新考的公务员，进了其它部门。她后悔没当上警察不？不，大学宿舍里四个女孩子，两个已经出事了。赵德发老师经常说的一句话是，每本书都有属于自己的命运。其实人生也是如此，我在骨科住过院，在眼科住过，这两个科室都是意外高发群体，例如截肢的人，昨天也许还活蹦乱跳的，例如眼球摘除的，也是如此。不要对孩子有过多的担心。我以前也总是犯类似的错。担心这，担心那。他们有属于自己的命运，若是我们有绝对的把握，例如我是医院的院长，我可以依托自己的资源去安排孩子学医、就业、提拔。准入门槛相同，但是提拔门槛不同。

2，普通人并不具备安排孩子余生的能力。县城里，若有，要么企业家，让孩子们接班。要么大局长，以上。咱就是普通人，与其按照咱对世界的认知去给孩子铺路，不如让孩子野蛮成长，有概率会飞的很高。三甲医院主任家的娃，主任还没退休，德高望重，都没有能力把娃安排进本地三甲医院，孩子考到了临县医院，每天来回跑……这个级别的人，在咱眼里，就已经是县城王了，他都很无力，别说我们这些屌丝了。

3，我接触的体制内朋友越多，越害怕，因为他们每个人都是半白半黑，给人的感觉城府很深，你不敢交心，总觉得他们时刻在套路你，这是什么造成的？他们同事之间彼此防着对方。把这种提防应用到了社会交往中。从这一点，我就觉得，体制内真的累，全是内耗，关键是工作没有成就感，说混日子不合适，每天也很忙，但是忙什么，他们也不知道。剩余点精力，都用在搞人与被搞上了。我不觉得这是一份好的工作，心累。

4，什么算是好的铺路？去年或前年，我参观了一家知名企业，老板的两个娃，一人分管一个厂，我当时就在想，这就是最好的铺路。还有一点，父母的言传身教，这也是好的铺路。虽然，我总是丑化我爹，但是我爹在我心目中，一直都是偶像级的存在。他永远都是村里最勤奋的那个。我也是同龄人里，最勤奋的。这就是耳濡目染。我娃们，虽然我骂他们是劣质基因，但是等他们长大了，他们也会是同龄人里最勤奋的，在不作恶的前提下，足够勤奋，有目标。不可能不成功。我爹是他们那个圈子里的有钱人。我是我们这个圈子里的有钱人。娃们也会是他们圈子里的有钱人。

5，买电脑跟买车一样，不能只看配置，要看品牌，看服务。很多青年买电车的逻辑也是这样，不看品牌，不看服务，只看配置。买国产电脑，就在两个品牌之间选，要么戴尔，要么联想，选电脑不能只看配置，确定好了大概价位的，随便选一个。

6，医学，哪些领域比较值得进入。急诊，肯定不值得。济南大医院罢工，多是急诊。干活多，油水少。外科医生，也累。牙科，这个我认为是可以的，但是对手艺要求很高，就是不仅仅要有一流的理论，更需要一流的实战能力，尤其是矫正、种牙，要手到擒来，牙医是真正的口碑饭。女性生殖类，辅助生殖、产后康复，都可以。康复系列，运动康复，手术康复。但是，这些也只是咱站在今天去看，一年一个变化，若是只看最近三五年，感受不到翻天覆地的变化。若是以20年来看？那真是翻天覆地。我跟大部分灰心派不同，我看多中国，看多中国的蒸蒸日上以及纠错能力。

7，现在，想去哪旅游就旅游，想要住几天就几天。在过去，这是不可能的，是真得痴心妄想。三件事，被动改变了这个局面。一是XXX案，当时《南方周末》报道了，全国律师呼吁。二是加入了WTO，用工荒，大家开始跨区域打工。三是大学扩招，大学生全国找工作。当时，社会是什么治安？老百姓是什么状态？镇上收税员在集市上收税，谁若是抗税？秤杆接着就给掰断，还会扇大耳光，现在谁敢如此？谁敢耀武扬威？一切越来越好了。是ENJOY，是发光，是发热。

8，有个国家队的运动员，我问他，外籍教练与本土教练最大的差别是什么？他说，本土教练喜欢讽刺、打击、挖苦，甚至动粗，外籍教练喜欢鼓励，从而队员在场上更兴奋。

9，教人写作，就一点，成为习惯，哪怕每天日更100 字，先成为习惯，当你每天都在想，我要写点什么的时候，其实就被动去观察了，被动去思考了，你知道作家是什么时候开始写文章的吗？看到或听到一件事的时候，已经归类进素材库了，也就是说，已经开始写了。但是，靠纯文字想吃上饭，很难，除非是写专业类的，例如你是程序员，写给程序员看，你是滴滴司机，写给滴滴司机看……

10，说个大家不喜欢听的，老师这个群体最喜欢举报，没有之一，内斗绝对专业。体制内为什么这些事比较多呢？因为它不同于社会上，社会上多是各有护城河，体制内不同，身边同事就是自己最大的竞争对手，就那么一个椅子，大家都要抢，抢不到自然就生恨……当如此近距离看到这些时，还是感觉到了人性之恶，这都是受过高等教育的，心胸却如此的狭隘、阴险。所以，我们一定永远也不要当举报者。
