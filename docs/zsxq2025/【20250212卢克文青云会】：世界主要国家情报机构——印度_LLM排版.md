
# 世界主要国家情报机构——印度

印度在1949年以后，在领土争议这个问题上，目前来看，它是占了便宜的。当然这个占便宜里面也牵涉到许多，比如说有大和尚的问题，英国人的问题，也有中印之间所谓南方国家的关系在里面。如果我们从领土这个概念上来看，其实印度人现在还是占了优势的，我们不得不承认这是个现实问题。

印度是1948年建国，我们是1949年建国，虽然我们和印度都是属于全球南方国家，但是印度在维护本身本国利益上，他始终是实力原则，跟我们不太一样。我们其实有一段时间不是以实力为原则的，我们是以意识形态为原则的。尤其是我们在1949年到1979年，这30年中，我们的维护国家利益的哲学的基础是意识形态。

一直以来，印度虽然他一方面说他是不结盟运动国家，另一方面他也说是全球南方国家，但是他始终还是相信实力原则，我有这个实力我就要搞定这个事情。这个实力原则，其实有两件事情是最典型的，第一件事情就是吞并西京，第二件事情就是孟加拉国独立，也就是东巴基斯坦独立。这两件事情，从情报学术的角度来看，是非常成功的两次这种行动。

可以说，印度在这两次行动中展示了他作为地区大国的实力，以至于后续的影响力持续到现在，到现在为止，在南亚大陆上，包括在印度洋，我们讲泛印度洋周边的国家里面，周边大大小小的国家对印度还是有忌惮的。比如说马尔代夫、斯里兰卡、孟加拉国、缅甸、尼泊尔、不丹，包括巴基斯坦，甚至阿富汗。

正因为它在南亚地区的这种实力原则，或者说它所展现出来的实力，也让他能够在国际舞台上具有一定的影响力。比如说他之前和前苏联的关系，后来和俄罗斯之间关系，包括后来和美国之间的关系，和英国之间的关系。现在是印度的一个比较好的窗口期，是因为中美现在在博弈，印度的战略价值就提升了。

当然这个话只能说到今年的1月20号，特朗普上台以后，对于印度的看法可能会有一些变化。我个人觉得可能在这个情况下，莫迪也好，或者说特朗普也好，可能也会重新调整他们彼此的对印和对美关系。

那回到底，是什么支撑着印度所谓的实力呢？一个是硬实力，一个是软实力。硬实力就是印度军队，印度军队虽然被号称万国造，但是这么多年的发展还是具备了一定的军事实力和国防实力，尤其是在国防工业上。我们中国人好像老是揶揄他，什么飞机摔了，轮船翻，枪打不响，炮弹打不响，但是不可否认，他这么多年在他那个底子上，就在他也不怎么富裕的家底上，经过几十年的发展，他确实基本上拥有了一整套国防工业体系。

至于这个能力高低那是另外一回事情，但是他现在已经有了，所以这个我们是要承认的。在软实力上，印度就是非常依赖于他的两大情报系统，其中最重要的情报系统就是我们今天讲的印度调查与分析局，也叫RAW。

另外还有一个情报部门，叫印度情报局，叫IB，但是印度情报局是近代历史上历史比较悠久的一个情报机构，它秉承于英国殖民统治时期的所谓的印度陆军的军事情报局。因为英国人在统治南亚的时候，他的陆军总部就是在现在印度的西姆拉，而西姆拉也是印度的一个避暑胜地。

你别看印度一到夏天，40度、50度，热死一大堆人，西姆拉和大吉岭这两个地方却是非常好的避暑胜地。西姆拉后来也成为了英国殖民者夏天避暑的地方，也在这个地方建立了当时的英国陆军的驻印英军总部以及后来的军事情报局。

今天我们讨论是RAW,IB 我们不讨论，因为IB现在已经有点开始被边缘化了。调查和分析局也就是叫RAW, 它是1968年成立的，它最重要的目标，最重要的工作就是关注中国和巴基斯坦，它就是为了这两个国家设立的。

经过这么多年的发展，它现在已经成为了印度政府提供战略性和全局性情报的主要来源，并且能够参与到最终的国家战略决策。而且RAW又与包括美国中央情报局、英国军情六处、以色列的摩萨德、俄罗斯的对外情报局等世界知名的情报机构，展开长期且深入的情报合作。

那么RAW在设立之初，就是直接归属于印度总理直接管理。它不属于印度国会，也不对印度的内阁负责，也不对印度的总统负责，它只对印度总理负责。它现在的主要架构根据公开的资料来显示，它有对外情报部门。对外情报司主要负责汇编和分析来自印度以外的情报数据，技术支持司主要是通过技术装备来搜集和整理情报，同时为RAW 的其他部门执行情报作业的时候提供技术支持。

航空研究中心是一支专门用来执行空中监视和侦察的任务。

在这个地方我稍微展开讲一点，印度对于西藏的企图心是非常长的。上个世纪的八十年代，印度空军的米格-25军用侦察机，因为米格-25是属于超音速，而且速度非常快的一种战斗机，但是它有一种侦察型的，它可以执行战术侦察任务。

当时印度空军的米格-25可以在我们的拉萨上空进行低空的通场。因为我们当时的空军没有飞机可以对米格-25进行跟踪和威胁，而且当时我们在西藏也没有完善的前进机场。所以在很长一段时间，印度空军在西藏上空是比较嚣张的。

那么这个局面是到后来我们开始在西藏建立了前进基地以后，随着第四代战斗机的开始入驻、开始轮训，印度空军的这种战术侦察行为就后来被压了下来。

但是，印度在这个方面其实一直以来虽然是比较欠缺的，以至于中印在洞朗事件期间，印度缺乏有效的空中侦察力量，不得已将印度海军的P-8反潜飞机调到前线，作为空中侦察飞机在用。P-8是一种反潜飞机，它主要执行的是这种长航时、大机动的反潜作业。

但是他后来实在没办法，因为他没有办法掌握到第一手情报，所以他不得不将P-8从海军调到洞朗地地区。足以可见，情报作业无论是在战略层面，还是在战术层面都是非常重要的。

其实，印度后来在洞朗事件之后，深感自己在航空航天侦察领域的不足，他就开始采购美国的无人机，后来又向美国的中央情报局和美国的情报单位购买高精度的空中侦察卫星照片，甚至他希望美国能够向印度提供实时的空中侦察情报，也就是像现在北约对乌克兰的这种情报支持一样。

他希望由美国方面派出侦察机，或者说是无人机，通过战术数据链，实时向印度在洞朗地区或者是在中印对峙的地区，就争议领土地区能够实时发送情报，这个要求是非常高的，你几乎要做到同步。我们有什么行动，他可以在第一时间知道。
