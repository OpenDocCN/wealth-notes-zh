
# ZIPPO中国-高婧文-从“上新”到“造浪”，新品营销如何完成跨圈层传播？

## 品牌历史和发展

ZIPPO品牌的发展历程可以概括为以下几个关键节点：

- **品牌起源**：ZIPPO火机由George Blaisdell在二战期间创立，最初作为军用打火机投放市场。
- **材料改进**：ZIPPO将战争物资黄铜改成钢铁，并在外壳喷涂黑漆，形成独特的“裂纹”效果，成为二战期间唯一的款式。
- **市场拓展**：ZIPPO进入中国市场，并在北京设立自营和天猫店铺，推出90周年纪念款，发布2024-2025设计计划。
- **多元化发展**：ZIPPO推出纪念版打火机，并进军户外、音乐节等场景，推出香氛产品等多元化产品。
