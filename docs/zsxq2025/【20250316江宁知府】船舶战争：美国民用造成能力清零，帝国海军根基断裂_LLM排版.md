
# 船舶战争：美国民用造船能力清零，帝国海军根基断裂

根据美国贸易代表办公室的数据，在中国制造的全球商用货船比例从1999年的5%激增至目前的50%以上，与之形成鲜明对比的是美国造船业快速衰落，海军舰艇也普遍出现延期交付现象。

面对这一困境，Trump政府近期频频就中美造船业发声，对外宣称要对使用美国港口的中方船舶征税，对内则提出重振造船业。

3月4日，Trump对国会参众两院发表演讲时公开表示：“为了加强我们的国防工业基础，我们将重振美国造船业，包括商业造船和军用造船。为此我宣布，将在白宫设立一个新的造船办公室，并提供特殊的税收优惠，让这个行业回归美国，这是它应有的归宿。”

为什么造船业会让美国政府如鲠在喉呢？

1. 二战期间，美国曾在短短数年内建造了1500多艘各式舰艇，其中包括数十艘航空母舰、巡洋舰和数以百计的驱逐舰、护卫舰。当德国潜艇部队司令邓尼茨从报纸上看到美国造船厂的下水纪录后，他长叹道：“看来，我们在这场战争中要失败了。”

放眼今天，美国的造船能力主要集中在四大船厂——通用动力电船公司、巴斯钢铁造船厂、纽波特纽斯造船厂、英格尔斯造船厂，其他一些中小型船厂已经失去了新造船能力。而在四大船厂的营收中，维持运转的订单高度集中于军船，其产品在民船领域已彻底丧失竞争力。

众所周知，寓军于民、军民两用是一种相对经济合理的军工生产模式，像美国造船业这种完全依赖军品的做法，只能无限推高军舰建造的成本，从长远看是不可持续的。

2. 一个广为流传的说法是“中国造船能力已经超过美国的200倍”，这个数据其实来自对过去几年中美新造规模以上商船的统计。美国每年建造约五艘商船，而中国建造1000多艘，被西方媒体称为“像生产饼干一样批量生产船舶”。

3. 麦肯锡顾问公司的数据显示，自20世纪50年代以来，美国的造船产量下降了85%，能够建造大型船只的造船厂数量下降了80%以上。知名智库战略与国际研究中心（CSIS）主任 Cynthia Cook 更是公开宣称“造船业显然是美国工业的一个薄弱环节”。

实际上，民船建造能力跟军船建造能力很多时候同样重要，因为征调民用船舶参与战时运输是常见且必需的。有美国学者表示：“一旦发生紧急情况，一旦与实力相当的对手发生冲突，以现在的状况我们很难将大量物资、设备和部队运送到大洋彼岸。”

4. 不只是民用船舶建造能力几乎清零，美国的军船建造能力也在迅速下降中，过去三十年里，美国海军舰队已从1992年的471艘舰船下降到现在的295艘。

在今年2月底的听证会上，共和党参议员 Roger Wicker 对候任美国海军部长 John Phelan 说道：“（海军的）几乎每项重大计划都落后于计划时程、超出预算或完全偏离轨道。”

去年美国国会曾进行了为期45天的海军造船审核，结果表明，许多项目严重落后于计划进度，例如“星座”级护卫舰的交付预计将推迟三年。

不仅如此，高昂且漫长的船舶维修过程正严重制约美国海军的战斗力，海军作战部代理副部长 Jim Kilby 上将表示：“我们得到一艘新船时，就会替换一艘旧船，因为旧船更贵，维护起来也更难。”

美军维护旧舰究竟有多难呢？以“尼米兹”级航母“华盛顿”号2017年8月开始进行的中期大修和更换核燃料为例，根据美海军声明，这类维护工程通常需要4年时间，但纽波特纽斯造船厂直到2023年5月才完成相关工作，导致排队等待维修的“斯坦尼斯”号航母的完工日期也大幅推迟。

实际上，目前已经有美军舰“出海就医”，到韩国、日本、印度等国家的造船厂进行维护保养，即修条船需要远赴重洋。

因濒海战斗舰项目失败，美国海军推出了“星座级”护卫舰项目，该舰设计排水量约7000吨，相当于052D驱逐舰的水平，然而这一被寄予厚望的项目一再延期，性能指标也在美海军内部存在诸多争议。

美国的产业保护政策通常分两个维度：打压对手+掠夺盟友。

根据美国贸易代表办公室最近公布的一项方案，压制中国造船业政策的核心是入港管制和减税措施。简单来说，如果中国船舶或扩大至由中国制造的船舶想使用美国港口，每次将征收100万至150万美元的入港费，并对中国制造的起重机等设备采取禁用措施。

光打压对手还不行，想重振美国造船业还得同时掠夺盟友。现阶段美国所占的全球造船份额仅为1%左右，与中国（50%）、韩国（26%）和日本（14%）差距很大，如果想迅速振兴造船业，中国的产能它“转移”不动，只能把心思打到日本和韩国身上，以“合作”为名抢夺日韩造船产能。

然而即便强取豪夺，也很难解决美国造船业劳动力队伍空心化的问题。造船业需要大量技术工人来维持运转，有美国海军将领忧心忡忡地表示：“没有人一开始就是造船工人，他们是焊工、蒸汽装配工和电工，如果没有稳定的工作，他们就会去哈雷戴维森、福特汽车公司、雪佛兰或其他公司工作。”

“早些年时，造船厂提供的工资远高于当地其他大多数工作，而现在，它们正在跟从咖啡供应商到快餐店的所有公司竞争。”

由于人员流失导致建造和修理能力大幅下降，一些原先技术领先的造船厂已经无法正常建造最尖端的舰艇。

对于一个大国来说，造船业衰落的后果有多严重呢？我们不妨看一下俄罗斯。

苏联解体之后，俄罗斯失去了在乌克兰黑海沿岸的大量造船厂，包括唯一能够建造航空母舰的尼古拉耶夫船厂。由于没有民船订单，只能依赖军费拨款，到头来俄国只有建造核潜艇的北方造船厂和个别建造破冰船的船厂能够勉强维系，其他大都倒闭关门。

在很多美国人看来，现在已经到了必须拯救造船业的时候，否则俄罗斯海军的今天，就是美国海军的明天。
