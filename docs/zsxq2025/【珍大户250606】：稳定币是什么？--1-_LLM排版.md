
# 稳定币是什么？

稳定币的作用与之前的地下钱庄类似，只是套了最新的数字技术，相当于地下钱庄搞了IT升级。回忆地下钱庄的原理：张三在A国往地下钱庄存入1000万美元，地下钱庄给他一个凭证，张三把凭证给李四，李四在B国就能把1000万美元取出来。这样张三和李四就没有能被官方发现的明处转账关系，由地下钱庄把钱转移走了。现在地下钱庄IT升级了，他们发Q币，张三存入1000万美元，钱庄发行了1000万Q币，张三把1000万Q币给李四，李四拿1000万Q币又能换回1000万美元，当然中间要损失一点手续费。

这跟地下钱庄不是一样吗？不过Q币发行的技术比当年腾讯要高，现在用上了区块链技术。

## 稳定币和比特币的区别

比特币之类的数字货币，当时发的时候固定了一个总数，而稳定币可以随时发新的，只要张三存入越来越多的美元，稳定币公司就可以按照存入的美元数量发新的稳定币。相反，稳定币也可以随时注销，只要李四取出越来越多的美元，稳定币公司就注销相应的数量。所以当年宣传的比特币能洗钱之类的，这么多年过去了，真用来洗钱的并不多，炒币的才是最多的，因为币值不稳定，哪个黑老板都不想3000万进去，1000万出来。但稳定币，这玩意挂钩美元或者港币之类的，这回是真的能洗钱了。

## 美元势力衰落与权利让渡

我感觉下来，对美国来说，这属于美元势力衰落之下的权利让渡，从而扶助美元势力。什么叫美元的势力衰落？本质是有其他越来越多的竞品逐渐出现，导致美元结算的比例持续下降，目前好像已经跌到49%，不到一半的水平了。竞品包括他国货币，比如欧元、瑞朗、日元、人民币等等。包括黄金和其他数字货币。包括大宗等价物，比如石油天然气结算、矿产结算等等。现在美国搞稳定币，等于在拉统一战线，本来数字币是美元的敌对阵营，而现在弄了对标美元的稳定币，是想把数字币拉到自己阵营，使美元版数字币来替代原有的数字币，这样，数字币的支持者可以被拉过来一部分人头，支持美元稳定币也就等于支持美元了，统一战线就扩大了，敌人的势力就减少了。

## 权利让渡的含义

意思就是说，之前美国强势的长臂管辖，可以以反洗钱为名，把瑞士银行的客户都筛一遍。可以以不同的名义，直接把俄罗斯这样的国家给禁掉，结果现在美元的国际结算占比比例一直在下降。此时美国如果说，我把反洗钱松一松，让之前被反洗钱禁掉的客户、和反战禁掉的国家，都能继续兜个圈子用美元，这不就能稳定一下美元的使用比例了吗？但直接说把历史上各种反洗钱废除了，那是不可能的。现在说把俄罗斯请回来，也是不可能的，帝国的脸面挂不住。但如果套了个壳，说对标美元的稳定币，谁都可以用，不就OK了吗。其实就等于在变相使用美元。

## 未来两种可能

比如通过拉长时间之后，美元的各种挑战都被打压掉了，美国又重回巅峰了，此时很好办。到时候再出个新法律，说稳定币公司涉嫌洗钱，要抄它的家，也就抄了。这就是典型的春秋战国之远交近攻，秦国最直接的敌人周边的邻居，是直接竞争对手，远方的国家反而还没有那种急的竞争关系，所以先跟远方国家结盟，先攻打邻居。对美元来说，最直接的竞争对手是跟他级别相近的其他国家的主流货币。相反，数字币的实力比较弱小，没有那么紧急的竞争关系。现在直接急迫的去抄比特币的家，没意思，应该主力放在打压欧洲和日本身上。如果最后赢了，以后自然可以收拾数字币。但也得等赢了以后再说。相反，如果未来美元再也强势不起来，那么恐怕还会更进一步的权利松绑，不仅要吸收数字版，恐怕还要想出更多的办法，来扶植使用美元。所以短期、以若干年内的视角来看，稳定币是属于受欢迎的东西，受美国政界和多国商界同时欢迎的东西。
