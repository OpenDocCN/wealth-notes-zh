
# 民主党正在悄悄发力

## 嘉宾：梦回安西

近期，就在外界的目光纷纷聚焦在新上台的懂王身上时，民主党却在欧洲，悄悄发力了，先后搞掉了两个关键的亲俄政治人物，分别是斯洛伐克总理菲佐，和罗马尼亚总统候选人乔治斯库。而这一切，会令俄乌停火的难度，明显增加。

先说斯洛伐克总理菲佐。很多人对菲佐的首次认识，来源于去年中旬他遭到枪击。由于是亲俄派，因而在枪击发生后外界格外关注。欧盟领导人里，有两名大刺头-欧尔班和菲佐，欧尔班的地位很稳固，其政党拥有绝对多数席位，一时半会搞不掉。而菲佐的呼声党只是相对多数（得票不到四分之一），地位谈不上稳固，他能当上总理，主要也是靠和其他党派合作，组建执政联盟。所以如果能让其他党派退出，菲佐政府也就垮台了。果不其然，12月份中旬起，斯洛伐克街头出现了大量的游行示威，要求停止亲俄政策，且人数越来越多，丝毫没有停下来的迹象。原因是总理菲佐访问了莫斯科，引发了欧盟和民主党的严重不满。1月，乌克兰掐断了天然气管道后，斯洛伐克的能源受到影响，菲佐选择报复，准备中止对乌克兰的人道主义援助。欧盟和民主党彻底炸了，向斯洛伐克执政联盟极限施压，其他政党顶不住压力，宣布退出。于是菲佐政府垮台。

比起菲佐，罗马尼亚发生的事情性质更恶劣。去年11月，罗马尼亚进行了第一轮总统选举，结果显示，亲俄的乔治斯库得票23%，排名第一，社民党主席乔拉库第二，得票21%，第三名是拉斯科尼，16%，第四名西蒙，14%。无人过半，就前两名进行二轮PK。考虑到第四名也是亲俄的，外加一些零零散散的亲俄票，乔治斯库在第二轮战胜乔拉库的可能性，极大。罗马尼亚是半总统制，总统的权力不小，乔治斯库一旦上台，可能大量减少对乌援助。所以第一轮的结果，引起了西方的恐慌。于是很快，乔拉库就声称选举存在舞弊，要求重新计票，并起诉到罗马尼亚最高法院，法院也是不要脸，直接宣布第一轮不算，明年重新选。理由是，乔治斯库涉嫌受到中俄的帮助，尤其是俄罗斯。最高法的依据十分搞笑，它们给出乔治斯库受到“中俄帮助”的依据，竟然是乔治斯库有个抖音账号(Tiktok)，他很会运营内容，吸引了不少粉丝，最高法院认为，一定是中国帮助引流了，增加了乔治斯库的人气，等于间接干预了选举。所以选举无效。实际上，乔治斯库和中国一毛钱关系没有，和俄罗斯关系也不算紧急，所谓不想援乌，就被贴上了“亲俄”的标签。由于过度不要脸，甚至引发了第三名，也就是拉斯科尼的反对，拉斯科尼是亲欧盟的。但欧盟和民主党才不管那么多，必须把一切风险，掐灭在源头里。等2025年再次举行选举，他们一定会采取一切措施，阻止那些不喜欢的人当选。接连干掉两名重要亲俄派，说明一个问题：民主党已经调整过来了。在大选结果刚出炉时，民主党上下表现得相当懵逼，不知道发生了什么，一度出现了瘫痪的情况，内部互相指责，在整个11月，民主党没有任何重大动作，就是这种懵逼的体现。但进入12月，民主党的状况明显好转，内部互相指责减少，一致对敌（懂王）的趋势，逐渐形成，先是联合共和党建制派，搞掉了懂王提名的司法部长候选人马特*盖茨，给了懂王胜选后第一个大挫折。接着开始全面发力，在各个领域给懂王挖坑，比如制定“算力种姓制度”，几乎完全不允许中国购买美国GPU，拜登明知这样做，等于向中国让出大片市场。却依然要这样做，反正棘手的是懂王。在俄乌问题上，民主党也持续发力，菲佐和乔治斯库被搞掉，就是发力的结果，带来的直接影响，就是俄乌停火的难度增加。懂王想停火，但阻碍不仅仅在于民主党，在于普京的要价，也在于欧盟。欧盟的保守派领导人，太少了，大多数是白左和建制派，这使得欧盟在整体步调上，是跟着民主党的，成为了停火的一大障碍。从获胜开始，懂王就在大力扶持欧盟内部的保守派。保守派往往也是亲俄派，比如就职典礼上，就邀请了梅洛尼等一大批欧洲的极右翼，马斯克更是亲自下场，扶持德国选择党和英国的改革党。不仅通过X给选择党导流，还要资助改革党一亿美元，等于向白左阵营全面开战。对比，民主党当然不甘示弱，于是先以极度不要脸的方式，做点乔治斯库，接着掀起街头暴力，强行瓦解了菲佐的执政联盟。目前来看，暂时是民主党占优。菲佐被搞掉后，下一个上台的斯洛伐克领导人，大概是要援乌的。事实上，除了斯洛伐克和罗马尼亚，民主党近期还在奥地利出手。去年年末的奥地利大选，第一名是极右的自由党，得票29%，另一个右派政党-人民党，本来已经同意和自由党联合组阁，但又诡异地推迟了，导致组阁迟迟无法进行，谁在发挥的作用并不难猜测。人民党内部有两派，一派赞成与自由党合作，另一派反对，有分裂的可能，如果民主党狠一点，让人民党分裂，自由党也将无法上台。只要无法在欧洲扶持更多的极右翼政府，懂王在西方阵营内部分，就会始终处于相对孤立的状态，一如上一个任期那样。到时候，不仅仅是俄乌停火，许多政策都会遭到掣肘，最终效果大打折扣。所以，尽管民主党这次大选输了，但依旧不能小瞧。
