
# 叫停外卖大战的真相：夫妻店快被打没了

之前京东入局外卖的时候，我就写了这篇文章《我希望东哥赢，但东哥很难赢》，里边已经把外卖这个行业分析得差不多了，看着好像挺赚钱，但整个产业链上依附着几千万人，这些人都背后又是一个个家庭，这么多人需要从这个产业里讨生活，最后被干成了一个薄利行业。

当时就提到了外卖行业的一个反常识的东西，就是外卖行业真正的护城河，恰好是这东西在形成巨大的规模效应之前根本不赚钱，有了巨大的规模效应后，依旧不咋赚钱。不仅某团不赚钱，其他那几个都不赚，比如饿了去年还在亏，亏了近一百亿。

当时我说东子最后大概率能切下一小部分市场份额，但不会太多，从现在的情况来看，这个分析和预测确实没啥问题。

不过东子在做外卖的过程中，意外发现一个新东西，那就是外卖是最高频的本地生活场景之一。高频=高用户粘性。做外卖这段时间，他的电商APP打开频率变高了。于是平台争夺的不再是单纯的外卖份额，而是用户手机上的“超级入口”地位。谁能占据这个入口，谁就能向用户交叉销售更多高利润服务（如到店餐饮、酒店旅游、金融、零售电商）。这是比外卖本身佣金更诱人的蛋糕。

这下各个平台格局打开了，于是又引发了一轮的“外卖大战”，本质也是争夺APP打开率的战争。也正因此，最积极参战的不是原本就做外卖的某团，而是试图寻找电商增量的某宝和某东。各方攻守都有其道理，都是规模巨大的平台，自然也就演变成了一场恶战。

但是到了最近，大家也都知道了，国家开始下场叫停了大战，我一开始还不太理解，毕竟这种让消费者拿好处的东西，为啥要叫停？

但是这两天接触了几个做餐饮的老板之后，看到了这场补贴大战的另一方面，也意识到了国家视野下，跟咱们看到的完全不一样。

首先第一个问题，消费者拿到了实惠，比如动辄1分钱的奶茶和其他外卖，那代价呢，代价平台和商家共同承担了。

比如一份30元外卖，消费者可能只需要支付12块，18块差额由平台和商家各承担一半，最终商家到手可能也就10块左右，去掉租金，原料，人工成本之后，赚不到钱是常态。

平台敢于出这个钱，主要也是对未来有信心，等切下来市场份额，再慢慢赚，本质也是用未来收益填现在的坑。这也是为啥三大平台的股价纷纷下跌，投资人都看清楚了，这三家花钱太猛了，未来预期被透支了。瑞银就认为，即时零售市场规模仅为电商的10%，带来的利润不值得巨头如此规模的商战。

而在商业逻辑之外，更该关注的是恶性竞争对民生和就业的伤害。这种赔本赚吆喝的背景下，那种大型连锁品牌还好，他们财大气粗实力雄厚，可以要求上游供应商共同承担成本，平台补贴也本就偏向大品牌，但那些中小企业就麻烦了，可能卖一单赔一单。

那你可能要说了，那商家可以不做线上啊？

问题就在这里，大规模补贴下，线下和线上差额太大了，以至于大家能线上消费就线上消费，甚至有极端操作，到了店也点外卖。以前线上线下价格相差没那么悬殊，社区老店还能靠堂食的口碑撑着不上平台，现在价格扭曲太严重，不得不认真考虑起外卖来。

这就陷入了一个无解的“囚徒困境”：做外卖吧，看着订单热闹，但增量不增利；不做外卖吧，门店堂食几乎没有任何顾客，消费者全被线上的低价补贴吸走了。

对于中小商家来说，做线上，是饮鸩止渴的“慢性失血”；不做线上，则是被流量抛弃的“急性休克”。更可怕的是，这场“慢性失血”竞赛中，最先倒下的往往是体量最小、抗风险能力最弱的夫妻档。

当然了，问题不仅于此。面对惨烈的竞争环境，商家总得活下去，总得想办法。大型连锁店自然有更多的选择，压上游，增加员工，引进效率更高的设备等等，中小商家怎么办？他们最常见的

选择就是降低餐饮品质，毕竟活下去才是唯一出路，都快活不下去了，啥事都能做出来。

于是这种惨烈的内卷，卷出来的可能不是品质，而是“成本狂魔”，比如大幅降低食材质量，甚至在外卖袋上做文章，用质量差有安全隐患的外卖包装，相当于说，最后代价又转回消费者那里去了，属于一种“劣币驱逐良币”。而那些倒闭的企业，主要是夫妻店和中小店，这些小微企业是社会就业的蓄水池和稳定器。

也就是说，这次外卖大战狂欢背后，其实是无数中小商家的哀嚎，而站位稍微高一些，把时间线拉长，就能发现这整体是个纯零和博弈，甚至是个“负和博弈”，对整体没啥意义，消费者占的那点便宜，背后是一堆商家的倒闭，平台补贴的那些钱，将来还要赚回去，对整个社会效益最大化一点影响都没。

甚至可能会进一步加剧通缩。一方面消费者意识到了物价可能还会更便宜，进一步加剧了通缩预期。另一方面大量中小商家倒闭带来的不仅是失业，更是小业主阶层财富的毁灭和创业信心的打击。这部分人群原本是消费的重要力量，他们的消费能力和意愿双双受挫，加剧了总需求不足。

这就属于系统性风险了，经济上行期这么做还凑合，现在再这么搞，肯定会被制止。

说到这里，又想起来一件事。几年前去日本，我就发现他们那边的夫妻档超级多。可能这些夫妻档小到只能容纳五六个客人，但是一家三代都在里面工作上班，每家都有点小特色，毕竟做的年头长。也正是靠着这种小特色，这种店铺能挺几十年，尽管赚得不多，但也能维持下去一家人的生计。

当时我对这种东西不屑一顾，最近突然反应过来，这玩意可能就是日本社会最重要的稳定器。比如很多年轻人根本不是个读书料，大学和公务员都考不上，除了去打螺丝或者送外卖也没啥好选择。在日本那边就直接回店里帮忙，将来直接继承店面，大富大贵指望不上，但也是个工作，而且相对来说稳定性强得多，可以一直干到老。

所以夫妻档这东西看着很low，但他们那点“小特色”，是对抗超级连锁的终极武器。尽管他们跟连锁餐饮价格差不多，甚至贵一丢丢，但是你去某个地方旅游，总不能继续吃麦当劳和吉野家吧？

如果没有大麻烦，大型连锁拿他们也没办法，长期看这些夫妻档才是社会真正的毛细血管，这些毛细血管生命力旺盛，整个社会才能健康发展。

这些年有个趋势，除了广东那边，这种夫妻档很难活下去，跟租金高、线上绞杀有很大关系。这些店大规模破产，从业者被扔到市场上，你们说说这些人能干啥？总不能所有人都去送外卖吧。

此外大量有特色但抗风险能力弱的小店倒闭，导致消费者能点的品类和口味趋于同质化，比如我这些年发现很多连锁烧烤店的味道都差不多。后来才知道，他们的上游供货商就那么几家（这几家供应链玩到了极致，最便宜），从肉到烧烤料，都基本上一样，味道可不就趋同了？

所以站在国家的层面，现在这种大战就是胡闹，为了一时虚假繁荣，你把毛细血管给搞死了，不如尽快别闹了。

很明显，健康的商业生态需要多样性（大中小商家并存、不同品类特色）。过度补贴大战加速了“赢家通吃”（大型连锁、平台），扼杀了中小特色商家生存空间，导致生态单一化、脆弱化。

餐饮行业尽管是个大红海，但它作为老百姓日常“衣食住行”中最重要的部分，一直都是个超级产业，我查了下数据，跟汽车产业比，餐饮收益规模确实比不上，但是从业者却比汽车多太多。

尤其是那些小店铺，长期来看是真正的“就业海绵”。

回到本文话题，这次外卖大战的残酷真相，就是各大平台在争夺流量入口的战争中，大量中小企业成批倒下。消费者短期看着得利，其实是一部分人已经付出了代价，长期所有人都要为此买单，妥妥的负和博弈，所以国家果断叫停。

接下来很长时间里，所有人都得思考一个话题，技术进步到底该怎么用于提升社会整体福祉，而不是让几家超级公司越来越大。比如现在完全可以

通过现有的直播技术，推行后厨透明化，平台想办法制定一套流程和标准，去帮助夫妻店解决食品安全问题，而不是自营取代它们。

这也是后续很长时间的一个主题，防止巨头们利用技术优势和成本优势不断侵占小企业的生存空间，最后对谁都不利，除了他们自己。
