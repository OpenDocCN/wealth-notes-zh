
# 岱岱-ZZ的艺术——成本转嫁论

关于内债和外债的辨别分析，曾引起美国国会的关注。世纪初，美国国会担忧国债不断攀升，要求华尔街金融家们解答。金融家们在国会科普，指出美国国债高企并非大问题，因为美国凭借全球领导力（即美元霸权的全球铸币权），欠中国、日本等国的国债实际上是不需要还的。这种观点让国会官老爷们意识到，外债并非绝对负担，关键在于国家是否有能力支付利息。

任何国家的债务都需要支付利息，只要经济增长速度能大于债务利息增长速度，就能通过信用背书不断举新债还旧账。解决债务问题的方法主要有三种：增发货币、征税还债、出售地方资产还债。

2008年后，中国经济增长模式转变为以软预算约束为基础，无节制地进行以基础设施为主的经济增长。这一阶段，银行贷款的60%投向了基础设施与房地产开发，制造业受到较大挤出效应。数据显示，日本和“欧猪五国”经历了痛苦的去杠杆过程，而中国非金融私人部门负债增加量是美国4倍左右，冠绝全球。

地方土地财政实际上是对居民部门的转移杠杆。楼市不仅是货币的蓄水池，更是另一个水龙头。在资产价格上涨的情况下，以按揭贷款形式交易会创造大量存款货币，推动房地产泡沫膨胀。楼市虽有部分“蓄水”作用，但放水的速度远比蓄水快，导致资产泡沫不断膨胀。

债务驱动经济的本质除了国内转移杠杆，还有国际成本转嫁。在金融资本主导的游戏规则下，强国吃弱国，国内上层吃下层。埃及学者最早提出成本转嫁理论，分析发达国家如何剥削后发国家，论证全球化经济秩序的不公平不平等。温铁军将这一理论移植到国内城乡二元结构，提出中国每次发展经济都是从三农提炼剩余价值，即成本转嫁。

成功的成本转嫁理论打通了国际国内的任督二脉。马克思的阶级论也是借鉴法国学者提出的阶级概念，推陈出新成自己学说中的核心地位。温铁军贯通埃及转嫁说，成一代大家，强调做学问不能闭门造车，要有国际视野。

介绍完债务经济和成本转嫁论，要聊到解决办法——财政。任何债务驱动经济体都急财政，财政是最重要的政治。历史上多的是因财政崩溃而崩溃的国家。清朝时期，尽管GDP全球第一，但财政收入远低于英国，主要依靠田赋收入，税基固定难以增加。而英国工业化程度高，税基多税率稳，财政收入比例远超清朝。

如今，全球国家基本都步入工业化现代化时代，财政收入占比在GDP的17%-22%之间。信息时代，大数据降低了征税成本，如金税四期，可以补上偷税漏税的部分。但财政收入占比并非越多越好，边际效益递减，过高的比例可能导致国富民穷。

美国也面临财政压力，近期国税局增添了新的条例，要求偷窃财产和非法活动所得必须上报。中美都是债务驱动经济体，财政都抓急。08年，中国选择了接盘发育的国运，但美国加息在即，有人仍要求中国继续接盘。中国只想和美国约炮，美国却想结婚，这种不对等关系需要警惕。

在2020年，明确表达了不与美国大放水承接泡沫的主张。美国是渣男本渣，拔屌无情，08年上过一次当，不能再次上当。中国需要自己努力攀科技树，即使争取到一段中美和平发展期，战略意义也不大。
