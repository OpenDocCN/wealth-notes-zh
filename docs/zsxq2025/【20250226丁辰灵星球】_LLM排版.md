# 隐喻故事疗愈

我刚发的这个下午的故事，给大家说下思维的过程。第一个骰子的图像是要隐喻整个主题，我忘记扔出什么了。第二个骰子到第五个分别对应一个完整故事的四个结构，即故事的背景，挑战，超越，和解决方案。

我扔出来的第一个图象，就是一个包子的形象，正好我的案主是个美女创业者，所以有了包子西施这个隐喻。

第二个图象是将军的头盔，所以挑战就是强权的压迫，不得不远走他乡。

第三个图像看起来像一碗面，所以我编了面馆的故事，这是整个故事我编的最长的，我属于一边想一边编，主要要编一个超越挑战的故事。

第四个图像看起来是一个刺猬，所以我给的解决方案是两家强强联手。刚才我根据记忆复述的故事漏了这个刺猬，当时讲的原话是，包子夫妻跟面馆老板诚恳寻求帮助和联姻时说了一句话，说我们自己这些年就像刺猬一样，对谁都扬着刺，现在发现没有必要，松下来做当下的抉择就好，选择比不选择好。这句话也对我的女案主有很强的隐喻疗愈效果。

大家可以试一试上面的思路来编故事，下午上课老师说，米尔顿艾瑞克森的疗愈故事为什么能有那么强大的疗愈效果，因为都是按照上面四个结构来的，最终人类的潜意识都能听得懂这些，所以能有疗愈效果，至于故事是怎么样随便编就好。

# NLP课程与隐喻故事

今天是NLP课最后一天，明天考试。今天下午的课题很有意思，即如何用隐喻讲一个故事来达到疗愈的效果。最难的是，老师给了一盒随机的骰子，即如果把故事分为五个结构来讲，每一部分编的故事要围绕着骰子上出现的意向来编，最后还要自治。

我的练习伙伴，一个女创业者，她给出的议题困扰是她有很多事不想做，明明觉得应该做，但她就不愿意做，会拖延或者忽视，结果带来很多新的问题。我就一边掷骰子一边编故事，如果你们看下面的文章，就会知道其中一个掷出来的是将军的头盔。（骰子每一面是一张图，不是数字）

整个都是没有提前准备的，就是一边掷一边编，最后同学说她感受到很大触动，让她的问题得到了一定的解决。她夸我的故事编的太好，有太多的隐喻，我第一次编，给大家分享下：

曾经有对夫妻在城里开了个包子铺，膝下养育了一个女儿。女儿逐渐亭亭玉立，出落的越发漂亮，在家的包子铺帮忙，得了一个包子西施的外号。全城的百姓乃至外地百姓为了看一眼包子西施，每天排着长队买包子。

提亲的人也多了，无论是街坊邻居，还是老夫妻的旧友同窗，甚至外地的商人，都被老夫妻以孩子还小拒绝了。他们觉得我女儿那么漂亮，不需要着急考虑这件事。更何况夫妻想，现在名声在外，又赚了那么多钱，普通人家哪里配得上。

有一天城里守卫的将军也听说了包子西施的美貌，于是托媒人上门提亲。

包子西施坚决的和父母说no，她有一段不好的回忆。十几年前国家发生了政变，一群武人杀来杀去，血流成河，你方唱罢我登场。这十年好不容易安稳下来，但上层斗争的依然很残酷。

包子西施觉得将军这种武人是高危职业，既可能动不动被其他将军取代灭族，也可能喜新厌旧把自己打入冷宫。她父母也是同样的想法，于是坚定的和提亲的说NO

几天之后的夜里，突然来了一伙蒙面强盗，他们打着火把砸了包子铺，抢走了很多存银，街坊邻居出来观看被警告不要多管闲事。还在门上贴了告示警告，让他们另谋生路，再继续开包子铺就会有生命危险。

老两口和包子西施猜测是拒绝了将军，被报复的结果。但他们没有证据，也不敢不听话继续开包子铺。而且他们也知道就算继续开，百姓们知道他们得罪了将军，恐怕也不敢再来了。

他们商议一下，和女儿提出要不离开避避风头。收拾细软，三人一路餐风露宿，包子西施也不免风吹日晒，容颜削减。

有一天来到一个城市，看到一个面馆，生意特别好，客人都在排队，但井然有序门口有个黝黑矮墩年轻男子，看到包子西施目不转睛盯了两眼移开，然后喊，太师面咯太师面，客官来吃完面吧。

一家三口坐下吃面，吃完后找老板结账。老板说你们看起来是外乡人，来这里走亲访友吗?于是一家三口愤愤的把自己家如何得罪权贵将军的故事说了一遍。

面店老板说：原来你们就是隔壁城市的包子西施啊，你们的故事我听客人说过。

这时候又有客人上门，一个长相普通皮肤黝黑的姐儿送面出来，老板说这是我女儿，在京师大学堂读书，最近学堂放假她回家，顺便来帮忙。

这时隔壁桌客人吃完离开，立刻有一个身材高大，长相普通的斯斯文文的少年来收拾桌子，刚送面的老板女儿也过来帮忙。很快桌子就收拾干净整洁，新的客人坐下。

老板说：这是我女儿的未婚夫，也是她京师大学堂的同学。小伙子没什么背景，当初读书村里老人凑了一些碎银子，还有几十个鸡蛋让他上京考的试。

然后指着门口那个揽客的矮墩男孩说，那是我儿子，这孩子不像他姐姐那么会读书，个头也不高，但他看了谁也不怵，看了谁都能吆喝进来吃碗面。

包子夫妻说，您这面做的也不错，但也谈不上多么惊世骇俗，生意怎么会那么好?您是什么时候开始卖面的?

面馆夫妻问，那之前您是做什么的?

面馆老板笑了一笑，指了指招牌，你没看到上面写了太师面吗?

“您原来是太师府的厨子?”包子夫妻问。

“不是，我是太师府的帐房管家，所有的账都是我在管，当然太师府的面肯定得天天吃啊。”

包子夫妻震惊了，太师是一人之下万人之上，太师府的管家是太师府的一人之上万人之下，怎么来做面了?

面馆老板说，十多年前的兵变改变了一切。换了新朝，为了生存，为了养孩子，我就去街头卖面了。

包子夫妻很惊讶，问道：“虽然换了新朝，但您是大帐房大管家啊，难道不能去其他的王爷将军或者商号继续当管家吗?”

面馆老板笑了一笑：“谁会请你这个前朝余孽?更何况，我过去一年工酬8000两银子，我就算愿意屈就找一个800两一年的糊口工作，人家商号也不信任我，觉得我一定会贪污。”

“再者，新朝新气象，京师大学堂持续扩大招生，年轻人不想从武的都去学算账商业，现在外面一个账房别说800两，给个400两也就不错了。

面馆老板继续说： “我若当初不甘太师府大管家的位子，还是去钻营各大将军幕府，恐怕早就被人挤了下来，甚至命也未必保得住。现在，太师面太师面，人人都觉得我这面是太师府的口味，生意兴隆，我这不也一年挣八千两吗?

最重要的是，我两个孩子成长了。我从太师傅出来时，两个孩子突然从锦衣玉食到流落街头。

但我挑着面担的时候，女儿儿子都跟着一起吆喝。

刚开始很多客人也是觉得娃可怜就来买面。

有些熟人刚开始看我笑话，天天茶馆酒肆说我去街头卖面，结果适得其反，更多人过来吃面了。

包子夫妻问：“那这些老熟人现在看你做大了，是不是后悔了经常来吃面?”

面馆老板说：“十年生死两茫茫，这将军幕府权力斗争激烈，很多人都不在了，有的杀头了，有的倒了，有的混不下去又怕人笑话，灰溜溜回乡下下了。总之吧，很多人都不见了。

包子夫妻继续问：“那以后呢?是不是打算继续扩大面馆，或者去我那个城市也开面馆开成连锁?”

面馆老板反问：“你认为那有意义吗?就像你的包子铺，离了你女儿这个包子西施还能有那么多客人吗?”

面板老板继续说：“你看我这十几年下来，女儿聪明好学，从小发誓要重振家业，她就去京师大学堂。她虽相貌普通，从不缺王公贵胄追求，她却选了这个村里出身的同学。她说这些王公贵胄只有资源，却没有脑子，既然是同学，以后做什么事用他们资源即好，何必嫁为人妻被他们控制?何况三十年河东三十年河西，以后谁当王公贵族还不一定呢?

儿子就不同了，他虽然资质平平，但他挺享受吆喝的。能吆喝，我这太师面终究是有传人了。”

包子夫妻大悟，他们说：“我们若在旁边租个铺子重开包子铺，凭我女儿姿色，到时必然传遍王国。届时整条街都会成为王国最繁华的街道。”

面馆老板笑问：“贤夫妻若囊中羞涩，老夫可相助一把。”

包子夫妻说：“听君一席话胜读十年书，若蒙不弃，我们结个亲家如何?”

面馆老板说：“这倒未必需要强求，我这笨儿子，你这包子西施的闺女未必看得上。要看年轻人自己的想法。”

包子西施经此人生大变，突然转念，过去看谁都看不上，现在突然觉得安住当下才是最好的幸福。

而面馆家的傻儿子，又哪有不愿意的道理。

从此太师面和包子西施铺缔结新的故事，果然传遍了王国，这条街成为王国最火爆的街道。
