
# 从央行货币委最新例会，教你学会看货币政策

## 央行三季度例会的新信号

与二季度相比，这次的例会通稿，删去了“风险隐患较多”的表述，暗示着之前一系列防风险措施已经看见成效，未来可以将更多精力与政策资源，从“防风险”转向“稳增长”这一核心任务。

对物价的描述，从二季度的“持续低位运行”，变成了这次的“低位运行”，删去了“持续”两字，意味着物价压力和通缩风险，是有所缓解的。

强调存量政策的落实。2024年底，中央经济工作会议提出了“适时降准降息”。随后，2025年央行一季度例会，明确为“择机降准降息”，释放了强烈的行动信号；二季度降准降息迅速落地后，例会表述随即调整为“灵活把握政策实施的力度和节奏”。

基于以上定调与变化，我综合分析四季度将处于一种“降准概率高，降息还需要契机”的态势。不着急降息，我认为主要面临多重约束：首先是银行净息差，当前的情况下，如果进一步降息会让这个净息差持续收窄，将冲击银行盈利与金融稳定；二是7天逆回购利率，它是银行间市场上的短期借贷利率，这个利率当前只有大概1.4%，下降的空间相对较小了，还需要为未来的不确定性预留弹药；三是上半年经济增速已经达到5.3%，全年目标压力不大，并没有触发“紧急降息”的条件。

相比之下，我分析降准则还存在空间，并且比较容易“一举多得”，比如能释放长期流动性，且不直接冲击银行息差；再比如能有效支持财政发力；还比如目前大型银行7.5%的存款准备金率，距离5%的“隐性底线”仍有2.5个百分点的充足下调空间。

因此，这样你就能理解为什么9月没有降息，我认为就当前形势来说，四季度降准更可能是一个优选项。

## 看不同层级会议的方法

作为顶层设计的经济层面的中央会议，每年最重要的，是一般在12月召开的中央经济工作会议。

具体到看中央经济工作会议的货币政策信号，你需要关注三个关键：

1. 对经济形势和工作要求的关键表述，比如类似“三重压力”，也就是需求收缩、供给冲击、预期转弱，是否被提及？是更强调“稳字当头”，还是更突出“稳中求进”？它们定义了政策的紧迫性，以及是防风险多一些，还是促增长多一些。
2. 直接的货币政策定调。例如，2023年的中央经济工作会议，关于货币政策的表述是“稳健的货币政策要灵活适度、精准有效”，“稳健”是基调。而2024年的中央经济工作会议特别提出，“要实施适度宽松的货币政策”，“适度宽松”是基调，并且直接提到要“适时降准降息”，提到“货币供应量增长同经济增长、价格总水平预期目标相匹配”，我理解这些就是强烈的加码货币政策的信号。
3. 重点工作任务。中央经济工作会议会部署下一年的重点工作，而货币政策作为服务整体战略的工具，就是围绕着重点工作来的，比如支持科技创新、绿色转型、普惠小微、数字经济等等。

其次，在中央经济工作会议的顶层设计之下，有三个关键会议，则负责将宏观战略转化为具体政策路径，并协调各部门之间的行动，以及进行对外的信号释放。分别是，中央政治局会议，国务院常务会议，和前面谈的央行货币政策委员会例会。

这里面最重要的，就是与经济有关的中央政治局会议，通常在每年的4月、7月、12月召开，少量时候在10月也会有，去年更特殊，提前到了9月。中央政治局会议，会更具时效性和针对性。

看中央政治局会议，一方面你需要关注它对当前经济的关键提法，比如是“稳中向好”，还是“稳中有变”，或者“稳中有进”，有没有“压力加大”，要不要强化“底线思维”，既定政策要不要“早出台早见效”，不需要“加强超常规逆周期调节”等等，尤其“超常规逆周期调节”，我分析往往就是加大财政和货币政策力度的信号。
