
# 深度交易通胀



黄金的突飞猛进，为白银留下了上涨空间。可以考
虑一下白银类投资标的。
川马新政对美元的意义，是否负面呢(强美元可以
持续吗)?其实，真正的减税降息并未开始，激进
的内政外交政策已经构成对强美元的冲击。强美元
已经是强弩之末了。
强美元的侧面就是通胀。西方对俄罗斯的制裁不会
立刻结束。贸易战当然不会立刻结束。而新的中东
战争已经箭在弦上。我认为，通胀才刚刚开始。是
否有恶性通胀，有待观察波斯湾油路。
我一直在思考：应该深度交易通胀，还是继续交易
AI泡沫。我趋向于保守，深度交易通胀。AI形成
高水平盈利还需要一个漫长的历史过程。人工智能
在现实中的运用，仍然处于试错阶段。因此，仍然
持盈保泰。
美在俄乌战争中的交易策略相当露骨。乌克兰成为
美国交易中东利益的筹码。甚至，整个欧洲都变成
了中东利益交易的筹码。俄罗斯会因为乌克兰而出
卖伊朗吗?我们拭目以待吧。
我国已经意识到，中美博弈不在乎外交、军事、贸
易、科技，唯金融(全球资本三流)而已。美刻意
阻止全球资本回流中国，但美元的质化紧缩和量化
紧缩皆已到达极限。我方处置得当，已日有孚。
我无意于提供具体建议。唯巳日有孚，可以考虑短
股的操作。操作逻辑不是 而是侧重于交易通胀。
因为，我担心大洋彼岸AI泡沫破裂。
至于何时结束长金，要看汇率能否恢复升值趋势。
不要太着急，稳住了。机会永远在。
一句话：继续坚持按既定方针办。
