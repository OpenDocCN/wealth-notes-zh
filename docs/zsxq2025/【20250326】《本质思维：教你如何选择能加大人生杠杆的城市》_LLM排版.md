
# 本质思维：教你如何选择能加大人生杠杆的城市

我们之前谈了，长三角、大湾区和成渝的战略关键和发展趋势。这是一块很重要的拼图，拿好这块拼图，我们才能更好地选择城市。后来有同学说，能不能给出些更高维的挑选区域或者城市的方法？好，今天我就来回答这个问题。

选城市这件事，到底有多重要呢？我们先达成一个共识：就是选择一座城市，就是选择我们的“人生杠杆”。

为什么这么说？你选了一座城市，在这里工作、买房、结婚、生子，其实就是把所有资产重仓在了这座城市，也其实就是买了一份“城市期权”，等着有朝一日能变现，给你惊喜。

如果这座城市的产业能持续扩张、基础设施能持续完善、公共服务能持续优化，那么你的资产价值，也会跟着水涨船高，你的选择才能回本，甚至获得几倍的增量。

所以，选择城市很重要，甚至可能超过了人生的其他选择，因为你的工作和另一半，也都是跟这个城市绑定的啊。

那么，选城市有什么方法吗？划重点了，就是 未来选城市，要选那些正在“从 1 迈向 10”的高成长性城市。

说白了，就是要选增量城市，找面向未来的、有潜力的、能给你带来几倍杠杆的城市。

增量城市的两大特征

我先讲讲增量城市的大致特征，至少有这么两个特点：

第一，是产业集聚带来高端产业的涌现。前面提到过，产业就是税收啊。战略性新兴产业就是“新税收”，未来产业就是“未来税收”，这直接关系到一座城市的子弹够不够。

第二，是人口集聚带动高端人才的涌入。人口既是消费力、也是生产力，而高端人才，就是高端消费力和高端生产力。一座城市的高端人才越多，服务这些高端人才的行业，也会发展得更好，做配套、做服务的人，也更容易赚到钱。

而产业和人口是城市跃进的两个飞轮，这两件事做好了，什么公共服务、交通物流，都是水到渠成、自然而然的事，你在这座城市的资产才可能升值，比如你的工资、你的房产、你孩子的教育等。

那么，这些增量城市就只是北上广深这些顶级大城市吗？不一定，甚至很多时候不是。

过去，我们都争着涌向北京、上海、深圳，因为那个时候，这些城市就是全国最大的增量机会。但现在，时代变了，这些城市已经是“带头大哥”了，他们最终只能变成更厉害的“带头大哥”，向上的增长空间很有有限。但跟在“大哥”后面的小弟们，可还在暗自较着劲，还有很大的进步空间，这些城市更值得关注。

也就是说，我们要重点关注一个城市未来创造新财富的能力，而不是行政级别或者现在的规模。当然，除了找增量城市，选城市时还要注意规避一类“收缩城市”。这些城市曾经家大业大，但现在正在走下坡路，具体我就不举例了，你估计有感知。对于这一类城市，不管现在规模多大，都是“天坑”，因为没有未来啊。这类城市，在西部、中部、东北和华北，都有分布，能离开的，尽量离开。

总之一句话，不要迷恋曾经辉煌的“绩优股”，要选择高成长性的“潜力股”。尤其是对于年轻人来说，因为很多大城市的财富增量，都被比你更高阶的人分走了，他们都分不过来，很轮得到你啊。所以你要去年轻人能分到新财富的新型高增长城市。

讲完增量城市的两大特征，接下来，我来讲更具体的“操作方法”：怎么找到增量城市？怎么确定这个城市是“真的”增量，还是“假的”增量？

在此之前，我还是要先提醒你一件事：这一轮城市格局和利益格局的巨变，的确是整个社会系统与经济系统的再造。但这一轮系统再造的强度，要稍稍弱于上一轮的城市崛起，这不是“水大鱼大”的大时代机遇，而是“池小鱼大”的新红利机会。换句话说，这不是大多数人的红利，而是少数者的机会。所以，我们要尽快找增量、抓机会，因为这样的机会不等人。

回到正题，城市的增量主要分成两类：一是，市场经济的增量，直接由经济规律决定；二是，国家意志的增量，这是由政府政策主导的。

找市场经济增量

先来看，市场经济的增量。这可以再细分为：人口的机会和产业的机会。

人口机会很好理解，它和城市的人口上限密切相关。比如上海，人口上限是 2500 万，但现在已经有了 2487 万，未来十年的增量空间不足 20 万。北京也是如此，人口上限是 2200 万，也要控制人口规模。深圳也面临一样的问题。

现在的问题是，这些城市还要继续发展新产业，还要引入新的人才，所以人口外溢就自然发生了，这是经济规律决定的。在交通一体化实现后，这些大城市周边的宜居城市，就是所谓的经济圈城市或者卫星城，也迎来了新的发展机会，这就是人口外溢带来的增量机会。

所以，如果你去不了大城市，就要找能承接大城市外溢人口的新兴城市。这方面，长三角做得很好，珠三角也还行，京津冀相对差点意思。

这里我再举一个反例，成渝周边也有所谓的宜居城市，但这些城市有类似的增量机会吗？大概率没有。因为重庆的人口上限是 3600 万，成都的人口上限是 2350 万，加起来接近 6000 万，空间还很大。这就意味着，成渝不是要人口外溢，而是要继续虹吸。更何况，成都还是全国唯一定位“公园城市”的超大城市。说白了，成都自己就是个“特大号”的宜居城市。

所以，如果你选择了成渝周边的二线城市，做所谓宜居城市，大概率等不来增量机会。

至于产业机会，是指在培育城市群和都市圈的过程中，一些城市会吃到产业链外移的双重红利，最典型的例子是江苏常州。虽然常州在当时的千亿城市中，面积和人口都是倒数，但它找到了一条新路，就是成为新能源车产业的配套城市。2023 年，常州成功跻身“万亿城市”，这就是产业协同带来的增量机会。

另外，因为我们上面提到的人口外溢和产业协同，也因为未来区域竞争，开始转向都市圈竞争，所以传统的“单个城市思维”已经过时了，我们需要从“都市圈思维”来重新理解区域的巨变。说白了，就是你要把城市放在一个更大的坐标系里去看，不是就城市本身看城市，而是就都市圈甚至全球产业链来看城市。至于都市圈思维下，哪些传统观念可能不再重要？我们在之后的课程中，会有更详细的解读。

找国家政策增量

说完了城市中市场经济的增量，再来看政策增量，也可以分为两类：

第一类，是新推出的产业政策。看产业政策的增量，有三个层次：第一，看有没有资金支持？第二，看资金支持的规模有多大？第三，看资金规模比其他城市多了多少？前两个，我之前谈了不少，今天重点看第三个层次。各个地方出台产业政策，相当于我们参加高考，资金规模相当于高考分数。那问题来了，高考的分数重要吗？不重要，重要的是排名。一样的道理，产业政策能不能真正奏效，也得看资金规模能比其他城市高出多少。

第二类，更关键的，看这个政策有没有独占性。举个例子，1980 年，深圳第一批获批成为“经济特区”。在这之前，没有一个城市是“经济特区”，这就是政策的独占性。只有我
有，别人没有，抢先占了“天时”。

所以，判断一个政策是不是增量政策，你要关注这些关键词：比如“首次提出”、“全国唯一”、“创新性政策”等等。我之前提到过，四川是全国唯一的国家级战略腹地。你看，这个分量有多重，这就是成渝经济圈最大的增量机会，只有它有，别人都没有。

当然了，听到这里，可能还有人不甘心。说马老师，那大城市不是也有新的市场增量、政策增量吗？当然有，我同意这个说法，但要特别提醒的是，这很可能不是普通人的增量。

很简单，北上广深都在进行大规模城市更新，但你想享受到这个政策红利，得先占个“坑”、买个房吧？你
看，享受大城市的政策红利，是需要付出代价的、是有成本的。比如北京和成都，成都和佛山，在上车成本上，是不是有巨大区别？巨大的上车成本，是不是就把这些可能的机会留给了更富有的中年人，而不是刚毕业的年轻人？所以我上面说了，有些大城市的增量财富，是给更强大阶层的增量，不是属于年轻人的。

继续推导，这些大城市同样都在扶持新兴产业，但是要求极高，因为这些城市需要的，是全国乃至全球最顶尖的人才，并且聚焦在少数产业上。想上这艘大船，你就必须具备这些城市需要的高技能、高竞争力，以及特别能拼抢能卷的能力。所以这些新增财富，背后都标好了价码的。简单说，你不能是个普通人。相对来说，我更推荐去的，是竞争没那么激烈的，对年轻人更友好的二线增量城市。
