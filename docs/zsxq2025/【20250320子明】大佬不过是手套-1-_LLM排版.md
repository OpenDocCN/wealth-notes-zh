
# 大佬不过是手套

《大佬不过是天龙人的手套》
(潘石屹=潘SHIYI)
源自3月19日群聊
发布于2025年3月20日

无论是李JC还是贝莱德，
别看那么多的光环，
实际上都是给天龙人打工的手套机构，
类似于贝莱德之前收购潘SHIYI的兔子资产，
其实跟贝莱德和潘 都没有关系，
只是两个代持人交换 一 下资产的持有手续。

实操过程，
类似于大连徐明奉命买下 一 处昂贵的海外别墅，
他既不会问价格，也不会问用途。
更不会问别墅里面会住谁，
他只会老老实实的交钱并完成这笔交易，
至于这套别墅是否有其他卖家出价或者看好，
就根本不是他考虑的因素。

同样，正如之前中央通过查税等手段，
阻挠了贝莱德收购潘 资产，
他没这个资格被中央针对，
也不是这个资产不能卖，
中央的行动也是针对白手套潘SHIYI身后的天龙 人 。

潘 卖兔子资产可能不请示自己的老板么?
这种巨额资产的撤离，
本就是一个政治山头的系统性决策，
代表的是一个利益集团的态度，
同样，李JC可能不打招呼就卖港口么?
他肯定已经事先请示了自己的东家，
而他的东家此事上没有跟我们勾兑，
因此，我们的愤怒并不是来源于李JC这个手套，
而是在于李JC背后的天龙人们跟我们离心离德。

想要进一步理解，可以借鉴潘 的时间线，
2020年10月，秘书董 落马
2021年6月，潘SHIYI抛售SOHO 兔子给贝莱德
2021年9月，贝莱德交易未通过兔子政府审查
2022年4月，秘书田huiyu落马，并以此为标志，
掀起来轰轰烈烈的金融系统反腐。
就像潘SHIYI的SOH0 最后还是会卖掉，
李JC的港口销售最终也阻挡不住，
审查更多是在进行对背后天龙人的施压，
而这背后实际上是天龙人之间的分分合合，
老百姓加入个人情绪其实没啥大意义。
天龙人也是人，
他们也会很小家子气的相互攻讦，
而天龙人手套的李 也会因此遭受到无妄之灾，
他剩余代持的兔子资产也会跟SOHO 兔子一样折价出售。
