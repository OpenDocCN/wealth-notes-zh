
# 卢克文8.29：参加9.3阅兵的名单公布，里面有什么玄机？

8月28日上午，官方终于公布了来参加9.3阅兵的外国首脑名单。说实话，这份名单，既有意料之中，也有意料之外，更传达出了不少有意思的信息。

## 名单里面，到底有什么玄机呢？

### 1. 普京与金将军的出席

名单上，一共有26个国家元首，最重磅的来宾是普京。但因为普京早就官宣，大家并不意外。真正意外的，是金将军。理论上，邀请将军没毛病，因为将军的爷爷金日成，在东北抗联当过师长，两国的抗日史是有交集的。邀请当年抗日战友的后辈来参加阅兵，从道义上是完全站得住脚的。

但中朝关系比较微妙，两国存在分歧，所以2015年阅兵时，将军没有来，而是派了时任劳动党中央书记的崔龙海来。相反，韩国方面倒是来了总统朴槿惠。有了上次的先例，很多人都猜测这次李在明会继承这个传统。万万没想到的是，李在明竟然给推辞了，朝鲜来的人变成了将军，这是1959年以来，朝鲜领导人第一次来北京观看阅兵。而且，我们把将军排到了第二位，仅次于普京。这也就意味着，这次将是普京和将军二人，一左一右站到中国领导人身边，这对从没参加过这么大规模多边外交活动的将军来说，可以说是极大的礼遇了。

### 2. 将军北京之行的意图

平时基本不出访的将军，这次这么痛快就来观礼了。军队刚刚在库尔斯克打了胜仗，这场战争证明了军队的强大战斗力，让世界对这个封闭、贫穷、落后的国家刮目相看，给将军提供了亮相国际舞台的底气。将军也可以趁着这个热乎劲，来北京进行一次外交突围。

不难看出，将军此次北京之行主要有三个意图：

**第一，表达中国抗战胜利的共同历史的承认和尊敬。**半岛从日本统治下解放，是半岛历史上的重大历史事件，谁一直坚持着抵抗最后赶走日本侵略者，谁在法统上就拥有正统性。将军出席中国的阅兵式，是借承认中国抗战胜利的历史，强调自己父辈主导的半岛抗日史，进而强调法统正义性。相比刚刚访问了日本、对日本侵略历史态度暧昧的李在明，起码将军更加尊重历史。

**第二，打破外界对中朝关系的猜疑。**近年来，西方世界一直在刻意挑拨中朝关系，特别是在朝鲜出兵援俄之后，把朝俄合作炒作成半岛对华疏远，目的就是制造一种中国“众叛亲离”的虚假叙事。现在，无论对朝鲜来说还是对中国来说，胜利80周年不仅是一个历史纪念日，更是传递团结信号的重要时刻。将军参加阅兵，也有助于打消这种谣言，告诉世界朝鲜和中国仍然是拥有共同防卫义务的血盟。

**第三，将军与中俄站在一起，本身就具备一种象征性意义。**要知道，近年来在美国的放纵之下，日本方面正在试图“重新解构侵略战争”，进而摆脱战败国身份，恢复对外动武的国家职能。而美国一边纵容日本(哪怕在今年日本投降纪念日上，美国也只是模糊地宣布“美国和日本结束了太平洋地区一场毁灭性的战争”),另一边也在不断逼印太国家站队，企图组建一个以日本为核心基地的亚洲版北约。这无论对中国、俄罗斯还是朝鲜，都形成了威胁。三国领导人并排站立，一方面是提醒亚洲其他国家不要忘记被日本侵略的历史，另一方面也向外传达出一种中俄朝一致对外的信号，毕竟，三国有够打世界大战的军工产能，有几乎用不完的能源和矿产，还有打仗不要命的军人，三国加起来，足够让所有有军事冒险企图的国家三思后行。特别是力量相对弱小的朝鲜，将军往天安门城楼上一站，就已经相当于获得了中俄的安全背书了。

### 3. 敏昂莱与伊朗总统的出席

除了普京和将军，还有几个意外的重磅来宾，就是大家一直忽视，但对中国非常重要的缅甸代总统敏昂莱和伊朗总统佩泽希齐扬。

和将军一样，敏昂莱也一直被西方社会敌视和制裁，但这中国并不在乎，中国只在乎敏昂莱能给中国带来什么。中国对缅甸的诉求很简单：

**第一，西南你要给我搞个出海口；**第二，不能和中国唱反调；第三，要稳定，不要影响我做生意。这才是中国对缅甸一切工作的出发点。在这个出发点的指导下，中国只能支持敏昂莱。因为缅北那帮只知道贩毒和搞电诈的民地武根本成不了事，也无法“小族临大国”来稳定治理以缅族为主体的缅甸。

所以我们看到，自从去年敏昂莱访华，全面配合皎漂港和中缅铁路建设后，中国对敏昂莱的态度就大变，先是说服了果敢同盟军与缅军停火，最近更是施压佤邦，停止了对德昂军的军火供应。这样一来，敏昂莱才稍微缓过来气，慢慢稳住局势。稳住局势后，敏昂莱能安心筹备今年12月底的大选了。在这个关键时候，敏昂莱必须要给中国面子，中国一邀请就赶紧来了。毕竟，能不能把自己这个“代总统”的“代”字去掉，就看中国的态度了。

伊朗总统佩泽希齐扬的出席也值得关注。咱们都知道，今年伊朗和以色列的冲突中，伊朗吃了大亏，不仅防空体系被彻底摧毁，大量高级军官被斩首，就连核设施也被美国的轰炸机给扬了。军事上的惨败，必然催生伊朗政治风向的转变，哪怕佩泽希齐扬这个一向以亲美为标签的总统，也开始不得不寻求维持政权延续的路径了。所以7月份以来，我们听到了很多伊朗向中国靠拢的消息，包括将卫星定位系统全面接入北斗，比如向中国采购红旗9防空导弹，甚至传言伊朗要采购歼10战斗机。显然，伊朗这是痛定思痛，想以中国标准来重建自己的国防体系了。9月1日，佩泽希齐扬要来参加上合会议，显然也是在寻求新的安全保护。既然上合会议都参加了，3日来看个阅兵也是顺理成章。相信眼前驶过的钢铁洪流，飞过的五代机，会让佩泽希齐扬认清现实的。

### 4. 名单分析

说实话，这次的元首名单亮相之后，网上有不少人说怪话。有的人说，这次来了26个国家元首，比2015年的30个少了4个。有的人说，这次来的都是“穷兄弟”,有钱的国家一个都没来。还有人说，这次西方国家集体缺席，特别是原本寄予厚望的特朗普没来，说明中国已经孤立于“主流世界”之外了。

其实吧，大可不必在意这些杂音。如果嫌参加的国家少的话，请大家想一想，以中国如今的国际地位，如果单纯为了面子，搞五六十个元首过来，是难事么？并不是。别的不说，中国一带一路那么多国家，只要延期部分债务，就有一票非洲国家元首愿意过来？为啥中国没这么干呢？没必要啊。家里办喜事，肯定要邀请和自己关系最好、利益关联最深的亲友，关系不好到那个份上，也真没必要让他们上桌，讲究一个亲疏有别。

毕竟，现在是2025年，不是2008年，中国没必要像当年那样追求万国来朝了，来的国家不求多，但求真正有含金量。不信我们可以看看这次来的国家。

东盟国家，除了正在抽风的菲律宾和总理被停职的泰国，基本都来了(新加坡派的是副总理)，这说明中国作为东盟国家最大的贸易伙伴，在东盟拥有巨大影响力。还有中亚五国，是中国一带一路的核心，中国新的市场和能源供给渠道，中亚五国一个不少都来了。社会主义兄弟，朝老古越，一个不少都来了。非洲国家，没有大面积邀请，只邀请了刚果(布)、津巴布韦作为代表就行了，体现了中国在非洲布局的关键节点。还有打得不可开交的阿塞拜疆和亚美尼亚，两个国家的元首竟然都来了，这说明人家相信中国的实力，愿意接受中国的调解。更不要说尼泊尔了，夹在中印之间的尼泊尔，历来唯印度马首是瞻，这次呢？莫迪不来，尼泊尔总理奥利却不考虑印度感受，选择来了，还不能说明问题么？

有人可能要说，欧洲国家怎么不来呢？其实吧，也不是不来，你看塞尔维亚总统武契奇、斯洛伐克总理菲佐，不是来了么？甚至就连日本前首相鸠山由纪夫，都以个人身份来参加阅兵式了，还不够有分量吗？总不能指望石破茂来吧？

至于有些人寄希望于的“西方世界”派人参加，或者嫌弃来的国家都是小国穷国，其实说到底，只是殖人思维罢了，总是把“来自西方的认可”作为判断事务的标准。

我们这是什么活动？是纪念抗战胜利80周年和世界反法西斯战争胜利80周年。看看那些西方国家吧。法国早早投降了，德国和意大利本身就是前法西斯国家，奥地利、捷克当年就是德国的一部分，还有丹麦、荷兰、比利时、保加利亚、罗马尼亚、匈牙利后来都成了德国的仆从国，还有波兰和波罗的海三傻，也属于德国总督府。这些国家，有什么资格庆祝反法西斯战争胜利80周年？中国就算邀请他们，他们好意思来么？相反，有资格纪念胜利的是谁？是牺牲了2700万人的俄罗斯(苏联), 还有坚持敌后作战的塞尔维亚，以及和中国人民并肩作战的朝鲜，这些国家来参加阅兵，本身就已经是对历史最好的回应，有毛病么？

再说了，疾风知劲草，板荡识诚臣，面对美国的施压，面对日本560亿的收买，这些国家在关键的时刻出现在北京和你肩并肩，本身就已经说明人家的立场了，你再嫌弃人家穷，讲道理么？这不是中国作为一个大国的格局。

就算来了几个西方国家，又怎么样呢？在意识形态有根本矛盾、产业结构存在竞争的大前提下，又能怎么样呢？人家就真的跟你一心么？你看，前几年西方国家印度如胶似漆的，现在不是照样翻脸了？

至于很多人寄予厚望的特朗普来参加阅兵，没来也没关系。毕竟特朗普这个人太难预测了！谁知道他来参加阅兵，能整出什么活来？此前就有传言，说美国国务院向中国提出了特朗普有意来北京参加阅兵，但要求安排特朗普讲话的环节……这不是搞笑么？这是中国的阅兵式，安排个特朗普的讲话算怎么回事？这么看来，中国也没必要非让特朗普出席阅兵式。中国人民的胜利，是中国人民用十四年的斗争，以无数牺牲换来的，苍天可鉴，日月可表，从来都不需要什么美国总统的“认证”。

谁来了，谁没来，没必要患得患失，中国既然走上了现在这条路，就要有被西方国家孤立和敌视的勇气。而我们，只需要静待9月3日的到来就好，相信这次阅兵，一定会震撼世界。
