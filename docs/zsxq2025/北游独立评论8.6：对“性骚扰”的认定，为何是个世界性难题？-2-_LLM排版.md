
# 北游独立评论8.6：对“性骚扰”的认定，为何是个世界性难题？

原创：我是北游 公众号：北游独立评论 25年8月6日

## 如何做到，即保护自己，又不冤枉别人?

这段时间，关于“性骚扰”的话题频繁引爆网络，因为指控性骚扰而最终败诉的案件，在最近几年频繁上演。近的有武大图书馆案，远的有阿里案，霍尊案，编剧史航案，某军案，东哥案……都曾轰动一时，涉及其中的当事人，即便胜诉，都付出了惨痛的代价。有些当事人至今都还在被小作文掀起的舆论所追剿，为什么？原因就在于，男女之间的情感，以及与“性”相关的言行的界定原本就是一个玄学，取证和界定极难，很大程度上是当事人自由心证，一旦当事人的主观和感受上出现了与事实不符的偏差，就会导致冤假错案的发生。

## 那么,真正的问题来了：

在“疑似性骚扰事件”中，当事人应该如何做到既保护自己，又不冤枉对方？

老朋友知道我的习惯，我向来对具体的事件没有兴趣，我从来不自认法官，也没兴趣网络审案。我聊具体事件或者案件都不是对事件本身感兴趣，而是关注事件背后的本质，那些具有普遍性的东西，只有普遍成立的观点才是真正有价值的部分。很多人一碰到男女之事就容易炸锅，尤其是动辄煽动男女对立的拳师，当一个人，乃至大多数人都情绪激动的时候，你跟他们根本没办法聊。这年头，阶层对立、性别对立是事实上的政治正确。所以，不少人盯着这碗饭吃，只要是挑动阶层对立，挑动男女撕逼的个人账号，那都是赚的盆满钵满。虽然不少账号因为过于偏激，求锤得锤了。虽然不少人开始厌倦了，他们厌倦了那些夸张叙事的文字，开始越来越多的说“让子弹飞一会儿”的话，而不会轻易盲从了。但是，男女话题依然很难不陷入自说自话的偏激状态，究其原因，就在于男女之间的事，基本属于玄学。

无论是霍尊事件、阿里事件、钱枫事件，还是以前的某东哥事件，哪一件没有出现反转或疑似反转的真相？事到如今，你搞明白这些事的真相到底是什么了吗？是不是还是一头雾水？这就造成了一个万分尴尬的局面，大家知道，法律是要讲证据的，是有原则的，比如“疑罪从无”，比如“无罪推定”，比如“谁主张谁举证”。这些原则，都源于一个最根本的目的，那就是我们不能冤枉一个好人，如果你证据不够严谨，没有百分百的把握，是绝不允许给人定罪的。如果这个法律原则不守住，用舆论和权力来构陷他人将变得轻而易举，这个社会上的所有人都有可能瞬间成为冤假错案的受害者，那将是一个多么可怕的局面。

那么,问题来了，当用法律在处理类似于“性骚扰”这样的案件时，举证难、生活场景的界限感的模糊性，就让法律介入变得极为困难。这种困难，绝不是思维简单的键盘侠可以用“渣男”、“绿茶”之类的词可以简单总结的。

那么,我们应该如何做，才能既保护自己，又不冤枉对方呢？

光说理论可能大家不太好理解，我举几个实例来进行说明吧。某东哥的事情大家都知道，因为众所周知的原因，我就不提这件陈年旧事的细节了。总之，东哥是被冤枉了，女主其实是自愿的。但是事件刚出来的时候，吃瓜群众可是群情激愤啊，一个超级资本家对一介平民，这是阶层矛盾；一个强势的男人对一个柔弱的女子，这是妥妥的男权，一个事件同时具备阶层对立和性格对立的特色，不用说群众有多愤怒了。然而，最后随着美国警方的调查结果，瓜众发现自己被愚弄了。

你一定要在连信息都不全的时候就去站队，就去宣泄情绪，只听取一面之词就去给事件当事人定罪，实在是很愚蠢的事情。我们不能被网上的各种信息牵着走，对待真相要冷酷一点、严谨一些，不要让情绪占据大脑，要善于分析比对，从而筛选出真正符合常识和逻辑的真信息来。在真信息并不充分的情况下，同妄加揣测事情真相相比，让子弹再飞一会儿，才是更加明智的选择。尤其是面对男女之间的玄学，更加需要谨慎。公众打开第一个盒子，马上大骂刘强东是“渣男”，公众再打开一个盒子，突然发现，上次笃定的结果，原来是错的。还别说外人无法了解，就是当事双方前一秒和后一秒的想法都可能天差地远，当对方表达出拒绝之意时，如果你产生误判，导致的后果也会天差地远。也许男方认为是默许的“半推半就”，女方却很可能认为是强迫的“性侵”；也许男方认为是社交场合里的“亲昵性友善”，女方则可能会认为是突破界线的“性骚扰”。男女关系就如同薛定谔的猫，当你不说，我不说，就会永远处于一半同意，一半反对的叠加状态。

对于疑似被侵犯的女性来说，举证很难，很难给对方定罪，所以有人说在法律无法定罪的前提下，在网络上爆料属于自我救济；对于嫌疑犯来说，要自证清白更难，而且网络审判一旦形成，就会处于失控状态，整个生活包括家庭都会遭受远超侵害程度的惩罚，造成罪刑不适应的另一种不公，还别说存在谎报的可能性。看看这次阿里事件，男主虽然无罪释放，但其家庭已经遭到了灭顶式的打击，即使清白了，代价已经付出了，今后哪家公司还会请他真不好说。

那么,这种事情是无解的吗？很遗憾的告诉你，是的!很多事情在法律上都是无解的。如果我们对于世界有清醒的认识，就会知道，我们总会有无法侦破的案件，总会有罪犯逍遥法外，所谓“法网恢恢疏而不漏”，不过是骗人的童话而已。我们可能永远不知道辛普森有没有杀害他的妻子，我们可能永远不知道MH370 失事的真正原因，我们可能永远也不可能拥有上帝视角去观察到章文和蒋方舟当时的具体情境和微妙的心理活动……

我们必须承认，世界偶尔会失控，不在我们掌握。虽然，这个事实会让我们感觉沮丧。

什么是“调情”？什么是“骚扰”？这两者的界限，是个玄学，是个世界性难题。法国主流报刊《世界报》前段时间刊发了一封公开信，这封由女性作家、演员、学者和商业精英组成的百人团队，在信中抨击以“MeToo”为代表的反性骚扰女权主义运动，是一种新形式的“清教主义”。她们在信中这样写道：“男性对女性的调情，对性自由而言不可或缺。强奸是一种罪行，但不论手段是否笨拙，是否穷追不舍，追求并不是一种冒犯，更不是一种大男子主义式的侵犯。一旦沾上‘性骚扰’几个字，男性就会立刻遭到报应，丢掉饭碗，但是有时候他们只是碰了某个女性的膝盖，发出了含有性暗示内容的短信或者只是想亲吻女性。”她们指出，大量的仇恨情绪涌向被指控人，却没有给他们辩解的机会。某运动不但没有让男女更平等，女性更独立，反而让女性加深了“女性是弱者”的刻板认知，让男女之间的正常相处受到了威胁……这是女权运动者们愿意看到的结果吗？女性展示自己的性感除了愉悦自己，当然也有愉悦异性的心理，男人也是同样，这是正常正当的行为，无可厚非，男人见到自己心仪的女性，欣赏对方的性感，甚至被对方吸引，试图追求和调情，这也是正常正当的行为，不该被道德斗士们无端批评。很多有识之士指出，应该正确区分调情和骚扰之间的界限。我也认为这个界限重要，但并不是那么重要。因为人的情绪和心理是微妙的、是会变化的，一位斯德哥尔摩症的患者可能会把加害行为误判成保护，而一个被社会屡屡暗示男人都是流氓的女性也会把男人的调情和好感误判为骚扰。重要的不是你当时是自愿还是不自愿，重要的是你需要及时制止伤害。行动指南是：当你觉得受侵害了，那么就要有勇气明确告知对方停止伤害，而不是当时隐忍，事后声讨，否则当事双方的权利都可能无法确保，这已被无数案例证明。
