
# 关于民营经济发展，学会从最高层原话中寻找关键信息

## 健全公正司法

《指导意见》的最后一部分，聚焦“健全公正司法体制机制，持续提升司法保障效能”，我这里挑了五个重点，依次和你讲讲：

第一，“持续优化诉讼服务”。核心就是 对符合法律规定的起诉，做到“有案必立、有诉必理”，重申了立案登记制的铁律，这背后是对一些地方民企立案难的回应。

第二，“规范适用保全措施”。财产保全是司法执行的一环。文件规定，“严禁超权限、超范围、超数额、超时限制封、扣押、冻结财产”。比如欠了 1000 万，却查封了 2000 万的设备，这是不行的；也不能案子都结了，还迟迟不解封。

另外，要求尽量采用“活封活扣”方式，什么叫“活封活扣”呢？比如过去查封一个工厂，可能是把机器贴上封条，让它完全停产，这样可能就把企业拖垮了，现在是允许企业在监督下，继续使用这些机器进行生产，只是不允许企业偷偷卖掉或者转移。

同时，也要追究错误保全的责任，要明确对滥用保全权利，恶意促使查封别人财产造成损失的，要追责赔偿。

总之，不能让它成为一些企业打击对手的工具，也约束一些地方政府的 行为。

第三，“依法规范执行工作”。执行是司法公正的“最后一公里”。针对过往出现的一些乱象，这次《指导意见》明确提出：“规范财产调查，避免错误查封案外人财产”。比如企业家个人欠款，不能把企业家亲戚的财产都查封了；规范财产查封，严禁明显超标的查封，优先选择对被执行人生产生活影响较小且方便执行的财产执行。

另外，对于已经查封的财产，给这些财产定价要依法进行，不能乱估价。如果符合条件，也可以允许被执行人自己想办法卖掉这些财产。这也是为了防止财产被贱卖，有人从中得利。对于采取拘留、限制出境、列入失信名单、限制高消费等措施的，要严格审批审查。

第四，“健全失信惩戒和信用修复机制”。传统的失信惩戒机制，存在缺乏弹性、修复困难等问题，如果“处罚过度”反而使民营企业丧失履行能力。对此，这次文件引入了分级分类和动态调整的理念：比如区分“失信”，有能力却不履行，与“失能”，暂时没有能力履行的情况，对轻微失信且积极纠正的，给予宽限期；建立信用等级动态调整机制，根据履行情况半年一调。

同时，对失信被执行人主动纠正失信行为的，人民法院要帮助他们修复信用，出具证明材料，让他们及时回归正常生产经营轨道；还要加强与国家发改委、金融机构、市场监管等部门间的信用修复结果互享互认。

第五，“坚决纠治涉企审判执行中的突出问题”。比如一些地方的法院为追求经济利益，本来没有管辖权，却要把民营企业的案件弄到自己本地处理，形成“诉讼主场”。这次文件明确，坚持“法定管辖为原则，指定管辖为例外”的基本原则，严防趋利性执法，坚决纠治对涉民营经济案件扩张管辖、人为制造异地管辖的行为。

从整体上来看，这一部分的内容其实系统回应了民营企业对公平、高效、便捷司法的迫切需求。到这里，这份《指导意见》的重点，我就为你梳理完毕了。

## 从最高层原话中寻找关键信息

而在最高法的《指导意见》出台后，8月16日出版的第16期《求是》杂志，也发表了最高领导人在2025年2月中央民营企业座谈会上的讲话原文的主要部分。

我这里多说几句，《求是》杂志这种核心央媒刊载的原汁原味的最高层讲话，非常值得一看，相比传统的新华社通稿，讲话原文里面有大量关键信息。

这里我结合自己的学习理解，带你初步感受下。

首先，关于民营企业的发展现状和贡献，最高层说：我国民营经济已经形成相当的规模、占有很重的分量。民营企业数占企业总数的92%以上，被认定为国家高新技术企业的民营企业有42万多家、占比也在92%以上；民营企业对进出口和税收的贡献都在五成以上，对城镇就业的贡献达到八成以上。

之前关于民营经济的贡献，一种常见说法是“56789”，就是民营经济贡献了50%以上的税收，60%以上的GDP，70%以上的技术创新成果，80%以上的城镇劳动就业，90%以上的企业数量。

最高层的新说法，突出了高新技术企业民企占比92%以上，以及民企对出口的贡献在五成以上，这里比常规说法多出来的两点，我的理解是，它体现了中央对技术创新和外贸中民企贡献的重视，也是我们新的导向性重点。

其次，最高层关于民企面临的困难和挑战，是这么说的，我这里边引原文，边给出我的理解供你参考，他说：新一轮科技革命和产业变革风起云涌，给传统产业领域的一些民营企业带来较大冲击（你看，这是新技术和产业冲击）；一些科技型和外向度较高的民营企业面临困境（这是外部压力，包括大国博弈带来的科技封锁和外贸压力）；一些民营企业盲目多元发展，市场把握不好、经营管理不善（这是对民企自身问题的要求，别贪大，别头脑发热，这个我认为跟过分内卷也有关联）；有的地方民营企业发展环境不优，落实促进民营经济发展政策措施有待加力（你看，这是对一些地方政府的批评和要求）；社会上对民营经济的认识还有一些杂音（这是对社会舆论的要求）。

同时，我在学习最高层讲话原文的时候，注意到还有几个值得和你分享的关键点。比如对要着力解决拖欠民营企业账款问题。最高层说：解开这个疙瘩，要从政府、从国有企业特别是中央企业带头做起，落实责任，用好新增地方政府专项债等政策，切实加快清欠进度……不能边清边欠、清了又欠、没完没了。同时，要严防民营企业之间互相拖欠，严防民营企业拖欠员工薪酬。

再比如，最高层特别强调：切实依法保护民营企业和民营企业家合法权益。这句话有两个关键词，一是“合法权益”，强调保护的是合法权益，而不是非法利益；二是“依法保护”，就是保护必须符合法律法规，不能搞“法外开恩”那一套。这些都是原话。

而关于认真落实各项纾困政策，最高层说“不少民营企业反映获得感不强”，所以要“实事求是，说了就要做，真帮实帮”。

而关于进一步构建亲清政商关系，最高层说：这个问题我讲过多次了，现在仍有不少干部认识不到位、工作片面化简单化。我再次重申，“亲”和“清”本质上是干事和干净的关系，是辩证统一的，完全可以并行不悖。

同时他也强调，“民营企业家也要心地坦荡地同干部交往，不要利诱、围猎、腐蚀干部。”

这篇讲话的原文链接，我也放在了文稿后面，你感兴趣可以详细看。

## 背后的两个更大信号

最后，我做一些进一步的分析，谈谈我从这些政策和表述中，理解到的国家在治理逻辑上的一些深层转变，我注意到两个比较鲜明的信号。

第一个信号，司法正从“事后裁判”转向“事前引领”。

我结合最近的很多动作，综合判断，最高法和各地法院的角色正在发生变化，从“事后裁判”转向“事前引领”。就是说，司法机关不再局限于在纠纷发生后的裁判角色，而是要通过提供清晰的规则预期和正向的行为激励，起到事前引领的作用。

这让司法成为国家治理体系中更为积极的关键一环，它的目标不仅是解决已经发生的冲突，更是要降低全社会的交易成本，从而提升经济发展的效率和安全性。

第二个信号，中国在经济发展上的逻辑，在从“政策驱动”迈向“制度驱动”。

过去支持民营经济，更多依靠的是周期性、特惠式的政策红利，比如税收减免、专项补贴等，特点是灵活性强，但预期稳定性较弱。而这次通过法律的形式，把促进民营经济发展，嵌入到司法的全过程，正是在向市场传递一个明确信息：对民营经济的保障不是临时性的“优惠”，而是长期性的“权利”，不是行政性的“给予”，而是法治化的“确认”。这是从“政策之治”到“法治之治”的转变，旨在从更本质的层面上稳定民营企业家预期。换句话说，要通过构建更先进的治理体系和法治环境来提升经济发展，最终目的都是要改善资本配置效率、激励创新行为、减少司法耗损、提升发展信心。
