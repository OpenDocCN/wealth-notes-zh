
# 洋恺宏观：三方博弈的策略选择

三方演化博弈模型是博弈论中的一个分支，它考虑的是三个参与者之间的互动和竞争。这种模型可以用于研究各种情境，如生物学中的物种竞争、社会学中的群体行为、经济学中的市场竞争等。在三方演化博弈模型中，通常会考虑每个参与者的策略选择、相互之间的影响以及演化的动态过程。下面是一个简单的例子来说明这个模型：

假设有三种动物，分别为狮子、斑马和羚羊，它们生活在一个封闭的生态系统中，彼此之间存在竞争关系。每种动物都有两种行为策略可供选择，例如：1、狮子的策略：狩猎或者休息；2、斑马的策略：逃跑或者觅食；3、羚羊的策略：逃跑或者吃草。每个动物根据其选择的策略和其他动物的行为来获取收益。例如，如果狮子选择狩猎而斑马选择逃跑，则狮子可以捕获斑马，而斑马则无法觅食。但是，如果斑马选择觅食而狮子选择休息，则斑马可以获取食物而狮子不会获得任何收益。

假如有三个枪手参与决斗，每轮只能射击一次。枪手 A 的命中率高达 80%，枪手 B 的命中率有 70%，枪手 C 的命中率只有 60%。看似枪手 A 占据优势，然而由于他的威胁过大，在第一轮决斗中另外两个枪手会共同朝枪手 A 射击，这导致枪手 A 反而最有可能被优先淘汰。

不仅如此，枪手 A 会把命中率仅次于他的枪手 B 视为最大威胁，并在首轮决斗中优先朝他射击。这使技术最差的枪手 C 反而获得战略优势，在另外两家决出胜负之前，枪手 C 都不会遭到子弹威胁。根据计算，在多轮博弈下枪手 A 的存活率为 4.3%，枪手 B 的存活率为 6.5%，枪手 C 的存活率竟然高达 75.2%。

假如出手有间隔的话，枪手 C 的战略优势只会更大。他可以选择隔岸观火，等待枪手 A 和枪手 B 其中一方被命中后，迅速朝残存的另一方开火即可。一战时期美国就是等到同盟国和协约国斗得头破血流后，才下场收割胜利果实，成为最大赢家。也就是说，看似最弱的老三，在战术上是劣势方，但在战略上却是优势方。老三能充分利用老大和老二之间不可调和的矛盾，既可以帮助老二共同对抗老大，获取更高赔率；也能联合老大压制老二，获取更高胜率。不仅如此，老三还能保持中立，待老大和老二打得奄奄一息后再下场争夺胜利果实。

弱势方的战略优势：在三方博弈中，看似最弱的第三方（如枪手 C）因威胁值最低，反而能利用两大强者（A、B）的矛盾获得生存空间。其策略包括：1、隔岸观火：等待强方两败俱伤后收割残局（如一战美国的中立策略）。2、灵活站队：通过在两大阵营间左右逢源获取利益（如冷战时期中国在中美苏三角中的角色）。

三国时期，魏蜀吴围绕荆州的地缘博弈，蜀汉通过联合东吴对抗曹魏，最终在三方中维持生存。冷战中美苏大三角，中国先倒向苏联获取工业基础，后转向美国融入全球化，成为冷战后期的最大赢家。21 世纪中美俄三角，中国崛起为"老二"后，俄罗斯退居"老三"角色，通过在中美间腾挪获取外交主动权。

威胁值决定攻击优先级，三方力量对比中，最强者因威胁值过高成为众矢之的，而最弱者因威胁值低反成"安全节点"。然而三方中任何两方的联盟都可能因利益冲突瓦解，第三方可以通过挑动矛盾实现"渔翁得利"。弱势方需隐藏实力，避免成为攻击目标；并寻求通过联合一方对抗另一方，逐步改变力量对比。

中国作为"老二"需警惕成为"众矢之的"，可通过分化美俄关系、利用地缘矛盾维持战略主动。另一方面，在多极化格局中，小国可通过在大国间游走（如东盟的对冲策略）获取最大利益。
