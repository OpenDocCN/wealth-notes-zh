
# Tiktok与巴拿马运河交易

特朗普在多个问题上的诉求与性质其实是一样的，都是美国资本要购买其股权。特朗普因政治需要，将其包装成国与国之间战争，把一个生意变成两个生意，在助推生意完成的同时，给自己招揽赢麻的功绩。

收回巴拿马，收回购入格陵兰，混改乌克兰，这些开疆扩土的行为，能够极大的激发红脖子们的兴奋。这些协议其实本质都是多方共赢的，甚至很多协议懂王为了揽功，也会出让一定的利益。但也正是因为懂王揽功和舆论宣传需要，也把很多原本简单的交易加入了政治因素和不确定性。

譬如乌克兰实际上的矿产储量都是未知指数，经济与安全双保障的协议是对乌克兰相对有利的。但获得欧洲支持的泽连斯基却坚决拒绝签署。譬如格陵兰岛实际上最后要的是矿产与油气的联合开发，对因俄乌冲突后能源紧张的欧盟也都相对有利，但是欧盟基于不能让懂王“赢麻了”的角度，也会通过激起民意来阻挠协议的推进。跟中方相关的Tiktok和巴拿马运河也是同样，虽然特朗普是在强抢，必然低于未来市场的估值，但懂王为了在中期选举前达成交易，也会让美国资本在价格方面给出一定让步，愿意用里子来换面子，使得这笔交易对于资产持有方来说并不太亏。

甚至账面还能显示出巨大的盈利，所以李嘉诚和他背后股东割肉割的很痛快。横跨了三届任期的Tiktok也是同理，懂王背后的资本在懂王的制约下，虽然是在掠夺，但并不敢狮子大开口，影响懂王“赢麻”给出条件对于张一鸣和他的股东来说也是相对有利，跟巴拿马运河以及泽连斯基的矿产协议类似，股东层面既能拿到丰厚的资金，也能解决未来的安全与庇护问题，实际上也是双赢。

因此接下来卡着交易的并不是Tiktok的股东，而是兔子用交易的审批权，接下来要跟特朗普博弈。

关于酒庄，群里大佬国外应该有酒庄吧？小燕子的酒当年买的不错的，不知道被老美抢了没？酒庄这玩意一般都是类似于炒币逻辑，如果庄主自己知名度不高，就很麻烦了。好酒之间的差别，差别类似于比特币与以太坊吧。囤酒，囤画，跟囤币，其实差别不大。好酒好烟好茶，都是傻傻分不清。当年很多人去买，不就是赚钱，跟买足球俱乐部，买球员啥的。

只要是能控制住发行端，就成。很多酒庄老板自己喝自己的酒，就跟拍卖了两幅画后烧了一副，性质差不多。骤然暴富，而钱的来路又不正，就需要把钱给洗干净。定价逻辑就是稀有性，然后自己搞标准，各种香型，喝不出来那是你不行。顶级的红酒，白酒，古玩，字画，比特币，甚至假币，都是类似的。越是产量低越值钱，就是奢侈品。

产量低只是一方面，更重要的是必须要控制发行端，不能像老蒋那种无限制发币。这方面法国人玩的明白。LV拉菲啥的。近代高利贷师祖，法国人，拿捏的死死地。本质上都是通过炒作定价，最后把他跟权力进行绑定，成为权钱交易的一般等价物。老班章，冰岛啥的，完全可以复制。是的，销量不行就把红酒都倒了，然后政府还补贴种葡萄和酿酒的，就是要维持价格不能乱来。我们就是卷价格就把产业卷没了。在西方看来，兔子这种内卷玩法太坑了。

东西争霸，在兔子看来，西方这种玩法，太坑了。赵薇搞酒庄，本质上是试图把一般等价物从自己的肉体，转变为红酒。老欧洲这套逻辑不一定被双方买账啊。毕竟自己已经年纪大了，精力不够了，很难再作为演员导演啥的来帮助大佬们洗钱了。我感觉东西相互别苗头最后老欧洲这套价值定价体系指不定就无了。毕竟一切价值体系的源头起码你得有贵族范儿吧，如果你都堕落穷困了，还怎么搞价值体系？

譬如那个嫖娼被抓的郑总，当年美人鱼的红酒，就是广告植入，因为周星驰这帮人也在炒和囤红酒。哪个郑总？还有香港那个红棍演员卖红酒翩，保护费不收了。他就是跟周星驰一起炒红酒的。很多时候，他们的酒庄，9就是个青瓷会所，负责钱权交易，负责给行贿的进行销赃，奢侈品都有这个功能。船长要是搞红酒的话，得有自己的交易圈。
