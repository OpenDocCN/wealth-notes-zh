
# 最新财新周刊特刊250922：新型政策性金融工具将出

新型政策性金融工具有望落地。2025年9月10日，国家发改委主任郑栅洁向全国人大常委会报告下半年工作时表示，将加快设立投放新型政策性金融工具。国家发改委在8月1日的新闻发布会上，就曾表示“将报批加快设立投放新型政策性金融工具”。

新型政策性金融工具最早在2025年4月底的中央政治局会议上提出。从此后的官方表述和地方政策解读会、项目调度会等透露的信息看，这一工具与2015年的专项建设基金和2022年的政策性开发性金融工具类似，是由政策性开发性银行——国家开发银行、中国农业发展银行、中国进出口银行通过基金来投放资金，主要用于解决投资项目的资本金问题。

与前两次的主要区别在于，这次将集中在数字经济、人工智能等八大领域，并有一定比例投向民企项目。

## 三轮准财政工具对比

多家政策性银行人士对财新表示，新型政策性金融工具规模初步定在5000亿元，其中国开行2500亿元、农发行1500亿元、进出口银行1000亿元；三家政策性开发性银行根据自身业务特点，各有侧重，如国开行侧重制造业等重要基础设施，农发行侧重农村领域重要基础设施，进出口银行侧重重要的外贸项目，主要是期限10年、15年、20年不等的长期项目。

近日多位相关银行人士告诉财新，这一工具有望在9月底落地。一位接近农发行人士透露，地方分行摸底的项目已经上报给国家发改委，准备工作已经完成，"届时项目批下来，很快就能投下去”。

此前专项建设基金和政策性开发性金融工具的推出，都是为了应对经济下行压力。随着2015年新预算法的实施，地方通过融资平台举债大大受限，而“开正门”的地方政府专项债券刚刚推出，当年规模只安排了1000亿元，地方投资承压，专项建设基金成为新的资金来源。2022年政策性开发性金融工具则是在新冠疫情、经济复苏不确定性加大的背景下推出的。

2025年以来，随着美国推出所谓的“对等关税”，中国经济发展面临的外部不确定性加大，经济运行仍面临不少风险挑战。中国经济体制改革研究会副会长李晓超提醒，固定资产投资增速连续数月回落，而且回落幅度较陡，需引起特别关注。

正如广东省佛山市发改局所言，当前地方发展很大的困扰正是缺资金尤其是缺资本金。新型政策性金融工具的推出，无疑会缓解投资的资金压力。但此举能撬动多少投资，对稳增长起多大作用?

多家市场机构认为，理论上的测算相对简单，根据项目最低资本金要求和新型政策性金融工具在资本金中的占比上限，即可简单估算出撬动倍数，但在实际操作中，往往还受其他条件影响，比如项目实际资本金比例可能高于最低要求、配套资金是否到位等。

华泰证券首席经济学家易崆表示，新型政策性金融工具能撬动多大规模的资金，可能仍取决于地方财力、银行配套贷款的规模，以及社会资本投资的意愿。

5月以来，广东、江苏、河北、湖南、山东等多个省份的地市级发改委及部分行业主管部门，组织开展了2025年新型政策性金融工具专题培训暨项目申报培训会议，以期抓抢机遇，精准对接项目和资金。

据财新了解，新型政策性金融工具的资金主要投向包括数字经济、人工智能、低空经济、消费基础设施、绿色低碳、农业农村、交通物流、市政园区等共八个领域；项目选择方面，原则上投向2025年底将要开工或已经开工但大部分未建设完成的项目。

多位政策性银行人士透露，与2015年和2022年的政策不同，此次还有个额外要求，5000亿元额度中的20%,即大约1000亿元，必须投向民营企业。

三家政策性银行基础设施投资基金在项目来源上采取“部委推荐、自主决策”。一位政策性银行人士介绍，对于部委推荐的清单，银行可以择优而投。

运作方式方面，新型政策性金融工具仍然由三家政策性开发性银行的基础设施基金公司来实施。“一套人马，加挂个新牌子。”一位国开行人士透露。

此前的专项建设基金、政策性开发性金融工具，均以基金的形式运作。2015年至2017年期间，国开行、农发行分别通过国开发展基金有限公司、中国农发重点建设基金有限公司入股项目公司；2022年是由三家政策性银行新成立的基础设施基金公司入股具体项目。

“这次还是由基础设施基金公司来操作，因为过去几年这几家基金公司已经积累了一些经验教训，能规避一些问题。”前述国开行人士说。

对于注入项目资本金的具体形式方面，多位政策性银行人士表示，与2022年的工具类似，此次新型工具的注资形式可以是股权投资或股东借款。

其中，股权投资是政策性开发性银行通过基础设施基金公司将资金投入项目公司，但不介入项目公司的日常经营管理，到一定期限后退出；股东借款是基础设施基金公司委托政策性开发性银行向项目公司股东提供借款，所借资金专项用于项目资本金投入，项目业主通过项目运营收入还款。

两者的关键区别在于，股权投资时，出资的基础设施基金公司风险自担。“我们不做股权投资，更愿意做股东借款。”前述接近农发行人士说。

与此前不同，此次股东借款资金可以直接给控股集团，再由集团分配给子公司的项目公司作为资本金。前述国开行人士说，此前股东借款资金必须直接打到项目公司的银行账户，这一变化是为了满足更多需求。

另据财新了解，与2022年一样，此次新型政策性金融工具也可以为专项债项目资本金“搭桥”。对于“搭桥”，此前政策的解释是，考虑到部分项目短期内可能无法通过发行专项债获得资本金，政策允许“搭桥”，即先以专项基金做资本金投入，待专项债发行资金成功到位后，再将专项基金置换退出。

资金成本方面，专项建设基金和政策性开发性金融工具的实施中，中央财政予以贴息。但多位政策性银行人士向财新分析，此次财政部大概率不再贴息。

熟悉专项建设基金操作的人士介绍，2015年至2016年国开行和农发行发行的前六批专项建设债券募集资金，财政部对债券利率的90%给予贴息，这样政策性开发性银行投资的年化收益率才低至1.2%，但第七批改为分档贴息，从0、50%、70%、90%不等。

2022年推出政策性开发性金融工具时，中央财政对政策性开发性银行发行的金融债贴息两年，且仅对实际股权投资额贴息。当时接受采访的多位银行人士和研究人士认为，财政贴息力度低于专项建设基金。

前述农发行人士说，2022年政策性开发性银行给予项目的资金利率是3.06%，由人民银行通过抵押补充贷款(PSL)给予三家政策性开发性银行低利率资金支持。“要么央行给PSL，要么财政给贴息，否则是做不了的。”一位政策性银行人士表示。

据财新了解，此次新型政策性金融工具仍由人民银行给予部分低成本的PSL资金，同时政策性开发性银行要自己发债募集一部分资金，“目前三家政策性开发性银行的资金混合成本是多少，还未确定”。

2025年5月7日，央行推出一揽子货币政策措施，包括把PSL利率降至2%，这有利于降低政策性开发性银行的资金成本。对于发金融债募集资金，业内人士预计，2025年以来发债利率低，估计是2%多一点，“如果财政不给贴息，应该也可以平衡成本”。

也有政策性银行人士认为，如果新型工具是为了解决项目资本金瓶颈，也没必要贴息。“资金利率稍微高一点影响不大，因为资本金是有撬动作用的。”
