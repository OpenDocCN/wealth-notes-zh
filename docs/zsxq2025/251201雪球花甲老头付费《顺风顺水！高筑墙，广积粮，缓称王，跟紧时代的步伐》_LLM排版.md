
# 顺风顺水！高筑墙，广积粮，缓称王，跟紧时代的步伐

## 前言：

固护城之墙，蓄可用之粮，敛争逐之心，择可为之机。2025年接近年底，想以本文梳理下这个阶段的思考和总结。

我们用眼睛看东西，如果贴到眼睛上，几乎是一叶障目，什么都看不到，所以必须要保持一定的距离，才可以聚焦，但是想总览更多的脉络和视角，就要站得更高，才能望远。你走在街上，能看到人流和店铺，但是站在摩天大厦的顶层，视角又是另一番景象。

思维也是如此，当局者迷，而且越钻牛角尖就越想不明白，看不清楚。所以要找摩天大楼，找山顶，找飞机，君子生非异也善假于物也，把自己置身于另一个场景和环境，你的思维的角度和视角，会焕然一新。所以，经常能够换位思考的人，他们的思维体系更扎实，他们的选择也是深思熟加。

虑，如何换位思考呢？首先就是要借助别人的眼光，别人的思维，别人的经验。想办法融合成自己的东西，这就相当于登高而招，臂非加长也，而见者远；顺风而呼，声非加疾也，而闻者彰。假舆马者，非利足也，而致千里；假舟楫者，非能水也，而绝江河。

从信息的分享来说，网络是一个好的发明，因为我们能获得太多自己认知以外的知识，经验和观点，而这些都可以让自己的思考和体系更完整。所以，同样是网络，有人用来学习武装自己的头脑，有人则当作了发泄情绪抬杠的场所，这才是人和人之间拉开差距的根源。

我认为，人生有很多重要的课程，但这些都不能依赖于学校，因为学校根本不教，而这些重要的课程，是需要自己去自学，你也会在路上遇到无数的老师，他们或多或少给你一些方向和经验，但最重要的是要有一种意识，那就是从根本上认识到这些课程的重要性。

我们的一生，其实处处都充满了投资的思维，甚至来说，你的每一次决策，都有投资思维的痕迹，比如简单的一个人自驾游，之所以你会选A路线，而不是选B路线，说到底，你是从心里看多A路线，可能是出于时间短，高速费少等一些性价比高的因素。

正因如此，我希望各位读者能够重视我接下来要聊的一些课程，我认为是人生的必备课程，这些甚至更为重要，因为学校第一是普及基础知识教育，第二也是选拔人才，而我所聊的一些方面，是人生的基础，是一生受用无穷的东西。

比如人性的弱点，比如子女教育，比如周期规律，比如心态的稳定，甚至对于财富和欲望的看法等等，在接下来的时代里，真的是需要系统性学习这些课程了，你会发现它们比很多学科都更有用，甚至会伴随一生都需要。

人最重要不是努力，而是在正确的方向努力，人最重要的也不是一次两次考试，而是看准了机遇顺势而为，在一个大的周期里，不要逆势而为。还有就是了解自己，任何事情，都要结合自己的实际情况，这是认识自己了解自己的过程，经历过这两点之后，一个成熟的人，要能学会掌控和驾驭自己。

就像驾驶一辆车，你根本不了解它的出场参数和性能，那怎么行呢？就很容易做出超越能力范围的事情，比如用低排量汽车挑战越野场景，最终车子报废。

有几个月没有写万字长文了，很多话就在这篇文章里记录分享吧，纵观历史，无论是往前看一百年，还是一千年前，或者一万年前，我们都是生活在一个最好的时代，所以，我希望各位读者，在这个伟大的时代里，要从容幸福，要尽量学习提高自己，不枉我们遇到的所有美好。

全文近两万字，我们身处时代，就要保持学习，紧跟步伐。

大家都有过运动的经历，无论是打乒乓球，打羽毛球，亦或者是跑步和游泳，我们都知道会有外力的影响，比如风向和水流，那我们应该明白，逆流而上和逆风飞翔是多么的艰难，除非你的能力非常强，已经可以无视外力的影响。

比如一个身强力壮的人，他打羽毛球可以给出很大的力气，即使有一些逆风，也能打出去，但是我们为什么不 能顺势而为呢？这是一个思路，甚至是很多人做决策的一个基准，那就是我们尽量去打顺风球，做顺水推舟的事情。

何为顺风？这就基于我们的思考和判断，周期有长短，其实以天为单位的炒作，这也算顺风，但是对于我们绝大多数的普通人来说，我们尽量抓大周期，因为周期长，容错率高。诸葛亮借东风火烧赤壁的故事大家都知道，试问，如果这场东风只有三分钟，诸葛亮会出兵击曹么？答案肯定是是否定的，因为这点时间太难把握了，而且风向一变，势头就不对了。

所以，我们要抓住大的周期，忽视小的波动，我们都是普通人，难免因贪婪和恐惧有失误，而这种大的周期，最好的优势就是容错率，你打羽毛球是顺风，对方就是很难接住，如果你一直顺风，你会一直轻松，占据了天时的优势。

说实话，如果游泳的时候，一个人普通天赋的人顺着水流游，另一个人是高手但逆流而上，那么最终可能还是普通人更快，从这一点大家能看出，外力有时候会弥补个人能力的不足，也就是一旦占据了天时，就事半功倍了。想一想这几年，有多少高手在逆风的市场里折戟沉沙，又有多少小白在顺风的市场里取得果实，这就是趋势的重要性，还是那句话，一定要把自己置于顺风的趋势里。

火烧赤壁的故事，其实有很多可以研究学习的点，诸葛亮为什么能借来东风？周瑜为什么知道火攻但是无能为力？曹操为什么坐拥百万雄师最终丢盔弃甲？

我曾经给孩子讲过三国这一段，着重给他讲了很多引申出来的想法，虽然三国的很多作品，是文学类的小说，但是我们亦可以在这里面，学习到非常有用的东西。

首先，最重要的就是学习，人不能停止学习，无论是三十岁还是八十岁都要学无止境，人和人之间知识储备是非常重要的差异，诸葛亮很明显不是神仙，而是他深谙天文地理等知识，东风不是诸葛亮创造的，更不是他借来的，而是他判断到的一个必然结果，他只是搭了一个顺风车。逆势而为必败，顺势而为则事半功倍。先是分析用火攻，然后判断风向，之后是等待，最终是行动，缺少哪一环都不行。

而周瑜想的是什么，他受限于自己的知识壁垒和思维困境，甚至是以往的经验主义，他虽然知道要用火攻，但是碍于“隆冬无东南风”的认知局限陷入绝境。

但事实上，长江中游地区特殊的地形（两侧为山脉，中间为长江谷地）会导致气流“狭管效应”与“地形阻挡”，当西北季风遇到江汉平原两侧的大别山、巫山等山脉时，部分气流会沿长江谷地向南偏转，形成局部东南风（类似“穿堂风”），且多在夜间或凌晨出现（昼夜温差导致的热力凝力环流加剧）其实曹操的失败，更多的是人性里的一些傲慢，因为他坐拥大军，兵力强盛，虽然北方士兵不善于水战，但是他也不认为在这个时候会有东风，越是自信的人，越难突破自己的认知盲区，很多重大的失败，往往都是一方过于自信的梭哈，然后忽略了小概率事件起到的作用。

大佬文果回队在这一局中，诸葛亮无非是赢家，不仅找到了火攻的优势，还判断到了东风的到来，甚至在人性上面，他也能赢得和周瑜的较量，如果诸葛亮过不了人性嫉贤妒能或者争强好胜这一关，他和周瑜都没法联合。东风不重要，没有火的配合什么都不是，火攻也不重要，因为没有东风根本赢不了，联合也很重要，如果没有孙吴联盟即使有东风和火攻也是赢不了的。诸葛亮早已预判到：火攻需要风向、火船、内应、追击等多重条件配合，他的每一步行动都服务于全局——筑七星坛是为了稳定军心、迷惑曹操，提前安排赵云接应是为了规避周瑜的加害，预判东南风的时长（三日三夜）是为了给火攻留足时间。这种“牵一发而动全身”的系统思维，让他在复杂局面中始终掌控主动权，避免“顾此失彼”。

有时候我曾经想过什么是神仙，这个世界上到底有没有神仙，后来我想明白一件事情，当你的知识储备，掌握技能和规律了解非常高的时候，那在别人的眼中，可能你就是神仙一般的存在。

试想一下，比如一个现代人，穿越到石器时代，那他就是神仙一般的存在，什么钻木取火，什么杠杆原理，什么夜观天象，这些在现在看着习以为常的东西，在茹毛饮血的部落里，大家都会觉得非常神奇。

别说石器时代了，就是现代一个高知的普通人，他具备的知识储备和学习到的经验，即使去了古代任何一个时代，他都是一名非常优秀的谋士，难道现在的人不知道草船借箭么？我们是已经站在过巨人肩膀的人。但是，有一个问题是，人性永远是不变的，这些年人性的进步远远滞后于知识与科技的发展，所以如果你回到古代，没有强大的内心，也是不行这就是我们现在需要重视的地方，一个是知识的储备，就是不断学习，二就是要了解自己，掌控自己的欲望和心态。

如果必要，真的需要科学的学习和训练，就像一个孩子上台演讲，他不仅要有脑子里有东西，可以脱口而出，还要有一个好的心理素质，不紧张，能够把自己发挥出来。所以，身处我们这个时代，没法和古人去做对比，任何一个大学生掌握的数理化都比古人要强，他们哪里知道牛顿三定律，哪里知道麦克斯韦方程，但是我们和同时代的人去比，仍然适用这个方法，那就是不断的去学习，增加自己的知识厚度，认识规律，把握内心的贪婪和恐惧，这个是基础，也是重中之重。
