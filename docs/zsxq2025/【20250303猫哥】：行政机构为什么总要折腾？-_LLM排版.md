
# 行政机构为什么总要折腾？

前天与一个体制内的朋友喝茶，期间听到他颇有抱怨提到，这几年体制内工作强度太大了，其中有很大一部分就是应付各种上级机构检查，填不完的表格，做不完的资料，还有各种考核，把人累的半死，他觉得很多考核检查都是形式主义，都耗费了下面人太多的精力。

然后他问了我一个问题，为啥我们的行政机构总要这么折腾呢？对于这个问题，我倒是有自己的思考。

行政机构是一种特殊的社会组织，与普通社会组织不同的是，行政官僚机构是基本没有生存压力的，没有生存压力的社会组织就必然遵循一个自然规律——组织会自然衰退。

纵观历史，官僚组织自然衰退的规律是非常明显的。以我国大一统的中央集权的官僚机构为例，历朝历代都非常重视前朝覆灭的经验教训，在官僚机构设计上都会刻意打上相应的补丁，或为了避免宦官专权，或为了避免后戚干政，或为了避免藩镇割据，或为了避免官员腐败等等。

所有的顶层设计初衷一开始都是好的，但是也无法避免官僚组织自然衰退的自然规律。只要一个朝代执政时间长一点，官僚组织系统总是会衰退得不成样子。

明末崇祯年间，满清入关劫掠，一个2亿人口的中华帝国，最后朝廷能调集出来野战的也就是宣大2万左右的军队，就这样的军队不但还欠6个月军饷，而且还经常断粮，明军在境内作战居然搞出敌后作战的感觉——最后宣大军队全军覆没，整个官僚系统居然无人担责。由此可见，在明末官僚系统衰退腐朽成什么样子。

明后的清朝也好不到哪里去，到了19世纪50年代，也就是满清执政200年左右，天平天国起义，一群没有任何军事素养的农民拿起锄头就将满清正规军打得落花流水，满清官僚系统好几年时间都无法阻止有效的抵抗。

为什么中国历朝历代大多只能生存200年—300年左右，原因无它就是官僚系统的自然衰退也就能维持200年—300年的时间。

那么有没有办法对抗官僚机构自然衰退的规律呢？有！

就是要对官僚机构进行不断的折腾。最简单的折腾就是不断的检查与考核，检查的目的即使对官僚机构进行摸底，在现有条件下，这个组织还有多大的战斗力，官僚机构设计一般都是有冗余的，如果摸底的结果是组织衰退不大，那么冗余部分就可以保证官僚机构还能满足相应的职能设计要求。如果这个组织战斗力已经衰退得非常严重，那么上级部门就要考虑进行组织变革，直接裁撤合并相应的机构。

这就是行政机总是检查考核不断的深层次意义。当然，也有一些机构专门应付检查的，表面文章做得很好，其实战斗力就是一个渣，在这种情况下，又设计出巡视模式来打上补丁，巡视模式可不是传统的检查可以应付的。这个巡视的模式就是鼓励组织员工向巡视组举报，然后根据举报内容对相关领导约谈。

检查、考核、巡视本质就是为没有生存压力的官僚机构人为创造危机感与紧迫感，从而对抗官僚组织衰退的自然规律。

正是这个原因，所以放眼全世界，我们行政机构是最能打的，特别是遇到突发危机事件，平时看起来吊儿郎当的公务员冲锋陷阵好不含糊，比较典型的例子就是2020年突发的疫情，城市管控，供应链组织生产井井有条，让全世界都目瞪口呆，后来就是让它们来抄中国作业也学不会。

为啥？因为承平已久，国外的官僚组织实际已经衰退得不成样子了，已经完全不具备基本的社会组织与动员能力。

其实在疫情之前也有很多案例，地震、台风、火灾、雪灾、洪水、甚至春运等等，从行政机构到我们的国企组织在这些极端环境下，都表现出非常强的战斗力。

人会衰老，组织会衰退，这都是自然规律。要对抗自然规律就要不停的折腾，人要抗衰老就要自律，要控制饮食，要健身运动，而官僚组织呢？同样也要不停的检查、考核、巡视，人为制造官僚系统的危机感与紧迫感这都是一个道理。

中国行政机构为什么战斗力强？深层次原因就在这些被体制内抱怨不断的形式主义的折腾中。
