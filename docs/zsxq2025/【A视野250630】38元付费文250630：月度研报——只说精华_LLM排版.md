
# A视野月度研报——只说精华

## 一、上半年的波澜壮阔

### 1、国内

1~2月，经济体感逐步上行，3月份达到上半年顶点。4月川普发动关税战后，增速略有下行，5月加速进入观望状态，6月则出现低估。

主要原因包括：
- 1~2月，川普刚上台，重要政策未落地，中国出现“Deepseek时刻”，打破美国AI炒作浪潮下的“美国例外论”。
- 1~3月，市场抢出口热潮，预期川普发动大规模关税战，出口对经济托举力度不小。
- 4月，川普发布全面关税战，但传导至国际贸易需时间，对市场信心（金融端）冲击较大。
- 5~6月，川普同意中美关税战休战90天，但前期中断的国际贸易产生直接影响（时滞效应），市场注意到有形之手不再公布新的救市举措。

休战后市场更加迷茫，因为市场认定只要出口延续（90天休战），有形之手不会公布新的救市举措。所谓关税战利多，因预期看不到举措，反而变成市场维度的“利空”。寒气在以地产链为主的大量产业表现淋漓尽致，加剧市场观望。

有形之手始终未做类似924行情时期的沟通，市场对未来预期充满不确定性，这是5～6月市场加速走向悲观的根本原因之一。

90天休战看似第一回合我们赢了，但也放大了不确定性。市场会追问，90天后呢？美国人完成补库存后是否会加码施压？中美90天休战到期是8月12日，进入7月，市场担忧会随日子临近而加剧。

### 2、国际

川普2.0至今政策落地情况：
- 几乎没有政策达成预期，包括关税战。
- 海湖庄园协议核心内容（美债化债置换）无法推进，倒逼他跟美联储交恶，美联储更不愿意配合降息减轻债务负担。
- 川普利用关税战收获年化6000亿美元以上财政增收，但以侵害美国及全球经济增长为代价，具有紧缩属性。
- 美国债务增长速度极快，严重影响《大美丽方案》未来经济成效。

由于无法成功化债，明年美国到期需要化债规模极大，构成2026年美联储不得不执行大规模降息/QE的动力。

海外购买美债动力快速边际递减，美国本土储蓄不足以支撑美债超发，未来美债买家只能是美联储。

放眼全球，俄乌冲突、印巴空战、中东乱局、美国与盟友贸易纠葛，原有国际分工体系加速瓦解。但需注意，不支持说美元霸权即将崩溃，川普所作所为是在为修复美元美债体系努力，全球储备货币市场份额无证据证明美元信用即将崩溃。

从时间轴维度，川普目前困境是：大量重要政策必须延续拜登时期政策，但还要推进竞选时承诺的举措。一方面沿用高息融资短期美债，一方面仍想控制美债增速推进减税法案，构成今年上半年全球金融、地缘与经济的错乱根本。

## 二、调研发现

### 1、宏观政策

尽管7月有北京重磅会议，但市场整体预期非常低，认定依旧是“托而不举”。对比“美国例外论”，国内对观点是“新版迷失30年”、“长期通缩论”、“经济结构转型论”。

市场中悲观预期主要情绪支撑是确信不会再有大规模刺激（长周期内，非短期）。短线而言，支持这种观点。7月会议可能采用“托而不举”，但长期来看不支持。

就今年7月份而言，国内经济延续6月压力可能性不小。90天休战给中国带来的直接出口美国订单有限，休战期平均关税率仍30%以上。

重大变量在于美联储降息时间，7月降息则后续央行必然跟进，对情绪修复意义重大；9月重启降息，则三季度压力巨大，尤其“托而不举”情况下，三季度经济体感将偏弱。

### 2、突破点

在大环境压制下，有形之手能马上发力的应是A股。A股修复所需资金体量不多、敏感度高、惠及人口数量庞大、有直观拉升情绪的价值。

924行情直接提振的也是A股>楼市。从历史来看，城市化、出口、基建等稳增长工具发力空间有限时，A股成为唯一可用工具（1998年、2014年）。

A股整体洗筹已超半年，主力高度控盘，进入7月，市场情绪逐步升温可能性大。需注意，进入7月，若“两融数据”创新高，需关注短线风险。

可关注主线仍以锚定中美博弈思路思考：
- 科技+国产替代+稀土：关税战从关注税率切换到科技战，双方都不肯妥协，有形之手扶持，护城河明显。
- 军工：中美都将军工作为压舱石，美国要求盟友大幅提高国防开支，全球进入类似50、70年代“全球军工超级周期”。军工板块远不止传统逻辑，而是对标下一代技术革命开启的重要意义。
- 稳增长：以旧换新、特别国债、地方化债等举措不断发布。Top多次宣誓要转型为“消费大国”，撬动海外庞大资源和战略利益。

当然，地产未能启动前提下，消费效果相对有限。但“新消费”赛道后续仍积极乐观，需关注短期过度拥挤问题。大金融是散户入场重要信号灯，若迟迟无动静，散户难入场。

- 香港市场：正在发生翻天覆地改变，回归亚洲金融中心迹象明显。从稳定币到港股扩容，不少回流资金愿在港股享受外汇自由进出便利，成为美元carry trade套利对象。但相对比，香港楼市仍需时间等待。

## 三、我们的担心

### 1、地产

核心城市新房与二手房市场近乎冰火两重天。瑞银推测，过去5年买房业主近一半出现房价击穿买入房价，负财富效应下，居民消费欲望无法有效刺激。

情况已超过91年日本泡沫破裂程度，回撤速度之快、力度之猛，创纪录。对地产不能仅局限于建筑投资维度，更要从需求侧作为居民家庭财富载体思考。

如无核心城市二手房明显量价齐升，整体稳增长难度相当大，与中国信贷周期息息相关。美元只能强、不能弱，对应人民币信贷不断缩减、无法快速放大，二者一体两面，相辅相成。

### 2、产能

目前各地仍以奖励生产为主，对需求托举有限，产能对比需求出现严重不匹配的过剩，汽车是例证。有形之手努力打补丁处理，但效果仍需观察。

此事非常关键，需求不足压力加速传导给供给侧，形成供给侧的债务和风险。地产优质开发商遇到的问题也是这个逻辑。

这种处境会让企业坚持内卷，无法自拔，也会带来更多债务风险，为后续可能的爆雷铺垫。

### 3、川普的不确定性

川普的不确定性是其对外谈判内生组成部分，也是TACO来源。无论当前如何淡化对华博弈，其每一步不是针对民主党，就是为将来针对我们做前置条件创造。

预计年内中美达成实质性条款可能性极低，势必带来两边激烈博弈，川普放大我们经济不确定性。基于此，美联储正式降息前，央行救市举措难度极大，国内市场将处于相对更久的悲观预期中。

预计下半年中美仍有至少一次非常激烈的博弈，但这对A股未必是利空。

## 四、总结

在中美博弈前景不明朗当前，7月北京重磅会议预期需放低。作为经济为数不多的亮点，进一步推动A股赚钱效应，以慢牛、长牛、局部赚钱效应等风格为主。
