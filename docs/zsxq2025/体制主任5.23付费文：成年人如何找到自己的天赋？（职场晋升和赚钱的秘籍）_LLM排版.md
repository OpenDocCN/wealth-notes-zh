
# 成年人如何找到自己的天赋？（职场晋升和赚钱的秘籍）

原创：主任 公众号：通透的职场人生 25年5月23日

天生我材必有用，只要你找到你的天赋，其实每个人都是天才，天赋就是你的灵性，也是老天爷留在你内心最深处的种子，人人都有。

之前我在公众号里说过，社会上的每个人都有自己独一无二的优势，我们每个人的身体里都有上天埋藏的一个礼盒，那就是我们的天赋，它一直在等待着你去打开。

很多人一辈子没找到自己的天赋，活得迷迷糊糊，他们既不知道自己的优势在哪里，也不知道自己该做什么,一生奔波劳碌，痛苦不堪，把整个生命都浪费掉了。

很多人穷其一生，都在寻找自己，认识自己，哪怕你现在人到中年，可能还没有真正的了解自己。

人和人之间的差距只有5%。柜台里的香水，95%都是水，只有5%不同，那是各家秘方。人也是一样，95%的东西基本相似，差别就是其中很关键性的5%.而这5%就是指人的思维、格局、眼界、认知，尤其是对天赋本质的认知。

可悲的是，普通人一辈子都在追随那95%的水，而自动屏蔽了起到决定性作用的那5%。

很多人，他们都有同样一个困惑，觉得自己并没有什么拿得出手的天赋，活了半辈子，硬是没有发现自己有什么可以对别人形成降维打击的优势。于是，他们认为自己大概率是一个普通的螺丝钉而已，没有什么核心竞争力，只能平凡地度过这一生。

很多人得出这么一个悲观的结论，原因在于人们对天赋存在巨大的误解，还没有真正明白天赋的本质。

而你，只要稍微清醒一点，找到自己的天赋，只要你找到，你的命运齿轮就会开始转动，接下来的人生就能重新设计。

也就是说每个人都是独一无二的天才，关键在于找到属于自己的赛道。一个人越早找到人生的“天赋轨道”,然后在剩余的人生聚焦、专注，在这过程中学习将你找到的天赋能力销售出去。

天赋的发挥，除了能让你事业蒸蒸日上和赚到大把的金钱，更重要的是，它会让你觉得生命充满了动力与激情。

一旦你找到自己的天赋之后，你的人生一定会焕发第二春。只有这样，你这辈子的精神和物质才能获得终将富足，再差，也会大器晚成。

本文会从4个方面告诉你怎么找到自己的天赋，再告诉你怎么利用好自己的天赋，如果你能把自己与生俱来的天赋挖掘出来、充分发挥出来，你的人生必将发生翻天覆地的变化。

那么普通人究竟该如何找到自己的天赋?据我总结，一共有4个标准：

其一：干什么不觉得辛苦? 你长期坚持做，却丝毫不觉得累的事情。想想看，在生活中，总有一些事情是你无论遇到多大的困难都愿意坚持下去的。长期坚持不累的，就是你丝毫没有感觉到是在干活，而是在享受，并且就算不给 你钱，哪怕没有任何回报，不管多累，都能坚持下去!比如，有的人对绘画有着执着的热爱，即使每天花费大量的时间和精力在上面，也不会觉得疲惫。

其二：干什么比别人快? 别人觉得难如登天，而你却觉得轻而易举的事情。每个人都有自己的优势和短板，当你在某一领域展现出远超他人的能力时，很可能这就是你的天赋所在。你觉得做起来不是很辛苦，水平也比大多数人都要高一些的，甚至是对别人来说需要付出很大努力，而对自己来说就像玩耍一样轻松有趣的事情。你要想想从小到大，有什么事，你不用怎么费劲，放在别人那就比登天还难、而放在你这，却要比喝水还简单!

其三：做了之后能让你废寝忘食、全然投入的事情。干了就废寝忘食的，做这个事情，你很容易进入一种心流的状态，忘我的状态，完全感受不到时间的流逝。这样的状态中的工作质量和工作效率都是非常高的。

你想想你做什么事会让你忘记时间(心流、热爱)? 你做那件事时，不管是偶然接触到，还是已经做了一段时间，有一种很爽的感觉、充满热情、很有兴趣、并且痴迷。如果有一件事能让你废寝忘食也孜孜不倦要做的事，那一定是你的天赋所在。因为只有真正热爱和擅长的事情，才能让人如此痴迷。

有人可能会问了：“是不是找到了这样的事，我就能就此平步青云，直达云巅了?”“打游戏、追剧、刷短视频最让我废寝忘食，可我怎么没有因此有什么成就?”

切记，天赋这张牌，不能单独出。千万别打着兴趣、天赋的幌子自欺欺人。“打游戏、追剧、刷短视频”这些事情，任何人根本不需要任何的力气就能做到。你说你打游戏厉害，那请问是有多厉害呢?有打职业联赛的水平吗?那些主流游戏的职业选手(比如《英雄联盟》),哪一个是因为“网瘾”就做上的?他们首先是在这方面有天赋，然后再加上大量的专业练习，才能达到那个水平，才能打出一一个个“亮眼操作”。

这里面有着大量的思考和繁琐的操作，你觉得这是你“爱打游戏”能比的?可以这么说，当上《英雄联盟》职业选手的难度，可能比考上清华北大还要高。当然，他们的收入也同样很高，年入可能有几百上千万，一点也不夸张。还有，你说你喜欢“追剧、刷短视频”,那你怎么就不能“拍短剧、做短视频”呢?做了之后能让你废寝忘食、全然投入的事情，并且还能赚到钱，这才是你的天赋所在。

其四：别人经常为什么求你?解决什么问题能让你兴奋?(价值感、成就感)。 你做什么事，做成了之后，不是光为了别人夸你几句，或者拿了什么奖那种高兴，而是你自个从心里往外觉得“嘿，这事干得真漂亮，有点小得意，甚至还想琢磨着怎么干得更好，这种能给你带来实实在在“成就感”的事。

如果遇到这样的事情，一定要立刻记下来，这很有可能就是你自己都没有意识到的天赋所在。这种让你有成就感，能让你持续开心的事情，这种天生的驱动力，或许是基因的理由，或许是你大脑的构造，也有可能来自于你后天的成长环境。但是不管怎样，这类事情你一定会比一般人更容易持续地去做。

对于我来说，解决读者的困惑和问题，听到读者升职加薪、有所进步的消息会让我感到兴奋，会让自己很有成就感，完全是一种超越物质的精神享受。

你做的任何事情只要符合以上四个方面的任何一个方面，都属于你独一无二的地方。

但是你发现了它以后还要持续精进，所有天赋得到发挥的前提都是，你不仅有天赋，你还得很努力才行。据我观察，身边那些做到厅级领导和赚到大钱的人，极少是凭借纯天赋就成功的。大多数都是先低下头猛干，取得一点点小成绩，形成正反馈，下一步就是不断的去精进并放大它，只有足够多的投入才能把天赋变成优势，否则它什么都不是。光靠自己的原始天赋是不足以在市场上脱颖而出的，因此需要我们通过刻意练习来提升和放大自己的能力。

对于我来说，我有写作天赋，但是这个天赋如果我后天不持续精进，也会变得平庸。没有了日日精进的动力，就不可能有深厚的积累，就不可能有远超常人的竞争力。

也就是说你写文章，你写的普普通通不值钱，你能够写大材料，你写过三五年七八年的工作报告、领导讲话稿，这种技能是值钱的。你是英语专业的，你的英语八级都没过，这种技能是不值钱的。你过了英语八级，有高级口译的证书，你能够跟老外谈笑风生，你对国内国外的各种考试，英语考试，它的技巧了如指掌，这种技能是值钱的。

所以一项技能，只有你做到超过身边80%以上的人才才是值钱的。当然了，是不是说你一定要达到超过70%、80%的人才能够开始?不是的，你刚开始做的时候，可能你只能够做到你的这项技能超过50%的人。

但是你慢慢做，你慢慢的积累，你在这个领域不断地学习，不断地研究，不断地精进，找到自己那块长板，把它磨得又光又亮，相当于把一口井深挖到1万米，量变必然引起质变，而质变之时，就是你成为顶尖之时，收获之时，你就能够成为这个领域的专家大咖。
