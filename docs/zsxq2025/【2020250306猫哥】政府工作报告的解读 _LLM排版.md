
# 政府工作报告的解读

## 猫哥原创
大家都很关心新鲜出炉的政府工作报告，谈一谈我的看法，我们挑紧要的说说。我看到舆论在炒作政府工作报告首次将“稳定股市与房市”写入，声称意义重大，意义当然重大，但是能否实现呢？我们用数据说话。

今年财政新增债务规模比去年增加了2.9万亿。这个数据看上去还不错，但是拆解一下结构如下：新增特别国债5000亿，这个主要是给国有银行补充保证金，这个5000亿与市场关系不大，所以真正投向市场流动的财政增量资金就是2.4万亿。

这2.4万亿组成包括，赤字率4%比去年增加1.6万亿，这是中央财政加的杠杆，地方专用债市4.4万亿，比去年增加5000亿，这是中央给地方财政加的杠杆。另外就是超长期国债1.3万亿，比去年增加3000亿，这个钱主要是给企业设备以及消费品购买补贴，相当于给企业与市场加的杠杆。

说实话，几个关键指标，赤字率、超长期国债、地方专项债都略微低于市场预期。赤字率是4%，市场预期是4%到4.5%。这个算是基本符合预期，也可以说是略微低于预期。地方专项债4.4万亿，比去年仅仅增加5000亿，这个是低于市场预期的。

因为政府工作报告说得很清楚，专项债增加部分主要用于投资建设、土地收储和收购存量商品房，消化地方政府拖欠企业账款，当然理论上除了增加5000亿之外其余专项债也可以用于上述领域。但是我个人感觉，这个钱不大够用。

以前我就给大家测算过，房市要企稳，国家至少要拿出2万亿。现在地方专项债一共就4.4万亿，新增部分只有5000亿，显然不大够，那么地方政府有没有可能将这个4.4万亿相当一部分拿出来收购商品房呢？基本没有可能。

我们不谈地方政府财政有很多坑要填，不可能拿出近一半来收购商品房，就说收储商品房本身，地方政府也缺乏积极性。因为现在积压的商品房一般都是100平左右的，而要收储商品房改造为保障房，后者一般是50平左右，这就存在结构性错位，如果地方政府将大量资金用于收购商品房，那么必然这些商品房大量变成沉淀资产，沉淀资产盘不活又要承担利息支出，地方政府是缺乏积极性的。

这也是去年央行安排3000亿专项贷款，而地方政府都花不完的根本原因。所以，如果仅仅靠着地方专项债收购商品房来实现让房地产企稳的目的很难实现的。但是让房地产企稳又是写入政府工作报告的。那么怎么办呢？唯一的办法是这样：

其1，今年大幅度降低土地供应量；
其2，全国大城市启动大规模拆迁（或者说旧城改造），拆迁资金由国开行提供专项资金，相对第一点第二点才是关键，但是政府工作报告没有提，最近好像有这个迹象，但是能否成为事实还有待观察，如果这个第二点不能落地，那么可以判断今年房地产企稳的目标是很难实现的。

房地产如果不企稳，今年要实现5%左右的经济增长目标难度就极大。去年能实现5%的目标有点特殊，因为四季度国外主要是北美市场都担心特朗普上台之后会搞关税大战，所以北美商家都在拼命囤货，这就是拉大了出口需求，让外贸成为四季度经济增长的火车头。但是今年特朗普已经上台，就基本排除这种可能性了。

要实现5%左右的增长目标房地产一定要企稳，房地产要企稳今年大城市一定要启动大规模旧城改造释放需求。超长期国债从1万亿增加了3000亿，这个也略微低于市场预期。本来市场预期超长期国债至少是1.5万亿到2万亿，现在是1.3万亿。这说明什么呢？

超长期国债主要用于补贴市场消费品购买，比如现在买家电、买手机、买电脑都有15%左右的补贴，市场预期这个补贴规模会进一步扩大，从部分消费品补贴品种进一步扩大化，甚至有没有可能扩大到装修补贴（讲一步刺激住房需求）。但是超长期国债仅仅增加3000亿基本就否决了扩大到装修补贴的可能性，大致就是比去年的消费品补贴略微扩大一些品种的范围。

去年有一个很关键的数据。2024年，企业部门与居民部门中长期贷款减少部分加上财政增加部分相对于2023年大致有3万亿的缺口。这3万亿的支出缺口（需求）也是导致经济陷入低物价循环的根本原因。今年财政增加支出是2.4万亿，如果房地产能企稳，这2.4万亿大致是够了，如果房地产不企稳（居民中长期贷款继续萎缩），这2.4万亿是不够的。

所以，赤字率、超长期国债、地方专用债等等关键指标都略微低于市场预期说明了什么？说明了管理层对经济现状与趋势的判断与市场的判断是不一致的，管理层对经济现状与趋势的判断相对市场要更乐观一些。

那么市场的判断与管理层的判断谁正确？答案也很简单。如果管理层手里准备打出大城市规模较大的旧城改造这张牌，那么毫无疑问，管理层正确。如果管理层并不准备打出大城市规模较大的旧城改造这张牌，我更倾向于市场更正确。三个月内即可见分晓。

相对房市未来还有一定的不确定，资本市场未来前景我觉得还明朗得多。为何？首先，老美那边美股有见顶的迹象，过去特朗普是要大力维护股市的，但是现在特朗普诉求有点差别，今年美债还要借7万亿美元左右（包括借新还旧债），美债利率超过4%，这么高的利率借7万亿财政负担很大。所以特朗普拼命折腾，想把资金从美股赶到美债市场，从而大幅度压低美债实际利率水平，同时也可以逼迫美联储降息。美股见顶，同时我们A股估值很低，所以东升西降的趋势很明显。

其次科技股这个赛道很有机会。一方面是国家大力发展，这次也写入政府工作报告，基本是明牌。另一方面，过去我们科技股估值是美股的小跟班，也就是对标美股企业对国内企业给个打个2折1折的估值；但是深度求索的横空出世，让市场发现，中国某些赛道的科技股居然还能对美股产生颠覆性效果，所以中国科技股某些赛道存在估值重估的巨大空间。

今天就写到这里。
