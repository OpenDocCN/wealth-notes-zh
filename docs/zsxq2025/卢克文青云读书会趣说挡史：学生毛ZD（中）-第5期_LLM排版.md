
# 卢克文青云读书会趣说挡史：学生毛ZD（中）-第5期

## 学生毛泽东(中):海纳百川(1915-1916)

大家好，我是杨闻哲，欢迎收听卢克文工作室的党史栏目。学生毛泽东在湖南一师这说长不长说短不短的五年半的时间，大体可以分为三个较为明显的阶段，第一个阶段就是上回说到的狂性自歇，也就是他意识到了自己不能浮躁狂妄，要沉下心来踏实做学问才能实现自己少年时期立志救国的宏愿，所以有了那篇《自讼》。基于此，才有了第二个阶段，也就是这次要说的海纳百川——不去媚上攀附，反而俯下身去走向基层。这是个重要的转变，说明他的知和行都开始往更为广阔的大本大源之处探求。这个阶段是个格局打开的阶段，不仅仅是他的社交网开始触及社会底层，同时他的眼界也开始甩开当时的精英阶层。

### 一、自我约束

学生毛泽东进入湖南一师上学后，前两年看似波澜不惊实则暗潮汹涌。由于少年时代就敏而好学，好读书求甚解，打下了良好的文化基础，所以进入学校后，咱们润之便很快地得到了老师的赏识和认可。只由于个性太强，太有主见，对学校的各种规定十分不爽，还经常无视校规，被不少老师视为狂妄自大，使得学校几次想要开除他，幸而有慧眼识英才的几位重量级教员保他，这才得以留在学校。而他自己又不想被学校束缚，苦苦挣扎于学习方法的问题，也几次想要退学。

这种痛苦纠结的状态持续了两年，到1915年上半年的这个学期，同学兼挚友的神童易昌陶的突然病逝让他悲痛欲绝几度从梦中哭醒以及新上任的校长张干特别地事儿，同时期袁世凯卖国求荣接受了日本提出的灭亡中国的二十一条引发全国众怒，在这多重负面情绪的作用下，一方面让他愈发频繁地向老师(主要是黎锦熙)请教学习方法的问题想抓紧解决自身的内部问题，另一方面又由于其少年时期就立志为国是个热血的爱国少年，袁世凯来这么一下使得他又急于找到一个情绪的宣泄点来回应外部问题。结果时任一师校长张干这时候就给了个神助攻，张干为了讨好官府，向学生们收钱，并且数额不菲，这最后一根稻草直接引发了学潮，学生们奋而团结起来准备造反，青年润之抓住机会，成功领导了学潮运动，将校长张干给开除了，但也落下一个记大过的处分。当然他自己肯定不在乎背处分，而且从杨昌济组织全校教员反对张干开除涉事学生也能看出，张干在一师混得是有多差，多不得人心。

张干其实不是个坏人，也不是没能力，看他的人生经历给人的印象就是他也想在国民党政权内好好混，但可能因为人品还可以，所以始终都没混出个模样。于是，1915年上半年的这个学期，就以学生毛泽东成功领导学生造了学校的反，轰轰烈烈地结束了。润之也从同学眼中的一个才华横溢的怪人变成了一位能开除校长的奇人，关键是老师居然都联名保他，客观上给他提供了相对宽松的成长环境。

如果事情就到此为止了，难免会让人觉得这只是一个狂妄的英才侥幸得逞，那润之自己在经历了这半年的折腾后可能也不会得到有益的成长，甚至反而会助长其狂妄的个性。可润之的过人之处就在于，他不仅没有更加狂妄，反而在暑假期间深刻地反思了自己，并在开学前写了一篇《自讼》猛烈地抨击自己，毫不留情地指出自己的缺点，让自己无地自容，于是成功地制化了自己的心性。你别说，人还得对自己下狠手，才能有所改变和成长，成大事之人，无不是对自己都狠的人。润之自此下定决心，以后不再狂妄浮夸。古人云：过而能改，善莫大焉。

### 二、格局打开

1915年上半年对润之来说就是一个命运的考验期。如果他经受住了考验，就会进入一个加速成长的时空，而如果没经受住考验，那可能来不了几次汪洋恣肆就会因为不善使用自己的力量而泯然众人乃至招来杀身之祸。学生毛泽东经受住了考验，于是命运很快给他提供了机会，引导他进一步升级乃至开挂。

1915年9月15日，新学期刚开学的的时候，远在北京的陈独秀主编的《青年杂志》(一年后改名为《新青年》)创刊。中国近代史上一次前所未有的思想解放和启蒙运动新文化运动正式拉开帷幕，鲁迅、胡适就是这时候开始走红，白话文也是这时候开始兴起的。经杨昌济介绍，毛泽东和他的好友蔡和森等人成为《新青年》的热心读者，并深受其影响。

紧接着，为了征求志同道合的朋友，学生毛泽东便以“二十八画生”之名，向长沙各校发出征友启事。启事说“愿嘤鸣以求友，敢步将伯之呼”。提出要结交能刻苦耐劳、意志坚定、随时准备为国捐躯的青年。他当时不是正好苦于在一师那个有限的小圈子里，能遇见的良师益友都太少了，想拓展圈子嘛，于是可能受到了《新青年》的启发，自己也开始全城招贴征友启事。

就相当于一百年后的我们在豆瓣上发布或者寻找同城活动的信息那样。这事在当年可新鲜了，用今天的话说就是遇见一社牛，满大街嚷嚷要和人交朋友。别说今天这种行为可能会引起非议，当年也一样，周南女中的老师就觉得这个二十八画生不怀好意，以求友之名行求偶之实。但显然，作为奇人的毛泽东当年发布同城交友广告不是像现在主要为了吃喝玩乐乃至男欢女爱，而是为了修身、救国、互相鞭策。

所以当时毛泽东择友的标准非常严苛，并且还制定了“三不谈”,即不谈金钱、不谈男女之事、不谈家务琐事。只是当时这种做法过于超前，符合要求的人则更少，所以回应者寥寥。据毛主席后来自己回忆说当时回应者只有三个，但最后在史料里留下确凿姓名的只有一个半朋友，一代风云人物罗章龙算是一个，而那“半个”则是党史上另一个赫赫有名的人物李立三。

罗章龙当时看到这个征友启事文采飞扬颇为感动，回去立刻修书一封表达交友意愿，3天后收到毛泽东的回信，二人约定在一个图书馆见面。第一次见面，两人相见恨晚，聊了两三个小时直到图书馆午休才作罢。临别时，毛泽东表示以后要和罗章龙经常见面，还说“愿结管鲍之谊。”青年润之一向择友甚严，用管仲和鲍叔牙为喻来表达心愿，可见对罗章龙的认可与看重。只是十几年后的1931年，作为中共核心领导人物之一的罗章龙在反对立三路线的斗争中受了委屈，便愤然另立中央，造成中国共产党早期的第一次大分裂，随即被开除党籍，亲手终结了自己的政治生涯，也是令人唏嘘。

所以我为啥一直觉得青年毛泽东在1915年暑假那次《自讼》特别重要呢，因为那是自我批评的开端，意味着从那时起，我们的润之便能时刻开展自我批评以保证自己始终不偏离正确的路线。若不是因为毛主席在学生时期就开始自我批评知错能改，主动约束自己的狂性，那么在以后那么多次党内斗争受委屈的时候，他是很难做到严守党的纪律的，也就不会否极泰来，一步步成为党的中流砥柱。干革命的，谁还没点狂性，但如果自己无法约束，受不了委屈，那往往就是一念之差招致毁灭性的结果。罗章龙不是个例，在他之前有陈独秀，在他之后有张国焘。

### 三、真知灼见

袁世凯复辟帝制只持续了100天，到1916年的3月22日，袁世凯便在全国的口诛笔伐中被迫取消帝制。5月29日，之前坚定拥护袁世凯称帝而在复辟期间备受宠幸的湖南将军汤芗铭，在各方面的压力和影响下，主要是在陆荣廷的护国军兵峰的逼迫下，被迫宣布湖南独立，并自封湖南都督。袁世凯得知后不停念叨“人心大变”四个字，然后仅仅过了一周，1916年的6月6日，袁世凯便一命呜呼了。

虽然没法说汤芗铭的反水直接导致袁世凯死掉，但由于这两件事挨得太近，所以汤芗铭的反水被许多人称为是给袁世凯的送终汤。当然汤芗铭其实也是假独立，只是为了拖延护国军进城的时间。这期间，谭延阁——也就是后来的湖南省长——觉得自己东山再起的时机到了，决定与汤芗铭联手，便拼凑了一支武装驰援汤芗铭，而汤芗铭却派出杀手去刺杀护国军的将领程潜以及陆荣廷，但结果都被护国军给抓获了，反而授人以柄，让护国军更加师出有名地挺进长沙。于是1916年7月4日，汤芗铭从长沙逃走。6日，程潜进入长沙。

湖南在6月底开始混战的时候，一师正好刚放暑假，毛泽东想回家却频频受到军队骚扰不能顺利回去，但仍然不顾兵荒马乱头铁地继续朝着家乡韶山前进，沿途看到汤芗铭的军队和护国军的士兵各种骚扰群众的情形。那时候的军队甭管你打着什么旗号，都跟普通老百姓没有什么关系，路上缺少啥物资就直接抢老百姓，对许多老百姓来说，去当兵也只是为了混口饭吃，哪有什么人民军队的概念。

可能正是见识过旧式军队的这种习气，润之便在心底埋下了改造军队的种子，后来1927年9月秋收起义失败后，毛主席在上井冈山前才进行了三湾改编，要打造一只区别于旧式军队的全新的革命武装，这个理念直到1929年底的古田会议时才算正式形成为我党建军的纲领。

湖南的混战算是告一段落了，汤芗铭被驱逐后，毛泽东随即返回长沙。暑假期间，润之频繁致信萧子升谈论对时局的看法，当绝大多数人都还在人云亦云地口诛笔伐汤芗铭时，还是个学生的毛泽东却认为汤芗铭被驱逐是有冤的。这一段论汤督的原始文字在《毛泽东早期文稿》这本书里有详细记载，其核心意思就是，不应该只将眼光局限于汤芗铭被袁世凯任命为湖南都督时屠杀了2万反袁人士，更应该看到汤芗铭治下的湖南社会安定和睦，路不拾遗，鸡犬无惊。

而这次护国军驱逐汤芗铭的主要动机是因为湖南处在南北必争之处，护国军从南向北进攻势必要夺取这个战略要地，但驱逐汤芗铭的理由却无非是其拥护袁世凯又杀人如麻。学生毛泽东认为，无论是拥护袁世凯还是杀人如麻，都不是汤芗铭独有的行为，当时的时局，全国其他地方的军阀很多是有过之而无不及。而当汤芗铭被驱逐离开湖南后，护国军又无力维护湖南的秩序，导致湖南社会陷入无政府状态的混乱之中，一些从前被压制的暴徒开始为非作歹滥杀无辜，淫荡之风火热，犹如法兰西恐怖时代的景象。

由此，学生毛泽东批判护国军有勇无谋目光短浅，认为成大事者一定要等待时机，要有所忍耐，为达到护国目的，不作出一些妥协和用一些铁腕手段不足以成事，而把这样做视为罪过，是那些不懂大计的蠢人。而这时候的毛泽东才多大?不到23岁。

1916年的7月25日，毛泽东在给萧子升的另一封信件中根据此前袁世凯接受的二十一条以及日俄之间的协定并结合欧洲正在进行的第一次世界大战的局势变化，清醒地看出，当时的日本无论什么人上台，对中国的灭亡瓜分政策都不会变。再三思考后得出结论：日本是中国最危险的敌人!国家应密切关注日本的举动。

这个认识甩了蒋介石不知道几条街，蒋介石直到七七事变爆发后还在与日本进行和平谈判，并幻想着英美能够管教日本。而此时的学生毛泽东已经作出了预言，说中国和日本在20年内必有一战。而现在我们都知道，中国的抗日是从1931年的9月18日开始的，从预言日1916年7月25日算起，十五年多一点，在二十年内；而全民族的抗日战争是1937年7月7日开始的，从预言日1916年7月25日算起，也不到21年。

如果润之论汤督让我们看到当时不满23岁还是个学生的毛泽东对于时事的看法已经能够做到全面掌握情况，清晰分析问题，头脑冷峻、思想深入，绝不人云亦云随波逐流，那他在随后所论的国际局面的观点则已经展现出一个卓越政治家的思辨和格局。
