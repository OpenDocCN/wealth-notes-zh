
# 中长期资金入市，道阻且长

现在的股市也只能指望国家队救了，而且还得是非一般的国家队。从9月24日瞬间拉起之后，横盘震荡4个月，经济会议说2025年要稳住楼市股市，结果一开年就大跌，其实原因很简单，横盘太久，没有足够的增量资金进入，自然就会一路下跌，其他如特朗普冲击、汇率贬值等因素都是次要的。要想根本解决，就必须让A股真正形成长期投资，但在牛短熊长的A股，说长期那都是笑话。最坚持长期主义的也就是广大散户了，好多人都是2019年公募这一波入市的，都已经快6年了，有几个散户赚钱了？在动荡不断的A股，说长期实在不现实。

在这一背景下，心心念念的中长期资金入市，终于有眉目了。1月22日六部门联合印发《关于推动中长期资金入市工作的实施方案》，次日国新办举行发布会，与9月24日类似，改在9点举行，包括证监一把手在内五部门部级官员出席，显然是想影响股市走势。从9点开始，股市也确实冲了一小会，但很快就像泄了气的皮球。

中长期资金入市仍然很难，不过是刚刚开始。这件事最早被提起，是2019年，当时还是易主席拿出了12项资本市场改革政策，他一直对美国401K条款支持下养老资金入市非常羡慕，希望推动中国公共养老金入市。之后易主席领导的证监系统也一直在努力，尤其是2022年到2023年初，努力了一整年，大会小会不停地提，而当融资端注册制改革基本完成后，对投资端的中长期资金入市关注度就更高了。但问题在于中长期资金中，只有公募资金归证监管，然而公募跟散户也差不多，玩的也是快进快出，干的也不是中长期资金的活。其他中长期资金不归证监管，对这些资金而言，入市的前提是资金安全，而不是冒险。于是就形成了悖论，证监说中长期资金入市才能让股市变好，而中长期资金说只有股市变好它们才会入市，就卡着了。

易主席最后一次努力是2023年7月中央定调“要活跃资本市场，提振投资者信心”，随即组织召开了全国社保基金理事会和部分大型银行保险机构主要负责人座谈会，想要再推一把，然而没有成功。很快易主席就被免，吴主席上任，开始组织救市，一开始重点是加强监管，整顿发行、交易、退市制度，4月新国九条提出构建支持“长钱长投”的政策体系后，重心逐步向中长期资金入市转移，到8月组织上任以来的第二轮座谈会，邀请包括全国社保基金、保险资管、银行理财、私募基金等10家头部机构投资者，开始落实构建“长钱长投”的政策体系。

但还是老问题，在股市低迷的时候，的确是中长期资金入市的契机，但问题是资金缺乏安全感，不敢用身家性命给证监抬轿子。除了公募基金以外，算得上中长期资金的还有五类，针对不同资金，证监计划采取的措施不同。私募基金：支持稳健发展，提升投资稳定性；保险资金：优化投资政策环境，完善绩效考核、投资运作等制度环境，这一块也是证监特别关注的；社保基金与基本养老保险：只说要完善投资政策，具体怎么完善，没有令人信服的章程；企业年金与个人养老金：表示提升投资灵活度；银行理财和信托资金：表示鼓励参与。

从上述可以看到，即使在资本市场得到最高层关注的情况下，证监能够调动的力量仍然很有限，也就是公私募基金还比较有把握，其他要么是企业年金这种一直没搞起来的，要么就是社保、养老、保险这类非常谨慎的资金，短期内就算有政策出来，也都是小政策，影响不了大局。

梳理一遍后，也就能理解本周中长期资金入市影响有限了。当然，在这个关键问题上能够有所进步，这当然是积极的，但意义更多在长期，短期指望靠这项政策拉动股市，是不现实的。

关键还是资金。本周中长期资金入市，各类资金都提到了：一是商业保险和权益类公私募资金，提到要对国有商业保险公司经营业绩实施长周期考核，还要提高权益类基金规模和占比；二是社保基金、基本养老保险基金、企业(职业年金基金入市；三是加大回购和分红。

真正能带来短期增量资金的也只有第一类的商业保险和权益类公私募资金，这块归属金管总局和证监管理，这两家目前协调起来相对更容易，如2023年底在协调央地责任时，就主动提出兜底。

近些年保险资金投资权益类资产的余额虽然在上升，但占比从2015年股灾后基本没变，一直在12%-13%之间波动。而文件说“力争大型国有保险银行从2025年期每年新增保费的30%用于投资股”，但“力争”也就意味着并非一定能实现，这每年几千亿增量资金到底能落实多少还要看股市走势。

最有希望的还是要求“公募基金持有A股流通市值未来三年每年至少增长10%”。力度还是不小的，因为最近三年远达不到10%，2022年增速是-12.32%，2023年4.52%，2024年1.56%，按照2024年底持有5.87万亿市值的基数测算，如果能完成10%的增速目标，那么接下来三年将分别提供5900亿、6400亿、7100亿的增量资金。

但问题是2019年公募潮后进入的散户们，大多数还没解套，还没进入市场的散户们也都“精明”了，公募的名声也差了，公募手里头掌握的弹药不足想要实现这种级别的增长，还是有难度的。

证监真正期待的还是第二类社保基金、基本养老保险基金。但从本周发布会来看，参会的财政、人社副部长表态并不积极，一个说推动有条件的进一步扩大，另一个说搞长周期业绩考核机制，并没有明确的具体措施。而且财政还说不会修改当前社保基金股债投资四六开的比例，这就意味着，如果想要实现更多社保基金资金入市，只能靠做大社保基金的盘子，也就是各地增加基本养老保险基金对社保基金的委托投资。根据人社数据，截至2024年底，委托投资规模是2.3万亿，而且每年投资运营增速很平均。

从2019年的0.99万亿到2023年的1.86万亿，每年也就增长2千亿左右，2024年才有较大增速，也只有4400亿，2025年大概率也就是2024年的增量，乘以40%的股市投资比，能够给股市提供的增量资金不到2000亿。

至于企业年金就更别指望了，2024年企业年金制度已经施行20年了，设立仅比公积金制度晚5年，但直到现在仍然没有解决众多基础性问题。比如不同于公积金的企业缴费和个人缴费都直接进入个人账户，企业年金的企业账户和个人账户分开，而企业账户资金发放就有太多操作空间了，于是同一家企业领导跟普通员工，企业年金发放差距能达几十倍，就算2017年修订要求控制在5倍差距内，但职工个人仍然说了不算。

除了两个账户资金分配的核心问题外，还有很多具体操作的障碍，比如一家公司想要设立企业年金，多给员工些福利，但设立的要求很高、流程也很复杂，于是能搞企业年金的基本是国企以及少数外企，20年了一直喊扩大企业年金范围，把养老的第二支柱搞实，但到2023年也只有3144万职工参加企业年金，领取年金的更是只有308万人。直到2023年末，20年积累，企业年金规模才首破3万亿元，达到3.19万亿元。

总之，六类中长期资金，短期内或者说2025年能够期待的增量资金，也只有公私募和部分商业保险资金，理想情况下也只能提供万亿的增量资金而且还要解决公私募快进快出的问题。相比股每天万亿的交易量，影响有限。

在这种情况下，不宜高估中长期资金入市对股市的影响。
