
# 嫁给喜欢的？还是合适的？

看到这个题目，读者们应该猜到了，我要聊婚姻话题，但这只是今天的三个话题之一。之所以采用这个标题，是因为真的有读者这么问我，她的原话就是：嫁给喜欢的？还是合适的。可我这里的嫁，是广义的。我今天要聊的不只有婚姻，我更要聊职业，投资。你嫁人也好，娶媳妇也罢，都是嫁给了一段婚姻。你选择A公司，B行业，C专业，D城市，都是嫁给了一段工作履历，一段求学履历。你选择了房子，黄金还是证券，都是嫁给了一段投资经历。这一切都是嫁，广义的嫁，人这一辈子，无论老人小孩，男人女人，你都跳不出广义的嫁，尤其是在这样一个重要且敏感的时刻，众所周知，三月初就要到了。

## 拆解喜欢与合适

喜欢？什么是喜欢？或者说，什么是你定义的世界里的喜欢？朱丽叶喜欢罗密欧叫喜欢呢？还是潘金莲喜欢西门庆叫喜欢？不要躲避这个问题，不要因为成见，上来就说罗密欧是好人，潘金莲是坏人，我赞美罗密欧，我鄙视潘金莲，所以我当然把自己代入到罗密欧身上。别这样。25年前，在学生寝室宿舍楼里，卧谈会，大家讨论喜欢哪种类型的女生，是黄蓉，穆念慈，小龙女，小昭，双儿，赵敏......

人会骗别人，也会骗自己，但数据会骗人么？大家沉默了。你嘴可以骗人，心可以骗人，无意识下，未曾掩盖的行为痕迹，难道也可以骗人么？

拆解完那个属于你自己的独特的喜欢，接下来做什么？拆解合适。合适？什么是合适？或者说，什么是你定义的世界里的合适？你觉得合适总要有个理由吧，这个定义是谁给你的？没有人会无缘无故的觉得合适，所有的合适都是暗示。

你去看男生的裤子，不是蓝的就是灰的，不是灰的就是黑的。以至于某一天，我儿子在店里选了一条白裤子，我和我太太都很惊讶。男款的裤子也有白的么？一问店员还真有，学生款的。接下来我们就觉得这个选择很奇怪，白色的裤子，怎么搭上衣呢？我们从小学过的美学搭配原则都是重色系在下面，你去看你们家，墙是浅色的，护墙板是深色的。如果反过来叫什么？叫头重脚轻。不信你用白色的护墙板，黑色的乳胶漆，你看看什么效果。同理，白色的裤子，上面搭黑色的外套，这不是很奇怪么？但那天，我儿子就那么选了一套衣服，黑色的上衣，白色的裤子。

## 环境与适合

我在想，为什么？想来想去，就一个原因，那就是他还没有被驯化。我在他那个岁数时，有没有属于自己的穿衣想法？我想应该有。只是常年的驯化之后，我已经全然忘记了，我只记得三种颜色，黑，灰，蓝，无论内衣还是外套，无论上衣还是裤子亦或者鞋子。因为我觉得，只有这三种颜色跟我是合适的，而这种所谓的我觉得，其实是驯化的结果。一个男生，一个女生，刚出生的时候，不可能有这些念头的。没有人会觉得这个就适合我，那个才适合她。所有的适合都是被教出来的，根据你的性别，会教你不同的适合定义，根据你的环境，你周围的人，以及你各方面的条件，也会被教给你所谓的适合的定义。

那么问题就来了，你所在的环境下，你被教授的适合，你认么？不同的人是不一样的，有些人与外部环境很融洽，大家怎么看他，和他怎么看自己，是一致的。爸妈觉得读书适合你，你像张飞一样，说，俺也一样。媳妇觉得赚钱适合你，你像张飞一样，说，俺也一样。这就叫一致。

有些人是闹别扭的，爸妈觉得你适合走读书的路，你说，最讨厌一种鸟，它自己飞不起来，下个蛋，老催别人飞。

你看，每个人所处的那个环境，或者说那个小世界里被定义的合适是不同的，而他自己是否认同，也是不同的。

到现在为止，我都没有回答你的问题，我只是反问你。你以为我问完了？没有，我刚开始。即便你明确了自己的喜欢到底是什么，它也有个问题，那就是你喜欢多久？喜欢一天也叫喜欢，晚上想想千条路，早上起来走老路。你不能否认那一晚，那千条路也是喜欢过的，只是短了点。喜欢一辈子也叫喜欢。那么同样，即便你认为合适，也有合适多久的问题。你以为《蜗居》里的海藻与小贝就没有过合适？你以为海萍和苏淳就没有过合适？你以为宋思明和姜淼淼就没有过合适？你以为只有兄弟是可以重定义的？不，合适也是可以重定义的。因为人是会变的。

## 容错度与选择

一个不明定义的喜欢，VS 一个不明定义的合适。一段不知长短的喜欢，VS 一段不知长短的合适。劳烦你告诉我，你是喜欢呢？还是合适呢？被我问倒了对吧？明明是来提问的，结果答案没拿到，带了一肚子的疑惑回去，回去干嘛？你是准备用AI把人类已知的喜欢的定义，长短，合适的定义，长短，都检索出来，挨个学习么？那就是诸葛亮舌战群儒时骂的话：春秋作赋，皓首穷经；笔下虽有千言，胸中实无一策。东吴儒生们真的有错么？并没有。他们就是古代版的人工AI，他们遵循的就是最传统的被训练多年的解决问题的通用方法。

就是就着问题去检索，无非东吴儒生在图使馆里检索，20多年前的学生们有了搜索引擎，不去图书馆，也能跨国检索。而如今，有了AI，检索的整理，归纳工作都帮你做好了。可是，这些都是什么问题？都是简单问题，都是可重复的问题。但是东吴面对什么问题？复杂问题，一个不可重复的问题，一个有关历史进程的问题。书上是没有答案的，要你自己来决定，所以诸葛亮才怒了，说：大人们在议事，你们这群AI实习生，都出去。定下大方针，要你干活的时候，你再来。

## 升维思考

双方之所以能打起来，恰恰因为彼此的答案不同，所谓你信的，我不信。有人卖了房子去买股，有人清仓了股票去买楼，他们在交易大厅里擦肩而过，互道一声：S内什么。才会怎么样？才会成交嘛。所以诸葛亮是在告诉东吴儒生们，你们要的那个打不打的答案，是检索不来的，是需要我和周瑜，去给你们创造的。如果你想要提前知道，除非有个更高维度的人，直接把你拉到空中，让你俯视大地。或者，像我当年琢磨赚钱这件事一样，最后，自己悟到了。

回到这个问题，嫁给喜欢的？还是合适的？这个问题真正的答案是什么？是一个指标，容错度。有的人，和80%的异性结婚，都会幸福；有的人，和10%的异性结婚，会幸福；而有的人，一定要遇到几亿分之一的极端喜欢，极端合适的真命天子天女，才会幸福。就像有的人，他做绝大部分品种，用绝大部分交易系统，最后都能稳定盈利，无非赢多赢少。而有的人，只有特殊品种，特殊环境，极端运气的情况下，才能赚到钱。如果你沿着前者的道路去研究，你迟早会稳定盈利，因为你走在科学的体系上。这个过程就像现代医学的诞生，你会看到治愈率不断提升，数据不停的给你反馈，你的盈利曲线是正的，而且越来越平稳。如果你沿着后者的道路去研究，你最后实际上变成民科了，你所谓的喜不喜欢，合不合适，就跟那帮研究彩票号码的人，是一样的。

## 嫁给不讨厌的

所以说，当你还无法了解自己真正的心意时，嫁给谁？嫁给不讨厌的。假如这个人，没钱，没貌，剥夺对方身上所有优秀的地方，剩下的，你依然不讨厌，依然愿意聊聊天，做个普通朋友。那这就是选项之一。无论男女，不要搞什么一白遮百丑，一赚掩所有。选择另一半，要在剥夺一切优点之后自己仍然不讨厌的范围内选。

## 职场选择

前一个话题，我们选取了一个读者的问题，以此为基点展开，直到推理出选择的底层逻辑。那么第二个话题，我想要聊职场，也同样选取一个读者的问题，以此为基点展开。春节档期的时候，我写哪吒的影评，我说，太乙就像名校生，不是师兄要罩他，是那个一损俱损，一荣俱荣的体系在罩他。申公豹就像个非名校步入职场的，活都是你干的，锅都是你背的，回头分仙丹（期权），人家一年分9000个，你600年，分6个。于是就有读者留言感慨，学历果然是王牌。

我看到了，当时想反驳，我想告诉这个读者，学历不是王牌，它是一张很重要的牌。重要到如果你会用，那它就是王牌，如果你不会用，那实际上也等于没牌。甚至，有些时候，还不如那个没牌却目标明确，知道自己有什么，要什么，愿意放弃什么的申公豹同学。只可惜当时没有例子，干巴巴的说出来大家也难以理解，凑巧，现成的案例送来了。
