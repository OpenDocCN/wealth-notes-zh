
# 特朗普是民企的恩人
# 发布时间：2025年5月21日

特朗普的贸易战与关税战不是坏事，对于中国一批优秀的企业家来说，是天赐的良机与上天送来的礼物。凡是被特朗普视为对手的企业和行业，都会获得一种“神圣的庇护”，能使得他们免受除了特朗普以外的所有攻击。而中国优秀的企业和企业家，并不怕特朗普商业化的明面攻击，他们怕的是隐性的非市场化壁垒和政策性打击，单纯拼效率中国人不会输给任何敌手。

譬如因李某某事件一度被舆论围剿的华为，凭借扛过特朗普的围堵，成为了民族英雄，手机成为了政府官员标配，垄断了高端市场，汽车绕过层层隐形壁垒，一路弯道超车……

半导体行业在科创板的支持下高歌猛进，国家大基金获得一连串的注资，政府采购被要求使用国产品牌与系统，军工科技项目一个接一个的上马，不符合条件的ant甚至可以披荆斩棘的申报IPO……

随着特朗普输掉了2020年大选，美国对中国一系列的限制和打击逐步取消后，我们也开始冷静下来，开始了算账，Jack和Pony遭遇了反垄断，大基金遭遇了一系列的清查，军工集团遭遇了严厉的反腐……

所以，特朗普2.0的回归对于企业家不是坏事儿，民营经济促进法加速出台了，平台经济大佬重回民营经济座谈会，高利润军工订单开始恢复了，国产替代与采购又被再次重视了，民营企业的在1.0时代的好日子又要回来了。为了跟美国正面对抗，特朗普要搞AI，我们就会给AI独角兽开绿灯，特朗普要发展半导体，我们就会斥巨资扶持，特朗普要斥巨资搞北美防御系统，我们也会加大军费开支，特朗普要遏制中国新质生产力融资，我们就会解决融资通道。

这四年里，企业只要能承受特朗普带来的市场冲击，凭借产品力和创新正面击败竞争对手就可以，就再也不必担忧过多的非市场化的限制与冲击。
