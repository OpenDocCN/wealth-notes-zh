
# 美团为骑手缴纳五险一金的背后

## 嘉宾供稿：梦回安西

2月19日，美团突然宣布，将为骑手缴纳五险一金。说实话，这是出人意料的，前面几年都不乐意，怎么这会又乐意了呢？很多人说，是看到京东加入竞争，美团急了。然而，京东才进来还没多少市场份额，一时半会无法产生威胁。再说，除了京东，抖音早就和顺丰合作介入外卖配送了，咋不见美团急呢？所以，京东的加入不可能是主要原因。事情的真实现状还得从零工经济之困说起。

通过这些年互联网平台的发展造就了大量的零工，其中美团是最具代表性的，因为美团有超过700万的骑手是最大的零工经济平台。虽然相对自由，收入也还可以，但骑手们长期以来都被手停口停所困扰，你干一单就得一单，底薪很难获得（全职骑手的比例很低），更不存在五险一金，这意味着缺乏保障并且常常是全年无休。如果算平均时薪，其实不算高。

对于要求缴纳五险一金的呼声，美团最大的理由就是自己和骑手不存在雇佣关系，一口咬定双方只是商业上的合作关系，平台不是雇主。既然你不是我雇的，自然就不用帮你交五险一金，连底薪都不用给。

为了避免和骑手构成雇佣关系，美团甚至会要求骑手注册个体户，再以个体户的形式接单，这样一来便可坐实商业合作关系，相当恶心人。这种心态的背后是美团试图把自己定位为一个企业，而不是一种基础设施。如果只是一个企业那么只需要专注于赚钱即可，其他事情不用管，只要不明显违法。如果是一种基础设施就得负担更多的责任，比如电网，你不能仅仅赚发电的钱还有义务保障居民用电，遇到大雪压垮了沿途线路你有责任迅速抢修。所以电网赚得多，但也是重资产的，利润并不高。从美团的角度它肯定希望自己是企业即可，不希望被认为是基础设施，这样责任就少得多，利润会得多，一般的企业可以这样认为，然而美团坚决不行。原因无他，美团事实上是一种基础设施。毕竟在现实中，许多商家的业务就是依赖外卖平台开展的，很多人的生活也和外卖平台息息相关。而美团又是外卖平台中的绝对垄断者，占了八成以上的全国市场份额。美团一个政策的变动就有可能影响数十万人的饭碗。说得夸张点对于许多商家来说，美团平台就是个“线上迷你版政府”。这种情况下美团能说自己仅仅是一个企业么？当然不能。既然你是一种基础设施，是“线上迷你版政府”，履行更多的责任与义务就是应该的，让你给骑手缴纳五险一金并无不妥之处。

当然我们也知道美团主观上是不乐意的，2022年被反垄断罚款后，美团就声称正在研究给骑手交五险一金，研究了两年也没个下文，直到2025年2月19日。问题来了，发生了什么呢？西方有一种“企业国家队”模式，也就是被纳入“国家队”范畴的企业，比如波音、洛马等，这些企业的员工不仅待遇更高，且通常不会被开除或者说很难开除。即使要开除，给的遣散费也高于平均水平。这些“国家队”企业的员工及其上下游供应链，构成了西方中产的主力，是社会的中坚。而纳入“国家队”的标准除了看科技，还看是否会对社会产生重大影响。比如沃尔玛，作为一个超市，谈不上科技含量，但沃尔玛毫无疑问是“美国企业国家队”的一员，总统出访时其高层经常会跟随。这是因为在美国许多地区，沃尔玛真就是基础设施，是贫民窟的稳定器。入选“国家队”的企业，会得到更多稳定的订单，并有政治上的一系列待遇。作为交换“国家队”企业不能轻易大规模裁员，要给员工相对高的待遇促进社会稳定，隔壁的日韩实际上也有类似的模式。所以我们发现日本部分大企业，是终身雇佣制的，员工很难开除。而咱们国家未来大概率采取类似的模式，美团作为基础设施显然有必要纳入。因此，2月17日召开的座谈会王兴是出席了的，而六年前那一次王兴可没出席，说明美团的地位得到提升，同时在短期内不太可能再次被反垄断罚款了。有交换筹码，才有动力去给骑手交五险一金。

当然，就算交也不可能全部骑手交，2023年美团的骑手有735多万，这是个很庞大的数字，全部交五险一金，确实超出了财力范畴。735万里，有许多人并非活跃的存在。比如，不少骑手是过渡性的，跑了一段时间，找到别的工作就放弃了，又比如，不少人仅仅是下班后跑一下，白天有别的工作。以上情况，交五险一金显然也不现实。可以预见，缴纳的核心将是两种骑手：全职骑手和长期兼职骑手，全职骑手美团有83万左右，长期兼职的数量缺乏准确的统计，但150万-200万是肯定有的。两者的数量相加，在230万-280万这个区间，约占美团目前骑手数量的35%-40%，给这几百万人逐步上五险一金是可行的操作。按照美团的计划，将从2025年二季度开始逐步推。

如果这一步能完成，零工平台将出现重大转变。别的不说，就说外卖行业，当美团缴纳后，你饿了么好意思不缴纳？那就别怪核心骑手跑路。当美团和饿了么都缴了，当外卖行业完成整顿，相关经验势必推广到其他行业，比如即时零售、快递等等，倒逼其提高员工待遇更加合规地经营。

只不过在喜的背后，忧也不能忽视，几百万骑手一年五险一金的费用，光是企业部分就是一笔巨大的支出，美团也无力全部支付，势必转移一部分。这意味着点外卖的价格，大概率要上涨。
