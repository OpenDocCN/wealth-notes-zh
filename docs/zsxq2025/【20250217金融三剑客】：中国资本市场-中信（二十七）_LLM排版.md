
# 中国资本市场-中信（二十七）

## 一、见效

各位在电视剧里看到的那些地下工作的桥段，就是根据刘副这个思路来的(表面上是某某掌柜的，其实是gcd某报社据点)。话说刘副的这个战术思路一箭双雕，在实际运用中效果很好，一下子解决了白区工作的两大痛点(穷/危)，再加上此前教员提出的统一战线和新民主主义革命的指导思想的指引，白区一下子就崛起了，各项工作开展的红红火火，直接助推了全国高涨的团结抗日，也间接推动了历史进程。

so，讲到这，各位就清楚了，刘副曾经能成为教员指定的第一接班人，并且被全d都接受与认可，是有其深层次原因的，其历史功绩也是有目共睹的。

除此以外，还有一层深刻原因比较隐晦(拉拢增量选票)，当时内部红白两区的斗争矛盾已经凸显，gcd方面在抗日胜利后初期，还想着和国民党和平建国的统一方案。教员的想法是：如果谈的好，gcd可以放下枪杆子，咱们进入国民议会和国民党打议会战争，争取人民的选票。这样一来的话，显然白区干部的重任更大(拉拢敌占区的选票)，是未来的希望(增量)所在(不打仗，红区干部和思想就用不大上了)。

## 二、体制

关于选票政治，很多读者感受不深。只是着美国那边每过一段时间来一轮。在最近春节期间上映的贺岁片里，你能看到他们美国人曾经对华人的欺压，甚至还动用了法律的手段。

但美国是一个新大陆，全世界去了很多个种族，除了华人，他们还针对哪个群体，专门制定过排斥法案呢？日本人。二战期间，罗斯福专门搞了一个法案，把10万多日裔美国人，送进了拘留所。

另外，在20世纪的不同时期，有针对穆斯林、南欧、东欧等地区，短暂执行过移民配额。但像对华人，搞的这么绝么？完全没有。

话说来了那么多个种族，对于新大陆来说，都是新人，别人不针对，单单针对华人，如果你只是从饮食、文化、偏好等表象层面去看，必然很肤浅。

其实在美国这个大熔炉里，德国人、英国人、法国人、爱尔兰人、意大利人、犹太人……大家在一起生活，也是相互看不上的。但并不妨碍他们，关起门来各自过自己的小日子。

但如果有一天，他们突然团结起来，被投机政客煽动起来，集中对付某一个群体，那一定是这个群体，伤害了他们共同的利益(比如他们之前说的：都是中国人抢了你们的饭碗)。

打个比方你就懂，你们公司原先有100个人，原本内部同事之间(各个国家移民之间)也是勾心斗角，相互看不上，有各种矛盾争端。但突然大老板，从外面新招来了10个工人。这10个人，都是从某地(东方某大国)来的，没日没夜的拼命干活，工作时长比你们多，工资还只要你的一半。

试问，你作为老同事，会欢迎他们么？肯定不会。你只想把他们，全都赶走，别在这里卷。

当年美国铁路热时，白人一天干8个小时，华工一天干12个小时；白人工资一个月50美元，华工只要26美元。

换你做老板，你会喜欢哪种员工？但你作为同事，你肯定不喜欢他们。

好，这个时候体制的差异就来了。

如果公司里，是大老板(集权专制)说了算，那就不会出现排斥卷王员工的惨案(会不停的玩骑兵迭代，发挥鲢鱼效应)；但如果员工有投票权，那拼命卷的员工，就会受到大家一致的排斥(人家只是想赚轻松体面的钱)。

这就是核心差异，人家员工有投票权，想过安生的好日子，不想和你卷自我迭代，卷利润。

## 三、生态

当全d都在教员的指引下，以为未来不是在战场(打内战)而是在议会中和国民党斗的情形下，此时白区干部出身就拥有着明显的优势。

当时全d认定的第一第二接班人，刘副和彭zhen(前面讲过的，重庆李庄案的后台老板)，这两个都是政工干部出身，这也是在内外部因素结合下做出的全d选择。

后面我们知道，重庆谈判没谈拢，内战也就不可避免。gcd在三年时间里如摧古拉朽之势解放了全中国。这时d内生态就一下子大变样了。

就又回到白区占优的情况，因为建国后不怎么打仗了，以经济建设为主，就是前面讲过的，学习苏联为主，这个时候，自然要重用知识水平更高的行政官僚，刘陈彭高等一批白区干部当仁不让就上位了。

简单小结下就是，在建d初期是白区一骑绝尘(搞学运工运)。南昌起义后，是红区蒸蒸日上(战略转型武装斗争)，白区日渐边缘。在延安之后，是白区迎头赶上(教员战略-刘副战术奏效)。在建国初期，则是白区大权独揽(经济建设为主)，红区干部养老归田。

但问题是，军队干部虽然是泥腿子出身，但天下曾经是这帮泥腿子打下来的，所以红区干部非常不满意，打天下是我们，坐天下是你们，这就建国初期的主要矛盾。

## 四、冲突

刘副同志相对彭和林来讲，最大的问题就是他在d内一直很难得到普遍支持，因为其一直在白区工作，几乎没有任何可以压过其他同志的战功，在军队里更是没有支持者。

在第一代领导人中，毛朱周自不必说，登在军中都有不小威望，陈yun同志又无意往上走，那么在七大上成为实际二号人物(教员赴重庆谈判期间，他代理主息职务)的刘副同志，建国后便很麻烦。你想想，高麻子为啥自始至终都不服他?

并且，他也一直不擅长农村工作，建国前土改中也偏"左";建国前后主张"富农剥削有理"的也有他，简单摘录一些便知：

"1，什么叫做剥削?现在必须要有剥削，要欢迎剥削。工人在要求资本家剥削，不剥削就不能生活，剥削是救人，今年关里大批难民到东北去，关外的富农能剥削他，他就会谢天谢地。过去每年有一百多万劳动力到东北去，若富农不剥削便能生活。

2，d员成为富农，其党籍怎么办?这个问题提得过早了。有剥削也还是可以做社会主义者的，例如雇难民去做工，不能叫做剥削，圣西门是一个资本家，但他也是一个社会主义者，虽然当时是空想的，假如今天东北有个资本家要照圣西门的办法去做，那么他可能办成功，而不垮台。凡是雇人劳动，扩大生产买马开油房增加了社会财富，不能叫做剥削。

现在是私有制的社会，d员生产发家了，要交公也交不出去，国家也不会要他的油房要他的马，不得已他只有暂时的私有，如果他现在发展了生产，将来在实行集体时，又能……"

显然，这些思想都是与教员相逆的。他这些思想之后又被延续到大跃进运动中。

在一五时期，我们优先发展重工业，这在一定程度上牺牲了农民阶级的利益，教员由此点明可能发生的农业危机和贫富分化。

因此教员主张虽然还要像过去一样优先发展重工业，但必须增加对轻工业和农业发展的投资；除利用和发展先进大城市工业外，还必须发展沿海与偏远地区工业；除大规模城市工业化外，还必须转向农村的中小工业；除中央部门的指导外，还必须要扩大地方zf因地制宜发展经济的自主权。

但这些举措引发了刘副同志的不满，“反冒进”就这样堂而皇之的登上了台面。1957年9月召开的八届三中全会上，教员对周、陈yun在刘副等人支持下提出的在1956年经济建设中进行“反冒进”的决策提出批评。

从1958年1月的南宁会议起，教员在中央召开的系列重要会议上，对1956年的“反冒进”批判日渐升温，认为“反冒进”泄了六亿人的气，和资产阶级右派只是五十步与百步之差，明确以后不能再提“反冒进”的口号，只能反右倾保守。

那刘周两个人在1956年搞的这个“反冒进”，实际上真的如字面意思的“反对激进”吗?非也，刘副的目的 是想让官僚机构进一步扩张、让专业化阶层与官僚阶层进一步巩固、让现代化中的城市化进程加快(即城乡二元体制)。这也是“包办代替主义”思想的体现。

so，为了巩固自己的地位，刘副在建国后，将自己老部下们纷纷授以高位，同时主动向老精英们(旧国民党公务员、知识分子、资本家)示好(如“剥削有功”论，如新民主主义的主张在干)。

嘛?

他在努力促成地下d系统+政工系统+传统上层精英的合流。当然，这些举措这也激化了与军队工农干部的矛盾。

即社会力量处于强势的军队干部在建国以后被边缘化，社会力量处于弱势的政工地下党干部却掌握大权，这也是建国后三十年时间里政局不稳的最重要的原因之一。
