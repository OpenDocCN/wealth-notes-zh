
# 嫁给独生子好，还是嫁给家里有兄弟姐妹的更好？

## 前言

主要对未来老公家里有兄弟姐妹的情况做一个大致阐述，以免单身女孩后悔。本篇已婚人士别看，男性别看，有恋爱对象的也别看。本篇主要写给单身女孩的，也就是已婚人士请绕道，有对象的女孩，请绕道，所有男性，请绕道。

女孩挑对象，这是为了谈恋爱，本篇讲的是找老公，两者并不一样，我们讲的是要登记结婚的。也就是你要跟他结婚并嫁过去，接下来大概率要生娃的，此后你跟他组建家庭、成家立业、双方父母成为亲家，你的姐姐他也跟着叫姐姐，你的阿姨和叔叔，他也跟着叫阿姨和叔叔。你俩的财产要公开、透明、合二为一。

其中一方婚前买房了，那另一方就没有了首套名额，买房首付款和房贷利率都会有所增加，购房套数也会被对方占用。后面你跟他一起将你们的孩子哺育、长大、毕业，直到你们的孩子成家立业又生了娃，你俩才算大致交差。

你要一起逢人客去、共同面对身边的所有事情，保不准因为异地和异心其中一方另寻新欢，也要一起承担上有老下有小的压力，包括其中一个甚至俩同时失业而导致的贫贱夫妻百事哀。

如果有不能克服的困难，则你俩会离婚，还要分割财产，有时房子不能卖，卖了两人包括孩子都没地方住，所以离婚后还住在一起，但明明已经是熟悉的陌生人，经济和亲子关系还要纠缠在一起。

其中协议离婚占据1/3，打官司离婚也有1/3概率，剩下的就是一辈子要离婚但是没能离得掉的，甚至有人将离婚作为毕生的唯一追求。

以上这个流水账过程，是提醒你，一别总记得或者憧憬被对方爱慕并一起浪漫约会吃饭看电影的短暂时光，后面长期的不浪漫几十年在等着你；二尤为重要的，挑结婚对象更要看人品、性格、身体健康、感情谈得来，绝不是说找啥“有车有房、父母双亡”的，你有没有想过，你41岁时他去世了你怎么办？

找老公的第二考察点，才是我之前说过的那些还让你打分的十来个要素：学历、年龄、老家所在城市、身高、长相、体重、兄弟姐妹、父母感情、收入、资产等。

至于星座、暖男、温文尔雅、玉树凌风、会修电脑、过去恋爱次数、对方父母工作情况、有几个兄弟姐妹、工作执行力强不强、鞋子里垫不垫鞋垫、先放屁不脱裤子还是先脱裤子再放屁，等等，这些都属于第三诉求，赶不上刚刚第一第二的权重，更不影响婚姻稳定或者幸福的大局。

所以本篇所讲“老公的兄弟姐妹情况”，虽然你婚后会发现也是很有切身体会的地方，但其对婚姻稳定的重要性，并没有那么大。属于次要因素之一，你心里有数即可。

我曾经提及过家里兄弟姐妹情况的打分，在总分100分里面最多占据10分。换位思考，用男孩找对象来打分，有哥哥的女孩无非比独生女高出个一两分，独生女继续比有弟弟妹妹的稍微高个小几分而已，这1-3%的悬殊不影响全貌。

## 一，找老婆，是找家里有姐姐的好，还是有个妹妹的好?

就光这个点，就已经可以单独成文，只不过这只是今天我们全文的一个引子而已。大家设身处地，假设你是个单身男孩，你找老丈人家的大女儿还是小女儿?

古往今来，尤其在亚洲，生儿育女主要诉求不过是养儿防老，所以从子女的角度来讲，要有在父母年老后的抚养责任，责任就是压力。对方家总共就俩女儿，你找了人家一个女儿做老婆，当然要承担一半的养老送终的责任。

但你也是有一定的好处的，那就是“6个钱包”。现在的生活压力大，你的孩子买房等重大开支事项，需要集合你们夫妻俩人、自己的父母、配偶的父母这6个钱包的力气。你老丈人丈母娘有俩女儿，所以你好歹能得到1个钱包，也就是你老婆父母的一半财产/家产。

我们都是有良心的，假设养老送终的过程中我们至少承担一半的成本，那后面纠结的娶姐姐还是娶妹妹，无非就是看老婆家产你获得大于50%还是比一半少一点点，就这么点差别。

所以，接下来就简单了，根据统计规律，父母一般会护犊子，也就是会更倾向于小女儿，所以正常来说，父母寿命正常能活到七八十岁的，会在两个女儿都成年后将自己的财产的大概40%给大女儿，留下大概60%给小女儿。

姐姐和妹妹相差出来的这大概20%财产分配的悬殊，就是你看到本节标题时在意的的东西。你自己评估吧，你老丈人家产200万，类似你多获得40万，或者少得到了40万。当然，少得的主要是姐姐，但父母也会瞒着给妹妹，不让大女儿尤其大女婿发现。

以上是一般规律，但是，也有特例。爷爷奶奶、外公外婆正常都更喜欢大孙女，所以大孙女会从祖辈那里将父母的偏心“补”回来。搞了半天，你找大女儿也不算亏，还有祖辈。只有你老婆的爷爷奶奶外公外婆都去世了或者毫无经济力道，你找姐和妹才有那20%的差别，否则都差不多。

下一个特例又来了，因为父母也不是一碗水都端平的，有些夫妻本来就一直吵架，吵到老的。正常来说，父亲喜欢大女儿多一点，母亲则喜欢小女儿多一点。他俩都健在，那还好说，他们可以达成平衡。

但是，在大女儿还没出嫁时，父母去世了，这就难办了，尤其大姑娘还读研了的。本科妹妹就希望获得绝大多数家庭财产，母亲本来就一直跟大女儿有“矛盾”，更喜欢小女儿，所以，在外表、内心和潜意识里，父亲去世后的大女儿，会比较惨。

不但因为没有父亲在而嫁给真正高富帅的概率陡然下降了，而且在家产方面母亲和妹妹都不大想让她继承太多。古往今来的故事，都是恨不得将她嫁给中山狼、矮锉穷甚至贾府里的焦大， 一辈子翻不了身就只能仰仗母亲尤其妹妹的施舍。用性格和命运来说，大家看下《红楼梦》中的贾迎春。在母亲强势尤其父亲去世早的家庭，大女儿更多是贾迎春。

以上是一般规律，假设再增加一个条件，则情况又截然相反，比如这俩姊妹再来个弟弟，那就完全不一样了，大女儿能得到大概10%家产，二女儿最惨，5%有时都没有，弟弟获得接近90%的家产。

所以，现实中，家里有姐姐有弟弟的女孩，也就是站在弟弟角度叫“二姐”的这位姐姐，会相对大姐“狡猾”了很多，因为她从小得不到家庭重视，就要争取资源、学会哭、诉苦，甚至赶在姐姐前将自己嫁了、带走嫁妆、嫁到外省。这是她们能想到的最大化自己的利益而最小化也就是逃避责任的一种方法，很多是潜意识，感受和体会，而不是她们真的去谋划过或者“聪明”。

当然，考虑兄弟姐妹情况，无非就是考虑，一，分到多少家产(5分),二，少承担多少义务(5分)。假如其他一切要素都相同的情况下，姊妹俩中，你找姐姐，只分到了5分的家产中的2分，承担赡养义务时妹妹逃得快、姐姐不愿意逃或者要承担更多，算5分中的2分吧；你找妹妹，则家产分到3分，逃避责任有3分。

所以，满分10分，找姐姐做老婆算4分(2+2),找妹妹做老婆6分(3+3)。找独生女呢?财产5分全得，下一个逃无可逃0分，加起来5分。

搞了半天，居然是找滑头妹妹更实惠一点，找傻大姐最苦，独生女居中。

## 二，你未来老公是独生子，会怎么样?

独生子，很显然，社会中的所谓妈宝男，主要就是独生子。这还不是武断，这是他们的成长环境决定的，你是个母亲，你也会这么宠着唯一的儿子。在他小时候你宠着，不让他做事情，培养成了小皇帝，这是你的爱子之心，等他大了后，你还是会宠着、疼爱他。

实际上父母尤其母亲从小将儿子培养成妈宝男，是有一定的目的性的，无论将来妈妈跟谁发生了矛盾，儿子都会不假思索地倾向于母亲这一边。

每一个妈妈都是用心良苦的，她的人生需要对付的“敌人”太多，包括丈夫、公婆、老公的情人、可能的小三上位、未来的儿媳妇、社会地位等等，她在成为妇女之后，如果亲爹不给力，是不会有干爹或者其他支持的，除了可以靠着儿子翻身。

可以说，从儿子的降临、儿子成绩好、帅气有魅力、甩开女同事的子女十八条街、将后妈比下去、考上好大学、工作好、女性追求者众多、事业有成、婚后婆媳之争向着妈妈、再生孙子孙女延续生命，等等，每一个环节，女性都指望儿子能给自己翻身至少不败的支撑。

即使儿子没啥出息，但只要他听妈妈的话，就是胜利。有人说，儿子是父亲活下去的支柱，不对，父亲还有其他出路，比如事业、第二春、钱、官衔和江山，但儿子是母亲活下去的唯一支柱。

说了半天，就是告诉你，你找独生子，要有一个心里准备，他的母亲已经大概率将他培养成了绝对支持她的人，确保在你60岁之前也就是她85岁的时候，你的老公都更听她的话而不是你的。
