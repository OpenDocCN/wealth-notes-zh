
# 稳定币对旧有国际金融秩序的影响

稳定币虽然和比特币、以太坊一样，都是基于区块链技术的加密货币，去中心化、不可篡改等特征都有，但本质上还不是一回事。最大的区别是稳定币不像其他虚拟币那样价格波动剧烈，顾名思义它很稳定，和法定货币是1:1锚定。所以，它不是用来炒的，而是用来花的。当然我们也可以简单理解为它是法定货币和虚拟币的一种杂交产物。

那么，为什么会出现这种杂交产物呢？这就要从它的前世今生说起。实际上稳定币并不是一个新东西，早在2014年就诞生了，其主要的目的就是给链上虚拟货币交易者提供一种更加便捷安全的变现渠道。

因为在它出道之前，比特币、以太坊等虚拟货币的交易其实非常原始，要么就是面对面手转账手转币，要么就是找个靠谱的平台，双方在平台的监督下进行相关投资交易。但不管哪种形式，虚拟货币的链上交易始终绕不开线下转账这个环节。因为毕竟虚拟货币在现实社会支付应用场景几乎为0，在虚拟货币价值波动非常大的状态下，不可能期望用比特币去买个汉堡，然后叫商家找补法定货币。

而且，既然是炒币，那么优先想的就是落袋为安，交易换来一堆筹码和有法定货币背书的稳定币是两码事。总结就是，币圈炒币的人需要稳定币来充当虚拟货币交易的中间商，去解决交易不方便、不稳妥等痛点。

稳定币从2014年出来后，并没有大范围得到应用，这个也可以理解，一个新物种的入市，需要经历一个漫长的摸索和试错的过程。比如之前硅谷银行暴雷，就差点连带着把稳定币USDC发行方Circle带坑里了，因为Circle有部分资金就放在该银行，硅谷银行暴雷直接导致恐慌挤兑，一度让一枚USDC的价格从1美元脱锚至0.87美元。甚至有些稳定币发行公司为了追求更高收益，选择不挂钩现金，而是锚定有价证券、企业债券等，这也为稳定币带来很高的安全风险。

但不管稳定币初期发展过程中踩了哪些坑，它还是凭借其交易便捷性、相对安全性，整体上得到了认可。到目前，稳定币的市值已经在2500亿美元上下，跨境交易结算达到34万亿美元，超过Visa＋万事卡的总和。

照说，稳定币创新初衷其实就是虚拟货币交易的一种衍生产品，为啥发展到现在，成功破圈了呢？核心逻辑就是稳定币的用户群体在快速壮大。除了币圈投资交易者以外，稳定币还受到哪些群体青睐呢？

第一是高通胀国家的居民，比如阿根廷、土耳其等国，这些国家通胀高的已经甚至超过100%了，为了避免收入缩水，肯定愿意将本币兑换成稳定币。

第二是做灰产黑产的群体，这个大家都懂，加密货币是他们洗钱的天堂，只不过随着监管加强，这块空间也在压缩，毕竟虽然交易加密，但账户是有主儿的。

第三是做跨境电商的公司，因为传统跨境汇款手续费达5%-10%，稳定币转账成本仅0.1%-0.5%，且分钟到账，这块对于中小企业而言诱惑力大大了。

所以，在面对这么一块超级蛋糕的时候，比如稳定币发行商，先不谈国家，就说摩根这样的金融巨头、科技巨头、各种加密公司、特朗普家族的政商联盟，谁不动心才怪。只要拿到发行牌照，就相当于成了第二个美联储，拥有数字货币的铸币权。

比如拿1亿美元投资短期美债，就可以发行1亿个稳定币，用户兑换稳定币又没有利息支出，属于0成本吸储，发行商不仅投资美债赚4-5%的收益，还可以收取交易费。

到这里，其实大家多少对稳定币有个初步印象了，对于我们普通人而言，只要没有参与跨境交易、国内也不存在高通胀、也不参与币圈交易甚至黑产灰产，那么稳定币其实和我们没有毛线关系，有钱干嘛不存银行或者直接投资国债，即安全又有收益。

回到大国叙事的方向。为啥特朗普政府要去推动《天才法案》的落地呢？首先，虽然现在全球绝大部分的稳定币锚定的法币是美元，但是有些发行商并不归老美管，因为公司注册地在其它国家。既然稳定币锚定的是美元，发行商搞借鸡生蛋，而且规模越来越大，老美肯定不会坐视不管，必须立法把鸡先圈起来。

《天才法案》，其核心就告诉稳定币发行商两点：其一，发行稳定币必须持有等值的储备资产，范围就是美元、活期存款和短期美债。其二，发行商必须落户美国，而且需要获得政府许可，也就是发行牌照。

其次，个人认为这背后至少还藏着特朗普的三重算计：削弱美联储的地位。说穿了，给稳定币一个合法合规的身份，本质上就是创美联储的根，不仅变相分流铸币权，如果将来上规模了，美联储的货币政策有效性都将极大受影响。大家都知道，现在特朗普和美联储正在就降息问题撕逼，推动法案落地，就是在给美联储上强度，你不听话，我就要给你放血。

培植壮大自己的金融势力。美联储虽然名义上是美国央行，但其股权结构自1913年成立起便由私人银行联盟主导。背后站的是华尔街金融世家，而华尔街和民主党的全球化利益政策致，所以要论两党和美联储的关系，民主党肯定更胜筹。未来在稳定币的牌照颁发上，向科技公司、算法公司倾斜，就是在培植和美联储抗衡的新势力。

显而易见，特朗普家族就是币圈的玩家，搞这出，就是捞钱。不管美元、美债将来会怎么样，先把钱捞了再说，毕竟稳定币本质上就是钱生钱的无本买卖。

再回到文章开头的问题，稳定币会当美债的接盘侠吗？毕竟天才法案要求储备短期美债。

我的结论是不会，为啥？其从规模上看，现在稳定币大致在2500亿美元的规模，根据渣打的预测，到2030年稳定币规模会达2万亿美元。乐观点，就算规模达到5万亿美元，并且全部买美债，也就是现在36万亿美元规模的1/7。按老美现在的尿性，美债规模不可能缩水，未来只能更加庞大。

其二，稳定币因为要满足即时兑付的问题，所以它对储备的底层资产流动性要求很高，它只能选择短期美债。而老美压力山大的是中长期美债，所以在结构上是错配关系，当不了接盘美债的主力，更可能扮演的是吸血虫的角色。再换个思路，如果钱都通过稳定币去买短期美债了，长期美债是不是更会失血。
