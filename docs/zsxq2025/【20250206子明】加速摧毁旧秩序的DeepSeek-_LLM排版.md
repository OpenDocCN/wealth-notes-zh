
# 加速摧毁旧秩序的DeepSeek

每一次的工业革命，都需要打碎一个旧秩序，建立一个新秩序。每一次工业革命准备进入中段，开始加速的标志，都是成本的大幅降低。而其标志意义也在于此。

## DeepSeek

第1次工业革命，英国法国需要打开亚洲的市场，肢解中国和印度的封建体系。

第2次工业革命，德日、美苏需要打开全球市场，肢解英法的殖民体系。

第3次工业革命，美国与西欧需要打开社会主义市场，肢解苏联主导的共运体系。

现在是第4次工业革命，大家就要合力打碎第3次工业革命中建立的中美全球贸易体系。本质上大家都是要根据生产力的变迁，建立一种新型的自由贸易体制，只是，由于每一次的工业产品不同，自由的方式与自由的地点也不尽相同。

对于被打开国门的中国来说，第1次工业革命是贸易港，是香港与五口通商的广厦福甬沪。第2次工业革命是经济特区，是浦东与计划单列市的深厦甬青连。第3次工业革命是互联网平台公司，是数字经济特区的深杭京沪。这一轮DeepSeek 代表的Al开源，放到历史上的生态位是：第一次工业革命的自由贸易，第二次工业革命的自由迁移，第三次工业革命的Internet互联网的互联互通。

而每一次的工业革命，都要通过庞大的中国市场与劳动力来降低工业成本，来加速实现工业革命。但凭借着中国强大的内生力，西方每一次使用中国加速，也都造成了对西方巨大的反作用。鸦片战争后的五十年后，西方要立法禁止中国的鸦片出口。侵华战争的五十年后，西方要立法禁止中国的纺织业出口。改革开放的五十年后，西方要立法对中国的全体贸易品加关税。此次的突破只是个开始。未来一批中国Al量产的新模式，不仅实现了工业革命中段必须要进行的降成本，实现了中国在资本主义社会中那不可或缺的位置，还提前实现了反向倾销，并逼迫着西方在降成本前，不得不提前进行限制。而这也标志着人工智能时代的正在加速到来。

## 宣传地位的时代变化

中宣部的地位变化，一直以来都跟中国的内外政策息息相关。改革开放之前，中宣部长一直都是最核心的岗位，肩负着与国民党、苏联和美国，乃至左右的论战。仲、陆、康、陶、耿、胡xun dingyi sheng zhu biao yaobang，每个人都是那个时代的风暴眼。但是79年改开之后，不谈主义，闷声发展经济。从王、邓到朱、王忍之。renzhong liqun, zehou，这些宣传一把手，就不再成为政治局委员。直到89年的那件大事儿之后，上面发现不控制好笔杆子要坏大事，对于宣传的逻辑来了一个U转，伴随着92南巡和新一轮改开，丁guangen 以政治局委员的身份担起了重任。到了2001年，中国加入WT0，要进一步打开市场，与西方进行文化碰撞，吸取了上一轮的改革经验，次年直接让文宣一把手的文明办主任入常，进一步加重了宣传的地位。

此后数届，中宣部长在书记处的排序也常年排名第二（第一是常委），一直到十九大。
