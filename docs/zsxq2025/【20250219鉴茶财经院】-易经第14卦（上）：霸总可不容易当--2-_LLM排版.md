
# 易经第14卦（上）：霸总可不容易当

会赚钱很难，会花钱更难。赚钱的时候经常会碰到各种巧合，有时候靠勤劳，靠机遇，嗖的一下就起来了。花钱可非常考验人的布局和规划能力，反映的是你原生家庭的烙印，是你的环境影响、认知系统、决策模式的总和。

以前网上有个段子说，屌丝千万别写富豪穿越小说，否则，写没钱生活的时候非常真实，苦的刻骨铭心，等有钱了，只会在苦日子的基础上加倍。

比如，"霸总一震杉杉西服的袖口，露出刚买的卡西欧手表”霸总在五星酒店大堂，耐心帮女主撕开餐具的一次性包装，挨个用热水烫了烫，霸总到了车库，一按车钥匙，各种吉利、奇瑞、宝马汽车闪着灯乱叫。

跟当年有些农民军造反一样，你问它想干啥，等打进了北京城，顿顿吃饺子，天天过大年。很多企业，到了一定阶段，就发展不动了，业绩死活上不去，有些人，身价到了一定阶段，也上不去了，玻璃天花板跟明显，根本原因，不是你不会赚钱，而是在小有钱之后，不会花钱。

"小的时候没有布局，将来哪来的“大有？

BATJM 发展到今天，百度明显落后，还是李彦宏会赚钱，但不会花钱，舍不得。陆奇要烧自动驾驶，他一下怕了。马老师有问题的时候，一帮人保他，它最终能够再回来，阿里技术的底色也没有掉多少，老师是出了名的会花钱。

08年就敢利润一分钱不要算砸了云计算，你是个人才就敢给高薪，超出你的要求和期望，魄力非一般人能及。大强子后来居上，会赚钱也会花钱。

有钱的时候，顺风顺水的时候，要好好规划排兵布阵，从容思考，才能从一个胜利走向另一个胜利。这就是易经第14卦，火天大有。

卦辞说，大有。元亨。象曰：火在天上，大有。君子以遏恶扬善，顺天休命。

上卦为离，代表火，火在天上，象征着光明和热烈，寓意着事物发展的态势很好，大有所成，大有收获。君子要止恶扬善，不是要你去当侠客，而是能够打破对自我发展不利的框架和思维，主动拥抱那些对自己长期有利的，潮流、主线，趋势，这才叫顺天应人。

把事情做到从0到1，需卦讲过了，很难。把1做到100，更难，这正是“大有要告诉我们的策略。

大有就是什么都有，如何都有，下篇分解。
