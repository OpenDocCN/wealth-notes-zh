
# 学习笔记

## 学习笔记，2025.1.24

1. 我走ETC通道的原则是，前面只要有车，我就等待20米以上，一方面防止蹭卡，一方面若是前面ETC失效，需要倒的时候，我不需要动。
2. 一个人，若是不能建立自己的客户数据库，那么越做越累，相反，若是有自己的客户数据库呢？越做越轻松。尤其是朋友圈的存在。可以做到大宝天天见。
3. 你有没有发现，微信里70%的人是不联系的，衣柜里70%的衣服也是不穿的，汽车里70%的速度是多余的，别墅里70%的面积也是不住的，人这一辈子赚的钱，70%也不是花给自己的。所以人生不要太用力，也不要太在意，守好自己的30%就可以了。
4. 选择与一个经济条件困难的人结婚，你不得不为他的无知、债务和困境买单，同时也要面对他所带来的各种不幸、错误的观念。不仅如此，你还得承担他家庭中的无知、懈念和混乱。好的婚姻过命，坏的婚姻要命，比起物质上的贫穷，心穷的男人，更致命。
5. 家长不要轻易怼学校怼老师，现在的学校只做人才筛选，不做人才培养，80%的教育责任靠家长自己。
6. 小时候父母掏出口袋的钞票，一眼就能看出他们的窘迫，如今，我们拿出手机扫码付款的时候，孩子却以为里面有用不完的钱。
7. 好好赚钱就是最好的修行，还分享给过她一段冷知识：只有赚钱才会重启人生。而早起读书，背单词，做笔记，瑜伽，锻炼，健身，都跟赚钱半毛钱关系都没有。只是看起来很“自律”，其实，大部分人初中高中时代都比这“自律”。
8. 体制内的人，99%都或多或少有类似的病，他从来都觉得自己是贵族，而老板们呢？是矮自己一层的，老板们对自己好，是应该的。他们对拿对占，永远是那四个字：心安理得。第一，老板们恭维的话，其实都是哄小孩，谁当真谁傻，但是体制内的人，往往当了真。第二，在老板眼里，那就是一个傻逼。他觉得，老板们都在跪舔自己。其实，在老板眼里，他啥都不是！
9. 要经营长在我们自己身上的技能或魅力，与角色无关，与职位无关，这样的好处是，随着我们的年龄与阅历的增长，我们是越来越有魅力的人，而不会出现所谓的断崖式。我为什么重新拾起了日记写作？就是，我原以为，我写不写，都是王者，大家都依然那么尊重我，结果我不写了，大家就不怎么搭理我了，使我产生了巨大的落差。
10. 曾经无知的认为高楼耸立，遍地名车才是发达国家，现在深知：安居乐业，食品安全，公平自由，法律保护弱者，老有所养，教育先进，活得有尊严，才是真正发达的国家。
