
# 首富启示录：大清第一灾星

在历史正剧大流行的时代，有一部《康熙王朝》是很火的，那首主题曲"还想再活五百年"，到今天仍有播放。康熙死于1722年，距今才刚过300年，如果按照电视剧里的意思，我们距离走出康熙时代还有200年。天呐，多恐怖的事情。因为按照科学家的说法，一个人的脑袋在50岁就基本固化了，50岁之后学一些新姿势小技巧还勉强，想有观念上的刷新会十分艰难。尤其是，对于高高在上上面没人的康熙帝来讲，不可能再有人帮他做观念上的刷新。

康熙帝说的，已经是完全正确。让他在完全正确的道路上狂奔，窃以为500年是撑不到的。所以能成康熙，旁边有个好宰辅是很重要的。就像在《康熙王朝》电视剧里，宰辅之一纳兰明珠过六十大寿，康熙亲笔写下祝寿匾额：亮辅良弼。这是最高的礼遇。亮，指的是诸葛亮，辅佐刘备，不仅帮刘备拿主意，更有一番隆中对，完全刷新了草鞋刘的认知。良，指的是张良，运筹帷幄之中，帮流氓刘邦灭项羽收群雄，一统江湖。在诸葛亮和张良面前，刘备或刘邦是一点都不敢坚持己见的。韩信打下山东，派使者来求封"假齐王"，刘邦一听火冒三丈，破口大骂才两句，张良从后面一脚踢到刘邦的屁股根。刘邦又疼又惊赶紧改口当什么假齐王，要当就当真齐王，韩信就该封齐王，我完全同意。

亮辅良弼是对一个宰辅的最高赞誉，与之相对，一句"老当差的"，则是最强诋毁。凡是看过一些非标历史回忆录的，都该知道这句“老当差的“是出自一位副统领之口来评价一位国之辅弼。而在更遥远无须讳言的历史里，也有一位自视甚高者盛宣怀如此评价他的老恩公——李鸿章。

说李鸿章是个"老当差的"，盛宣怀是有理由的。作为洋务派干将，大清航运、纺织、通信、铁路、新式教育以及红十字会等诸多产业的奠基人和操盘手，盛宣怀十分看不上下李鸿章对慈禧太后一路尾随不敢违拗的样子。"她就是个大门不出爱听戏的老太太，懂个P。你李鸿章游历世界各国什么都懂，干嘛事事都要向她请示，她放一句P你就屁颠颠跑断腿，真天生一个贱骨头，老当差。"

在讲究辈分和出身的大清朝，盛宣怀这样说李鸿章是很不对的。因为李鸿章首先是他长辈，是他父亲的朋友。其次盛宣怀得到重用，成为“官督商办“第一经理人，完全靠的是李鸿章提拔。眼看李鸿章已经七十多岁老眼昏花没几天活头了，却仍放不下对慈禧的敬畏，盛宣怀很不服气。尤其当义和拳闹起来，慈禧利令智昏率领拳民对十一国宣战，电召各地总督率兵勤王的时候，盛宣怀命令各地电报局将慈禧的诏旨扣压，力劝李鸿章别理会慈禧，私下里与列强签订《东南互保条约》。更进一步与各督抚达成默契，一旦慈禧完蛋，共同推举李鸿章出任大总统以接管大局。

盛宣怀是敢等着慈禧完蛋的，但当差一辈子的李鸿章不敢，当京城被列强攻破，慈禧仓惶逃到西安并急电李鸿章赴京和谈的时候，他还是坐不住了，迈着77岁的残躯奔去给慈禧收拾残局。临行之前，李鸿章要盛宣怀一起去，他拒绝了。望着老恩公踉踉跄跄登船北上的身影，除了心里送上一句“老当差的”，盛宣怀不由握握拳头。李鸿章行将就木，他盛宣怀掌大局的时代要来临了。

人都是短视的，因为人的脑袋长在屁股上，脑中的观念被屁股决定着。这方面，一位跨世纪的宰辅曾说过，他在上海任职的时候，就想着让上面给政策放权限尤其不要动辄干涉。等他到了上面，立刻反过来，又一心想着怎样收回权限让地方服服帖帖。盛宣怀也是跨世纪的。在20世纪之前，他作为李鸿章的左膀右臂，十分愤怒于以慈禧为太后的清中枢干涉。修段铁路，说蒸汽机轰鸣惊动龙脉，修好的铁轨被扒掉；架设电线，说电流传播会摄人魂魄，严禁电线杆子进京；即便是烂到极致早该淘汰的漕运制度，也一直拖拖延延无法废除，理由很多，荒唐可笑：如果粮食不走大运河改走海运，遇到列强封锁海岸线怎么办?那满城贵胄岂不要饿肚子?把粮食牢牢掌握在自己手里，大运河这项法宝不能丢。

说出来你可能不信，直到1903年，清朝的漕运制度才正式废除，大运河的纤夫编制终于裁撤，粮食改由海运和铁路运输到京。1903年，李鸿章已挂，盛宣怀也已调入京城，加盟中枢。屁股挪地方了，步入到20世纪的盛宣怀，观念就立刻变了。他积极谋划，主动进取，想方设法帮清中枢收权。这是很讽刺的，因为就在几年前，他还串联各地方不要听从慈禧指示，搞东南互保，是此后数十年军阀割据的始作俑者。

收权，盛宣怀是专业的，因为他最懂在一个现代国家里哪些权力最重要。当那些满清贵胄忙着遛鸟斗鸡为谁家福晋能陪太后听戏吵个不停的时候，盛宣怀建言：当今之计，要先把铁路收归国有。大清虽大，四通八达，只要掌握铁路，平时可以收税，钱不够花了就加税添费。遇到潮暖环境，哪里生出乱子，有铁路运兵，朝发夕至。

确实，在此后很长一段时期里，谁掌握住那几条铁路大动脉，谁就掌握着神州主导权。即便是没有能力掌握了，也要想方设法破坏，不让对手舒服享用。花园口决堤因此而起，铁道游击队也是这个意思。这里要提一嘴1938年的黄河花园口决堤，直接受灾人数逾千万，死亡数老蒋当局说是10万+，那肯定是不可信的。我问了一下chatgpt，说学界主流估计89万。问deepseek，说在数十万至百万之间。我不知道哪个数可信，但假如万一某一天郑州方面要建一座纪念碑的话，上面的数字将震撼不了谁。因为这口是老蒋下令扒开的，老蒋本质上算自己人，扒口子是为一致对外，让黄河水破坏掉京汉铁路，阻止日军从华北直插武汉。这里的“京汉铁路”，就是盛宣怀主持修建的，也是清王朝修建完毕的唯一纵贯南北的干线，全长2500余里。

此路完工，慈禧大喜，自觉江山稳固了许多。京城的兵开到武汉，只需一天一夜；武汉是军工重镇，既有汉阳兵工厂，又有武昌军械库，可以辐射整个西南。慈禧在的时候，尚且谨慎；慈禧一挂，袁世凯被免，新上的隆裕太后和摄政王除了安插满清贵胄，汉人里面最器重的就是盛宣怀，他被任命为邮传部大臣。

邮传部可不是邮政局。在清朝末年，邮传部总管邮政、船政、铁路、通信事务，掌握大清经济命脉，堪称半个宰辅。盛宣怀要大显身手了。他上台后做的第一件事就是把原先由地方自建的铁路收归中枢。这一措施自然会受到地方壬绅的抵制，但盛宣怀不怕，有了京汉铁路，枪把子朝发夕至，于是他把实施重点首先放在了两湖和四川。

后面的事情我们都知道了，一向岁月静好的四川，闹出了保路运动。湖北的兵被调入四川，武汉空虚。几个接受了新思想的士兵一呼百应，连夜攻陷了武昌城内的军械库。此事发生在10月10日，到了10月26日，已成众矢之的的盛宣怀被革职，他自觉不妙，连夜坐上船一溜烟去了日本。

华夏文明，博大精深，有些道理是放之四海皆准的。盛宣怀是个无比精明的人物，办企业的好手，虽然顶着官督商办的名号，不实际拥有股份排不进富豪榜，但论手里可掌控的财富，他才是晚清首富，远超胡雪岩和张謇。他当督办的招商局，只用几年时间便在激烈的航运竞争中占得上风，并一举收购美资的旗昌洋行，成为行业第一。

这就像是李嘉诚，通过华人特有的勤奋和低成本，在市场竞争中击败香港的英资，并通过收购英资第二大洋行和记黄埔，接管了香港经济半壁。

但无论是盛宣怀还是李嘉诚，当手握垄断权后，表现均堪称灾难。盛宣怀直接搞没了大清，上面已经说清楚了。李嘉诚呢?他在上世纪80年代登顶香港首富后，开启香港炒楼时代。房价飞涨，产业空心化，年轻人翻身变得越来越难，直到沉浸于靡靡之音，去向英资缅怀。

宰辅当用读书人，莫让商人掌命脉。

这句话，真应该翻译成得体的英文，让陷入川普时代的美国人看一看。
