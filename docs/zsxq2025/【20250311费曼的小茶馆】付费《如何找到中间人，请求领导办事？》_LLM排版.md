
# 如何找到中间人，请求领导办事？

中间人是提拔资源的联系人，找到中间人，是提拔的第一步。全文5900字，说透找中间人联系领导办事的方法。无论是职务提拔、还是工作调动、职级晋升，以及遇到其他各种公事私事需要找领导帮忙，如果自己有关系和人脉是最好的，但很多时候，绝大多数人是没有第一手资源可用的，尤其是有些事到临头火烧眉毛必须着急行动的事情，只能通过现有人际交往圈来找关系和领导搭上线，这个时候如何找到合适的中间人？如何通过中间人找到领导？找到领导以后如何进行沟通联系把事情办好？具体和领导沟通联系的话术是什么？求人办事的具体开口套路说法是什么？求办事前前后后如何体现出高水平的人为处世能力，如何说如何做才能得体应对？这些都是绝对的稀缺内容。全部手把手教你具体语言话术和操作，绝对的干货。

## 如何理解办事要找中间人的逻辑？

首先要理解找中间人的本质。比如我孩子要读书，我直接找到管事的领导，比如教育局局长，他肯定能够把事情干成，但是他不熟悉你，也不信任你，更无法接纳你，所以多少人想读优质学校，多少人家长排队送钱他都不会答应。反过来说，教育局局长能够解决你孩子入学难的问题，但是他不知道你的存在，不知道你的需求；也可能他知道你的需求，但是你没有向他表达足够的诚意；也可能你向他表达了足够的诚意，但他很难说服自己爽快放心地收下。所以这个时候就体现出中间人的价值，中间人的重要价值就是帮领导筛选客户，帮助客户顺利安全地和领导见面沟通。

## 找到合适的中间人？

第一种情况是本单位内部找关系。比如有些事情是本单位一把手可以决定，比如你个人的本单位内部换岗、个人职务职级变动、提拔进步问题，你想突击找领导帮忙，或者想和领导先搭上话再长线经营关系，但你和一把手并不熟悉，这个时候分管领导能够说得上话，那么分管领导就是我们的中间人，你请分管领导给你帮忙，只要他愿意帮忙，你的事情就相对好办了。

第二种情况是跨单位领域找关系。有些事情不是你本单位领导能够解决的，需要找外单位一把手或者外单位其他领导帮忙才能搞定，甚至你都不知道应该找谁搞定合适，这个时候你就要去找本单位的领导探探口风，问问他看看是否认识人能够搞定，如果可以那么就需要我们找本单位的领导作为中间人。

第三种情况家族关系有认识。再有些比如一些领导，比如我们有长辈、有亲戚、有朋友，有原生家庭的关系，可以和领导搭上话，那么这些原生家庭关系的长辈、亲戚、朋友就可以作为我们的中间人，我们可以通过他们来谈谈口风，了解情况，用他们的人情请领导出来，或者先咨询咨询领导，这是找中间人的四种情况。

## 找到中间人以后如何和领导沟通联系？

找到中间人以后，如何和领导联系，这里也是要分两种情况。第一种是中间人直接带你去的。这种情况，比如因为工作调动的事情，比如因为子女上学的事情，比如因为家里人做生意或者其他个人私事等等。这些情况下间人直接带你去的话，这件事情就比较简单。中间人肯定会事先给领导打好招呼，至少是电话沟通清楚，你们才有见面的可能。说明只要领导答应见面，就说明中间人关系够铁，同时这个事情领导也不会觉得麻烦，他只要愿意见你，这件事情就应该能够搞定。至于他愿不愿意帮忙，就看你见面领导时候的表现。所以一定要提前做足以下功课，完成以下内容。

第一个功课，就是要和中间人先商量清楚，见到领导以后你说什么。究竟是你说为主，中间人在旁边敲边鼓，还是说中间人说为主，你在旁边作为辅助。中间怎么说，说什么，这些情况、角色、表现都要和中间人提前商量好，让他来教你怎么做。

第二个功课就是是要和中间人商量好给什么“心意”，具体给多少，也要咨询清楚中间人。你可以和中间人说：“领导这边心意您觉得大概给多少合适？需要不需要请领导吃饭？还是哪天直接去拜访的时候直接表达？具体打点多少等等。”这些要和中间人商量清楚。当然，千万别忘了要感谢中间人，这个要提前说清楚，毕竟都是中间人的面子。

这些功课做足了以后，接下来就是中间人带你到领导家以后的 具体表现了，这里的注意事项就是，前期沟通的话术一定要流利地表达出来，同时到领导家以后，一定要有眼力劲，首先点头哈腰，给领导请安问好，主动点头陪笑，甚至包括给领导倒茶续水，在服务工作上反客为主，让领导看到你的礼貌，让领导看到你的眼力劲。同时说话的时候要表现出求人办事时候该有的样子。比如领导直接问你具体情况的时候，你可以说：“我有一个不成熟的想法，如何如何，我自己想能不能够这样，不知道行不行，不知道可不可以。” “我的想法大概就是这些，肯定还不太成熟，所以要领导多批评指正，多给我指指路。”“这件事情大概就是这种情况，我觉得不知道应该怎么处理，所以今天也是请领导指点指点，有劳您费心了。”要把这些话术说出来，表现出谦卑谦和的姿态，不要把话说得太死，这样双方都有余地，领导也会觉得你讲规矩、明事理。

第二种情况就是中间人说给领导打个招呼，喊你自己去找领导。如果是这种情况，那么中间人帮你提前打好招呼的话，那么你就要根据中间人的要求，抓紧时间和领导联系。先给领导打个电话，具体约一个时间，是去领导办公室还是领导家里，或者找其他地方坐一坐，先和领导约时间。约好时间见到领导以后，具体话术如下：

首先，客套寒暄，“某某领导您好啊，今天很高兴见到领导啊，幸会幸会。是某某领导（中间人），喊我和您联系。” 首先是客套寒暄，给领导问好，点头哈腰。接着自我介绍，自己是谁，今年多大年纪，具体负责哪些工作等等，让领导对你有一个初步的印象。自我介绍完以后，给领导笑一笑，甚至可以拍拍领导马屁。然后，领导可能给你说两句开玩笑的话，缓解缓解气氛，喊你坐下，喊你喝茶，给你倒水，这样就把人际关系距离慢慢地拉近了。接下来聊天的时候就交代一下你和中间人的关系。比如说：“我是某某局长（中间人）的手下，我在他手下主要做什么什么工作。可以简单介绍一下你的成绩，体现一下你的价值。然后说：‘某某领导也对我很关照，也非常认可我。并且他也跟我说了您为人非常热心，对年轻人也特别关照，我也是听了他的话，才找到您这里的。’这段话的意思，就是大概介绍一下你和中间人之间的关系背景，也顺道介绍一下你自己的成绩，让他对你有一个良好的印象，同时也熟悉你、拉近距离的一种方式。”

前面简短的寒暄介绍完了，基本上前期的铺垫和破冰之旅就结束了，也就是人际关系废话铺垫阶段结束了。接下来我们就进入正题，这个时候你就不要再老去扯你和中间人的关系了，因为简单介绍过就够了，免得让人以为你是拿关系去压别人，如果给人造成这样的感觉就非常不好了。前期铺垫结束了，你就应该开门见山地说：“某某领导，今天过来呢，是自己有一些想法，有一些情况需要跟您汇报一下。” 接着就把你个人的情况讲清楚，遇到的麻烦和困难讲一下，接着提出你的要求和想法。提要求和想法的时候，要说：“像这样的情况呢，我个人有个不成熟的想法，我想如何如何如何，不知道您看这样妥当不妥当？也想聆听一下您的教诲，您看指条路，我这个方法如果不行的话，您觉得您觉得怎么做合适？我想多听听您的意见建议。”把这些情况说完，基本上他肯定就知道情况了，你的想法对不对，你的思路、你的诉求，他能不能满足。如果你的诉求不合理，那么他这个时候也会考虑给你指一条路。总之这个时候他就会给你答复了。这个时候，你就要根据他的答复，记住他的答复，按照他的答复抓紧去办，把该落实的落实。同时也要记得回头马上把相关情况汇报给中间人，让中间人给你参谋参谋。因为对于不太会办事情的人，没有这方面经验的年轻人，也很可能有你还没有听懂领导的弦外之音，也许会错过一些重要信息。所以必须和中间人全盘托出，一起商量参谋一下，避免信息差造成误会。

第三种情况就是家里的长辈、亲戚打过招呼的这种情况。这种长辈亲戚打过招呼的情况，哪怕长辈是大领导，比如你大伯是市局局长，你现在在某某区局，你大伯和区局局长打过电话、打过招呼了。那么这种时候，你同样需要对区局局长讲清楚，表明想法，适当感谢。因为大领导有大领导的面子，小领导有小领导的面子。千万不能让小领导觉得你是拿大领导的权势去压他，或者拿长辈的辈分去压他。一定要表现出长辈也只是中间人，关键是你会事，你的为人处事到位，让小领导开心，觉得你这个人值得交往，所以他才给你帮忙，一定让他产生这样的感觉。千万不要觉得有长辈、有亲戚给你打了招呼，你就可以无所谓了。你要知道，长辈都是会退休的，他们这个时候给你仙人指路，为你搭桥铺路，但是这条路未来走得怎么样，能不能把它越走越宽，就是看你懂事不懂得事了。哪怕小领导看在你家大领导亲戚的面上，必须要帮你办，你也要给人家的足够脸面。只是说因为有大领导亲戚这样的关系，那么你办事的“价位”可能低一点，比如给些高档烟酒就够了，就不需要送现金了，但是你一定要去送。就是长辈给你指路，但是路要你自己去铺，后续的关系要你自己去维系。否则小领导这一次帮你忙了，后续的事情怎么办呢？要形成家里长辈亲戚为你指路铺路，后续的路你就自己走通，以后有问题有麻烦就不需要劳烦家里的长辈，你自己就能把关系给建立起来，要达到这样的效果。现在很多年轻人毕竟刚出社会，没有经验，加上从小学习压力比较大，没有深度参与社会，所以被保护得比较多，对人情世故不懂，以为长辈亲戚打过招呼就无所谓了。记住，该走的流程，我们一定要走。

第四种情况就是通过中间人办事的几点注意事项，避雷指南和后续事宜。

## 学会感谢中间人

你要知道你能联系上大领导，把事情办好，根本原因在于中间人的牵线搭桥。大领导给的是中间人面子，不是给你这个小白面子。所以一定要把中间人维护好，尽可能把中间人的资源慢慢承接过来。我这里上个月都遇到一个年轻人，通过中间人办完事以后，一个劲地想办法要感谢领导，要和领导建立关系，却忘记了要对中间人的感谢。情况弄得非常尴尬，因为他对大领导过度热情，大领导不接招，然后他又忘了给中间人合适到位的感谢，中间人也觉得这个人做人不好。他还没有意识到这个问题，当他告诉我为什么之前都好好的，结果事情办完以后，怎么大领导也不太愿意理他，中间人这里态度也不好，还问我什么原因。当我把情况讲给他以后，他才恍然大悟，说原来自己怎么这么傻。这样的大学忌，我们一定要避免。对于中间人，我们该请吃饭要请吃饭，该直接感谢直接感谢，同时，逢年过节该走动也要走动，尽可能把中间人的资源尽可能多承接到自己身上来，即便承接不到自己身上来，保持好关系绝对是应该的。因为后续你依然有可能有麻烦到别人的地方。
