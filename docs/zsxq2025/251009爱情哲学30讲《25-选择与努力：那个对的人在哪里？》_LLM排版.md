
# 25 选择与努力：那个对的人在哪里？

上一讲我们谈到，单凭技巧和攻略其实很难帮你“赢得爱情”，哪怕通过技巧“俘获”了一个合意的对象，如果缺乏真实和真诚，也并不是真正的爱情。有些同学会说：“道理我都懂，那又有什么用呢？我还是渴望真正的爱情，但问题是，怎么才能找到那个对的人呢？”确实，这是许多人关心和疑惑的问题，也是一个难题。这一讲，我们就来探讨“那个对的人在哪里？”。

运气的确存在，但完全依赖运气，但我们知道：运气不能作为方法，也不需要方法。所谓“方法”，必须在面对现实的时候，具有可行性。对于绝大多数人来说，现实的真相不是传奇，而是平凡无奇，很难有非凡的运气。

寻找“对的人”是一种执念。这可能和浪漫主义的爱情叙事有关。这种叙事常常暗示，每个人都有一个“命中注定的另一半”，就像柏拉图的《会饮篇》中，喜剧作家阿里斯托芬讲述的神话：人类原本是完整的球体，后来被神劈成了两半，此生注定要不断寻找那失落的另一半。这个神话在西方文化中影响深远，直到今天，很多人仍把爱情理解为“找到你灵魂的伴侣，然后两个人融为一体，达到圆满”。它确实契合了我们内心对完整和归属的渴望，而且描绘了一种理想化的爱情状态：好像只要找到了那个“对的另一半”，所有问题都会迎刃而解，从此就能“幸福快乐地生活在一起”。

但我们在第三讲中提到过，这个“1+1=1”的神话并不是柏拉图赞同的观点。然而，它却在各种故事、电影、歌词和广告中被反复传播，构成了浪漫主义爱情叙事的一个主题——只要你找到那个对的人，一切就都对对了。

这背后的潜台词是，你现在的痛苦、挫折和迷茫，都是因为还没有遇到那个对的人。于是，“那个对的人”成为了一种象征符号，一种无望的寄托和幻想中的拯救。它对爱情中挫败与无助给予了巨大的安抚，仿佛在说：你值得更好的，只是还不够幸运。而保持这种执念，是对命运仍然可能逆转留有一丝希望。

但现实的真相是，对于大多数人来说，那个命中注定的“对的人”可能并不存在。即使存在，你们大概也不会相遇。就像绝大多数人的彩票并不会中大奖。如果一定要回答“那个对的人在哪里”，我的答案是：如果存在，只可能出现在未来。只有在爱情的选择和努力的过程中，经过尝试和相互塑造，彼此才能成为那个“对的人”。

换句话说，对的人并不是“现成的”，而是“造就的”，是爱情实践的结果，而不是爱情的起始条件。

努力与选择的关系。有一种说法是，你先要自我修炼，让自己准备好才能投入爱情。这种观点强调的是“自我完善优先”，认为自身越优秀，就越能吸引到适合你的人。只有当你足够独立、自信和成熟之后，才能进入一段真正的爱情关系。

这种说法有一定的道理，但也可能让人陷入“过度修炼”的停滞。因为现实生活从来不等你准备好才开始。几乎所有人，都是在还没有完全准备好的情况下，跌跌撞撞地踏上了爱的旅程。这就像学游泳一样，如果你一直在为游泳做热身准备，却从不下水，那永远也不会游泳。同样，爱的能力，包括选择的能力，也只有在爱的实践中才能不断发展。

关于选择。其实，选择是一个探索和试错的过程，你的标准和判断，也会在这个过程中逐渐澄清和变化。英国作家阿兰·德波顿就写过一篇很有名的文章，叫《我们为什么总是会和错误的人结婚》。他说，人们对“相爱”的幻想，其实只覆盖了恋爱最初的半年到一年。在这个阶段之后，真正的考验才刚刚开始：我们面对的是“对方真实的面貌”，而不仅仅是他最初的魅力。

既然选择是一个试错过程，那么，一个现实的问题是：选择的时候，需要理性计算吗？一般说来，多数人不可能完全脱离那种通用标准，比如在相亲网站上填写的各种数据指标：身高、体重、外貌、学历、收入、职业、个人兴趣爱好等等。

但爱情中的选择，并不是在巨大的商品货架上，挑出一瓶指标最佳的健康饮料，它具有很强的个体性和具身性。除了数据指标，我们还会格外关注那些无法被指标化、数量化的个体特征。你可以想象，如果让电影《泰坦尼克号》的男女主角，或者罗密欧和朱丽叶，在今天的相亲网站上相遇，会怎么样？他们可能都不会选择对方，因为指标明显不般配。为什么在故事中，他们选择了彼此呢？

正像前面我们讲过的，真正吸引你、打动你的，不是那些抽象的指标，而是品质的具体表现。忘记的同学，可以回到第 22 讲复习一下。这里再举个反例：一对男女毕业于同一所学校，现在同样年薪 30 万，从数据指标上看都是相同的，但他们是否热爱自己的职业、具有什么样的工作风格和才能，可能大相径庭。这意味着，他们的精神状况很可能各不相同。

所以，一个能让你由衷喜欢的人，可能会超越指标上的般配。而仅仅指标合适的人，你可能又未必喜欢。这里正好回应了一个常见的问题：到底是要选“合适的人”还是“喜欢的人”？在我看来，喜欢的人不一定就合适，但不喜欢的人一定不合适。喜欢无法与合适分离。因为喜欢本身会激发你的意愿去创造和努力，而这种努力又会增强爱的能力，从而进入一个正向的循环。

关于努力。说到努力，什么样的努力才是有效的呢？努力不是一时的意愿，而是一种持续的行动：当热情退却时，是转身离开还是重新复苏？当冲突来临时，是逃避还是沟通？当面临生活的琐碎事务，是心烦意乱还是保持耐心、彼此激励？所有这些日常细节中的行动，才构成了爱情最真实也最艰难的努力。但也正是在这些日常的共同努力创造中，一个最初选择的“合适的人”才有可能慢慢成为“那个对的人”。

说了这么多，选择与努力的关系到底是什么呢？一句话，选择和努力在实践中就不是截然二分的，而是互相交织的：如果选择是一个试错和探索的过程，其中必定有努力的介入，而努力的行动也蕴含着一次次可能的选择。

这让我想起东北的方言对“谈恋爱”的表达，叫做“找对象”和“处对象”。对象不是单纯“找到”的，也要“相处”才行，而相处也可以是选择过程的一部分。

总结一下：我们寻求的“对的人”不会从天而降，ta 不是现成的，而是彼此造就的。ta 不完全是选择的结果，也不是单凭努力就能造就的，而是一种在选择中开始，在共同努力中创造的实践成就。我们无法确定“那个对的人”在哪里，但我想说，那个愿意和你一起努力，坚持不懈成就彼此的那个人，更有可能成为“那个对的人”。到那时你会说，多亏“我在人群中多看了你一眼，但绝非‘只是因为’那一眼”。

思考题：你觉得通过努力实践造就的爱情，和那种天赐良机的超级传奇相比，哪种更值得你追求？或者做个类比，是艰苦创业成功的收获，还是中头等彩票？哪个会让你感到更幸福？欢迎在留言区讨论。
