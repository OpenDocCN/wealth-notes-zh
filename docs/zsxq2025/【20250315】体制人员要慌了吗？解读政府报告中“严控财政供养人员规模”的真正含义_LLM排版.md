
# 体制人员要慌了吗？解读政府报告中“严控财政供养人员规模”的真正含义

在2025年的政府工作报告中，总理关于这个严控财政供养人员规模的表述就引发了广泛的关注。大部分自媒体的解读都是说这是钟帼政府面临着财政压力的现实考量。其实，严控财政供养人员不等于现在就已经开始砍铁饭碗了，它更像是中央高层对财政供养这一行为的边界调整。而且真正带来变数的其实是人工智能的发展。此外，钟帼正面临着一场涉及到经济、政治以及社会结构的整体大变革。政府也未必是想立刻就进行断崖式的裁员，更大的可能是政府正在进行一场新的摸着石头过河。

政府可能正在观察AI、机器人时代的经济转型情况，再决定下一步要如何改变自身。现在的局势就如同这个克劳斯·施瓦布所说的：政府角色也需要与时俱进，不能再沿袭过去大而全和僵化冗余的旧模式了。在第四次工业革命中，科技将以前所未有的速度去冲击所有行业，改变政府的治理模式和传统的就业结构。

在北宋时期，中央政府出于天下无乱的考量，会在灾年的时候，把大量的流民和社会闲散人员编入军队，这就是北宋军队出现冗兵现象的根源。这样的政策在灾年的时候确实是能够迅速的安抚民心，减少动乱，但是后果就是会给国家财政带来难以承受的重负。当代的钟帼政府在不同的时期也是运用了类似的思路的。过去的几十年里面，每当经济下行压力加大的时候，政府往往就会通过公务员、事业编制以及国企岗位的扩招来拖住就业。比如在2008年金融危机爆发以后，很多地方政府就迅速的扩招了公务员，并且让国企也增设了很多岗位。这在官方有一种正式的称呼，叫做“救市式的吸纳”。

而在3年口罩时期，为了稳定社会和就业，很多地区也对基层岗位和社区岗位进行了大规模的扩招。而且这次的覆盖面积更广，除了公务员、事业编以及国企以外，很多和政府存在着财政或者行政联系的外围机构也加入了兜底的行动。比如说各类的协会、研究院、科技服务平台以及公共媒体等。这些半市场化的组织成了当下规模异常庞大的就业蓄水池。也正因为如此，财政供养人员的总量在疫情期间是骤然膨胀的。这当然就会给国家财政带来严重的负担，也同时严重影响了社会的基本公平。

在历史上，钟帼政府也会在恰当的时候对庞大的财政供养队伍进行瘦身。一般是在经济回暖和社会稳定度较高的时候，或者是科技革命能够支撑出新的就业形态的时候，比如上个世纪的90年代初，中国在加入WTO后，国企就展开了大规模的裁员。当时主要的做法是“留住核心，放走边缘”，这成功为后来的深化改革赢得了财政余利。再比如说，在2013年前后，互联网和电商经济迅速崛起，各类的网络平台就吸纳了海量的就业，在这个节点上，国务院和党政机构就又实施了一轮颇具影响力的改革，当时是大部门制改革和机构合并的接连落地。

实际上，他们遵循着一条共同的逻辑，就是在风险或者危机出现的时候，钟帼政府往往会通过扩编来稳定民生，而在经济转暖或者财政压力爆发的时候，则会通过体制改革、机构合并、自然淘汰以及招录限制等手段来削减这个冗余人员。换言之，财政供养在钟帼，有点像是特有的社会稳定器。因此，要真正读懂李总理的“严控财政供养人员规模”这句话，就必须要先理解，在钟帼政府的语境之下，这个财政供养从来都不是单纯的政府规模失控，而是一种社会稳定器的策略性选择。它既承担了兜底救市的功能，也注定要在风雨之后进行一场剧烈的规模收缩。

李总理这次虽然是说了重话，但未必就是想立刻进行断崖式的裁员。虽然AI、机器人等新兴产业正在蓬勃发展，但真正要想形成大规模的产业链就业，还需要很长的时间去进行基础配套设施的建设，比如说算力、算法生态以及AI芯片制造等。如果此时就进行政府部门断崖式的裁员，无疑是把大批人推向一个还没有成熟的市场，既难以做到吸纳这么多人，也很容易形成新的社会风险。

此外，即使AI已经在某些领域崭露头角了，但是劳动者和新岗位之间还是存在着很大的技术鸿沟的。中国庞大的基层人员未必具备直接转移到AI、机器人等新产业的技能。另外，当前民营经济的复苏力度是远远没有达到旺盛吸纳劳动力的程度的，目前通缩还是在延续的。这是一个基本的事实。

虽然地方政府的财政压力看起来已经是迫在眉睫了，但是如果砍编制、砍人员的力度过猛，不仅容易引发体制内的强烈反弹，也可能会造成更大的社会不稳定。这样做看起来可能是省钱了，但是更大的可能是会极大的增加维稳经费。因此，砍编制是未必可行的。所以从形式上来看，李总理的设想是不具备激进实施的社会基础的。一场真正可行的精兵简政是需要做好市场接盘的铺垫的。

有很多体制内的朋友说，害怕自己未来可能会被取代，因为当下的AI公务员已经日益成熟了，呼声很高。这个事情是这样子的：从长远来看，政务数字化是减少人力成本、提高效率的必经之路。但是搭建这些系统本身是需要大量的资金和技术投入的，也需要有时间来重塑原有的人工流程。数字化，不会在一夜之间就把所有的人工岗位都砍掉的，所以当下所提的严控财政供养人员规模，更可行的方案是稳步的减肥。钟帼政府可能会倾向于在技术的转型期先严控增量、自然减员，而不是像马斯克对美国政府一样，直接采取过激的行动。

今年的提法，可能是对未来的政府结构性改革释放一个强烈信号，但也仅仅是一个信号而已。同时，如果再结合政务的发展趋势来看，AI，一个残酷的结论就是大规模改革是一定会出现的，这一点是已经基本断定了，只不过当前还是处于酝酿期，而不是启动期。那么如果一定要对钟帼政府的启动期做一个预测，估计应该是在AI机器人生产极大的降低门槛的时候，那个时候中国的小工厂也能轻松的参与、设计和生产民用机器人了，这时候民营企业才会出现新的繁荣，这时候才能重新具备容纳大规模就业的能力，此时的钟帼政府才有了精兵简政的外部条件。

如果按照宇树机器人CEO王欣欣的预估：机器人大规模推广是在5年以后。如果再考虑到当前资本对AI机器人赛道的热情，那么可能性最大的启动时间点，应该是会比王欣欣预估的5年可能会更快一些。那么这一场钟帼政府结构性改革，它的基本蓝图是怎么样的呢？具体怎么改呢？首先，按照钟帼政府官方的说法，政务的改革基本蓝图是政府要尽可能的平台化，AI。目前的钟帼政府是以传统的模式在运转的：其基本的特征是人制与层级式官僚，在经济和社会领域的直接参与度非常高，通过官僚体系和政策指令来维持运转。而未来AI政务的趋势会让大量常规性、重复性的审批工作交给算法，也就是说，政府将不再依赖大规模的人力来审批，而审批越少则平台会越大，这就是未来的趋势。

虽然一线的执法公共服务人员仍然需要一定的保留，但是人数肯定会相对精简的，对专业化的要求也会更高。而关于中央政府的一些核心职能，譬如国防安全、司法、外交以及制定宏观经济政策等，就必须要组建更高精尖的专业团队来统筹，其入职门槛势必是会水涨船高的。而在地方层面，一些非核心的职能，譬如像城市管理、环卫、特定的技术类政务服务等，大概率可能会采取外包模式，让小政府与大社会联动，用社会组织或者企业的专业度来代替这种传统编制的冗余。这可能是未来一个趋势。对于那些目前还处于事业单位里的人来说，他们未来很有可能会被转移到政府签约的社会组织或者企业里面去。同时，行政层级的扁平化以及地方政府的合并瘦身的改革也会继续深入。具体是有两个方向的：第一，大部门制将会被继续推动，中央的环节也会继续减少；第二，各级政府利用新的AI政务系统也将不再需要大批跑腿式的中介机构了。

以上所提的这些改革进程其实是目前已经开始了，只不过各类媒体还没有进行大规模宣传而已，可能这也是顾及到社会的恐慌。另外，从历史的宏观来看，这场AI政务的整个改革是被划分成三个阶段的，目前还是处于第一阶段。具体是这样子的：

第一阶段就是把很多业务放在已经有了的政务或者网上办事大厅上面去办，APP，这种模式大多数只是把线下的流程搬到线上，实际上仍然是需要大量的人工在后台操作的。以后这种模式可能会转变，会转变成真正的全流程的智能化，大部分的工作会完全由AI系统自动完成。

第二阶段就是钟帼的官员们会全面启动来辅助AI决策，比如说现在有些地方已经实验性的使用数字城管或者数字监管了，并且还运用了AI来监测公共安全、舆情以及环境违规等。外媒对此的抨击是钟帼净府在搞数字威权主义，实际上不是的，这是AI政务改革的一部分。预估未来的方向可能会进一步发展成官员的“决策模拟”——是可以做到根据海量数据来推导和AI模拟出政策效果的，这样大大削弱过去依赖经验主义的官员对政策制定的主导权。其实这样也好，这样可以有效的避免一些地区的恶政频出，虽然“老人治政”的经验可能也会被淹没，但是过去很多极难根除的地方治理顽疾也有望迎来真正的技术性解决。

那么第三阶段是什么呢？第三阶段就是由AI完全取代人类进行各个城市的综合治理，这个目前还只是存在于理论阶段的，有太多的领导对此是不放心的，预估这个在未来几十年之内应该是看不到的，至少当前这一代人是不必过多担心这个问题的。

那么根据这个趋势，正在被财政供养的朋友们个人应该如何应对呢？首先，公务员精英化的历史进程现在是已经启动了，这个只能去正面面对，未来的政府对高端的专业人才和高级管理能力的需求，只会越发的水涨船高，尤其是对复杂性执法、政策研究、国际关系以及核心公共服务等领域，政府是会有大量的需求的，所以公务员招录时的考核就更会关心你读过什么、做过什么，以及核心专业技能是什么，这些将会成为体制内的硬通货。同时，在重复劳动交给AI以后，未来在体制内留给人做的事应该就是理解社会情绪、政治环境、宏观政策以及做那些AI替代不了的复杂判断。此外，优化和管理AI也是需要人去做的，比如说与技术部门合作提出新的业务需求，还有对AI决策的结果进行人类视角的审视和把关等这些都只能依靠人比较特殊，还有一个就是面对突发情况的时候，也必然是需要人机协同的，这个领域现在看起来也是急缺人才的。所以体制内的朋友如果还年轻，就需要认清AI的大事，聚焦机器人无法替代的工作，或者说把自己打造成懂技术、懂政策、能管理的复合型干部，这可能是体制内朋友唯一的出路。

当然了，如果从另外一个角度来看，这也是对年轻干部的一个重大利好，因为这一次是年轻干部对中老年干部的一次弯道超车的特殊机遇，现在还是处于改革红利期的，是存在着很多提拔机会的，主要是因为很多的中老年干部根本就无法理解AI，这当然就会给年轻人机会，AI时代将会促成社会财富大规模向年轻人转移，同样的道理，其实也在体制内也会发生类似的变化，时代将淘汰大量的中老年干部，AI，同时也将使大批量的年轻人弯道超车。

很多朋友可能会说，体制内的提拔关键是看关系而不是专业，其实这一次不一样，如果个人足够的专业，而且在专业人才稀缺的大环境里面，那么专业人才就能靠自己的本事攀上关系，毕竟领导也是要用人的嘛，如果对这番话感到模糊的朋友，推荐看一本书叫《大江大河》，这本书讲一个没有背景的技术类官员是如何在时代浪潮中脱颖而出的。另外，现在政府的内部也是存在着很多的AI试点单位和重点项目的，这些部门往往经费很充足，也有更多的机会对接国内外的先进团队，年轻干部其实是可以去主动拥抱此类部门的，最好能积累一些技术、治理、民政的经验，预估未来的中国政府是需要大量的这种专业的高级官员的，所以总的结论就是大的结构性变革现在已经不可避免的，越早转变观念，对个人可能就会更有利。

那么在这场官员队伍的大变革中，体制外的朋友又有什么机遇呢？其实有非常多商机，首先，政府部门的瘦身趋势一旦出现，那么就必然会伴随着大量非核心职能的外包，比如说城市管理、环卫、辅助性的政务服务等，如果能建立一家有专业能力和合规意识的公司，就很有机会去主动承接，当然了，这可能要对这个招标规则有很深入的了解。其次，在这个AI政务转型的早期，很多地方政府自身的技术储备肯定是不足的，那么AI方案的集成和维护就会成为刚需，那么时就迫切需要这种IT企业、咨询公司或者是那个技术团队。做这种系统化AI方案的集成维护，与普通互联网项目相比，AI政务对数字化的安全性、数字合规和保密等也会要求更高，这也是一般民企涉足不深的领域，如果能率先建立一套可行的方案，就会具备先发优势。此外，海外也会有很多新的机遇出现，这也是一个规模巨大的蛋糕，未来在一带一路上很多的发展中国家也会很希望能够借鉴中国的AI政务模式的AI，如果你的公司有了很多国内的相关经验，那么来自世界各国的需求也会带来海量的订单。

总而言之，每一次体制内的大调整，都会衍生出新的产业形态和商业模式，这一轮AI政务的改革也是如此，其实，围绕着中国政府去做生意，它既是一个古老的传统行业，也是一个会随着时代更新的行业，关键还是看自己应该如何去应对这个时代的浪潮。
