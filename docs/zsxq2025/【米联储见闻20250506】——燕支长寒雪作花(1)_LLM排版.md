
# 米联储见闻20250506——燕支长寒雪作花

这个五一其实颇为平静，内外都没有什么大事情发生。川普引来了自己的百日总结，虽然民调创80年来最低，但川总自我感觉良好。马斯克退出内阁，川总内部也做小幅调整，在关键问题上让步，不再要求替换美联储主席，并明确表示不会有第三任期。可见有人给川总带上了缰绳，不要飞的太远。但是对华的攻势并没有减弱，相反，很多反对川普国内改革的人正在因为川普对华遏制开始收效对冲了一部分对川普的负面印象。川总的团队很快就会意识到，他所做的这一切事情想要做好做成都在于能不能压制住东大。如果压制住东大，美国国内的改革也能减少阻力。如果在压不住东大的情况下对美国国内进行改革，就会快速走向崩溃。关于攘外安内的问题，最起码对川总来说，眼下一定是攘外的优先级大于安内。这就像战时的罗斯福也并非在国内没有政治危机，但是因为确定了美国在二战中的制胜优势，甚至成为二战赢家，才会借二战之力解决了1929年大萧条给美国留下的政治经济创伤。现在美国虽然问题很多，但解决这个问题的唯一解其实就在外部，所以意识到这一点之后，川总很快开始调整对外策略，对华的态度也在反复横跳。民主党大概也意识到，如果川总真的集中火力恶化中美关系，实际上留给庙铎勹肴政治空间将会进一步压缩，民主党尚未其森念美国内部政治生态，或§民生经济生态恶化，辱仑下来，不管是川总，还是民主党，包括暴冷甘韩，可能都会调整节奏和态度。现在是一怜＼鲁局势一日三变的状态，法无定法，实事求是。

纽约时报假期发了一篇文章，认为川总特加征的关税巳开始伤及中国的制造业。目前关税战打响足足一个月了，国家统计局周三发布的制造业调查报告显示，4月中国制造业活动出现了逾一年来最剧烈的月度萎缩。PMI指数从3月的50.5降至4月的49.0。该指数低于50表明制造业活动出现恶化。受关税生效前订单增加的推动，前两个月该指数均高于50。4月的数据低于经济学家的预期。出口商品新订单的读数创下了自新冠疫情以来的最低水平，拖累了整体指数。此外，衡量制造业就业人数的指标也出现萎缩，处于自2024年2月以来的最低水平。有机构评估：受川普关税政策影响，中国经济今年将增长3.5%，远低于政府设定的5%的增长目标。野村证券的一份研究报告称，如果中国对美国的出口下降50%，中国将有570万人立即失业。一旦长期影响波及整个经济，失业人数将增至1580万。

视频最后捍卫中国硬杠美国的做法说，当每个国家都挺直脊梁，世界必将冲破霸权的高墙。这段宣传视频采用大量具有冲击力的文案和画面，包括老毛1946年接受美国记者斯特朗采访时下的论断：一切反动派都是纸老虎；孟晚舟被戴上电子脚镣的特写镜头，以及她在被羁押近三年后回国，坐中国包机抵达深圳的画面。这个视频与节前政治局会议把中美关税战“定调为国际经贸斗争的舆论口径定调一致，这种类似于摊牌的动员视频，本身传递了中方的多重态度：首先，对内的团结动员，向中国民众展示官方与美国斗到底的决心。视频当天上线后，确实可以看到朋友圈有很多人转发，支持官方对美做长期斗争。也确实有网友认为《不跪》这个标题起得不好，认为讨论跪不跪有点气短，应该想办法让美国跪。可见民间认知与官方认知对与美斗争的看法有差异，民间认为中美应该是势均力敌，甚至认为东升西降巳经进入收官期，中方优势更大，态度更加乐观；但显然官方人士中美在实力上仍有明显差距，与美斗争是一场艰难的持久战，耐力战，形势并不是乐观，所以才会出现民间和官方对不跪两个字的不同看法。

对外而言，这段视频也算袅攫＄美国，中方不会在关税战中轻易让步，一开雄流出奉陪到底的强硬立场，在关税数额上以柔汽护，也采取一系列非关税措施，例如限、对美企实施出口管制等。不仅释出不会回对等关税前谈判的信号，过去一周更是三番笠次在川普宣称两国巳就贸易谈判接触后，第一时间公开否认，通过这段视频，中方强调自己站在公理一方，弱者一方，正义一方，批判美国霸权破坏规则，喊话国际社会和中国站同一阵线，对美国的不合理要求说不。这也是王外长最近在一系列外交场合反复呼吁和宣传的中方立场，呼吁各方不要在美国漫天要价下妥协退缩，让霸凌者得寸进尺。

从经济角度看，硬碰硬未必是最佳方案，但显然中国现在的考虑不是经济角度，如前面所说，如果川总成功遏制了东大就会降低美国内部改革的压力和阻力，实际上等于是缓解了美国的内部矛盾，所以，中国实际上就是要恶化美国的外部斗争局势，让川普不能轻易在外部斗争中占到便宜，甚至连口头便宜都占不到，抱着自伤八百也要跟你死扛到底的心思，加速美国自身内部恶化。因为中国目前的判断是，美国将长期把中国视为战略对手，不惜一切大家打压中国，并且将之作为一个长期目标，战略目标，这意味着，美国自身实力越强，中国就越危险，真正能够让中国免于灭顶之灾的出路就是要让美国在危机中导致综合。

这样的谈判根本没有任何信任基础。不过态度虽然是坚定，实操却是灵活的，川普一边嘴硬一边开始作出延期调整，而中方其实也在一些关键产品的进口关税方面进行减免，比如医疗设备、航空零部件、乙烧等美国商品，也开始讨论关税战可能对就业、企业、市场和预期产生的冲击。今年五一，很多地方为了吸引游客，甚至放开了政府和事业单位的停车场和食堂，这里面肯定有中央精神的鼓励，也足见地方在经济方面的压力，现在还不是最难的时候，第二季度的数据出来以后，很多事情还会有剧烈的变化。

出海捞金和消费降级仍是未来的主旋律，国外赚钱国内花，应该是当下最佳选择，把中国打造成全球最物美价廉的消费目的地，同时把中国高效的产能和生产经验转移到海外，赚高额利润，相比于过去国内赚钱国外花，现在的模式需要彻底调转了。所以，消费升级仍是大方向，这里面还有很多文章可以做的，期待一下五一的消费数据再说。
