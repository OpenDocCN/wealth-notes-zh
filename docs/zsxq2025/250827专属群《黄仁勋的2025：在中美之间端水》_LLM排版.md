
# 黄仁勋的2025：在中美之间端水

黄仁勋，英伟达创始人，被许多人视为AI革命的全面赢家。他不仅将英伟达市值带上了4万亿美元的世界纪录，还成为了全球资本、各国政府和科技大厂追逐的对象。作为黄种人，他的成功更加引人瞩目。黄仁勋的身家已经超过了股神巴菲特，在事业与财富两条成就线来看，都足以被崇拜。

然而，在AI之外，很难想象美国未来还能在哪个产业再跟中国拉开技术代差，并实现多赢。就连当下的AI产业之争，也是“我们的中国人”和“他们的中国人”之争，扎克伯格和马斯克们砸重金吸引的人才，一多半也是华裔面孔。

2025年，黄仁勋在中美博弈前线为英伟达争取空间，展现出了“最后一个科技偶像”的特质：近乎外交家的手腕与分寸感。这远远超出崇拜者们对一个工程师型CEO的预期。

## 端水大师

黄仁勋对特朗普阵营的经营成果，在上一讲中已经有所提及。而在中国这边，黄仁勋7月来参加链博会时，带来了最大的礼物：美国政府解禁H20芯片出口的消息。此外，他还主动脱掉标志性的皮夹克，换上了唐装，走上链博会开幕式的讲台，用并不熟练的中文说开场白，然后在接下来的英文演讲里，赞赏中国的供应链堪称奇迹，中国的开源AI是推动全球进步的催化剂。

根据公开信息，黄仁勋到访的这一周主要行程如下：

- 7月14日，黄仁勋与小米CEO雷军会面，并在社交媒体上公开了两张与雷军的合影。
- 7月15日，黄仁勋告诉中国，英伟达H20芯片获得了美国政府的出口许可，将重启对华销售。此外，英伟达还将面向中国发布一款新显卡，名为RTX Pro，来服务中国正在发展的数字孪生与机器人产业。
- 7月16日，黄仁勋上午到链博会发表演讲，随后接受央视《面对面》节目访问，下午又在他落脚的四合院里，组织了30多家中国媒体做沟通。
- 7月17日，黄仁勋回到链博会会场，跟阿里云创始人王坚对谈。下午拜会商务部部长王文涛，晚上又出现在北京著名的景点什刹海，泡酒吧、逛街，跟认识他的路人合影、签名。
- 7月18日，他又跟“AI六小虎”之一的MiniMAX创始人闫俊杰见面。

在中国短短几天的行程里，黄仁勋疯狂给中国点赞。他说早就知道雷军会成功，说敬重华为，说DeepSeek是毫无疑问的技术突破，说中国电动汽车是全球惊喜，说中国机器人发展极度乐观，说中国培养了世界最好的一批AI研究者，还说中国AI市场有没有英伟达都会进步。

而等他的中国行程结束，7月23日，现身美国的人工智能峰会上，黄仁勋又毫不犹豫地赞美起了美国。他说：“美国所拥有的、其他国家无法企及的独特优势，是特朗普总统。”事后，特朗普亲自在社交平台发布了这段赞美视频。

黄仁勋这种强悍的政治平衡能力，在中国互联网上，通常被尊称为“端水大师”。硅谷和华盛顿虽然没有这种说法，但也为之震撼：这人半年前真的是个政治门外汉吗？怎么突然就超越了库克、马斯克等等前辈，在最凶险的国际舞台上、成了最能左右逢源的企业家？

## 对等交换

黄仁勋突然展现的这种高段位社交能力，超出我们对工程师的预期。但这倒让我想起一段旧闻。

过去这些年，黄仁勋跟采访者聊起自己成长经历时，好几次提到这么一段故事：

他10岁那年被父母送往美国，阴差阳错进了一所青少年行为矫正学校去寄宿读书。啥叫“青少年行为矫正学校”？就是问题少年收留中心。黄仁勋的室友当然也是一个问题少年，大他7岁、经常打架械斗，一见面就用身上的刀疤来吓唬他。

但10岁的黄仁勋很快找到了保护自己的法子。他跟室友达成了交易：他教对方认字，对方教他做俯卧撑。从那个时候起，他跟室友成了好兄弟，再也没被欺负过。也从那个时候起，黄仁勋养成了每晚做100个俯卧撑的习惯，一直坚持到现在。

让对方教俯卧撑这个细节，后来经常被我想起。任何一个好学生要跟小混混长期相处，都很容易想到卖艺保平安的办法，用自己的作业或学识，来为对方创造价值。

但黄仁勋在这个基础上，还多想了一步：他向对方也提出了需求，要对方也教他东西。学什么？俯卧撑，一种看看就会、不需要学的健身方式。

事情妙就妙在这里。

俗话说“大恩如大仇”，人在接受恩情无法回报时，容易产生心理负担，滋生反感和敌意。这个道理放在小混混身上也有效，一个学霸单方面教小混混写字，日子久了不免让对方产生“欠人情”的不爽感。而学霸教写字的同时，让对方教自己俯卧撑，甚至再请对方每晚监督自己练一练，这能让对方觉得自己被需要，两人更容易形成对等的交情。

你说，这是年幼的黄仁勋不小心趟对了路，还是用理性拿捏了人情世故？

等到了2025年，黄仁勋游走在美国、中东与中国市场之间，险象环生地化解危机、巩固生意，同时还不忘为英伟达下一个十年的生存确定性下注。我觉得，答案终于显露了出来。

## 两大危机

但是话说回来，今天的AI产业界显然跟问题学校已经不可同日而语，那为什么一个取得了卓越技术成就和商业成就的工程师型CEO，还要花这么大精力去在中美之间端水呢？

很现实的原因是，当前的英伟达，面临两个巨大的生存危机。而黄仁勋又是一个对危险和失败无比敏感的人。

他的这种个性，专栏在第三季已经跟你聊过。

哪两大危机？

第一个，全球AI产业链的“去英伟达化”。这两年，英伟达的竞争对手、上游客户和供应商，受到行业需求和利润的刺激，都启动了芯片自研战略。

英伟达的竞争对手就不用说了，博通、英特尔、AMD都跟得很紧。

在它的客户群里，亚马逊AWS、微软Azure、谷歌云、甲骨文OCI、阿里云、华为云这些全球主要云计算厂商，都有自研芯片的计划和行动，有些都已经量产有客户了。

它的上游呢？就在8月份，原来给英伟达卖技术架构的ARM公司，也宣布要亲自下场造芯片，跟老客户高通和英伟达抢饭吃。

这些变化，都让英伟达有可能从垄断龙头变成普通选手。这是黄仁勋不能接受的。

英伟达的第二大生存危机是，它在中国这个超大市场的生意，突然遇到了三重阻力。

一个阻力来自产业上下游。它在西方市场遭受的产业链竞争，在中国照样得来一遍。中国的一批国产芯片公司，寒武纪、摩尔线程、壁仞科技都在融资紧追。华为的自研芯片昇腾、百度的自研芯片昆仑，也都已经用来训练大模型。

另一个阻力来自美国政府。英伟达这几年接连受到拜登政府和特朗普政府的限制对华出口，也在变相给中国国产芯片厂商们腾市场。

IDC的统计说，2024年中国市场AI芯片的总出货量超过270万张，其中国产芯片出货量超过了82万张，同比增长了310%。同时，券商伯恩斯坦预计，英伟达在中国AI芯片市场的份额，将从2023年的66%下滑至2025年的54%。

第三个阻力来自中国政府。就在黄仁勋终于拿到美国政府出口许可之后，中国政府公开表达了对H20芯片的怀疑，并通过约谈，向英伟达的中国大客户们释放了某种戒备信号。

如果英伟达错过这样一个中国市场，也是黄仁勋不能接受的。在他的判断里，中国AI芯片市场有两个吸引力：

一个是，这个市场未来的规模有望达到500亿美元；另一个是，中国有望孵化出一套独立于美国的AI产业体系。

## 总结

那么，黄仁勋应对这两大危机的方式，就只有端水、砸钱和说好听的吗？当然不是。正好他7月到访中国这次，跟中国媒体和产业界进行了一些非常有质量的对话，我们从他的对话里可以发现一些他给英伟达安排的未来生存和增长之路。
