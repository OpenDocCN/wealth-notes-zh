
# 从一件魔幻的小事，看2025年的国运

原创teafox 茶狐看世界2025年02月27日07:25

前几天，在联合国，乌克兰提出针对俄罗斯的决议草案，结果93票赞成、18票反对，65票弃权。其中，东大投了弃权票。投反对票的18个国家包括：美国、俄罗斯、白俄罗斯、朝鲜、以色列等。我做梦也没有想到，2025年，这几国居然能达成共识。

国家之间从来都是丛林法则，只不过，在2025年，这条残酷的法则更加清晰。要想不被列强蹂躏，唯一的方法就是成为列强。哪怕只有月薪三千，国家的强弱也与你息息相关。

我每年都会写几篇关于大势的文章，根据我行走各国的经验，从一个普通人的视角出发，观察世界的变化，不一定专业，但一定有凭有据，有第一手资料。

为了写作，我经常「没事找事」，目的就是体验生活，观察社会。有些事情，如果不是亲身经历，编都编不出来，现实的生活，往往比戏剧更加荒诞。从有些小事的细节，就可以观察一个国家的运势。

我们国家与其他工业国家最大的不同，在于我们的经济结构——同时存在国企和民企，这就是中国特色的市场经济。

先说民企，我一直坚持，市场经济是为中国人量身定做的，我们勤奋、聪明，渴望成功，热爱财富，甚至连新年祝福，都是「恭喜发财」！今年以来，民营企业在中美科技竞争前沿一线的重要性再度凸显。出自民企之手的、以DeepSeek 和《哪吒2》为代表的现象级产品，无疑是振作市场信心的直接“导火索”，这背后的重要原因是，民企已经成为创新源泉。

最近十年，在科技创新前沿，开疆破土的企业，几乎都是民企，比如说：华为、字节、大疆、比亚迪、宁德时代、宇树……这些企业出海之后，都是降维打击。

相比民企，国企最大的优势就是「稳定」，为经济发展提供支撑，如同一根定海神针。我们的制造业强，背后有很多原因，其中，我们的独门秘笈就是——基础设施，我们有世界上最强的高铁，最长的高速公路，最大的发电量，最好的电网，便宜的电价，等等等，这些都是国企的贡献，也是其他国家不具备的优势。

民企善于创新，但创新过头了，就是泡沫，国企胜在稳定，但稳定过头了，就是僵化。泡沫可以用市场调节，周期循环，优胜劣汰。国企更依赖国家政策的调控。

让国企在发挥稳定作用的同时，保持活力，对我们来说，极为重要。目前，我们国家，到底采取了什么措施，保持国企的竞争力，这是我很关心的一个问题。

最近一段时间，我亲身体验了一个真实案例，亲眼目睹了两个世界500强国企之间的惨烈竞争，很多细节，让我瞠目结舌，我记录下全过程，与大家分享。从这个案例管中窥豹，可以了解目前头部国企的真实状态。

有些细节可能引起争议，虽然不一定是商业机密，但至少是摆不上台面的竞争手段。

这两家国企就是移动和联通，根据2024年《财富》世界500强排行榜，中国移动以1428亿美元的营收位列全球第55位，而中国联通则以526亿美元的营收位列全球第279位。

去年底，我打开手机，看到一个未接来电，是陌生号码，系统提示为「骚扰电话」，原本打算直接拉黑，但那天我闲来无事，就打开一看，好家伙，这个号码已经给我打了5次电话，每次响铃都超过30秒。

看样子非常执着，也许此人找我真的有事，于是，我就拨了回去。

电话那头，是一个女声，开门见山地说：「我可以把你的手机上网费砍掉一半，而且流量更大，信号更好。」

居然还是营销电话，按照常规，我一定挂断，然后拉黑，但这次没有。

于是，我没有挂断电话，而是让她继续说，对方介绍：「68包含130G流量，500分通话，一条1000兆宽带，额外再赠送370元话费。」

我对她说：「听起来不错，考虑一下。」

她说：「好的，我明天再联系您。」她说话干脆，态度近乎谄媚，听得出来，她似乎看到了业务成功的希望。

我写文章记录生活，最大的好处就是有据可查，正所谓「好记性不如烂笔头」。2023年的条件是：「58元，40G全国流量、通话800分钟、500兆的宽带，没有额外赠送话费。」

价格差不多，但服务更好了，宽带网速提高，手机流量增加了90G，对我来说，挺实用。更吸引我的是，移动公司的国际漫游业务，真的很强，上次在古巴，只有移动公司有信号。为了出国更方便，我动心了。

于是，我尝试「转网资格认证」，就是把一串字符“sqxz#姓名#身份证号码”发给联通10010，几分钟后，就可以知道你的手机号码，能不能转网。

我得到的回复是：「尊敬的用户您好!您的号码因2个原因影响携号转网……」

和上次一样，我刚操作完「转网资格认证」，就马上有联通业务员给我打电话，对方说：「真抱歉，这么晚还打扰您，系统提示，您要转网？是嫌价格高吗？」

其实，我现在套餐的价格已经很便宜，再砍价，连我都不好意思了。

我只能如实说：「价格倒其次，主要因为移动出国漫游业务比较好。」

她训练有素，立刻解释道：「每个公司的网络不同，有些国家他们强，有些国家我们强。」

她说的也有道理，其实，除了国际漫游，我还有一个动机，那就是想看看，现在国企电信公司之间的竞争，到底发展到什么程度了。这一点，当然不能对她讲。

她最后说：「您现在因为有合约在身，不能转网，等合约到期了，我再给你打个折，你是我们的老顾客了，你放心，不会让你吃亏。」

面对这种情况，我也无话好说。联通的价格实惠，服务也不错，我决定放弃转网。

第二天，移动业务员的电话如约而至。

我说：「还有合约在，不能转网。」

她没放弃：「你有领过手机吗？」

我说：「没有，只是套餐。」

她说：「您放心，有套餐也没事，我们帮你付违约金。」

我说：「还是算了吧，我不想去营业厅，太麻烦，浪费时间。」

她依然不放弃：「您不用亲自办理，我们的业务员会上门服务，一切交给我们就可以了。」

被她这么一说，我的好奇心又被点燃了。

现在国企的办事效率让我吃惊，移动的业务员，居然当天下午就上门服务。

接下来的操作，更是让我大开眼界！

业务员非常熟练地打开我的联通APP, 粘贴复制，迅速完成了三步操作。

第一步，携转信息。

信号质量差；无法携转原因：业务限制；收不到携转码；无法线上办理：过度维系挽留。

这段话非常专业，作为一个外行人，我半懂不懂。

第二步，投诉内容。

没空去营业厅，帮我把宽带退掉，影响我转网合约，全部取消。合约都是在不知情的情况下，办的。今天帮我退掉。不退升级投诉。

这步操作，语气更加凶狠。

第三步，来电转接。

业务员在我的手机上设置来电转接，把所有联通客服电话，都转接到他的手机上。

他说，接下来联通会打电话，要么挽留，要么核查，要么找借口拒绝。这些手段，对顾客来说，很麻烦，浪费时间，而对他来说，却很好处理。这是他的职业，每天要处理很多案子，已经熟能生巧。

临走时，他拍胸脯对我保证：「你放心，两天之内搞定！」

过了几天，他给我打电话：「联通说，您2023年又拿了他们新的设备，所以违约金变为1700。如果去营业厅把设备还给他们，就能少个八九百块钱的违约金。」

这时候我想起来了，2023年，我也是因为好奇心办了一个1000兆宽带，每个月加了30元，联通给我一些设备。

我说：「那怎么办?我可没时间去营业厅。」

他说：「不去也行，我付1700没问题，但付了钱你一定要确定转网，不然我就亏大了。」

他的话，让我很吃惊，移动公司为了从联通争夺客户，居然不惜代价，愿意付出1700的违约金。要知道，移动跟我签的合同，三年有效，每年才700多，三年总共才2100，此外，移动还要免费给我设备，付出人力物力。这一单携号转网，移动几乎要赔本。

我问他：「移动公司这么干，图什么呢？」

他说：「一方面，携号转网是工信部发起的，2019年开始，三大运营商都可以互转，目的是让运营商互相竞争，保持活力。另一方面，运营商都有增加用户的压力，宁可牺牲眼前利益，也要从对手抢夺客户。」

接下来，联通与移动的勾心斗角，让我更加意外。

他说：「本来我可以直接打钱给联通，但是我的账户打钱的次数太多了，估计被联通标记了，他们可能不会给我开发票，所以到时候麻烦你打一下。」

我没听懂，问道：「什么意思？」

他解释：「我把钱给你，你去把钱打给联通。」

我说：「没问题。」

又过了几天，联通公司加我微信，对我说：「需支付512.1的违约金+FTTR的设备905+光猫设备费260元=1677.10元。」

就在我准备打钱的时候，意外又发生了。

移动的业务员给我打电话：「你这个单过几天再说吧，我现在不敢做，这几天，联通不给我开对应的发票，不知道什么原因，要不等个几天吧？」

「我们只能报销违约金，但联通的发票都标注建筑服务，故意为难我们。」

原来，联通为了刁难移动，故意在发票上动手脚，这样一来，违约金就不能报销，顾客就不能携号转网。

我万万没想到，两个世界500强国企，为了竞争，能如此挖空心思，不择手段。能不能携号转网，对我来说，无关紧要，让我欣慰的是，我们国企，居然能保持这种战斗力。
