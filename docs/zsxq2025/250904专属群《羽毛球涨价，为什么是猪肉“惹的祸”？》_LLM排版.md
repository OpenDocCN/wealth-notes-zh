
# 羽毛球涨价，为什么是猪肉“惹的祸”？

前段时间，我在营销专家小马宋老师那看到一个有趣的事，说的是羽毛球为什么涨价？从去年开始，各个羽毛球品牌接连宣布涨价，一些高端产品的价格几乎翻倍，原来100块出头的 一筒球，现在要200多块才能拿下。还有人算过，2024年这一整年，黄金的涨幅是88%，而羽毛球的涨幅是96%。

为什么羽毛球的价格会涨得这么厉害呢？羽毛球运动越来越火，买球、打球的人多了是一个原因，但还有一个原因你可能想不到，跟猪肉降价有关系。

羽毛球的原料一般是鹅毛或者鸭毛，一颗球要用16根羽毛。但关键在于，羽毛球对羽毛的硬度要求很高，一只鸭子身上，能拿来做羽毛球的毛大概就七八根。换句话说，一颗羽毛球，得两只鸭子来供应。

而这几年，鸭子和鹅的出栏量，都在下降。我国商品鸭的出栏量，从2019年的48.78亿只，下降到2024年的42.2亿只，同样，商品鹅的出栏量，从6.34亿只，下降到5.69亿只。再加上，养殖技术也在进步，很多肉鸭肉鹅出栏时，羽毛的硬度还没长到适合做羽毛球的程度。这么一来，就导致羽毛的价格水涨船高。

那么，为什么鸭和鹅的出栏量减少了？不是技术上或者供应上的事儿。其中一个关键因素就在于，猪肉降价了。在咱们国内的肉消费市场上，鸭和鹅肉，一定程度上是猪肉的补充和替代品，猪肉价格相对贵，消费就会更多地转向消费鸭和鹅。

而最近几年，猪肉供应持续上升，猪肉价格下降了，那么消费者对鸭和鹅的消费需求就降低，选择猪肉的人变多，相应地选择鸭和鹅的人就减少，于是鸭和鹅的养殖量就下降了。这个链条一路传导下去，终于羽毛球的原材料供应出现了问题，价格飞速上涨。

借用营销专家小马宋老师的评价，这是一个商业供需上的蝴蝶效应。羽毛的供应，受到鸭鹅数量的影响。而鸭子和鹅的供应量，又在肉食市场里，受到猪肉供应的影响。这不是一个单向传导的链条，而是两个市场互相关联，碰撞出来的结果。

换句话说，我们可以把猪肉降价导致羽毛球价格飞涨，看作是一种系统性的因果，也就是，一个变量被输入复杂系统之后，经过系统加工获得的那个出人意料的结果。

其实，在商业社会中，这种曲折的因果关系也并不少见。比如，原本已经稳定下降的银价，这几年突然出现了飞涨。2019年，白银的价格是每盎司15美元。而到了2024年，涨到了每盎司30美元，翻了一番。根据投资机构今年年初给出的预测，2026年，白银价格会涨到每盎司38美元。而在国内的饰品市场上，年初的两个月，白银的价格就上涨了13%。

按理说，白银的稀缺性问题已经被解决了，现在70%以上的白银并不直接来自银矿，而是来自其他金属生产的副产品。理论上说，只要其他行业的产业链稳定，白银就会跟着一起稳定。那么，为什么白银的价格会掉头上涨呢？

放在一个更大的系统中看，白银涨价，不是因为它的贵金属身份，而是因为它作为一种工业原料，变得越来越稀缺。5G设备、光伏发电以及电动车电池，这些前沿科技产品的生产中，白银都是一种关键金属。其中最耗白银的，是太阳能光伏行业。

光伏的电池，由硅片、钝化膜和金属电极组成。而做这个金属电极，银是最佳选项，它同时具备高导电性、耐腐蚀性、高电极附着性以及优秀的光反射性能，且目前还没有很成熟的替代方案。

2022年，全球生产的白银有10%用在光伏板上。同时，这个需求逐年走高，根据国际能源署的估计，2030年，全球30%的白银会用在光伏板上，到2050年，现有产量的90%会用在光伏板上。

你看，银饰涨价，跟太阳能电池产业的发展有关，乍一看出人意料，但仔细推敲又非常合理。

其实，类似的情况还有很多。再比如，全球变暖导致暴风雪增加，乍一看好像没有道理。按照通常的设想，暴风雪的前提是低温，气温升高暴风雪应该减少才对，怎么会增加呢？这其实是因为，系统中有海洋活动的参与。全球变暖导致海水蒸发，水分会被吹向北极。到了冬天，水分会变成雪，覆盖在北美洲东海岸。这样一来，暴风雪出现的概率就增加了。

再比如，北极的冰雪在消融，请问，南极的海冰面积会怎样？按照通常的想象，北极的冰都化了，南极自然也是跟着融化。但根据研究人员的测算，南极海冰的范围从1979年以来基本保持不变。这是因为南大洋中有大量的直径在10到20公里之间的漩涡。这些漩涡把海水从大气中吸收的热量，向赤道的方向输送。这种输送把热量带离了南极周边。这样一来，南极附近海域的升温速度就变慢了。

你看，从系统性的视角出发，很多所谓的悖论，其实是系统自发调节出的一种规律。

再比如，交通领域有个著名的悖论，这就是，路越多，车越堵。怎么缓解交通拥堵？咱们很容易能想到的一种方法，把马路变得更宽。但事实上，假如真这么干了，反而城市交通会更拥堵。美国和英国的研究者做过实证研究，都证实了这个结论。

这是为什么？因为其中有人的参与。拓宽马路，人们会觉得路一定会更通畅、更好走，于是，原来绕道的，或者不开车的人，都来走这条路，这条路就会更拥堵。

再比如，有个很经典的交通案例，在20世纪60年代，德国的斯图加特市为了解决交通拥堵，额外增加了一条街道。按理说道路多了，交通应该更顺畅，但事实上，斯图加特的交通反而变差了。后来政府关闭了这条路，交通又恢复了。不是因为不该增加路段，而是这个路段增加的位置有问题，它有种显而易见地快，导致所有司机都来走这条路。当然，实际情况要更复杂一些，我们在这就不展开说了。

德国数学家迪特里希·布雷斯还专门分析过这个问题，这类现象也因此被称为布雷斯悖论。布雷斯悖论也给了当代城市治理一个提醒，面对任何复杂问题，都不要简单地做加法，而是要看到其中的系统性因果。

再比如，科技领域也有一个著名的杰文斯悖论，说的是，技术进步提高了能源利用效率，但是，整个人类社会消耗的总能源却会上升。杰文斯悖论的背景是19世纪的英国，瓦特改良蒸汽机，大幅提升了燃煤效率，理论上可以使用更少的煤炭来从事生产。但是，因为这个改进带来的效率提升太高，反而引发了更大规模的煤炭使用，这让煤炭的消耗加速增长了。

按照这个理论，咱们也能理解正在发生的很多事情。比如，年初DeepSeek横空出世，它最核心的杀手锏是降本增效。训练最先进的大模型，DeepSeek只用了2048块英伟达的GPU，而OpenAI要用10万块，论推理成本，DeepSeek是OpenAI的1/30。

那么，人们对算力的需求会下降吗？按照杰文斯悖论，并不会。DeepSeek相当于给行业树立了一个范本，一家AI成本大大降低，会刺激更多同行探索怎么降本，同时也会刺激更多企业、机构部署大模型，AI会越用越多，社会整体的算力需求反而会大幅增长。

这就好比淘金热，人们突然发现金子比以前容易挖到了，那么买铲子的人肯定会增加，而不是减少。

好，刚才说了很多案例。从中，我们至少能获得这么三个启发。

第一，什么是复杂性？法国哲学家埃德加·莫兰，提出过复杂思维理论。他说，复杂性不是混乱，而是混乱中的秩序。在一个复杂系统中，因果关系并不是简单的“A导致B”，而是循环、反馈和相互嵌套的。

第二，任何一个反常的结论，只要把它放进一个更大的系统，或许就能得到合理的解释。所谓悖论，其实是经过反复博弈后，达成的均衡。

第三，回到我们自己，万维钢老师有个建议特别值得参考。他说，面对复杂系统中的博弈，最重要的是先入局，先做一个参与游戏的玩家。有一点参与游戏的精神，我们就有权在可控范围内展开最有利的行动。有一点参与游戏的精神，我们就是积极主动的，并且不会有那种把世界简单化的幻觉。
