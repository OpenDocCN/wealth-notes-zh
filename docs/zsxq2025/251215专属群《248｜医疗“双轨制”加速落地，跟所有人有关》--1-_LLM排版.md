
# 医疗“双轨制”加速落地，跟所有人有关

## 医药支付开启“双目录”时代

12月5日，国家医保局发布了2025年的《国家基本医疗保险、生育保险和工伤保险药品目录》及《商业健康保险创新药品目录》。经历了数月的遴选谈判，今年的国家医保谈判药品，简称“国谈药”，目录结果最终落地。

按照惯例，国家医保局的医保药品目录每年调一次，这一次，商业健康保险（商保）创新药的目录也终于出炉，从此开启了中国医药支付领域的“双目录”时代。

今年国家医保药品目录新增了114种药品，有50种是一类创新药，总体谈判成功率88%，高于2024年的76%。

这次的国家医保谈判相比以往，有几个非常突出的特点：

第一，进口创新药的准入提速，多款“明星”进口创新药被纳入了国家医保基本目录。比如美国礼来的成人2型糖尿病特效药替尔泊肽注射液，阿斯利康的乳腺癌创新药物荃科得，通用名：卡匹色替片、罗氏制药的伊那利塞片、美国公司研发并由复星医药引入的盐酸替那帕诺片等等，覆盖肿瘤、慢性病、精神疾病、儿童用药等多个领域。

第二，首次设立商保创新药目录，定位为纳入超出基本医保支付能力、但临床价值显著的创新药。从121个申报品种中最终遴选出18家企业的19个药品，包括5款上百万一针的CAR-T疗法，这是一种治疗肿瘤的新型精准靶向疗法、两款阿尔茨海默症新药及多个罕见病药物，都被纳入了商保创新药目录。

第三，这次的一个新信号，是突出了药企药品的商业化价值。根据知名医药平台米内网的报道，114个新增进入国家基本医保目录药品中，有105个为独家品种，新增进保的中成药则都是独家品种，涉及以岭药业、方盛制药、华润三九、广东思济药业等知名药企。此外，信达生物、康方生物等头部药企的核心产品，也悉数进入医保或商保目录。总之，我看下来这次国家医保谈判有明显的“放大商业价值”的态度。

从2018年“我不是药神”电影宣传后，医药高价问题进入了中国社会的视野，同年，国家医保局正式成立，并在随后开启了药品国谈、药品集采的一系列行动，以低价集采为核心的药品改革推行开来，至今已经进行了8轮药品国谈和11轮药品集采。

成果自然毋庸置疑，用药价格大幅下降，但也引发了部分关于药品是否会因低价而影响效果，以及进口特效药消失的担忧；与此同时，很多创新药也要在“保销量”和“保利润”之间抉择，这客观上不利于创新药的发展。特别是对于动辄投入数十亿、研发周期超十年的真正创新药，比如肿瘤、罕见病等领域的突破性疗法来说，容易抑制研发的积极性。

国家也一直在尝试解决这些问题，而从这次的国家医保谈判药品目录中，能够很明显地看到更优化的倾向：

首先，医保衡量药品的核心权重正在转变，从“极致低价”转向“药效价值”，也就是从“价格导向”转向“价值导向”。

过去多轮谈判和集采，核心目标是快速挤出药品生产流通环节的虚高水分，解决“看病贵”的燃眉之急，因此“降价幅度”成为最显性的KPI。而现在则是将药品的临床疗效、患者生存质量改善、对现有治疗方案的突破性意义等“实际临床价值”，纳入核心考量。

其次，中国加速进入医疗“双轨制”时代，基本医保目录和商保创新药目录，构建起了“保基本”与“促创新”的格局。

基本医保目录覆盖广，而商保创新药目录，则为具有突破性疗效、但价格高昂的“救命药”开辟了商业化通道，形成了“基础层+创新层”的支付分层。

而对普通人来说，支付体系的分层，对应的是治疗选择的分层。普通疾病，一般就选择性价比高的国家医保；而面对重大、罕见疾病，一个可能的选择就是购买相应的商业保险，获得使用前沿“救命药”的机会。

“用得起”和“用得好”，会是两种不同成本的抉择和获得感。这会是一个越来越清晰的选择问题。

再叠加上国家已经在全国范围内广泛铺开的DRG/DIP支付，就是从过去的传统医保按照项目结算，改为按病种付费，疾病分组定价。总之，这意味着，以后医疗会变得越来越精细，也越来越分层。国家医保和公立医院保基本需求，让所有人看得起病；商业保险和私立、外资医院保高端需求，提供更好的服务。你需要更清楚已经确定的变化，并早做打算。

具体说，我有这么三个建议，供你参考：

第一，可以关注每年的医保目录调整和商保药目录发布，了解哪些高价值药被纳入、对应哪些疾病。这部分我们这门课也会带你持续追踪。然后针对性地评估自身与家庭的疾病风险，尤其是重大疾病与罕见病方面的保障需求。建议每隔1—2年，重新审视家庭医疗保障配置，及时查漏补缺。

第二，如果家庭预算允许，可以考虑投保覆盖“特药目录”的商业医疗险或重疾险，重点关注保险条款中是否包含各类罕见病用药等保障。但你要注意区分产品是否涵盖“院外购药”“特定药品清单”等关键责任，因为有些产品虽然宣传覆盖特药，但实际清单范围很窄，或者报销条件苛刻，这点要特别注意，避免买了保险却发现关键时刻用不上。

第三，就医时主动沟通治疗选择，如果面临重大疾病诊疗，可以和主治医生探讨是否有纳入商业保险目录的创新药可选，以及如何通过保险或援助项目减轻负担。

好，第一个话题先谈到这里。

## 怎么看具身智能新本科专业？

再说第二个话题。

11月14日，教育部公布了首批申请增设“具身智能”本科新专业的7所高校的名单，分别是北京航空航天大学、北京理工大学、北京邮电大学、东北大学、上海交通大学、浙江大学和西安交通大学。

我之前在第243课谈国家发改委警示人形机器人产业时，明确跟你说过，具身智能的产业发展和具身智能这个专业本身的前景，是不会被当下人形机器人企业的低水平竞争所影响的，并给出了一个选择决策模型，你可以再复习下。

而这次教育部新设的具身智能本科专业，我认为其实就是一个明确的红利信号，产业还处于早期，进入的人也不多，所以就业不会那么卷，明年应该是会首批录取的，如果有条件，值得特别关注下。

而从这次教育部和高校的动作来看，发现了几个非常明显的特点：

第一，首批设立该专业的7所高校，无一不是顶尖工科强校，且都在机器人、控制科学、计算机领域有深厚积累，这说明国家对这件事很重视，就是要挑底蕴深厚的学校加快搞成，我进一步判断，这是个明显要完成的KPI指标。就是要给具身智能专业投资源，系统化、精英化地来培育高端人才。

第二，这个专业是直接以“具身智能”命名的，而具身智能领域本身，就是一个强交叉学科的产物，要组建这样一个学科，必然要打破传统高校中机械学院、计算机学院、电子学院、材料学院等传统院系间的壁垒，现在明显是在加速推动这种“新工科”的深度融合范式，未来的课程设置、实验室建设、师资配置都将围绕此展开，这背后其实还是集中砸资源。

第三，观察这7所高校的地理分布，北京，长三角的上海、浙江，东北的沈阳，西北的西安。这恰好覆盖了京津冀、长三角、东北老工业基地、西部的区域科技创新中心等关键区域，至于珠三角的大湾区虽然这次没有大学列入，当然这一定程度上说明了大湾区的内地高校还要继续加油，但我一点不怀疑上面的七所大学，可以通过自己在深圳的研究院所以及校企合作来覆盖大湾区。总之，这背后，我认为会突出与相关先进制造业集群深度融合的“订单化”培养，这意味着更好的个人发展。

第四，从培养方案上看，各个高校的具身智能专业，都呈现出高学分、高交叉、强实践的特征，感觉是要把四年制本科变成“迷你直博”，加速提升高校的资源竞赛。

另外，同样是增设了具身智能专业，但首批各个高校的侧重并不相同。

我观察到，上海交大、北京邮电、北京理工这三所高校，是把具身智能专业设置在了“人工智能学院”中，意味着可能会主要以软件研发为核心，比如算法、软件、智能系统、认知与交互、虚拟仿真、数字孪生等。

而浙江大学、北京航空航天、东北大学、西安交大，则是把具身智能设置在了机械工程及自动化、机器人科学与工程、控制科学与工程等“工程类”学院之下，意味着这些高校的方向可能更偏“硬件研发”，比如仿生结构、材料研发、能源动力、专用硬件芯片等。

所以你自己或者家里孩子在报考具身智能专业的时候，也要仔细甄别未来的发展方向，看看自己是适合“偏软”还是“偏硬”。如果不清楚或者尚未明确偏好，那么或许可以选择学科综合实力更强、跨学院合作更紧密的高校，比如上海交大、浙江大学，以便在校内探索后再聚焦。
