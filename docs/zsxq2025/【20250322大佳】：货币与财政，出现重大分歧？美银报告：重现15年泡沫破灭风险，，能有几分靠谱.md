
# 货币与财政，出现重大分歧？美银报告：重现15年泡沫破灭风险，能有几分靠谱？

## 财政部VS央行，不同的kpi导致的分歧

周五，市场经历了一个温柔但又普遍的回调。关于市场风险的讨论开始陡然增加。尤其是美银这周的一篇报告，被广泛热议。而截至上周五，我们A股组合和港股组合，均创历史新高。周五，A股组合上涨0.31%，总收益创历史新高。3月超额收益转正，今年超额为2.8%。港股组合上涨1.13%，总收益为16.85%。今天，我们和大家分享一下，我们对于通缩和美银这篇报告的看法与研究。以及对未来的政策推演，和我们自己的布局方向。在我们讨论A股的杠杆风险和获取超额收益的投资策略之前，我们需要明白A股市场的特性。A股是一个波动率极高，且每一次巨大的波动都和政策息息相关的市场。因此，在我们将A股应该如何投资，尤其是如何择时的时候，就一定要要梳理政策。尤其是央行和财政部，两大最核心部门的政策。前者，主导货币政策。后者，主导财政政策。这两个部门的kpi有着非常明显的区别，因此，在某些特定的时刻，都会有不一样的想法。周五，我们，金融市场最重要的消息，就是这两个部门的“不一样的想法”。今年2月以来，债市经历了腥风血雨的回调。1.75—1.830—中国：10年期国债收益率经过一度兵临1.9%城下。30债回调更加猛烈，一度跌穿到了9月之前的位置。这是在今年9月正式宣布降息之前的位置。国债猛烈回调，利率飙升的后果就是，整个债市，包括信用债之内的所有品种，利率全部开始飙升。再接着，就影响到了国债和地方债的发行。因为利率的飙升，意味着国债和地方债发行成本都会走高。对于这个结果，财政部显然是不满意的。那么,问题来了。为什么央行坐视利率飙升而不采取行动，及时的向市场注入流动性，而降准降息，又一而再再而三的推迟呢？很多人的观点，都认为是汇率。我们认为，汇率只是其中一部分的影响。实际上，今年以来，美元指数大幅度走低，汇率压力已经很大程度上缓解。其中，有一个被忽略的因素就是银行的净息差。每一次降息，每一次降准，都意味着货币政策的空间逐渐的缩小。同时，整个金融体系的风险在逐渐放大。尤其是贷款利率的下降，会让本来已经比较危险的中小银行的净息差雪上加霜。对财政部来说，国债利率走低，国债发行成本下降，财政负担降低，自然是一件好事。但对于央行来说，每一次降准降息，都在增加金融体系的潜在风险，自然是不那么情愿。这就是近期债市大回调的根本原因：国债和地方债大规模发行，但是央行不肯及时降准降息来进行配合。之前押注降准降息的交易盘被迫大规模止损。这也是近期债市企稳反攻的根本原因：利率的飙升，已经严重影响到了国债和地方债的发行。因此，被迫三个交易日注入5000亿+流动性，来防止利率继续飙升。

## 进一步萎缩的信贷和向股市传导的通缩

2月份的经济数据，我们在上周六的文章已经分析过了。我们的结论是，信贷进一步萎缩，通缩肉眼可见的加剧。而2月的时候，利率还没有飙升得如此明显。10债是进入到3月之后，才跌破1.8%关键点位的。那么3月的情况到底会如何呢？我们可以从黑色系走势，管中窥豹。南华黑色系指数一路走低，跌穿了今年1月的最低位，并且已经来到了去年9月大反弹之前的最低位。整个3月份，黑色系下跌的速度反而加快了。其实房地产销售数据更加明显。目前，一线城市维持了一定的热度。但是一线城市的销售主要集中在二手房。二手房的活跃并未带动一手房市场。根本原因是，二手房的业主出售房子之后，回收了现金，清偿了债务，但并不进入到一手市场。很明显，这种行为就是一个去杠杆。二手房的火爆和一手房的遇冷，更加说明居民的去杠杆进程并没有中止，而是在继续。虽然三月份还未结束，但是我们可以合理推测。在利率飙升的大环境之下，再加上已经生效的关税影响，三月份的数据会延续二月份的下滑状态。这种持续性的下滑，不只是反应在商品市场，也正在逐渐反应股票之上。
