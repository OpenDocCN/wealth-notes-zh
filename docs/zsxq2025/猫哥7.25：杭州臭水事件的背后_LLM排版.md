
# 猫哥7.25：杭州臭水事件的背后

聊一聊杭州臭水事件，这是一个很好的案例样本。通过这个样本，我们可以了解这些与老百姓生活密切相关的公共产品供给的一些深层次东西。

首先，我们复盘一下杭州臭水事件经过。7月16日，杭州余杭区仁和街道、良诸街道部分小区出现自来水味道异常，问题源头初步判定为东苕溪自来水厂取水口附近因高温与降雨导致藻类植物大量繁殖并厌氧降解，从而产生异味。

## 臭水真相真是如此简单吗?

我通过AI搜索一下高温+降雨导致藻类植物大量繁殖并厌氧降解的机制是这样，高温会刺激藻类植物代谢，但是代谢需要氮、磷等营养物质的参与，降雨会冲刷地面，将地面的氮、磷等营养物质冲入水体，两者叠加，雨后藻类植物迅速爆发。同时，暴雨导致水体浑浊，光照减弱，藻类植物死亡后沉降分解，大量耗氧，引发最后的厌氧降解。这一切看起来就是很正常的自然反应，但是问题又来了，为什么临近饮用水源水体附近地面会有大量氮、磷等化学物质?

这个就要从头说起了。杭州最重要的水源地是千岛湖，千岛湖水源的来源是安徽的几条河流，这里就有一个矛盾，安徽要发展经济，就可能影响千岛湖水源的水质，所以，2012年开始，中央出资3个亿，浙江与安徽各自出资1个亿，对安徽几条流向千岛湖的水源进行治理，只要水源达标，浙江每年补偿安徽1个亿，如果水源不达标，安徽反向补偿则将1个亿，这就是三方治理水源机制。

2015年中央财政退出，水源治理变成安徽与浙江各自出资2个亿，2023年，这个治理金额提升到10个亿，浙江与安徽各自出资4:6。从这个过程可以看出，浙江为了保证杭州饮水安全是花了大力气的，光是为了水源安全每年就要给上游安徽补贴几个亿。

但是杭州城市规模大人口多，光是一个千岛湖是保证不了全城饮水需求的，所以杭州城北一部分，主要是余杭区饮用水主要来自东苕溪，东苕溪水源来自于湖州市德清县的河流，为了保证上游水源地安全，余杭区与德清县也签订了生态保护补偿协议，模式也是一样的，水源达标，余杭区每年给德清县多少钱，具体金额没有公布，但是我估计这种区县之间合作，补偿金额不会高，大致就是几百万顶多千把万。

生态补偿对河流的治理有钱就是有钱的干法，钱少就是钱少的干法，比如如果钱给够，那么在水源地附近肯定是禁止农业养殖的，防止家禽粪便排污进入水源地。方法也简单，地方政府直接给附近农户一笔钱，以补偿其禁止养殖的损失。

但是如果生态补偿金不够，那么所谓的生态治理就不可能到位的，比如禁止农户养殖就做不到，农户养殖家禽排出的粪便虽然可以做到不直接向水源排污，但是地面积累相当多的粪便是必然的，平时没啥事，遇到暴雨，这些地面的粪便就会被冲入水源，叠加高温天气，就容易产生藻类植物爆发并且厌氧降解。

杭州市对饮用水安全还是比较重视的，但是平时监管的重点都在千岛湖，这个湖关系到杭州80%以上的饮用水安全，但是对于东苕溪饮用水的监管就相对薄弱了，上游补偿不到位叠加取水口监管薄弱，再叠加余杭区一些老旧小区二次供水设施改造尚未完成(杭州市规定新建住宅不得使用屋顶水箱，减少二次污染二次供水设施每半年清洗消毒一次)，几个因素阴差阳错叠加到一起，就出现了杭州臭水事件。

实事求是的说，浙江对饮用水安全的管理体系是全国最完备的，浙江也是全国少数省份将饮用水安全纳入县级政府政绩考核的——能做到这一点，不是全国其他省份缺乏意识，而是浙江有钱。

给大家一个数据，光是杭州一个市覆盖不过1000万人口，每年在饮用水安全投入就不低于200亿(包括农村供水、水环境治理、生态修复等等)。

我们再来对比一下西部城市。以重庆为例，每年在饮用水安全投入大致是120亿，重庆的优势是伴水而居，主要水源就是长江与嘉陵江，为了保证水源安全，光是长江川渝段治理，每年就要给四川省5个亿，四川省另外投入5个亿，川渝段长江治理金额就是10个亿，对于长江经过重庆段治理每年又是3个亿，当然这3个亿有一半是湖北买单。

因为人口集中在水源旁边，叠加水源地集中，叠加西部水源整体品质要高于东部水源(污染更少)，所以重庆饮用水品质在全国大城市长期排名前三，直辖市中仅次于北京。

我们再来看四川，四川每年在饮用水安全投入大致是88亿，88亿资金要解决覆盖多个河流8000多万人口的饮用水安全就有点捉襟见肘，所以四川在饮用水安全方面主要集中在本省河流治理，没有给上游(青海、西藏)生态补偿(当然青海、西藏经济不发达，对水源污染也少)。

几方面原因叠加，四川省饮用水水质也只有成都等少数几个城市能与重庆媲美，其余大部分城市存在汛期水质恶化(没钱给上游做生态补偿修复)，缺乏完备的多重闭环监管体系(重庆市水厂自检、区县检查、市级抽查)，一半县城顶多做到水厂自检，县城抽检；多数城市都做达不到饮用水安全106项指标全面自检能力(缺钱)。

通过上述描述，我们就可以知道，老百姓生活品质其实很大程度取决于公共产品的供给，而政府要提供优质的公共产品是特别花钱的，一个城市光是要提供优质饮用水，随随便便几十亿上百亿都是要花出去的。

这也是过去我撰文在知识星球提出，所谓的逃离大城市去小城市养老就是一个伪命题，小城市财政收入有限，提供的公共产品与大城市差距很大的，可能平时大家喝水感觉不出来，但是小县城养老平均寿命比大城市低6—10岁却是不可争议的事实。

说说杭州臭水事件的后续，杭州还是动作很快，现在在监管体系已经建立藻类植物高峰期加密监测机制，叠加杭州老旧社区二次供水系统今年改造完成，未来杭州大概率不会发生类似事故。

说实话，也就是杭州这样的国际性大城市，出现臭水事件马上全国都知道了，如果在一个小城市，大概率无声无息就过去了。
