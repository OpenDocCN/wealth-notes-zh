
# 大佳和他的朋友们4.23：关税，川第一次正式释放缓和之后的中性预期

今天，是川正式释放缓和信号的第一个交易日。我们首先确认一件事情，川是在释放缓和信号吗？以及释放缓和信号之后，接下来会怎么做？

1. 川是在释放缓和信号。
2. 川的缓和信号在于两点：
   - 认为现在关税过高。
   - 认为需要达成一个新的协议，而不是这种无序的状态。

我们认为，这次缓和信号，并不是因为川态度改变，而是因为制造业回流需要时间，且短期完全中断中美贸易，美国公司的供应链也同样受到极大的压力。

一开始，川本身征收的关税也就是在60%-70%。且这个关税本身就是为了谈判做的准备。所以一开始，最终可能达成的关税目标也就是在60%以下。

我们认为，最终的关税水平，可能在40%-60%。理由如下：
1. 40%-60%是一个双方可以接受的目标。双方都可以宣布，在关税上战胜了对手。
2. 在这个目标之下，中美贸易并不会完全中断。中美之间的必需品，依然可以正常交流。只不过变得更贵。关键商品，贵是可以接受的。
3. 这个关税水平，可以一定程度保证美国的供应链不中断，并且给美国重新塑造供应链时间。
4. 这个关税水平，中方可以加大退税与补贴，再配合一定程度的贬值，可以进行一定程度的对冲。并且，给国内政策留下空间。

那么，对市场有什么样的影响？
1. 我们认为，即使达到这么一个比较理想的水平，对出口的打击依然很大。国内依然需要出台以内需为主的政策。并且会加速我们经济转型。
2. 我们认为，对出海的企业来说，需要巨大的投入，来规避关税的风险。相当于无形增加了很多资本开支，并对未来盈利造成影响。所以，估值需要下调。当下的价格显然过于乐观。

今天市场的反应，看起来很奇怪，但具有合理性。
1. 商品市场大涨。主要是情绪修复，触底反弹。
2. 债市大跌。主要是MLF预期落空。且中美存在谈判空间，所以货币政策的预期可能会推迟。
3. 股票市场高开低走。主要是因为之前上涨，提前透支。且这个缓和的消息，并不是多么大的利好。因为40%-60%的关税，本身就是回到贸易战刚爆发的水平。

4. 港股大涨，但南下资金大卖。这是因为地缘风险消除之后，很多被动资金重新配置港股。南下大卖是因为净卖出逐渐解除之后，机构开始卖出降低仓位。机构一般会选择先卖港股后卖A股，或者多卖港股少卖A股。

需要注意的是，净卖令是逐渐解除，而不是一次性放松。

同时，今天的MLF保持不变，加强了我们的一个观点。即4月底，不会推出特别超预期的政策。真正强力政策，可能会在7月底。
