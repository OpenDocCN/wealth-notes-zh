
# 卢克文环球军情第68期-世界主要国家情报机构-土耳其

土耳其在中国国内的互联网，尤其是简中互联网上，大家好像经常讲什么狗中哈士奇、国中土耳其，对土耳其颇有种不屑的感觉。当然，这个我能理解，因为这个世界上，能够有中国这样悠久的历史和深厚文化的国家是没有的。

平心而论，如果我们把视角放平，土耳其这样一个横跨欧亚大陆，8500万人口的唯一的北约穆斯林国家，他所取得的成就确实可以让土耳其在一些领域有他的自豪感或者国家认同。当然你可以说他历史上被俄罗斯揍了很多次，强如俄罗斯不也曾经被拿破仑和希特勒追着屁股打，我们也曾经遇到过百年耻辱，所以我们如果把视角放平的话，土耳其这个国家以前可能他只是在地区，或者说是近东地区有他的国际影响力。

但是埃尔多安执政的这么多年以来，土耳其在国际上发挥着越来越大的作用，我们姑且不说他国内经济情况怎么样，但是至少他这些年在国际事务上扮演的角色越来越重要，越来越具有影响力，这个是不可否认的。我们举个很简单例子，2024年8月1日，土耳其国家情报局简称MIT，这个MIT不是美国MIT，是土耳其语缩写。

土耳其国家情报局成功斡旋了美国、俄罗斯等7国参与的冷战结束以来，最大的交换囚犯的行动，交换的地点就是在土耳其的安卡拉，双方一共交换了26名囚犯。就说在纵观整个从冷二战结束到现在，敌对的双方，或者在情报作业中能够进行交换情报人员，是所有的情报行动中极为谨慎和机密的行动，尤其是需要由第三方来斡旋的时候，那么第三方要取得另外当事两方足够的信任，才能够进行这种换服行动。

这个月在阿联酋、美国和俄罗斯也进行了有一次交换囚犯的行动。俄罗斯将一名持有俄罗斯和美国双重护照的一名被逮捕的人员交换给了美国，美国把一名在押的持有俄罗斯和德国双重护照的情报人员交换回了俄罗斯。那么俄罗斯被交换回去的这个人，他是在塞浦路斯从事所谓的非法军事装备采购的一个商人，他是做国际贸易的，但是他主要是采购电子设备的，主要的活动基地是在塞浦路斯，后来是因为他触犯了美国商务部和美国国务院的禁令，在俄乌战争爆发以后，仍旧向俄罗斯运送这些被管制的就是所谓的军民两用的电子产品。

去年8月1号，土耳其国家情报局能够斡旋美、俄、德等这些国家的情报人员在安卡拉进行交换，本身来说土耳其国家情报局获得了所有当事方的同意。应该说，这一点足以证明土耳其国家情报局在整个情报行业里面，它是具有举足轻重的影响力的。

2025年的1月份，土耳其国家情报局再次展现了其强大的这个情报能力。他通过与哈马斯的接触，将5名泰国人质从加沙解救了出来。这里面土耳其国家情报局游走在哈马斯、以色列、巴结组织、泰国政府四个方向，四个当事方之间取得了四个当事方的统一的信任，最终将这5个泰国人带回了泰国。

2025年的1月和2月，土耳其国家情报局的一把手在大马士革与叙利亚的新政府领导人，也就是朱拉尼，展开了接触，进一步凸显了土耳其在地区事务中的影响力。也就是说土耳其不仅把泰国人质从加沙带出来了，同时他在第一时间与HTS，也就是所谓的朱拉尼这些组织的人接上了头，可见土耳其在近东地区，在叙利亚、在中东在这些问题上确实扮演着很重要的角色，否则他不可能能够和这么多当事方发生这么多这么深刻的互信关系。

那么土耳其国家情报局的这种能力也不是一蹴而就的，罗马也不是一天建成的。长期以来，就说土耳其国家情报局始终在国际情报领域里面把自己塑造为一个值得信赖的合作伙伴。那么尤其是在调解人质危机中，土耳其方面确实展现出极强的协调能力以及战略智慧，他能够极好的在不同层面上与不同的组织满足他们不同的诉求，这一点是极其需要耐心和战略眼光。

举个很简单的例子，就跟你在家里做饭一样，你们家有5个人，甜酸苦辣咸5个人吃5种口味，那么作为主厨的，你又怎么能够让他们5个人都能够满足，而且还不发生矛盾呢？这个是很需要智慧的一件事情，更何况这个是人命关天的事情，你如果稍有不慎，这些人质可能会出现危险，像哈马斯或者HTS这些组织，其实他们的情绪有的时候并不是那么稳定的，所以这个是很需要技巧的一件事情。
