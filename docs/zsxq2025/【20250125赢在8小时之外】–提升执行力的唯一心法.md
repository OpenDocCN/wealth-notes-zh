

# 提升执行力的唯一心法 已付费

原创 赞赏U君 赢在8小时之外  
2025年01月21日13:33 海南  

大多数人都是拖延的高手，而且对于消耗自己生命和能力的事物欲罢不能。  

即使自己已经意识到了问题的严重性，但是由于缺乏正确的方法和思维，他们的身体仍然拒绝行动。  

① 有个人整天刷短视频，已经形成固定的习惯了，有时候甚至会出现这样一种状态：  

自己看到短视频，内心就会泛起一阵恶心，而且自己也非常清楚：继续刷下去，对自己百害而无一利。  

但是，自己的手就是停不下来。  

结果是，每天晚上躺下时，心里都会充满悔恨，他非常清楚自己的生命就是这样被一天天地浪费掉了。  

② 有个人受到了某些博主的忽悠，花了80元钱去电影院看一部新上映的贺岁电影。  

刚看10分钟，他就觉得这部电影非常垃圾，无论是剧情还是演技，都非常差劲，甚至令人恶心。  

但是呢，一方面他不想让自己的票钱浪费掉，另一方面他还期待着剧情会变好。  

其实潜意识里他非常清楚，越看下去就会越恶心，但是他还是忍着恶心把电影看完了。  

结果是，他不仅浪费了两个小时的生命数，而且记忆里又多了两个小时的恶心时光，每次想到这件事都会恶心一下。  

③ 有个人长得非常胖，他非常清楚‘少吃多动’的重要性，如果再不减肥，那么一系列的富贵病都会找上门来，甚至会要了自己的命。  

但是呢，即使他的大脑里已经波涛汹涌，塞满了恐慌和焦虑，但是这些念头都被困在一个拒绝行动的身体里。  

结果就是，他的灵魂在尖叫，但是他的身体仍然坐在沙发上吃薯片。  

时间长了，他会对自己非常失望，能量场和自信心都会严重受挫，最终变成个一事无成的废物，一过40岁就百病缠身。  

那么问题来了，到底该如何提升执行力呢？  

我总结，最重要的心法就是‘切割’。  

这个切割主要分为两个维度，一个是物理层面的，一个是心理层面的。  

只要你能够领会我心法，并且愿意去执行，那么你一定会拥有牲口一样的执行力。  

另外，拿死工资的打工者就不要继续向下阅读了，因为打工者需要融入一个大的‘集体’，身边聚集了很多穷鬼和废物。  

如果你敢搞立独行，那么他们肯定会把你当成神经病。  

再说了，拿死工资的打工者提高执行力也没有太大意义，一天8个小时，无论你是执行力爆棚，还是磨洋工，你的工资都没有什么区别。  

由于我会提到一些非常反常识、毁三观的思路，所以我必须设置成付费阅读。  

## 1. 物理切割  

互联网越发发达，信息流越丰富，那么自制力差的人越倒霉，因为99.99%映入你眼帘的信息流都是有毒的，都是消耗你生命的。  

比如，我跟大家细数一下最近几年的热点事件：鹅乌冲突，铁链女，唐山打人，明星出轨，发布新iPhone，美国大选，缅北诈骗，特郎普发市……  

这些事情你关注过几个？  

如果你关注过哪怕其中一个，而且是作为消费者去关注的（而不是自媒体人、新闻从业者），那么你就是有毒信息流的受害者。  

我能想象，你每天会花几个小时浏览这些跟你没有一毛钱关系的新闻，你的情绪会被有毒信息拉扯上下波动几十次。  

到头来，你的收入增加了吗？你的幸福感提升了吗？  

大家不要以为，坐在家里、刷刷手机，就尽知天下事。  

这件事的本质是：你把全天下的毒都吸过来了，不断毒害自己的大脑和灵魂。  

所以，必须从物理层面，对这些有毒信息进行切割。  

比如，把手机里所有的社交软件、购物软件全部卸载。  

具体来说，手机里只保留微信、支付宝、喜马拉雅FM和一些刚需购票软件就行了。  

其他所有可以‘刷’的软件，一律卸载，比如短视频APP，购物APP，小红薯、知乎等等。  

卸载短视频软件很好理解，因为它是人类历史上最大的信息毒瘤。那么为什么要卸载购物APP呢？  

原因也很简单，很多APP的开屏广告都会自动跳转到购物APP。  

而且大数据非常聪明，你最近在浏览什么商品、对什么商品有购买意向，他们都一清二楚。  

所以，只要你手机里有购物APP，那么你点开的概率非常大，而且打开一次，就要消耗你一两个小时的时间。  

你本来是想买A产品，但是下面会有很多相关的、价位差不多的B产品、C产品、D产品……  

你一浏览，一个小时就过去了，两个小时就过去了。  

所以，手机上绝对不能安装购物APP，如果有购物需求，就通过电脑网页解决。  

为什么我说这个方法不适合打工者呢？  

因为大部分打工者所处的集体，是一个弱智群体，他们的注意力和灵魂都不属于自己。  

比如，无论你在什么单位、什么办公室，你们闲聊的话题无非就是现在比较热门的新闻，比较当红的网红或者明星，以及新发布的各种电子产品、汽车等等。  

如果你听了我的建议，把那些获取信息的APP全部卸载了，你的手机变成了老年机，那么你的同事会用什么眼光看你？  

提到老年机，如果你去购物网站看一下，很多购买老年机的并不是老年人，而是大学生，他们在备考四六级、考研、考公的时候，都会购买老年机。  

因为他们非常清楚智能手机的毒害，以及自己的自制力有多薄弱。  

## 2. 心理切割  

下面我跟大家说一条犀利的、极端的、让人汗毛孔竖起来的心法（只适合极少数创业者、老板，所以我必须设置成付费阅读）。  

只要你领会了这条心法的十分之一，并且严格去执行，那么你的收入绝对会突飞猛进，但同时也会牺牲掉很多自己珍视的东西。  

这条心法就是：把自己当成赚钱机器，把自己的灵魂献给金钱。  

每天从早到晚、每年365天，只吸收跟赚钱相关的信息流，只接触能帮助自己赚钱的人，只做跟赚钱相关的事情。  

也就是说，你得从心底认同，这个世界上最重要的事情就是赚钱，自己活着的唯一使命就是赚钱。  

而且你要戴上有色眼镜，每当你看到有人在刷短视频，在刷购物软件，你就能看到他们的灵魂正在被某个恶鬼吸食。  

你自己一定不要沾染。  

至于跟同学聚会、跟朋友聚餐、跟朋友去演唱、亲戚朋友婚丧嫁娶，这些一概不参加。  

你必须一心一意、全心全意把自己投入到赚钱的事业当中去。  

有人会说：活成一个冷血的赚钱机器，不觉得悲哀吗？生活中还有很多比赚钱更有意义的事情。  

还有人说：等我赚到多少钱之后，就躺平/环游世界/包养多个二奶……  

这两类人的心智模式决定了，他们注定赚不到钱。  

前者如果去做‘比赚钱更有意义’的事情，不管是泡妞也好，还是跟同事聚餐也好，还是陪家人也好，还是旅行也好，最终一定会陷入‘无聊’的状态。  

后者如果认为赚够了多少钱之后就去做某个‘跟赚钱无关的事’，那么说明他的信仰出了问题，财神压根就不会光顾他的家门。  

之前还有人问过我：U君，你平时在工作之余，有什么爱好和娱乐吗？  

其实，我真的没有什么娱乐活动，因为传统意义的那些娱乐活动，压根就不能给我带来愉悦感。  

写作、收钱、搞新项目这些事情才能给我带来愉悦感和快感。  

换句话说，我的生命里几乎只有‘搞钱’，其他的事情都无法吸引我的注意力。  

另外，很多人之所以没有执行力，是因为他们没有危机感。  

每天磨洋工8个小时、每个月赚5000元就挺好，也饿不死，那干嘛拼什么？  

但是呢，这片土地很特殊，跟世界大多数土地都不一样。  

这是一片盐碱地，一旦你过了30岁，40岁，你就会认清这片土地有多贫瘠、人均资源有多稀缺了。  

到了那时候，如果你仍然没有执行力，那么你将会长时间地在苦海里挣扎。  

不仅没有保障，他们还都是被收割的韭菜、待宰的羔羊。  

我最佩服的华人企业家就是潘石屹和李嘉诚。  

如果把人比作猪的话，普通人知道被捉住了、捆上了，才开始发出惊天动地的哀嚎。  

而潘和李这样的智者，只要听到风中飘来‘烧水’两个字，就会撒开蹄子开始外跑，要么跑进深山老林，要么跑到大洋彼岸。  

还有一周就到乙巳蛇年春节了，我希望各位都好好感受一下，到底是做事业、赚钱这件事给自己带来的愉悦感更充实，还是串亲戚给自己带来的愉悦感更充实。  

另外，你千万不要说自己身不由己，没人拿枪逼着你去串亲戚。  

你是一个有独立思考能力、有自制能力的成年人，千方不要说什么外在的世界约束你。  

真正束缚你的是你内心的恶魔。  

如果你不肯杀死内心的恶魔，那么你将会长时间地在苦海里挣扎。
