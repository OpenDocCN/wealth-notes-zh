
# 如何理解5%的增速？

在四季度超预期增速5.4%的带动下，中国2024年全年GDP增速为5%，完成全年目标。但关于5%的增速以及中美GDP差距拉大等，有很多容易迷惑的地方，不同说法很多，比如1月16日贝森特在财长提名确认听证会上，说中国经济可能有一个4%的通缩。

而对美国经济，既有人说拜登就任四年，美国GDP从2020年的21.35万亿美元增长到2024年的预计29.2万亿美元，增长超过35%；但查看美国公布的每年增长，除了2021年暴增6%，其他年份都在3%以内，这两个数对不上。看起来很糊涂吧。其实主要是口径不同。

一般说总量的时候，用的是名义GDP，当期价格；但当谈到增速的时候，用的是实际增速，也就是剔除价格变动的不变价格计算，以反映经济实际产出变化。而名义增速与实际增速的差值就是GDP平减指数，相比CPI、PPI，这个指数更能全面反映物价变动情况，一般认为GDP平减指数长期为负，就认为进入通缩阶段。2024年中国名义GDP增速为4.23%，实际增速为5%，因此2024年GDP平减指数为-0.77%。

自2023年二季度开始，中国的季度GDP平减指数已经连续7个季度为负值，而且预期未来一段时间也无法转正；30年来，中国只有在1998年-1999年曾经出现过连续6个季度的平减指数负值阶段，这段时期被公认为出现了通缩。因此，中国目前已经处于通缩阶段，这也是国家最近加大刺激力度的关键原因之一。

中国每年3月公布的年度GDP增速目标，一直是实际经济增速，这也是因为中国长期以来，大多数情况下经济更容易过热，选择不变价格的实际增速目标更有利于跨年对比实际增长情况，而且大多数情况下，名义增速大于实际增速，只要完成年度增速目标，名义经济增速也肯定没问题，所以长期以来，名义增速也不被看重。

但现在情况变了。近期越来越多的经济学家认为要加大对名义增速的重视度，希望国家加大刺激。参考1998-1999年经验，当时就是采取积极财政政策加大基建，再叠加体制机制改革和加入WTO，最终摆脱了长达一年半的通缩。

但现在情况跟20多年前大不一样，当年中国整体上是供不应求，只要扩大投资、提高产能，就能够经济增长；但现在是产能过剩，基建已经达到相当高度，再加上国际贸易摩擦加剧，改革又进入深水区，再想用过去的老招数很难奏效，也就意味着这次通缩持续的时间还会更长。

搞清楚不同口径的差异，也就能理解，不同说法都是对的。中国确实完成了5%的增速目标，美国人说通缩-4%也不是胡说。美国的说法是以名义GDP增速为标准，再叠加人民币汇率贬值的影響。人民币兑美元从年初的7.1左右，升到年末7.3左右，人民币全年贬值3%左右，加上平减指数的-0.77%，大概就是贝森特口中-4%的出处。

综合上述，因为中国近两年来名义增速较低，且人民币贬值，就导致折算为美元，中国经济总量占美国经济总量的比重就从2021年峰值的75%，一路下降到2024年的约64%。

按照大国博弈标准，一般认为新兴大国占守成大国的比重在60%-85%之间时，是新兴国家最为承压的时候，超过60%就会导致守成大国全面狙击新兴大国，而超过85%一般认为新旧大国之间的博弈从不对称博弈转变为对称博弈，守成大国就得开始转变策略，从单方面压迫转为谋求共存。

这也是近期对中国名义GDP和通缩关注度上升的原因之一。因为按照2021年当时的乐观预期，国内外普遍认为2025年中国经济总量占比大概率突破85%，中美大国博弈将开启共存博弈的新阶段。虽然这些经济理论都是总结历史经验，并不能真的全信，但这也是很重要的参考因素，如果按照2022年以前的趋势发展，大概到2027年，中国经济总量将超过美国，美国就必须转变对华战略。所以在美国2022年发布的诸多报告中，都认为中美战略博弈的关键节点是2027年，还认为中国将在2027年前后实现统一等，认为留给美国的时间不多了。

只不过2023年之后形势变了，中美经济总量比重进一步拉开，目前64%的大致比重已经回到了2018年以前的水平。或许拉大中美经济总量差距，也是美国加大刺激的原因之一，毕竟保持世界第一，对美国来说很重要，关系到霸权的稳固。

当然，短期的波动很正常，美国这几年名义增速远超实际增速，这也是很严重的问题，意味着增长更多靠通胀拉升。而在中美博弈焦灼的这几年，中美比重短期内波动并不会改变大势，但客观地说，从2021年到2024年，以美元计价，中国规模与美国进一步拉开，确实需要引起关注了。

至于有些人质疑四季度增长5.4%。其实也好解释。9月之后加大刺激力度，首先影响到生产端，而季度GDP统计用生产法而非消费法统计，四季度生产端的工业、服务业增加值同比增长分别为5.7%和5.8%，高于需求端的社零同比增速3.8%，因此季度核算口径的不同就导致较低的社零和投资增速，带来了较高的季度GDP增速。

另外还因为预期美国加征关税，所以四季度抢出口。海关方面统计四季度外贸实现了11.51万亿元的季度历史新高，占全年43万亿元的26.77%，12月更是首次突破4万亿元，同比增速达6.8%。当然也可以说是为了完成全年目标，所以四季度用力很猛，但这个数据是说得过去的。毕竟也只是初步核算，来年还需要进行终审核算，比如2024年12月公布的2023年终审核算，就必初步核算数增加了33690亿元。

以上分析零零碎碎的，但综合上述，可以对未来政策进行预判：
1、因为连续三年以美元计较，中国GDP较美国占比持续下滑，已经降到64%的位置，接近60%关键节点，这就意味着接下来必须保持汇率稳定，央行工作很多很繁重，但保汇率是最优先的。
2、既然保汇率更为优先，而2024年四季度在发力后实现超预期增长，那么短期内就会暂缓降息，避免进一步拉大中美利息差，导致面临更大的贬值压力；而暂缓降息，也就意味着去年定下的货币政策适度宽松，也会缓一缓。

暂缓降息、保汇率的政策预判就更成立了。
