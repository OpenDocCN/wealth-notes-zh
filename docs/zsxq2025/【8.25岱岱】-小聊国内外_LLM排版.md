
# 【8.25岱岱】 小聊国内外

李在明走钢丝的技术还是挺高的。李在明的主要投资来自于中国，我们看好他投资了他，给了他朝鲜族的保安帮他在那次政变中安然度过，但他上位后很细心的在中美之间走钢丝。我们阅兵是邀请他的，他说不方便来，懂王面子上是可以接受的。那李在明也是要首访的啊，之前韩国首访基本是美国，李在明不去阅兵就是不去中国，如果再去美国就有倒向美国之嫌。所以，李在明去了日本。首访放在日本，而且去日本谈关税结盟一起和懂王谈判的事，还暗示要对懂王打美韩联合部队的筹码，不接受台湾有事韩国部队要介入台海。李在明这么做，中国也满意了。

中国更满意的是李在明看普京懂王在谈半岛，立马出来出不接受任何形式的半岛统一。韩国也出了他们的安倍晋三。现在日本首相石破茂比较危险，基本要被懂王搞下去，就看能拖多久，石破茂能拖到关税谈判结束后下台，就完成历史使命了。日韩目前没多大问题，这是好事。然后是印度莫迪要访华了，阅兵来。和巴基斯坦一起来看阅兵。莫迪的脸皮也是真的厚。现在中印在谈判，不要寄予厚望，印度是拿和中国谈判来促进印美关税谈判的，印度放风故意说中国肯产业转移肯卖盾构机啥的，来刺激懂王。

中国拖着印度就行，真给了莫迪立马转身卖了美国和懂王重归于好了。马上9.3阅兵了，估计这几天还有台海演戏。8月17号，由渤海钻珠号、渤海宝珠号、渤海翡珠号、中华复兴号、永兴岛号、长山岛号和普陀岛号等7艘滚装船组成的船队南下，分析认为这是要在南方进行进一步的“抢滩登陆”演练，更加贴近实际使用环境。鉴于岛内媒体宣称20日上午，8艘汽客滚装船在泉州外海集结待命，因此不能排除，上述电子战和情报收集飞机在泉州和台中以西空域进行严密布控的主要战术目标，第一是对登岛演习编队实施空中电磁/电子战防御作业，另一方面也是与台军侦察飞行单位进行电子对抗和电子情报收集而来。对于这般大阵仗，美军也是十分重视，派出了C-135W侦察机在东海、黄海上空晃悠了半天进行抵近侦察。

前几天赖清德发言很嚣张，9.3前后会有大规模登陆演练回应赖清德的。聊下国际，看下国内。前天干结文章很多瓜友是没看仔细，说干结还有新闻出来啊。那是22号的新闻，23号没新闻。22号是各个代表团到各地方慰问，都有新闻，23号新闻是代表团返京。返京当天上午各团都参加了活动，唯独没报道他。看新闻要看仔细时间啊。而且，叠加狗头侧去山东的新闻连起来看，就更明显了。

除了干结就是郭树清了。但郭树清在山东只是省长，他主要是去当救火队长的，帮山东化解200多亿的债务，要查郭也是查银行查证监会，不会选山东查。拳击赛的确到了白热化程度。马上9月，9月初阅兵，然后9月下旬美联储降息，10月马上大会，然后是出国访问会晤懂王，中美关税停战时间到立马又要开始谈判。9月10月都是紧锣密鼓的。8月明显要提速，才能为后面两个月准备。瓜友说我最近的文章太刺激了，刺激到都不敢看了。没办法，拳拳到肉就是这样的血腥。

这种高强度的斗争也无法长时间持续，搞到10月希望能消停一会吧。届时我们就盯着几个点看。一个是我们三个位置是否会让出一个。一个是是否有太子谁是太子。一个是侠是否会成为戎装常委。这三点，基本就能看出拳击赛谁赢谁输了。当下一系列眼花缭乱的反腐斗争和大案深挖，无非就是围绕这三点展开罢了。然后是股市。股市能稳住，稳到9月的美联储降息，就是胜利。不要再拉了，再拉了大家反而会心有不安。但现在貌似是国家队不想怎么拉，机构和散户上头了，生怕牛市起飞自己踏空，一个劲的往里头冲。去年拉股市先胜后败，无非内因和外因。内因是六尺巷神仙打架，金融亲美派踏空牛市搞不配合要倒车接人。外因是美联储反应过来坚决叫停了降息不给中国持续拉牛市的政策空间。9月美联储降息基本笃定，这个外因没了，现在就盯着看六尺巷这个内因。

我不想看到肖钢这个人出来，上次我们外访那几天，肖钢就出来唱空股市，我就说大事不好。当然，证监会那个退休很多年的人都进去了，深挖15年股灾黑幕，这把火是烧向肖钢去的。肖钢要是再出来，要么是唱空继续和我们斗，要么就是绣红旗求我们饶命了。然后是那个持刀杀小孩的事。
