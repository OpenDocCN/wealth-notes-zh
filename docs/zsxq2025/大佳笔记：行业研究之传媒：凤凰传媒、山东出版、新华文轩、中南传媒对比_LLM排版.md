
# 大佳笔记：行业研究之传媒：凤凰传媒、山东出版、新华文轩、中南传媒对比

## 1、主营业务分析
四家企业属于新闻和出版业，行业整体发展较为平稳，具有较明显的弱周期特点。四家企业的主营业务包括：图书、报刊、电子出版物、音像制品的编辑出版、发行，公司教育出版、一般图书出版、图书发行。且呈现地区化垄断性。凤凰传媒在江苏，山东出版在山东，新华文轩在四川，中南传媒在湖南；几乎省内的出版发行生意都归他们所有。四家公司的主营业务其实大同小异，我们做一个简单的对比分析。

### 1）凤凰传媒
教材和教辅优势明显，拥有18套国家目录教材，同时长三角属于教育较强的地区，在长三角的教育生态中拥有巨大的优势。同时拥有学科网平台（合作学校4万所）提供题库、教学资源服务。在教培领域，凤凰传媒的优势不仅是局限在江苏省，而是全国性的。在生活和文学类图书市场，凤凰传媒也具备很强的竞争力；生活类、文学类图书零售市场常年排名第一。

### 2）山东出版
聚焦教材教辅，主要在山东省内市场主导，年出版图书1.4万种，再版率76%。推进智慧教育平台、研学文旅、和数字出版与教育信息化融合。

### 3）新华文轩
四川省唯一中小学教材发行商，教材教辅收入占比超60%；覆盖全学科教辅、近两年在布局教育信息化。

### 4）中南传媒
聚焦教材教辅市场，收入占比超75%；在湖南省内市场主导。同时职业教育教材走出湖南，覆盖率全国市场。近两年在布局数字教育平台。

综上来看，四家过往的业务基本盘和未来的规划布局大同小异。主业基本盘都是专注于教材教辅市场，同时在探索数字化、教育信息化等领域。

## 2、财务情况对比（凤凰传媒、山东出版、新华文轩、中南传媒依次）

### 1）营收
从营收体量上看，在经济教育大省的凤凰传媒排名第一，中南传媒排名第二，山东出版和新华文轩接近。从增速来看，新华文轩在2020-2024年连续保持了5年的连续正增长，增速水平排名第一；中南传媒排名第二，山东出版和凤凰传媒差不多。2025年中报，四家企业同时出现营收下滑，原因有以下两点：1.四家企业的营收发力点都在教培教辅领域；在校学生数量见顶，但全国中小学在校学生数量因出生率下降进入下行周期。江苏和四川虽然勉强维持了人口净流入，但增速也放缓了，因此导致教辅的需求量下降。山东出版下滑幅度最大是由于山东省提出了“教辅征订方式新规”， 学生和家长有权自主决定是否订购教辅，学校不得强制；因此压缩了教辅销售规模，进一步拖累营收。2.2025年上半年全国图书零售市场同比下降9.64%，经济下行的周期中，大众图书消费意愿下降。

### 2）扣非利润
从扣非利润来看，新华文轩依旧是最优秀的，保持了连续五年的正增长，且2025年也有望实现正增长。凤凰传媒在增速上领先中南传媒，山东出版垫底。营收下滑，除了山东出版外，另外三家扣非利润增长的原因在于：1. 通过削减职工薪酬、宣传推广费等降低销售费用。2. 所得税优惠或免税政策延续，导致所得税费用减少。3. 各家线上教育、数字化教育发力。

### 3）毛利率
从毛利率来看，中南传媒的近五年的平均毛利率最高，超过40%。另外三家的毛利水平差不多，新华文轩和凤凰传媒2025年的毛利率有较大幅度提升，而山东出版的毛利率在25年有所下降。原因在于山东省的教辅新政导致的需求疲软，进而导致售价下滑。

### 4）应收账款
从应收账款来看，四家企业的应收账款在2025年大幅度提升。里面表现最好的是新华文轩，应收账款增长幅度最小；山东出版和中南传媒的应收账款环比2024年年报增长了接近50%，凤凰传媒表现最差，环比增长接近200%。主要的原因还来自于行业需求萎缩和地方财政支付能力下降。

### 5）库存
从库存的情况看，2025年上半年，除了山东出版，另外三家的库存环比2024年出现下降。说明出版企业已经意识到了需求不足，开始从生产端控制生产，进入到清库存的阶段。

## 3、估值对比&分析师一致性预测
由于四家企业属于传统行业里的传统企业，我们参考PE（TTM）和股息率来看当前估值水平。

### 凤凰传媒
股息率4.7%；PE为13.82X，位于近一年的29.75%分位，近三年的70.07%分位，近五年的81.42%分位，近10年的49.65%分位。

### 山东出版
股息率3.99%；PE为13.66X，位于近一年的25.62%分位，近三年的73.93%分位，近五年的84.39%分位，近10年的82.95%分位。

### 新华文轩
股息率4.04%；PE为10.83X，位于近一年的9.09%分位，近三年的29.66%分位，近五年的55.57%分位，近10年的30.15%分位。

### 中南传媒
股息率4.49%；PE为13.53X，位于近一年的1.24%分位，近三年的43.45%分位，近五年的63.09%分位，近10年的31.66%分位。

从PE估值的角度看，新华文轩的估值水平在历史区间来看是较低的，其次是中南传媒，凤凰传媒，山东出版由于业绩较差，导致估值水平位于历史的较高区间。

## 4、投资判断
综上分析，新华文轩从几乎所有的指标来看都是这四家传媒龙头里的优等生。中南传媒和凤凰传媒次之，山东出版受到省内教辅政策的调整影响较大，未来几年的业绩可能会持续受影响。未来如果要长期跟踪出版这个赛道，需重点关注以下几点：1.政策持续性，地方的免税或税收优惠政策是否延续；2.库存和应收账款的变化情况；3.数字化教育的转型和线上渠道的进展情况。
