
# 教父的交易

原创 顾子明 顾子明说2025年03月22日00:14

中文自媒体，总喜欢把一些不能摆上桌面的生意，套用正常生意的逻辑，在这轮港口争夺战中展现得淋漓尽致。其实，港口这个生意，就不是一般人能够玩的，都是最顶级的家族在操纵。

譬如中国买的澳大利亚Cape Preston港时，是华人届最顶级最尊贵的两个家族联手，论地位，李家跟他们比还真不够看。同样，买走李嘉诚巴拿马港口贝莱德，目前可以看出被穿透的股东，地中海航运背后则是尊贵的西西里“教父”。

教父们做的是什么生意，看过电影《教父》自然很清楚，MSC港口航运的那点利润率，教父们是肯定看不上眼的。但是，遍布全球的航线，能够为教父们的非法生意，建立全球的交易网络，而巨额的跨国贸易流水，也能帮助教父们把大量的黑钱洗干净。

凭借着欧洲最大洗黑钱的这个生意，教父们就能够跟欧洲最顶级的权力者和寡头们平等的合作，无论是红衣主教还是参议员，也都是可以暗中操纵的。甚至这一轮俄乌冲突，普大帝想要绕过西方制裁获得关键的物资，也需要求着这些掌握着航线的教父们，帮助他把石油换成欧元，再用欧元换成前线急需的物资。

对照干同样生意的霍家荣家就知道，他们都是各国最高权力者尊贵的客人，在需要的时候，超级大国的军事力量也会为他们的海外利益保驾护航。

而这些重资产，对于正常的生意人来说，是根本搞不定的。看过《美人鱼》对吧，没有上层的关系，政府光靠环评就能把港口卡的死死的，更不要说，西方强大的工会组织，以及那些秘密结社的力量，你要是没有点黑手腕，能让你的港口一年300天趴窝。

正常商业组织来做这个生意，搞正经的贸易，不去参与洗钱和走私，搞这玩意，最终的结果只能是亏成狗。而只要你想明白这一点，就会发现。国际上的这类重资产，跟中国的央企其实一模一样，在各个领域都是一副薛定谔的状态：盈利能力很强，但分给股东的利润很低，价值很高，但资本市场的估值很低，在经济体系中很重要，但大众视野里的知名度很低……

你可以从二级市场上买到股票，但不好意思，你永远也无法从二级市场买到资产的控制权，这些资产的控制权根本就不可能给你。

所以呢，港口买回来这事儿现在就别想了，这相当于允许美国获得国资委下属央企的实控权，这就从来都不是价格的问题。一旦这些非卖品的重资产换手，往往意味着顶层教父们的站队出现了变化。

资产变更的背后，是教父和教父们勾兑后的交换地盘和重新组建联盟关系，是大佬们闭门会议商讨出来的结果，是他们在站队上，明确支持了那个西边的太阳。至于那些代持的职业经理人，也只不过闭门会议结束后，听从安排来做交易而已。

因此，我们的敲打，也不是针对李某人，所谓打狗要看主人，很多时候打狗也是给主人看的。
