发布于2025年4月1日

懂王在上台前，一直期待自己仅剩的几个东方老朋友能够给予他支持，帮助他一起对抗美国 Deepstate，设定了俄乌24小时停火，巴拿马交易，TikTok 交易，甚至期待把自己的首次出访，破天荒的给于中俄，搞出了四月访问北京，五月访问莫斯科的计划，试图在莫斯科的红场上搞出一个三皇同盟的震惊画面。但目前看来，中俄都在跟特朗普打太极，虽然三位一把手的私交都很好，但是各种力量都在阻挠三皇之间的友谊，譬如有人支持泽连斯基搞砸协议，譬如欧洲宣布全力援乌，譬如缓和的台海又被推着成为漩涡，譬如早就知晓的巴拿马事件被炒成热搜，譬如各方共赢的 TikTok 协议也迟迟未能签署。
