
# 天机不可泄露：当前社会的阶层攀登路径，就剩这么几个了

有人看了我的文章后特别激动，就留言质问我说：人和人之间都是平等的，没有谁比谁高贵，社会也应该是平等的，为什么你一定要把社会人为地分成上、中、下几个阶层呢？你是不是在制造矛盾？

我想说，人和人之间地位是否平等、有没有高低贵贱的区别，这个仁者见仁、智者见智，每个人都可以有自己的看法，没什么好争论的。但社会阶层这个概念，虽然是人为定义的，但也确实是客观存在的现实，谁也无法否认阶层的存在。

纵观古今中外，到目前为止，还没有出现过一个没有阶层的社会。不承认社会阶层的存在，就像是掩耳盗铃，这样做只会让自己变成聋子，但别人还是听得见铃声的。

如果自己给自己原本明亮的双眼，蒙上了一层厚重的迷雾，对于我们观察和分析社会现实、提高对社会认知，实在毫无帮助。

当然我也理解这些人的激动心情，毕竟很多人从小都是受“伟光正”的思想灌输成长起来的，看到残酷的社会现实或者一些没人敢讲的社会真相，一时难以接受也很正常。对这类人也不必和他们争论，因为他们要么有一天会在现实的毒打中自己醒悟过来，要么会永远带着幻想过完虚幻的一生，不一定是坏事。

当然，这些跟我都没什么关系了。毕竟，还是有很多人，有足够的阅历和理解能力，能够听懂我说的话，也知道我说的东西，对于一个人的一生的影響有多大。

其实，认识社会、认清社会阶层的本质与运动，本就应该是一个成熟的社会人，进入社会之前的必修之课。很可惜的是，我们能听到的、能学到的，大多是很虚假空洞的东西，很少有人能够幸运地在人生的早期学到这一课，包括我自己。我想，如果我们能够更早地懂得这些，那么人生可能会走的更远、更充实。

关于社会阶层的结构，我在之前的文章里和群里都提过很多次了，有兴趣的可以去翻之前的文章，或者也可以进群和我交流。本篇文章来讲讲，当今社会的主要的阶层攀登路径，供大家结合自己的情况去思考和选择。

上图的金字塔型社会结构图，是我认为对当前社会认识较为全面的结构图。社会犹如一个瘦长型金字塔，上端尖锐细长，底部宽大厚重。越是靠近顶点，越是社会精英，社会地位越高；越是靠近底部，越是社会底层，社会地位越低。整个金字塔社会分为上、中、下三个大阶层， L1-L9 九 个 小阶层。三大阶层划分的核心依据是资源。也就是说，不同阶层掌握的资源不同，对资源的掌握程度也不同。

是否拥有资源，拥有什么样的资源，对资源的掌握程度(即拥有资源的分配权、使用权还是经营权等)，是社会分层的核心逻辑。这个逻辑，同样也是一个人能站上什么阶层的决定性因素。

很明显，三大阶层所拥有的资源是不一样的。

上层拥有的是政治资源。
什么是政治资源？政治资源本质上是分配的权力。上层在国家层面、地区层面、行业层面，拥有资源分配的权力，能够决定国家、地区或者行业的规则制度、政策走向、经济方向。比如零八年开始的大放水、棚户区改造，造就了房地产行业的辉煌，同时也直接让一大批拆迁户、炒房客成为新的富裕阶层；而对新能源的补贴、扶持，造就了最近几年新能源汽车行业的突飞猛进。

中层拥有的是经济资源。
经济资源是指经济活动过程当中的生产资料，中层拥有其所有权、使用权或者经营权。这个阶层跟上层关系密切，一般是制度的实际推行者、政策的承接人。换句话说，上层有资源的分配权，而中层就是负责把这些资源分下去的任务执行人。俗话说，近水楼台先得月，因此在这个阶层的人，也是最容易被获得实际的经济资源的人。

下层是人力资源的提供者，很少拥有资源。这里我说的人力资源，是指个人所拥有的体力、脑力、心力等，个人先天具有或者通过后天习得的，可以创造社会价值的能力。根据各人提供的人力资源属性的不同，下层大致可以分为体力层、知识层、管理层。

下层的核心特点是，不拥有政治资源或者经济资源，也很少拥有生产资料。白话就是有一些钱但不多，只能维持生存、生活，无法用来创造收益；有一些个人权利，但没什么社会权力，无法决定资源分配或参与资源分配。

以上是三大阶层的分层逻辑，能看懂的理解能力已经不错了，看不懂也没关系，可以在群里和我交流。至于是否有道理、是否客观存在，你可以自己判断一下。

下面开始讲当今社会的阶层攀登路径。我盘点了一下，目前合法的、常见的阶层攀登路径，真的不多了，大概只有四条。

第一条路径是大家都知道的：打工人路径。这个路径能达到的阶层上限是中层 L6 层。打工人适合底层的阶层攀登，因为它的门槛低、起步快。比如二十年前，城市化刚刚开始，六、七亿农民几乎都处于最底层的 L9 阶段。城市化运动的最大作用，就是让大量农民去城市打工，把他们的身份从农民变成城市工人、新市民。

试想一下，一个农民辛苦务农一年，收入不过几千块钱；但是变成打工人之后，年收入就能迅速提高到几万元，阶层也能很快从 L9 层上升到 L8 层。所以，打工人路径是底层快速提升小层级的最快方法，但对于普通的农民来说， L8 已经是上限了。

在打工人路径，要想往上走，核心“装备”是文凭，更准确地说，是大学文凭。没有文凭，能选择的工作很有限，只有在工地、工厂或者服务业等地方谋生。在这些地方干得再好，拼尽全力大概也只能升到下层的L7。有大学文凭就有了敲门砖，能够成为白领，选择更多的职业、企业和工作岗位。只要积累一定的工作经验，混一些资历，进入 L7 层是很容易的。如果是重点大学或者热门专业，或者运气好一些，进入了
一个核心行业、一个快速发展的行业，那也有可能跨一个
大阶层，踏入社会中层的 L6 层。如果是顶级的大学毕业，一毕业就可以进入 L6 层也是很
常见的事情。学历，某种程度上包含了了一个打工人的智力、知识专业、
社交人脉等等实力。

不过，随着阶层的上升，文凭的效用和价值是边际递减的。一个打工人想要登上更高的阶层，还需要做好选择。到了中层的L6 之后，学历就不再是能提供额外收益了。打工人路径的阶层攀登上限，也大多就止步于中层的L6层，只有极少极少的打工人可能进入L5 层。原因很简单，中层的核心是经济资源，而打工人提供(出
卖)的是人力资源，无非是自己的体力、知识、管理能力
等。这些东西在下层很被看重，但还无法和经济资源的价值相
比，能力估值能到百万级别的人已是凤毛麟角。所以，大部分打工人到了一定时间比如35岁，就碰到了
天花板，要么开始躺平，要么就选择转职了。

总结一下，打工人路径，核心是出卖体力、智力、管理能
力等，上限是中层的 L6, 需要的核心“道具”是文凭，
属于起效快、上限低的路径。

第二条阶层攀登路径：创业者路径。创业者路径能到达的阶层上限是上层的 L3 层。和打工人路径相比，创业的风险和收益都是几何倍增的。

创业者路径，有两个阶段，第一阶段是个体户或者自由职业者，第二阶段是企业老板。很多企业老板，都是从个体户或者自由职业者开始起跑的。

创业者路径最大的特点是， 上限很高，下限很低，因此，
创业成功能够带来几个阶层的飞跃，而创业失败也可能造
成几个阶层的跌落。创业者路径没有“道具”门槛，不像打工人路径需要学历
文凭作为支撑，谁都可以来创业，所以门槛很低。但是，创业者路径对个人的韧性、认知、资源等等的要求
又很高，所以它的隐形门槛很高。创业者路径，就是一个向中上阶层发起冲击的路径。中上层的核心是什么?是资源。所以要想升阶成功，就必
须要拥有自己的核心资源，体现在创业过程上，就是要有
自己的核心生产资料，比如资本、技术、品牌、流量、专
利等等。如果你想要创业， 一定要找到自己的优势资源。

创业者路径，可以很简单，也可以很复杂。但好处是，你
可以从初级开始，由浅入深，渐入佳境。创业的过程，就是获取核心生产资料的过程，就是一个造
产品、营销客户、获取利润的过程。说得复杂点，打造产品需要资金、技术、专利等，营销客
户需要品牌、推广、流量，获取利润需要一个高效的盈利
模 式 。但复杂的东西如果看到本质，往往都很简洁。很多人无法
开始创业，或者创业失败，就是看不到本质。把创业的过程精简，其实最后就是一个东西，盈利点。只要能抓住盈利点，不断地优化、复制、扩大盈利的过程，
最后就能得到一个优秀的项目。而这个过程，就是从创业的初级阶段向高级阶段转变的过
程。

第三条阶层攀登路径：投资人路径。投资人路径能达到的阶层上限，是没有上限。在欧美一些国
家，政府的背后都是大财团，资本是社会的最高掌权者。
当然国内略有不同，这里就不多说了。

投资人路径的起点高，要求高，上限也高。

起点高，因为投资人路径的核心资源是资本，没有资本也
就无从谈起投资。至少在中层的 L5 层，才有可能真正开
始投资人路径。做投资的人很多，比如证券公司、投行里，有一大堆的投
资顾问、投资经理，但这些人大部分都是打工人，不是真
正的投资人，他们的主要作用是给投资人的投资做调研、做中介、做决策参考。真正的投资人，首先你的有“资”可投。拿一点工资收入
来炒股，这不叫投资，这叫投机，顶多算是理财。更直白
的说法，这叫为社会经济发展做奉献。

其次，你要能承担风险。小散户没有风险承受能力， 一点
波动就心惊肉跳，亏一次就重伤，这样的很难称之为投资
人。

投资人路径也有两种，一种是做经济投资，也就是在各类
资本市场活跃度；另一种是政治投资，这个就很少人接触
到 了 。

举个简单的例子，某大国选举之前，各个财团都会就行政
治押注，决定支持哪个人选，就要对其就行投资。压中了
回报巨大，没压中可能就要受到打压和清算。政治投资古已有之，不是新鲜玩意儿，只不过普通人很少
接触 。

古代的世家门阀，遇到皇家争储的时候，也会进行政治投
资，但又不同的投资策略，有的孤注一掷，赢了平步青云，
输了满门抄斩；有的则两边押宝，胜在稳妥。

投资人路径并不好走， 一是条件苛刻，要求高，不仅要有
资本，还要有强大的社会认知；二是风险巨大，必须擅于
抓住时机，能进能退。但是投资的逻辑，是值得普通人借
鉴。

第四条阶层攀登路径：依附者路径。依附者路径，就是你通过依附于某个高阶层的人，或拥有权
力、金钱等稀缺资源的人，和他成为利益伙伴或者情感伙伴，
然后能够分享他所拥有的资源或权力。

情感层面的依附，一般是通过婚姻或者其他情感方式进行依
附。比如灰姑娘嫁给王子，平民女嫁入豪门，这些走的都
是依附者路径，从底层直接跃迁到顶层。

另外，官二代、富二代这些阶层，本质上都是依附者，他
们自身并没有这个阶层的实力，但是通过情感依附，获得
了高于自身实力的阶层地位。

利益上的依附，包括一些认干爹、干妈的行为，拜师或者加
入某个门、某个帮派等等，同样也是如此。

依附者路径的核心有两个， 一是自身的特长，比如能提供
外貌价值、情绪价值或者技能价值；二是作为依附者的心理臣服和心理认同。

总之，这个路径是一个相对的捷径，但同时也需要一定的
条件和自我牺牲，以及一些能力和运气。

以上四条路径，就是当前社会普通人能够攀登的路径了，
当然还有一些其他的路径，比如政治路径，这个就更难更
复杂了，也不太方便讲；还有一些违法犯罪的路径，这个
大家 一 定不要去走， 一定要保证自己的合法性。
