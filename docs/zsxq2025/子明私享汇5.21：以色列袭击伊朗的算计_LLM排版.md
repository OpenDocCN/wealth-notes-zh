
# 以色列袭击伊朗的算计
## 发布时间：2025年5月21日

近期以色列号称准备袭击伊朗核设施，跟几个事件会比较类似，一个是1999中国驻南斯拉夫大使馆遇袭，一个是2019年朝鲜驻西班牙大使馆遇袭。前者是克林顿跟中国签WTO协议前，后者是特朗普跟朝鲜签停战协议前，都是美国的既得利益集团不满总统的决定，但又无力阻止总统的决定，于是安排打手制造对手无法下台的事件，阻挠双方已经秘密达成的协议。

这种事情，幕后的力量集团必然非常强大，譬如2018年阿根廷G20中美元首会晤前，安排国安顾问博尔顿要求加拿大抓捕华为孟晚舟，加拿大方面根本无力拒绝这种要求。

这次以色列宣称要袭击伊朗的核设施，也是美国内部不希望美伊草签协议落地群体的努力。只是把打手的工作交给了以色列的军部，而以色列军部也清楚这任务搞大了容易引发全面战争。索性把事儿先挑开，争取伊朗内部反对协议力量的鹰派支持，给他们提供阻挠协议的筹码。

类似于3年前高喊“双方鹰派通过shoot down，一场低成本的‘冲突’，来携手终止鸽派协议的签署。”
