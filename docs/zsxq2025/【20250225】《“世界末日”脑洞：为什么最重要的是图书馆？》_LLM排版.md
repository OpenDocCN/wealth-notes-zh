
# “世界末日”脑洞：为什么最重要的是图书馆？

最近，关于2024年小行星的消**息很多。据说有科学家预测，这颗小行星有可能会在2032年12月，从距离地球大约12.8万公里的地方飞掠而过。

这颗小行星有没有可能引发灾难？前段时间，有媒体采访了天文普及专家曹林老师，其中提到了三个关键信息。

第一，撞击的概率，大约是3.1%。这颗小行星最早被发现在2024年12月，由夏威夷大学开发的小行星撞击早期预警系统捕捉到。随着科研人员预测，它有1.2%的概率，在2032年12月22日撞击地球。而到了2025年2月，NASA做了最新估算，这个概率又提高到了3.1%。随着时间临近，这个数据还会进一步调整。这个测算过程就像天气预报一样，时间越近，预测越准。

第二，这颗小行星的体积，大约相当于一个足球场。它的直径在40米至100米之间。假如它冲进大气层，那么高温冲击会消耗掉一部分体积，又或者直接让这颗小行星解体成一堆小块，并且在大气层燃烧殆尽。即使接触地球，也只是接触了大气层，碰不到别的地方。

第三，人类目前的科技水平，已经可以抵御相当一部分撞击风险。比如，各个国家都建设了小行星监测系统，可以探测危险小行星，并且对撞击时间、落点，以及危害程度做出预报。同时，我们还可以主动对小行星发起撞击，改变它飞行的轨迹。

综合以上因素，小行星造成灾难的概率，已经极大程度降低。

其实，人们一直都在观察那些有撞击概率的小行星。像贝努小行星，预测将在2182年接近地球，撞击概率是0.037%。这个概率不算高，但这颗小行星的直径达到490米。韩国釜山大学还做过模拟实验。假如贝努小行星撞击地球，最坏的结果是同时触发冰河期、臭氧空洞和生物大灭绝。但这 是160年以后的事，其间人类的科技还会获得巨大的进步，帮助我们抵御风险。

好，咱们先把这回的小行星事件放在一边。顺着这个新闻，我们可以开个脑洞。假如，只是假如啊，万一有个跟地球一模一样的星球，有一天发生了大规模的灾难，夺走了大量的生命。而 你特别幸运，活了下来。这时，你背负一个艰巨的任务，重建整个人类文明。那么请问，你该怎么做？

其实，这些年好莱坞正在出现一个新的类型片，我们可以管它叫，后灾难电影，专门讲灾难之后的故事。主要是因为这些年灾难电影拍烂了。光是自由女神像在电影里就不知道被毁了多少次，被水淹、被火烧、被外星人打碎，等等。

导演们眼看灾难片没得拍了，怎么办？大家一琢磨，干脆拍灾难之后的故事吧。于是，就有了后灾难电影。题材包括但不限于，地球被外星人占领之后怎么办？地球被沙漠吞噬之后怎么办？全人类失去生育能力怎么办？

但是，电影毕竟 是电影，为了戏剧效果难免牺牲一些现实层面上的严肃性。

因此今天，咱们换个相对严肃的视角。伦敦大学学院有位学者，叫路易斯·达特内尔，他同时也是位科普作家。路易斯写过一本书，名叫《世界重启》。书里主要研究的就是，地球在经历重大灾难之后，如何快速再造人类文明。

注意，尽管这个话题怎么听都像在开脑洞，但路易斯的写法相当严肃。这本书几乎没什么故事性。路易斯的出发点是，人类要想重建文明，最重要的是知识。尤其是农业、医药、能源、交通、通信等方面的知识，因此这本书有70%以上的篇幅就是在教这些事，包括怎么选择适合种植的土壤？怎么使用粪肥？怎么给孕妇接生？等等。

路易斯的大致设想是，把所有重建人类文明需要的知识线索，都写进这本书。注意，只是知识线索。你要想进一步了解，可以顺着这些线索去图书馆查找。没错，他认为重建人类文明最重要的东西就是这本书，外加一座图书馆。

而今天我们没法把这些专业知识展开，只介绍路易斯的大致设想。好，咱们正式开始。

首先，既然有人类能幸存下来，说明灾难并没有把一切都毁掉。地球上尚且存在一些生存资源。

因此，你要做的第一件事就是，找超市。按照路易斯的估算，假如一个人找到了一间中等规模的超市，那么里面的物资大概够他生活55年。假如不挑食，把猫粮狗粮全都算上，那么这个年限能延长到63年。对于一个30岁的成年人来说，一间超市能养活他到93岁。路易斯把这段 时间叫做灾难之后的宽限期。就像疾病一样，即使再严重的病也不会让一个人马上死亡，总会给他留一部分求生的 时间。

当然，作为一个高尚有追求的人，你不可能独自霸占这些资源，更不可能窝在超市里不出来。你会收留更多需要帮助的人，因此超市里的物资很快就会消耗掉。你要趁物资被消耗掉之前，赶紧行动下一个落脚点。

找哪呢？路易斯说，最明智的选择是逃到城市周围的农村。因为农村更接近自然资源。你看，城市里的水、电、燃气，本质上都是一套严密的工业系统，需要大量的人工协作。一旦发生灾难，这些协作系统会马上停摆。首先是水资源被污染。其次是供电系统崩塌，失控的电压会引发火灾。火灾又会威胁到燃气系统，引发大大小小的爆炸。

因此，路易斯认为，前往农村是相对最安全的 选择。至少这里更接近自然资源，有柴火可以烧火，或者有河流可以取水。

但路易斯特别强调，不能前往偏远的农村，必须是离城市近的农村。为什么？因为你的目标是重建人类文明，而要想完成这个任务，城市里有你必须依靠的东西，你不能离它太远。这个东西就是，图书馆。

即使没有电力，电脑打不开，但书依然存在。要知道，绝大多数现代人的大脑里，只有高度专业化的知识，很少有人熟悉那些文明最底层的、跨领域能技术。

比如，医生会开药，但未必知道生产药品的工业流程。程序员会写代码，但几乎没有什么程序员可以独立掌握制造芯片的技术。钢铁铸造厂里的经验丰富的工人，未必掌握了冶炼金属的所有步骤。

说白了，没有任何个人掌握所有维持社会运转的关键知识，更别提实际应用这些知识。因此，要想找到这些知识，你必须依靠图书馆。

而假如你做个统计，就会发现，在多数灾难电影的设定里，最珍贵的资源，一是饮用水，二就是书。比如，《疯狂的麦克斯》里，人类分化成了无数个小型暴力集团。但即使最恶劣的暴力集团，里面都得设置一个角色，叫历史人。历史人的工作就是传播知识。再比如，阿西莫夫的《基地三部曲》，里面有个情节是，先知预测到文明即将毁灭。而要想快速恢复文明，最重要的 事情就是写一本书，记录下所有重要的信息。

好，书归正传，回到你的主线任务。即使你已经找到了图书馆，但光有专业的书还不够，还得有专业的人。因此，你接下来的重点是找到更多幸存者，然后跟他们合作。

同时，假如你自己就是个有专业知识的人，比如医生、护士、机械师，那么路易斯建议，在你遇到能够庇护你的群体之前，最好不要轻易展露你的技能。因为一旦被人发现，你可能就会被暴力集团胁迫，成为他们的人。

那么，作为召集者，找到多少人才算够呢？路易斯的测算是，最低140人，他们最好是由70对夫妇构成。路易斯说，从生物学的角度考虑，70对夫妇是维持人类可以正常繁衍的最小数字。注意，只是最小数字，哪怕是按照人类历史上出现过的最快人口增速计算，70对夫妇也得经过800年的时间，才能达到工业革命前世界人口的数量。

因此，路易斯认为，最好找到一万人。有了这个人
口基础，人类就可以早点利用现有条件恢复农业生产，逐渐进入稳定状态。

那么，重建后的人类文明是什么样的？会和现在的一样吗？按照路易斯的设想，肯定不会。你大概会看到这样的景象，很多人坐在马上，但用手机和笔记本电脑。因为这个时候新能源大概率是用不上的，人类只能烧木炭，然后用蒸汽机给手机和电脑充电。而由于缺乏燃料，汽车只能是探险队的专用交通工具，普通人只能骑自行车或者坐马车。

但好消息是，这个重建文明的过程一定是加速展开的。因为跟以前的人相比，幸存者最大的优势在于他们了解历史，可以绕开历史的偶然性。比如，青霉素的发现，来自弗莱明的一次偶然观察。电磁知识的跨越，来自法拉第的偶然发现。但现代人已经具备这些知识，可以把这些科学进展一步设计到位。这就在很大程度上提高了文明的重建速度。

最后，路易斯也提醒，在你做这一切的时候都别忘了，依然要把所有的过程和方法记录下来。万一星球再出现危机呢？毕竟，路易斯始终认 为，重建文明最重要的东西，永远都只有知识。

今天咱们算是开了个脑洞。假如你有兴趣，可以去读一读路易斯的这本书《世界重启》，得到就有电子书，以及这本书的听书解读版。看一个人用极其严肃的方式，去思考一个被很多人看作调侃的问题，也是一种非常有趣的体验。
