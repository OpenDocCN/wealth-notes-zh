
# 学习笔记，2025.7.31

1. 男欢女爱的本质是：你的基因认为它终于找到了“有利于自己繁衍”的另一个基因。于是它命令大脑分泌大量的化学物质对你进行奖赏，使你感到巨大的愉悦感。这就是为什么你在坠入爱河的时候会有明显的幸福感。说白了，其实就是基因在催你生娃了。
2. 做真实的自己，会给你带来很多附加的好处，有人会因为你的真实而喜欢你，成为你的铁杆粉丝；当然，也有人会因为你的真实而讨厌你，然后就会主动消失在你的生活里。这笔账怎么算都划算。而且你越真实，筛掉的人就越多，但留下的，一个顶十个。
3. 有毒的人际关系和糟糕的环境，就像慢性毒药一样，会无声无息地侵蚀一个人的灵气与生机，让人产生各种生理和心理方面的损耗：抑郁、呆滞、情绪失控、判断力下降。因此，请选择那些能让你情绪舒展的人际关系，选择那些能滋养你而不是消耗你的环境。
4. 永远不要低估痴迷的力量，你爱过的东西，最终都会回到你身边。某件你曾经尝试过的奇怪小事，某个放弃的爱好或者荒废的乐器，它们都会在未来的某一刻回到你身边，在你需要的时候为你服务。那些你为之投入的事物，不管多么渺小、短暂，都不会被浪费。
5. 凝视是来自上位者的软暴力，因为它不仅仅是观察，还包含了评价甚至攻击。如果你总感觉自己被凝视、被评判，在别人面前无法正常思考和做自己……你要学会凝视别人。你是判官，也是面试官，而不是被观察的角色。你可以直视任何人，全世界都正接受你的审判和面试。
6. 人生最好的状态不是一帆风顺，而是相信自己“永远有解决问题的能力”。如果你真的无所畏惧，永远相信自己，那么当你有一天早上醒来，或者独自走在路上，突然间，一件奇怪的事发生了，未经邀约、未经请求、未经追寻，那个可以被称为爱、真理或幸福的东西，忽然就降临了。
7. 最近发现低精力人的第一原则应该是，别掺和。本来精力电池就少，还总要被别人影响，别人说什么、做什么都能打扰你，最后被卷进本来和自己没关系的麻烦。低精力的人更应该守住自己的注意力，因为你的精力真的太稀少太珍贵了，只够你专注地走在自己的主线上。
8. 更重要的是重拾自信。女孩子选择开始健身，不久之后你就会发现，她一定会变美。因为脂肪更少了，全身的皮肤紧实了。运动后，皮肤也更佳有光泽有弹性了，人会变得乐观开朗了，变得比之前更加自信了。想要瘦下来，首先要改变的就是生活态度和方式。变瘦只是第一步，更重要的是重拾自信，以及拥有身体和心理上的健康。
9. 假如你想对这个人好，那当你做出“对他好”的行为后，这件事就结束了。别人有没有反过来对你报恩，你都不要去管了。同样的，假如你确定自己没什么问题，那当你做出拒绝的表达后，这件事就结束了。别人有没有因为你的拒绝就对你心生怨气，你也不要去管了。因为对方的反应其实和你没什么关系，那是他自己的事自己的课题，不是你要负责的——做到这点后我发现人际关系很多问题都迎刃而解了，变得彼此都轻松自在，还能顺便筛掉一些消耗人的关系。
10. 只听有结果人的话。除非那位长辈有很大的成就，不然不要理会他们的意见。毕竟他们混的怎么样，你已经亲眼目睹。
