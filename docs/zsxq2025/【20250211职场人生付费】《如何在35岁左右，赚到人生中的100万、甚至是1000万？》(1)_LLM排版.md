
# 如何在35岁左右，赚到人生中的100万、甚至是1000万？

前言

赚钱的实操方法

今天这里所说的100万、1000万是不包括房子、车子等实物资产，就是纯现金存款。

对普通人来说一年赚1个小目标是遥远一些，设定一年赚100万，还是可以实现的，拥有一百万后，后面的钱会越来越越来越好赚。

赚第一个100万和后来赚1000万，哪个更难？

“赚第一个100万更难。”我很明确地告诉你。

赚100万是冷启动，特别难，赚1000万是热运行基础上有了加速度，相对容易的多。

所以，对于20多岁的年轻人来说，我的建议是，一定要早日赚到第一桶金。

一年赚10W，在体制内一半以上的人都能达到，在体制外当一个小领导，努力搞定你的上司，或者做个体户开个小店、又或者做一名骑手，一天跑14个小时也是可以实现的，也就是说一年赚10W元，只要体力上使劲，就够得上，一年赚10万也不是很难。

但是一年赚10W的和赚100W，是两种完全不同的思维。

所以，如果你还年轻，30岁左右，看我这篇文章极其重要，你还有很大的可能性“翻身改命”。

你除了读这篇文章，还要打印出来，要时不时阅读并且实践，本文全是我自己亲身的经历，全是些实操方法，你照着去做，赚到100万真的不难。

一、拆解目标，建立赚100万的目标

年收入100W是一个大目标，将它拆解成很多小目标，你实现的每一个小目标，积累起来就成了大目标，看似遥不可及，实则触手可及。先赚10块，再赚100块，然后赚10个、100个100块，慢慢就赚到了大钱。

比如：你是做种植人参生意的，每卖出去一盒人参，利润就是100元。一年赚100W，那么，你一年之内必须卖出去10000件，一个月你要卖840盒，每天28这这时候你的目标从年收入100W就变成了每天卖28盒的人参。如果你是实体店的生意，一天28盒有点难，因为现在线下的生意受到互联网的冲击，流量分散。但只要你会手机，你就能利用互联网的平台实现你的目标。

对于没有经验，没有流量的小白而言，一天卖28盒也不容易。继续细化目标。

一天成交28单，你需要让多少人看你的产品？这时候你就得从同行那里预估一个转化率，假如你从同行那里预估转化率是10%，也就是100个潜在客户就会有10个下单成交，那你就要引流280个潜在客户。此时的目标由一天卖28盒人参变成每天引流280个潜在客户，能够一天280个潜在客户关注到你 的产品，要么你 的人参宣传的内容足够吸引人，要么你的内容曝光多，下边就是从线下或者线上搞流量。

再比如说：你做自媒体，100万不是一笔赚100万，而是每天2700元。每天赚2700元不是一单赚来 的，比如一单商品你售卖270元，你需要每天销售10单。每天10单，如果是5%的转化率，对应200个人浏览你的产品带来的。

所以到这里，你拥有了非常明确具体的流量目标。

具体的执行方案：

如果一篇文章，平均可以获得100次的曝光。那么，每天需要发布2篇的高质量内容，才能实现你的目标。“每天发布2篇高质量内容”就是你真正要完成的目标，你的最小行动单位。

你拥有了具体可行的动作，而不是模糊的让人不知所措的“我要赚到100万”。所以：把模糊的目标，变成具体的、清晰的、明确的日常行为。你才能实现赚到100万这个数字目标。

再举个例子举例，你觉得一年赚100万这个目标有点大，你说要在3年内完成。那么我每年需要赚的是30万左右。3个项目，每个项目月入一万。

比如自媒体这个项目：每个账号平均每月能带来3000的收益，我就需要同时运营3个账号。如果每个账号能带来5000的收益，我需要运营2个。

主业工作，能带来1.8万收入，那么我只需要再发展一个月入7000的项目，就可以实现我的目标。于是我的计划细化到日常：

1、每天上班；2、每天写2—3篇高质量文章；3、每天发布1—2篇文章

最后一步：让系统运转起来。

当你的系统运转起来时，结果就会自然到来。你需要的是按照最小行动单位，坚持去执行它。然后等着收获。

所以，你真的不必焦虑：做什么才能赚到100万？

你要专注于“今天的2—3篇内容写完了吗？”“今天上班了吗？”让你的系统运转起来，结果自然会达成。

最好，找到你的配偶一起去执行系统，重复完成，100万，就是精确的日复一日。

这时候，年入100W的目标就非常清晰了。

做生意从0→1比1→100都要艰难，考验你的不止耐心，还有信心。前期你很努力，可能每天也是十几个，甚至是个潜在客户，没有达到拆解的目标。所以，绝大多数人坚持30天可以，坚持3个月可以，但是坚持6个月，1年，3年，他自己先放弃了。

二、主副业搭配模式，既有主业的稳定，又有副业的灵活

看 我文章的大多数是体制内的，你有一份稳定且待遇不差的 工作，又或者进入互联网大公司工作，能拿到比较好的薪资，和基本的福利待遇，保证下限，这都是你的优势。

一个提供现金流的主业工作及一两项副业，你需要持续的积累虚拟的资产如知识和经验。

如果你一项技能你只掌握到百分之四五十，这项技能就是很平庸的，就是大众化的，每个人可能都会，不可能给你带来任何经济上的回报，顶多就自娱自乐，丰富一下业余生活。如果你能够把一项技能打磨到超过90%的人，这就成为了你的优势和特长，就能获得经济上的回报。

当然也不是你的技能到90%的时候去做，50%的时候就可以去了，继续深耕，提到60%、70%、80%、90%。

总之，赚钱要从自身去定位，拿自己最擅长的东西去开拓赚钱的路子，这样成功的概率会高很多，你也更容易坚持下去。

我观察过，赚到大钱的人，很少只是靠天赋成功的。

多数是先埋头苦干，有点小成绩，形成正反馈；然后天赋才能发挥作用。最后，不断扩大战果，成为行业专家。

其实就是：认真且持之以恒的反复学习实践。哪怕只用1年的时间，在一个行业反复训练自己的技能，不断用实践复盘，每天把做的事都记录下来，并形成计划，复盘总结。

如果什么都做，很容易一事无成。最好聚焦于一个东西，让自己成为独一无二的专业家。

所有真正厉害的人，都是只做自己喜欢的一件事，短期内锁定一个确定的方向，用长期主义信念去深耕细作、持持久战。

赚钱从来不是一蹴而就的事，而是一个长期、甚至是一辈子的事。如果这时无法晋升，那么也可以转型，把副业变主业。当然在40岁以前，主业都是你最大的事业。

只要能在这个行业深耕一年，都会有很多人把你当成专家请教，你领悟的速度就会在实践中越来越快，然后突然就会在量变中积累到质变，迅速全方位开悟。你的认知碾压别人、你对别人有价值、别人对你有所求，你很难赚不到钱。

三、想赚大钱的话，需要有自己的生产资料

为什么一个熬夜12年戴着400度眼镜的清华学霸，专业成绩排名第一，却赚不过小学毕业出身、卖饲料的土老板？其中的奥秘就在于谁拥有生产资料。

为什么靠专业技能赚钱很慢？很难赚大钱？因为当你每天走进办公室，按照时间价值计算工资时，你离财富就已经很远了。

没有自己的生产资料，不论是体制内还是体制外，都是在出卖自己的劳动时间来获取报酬。

在体制外老板组织起一群打工者，购买他们的时间，来打造自己的生产资料。可能偶尔会有一两 个拿着高薪的打工皇帝，他们的经济状况可能比普通打工者好一些，但其实也好不到哪里去。

很多打工皇帝之所以能够赚到大钱，很大程度上是因为拥有公司股份，可以获得分红，这时候相当于占有一部分生产资料。但普通人家庭几乎没有接受过关于“生产资料教育”，学校也不会教你怎么赚钱，学校只关心考试成绩，就算是老师也没有积累生产资料的意识。

潮汕人的“宁愿卖葱，也不打工”，就是典型的“生产资料教育”，即生产资料的重要性优先于生产力工具的重要性，产权价值大于劳动力价值。

实际上，富人家 庭和有深厚经商传统的家庭都非常重视生产资料教育，而大多 数普通人家 庭教育则忽略了这一点。

没有哪本书或者哪个专 业会告诉你，掌握资源远比学历重要，掌握生产资料的能力是赚钱最重要 的能力，而不是具备某个专业技能。

这才是赚钱的最大奥秘。

拥有生产资料的本质，是拥有一个东西的产权，而这个东西是可以持续不断地给你产生收益的。

真正的富人，一定是“生产资料”的拥有者，他并不一定有多大的身价，但是每个月一定会有源源不断的现金流，而且不需要自己付出太多的时间和体力。当你拥有生产资料，就相当于拥有了被动收入，拥有完全不一样的生产关系，而不仅仅 是依赖劳动力和劳动时间来获取报酬。

而我个人认为，一个人真正的开悟，其实就在 于产权意识的觉醒，在于生产资料意识的觉醒。没有生产资料，没有商业思维，不管多么辛苦，工作时间多长，赚的钱永远都是辛苦钱，永远都发不了大财。数字时代、互联网时代的生产资料

随着社交媒体的兴起和智能手机的普及，数字时代的生产资料正在发生天翻地覆的变化。这是一个时代的礼物，一个人也可以活成一家赚钱公司。

现在无论你是谁，在互联网和短视频平台的赋能下，你都可以摆脱组织的束缚，自己创业，自己赚钱。在这种情况下，我们应该顺应趋势，让自己成为生产者，积累自己的生产资料。

当你真正有意识地去积累生产资料的时候，你就会很有目标感和专注地做事，心无旁骛，万事可破。

打造自己的资料，就会拥有源源不断的被动收入

批发时间而不是售卖单个时间

按照马克思主义政治经济学的观点：价值指的是凝结在商品中的无差别的人类劳动。

看不懂没事，也不用懂。说白了，就一句话：价值是用时间来衡量的。

而每个人的时间都是一样的，都是只有24个小时的，之所以有人赚的少，有人赚的多，取决于以下二个因素，而你想赚更多的钱，也就只能从这二个因素上着手开拓。

单位时间能够创造的价值

这个很好理解，同样是打工，同样是工作八个小时，有人赚得多，有人赚的就少；同样是出去讲课，有的老师一天的课时费是2000，有的老师一小时就能是1万。

所以如果你想多赚钱，要干的第件事情就是想办法的提高自己的单位时间价值。我举个例子，一个大学 生，上学期间为了赚一些钱，每天出去麦当劳、肯德基、必胜客打小时工，假设一小时可以赚20块钱。一个月也许能赚个2000块钱，四年下来赚个听起来“很惊 人”的数字—10万。

但为了赚 到这10万块钱，这个学生的业余时间几乎就没 有了，甚至可能还要占用上课时间。没有时间学习，成绩自然不会好，严重的可能还会挂科，甚至无法毕业。

而另外一个学生，在学 期间刻苦读书，除了以优异 的成绩 毕业外，还考过了英语六级和 其他一些含金量很高的证书。当然，没有赚到10万块钱。

但是毕业 后，成绩优异 的学生拿到了30万的年薪；在麦当劳打工的，依然在麦当劳打工。所以，当你依然执着于（或者不舍于）一小 时几十块钱的“稳定收入”时（比如跑滴滴、送外卖 等），不妨想想这个例子。
