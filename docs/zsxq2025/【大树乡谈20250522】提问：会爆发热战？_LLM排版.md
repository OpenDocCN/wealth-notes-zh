
# 提问：会爆发热战？

## 镇长回答：
先说结论：99%以上不会。

## 这种担心可以理解，但忽略了三个关键：
一是认知更新。目前全球，像兔子一样为战争认真准备的国家并不多，大国中也只有兔子和俄罗斯，西方普遍马放南山、军备废弛，不能继续用二三十年前西方军力的巅峰去看待当前。
二是要注意本土守备和出境进攻是两个概念， 一个国家能够外出作战的军队占比并不高。
三是战争的目的是什么?美国有没有必要同兔子发生热战。在小镇看来，无论中美竞争谁胜谁败，都

不影响败者继续作为凌驾于其他国家之上的第二国，中美不可能两强相争，令其他国家渔翁得利。事实上,中美无论贸易战、关税战还是科技战，最终的结果都是其他国家被削弱，而中美更强，这并非刻意为之,而是国家优势决定的。

关于中美会不会爆发热战，最近看到有些讨论，担心美国会不会在关税战、贸易战、科技战失利的情
况下，动用包括热战等在内的其他招数。

4月关税战升级，在中美接近贸易禁运的情况下，
兔子经济数据反而非常不错。比如上周5月19日介绍
2025年4月国民经济运行情况，各项数据都非常喜人，
呈现了极强的韧性，以事实证明，就算在美国倒行逆
施的极限施压下，兔子经济也完全扛得住，甚至更好

进出口方面的数据小镇之前在公号文章已经提到
了，4月货物进出口同比增长5.6%,出口增长9.3%、进
口增长0.8%,注意这是在对美出口下降超过20%的情况
下取得的，无论是直接对他国出口还是转口贸易，都
以事实证明了，消费国发动的关税战必然失败，这是
人类历史上实践证明的真理。

而且也正如小镇之前预料，最受中美贸易战影响
的就是服装鞋帽家具家电类生活消费品，这类产品出
路在于扩展全球其他市场需求，4月也出现了可喜的变
化，这类生活消费品的出口量并未出现明显下滑，虽
然有部分企业遭遇困难，但整体上通过开拓其他国际
市场，受冲击程度可控。出口的去向就是” 一带一路”沿线国家，目前
“一带一路”沿线国家外贸占比已经超过一半，4月对东
盟的出口也大增超过20%,基本填补了对美出口的下降
缺口。还要注意到，不仅是“一带一路”沿线国家
包括对日、对欧等国家，出口也在快速增长，全球化
是拦不住的，更何况美国本身也不是铁板一块。

而在外贸之外，工业、服务业增长都比较理想，
同比都超过6%;固定投资增长也 开始恢复。特别值得
关注的是4月社零销售总额同比增长5.1%,环比增长
0.24%,注意这是在CPI 同比下降0 .1%的背景下取得的，
意味着实物消费增长。

关于社零消费，小镇之前在公号也谈了。从实物
消费量角度，兔子早已是全球第一大消费国，体量远
超美国，可能接近整个西方之和，而且兔子自身消费
也在持续升级。一个特别值得关注的指标就是“粮油食品类”限
额以上单位商品零售额，这个指标常年保持两位数的
增长。小镇之所以关注“食品粮油”这个指标，与兔
子人的消费习惯有关，毕竟兔子人对吃情有独钟，吃
不仅是为了生存更是一种享受，而限额以上单位的食
品粮油指标还体现了产品升级的趋势，这代表民众对
食品品质要求提升、规模化生产等趋势。

2015年之前，食品粮油和饮料烟酒合并统计，之
后分开了，从历史数据看，食品粮油的走势相比社零
有一定的滞后性。2020、2022年社零负增长的时候，
限额以上食品粮油仍然保持9.9%、8.7%的同比增长，
2023年增速反而低于社零，到2024年基本恢复到过去
十几年一直维持的10%左右同比增速。

这个数据在今年出现增速持续上升，1-2月、3月
、4月同比增速分别为11 .5%、13.8%和14%,虽然这只
是一个局部指标，但考虑到目前价格下降，也反映了
兔子内部消费稳中有升。

还是小镇一直说的，兔子强调扩大内需，不是因
为自身消费弱，而是因为外贸萎缩，不得不短期靠内
部消化一部分。

就业方面，也出现了向好的变化，4月城镇调查失
业率为5.1%,环比下降0.1个百分点。小镇昨天 在公号
《面对即将到来的爆发式发展，必须做好抗冲击准备
》,介绍自然失业率 目前大概在5 .1%,这就意味着调
查失业率基本接近充分就业的均衡状态，当前失业压
力更多来自科技和产业升级带来的结构性摩擦。不过
工作时间还是偏长，4月企业就业人员 周平均工作时间
为48.3小时。

不管怎么说，中美4月关税战已经证明了，关税战
没什么可怕的。至于科技战，早就破产了，相比2018
年，中美科技和产业之间的差距大为缩小。

以上是背景，在美国关税战、贸易战、科技战受
挫的情况下，未来会有声音更关注战争的风险。
这种担心可以理解，但完全没必要。

这次巴基斯坦空战已经证明了兔子现代化军改路
线的正确，兔子以实战也就是同等级别的大国交战作
为军改目标，而非把军力浪费在欺压弱小的治安战上
,再加上兔子强大的制造能力和基础教育，极大增强了
兔子国力、军力。可以非常直白地说，就算全世界
所有国家联手，也不足在兔子周边对兔子形成足够的
军事威胁。

各国表面的军事数据做不得真，兔子并不是武力
扩张型国家，要跟兔子大战，肯定在兔子周边进行。
那么各国本土军事力量和能够派出到国境外的进攻性
力量，这是两个概念。

比如现在的欧洲，去年北约峰会，欧洲国家为了
换取美国延缓撤出欧洲，曾提出派出军舰到兔子周边
,帮助美国对华施压。但这不过是说说而已，欧洲各
国连支援近在咫尺的乌克兰都很难做到，西欧的英法
德，基本是欧洲军力的最洼地，三国竭尽全力能够派
出干涉乌克兰的军队都不会超过2万

比如德国要在立陶宛部署一个5000人的装甲旅，
说了很多年了，但现在也不过只有几百名文职人员。
今年宣布正式启动，但也只敢预计到2026年夏季扩增
至2000人，估计基本是后勤文职人员，作战人员占比
不高。这也是英法德军队目前的通病，文职人员占比
过高，比如英国，陆军将军数量都比坦克多， 一艘几
干吨的军舰能分到十几位将军、舰长。

欧洲真正有战斗力，并且为大规模战争做准备的
国家，也只有希腊、意大利、西班牙、波兰和土耳其
,但希腊和土耳其互相制约，波兰要盯着俄罗斯， 西
班牙对参与战争毫无兴趣，与意大利一样还要面对来自
非洲的压力。

如果爆发战争，整个欧洲都不用考虑，仅俄罗斯
就足够了，甚至还需要美国派出大量机动力量驻扎，
毕竟压力不仅来自俄罗斯方面，还有以色列的需求。

目前兔子周围的确是人类武力的高地，但还是要
具体分析。
俄罗斯不用担心，反而是整个北部以及半个西部
的盾牌，无论有多少“美俄轴心” “俄罗斯背刺”的
扭曲，5月抗战胜利80周年已经证明了，中俄关系非常
稳固，关系还进一步深入了，已经成为不是盟友的铁
杆盟友。

印度方面， 之前或许还要担心，但最近的情况已
经很清楚了，印度能够应对巴基斯坦就很不错了，而
且印度也不想扩大战争。小镇之前就说印度会自己找
台阶，果然印度在国内搞10天的 胜利纪念活动，明天
结束，还在印度网络上严控一切宣扬印度失败的信息
,这充分说明印度还是很清醒的。

剩下的还有谁呢? 不至于有人会担心东盟吧?
先不说东盟的军事力量如何，兔子与东盟的关系
非常深，怎么可能针对兔子?5月20日，兔子、东盟共同宣布全面完成兔子-东盟自贸区3.0版谈判，还是要感谢特朗普的 疯狂，要不
然也不至于推进得这么快； 再加上 此前宣布激活沉寂
十几年的清迈机制协议，还要 把这个机制坐实，都说
明兔子与东盟之间的地区协作 更加深入，合作深度并
不亚于欧盟、美加墨。

剩下的也就只有日、韩、 朝以及兔子台湾地区
了。之前赖满嘴大话，说要在5月20日搞事，结果还是怂
了，这并不奇怪 ，统一根本不需要动用武力，军队控
制住外围，禁止任何外部力量干预，然后执法力量登
岛执法就行了， 把所有分裂分子全部逮捕宣判就行
了，不会有任何漏网之鱼。

至于日本，20多年前“八八舰队”多么强大，但那
就是日本军力的巅峰了，日本军事建设一个很大的
问题就是什么都想国产化，但日本作为非正常国家，
军队装备需求很低，什么都国产化就意味着生产效率
低、成本高、质量差，其实反而不如搞采购整合，自
己抓住部分核心部件。

更重要的还是本土守备和境外进攻的区别。日本
目前能够用于海外部署的，满打满算也就是两三千人
,其中作战部队还要折半，就算人人是美国队长，又
能怎样?何况日本军队被严重阄割。

至于兔子台湾地区，也早不是20多年前了，现在
征兵这么难，怎么可能维持足够规模的作战力量，空
军海军早就军备废弛，陆军纸面作战部队满打满算也
就几万人，这里面还大量充斥着严重缺乏基本训练的
样子货，虽然老装备依托半永久工事具备一定作战能
力，但也不过是聊胜于无。

目前战斗力最强的就是朝韩，但朝鲜怎么可能发
疯了去帮美国，而只要有朝鲜在，韩国动都不敢动。
虽然韩国军事技术和装备更加先进，但金说得很清楚
“现代战争仍然是炮战”,韩国经济精华的首尔地区
,就在火炮射程范围内，朝鲜能够给俄罗斯支援上百
万枚炮弹，就说明自己手里头还有更多。而朝韩的对
峙，也决定了两国军力都腾不出手。

算来算去，还不如1950年，那时候美国还能集结
大量盟国军队，而到了2025年，美国只能靠自
己。但美国的远洋运输能力大降，在太平洋上都
没有可用于作战的航母了，只剩一艘本意宣布退役的
老破航母勉强维持纸面存在，又怎么可能搞正面对抗
,而随着时间推移，兔子军事优势更大，预计到2027
年左右，仅计算纸面，兔子海空军也将超越美国，而
单体作战力更强于美国。

所以，关于热战。
只要兔子不允许，世界大战就打不起来。因为一
二战都证明了，当有一个强大的制造业国家置身事外
,这个国家将决定战争最后的胜利方，兔子不动，谁
又敢乱动?
但还是如小镇之前分析的，局部冲突很可能加剧
,恐怖袭击等特殊威胁加剧，另外需要警惕美国操纵
舆论等工具控制欧洲部分国家的选情，罗马尼亚就是
典型，不过这就不是安全威胁了，是欧洲人的麻烦。
所以兔子目前更加关注周边安全以及关键枢纽位
置国家的安全与稳定，如果美国继续强化对世界和地
区安全的威胁，这必然会加速其他国家的应对，比如
兔子与东盟，也完全可以参考上合，建立区域安全合
作机制，把目前国与国的“2+2”战略对话机制全面
推 广 。
