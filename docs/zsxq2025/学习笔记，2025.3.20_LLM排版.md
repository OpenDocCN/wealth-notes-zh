
# 学习笔记，2025.3.20

1. 永远不会被琐事消耗其实穷人和富人的自制力差别很小，但穷人需要在处理日常琐碎事物中消耗掉大量的注意力。以至于在处理某个具体事件时，穷人的自制力就会显得比富人更弱。

2. 只跟靠谱的人合作，远离那些不靠谱的，这样可以节省很多时间和精力。不要试图“让他人变得靠谱”，这几乎是不可能的事情。世界上没有免费的午餐，价值连城的信息不会出现在狐朋狗友的免费群。

3. 做一个账号的姿态是什么？就是百分百投入，把它当做自己的孩子一样去呵护；把自己当做农民，这是你唯一的一块土地去守护；把自己当一个艺术家，这是你此生唯一的一个作品。姿态，决定高度。

4. 人需要有一点贪念，贪念到不轻易将自己最重要的需求委身在凑合的对象上。同样的，人还需要一点知足，以至于那个恰好的版本出现后，就不再奢求更好。我们能追求的并非是完美，而是更高水平的守恒，求满必然招损并折伤福缘，炽盛和极限本就是事物走向衰变的起始。一个人能在不凑合和不奢求之间精确取值的能...

5. 我找合作伙伴，第一点看中的是Ta是否善良，是否有格局，是否有执念。能力不是最重要的。看Ta做事情的目的，肯定绝大多数是为了赚钱，但也有极少数的人是因为兴趣，或者责任心或者成就感，那这些人会做的更用心一点。再回到绝大多数，既然是为了钱，那看Ta是想快钱赚一波，还是想持续赚钱。合作伙伴尽量找后者，做事情眼光会长远一些。这是我跟别人合作的一点小体会。

6. 人与人之间最舒服的相处方式，就是我用真心待你，但不控制你，活在缘分中，而非关系里。我们总喜欢拿顺其自然，去面对人生路上的荆棘坎坷，却很少承认，真正的顺其自然，其实是竭尽所能之后的不强求，而非两手一摊的不作为。能够长期维持下去的关系，是靠很少的攻击，很少的对抗，很少的强人所难，给予更多的允许和接纳，爱是如你所是，而非如我所愿。道德经有云：“上善若水，水利万物而不争。”

7. 女人怎么才能变漂亮？一，要变得有钱，要么嫁给有钱人，要么自己成为有钱人。二，坚持健身。有钱人，都耐看。文人习惯了穷酸，你看文人写的富婆，全是丰乳肥臀，实际上呢？丰乳肥臀多是农村娘们，真正的富婆则是40岁看起来像30岁，身材好，皮肤好，颜值好。

8. 有人年纪大了之后，就以记忆里变差为理由，开始拒绝学习。实际上并不是大脑出了问题，很多研究都证明了脑细胞有可塑性，出问题的是人的注意力，每天把注意力放在国际大事上，放在应付工作上，难得一些空余时间，还要放松一下刷短视频。脑子越不用就越显得不好用。

9. 职场官场上永远都是两条运作小路，一条是台面上的程序，规定，会议，另一条是台面下的谋划，商量，迂回，和规避。

10. 以前，我总觉得，青春很长，养老这个话题离我们很遥远，但是我现在越来越深刻的意识到，每个人的人生都是抛物线走势，对于我们这个年龄而言，其实已经过了抛物线顶点了，开始走逆风盘了，要送父母走，要为孩子规划人生，自己又病恹恹的，若是40岁以前没有做好养老计划，真是越走越累。
