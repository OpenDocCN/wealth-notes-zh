
# 洪灝的宏观策略7.16《恒指新高；中国资产突围》

新高意味着什么。今日早盘，恒指创出了年内和三年以来（收盘价比较）的新高。早在今年三月十八日的一篇专属会员文章中，我为会员读者预警了即将到来的全球市场历史性暴跌，然而也预测了中国市场今年下半年的新高。当时，我们观察到美国市场的隐含波动率和依然处于长期均值之下，然而全球贸易和政策的不确定性却在历史性新高。

如是，两个市场波动性指标的背离意味着其中一个指标必须向另一个指标回归。特朗普一天在位，全球政策和贸易的不确定性就会继续高企。因此，那是即将到来的因子回归必然是全球市场波动率的飙升。在接踵而来的“解放日”历史性暴跌中，当时市场隐含波动率 VIX 在那几天的飙升是历史上最剧烈的动荡之一。

我们当时还观察到了全球资金轮动的新的方向：从美元资产开始向非美元资产的轮动。在四月那场历史性暴跌中，美元、美债、美股同时暴跌，美元和美债失去了它们传统的避险资产的地位。这种盘面只有全球资金流向转向才可以解释。如是，当资金从美元资产流出之后，主要的非美元资产，如中国香港、新兴市场、贵金属、加密货币等都应该受益。同时，我们还看到当时的北水南下的净买入也达到了历史新高。今年，北水南下净买入为香港市场贡献了新的流动性，也造就了今年热火朝天的香港打新市场（图一，三月报告原图更新)。

昨夜，美国通胀数据基本符合预期，甚至核心通胀数据略低于预期。虽然贸易相关的商品通胀显示了一定的关税上调的影响，比如家具等，但它们的价格上涨已经在市场的预期里。与此同时，我们看到关税的影响其实比市场的预期还是有所滞后的。同时，服务型通胀继续符合或低于预期。即便如此，市场还是下调了今年下半年美联储降息两次的概率，20 年美债收益率飙升到了 5%以上。我在前述报告中已经讨论了服务型通胀将如何影响美联储货币政策的选择。这里不再赘述。

昨夜出乎市场意料的是，即便美国通胀大致符合预期，美股还是高开低走。让一众因此而看多的卖方分析师大跌眼镜。然而，尽管如此，虽然标普上探了 6300 点后下跌，中概股和中国相关资产的价格却大幅上升，一个看似弱势的资产类别在一个走弱的市场里反而上涨，这种盘面应该引起交易员足够的重视。

昨夜另一则重要的消息，是英伟达重新获得了向中国出口改装芯片的许可。一直以来，老黄对于特朗普政府对于中国的科技禁运的论点是，这样的禁运反而伤害了美国科技公司的利益。因为中国已经在高科技领域获得了长足的突破，比如华为已经自造出了 GPU 组群结构。如果中国公司不能使用美国芯片，那么它们将倒向中国自己制造的芯片，令美国科技公司失去市场份额。

于此同时，特朗普还宣布了与印尼达成了贸易协议，向印尼出口征收 19%关税，而美国向印尼的出口则免税。特朗普还称与印度也将很快地达成类似的协议。如是，这些新的贸易协议都是向美国利益倾斜的。然而，在这些有利于美国的贸易协议里，美国科技禁运的放开却是一则逆趋势的、利好中国的消息。而中国昨天也开始接受波音公司的飞机送货了。早在五月 的会员专属报告中，我已经与读者讨论了，贸易战今年最坏的情景已经过去了。

近来，我们还看到了中国市场流动性的改善。央行在逆回购上频频出手，保持市场流动性的充沛。中国的狭义货币和广义货币的条件都在改善。而全球流动性的改善是我在下半年展望中对于各个主要资产类别的分析里最重要的因子之一（图二、三）。这些情况在未来几个月都应该利好中国的风险资产表现。

从 2020 年以来，我们观察到中国大型股的表现和美债十年的表现开始负相关。换言之，美国债券收益率越高，对于中国大型股的表现就越显压抑（图四）。这很可能是因为我们使用的中国大型股指标是美国上市的、以美元计价的 ETF。因此，这个指标与美国的相关性更强。美债收益率上升的时候，也是美元资产折现率上升压抑美元资产价格、资金从美元资产流出之时。因此，由于我们上述对于资金流向的观察，资金流向的改变很可能将继续影响中概股的表现。然而，中国在贸易战中的相对优势和中国市场流动性条件的改变将是影响力更大的因素。

上周五 A 股指数冲高回落之后形成的日线级别的“暮星”形态。这是一个胜率一般的、短期的技术指标。我们看到一系列大盘股指标都出现了这个“暮星”的形态。因此，在为来几天，交易上很可能将表现为重要的技术阻力位久攻不下，市场出现一定的调整。但是这样的短期技术型调整不应该是作为看空的原因，而是应该逢低吸纳的理由。

总结一下：我们之前讨论过的，由于流动性条件改善而导致的港股新高已经开始逐步实现。而中国流动性还在持续改善，利好中国风险资产。然而，短期美国通胀将继续左右美联储降息路径。美债，尤其是美国长债的收益率飙升对于中国资产价格之前并不是一个利好。但这个因子对于中国资产价格波动的影响将开始减弱，中国流动性条件和下半年政策的应对才是决定中国市场走势的关键。

在上周五中国股指冲高回落之后，短期日线级别主要指数都出现了“暮星”的日线级别的技术形态。未来几天，交易上很可能将出现一定的阻力。然而，这不应该是看空的借口，而应该视为阻力位突破后回踩的确认。如果确认成功，之前的阻力位置将转身变为未来的支持位置。但是如果确认不成功（小概率事件），那么大家都知道意味着什么。这里也不需要赘述。
