
# 学习笔记，2025.4.1(1)

1. 好多人不知道买什么保险，又因为卖保险的口碑不太好不想去咨询。认准一点就行，不买理财性质的保险，那种承诺买多少年返多少钱的，基本都是坑。有社保，基本就够用，主要是医疗和养老。想加个商业险，买意外、重疾、医疗类，一年一付那种。

2. 沉默也是一种表态。你不去反对那些劣币，就是在用沉默给劣币投票，以后就别怪劣币驱逐良币，这是自己选出来的结果。你不想把世界的话语权让给那些蠢货，就该去试着表达，把话语权从他们手上抢过来。

3. 不存在说，人到了哪个年纪，就自动拥有了什么，也不存在人越长大就一定会变得越来越好。成熟、稳重、财富、家庭……这些都是自己奋斗出来的东西，而不是天上掉下来的。很多人毕业之后就几乎停止学习了，剩下的日子就是重复，除了年龄的增长，其他没什么变化。

4. 搜索是学习的元技能，考验的是提问能力和查找信息的能力，有这两个能力，想学什么都能学得会，辨别真伪的能力也会更强一些。没这两个能力，只能被动接受别人的投喂的信息，就约等于把自己的命运寄托在自己别人身上。

5. 学新东西，难点不在新东西，而在旧观念。人总是会先入为主地用旧观念解释新东西，于是误以为自己了解新东西。就像，现在很多人都在学AI，用旧有观念来理解，就是一个巨大的知识库，把AI当成搜索引擎入口。但AI能做的远不止这些，还有很多新功能等着人们去探索、发现。

6. 人生是一个积累的过程，记忆在积累，痛苦在积累，快乐在积累，财富在积累。不会突然消失，也不会突然爆发，好事、坏事，都是一个持续积累的过程。人会暴富，是经验多年积累，在某个节点得到了爆发式增长变成了财富。人会返贫，是思考多年偷懒，在某一段时间内，自己认知配不上的财富被别人骗走。总想着短时间能有巨大变化，内心就不能安稳，总想着去找一些奇技淫巧，就会越来越浮躁。

7. 人不怕走弯路，就怕不上路。走弯路还能绕回来，不上路，前面有再多的直路，也没用。想做什么事就去做，别光想，想太多不去做，只会徒增焦虑。

8. 财富自由是工具，不是目的。目的应该是注意力自由、时间自由，想把时间用在哪就用在哪，想不用在哪就不用在哪，自己的人生自己说了算，不用看别人眼色做事。现在大多数平台，都在想尽一切办法做留存，吸引你的注意力。你的注意力，对平台来说就是财富，你自己不珍惜，他们就会拿去变现。反过来想，如果你有了愿意投入时间和注意力去做的事业，你的自由就在这里，又何必太在意背后变现的数字。
