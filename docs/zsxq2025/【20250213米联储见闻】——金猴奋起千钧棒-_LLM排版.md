
# 金猴奋起千钧棒

我很少用这句诗做标题，但我觉得今天算是一个应景的日子，一部中国动画片依靠绝大部分的中国观众中国市场，在短短15天打出100亿票房，算是创造了电影票房史上的一个小神话，在全球质疑中国衰退，经济下行，投资缺乏亮点，消费不振的当下，这已经不是一剂强心针那么简单，更重要的是，哪吒2并不是孤例，从8月开始的黑神话悟空热潮到六代机横空出世再到深水引爆科技酱缸，再到哪吒挑枪打破消费力不足的魔障，加上开发署停发狗粮，导致不少外媒借机传播这部中国电影来要挟美国尽快发放狗粮。在全球票房榜里，前五十名都是美国电影，哪吒是唯一一部非好莱坞冲击榜单的挑战者，这背后不仅是中国工业实力的推动，还是中国人集体意志的推动，这场景完美再现了哪吒2电影中，一众海兽推举定海神针，打破天庭垄断的画面。恰恰是这个逆天的票房终于让傲慢的全球资本不得不重新审视中国的潜力，毕竟谁会拒绝钱呢？一部投资5亿的电影在15天创下100亿票房，这是近乎20倍的回报，而且是在单一市场完成的，没有人能忽视这股潜在的消费能量。

从某种程度来看，哪吒百亿票房这件事，不仅仅只是电影圈的破局，这也是为什么广电总局拿出巨大的媒体资源来配合哪吒做宣传，导致这个票房记录在短期内后无来者，这是天时地利人和的结果，如果是在去年春节上映，同样的哪吒2也达不到这样的印钞速度，经过去年一张的盘整磨底，经过这一轮一又一轮的认知冲击，要打破美国天下第一，中国始终不行的成见，需要这种真正硬核的产品，比如黑神话悟空，当IGA决定将年度游戏颁给别人时，没人会质疑黑神话，只会质疑IGA的含金量；当六代机飘过天空时，关于中国军工全靠偷的说法立刻成了段子，当deepseek选择开源时，全世界再也不敢说小看中国的高科技实力，而是老老实实来学习；当哪吒15天拿下百亿票房时，连长期黑中国的纽约时报都不得不反思，中国是不是已经不再需要好莱坞，毕竟2023年中国票房前十没有一部外国电影，即便是在中国本土电影非常低迷的2024年，也只有一部好莱坞大片进入前十，对于美国故事中国人已经相当不感兴趣了，而不管是黑神话还是哪吒，六代机还是deepseek，本身的都是中国叙事的不同表达形式，更重要的是，不管是刻意还是偶然，这几件硬核产品背后都是全中国团队，有些甚至连留学生都没有，真正的中国制造2025，从内到外的中国本土制造，连殖人都难以阴阳怪气，当然，仅靠这些东西想要撑起中国现代化之路还远远不够，但正如冯骥所言，踏上取经路比抵达灵山更重要。四个自信喊了这么多年有多少人真的相信，只有那些以行动丈量差距，以坚持突破困境的开拓者，在用真正的中国力量撑起四个自信，首先要相信自己，才敢于打破成见，如果连自己都无法相信自己能做到，成见终将是人心中难以搬动的大山。没有什么是真的无法战胜的，也没有什么是不能挑战的。中国人几千年前就笃信王侯将相宁有种乎，不信命不仅仅是哪吒的命，是整个中国民族的命格，是华夏文明刻在骨子里的信念，愚公移山，精卫填海，这些上古神话都在讲述一个不会被现实打垮，不会被成见摆布，脱皮剔肉都要逆天改命的故事，所以，哪吒的故事只有中国人会懂，而且是秒懂，外国人最好看不懂，因为这么优秀的品质，不要让他们学会最好。

在中国人忙着看电影的时候，川普和普京的双普通话终于落地了，两人也没有虚与委蛇，一上来就谈最核心的问题——俄乌冲突怎么结束？让欧洲和乌克兰集体破防的是，这场通话川普事先既没有知会参战方乌克兰，也没有问北约其他国家的意见，而是直接拨通普京的电话，就像大人聊天小孩别插嘴一样，坐实了北约其他国家和乌克兰都只是肉鸡的事实。这场通话发生在慕尼黑安全会议前一天，川普的百日俄乌停火大计将会是这场会议的主菜，只是北约万万没想到，川普先跟普京打电话再去开安全会议。而且这场通话持续了一个半小时，在没有跟欧洲盟友和乌克兰商量的情况下，两人同时发布了同意马上启动乌克兰的和平谈判的消息，欧洲全体工具人既视感，就喜欢川普这种连装都懒得装的爽片感。

更打脸的是，川总在打完电话后先发了一条社交评论，然后才说会打电话给泽连斯基告诉他和普京沟通的结果，等于作为参展方的领导，泽连斯基可能比网友还晚知道老大已经跟对手牵手手了，这真是骑脸撒尿的节奏，更离谱的是，川普表示普京会访美，他也会访俄，不过两人会先在沙特见面。这让享受惯了国会演讲待遇的泽连斯基情何以堪，果然是180度大转体了，打了三年，发现自己居然是个外人。泽连斯基反正也是做狗，同川普通电话后，假装若无其事，附和通话有意义，说“没有人比乌克兰更想要和平”，表示要“如川普总统所说一样，让我们把它完成吧。”我只能说乌克兰人是真能忍呀，100多万人死于战争，国破家亡，美国人说要打，乌克兰人就去送死，美国人说别打了，乌克兰总统就说，那好吧!这尼玛什么贱骨头呀，没有根的民族活该被蹂躏。乌克兰好忽悠，欧洲可不全是傻子，双普通话后，欧洲立刻炸了，英、法、德等国及欧盟委员会发表联合声明，指乌克兰和欧洲必需是任何谈判的参与方，表明乌克兰获得公正的和平是跨大西洋安全的必要条件。但川普会鸟欧洲吗？没有美国主导的北约能撑场面，欧盟主席和北约估计都要断粮了，现在轮到芬兰傻眼了，我尼玛鼓起多大勇气来造俄罗斯的反，现在北约都还没加入，组织尼玛不玩了，这是什么鬼呀。还有刚加入北约的瑞典，钱和人都准备好了，突然一下不知道给谁了，加入个寂寞，被炸了北溪管道的德国更是有苦没地方说，美俄打个电话就要和好了，其余的全部欧洲买单，刚刚与俄罗斯电网脱离的波罗的海三国，估计心里也想骂妈妈皮。

当川普政府连美国内部的大佬都不给面子，更不给欧洲的政客什么面子，这些人在川普眼里，都是摁着美国脖子喝血的寄生虫，正在布鲁塞尔出席会议的美国国防部长海格塞斯直接表态，恢复2014年之前
的乌克兰领土是一个不切实际的目标，只会延长战争、造成更多苦难。他强调，美国不相信乌克兰的北约成员地位是合乎现实的谈判结果，这种安全保证必需由有能力的欧洲和非欧洲军队支持，美国不会自己派兵，而其他北约成员国未来如果在乌克兰驻军，这些军队将不会受到北约联防条款的保护。此话一出，乌克兰和欧洲盟友心里都凉凉了，说白了，你们谁要保护乌克兰自己去，不关北约的事。欧洲国家心里想，我是有病吗？当初不是美国怂恿，谁尼玛有病去拿棍子戳大俄呀。美国再次以实力证明了，做美国的盟友会送命。川普随后在记者会上重复了这一结论：乌克兰要拿回2014年失去的领土不太可能，而乌克兰加入北约也不实际，两记闷棍彻底打掉了这仨年俄乌战争的全部意义。正如导演饺子在很多年前拍的那个动画片——打，打个大西瓜呀!

现在川普任命国务卿鲁比奥、中央情报局局长拉特克利夫、国家安全顾问沃尔兹和原来的中东问题特使威特科夫去领导俄乌停火谈判。而素来对俄强硬的俄乌特使凯洛格不在谈判领导名单上，凯洛格正在去欧洲出席慕尼黑安全会议的路上，他原本计划同欧洲各国沟通讨论对俄乌停火协议的想法，并访问乌克兰，此前有报道指他将会在慕尼黑公布和平计划的一些内容。凯洛格对俄持较强硬立场，认为美国可增强援助乌克兰、加大制裁俄罗斯来作为乌克兰的谈判筹码。但是没想到他人还在路上，川普就釜底抽薪直接跟普京通话了，而且做出了大方向的表态，等他到达慕尼黑，等待他的肯定是来自欧洲各盟友的口诛笔伐喷口水，而他在安全会议后发布的信息也已经没有价值，因为美国总统已经跟俄罗斯总统达成共识，特使的功能从决策前沟通放风，变成了给总统独断专行擦屁股，只能说川普坑自己的队友也是相当专业的，又或者这个特使是不是某些军工复合体基于利益强加给川普政府，结果不但没有保住军工复合体在俄乌冲突上的利益，反而被川普摆了一道。明天，川普就可以双手一摊对全世界说：我说了，结束战争没那么难。“世上无难事，只要肯放弃”，如果肯认输，哪里还会有什么战争。反正俄乌冲突富裕的是民主党的口袋，川普是一天都不愿意多等了。

与此同时，美国财长贝森特也访问了基辅，与泽连斯基商谈以乌克兰稀土资源换取美国援助的协议，现在川普想的是，如果尽快入场把美国投资这场战争的收益拿回来，川普这个吃相必将引起整个欧洲的恐慌，毕竟大家都出力了，结果到了要分赃的时候，美国一个人先冲进去了。贝森特称矿产协议是更大的和平协议一部分，美国同乌克兰的经济合作在冲突结束之后将会为乌克兰人提供长期的安全盾。言下之意，你反正人都打完了，还想要人保护，怎么滴都要卖点家当做保护费吧。乌克兰走到这一步真是彻底把国家败光了，乌克兰人不撕碎泽连斯基都对不起死于战争的100万同胞。

所以留给乌克兰的出路很明确，拿回2014年的土地不可能，加入北约不可能，美国可以提供保护，但要
用稀土矿来换，这就是美国对乌克兰的最大支持了。好好的一个国家，活活被打成另一个国家的矿区，人变人矿，国变国矿，真是最生动的历史教训。

更令人搞笑的是，知道今天泽连斯基还在渴求美国的保护，在接受经济学人的采访时，泽连斯基终于说了句大实话，没有美国不行，北约都没用，现在整个欧洲的现有可用兵力人数比乌克兰还要低。这就难
怪川普一查帐立刻翻脸了，搞了半天北约，原来北约是这副垃圾家底，战斗力还不如快打穿的乌克兰，就
这个实力还想上谈判桌，滚犊子吧!
