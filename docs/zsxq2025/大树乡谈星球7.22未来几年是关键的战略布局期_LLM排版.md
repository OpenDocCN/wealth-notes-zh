
# 大树乡谈星球7.22未来几年是关键的战略布局期

近年来，围绕中国的政策倾向，有一些争论。有部分机构和专家认为中国应该减少投资，更多投向消费。对此小镇已经谈过一些了，很多机构和专家之所以与国家决策相差甚远，关键在于视野和格局，站在国家层面，资源是有限的，“投资于人”是重要但不紧急的任务，未来几年则是宝贵而又关键的战略布局期，需要进行更多资源倾斜。

如何理解？小镇将分三部分，第一部分是理念，第二部分是趋势，第三部分则举例具体阐述，第三部分只发在星球。

一个伟大国家的真正崛起，既需要实际崛起，也需要被周边国家以及全世界公认，从而彻底重塑周边及全球秩序，通过新秩序被普遍承认实现低成本的全球治理。笼统而言，有两种途径：一快、一慢。

一快：就是通过暴力、战争实现领导权的转移，一般称之为“霸道”。西方国家的全球领导权转移基本通过这一方式，先是西班才帝国通过殖民掠夺建立霸权，之后无敌舰队被英国击败，霸权更迭，美国则因为与英国复杂的关系，更借助两次世界大战的惨烈，实现了看似和平的霸权更迭，但实质仍然建立在战争基础上；而欧陆更是列强战乱不休。

一慢：以更加温和的手段实现秩序重塑，一般可称之为“王道”。这两条路有利有弊，前者相对简单也更快但衰败也快，后者更难更慢但也更稳，目前全世界几百年来，走“慢路”的后发崛起国家，也只有中国一个。

这两条路都要求获得强大的力量，前者一旦获得强大力量之后，就要从“实力”出发去谋求相应的地位，这条道路开不在意厂大中小国家的意志。“实力与地位”也是遵循这条道路的国家最常说的话，也就是说两步走：获得力量、以力量博取地位。

然而后者就要复杂很多，获取力量只是基础中基础，之后才是更加艰难的时刻。一方面要在已经拥有强大力量的同时，克制内部动用力量解决问题的冲动，这已经非常困难了。因为随着国家的崛起，必然开始面临霸权国设置种种制约，而霸权国主导的全球秩序以及其他国家的惯性选择，也导致后发崛起国面对越来越多的发展天花板问题，这些问题如果按照“霸道”的做法，只需要用力量去打破，虽然长期不利，但短期确实很爽，这就是“身怀利刃，杀心自起”。

小镇之前还在文章《战争改变国家，未来又该如何?》提到，战争往往是平民彻底改变阶层的捷径，武力的运用会迅速形成一个强大的利益共同体，会逐步影响甚至主导一国的发展方向。

另一方面对外则要解决，如何在克制力量使用的情况下，令其他国家相信崛起国真的拥有了足以抗衡霸权国的力量，更进一步要让包括霸权国在内的其他所有国家，尤其是周边大国，接受崛起国拥有如此强大的力量，最终令其他国家从害怕、抵制崛起国拥有强大力量，转而期盼崛起国拥有更强大的力量，崛起国拥有的力量成为维护全球秩序稳定的基石，这就实现了秩序的重塑。

上述逻辑小镇画了一张“快慢崛起路”示意图，放在文末。

中国目前，显然已经拥有了足够的力量，在美国看来，现在的中国就相当于苏联、日本、德国等所有后发挑战国之和，还要更强。毕竟历史上，从未出现过一个非西方国家，制造能力超越整个西方之和，甚至直接间接掌控的全球生产力已经超过一半，虽然美国也曾经达到过如此占比，但当时是二战的特殊情况，而且人类科技水平还很有限，质量仍然无法完全替代数量。

而在拥有人类历史空前物质力量的同时，也就意味着中国对内、对外克制力量使用冲动的压力变得更大，也会引发来自霸权国、周围大国对中国强大力量的猜忌甚至是恐慌。当然在周边以外的其他区域，比如长期习惯外部力量进驻提供秩序保障的中东等地区，更疑惑为什么中国不肯派驻军力，这也影响了国家之间关系的进一步升级。

毕竟在国家之间关系深度方面，经贸往来实际上是最低层级的，之上是金融，最终则是军事和安全。正因此，小镇在最近几个月谈周边工作变化，特别强调中国与东南亚关系，正在从传统经贸关系向更深度的安全合作迈进，这是非常关键的转折。

这种从不相信到猜忌再到恐慌引发应激反应，最终清醒认识到中国力量的客观存在，并且转而选择承认拥护，这个过程需要的时间很长。当然在美国看来，短短20多年就实现了如此跨越，已经太快了。

简单举几个例子体会下：

从霸权国也就是美国的角度：近10年越来越清醒认识到中国国力、军力的迅速壮大，美军甚至成为中美和平的积极拥护者，这个变化的节点是2016年南海、2018年首轮贸易战、2022年航母对峙以及2025年中国双航母突破第二岛链和中美第二轮关税战对垒。现在美军方面普遍认为，2030年以前中美军事力量在纸面上就会出现历史性逆转，现在美国内部已经开始讨论如何实现对中国的区域拒止，也就是阻止中国进一步深入美国的传统势力范围。

从周边大国角度，典型代表是印度：印度心里一直有一根刺，刚立国的时候，印度基础远比中国好，同为发展中大国，印度一直希望成为全球南万的旗帜，然而偏偏有中国仕刖，更仕朝鲜战场打出了军威国威，这种复杂情绪在1962年后彻底升级。印度一直觉得“中国行，我也行”，但实际印度确实不行，最近10年尤其是2017年的洞朗对峙后，印度之后一直采取挑衅、进攻姿态，在边境大规模增兵，尤其以非法的“麦克马洪线”为由侵蚀中国国家主权。然而印巴5·7空战，令印度开始冷静下来。在雅江水电站项目上，印度在5·7空战前后态度截然不同，年初还坚决反对的，现在也转而认为雅江水电站的建设有利于提高印度抵御灾害的能力，也就是我国长期主张的“不仅无害，反而有利”。

其他中小国家的态度转变更加明显，典型就是缅甸：曾经在港片中作为背景最终极大BOSS的神秘势力，结果被中国一个地级巾公女局的通缉就瓦解了，这就是实力和影响力。

更多例子不多列举了。

总之参考“快慢崛起路”的示意图，2025年的当下，包括霸权国在内全球各国，已经开始承认中国已经拥有了强大力量，越来越多的国家已经不再忧虑、猜忌乃至恐慌中国力量的增强。

网上有很多段子，比如有段子说，当中国一整个集团军开始动员的时候，东南亚各国反而不紧张，毕竟全世界能够抵抗中国一个集团军的国家，没有几个。

当然，更重要的是，中国几十年如一日坚持“和平共处五项基本原则”，始终保持平等合作、协商共赢，以实际行动践行“人类命运共同体”，更重要的是说到做到，反而对最强大的美国针锋相对，这就让全世界各国相信中国和过往西方国家真的不一样，确实拥有足够的自我克制能力。

现在的美国也已经开始转变，已经逐渐承认中国与美国拥有同等实力和地位。这从今年的关税战就能看出，特朗普现在对全球包括欧盟、日韩、印度等国采取非常强硬的态度，然而对中国反而采取怀柔态度，从各个渠道表达希望同中国友好协商的姿态，几个月前的针锋相对就像是做梦一样，似乎关税战与中国已经无关了。美国的这种姿态，也影响到了印度、日韩以及更多国家对中国的态度。

现在中国需要做的，就是尽最大限度避免意外冲击，继续保持稳扎稳打的姿态，度过最后的厄险过渡期。这个厄险开个是说中国会大败，而是避免不必要的战争与破坏、避免中国从王道坠入霸道，目前国内已经出现了越来越强烈的使用力量的呼声，而美国也在煽动如菲律宾等国家，诱导中国动用武力，从而破坏中国几十年建立起来的国家信誉和王道形象，引发其他国家的恐慌。

以上就是背景和理论的阐述，接下来要结合现实。

最近新成立了新央企集团，直接位列第22位，而在官方通稿中强调之所以成立该集团，目的是“深入落实总体国家安全观、能源安全新战略和新时代党的治藏方略”，这句话一定要好好品味，更要结合前面阐述的理念和形势变化，意味无穷。

该集团所属万亿水电站的主要目的是发电，但集团本身可并非仅仅为了发电，更重要的是服务国家战略。这一块作为最后一部分的具体案例详细展开。

总结近年来新央企设立，能够更加清晰地体会宏大的国家战略。

2019年安能建设集团和石油天然气管网集团，2020年融通资产集团和南水北调集团，2021年卫星网络集团和稀土集团，2022年矿产资源集团，2024年资源循环集团和数联物流，2025年新央企集团。

可以清晰地看到三条主线：

一是科技创新木米广业，如卫星络和驮物流；

二是机构改革深入，如涉及军改的安能和融通；

三是战略资源：包括石油天然气管网、南水北调、稀土、矿产资源、资源循环和新央企集团。

这几条线一个共同点是战略和安全布局。既是满足国内进一步发展的需要，也是以和平方式完成对外斗争的关键节点布局。

比如最近成功卡住美西方脖子的稀土和关键矿产，还有提高中国资源循环利用效率从而降低对外资源依赖度，加强立足国内能源和矿物的能力储备，以及对抗包括冲突、气候变化等极端情况的深度布局。小镇昨天谈的“国家水网”也是这一战略趋势的重要动作。

把这一切与前面提到的中国“王道”之路相结合，重点考虑周边布局以及周边国家对中国力量强大的态度转变，尤其是南亚和东南亚方向，站在这个角度，更加深入地理解新央企集团的意义，这才能深刻体会为什么集团定位是“深入落实总体国家安全观、能源安全新战略和新时代党的治藏方略”。

最近高志凯教授有一个非常精彩的主张，就是提出以恒河划定中印边界的“高志凯线”。官方这么做不太合适，但民间以及1.5轨完全可以提出，不得不说高志凯和张维为两位作为小平同志的外文翻译，水平就是高。

围绕这三条线的明争暗斗，已经长达几十年。实际该地区自古以来自古以来就是中国领土，主权归属十分明晰，这一所谓主权争议，就是英国一手炮制的。

但现实很严峻，关键是后勤补给保障能力。1962年对印反击大胜，但不得不快进快出，就是因为补给太难，对印度来说，喜马拉雅山以南是一片平原，而中国需要跨越世界屋脊和喜马拉雅山，以当时中国的国力而言，很难维持在当地的充足力量，如果力量不足，反而容易陷入被动。

结合上述背景，就可以更加深度理解在书面之外，重大工程的潜在战略意义。

1X水电站就需要十几年建设，而开始土体工程建设之前，当然要先解决物资运输和施工保障问题，除了保障钢筋水泥等基础建材运输，也需要建设一些配套工厂，还要维持庞大施工队伍的长期生活。目前派墨公路新改建正在进行，墨脱支线机场、察隅通用机场完成选址踏勘，而川藏铁路全线预计2030年通车，实际肯定会提前。

围绕超级工程建设，必然要求道路、后勤保障、公共服务、基础设施等等向前推进，这是工程建设的需要，也客观上形成了一个稳固的桥头堡和经济示范区。这个桥头堡背靠超级工程的建设和运营，获得了雄厚的经济基础，能够实现自我造血。

产业自我造血和转移支付补贴是截然不同的。

之前保障工程建设的道路、工厂，肯定不会在工会完工后拆除。而更重要的是，中国直接把更先进、更富裕的生活方式展现出现，这就是民心所向。后续，用六个字就可以总结“进可攻、退可守”。

未来几年，这些战略工程，优先度必然更高，也是为后续加力民生消费做准备。
