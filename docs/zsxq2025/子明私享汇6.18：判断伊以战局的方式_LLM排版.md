
# 判断伊以战局的方式

舆论战场也是战场的一部分，充斥着各个利益集团释放的假消息，辨别战场上的趋势变化，最好的方式就是对比超级大国的态度变化。以22年的俄乌冲突为例，早期默克尔、马克龙和基辛格都站在俄罗斯一边拉偏架，到了6月份后，他们纷纷跳船去了乌克兰这一边，巴拉特也免去了高呼“无上限”的预备外长L，之后就是乌克兰的大反击。同样，25年的这一场伊以冲突，最近几天的时间里，几个超级大国的态度也都迅速发生了巨大的变化，德国默茨对以色列从批评，变成了17号的强烈支持，巴拉特以色列军事行动也从14号的强烈谴责，变成了17号的反对。尤其是美国那个好大喜功的特朗普，从早期的主张谈判置身事外，骤然变成了考虑亲自下场，甚至喊出了要求无条件投降(UNCONDITIONAL SURRENDER)，颇有一种突然发现自己还没上车的即视感。

政治人物可以坏，但是不能蠢，他们掌握着这个世界最精准的外交与情报网络，就算一开始搞不清楚情况，很快就会搞到战场的真实情况，不会被舆论战所引导。而国际支持都是靠自己实力争取的，国军打了个豫湘桂大溃败，美援马上就锐减，共军进入汉城，苏联的军援和156个项目就拍马赶到。这场伊以战争才刚刚几天，作为裁判员的超级大国纷纷调口径，原因就是伊朗目前局势比我们看到的还惨，几个大流氓都在调整策略，准备上餐桌。

基辛格否认“建议乌克兰割地求和”：我从来没说过
来源：观察者网发布时间：2022年7月4日
今年5月，美国前国务卿基辛格在达沃斯论坛上呼吁俄乌边界“恢复原状”，但随后被外界解读为建议乌克兰“割地求和”。据美国《时代周刊》7月3日报道，基辛格在接受该刊专访时做出澄清，称“我从没这么说过”。

又一盟友态度转变?默茨：以色列“不要太过火”
来源：环球网发布时间：2025年5月27日
据英国路透社、《以色列时报》报道，德国总理默茨当地时间5月26日批评以色列在加沙的军事行动，称以色列对加沙平民造成严重伤害。欧洲新闻网称，默茨的言论表明，又一个以色列盟友的态度发生转变。

默茨惊人发言：“向以军致敬”
来源：参考消息发布时间；2025年6月18日
据德国《时代》周报网站6月17日报道，德国总理默茨认为，以色列袭击伊朗是在为西方盟友效劳。默茨在加拿大七国集团峰会期间接受德国电视二台采访时说：“这是以色列为我们所有人在干的脏活儿。我们也受到了那个政权的影响。”

默茨说：“我对以色列军队和以色列领导人有勇气这么做，致以最崇高的敬意。”此外，默茨还说，他希望伊朗政府垮台。

特朗普：要求伊朗“无条件投降”
来源：参考消息2025年6月18日
美国总统特朗普17日下午在白宫召集国家安全团队开会，讨论以色列与伊朗正在持续的军事冲突。多家美国媒体报道，特朗普正考虑一系列选项，包括加入以色列对伊朗的空袭。

据报道，特朗普6月17日早些时候对媒体表示，“我没有太多心情与伊朗谈判”，并称他针对伊朗的目标是“结束，真正的结束，而不是停火”。他当天还在社交媒体上数度发文，称“我们的耐心已经耗尽”，要求伊朗“无条件投降”。

美国有线电视新闻网6月17日援引两名知情人士的话报道，特朗普正越来越有意动用美国军事资产空袭伊朗核设施。不过，如果伊朗作出重大让步，特朗普仍对外交解决持开放态度。
