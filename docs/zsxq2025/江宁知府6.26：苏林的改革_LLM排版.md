
# 江宁知府6.26：苏林的改革

## 前言
苏林在就任越共领导人后仅两周即火速访华，里面其实有寻求中方支持（以巩固内部地位）的意味，类似做法在社会主义大家庭时期很常见。“精简一扩张一再精简”几乎是各种机构一致的演变路径。自1975年南北越统一以来，越南共进行过三次大规模机构改革：第一次是1981年黎笋实施的“苏联化”改革；第二次是1992年杜梅和武文杰进行的“去苏联化”改革；第三次是新千年之初农德孟推行的改革，包括大部制、市场化、增加国会权力等。就效果来看，第一次全面失败，第二次和第三次成效显著--不仅使越南逐步摆脱经济危机，还直接推动了长达三十年的持续稳定增长。本轮由苏林推动的“开创越南历史新纪元”改革为第四次，主要目标是精兵简政，其机构改革力度之大十分罕见，包括辞退大量公务人员、合并一半的省/直辖市以及改“省-县-乡”三级结构为“省-乡”两级结构等。根据越南官方说法，这次大规模机构调整早在阮富仲生前就已达成了共识，苏林任内属于全面铺开。这某种意义上表明苏林仍在其“政治导师”阮富仲的轨道上前进，或者说仍需要借助后者的影响力。中越作为治理模式相似的两个国家，存在许多共性问题，一些改革措施也可以/可能彼此借鉴，正所谓“他山之石，可以攻玉”。

## 1、体制内退出机制
越南在此时进行大规模体制内人员清退显然是经过深思熟虑的，这是一个精心挑选的时机。越南目前按美元计价人均GDP算大致相当于中国2008、2009年的水平，按融入全球产业链时间算则相当于中国2004、2005年的水平。这一时期，恰好是外资企业全面进入，外企收入明显高于体制内收入的历史阶段。目前越南国内的三星、英特尔、富士康、LG、本田、丰田等大厂的薪资福利是优于公务员的，就业市场上非常受追捧，市场化就业取代“铁饭碗”心态的转折点已经到来。同样的逻辑，假如未来中国进行类似的操作，所挑选的时机也应该是市场化新兴岗位大爆发的阶段。

对于高度膨胀的体制内人员，越南在此次改革中大胆尝试了“带补偿的非惩戒性退出机制”。2025年2月18日，越南国会正式批准了几十年来力度最大的政府改革计划。越南政府部委的数量将从之前的18个减少到14个，同时仅保留三个部级机构以及五个政府直属机构，此举将把部委的人员规模缩减约20%，预计10万名公务员受影响。6月，越南国会再批准地方省市的合并计划，把全国63个省市行政机构精简至34个，地方层面改现行的“省、县、乡”三级结构为“省、乡”两级模式，当前10035个乡级行政区重组后将保留约5000个。对于一些重点乡镇，合并后将会出现“省级常委兼乡镇党委书记”的情况，听起来十分魔幻。机构撤并必然涉及人员优化，那越南方面是如何操作的呢？

苏林拜访农德孟。农德孟1940年生，2001至2011年间为越共一把手，其为人温和谦恭，深受各界好评。作为越共内部没有直接参与过越南战争的新生代，农德孟在政治上持温和中间立场，属开明务实派（西方媒体称之为“体制内的妥协派”）。下面是越媒报道的一则案例。M先生是越南部委的一名公务员，工作时间15年，职级为5级，目前月薪1070万越南盾（约合2900元人民币）。由于组织机构调整，M先生自2025年6月1日起辞职，他将享受一次性社会保险（即一次性提取社保账户中的钱，不再按月领取养老金），同时获得总额为7.9亿越南盾（约合22万元人民币）的补贴，具体包括：遣散费5.2亿越南盾、津贴4亿越南盾、求职补助3000万越南盾。以月薪进行换算，相当于一次性支付6.3年的工资（不含一次性社保提取）。根据越南官方资料，公务员年终奖金和津贴约占总收入的30%，假如算上这部分钱，对于这位M先生来说，实际支付的裁员赔偿大致相当于4至5年的总收入。在当下越南的机构改革中，往往是一些年长的公务员比较接受被辞退，或主动申请被辞退，因为他们拿到的补偿更多，若被优化可当作“提前退休”，心理负担较轻，甚至可以领完补偿后再去私营部门兼职。而年轻人因补偿较少，大都不愿意离开政府机构，很少有40岁以下的公务员主动申请离职。这种“吐故留新”的政策引导某种意义上也是为了进行年龄优化--老员工主动离职可以为年轻干部腾出岗位空间，让改革更具活力。

越南这一机制颇具启发意义，因为中国目前对大龄晋升无望的公务员也缺乏有效退出通道，不排除将来做相关探索。不过需要注意的是，虽然中越在编制膨胀、财政压力、干部年轻化、机构效率等领域面临共性问题，但双方国情与现实考量并不完全相同。越南没有像中国那样的全国/全省统一公务员考试，其公务员招聘通常由各个政府部门或机构自行组织，采用不同的招聘方式。这让越南公务员的身份/编制灵活性更大，不像中国那样“正式”。除此之外，当下越南面临的外部环境较为宽松，中美俄欧日韩都在争相拉拢越南，而中国则处于“紧绷”状态，对保持内部稳定看得格外重。

## 2.4 越南公务员工作场景
根据越媒报道，在人员裁减过程中出现了各种临时聘用人员、合同工以及挂靠人员现象，所谓“裁编不裁人”，导致“暗养”成本增加。实际上，行政体系优化的核心在于“减事”，把原本由政府管辖的事情放开让市场调控，让“非必要行政行为”退出社会运行主线，也就是我们常说的由“管制型政府”向“服务型政府”转型。可当下体制内部门越来越“泛行政化”，招商引资、企业招工、学生健康、青年就业、城市卫生、邻里纠纷、农业活动等等事情全都要由政府负责，导致疲于应付，出现形式主义。尤其是基层单位，成为“事多、人少、责任重”的典型，只有先“减事”，才能后“减人”，否则很容易造成治理坍塌。

## 3、行政区划撤并
本轮越南机构改革中，地方上的变动远比中央部门大，并省、撤县搞得不亦乐乎。在大家比较关注的层面，河内市保持不变，胡志明市获得极大扩充--巴地头顿省、平阳省将和胡志明市合并，形成新的胡志明市。目前河内人口为900万，约4000亿人民币；胡志明市人口1000万，约5000亿人民币，南北两大城市综合实力相当。而行政区划调整后，新胡志明市将成为一座人口高达1370.6万的特大城市，拉开与河内的差距，在东南亚七大城市中有望从“下半区”冲击至“上半区”（新加坡、雅加达、曼谷、胡志明市、吉隆坡、马尼拉、河内）。

对于许多体制内朋友来说，发生在越南的并省、撤县同样有可以借鉴的启示。越南的省大致相当于我们的小地市或大县（一两百万人口），越南的县大致相当于我们的小县和中心镇（约十万至十五万人口）。一个小县城，十几万二十万人口，吃体制饭的得有一万多（包括各种编外人员），大部分县区都是以这些人的开支和行政支出作为最核心的一环，然后往外扩散，层层受益。换句话说，小县城其实很依赖于体制内人员的收入进行经济循环，只要体制内降薪，经济就很难好，因为这是负反馈。然而从长远看，随着信息传播愈发便利、交通更加发达，对于人口不断流失的小地市、小县城来说，撤并将是大概率事件。

目前许多县级财政日常第一大支出就是给老师发工资，叠加学龄人口逐渐减少，其裁减教师编制动力很充沛。城市撤并的思路无外乎两种：1、相邻城市如果是“强-弱”结构，则“弱”并入“强”，比如1997年重庆直辖，就合并了周围的涪陵市、黔江地区和万县市。2、相邻城市如果是“强-弱-强”结构，则有可能两“强”瓜分一“弱”。因此，无论体制内人员还是普通人，都应提前去大城市、去相对能量级较高的城市发展，比如副省级城市，因行政级别高，在城市合并中就占有很大优势。即使不能去副省级城市或省会，也尽可能到小区域的中心发展，比如江苏徐州、山东临沂、湖北宜昌、陕西榆林这种。

## 4、越南未来走向
2025年5月30日，苏林在纪念南方解放、国家统一五十周年庆典活动上发表讲话，里面有几处细节颇耐人寻味。柬埔寨军队和前首相洪森出席了阅兵庆典，与越南渊源深厚的洪森（主席台穿军装者）坐在了越南国家元首梁强和总理范明政中间的位置。苏林讲道：“在这神圣的时刻，我们深切缅怀并无限感恩敬爱敬爱的胡志明主席——我们党和民族的天才领袖，越南革命的伟大导师，民族解放的英雄，世界文化名人，国际共产主义和工人运动的杰出战士。”众所周知，南北越统一是在黎笋任内完成的，胡志明1969年就去世了，但苏林全篇演讲没有提到黎笋，而是将其包含进“历代革命老前辈”里（在同期发表的署名文章中提到过黎笋一次，但频次和力度远远少于胡志明）。这种高举胡志明旗帜而淡化黎笋的做法是革新开放后越南“北方派”的一种务实思路，当下被苏林所继承。

总结抗美救国事业取得胜利的原因时，除越南自身因素外，结尾处苏林还放了这样一段话：“（胜利）源于苏联、中国以及其他社会主义兄弟国家给予的巨大、正义而有效的支持与帮助；源于越南-老挝-柬埔寨三国之间的特殊团结；源于世界各国爱好和平与进步的人民和力量，其中包括美国进步人士的支持。”当中不难体会越南试图平衡大国关系的微妙心态。苏林的家乡与仕途履历全部在越南北方。尽管目前越南政坛已不再区分所谓的“北方派”和“南方派”，但整体而言苏林的立场是贴近于北方的--以河内为基础的北派政治人物长期与军警系统走近。实际上，苏林在就任越共领导人后仅两周即火速访华，里面其实有寻求中方支持（以巩固内部地位）的意味，类似做法在社会主义大家庭时期很常见。

如果说目前的各种调整只是苏林权力的试金石，那么2026年1月将成为后阮富仲时代越南政治生活里真正的十字路口。届时“四驾马车”谁上谁下，“阮规苏随”是否继续，越南何去何从，谜底将一一揭晓。
