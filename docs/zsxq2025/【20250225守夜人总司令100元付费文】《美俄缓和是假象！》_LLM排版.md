
# 美俄缓和是假象！

许多人认为现任美国总统一特朗普是一个亲俄罗斯的人，这是一个误判。特朗普本质上是个商人，特朗普的首要任务是把自己的油气、粮食等产品卖给别人，主要潜在买家包括欧洲，印度，中国。所以特朗普会与俄罗斯和伊朗甚至沙特形成竞争关系。

另一方面，特朗普的任务是让资金回流美国，要让资金回流美国，目前唯一的想象空间就是科技创新，其他方面都缺乏号召力。贸易制裁的主要目的有两个。一个是为创造谈判的筹码，另一个就是让别人到美国那里投资。之前有一篇叫《24年1204：鹰酱的阳谋》那就是美国的目的，不明白的可以回头再看一遍。

你不要看特朗普在俄乌冲突上的立场如何的偏向俄罗斯，这都是障眼法，项庄舞剑意在沛公，这个沛公肯定不是乌克兰总统泽连斯基这样的货色。特朗普此举的目的在于一箭双雕，对外逼迫欧洲妥协让步，对内则攻击特朗普自己的反对势力，因为特朗普现在还并不太稳，真正的稳不是靠你说什么，而是要为反对特朗普的势力找到利益，这个为内部反对势力谋利益可以从外部攫取。

如果说传统能源和粮食的销售，视为特朗普内部支持势力的基本盘谋求利益，那华尔街的利益呢？硅谷的利益呢？好莱坞的利益呢？这些特朗普内部反对势力则需要通过科技和金融方面的获利来安抚。目前不管是无人机还是其他相关的科技产品的核心部件，甚至是钢铁和铝的供应，如果抛开中国，美国的产业体系是无法独立运转的，所以，美国需要为这些美国自身无法独立运转的产业打补丁。打补丁的方式就是向外掠夺，向欧洲掠夺，向日韩掠夺产业和技术同时还需要掠夺原材料，所以美国今天要占这个岛，明天要抢那块地，阿根廷有很多锂矿，阿富汗也有，但是阿根廷的矿场是成熟的，阿富汗的矿产根本就不在塔利班手上，而是在阿富汗北方割据势力手上，所以，现在中国和美国都没有办法。

阿富汗北方部族势力虽然跟西方亲近，但是现在塔利班当家，没有与地方建立直接联系。中国虽然跟塔利班关系还行，塔利班也希望中国去投资，但阿富汗北方势力那块地儿根本就不在塔利班的有效控制范围之内。你要真去投资当地部族可不认你跟塔利班签署的协议，如同在我们认为中国跟巴基斯坦关系很好，但巴基斯坦真正能控制的区域非常小，关键的港口和沿途地区，虽然名义下是巴基斯坦的国土，但真正控制的是当地的部族，在当地人眼中，当地部族可不认中国跟巴基斯坦政府官方签的协议。

因为这个世界很多地区，政权的官方就是一个代表而已，这就好比当年德国需要进口钨矿，德国跟南京国民党谈的协议没用，最后要找人跟苏区共产党谈才有用，因为江西的钨矿控制在共产党红军手上，南京是国民党官方，但管不了江西的钨矿！乌克兰有许多矿产甚至是稀有金属，但是跟乌克兰谈没用，签什么协议都没用，因为矿在乌东部地区，连乌克兰最大的煤矿，乌克兰都丢了！如果美国真的想获取这些东西，就只能找俄罗斯谈，为什么不支持乌克兰把这些地方夺回来呢？从2022年开始的过去三年不是没支持乌克兰，西方全力以赴支持乌克兰以后还是不行。所以，特朗普要跟俄罗斯谈，特朗普要获得特朗普想要的东西。大毛就需要获得俄罗斯想要的的东西。俄罗斯和美国会不会达成一个协议：俄罗斯给美国所需要的矿产，美国同意俄罗斯对已占有地区的所有权，并部分解除制裁呢？不包括这样的可能，但是欧洲不愿意，因为出钱出力的的是欧洲，欧洲最后什么都没有到，所以欧洲坚持要打。

如果俄罗斯在获得了西方的默许之后，不把那些美国想要的东西给美国怎么办呢？俄罗斯没有太多选择。俄罗斯的经济按照现有的方式，难以支撑起长期封锁下的战争，如果这场从2022年开始的俄乌冲突战争打上十年八年，最后俄罗斯就真的成了一个大的沙特。美国就是看到了莫斯科急于达成协议解除制裁，所以在给俄罗斯创造一种值得期待的假象，好像是美国站在俄罗斯这一边，人一旦有了期待，而且这种期待越强烈，就越容易做出让步，特别是深陷困境之中的时候。

除了为俄罗斯创造期待，从而让俄罗斯更愿意做出让步之外，此举也是在向欧洲施压。特朗普会逼日本，逼韩国逼沙特，甚至逼中国加大对美国的直接投资，以关税的方式创造筹码来进行交换，其他人都不好逼迫，中国不会轻易妥协。中国最多加大力度去买美国的农产品和油气能源，拒绝不需要理由，但需要底气！在过去的5年中，中国房地产在整个经济中的占比从25%下降到了百分之21%，而科技产品以及精密制造的比重从11%上升到了16%，汽车产业做成了一个很大的产业。新旧产能轮回，现在中国国内房地产造成没有办法解决，那就只能拖，拖到房地产在整个经济中的占比越来越小，这就是解决的方式！当然了，在这个拖的过程中，存在一个很大的债务风险，这种感觉就如同开封市上有一个超出地面7米的黄河一样！你要时刻刻注意各种会导致地面黄河决堤的因素，并作出应对措施。

从目前的数据来看，2025年中国官方会持续的执行这种房地产要在整个经济中的占比越来越低的拖延政策。同时，把资源尽量向能够创造增量和产业切换的地方投入。不要看官方政府怎么表态，所有的表态都可能是安抚和诱导，从而为官方政府解决核心问题出力。
