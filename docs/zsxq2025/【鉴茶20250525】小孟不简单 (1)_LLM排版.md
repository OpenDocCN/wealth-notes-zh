
# 小孟不简单 (1)

## 回滚也回
5月23号的董明珠和孟羽童直播， 大王看了好几遍， 网上有切片大家可以去看。

## 血免费进资料群
董明珠说什么，
“你丧失了一次最好的机会， 格力平台很大， 你以为自己搞代言、 给别人做广告、 好像收入有了一点， 但你失去的更多。
“最后一次叫你离开的时候， 确实是你犯了很大的错误。 因为要讲规矩， 我们在工作当中 一定要讲原则， 情感是另外一回事。
“年轻人三观要正， 不能说你三观不正， 只是要打一个顿 ” 。
现场我们看到什么，董明珠一直要证明自己是对的， 自己不是不懂互联网， 不是冤枉孟羽童， 错不在自己。 还要逼着小孟 承认。

## 董的问题有什么
第一是心结太重， 也是那一代人的 ＂ 挂相 ” ，总要争是非对错。
事情都过去了， 不肯冰释前嫌总要占点便宜回来， 本质是不肯放过自己。
强烈情绪驱使能让人做成大事， 但不会让你有太强的幸福感。
第二是缺乏胸怀， 袁世凯都说过， 扬善于公庭， 规过千私室。批评人哪能公开说， 还上升到三观不正的角度。
孟羽童如果这么不堪， 你当年招她入职， 岂不是用人失察？
但重点不在这， 而在孟羽童面对如此重 指控 ，根本没有和董明珠去拼扯这些东西， 随口打哈哈应付两句， 剩下的时间面 不红心不跳的吧直播搞完了， 有点泰山崩千前而不改色的大将之风。
这说明什么，人家根本不会纠缠情绪， 不在乎你说什么，人家有一套自洽东西， 这是99％的人都很难做到的。

## 因为这些需要你有意识的， 长期的， 从小的训练。
孟羽童从小出生在部队大院和富足的家庭， 从小就参加各种运动会、 学习小组、 社会实践、 外语节、 青春剧场、 模拟联合 国等， 还当导演。
这就是 从小完成了社会化训练 ，自己很幸福， 没有重大的缺憾， 不需要外头提供情绪价值， 各种场合和人经历的多了，
对他人也没什么期待。
这就是一种 “ 应无所住而生其心 ” 。
所以我们培养孩子， 最重要的一条是为其创造 “ 稳定 ” 的环境， 让其自洽的成长， 而不是充满落差和期望。
董和孟， 是中国截然不同的两代人，
一代是因为匮乏和重大缺憾， 放手一搏最后功成名就。
一代人是因为充实和饱满， 坚定的做好自己脱颖而出。
我们与自己和解， 就是与自己的各种 意难平 和 重大缺憾 和解。
我们培养下一代， 不能按匮乏的模式， 而是要创造环境， 使其自信， 平静， 蓬勃， 就像特朗普曾经评价他老爸， 他让我感 觉像世界里的王 。
孟羽童知道， 她如果和董明珠谈对错， 完全是鸡同鸭讲， 毫无必要， 还不如不说算了。
这个道理， 估计她很小的时候爸妈就给他讲过。
