
# 学习笔记，2025.7.13(3)

1，古镇在开发时，每一家都想做出特色。可是，做着做着，就向市场妥协了。
光投入，不赚钱。最终，就允许市场自由发展，那么各商铺自由出租，能看中这个流量的，要么搞住宿的，要么搞餐饮的，要么搞拍摄的……
市场，相互选择的结果，没有对与错。

2，那上海的 DNA是什么？尊重，你有没有发现，上海很少有红色标语，也没有“我爱上海”这类的口号，上海不逼每个人爱上海，足够开阔，足够尊重，你若在你老家发个抖音，说自己讨厌沂水，你想想会惹来什么暴风雨？但是，你在上海，随意吐槽，无所谓，没人在意，每个人都可以在这里尽情的做自己，没有人来绑架你。
深圳也是如此。深圳是因为太年轻，太匆忙，大家没有时间，而上海呢？是足够沉淀以后，依然足够尊重。

3，我喜欢现在的自己，当时的我，其实德不配位，在一个不属于自己的位置了，时而自负，时而自卑，内心高度撕裂，为什么我突然从上海回到了农村？就是想回到内心的安全地带，然后再一步步成长起来。
这次成长起来是伴随着自己的德智体全面发展，修行越来越好了，内心越来越有爱了，财富值也越来越高了，还有，对读者越来越尊重了，过去人家一联系咱，男的咱就想弄点钱花花，女的咱就想上人家。现在呢？更多的是尊重。

4，我觉得要跟上时代，不管怎么说，当成为一个现象时，就应该积极去体验，无论是正的还是负的，体验过了，就能分享一些心得给大家。

5，目前，国内自动驾驶做的最好的，一是华为，一是小鹏。
一说小鹏，我就想起了小时候看的《雪山飞狐》，里面有个人，偷了胡家刀法的一页，就已经成了顶级的武林高手。
不过，小鹏汽车发过声明，说没偷过特斯拉的自动驾驶代码，那个被特斯拉起诉的前员工，也跟特斯拉和解了。
何小鹏本人也是特斯拉超级粉丝。
自己有四辆特斯拉，还持有特斯拉的股票。

6，为什么我如此的小心翼翼，因为我是家族独苗，是家里的顶梁柱，我倒下了，家里的天就塌陷了。父母马上回家种地，媳妇和儿子，也瞬间跌落，而且无论父母还是媳妇儿子，当回忆起之前的生活时，都仿佛做了一场梦，是那么的不真实，
若是讲给别人听，仿佛是痴人说梦！

7，跟在211工作的大学同学聊了一会，她跟我讲，高考是最好的筛选机制，挂科的多是二本考上来的，跟我讲如何决定让谁挂科，有的同学，平时作业不做，哪怕及格了，也让挂。
什么是优秀的作业？
她的原话是：在我眼里，有一种高质量是分数高的作业，但还有一种高质量，那就是表现出自己的笨拙错误、和不完美，但是在努力的作业，这些都是高质量作业。
关于拿球，关于买单，这其实都是常识。
不拿不占，积极付出。
这是一个优秀人的基本姿态。

8，整个国家，一定是越来越科学的，越来越国际的。
有些时候，你不能光看口号。要看，趋势。
只要发现此路不通，立刻止损。
其实没有运气好坏的区别，拉平看，大家运气都差不多，真区别只是调头的快慢。

9，我的性格是酒香不怕巷子深，我一直在等一个人，应该是五年后，也可能是十年后。他突然如发现宝藏一般，发现了这么一个账号，他一定会说，卧槽，这么多年，竟然一直在默默的做一件事。
会有这么一个人吗？一定会的，就如同那个拳击跳的冰碴子，我刷到她，如同刷到了一个宝藏，接着在健身群里分享了。然后，我就发现，健身群里好多人她成了好朋友，又过了一段时间，她也关注了我。

10，一个人，最难跳出去的，不是县城，而是原生家庭。
有些人，靠一已之力刺破了原生家庭巨大的阻力。把这个家所有的劣势终结。
这类人光靠努力是远远不够的。他要克服家庭带给他的懒惰、恐慌、敏感、懦弱等一系列的心理状态。在这个过程中，他可能经历自我怀疑、抑郁、各种煎熬，但是他爬出来了。（这段话，很值得背诵）
到我们这个年龄，看初中同学、高中同学、大学同学，真正实现阶层越级的，很少。
实现越级的，要么利用了互联网杠杆，要么如有些女同学，嫁对了人。大部分人，都没有跳出原生家庭给画的圈。
