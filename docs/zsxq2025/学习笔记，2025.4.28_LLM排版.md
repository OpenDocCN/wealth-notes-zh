
# 学习笔记，2025.4.28

1，人很多行为都是习惯，暴力是习惯，怼人是习惯，安静是习惯，逃避是习惯……在习惯里，可以不用思考，是一个人的舒适区，只要交给“我以前都是这么做的，这一次也可以这么做”就行。如果不在上面吃一次大亏，很难发现习惯对自己是好是坏。

2，越是情绪不好的时候，越不该做重大决策，因为这时候做出的很多决策都是不过脑子的，大多会朝向很坏的方向发展。但大多数人的重大决策，都是在情绪主导的情况下做出的，恋爱、结婚、换工作、换城市……都是感动/愤怒了、爽了/不爽了、开心/不开心了，所以去做了什么事，从来没想过，自己真正想要的是什么。不要成为大多数。

3，很多人在找外部原因的时候，能找到很多看似合理的原因，行业不行，周围环境太差，同事不给力，小人爱找茬……唯独到自己身上的时候，能找出一大堆自己出错的合理原因。凡事先在自己身上找原因，因为你能改变的只有自己，别人再好再坏，不是你能管得着的。一个人觉得环境不好，还不逃，那不正说明他问题很大吗？

4，想学成功人士的经验，多去看那些一代是怎么奋斗怎么翻身的，少看那些二代三代怎么讲各种规则、礼仪，那都是钱多了闲着研究出来的规矩，富人过家家用的。不能只看别人的结果，要多看别人的经历对自己有没有启发。

5，老好人总觉得自己能维护好一切关系，为了不得罪人，哪怕自己利益受损也无所谓。越是这样，别人对他就越没有边界，就越会过多侵占他的个人利益。这种思路很不对，因为一段关系里，舍不得一个人的很大原因，不一定是那个人对他有多好，而是他在那个人身上投入过很多成本，不想这个成本就这么被清零，总觉得自己还有机会赚回来。别人在你身上没有投入过任何东西，怎么会把你当回事呢？

6，人，充满了不确定性，没有哪个“人”值得成为信仰，因为人一直在变化，尤其涉及到利益的时候，变化会更大，为了利益翻脸不认人的大有人在。很多时候是否要信任一个人，取决于这个人的背叛成本有多高。把房产证押给银行，银行就愿意借给他钱，他敢不还，银行有得赚，所以银行信他。上嘴唇一碰下嘴唇，就能在你那里套出来几十万，没有太多成本，他就倾向于赖账不还。

7，不要把注意力放在别人身上，新闻热点，娱乐八卦，大多数别人的事都和你没什么关系，持续去关注这种事，只会浪费精力、逐渐丧失深度思考的能力。把注意力收回到自己身上，收回到身边具体的人和事上面，多关注自己能力方位内的事，多关注能让自己变得更好的事。

8，想要有所成长，一定要逆人性做事。因为自私的基因给人性的指令，总会偏向于低能耗做事，不动脑子、不动身子，能懒则懒，省下来能量，多长点肉，不去到处瞎逛，才能让基因存活时间长一些。想要变好，不被任何人操控，就得逆着这种基因本能去做事，不能懒。

9，想改变和要改变，是两回事。很多人都想改变，但是想想就完事了，以后该怎么做还怎么做，甚至因为自己想过改变但没有实际改变，还会增加一些内耗，觉得自己什么都做不好。要改变，真正在改变的人，只是少数，他们没有太多时间去喊口号。

10，以前有人怀才不遇，是因为它可能连小县城都出不出，千里马被伯乐看到的机会很小。互联网时代，没有那么多的怀才不遇，一个人可以通过互联网向全世界展示自己的才华，如果全世界都没人欣赏自己的才华，就得好好想一想那到底是个什么东西了。
