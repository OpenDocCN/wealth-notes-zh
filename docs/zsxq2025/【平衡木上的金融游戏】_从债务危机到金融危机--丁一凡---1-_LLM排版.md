
# 【平衡木上的金融游戏】_从债务危机到金融危机 (丁一凡) (1)

## 日本媒体爆出猛料
一直有传言，美国商务部长卢特尼克和财政部长贝森特，存在严重矛盾。比如5月12日，中美在瑞士达成协议，美方
的谈判代表是贝森特，卢特尼克没有参加。
于是卢特尼克怀恨在心，让商务部下属的工业与安全局颁布
公告，全世界谁敢用华为昇腾芯片，就逮捕谁。等于给贝森
特上眼药，破坏中美谈判。
但这种传言无法实锤，直到6月6日，《日经亚洲评论》(隶属于日本官媒《日经新闻》)爆出一记猛料，实锤了两人的
巨大矛盾。

## 那么,这是怎么回事?
《日经亚洲评论》爆料，日本时间5月2日(美国时间5月
1日),美日在华盛顿举行了第二轮关税谈判。日方的代表团
团长，依然是“经济再生担当大臣”赤泽亮正。
美方的领头人，则是商务部长卢特尼克、财政部长贝森特和
贸易代表格里尔。

会谈期间，卢特尼克和贝森特、格里尔三人，竟然当着赤泽
亮正的面吵了起来。
把赤泽亮正给整不会了。
卢特尼克是极端贸易强硬派，让美国人去打螺丝，就是他的
言论。既然是强硬派，那么对日协议必须强硬。日本不仅得
承担高关税，购买美国能源和农产品，还得赴美大量投资，
升值日元，日本对美出口实行配额制等等。
总之，和《广场协议》没区别。
卢特尼克认为，如果不能再次达成《广场协议》那样的苛刻
条款，不如不签任何协议。 一份宽大的协议，容易给其他国
家释放错误的信号。
尤其是中国，可能会觉得，既然日本的体量，都能拿到一个
好待遇，那我一定能拿到更好的。
这样一来，中美谈判就难了。
卢特尼克甚至觉得，在中美达成协议前，美国不应与其他国
家达成任何协议，否则不利于中美谈判。
而贝森特相对务实，认为再次达成《广场协议》的难度相当
大，今时不同往日，不如放宽一点。
只要日本让步，买点美国农产品、能源啥的，再承担一部分
关税，愿意到美国投资设厂，咱们就给个面子。
由于日本是第一个与美国展开正式谈判的国家，谈判结果具
有示范效应。要是日本服了，拿到相对好的待遇，有利于后
续的谈判。其他国家一看，貌似美国的要求也不是很苛刻，
很可能主动与美国谈判，不再观望。那时候，美国能提要求
的地方就多了，这有利于孤立中国。
两派互不相让，贸易代表格里尔，选择支持卢特尼克。
两派分别向赤泽亮正施压，提出不同的要求，导致赤泽亮正
不知该如何是好。
其实，卢特尼克就是故意的，他和贝森特的矛盾，来源于大
选结束后。
卢特尼克从事金融业，和特朗普相识于2000年，2016年大
选期间，卢特尼克支持的是希拉里。
2019年5月，卢特尼克首次明确站队特朗普，为特朗普筹集
了500万美元政治捐款。“国会山事件”后，卢特尼克是少
有的、继续支持特朗普的金主。

2024年大选，卢特尼克帮助特朗普筹集了大量资金，是为大
功臣，被任命为过渡团队联席主席， 一般来说，联席主席会
是新内阁的核心人物。
卢特尼克自己，盯上的是财政部长一职，一开始，财政部长
呼声最高的，的确是卢特尼克。但我们知道，财政部长通常
由华尔街的人担任。
卢特尼克虽从事金融业，却不是华尔街的人。多位特朗普的
金主，表示不希望卢特尼克成为财政部长。
最终，特朗普选择了贝森特担任财长，卢特尼克则成为商务
部长。商务部长也是大员，但和财政部长相比，影响力还是
有些差距。
卢特尼克加入阵营较早，贝森特是后来的，凭什么位置在我
之上?
于是乎，卢特尼克十分不满。
后面的中美瑞士谈判，卢特尼克甚至没有参加，贸易是商务
部主管的，卢特尼克这个商务部长，却不被允许参加，风头
全让贝森特抢去了。
等于卢特尼克被边缘化了，换了你是卢特尼克，你会怎么想?
肯定得下绊子。

所以，贝森特那头，要与中国达成合作，卢特尼克就拼命出
台各种反华政策。
你别说，卢特尼克这么一闹，还真的有效果。
中美通话后，确定将举行下一轮谈判，卢特尼克出现在了代
表团名单里。
其实，卢特尼克和贝森特的矛盾，是特朗普内阁的缩影。
当初，特朗普为了击败民主党，选择了最大限度团结各方势
力，于是共和党建制派、马斯克等硅谷右翼，以及贝森特这
样的索罗斯门徒，都被纳入内阁。
一方面，可以说特朗普在政治上成熟了，懂得“把朋友搞得
多多的”。
但另一方面，内部成分过于复杂，会给执政带来麻烦。
不同派别，主张必然不同，互相存在矛盾。大选前，大家为
了打江山，还能保持团结。 一旦进入分蛋糕的节奏，矛盾势
必浮出水面。
马斯克和卢比奥吵架，马斯克和特朗普本人的骂战，都证实
了这一点。

卢特尼克和贝森特， 一个MAGA 派，一个华尔街的人，本就
谈不到一块去。
特朗普偏向谁，都不好。
偏向贝森特，会引发MAGA 不满，偏向卢特尼克，则华尔街
不满，所以，特朗普的做法，明显类似于袁绍对河北派和颍
川派的态度。
一会儿支持贝森特， 一会儿支持卢特尼克。
袁绍已经用亲身经历告诉我们， 一会听这个一会听那个，是
最糟糕的，要么你只听一个，要么都不听。
但和袁绍类似，特朗普的阵营结构决定了，他不能完全抛弃
两派中的任何一个，只能和稀泥。未来，这俩人的矛盾，会
对特朗普政府造成巨大干扰。
正如对日谈判那样，你内部都无法达成统一方案，又怎么指
望其他国家对你妥协?
当着谈判对象的面吵架，等于鼓励对手不要妥协。
就算想投降，也不知道该向谁投降，向贝森特投降?
卢特尼克不答应。
向卢特尼克投降?贝森特不答应。

再想想，特朗普政府里有类似矛盾的，只有这俩人?
当然不止。
类似的吵架，后续不会少。
内部如此矛盾重重，注定了特朗普雄心勃勃的关税目标，将
很难实现。
