
# 体制生存手册（体制内必读）

## 第一部分 体制内学生思维

### 第一节 学历崇拜与综合权衡

你觉察不到的学生思维如何在体制内毁了你。大部分人的思维和认知都是两个来源，一个是原生家庭，另外一个是学校的教育。原生家庭每个人都不一样，如果你的家庭条件好一点，父母也比较有文化，可能对孩子的教育还比较到位，如果父母没有什么文化，基本上生下来就是只要你能吃饱穿暖就行了，谈不上什么教育，甚至父母的言谈举止一些不太正确的价值观还会影响到你，所以很多人尤其是我们那一代人，父母都是经历过文革读书的时候，又遇到上山下乡，基本上就没有读多少书，所以这样一来，我们很多人的认知，很大程度上还是依赖学校的教育，所以大家想一想，如果你22岁大学毕业的话。六年小学，三年初中三年，高中四年，大学22年当中，你有16年的时间都是在学校里面度过的，如果你还读了研究生，那么你在学校里面呆的时间更长，这样的话，就会产生一个问题，就是这么长时间的学生身份，你的思维方式很可能会被潜移默化成的学生思维，学生时代十五六年的教育和形成的思维方式，对我们的影响实在是太大了，所以进入社会以后，我们带着学生时代形成的这种惯性思维去面对这个复杂的社会，包括我们进入体制内，我们经常会碰得头破血流，这个社会的运行规则对我们内心的价值观发起了剧烈的冲击，所以我们进入社会以后，进入职场以后，我们的任务不是把我们学到的东西应用到职场中去，或者是怎么样把学生时代形成的思维方式给他破除掉，形成一套新的私营社会规则的思维方式，我一直以来有一个观点，人与人之间最大的区别是什么不是高矮胖瘦，不是颜值高低，也不是财富的多与少，而在于人的思维，人和人之间最大的不同在于思维的不同行为只是一个人的外在表现，而思维才是一个人的真正决定性的因素，也就是说，思维决定了我们的行为，很多人在体制内过得很纠结，很郁闷，遇到很多问题，其实从根子上来找原因都是学生思维在作怪，你要实现逆袭，首先必须升级你的思维，当然，打破学生思维的过程，是一个非常痛苦的过程。而且会是一个相当长时间的过程，有的人要花35年的时间，有的人甚至要花一辈子的时间，所以，今天我就跟大家聊一聊学生思维都有哪些表现有哪些危害要如何来破除这些学生思维那么首先来讲，第一个学历崇拜与综合权衡，什么叫做学历崇拜就是在我们很多人的
