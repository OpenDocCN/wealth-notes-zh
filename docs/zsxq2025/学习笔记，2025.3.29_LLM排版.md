
# 学习笔记，2025.3.29

1. 焦虑，是因为看了太多、想了太多离自己太远的东西，下一步迈不出去、没处落脚了，心里就慌。停下来，多看看脚底下的路，多去关注自己看得见摸得着的东西，把脚落下去，行动起来，就不焦虑了。

2. 不要期待别人感谢自己，也不要反复提及自己帮过别人。自己认为值得做的事就去做，做事过程的经历，就是对自己最大的回报，如果把关注点放在别人的感谢上，就容易动作变形。把事做好，是自己的课题，感谢与否，是对方的课题。

3. 想了解一个人，别看他说什么，要看他读什么书、做什么事、有什么习惯。说出来的东西，上嘴唇一碰下嘴唇，没什么成本，但是习惯的养成，却是需要长期投入时间精力的，包括坏习惯，都是用时间堆出来的，轻易无法改变。

4. 求人不如求己，遇到什么问题，先问问自己，有没有努力去找解决方案，有没有搜索了解过别人是怎么解决类似问题的，这个探索的过程，也是对个人能力的一种锻炼。遇到事就想找人帮自己解决，就像做题先翻答案，抄一百遍答案，也还是不会自己解题。

5. 不要预设别人是一个“好人”，这样，你会对他期待太高，然后无法接受他的不完美。对自己也是，不要在外给自己立太伟光正的人设，会被别人捧杀，做很多自己不愿意做的事。当个俗人，贪财好色，挺好。

6. 人应该独立思考，但是经常把独立思考挂在嘴边的，要么是准备忽悠人花钱去跟他思考，要么是被人忽悠了把这四个词当成了武林秘籍。就像，人要遵纪守法，但把遵纪守法挂在嘴边的，大概率做了很多擦边的事。

7. 物以类聚，人以群分。如果觉得自己所处的环境，周围的人太差了，和自己不匹配，就得停下来想想，是不是高估了自己，自己是不是能力就配到这个环境。如果不是自己太差，你还不逃，等着过年吗？

8. 人不分亲疏远近，是一种理想，事实是，人就是分亲疏远近的，你看到一个身边的人遇到麻烦和看到一个其他国家陌生人遇到麻烦，感受是不一样的。对所有人都去无差别共情，悲天悯人，不适合大多数普通人。普通人的精力和情绪阈值是极其有限的，要多用在自己身上和关系友好的人身上，不要浪费时间去和不喜欢的人抬杠。

9. 很多时候，能够真正对他人“同情”，是因为自己也有过类似的经历，能在一定程度上感同身受。无差别的对别人同情，去共情，尤其是喜欢同情别人的弱小，无法共情别人的喜悦，是一种喜欢站在上位视角的优越感，是一种“还好我没有遇到那些破事”的侥幸。

10. 当别人遇到问题的时候，我们直接给他答案会更好吗？通常不会。他少了主动探索的过程，下次遇到问题，还是不知道怎么办。而且，对很多成年人来说，不请自来的答案，不是指点，是指指点点。
