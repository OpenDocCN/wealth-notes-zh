
# 冷眼局中人6月必读付费文修改之处

回头看估计还得回到这个区间带里。另外周四早上6点左右在陆家嘴做了一个梦，梦见了周金涛总路演后一起吃饭，我赶紧借机问他现在周期品有什么机会吗？他轻轻地说了两个字。我听完一个激灵惊醒了，然后赶紧打开电脑看图，一看吓了我一跳。发现这个品种(是一个板块)每隔4年的7月都会大涨，上一次是21年7月。于是在周四路演和机构们说了这个事，并加了一张图，起名叫周期一梦。现在公众号有个好处，文章可以修改图和120个字了。我把修改的内容放到了6月必读最后，大家可以自己去看。
