
# 金融三剑客：中美博弈+资本收割模型（四十一）

## 一、关键

那现在的AI能不能帮你交作业？能。但是交上去的作业，到底能不能过关，那就不在于它，而在于你。你在人家导师（董事会）眼里重要，你怎么都能过关，你在人家眼里不重要，你怎么也过不了关。那你因为是那篇论文在起作用吗？不，是你这个人，在导师眼里的地位，在圈子里的地位和影响力。只是一个加强版的搜索引擎（继续缩短信息检索和加工的时间），它能帮你更快的敷衍平日里那些工作报告。但是，你到底能不能过关，取决于在老板（董事会层面）眼里，你这个人，到底有没有分量和话语权（解决问题的能力）。

所以看到今天很多年轻人，以为AI是用来寻求正确答案的，剑客就苦笑。你想想看，20多年前，马云那波人，是利用互联网去寻找了一个什么所谓正确答案而崛起的吗？大错特错啊。

## 市场

如果你站在这个角度看，未来是不是非常有前景？当然是，而且一直是。有些读者总担忧AI会取代白领，机器人会取代蓝领，这就是操心的步骤出了问题。就业问题的本质是什么？是消费不足，是资本市场融资困难（没钱烧，哪里来的高薪岗位）。

没有再没有AI技术的突破，你让富老头们凭什么拿钱出来给你穷屌丝创业？如果没有机器人技术的突破，你让那些拥有81%以上存款的2%的人口买什么东东？

so，如果咱们把消费市场分成两部分，第一部分人是什么画像？他有消费欲望，当下提供的产品里面就有他想要的，但他囊中羞涩。这一部分，就叫做正在修复的居民部门资产负债表（类似日本失去的三十年）。他也许家里有很多贷款要还，也许工作不好找，总之，入不敷出（没能力消费）。

如果你要盯着这部分市场做生意，那是你自己能决定的么？不是。除非有居民，企业部门之外的三张表出动，说它愿意扩张，把居民和企业部门的担子接过来。用土话讲就是按人头发钱。

说白了，全世界的那些发达国家都是这么解决此类问题的，因为理论上第三张表累积多少债，也就是记个账而已。你去看隔壁的小日子，垒得那么高，有什么问题么？不还在继续累积么？

如果你还盯着这部分市场做生意，那你只能等（操盘手动作）。你的未来不是你能决定的，因为你的客人们，他就没钱啊。

那我们跳出这个画像，你去看消费市场里的另一部分人，是什么样的？他们是没有欲望了么？不，他们 是 揣 着 钱（超过81%的存款），买不到自己想要的东西。

我们很多做企业的读者，在过去的许多年里，都喜欢讲消费升级。但实际上，有多少人，真的去升级了他们的产品本身？很少很少。

因为我们是个非常大的市场，以至于口碑效应在高速发展时期的作用，是不那么明显的。什么意思呢？就是说他完全可以以次充好，就算这个客人不复购，他挨个晃点，也还有14亿等着他呢。

于是我们所谓的消费升级是什么？是同一批次的茶叶，给你换个包装，打着大师的旗号，就成了高端货（小罐茶）。话说他忽悠了那些有钱人一次，他们有钱人也就不愿意被忽悠第二次了。

不过，这种玩法在前些年居民部门还有加杠杆潜力的年代，总架不住有些想要效仿有钱人生活方式的，愿意被他接着忽悠（他就这样连续收了六次的智商税）。于是他那种以次充好的消费升级就可以存在。

但当现在居民部门已经没有加杠杆空间的时候，接盘侠怎样？消失了。

以前讲过的：中产（伪概念）=接盘侠（加杠杆接）

so，他那种套路也就玩不下去了。可是，套路玩不下去，真诚也玩不下去么？非也，你去胖东来看看，人山人海，连那些帮着从胖东来购物的二道贩子们，都有人月入六位数的。你再去山姆看看，人山人海。

## 四、未来

所以任何时候你去看，市场一定是分化的。80年代的时候，南下打工很挣钱，你待在原单位，一个月就几十块。90年代的时候，做外贸，做服装，很挣钱，与此同时，很多此前的企业，倒闭了，很多人下岗了。

2000年之后，做房地产的，很挣钱，再后来做互联网的很挣钱，与此同时，女装街开不下去。

说白了，现实中你看到的从来都是一半是海水，一半是火焰。

回到10年前，浙商会上，有大佬公开讲，不要总说生意不好做，是你的生意不好做，有人生意好做得很。他说的对不对？他说的非常对，尽管数年之后，他自己的生意也不好做了，但是，这并不掩盖彼时彼刻，他说的那句话的价值。

未来什么样？未来对不同的人，是不一样的。对有些人满面春风，对有些人青面獠牙，这就是未来。任何时候的未来，它都这样，它（机会）都是一个看人下菜碟的模样。进群加微信 你会发现，未来始终都有两张脸，雷霆雨露，皆是未来。

所以你想讨论宏观，那也是分开讨论。你是要讨论海水呢？还是要讨论火焰呢？

你以为今天才诞生的润学？不，这个词儿从90年代就开始了。年年都有人说不行了不行了只能润，然后呢？你不行，不等于别人不行，这片土地上总有人行。总有人跳出来打脸，行给你看。

比如最近刚召开的大会，2018年11月和2025年2月，上峰召开了两次民营企业座谈会，但参与的企业家各有不同（有得一直行，有得就换人行了）。这就叫江山代有人才出，各领风骚三五年，6年时间，两份名单，每个被邀请的人选，都是对当下和风口的不断定义。

什么意思？就好比饺子导演的妈。孩子你想搞什么，你就去搞，我只负责给你提供后勤。也就是说360度自由生长，你干嘛，他不限制你的。

那么另一种家长是什么？是限定方向式家长。孩子你要好好读书，你选科一定要这么选，你选专业一定要这么选，你选学校一定要这么选，你将来一定要回到哪个城市，你一定要去面试什么工作，你一定要娶谁谁谁，你一定要买房子买在啥地方……
