
# 孤独的永定门公园

周末，趁着阳光明媚，兜兜转转，再次去了一趟永定门公园，享受一波快乐的休闲时光。永定门公园建成于 2004年 9月，整个公园占地约28.5公顷，绿化面积达到13.8 万平方米。其中，永定门城楼是北京中轴线的“南起点”，其复建工程开始于 2004 年，2009 年落成。城楼建成标志着北京中轴线由明北京内城延伸至外城 3公里，最终形成完整的 7.8公里。然而，当安邦智库（ANBOUND）研究人员走在御道上，向南遥看，在帝都深蓝的天空下，只觉得永定门公园尤其是永定门城楼透着莫名的孤独，没有往昔的高大雄伟与庄严大气。也许这是时代的忧伤，即使正在弥补，却一去不复返，唯有追忆之。

经常去的人肯定知道，虽然永定门公园宣称是开放式的管理，不需要预约，可以自由进入，但是实际上不太好实现，反而显得有些封闭。此话怎讲呢？实际上，永定门公园是被两条很宽的城市道路和周边片区几乎完全隔开。平时，这两条道路上是车来车往，络绎不绝，行人进去只能去固定的门，还得等待时机。因此，永定门公园就成了一个“现实版”的孤岛，被交通路线“包围”，导致可达性不好。穿梭的汽车像是飞翔的箭镝，恍如一段段坚硬冷峻的历史不断闪回。如果第一次去，游客是不太容易找到公园入口的。尤其是从南门进入，游客还要穿过迷迷糊糊的地下隧道。

另外，除了永定门城楼外，公园整体上似乎也没有什么特色，只是一个现代化城区包围的绿地和广场而已。有报道说，晚上，永定门公园在灯光亮后可以感受到那种现代化的璀璨之美，但是历史的光环却在沉默中黯淡，没有太多游客清楚，也不会关心。目前，也不知道是何原因，永定门城楼也是不开放的，仿佛一个无言的摆设，也就在很大程度上断绝了游客站在城楼上俯瞰北京城的美好期待。因此，午后的公园里也没有多少游客，稀稀落落。只见几个人在走动，几个人在打网球，还有几个人在木凳上闲坐。实际上，2023年初秋第一次去永定门公园，同样是深蓝的天空，只是人更是少得可怜。没多少人去的公园，仿佛被遗弃的孤儿。

作为永定门公园的主体，永定门城楼是过去北京老城外城的正南门，也是外城城门中体量最大、建筑等级最高的一座。如今，复建的永定门城楼“修旧如旧”的匠心随处可见。它是根据 1937 年北平市文物整理委员会对永定门城楼的实测图、1957年拆除时绘制的建筑结构图以及故宫博物院保留的永定门建成以来各种文字、图片资料进行复建的。新建成的永定门在原址采用原材料、原形制、原结构、原工艺进行；城楼彩绘采用中国传统的建筑中常见的装饰手法—雅五墨旋子彩画；使用的城砖是原来嘉靖年间制造的老城砖。

不过，这些复制的仿品只有当年形貌的躯壳，其历史与文化的内涵恐怕早已荡然无存，不免让人扼腕叹息。毕竟，有的瑰宝一旦毁掉，就很难再恢复。更重要的是，永定门城楼与周围的环境格格不入，未进行有效的融合。如果没有周围环境的契合，建筑就没有了灵魂。由于周边环境已缺失北京旧城原有肌理与尺度，永定门城楼在南侧宽阔的二环路与北侧空旷的永定门公园的大尺度环境夹击下，也失去北京南城门原有的宏伟尺度，看起来比原来的小，或许是我们早已习惯了高楼大厦。有学者也表示，环境大了，你看环境中的东西就小了。环境小了，你看环境中的东西就大了。

进一步回顾历史，或许大家就能感受到，永定门公园更显孤独，是残缺不全的。当时为了防御南倭北虏，特别是北方游牧民族的威胁，巩固大明江山，明世宗嘉靖开始在护城河北边修建外城城墙，建永定门。当时，因连年战乱，国库空虚，财力有限，只修建了永定门一个城门楼。瓮城是在嘉靖四十三年（1564 年）补建的。箭楼是在清乾隆十五年（1750 年）增建，并又重修了瓮城。至此，永定门城楼历时 197年才算完全建成。1900年庚子事变，八国联军攻进北京，先将永定门城墙西面拆了一段，修了一条铁路。此后，永定门因为年久失修，逐渐荒废。1950年，为打通北京环城铁路，永定门瓮城城墙被拆除。1957年，城楼和箭楼又因妨碍交通和已是危楼的理由，相继殒命于锹镐之下。这个以护卫京城安危为使命，顶天立地的威猛武士，在现代文明升帐的年代，以极不体面的理由被清除出它所深爱的世界。2004 年，永定门重建，但是只是建设了永定门，而原永定门箭楼和瓮城因其他原因而缺席，早已拆除的城墙更是不可能重建。

此外，永定门城楼南面是东西横贯的永定门滨河路，下面是下沉式现代化立交桥，虽与旧时永定桥同名，却已是大相径庭。20年前恢复城楼，值得点赞，但可能更多是用于申遗的功利性目的，缺乏整体性、历史性、长远性的思考。这意味，只有城楼，没有了瓮城和箭楼的衬托，也没有城墙，永定门孤零零地矗立在中轴线的南端，显得有些单薄，只能感受历史的沉重和沧桑。

如今，永定门公园作为一个现代化公园，蕴含了恢复重建的思考与心力。然而，历史建筑与文物展示着一个地方、一个民族、一个国家的历史文化的底蕴与厚重记忆。一旦遭到巨大的破坏，即使恢复重建，其内涵与价值可能就一去不复返了。因此，当前城市更新与改造一定要慎重，要具有战略性、前瞻性与长远性的眼光，绝非按照所谓现代化的眼光一拆了事。否则，悔之莫及，徒增无奈。

最终分析结论（Final Analysis Conclusion）：
如今，永定门公园作为一个现代化公园，蕴含了恢复重建的思考与心力。然而，永定门公园是孤独的，看似开放式管理，实则并非如此；“修旧如旧”的匠心随处可见，但难以展现南方大门的庄严大气；只有城楼，没有了瓮城和箭楼的衬托，也没有城墙，显得有些单薄。因此，城市更新一定要慎重，避免一拆了事。

【形势要点：国内推动中小企业合规管理与国际接轨】
3 月 13 日，工业和信息化部等 15 部门办公厅发布《关于促进中小企业提升合规意识加强合规管理的指导意见》。《意见》明确合规管理重点领域，具体包括劳动用工合规、财税合规、产品和服务质量合规、安全生产合规、节能环保合规、知识产权合规、网络和数据安全合规、公司治理合规、国际化经营合规、供应链合规等。《意见》还提到，将合规管理纳入中小企业经营管理领军人才和专精特新“小巨人”、专精特新中小企业培训内容，面向中小企业经营管理人员普及合规知识，帮助中小企业了解合规、重视合规、精通合规，推动合规管理成为企业自觉行为。鼓励具备较好企业基础、工作条件比较成熟的地区，研究出台促进中小企业合规管理的相关政策措施，研究构建合规风险预警机制、重大风险快速响应机制，依法依规探索在相关评比、认定、考核工作中引入合规方面指标，引导、帮助优质中小企业提升合规管理水平。需要指出的是，该《意见》的提出，体现了国家从政策层面大力支持中小企业发展。在经济全球化的大背景下，国际市场竞争日益激烈，各国之间的经济联系和依存度不断加强。中小企业作为经济发展的重要力量，也越来越多地参与到国际经济合作与竞争中。推动其合规管理与国际通行规则接轨，有助于中小企业更好地融入全球产业链和供应链，突破贸易壁垒，降低国际经营风险，在更广阔的市场中实现资源优化配置和可持续发展。

【形势要点：量价分离阶段需确定通胀目标下限】
长江证券首席经济学家伍戈日前表示，绝大部分行业现在正在经历非常深刻的以价换量的过程，同样在宏观层面也看到了量价的分化。他指出，去年四季度我们看到经济有一个脉冲式的反弹。但如果我们看价格，特别是以 GDP 平减指数为代表的价格，去年四季度的 GDP 平减指数与价格仍在持续负增长区间，甚至还在下行。他认为，中国作为一个制造业大国，生产能力如此之强，那么我们确实可能就处在一个量价分离的重要阶段。这种阶段对于我们宏观政策以及宏观政策的引领，是非常关键的。这也是未来要考虑的所谓货币政策、财政政策、地产政策，回避不了的问题。这是一个目标的问题，而不是工具和手段的问题。伍戈以日本遇到同样的困境得出结论，如果 CPI 只是零点几的增速，既不足以改变老百姓消费者的预期，也没有足够的力量改变投资者的信心。后来日本开始像很多成熟国家一样，把价格目标定在 2%左右。他指出，这也是现在包括央行在内的各级政策制定的一个目标。这个目标并不是要实现的上限，而是下限。也就是说，如果没有实现这个目标，所有的宽松政策都不会止步。对于中国而言，今年两会的价格目标是下限还是上限？若需实现价格真正转正，政策需要超常规。

【形势要点：持久的价格战使汽车经销商不再靠卖车挣钱】
近日，中国汽车流通协会发布的《2024年全国汽车经销商生存状况调查报告》。该报告显示，2024年，48.2%的经销商超额完成年度目标，2023年度该占比不足 30%，豪华品牌目标完成率略好于合资品牌和自主品牌。报告还显示，2024 年度汽车经销商盈利比例为 39.3%，持平比例为 19%。经销商亏损面较2023年的 43.5%收窄至41.7%，较 2024年上半年的50.8%收窄了 9.1 个百分点，但是超过 40%的亏损比例依然偏高。就经销商毛利的具体构成来看，在经销商的毛利构成中，新车、售后和金融保险的毛利贡献分别为-17.7%、61.6%和38.0%。就新车销售的毛利贡献率来看，去年全年相较于去年上半年已经有明显的改善，但是却依然处于负数值区间，可见去年车企的多轮价格战已经显著伤害了经销商的销售收入，经销商被迫不再靠卖车赚钱。值得注意的是，就新能源汽车独立品牌经销商（代理商）而言，新车销售表现出了相对较好的盈利能力。此次报告样本数据中，新能源独立品牌的经销商或者代理商，占据了 16.8%的比例。新能源车经销商也主要靠着售后在赚钱。
