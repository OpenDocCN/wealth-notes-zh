
# 鉴茶财经院250312:巩固

两会结束后，村里落实精神核心是两句话。一是全力巩固市场回稳向好势头。二是在支持科技创新和新质生产力发展上持续加力。

巩固，就说明对目前态势还是比较满意的，稳中有进就行，下面有底，上面随缘。下面的底，3233一线。加力，就说明科技仍要当主线，市场怎么走不知道，但上峰这样想。

尾盘，AIDC出现了一波大跳水，既有港股波及的原因，也有分歧加大，有些主力资金的原因。从我们说算力租赁是错杀，很多标的的振幅高达60%甚至100%，短期大赚一笔总要落地为安。最关键的是，业绩披露期即将来临，有些纯属蹭概念的，怕被打回原形提前跑路。这时就要看，谁是真有英伟达卡的人了，大厂们可不傻，没东西谁租你的。国内的很多数据中心，都是IDC而不是AIDC。

## 短线选手有什么应对策略?

1. 辨别今天放量下跌的，尤其换手率成交额远超以往的。比如你以前，换手5%，成交3亿，今天12%，成交9亿。
2. 进行减仓，都赚多少了。
