
# 泪痕春雨漫评靖难之役

朱允就藩王驾驭缺乏自信。朱元璋曾问他如何处理不听话的藩王，朱允回答会进行思想工作，削减待遇，甚至废为庶人并监禁。朱元璋对此满意，但朱允内心并非如此。他实际上认为藩王削亦反，不削亦反，分封藩王是开历史倒车。

朱元璋作为武夫，因运气好当上皇帝，不懂政治，留下祸害。朱允的这些想法虽然大不敬，但反映了他的真实想法。从朱允的后续行为和后世读者对历史的评论可以看出，他们都认为分封藩王是错误的。

朱允认为朱元璋构建的权力格局漏洞百出，他想彻底打破并重新构建。朱允的这种自信和勇气令人奇怪，他可能认为朱元璋留下的摊子太乱，而不是他自己的问题。

齐泰和黄子澄是朱元璋的亲信重臣，但他们的失败与朱允的失败无关。朱允认为自己的失败不是因为摊子太乱，而是因为自己比朱元璋更有政治头脑。

历史书通常只关心证明政权的合法性、合理性和皇权的神圣性，而不是历史的真相。因此，人们熟读历史书后，往往无所不能，但往往缺乏历史经验。

朱允认为朱元璋缺乏政治常识，不懂吸取历史教训。如果他是朱元璋，他不会这样做。朱允的自信和勇气令人佩服。

朱允分封藩王的主要原因是帝国政治重心在长江一线，军事重心在长城一线，两者相隔数千里，对帝国非常危险。朱允认为，把长城一线的军事权力交给自己的兄弟、叔父，皇帝都不放心，交给外人就更放心不了。

朱允认为，长城一线的藩王有十三个，每个藩王权力有限，只是军区皇族代表，军队实际管理者是都指挥使，藩王只能控制藩王府内的军队。面对这样的藩王，皇帝同样会如临大敌。

人们常常有错觉，认为如果不分封藩王，就没有靖难之役。实际上，长城一线的藩王互相制衡，藩王、地方行政主管、军事主管、监察主管四股力量互相制衡，小皇帝都没有安全感，最终导致藩王反杀帝国都城。

真正威胁皇权的不是皇族、外戚，而是皇位作为天下利益汇集的中心，在特定历史背景下，不管皇帝如何布局，它都是危机伺伺的。

皇族、外戚对皇权的威胁虽然巨大，但与其他力量相比，却是最安全的。因此，历代皇帝都重用皇族、外戚。直到五代之后，随着小农经济的发展，皇帝不再需要皇族、外戚巩固皇权，但在特定历史背景下，仍然需要这样做。

朱允认为，周王、代王、齐王、岷王被废为庶人并关进监狱后，小皇帝一时非常被动。湘王则领着全家自杀，湘王的这种操作让小皇帝承担了太多的道德风险和舆论压力。

弱者被欺负时，最绝望的反击就是摆烂与自虐。周王、齐王、代王、岷王和湘王都是这样玩的。他们用自己的可怜与无助，悲惨与不幸，把小皇帝推到道德与舆论的风口浪尖之上。

燕王也开始摆烂与自虐，他知道小皇帝不敢扣留他的儿子当人质，因为那样会承担无与伦比的道德风险和舆论压力。小皇帝知道，自己绝不能这样做，因为那样会留下更多藩王等着解决。
