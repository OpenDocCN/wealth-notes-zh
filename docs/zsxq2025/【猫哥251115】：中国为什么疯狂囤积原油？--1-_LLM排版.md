
# 中国为什么疯狂囤积原油？

今年在进口原油方面，我们的动作把老外惊呆了，日均进口量和沙特这个油老虎的日产差不多。可以说，我们凭自己之力，把国际原油价格撑在60美元以上。今年前三季度，我国进口原油规模约4.23亿吨，同比2024年前三季度增长2.7%。单从同比来看，增长也就不到3个点，似乎谈不上用“疯狂”二字来形容，但要知道的是今年宏观经济并没有强力反弹。要知道，2025年前三季度另一个大宗能源 - 天然气，同比还下降约11%。说明我们目前整个经济对能源的需求是下降的。

要知道，2024年末新能源汽车保有量约3100万辆，而当下估计上4000万辆。新增1000万辆就代表每年大致节约汽油2500万吨，对进口的原油的总体规模影响减5%。也就是说，在需求端萎缩的前提下，进口端却是在扎扎实实得买买买。

大口吃进原油，除了自用以外就是储备。今年我们平均每天的囤油量高达120万-140万桶，导致今年我们的新增原油储量占全球新增储量的90%，这个数据是不是相当炸裂。根据估算，我国目前存的油大致在12亿-13亿桶左右，如果加上每年自产的2亿吨原油，极端情况下，就算没得进口也能保障180天需要。这个储备规模虽然已经远超国际能源署建议的90天安全线，甚至比日本的150天储备还多。但对于我们而言，也许还不够，看阵仗是冲着200天甚至以上去了。目前全国各地还在新建多个大型储油基地，也就意味着场地、基础设施、管理维护等方方面面的工作正在跟上。所以，囤油还不是一个短期行为，而是一个长期战略选择。

那么问题来了，我们疯狂囤油的背后，究竟在打怎样的算盘呢？

很多人解读是因为政策驱动导致。因为新的能源法在1月1日起就宣布实施，其中就明确要求建立健全高效协同的能源储备体系，企业必须承担储备义务。也就是说无论是国企、地方还是民企，都要储备，而且不是指导建议，是刚性要求。

但这项政策背后逻辑、出发点又是什么？我们梳理一下能源法的几个重要时间节点。2024年4月对草案进行初审。2024年9月对草案进行二审、11月通过。其实不难发现，我们的准备工作早在去年4月以前就完成了。而真正把它端出来，特别是二审、通过的时间点，几乎是和美联储第一次降息和美国大选同步的。也就是说，要理解囤油背后的逻辑，需要放在中美博弈这个大框架内进行解读。

首先，是增加对全球原油价格的议价能力。2006年，我们的原油储备仅有7天保障能力，所以才导致面对08年147美元每桶的高油价束手无策。那么未来油价会不会涨呢？个人认为涨是大概率事件，为啥？我们都知道老美的标准动作，就是通过美元潮汐收割全球。先加息制造全球美元流动性危机，引导资金回流拉爆几个经济体。然后降息印钞，释放流动性抄底各国资产，导致美元泛滥大宗涨价。以往这种操作无往而不胜，但现在这招变得越来越不灵了，核心原因是我们在前面顶住了，导致老美到目前为止还没拉爆任何一个经济体，所以着急的川普现在打算明抢，威胁南美的委内瑞拉、非洲的尼日利亚就是基于如此。别看美联储降息扭扭咧咧，但大趋势不会变。既然只是时间问题，那何不在QE高潮来临之前，手上多几口油田呢？要知道川普背后的金主是传统石油公司，这些人无时无刻不想通过涨价发财。

反观我们，虽然是原油全球最大的进口大国，1年7亿吨消费量，将近5亿吨来自进口，但没得定价权。一旦我们成为全球最大的原油储备国，那就有足够的底气应对未来可能出现的涨价，甚至不排除届时我们会释放储备打压国际油价。说实话，当你手上掌握20亿吨甚至更多的原油储备，能保障自给自足200天以上，还是能解决很多事。

其次，脱虚就实，逐步推进去美元化战略。这是基于金融安全层面的战略考虑。我们一方面手握3万亿巨额外汇储备，其中美元资产占比接近60%，另一方面又通过巨额顺差赚取美元，这么多美元资产在手，总得找一个安全的蓄水池吧。以前我们是购买美债，但现在呢？美国联邦债务已经突破38万亿，虽然短期崩盘的可能性几乎为0，但风险却是越来越大，连西方自己的信用评级公司都提示预警。所以别说购买美债了，我们是不断的在减持美债。那么什么才是可靠安全的资产呢？答案显而易见，就是涉及黄金、原油、矿产等硬通货。除了今年大规模囤积原油外，我们已经连续12个月增持黄金。

当然我们也并不是一味寻求全面推进去美元化战略。因为哪怕这个体系现在千疮百孔，但毕竟我们还是这个体系除老美外，全球第二大受益者，不到万不得已，不会率先掀桌子。当下，我们要做的事，还是以优化外汇资产结构为主动，防范金融风险。当你手上有足够多的硬通货，才能有效应对国际货币体系未来的不确定性。

最近我们在香港发行40亿美元，认购倍数超过30倍，比老美自己发行美债的市场认可度都高得多，就很能说明问题。国家信用的背后是综合实力的体现，这其中也包括你手上掌握多少的硬通货。

其三，是备战需要。梧桐一直是我们选项之一，虽然我们不会轻易开启，但如果民进党要兴风作浪，肆意践踏红线，那就不得不为之。这里面就会涉及2个考虑：一方面是美西方以此为由，联合卡我们能源供应链，比如搞能源禁运，封锁马六甲海峡、贸易制裁、禁止使用结算系统等等。现在它们玩硬的不敢，但使用这些软手段，如果没得充足的准备，也会让我们经济很难受。也许有人会问，梧桐哪里需要200天，也许20天甚至2天就差不多了，这里就涉及另一个考虑：日本。高市早苗在当选日本首相后，连续搞出涉台不当动作和不当言论，已经在国内舆论场炸锅了。但可能你不清楚，它是第一个发表涉台言论的在任日本首相，而且日本目前国内支持率根据日本新闻网的11月最新民调，已经超过80%，居然还在提高。这是一个非常危险的信号，说明日本几乎正在全面右翼化。虽然目前就军事能力而言，10个日本绑一堆都未必是我们的对手，但不要忘了，日本这个国家是很疯狂的，以小博大的事对他们来说驾轻就熟。中日甲午战争、日俄战争、甚至太平洋对美国，哪个不是以小博大。届时只要老美松绑，日本大概率会第一个冲出来。

所以，增加能源议价能力、推进去美元化战略、战备需要以上三点，就是我们出台能源法，囤积原油的核心逻辑。

不谋万世者，不足谋一时。不谋全局者，不足谋一域。只能说大国背后有高人，为我们这艘大船，穿越惊涛骇浪保驾护航。
