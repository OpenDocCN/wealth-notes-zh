
# 老年人要的不是“占便宜”，而是“信得过”

## 老年人消费的三个普遍误解

有人可能会觉得做老年人的生意，就是送米、送油、送鸡蛋，嘴甜点，生意不就来了吗？但事情真是这样吗？这就要说到，关于老年人消费的三个普遍误解。

### 第一个误解：老年人爱占小便宜

从真实情况看，中国的老年人其实是最有钱，也最有闲的消费群体。靠小便宜吸引顾客的老板，最后也经常只能赚到小便宜。比如有的超市，为了吸引客流，把青菜价格降得很低，原来5毛，现在卖3毛，结果是1/8。

### 第二个误解：老年人无关紧要，年轻人才是核心客户

不好意思，在很多线下的门店里，老年人才是最关键的客群。根据黄碧云老师的调研，银发族已经占了超市日常客流的70%以上。每个老年人平均一周要去三次。还有药店、水果店、生鲜店，等等，都必须认清这个现实，银发族是最重要的客群。

### 第三个误解：老年人难伺候，很挑剔

事实上，挑剔只是结果，真正的原因是，针对银发族的供给不足。明明有那么多已经退休，或者即将退休的新老人，但商场里的银发产品依然很少。还有文旅服务、智能家居、手环之类的健康硬件等等，老年人都是最重要的客群之一。

## 如何做好银发族的生意

黄碧云老师调研了几类实践案例，接下来咱们就一起看看。

### 第一类：超市

怎么吸引银发族？关键在于四句话：周中处关系，周末做生意。促销没有用，能玩最要紧。

**周中处关系，说的是周中的时候顾客本来就少，你不要想着做什么活动，搞点什么优惠就能来一大堆人。不要心急，重点不是拉新客，而是拿出细水长流的心态和客户处好关系。**

**怎么才能和叔叔阿姨们处好关系呢？这就需要设身处地去换位思考了。你看，多数老年人每天都有个固定的日程，就是接幼儿园或者小学低年级的孩子放学，然后再带孩子去外面溜达一会儿。只要能把小孩子吸引来，哄开心，爷爷奶奶们怎么会不满意？**

超市吸引小孩的办法可太多了。比如，可以让孩子们体验采摘，把整棵的蓝莓树搬到店里来，既能采摘又能吃。再比如，还可以把门店变成一个农业知识小基地。盒马超市今年把桑叶变成蔬菜来销售，店里同时就做了蚕宝宝的科普展示，同时还卖蚕宝宝加桑叶的套餐。孩子们既可以观赏养蚕，又能涨点知识。再比如，还可以把应季的鲜花搬进门店，做成小花田。在孩子们玩的同时，爷爷奶奶再拿手机拍个视频发到家庭群里，全家人对这个超市好感度都会提升。

在春节前后，超市里还可以举办财神节，工作人员扮成财神爷给小朋友送金币送小零食。家里有娃的人肯定清楚，这套设计对小孩子的诱惑力有多大。

老年人往往掌握着全家一日三餐的财政大权，平时能把他们吸引到店，就算赚不了大钱，但至少日常的基础流水能保证。更重要的是，平时把他们服务好，周末他们就会带着全家一起来。一旦爸爸妈妈也来参与购物，超市的流水不就上去了吗？

### 第二类：线下实体药店

医药行业属于典型的信息不对称行业，药店最宝贵的资产就是用户信任。而且信任这件事很容易走极端，一旦客户信任你，就会死心塌地追随你。反过来，一旦不信任，那么你给再多的优惠都拉不回来。

要想建立信任，关键在于，抓住店里的慢变量。也就是，那些长期存在，并且需要用药物来解决的问题。比如高血压之类的慢性病。

连锁药店健之佳在全国有3000家门店。他们从2011年开始就提供针对慢性病的服务。根据健之佳的观察，要想获得叔叔阿姨们的信任，至少需要做到这么三件事。

**首先，药剂师必须要会聊天，懂得倾听老一辈的念叨。像云贵川地区，很多年轻人在外地务工，只剩老人自己在家。而因为慢性病的缘故，老人们活动范围也受限，社交圈很窄，身边也没什么能说话的人。因此，多陪他们说说话，聊聊天，这是建立信任的关键。**

**其次，初步建立信任之后，药剂师要提供足够细致的服务。包括替老人建立健康档案，并且定期回访。看看人家有没有准时吃药？血压有没有改善？**

**最后，也要给到老人们实惠。比如健之佳的一些门店，一般会在周四举办慢性病科普活动。但是，会员日优惠是放在周五的。为什么要隔一天？这是为了给老人们一个时间，去货比三家，确定价格合适再来买。假如在科普日当天搞活动，老人没时间货比三家，他可能就不会买，或者万一买贵了，就会对这家药店失去信任。**

老百姓大药房吸引顾客的方法是，从低频向高频拓展。毕竟，买药是个相对低频的行为，因此，他们就会卖一些相对高频一点的产品，比如花生、红薯片、山楂类零食等等。这些品类既不和菜市场形成竞争，又能让顾客增加到店频次，顾客来得多了，自然就容易建立信任。

## 低信任度行业的通用思路

类似的思路不仅可以用在银发生意，在其他地方也通用。比如，茶叶、名酒、装修，这些都属于典型的低信任度行业。而且普遍是高单价低频次，对这类生意来说，只有获得用户信任，才能有长期的复购率。

怎么建立信任？不是做促销，也不是简单地套近乎。而是要想办法，让你的客户也变成专家。顾客之所以怕被坑，就是因为自己不懂。假如让顾客也成为至少半个专家，他自然会对你产生信任。

比如，装修。贵阳有个喜百年装修公司，只要用户去这个店里，不管买不买东西，都可以免费领三本册子。分别是《装修白皮书》《装修红宝书》《装修避坑指南》。在避坑指南上，详细列出了在装修过程中可能踩到的坑。假如你能让用户搞懂其中的门道，他对你自然就多了一份信任。

胖东来卖茶叶为什么能火出圈？主要原因之一，就是他们会教顾客怎么鉴别茶叶。在每一款茶叶的包装上，胖东来都会列出详细说明和选购指南。用户知道来这买茶叶，未必是最便宜的，但至少不会被骗。包括胖东来卖保健滋补品，用的也是类似的方法。

借用黄碧云老师的话说，越是有信息差的行业，越要教顾客变成专家。越是低信任度的行业，越要真诚地教客户新踩坑。越是消费频次低的商品，越要跟顾客建立长期的信任关系。真正吸引人的，不是占便宜，而是信得过。这不仅适用于银发生意，对很多行业都适用。
