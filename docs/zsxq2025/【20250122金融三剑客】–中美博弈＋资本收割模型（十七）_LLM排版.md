
# 金融三剑客 – 中美博弈＋资本收割模型（十七）

## 一、远见

1993年3月27日，八届人大一次会议上，荣毅仁同志如愿以偿当选为新一任国家副总统。当年曾豪言“五星红旗上的一个星就是我们荣家”的他也算是得偿所愿。所以，很多人在那个时候就不得不佩服教员的远见，他在临终前曾拉着叶剑英的手，想说话却始终说不出来的境遇……

老人家到底想交代叶剑英些什么？垂危之际，他对权倾一时的叶剑英到底有什么不放心？这在官方出版的多部《毛泽东传》或党史书籍中均有记载：

“1976年9月8日，各个肠胃、政治委员在床边与生命垂危的教员道别，教员当时已经完全说不出话，甚至睁不开眼，只有对叶剑英有反应。在叶剑英行将转身离去时，教员手指颤抖，护士发现后立刻叫住叶剑英，他握住教员的手询问‘您要说什么？’但教员只是看着他，嘴巴微微抖动，十分吃力，但还是一句话也说不出来……”

其实近半个世纪，叶剑英能够久经风雨不倒，周是一个至关重要的因素。包括温格时期，陈伯达等人将矛头指向萧华时，周也是第一个站出来保护的，因为他知道，下一个目标就会指向叶剑英。甚至“二月逆流”时期，也是周的保护让叶剑英未陷谷底，直到1971年让叶剑英出山、一同接待访华的基辛格，正式宣告叶剑英的全面平反与复出（当时接待基辛格的工作就是由周全权负责的）。

同年913事件后，那个制约周和叶剑英的最大阻力：林总从此消失，随即在周的操作下，撤消军委办事组，成立军委办公会议（正是交由叶剑英负责主持）。也就是从那一刻起，奠定了叶剑英在党内“军权第一人”（枪杆子在手）的实际地位，且随着毛泽东、周总理的身体状况越来越不好，叶剑英的重要性也与日俱增。再至后来，他扶登老上位取代华，也是叶剑英的又一得意之作。

说白了，中信上层权贵家（情报就在他那里产生嘛，情报的源头）早就出过类似军队情报泄露的事了，但在这个事（姬老儿子）上，那次各个大佬都守住了底线，愣是没让姬老把自己儿子捞出来。

## 二、涉局

那这回火箭军高层的孩子捞出来没？捞出来了，只不过是在老美的地盘上。他托的谁呢？时任驻美大使：刚子。按照纪律，刚子应该给上面报告的，但他存了私心没有上报。

按说这事到这也就结束了呗，咱们内部也没人知道老美已经获取了机密情报。结果还是后面截获了美五角大楼的一份情报，那份情报显示了咱火箭口的导弹井盖位置（都在积极搞情报战，近期双方还交换了三位线人，后面讲）。

这种事高层不知道也就算了，这知道了还得了，首长遂下令严查，就查到前面说的那个火箭军高官，然后他儿子和刚子的事也一并查出。

不过前阵子小道消息是瞎传，传成了刚子对美国出卖枪杆子情报（纯属误会）。那为什么刚子还是落马了呢？涉局了，不过涉的是另一个“知更鸟计划”的高端局部分。

这个局的间谍多为女记者。她们的出身都还不错，都是成长于70-90年代的内地人。正因为起点高，才能在全国人民还憧憬着吃饱的时候就能早早出国喝洋墨水。

客观来讲，那时候内外的落差还是很大的，大到能颠覆人的三观。而且你在成长过程中形成的观念和心态真的会伴随你的一辈子，这就好比你对青春时期白月光的记忆比真人站在眼前更有杀伤力。

因为一个人的认知、心态、观念、喜好和看法都存在严重的路径依赖（也是这次紧急更换人教版教材的原因），恰恰是这种内在的惯性（人之所以难改命，最大的问题，就是习惯和惯性，在没有外力和内因的情况下，人会在原有轨迹下保持惯性，这也符合物理学规律）。

人想改命，早期没定型时候还容易，因为惯性弱，越到后面越难。足以让她们成为老美操纵下的最好的棋子。

话说她们年轻的时候就想着去占美国那边的好处（能成为高华的人都是当年的优秀份子）。但现在没啥价值了，你们（内地）不是觉得她们优秀吗？那就按照优秀高端人才引进的方式来安置她们啰，给到她们一些在国内的特权。

比如澳大利亚的女间谍成蕾潜伏在央视这样的地方，充当财经记者（这个岗位有其特殊性，能接触大量政商界大佬，比如最近上海第一财经女记者蔚笛成功上位杉杉的操盘手）。

那这回把刚子（有一说是博古的二公子，同名，是否同一个人有争议），组织精心培养的副国级外交接班人拉下水的又是谁？凤凰卫视的那位女记者：傅晓田。

经过常年驯化的她们，哪有省油的灯？后来刚子的这个小三（傅晓田）带着孩子回国要求上位，就和正宫斗，就闹大了。

对组织而言，这个情人问题叠加孩子是美国籍（算裸官性质，内部有规定，裸官不能担任主要领导职务），再加上前面那个破了纪律的事（隐瞒不报），这两个事叠加，刚子还怎么保？尤其是裸官。

然后以这几个人为突破口，军方那几个高层都被查出有问题，然后全部撸掉，只是首长没想到后面会涉及到如此高的层次，之前剑客发过详细名单的。

## 三、收缩

但你懂的，裁员裁到大动脉的时候，都得非常谨慎，由于怕枪杆子再给大家惊喜，短时间内也不易有这么多动荡，应该慢慢来，所以后续枪杆子的反腐，就开始收缩了。

包括这次的D军事件 英国《金融时报》突然发文报道，防长董某已被控制正接受调查。这次消息的发布就有些吊诡，董五天前还在出席活动，上次秦某出问题还是长时间消失导致外媒猜疑，而这次仅仅是五天后外媒就有了确切消息（消息哪里来的？）。

实际上今年以来军方一直有干部落马，都是各军种的一二号人物，尤其是海军，一号二号人物都在第三季度相继接受调查，董之前也来自海军，奇怪的是今年两会没有入军委（未兼任军委委员，因为依惯例，国防部长通常在军委委员中排名第四，位居军委主席及两名副主席之后），可见内部有考量，根据内部人透漏：过渡用用。

有意思的是，在军方否认董出事以后，又半官方了军委苗某被调查的消息，这个就有点刻意了。苗其实是在11月中旬在办公室被架走的，距今也就是不到两周的时间，而且苗当时被控制后，不管是内媒还是外媒都没有发消息，可见苗的问题没有那么敏感，也不是可以用来当筹码的存在。

但是在董的消息发布后，官方又主动披露了苗的信息给董这个信息降温兼转移舆情，似乎在营造出一种你们外媒搞错了的感觉：你们把苗出事当成了董出事。

这个操作在内行人看来：就真的有点此地无银三百两了。

那为什么还要呢？显然对于董如果再出问题（还是苗某推荐的）的话，高层的神经已经极度敏感的，毕竟是接连第三个防长出问题，不管从哪个角度来考虑都会是一件很动摇军心的事情。

不过，目前来看，可能个人问题并不大，苗被调查，董肯定要协查，因为当时推荐董的就是苗，而且推荐董时就有很大争议，因为他是海军机关干部，没有在第一线干过，不过国防部长的本身也是虚职，主要是一些外事工作的沟通协调，并不是真正控制军队的直接管理者。

恰恰因为董是机关的，是文职，所在部门没什么油水，所以从这个角度考虑风险较低（想寻租，机关干部没有太多的实权），才提拔了他。

## 四、博弈

类似这类事呢，外媒就很喜欢炒作，因为可以显得首长有些不太知人善任，来打击新领袖威信。

话说秦某出事后，美国就开始在半岛搞军演，十分高调，宣扬要搞大规模军演15天，一方面是半岛危机（朝不愿意美撤离东亚），另一方面也是它看准了我们枪杆子内部有混乱，来探探虚实，有没有得搞头。

但咱们关键时候没掉链子，还是展示了足够的控制力，并且主动反制了美国，也搞了大规模军演，且十分大胆，我方核潜艇公开上浮到美军几百米前，看美军军演。

话说如果你反腐反到枪杆子都握不紧，你肯定是不敢和美国如此极限博弈的。后面老美看咱们军演搞得也十分有章法，也显示我们对军队的控制力依然在，最终，美国综合评估下来，我们枪杆子的威力，没受到内部因素影响。做出决定：原定的15天军演，只举行了5小时就匆匆宣布结束了。

换言之，中国枪杆子的动员能力还在。你再想，之前新冠疫情爆发的时候，给武汉派了那么多军队去抗疫，这是间接向西方展示什么？也是强大的动员能力啊，否则你以为呢？
