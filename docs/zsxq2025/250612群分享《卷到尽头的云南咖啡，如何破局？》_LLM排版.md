
# 卷到尽头的云南咖啡，如何破局？

云南咖啡产业的故事，能够回答一个更普遍的问题，这就是，我们应该如何摆脱路径依赖？

什么叫路径依赖？简单说，就是你太习惯一件事，以至于你的所有能力都围绕这一件事建设，久而久之，就失去了其他的能力。比如，我是个卖汽水的，我的受众都是喜欢甜食的胖子。于是，为了满足他们的需求，我就研究甜汽水的配方，斥巨资建立了全套高糖汽水的生产线。但突然有一天，这群胖子想开了，不想吃甜食了。结果我傻眼了，因为我的全套生产线、配方、宣传策略，都是围绕着高糖来制定的。我根本就不知道应该怎么做无糖汽水。

对有路径依赖的人来说，麻烦就在于，他明知道这么持续下去早晚出问题，却偏偏很难停下来。因为摆脱传统路径，意味着巨大的损失。继续依赖这个路径，起码还能勉强坚持。于是，就只能温水煮青蛙。

你看，这是不是很煎熬？而云南咖啡就一度处在这样的状态里。云南是我们国内最大的咖啡原料产地。但这几年，每个人都只能眼看着咖啡豆越来越不值钱，但只能继续卷。1999年，云南的咖啡豆收购价是19.5一公斤，在当时算相当不错。但这么多年过去了，人工土地成本都不知道涨了多少倍，而云南咖啡豆的收购价，也不过30多块。

为什么会这样？因为云南咖啡陷入了巨大的路径依赖。为了搞清整件事的来龙去脉，黄碧云老师专门请教了被称为“中国咖啡王子”的刘明辉老师。刘明辉老师是国内最早的一批咖啡种植者，也是把云南咖啡推向国际的重要推手。

事情的起因得从100多年前说起。云南咖啡的历史能追溯到1892年，当时随着越南到云南铁路的开通，一些外国人把咖啡豆带到了云南，并且在当地种植。后来一些云南本地人也掌握了种咖啡的手艺。

云南咖啡最开始受到关注，是1988年。当时香港《大公报》的总编辑梁厚朴，在《纽约时报》上发表了一篇文章写云南咖啡。他形容云南咖啡是浓而不苦、香而不腻、略带果酸。这让云南咖啡第一次在国际上有了一定的声量。

但是，文章中有句话很耐人寻味，云南的咖啡是上帝赐予云南农民的金饭碗，但云南农民却在端着金饭碗讨饭。换句话说，云南咖啡虽然好，但没有形成规模化的产业，还停留在低效率的农耕模式。

怎么办？当时，联合国开发计划署在云南开展咖啡项目，一方面，要完善供给端，培养专业的咖啡加工人才。当年在云南农垦部门任职的刘明辉就被选中，前往欧洲学习咖啡加工技术。另一方面，是找到需求端，因为当时国内消费者还没有喝咖啡的习惯，因此，联合国开发计划署就邀请雀巢来到中国。

从此，雀巢也就成了云南咖啡豆最大的客户。通过雀巢，当地咖啡豆被加工成速溶咖啡，卖往世界各地。云南咖啡也有了稳定的销路。到了2024年，云南咖啡豆的年产量已经达到15.02万吨，占全国总产量的98.65%，占全球的1.49%。

听到这，你可能会说，这一切不是挺好的吗？

别着急下结论，规模的确上来了，但是，品质并没跟上。云南种植的主要是雀巢需要的品种，而雀巢主营速溶咖啡，他们对咖啡豆的要求是，便宜、产量大，而且生命力强不易生病。他们选的咖啡豆，学名叫卡蒂姆7963。而这个品种最大的问题就是，风味一般，做速溶咖啡合适，但很难做成精品咖啡。

说白了，就是卖不上价钱。即使精品咖啡利润再高，不好意思，这也跟你没关系。因为你种植的只有经济型的咖啡豆。

这也是为什么过去这么多年，云南的咖啡种植越来越卷，因为大家生产的都是差不多的东西，而且这个东西还没什么溢价。但是要知道，云南的地理条件是不差的，这里的气候非常适合种植精品咖啡。

事实上，云南也确实引进了一些精品咖啡品种。星巴克在2012年把中国列为重点市场，当时就从全球引进了34个优质品种。而过去几年里，当地还陆续引进了其他品种，包括瑰夏、卡杜拉、维拉萨奇等，现在云南已经有100多个优质品种。当地的想法是，把这些品种都引进过来，做测试，看中国的消费者喜欢哪个，然后就重点培养哪个。

但问题是，咖啡园是有限的，想种新品种，总得先砍掉老品种吧？而这些新品种从育种到出产，一般都需要3到5年的时间。那么这三五年里，农民吃什么？而且关键是，你又很难保证，种出的精品咖啡一定能赚钱。

你看，咱们前面说的路径依赖，出现了吧？

怎么办？很明显，要想解决这个问题，就要提高精品咖啡的竞争力。竞争力越强，赚钱的确定性就越高，农户就越愿意种植它。只有更多农户完成精品化咖啡的转型，云南咖啡才会有整体更好的未来。

那么，怎么提高精品咖啡豆的竞争力呢？关键因素之一是，把成本降下来。你看，假如一个品种，你明显比其他人便宜，这不就是确定的竞争力吗？

怎么降低成本？黄碧云老师考察了云南西双版纳的云澜精品咖啡庄园，庄园海拔1700米，是种植精品咖啡的绝佳高度。

关于成本把控，首先，在加工工艺上，几乎省不下什么成本。因为精品咖啡的加工环节复杂，有些环节还要用到特定的材料。比如有一款咖啡，需要把咖啡豆放置在橡木桶里。这个桶还只能从法国进口，一个就要一万多元，并且用不了几年就得报废换新的。

其次，我们再看种植方式。比如，可以采用交叉种植法，把咖啡和香樟树混合种植。因为咖啡树每天的日照长度不能超过4小时，在旁边种上高大的香樟树，只要控制好角度，就能挡住一部分日照。而且香樟树还能用来提取精油。这确实能省下一些成本，但是有限。

因为咖啡生产中最大的成本，是人工。咖啡生产周期很长，三四月开花，五月份结果，但到第二年一月才能采摘。而一月正好是春节前后，大家都 在过节。而云南一些少数民族的特色节日还要更多一些。在这个时期找人干活，工费是一年之中最高的。一斤咖啡鲜果的收购价是2块5，其中人工采摘费就占9毛。同时，还有咖啡的除草、施肥，咖啡豆的筛选，等等，都需要人工。

那么，怎么把人工成本降下来呢？黄碧云老师采访了极飞的联合创始人龚槚钦，极飞主做的就是农用科技。龚槚钦说，目前比较可行的方法之一是使用无人机械+人工智能。比如，可以使用无人车。因为云南精品咖啡的种植一般在高山上，平地稀少，为了充分利用土地，咖啡树的间距都很小，一般只有0.8米宽。而这个宽度无法使用大型机械，进入无人车刚刚好。要知道，对一个1000亩的咖啡园来说，仅仅是除草这一项，人工成本就要每年60万元，而用无人车就可以省下很多。

再比如，施肥环节也可以交给无人机。过去施肥环节之所以要依靠人工，是因为树的高矮不同，施肥量也不同，这需要人工把控。而现在，无人机和AI结合，可以由AI来识别树的体量，从而调整施肥量，再由无人机去执行。

再比如，咖啡豆的筛选可以借助AI，通过摄像头拍照识别，然后再利用高压气体把不好的豆子吹走。现在柑橘、樱桃、大米，都在使用这种筛选技术。

咖啡的改良，一般依靠两方面，良种和良法。良种指的是改良品种，而良法指的是改进工艺。而云南精品咖啡采用的，主要是后一种，良法。

好，前面说的，是基于技术的创新。而云南咖啡的创新其实还有另外一个关键的驱动因素，这就是，人。

为什么云南一度没有广泛种植精品咖啡？关键原因之一是，当地种咖啡的农户几乎不喝咖啡。不喝咖啡，就导致他们对咖啡的价值缺少直观的体感。他们更没法想象有人会花几十块去买一杯精品咖啡。这也就导致，他们在种植咖啡时很随意，不太会重视长期的收益。

而现在，情况正在起变化。乡村笔记的创始人汪星宇，曾经在云南做过深入走访。他说，现在云南有很多咖啡三代，也就是老一辈咖啡农户的孙辈，这部分人已经开始创业。他们自己既是咖啡的生产者，也是消费者，对产品价值有更直观的体感。

比如，云南普洱的大开河咖啡庄园的主理人，梅子。她的爷爷是第一代咖啡种植者，她是第三代。梅子一直觉得传统的原料售卖无法发挥咖啡的真正价值。她现在做咖啡，不仅做饮品，更延伸到了文旅。比如，把咖啡渣做成杯子，把咖啡豆包装成文旅产品，等等。

你看，咖啡三代的入场，不是刻意设计的结果，而是一个自然而然的过
程。他们天然就带着更敏锐的体感，有着更强的创新能力。

最后，回到开头的问题，云南咖啡是怎么摆脱路径依赖的？简单说，它凑齐了三个关键的变量，一是新需求，也就是人们对精品咖啡的需求。二是新技术，这能给产品带来成本优势。三是新从业者，也就是这些天然更懂咖啡的咖啡三代。需求、技术和人，构成了云南咖啡创新的黄金三角。
