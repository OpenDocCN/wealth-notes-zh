
# 常年住酒店是一种什么体验？

过去一年里，至少有10个月我都在住酒店，国内国外的都有。我住的主要是3-4星级酒店，在“居住”这个功能上，平均每个月的花销应该不到1万元。有时候去海南这样的地方，朋友让我住他的度假房，我住了几天就不住了，然后又去住酒店了。因为我真心觉得40平米的市区酒店房间比140平米的海景居民房要舒服多了。

那么问题来了，一年中有300天都住酒店的话，是一种什么体验？跟买房、租房相比，有哪些优缺点？常住的话该如何跟酒店协商价格？从心理、社交、归属感等维度来看，常年住酒店适合什么样的人？

下面我就详细聊聊上面这些问题，想到哪里说到哪里，全文一共2000字。

## 1. 迁徙/搬家

有一类人特别适合长期住酒店，那就是一年四季都在不同城市居住的人。他们可能是经常出差的高管、自由职业者、网红、明星、旅行博主……还有一类人，他们单纯就喜欢到不同的国家/城市去居住、去生活。当然，这些都要有一个大前提：预算充足。因为你无论去任何一个城市，住酒店的价格肯定是远高于租房的价格（同样的面积）。另外，心理上还不能有太多对人、对物的牵绊。比如，如果你上有老下有小，公司里还有一帮人，他们都需要你照顾，那你肯定只能一直在他们身边打转转。还有的人连搬个家都要做很久的心理工作，他们总会想：我的人脉圈、社交圈都在这里，我去新的城市还要从0开始积累……还有的人被物品绑定了，比如他的家里的设施非常齐全，有影音设备、健身设备，还有各种各样的按摩椅、咖啡机、面包机、书房……甚至还有猫猫狗狗、花花草草需要照顾。总之呢，以上这几类人都不适合常年在外面住酒店。

## 2. 价格

我对酒店的要求主要有以下几个：前台会微笑，房间窗户是双层玻璃，房间里有冰箱，酒店早餐丰富，有洗衣房、烘干机和健身房。满足这几个条件的酒店，在二线城市的“非旅游旺季”，标价可能是350元一晚。但是，通过下载他们的APP、成为会员的话，由于长期居住，很快就能升到最高等级，连住半个月可以打85折，300元一晚。我问前台，连续居住一个月、两个月的话，还有什么优惠吗？前台会说，APP上连续居住15天那个已经是最大的折扣了，没有更高的折扣了。然后我就会让前台把他们销售经理的电话给我，我直接给销售经理打电话，问经理居住一个月多少钱。经理给出的折扣还要更大，连续居住一个月的话，平均一晚可能只需要260元。当然这个操作流程要分城市、分酒店品牌，有的品牌就可以这样操作，有的品牌就是死心眼。那么问题来了，用这个价钱来租房的话，是不是能住很大的房子？其实还真不一定。首先，按月短租的房子要么很贵，要么很差。其次，酒店的价格其实包含了很多服务——除了居住之外，水、电、空调、网络、家具、电器、床上用品、洗浴用品、早餐、下午茶、夜宵……

## 3. 时间

有人会说，上面这些费用加到一起也没多少，最终还是比租房贵很多。我想说，时间也是金钱。比如，我住的酒店如果单独买早餐的话，价格是60元，实际上每个食客吃到嘴里的食物成本应该不超过20元。但是，如果自己买食材、自己做的话，需要消耗大量的时间。比如，酒店的早餐很丰富，种类很多，有几种炒菜、几种凉菜、几种水果、几种面条、几种饮品，还有红薯、玉米、面包、炒饭等多种主食。我每样都都来一点，盘子里就会有七八种食材。还有很多酒店有免费的下午茶和夜宵。另外，如果住酒店的话，打扫房间、洗衣服的时间也都省掉了。上面这些事情如果都让我自己做的话，先不用说能省多少钱，主要是费时费力。如果请家政人员来做的话，也得需要一笔额外的费用和各种沟通成本。如果你住在自己买的房子里，或者自己租房住的话，还需要跟物业、水电燃气、宽带公司的人打交道。如果下水道堵了、空调坏了、门锁出故障了，你也要找人维修，或者购买、更换，这些都需要沟通、需要时间。如果住酒店的话，这些时间都能省掉，如果这个房间住着不满意，直接打个电话就可以换房间。

## 4. 社交/归属感

人是群居动物，需要社交、需要归属感、需要亲密关系。所以，如果你在外面住酒店的话，上面这些都要损失掉，那你就太亏了。我的建议是，要么你有一个稳定的、愿意跟你到处跑、自己也能赚钱的伴侣，要么你在各地都有非常靠谱的朋友和社交关系。否则一个人在外面东奔西走的，可能会非常孤独。当然，如果你喜欢结交新朋友，也有很多思路，比如下面这几种：

①大城市的酒店有泳池、健身房、行政酒廊，旅游城市的民宿也有很多共享的区域，在这些地方都能结交新朋友。还有一些酒店会组织欢迎酒会等各种各样的文化交流活动，这时候也可以结交新人。

②根据自己的兴趣爱好，通过社交媒体、线下活动平台等加入当地的兴趣小组或社群，如摄影群、读书俱乐部、骑行群等。定期参加他们组织的活动，与志同道合的人交流互动，不仅能满足社交需求，还能丰富自己的生活。

③积极了解和参与当地的传统节日、文化庆典、艺术展览、音乐会等活动，深入体验当地的文化和生活方式，让自己更好地融入当地环境，从而产生一种对所在城市的归属感。比如U君自己如果去泰国或者日本旅行时，只要能赶上当地有什么民俗活动，我肯定是要去凑热闹的。
