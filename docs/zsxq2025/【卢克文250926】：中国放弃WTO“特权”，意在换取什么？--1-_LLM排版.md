
# 中国放弃WTO“特权”，意在换取什么？

中国商务部明确表态，中国仍然是世界上最大的发展中国家，但在WTO当前和未来谈判中，不再寻求“特殊与差别待遇”。

中国加入WTO二十年来，经济实现了火箭式腾飞，外贸是关键。身为发展中国家，“最惠国待遇”这个“特权”，极大促进了我们的出口。以至于不少声音说，咱就是靠“WTO最惠国待遇”起家的。

那么，为什么当下的中国反而要“自废武功”，主动放弃好不容易争取来的“特权”呢？

## “特殊和差别待遇”是啥？

二战后，各国反思历史教训，意识到高关税壁垒和贸易战会引发战争。为此，1947年，以美国为首的23国签署了《关税与贸易总协定》(GATT)，旨在降低关税、推动全球自由贸易。

这个组织最初就像个“富人俱乐部”，规则都是发达国家定的。后加入的发展中国家发现，原有规则几乎没有考虑发达和发展中国家的差距。

这就像让一个新手和重量级拳王按同样规则比赛，显然有失公平。

于是在发展中国家集体争取下，1965年，“特殊与差别待遇”被正式写入条款，核心就是承认发展差距，不要求发展中国家和发达国家对等开放市场。1995年WTO成立后也延续了这一原则。

具体来看，这个“待遇”主要包括：

1. 最惠国待遇关税：发达国家对发展中国家出口给予更低关税，方便发展中国家向发达国家卖货。
2. 市场保护权：发展中国家可对关键产业维持较高关税，以扶持本土企业。
3. 国际援助：发展中国家获得世界银行，国家开发机构等的资金支持。
4. 最不发达的国家，还可获得零关税的特殊待遇。

说白了，这就是WTO为发展中国家设置的“新手保护期”。

对中国来说，2001年加入WTO是一个很重要的转折。借助包括“特殊和差别待遇”在内的全球化机遇，中国逐步成长为“世界工厂”，并于2010年GDP超过日本，成为世界第二大经济体。

如今，中国主动宣布在未来谈判中不再寻求这一待遇，相当于不再需要“新手保护期”。

## 为什么我们主动这样做？

其一，中国的“块头”早已不是新手。2001年刚入世时，中国GDP仅1.34万亿美元；到2025年，这个数字已增长十余倍。

想想看健身房里，一个腹肌八块的壮汉，还硬要穿“新手训练营”T恤，很容易招来“开着奔驰领低保”的批评。这几年美国等国家不断质疑我们发展中国家的身份，觉得我们在“钻规则空子”。

其实吧，以中国的贸易竞争力，也确实不怎么需要“新手保护期”（除了小部分特殊产业）。

其二，很多发达国家早就不给咱们“最惠国待遇”了。

去年11月就有32个国家，包括英国、加拿大、土耳其这些国家，正式取消对中国的普惠制关税优待，理由就一条：中国已不是发展中国家。

虽然我们觉得，我们是发展中国家，但人家压根不认。

而自2018年中美贸易战打响以来，美国对华关税已从平均2.2%大幅提升至51.1%。中国商品在美国进口总额中的占比，从2018年的高点(约21%)下降至2024年的13%。

除此之外，美国国会这几年不断有人提议，要彻底取消我们的“永久正常贸易关系”地位。加上特朗普重返白宫，中美经贸会走向“事实性脱钩”的猜测，不绝于耳。

也就是说，中国所面临的早已不是“要不要特殊待遇”的问题，而是现实环境中，“新手保护”早已名存实亡。

与其被动等待别人收回，不如我们主动调整姿态，在新的规则博弈中拿回主动权。

## 美国对WTO的系统性“拆台”

当年克林顿支持中国加入WTO，本想着把中国拉进美国的圈子里，指望我们慢慢“听话”。

但到奥巴马时期，美国发现，在WTO框架内，不仅没能规训中国，中国反而学会了利用规则。他急了，于是对WTO开始系统性“拆台”。

核心一招就是，阻挠上诉机构法官任命，让WTO的“最高法院”彻底瘫痪：

2011年在美国对华钢铁征收反倾销和反补贴税一案中，上诉机构推翻了专家组裁定，支持中国的主张。

同年，美国以“需要改革”为由，拒绝支持本国提名的法官詹妮弗·希尔曼连任，打破了法官连任的惯例。

到了2016年，美国直接撕破脸，硬是拦着韩国法官张胜和，不让他连任，说他在判中国反补贴的案子里“管得太宽”。

WTO的顶梁柱——上诉机构，就被掏空了。

当这个最高仲裁机制无法正常运转，WTO的约束力就名存实亡，美国便可以不受制约地采取单边行动。

紧接着，奥巴马就开始“拉小群”——全力推动《跨太平洋伙伴关系协定》(TPP)，这个协定用极高的环保、劳工、知识产权标准来主导亚太地区经济合作，目的就是要把不能达标的中国排除在亚太贸易体系之外。

所以你看，美国绕开WTO，另立山头的心思十多年前就埋下了。

而奥巴马的接任者拜登，更是个擅长拉帮结派的高手。

拜登政府嘴上说支持多边合作，却没有采取任何实际行动来恢复WTO的上诉机构。

在贸易政策上，继续搞小圈子，拉拢盟友搞“印太经济框架”，与日本和欧盟推进关键矿产、半导体等领域的供应链协定，推动“小院高墙”技术限制，本质上还是贸易保护主义那套，连特朗普时期对华高关税，也原封不动留着。

对付中国就更明显了，2023年3月，美国众议院甚至以415比0的离谱票数，通过一项“中国不是发展中国家”的法案。

表面理由是，中国已是世界第二大经济体，但背后藏着两本账：

一本是明账：想取消掉中国在WTO、世行等国际组织里享受的关税优惠、资金支持等发展中国家的特殊待遇。

一本是暗账：削弱中国作为全球南方代表的身份认同，分化发展中国家团结，进而孤立中国的外交地位，维持美国在国际规则制定中的主导权。

这套组合拳，中国看得清清楚楚。

在24号的发言中，商务部明确强调：“中国仍然是世界上最大的发展中国家。”

这可不是嘴硬：

WTO从没有明确定义过什么叫“发展中国家”，其身份主要由成员自我宣示。新加坡这种国家都宣称自己是发展中国家。

中国的人均GDP才1.2万美元，离发达国家3万美元的门槛，还差着一大截。

中国始终是全球南方的 一员，会坚定不移与发展中国家站在一起。

未来更要带着中国式现代化的经验，和大家一起搞发展，促团结。

美国想给我们戴高帽，使离间计？没门。

## 中国的选择：维护WTO多边框架

拜登在时，美国还只是拉小群，到了特朗普2.0时代，画风突变，美国开启无差别攻击，连传统盟友，也难逃关税大棒，全球贸易体系摇摇欲坠。

在这个紧急关头，中国主动站出来，宣布在未来谈判中不再寻求“特殊与差别待遇”，其实质意义，是表态维护WTO多边框架。

当然，要想让“旧群”的成员不被“新群”吸引，光靠姿态还不够，总得拿出点实实在在的诚意。

在世界贸易组织召开的发展系列会议上，中方代表向世贸组织通报：中国已对所有建交的最不发达国家全面实现100%税目零关税。

2025年第一季度，中国自非洲最不发达国家进口额同比增长15.2%，其中咖啡、可可豆进口量分别大幅增长70.4%和56.8%。

卢旺达干辣椒、埃塞俄比亚咖啡等农产品，凭借零关税迅速打开中国市场。

对外投资上，中国通过“一带一路”倡议为发展中国家注入发展动能：

中老铁路累计货运量已突破5400万吨，旅客发送量增长5倍；

非洲首条万吨级智能化水泥生产线在埃塞俄比亚投产；

拉美重要枢纽秘鲁钱凯港开通，被当地视为“改变美洲航运格局”的关键项目……

在前沿科技上，中国反对科技垄断与科技霸凌，主张用合作支持其他发展中国家科技进步：

中国与巴西联合进行航天技术研发；

中国协助巴基斯坦、玻利维亚等国发射卫星；

2025年，DeepSeek用接近“白菜价”的开发成本，搞出了和世界顶尖AI比肩的高性能。以后，广大发展中国家都能用得上、用得起先进的人工智能了。

中国的逻辑很简单：培养消费者，而不是剥削生产者。

发达国家的传统援助，往往是把高污染、高能耗的产业转移出去，再低价收购你的资源或初级产品，本质仍是剥削逻辑。

而中国的做法是：帮你把路修通、电接通、港建好，只有让你的经济发展起来，老百姓有了钱、成了有消费能力的群体，才能最终消化我们自己的优质产能。

很多人会骂，说中国是打肿脸充胖子，自己还有那么多人月收入不高，为啥要去帮别人?

他们最担心的是：你把产业链都搬走了，本国老百姓干啥?吃啥?

这里有个关键误解：中国转移出去的，是已经过剩的产能，不是整个产业链。我们绝不走美国那种“产业空心化”的老路。

就像制衣行业，中国现在重点发展高端面料研发、智能服装设计，还有上下游的配套，而把一些技术成熟的普通加工环节，转移到有劳动力优势的邻国，比如越南。

之后，中国本土的工厂会转向生产更高附加值的产品，需要的是更专业的技术工人，这不是丢掉了饭碗，而是升级了饭碗。

“全球南方”国家心里都有杆秤，很清楚跟着谁能真正解决吃饭的问题。所以，现在大家不爱搞小圈子那一套，更想抱团合作。

对中国来说，最好的策略就是带头推动改革，利用好这个现成的平台，主动推进WTO开放，和大家一起把这张有点旧的桌子修好、变大，让更多人都有位置坐。

目前，全球经济治理正在形成两套系统：

一边是“经济上的北约”，会员制的高端俱乐部，只追求供应链安全和技术保护，对外部有一定排斥性。

一边是“经济上的联合国”，像WTO这种老牌大平台，中国正在通过开放市场、深化南南合作等方式，继续为大多数国家服务，讲究包容和共同发展。

这两套体系会长期并存，规则和玩法都不一样。对很多发展中国家来说，这反而意味着更多选择——可以根据自身需要，在两边灵活参与，不必非要选边站。

这种多元共存的局面，说不定反而能慢慢催生一个更加平衡、不再是一家独大的全球经济新秩序。
