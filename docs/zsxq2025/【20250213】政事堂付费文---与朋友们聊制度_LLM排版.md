
# 与朋友们聊制度

我们这为什么没有合法性? 人家写在两张纸上就有合法性?笑话!我们从秦祖龙开始，只要上台就是合法性。不需要写在纸上，也不需要地球另一端的人承认。我们写甲骨文的时候，他们还是衣布遮体的野蛮人。近期读了李录的文明，现代化。现代化和政治体制没啥太强关系。现代化主要就是自由市场和科技这两个。最后就是唯一的大市场。只要开放，哪怕只是开放浦东，对未来都没有必要悲观。更何况当，前，在全球看来看，我们这边已经开始全球市场了。对未来一二十年有信心还是要有合法性，礼崩乐坏，那不就是西晋了么。

文艺复兴开始，西方科技确实点亮了人类文明之光，是不得不承认的事实。但是三十年河东三十年河西，谁都有辉煌和没落，没有什么是永垂不朽。以前说中华文明我不理解，但这两年看看，确实有它核心的力量，我们不必骄傲自满，也不必妄自菲薄，更不需要别人来承认我们的合法性。

我们的中华文明，上台了有权利，但也要代表广大人民群众的利益。不然造反就是诛一独夫耳，这是朱元璋想删掉的孟子。还是要有合法性，礼崩乐坏，合法性不是个机械的概念，也未必是写在纸上就有合法性，不写上去就没有合法性。

“我们的中华文明，上台了有权…”是的，代表最广大人民的最根本利益，就是合法性。大多数王朝开始的时候，以及目前至少现阶段，都是上述的代表。我们最上头的初心还在。子明：我喜欢三个代表总结的合法性。所以支持人民对于当权者的监督，真正践行人民是阅卷人，这样才好。

“我们的中华文明，上台了有权利，但也要代表广大人民群众的利益。不然造反就是诛一独夫耳，这是朱元璋想删掉的孟子”。看看晋朝，以孝治天下眼镜的话，过后总是越看越有道理。五胡乱华，神州陆沉，两脚羊遍地走，东晋他们自己托孤都只敢说如诸葛武侯，王丞相故事。

子明：所以，我很不喜欢这种改革，代价太大，可抗因素太多，但如果走了这条路，也就走下去吧，反复折腾更惨。这是动态的平衡，其实是最难的。就像马太效应一样，天平总是趋向一端倾斜，要对抗这种趋势，更是难上加难。不过现在有科技的加持和科技的管理工具，比古代要好很多了。

子明：汉末魏晋南北朝时期到隋唐，相当于君主制改总统共和制，引发的长期混乱；唐中期五代到北宋，相当于总统共和制改议会共和制引发的混乱。希望别搞得，都来十五帝，拨乱五十秋，这就是合法性大打折扣的弊端。读历史，往往感觉，我们喜欢下一个朝代矫枉过正，然后玩死。

一直以为秦到清都是君主专制。明朝是典型的君主专制。我个人认为，我们古代的管理机制是地球上最完善的，和其他地方比，我们的总内耗和总代价是最小的。子明：政治制度在明朝之前，一直都是向前改进的，朱元璋上台后开倒车。那个动不动就哭孔子，哭坟，居然能左右政策。

其他地方感觉都是打打杀杀和遍地神棍，惨的一比。康乾隔代开了波猛的大倒车，还伴随很强的不行了咱楼一票润宁古塔暗示。政治选贤和经济选贤，这两个都是最终的结果。自由市场和科技是动力。当时看书，说朱元璋给户定职业世袭，就觉得离谱，乞丐出身确实有局限，而且元朝对旧利益集团破坏，阻力也小。子明：北宋王安石改宅地法改革失败，地主阶级爆炸。明朝是把之前一千多年的改革都给废掉了，直接回归农奴制。九十年代铁路职工还是继承呢。

子明：那么多的牺牲都废了。怎么看那时的权力结构?按史书记载一般都是皇帝绝对掌权。除了曹操等少数时候，那时候就标志了农业时代的终结。子明：共和制和议会制下，皇权都不大的。没有吧，丞相，大夫，开府，从春秋开始就有了。子明：宋朝的皇帝，想杀个士大夫都很难；隋唐的皇帝，也要看门阀的脸色，人家都不愿意把孩子嫁入皇室。以前看史书，想不通有时候皇帝怎么会那么憋屈呢?

子明：中国一直以来都是在不断的改革进步，就是明清两朝开起了倒车，于是被西方公司反超了。确实，读历史读到明清开始感觉憋屈。没看到过写皇帝憋屈的，可能看的太少了。然后。…。到了，.他们是说唐宋的皇帝憋屈。所以大明就变异出了宦官集团和东林党两个超级变态体来平衡皇权。东林党不就是士大夫，宦官就是皇权。我说的就是整体的感觉，不是皇帝的角度。

子明：宋朝是议会制+宅地法+言论自由+职业军队+自由贸易……放到维多利亚时代都最最顶级的政治制度。以前不明白，感觉东林党的人脑子都坏掉了，为了那一个名声，被杀头也前赴后继，勇于争先。子明：明朝直接改成了跟沙俄一样，职业军队在火炮科技点出来之前，打征召军队肯定吃亏的。你一个兵的成本是人家十个兵的成本，没有科技优势肯定打不过。生产关系超过了生产力的发展，也是不适合的。子明：但是不要大国叙事，站在个人的角度，职业军队意味着死亡率大幅降低了。

从汉到唐，血脉，世家和地方豪强，继承脉络非常清晰，到大宋也能跟得上。朱重八开局一个碗，完全是重新洗牌。到教员又是一次重新洗牌。小时候大人们总说宋鼻涕…说宋朝各种挨打，但是从描述里总感觉群众很幸福。。子明：去过征召部队的死亡率是极其夸张的，但是职业化军队意味着死亡率很低了。宋朝之前，动辄几十万大军，到了大宋，过万都不容易了。参考二战德国东线战场，职业军队面对乌泱泱的征召部队，不被虐成菜才怪呢。
