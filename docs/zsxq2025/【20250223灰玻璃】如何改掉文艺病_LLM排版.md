
# 如何改掉文艺病

想跟您探讨一下文艺女青年的生活方式与前途命运。我从以前的纯靠性情去恋爱，变成如今脑子里也会权衡利弊。但理性和感性博弈的时候感性还是会占据上风，还是忍不住疯狂上头。明知不可为而为之的那种人。我从小很喜欢凄美文学，喜欢悲剧为结尾的电影，甚至会享受自己的痛苦，沉浸在自怨自艾，反复拉扯又精神自我虐待中。我不理解为什么自己如此喜欢哲学、艺术、文学、心理学。我能理解张爱玲、陆小曼、琼瑶这类女人。欣赏林，却做不到林那么理性，可能我本人更像陆一些。小时候喜欢悲伤的音乐、悲伤的文学、悲伤的电影艺术，也几乎不会爱上阳光的男孩事物，会被阴郁脆弱又“强制爱”的男人吸引。不知道学姐有没有看过王朔的文学《一半是海水一半是火焰》，我总觉得自己很像那个女主吴迪。第一任男友占有欲极强，第二任男友是个软饭硬吃的渣男，第三任男友很爱我，第四任是轰轰烈烈的姐弟恋弟弟很疯也挺花。几乎没谈过啥正常的平淡如水的恋爱。

热衷于拯救浪子，享受痛苦，这是什么样的心理呢？会跟个人价值高的男生谈恋爱，但真真正正能挑拨我心弦的还是善良的“混蛋”。自己可能是父母亲戚朋友眼中非常优秀的女孩，但我知道自己内心有多阴暗和疯。我就像个矛盾体，生活在优等生的圈子里却喜欢江湖气的人，生活在梏中又找刺激寻自由。希望学姐能帮我分析一下原因，以及如何改掉自己的“文艺病”。

文艺本身具有自毁性，学姐真的不建议女性过早不加分辨地接触文学。民间俗语云少不读水浒、老不读三国，学姐将近不感之年才体会到这句话背后的阅历，更何况文学作品中很多是满足女人单方面自恋的爽文，即使正统文学很多作家是有心理学方面障碍或社会化不足的问题，比如有重度抑郁症三毛、川端康城、海明威都因抑郁症而自杀、重度社交障碍比如张爱玲、卡夫卡，这些作品中携带的思维痕迹会不知不觉影响读者的思考惯性。即使写爱情文学《简爱》之流的作家，她们的现实婚姻很不如意，用现在的话说没有拿到结果。不是说他们不优秀，就文学艺术单维度而言，他们的作品极具艺术感，但你要是用来指导现实生活，会把你带沟里。

比如《窄门》，阿莉莎很明显是回避性依恋，是需要看心理医生积极治疗的，但受到了文青的共鸣和追捧。社会化不足，也没有哲学、心理学积淀之人，是无法读出文学的批判性的。如果在价值观尚未成形的青春期，深度沉迷于这种反现实的戏剧性情节中，长期以往，人会废掉。同理，电影、电视剧、短视频看多了，人也容易变得弱智，真实世界很多事情的演化极为复杂迂回，大部分现场的实际体验是枯燥，单调，无聊，长期间缺乏有效反馈，和各种挫折。电影、短视频乃至小说，因为时间限制，必须在最短时间内把最戏剧性的核心场景表现出来，所以比例感是完全扭曲的，和真实世界完全不一样。而且在大脑里容易形成思维钢印要吃很多亏才能扭转回来。总之，喜欢文艺也是一种自恋，它有一种小资性，看多了会让自己有种孤芳自赏、自命清高的感觉。学姐希望大家都能脚踏实地地参与社会运作，不去做漂浮在半空中的人。

题主是否是别人眼中的乖乖女呢？如果从小到大太过于压抑自我，总是习惯去迎合别人的话，会容易喜欢上有点痞气和社会气的男人，他们身上释放出来的攻击性是乖乖女没有的，因为补偿心理，对于乖乖女来说会有致命的吸引力，可以参考马思纯。我看完也有一样的感觉，感觉题主好像一直被一种有形或者无形的力量压制，无法反抗或者反抗无效，有一种自暴自弃非贬义，只能从其他方面去找突破口的感觉。“人须在事上磨”，反而是聪明又充分社会化的男性，更可能相处宜人不累。如学姐说的，能明白人世的灰度，入世通达圆融，出世能激发自己和伴侣的神性。从头到脚都透着一股不同于纯文艺青年的大气。
