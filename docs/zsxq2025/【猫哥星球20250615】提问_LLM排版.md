
# 提问：

猫哥您好!学习你的专栏多年了，第一次提问。我
及我的祖父辈都是普通工薪阶层。我的儿子很优秀，
凭自己的能力，今年申请到了美国哥伦比亚大学的MBA
。现在已经办理好的赴美签证。鉴于目前中美关系愈
发紧张，我对于孩子赴美留学十分忐忑。另外，两年
300W左右的学习费用，除去10W美金的奖学金，还有不
小经济压力。请问猫哥，对我孩子去美留学有什么建
议?

## 猫哥回答：

花几百万去美国混个学历将来是个什么结果?
大概是两个方向。

其1，留在国外成为边缘人，未来美国大概率与中
国关系越来越差，华裔在美国就是出气筒，当然也可
以移民加拿大澳洲这些国家，但是很难融入主流社会
，边缘化是一定的。我的一个朋友儿子花了几百万在
加拿大拿到法律博士，移民加拿大后现在在一家律
师事务所工作，三十岁的人还有博士学历，依然做的是
跑腿打杂的事情。

其2.回国，大概率做不了公务员。进不了国企央
企，没有大学人脉，一切从头开始，至于能混个什么
名堂，就看自己造化，我觉得工薪阶层真的没必要砸
锅卖铁把孩子送出去，反正我坚决反对这样做，对自
己孩子也是如此。
