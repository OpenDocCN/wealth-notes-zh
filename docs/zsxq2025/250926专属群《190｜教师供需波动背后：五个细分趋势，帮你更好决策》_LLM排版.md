
# 教师供需波动背后的五个细分趋势

## 不同区域的生源和编制配置

不同地区、城乡之间的学生分布差异将长期塑造教师编制的配置格局。华南师范大学特聘教授刘善槐在《人民教育》的论文中指出，学龄人口持续向经济发达区域集聚，呈现“东密西疏”的态势。2013-2023年，东部地区义务教育学龄人口占全国的比重由37.2%增长到41.1%，而中西部占比则呈现下降趋势。

省域范围内，省会城市人口集聚效应不断增强。以中部某省为例，2013-2023年，该省省会小学阶段在校生数占全省学生数的比例由10.43%提升到14.63%，且比例持续上升。

县域范围内，教育质量差距因素驱使下，学龄人口呈现“向城性”，即向县城流动的趋势。

福建师范大学陈坤在《中国教育学刊》发表的论文指出，教师配置在区域间存在“无编可用”与“超编冗余”的结构性问题。城区学龄人口流入端学生骤增，但编制总量并未增加，如深圳、成都等市区中小学“无编可用”现象突出。而在学龄人口流出端，教师“冗余”严重，如东北某县义务教育的平均生师比不到5:1，远超国家法定标准。

## 不同学科老师的不同需求

教师需求还取决于学科，不同学段、不同学科的老师需求呈现分化。北京师范大学胡咏梅教授在《中国教育学刊》发表的论文指出，小学阶段的教师资源配置呈现明显的学科结构失衡，城、镇、乡普遍缺少科学、体育课程教师，严重缺乏劳动、综合实践活动和道法课程教师；而镇区与乡村在美术教师的配置上也明显不足。初中阶段，城、镇、乡在多数学科上出现了教师资源的冗余，但在综合实践活动、劳动等方面存在明显的教师需求缺口。

东北师范大学谭卓在《现代教育管理》杂志的论文中预测，2026-2030年，小学阶段语文、数学、外语、艺术、信息与科技等学科的师资供给相对过剩；而道德与法治、科学、体育与健康、劳动、综合实践活动等学科教师数量存在一定程度短缺。2031-2045年间，小学阶段各学科教师的需求数量将总体趋于平稳，只有劳动学科可能出现师资短缺。2046-2050年，除劳动、体育与健康两门学科外，其余学科的师资都将处于富余状态。

初中阶段的老师，2026-2030年除体育与健康、劳动和综合实践活动三门学科外，其余学科都表现出教师富余的现象；2031-2035年和2046-2050年两个时期内，劳动和综合实践活动这两门学科仍然面临师资不足的问题，其他学科的教师资源较为充足。

## 师范生培养规模与市场需求不匹配

师范生的培养规模与市场需求不匹配，本科层次有所过剩，研究生层次却供不应求。上海大学人口研究所程顺祺等人发表在《教育与经济》上的论文说，假定未来五年（2025-2030年），本专科师范生毕业生规模，降到2016-2020年间的318.9万人的平均水平，面对当前市场上只有100万的岗位需求，即便100%录用师范生，也将只有不到1/3的师范生进入基础教育教师队伍，比例持续下滑。

早在2015年，《中国青年报》就报道过，全国有60多万名师范生毕业，但整个基础教育领域的教师需求只有25万，其中至少1/4的名额被优秀综合性大学的本科生、硕士生甚至博士生竞争走。十年前，师范毕业生与基础教育教师岗的供需比例高达3.2:1，换句话说，只有三成左右的师范生有机会当上“有编制的老师”。

东北师范大学谭卓的论文认为，直到2050年，小学和初中不同阶段的本科学历教师需求是在不断减少的；但研究生学历的义务教育专任教师将长期处于短缺状态。小学阶段，本科学历教师的需求数量，在2026-2035年呈现急剧下降趋势；而研究生学历教师的需求数量，在整个预测期间持续增长。初中阶段，专职教师的需求数量也呈现类似趋势。

## 高中教师未来仍在阶段性扩张

高中还在继续阶段性扩张，叠加普职分流政策变化，使得高中教师的需求仍在上升。教育部发展规划司司长提出，要紧盯高中学位不足这一突出问题，新建改扩建1000所以上优质普通高中。中央发布的《教育强国建设规划纲要（2024-2035年）》提到“加快扩大普通高中教育资源供给”，包括“探索设立一批以科学教育为特色的普通高中”、“办好综合高中”、“实施县域普通高中振兴计划”等。

高中的扩张背后，除了学生自然增长外，还有“普职分流”政策的新变化。在全国层面，除了极个别地区外，普职比例已经不再维持五五分流的比例。这意味着，就算学生总数不变，普通高中的学位需求也会迎来一波“小阳春”，原来可能被分流到中职的学生，有机会读上高中。宁波市北仑区的例子显示，2023年普通高中招生2960人，普职比约为5.4:4.6，但在2024年，普通高中招生计划增加到3729人，中职招生计划则只剩1195人，普职比进一步拉大差距，变成7.4:3.6。
