
# 埃尔多安陷入了危机
## 嘉宾：梦回安西

近期，土耳其爆发了针对埃尔多安的全国性抗议，其声势之浩大，不同于以往任何一次。"埃尔多安要被推翻"，成为了西方媒体的热门话题。

那么，统治土耳其十多年的埃尔多安，是否真的要被赶下台了呢？这一次全国性的爆发，直接导火索是3月19日，土耳其警方以涉嫌贪腐和协助恐怖组织为由，拘留了伊斯坦布尔市长伊马莫卢。

伊马莫卢是埃尔多安的最大竞争对手，2019年成为伊斯坦布尔市长，2023年连任。普遍认为，他会在下一次大选中，代表共和人民党出战(土耳其最大在野党)，挑战正义与发展党(埃尔多安所在的党，简称正发党)。

伊马莫卢的实力很强，而正义与发展党这边，情况则不容乐观，由于已经连任过一次，埃尔多安不可能再参加下一次大选，除非修宪。但修宪需要三分之二的国会席位同意，正发党加上盟友都没这个数，希望不大。

如果有别的强劲选手，倒也无所谓，问题在于，正发党现在找不出第二个埃尔多安，能够出战的选手，实力明显不如伊马莫卢。

以正发党这些年干的事，如果输掉大选，他们很清楚会面临怎样的清算。

于是，埃尔多安故技重施，抓了伊马莫卢的同时，还取消了伊马莫卢的大学学位，理由是当年伊马莫卢大学转学时程序违规。

其实被捕还不算大事，取消大学学位才是严重的。因为土耳其总统参选人规则有规定，没有大学学位者，禁止参加大选。

这样一来，伊马莫卢没法参加2028大选了。

实际上，伊马莫卢在2023年大选前，已经被埃尔多安黑过一次了，当时找了个微不足道的理由，将其判了两年，虽然没有真的坐牢。

但有案子在身，伊马莫卢就无法参加2023大选。

人是有路径依赖的，埃尔多安还想来一次。

只不过，这一次和2023年不同。

毕竟，同样的手段用两次，傻子都知道怎么回事，实在过于不要脸。

何况，土耳其近些年经济危机愈演愈烈，尤其是通胀利率的爆炸，动不动就50%，土耳其的货币里拉贬值严重，许多人的生活难以为继。

2024年的土耳其地方选举，正发党就遭遇了惨败，连首都安卡拉都丢了，是十多年来，第一次输掉首都。

土耳其社会，特别是西部沿海地区和城市，积累了对埃尔多安的巨大不满，就像干柴，而伊马莫卢的被捕等于烈火，一点就着。

抗议浪潮迅速席卷全国，声势明显比以往大不少，连埃尔多安的母校--马尔马拉大学，都忍不住参与其中。

共和人民党在3月25日宣布，确认伊马莫卢为2028年总统大选候选人，摆出了要和埃尔多安刚到底的姿态。

埃尔多安也不怂，一出手就是逮捕一千多人。

西方媒体报道十分起劲，一副不推翻埃尔多安，誓不罢休的样子。

但埃尔多安是否会被推翻，取决于大国的反应，尤其是美国的态度。

目前来看，特朗普没有这样的兴趣，为什么呢？

一是战略收缩，不想管事。

二是相比埃尔多安，特朗普更不喜欢共和人民党。共和人民党素来以“大西洋主义者”自居，“大西洋主义者”指的是美国和欧洲的大西洋联盟。虽然我们都知道，土耳其不是西方国家。

但土耳其和西方交往密切。

军事上，土耳其加入北约多年，是北约的中坚力量；

经济上，承接了大量欧美的产业转移，比如欧洲许多制造企业会在土耳其建厂，利用土耳其的廉价劳动力。

土耳其在产业链上，和西方绑定的程度不低。

这一切，让许多土耳其精英认定，土耳其已经是西方国家的一员。尤其是西部沿海和城市中产，这些精英的代表，就是共和人民党。

在意识形态上，共和人民党与民主党是一伙的，其官方意识形态是社会民主主义，也就是“社民派”，类似于德国的社民党。

共和人民党和特朗普是谈不到一块的，比如伊马莫卢本人，对于气候变化的态度就颇为激进。

所以，尽管特朗普不喜欢埃尔多安，但也不会帮民主党做嫁衣。只要美国不下场，单靠欧洲，成事概率不大，埃尔多安不带怕的。

他这辈子别的不一定厉害，唯独在拿捏欧洲方面，堪称炉火纯青。

目前，在土耳其国内还有几百万叙利亚难民，绝大多数是想借道土耳其，前往欧洲的，只是土耳其和欧盟有协议，不能让难民过来。

把埃尔多安逼急了，把边境一松，保准欧盟焦头烂额。

站在中国的立场上，埃尔多安在台上，对我们相对有利(注意是相对)。

虽然在涉疆等问题上，埃尔多安非常恶心，但他不是不懂合作的人。

比如2024年6月，埃尔多安政府突然宣布，将对我们电动汽车加征40%关税。但过了一个月，一些中国厂商表态愿意在土耳其设厂后，关税便取消了。

土耳其的优势是和欧盟有共同关税区，中国厂商在土耳其设厂，从国内进口零部件再组装，可以避开欧盟的高关税。

土耳其还用了不少华为设备，美国一度以禁售F-35战斗机作为威胁，要求放弃华为设备。

埃尔多安反复权衡后，终究没答应。

早在2015年，中土两国就签署了《一带一路合作谅解备忘录》。我们和埃尔多安的关系，属于既冲突又合作，打交道多年，已大体熟悉他的套路。

如果换了共和人民党，意识形态上脑，说不定就不合作了。

在涉疆问题上，共和人民党好不了几分钱，甚至比埃尔多安更恶心，他们经常批评埃尔多安，对我们太软弱。

这意味着，共和人民党上台，土耳其也是要在涉疆问题上恶心我们的，没有区别。

既然两个都是“害”，那么，只好两害相权取其轻。
