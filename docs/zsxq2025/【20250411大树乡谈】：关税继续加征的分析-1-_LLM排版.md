
# 关税继续加征的分析

特朗普在超过100%加征关税基础上继续增加关税，属于黔驴技穷，小孩子胡闹，毫无意义。这不仅说明特朗普手里没牌了，也说明特朗普很可能就没想真的想执行，只是作为一个讹诈的工具，甚至仅仅是为了炒股。因为实际美国出口到中国的商品，如果严格稽查，70%以上再增加就已经没啥实际意义了。

加征关税是一件很严肃的事情，不是拍脑袋说说就行的，几天时间从34%不断往上增加到84%、125%、145%，这是非常幼稚的做法，属于朝令夕改，根本就没考虑如何执行。这跟中国反制截然不同。具体小镇昨天公号《在实干者眼中，特朗普的关税战不过是纸上谈兵》已经说的很清楚了，决策只是开始，重点是执行。

对如此胡闹，中国没必须继续跟，不仅没有意义，还对实际对美制裁工作带来负担，不利于对美有效施压。不仅增加关税需要往上报批，重点是宣布加征之后的执行，中国不像特朗普不过是说大话，中国是真的要执行的，改来改去，海关老是重复工作，不停的对全国下通知、调整系统、跟其他单位协调，改来改去这谁受得了，也太不严肃。

因此没必要陪特朗普胡闹，要把精力用在更有价值的事情上。比如抓住机遇团结全球其他国家、针对美国知识产权等服务贸易发起调查等。

下午4点就宣布了，对等加征到125%以后，表示“鉴于在目前关税下，美国输华商品已无市场接受可能，如果美国后续继续对
中国输美商品加征关税，中方将不予理会。”
