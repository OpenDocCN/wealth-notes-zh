
# 猫哥星球251219：圈层的问题

最近一段时间，向上社交作为成功学最热门的话题是很多人都关心的热点，今天就借着一个舆论热点给大家聊一聊圈层的话题。
前几天一段谈话类的采访比较热，谈话的主角是段永平与王石夫妇，双方都是知名大佬，但是这段谈话被很多人诟病，这里就谈谈几个细节。

首先是身份不匹配，相处就尴尬。
虽然段永平与王石都是颇具影响力的大佬，但是两者的身份或者说身价差距是很大的，段永平的身价大概是千亿级，王石呢？勉强能达到十亿这个标准，双方的身价差距是百倍级别的。这种财富量级的巨大差距就决定了双方即便坐在一起，表面上客客气气，实际上那种阶层的鸿沟几乎一眼就能看出来。

比如，段永平是不大看得起王石的。
虽然王石是老前辈，可能段永平理智也觉得自己应该尊重这个老前辈。但是在段永平的潜意识里，对王石是不大尊重的或者说是看不起的。有两个细节。

细节一，王石夫妇拜访都是规规矩矩穿着正装，而段永平呢？穿这一条短裤就坐上来了，还是在镜头面前，这个穿衣的选择就明显表现出段永平的心态。
大家想一想，假如拜访段永平的不是王石，而是马云或者字节的老板，这些财富量级与段永平差不多甚至在他之上的大佬，而且是在镜头面前的采访，段永平是否还会穿一条短裤出席呢？
绝对不可能！

细节二，王石老婆田朴珺在舆论场口碑不好，我也觉得这个女人是一个草包，绣花枕头一样的草包，但是不管怎么样，田朴珺也是王石的老婆，如果段永平对王石有足够真正的尊重，那么即便田朴珺是一个草包，段永平也应该有三分客气的，但是采访过程中，我看到的是段永平对田朴珺根本就不客气。

段永平与王石谈论当年怎么投资茅台的问题，田朴珺急忙插话恭维段永平投资茅台如同当年巴菲特投资可口可乐，结果直接被段永平很不客气的否认，认为自己投资茅台的逻辑是巴菲特投资喜诗糖果。

这种断然直接否认在这样的公众镜头面前是很不客气的，正常情况下否认不是不可以，应该用一种比较委婉的表达，这种委婉的表达本身就是一种尊重或者说客气，而没有委婉直接否认，某种意义上就是段永平潜意识认为田朴珺不值得尊重。

实事求是的说，段永平还是很有思想的，他能接受采访，很多人包括我自己还是很期待能看到段永平一些有深度思想的阐述，但是这个谈话节目采访者是田朴珺这个草包，田朴珺虽然跟着王石也见过世面，但是思想层次还停留在傻白甜的阶段，所以田朴珺的提问完全掌握不到重点，或者说完全跟不上王石与段永平谈话的境界，段永平大部分回答田朴珺的提问基本就只能讲一些正确的废话。

而王石呢？整个过程唯唯诺诺，精气神状态看起来很一般，要知道巅峰时期王石号称是精神教父的，随便说点啥都被奉为金句的，但是现在老人迟暮，在段永平面前已经缺乏起码的自信，就只能看着自己的草包老婆在那里提一堆傻白甜的问题。

我不知道别人是怎么看的，反正如果我把自己代入王石的身份，坐在那里绝对是一种煎熬，对比巅峰时期甚至可以说是一种折辱。

按道理王石都七十几岁了，财富自由式没有问题的，为何还要天天上班劳累还要受这份煎熬呢？
原因无它，就是王石娶了一个绣花枕头还特别爱折腾的老婆。

为了自己晚年的形象，王石只能把老命都拼上了陪着老婆折腾——比如，厚着脸皮陪着老婆去采访段永平，段永平虽然给了面子，但是双方身份身价差距太大，硬着头皮坐一起也是尴尬。

如果仅仅是私人场合会晤，即便有点小问题大可以一笑而过，但这是在镜头面前也就是在公众审视之下，这就是一种煎熬了。

段永平这个老江湖自然看透了这一切，所以会晤结束之后，对王石说了一句话——对自己好点。

我的理解就是，段永平的意思就是，王石这个年龄了没必要活的这么辛苦，老婆爱干嘛就让她自己去折腾，自己活的轻松一点。

从这个案例中大家有何感悟？
王石与段永平都是几十年的老熟人了，即便如此，因为身价差距太大，坐在一起也尴尬，这种身价的差距总是从各种点滴细节中一点点渗透出来，让身价更低的王石浑身不自在，甚至可以说是一种煎熬。

同样，平时各种成功学大师说的什么向上社交——那有这么容易！人与人之间身份地位差距太大，即便有机会接触，也有一道巨大的看不见的鸿沟，这道鸿沟决定了即便有接触也留下联系方式，未来也很难有交集甚至是合作。

我有一个老大哥，巅峰时期身价有十个亿这个量级，曾经有人打算介绍真正的大佬与之结识(相当于副国级那种大佬)，但是大哥果断拒绝了，原因很简单，就是双方身份地位差距太大了，即便接触，也不可能有什么交集。

所以向上社交是一个循序渐进的过程，只有自己能给大佬带来价值或者说独特的价值——向上社交才有意义，否则都是无效社交——甚至整个过程都是体验很差的煎熬。
