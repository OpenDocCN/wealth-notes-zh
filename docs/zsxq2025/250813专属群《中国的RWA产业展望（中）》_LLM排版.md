
# 中国的RWA产业展望（中）

## 产能上链

中国入局稳定币与RWA催化的加密市场，会给这个世界带来什么影响呢？有几个答案你很快就能反应过来：

首先，全球资产代币化的资产池会扩大，大规模增加中国资产。其次，全球稳定币市场结束100%是美元稳定币的垄断格局。再次，全球稳定币与RWA的合规标准当中，会加入中国方案和中国意志。

而其中第一个影响，促进全球RWA资产池的扩张，我认为又会是最有意思的，打算跟你展开说说。因为这种扩张不光会体现在进场资产的数量上，还会体现在品类和模式创新上。

这么说的逻辑，我在前面的专栏已经跟你汇报过了：中国的充裕产能，其实也可以变成链上的RWA，像碳排放权、充电桩运营权那样，放到合规的加密市场里交易。

让产能成为链上资产，这个思路在脱实向虚的金融市场里很难成立，在中国却浑然天成。因为中国拥有全球接近1/3的实体产能。如果它们能进入全球链上市场，不仅能丰富全球投资者的投资选择，也能帮中国企业老板更有效地盘活资产、降低融资成本。甚至，还有机会帮加密资本和传统股权资本解决资产权益交叉混乱的问题。

我们解释一下这件事，先举个例子看看当前的混乱。

## 股票代币的存在危机

我们前面提过，2025年以来，美国加密市场流行起了“股票代币”：有些第三方公司开个外盘，自己买入一笔苹果、亚马逊这类热门公司的股票来当作RWA，自行发行代币，价格直接锚定苹果和亚马逊的股价，来供区块链玩家们交易炒作。

外盘这个概念，要先给用户里的好宝宝们解释一下。赌场里外盘的参与者们，在赌客押大押小时，也会跟着下注。但他们下注的，不是骰子的点数，而是“下注”这件事本身。他们押的是赌客们谁赢谁输、几轮之后的变局。他们是围绕赌场赌局本身，再开一场赌局。

外盘与赌场之间是什么关系？是高度依赖，但又不完全受控。

股票代币就很像这样的外盘。它不是去直接买卖已经公开发行的股票，而是围绕苹果、特斯拉这类热门公司的股价，再构建一套链上的“镜像赌场”。

股票代币跟股票相比，确实带来了一些新优势。比如，它可以24小时交易，不受交易所开盘的时间限制；它可以把一股再拆成100份，参与门槛更低；它的清算周期是T+0，几秒就到账。

但这事很快惹出了麻烦。比如，苹果和亚马逊的代币，刚上市两周就被炒飞了，价格远远高出了它们所锚定的真实股票。

追踪亚马逊股价的代币AMZNX，价格一度飙升到真实股价的4倍以上。在加密货币平台Jupiter上发生的事情更极端：7月3日，有用户试图购买约500美元的亚马逊代币，结果短暂把代币价格推高到了2万多美元，超过亚马逊前一日股票收盘价的100倍。

这就带来了至少四个疑问：

第一，苹果和亚马逊的市值，到底由股价说了算，还是由币价说了算？第二，代币价格的暴涨，背后有没有发行方和交易平台的操纵？第三，这些代币后面，真的存在足量可兑付的现实资产吗？第四，同一笔资产上市又上链，在股票所有者和代币所有者之间该如何分割？

## 权益分层的厂东模型

那么，RWA想要找到自己的需求场景，还能跟传统股票相安无事，可能吗？

可能。今天的中国RWA市场已经隐约摸到了一个创新模型，我们先把它称作“厂东模型”。

什么是“厂东”？你肯定知道，航运行业有船东，房地产行业有房东。船东，拥有船舶产权，但不一定参与运营，他们可以把船舶租给航运公司，让航运公司去跑船运货。出租船舶，就是在保住船舶所有权的前提下，把船舶的运营权，分时段或分频次“卖”出去。

房东，也拥有住宅、商业地产或工业园区的产权，他跟船东类似，可以亲自拿房开店或亲自招租，也可以把房产的运营权外包给物业公司或商管公司。你把房子委托给链家出租，也就是在跑房东模型。出租房产，也是在保住房产所有权的前提下，把房产的运营权“卖”出去。

那么，把船东和房东的盈利模式平移到制造业，就出现了我们说的“厂东”。中国产业链里大大小小的工厂老板们，他们拥有厂房、仓库、设备、产线，但不一定非要亲自做产品，也可以把产能的运营权“卖”出去。

所以简单来说，厂东模型，就是把资产的所有权和运营权分层，让股权资本去交易资产的所有权，让加密资本去交易资产的运营权。

我们拿特斯拉做个思想实验：

特斯拉的所有权，仍然以股票形态，在纳斯达克平台上交易；而它的产能运营权，比如“上海工厂10GWh”产线的一年产能，可以做成链上资产RWA，对应发行1000万枚产能代币，放在区块链平台上交易。投资者可以购买这些代币，根据特斯拉工厂每月或每天的产量与绿电数据，来获得稳定币分红。

甚至，如果有另一家名叫“大米”的汽车公司，遭遇了产能困境，也可以在链上把1000万枚特斯拉产能代币全买下来，然后像期货商交割豆粕那样，要求交割这笔RWA对应的真实产能。交割完后，这1000万枚代币就可以注销了。一年后，特斯拉工厂腾出产能，可以再次发币。

类似的，在传统的供应链金融里，一台价值1000万的机床只能整租或整卖，而RWA可以把机床拆成1000万个1块钱的代币，让只有10块钱的散户也能买到“机床收益权”。

一条生产线的租金、残值只能靠季度报表呈现，估值非常滞后。但在区块链上，所有产能高度数字化，生产线的开机率、良品率数据都能上链，产能代币们可以通过链上合约绑定这些细节数据，确保分润的公平和可追踪。造假和利差空间也都被压缩了。

“厂东模型”，还真不是我们的异想天开。蚂蚁集团做的新能源资产RWA融资，其实已经走在这条路上了。

我们回想它的几个尝试：

它跟朗新科技合作，把9000多台充电桩放到链上，把它们未来5年的经营收益切成1亿元面额的Token，卖给境外投资人，同时又让朗新科技保留这些充电桩的所有权。

它跟协鑫能科合作，把一批分布式光伏电站的2亿元未来现金流给代币化，同样是只出售经营收益，不转让所有权。

跑通了这个模型之后，你就会明白，蚂蚁后来推出RWA交易平台，把1400万台新能源设备变成RWA，其实就是在帮“工厂老板”转型变成“厂东”，而自己成了某种“链上物业公司”。更多的新能源资产公司，只要把资产放进蚂蚁的“链上物业系统”，就能秒变全球厂东，随时用稳定币收租。

## 内生价值

你肯定记得，巴菲特和芒格这对神仙搭档不喜欢黄金，更讨厌加密货币。他们反感这两类资产的逻辑之一是相通的：黄金和比特币都没有“内生价值”，也就是资产没有自我增值能力。

而“产能上链”，恰恰能完美回应巴芒二人对投资标的物的挑剔：一条汽车生产线，折旧周期内也可以产生出远高于它造价和运维成本的产品；一座光伏发电站，也能在收回投资后继续采光发电。

蚂蚁集团今年有高管解释过，为什么接连几笔RWA融资锚定的都是新能源资产。这位高管说，蚂蚁数科拒绝了很多资产，而新能源所在的中国的高端制造业是非常完美的底层资产，通电联网、数据刻画、数字孪生，从新能源行业到高端制造一定是蚂蚁寻找的方向。

这话翻译过来意思也很清晰：新能源资产是“产能”，有稳定的内生价值，有翔实、不容易篡改的运营数据。

所有的供应链资产，都自带真实贸易产生的稳定现金流。这也是为什么就连宗氏家族的财富之争，焦点之一也是争夺娃哈哈集团的供应链现金流：一方借助供应链分红来不断给家族信托充值；另一方则在不改变集团股权的前提下，逐渐把高毛利的供应链资产转入己方阵营。

## 总结

这是站在香港试水稳定币、RWA概念也在中国金融圈流行的2025年，我想要提前为你开出的脑洞。

如果这次加密世界与现实世界的融合成功，对我们这些产能大国的普通公民来说，未来几年我们可能有机会看到，全新的金融时代拉开帷幕，也看到厂东模型快速覆盖各种供应链领域。

作为普通投资者，我们也许就能通过数字钱包直接购买“美的智能工厂180天现金流”“华为供应链资产包”“京东存货抵押资产包”等等，以非常低的认知和资金门槛，去参与全球优质资产投资。

这种模型对中国企业来说，也意味着更低成本、更高效、更灵活的融资渠道，供应链上下游的中小企业将获得更多发展机会，当然了，也能迫使低效产能加速出清。

这两讲可能有点令人挠头，但相信我，事情的起点就是理解整件事最容易的时机。
