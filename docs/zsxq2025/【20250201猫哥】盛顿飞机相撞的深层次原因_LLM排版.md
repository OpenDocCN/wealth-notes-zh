
# 华盛顿飞机相撞的深层次原因

美国东部时间2025年1月29日晚，美国太平洋西南航空公司一架载有64人的庞巴迪喷气式客机，在华盛顿里根国家机场降落过程中，与一架载有3名军人的“黑鹰"直升机相撞后坠河。两架飞机直接相撞的事件极为罕见，更别提发生在美国首都，简直令人难以置信。但这看似偶然的事件背后，却有着诸多深层次原因。

关于事故原因，已有一些初步分析。从网上视频可见，两架飞机各自的航行灯相互接近，直升机切入了客机的降落航线，从而导致相撞。机场塔台与直升机的通话记录也已公布。两机相撞前不到30秒，塔台曾询问直升机是否看到前方的客机，直升机飞行员回答看到了。

既然已经看到，直升机为何还直接撞上去?美国媒体给出诸多解释，比如可能是直升机误判了两架飞机的距离和速度，也有可能是直升机飞行员看错了飞机。但这些说法都未触及这次事故的本质，看似偶发的事件，实则有其必然性。

首先，美国空管部门有很大责任。从这次塔台空管人员没能全力避免碰撞就可以看的出来。美国媒体指责塔台空管人员不够专业。塔台通过空中感知系统预见到两架飞机有碰撞风险，并联络了直升机。

但空管人员语气过于平和，既没有发出具体指令，也未引起直升机足够重视。很可能是他们认为黑鹰直升机飞行员有足够能力避免碰撞，毕竟此前黑鹰飞机和驾驶员被美国军方吹嘘得神乎其神。

而且当时机场通讯频道拥堵，空管人员担心过多信息传递会导致混乱。或者未能全面掌握态势，未意识到碰撞的可能性，也许是当时空域内飞机分布复杂，他们难以准确预判。

还有一个严重的问题是，机场空管人员人力不足。根据美国民航局2024年人力评估报告，华盛顿空域的日均流量较2010年暴增180%，但空管员数量反而减少了15%，平均每人每分钟需处理2架次航班。这相当于让空管人员每分钟都处于高度紧张的“死亡游戏中"。

国际民航组织建议，每名空管员同时处理的航班不应超过5架次，而里根机场却常态化要求每人监控8至10架次。据美媒报道，美国空管人员的缺口在3000人左右。具体数据显示，过去五年中，因空管人员不足导致的航班延误事件平均每年增长20%，这不仅影响了旅客的出行体验，也增加了航空安全风险。

不管是专业度不够还是人员不足，那都是管理部门无能的体现。

特朗普和万斯双双出席了飞机相撞事件的新闻发布会。

别有用心的记者质疑特朗普对美国航空管理局的清洗，促成了这次空难。1月20日特朗普正式就任总统，当天美国联邦航空管理局局长就辞职。

次日，空中管制人员就冻结招聘，紧接着航空安全顾问委员会解散。也就是说，特朗普没有解决人员短缺的问题。

为了转移矛盾，特朗普在发布会上抓住空管专业性不足的问题大做文章，把奥巴马和拜登推行的 计划批得一无是处，称该计划导致美国联邦航空局为兼顾招聘不同人群而降低标准，雇佣了大量“能力不足”甚至“有精神问题”的人员。

事故发生后，他要求联邦航空局更换不符合资格标准的人员，还审查过去四年的所有招聘协议和安全协议。这哪里是在解决问题，分明是在甩锅和攻击民主党。

但无论是共和党还是民主党，美国空管部门的管理混乱却是长期存在的。

其次，美国军工复合体的三宗罪才是导致事故的更为关键原因。

第一宗罪：美军飞机随意进入民航航线。

美国军方人士称，华盛顿及周边地区政府、军事单位多，军用飞机执行任务频繁。此次涉事的“黑鹰”直升机在执行夜间避免与其他飞机相撞的训练任务，结果本是为避免相撞搞的训练，却偏偏发生了相撞事故，实在荒唐!

那么，军方是否有权随意进入民航航线?美国《民用航空法》《国防授权法案》《空中交通指挥和控制协议》和《紧急空域》规定，在紧急情况下，军用飞机可以优先通过民用空域，尤其是在执行战斗任务或紧急救援时。

然而，路透社公布的一份报告显示，此次出事的华盛顿里根国家机场，每年出动军用直升机的次数高达1万多次。显然，其中绝大部分都不属于紧急情况。正所谓“常在河边走，哪有不湿鞋”,这次就出事了。也就是说，正是由于美国军方毫无底线的肆意妄为，才最终导致这场悲剧。

2022年，美国军方因军机随意乱入民航线路被质疑。美国国防部长奥斯汀在听证会上承诺严格执行民航避让原则，可结果呢?前里根机场空管员玛丽莎 ·吴透露，军方随时会用红色电话“劫持”空域，民航方面连抗议的时间都没有，这所谓的承诺岂不是成了空头支票?

据相关统计，在过去的十年中，美国因军用飞机与民用飞机发生冲突或险情的事故多达数十起，其中不少事故是由于军用飞机未经许可进入民航航线造成的。

这些事故不仅给人员生命和财产带来了巨大损失，也严重影响了公众对航空安全的信心。

例如，此前曾发生过军用飞机因未提前通报而惊扰民航航班的紧急情况，导致乘客恐慌，航班被迫紧急调整航线。这种军机随意进入民航航线的情况，是导致此次事故的潜在风险因素之一。

第二宗罪：美国军工企业故意推迟合同，导致黑鹰直升机未安装ADS-B系统。

所谓ADS-B 系统，就是在两架飞机有可能相撞的时候，系统会给飞机发出警告和指令。如果这架黑鹰直升机接收到系统相撞的明确指令，就能及时规避碰撞，因为那架庞巴迪喷气式客机已经进入了落地降落的稳定飞行路线。

为避免飞机在空中相撞，中国民航局早在2020年就全面实现了军民航ADS-B 信号100%强制接入。并且，在系统的后续维护和升级方面，中国也投入大量资源，确保系统能够稳定、高效地运行。

那么问题来了，为什么在首都服役的黑鹰直升机都没有安装这套系统呢?
