# 川普对欧盟加50%关税，美欧贸易战开打？

文/卢克文工作室嘉宾 云梦竹

特朗普23日在社交媒体发文称，他建议从6月1日开始，对来自欧盟的商品征收50%的关税。他表示，欧盟成立的主要目的就是“在贸易上占美国的便宜”，美国与欧盟的谈判“毫无进展”。

当天，欧洲股市出现震荡，英国富时100指数下跌0.24%，法国CAC40指数下跌1.65%，德国DAX指数下跌1.54%。

26号，特朗普又表示，在与欧洲进行友好会谈后，决定将达成协议的最后时限，延长至7月9日。这一轮关税风波也算是短暂度过去了，无论特朗普本次发言是蓄谋已久，还是一时兴起，欧洲算是彻底输麻了。

## 1. 欧洲战略失误

在5月举行的G7会议上，美方官员披露，美国财政部长贝森特等官员甚至在私下给其他各国提出过“统一对华关税政策”，企图联合盟友对华施加压力，但很多国家都没接茬。而在很久之前，这就是欧盟所设想的最好状态。

在欧洲的战略规划中，欧洲一直寄希望于和美国一起对中国施压，再以盟友的身份与美国进行贸易谈判，也叫祸水东引战略。

但欧洲的本意不是要像美国一样，试图压制中国制造。而是借着中美的贸易冲突，扩大欧盟的利益，也就是鹬蚌相争，渔翁得利，想让美国替自己“打工”。

根据欧洲一家经济研究所的观点，如果美国继续推动激进的关税政策，那么各国在评估与美国的经济风险后，可能会选择美元之外的货币进行结算，而欧元在这个阶段，结算份额将会进一步上涨。

4月14日，在中美贸易战交战正酣时，欧洲选择与中国签订电动车关税协议，给了中国一个巨大突破口，也是欧洲战略的具体体现。

欧洲本来计划是趁着美国产品因为高关税而失去中国市场的契机，快速用自家产品，填补市场空白。结果没想到，仅仅一个月后，中美日内瓦会谈达成了协议，让欧洲借助中美相争，自己牟利的计划化为泡影。

对于欧洲来说，其上策是与中国一起，跟美国展开拉锯，双方都不对美国妥协，然后美国自然会拿出更好的条件来拉拢自己。这样一来，中欧都能攫取巨大收益。

中策，是在中国与美国达成协议之前，欧洲先一步，分别与中美达成协议。这样一来，在中美激战之际，双方都需要给欧洲做出巨大让步。

而现在，尴尬了，欧洲既没有与中国对美国施加压力，也没有与美国一道，组成制裁联盟，在中美达成协议后，欧洲在谈判中的地位直线下降。这才出现了今天这种局面。

## 2. 直接原因

此次特朗普之所以发怒，是因为欧洲没有按期达成新的贸易关税协议。

而欧洲方面的解释也很清楚，欧洲已经在几乎所有关键问题上让步，除了小部分的敏感农产品领域之外，几乎所有的工业与农业领域，都执行0关税。

同时，欧盟还愿意在人工智能等领域与美国进行合作。甚至包括放宽数字监管以及共同努力遏制所谓中国“产能过剩”。虽然欧洲内部达成了协议，但最终签署需要欧盟27国一致同意，这不是短时间内可以完成的。这个欧盟体制多年以来的弊端，也被欧洲各学者多次提及。

27国一致同意的民主原则，要求欧盟在处理这种复杂问题时，必须顾及内部所有国家的感受。而要命的是，本轮特朗普计划征收关税的项目，总计近5000亿美元，占欧盟对美出口总额的97%。如此大的范围，涉及的利益冲突将会更加剧烈。

这里打个比方，对于德法这种产业出口国，肯定希望尽快达成协议，规避关税风险。比如德国大众，每5辆“德国制造”就有1辆销往美国，而美国市场贡献了大众集团27%的净利润。更致命的是，欧盟汽车产业链高度依赖美国半导体——一颗车载芯片被加税10%，整条生产线成本就要跳涨3%。

对于德法这种工业国而言，无论是美国对欧盟增加关税，还是欧盟对美国增加关税，都会极大增加企业整体成本。但如果想要达成协议，也不是那么简单。

美国的需求中就有一点，要求欧洲放开农业市场，对于以农业为主要收入的国家，则必须顾虑美国农产品的冲击。

各国不同的经济结构，让欧洲在面对这种问题时很难达成统一意见。如果欧洲放开监管，让美国农产品进入欧洲市场，那么以出口工业产品为主的西欧各国，势必要给农产品生产国足够补偿，才能弥补这些国家的损失。

这里面的利益协调，肯定不是短时间内可以完成的。这仅仅只是一个例子，在无数的征收项目中，这种例子可能有成百上千例。所以，哪怕特朗普将期限延长了一个多月，情况依旧不乐观。

大家可能不明白了，为什么欧洲做出这么大的让步，既愿意全面放开国内市场，又愿意在科技领域，配合美国政策。但特朗普依然紧追不舍。其实，这就是商人治国的技术手段——逼单。

在当前协议有利于美国的情况下，特朗普希望利用设置最后期限的方式迫使欧洲尽快签署。销售产品的过程中，往往会以折扣取消等方式，迫使客户尽快签约，避免夜长梦多，本轮征收关税，可以看做销售在逼单。

## 3. 结尾

在前文我们提到，美国财政部长贝森特邀请各国参与“统一对华关税政策”的联盟，但各国都没有回应。哪怕是如此期望与美国搞好关系的欧洲，在这种问题上也非常避讳。这也体现了，欧洲各国对于特朗普政策的不信任。

在理解特朗普的决策前，首先需要考虑一个问题，特朗普的决策理智吗？

2019年7月，英国外交部秘密电报被媒体泄露，其中驻美大使金·达罗克爵士用最犀利的语言，描述了特朗普本人的无能与幼稚。

他告诫伦敦当局，在与特朗普打交道的过程中，需要赞扬他最近的行为，以攫取好感，才能让事情顺利推进。并且观点要简单且直白，不要有弯弯绕绕，因为特朗普无法理解过于复杂的问题。

这篇报道很快成为了反特朗普的武器，也让英国外交部成为众矢之的，特朗普明确表示将不再与达罗克男爵打交道，达罗克男爵被迫辞职，并于当年从英国外交部退休。

但实际上，现在无论是沙特还是俄罗斯，都敏锐地捕捉到了这一点。无论沙特还是俄罗斯，都会在谈到各方攸关的利益问题时，给特朗普戴明显的高帽，在之前俄罗斯关于俄乌谈判的公告中，先指出特朗普说了什么，再写俄罗斯做了什么，虽然都是一些客套话，却给足了特朗普情绪价值。

相反，泽圣在白宫甚至连谢谢都不说，就很不礼貌了。

对于特朗普的敌人们来讲，他们已经摸清了如何与特朗普打交道，反倒是美国的盟友们至今不清楚。

就如同泽圣不会相信美国会在支持乌克兰多年后，政策出现180度大转弯一样，欧盟也绝对没有想到，美国可以在与中国撕破脸皮后快速达成协议。

事实上，欧洲鹬蚌相争，渔翁得利的战略非常精妙，但唯独漏算了一点，精妙的计划往往是脆弱的，因为任何变量都会对计划产生影响，而特朗普本人，就是一个巨大的变量。
