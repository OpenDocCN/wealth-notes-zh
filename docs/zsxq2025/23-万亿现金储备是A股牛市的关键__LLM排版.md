
# 23 万亿现金储备是A股牛市的关键

## 市场
A $23 Trillion Cash Pile Holds Key for Chinese Stocks’ Bull Run

中国股市的上涨势头有望获得散户投资者的助力，他们庞大的储蓄将推动市场迎来新一轮上涨。
中国月家庭存款从月的历史高位回落，降至160.9万亿元，表明投资者正将资金投入运作。
摩根大通公司预测，在年月至明年年底期间，约亿美元的额外储蓄可能流入股市，推动股价上涨逾。

中国股市的上涨势头有望获得散户投资者的助力，这引发了市场对散户巨额储蓄将推动股市下一轮强劲上涨的期待。
基准沪深300指数持续飙升，8月涨幅达10%，在流动性驱动的上涨浪潮中成为全球表现最佳的股指之一。尽管对冲基金活跃于市场，但分析师指出，国内散户投资者可能正处于重大资金轮动进入股票和股票基金的初期阶段。

## “Cash makes bull markets, and deposits shifting to stocks is going to be an important driver of this rally,”
北京金通私募基金管理公司基金经理徐大伟表示："资金是市场的核心驱动力，存款转向股市将成为本轮上涨的重要推手。这一趋势已然启动，且不可逆转。"

储蓄过剩是推动华尔街银行上调中国主要股指目标价格的因素之一，也助长了市场对A股涨势将延续的预期——尽管企业盈利疲软且经济健康状况持续存疑，但A股迄今仍逆势上扬。
高盛集团策略师在调高中证300指数目标值时，特别指出家庭储蓄过剩的现象。该行前预测未来12个月该指数将上涨约10%。汇丰控股在调高中国两大股指目标值时，则将储蓄池视为潜在的"极强劲催化剂"。

## 错失良机
28岁的北京科技公司职员毛达尔文自去年九月起，就一直考虑转向股市投资。
当时，中国央行的刺激政策闪电出手，令股市飙升，终结了因经济担忧而持续多年的抛售潮。沪深300指数一周内跳涨约25%，引发本土投资者争相抢购。直到今年八 月，该指数才突破了当时创下的高点。

"股市涨得太快，我没来得及买入"，毛先生表示，这次他决心不愿错过良机。"七 月底我趁机投入了部分闲置资金，此后一直 在增持仓位。我认为这轮涨势将持续到今年年底。"

中证300指数在过去10周中有9周上涨，较今年4月初的低点累计涨幅达25%。投资者对当局将在9月3日阅兵前持续提振市场情绪表示信心，此次阅兵旨在纪念二战结束80周年。中国历来在重大政治活动前支撑股市，以展现稳定形象。

## 中国股市暴涨万亿引发投资者风险担忧。徐文慧解析这场股市狂热的驱动因素，并探讨这轮牛市是否好得令人难以置信。

部分策略师已指出市场过热迹象，若干技术指标正发出超买信号。例如，神威芯科技公司股价在8月翻倍上涨，促使这家人工智能芯片设计公司警告投资者其股价可能已脱离基本面。该消息导致该公司股价周五暴跌。

迄今为止，储蓄转向股票的趋势仍属涓涓细流：7月份非金融存款（作为股票、基金和信托账户流动性的替代指标）约2.1万亿元的增幅，仅为2月以来的最高值，且仅略高于过去十年的季节性平均水平。

但分析师认为，股市正受益于"别无选择"（TINA）环境的推动。债券收益率处于历史低位，而房地产——这个曾是中国民众致富首选的投资渠道——仍未从长达数年的低迷中复苏。中国大型银行的 一年期定期存款利率现仅为0.95%，创历史新低。

"中国可投资资产存在短缺。"美银证券中国首席策略师吴文妮表示："若股市能产生明确的盈利效应，投资者将愿意投入更多资金。"
