
# 体制内女博士当选调生2年后的职业选择

## 个人情况
强势大学博士毕业，女，30岁，传统东部省份非省会城市的选调生，出身普通，努力上进，有职业理想和追求，今年是工作第二年。

毕业时综合考虑性别、年龄、家乡、个人发展等因素，放弃北京、上海等省份，选择选调回家乡，在某市直部门任职（专业性强，与所学专业对口，但面向较窄），是市里第一个毕业于该大学的博士，在哪里都被人高看一眼。

选择原因：虽然在北京上学，但临近30岁的外地女性，家庭条件一般，不是学校本科土著，在北京并无竞争优势，唯一的学历优势无法发挥，加之不想在专业对口的末流部委工作一辈子（级别层次越高越难跨部门专业），所以才向下走，做出了自认为是扬长避短的策略选择，走上了很多人没想到、也不理解的路。

## 具体困惑：两年基层锻炼期满后是否回到原单位

选调基层锻炼满两年后，有一次选择机会，可以回到原单位，可以留在乡镇。

原单位的优势：专业对口，比较舒适，离家近，而且专业优势发挥空间大，不可替代性强，短期有利。

问题：接触面比较窄，呆得越久越难出来，担心一旦进去再也无法出来，而且能力锻炼也不够全面，长此以往，经验阅历难与同龄人竞争。如果这样，还不如一开始去省城。

乡镇的优势：在学校的、学历的加持下，至少能给一个党委副书记，算是开始走上领导岗位，能真正做些事；乡镇更综合，锻炼更全面，以后去哪里发展限制都少，算真正逃离专业的桎梏：名校女博士自愿留在乡镇的少之又少，可能会被推为标杆，有一定红利，而且听说越往上，拥有乡镇主要领导岗位任职经历的女性越稀缺，长远似乎更有利于个人成长和发展。

问题：担心层次太低，限制了格局和眼界，按触大领导的机会又少，加之本身没有靠山，存在“折”在基层的可能。毕竟我所在的市不像福建、河北等，能明显感觉出对名校的重视，也没有出台政策保障名校选调生的发展路径和速度。
