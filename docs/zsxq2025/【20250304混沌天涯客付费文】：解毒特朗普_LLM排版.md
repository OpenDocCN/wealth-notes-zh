
# 解毒特朗普

原创混沌天涯客 2025年03月04日15:51 加拿大

川普的偶像是谁? 欲了解川普，必先答这个问题。偶像的成立，是需要带点幻想的，最迟不能越过想入非非的青少年时期。川普1946年生人，1966年已经20岁，所以他的偶像不可能是里根。里根80年代当总统的时候，川普已是个成功的商人，内心里，一个成功的人，是不愿意崇拜同时代另一个成功的人。世所公认，里根是打赢冷战、将庞然大物前苏联推向解体的罪魁。他1989年1月卸任总统，8年执政，将前苏联拖得眼看要散架；而这8年，也是中美自40年代末尾以来最浓郁的蜜月期。

如果你能使用山寨版老百度的话，不妨搜一个关键字"里根中国",出来的第一张照片，就是里根撑着伞，与一位中国老乡手挽手，哈哈笑着漫步在草地。这画面让我想起诗人戴望舒的“雨巷”。“撑着油纸伞，独自彷徨在悠长，悠长又寂寥的雨巷。我希望飘过一个丁香一样地结着愁怨的姑娘。”联中扛苏，里根是贯彻最彻底的，为此他释放出最友好的待遇，大范围对华放开技术转让，包括军事科技。这一点是之前和之后的所有美国总统都没做到的。长年以来，我们继承和发展的都是苏式科技，当美式科技如旋风般横扫80年代的大街小巷，普罗大众震撼，执牛耳者愁怨。震撼与愁怨相对冲。终于，在里根卸任总统几个月后，与他一同撑伞走过草地的老乡，独自彷徨走进了悠长又寂寥的雨巷。

所以川普的偶像绝不可能是里根，自第一个任期开始，他就对苏老大之继承者俄大哥极力靠拢，反而是在贸易上、技术上对东大竭力压制。第二个任期更是变本加厉。甫一上台，先是施压巴拿马，抢夺巴拿马运河控制权，逼其退出东大大蓝图。又加关税，加拿大和墨西哥是陪衬尚未实施，对我方两次10%的关税已结结实实上路。更重要的是他在以最快速度从欧洲抽身，甘冒天下zz 正确之大不韪。白宫那一起吵架全世界都看到了，川普努力了，是小泽不想签。接下来小泽想继续干，那是欧洲的事情了。果然，小泽第二天就飞到了英国，与英国首相亲切拥抱，头一次获得查尔斯国王接见，并与欧洲十余个国家一起，达成了援乌新协议。欧洲的事，他们爱怎么玩就怎么玩吧。川普小爹马斯克甚至放风，要退出北约。中立 于欧洲，到东方插手，川普一党所作所为，学习的是他真正的偶像，西奥多 ·罗斯福。

美国立国二百多年，头一百年里重心是在东海岸，对着大西洋。是西奥多 ·罗斯福，意识到太平洋的重要性，促使巴拿马独立，开工巴拿马运河。有了这条运河，美国东西海岸资源得了全面贯通，是战略向东方转移的基础。川普发现，如今的巴拿马已被中资占领，他当选后即开始施压，没几天功夫就夺回了运河主导权。

川普向西奥多 ·罗斯福学习的第二件事是，强势介入战争调停。只不过120年前的罗斯福总统，调停的是另一场日俄战争。那场仗是在东北打的，清朝名义上保持中立，实则是暗中大力支持日本。尤其是民间，无论是大V还是小网红，都对日本这个东方后起之秀能击败欧洲老牌列强俄罗斯赞不绝口、体液横流。

向日本学习， 一大批爱折腾的人开始东渡日本，其中有孙中山、黄兴、蒋介石、汪精卫，也有一心想给国人治病的鲁迅先生。我们在中学课本里都学过那篇《藤野先生》，鲁迅在学医的过程中看日本的新闻片，里面有日俄战争的场景，一名东北老乡给俄国做奸细，被日军抓住要枪决示众。围着看热闹的，也是一圈东北老乡。这深深刺痛了鲁迅，让他明白身体健康虽然重要，但心理健康更是重中之重。为此他弃医从文，走上了百年文豪之路。

请注意，日俄战争的停止，就是西奥多 · 罗斯福一手促成。他召集日俄代表在美国的新罕布什尔州朴次茅斯港进行和谈，连哄带骗，威逼利诱，让两国在《朴次茅斯和约》上签了字，为此西奥多 · 罗斯福获得了第二年的诺贝尔peace 奖，奖励因他一人之力挽救了无数生命。这些生命，有俄国人，有日本人，但更多的应该是东北老乡。因为仗打得实在太激烈，旅顺会战，辽阳会战，奉天会战，巨炮对轰，机枪横扫，最先进的下濑火药大放异彩……每一发炮弹响起，都要带走无处可躲的无辜生命。尤其是奉天会战，25万日军vs37万俄军，在一马平川的东北平原拼命厮打，短短20天就有十数万死伤。这可都是日俄两方的精锐部队，如果让他俩继续打下去，从沈阳一直打到哈尔滨，即便有一方最终赢了，那也是惨胜。必然元气大伤，没个十年翻不起大浪。

两败俱伤，渔翁得利，这正是晚清外交策略的精髓。从李鸿章开始就擅长以夷制夷，最成功的例子是《马关条约》之后的三国干涉还辽，本已割让出去的辽东半岛在俄、法、德三国干涉之下，又还了回来。但这也造成了俄国在东北的做大，尤其是几年后赶上义和团，俄国以保护侨民为借口，调动20万大军侵入东北，从哈尔滨到沈阳再到锦州，一路烧杀抢掠，迅速占领了东北全境。来了，他就不想走。义和团平息后，清朝与俄方签了《交收东三省条约》，签是签了，但俄方找各种借口推脱，一直到1905年俄日战争爆发，依旧牢牢控制着东三省。

怂恿日本来打俄国，是清朝的外交谋略。赶跑一只狼引来一只虎固然不妙，但如果让狼和虎打得两败俱伤，不失为让大清获得喘息得以发展的好机会。确实是打上了，大打出手，但是才打了一年半，两方正火气四溅，就被西奥多 ·罗斯福给调停了。这直接导致，狼还是狼，虎还是虎。
