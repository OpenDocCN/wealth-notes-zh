
# 拜登四年

拜登从一个将川普拉下台的变革性人物，变成了两届川普任期之间的过渡性人物。曾经的意气风发，如今只剩下失意，甚至悲情。

《纽约时报》去年12月发表报道记录拜登在白宫的最后时光，标题是“疲惫、虚弱与感伤”。

本期新闻实验室会员通讯，我们来回顾拜登这四年，并聚焦到他与媒体的关系之上。

## 在这个需要罗斯福的时代，他只是拜登

在拜登位于白宫的椭圆形办公室，壁炉上方挂着富兰克林·罗斯福（FDR）的肖像。从1933年到1945年担任美国总统的富兰克林·罗斯福，是史上最重要的美国总统之一。他开启的“新政”（The New Deal）让美国人摆脱经济大萧条，其后又带领美国赢得二战。拜登希望自己的政治遗产可以接近富兰克林·罗斯福—至少从刚上任的时候看，疫情中的美国一定程度上经历着和大萧条类似的危机；而“终结川普”、“挽救民主”也被很多人认为是极为重要的任务（或类似于一场战争）。在竞选的时候，拜登就屡次致敬罗斯福，并认为罗斯福重视工人阶级、保护弱势群体的理念与自己一致。参议员桑德斯在拜登上任之后不久便表示，拜登曾经告诉他，自己会成为“自FDR以来最进步主义的总绡”。

拜登的确按照自己的理念做了不少事情。如《纽约时报》的政治newsletter总结的：他是第一位现场参与工人罢工活动的现任总统；他免除了500万人的学生贷款债务；他的政府采取积极措施遏制企业权力。

但比起做了三任多总统的罗斯福，拜登显然没有那么多的时间展开他的变革。而拜登和他的副手贺锦丽未能阻止川普卷土重来，从根本上说，又与拜登在执政能力上远不如罗斯福有关。

## 甫一上任，拜登的确做了不少重要的事情

其中最重要的事情，是在2021年3月推出美国救援计划（American Rescue Plan）。这项1.9万亿美元的计划，旨在为受疫情影响的家庭和个人提供经济援助，加快从疫情引发的经济衰退中复苏。除了刺激性支出外，该法案还发起了一场迅速的疫苗接种运动，使社会交往重新变得安全。

除了疫情救济之外，他的任内还大力投资基础设施建设、制造业和清洁能源。他还确保通过了旨在抑制老年人处方药费用的立法，包括胰岛素每月35美元的上限。

然而，拜登的支出法案在刺激经济的同时，也助长了自吉米·卡特总统任期以来最高的通货膨胀率，并使他的政府以及贺锦丽的选举都背上了最大的包袱—面对飞升的物价，选民们很不满。而民主党人在这样的情况下依然将经济作为拜登任内的亮点宣传，也被认为是输掉大选的原因之一。

## 拜登的致命问题

罗格斯大学研究美国历史的学者David Greenberg撰文对拜登的这四年提出了更尖锐的批评。他说：“归根到底，拜登习惯性地无法全面衡量和解决问题：在新冠疫情之后，他看到了振兴经济的必要性，却没有考虑到通货膨胀的危险；他为乌克兰和以色列提供保护，却没有预见到这些战争可能发展的方式；他坚定地认为川普威胁到了民主，却没有采取任何措施来加强对民主的保护。拜登总是兢兢业业，但从未有过远见卓识。在一个更加和平的时代，他可能会成为一位出色的总统。但在当前这场需要罗斯福的危机中，他只是一个拜登。”

而《卫报》的一篇文章则认为：拜登的致命问题在于，不知道放手，明明身体状况已经不可能再支持做4年总统，但直到一场灾难性的辩论之后才考虑退出，然后贺锦丽匆忙上阵。如果 他更早退选，让民主党内部可以有自由公平的竞争，然后让胜出者可以有充分的时间展开竞选，结果会不会不一样呢？

## 拜登本可以为媒体做些什么

David Greenberg的文章中提到：拜登以及民主党人认为川普是民主制度的威胁，但是却没有做任何有意义的事情来保护民主。

如何理解这样的说法？我们可以从媒体这个更具体的视角来切入。正好，《哥伦比亚新闻评论》最近的一篇文章谈的就是：拜登本来可以为媒体、为民主做些什么，但他错失了这些机会。在离职之前，拜登在讲话中表示美国的自由媒体正在崩溃，但是在任期内并没有做什么来改变这一趋势、挽救媒体。“往好了说，这是一个错失良机的故 事；往坏了说，这是一个赤裸裸展示虚伪的故事。至少，我们很难断定拜登曾把阻止媒体 的崩溃作为首要的任务。”

为了证明这样的观点，文中细数了拜登任内在媒体领域的成与败。

在他任内，两位被俄罗斯关押的美国记者——《华尔街日报》的Evan Gershkovich和自由欧洲电台的Alsu Kurmasheva重获自由。但是，他们是在美俄换囚计划中被释放的—也就是说，他们被释放的代价是美国也释放了几名俄 罗斯囚犯。《哥伦比亚新闻评论》的文章认为：这可能会给普京等人释放一个错误的信息，让他们认为可以通过逮捕美国记者的方式来挟持他们为人质，以救出俄罗斯方面被美国关押的人士。也就是说，这两位记者的被释放，并不意味着在俄罗斯等国工作的美国记者变安全了—相反，他们可能面临更大的危险。

在他任内，拜登并未将新闻自由置于外交政策的重要位置，他不愿为新闻自由和记者安全的议题去得罪美国的盟友。2021年2月底，拜登政府发布报告，公开了一个早已被很多人知道的事实：2018年遇害的《华盛顿邮报》记者Jamal Khashoggi是在沙特王储Mohammed bin Salman（简称MBS）的批准下被杀的。然而，拜登政府拒绝制裁MBS本人，理由是沙特是美国的盟友。这一举动在新新闻界引发了大量批评，人们指责拜登让一个杀人凶手就这么逍遥法外了。

2022年，以色列军队杀害了美国公民、半岛电视台记者Shireen Abu Akleh。但是，拜登政府并没有采取什么强硬的回应。本轮加沙战争开始后，许多记者不幸遇难，但拜登政府同样因为和以色列的盟友关系而选择睁一只眼闭一只眼。在国务卿布林肯任内的最后一次记者会上，有几位记者高喊“罪犯！”，指责拜登和布林肯为以色列提供大量武器与外交支持，并最终被强行带离了房间。

而在美国国内的新闻自由议题上，拜登政府为媒体提供了一些保护。他的司法部长发布了一项新政策，为通过泄密的内部信息进行报道的记者提供保护，避免他们的通信记录被法庭传唤获取。然而，这只是一项政策而已，川普上台后很容易就可以将其推翻。

《哥伦比亚新闻评论》的文章指出：拜登政府并没有将对新新闻媒体的保护写进法律里面。2024年11月的大选结束后，媒体界对于第二届川普政府可能对新闻自由带来的冲击产生了担忧，他们敦促拜登政府尽快通过《新闻法案》（PRESS Act）——该法案已在两党支持下在众议院获得通过，它将避免记者被迫交出消息来源。然而，法案在参议院被搁置，其中固然有共和党参议员的阻拦，但民主党参议员们并未将该法案作为工作的重点。实际上，该法案在2021年就被第一次提出了，当时民主党还掌握着参众两院，但从未积极推动它。

文章还认为：拜登本来可以推动一项向地方新闻机构提供联邦补贴的积极计划，挽救地方媒体，或者推动成立某种委员会来重新想象媒体的商业生态可以怎样重构，让更多媒体可以经营下去。但是，他都没有做—当然，他可以说自己有更重要的 事情要做，但媒体与新闻自由真的没有基建和清洁能源重要吗？

或许，他在去年大选之前真的认为，川普的政治生命已经结束，美国的民主和新闻自由不再面临那么大的威胁，所以也不需要通过法律等更积极的方式去保护。

但是他估计错了。在他离任前的演讲和采访中，拜登屡次提及：自由的新新闻界正在崩溃，可靠的记者编辑正在消失，社交媒体放弃了事实核查，真相被权力和利益的谎言所掩盖。他还认为，已经形成了一个“科技—工业复合体”，几个极为富裕的寡头掌握了巨大的权力。

这些话在新闻界听来，并没有产生心有戚戚的感觉，而是觉得：早知如此，你在过去四年是不是该多做什么？这个“科技—工业复合体（tech-industrial complex）”，在拜登卸任的时候显然比他上任的时候变得更强大了。而新闻业，在今天显然也比四年前更虚弱了。

## 未能复制“炉边谈话”

回到拜登对成为第二个富兰克林·罗斯福的渴望—其实，拜登不仅在政策上远不如罗斯福，他在另一方面也远远输给了罗斯福，那就是公众形象。

David Greenberg的文章中说：“拜登令人抓狂的低调公众形象使他无法提供时代所需要的有力的公共领导。毕竟，富兰克林·罗斯福不仅实施了为期100天的全面改革，还为我们带来了‘炉边谈话’——他口语化、雄辩地向全国发表广播讲话，这是总统领导力的一项突破，是在窘迫时期凝聚团结、鼓舞士气的工具。川普的第一个任期之后，美国同样需要一场持续的运动来击退政治中弥漫的撕裂、阴谋论和仇恨煽动。拜登没有，或者说无法发起这样一场运动。”

他所说的“低调公众形象”，也是拜登令记者们非常不满的地方：和前任总统们相比，他很少坐下来接受媒体记者的采访，举行记者会的次数也比前任少。他虽然和记者们有互动，但大多数时候只是 在非正式的、转瞬即逝的场合聊两句。比如，最近访问非洲的时候，他对记者提问的回答加起来只有14个词。而南美洲访问时，他的回答只有一个词。

总统接受媒体采访，是 对媒体做出好内容的一种直接支持。因为这样的采访并不是终点，而是可以 是更重要的报道的起点。

在主流媒体之外，拜登也未像川普那样获得许多内容创作者、播客主播的青睐，和他们建立起紧密的关系。

如果说广播是罗斯福时代做炉边谈话的媒介，那么今天这样的媒介或许是播客，或许是短视频，或许是直播，但拜登没有很好地利用任何，也就失去了展现领导力、团结民众的机会。

在美国即将进入川普2.0时代的时候，回顾拜登的失败与失意，一个政客在时代当中的角色与局限变得更加清晰。拜登是一个传统主义者，他未能跳出传统的框架，回应川普崛起、科技巨头、社会裂痕等关键挑战。接下来的川普四年，更多的冲击或许将会海啸般来临；而真正有远见卓识和破局气魄的回应者，也 许会在这样的环境下面出现。

至少，不少美国人在等待一个真正的FDR。
