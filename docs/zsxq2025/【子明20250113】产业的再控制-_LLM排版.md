
# 产业的再控制

延展自政事堂1月10日文章《五粮液董事长，也落马了》
发布于2024年1月13日

任何利益集团上台的目的，
都是推动自身力量对各行各业的改造，
以实现对更多产业链与利益的控制。
以电子工业部(工信部)为根基的Red 党在18大胜
后，
代理人马云到处搞“赋能”,
从通讯骨干网到高铁再到物流网，从高端白酒到
大型商超再到饿了么,衣食住行被他颠覆了个遍。
各位互联网大佬们也跟着电子工业部爽的一笔。
直到马云把手伸向金融，
试图将金融改编并吃到自己肚子里，
导致原本的盟友变成的仇敌，
引发了几乎所有被他“混改”势力的联合反击，
也才触雷并导致了功亏一篑，
随后整个互联网平台经济都遭遇了血洗。
就像工信大佬们退出茅台一样
现在的很多产业动作，
无论是对房地产、教育、医疗等
说的更赤裸一点，
本质上是去工信部化(J), 去 央 行 化(Z).
是要去除他们身上的互联网与金融属性。
当然，这些动作并非是要“解放”他们
而是要让“高质量”集团再对他们进行“新赋能
以便于刚刚换完政治房本的新政治力量，
对这些传统产业实现新的控制。
