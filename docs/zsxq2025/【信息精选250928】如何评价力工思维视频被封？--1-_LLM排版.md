
# 如何评价力工思维视频被封？

蒋天明

社会群体之间是有博弈的。偷奸耍滑的一方可以一时占便宜，但是很快会被反噬。传统婚姻形式，就是这个博弈的平衡，其中配套了社会舆论、意识形态教育，道德约束，甚至刑事惩罚。而力工思维背后，本质是对传统婚姻稳定性、对女性道德性的无条件信任。这种信任因为历史惯性，已经被力工们视为天经地义的。但是这种社会意识其实已经落后于社会存在，因为女性已经率先完成了去责任化。女性去责任化，会带来消费繁荣；但是男性去责任化，直接就是失去了鞭打牛马的鞭子，生产能力都会退化。但这是现代工业化社会对于传统婚姻的必然解构，谁都无法阻挡。

婚姻作为套在男性身上的社会枷锁，本来是通过对女性婚姻道德的强制干预来实现的（三从四德、通奸惩罚等）。现在社会把女性的婚姻道德限制去除了，还希望男性的枷锁继续存在，这有点侮辱男性的智商了。社会存在决定社会意识。社会不会因为力工思维的视频被封了，就能够压制男性的博弈策略。其实在力工思维之前，“龟男”、“供养者”这些概念已经存在了， 男性的去责任化浪潮已经开始了。只是“力工思维”,就像“taco”一样，简单、准确、生动、易于传播，把男性的觉醒活动推到了高潮。

另一方面，力工思维是比以往口号更为深刻、影响面更广的slogan。

1. “力工思维”将矛头直指婚姻，而不是恋爱，把反女拳的叙事范围从年轻人扩展到已婚人士。近两年来所谓的“龟男”理论、“舔狗”理论，更多只是抨击了男性习惯性利女的恋爱态度，范围局限于小年轻。而“力工”理论，血淋淋地指出女性在婚姻中的责任缺失和男性悲惨的经济遭遇，并且直接把问题核心指向了婚姻合作模式本身，而不仅仅是年轻人的恋爱。于是很多80后以前的已婚男性，也会被卷入到对自身婚姻中责任是否对等、收入与家庭地位是否匹配的反思中来。其中很多估计一反思，就破防了。

2. “力工思维”以性别叙事超越了阶级叙事，将叙事范围扩展到全阶级男性。无论是高官、资本家、娱乐明星还是底层打工人，“力工思维”诱导男性去思考：你有没有在两性关系中对女性和所谓婚姻做无条件让步?有没有自己冒风险贪污，认罪坐牢，老婆孩子拿着赃款在国外挥霍?老婆有没有找茬离婚分走一半公司股权?就算你长得帅、能挣钱给家庭，老婆有没有出去做头发?

所有的一切都试图向全社会的男性证明，女性在婚姻模式中是不可靠的，男性自己的劳动成果不应该轻易地与缺乏婚姻道德的女性分享。

综上，我愿称“力工思维”是男性被迫对女拳压迫作出反抗以来，最强的大规模杀伤性武器。它是男性反抗女拳的“核武器”。
