
# 学习笔记，2025.5.25

1，正常的创业流程是，一个人上班看到天花板了，在职场上影响自己发挥，想去追求一下更高的可能性。网友的创业流程是，上班做不出绩效被裁了，想当老板去制裁别人，想花钱买个铺子躺着赚钱，让员工去累死累活，这种情况，输掉家底是轻的，很多人最后会背上债，打工还债。打工，只需要负责商业行为的一环，创业，则需要自己对所有环节都了解，稍一松懈，就会被人坑，难度比打工高了很多级。打工都没打明白，千万别创业，别听那些教人创业的博主瞎忽悠。

2，爱一个人的前提是，能够做到好好爱自己。如果一个人连自己都不尊重、不够爱，他不可能对别人有真爱，这种更多的是一种索取，希望通过自己表现出来的热爱，换来对方的关注。当这种给出去的爱长期得不到自己想要的回报的时候，就会慢慢转变成恨，认为对方在耍自己、不尊重自己。

3，不是只有把钱放到某个理财产品，才叫投资。投资自己也叫投资，认知的投资、生产资料的积累、技能的提升，这些都会随着时间的增长而带来更多回报。对普通人来说，通过工作赚一百万，要比通过投资赚一百万，要容易得多。换个角度想，如果一个人不能通过工作赚到一百万，对市场肯定有误解，不可能做好投资市场。

4，很多关系，上次见面已经是最后一面了，同学、同事、亲戚……而且，哪怕同在一个城市，也不一定有再见面的机会。

5，现在的自媒体，就是一场大型表演，都在立人设，观众喜欢什么，博主就是什么。喜欢爽文，博主就是草根逆袭大女主/大男主人设。喜欢抱怨，博主就是怼天怼地看谁都不爽人设。自媒体内容，基本以情绪内容为主，所以会一直刷刷刷，停不下来。你有欲望，你控制不了自己情绪，别人就有的是办法抢走你的时间和注意力。

6，一句话，说出去，解释权就不在自己了，正着解释反着解释的都有，怎么用都有可能。一些容易存在误解的表达，尽量不说。可说可不说，不说。

7，怎么判断自己是不是真的喜欢一个人？核心标准就一个：你能接受自己的孩子以后变成他这样吗？简单来说，找对象最该看重的是内在品质——性格是否合拍，人品是否可靠。这些都是真正需要优先考虑的因素。别只看学历、外貌、收入这些外在条件。这些都会变化，现在觉得好，以后未必还行。一个人的本质特点和价值观才是最关键的，这些才会决定一辈子的生活。你可以问问自己：你希望孩子将来跟他一样吗？做事能坚持，做人守规矩？再认真想想：如果孩子既有你们的优点，也继承了双方的缺点，这样的孩子你能接受吗？要是连这点都做不到，那这段感情可能就得重新考虑了。

8，评价是容易的，上嘴唇一碰下嘴唇，谁都能评两句。好的评价是不容易的，得真正了解、体验过，才能给出客观真实的评价，甚至有些评价可以比原作品高级。多体验、多实践，不要成为一个只会动嘴的评价者。反过来也一样，别人评价你的时候，问问自己，他做到过什么，他配吗。不是谁的评价都有必要听进去。

9，人想得太多，做的太少，没有自己的核心事务，就容易焦虑。人如果有一个不得不做的事或者热爱做的事，就没工夫关注太多事情之外的事，会把大量时间用在做事上，外界如何变化，对他来说，都不重要。

10，人生如登山，不必等到登顶才相信自己能到。先在心里站上峰顶，脚步自然会跟上。想改变，先想象已成为的模样。比如想减肥，别盯着体重秤，而是幻想自己穿上心仪的衣服，或站在舞台中央光彩照人——这画面越具体，行动力越强。习惯是旧自我的铠甲，打破它需要两样东西：极致的渴望，或刺骨的痛。若没有，就给自己造个“不得不”的理由：想被爱慕？想被仰望？让目标像钩子一样拽着你向前。记住：先活成想要的自己，世界才会信你配得上。
