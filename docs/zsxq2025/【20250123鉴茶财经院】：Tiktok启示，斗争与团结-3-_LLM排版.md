
# Tiktok启示，斗争与团结

美帝刚制裁tik,要求其非售即禁的时候，某司那些高管们都懵了。他们常年受西式教育，以为那票人真是自由平等，讲规矩的。什么信息安全，服务器等等，能做的都做了，美帝不依不饶。说白了跟这些没关系，人家就是要吞并，恶意的。你以为的理由，都是借口。

当年带着小张出道的一个老师，是大王的一个老大哥，他说一鸣把自己关在屋子里，觉得三观被颠覆了，世界都崩塌了。后来，上面派人去给支了招，核心就一句话，以斗争求团结则团结存，以妥协求团结则团结亡。

你耍流氓，我就拿出宁为玉碎不为瓦全的架势，反正不卖，宁肯关了也不卖。否则，一旦开了口子，今日割五城，明日割十城，什么时候是个头。1月19日，洋抖直接发通知说不提供服务了。这下美帝慌了，又给宽限三个月，川普提出个要50%股份的方案。从不给就禁，变成了继续经营，从全吞，变成了半吞。美方的气势弱了，高低位切换了，后面很多事就能谈了。

这件事给我们什么启发，第一，当你能力实力不足的时候，要敢于大踏步后退让步，不要磨磨唧唧，想给不想给的，事没办成，纠结内耗还搞了很大成本。红军当年倒围剿失败，到遵义后，教员说不要在南方折腾了，直接去陕北另起炉灶。第二，当你符合公序良俗，非常正确，又是自己核心利益的时候，一点都不能让步，必须有自己的框架，让对方清晰的知道，你什么事情是绝对不接受，碰了就死。

群里很多人问过感情的问题，其实都是因为自己一再牺牲核心利益，不敢表达诉求，最后关系失衡，你非常痛苦还没解决问题。比如两个人谈恋爱，你是焦虑型人格，连发四五条消息，对方两三天都不回复，搞的你患得患失，疲惫消耗。错不在你，反而是你要直接讲，下次24小时没回音，视为毫无诚意，直接删除。反者道之动，弱者道之用，你不怕失去，才能最终得到。

第三，不断的精进，优化，提升自己。美帝为何最终不敢对tik下死手?东西是真好，看看小扎，老贝，他们做的社交媒体啥玩意。洋抖是真好，真有分量，真有价值的。很多时候，你没有得到一个东西，不是你不努力，也不是对方和客观条件，而是你“没配上”，当你配得上很多东西的时候，自然就有了。轻松的拥有，本质是一种向下兼容。
