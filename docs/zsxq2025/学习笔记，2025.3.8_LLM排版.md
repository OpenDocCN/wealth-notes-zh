
# 学习笔记，2025.3.8

1，在网上看大 V、讲师们分享，不能只看他说什么，要看他在做什么。比如，我在网上说，要少一些情绪、多一些事实。这并不代表我没有情绪，也不代表我不去当标题党玩弄情绪。我可能前面跟你说，要长期主义，要稳下来做有价值的事，转头就去搞 AI 写猎奇爆文薅流量主广告赚快钱，创造垃圾，让人们陷入情绪之中。前几天还看见一篇有意思的报道，史玉柱，搞脑白金的那位，开始搞年轻人的“脑白金”了，想打造健康咖啡的概念，但是年轻人基本上不买他的账。别看他说什么，看他做什么。还有最近学雷军的某个大网红，以前做的缺德事一大堆，现在营销的再成功，年轻人也不买他的账了。

2，一个人没法成为自己嫉妒的人，没法成为不想成为的人。一个人嘴上说着想要变富，但是内心却仇富，那他就不可能变得富有。

3，人想要自救，就要先找到那些让自己陷入深渊的问题，比如导致情绪波动的人，导致身体不适的行为，等等。发现问题、正视问题，才有可能解决问题。

4，人要适当克制自己的表达欲，表达对了人，是加深关系，表达错了人，是徒增烦恼。大多数情况下，表达欲起来的时候，都容易跟着情绪走，表达错了人。

5，持续自学、善用搜索，才是一个人最底层的能力。有这两个能力，只要你想，就能掌握任何你想掌握的东西。

一个不愿意自我迭代、持续学习的人，被社会淘汰是早晚的事。就像很多曾经叱咤风云的商界、政界大佬，一旦下台，没了那个心气儿，慢慢也就会被卖保健品的忽悠。他不是错了，只是老了，心态老了。

6，一个人自身有价值，能给别人提供价值，才能吸引更多同频的人。一个人不能给别人提供价值，别人还愿意带他玩，那大概率不是一种平等关系，而是一种施舍，是一种居高临下的慷慨。很多人在追求的就是别人的这种慷慨，希望有人能来当自己的救世主，当自己的外挂老爷爷。

7，常有人问「努力真的比不过天赋么」，但很少有人问自己「真的努力了么」？

8，人如果不承认自己的局限性，就会下意识反驳所有和自己不同的观点。当别人不愿意接受他的观点输出的时候，他就会情绪爆发，开始用激烈的方式引起对方的关注，让对方来和自己辩论。网上的杠精，大多都是这种人，自认为全知全能，每个行业都能指点一下。

9，AI 的思考模型，让很多不思考的人误以为自己也有了思考能力，开始把AI的话当成圣经，当成自己思考的结果，用AI的回复去反驳别人。尤其是AI的联网模型，现阶段联网搜索的基本都是自媒体的标题党文章，不可能有权威性，什么人口预测、房价涨跌，都是胡扯。AI有智能，也确实可以涌现出一些有新意的回答，但是用好AI 的前提是，你能分辨AI内容的真实程度，否则他就是一个爱说瞎话的孩子，半真半假，说不定在什么地方坑你一个大的。这和当初用搜索结果去质疑医生是一样的道理，换了个看上去更高级的模式而已。

10，人爱别人、爱事业，都不该超过爱自己。爱自己是基础，一个不爱自己的人，很难做到真正爱别人、爱事业。要优先照顾好自己的身体和情绪。
