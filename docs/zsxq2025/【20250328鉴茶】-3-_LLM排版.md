## 放弃对他的期待， 你不要了。
比如以前， 你给10,希望她回报3,哪怕有1也满足， 结果， 她给你－5,你难受不。
你继续该给10给10,连这个1也不要了， 你爱咋咋地， 这样你就不会收到－5。
你希望得到的3,你自己给。
这就是无欲则刚。
这世界上， 深度纠缠亲密的关系之间， 有两种模式，
一种叫 “ 合 ” ，两人心意相通， 很有默契， 琴瑟和谐， 离不开。
比如， 属鼠与属牛， 属虎与属猪， 属兔与属狗， 属龙与属鸡， 属蛇与属猴， 属马与属羊， 这都是六合。
十二生肖， 对应太阳的黄道十二宫， 其实是能量的十二种体现方式。
一种叫 “ 刑 ” ，两人就是非常拧巴， 非常消耗， 搞不到一块去， 但也离不开。
比如， 牛和马， 蛇和虎， 羊和鼠， 猴和猪， 龙和兔， 都是弄不到一块的， 另外还有自刑， 比如辰辰， 两个龙， 亥亥。 两个
相合相刑， 都是能量的深度纠缠， 一团乱麻就是解不了。
当然仅看生肖太宽泛太粗了， 不精准的很多， 命理大模型54万种类别， 就非常详尽。
碰到合的人， 那是帮你的， 你尽情做自己， 尽管释放欲望。
碰到刑的人， 那是渡你的， 你调整自己的心态， 学会摒弃欲望。
痛苦的时候，
要么改变环境， 要么改变心态。
