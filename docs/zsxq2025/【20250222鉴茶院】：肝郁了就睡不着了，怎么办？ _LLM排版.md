
# 肝郁了就睡不着了，怎么办？

## 睡觉的好处

*   **子时大睡**：23~1点进入沉睡状态，肝血排毒，推陈出新，脏腑得新鲜血液滋养而充满活力。
*   **午时小憩**：上午消耗的全是阳气，午时则开始阴生，11~13点小憩有助于养心阴、补心阳。

## 睡不着的根源在肝

*   **肝藏魂**：魂指思维意识，由肝血化生和涵养。当肝郁气滞，气血不和，魂失养而离舍，思维、意识就会很活跃，翻来覆去睡不着；当肝气腾达血充足，魂有所归，思维就不会到处游走，就能睡得好。

## 女性更易肝郁的原因

*   **易郁易怒伤肝**
*   **激素变化**
*   **月经耗肝血**

女性心细敏感，遇事藏怒发，怒则肝气逆行往头上窜，肝气不顺而久郁。

## 肝气郁结的表现

*   乳房：经前乳房易胀痛
*   皮肤：暗沉/黄/长痘斑
*   月经：经量少/多/不调
*   头部：伴有头晕/痛痛
*   眼部：眼干/眼涩/眼痛
*   睡眠：难入睡/易燥心

## 养护指南

### 睡前3做，助入眠

*   引血入肝，才能安。
*   按脚趾、压头顶、揉手腕。
*   引血归肝，让头顶是百会穴，用指腹敲打或用手抓此处头发向上提，调和脾胃。

### 内服，康祺解郁丸

*   **中药成分**：白芍、柴胡、当归、郁金、茯苓、百合、合欢皮、甘草、大枣、小麦。
*   **功效**：疏肝解郁，养心安神。
