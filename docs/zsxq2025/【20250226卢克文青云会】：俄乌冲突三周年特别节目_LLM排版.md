
# 俄乌冲突三周年特别节目

这段时间大家应该也都知道，短短的半个月不到，俄乌冲突发生了巨大的戏剧性的转变。如果之前经常听我直播节目的老听众应该都知道，我在很早以前就对俄乌冲突有过一个基本的看法。

我当时就说，俄乌冲突是美俄重新划分欧洲各自势力范围的一次军事行动。从现在美俄开始在沙特的谈判以及一系列美方的操作来看，确实基本上符合我当时的判断。这场战争既不是所谓的能源之争，也不是所谓的反法西斯或者反纳粹、去乌克兰军事化的这种战争，这些都是借口。而其根本军事冲突的目的是在于就是要重新划分美俄在欧洲的势力范围，因为北约东扩已经严重影响了俄罗斯的地缘政治安全，所以俄罗斯是绝对不会容忍的。

那么同样的范围，美苏冷战结束以后，当一种全新的地缘格局政治没有形成的时候，毫无疑问战争是最好的一种方式，让双方能够达到再平衡，确实俄乌冲突让美俄在欧洲的各自的势力范围达到了一次重新的平衡。所以俄罗斯必须要发动这场战争，美国当然也作为欧洲事务的重要参与者，他也会介入。

现在很明显，俄罗斯依旧是一个大国，这一点毫无疑问。我也看了很多短视频平台上的很多人，要么讲俄罗斯必败，要么讲乌克兰必败。我觉得乌克兰和俄罗斯是否战胜，在这场战争中本身来说并不重要，重要的是他需要这样一场战争来让美俄或者欧洲的势力范围得到重新的平衡。

我基本上认为俄罗斯不太可能会被这场战争所拖垮的一个比较重要的看法，可能跟很多常规的看法可能不太一样。上个星期我们在做一次内部的讨论的时候，我也提出了这个看法，今天我也可以跟大家来分享我的一个看法。当然这个看法， 一般来说，外面你们可能不太听到得，当然这个也是可以公开讲的，因为本身只是我的一个看法。

就是我认为，一个在历史上能够创造出诸多优秀文化作品的一个国家或者一个民族，他是很难被真正地用一场战争的方式拖垮。那么这个话怎么来理解呢？俄罗斯作为一个民族也好，或者是作为一个俄罗斯联邦国家也好，如果我们回过头去看他的历史，他确实在人类的文化中贡献了很多。

比如说他的文学作品，《战争与和平》、这些如此宏大的巨作，是俄罗斯人写出来的，还有油画、芭蕾舞，现在基本上还是俄罗斯独占鳌头。俄罗斯人对于文化和艺术的感知能力是非常强的，也就是说换而言之，能够创造出如此优秀的文学作品或者文化作品的一个国家，其实是很难被一场战争所拖垮的，因为它有足够的民族韧性。

所以我当时就说，也许俄罗斯战争打得不怎么样，但是你想要说我通过这样一场局部的战争能够把它拖垮，乃至于说甚至有人会认为俄罗斯可以被解体，这个基本上不可能。如果解体的话早就解体了，因为他已经经历过一次解体了，所以从本身来说，俄罗斯本身自己不太可能会被拖垮。

当然从斯拉夫人的民族的角度来说，其实这场战争是一个斯拉夫人种族的悲剧，因为如果稍微了解一下斯拉夫人历史的都知道，其实基辅，是最早俄罗斯文明的诞生地。基辅至于斯拉夫人就相当于西安至于中国，但是白罗斯、俄罗斯、和乌克兰三个国家其实是斯拉夫民族的，应该是一体的，现在完全就是兄弟相残，其实从种族的角度或从文化民族的角度上，这是一种悲剧。

其实这个对他们自己来说也是一场悲剧。那么基本上我跟大家分享一下，我对这场战争到了打了三年下来的一个基本的一些看法。

在这场战争中，俄罗斯和乌克兰双方在情报领域确实是投入了大量的精力，而且也成功的向外界展示了一场在现代信息条件技术下的一场情报攻防战，这个对于中国来说是具有极其现实的借鉴意义的。

因为我们本身就有很多就是说亟待解决的现实地缘政治问题，两岸问题、南海问题、中印问题，这些都是就中日之间的问题，中韩之间的问题。所以我觉得我们通过俄乌这场现实的血淋淋的战争，可以看到现代战争中所要动用的涉及的面已经越来越广了，不是以前简单的造枪造炮的这种，而是一个它整体做了一个全国的动员。

首先我们从俄罗斯的角度来看，俄罗斯进行了哪些所谓的反情报工作？最早在2021年的时候，其实以美国为首的北约就开始对俄罗斯进行抵近侦察。那么由于俄罗斯在这个俄乌边境借由举行军事演习的名义，向边境地区持续地增加兵力，所以北约就出动了这空中侦察机、海上侦察平台、陆上侦察平台、天空的侦察卫星等等多个维度的侦察力量，对俄罗斯进行严密地战场侦察。

同时对俄罗斯的兵力调动、通讯波段的监听、武器装备的使用、部队的训练、士气等等这些进行持续地侦查，为北约提供了大量实时的情报数据。这些情报数据后来在俄乌冲突爆发以后，也持续与乌克兰方面进行共享。

这个全方位的情报侦察，其实无论是从战场的角度，还是从情报作业的角度是很有必要的。因为我们在展开进行一场军事斗争的时候，前期的前期侦查工作是非常重要的，这个侦查工作其实主要来说就是战场经营。

所谓的战场经营就是你要了解你当前的这个对手，他武器装备的情况，包括人员、后勤、士气，道路桥梁。比如举个很简单的例子，当面如果是有桥梁，这些桥梁的最大承重是多少？因为我们都知道现代的很多主战装备的重量是非常重的，很多时候是很难通过一些桥梁的，必须要进行事先地加固。

我在这儿展开一下，比如说像今年美国向台湾提供的第一批M1A1T这个主战坦克，但是这种主战坦克第一批大概提供了38辆，被台湾认为是台湾军队目前为止最先进的装甲力量。

如果单纯从账面角度上来看，确实M1A1T这款坦克确实是能够列入世界先进坦克之列的。无论是它燃气轮机的所爆发出来的高速和机动能力，以及它的精准的这个，以及大口径火炮，以及它较强的装甲防护能力。

但是对于台湾这样的一个以山地为主的地区，M1A1T所能发挥的作战效能会受到严重的制约。为什么？因为台湾大多数的桥梁是不能够承受M1A1T坦克的通过的， M1A1T 坦克的全重至少是在55吨以上。

那55吨的载重能力对于很多桥梁，尤其是很多台湾的桥梁是没有办法做到的。它必须要进行加固，这个加固的过程中就会大大延缓它的行动能力，因为除非你把台湾所有的桥梁都进行加固，否则你很难在他需要的时候让这些坦克能够到达战场。

因为你不知道未来的人民解放军会在什么地方展开所谓的登陆作战，以及你也不知道他们作为解放军来说，他可能会采取的登陆手段。所以在这一点上，对于桥梁是否能够加固，加固到什么程度，它是不是能够承受这么重的坦克以及后续的车队的运营，这其实就是战场经营。

乌克兰方面其实得到了北约方面的这种大量的情报数据以后，确实它对于整个当面的俄军有比较清楚的了解，所以这也就是为什么当天爆发了冲突以后，俄军没有在第一时间能够拿下基辅国际机场的一个很重要的原因就是，乌克兰方面已经有所警觉了。

这一点确实应该得益于北约的情报作业很成功。

那么另一方面，乌克兰方面，因为乌克兰也知道，如果论情报作业，乌克兰是防守一方，防守一方是很难知道俄罗斯的主要攻击方向的。所以乌克兰就发动所谓的民众情报作战，就是将所有的民众动员到、参与到情报作业中。

所以整个乌克兰情报部门和乌克兰国防部就利用所谓的电报社交软件开通了一个信息上传的频道。也就是说你只要在你的马路上或者你的小区里，或者你的什么地方发现有俄军的车辆、人员，你就可以拍照，直接通过电报这个软件报告给乌克兰军方。

而乌克兰军方就可以通过你所提供的视频照片来大致，或者说迅速寻找俄军的踪迹。后面等会我会讲到，那么这个踪迹是怎么获得的？这个里面其实又有一个很重要的技术手段，就是人脸识别技术手段。

基于我刚才讲的，比如说基于这些政治上的、文化上的，包括心理上的这些维度，那么他利用他自身的这些领域上的特点，再用信息平台的方式构建了一个所谓混合作战的理论。

混合作战理论确实一直以来是被西方军事专家所讨论的，但是混合作战长期以来停留在所谓的理论层面，没有一场战争对它进行加以实践，或者加以验证，而这次俄乌冲突就成为了混合作战理论的最好的一次试验。

简单来说，混合作战就是利用多元的主体，利用多样的手段在不同的领域发动战争。因为以前我们都知道传统战争是什么？兵来将挡，水来土囤。你有坦克，我有飞机。你有飞机，我有导弹。

就只是在一个军事层面，或者说是工业层面的一场战争。但是现在的混合作战已经发展到了军事、信息、经济、政治、文化、心理、语言等各个方面，因为每一个领域都有一个自己的空间，而将这些空间全部纳入到整个战争中来，并且利用不同的手段，针对不同的人群发送消息。

比如说，举个很简单的例子，俄军和乌克兰方面会利用在战场上搜捕到的对方被打死的士兵的照片，向这些士兵的家人定点发送照片和信息。

也就是说可能是一个俄罗斯的母亲，或者是一个乌克兰的母亲，会在她自己的手机上看到自己的孩子被打死的照片。这种所产生的心理震撼以及强烈的情感，就是混合作战中很重要的一部分。

我们以前都知道，在美国发动越战的时候，有越南妈妈走上街，走向华盛顿，要求美国从越南撤兵，而美国后来从越南撤兵，确实也受到了这些失去孩子的美国母亲强烈的道德和舆论的压力。

后来的现代军事学家就把这种引导情绪和引导舆论的方式，作为混合战争中重要的组成部分，而在这场俄乌冲突中得到了充分的验证，确实得到了成功。

我们后来看到了几次双方的交换俘虏，双方各自利用自己的媒体，向全世界传播着有利于自己的信息。很显然，我们国内无论是支持俄罗斯的，还是支持乌克兰的群体，你不能否认他们没有受到过类似信息的影响。

我们也在观察中国的社群，尤其是对俄乌冲突这种社群的观察，很多人就因为一两张照片或者一两段视频受到影响。这些照片和视频是不是真的呢？姑且我们把它看作真的，但是为什么他们要利用这种方式呢？

很显然，他们就是利用这种强烈的情感造成一种情绪的共鸣。

因为人是一种会共情的动物，当越来越多的俄罗斯母亲或者乌克兰母亲看到自己的孩子被打死的照片的时候，我想这种力量是非常巨大的。

这一点我觉得大家要有一个认知，这个照片绝对不是轻易就给你发出来的，某种意义上它也展示了俄罗斯或者乌克兰的一种情报。

我不但能够知道我打死的这个人姓氏名谁，我还能够知道他母亲的手机号码是多少，而且我还能够成功地将这些照片发送到这些母亲的手机里面，这本身就是一种情报能力。

因为用户的手机号码和个人信息是隐私，按道理说，电信运营公司是要保密的。

他们是怎么获取的呢？第二个问题是他们利用了大量的黑客技术，等会我后面也会讲。
