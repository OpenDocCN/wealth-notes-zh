
# 卢克文5.23：中国发布藏南第五批命名，释放出什么信号？

## 中国发布藏南第五批命名，释放出什么信号？

5月11日，民政部依据《国务院地名管理条例》，公布第五批藏南地区标准化地名，共27个，涵盖错那市、墨脱县等地，包括山峰、河流、居民点等类别，并标注精确经纬度。

大家知道，藏南是我国固有领土，但因历史遗留问题被印度非法控制部分地区。选择此时宣示主权，释放出什么信号？后续会如何发展？

### 1. 中国对藏南地区标准化命名是宣示主权的常规操作

实际上，我国对藏南地区标准化命名是宣示主权的常规操作，2017年、2021年、2023年、2024年和2025年已经先后公布了5批命名。但每次公布的时机都是有讲究的，基本都是三哥内心有点小躁动，需要敲打的时候。

三哥有时候服，有时候不服。

#### 2017年

2017年那次三哥就不服。当年4月13日，我国公布第一批对藏南地区的标准化命名。印度不仅抗议，边防部队还更加活跃起来。发展到6月份，甚至嚣张到越过洞朗地区，阻挠我方修路，引发长达73天的军事对峙。

#### 2021年

2021年那次三哥就服了。当时印度持续在藏南增兵，推动“东进政策”基建项目，并通过《国家安全法》修正案，强化对所谓“阿鲁纳恰尔邦”（我国藏南）的法律管控。在中国针对性公布第二批15个标准化命名后，三哥嘴上说不要，外交部回应称我方命名“无效”，但身体还是比较诚实，暂停了部分边境基建项目。

#### 2025年

第五批，也就是这一次，选择的时机是印度违反印巴停火协议再次开火的第二天（5月11日）。意思很明显，就是点醒三哥：差不多行了，别太兴奋。

不过三哥这次没有服，大概还在气头上吧。

### 2. 俾路支问题与中巴经济走廊

5月14日，流亡欧洲的俾路支分离运动领导人米尔·亚尔·俾路支在X平台上发布视频，宣布成立“俾路支斯坦共和国临时政府”，呼吁印度和国际社会承认其主权。此后该消息经印度媒体翻译和炒作后在全球引发热议。

同日，俾路支解放军(BLA) 等武装组织在俾路支省51个地点发动代号“英雄行动2.0”的袭击，包括炸毁政府军据点、安全部队监控设施，进攻路线一度逼近中巴经济走廊。这是自2023年以来该省规模最大的一次武装行动。

虽然印度方面一再否认，从5月15日起就在多个场合强调印度支持巴基斯坦的主权和领土完整。印度驻联合国大使甚至在X平台上发布多条推文，称“印度是巴基斯丹动荡的最大受害者”，要求国际社会关注“跨境恐怖主义网络”。但明眼人一眼就可以看出是印度（可能还有美国）在背后搞的鬼。

印度被俘将领的供词、巴方截获的银行记录、缴获的武器，都指向印度在向俾路支独立武装提供包括情报、资金、军备和舆论操控等系统性的支持。

李敖曾经有过一个精辟的比喻：“台湾就像中国的睾丸，美国人不痛快了就要去捏一捏。”这个比喻用在俾路支身上也是合适的——俾路支就像巴基斯坦的睾丸，印度人不痛快了就要去捏一捏。

这次印度捏的力道大了，矛头直指的龙哥：中巴经济走廊最重要的枢纽瓜达尔港就位于俾路支省海岸。

俾路支闹这一出，多少威胁到了中巴经济走廊的安全和后续的建设速度。

### 3. 俾路支问题的现状与影响

在之后，自媒体上各种捷报频传，说巴军已经将俾路支武装全面清缴。甚至还有细节：头目被抓，主力被歼灭，少部分逃往山区；并按暗示作战中巴军使用了我方提供的机器狗等高新技术武器等等。但可信度不高，既找不到权威的信息来源，也不符合逻辑。

从逻辑上分析，俾路支武装是内部分裂问题，武装分子和平民百姓浑然一体，军方动起手来投鼠忌器、极为棘手，不是军力或武器占优就能解决的。

此外，俾路支省地处伊朗高原东部，境内超过80%区域为山地和高原地貌，比如苏莱曼山脉和基尔塔尔山脉，平均海拔达1200米以上，地势起伏剧烈，沟壑纵横交错，易于藏匿，难于搜捕。要清剿包括“俾路支解放军”(BLA)、 “俾路支解放阵线”(BLF) 等至少6个独立、松散的主要武装，短期内几乎不可能。

官方媒体最新的报道也印证了这一点。

新华社伊斯兰堡5月19日电讯，巴基斯坦西南部俾路支省18日夜间发生一起自杀式爆炸袭击事件，造成至少4人死亡、20人受伤。

新华社伊斯兰堡5月21日电讯，巴基斯坦官员21日证实，一辆校车当天在该国西南部俾路支省胡兹达尔地区遭遇炸弹袭击，已至少造成包括3名儿童在内的5人死亡，另有40人受伤。（巴基斯坦指称是印度策划，其俾路支代理人实施）

看，俾路支武装力量还活蹦乱跳。

从目前的信息看，从中国公布藏南地区命名到印度还手，这一来一回中国吃了一点暗亏，印度找回了一点心理平衡。

但中国输了吗？并没有。

将军赶路不追小兔，欲成大树不与草争。

大国聚焦于自己的长远目标，很少做不必要的意气之争。现阶段中国在印度次大陆的核心利益是中巴经济走廊，而不是打倒印度。这一次公布藏南命名，目的也是防止印巴冲突进一步升级，给中巴经济走廊造成不可控制、不可挽回的损失。这一目的已经达到了。

5月20号，法新社报道称，巴基斯坦一位高级安全官员证实，巴印两国已同意在5月底前将在最近冲突期间部署的增援部队撤至和平时期的阵地。双方冲突升级的可能性已经很低了。

至于俾路支独立运动，已经是巴基斯坦的老大难问题了。从巴基斯坦1947年立国至今冲突无数，多这一次不多，少这一次不少。虽然没办法迅速解决，但不会伤筋动骨。这一风险成本，早在中巴经济走廊立项时就在计算之中。所以这一次俾路支宣布独立，并没有对中国造成什么额外损失，一切仍然在轨道中。

### 4. 中国的长期战略

综上，这次宣布藏南地区命名是达到预想目标的，吃点小亏让印度心理平稳，使中巴经济走廊避免更大的风险，划得来。

这一次中国公布藏南地区标准化命名，还说明一个问题：中国并不想现在就毕其功于一役，解决藏南问题。持续宣示主权，是为后来的大动作保留正当性，这是将问题押后解决的一个信号。

从全域看，至少台湾问题、南海问题优先级别在藏南问题之上，在解决这两个问题之前都应该先稳住三哥，免得三哥头脑一热去当美国印太战略的急先锋。

从南亚局部看，中巴经济走廊优先级别也在藏南问题之上，中国无论如何也要先保证这条能源通路的顺利建设，避免节外生枝。实在是这条通路对中国的能源安全太过重要。

中国目前年自产原油约2亿吨，每年还需进口约5.5亿吨，进口依赖度约77%，是世界第一大原油消费国和进口国。而进口的原油有80%需要经过笼罩在美国军事影响力之下的马六甲海峡。

马六甲海峡最狭窄处在新加坡段，最窄处仅有37公里。而美国在新加坡有四个军事支点：樟宜海军基地、森巴旺海军基地、巴耶黎巴空军基地和樟宜国际机场。所以非常时期只要美国愿意，它是有能力在马六甲海峡切断中国的能源生命线的。

因此，中国向来有“马六甲之困”的说法。

而中巴经济走廊就是解决马六甲之困的核心替代路线之一：瓜达尔港可以接收从中东过来的石油，然后经由管道和陆路运输进入中国西部，航程缩短85%，完全绕过马六甲海峡，设计目标是保障中国原油进口的45%。

所以你看，中巴经济走廊对中国来说几乎是容不得有失、不惜代价的。曾经担任邓小平英文翻译的高志凯在印巴冲突时期的高调表态，在一定程度上代表了中国政府的态度，他面对国际媒体时反复表示：不要低估中国对巴基斯坦的承诺！

这种对印度来说近乎威胁的姿态，背后的实际含义是——不要低估中国维护中巴经济走廊的决心！

后续要使中巴经济走廊达到预想的战略效果，中国接下来还是要帮助巴基斯坦逐步解决俾路支问题。核心将是策略的输出，而不是武器的输出。中国有56个民族，民族融合经验之丰富，全世界首屈一指。民族融合经验才是解决俾路支问题的关键。

我曾经参与处理过一些群体性事件，核心的逻辑就三个：分化一部分、拉拢一部分、打击一部分。先是使群体内部利益对立，不能拧着一股绳；然后对可以争取的大多数采取怀柔政策，给予感情和经济支持；对起事的刺头，则杀鸡儆猴。当然我参与的是小规模群体事件，但民族问题，道理也是一样，只是在分化、拉拢、打击之后再增加一个环节，就是同化。

比如推广官方语言、节日、服饰，通过社交媒体促进民间互动等等。只要让俾路支普通百姓的日子过得好点，切断他们和武装分子的联系，逐渐改变他们的身份认同，那如无根之萍的武装分子就翻不起什么风浪。至少缓和俾路支问题，不至于影响到中巴经济走廊的大局。

印度和藏南接壤的是东北六邦，包括梅加拉亚邦、曼尼普尔邦、那加兰邦、阿萨姆邦、特里普拉邦和米佐拉姆邦。这东北六邦是通过一条狭小的陆地走廊和印度本土连接在一起的，这条走廊叫西里古里走廊。

西里古里走廊夹在我国和孟加拉国之间，最窄处大概只有20公里。接壤处是我国朗洞地区，离我们日喀则市亚东县直线距离也就100公里。真打起来，以我军现在的实力，自上而下掐断这条陆地走廊有多难？

一旦掐断西里古里走廊，印度的东北六邦就成为一块飞地。

而这东北六邦本来就有独立的倾向。它们原本就是多个独立王国或部落领地（如阿洪姆王国、曼尼普尔王国）构成，19世纪被英国通过英缅战争强行划入英属印度。印度独立后继承了这一版图，但并没有解决当地民族认同问题。这六邦有自己的宗教、语言和传统，不觉得自己是印度人，骚乱基本也没有断过。

比如，曼尼普尔邦在1947年曾短暂宣布独立，后被印度武力镇压；2019年《公民身份法》将穆斯林移民排除在难民保护之外，引发东北六邦大规模抗议，阿萨姆邦爆发骚乱导致17人死亡；2023年，曼尼普尔邦的库基人与梅泰人因土地纠纷，爆发骚乱导致3.5万人流离失所……

如果说俾路支是巴基斯坦的睾丸，那么西里古里走廊就是印度的睾丸。
