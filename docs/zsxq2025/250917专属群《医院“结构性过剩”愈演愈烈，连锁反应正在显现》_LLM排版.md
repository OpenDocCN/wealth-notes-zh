
# 医院“结构性过剩”愈演愈烈，连锁反应正在显现

## 医院亏损降薪

近年来，无论是公立医院还是民营医院，都面临着经营困境：

首先，亏损情况普遍存在。2020年度全国三级和二级公立医院绩效考核国家监测分析情况显示，全国20个省份医疗盈余为负，占比62.5%；超过四成的三级公立医院医疗盈余为负，四成的二级公立医院出现亏损。2021年，仍有43.87%的二级医院出现亏损。

民营医院情况同样不容乐观。《中国卫生健康统计年鉴2021》数据显示，全国2.35万家非公立医疗机构一年总计亏损1300亿元，平均每家亏损高达553万元。《中国医院院长》杂志报道，公立医院和民营医院的亏损率都稳定在45%左右，近一半医院处于亏损状态。

其次，负债问题严重。2009年至2021年，公立医院资产负债率从31%增长到45%；部分民营医院的负债率更高。

债务居高不下的重要原因在于，许多医院多年来采取“摊大饼”模式，依靠扩建院区、新增分院、增加设备来扩张产能。根据《2023年我国卫生健康事业发展统计公报》，2023年末全国医院床位达到800.5万张，相比上一年增加了34.2万张，医疗产能不断扩张。

但医疗产能扩张的前提是需求同步增长，一旦就诊量下滑，过度扩张的投资就可能形成部分医院的“产能过剩”，从而成为压倒医院经营的稻草。

在这种情况下，医院的财务压力传导到现金流层面，一些公立医院不得不通过降薪甚至拖欠工资来维持运转，而对抗风险能力更弱的民营医院，一部分开始走向破产。

就公立医院来说，财新网2024年12月的报道说，广州、深圳多家三甲医院医生反映收入降低30%或“腰斩”。2024年，梅州市嘉应学院医学院附属医院这家公立二级医院，在欠薪10个月后破产。实际上，公立医院虽然是事业单位，但只是公益二类差额拨款的事业单位，国家财政对公立医院的投入不超过8%，公立医院的运营里面，绝大部分要靠自己创收。

而一些民营医院则直接走向了破产。通过全国企业破产重整案件信息网站搜索，2021年医院破产相关案件为77件，2022年为104件，2023年达到了316件，2024年更是大幅增长到了614件，增幅达到了94%。

据《中国医院院长》杂志采访的专家分析，未来可能继续面临倒闭风险的医院有三批：第一批是医疗服务能力不强、全靠医保吃饭的医院，其中倒闭的民营医院会逐步增加；第二批是没有财政补贴的公立医院；第三批是有财政补贴，但缺乏患者来源，负债较大的医院。

## 什么样的医院在“过剩”？

这些现状可能和我们的直觉有点相悖，按理说医院应该不会过剩才对，因为老龄化社会，老年人的就医需求在不断增长，想看病可能都排不上号，怎么会过剩呢？

从整体上来看，医院确实是供不应求的。根据《2023年我国卫生健康事业发展统计公报》，全年入院住院人次增速达到了22.3%，远远超出床位量的个位数增速，短缺现象比较明显。

但从结构上看，就完全不一样了，不同等级的医院面临的需求量截然相反。

简单说，中国的医院分为三个等级：一级、二级、三级，一般等级越高，综合水平越高；每个等级下，再细分甲、乙、丙，其中甲级最高，乙级次之，丙级为基本达标。

其中一级医院，包括乡镇卫生院、社区医院等；二级医院就是很多县、区医院；三级医院就是很多市里的医院了，而更有实力的医院，就是大家说的“三甲医院”。

那么，不同等级医院目前的需求量分布是什么样的呢？这里主要讲讲二级和三级医院的情况。

数据显示，三级医院的门诊、住院量占医院总服务量的比重，在2023年双双突破60%；而二级医院诊疗人次占医院总诊疗人次的比重，却从2019年的34.9%，持续下滑到了2023年的28.64%，入院人次占医院总入院人次的比重在同一时期，由39.56%连年下滑到了30.74%。

从数据可以看出，此消彼长的态势非常明显。三级医院的需求是在持续上涨的，而二级医院则每况愈下。很多二级医院，在过去过度扩张了院区、新分院，增加了大量的设备，却因为就诊需求的持续流失，导致无法消化扩张的医疗产能和过重的负债。

同样不可忽视的，是在过去的几年中，很多中小城市，尤其是东北和西部地区的人口外流日益严重，一方面常住人口在减少，另一方面，越来越多的医院升级为三级甚至三甲医院，这种情况下来，一些地方的三甲医院都不能吃饱，何况是二级医院。

除了居民直接需求的降低之外，医疗政策、医保政策的变化，也在挤压这些二级医院的生存空间。

比如药品加成被取消。2017年，药品“零加成”政策全面实施，医院不得对药品额外加价，必须以成本价向患者销售。随后2019年，公立医疗机构的医用耗材也开始实施“零加成政策”。这个政策极大地减轻了老百姓的负担，但它带来的连锁影响就是，医院的药品、器械收入不断萎缩，对应的补偿办法则是提高医疗服务的收费和增加财政补贴。这个政策对很多地方作为主力的三级尤其是三甲医院来说，还撑得住，但是对一些经济和人口不强的地方的二级医院来说，它们实际上并没有更多“腾笼换鸟”的途径来消化，甚至经常财政补贴也不到位。

## 医院“过剩”的影响

在我看来，近几年很多二级医院的“过剩”，其实是多方面的综合结果：既有患者就医需求的减少，也有前几年扩大产能的影响；此外，在过去这几年，医疗一方面在提高质量，另一方面在下沉基层，所以优质的三甲医院和基层医院两头，获得了更多资源。而中间层的二级医院，则难免会面对多重挤压，愈发艰难。

另外，很多属于一级医院的乡镇卫生院，过得也并不好。

不过这些医院和国家层面也在积极破局：

一方面，是医院在主动降级。比如陕西宝鸡高新中医医院、四川广元皮肤泌尿专科医院等，从二级医院降到一级；临夏市明瑞医院，更是从一级医院降为门诊部。更夸张的是乐山华城耳鼻喉医院，三级医院的名头直接不要了，自降到二级。

它们之所以主动降级有两方面的考虑：一方面，对于不同等级的医院，国家的硬件设施、人员配置、科室设置的要求完全不同，降级之后，可以减少人员和不必要的投入，控制经营成本。另一方面，低等级的医院，医保往往会给予较高的支付比例。

而另一个办法，是政策层面的转型引导。国家卫健委2025年7月发布了《关于印发医养结合示范项目工作方案（2025年版）》的通知，明确提出要“引导支持医疗资源丰富地区的部分二级及以下医疗卫生机构转型为康复医院、护理院”。我的理解，就是医疗资源丰富、竞争激烈的地区，一些弱二级医院和一级医院，就别做综合医院了，面对老年人做康复医院吧，甚至都别做传统意义上的医院了，做护理院吧。

最后，再简单谈谈“结构性的医院过剩”的影响：

首先，对医生群体，特别是部分身处弱二级医院的医生来说，面临薪资下降、发展前景有限的压力，往往面临两个抉择：要么向上流动，努力进入大三甲医院；要么可能要向下流动，或者跟着医院向康复医院等方向转型。这些都需要提前打算。

其次，对于还没毕业的医学生来说，也面临类似的风险。实际上，这种挑战可能不仅限于二级医院，前面提到了三级医院的亏损现状，因此医生这个行业越来越明显地显示出就业竞争内卷和薪酬降低的问题。

因此对医学生来说，未来在选择医院方面要特别注意，建议尽量选择人口流入的东部城市，谨慎选择人口流出的东北和西部城市。

另一个现象是，今年看起来临床医学这个以往的超热门专业，正在降温。山东省委机关报《大众日报》的报道说，记者统计了37所高校的数据，这些高校今年临床医学专业在山东的投档位次，都出现一定程度下滑。

比如清华大学、北京大学临床医学类最低投档位次，比去年分别下降了26个位次和34个位次；浙江大学医学院临床医学（5+3 一体化），下降了283个位次等。一些非名校下降得更厉害。不过我也提醒下，一些学校位次下降有比较复杂的原因，比如本科的扩招，很多双一流院校的医学院都在扩大招生规模，这也会导致投档位次的降低。

但无论如何，医学生尤其非名校的医学生，本身就业内卷加剧的大趋势还是比较确定的。
