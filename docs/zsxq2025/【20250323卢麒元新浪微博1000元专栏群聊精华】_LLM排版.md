
# 卢麒元新浪微博1000元专栏群聊精华

黄金终于站在3000之上了。很多朋友认为很高了，
或者这才仅仅是开始。很快，人们会习惯3000,
并开始建立底部的感觉和认知。须知，当大家都疯狂的时候，才是游戏结束的时候。
昨天的聊天未能尽兴。历史说得多了些，现实谈得
少了些。请注意我的叙述，贝森特并非做空美国，
贝森特的目的是做空全世界。当然，首要目标就是
中国。关税战、贸易战、海运战、汇率战紧锣密鼓
的到来。
我今天使用速冻经济的概念。我想贝森特们也是这
样设计的。必须冻结商品流转、资本流通、人员往
来，才能使全球经济进入休克状态。这样美元重置
的外部条件就具备了。现在，他们干的就是这套逻
辑。
我有些忧虑，我国在预判对手方面，仍然是在被对
手塑造认知逻辑。我们的声音被精准屏蔽，举国上
下并不认同贝森特们做空全球的推论。也确实没有
预置预案，现在启动经济的思路，与应对一场危机
的思路完全是两回事。
我在密切关注中东局势。因为，我认定那是速冻全
球经济的起点。也可以说，那是美元重置的起点。
一些专家学者对我有意见，说我不使用学术语言。
我不用他们的语言，就是为了跳出他们的语境。请
注意：美元重置；速冻全球经济。我会给出精准的
定义。我就是不屑于使用他们的语言。
巳日有孚。其实，时间已经很近了。会不会大洋彼
岸的资本市场出现崩盘，我无法给出精准答案。不
过，我仍然希望大家谨慎一点。至于国内股市，我
不觉得有大问题。提醒一句，短股要在短字上面。
不要太过激进，真正的风暴还在路上呢。

这个策略就一定要坚持下去。再说一遍；按既定方
针办。
