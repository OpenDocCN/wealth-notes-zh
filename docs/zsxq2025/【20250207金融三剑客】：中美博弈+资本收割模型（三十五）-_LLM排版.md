
# 中美博弈+资本收割模型（三十五）

## 一、低保

说难听点，那种人无论他在欧洲，还是在非洲，都只能吃低保。对于一个只能吃低保的人来说，那当然是发达国家更好。可是在我们国内，有很多白领他的年收入都还没有到10万7千块(包括上面讲得医师律)。那老美那里贫困线定得高意味着什么?意味着低于那个收入，你在那里当老板是招不到人的。

你想啊，人如果能够通过救济就能领到免费的东西，为什么还要去给你劳动来获取呢?多一个在他看来类似于被压榨的环节呢?这也就倒逼他们美国企业的利润率，不可以那么低(人工成本因为最低贫困线被动抬升了)。

## 二、内卷

可是我们反过来，当以上这两点都不存在的时候，这些人也不可能说年收入低于10万就有地儿领救济。也就是说，咱们这里老板看到的是，外面有得是愿意工作的人。so, 即便你现在对外垄断了某个行业，你也很难抬高利润。因为敌人并不在外部，敌在本能寺，你的竞争对手，就是咱们自己人。

换言之，咱自家的商人们会去抢你的这门生意，利润再微薄，他都要抢，为啥?因为他总是能够找到愿意低价来干活的人(背后各地操盘手求之不得啊，你现在知道内卷怎么来的了? )。

所以就会出现一个奇观，你明明已经抢占了整个行业，行业也很好，很有前景，需求很旺盛，但是老板们就是赚不到钱。就是胖东来老板说的，不赚钱的事情，这帮人还整天忙个什么劲?

试问，如果连老板们都赚不到钱，你觉得员工们能赚到么?你看着这些行业的从业者们很热闹，但是却没有留下利润。因为并没有在挣钱，大家只是在争就业。

所以，如果你非要问剑客：怎么在一群人和你卷就业的局面下赚到钱，你让我怎么回答你?

## 三、风口

所以说，我们实际上平日里利润是很微薄的，制造业更是薄如刀片，少数人发财并不是靠平日里一单一单卖商品赚来的收入，而是靠一次又一次的捕捉风口(风口原理)。谁抓到了，谁就发财。

这也是为什么国内这么多人，从富有到贫穷，从赚钱的成年人，到报考志愿的学生，都在把目光聚焦在捕捉风口上。

这就是剑客在中信系列里讲过的那三大改革。像过去的房改、现在的新房改、土改等等，改来改去背后更深层的本质是：

上峰到底允许谁可以富起来!

因为大家默认的赚钱的路径就是风来之前先卡位，然后把卡位的那个位置，资产变现给接盘者。大家放眼望去，大部分有钱人，包括中产的钱，都是这么挣的，所以大家当然会习惯性

的这么看问题。

但是，太多人，只是凭借命运的安排，在随机的碰撞风口，他们根本就没有研究过风口的原

理。比如风口是怎么诞生的，为什么会诞生，以及会在哪里诞生。

这些，是很多人不曾研究的，那么凭运气赚到

的钱，最后就会凭本事输掉，因为他连好风凭借力，送他上青云的那个风，都不想弄清楚。

## 四、层次

但与此同时，对于赚到了的读者们，我们更要明白，那只是个游戏。明知是游戏而玩游戏则可，不知是游戏而精神内耗，则大可不必。

你看到了董宇辉那位置，真正能带给他进一步快乐的，是他能提供很多很多的就业岗位。但现实是什么?他有100个小目标，只有几十人的小团队；别人有100个小目标，却有几万人的大团队。那后者在国内的生命体验，是前者所仰望的。

前者是个顶流网红明星，而后者，是业内巨

佬。

不可能有了。有些事生来有就有，没有就没有了。

换言之，你生来没人罩你，这辈子就没有了。你后天所有能找来的所谓的那个后台，都是来

利用价值，分分钟就会把你当脏了的手套清理

## 五、进阶

而美国在这方面就发展的更进一步了，他们其

实是不太在乎本国底层民众的。

按常理说，上层的利益是来源于下层的规模和

出油率的。但现在欧美社会的种种抽象行为，

都在将本国的中下层民众导向自我毁灭的境

地。

那从食物链的角度看，难道它们这些狼就不害

怕羊群不够了，狼也会被饿死么?

其实它们现在还真的不用担心，因为当社会生

产力发展到一定阶段之后，大多数人即便什么

都不干，也不会饿死。即便是最底层的流浪汉

也能获得免费食物。

但是，中下层的民众并没有资格进入主流社

会。即便是采取公开、公平、公正(三公 )的

选拔方式，也让他们拥有自主的选择权，他们

也依然进入不了正常节奏的主流社会。原因很

简单，“脑袋”已经堕落的坏掉了。

其实就算是纯靠运气，也是不公平的事情，这

个比较隐晦，它和身高体重一样，每个人都不

同，只不过几乎所有的共识都认为人和人的运

气是一样的。所以，抽奖，抽签，抓阄，大家

都觉得是正常的。

话说只要按照看得见的属性去分类，大家就

觉得不公平，按照看不见的去分类 (比如彩

票), 所有人就觉得好公平，大家概率一样，

一样个P啊。

话说如果一个人你运气差，可能别的地方会

好，偏财不旺正财旺也有可能，财运差点身体

健康也是可能的，反正每个人的初始属性不

同，善于发现自己的优点是需要大智慧的。

你再看英国，它的社会结构同美国也差不多，

40%的收入都来自于海外投资，那你觉得这种

投资跟英国的中下层人群有关系么?为什么小

英不遗余力的支持乌克兰战争?

其实并不是出于什么国际上的道义，纯粹是出

于他上层自身的利益。

因为脱欧之后的英国是美元在欧洲的离岸中

心，它在后面借小兰打击欧元区，会让欧洲的

财富因为避险需求而切换为美元来保值，从而

会优先流向自己的伦敦金融城。这是英国精英

们的小九九。

而美国社会的小九九也差不多，上面的精英人

群卷的非常厉害，它们主要从事金融、科技、

人工智能、航天和生物医药等高利润的产业。
