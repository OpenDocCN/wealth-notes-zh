
# 气运是个啥

## 猫哥原创

今天我们不谈唯物主义，讲一讲唯心主义。熟悉我的读者都应该知道，我是比较相信气运的，在过去文章中多次谈到国运、气运这个问题。对于这个问题，很有读者都有意见，认为气运是虚无缥缈的，甚至就是不存在的。个人也好，国家也罢，要成功都必须靠自己努力。那么气运真的就不存在吗？个人努力就一定能成功吗？

我们来看一段历史。以汉代为例。任何朝代都会出现权臣，也就是那种在朝堂之上能一手遮天，相权大于皇权的权臣。

汉朝第一个权臣是霍光。霍光在历史上评价是很高的，西汉能够中兴，很大程度就是霍光的功劳。但是霍光的结局却不大好，霍光死后，其家族被霍光身前一手扶持拥立的汉宣帝杀得干干净净。霍光被灭族历史书上的理由是家族谋反，但是所谓的霍光家族谋反一说其实也是被汉宣帝逼迫的结果，即便霍光后人没有谋反，汉宣帝大概率也会找个理由将霍光家族杀光。

为何？因为霍光身前干了废立之事，对于皇权而言，权臣行废立之事——即便自己就是这个废立之事的受益者（汉宣帝），也是无法容忍的。万一以后有其他臣子效仿，对皇帝威胁就太大了，所以一定要给后人树一个标杆——凡是行废立之事的权臣，其子孙都会被族灭。

这就是霍光后代被族灭的权力逻辑。霍光之后，第二个权臣是王莽。王莽一看霍光的先例，怎么能不吸取教训？既然干了废立之事，未来就一定没有好下场，为子孙计，只能一劳永逸的解决这个问题——自己去当皇帝。

但是王莽篡位之后，名声一下子就臭大街了，全天下群起而攻击，最后王莽成为一个短命皇帝，子孙后代也被杀个干干净净。

王莽之后第三个权臣是曹操。曹操一看历史，不能学霍光，否则后代下场不好，所以篡位是一定要篡位的，但是不能学王莽，这个篡位也要讲究水磨工夫，自己不能直接干，得让儿子干，也就是花费几十年时间让天下人慢慢接受曹家代替刘家的事实。

曹操算盘打得很好，但是奈何气运不顺。儿子篡位称帝之后，因为太喜欢甜食得了糖尿病，很快就挂掉，孙子继位之后也是英年早逝，三代人都死掉之后，以后的子孙就不打成气了，偏偏遇到一个老不死的司马懿寿命太长，熬死了曹家三代人之后成为权臣。

高陵之变后，司马懿将曹真全家杀了干干净净，环顾左右，好像没有权臣能威胁司马家，然后准备学习曹操的先进经验，自己不篡位，留给儿子篡位。但是这个篡位时间拉长了就有变数。权臣没有了，偏偏遇到一个傀儡皇帝曹髦性格刚烈，带着几百个仆役就要去诛杀司马家，结果在大街上被司马家的人诛杀，当街弑君，司马家的名声一下子就臭大街了。

历朝历代基本意识形态都是以忠孝为牌坊，司马家当街弑君实在不好意思再谈忠，最后只能以孝立天下，意识形态的缺陷导致司马政权先天不足，后面才有了八王之乱，五胡乱华，司马家子孙结果也不好。

从以上案例我们可以看出什么？从王莽到曹操到司马懿，哪一个不是充分总结前人经验，然后制定了相应的补救策略，本来都希望子孙能有一个好结果，但是偏偏在执行过程中出现各种变数，导致策略失败。

这些是王莽、曹操、司马懿等等不够努力吗？不是。就是他们气运不够。

所以，老祖宗给我们留下一句话，谋事在人成事在天。能不能成事不是自己努力够不够，而是老天爷给不给你这个气运。老天爷不给，再努力也白搭，霍光、王莽、曹操、司马懿对此应该纷纷点赞。

还有一句话叫做科学的尽头是玄学，人类历史上最聪明的人从牛顿到爱因斯坦都是相信这个世界是存在造物主的。所以气运一说严格意义上不是简单的唯心主义，而是更高层面的唯物主义。
