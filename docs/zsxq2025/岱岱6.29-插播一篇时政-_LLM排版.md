
# 岱岱6.29 插播一篇时政

今天暂缓统战论系列，先插播一篇时政。
现在国内国外局势，到了一个屏息等待的微妙时刻。
在几天前，我就知道，河里会议开了。
是的，提前开了。
按理说是8月开的，提前到七月初开了。
河会绝对是提前的。
因为我们本来预定七月初去参加巴西的金砖会议的。

这些年是中美博弈的高潮，我们很注重金砖上合g20这类国际会议，除了三年口罩没出门还有上次印度主场举行g20因为边境摩擦我们没有去外，其他国际峰会我们都没有缺席过。
这次我们缺席，原因就是河会提前开了，冲了7月初的外访计划。
我之前说过，河会提前开，是不利于我们的信号。
要知道，我们在6月初的时候是开了一周左右的闭门局扩大会，局会的结果是我们和懂王打了那通电话，有了关税战的第二轮伦敦会晤，以及糊弄懂王般的答应懂王来访华和我们访美。
6月中旬我们去中亚开会，那么6月懂王访华的算盘就落空了，懂王只能指望7月8月能访华成功。

现在看来，6月局扩大的另一个结果，就是提前召开河里会议。
这次提前召开河会，必然会谈对美关系，谈经济问题，甚至谈人事梯队问题。
结合相关新闻就可以看出。
1、这几天小丁重庆一号等人的出镜率很高，对应人事梯队问题。
2、在河会召开前夕，苗案尘埃落定，当然还有一个最新的海军参谋长落马。

河会必然会聚焦套利这轮枪杆子反腐，这关难过。
苗会被定性为有政治野心是两面人，这个你懂的。
同时，审计署罕见的大尺度披露了地方债问题。

地方债问题也会摆上桌谈，这个是央地博弈了。
内事纷乱，我们为了稳住内部，估计对美是作出经贸的妥协的。

因为在召开河会的几乎同时，懂王说中美达成贸易协定了。

中美都没有曝光具体协议，考虑到我们这边河会提前，急于熄火外部关税战来做好内部斗争准备，估计这次中国没占到多少好处。
甚至懂王蠢蠢欲动，似乎认为这次河会有利于懂王的访华。

河会已经从八月提前到7月初开，已经是不利的信号了。
如果懂王访华从9月阅兵提前到七八月访华，那么不利信号会更进一步——进一步锁死七八月的梧桐可能。
我们当然无法知道会议开的到底怎样了。
只能旁敲侧击的去分析。

按惯例，河会会开三天，第一天地方大员参与谈经济问题，第二天地方大员回去元老们继续开谈政治人事，所以地方大员要规避，第三天则是总结结束。
这是惯例的开法。
如果开的比较漫长，不止三天，那就是拉锯战。
如果地方大员不仅呆了一天，而是呆了两天乃至三天，那信号就更加玩味了——战局更混乱。
胜败利钝究竟如何，现在还未可知。
想起四爷的畅春园之夜。
康熙在畅春园的最后一晚处于弥留之际，宣召各路皇子，四爷前去内心忐忑不安，不知前途命运如何，不知自己是一步登顶还是跌落深渊，四爷出门之际雪夜回头，令人感慨。

不用等到四中，也不用等到明年两会，这波内事到底如何收尾，对内对外政策是否会迎来拐弯，特别是人事问题究竟如何，这一切随着河里会议的推进，已经是时候慢慢给出答案了。
我们的畅春园之夜时刻，差不多到了。
四爷会也回头，还能看到邬思道，我们这次雪夜回头，怕是连一个有足够能力可以信赖的邬思道，都没有。
一切祝好，静等消息吧。
