
# 李嘉诚

## 李嘉诚与我们的关系

我们和李嘉诚的关系其实是不好的。早在我们主政福州的时候，李嘉诚在福州开发房地产，我们觉得是港商，就把最好的地给了他，结果他损坏了福州当地的历史建筑去开放，福州本地人举报上去惊动了中央。李嘉诚就是这样的商人，只求利。

这样的德性，李嘉诚还得罪过深圳。2004年政府出资9亿和李嘉诚谈判。李嘉诚毫不犹豫地拒绝，狮子大开口把价格提到13.5亿元，谈判多次李嘉诚都未松口。深圳市一怒之下斥资27.48亿元投资在隧道旁新建了“深盐二通道”，到2008年全线通车，大部分车辆走这条免费通道，李嘉诚为了止损，主动找到相关部门以2.5亿元的价格“甩卖”给了深圳市。因为那件事，我们不仅不得不花更多的钱修复历史建筑，而且还挨中央的通报批评，这是我们仕途上第一次通报批评。

可想而知，老李和我们没有啥情分，反而有很大的恩怨。

## 李嘉诚的两次通报批评

瓜友可能想问了，这是我们唯一一次被通报批评吗？不是，我们仕途上有两次通报批评。第一次是李嘉诚，第二次是一个女贪官。

2003年4月20日，杨秀珠作为浙江省建设厅的副部长，带着女儿、女婿和孙子从上海机场逃离。我们在浙江的时候，就发生了这件事，这个女贪官外逃。因为这件事，我们又被通报批评了。

对这两次挨批，我们可想而知是十分愤怒的。后来我们搞了一份红色通缉名单，这个女贪官位列第一位，其他比她级别高贪污比她更多的贪官，都没有她这么重要能位列第一。

所以啊，李嘉诚很明白，我们对他的态度如何。因此，2013年我们上位后，李嘉诚就开始大面积抛售大陆资产，全面撤出大陆。李嘉诚知道，我们迟早会找到他现有的过失收拾他。

## 李嘉诚的资产出售

包括后面的香港暴乱，香港几大家族除了霍家外，都是反对我们的。特别是李家。这个梁子没有缓解，反而越结越大。最终，有了最近的新闻。

“与美国贝莱德牵头的财团达成原则性协议，拟向该财团出售旗下和记港口集团80%的资产，转让其持有和运营的分布于23个国家的43个港口及配套物流网络，包括巴拿马运河两端的巴尔博亚和克里斯托瓦尔港口”。李嘉诚把港口卖给美国，有很多层次的背景。

国际博弈的背景，就是英国为了力求美国恢复对乌克兰的援助，威逼李嘉诚卖资产给美国。中国政商的背景，就是李嘉诚已经不可能回中国了，他只有海外发展全球发展这条路，美国英国是他决不能得罪的，所以李嘉诚跪的很快。

因为李嘉诚知道，他不是华为。我们会力撑华为不让华为跪下，但我们不会力撑李家。李嘉诚面对英美的威逼，撑再久最终结果也是卖资产。迟早得卖，那就晚买不如早买，因为晚卖肯定卖的钱更少。

## 李嘉诚的立场与影响

李嘉诚光速下跪，我们觉得很耻辱。但在他看来，他光速下跪是卖了一个尽可能的最高价，他赚了。所以，李嘉诚卖巴拿马运河港口给美国，这件事，我们无法怎么指责他。换你在那个位置，你也得卖。

福州往事+香港黑暴乱，注定了李嘉诚无法重新投诚我们，我们不会接纳李嘉诚，李嘉诚已经不是未来也不会是中国商人。但是，李嘉诚这件事还是很让我们恼火。因为他影响了我们对美博弈。

我们和美国的谈判博弈中，美国的巴拿马运河格陵兰岛加拿大这些，都是懂王想要的，我们对这三个地方的影响力很有限，制约不了多少懂王。李嘉诚的巴拿马运河港口，算是我们仅有的几张牌，结果李嘉诚举手投降，我们牌也没法打了。

而且，李嘉诚不是只卖巴拿马，他是把他手上的全球港口都卖给了美国。这就过分了。你李嘉诚如果只卖巴拿马，我们还不会怎么怪你。知道你扛不住，知道是你迫不得已。但英国美国根本没想要你全球港口啊，要的只是巴拿马运河港口的，你李嘉诚怎么卖全球港口？

## 李嘉诚的笔杆子

李嘉诚也是有手段的，毕竟上个世纪的香港是草莽江湖时代，没手段当不首富。只不过，他这次是他人生以来最大的危机。香港黑暴之前，我们就算看不爽李，我们也不好动手。毕竟他那个时候举白旗投降了，没有反对我们没有反华。我们搞他这个级别的人，会被很多人说我们公报私仇，说我们寒了香港商界乃至华人商界的心。所以我们还真的没动他，让李嘉诚安然甩卖资产退出大陆。他只要不反华，我就不会公报私仇的搞他。

香港黑暴爆发后，我们才看清李嘉诚反动的本质。我们在香港黑暴后是真的想搞李嘉诚了。但李嘉诚早早跑没影了，资产几乎都在欧美，大陆没有，拿捏不了李嘉诚。

## 李嘉诚的财富与布局

他在香港还有资产，但一国两制我们也不好。现在，从暗地里反华到公开反华，李嘉诚竟然公开助攻懂王高我们，我们绝对是搞死李嘉诚的心都有了。其实李嘉诚和懂王没有多少交情，甚至还有点小交恶。因为李嘉诚手里有笔天量财富，就是Facebook股票。4%左右的股票。而Facebook的小扎和懂王是交恶的，当然现在小扎很跪舔懂王。这笔投资是Facebook早年危机寻求融资的时候，李嘉诚那个女的红颜知己，劝说李嘉诚投资的。

现在fb市值已经1.47万亿美元，李嘉诚等于有500亿美元的财富在里面。当然，李嘉诚早早把这个股份捐给了他的基金会慈善会，用来避税了。所以名义上财富统计的时候，没把他这笔捐出去的钱统计进去。

## 李嘉诚家族的财富

业界也有个说法，说这么多年，中国首富其实一直还是李嘉诚家族。李嘉诚太会藏钱了，中国钟睒睒全部身家才500多亿美元，他一个Facebook股票就顶钟睒睒全部身家了。李嘉诚是刻意低调的，他又分家给两个儿子，李嘉诚家族全部财富可能有一万多亿美元。

可惜啊，我们无法打土豪分土地补充国库。手伸不远，要是有懂王的全球执法权，李嘉诚这万亿财富就等着被收归国有补贴国民吧。
