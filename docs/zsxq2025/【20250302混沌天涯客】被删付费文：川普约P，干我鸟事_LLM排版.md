
# 被删付费文：川普约P，干我鸟事

“三十亩地一头牛，老婆孩子热坑头。”这是一句名言，出自一位著名人物之口，时间是在1947年，地点在东北。这句名言描绘出一个梦幻场景，该场景世代代代萦绕在小农社会每一位十八代农民心头：有地，有牛，有老婆和热炕，养得起孩子。这一画面感十足的美梦，激励着1947一代农民兄弟放下锄头，踊跃加入组织，水里水里去，火里火里去，赶跑了蒋光头及其走狗，实现了美梦。

我爷爷即是那一代农民，他没分到牛，但分到了地，娶到了老婆，生养了孩子。虽然没几年地又收回了，但老婆和孩子没跑路，五个孩子躺一条炕上，饿是饿了点，但都长大了。

我爷爷那时候能实现的梦想，臣妾如今是做不到的了。五个孩子能吓尿，四个孩子要吓醒，三个孩子吓出一身觉悟，两个孩子勉为其难，一个孩子足足够够。从生物繁衍的角度，这是一种倒退。从我爷爷的黄金时代，倒退到现在的好时代。无独有偶，同样的倒退也发生在美国。美国的黄金时代在上世纪40年代末50年代初——即我爷爷分到地的同时期——达到高潮。

携二战余威，美国的农业、工业、科技均繁荣的无以复加。落到一个普通美国人身上，哪怕是一名水电工、装卸工、挤奶工，只要按部就班工作，不用996或007，就可以住得上house，娶得到老婆，生养四个孩子，楼上楼下，电灯电话，家里有车，每周超市大采购。

那时候，美国的公路宽阀高楼摩天，纽约地铁是世界样板，U2 飞机让地球惊叹。现如今，那一段美国梦已经过去很久了。U2飞了67年仍在飞，纽约地铁充满尿骚味，摩天大楼变得灰暗陈旧，公路勉强还能跑，但远比不上东大的普通标准八车道。落到普通的美国家庭头上，一个产业工人有房有车养四个孩子的美国梦，做梦去吧。美国产业空心化，工人找个工作都难。原来的世界工业基地五大湖地区，失业率常年居高不下，到处是脱不掉长衫的孔乙己。

失业导致失意，失意需要麻木和刺激，由此，芬太尼等药物在美国泛滥。原本的产业工人不仅没了工作，也没了身体。这种情况，在副总统万斯的自传回忆录《乡下人的悲歌》刻画的淋滴尽致。这本书曾连续两年高居美国畅销书榜首，万斯凭此书打响名气，被川普看中，成为MAGA 党的后起之秀。

在一名乡下人看来，那些高尚的英明的光荣的业务，什么非洲儿童饱受寄生虫困扰，中东群众打得不可开交，南美洲雨林又少了三百亩，俄大哥乌小弟三年里排放的二氧化碳到底知多少。。。这些东东，一言以蔽之，干我鸟事。

这就像是那个年代里，没分到牛却被分配在生产队耕地的我爷爷，听着村里的大喇叭广播：阿尔巴尼亚兄弟又来分享我们丰收的喜悦了，巴铁又来感谢我们的国际胸襟了，巴拿马对美帝的斗争正使用我们的法宝，柬埔寨的高棉弟兄在活学活用我们的精神。。我爷爷耕地间隙难得停下来抽口早烟，边抽边说同样一句，关我鸟事。

说这样的话，觉悟很低级。我承认我爷爷觉悟很低，但看在他为生产队干了半辈子活临了没领过一分钱退休金的份上，应该允许他觉悟是0分。无独有偶，商人川普和乡下人万斯的觉悟，在当下的世界看来，也是0分。

川普上台之前曾发出豪言，24小时结束俄乌之战。上台后的第一天就被记者逮住问，你都上来小半天了，战争结束了没。狡诈的川普打个哈哈，我还有半天时间。现在来到3月份，川普上台已满40天，虽然战争仍未停止，但全世界都看到了，川普是真的想让战争停下来。

停战，才有和平。和平了，才好做生意。虽然打仗本质上也是一门好生意，但川普这辈子做的生意是房地产、度假村、高尔夫球场、娱乐电视秀……他没做过跟打仗相关的生意。在他看来，放下生意改打仗，实在是很划不来。他反反复复念叨，三年里美国向乌克兰掏了3500亿美金，这么多钱掏了，却什么都没换来，只有交战双方伤亡百万人。关键这并不是终点，虽然普大哥和小泽司机都表示期盼和平尽快到来，但你推我我推你，没有一个人肯放下武器。

看架势，这仗还要继续打下去，没完没了。按老规矩，美国还要继续掏钱给武器，看不到尽头。

没办法，这世界就是这样，无论东方or西方，都被架到了某种高尚英明光荣正确上。东方有东方的正确，西方有西方的正确，必须要沿着这个正确的模板行事。乌小弟说领土一分也不能少，这代表着无上的正确；俄大哥说自古以来这些土地就是我的，也具备着不可反驳的高尚。为子维护高尚和正确，打仗就是必须，为打仗花的钱死的人排放的二氧化碳就不算个事。

多讽刺啊。一边喊着节能减排气候变暖，一边源源不断提供着炮弹。在西方，哪个政客不同时做这两件事，就是人类公敌。偏偏上来个川普，搭配着乡下人万斯，两件事都不想干，一上来就退出“巴黎气候协议"。然后针对俄乌，两人合起伙儿在世界人民面前演了场戏。

不是世界人民，而是美国人。川普喊的口号是美国优先，他以压倒性优势在选举中获胜，选票均来自美国人。

这场戏演得太成功了，都在看，都在传，都在讨论。在世界人民的眼光里，小泽的身影多了一层悲情，以前他是扛俄，现在又加上杠美。两大强敌一肩挑，我命由我不由天。

但问题的关键是，川普与万斯根本就不在乎美国之外的世界人民。这一场戏在美国人眼里会是怎样?

很简单，川普尽力了，他尽力为美国争取利益，尽力让俄乌停下来签协议。虽然川普的协议俄乌两方面都不满意，一边骂他投降，一边批他无耻。但在美国人看来，他们满意不满意，干我鸟事。

太阳底下没有新鲜事，历史上很长一个时期，美国是奉行孤立主义的。面对世界纠纷，高喊关我鸟事。历史也一再证明，孤立的美国是最可怕的，尤其是对我们。欲知详情，且待下期。
