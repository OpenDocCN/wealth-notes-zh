
# 当世界进入“战国时代”

目前有一种说法，即人类社会正逐渐由“春秋时代”进入“战国时代”,现在可能处于“春秋末期”的感觉。春秋时期的政治逻辑是通过战争、外交等各种手段去争霸，争一个国际地位、势力范围或话语权，很少涉及谁灭掉谁；而战国则是赤裸裸的灭国吞并，你死我亡，手段更加残暴。

在“战国时代”,核武器与核战略的意义将急剧放大，获得核武器的国家可以获得更强的战略权力，而没有发展核武器的国家则将沦为猎物。1964年4月，美国国务院政策设计委员会曾起草题为《针对中国核设施直接行动的基础》的报告，提出四种摧毁中国核武器设施的办法，包括公开的非核空中打击和秘密攻击等形式。

作为现阶段人类手中的“最终暴力”,核武器自诞生之日起就对国际秩序产生了深刻影响。二战末期，美国之所以能够“随心所欲”地安排战后秩序，经济实力固然是一方面，长崎和广岛两颗原子弹的威力同样功不可没。1949年8月29日，苏联试验了首颗原子弹“斯大林1号", 标志着两个超级大国间的核军备竞赛正式拉开帷幕。

冷战期间，美苏核力量平衡是维持两大阵营均势的根本性因素，核力量向下才是常规军事力量。随着冷战结束和苏联解体，全球核态势发生了显著变化，最紧迫的威胁来自前苏联核武器/核原料扩散，于是美国开始力推“核不扩散”政策。某种意义上讲，当今国际秩序其实是建立在“核不扩散”的基础上，而“核不扩散”又建立在美国庞大的盟友体系以及“延伸威慑”战略上。

什么叫“延伸威慑”战略呢?简而言之，对于那些有能力拥核的“友好国家”(如日本、韩国以及一些欧洲国家), 美国同其签署安全保障条约，承诺将他们纳入“核保护伞”,从而避免其独立拥核。如此一来这些国家便跟美国形成了坚固的安全绑定关系，并以此为基础延伸到经济、贸易和国际外交领域，最终形成“以美国为首的西方世界”。

对于那些试图拥核的“非友好国家”(如朝鲜、伊朗、伊拉克等), 美国予以巨大军事压力，实施强硬制裁，甚至不排除先发制人式打击。那么问题来了，如果美国的安全保障和军事威慑不再可靠，会产生怎样的影响呢?简单来说，越来越多的国家或地区(包括美国的盟友)将尝试拥核，进而加速点燃潜在的战争危机——核扩散进程最危险的时刻是"即将跨过核门槛的时候"。

在这些对拥核跃跃欲试的国家和地区中，一半在欧洲，另一半就在中国周边。驻台南基地的美军于营区内形成一个完整的生活圈，一般台湾军民不允许进入。1952年至1961年间，美军曾在台南空军基地部署12枚原子弹。《外交事务》杂志近日刊登了一位德国学者的文章，题为“为下一个核时代做好准备”。

文章认为，美国政策制定者在经历过两次世界大战和经济危机之后，于1940年代开始重建国际秩序。他们从二十世纪上半叶吸取的教训很简单：如果纯粹根据粗暴的短期利益行事，将导致各国采取以邻为壑的经济政策和推卸责任的安全政策，这反过来会造成经济与社会动荡，促进侵略性政权的崛起，并最终引发世界大战。为了避免这种模式重演，华盛顿尝试以开明的长期利益为出发点，将国际政治改造为一个正和游戏，参与者可以共同成长，而不是以牺牲彼此为代价。

当然，这一构思从一开始就建立在美国拥有绝对实力和超然地位之上。经历过几十年的权力消长与弊端累积后，目前Trump政府正迅速瓦解战后国际秩序的关键要素，并没有认真考虑其行为可能导致的一些重要后果，比如引发新一轮的核扩散——这一次不是恐怖分子或伊朗朝鲜，而是美国的盟友。

核不扩散制度是一种自愿协调的国家约束行为，绝大多数国家都愿意遵守这一制度，因为大家觉得有这个制度比没有这个制度更安全。但这种安全很大程度上依赖于一个更广泛的国际体系，具体来说，是由整体上温和的国际体系。而现在Trump正在撕碎这个美国亲手创建的国际合作体系，包括北约、联合国、WTO等机构。如果西方主导的现行国际秩序垮台，防核扩散制度也会随之垮台，除非白宫立刻改变路线，否则未来几年国际政治很可能由核危机来定义。

一种观点认为，核武器扩散越多越好，届时所有国际竞争都将通过相互确保毁灭的前景而持久稳定。可即便如此，核扩散也将加速点燃潜在的战争危机，诱使更多的“先发制人式打击”出现。以台海问题为例，岛内(尝试)拥核几乎可以被认为是大陆最红的红线之一，将直接引发战争。

青海的二二一厂见证中国第一个原子弹的研究和测试。

截至目前，世界上有九个国家拥有核武器，包括核俱乐部正式成员美、俄、中、英、法外，以及非正式成员以色列、印度、巴基斯坦和朝鲜。具体细分一下，可以认为美俄一档，中英法一档，印度、巴基斯坦和以色列一档，朝鲜单独一档。

这里面法国是个很值得关注的案例，近期被许多欧洲学者拿出来讨论。1954年，法国秘密启动了核计划，该计划于1958年公开，并于1960年成功测试了第一枚核武器。为什么法国在已经被美国核保护伞覆盖的情况下仍执意得到原子弹呢?这是因为时任总统戴高乐根本不相信华盛顿会履行其安全保证。戴高乐认为，延伸威慑就是一场骗局，要让巴黎真正安全，法国别无选择，只能自己获得核能。他在1963年这样讲道：“美国的核武器仍然是世界和平的重要保证，但美国不一定会立即对有关欧洲和法国的所有可能性做出核反应，因此我们决定为自己配备核武器。”

几代人以来，大多数非法国分析家都对这种推理嗤之以鼻，认为它反映了高卢人的过度骄傲或偏执，而不是冷静的战略逻辑。直到最近一段时间，欧洲人才突然意识到戴高乐似乎是有先见之明的。现阶段，法国与英国是欧洲唯二拥有核武器的国家——法国拥有近300枚核弹头，可从本土战机或潜艇发射，英国约有250枚。二者关键差异在于，法国核武库独立研发、完全自主，而英国依赖美国技术支援，有使用限制。

那么自主拥核与"延伸威慑"相比究竟管用不管用?很管用，能够对对方决策者构成极大影响。前文提到过2005年波兰公布的华沙组织机密档案，在1979年华约进行的“七日抵达莱茵河”兵棋推演中，苏联将对西欧发动大规模核反击，而唯—一个被排除在外的欧陆国家即法国。因为站在莫斯科的角度，为了保卫东欧卫星国或扩张势力范围对法国进行先发制人核打击是得不偿失的，风险收益不对等。

简而言之，如果是为了保卫莫斯科或圣彼得堡，苏联人有可能甘愿冒核战争的风险，而“抢地盘”这种事情本来就可多可少、可有可无，犯不着走到核大战那一步。不难体会，独立拥核确实在很大程度上巩固了法国的国家安全。

1984年大阅兵时，战略导弹部队第一次揭开神秘面纱。视察战术核弹头。朝鲜已于2006年拥有核武器，领先于伊拉克(寻求核武库)和伊朗(现在正处于临界点)等国家。

介绍完法国的例子，接下来把目光转向乌克兰。苏联解体时，乌克兰境内有约1900枚核弹头，为世界第三大核武库，这对于核不扩散显然是巨大的威胁。经过漫长而复杂的谈判，在各方威逼利诱之下，乌克兰、白俄罗斯和哈萨克斯坦三国最终与美、俄、英达成一份名为《布达佩斯安全保障备忘录》的协议。根据协议条款，乌克兰、白俄罗斯和哈萨克斯坦三国同意弃核，以无核国家的身份加入《核不扩散条约》。作为交换，三国得到备忘录签字国俄罗斯、美国和英国的安全保障。

需要说明的是，在国际外交术语中，“安全保障”(security assurances) 与“安全担保”(security guarantees) 是两码事，前者仅表示“我不打你”，后者才表示“你被打时我会来救你”。实际上，早在1994年12月《布达佩斯安全保障备忘录》签署之前，就有学者提出了质疑，比如大家熟悉的美国学者米尔斯海默。米尔斯海默在1993年的《外交事务》杂志上撰文指出，乌克兰最终需要对抗俄罗斯的复仇主义，而保持核能力是实现这一目标问题最简单的方法。他在文中写道：“乌克兰无法用常规武器抵御拥有核武器的俄罗斯，包括美国在内的任何国家都不会向其提供有意义的安全保证，乌克兰拥核是对抗俄罗斯的惟一可靠威慑。”

众所周知，米尔斯海默因俄乌战争期间的一系列言论在西方学术界颇受排斥，比如宣称西方对乌克兰危机负有主要责任；因为立场问题，这几年他经常到一些东方的场子里来寻求“温暖”。

其实归根结底，米尔斯海默等人属于冷血的现实主义派，他既是1994年极少数反对乌克兰放弃核武的学者，也是2014年之后建议基辅不要轻易相信西方承诺、认为西方支持基辅对俄强硬最终会毁掉乌克兰的人。和戴高乐一样，米尔斯海默也被证明是正确的，所谓“延伸威慑”根本就是一场骗局，任何相信它的人都是傻瓜。

1、欧洲

既然美国已经成为一个不可靠的盟友，那么寻求保护的欧洲发达小国势必会考虑从不同的提供者那里获得威慑资源，比如法国或英国。可以猜测，欧洲核共享或欧盟核共享很快就会被提上讨论日程。然而伦敦不信任华盛顿保卫它，巴黎也不信任华盛顿或伦敦，为什么其他国家应该信任伦敦和巴黎呢?因此，一些有实力的国家可能会决定自主研发核武器，或者要求拥有共享核武器的战时使用权。刚刚过去的3月7日，波兰总理图斯克在对议会演讲时表示，“仅仅购买常规武器和传统武器是不够的，波兰必须寻求最现代的可能性，包括核武器”。总理也愿意探讨法国核武器如何保护整个欧洲大陆。
