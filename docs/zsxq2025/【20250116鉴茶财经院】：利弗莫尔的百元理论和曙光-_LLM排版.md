
# 利弗莫尔的百元理论和曙光

利弗莫尔的百元理论标准是：
"如果一个个股首次超过100元，200元，并且趋势仍在的情况下，那么，它还可以再涨。"

利弗莫尔在95元买进了伯利恒钢铁。请注意两个前提，超过100，趋势仍在。它95没冲到100，那就冲击失败，趋势不再。利弗莫尔的胜率6成，已经非常厉害。

我们提紫禁之巅是十月前后的事情，有些同学大赚特赚了，还是不走。这也是我们多次讲的，管理预期，自己要有纪律，比如+20%卖， -8%走。

请问到底价值流，买公司，还是操作流，买涨跌？搞公司，你就不要管他短期涨跌，搞操作，你就要坚守纪律，哪怕不会去弱留强，也要用好说烂了的5日平台。

我们的有些同学，总是既要又要， 一会价值流，一会操作流，那也行，我们给了个箱体概念。箱体总会看吧？怕的是，你平台稳步向上，自己拿不住，卖飞了，你跌破箱体了又不走，觉得公司不行。

我们就说中科曙光：

1. 董事长，李国杰，这可是工程院院士，计算机领域泰斗，徒子徒孙遍天下，大院长当董事长的企业有几家？李院士已经80多了，领导一再要求，才继续留任。算法到最后烧的都是人才，雷布斯得千万年薪挖人，你跟中科院拼人才？

2. 再看公司，大家知道的是，中科曙光持有海光信息27.96%、曙光数创62.07%、中科星图15.67%。这几个都是上市公司。

3. 看激利润，三季度末公司毛利为26.81%，连续第7年保持增长，创下2013年以来新高。而竞争对手浪潮信息、紫光股份则在最近2-3年持续下滑，一样干服务器，山东国资委的浪潮信息700亿，中央的曙光不值1000亿？

4. 它战略和要干的事什么？用自己徒子徒孙的寒武纪的芯片，海光信息ASIC的CPU，自己去做芯-端-云-算的综合化解决方案，国内只有它和华为能干。

5. 它还布局了什么，液冷，超算，量子计算等等。前两天黄教主在CES上吹牛的液冷方案，现在字节跳动50%的液冷服务器都是曙光数创搞的。大家不是拼算力吗？大不了用最后兜底的大方案，超级计算机。神威太湖之光，每秒最高峰运算12.5亿亿次，持续计算速度每秒9.3亿亿次。美帝在这个领域已经放弃了。

这样的公司，大涨时候你没卖，跌下来值得埋怨吗？你会因拿个垃圾而难受？

国资减持，很可能是去布局用的，不然你怎么过会？懂的同学都懂，就算不投新项目，减持的2%，6个月内不准卖。有人帮你锁半年这个价格。公司没有一直在涨的，也没有一直会跌的， 一体两面。

最近市场环境不好，主要是央行等这些人马去干预汇率，市场暂时是个空档期。

在3080-3150之间横盘做个箱体。大国的趋势和国运怎么走，格局还不够明朗？以前你不行，别人骂你中国威胁论，现在你看美帝和英法，还说你威胁吗？因为你真有。真有了，还用得着端起碗吃肉，放下碗来骂娘，赌国运没有三天耐性？困难不会太久了，"我们的同志，在困难的时候，要看到希望，看到光明"。
