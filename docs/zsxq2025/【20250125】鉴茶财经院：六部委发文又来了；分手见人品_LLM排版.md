
# 鉴茶财经院：六部委发文又来了；分手见人品

## 一、六部委发文又来了

1998年以来，六部委发文有不少，包括：

* 振兴内需
* 产学研结合
* 环保风暴
* 棚户区改造
* 房地产调控

这一次的主题变成了《推动中长期资金入市》。需要六部委联合发文，就说明背后的利益很复杂，阻力非常大，纠葛特别多，但能把这几家攒一起让他们表态，也同样说明，最上面的意志非常强。所以我们看到很多六部委级发文，一开始都踉踉跄跄，但最后基本都办成了。

## 二、分手见人品

拜登在离开白宫前，给懂王留了一封密信，原意是不要公开，但懂王啥性子，直接透露给媒体。

## 拜登在信中说：

“我祝愿你和你的家人在未来四年里一切顺利。美国人民以及世界各地的人们，都期待白宫能在不可避免的历史风暴中保持稳定，我祈祷未来几年里，我们的国家将迎来繁荣、和平和恩典。愿上帝保佑你并指引你，就像他自我们建国以来一直保佑和指引我们所爱的国家一样。”

瞧瞧，这就是格局，临走了，没有任何指责和怨言，祝福下一任，希望他能把事做好。虽然懂王多次攻击拜登老迈、昏聩、痴呆，然而，登子从没在人品上否定指责过懂王，在任时，也拒绝了民主党的要求，用自己的政治威望去打击报复川普。登子要真那么做了，懂王早挂了。

而你看懂王怎么说拜登呢，就在前两天的就职演讲里说，懂王批评登子那票人，把国家搞的四分五裂，一团乱遭，邪恶暴力，激进腐败，支柱断裂，完全失修。上届班子还在现场呢，这几乎是全盘否定，指着鼻子骂。

的确，懂王在上次败选后是遭遇了不公，但是现在，你都上台了，大比分赢了，无限荣耀光环加于一身，扎克伯格等富豪早就降了，就必须展现出大度和胸怀，团结一切可以团结的力量，这就是政治家的气量。

古往今来，这样的案例多了，晋曹操烧掉了手下给袁绍的书信，戈尔当年败给小布什，两个人也没有闹僵。

其实，懂王完全可以和拜登搞好关系，甚至聘他做一个顾问。登子毕竟代表了老白男，美帝精英，各种建制派。如果大家看到你连敌人都能包容，那么势必给出更多的支持和心服口服。那么，懂王就能做到真正的一统江湖和号令天下。

然而，懂王赢的很爽，上任前后多少有点泄愤和自我表演，格局就差了很多。他也只能吸引来见风使舵和唯利是图的投机客，难以团结真正的美国中坚和精英，比如耶伦、鲍威尔等人。那些高管才是改革的核心力量。

所以各位如果你是注定要做大事情的人，就要学会扩充自己的胸怀，驾驭自己的情绪，能够给他人留一线，善于和别人握手言和，善于团结一切能够团结的力量。江湖风雨，谁还没个委屈不成？关键在于，在面对这些事情的时候，大家看出来你真正的人品和内心力量。

登子为何临走前大赦2000多人，他已经摸透了懂王的心眼啊，他不觉得川普有那个气魄去化敌为友。最了解你的，往往是你的对手。

齐景公问政于孔子，怎么当领导，怎么管好队伍。孔子说，君君，臣臣，父父，子子。臣要有臣的样，君要有君的样。各自做好自己本应该做的事情。当了领导，你就得有格局胸怀，这和你是什么性格，没有关系。
