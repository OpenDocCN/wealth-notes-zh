
# 懂王不是下场中东，而是退场中东

等了一星期了，终于在今天，懂王空袭硬解伊核。特朗普搭乘空军一号前往新泽西州贝德敏斯特高尔夫球俱乐部，下机受访时已语带转折，称“两周只是最长期限”，实际可能更早。更多文章加薇：jntsg2

特朗普与副总统万斯、国务卿鲁比奥、CIA局长拉特克利夫、美军参谋长联席会主席凯恩进行了密集沟通。特朗普还在高尔夫球俱乐部，但白宫工作人员已获知将有重大行动。特朗普乘机前往白宫，下达了空袭指令。晚上，特朗普及多名内阁成员进入白宫战情室观看空袭直播。美军派遣6架B-2隐形轰炸机，由密苏里基地起飞，历经37小时连续飞行与空中加油后，抵达伊朗空域，共投下12枚GBU-57巨型钻地弹，轰炸福尔多深层核设施。另有1架B-2轰炸机也向纳坦兹核设施投了2枚钻地弹。美军核潜艇自波斯湾400英里外发射30枚战斧巡航导弹，精准命中纳坦兹与伊斯法罕核设施。但是，越来越多的细节证明，懂王不是在下场中东，而是在退场中东。

看起来，懂王用了B2，用了钻地弹，一切都按照我们的最美剧本走，一切也都按着内塔的挟裹走。但是，以下几点不能解释。第一，伊核没有核泄漏。沙特阿拉伯核与辐射监管委员会22日在社交媒体上说，美军打击伊朗核设施后，在沙特及阿拉伯海湾国家的环境中未检测到任何放射性影响。伊核如果被硬解，大概率是要核泄漏的。特别是福尔多设施，那里有武器级浓缩铀，地下工厂如果被炸穿，核泄漏无可避免。国际原子能机构2023年在这里检测到丰度高达83.7%的铀颗粒，距离武器级浓缩铀仅一步之遥。

这个级别的浓缩铀如果被轰炸，不会引发蘑菇云式的核爆炸，但它绝对会导致灾难性的放射性物质泄漏（核泄漏），其危害程度不亚于甚至可能超过某些核事故。但是至今，都没有相关机构观测到伊朗出现核泄漏。甚至，伊核设施附近的道路都重新开通了。伊朗库姆新闻：德黑兰-库姆高速公路已通车，根据市民观察，德黑兰-库姆高速公路目前已通车，车辆通行顺畅。该高速公路距离福尔多核中心仅几公里，对过往车辆不构成危险。伊朗如果发生了核泄漏，那么绝对不会开通这个高速公路。

能开通这个道路，就说明伊朗自己已经确定，没有核泄漏。第二，懂王是扔了钻地弹，但没炸穿了。这个巨型钻地弹能炸穿61米厚的钢精混泥土。但福尔多设施在地下90多米，而且在山体内有更坚固的岩层保。这里的花岗岩硬度甚至比超强混凝土还要硬。而GBU-57对中等硬度的岩石钻透深度为40米，对更坚硬的岩石钻透深度肯定更浅。根据科学计算，后一枚钻地弹就算打进前一枚钻地弹炸开的洞中，继续增加的钻透深度只有前一枚的25%左右。

五角大楼评估，即便用两枚GBU-57连续命中同一目标，穿透深度也只能达到80米岩层左右，仍不足以摧毁福尔多的核心区域。美军如果用GBU-57进行攻击，2枚或许才能钻透50米左右的花岗岩岩层。如果要钻透100米左右的深度，至少需要7枚。然而，2枚钻地弹落在一个坑中都非常难，更别提7枚“鱼贯”向下钻。毕竟这款巨型钻地弹采用的是GPS制导方式，精度能达到3米以内就算表现良好，实现7枚钻地弹落在同一个坑里的精度是难以实现的。

果然，从卫星地图显示，一共炸了三个坑，而不是只炸出一个坑。这是两组卫星图，图一是 Planet Labs PBC拍到的，图二是 美联社发的，来自Maxar，都发现是2x3个弹坑。另外这三个间隔很近的弹坑显示：GBU-57的精度根本不足以让第二枚GBU-57沿着第一枚形成的坑道继续深入打击90米岩层以下的核设施，福尔多核设施的主体结构很可能基本完好，所以多国都未检测出异常核辐射。美国耗费巨资，还玩声东击西的计策然后打了个寂寞。

更无语的是，懂王还炸了地下设施的出入口。美国只有20多发这种钻地弹，用一发少一发，重新制造要耗费时间，这么珍贵的钻地弹，不去打地下浓缩铀，竟然对准地面出入口来了一发，真是暴殄天物。也可以看出懂王的真实用意——

“不能再对准窟窿炸了，万一真的炸穿了怎么办？什么？你说带的钻地弹有点多演戏要演到位？那好吧，你随便找个新的地方扔了就算了，反正别真的炸穿就是。”

第三个细节，就是懂王在没确认伊核是否彻底硬解之前，就宣布大赢特赢。懂王还没确认伊核是否硬解，就说这是全部军事行动，只要伊朗不反击美军就走人了。

各方面消息源也表明，懂王不想再继续战争行为——知情官员指出，川普目前无意进一步对伊朗发动新一轮打击，而是希望这次果断出手能促使德黑兰重返谈判桌，结束战争。CNN全球事务分析师麦格克指出，他与一位美国高层官员交谈后得知，白宫对这波空袭的效果感到满意，并明确向中东各国与伊朗表示，“这就是全部，这次行动只针对核计划，不会扩大成战争。”美国哥伦比亚广播公司新闻部21日援引消息人士的话报道说，美国当天通过外交渠道向伊朗表示，打击伊朗核设施是“美方全部计划”，美方不打算推动伊朗政权更迭。

伊核到底有没有被硬解，都还不清楚。懂王就急忙忙的宣布停战退场，甚至还要重新和伊朗谈判，所以其中的猫腻你懂的。

第三个细节，就是懂王空袭伊朗之前，甚至告知了伊朗——据美国有线电视新闻网 (CNN) 报道，两位知情人士透露，美国通过秘密渠道讨论向伊朗传达了如下的信息：美国总统特朗普下令对伊朗核设施发动空袭，这次空袭行动（范围）将得到控制，并且未来不会再计划进行空袭。然后伊朗就说，我在懂王轰炸之前，就偷偷转移了浓缩铀，说懂王炸的是一个空壳子。甚至说地下都没炸穿，只炸了地面的。

伊朗继续声明没有核泄漏，并开通了附近的高速公路。伊朗目前只是嘴炮说要择机反击，说中东美军基地是目标，威胁说要退出核不扩散条约，乃至封锁霍尔木兹海峡。但是，伊朗目前都只是高官嘴炮，哈梅内伊还没有出来下令如何反击。大概率，哈梅轰炸下中东美军基地，就应付交差了。反正之前也轰炸过，美军当年被炸也没下场。反倒是其他中东武装，一个个出来宣布反击美国。比如胡赛说要重新封锁。比如伊拉克民兵组织说要攻击美军中东基地。这些小弟们倒是真刀真枪的干了。就等哈梅表态反击了。

如果真的被炸出核泄漏了，如果真的伊核被硬解了，哈梅强力对准美军反击还好。但如果懂王只是装逼而已，那哈梅就不会真的封锁海峡真的扔以色列脏弹。

毕竟，懂王甚至都提前告知伊朗袭击计划。结合种种细节，我认为，这次空袭，并不是我们期待懂的懂王下场中东的最美剧本。恰恰相反，这是懂王被内塔架在火上烤后，没人给他下台阶后，懂王自己给自己找的下台阶。懂王给自己退场中东，找的下台阶。根本不是真的硬解伊核，只是空袭装装样子给自己退场找下台阶，然后宣布自己大赢特赢。简称，懂王装完逼就想跑。

哈梅知道这是懂王退场中东的下台阶，哈梅就不会过分报复，逼懂王再度回到中东。伊朗外长已经准备去俄罗斯见普京了。伊朗肯定会找普京劝和，让普京当中间人再度和美国回到谈判桌，缓解中东局势。而可想而知，内塔现在肯定是知道被懂王忽悠的。内塔很生气，后果很严重。

特别是内塔在第一时间还被蒙在鼓里，说谢谢懂王懂王伟大。后续内塔还会继续作妖的想办法挟裹懂王。同时，中国也不会让懂王就这样轻松的退场中东。胡赛第一时间宣布重新开打，背后难说没有中国的意思。现在可能只有伊朗一家，是乐见懂王装完逼就跑，退场中东的。内塔和中国，都不乐见懂王装完逼就跑的样子。后续就看内塔和中国如何出招，将准备退场的懂王，再度挟裹到中东泥潭了。特别是中国。

懂王宁愿对全球撒一个弥天大谎，也要退场中东，还不是顾虑到中国，顾虑到台海局势。各方好不容易把懂王忽悠到中东的悬崖边上，现在懂王见势不妙装个逼就想跑，没那么容易呢。
