
# 《李嘉诚也爱国？》

源自 3 月 14 日群聊
发布于 2025 年 3 月 14 日

除了马斯克这种另类，
台面上的大部分的知名首富，
本质上都是资产的代持人，
他们得以手握天量的财富，
是因为需要服务一个政治山头或者政治团体。
王健林和马云如此，
徐明和丁书苗如此，
给大英帝国老爷们搭理的李嘉诚也是如此，
他们很多买卖资产的动作，
尤其是涉及海外资产，以及疯狂的搞慈善
和捐赠，资助女明星，
基本都不是自己需要，
而是在替自己的幕后出资人来掏钱和代持。
这些大佬们，本质上是不存在所谓爱不爱国的，
因为他们的后台，本就是国家的核心组成部分，
他们一定是最爱国的一批人，
可以说是与国运休戚与共。

但问题是，
不同于中国的大企业家的后台是国内家族，
李嘉诚的后台，是一群英伦老钱家族，
李超人真的是爱国的，但是他爱的是大英帝国。
