
# 怕你睡不好觉，国家又出手了

## 中国睡眠健康调查报告

2025年3月，中国睡眠研究会发布了第一份面向全体国内居民的睡眠报告。这份报告显示，虽然2024年我国的总体睡眠情况有所改善，但睡眠困扰率依然高达48.5%。

### 睡眠现状

- **睡眠指数**：2024年，我国居民睡眠指数为68.74，是2021年以来的最高值，但睡眠状况本身并不达标。
- **睡眠困扰率**：18岁以上人群的睡眠困扰率高达48.5%，主要问题包括入睡困难、早醒、夜间易醒等。
- **性别差异**：女性群体的睡眠困扰比例（51.1%）高于男性（45.9%），尤其是成为母亲的女性。
- **年龄差异**：65岁及以上人群的睡眠困扰率最高，达到73.7%。
- **职业差异**：家庭月收入在1万到1.5万元的新就业形态人员（如网约车司机、快递员等）失眠问题最严重。
- **收入差异**：家庭月收入在4.5万元以上的传统正规就业者平均睡眠时间最晚，中等收入群体每晚平均睡眠时长为6.85小时，超过45%的被调查者凌晨零点以后才入睡。

### 睡眠习惯

- **“00后”群体**：睡眠时长最长，但入睡时间普遍在零点以后，甚至有12%的“00后”是凌晨2点以后才入睡。
- **个体户和自由职业者**：睡眠相对较好，因为他们可以自主调整工作和休息时间。

### 影响睡眠的因素

- **吸烟**：我国15岁及以上人群的吸烟率是24.1%，吸烟会降低睡眠质量，戒烟者有睡眠困扰的比例高达65.4%。
- **电子设备**：91%的人群习惯在卧室使用手机，睡前使用电子产品的人，睡眠困扰率高达51.5%。
- **提神饮料**：睡前饮用咖啡或浓茶的人，睡眠困扰率达到64.8%；晚餐过饱的人睡眠困扰率是57.7%。
- **饮酒**：虽然酒精有镇静效果，但会打断深度睡眠，反而降低睡眠质量。

### 如何改善睡眠

1. **建立标准睡眠**：健康睡眠是指入睡时间在30分钟以内，夜间醒来的次数不超过3次，且醒来后能在20分钟内再次入睡，同时睡醒以后感到精神饱满。
2. **改变生活习惯**：排除干扰因素，如吸烟、睡前使用电子产品、饮酒等，并控制午睡时间在25分钟以内。
3. **使用安眠药物**：如果睡眠问题严重，可以考虑去正规医院开具安眠药物。目前市面上的安眠药可以分为三类：苯二氮䓬类药物、“Z药”、褪黑激素受体激动剂和食欲素受体拮抗剂。

最后，睡觉是我们日常行为中，最没有替代方案，最需要认真对待的事情。祝从今天开始，每天都能睡个好觉。
