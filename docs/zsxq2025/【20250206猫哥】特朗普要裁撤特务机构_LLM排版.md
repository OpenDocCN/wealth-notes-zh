
# 特朗普要裁撤特务机构

嘉宾：梦回安西

在裁了“美国国际开发署”后，懂王做出了更为惊人的举动--对FBI和CIA 进行降本增效，大量的特工在自己不知道的情况下，就被解雇了。

大家原先设想的是，懂王和马斯克最多裁撤驿站，没想到竟然要裁特务机构，只能说，现实远比小说魔幻，我们的想象力已经跟不上了。

明面上，懂王这样做是为了省钱，但谁都知道，他和特务机构的恩怨很大。

FBI在第一个任期内，就是“通俄门”的调查机构，这事让懂王耿耿于怀，新上任的FBI局长帕特尔就明说了，要把参与“通俄门”调查的人员统统揪出来，该解雇的解雇，该起诉的起诉。

除了“通俄门”，还参与了对“国会山事件”的调查、闯入海湖庄园事件，桩桩件件下来，懂王对早就恨得咬牙切齿了，不往死里整才怪。

自己不整FBI，说不定卸任后又要被FBI整，或许都不用等到卸任，在任时FBI都敢，第一个任期就发生了类似的事情。2017年，就在荒郊野外包了个别墅，悄悄作为调查懂王的办公场所。

有这么多前车之鉴，懂王与其赌一把FBI的忠诚度，不如先下手为强。

有权不用，过期作废。

CIA倒是没明着跟懂王作对，主要是CIA的职能偏向对付外部，和美国内部的关联，相对没有那么强，只是双方各种摩擦还是少不了。

比如，从第一任期起，懂王就希望将“南部边境”问题，列入国家安全问题的范畴，进而名正言顺地动用情报、军队等组织，进行打击。

这时候就要发挥作用了，它是美国最大的情报机构，在国家安全问题上有很重的份量。如果CIA不愿意，那么就显得名不正言不顺。

CIA是怎么回应懂王的呢？三个字：软对抗。

CIA每年撰写的国家安全报告中，都不把“南部边境”问题列入其中，哪怕懂王强烈要求，CIA也只是轻描淡写，一笔带过，让懂王很不满意。

除了软对抗，许多红脖子都怀疑，与懂王的枪击案存在联系，尽管没有证据，但以懂王的为人，会因为没有证据就不收拾你？

虽无显迹，意有之，不服么？

考虑到只是软对抗，加上份量比FBI重，因此给CIA的裁员待遇要好得多，FBI的许多人都是突然被裁，事先完全不知道。

而CIA是有事先通知的，还有签字同意的流程。

FBI的人普遍拿不到什么补偿，能拿走这个月工资就不错了(美国政府裁员不存在N+1)，但CIA能获得“买断工龄”，最高拿八个月薪水补偿。

当然，“最高八个月”必须是在2月6日前签字的人才能领到，过了2月6日就没那么多了，越往后越少。

尽管大部分人拿不到最高，但拿几个月的问题应该不大。虽说对于老员工而言，几个月乃至八个月，肯定是吃亏的，也总好过没有。

对FBI和CIA进行降本增效的计划传出后，立刻让美国政坛炸了。

毕竟特务机构可不是一般机构，是手握大量黑料、关键情报的，万一特工们被裁撤后，再就业困难，经受不住敌对国家的金钱诱惑，偷偷卖情报咋办？

特工们还手握很多犯罪的东西，比如毒品，万一被裁后直接转投黑帮怎么办？不过相比于卖情报、投黑帮，真正的大问题是两点。

一是美国颜色革命的输出能力，会被重创。

从来就是美国对外输出动乱的主力机构，它和“国际开发署”构成了一对组合，负责水面下的部分，暗杀、关键情报刺探、军事训练等等。

比如“猪湾行动”，就是CIA组织的。

“国际开发署”则负责水面上的部分，包括资助NGO、鼓动群体对立等等。

两者互相配合。

现在“国际开发署”瘫痪，被大量“买断工龄”，后续美国的颜革输出能力，还能有保证？别说钱不钱的了，连执行机构都严重削弱了。

其实这些年，颜革的效果在下降，比如塞尔维亚，搞了好几波都没成，只是把总理给拉了下来，距离搞掉总统武契奇还有很大距离。

主要原因是，各国被颜革得多了，也学会应对了，只要把颜革掐灭在苗头阶段，就多半起不来，而掐灭的关键，是敢于下狠手。

以往颜革能成，靠的就是制造出巨大的声势，似乎所有人都不支持颜革似的，导致政权方不敢下手，畏畏缩缩，但随着时间的推移，不少国家意识到，这只是西方媒体配合互联网制造出的幻象。

他们的实际支持者，远没有看起来那么多。只要你敢来硬的，往往能试出成色，前段时间格鲁吉亚就是这么干的，还真就顶住了。

本来颜革的成功率已经不太行，搞了格鲁吉亚，都有点搞不动了，懂王还要来这么一手，那么“颜色革命”事业就是基本废了。

此举等于把民主党的饭碗都砸了，一点情面不留，从侧面说明其重回“孤立主义”的决心，对美国之外的事情，是真的不想管了。

对我们来说，是大好事一件。

只不过，这并不意味着中文互联网会清净太多，因为在中文互联网上带节奏的，除了美国，还有日本和台湾，后面两个投的银子，不比CIA少。

可以毫不夸张地说，精日是各个群体里最死硬的，比精美都要死硬不少。

说回懂王，裁撤“厂卫”这么大的事，势必引起敌对者的猛烈反扑是一定的，民主党那边早就炸了，所谓的西方主流媒体，这几天一直在大骂懂王。

因为裁撤“厂卫”，不仅涉及美国的利益，也涉及到许多盟友的利益，比如BBC，BBC的最大“捐款者”是CIA，占BBC年度经费的25%左右。

一旦被削弱，如此多的利益攸关方，得到的银子也必然会减少。

他们能善罢甘休？不可能的，暴风雨终将来临。
