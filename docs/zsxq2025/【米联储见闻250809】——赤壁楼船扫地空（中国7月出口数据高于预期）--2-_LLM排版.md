
# 米联储见闻250809——赤壁楼船扫地空（中国7月出口数据高于预期）

7月出口数据出来了，中国出口激增比预期还要高，外媒和机构都心理不爽了，尤其是关注到对东盟出口的大幅增加，让川普超过40%的转口税变得越来越迫切。7月，中国整体出口同比增长了7.2%，进口增长4.1%。对东南亚和非洲的出口增速是整体出口的两倍多，大家都知道这些地区是转运至美国的重要中转地，此外，中国对欧盟的出口也大幅增长，面向最大出口目的地东盟的出口同比增长17%，越南同比增长28%，泰国同比增长26%，均大幅增加。欧盟同比增长9%，日本同比增长2%。由于美国买家对支付川普额外加征的30%关税持谨慎态度，中国7月对美直接出口下降了超过五分之一，7月对美出口同比减少22%。减少幅度比6月(16%)扩大。7月对美出口金额为358亿美元。从降幅来看，4月同比减少21%，5月同比减少35%，而6月曾缩小到16%，7月是连续减少的第四个月。对美物流需求不断减少。

浙江省宁波舟山港对美西海岸出口集装箱价格指数(截至8月1日)与美国大幅降低关税后的5月30日相比，下降了近7成。对美东海岸的价格也下降了近6成。需求下滑对价格造成了下行压力。去年中国贸易顺差接近1万亿美元，其中制成品顺差相当于经济总量的十分之一。打击中国的进出口贸易是美国遏制中国制造和中国综合国力的关键手段，所以每个月看到中国的贸易数据在增长，不管是外媒还是机构，心里都是极其不痛快的。尤其是在美国对中国进口商品加征30%的情况下，中国的进出口贸易还在大幅增长，这意味着美国的遏制手段没有起到真正的作用。

过去十几年，中国对美出口额与自美进口额的比例曾经高达4:1，而6月录得美国对华贸易逆差21年来最低点，仅95亿美元，美国6月整体贸易逆差也收窄至602亿美元，为近两年来最低水平。但中国7月的数据显示，其对美出口仍为自美进口的三倍。7月中国对欧盟27国集团的出口增长了12%。这是比较积极的变化。毕竟欧盟仍是替代美国的最佳选择。相比于东盟作为转美贸易的中转点，始终将面临美国高关税惩罚的潜在威胁，而中欧贸易的规模相对于这些贸易质量更高，利润更高，且相对更稳定。

目前中国还面临一个大问题，限制转口贸易的程度和执行力度，目前已开始对转口商品加征40%，但是关于认定规则还未公布，一件来自泰国的产品是适用于19%的关税，还是40%的关税，美国究竟将如何判定才是关键。这也是眼下东南亚国家和中国国内生产商最关心的问题，因为只有划定了规则，大家才能思考如何寻找漏洞，而川总一天不发布规则，进出口商提着的心就没法放下来。

今年上半年中国对东南亚出口同比增长12.9%，7月增速进一步加快，达到16.6%。自川普第一任期开始以来，这些出口已增长近两倍。因为担心中国商品无法真正出口到美国，印尼等东南亚国家过去两年已对非转口中国商品大幅提高了关税和其他税费。此外，7月中国对非出口同比激增42.4%，这个增长速度，让非洲国家也开始担心，比如跟中国关系不错的南非，被川普扣上了30%的关税帽子，最近也开始表达担忧，称其制造业正被中国竞争压制。

东南亚方面为了让美国刀下留情，不得不说出心里的实话，表示，如果美国设定一个非常高的标准，东南亚国家将无法达标，因为这些国家还没有建立起上游产业，也就是说从中国进口原材料和零配件几乎是没有办法的事情，东南亚需要时间来提升本地产业链从而确保产品达到美国去中国化的要求，实际上，从川普1.0时代开始，这种去中国化的操作一直在执行，但恰恰是美国自身政府的态度左右摇摆，导致企业在成本面前难以下决心，最终导致去中国化一直在提，但大家依然依赖于从中
