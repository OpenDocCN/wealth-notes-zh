
# 为什么有钱人都很抠门？

原创： 君 公众号： 君笔记 25年10月13日

有钱人的抠门，跟穷人的省吃俭用，根本就不是一个概念，背后也不是一个逻辑。比如，有钱人开着价值百万的豪车，但是为了省5元/小时的停车费，宁愿把车停远一点，然后多走200米的路。比如，有钱人给别人发几十元的红包，连眼睛都不眨一下，但是几乎从来不去外面喝几十元一杯的咖啡。比如，去高档场所一晚上就消费几万元，但是你如果开口向他借一万元，他会毫不犹豫地拒绝你，即使你们是亲戚关系……那么问题来了，有钱人为什么会有这种脑回路?我总结，一共有5个原因，这些既是他们成功的原因此，也是他们成功的结果。如果你是个花钱无节制、认为抠门很可耻的穷人，我建议你好好吸收一下本文的精髓，你的财富值肯定会直线上升。

## 1.保持对金钱的敏感度

有钱人不怕花钱，但是要确保花出去的每一分钱都能带来超值的回报(哪怕是1%的回报)。所以，他们对每一分钱的去向都会保持着高度的警觉和控制欲。他们潜意识里知道，一旦丧失了这种敏感度，那自己离破产就不远了。举个例子，我自己去其他公司拜访、去游泳馆、去 spa 会所时，经常会加前台的微信，只要她们接待过我，我都会发一个58或者66元的红包。表面是感谢她们为我提供服务，实际上我是想从她们嘴里获取一些值钱的商业信息。比如，她们公司的客流量如何?有没有推出什么新的产品或者服务?俗话说，知己知彼百战百胜，如果我在各行各业都有“眼线”,那么我看待各行各业都是透明的。这时候我做出的每一个决策，成功率也会大大提升。但是，你让我花60元去喝一杯咖啡、买一份甜点，我肯定不会去的，我认为这份钱花得太冤枉。

## 2.穷人总是因小失大

在乱投资和诈骗案件中亏掉几万元、几十万元的，基本都是穷人和中产群体。他们之所以上当受骗，就是因为他们对金钱不敏感，没有养成“管理金钱”“计算投资回报率”的好习惯。要知道，你越是在小事上保持敏感，在大事上就越不会出昏招，因为自己会形成肌肉记忆。举个例子，月薪5000元的普通工薪阶层，基本都是花钱无度的蠢货。比如，周末花30元去喝咖啡，花200元去跟狐朋狗友聚餐，花500元去买一件华而不实的衣服，花1000元去参加注定没有“回头钱”的同事的婚礼……这样做的恶果有两个：①自己手上永远也攒不下来钱。手上没有闲钱，就永远无法培养自己投资的能力，那么就永远不会获得被动收入，自己只能年复一年做牛马。②永远都不明白金钱的真正价值。富人都知道，金钱的真正价值不在于花出去的那一刻能换回什么商品或者服务，而在于它创造的未来。你每节省下来的10元钱，将来都有可能变成50元、100元，甚至更多。一般来说，只要花钱时不考虑“回报率”的人，他永远都无法形成“投资思维”,结果就是，一发工资就想着买这买那，完全没有投资意识。

## 3.穷人买奢侈品，富人买奢侈品公司

一件T恤的价格如果超过1000元，一个包的价格如果超过1万元，就算是奢侈品的行列了。而购买奢侈品的大多数群体都是打工族和中产，他们以为这些奢侈品能彰显自己的身份，让自己跨入富人的行列。实际上，真正的有钱人很少买这些东西，因为他们非常清楚，这些东西只能满足自己的虚荣心，无法带来实际的回报。但是，他们会购买这些公司的股票(比如 LVHM 公司),实现某种意义上的躺赚。换句话说，蠢货通过“买奢侈品”试图让自己成为有钱人，而聪明人通过“卖奢侈品”来让自己成为真正的有钱人。举个例子，U 君自己就通过纳指持有了苹果和特斯拉的股票，它们的销售情况越好，我越躺赚。有人会问，我穿着奢侈品衣服、拿着奢侈品包包，不是可以混入上层社会、获取更多人脉资源吗?我的回答是：这些东西只能为你锦上添花，而不能为你雪中送炭。人脉资源的基础是：你有实际的利用价值。比如，大家在见面、聚会时，会从正面或者侧面打听对方的具体业务，营业额是多少。如果你只是一个普通上班族或者中产，那么到这个环节你就全露馅了，因为你嘴里吐出来的全是不上道的“外行话”。这时候，即使你穿的名牌都是真的，大家也会认为是水货。

## 4.富人更能接纳二手和租赁

有数据显示，在年收入50万元以上的人群中，开二手车的比例高达30%,而那些月薪几千块的年轻人，宁愿贷款也要买辆新车。这就是认知差距的根源，也是财富差距的根源。我自己就是购买二手货的践行者，我刚参加工作、收入还不到1万元的时候，我执意要买新车(开了3年就卖了)。现在我不喜欢开车了，如果我再买车的话，100%会买二手的豪车。一方面是因为，很多豪华车型都是没有现车的，你得加价、预约、排期，非常麻烦。另一方面是因为，豪车的贬值速度太快，或者说，买二手豪车能省去很多额外费用。举个最简单的例子，保时捷某款车型的新车裸车价是80万元，但是你想要上路的话，得花120万元(各种选配、各种税费、各种加价)。但是，如果你买那些开了两年、行驶两万公里二手车，价格可能只需要70万元。无形之中你就节省了50万元，这些钱你拿出来再投资，他不香吗?穷人之所以很难接受“二手货”，就是因为有道德洁癖，这也是限制他成为富人的核心原因。实际上，真正赚到过钱的人都明白，赚钱就是要把双手弄脏，不可能太体面，吃相都很难看。如果你总是想站着把钱赚了，那么你早晚会被竞争对手生吞活剥。除了二手货之外，富人对“租赁”的接纳程度也远高于穷人，或者说，两者的出发点是不一样的。

## 5.“抠门”会提高自己的鉴别能力

只要你每次在花钱时，都要计算这笔钱的投资回报率，那么你就会越来越聪明，鉴别能力也会越来越强。当你养成了这种习惯之后，你的鉴别能力就会大大提升，然后轻而易举地就能识别什么是机会，什么是陷阱。会花钱的人才会赚钱，就是这个道理。你看那些动不动就被骗走几十万元的人，他们大概率都不是会赚钱的人，也不是会花钱的人。他们很可能是没有什么实质性收入的家庭主妇，老公在外面赚钱、交给自己保管，然后自己把钱拱手相让给骗子。真正有赚钱能力的富一代，大概率一眼就能识别出谁是骗子。因为他们对“投资回报率”有着非常客观的认知。比如，“100万元本金”这个量级，如果能做到6%的年化收益率，就已经是非常了不起了。但是，没有下场赚过实质性大钱的人，根本就理解不了这句话的真正含义。有钱人不仅看项目很透彻，看人更透彻。回到文章开头的那个案例，为什么你的亲戚身价几千万、去高档场所一晚上就消费几万元，但是你如果开口向他借一万元，他会毫不犹豫地拒绝你?原因也很简单，他认为你没有前途，是个不值得投资的人，风险很大，这时候即使在你身上投资100元，他都觉得冤。
