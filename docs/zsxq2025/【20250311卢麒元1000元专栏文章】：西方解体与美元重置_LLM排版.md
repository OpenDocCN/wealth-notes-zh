
# 西方解体与美元重置

## 祛魅金融

金融在国家治理中占据核心地位，尤其在殖民时代末期，金融殖民成为主要形式。由于对金融的忽视，苏联解体、平成战败等悲剧发生。金融的重要性体现在其决定国家生死存亡的能力上，掌握金融如同掌握命脉。

金融作为一种工具，其效用远超其他工具。汉武帝时期，通过桑弘羊等人的金融手段，收回中央铸币权，控制了大汉的财政金融，实现了削藩。盐铁官营、中央税收一杆子到底，有效控制了藩王的经济权力。均输平准则严厉打击了投机行为，稳定了大汉的经济。

回顾我党历史，从井冈山斗争到解放战争，胜利的关键在于掌握了财政和金融主权。土地政策和金融控制是胜利的基石。古今中外历史表明，不懂财政和金融，谈治理二字将非常危险。

金融必须忠诚和忠贞，金融官员要极度忠诚，不能有个人、机构或部门利益。美国和西方出问题，根源在于金融主权旁置，负责金融的人缺乏忠诚和忠贞。

唐朝时期，李林甫的金融改革导致通货紧缩，土地兼并加剧，正规义务兵无法得到供养，变成了雇佣兵。大唐的衰落，源于金融改革缺乏忠诚和忠贞。金融是国家信用和执政党信用的体现，缺乏忠诚和忠贞，一切技巧和技术毫无意义。

金融极简，妈妈是天生的财政部长，她再穷也能让孩子吃饱，再富有也能节约。金融的逻辑应该让老百姓听懂，服务于党、人民和国家。

## 金融权力的核心

金融权力的核心包含征税权和铸币权，构成一个国家的核心主权。征税权是财政权，铸币权才是金融最核心的权利。国家主权最核心的经济主权就是征税权和铸币权。

川马新政关注的焦点是铸币权。马斯克和川普重视数字货币，因为他们手上有独立的数字货币。他们要查金库的账，黄金出现暴涨，目的是为了一个新的五铢钱。

美元的含金量面临挑战，美国、美国人围绕美元形成的负债总负债是230万亿美元。美元走进尾声，必须有一个解决或了断。发行新的货币会有问题，旧美元充值和重置是最佳选择。

充值的方法，如贸易战开打，格陵兰岛、加拿大、墨西哥等将为美元充值，但意义不大。庞大的经济体才能帮助美元充值，而目前只有中国具备这样的条件。

其他充值方式，如战争，可以导致美元资本回流美国，但不能解决根本性问题。美国人想再造全球的资产和商品交易模式，将实物交易模式全部上线，并数字化、加密化，变成Web3、通证化及稳定币。

美国错过了最佳时期，现在再想完成这样的部署已经没有机会。中美一体化可能创造奇迹，但美国的头部没有解决意识形态问题，他们仇中和反中深入骨髓。

## 中美博弈

中美博弈不是中国和美国两个国家的博弈，不是在一个系统内或一个生态内博弈。中美博弈是两个生态体系的博弈，即美国为首的西方生态体系与以中国为代表的正在生成的东方生态体系。

中国自2012年以来，特别是2014年央地税合并、2015年取消与美元联汇后，开启了长达十年的生态重构。中国正在用小多边外交、小多边政治经济往来、小多边关系，在重建与世界的关系。

美国和西方是一个完整的生态体系，他们在打压不发达国家和南方国家。中国与一些国家建立了良好的政治关系和经济关系，形成了围绕中国展开的东方生态，这个生态正在成型，并迅速发展和壮大。

两个生态体系正在形成竞争，西方生态体系要抛弃踢开脱钩断链的生态体系。中美博弈是两个生态体系的博弈，而不是两个国家在博弈。
