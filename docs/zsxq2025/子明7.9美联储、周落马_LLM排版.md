
# 子明7.9美联储、周落马

## 2020年的武汉市长周先落马
在03年的冲击中，由于使得北京被动，4月20日，农业部北京市委同时换帅，海军军医系统出身的张文康卸任卫生部部长，财贸世家出身的孟学农卸任北京市长，动作迟缓的鸽派都被一股脑的挪走，临危受命的W借此开启了他波澜壮阔的政治生涯。20年2月13日，湖北武汉同时换帅，湖南金融系统出身的蒋卸任湖北书记，上海宝钢系统出身的马国卸任武汉书记，两个让加微信号jntsg2，免费进资料群。

北京被动的“鸽派”主官同时被挪走，临危受命的Y、W以及C，也开启了仕途的一路高歌。2000年前后，湖北与金融系统进行了一波交流，其中有两个代表人物，湖北的周先wang去中国银行总行挂职了个部门副总，金融的蒋chaoliang去湖北挂职了一个副省长，后来周先wang成为了武汉市长，蒋chaoliang成为了湖北书记。21年，在令北京震怒的蒋chaoliang卸任湖北的一年后，“甩锅”的周先wang也黯然卸任了武汉市长。

而这种隐性关联又延展到了4年之后，25年2月21日，周先wang的老上级蒋良超落马，25年7月4日，周先wang全家被带走调查，几处住所也全部被查抄。

## 美联储影子主席
发布专栏：知几共话
一个注定走到尽头的主席，他的倔强已然无力。2025年7月4日，据央视新闻报道，当地时间周四，美国财政部长贝森特表示，特朗普政府将在今年秋季集中精力选出美联储主席鲍威尔的继任者，并称有很多优秀的候选人。此前，特朗普已多次抨击鲍威尔。值得注意的是，贝森特本人也是美联储主席一职的竞争者。由于鲍威尔始终不愿配合特朗普政府的降息政策，后者已多次公开抨击前者。本次提前为美联储物色候选人也是特朗普在表达不满。

那么，本次贝森特的发言隐含了哪些信息呢？提前选出掌舵人又将对美联储产生怎样的影响？贝森特说要在今年秋季选出美联储主席候选人，看起来懂王对鲍威尔的不满已经到顶点了。按理说，鲍威尔的任期到明年五月才结束，美国的惯例则是总统会一直等到现任主席任期的最后几个月才着手任命继任者，这次明显是提前了。

子明：特朗普选出一个影子联储主席。可以理解为在央行YG行长上面安了个GSQ书记。这是相当于给美联储搞个双主席啊，是要预期管理吗？但我不理解的是，如果人选出来了，新主席又没有实际权力，该怎么操作才能配合懂王去影响市场呢？

子明：你可以这么理解，特朗普安排了一个美联储工作领导小组组长……小组治国。就比如，鲍威尔说绝不降息，新主席说我一上任就要降200个点，市场的预期就被均衡了。

子明：联储的官员们会分得清到底是主席大，还是书记大。

子明：对于一个政治人物来说，如果他是一个明年注定要卸任的主席，连接班人都确定了，那么他的政治生命实际上已经提前走到尽头。可以类比于皇帝对于皇太子的猜忌，基本都来源于此。像汉武帝、康熙正是因为这个原因要废太子。都明确接班人是谁了，大家还不下注，那就是傻子了。再比如，贵州、重庆、广东，三个60后都走向了没落。相应的，长期投资这几位的资本，也随之吃瘪了。

但是美联储降不降息，真的会因为一个影子主席就改变很多吗？毕竟懂王只剩四年了，可他们这种人是要在华尔街混一辈子的。

我也有一点这方面的顾虑，懂王只是换了一个主席，其他有投票权的都是有百年根基的利益集团。新主席固然是特朗普推上去的，但特朗普并不是一个人，他的背后也是一个利益集团。

稍微延伸一下，随着美联储阻碍的消失，民主党可能要垮了。懂王现在除了少数族裔，都已经团结的七七八八了。随着鲍威尔继任者的遴选和最后揭晓，美联储将开启一段由“双主席”领导决策的新阶段。等这位“书记”进驻美联储，我们不妨期待一下，美国乃至全球的货币政策，究竟会发生什么。
