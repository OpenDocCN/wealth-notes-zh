
# 香港国际调解院：一场“静悄悄的革命”

在现有的国际规则框架下，解决国际争端主要有三个路径：即联合国安理会、WTO争端上诉机构和海牙国际刑事法庭。但是，现在上述三个路径已经名存实亡。比如联合国安理会，现在就像是一个大型撕逼现场，因为老美滥用一票否决权，导致很多重大问题除了吵架基本解决不了争端。比如WTO争端上新旅翁，因美国持续阻挠上诉机构法官遴选，自9年起就陷入瘫痪，至今已有30份裁决因无嚎诉而作废。再比如海牙国际刑事法庭，现在更多的是充当西方国家的伶子手，专门针对不听话的国家和政治人物下手，最近几年闹得比较大的就是对普京发出逮捕令和调查拘押杜特尔特，而对老美等西方国家的种种恶行却搞选择性失明。

除了以上问题外，走上述路径门槛也高得吓人，比如通过诉讼和仲裁打一个跨国官司，背后没有力量加持，少说打个3-5年时间，整的当事人欲哭无泪。于是对于很多吃了亏都没处说理得发展中国家，心头早就憋着火，恰好最近几年我们在科技、军事等各方面发展比较好，已经具备和老美扳手腕的实力，所以由兔子牵头另辟蹊径搞一个国际调解院就变得顺利成章。比起诉讼和仲裁，调解这个方式更凸显来自东方的智慧，特别是在友好、灵活、经济、便捷的环境下，更是得到了很多国家的认同，连美国的死忠小弟匈牙利都退出海牙国际刑事法庭跑来入伙。这事儿说白了，就是一群被西方法律霸权欺负狠了的国家跟着兔子集体掀桌子，自己搭个讲理的新场子，准备把西方主导的国际司法体系给捅个窟窿。

我们这回牵头搞国际调解院，绝对不是拍脑门的热血上头。早在7、8年前，就有有志之士提出了相关构想。并在2022年兔子与近20个发展中国家（包括印尼、巴基斯坦、柬埔寨等）签署《关于建立国际调解院的联合声明》，算是完成了0-1的重要一步。紧接着2023年2月，国际心备办公室在香港正式挂牌，标志着组织国际贷遠判及前期筹备工作正式拉开帷幕。比如推动伊沙这对千年冤家的握手言和。比如帮助巴勒斯坦114个派系签下《北京宜言》，结束内斗。正因为有了这些以和为贵的成绩，才让各国心里有了底，也才有了5月30日来自全球60个国家和20多个国际组织的代表齐聚一堂，共同签署《关于建立国际调解院的公约》的盛况。虽然现在参与国不到全球一般，但不要小看这个阵仗，光创始国（类似于原始股东）就涉及19个国家，不仅包括印尼、巴基斯坦、老挝、柬埔寨，还有顶着欧盟压力入伙的塞尔维亚，横跨欧亚非的刺头的白俄罗斯、苏丹、阿尔及利亚、吉布提等，他们都是被西方法律霸权欺负惨了的国家，早在3年前就立马响应，举双手赞成，跟着兔子写公约草案。其次还有属于B轮融资进来的41个国家。
