
# 《李嘉诚的图谋》

源自 3 月 27 日群聊
发布于 2025 年 3 月 27 日

李嘉诚二儿子钓鱼台与总理同框，
跟李嘉诚本人遭受打击，被迫出血，
两者并没有什么矛盾，
也不要把两者混为一谈，
该打击打击，该合作合作，
两边都会加速的推进。
而对于李嘉诚本人来说，
这是非常成功的一盘大棋，
通过巴拿马港口事件，
成功将自己从一个手套家族晋级世界顶级家族。

任何一个大家族的崛起，
都需要做到不完全依赖于一股政治力量，
琅琊诸葛氏分仕魏蜀吴，
颍川荀氏分仕袁曹司马，
家族的成功穿越了历史的周期，
三国结束进入晋朝都依然是最顶级的家族，
后来，清河崔氏、兰陵萧氏、赵郡李氏、琅琊王氏，
这些最顶级家族的标志
就是不同的分支能够分仕于不同的政权，
并不需要家族对一个政权无条件的效忠，
近年来我们最熟悉的，莫过于绍兴俞家。

李嘉诚早早的就对两个儿子做了分拆，
大儿子继承了自己的家业，服务老钱持有重资产，
二儿子拿到现金创业，跟新钱圈子混在一起，
巴拿马事件，成功的让大儿子和二儿子分道扬镳，
一个继续混五眼圈，收获西方太阳的友谊，
一个继续混大中华圈，则接住东方的橄榄枝，
兄弟之间保持相互的斗争，
避开了强制的政治站队，
让整个李氏家族作为百年家族能够传承下去，
这才是华人首富级的布局与大智慧。
