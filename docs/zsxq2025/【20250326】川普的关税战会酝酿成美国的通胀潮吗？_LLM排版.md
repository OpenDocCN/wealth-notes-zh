
# 川普的关税战会酝酿成美国的通胀潮吗？

川普政府现在广泛执行的关税战的目的就是逼迫制造业回流。钟国对美出口的关税，已经累计叠加到了39%。这意味着钟国对美出口已经没有利润可言，维持现有的出口价格完全不现实，必须大比例涨价。这样的涨价消费者无法接受，于是美国的进口商只能另寻货源，或者更干脆一点，直接在美国开厂。直接开厂，就是川普政府意图达到的目的。

再比如加拿大，看起来似乎人畜无害，川普政府同样对他发起了关税战，关税一加就加到了25%。这场关税战的指向性非常明确——加拿大的汽车和化肥产业。这些汽车厂和化肥厂全都是从美国国内的五大湖区搬到了加拿大的安大略省，搬迁路径非常短，也就是从五大湖的南岸搬到了北岸。安大略省因此成为加拿大经济第一强省，其GDP占全国的比值，长期稳定在37%。安大略省的人口占加拿大的比值也长期维持在40%左右。

2024年，加拿大对美国的汽车出口规模为150万辆，占美国本土新车销量的9%。加拿大对美国的钾肥出口总量为1050万吨，占美国钾肥需求总量的85%。此外，美国的糖果生产和影视后期制作在近10年内也大规模转移到了加拿大。美国国内的饼干和巧克力生产基地已经全都搬到了加拿大。影视后期制作这一块，好莱坞大部分的影视公司比如索尼、环球等等等的后期制作全都放到了加拿大，每年可以创造120多亿美元的GDP。

川普政府现在还没来得及对这两个产业发起关税战，估计下一步就要针对这两个产业开打了。也就是说，加拿大的核心产业，基本上就是从美国搬到加拿大，然后再从加拿大向美国出口。2024年，加拿大对美国出口占其出口总量的75%，并因此创造了760亿美元的贸易顺差。加拿大经济可以直接总结为趴在美国身上吸血的水蛭，吸不到美国的血，加拿大经济当即就会崩溃，没有任何生还的可能性。就美加之间这样畸形的经济关系，加拿大已经完全变成了美国经济的附属品。所以川普给出总结，你加拿大在经济上没有任何独立性，不如直接并入美国好了。

现在这一刻，川普政府的目标非常清晰，以关税政策逼迫产业回流，从美国搬到加拿大和墨西哥的制造产业必须要回流美国。你不回流，我就给你加关税，增加你的成本，让你无利可图，让你只能选择回流。当然了，对企业来说，他也可以赌一把熬4年，熬到川普卸任，下一任如果是倒行逆施的民主党上台，那就又能够继续在加拿大过好日子，不需要回美国。到这里就轮到博弈论登场了，你不回流，总有其他生产商选择回流，他们将会占据领先优势，将你这个不肯回流的生产商赶出主流市场，并且伴随着川普上任之后的一通乱拳，马斯克领导的doge将民主党政府的大规模贪腐，充分暴露了出来，民主党现在的支持率，低到了丢人现眼的程度，从年初的40%出头，跌到了30%以下，创有记录以来的最低，也就是说，博弈的结果就是。

只要你现在不肯回流，你未来丢失美国市场的概率高达100%，没有任何企业能够承受这样的风险。所以现在所有在加拿大、墨西哥的汽车厂和化肥厂都在积极筹备回流，或者在美国国内选址建新厂，或者直接恢复此前被废弃的旧厂。美国经济居然因此呈现出一片兴旺景象，大量的新厂筹备开业，并因此有效阻止了所有左派经济学家臆想中的所谓衰退潮和失业潮。按照各位左派经济学家的狂想，川普上任之后这两个月的一通乱拳，应该会迅速造成衰退，并因此导致猛烈上升的失业潮。然而，截止3月中旬，美国初次申请失业救济的人数始终维持在22万人的历史低位，持续申请失业救济的人数也始终保持在190万以下的历史低位，丝毫没有萧条的迹象。更关键的是，美国持续了整整4年的通胀现象也终于出现了缓解迹象，2月份美国同比涨幅2.8%，CPI较1月份的3.0%继续回落，并且最为关键的食品和能源和交通价格已经出现了清晰的环比下降迹象。

3月份，伴随着川普新政的进一步落实，美国的食品、能源和交通价格必然会进一步下降，从而进一步遏制通胀。左派经济学家关于川普发起的关税战必然加剧通胀的预期，基本上完全落空。为什么川普对外加关税并不会引发通胀？大家一定要清晰地意识到，美国是全世界最大的农业国，它有着全世界效率最高的粮食和畜牧生产基地，也是全世界最大的农产品出口国，没有之一。2024年，美国光大豆的出口量就高达4967万吨，本来美国根本不可能有农产品的短缺问题和涨价问题。然而，拜登政府对农业提出了极其严苛的极端环保要求，比如联邦官员会去检测大豆种植的碳排放量，并要求所有的农场主缴纳碳排放费。

此外，联邦官员还要求所有化肥的吸收必须达到99%，施肥的时候不能在农地上大面积抛洒，必须直接施在农作物根部，同时必须将施有化肥的土壤与其他土壤进行物理隔离，绝对保证化肥残余不渗入大地。在运输的时候，拜登政府要求所有农场主，必须使用清洁能源进行运输，也就是只能使用电车运输。这些扯淡的要求极大地提高了美国农场主的生产成本。根据美国农业部的数据，以伊利诺伊州为例，玉米种植的非土地成本，也就是日常的化肥、机械、人工和运输成本，2019年每英亩为587美元，到2024年增加到772美元，增幅高达31.5%。大豆种植的非土地成本2019年363美元，2024年达到512美元，增幅41.0%。这里的成本增长基本上都是拜登政府的倒行逆施导致的，并因此极大地推升了美国的通胀率。

现在川普政府上任，第一时间就退出了巴黎气候协定，并宣布废除拜登政府对农业生产强加的所有苛政，就意味着，即使不考虑川普新政带来的其他刺激作用，仅仅只是废除极端环保政策，美国农场主的非土地成本就可以大幅下降三成左右。同时，川普政府全面废除了拜登对于国内能源开采行业的严苛禁令，全面开放页岩油的开采与运输，一方面这本身就会拉低美国国内的能源价格，另一方面，这也会大幅降低农场主的运输成本，从而进一步拉低农产品价格并进而拉低通胀率。基于这样的因素考虑，美国对加拿大的化肥产业增加关税，短时间内可能确实会造成钾肥的价格上涨，不过，钾肥在农产品生产中的成本占比只有1%左右，影响力本身非常低，而废除极端环保政策对美国农场主的生产成本下降是立竿见影的，所以美国的食品价格一定会继续下降，当然，能源价格也一定会继续下降。
