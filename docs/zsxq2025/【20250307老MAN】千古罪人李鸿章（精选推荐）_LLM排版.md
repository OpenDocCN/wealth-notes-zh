
# 千古罪人李鸿章

李鸿章拥有过改变中华民族命运的能力。在很长一段时间里，只要李鸿章愿意，他能够轻易的扭转满清帝国这艘即将沉没的巨轮的航向。然而李鸿章最终什么都没做。在历史的十字路口，李鸿章只是眼睁睁的看着汉民族从一个黑暗时代沉入了更深的黑暗时代。从这个角度上说，李鸿章算是汉民族的罪人。

满清帝国趁着明王朝内乱的机会抢到了中原地区的地盘，然后通过艰难的苦战击败了南明政权和陆续作乱的藩王。一直到公元1681年，也就是康熙20年的时候，平定三藩之乱，才算是真正实现了统 一。

康熙一手建立起了整个满清帝国的基本制度框架，他充分吸取了蒙元帝国百年即亡的教训，从军事、文化、基层治理这三个方面入手，对帝国的制度进行了全面重建。

中华大地上的所有帝国最重要的制度始终都是军事制度，没有之一。满清帝国参照蒙元帝国建立起族兵制度，凡满清子弟从出生的那一刻开始，就是满八旗军中的职业军人，这些卒兵就是满清帝国的主力军。满清建国初期，满人只不过有30余万的成年男丁，但是征募了其中10万余作为职业军人，10余万满清军人的数量，面对当时8,000万的汉人，数量远远不够，于是满清朝廷又组建了以汉人士兵为主体的绿营，八旗军人和汉人的绿营军人，都是职业军人，以当兵打仗作为终身职业，因此可以比较长时间的维持军队战斗力。

当然绿营军人的待遇和地位都要远远低于八旗士兵，绿营的补给受到严格限制，高级指挥官也一定要由满族军人担任，以防止绿营军人有任何叛乱的可能性。军人的职业化在中国历史上一定会引起严重的军阀割据问题，但是在满清帝国这里居然不再是问题，这是因为十几万满清职业军人往中华大地上一撒，基本上就看不到人了，满人要维持住统治都不容易。至于军头们找个地盘割据一把，纯粹就是瞎扯淡，根本就做不到。

在文化领域，蒙元朝廷完全是自由放任的态度，因此元朝时期中国文化大爆发，戏曲小说话本的艺术水准都达到了非常高 的程度，但是满清朝廷根本就不吃这一套，他开始了极其严厉的文化钳制，审查所有的出版物，乃至是所谓的文字作品，包括私人的书信，但凡他认为存在一丝一毫的对满清朝廷的不敬，就会发作一场规模空前的文字狱，轻则斩首示众，重则株连九族。有清一代大型文字狱案件超过180宗，老百姓从此不敢谈论国事，也没有了任何值得一提的科技文化成就，整个国家成为了一潭死水。

在基层治理方面，中华大地上的历代王朝原本实行的皇权不下乡模式，政权架构只到县一级，广大的乡村地区主要都是依靠民间自治，但是满清帝国不满足于这一点。多尔衮从历史的垃圾堆里翻捡出来了王安石发明的保甲制度，到康熙朝予以系统化完善，中华大地终于出现了基层管控体系。满清的保甲制度有两个主要内容，第一，由所谓保长负责整个保的户籍登记、治安征税和民间纠纷处理。第二，一保之内有人犯罪全保连带担责，主要担责方式是罚金。保甲制度的实行进一步钳制了老百姓的自由，中华大地彻底的黑暗下来。

然而历史的吊诡之处就在于从来都没有完美的制度。从多尔衮到康熙，满清三代统治者殚精竭虑设计出来的种种剥夺中国人的自由，扼杀中国人的创造力的管控措施，终究还是会撞上制度的天花板。

先说军事制度，族兵制度与我们前面详细讲述过的兵库制度一样，它的核心问题在于难以持续，到第二第三代人之 后，所谓家族传承的战斗技能就会丢失。满清朝廷的办法是持续的打仗找尽各种借口打仗，以淬炼八旗军人的战斗力。满清开国之初，顺治朝击败了南明小朝廷的百万杂兵，康熙朝则一鼓作气击败了同样堪称精锐职业军人的吴三桂军团，然后挥军蒙古击败葛尔丹，雍正朝击败了青海的罗卜藏丹津，乾隆朝则击败了新疆的大小和卓叛乱，打到这个时候所有的内外敌人都被消灭得干干净净，满清帝国竟然获得了长期的和平。我们看看前面的那几场战争和南明朝廷以及吴三桂军团之间的战争还算得上是有必要。

至于康熙远征葛尔丹，雍正讨伐罗卜藏丹津和乾隆诛灭大小和卓，这些战争其实根本就没有必要性。这些个背负着反叛罪名的部落首领，其实根本没有干过啥背叛满清朝廷的事，撑死了也就是在各自部落内部的权力更替斗争中，没有第一时间征求满清皇帝的同意罢了。然而身系族兵制度弊端的满清帝王们，依然坚定将这些部落作为了满清八旗兵的磨刀石。乾隆朝之后，稍微像样一点的战争借口都不太能找出来了。边境上的游牧民族老老实实的生怕惹事，京城里的八旗兵走上街头一看，汉人一个个都自称奴才，磕头磕得一个比一个熟练，甚至还有了磕头培训班，专门教育汉人怎么把头磕得又快又响，这还打个啥仗，练个啥兵。

八旗子弟们志得意满，于是堕落成了整天喝茶斗鸟的闲爷们，八旗军的战斗力，在随后的100年里一步步的丧失干净。至于文字狱在乾隆时期达到顶峰，到嘉庆道光年间，这种持续的高压政策没人受得了，连官员们自己都受不了。尤其是道光年间官场法纪废驰，人人都醉心于贪腐捞钱，官员们对于兴起文字狱也没有了任何兴趣。满清朝廷对于大众思想的钳制就此放松了下来，于是基督教等西方宗教的传教士们居然得以在中华大地堂而皇之地传教，这要放在乾隆朝是不可想象的，而保甲制度则因为人口大爆炸而带来的巨量人口流动同样是无法维持。

满清时期面对的最大问题就是人口大爆炸。在中国历史上最为高压的制度管束之下，老百姓唯一可以干的事情就是生孩子。清朝立国之初人口约8,000万，在立国之后的长期战乱之中，人口减少到7,000万，之后用了100多年时间，仅仅只是到道光帝后期就增加到了4亿，如此迅猛的人口根本不可能将他们长期束缚在一个小地方不动弹，必须要允许他们出去寻找生路。于是从乾隆朝到道光年间，中国出现了持续了100多年的人口大迁徙运动，湖南江西的山区穷苦老百姓大批迁徙到四川，而中原地区的穷人则往南迁徙到福建和两广，成为了新客家人。

这些新客家人与当地的原居民争抢田地，矛盾十分剧烈。这些矛盾一方面摧毁了原本严密的保甲制度，另一方面则带来了造反的契机。1850年新客家人群体在广西发动太平天国起义，这个时候满清建国已经210多年，距离最近的大小和卓之战也已经过去了足足100年，满清的八旗兵和绿营都已经彻底丧失了战斗力，连盔甲的束带要怎么绑都忘记了。于是太平天国在两年之内就横扫整个南中国，如入无人之境，满清军队毫无抵抗之力，看着洪秀全在南京称帝，一点办法都没有，然而太平天国本身实在是不堪入目，洪秀全以及杨秀清、萧朝贵之类的太平天国首领，走的 是邪教立国的路线，核心的玩法大致是道教的请神上身，加上佛教的因果报应，再加上基督教的神话体系，以此忽悠信众。

在治政方面，这帮邪教首领复制的居然是满清开国时期的那一套严密的底层保甲式管控和残酷的思想钳制，乌烟瘴气，一塌糊涂。江南大地在10年时间内人口锐减一半，除了少部分是逃出去的，其他绝大部分都是被太平天国政权以各种残酷的刑罚杀掉的。原本全国的汉族士绅都在观望，对于当时已经孱弱不堪的满清朝廷，并没有什么忠诚之心，结果太平天国竟然更加不像样，贪婪残暴比满清朝廷还不如。所以江南大地上的汉族士绅们开始自行组建雇佣兵团，以对抗太平天国。

1853年曾国藩在湖南组建了湘军，他的学生李鸿章则在安徽组建起了淮军，在与太平天国的死磕过程中，这两支雇佣军团逐渐成长起来，成为了精锐的职业军人。曾国藩和李鸿章也一步步拿到了地方上的人事权和财富权，有权提拔人才，供养军队。1864年，经过11年血战，倒行逆施的太平天国覆灭，湘军和淮军成为了当时中国大地上的最强武力。满清朝廷手里残存的那点八旗军和绿营军根本就不堪与之相提并论。湘军和淮军这种汉族雇佣军团当然深受满清朝廷的猜忌，当时已经53岁的曾国藩顶不住压力，在一个月内就将12万湘军裁撤到只剩几千人。

这部分人辗转几年之后，大部分去跟了左宗棠，湘军算是完全退出了历史舞台。曾国藩手上没有了军队作为靠山，从此也没有了任何值得一提的历史成就，被满清朝廷从各种官位上调来调去反复折腾。

1872年61岁的曾国藩在两江总督的位置上郁郁而终，不过李鸿章做出了完全不同的选择，剿灭太平天国之后的李鸿章还算年轻，刚满41岁，在官场还算是年轻一代，这位朝气蓬勃的将军借口，他还要继续在山东剿灭捻军，硬是保留下来了5万淮军编制，一个兵都没裁。

有兵就有一切。于是接下来的世界就属于这位当时还年轻的英杰。1865年，在太平天国覆灭之后的第二年，满清朝 廷把手头上的1万多八旗骑兵集合起来，全部交给当时的满清第一名将僧格林沁，指望这位爷能在剿灭捻军的战场上，露露脸，长一长满人的威风。结果面对一帮作战方式是挥舞木棒和投掷石块的山东农民， 一万多八旗骑兵全军覆没，僧格林沁战死，丢人丢得举世皆惊。这一仗之后，全国范围内唯一能打仗的队伍就只剩下了李鸿章的淮军。于是从1865年开始，李鸿章就成为了中国最有权势的人，他花了三年时间肃清了捻军，并参与镇压了其他的一些小规模反叛。淮军趁势扩张到了10万余人。1870年权势滔天的李鸿章就任直隶总督，兼任北洋大臣，也就是当时的商务部长兼外交部长。这个时候的李鸿章执政京畿重地，手握重兵，并且控制着商贸和关税收入，手里要人有人要兵有兵要钱有钱，已经成为 了事实上的最强军阀。而当时的满清朝廷实际的掌舵人慈禧太后没钱没兵，对李鸿章一点实际控制力都没有，这个时候的李鸿章面前有很多选择，最强 猛的做法，是干脆终结满清的统治自己做皇帝。

游历过欧洲的李中堂对君主立宪制度不算陌生，如果他认为传统的帝国模式已经不能适应当时世界的话，那么他建立君主立宪制，自己做立宪制下的名义君王也是可以的。以他当时巨大的威望和无比匹敌的实力来说，他完全可以凭一己之力将中国带入君主立宪制模式下的宪政时代。如果这些变革都需要以民智已开作为前提条件的话，那么最简单的，李鸿章至少可以强迫满清朝 廷恢复宰相 的官位设置，让自己以宰相身份光明正大的压制皇权，进行政治制度上的变革，让当时还搞不清楚状况的满清皇族，解除高度专制状态，向民众放权让利，恢复民间思想和经济的活力，推动经济文化的全面发展，从而建立起能够真正实施军事总动员的基础。

大佬文集圈这样当外敌再次入侵的时候，身为宰相的李鸿章可以名正言顺地动员全国的人力、物力、财力以对抗外敌，但是李鸿章啥选择都不做，他虽然已经是全天下最有权势的人，但是一直满足于直隶总督兼北洋大臣这两个不高不低不尴不尬的位置，完全不谋求政治体制方面的任何变革。

1870年之后的李鸿章做了一些对大局根本就没有实际意义的工作，他很支持洋务运动，兴建了中国最大的纺织厂，兵工厂和航运企业。他将原来的淮军以西方的标准进行重新操练，升级为了新式陆军，也就是后来的北洋军阀。同时他还组建了北洋水师，当然了，无论北洋军阀还是北洋水师，都算是他李鸿章的私兵，由他负责找钱养活。他主持签订了30多个外交条约，每一个条约都让他背负卖国的骂名。在每一次签约谈判过程中，李鸿章都在积极为当时孱弱的中国争取外交权益，但是统统都徒劳无功，这些琐碎的工作耗尽了李鸿章的精力，让李鸿章迅速的衰老下来，但是做这些有什么用？这个国家没有在正治体制上进行任何的变革，满清皇族趾高气昂地呆在紫禁城里，维持着高度的中央集权体制，生怕向老百姓让渡一丝一毫的权利。在列强环伺的末日背景下，这个国家竟然从未实施过军事方面的总动员，连宰相身份都没获得过的李鸿章也不可能发起这种总动员。

于是1894年决定国运的甲午海战，李鸿章只能靠自己区区一个北洋军系的力量，与日本这一整个国家拼消耗这么拼下来的结果，当然只能是北洋水师全军覆没。甲午海战失败后，李鸿章主持签订马关条约，这是满清帝国签订的所有对外不平等条约中最惨烈的一份：割让台湾列岛，赔偿2亿2两白银，允许日本在中国投资开厂。

李鸿章这一把就将整个北洋系，整个清朝的财政和整个中国的前途全部赔了个干干净净，李鸿章从此身败名裂，心如死灰，再也没有任何作为，到1901年就黯然辞世。这位最有权势的人放弃了政治改革，那么满清朝廷也不再有任何政治改革的动力，就这么再拖了10年。孱弱的满清皇族却始终坚持维持着高度的中央集权制度，这就是弱小的身躯，却硬顶着一颗硕大的头颅，让每个人都恶心的要命，终于到1911年就跟开玩笑一样，武昌几个小兵随随便便的开了几枪，人都没死几个，北洋军阀出来跟着喊了两嗓子，满清朝廷立刻就垮了。

然而此后的中国再也没有了类似李鸿章这样可以团结绝大多数人的权势人物，全国处于一团散沙的状态。很快，中华大地就四分五裂，军阀混战陷入了彻底的黑暗。李鸿章一辈子最大的闪光点就是拒绝解散淮军，这算是他最有担当的时刻。但是此后 的李鸿章辜负了自己的历史使命，历史给了他至少可以做宰相的地位，即使再往上走一步，做立宪制下的皇帝或者真正的皇帝都有极高的可能性，但是李鸿章却拒绝了，他的一生就浪费在了开几个洋务公司，练几支新军的琐碎小事之上。这位唯一有能 力将中国带离黑暗的人没能尽到自己的历史责任。

我们今天回顾历史，只能这么评价，李鸿章从来没有意识到在中国实施总动员的必要性，他早年在曾国藩麾下击败太平天国之后，他作为主帅击败捻军，都没有实施全民总动员，靠的都只是他东拼西凑出来的一帮私兵而已。这一段经历深深的烙印在了李鸿章的血脉深处，令他深信只需要办几个厂，领着直隶总督和北洋大臣这样尴尬的身份，再靠一点关税和商税收入，就足以养出和列强对抗的强悍军队，护卫整个国家的安全。他的这种幻想原本其实真有道理，从鸦片战争的英军一直到八国联军，其实欧美列强都没有对中国派出真正的重兵，都属于小打小闹的性质，对付这种长途而来抢东西的敌 军，李鸿章的北洋陆军加上水师已经足以应付了，根本就没有实施总动员的必要。

在李鸿章的假想敌里，从来就没有日本这种尽举国之力而来，要与中国打一场国战，并且有意要吞并整个中国的对手。在李鸿章的世界里连想都没想过，中国会遇到一个一心要吞噬掉他的对手，当然就没有必要搞正治体制方面的改革，更没有必要搞总动员。所以当日本人的野心在甲午海战之中暴露出来，日本全国的老百姓穷的全家都穿不起裤子，也要捐钱给海军打仗的时候，李鸿章的世界观轰然破灭，这个世界不再是李鸿章可以理解的世界，所以他也只能在无尽的忧惧之中死去，也不知道当李鸿章去世的时候，他会不会对自己辜负了历史的责 任，而感受到一丝的后悔。
