
# 美俄要重启北溪管道，德国总理为什么拒发许可？

## 美俄要重启北溪管道，德国总理为什么拒发许可？

北溪管道，曾经是德国的“经济大动脉”，为全球领先的工业集群源源不断地注入俄罗斯廉价能源。俄乌冲突爆发后，德国决定对俄罗斯实施经济制裁，包括停止从俄罗斯进口煤炭和石油，同时俄罗斯也停止通过北溪向德国供气。2022年9月6日，北溪管道1号和2号被同时炸毁，真相至今成谜。

失去了廉价能源的德国工业，也就失去了竞争力。欧洲能源危机后，德国经济已经连续两年萎缩（2024年萎缩0.2%，2023年萎缩0.3%），雄心勃勃的工业4.0战略成为泡影，快速的“去工业化”正在把德国工业4.0“降维”成404或0.4。

现在，有一个机会放在德国新总理默茨面前：让北溪管道重启。美俄近期就俄乌停火举行谈判时，讨论了重启“北溪-2”项目的可能性。双方拟议由美国投资方从俄罗斯购买天然气，再贴以美国品牌向欧洲转售。5月15日，默茨给出的回答偏偏是，“不同意”。德国不会批准这条管道运营。

为什么默茨拒绝发放北溪管道的运营许可？

默茨作为基民盟主席，政治主张是大西洋主义，经济自由主义，向新自由主义靠拢的保守主义，信奉家庭、子女、教会、祖国，以及尽可能少的外来移民。他是坚定的亲美派。亲美派，现在拒绝美国的提议，并不奇怪。因为这个亲美派亲近的美国，是两党建制派当政的美国，是深层政府的美国，而不是特朗普的美国，不是民粹主义MAGA的美国。

特朗普上台之后，大幅度调整美国的外交政策，对俄改善关系，推动俄乌停火，对欧加征关税，还要求欧洲将国防开支提升至GDP的5%，引发了欧洲国家的强烈不满。更重要的是，特朗普的政策在沉重打击跨国集团的利益，比如所谓的“对等关税”。

欧洲的建制派，大西洋主义者们团结在一起，十分纠结地反对特朗普的“错误政策”，等待建制派重掌美国，主张欧洲团结独立，又竭力避免欧美关系彻底破裂，维持“欧美亲善”的传统，维护全球化和全球市场。

美俄重启北溪管道，卖给德国廉价能源，虽然对德国民族工业发展有利，但是对基民盟所代表的经济力量（跨国集团）来说不友好，或者说，时机未到。跨国资本还没能完成对德国工业、乃至欧洲工业的收购和整合。默茨正试图“转能源危机为机遇”，一方面推动欧洲能源市场的整合以降低德国企业成本，借“欧洲芯片法案”加速产业升级；另一方面依托欧盟“全球门户”计划拓展海外基建投资，为德国技术标准与设备出口开辟新空间。

同时，面对中美博弈加剧与多极化冲击，默茨将“欧洲”顶在最前面，能使德国在外交上更加灵活地处理与各大势力的关系，避免过度卷入大国竞争，换取谈判筹码，维护关键利益。重启北溪管道将打断默茨的所有战略，意味着德国放弃欧洲的政治经济领袖地位，单独与美俄媾和，从而放弃“欧洲自主化”和“欧洲安全”议题，以及藏在这两个议题下的“德国正常化”进程，损失已经投入的大量沉没成本。

欧洲在用制裁俄罗斯能源做筹码，换取利益。2025年5月6日，欧盟委员会公布了逐步终止进口俄罗斯能源的计划“路线图”，打算到2027年底停止所有俄罗斯天然气进口，并限制进口俄罗斯的石油和核能原料。除非俄罗斯与乌克兰达成停火协议，而且是有欧洲参与的协议。

在俄乌冲突中，现在有四方力量——美国、俄罗斯、乌克兰和欧洲。美国和俄罗斯都不希望看到欧洲“坐上餐桌”，从战后的俄乌双方分到一杯羹。乌克兰则希望欧洲“上桌”，能让美国、俄罗斯别吃得太难看。欧洲想“上桌”，但是手里的牌不多，只能威胁制裁俄罗斯能源出口，以及俄罗斯能源的大客户——也就是欧洲自己。

5月16日，德国媒体报道称，欧盟成员国正在讨论更为严厉的制裁措施，这些措施与俄乌双方在伊斯坦布尔的谈判结果密切相关。其中包括全面停止进口俄罗斯天然气（目前俄罗斯天然气占欧盟进口总量的近20%），以及限制从俄罗斯进口用于核电站的铀原料。

俄罗斯外交部长拉夫罗夫曾表示，反对重启北溪管道的欧洲政客“要么是精神有问题的人，要么就是在走自杀路线”。但欧洲也没有别的办法。如果北溪管道重启，俄罗斯能源变相恢复对欧出口，欧洲将失去胁迫俄罗斯，坐上俄乌谈判“餐桌”的重要筹码。

德国当然参与（甚至主导了）路线图的制定，也想以欧洲带头大哥的身份在俄乌手里分一些利益。默茨不可能突然改变政治立场，支持北溪管道重启。德国和欧洲不希望美国完全掌控其能源安全，美国不希望欧洲“联邦化”。

2024年，德国有92%的液化天然气要从美国进口。但德国仍有选择的空间，还能花钱从其他国家购买，甚至保留了“万不得已”与俄罗斯妥协，重启北溪管道的机会。如果北溪管道也被美国所掌握，德国将进一步失去其能源安全，同时失去与俄罗斯拉近关系的“最后一招”，失去与美国博弈的所有底牌。

这是默茨不愿意看到的。

如果将视野上升至欧洲层面，就能看到特朗普的美国打算彻底“包围”欧洲，对欧盟进行全方位的能源控制，方便进行经济收割和政治操纵。特朗普在俄乌矿产协议谈判时，曾经明确提出，要由美国国际开发金融公司控制从乌境内向欧洲输送俄罗斯天然气的管道，如果再加上北溪管道，就基本控制了欧洲东面的能源命脉。

通过与胡塞武装对抗，美国航母舰队实际上已经控制了红海和苏伊士运河。美国还要吞并格陵兰岛。从外部输入欧洲的能源通道还剩几条？如果特朗普实现了对欧洲能源的控制，那么下一步推动欧洲右转，支持代理人或右翼政党上台，甚至颠覆欧盟，将欧洲重新变成听话的美国经济殖民地，整合“西方世界”的力量，更好地应对中国挑战，就会变成一种可能。

根据多家欧美媒体的披露，2025年3月11日下午，美国“传统基金会”举行了一次闭门会议，主题是如何搞垮欧盟。特朗普要对欧盟不利，既因为美国实力在衰落，需要攫取更多的外部利益，也因为欧洲正在团结起来，欧盟委员会的权力越来越大，逐渐接管欧盟各国的财政和安保权力，从而走向一体化。

如果欧盟在一体化的道路上走太远，就会“联邦化”。美国对这种趋势最熟悉，它就是这么走过来的。如果欧盟联邦化，不论是否能够搞活经济，都将对美国在欧洲的利益，以及美国在全球范围内的权势造成永久性不可逆的巨大伤害。

当然，害怕欧洲联邦化的不只是美国，还有欧洲国家自身。特朗普抛出美俄要重启北溪管道的议题，是高明的试探。他不怕默茨拒绝。德国国内还有选择党，今年议会选举时，魏德尔就公开表示，如果德国选择党能够取胜，就将推动重启北溪管道。默茨拒绝重启北溪管道，这个议题就会加剧德国国内的政治分裂。

这也是为什么德国国内一直在讨论“取缔德国选择党”。因为AFD给了以特朗普为首的外部势力无数极好的“政治抓手”。

俄罗斯没有公开反对，拉夫罗夫的表态是，“如果美国利用其对欧洲的影响力，迫使欧洲不再拒绝接受俄罗斯天然气，那将耐人寻味。”俄方估计会要获利的美方用其他利益兑换。反倒是默茨，其实没有什么选择，必须拒绝。

作为新任德国总理，默茨接手的经济是个“烂摊子”，政治上也是一团乱麻，但是作为欧洲工业最强的国家，德国未尝没有翻盘的希望。对中国来说，德国在多极化世界中有着极其特殊的地位。如果默茨领导的德国能够扮演“务实引领者”的角色，推动欧盟对华合作，将会大大促进双方的经济发展，改变全球的经济产业格局。如果正相反，就将大幅度影响德国的经济复兴，拖累欧洲经济，导致更多的产业外流，同时让出更大的全球市场，对中国来说也不是太坏的事情。

不论默茨如何抉择，都将深刻影响德国的未来，影响欧洲的未来，影响中欧关系的未来。
