
# 大佳10.7：信息

## 行情私货
港股在上周四大涨之后，周五和今天，均小幅度回调。除了部分科技股，尤其是芯片之外，基本回到上周二收盘节点。今天回调，一个很重要的因素是，美元指数强力反弹。这里和日本股市的对比很鲜明。日本股市上涨，是因为新首相上台之后，加息可能会被推迟。一方面导致日元暴跌，一方面也导致股市暴涨。这里，日元暴跌，引发了连锁反应。美元指数大涨之后，欧元也突然加速下跌。而美元的大涨，却没有导致黄金大跌，黄金反而加速了。推测是因为，欧元和日元下跌之后，增加了这两个经济体，关于黄金的买盘。也就是说，虽然美元指数反弹幅度很大，但资本产配置的角度上来说，大家似乎更喜欢押注黄金。当然，因为港股几乎都是跟着美元指数来波动，因此，这几天都没有找到很好的位置来进行交易。港股组合，在假期保持不变。

今天是中秋节，祝大家假期愉快~明天我们会陆续到岗，开始进行部分工作。似乎很多用户都很关心我们的组合的，关于资金跟踪的情况。这里详细说明一下：
1,A股和港股，两个组合，都有很多资金在跟踪。尤其是A股组合。不仅仅是我们自己，而是有明确的资金实力雄厚的大户和一些机构在跟。港股本来跟踪的规模会更大，但被我们婉言谢绝了。主要是07552引发了很多争议。本身对我们来说，这就是实验性很强的一次对冲。至少需要一年以上的时间来验证。等时间超过一年之后，我们会写一份报告。
2,关于三个20组合。同样，有资金在跟踪进行试验。市面上其实对于这种主题策略的需求很大。尤其是红利20,因为有中证红利作为基准比较，所以关注度其实很高。但这一部分，和我们并没有特别的协议。但我们会一直完成长期的跟踪和轮换。
3,关于9月底公布的高股息轮动。这是一个纯粹的低频量化策略。目前在实验阶段。虽然做过回测，历史表现很不错，且逻辑上没有问题。但因为是我们第一个低频量化，所以我们更多的是呵护和共同成长。未来，选股的参数上，可能会发生一些改变。但我们会在更新的时候，继续进行开源处理。不会有任何暗箱操作，保证透明性。如果大家对这个轮换策略感兴趣，也可以私信我们。

总之，在国庆之后，我们会对我们星球的内容，进一步强化。主要方向是，彻底摆脱传统的吹票模式，转化成策略和观点模式，策略甚至在观点之前，观点为策略服务，尽可能更加贴近大部分用户的实操。我们的目标是，让绝大部分用户都会觉得，物有所值。也即，星球提供的内容和服务，远远超过星球费用本身。其实，这本来就是我们今年以来，一直改革的方向。早期加入星球的朋友们，应该能够感觉到我们的东西越来越多，而不是越来越少。当然，中间也有很多不足的地方。比如，有一些策略，虎头蛇尾。比如，因为跟票跟交易的形象特别突出，导致跟踪我们的交易，出现了比较明显的亏损。而最近的一系列风波，又会在主观感受上，放大以上的问题的影响，从而影响我们改革的进程。对于，现状我们都接受。但接受不等于摆烂，受影响不等于是足不前。今天就到这里，再次祝大家中秋节愉快！请务必，多多陪陪家人。哪怕说一些简简单单的话都好，陪伴才是最重要的。
