
# 学习笔记，2025.1.18

1. 焦虑不可避免，坦然面对它，学会转移负面情绪，给自己做好心理调节。不要总和别人去比较，给自己带来不必要的焦虑。守住初心是根本，别因为一些小私小利毁了自己的底线原则。过程永远比结果更重要，不管成功与否，有所经历就是一种收获。

2. 一句话总结，现在年轻的女人，普遍浮躁。年龄大的女人，普遍躺平，混日子。能踏实干活的，就是30到35岁之间的。还有一点，40岁以上的人，例如我现在40岁，普遍开始研究养老了。

3. 绝对敬畏概率，尊重概率，不挑战概率，不轻易突破自己，尤其是户外、探险这些领域，玩沙漠，玩无人区，凡是我认为超出了我的可控范围的坡或沟，我都绕行，哪怕大家都说没事，我也绕行，我是一个非常小心翼翼的人，我写安全驾驶，不是说出于市场需求而写的，而是出于性格，我觉得我就是这样一个人，我爹年轻时是基建营的拖拉机手，我奶奶说，方圆十里的人赞美我爹，都是同一句话，开的拖拉机不紧不慢的。这玩意，我认为是一种天性。

4. 佛教缺少一个马丁路德，佛教的主基调还是在宣扬生死轮回，在封建社会，这是有市场的，在科学时代，这个受众就会越来越小，因为越来越难以自圆其说，佛教现在相比基督教而言，最大的问题是中介制，信徒不能直接跟佛祖对话，而是通过寺院和僧侣进行中介，有中介就有腐败，过去基督教也是这个模式，有人扮演神父，现在普遍是牧师，是帮助你与上帝直接对话的，是老师，不再是中介。出发点也要改。现在的修行，动能多是恐惧。害怕下辈子成了牛马。要把动能置换成 LOVE，这一点，基督教走在了前列……

5. 在任一领域，再小的名气，也能吸引到舔狗。哪怕只是在村里干个村主任。也能！

6. 被拒绝是常态，被接纳才是恩赐。乞食的过程并非是对食物的追求，多次碰壁的求而不得也是对心境的一种修炼。这方面，我感觉自己也进步了。

7. 去深圳时，我坚决要求去莲花山打卡。没有改革开放，就没有今天。当然，有朋友后来给我悄悄的科普了一个数据，真正腾飞是WTO。但是，我也觉得改革开放是第一跳，WTO是第二跳。没有第一跳就没有第二跳。期待，改革春风早日刮到对面，当然，有个比改革春风更牛逼的方案，那就是柏林墙的倒掉，那样的话，丹东会成为下一个香港。

8. 对面有很多优秀的画家，贵的一幅也要几万元人民币，他们偶尔也会到北京到上海，还在那边有工作室，钱也会留在中国，我曾经问过书画商一个问题，他们这些人这么有钱，当他们回到了那么一个环境里，会不会很压抑？会不会很另类？书画商的说法是，这些开眼看过世界的人，当回到原来的世界时，会表现的比那些人还那些人，彻底把自己融入群众当中，把自己好好的保护起来。我突然在想，那些大放厥词的教授，是不是也在如此？在故意表演着什么！一类是，出去后，依然是中国人。一类是，出去后，成了当地人。一类是，出去是当地人，回来是中国人。

9. 高门槛是朝下限制，低门槛是朝上限制。怎么理解呢？若是你组织了一个群，不要钱，谁都能进，那么，优秀的，是拒绝的。没有门槛就是拒绝了真正优秀的人！这一点，悟透是非常难的，大作家多有自己的书院，书院就需要做个群，自己的朋友，出版圈的，新闻圈的，写作圈的，优质读者。听起来非常高大上吧？尤其是自我介绍时。我看N多处以上的，但是，这些群，最终全乱了。因为，活跃的就那么几个，他们会把自己写的文章分享进去。每天就成了广告。就他们几个……现在，免费的群，我基本都不加。浪费时间。浪费心力。所以，要做圈子，应该从隐蔽、神秘、小众出发。

10. 其实小微企业反而希望每个月纳税，一是为国家奉献一点力量，二是求个心安，这个月纳了就没心事了。尤其是听说财政压力很大时，不由的感叹这个点。其实呢，大家内心能接受纳税，觉得开门做生意，纳税天经地义，虽然纳的少，但是毕竟群体庞大。反过来，因为常年不纳税，类似农民现在不用交公粮了，总担心突然来次大的……电商公司注册在海南或杭州，不要注册在东北。东北只适合干一件事。种地。
