
# 欧大2025腾飞元年线下年度演讲

OK。OK大家好！今天是1月19号，还有十天就要过年了。我们在中国的一线城市深圳，在一个非常美丽而且气候很温暖很舒服的一个城市。小孩子从北京过来，但都不想走了，所以这是个非常美的地方。

今天欧老师的演讲，我的标题叫做2025终之章。终结的结中指长什么意思呢？意思说很多事情我们等了五年了，等了十年了，等了很久很久。很多事情到了2025年要画一个句号。2025年注定是一个不平凡的年份，是一个跟之前的三年、五年都不一样的年份，是一个历史的很多事件发展到今天来了一个总结局的一个年份，2025必将载入史册，2025是过去的很多的蛛丝马迹，很多的草蛇灰线，到今天我们要画一个句号。

那为什么呢？提高老师慢慢往下降。首先的第一件事情就是太平盛世太平。今天大家来深圳，坐在我们这个富丽堂皇的大厅里面，你们感觉日子过得怎么样？好，中午饭吃了吗？三点半有茶歇，茶点也很好吃的。我们在这里无忧无虑，无灾无难，风和日丽，气候又好吃的又好身体又好，人生还有什么不满足的？我们是不是在太平镇死。

我今天和老师演讲的主题是什么？是S3。我待会59页PPT讲的都是S3，你们最喜欢这个S大 家会说我。为什么我们放的这么美丽的，这么舒适的我们的太平生活。不过你要去搞什么S3，你是不是一个战争贩子？是不是个窝囊的，是不是鼓吹战争的人，是不是一个好日子不过非要过荡不安的意思。如果大家把眼睛一闭，当做什么事情也没有发生。如果大家把眼睛一闭，把今天的日子继续过下去，能不能过到这个？

如果我们忘掉过去的，我们忘掉一切的一切，就今天做什么，明天还做什么，行尸走肉一样的继续去上班，继续的去工作，继续去消费，当做什么也没有发生。我们能不能再过一个完美的太平生死？我想过能够可不可以？为什么我们要动荡？为什么我们要惊涛拍浪？为什么欧老师要像个战争贩子一样在这里推销S3？我们能不能就这么过下去？

盛世太平，能不能继续？盛世欢迎继续的一个前提就是我们的所有的存款要全部清零。你能不能接受？你奋斗了一辈子，存了200万，存了300万，你省吃俭用一辈子，你有500万出积蓄，500万理财，想靠这个养老想活下辈子。但是我跟你说银行要破产，你银行的钱取不出来，它归零，你能不能接受你的存款归零？如果我们在座的所有人都能接受我们的存款清零的话，那么我们的态度甚至可以再继续实行。这个是2025年经济的现状，说了能不能不能存款清零，我们再过我们的太平日子。

不说另外一个话题，如果我们像今天这样苟延残喘走下去，如果我们不做任何改变，像老板这样天天发个通告，疾病增长百分之，形式不是小好，而是大好一片大好，形式是个获得了质的提升。再小如果我们自我陶醉掩耳盗铃，中国会怎样？目前我们这个城投债欠了100万亿，会导致中国发生什么事情？如果我们不做改变的话，有什么力量促使我们改变？

非常简单，中国在减速器，你们有没有发现，中国的经济一年比一年差，一年上一个台阶。GDP增长不要说八点几、六点几、七点几、五点几、四点几、三点几、二点几，失业率一年比一年高，生意一年比一年难做。中国经济在急剧的建设。如果我们不印钞票，如果我们就这样当做没事发生，把眼睛蒙起来，掩耳盗铃。那么我们会面临的情况是中国经济越走越慢，越走越慢，他跑不动。

中国经济为什么会跑不动？因为我们整个社会的信用崩塌的，我知不知道地方政府叫B2B businesstogovernment，现在鼻突肌的声音根本没有人听。央企国企的单子以前是最好的。金国以前要请客吃饭，要去KTV，要请大家喝酒才能接下来订单。现在国企央企送给你这个工程都没有人接了，为什么呢？收不到钱，收到钱。目前所有的B2C的项目都没人接，因为我拿不 到钱。BtoG没人接的话怎么办？

地方政府动员能力接近他们，地方上没有行政能力。以前一声令下，我们去做志愿者，我们去做大白。以前我要了10万20个志愿者，一声令下，一堆的人全都帮你干活，现在还有没有人帮你干活？在你的信任，我已经崩塌掉了，人设塌掉了。现在做任何事情，对不起，我家里有事，我说小孩子要哭，小孩要照顾，今天来不及任何事情没有人跟着你干，整个社会组织能力接近瘫痪。

我们现在在深圳，深圳还好一点，深圳没有银行危机。你知道锦州银行、赣州银行、平顶山商业银行、小银行的地方，河南村镇银行跑路跑了400亿，跑路小地方也。中国2025年有接近十家地方小银行发生了信用崩塌。整个城市它。整个城市银行不能被信任，任何生意做整个城市河南村镇银行一款多少人，我的钱一辈子在里面的钱没办法支付。我们的工资大量拖欠，教师、公务员欠多少钱的工资，整个社会到处是欠钱，没有钱来结算。所以中国生意作为经济搞不了，整个系统越绷越慢。有一点是共识，就是我们不解决这个问题的话，我们这个社会就越跑越慢，越来越跑不动，越来越不好。

欧老师跟你们说，今天我们是盛世太平，在深圳这么一个美丽的城市，在我们大家都吃饱喝足，日子过得很逍遥的日子。但是很可惜，明天不能这么过。如果我们不把这件事情清掉以后，把这个账掉的话，它会越跑越远。他会越好。

有一点是任何人都无法质疑的，反驳的，就是中国的经济在逐年下降。是不是郭老师说的对不 对？逐年下降说的对的，给个掌声。

我们在外面做生意都是要求回报的，你格局你不要回报。但是任何人都说了，如果我去打工，老板结不出工资，如果我去做公务员，做老师，半年不发工资，那我就不能去干这个活了。在政府的号召力，志愿者根本使唤不动，地方政府的控制能力接近他。如果我做生意，做生意的 话就要结钱，就拿钱。我自己垫了这么多的钱，帮你搞个工程，帮你修个路，修个桥，到最后工程款接不到，那我去桥路。

现在中国是什么生意都做不了，因为哪里的钱都借不到。我做你的生意，有可能你是前几百亿，你是破产，我放在银行的钱，有可能银行是破产的，银行亏空了，银行的钱取不出来。在中国夸张到什么程度，你一个一等良民放在阶级卡的钱都能取出来。你感受到中国经济到了这个地步，他转不动了，他会越跑越慢。
