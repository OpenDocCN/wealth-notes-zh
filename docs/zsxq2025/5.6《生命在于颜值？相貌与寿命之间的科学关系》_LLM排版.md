
# 生命在于颜值？相貌与寿命之间的科学关系

## 数学门诊的设立与意义

数学门诊的全称叫做“空间与数学学习困难门诊”，是由上海交通大学医学院附属上海儿童医学中心与上海交通大学心理学院联合设立的，2024年10月正式开诊。开诊第一天，预约号就被抢空了。

数学门诊主要是面向学习数学、物理、化学等感到困难，以及有空间感知障碍、计算障碍的学生，看他们的数学问题到底是由于空间障碍，还是心理焦虑。

数学门诊的设立，其实是给了我们一个观察孩子学习行为的新视角。全世界的孩子数学成绩正在出现不同程度的下降。根据世界经合组织开展的一项国际学生评估（PISA），全世界孩子的数学分数低了15分，相当于在数学方面落后了9个月。

数学学习障碍的患病率大约为3%到6.5%。咱们国家也差不多是这个比例。每100个学生当中，就至少有3个存在数学学习障碍。59%的孩子会担心自己学不好数学。

数学更容易让人焦虑。同样是学不好，数学方面学不好就要比语文方面学不好更焦虑。同样不及格，数学不及格的孩子就要比语文不及格的孩子更抑郁。

数学门诊不仅是为了解决孩子的焦虑，也是为了解决家长的心病。开设数学门诊，不仅是为了解决孩子的焦虑，也是为了解决家长的心病。

## 颜值与寿命的研究

来自江苏师范大学的团队研究发现，运动确实能提升颜值。运动对女性颜值的提升效果略高于男性。运动对颜值提升最明显的人群是，40到59岁，城市里的女性。

颜值吸引力过低，会提高死亡风险。高中毕业时长相吸引力得分最低的人，在各个年龄段的全因死亡风险，比其他人要高17%。吸引力高的人，能获得更多的社会互动，这能提升心理健康水平。

严格来说，这个研究的最终结论不应该叫颜值影响寿命，而是，在各方面对自己有要求的人，更有可能长寿。

关于颜值与寿命的关系，也有很多人认为，不是颜值决定寿命，而是寿命决定颜值。也就是，一个人之所以有更强的外表吸引力，是因为他的健康状态更好，而不是因为他有吸引力才健康。

人体的免疫力与外貌吸引力之间，存在着一定的相关性。NK细胞的免疫功能越强，说明这个人的免疫能力越强。对男性来说，NK细胞的功能越强，他的外貌得分越高。对女性来说，同样是免疫系统状态越好，外貌得分越高。

总结一下，医院为什么开设数学门诊？因为数学更容易触发学生的焦虑，这些焦虑的根源也许不在数学，而在学生的家庭。而数学门诊的对象不仅是学生，也包括改善家长对待孩子的方式。颜值为什么会影响寿命？很多时候，人不是因为好看才健康，而是因为健康，因为对自己有要求，才变得好看。也祝你，又健康又好看。
