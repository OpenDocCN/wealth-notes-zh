
# 大盘能够一鼓作气地站稳在3400点上方吗？

上周五的文章，我用大标题明确提示："3300点是近期A股正常调整的关键位"。本周二，上证指数在触及3297点时迅即就被多方拉起。显然，多方主力也同样意识到了3300点的得失对近期走势的强弱和人气都有着十分重要的作用。

本周五，上证指数最高上摸至3390点，距3400点仅一步之遥但仍然没有站上去。可见，我前两期的文章中反复提示的3400点上方的近期有很大阻力的判断也确实在实际走势上获得了验证。

现在的问题是，在本周最高上摸3390点但尾盘又回落至3372点的情况下，上证大盘还能不能延续本周的上行势头呢？换言之，在周线图上最近的三根周K线的组合形成的类似双阳夹一阴的多方略微得势的组合态势下，近期大盘究竟能否一鼓作气地站上3400点呢？

我的看法是：近期上证大盘即便能够站上3400点，但对于3400点的多空争夺战仍然不会很快结束。并且，如果下周大盘没有能力再度摸高本周的3390点的高点，投资者甚至还要提防未来几周的大盘仍将在3300-3400点的箱体区间内进行来回震荡。

作出这个判断的基本理由是：

1、波浪结构上，从3140点以来的回升浪型看，在分时走势上可以有两种分形。一种是：3140-3388点，已经完成了一组上升5浪的结构。如此，3388-3297点的回调应当属于a浪调整的性质，而3297-3390点只是一个浪反弹。按照这个浪型结构，那么，下周大盘将会向下走调整c浪。c浪调整的目标，将再度考验3300点的支撑力度。

2、如果从3140点上来的这组结构中，3388-3297点的回调是属于一个简单的4浪回调的性质，那么，本周3297点开始的回升就属于第5上升浪。若果真如此，从本周低点3297点上行以来的分时走势，到本周五为止应该还尚未走完。按照这个浪型推算，下周大盘的高点是可以摸高3420点附近的。从而完成整个一组上行结构。然后再运行下一组调整浪的结构。

3、即便上证大盘是按照第2条的结构来运行，其后大盘的调整，理论上仍然要再度回试3300点附近的支撑力度。

由此可见，无论下周大盘是能够摸高到3400点的上方，还是直接向下震荡调整，理论上，其共同点都是还要再度验证3300点的支撑力度。所以，从这个意义上来讲，未来几周，大盘的主要运行态势就是在3300-3400点的区间内来回震荡。这是目前最为合理和理性的预判。佐证这个预判的一个重要因素是：本周的周成交量是有所下降的。

当然，最为合理的预判也只是一种预判。实际走势会受到政策层面、美股走势以及中美关税战等各种因素的影响。例如，有投资者问：有没有可能在某种强力因素作用下，直接走成我之前的专栏文章里提到的将目标指向3500点方向的大浪？我当然无法排除各种可能性。虽然，我目前认为这种可能性极小极小。合理的走势应该是在3300-3400点之间通过震荡夯实台阶，然后再走第二段指向3500点方向的回升行情。这是比较理想的结果。别忘了，高层反复强调的方针是“稳定楼市股市”。注意，是“稳定”！为什么是“稳定”？投资者不妨认真想想。

要提醒注意的是，上证大盘在3300-3400点区间里的震荡，理论上是不可以再有效跌破3297点的前低的。因为那样的话，A股大盘今年上半年的整个行情很可能就在3400点附近走到头了。

这里，我还要提请投资者注意：上述分析和预判，我虽然是通过技术语言来描述的，但其中已经蕴含了包括政策因素、中美因素和美股震荡等因素在内的综合判断。毕竟，我是有着深厚金融学背景的市场专业人士，不是仅靠拍脑袋来凭空想象几条技术路线！

既然是预判，就不可能百分百的正确。投资者还是要有自己的判断和认知。

顺便提一下：本轮行情虽然各种概念眼花缭乱，个股也精彩纷呈，但真正影响港股和A股市场整体走势的，我认为只有两个风向标：阿里和腾讯。根据我对一段时间以来每个波段走势起落的观察发现，阿里和腾讯强，港股就强，A股也强。阿里和腾讯若走弱，A股自身往往就直不起腰来。

而科创和人工智能等科技板块，最具风向标的是寒武纪。因为寒武纪在相关ETF中的权重最大。所以，某种意义上讲，寒武纪的强弱，也往往意味着科技股和科创板的强弱。这也是为什么我在之前一两年大盘的下跌过程中，反复提示要定投中概互联ETF和科创50的原因！那些认同我的理念和思路的粉丝投资者，无疑，是真正的有缘人！
