
# 帝王术实操

假如有一天，又有一艘大游轮撞上冰山了。那幸好，这回他们备下了足够的救生艇。杰克和肉丝，再也不用生离死别，他们跟另外的998个乘客一道坐着救生艇，划到了一座与世隔绝的荒岛。非常走运的是，你正是这幸存下来中的一员。就在其他的凡夫俗子还在庆幸自己好不容易活下来的时候，那志向高远的你已经在考虑，接下来怎么样才能奴役其他的999个人，甚至建立自己的王朝。

那说到奴役他人，大家脑海里第一时间恐怕会想到那一群衣不遮体，像伏尔加河上的纤夫那样的可怜虫，在鞭子的驱赶下，生无可恋的干活。其实，在现实中，这样用鞭子赤裸裸的奴役是极少数。毕竟，拿鞭子打人这个活，你让猴子学两天他都能学会。要是我们只懂这个，简直白瞎了人类几百万年的进化。那如何奴役、如何统治，这是一门技术活。

这门技术，在古代有个非常牛叉的名字，叫做帝王术。那这门技术的实践与运用，不少人写过说明书，比较出名的有商鞅的《商君书》和马基雅维利的《君主论》。马基亚维利只不过是把桌面下的一些手段摆在明面上讲了而已。那跟遥遥领先完全没有道德底线的商鞅相比，那真是给商老师提鞋都不配。那当然了，这历朝历代都是禁书。那如果以前老百姓研究这玩意，轻则脑袋搬家，重则一家人整整齐齐。今天这篇文章，我们就稍微检阅一下，一边理论，一边用刚才的例子去追随那些帝王将相，追随那些人民的慈父和红太阳走过的路，看看这神乎其神的帝王术里，究竟藏着些什么玩意儿。

那回到我们的小岛，在遥远的未来，经过多年奋斗的你已经垂垂老矣，这天，你引着尚且青涩的太子，迎着午后的阳光，爬上了岛上的最高峰。这座山，刚刚被你划成禁区。一般人等除非得到最高层也就是你的允许，否则是不能擅自进入的。那当然了，你做出这个决定，完全是为了大家的利益。毕竟这座岛上的一切，包括这座山的所有权，都归属全体人民。你所拥有的，仅仅是每周七天，每天24小时的使用权而已。

在山顶的微风中，你们父子俩看着脚下，一群群工蚁般干活的人们。那这个时候，你本该拍着儿子的肩膀，指着山脚下的人说，看，这就是朕为你打下的江山。但是，这个时候你什么也没说。一阵温暖的海风拂过你满是皱纹的脸庞，把你的思绪吹到了多年以前。那是你们一群人海上漂浮多日，刚刚狼狈的爬上陆地的那个遥远下午。那一项伟大事业的开端啊，往往不像书里写的那么光芒万丈，大多数，都很仓促，甚至有些狼狈。这回也不例外，当幸存者们头上顶着海藻爬上这座小岛的时候，每个人都惶惶不安，对未来一片迷茫。

不过，还有一个人是个例外，怀揣伟大理想的人，当然不会被这些无谓的情绪困扰。虽然前途是光明的，但是，饭得一口一口的吃，路得一步一步的走。你看着海滩上这些劫后余生的男男女女，有些嚎啕大哭，有些互相安慰，更多的人，是茫然若失地坐着。那这些蝼蚁就是你未来的子民，未来的奴役。不过，我们低头看看狼狈的自己，放在人群中毫不起眼。那么这个时候你会问自己一个问题，凭什么，凭什么是我，在这么一群人里面，有的身强体壮，有的学识渊博，有的身份显赫，还有智商高的。那这么多人，凭什么轮到你呢，总不能光凭你长得帅吧，这个理由也不太能够啊。

不过，好在我们懂一点点的历史，知道已行之事势必再行，日光之下并无新事。于是，赶紧翻翻前人的智慧，看看历史上那些成大事的屌丝们是怎么起家的。曾经，有个泥腿子叫陈胜，整天都不好好种地，就爱到处的吹牛叉，说什么老子以后富贵了，不会忘了你们这些穷屌丝的。大伙听了这个话，都觉得这丫病得不轻。后来，陈胜领那几百号人去前线当炮灰，走到半路，就开始琢磨怎么鼓动大家造反。那彼时彼刻，恰如我们的此时此刻，那个时候的陈胜也是光棍一条，啥啥都没有。那他的创业项目又是怎么启动的呢，那陈胜老师的方案，就是编鬼故事。那其中最出名的，就是半夜三更不睡觉，跑到附近去学狐狸说人话。老实说，这个操作我到今天都没有想明白，一个人是怎么模仿狐狸说人话的，可见，陈老师在口技方面，确实是有两把刷子的。那最后的效果就是，大伙半夜三更都被狐狸吵吵的睡不着觉，那个家伙嗷嗷的叫着什么：陈胜王，把大伙唬得够呛，私下都在传，哎呦，这个陈胜原来是狐仙转世啊，怪不得身上这么大味儿呢，啊，之前还在心里嫌弃过他，罪过罪过。

那与陈胜同时期，还有个叫刘邦的，有天，走到道上砍了一条白化病的蛇，那就这么个小事情，就被他可着劲的到处跟人摆显。后来还觉得不过瘾，又编了个爬行动物给自己老爹戴绿帽子的故事。没过多久，又有一个南华老仙的亲传弟子张角，张老师喊出了：苍天已死、黄天当立。那离我们最近的还有耶稣他老人家的亲弟弟洪秀全，洪老师带着一帮子天帝、天妹、上帝，那么一家子下凡，半路还搞出幕天兄杀天父的人伦、神伦惨剧。历史上，这种拿着神仙盖章造反的事情，跟蟑螂一样多，就连宋江这么个土匪山贼，都要搞一套九天玄女亲笔签名的秘籍。

那这里面的原因到底是什么呢，那可能有人会说了，这有什么好研究的，不就是拿着神仙背书去忽悠那些愚夫、蠢妇，然后煽动大家给自己卖命呗，那这么说，错倒是没错，但是，如果仅仅满足于这样简单的解释，那这个世界就没有什么事情值得研究了。那所谓航天计划不就造个金属壳子，里面塞几台发动机吗，然后一点火就飞上去了，那所谓芯片制造，无非就是抓一把沙子，然后烧一烧，最后涂上胶，再用光照一照就好了，按照这个思路，万事万物都没有什么好说的了，所以，有些事情还是值得我们花点时间，去深挖一下的，比如说，刚才帝王术这个第一个重点，一个人活着就会产生各种各样的欲望，或者说愿望，视为愿力，比如说，有些人想要漂亮的女朋友，有些人想中六合彩，那在大多数情况下，这些愿力各不相同，也零散于各地，也没有什么意义，但是，总有些时候，总有些人会对某件事情产生相似的愿望，比如说，皇帝老二逼得大家活不下去了，很多人，想换个活法，这个时候的愿力就会格外的强大，但是，我们是普通人啊，没有办法像元气弹那样去吸取这些愿力，这个时候，就需要一个管道，或者说一个抓手，让这些愿力为我们所用。

那在游戏或者网文里面，主角一般都是天生自带一套系统，可以把万千臣民的愿力转化成自身的力量，那么这个人，就有能力做出常人所不可企及的事情，比如说，一刀劈开一个小岛啥的，那当然了，我们这里不是写玄幻小说，不过，在日常生活中，这套愿力系统其实一点都不玄幻，甚至，多到烂大街的程度，那这个玩意儿，一般我们管它叫做选票，政客的竞选，就是获取他人愿力的比拼，最后赢下来，那个人，也就意味着获得了最多的愿力，也有能力做出常人所不可企及的事情，比如说能拿导弹炸个小岛啥的，其实古代所谓的神仙背书，跟政客参选也是一样的，本质上，都是为了获取更多人的认同，进而把万千的愿力化为己用，只不过，路径不同，表现的形式也大相径庭。

现代选举，是让人们的愿望直接化成一张一张的选票，汇聚在个人或者组织上，这种汇聚是直接的，是没有任何中间商的，天然就具备合法性，但是古代的愿力获取，先是大众对于超自然力量的信仰，把没能满足的愿望，比如说，能活下去，寄托于漫天的神佛，那有志青年们，再通过各种神仙的背书，拿到汇聚愿力的抓手，进而获得权柄，那跟直接选举比起来，多了一道中间的手续，所以，不管是政客的竞选演讲，还是陈胜老师的口技表演，最终的目的是能够获得认同，把人民的愿力与自身做出强绑定，鬼故事只是表皮，内核其实跟现代是一样的，可以把愿力理解成一种特殊的选票，他们在拉票，所以一直以来，搞事情的人都会讲究一个大义名分，名分就是汇聚愿力的抓手，这一点，对创业非常的重要，尤其是不带任何资源的冷启动。

那写到这里，可能有小伙伴就会说，那不是一直都在讲，天子兵强马壮者当为之，喊得再响又有什么用呢，那这么说也没错，但是，历代那些世家大族或者实力派，比如说跟上面两位同时期的项羽，搞事情的时候，就不会那么多神神叨叨，上来直接就是干，那因为都兵强马壮了，都准备IPO了，那肯定不用考虑什么A轮、天使轮的融资问题了，我们在说的是创业初期。

那什么最容易汇聚愿力呢，那肯定是宗教，包括愿力这个词就是个宗教用语，这也就是为什么，历史上打着宗教名头创业的特别多，因为宗教天然汇聚了这些信徒的愿力，而且，具备相当的组织度，成本低、启动快，用过的都说好，当然了，这也是历朝历代朝廷，都要对宗教组织严加控制，或者说把宗教领袖纳入体制的原因，好了，讲完这些，我们再回到海岛上，现在我们要回到第一个问题，凭什么，那不是说自己需要从兜里掏出个王母娘娘发的毕业证，还是那句话，表象什么的都不重要，关键是怎么样把大家的愿望与我们个人相绑定，人民相信神仙，你就可以掏出太上老君盖的章，人民期待救世主，你应该扮演这个时代的红太阳和慈父，那么，当下岛上的人们期待什么呢，这很简单啊，劫后余生，每个人都希望能在这座岛上活下去，好好地活下去，所以，我们现在要做的第一步事情，就是找到自己身上可以、愿意强制绑定的优势点，包括但不限于，你有专业技能，你可以捕鱼、打猎、造房子，你肚子里能装着各种各样生存的知识，那你有发达的肌肉，一个人干好几个人的活，你的口才非常了得，能跟希特勒似的鼓动大家，或者实在不行了，就拉拉关系，搞搞小团体，哪怕，你只会用鼻子吹鼻涕泡，只要有本事让大家相信，你吹出来的鼻涕泡，对未来的生存至关重要，那都是可行的，那到了这里，我们总得给自己安个什么能力吧，才能继续往下讲，那太强，那肯定不行了，什么野外生存点满呐，什么打猎、盖房子样样都会啊，那写起来太复杂，关键是我自己也不懂啊，那至于性格魅力，组织能力啊，同上，而且呢，我们也不是要写小说啊，身强力壮，倒是可以，不过呢，最好再稍微弱化一点，能再具象化一点，那想了一想，就让系统给发个道具吧，那比如说发条打狗棍，那这根棍子的作用，是单挑有优势，打两个人以上，就歇菜，那就这么个小道具吧，是我们唯一的优势，那接下来我们是不是要拎着棍子，天天找人单挑，一个一个把人打服了吗，那如果是这样的话，是看不到第二天太阳的，那甚至，可以这么说，等你将来登上皇位的那一天，看着手里的棍子，突然发现，原来这么多年了，这根棍子一次都没有用过，当然，这是后话了，好，拿到道具以后，接下来就需要表态了。

那我们拎着棍子，就是为了大家服务的，怎么服务呢，为了活下去，那必须得分工吧，那既然要分工，起码有一套分配制度吧，这也是最基本的规矩，但是，这么大个群体总会有些矛盾，有些争执，万一有谁心术不正，想多吃多占呢，这个时候，就需要有一个人来调解矛盾，维持秩序，而，就勉为其难干这个得罪人的事情吧，总之，你得让大家相信，这根棍子在你手里是永远不会威胁到他们，不但不会威胁到他们，还能保护每一个人，是大家活下去的重要保障，那这是真的吗，当然肯定不是了，那既然是假话，那就更需要我们把它重复1000遍，开始，肯定没有什么人会完全相信，坚持下去，大家会从完全不信到将信将疑，再到相当的一部分人完全相信，你说这么干有什么好处呢，那规矩嘛，肯定是大家定的，我们就是跑个腿而已，那这就想简单了，这个执行的文章就深了，这中间，还有一个自由裁量权，那这个以后再说，到了这一步，就拿到了至关重要的执法权，现在你的计划，终于完成了热身的准备了，下一步，就像我们前面说的，拿到大义名份，就像创业初期拿到了天使轮融资，下一步该干什么呢，是不是该打造产品了，那么我们就需要开始用手头的资源，打造帝王树的第一款产品了。

关于权力，教科书式的定义是：权力是影响他人行为或意志的能力，这个描述，确实很惊艳，但是过于学术，对于我们理解权力的本质并没有什么用，那也有人说，权力即是给予和剥夺，手握奖励和惩罚，自然就拥有了权力，那这个解释倒也没有错，就是多少流于表面了，那现在需要我们在小岛，构建自己的权力体系，那么解构权力的本质就是不可回避的问题，在这个过程中，我们会一边分析权力的逻辑，一边完成岛上帝王的基础搭建，上文写到，你拎着打狗棍，开始为人民服务了，接下来，就需要打造第一批，服务于人民的产品了。

具体怎么做呢，最简单的，就是谁不听话，咱就拿打狗棍揍他，谁反对我们多拿多占，揍他，谁反对我们霸占小姑娘，还是揍他，这么一通操作下拉，不用一个月，不用一个星期，当天晚上我们就得凉凉，你永远永远不能仅仅依靠打狗棍，也就是暴力，还是最初级的暴力去建立统治，这是自寻死路，先不说在背景设定里，咱们压根就没有碾压性的力量，就算有这样的统治，也是让人无法忍受的，那在这样的统治下，你必须时刻让暴力机器保持全天候，满功率的运转，去浇灭那些野火烧不尽的反抗，直到暴力机器崩解的那一天，革命的大火会把你和你的统治烧成灰烬，那暴力重要吗，重要，那就像《血酬定律》里说的，暴力是元规则，是一切规则的规则，那在大众的印象里，暴力也天然和权力相关，那权力和暴力之间的关系到底是什么，比如说，电影《第一滴血》里的RAMBLE暴力到极点了吧，他除了能杀杀杀，半点权力也没有，但那反过来说，你什么力量也没有，就是知道怎么领着村里的狗剩和二蛋，去隔壁村偷苞谷，这个时候你反而有权力了，你可以先指使狗剩去放风，命令二蛋先去探路，最后偷来的苞谷，你还得拿大头，不然下回不给他们去了，那从这两个事情我们可以看出来，所谓权力往往是为了某个目标，比如说偷苞谷，然后，围绕这个目标建立起某种秩序，随着秩序的建立，等级也就出现，那权力，也就由此产生了，总结一句话，就是秩序中诞生权力，权力依附于秩序，暴力的重要之处在于，暴力可以推翻旧秩序，同时为新秩序保驾护航，光有暴力，没有秩序就会像RAMBLE那样，杀起来是挺过瘾啊，但是永远跟权力不沾边，暴力可以让你推翻一个旧世界，而权力取决于你能不能建立一个新的世界。
