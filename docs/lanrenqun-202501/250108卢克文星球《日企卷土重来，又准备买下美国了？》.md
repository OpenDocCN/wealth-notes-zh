

# 日企卷土重来，又准备买下美国了？

250108 文/卢克文工作室嘉宾 小小罗
整理：公众号懒人搜索，懒人专属群独享
懒人微信：lazyhelper

![](img/d1f1a3b6f6c6e68a9762d7cab6ae62f7_0_0.png)

当地时间1月3日，美国总统拜登发表正式声明宣布，他已签署总统令决定，禁止日本制铁公司对美国钢铁公司的收购，理由还是老一套的“威胁国家安全”。

但日铁董事长桥本英二今天已经宣布，将以拜登为主要对象提起诉讼，显然是不打算放弃收购，莫非在泡沫经济被戳穿几十年后的今天，日企卷土重来，又准备买下美国了？

2023年12月，日本制铁公司向有120年历史的美国钢铁公司发出了收购要约，价格高达149亿美元。

收购这种老牌美国“央企”难度可想而知，但日本人志在必得。

美铁公司是美国重要的高炉工艺钢、汽车板、电工钢生产企业，拥有全球先进的小型电弧炉工厂以及在北美自供的铁矿山开采区等等，目前产量约1575万吨/年，规模仅排名全球第24位，在美国国内也仅列第三。

如果收购成功，日铁有望一举跃升为全球第三大的钢铁公司。

长期以来，由于人力成本高昂及技术落后等叠加影响，美铁的经营境况十分惨淡。

相关数据显示，近十年间公司业绩持续走低，净利润一直围着盈亏平衡点打转。2023年，美钢营收较2021年的204亿美元下降至180亿美元，净利润则从41亿美元狂跌至9亿美元，其2023年的粗钢产量甚至不及宝武钢铁的零头。

要救活这样在国内国际市场上严重缺乏生存能力的企业，唯一的办法就是向外部寻求投资。

日铁愿意出149亿美元，美国钢铁公司的股东也很高兴，自己快撑不住了，居然还能卖出这么好的价钱，简直天上掉馅饼，去年4月召集的临时股东大会更是以98%的支持率通过了日铁的收购方案。

日铁收购美铁还有一个大背景。2021年拜登政府推出了价值1万亿元的《两党基础设施建设法案》，这是50多年以来美国最大的基础设施法案，称要在五年内新增约5500亿美元投资用于交通基础设施。

日铁这次的收购，也是押注美国内钢铁市场有望趁机实现复苏。

而且面对美国持续至今的针对进口钢铁征收25%关税的保护主义政策，日铁收购成功后可直接继承美钢的本地工厂并扩大在美生产，顺理成章地规避关税大棒，长期利益相当可观。

但是日铁收购美铁的过程却从一开始就遇到了极大的阻力。

第一道难关就是美国钢铁工人联合会的反对。由于全球钢铁行业不断洗牌，多家美本土钢铁工厂近年来关闭或向低成本国家转移产能，导致工人失业危机频发。在此背景下，美国钢铁工人联合会对外资介入已难存信任。

而且美铁公司所在的宾州是有19张选举人票的关键摇摆州之一。

去年大选，民主党在几个摇摆州全军覆没，如果这次继续再得罪钢铁工人联合会，对民主党来说是灾难性的后果，为稳妥起见，拜登还是否决了并购案。

第二道难关便是民主共和两党高层。早前收购案公布之后，特朗普、拜登、哈里斯等人都明确表示反对收购计划，原因不外乎为了争取宾州的庞大工会群体选票以巩固票仓。

为了化解美国内部的反对声浪，日铁公司采取了积极行动。除了承诺保障工作机会之外，也宣布将对员工发放总计2000万美元奖金以及投资近14亿美元翻新扩建老旧厂房、承诺不裁员也不关厂等举措。

去年9月，岸田文雄访美，直接向拜登提出了希望美国政府支持日本制铁收购案的要求。而石破茂上台后，也于去年11月致函拜登，希望批准这起收购案。尽管日本拿出了如此大的诚意，但拜登还是不给面子。

由此可见，再要好的盟友，涉及到核心利益，照样难逃欲加之罪。

日铁作为日本第一大钢铁企业，涉及的又是针对美国上百亿美元的收购案，背后肯定有日本政府的撑腰和力挺，而且此桩收购案约定了近5.6亿美元的天价违约金，所以日铁绝不甘心赔了夫人又折兵。既然美国人先不讲武德，日铁也就有恃无恐，决心起诉美国政府。

其实，无论日铁与美铁双方，还是特朗普与拜登双方，对待这次的收购方案都是基于“对抗中国”的思路走的。

日铁这次执意收购美铁，多少有点像是不甘心的无奈之举。

想当年，日铁按照中日两国领导人达成的协议，出大力气帮助宝钢的早期建设，但到了1996年，中国的粗钢产量就超过了日本。进入2000年之后，中国钢铁产能更是爆发式增长，把日本制铁远远甩在后面。

弹指一挥间，曾经的学生成长为最有力的竞争对手，日铁的心态五味杂陈。

去年7月23日，日本制铁宣布撤出与中国宝武钢铁集团旗下的合资业务——宝钢日铁汽车板有限公司，将自身50%的股权卖给宝钢股份。

日铁撤资的消息震惊了钢铁业界，标志着双方长达半世纪的合作出现重大转折，一时间引发众多猜测。其实，这份合资合同是属于正常合作到期（20年），只是日铁方面选择了不再续约而已。

不续约的原因有两个：
- 一是日铁方面坚持认为宝钢在2020年左右开始向丰田供应的无取向电工钢的产品技术侵犯了他们的高功能钢材专利，双方一度打起了官司，公开决裂。
- 二是因为近些年来，日铁的主要客户——丰田、日产和本田等日本汽车制造商在华销售量逐年下滑，日铁也认为未来难以实现业务增长。

于是日铁积极配合美国产业链供应链“去中国化”战略，主动收缩了在中国的产能，把目光瞄准全球老二印度和全球老四美国，打算通过兼并联合做大做强，重新拿回市场地位以对抗中国。

而拜登和特朗普双方的意见都是要保护美国的钢铁行业。中国的钢铁产量太大，而价格太低，如果美铁公司不在自己手里，美国的钢铁行业可能要全面崩溃了。

美国钢铁行业的衰落源自于对钢铁行业实施长期的关税贸易保护，以至于在过去很长时间内无法与世界先进技术竞争，导致钢铁价格飙升，国内需求骤降，盈利能力大减。包含美国在内的整个北美洲，消费能力不到全球产能的一成，最终迫使美铁公司走上了被收购的道路。

而日本这边呢，随着特朗普再度执政，中美关系变数犹存，却也间接影响了中日关系的走向。

石破茂政府不甘心坐以待毙，在外交政策导向上对华释放出了史无前例的友好信号。在2024年岁末之际，两国关系改善迹象愈发显著，中日关系熬过了寒冬，迎来近20年最佳窗口时期。

在百年未有之大变局的影响下，一向慕强的日本人终于开始认清了自己的定位。毕竟对于日本来说，既然要重视中国，那么不能时刻都看美国的脸色，中日的未来合作，更不能任由美国指手画脚，要做到让中方放心。

但想要进一步合作（解禁水产品进口），石破茂就必须拿出务实的动作来展示姿态，新年第一枪对准美国成为了访华前向中国递交的投名状。

几十年前是一个强势的日本被更强势的美国给拍回去了，现在是一个疲软的日本被另一个疲软的美国给怼回去了。可以看到不论美日有多么联系紧密，本质也不是穿一条裤子，不论美国强盛还是疲软，都不会允许日本染指一二。

只有独立自主，走出自己的路，才是真正的立国之道。
