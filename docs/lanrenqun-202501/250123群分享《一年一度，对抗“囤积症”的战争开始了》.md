

# 一年一度，对抗“囤积症”的战争开始了

250123

整理：公众号懒人搜索，懒人专属群独享

懒人微信：lazyhelper

![](img/dac4231d301ad8fe0a64009a355a8ba9_0_0.png)

眼看着快到春节了，按照咱们的传统习俗，腊月二十四这天是做大扫除的日子，所谓“二十四，扫尘日”。尘土的尘和陈旧的陈同音，所以，新春扫尘，有“除陈布新”的含义，意思是要把一切“穷运”“晦气”都扫出门。

趁着岁末大扫除，咱们今天就聊聊断舍离，以及它的反面，囤积症。

咱们先从囤积症说起。本来，在家里囤东西不是什么大事。但问题是，囤积这个现象发展到极致，可能就会影响到家庭里的正常生活。

B 站上有个 UP 主叫马俐管家，是一家专门做垃圾房清理的家政公司，发布的视频都是严重的垃圾房。其中有一期被网友称作是“年度最炸裂视频”，是一位老人独居了几十年的家。家里囤积了大量的垃圾，堆满腐烂的过期食物，甚至宠物的粪便和尸体，地面上还有三层污垢。这些混合起来的难闻气味引来全小区的投诉。

子女要清理，老人还不乐意，最后是费了大劲说服，老人才同意找人上门清理。

这种严重的过度囤积行为，显然已经不是正常举动。事实上，这确实是种精神障碍，学名叫“囤积障碍”，也叫囤积症。2013 年的第 5 版《精神疾病诊断与统计手册》里，把囤积障碍列为一种独立的精神疾病。《默沙东诊疗手册》里对囤积症的描述是，患者在舍弃自己的所有物方面长久存在困难，以至于物品积攒，并凌乱地堆满生活区域，使得生活区域无法使用。

这样的情况其实并不少见。根据统计，全世界有 2%—3% 的人受囤积症困扰，并且主要以中老年人为主。在 55 岁以上的人群中，囤积症的患病率超过 6%，比例比其他年龄的人群要高得多。

虽然囤积症人群中老人居多，但是很多人从青少年时就会表现出轻微的囤积趋向，比如扔东西的时候很难下决心，或者总是拿不准应该把东西放在哪儿。

囤积症的病因现在还没完全明确。不过，有一些心理学研究者发现，囤积症患者往往有一些特殊经历，比如经历过童年创伤，或者是突发的压力事件。说白了就是受过严重的刺激。

比如，2005 年，艺术家宋冬做过一个装置艺术展览《物尽其用》，展品就是，他的父亲去世后，母亲囤积在家里的上万件日用杂物。宋冬母亲家的日历停留在父亲去世的那一天，这天之后他的母亲就开始大量囤积物品，直到杂物塞满一个两室一厅的家。同样，她也拒绝其他家人扔掉这些东西。

再比如，日本综艺《可以去你家吗》，节目组跟随一名女生探访她的家。女生自己打扮非常精致，但家里堆满垃圾和杂物，进门只能在垃圾上行走，只有床铺还算干净。为什么她的生活会变成这样？原来她经历过一次非常严重的创伤。她曾经有一个非常相爱的男友，但准备结婚的时候才发现对方是自己从小分离的双胞胎哥哥，最后，男友受不了伦理的束缚自杀身亡，在那之后，她就再也没有清理过自己的屋子。

那么，对于有严重囤积障碍的人来说，怎么帮助缓解这个症状呢？

帮他们拓展存储空间，或者他们做一次大扫除，只能算治标，不能彻底解决问题。根据《默沙东诊疗手册》，通常的治疗建议有两种。

- 一是认知行为疗法，也就是，帮助患者改变思考角度和行为方式。可以引导患者思考为什么很难扔掉家里的破烂儿，或者说服患者至少别再往家里带新东西。当然，他们很可能不愿意配合，所以还得想点儿激励措施，总之，得让他迈出少囤、多扔这一步。
- 另一种方法，就是药物治疗，现在主要使用到的，是抗抑郁和抗焦虑的药物。

当然，说了这么多囤积障碍，大部分人的囤积行为其实没有到这么严重的地步。很多人，尤其是老人，愿意攒点儿东西，这是惜物的表现。

中国人民大学的哲学教授王小伟写过一本书，叫做《日常的深处》。这本书里就说，老人爱囤东西是一个普遍现象，不光因为这些物代表过去，还因为这些物，指向未来。

你看，对老人来说，这些东西不要扔，是因为以后还能用到。老人在通过这个举动，确认自己依然拥有漫长的未来。当一位老人每次掏出一个藏好的塑料袋去买菜，对他来说，都是兑现了一份曾经期盼的将来。这算得上是一种心灵的疗愈。因此，一些简单的囤点儿东西的行为，也是完全可以理解的。

接下来，咱们再说说，囤积的反面，也就是清理。

有人大量囤积物品，就有人奉行断舍离。这方面存在感最强的应该是日本人。一说到断舍离，很多人就能想到一个画面，一间空空如也的日式白房子，一个人身披禅衣，枯坐其中，身外杂物，越少越好。《断舍离》《告别事物：新日本极简主义》之类的书籍，也全是由日本作家写出来的。

虽然断舍离和极简主义，是现在最流行的日式风格。但真实的日本，也存在另一个切面，它有着疯狂的囤积史。

比如，19 世纪五六十年代，明治维新前后，日本崇尚复杂的艺术和手工艺品。日本人的居所里，通常会摆满各种精美家具、陶瓷装饰、金属工艺摆件，还有各类餐具、小玩意和装饰品，总之是越繁复越好。现在一些复古的日料店里，也还可以看到这种装饰的影子。据说这些物品当时还出口欧洲，在欧洲引发了日本主义热潮，间接影响了很多奢侈品的艺术取向。

再比如，20 世纪八九十年代，日本电子产业鼎盛的时代，整个日本被消费主义席卷。日本家庭中往往会填满大量电子产品和娱乐设施。每个家庭里都有随身听、录像机、游戏机、电视、音响、卡拉 OK 机，有无数玩具、电子游戏，无数卡通和漫画。那时候的日本人买得多、囤得多，扔得也多，经常有人在垃圾场蹲守，等着淘一台功能完好的电视机。这期间很多日本艺术作品也走的是杂乱风，比如《千与千寻》中就有很多餐桌堆满食物、卧室堆满杂物的场景。

再比如，到了今天，日本很多城市的郊区，还存在大量堆满杂物的房子。在一些空间特别有限的地方，杂乱的住宅和商店甚至会垃圾爆发，很多东西从房屋里面倾倒到街上。日本的城市规划师给这种现象起名叫“溢出”，而日本民间还产生了一个讽刺性的词语叫做“垃圾豪宅”。

你看，这就是咱们今天能看到的局面，物品过剩和过度清理，两种主张出现在同一个地方，极简和杂乱，构成同一枚硬币的两面。就像有一名旅居日本的美国作家写的，别被所谓的极简主义骗了。日本大量输出类似的作品和主张，让我们误以为日本就是这样一个极简主义天堂。其实，日本之所以会出现这些书籍，正是因为，日本人是最需要学习怎么处理物品的。假如日本真是一个极简主义天堂，这些人还写书干吗呢？

其实，断舍离主张的兴起，恰好伴随着日本人的一次集体心理转变。日本的经济泡沫破裂后，曾经那种豪气的消费主义也难以为继。这些商品原来代表着经济富裕，现在却成了失控的证据。日本人转而催生出一种物品羞耻，迫切渴望摆脱。

就在这个时候，以《断舍离的艺术》《新断舍离术》为代表的一批书籍面世了。断舍离这个词，来自瑜伽术语，意思是放弃对世俗的执着，日本人用它来鼓励人们积极地丢弃没用的物品，发动对物品的抗争。这多少迎合了日本人当时渴望摆脱物品的心理。

说了这么多，回过头来看，囤积和清理，其实原本就是联系在一起的行为。一方面，我们的生活中确实围绕着大量物品，囤积行为甚至从线下蔓延到线上，数字囤积也是一种囤积，购物车、收藏夹，和真实的衣柜、书桌一样需要清理。而另一面，在一个同质化越来越严重的社会里，杂乱和丰富，又会形成另一种个性。

而最终人们需要关注的，可能不是物品的多与少，而是怎样对待身边的物品。借用王小伟老师的话说，生活需要细节，而不是罗列几条梗概。人们所感觉到的幸福，通常是在无关紧要的细节中酝酿起来的。
