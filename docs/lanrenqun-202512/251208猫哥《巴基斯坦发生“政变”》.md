

# 巴基斯坦发生“政变”

251207 猫哥

整理:公众号懒人搜索,懒人专属群独享懒人微信:lazyhelper

![](img/85ff4d8efef576941f33c1ce301e9359_0_0.png)

近期，鲜有人注意到，巴基斯坦发生了一场“政变”，巴基斯坦军方的权力大幅增加，再次站到了台前，这对中国来说不是好事。

那么，巴基斯坦好好的，怎么就“政变”了？

2025年11月13日，巴基斯坦议会通过了“第27次宪法修正案”，总统扎尔达里光速签署，修正案立刻生效，巴基斯坦舆论马上炸了。

“第27次宪法修正案”不是单一条款，而是许多个条款打包起来的组合。

它的主要内容包括：

- 将最高法院的宪法审查权剥离，移交给新成立的宪法法院，组建由总理控制的司法遴选委员会，负责选拔宪法法院的法官。
- 废除“参谋长联合会议主席”一职，新设立“国防军总司令”，作为海、陆、空三军最高领导者，由陆军参谋长兼任。
- 赋予军队元帅终身司法豁免权。

这是军方和文官政府的一次利益交换，文官政府和军方都扩大了权力。细究起来，还是军方好处拿得更多，尤其是陆军元帅穆尼尔。

先看第二条，由陆军参谋长兼任“国防军总司令”，统领海、陆、空三军。作为现任陆军参谋长，穆尼尔因此一跃成为三军总司令。

第三条，“赋予军队元帅终身司法豁免权”

穆尼尔不仅是元帅，还是唯一的现役元帅，是该条款的唯一受益人。

说难听点，本次修宪，就是为军方乃至穆尼尔本人，量身定做的。文官政府看似扩大了权力，其实只是跟在军方后面吃了点残羹剩饭。

因为修宪过后，穆尼尔成了军队总司令，还负责统领海、陆、空三军，意味着军权，落到了他手里，文官政府手中，不再有军权。

巴基斯坦舆论担忧，穆尼尔要成立军政府。

这种担忧并非没有道理，1947年建国以来，巴基斯坦长期处于军政府统治之下。

1958年，军头阿尤布·汗发动政变，推翻文官政府，首次建立军政府。

1971年，因丢失孟加拉，军政府颜面扫地，只得将政权归还给文官。

但六年后的1977年，军方卷土重来，陆军参谋长齐亚·哈克发动政变，推翻了阿里·布托领导的文官政府，并处死阿里·布托。

11年后，齐亚·哈克死于一场离奇的事故，群龙无首的军方被迫再次还政。

1990年，按耐不住的陆军参谋长穆沙拉夫，又又又一次发动政变，第三次建立军政府。直到2008年，军政府面对铺天盖地的批评，不得已还政。

从历史可以看出，巴基斯坦由文官政府稳定治理的时间，还不到20年，而军政府的时间，要长得多。即便在过去十多年的文官政府里，军方仍然扮演着重要角色。

巴基斯坦国内发生的很多大事，都有军方的影子。比如前总理伊姆兰·汗被推翻，外界普遍认为，与军方的默许，脱不了干系。

因为伊姆兰·汗与军方的关系不怎么好。

巴基斯坦人对军政府统治印象深刻：当修宪通过时，当穆尼尔成为修宪的最大受益人时，舆论难免陷入担忧，他到底想干嘛？是不是要再次建立军政府？

穆尼尔敢突破界限，原因有两点，一是巴基斯坦社会对文官政府的不满。

首先是经济。

疫情后，巴基斯坦经济濒临崩溃，长期靠向 IMF 借高利贷艰难度日，但借高利贷的代价不小，比如降本增效、提高国内电费价格等等。

总之，IMF 的要求很多。

目前，巴基斯坦许多地方别说经济发展，连基本的电力供应都成问题。

## 其次是安全形势堪忧。

稍微关注巴基斯坦新闻的朋友，都知道巴基斯坦的安全局势非常不稳，每天不是这里发生了恐怖袭击，就是那里出现了部落冲突。

根据“伊斯兰堡智库”的统计，2023年，巴基斯坦国内的恐袭事件，共造成 1500 多人死亡，到了 2024 年，死亡人数变成了 2500 多人。

短短一年时间，死亡人数大幅增长。可见，巴基斯坦的安全形势有多严峻。

经济崩溃和安全形势堪忧，直接导致民众极其不满，觉得文官政府是一群饭桶。总理都换了多少个了，选举都选了多少次，有用么？

这种情况下，即便军方掌权，相当一部分巴国民众，会持默许态度。

他们的心理是：文官政府都那么烂了，换个统治者也不会比文官更烂。

至于民主、自由，不是不在乎，但对经济和安全的需求更为迫切。

## 第二，是“5·7空战”的影响。

穆尼尔是“5·7空战”的指挥者，7比0大胜印度后，穆尼尔的声望暴涨。随之而来的，是特朗普的拉拢。

6月中旬，穆尼尔受邀访问白宫，与特朗普共进午餐，这是一个明确的信号。

正常情况下，外交接待的惯例是，文官接待文官，武官接待武官。特朗普一个文官，接待穆尼尔这个武官，还是在没有巴基斯坦总理陪同的情况下。

8月，穆尼尔再次受邀访问白宫。

双方随后达成了合作，巴基斯坦向美国出口石油（其实是和特朗普家族合作，一起走私伊朗石油），换取美国将关税降到19%。

9月25日，巴基斯坦总理夏巴兹·谢里夫，与穆尼尔一同访问美国，并在白宫与美国总统特朗普举行了时长约80分钟的闭门会谈。

会谈内容，涉及军事、经贸、能源、核能、矿产资源、航空航天等各个方面。

按理说，穆尼尔作为军方代表，只参加涉及军事的会谈，就足够了，其他的留给总理。

但事实上，穆尼尔全程参与。

短短四个月，穆尼尔受邀访问白宫三次，身为总理的夏巴兹·谢里夫仅访问白宫一次。

特朗普的意思十分明确，只要能合作，他不在乎对面是文官还是武官。巧了，历史上，每一次巴基斯坦军方政变成功，第一件事都是寻求美国认可。

有了特朗普背书，穆尼尔的胆子越来越壮，于是有了这次争议极大的修宪。

巴基斯坦军方，从幕后走到了台前。当然，穆尼尔吸取了历届军政府的教训，名义上保留了文官政府，因为赤裸裸的军政府会引发巨大反对，还意味着巨大的成本。

所以，他在获得好处的同时，尽量避免增加成本。

只要我不发动政变，你们就没有理由指责我是军政府，但明眼人都看得出来，这是一场“软政变”。

谁当巴基斯坦老大，我们无所谓，我们要的是一个稳定的巴基斯坦，以牵制印度。但穆尼尔这么搞，很容易让国家陷入动荡。

他现在不迈出最后一步，谁能保证将来不会？又或者他的继任者不会？所以，巴基斯坦修宪，绝不是好消息。
