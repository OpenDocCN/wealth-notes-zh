

# 你的不开心，来自对“世界善意”的低估

251202

今天我们要说一份关于情绪的报告，叫《2025 世界情绪健康报告》。这份报告是统计机构盖洛普，以及世界卫生峰会在 11 月联合发布的。

一年当中，关于情绪的调研报告有很多，那么，这份报告有什么特别呢？主要就在于，它涵盖的样本数足够多。今年的报告，一共涵盖了全世界 144 个国家和地区，样本数在 14.5 万人左右。

换句话说，你可以把这份报告当成是一个观察世界总体情绪的参考资料。我们能在一定程度上看到，全世界不同国家、不同年龄、不同身份的人，在情绪方面到底有哪些共性。我们正好也可以趁机看看，自己过去一年的情绪坐标，到底处在哪个水平？是比大多数人开心？还是比多数人焦虑？

以及要提醒一句，这些结论和数据，仅供参考，它只代表盖洛普这一家机构在宏观层面上的观察，不代表每个个体的具体感受。

好，咱们正式开始。

在今年的报告里，有两组最核心的结论。

## 第一，是关于全世界的情绪总基调。可以用一句话概括，人类这个物种的情绪，从未如此鲜明而丰富。

注意，是丰富。什么意思？简单说，就是人类的“开心程度”和“不开心程度”，都同时增加了，而且达到了近年来的最高值。一天当中，开心的时候变多，不开心的时候也变多了，糟心的事多了，但幸福的事也多了。就像坐过山车一样，波峰和波谷在同时增加。

首先，我们先看“不开心”的一面。

### 报告里有句话是这么说的，叫，人类已经走进一个负面情绪基线被整体抬高的时代。

负面情绪，整体抬高。什么意思？简单说，就是“不开心”这个事，变得越来越容易了。就好比，以前我们得遭受一记重拳才会难过，但现在呢，只需要轻轻推两下，就会心里堵得慌。

根据报告里的数据，2024年，全世界有39%的人表示，自己一天内，出现过担忧情绪。什么概念？大概就相当于，你在大街上找10个人，问他们：“你最近一天有什么不开心的事吗？”然后4个人会告诉你说，“有，我有不开心的事”。

同时，还有37%的人感觉到有压力，32%的人在情绪问题的同时伴有身体疼痛，26%的人感到过悲伤，22%的人感到过愤怒。

注意，这个“不开心”的趋势，不是今天才出现的。根据盖洛普的分析，从2007年以来，全球的“负面体验指数”就在一路上升。2020年疫情暴发时，达到最高位。2023年小幅回落，现在又开始抬头上升。

但是，好消息在于，人类总体的“开心时刻”，也在明显增加。

在盖洛普的调研里，有88%的人表示，自己过去24个小时，感受到了来自他人的尊重。这个数据是盖洛普有记录以来的最高水平。同时，有73%的人说自己“昨天笑了很多次”，73%的人觉得“有些时刻很享受”，52%的人，在一天内做过自己觉得有趣的事，或者学习过一些新东西。

听到这，有人可能会说，既然我们一边“更开心”，一边又“更不开心”。这两个情绪会不会相互抵消，等于什么都没发生呢？

并不会。根据盖洛普的观察，当正面情绪和负面情绪打架时，短期看，负面情绪占上风，但长期看，赢到最后的往往是正面情绪。说白了，负面情绪是来得快，去得也快。虽然来势汹汹，但持续不了多久。但相比之下，正面情绪的影响要持久得多。

借用报告里的一个观点，“正面情绪就像是你心里的底层资产，即使面对外部压力，我们依然能从日常的人际互动中持续获得快乐”。

这就是报告的第一个结论，过去一年，人类的“开心时刻”和“不开心时刻”，都在变多。但好消息是，当二者相遇时，“开心时刻”更有耐力，它总能赢到最后。

说到这，我想插句题外话。之前有学者在美国的中学里做过一个调研，让学生们选出，这个学校里最“受欢迎”的人和最“不受欢迎”的人。按照通常的设想，这两伙人一定是脾气秉性完全不同的两类人。但结果呢？学生们选出的最“受欢迎”和最“不受欢迎”的，居然是同一拨人。最招人喜欢的是他们几个，最讨厌的也是他们几个。

这个结论其实不难理解。你看，受欢迎的人，往往是存在感很强的人。可能是啦啦队队长，可能是橄榄球队员，同学们都很羡慕他们。但问题是，羡慕和嫉妒，有时候就是一线之隔。而且越出挑的人，遭受的嫉妒就越深。

换句话说，过去我们觉得，“欢迎”的反面是“讨厌”。但其实并不是。它的反义词应该是“被忽视”。因为喜欢也好，讨厌也罢，前提都是“重视”。假如被忽视，哪里还谈得上喜欢还是讨厌？

同样，回到这份报告。我们或许也可以说，“开心”的反义词，不是“不开心”，而是“麻木”。

你看，开心也好，不开心也罢，都说明你对这个世界依然有足够的感受力。有感受就有思考，有思考才有行动。它们总归都要好过“麻木”。你看，形容一个人最糟糕的状态，我们不会说“悲伤欲绝”，而是会说“行尸走肉”。不怕你对这个世界“不喜欢”，就怕你对它“没感觉”。

从这个角度看，我们也许能从报告里读出一个总体积极的信息，人类这个物种正在变得“远离麻木”。

这是报告里的第一个重点，关于人类的总体情绪。

第二个重点要更具体一些，是关于不同人群的情绪体验。也就是，什么人更快乐？什么人更悲伤？

首先，从性别角度看。女性的负面情绪指数，要普遍比男性稍微高一点。比如，担忧情绪出现的比例，男性是37%，女性是41%。悲伤情绪的比例，男性是24%，女性是28%。愤怒情绪的比例，男性是20%，女性是24%。

注意，尽管女性的负面情绪比例更高，但她们的“生活评价指数”也更高，比男性大概要高出2个百分点。换句话说，我们大概可以认为，尽管女性有着更多的“不开心时刻”，但这并不影响她们更加“热爱生活”。

其次，从国家的层面看。简单说，国家越弱，环境越乱，国民越是“不幸福”。这和我们的常识完全一致。比如，2024年，负面情绪占比最多的国家，有非洲的乍得、约旦、塞拉利昂、几内亚等。这些国家普遍处在战争、动荡，或者政治危机当中。

还有，从年龄的角度看。每个年龄有每个年龄的烦恼。15到49岁的男性，最容易感到愤怒。30到49岁的人，感觉生活压力最大。50岁以上的人，更容易感到悲伤。简单说，就是年轻人更易怒，中年人压力更大，上年纪的人更容易悲伤。这也和我们的通常认知相近，因此就不展开说了。

好，关于盖洛普的《2025世界情绪报告》，我们先说到这。

但是，注意，今天的内容还没结束。最后，借着这个话题，我还想跟你说一个，大概率上会让你觉得开心的事。来自今年上半年，盖洛普与牛津大学福祉研究中心联合发布的另一份报告，叫《2025 年世界幸福报告》。这份报告里有一个非常有趣的研究。说的是，**我们总是低估，来自他人的善意。说白了，周围的人，要比我们想象得更加善良。**

这项研究有 5000 人参与。具体过程是，让参与者针对两个问题作出判断，而且要事先承诺，必须诚实作答。

第一个问题是，你觉得他人有多大几率，去愿意帮助那些“不开心”的人？第二个问题是，你自己有多大几率，去帮助那些“不开心”的人？

你看，假如人们对他人的判断是准确的，那么这两个问题的答案应该差不多。但实际上，第一个问题，志愿者认为“他人会伸出援手”的概率，是 23.9%。而第二个问题，认为“自己会伸出援手”的概率，是 95%。

注意，23.9%和 95%，这两个数差的不是一丁半点。即使参与者有私心，想把自己说得善良一点，但这么大的数值差距，依然值得重视。

哪怕这个数字不能代表实际情况，哪怕有一些人并不会真的帮助别人。即便如此，**我们至少也可以说，每个人内心“想”帮助他人的意愿，要比想象中高得多。**

注意，别小看这个心态问题，因为它会直接导向行动。你看，很多人为什么不爱跟人打交道？因为他们觉得别人是“不友善”的。当一个人不相信别人会善待自己时，就会减少社交，就会拒绝自我暴露。而一个人的社交越少，对周边真实的人的感受就越少，他们就可能会进一步加深对周围人的误解。久而久之，这就成了恶性循环。

说白了，很多时候，不是这个世界不够友善，而是人的悲观认知，把自己推向了真正的孤独。

从这个角度看，主动释放一点善意，主动联系许久未见的朋友，主动对周边世界多一些互动，也许都是打开“积极情绪循环”的一把钥匙。

换句话说，很多时候，所谓的不开心，只是来自对这个世界善意的低估。
