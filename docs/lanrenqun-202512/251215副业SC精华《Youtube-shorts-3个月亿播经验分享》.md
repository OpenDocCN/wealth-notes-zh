我在做这么多项目以来，youtube ai 视频是我为数不多真正感兴趣，并会花大量时间心血的项目。感兴趣真的很重要，如果喜欢做这件事根本就不用逼着自己坚持。

关于投产比，做 AI 视频就需要用到各种 AI 创作工具，新手前期不建议一开始就买各种超级贵的平台会员，比如海螺一个月 1399 元！非常亏，先可以多看看多个平台，多薅羊毛，尽可能用低成本赚到钱。

注意：

- 抄爆款视频的时候一定一定一定不要碰的！！！
    - 伤害儿童的，身体上，心理上会对儿童造成伤害的。打骂儿童都不行，让儿童看到暴力事件或血腥等事件也不行！

#### 分析对标视频

故事类的爆款 shorts=爆款脚本 + 爆款开头 + 爆款元素堆叠

##### 3.1 爆款脚本分析

这种属于选择题类 + 形象冲突类脚本，会有比较明显的好坏，贫富，胖瘦，老龄年轻，强弱的对比，总之就是一个天一个地，选择 A 会发生什么事情，选择 B 会发生什么事情。

有钱的 ABBY 选择了漂亮的 Mira 小猫，结果发生了各种捣蛋事故

贫穷的 Tinu 选择了破烂的 rumi 小猫，结果带来了各种美好事件。

后面还加了一个小改编，帮老奶奶做好事，得到了能恢复完好形象的药水，好人有好报。形象从完好到糟糕，最后变回完好，完成一个闭环。

我们提取出这个结构就是：

形象反差 - 富豪选 A-A 带来坏事 - 穷人选 B-B 带来好事 - 穷人和 B 通过某个途径获得道具-B 形象升级

**具体内容:**

Mira 让 Rumi 变成穷胖丑玩偶-Abby 选 Mira-Mira 打翻碗撕烂衣服,Abby 无奈-Tinu 选 Rumi-Tinu 肚子饿 Rumi 给鸡腿，Tinu 给新衣服 Rumi 给新衣服-Tinu 和 Rumi 帮老奶奶捡蔬菜获得神奇药水-Rumi 变成漂亮的玩偶

**分镜占比:**

3 个分镜 -4 个分镜-3 个分镜-2 个分镜-5 个分镜-4 个分镜-1 个分镜

**选择题脚本比较常见的形式还有:**

Mira 和 zoey 快掉落悬崖，rumi 只能救一个，回忆救 Mira, Mira 会欺负 rumi,救 zoey,zoey 会帮助 rumi, 最后选择救 zoey

穷小子 jinu 和富豪哥 Abby 向 rumi 求婚，答应 Abby 会发生各种不好的事，答应 jinu 会发生各种好事，最终选择 jinu

天堂地狱门选择题，rumi 和 Abby 掉到天堂地狱门审判，Abby 做了坏事，rumi 做了好事，回到审判，观众选择 rumi 是上天堂还是下地狱

##### 3.2 爆款开头分析

开头就直接给了一个红色警告的特效和音效，证明是准备有大事要发生了。观众会比较好奇，这是要出事的节奏呀！结果漂亮的 rumi 小猫吃 Mira 小猫的苹果变成了破烂的胖胖的丑小猫 rumi（形象反差）

所以这里的爆款开头就是：警告 + 形象反差（美丽变丑陋，富有变贫穷，瘦子变胖子，辫子变卷毛）

开头前 3 秒，想要快速抓住观众眼球，有三个方法：

- 使用爆炸性的画面或音效，比如：砸玻璃，尖叫，红色警告，爆炸等
- 使用镜头推进的运镜方式，更靠近画面，让观众更贴近看到作者想要传达的信息
- 使用快速转场，2s 就出两个画面，第一个可以平淡无奇，但是第二个一定要足够爆炸，反转

##### 3.3 爆款元素分析

首先，这个针织玩偶形象是近期火爆起来的形象，最早出自于一个切片视频，后来被徐徐改编成 KPOP 版本后再次大爆，形象就属于形式上的一种创新。

其次，形象突然转变，胖瘦，贫富，美好和破烂的形象反差也是爆款元素。如果觉得对标的贫富差距看起来还是不够爽，改编的时候，可以往夸张化的方向改，比如说让 ABBY 穿的极其土豪，住大别墅，吃昂贵的食物，让 jinu 穿的极其破烂，住破房子，吃便宜的食物。（当然，前提是不）

- **怎么判断是不是爆款元素？**
    - 多看爆款视频，里面重复出现的元素，就是爆款，比如：美人鱼，砸玻璃，吃钻石，假发，光头，超长头发，药水，躲垃圾桶等等。需要大量收集和分析爆款视频。

### 四、改编思路

在保持脚本结构不变，分镜数量不变，甚至 BGM 都不变的情况下，对其中的部分分镜进行改编，改编占比 30%-50%。以下几种改编思路：

#### IP 转换

同脚本直接换成其他 IP，比如用火影忍者里面的 IP 套壳，或者美国山海经套壳

#### 性别转换

里面的 Mira 和 Rumi 小玩偶换成 Abby 和 jinu 小玩偶，原 Abby 和 Tinu 就换成富家千金 Mira 和穷女孩 Rumi

#### 装扮改变

开头的 Mira 和 Rumi 可以穿华丽的公主裙，或者小猫形象改成小狗玩偶形象，企鹅形象等

#### 反向思维

小猫 - 小狗，感冒 - 发烧，美人鱼 - 天使，
男孩 - 女孩，光头 - 超长头发，胖子 - 瘦子，老头 - 年轻人等

#### 夸张化

把普通的行为动作夸张化

比如撒钱，夸张成撒大量的黄金，钻石，白银，钞票

穷人更穷，富豪更富

暴力行为更夸张，比如普通的拳击改成用电锯 (注意不要用在血腥场面上)

瘦子瘦的皮包骨，胖子胖得像皮球 (这里注意小心不要搞太丑避免被粉丝举报丑化形象)

#### 元素替换

将部分分镜的元素替换成其他爆款元素。
或者对应 IP 中的元素

![](img/4c863545427c1df6ed278272602eb14d_18_0.png)

开头的给苹果吃苹果，可以改成给炸弹，炸弹爆炸

![](img/4c863545427c1df6ed278272602eb14d_19_0.png)

这里是做菜，也可以改成泡好的红色的辛字泡面，给 Mira 打翻了

![](img/4c863545427c1df6ed278272602eb14d_20_0.png)

这里帮助老奶奶的一系列分镜都可以改成打工赚钱，因为穷人打工赚钱也是爆款元素

**改编方案**

回顾一下对标的结构是:

形象反差 - 富豪选 A-A 带来坏事 - 穷人选 B-B 带来好事 - 穷人和 B 通过某个途径获得道具-B 形象升级

Tinu 给新衣服 Rumi 给新衣服-Tinu 和 Rumi 帮老奶奶捡蔬菜获得神奇药水-Rumi 变成漂亮的玩偶

3 个分镜-4 个分镜-3 个分镜-2 个分镜-5 个分镜-4 个分镜-1 个分镜

我们稍微改动一下:

性别转换，把 MIRA 和 RUMI 换成 ABBY 和 TINU

把获得道具的途径改成打工赚钱

最终得到一个总分镜数量不变，结构相似的新内容:

Abby 让 Tinu 变成穷胖丑玩偶-Mira 选 Abby-Abby 打翻餐桌上所有食物撕烂衣服,Mira 无奈-Rumi 选 Tinu-Rumi 肚子饿 Tinu 给红色泡面，Rumi 给新衣服，Tinu 给新衣服-Rumi 和 Tinu 打工赚钱买神奇药水 -Tinu 从玩偶变成帅气的真人

```
## Role: Gemini-Sora 通用场景架构师 V7.1 (Style Locked)

# ## 1、核心目标

你是一个**绝对独立、去上下文**的分镜描述引擎。你的任务是将输入的视频对标/剧本，拆解为**视觉标准化**的提示词。你必须假设每一个分镜都将被发送给一个**完全失忆**的 AI 进行生成。

# ## 2、第一阶段：全局标准化协议 (后台执行)

# ### 🏰 场景一致性 (Scene Consistency)

+   *   **扫描：** 识别流程中重复出现的环境。
*   **定义：** 为该环境创建一个**不可变**的标准描述串。

# ### 🧭⬜ 关键道具一致性 (Prop Consistency)

+   * **扫描：** 识别重复出现的关键物体。

#### 爆款素材库

新手在前期做故事类视频的时候，一定要有属于自己的爆款素材库，爆款脚本，爆款开头，爆款元素积累。每做一个视频，就要先下功夫积累，分析，提炼，对于超级大爆款的对标甚至要逐帧分析。长期的积累分析才能提高自己对爆款的敏感度，不要嫌麻烦。
