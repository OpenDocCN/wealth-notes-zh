

# 总要为后人留点什么

251208 大英良心汉弗莱

整理：公众号懒人搜索，懒人专属群独享

懒人微信：lazyhelper

在2020年下半年，首席写过《疫情以后》系列，提出：我们需要一个缓慢衰落的美国，一个过快衰落的美国对我们是不利的。

2025年年底，美国人发表了国防安全战略。在1823年12月2日詹姆斯·门罗正式提出门罗主义以后（核心是反对欧洲列强在美洲殖民、要求欧洲不干涉美洲事务、承诺美国不介入欧洲争端三大原则），200年来又一次提出了门罗政策。

这是美国从战略进攻转入战略相持的标志性事件。

站在这个历史时间点上，我们要看到危机，有危，也有机。

作为共和国来说，我们的责任是确保红色江山不变色。站在此刻，我们有必要想一想，总要为后人留点什么。

以下内容仅“小伙伴”可见。今天讲第一个东西：统一。

## （一）稳，也不能拖

朝鲜战争前的1950年，杜鲁门就试图抛弃台湾以换取与中共某种形式的“联合”，朝鲜战争爆发与中国参战使杜鲁门的设想化为泡影。

40年后的1972年，尼克松为了与中国改善关系以抗衡北方苏联时，也曾有限地抛弃过台湾。

这两次都为日本插手台湾提供了机遇。

又是一个40年过去，在艰难中爬起来的中国人，遇到了美国人打出来的连环进攻：2018年的贸易战、2020年的生化战、2022年开启的扼杀战、2025年再次开启的贸易战。

在这些连环进攻下，我们没有衰落，也没有倒下，反而更加强大。

当然，收缩是为了进攻，进攻的目标目前看还是中国。

坎贝尔代表的深层政府，在2024年就讲的非常明白：中国是美国有史以来面临最大挑战。

但问题在于美国人不会一味进攻，因为当前美国的力量不够支撑它来反复进攻了。

只能是先恢复国力，恢复国力的方式一定是养精蓄锐：统领为代表的这一波人上台，就是为了休养生息，聚集力量，然后唤醒我们国内的第五纵队，进而开展下一步进攻，这种进攻有可能是内部不断的制造话题。

但我们得从这些纷繁的世界里找到主线，尤其是找到我们应该做的工作：那就是统一。

因为，用一些莫须有的东西来扰乱我们的决策是他们最擅长的，就如同里根用“星球大战”将苏联引入军备竞赛的不归之路一样。

AI这些东西也有这个特点。

面对这些东西，我们必须加快军备，同时也要准备谈判，但是以军事压制为主轴，不能有任何放松。

统一对我们来说，时间窗口很有限，如果统领在任内没有办法压制住左派，那么左派上台后，按照他们那个“志大才疏”的尿性，必然又开始干涉，那我们统一的时间又会往后退。

既要稳，也不能拖。

## （二）立，也不怕破

既要敢于破立，也不能害怕破。

对于当前而言，我们的大战略就是敢于破，更要敢于立。

这个破就是：敢于牺牲一部分经济代价，这个立就是：换取我们未来在东亚、东南亚无限广阔的政治战略空前。

在统一这个千年大计面前，我们必须敢于牺牲一部分经济。所谓牺牲，就是面对美国人的贸易战，我们必须打破跟美国人的贸易关系，勇于脱钩，虽然这个脱钩，带来了国内经济相当长时间的低迷，未来可能还有一段时间的低迷。

但是跟我们在东亚、东南亚会取得的未来战略地位比，这些低迷是值得的。

一个非常明显的印证就是，我们在佩洛西来了以后，从2022年开始全面封锁台湾；同样，这一次高市早苗讲话以后，我们将从2025年开始演练如何封锁日本。

很多读者说，首席，你在做梦吗？

首席说过，我们总要为后人留点什么。我们这代人不敢想，那后人会如何看待我们，何况这是我们的根本战略。

从唐朝“新罗有事，就是日本有事”，到明朝“浙江有事，就是日本有事”，到清朝“朝鲜有事，就是日本有事”，2000年来，我们基本上跟这个国家处于你死我活的斗争中，现在有什么不好下手的。

大家可能很难理解什么是战略地位：1972年前后，毛主席利用美国战略收缩让出的新的战略空间，在1971年10月25日恢复了中华人民共和国在联合国组织中的合法权利；迫使美国于1978年最终接受对台“断交、撤军和废约”三原则，并与新中国建立外交关系，从而为后面的40年奠定了坚实基础。

当前，美国人不停的打贸易战，无非就是想扰乱我们的心性。

而我们为什么让李成刚副部长直接刚正面？原因就是告诉美国人，我们不在乎你们在次要战场的任何挑衅，我们的主要方向一直在统一这个主轴上。

而在国内，我们忍受了大量“砖家”和“文人”的批判，坚持打贸易战的根本目的，就是在一定程度上清洗一部分买办（注意是部分）和依赖美国的小资产阶级，这对于我们的统一战线建设是极大的益处。

同时，坚持打贸易战，而不对美国人对我们的实体清单进行大规模反击，还有一个好处就是倒逼国内的产业界提升能力。

## （三）谈，也不怕打

首席曾经讲过一个观点：民进党到底会不会逆转？

什么意思呢？就是会不会出现某一天，民进党突然说，我们跟大陆开始谈判了。这一点是让大家最意外但是确实是意料以内的事？

首席之前做出了判断，短期内不会，但是长期内有可能。

这一点，在目前台湾内部出现了这样的印证，那就是苏起。苏起之所以要说岛内要联合三个党，一起跟大陆谈判，难道真的是为了台湾？

国民党这样一个百年烂党，心里有台湾人民？不是吧，根本目的还是怕民进党甩掉国民党，单独跟大陆谈，这样国民党彻底被干掉。

因为干掉国民党，是民进党和我们的共同想法。

这里首席多讲几句：

应该说，孙中山先生是一个理想主义者，也是资产阶级民主派的主要代表。但是带领队伍还是差点意思。

从国民党成立一开始，在组织上是一个成分非常复杂、组织涣散非常无力的党派，或者叫做松懈的阶级联盟。

它们有五无：无明确的反帝反封建纲领，无革命目标，无组织纪律性，无革命精神、无稳定的民众基础。苏联的近距离观察就非常明显：“国民党中不真心想在中国进行彻底的社会改革的党员太多”。

在这种基因下诞生的国民党，即便退守到台湾，经过了蒋氏父子的改造，仍然是一个没有任何稳定性的松散联盟。

因此指望这样的政党是不可能成事的。而民进党某种程度上，很像 1910 年代左右的中国政党，还有一丝丝血性，在这种情况下，赖清德讲的“大公司、小公司”论，似乎就是内部的某种声音被传递出来。

但是，无论是怎么谈，时间一定不能拖。拖到某个时间点，等日本、美国人的能力恢复起来，再来干涉，这是断然不行的。

而在这种情况下，我们必须坚持三个原则：

- 第一是战争优先的原则，要坚持封锁战、宣传战、远海海战、空战等各种形式全面演习的原则，能打就打。
- 第二是坚持治理同步的原则，要做好成立台湾省委、省政府和台湾各地市政府的准备，为后续的治理做好安排，尤其是要汲取香港的教训，做好一切治理安排。
- 第三是要准备好迎接在统一以后，整体国家再次开放的准备的原则。很多读者说，目前的经济太差了，那是因为我们主动脱钩的结果。

但是一旦统一，我们将迎来更大规模的开放，那个开放，将是在战争结束后的开放，主轴将是在全国的边疆地带，这个边疆将是在海岛一线，特别是在琉球、台湾一线，和东北、内蒙、新疆、西藏一线。

这样内陆，尤其是两湖、四川、安徽、河南、河北、陕西、山西、江西、贵州这个连片的中部地区，它们的压力将会减轻，会有更多的资源投入到发展中。因为当前有很大一部分是在做备份准备。到那个时候除了底 而随之而来的是，我们将向外输出社会主义思想，伴随着社会主义思想的输出，东南亚、日本、韩国将会发生社会变革，这种社会变革为我们输出产能将提供极大的机遇。

## 为什么是琉球呢？

首席讲过，我们还要给我们的生存留一个巨大的出口。如果只是统一台湾，那么我们的出海口宛如一个茶壶的壶嘴。

但是如果让琉球独立，从种子岛到台湾垦丁，长达1500多公里的漫长的深海，都是我们的出海口，这才能让我们的国家安稳。
