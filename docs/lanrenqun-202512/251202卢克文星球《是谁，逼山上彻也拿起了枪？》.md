

# 是谁，逼山上彻也拿起了枪？

251202 文/卢克文工作室 壹壹壹
整理：公众号懒人搜索，懒人专属群独享
懒人微信：lazyhelper

![](img/8bf31bbcbbdcfc54865b5baccabbae50_0_0.png)

2025年11月18日，日本奈良地方法院，一场备受瞩目的审判拉开帷幕。坐在被告席上的，是刺杀前首相安倍晋三的山上彻也。

这次审判最引人注目的，是山上的妹妹作为关键证人，首次出庭作证。在她强行压抑着啜泣声的讲述中，一个日本青年被统一教剥削，家破人亡，愤而向首相复仇的故事，呈现出更为完整、清晰的脉络。

根据山上妹妹的证词，其父亲自杀后，母亲将超过一亿日元家产全部捐给了统一教；长兄因病失明，在贫困与绝望中自杀；山上彻也为了将死亡保险金留给她，也曾试图自杀。

一家五口人，三个人被逼到自杀，就连最阴郁的日本小说，也写不出这么惨绝人寰的悲剧。

而这场罪恶的根源，要从日本经济泡沫破灭说起。

山上彻也从小家庭经济优渥，父亲是京都大学毕业的建筑工程师，母亲是奈良当地一家建筑公司的社长千金。他的命运，本来是考上京都大学，成为一名社会精英，但由于房地产泡沫破裂，父亲工作一落千丈，家庭经济陷入困难，在他年仅4岁时，父亲跳楼自杀，留下6000万日元身故保险金。

母亲深受打击，精神崩溃，“统一教”乘虚而入，母亲渐渐被洗脑了，她加入教会，先后捐献近一亿日元（约466万元人民币），捐到全家破产，逼得成绩优异的山上彻也弃读大学，靠伯父资助才读上了技校。

为了养活家人，山上彻也只好加入自卫队。

2005年大哥癌症复发，妹妹急着要学费，走投无路的山上彻也，试图自杀骗保，想用命换哥哥的医药费和妹妹的学费。

自杀未遂后，他被自卫队开除。

山上的妹妹曾求助儿童咨询所和律师，被告知：“成年人自愿捐款，我们无权干涉。”

大哥曾持刀威胁母亲同归于尽，也冲去教会想找头目算账，结果刚进门就被打了出来。

最终，大哥不堪折磨自杀，妹妹被母亲强行带往韩国“朝圣”，被迫参加统一教的集体配婚，痛苦不堪。

山上彻也，自此走上了复仇之路。

最初，他想杀统一教教母韩鹤子，但找不到下手机会，才将目标锁定为安倍晋三。

从2006年开始，安倍多次为统一教活动站台，教会报纸也明晃晃印着安倍的照片。

2021年，山上在教会视频中看到，卸任后的安倍仍和教会密切来往，产生了强烈的“绝望感和危机感”，认为邪教之所以害死他的家人，就是因为政治大人物的公开背书。

2022年7月，山上通过教团，获取了安倍的行踪信息。2022年7月8日中午，奈良西大寺站前响起两声枪响，尖叫声瞬间炸开，安倍晋三应声倒地，山上彻也平静地放下手中的自制枪支，站在原地等待警察到来。

当警方搜查他13平米的公寓时，发现了令人震惊的证据：购物记录显示，山上彻也跑遍超市、刷遍网站，用六十万元日元前前后后买了八百多个零件。

他本想用炸药，那玩意儿威力大，成功率也高，但转念一想，炸药容易伤及无辜，还是放弃了。

那些日子，他晚上在车库里打磨铁管，周末跑到没人的山里偷偷试枪，一遍遍记录数据，改进他的“作品”。一个被命运逼到墙角的普通人，准备了整整两年时间，完成了一场震惊世界的复仇。

这朵由岸信介亲自种下的恶之花，最终由自己的外孙咽下了苦果。

刺杀案件发生后，统一教的种种恶行，逐渐露出水面，日本民众对山上的情感，从愤怒转向同情。拘留所里的犯人觉得他“替天行道”，还有无数人给他求情，律师团队免费辩护，还有不少人给他送现金，他的妹妹和伯父也收到一大堆慰问信和礼物。

他能得到那么多人的支持，还有一个原因，就是很多人恨死了安倍。

因为安倍所主张的经济学，表面上承诺带领日本走出经济困境，实际却加剧了社会贫富分化。

其核心政策“量化宽松”让日元贬值了不少，丰田这类出口企业赚得盆满钵满。

可企业赚来的钱，要么存起来纹丝不动，要么送到海外投资，就是不愿给本国员工涨工资，也不肯在日本国内投资。与此同时，日元贬值导致进口能源和原材料价格大涨，中小企业和普通家庭压力沉重。

大企业们赚翻了，普通人却要面对牛肉涨价120%、燃气费涨65%的现实。更不公平的是“负利率陷阱”：普通人存100万日元，一年利息才500日元，物价一涨，财富反而缩水；财阀却能用低成本资金投资海外，轻松赚取8%以上的收益。

这种“穷人失血、富人吸血”的循环，让日本基尼系数从1990年的0.24升至2023年的0.38，接近美国水平。

“安倍经济学”让大企业和富人更富了，普通人的生活却越来越难。

所以，不少人一看，被杀的是安倍，多少有点“你小子也有今天”的感觉。

这场仇杀，彻底揭露了邪教长期对日本社会系统性吸血的现实。

统一教对日本社会的毒害，早就不是山上彻也一个人的悲剧了。

官方数据显示，因为这个教派造成的家庭悲剧，光是自杀人数就超过300人，至少2000个家庭被彻底榨干、宣告破产（这个数字，明显低于真实数据）。

统一教的收割手法，靠的是一套精心设计的精神控制加金钱收割。

教会先是给你洗脑，说你祖上犯了罪、家里有诅咒，想化解就得买他们天价的“祈福物品”，一个花瓶就能卖到上亿日元。再不断灌输“这里有爱与和平，只要你行善（就是买统一教生产的高价产品或向教会捐款），人生就会美好”。

更可怕的是精神牢笼，前信徒回忆说，教会要求他们每天汇报思想，不准看外面的书，连给家人打电话都要录音监控。很多人就像山上彻也的母亲，被洗脑到倾家荡产还坚信这是在“赎罪”。

邪教这么乱，政府为什么不管？

这事儿得从二战后说起。

当时美国为了渗透日本，在日本大力推行宗教自由（方便美国宗教势力进入日本），给宗教团体各种优惠政策，结果，宗教团体不仅活动自由，还有免税待遇。

日本《宗教法人法》更是留下漏洞，宗教相关的收入几乎不用交税。

统一教就钻了这个空子，把高价商品包装成“宗教圣品”，把商业交易说成“自愿捐献”，完美避税。

更关键的是，政客们跟这个教会牵扯太深。

当时日本活跃着各种邪教，专门收割陷入精神空虚的日本群众，最活跃的组织之一，就是统一教。

统一教并不是日本本土宗教，而是邪教狂徒文鲜明1954年在韩国创立的。明面上是个宗教，但实际上，文鲜明靠宗教敛财，投资造船、房地产构建商业帝国，再以资金和信徒选票为筹码，换取各国政要保护。

上世纪60年代，为对抗日本共产党，岸信介主动找到统一教出面干脏活，日共候选人在街头演讲，统一教信徒就上去撕标语、打人、捣乱。等警察一来驱散人群，演讲也就黄了。

1970年5月，文鲜明在日本办大型集会，岸信介、佐藤荣作等自民党高层出席，双方正式结盟，合作长达半个多世纪。

一个从韩国漂来的邪教，就这样在日本混得风生水起。自民党里头，跟统一教有关系的议员多达百人。岸信介是统一教徒背后的大金主，后来安倍晋三，不过是接过了外公的政治遗产。

靠着政治保护，统一教在日本风生水起，敛财无数。

《华盛顿邮报》报道，仅上世纪80年代，统一教就从日本转了超8亿美元到美国，至于塞给日本和韩国政客的钱，更是数不清。

作为回报，每次选举，统一教都会号召信徒给指定的候选人投票。

安倍晋三2013年靠统一教帮亲信拉票，2014年又派心腹参加教会活动，卸任后甚至还去当了他们的“特别讲师”。

安倍的接班人高市早苗，更是在统一教旗下媒体活跃了将近二十年，时不时上统一教的媒体节目。

有这样的政治保护伞，谁敢动他们？

山上彻也刺杀安倍两个月后，39个民间团体联合署名，超十万人签名请愿要求取缔统一教。民意汹涌之下，政府终于加快了动作，案发仅五个月，国会就火速通过《统一教被害者救济法》。到2025年3月，法院最终对统一教发出了解散令。

山上彻也的那一枪，真的能打死日本的邪教吗？

不能。

所谓的“解散令”，只是剥去了“统一教”的宗教法人身份和免税资格，就像摘掉了一块招牌。它失去了官方认证，不能再以“宗教法人”的名义享受税收优惠，但法律保障宗教信仰自由，无法将它从物理上彻底消灭。

换个马甲，还可以继续活动，而远在韩国的总部更是毫发无伤。

这样的邪教，日本还有很多。

走在东京街头，你时不时能看见一些举着“我是某某教会受害者”牌子的人，这就是日本社会邪教众多的缩影。

邪教的生命力，远比我们想象的更顽强。它们擅长用狡猾的手段渗透社会：隐瞒真实名称，通过街头问卷、书店搭讪甚至瑜伽课程这类“低威胁”方式接近潜在目标。

一旦建立信任，便逐步诱导对方参加“学习会”，用精心编织的阴谋论和逻辑漩涡完成精神控制。

那为什么总有人上当？

“失去的三十年”，让日本经济逐渐下行，很多日本人过得挺憋屈，职场压力大、社交圈子窄、内心孤独是常态。

每当境遇不佳，人们会倾向于寻求精神慰藉，这是规律。可一看心理咨询的价格，一小时收1.5万日元（约700元人民币），直接劝退。

就在这个节骨眼上，邪教来了。

邪教不用咨询费，0门槛，还精准拿捏了你的这种心理，摆出一副“我懂你”的姿态，防范意识弱的人，很容易就被骗进了它的陷阱里。

像统一教就用“家庭联合”的温暖外壳，吸引渴望联结的年轻人，然后用集体婚礼来绑定忠诚。邪教的高明之处在于，它不是简单地骗你，而是让你心甘情愿地献出一切，许多在社会上得不到认可的人，在这里能找到认可。

这就使得，许多信徒真是心甘情愿的。即便付出生命的代价，有时也唤不醒深陷其中的他们。

整场审判，最触目惊心的是山上彻也妹妹的一句话。

> 妹妹说：“这个人已经不是我的母亲，只是长得像母亲的教徒，但是，我无法推开她，因为她长得像我母亲。”

这个被邪教洗脑的母亲，何尝不是整个日本的缩影？

安倍虽然倒下，但政教勾结的利益链条仍未彻底斩断；山上彻也虽然引发了社会对邪教问题的关注，但根本性的改革始终步履维艰。

就像被邪教洗脑无法被唤醒的教徒们一样，整个日本社会也似乎陷入了某种集体迷茫。他们不敢直面历史遗留问题，不敢正视现实困境，更不敢开辟新的道路。

即便山上彻也的枪声震耳欲聋，即便现实已经如此严峻，这个国家依然选择在旧路上蹒跚前行。
