

# 聊一聊日本自卫队

251130 猫哥

整理：公众号懒人搜索，懒人专属群独享

懒人微信：lazyhelper

谈到日本自卫队，大家第一时间想到的是什么？

是不是“武士道”、“神风”这些日军历史印象。

是不是目前差不多能排进全球前8的综合战力。海陆空三大自卫队军种，特别在海空方面，可能已经进入全球前5序列。

比如海上自卫队：
- 拥有约150艘舰艇，总吨位约65万吨，包括“准航母”的出云级直升机驱逐舰（经改装后可搭载F-35B隐身战斗机），以及8艘配备“宙斯盾”作战系统的防空驱逐舰。海自的反潜作战能力也被公认为世界顶尖水平。

比如航空自卫队：
- 除了拥有200架F-15T战斗机，还有E-767预警机、E-2D预警机和KC-767空中加油机等支援机型，甚至F-35A/B隐身战斗机正在快速列装。

但是，我想说的是，错了。现在的日本自卫队早已不是原来的日军，而且差别非常大。

虽然军队战斗力的强弱主要体现在装备层面，但仅仅是针对“虐菜”这个级别。在面对装备上同级别对手，或者比你更强大的对手时，其实比的还是军队人员素质。

那么，现在日本自卫队的人员素质怎么样？

简单给大家罗列几个重点：

- 第一，目前日本自卫队的平均年龄已达38.5岁，而且三分之一以上是40岁以上的“大叔兵”。
- 第二，日本自卫队和其他国家军队不一样的是，它们全员都是公务员，包括军官和士兵。他们采取朝九晚五的上下班打卡制。除了值班人员，绝大部分官兵都不在部队过夜。一到下班点，就各回各家，各找各妈。为了保障官兵日常起居生活，日本征兵采用就地取材模式，比如第四师团，就在大阪征兵。

是不是又刷新了大家的三观。

- 第三，日本自卫队新兵训练，间隔30分钟就要休息一次，补充能量。气温超过32度就停止训练。

看完上面的介绍，是不是认为在日本当兵不要太爽？

但就算这样，日本自卫队近年来仍然面临招兵困难的境况，2022年到2024年，实际招兵数量也就大致占计划招兵的50-70%。这还是使了牛劲后的成绩。

我们印象的招兵广告是铁血、保家卫国的画面，而日本自卫队招兵广告是卡通、明星美女的画面，这放在全球都是独一份。

那么，为啥日本自卫队混得连招兵都困难呢？

总结原因，主要涉及以下几点：

- 其一，工资待遇低。
自卫队员的底薪仅约为日本社会平均水平的75%-80%。
高级士兵，也就是上等兵，一个月能拿到工资换算成人民币也就是9500-12500元左右。
高级士官，比如士官长，一个月能拿到工资换算成人民币也就是11000-20000元左右。
是不是单看金额好像也不低，但不要忘了日本国内的物价是个啥水平，看看日本大米的价格就知道。可以说，日本是把军费都用在进贡老美军火商身上了。

- 其二，没啥保障、口碑也差。
不仅军队内霸凌现象时有发生，而且日本居民也不待见，经常出现一头搞整训演习，另一头因为扰民，军官挨家挨户跑去道歉。
本来工资就只够糊口，结果年龄到了退伍的时候，还得继续找工作。在军队里搞了几十年，脱离这个体系，哪又能找啥体面的工作？

- 第三，2014年，日本解禁集体自卫权，就导致日本年轻人参军意愿大幅下滑，不到万不得已，不会去加入自卫队。因为大家都清楚，这意味着随时都可能参与战争，成为日本政治的炮灰。

所以，不要看日本右翼浪潮高涨，绝大部分年轻人都是在口嗨发泄不满，真到需要流血贡献的时候，又有几个真能蹦出来？

更深刻的说，日本“失去的三十年”已经重塑了当下的日本人。

上世纪60年代-80年代，因为日本经济高速增长，国力强盛，社会充满机遇和乐观情绪，所以它们信奉努力、拼搏、为公司和国家奉献，相信努力必有回报，也被称为昭和男儿。

你看现在，经济长期停滞，阶层固化，未来不确定性高，就导致很多日本青年成了社畜，沉迷在动漫、游戏、网络、AV里面不能自拔，从而寻找快感和成就感。

日本青年从“昭和”到“平成-令和”的精神状态变迁，说穿了就是个人意志已经垮掉了，这是它们对一个机遇收缩、激励失效的社会系统所做出的系统性、适应性反应。“低欲望”和“向内收缩”，是在新环境下一种务实的、甚至是清醒的生存智慧。

在这种大环境下，生长出来的小本子，战斗力能有多强？一个国家的军队，一定要有责任感和使命感，这是一切的源动力。

1998年抗洪抢险，2008年汶川地震，我们的官兵体现的是一种为国家、为人民，无畏生死的牺牲精神，甚至苏联发生核泄漏事件，苏联军队都在舍身犯险，你再看福岛核泄漏，日本自卫队在哪里窝着？

面对这样一个日本自卫队，对我们而言又有多大的威胁可言。我们需要做到的就是战略上藐视，战术上重视即可。

对于现在日本，之所以我们有所顾虑，无非出于就当下环境而言，我们的叙事主线还是对美对台方面，虽然小日本和我们有血海深仇，但目前也是支线，从服务于大局角度审视，更希望放后面再说而已。

但如果继续作死下去，我觉得好好修理一下也未尝不可。大家不必过于看轻自己，也不必过于抬高对手。

最后，聊聊当下日本叫嚣击沉福建舰这事，11月27号，在国防部举行的例行记者会上，关于日本自卫队内部人员称“日本和美国有能力击沉福建舰”的问题，我方用“痴人说梦，不自量力”这8个字进行了霸气回应。估计美军听了想死的心都有了，头上100个草泥马涌现，你自卫队想去送人头，干嘛非得把我拉上。啥事没干，啥话没说，无缘无故就成了痴人说梦，不自量力的背景，以后还怎么当大哥？

关于福建舰的战力，我们从编队情况看就一目了然。

除了自身搭载歼-15T、歼-35、歼-15D电子战机、空警-600、直-20反潜机外，周边打辅助的还有核潜艇、055、052D、054B以及综合补给舰。

但凡有点军事常识的人，都清楚这个战力摆在亚洲都是无敌的存在。远的不说，大家有没有知道前段时间055横切老美航母编队这事吗？

手上没得金刚钻，哪敢揽瓷器活。

说实话，无论是水下、水面还是空中袭击，就日本手上那点家当，再外加驻日美军，都毫无胜算可言，除非来一次饱和式攻击。

但我们会主动进入日本本土火力覆盖圈内吗？

退一万步说，就算遭遇日本的饱和式攻击。那届时，日本还在不在这个地球上，都难说？

这个交换比，大家认为如何？

所以，日本自卫队内部人员能说出这番话出来，也不怕自己被噎死。除了拉美军垫背，一点效果都没有。

老美现在除了“虐菜”有点积极性外，要在东亚和我们直接来个硬碰硬，除非脑子进水，找不到其它原因。最多就是暗地里把对日军售口子开大，一方面发笔横财，另一方面让小日本自己去赌国运，他在后面捡落地桃子，反正死道友，不死贫道。

那么，在看了上面对日本自卫队分析后，你会认为它能主动扛起这项“历史重任”？

反正我的认识是：一万个不可能。
