

# 段永平、冯柳：赢在对“不犯错”的偏执

我在为重新理解芒格的不犯错哲学做功课的时候，读了步步高创始人、著名的价值投资家段永平的一本新书《大道》。我发现，段永平的投资哲学跟我们上一讲讨论的芒格启发有高度呼应。

《大道》这本书，其实不是段永平自己出的，而是有心人把他过往十几年在雪球投资社区里跟网友的讨论整理出来，征得了他的同意而出版的一本投资问答录。

段永平，在中文投资圈里被誉为“中国巴菲特”，被认为是华人当中最懂巴菲特的投资者之一。他从2001年转型投资以来，身上有不少投资传奇，比如投资网易实现回报140倍，投资苹果实现了炒股20倍回报，早期投资拼多多更是实现了1000倍的回报，等等等等。根据他自己公开的信息和一些第三方研究机构测算，他最近五年，美股的持仓年化收益率达到了49%，近20年来的长期收益率不低于巴菲特，也就是年化超过20%。

而这同时，他并不是一个“卷生卷死”的企业家，他早早就进入了半退休状态，主要精力用来陪家人、打高尔夫球、玩游戏，以及在雪球社区里给网友们“布道”。

说了这么多，我的目的是向你证明，他的投资哲学同样值得研究。

所以这一讲，我主要想摘几段我很有感触的问答原文，来帮你感受一下，也加强我们对价值投资的理解。

整体上，他在书里反反复复在论证两个原则：

第一个叫作“做对的事情，把事情做对”。什么意思？我的理解：做对的事情，就是选准最有把握的方向，不要去做错的，或者没有把握判断对不对的事情；把事情做对，就是在对的方向上，进一步选出最优的行动路径。

第二个是他在挑选企业的时候，集中看三个维度——商业模式、企业文化和价格。其中前两个又最重要，任何一个让他不喜欢，他就会坚决放弃。你看，这是先用情感认同来过滤投资标的。至于价格，可以慢慢等。

我个人认为，这两个原则也是互相嵌套的。根据商业模式和企业文化去选公司，就是“做对的事情”；等价格，就是把“事情做对”。请你带着这两个观念，来看接下来他的几个讨论。

## 【高手在于错误率低】

第一个讨论，高手在于错误率低。段永平说：

今天重新看了一遍《赢在中国》，听到马云的一句话很矛盾，他说不犯错将是最大的错，而巴菲特先生的原则却是不要犯错，这要怎么理解怎么把握呢？这段话引自网友，我觉得是个很好的问题，故说说自己的看法。

我不知道这里所提的巴菲特原话是什么，我只记得巴菲特说过大概这样的话：第一，不要亏钱；第二，不要忘记第一条。我个人理解，他老人家在这里讲的是投资安全的问题。如果一个人投资到他不懂的东西里，就有可能亏钱，所以要避免。

这并不意味着巴菲特就不会亏钱，事实上，他亏过的钱的总额比我们谁都多，因为他也有犯错误的时候，光是他投在康菲石油上亏的钱就有几十亿美元啊。我一直没看明白他为什么要投康菲石油，也许他也犯了以为看懂实际却没看懂的错误？

但是，巴菲特有今天，我个人觉得最重要的就是他很少犯“原则性错误”，也就是说，他认为他能力圈以外的事，坚决不碰。几十年下来，他犯的错远远少于同行，仅此而已。

## 【做对的事情大于把事情做对】

巴菲特之所以是巴菲特，最重要的是他能坚持做正确的事，也就是原则性错误的事不做。比如，他认为自己不懂的事就坚决不做，我从巴菲特那里学到的最重要的东西也就是这个。

马云讲的是另外一面，是如何“把事情做对”的问题。无论是谁，在“把事情做对”的过程当中都是有可能犯错的，好像也没听说哪个所谓成功人士没犯过错。把事情做对的过程，往往是一个学习的过程，犯错往往是不可避免的。不犯错的办法只有一个，那就是什么都不做。什么都不做有时可能就是最大的错。

不知道这么解释你是否能明白？有时想想中文真是很有意思。一会儿说“慈不带兵”，一会儿说“爱兵如子”；一会儿说要“三思而后行”，一会儿又说要“当机立断”，好像并不矛盾啊。它们说的都是事物的不同面而已吧？

第二个讨论：

巴菲特股东信里讲的东西无非就两个：一个是做对的事情，就是去找好公司，关注公司长远的未来；另一个是如何把事情做对，就是如何找到好公司，如何看到公司的未来。

第一个问题其实就一句话，非常典型的“大道至简”，但非常难，绝大多数人在这点上过不去，才会过于关注市场。

第二个问题，是每个人的能力圈问题。做投资，每个人都该找自己能搞懂的好企业。巴菲特懂的我经常不懂，我懂的巴菲特也可能不懂。我大致看过一些巴菲特的股东信，也很认真地听过他在几个大学的问答式演讲。他的语言精练、诙谐、逻辑性极强，对我来说，听起来就好像听音乐一样，非常舒服。对于能看懂他的东西的人来说，这些已经足够好了；对于那些看不懂的人来说，其实就算详细讲解也是没什么用的。

总而言之，在投资里，能说的巴菲特都说过了，我没见过谁能比他说得更简单明了！

道的东西确实难教，必须要靠自己悟。心中无道的人你怎么说也是没用的。大多数人都是不容易开放的，很难接受跟自己以前认识不同的东西。我有时候感觉很奇怪的是，如果是个已经很成功的人，不太接受新东西我也能理解，因为人家已经有足够成功的经历和理解了。现实中我看到的情况刚好相反，那些不开放的人常常就是那些不太成功的人。

在另一个讨论里，段永平说：

大家一定要分清“做错的事情”和“在把事情做对的过程中犯错误”的本质区别。在把事情做对的过程当中，任何人都是会犯错误的，包括巴菲特。但由于坚持不做错的事情，所以犯大错的概率低，很多年以后的结果就完全不一样了。如果还不明白的话，就请看看那些一直很聪明但老是不如意的人们这些年都在忙啥就明白了。

往回看个几十年，你会看到很多很“聪明”的“聪明人”成就很小，原因可能在于他们没把聪明放在做对的事情上。

有网友说，自己发现了一家做饼干的60年老公司，研究了一下发现价格很让人动心，想买了赌退市，问段永平意见。他没有直接给答案，而是给了判断逻辑。他是这么说的——

先假设你看到的数据都是真的（小公司，尤其是家族公司，这点其实非常不容易确认）。你先问你自己几个问题吧：

- 你喜欢他们的商业模式吗？
- 你喜欢他们的企业文化吗？
- 如果你有钱，愿意把这家公司按这个市值买下来并继续交给他们经营吗？（不要想着去改变任何东西，那是不一样的事情。）
- 60 多年的品牌终于走到了今天，你觉得它们再过 60 年会怎么样？
- 你愿意把你的闲钱都投进去吗？
- 当你需要问是不是投机的时候，那就是投机，当然也可以叫投资在机会上。”

你看这个回答，其实回到了段永平的商业模式、企业文化、价格三件套。这家公司价格是对的，但仍然需要从感受上，判断自己对公司的商业模式和企业文化是不是认同。这等于是把自己当作一个感受过滤器，去判断这家企业在长期是不是能被人群和社会喜爱，能拥有长久的生命力。这个是“对的事情”。相反，如果不考虑这个前提，即便遇到好价格，等于是在错误的方向上把事情做对，风险就太大了。

## 【弱者体系】

你看，价值投资的高手们的执念出奇地一致：先少犯错，用少犯错来增加胜率，再借助时间复利把最后的总赔率拉高。

中国资本市场上还有一个投资者，这些年也不断被后来人当作价值投资的高手来研究。这就是高毅资本的合伙人冯柳。冯柳是散户出身，2015 年因为投资成绩太过出色，被高毅资产邀约加盟，成了私募基金经理。

冯柳也有一套著名的“弱者体系”。什么意思？做市场的强者，得深度研究公司、行业、宏观，什么都要抓，让自己尽量不存在盲区。但做市场的弱者，就是预设自己在信息获取、理解深度、时间精力、情绪控制、人脉资源等方面都处于市场的最差水平，于是基于最简单也最深刻的常识，来做投资。

换句话说，弱者体系就是老老实实承认自己的弱者身份，不求战胜市场，而只求不被市场战胜，在长期实现复利。

9年下来，冯柳在高毅资产管理的旗舰基金，专注A股，年化回报率也能超过20%。他的“弱者体系”投资哲学，也越来越被投资圈认为是价值投资的顶级本地化版本。雪球上有散户从2021年开始跟随冯柳践行弱者体系，到了2024年底，累计收益率35.25%，比沪深300指数多了接近六成的超额收益率。

## 【总结】
这一讲，我们讲了两个跟巴菲特、芒格心灵相通的中国投资者，一个段永平，一个冯柳。我们说，他们的共性，就是对“不犯错、少犯错”这件事偏执到了极致，也因此大幅增加了自己在无限游戏当中的胜率。
