

# 银行股为什么涨幅惊人?

最近一段时间,资本市场最大的主题就是银行股涨幅惊人,很多银行股价都接近翻番,那么问题来了,在经济大环境不景气的背景下,银行股价为什么节节攀升呢?

首先,是银行业绩很牛吗?答案是否定的。

以今年一季度为例。

工商银行利润同比下降4%,建行同比下降4%,农行同比上升2%,中行同比下降3%,交行同比下降2%。

银行业绩大多同比下降是符合预期的,现在国内大银行核心业务还是一个放贷机构,主要就是靠着存贷差赚钱,但是现在大环境不好,一季度居民存款创新历史新高,高达近10万亿,与2021年居民新增存款相同,存款如潮一样涌入,但是贷款却放不出去,所以银行业绩不大理想自然也在情理之中。

# 那么在银行业绩不大理想的情况下，究竟是什么资金不断的买买买银行股票呢？

是保险资金。

保险资金持续买入银行股是多种因素决定的结果。
- **政策面**：证监会有关规定，保险公司每年新增保费至少要拿出 30% 买入股票，低于 30% 就是违规。财政部以及金融管理总局也在推波助澜，不但将部分保险公司投资权益类资产上限调高到 50%，同时调整保险公司考核机制，为保险公司买入股票扫清各种制度性障碍，今年上半年，已经落实不低于 1000 亿保险资金长期股票投资试点。
- **基本面**：保险公司吸纳资金平均成本大致在 2.3% 左右，现在大环境不好，保险公司拿着资金也面临着资产荒的局面，筹集的资金该怎么投？银行存款只有 1%，就算是理财也只有 2.5% 左右；国债收益率目前低于 2%。地产处于下行状态。看来看去也就是银行股更顺眼，银行虽然业绩不大理想，但是分红收入对应的股息平均达到 4%，部分国有银行股息甚至超过 5%。

保险公司资金成本是 2.3%，如果买入银行股就有 2—3%的套利空间。

最为重要的是，国有银行大部分股权都掌握在财政部、中央汇金、中央证金、社保等国家队手里，以工商银行为例，国家队持股就超过 93%，如果再剔除保险资金、指数基金持有的股权，真正市场上可以自由交易的工商银行股票占比不到 2%，市值 500 亿左右。

也就是说，保险资金大规模买入银行股，只要国家队不减持，股价基本是安全的，不会有较大幅度的波动——即便某种情况下国家队大规模减持银行股，肉也是烂在锅里，险资对监管层也有理由交代，综合下来比较符合保险资金厌恶风险的需要。

所以，保险资金大规模买入银行股符合政策面引导方向+自身套利需要+波动性较小等等多重逻辑。正是因为这些多种逻辑合力，导致前段时间银行股买盘一直大于卖盘，几乎每天都是净买入，股价自然就节节攀升。

所以，银行股价上升，与业绩基本没啥关系，本质就是前期资产估值太低（目前还有银行股 PB 小于 1），在资产荒的大背景下，匹配了保险资金的需求。

银行股价的节节攀升，带动了大盘指数的上升，算是为资本市场搭建了一个比较好的舞台。那么，这个舞台下其他企业股价能不能有好的表现？接下来就要看国家层面反内卷的成效了。

过去我们说通缩，比较关注的是物价指数CPI，但是其实中国作为一个工业性大国，PPI（工业生产出厂价格指数）也相当重要。逻辑很简单，如果PPI持续下降，意味着出厂价格持续下降，也就是工厂越来越内卷，国内消费者固然可以享受低物价的红利，但是也面临着企业打工人工资收入持续下降的恶果，这就会极大影响消费者的消费预期。

从目前数据来看，PPI指数依然不乐观。6月份CPI同比+0.1%，勉强回正，但是PPI同比-3.6%，环比-0.4%，同比降速还在进一步扩大，说明各行各业产品价格都在加速下降，企业利润在枯竭，发到打工人手里的钱自然越来越少。

所以，现在国家旗帜鲜明提出反内卷，但是整治内卷能有多大成效还有待观察，因为内卷的本质是产能过剩，产能过剩的原因是地方官员考核机制是GDP为主，任何一个地方要拉动GDP最快最有效的方式就是招商引资，另外，我国基本税收体制还是向企业征税，所谓的地方保护主义本质就是地方政府保护自己的钱袋子。

所以，反内卷要取得很大的成效短期来看难度很大，希望在行政干预下，企业的内卷能有部分改善，PPI回升，企业利润回升，居民收入增加，那么支撑资本市场走强的条件就具备了。现在还不具备。
