

# 从宗教控制到军事势力的转变！

250707 守夜人总司令

从某种意义上来说，以色列帮了伊朗内部某些势力的大忙，那是一个以宗教为核心粘合起来的社会。伊朗试过各种方式，最后发现只有这样才能勉强维持。伊朗这个社会的发展历程呈现出一个典型的后发国家的困境：西化改革会造成社会对立，上下脱节，主权丧失，最终沦为政治和经济的殖民地。

回归传统，向社会占绝大多数的底层群体的状况妥协，虽然能压住矛盾维持秩序，但会长期难以发展：内部的精英不支持，外部的西方又敌对和封锁。到目前为止，只有一个国家走出了这个像死循环一样的困境，那就是中国！

为什么最终选择宗教呢？因为只有宗教，才能成为那个散碎的社会的最大公约数。伊朗巴列维国王的改革，不可谓不激进，西方的投入，不可谓不多。然而，一顿改革下来，试图把伊朗这个社会带入文明和现代化的巴列维国王，却成了社会所有阶层的公敌！在传统的农业社会中，教士、乡绅、喇嘛、阿訇和牧师，不仅仅是一个职业，还是一个社会群落的纽带，而且他们天生就是知识分子，具有高度的组织性。

如此一来，才能在广袤的土地上，把散落又原始的人群组织成一个共同体，这是一种低成本的组织方式。而世俗化和现代化的公共秩序，是一种高成本的解决方式，不仅需要现代化的配套设施，也需要常备化且由公共开支供养的统治机器来维系，这一切最后都落到一个字上：成本！

读过《24年0530：人民的选择！》就能理解中东诸多国家的世俗化和现代化改革为什么最终都无疾而终。包括最近的叙利亚，在几十年的世俗化尝试之后，还是不可避免的走了回头路！低成本的统治和一盘散沙的状态符合内外各方的利益。

伊朗是一个有着丰富资源的国家，曾经在西方主导的情况下，建立了一定的工业基础设施。只不过，在过去的40年中，这个基础设施没有得到发展，既有外部的原因，也有内部的原因。在美国最强盛的历史阶段，伊朗选择与之对抗，而在美国逐渐虚弱的时候，伊朗却迫不及待的想要投降。这种迷之操作源自于现实的迫不得已和力不从心。

有一个值得玩味的现象：伊朗存在两套武装体系，一套是国防体系，另一套是宗教卫队体系。宗教卫队体系比前者国防体系更能优先得到各种资源，也更加强大。然而，后者的费用是不公开透明的，后者不仅掌握武装，还掌握类似于国企体系的经济组织。更重要的是，宗教卫队体系不用交税，也不透明，还不听政府的。恰恰相反，近20年来，越来越多的高官都出自于宗教掌握的卫队体系！卫队体系掌握在教宗手中，正因为如此，教宗与行政首脑的矛盾一直没有断过。

不管教宗选择什么样的人来说出任这个管具体事务的职位，最后都会闹翻！为什么会这样呢？对这个岗位是解决具体问题的，解决具体问题就需要钱，钱都在你那里，事要我来办，而且，你们花钱都不透明，自成体系无法无天。没有钱办不了事，大家都怨恨我，你为了维护自己的形象也指责我，你让我怎么办呢？这就是所有负责行政具体事务的人，最终都站到了对立面的原因。唯一一个还没有站到对立面的人，还没等到证明身份那一刻的时候，就飞机失事摔死了。这个是很蹊跷，完全不排除是内部人的手笔！

为什么伊朗有这么多的内奸呢？被渗透的像筛子一样，就是因为内部的许多矛盾一直没有一套相应的机制来调和和解决。积压的太久了，人心思变，有些人就被收买了，或者主动求降，试图借用外力来打破内部的体系。

以色列国防军派出了7个装甲和步兵旅与不到2万人，连重武器都没有，在跟巴勒斯坦哈马斯周旋那半年，结果损失了100多辆坦克装甲车，高达5000多人阵亡。从数量和武器装备的编制上来看，不亚于第一次车臣战争中俄军先头部队的全军覆没。那种损失对于俄罗斯这么大的体量都是难以承受的损失，对于以色列这种体量，更加是一种惨重的损失。

这意味着在常规的战争模式下，以色列已经搞不定这些武装了，只能采用非常规的手段，所以随后改变了战法，首先用BB机爆炸的方式，干掉了真主党武装的指挥体系。在失去指挥体系的情况下，各自为战的武装在不对称的力量下就是待宰的羔羊。随后又用定点清除的方式，不断的干掉对方的老大，新上了一个就干掉一个，这种斩首行动不仅在指挥体系上给对方造成毁灭性的打击，更在心理上让人感到恐惧和绝望，彻底瓦解对方的抵抗意志。

以色列把这种成功的经验，复制到了伊朗身上。伊朗的关键人员接二连三的被定点清除，关键人物定点清除所造成的威力，比成建制的消灭伊朗一个团、一个旅，甚至一个装甲师，更能瓦解对方的抵抗意志并增强恐慌的蔓延。一方面，关键人物定点清除所造成的威力会让内部的力量此消彼长。任何一种稳定的格局都是由人来搭建的，一股势力的人马不断损失，另一股势力的人马就会去补位，最后造成内部原有格局的失衡，从而形成内爆或者重新洗牌的效果！

更要命的是，这种封闭状态下的统治都存在一个共同的特点：就如同一个控制孩子的父母，会将外部的事件描述得非常恐怖，以强化孩子对自己的依赖，并且以保护神的面目出现在孩子的心中。对于这样的父母，最害怕的就是：让孩子看到自己的软弱和无能！

因为这个保护神形象的破灭，就会让父母自己显得多余。孩子会认为反正都要被欺负，有你没你都一样。所以这样的父母一定是逞强的，哪怕是对抗外特别怂，反而极力的营造出一种无所不赢的假象。这种假象无法通过正常的方式撕破，然而，对方如果不断的采取定点清除的斩首行动，而那个总是以无所不赢的形象示人的父母却没有还手之力，没有什么比这个更能让孩子意识到父母的无能！

如此一来，孩子不仅会感到愤怒和恐慌，甚至会为自己一直忍气吞声的承受父母的各种苛刻对待感到愤怒不已。这种情况下的孩子不会再愿意听信父母的任何解释，也不再愿意忍受一分一毫。这对于父母来说，是一种灾难性的局面，一种比外部的危机更难以承受的危机。

当然了，这种事情是有好处的，最大的好处就是不得不提拔新人来补位，因为原有的人都被炸死了，总要有人来干活。新人被提拔的人，与教宗的关系并不密切，如果密切早就被提拔，就是因为关系密切的人都打死了，才不得不临时提拔不密切的人来补位。如此一来，很多位置就会被那些不密切的人占据，这种跟教宗原有体系不密切的人一旦多起来，情况就会发生微妙的变化。就算在世的时候不会，一旦过世，后继的娃娃是稳不住局面的，整体的改弦更张就会发生在那个时候。

总的来说，宗教势力会逐渐被世俗的军事和非宗教政经势力替代。只要这种替代过了临界点，后面的事情就不可避免...可以拭目以待！
