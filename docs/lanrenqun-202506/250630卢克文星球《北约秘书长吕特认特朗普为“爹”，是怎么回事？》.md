

# 北约秘书长吕特认特朗普为“爹”，是怎么回事？

特朗普喜当“爹”了。

6月25日，北约峰会时，北约秘书长马克·吕特恭维特朗普结束了以色列和伊朗的冲突，将其比作“爸爸”，称“爸爸有时不得不使用强硬的语言”。

之后，吕特还给特朗普写了一封肉麻的短信，里面说，“特朗普总统，亲爱的唐纳德，您让这一改变成为可能”。

对于别人的“认爹”，特朗普自然乐开了花，不仅公布了吕特的短信，还在记者会上调侃吕特喊他“爸爸”。

“认爹”新闻一出，惊起欧洲千层浪。有记者向吕特求证。吕特倒也实诚，说，是有这么回事，不过他辩解，他是从欧洲人担心美国是否还会与欧洲站在一起的角度使用“爸爸”这个词，并非真的称呼特朗普为“爸爸”。

但是吧，不管是比喻也好，还是真认也罢，吕特“认川（普）作父”是跑不了了。

中国网友看热闹不嫌事大，各种调侃，各种表情包横飞，甚至将吕特比作三国的吕布。整个中文互联网充满了快活的气氛。

“认爹”闹剧，是这次北约峰会奉承特朗普诸多动作的一部分。仔细分析这次北约峰会，会发现宗旨只有一个：千万别让特朗普不开心。

为了让特朗普开心，北约秘书长吕特忙前忙后，使出了浑身解数。

峰会前，为符合美国要求，吕特苦口婆心给北约各成员国一一做工作，嘴巴都磨秃皮了，终于让他们同意了GDP 5%的军费目标，其中3.5%属硬开支（即武器），1.5%属软开支（国防和安全投资）。

虽然年限是2035年前，但这个比例在去年欧洲人想都不会想，可以想见，吕特费了多少口水，掏了多少心跟肺？

所有人都知道，特朗普没有耐心。于是，吕特十分体贴地缩短峰会议程，仅安排一纸公报，一次工作会议，剔除一切繁文缛节，只为了让年事已高的特朗普“不那么累”。

- 四、安排了皇家晚宴款待特朗普，让他享受了一次“国王待遇”。这个马屁一下拍到特朗普的心坎上去了，因为此时在美国国内，一些美国“刁民”正在举行反特朗普游行，打出的旗号就是“不要国王”。你美国不要国王，我老欧洲就让川大统领感受一次“国王待遇”。
- 五、知道特朗普现在对泽连斯基不爽，于是，干脆不邀请小泽参加领导人会议，只允许他参加晚宴，而且座位还安排离得特朗普远远的，让懂王“眼不见心不烦”。

你看，这一系列操作下来，你觉得吕特将特朗普比作“爹”还算个事吗？看到这，有小伙伴就纳闷了，至于吗？你吕特当年也是当过荷兰首相的人，怎么一当上北约秘书长，就要对特朗普如此卑躬屈膝？人格呢？国格呢？

这么说吧，当荷兰首相的吕特对懂王不一定会阿谀奉承，但作为北约秘书长的吕特一定会对美国（无论谁当总统）奉承讨好。

问题不出在吕特身上，而是出在北约身上。

## 「02」

北约现在的问题很大。

最大的问题是，美国“不想要”北约，至少特朗普想“离婚”，但欧洲需要北约，他们恳求特朗普“不要抛下”他们。

北约是典型的冷战产物，是当年美国为了围堵苏联，拉着一帮欧洲国家建立起来的。后来苏联为了与之对抗，也拉着一帮社会主义国家组建了“华约”。按理说，苏联没了，北约没有了存在的必要。

但是，作为一个在欧洲深耕几十年的组织，北约已深入欧洲的方方面面，形成了一个“深层政府”组织，构建起了一个强大而根深蒂固的利益共同体。

这个“深层政府”组织，主要是由政府官僚、公务员、军工复合体、金融界、财团、情报机构等高层组成的组织，他们掌握了巨大的权力和资源，幕后控制国家。

多年来，欧洲的深层政府组织与北约相互影响，甚至本身就是北约的一部分。

典型的例子是丹麦的情报系统。

丹麦的海底光缆网络极其发达，如同血管一般将丹麦与欧洲其他国家紧密相连。据统计，全球互联网流量97%经此传输，这使得丹麦成为欧洲的关键数据枢纽。上世纪90年代末，丹麦情报系统就跟美国签订了“电缆协议”，丹麦藉由向美国提供情报，换取美国的特殊支持，并提升自身在北约中的影响力。

2012 年开始，美国国家安全局和丹麦国防情报局在哥本哈根南部阿迈厄岛合建了一个数据中心，主要用于光缆数据的截取，目的是监听欧洲政要（包括法国总统马克龙、德国前总理默克尔）。

这种深层政府就跟美国“影子政府”左右美国政局一样，他们也左右欧洲政局，更重要的是能一直分食利益。

但在特朗普看来，这些利益跟他又没有关系，他又分不到一杯羹，此其一：
- 其二，现在美国债务危机十分严峻，而北约还像吸血鬼一样一直在吸美国的血，他能忍？
- 其三，北约即便存在，也只是便宜了美国的军工复合体，但军工复合体跟他懂王不是一条心，多年来针插不进水泼不进。

在这种情况下，懂王自然要嚷嚷退出北约。这就急坏了北约，尤其是急坏了北约的最高领导吕特秘书长。

如果美国真退出了，北约就散了，北约散了，欧洲诸国就要自己承担各自防务了。之前网上有一句话，几十年来欧洲之所以过着好日子，无非是在薅中美俄三个大国的羊毛，即：俄罗斯的廉价能源、中国的廉价商品、美国的廉价防务。

如果北约解散，欧洲自行承担防务，那必定会增加军费开支，而增加军费开支，势必又会增加财政支出和减少社会福利支出。而这两项则直接损害“深层政府”的利益。就拿前面提到了丹麦情报系统来说，丹麦政府如果将过多的财政收入投到国防开支上去，势必会减少在情报系统的投入和财政支持，情报系统就会面临着缩编和裁员，这还怎么玩？

损人利益，如同杀人父母，过惯舒服日子的老欧洲们怎么可能同意北约散伙？

无论如何，也得把美国留在北约内。只要美国还在领导北约，“深层政府”的利益就不会被削减。

现在，你理解作为北约秘书长的吕特为什么要对特朗普阿谀奉承、喊他“爹”了吧。

至于特朗普要北约各国将国防开支比例提到5%，那完全不是事。提升到5%和完全自行承担国防开支，完全是两个概念，再说了，提升的年限是2035年，到时特朗普已经下台了，是不是真提升到5%，他也管不着了。

总之一句话，将特朗普糊弄好、奉承好，对吕特和北约来说，比什么都重要。作为北约一把手，他必须为北约利益考虑。

## 「03」

有人就说了：不对啊，欧洲真需要北约吗？法国总统马克龙不是一直吵嚷着“北约已经脑死亡”、要实行欧洲防务自主，甚至要成立欧洲军吗？

其实，吕特之所以如此讨好特朗普，另一个深层逻辑就是要打压法国和德国这两个刺头。

这次北约峰会，法国总统马克龙提出一个建议：欧洲应该直接和俄罗斯就“恢复信任”展开对话。

这个提议让吕特很不爽，不点名地批评了马克龙。

二战时法国虽然投降快，但并不妨碍它想成为欧洲老大。从戴高乐时代起，法国就一直看不惯北约。近年，法国总统马克龙就一直在不同场合变着花样抨击过北约，当然，他也知道以法国单独的实力还不足以与北约分庭抗礼，于是就想拉着北约经济实力最强的德国一起“造反”。也就是说，他们想自己来领导欧洲，而不是让美国通过北约来领导欧洲。

法德之心，路人皆知，但他们的意图，北约不会同意，欧洲其他小国如波兰、荷兰、波罗的海三国等等等等更不会同意。

道理很简单：如果欧洲这些国家真遇到战争危险，他们是相信你法德，还是相信世界老大美国？一直以来，这些国家是求着美国去驻军，而不是求着法国和德国去驻军，他们只是国家小实力弱，但不傻。

如果北约散了，美国走了，最先遭到战争威胁可就是他们这些中东欧小国，真等你们法国和德国来救，他们早就灭亡了。

再说了，你法国能保护他们什么？

对于法德的心思，美国是一直心知肚明的。2018年，马克龙提出要成立欧洲军的时候，就遭到过懂王的炮轰和讥讽：“法国还是先将在北约的军费比例提高再说吧。”

这次峰会，马克龙提出要和俄罗斯直接接触的时候，不等懂王开呛，吕特就先批评上了。之后转头就对懂王“喊爹”。

这一举动，显然是在打法国的脸，更是给了欧洲所谓防务自主一记耳光。吕特就是在告诉法国和德国：别出头，北约永远是美国爸爸的北约，大树底下好乘凉，你小屁孩懂什么？

同时也是在告诉懂王：你永远是我们欧洲的爹，是欧洲的主心骨，一些反骨仔翻不起风浪。没有你的带领，欧洲就如同黑暗中迷路的孩童。

听到这，懂王满意地轻拍吕特的头，颔首而笑。

在吕特这一操作下，北约的价值得到了无限突出和放大，而法国（以及德国）反倒成了犹大。

北约，这个冷战时代的组织，发展到现在，每一寸皮肤都散发着不合时宜的气味。早就应该被扫进历史垃圾堆的东西，如今依旧在垂死挣扎，这样的组织，闹出“认爹”丑剧，一点都不稀奇。

只要它继续存在，若干年后，只怕会闹出更大的笑话。
