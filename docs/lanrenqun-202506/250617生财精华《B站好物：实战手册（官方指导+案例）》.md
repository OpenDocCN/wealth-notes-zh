## 拆解：超级标（说清楚，为何这个项目有机会）

- 平台政策好：B 站开放电商跳转口，允许创作者把流量带走卖货，评论区挂蓝链等，为带货提供便利。
- 流量有优势：视频 SEO 友好，能获得长期精准流量，还可通过推荐、搜索等带来更多流量，形成流量复利。

## B 站好物：悬赏带货（蓝链接）

B 站悬赏计划是 B 站官方电商带货服务，UP 主可以在视频播放器内外、动态、专栏、橱窗等位置通过官方悬赏带货组件配置商品链接，并通过带货获得销售分成，也可以为 UP 主自有店铺的商品进行带货。简单来说就是基于电商平台联盟 CPS 体系支持下的视频带货功能。

![](img/50db70064a79f05c4335b3d96fcdc1bc_2_0.png)

悬赏带货实现了 UP 主、粉丝、品牌的三赢：

- 粉丝获得心仪好物 - 通过 UP 主更高效地匹配需要的东西
- 品牌取得推广曝光 - 通过 UP 主更精准触达潜在用户
- UP 主获取商业价值 - 通过带货获取收益持续内容产出

#### 1. B 站蓝链带货指的是什么？

B 站蓝链带货其实就是一个 B 站通往站外的跳转链接。

![](img/50db70064a79f05c4335b3d96fcdc1bc_6_0.png)

#### 2. 它的展现形式是怎样的？

通常出现在评论区，在置顶评论里的蓝色链接。

在蓝链的前面会有一个小标，一般是“淘”/“JD”的字样，如果是“淘”，代表的是商品来自淘宝，如果是“JD”，那就代表是来自京东。目前B站蓝链带货支持淘宝和京东两个平台。

#### 3. 蓝链带货转化链路是怎样的？

下面是具体的蓝链转化链路演示：

开通橱窗后，在（移动端）【我的】推荐服务栏里点击【橱窗管理】即可进入橱窗页面，通过橱窗可以自由添加、管理商品。

申请橱窗后，证明该 UP 主可以挂蓝链了，具体要挂什么商品，需要商家提供特定的链接。

#### 4. B 站官方审核橱窗申请标准是什么？审核商品标准是什么？

审核橱窗是只要 UP 主满足上面我们说的几个条件，基本都会通过的。

审核商品主要是看类目，某些品类可能会受到限制，比如医疗器械、或是一些特殊品类，得需要 B 站特定开白，针对某个商品开白，不是整个品类开白。

#### 5. 企业有蓝 V 号可以添加蓝链带货吗？自己带会比找 UP 主效果好吗？

企业蓝 V 号也是可以用蓝链带货的。

效果的话，虽然说企业号在 B 站的定位跟 UP 主是不一样的，相当于品牌私域，但最终的带货效果，本质上还是要看视频内容。

## 投放模式

- CPC (Cost Per Click)：以每点击一次计费，bilibili单次点击成本0.5元起竞价，具体以实际消耗为准，成本可控，适合电商、应用类客户。
- oCPC (Optimized Cost Per)：以目标转化为优化方式的点击出价，智能竞价。转化目标：订单提交、有效线索、表单付费、微信负责、应用内访问、应用激活、落地页调起、店铺停留、应用下载、应用留存、应用内付费等。
- oCPM (Optimized Cost Per Mille)：目标转化：稿件播放。单次视频播放0.05元起竞价，即用户点击推广视频播放所产生的费用，实际根据市场行情投放而产生计费为准，可能低于0.2元也可能高于0.2元。
- oCPM (Optimized Cost per Mille)：目标转化：自然粉丝关注。用户关注2.5元起竞价，即获取单个自然用户成本出价2.5元起，实际根据稿件质量投放而产生计费为准，可能低于2.5元也可能高于2.5元。
- oCPM (Optimized Cost per Mille)：目标转化：评论区链接点击 / 框下链接点击 / 小黄车点击。目标转化5元起竞价，即获取单个自然用户有效转化成本出价1元起，实际根据稿件质量投放而产生计费为准，此计费模式是针对超推账户。
- CPM (Cost Per Mille)：展现成本，或者叫千人展现成本，bilibili千次曝光成本5元起竞价，具体以实际消耗为准，适合更大范围曝光。第一时间内提升视频曝光率冲刺热门与排行榜。

## 一、 什么是悬赏计划（B站好物）

bilibili 悬赏计划是 B 站官方电商带货服务，UP 主可以在视频播放器内外、动态、专栏、橱窗等位置通过官方悬赏带货组件配置商品链接，并通过带货获得销售分成，也可以为 UP 主自有店铺的商品进行带货。简单来说就是基于电商平台联盟 CPS 体系支持下的视频带货功能。

![](img/50db70064a79f05c4335b3d96fcdc1bc_22_0.png)

悬赏带货实现了 UP 主、粉丝、品牌的三赢：

- 粉丝获得心仪好物 - 通过 UP 主更高效地匹配需要的东西
- 品牌取得推广曝光 - 通过 UP 主更精准触达潜在用户
- UP 主获取商业价值 - 通过带货获取收益持续内容产出

发布 180 天以内、原创视频可挂载商品，动态/专栏需在发布/编辑时添加商品，且稿件和广告创意需要全部通过审核。
稿件和广告创意都通过审核的 5 分钟之后，手机端可以查看资源位。
广告创意标题需避免出现广告法禁止的“最”、“第一”等词语。
如创意长期处于“审核中”的状态，建议修改标题文案。
注意电脑端发布的视频，必须要勾选商业报备。

### Step 6: 查看收益&结算

结算规则：
淘宝联盟和京东联盟带货 B 站平台不抽佣，联盟服务费按联盟规则收取，如淘宝联盟收取佣金收益的 10%（GMV*佣金率*10%）作为技术服务费。
会员购联盟结算规则：用户确认收货后的 T+15 天，结算会员购佣金，佣金=（商品原价-商单承担优惠金额）*带货分成比例；会员购带货佣金会在次日更新。

查看收益和提现金：
淘宝联盟和京东联盟的带货收益 B 站暂不显示，详细带货数据和收益明细可至联盟端查看并进行提现操作。以淘宝联盟为例，登录个人账号进入淘宝联盟，点击【效果报表】，选择【订单明细表】，订单明细表页面选择视频发出时间段，点击下载报表，即可查看佣金收益。

会员购联盟带货收益，在佣金到账后会在【悬赏带货-我的收益】里提现。当月整月的佣金会在次月统一打款至贝壳（预计5~10日可提现），UP主可至【我的钱包-贝壳】进行提现操作。

## 玩转 B 站带货业务-常见问题 QA

### 关于权限开通

Q：为什么我不可以加入悬赏计划呢？

A：如果您达到了加入悬赏计划的门槛但无法加入，可能是 UP 主之前有过不诚信行为，类似收到免费的样品拒绝拍视频等，我们会取消 UP 主带货权限，可根据提示发送相应邮件进行进一步沟通。

Q：开通完就可以立即带货了吗？

A：不可以哦。开通完毕后需要 UP 主绑定 PID，这样 UP 主才可以获得带货相应的佣金。建议 UP 主大大开通悬赏计划后立即绑定 PID 哦。

### 关于淘宝 PID

Q：PID 可以自己填写吗？

A：不可以哦~PID 作为 UP 主在 B 站和淘宝联盟唯一识别 ID，需要按照步骤绑定或者生成相应的 PID 哦。

Q：在淘宝端绑定 PID 时，显示我的昵称不可用怎么办？这就是我在 B 站的昵称啊？

A：建议大大替换 B 站昵称的 1-2个字呢，这个不要求完全同步一致的哦，昵称不一致不影响佣金结算哒。

Q：怎么删除 PID？

A：目前不支持删除 PID，建议新建推广位及 PID，重新进行绑定。每日可修改 PID 一次。

Q：多次绑定 PID 显示失败？

A：请 UP 主检查一下：联盟账户、支付宝、淘宝三者绑定的是不是同一个手机号；再次执行操作绑定的时候需要登录对应的淘宝账户。

### 一、 选品逻辑

1. 什么样的品适合B站？

通常来说，在短视频平台选品基本上都需要符合以下特点：

- 低中等客单产品：300 以下低客单价的产品决策效率低，ROI 更易达成；
- 适合视频展现的产品：产品价值能视觉化和场景化表达——好看好玩好展现，真实有共鸣
- 有特点的产品：击中痛点/有独家优势（优惠机制、相较竞品经得住横向测评）/新奇特

B站用户以Z时代和都市白领为主，主张悦己和理性消费，更会为产品实际价值、颜值和情怀买单。

- 类目特点：美容个护、家居家电、营养健康、仪器工具等较受用户欢迎
- 价格特点：100-300元客单产品更匹配B站用户消费能力，ROI更易达成要求，高客单适合粉丝量级较高或专业类UP主带货
- 内容特点：有高新技术、适合测评、击中垂类人群痛点的刚需或黑科技产品更受欢迎
- 产品外观：有颜值、成分专业、体现人文情怀科技属性的产品更受欢迎

## 玩转 B 站带货业务--定位核心运营指标

如何把重点运营指标拆解成切实可执行的运营策略。

```
GPM = GMV / 视频VV * 1000 = 商品点击率 * 下单转化率 * 客单价 / 视频VV * 1000
```

从这个公式中，我们可以拆解出提升销售额的核心运营指标：商品点击率、下单转化率、客单价以及视频的 VV。

那么定位了这些运营核心指标，我们如何变成有效的运营手段，提升视频带货的销售额呢？接下来我们会把这几个率一次拆分成 3 个细分场景：电商的三大要素：人、货、场

![](img/50db70064a79f05c4335b3d96fcdc1bc_84_0.png)

商品点击率及视频 VV 主要是从“人”的场景出发做优化，重点可以从 IP 定位、内容结构及创意、巧蹭话题热度、优化产品呈现及口播话术等几个维度进行优化。

客单价则主要对应的是“货”的场景，主要从选品结构（选什么样的品，价格区间怎么定位，如何打造爆品）、以及匹配目标人群（我的粉丝画像是什么，要什么，他们的痛点是什么）等维度进行考量及优化。

商品转化率则对应的是“场”的运营，分别从如何打造营销氛围、如何通过营销活动及福利吸引粉丝下单、以及运营粉丝增强互动等几个维度进行优化。

##### 1)结算规则
UP 主带淘宝联盟、京东联盟和多多进宝的商品，联盟服务费由对方联盟平台按规则自行收取，B 站平台目前不抽佣，也不参与结算。

B 站内部商品结算规则：买家支付完成后的第二天，可查看带货订单明细；买家确认收货后 15 天，可获得佣金。如确认收货时间是 2022-04-10 09:00:35，那么结算时间就是 2022-04-25 09:00:35；已结算的收益会在次月统一打款至贝壳，预计 5~10 日可提现。

##### 2)查看收益和提现
**淘宝联盟-WEB 端**：登录个人账号进入淘宝联盟，点击【效果报表】，选择【订单明细表】，可进行查看；点击下载报表，即可查看详细佣金收益。

![](img/50db70064a79f05c4335b3d96fcdc1bc_98_0.png)

**淘宝联盟-APP 端**：登录个人账号进入淘宝联盟 APP，点击【收益】-【推广订单统计】，可查看实时/昨日/近七日/近 30 日/本月数据情况；点击【订单明细】，即可查看订单维度的详细佣金收益。
