# 前言

与平常人交往，共享乐易，共患难难；与天子交往，共患难易，共享乐难。马斯克应该已经体会到了这个道理。

《大明王朝1566》中，在因改稻为桑引起的这场暗流涌动的博弈里，有许多因此丧命的人物，其中一位早期下线的官员为杭州知府马宁远。

根据剧情设计，御前会议是在正月举行的，改稻为桑的政策也是此时颁布的，但浙江方面在推动时遇到了重重困难。

马宁远曾这样对下属和百姓喊话：“改稻田为桑田，上利国家，下利你们，这么天大的好事，就是推行不下去！”有一点气急败坏的感觉。

作为马宁远直接上级的浙直总督兼浙江巡抚胡宗宪早就意识到改稻为桑是《一步死棋》，但话又不能明说，只能寄希望于通过推行过程中遇到的阻力让上级意识到事情的复杂。

可马宁远非但没领会到胡宗宪的良苦用心，反而理解偏了。

到了五月，端午汛来了，人在京城的“小阁老”严世蕃想出一个馊主意——借着汛期毁堤淹田。到时候你们水稻种不成，改也得改，不改也得改。

经过几个月的上下博弈，严世蕃已经感受到了胡宗宪对改稻为桑不积极，因此直接给胡的两位下属写信授意，即浙江布政使郑泌昌和浙江按察使何茂才，并得到江南制造局兼浙江省市舶司总管太监杨金水的支持。

了解到严世蕃意图后，杨金水、郑泌昌、何茂才三人一起约见了杭州知府马宁远，让他背着胡宗宪去毁堤淹田。

杨金水是这样做马宁远思想工作的：
杨金水：我问你，你听胡部堂的？
马宁远：那胡部堂听谁的？
杨金水：这不就结了？肯干事、认上司，这是你的长处。可是肯干事，不是指一指、拜一拜。你认胡部堂，胡部堂认阁老……

马宁远是胡宗宪一手提拔的亲信，三人之所以让他参与毁堤淹田，一方面固然是由于属地管理问题，另一方面也是趁机想把胡宗宪拖下水。

凭借自己有限的政治智慧，马宁远觉得胡宗宪不是不愿意改稻为桑，而是太难没法改。除此之外，因来自裕王府（严嵩政治对手）的谭纶长期在胡宗宪身边，马宁远觉得领导可能言行受限，所以便大胆揣测胡宗宪是希望改成的，于是自作主张参与了毁堤淹田。

- 1、改稻为桑是朝廷下的明文政策；
- 2、领导的领导（严世蕃）暗中授意通过毁堤淹田来推动改稻为桑，两位非直接领导（郑泌昌与何茂才）也都支持；
- 3、唯独直接领导（胡宗宪）态度模棱两可。

这其实是一个非常经典的场景，接下来的故事就围绕它展开。

胡宗宪：你知道朝廷的水，有多深吗？无论做官还是做人，首先要识大势。对于官场来说，“大势”主要分两种：一种是“进取型官场生态”，强调创新、表现、政绩，不露脸就没人记得你，没人记得你就没有晋升机会；另一种是“防御性官场生态”，强调守规矩、防风险、讲忠诚，以稳定为先，低调行事，从“快马加鞭搞建设”变为“收紧缰绳保安全”。

在《大明王朝1566》里，朝廷内部的倒严势力与严党已经到了水火不容、濒临决战的阶段，此时任何风吹草动都可能引来滔天巨浪，更不要说涉及严党的重大违法行为（毁堤淹田）。

马宁远作为知府层级的官员，不仅没有意识到官场生态已发生重大变化，而且还在直属领导胡宗宪态度不明朗时妄自揣测。上级的上级既然想越级指挥你做一些灰色地带甚至违法的事情，这里面肯定有许多隐藏内幕，此时最保险的做法就是向直属上级汇报。

什么叫心腹？心腹的本质不是能力强，而是方向对，最大资本来源于领导的信任，能够充当领导意志的延伸。

“勤请示、多汇报”作为心腹官员的一种角色管控机制，目的是确保自己的判断与领导保持高度一致，不能让领导没有安全感，而要让其觉得一切可控。简单来说，遇到关键事情不要自己拿主意，去请示，仔细观察领导在指示过程中的言行神态，从中尽可能多地汲取信息。

很多重大决策都存在潜在风险，如果你擅自行动，哪怕初衷是“为领导好”“为单位好”，一旦发生问题你就是替罪羊。请示本质上是“授权确认”，把责任提前向上转移，在执行一项重要任务时，一定要让周围人明确知道“你是听了谁的话才做的这件事”。

既然是心腹，更加不能刻意隐瞒，而要把尽可能多的信息原汁原味传递上去，让上面拿主意。

马宁远：朝廷改稻为桑的国策能够施行，部堂您不再夹在里面为难。
胡宗宪：这么大的事情居然伙同他们，瞒着我去干，还说你这颗心对我是忠的？

上级很忌讳被下属做信息隔离，剧中与马宁远对应的政治人物是杨金水，尽管也卷入了漩涡，但最终得以脱身。吕芳对杨金水说：“难得是你不隐瞒，这便是最大的忠。”嘉靖对吕芳说：“你用的这个杨金水，还是得力的。”

那如果察觉到风险而领导坚持要做该怎么办呢？这便到了政治决断的时刻，比如胡宗宪对严世蕃，孙连城对李达康。他们的办法很简单，即不否认但也不推动争议性工作，让时间去消化矛盾。以孙连城为例。

李达康当众让孙连城接任“光明峰项目”总指挥时，从言语表情中能看出来孙连城是不太愿意干的，因为他知道“光明峰项目”是个大雷，早晚得出问题。后来因财政垫资问题，李达康授意孙连城重新拍卖大风厂土地，更是把他往火坑里推。孙连城深知一旦出事没人救得了自己，只能以拖待变。

剧中孙是因为信访窗口整改问题被沙瑞金和李达康定性为懒政官员，最终去少年宫看星星。假如抛开这个专门设计的情节不谈，孙连城的策略其实是成功了——随着山水集团和它的后台垮掉，大风厂难题迎刃而解。所以有些争议性工作不用急着推动，之所以推不动很可能是存在不可言说的深层次原因，走着走着或许就柳暗花明了。

李达康：“拍卖那块地呀……形成的东西全部推翻？”
孙连城：“这个……”
孙连城：“你跟你们高总说，这事呢，得慢慢来。”

再回到剧情。端午汛决堤的事情闹大后，胡宗宪果断斩杀了马宁远、两位知县以及河道监管，理由是修堤不力、贪墨公款，而不是毁堤淹田。这样做是深思熟虑的结果。因为毁堤淹田事关重大，直接牵扯到严世蕃和大明高层的争斗，胡宗宪区区一个浙直总督做不了主，必须交由嘉靖裁决。

那为什么要先斩后奏呢？原因也是怕事情闹大——倘若留着杭州知府马宁远、河道监管李玄等人，裕王和徐阶、高拱、张居正一方必定会想方设法来审问他们以拿到严党罪证。站在胡宗宪的角度，于私，他是严嵩一手提拔的人，于公，他不希望大明朝登时生乱。所以尽快把毁堤淹田这件事从公开层面解决掉，就是最明智的选择，至于私下向嘉靖的沟通则可以慢慢来，见机行事，用口头汇报的形式而不是落成黑纸白字。

当然，为了让马宁远死得更有价值，胡宗宪还上演了一出“口供大戏”。马宁远是毁堤淹田的关键当事人之一，而郑泌昌、何茂才、杨金水曾亲自向他表示这是“阁老、小阁老”的意思——他们都在一条船上。在确认郑、何有意把自己推出来顶锅后，心如死灰的马宁远写下亲笔口供，并交给胡宗宪，这份口供遂成为胡宗宪的杀手锏，用来逼迫郑、何、杨三人在改稻为桑一事上与他统一立场。

剧中杨金水、何茂才、郑泌昌看到马宁远把口供交给胡宗宪时的表情是这样的：“照部堂的意思，改吧。”接下来的事情发展有些超出胡宗宪预想，他未能在御前会议之前见到严嵩（未把马宁远供状一事向严嵩提前交底），最终嘉靖、严嵩、裕王、胡宗宪等人在彼此信息不对称的情况下面对面摊牌。

嘉靖此时已从杨金水那里知道了“毁堤淹田”是严世蕃的主意，但没有过硬证据。严嵩并不清楚嘉靖已经知道“毁堤淹田”的事情，虽然已经生出怀疑。胡宗宪手中掌握着马宁远供状，但这份供状到底是何内容，除他自己外没有别人知道。最终结果是胡宗宪在公开层面向嘉靖隐瞒了“毁堤淹田”的真相（一旦公开相当于立即逼皇帝做是否“倒严”的决断），同时暗示嘉靖此时不宜“倒严”，并趁机辞去烫手山芋的浙江巡抚一职，专注于抗倭作战。

嘉靖同意了胡宗宪的辞职请求，也没有深究毁堤淹田一事。事实表明嘉靖没有那样在意毁堤淹田的真相，他更重视改稻为桑能否成功，能否迅速带来财政收入。而胡宗宪很会边观察边表达，注意到领导态度后，他没有再向嘉靖质疑“改稻为桑”政策的合理性（跟领导提要求不能得寸进尺），事后该任务被郑泌昌、何茂才认领，直到矛盾全面爆发。

有观点认为，马宁远死前实际上拟了两份供状：一份是揭发杨金水、郑泌昌、何茂才，记录毁堤淹田真相的；另一份则是在胡宗宪授意之下，自认贪墨修堤公款、监察不力的。胡宗宪交给嘉靖的是第二份，目的是息事宁人，压住毁堤淹田的风口，别牵扯到朝廷。

某种意义上讲，胡宗宪算是生动演绎了剧中“思危、思退、思变”的官场哲学。知道了危险才能躲开危险，这叫思危；躲到人家都不再注意你的地方，这叫思退；退出漩涡后仔细关注局势发展，思考自己之前哪里错了、哪些可以改进，这叫思变。总而言之，在一个“防御性官场生态下”，惩罚机制大于激励机制，因此守住底线往往比抓住机遇更重要。进取看能力，防御看智慧，缺一不可。

# 番外
特朗普宣誓就职的那一刻，二人关系便从兄弟转变为君臣。

5月30日，马斯克最后一次出席了白宫新闻发布会，当时他的表情显得有些落寞、失望和不甘。仅仅一天之后，特朗普突然宣布撤回马斯克支持的NASA局长人选任命。

6月3日，马斯克公开批评特朗普力挺的《税收与支出法案》为“令人作呕的怪物”——这番言论在白宫引起震惊，被视为二人关系破裂的标志。

6月5日，双方在社交媒体上正式互撕，马斯克说出了“没有我你赢不了选举”这样极其犯忌讳的话。成功人士都会坚定认为成功的主因是自己，“没有我你就没有今天”是他们最不能容忍的。

作为特朗普2024年竞选期间最重要的资金支持者，马斯克有着“拥立大功”，但他似乎不清楚一点：帮助总统上位的人往往不能索要重权，否则便容易功高震主。历史上，打天下和坐天下时用的是两批人，套用《雍正王朝》里一句话：“与平常人交往，共享乐易，共患难难；与天子交往，共患难易，共享乐难。”对特朗普来说，马斯克其实是一枚定时炸弹，他在2024年大选期间极有可能掌握了特朗普阵营的一些灰色选举策略内幕，借助其掌控的X平台，马斯克有能力在美国政坛掀起滔天巨浪。

因此，特朗普对马斯克的态度充满矛盾——既有感激，又有忌惮；既要利用他的影响力，又要防止他成为不可控制的威胁。而马斯克显然没有足够的城府去平衡特朗普矛盾的内心。抑或经历过五个月的摸爬滚打，马斯克意识到政府是一个比企业复杂百倍的组织，企业里“开源节流”那套做法根本不适用，因此他对MAGA事业和美国财政状况的前景感到担忧，正提前进行一种策略性的“政治切割”，把自己跟“美版改稻为桑”这场大戏摘干净。

电视剧《雍正王朝》雍正（饰演者唐国强）问邬思道：“大隐隐于朝，中隐隐于市，小隐隐于野，先生想怎么隐？”
邬思道：“臣想半隐。”

《雍正王朝》中，四爷拿下皇位后第一时间回去“处理”自己的顶级谋士邬思道，因为邬知道他太多夺位过程中的阴暗面，最终邬以“半隐”（在一个雍正能随时看到的地方做闲差）为退路获得许可。识人心，懂进退。在权力的世界里，合作是暂时的，忠诚是单向的，冲突才是永恒的。
