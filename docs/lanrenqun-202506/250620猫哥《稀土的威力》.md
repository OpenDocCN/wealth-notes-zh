

# 稀土的威力

250620 猫哥

整理：公众号懒人搜索，懒人专属群独享

前日，欧盟委员会主席冯德莱恩公开指责中方把稀土武器化，对此中方予以毫不客气的驳斥。

那么冯德莱恩为什么失心疯地指责中方呢？

原因无它，中方对稀土出口的管制，让欧盟制造业苦不堪言。

目前中方虽然恢复了稀土出口，但是仅限于民用领域，为了保证中方出口的稀土不被挪用，中方对稀土出口的管制空前严格。

首先，国外制造业要申请稀土出口许可证需要提交一系列复杂的资料，这些资料包括生产线的视频（确认企业生产线确实要使用稀土）、客户名单以及产品订单的详情。

意大利 Speakers 公司供应链主管对记者透露，他们仅仅是因为模糊化处理了客户的订单资料，然后中方商务部就直接停止了审批程序。

对于企业而言，客户名单、订单详情、生产数据都属于商业机密，但是要获得稀土出口许可证的审批，这些资料是强制性要求提供的，否则就不能获得稀土出口许可证。

不仅如此，国外企业获得稀土之后，中方还会要求派员进驻企业检查，确保这些稀土只用于企业自身需要，日本铃木公司拒绝中方派员的要求，然后就被中方停止稀土供应，因此铃木雨燕系列产品停产。

现在中方在稀土出口管理上空前严厉，国外制造业或者完全配合中方的诉求才有可能获得生产必须的稀土，否则就只能眼睁睁看着稀土被卡在海关。

虽然中方要求提供的资料与数据对于企业而言是非常敏感的——客户与订单的泄露可能被竞争对手掌握市场布局，生产线细节可能削弱工艺与技术优势，但是为了获得稀土，西方制造业别无选择！

以上就是冯德莱恩指责中方将稀土供应武器化的缘由。但是冯德莱恩可能忘记了，去年中方向欧盟出口电动车的时候，欧盟以中国电动车可能获得政府巨大的补贴为由要求中国电动车企业同样提供一系列的资料，其中甚至要求中方企业提供核心电池技术的配方，对于不配合交出绝密核心技术的中国电动车企业欧盟蛮横无理的加36%的关税！

所以这个回旋镖现在该欧盟吃下来了。

当然，中方对稀土出口的管制不是为了报复欧盟过去的做法，真正的原因就是严格控制稀土出口只能限于民用领域，防止中方出口的稀土被西方军工企业所掌握。

这是至关重要的一张牌。

大家应该还记得，中美伦敦会晤实际上是谈崩了，中方提高了要价，这个要价让美方最鸽派的代表贝森特拂袖而去，而美方其余代表则宣称，中美没有后续的谈判计划——潜台词就是，对于中方的要价，美方认为没有继续谈判的必要。

当时我就想，美方军工企业对于中国稀土望眼欲穿，为什么中方提高了要价（实际上是与中方稀土价值是匹配的），美方就有底气不谈了？

现在回头看，大概是美方认为，即便在正常渠道弄不到稀土，通过其他渠道也能弄到稀土。

但是，最近中方对稀土出口空前严厉的管制手段击碎了美方的美梦。

所以，6月18日，在伦敦会议上提前离场的贝森特又突然访恢复对美方军工企业的稀土供应。

这就是稀土的威力，只要中方认真，发狠切断对美国军工企业的稀土供应，并且堵住美方军工企业从其他途径搞到稀土的渠道，美方终究会坐下来与中方谈判的，即便中方要价高一点。

为了给未来“3周后”中美会晤铺路，美方官员对媒体暗示，在日内瓦达成的中美互降关税宽限期8月10日到期后，计划再延长90天。

那么90天到期之后会怎么办？只要我们用稀土卡住美方脖子（比如供应是按3个月量限定），美方届时大概率还是会继续延期的。

稀土这张牌的威力在逐渐呈现，未来甚至可能发挥更大的作用。

只要西方慢慢适应中方对稀土出口的管制，未来在地缘政治冲突中，中方可能会获得更大的话语权。打个比方：

比如以色列不顾国际舆论，继续轰炸加沙，制造大面积的人道主义灾难，那么中方未来就有可能动用稀土这张牌来卡以色列的脖子，中方完全可以宣布，任何给以色列提供武器的企业将被切断稀土供应。

只要道义上能站在制高点，使用稀土这张牌威力将巨大无比。
