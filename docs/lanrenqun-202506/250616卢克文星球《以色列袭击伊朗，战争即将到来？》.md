

# 以色列袭击伊朗，战争即将到来？

2025年6月13日凌晨，伊朗首都德黑兰东北部马哈拉蒂街区，一片夜深人静。

修剪整齐的树篱后，一栋栋住宅楼的轮廓在月光下显得规整而森严，这里是大量伊朗伊斯兰革命卫队（IRGC）高级人员及其家属的住宅区。

厚重的窗帘后，这个国家军事力量的最高层大脑，革命卫队总司令侯赛因·萨拉米少将，正在呼呼沉睡，丝毫意识不到危机的到来。

凌晨3时20分，刺耳的防空警报响起，萨拉米少将被惊醒了，他意识到，将有大事要发生。

但一切，已经太迟了。

一枚以色列导弹穿透云层，精准击中了萨拉米的住宅楼，闪光炸开，瞬间吞噬了目标建筑的轮廓，整个住宅楼陷入一片火海。

这场突如其来的袭击，再次令伊朗损失惨重。

以色列本次行动的代号，为“狮子的力量”，出动了200架战机，型号包括F-15、F-16和F-35。

以色列国防军发言人埃菲·戴弗林表示，仅13日凌晨对伊朗的首轮空袭中，以军就出动了200架战斗机，投掷了330余枚弹药。

这场典型的饱和式攻击，瘫痪了伊朗的防空系统。

根据网络流传的视频显示，遇袭区域包括首都德黑兰市、东阿塞拜疆省、克尔曼省、伊斯法罕省和哈马丹省。

值得注意的是，位于伊斯法罕省的小镇纳兹坦拥有伊朗最重要的核设施基地。“纳兹坦基地”遭到了两次精准打击，但从爆炸特征判断，很可能是航空炸弹而非导弹。

有胆子大的伊朗媒体，在现场开直播，直播画面显示，事发地点浓烟弥漫。值得庆幸的是，此次袭击并未引发核泄漏事故。航弹仅击中了地面设施，而核设施的核心区域通常深埋地下，因此影响有限。

损失真正重大的，是人员。目前，可以确认的伊朗关键人员损失有：

- 伊朗伊斯兰革命卫队总司令侯赛因·萨拉米少将
- 伊朗武装部队中央司令部司令、副参谋长古拉姆·阿里·拉希德少将
- 伊朗武装部队总参谋长巴盖里少将（不确定，有关他身亡的消息反转了好几次）
- 德黑兰沙希德·贝赫什提大学核工程学院院长阿卜杜勒哈米德·米努切赫尔（核科学家）
- 沙希德·贝赫什提大学核工程学院教师艾哈迈德礼萨·佐勒法加里（核科学家）
- 伊朗原子能组织副主席兼核科学技术研究所所长阿米尔侯赛因·菲格希（核科学家）
- 法里东·阿巴西（核科学家）
- 伊斯兰阿扎德大学校长马塔尔比扎德
- 穆罕默德迈赫迪·德黑兰奇（核科学家）

损失如此惨重，并不能全怪以色列渗透，伊朗自己粗心大意也是真的。6月10日，以色列秘密向美国通报，即将对伊朗发动袭击。

次日，美国政府紧急下令，要求驻伊拉克、巴林和科威特大使馆的“非必要人员及家属”迅速撤离，并撤出了部分驻伊拉克美军。

美方公开解释称，此举是担忧以色列对伊朗的袭击可能导致局势升级。

这无异于直接向伊朗发出预警：以色列即将动手。

随后，特朗普更是通过媒体放风、社交媒体发文等方式，对伊朗进行各种暗示——若将此比作一场考试，特朗普几乎把标准答案都递给了伊朗。

然而，伊朗的反应却出人意料。

在如此敏感的时刻，革命卫队总司令萨拉米少将竟安然在家中睡觉，而且还是在革命卫队的公开宿舍楼内，显然完全没把特朗普的警告当回事。

要知道，革命卫队的宿舍楼并非秘密地点，普通人通过公开信息就能查到，以色列不可能不知情。

据伊朗媒体解释，萨拉米少将当时在家是因为近期患病休养。

即便如此，临时转移也并非难事吧？

更令人费解的是，与他同住一栋楼的还有革命卫队副参谋长古拉姆·阿里·拉希德少将——这位可没生病，却也未做任何防范。

而巴盖里少将遇袭的地点，竟是同样地址公开的革命卫队总部，但凡他躲在地堡里，以色列也没那么容易清除他，即使用钻地弹。

再看看那些身亡的核科学家，遇袭地点清一色是在家里。

但凡他们移动一下，换个地点，总能活下来一两个。

毫无疑问，伊朗高层低估了局势的严重性，甚至可能认为以色列只会象征性打击，因此未疏散关键人员。他们完全没料到，以色列这次是动真格的。

以色列如此大动干戈，背后自有其深层考量。

## 02

内塔尼亚胡一直希望，特朗普上台后，美国能“重返中东”，帮自己收拾伊朗。竞选时，特朗普对伊朗态度强硬，上台后却换了一副模样。

从今年3月中旬到6月初，美国和伊朗共举行了五轮核问题相关谈判，有重大进展。

《纽约时报》披露，特朗普私下提出，同意伊朗“进行低水平铀浓缩活动”。

实际上，就是承认伊朗浓缩铀活动的合法性，只要伊朗不真的造出核武器，其他的不管了。

伊朗的浓缩铀丰度早已超过60%，距离核武器级浓缩铀的90%丰度，已经不远。

近期，伊朗又建造了一处新的核设施，并对原有的核设施进行了大规模升级。

用伊朗原子能组织发言人贝鲁兹的话来说，“我们将用第六代离心机取代第一代离心机，铀浓缩的产量和效率都将大幅提高”。

换句话说，伊朗距离制造出核武器，仅一步之遥。

这个时候，特朗普要做出重大让步，以色列不疯才怪，必须搞一次大新闻。

把伊朗狠狠炸一次，中断美伊谈判。

2025年4月末，特朗普把国安顾问迈克·华尔兹调离，改任驻联合国大使，等于降职。

我在以前的文章里说过，华尔兹被降职的原因，不是“群聊门”，而是私底下和以色列讨论打击伊朗的计划，特朗普从未授权他这样做。

将华尔兹降职，有几分警告内塔尼亚胡的意思。

5月7日，特朗普干脆和胡塞武装达成了停火协议，根据协议，胡塞只对美国停火，不对以色列停火，不符合以色列的利益。

多家美国媒体报道，特朗普在达成协议前，没有和内塔尼亚胡通气。

内塔尼亚胡很不高兴。还有叙利亚局势。

从4月起，美军从叙利亚撤走了1000人，未来几个月要再撤走500人。这样一来，驻叙美军只剩下500人。

权力是不会出现真空的。美军撤退后，土耳其很快填补了生态位。

过去几个月里，土耳其军队大举进驻叙利亚，不仅控制了北部，土军不仅控制叙北部，更深入中部巴尔米拉，甚至在霍姆斯省和哈马省出现。

6月3日，土耳其国防部长居勒表示，“现在讨论土耳其在叙利亚的2万多名士兵撤离或重新安置的可能性还为时过早。”

这摆明了要“住”在叙利亚。

土耳其在叙利亚的势力扩张，严重威胁以色列的利益。

5月初，在土耳其的支持下，HTS的军队，成功进入叙利亚南部的苏韦达省，扶持了一个傀儡政权，引发了德鲁兹人大分裂。

叙利亚南部的苏韦达省、德拉省，是德鲁兹人的地盘，向来不服从大马士革中央政府。

以色列希望扶持德鲁兹人“自治”，让德鲁兹人作为以色列的北方屏障。以色列国内有不少德鲁兹人，军队里的德鲁兹人就有1000多。

双方是有沟通渠道的，计划成功的可能性很大。但HTS在土耳其的力挺下，成功进入苏韦达省，导致一部分德鲁兹人见风使舵，当了带路党。

叙利亚德鲁兹人，分成了两派，一派主张抵抗，一派主张当带路党，目前后者占上风，HTS已在苏韦达省站稳。

如无意外，土耳其势力很快就会进入南部。

以色列的“北方屏障”计划，遇到了大麻烦。为此，内塔尼亚胡专门召开内阁会议，会议主题叫“土耳其接管叙利亚”。

没有美国帮忙，以色列难以单独抗衡土耳其。

但特朗普对伊朗的态度，对胡塞的态度，以及对叙利亚的态度，都表明他不想“重返中东”。为了围堵中国，特朗普正在把更多的资源调到印太地区。

在此困境下，以色列只能铤而走险，通过制造重大事端迫使伊朗强烈报复，进而要求美国重返中东。

这种行径，恰似未能如愿的孩童在撒泼打滚。

再不满足我，就摔东西给你们看。

## 03

袭击发生后，伊朗紧急向以色列发射了100架无人机，对以色列伤害不大。真正能造成伤害的，还得是导弹，伊朗储备的导弹有数千枚。

接下来，伊朗肯定会对以色列进行导弹打击。

如果和前几次导弹打击一样，效果不疼不痒的话，对伊朗的最后一点“指望”，可以彻底死心了。

以色列这次空袭，简直把伊朗的老底都给掀了。

关键时刻，革命卫队总司令在家睡觉被炸，总参谋长和一众要员也没疏散。更可笑的是，美国提前几天就通风报信，伊朗高层却连基本的人员疏散都没做。

这种水平，已经不能用拉垮来形容，再次给提倡“中俄伊三角”的人泼了一盆冷水：

伊朗这个“盟友”，短期内根本指望不上。

伊朗既没有足够的实力，也缺乏斗争的决心，加之内部问题重重，所以总是首鼠两端、缩手缩脚。

继续和伊朗做生意没问题，但搞“大三角”就算了吧。

中东这盘棋，终究还得靠自己的实力，而不是指望不靠谱的队友。多和阿联酋、沙特等国深化合作，都比单纯押注伊朗要靠谱得多。

中国目前在伊朗，走的就是这一步。伊朗的拙劣表现，正应了那句话：不怕神一样的对手，就怕猪一样的队友。
