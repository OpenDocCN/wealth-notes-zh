

# 驾驭消费：买相处

欢迎打开，我是蔡钰。这里是栏中栏《超级图景与问题意识》。

接着上一讲的话题，我们来讲第三个消费维度的建议：买相处。

买相处其实就是买交情，这是滋养我们生命里的重要负熵。但我担心直接跟你说买交情，你会把它简化理解成简单的情感消费，就像课程前面提到的那个租妈妈的案例。我们说的买相处，不是去简单雇个旅游搭子，或者体验cos委托，收获现成的情绪资源，而是以消费为引子，投入时间、交换思想和心情，来增强交情。

## 跟物品结婚

买相处，首先是跟物品的相处。

我给你推荐一种消费心态，叫“跟物品结婚”。你可以在购买最常用的5-10件耐用品的时候，比如水杯、沙发等等，调用这种心态来做选择，假设这辈子只能用它了。你会发现，因此选出来的产品会截然不同。

物品买到手之后，也请你保持这种结婚心态来使用和爱惜它，和它建立交情。如果我们身边有这样几件十来年甚至二十年交情的物品，它们就能从时间上帮我们强化对人生脉络的感知，也帮我们增强对“附近系统”的主场感。

这样的物品，可以叫做长情物品。如果你要更换，也请怀着“离婚再婚”的心态来进行。你如果有更多的兴趣，不妨整理一下自己每次更换物品的心情和思路，这是在帮我们理解自己，也是在以物为节点记录人生。

在长情物品之外，我们还可以按周期留出一笔新奇趣预算，来帮我们创新体验。什么东西有新意？什么消费在流行？什么物件让我看见了眼前一亮、心情变好？在新奇趣预算内，买它。

在长情物品和新奇趣物品之外，出于对补给生命力的考虑，我还建议你对“商务礼品”这类产品，制定常态化的转手纪律。这也是在给咱们的物资系统搭建“耗散结构”。

具体来说，如果你在事业上经常有人情往来，肯定经常收到茶叶、月饼、日历、雨伞、水杯等各种企业礼物。如果一个礼物没能当场被你归入前两类物品里，我建议你把它挂闲鱼，或者送给点头之交的邻居、门卫、保安、保洁、外卖小哥、小店老板娘。

你还可以在楼道里放个纸箱子，写上自家门牌号，鼓励邻居们自取。

为什么要这么做呢？我有个偏见：商务礼品和各种积分兑换的礼物，是对我们选择权的一种封印。它把我们的消费乐趣锁死在了送礼者的眼界，让我们腾不出人生去拿真正喜欢的笔、打真正喜欢的伞、喝真正想喝的茶。

所以，把并不合意的礼物转送出去，既是还给自己选择的自由，又是在借它们跟附近建立小小的交情，这是这些礼物对我们的真正加持。

# 消费+1

说到这，我们进入第二种值得买的相处，跟他人的相处。

其实，我们约朋友参与共同活动而发生的一切消费，无论是吃饭、看戏、旅游，都是在买相处。

还有一种消费心态，很适合我们用来跟人相处，我叫它“消费+1”。

什么意思呢？我每次购买了有意思的物件，无论是书、零食、文具、日用品，用过之后通常会在自己的朋友里想一圈，看看它还可能适合谁，然后再下一单寄给对方。

过去一年，我给朋友们送过书、课程、鞋子、蓝莓、便携显示器、净化器等等，有次甚至卡着时间，买票把异地的朋友送进了电影院。反过来，我也收到过不少带有朋友脑洞和心意的意外物件。

这样一来，这个小礼物可能就替我们向朋友表达了没空见面的歉意，或者成为下次我们见面的谈资。这次做《超级图景》这个栏中栏，我给一些催更过的朋友寄了一份小礼物，就也是这个思路。

“消费 +1”这个思路，最直观的好处就是，维护了朋友间的交情。但更重要的是，相同的商品，在我们和朋友之间创造了一个共同信息场，来互相分享使用感受和衍生联想。过程中，我们就是在识别和强化跟对方的“志趣相投”，在扩大和增强我们的价值主场。

即便对方暂时给不出反馈，你也可以告诉他，你认为这个物件跟他合拍的理由。这会让对方不止收到你对他的关心，还收到了你对他的新理解。谁能抗拒来自他人的善意连接呢？于是，我们自己虽然可能只使用了这个物品一次，但却在“消费 +1”的操作当中，获得了双倍的信息增量和情感增量。

信息增量和情感增量，就是我们通过“买相处”而真正追求的负熵。买相处的本质是，通过消费在我们和他人之间建构一个共同信息场，让信息增量和情感增量有机会碰撞诞生。

基于这个逻辑，我们也可以带着问题意识去约新朋友见面吃饭，问他近期的特别经历和特别感触，问他对流行事件和公共议题的看法。

如果我们面对的是再熟悉不过的家人和朋友，就没有必要在日常生活里塞满那么多的深度思想交流了，我们可以侧重关注哪些消费能帮我们增进情感交流。

从这个角度看，你会发现，包饺子、大扫除不是简单的家务分摊，而是非常好的家庭团建项目，它们需要全员参与，需要配合，有价值共创，能凝聚认同。我们要做的就是把面粉和肉馅、把奇怪的围裙和手套，当成团建道具来买。如果你能偶尔给这种团建准备一些奇特的奖杯、奖状、颁奖词，会更能增加家庭系统的情感能量。

这两年，中国民间还出现过两类特别有趣的拉横幅举动。

第一类是高考考生家庭拉的横幅。标志性事件是2024年西安一个张姓人家，全家8人出动来接考完了最后一门英语的儿子，还拉出了一条横幅，写着“终于可以谈恋爱啦”，这在全网上了热搜，“张同学，全网都知道你可以谈恋爱了”。

另一类是人们开始在吃饭聚餐的时候拉横幅。什么“刘秋香女士喜提30岁生日聚会”“祝贺王富贵脱单失败庆功宴”“三年二班老同学茶饭研讨会”“球友微信群2024年尾牙争霸赛”等等等等。

这都是把相处给事件化，用叙事赋予意义，用消费创造共同信息场，创造信息增量和情感增量。

# 总结

好，这两讲，我们讨论了三种通过消费来补给生命力的操作思路，分别是：

- 买休息。借助消费，完善我们的休息工程，增加我们个人系统的存续时长。
- 买事件。通过消费，参与系统的共同事件，增强系统认同和自我认同。
- 买相处。用消费在我们和他人之间创造共同信息场，促进信息增量和情感增量的诞生。

如果你是个精明的创业者，你肯定已经意识到了，这三件事反过来当成供给，就是绝佳的创业路径。

《商业参考》在第三季曾经提出过一个产品模型，叫门票与乐园。意思是，在精神消费、情绪消费领域，我们购买的实体产品，比如泡泡玛特的潮流玩具、酒馆里的精酿啤酒，本质上都是在购买一张通往某个精神乐园的门票，我们真正消费的是凭票进入的那个乐园。

在需求端先行觉醒、开始尝试自创供给的今天，我们亲自尝试买休息、买事件、买相处，也是在自行搭建精神乐园。当前这个产品模型还处在一个更早期版本，叫“建材与乐园”。我们这些通过自主定制消费方案来补给生命力的人，何尝不是自己精神乐园的建筑师呢？

在过去很多年里，商业市场站满了销售高手，都在研究怎么把梳子卖给和尚。但在今天的时代拐点下，在完备的产业链旁边，我们的创业者应该有底气扔下梳子，去关心和尚真正需要什么。我们在日常善待自己的过程中，没准就能把善待自己的方案，扩展成服务某一群受众的系统能力。这一轮中国服务业崛起的过程，“精神管家”这种新职业和新事业，没准儿也会涌现出来。

当然，梳子也不是完全不能再卖。今天无论你在制造什么产品、提供什么服务，都可以把它看作一张门票，去理解和想象，这张门票能带用户通往怎样的精神乐园。乐园里真正售卖的可以是休息，可以是相处，也可以是事件，用这些内涵，来真正打动你的用户。

要搭建这样一座乐园，我们的个体能力是有限的。但没关系，我们不妨承认自己的有限，去跟系统对齐，跟系统成为利益与价值观的共同体，再借助系统资源，给自己搭建一个巨大的供应链和信息网络系统。

在这样的系统里，学会区别看待自己的能力和目标，变得非常关键。只要我们追求的目标同时也能替系统、替同行和上下游实现熵减，就能够争取到更多的资源的支持，把我们的目标当作共同愿景，共同实现出来。这个过程，我们就是在建构自己的价值和利益主场，成为产业链的链主了。

YouTube 网红甲亢哥做高质量的中国旅游直播，DeepSeek R1 发布后快速被整个产业界部署到本地，“胖东来同款”成为淘宝商家的信用标签，《哪吒2》身后也有 138 家中国动画公司的支持，都在不断证实这个逻辑。

切回到社会图景里。今天渴望身心回血和寻找意义的人们，可以通过买休息、买事件、买相处的方法，获取短效的情绪资源和心理能量，恢复部分生命力。但想要真正尝试解决问题，我们还是得退回到时代图景的更底层，把空心困境放进图景里去理解，找出“怎么了”后面的“怎么回事”。
