# 一、项目思路

## why - 市场 - 为什么要做，为什么在那个时候做？

2023 年，某个在我国曾经火过的日本动漫 IP 的电影在国内上映。

我是该 IP 的老粉。在电影上映后，刷刷社交媒体，参加线下活动，与同 ip 的粉丝交流几句。

那是一个被粉丝们火一般的热情所包围的夏天：

- 粉丝群自发组织的包场观影活动和其他线下与线上活动（"only 展"、角色应援、cp 应援）在全国各城市遍地开花，每周都有若干场；
- 电影的上映激活了该 IP 原本的 80 后至 90 后的老粉，同时也拉新了一批 18 岁~25 岁的热情高涨、创作欲与购买欲旺盛的年轻粉丝；
- 与上映前相比，电影上映后的 1 年时间里，各内容平台中（lofter、微博、小红书）此 IP 的二次创作内容的数量指数级增加，数据大幅提高；
- 很多粉丝自制并售卖该 IP 的二次创作周边产品，做得好的会有一定销量（几十~几千件不等）；核心粉丝当时购买二创产品花销大致在几百~几千元/月；看到喜欢的制品就会马上买，因为知道喜欢的制品可能只做这一批，过了这个村没这个店；

粉丝购买自制周边的心理因素：

- 官方周边种类少且丑，群众需要高审美、多风格、多种类、多生活场景使用的周边；
- 应援角色，与对角色和作品的认同感、信念感、展示/炫耀/攀比等
- 对二次创作产品的喜爱：圈子内有很多二次创作写文、画画、创作漫画的才华横溢的创作者（二次元圈内尊称"老师"），她们的作品提供了巨大的情绪价值：审美好/有趣/感动
- 线下活动中的社交需求也是购买自制周边的原因：见面时互换"无料"（即互送周边作为礼物）是二次元圈子内潜在的规则与礼节；痛包展示（痛包即放了动漫周边制品的包）；线下打卡拍照等等

我身边的粉丝们购买过的粉丝二创自制周边包括：手办，队服，球鞋，吧唧，透卡，色纸，明信片，小卡，卡套，卡册，立牌，同人本（小说集或画集），光栅卡，海报，打卡棒，日历，棒球帽，t恤，卫衣，金属徽章，钥匙扣，发饰，毛巾，围巾，水壶，玻璃杯，碟子，包包，动物塑玩偶，角色棉花娃娃，棉花娃娃的娃衣，贴纸，丝巾，收纳盒，御守，塔罗牌……你能想到的每个小件生活用品都有被做成周边并变现的潜力。

举几个该 IP 圈内普通人轻投入赚到钱的例子，

一位大学生做了一个角色玩偶，趁着作品热度高时开卖，卖了几千件，单件产品利润 5~10w 左右；

一位审美很好的女生设计了角色姓名的丝巾，80 元左右一条，一共出了十几个角色的，我身边的粉丝们都是至少几条几条地买，复购很高；

一位大三女生在几个热度高的动漫IP圈子中做了多款二次创作周边，以及举办各IP的多场线下活动，2023~2024年的她盈利推测在40~100w左右。

## 总结：

短时期内的强需求：动漫IP的粉丝黏性很高，而该IP电影上映及二创内容的发发达拉新了用户，也激发了大家的购买欲望；而这个热潮是有时限的。

粉丝自制周边市场是高度有机化的，即每样产品量不大，但可以开发多个品类卖给同一批人，很适合爱好者以极低的成本起步练习做生意 -- 如何做产品 + 如何做营销。

在2023年之前，我在二次元圈子里只是玩一玩看一看，并没有想通过爱好赚钱。而这个夏天，我体会到了这股强需求的热浪，看着身边的粉丝姐妹、阿猫阿狗普通人们纷纷自制周边开卖，于是我抱着「练一练手，为爱发电，亏不了几个钱」的预期，决定亲自潜入水下，试试水温。

## what - 产品 - 普通的我，竞争优势是什么，能做什么产品？

在二次元圈子裡，我是一个非常普通的人：不会画图，不会写小说，不混圈。

因此在我看到需求，想要售卖产品的时候，首先想的是：我不是圈内的 "老师" "女明星"，即社交平台账号没有粉丝积累；如何在这种情况把东西卖出去？

关键在于精准找到用户聚集地，并向他们展示我的产品（即使是0粉）。下一节和"实操经验"章节，会详细讲我是怎么做的。

我略有优势的地方是什么？

1、能做简单的设计，有审美，用好东西。（于是后来 90%的设计图都是自己的，没花钱约稿）
2、抠细节不嫌麻烦，有执行力。（后来做定制图案包包时，老板认为做不出来，我自己买原材料拼出来图案把样板拍照发给他，老板就同意做了）
3、在该IP圈子里浸泡了几年，对"太太"们很熟悉，并且知道谁的文风/画风/设计风格为何会被喜欢、被追随；尽管不写文不画图，但我在小红书和lofter上写过几篇该IP圈子内数据不错的分析型文章，知道什么样的内容受欢迎。

前两个指向做产品，最后一个对应做营销。

在分析了市面上大家卖的制品后，我飞快地制定了产品策略：做日常每天可使用（show给别人看到）、有潜在需求、别人还没做、而我能做的品类。

从生产角度，很关键的一点：这个商品要能在控制成本的前提下，可以小批量定制。起订量1件～100件，最高不超过100件。

最终我做了几个大热角色的围巾和毛毯，后来又做了一款我推的 cp 的丝绸包，以及透卡、立牌等常见的动漫周边产品。选择的人群是本 IP 圈子里购买力最强的一批人。

## how - 营销 - 人群在哪里，怎么营销？

上章节提到，我的 Lofter 、微博、小红书账号都基本没有粉丝基础，如何把东西卖出去？

当时作为一个做生意的新手，我会做的是：在有精准人群的地方，二话不说就是发（最来混最重要的是出来）；并尽量把宣传内容在视觉上做得更好看，在文字/标题/封面上做得更吸引人。

根据我在该 IP 圈子里浸泡了几年的了解，当时的精准粉丝出没于下面几个地方：

线下活动：综合性漫展（如 Comic Up 展会），该 IP 的 only 活动，线下观影活动，角色应援活动，CP 应援活动。

线上渠道：

- 微信群（线下活动群，线上云活动群，观影活动群，聊天群，买制品群）
- 小红书（算法精准推送）
- Lofter (动漫二创内容平台, 算法精准推送)
- 微博 (可以发到 IP、角色、cp 的超话广场里和相关 bot 账号里)
- QQ 群 (买同人本群, 角色的棉花娃娃购买群)
- 其他粉丝会出现的内容平台 (例如爱发电, ao3)

思路大致定了,我决定不想那么多,就是一个干。执行力和速度很重要。执行的具体细节在"实操经验篇"分享。

# 二、在二次元领域赚钱,与别的领域有什么不同?容易在哪里踩雷?

在二次元冲浪,可能会遇到很多很有意思的情况:

- 对面疯狂骂你的厕妹,可能小学还没毕业;
- 追文追了很久的太太说要断更一段时间,因为要去中考了;
- 在转发区提了一下逆 CP,被群众讽刺了 1000 个转发;
- 买了上百个棉花娃娃和娃衣的女生,她的孩子都快小学毕业了;

孩子成熟化，寻找伊甸园；成人儿童化，守在永无乡；因为同样的喜爱而聚在了一起。

这个圈子里有很多自己的规矩和文化。那么如果想在其中做生意，有哪些需要注意的地方呢？

## 1、不同之处：

- 强情感驱动 & 社群属性：购买基于热爱和认同感，而不仅仅是实用。口碑和社群传播效应极强。
- 对“柄图”(画稿)的极度依赖和溢价：画师知名度、画风是否受欢迎直接决定销量和定价上限。
- 时效性与 IP 热度强绑定：电影上映、周年庆等有事件热度的活动期是黄金期，热度下滑后需求锐减。
- 「为爱发电」与商业化的微妙平衡：定价过高或过度商业化易引发粉丝反感；需要找到“合理回报”与“同好分享”的平衡点。
- 版权问题的灰色地带：同人创作的边界需要时刻注意，必要时请咨询律师。

## 2、容易踩的雷：

- 版权雷区：直接盗用官方素材、未获授权使用他人画稿、设计过于接近官方或知名同人作品。
- 品控雷区：周边制品粉丝对质量（尤其颜色、材质、工艺细节）非常敏感，品控不好口碑瞬间崩塌。
- 定价雷区：定价过高（被骂“割韭菜”）或过低（破坏市场、显得廉价、自己亏本）。定价时需要考虑的因素：成本，合理利润，圈内同类产品参考，粉丝承受力，画师溢价，稀缺性和审美性等。
- 沟通与服务雷区：发货延迟、信息不透明、售后差、在社群中态度傲慢等。
- 库存雷区：对热度判断失误，生产过多造成压货。小批量定制的重要性再次体现。
- 「混圈」文化雷区：不了解圈内规则、黑话、敏感点（如 CP 逆位、角色争议），容易说错话做错事。

# 三、实操经验

## 1、产品生产篇

几个月时间内，我做了几个不同品类的动漫周边制品：

- 围巾和毛毯
- 丝绸闪钻包包
- 透卡及立牌

其中第三项是很常规的动漫周边制品，淘宝拼多多随便找一家做就行，起订量也很灵活；第一项和第二项都麻烦很多，因为需要找工厂定制化生产，且不是常见的动漫周边种类。

拜这件事情所赐，我学会了日后对我来说一件很有用的通用型经验：如何从无到有做出一个任何种类的实物产品：涉及产品设计、选工厂的原则和渠道、生产流程和货期把控等。

从0开始做一个普通日用消费品，生产端基本是这几个环节：设计产品 - 选工厂 - 打样 - 做大货

### 1.1 围巾及毛毯-设计与生产

在大致定好选产品品类的策略后，我在1688广泛地找想做的各种品类商品的制作工厂，例如T恤，棒球帽，丝巾，卫衣，发饰，戒指，皮标，绣标，收纳袋，帆布包，皮包，各种材质的围巾和毛毯等等，看了很多品类和工艺。

对从0开始的小白来说，以目标为导向是最快的学习方式。由于已经是秋天，又看到大家在某个买东西的微信群里的强烈要求，于是决定先做单角色的围巾和毛毯。

我飞快地用Photoshop画好了设计图。拿着设计图问了几个圈子里的朋友的意见，以及工厂老板生产方面细节是否有困难，又改了两次形成终稿。

设计图是自己做的，没有花钱约画稿（因为想省钱+认可自己的审美），以及作为i人实在是懒得当甲方去约柄图（即画稿），省下了沟通成本和精力。

在对比了几家工厂的样品、能做的工艺、我想要的材质、工厂起订量、打样报价和大货报价后，给几家老板看过设计图，最终确定了一家能做我想要的效果和材质、报价不高、起订量也能接受的工厂，开始了选纱线、选色、以及打样环节。

我选的工厂做货的质量挺靠谱，分给我进行设计图到生产对接的师傅也很细致，打样样品和大货的质量最终都很好。但怎么可能这么完美呢？我遇到的问题在另一个方面上：正值深秋，工厂那边的订单很多，最终打样的工期拖后了一段时间；由于围巾和毛毯是季节性产品，对之后的销量有些影响。

我选的围巾和毛毯在二次元周边领域属于 hard 模式，因此也没人跟我竞争，而根据生产改设计图、催打样、生产对接等花了不少时间。

### 1.2 丝绸闪钻包-设计与生产

在第二年春天时我刷淘宝，看到一款缎面仿丝绸材质、闪钻卡通图案的包包。黑色和粉色两个颜色，是很多女生会喜欢的款式，价格便宜，又显得高级，非常有性价比。

我上1688找到了做这款包包的源头工厂，工厂给的报价是几十个订量按20几块钱一个，起订量也很灵活。对于我很理想，有利润空间且定价不高，把包包自带的闪钻图案改成我画的我cp的定制图案，应该挺受欢迎。

我没有绘画基础，摸索着画了两个尽量简单的可爱大头Q版图案，打算印在包包正面；包包背面印我cp名字的罗马拼音加上爱心图案。

感觉自己画得挺可爱，正美滋滋呢，把图案拿给老板一看，天塌了：老板拒绝了我，不给做，说这个图案用钻拼不出来。这可怎么办？懒人微信：lazyhelper

我想了一会儿，拿着包包的样品对比着我的图案的大小，估计着其实是能做出来的，我画图做设计时有考虑到如何用钻拼成线条和图案细节。只是图案复杂一些，比较花时间，工厂那边觉得有拼不好的风险，所以不想找打板师傅把图案拼出来。

于是我上1688买了一大堆跟包上同样大小的钻，以及两张透明塑料膜：一张画上我设计的图案，另一张放在第一张上面，用钻把图案拼了出来，拍了照给老板看，于是解决了这个问题，老板同意打板和后续生产。

后来这款包包在淘宝上和两次线下活动中都卖得不错。

### 1.3 透卡及立牌-设计与生产

这两类制品在动漫周边届都属于便宜好做又成熟的产品了，生产很简单，淘宝或者拼多多随便找即可。

由于我不会画画，也懒得约稿跟人对接，柄图（画稿）是厚着脸皮跟一位平时爱好画画的同学朋友要的授权。

朋友是业余画着玩的，按照她的水平和图的复杂度按市场价格给约稿费用的话大约是 200-400 元/张，我给她红包她没要，于是后续我给她寄了我做的毛毯围巾包包和立牌等周边，也是同担之间的一种正向流动。

### 关于利润空间：

吧唧，透卡，立牌，色纸等这几种类型的动漫周边制品是大家做得最多的，利润空间很足。

例如透卡我一次做几十张的成本大概是 1 元/张，量大会更便宜。当时我的售价是 8 元/张，属于正常价格范围。

### 关于约稿+自己开发制品企划的模式：

自己不会画画的话，可以找画师约稿，之后自己再拿画稿做制品企划。稿价根据画师水平，几十到上万元/张不等，圈子里很多创作者和画师靠这个赚钱，做二创周边制品企划的人也能赚。

画稿的成本是该类型周边制品的一个占比比较大的成本，剩下的大头是生产成本。所以更多的情况是创作者/画师用自己的画稿做了各种周边制品去卖，等于用人力省掉了约稿成本，但也会付出更多时间（设计生产制品、宣传等等）。

有的企划是一张柄图（画稿）做成好几个类型的制品，即约到一张稿子后可以一鱼多吃，那么单类制品的成本就降低了。

例如我最近买的一套某IP的粉丝自制周边，定价是100元/套不拆卖，里面是一张色纸+一张透卡+一张明信片+一块流麻+一个吧唧，柄图约的是该IP圈内一个很受欢迎、排期很满、接官方周边稿图也接同人谷稿图的画师。她的画稿出的周边，很多人是直接盲买的（例如我。因为能画出角色神态精髓+很好看）。

她的画稿价格，无背景半身图在1.3w~1.5w左右，属于稿件价格高的。这个类型制品能卖出量的关键是柄图是谁画的、好不好看、稀不稀缺、是不是大家喜欢的类型，以及企划能力（比如整套周边的理念是如何设计的，对柄图的附加创意再设计）和营销能力。

### 产品生产-复盘：

对于小批量定制型商品，起订量，及其报价，是个特别特别关键的数据，决定了你一开始投入的成本高低，及会不会亏本。

大多数常见动漫周边制品（吧唧、透卡、立牌、色纸等）都可以一件起订，量越大单价越便宜。如果你的目的是在二次元领域赚钱且不容易亏本，请从这些常规制品做起。这类制品，柄图受不受欢迎、是谁画的，是最关键的。

如果再选一次，我会先做刺绣棒球帽和T恤，这两个会比我选的围巾和毛毯更好卖，生产对接环节也更简单。

工厂在忙季的出货时间说不准，生产肯定会优先量大的和老客户。小批量定制产品的打样和生产请打出时间余量。工厂出货品质也有可能跟预想的有差别，请多问几家工厂，多要样品，多跟工厂沟通。

产品做出来了，但作为 0 粉素人，怎么让目标客户看到并愿意购买？这就是接下来‘营销篇’要解决的挑战。

## 2、营销篇-素人小红书 0 粉起步，怎么把东西卖出去

### 2.1 线下渠道篇

在线下渠道，我的策略很简单：发传单「地推」。

那几个月赶上一些线下活动，比如综合性漫展 CP30，上海的该 IP only 活动，北京的该 IP only 活动，粉丝办的各种线下活动例如音乐会，角色应援场，CP only 应援场，观影应援场等。

我设计了传单，上面有设计图、宣图以及 VX 群二维码，带着产品的打样，在线下活动里看到同担就过去互换无料礼物，然后聊两句，顺便也塞个传单给大家。

如果是有其他人摆摊位的情况，我会给每个摊位上的每个老师发一张传单，顺便交换一下无料。

有两个活动是我在活动里申请摊位，自己摆了摊儿在活动上出售的;

CP30 那场活动我没有去，采用了寄送模式，把产品寄到 IP 在活动上的「街道负责人」那里，他们负责帮忙卖产品，多余的没卖完的再寄回来。

通过「线下地推」的方式，我找到了我的第一批客人，这个渠道的人数大概占了 30%-40%左右。

### 2.2 小红书篇-素人 0 粉起步做营销

在小红书，我的策略是：精准找到用户聚集地 - 小红书算法帮我完成了「精准」，并通过高颜值内容向他们展示我的产品 - 即使是 0 粉。

在当时发了一些笔记之后的感受是，小红书内容推送更倾向于素人，很「平权」，且推送精准，对想通过做好内容拿到免费流量做个小生意的商家很友好。

我在小红书上通过做内容获取免费流量，最终引过来的人数大概占了我的买家的40-50%左右。

我账号的定位：账号的定位是该 IP 爱好者，内容着重我喜欢的几个角色和 CP，并且这几项角色和 CP 是与我要卖的产品相匹配的。

内容分配大致是营销内容+非营销内容，营销内容的比例在 30-40%左右。

非营销内容发得兴之所至，作为爱好者，手头有什么、想发什么就发什么：

- 该 IP 角色的 MBTI 分析：从大概 2022 年开始，MBTI 内容很火，而我 IP 的圈子里除了我，没有写这个写得好的。于是我把我在 lofter 平台上的一个数据不错的 mbti 分析长文章按角色拆开，重新润色一下，每个角色发一篇，在小红书重新发了一遍。其中有一条在本 IP 圈子里数据很高。
- 去线下活动经历的 vlog 和图文
- 买周边的开箱视频

## 买同人本和其他谷子的照片和感想

分析人物性格，看文的感想，对作品的感想与分析，对 cp 的感想。

晒娃（该 IP 粉丝自制的棉花娃娃），求娃（想买娃买不到就在小红书上问）。

热度话题讨论（例如跟 CP30 漫展相关的讨论）。

## 营销内容：
- 做货过程：选设计图，选色板，大货细节拿不定主意请用户帮忙做选择等。
- 设计图+宣图+开团说明。
- 打样照片拍摄：围巾/毛毯会加穿搭或者其他使用场景（如沙发等），包包加了穿搭去拍，也有当作痛包/娃包的摆拍。
- 大货照片拍摄。
- 发货过程拍摄。
- 线下活动摊位照片、感想等。

在小红书的娃群、和该 IP 的其他爱好者群里进行互宣或者宣传。

## 视觉是关键：
前面说"把宣传内容在视觉上做得更好看"。

当时为了给产品拍照，我简单学习了怎么布景，怎么打光、构图之类的，买了一些拍摄道具和设备，也参考了怎么穿搭配产品更好看等。

现在再看，做得很一般，不过小圈子里要求也不太高，做我这个的没什么竞争者，大家就凑合看看也能卖。

# 四、总结
这是一个由爱好转为认真做小生意的项目。

作为普通人，我「0粉起步赚到钱」的关键步骤：
洞察强需求 -> 发挥自身优势选品 -> 解决生产难题(小批量) -> 精准渠道(线上+线下) + 高颜值内容营销(尤其小红书) -> 做好基础服务（售后发货等）

「为爱发电」是我的初心，找到「爱好」与「生意」的平衡点，合理的商业逻辑让这份爱走得更远：满足同好的需求，合理定价，做好服务。

5W+ GMV 对各位大佬不多，但对一个利用业余时间、0粉起步、尝试新领域的素人来说，是很好的起点。

从个人收获的角度，这件事我积累到了 4 点：

- 1、作为做生意初学者，用非常少的成本，跑通了做一个小生意的产品+营销的全流程。
- 2、掌握了从0开始做任何实物产品的流程，例如做女装、开发硬件产品等等，大体的流程一致，只是领域与难度、细节颗粒度不同。
- 3、对小红书的算法、推送逻辑、什么内容受欢迎有了"体感"，为我 2024 年在另一个领域起号时打下了基础：在那个号上我做的第一条视频就是爆款，当天涨粉 1.3w。
- 4、使我兼具「生产者视角」与「用户视角」，去看待所有的泛二次元/年轻人情绪经济生意。

如果你有冲动与热情，且发现了机会，请立即行动起来。请从细分领域的「小」需求出发，从「小」处着手（比如二次元领域就从最简单的透卡、吧唧开始），在行动中不断学习与迭代，积累属于自己的长坡厚雪。
