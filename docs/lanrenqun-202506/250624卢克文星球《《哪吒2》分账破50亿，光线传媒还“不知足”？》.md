

# 《哪吒2》分账破 50 亿，光线传媒还“不知足”？

当《哪吒2》全球狂揽 159 亿票房，大家为国产动画扬眉吐气欢呼雀跃时，电影圈内部却因“钱怎么分”吵翻了天。

在第 27 届上海国际电影节开幕论坛上，《哪吒2》的出品方、光线传媒的老板王长田现场倒苦水：“100 块票房里，片方只能分到 33 块，扣掉成本，赚的钱无法维持行业基本投入！”

“王长田不满分账比例”顿时登上热搜，有人吐槽：按照王长田说的分配比例，一部电影分账破了 50 亿，光线还不知足？但也有人拍手叫好：王长田这话说得实在，直接把国产电影“盛世危机”下的难题给捅了出来——光线虽然在《哪吒2》上分了 51 亿，但这部片子做了 5 年，光特效镜头就 1900 个，成本高得吓人。票房破纪录背后，原来是制片方在咬着牙硬扛。

这件事上，到底谁说的对？

内地电影票房分账这事儿，咱们来算笔明白账：

一张 100 块的电影票，先被猫眼、淘票票这些购票平台以“服务费”名义切走 10 块。（有意思的是，光线传媒是猫眼娱乐的股东之一，这钱相当于左口袋进右口袋。）

剩下的 90 块，国家电影专项基金和增值税要扣掉 8.3%，差不多 7 块多。然后剩下的 82 块多，才是片方和影院分账。按行业规矩，片方拿 43%，算下来 35 块左右，院线拿 57%。但片方这 35 块还得扣掉发行费、宣传费这些，最后实际到手大概 33 块。巧的是，《哪吒》的发行也是光线自己做。

七七八八算下来，光线至少能拿 46 块，可王长田喊冤的时候，绝口不提猫眼和发行赚的钱，光强调片方只拿 33 块，说分账不公平。

难怪大家说资本家贪得无厌，票房越高眼睛越红，完全不管影院死活。

但王长田有自己的说法。

作为《哪吒》的出品方和制作方，光线传媒承担了电影制作的几乎所有工序。作为发起者，光线既要找投资、拉赞助保障项目初期创意落地，又要聘用制作人组建主创团队，从大纲、剧本到拍摄、后期特效全程跟进。

这中间还要过审拿“龙标”，这关要是过不去，前面的投入就全白费了。可以说，片方投入了 100%的成本，承担着 100%的风险，但最终却仅拿回 33%回报，实在不太合理。

尤其是，很多人只看到《哪吒 2》用 5 亿成本撬动 160 亿票房，觉得光线赚得盆满钵满，却忽略了这家公司押注国产动画的“十年孤勇”。

时间倒回 2013 年，中国动画市场还被“喜羊羊”式低幼片统治。光线传媒的王长田却从一档娱乐节目的收视数据里嗅到转机——动画片 18-35 岁观众占比高达 41%。2015 年《大圣归来》三天破亿、最终斩获 9.56 亿票房，证明了成人向动画的商业潜力。同年，光线正式成立彩条屋影业，并找到了那个“用一台破电脑做出获奖短片”的狂人——饺子。

“推掉所有外包，我们用五年赌一部作品。”彩条屋 CEO 易巧的承诺，让饺子团队得以不计成本打磨《哪吒》，剧本狂改 60 版，最终成就了“我命由我不由天”的国漫神话。而光线这十年间推出的《大鱼海棠》《姜子牙》《深海》，无一不是用真金白银为行业试错。

但辉煌背后藏着产业的隐痛。《哪吒 1》片尾滚动着 100 多家外包公司名单，易巧坦言：“这是中国动画的悲哀——没有成熟产业链，只能靠人海战术。”

从零搭建动画工业体系，光线既要当“投资人”，又要做“铺路人”。

正因如此，王长田必须“贪得无厌”。早在2016年，王长田在公司内部公开信里写道：“只有商业上的成功，我们才有可能在电影行业立足，才有机会站在电影行业的中心而不是边缘，才有能力去补贴、分担那些亏损的影片，那些创新作品和艺术作品。”

分账问题并非王长田首次提及。早在《金陵十三钗》时期，张艺谋就曾联合制片方要求将分账比例从43%提升至45%，但遭到院线强烈抵制。最终在电影局介入下，部分影院通过提高最低票价实现调整。

王长田推崇的美国分账模式，通常采用阶梯分账：首周片方可分60%～80%，随后每周递减约5%～10%，到第四周大概降至25%。综合算下来，片方最终能分到50%的收益。

不过，他忽略了美国院线约85%的利润源自零食饮料销售，对票房分成的依赖程度较低。以AMC院线为例，其爆米花毛利率超过80%，“爆米花经济”赋予了院线更大的让利空间。反观国内，中国观众没有这种消费习惯，而且当下多数影院经营处于亏损状态，根本无力做出让利。

片方称要回馈影迷，加大投入打造诚意之作，为此拼命多赚钱、在产业链中多分钱。

然而，想出精品电影，片方多投入就一定有效吗？

未必。

2024 全年总票房合计 425.02 亿，还赶不上 2015 年的 438.1 亿，市场退步到十年前已成行业共识，但电影成本根本压不下来。

万达电影董事长陈祉希坦言：“十年前我拍《煎饼侠》只需 1200 万，如今要达到同等水准，1.2 亿都难。”

这一现象背后，有新锐导演追求更高制作标准导致投入增加、周期延长的原因，但更多是电影已经沦为资本“击鼓传花”的游戏。

2010 年代初期，互联网的热钱涌入电影业，票补让不少人花 9.9 就能看场电影，中国票房规模十年间翻了 5 倍，也让不少资本盯上了电影这块蛋糕。

这期间催生出了流量明星 + IP 的玩法，像郭敬明的《爵迹》系列，豆瓣评分虽然只有 3.8，却能斩获数亿投资，知名演员和 IP 价格跟着水涨船高，导演和头部演员话语权与定价权增大，但电影收益押注在某个人身上风险极高。《爵迹》就因为口碑崩塌，最终票房惨败收场。《爵迹2》因为演员问题，无法在院线上映。

保底发行就在此期间发扬光大，主创与投资方对赌：预先设定保底票房，若实际票房未达标，主创需自担亏损；若超额完成，则按约定比例分成，发行方(一般是主创团队)可优先收回宣发费用并获取额外代理费。

表面上看，这种模式分摊了风险，可实际上却引发了电影成本能出十几个版本的乱象。

为了快速回笼资金，电影制作在保底发行的版本上发展出了估值倒卖的模式。

假如一个知名导演拍一部电影实际花了5000万，给主控出品公司报价1个亿，主控方转手以1.3亿价格出售给投资方A30%的份额。到了粗剪阶段，报价又涨到1.6亿元，再给投资方B出售30%的份额。只要市场上有人看好，投资方A和B又可以加价出售电影份额。

这么一来，一部电影就有了十几个版本的成本数字。制片导演一开始就赚得腰缠万贯，主控出品公司在付出3000万宣发费用后，票房1个亿就能回本。投资方A和B得等票房到2亿，其他联合出品方要期待5亿票房，二级市场的投资者可能要等到7-8亿才能保本。

也难怪说，现在的电影先是金融投资品，其次才是艺术。在这种环境里，谁还愿意花十年磨一部好片？

王长田记者出身，对传媒电影有天然的热爱。所以他才愿意注资 3000 万解救彼岸天，这才有了叫好又叫座的《大鱼海棠》。

这部灵感来自《逍遥游》，用 1109 张手绘背景打造的奇幻电影，用 5.75 亿票房刷新了国产 2D 动画票房纪录，但这部作品耗时十二年，成本回收率仅 117%，远低于商业动画 300%—500%的行业基准。

好作品都难达标收益线，王长田作为行业人愿意孤注一掷，可不能指望所有投资人都为爱发电吧？

毕竟，来投资电影的大多是奔着赚钱去的，他们根本不在意电影拍得烂不烂，要是让他们去投资一部得十二年才能有回报的电影，哪怕最后真的能赚大钱，也肯定没人愿意投。

## 03

单靠票房已难以支撑电影产业的可持续发展，开发 IP 衍生价值成为行业共识。

王长田透露：“《哪吒 2》推测能带来超 2000 亿元人民币的 GDP 增量，我们授权的部分产品，一个门类销售额就超百亿，要是没那么多盗版，上千亿衍生品销售额完全有可能。”

实际上，全球影业早已验证了 IP 衍生品的“吸金”能力。迪士尼的《冰雪奇缘》衍生品销售额超过 100 亿美元，远超票房收入。

哪吒两部电影全球票房超 233 亿元，但与其在潮玩、联名商品等领域创造的商业价值相比仍相形见绌。《哪吒 2》上映两个月后，哪吒联名品牌产品仍在热销，潮玩品牌 iFoodToy 的“哪吒 2 联名玩具惊喜盒”卖出近 6000 万元，至今仍位居“抖音潮玩盲盒好评榜”榜首。

不过，这些可观的销售额和光线关系不大。一方面，《哪吒 2》的制作团队把业务集中在内容本身，授权联名合作采用的都是一次性买断模式，电影出品方不参与后期销售的分成。另一方面，《哪吒 2》的爆火在意料之外，起初授权时，授权金收的都是白菜价，随着电影热度不断攀升，授权金才涨到了 900 万 - 1200 万元。

所以，尽管《哪吒 2》周边销售规模上千亿，但只有百分之几的收益与光线传媒有关。

抱着金鸡给别人下蛋，王长田觉得太亏了。所以这次光线加快拓展衍生品、流媒体等收入渠道，通过多元化收益以及对电影行业的反哺，来证明自身价值。

但王长田低估了这条路的难度。

王长田一心想把光线传媒打造成中国版“迪士尼”，为了拿到 IP 的绝对控制权，光线的动画电影系列从不接受外部投资。

这意味着，光线能拿到更加丰厚的回报，但走的路也会更加艰难。

王长田不喜欢跟风买明星的流量，他更喜欢实打实地一步步培养人才。

《西游记之大圣归来》从筹备到上映花了 8 年时间。饺子一部作品要等 5 年，《大鱼海棠》从创作到上映 12 年，但在如今这个“酒香也怕巷子深”的时代，这样的更新速度观众等不起吧。

更何况，作品上线不代表就能火，像旗下的《小倩》和《深海》就没有溅起多大的水花。 而王长田自己本人也承认《哪吒 2》作为个例，不可复制也难以借鉴。

另一面，光线的实景娱乐布局进展也很迟缓——2014 年启动的“中国电影世界项目”，十年间仅扬州一期建成，与迪士尼的成熟 IP 乐园相去甚远。

中国电影 IP 开发正站在千亿市场的门槛上，但要“从爆款偶然走向持续变现”，仍需破解盗版治理、授权模式创新、全产业链运营等多重难题。光线传媒在这条路上搞探索，做法挺理想主义的，可也把咱本土文创产业升级要经历的那些难处都暴露出来了。这条路不好走，既要匠人精神，还要商业智慧。
