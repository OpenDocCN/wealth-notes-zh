

# 中央打明牌，金融大转向趋势定了

今天这节课，我们谈谈最近备受关注的上海陆家嘴论坛，以及背后的金融大转向。

6月18号，2025年上海陆家嘴论坛正式开幕，央行行长潘功胜宣布，将在上海实施八项金融政策举措；证监会主席吴清宣布，推出进一步深化科创板改革的“1+6”政策措施；而央行副行长、国家外汇管理局局长朱鹤新，则宣布了跨境投融资外汇管理改革的九大措施。

在前面的课程里，我们讲到了中国正在加快构建新的金融体系，还讲到了深圳和上海正在进行一系列的金融改革，加上这次陆家嘴论坛上，各个金融部门“一把手”推出的各项措施，我可以很确定地告诉你，中国金融战略在进行大转向，这将对我们产生一系列的连锁影响。

## 陆家嘴论坛说了什么

懒人微信：lazyhelper

好，我们先来谈一谈这次的陆家嘴论坛上，各金融部门的“一把手”们都推出了哪些具体措施。

首先是央行。

央行行长潘功胜宣布，将在上海实施8项金融政策，我从中提炼出了四项具有突破性意义的政策，你先听一下：

- 一是设立数字人民币国际运营中心，将加速人民币跨境支付系统的全球化；
- 二是在上海临港新片区，开展离岸贸易金融服务综合改革试点，实质上是打造“境内离岸市场”；
- 三是发展自贸离岸债，这个自贸离岸债，是指在自由贸易试验区内发行的债券，这相当于为中国企业在自贸区里开了一个国际融资“小窗口”，能吸引境外投资者来买中国企业的债券；
- 四是优化自由贸易账户功能，这是要让境内外资金更加自由地进出。

我提炼了重点后，可能还是不太好懂，没关系，我直接给到你判断，这些政策背后，体现了高层的两个核心意图：

- 第一，强化上海的国际金融中心地位。这些创新政策都在上海落地，实质上是把国内大循环，包括银行间市场、征信体系，与国际大循环，包括离岸债、外汇期货的关键节点，集中布局在上海，让上海成为内外资金、规则、信息的转换器，既能服务国内实体经济的发展，又能提升国际的定价权。未来，自贸离岸债对标欧洲的美元市场，人民币期货瞄准新加坡离岸市场，离岸贸易的金融服务对标纽约。
- 第二，区域之间进行差异化金融布局。上海侧重“规则开放”，深圳聚焦“科技+金融”，香港保持“离岸自由港”优势，三个地方形成互补，共同架起中国金融的开放格局。

这些政策几乎都是为上海“量身定制”的金融大礼包。值得注意的是，这是继今年4月《上海国际金融中心进一步提升跨境金融服务便利化行动方案》后，上海在短短两个月内，获得的第二轮重大政策支持。这种政策密度和力度，充分体现了中央对上海国际金融中心建设的重视。而这些金融领域的频繁大动作，明显在构筑一个更大的开放格局。

其次，是证监会。

陆家嘴论坛上，证监会主席吴清的讲话，揭示了金融资源将怎样重新配置。他宣布的十项措施中，有六项直接服务于科技创新，包括深化科创板改革“1+6”政策措施、创业板启用第三套上市标准、发展科创债等。我知道政策比较枯燥，没关系，你就知道下这些政策，都是为了让资本市场成为科技创新的助推器。

最具突破性的，是重启未盈利企业适用科创板第五套上市标准。这意味着，那些还在研发投入期、尚未实现盈利的硬科技企业，将获得更多的上市融资机会。同时，除了科创板外，创业板也正式启用第三套标准，支持优质但未盈利的创新企业上市。

总之，中国资本市场正在打破传统盈利门槛的束缚，向更符合科技创新规律的估值体系转变。而且这些改革并不是孤立进行的，而是与“并购六条”，全称是《关于深化上市公司并购重组市场改革的意见》、以及《上市公司重大资产重组管理办法》等文件，形成政策组合拳，构建从初创期到成熟期的全生命周期融资服务体系。

最后，是外汇管理局。

在陆家嘴论坛召开的同一天，国家外汇局发布了《关于深化跨境投融资外汇管理改革有关事宜的通知（征求意见稿）》。

其中最吸引我的，是《通知》缩减了资本项目收入使用负面清单，取消了资本项目外汇收入及其结汇所得人民币资金，不得用于购买非自用住宅性质房产的规定。

用大白话讲就是，以后外资企业和外国人，可以直接在国内投资房地产了。之前是不行的，之前只允许外国个人购买自住的房产，现在则是允许“以投资为目的”来买房，范围也进一步从个人，扩大到了所有外资。

在我看来，放开外资购房绝不是简单的房地产政策，而是人民币国际化战略的关键落子。要知道，对中国来说，不动产是最具粘性的人民币资产，通过允许外资用外汇直接购房，实际上建立了“房产-人民币-外币”的价值锚定，未来，北上深广等一线城市和成都、杭州等新一线城市的房产，可能会迎来国际资本的重新定价，这对大城市房产的持有者来说是个利好消息。

## 金融战略的大转向

而这一系列的政策背后，是中国金融战略的重大转向。什么意思呢？

之前二十多年的金融战略，聚焦于基建和房地产，并形成了“土地财政拉动基建投资，带动房地产繁荣”的循环，但这套模式在中美科技竞争日益激烈的背景下，难以为继了，必须要改变。

而现在，中国金融战略的核心则转向了两方面：科技创新，和国际化战略。

对内，金融资源要大幅配置到科技创新上，构建起金融的内循环：

- 资本市场方面，要做大的科创板这个“硬科技试验田”，重构价值评估体系，为科技创新输送资金。
- 银行体系方面，要从传统的抵押品定价，转向科创定价，成为科技创新的“加速器”，而不是房地产和传统制造业的“提款机”。
- 地方政府方面，也要从土地金融，转向科技金融，用科技产业基金构建起“科技资本化”的通道，推动经济结构升级。

而对外，金融则服务于人民币国际化、企业出海等业务，构建起金融的外循环：

- 一方面，香港是金融外循环的“前门”，上海、深圳是金融外循环的“后门”，三者不仅是资金进出通道，更是人民币资产定价规则的输出平台，要持续扩大话语权和影响力。
- 另一方面，构建独立于美元体系之外的国际融资市场和金融基础设施，既不完全脱离美元体系，又能在关键节点形成自主闭环。

而站在更大的历史视角看，中国金融战略的转向，实质是在回答“中等收入陷阱”的世纪之问。当传统的增长模式难以为继时，只有把金融资源从投入钢筋水泥，转向投入科技和国际化建设，才能完成发展动能的根本转换。

我知道这一节内容可能听起来比较枯燥，但哪怕不太好理解，我也先喊一嗓子，这是一个大趋势，值得你的关注。

## 背后五大影响

金融是现代社会发展的底层基础，当金融战略发生转向，会带来深远的影响，我们依次说一下。

- 第一，财富积累的方式变了。以前房地产时代，财富增长遵循土地的稀缺性逻辑；而在科技时代，财富创造则呈现“知识裂变”的特征。比如一项基因编辑技术的突破，可能催生千亿级的生物医药产业；一个新型电池材料的发现，可能颠覆整个新能源格局。未来，谁拥有核心技术，谁就将站在财富金字塔的顶端。这种变化不仅会改变财富的分布，更将重新定义社会阶层的流动规则。

- 第二，传统规律和周期可能失效。之前围绕房地产形成的经济周期和债务周期，具有很强的规律性和周期性，而现在科技创新主导的新发展方式，则呈现出“量子跃迁”的特征——就是长时间没什么动静，一旦突破就会是爆发式增长。这种非连续的发展模式，可能会让传统规律和周期失效，“二八现象”可能越来越明显，这是你要思考的一个新逻辑。

- 第三，精英阶层正在更替。传统地产富豪的退场与科技新贵的崛起，只是表象，更深层的变革在于整个精英阶层的认知重构。过去二十年，财富爆发源于对土地价值的敏锐嗅觉和对政府关系的精准把握，而未来将转向对技术和创新的预判，对国家意志的理解，和对地方比较优势的把握。未来的精英，是懂产业、懂技术、懂政策的人，那些还停留在传统思维里的精英，必然会被淘汰。

- 第四，人民币国际化的方式也在变。过去主要依靠贸易结算推动人民币走出去，而现在允许外资投资境内房地产、股票等。当外资持有更多用人民币计价的房产、股票、债券等资产时，自然就提升了人民币资产的吸引力，更能稳定人民币的价值，相当于为人民币找到了新支点，未来可能不再由美元波动来定价，而是由人民币资产来定价。

- 第五，城市竞争的重点变了。深圳和上海的新一轮开放，都在证明一个信号：制度创新的容错空间和对外的政策灵活度，正在成为新的核心竞争力。未来，谁能为科技企业提供更大的试错空间，为国际资本设计更适配的规则，谁就能脱颖而出。未来城市之间拼的，不只是资源，而是能不能提供适合创新发展的好制度、好规则。而这种竞争，将淘汰一大批城市。这些变化，就是我讲这一节的意义。
