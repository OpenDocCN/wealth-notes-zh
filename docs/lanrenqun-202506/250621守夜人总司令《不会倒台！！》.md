

# 不会倒台！

摧毁核设施和配套的人员及产业没问题，但要彻底颠覆是不能接受的。读过《骑虎难下》就会明白，在那个生存结构中，需要一个对抗者来搅动整个地区的不安，同时，这个角色又不能失控。

伊朗不是叙利亚，体量和能源的储备和供应能力完全不一样。如果这样的角色消失了，整个地区就会一边倒，这种情况最大的恶果就是，油气的定价权会再一次从目前的不确定状态重新变成确定状态！不要忘记美国也是一个产油大国，除了华尔街的定价权之外，美国自己的份额也不低，如果中东和美国的份额加在一起，再加上拉美的，即便是用市场规则，也是占大额从而拥有定价权。

以色列攻击伊朗油田的时候，为什么最先跳出来的是日本反对呢？按道理说日本不应该跟美国保持立场一致么？因为日本在2022年2月开始俄乌冲突中站队乌克兰，导致俄罗斯对日本的油气供应中断，即便从西方或者东南亚买二手的油，也是成本高企供应不足。

如果油价继续推高，而且航路中断，让进出口的成本进一步增加，已经连续两年陷入贸易逆差的生产型国家日韩就会更加的麻烦。日本人已经开始用粮票了！这可不是玩梗，真的开始在粮食上采用我们中国当年的配给制。你能买多少粮食不取决于你有多少钱，而取决于你有多少粮票；如果状况持续恶化，生活领域搞粮票，工业领域是不是要搞油票么？

除了英国跳得高之外，生产型的西方国家并不具有太强的参与意愿，反而试图让局势降温。为什么美国现任总统特朗普不遗余力地火上浇油呢？因为特朗普上任以来没有搞成一件特朗普承诺的事：不管是2022年2月开始俄乌冲突的和解，还是对中国发动的贸易战，最终都不了了之。

现在唯一的机会就是把油价搞上去，为中期选举打造一点稍微拿得出手的业绩。

中国为什么袖手旁观呢？中国并非不参与。中国在等一个故人交出他的筹码，这个故人不是伊朗，伊朗就算愿意给点东西，也送不过来，而且依然因为被封锁缺乏技术更新，伊朗的油品是最差的，杂质量最高，这个故人是俄罗斯！大佬文集圈里我们曾经推演过，如果霍尔木兹海峡被封锁，我们的油气缺口会有多大呢？最坏的局面是有20%—30%会不确定。

虽然这部分可以从东南亚购买，但是这些国家自己的产出比较少，大部分是转卖的。那么谁可以补上这个缺口呢？俄罗斯和美国都可以！上次中国高层打电话邀请美国访问，美国现任总统特朗普肯定是要卖美国的油气，访问之前推高价也有利于谈判。

有利于特朗普就不有利于你，最好是有一个稳定的供应者先给你吃个定心丸，比如说签订一份长期合同，以更低的价稳定供应，不受市场价格波动。这个筹码只能俄罗斯给，其他人也给不了。

你要让俄罗斯愿意给，你要么能给俄罗斯点什么，要么有个困境会让俄罗斯损失更多，俄罗斯不得不做这个交易。现在的情况就是：中国越沉得住气，俄罗斯就越慌。以色列宣布要像叙利亚那样，在伊朗反制一下。

中国除了空喊的说了几句不痛不痒的话，没有表态，反而是俄罗斯明确地表示这是不能够被接受的！如果真的发生这样的事，在被封锁的情况下，俄罗斯去年试图通过阿塞拜疆再经过伊朗抵达红海的管道还有没有戏呢？肯定没戏了！

目前唯一通往西方的管道就是经过黑海底部抵达土耳其再往欧洲，现在命门捏在土耳其手上，俄罗斯不好受也没办法，这可是俄罗斯唯一的生命线！俄罗斯的收入70%来自于油气产业，如果只有东边的，没有了西边的，在收入锐减一半的情况下，怎么支撑起现有的局面和还没有结束的战争呢？俄罗斯去年的军费已经高达GDP的38%了，所以，俄罗斯很着急。

除了眼巴前的这个危机，从长远来看，一旦没有了伊朗，伊拉克和埃及也在挣扎，那么沙特的态度就是唯一的态度。沙特一旦跟以色列和解，再次跟美国捆绑，就是用现在市场的份额和规则，美国也会拥有定价权，这对俄罗斯来说是不能接受的灾难性后果。

当然了，对中国来说也不愿意接受，但是，中国没有那么急，中国越是不急，俄罗斯就越急。所以接下来你会看到，俄罗斯会拿出诚意来，与中国协商一个统一的立场，尽量发出一个共同的声音。早些天俄罗斯宣布要调解的时候，美国现任总统特朗普嘲讽道：“俄罗斯还是先把自己跟乌克兰的冲突调解好吧，中东不劳烦俄罗斯！”

国家之间所有的情谊背后都是交易，交易达成之后，在电视上就都是情谊，都是为了生存和发展，朋友和敌人没有那么明显的界限。当下是战国，以阵营来划分敌我是幼稚的想法，战国的情况就是：所有人与所有人博弈。
