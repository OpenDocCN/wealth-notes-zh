

# 举重若轻是一种大智慧

251114 猫哥

整理：公众号懒人搜索，懒人专属群独享

懒人微信：lazyhelper

![](bbox=[60, 149, 823, 270])

少时读书，常常看到一句话：善战者无赫赫之功。当时理解不到这句话的内涵，但是随着年龄的增长，对这句话才有深刻的认识。

今天给大家讲一个历史上可能是最低调的开国皇帝。

这个皇帝就是赵匡胤。

与其它开国皇帝相比，赵匡胤看起来是最平庸的。

生平经历既无赫赫战功，比起李世民、朱元璋等都差了八条街，也无执政非常突出的成就，与文景之治、贞观之治等等相比，赵匡胤也差了八条街。

赵匡胤给历史留下的也能让大家记住的就是两件事：

黄袍加身与杯酒释兵权。

黄袍加身让赵匡胤篡了后周的基业，轻轻巧巧当了一个便宜皇帝。

杯酒释兵权让赵匡胤结束了五代时期藩镇割据尾大不掉的现状，让这个皇帝做得实至名归。

所以后世总有人说赵匡胤运气太好，没怎么费力救得了天下。

但是，真的是运气好吗？我们对比一下历史。

同样是篡位，赵匡胤的前辈付出的代价是非常高的。

司马懿发动高平陵之变后，司马家已经掌握魏国的军政大权。但是依然经历了司马懿、司马师、司马昭和司马炎几代人，反复对朝臣进行清洗、不断平定叛乱，才最终夺得皇位。整个过程耗时数十年，期间爆发过数次战争，死伤极多。

南朝宋武帝刘裕篡位，大肆诛杀司马氏满门。后来齐高帝萧道成夺位，又灭了刘氏满门。过程血腥残酷实在是一言难尽。

五代时期，朝代更迭更加血腥。赵匡胤的老上司郭威，也曾上演过黄袍加身的戏码，他付出的代价是：允诺士兵在开封城中大掠三日。开封城死伤无数，堪称人间地狱。

与上面三者相对比，赵匡胤发动的政变，历时极短，而且开封城里都没怎么动乱，全程只有一人死亡。

这个篡位的代价，可以说是低到了极点。

那么问题来了，为什么赵匡胤的前辈篡位就这么血腥，代价这么大，而赵匡胤则这么轻轻松松就实现了改朝换代的目标？这仅仅是运气吗？

再说说杯酒释兵权。

任何人当了皇帝之后，都想着要削减武将权力，将兵权收归中央，但是要做到这一点谈何容易？

我们不说历史上的失败案例，就说几个著名的案例。

汉高祖刘邦怎么做的？

刘邦收权过程就是各地不断叛乱过程，为此刘邦基本把老命都赔上了，死了后还是爆发七国之乱。

唐代李渊要收秦王兵权，结果爆发了玄武门之变，最后连皇位都丢了。
朱元璋要收武将兵权，手段就杀杀杀，把开国元勋杀了干干净净。

对比一下，赵匡胤收权简直不要太轻松，喝一杯酒，就轻轻巧巧将兵权收归中央了，过程中甚至没有死一个人。

从刘邦到李渊到朱元璋，哪一个不是人中龙凤，为什么他们收权这么艰辛，而赵匡胤却能干得这么轻松？

所以，赵匡胤能成为宋朝开国皇帝，这个人绝非看起来那么简单！

之所以从黄袍加身到杯酒释兵权做得这么顺利，就是赵匡胤做事不仅谋定而后动，而且他具备一种举重若轻的大智慧。

对，就是举重若轻！

无论是篡位还是收权，都是重得不得了的大事，但是赵匡胤却能干得很轻松。

限于篇幅，我们今天就讲一讲杯酒释兵权。

少时读史，看到杯酒释兵权的时候，我就在想，这些将领交出兵权之后赵匡胤是这么处理的？总不能自己直接去接收这些兵权吧？那么赵匡胤又怎么防止继任者不变成五代时期那些新的藩镇呢？

赵匡胤的做法就是两步走，分权制衡与尽收精兵。

我们一个一个来说。

在五代时期，地方节度使是军政一把抓，因为控制了民政，所以藩镇就能给自己的军队提供后勤与粮饷，这样的军队就变成节度使的私人军队。

赵匡胤就是将节度使的民政权力剥离，中央派出知州管理民政，并且定期轮换，同时在各州设立通判，监督知州，形成权力制衡。

然后收取藩镇精兵做得更是高明。

所谓收其精兵，就是把各地节度使手上的士兵进行拆分，把其中的精锐收归中央，老弱留在地方，以此削弱节度使的实力。

这个事情说起来很简单，但是真操作起来问题非常多。且不说会不会遇到反抗，会不会逼反地方这些大问题。就说一个最现实的：地方节度使完全可以把手上的精锐藏起来，把老弱送到中央去，你有什么办法？

那赵匡胤是怎么解决这个问题的呢？很简单，他向各地藩镇下了一道圣旨：为了提高地方军队的待遇，增强部队战斗力，朝廷决定从国库拨出一大笔钱，直接发给各节度使麾下的精兵强将，按人头发放——相当于给这么精锐士兵发放岗位津贴。

为了防止部分地方节度使浑水摸鱼，甚至领空饷，圣旨中要求各镇节度使所给出的名单，要有详细的资料，包括籍贯、姓名、身高、气力、特长等等，以供中央派人查验。只要查验无误，确认此人满足精兵的条件，这笔钱就足额发放。

圣旨下达后，各地节度使纷纷行动起来，将手上的精兵名单一一汇总递交到中央。而朝廷也果然拨下了一笔巨款，每个月都给这些士兵发放额外的军饷，自然皆大欢喜。

然而转过年，赵匡胤的杀招就来了。他又向藩镇下了一道新的圣旨：

为了加强大宋的国防力量，朝廷决定从地方抽调精兵充实禁军，精兵名单随圣旨一道下发，凡在名单上的人，地方不得以任何理由扣留，必须将他们送入开封，充实禁军。

杀招一出，节度使们面面相觑，完全没有还手之力，只能乖乖奉旨，丝毫不敢讨价还价。因为那些精兵领了几个月的奖金都是中央发出来的，如果扣留这批精兵不要说就是对抗圣旨，最关键的是，精兵已经习惯领取岗位津贴了，节度使不放人，自己又没有民政权力，拿什么去发岗位津贴，如果把这批士兵津贴给停了，都不用中央动手，士兵也会给节度使好看！

你看看，赵匡胤收取兵权的解题思路与历朝历代都不一样，没有选择最血腥的清洗的模式，而是谋定而后动，将重的不得了的事情拆解成几个步骤，用政治手段叠加阳谋就轻轻松松办下来了。

说实话，虽然赵匡胤看起来很平庸，从黄袍加身到杯酒释兵权都顺利得不得了——那是善战者无赫赫之功！这份举重若轻的本事历朝历代皇帝中，赵匡胤当居首位！

回到当下。

现在大家都关注的台湾问题，我总觉得国家处理这个问题很有点举重若轻的味道，虽然大家现在都在喊武统，但是我有一种很强烈的感觉，说不定几年后，这个事国家轻轻巧巧的就办下来了——付出的代价会小得多，甚至可能都死不了几个人。

那么对于我们个体而言，赵匡胤的历史也是有借鉴意义的。

我们在生活与工作中经常遇到比较棘手的问题，这些棘手的问题对于一般人可能也是重得不得了。

但是如果我们多思考，谋定而后动，将复杂的问题拆解成几个关键环节，用创新的思路去解决这些环节，那么我们也能做到举重若轻，用最小的代价解决最大的麻烦。

不要遇到困难就有畏难情绪，再大的事情能有篡位谋国大？能有收取兵权大？历史上赵匡胤都能举重若轻将这些事轻松办下来，现成的历史教材摆在这里，我们又有啥事情做不到的？
