

# “十五五”详解四：扩大内需，上强度，给红利

欢迎来到《政经参考》，我是马江博。

今天我们继续研读“十五五”规划建议稿，谈第三个任务“建设强大国内市场，加快构建新发展格局”。这就是最近这两年一直在谈的扩大内需，也就是消费和投资，这部分和每个人的钱包直接相关。

这节课我会带你看清：国家扩大内需的主要任务是什么，哪些消费会有政策红利，哪些具体领域会迎来投资红利，以及哪几类企业会在市场重塑中受益。

## 总体论述

在这部分的开篇，建议稿就直接说，“强大国内市场是中国式现代化的战略依托”。

什么是“战略依托”，我理解就是未来五年，“强大国内市场”必须成为我们大国博弈和实现中国式现代化的底气和堡垒，而且“强大国内市场”和“科技自立自强”一样，都突出的是“自主可控”，为安全和未来可能的中美之间的独立平行系统做准备，我把这个关键先点出来。

因此，建议稿说，要“坚持扩大内需这个战略基点，坚持惠民生和促消费、投资于物和投资于人紧密结合”。这里你注意两点，第一是“战略基点”的表述，我分析这意味着未来五年的政策工具箱里，促进消费和投资的优先级会非常高；第二，消费放在了投资前面，更有紧迫性。

具体到消费方面，“坚持惠民生和促消费紧密结合”，是把改善民生和刺激消费绑在了一起，要先把社会保障做好，大家才能更多花钱，这个顺序现在是很明确的，而且补贴了同样的钱，中低收入者拿来消费的比例，比富人要高得多，毕竟这些补贴可以切实改善他们的生活质量。

而投资方面，“投资于物和投资于人紧密结合”这个提法比较新鲜，特别是“投资于人”，这个词今年首次在《政府工作报告》中提出，也是首次在五年规划中出现，投资于物简单理解就是修路建桥、搞新基建这些；而投资于人，就是发展以人为核心的教育、培训、医疗等。这个意思就是要平衡“硬投资”和“软投资”。

我综合判断，未来五年中国经济政策的一个关键转向，是从出口更多转向内需，从投资拉动更多转向消费驱动，从撬动项目转而加大撬动人。具体来说，建议稿主要提了三个子任务。

## 大力提振消费放在第一位

第一，是“大力提振消费”。建议稿说要“统筹促就业、增收入、稳预期，合理提高公共服务支出占财政支出比重”。

首先，“促就业、增收入、稳预期”的次序很重要，“促就业”是基础，有工作才有收入；“增收入”是核心，钱多了才能多花；“稳预期”是花钱的信心关键。我判断这接下来也会是政策发力的优先排序。不过实际情况中，稳预期会更容易些，包括我们最近半年股市的拉涨表现，也是“稳预期”的举措之一。

而“合理提高公共服务支出占财政支出比重”，我认为则是一个宣示，意味着未来五年民生保障类行业将迎来持续的、确定性的政策红利和资金投入。对个人来说，类似职业教育、医疗健康、养老服务等领域，会开启一个红利期。

然后，建议稿先提了“服务消费”，把它放在了“商品消费”之前，我们课程反复说过，服务消费未来是最大的消费新增量，和普通人的就业蓄水池，“十五五”规划建议稿再次明确了这一点。

在服务消费方面，具体说要“以放宽准入、业态融合为重点扩大服务消费。”“放宽准入”是扩大对民营和外资的准入。“业态融合”很有意思，怎么理解呢？举个例子，比如现在很火的“苏超”，就是“体育 + 商业”，体育赛事赚流量，商业配套赚利润；还比如“医疗 + 康养”，医疗提供基本服务，康养提供高价值利润点。这就是“业态融合”，其中一个服务或场景用来“引流”，另一个用来赚钱。这种融合创造的新体验，既是创造新需求的关键，也是新的消费增长点。

然后是“商品消费”，措辞是“推动商品消费扩容升级”。比如同样是家电，不只是要能制冷、能洗衣服，更要能和全屋智能家居系统联动。再比如国潮品牌和新消费品牌，要能利用新技术、新设计、新营销，打造新体验、新需求。

而且这次强调，要“打造一批带动面广、显示度高的消费新场景”。带动面广就是要一带多，吃喝玩乐、文创科技等都融合进来，而“显示度高”在这种文件里是个特别新奇的词，我理解就是要有足够的吸引力和话题性，这个词出现，代表了一个国家顶层设计中的流量思维，很有意思。

建议稿还谈到，要“培育国际消费中心城市”，目前北京、上海、广州、天津、重庆，已经被列为首批建设试点。

建议稿继续说要“清理汽车、住房等消费不合理限制性措施”。很多人看到后问说，是不是北京、上海这些大城市的车牌摇号和购房限购政策，要给“清理”了？我的看法是：首先这些限制政策肯定将会被全面审视，在非核心城市会清理得很迅速，但北上这种城市更大的可能性是小步渐进放松，比如对于车牌可能加大释放数量，而目标方向可能是从限制购买，转向“调配使用”，房产限购也会是类似逻辑，渐进式满足改善需求。

最后建议稿还说了一句“落实带薪错峰休假”。落实这个词的意思是要真做到，如果真能落实，这将让休闲消费更加常态化。那些能够提供全年稳定服务的旅游目的地、度假酒店、娱乐场所，以及能够满足个性化、分散化休假需求的定制旅游、短途周边游等产品，将更受欢迎。

## 投资突出“有效性”

谈完了消费，我们谈投资，“建设强大国内市场”的第二个子任务，就是要“扩大有效投资”。

请注意措辞，这次强调“有效”二字。我分析，有效对应的自然是之前的某些无效，既有一部分政府投资的无效，也包括一些企业投资的无效，沿着这个逻辑，中央和地方投资的合理分工，投资的进一步市场化，以及加强“反内卷”、避免无效产能和恶性竞争，就是必然。

这部分，建议稿说要“提高民生类政府投资比重”。这和我们上面讲的，提高公共服务支出占财政支出比重这个转变是一致的。政府的钱，要更多地用于老旧小区改造、城市地下管网更新、城乡医疗卫生体系建设、普惠性幼儿园和养老服务等，这些未来五年是确定性的红利机会。

然后是，“高质量推进国家重大战略实施和重点领域安全能力项目建设”，这里的“安全能力”，指的就是粮食安全、能源安全、产业链供应链安全、数据安全等。比如投资大型水利枢纽、建设国家粮食应急保障基地、构建国家算力网络等，这部分也是大投资。

建议稿随后说，要“加强人力资源开发和人的全面发展投资”。我的看法是，这就是要加大投资人的知识、技能、创造力，中国经济要从“人口红利”向“人才红利”转变。我认为未来几年，政府尤其会着重支持各类教育和职业技能培训，这些将成为新的“基础设施”。

然后，建议稿强调，要“进一步明确中央和地方投资方向和重点”。如果说前几条说的是政府“投什么”，这条说的就是“谁来投”和“怎么投”。过去地方政府投资容易“一窝蜂”，未来五年中央会规划好“赛道”，我分析下来，中央主要负责跨区域、战略性、安全类的重大工程；地方则主要负责本地的民生和公共服务，以及特色优势产业升级。

下一句同样关键，就是要“激发民间投资活力、提高民间投资比重，增强市场主导的有效投资增长动力”。我认为这句话的潜台词是，投资不能政府唱独角戏，经济的活力最终要靠广大的民间市场主体，另外在类似关系国计民生的重大项目和未来产业布局中，不仅要让民间资本参与，还要鼓励它扮演更重要的角色，提高它的投资份额，而最后目的则是要增强“有效投资的增长动力”。总之，在投资上，政府和市场，要在各方面形成合力，这也是“新型举国体制”的另一种体现。

## 物流值得重点关注

有了强大的内需，和有效的投资，但如果市场机制不通，那也不行。所以建议稿用不小的篇幅来讲最后一个子任务，就是“坚决破除阻碍全国统一大市场建设卡点堵点”。

建议稿说要“消除要素获取、资质认定、招标投标、政府采购等方面壁垒”。这些壁垒是什么？其实就是地方保护。然后，建议稿强调了“综合整治‘内卷式’竞争”和“强化反垄断和反不正当竞争”，这背后是中央在强力重塑市场秩序，包括平台经济。而最终目标，建议稿说要“形成优质优价、良性竞争的市场秩序”。简单说，好的产品，可以卖得贵，还能卖得好。只有这样，大家才有动力去创新，去做好产品。

对此我的整体判断是，未来几年，针对地方保护和市场分割的“统一大市场的”改革会强力加速，反内卷和反垄断、反不正当竞争的监管，将日益常态化，成为基础环境，这将重塑新能源、平台经济等很多行业的竞争格局。

在这个过程中，我认为未来三类企业可能从中获益：

- 第一类是部分全国性布局的企业。对那些有能力在全国范围内统一运营、统一品牌、统一供应链的企业来说，过去因为地方保护被挡在门外的市场，现在更有机会进入了。它们的标准化优势，可以快速释放出来。
- 第二类是“专精特新”企业。这类企业不是拼规模和关系，可能就是靠一项核心技术，在“优质优价”的市场新秩序下，它们将活得更好。
- 第三类则是提供合规服务的企业。随着反垄断和反不正当竞争执法的强化，企业对合规经营的需求会加大，这类专业服务，将大有用武之地。

在市场机制这部分，建议稿最后说要“降低全社会物流成本……优化企业总部和分支机构、生产地和消费地利益分享”。

这段话背后也非常有信息量。目前，我国物流成本占 GDP 的比重（2024 年：14.1%）仍明显高于世界平均水平（10.7%），“降低全社会物流成本”是建设统一大市场的重要一环，这需要从基础设施、技术应用、管理体制等多方面入手。

不知道你有没有注意到，“物流”这两个字，我们在之前课程中经常提到，它的地位越来越高，所以机会也越来越大，不管是关键的物流枢纽，高端的智慧物流，还是在县乡之间补短板的升级的普通物流，这方面都值得重点关注。

而“优化企业总部和分支机构、生产地和消费地利益分享”，则是要解决跨区域经营的利益分配问题。过去，企业总部所在地可能获得大部分税收，而分支机构所在地虽然承担了环境、交通等成本，却获得较少税收，这导致不同地方政府之间收益的不平衡。优化利益分享机制，就是要合理分配税收等利益，让各方都能从统一大市场中受益。

综合下来，我的判断是，未来五年，物流效率将显著提高，物流成本将明显下降，并且会打破更多的市场分割壁垒，这是个重要的基础判断。
