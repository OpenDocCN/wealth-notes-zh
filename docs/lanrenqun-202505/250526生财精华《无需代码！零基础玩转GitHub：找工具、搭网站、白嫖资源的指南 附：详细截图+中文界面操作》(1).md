# 第二章：快速上手：注册与汉化

## 步骤 1：注册账号

访问 github.com，点击右上角 Sign Up → 填写邮箱/密码/用户名。

避坑提示：用户名只能包含字母、数字或连字符。

![](img/09931abf6aea66e3a0297eb6d005e8ae_5_0.png)

## 步骤 2：安装汉化插件

搜索“汉化插件”→选择星标最高插件。

点击“安装”→刷新页面自动生效（可以根据自己的浏览器来进行选择）。

![](img/09931abf6aea66e3a0297eb6d005e8ae_5_1.png)

进入之后往下滑可以找到【浏览器与脚本管理器】，根据自己的需求选择合适的脚本进行安装即可。

## 2. 筛选指标

这个筛选指标是通用的，大家可以根据自己的需求来，可以选择复刻多的，也可以选择星标多的。

![](img/09931abf6aea66e3a0297eb6d005e8ae_9_1.png)

一般情况下星标和复刻越多就是这个项目越受欢迎，咱们可以随便点击一个进去它的介绍，基本上每个打开都是这样的一个界面，大家可以看一下我的标注。

网址基本上都是主域名 + 开发者名字 + 仓库名称组合而成的，如果找不到相关的展示链接可以按照这样的顺序进入展示的页面。

最左边就是代码部分，可以点击进去查看代码的情况，不过我看不懂哈哈哈哈。

⚠️ **注意：若项目最后更新时间为 2 年前，可能已失效！**

# 第四章：Github 隐藏功能

1. 社区互动（Issues 提问于讨论）

如果你觉得项目存在哪一些可以继续更新完善修改的，也可以将你的修改通过主页的【拉取请求】合并进作者的主代码库，为 Github 贡献出你的一份力。

首次为 maboloshi/github-chinese 做贡献？
如下图，当前是默认的公开系统，就是默认的公开系统的议题回答，收费是免费。

如果你不知道如何修复一个议题，可考虑为它打开一个拉取请求。当然，你在做项目的过程中如果遇到问题不知道怎么解决的话也是有迹可循的，主页中间有一个议题，这里就是往期其他适用过本项目的小伙伴遇到问题归纳和总结了。在【议题】中你可以与项目作者和其他爱好者互动讨论。

用户进行讨论。还可以点击右边绿色的按钮【创建议题】发起一个讨论。

【议题】分成两种，【打开】跟【关闭】的，其中【打开】一般指的是这个项目在使用过程中还没有解决的 bug 或者还在讨论的问题，【关闭】则是指的已经解决的 bug 或者已经结束的讨论。

![](img/09931abf6aea66e3a0297eb6d005e8ae_15_0.png)

## 2. 个性化推荐（探索页和星际联动）

平时你遇到觉得不错的仓库或者项目可以给个星标，GitHub 会自动根据你的星标的项目给你进行推荐类似的（有点像是小红书你点赞了哪篇笔记，系统会自动再给你推荐类似的）

可以在自己仓库的主页右上角找到【探索仓库】，下面的【探索更多】，点开之后你就可以像逛小红书一样逛 GitHub 了，基本上都是基于你星标项目给你推荐的

## 公众号懒人搜索，懒人专属群分享

在探索仓库的下面还有一个更多热门仓库，你点进去之后会看到最近比较热门的仓库/项目排行榜

那么怎么找到这个功能呢？右上角有个筛选功能，可以找到今天、这周甚至这个月最受欢迎的项目，如果找不到的话可以直接点击 https://github.com/trending 这个网址进入，你还可以筛选中文类的项目，对于程序员可以筛选基于某种程序语言的项目，

遇到喜欢的还可以点个星标，或者觉得有异常值的还能拿来当作风向标
