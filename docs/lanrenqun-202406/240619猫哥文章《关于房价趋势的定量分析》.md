

# 关于房价趋势的定量分析

看了一眼，后台提问的还是关于房子的问题最多，其实大家应该都知道我对房市未来趋势性的主要观点，不过我的观点都是趋势性的定性分析，现在很多人纠结的问题基本是具体到自己的房子的短期价格趋势——比如现在持有的房子该不该卖？心仪的房子现在该不该买？今天就给大家一套对房价趋势的定量分析方法。

首先我们来看大环境。

自从5月份国家出台一系列救市政策之后，现在刚好一个月，那么这一个月救市效果怎么样呢？确切的说，很不理想。

6月17日，国家统计局公布了5月份全国70个大城市的房价数据，环比4月没有一个城市上涨，全部都在下跌。如果是同比去年5月，全国70个大城市房价基本下跌了5%—10%左右（大家记住这个数据）。

不但房价下跌，销售量也在下跌，也就是量价同跌。

那么国家管理层对房市又是什么态度呢？

国家对房市的基本态度就是，救肯定要救，但是救市的力度基本是托而不举。也就是说，国家对房地产基本态度就是企稳就够了，国家不希望救市力度过大而导致房价暴涨，这也是 5 月央行拿出收购存量房资金只给了区区 3000 亿的根本原因，3000 亿资金，配套商业银行贷款一共也就 5000 亿，对应近 7 万亿的商品房存量，基本也就是杯水车薪。

这里多啰嗦几句，国家对房市托而不举的基本定性原因就是，我们房地产以及关联产业链存在巨大的产能过剩，这个巨大的产能过剩必须通过市场化的方式出清，才能让房地产以及关联产业链未来能健康发展。

既然国家对房市是托而不举（也就是政府干预的力度远远不足以改变市场基本的供求关系），那么房地产趋势就要遵循基本的经济规律。

这个经济规律是什么呢？

如果我们把房市类比为股市，那么过去房市相当于高成长股票，大家对房价上涨的预期很高，在这种情况下，就会给这个股票很高的PE（市盈率）。

但是现在房子的金融属性被剥夺，也就是价的剥夺，在这种情况下，市场就会回归理性，去追捧高股息股票（也就是分红比例高，市盈率低）。

对应到房子这个市盈率就是房价的租售比，因为在市场对房价成长性不看好的时候，租金回报率就是房价的价值之锚。

那么这个租售比在什么位置是比较合适的呢？

这个问题大家不要与老外大城市相比，原因是他们有房产税有持有成本，而且他们资产价格之锚与我们也不一样。

在中国所有金融资产定价之锚是三十年国债收益率（老美是十年期国债收益率）。

今年年初，我们30年国债收益率一度冲到接近2.8%，国债收益率大涨直接导致我们股市暴跌（相当于静态市盈率高于35的股票都没有投资价值），房市也暴跌（去年我们50个大城市平均租售比是1.95%）。所以央妈必须亲自下场操盘压制国债收益率。

现在我们三十年国债收益率在央妈亲自操盘下压制到了2.52%。去年我们50个大城市平均租售比是1.95%，今年大城市房价同比去年下跌了5%——10%左右，按照这个数据测算，目前我们大城市租售比大致是 2.05%——2.16%，与目前国债收益率相比还有差距。

这就是目前房市还不能企稳的根本原因，如果我们把国家 5 月份出手救市理解为政策底，那么还有一个市场底。

那么这个市场底在哪里呢？

首先，我估计央妈还会想尽一切办法继续压低国债收益率，到今年三季度左右，预测我们三十年国债收益率可能会达到 2.3%左右。

我们以这个 2.3%为测算一下，大城市房价估计还有 10%以内的下跌空间。

这就是市场底。

所以对于纠结于卖还是买房子的朋友，现在我就给你们一个简单的定量化的指标。

租售比 2.3%。

如果你的房子租售比高于这个指标，可以等等看，如果低于这个指标可以直接卖掉。

同理，如果你心仪的房子也可以按照这个指标做决策，租售比高于这个指标直接买，租售比低于这个指标可以等等看。

另外，2.3%只是大城市包括省会中心城市的参考指标，如果是三线城市，这个指标还要下调 20——30 个基点，也就是将租售比指标调整为 2.1%——2.0%左右，因为城市越小，商品房供应越是过剩，租售比指标越低。

以上就是关于房价趋势的定量分析模型。这个模型仅限于住宅，商铺价值衰减速度远远高于住宅，如果同样用资本市场类比，商铺相当于 ST 股票。
