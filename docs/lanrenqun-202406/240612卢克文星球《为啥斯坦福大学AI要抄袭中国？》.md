

# 为啥斯坦福大学AI要抄袭中国？

240612

文/卢克文工作室嘉宾 漫游指南

整理：公众号懒人搜索，懒人专属群分享

懒人微信：lazyhelper

在5月29日，一个名为Llama 3-V的AI引爆了科技圈。开发团队就3人，2名斯坦福大学本科生加1名南加州大学毕业生。还说性能不输GPT-4这类一线AI，训练只花了500美元。

正当一众看客以为自己即将见证下一场工业革命时，有人发现这个Llama 3-V和中国面壁科技的MiniCPM-Llama3-V 2.5大模型除了变量名字不一样，其他的都一样。

随着无数铁证的上传，Llama 3-V团队在6月3日承认抄袭并公开道歉，但是发声的只有斯坦福的2名本科生Aksh和Siddharth，拿了南加州大学学士学位的Mustafa却杳无音讯（名字太长，后面叫他们A，S和M）。

道歉信中，A和S把自己的责任摘得一干二净，宣称自己只负责宣传，开发从头到尾都是M一个人在做，俩人只看过演示没看过代码。还说事发后，M人间蒸发了，他们俩也联系不上。

事就是这么个事。Llama 3-V和MiniCPM-Llama3-V 2.5表现一模一样，用不同数据和不同策略训练出来的AI不可能有一样的结果，所以抄袭铁证如山。

除了抄袭本身，大家一定也发现这件事其实还有不少疑点，A和S的话自相矛盾有不少漏洞。但是与其掰扯这3人的谁是谁非，不如花时间研究一些更根本的问题，比如为什么科技圈一开始就如此关注Llama 3-V项目？以及现在的AI大战是个什么局势？

先回答为什么Llama 3-V项目有如此的关注度。答案不在斯坦福大学的名头上，AI圈天才无数，斯坦福的名头在这里激不起任何浪花。热点在成本只要500美元上，这也是A和S最大的可疑点，他们俩都有技术背景，不可能不知道这500美元代表着什么。

500美元代表什么呢？AI训练是很贵的，一次几百万美元都未必打得住，如果Llama 3-V所说500美元属实，等于这3人找到了点石成金的办法。别人要100块钱的油才能跑完的路，你5块钱的油就能跑完，还跑得更快。简直是划时代的。

拿现在的AI标杆企业OpenAI为例，其训练GPT-4的成本差不多是1亿美金，训练一次需要花大约6400万美元。这个数据只是参考，比如谷歌训练AI就花了更多的钱，还比如这次被抄袭作业的面壁科技做到了花相对更少的钱做出了性能一样的AI。

但是无论设计如何精妙，训练AI都要花费一笔巨额资金。为什么一定要花这么多钱呢？

训练一个AI贵是因为必须要运算海量数据。还是以OpenAI为例，OpenAI训练GPT-3用了大概753GB的数据，训练GPT-4用了大概上百TB的数据。

训练AI和小孩学习一样，小孩学得越多就懂得越多。AI也是一样，训练过的数据越多，模型越准确，AI性能越高。

无论设计多么巧妙，数据越多AI越强都是铁律。而运算海量数据就必须要海量算力；同时因为算力过于庞大，还得考虑如何维持算力平稳运行；说人话就是买芯片的钱和电费。

所以训练AI便宜不了，即使极限压缩数据和人力成本，算力成本依旧吓人。现在算力市场是卖方市场，买家多卖家少，算力自然便宜不了。

现阶段算力市场是英伟达一家独大，甚至可以说英伟达吃下了AI算力市场几乎所有的份额。无论英特尔还是高通都在AI芯片领域被英伟达压得死死的。

英伟达的优势在于买他们家的算力最划算，而且现阶段训练AI必须的CUDA软件是英伟达的独门秘诀。所以英伟达的高端AI芯片现在一卡难求，不少大公司整批整批的采购英伟达的芯片，就比如OpenAI背后的微软。

那么这个时候一定有朋友会问，既然要的是算力，那为什么不像超级计算机一样用多张芯片组在一起凑算力呢?

因为高端芯片的优势不光在算力上，更在能耗上。同样的算力下，高端芯片可以耗更少的电更快的算出结果。训练AI付的电费中有不小的部分浪费在给芯片散热上，高端芯片的一大优势就是减少这部分浪费。

这也是为什么美国针对中国的AI产业瞄准的是高端芯片产业链，因为凑算力不如直接买高端芯片划算，表面上中国公司可以通过凑算力来规避制裁，实际上制裁直接导致中国企业AI的训练成本和运营成本比美国同行高。

那么拜登的制裁有用吗？现在看，制裁在AI领域只起到了象征性的作用，因为AI的优劣不在于算力，而在于算法和接触到的数据量，这点中国和美国在同一水平线上。

就比如这次被抄的MiniCPM-Llama3-V 2.5。面壁科技的算力成本确实更高，不光算力更贵，电费也更高，但是训练出来的AI不比OpenAI和谷歌的当家AI差，这点Llama 3-V项目有实证。

Llama 3-V的性能经过验证确实强，这也反向证明MiniCPM-Llama3-V 2.5的性能足够强悍。然而面壁科技只是中国茫茫多AI公司中的一家，他没有最强的工程师，也没有最雄厚的资金支持。这足以说明中国AI在世界地位。

相反，美国的制裁导致了美国芯片公司在中国市场少赚了不少钱，还促使中国公司开始自研自产高端芯片。

更讽刺的是，拜登一直在促进高端制造回归美国本土，其中芯片就是重点项目。拜登计划在美国建造一个世界最大的集芯片研发、设计和生产的产业中心。那么问题来了，多造的芯片卖给谁？不卖中国可就没人有能力吃下这么多芯片，这个产业园不就是成立即荒废吗？

既然美国的制裁没什么大用，那么现在中外AI大战的局势到底如何呢？答案是各有优劣，没谁绝对占优，也没有谁完全掉队。现在AI大战的最大赢家反而是芯片产业，不是AI科技公司。

有心的朋友应该注意到了，抄袭的Llama 3-V和被抄的MiniCPM-Llama3-V 2.5的名字中都有一个Llama 3。Llama 3是美国脸书公司的开源AI，面壁科技就是基于脸书的Llama 3框架才做出的MiniCPM-Llama3-V 2.5。

注意，这里不意味着面壁科技只是拾人牙慧，或者说得难听些抄袭。面壁科技在脸书的Llama 3架构上做出了不少的改进。正如前文所说，架构和数据才是一个AI的核心竞争力，如何训练AI以及训练完后如何改进是难点，也是面壁科技的比较优势。

可以说面壁科技做到的事情，脸书未必可以做到。大家对面壁科技的误区，也是不少朋友对现在的AI开源以及互联网开源文化的误区。现阶段确实有不少中国公司用美国的开源AI做出了自家的AI，但是也有不少美国公司用中国的开源AI做出了自己的原创。

开源实实在在的促进了世界AI的发展，不光是中国的，也有美国的。从根子算起，AI的三大基础架构都是中国人提出来的。所以这里不是天下文章一大抄的环境，而是大家一同进步你好我好大家好的环境。

开源文化在互联网诞生之初就有，是极客们对权威的反叛，也可以说是小偷的狂欢。但是不可否认的是，当今世界并没有开源或者技术无国界生存的土壤，这两个理念固然美好，却一直是乌托邦式的美梦。

都说不要把技术问题政治化，结果碰到技术问题大家都会下意识的政治化。所以理想主义者们固然可笑，但是请不要嘲笑他们的理想。

现在美国的AI声浪确实大，但是中国的AI性能也不差，大家的差距其实拉开太多。

比如，谷歌的Gemini就爆出过中文内容抄袭百度的文心一言，再深挖会发现谷歌基本上抄了所有的中国主流一线AI。谷歌的例子说明，声浪大不代表真的强。

那么为什么现在AI热潮中的赢家是芯片产业呢？因为现阶段全世界的AI都面临着同一个问题，训练出的AI确实厉害，但是怎么用AI赚钱呢？

谷歌、OpenAI和微软这些科技巨头都已经开始提供大模型AI服务，但是这些业务的具体营收，巨头们都不怎么提。OpenAI在法理上甚至都不算是个公司，是个公益非盈利组织。

相反，那些专注于某一特定领域的AI在市场上的表现更好，比如抖音上专门抠图和加特效的AI，每一家科技巨头都在疯狂投资源进AI，但是现阶段没有企业告诉世界AI怎么赚大钱。

反映出的问题，就是一面AI在蓬勃发展，另一面各行各业还是需要真人才能干活。

不可否认，现在的AI已经展现出了替代人类的潜质。但是潜质是潜质，AI到底还有多久才能替代人类？替代人类有哪些技术难点需要攻克？AI代替了人那人如何在社会上生存？

这些问题没有答案，需要各位朋友在未来自己拼凑出答案。大家不必杞人忧天，也不能躺平觉得高枕无忧。只需要谨记一点，无论浪高浪低，聪明人总有赚头。
