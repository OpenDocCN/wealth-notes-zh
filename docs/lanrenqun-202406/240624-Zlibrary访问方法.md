# 2. 国内网络

## 2.1 代理网站

国内网络的话，可以利用代理网站，绕过网络限制。以下为网友总结的几个代理网站：

- https://proxyium.com/zh-CN/
- https://www.croxy.network/zh/

浏览器访问任一网站，然后在下方输入框里，输入任一Zlib官方网站即可访问。

![](img/cbdfae27df9d3f508512476a536af74b_2_0.png)

第一部分已经给出了ZLib网站了，这里还是再贴一下：

- https://zh.singlelogin.re/
- https://zh.singlelogin.rs/
- https://z-lib.gs/
- https://it.1lib.sk/
- https://es.1lib.sk/
- https://1lib.sk/

注意：同样需要登录Zlib网站才能下载，下方的推荐不能下载，只能通过搜索后下载~

## 2.2 套壳浏览器

Github上有网友开发了一个Zlib专用浏览器，其实原理也是走一层代理。

Github开源地址：https://github.com/louiesun/Unofficial_Z_Access/releases/tag/V5

下载里面的 `dist.zip`，解压后运行即可。

![](img/cbdfae27df9d3f508512476a536af74b_3_0.png)

为避免部分bro访问不了Github，小懒这里准备了一份压缩包。

下载地址：https://lazyman.lanzoul.com/inJ6n22kdmlg

这个浏览器除了zlib还内置了安娜的档案。
