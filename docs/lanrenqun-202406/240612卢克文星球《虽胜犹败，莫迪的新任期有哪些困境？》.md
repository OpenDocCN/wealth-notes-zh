

# 虽胜犹败，莫迪的新任期有哪些困境？

2024年6月4号，印度大选到了最终开票的日子，这场历时六周、选民10亿、热死上百人的选举，也终于有了结果。现任总理莫迪领导的印人党赢得240个议席，成为第一大党，但这一成绩远远不如2014年的282席和2019年的303席，且未及超越半数的272席“分水岭”。所以，印人党必须依靠其联盟中至少3个政党共同组阁，这是莫迪任期内印人党首次面临无法单独组阁的局面。难怪有人讽刺莫迪是“不败而败”。

那么问题来了，本来胜券在握的莫迪老仙，怎么莫名其妙地翻车了呢？

在大家的印象中，莫迪老仙作为印度的“天降伟人”，一直是法力无边，支持率非常高。其实，不只有中国人这么认为，就连很多印度人，包括莫迪自己，也是信心满满，喊出了“超越400个席位”的口号，这显然是要瞄准宪法改革去的（按印度法律，修宪需要超过三分之二）。但是现在莫迪的票数，面临的已经不是修宪的问题了，而是想独立组阁都不可能了，为啥翻车翻成这样了呢？直接原因，是一些印人党的铁杆票仓叛变了，分别是北方邦、马哈拉施特拉邦、拉贾斯坦邦、卡纳塔克邦等。

举个例子，北方邦，这个邦人口有2亿2792万，是印度人口最多的邦，而且在印度宗教文化上占有重要地位。因为人口多，再加上宗教重镇，所以该邦在印度下院占有80个议席，一直有“得北方邦得天下”一说，上次莫迪在这个邦拿到71席。可是这次呢？只拿到了32个席位。在其他几个重点邦的情况也差不多。

为啥这几个邦叛变了？因为莫迪的征地法案，把这几个邦惹毛了。我们都知道，近几年，西方国家为了“去风险化”，不停地把供应链从中国迁出，转移到印度。问题在于，工厂来了，发现没地方建厂！因为印度的土地属于各邦，而且根据印度《征地和重新安置法案》，规定征地方必须向农村和城市的土地所有者分别支付最高达到土地市价4倍和2倍的收购价格，而且征地时，必须事先取得80%土地所有者的同意。这使得拆迁难以进行，印度的外资企业最头疼的就是征地，连莫迪都无奈地说“显而易见，没有土地，修不了路，也建不了厂，吸引外资和印度制造无从谈起”。

所以在2015年，莫迪的印人党提出了《征地法案修正案》，核心内容是用以工业开发、房地产开发和基础设施建设的土地，可以绕过“80%土地拥有者同意”这一环节，直接进行市价补偿。但问题在于，印度土地属于地主，农民是打工人，征地之后，地主拿了钱，农民上哪去谋生？要么饿死，要么到城市贫民窟流浪，相比之下还不如种地。

更关键的在于，印度有很多大地主，农民对他们是依附关系，他们可以随意欺辱农民，如果征地改革了，征地之后自己是拿到了一些钱，但农民和他们的依附关系没了，上哪去寻找优越感？所以地主对征地法案也很反感。在各邦历来有集团投票（地主指使佃户农民倾向性投票）割据地方的传统，各邦选出的议员，就算再支持莫迪，也不敢背叛自己的出身啊，结果莫迪的征地法案不出意外地被卡了，吓得莫迪连续几年没敢提这事。

如果说征地改革失败过去莫迪能忍，现在莫迪已经忍不了了，为啥？因为再不改变就没机会改变了，现在是产业链转移关键时期，也是印度发展制造业千载难逢的好机会，过了这个村就没这个店了，印版“战略机遇期”绝对不能再被征地卡住。

所以莫迪在去年年底再次透露，自己第三任期开启后，会再次推行征地法案，这下很多人都炸了，北方各邦以农业为主，得罪了这里的地主和农民，莫迪怎么可能拿到优势票数？

除此之外，莫迪的画大饼作风，也开始让很多人厌倦。莫迪这个人，是个很好的画饼专家，在2014年竞选时，就画了很多大饼，比如2020年超越中国，2030年印度成为超级大国，很对心怀大国梦的印度人的胃口，于是莫迪顺利当选。此后，莫迪不断加强这种形象工程，一会说“21世纪是印度的世纪”，一会说“印度制造崛起”，一会登月，一会造航母，其实都是为了给印度民众打强心针，以一种虚幻的“大国梦”，变成印度人的良好预期。

学过经济的都知道，当预期好的时候，投资的欲望就强，消费的意愿也强，形成欣欣向荣的大好局面，吸引外资注入，这又反过来能够刺激印度经济发展，称为莫迪政绩的佐证。但问题在于，这种玩法，时间长了也会腻的！虚幻的梦想，比不上现实的残酷！

你看看莫迪竞选口号中提出的三个重点，青年，妇女，穷人，号称要给青年更多的就业机会，给妇女更多人权保障，给穷人摆脱贫困的投入。但是结果呢？印度青年人占失业劳动力的近83%，针对女性的强奸案频发。至于穷人就更不用说了，莫迪的农业改革法案几乎把所有农民佃户都惹毛了，开着拖拉机去抗议了。莫迪搞了这么多年，虽然从数字上GDP增速飞快，一度达到8%，但人均GDP平均每年增长仅为4.3%，这也意味着，印度的人均GDP水平是在下降的，甚至被隔壁的孟加拉国反超了，把许多印度网友气得够呛，这也是莫迪票源流失的原因之一。

还有在这次大选之前，莫迪还安排执法部门和税务机关逮捕了一大批政客（其中包括德里首席部长），这些人都是反对派的，甚至连国大党的银行账户都被冻结了，国大党连竞选的钱都拿不出来。更不要说印度公然刺杀锡克人领袖的事情了，给印度制造了不小的外交纠纷。搞政治应该徐徐渐进，莫迪这种激进作风，就算他真的能压制国内舆论，营造出一片莫迪形势大好的印象，但大家心里是有杆秤的。如果不是反对派联盟太过拉胯，内部互不服气的话，恐怕莫迪还会更危险。

从现在的情况来看，莫迪只能联合其他党派成立执政联盟组阁。拉谁呢？主要是两个地方党派：泰卢固之乡党（TDP）、人民党联合派（JD(U)）。泰卢固之乡党是印度安得拉邦地方政党，也是印度全国最大的地方政党之一，其领袖是安得拉邦首席部长奈杜，这次大选拿到了16个席位，虽然是莫迪的盟友，但这个党派却反对“印度教优先”的做法。而人民党联合派（JD(U)）的根据地是比哈尔邦，领袖是首席部长尼蒂什·库马尔，这个邦人口超过1亿，这次拿到了12个席位。这个党的领袖库马尔和莫迪关系很不好。

有趣的是，这俩党在不久之前，都还是国大党的盟友，一直到年初才“反水”加入了莫迪阵营。这种左右横跳的属性，使得他们可能会狮子大开口，要求分享内阁部长的任命权。所以6月4日晚上，选举结果出来之后，莫迪亲自会面了两党领袖，就利益分配进行了商议，奈杜出来后就宣布支持莫迪。除此之外，这俩党还可能在议会中与莫迪讨价还价，对莫迪的各项政策各种掣肘，借此谋取更多的党派和地方利益。也正因为如此，大选结果出来后，印度的股市、汇市、债市都出现了比较明显的雪崩，显然，在权力分散的形势下，外资对于莫迪还能否持续其现有政策，也是非常怀疑的。

那么在这种形势下的莫迪第三个任期，会面临什么困局呢？首先，是莫迪的宏伟蓝图将遭遇巨大挫折。莫迪其实任上主要死磕三件事，一是国族再造，二是印度制造，三是以人民为抓手加强中央集权。国族再造，就是强化印度教民族主义，用印度教的宗教认同，代替一盘散沙的种族认同，借此创造出一个完整的“印度人”认同，同时通过孤立伊斯兰教和其他宗教，来增强印度教徒群体内的凝聚力。印度制造，就是借着产业链转移的东风，尽可能多地承接从中国转移出来的产业链，借此实现印度的崛起。中央集权就更好理解了，印度这种“一盘散沙”的局面，中央说往东，地方说往西，想靠“印式民主”把国家发展起来，根本没戏，必须依靠集权。但地方各有各的利益，很难让他们就范，于是莫迪就换了个方式，以印度教为意识形态，以沙卡准军事化基层组织，然后再以印人党通过地方选举来掌握地方行政权力，以党权支撑中央政权，最终实现中央集权。

从效果来看，莫迪其实干得不错，在近两年的地方选举中，印人党已经掌控31个邦中的20个。但问题在于，这是在莫迪占优的情况下实现的，现在印人党尴尬的选举结果，显然这三条强国路径未来都将困难重重。比如泰卢固之乡党，是泰卢固人自己的党，成立的主要目的，是争取泰卢固人的权益，所以一直在搞“语言邦运动”，说白了就是说自己的语言，不说印地语，明摆着不会把自己同化为“国族”的。那么如果莫迪想再出一些国族政策，肯定要受到泰卢固之乡党的掣肘，考虑到泰卢固之乡党属于执政联盟，莫迪也不敢闹得太僵。还有制造业的问题，莫迪玩的是韩国模式，搞了一群亲自己的财阀（比如印度首富阿达尼），借富豪的手去集中精力搞制造业。但问题在于，一方面这里面产生了严重的腐败和分配不均，在莫迪上台的2014年至2022年期间，印度亿万富翁的净财富增长了280%以上，是同期国民收入增长（27.8%）的10倍。另一方面还是绕不开制造业征地的问题，产生财阀与地方的矛盾。而人民党联合派呢？巧了，其口号就是反腐败，同时也是地方利益代表，很难把土地让出来搞制造业。所以莫迪的白手套阿达尼旗下的公司的股票，6月3日还创下历史新高呢，结果4日就暴跌了20%，其主要上市子公司股价甚至跌了25%。至于加强中央集权，跛脚的莫迪可能无力推动这一战略了。

所以，莫迪的第三任期，莫迪可能会把精力主要放在民生和外交上。民生方面，主要是想办法“有增长无就业”的窘境、抑制通胀、解决发展不平衡等问题，让更多的印度人从印度发展中获得红利，借此来挽回自己的支持率。而外交方面，主要还是亲美、疏中、欺巴。亲美不必说，美国主导的制造业转移，并不是只有印度一家，印度、越南、巴西、墨西哥乃至印尼都是潜在对象，所以印度必须讨好美国，用行政手段拉投资。疏中，主要是和中国保持距离，甚至制造一些摩擦和紧张气氛，借此谋取美国的肯定。但以莫迪“无利不起早”的习惯，与中国发生大规模冲突的可能性很小，冲突规模取决于美国给的有多少，就算美国给得多，以莫迪的作风，大概率也是糊弄了事。还有欺巴，印巴边境可能会迎来新一轮的冲突。不过民粹主义是一把双刃剑，极端民粹主义和印度教密不可分，火一旦点起来，烧向哪里，可就不好说了。所以，莫迪的第三任期注定会如履薄冰。当初的印度之光尼赫鲁，同样是连任三届，同样是风光无限，但在第三届的时候遭遇了中国对印自卫反击战，气的中风，两年后病逝。那么莫迪结局会如何？我们拭目以待。
