

# 极右翼大胜，欧洲要变天了？

6月9日，欧洲议会选举结果公布，极右翼取得了空前的胜利，其中在法国，极右翼的国民阵线拿下了31.4%的席位，而马克龙所在的复兴党不到15%。

在空前的惨败下，国民阵线要求解散法国议会，提前选举，马克龙也顺应了他们的要求，宣布解散了议会。

隔壁的德国，极右翼的选择党拿下了16.4%的席位，名列第二，超过了绿党和社民党，虽然基民盟拿下了30%，高居第一，但如果看地域分布的话，选择党在东部已经取得了明显的优势。

于是很多人说，欧洲要变天了，这是不是真的呢？其实，这并不完全准确。

欧洲议会有六个主要党团，分别是“欧洲人民党”、“社民党党团”、“欧洲复兴党团”、“保守与改革党团”、“身份与民主党团”、“绿党党团”、“左翼联盟党团”。

| 党团名称 | 席位数 | 备注/排名 |
|---|---|---|
| 欧洲人民党 | 189 | 排名第一，成员包括德国基民盟、法国共和党等建制派右翼，领袖为冯德莱恩 |
| 社民党党团 | 135 | 排名第二，成员包括德国社民党、西班牙工人党等建制派左翼 |
| 欧洲复兴党团 | 83 | 排名第三，中间派核心（法国复兴党），较上次减少19席 |
| 保守与改革党团 | 72 | 由东欧和南欧国家的极右翼组成 |
| 身份与民主党团 | 58 | 由法国和德国的极右翼组成，较上次大幅增加 |
| 绿党党团 | 53 | 欧洲绿党联盟，较上次减少18席 |
| 左翼联盟党团 | 35 | 排名最后 |

从上面的席位数就能看出，极右翼虽然有了壮大，两个极右翼党团加起来有 130 个席位，这是前所未有的，但从数量上看，建制派左和建制派右依然处于绝对优势，两者合计有 324 席，短期内还是建制派说了算。

而且，极右翼自身也不团结，东欧和西欧的极右翼，当然在反对 LGBTQ、反对移民等方面有共同价值观，但他们也有不少的分歧。

尤其是对待美国的态度，东欧国家处于抗俄的第一线，不管极左还是极右，对俄罗斯都极为恐惧，非常需要和美国合作，引入美军。而西欧不处于抗俄一线，反而对俄乌战争造成的负面影响颇为不满，当地的极右翼自然不希望看到那么多美军，这样的分歧让两者无法合力，才会弄出两个党团。

并且，极右翼的席位数上升，还与他们对基本盘的动员能力更强有关。这里要普及一个冷知识：领导人选举，是全民主的，而议会选举则很大程度是靠基本盘。因为很多人，尤其是年轻人对于频繁的选举不感冒，往往只参加领导人选举，而懒得去给议会选举投票，毕竟选举是需要花时间、花精力的。

投票可能得排队几个小时，要被太阳晒，皮肤要变黑，不少人嫌麻烦就干脆不参加。愿意参加的，大部分是政党的基本盘。而极端派对基本盘的动员能力远强于中间派和建制派，在议会选举中就会更占优势，席位数通常大于实际社会支持率。

这次欧洲议会选举，极右翼能够大胜，也有这样的因素。

所以，欧洲变天的说法是夸张了，短期内还不至于变天，不过，这不代表一切都好，本次欧洲议会大选体现的问题，还是很明显的。

首先，是绿党党团损失惨重，比上次少了18席，总席位数已经远远少于极右翼。在俄乌战争的大环境下，反俄反华亲美属于政治正确，全世界都知道绿党亲美，此时大选是有利于绿党的。但是俄乌战争带来的通胀飙升、工业外流等问题，令许多人迁怒于绿党，又不利于绿党。所以本次欧洲议会大选，是欧洲绿党的一次大考。

如果他们支持率上升，就说明用政治正确绑架欧洲的做法成功了，如果支持率下降，则证明失败。

而结果表明他们失败了，绿党那套强行绑架的做法，除了他们那一点基本盘以外，并不得人心，这对中国来说是好消息。俄乌战争这两年，我们最害怕的就是美国通过政治正确捆绑欧洲，让欧洲彻底倒向美国。但现在看来，欧洲人没有那么傻，绿党的整体支持率很难突破 20%。只要绿党无法成为第一大党，无法单独执政，欧洲就不会整体倒向美国，这也让我们松了一口气。

## 其次，是极右翼在德法的空前壮大。

其实就席位而言，极右翼在本次欧洲议会大选增加的数量不算多，但是偏偏在德国和法国，这两个欧盟核心国家，实现了大幅增加。这也是外界惊呼“欧洲要变天”的原因。

在法国，败得特别惨，仅得14.9%，勒庞领导的国民联盟，能在法国拿下31.4%的欧洲议会席位，可谓出人意料。如果看选区的话，国民联盟在大部分选区都取得了优势，而不只是个别地区，这才是最令马克龙破防的。

而在德国，虽然基民盟成功在全国范围内压制了选择党，但选择党在东部，也就是原东德地区的优势十分明显，东德地区正在变成选择党的势力范围。如果说，前几年极右翼在德法的基础还不算太稳固的话，那么现在他们已经日益稳固了。

考虑到法国和德国的政治碎片化本来已经很严重，法国是总统制，议会的制约没有那么大，还好一些，议会制的德国就难搞了，没有一个党支持率突破35%，要好几个党联合起来，才能勉强执政。过去三年，由社民党、绿党、自民党组成的“交通灯联盟”，用一事无成来形容不为过，三党本来就南辕北辙，你要向东，他要向西。

政治上的根本分歧，导致无法正常施政。而极右翼的壮大，势必让德国政坛的碎片化进一步加剧。

对中国来说，极右翼在欧洲的壮大，有好有坏。

好的一面是，极右翼更务实，他们虽然口头反华，但不排斥对华合作，比如意大利上一届的极右翼政府，就和中国签订过“一带一路”合作协议，德国选择党也明确支持“一带一路”。退一步说，即使欧洲极右翼也反华，那也比白左好得多，极右翼的反华是有得谈的，无非是利益的分配问题，而白左纯粹是无脑反华，是没得谈的。

就像懂王在位时，我们还能和他达成重要的协议，而拜登，你压根没法谈，朝令夕改、出尔反尔，谈了也是白谈，不可能得到落实。所以，我们并不反对极右翼在欧洲的壮大。反倒是美国，有得头疼了。

欧洲向右转，意味着在意识形态上，和美国民主党的碰撞会越来越大，两者逐渐水火不容，俄乌战争要继续进行下去，就不能指望欧洲继续出力。欧洲援助乌克兰越多，社会不满情绪就越大，极右翼的支持率就越高。欧洲不出力，就只能是美国出力，那样俄乌战争就会变成对美国的放血。

原先想放欧洲和俄罗斯的血，现在变成了放自己的血，史诗级的回旋镖。

也许有人会说，欧洲极右翼和美国极右翼，意识形态不是一致的吗？懂王上台，大家一起右转不就行了？

这其实更不现实了，各国的极右翼都是希望本国优先，各家自扫门前雪。双方本来就在贸易、产业等问题上，有着不小的分歧，比如波音，常年在世界各地打压空客。空客之所以忍气吞声，很大程度上是看在美国控制欧洲的份上，不敢公开化矛盾。要是大家都是极右翼执政，你优先我也优先，冲突必然爆发，多年精心构筑的美欧关系网，将面临崩塌。

颇为讽刺的是，走到这个十字路口，必须感谢拜登策划的俄乌战争。

俄乌战争前，虽然极右翼已经在崛起，但速度还是比较缓慢的，在法国拿下欧洲议会选举的第一名，则更是难以想象。是俄乌战争，给欧洲社会带来了强烈的不安全感，让越来越多人认清了现实，才给了极右翼壮大的土壤。原先在美国的计划中，这种不安全感会让欧洲全面倒向美国。

然而事实是，欧洲只是表面上看起来倒向了美国，平静的冰层下，是不断涌动的暗流，欧洲，已经来到了巨变的前夜。
