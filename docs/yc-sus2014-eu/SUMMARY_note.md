# Y Combinator 创业教程 第1课：课程概览与导师介绍 🧭

在本节课中，我们将一起了解Y Combinator创业教程的整体框架，并认识本系列课程中即将登场的各位重量级导师。通过他们的分享，你将获得来自顶尖创业者和投资人的第一手经验与洞见。

上一节我们明确了课程目标，本节中我们来看看本系列教程的具体内容构成。

这个列表代表了本教程的核心内容模块，每一讲都由一位或多位在创业领域取得卓越成就的导师主讲。他们将从**产品构建**、**团队管理**、**融资策略**、**市场扩张**等不同维度，为你系统性地剖析创业成功的关键要素。

接下来，让我们简要了解一下部分导师的背景，这有助于你在后续学习中更好地理解他们的观点：

*   **Adora Cheung**：家政服务平台Homejoy的联合创始人，她的课程将聚焦于从0到1启动一个创业项目。
*   **Alfred Lin与Justin Kan**：前者是红杉资本合伙人、Zappos前COO，后者是Twitch联合创始人，他们将搭档分享关于规模化增长与团队文化的见解。
*   **Patrick Collison**：在线支付巨头Stripe的联合创始人兼CEO，他将深入探讨技术驱动型公司的构建与创新。
*   **Paul Buchheit**：Gmail的创建者、YC合伙人，他发明的“**Don‘t be evil**”口号和关于产品市场契合度的思考极具启发性。

每位导师都带来了独特的视角和经过实战检验的方法论。在后续课程中，我们将逐一深入学习他们的核心思想。

本节课中我们一起学习了Y Combinator创业教程的整体结构，并初步认识了即将引领我们探索创业世界的各位导师。从下一课开始，我们将正式进入第一讲，跟随Adora Cheung学习如何迈出创业的第一步。准备好开始这段旅程了吗？让我们继续前进。
