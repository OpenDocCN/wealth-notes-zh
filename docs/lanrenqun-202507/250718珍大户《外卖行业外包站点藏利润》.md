

# 外卖行业外包站点藏利润

250718 珍大户

整理：公众号懒人搜索，懒人专属群独享

懒人微信：lazyhelper

![](img/2e5ae0cd7b27a35aca517cdf8f705d2b_0_0.png)

@总钻风问“京东藏利润啥意思”，是我写东西省略过头了，意思是“之前点评京东美团外卖大战时，说的外卖行业藏利润这件事”，当时说的是，美团的外卖环节的利润在外包站点上，是有表外隐藏的———支付给外包站点环节，显示不到财报里，算是一种环节上的贪污，或者说环节上的损耗。

京东砍掉外包站，外卖员直营的话，类似等于供应链环节砍掉中间商。

就类似于说，如果一个卖花盆的企业投50%提成比例的大V带货，然后又拼命说自己不赚钱，很累，然后从财报上，你也确实也看不出来利润，因为成本很高。

但指不定非要投大V带货是因为大V给返点了，支付出去的成本在私下返还给管理层，就方便进个人口袋。

此时如果另一个企业下场，不找大V带货，而是自己经营直播间，是能实现环节节省而盈利的。

此时如果外界一直叫唤，说买花盆就是一个低利润不赚钱只能做慈善的行业，不信你看原来只有a企业活着，你看财报吧啦吧啦。

为什么我说外卖行业的外包站点一定藏利润了？我提问一句，大家想想：你见过转让美团外卖站点的吗？

你想过去经营美团外卖站点吗？我们圈子马上7周年了，这七年里，我收到过至少6次以上的提问“能不能接手菜鸟驿站”的提问，至少3次以上的“能不能接手香烟店”的提问，至少3次以上的“能不能接手彩票店”的提问。但是没收到过一次“能不能接手外卖站点”、能不能接手“茅台分销商网点”的提问。

哪个赚钱，哪个不赚钱，可见一斑。

但神奇的是，所有买茅台的股民都能bb几句，说茅台有大量利润藏在茅台分销商手里，只要取消经销商，茅台上市公司的利润能翻倍，因为市场价2500，给经销商的价格1300。

但没有任何买美团的股民bb过，“有大量美团的利润藏在外卖站点，只要取消外卖站点，美团盈利能翻倍，因为每个外卖站点都年利润几百万上千万”。

我现在非常好奇的想知道，加盟一家美团的外卖站点，是什么样的条件，普通人有可能经营的了么。大家可以搜索一下做一下功课，看看条件是什么样的。

这部分基本没见买卖美团的股民去提，当初我最早说京东能赢的时候，所有人都在跟我 bb 外卖行业的利润是有多低，非说只有美团才能精耕细作，在微薄利润的行业里勉强盈利。说京东做不到。

在我看来，增加外卖站点外包这一层，本身就是管理能力弱，管理外包的结果，包括淘系，抖系，主要经营的都是【平台】，所谓平台，其实就是把商品的“周转率管理”外包给各个商家自己，压货、积压、退货等等成本，是由商家承担，把物流速率管理外包给各个快递公司，货运速度由快递公司承担，这些各种环节外包，本就是管理能力弱的结果，如果管理强势，则一定是各环节自己一手抓的。

京东擅长的恰恰就是供应链，是各种货物库存的“进存销”调配，是物流运转管理能力，这些一同构成了商业上很重要的“周转率”的概念，他们能把商品价格做到跟其他平台几乎持平，说明进存销调配能力很强大了，能把物流价格做到比顺丰低，还时效快服务好，从四通一达手里抢份额，说明物流转运管理的能力已经超级强了，这种精细化控制的能力，是普通平台所比拟不了的，这才是京东的护城河。

至于美团，随着时间推移，大家早晚会看到，外面网友讨论的什么美团的“护城河”都可笑的不得了。

美团没有什么可以称道的护城河，经营年头根本不算，以前能赢是因为竞争对手都太弱，不是美团实力强，而是当时竞争外卖的对手，一个是做什么什么不行的百度，一个是收购什么什么做不好的阿里，都是被对手映衬的。

最近阿里闪购把美团映衬的又变熊包了，原因还是阿里把饿了么收归本部了，阿里本部实力不弱，弱的都是之前各种收购后外部放养的。

之前只有京东和美团两家的外卖大战里，京东要用自己的供应链优势，去管理外卖派送供应链了。

这种优势，往大说，有可能性能掀翻美团，往小说，能吃掉10%～30%的市场份额，但只要保守业绩达成，就能证明美团的护城河没有，能证明自己可以靠外卖加持增加电商业务，从而整体获益，那么股价一定会有积极表现，这是我当时看好狗东，看衰美团的原因。

现在出了一点变数，就是阿里亲自下场了，阿里全平台的电商吸引力是整体大过京东的，用户数量、打开频次都高于京东，这对京东绝对是个杀伤力。

所以目前能看到东子蛰伏了，敌强我弱，先藏，但是这一仗不可能完全放弃，因为如果东子在商场上还有野心的话，想要做大就不可能不发展，发展的话必然要多战线走路，没有更大的基础业务做匹配，光靠稳定币是翻不出大水花的，要有足够多的底层业务规模做支持，那么加持稳定币才能再发展出一个“跨国地下钱庄”功能来，基础业务就包括电商要扩的更大，也包括要向海外发展。

电商部门规模又依赖“用户数量×用户频次”，怎么算做外卖都是既能拉新，又能增加打开频次的好主意，所以外卖业务不可能放弃，目前在阿里的强龙势头下，只能暂时蛰伏，就是“敌强我弱先藏”

那么后续的路线，很可能是“敌疲我扰”，要等两家势头弱一些时，再找什么借口做营销吧。
