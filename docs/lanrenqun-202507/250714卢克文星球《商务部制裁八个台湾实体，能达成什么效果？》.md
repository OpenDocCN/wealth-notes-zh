

# 商务部制裁八个台湾实体，能达成什么效果?

250714 文/卢克文工作室嘉宾 特拉维斯
整理：公众号懒人搜索，懒人专属群独享
懒人微信：lazyhelper

![](img/dbba3875da7dc63c4896ee57f3e93a48_0_0.png)

2025年7月9日，商务部公告，将汉翔航空工业股份有限公司等8家台湾地区实体列入出口管制管控名单，禁止向上述8家台湾地区实体出口两用物项，正在开展的相关出口活动应当立即停止。

那商务部为什么要把这8家台湾实体列入清单？这背后有什么战略考量？

我们来看看这8家实体都是什么来头，根据商务部名单这8家实体分别是汉翔航空工业股份有限公司、经纬航太科技公司、台湾中山科学研究院、仲硕科技股份有限公司、国际造船股份有限公司、中信造船股份有限公司、龙德造船工业股份有限公司、攻卫股份有限公司。

这里，首当其其中的就是台湾中科院。

台湾中科院是台湾军事装备领域集研发、试验、生产于一体的核心单位。相当于美国的“国防高级研究计划局”加洛马与雷神的复合体，具有极高的战略价值。我们熟知的雄风系列反舰导弹、天弓系列地对空导弹、云豹装甲车，还有各型舰载雷达、指挥系统、空空导弹都由台湾中山科学研究院研发生产。研究院董事长由台湾所谓的“国防部长”兼任，院长均由台军现役军官担任。

其次是汉翔航空，原是台湾中山科学研究院下属航空工业发展中心，2014年又改制为民营公司，董事长和总经理均由台空军中将以上退役人员担任。上世纪90年代自主研发的经国号战斗机是其明星产品。进入新世纪后则主要以教练机、运输机为主，同时也承担F16、经国号战机的维护升级任务。

这两家主体可以说是未来台海军事冲突中，台军最具威胁武器的研发和生产单位。雄风2E导弹可直接威胁我国沿海城市，而F16和F-CK-1经国号战机则是我们夺取制空权的阻碍。导弹、雷达、航空零部件等对稀土依赖极大，且短期内无法替代，特别是钕铁硼磁材等材料的部件。此次被限制无疑将对这两家实体造成战略级打击。

再来看看经纬航太和仲硕科技，这两个实体又是做什么的呢?

经纬航太由前汉翔航空工业公司总经理罗正方于2024年创建，公司以卫星定位技术、遥测技术与地理空间资讯技术为核心，为台军及政府提供高精度卫星地图、定位导航、影像分析、无人机侦察等产品和服务。经纬航太的卫星影像数据主要有两个来源，一个是台湾自己的福尔摩沙系列侦察卫星，你没听错，台海之上是有台湾自己的侦察卫星的，能够提供2米分辨率的彩色地球影像。另一个则是近些年大力发展的小型智能化无人侦察机。

仲硕科技成立于2001年，主营各类军民两用无人机制造，其生产的AT100M狙击手和AT250M毁灭者，号称全球最具性价比的自杀式无人机，能搭载1至4.5公斤炸药，实现视距外打击。此外还有HY100猎手无人机，可搭载不同模块任务包实现空拍测绘、情资侦搜等功能。2019年和2023年分别从台军和台海巡署获得185万和2.1亿的军用无人机订单。

之所以把这两家实体也列入管制清单，想必大家也能理解。

俄乌战争开始后，无人机彻底改变了战场的作战模式，尤其是今年6月乌克兰发动的“蛛网行动”仅仅是用百十来架无人机，就摧毁了俄罗斯几十架军机，其中还包括几架战略轰炸机和一架预警机，损失估计达70亿美元，战争收益比惊人。在当下世界各国都加大了无人机在军事领域的应用，台湾省自然也不例外。而经纬航太的高精度卫星图像、GIS地形数据，可以协助台三军联合指挥所构建模拟作战地形，大大提高了台军对战场的态势感知能力，和信息化能力。可以说把这两家实体列入名单，就是瞄着台军的无人机战、信息战等新型作战能力打。

我们继续看国际造船、中信造船、龙德造船三家实体。台湾国际造船厂主营船舶建造，像油轮、货柜船、矿砂船、散装货轮等，另外也是台海军舰艇的核心生产单位，主要生产5000吨以下的舰艇，导弹快艇、运输舰、登陆艇、巡防舰、港巡艇、缉私舰等。比如万吨级的玉山两栖船坞登陆舰，5000以下的各级巡防舰如成功级、康定级和沱江级巡防舰。此外还是台湾唯一具有潜艇自主设计建造能力的船厂，设计制造了2500吨级的海鲲号柴电动力潜艇。

剩下的中信和龙德两家船厂，除了民用船舶外，主要是为台军和海巡署造一些小的巡逻艇，一般在1000吨以下，而且还为新加坡、菲律宾等国生产警用巡逻艇。此次将这三家实体列入清单，对台湾国际造船厂的影响还是比较大的，剩下两家影像较小一些但也不得不面临更换供应链的风险。

最后是攻卫股份，2008年成立，主要为台湾政府及军方提供军规级防弹衣、盾牌、特勤服、防割手套等这类消耗品。乍一听“股份公司”感觉规模应该挺大，但是实际上根据台湾官方媒体信息显示，这家公司员工数小于 25 人，而且 2019 年在为台军供应防弹布料的合同中，违规使用大陆防弹布料代替自产布料，企图蒙混过关，结果被台国防部调查后剔除供应名单。

根据公开的招标信息显示订单和生产量级，结合台湾官方媒体的爆料证据推测，攻卫股份极有可能是一个“台湾贴牌倒爷”，通过关系从政府拿标，找大陆企业生产，回台湾再贴牌，不得不说实在是玩的花啊。

虽然台湾能提供防弹衣的企业还有像利勤实业、勤崴国际等安防生产企业，但是无论从规模还是台军订单数量都和攻卫没法比，攻卫依然是台军及政府最大的安防装备供应商。所以攻卫依然被纳入的实体清单。

到这里，我们可以看出这 8 家实体几乎是覆盖了台军的全部军事工业单位，而且连警务安防部分也覆盖了。但是稍微了解下就知道，根据台军的《国防采购法》，台军方一直要求其军事生产单位的供应链排除中国大陆的原材料，打造所谓的“非红供应链”。

这个实体清单的制裁有用么？能打击到台军么？能，可以说是釜底抽薪。首先虽说台军的想法是美好的，可是现实是骨感的。

中国大陆是工业门类最齐全的，产业链丰富且价格具备很强的市场竞争力。根据台湾海关的数据，从大陆进口的航空配件、光学精密仪器、各类合金金属及矿产常年来一直是台湾进口的大头，占其全球进口份额的20%-30%，来源地排名第一。

尤其航空航天中的稀土类配件，连美国都无法避免，别说台湾了。像军规级防弹材料等一些非独家产品，虽然可以用其他国家代替但是替代成本普遍较高。

其次，这次限制的是军民两用物项，也就是说连民用部分也给砍掉了。

而这8家台军企业呢，除了生产军用品，更多的是还生产民用部分，有的占比还不低，比如汉翔航空年营收390亿新台币，利润22亿左右，台军订单占比为40%左右，台湾国际造船厂占比也是低于50%，如果没有民用部分的利润支撑，军方订单也几乎保障不了，就像美国现在造船业萎靡，利润撑不起产能，关停不少船坞，连军舰都得去外国修。

可以预见未来几年，这几家台军企业将面临严峻的产能与盈利挑战。

我们再说说到为啥是现在这个时间点？以前为啥不限制？因为情况已经十分紧迫了，现实和策略都发生了转变。

我国商务部根据《出口管制法》与2020年9月公布《不可靠实体清单规定》和《出口管制清单制度》，其主要目的是为了保护国家安全与利益，对外国企业或组织的敌对行为做出回应，故主要的应用对象是外国企业，有点类似美国的实体清单和SDN制裁清单。

那么台湾作为中国的一个省，一开始自然是不适用的，即便一些进出口类目比较敏感。

一方面是2024年赖清德上台，就职时提出两岸“互不隶属，台湾是主权国家”两岸关系全面紧张化，台独势力加速“实质台独”。提出17项台独措施，包括恢复军事审判制度，严查陆配，禁止民间交流，加强公职人员赴陆审查，严查社群媒体网红统战宣传等。

而且，美国也改变了对台独的支持策略，不再依靠对台军售这种慢节奏工具，转而在2024年开始推进对台《紧急军援机制》，演练战时快速调拨美军库存武器到台湾，企图将乌克兰模式复制到台湾。以前是那个美国实体卖武器给台湾，我们就可以将该实体加入清单，但是现在是美国直接调拨库存，总不能把美国政府放到清单里吧。

所以我们对实体清单工具的运用逻辑也发生了转变，把台湾企业当作是“违法协助敌对势力的单位”对国家主权、统一和领土完整造成威胁，依法适用国家法律进行管理和处罚，这样在法理上自然不存在任何问题。

另一方面，最现实和直接的原因是自赖清德上台以来，台军加速备战，未来两三年将是台军武器装备发展和生产的重要阶段。试想一下，如不立刻快速遏制，那在我们统一的时候，这些武器将对我们的解放军构成巨大威胁。

所以此次将台军事工业踢出中国产业链，也是顺理成章的动作了。
