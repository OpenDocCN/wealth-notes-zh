

# 今天的硅谷，像个巨大的华夫饼

今天我们从一个硅谷的消息说起。上周五，7月11日，谷歌的DeepMind被爆收编了AI初创公司Windsurf的核心团队。说白了，就是挖人，谷歌开出的价格是24亿美元，相当于人民币170亿左右。

而在此之前，据说OpenAI已经和Windsurf谈收购问题，而且金额是30亿美元，比谷歌还要高。但不知道到底是什么原因，也许是条件没谈拢，也许是理念不合，也许是别的什么因素，总之，OpenAI这笔收购没有兑现，反而是被谷歌给“截和”了。

抛开具体的细节不说，你有没有觉得这件事的整体画风有点眼熟？没错，就和一个月前Meta挖人的情况差不多。

今年6月，Meta宣布要重组AI团队，宣布成立超级人工智能实验室。那么事情从哪入手呢？最关键的切入点之一，就是疯狂地挖人。

其中最有代表性的，就是Meta用2亿美元的薪酬包，挖走了苹果的前基础模型负责人庞若鸣，合同期限4年，平均每年5000万美元。什么概念？这比庞若鸣的前老板库克都要高。相当于1.5个2024年的妮可·基德曼、1.1个2023年的汤姆·克鲁斯，或者1.1个超级名模吉赛尔·邦辰。

说得再具体点，有人曾经算过，汤姆·克鲁斯平均说一句台词的价格是6000到7000美元，没错，一句话7000美元。而吉赛尔·邦辰巅峰时期的纪录是，在T台上每走一步45万美元，注意，是一步45万美元。而现在，庞若鸣比这些人通通都要赚钱。

而且这还只是Meta砸钱的冰山一角。光是挖人这一项，Meta就先后挖走了OpenAI多模态模型核心人员毕树超、OpenAI o3-mini的研发者任泓宇，还有Scale AI的前CEO亚历山大·王来担任新团队的负责人之一。据说为了请亚历山大·王，Meta是直接花了143亿美元收购了他公司49%的股份，相当于买公司送CEO。

特别值得一提的是，目前可查的Meta从各个公司挖走的AI科学家，一共14人，而其中有9人是华人，而且是清一色的年轻人。其中作为负责人之一的亚历山大·王只有28岁。

在这里姑且做个猜测，Meta的超级人工智能实验室，可能囊括了全世界最赚钱的一群“90后”雇员。

但是，你可千万别觉得Meta只是靠蛮力砸钱而已，其中还有很多微妙的设计。

比如，在挖人期间，Meta采取了48小时的极限施压策略。你同意也好，不同意也罢，就给你48个小时的时间考虑。行就签字，不行就拉倒，总之过了这个村就没这个店。而一旦你签字同意，Meta马上就启动后续的HR流程，行动相当迅速。

再比如，这场挖人不仅有“利己”的成分，也有打击对手的意味。据说这回挖走的AI科学家里，有不少都是OpenAI多模态团队的核心人员，他们的出走可能会导致OpenAI的相关技术进展明显推迟。

而且最关键的，Meta这回在人才上的投入只是整体策略的一部分。除此之外，Meta还启动了“算力霸权”战术。比如，和台积电签订独家合约，提前锁定了台积电2025年H200 GPU产能的70%。什么概念？H200是目前AI领域最依赖的GPU之一，而全世界只有台积电代工生产。现在等于宣告，2025年台积电生产出的H200 GPU，七成归Meta，其他公司只能分剩下的三成。

Meta还建立了新的数据中心，叫AI沙丘，据说算力密度是OpenAI的5倍。扎克伯格承诺，Meta将为AI科学家提供不设上限的算力支持。说白了，只要你有想法，钱上面我绝对不差事儿，奉陪到底。

这场挖人大战，只是目前硅谷AI博弈的一个缩影，关于AI的竞争从来都没有停止过。

那么，作为挖人大战中最大的员工流失方之一，OpenAI是什么态度呢？CEO山姆·奥特曼在一次访谈中说了两点。首先是惊讶，对于对手开出的天价薪酬，连投资人出身的他都感到惊讶。其次是表达信心，奥特曼说，直到目前为止，团队里最核心的人才依然没有被挖走。他还说，挖人不是光靠钱就行的，还有企业的愿景、企业的文化，这些都是比钱更重要的东西。

当然，就在奥特曼说这话的同时，据说OpenAI已经在给员工涨工资，而且根据科技媒体报道的数据，OpenAI过去一年的员工保留率，还不到70%，有三分之一的员工都流失了。

但是，这和奥特曼说的话并不冲突，因为在今天的硅谷，各个公司确实已经分化出了泾渭分明的企业文化。

几十年前提起硅谷，很多人都会笼统地觉得，这里虽然有很多企业，但公司之间都有着某种相对一致的气质。

比如，极客精神、工程师文化、技术至上主义等等。之所以会这样，是因为当年的硅谷刚刚起步，有很多公司的创始人都是同宗同源的师兄弟。比如，当年从肖克利实验室走出的“八叛徒”，创立了仙童半导体。后来八人中的诺伊斯和摩尔又创立了英特尔，杰里·桑德斯又创立了AMD，在1960年代，硅谷半导体公司的高管里有37%来自仙童。

因为经历类似、领域相同，所以当年的硅谷就像一棵大树，看起来有很多分叉，但很多枝丫都来自同一个根系。

而现在呢，借用刘润老师的比喻，硅谷更像一个巨大的华夫饼。远看这群人好像有很多相似之处，但近看的话，里面全都是一个一个小格子。因为过去这些年，公司之间的业务模式不同，彼此之间已经演化出了大相径庭的企业文化。

就拿Meta来说，为什么它的挖人策略这么激进？除了AI人才很重要之外，也和它的企业文化有很大关系。

首先，Meta内部有强烈的“恐惧文化”，员工采取末位淘汰制，每个季度还要签署绩效承诺书，假如不达标，可能就会成为下回裁员的备选。这也就导致Meta内部对失败的容忍度很低，而且各个部门之间都会拼命争抢资源。

这种企业文化一度为Meta带来强大的竞争力，但是也导致一些项目失利。

比如，Meta的上一代大模型Llama4，据说因为Meta内部的部门之间因为争抢资源，导致协作不顺畅，最终让Llama4在很多指标上比GPT-4.5要落后10%以上，在很多能力上也不如DeepSeek。据说开发团队的员工都以此为耻，以至于他们离职以后在简历上都隐去了这段工作经历。

同时，前面说过，Meta不太能容忍失手。而且关键在于，Llama4还不是第一次失手。要知道，Meta原来可不叫Meta，而是Facebook，后来是扎克伯格决定进军元宇宙，才把公司改名叫Meta。目前看，元宇宙显然还没有玩明白，假如AI上再失手，扎克伯格大概率不能接受。因此，他们一定会在后面的Llama5大模型上背水一战。那么，Meta的这种企业文化到底有多特别呢？我们可以做个对比。

跟Meta比，谷歌强调的是包容创新，欢迎试错。比如，谷歌有个很著名的管理设计，他们会给员工留出20%的时间，鼓励他们去创新。这也带来很多意外的惊喜。比如谷歌的Gmail就起源于员工的业余创新项目。

而Meta强调的是结果导向，任何功能都是越快上线越好，快速迭代，快速完善，不行就果断放弃。

定期举办编程比赛，也就是行业里说的“黑客松”，鼓励员工快速验证想法。据说有人统计过，Meta单位时间里的代码提交量是谷歌的3倍，但Bug也比谷歌高出40%左右。

再比如，Meta和亚马逊相比，亚马逊强调的是长期主义。贝索斯最著名的一句话就是，很多人关心未来十年什么东西会变，但很少有人关心未来十年不变的是什么。因此亚马逊对一个产品的打磨，可以历时很久。但Meta强调的是“冲刺文化”，即使像元宇宙这样的长期项目，也要求团队必须每季度交付可展示的里程碑，做不到就裁员。

再比如，Meta和微软相比，微软现任CEO纳德拉推行的是成长文化，鼓励内部员工成长。但Meta要求的是立竿见影地变强，因此他们在挖人上非常果断。

你看，当前面这些公司聚集在一起，是不是有点像一个巨大的华夫饼？内部有无数个小格子把彼此区分开。

注意，这可不是说只有Meta的员工压力大，硅谷的压力都不小，只是这些压力的表现形式不一样。同时，我们也并不是要评判各个公司文化的高低。事实上，这些企业文化在很大程度上，都是一个个选择积累出的结果。它们也许并不存在好坏之分，只是适用的环境不同。

首先，我们看看Meta的创始人扎克伯格的个人风格，他本身就是写代码出身，信奉的是“数据即决策”，所有的事情都要拿数据说话。这也在很大程度上决定了Meta的行事风格。

当然，最终决定Meta企业文化的，还是它的赚钱方式。尽管号称要进军元宇宙，但至少目前，Meta最大的营收依然来自广告业务。广告从哪来？有用户关注才能有广告。因此，Meta必须想尽办法保证用户一直增长，一刻都不敢停下来。

那么，怎么保证用户一直增长？很明显，你需要不断给用户提供些新功能、新产品。因此，Meta特别注重技术的商业化，他们的创新也更多集中在应用层。这也间接导致了Meta内部激进的绩效文化，所有的事情都要求马上看到结果。

现在，你再看一眼Meta的挖人策略，从激进地砸钱，再到48小时极限施压，是不是跟这个风格一脉相承？

换句话说，即使是再大的公司，对它的企业文化起决定作用的，也许依然是它赚第一笔钱的方式。你的第一笔钱是怎么赚的，将在很大程度上决定你以后很多年的行为方式。
