

# "天价耳环"二次通报，但已经注定没人相信了

7月16日，雅安市联合工作组对黄杨钿甜“天价耳环”事件，进行了再次通报。

不过从通报效果来看，并不尽如人意。

那么，事情真相到底是如何呢？

相比于相对仓促的第一次通报，第二次通报从内容和细节上增加了很多内容，而且央视还发布了《黄杨钿甜“天价耳环”事件追踪》的文章，对照通报来看，就能还原出不少过去不为人知的真相。

第一，关于黄杨钿甜的背景。

事情的起因，无非就是黄杨爸爸，杨伟曾经的公务员身份，引发了网友关于在担任公职期间贪污了汶川地震救灾款的质疑。

所以在第一份通报中，详细解释了黄杨爸爸的问题。黄杨爸爸不是公务员，而是事业编，当了三年城管，后来借调到科级事业单位雅安市投资促进服务中心（科级），一直到辞职，也不过是个科员。

这种工作履历，显然不可能有什么不得了家族背景。

于是网友又把怀疑的目光投向了黄杨的妈妈，因为黄杨的第一个字是黄，很可能是随母姓，那么母系家族肯定是什么权势家族。

于是，在第二份通报和央视报道中，清楚地揭示黄杨母系家族的情况：

黄杨的母亲根本不姓黄，而姓司，黄杨的外公是司某某，外婆是岳某某，均为河南商水人，长期务农。

可能是预判了不甘心的网友会再把矛头指向黄杨的父亲，文章把她爸爸的父系也一起调查出来了：

黄杨的爷爷黄某某 1979 年从部队义务兵退役，后在一个村办企业当货车司机，企业倒闭后在家务农。黄杨奶奶杨某某为家庭主妇。黄杨的叔叔和姑姑都在成都或广东打工。

你看，这就是个普通家庭嘛！哪像有些人说的“权贵家庭”？

其实稍微懂点逻辑的都能看明白，如果黄杨母系家族真是权贵的话，为啥会看上这个城管队员？（2010 年代城管队员的名声可不咋好）就算看上了，又怎么会让女婿当一个小小的科员？

那么问题来了，黄杨的爸爸明明姓杨，为啥黄杨的第一个字却姓黄呢？

答案很简单，黄杨的爷爷姓黄，奶奶姓杨，爷爷属于上门女婿，要跟着奶奶姓杨。根据“三代还祖”的风俗，到了黄杨钿甜这一辈，改回了黄姓，仅此而已。

说实话，这个调查结果是可信的，因为身份问题太难造假了，也太容易被证伪了，调查组不会那么傻。

第二个就是她爸爸在当公务员期间经商办企业的问题。

澄清了黄杨家族问题之后，那么就要解释黄杨家的钱是从哪来的了。

第一份通报中，澄清了黄杨爸爸侵吞善款的质疑，一个城管队员根本接触不到善款好么？

就算黄杨爸爸后来去了投资局，就算真有侵吞几亿善款这种事情，也轮不上黄杨爸爸这个小小的科员啊！

于是不甘心的网友又扒出，黄杨爸爸在 2015 年（工作期间）涉嫌违规经商办企业！是不是通过这个企业官商勾结挣的钱呢？

这个嘛，的确是违规了，也是官方立案调查的重点。

结果呢？调查发现这个公司就是个实缴 8 万块钱的小公司，是黄杨爸爸和 3 个同事合开的，做一些办公用品和耗材，以及一些展览业务等等。结果，2年时间，才赚了12万，一家分了3万块钱，说实话，这点钱还不够费劲的呢。

虽然这种行为的确违规了，但按《公职人员政务处分法》有关离职人员的规定，没收违纪所得就行了，不会有其他处理。

至于在工作期间违规生育二胎的事情，这也的确是真的，不过呢？毕竟人家都离职了，而且不能生二胎的规定早就废除了，再拿作废规定处理人家也不好，所以也就不再追究了。

第三就是最关键的，耳环到底值不值230万的问题了。

第二份通报发布后，哪怕经国家级检测机构检验为玻璃，照样有很多网友不信是假的。但是从央视报道来看，这个说法是非常符合逻辑的。

黄杨妈妈司玲霞2014年去深圳学习钻石珠宝鉴定，认识了刘芳芳，二人成为朋友。2016年，刘芳芳在水贝珠宝市场逛街时，花300元买下了这套饰品，一共三件，一对耳环和一条项链，后来司玲霞去找刘芳芳玩时，看到了很喜欢，刘芳芳可能觉得也不值钱，就把耳环送给了她。

而且刘芳芳还出示了带拍照日期的项链和耳环摆在一起三件套的照片，确实是2016年拍摄的。

很多网友可能还不服气，谁能保证送去检验的假耳环，就是黄杨戴的那一套呢？毕竟照片是5月11日发的，送去检验是5月29日，这中间有十几天时间，足够买一套假的了。

的确，这一点很难证实，不过调查组也考虑到了这一点，在送检的同时，专门查阅了GRAFF的交易记录，没有发现杨伟及其亲属的购买痕迹。

如果耳环是真的，在没有购买记录的情况下，又是从何而来呢？

至于照片拍出的耳环火彩问题，这个更容易解释了，大家去小商品市场找个仿制钻石首饰去拍一下，角度合适的时候，一样可以拍出火彩的。

除此之外，整个事件中，已经有这么多质疑被证伪了，那么其他质疑，是不是其合理性和真实性也值得怀疑呢？

比如黄杨爸爸“千万注册费”的公司紧急转让更名的问题，因为公司注册在疫情期间，被网友怀疑参与了核酸和疫苗生意。

但事实上呢？这个公司并没有实际运营，认缴资金是0元，原本是为了做化妆品生意的，后来没做成。早在2025年3月（舆论爆发前）就与中介公司签了转让协议，也谈不上紧急转让公司的事情了。

此外，第二次通报发出后，网友又对前期耳环来历说法不一提出质疑，说“经纪公司说跟品牌方借的，本人说是问妈妈借的，爸爸说是买的仿品，官方通报是玻璃，到底听谁的”？

这个说法看起来的确很可疑，但翻翻经纪公司的声明，并没有“跟品牌方借的”的说法，而黄杨爸爸的声明，也只是说了“非正品”，而不是“买的仿品”。

如果把这两条不实信息去掉之后，黄杨“问妈妈借的”的说法其实是和官方通报相匹配的。

然后又有网友提出，这款耳环是2018年才发布，咋可能在2016年买到？

这个说法证伪其实很简单，我们去GRAFF官网查一下就知道了，2018年并没有发布这款耳环，相反2016年出现过不少这种耳环。

好了，这就是通报的大致内容，那么，如何看待呢？

# 2

说实话，通过第二次的通报和央视的报道，这件事的脉络已经清楚了，一个个疑点也都解开了。

但问题在于，没人会信。

> ——从表面来看，是因为通报虽然很详尽，但没有解释两个关键问题：

第一，耳环的购买记录。

虽然刘芳芳说是在深圳水贝珠宝市场逛街时买的，虽然刘芳芳也出示了照片和拍摄时间，但问题在于，刘芳芳说的就是真相吗？口说无凭啊！这么关键的问题，怎么能用“工作组通过实地走访核实，网传黄杨某甜佩戴的天价耳环系司某霞朋友 2016 年所赠，为仿制某大牌饰品。”一笔带过呢？

这个说法，实在是太含糊其辞了。

你走访了水贝珠宝市场相关摊位了没有？查了商场的购买记录了没有？最不济，你想办法找到摊主做个证也行啊！

其实还有个最简单的办法，把刘芳芳手里拿个项链要过来，和黄杨家送去的耳环放在一起做个检测，如果项链有关铜合金的金属成分含量比例，和耳环的相关成分完全一样的话，那才能证明耳环和项链真的是一套，真的是 2016 年所购买，而不是 2025 年购买。

在官方拿不出购买记录也拿不出对比检验记录的情况下，也难怪网友会怀疑。

第二，黄杨家的钱是从哪来的？

黄杨出道时，经纪公司包装的人设是“富家千金”，从她微博的照片来看，她家有一个巨大的首饰柜，这些首饰，总不会全是假的吧？

显然，黄杨的家境是非常殷实的。

那么，一个科员，是怎么在 2017 年辞职后短短 8 年时间，就突然从城管变成经商天才，挣出来如此多的钱的呢？或者说，这些钱是雅安工作期间积累的呢？

根据“巨额财产来历不明罪”，“国家工作人员的财产、支出明显超过合法收入，差额巨大的，可以责令该国家工作人员说明来源，不能说明来源的，差额部分以非法所得论。”

既然网友有质疑，如果黄杨家的钱真的来得干净的话，通报为什么不写这些钱从何而来呢？

这个问题不解释，那网友的质疑就不会消除。

联系到这些年层出不穷的周公子、北极鲶鱼、4+4 等等事件，如果我们观察一下本质，就会发现，不是人性变了，而是群体的心态变了。

# 3

说实话，中国改革开放的本质是什么？

是打破了效率低下的大锅饭之后，允许生产力和生产要素集中到少部分人手里，让这些人先富起来，然后“先富带动后富”，渐渐实现共同富裕。

“先富带动后富”这种口径，在国家宣传力量的宣传之下，成了社会认知基石，并成为群体的信仰。

咱们实事求是地说，“先富带动后富”是有道理的，人的确是有多样性的，有些事情，也的确是个别“有本事的人”能干好。

但问题在于，在中国改革开放的草莽时代，你很难说某些“有本事的人”做的事情是完全合法的。

婚外生子和超计划生育的有之，没有造车资质仍然造车的有之，背靠保护伞打压竞争对手的有之，官商勾结侵吞国有资产的也有之。

对于这些事情，其实很多人也都清楚，也是公开的秘密，不过呢？大家顶多骂两句，不会有其他动作。

为啥？因为在中国经济高速发展的时代，大部分人有钱赚，有奔头，有前景，没人愿意在这上面浪费精力。

所以，关于“先富”群体的一些黑点，处于一种“民不告官不究”的微妙平衡状态之下。

但是，随着国际形势和经济形势变化，群体的心态也变化了。

钱不好赚了，学历贬值了，上升渠道变窄了，劳动价值缩水了，个人再奋斗也触碰不到高层的地板了，种种因素叠加，形成了改革开放以来最严重的一次群体心理危机，然后一些二代、三代的炫富、特权行为，又进一步刺激公众，形成了一种群体的愤怒和拷问：

不是说先富带动后富吗？你们这些人靠着黑历史发家，怎么转过头翻脸不认人了？我们当年把生产力和生产要素让渡出来，现在我们的利益谁来保障？明明都赚了钱了还出来炫富，你们这个群体不得好死！

当社会给不了答案，那么群体的矛头就会指向一切有钱、有权、有地位的群体，一旦发现有一个可疑的点，马上就会成为群体性情绪发泄的突破口，近些年发生的一系列舆论热点，其实大都因为如此。

只要形成席卷全网的热搜，就能倒逼官方下场调查，也总能找出一些黑点出来，做出相应的惩罚和处理，这样就形成了一段“网友剥丝抽茧发现惊天丑闻，不良男女/家族/集团终得报应”的佳话，看着曾经高高在上的人倒塌凡尘，社会情绪狠狠出了一口恶气，然后转头看向下一个目标，舆情也就平息了。

但这样的舆情平息有一个前提，那就是调查的结果证实了网友猜想。

如果调查报告证实被质疑的人是清白的，许多人不可能不破防。

毕竟在历次舆情事件中，因为人情社会、历史原因、维稳考虑等等因素，有许多黑历史，甚至闹出指鼠为鸭的笑话，直接把公信力给打崩了，然后这种小地方的公信力崩溃，又被延伸到了整个国家的公信力，让整个体制陷入了逃不脱的塔西佗陷阱。

这个时候，无论再怎么调查，也是没人信的。这次也是一样，有不少疑点未能解答，甚至没有触及，那就不能怪大家不信了。
