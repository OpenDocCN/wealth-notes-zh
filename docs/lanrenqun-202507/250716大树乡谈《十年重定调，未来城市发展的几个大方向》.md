

# 十年重定调，未来城市发展的几个大方向

2015年，时隔37年，召开了第五次城市工作会议，这也是改革开放之后的第一次。昨天下午新华社刊载重要文章《走出一条中国特色城市发展道路》，显然，改革开放之后的第二次城市工作会议正在进行，很快大家就能看到通稿，最快就是明天。考虑到城市在经济社会发展中的关键性作用，十年一度的城市工作会议，每一句话都值得认真分析，这都是非常重要的未来方向判断。在通稿发布前，小镇先简单谈谈自己的想法。这篇文章本来应该昨天发，结果被某公司的侵权投诉耽搁了。

最近，市场和各方情绪有所激动，小作文又开始泛滥，鼓吹即将迎来新一轮货币化棚改，还将加大财政投入收储存量房，总之就是要救房地产。这显然是炒作而已。中国城市发展的环境已经发生了根本变化，已经不可能回到10年前了，更何况十年前的城市工作会议，目的就是要对过热的房地产炒作降温。2016年的货币化棚改更应该视为种种原因导致的错误，也很快被纠偏；而2023年已经判断“我国房地产市场供需关系发生重大变化”，这包括总人口下降、城镇化进程放缓、人均住房面积接近发达国家水平、户均也已接近1.1套。从2022年房地产开始下行，到现在仍然处于“止跌”阶段。因此，又怎么可能回到过去通过大拆大建刺激经济增长的旧模式？更何况土地财政大降，地方又哪来的钱去搞新一轮货币化棚改和加大收储力度？难不成指望国家财政大放水？这不过是一厢情愿的幻想罢了。

一定要明白，中国城市高速扩张的时代已经结束了，未来要进入的是一个崭新的阶段，可以理解为“高质量发展”“精细化改造”的阶段。未来的主基调是什么呢？在城市发展基调上，核心有两条，来自“政府和市场”的关系，这也是今年初提出的五个“必须统筹”中的第一点：必须统筹好有效市场和有为政府。

先说市场角度。代表人是《大国大城》的作者陆铭。就在上周，陆铭在接受采访时提出《上海可以再“大”一点，再“活”一点》，认为上海还不够大，提出需要从都市圈而不是行政管辖区的角度去理解。比如陆铭提出，相比东京都市圈的3700万人，以上海和苏浙中小城市形成的潜在都市圈目前人口只有3100万人，显示上海城市开发强度不够，原因在于过去根据行政辖区概念设置的种种城市开发“红线”制约，比如根据行政区域面积限制城市建设用地总量等。陆铭的观点就是尊重市场化和人口的自然流动，应该适度放开限制，倾向于中心城市、核心大城市应该更大，直到不同区域间人均劳动效率基本一致，这也是陆铭的一贯观点。

然后是政府角度。倾向于加强管控，通过把握城市发展规律对城市发展进行人为干预，或者说规划。这个规律基础是考虑城市的承载能力，还要注重区域协调发展，避免一城独大，承载能力包括水土、能源等资源禀赋，以及环境、基础设施、公共服务以及产业布局等。而这也是自2015年以来决策层一直遵循的思路，典型有二：一是北京等超大城市陆续确定人口总量上限、生态控制线、城市开发边界“三条红线”，这也是陆铭提出希望适度放开的“红线”限制；二是雄安，代表了决策层对未来城市的美好想象，雄安就是一次大规模的实践探索，理想未必能完全实现，但实现理想过程中的探索一定会应用在其他城市。

从新华社文章就能看出来，第一章节主题就是“认识、尊重、顺应城市发展规律，端正城市发展指导思想”，开头就以雄安作为例子，强调对城市发展的系统规划。所谓“规划”，自然是强调政府作用。也正是在2015年的会议上，提出了城市发展的“一中心一尊重五统筹”，即“以人民为中心”，“尊重城市发展规律”，“统筹空间规模产业、规划建设管理、改革科技文化、生产生活生态、政府社会市民”。

新华社文章，主调并没有变。一定要注意，这种配合会议召开的文章，都是由专门的课题组撰写的。那么哪一个会成为未来的主基调呢？当然是后者，也就是继续强调政府统一管控或者说规划。对政策，小镇已经说了很多次了，要关注不同的观点和视角，避免陷入认知茧房，但把握政策，一定要搞清楚决策层怎么想，搞清楚客观上“政策是什么”，减少主观想象。

基于这一主基调，未来的城市发展主要方向，小镇觉得有“四明两暗”。“四明”是已经非常确定的，也会加大投入的重点领域；“两暗”是非常重要但投入不足，有些核心问题还没想清楚，还需要进一步探索的重点领域。

- **方向一：** 限制单体城市规模，鼓励发展中小城市，打造都市圈、城市群。未来超大城市、大城市仍然要限制单体城市的规模，设置的红线不会打破，顶多在个别领域有所放松，比如在上海等个别城市，以及部分重点项目、跨区域项目等，可以进一步放开城市间、区域间指标流转。
- **方向二：** 强化区域一体化建设，以改革解决卡点堵点。为了实现都市圈、城市群，就要强化区域一体化，就必须解决目前阻碍区域一体化的卡点堵点。比如公共服务不均衡、区域内产业同质化严重、城市间协调机制不健全等问题，这些都阻碍了区域协调发展尤其是同质化竞争，这也是导致地方保护、“内卷式”恶性竞争的关键原因。在中国目前基于产出法的GDP核算标准下，一个地方更看重的是产业带来的产出和就业，而这个产业是否具备竞争优势、利润多少是相对次要的，于是招商引资成为重点，什么产业火就一哄而上，争抢有限的资源，还动用地方保护延缓劣质企业退出，于是搞得优质企业也不得不持续失血，这才是国家整治恶性竞争的重点。

城市之间的关系主要是以上两个方向，然后就是城市自身的发展。过去“大拆大建”“摊大饼”的模式肯定不会再走了，未来的重点是精细化、高质量，也有两个需要关注的主要方向。

- **方向三：** 注重高质量发展、精细化改造的城市更新。2023年为了稳定经济，曾经提出了包括建设保障性住房、“城中村”改造、“平急两用”公共基础设施建设在内的“三大工程”，年底经济会议再次提出加速，但注意，这“三大工程”很快就降温了，2024年基本淡出政策讨论，虽然政策惯性仍然存在，比如去年9月提出“新增实施100万套城中村改造和危旧房改造”，2025年3月又提出在新增100万套的基础上继续扩大规模，但这绝不是货币化棚改，更不是拆迁，而是城市更新的概念。还要注意，并没有说什么时候完成，很难评估到底有多少是在常规工作之外新增的。至于保障房建设或者小作文炒作的政府收储存量房用作保障，只能说想法很好，实际执行难度很大。经过这几年的房地产调整，租金已经明显下降，比如北京房租普遍下降15%，闲置商品房增多，位置偏远一些的保障房吸引力更低了。而在商品房已经过剩、政府财政紧张的情况下，政府又怎么可能继续加大保障房建设？

收储就更难了，有三个基本问题：一是向谁收储？政府能安排人员一套套地谈吗？显然只能对接开发商成规模的收储。二是以什么价格收储？低了房地产商不干，高了是不是涉嫌腐败和利益输送，关键钱从哪里来？三是收储后的运营收益，如何出租？租金怎么算？如果亏本，那是不是国有资产流失？所以，除了正好适合的项目，否则地方对于收储存量房兴趣很低，还不如要求房地产商配套建设保障房。

过去的建设扩张思路就不要想了，习惯了项目建设逻辑的发改系统，一直在想办法找新城市开发项目，但体量远不如过去。比如4月新闻发布会，发改在谈到城市更新时，强调城市发展已经进入增量建设和存量更新并重的阶段，在增量方面，举例提出城市地下管网建设改造，测算未来五年资金总需求约4万亿。管网建设当然是未来城市更新的重点，也一定会加大支持力度，但往大里说，也不过就是“五年4万亿”，这还包括了每年本就不有的数千亿投资，增量其实远低于“4万亿”。当然这里面的机会还是很大的，小镇这里谈的是宏观大方向，实际上任何一个国家提出的方向，都足够庞大了。

不同于“大拆大建”，城市更新更倾向于改造，重点就是解决“城市病”，提高居民生活体验，也是要在城市发展形式发生巨变的情况下，继续拉动投资和就业。所以只要城市生活中的痛点，都会得到一定的政策支持，比如提高城市安全性的生命线工程、供排水、城市体检等“里子工程”，还有提高人文居住体验的人居环境、文化建设，以及如社区养老等在内的民生保障设施建设。

- **方向四：** 数智化、绿色化建设与改造。这个方向实际也属于城市更新范畴，只是特别重要，也不同于过往传统，因此单列。如果关注这几年的高层调研，就会发现城市数智化建设已经成为调研的重点，这一块也非常符合发改大规模项目建设拉动经济的传统逻辑，比如目前推动的数据要素和数字经济，实际上还是落到了项目建设的方向上，强调的是“数字基础设施建设”，城市数智化，就是重点之一。

这不仅仅是过去的智慧城市、城市大脑等单点建设，而是要在改造基础上将整个城市连为一体，比如去年开始推进的自动驾驶“车路云一体化”，就是典型。日常城市生活所见的一切，都会被“数智化”改造一遍，并伴随绿色化改造。比如路灯灯杆正在进行智慧化升级，同时还伴随绿色节能改造，有的城市还把电动自行车的换电柜整合进了智慧灯杆，在设想中，换电柜里的电池还具有了一定的储能作用。还有公共站牌、红绿灯、城市监控设备等等，都需要升级改造，还要连成一片，最终通过人工智能来提高整个城市的运转效率。如高德地图的“云信”，就是通过智能化手段优化路口红绿灯配时，以提高道路通行效率，已经在一些城市进行试点。

在城市数智化的基础上，城市大量闲置资源也有望被盘活，同时解决部分城市病。比如大城市停车，其实闲置停车位很多，想要利用起来就需要实时更新停车位情况，更有效率地指引。设想这样一个场景，当一个人开车去某个停车紧张的区域，靠自己很难找到停车场，可能转好几个停车场都是满的，而实现了城市数智化，还能在高德地图等APP上直接面对所有个体，那么就可以直接引导车辆前往不同的停车位，还可以提前锁定目标车位，而车位上普遍安装低功率的慢充电桩，一两个小时的补能就完全够用了。基于此，还能更高效利用单位、企业的内部停车场，这就跟目前正在探索的以“V2G技术”为核心的车网互动模式很类似。当实现了一个小区的微电网智能化，就可以统一安排小区内车辆的充电、放电，有限的电网容量得到最充分的运用。今年4月已经发布了《关于公布首批车网互动规模化应用试点的通知》，划了9个试点城市。在数智化的基础上，基于可再生能源的绿色化改造，才能真正发挥威力。比如去年建筑科技展特别关注的光伏建筑，发的电要想充分利用，必须建立在智慧电网的基础上，储能仅仅是辅助。

小结一下，未来城市发展，有四个大方向：一是限制单体城市规模，鼓励发展中小城市，打造都市圈、城市群；二是强化区域一体化建设，以改革解决卡点堵点；三是注重高质量发展、精细化改造的城市更新；四是数智化、绿色化建设与改造。

还有两个特别重要，但难度很大、关键问题还没想清楚，目前重视度和实践脱节比较大的方向，也就是“两暗”。

- **一是基层社区治理。** 近几年，最高层赴地方调研，最多的就是社区调研，新华社文章也专题谈了社区治理。这一块也非常影响城市居住体验，比如物业问题还有养老问题等，有不少成功探索，但就像新农村建设一样，能做成的各有各的优势，缺乏从点到面的推广。
- **二是以农民进城为核心的民生保障。** 未来农村人口仍然会持续流入城市，城乡融合方面相对容易改的都已经改了，接下来怎么办？要不要取消户籍？如果取消户籍，如何调整基于户籍的资源分配？从常住人口角度，2024年末城镇化率已经达到67%，但从户籍角度，才只有48%，意味着超过2亿人长期生活在城市、为城市做贡献，但并没有获得城市户口，更没有享受城市户口带来的包括教育、养老、医疗等在内的民生保障福利。然而之所以限制户口，就是因为资源有限，仅仅给一个户口，而不弥补民生保障的缺口，这个户口也没什么意义，这就是“租售同权”改革没什么实际效果的关键。那么，中央财政要不要加大在这方面的投入？不可能如某些声音说的大规模刺激，还是更强调稳步增长，需要关注的变化是，在“投资于人”思路转变的情况下，今年有一些探索，当对“投资于人”带来的增长乘数效应等有比较清楚认识后，会不会加大投入。

其他还有一些分散而具体方向，比如进入存量时代的改善型住房需求，以及新房领域提出的“好房子”概念；针对存量房的改造需求，如大批商品房老化后带来的维修改造，适应老龄化的适老化改造，解决老旧房子问题的降噪、防水改造等等。
