韩国前总统尹锡悦和他老婆金建希的事儿有新情况了。

本月7日，“金建希特检组”以涉嫌违反《资本市场法》等法律为由，向韩国首尔中央地方法院提请批捕金建希，首尔中央地方法院12日深夜签发了对金建希的逮捕令，随后将金建希收押进拘留所。

在逮捕金建希的问题上，韩国舆论出现了两种倾向，右翼媒体称其为前第一夫人受难，隐喻着本案件的政治属性。而左翼媒体则将案件视角扩大，集中火力去攻击韩国检察系统的包庇，毕竟在之前针对金建希的历次调查中，韩国检察机关都做出了不起诉决定。

而在金建希被捕当天，李在明提出了改革检察系统的宏伟愿景，并计划在2030年解散韩国检察院，将检察系统的调查和起诉权分立，彻底终结韩国检察系统无冕之王的存在。

# 韩国检察系统与左派的斗争

要理解李在明与韩国检察系统的恩怨，就必须知道韩国检察系统，这头权力怪兽从何而来。

1808年，法国大革命后，拿破仑改革了刑事诉讼法，要求赋予掌握起诉权的检察官直接调查权。当时就有法学家认为，这项改革会让检察官成为威胁国家的暴君，多年后，法国认识到了这个问题，最终形成了起诉（检察院）、调查（警察）、审判（法院）分离的原则。今天全世界的检察系统，都遵循这个最基本规则，唯独韩国是个例外。

日本在明治维新后，以法国为蓝本，塑造了国内检察系统，并将这套集合了调查、起诉的检察体系带到了日据朝鲜，形成了今天韩国检察院的前身。虽然当时日本法学家已经认识到这套系统不合理之处，但是为了维护殖民统治，日本依旧对检察官系统予以了加强。根据日本制定的《朝鲜刑法》规定，警方应在检察官的指导下调查犯罪，并服从检察官的命令。

在1945年韩国光复后，韩国有识之士就推动检察院改革，名义就是推翻日本殖民司法体系。在这一阶段，美国主导了韩国检察系统改革，警察获取调查权，检察官获得起诉权，两方分开，结果一分开出事了。

韩国警察以为美国给他撑腰来了，直接拒绝在任何事项上配合检察机关，无视检察机关的协助请求。而检察机关也制定了内部政策，拒绝警察的一切报告，并对任何涉嫌犯罪的警察无情地提起拘留和起诉。这就是韩国历史上著名的警检冲突。

有趣的是，此时社会舆论站在了检察院一边，因为掌握了调查权的警察，在李承晚和美国的指导下，犯下了一系列罄竹难书的罪行。检察系统对警察系统的政治报复，被民众视为捍卫民主的正义举动，最终让调查权再次回到韩国检察机关手中。

而在韩国军政府的推动下，检察系统权力再次加强，成了军政府手中的一把利刃，可以说，韩国检察体系，某种程度上不是一个法律机关，而是一个政治机关，更类似于苏/俄的内务部或者契卡这种负责清洗的单位。

等到韩国走向民主化之后，再也没有势力能够控制这台以清洗为生的政治机器了。而一轮一轮的清洗，缔造了韩国总统前仆后继入狱的悲剧。无论左派右派，都逃不出这个轮回，甚至连曾经担任检察总长的尹大总统和他老婆都难逃厄运。

文在寅在卢武铉自杀一事 后，认识到了检察系统的威力，并将其视为韩国独裁体制遗留的顽石。李在明则坚持了文在寅的政策，借助尹大总统政变的东风，试图推动对检察系统的改革。如此看来，尹大总统真是左派的恩人，如果没有这轮政变，韩国检察系统的改革，可能一百年都没法开始。

# 德意志股票操纵案

在对金建希的调查当中，特检组第一个援引的法律条款，就是《资本市场法》。这很明显是针对德意志汽车股票操纵案去的。

德意志汽车是一家韩国汽车经销商，负责宝马汽车在韩销售。2009年，当时金建希就与股票操纵“高手”李钟镐一起，通过操纵股价的方式攫取收益，整个过程持续三年。他们总计攫取了大概24亿韩元的利润。在这期间，金建希与尹锡悦相识并结婚。金建希的母亲，也就是尹锡悦的岳母也被卷入其中。

此案在2020年暴雷，参与操纵股票的几人，包括德意志汽车董事长权五洙都被逮捕。然而直到2024年，金建希才被韩国检察院调查，在调查中，金建希一口咬定自己只是普通投资者，完全按照投资顾问的要求下单，至于为什么能赚这么多钱，都是运气好。

在经历了数个月的调查后，韩国检察机关认可了金建希的说法，认为第一夫人是无辜的，她只是一个普通的投资者，她又有什么错。于是去年10月，首尔中央地方检察厅宣布金建希无罪，不予起诉。

至于金建希案件的关键证据，就是第一夫人与股票操纵者长达三年的录音，其中包括要求金建希上交40%的利润作为操盘费，自然也被韩国检察机关以各种理由不予采纳。

所以，从这里就可以看出韩国检察机关的权力之大，韩国检察机关可以决定起诉还是不起诉。大家千万别小瞧了这个权力，假设你有罪，韩国检察院可以给你无罪释放。假设你无罪，韩国检察院可以对你执行长达一年乃至数年的调查-起诉套餐，等到法院宣布你清白的时候，一切都晚了。

在德意志股票操纵案中，原本韩国警方想要调查搞个大新闻，去向韩国金融监管局申请数据查案，结果韩国金融监管单位表示，这种调查只能检察院来，拒绝向韩国警方提供股票账户信息。

言外之意，警方是什么东西，也配调查这种案件。韩国一位警察就匿名向调查记者透露，只要韩国检察院和金融监管机构愿意，韩国警察无法对任何金融案件进行调查。

现在，金建希特检组重新就此事发难，一旦金建希被定罪，下一步就要调查首尔地方检察厅是否涉及渎职和包庇了。然后，这个调查会直接指向尹大总统，成为其利用检察系统制造法律不公的典型案例，并最终推动李在明的检察系统改革，完成文在寅等人，未能实现的心愿。

# 结尾

8月13日，当金建希带着手铐独自走进拘留所，李在明正在青瓦台，出席总统国政规划委员会的发布会，并发表讲话。在这里他提出了，23项战略举措和123项课题。其中最为关注的依旧是国内权力改革。

其中对于警察，检察系统的改革，将会成为李在明政府未来政策的核心。韩国检察系统，调查与起诉权将会分离。目前的规划中，调查权将会下放给警察，随后会对警察系统进行改革，彻底铲除殖民地司法遗留。

韩国检察体系自日本殖民时期一脉相承，至今已历经124年，这种依附于殖民地统治系统的不正常检察权，依旧在影响着韩国的方方面面。随着现代社会经济发展，社会运作逐渐复杂，法律越来越难以对细节进行解释。在这种黑与白的交融地带，韩国检察官就具备了巨大的权力。

同时，韩国法律有着一系列模糊条款，乃至毒丸法律，这些规则的存在进一步赋予了检察官控制社会的力量。在这个问题上，韩国左派认识是不断加深的。在1998年，金大中曾经误以为检察系统是法律的化身，甚至高呼让检察系统站出来，维护民主社会。直到卢武铉从山崖一跃而下，文在寅才认识到这只权力怪兽的可怕。

对于李在明而言，过去很长一段时间，他都是韩国检察系统重点关注对象，是差一点就要进监狱的主。如果不是尹大总统政变失败，今天李在明能否活着都是一个问号。毕竟在通缉逮捕名单中，李在明可排在前列。

当前，李在明唯一的活路就是在任期内完成对检察系统的改革，这一战，既分高下，也决生死。

虽然从案情考虑，金建希的案件还是有一些值得玩味的地方。但从政治角度来说，只有彻底将金建希案钉死，李在明才能推动其改革进程。

而这就是为什么韩国检察系统，长盛不衰的原因。无论左派右派，都希望有这么一柄摧毁政敌的刀子。

尹锡悦如此，李在明，也是如此。
