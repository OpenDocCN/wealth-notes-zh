

# 最新政治局会议定调下半年，看懂关键表态和背后影响

7月30日，中共中央政治局在北京召开会议，专题研究了下半年的经济工作。这次重要会议传递了不少关键信号，今天我就和你分享一下我的个人理解，带你把握这次会议的核心定调，看清下半年经济工作的发力重点，以及房地产、股市这些跟我们钱袋子有关的重要话题。

## 会议的核心定调

首先最核心的，7月份的政治局会议是个定调的会议，我们就要搞清楚，这次定了什么调。会议定了两个调，第一个是关于马上要制定的“十五五”规划的调。这部分，回头我会在“十五五”规划建议稿出台时专门解读，今天就不详细说了。

第二个定调，是对今年下半年宏观经济政策的定调。这次一个非常重要的转变，就是卸下了之前“应急状态”的政策盔甲。为什么这么说呢？我们拿4月关税战时的中央政治局会议来做对比。

4月份会议的说法是，要“统筹国内经济工作和国际经贸斗争”，要“应对外部环境急剧变化的不确定性”，同时当时的要求是“既定政策早出台早见效”，还格外强调要“加强超常规逆周期调节”等等。当时经贸斗争成为影响国内经济的短期最大变量，所以政策具有急迫性，甚至可以说进入了一种“应急状态”。

但之后情况转变了，我们都经历了，这里我回顾总结一下：

一是中美经贸关系阶段性缓和，虽然还是会有摩擦，但是短期内大动干戈是小概率了。
二是经济并没有想象中的那样受到猛烈冲击，5月份出口增速6.3%，6月份出口增速7.2%，包括上半年的GDP同比增长也达到了5.3%，主要的宏观经济指标没有遭遇预想中的巨大冲击，甚至比较亮眼。

在这种情况下，七月份的政策定调也发生了变化。会议的说法意味深长，说“坚持稳中求进工作总基调……保持政策连续性稳定性，增强灵活性预见性，着力稳就业、稳企业、稳市场、稳预期。”

这段话里，“稳中求进工作总基调”和“保持政策连续性稳定性”一出来，还是反复说的“稳”字，配合“增强（政策）灵活性预见性”的表述，我认为下半年的基调是：接下来总体上不会搞强刺激或者大水漫灌，比如房地产领域，同学们对这方面不要期许太大。但是面对各种具体困难，在具体政策上会变得更灵活，会有很多的细微政策，精准刺激。另外，增强预见性，就是如果看到形势有大变化，就别犹豫及时出政策。这其实反映出决策层对经济韧性的信心，以及避免强刺激政策副作用的审慎态度。

而这些政策的核心指向，还是要“稳住”，稳就业、稳企业、稳市场、稳预期。换句话说，**政策关心的是必须托住经济基本盘，而不会因为强刺激制造短期泡沫，下半年的政策很可能是“大雨没有，小雨不断”**，这个判断，我觉得已经很明确了。

## 下半年经济工作到底要怎么搞？

理解了这个大前提，接下来，我们进入具体的政策，带你读懂会议通稿中对于下半年经济的一系列安排：

在财政货币政策上，会议的说法是：“要落实落细更加积极的财政政策和适度宽松的货币政策，充分释放政策效应”。这个意思是，财政和货币政策在上半年其实已经集中出台很多了，下半年的任务就是细化和落实这些政策，而且这些政策也存在效果延迟，还需要时间来观察。

在货币政策上，继续强调要“保持流动性充裕，促进社会综合融资成本下行”，不过没有提到“降准降息”等表态。有人可能觉得力度不够，但我认为这并不意味着下半年就不会降准降息了，关键还是看外部环境。下半年如果美联储进一步降息，国内货币政策空间打开，在经济需要的时候，仍然可能会进行降准降息，因为中央说了，要“增强（政策）灵活性预见性”。

另外，如果从另一个角度来解读，其实“降准降息”只限于银行端，并不一定能传导到实体。我在第44讲课程里说过，现在的问题是大量资金沉淀在银行中，没有流通到市场上。所以现在说“促进社会综合融资成本下行”反而是一种信号，就是中央认为光给银行降息降准可能没太大效果，要想办法把居民和企业的实际融资成本降下来。而政策也确实这么在干，比如7月31日，国务院常务会就宣布，要对个人消费贷和服务业经营主体进行贷款贴息。

针对今年以来备受关注的反内卷议题，会议说要“依法依规治理企业无序竞争”和“推进重点行业产能治理”。而在7月1日的中央财经委会议上的说法则是“依法依规治理企业低价无序竞争……推动落后产能有序退出”。对比一下，这次表态删除了“低价”的表述，“落后产能有序退出”则被“产能治理”的表述替代。

很多人说是“调门变轻了”或者反内卷力度可能不够，但我不这么认为。

我觉得这反而是一种很大的转变，就是高层不再把低价竞争等同于“无序竞争”，技术进步和规模效应带来的合理降价是被允许的，而不是搞一刀切，所以未来更倾向通过技术标准、环保、能效等市场化手段，来实现“良币驱逐劣币”。

而要达成这个目的，则必须要规范地方政府的行为，所以会议明确提出要“纵深推进全国统一大市场建设”，还要“规范地方招商引资行为”。中央的思路是，反内卷首先就要从城市间、地方政府间的恶性内卷开始，打击狭隘的地方主义和山头主义，构建起一个统一的全国大市场，这样才能从根本上一层层解开市场、行业、企业的内卷。这个思路肯定是对的，当然，这么做难度也更大，我会带你一起关注下半年有什么更进一步的举措。

此外，会议还有一个核心表述，就是我们反复提到的“新质生产力”，会议说要“坚持以科技创新引领新质生产力发展，加快培育具有国际竞争力的新兴支柱产业，推动科技创新和产业创新深度融合发展”。

“具有国际竞争力的新兴支柱产业”这个提法，映照了我前面提到的，中国未来要在大国博弈中，保持战略积极态势的判断，这是大指挥棒，更是产业红利。而“推动科技创新和产业创新深度融合发展”，还记得我在课程中强调的，大学正飞速嵌入产业链吗（第128讲）？包括专业改革（第84讲），深圳科技政策试点（第118讲）等等，这些都可以串在一起思考。这就是大趋势，以后科技和产业的结合，是最大的时代红利。

## 关于房地产和股市的两个信号

最后，和我们个人财富息息相关的房地产和股市领域，也有两个关键信号，我依次讲讲。

先说房地产，这次政治局会议罕见地没有提房地产，而只说了“高质量开展城市更新”。这让市场感到非常意外，结合其它因素，甚至引发了一些股市的波动。房地产最近几个月的低迷很明显，在这种情况下，政治局会议没有提房地产，确实值得思考。

这里我提醒你注意，这次会议通稿在开篇部分就强调要“推动人的全面发展”。我在介绍《住房租赁条例》的时候（第151讲），讲了国家推动“租购并举”和“租购同权”的政策意图。其实，《住房租赁条例》在征求意见稿征求了五年意见后，突然加速出台，并强调推动加快建立“租购并举”的住房制度，已经表明了政策的态度。毕竟促进租房市场繁荣的另一面，必然是购房市场的一部分需求被替代。

实际上，住房压力大一直是年轻人不结婚、不生孩子的重要因素之一。最近国家宣布了全国性无门槛的生育补贴政策，显然也是政策重心变化的一个注脚。一般来说，这种中央和地方的共同事权，中央惯例是只出少的一部分。但7月30日财政部官员说，今年初步安排育儿补贴补助资金预算900亿元左右，中央总体承担约90%。而前不久（7月25日）的国务院常务会议上，还提出要“逐步推行免费学前教育”。显然，生育的政策权重，在一步步增加，而对应的，就是要解决阻碍生育的各种成本问题。

了解了这些背景信息之后，再回到房地产上，你会发现，最近两年的政策，包括房地产新模式，以及房地产“市场+保障”的双轨制，体现出了房地产的民生意味正在一步步增强。总之，在房地产市场过去五年实际情况重大变化，居民杠杆居高不下，以及住房等压力让年轻人结婚和生育减少的大背景下，我们的政策取向必然会既面对现实，也面向未来。这个判断，也提醒你注意。所以，对于下半年的房地产市场，我的看法是，或许“慢跌”也是可以被接受的。

这里我多说几句，关于民生政策的取向，还不止这些。比如在消费方面，从“两新”政策到国补，从加码服务消费到最近的消费贷补贴，很明显国家开始采取一种更直接、更精准的政策思路。国家的真正目的，就是这次政治局会议所说的，要“在保障改善民生中扩大消费需求”，形成“补贴居民—提振消费—带动就业”的新循环。这背后的本质，就是我在第88讲课程说的，中国地方的发展模式，正在从土地财政向人口财政进行深度转型。

再比如就业方面，会议也提出“突出就业优先政策导向，促进高校毕业生、退役军人、农民工等重点群体就业”。“就业优先”被突出强调，而且促进大学生就业排在了首位，说明了年轻人就业是重中之重，背后也显示出这个群体就业困难确实有点大。

再来看股市。这次的说法是：“增强国内资本市场的吸引力和包容性，巩固资本市场回稳向好势头。”这是一个非常积极的表态了，“增强吸引力”的说法是首次在政治局会议提及，这不仅是提升对国内资本的吸引力，也要提升对国际资本的吸引力，这意味着股市要努力保持基本繁荣；至于包容性，我认为，包括 IPO 和重组上市的放宽，也应该是一个可能。

总的来说，虽然下半年政策的总体基调是“稳中求进”，不会大雨强刺激，但是在具体政策上会有很多细微的精准刺激，我认为，股市就是精准刺激的代表。还是我刚才说的，未来很可能是“大雨没有，小雨不断”，这个判断很重要，因此我这里再重复一遍。当然，关于股市这部分，我只是在说清楚政策意图，不代表具体的市场表现，你还要结合自己的情况理性判断。
