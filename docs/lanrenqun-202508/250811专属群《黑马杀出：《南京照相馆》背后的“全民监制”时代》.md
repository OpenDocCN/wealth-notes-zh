

# 黑马杀出：《南京照相馆》背后的“全民监制”时代

![](bbox=[53, 176, 826, 298])

这个暑期，电影《南京照相馆》几乎已经确定是最大的黑马。电影7月下旬上映，6月27日发布第一支预告片，但上映两周后，票房已经超过18亿元。

假如不出意外，在你看到这期节目时应该已经超过20亿元。而且从目前的趋势看，《南京照相馆》很可能成为暑期档票房最高的电影，也是2025年继《哪吒2》之后最叫座的电影。

假如你没看过这部电影也没关系，我们先在不作额外剧透的情况下，仅就制片方已经公布的梗概，来简单介绍一下它的剧情。

《南京照相馆》的故事背景，是血泪纵横的南京大屠杀。男主角阿昌是名邮差，为了活命进入吉祥照相馆冒充学徒。在冲洗照片的过程中，他发现了很多日军屠城的罪证底片。与阿昌经历相似的还有另外几个人，他们最初都只是求自保的普通人，但后来都渐渐觉醒，最终他们决定去完成一项壮举，把这些底片送出去，揭露侵略者的谎言。

这段剧情脱胎于历史上的真人真事，吉祥照相馆的原型是华东照相馆，位于南京市长江路估衣廊一带。主人公阿昌的原型，是当年在华东照相馆当学徒的罗瑾。而且影片中还有很多回应现实的设计，比如阿昌的邮差编号是1213，而12月13日这天在历史上是南京沦陷的日子，为了纪念这段历史，这天也被定为国家公祭日。

好，以上是这部电影本身的基本信息。

回到电影的行业视角，这里面有几个特别的地方，非常值得留意。

接下来，我们就跳出故事本身，从电影行业的层面来说说它的特别之处。咱们主要说两个角度，创作和宣发。这也是一部电影最核心的两个方面。

## 创作层面

首先，从创作层面上看，一般来说，以民族苦难为背景的电影，有两种拍法。这两种拍法未必有高下之分，但是侧重点不同。

第一种是战争片的拍法。注意，从故事层面上看，战争片的核心特征，不在于战争场面，而是它的整个故事架构，一开始就是基于“大是非、大判断、大情感”。大是非，指的是这些故事中往往是非鲜明。大判断，指的是每个角色都有清晰的好坏对错之分。而最后这个大情感，指的是故事里的角色往往身负大义，在这些大义面前个人的感受、个人的生死，并不是故事展现的重点。

这类创作手法不局限于历史题材，比如前几年的《战狼》也属于这一类。从头到尾都贯穿了“大是非、大判断、大情感”。

第二种是故事片的拍法。这种方式肯定也要以正确的是非观为前提。**但在这个基础上，更注重以普通人为切入点**。

这些普通人也会考虑个人得失，也怕死，也有恐惧，也有各自在意的个人利益。但最终，他们都会在一连串的经历中完成蜕变，进而超越个人得失，甚至超越对死亡的恐惧。

而在这类电影里，国内的上一部现象级作品还是张艺谋的《金陵十三钗》。而这类故事目前在电影网站上评分最高的，是1993年斯皮尔伯格的《辛德勒的名单》，在奥斯卡上一口气获得了7项大奖。即使是斯皮尔伯格本人，后来也再没超越过《辛德勒的名单》的口碑成就。

《辛德勒的名单》的背景是二战中纳粹发动的犹太人大屠杀。而斯皮尔伯格本人就是犹太人，当年电影上映时，斯皮尔伯格还受到了不少批评。有人说他对这场惨剧的刻画不够明确，没有正面描写抗击纳粹的过程，而且男主角奥斯卡·辛德勒这个角色也有瑕疵，比如他年轻时贪财之类的。

注意，我们的重点不是要说《辛德勒的名单》，而是想说，用这样的方式讲故事难度非常高。

这就对导演提出了很高的要求，你需要在一个足够锋利的题材上，对人物的情感起伏有恰到好处的把握。既要让观众感受到这是一个好故事，同时能唤醒人们对背后价值立意的思考。

《南京照相馆》的导演申奥，之前拍过《受益人》。这部电影讲的是一个假结婚，然后试图杀妻骗保的故事。他还拍过《孤注一掷》，讲的是关于缅北诈骗的事。而申奥目前正在筹备的新片叫《用武之地》，讲的是中国公民在境外遭到绑架，之后踏上逃亡路的故事。

你看，这些题材都非常尖锐，都要求导演对故事的分寸有准确的把握。从这个角度看，之前的经验铺垫，或许也是申奥拍摄《南京照相馆》的关键前提。

好，这是创作层面上的观察。

## 宣发层面

其次，咱们再说说宣发层面。

不管你看没看过《南京照相馆》，都不难发现一件事，《南京照相馆》的宣发远远不像其他电影那么张扬。

但这并没有影响它后续的票房表现。

为什么会这样呢？你可以说酒香不怕巷子深，是金子总要发光。但是，在这背后，还有另外一个变化值得留意。这就是，今天的电影和观众之间的关系，正在发生转变。

在社交媒体的参与下，电影行业正在进入一个“全民监制”的时代。

什么意思？我们可以把电影和观众之间的关系，分成三个阶段。

第一阶段，是“全民观众”时代。主要是指互联网崛起之前，当时，各地都在建立电影院，每个城市都多了很多块银幕。从此看电影的人越来越多。但是，大家都只是观众，看完之后，满意也好，吐槽也罢，都只能跟周围的人说说。而能在媒体上发表评论的，只有少数的影评人。

但紧接着，互联网大规模普及，情况发生了变化。电影与观众的关系也进入了一个新阶段。

第二阶段，是“全民评委”时代。随着互联网普及，影评不再是少数人的专利了，每个普通观众都有了发表评论的机会。

当时观众发表观点的主要渠道包括，各式各样的论坛、像豆瓣这样的评论网站，或者电影片段在B站上的弹幕。这让很多之前从未进入导演视野的话语，第一次被导演看见。也由此开启了电影史从未有过的一个奇观，导演与观众的对骂。

观众觉得自己花了钱，就有权给评论，就像商品允许给差评一样。而导演觉得你们哪懂电影，我付出的努力你们根本没看到。而在所有的互怼中，几乎百分百，败下阵的一定是导演。只要双方开炮，往往紧接着就是电影评分下滑。导演中更有甚者因此破防，扬言要起诉打差评的观众。结果刚暴露动机，评分马上雪上加霜，没有最低，只有更低。

这期间不排除有些电影是被导演拖累了。因此你会看到很多影视剧，上映的时候分数很低，但时间长了，分数又慢慢反弹回来。

这个阶段有个核心特点，导演和观众的遭遇战往往发生在电影上映之后。而且一般只打一轮，电影的档期结束，互怼也就结束了。

但是，当移动互联网普及，尤其是短视频风靡之后，情况又变了。电影与观众的关系，从第二阶段的“全民评委”，进入了第三阶段。

第三阶段，是“全民监制”时代。也就是，在短视频平台上，几乎所有预算充足的电影，都会开设自己的抖音账号，或者其他账号。并且在筹备阶段就开始频繁发视频，包括怎么选角，幕后如何拍摄，导演和演员之间怎么交流。等到电影的路演阶段，现场请网红，主创人员跳“科目三”“加绒摇”更是几乎成了标配。前两年电影《三大队》的路演中，演员张译因为拒绝跳“科目三”还上了热搜。总之，一个电影开始从筹备到上映，全在观众的注视下完成。

在互联网时代，观众看到的只是电影本身，他们只能对结果发表评论，充其量是“评委”。而现在，观众能够见证电影的制作全过程，几乎每个观众都成了监制。

电影到底哪里好，哪里不好，他们都心里有数。当然，观众可能未必懂那么多专业层面的知识，但是你做得用不用心，态度诚不诚恳，片子本身好不好看，这逃不过他们的眼睛。即使电影本身没有建立账号，只要有初期热度，相关的内容也会在短视频平台上快速出现。

在这个“全民监制”的状态下，电影主创团队和观众的沟通变得空前密切，他们可能会随时因为一个细节而同频共振。同时导演也更意识到与观众沟通的重要性，比如导演申奥在面对观众的意见时，第一反应是，假如你有这个感受，说明我们没有把故事讲清楚。而在这条视频下面的第一条点赞最多的评论就是，这才是导演该有的态度。

因为观众的关注，已经渗透到了电影的每个环节，这就导致，他们对片子的观察更彻底，他们更能形成自己的判断。假如一个电影不好，很难糊弄过去。但确实好的电影，也不用担心被低估。

与其说是全民监制，这更像是全民共振。导演和观众之间，随时可能会因为某个细节的心有灵犀，而同频共振。

姜文曾经说，多用点脑子，少看点手机。但是从“全民监制”的角度看，未来，这两个事或许都值得重视。
