- **1.** 灏泽我自始至终认为，人这一辈子想要发财，前提就是你必须经历一次“实实在在的大行情”。因为只有经历过一次大行情以后，你对资本的认知，才能真正的落到实处。然后这段经历，才会转化成内功，为你打下财富的江山。以后你就会知道了，一个人只要内功足够深厚，那么就算千金散尽还能复来，就算一无所有还能东山再起。

- **2.** 按照目前的趋势来看，大行情真的离我们很近很近了，甚至于可以说现在就处于筑底布局的阶段。其实很多人一辈子都没明白，真正的大牛是有其很清晰的布局过程的。先是大量的资金，涌入龙头大哥和嫡系巨头，然后来来回回调整几次，把各种意志不坚定的小虾米抖掉，接着再是券商开始猛的发力，之后就是一轮又一轮的推动。直到某一天，人声鼎沸时……

- **3.** 小趋势看经济，大趋势看局势，而目前的局势就是需要有一项能让大家情绪都开心起来，能实现全民参与全民欢乐的事儿。而最能实现这件事儿的，那就莫过于我们的大A了。更重要的是，目前美元就是开始了新一轮的大贬值&降息，财富的流通性又会重新提振起来，并影响到我们。

- **4.** 可千万不要低估这一轮的行情，我们可以质疑这一轮行情的到来时间，但绝对不能质疑这轮行情的庞大威力。一轮行情的力量，基本和它的“沉淀蓄水期”直接划等号，按捺压抑的多久，爆发的时候就多汹涌。因为世面上所有的资金和欲望，都会如同漩涡一般聚集在这个领域里，以至于理性和逻辑统统丢失。

- **5.** 那么这个阶段，哪种人是肯定赚不到钱的?
行事胆小如鼠、内心自命不凡、却又对财富极其贪婪，妄图一把翻身的人。
还有。
就是那种整天自诩聪明通透，可是从来没有在投资这门领域，真正下苦功的人。

- **6.** 那么哪种人，是能赚到钱的?
曾经完整经历过一两轮行情，并且非常认真的总结出了自己的投资思路，并时时刻刻都在优化改进，而且清楚知道自己“段位”的人。
以及。
明白自己天资普通，所以只用极小的资金，去体验沉淀经验，然后不停的学习顶级高手的思路，再慢慢把财富雪球滚起来的人。

- **7.** 注意，以下不是操作建议，仅仅是灏泽我自己个人的思考。
很多人听到“还会有几次调整”，就感觉自己已经错过了最好的布局时间，这其实是错误。真正把握住大行情的人都清楚，在券商正式发力以前，一切都是铺垫。因为在券商发力以前，普通的众生是几乎没有什么耐心和魄力，真正的埋伏在底部的。别看无数人整天在那里说“买在悄无声息时卖在人声鼎沸处”，真正能践行的，压根就没几个，各个都是只懂得打顺风仗的主。

所以，券商发力以前，真正进场的资金普遍都是嫡系部队，他们熬得住也耗得起，所以反而能拿到最实惠的筹码。

那为什么每一轮行情，都是以券商猛涨为特征呢？

很简单，如果券商都拉不起来，那就不要指望大牛了。

- **8.** 对普通人，灏泽最大的建议就是，如果你的主业很有前途并且能为你带来很宽裕的财富回报，那么你的确应该认真上班。但如果你的主业，是一份弃之可惜的鸡肋，薪资低、机会少、烦事多。那么你就不应该再浪费时间了，灏泽不要你辞职，但是你应该把精力投入到“体验市场、沉淀内功”上头去。

或许这件事并不能直接的为你带来财富，但它能极大的提升你的内功，你会更加清晰的掌握局势、利益、企业、财富等各项重要能力。这些能力，会在以后爆发出极大的力量。

- **9.** 如果要报出个比较健康的数字，那么我的意见是这两年就尽量控制开支，努力的存钱。存款中的15%可以作为先遣部队，不停的磨练自己的盘感和直觉，另外的25%可以作为主力等市场大调整后，再进行布局，还有的10%就作为应急的资金，可以看哪些机会是适合抄底的，再安排。随后，其余你还有50%的存款，就不要拿来做任何风险性操作了，就是稳稳地守住，作为生活的保底即可。

- **10.** 之前灏泽我一直认为银行很好，当然我现在也觉得银行不错，只是作为领跑的其力量可能就不如别的新秀了。所以，如果是从中期来看，上面一再提及的概念，你可以观察。然后，依然还是高股息的高精制造业是主角，其次再是高股息的半垄断行业龙头。注意，如果上述这些企业，你自己完全没办法找到，那么你就真的真的不适合做任何投资，至少现在是肯定不行。因为这说明你就连最最基本的入门，都没有过，所做的决策也自然一定是会犯错的。

- **11.** 不出意外的话，下一轮世界经济大爆发，一定是以人工智能+机器人+电力基础+加工制造业为核心的。而具备综合实力，把这些玩意运作起来的，也就我们和白头鹰两家，其他的玩家都已经彻底退出竞争。我们的优势也明显，那就是极强的跟进能力+超强超稳定的基础建设。所以，如果你能把上面这段话看明白，你就自然知道对应到现实投资里，哪些企业是可以提前关注的，而且这些企业绝对是会实现从“草根到龙头”的强大逆袭。

- **12.** 这一轮行情还有一个很特殊的点，就在于如果你这一轮把握住了，那么你就等于真正建立了自己的“财富地基”。因为，只要你耐心选中了一个长期高股息回报的好企业，那么哪怕只是二三十万的数字，它也是能慢慢滚动并壮大起来，形成很好的正向势能，并最终为你带来源源不断的回报。很多人不明白，总以为发财是¥0 → ¥10,000,000的一步登天。殊不知，这种一步登天背后往往风险极高，而且就算赚得到却也守不住。而真正的大财富，一定是¥0 → ¥10,000 → ¥50,000 → ¥100,000 → ¥200,000 → ¥500,000的循序渐进。对有天分的人来说，当你跨过了¥500,000的坎儿，并且能很好的用高股息好票来守住，那么人生财富自由之路，基本就是提前锁定的事。
