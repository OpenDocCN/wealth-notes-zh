

# 中国正转向消费型社会，这是几十年来难得的重大转变

最近，好几位同学说，想听我讲讲对于国家育儿补贴的个人理解，就是全国普惠性给不超过3岁的孩子发育儿补贴。其实，就人口和经济的话题，我在第114、115讲聊过自己的观察。今天，我想请你关注的是育儿补贴等一系列事关消费的政策背后，一个更大的本质趋势。

最近非常多的政策动作，都加速指向了扩大居民保障。除了育儿补贴，还包括我们在第81讲课程提到的，把毕业后未就业的困难大学生，纳入救助范围，以及最近的消费贷贴息政策和新一轮国补，还有各地保租房加速入市等等。

这些政策背后，其实就是7月30日的中央政治局会议指出的，要“在保障改善民生中扩大消费需求”。我认为这个意思很明白了，就是消费这件事，光靠刺激是不够的，必须要通过保障和改善民生，让大家压力小了，愿意花钱，才能最终振兴消费。

关于提振消费，国家的政策还不止于此。早在今年3月的《政府工作报告》中，就提到一个关键说法，叫“投资于人”。最近，这个说法又有了新一轮的政策吹风。8月3日，《人民日报》发表以《加力民生保障，激发消费潜能》为题的文章说，要把更多资金资源“投资于人”。

注意，这可是国家发展思路的一个重大转变，简单说，就是要从“投资于物”的生产模式，大力度转向“投资于人”的消费模式。国家发改委下属的《中国改革报》8月4日发表文章直接说，从生产型社会向消费型社会转型是必然选择。在我看来，这是几十年来难得的重大转变。同样的，前面提到的一系列扩大居民保障的政策，也是这个转变的一环。

所以我说，从生产型社会，到消费型社会的转变，才是真正值得你关注的大趋势。

## 为什么要转向消费型社会？

那么，为什么现在要进行新的转变呢？

我们之前说过，中国过去的经济增长，高度依赖地方政府主导的投资驱动模式，而投资，投向的主要就是产能。但这种模式当前面临很大挑战，我们先看看内部视角下的情况：

- 一是债务风险积累。一些地方政府债务、城投债务的规模迅速膨胀，这部分我在第157-159讲课程都详细说过了。从这些地方政府，到企业，到居民，都是高杠杆，债务太多，没法再加了。
- 二是存在投资效益下降问题。随着基础设施日趋完善，新增投资效益在不断减少。
- 三是资源的错配。不仅是一些传统产业，连一些新兴产业也在特定领域出现了产能过剩问题，产能过剩又带来领域内的严重内卷，这就形成了一种负反馈机制。

而从外部视角来看，中国的生产型社会，同样面临挑战。

我认为，中国过去四十年的经济崛起，本质上是全球产业资本与中国生产要素的完美结合。中国生产型社会的核心逻辑在于：通过制度化的低成本优势，比如劳动力、土地、资本、环境，来承接全球产业转移，用规模经济，摊薄固定成本，用出口导向，实现资本积累。

这种模式创造了人类经济史上最快的工业化进程，使中国在短短几十年内，完成了西方三百年的工业化。

然而，这种模式发生在全球化的黄金时代这个特殊历史阶段，更具体地说，它建立在两个大前提之上：第一，当全球产业分工处于扩张期的时候，后发国家可以通过技术模仿，实现追赶；第二，当全球贸易增速持续高于GDP增速的时候，生产型社会可以依靠不断扩大的外部市场，消化产能。

但这两个前提，在当今世界都已经发生了根本性变化：一方面，技术扩散红利正在衰竭，在芯片、生物医药、高端制造业等关键领域，中国面临发达国家的强硬技术封锁；另一方面是全球化退潮，全球贸易增速从2008年后，连续多年低于GDP增速，以美国为首的发达国家，试图进行贸易脱钩，中国作为“世界工厂”面临挑战。

所以我认为，我们已经进入了一个“不得不转”，也是“必须要转”的阶段，而解决办法就是要从生产型社会，转向消费型社会，这也是过去一段时间和未来很长时间，我们很多政策的出发点之一。

听到这里，你可能要问了，这个消费型社会到底是怎么样的呢？

根据我的观察和研究，消费型社会并不是简单地鼓励居民多消费，而是构建一种以人的需求为核心、供需良性互动的经济体系。在消费型社会中，消费在经济社会系统中发挥核心调节功能，具体包括，消费需求引领投资方向，决定生产结构，塑造产业形态，形成“需求牵引供给，同时供给创造需求”的新平衡。

此前的生产型社会中，劳动者报酬占GDP的比重偏低，一定程度上抑制了消费能力。而消费型社会则强调，提高居民收入、完善社会保障，形成内需驱动的增长模式。我认为，这也是7月30日中央政治局会议提到的，要“在保障改善民生中扩大消费需求”的真实含义。

另外，转向消费型社会，也是应对当下技术冲击的一个解决方案。目前，人工智能、自动化等技术的发展，正在改变传统生产模式，许多低端制造业岗位可能被替代。如果社会仍然以生产为中心，提供的新增岗位有限，可能没办法容纳那么多就业人口。而消费型社会更注重服务业，能够创造更多的就业机会，来应对科技进步带来的就业冲击。

而从生产型社会转向消费型社会，是一个国家要跨过中等收入陷阱的关键，欧美日韩等国家，几乎都经历了这种转变，才最终成为了发达国家。

## 三个关键问题

但要实现这种转型，绕不开三个关键的问题：

- 第一，谁来生产，谁来消费？
- 第二，生产什么，消费什么？
- 第三，怎么生产，怎么消费？

对于这些问题，美欧模式的答案是“全球化产业转移+福利政策”。简单说，就是把工业制造业，向其他国家外迁，利用外国的土地和资源，来生产，同时用本国福利政策培育出来的居民购买力，来消费。这里面，外国生产制造的中低端产品，由美欧消费，美欧生产的金融和科技产品，由全球来消费。

但显然，我们没办法，也不会简单复制西方经验，中国要探索的是超大人口规模下的消费型社会发展新路。这是因为人口庞大的中国，有着同样庞大的地区间的发展不平衡，比如东部和西部的就差了一大截，东部能很容易转向消费型经济，但一些西部甚至连真正的工业化都还没开始。

所以，我们现在观察中国当前的一系列政策，其实都是围绕着前面三个问题展开的，我们依次解释。

对于第一个问题，谁来生产，谁来消费，我们的答案是要自己生产，自己消费。

在生产端，中国要通过新质生产力和新型工业化，实现东部制造业的升级；而中西部，则承接东部相对低端的劳动密集型产业转移。总之，不论高端还是低端制造业，最好都捏在手中，既要打破西方制造业的技术封锁，也要避免产业外迁导致的空心化。

与此同时，国企与民企进行详细分工，国企主导能源、安全等重大战略领域，民企聚焦消费、创新等市场领域，同时鼓励中小企业，满足细分的市场需求，激发创新活力。

而在消费端，政策则是要通过一系列方式，来激发城市家庭的消费，包括提高工资性收入，拓宽财产性收入渠道，加强社会保障兜底，优化“一老一小”服务供给，保障普通人双休权利，加强保障房建设、消费品以旧换新等；同时，通过乡村振兴、农民市民化、电商下沉等，来激活农村家庭的消费。

对于第二个问题，生产什么，消费什么，我们的答案是生产要升级，消费要扩容。

生产什么，消费什么，这里面的核心矛盾，就是低端产能过剩与高品质供给不足。所以这两年的国家相关政策，特别强调要严格反内卷，要淘汰落后产能，对一些传统产业进行包括环保和限产在内的措施，对地方政府的招商引资进行严格的规范，确保低端产能能够出清。

与此同时，大力发展数字经济、低空经济、海洋经济、冰雪经济以及平台经济等，鼓励提供更高品质、更高附加值的好产品，并扩大服务消费，特别是扩大养老、托育、教育、医疗等短缺服务的供给。

对于第三个问题，怎么生产和怎么消费，我们的答案是智能化、数字化生产，和构建全域消费场景。

生产端，是要推动生产方式的智能化、数字化转型，改造传统产业链。其中，由国企主导，统筹关键领域技术攻关，民企依托数字平台，实现柔性制造，同时从东往西，进行高中低产业的梯度构建。

消费端，则是要构建“全域消费场景”。比如通过免税经济、夜间经济、文旅IP开发，释放消费潜力，通过县域商业体系，打通农产品上行与工业品下乡的双向通道。这背后，要通过“全国统一大市场”，来破除区域壁垒，使消费基础设施与供应链深度匹配。

在这个过程中，今年国家也在大力解决对中小企业的拖欠账款问题，要让财富流下去，流向更多中小企业和普通人。

所有这一切，我认为都是为了形成“生产-分配-流通-消费”的良性闭环，通过从“投资于物”向“投资于人”转变，实现从生产型社会向消费型社会的转型。这件事不容易，但是目前看国家下决心要做，因为只有消费起来了，更多的普通人才能真正挣到钱。
