

# 许倬云：找到你自己的桃花源，哪怕世界永远不平静

250806

整理：公众号懒人搜索，懒人专属群独享
懒人微信：lazyhelper

![](img/a29136779139f19c94bd6f5cea071a86_0_0.png)

今天从一个很遗憾的消息说起。这周一，8月4日，著名的历史学家，也是华语世界最具影响力的历史学家之一，许倬云先生去世，享年95岁。

很多外行人可能对“历史学家”这四个字感受模糊。怎么判断历史学家的高下？是知道的事情多？查阅的史料足够细致？这些都不全面。学界有一个普遍的标准，叫“做对事，讲透理”。做对事，指的是你能从那些数量庞大，真假混杂的史料里找到最可信的部分，讲事实，辨真假。而讲透理，就是要在这个基础上追问一句，为什么？当事人为什么这么做？形成这些的原因是什么？

假如说历史是关于现实的，那么历史学家就需要超越现实。比如，讲中国古人的“天下观”，古人认为自己是世界中心。有的电视剧里可能用一句“古人地理水平有限”带过去，而许倬云先生就会由此追问，为什么黄河流域稳定的农耕文明催生了这种观念？这个观念又怎样影响了中国与周边民族的互动？等等。

历史学家葛兆光先生曾经评价说，大历史要有大判断，非博览硕学之士，不能下大断语。在许先生这种大历史著作中感受最深的，就是那种截断众流的大判断。如今，历史知识被各种各样的原因歪曲、遮蔽和改写，特别需要真正专业的学者，用不是“戏说”或“歪批”的方法，来给大众普及和清理。

其实对于死亡，许倬云早有打算。2023年4月，他在一次采访中说：“我死了要埋葬在家乡，我买了坟地，碑都刻好了，就在父母的墓葬旁边。”后来，许倬云先生的太太孙曼丽在采访中也说，他不恐惧死亡，理性上他已经做好准备了，只是在感情上还有很多的舍不得。

要想理解这份“既无恐惧，又舍不得”的情感，我们得回到许倬云的经历。

1930年，许倬云出生在厦门鼓浪屿，他和弟弟是双胞胎，也是家中最小的两个孩子，许倬云排行老七，弟弟老八。许倬云先生的名字取自《诗经·大雅·云汉》的首句，原文是“倬彼云汉，昭回于天”。这句话描写的是银河璀璨、蜿蜒曲折的天象。其中“倬”字的意思是，广大、光明。

但假如看过《诗经》原文就会发现，这首诗歌其实围绕周宣王时期的旱灾展开，全诗以周宣王的视角，表达了对上天的敬畏、对民生的忧虑以及祈雨的迫切。

而这大致也跟许倬云的生平贴合。由于母亲早产，许倬云出生时只有两斤七两重，而且患有先天性肌肉萎缩，直到七岁时才能勉强坐在椅子上。后来他好不容易有了一点行动能力，结果 1937 年抗日战争全面爆发。因此，他只能跟随家人四处逃难，颠沛流离。

所幸，许倬云先生本人非常争气。在他十六岁那年，抗日战争结束，他也有机会进入无锡的辅仁中学就读高中。20 世纪 40 年代，他跟随家人赴台，考取了台湾大学。更幸运的是，1957 年，得益于胡适的帮助，许倬云获得了去美国芝加哥大学攻读博士学位的机会。

1962 年，他学成归来到台湾任教。1970 年，许倬云先生接受了美国匹兹堡大学的邀约，前往美国担任访问教授。从此许倬云得以更加专注地从事历史研究，他的学术代表作《西周史》《中国古代社会史论》《汉代农业》被称为“古代中国三部曲”。1999 年他从匹兹堡大学历史系退休后撰写的大众史学作品，尤其是《万古江河》《说中国》《中国文化的精神》三部作品，被誉为“中国文化三部曲”。

当然，很多人认识许倬云，并不通过这些书，而是他说过的一些流传很广的话。

关于读书，他说，读书不是为了学位，读书是为了获得一种判断世界的能力。有了这种能力，人就能看见环境，懂得环境。

关于人生，他说，这一辈子，“完整的人”这四个字是你的责任。保持你的完整，不屈服、不腐化、不猥琐。

关于变化，他说，这个世界上，唯一不会改变的事实，就是我们经常不得不改变。

关于躺平，他说，你的存在就是你的能量，躺下投降的人，未来就报销掉了。

许倬云先生还说，他最重要的人生态度之一是，要有一个远见，能超越你未见。那这个“远见”具体是什么呢？他的原话是，拿全世界人类曾经走过的路，都要算是我走过的路之一。

那么，许倬云先生走过的路是什么呢？有几段经历特别值得了解。

首先，是面对困境时的乐观。

许倬云先生因为先天肌肉萎缩，一辈子离不开拐杖。年幼时他经历抗战，错过了手术治疗的最佳时机，直到28岁时才动手术矫正了畸形的双脚。但这也只是勉强让他挪动步子。而到了晚年，他的行动更是不便，起床只能依靠电动的吊兜，把身体吊到轮椅上，出门时要从轮椅抬到汽车上。

要是普通人，可能很容易就被先天的缺陷打败了。但许倬云说，我自己没有什么了不起的修养，我虽然残废，但我不惭愧。儿时他的父母就时常对他说，老七你虽然不方便，但我们疼你跟老八完全一样，并不因为你的外表，因为你是我们的孩子，因为你的心、因为你的笑容。

后来在很多公开的采访中，许倬云先生都反复强调，真正的智慧不关乎外表的美丑。

还有一段经历，发生在他即将从台湾大学毕业那年。当时有一个数额不小的奖学金，专门用来资助赴美留学的青年学子，选拔过程很艰难，而许倬云还是考了第一名。但基金会的秘书回话说，出资人生前有遗嘱规定，获奖者必须身体健全，由于许倬云是残障人士，资格就被取消了。当时，台湾大学的校长钱思亮以及国际考试委员会的工作人员都非常气愤。

但许倬云先生去见校长和院长时说的是，老师你们不要难过，我现在在你面前，我变了一点没有？我有没有更矮了一点？我有没有更高一点？还是更丑了一点？即便没有奖学金，我仍然跟本来一样。

这种乐观态度一直贯穿许倬云的一生。在疫情隔离期间，他面向大众说，欲望达不到的时候，你必须知道，人不可能所有欲望都达到的，每个人都有抓不到的云，都有做不到的梦。你要理解。抓不到的云让它飘走吧，做不到的梦，有机会再做也好，没机会再做，你做别的梦。要先找到自己，找到真正的问题所在，才能往里走，安顿自己。

说完许倬云先生对待自己的态度，我们再看看他如何对待外部变化。

许倬云先生的人生经历跨度非常大，前半生经历了抗战、二战、冷战，晚年又见证了人工智能等新技术的发展。

许倬云先生就说，变化之中可能是机会，变化之中可能也是危机。尤其在现在这个时间段，人生的惊涛骇浪是免不了的。因此他的叮嘱是：第一，不能慌张；第二，不要放弃，最要紧的是扎实自己。

这种态度与他本人的经历有很大关系。抗日战争打响后，许倬云的父亲许凤藻在湖北沙市任职。

在年幼的许倬云的记忆里，经常有人到沙市投奔他们。有一回，一个姓廖的海军军官带着两个小兵在他家住了一周，天天给他讲故事。有天深夜，廖队长一行人辞别。后来许倬云才知道，就在那一晚，廖队长带着两个兵，乘着一船炸药划到日本军舰旁边，发动了自杀式的袭击。

后来沙市沦陷后，他们一家又开始逃难。在路上，小许倬云目睹过被日本飞机轰炸过的战场。有些时候，上午还在一起玩耍的小伙伴，下午就变成了一堆残骸。直到1957年到美国读书时，他还会常常在睡梦惊坐，恍惚中以为是日本的轰炸机又来了。

由于见证了逃难时期的人间百态，许倬云一直非常关注历史中的常民，也就是普通人。他在采访中说，满路的人奔走，没有人欺负人，挤着坐船坐车逃难，上车上船都推着老弱妇女先上，自己留在后面。大路上奔走，多少老年人走不动了，给孩子说，你们走，走。

而这段经历也推动他后来做大众史学。2006年，他的著作《万古江河》出版，相比于他之前出版的上古史研究专著，《万古江河》这本书中没有帝王将相，也不讲开疆拓土，而是关注老百姓的衣食住行、思想信仰。

许倬云说，为生民立命，就是为世界帮忙，这是儒家的本分。

换句话说，也许是因为那段不知道下一站是哪里的日子，让许倬云先生看见了每个个体的苦难，也看见了人与人之间的温情。这也为他之后的研究，奠定了不可或缺的基础。

最后，我们就借用许倬云先生的一段话，作为今天的结尾吧。我们都听过那个很有名的故事，陶渊明寻找桃花源。但桃花源是别人的住处，万一别人的桃花源不接受你怎么办？许倬云先生说，要找到自己时代中的位置，找到自己的桃花源，家庭里的和谐，夫妻之间的相濡以沫，朋友之间的依赖，这些都是桃花源。

世界永远不可能平静，你只能求自己的安定、冷静。
