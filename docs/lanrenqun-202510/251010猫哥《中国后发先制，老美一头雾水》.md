

# 中国后发先制，老美一头雾水

## 251010 猫哥

本期内容我们聊聊9月29日国务院发布的最新版《中华人民共和国国际海运条例》，算是国庆长假结束后的一个补课。

9月12日，国务院第68次常务会议通过了关于修改中华人民共和国国际海运条例的决定，并于国庆长假来临前2天公布并生效。我们捡重点修改内容做个梳理：第一，补齐对数字航运的监管，将国际航运线上交易平台纳入我们监管范围内。

第二，要求平台经营者主动向我们报备信息，包括服务协议、航运交易规则等，如果逾期不报备，除了罚款外，还可以升级为停止开展相关业务。

这样做的目的除了保护交易公平和数据安全外，更重要的是让我们摸清具体情况，便于未来精准监管。这块很重要，因为我们现在是国际航运最大的利益相关方，不能搞一刀切，否则最受伤的是自己。

第三，也是重头戏。任何国家或地区对中国航运企业、船舶或船员实施歧视性禁止、限制措施，我们都可以启动反制措施，包括：
- 1、对相关国家船舶收取特别费用。
- 2、禁止或限制其船舶进出中国港口。
- 3、限制其获取中国航运数据或经营相关业务。

最值得注意的是，条款删除了过去“对等原则”的表述，改为“根据实际情况采取必要的反制措施”。这意味着反制措施不必在形式和规模上与对方完全一致，可以根据实际情况更灵活、更有力地回应，增强了策略上的主动性。

那么，为啥我们要抢在国庆放假前，要将修改内容进行公布并立即生效呢？

虽然字里行间中没提针对哪个具体国家，但答案再明显不过，就是冲着老美去的。

因为早在4月份，老美就针对我们造船航运业启动了301调查行动，并宣布对我们拥有、运营或建造的船舶征收港口费，从10月14日起每净吨50美元起，然后未来三年阶梯式上涨，初步目标是涨至140美元/吨。

这项变被动为主动的战略举措，我们倒是安心欢度国庆8天长假了，但却让老美那边却炸开了锅，为啥？

因为川普搞的这个动作，原本就是李嘉诚出售全球43个港口资产给贝莱德等美西方财团的配套政策，手上掌握全球更多的优质港口资源，这项加征港口费的计划才有意义，但大家都清楚，由于我们出手，导致该笔交易未通过反垄断审查。

到目前为止，这笔交易还处于停摆状态，还在设计交易方案，包括引入中资股东等。

所以，从6月份后，老美方面就呈现出“雷声大、雨点小”的状态，没得任何跟进措施。以至于马上到10月14日，老美的航运业内人士仍然不清楚具体实施细节。

本来就是一头雾水，现在我们又突然出招，修改海运条例，你说老美那边是不是更懵逼。

现在我们把难题甩给川普了。

要么选择认怂，放弃加征港口费。

要么一条道走到黑，继续选择10月14日加征港口费。

对于川普而言，两条道都不好走。

认怂？不大可能，自己一直标榜的就是“赢学”祖师爷，怎么可能轻易认怂。

按原计划执行吗？

缺少李嘉诚43个港口资源，手上筹码少得可怜。现在全球前10大港口，我们独占7席，包括上海、宁波、深圳、广州、青岛、天津、香港。而老美拿得出手的洛杉矶—长滩和纽约，也就排在14和15位。

要论港口吞吐量，我们对老美完全处于碾压级别，就算把前十的另外三个港口（新加坡、釜山和鹿特丹）捆在一堆，也差得远。

如果川普选择头铁，损失会很大，毕竟全球90%以上的物流是依托海运，而美国的农产品、能源要出口到亚洲，大部分还要依托中国港口周转。

假如我们也每吨征收50美元，信不信川普分分钟被红脖子喷得体无完肤，中期选举民主党基本可以提前宣布胜利了，要知道农业和能源是川普的基本盘。

所以，这个国庆节，估计白宫的电话被打爆了。川普原本可能以为收购李嘉诚港口这事既然出现僵局，那加征港口费这事最好就不了了之，但没想到我们突然将军，又演一出求锤得锤的范例。

制造业回流、振兴美国造船业，说实话，连川普自己都不信，无非就是想凭空造牌，制造和我们博弈筹码，没想到又把自己架在火上烤。

而且，我们这次对老美的反制还不是单纯的对等加港口费，是以组合拳套餐形式端出来。

不仅如此，我们提前还做足了功课。

比如，从川普宣布准备要加征港口费后，我们就开始调整优化航线，主动避开老美港口停靠。虽然时间长点、费用多点，但羊毛出在羊身上，绝大部分是老美自己买单。包括最近几个月，像沃尔玛、亚马逊这些主体，就不停抱怨商品供应跟不上，价格提高，消费者购物体验下降等等。甚至连以往繁忙的洛杉矶—长滩港口，现在也变得稀稀拉拉。

国际航运本身就是复杂程度相对比较高的产业，就算船是我们制造，但投资人有可能不是我们，注册地有可能不在中国，运营方也有可能不是中企，又参与国际航运，成天到处跑，真能分得清楚吗？

而且，虽然美国航运业不咋地，但美国金融投资业发达，保险业务、融资租赁业务、私募股权业务早就深度参与国际航运业务。

所以，针对中国船舶搞的301调查，本质上就是与全球PK，当然也包括美国相关资本，搞得下去才怪。

说实话，对于单极管思维的川普政府，在面对我们这样体量的对手，处理复杂问题的时候，吃瘪就是早晚的事。

当然，最最核心的因素，还是当下我们有对抗老美的硬实力。

根据老美自己的研究中心发布的最新报告显示，按总吨位计算，今年1-8月，中国船厂承接全球新订单占比已经突破50%，超过2023年全年。

这还仅仅只是造船这一项，并且优势还在继续扩大。还有前面提到的港口吞吐量、集装箱拥有量等等。特别是稀土，现在全球90%稀土运输都必经我国港口，说不定哪天连商用都给禁了。

其实就个人观点而言，还要再次感谢川普，没得川普急吼吼的对我们下手，我们贸然修改国际航运条例，可能存在冒险激进。现在我们出台线上交易纳入监管、要求报送信息、一系列反制措施，就显得顺理成章。

我们是在合理合法维护自身权益，其它整天琢磨着想打歪主意的相关方就算要告状，也没得抓手。

现在的中国，某种程度上欢迎川普来踢馆，因为每次踢馆后，我们都正大光明地前进一步，稀土如此，造船航运亦是如此，助力我们从海洋大国迈入海洋强国。
