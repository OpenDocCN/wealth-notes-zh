

# 王健林被限高的背后

251005 猫哥

**整理：公众号懒人搜索，<u>懒人专属群</u>独享**

**懒人微信：**lazyhelper

![](img/faa77e8aeaff36528f4cc2488ef47c63_0_0.png)

这两天，王健林（简称：老王）先是被限制高消费，接着又取消，许多人的第一反应是，这是房地产周期的缩影，王健林仍深陷其中。

2016年退市前，万达在港股的总市值，仅为2376.8亿港元（人民币约1924亿元），每股最低价到过31.1港元，跌破了发行价不说，还不如万达的净资产。

王健林对这个价格严重不满，认为太低了，决定将万达私有化，从港股退市，寻求在大A上市。私有化需要钱，万达彼时缺乏足够的资金。

找外援的话，就得签苛刻的对赌协议。

但那时，地产行业还处于上行周期，万达觉得，就算对赌也没什么，于是和企鹅等多位战略投资者，签下了利息高达12%的对赌协议。

如果拿了他们的钱，却不能在大A上市成功，就得付出高昂的回购代价。

老王对此并不担心，在老王的计划中，2017年后，万达将逐步抛弃住宅业务，2020年后，除万达城配套的住宅外，万达原则上不再新增住宅开发。

等万达城开发完，就从地产开发行业退出，转而布局文旅、商管、电影和投资四大业务，实现转型。彼时，老王在年会上大吐口水：

> “我从事地产行业28年，再两年就30年了，被调控了十几次。中国地产好年景没有持续四五年的时候，基本上三年左右来一轮调控。行业周期性太强，造成现金流不稳定，预期也容易经常发生变化”。

但老王的雄心勃勃，遭遇了现实的一次次重击。后面的事情，大家都知道了，万达一次次在大A上市失败。

无奈之下，老王只能不断放弃重要资产，先是大部分万达广场的控制权。

2023年，珠海万达商管，也就是万达广场背后的主体公司，王健林的持股比例，从60%下降到40%，作为战投的太盟成功入股。

在入股后，太盟逐步把自己人派了进来，把万达的旧部踢了出去。

其次，是万达电影，也是在2023年，中国儒意先是获得了万达电影49%的股权，随后，中国儒意又获得了万达投资51%的股权。

由于万达投资持有万达电影部分股权，那么，中国儒意在控制万达投资后，等于间接获得了万达电影的部分股权，加上前面的49%，完成了对万达电影的控制。

投资和电影，老王寄予厚望的，要转型的两大板块，都被中国儒意给拿下了。

中国儒意的创始人，名叫柯利明，虽然腾讯旗下的Water Lily Investment Limited公司，为中国儒意的第一大股东，但这不等于腾讯控制了儒意。实际控制者，还是柯利明。

就像腾讯曾经是京东的第一大股东，但控制不了京东，京东依然属于强子。

当然，腾讯能从这两桩买卖里获益，是一定的。

说回老王，到了2025年3月，老王又失去了“万达酒店管理”的控制权，“万达酒店管理”被出售给了同程。

接连失去电影、投资、酒店管理三个板块，万达广场也不断卖卖卖，老王手上仅剩的大牌，就是文旅。

文旅，成了“全村的希望”。

然而，文旅也不是好走的路子，本次被限高，就是文旅“惹的祸”。

2021年，万达想在武汉开发一个文旅综合体，内部包含滑雪乐园、酒店、古风小镇等等。当时，万达的资金状况已不容乐观，自己是没钱的。

所以，万达找了光大兴陇信托，进行融资，光大兴陇信托的背后，是甘肃国资委。但钱借出去后，项目进展不顺，万达还不上。

于是，光大兴陇信托一纸诉状，将万达旗下的部分资产冻结了，并对老王进行了限高。

实际上，2018年后，万达的游乐园逐渐卖光，在文旅业务上也逐渐转型，采取轻资产路线，像武汉项目那样，搞重资产模式的已经很少了。

说白了，就是不持有实体资产，而是和实体资产的所有者（大部分是地方政府或者地方国资）合作，一起开发，获得的收益按比例分成。

这个模式只是看似美好，实则缺陷大。

实体资产在别人手中，说不和你合作，就不和你合作，不确定性非常大。比如，万达和大同市政府合作开发的大同古城项目，双方签完战略合作协议，就没下文了。

类似的事情，不是一次两次。

归根结底，对文旅行业而言，景区，或者说，优质的实体资产，才是核心资源。手握核心资源的地方政府，和谁合作不是合作？

万达的开发能力，也没到无与伦比的地步。不是离了你，就不可以。

其次，文旅这碗饭，是典型的看着香，吃起来不容易。

文旅行业的本质，有点类似于网红。从大部分游客的角度，尤其是非本地游客，他们往往不会重复去一个景区，顶多是尝鲜一次、体验一次，就很难再去第二次了。

毕竟，全国、全世界，有那么多地方能够尝鲜，除非体验真的很独特。如果体验不好，或者只是一般般，那么更不可能有第二次。

这是因为，文旅的门槛，并不高，一个地方依靠一种玩法走红后，会被迅速复制粘贴，全国各地的古城、风景区，以及各种游乐园综合体，玩法都差不多。

所以，大部分爆红的景点，往往热度只能维持几个月，就像一阵风。吃一波红利，后面就没有了。

而那种能长红的项目，通常景区本身，就是优质资产，不需要和外部合作太多。

重资产，万达不想搞，轻资产，不确定性大，优质的文旅项目很难合作，能合作的，本身往往不优质，前期需要大的投入力度，于是万达的文旅业务陷入了尴尬。

万达目前的有息负债，超过 3000 亿人民币，每年光是还利息就压力山大。诚然，老王手中，还有一些万达广场，靠卖广场和其他资产，还能撑一阵子。

如果文旅这个全村的希望起不来，又没有其他新业务作为开源（万达目前的状况，也很难开展新业务了），那再怎么苟延残喘，也有撑不住的一天。

老王有一句名言：清华北大，不如胆子大。

这句话生动地体现了，那一代成功人士，往往不知道自己因何而成功，总以为是自己的成功是靠奋斗，靠胆子大，靠各种超前的眼光。

殊不知，本质上是时代红利。

大多数产业的成功者，如果没有时代红利，几乎不可能取得成功，房地产行业亦然如此。城镇化阶段，他们相对容易吃到巨大的红利。

等城镇化完成，再怎么有眼光，再怎么闪转腾挪，也只能是延缓下滑，改变不了最终的命运。老王从风光到困境，就是这一切的生动体现。

而一旦老王撑不住，小王就很难潇洒了。毕竟，当债主们讨起债来，可不是闹着玩的。
