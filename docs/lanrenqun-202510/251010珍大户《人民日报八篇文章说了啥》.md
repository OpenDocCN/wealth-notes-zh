# 人民日报八篇文章说了啥

![](img/26c3d8c3cc036d31dc4199f60ecee704_0_0.png)

![](img/26c3d8c3cc036d31dc4199f60ecee704_0_1.png)

那么，这次的文章发表，大盘会不会像上次那样，也是大幅下跌的开始呢？

珍大户：

首先不能这么简单的比，当年有个啥事，如今又有个啥事，是不是就一样？这完全属于刻舟求剑了。重要的是因果关系，而不是同时出现。

2015年的大跌有多重原因，一个重要原因是五六月份开始严打配资和杠杆交易，很多查杠杆的仓位被迫卖出平仓。刚一开始是1：10配资的被关停卖出，卖出导致市场下跌，然后1：8杠杆的到1：5杠杆的又开始被关停卖出，继续下跌。

另外是ZZ因素，很多年后传出来的消息，是当年有人利用金融市场搞个大幅下跌来逼宫。

反正不管怎么说，都跟人民日报文章没一毛钱关系。

但是早上进圈子后看到好几个圈友提问，我就赶快去找了原文去看了一下。

感觉8篇文章像是一篇长文拆出来的，每篇讲一小个主题。第一篇：人民日报钟才文：从全球视角看新时代中国经济的跨越与蝶变，这篇显然是“总分总”长文格式里的“总”的部分，主旨就是在说，我国经济成果斐然：GDP增长快，制度改革好，创新牛，全国各地都在发展，还能兼顾绿色环保，还能对外开放，老百姓生活质量节节高。

第二篇：人民日报钟才文：深刻认识中国经济长期稳定发展的内在逻辑，这篇是在讲，我们为什么这么牛啊？

- 1、党的领导好，不像老外，今天他当选，明天他下台的，三天两头的换人，成不了大事。
- 2、规划做的好，一直在搞五年规划，每年还有宏观调控，总之很棒！
- 3、本心没有变，一直为了人民，才能发展这么快..

反正第二篇的主题就是讲原因。

第三篇：人民日报钟才文：科学客观看待我国当前经济发展态势，因为有讲“辩证法”的思考模式，也因为文化里的谦虚、吾日三省吾身这种基因存在，所以第三篇还给自己找不足，说还要继续创新，还要继续化解风险、兼顾民生保障工作，要做好产业升级。

（但注意，这篇的“自我批评”很隐晦，依然用一种“我们超牛”的方式在传递。这种态度值得思考。）

第四篇：人民日报钟才文：中国经济转型升级蕴含重大机遇，第四篇是吊胡萝卜，并且尤其是对老外说的，说我国产业升级呢，转型呢，里面会有重大的投资机会（潜台词：你们不要光看英伟达啊）。

说我国每年500万的STEM工程师人才，（潜台词：全球谁敢跟我比）说我国这么多人，还有钱，每年买新车的量全球第一多，多大的消费市场啊，（潜台词：不要老以为美国人才消费力高）说我们还要继续城市化，继续新型基建，这过程中还有大量投资要拉动，（潜台词：我们还能当金主爸爸呢。）说后续民生保障医疗教育养老的投入越来越高了，（潜台词——这个目标明显是奔着之前每年大量外汇用于出国购买教育医疗养老的各种服务了，这里是想把国外那些服务机构吸纳入国内。）

第五篇：人民日报钟才文：全面认识把握中国经济高质量发展的确定性，第五篇依然是胡萝卜，并且非常确定的是对国外说的。不过是从风险角度，说国外动荡、不确定性高，只有中国的确定性是最高的，原因就是：第一，我国长期战略是确定的，不像老外，今天偏左，明天偏右的。

第二，每年的政策都是确定的，不像老外，三天一换人，换人了前面的章都不认了。

第三，我国人口多，不可能立马人少了吧。所以消费力是确定的。

第四，我国斗争能力强，面对老美强压，嘛事没有。不像其他国家，就算你们想确定性发展，你能斗得过川普吗？川普一压，你垮了，你不还是完蛋吗。

第五，我国政策欢迎国外，来吧。

第六篇：人民日报钟才文：中国是全球包容性发展的典范，第六篇是安抚国内群众的，说我国政策是不让人一个掉队（以脱贫攻坚为例），不让一个地区塌陷（以转移支付为例），让每个普通人都能有人生机会（以义务教育和大学教育为例），发展成果由全民共享（以全民养老金和全民医保为例，注意这里，再回忆我说的，未来我国会逐渐转型成福利社会，然后发福利的凭证可能就是靠社保资格之类的）。

第七篇：人民日报钟才文：中国经济发展具有开放共赢性，第七篇依然是对外的，潜台词是，我们会当好大哥的。

先说，过去多少年里，是中国对国际经济增长贡献率超30%啊，不像某些国家，控制不住自己国家的劳资对立，然后就左右搞事情。然后又说，只有我国才是全产业链完整的，能够保证全地球不掉链子的存在啊。然后又说，我们搞的一带一路，就是拉你们一起发展的、合作共赢的平台啊。

最后说——“反正我话放这了，欢迎来合作，就看你来不来呗”。

## 第八篇：人民日报钟才文：深入了解中国就会坚定相信中国

这是“总分总”格式的最后一个总。

啊～中国的理念超前～！

啊～中国是有正义、道义、责任的～！

啊～中国做了那么多年贡献，历史可鉴证啊～！

啊～中国可是屹立引领了上千年啊～！

总的来说，看完8篇文章后，我的直观感受是，觉得这都是正常的文章啊，好像没有啥新信息可说，甚至不值得写一篇“政策解读”出来。

原本是不想写的。然后我就开始去回顾国庆期间外围都发生了啥，觉得假期结束该补补课了。结果一不小心，看到高盛说看多全球股市，大概意思是，认为全球的经济开始改善了，美联储又要新一轮货币放松了，全球各国都要宽松了，所以未来一年乐观乐观乐观啊。

我顿时就来精神了。

确实，人的本性就是今朝有酒今朝醉，长期的让人绷着弦是反人性的，长期的奋斗、为了目标而努力，长期的节俭储蓄（比如我刚工作那些年的生活），这些都是反人性的，就如同上面第五篇里，不断在嘲讽老外，今天偏左，明天偏右，三天一换人，但这种才是人性。

像我们国家这种，有一个巨大的“体制”，像结成行军蚁一样能把国家不断往前推进，从根基上去发展，这种才是反人性的。

就像三体小说里，有一个情节，说的是三体人侵略计划曝光后，地球人曾经陷入恐慌，全球人都开始先减少消费，开始埋头研究对策，然后经济大萧条，人们节衣缩食，生活质量一度大幅下降。但是很快，大部分人就不干了，觉得三体人要到400年后才抵达，凭什么我们要为400年后的人奉献自己的一代呢，为什么不花天酒地的享受，让自己当最后一代，把所有能享乐的都享受光，把地球造光了，败家败到底，反正死了两眼一闭，关我毛事。

你看，这才是真实的人性。

所以，前面几年，不仅是中国困难，其他国家也很困难。他们处在的就像是刚得知三体人要来的时候，但是坚持不了多久，就会变成能享受就先享受吧，所以高盛推断的，后面马上要全球开始放松，是真的。

再看我国的态度，我感觉我国态度最近三年里有一个明显的变化，就是之前还是低调的，不会管外界怎么看、怎么评、怎么做，只要低头做好自己。

后来突然发现BUG了，就是发现原来吹牛逼是真的能救经济啊、原来改统计数据是真的可以美国第一啊，（怀疑是受美国最近几年做法的启发），

所以现在明显有一种，寇可往，吾亦可往的姿态。你们吹AI英伟达半导体，中国也香。你们吹制度，中国也香。你们吹美国暴打外星人，等着别急，你看我们马上来中国180度花式吊打外星人。你们上火星？别急，我们登月，上火星，飞深空宇宙。

我觉得可能是想开了，因为你想想，如果全球都继续大放水，全球吹泡泡，过个五六七八年，又危机了，就算这个时期里中国低调，不吹泡泡，中国到时候就能不受牵连，就能安全无事了吗？不可能。

我们保守了，低调了，吹泡泡时自己没享受到，结果泡爆时我们连累一起惨，那多亏啊。

所以我突然明白这8篇文章的意义了。吹~！一起吹！

先吹起来再说，到个五六七八年之后，万一中国吹的泡实现了60%，国外连20%都没实现，没准到时候吹到全世界都更信中国了，反而中国还更安全呢。

不吹白不吹。

所以我现在突然意识到点什么了。在上涨的周期里，不要贸然喊回调。黄金如此，大A也如此。

一些城市的房地产可能也到底了，大家国庆期间不知道有没有四处看车？记得后面也追加在各自所在的城市里探一下盘，都发贴汇报一下看了几个盘，均价总价、位置配套如何，因为后续可能又快到了全球集体放水的时期了。
