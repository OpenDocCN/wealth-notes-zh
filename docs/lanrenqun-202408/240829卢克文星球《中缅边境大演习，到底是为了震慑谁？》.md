

# 中缅边境大演习，到底是为了震慑谁？

240829

8月26日，南部战区在中缅边境一线我方一侧，举行了联合实兵实弹演习。

而就在十几天前，中国外长王毅刚刚访问了缅甸，会见了缅甸的最高话事人、国防军总司令敏昂莱。

刚刚访问就在边境演习，还是在今年4月刚在边境演习完、短短4个月之后再次演习？这说不通啊！显然，这次演习，别有深意，目的是在震慑某个势力。

那么，震慑的是缅甸军政府吗？并不是，恰恰是前一阵子“打击电诈功臣”——果敢的民族民主同盟军。

咱们都知道，自去年10月27日，果敢同盟军与德昂民族解放军、若开军成立了“三兄弟联盟”，向缅军发起代号“1027飓风行动”的攻势之后，在短短2个月里，就成功拿下了果敢全境，完成了过去8年未能实现的战略目标。

随着《海埂协议》的签订，战乱的缅北总算迎来了短暂的和平。但进入2024年第二季度，情况又有了新变化。

- 第一是战火重燃。
双方休整了几个月，觉得自己又行了，重新开打。不得不说，敏昂莱有点太自信了，在与缅北民地武开战的时候，自己那边的人民防卫军（PDF）还没剿灭呢。

PDF是被缅甸军政府推翻的民盟组织的武装，因为有合法政府这个大旗，所以不少武装都投到防卫军旗下，比如钦兰军、克钦防卫军、克伦尼军等等，他们可没签《海埂协议》，和缅甸军政府处于战争状态。

稍微有点军事常识，就知道不能两线作战，而敏昂莱呢？竟然重开战火。后果大家都知道了，缅北局势糜烂，同盟军攻占腊戌，德昂军偷袭曼德勒，而人民防卫军更狠，在伊洛瓦底三角洲四处开花，核心区也不安全了，各路地方武装看到缅军的颓势，也纷纷揭竿而起。

截至目前，缅甸军政府实控的缅甸领土不足总面积的50%，敏昂莱政权有崩溃的危险。

第二是美国的黑手已经伸进了缅甸。民盟的出身，就是西方国家扶持起来的反政府势力，而民盟被军政府政变赶下台后，民盟更是受到西方国家支持。某种意义上说，民盟的人民防卫军PDF，本身就是美国支持的克钦独立军帮忙训练和组建的。

克钦独立军的宗教来自美国，文字来自美国，武器弹药也来自美国，是美国安插在中国边境的一个钉子。

而就在8月16日，美国国务院发表声明称，美国两名高级官员和缅甸“民族团结政府”（NUG，被政变推翻的民盟建立的）及数个地方武装举行了远程视频会议，明确表示支持。

同时，联合国也派了一个“联合国缅甸问题特使”访问中国，这个特使是谁呢？就是澳大利亚前外交部长朱莉·毕晓普，她一直是个对华强硬派，来中国大有兴师问罪的架势。

如果缅北问题不能快速解决，就有国际化的可能，引来他国干涉。

显然，美国的意图是以NUG为大旗，以克钦独立军为骨干，整合各个反政府力量，在国际上再拉起一个类似支援乌克兰那样的小圈子，各方力量聚集起来一起推翻敏昂莱。

能推翻敏昂莱固然好，我们对敏昂莱也没什么特殊交情，但怕就怕扶持一个反华政权上台，中国和缅甸签的一系列协议恐怕都要作废了。

第三是随着局势的糜烂，难民潮、弹药跨境误击等风险将急剧上升，对中国的社会稳定与民众安全构成了直接威胁。针对这些新变化，中国的应对策略也做了调整，避免缅甸局势恶化下去。

首先就是8月14日，王毅访问缅甸，会见了敏昂莱。在事后的通稿中，王毅提了“三个坚定”和“四个坚决”。

“三个坚定”第一条，就是坚定支持缅甸维护领土完整。

那么也就意味着，中国同意，所有民地武分裂缅甸的行为，都是叛乱。至于“四个坚决”就更有意思了，每一条都意有所指。

坚决反对缅甸生乱生战？那谁生乱生战，谁就是中国反对的对象。

坚决反对外部势力对缅干涉渗透？谁干涉渗透？自然是美国。

坚决反对任何破坏缅甸稳定和发展？这话显然是说给民盟听了，谁让你跟美国眉来眼去呢？

最后一个就更有意思了，坚决反对任何企图离间中缅关系、抹黑污蔑中国的言行。

这话是从何说起呢？

咱们回忆一下，去年同盟军1027行动之后，中国的互联网上，是不是出现了很多“同盟军在森林里捡到了中国武器装备”的传言？给人的印象，就是中国武装了同盟军，是同盟军幕后操纵者。

虽然这种传言听起来很“嗨”，但在缅甸军政府耳朵里，是什么感受？你中国不是不干涉他国内政吗？怎么说一套做一套？

现在的情况是，缅甸亲军政府的势力，认为中国在支持缅甸分裂武装，而缅甸支持NUG的势力，认为中国劝和促谈是在帮缅甸军政府续命，也在骂中国，中国两边不是人。

所以这种传言，就是“离间中缅关系、抹黑污蔑中国”，必须反对。

而王毅访问缅甸之后，我们会发现，在中国自媒体平台的“平安果敢”“果敢资讯网”等同盟军官方账号，以及缅甸阿堂、小白龙、朱玉生等宣传同盟军的个人账号，也都被封了。

这种封禁行动涉及微博、微信、头条、抖音等多平台，肯定不是平台自己封的，而是主管部门下了通知，几乎也就在封禁的同时，中国关闭了中缅边境的清水河、南伞杨龙寨、棒塞、金三、芒卡、南坎等多个口岸，同时，再次对果敢老街断电。

要知道，同盟军的武器弹药虽然不是中国提供，但其粮食、商品、电力却是非常依赖中国民用物资贸易的，口岸一封，电一断，同盟军就过不下去。

有意思的是，不仅中国和果敢接壤的南伞杨龙寨口岸被关了，就连和佤邦接壤的芒卡南邓口岸也关了。

为啥要关佤邦的口岸？因为同盟军某种意义上就是佤邦的白手套，同盟军打下的很多地盘，都移交给了佤联军，就算中国封锁果敢，同盟军也能从佤邦获取物资。而中国关闭连接佤邦的口岸也是在警告佤邦：别以为我不知道你和同盟军那些破事。

光封锁力度还不够，南部战区搞了一次“例行演习”。演习的地区很有意思，瑞丽和镇康。

瑞丽对面啥地方？就是最近同盟军要围攻的木姐。

很多人不知道的是，木姐，是目前硕果仅存的中国和缅甸军政府联络的通道了，其他接壤的口岸全部被缅甸民地武占领。如果木姐再丢了，中国与缅甸军政府最后的陆上通道就断绝了。所以，木姐必须控制在缅甸军政府的手中。

解放军在对面演习，就是警告对面的各方势力，不论是同盟军，还是德昂军，或者若开军，绝对不能进攻木姐！

演习的内容则更有意思，官方说法是“侦察预警、立体封控、火力打击”，如果说“侦察预警”和“立体封控”，出发点还是防止难民潮的话，那么“火力打击”，就是赤裸裸的威慑了。

很多人有点奇怪，果敢同盟军不是帮助中国打击电诈的吗？怎么突然不支持了？

其实，我们必须真切、客观地认识到一点：中国从来没有支持过同盟军。

在中国的互联网上，很多人因为同盟军的华人血统而对同盟军心有好感（甚至不乏越境去参加同盟军的）。

同盟军是华人武装不假，但并不意味着，华人武装就一定亲华，中国就一定要支持华人武装。

现在同盟军的旗号是“打击电诈”，似乎是一支正义之师，但很多人似乎忘了，早些年在果敢种鸦片的是谁？骗中国人去赌博，钱输光了就关水牢，嘎腰子的是谁？就是同盟军的控制者彭家啊。

当年中国云南外事部门多次劝告彭家远离灰产，但他们为了凑够养兵的钱，仍然不放弃灰产。当他们用毒品毒害国人的时候，将中国人关进水牢的时候，曾顾念过同族情谊？

至于去年同盟军的1027打击电诈行动，也不过是看到了中国打击电诈的大形势，借势拉起一面大旗而已。只不过，同盟军的行动客观上有利于打击电诈，中国才默认果敢变天，对同盟军水军在网上散布“中国支持同盟军”的说法也没有公开澄清。

如果同盟军拿下老街后，彻底根除灰产，老老实实搞橡胶、甘蔗等热带经济，发展边境贸易，没准中国还会正常对待同盟军。但现在呢？同盟军竟然把腊戌打下来了，甚至在NUG的鼓动下还有南下打曼德勒的趋势（NUG曾经许诺打下内比都，让彭德仁当缅甸副总统），这显然是捞过界了。

你同盟军说要光复果敢，中国默认了，但你现在拿下腊戌，没有法理依据啊！这地方从1948年缅甸独立开始，就不是你的地盘啊！

更关键的在于，腊戌是什么地方？是北掸邦的首府！是缅甸东北军区司令部所在地！任何一个政府，不论是民选政府还是军政府，都无法容忍这里丢掉的。同盟军拿下这里，就是和缅甸军政府不死不休！

只要腊戍在同盟军手中，双方和谈的基础就不存在，中方连斡旋都没法斡旋。中国对缅甸的政策是一贯的，不能影响西南出海通道，也不能影响一带一路，更不能反华。

如果缅北战火绵延，大大小小几十个势力，这些战略还怎么实施？

最简单的道理，中国建设了皎漂港，要修铁路往国内运吧？修铁路是找缅甸军政府一个势力谈容易呢？还是找一路上乱七八糟的几十个势力谈容易呢？

所以，出于中国自己的利益考虑，就必须维持缅甸现有军政府的稳定，这不是中国喜欢敏昂莱的军政府，而是需要他把协议一点一点地落实了（特别是皎漂港）。

当然，为了拿捏敏昂莱，中国也要手握筹码，筹码就是佤邦、勐拉这种还算是听话的民地武（虽然很多时候是阳奉阴违），让他们和缅甸军政府形成一种平衡。

无论是缅甸军政府灭掉民地武，还是民地武坐大，都是不符合中国利益的，现在同盟军捞过界了，中国就要向同盟军施压，让它回到原来的地盘上去。

从目前来看，中国希望双方回到海埂协议的框架上来，具体愿景为：

- 1. 腊戌交还给缅军，同盟军控制果敢全境和滚弄（滚弄是果敢门户，把这里给同盟军，能让同盟军安心）等地区。
- 2. 缅甸政府承认同盟军对果敢老街的统治，参考佤邦模式，重新给予一特区的编制。作为对应，同盟军承认敏昂莱政府，不再和民盟眉来眼去。
- 3. 果敢地区象征国家主权的国旗、身份证、车辆牌照、护照、货币等仍然保持与缅甸一致，但除海关之外，其他的官吏、警察、司法和军队，都由同盟军安排。

如果双方能达成协议，那么对各方都有好处。

同盟军这边，有了一个相对稳定的发展环境，能够好好修复战争创伤，消化刚刚占据的地盘，稳定自己的统治。

而缅甸军政府这边，收复腊戌面子上能好看很多，而且还能收缩战线，把有限的兵力抽调去对付人民防卫军，稳定军政府统治。

中国，也不用担心中缅石油管道的关键节点的腊戍，再陷入战火了。

如果相关和平协议能够达成，而且敏昂莱能全力配合中国迅速搞好皎漂港和配套的中缅铁路的话，那么缅甸拿到足够的军火，比如精确制导炸弹、温压弹、燃料空气炸弹等，让敏昂莱迅速平定人民防卫军，也不是难事。

由此可以看出，中国对待缅甸局势，立场一致是很清晰的，那就是中国利益为先，而不是感情为先。但问题也很明显，除了同盟军未必听话外，敏昂莱也不是省油的灯，毕竟他撕毁协议的事情，可不是一次两次了，所以，要达成并执行协议的难度，还是很大的。
