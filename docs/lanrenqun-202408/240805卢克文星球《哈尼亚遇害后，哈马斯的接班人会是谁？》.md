

# 哈尼亚遇害后，哈马斯的接班人会是谁？

最高领导人哈尼亚在德黑兰遇害后，哈马斯面临的首要问题，是选择谁作为新的接班人。

虽说哈马斯的最高领导人，实权并没有想象中那么大，而是更多地履行外交官的职责，但再怎么小也是名义上的最高领导人，重要性不可低估。

那么问题来了，哈马斯有哪些合适人选呢？

目前来看，至少有四个人，分别是叶海亚·辛瓦尔、哈利勒·哈亚、阿布·马尔祖克、哈立德·马沙尔。

叶海亚·辛瓦尔是哈马斯的二号人物，哈尼亚死后，二号人物直升一号，理论上是最合理的，但哈马斯内部的情况比较复杂，不能这样简单粗暴地处理。

哈马斯有个不成文的规矩，一号人物一般由海外派担任，也就是在加沙之外的人，二号人物则由留在加沙的人担任。

之所以这样，是因为加沙地方实在太小了，为了分散风险，避免被一锅端，哈马斯在许多国家都设立了办公室和分支，包括卡塔尔、伊朗、土耳其、叙利亚、黎巴嫩等等。

这样的好处，是以色列无法消灭他们。

坏处就是，久而久之，在海外那帮人（海外派），和留在加沙的另一批人（本土派），必然会产生矛盾，就像国内，很多喝过洋墨水的人，通常瞧不起乡下泥腿子一样。

为了平衡两边的利益，缓和矛盾，共同抗以，双方就形成了默契：一号人物的位置给海外派，二号人物的位子则留在本土派。

我在别的文章里多次提到过，哈马斯的一号人物是名义上的领袖，二号人物才是实际上权力最大的人。

由于一号人物实权不大，主要实权在二号人物手上，这样安排的结果，是海外派得到了面子，本土派保留了里子，两边都能接受。

在哈尼亚遇害前，结构就是这样的，一号人物的哈尼亚是海外派，二号人物叶海亚·辛瓦尔是本土派，如果贸然打破这一格局，存在引发内部矛盾的可能。

## 第二个人——阿布·马尔祖克。

因此，辛瓦尔并不适合升任一号人物。他本人也是个有大局观的人，大概率不会去争。

90 年代时，哈马斯创立没多久，便遭遇了第一次严重的打击，亚辛、兰提西等创始人被捕，组织运作几乎瘫痪。

得亏马尔祖克及时从海外回到加沙，领导实施了组织历史上的关键改革，重新焕发了组织的活力，曾挽救组织于危难中的经历，让他在内部很有威望。

1992 年，哈马斯首次设立政治局时，马尔祖克就是第一任政治局局长。

同时，马尔祖克和各方、尤其是中俄的关系不错，《北京宣言》就是马尔祖克代表哈马斯，来北京签署的。

比较神奇的是，马尔祖克和美国的关系也不算坏，青少年时期，他曾经在美国呆过14年，在科罗拉多州立大学和路易斯安那理工大学，均获得了学位。

也许是出于这个原因，美国觉得马尔祖克能够被他们影响，直白点说，90年代时，克林顿政府相信，可以通过马尔祖克来影响哈马斯。

于是当以色列追杀马尔祖克时，克林顿还保过马尔祖克一段时间。意识形态上，马尔祖克属于强硬派，坚决不承认以色列，由于常年在海外的经历，他同时也是海外派。

不过马尔祖克的缺点明显，年龄比较大，74 岁了。

同时，他和伊朗、卡塔尔、土耳其的关系不够紧密，而这三个国家恰恰是哈马斯的大金主，尤其是前两者，没有前两者的支持，很难上位。

这里简单说一下哈马斯和三个金主的关系。

卡塔尔主要给哈马斯提供资金支持，伊朗更多是给哈马斯提供技术和人员培训方面的支持，至于土耳其，则是资金和技术兼有。

三个金主里，论谁对哈马斯的影响力更大，那无疑是卡塔尔，然后到伊朗，最后是土耳其。

以哈尼亚为例，他就是常年呆在卡塔尔的首都多哈，而不是伊朗的首都德黑兰，或者土耳其首都安卡拉。

小小的卡塔尔能有大大的影响力，关键就是它身为资源土豪，舍得砸钱，每年会打给哈马斯几亿美元，这是伊朗和土耳其力所不及的。

如果卡塔尔不点头，哈马斯新领导人很难上任。

某种程度上说，如果马尔祖克上位，等于中俄挤掉了卡塔尔和伊朗的影响力，他们不会乐意，我们也大概率不会这样做，因为没有必要在这个时候内讧。

排除了辛瓦尔和马尔祖克后，那就只剩下哈利勒·哈亚和哈立德·马沙尔。

先说哈立德·马沙尔。

他在创始人亚辛、兰提西死后，一直担任哈马斯的领导人，直到2017年。

他领导了哈马斯的第二次改革，也就是进行世俗化，减少宗教化倾向，由于改革的成功，2005年大选，哈马斯击败法塔赫，获得了多数席位。

当然，由于西方不承认，挑唆法塔赫搞事，最后双方内战，巴勒斯坦事实上分裂，那些席位也就没有意义了。

在 2011 年叙利亚内战爆发后，卡塔尔要求哈马斯站在叛军一边，推翻阿萨德政府，哈马斯起初试图两头不得罪，但在摇摆了一段时间，最终倒向了叛军。

不仅在嘴上支持叛军，还身体力行，派出了 1500 人参加战斗，这一切令伊朗非常不满。而马沙尔作为当时哈马斯的一号人物，在倒向叛军的过程中发挥了很大的作用。

所以，伊朗并不待见他。

但马沙尔的优点也明显，除了和卡塔尔土耳其关系好，还有大局观。

2017 年，金主卡塔尔意识到叛军无法获胜，便着手修复与伊朗的关系，金主都转向了，哈马斯自然也要转向，和伊朗恢复关系。

马沙尔知道，如果自己继续呆在位子上，与伊朗就难以修复关系，于是他主动让贤，辞去了政治局主席的职务，哈尼亚才得以接班（当然，伊朗肯定有施压）。

最后是哈利勒·哈亚。

关于哈利勒·哈亚的资料还很少，只知道他是哈马斯政治局的副主席。能做到这个位子的，必然是在内部呆了很久的。

虽然不像马沙尔那样担任过一号人物，但也是高级成员之一，威望和资历的问题不算大。

这半年多来哈马斯与以色列的谈判，是他直接负责的，他和伊朗的关系相当不错，去过德黑兰N次，反倒是和卡塔尔、土耳其两国的关系，只能算一般般。

哈尼亚在任时，和三个金主国家都处得不错，而目前的人选，没有一个能达到这个高度。

从伊朗的角度，肯定更希望哈利勒·哈亚上任一号人物，而卡塔尔倾向于马沙尔。

如果两国能谈好，大概率是哈亚和马沙尔两人里二选一，如果不能，也不排除第三种可能性，扶持一个和双方关系都不算紧密的、且相对弱势的人上位。

但无论是谁，有一点是可以确定的，那就是上台的人必然是对以色列强硬派，哈马斯高层根本就没有妥协派。鲜为人知的是，哈尼亚是哈马斯高层里少有的温和派，因为他愿意承认以色列的存在，承认两国方案，而其他人普遍不愿意。杀死温和派的结果，就是得到强硬派。这就叫自作自受。
