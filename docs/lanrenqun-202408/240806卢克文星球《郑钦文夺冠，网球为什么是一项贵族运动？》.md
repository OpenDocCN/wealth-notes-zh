

# 郑钦文夺冠，网球为什么是一项贵族运动？

8月4日凌晨结束的巴黎奥运会网球女子单打决赛，中国选手郑钦文战胜了克罗地亚选手维基奇，获得冠军，创造了中国网球的新历史。

这一枚金牌分量很重，哪怕传奇如李娜，也没拿过奥运会金牌。

因为网球超强的商业价值，毫无疑问郑钦文会成为下一阶段中国最耀眼的体育明星，也会大大促进中国的网球运动。但是，很多人却认为，中国不具备全民普及网球的能力。

为啥？因为打网球太贵了，是一个贵族运动。

为什么说网球是贵族运动？因为和诞生于街头巷尾的足球不同，网球还真的诞生于贵族阶层。

网球最初起源于法国宫廷，一度被视为特权阶层权力的象征，参加网球运动的只有贵族阶层以及宫廷内部人员等权力阶层。为了保持网球运动的贵族血统，法国国王路易明令禁止普通平民参与网球运动，并规定这是宫廷中的特权游戏。

所以，网球运动从创始之初就被贴上了特权阶层的标签，代表了特权阶层的权力。

后来，网球传入英国并流行于贵族圈内，并把绳帘改用为小方格网子，拍子改用穿线的网拍，由此诞生了现代网球。1877年，英国举办了全英草地网球男子单打锦标赛，即后来闻名于世的温布尔登网球赛。

虽然现代网球已经不只是贵族的专属，但从网球的球具、衣着、判决、技巧、礼仪等等规矩来看，仍然透露着浓厚的“贵族范”，最大程度地保留了贵族般的优雅。

而从运动成本来说，打网球，无论当职业球员，还是随便玩玩，成本也远高于其他运动。

我们先算一下，如果是随便玩玩，要花多少钱。

先说装备，网球装备里包含的种类太多了：球拍，球线，训练球，球包，护具，球衣，球鞋等等。咱们以城市中等收入家庭能接受的标准来算，球拍800元；球线70元；网球上衣和网球短裤共240元；网球鞋400元；护腕40元；低端吸汗带、避震器共20元；普通网球包150元，总计1720元。

这是固定支出，还不算球鞋、球袜、球拍手胶等等耗材的费用。再看场地，场地是个尴尬的问题，最大的难题就是公共场地不足。

中国为啥乒乓球和羽毛球很普及？因为乒乓球，支个水泥台就能打；羽毛球，一块平地就能打。而网球场呢？建一个网球场相当于2个羽毛球场或者篮球场，从决策者角度考虑，建一个这么大的网球场，还不如多建几个乒乓球台子划算。

就算建了网球场，信不信你根本抢不着？晚上就是广场舞大妈的乐园。

所以，要想好好打球，就必须进付费场馆。一般的场馆在120元-200元每小时，一周打上六个小时不多吧？就是1000元左右。

更不要说网球的门槛比较高，没有教练带着打基本就是瞎玩，教练每小时也要一两百块。如果你还要发球机+球童（没有球童的话，大部分时间都用来捡球了，体验很差），那还要再加200元。

此外还有通勤、停车、吃饭、能量棒等等开支，光打球，一个月就要花进4000块进去。

说实话，现在的城市居民，这么有钱有闲的，能有多少人？所以网球基本多见于大学，平时在社会上根本见不到。

## 那么走职业球员路线呢？

和很多体育运动一样，职业网球运动员是童子功，在6岁之前，就要判断是不是具备一定天赋。郑钦文的父亲是一名田径运动员，所以郑钦文从小就不爱抱，只爱跑，后来2008年郑钦文和父亲去看奥运会，幼年的郑钦文被网球吸引，她的父亲才决定让她走这条路。

于是，花钱之路就此开始。

先是请教练，网球看起来简单，实际上入门很难，而入门指导教练非常重要。一般的教练肯定入不了郑钦文父亲的法眼，所以必须用资深教练。按照每小时250元（这个价格很低了），一天练两个小时，一周就是2500元，一年按50周算，就要12.5万，三年就是37.5万。

郑钦文10岁之后，已经展现出了超乎同龄人的网球天赋，那就必须请专业的网球教练了。所以父母把她送到武汉，聘请了李娜的启蒙教练夏溪瑶为其教导，后来又给李娜前教练余丽桥跪下，求余丽桥收下这个弟子。

这俩人，都是中国青少年网球培训的大神级人物，价格自然是很贵的，有媒体报道光教练费用，一年就要50万。

2014年湖北省第十四届运动会，12岁的郑钦文拿下2枚金牌，进入北京匠心之轮网球学校，师从李娜的阿根廷籍教练卡洛斯，教练费用两年100万。

14岁之后，郑钦文又只身一人远赴欧洲及美国学习，并进入世界上最大的网球寄宿学校——IMG学院学习。IMG学院前身是由传奇网球教练尼克·波力泰里尼创办的尼克网球学校，网球名将大威廉姆斯、小威廉姆斯、扬科维奇都曾在这里受训。

IMG学院的学费大概8万美元，算上日常开销，起码10万美元。这还不够，要想提高成绩，还要找名将教练“喂招”，一年没有5万美元下不来。

虽然在IMG可以参加ITF青少年比赛赢奖金了，但ITF的奖金较少，相反外出比赛却很多，机票、住宿、注册等等费用下来一年也要几万美元。这样林林总总算起来，郑钦文在美国4年，要花五六百万人民币。

所以在2018年郑钦文夺得U18冠军之前，这些钱基本都是纯投入。

等好不容易郑钦文开始拿奖金了，也要开始组建自己专业的训练和保障团队了，相关的花费也水涨船高。

作为一个球星，体能师、医生、营养师、陪练、康复师、公关都不可缺少，整个团队的工资和开销都需要郑钦文负担，包括机票食宿。

养这些人员，每年最少都要200万，更别说私人教练了，先是佩雷·里巴，然后是比利时人费塞特，这俩人一年没有50万美元是下不来的，光靠奖金可不够。

要说郑钦文的家庭，也算是殷实之家。她父亲曾经是运动员，退役后创办了一家名为“十堰鑫胄科工贸”的公司，主要经营汽车、钢材、化工产品、建材的销售，收入不菲。但是面对出国训练的高昂花费，仍然入不敷出。郑钦文曾回忆说：“我还记得在我十四五岁（正是在美国期间）的时候，我爸爸甚至为了让我打球，想要卖掉房子，因为他看出我很有潜力，所以想要倾尽所有来帮我实现网球梦想。”

最后，还是郑钦文的父亲从湖北省队拉来了点钱，又以承诺签约IMG为代价（IMG不仅仅是学校，还是体育经纪公司，谷爱凌签的就是这里），要过来一点钱，才勉强渡过难关。

这样算下来，外界传言的郑钦文为了走网球这条路花了2000万可能还说少了，这对普通中国家庭来说，完全是一个天文数字。

更关键的在于，现在郑钦文成功了，广告代言都接得手软，已经登上收入前十的女运动员榜单，以前那些投入，现在连本带利都能收回来。但问题在于，在押注投入的时候，谁知道最后会不会成功？如果拿不到成绩，前面花的钱，基本都打水漂了。

这是一个豪赌，除了郑钦文的父亲这样的，恐怕真没几个中国家长敢下这种血本。

这种情况不仅发生在中国，纵观世界，打网球的球星，也大都非富即贵，比如女单八分之一决赛中被郑钦文击败的美国选手艾玛·纳瓦罗，就是个超级富二代，她爸爸拥有一家金融服务公司，是亿万富翁，所以才有钱去学打网球。

说白了，西方国家就是故意抬高网球的门槛，提升培养职业球员的成本，把后发国家拒之门外，然后一方面彰显白人的优越感，另一方面还可以彻底垄断这一极具商业价值的体育产业。

### 网球贵族化，对中国是好事还是坏事？

简单来说，对拿金牌来说，是好事。

但对建设体育强国来说，没有什么太大帮助，在现在的经济形势下，绝大多数普通家庭在核算成本后，都会放弃网球，哪怕孩子再有天赋也不行。

为啥？因为中国体育体制改革了。

咱们都知道，中国的体育体制，曾经是“举国体制”为主流。

“举国体制”是从原苏联学过来的。

原苏联就是由政府选拔和培养有天赋的运动员参加比赛，这样做的最大的好处就在于可以集中力量，使优秀的教练员都有统一的观念和训练计划。在这种制度下，运动员守纪律，训练刻苦，求胜欲望非常强烈，这在奥运比赛中是拿金牌的关键。

所以，一代一代孩子从小就被送到体校，接受严格的训练，然后进入省队，再进入国家队，然后在各省尖子的内卷中诞生奥运冠军。

实事求是地说，举国体制在过去，帮中国拿下了很多金牌，功不可没。但问题在于，这种体制也有缺点，那就是太贵了。

每次奥运会，中国差不多能拿三四十块金牌，但这三四十块金牌的背后是什么？是国家体育总局下属群众体育司、竞技体育司、体育经济司等11个司局级单位，还有体育科研所等事业单位，数十个国家队，数百个省队，上千所体校，3000多个教育体育局，以及围绕这个庞大链条形成的各种协会等等。

这个链条上的人，大部分都是“有编”的，这是多么大一笔财政支出？

曾经有一篇热文《奥运金牌的陷阱》上过各大门户网站，文章算了一笔账，雅典奥运会备战4年，中国在体育事业上花费了200亿元，而中国队在雅典奥运会拿了32枚金牌，平均下来每枚金牌成本就差不多是7亿元，“可谓世界上最昂贵的金牌”。

这是什么概念？中国搞嫦娥计划，从一期工程干到六期工程，花费也就100亿多点。

虽然这个算法并不十分科学（因为有相当一部分钱花到了全民健身上去），但如此巨大的付出，仍然让社会反应激烈。

最终，国家进行了一次全面、深入的体育体制改革。

改革分为两层，第一层是那些观赏性不强、商业价值不足，但又是夺金热门的项目，中国仍然坚持体校路线，一级一级选拔苗子，进行重点培养。

比如最近大放异彩的跳水冠军全红婵，她就是在小时候玩跳格子游戏时被教练发现了天赋，于是从7岁就开始接受系统性的跳水训练，先是在市体校，然后是省跳水队，最后再进入国家队，一路走下来成为世界冠军。

另一层呢？就是那些观赏性强、商业价值大的项目，直接推向市场，让他们找资本要钱，国家不再负担培养费用，所以体校大量裁撤这种商业项目，转由俱乐部或者专业学校培训。

比如足球，搞了个“大国家队”，除了原来的男足国家队、U23国足之外，还有U21国青队、U19国青队、U16、U15、U14国少队等等。

那么国少队的球员怎么选拔呢？主要从足球学校（也有俱乐部）中选拔，这样搞，据说是为了扩大培养面，吸纳更多的人才，也省去了前期的培养费用。

但问题在于，这也就意味着，原本在体校中由国家承担的训练成本，都要靠个人和家庭来承担了。不过不用担心，中国想让孩子当球星的殷实家庭，比比皆是。

这倒是个体育发展的新路径，举国体制的归举国体制，市场的归市场，可以有效解决过去中国金牌种类单一的问题。

你看，中国夺金优势项目，都是举重、体操、跳水、乒乓球、羽毛球、射击、女子柔道等等，每次中国奥运会金牌的80%都是这些项目，因为这些项目更讲究技巧，可以通过体校集训来快速提升成绩。但是在像高尔夫、马术、网球等“贵族运动”，中国队与世界强队间差距巨大。

如果国家花巨资投入那些有较高门槛的“贵族运动”，不仅不一定能确保出金牌，而且就算拿了金牌，这金牌的性价比也太低，反而会挤占原本就不富裕的其他夺金项目资金。

所以，索性把培养交给市场，有钱的精英家庭，如果不想走学习内卷这条路，完全可以选择“贵族运动”这一路径嘛，不仅缓解了这些家庭的出路焦虑，而且还能时不时地带来个惊喜，给中国拿一枚意想不到的金牌。

这一路径，甚至还可以和“归化”相结合，只要是外国的优秀运动员，中国完全可以将其作为夺金的利器，不用投入，只收割成果，相当划算。

当然，这对那些孤注一掷的家庭来说，也是双赢，夺金后带来的巨大商业价值，就是这些家庭前期投入的最大犒赏。

另一条道路也不能偏废，中国仍然有千千万万个不富裕的家庭，这些家庭渴望着通过练体育实现阶层跨越，只有保持传统的优势项目，体育好苗子们才能源源不断挖掘出来。

所以，“两条腿走路”，才是新一阶段，中国体育新的战略方案。
