# 网红“铁头”被抓，原因为何？

近期，在互联网上靠“打假”红极一时的网红“铁头”，被杭州市公安局以涉嫌敲诈勒索罪依法立案侦查，他上一次如此吸引大家的目光，还是在6月份的靖国神社涂鸦事件。

根据通报，他涉嫌敲诈勒索罪，以爆“黑料”为威胁，向某带货主播索要数百克黄金，许诺交付后向其提供“保护”。这种漫天要价收“保护费”的行为，实际上是妥妥的黑社会作风。

早在今年1月，他就在直播间里主动承认自己曾经涉黄的经历，引发网友纷纷对其举报和抵制，其旗下的多个账号，也在广大网友们的正义执行下被封号，这次被爆出敲诈勒索，属实不稀奇。

## 01

网红铁头，原名董光明，生于1987年。2023年，开始涉足短视频领域。4月，董某开始在短视频平台发布视频，一开始，董某的视频账号并没有多少人关注，因为其短视频的内容大部分类似于帮助路人实现个人心愿和街头善意测试的题材，比如说假扮流浪汉去超市门口要食物、测试街头流浪汉倒地路人的反应等，这类题材没有让铁头真正红起来。

让他真正火起来的，还是2023年发布的一系列打假视频。2023年7月，铁头在三亚市友谊路的一家海鲜店里买海鲜，遭遇了“五两秤”，随即拍摄了视频进行打假。随后，三亚市执法部门证实了商家的确存在缺斤少两的行为，并责令该海鲜加工广场将涉事商户清退出场，罚款30万元。

这次“打假”让“铁头”一战成名，引发了网友的广泛热议和共鸣，他的视频账号“铁头惩恶扬善”在5天内涨粉200多万，一时间成为了全网舆论热议的核心。这件事也让他成为了全网新时代“咆哮式”打假的代表人物。

也许是成功让他有些飘飘然，8月4日至7日，他连发四段视频，公开举报杭州新东方“违规补课”，向杭州市拱墅区教育局递交了相关资料。铁头在视频中称，他要求新东方退还家长全部费用，并且希望拱墅区教育局立即取缔违规补课。

不出意外，新东方被罚了。从国家规定的角度来说，新东方违规补课是不对的，在国家力推“双减”的时代，从大了说，违规补课是顶风作案，从小了说，的确是在加速学生之间的“内卷”，让学生们增添了无形的压力。

然而铁头忽略了一个现实，就是中国大部分的家长都是有“望子成龙”的心愿的，所以这件事引发了家长圈内部的巨大争议，成为一些网友对其“粉转黑”的开端。当然，有争议才有流量，他也确实因此获得了更多的流量。

最让铁头“吸粉”，把他推上流量顶峰的，当属打假杭州理发店诱导充值事件，在24年年初，铁头连发多个视频，称杭州滨江区一个理发店诱导顾客充值，他在询问理发价格时，理发店称办卡可以享受超低折扣，并推销店内的其他业务，如果不办卡理发的价格就要在七八百元左右，后来董某称自己被诱导充值了7000元。

双方为此事在店门口发生了激烈争吵，铁头将这一过程拍摄了视频，并反馈给了警方，在警方人员来到现场后，此事才平息。这件事无疑再次拯救了铁头的风评，因为理发店的诱导消费行为实在太过恶劣，违法行为都被视频记录得一清二楚，铁头再一次赢得了网络舆论的叫好。

“铁头”发布的一系列“咆哮式打假”视频为他本人带来了百万级流量，大量支持他的网友认为，虽然他采取了许多不恰当的，甚至过激的方式进行“打假”，但他的行为客观上揭露了诱导充值、缺斤少两等一系列市场乱象，保护的最终是消费者。

只不过，铁头的辉煌期并不久，很快就翻车了。

## 02

资料显示，2014年4月20日，铁头因涉嫌犯开设赌场罪被刑事拘留；同年11月21日，因涉嫌非法拘禁罪被刑事拘留，2017年5月23日，其服刑期满出狱。

对于刑满释放人员来说，想要社会重新接纳他们，客观上存在一定难度，找个正经工作养家糊口，不是那么容易。

而在这个流量为王的时代，制造爆点，迎合看客需求，不仅来钱快，而且也会收获无数用来营销的名气，这种“名利双收”的事，自然被铁头盯上了。

过去涉嫌犯罪的经历，使得他压根不会在拍视频时注意所谓的“底线”，手段越激烈，情节越起伏，观众越爱看，整活越没有顾忌。2024年7月，网上曝出一条他“打假”的视频，视频显示，他闯进一家金店，指明要“一口价黄金”，甚至还大声指着店员的鼻子辱骂，当店员想去劝阻他时，他直接躺在地下倒地不起，做出一副要讹人的架势。

人只相信自己眼睛看到的东西，不明真相的群众肯定认为金店在仗势欺人，铁头树立的为民打假的“光辉形象”，就能够保持的住。

铁头背后的团队，也为他整活的事迹出了不少的力。许多创意并非出自铁头一人之手，他背后的团队“碰瓷”能力极强，堪称谁火就盯上谁。他背后团队商业嗅觉是十分灵敏，如果在一个比较火的带货商家能够打假成功的话，那么所引发的吸睛效应是十分明显的。那么铁头就可以再次把自己塑造成为打假成功、不畏权势的英雄。

哪有钱冲哪跑，谁火就咬谁，非常符合铁头和他的团队的调性。

2023年10月在董宇辉老师和他的东方甄选火起来之后，铁头就号称自己要打假东方甄选，结果引发了对方的警告，最后因为实在找不到什么可以被打假的黑料，只能不情愿地道歉。

当然，他的道歉并非是诚心的道歉，只是吃不到流量后的一种无奈之举，甚至我们会发现他在道歉视频之后还做出了一个紧握拳头的动作。试问如果真的是诚心诚意的道歉，怎么可能会有这样的表情和动作呢？

问题来了，为什么这次会翻车呢？很简单，铁头缺钱了。

打假这种事情是需要花钱的，拍摄视频、路费啊，包括各种的买卖行为都是要花钱的，铁头公开的数据显示，其一个月的开销就要十几万左右。到铁头被采取强制措施为止，他已经花了110多万了。

因此，铁头不得不想办法创收，在这次案件中，如果他能够成功勒索500克黄金，那么对应到黄金的价格，就是30多万。

而平台给他的流量钱，一共也就五六十万元，所以铁头自然想到了去干这种下三滥的事情。

铁头事件恰似一面镜子，社会大众对铁头惩治不公，伸张正义，为大家维权的行为抱有浓厚的兴趣，并且给予了支持。这反映出我们这个社会对于公平公正的热烈渴望。

然而作为公众人物，尤其是展现所谓“正义”形象的公众人物，自己首先就要做到以身作则，否则“屠龙者”，很容易变成恶龙。
