

# 让别人满意有什么不对么

先说个题外话，最近谷歌创始人在斯坦福的演讲实录刷屏了。根据一些解读，据说他原本以为是一次未公开分享，所以讲的比较奔放，但后来没想到内容泄露出来了，那么这里的内容就会呈现出一些违和感。

也就是，讲的很真实，但是很多地方，不太符合所谓“表达正确”。这里也提醒一下，所有所谓的大佬公开分享，无论东西方，基本都是建立在“表达正确”的前提下。在这样的环境里，你几乎听不到太多“真实”的观点和见解。

我线下分享会的时候，一直要求不录音，不视频，不公开。很多人会误以为我有什么大料要爆，或者误以为我会讲什么不和谐的言论，其实不是。只是公开场合，有太多所谓“正确”的束缚，很多言论就只能照顾公众的情绪，特别是那些敏感又脆弱的群体，而不敢讲的太真实。但只有私下，不公开的场合，才敢说一些不那么正确，但更真实的内容。

讲这个题外话，就是想说，今天想讲一点真实的认知，但是说出来就不那么正确，所以，付费吧。

我们身边有很多好人。所谓好人，可以说是有责任心的人，他们愿意也敢于承担责任，做个让别人满意的人，这里包括让上司满意，让伴侣满意，让伙伴满意。当然，这通常也是我们认为属于优秀的典范。我并不是否决这一点，但今天我说点真心话，做到让对方八分满意，很好了。该交付的交付，该兑现的兑现，很优秀了。不要去追求别人对你的十分满意，追求太多，反而是错的！我这里有几个前提希望读者理解一下。

- 第一，你做任何决策的时候，请不要假设你的上司/伴侣/伙伴/同事一定是好人，请不要把这个作为决策的前提假设。
- 第二，即便他们真的是好人，至少概率上是存在的，请不要假设他们所认为的对你好，真的就是对你好。

先理解这个前提，其实这个文章的意思基本上就明白了。

我举一些例子。比如一个技术大牛，在职场承担很关键的角色，很受领导器重。那么领导会说：“你好好干活，好好做事情，公司不会亏待你，不要天天微博上扯东扯西的，浪费时间，对工作没帮助。不要总是去参与那些技术峰会，咱们技术遥遥领先，他们都是来套你方案的，别搭理那些人，瞎耽误功夫；不要总是和竞争对手公司的那些同行天天勾搭，我知道你们是老同学，老同事，以前关系不错，他们现在就是利用你，从你这里偷情报。我当然相信你人品，但你还年轻，太单纯，很可能就不注意被他们利用了。相信我，我可是为你好，不想让你被人利用。”

听上去领导非常推心置腹有没有。

其实潜台词是什么？你的能力，你的价值，最好只有我知道，只有我能给你开出匹配的薪酬。在别人眼里，最好你就是个 nobody，这样我的业务线才安全，你才能安心给我干活。对吧，让员工不要图虚名，不要无谓社交，这不都是老板们爱说的话。

伴侣关系也是如此。如果一方为了另一方付出太多，为了让对方满意而放弃自己的社交、职场前途，那么一旦双方感情发生问题，将无路可退。我讲过一切皆博弈。顺便说一下，请不要讲资本家剥削，忘掉“剥削”这个词，换成“博弈”这个词来理解劳资关系。

那么回到这个话题，就是，在很多场合，你和对方存在博弈关系，你必须保留自己的筹码。而从对方角度，你没有筹码才是他最满意的状态。能理解这个底层逻辑么。永远不要用彻底牺牲和放弃自己竞争力为代价来无限满足对方的诉求。

这是第一条。你的影响力，你的社交资源，你的信用，你的学习态度，都是你的竞争力。不要为了让对方满意，轻易舍弃。别人可能希望你完全符合这个体系的诉求，但是你必须明白，这个体系可能能不能保障你的一生。这时候，你必须要有备选方案，有备选资源。这是你对自己的基本保障，不能为了满足对方的诉求而轻易舍弃。

然后是讲讲人性。你越是去不断的放低自己满足对方，对方越不会把你当回事，会认为你是很容易妥协、很容易屈服的目标。在一些不容易平衡的利益分配场合里，人家会把牺牲你的利益当作是最容易做的决策。这也是我之前说过的，其实全世界都是在欺负老实人。过度满足对方，会失去对方应有的尊重，最终发现你永远满足不了，只能成为耗材和牺牲品。

回到前面，我不是说有责任感不好，不是说满足别人不对。员工满足老板的业务诉求，丈夫满足妻子的生活诉求，都是理所当然的。但是，这里存在一个合理区间。所谓合理区间就是：不以牺牲和放弃自己的竞争力为代价；不以丧失基本的尊严和平等为代价。在职场，在社交场合，在两性关系中，均是如此。
