

# 宁波港口发生爆炸，是怎么回事？

240812

2024年8月9日13时50分左右，靠泊北二集司的阳明船公司的集装箱船“动明”号上发生燃爆，初步判定，爆炸的是过境的危险品集装箱，目前暂无人员伤亡。

注意关键字：“靠泊”和“过境”。

靠泊就是船停在码头上装卸货物，集装箱船靠泊时，除了等码头吊机装卸货物，人员可能上岸休息以外，还有个步骤就是“换电”。

船舶在海上航行时，电力是由主机提供的，而靠泊港口时，主机一般停机，改为使用码头提供的岸上电力供电，这样可以大幅度减少船舶在港区的大气污染物排放，有效的改善港区大气环境。

而过境是指过境的集装箱。根据阳明公司（湾湾的航运公司）资料，发生爆炸事故的“动明”号，装载能力是6589个标准集装箱，在挂靠宁波港的前一站是上海外高桥码头，最后会驶往中东港口。

> 沿途会挂靠许多港口并装卸集装箱，发生爆炸的集装箱既不是在宁波装的，也不会在宁波卸下，「只是装在船上路过宁波港而已」。靠泊期间爆炸的集装箱，就是安静地躺在船上，由“动明”轮实际控制并管理。

根据初步调查，事故的原因是货主瞒报。

目前行业内有一则消息称：爆炸货物为过氧化剂，过氧指的是两个氧原子连在一起的基团，例如过氧化氢（双氧水）就是H2O2，即 H-O-O-H。

过氧化剂属于五类危险品，高温下分解会释放氧气，碰撞，摩擦都容易爆炸，所以需要低温保存。过氧化剂海运的话一般是用专门的冷藏集装箱保持低温。冷藏集装箱需要插电来制冷，船上有配备插电功能，专门放置冷藏集装箱的位置。

如果正常插电制冷，是不会发生爆炸的。目前宁波的温度大概是三十五度，集装箱内温度则可能高到七八十度。这一点刚进入露天停放没开空调的轿车的人应该有经验。没插电无法制冷，保持低温的过氧化剂可能会爆炸。

按照管理规则，船舶装箱时需要进行危险品申报，装运危险品的船舶挂靠前需要向港口监管当局和海事部门提交船上所有货物的舱单信息，危险品要提前申报。

经手危险品申报的不止是船公司，是否瞒报不难查清。如果装运港和过境的宁波港监管当局接到过这个箱子的危险品申报，那说明货主没有瞒报。

无论是哪种原因，跟宁波港的关系都不大。宁波港知道船上有危险品，但也仅仅是知道。由于集装箱和集装箱里的货物是由船方实际控制，插不插电都是船方和船员的职责，没什么异常谁也不会去动它。

但宁波港和宁波港的用户是最大的受害者，“动明”轮停靠的北二集司是指宁波北仑第二集装箱有限公司，俗称三期码头，是宁波港集装箱的主力码头之一。爆炸发生后，三期码头就停止提箱了，所有码头操作都会停止，作业计划全部打乱。

同时停靠的几艘船舶可能无法按时离港，预计停靠的船舶要先在锚地等待，改挂靠其他码头。已经装了货物进入三期码头等待装船的重箱不知道有没有受损，也不知道什么时候能装船。所有的船都会延后，甚至影响下一个挂靠港口的计划。

不止三期码头，离三期码头几公里外，就是给镇海炼化配套的算山码头，用于停靠超过 20 万吨的超大型油轮(VLCC)，现在所有超大型油轮都无法挂靠，都在海上飘着等通知。

如果调查结果出来，阳明船公司的声明属实，货主确实瞒报了，那么就是货主的重大责任。不过不得不说的是，不管是危险品生产单位还是运输相关单位，对于危险品瞒报的感情是复杂的。

许多工业品都是危险品，只要我们走工业化的路，就躲不开危险品。公众印象中提到危险品就会出现两个字：严管。

但严管是需要成本的，不止是执法成本。还有危险品企业的经营成本。经营成本的直接体现就是各种证件，经营需要《危险化学品经营许可证》，运输需要《道路运输证》（加盖道路危险货物运输专用章）《槽罐质量检测合格证》，《剧毒化学品公路运输通行证》。危险品海运更是要危包证，危险品监装资格，海事申报，甚至车船交接等一大堆手续。

各种证件加持下，再加上日常操作中出现的各种事故导致的处罚，市场上合格的能提供危险品服务的企业就越来越少。

于是各种乱象就出现了：危险品运输企业顶着处罚运营，危险品仓库没位置，危险品不得不到更远的仓库装箱，还有危险品监装资质挂靠等等。至于什么高收费，排长队，都已经不算事了。

以锂电池海运和空运为例，船公司和航空公司要求包含锂电池的产品提供鉴定报告，鉴定报告可以由不同的鉴定机构出具。

然而在实际操作中，经常会出现上海只认A鉴定机构出具的报告，深圳只认B鉴定公司出具的报告。

如果某电池生产企业只做了A报告，那么只能在上海出。如果这个企业把产品卖给了深圳的某个工厂装进了某些产品里。那又只能运回上海出口。导致企业不得不在每家鉴定机构重复做鉴定报告。

更要命的是，由于执法难度，所有的严管措施都会施加到如实申报的企业上，而瞒报的企业，成本低得多，差不多就是无人监管。业内对于危险等级不高的危险品打擦边球的现象屡屡发生，这就会形成劣币驱逐良币的情况，加大出事的几率。

危险品行业的从业人员是最不想发生事故的，一旦发生事故，轻则丢饭碗受处罚，重则甚至坐牢丧命。谁都想高高兴兴上班来，安安全全回家去。就算事故跟自己无关，也会迎来安全大检查，这个时候一般都会告诉涉及危险品的客户：

能别出就别出了，等风头过了再说。

有必要打破单纯靠“严管”的思维，从培育产业的角度出发，在保证监管的基础上，降低企业的运营成本，加大到瞒报的监察。才能从根本上减少事故的发生（完全杜绝不现实），走出越严管，越出事的循环。
