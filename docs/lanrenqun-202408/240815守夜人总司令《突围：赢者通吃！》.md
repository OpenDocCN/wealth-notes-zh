

# 突围：赢者通吃！

有一件事情为大家所忽略却无比重要：随着产业的发展，准入门槛会越来越高。这就意味着，在超级主导者和下游附庸之间，中间角色将越来越难以保证自身的独立性。这一点不仅在国家之间是如此，连个体也会如此发展！为了更清晰的理解这一点，我将举例说明。

产业的发展在超级大国和附庸之间的中等型国家越来越难以保持自身在政治上的地缘影响力和经济上的独立性，甚至在军事上都会越来越缺乏独立性和威慑力。在我们建国之初，越南可以威慑和干涉中南半岛上的周边国家——但此时此刻就再也别想了！因为这些国家显然会找老大给它出头。柬埔寨一直被越南卡住海运入口的脖子，他自己没有资金和技术摆脱这种局面，也没有相应的军事力量改变这种地缘。但是，老大有钱有技术有力量，只要他愿意，老大也愿意，越南就毫无办法了！如此一来就只存在两个身份：超级老大和一群无差别的小弟。你如果没有办法成为超级老大，你就会沦为其他小弟中的一员！一旦你无法整合周边形成自己独立的体系，就只能选择加入到别人的体系，并扮演别人给你的角色。这个角色不仅仅是地缘角色，也是经济体系中的产业角色。而这两者是相互锁死的！

我们一下子难以理解欧洲为何会在短短的20年间被美国渗透和控制成这个样子。只要想一想欧洲最缺的东西就明白了！欧洲不仅仅缺主权的独立性，也缺产业的独立性。欧洲有自己独立的卫星系统吗？好像没有！不管是军用还是民用都是GPS提供的服务。欧洲有自己的数字产业么？有自己的新能源产业吗？似乎也没有！美国有特斯拉，有微软，有英特尔，有谷歌，有社交网络，有苹果和安卓。欧洲有什么？欧洲拥有优势的传统产业正在逐渐外迁，或者被新兴工业势力追赶上逐渐替代。起决定性作用的新兴产业和关键服务都高度的依赖美国！一直以来欧洲人用美国的互联网技术和社交媒体平台挣钱还需要向美国交税。哪怕是服务提供者的服务器就在欧洲而且主要用户也是欧洲人，也不例外。美国去年才跟欧洲谈那个数字产业的协议，如果是上面这种情况欧洲可以自己征15%的税，而不是全部交给美国。

中国的相关产业，虽然没有美国相关产业影响力那么大，但是都有对应的东西并自成体系。这是具备成为独立经济体的前提条件！有了这个基础，那些不愿意接入美国体系的反叛者才有另外一种选择。为了他们自己的独立性，他们会用自己的资源和影响力去帮你完善这个体系——帮你也是帮他自己！

如此一来，很多事情的发展轨迹其实就会越来越清晰。不管你愿不愿意，它都会逐渐像滚雪球一样形成一种必然。明白了这一点，就会明白欧洲在整个棋局中会扮演什么角色以及能够扮演什么角色！欧洲永远不可能作为一个整体去发挥它的独立作用。如果作为整体发挥作用，也是作为鹰酱的附庸来发挥作用，而不是作为独立的一极去发挥作用。所以，兔子对欧洲的策略是分化瓦解，团结其右翼和缺乏产业优势的成员。

之前我们抓住的是匈牙利，也试图与意大利和德法形成某种利益捆绑。当年意大利破产了，走投无路，选择加入一带一路。在这个过程中，意大利获得了回血的机会。然后在鹰酱的压力下合同到期后就退出了！这一次右翼上台，把欧盟好好的玩了一把：先在欧盟内部高调指责中国新能源汽车产能过剩配合冯德莱恩背后鹰酱势力的意图。欧盟27个国家有26个签署了限制中国新能源汽车的法案之后，她自己又跟冯德莱恩吵了起来。然后跑到北京，要求签订合作协议——在她的任期内，兔子向意大利投资建设新能源汽车的生产线并培训相关技术。

此时此刻，德法掐死她的心都有！在以前的文章中说过，法国搞一堆动作就是为了让中国向法国投资建设新能源产业。结果让意大利摘了桃子！一旦建成之后就是意大利后来居上了——意大利在欧盟中的份量比匈牙利要大的多，欧盟可以威胁把匈牙利踢出去，无论如何都不敢威胁把意大利踢出去——就如同苏联不会威胁把俄罗斯或乌克兰或白俄罗斯开除出联盟一样——这种角色被踢出去，联盟本身就不存在了！

欧洲无法作为一个整体整合起来，就会逐渐变成了一个松散的中等国家俱乐部。彼此各怀鬼胎又能力和体量都有限。随着时间的推移就只能整合到别的体系之中。如果说20年前欧洲还有希望，现在及以后是彻底没有作为一个独立体系的希望了！当这个联盟不具有成为独立体系的希望之后，成员之间就不再是合作关系，而会成为竞争关系。竞争什么呢？竞争嵌入某个独立体系时占据更优势的位置！英国脱欧并占据美元的欧洲离岸中心那个位置，对于其他接入那个体系的欧洲国家来说，就更具有优势。为什么是意大利和匈牙利这样的国家率先掉头？因为他们在原有体系中被边缘化了！德法会成为最纠结和最犹豫的存在。

大毛最开始拒绝承认自己在经济和产业上不具有建立独立体系的角色定位。所以牢牢的抓住前苏联地区并染指中东。大力发展农业和能源，试图通过粮食和能源的绝对优势整合其他相同生态位的国家，形成自己的主导体系。经过20年的内部挣扎之后——特别是俄乌冲突的现实反馈，让大毛和它所试图控制的地区都清醒的意识到，大毛无力形成独立的体系。虽然大毛依然是一股强大的地缘政治军事力量。但在产业和经济秩序的体系建设方面大毛也只能选择嵌入某个体系——20年来大毛主导的亚欧经济带不仅毫无进展而且经此一役之后越发形成虚设了！

中吉乌铁路是一个信号，也仅仅是一个开始：它意味着大毛在两害相权取其轻的考虑下接受现实，放弃对这个区域的绝对控制。与其让这些昔日的小弟去嵌入美国主导的体系，不如让他们嵌入中国主导的经济体系，至少自己在兔子主导的经济体系中有着举足轻重的份量，而在鹰酱主导的经济体系中，什么都不是，还经常被针对！这个事最大困难的地方是接受自己的非主导者定位。

随着科技和产业的发展以及发展相关产业的前期投入和各种配套门槛的指数级递增。整个生态就会形成一个超级主导者和一堆下游普通成员的格局。即便是处于中间位置的角色也会持续弱化，而不是强化。

这一个特征不仅存在于国家之间，也存在于任何社会的个体之间——随着资源的加速聚集和各种门槛的水涨船高。由中间层级越升到上层的概率越来越低，而由中间层级坠落到下层的概率会越来越高。而下层的独立性越来越难以保持，依附性会越来越强。举一个最简单的个体例子：当越来越多的农民进入城市成为新市民，衣食住行都需要成本。这种城市化进程会形成一个更大的消费市场，为社会生产提供拉动力。但对于个体而言，为了保证最基本的生存，就不得不将自己嵌入到某种分工协作的体系之中去换取生计所需，而这套体系一定不会让个体拥有资源和时间上的剩余——必然会通过改变游戏规则，强化竞争力度来消耗掉。

以前，大毛周边的小弟没有选择！那是一种多层次的深度捆绑：即便想与外部互通有无也没有资金技术和能力去修建通道，只能与大毛互通有无。这就如同外蒙只能与中俄做生意一样——外蒙倒是想把矿卖给美国，未经允许空运都出不去。中吉乌铁路和中塔铁路线，以及哈萨克斯坦的里海港口和与之配套的基础设施建设，不仅会形成新的亚欧商贸通道，更会让这些国家进一步摆脱对大毛的依赖，并嵌入另外一个规模和分工协作密度更大，体系也更健全的经济秩序中。

随着时间的推移和利益的深度捆绑，体系的主导者就会为他们提供一种保障——如同中南半岛上柬埔寨和老挝不再受越南的控制一样。

大毛家理性的学者和政治家无不叹息苏联的解体。这不仅仅是个地缘政治的瓦解，随着时间的推移，人们会越来越清晰的发现：毛熊家族的成员，因为苏联的解体，彻底丧失了构建起一套具有主导地位的经济产业体系的可能性。不管苏联内部产业结构多么的不合理，但它依然是一个完整的体系！而且这个体系，不仅比改开时的兔子更完整和更先进，而且下游的资源供应，更是不知道比兔子好多少倍！兔子到2009年全国战备石油储备都只能保持30天。不管是粮食、能源、原材料，随着工业体系规模的发展，每一项都成了一剑封喉的软肋。为了保证这种供应的安全，花费太多的代价。而这些东西对苏联来说天生就是安全和富足的！如果苏联不解体，苏联比兔子更加具有建立起主导性地位的经济体系的潜力。如果是那样的话，随着时间的推移，兔子的角色定位就只能跟印度并驾齐驱了！

欧洲的陨落，表面上看起来是妖魔鬼怪横行，实际上是以鹰酱为主导的经济体系中，中间层级角色逐渐退化的结果——欧洲与鹰酱的关系如此密切，为什么还会错失数字产业和新能源产业？无论是社会制度还是文化和生活习俗，欧洲都与鹰酱高度一致，完全不存在自我封闭和后知后觉的情况。那为什么还会错失呢？反而是与之格格不入的兔子却没有错失！同样属于东亚文化圈的日韩也错失了。你说奇怪不奇怪？其实，一点都不奇怪。因为你嵌入的越深，在具有超级主导者的体系中，独立性就越差，就会加速弱化成一个纯粹的依附者角色。在这个体系秩序中，超级主导者只会赢者通吃。

在过去的40年中，兔子与鹰酱主导的体系一直是合作关系不是嵌入式的依附关系。虽然内外都存在大量的施压，试图让兔子成为鹰酱主导的体系中的依附者。但是，兔子家的决策者顶住了这种压力——哪怕背负各种骂名，也绝不松口。

接下来，高校中那些不遗余力试图让兔子完全融合到那个体系中的学者，将会陆续被清理掉。因为事实已经证明了他们是错的，而且错的离谱！

在《布局》那篇中，系统的阐述了决定产业发展的底层制约因素。这些决定性因素由技术和产业构成。所谓布局，就是对这些具有决定性意义的技术和产业持续投资，形成决定性的垄断优势。有了这个基础，就如同在最底层掐住了潜在挑战者的命脉。技术的扩散是不可避免的，但是从技术到产业需要时间，而从产业到基础设施，更需要时间、资金、人才等各种门槛。只要在这个过程中超级主导者在更多的领域形成优势，那些行为就会变得越来越没有意义，因为赢者通吃！

当我们说到突围和转型的时候，不仅是一个产业替代另外一个产业，也不仅是一些国家的贸易替代另外一些国家的贸易。同样不仅是一种经济增长的拉动方式替代另外一种经济增长的拉动方式。而是角色的彻底转换！在这个过程中，唯一也是最大的危险，就是遭遇类似于苏联解体的危机。除此之外，只要不自己作死，徐图改进，角色转换成功之后，想象空间非常大。这一篇阐述的是框架性的原理，下一篇再就具体的主题细节展开阐述。
