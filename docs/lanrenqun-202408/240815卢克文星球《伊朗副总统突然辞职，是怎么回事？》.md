

# 伊朗副总统突然辞职，是怎么回事？

8月11日，上任仅10天的伊朗副总统扎里夫，居然宣布辞职了。

虽然扎里夫在社交平台X上称辞职是出于“对工作结果不满意”，准备回归大学继续学术道路，但很难不让人浮想联翩。

## 浮想联翩，这位新副总统为何突然辞官？

国人对扎里夫不算熟悉，上任第二天才确定他的中文译名。

不过对熟悉伊核问题的朋友来说，扎里夫这个名字早就耳熟能详。2015年7月14日，他曾作为伊朗外长，与六国达成伊核问题全面协议，那一天，在富丽堂皇的维也纳联合国中心，扎里夫亲自促成了这份历史性协议的诞生。

当时他表示：“我们达成了一个对所有人都不完美的协议，但这是我们所能达成的协议，对我们所有人来说都是重要成果。”

把扎里夫的这项经历，和改革派总统佩泽什基安任命他为副总统，联系在一块看，不难看出他是一位主张温和外交的改革派人士。

8月2日，他又被任命为“战略事务副手”和总统战略研究中心主任。这样的安排，无疑在宣告伊朗未来的外交政策将主张缓和矛盾，多边合作。

伊朗近年来由于制裁而经济低迷，所以改革派在竞选时主张外交上对西方缓和，恢复伊核协议以求解除制裁，并力求抓住机遇，加强与周边国家的关系和与中俄等大国的合作。

这样的主张有利于恢复经济，缓和外部矛盾，也赢得了伊朗民众的支持，改革派就这样成功竞选上台了。

不过有些国家不乐见一个温和、务实并与西方走向缓和的伊朗政府出现，比如以色列。之前《聊聊哈马斯领导人被暗杀》一文已经解释了内塔尼亚胡急于拉伊朗和美国下水，破坏伊朗与西方关系缓和的动机，这里无需多说。

于是在7月31日，佩泽什基安就职典礼当天，以色列就送了份“大礼”，在德黑兰把哈马斯领导人哈尼亚送上了天。

这一炸只杀死了两个人，破坏力却巨大。从伊通社近10天左右发布的政治新闻看，谴责“犹太复国主义政权”和声援巴勒斯坦的内容几乎霸屏，占了大部分内容。这足以看出，以色列的刺杀行为已经给伊朗新政府的正常施政带来了极大干扰，这事绝对不能算完。

不解决好这件事，骑虎难下的佩泽什基安政府就无法向伊朗人交代，遑论接下来的执政。

千千万万双眼睛在看着呢。贾姆卡兰清真寺的复仇血旗已经升起多日，那可不能白升，这次不是打几发导弹就能敷衍过去的了。而近年来，改革派和保守派在政治经济上已经有了更多共识，改革派主张和缓、务实的外交政策，但不代表无脑跪舔西方。

虽然伊朗内鬼和带路党确实多，但以色列整这么大活出来，他们明面上也得装出同仇敌忾的样子，否则被愤怒的群众用石头砸死不是没可能。就连伊朗议会的犹太人代表都称：“有必要就哈尼亚遇刺对犹太复国主义政权进行报复。”

总之，伊朗难得地上下团结一心，要严厉报复以色列。伊朗总统、陆军总司令及代理外长这段时间相继强硬表态，称“以色列的无耻行径必将得到强有力的回应”。

由于伊朗响应“奥运非战倡议”，将报复计划延迟到8月11日巴黎奥运闭幕后当然是合情合理的。不过如今伊朗仍在按兵不动，副总统又突然辞职，这就让伊朗未来的举动又变得更加扑朔迷离。

伊朗的报复行动何时发生，如何报复，我们不得而知。但从扎里夫辞职和新内阁成员名单的关系中分析，或许能窥见伊朗政坛变动的蛛丝马迹。

这次的新内阁成员名单列出了19位部长，无需一一分析，只要关注经济、外交和军事这几位重要部长的人选，就足以看出伊朗新内阁大致是啥成分。

伊朗央行行长阿卜杜勒·纳赛尔·赫马提被提名为经济事务和财政部长；赛义德·阿巴斯·阿拉格希被提名为外交部长，他曾是扎里夫任外长时的副外长；伊朗武装部队第一副总参谋长阿齐兹·纳西尔扎德被提名为国防部长。

先来看财长阿卜杜勒·纳赛尔·赫马提，关注过2021年伊朗大选的朋友或许不会陌生，他当时作为改革派的“黑马”候选人受到广泛关注，虽然不敌莱希，但这位央行行长凭借着低调、务实的作风赢得了很多伊朗人的好感，更是在当年的电视辩论中大胆地以西方制裁、大规模抗议等敏感问题炮轰保守派。

而自称改革主义者的赫马提早在竞选总统时，就计划对伊朗进行一系列大刀阔斧的经济改革，担任财长，肯定是有利于他和改革派大展拳脚的。

而阿格拉希作为扎里夫多年的搭档，说他不是改革派，别人都不信。当年伊核协议达成，也有阿格拉希一份功劳，直到2021年，他的副外长职位被目前的代理外长阿里·巴盖里取代之前，他都在致力于恢复伊核协议和取消制裁。

而阿里·巴盖里是一位强硬派的外长，最近他频繁发声，又是声称以色列“有罪不罚”将使其得寸进尺，又在呼吁伊斯兰国家团结起来支持巴勒斯坦。在12日还和我国外长王毅通电话，获得了中方对伊朗制止以色列侵略和促成加沙停火的支持。

虽然他是个强硬派，或者说最激进的保守派，但任代理外长的这段时间，巴盖里也算恪尽职守，更符合现政府的反以路线。干得好好的，你把他换了干嘛？

唯一的解释就是，从改革派政府的长远外交政策考虑，他不能长干下去。为哈尼亚报仇，制止以色列侵略固然是目前的刚需，但改革派不可能接受长期和西方对立下去。

更何况，阿格拉希曾长期担任伊核谈判代表团团长，在伊核问题上外交经验丰富，改革派还指着他恢复伊核协议，解除西方制裁呢。

对抗是必要的，但是对抗完了，日子还得过，关系还是要缓和。

再来看担负着对抗和报仇的重责大任的防长。

纳西尔扎德作为伊朗军队重要人物，以前就经常出于“工作原因”往莫斯科跑。来而不往非礼也，8月5日，绍伊古率团访伊，而不到一周之后，纳西尔扎德就被提名为防长。谁都看得出来，此举是为了促进伊俄关系，增加伊朗在军事上对抗美以的筹码。

这么一看，伊朗新内阁成员的提名，既扶了大批改革派上台，又搞强硬温和两手抓。这有两重意思：西方想欺负伊朗，那是不可能的，但想缓和关系，我们也是可以谈的。

回到扎里夫的辞职，按道理，扎里夫绝不可能对改革派内阁有何不满，他既不是保守派，外长阿格拉希又是他的老部下。如果说他无意仕途，做了那么多年外长，又在2019年辞职风波中被总统挽留下，这也解释不通。

最大的可能性就是改革派内部玩了一手“弃车保帅”，用扎里夫的辞职来转移保守派的视线，为新内阁的上台铺路。为了缓和政坛矛盾，维护国内稳定，这步棋似乎也不是不能理解。

就像以色列刺杀哈尼亚，在伊朗政坛的水面上击出强烈水花，扎里夫的辞职，也将激起一片涟漪，这对未来伊朗会怎么报复以色列，在这之后以何种态度缓和外交关系，可能会产生很大影响。
