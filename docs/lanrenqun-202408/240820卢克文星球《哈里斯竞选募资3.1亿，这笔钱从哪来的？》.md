

# 哈里斯竞选募资 3.1 亿, 这笔钱从哪来的?

240820

美国政坛风云骤变，哈里斯在短短半个月内就实现了惊人的蜕变，从昔日坐冷板凳的副总统角色里脱颖而出，一跃成为赢得民主党内超90%支持率的总统候选人。

看来政治这盘棋，变化比翻书还快。特朗普那边枪声一响，民主党就迅速调整队伍，齐刷刷地站在哈里斯身后。现在连民调，哈里斯都能领先一步，国际知名的民调公司益普索公布的最新全国性民意调查结果显示，哈里斯以42%的支持率领先特朗普的37%。更为显著的是，在七个至关重要的摇摆州中，哈里斯的支持率还领先特朗普2个百分点。（这个数据真假存疑）

这里，就不得不提哈里斯那惊人的“吸金”能力了。

在短短两周内，她竟然筹集到了高达3.1亿美元的竞选资金，这一数额比特朗普的1.387亿美元的两倍还多，创下了美国总统大选筹款史上的新纪录。

那么，这么多钱，哈里斯到底是怎么“吸”来的呢？

哈里斯宣布总统竞选后，24小时的筹集金额就达到了惊人的2.31亿美元。这笔巨款中，包括了8100万美元来自小额捐款者的热情支持，以及来自大额捐款者的1.5亿美元“重磅炸弹”。说起来，这1.5亿美元的捐款，主要源自拜登竞选那会儿的“金库”——旗舰超级政治行动委员会“未来前行”（Future Forward）。

拜登一说退选不玩了，这“未来前行”就立马转身，给哈里斯当起了后盾，24小时内就成功向民主党的金主敲定了这笔新资金承诺，其竞选团队也果断地更名为“哈里斯竞选总统”。

这意味着，哈里斯将全面继承拜登的竞选资源，不仅能拿到不菲的竞选资金，还能全盘接手一个超千名专业精英组成的竞选团队，这便宜捡的让希拉里都得感慨望尘莫及。

当然，最郁闷的还得是特朗普，他本来花了几百万美金雇黑粉黑拜登，结果这下全泡汤了，搞得特朗普现在一听见哈里斯的名字，就情绪不正常。

但要是就此评价她只是个运气爆棚的花瓶，那可就太小瞧人了。

看看这次公开为哈里斯站台的人就知道了：

迪士尼财富的继承人、电影制片人阿比盖尔·迪士尼说：“我全心全意地支持她。”

民主党“超级捐款者”、金融巨鳄乔治·索罗斯的儿子亚历山大·索罗斯在社交媒体上发帖称，“现在是我们团结在哈里斯周围、击败特朗普的时候了。”

民主党的大财主，微软董事会成员里德·霍夫曼也已计划向哈里斯捐款，捐款规模甚至会超过此前给拜登团队的捐款。

都说爱笑的女孩运气不会差，但实际上，哈里斯整天这么乐呵呵的背后，还是因为她有着手眼通天的门路。

回顾哈里斯的崛起之路，不难发现她在结交权贵方面天赋异禀，她擅长捕捉机遇，灵活应变，这种见缝插针、见风使舵的智慧，让她在职业生涯中不断打破天花板，一路高歌猛进。

哈里斯的职业生涯始于地区的县检察官，因为加州众议院议长威利·布朗的赏识，哈里斯很快就结识了当时还是参议员的奥巴马。

2008年，奥巴马竞选时，哈里斯忙前忙后，为奥巴马筹集资金，这份衷心和“友谊”也在日后带给哈里斯超额回报。

2010年，哈里斯正式被任命为加州检察总长，身居高位，总统支持，又有黑人女性的特殊光环，此时的哈里斯人气扶摇直上。一般人到这可能就知足常乐了，但哈里斯的胃口显然没那么小，她更进一层楼地结识了拜登大儿子博·拜登，并很快正式踏入政坛，成为了加州民主党的一名参议员。

要说哈里斯人生最成功的一步棋，还得是2013年嫁给了犹太裔律师道格。

道格在好莱坞和华尔街有着深厚的人脉，且与以色列政府保持着紧密的联系，2016年，为了支持哈里斯竞选参议员，道格慷慨解囊投入了数百万美元，后又利用他的人脉和运作手段让哈里斯在2020年成为拜登的副总统。这也就解释了为什么，拜登一说退选，哈里斯就能在短时间内得到华尔街金融界的鼎力支持。

然而，与权势人物过从甚密往往也会惹人非议。马斯克就曾直言不讳地指出，哈里斯实际上是金融大鳄索罗斯的“傀儡”。而拜登决定不寻求连任，背后反映出民主党内部的一种倾向——他们更希望拥有一个能够更好服务于自身利益的“木偶”式总统。

美国的总统选举，往往被形容为一场凶猛的烧钱竞赛。其中充斥着巨额资金的流动与较量。

富豪们通过超级政治委员会等机制，巧妙地脱离联邦竞选财务法监管，得以不受捐款金额限制，无上限地向候选人注入资金，强力输血。尽管候选人上台后未必会完全成为富豪们的傀儡，但这些财力雄厚的支持者自然不会错过利用美国政府内部强大人脉资源的机会。

日常的联络问候自不必说，寻求适度的政策倾斜以期获利也无可厚非，许多超级富豪会慷慨解囊，向两党的候选人捐赠巨款，希望自己扶持的候选人上台之后，会通过政策调整或人事任命等手段，为自己或所在行业带来实质性的利益。

从资本的角度来看，哈里斯显然是个比特朗普回报率更高的总统候选人。一来，哈里斯跟华尔街硅谷走的更近，二来，哈里斯的正式从政经历仅有8年，其中还包括了四年作为拜登副总统的“吉祥物”时期，而她目前除了身份政治正确的喊口号外，还缺乏一套完整、系统的政治纲领。

这样一位资历尚浅，政治目标混沌的政客，显然比特朗普那个犟老头，好打交道的多。

所以，在拜登宣布退选并转而全力支持哈里斯竞选总统之后，民主党的大金主霍夫曼毫不犹豫地表示，将向哈里斯提供比原先给予拜登的700万美金捐助还要高出一倍的资金支持，助力哈里斯战胜特朗普，成为美国历史上首位女总统。

而作为交换条件，哈里斯要在就任总统后，撤换掉联邦贸易委员会(FTC)主席丽娜·可汗(Lina Khan)。今年只有35岁的可汗或许是美国大企业最痛恨的人。

三年前，拜登上台后就提名了不到32岁的可汗出任FTC主席，掌管美国最重要的反垄断部门。这一手操作，当时可是炸了锅，毕竟可汗是反垄断界明星级的鹰派学者。早在2017年，还在耶鲁念书的可汗就写了篇《亚马逊的反垄断悖论》，直接点名电商巨头亚马逊的竞争模式早已触达垄断，而当前美国的反垄断法律体系却对此束手无策。此文一出，年仅27岁的可汗就火了，还跟民主党的大腕儿沃伦参议员搭上了线，为日后从政铺平了路。

而在执掌FTC这三年多，可汗和大资本真刀真枪对着干，针对亚马逊、Meta等科技巨擘发起了一系列强有力的挑战。在她的领导下，FTC成功阻止了多起天价并购案，其中最为人瞩目的便是英伟达拟以400亿美元收购Arm的交易，FTC坚决反对，认为此举将对行业创新构成严重威胁。

微软的霍夫曼对可汗抱有深深的敌意，原因不难理解：微软同样多次成为FTC的“重点关注对象”，虽然没像其他巨头那样吃官司，但调查诉讼也是家常便饭。特别是微软斥资687亿美元收购动视暴雪的案例，FTC更是坚决反对，双方为此展开了长达近两年的法律拉锯战。

尽管微软最终赢得了官司，但FTC并未放弃，不依不饶的准备继续上诉。可汗的想法是，就算输了官司，也得给国会提个醒，让他们知道反垄断法该更新了。

而今，可汗正试图将硅谷最热的AI风口纳入监管雷达。亚马逊、Meta、微软、谷歌、OpenAI等AI巨头纷纷被FTC召去“喝茶”。可汗还计划在全美范围内废除竞业禁止协议(高管和核心员工除外)。这样一来，美国的打工人就能更自由的跳槽，而美国劳资关系的游戏规则也会被重写一遍。

可汗一直被认为是拜登提拔上来对付大资本的一块硬骨头。虽然，拜登任期内，有不少超级富豪曾跳出来公开反对过可汗，可执政经历超40年的拜登已经是个老江湖了，自有手腕，稳住局面。

若特朗普复辟，鉴于其“美国至上”政策与全球化、虚拟资本的冲突，他或将重用可汗。而哈里斯这边，她和民主党紧紧抱住了华尔街和硅谷的大腿，特别是现在情况这么紧张，她得让金主们开心，不然那些钱可白花了。

眼下，哈里斯在谷歌和脸书的数字广告支出就高达5700万美元，特朗普那边才560万，连哈里斯的十分之一还不到，真是小巫见大巫。

这些撒出去的金币，哈里斯可不白花，仅是名声大噪，广告还帮她拉了不少票和钱。哈里斯的多数数字广告都包含募款呼吁，这招儿挺灵的，给她小额捐款的人特别多。

特朗普那边就保守多了，原来他广告砸的也猛，但现在法律官司缠身，花钱如流水，竞选预算里六分之一都搭进去了。这么一来，他在广告上就没那么大方了，整个竞选策略都可能受影响。

说到底，美国选举就是一场金钱游戏。

早在100多年前，俄亥俄州的共和党大佬马克·汉纳就一语道破：“要赢得选举，需要两样东西，一是金钱，第二个我就记不得了。”

这告诉我们，在美国政治圈，支持候选人可不是动动手指点个赞那么简单，得实打实地砸真金白银才行。
