

# 从“油罐车事件通报”，看食品安全的“黄金三角”

今天要为你带来的，是零售咨询专家，消费行业专家黄碧云老师，每半月一期的消费观察。而且今天的题目特别应景。我们要说的，是一个跟所有人都关心的行业，食品行业。

正好，就在前几天，油罐车运输食用油的事件处理有进展了。8月25日，国务院食安办、国家发改委、公安部、交通运输部、国家卫健委、市场监管总局、国家粮食和储备局，针对媒体反映的“罐车运输食用植物油乱象问题”，组成联合调查组，分别前往河北、天津、内蒙古、陕西展开调查。

借用《人民日报》的评论，食用植物油是家家户户日常生活的必需品，对其安全监管必须严字当头，坚持最严谨的标准、最严格的监管、最严厉的处罚、最严肃的问责。

你可以从措辞里感受一下食品安全问题的重要性。这也是今天最重要的社会议题之一。

而黄碧云老师为我们带来的这期消费观察非常特别，其中的案例，正好处在三个议题的交集上，分别是，食品溯源、有机农业，以及降本增效。没错，都是食品行业最关心的事。

而且更有趣的是，这三件事之间，正好组成了一个黄金三角。这三件事互为支撑，互相辅助，你很难单独解决其中一件事，要干好其中一件，就必须同时干好另外两件。

这么说可能有点抽象，咱们还是先进入具体的案例。

黄碧云老师这回走访的，是几个盒马村。盒马村不是一个行政单位，而是指与盒马深度合作，而且品质达到盒马品控标准的种植养殖基地。大概就相当于盒马的农产品供应商。而其中最关键的合作品类，就是有机农产品。

前面说过，盒马村做有机农产品，恰好诠释了食品行业的黄金三角。也就是，你要想把有机农产品做起来，就必须同时解决农业溯源和降本增效的问题。

为什么？咱们先从一个问题说起。有机蔬菜为什么贵？这主要是因为，有机蔬菜有一组额外的硬性成本。

比如，土地租金，因为有机农场要建一个隔离带，隔离带大概要占掉10%的土地面积。这就导致有机蔬菜的土地成本，要比一般的化学种植高10%左右。

再比如，认证成本。有机蔬菜的认证，需要检测水、土壤和空气，土壤符合GB15618标准，灌溉水符合GB5084标准，空气符合GB3095标准。而且农场必须有专业人士负责做农事记录，还要承担认证机构检查员的交通住宿费用。这么算下来，500到1000亩的土地，80个左右的蔬菜品种，一年的认证费用大概是20万元。

再比如，有机蔬菜的亩产更低。按照相关标准，有机蔬菜是不能用化肥催长的，因此有机蔬菜的亩产会比化学种植低30%左右。

再比如，有机农场的管理成本更高，还有除草成本、生物农药的成本、土壤改良成本等等。

前面这些成本叠加在一起，有机蔬菜的成本理论上，是化学种植的2到3倍。这也是它为什么卖得贵。

黄碧云老师走访的其中一个盒马村，是云南的芸岭鲜生。按照芸岭鲜生的说法，有机是过程，而不是结果。因此，商家为了让成本变得可感，同时也让顾客买得放心，就需要做好一件事，可溯源。比如，芸岭鲜生有一套自己的溯源系统，在每一份菜的筐子上，都有一个溯源码。每经历一个环节，比如产出、运输、入库，就要扫一次码。这么一来，每个环节都可以追溯。

但是，说到这，你可能也发现了，还有一个终极问题，这就是，成本。本来有机蔬菜的成本就高，再加上这么一套溯源系统，就会卖得更贵。很多农户就会担心销路，要么不敢种，要么着急收割。

比如，北京平谷的桃子，原本品相不错，但农户害怕销量不好，担心桃子烂在地里，就不等到全熟，而是六七成熟就采摘了。这么一来，桃子确实容易卖出去，但因为提前采摘的桃子口感略差一点，会影响平谷大桃的口碑。同时，因为很多桃子被提前采摘，留到全熟的桃子变少，到时卖得就更贵。你看，这就成了死循环，农户担心太贵卖不出去，就提前采摘，导致品相好的桃子变少，进而卖得更贵。而更贵的价格，就会让农户更担心，更着急采摘。

注意，说到这，我们可以读出一个真相。有机农业最核心的关口，其实就一个，这就是，确定性。

只要农户能有确定的销路和确定的价格，他们就会大范围种植。供应量提高，成本就会下降，就可以卖得更便宜。销量一上来，自然就有更多的利润，去做产品升级，去搭建食品溯源系统，让安全和品质获得进一步的保证。

这也是为什么说，食品溯源、有机农业、降本增效，这三件事组成了一个彼此勾连的黄金三角。要么全都做好，要么全都做不好。而撬动这个黄金三角的起点，就在于给农户提供足够大的确定性。

而盒马鲜生和盒马村之间的合作，与其说是买卖商品，不如说是在交换确定性。

比如，黄碧云老师调研的盒马村之一，云南的芸岭鲜生，这也是最大的一个盒马村。芸岭鲜生的有机蔬菜种植面积，占云南省的7.1%。他们2020年创立以来，最高年增速是107%，直到现在还保持在年均40%的增长。

# 芸岭鲜生和盒马之间，是怎么交换确定性的呢？

比如，针对销量，盒马和芸岭之间的合作，采取的是订单农业的模式。盒马每年上半年的四五月份，跟芸岭制定第二年的种植计划。双方约定好种植的品种、数量，盒马采取恒价采购，也就是恒定价，农户按照约定种植出来的，一口价照单全收。这么一来，农户就有了最起码的确定性。

但是，注意，这还只是最初级的部分。真正厉害的部分还在后面，你可以感受一下，盒马对确定性的交付，已经精细到什么程度？

比如，盒马发现，销量最好的包装规格，大概是叶菜一包250克，根茎菜一包600到750克，这恰好是一家人一盘菜的量。这是个确定的数字。而基于这个初期的确定性，就可以衍生出更多的确定性。

一来，我可以确定，在盒马，小包装的有机蔬菜卖得好，因此在上游的供应环节，芸岭就会把蔬菜提前做成小包装。这么一来，不仅方便销售，还可以防止传染。因为叶菜在运输过程中，一旦挤压烂掉，就会传染给更多的蔬菜。而芸岭鲜生为了避免传染，索性把蔬菜的打包加工间建在了种植基地旁边。蔬菜采摘之后，就马上完成打包。这就降低了损耗，提高了利用效率。

二来，我既然知道蔬菜需要打包售卖，我就可以控制蔬菜的尺寸，让它处在一个最适合打包的尺寸里。没错，蔬菜的尺寸也可以提前确定。比如韭菜，长度绝对不能超过40厘米，最好不超过35厘米。因为只要不超过35厘米，韭菜就是直着向上长的，超过这个尺寸，叶子就会耷拉下来。而这部分耷拉下来的叶子不仅不方便包装，还可能会在除草时被来往走动的工作人员踩坏。处理这部分烂叶子又需要一笔时间成本。因此，韭菜控制在40厘米上下是利用效率最高的。

再比如，空心菜，原本可以长到45厘米，爱吃的人也多。但是，自己在家处理起来比较麻烦，洗净一把空心菜要五六分钟。因此，芸岭鲜生就把空心菜做成了打包的净菜。而且长度控制在35厘米，比一般的空心菜要短10厘米。这其实是他们计算过的结果。因为35厘米的空心菜，对半切开后，大概12厘米。对于家里30厘米直径的炒锅来说刚刚好，不容易炒到锅外。做起来更方便，顾客也就更愿意买。

再比如，奶白菜不超过25厘米，生菜也不能超过25厘米，这都是盒马测试得出的确定的数据。有了这个确定性，作为供应商的芸岭就可以严丝合缝地种植，不多耽误一天，也不多付出任何无效的种植成本。

还有个很厉害的细节，是关于蔬菜的品相。一般来说，芸岭鲜生采摘叶菜时，会选那些最外面的叶子有点老的菜。这是因为，蔬菜在运输到超市的过程中一定会失水，而最外面的菜叶能起到保水的作用。假如短途运输，最外面失水的叶子大概是两片，假如长途运输，失水的叶子会增加到4片。换句话说，只要知道这些菜是运到哪里的，我就可以控制采摘标准。比如长途运输，就选那些外面4片叶子有点老的菜，因为早晚都是要被去掉的。而且要特别说明的是，这些被去掉的叶子只是卖相不好，成分并不差。因此符合标准的叶子还可以回收，作为果蔬汁的原料。

换句话说，这个确定性，已经不仅仅停留在蔬菜的产量，就连蔬菜的尺寸、叶子的数量，都能被提前交付了确定性。

而反过来，芸岭作为供应商，提供的最大的确定性之一是，供应稳定。要知道，对于顾客来说，假如这个蔬菜只有一两个月有，其他月份都没有，他们来年就未必想得起来再买，这部分顾客就会流失。因此，供应商要确保全年有货。这就要说到芸岭的优势了。芸岭地处云南，而云南有丰富的海拔分布。比如芸岭的11个有机产业园，分布在云南不同的海拔上。海拔的变化，就对冲掉了季节的变化。有的蔬菜，夏天可以在海拔高的地方种植，冬天就可以放在海拔低的地方。目前，芸岭已经有50多个品种的蔬菜，可以确保全年稳定供应。

再比如，这套合作思路不仅应用在蔬菜上，也应用在养殖业。像湖北特产，大别山的黑山羊，原来一只羊可以分出6个部位，利润是2000多元。而基于盒马的用户行为数据，现在的大别山黑山羊，已经开发出53个部位，一只羊的利润从2000多元变成了5300元。

你看，在盒马村的案例里，我们能读出一个微妙的连锁反应。对盒马村来说，那个最初的积极改变，是怎么发生的？不是因为有了更多的订单，不是因为有了更多的产品，而是因为，这个原本靠天吃饭的系统里，突然被加入了一点确定性。以前的很多不确定因素，包括销路、价格、供应，突然有了一个起码的确定性。然后基于这个确定性，整个系统开始快速向好的方向进化。

就像冯小刚的《非诚勿扰》里有句台词，要诚实，这是我们今后在一起的前提。放到商业世界里也一样，所谓诚实，不仅在于你的态度，更在于你能否交付足够多的确定性。
