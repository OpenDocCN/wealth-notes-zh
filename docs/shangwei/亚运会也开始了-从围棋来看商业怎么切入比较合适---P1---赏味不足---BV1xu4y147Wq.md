## 围棋领域的商业环境

围棋属于竞技类项目，但其商业表现一直不佳。类似的小众领域做商业很难，需要先观察大环境。

以下是分析大环境时需要关注的几个层面：
*   **国家与政府**在推动什么。
*   **高校**在研究或培养什么。
*   **企业**在投资或布局什么。
*   行业的**地域分布**（如南北方比例）。
*   行业的**投入产出比**。

不能仅凭个人感觉（例如觉得AlphaGo很厉害）就盲目进入一个领域。一个核心原则是：**在一个行业中，“卖水卖铲子”的生意总能赚钱**，且风险相对较小。无论行业本身发展好坏，为从业者提供工具、培训或服务的模式往往更稳健。

上一节我们分析了行业大环境，本节中我们聚焦于围棋领域内部的商业模式。

## 圈内商业模式的局限

圈内商业主要指围绕协会、培训机构和竞赛展开的业务。这与买彩票类似，最终能成为职业选手或打出名堂的人凤毛麟角。

竞技体育领域普遍如此，大众只看到成功者，却忽略了大量不成功的“炮灰”所付出的精力与金钱。圈内最大的商业就是**培训**。

以下是圈内商业模式的主要问题：
1.  **客单价低**：受众（广大普通家庭）支付能力有限，导致培训课程价格难以提升。
2.  **非刚需**：围棋在升学和主流竞赛中并非硬性要求，市场难以大规模扩张。
3.  **从业者价值天花板低**：顶尖职业选手除了比赛，主要出路是教学或解说，但这两者的商业回报（课单价低、工作累）与其投入不成正比。
4.  **行业缺乏财富效应**：无法为从业者创造足够的财富，导致圈子难以自然成长，陷入僵局。

公式可以概括为：`圈内商业价值 ≈ (有限用户数 × 低客单价) - 高培养成本`。这导致投入产出比不高。

既然圈内突破困难，上一节我们探讨了其局限性，本节中我们来看看向圈外拓展的可能性。

## 向圈外拓展的商业策略

当圈内难以突破时，应转向圈外寻找机会。核心思路是**以围棋为切入点，拓展其他技能或商业价值**。

“卖水卖铲子”是其一，例如做围棋自媒体教学、解说。更有效的策略是“出圈”。

**出圈的本质是切换受众和关注点**。大众关注柯洁或战鹰，并非因为懂围棋，而是因为对**AI对战**、**年轻天才**或**网络梗**感兴趣。围棋本身变成了一个标签（Tag）。

出圈之所以有效，是因为：
*   **受众基数大**：圈外潜在用户远多于圈内。
*   **评价标准变化**：圈外用户不再苛求专业围棋水平，更关注**个人特质、故事性、娱乐价值**。
*   **流量变现潜力大**：有了广泛的受众基础，才可能产生可观的商业流量。

![](img/e51cb893964148c2f6165531bfdb318c_12.png)

此外，还有“高举高打”策略，例如组织**海内外交流**或面向**高净值人群**的培训。此时，高客单价并非源于围棋本身，而是源于“国际交流”的形式或高端客户群的身份价值。

上一节我们介绍了出圈和高端化策略，本节中我们来总结具体的切入方法。

## 商业切入方法总结

对于围棋这类小众领域，商业切入主要有以下路径：

![](img/e51cb893964148c2f6165531bfdb318c_14.png)

![](img/e51cb893964148c2f6165531bfdb318c_16.png)

以下是几种可行的商业切入方向：
1.  **培训**：基础模式，但受限于非刚需和低客单价，难以形成规模效应，投入产出比相对较低。
2.  **解说与科普**：依赖个人“整活”能力和整体娱乐环境。如果只围绕纯围棋内容，很难拓展用户。更优解是尝试“**围棋 + X**”的交叉学科模式。
3.  **出圈**：最靠谱的路径之一。关键在于找到能让圈外大众感兴趣的连接点（如AI、人物故事、娱乐化内容）。成功出圈后，商业价值将脱离原有专业范畴。
4.  **服务高净值人群或国际交流**：通过提升服务形式（如高端私教、国际赛事游学）来抬高客单价，目标客户付费的是附加价值而非围棋本身。

这个思维模式可以迁移到任何小众领域。如果在一个领域内做到金字塔尖很难，就不要只执着于一条路走到黑。可以考虑：
*   为该领域从业者“卖水卖铲子”（提供工具、服务）。
*   与其他领域结合，做交叉学科。
*   提升服务对象层级，做高净值生意。

许多人容易“当局者迷”，局限于自己熟悉的领域，即使其天花板明显。关键在于认清形势，敢于向性价比更高或更广阔的领域探索和迁移。
