# 课程一：停止内耗，聚焦基本盘 🎯

在本节课中，我们将探讨一种普遍存在的心理状态——“想东想西”，即过度思考与无谓焦虑。我们将分析其本质，并提供清晰的行动思路，帮助你停止与“空气”斗智斗勇，将精力聚焦于真正重要的事情上。

## 主题定义：何为“想东想西”？

今天讨论的主题，可以通俗地称为“想东想西”或“戏多”。更正式的说法是杞人忧天。它指的是在人际交往或处理事务时，在事情并未实际发生的情况下，内心已经上演了复杂的内心戏。

这完全是与空气斗智斗勇。就像把全部资金投入股市后开始焦虑，但焦虑本身无法影响股市走向。你以为自己能控制全局，但实际上连局部都无法控制。

## 核心误区：试图控制不可控之事

上一节我们定义了问题，本节中我们来看看其核心误区：人们常常为无法控制的事情焦虑。

以下是常见的思维误区：
*   **担心朋友不开心。**
*   **担心同事不喜欢自己。**
*   **担心领导未来不帮助自己。**

首先需要明确：**这三个问题是你能控制的吗？** 最根本的问题在于，许多人不懂得去寻找那些本就认同和喜爱自己的人，却总想着如何让所有人来爱自己。

这就像很多人会问：“这件事做不成功怎么办？” 但作为普通人，我们做任何事都不可能保证一次性成功。既然失败是可能性之一，过度担忧便没有意义。社交中也一样，被人拒绝是正常现象，我们的目标是寻找互相认同的人，而非说服所有人认同自己。

这里有一个关键真理：**舔狗不得好死**。即便你做得再好，对方若不领情也是徒劳；若对方领情，也意味着谁舔都行，你将陷入无休止的内卷竞争。

## 焦虑的本质与正确应对

有的人认为，想得多是因为对不确定性的焦虑。但这个逻辑并不成立。

无论是等待晋升、录取通知、恋爱结果还是商业合作，甚至在考试、面试、相亲结果出来前，许多人都会焦虑到失眠，甚至去“赛博烧香”。我的观点恰恰相反：**焦虑情绪本身无法完全消除**。

我们的目标不应是消除焦虑，而是建立对时间和全局的正确认知。花时间焦虑只能得出两个结论：

1.  **你的时间可能不值钱。** 如果你有时间焦虑，不如向前看，思考如何进步或赚钱。即便不向前看，用于休息放松也是更好的选择。如果你自己都不珍惜自己的时间，别人更不会尊重你。
2.  **你缺乏备选方案（Plan B, C, D...）。** 我们能坦然接受任何结果的前提，是拥有应对不同结果的准备，而非孤注一掷。**焦虑是一种情绪，它不影响结果的发生**，我们不是活在科幻片里。

## 行动准则：聚焦可控，接受多元

我们不应该试图控制无法控制的事情。任何事件都可能产生多种结果，存在不同的发展路径（类似平行宇宙概念）。这就是“蝴蝶效应”：一个微小变动可能引发不可预见的连锁反应。

我常建议别人：**先做出一个60分的产品或服务**，不要一开始就追求完美，陷入过度思考和细节纠缠。但十个人里有九个会反驳：“口碑很重要，做不好会坏口碑。”

这又陷入了误区：
*   **首先，你无法垄断行业，也并非世界名人，不必高估自己的“口碑”影响力。**
*   **其次，口碑好坏与你的产品服务没有绝对关系。** 你无法让所有人满意，同样，你也很难让所有人都不满意。**在社会中，只要基数足够大，对任何事都大约会有50%的人支持，50%的人反对。** 人性就是如此。

有人会问：那些匠心独运、做大做强的人是如何建立好口碑的？需要明白：**我们已经成年，不是活在童话里**。等你做大做强后，自然可以这样宣传，而不是在什么都没有的时候，就幻想这些不切实际的事情。

## 实践路径：积累基本盘，直面问题

那么，正确的做法是什么？**有问题，就解决问题**。但在此之前，有三件日常需要持续积累的事情：

以下是需要持续积累的三个基本盘：
1.  **金钱**
2.  **人际关系**
3.  **健康的身体**

除了这三样，别的无需多想。你不会每天担心明天是否还活着，或者先睁眼还是先遭遇地震。很多事情，想与不想并无作用。例如，有人总怕被举报、被网暴、被拉黑而感到伤心。

需要客观认识到：**只要与外界产生连接，就必然会有麻烦和问题，这是人类社会的常态**。人活着就有喜怒哀乐，不能只看到“怒”和“麻烦”，也要看到你获得的好东西。与人交往，在获得积极情绪价值的同时，也可能获得消极情绪价值，不能永远沉浸在后者中。

因此，**你想赚钱、想折腾，你得到的越多，针对你的人也自然会越多，这是社会和人性规律**。所以才有“闷声发大财”的说法。我们需要做的是正视这一客观规律，而不是陷入“我是不是不行”、“我是不是不适合”的自我怀疑。

任何事情都有两面性。**你的目标越大，面临的挑战自然越多**。正如“一千个读者眼中有一千个哈姆雷特”，你无法知道每个人如何看待你。但只要明白基数定律，就知道总会有人觉得你好，也有人觉得你不好。**这与你个人无关**，你只需将其视为社会与人性规律即可。

---

本节课中我们一起学习了“想东想西”这种内耗模式的本质与误区。关键在于停止试图控制不可控之事（如他人看法），停止无谓焦虑，转而将精力聚焦于持续积累**金钱、人际关系、健康**这三个基本盘。接受世界的多元与复杂，**有问题就解决问题**，在行动中前进，而非在幻想中纠结。
