# 一人可行的赚钱篇3：我们找到了切入点，接下来就是找npc搜集信息---P1---赏味不足---BV1

在本节课中，我们将要学习在确定了一个可行的商业切入点后，如何通过寻找并接触关键人物（即“NPC”）来系统地收集信息，从而将想法落地。

上一节我们介绍了如何寻找政府、企业或高校每年必须支出的经费作为切入点。本节中我们来看看如何围绕这个切入点，通过“找NPC”来获取关键信息，推动项目前进。

## 第一步：明确信息搜集的两条并行路径

找到切入点后，你需要同时从两个方向搜集信息。这意味着你既要接触作为需求方的甲方客户，也要接触曾经为他们提供服务的供应商。因为双方提供的信息存在差异，结合两者才能获得更全面的认知。

以下是两条路径的具体说明：

*   **甲方客户路径**：直接接触有经费预算的部门或负责人。例如，企业的培训费可以找人力资源部门；高校项目可以找学院老师或院长。
*   **供应商路径**：接触曾为这些甲方提供过服务的机构。对于政府项目，建议优先寻找曾承接政府项目的培训机构或咨询公司，而非直接联系政府单位。

## 第二步：如何寻找并接触“NPC”

寻找这些关键人物需要利用公开信息进行搜索，并以恰当的方式建立初步联系。

以下是具体的方法与步骤：

1.  **利用公开信息搜索**：在网络上搜索如“2024年 校企合作”、“政企合作 项目圆满结束”等关键词。成功合作的项目方通常会发布宣传文章，你可以从中找到企业或机构名称。
2.  **定位联系人**：找到目标公司后，通过其官方网站、公众号等渠道，寻找商务或销售部门的联系方式。
3.  **进行初步沟通**：联系时，需进行适当的自我包装。不要直接表明来意，而是根据对方的业务寻找合作切入点。例如，对方做招生，你可以提及自己拥有生源渠道。目的是争取见面机会，因此沟通需专业、得体。

## 第三步：以培训项目为例深入操作

我们以企业、高校或政府的年度培训经费为例，说明如何将信息转化为机会。

通过之前的了解，你可能已经找到一些曾承接大数据、云计算等培训业务的机构。此时，你的重点是了解当前及未来的需求。

以下是关键操作点：

*   **关注新方向**：重点了解如AI、数据资产、新质生产力等新兴领域的培训需求，而非竞争激烈的传统项目。
*   **摸清具体需求**：通过与培训机构的交流，摸清甲方的具体需求，例如：需要什么级别的讲师（AI硕士/博士）、培训内容大纲、培训天数等。
*   **匹配市场资源**：在摸清需求后，再到市场上寻找能够提供相应讲师的供应商。这种模式的核心是资源整合与对接，一旦走通，利润可观。

## 第四步：如何处理软件或技术转移项目

对于政府或产业园区的软件交付、创业扶持、技术转移等每年固定支出的小额项目，操作逻辑类似，但对接方可能更分散。

以下是操作要点：

*   **寻找对接入口**：这类项目可能需要对接地方文旅局、科技局、科委、教育局或相关行业协会等多个部门。
*   **利用成功案例**：寻找曾经成功申报过此类项目或获得过基金扶持的企业。这些信息通常会在地方政府网站公示，或由企业自身进行宣传。
*   **选择你的角色**：你可以选择两条路。一是直接对接甲方，作为乙方承接项目；二是对接曾服务过甲方的乙方公司，为其提供新的服务（如低空经济、AI解决方案），成为乙方的外包商。核心目标是先参与进去，融入流程。

## 第五步：沟通时的关键技巧

在与“NPC”沟通时，技巧至关重要。如果你的社交能力不强，这条路会较难，但其优势在于能持续获取年度经费机会。

以下是在沟通中需要掌握的技巧：

*   **拓宽话题范围**：不要只聊你事先设定的单一方向。例如，在聊人才补助时，可以主动试探对方是否还有培训经费、竞赛经费、创业经费等其他预算。你需要主动引导和发掘潜在机会。
*   **善用行业协会**：行业协会汇聚了多家企业，是高效获取综合性信息的渠道。但与之沟通需要你具备更好的个人包装和沟通能力。
*   **保持耐心与投入**：不要指望接触十几个人就能成功。信息搜集和关系建立需要投入大量时间和精力，缺乏耐心很难取得成果。

本节课中我们一起学习了在确定商业切入点后，如何通过双线并行的策略寻找关键“NPC”，利用公开信息进行接触，并在沟通中拓宽信息面以发现更多机会。核心在于主动出击、耐心积累，并灵活选择自己在项目链中的角色，目标是先“上牌桌”参与进去。

> 注：如果你在职业规划、主业发展或副业赚钱方面有具体问题，希望结合个人情况获得更接地气的建议，可以整理好你的个人背景与具体问题。
