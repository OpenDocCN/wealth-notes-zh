# 课程：如何为自己创造幸福感 - 第1课 🧠💖

在本节课中，我们将学习一个至关重要的主题：如何主动为自己创造和维持幸福感。我们将探讨情绪的来源、工作与生活的平衡，以及通过具体行动提升日常满足感的方法。

上一节我们介绍了课程的整体目标，本节中我们来看看导致情绪问题的深层原因。

## 情绪问题的根源

情绪低落通常是长期积累的结果。许多人认为心情不好是孤立事件或短期压力造成的，但事实并非如此。

根据更广泛的生活观察，日常情绪问题主要源于两个特点。

以下是导致压力积累的两个常见模式：

1.  **日常给自己压力过大，过于紧绷**：过度担忧当下表现和未来发展。这常是潜意识或被动行为，但会持续自我施压。
2.  **日常生活过于节俭**：在衣食住行上长期保持节俭。虽然短期内没问题，但时间一长，作为普通人会感到难受，缺乏幸福感。

人不是圣人或机器，需要情绪价值。也许未来机器人也需要，更何况是人。

有人会认为，因此更应该拼搏、奋斗和节俭。但核心观点是：奋斗的前提是所做之事必须是**自己能控制的事情**。

**自己能控制的事情**定义：想做就做，不想做就不做。有能力多赚就多赚，想停摆今天就可以停摆。

所有受雇工作、按指令行事的行为，都不是个人能完全控制的。对于自己不能控制的事情，就不应为之过度拼搏、奋斗或节俭。不要认为吃苦本身是光荣的。

理解了情绪积累的原因后，接下来我们看看与大家最相关的工作领域。

## 关于工作的心态调整

工作是每个人最息息相关的事情。许多人反馈，他们总想做好工作、完成他人安排的任务、给公司和同事留下好印象、向别人证明自己。

需要明确：人活着是为了自己，不是为了别人。向他人证明自己可以，但至少要证明**自己可控的事情**。

许多人的难过、伤心、自卑和自责，源于对团队项目的过度负责。在大多数工作中，个体只是螺丝钉，无法左右整个项目的质量和走向。

对于这类工作，我们的控制力是有限的。你可以努力，但不必把责任往心里去，更无需自责。这没有意义，因为你无法左右整个项目。

公司和社会的某些人可能会进行PUA，将项目责任归咎于你。但你自己不能PUA自己。要明白，很多事情不在我们的控制范围内。

**我们自己可控的事情**是：由自己主导，自己真心想做好的事情。可以用心去做，不会因为领导或他人的问题而让自己背锅或行为扭曲。

从签劳动合同那一刻起，它就是一份不平等条约。整个项目若真的没做好，也不是你一个人的问题。只要问心无愧即可，无需自责。

调整了工作心态，我们再来关注直接影响情绪的日常生活部分。

## 日常衣食住行的情绪价值

很多时候，我们需要一杯好咖啡、一份好饮料、一顿美餐、一件让人开心的衣服、一个不错的妆容或一个温暖的拥抱。

这或许是因为身体需要更好营养，但更重要的是，人需要这些事物带来的**情绪价值**，以维持快乐状态。

这类似于游戏中的精神力。一旦精神力低于某个阈值，就会出现大量负面情绪。

许多小伙伴目前经济拮据，不舍得花钱。但很多东西其实很便宜。例如冰箱贴或贴纸，花费不多但非常治愈，平时看到也能开心。这就是情绪价值。

我们每天可能很忙，无暇关心和爱护自己。最近对朋友说得最多的一句话是：你要多爱自己。

再累再辛苦，一方面要分清什么是自己可控的。对于不可控的事，尽量不要耗费过多精力。人的精力有限，为何要为不可控之事耗尽精力？

另一方面，要多给自己买点好吃的、好喝的、好玩的，多为自己提供情绪价值。否则，负面情绪一旦积累，自己往往察觉不到，等到爆发时就得不偿失了。届时会发生什么，无人知晓。

了解了如何从日常获取快乐后，我们来总结幸福感的根本来源。

## 幸福感的根本来源

幸福感从何而来？核心有两点。

以下是获得幸福感的两大支柱：

1.  **少操心不是自己的事**：包括父母的事。父母是独立个体。该养老养老，该付出时间付出时间，但不要额外被其PUA。许多人遇到父母借钱不还、赌博等问题。要明确界限。
2.  **多赚钱**：两点结合。第一，少操心。只要是自己不能控制的、不在整个商业链路中占大头的，就别操这个心。不要被亲情、友情等束缚，否则容易陷入困境。有能力帮忙当然可以，但大部分人在自己尚未过好或未来不明时，累死累活本质是在帮别人赚钱。这也是为什么支持大家拥有自己现金流业务的原因。再累再苦，为自己做，开心。

多赚钱的本质是：99%的问题钱能解决。剩下的若不能解决，部分原因可能是钱还不够多。

说句俗话：当你真的赚到钱时，会发现很多人对你很好，很多矛盾自然消失，情绪也会变好。人毕竟是物理世界的生物，花钱能带来更多幸福感，现实如此。

需要补充的是：这些人对你好，并非真心想对你好；这些矛盾也并非真的消失。而是赚到钱后，你能避开这些东西。我们解决不了这些问题，但可以避开。

最重要的核心不在这里，而在于：**你想做就做，不想做就不做**。你无需在意他人，无论对方多厉害、是不是领导、是否想压榨你，都与你无关。完全可以按照自己的想法来。

这才是幸福感。不是指996、007，年入百万就叫幸福感。这不是真正的幸福。从许多高薪但身心俱疲的打工者身上，看不到任何幸福感。
