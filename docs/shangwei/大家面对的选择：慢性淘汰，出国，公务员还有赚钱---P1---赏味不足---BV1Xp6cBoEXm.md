# 大家面对的选择：慢性淘汰，出国，公务员还有赚钱---P1---赏味不足---BV1Xp6cBoEXm

## 概述

在本节课中，我们将要学习并分析在当下社会环境中，个人可能面临的几种主要发展路径。我们将逐一探讨“慢性淘汰”、“出国”、“公务员”与“赚钱”这四条道路，分析其背后的逻辑、潜在问题与核心要点，帮助你理解不同选择所对应的现实情况。

## 慢性淘汰：无法回避的宏观趋势 🔄

上一节我们介绍了本课程将要探讨的四个方向，本节中我们来看看第一条路径：慢性淘汰。

我所说的慢性淘汰，也可以理解为慢性死亡。这个现象很简单，无论你身处哪个行业，无论年龄多大，只要是在中国打工，都会面临这个情况。女性还额外面临结婚生育的问题。以2026年为时间点估算，女性大约在29至31岁，男性大约在31至33岁，就可能开始面临职业挑战。

这并不是说在这个年龄段就找不到工作，而是找到的工作薪资会越来越低，整体的待遇包会下降。这个判断有一个依据：从现在到2034年，每年的应届毕业生数量都维持在高位。虽然生育率在下降，但其对劳动力市场的影响要在十几年后才会显现，在当下这个时间点，我们不会因此受益。

相反，你会受到另一波冲击：大量幼儿园、小学、初高中因生源减少而关闭。这些机构里的教师和其他工作人员会流入社会，在就业市场上与我们形成竞争。这意味着，在一个周期内，你将面临非常激烈的竞争。

有人会问，为什么认为我们会被淘汰？我从不认为具体某个人一定会被淘汰。我的意思是，从宏观角度、从整个社会的金字塔结构和市场上的岗位数量来看，如果市场上的人数是 **X**，岗位数是 **Y**，那么总有 **X - Y** 个人找不到工作。

随着时间的推移，为了社会稳定，社会总是倾向于让年轻人获得更多工作机会，而不是让那些上有老下有小或年纪较大的人持续占据岗位。这不是我们现在独有的问题，而是任何人类社会金字塔结构都会面对的问题。

金字塔之所以能持续运转，在于它是一个血液替换或迭代的过程。所谓迭代，就是新人上来，老人下去。如果按照“有经验就该一直留下”的逻辑，每年应届生和年轻人也需要工作，那么市场上的岗位就应该无限增加。但如果没有淘汰机制，淘汰谁呢？最终只能淘汰年纪大的。

从个人细节角度看，你可能能力比别人强，这我认可。但我重申，我不是在针对具体个人，我只是从宏观层面指出：年纪大的群体就会慢慢被淘汰，这是2026年到2034年期间将要面对的现实。

所以，我一直强调，无论你打什么工，不要和我谈行业或你的个人能力。只要你不是行业内的前0.1%或前1%，你迟早会被淘汰，只是时间早晚的问题。

这条路最核心的问题还不是被淘汰本身，而在于当你33岁、35岁甚至40岁被淘汰后，你并非直接养老或退休，你还需要想办法生存。但到那时，你还能想出什么办法呢？因此，我在这里写得比较委婉，用了“慢性淘汰”，其实更准确的应该是“慢性死亡”。因为到那个时候，问我还有什么解决办法，我认为没有了。

## 出国：机遇与挑战并存的地图 🌍

上一节我们探讨了“慢性淘汰”这一被动路径，本节中我们来看看主动选择“出国”可能面临的情况。

说实话，我认为国内外现在是一盘棋。国外当然有好处。第一，海外的生活压力很多时候没有国内这么大，相对而言。第二，如果你能在海外找到工作——前提是你能找到——那么打工的性价比通常比国内高，这一点我比较有信心。

但是，海外有一个核心问题是签证。你到底能获得一年、两年的签证，还是一直能续签下去？这是个比较大的问题。所以，基本上我觉得疫情之后选择出国的人，比较多的要么是能力很强，能够技术移民；要么就是直接在那边找一个有绿卡、有身份的异性结婚。

在我看来，出国这个选择，最不应该选的就是“出去一两年再回来”，或者“出去水个硕士、博士再回来”。千万不要。你想想，你在国内欠缺的是与别人的竞争力，因为你没有背景，没有资本，也没有太多经验。你出国之后，不管两年还是二十年，你在海外积累的很多经验，包括对国情的理解，在国内是行不通的。

再说句不好听的，你出去多少年回来，该不会的还是不会。我指的是，该不会赚钱还是不会赚钱。你出去几年，既没有学会赚钱，又没有在国内积累资源，那么出去兜一圈回来干什么呢？至少在我看来是这样，因为大部分人出去就是兜一圈就回来。

你可以去看看现在的市场，问问你认识的政治家、资本家、商人或任何市场人士，海归身份已经不具备太大的优势了。这是第二点，希望你们能想明白。

## 公务员：稳固金字塔中的位置 🏛️

上一节我们分析了出国这条路的利弊，本节中我们转向第三条传统路径：公务员。

公务员这个事情也比较简单。你要想清楚，到2026年是什么阶段？现在可以说是“官商红”军政的时代。放到六七十年代是第一代，八九十年代是第二代，那么到2026年，好歹都是二代、三代了（四代暂且不谈）。

如果你是一个老百姓，和我一样是人民群众的政治地位，那么你去考公务员、选国企（事业编）、选教师编，你上面面对的领导或竞争者，很多都是二代、三代。他们可能是直系亲属，也可能是发小、恋人、旁系亲属等等。到2026年，社会的金字塔结构其实已经很牢固了。

你想考进公务员并往上走，那就问问自己，你觉得有这种可能吗？你我的出身就注定了我们不可能往上走。明白吗？这不是“铁饭碗”是否稳固的问题。

因为事实上，今天就算给我一个铁饭碗直到退休，我差的也不是这个饭碗。你们差的也不是这个饭碗。人活着不是为了一个铁饭碗，人活着是要去生活、去享受、要有自己的爱好。我不是说每天在那里当牛做马。但问题是，但凡你是人民群众出身，家里没有积累，你就只能给那些二代、三代当牛做马。这是一个现实的政治结构。

另外，我今天作为一个人民群众，好歹还能跟你们抱怨抱怨。如果你们已经是公务员了，我跟你们讲，你们没有任何机会抱怨，你们敢抱怨吗？这是第三个选择。

## 赚钱：掌握人生主动权的核心能力 💰

上一节我们讨论了公务员体系的现实，本节中我们来看看最后一条，也是我认为最核心的路径：赚钱。

我说的赚钱，是指不管你是通过互联网、借助AI的能力，还是在社会上混人脉、混关系、察言观色、进行各种沟通，只要在我的价值观里，你不违反刑法，不踩政治红线，不涉及黄赌毒，你就应该去尝试。这是你最后一条可以去选的路：先去让自己赚到钱，或者说让自己拥有若干个赚钱的能力。

这是第一步。因为只有在你未来的人生中，把这个赚钱的雪球滚起来，你在这个过程中才相当于向水里扔了一颗石子，泛起了涟漪。接下来你赌的是，在能够赚钱的情况下，在过程中能碰到很多天时地利人和的人，能碰到一些合作伙伴，能给你带来更多的业务。其实我们每个人走的就是这条路。

如果你作为一个老百姓，问我2026年还能做什么选择，我就告诉你，你现在站在一个四条路的路口，四选一，你自己选。你没别的路了。

你打工，不管是大厂小厂，都是我刚刚说的那个结论。出国面对的问题很多，你们可能大部分人不了解，只是觉得“我要逃离”，但单纯逃离没有用。也就是说，无论你选择大公司、小公司还是出国，你其实都没有一个稳定的、确定的未来，也没有一个确定的赚钱方式。你在国内赚不到，出国也赚不到。不可能说在中国赚不到，到英国就赚得到，这没有逻辑。

至于公务员，我真的跟你们讲，很多人没去过，去了后悔就来不及了。但凡你能明白当下社会的政治结构，我建议你，如果家里不是有积累的，就不要去，没意思。这跟你读书读得多好没有关系，学历再高，你人民群众的地位，跟人家二代、三代有什么可比的？

另一方面，你也不要跟我谈公平，你有什么公平可言呢？如果有公平可言，那就不存在二代、三代了。所以我一直以来一而再再而三地跟你们讲，要去赚钱，要去自己积累一些业务，要让自己有现金流的能力，要让自己在未来的人生时间线里，有这么若干条线能够往前推进。

为什么？因为只有这样，你对你的人生才有主控权。同时，你才有可能积累到一些能够跟你合作、未来能够给你帮助的关系链。否则，无论你选择第一条慢性淘汰，还是选择出国，还是选择公务员，本质上这个漏洞就是——你的命都在别人手上，能听懂吗？你在自己手上是没有主动权的，那你怎么往后走呢？

这条路，其实至少在我看来就这么简单。你想想看有什么区别吗？没有区别。就是你不要去在乎那些表面的东西，说什么“我今天去创业公司、大公司”，还去什么“电网”、去什么地方，然后“我是不是做公务员”，这都是表面现象。

你要看核心。核心就是你有没有拥有自己赚钱的能力，你有没有拥有围绕你、以你为圆心的一些关系链。你如果没有，你就得去积累。你怎么积累？就得通过那些赚钱的业务去积累。这是核心。

你说我打个工，从我角度来讲，我不管你年薪是10万还是300万，你都是明天老板说你没有工作就没有，HR说你没有就没有。有什么区别呢？你有什么价值？你没有，什么都没有。你想想是不是这个道理。

## 总结

本节课中我们一起学习了当代个人可能面临的四条发展路径：慢性淘汰、出国、公务员与赚钱。我们分析了每条路径背后的宏观逻辑、现实挑战与核心问题。慢性淘汰是多数打工者无法回避的宏观趋势；出国充满不确定性且归国优势不再明显；公务员体系在稳固的金字塔中留给普通人的上升空间有限；而主动培养赚钱能力与构建个人关系链，则是掌握人生主动权、应对不确定未来的核心策略。希望你能根据自身情况，审慎思考并做出选择。
