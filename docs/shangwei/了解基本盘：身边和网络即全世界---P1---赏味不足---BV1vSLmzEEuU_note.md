# 了解基本盘：身边和网络即全世界---P1---赏味不足---BV1vSLmzEEuU

在本节课中，我们将探讨一个核心概念：“基本盘”。我们将分析为何许多人会不自觉地认为“身边即世界”或“网络即世界”，以及这种认知如何导致信息茧房和决策失误。通过理解这些现象，我们可以学习如何打破局限，更客观地认识真实世界。

## 因果论：身边即世界的陷阱

上一节我们介绍了课程主题，本节中我们来看看第一个核心观点：在沟通中，人们常犯的一个错误是**将个人有限的身边环境等同于整个世界**。

例如，当有人向“陈老师”抱怨身边环境不如意时，得到的回应是：“那是你生病了”。对方可能会反驳，认为自己接触不到其他环境。但事实是，许多人的生活轨迹高度同质化：学校、公司两点一线，或沉浸在抖音、小红书等网络平台。在这些场景中，接触的多是“工具人”、“消耗品”或是网络上虚构的“年薪百万网友”，而非真实、立体的活人。

越是接触不到真实多元的世界，个人的**信息茧房**就越牢固。这会导致信息日益闭塞和片面，从而使人做出错误的决定，陷入恶性循环。真正“与世界接轨”的人，能清醒认识到自己的无知，明白世界存在许多自己未知但真实存在的部分。而越不接轨的人，则越容易坚信“身边即全世界”，对自己不知道的事物采取“鸵鸟心态”，甚至拒绝接受相反的信息。

## 两种典型的认知偏差

理解了“身边即世界”的普遍性后，我们来看看现实中两种具体的认知偏差。以下是两种常见的片面结论：

1.  **“朋友圈欣欣向荣”偏差**：有些人根据自己朋友圈的光鲜生活，认为普遍情况很好，从而质疑他人所描述的困难只是“幸存者偏差”。
2.  **“身边都是牛马”偏差**：另一些人则看到身边人都在辛苦工作或忙于考公，觉得环境压抑，找不到志同道合者，因而感到无力突破。

针对第一种偏差，建议是**实地观察**：去上海世纪大道、北京西二旗看看通勤人群的状态，或乘坐几趟火车卧铺，看看普通百姓的生活，以此理解何为真正的“幸存者偏差”。

针对第二种偏差，核心在于**主动拓展**。如果不主动打破原有圈子，接触那不同的5%，那么视野和思维将永远被那95%的同质化环境所局限。突破需要自己主动迈出脚步，因为外界不会有人拉着你突破。

## 网络的“盖中盖”幻象

上一节我们讨论了线下环境的局限，本节中我们来看看网络如何加剧这种认知扭曲。网络信息茧房是“一层套一层”的，宛如“盖中盖”，让人难以辨别真假。

网络制造的最大错觉主要有两个：
*   大家都很有钱。
*   现在毕业生起薪都很高。

这种对“钱”的扭曲认知会衍生到其他领域，例如找对象。网络上充斥着**人设**，但许多人却信以为真，仿佛人均收入百万。实际上，只需查看任何城市的应届生起薪**平均值**或**中位数**数据，就能发现这与现实相去甚远。

此外，网络无限放大了**比较心理**和**攀比心理**。这容易让人产生“只有自己穷”的焦虑，并对当下生活产生不满。然而，每个人的现状都是由自身过往选择决定的，与他人过得如何**本质没有关系**。将攀比心理投射到伴侣身上（如比较礼物、旅行消费），除了扰乱自身生活和感情，并无任何益处。

## 信息实验与多元化思维

前面我们分析了身边和网络环境的误导性，最后我们通过一个思想实验来理解其深层机制。以前有个著名的“微信群实验”：

假设一个500人的微信群，每天都有499个人对你进行夸张的赞美（“夸夸群”模式）。不出半个月，你就会膨胀，失去对自我、环境、社会的客观判断，变得盲目自信。但客观上，除了言语赞美，什么都没发生。

反之，如果群里499人整天刷屏百万年薪、别墅豪车的信息，你会觉得全世界都是富豪；如果刷屏的都是贫困苦难的信息，你又会觉得世界即将毁灭。

这本质上是一种**垃圾信息洗脑**，也会形成一种**幸存者偏差**。问题不在于信息本身（确实有人夸你，也的确有人买得起别墅），而在于**接收信息的普罗大众缺乏多元化的思维**。他们容易从单一维度的信息中得出极端（二极管）结论，例如“大家都很有钱”或“世界很苦”。

**基本盘**的逻辑往往是：道理都懂，但走不出去。走不出去的根本原因在于**格局不够大**，而格局小的根源在于**对真实世界了解太少**。大多数人的认知来自家长、老师、学校、网络的灌输，而非亲身经历。很多人甚至从未离开过自己长期生活的地方。

那么，该怎么办？关键在于认识到，**除了亲眼所见、亲身经历，其他一切信息都仅供参考**。包括本课程所讲的内容，也旨在提供一种视角，真正的突破仍需你主动去观察、体验和思考。
