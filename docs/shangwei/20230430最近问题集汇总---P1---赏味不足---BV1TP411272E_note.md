## 政策与赚钱的关系 🧭

上一节我们介绍了课程概述，本节中我们来看看第一个核心问题：政策与个人赚钱的关系。

![](img/99ca31dbe34660b26475d16f2c22b7f6_11.png)

有观点认为宏观政策与普通百姓无关。这种看法是错误的。要想在未来赚到钱，必须关注并理解政策动向。不关注政策，未来赚钱将非常困难。

![](img/99ca31dbe34660b26475d16f2c22b7f6_13.png)

未来的经济形势非常严峻。如果认为政策与自己无关，那么可能只能停留在打工层面，难以获得更好的发展机会。

## 合作模式与行动力 💼

![](img/99ca31dbe34660b26475d16f2c22b7f6_15.png)

理解了政策的重要性后，我们来看看如何采取行动以及可能的合作模式。

以下是关于合作与行动的两点说明：

1.  **关于合作**：比起雇佣关系，更期待商业合作。可以分享大部分利润，追求共赢。
2.  **关于行动**：很多人表示“不知道怎么做”，根源在于缺乏实践。所有交流应建立在实践基础上。没有行动，空谈没有意义。

## 读书的价值与运用 📚

上一节我们探讨了行动的重要性，本节我们来分析另一个常见疑问：读书的价值。

读书肯定有用，但取决于三个因素：
*   **读什么书**：选择正确的领域。
*   **是否读进去**：深度理解而非浅尝辄止。
*   **能否创造价值**：将知识转化为为他人解决问题的能力。

公式可以概括为：**读书价值 = 书籍选择 × 理解深度 × 价值转化能力**。

![](img/99ca31dbe34660b26475d16f2c22b7f6_21.png)

单纯追求学历（如本科、读研读博）而不考虑其市场价值，意义有限。有用的金融或商业书籍很多，关键在于阅读后的应用。避免“非黑即白”的二极管思维。

![](img/99ca31dbe34660b26475d16f2c22b7f6_23.png)

## 应届生与普通人的现实策略 🎓

对于资源有限的应届生或普通人，需要采取务实的策略。

当前及未来5-10年的经济环境将非常严峻。对于家境普通、无一技之长的应届生，策略是：
1.  **降低预期，先找一份工作稳定下来**。
2.  **放弃“既要、又要、还要”的心态**，能屈能伸，认清现实。

![](img/99ca31dbe34660b26475d16f2c22b7f6_27.png)

关于如何利用政策、获取补贴等具体方法，这些是具有商业价值的信息，无法免费公开。核心逻辑在于：**有资源者可探讨合作；无资源者需抓紧时间窗口（约2-3年）进行积累**。

## 核心竞争力的本质 ⚙️

![](img/99ca31dbe34660b26475d16f2c22b7f6_31.png)

在严峻的环境下，什么是真正的核心竞争力？

![](img/99ca31dbe34660b26475d16f2c22b7f6_33.png)

核心竞争力在于 **以结果为导向，为他人创造价值的能力**。简而言之：**能否帮别人赚到钱**。

![](img/99ca31dbe34660b26475d16f2c22b7f6_35.png)

学历、技能、方向都不是核心。如果不能将这些转化为为他人盈利的能力，则不具备不可替代性。可替代性强，就没有核心竞争力。

## 打工、副业与AI焦虑 ⏳

![](img/99ca31dbe34660b26475d16f2c22b7f6_37.png)

![](img/99ca31dbe34660b26475d16f2c22b7f6_39.png)

我们来分析打工与副业的关系，以及如何看待技术变革带来的焦虑。

1.  **打工与副业**：两者都重要。打工提供基础稳定收入；副业是应对主业不稳定的必要补充。应从现在开始积累副业能力。
2.  **AI焦虑**：AI只是加速器。即使没有AI，许多人也会因激烈的竞争（如年轻劳动力、岗位缩减）而被淘汰。努力工作也可能被卷走，这是基本逻辑。

## 学生与职场人的行动指南 🗺️

针对不同阶段的人群，行动重点有所不同。

以下是针对两类人群的建议：

*   **对学生（大一大二/在读）**：
    *   首要保证能毕业。
    *   尽早接触商业、实践项目、尝试赚钱，踩坑越早越好。
    *   避免局限于校园，要对行业和社会有认知。

*   **对职场人**：
    *   工作中遇到的多数人际或琐碎问题，从长远看都是尘埃，不值得耗费情绪。
    *   聚焦于对自身成长和赚钱有益的事。

---

## 副业规划与职业规划 🔮

最后，我们来探讨具体的规划思路。

1.  **副业规划**：应像投资一样，采取多策略并行。
    *   **代码示例（策略模型）**：
        ```python
        # 副业组合示例
        side_business_portfolio = {
            “plan_a”: {“type”: “长线”, “投入”: “高”, “产出”: “潜在高”}, # 如创业项目
            “plan_b”: {“type”: “短线”, “投入”: “低”, “产出”: “稳定低”}, # 如接零活
            “plan_c”: {“type”: “持续性”, “投入”: “一次性”, “产出”: “持续”} # 如数字产品
        }
        ```
    *   不要只依赖单一副业（如仅做自媒体）。

2.  **职业规划**：未来几年的核心目标是 **生存**。
    *   获得一份相对稳定的主业收入。
    *   核心是积累人脉、商业逻辑、构建个人护城河。
    *  过去的线性职业规划在当下基建完善、增长放缓的时代已不太适用。

## 总结 📋

本节课中我们一起学习了多个关键点：

1.  **政策认知**：关注政策是赚钱的必要条件。
2.  **行动与合作**：实践高于空谈，推崇商业合作。
3.  **价值创造**：读书与学习的意义在于转化为替他人解决问题的能力。
4.  **务实策略**：普通人需降低预期，抓紧时间窗口积累。
5.  **核心竞争力**：本质是 **为他人创造价值** 的能力。
6.  **风险对冲**：打工与副业并重，规划多线收入来源。
7.  **聚焦重点**：学生尽早实践，职场人忽略琐碎，聚焦个人成长与盈利。

![](img/99ca31dbe34660b26475d16f2c22b7f6_49.png)

![](img/99ca31dbe34660b26475d16f2c22b7f6_51.png)

核心思想是：在充满变化的时代，抛开幻想，以 **现实和赚钱** 为导向，通过持续实践和积累，构建自己的生存与发展体系。
