# 课程：致所有想创新、改变、赚钱的人 - P1 🎯

在本节课中，我们将探讨一个核心主题：在快速变化的社会中，个人如何通过深入理解世界的运作逻辑来实现创新、改变与赚钱的目标。我们将摒弃表面认知，强调实践与深度体验的重要性。

## 前期提要：关注自我，而非他人看法

上一节我们介绍了课程概述，本节中我们来看看一个重要的前提。

核心是不要过于关心任何人的看法。他人的想法对你而言并不重要。重要的是你对自己状态的认知。你清醒与否，只有你自己知道。

## 第一点：年轻人是未来的希望

上一节我们强调了关注自我的重要性，本节中我们来看看为什么年轻人是关键。

你们才是未来的希望。因为你们年轻。在我这个年龄段的人，绝大部分已经被时代抛弃。他们或安于现状，或心有余而力不足。社会发展太快，若非业务属性所需，我也可能早已落后。

跟上社会不仅意味着知道新名词，如 **AI**、**Web3**、元宇宙。关键在于理解其运作逻辑、发展趋势和本质。这非常困难。我身边许多同龄人的认知早已被淘汰。我没有兴趣与他们交流，因为他们缺乏价值。

人类就是这样一种生物。无论之前智商、情商、学历如何，如果只看表面，最终只会变成工具人。无论之前多优秀，都没有用。

## 第二点：凝视深渊与体验深渊

上一节我们讨论了认知深度的必要性，本节中我们来看看如何获得这种深度。

很多人说要创新、改变、赚钱。有一句话叫做：你凝视深渊时，深渊也在凝视你。意思是，如果你只看到是非黑白中的“是”和“白”，而看不到“非”和“黑”，你如何理解世界？如何创新？如何赚钱？如何改变？

人类社会有太多东西不会出现在表面上。这就是冰山理论：只有20%浮在水面，80%隐藏在水下。你没有体验过深渊，没有体验过“黑”和“非”，就无法真正了解。靠看书、听人说、看视频都没用。

要创新、改变、赚钱，前提是先了解和体验。如果觉得时代太快，没有时间，那么你需要压缩时间。别人一天睡八小时，你睡四小时。别人不敢做的，你做。别人在娱乐时，你不娱乐。别人不会做的，你也做。别人不了解的，你去了解。

这个过程是：从表面看到深渊 -> 体验深渊 -> 成为深渊 -> 改变深渊。但很多人体验深渊后，未必能成为深渊，这需要天时地利人和。成为深渊后能否改变深渊，则取决于初心。屠龙者最终变成恶龙，这是常有的事。

## 第三点：皇帝的新衣与个人能力

上一节我们探讨了深入体验的必要性，本节中我们来看看勇气与能力的关系。

皇帝的新衣故事里，大家都喜欢那个孩子的勇气。前辈也教导我们要勇敢。但你现在还有这种勇气吗？即使有勇气，你有能力看到那件“新衣”吗？

当你看到“新衣”时，可能会发现不止一件，而且到处都是。你会看到太多不公、委屈和问题，但没有能力改变。因此，本质上你必须先强大自己。

我看到的世界非常割裂。例如，有人带着小孩在路边乞讨，我会捐钱。同时，我的伙伴可能在 **Web3** 投资中亏损数十万美金。这些钱本可以救济很多乞讨者，但不可能，因为钱是别人的。

有的人寒窗苦读数十载，得到一份工作，好一点的能养家，一般的可能温饱都难，工作还不稳定。但有的人，吃顿饭、喝次酒、做个项目就能赚几百万上千万。这个数据毫不夸张，只是你们没见过。

大家都只活一次，凭什么？当然，每条路上都有成功者和失败者。但关键在于，那些不走寻常路的人，无论成功与否，他们对社会、黑暗、深渊的了解要深刻得多。而那些循规蹈矩的人，无论学历智商多高，对冰山下的世界了解甚少。

## 第四点：视频的局限与自我实践

上一节我们讨论了现实世界的复杂性，本节中我们来看看获取真知的途径。

大家能看到我的视频已经两年了。你们往前翻，可以看到我做过游戏直播、视频、美剧解说等。从大部分人看到我视频到现在，快满两年了。这两年所有的视频内容，只是整个社会运作规律和模式的冰山一角中的一角。

除非你们能把我走过的路走一遍，否则单纯看视频是没用的。我所有视频只阐述一个核心思想：按照现在社会的发展，读书、看书和上网**不可能**带来任何有效信息。现实生活套路太多，到处都是老狐狸和骗子。

如果你指望别人，一路走来，最终在某个年龄段会发现，除了你自己，别人都靠不住。人与人的差距很大。能明白这些的人不少，他们在各自领域影响着更多人，但这并非义务。

作为年轻人，最大的优势就是年轻，活得比我们长。但数十年后，你们问问自己：活了这么久，求了这么久学历，认识了这么多人，做了这么久工作，你对每一个组织、企业、协会、商会的运作逻辑了解多少？对金融了解多少？对各行各业，无论是传统线下、线上、会销、直销、电销，还是每年出现的新技术背后的盈利模式，你了解多少？你还有多少时间去了解？更何况这些可能只是表面，背后还有政治、国际关系、人性等深层因素。

这些事情不用我来讲。梳理这些，或者问你身边的人，真的能找到答案吗？按我的想法，**不能**。

## 第五点：关注自身与实践是唯一途径

上一节我们指出了外部信息的局限性，本节中我们来看看解决问题的根本方法。

就像一开始说的，为什么要一直关注自己？因为本质上，无论你羡慕、嫉妒、抨击他人，说成功者都是二代，说自己就是劳碌命，说“我命由我不由天”，或者说我们改变不了命运，这些话都随便说。但这都改变不了你对社会了解多少这个事实。

除了调动情绪，没有用。你了解多少就是多少。要去了解，只能靠实践，没有别的可能性。这个实践就是我说的：体验深渊 -> 成为深渊 -> （或许）改变深渊。看个人造化。

所有人都像在练内功。如果你刚从学校出来，对社会一无所知，没有“内功”，然后你说要去赚钱。不是说你赚不到，而是在赚钱过程中，该踩的坑一个都不会少。这个坑踩多久、多深，取决于你做事的灵活程度。

坑不是随便踩的。有的人命好，踩坑后能复盘、学习。但也有不少人，踩一个坑就可能进去踩十年缝纫机。所以说，为什么要了解、要体验？因为你不了解、不体验，就没有任何“内功”，就等于在凭运气随机踩坑。只有了解之后，才能真正明白风险在哪里。
