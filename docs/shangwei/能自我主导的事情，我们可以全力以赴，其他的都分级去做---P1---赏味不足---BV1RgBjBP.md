# 能自我主导的事情，我们可以全力以赴，其他的都分级去做---P1---赏味不足---BV1RgBjBP

## 概述

在本节课中，我们将要学习一个核心观点：**对于你能自我主导的事情，可以全力以赴；对于你无法主导的事情，则需要分级处理**。这个观点源于许多认真负责的职场人常陷入的困境——因被误解、被甩锅而感到委屈，并试图通过“摆事实、讲道理”的方式去争论，结果往往适得其反，消耗大量情绪却收效甚微。本节课将深入分析为何“讲道理”常常无效，并提供一个清晰、可操作的“分级应对”框架，帮助你在复杂的工作关系中保护自己，更高效地分配精力。

## 为何“摆事实、讲道理”常常无效？🤔

上一节我们概述了本课的核心主题。本节中，我们来看看为什么在工作中，尤其是面对误解和指责时，试图“摆事实、讲道理”往往不是最佳策略。

许多人喜欢辩论，认为摆事实、讲道理显得自己非常理性。但是，你们要明白一点：在人类社会中，摆事实、讲道理有一个前提，那就是**双方是平等的**。事实上，请问你跟别人平等吗？或者别人跟你平等吗？你往往在很多事情上面，在这件事情还没有发生的时候，在一个原始的框架下面，就已经不平等了。

放到公司里面，你面对的误解往往来自你的同事、你的领导、你的HR，甚至是你领导的领导。你的同事跟你可能是平级，但你们所在的业务事业部、所做的工作、彼此的关系网络，都可能存在不对等。就更别谈你的Leader、你的Manager、你的HR，在公司的整个框架下面，你们原本就是不平等的。

你有什么好去摆事实讲道理的呢？你越摆事实讲道理，在别人眼中，你可能就越像一个死缠烂打、有错不认、屡教不改的人。到最后，除了你自己伤心难过、感到委屈，还有什么用呢？事实就是如此。你可能会问，为什么他们不讲道理？那我反问你，为什么他们要跟你讲道理呢？谁告诉你在一个企业框架下，老板、HR、你的Leader必须跟你讲道理？并没有这样的规定。

## 核心：区分“主导权”与“责任感”⚖️

理解了“讲道理”的局限性后，我们需要建立一个更根本的认知：区分事情的“主导权”和个人的“责任感”。

工作上的许多事情，我们是没有主导权的。你想用理性的方式去思考或争论，有一个前提，那就是**这件事你得有主导权**。我以前也跟很多人说过，如果项目没做好或犯了错，不要把锅全往自己身上揽。即使别人把锅甩给你，你也不要主动去接。因为一个项目是团队合作的结果。一个几十人、几百人公司的项目失败了，怪到一个执行者身上，这合理吗？

但是，很多理性上明白这一点的人，由于过于认真、负责，很容易自责。别人可能只是一句话，他却能自责一周甚至一个月。很多抑郁症、焦虑症等情绪问题，往往都发生在那些认真、负责的人身上。因为他们自我内疚感深，反而容易患上心理疾病。那些不那么在意的人，反而没有这种困扰。

所以，看问题要让自己站在一个第三方角度。永远不要过度代入自己，而是去评估其中的利害关系。你会发现，除了你自己觉得有道理，在别人的视角里，你可能就像一个不想认错、不想背锅的刺头。这对你有什么好处呢？

## 方法论：如何“分级”应对？📊

上一节我们明确了主导权的重要性。本节中，我们来看看具体如何操作，即“分级应对”的方法。

面对委屈、误解或被甩锅时，不要采用“二极管”思维。不是只有“忍气吞声”和“激烈对抗”两个极端。处理方式就像一个有1到10级的刻度盘。

以下是应对级别的划分示例：

*   **第1级（最低）**：语气平缓、态度诚恳地表达。“我知道问题了，这是我的不对。但我想说明一下，这个问题可能出在XXX地方。” 这只是轻微地澄清事实。
*   **第2-9级（中间）**：这些级别是1级和10级之间的过渡。你的态度、语气、措辞的强硬程度逐级递增。例如，从“建议”到“指出”，再到“强调”。
*   **第10级（最高）**：态度最强硬，完全不认错。“这跟我没有关系。我摆事实讲道理，问题是出在你们部门/别的事业部，不要甩锅给我。” 这是一种明确的边界宣示。

你不是只有1和10的选择，你还有中间2-9级的广阔空间。选择哪个级别，取决于你想给别人塑造什么形象，以及你愿意在这件事上花费多少时间和情绪。

*   **如果你想塑造“不好惹”的形象**，告诉别人你不是软柿子，可以选择较高级别（如7-9级）。
*   **如果你处于弱势（如试用期、实习期）**，可以选择较低级别（如1-3级）暂时忍耐。但这不代表一直忍，该记录的证据要记录好，等自己“翅膀硬了”，再找合适时机处理。

关键在于，对于工作这类你无法完全主导的事情，建议投入60分的精力即可。做到60分以上往往依赖团队协作，而你又不是团队的主导者，那么过度投入和自责就与你关系不大了。很多人不够强大，项目成功时功劳没有你，项目失败时锅是你的，情绪最低落的也是你。这并不公平。

## 心态调整：格局与自我保护 🛡️

掌握了分级应对的方法后，我们还需要从心态上进行根本的调整。

每到这种时候，总有人会说：“总得有人改变，我不改变，你也不改变，以后就没有改变了。” 但是，我希望你们格局大一点。与其在底层每次跳出来反抗、每次自我受伤，不如**忍辱负重，先努力走到更高级别、更有话语权的位置，再去推动改变**。这样性价比不高吗？不要陷入无意义的自我感动。这个世界、这个社会，甚至你身边的同事、领导，可能都没那么关心你的委屈。

我不希望你们因为善良、负责这些正向属性，反而被职场政治或其他因素利用，让自己深陷情绪内耗。你们当中很多人很有潜力、能力和未来，但一事无成的原因，常常就是过于负责，陷入了本不需要存在的情绪和道德负担中。

就像有人站在道德制高点批判我时，我为什么不关心？因为我不配关心。那么多比我厉害、地位高的人都没负责，为什么要我来负责？人家怎么不自责？搞到最后我自责，为什么？好人就该被枪指着吗？凭什么？

所以，希望大家明白，尤其是对于工作上你无法控制的事情，不要投入过大的情绪，不要反复纠结。你的纠结除了伤害自己，没有任何好处。

## 总结与行动建议 🎯

本节课中，我们一起学习了职场情绪与责任管理的核心方法。

1.  **认知层面**：认识到在工作等非自我主导的领域，“摆事实、讲道理”常因权力结构不对等而失效。
2.  **核心原则**：严格区分事情的“主导权”与个人的“责任感”，不要为无法主导的团队结果过度自责。
3.  **实操方法**：采用“分级应对”策略，根据情境（自身处境、想塑造的形象）在1到10级的回应强度中灵活选择，避免非此即彼的“二极管”思维。
4.  **心态建设**：提升格局，优先追求个人成长与职位提升以获得更大影响力；对于无法控制之事，学会情绪隔离，保护自己的心理能量。

记住核心准则：**能自我主导的事情，全力以赴；不能自我主导的事情，分级处理，60分即可**。将你宝贵的热情和精力，投入到你真正能主导的个人成长、技能提升或副业发展中去。
