# 能力强关系到位也好，什么都没有也罢，无非是走不同的道路---P1---赏味不足---BV1XiJXz

在本节课中，我们将要学习一个核心观点：无论个人能力、家庭背景或社会关系如何，每个人都能找到适合自己的发展道路。成功没有唯一的标准路径，关键在于认清自身条件并选择匹配的策略。我们将通过一个具体案例，分析不同价值观的适用性，并探讨如何在资源有限的情况下起步。

上一节我们介绍了课程的核心主题，本节中我们来看看一个具体的现实案例。

## 案例背景：被“劝退”的起步者

昨天，一位小伙伴向我表达了失落之情。他刚准备迈出第一步，就被一位“大佬”劝退。这位大佬评估了他的家庭背景与个人情况后，认为他作为一个普通老百姓，资源匮乏，准备不足，难以成事，建议他安心工作，不要多想。

这位小伙伴因此感到被浇了一盆冷水，他问我：“这位大佬说得对吗？到底应该怎么看这个问题？”

## 核心观点：没有绝对的对错，只有不同的道路

首先，一个基本前提是：做事方式取决于自身，他人的看法并不重要。每个人都有适合自己的行事方式，不必过分在意他人评价。

所以，大家要找到适合自己的路，而不是一味争论对错。你问我那位大佬说得对不对？

**在我看来，对与错是相对的。** 他的话可能正确，但仅对他所代表的那类人正确。

## 价值观的多样性：以我为例

好，我们就拿我来举例子。

我的价值观是“五五开”。如果以善恶黑白来衡量，我各占五成。我并非一个时刻保持纯粹善良或正直的人。

因此，你们会发现，我的行事方式并非主流或普世价值观中那种“脚踏实地”或“绝对正直”的体现。如果你们能理解我视频的核心意思，可能会认为其内核有“正”的成分，但我表现出的外在不可能“时时正”。

**因为社会本身就不是“时时正”的样子。** 我所表现出的，是我的价值观，也是我所认知的社会模样。同时，我们需要接受其他价值观的存在。

那位小伙伴遇到的人，可能就是我所描述的“九成善、十成善、九成正、十成正”的人。在外界看来，这些人非常“正”。他们的观点是：做一件事必须做好万全准备，必须要有关系、有积累才能去做。

你问我对不对？这话怎么不对呢？但仅仅是对他们而言。

为什么？因为他们可能天生有积累、有家庭关系、父母比我们更优越，或者投胎运气更好。这些我们无从得知。

那么，在我们没有他们那种投胎运气和资源积累的情况下，如果强行采用他们那种“九成白、十成白”的价值观去做事，我们这辈子将寸步难行。

我们这辈子不可能有他们那种“十成”的准备。你说我们要先花一辈子去追上他们父母的积累？这无异于一条死路。

所以你问我是否认可他们？我理解他们。谁不想在做事前有十足把握或成熟的关系呢？

**但我们没有啊。** 在我看来，那位大佬并非PUA，他只是在强行灌输自己的价值观，而他的价值观并不适合我们。强迫我们使用这种价值观，就是把我们逼上绝路。

## 社会的多元化：条条大路通罗马

第三，我以前就说过，社会是多元化的。

什么路都有人走，而且走了之后都能赚到钱，这就是社会有趣的地方。我们举个例子：

*   **产品路线：** 有的人融资六七轮，走正规产品路线，赚到了钱。
*   **关系路线：** 有的人依靠更高的关系链，也获得了成功。
*   **灵活路线：** 有的人在合法合规范围内，通过一些非常规策略（可理解为“坑蒙拐骗”），同样赚到了钱。

他们的价值观和做事方式肯定不互通。但当我们遇见其中任何一个人时，能去评判谁对谁错吗？

无法判断。因为我们不能单纯以“是否赚到钱”作为对错的唯一标准。到底选择什么样的路，取决于我们每个人的具体情况——**你手上有什么牌**。

就像一个一路顺风的人，他一定会告诉你：“做事情一定要有万全准备，你看我是有积累的。” 但问题在于，他一路顺风。

如果是一个一路坎坷走过来的人，他肯定不会这么说。

## 关键警示：避免选择与自身条件严重不匹配的道路

不管怎样，我们千万不要走上这样一条路：**明明自己什么都没有，投胎也是普通老百姓，却非要选择那种看上去“很正、很白”、需要“九成正、十成正”的实打实的道路。**

我跟你讲，真的不是我对你们或对自己没有信心。实在是考虑到2025年的社会上下层结构，像我们这样的人走那种纯粹的“实打实”路线，跟走死路没什么区别。

## 行动建议：如果我是那位小伙伴

第四，如果我是那位小伙伴，我的态度很明确。

**正因为我什么都没有，没有积累，所以我更要去做，现在就要去做。**

而且，我要走的路线，一定不是去找红杉、经纬这类顶级机构融资——我不配，也没这个能力。

我想要的是：先通过一些小项目混关系、混资源，让自己从“没有”到“有”，先混到一丢丢东西。这就是我的目标，我不求大富大贵。

然后，再慢慢地往上混。能混到一些赚钱项目最好，混不到也无所谓，我就当了解社会、接触更多人。

我不奢望融资ABCDE轮，更不奢望IPO，也别跟我谈给社会创造巨大价值——不好意思，以我当下的能力，这不是我能做的事。

但我不会因为“什么都没有”就选择“不做”。因为不做，就更不会有。

那些有能力、有关系、有父母支持的人，他们好好走正规路线去赚大钱，那是他们的路。

**如果我是他，我就走野路子，赚小钱。** 你不要来跟我讲你的大道理，你怎么样，跟我没关系。如果某一天运气好，真的积累到一些真正牛逼的资源，我再去做一个“九成正、十成白”价值观的人也行。但现在不可能。

## 总结与思考

所以你会发现，很多人讲的东西，对不对？对。但它只会对我们造成困扰。

为什么？因为他根本不懂，作为普罗大众、普通老百姓，我们真正面对的困难是什么。

就好像我毕业时，在超市卖水饺，一个月200块钱。如果当时有人跟我说：“你要出去做咨询、做培训？你一没背景、二没技能、三没家庭，你不行。” 请问这话对我有什么用？

除了打击我，让我更难过，没有任何用处。

**因此，当你们听到这种话时，请先掂量：对方是谁？他的各种情况和你是否一样？**

如果他的家庭、成长环境与你完全不同，那么他的话可以参考，但你千万不要把自己陷进去。不要搞得好像还没开始做，就被一盆冷水浇灭了热情。

没必要。
