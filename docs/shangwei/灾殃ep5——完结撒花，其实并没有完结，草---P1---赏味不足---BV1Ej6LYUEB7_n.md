## 环境探索与初始谜题

## 核心追逐战与躲避机制

在熟悉环境后，游戏会引入追逐战环节。玩家需要躲避敌人并利用场景障碍。

核心追逐机制遵循一个简单的**条件判断**逻辑：
```python
if 玩家被敌人发现:
    触发追逐状态
    玩家需要逃跑或躲藏
else:
    玩家可以安全探索
```

以下是成功躲避的关键步骤：
*   观察敌人的巡逻路线（“一步三回头”模式）。
*   利用场景中的掩体（如棺材、墙壁）进行躲藏。
*   掌握时机，在敌人背身或离开时快速通过危险区域。
