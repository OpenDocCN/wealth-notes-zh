# 为什么国内的牛马对加班容忍度那么高---P1---赏味不足---BV1yFk5YtEaE

## 概述
在本节课中，我们将探讨一个普遍的社会现象：为什么国内部分职场人士对加班表现出较高的容忍度。我们将从社会认知、教育背景、个人心理和职场文化等多个角度进行分析，旨在帮助初学者理解这一现象背后的复杂动因。

---

## 社会认知的缺失
上一节我们概述了课程主题，本节中我们来看看问题的核心。许多人将加班容忍度高归因于劳动法监管缺失、企业主压榨或员工逆来顺受。然而，在我看来，问题的根源在于广大民众对社会及自我缺乏清晰的认知。

当个体缺乏认知时，其思想或行为上就会出现一个“缺口”。这个缺口要么由自己填补，要么由他人填补。在中国，应试教育的影响深远，其核心在于鼓励学生遵循既定路径，而非培养独立思考能力。这导致许多人并不真正思考加班的意义，而是被动接受现状。

**核心逻辑**可以概括为：
```
认知缺口 -> 自我填补 或 他人填补 -> 被动接受或扭曲应对
```

## 扭曲的职场情感
理解了认知缺失的基础后，我们进一步探讨由此产生的情感反应。关于加班，许多人的情感并非简单的“反抗”或“容忍”，而是出现了第三种更为扭曲的状态。

以下是几种常见的扭曲心理：

*   **“热爱”加班**：部分人将“内卷”和加班视为人生意义，认为只有通过拼命工作才能超越他人。这种心态常源于家庭或职场长期的PUA（精神控制），例如不断被灌输“家境贫寒，必须发愤图强”的观念，最终异化为“用命去卷”。
*   **“薅羊毛”心态**：另一部分人并非认同加班，而是因为下班后无事可做，或为了享受公司的加班福利（如打车报销、夜宵），选择留在公司。他们认为“有薅不薅是傻瓜”。
*   **从众心理**：当个体都采取上述策略时，便形成了“996”、“007”的集体氛围。每个参与者都觉得自己没有直接影响他人，但实质上都是助推这种文化的一份子。

这并非容忍，而是一种集体无意识的、扭曲的参与。

## 为他人而活的行为模式
这种扭曲心理不仅体现在加班上，也渗透到生活的其他方面。人们的行为往往不是为了实现自我价值，而是为了满足他人的期望。

以下是几个例证：

*   **关于时间**：人们口头上不愿浪费时间，但行动上却可能为了“积累经验”而从事并无长期价值的内耗。
*   **关于婚姻**：不愿草率结婚，却又可能迫于社会压力去相亲。
*   **关于学历**：明知学历贬值，却依然投入巨大的精力去“卷”学历。

在职场中，这种模式尤为明显。许多人努力工作、争取项目经验，表面是为了“更好的晋升”和“未来跳槽”，但实际上：
1.  内部晋升常取决于运气、项目机会和与领导的关系。
2.  在极度内卷、每年有大量应届生涌入的就业市场中，大部分普通从业者到一定年龄后面临淘汰的风险，这与他们早期的“技术经验”关联度可能很低。

**市场现状公式**：
```
个体价值 ≈ f(可用性， 成本， 年龄)  // 价值函数更取决于即时可用性、成本和年龄，而非长期积累的经验。
```

当个体缺乏自主想法，只是为父母、社会价值观、公司或领导而活时，就谈不上真正的“容忍”或“反对”，只剩下被动的“接受”和“浪费时间”。

## 职场中的“逆淘汰”案例
为了更具体地说明上述现象，我将分享两个亲身经历的案例。它们揭示了当扭曲成为常态时，正常反而会显得异类。

我曾遇到两位从外企来的领导，他们都在一年内因“水土不服”而离开。

*   **案例一：鼓励不加班**：一位领导提倡高效工作，明确表示“没事别加班，不用做样子”。结果第二天就被举报，理由是“缺乏团队协作精神”。
*   **案例二：优化代码架构**：另一位领导发现产品代码架构有问题，为了长远“降本增效”而计划重构。这同样遭到举报，理由也是“缺乏团队协作意识”。深层原因是重构会触动现有人员的利益，增加其工作量，影响其KPI和晋升。

这两个案例表明，许多人的工作核心并非为了把事情做好，而是为了**保护自身利益、避免犯错、在内部竞争中胜出**。在这种环境下，讨论对加班的“容忍度”已失去意义，因为行为动机早已扭曲。

## 海外文化的参照
最后，我们简要对比一下海外常见的职场文化，以提供一个参照系。我接触过的海外公司或职场人，无论是华人还是本地人，整体风格更为“随性”。

像“加班容忍度高不高”这类话题，在海外职场很少会成为需要严肃讨论的议题。这本身也反衬出，在国内这成为一个问题，正说明了环境的某种非常态。海外职场普遍遵循“有事做事，无事休息”的原则，例如在日本，加班后同事去小酒馆交流是常见的社交，而非为了“做样子”而留守。

此外，一个鲜明的文化差异体现在责任归因上。我接触的海外学生或职场人，如果事情没做好，通常会就事论事地分析原因，**几乎没有人会将“我还年轻”、“我还是学生”作为预设的免责理由或对未来的担保**。这种“因为A（年轻），所以B（做不好），将来C（老了就会好）”的逻辑，在国内却颇为常见，这本身也是一种不成熟的认知模式。

## 总结
本节课我们一起学习了国内部分职场人士加班容忍度高的现象。我们分析了其根源在于**社会与自我认知的缺失**，探讨了由此产生的**扭曲职场情感**（如“热爱”加班、薅羊毛心态），指出了**为他人而活的行为模式**如何渗透到职场抉择中，并通过案例揭示了职场**“逆淘汰”** 的怪象。最后，通过与海外职场文化的简单对比，我们获得了更广阔的视角。理解这些，有助于我们更清醒地看待职场环境，并思考如何建立更健康、自主的职业观。
