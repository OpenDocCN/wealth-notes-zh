# 课程一：基本盘篇：善良、老好人、感性 - P1 🧠

在本节课中，我们将要学习“基本盘”的核心特征，即善良、老好人与感性。我们将探讨这些特质在现实社会中的实际影响，并分析为何单纯依赖它们可能无法有效保护个人利益。

## 概述

基本盘是指一类普遍存在的思维与行为模式，其核心特点是善良、充当老好人以及依赖感性判断。本节课程将深入剖析这些特点，并解释为何缺乏理性支撑的“基本盘”模式在现代社会中容易导致个人利益受损。

## 基本盘缺乏理性

上一节我们介绍了课程主题，本节中我们来看看基本盘的核心缺陷。

基本盘是没有理性的，从古至今都是如此。不要否认这一点。要真正改变，首先必须学会接受现实。无法改变的人，就是如此。

而且，如果只是普通老百姓，没有家族背景或特殊关系，就越可能具备这种基本盘特性。

## 老好人特质及其后果

以下是基本盘最大的特点——充当老好人。

你会发现，无论是父母还是身边的人，都说这是为你好，那是关系好。这个人帮你推荐工作，那个人帮你介绍对象，这里帮个忙，那里帮个忙。

但当你追问关系怎么好、具体帮了什么忙时，他们往往一问三不知。如果继续追问，他们甚至会说“人家愿意帮你就不错了”。

国内的基本盘从小到大都是这样被教育过来的，从上一代到你们这一代，甚至可能延续到下一代。但这种模式必须改变。

因为在社会上，老好人只有一个结果：**吃亏**。

根据我的人生经验和观察，可以得出一个结论：好人往往没有好下场。你们可以不信，也可以反驳，但请记住这句话。未来某天，你们自然会明白。

基本盘，尤其是父母辈的逻辑，教育孩子宁愿自己吃亏，也要对别人好。即使别人欺负到头上，也要笑脸相迎。在我看来，这很奇怪。

## 利益的唯一保障：法律

口口声声说要保护自己利益，但真遇到事情时，老百姓的第一反应往往是求助于人情、关系或感觉。

然而，这个社会能保护我们利益的，客观来说，只有一个东西：**法律**。就这么简单。

这意味着，我们需要的信息和依据，必须在法律层面有意义、有用。不要谈感觉或关系，这些在法律上没有依据，也没有意义。

你们可以去公证处了解一下，哪些东西是可以公证的。感觉有用吗？人情有用吗？元气有用吗？感性有用吗？老好人有用吗？没有用。这三个东西与法律毫无关系。

## 实例分析：介绍工作

例如，父母说有人帮你介绍工作，声称是“好工作”、“国企”、“能学到东西”、“不会吃亏”，还叮嘱你要多请人吃饭、送礼。

在我的观念里，这些都是空话。如果介绍工作，我只关心几点：
1.  **岗位名称**：正式的岗位名称是什么？
2.  **合同条款**：合同怎么签？
3.  **薪酬待遇**：合同上的金额怎么写？薪酬包（package）如何构成？
4.  **流程**：面试流程是怎样的？

那些“能学到东西”、“不会吃亏”的话太虚了。有本事就把“学到什么”、“保证不吃亏”写进合同里。没本事就别说了。

你会发现基本盘很奇怪：口口声声说要维护利益，但真到维护利益时，却又退缩了。

如果真要“帮忙”或“走关系”，那关系应该体现在：**能直接签合同**、**减少面试流程** 或 **提高薪酬**。例如，本来年薪30万，走关系后变成40万。

如果不清楚岗位、合同和薪酬，这算什么帮忙？大概率，你们的父母甚至你们自己都不知道要关注这些。他们可能觉得关注这些是“不近人情”、“社会上会没朋友”、“混不下去”。

但问题来了：我们需要那么多朋友干嘛？能否在社会上立足，取决于**利益关系**，而不是人情或感觉。活在世上，“近人情”有什么用？我同情别人，别人就会同情我吗？我对别人好，别人就一定对我好吗？

如果不关心薪酬、合同和实际岗位，如何保护自己的利益？最后只能吃哑巴亏，甚至还得碍于所谓的关系陪着笑脸。在我看来，这毫无必要。

## 实例分析：商业合作

第三点，这可以称之为“封建”思维。

以前有人给我介绍商业合作，说某公司或教授团队有项目可以一起赚钱。我说可以，于是开线上会议，介绍项目、合作方式和盈利模式。

然后对方开始不断向我要各种方案、文件和案例，甚至要求我现做。看在以前合作过的份上，我做了两个。但对方没完没了。

于是我直接问：“你一直问我要，你觉得合适吗？我们要么跟教授签合同，要么让他付钱。否则项目拖到什么时候？我被你白嫖没关系，但我们不能被教授白嫖。”

你们知道合作方怎么回复吗？一个二十八九岁的人说：“我们要跟教授搞好关系，不能催他，教授很忙，关系搞僵了不好。” 我从此再也没和这个人合作过。

你们一定要明白核心逻辑：社会上的人和事，不是你跪得姿势多好，或礼送得多到位，就一定有用的。社会早就不是这样了。

## 普遍的逻辑困境

第四，根据我的沟通经验，95%以上的人逻辑都差不多。

无论是父母、导师、领导、朋友还是合作方，当我问及社会意义或法律意义上的具体问题时（如职称、合作合同、项目文件或新闻链接），他们基本都不知道。反而会说“对方地位高”、“关系好”或“有项目可做”。

一旦追问具体细节，就一问三不知。

基本盘很奇怪，几乎所有人内心都默认排斥谈论这些具体问题。他们觉得问合同“感觉不好”、“伤感情”。但如果不问，你打算什么时候问？还是永远不问？如果不问，如何保证你的利益？

“感觉”对方不会坑你，“感觉”有用吗？如果感觉有用，你怎么没去清华？怎么没成为首富？

所有这些自我标榜善良、老好人、感性的人，觉得不能跟别人谈直白的东西，怕别人不高兴——这种人就是传统意义上的“软柿子”。别人不捏你捏谁？捏了你还不敢吭声，怕搞僵关系。

在我看来，人与人是平等的。要搞好关系，可以，但前提是**一起赚钱**。空谈关系毫无意义。

## 总结

本节课中我们一起学习了“基本盘”的核心特征：善良、老好人与感性。我们分析了这些特质如何在缺乏理性支撑的情况下，导致个人在社会中利益受损。关键在于，保护利益需要依靠**法律**和**明确的契约（合同）**，而非空洞的人情与感觉。改变思维，关注具体、可衡量的条件，是摆脱“基本盘”困境的第一步。
