# 千禧年梦核：大家一起梦回千禧 - P1 - 赏味不足 🎮

在本节课中，我们将跟随游戏实况，一起探索《千禧年梦核》这款充满千禧年代怀旧元素的游戏。我们将了解游戏的基本操作、环境氛围以及其中蕴含的独特文化符号。

## 游戏初始设置与界面 🎛️

上一节我们介绍了课程概述，本节中我们来看看游戏的初始设置与界面。

游戏开始前，首先调整了镜头晃动和鼠标灵敏度设置，具体数值为**0.5**。游戏画面中充满了千禧年代的经典元素。

以下是游戏开始时观察到的部分怀旧物品：
*   弹子跳棋
*   闹钟
*   存钱罐

游戏角色设计为马赛克风格。电视中播放着《葫芦娃》和《西游记》中“孙悟空大战二郎神”的片段。

## 基本操作与初期探索 🕹️

了解了初始界面后，本节中我们来看看游戏的基本操作与初期探索。

游戏的核心操作之一是使用相机。按**鼠标左键**可以进行拍照。游戏初期，玩家在家中醒来，需要找到桌上的20元钱出门。

出门后，场景转变为典型的2000年前后无电梯居民楼小区。游戏提示需要“找娃”。在小区里，可以看到自行车棚和“新世纪小区”的标语。

## 购买任务与场景切换 🛒

完成了初期探索，本节中我们来看看一个具体的购买任务如何推动场景切换。

根据对话提示，玩家需要前往小卖部用20元钱购买“脑黄金”。小卖部旁有装修的噪音。与店主对话后，花费20元获得了脑白金。

随后，场景切换至“外公家”。在此处获得了一个手电筒和一把扇子，背景音乐也随之改变。电视中播放着诡异的画面。

## 探索“外公家”及诡异楼道 🚪

拿到关键物品后，本节中我们进入“外公家”及后续的诡异楼道进行探索。

“外公家”的布置不同寻常，桌上有一张陌生人的照片。从窗户向外看，场景变得超现实。离开“外公家”后，进入一个灯光闪烁、结构异常的楼道。

楼道中有绿色和红色的“安全出口”标志。选择红色出口后，会进入一个循环空间，墙上出现不应存在的投影窗户。最终通过绿色出口离开。

## 商业街探索与更多怀旧元素 🏬

离开了封闭的楼道空间，本节中我们来到一个充满年代感的商业街进行探索。

场景变为“东方广场购物中心”，时间显示为2002年（马年）。街道两旁是各种清仓甩卖的店铺招牌，例如“时尚文胸”、“江南皮革厂”等，但空无一人。

以下是商业街中出现的部分怀旧物品与场景：
*   需要充点卡的网游广告
*   老式台式电脑
*   小霸王游戏机
*   拍大头贴的机器
*   公用电话亭

玩家在此获得了一盘磁带和录像带。

## 进入异常空间与最终循环 🌀

收集了部分物品后，本节中我们进入一个更加异常和诡异的空间。

通过一个电视屏幕般的入口，玩家进入一个装饰风格奇特、配色诡异的家庭场景。随后又进入一个“书法培训”房间。

最终，玩家通过一个“浴池”入口，进入一个红色调的异常空间，内有天气预告电视和许多马桶。该空间是一个循环，最终将玩家送回到商业街起点。

本节课中我们一起学习了《千禧年梦核》游戏初期的流程。我们操作角色探索了充满千禧年元素的怀旧场景，完成了购买任务，并逐步接触到游戏超现实和诡异的一面，包括循环的楼道、空无一人的商业街以及最终的红色异常空间。游戏通过环境氛围和物品细节，营造了一种独特的“梦核”体验。
