## 核心方向预览

### 1. 不转行篇 🔄

![](img/eda6815bc7993476bf128000d4cecc11_27.png)

对于很多人来说，当下情况不佳，转行也不现实。不转行篇将探讨如何在现有领域内深耕和发展。

以下是“不转行”方向下可以细分的几个路径：
*   **测试开发 (SDT)**
*   **手工测试**
*   **自动化测试 (Automation)**
*   **性能测试 (Performance)**
*   **质量保证 (QA)**

![](img/eda6815bc7993476bf128000d4cecc11_29.png)

![](img/eda6815bc7993476bf128000d4cecc11_31.png)

### 2. 跳槽篇 🚀

![](img/eda6815bc7993476bf128000d4cecc11_33.png)

如果考虑更换工作，跳槽篇将提供相关指导。

![](img/eda6815bc7993476bf128000d4cecc11_35.png)

以下是“跳槽”时需要关注的两个方面：
*   **面试技巧**
*   **职业规划**（包括作为“老人”或“新人”的不同策略）

### 3. 兼职篇 💼

如果既不想转行，又不想在现有公司“躺平”，可以考虑利用技术做兼职。

![](img/eda6815bc7993476bf128000d4cecc11_37.png)

以下是“兼职”可以尝试的几个方向（开发与测试人员均可）：
*   **技术顾问**
*   **培训讲师**
*   **技术咨询**
*   **其他自由职业**

![](img/eda6815bc7993476bf128000d4cecc11_39.png)

![](img/eda6815bc7993476bf128000d4cecc11_41.png)

### 4. 转行篇 🧳

![](img/eda6815bc7993476bf128000d4cecc11_43.png)

转行是很多人考虑的选项，但风险较高。本系列希望能给大家一些启发。如果下定决心，可以找我进行收费咨询。

![](img/eda6815bc7993476bf128000d4cecc11_45.png)

![](img/eda6815bc7993476bf128000d4cecc11_47.png)

以下是“转行”的两个可能路径：
*   **跨行业，但岗位不变**：转换到不同行业，但继续从事开发或测试工作。
*   **跨行业，且岗位转变**：转换到不同行业，并从事其他岗位，例如产品经理、销售等。

## 总结与建议

本节课中我们一起学习了在当前行业背景下，开发与测试人员可以考虑的几大职业发展路径。

总体而言，大前提是近两年行业情况不佳。我在其他平台也回复过，目前最好的方式之一可能是去读书（如攻读硕士或出国留学），以度过这段困难时期。

现状是竞争越来越激烈。当岗位稀缺时，内卷的空间也变得有限。所有从业者，包括开发和测试人员，都需要面对一个现实：随着年龄增长，以及互联网行业整体下行，必须为未来做打算。现在不考虑，未来会更被动。

我的分享并非贩卖焦虑或唱衰行业。行业的发展不会因个人言论而改变，其本质是正处于下行周期。因此，我们才需要认真考虑这些应对策略。
