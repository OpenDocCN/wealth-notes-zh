

# 怎么知道自己适合干什么？

241125 野心范

整理：公众号懒人搜索，懒人专属群独享

懒人微信：lazyhelper

![](bbox=[60, 152, 825, 274])

我在大学以前，是从来没有认真想过这个问题的。

小时候我看星星月亮时，倒是想做一名科学家，没别的原因，就是出于好奇而已，因为在那时看来，科学家是无所不能的大神，什么都懂的，是可以深入到月亮或者太阳内部，摸一摸它们到底是什么东西，到底有多烫手的。

即使到了高考选专业时，我也不懂自己以后想干什么，一则是因为没有阅历，什么都不知道，天知道自己想干什么；二来是家里就没有一个懂的人可以给我咨询，没有有见识的父母所提供的指导。

所以就看看自己的分数，看看学校哪个专业的名称听起来高级一些，就这么填志愿了，所以至今我都觉得，普通家庭的孩子太亏了，亏死了，亏出血了，连个人问都没有，一点规划意识都没有。

这也是我写微博的一点初衷，可以分享一些经验给需要的人。

但是到了大三这样，我开始意识到，我他妈的错了，错出血了，人家有些家庭是有预谋的做事和规划人生的，我们这种粗野式的发展是要完蛋的节奏！

那时我们已经开始去工地见习了，我到了几百米深的地下，感觉尼玛的这不是工作，这是要命啊，于是心里想，我虽然不知道我现在要干什么，但是：

起码我知道我一定不要干什么！

这一点很重要，你首先要知道，你真的特别抗拒什么，你就不要去做那个领域了。

比如有些人很受不了一天对着电脑画图写代码，就是不太坐得住的那种多动星座，那就不要搞技术了，有些人喜欢宅家里，打死都不愿意多说几句话的，觉得去拉业务好low的，那就不要去搞销售和协调类工作了，不然真的觉得超级痛苦。

那排除了自己不想干的事情，怎么知道自己适合干什么呢？

我当时就想了好长一段时间，估计持续一两年吧，其实也会有很多怀疑，反反复复的，因为没有实践验证过，所以这个心理过程是很正常的。

那就回归到最终极的问题上来，我到底想做一个什么样的人呢？

不说人生梦想，我们就讲生活理想，你到底是想过上什么样的生活呢？

这是一个很实际的问题！

当时我家里的情况很糟糕，帮不上我，还负债，有很有钱的叔父辈，但没有一个愿意出手相助的，那段时间是家里最难过的时候，我的心灵也感受到了这世界最深的恶意，心里有一口气憋着。

那没有什么可想的，我缺钱呀，我要出人头地呀，我要做收入高的工作，要做有爆发性的工作！

说白了，就是我能忍一段时间，积累出一个超级爆点，我想好好干几年，然后发财！

家里没有人当大官，想从政，是漫长而艰辛的，可能能力强，运气好，也要到40岁才能混个实权的位置，才能有资源有大钱。

我真的等不了那么久。

那就搞商业吧！

而商业离钱最近的环节，就是销售了。我销售的想法起源，就是这样的背景，就是单纯的想赚钱，想改变命运，又没有其他更好的路径。

所以，在你不知道你要干什么的时候，你首先排除掉那些你绝对不想干的事情，接下来就是细数你的想法。

你到底想要一个怎么样的人生，你迫切的想要在哪方面取得成绩？

想赚钱，还是不缺钱但是想去搞搞政治？还是想搞搞艺术？

搞搞科学？

这才是你一切的出发点，因为你的人生，就是为了这个而活的呀，或者说，你目前为某种大难题所困的话，你是要解决他的呀，那你只能从这里出发，想想什么工作是接近你的理想的。

你的理想就是你的出发点，不要为了其他的旁支原因而跑到其他赛道去。比如你本来应该做业务线条的，结果因为技术线条底薪高了 500 块，你就去技术岗了，这是大错特错的，是有违初心的，是浪费时间的，是自欺欺人的。

一个行业你是否做得好，主要看的就是这点，它做好了，是不是能满足你的人生理想！

这点明确之后，就是考虑自己是否做得来的问题了。

其实我一直认为，我就是个内向的人格，我小时候也是个软弱的性格，我也不是特别喜欢交际和热闹的人格，我居然做了大客户销售？

其实跟大家想的相反，真正生意做得好做得大的老板，或者说官做得大的领导，都不太是那种看起来外向的性格，这是很反常识的一点。

也就是说，只要你价值观上不反感这个工作，你就是可以把它做好的。

就比如说，我们大客户销售行业，它就不是一个吃喝嫖赌吹牛逼就可以做好的行业，不管什么性格的人，都有做得特别好的，因为它需要的本质上不是一个人的性格要怎么样，而是你怎么去做这个事的问题。

举例，假如你显得闷，但是我告诉你，要勤打招呼，打招呼要响亮，晚上或者周末要约客户出来吃个饭，某个问题用什么话术去跟客户讲，这些都是方法呀，跟一个人的性格没有关系。

你只要去做了就行了，虽然体现出来的风格会不一样，但是效果是差不多的。

而且，你是什么性格，你就会吸引什么样的人，同样会有“粉丝”。

我当时想了很多，不是太确定自己要做销售，我骨子里是个书生文人，做销售，好像有点羞耻。

后来，我又问了宿舍的人，身边的人，他们都说，我就适合做销售，说我平时能说，有想法。

他们这种反馈，很朴素，就给我个“能说”的概念，就认定了我适合干销售。不过正是这种外人对于自己不假思索的评价，可能真正的反映了自己的特质。

因为你往往没有别人了解自己，或者说，你认识的自己，是你思考过度的自己。

很多时候，你并不是你认为的自己，别人说你是什么样的人，你就应该想想是不是有道理。

这是一点很重要的发现。

因为在别人看来，你是什么样的，这是你的特质，很大程度上反映了你过往十几年所关注和学习的东西。

天才在于积累，聪明在于勤奋，这十几年的关注和养成，就是你这方面的积累所在，所以你大概率就是应该往这方面发展的。

总的来说，你首先要回归内心，想想自己真的要什么，然后打量一下自己，问问身边人，觉得你适合什么，你就干什么。

这两者重合的话，你一定能做好，如果冲突的话，那就理想导向，听从你内心的声音，而不是别人的评价。

以上才是重点，其他的都不重要，如果你觉得重要，可能你不够干脆，不够狠心，估计大概率要后悔！
