

# “她势力”的崛起

每一种思潮都有它的受益者和受害者。在《风向真的变了！！》中阐述了一个事实：随着社会产业的发展，市场经济必然会导致人群的迁徙，人群因为流动和自主选择而产生分化，最终造成社会基本结构的失衡。在一个失衡的社会结构下，以前天经地义的事情，就会变得难以企及！

置身其中的芸芸众生，往往难以理解这一点。只会根据个体的感受，把那种难以启齿的欲求不满，归因到自己能够接触的个体身上！乡下结婚难是因为彩礼高么？其实并不是。你猜她们为什么要这么高的彩礼还有人愿意给？就是因为供需失衡导致的物以稀为贵！供需失衡的问题解决不了，不管用什么办法去压制高彩礼，都会变成按下葫芦浮起瓢的状况。这就好比疫情时的上海，为什么送外卖的人能够接到 1000 块钱一单的跑腿业务？就是因为人家家里确实有人生病了，出不去又买不到药。如果不能解决这个供应问题。不管花多少精力去打击那些跑腿的，不许他们接这样的单，依然会屡禁不止。这对于送外卖和跑腿的群体来说是个挣钱的捷径，他们平时送一个月还没这一天挣的多。他们就算使出浑身解数与你斗智斗勇玩猫捉老鼠的游戏也不会放弃这个挣钱的捷径！

基于同样的道理，乡下光棍多适龄女性少，这种供需的失衡问题不解决，即便打击彩礼，也会以另外一种方式来完成同等价位的财务转移！比特币不合法，但他们依然想出了合法的方式利用比特币把国内的资金转移了出去数万亿。譬如说，不交易比特币，但是合法购买电子钱包。只不过用一笔很大的资金去买一个电子钱包。虽然让人匪夷所思，但合乎法律流程。我不能买你的夜明珠，但我找你买个盒子总可以吧？我用巨款买你那个盒子，虽然显得有点傻，但也是合法的行为。我买了你的盒子之后，你在我的盒子里放了一颗夜明珠，然后在国外把夜明珠卖掉。这就间接完成了资金的转移！

就算打上法律补丁，禁止交易盒子。那也会想出另外一种方式。以前有些香港商人行贿的方式就很别致：随便拿一个极不值钱的瓷瓶作为礼物送给一个官员。几十块钱的东西，对方就算收了也不违法。然后找个专家证明那是“文物”，再花高价把这个“私人收藏的文物”买回去。如此一来就把钱送出去了，而且走的是正规流程。有一些非常高档的商场里会有一些卖高档工艺品的店，但一年到头你看不到它有几单生意。有时候你会很纳闷：它都没什么生意，如何在这么贵的地方长期存在下去？因为这些店里面那些非常值钱的工艺品并不是用来卖的！而且这些店的存在本身就不是为了做生意……

扯的有点远了，回归主题：任何一种思潮的泛滥，一定存在广泛的群众基础。对自身的处境和未来感到绝望的人越多，就越会追逐那些流行的极端思潮。所有宗教信仰特别虔诚的地方，是不是都特别的愚昧落后和贫穷？与之同时他们又会莫名其妙的感到骄傲！为什么会出现这样的情况？越是在现实中感到绝望的人群，就越需要一种极端的精神支撑。否则如何让这些人去面对日益严峻的现实困境呢？

你看那些念经的地方，不但贫穷而且特别的愚昧和残忍。这种愚昧保守残忍和封闭，恰恰是一种能以极低的成本维持住当地社会秩序的有效方式。富裕的家庭里面才有包容和尊严，贫穷的家庭里面没有办法做到包容和尊重。因为无力支付试错的成本！在做所有的生活决策的时候，都会本能的选择低成本的方向。但是这种低成本的选择往往又会显得非常的保守愚昧和残忍，甚至会导致更糟糕的结果！最终就形成了一种闭环的宿命。

大城市里面的女生特别的恨男人不上进。舆论上把男性骂的一无是处，但现实中的相亲局参与者有90%都是女生。舆论场是现实的反向放大器。为什么会出现这样的反差？因为淤积在大城市里面的大量普通家庭的女生，真的没有办法靠自己的力量在大城市长期立足。随着时间的推移，能够冒尖的是万里挑一的少数，绝大多数女生在职场中连中层都升不上，只能淤积在基层。即便是在体制内和事业单位也是呈现这样的分布——在过去的20年中，医疗和教育领域所提供的事业编岗位接纳了数千万的女生。然而，这些岗位正面临着压缩和裁撤！至于体制外的岗位，随着经济周期的变化，岗位日益减少，竞争持续加剧。男性都存在35岁的职场瓶颈，女性的职场瓶颈只会更加的严峻。一个80后的女生和一个00后的女生去抢一个同样的基层岗位，显然企业更愿意用那个00后的女生——因为她更好骗也更便宜！

这种迫在眉睫的现实压力和自然而然的职场代谢，会让许多遭受职场瓶颈的人，失去保证自己在大城市里生存下去的收入来源！许多混不下去的大龄男性在被竞争环境和岁月双重淘汰之后，主动选择退回到家乡。但对于许多人淤积在城市里的女生来说，既没有退路，也没有未来！恰恰是现实中的这种绝境，会让她们对自己的人生感到无比焦虑。如果婚姻能够改变这种绝望的处境，对她们来说就是一种拯救——这恰恰是为什么舆论场上一边倒，但在现实中又呈现向另外一边倒的原因。她们希望在婚姻中能够找到的对象，是能够将自己从看得见的绝境中拯救出来的盖世英雄。然而，绝大多数社会中下层男性都不符合这个条件。而符合条件的少数男性又拯救不了这么多！所以她们才会痛恨那些普通男生不上进。

事实上，这是人群分布的结构性失衡。现实中的生活成本水涨船高，而这种水涨船高本身就是一种社会筛选的方式。那些被筛选掉的男性回退到了自己出生的小地方。只不过，同样被筛选掉的女生，不愿意回退到自己出生的地方。她们就是为了逃离那种让自己感到窒息的地方，为了让自己有更多的自主权和自由度，才拼命的试图留在大城市。然而，现实中的处境和竞争趋势又不断戳破她们内心的这种渴望。而且她们面对于这种残酷的现实，往往无能为力！越是感到绝望，就越会滋生出恨意。只要这个群体随着时间的推移变得越来越多，那些将恨意拉满的思潮就会像野火燎原一样越来越旺——任何方式都阻止不了！这就像当年西藏的拉藏汗试图推行世俗化，最后彻底被宗教势力打败了一样。只要那个地方的生存环境无法改善，绝望的人群对宗教的热情只会越来越高涨。拉藏汗无力改变当时西藏的生产力和生存环境，所以他的世俗化政策，最终敌不过宗教安慰对绝望人群的吸引力。

虽然舆论场在不断的渲染“她势力”的崛起，然而，这不过是另一种形式的宗教思潮而已。这就如同那些恢宏的建筑和先进的战机，能给底层的人群带来一种虚假的力量感一样！随着产业的新旧更替，那些没有文化也跟不上时代的人群，处境只会越来越糟糕。他们在现实中既没有退路也没有未来。然而，他们依然可以在潮湿地下室中的单间出租屋里，通过手机观看视频的方式来感受到祖国的日新月异。这些视频中所呈现出来的强大与先进能够赋予他们尊严和力量，从而让他们暂时的忘却自己在现实中寸步难行的生活！
