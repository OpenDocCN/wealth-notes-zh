

# 姜萍事件水落石出，锅到底是谁的？

241105 文/卢克文工作室嘉宾 星海舰长

整理：公众号懒人搜索，懒人专属群独享

11月3日上午，阿里巴巴终于对外公布了2024年阿里巴巴全球数学竞赛获奖名单。

意料之外似乎又意料之中地，获奖名单没有今年6月份轰动一时的“中专天才少女”姜萍。与此同时，涟水中专也发布通告，宣布王闰秋在考试中对姜萍进行了指导，违反比赛规则，对王闰秋进行了处罚。

至此，“中专少女数学竞赛逆袭”事件，终于尘埃落定。

这件事的反转早有预兆。

正在挺姜派和反姜派吵得一地鸡毛的时候，6月22日，2024阿里巴巴全球数学竞赛决赛正式结束，成绩定于8月公布。但是，8月过去了，没公布，9月过去了，没公布，10月过去了，还是没公布。

与此同时，姜萍彻底消失了，新学期开学后，姜萍未参加学校报到注册，就连姜萍的家也人去楼空，院子长满了杂草。

主流媒体，涟水中专，达摩院不约而同地悄悄删除了姜萍的视频。

这么多线索串联到一起，已经很说明问题了，最后结果的公布，只不过是把这件事敲下了最后一颗钉子罢了。

我们也许可以从阿里巴巴的数学初赛说起。

初赛是一场 48 小时的线上赛，虽然比赛要求独立完成，但在这 48 个小时里面，你总要吃饭、上厕所，睡觉的吧。而且监考也没法通过摄像头监控那么多人，所谓的“禁止与他人讨论”基本形同虚设。

王闰秋作为一名数学专业的老师，对这个比赛是很有兴趣的，他已经参加三次了，所以对于这个比赛 bug，也是心知肚明的。

于是，他采取了一种作弊手段：多账号答题。

为什么这么玩呢？因为在预赛是开卷，理论上可以一个人答题，然后把答案给其他人，只要自己水平过硬，就能把其他人带入决赛，成为自己的“炮灰账号”。

而到了决赛，情况就不一样了，决赛共分五个赛道，分别为：代数与数论、几何与拓扑、分析与方程、组合与概率、应用与计算数学。选手需要选择一个赛道才可以进入答题界面（赛道一旦确定，不可改变）。

每个赛道共六道题，必须在开考日的任意8小时内连续完成答卷。

这个时候，事先准备好的进入决赛的炮灰账号就有用了，让他们在开考日0点登录炮灰账号，让炮灰账号分别查看5个赛道的题目（或者炮灰账号都选同一赛道，查看单一赛道题库的题目），记下来，提供给主账号。

这样一来，主账号就额外得到了16个小时来提前线下答题，还可以根据难易程度，选择自己把握最大的赛道，等选定赛道的题都做出来之后，主账号在最后8小时登录，写下来自己提前解出来的答案，拿到高分。

为什么我们可以肯定王闰秋多账号答题了呢？我们看看去年的预赛成绩就知道了，他和妻子的分数是一样的，名次也紧挨着，分别是432名和433名，这可能是把同样的答案，用两个账号提交了两次。

曾经有网友对进入决赛的所有参赛者的学校和成绩作了聚类分析，发现一个有意思的现象，同一学校得分相近，不同学校则相距较远。这说明什么问题？说明很多人都搞了这种“团队答题”，很可能有不少人手里都有拿同学身份证注册的“多账号”。

去年的时候，可能王闰秋的账号不够多，在决赛中并没有拿到所有的赛道考题，导致决赛失败。所以今年他需要更多账号。

一开始，他找到了自己的好友，拿到了账号，加上自己妻子的账号，也才3个，剩下的账号怎么办？只能从自己的班里物色有数学基础的学生。

我们不知道王闰秋找了几个人，但姜萍一定是其中一个。

姜萍这个人，我们不能否认其对数学有一定的兴趣，否则王闰秋肯定也不会找她。而且通报中说接受了老师王闰秋指导，而不是由老师提供答案，说明姜萍也是有一定数学水平的（只不过肯定没有传说中那么厉害）。

但是在关键的一道题上，可能王闰秋也犯了难，因为他可能算出了不一样的答案，自己也不知道哪个是正确的，索性把不同的答案给了不同的人，增加成功率。万万没想到的是，他给姜萍的那个答案才是正确答案，这也就是为什么姜萍12名，王闰秋125名的原因。

结果就是这样一份结果，把达摩院给震惊了！

说实话，之前不是没有大专中专生进入决赛，但名次这么高的，绝无仅有！

阿里数学竞赛已经办了几届了，虽然在圈内有一定名气，但一直没有“出圈”，需要一个舆论事件引发全社会的关注，千金买马骨，遂有壮士来，需要吸引更多的人来参与这个竞赛，提高竞赛的含金量和知名度。

于是乎，赶紧成立一个采访报道组，赶往涟水采访王闰秋和姜萍。

对这种情况，王闰秋和姜萍也是措手不及的。

王闰秋能说自己多账号答题吗？

肯定不能啊。

那怎么解释姜萍成绩比自己好呢？那只有说孩子有天赋这一条路了。

我们可以试想一下，一个17岁的中专女生，涉世未深，面对突如其来的采访，一定手足无措，那她会如何呢？父母不在身边，唯一可以依靠的，不就是老师了么？老师说什么，就是什么，哪怕姜萍面对镜头说的那些话，也很可能是王闰秋给她的。

注意，此时姜萍对于事件可能带来的巨大影响，是没有一点概念的，自己也不知道这样做对不对，但老师说对，那就听老师的吧，于是就这么出镜了。

没想到，视频放出，一下子火了！

17岁农村中专生逆袭，击败清北数学系，完全是个爽文好么？这绝对是底层占大多数的社会喜闻乐见的，深深切中了大众的爽点，舆论一下子爆了。

然后轮到主流媒体上了。

现在很多主流媒体的融媒体部门，都是流量至上，各种粉丝量、阅读量、转赞评等等指标会列入 KPI 考核，看到个热点，谁不想扑上去吸两口血？于是一众主流媒体也扑到了这场造神运动中，姜萍就这样被捧上了神坛。

然而，假的就是假的，质疑很快就来了，因为破绽实在太多了。

可能是因为阿里视频摄制组大多是新闻专业人员的原因，在视频中出现了很多违反常理的事情。比如展示姜萍一遍翻看英文字典，一边学习英文数学教科书，看似很励志，但问题在于，这个书有中文版啊！何必再看英文版呢？

还有板书的过程，姜萍把十几处的数学符号都写错了，就这都没人看得出来，视频就这么播出去了，所以很多专业人士表示质疑。

但对媒体来说，不管正面反面，有流量就是好事，在各种媒体和平台的推波助澜之下，全民都陷入了一场关于姜萍的巨大争议之中，有人去网暴浙大等高校，要求立刻录取姜萍，也有人和阿里达摩院负责人对线，生生把这场闹剧变成了一个全社会的现象级事件。

对此，其实达摩院是最慌的，质疑声起来的时候，达摩院就一定赶紧去核实了，得出的结论自然对自己不利，但在热度最高的时候站出来说姜萍作弊，必然成为众矢之的的。这也是达摩院选择过了3个月才公布真相的原因。

达摩院同时也宣布，“组委会将认真听取各方的批评和建议，吸取教训，优化规则，让竞赛活动更加规范。”事实上承认自己的组织有问题了，至此，姜萍事件才最终尘埃落定。

## 2

姜萍作弊的新闻爆出之后，姜萍瞬间成了众矢之的，迎来了一场凶狠的舆论反噬。

无数把她夸上天的媒体，第一时间光速切割，并炒作“姜萍作弊”的新闻，又赚了一波流量。

无数曾经力挺姜萍的人，秒变受骗者，对姜萍破口大骂，仿佛她才是这一切的元凶。

姜萍有错么？有错。

她不该违反比赛规则，不该听老师的，也不该在接受采访时撒谎。

但问题在于，她毕竟只是个17岁的孩子，从她接受采访的状态就可以看出，绝对是那种涉世未深、见识短浅的孩子。

咱们将心比心，你我 17 岁的时候遇上这种事情，面对老师的要求，面对添油加醋只为提高招生率的学校领导，面对一门心思想搞个大新闻的记者，会怎么做？一定能做出完美正确的判断？

等她发现事情闹大的时候，已经骑虎难下了。

姜萍有错，但让整个事演变成一个上升为“逆袭”“突破阶级”和“天才还是骗子”的巨大社会事件，又是谁的错？

如果说姜萍的错有 10%的话，那么她的老师王闰秋，起码有 20%。

王闰秋首先就不应该作弊，但这不是他的最大错误，毕竟搞“团队答题”的又不止王闰秋一个人，当很多人都在犯错误的时候，没必要揪出一个人大加挞伐。而他最大的错误，是不敢承认自己作弊，为了自己的考博前途考虑选择了撒谎，还说自己怎么培养了姜萍。

而达摩院呢？起码占到了 30%。

短视频脚本，是谁写的？短视频审核，是谁做的？短视频发布，是谁发的？给姜萍站台的人又是谁？能在达摩院混的，都是人中龙凤，在宣传姜萍之前，组委会、公关部、视频拍摄团队，就没一个人质疑为啥一个中专女生这么厉害吗？或者说，达摩院本来就心知肚明，但为了流量和曝光度，有意装糊涂呢？

而各种推波助澜的媒体，责任无疑更大，占到了40%。

值得注意的是，这件事第一波冲上去吸流量的，并不是自媒体，而是主流媒体。

按道理来说，主流媒体应该更为严谨，底线更高，但是事实证明，占据舆论高地、掌握舆论权力的他们，不怕事后追责，所以行事肆无忌惮。

在整个过程中，主流媒体先是对单一信源不核实不质疑，只管转发，不管检验。哪怕知道自己是学新闻的鉴定不了数学，但起码也找个人问问啊！结果呢？就是傲慢地不问，而且对各种破绽视而不见。

面对公众质疑的时候呢？只会装死，也没有尝试去调查的勇气。等结果最终出来，光速切割，反戈一击，仿佛他们什么责任都没有一样。

为什么媒体会选择姜萍？因为姜萍话题有流量。

说实话，在当前社会机会少的情况下，大众对于阶层固化是相当焦虑的。而在底层民众中，也有大量的人群对自己的境遇不甘心，他们相信自己是怀才不遇，抱怨中国教育制度不公，抱怨自己没有上升渠道，积压了太多的怨气需要发泄。而姜萍，就是那个承担了底层怨气的符号。

在底层眼中，姜萍就是底层的野草，但野草也有锋芒。

姜萍就是被教育制度埋没的天才，有第一个就会有第二个。

姜萍就是中专生大专生逆袭的偶像，姜萍行，我也行！

如果你说姜萍是假的，他们会跟你急眼！说你歧视底层！

然而，这种汹涌却缺乏理性的情绪，往往会被资本和媒体收割的最佳目标，为了流量，媒体们不介意去造一个神。

就这样，底层相信神，媒体喜欢造神，还有人通过捧神得到利益，最终，造神成功了，所有人都得到了想要的，却忘记了，假的始终是假的。

姜萍事件结束了，却又没结束。

只要心有不甘的群体，只要有心存侥幸的阴谋者，只要有不谙世事的孩子，只要有唯流量是图的媒体，只要有推波助澜的吃瓜群众，那么今天一个姜萍倒下去，明天还会有更多的姜萍们被捧起来。

然后，再重重摔到地上。
