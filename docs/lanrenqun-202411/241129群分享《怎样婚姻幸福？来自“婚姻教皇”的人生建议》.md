

# 怎样婚姻幸福？来自“婚姻教皇”的人生建议

今天我们说说婚姻大事。最近，眼看着到年底了，有很多跟婚姻有关的数据相继出炉。

比如，前段时间（11月3日），民政部官网公布了《2024年3季度民政统计数据》。统计显示，今年的前三季度，全国的结婚登记数是474.7万对，离婚登记196.7万对。

再比如，国家统计局官网发布了《中国统计年鉴2024》。数据显示，2023年全国初婚人数达到1193.98万人，跟2022年相比增幅为13.52%，这也是自2014年以来我国的初婚人数的首次增长。

尽管在数据上，初婚的人数出现了增长，但总体结婚人数变少，离婚人数增加，已经变成了一个事实。比如，具体到2024年第三季度，结婚登记数同比下降了将近25%，创下了自2008年以来的最低值。而反观离婚登记数，同比增长了5.64%。

注意，可不只是咱们国家，放眼到全球，结婚率下降、离婚率上升已经成为普遍趋势。

这就引出一个问题，离婚的人都是因为什么而选择放弃婚姻呢？

就在今年，《福布斯》还专门调查了夫妻离婚的原因，结果排在第一位的原因是缺乏承诺。没错，就是夫妻当中有一方对婚姻关系不认真、不投入或不付出努力，导致另一方失望而离婚。而像婚内出轨、财务问题、家暴等原则性的问题，分别排在第二位、第五位和第六位。

换句话说，很多人可能是因为原则性的问题导致婚姻破裂，但还有更多人是因为失望累积而导致婚姻走到尽头。说白了，婚姻当中消磨激情和耐心的，可能是那些平时不太惹眼的琐事。

那么，回到最根本的问题，怎样才能拥有幸福的婚姻呢？关于这个问题，有一个人的建议特别值得仔细听听，这个人叫约翰·戈特曼，人际关系领域里的大师级人物，他还有个很厉害的绰号，叫“婚姻教皇”。

约翰·戈特曼对婚姻关系做了长达50多年的跟踪研究，涉及3000多个美国家庭和700多对新婚夫妇。据说，约翰·戈特曼本人可以在5分钟之内判断一对夫妻未来一年内是否会离婚，而且准确率高达91%。

回到约翰·戈特曼个人的婚姻状况，他跟妻子朱莉·施瓦茨·戈特曼已经结婚37年，同时朱莉也参与了约翰·戈特曼的很多研究，是最重要的合作伙伴。说白了，至少在婚姻这个话题上，约翰·戈特曼是有一定话语权的。

接下来，我们就说说戈特曼对于婚姻的观察。

## 婚姻危机信号：“末日四骑士”

首先，咱们来说说底线问题，也就是什么样的婚姻关系容易出现危机？约翰·戈特曼说，婚姻关系当中有4个很危险的信号，也被称为是“末日四骑士”。

- 第一个信号是：批评。注意，是批评而不是抱怨。按照约翰·戈特曼的说法，抱怨只涉及对方做错事的具体事件，但批评的打击面更广，包括对性格以及人格的负面评价。说白了，抱怨是对事，而批评是对人。比如，你说：“你怎么又没辅导孩子的功课？说好我们轮流来的。”这就属于抱怨。但假如你说：“你为什么总是记不住要辅导孩子的功课？你就是对孩子的教育不上心，就是没有责任心。”这就属于批评。批评最大的坏处是，当它积累到一定的程度，就会变成另一个危险的信号——蔑视。
- 第二个信号是：蔑视。在婚姻关系当中，当你持续地批评另一方时，就会感觉自己比对方优越，就容易蔑视对方。咱们继续刚才给孩子辅导功课这个场景。比如，这个时候对方说：“我先休息15分钟，再辅导孩子的作业。”那出现蔑视的情况是，你可能会说：“你确定休息15分钟后会给孩子辅导作业？你就知道休息，要么就是一回家就躺下，要么就是玩手机，难道休息15分钟，对你就那么重要吗？”没错，在对话过程中经常出现挖苦和冷嘲热讽，就是蔑视的表现。而根据约翰·戈特曼的观察，蔑视是预测离婚的最强烈的信号。为什么这么说呢？
- 第三个信号是：防御。没错，一旦出现蔑视，双方就很容易陷入防御状态。这个防御可不是一种好的状态，而是不退让也不道歉。实际上隐含的潜台词就是：“都是你的错，我懒得搭理你。”说白了，防御其实是一种更冷漠的指责。
- 第四个信号是：冷战。而防御到最后会升级成第四个危险信号，也就是冷战。想象一下，一对夫妻，有一方为了避免冲突开始频繁回避交流，甚至开始回避见面，这不就代表这段婚姻出了问题吗？而且根据约翰·戈特曼的说法，85%的婚姻中，丈夫都是冷战者。注意，这可不是说他对男性有意见，这也许跟人类最初的基因设定有关。

之前，加州大学伯克利分校的学者通过研究发现，当男性和女性同时突然听到一个非常响亮且短促的声音时，像车胎爆裂的声音，男性的心跳普遍比女性快，而且心跳加速的持续时间更长。还有研究表明，当男性被粗暴对待时，大约在20分钟后，他们的血压会再次升高，这种状况会一直持续到他们反击为止。说白了，男性在生理层面上，要比女性更记仇。因此，发生冲突事件时，对男性的消耗通常大于对女性的消耗，因此男性就更容易出现回避的情绪，进而引发冷战。

注意，咱们前面说的这4个信号，未必一定是按顺序出现的，有时候会反反复复。那么，回到现实，具体应该怎么经营自己的婚姻生活呢？

## 经营婚姻的三个实用方法

之前，约翰·戈特曼在《幸福的婚姻》这本书里提供了很多实操性很强的建议，同时结合他在接受各种播客的采访时提出的观点，咱们今天说其中的三个最实用的方法。

简单来说，约翰·戈特曼的核心观点是，婚姻关系的核心其实是友谊。就在今年5月，清华大学中文系的教授汪民安，出版了一本新书，书名也叫《亲密关系的核心是友谊》。具体来讲，怎么去经营夫妻之间的友谊呢？

- 第一，完善你的爱情地图。什么是爱情地图？其实就是你对你伴侣内心世界的了解程度。最简单的方法就是，记住一些对方人生中的重要事件，而且当这些事件发生变化时，要及时更新信息。比如，对方最好的朋友是谁，他曾经的梦想是什么，以及他生命中的一些重要时刻等。说白了，就是深入了解你的另一半。可能有人会问，才能获得这些信息呢？最简单的方法就是直接提问。按照约翰·戈特曼的建议，当你想改变某段关系时，你可以试着把50%的对话变成一个开放式的问题，多给对方表达的机会。
- 第二，尽量多表达你的喜爱和赞美。这句话乍一听很好理解，但想真正做到却不简单。根据约翰·戈特曼的统计，婚姻关系非常稳定的夫妻，释放积极信号和消极信号的比是5:1。而那些最终离婚的夫妻，这个比例是0.8:1。没错，几乎没有办法避免消极信号，只能通过增加积极信号来稀释它。具体怎么做呢？其实，最容易操作的办法就是，把发掘和欣赏伴侣身上的优点，变成一个日常习惯。这背后还有一个潜台词是，婚姻关系当中，不能总是一方占据道德高地。按照《关系攻略》的主理人熊太行老师的说法，经营婚姻关系时，交替领先的团体赛模式，才是最理想的状态。
- 第三，要在那些微小时刻，彼此靠近。比如，有一天对方刷手机时忽然说：“你看，这是我们之前看过的那部电影。”这个时候，假如你只是回复“对、嗯、没空看”等，就会导致对方丧失分享欲。但假如你说：“是啊，还记得当时我们看这部电影的时候很感动，尤其是电影的结尾太精彩了。”你看，这个效果肯定会不一样。约翰·戈特曼还给这套理论起了个名字，叫“爱情银行”理论。也就是，这些看似微小的事件对夫妻关系至关重要。而每一次互相回应，就是在往“爱情银行”里存款，这样在遭受巨大的生活压力和冲突时，就能用之前积累的善意来缓冲。

最后，你可能会说，即便是掌握了这些方法，夫妻关系里还是有很多冲突没有办法彻底被解决。没错，事实也是这样。按照约翰·戈特曼的说法，即使我们完善爱情地图，彼此表达喜爱和赞美，在日常中互相靠近，婚姻当中69%的冲突也还是无法避免的。

怎么办呢？戈特曼说，你可以改变对冲突的认识。实际上，不是所有的冲突都需要被解决。很多时候夫妻爆发冲突的地方，往往是彼此最深层次的东西，而在一些最根本的问题上，双方都很难妥协。这个时候，你可以把婚姻中的这一面，当成是一个陌生的领域，通过提问、靠近、理解的方式去探索，然后创造一个夫妻之间的共同任务。换句话说，在那些看似糟糕的冲突背后，反而可能隐含着亲密关系最大的源泉。

最后，假如只记住一个秘诀，那么约翰·戈特曼的这句话也许很合适带走。他说：“幸福的夫妻都遵循这样的信条，当你感到痛苦时，我将放下一切事情，只专注倾听你。”
