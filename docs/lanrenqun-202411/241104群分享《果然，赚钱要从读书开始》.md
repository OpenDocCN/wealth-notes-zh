

# 果然，赚钱要从读书开始

前段时间遇到消费行业的专家黄碧云老师，她特别兴奋地说，她正在考察一个特别神奇的案例，没错，也是东茅街茶馆。不瞒你说，之前我对东茅街茶馆几乎是没听说过。直到看到黄碧云老师带回的观察报告，那真叫深深给我上了一课。

过去我们总说一个词，叫层次感。形容一个运动员厉害，会说他的技术有层次感。形容一个歌手很牛，会说他的声音有层次感。而东茅街茶馆特别的地方就在于，它会告诉你什么是做生意的层次感。

接下来，我们就说说黄碧云老师带回的，关于东茅街茶馆的观察。

先交代个背景，东茅街茶馆地处长沙芙蓉区的丰泉古井社区，离着大名鼎鼎的解放西很近。这个地方就相当于北京南锣鼓巷旁边的胡同，或者上海城隍庙附近小区。总之，你就想象你所在城市里，离那个最火的步行街不远处的某个老小区，东茅街茶馆大概就在这么一个地方。

茶馆今年 9 月底刚刚开业，已经在长沙本地上过几次新闻。它为什么那么受关注呢？我讲三个细节你听听。

- 首先，茶馆全天客流量不断，从一大早六点半开始卖早点，直到夜里卖夜宵，全天能接待 15000 人，人流始终不减。
- 其次，茶馆的风格很复古。东茅街茶馆是由原来当地的一个大礼堂改造而来的，叫二轻集体大礼堂，1952 年就建成了，茶馆就在大礼堂的基础上做了翻修。
- 最后，茶馆的客群很丰富，有附近的邻里街坊、有当地的年轻人，还有外地游客。

听到这，有人可能会说，这有什么特别的吗？这不就是一个玩情怀，主打文艺的茶楼吗？这样的地方很多啊。

别着急，你仔细想想，前面这三个场面，其实都并不多见。

首先，不管多繁华的地段，很少有客流全天都密集的。网红大集人多吧？一过中午就散了。三里屯、解放西、南京路热闹吧？一到早上就没什么人了。嘉御资本的创始人卫哲老师说，做生意的目标就是要让一天之中都没有淡季，做到这点并不容易。你看，东茅街茶馆做到了。

其次，关于复古风格，要知道，一个1952年的破旧老建筑，别说经营了，不被拆除就已经是万幸了。换句话说，你看到的东茅街的所有复古，都是重新做出来的旧，都是刻意设计的结果。茶馆的复古还原度高到什么程度？店里那个装米粉的碗，因为跟过去自己家用的太像，很多人居然提出不买粉只想买个碗。你看，能把复古做到这么细致，说明其中的手法一定有讲究。

最后，说到客群，其实很少有网红店能同时吸引老人和年轻人。比如北京的南锣鼓巷，一条巷子隔开了两个世界。巷子里是来旅游的年轻人，巷子旁边的老胡同住的是本地的老居民，这些老人几乎从来不去南锣鼓巷溜达。而东茅街呢，一个开在解放西附近的茶馆，从早上开始，附近的老人就来这里吃早点，不一会儿就有四面八方的年轻游客。全天看下来，这个茶馆里既有点上一杯茶，坐上一天聊天打麻将的本地老人，也有来打个卡，或者来开直播的年轻人。这个场面是不是不多见？

换句话说，东茅街茶馆厉害的地方就在于，它包容了大量的，看起来截然相反的因素。有了这个极强的包容度，自然就有消费的梯度，再加上全天庞大的客流，营收的基本盘自然也就有了保证。

那么，东茅街是怎么做到的，这背后的策略是什么呢？

东茅街的创始人叫简名，在商业文旅领域算是成名已久。简名是云梦山海的创始人，长沙7UP 购物美学中心的联合创始人，也是广州超级文和友、深圳超级文和友的操盘手。东茅街茶馆就是他的新手笔。

在黄碧云老师的采访中，简名说了很多细节，我们今天主要讲其中的两个重点。

## 第一，穿透方志。

什么意思？问你一个问题，假如你要设计个带点地方特色的餐馆茶楼，请问，你该去哪里寻求思路？有人可能会请专业的顾问，但也有不少人会直接照搬其他的网红店。

但事实上，一直有个特别重要的参考资料，只不过查询起来有点费事，这就是方志。方志，也叫地方志。城市有城市的方志，村子有村子的方志。你可以把方志看成一个地方的简历，它从存在以来，都发生过什么大事，出过哪些名人，百姓的生活习俗什么样，方志里几乎全有记载。

简名说，随着现代化的进程，现在的城市之间越来越相似。那么，你去哪寻找一个城市专属的特色文化？答案就是方志。要想在一个地方做文旅，第一步就是要熟读当地的方志。

你可以看看简名在长沙的方志里都发现了什么。

比如，长沙的方志里有记载，在长沙这个地方，存活时间最久的商业业态之一就是茶馆，已经有上千年的历史。而且你再观察长沙的历史，越是经济下行的时期，老茶馆的经营反而越坚强。那么，怎样才算是传统意义上的老茶馆呢？这个老字该怎么理解呢？你也能从方志里看到答案。根据简名的概括，中国的老茶馆就相当于国外的广场，需要承担三个功能，基础消费、基础社交、基础空间。最后这个基础空间指的是，只要花点小钱就能心满意足地待上一天的地方。借用今天的话说，就是花点小钱就能获得高情绪价值的场所。

你看，方志等于是在告诉你说，从长期看，老茶馆是一门确定性非常强的生意。这个确定性能帮你锁定一个正确的方向。

再比如，长沙喝的是本地特色的烟熏茶、沱茶；长沙人吃的米粉是扁粉，而常德的米粉以圆粉为主；以及过去很多长沙人的院子里有梧桐，注意，这可是长沙本地的土梧桐，而不是后来大规模种植的法国梧桐；还有老长沙人最舒适的那个状态，叫，一杯茶，二两粉，三五好友。这些关键信息大都能在方志里找到线索。

你看，有了这些发现，后续重建茶馆的时候就知道该怎么设计，该还原哪些场景了。比如场地里原本的梧桐树，正好就是过去的土梧桐树，那就不应该砍掉，留着它更能吸引本地人。同时，根据方志里的发现，茶馆也可以确定大致的经营品类，以茶、粉、面，以及长沙本地的包子、胡椒饼、油薄脆为主。

换句话说，假如你要寻找短期的趋势，可以去观察眼下的网红店。假如想发现长期的确定性，就需要具备更长期的历史视角。而方志提供的，就是这个视角。

比如，海南的方志会告诉你，客至敬槟榔，这是黎族的待客传统；而庆阳的方志会告诉你庆阳香包的历史故事，这些信息没准都能成为文旅行业的抓手。

很多人说，赚钱要从读书开始，大概就是这个意思。

## 第二，设计有层次感的商业生态。

前面说了，东茅街茶馆处处都在考虑老年人的需求。比如，茶馆的入口有专门的坡道，腿脚不便的老人坐着轮椅也能来。再比如，茶馆的定价很便宜，对标的是长沙麻将 258 的娱乐体系。也就是，2元的葱油粑粑、5元的花生瓜子、8元的红茶绿茶，人均不到10块钱就可以闲坐一天。

但是，有人可能会奇怪，这样利润是不是太低了？这就要说到刚才那个词，层次感。这背后是一组非常丰富的层次。

- 首先，茶馆开在步行街附近的老社区。这是因为，步行街满足的主要是年轻人的需求，但附近其实也住着大量的老人。这时，开个茶馆，一来可以满足老年人的需求，二来因为老年人起床早，也确保了早间时段的客流量。而这些都正好是市场的空白点。三来，住在附近的老人很多，他们也能给茶馆提供一个客流的基本盘。
- 其次，前面的经营利润率确实有限，但别忘了，一旦附近的老居民聚集起来，让茶馆有了人气，这在年轻人看来，就是现成的烟火气，现成的人文风景啊。没错，第一层的客流，是第二层客流眼中的风景。旅游业一直有个说法，叫，最美的风景是人。尤其长沙人本来就谈吐幽默，听附近的老人聊聊天，是个非常有趣的体验。
- 最后，有了当地老居民，有了外来的年轻游客，就可以以这个热闹为基础，做第三层设计，这就是直播。你看，这么热闹的网红茶楼，就是现成的直播背景。茶馆里有主播全天直播，再配合投流之类的手段，线上的流量也能跑起来。流量跑起来，来的人多了，总营收就高，茶馆就可以进一步控制好基础品的价格，并且通过这些基础品，吸引更多的附近居民。你看，这么一来，一个增长的循环不就跑起来了吗？

说到这，我们可以再回过头打量一个概念，生态。这个词现在动不动就出现，好多人都觉得，生态就意味着丰富，意味着商品多。但从东茅街茶馆的设计里，你会发现关于生态的另一个真相。生态的本质不在于你的商品种类多，而是你生意中涉及的各个元素，它们之间能不能彼此促进，形成共生关系。只要能像自然界那样形成共生，彼此促进，那么这就是一个好生态。
