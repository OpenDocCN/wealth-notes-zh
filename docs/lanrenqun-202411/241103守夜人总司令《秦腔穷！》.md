# 前言

我们并不是同情他们，我们只是同情曾经的那个自己。

然而，曾经的那个自己并不是像他们现在这样！他们莫名其妙的优越感主要来自于两个地方：
- 1. 所谓的信仰驯化的优越感，
- 2. 西方叙事框架把他们称为中东，把我们称为远东，让他们误会自己更加的优越和更靠近“文明”。

在李小龙拍的一部老电影中，影片中的西方人就是用这个词称呼李小龙的。
这个侮辱性的称呼源自晚清时期的华人在美国被西洋人视为虫子——秦是中国的意思，腔穷其实是“长虫”的西方发音。

晚清时积贫积弱的中国和愚昧落后的面貌让中国人饱受外人的鄙夷。因此，不管是西洋人，还是东施效颦的东洋人，都给我们起了很多侮辱性的外号，这种侮辱性的词汇一直被一些外国人用来称呼中国人，只不过在资讯不发达的时候，外国人侮辱我们的信息被屏蔽了。

官方媒体所传递的信息自然是以大局为重的正面信息，然而，国外的官方人员通过我们的官方媒体传递给我们的信息是基于利益的理性表达，而那些国家的普通民众对待我们的态度才是那个社会看待我们最真实的样子!

中国人在自己的信息茧房中普遍认为当下的中国已经遥遥领先的存在，至少也是与美国处于并列地位的状况。自然会得到全世界的赞赏和羡慕以及尊重。特别是那些落后地区的人更会把兔子当成拯救自己的那道唯一的光。
不可能学着西方人的样子比喻和侮辱我们!这种观念一方面源自于我们以己度人的惯性思维，另一方面源自于我们的官方媒体有选择性的报道。

事实上，许多国家的民众还没有从认知惯性中回过神来。就连社会相对发达，能够到全世界去旅行的欧洲，都有许多人对中国的看法停留在四五十年前大家都骑自行车穿着军绿色和蓝色工作服的印象中。对于那些还不如欧洲的地方，更不可能了解这二三十年中国的突飞猛进。而且，他们所能接触到的媒体也不会报道真实的中国。

那些极度反美的地区，骨子里反而特别的羡慕并渴望依赖西方——尤其是普通民众。

中国人自己遭受过西方的欺压，在苦难深重的绝境中凭借三代人的牺牲和接力，终于从绝望的泥潭中挣扎出了一个人形，并且开始洗刷身上的屈辱，重新迈向属于自己的高峰。

在这个过程中，我们会本能的以己度人，认为那些同样遭受过西方殖民者欺压和蹂躏的苦难民族也会和我们一样拥有奋发图强的意志和心态。事实上，除了我们以外，这个世界上绝大多数国家的人都很认命！

他们根本没有“将相无种，人当自强”的观念。我们这个社会之所以会那么的卷，就是因为我们都有强者心态：哪怕是贩夫走卒都会想办法让儿女去读书，试图让下一代能青出于蓝而胜于蓝！放眼全球，似乎只有我们是这样，其它国家和民族的人真的不会这样。

他们真的很认命，以能成为贵族的仆从为荣，而中国人个个都想出人头地，人人都怀揣鲲鹏之志。中国人希望自己的国家在任何领域都出类拔萃，独占鳌头，遥遥领先，否则就是不行。

中国的父母希望自己的孩子，在任何方面都很优秀，做一个普通人平凡的过完一生，似乎是一种被迫无奈的选择。这些我们习以为常的事，并非适应于全世界的所有人，他们跟我们真的不一样！

东亚儒家圈的人还有点进取之心，东南亚普通人的进取之心就会弱很多。到了中东北非拉美之类的地方，那简直是一种让人非常无语的存在！这个世界上许多地方的人并没有像我们一样经历过漫长的大一统的封建时代。

包括欧洲在内的地方，社会组织方式和社会各阶层的流动方式，仅仅停留在秦朝统一之前的那种状态就再也没有继续向前进化了。要不是科技革命掩盖了这些社会治理体系的进化不完整，它们就会呈现出中世纪的那种野蛮和血腥。

今天美国的文盲率还有 21%，但这并不妨碍他成为世界第一。它的精英教育非常的内卷，但是快乐教育也非常的松弛。他们的社会精英对待社会底层比我们想象的更加的恶毒和无底线。他们可以人人都合法拥有枪械，校园枪击案此起彼伏。

但是他们有几千万流浪汉，包括 4000名退伍的美军沦为流浪汉。然而，他们并没有组织起来反抗。可见它们社会的控制和摧残比我们想象的要更加的可怕！否则，不至于造成他们如此的麻木，像行尸走肉一样完全没有反抗的意识。

通过发达的社交媒体，我们看到了以色列是如何对待加沙地区的民众的。

事实上，它们一直就这样对待彼此。

德国人对待犹太人，犹太人对待中东人，英国人对待爱尔兰人，乌克兰西部的人对待东部的人，它们一直是如此残忍。

它们甚至习以为常，只是我们觉得太过分了而已。它们对自己国家的人也是非常的没底线。

它们社会的那种不公往往是超出我们的心理底线的存在。

国外许多社会是直接从部落和奴隶制直接被拽到现代社会的。整个社会组织方式和人的行事风格以及心态都还停留在部落和奴隶社会的惯性下。只不过外部装饰和生活器物因为外部力量的统治而变成了现代社会的样子。

但那个社会结构以及社会心里依然在原有的惯性之中。这一点我们可以从大凉山的当地人身上获得验证——它们在解放前处于奴隶社会，没有经历过古典文明的长期浸染和雕塑，它们是被外力强行拉入现代社会的。它们的心态和行为依然延续奴隶时代的惯性：扶贫的人送给它们猪仔，教它们饲养，试图帮助它们脱贫。

结果它们当天晚上就把猪仔杀了端上了桌子！在它们的观念中，这没有什么不妥。因为心智处于奴隶制惯性下的人，眼前能吃一口就吃一口，下一顿还不知道能不能吃上。

这种状态下的人不会有任何指望，更不会有任何规划，活得非常的原始。这一点你在非洲人和一些中东人身上也能看到！

中东那个地方的人被以色列来回的虐了几十年，他们为什么就不能团结一致呢？他们彼此之间的相互戒备远远高于他们对以色列的敌意。当年阿拉法特来华访问，兔子教他们如何建立统一战线，如何去争取阶段性的目标。结果，他完全不听。

不是去追求建国共存这个切实可行的阶段性目标，而是一味追求不切实际的极端主张——要把以色列彻底抹去。这怎么可能！且不说他们没有这样的能力和力量，读过《不是以色列厉害，是所有大国都不允许它灭亡！》的人就会明白，任何世界大国都不会允许以色列在那个区域消失，不仅仅是美国不允许！

他们的脑子大多呈现二极管的特征，心态非常的极端和狭隘——如果忠诚不绝对，就是绝对不忠诚。追求的目标又极度不切实际。所以，他们一直遭受苦难。这是他们自己的选择，也是他们必然的命运！

当年约旦接纳巴勒斯坦难民，这些难民不仅不感激约旦的收留，还指责约旦没有不惜代价去消灭以色列。这就属于道德绑架了！最后发展到：阿拉法特他们试图夺取约旦作为反以色列的基地，鸠占鹊巢。此举被约旦发现，与他们打了起来，最后把他们驱逐出去了。

当时约旦向周边各国求援，没有任何一个国家伸出援手，派出一兵一卒。只有以色列赶来帮助约旦守住了自己的国家。从此以后约旦都站在以色列那边。包括这一次以色列和伊朗的回合制中，伊朗打以色列的时候，约旦会拦截伊朗的导弹。

以色列打伊朗的时候约旦会开放领空。大家都指责约旦为什么不站在同一信仰和同一族群的立场上，他们不清楚这些所谓的同一信仰和同一族群的人都是以前什么样的人！

他们为什么还会学着西方的论调鄙视我们呢？那是因为他们的普通民众，在心理上认为自己比我们高人一等。一方面来自于所谓的信仰的驯化，另一方面来自于西方的叙事框架：西方中心视角，把他们那里称为中东，把我们称为远东。于是，他们就认为自己更加靠近“文明”！至于信仰会让一个人变得更加的自以为是。

这一点你可以通过国内的某一些群体就能看出来——明明自己生活的地方穷的一塌糊涂，却看不起这个看不起那个，因为别人都没有信仰，不够虔诚，所以不如自己高贵。以此类推，两者叠加起来就造成了他们的这种行为，学着西方的样子称呼我们为“秦腔穷”！

大毛老大去隔壁三仔家，端着一杯酒半天不喝，三仔不喝他也不喝。三仔去大毛家，大毛家有个欢迎传统，就是拿面包蘸盐吃，三仔做个样子不吃。越是表面上表演关系很紧密，越是信不过对方，相互戒备。

反而是大毛来兔子家，不仅是礼仪程序上会直接吃，开会时前面摆的小吃都被大毛的老大吃光了！兔子去到大毛家，在机场欢迎流程上，也是直接吃面包。土匪最喜欢把义气挂在嘴边，但最不讲义气的就是土匪，往往为了蝇头小利就会相互背后捅刀子。这就是为什么加沙被虐成那个样子，但周边同宗同族统一信仰的国家都不愿意接受他们的难民的缘故。因为已经表演过一次了！

只有苦难而不懂得进化的人群，苦难是他们的必需品。其实我们也并不是同情他们，我们只是同情曾经那个自己。即便是曾经的自己，也并不是他们现在的样子！我们在山河破碎的时候，不是他们这样听天由命和自以为是。

当我们被日军围困到只有一条生命通道的时候。20万云南父老在没有施工机械的情况下，妇女儿童一起上阵，修通了滇缅公路。每一米路就损失三个人！有一个当时美国人拍的纪录片：四五岁的孩子工地上砸石头，妇女用背篓背着自家的婴儿在工地上干活，而且是自带工具和干粮！

中东那地区的人会有这样的民族性格？所有的镜头都只是在不断的展示他们的苦难和茫然无措，渴望得到外界的同情和帮助。但他们自己呢？西边在打，东边在看！一个屁大的地方还分成14个派系相互不对付。
