

# 普京能搞定俄军腐败问题吗？
## 241117 文/卢克文工作室嘉宾 怀青

  俄乌战争即将迈过第四个年头，乌克兰被抛弃已成定局，但俄罗斯方面的表现，也和人们的预期相差甚远。战争把俄军长久以来存在的腐败问题暴露得一干二净，腐败渗透到了方方面面，成为普京不得不解决的问题。

  今年以来落马的俄军将领非常多，几个影响比较大的比如，卢甘斯克人民共和国英雄称号获得者、前国防部副部长绍伊古的心腹帖木儿·伊万诺夫；俄联邦英雄、前国防部副部长，负责俄军后勤系统的德米特里·布尔加科夫大将；俄罗斯武装力量通信部队总参谋长兼俄总参谋部副总参谋长瓦吉姆·沙马林中将；国防部国防军令保障局局长弗拉基米尔·韦尔特列茨基。

  其中行为最恶劣的是布尔加科夫大将，这位1976年从军事后勤学院毕业后一直担任后勤管理军官的大将，参与制定了现在俄军的后勤运行体系。

  可俄乌战争开始后，俄军士兵抱怨军粮供给不足的问题就频频发生，后来又爆出乌克兰士兵缴获过期7年的俄罗斯军粮，证明俄军后勤系统存在大问题。

  军粮量不足还过期，但没想到质也差，布尔加科夫大将在收了钱后，帮助供货商以次充好，高价向部队供应低质量口粮，士兵口粮中的牛肉被更廉价的鸡肉和猪肉替代，打仗是件耗体力的事情，连饭都吃不好，士兵的战斗力可想而知。

  布尔加科夫主管俄军后勤长达十四年之久，贪腐行为影响了整个后勤系统。

  从今年开始，俄军打击贪腐行为的力度明显加强，一边打仗一边打老虎，普京这回是来真的了。此前俄军进展缓慢，外界一直有传言绍伊古会成为这口锅的承担者，可没想到在风声最紧的2023年，哪怕出了普里戈津“清君侧”事件，老伙计依旧稳如泰山，普京似乎一如既往地信任着绍伊古。

  到了2024年，绍伊古突然被换，新上来的别洛乌索夫没有在军队中任职过，背靠着普京手起刀落，严厉打击贪腐问题。仅在今年前8个月，俄罗斯联邦调查委员会在相关领域的立案数量就达到了497起，远超2023年同期。

  今日果是往日因，造成今天这个局面的因可以追溯到苏联解体后。

  1993年，俄罗斯爆发宪政危机，得到军队支持的叶利钦成为胜利者，为了报答军头们的支持，同时也为了拉拢人心，叶利钦对于将领们倒卖武器、虚报数量吃空饷、挪用军费的事情，闭上了双眼。

  贪腐之下，士兵士气自然低落，后勤武器都供应不足，军纪想不涣散都难，于是在指挥失误、战术判断不准的叠加因素下，1994年的第一次车臣战争以失败告终，叶利钦颜面尽失。

  1999年，叶利钦做了可能是他一生中，对自己祖国做的唯一一件有益的事情，选定了普京作为他的接班人，并在1999年的最后一天撂挑子不干，让普京顶上。

  在叶利钦时代，俄军的总参谋长和国防部长的职责存在重叠，经常发生冲突，毕竟都是当主官的，没人愿意分权。普京接任俄罗斯总统后，总参谋长与国防部长分别是克瓦什宁大将和谢尔盖耶夫元帅，都是叶利钦时代留下的高官。

  在“双长制”的背景下，两位大佬为了争权夺利，发生内讧。此时的普京刚刚上台，对于军队的掌控力有限，只能冷处理。2001年谢尔盖耶夫退休，普京立刻安排亲信伊万诺夫接任谢尔盖耶夫的职位，伊万诺夫也成为俄罗斯历史上第一位文职国防部长。

  此时普京的重点集中在对权力的掌握上，但即便是这样，普京对于俄罗斯军队的腐败问题，也在不停地打补丁，不停地安排亲信掌握核心权力，加大对腐败的惩治。不过这些行为只能算是止血，始终没有把烂肉挖出。

  除开精力方面的问题，在制度方面俄罗斯反腐机构设置始终存在职权分散，没有统一、独立的总管部门，对政府官员的监督都有不足之处，更别说对军队的监管。何况军队因为涉及“机密”，调查监管难度更大。

  普京在反腐上真正开始投入精力是在2008年的俄格战争后，由于腐败问题没有被根治，俄罗斯虽然没有惨败，但也没有大胜，这场仗打得并不好看。据统计，当时大约30%的国防军费要么被中饱私囊，要么被挪作他用，而俄军由于缺钱，武器长期得不到升级换代。

  尽管时任总统是梅德韦杰夫，但普京在背后一方面增加军费，一方面打击贪腐。2008年俄罗斯先后颁布了《国家反腐败纲要》和《反腐败法》，还在2009年之后要求俄罗斯高官公布财产，完成财产申报。

  结果很打脸，2012年俄罗斯军方爆出惊天丑闻，时任国防部长、普京的心腹阿纳托利·谢尔久科夫存在严重贪污行为被解职。

  谢尔久科夫主要牵扯到了Oboronservis案。Oboronservis是一家负责航空设备的维修、信息安全系统和自动化管理系统的研发、武器和军事装备的维修、俄罗斯国防部住宅设施的管理，是一家大型的国有军工企业。

  叶夫根尼娅·瓦西里耶娃是这家公司的董事，同时也是国防部财产管理部门的负责人，她还有一个身份，即谢尔久科夫的情妇。通过Oboronservis，她将属于国防部的土地和房地产以低价出售，自己从中收取回扣，而授权文件自然是谢尔久科夫签署的。

  事情败露后，瓦西里耶娃被判犯有欺诈、挪用公款和洗钱罪，谢尔久科夫被控犯有过失罪，最终落马。谢尔久科夫的继任者就是绍伊古，和前任不同，尽管同样不是职业军人出身，但当过紧急情况部部长的绍伊古，揽权是一把好手。

  当上部长后，绍伊古首先把老部下调到身边，然后叫停谢尔久科夫的军改计划，作战单位从“师改旅”改回“旅改师”，扩大士兵数量。为了拉拢军中势力，不断增加副部长职位，最高峰时设立了一名第一副部长、十二名副部长，加上绍伊古他自己，一共14人。我们常说一个萝卜一个坑，这么多坑，加上绍伊古推行的集团军模式，带来的连锁反应就是俄罗斯高级将领暴增。

  大权到手的绍伊古对普京并没有生出二心，但是因为一家独大，担任要职的都是自己人，管理松懈是自然的，期间贪腐事件屡禁不止。

  2012年，Zaslon公司获得了约8亿卢布的资金，用于开发一款拦截非战略导弹的飞机系统。2013年开发工作被冻结，2016年国防部决定重新开发该系统并希望将所有设备转移到国防部中央科学研究所时，他们只发现了两台旧笔记本电脑、一台损坏的望远镜、一台座机电话、一个损坏的导弹模型。

  调查显示，为了制造研究活动的假象，Zaslon公司与空壳公司签订了虚假合同，其中一些公司的注册地址竟然是公共厕所。

  2015年Prompostavka防务公司前负责人鲁斯兰·苏莱曼诺夫被抓，涉嫌利用虚假合同套取高达18亿卢布的军费。

  2017年有5家公司中了国防部的标，负责建设价值超过230亿卢布的营地，但这些公司都是在2015年8月到9月期间集中成立。进一步调查显示，这次事件涉及至少28家空壳公司，追根到最后，有11家公司的背后是和普里戈津有关。

  从2017年开始，普京每年下令解除将军职位的人数都是两位数起步，大部分都是因为贪腐问题。要不是俄罗斯的经济问题牵扯住了普京的大部分精力，按照他的性格应该早就下手了。

  时间来到2022年，俄乌战争开打，绍伊古表现糟糕，俄军的贪腐问题被全世界知晓。前方吃紧，后方紧吃，从上至下的腐败问题大大影响了俄军的战斗力。

  普京忍不了了，决定刮骨疗伤，借着俄乌战争对军队展开大换血，更好地实现新一轮的军改。

  2023年俄罗斯宣布，在2023年到2026年期间，俄军将经历一次“重大变革”，部队组成和行政组织结构都将进行大规模调整，并将着力提升俄海军、空天军和战略导弹部队的作战能力。提升作战能力势必加大投入，最新的消息是2025年俄罗斯军费达13.5万亿卢布，占GDP的6.3%，比今年高四分之一。

  牙缝里省下来的钱，再让这些蛀虫中饱私囊，是决不能接受的。连任成功后的普京个人权力达到顶峰，有能力也有实力对这帮老伙计下手而不伤根本。

  关于普京这次反腐，有一种阴谋论的说法是普京正在借惩治军队贪腐问题，对军队将领进行清洗。但实际上现在的普京对于军队有很高的掌控力，绍伊古或许贪权古板，但有反心的可能性不大，这也是为什么他现在只是在坐冷板凳，而不是在唱铁窗泪。

  换句话说，这次反腐是因为以普京为首的核心高层和贪腐将领在当前形势下存在根本性矛盾。在经济收入有限的情况下，继续推行军改，普京需要保证资金投入用到正途，不再被挪用。所以在普京的授意下，别洛乌索夫会继续推进反腐，反腐其实是为了给军改扫清障碍，这是俄方的最终目的。

  另一方面，俄乌战争进行到今天，俄罗斯虽然占据上风，但是付出了巨大的代价。军队反腐不仅可以巩固军队战斗力，应对乌克兰战争的持久化；还能提振士气，恢复俄军的社会形象。

  高层腐败案件被公开处理，不仅是对腐败官员的威慑，也是对普通士兵的激励。通过公开审判军队贪腐官员，加上提高待遇，士兵们的怨气被平复。军改的目的是为了强军，强军不仅要强兵器，更要强人。士兵是军队的基石，军心不稳，何谈强军。

  所以这腐败，普京必须反。这颗在俄罗斯联邦诞生之初就有的毒瘤，再痛也要刮掉。毕竟如果连普京都无法去除这颗毒瘤的话，后继者再想根除，恐怕就不止伤筋动骨这么简单了。
