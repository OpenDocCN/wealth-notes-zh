

# 觉悟社问答：关于未来几年房价的走势

提问：

司令好，坐标北京海淀，目前有换房计划，计划从一个小两居换成一个大三居（都是次新小区，小区品质在周边还算可以），但要背负几百万的贷款。个人工作稳定，收入稳定，不存在还贷还不起的风险。房子各方面也很满意，也确实有换的需求。

唯一的顾虑就是之后几年内房价走势，因为目标房子和我自己的房子均价一致，基本是同涨同跌，如果未来房价还要下跌，相当于我补的差价也亏了，虽然还贷不成问题，但凭空亏了几十万甚至一百多万，也非常不划算。

所以想请教下您对于未来几年房价走势的看法？以及您对于我的建议，再等1-2年形势明朗后再换是否更合适？

回答：

人的一生中重大的决定往往并不取决于理性。所以决定只能由当事人来做。有时候真正能够影响最终决定的核心因素，甚至不是确凿的证据，而是一时的情绪。这种权重极高的情绪，往往只有当事人自己能够理解。两年前曾经有人问过类似的问题。当时给出的建议是赶紧出手，因为趋势会向下。有人及时出手了，有人觉得这个判断是对的，但依然犹豫不决越拖越深。显然不同的决策者，即便接收到同样的信号，也会采取完全不同的反应。所以，你更需要扪心自问的是那些真正会影响你决策的因素是什么？

关于房子的走势，趋势已经形成，暂时刹不住。在过去的十五年中，房子不是居住的供需市场，而是个金融市场。不管官方如何呼吁，只要不掏出钱来把筹码都吃掉，这市场价格就很难稳定下来。为什么呢？因为筹码高度集中在少部分人手中，而且这少部分人，不是官方就是上市公司！

如果你仔细看报表，你就会发现，最近这一两年，许多经营不善的上市公司，特别是那些将要退市的公司，都在卖房子保住自己的上市企业地位。这个操作虽然是合法的，但不免让人想起一个问题：一家做半导体的企业，为什么手上有这么多房子？一家做汽车的公司为什么能够卖出这么多房子？你最后就会惊讶地发现：大量非房地产行业的企业，手上有大量的房产。而且国资委手下的一些投资公司手上，也动不动有上千套房。

如果你按照商品的方式去理解，肯定没法理解这种事。如果你按照金融的方式去理解，就一下想通了：在过去的15年中，没有哪个金融产品比房子更优！所以大量的资金都在持有这种筹码，包括专业的投资机构和各大上市公司的资金。当然也包括市井小民的资金。

那些上市公司的主业做的一塌糊涂也不要紧。他们依然会想尽各种办法扩大规模。为什么会这样操作？因为他们不是靠主业的生产和贸易来获利而是通过主业的规模来获取更多的银行授信和商誉借款，并把这些资金用来持有房子这个筹码！

三年疫情和外部对抗造成的经济周期，打乱了他们的抛售节奏，最后造成集中抛售。现在没有人接。刚需族太珍贵了，但数量越来越不够。本来官方出了个主意，说让各级地方把这些房子都六折收了，然后做成长租的廉价房，允许转租金。并且上面还愿意借钱给下面各级地方去收。大家都不是傻子，这一笔收的钱是借款，不是给钱。而且做成长租的廉价房，猴年马月才能把这笔钱挣回来。上面和下面博弈了一年，这个事情还没有什么效果和结果。没有结果，就需要继续恶化逼迫做出让步。这个世界充满了阴谋和诡诈。许多时候，暗藏在水下的博弈，总是把普罗大众当棋子玩！哪个市场都是一样，只要有人的地方就有江湖。而且江湖中的大鳄，往往就是，正如一部著名的电影中那个军火贩子对拘捕他的美军军官说的那样：你们全世界围剿我，是因为你们认为我是罪大恶极的军火贩子，然而，过不了多久就会有一个华盛顿的高级将领给你打电话，命令你把我放了。而且他还不是那个最终下命令的人，最终下令的人可能来自白宫。因为他们还需要我——真正的贩卖者不是我，我只是一个跑腿的，挣点辛苦钱！

扯的有点远，总之一句话：整体趋势继续向下，银行利率只要继续向下，趋势就刹不住，而且很严重。
