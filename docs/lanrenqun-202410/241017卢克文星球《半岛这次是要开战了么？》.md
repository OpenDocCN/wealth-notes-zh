

# 半岛这次是要开战了么？

这几天，新闻报了很多朝韩方面的简讯，注意，都是字很少、但让人看上去觉得事很大的那种，于是很多人怀疑，半岛是不是要开战了。

简讯的优势就是冲击力强，读者会下意识地放大刺激类信号，比如朝鲜炸毁公路铁路、韩国要终结朝鲜政权、朝鲜下达炮兵射击指示、韩国已经开炮射击等等。

但简讯的诡道在于，时常语焉不详，甚至淡化掉某些关键条件，让人看不清来龙去脉。所以，在诸多简讯的叠加轰炸下，朝韩此次对垒交锋，再次让大家生出“半岛是不是真要开战”的疑问。

先说结论：打起来的概率几乎为零。首先，简要捋一下此次紧张局势的脉络。

事情起始于10月9日，朝军发了个声明，说即日起要完全切断与韩国通联的朝方地区公路和铁路，并进行防御设施要塞化加固。韩方当天随即回应，朝鲜要是敢率先挑衅，那么韩国将对朝进行打击。

紧接着10月11日，朝方突然又发声明，谴责称韩国的无人机近期多次渗透进平壤，还撒了一堆恶毒的传单，朝方下最后通牒，如果韩国再敢放无人机进来，再不停止对朝挑衅，那朝方就要给予反击了。而韩方当天回应说：我没放，不是我。

10月12日，金与正公开痛骂了韩国一顿，口径与11日基本一致。

10月13日，韩方强硬回怼，说你朝鲜不也向我放气球么？还挂着翔、玩埋汰的，你也威胁到我了，你要再威胁我，我就终结你政权。

听到这话，朝鲜绷不住了，13日回应称已向前线部队下达命令，注意了，指示命令的完全体是“国境线附近的炮兵部队、火力部队进入射击「准备」状态”“按照战时定员编制把8个炮兵旅转入射击「待命」状态”。

10月14日，金家兄妹齐上阵，一起痛斥韩国是美国驯养的犬科，不过发言核心依然在于，朝鲜是从不屈服的，朝鲜也是永远强硬的，但朝鲜一切动作的本质，就是启动防卫力量、行使自卫权。

10月15日，韩方用动作给出了个回应，对着三八线以南韩国这边的无人区放了几炮。

下面，分解一下这其中的细节和奥妙。

如果非要给故事安一个开头，那本轮紧张局势姑且算是朝鲜挑起的吧。朝鲜先说要切断对韩公路铁路，其实这个根本就没啥营养，因为自从尹锡悦上台以来，朝韩的实际联系接触已经全面中断，朝鲜在这种背景下再切断物理通联，相当于“我们已经分手两年，但我现在要删了你的手机号”。那朝鲜为啥突然提这茬呢？

去年金正恩正式将以往朝韩“我们民族之间”“南北之间”的官方式昵称表述，全都改成了“朝韩之间”，以此来彰显割裂感，表明咱俩不处了，没啥好说的了。

此后的这大半年以来，朝鲜不断吹风说最高人民会议要修宪，明里暗里提示可能在宪法里也把“南北关系”改成“朝韩关系”。

然而，这毕竟是大事，很多朝鲜人估计无法接受，所以今年10月初朝鲜的全国代表大会，最终没有修宪。

既然行动上露了怯，那嘴炮上就要找补回来，于是大会刚开完，朝鲜就整了“删手机号”这么一出戏，以示朝韩关系依然很紧张，没有爱了，全是恨。

朝鲜又特别喜欢炸东西，炸过联络大楼，炸过纪念碑，这次又炸公路铁路，似乎有种消灭物质象征就能摧毁对方精神的迷信感，仿佛放生100条鲤鱼就能胜造一级浮屠似的。实际上，南北心里都很清楚，炸完再建、建完再炸呗，千锤百炼不过是花点钱的事罢了。

更重要的是，朝鲜在声明中还说，要在那片区域埋地雷、放反坦克路障、加固哨所碉堡和阵地之类。而这种行为本身就是防御性的。

因为朝鲜只有陆军而且是炮兵最强，空军海军可以说是弱爆了，至于所谓战略导弹部队，怎么说呢，且不论那些东西实战水平到底如何，单说朝鲜的经济状况和导弹库存，恐怕到时都捉襟见肘。

更何况，战略导弹这种王炸，朝鲜不会轻易出，作用是拿来威慑。因此，朝鲜对韩国如果真要发动攻击，主力就是大炮加火箭炮，打完第一轮毁伤后，除了特种作战，基本就只剩阵地防御了。

如今，朝鲜把三八线附近的路炸了，再埋上地雷、搞上碉堡，主打的就是一个你别想进来、我也不出去。

总的来看，朝鲜发这个声明，就是维持紧张局面的防御性表态，没有任何想动手的意思。当然，韩国当天也对此回了几句片汤儿话，同样没啥营养。

不过，朝鲜向来都遵循“没热点也要制造热点”的原则，趁着热度立马又抛出了无人机事件。这个事很有意思，朝鲜说韩国连着几次放无人机侵犯平壤领空，还在党政核心区域撒反动传单。

从朝韩边界线到平壤有160多公里，小型无人机根本飞不了那么远，只有燃料充足的中大型固定翼机型才能飞到。朝鲜情况又很特殊，不像俄乌冲突中，乌克兰人员可以渗透到莫斯科附近，然后近距离操控小型无人机突防袭扰，而韩国无论是军方还是NGO，根本就进不去朝鲜境内，只能从三八线以南或者海上放无人机。因此，如果是韩国放的无人机，就必然是中大型无人机。

问题来了，中大型无人机入侵领空，都杀到领导机关头顶上了还没被打下来，还能顺利撒传单，那朝鲜的防空体系，又算是个什么呢？

所以这其中就出现个有趣的悖论，即如果是韩国放的无人机，那么朝鲜的防空就是个摆设；如果不是韩国放的无人机，那么就是朝鲜自导自演造了假。

这里不作任何倾向性解读，因为在国际关系中，除了当事方的参与者外，真相是永远不会公布的，无人机究竟是谁放的，已然不重要了，谁能通过事件拿到最大溢出效应，才重要。

朝鲜把无人机事件上纲上线，于是，韩国就说从今年5月以来，朝鲜总共对韩放了约30次“农家肥气球”，是你错在先。双方用这种低端手段打心理攻防战，已经几十年了，不过每次都是以口水仗告终。

特别是韩国，从最开始投放脱北宣传册，到形形色色的杂志、韩流影音类制品，再到这次曝光金正恩的物品，都很有意思。

至于大家很好奇的传单内容，从朝鲜公布的画面看，上面确实是类似百达翡丽或万国手表，然后搞了些换算，意图就是给朝鲜民众拉金家的仇恨。至于这传单是此次被无人机扔到平壤的，还是以前被扔到朝鲜其他地方的，那就不得而知了。

尹锡悦搞心理战的重点，就是向朝民众揭露金家的腐败奢侈。

但这套打法略显一厢情愿，因为朝普通民众长期处于封闭状态，认知程度很低，平壤这类大城市的民众，好歹还见过些有限的世面，而更多数地区的农民，就算面对面跟他们讲百达翡丽的品牌溢价，他们既不懂，也不会相信，因为确实是超纲超太多了。

更重要的是，朝鲜这几年安保抓得很凶，韩国投放过去的东西，拾到者不仅被管束严禁传看，还得第一时间上交相关部门。

所以，这里又有一个悖论，高级黑的宣传单，乡村或边境的老百姓捡了看不懂，看了也理解不了；而守备森严的平壤，老百姓压根就不敢去捡，也就等于看不到。

真不如放点韩流或资本主义灯红酒绿的东西，可能更贴合平民朴素的认知猎奇。

韩国军方否认释放无人机的话语，再次激起朝鲜“愤怒”，然后就有了新闻简讯中，朝鲜让炮兵待命、进入射击准备状态。这里，朝鲜强调的是待命、准备，也就是说，只要韩国不主动攻击，朝鲜也不会主动攻击。

面对朝鲜的“强硬”，韩国也不认怂，随即对着三八线以南无人区放了炮。但其实双方每年都会打着各种理由或旗号，往无人区里放几炮。

所谓的紧张局势，也就这个程度。

今天，朝鲜说召回140万青年学生参军复员，这也是常规操作，其实就是统计工农赤卫队的人员情况，跟我们武装部例行统计预备役人员差不多，根本不是上前线打仗。

朝韩如此反反复复拉扯折腾几十年了，每次都像是要动真格的，但每次又都是雷声大、雨点小，近些年来最危险的一次局势，还是2010年延坪岛炮击，那真是炸死了韩国军民，最后双方也点到为止。

后来无论是朝鲜搞核试验、一年搞几次导弹试射，还是美韩大规模军事演习，无论是在三八线无人区，还是在北方限界线争议海域实弹射击，朝韩乃至美国都保持着见好就收的默契，即

> 「紧张局势螺旋上升闹得如火如荼，但直线降温来得更加猝不及防。」

此次局势跟以往比起来，烈度算是小的了，即使后续捎带出现点摩擦，也升级不成半岛危机。

最后讲讲，朝韩为什么非要闹这次紧张局势。

对金正恩而言，紧张局势是氧气、是必需品。没有紧张局势，朝鲜在大国中的地缘战略作用，就凸显不出来了，作用不凸显，就没人搭理他，就没法换取实际利益。

拜登政府这4年，既不减轻制裁，也不搭理朝鲜，把金正恩晾到怀疑人生，只要民主党在位，就不会跟朝鲜谈判，那金正恩就没机会解开脖子上的枷锁。

眼下，美国马上要大选，金正恩希望特朗普卷土重来，因为特朗普的对朝策略更为激进，朝鲜就有望获得解锁的可能性。在美国大选的时段，朝鲜只要制造些紧张局面，会间接地不利于民主党选情，同时让特朗普注意到朝鲜的作用。

另一方面，普京6月访朝，与金正恩拟了个《朝俄全面战略伙伴关系条约》，这个条约最重要的内容，就是朝俄任何一方遭遇战事，另一方都可给予支持。

只要这条约在俄那边通过，金正恩就跟普京绑在了一起，未来美韩倘若对朝动武，俄就会被卷进战争。当前，普京在欧洲的战略是被堵住的，整体战略要转向东方，朝鲜半岛的位置对俄来讲，也不是以前若隐若现就可以，而是变得越来越重要，因此金正恩制造半岛紧张局面，能够刺激普京更加重视朝鲜的事。

总之，金正恩的策略就是拉越多的大国下水入局，他在中间左右要的价码就越大、机会就越多。

再看尹锡悦，当前执政支持率降到新低，不管内政还是外交，都被国内痛骂，制造点看上去可怕、但实际上又比较可控的南北摩擦，能够有效转移压力，何乐而不为。

当然，尹锡悦肯定不想闹大，如果真是韩国放的无人机，那也肯定是有政府支持背景的NGO放的，毕竟军方直接参与会被抓把柄，非政府组织才是白手套、背锅侠的最好选择。

面对朝鲜的嘴遁攻势，韩国利用完紧张局势，只需在适当时刻下令禁止NGO搞无人机，这事分分钟就下了台阶。暂且消停一阵后，NGO又可以在之后的任何时间出来搞事情，大家讲究的就是个心照不宣。

综上，朝鲜半岛总体就是这么个情况，在核心大国斗争达不到破局的临界点时，朝韩这两位配角，也就只能不断演着同一场剧情无限轮回的泡沫剧。
