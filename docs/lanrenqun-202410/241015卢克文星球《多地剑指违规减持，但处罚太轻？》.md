

# 多地剑指违规减持，但处罚太轻？

241015 文/卢克文工作室嘉宾 文如刀歌
整理：公众号懒人搜索，懒人专属群独享
懒人微信：lazyhelper

![](img/d51fb2245ed6ad3a231d91394762bc6c_0_0.png)

国庆节后，10月8-9日的A股市场，盘面走势与绝大多数散户的预期产生背离。

这几天各种分析的声音都有，包括高层不希望疯牛，主动踩刹车；违规贷款加杠杆炒股太疯，泼冷水清醒清醒；暴涨必急跌是自然规律。

另一类分析则包括：上市公司股东知道自身垃圾，提桶跑路；第二，主力资金吃相难看，一见新股民仿佛最后一顿吃席，迫不及待收割；第三，“牛”早跑了，不过是短线圈一批新韭菜高位接盘。

这几种观点都负面，属于“市场有坏人，散户必被收割”的看空派，甚至有认为3674就是未来几年高点的极端观点。有许多读者朋友问，节后市场不及预期，到底是什么原因导致的？

从政策面和监管层在刚刚周末的联合亮剑来看，两方面因素都影响了大盘。过去的周六，北京、上海、江苏、深圳四地证监局联手，对“大宗交易违规减持”亮剑，发出了严厉警告。

从这次四地监管局发布的公告来看，针对的是汪小清、孙肖文、魏昌伟违规减持套现多家上市公司股份获利的具体案例和处罚措施。

透过监管局发布的文件，我们也能看到这几位“大神”违规减持的肆无忌惮以及获利丰厚。

汪小清在多地市场违规减持，其中在北交所交易最疯狂，获利也最丰厚。他受让华信永道 93.2 万股，买入金额合计为 3069.1 万元。然后半年内违规减持，将 93.2 万股全部卖出，卖出金额总计 5196.9 万元，这一次违规减持，就获利超过 2100 万元。

汪小清还受让上海艾融软件 130 万股，买入金额 222.4 万元，违规卖出套现 3296.1 万元，一次违规减持，获利就超过千万元。孙肖文同样是受让上海艾融软件，买入 80 万股，金额是 1478 万元，违规减持套现 1755 万元。虽然套现获利仅有三百万，远小于汪小清，但两人大笔受让和违规减持的都是上海艾融软件这样的小盘股，两人的联手操作起到了巨大的操纵与砸盘效果。

在江苏省的处罚公告中，还是汪小清，10 月 8 日大笔受让百佳善达持有的百甲科技 20.69 万股，购入价 99.73 万元。10月9日，汪小清就减持百甲科技股份1.69万股，交易金额8.23万元。

这次违规减持虽然套现金额比例并不高，但是他购入的是超出5%的大宗交易，而且购入和减持就发生在10月8日与9日，很显然是利用节后利好，短期套现获利。

深圳证监局通报的魏昌伟，他的大宗交易受让和违规减持同样发生在10月8日与9日。他10月8日受让深圳市鑫汇科股份26万股，买入金额409.76万元。10月9日，魏昌伟将26万股全部卖出，违规减持套现434.98万元，虽然获利比例不到7%，相比较那两天大幅震荡的市场和破纪录成交额并不显眼，但他触犯的是“受让5%以上股份后，半年内不得违规减持”的规则红线。

北京、上海、江苏、深圳四地证监局给出的处罚措施，都是依据《上市公司股东减持股份管理暂行办法》第二十九条的规定，责令购回违规减持股份并向上市公司上缴价差的监管措施，并记入证券期货市场诚信档案。

直白的说，针对大股东违规减持，处罚就三条：
- 减持的股份全部回购
- 获利价差全部补偿给上市公司
- 记入市场诚信档案

这相当于抢劫银行后，处罚仅仅是需要退回抢劫资金，再把花掉的补回来，就没事了。对于违规减持的个人来说，处罚措施仅仅是将减持的买回去，中间赚的钱退回去，回到卖出前的状态，并没有更严厉的大额罚款警告。包括后期资本市场的运作资质，也没有哪怕短期的“禁业”处罚，仅仅只是记入诚信档案，简直是隔靴搔痒（也许还都不算）。

另外从补偿损失的角度，这些大股东违规减持，导致带崩个股或者大盘，受损最严重的，应该是广大的散户股民和高位接盘者。可是最后这些违规大股东“上缴的获利”全部补偿给上市公司，那么股民散户的损失谁来赔？股民散户的利益谁来保障？

虽然广大散户对监管层的处罚力度很不满，但这却是现阶段根据《上市公司股东减持股份管理暂行办法》能做出的最大处罚了。这样力度的处罚属于“依法办事”，再超出力度重罚，就属于无法可依了。难怪这些人肆无忌惮。下一步散户们要推动的，是在《上市公司股东减持股份管理暂行办法》提升为正式法律的过程中，呼吁加重对违法减持的处罚力度，同时将损失补偿重点向散户投资者倾斜。

只有最尊重散户投资者的市场，保障散户利益的市场，才能让散户意志与国家意志同频共振，跑步入市。

当然，这次四地证监局对违规减持的处罚并不重的另一原因，是许多专家认为，从这些违规减持可以看出，相关市场的监管漏洞比个人投资者的操作“责任占比”更大。

什么意思呢？这就相当于银行柜员机不断向外面吐成捆的钞票，考验个人是否“将钱捡走”的意志。这次监管层的处罚通报，给散户老百姓的第一印象，是这些人的“大宗受让和减持”都没有锁定期的么？

几十万股，几千万的金额，都能做到头天买第二天卖，明显违规。理论上，这些人一旦持股超过5%，这些股份立刻进入T+185天的锁定交易，很难实现么？

很显然是过去的交易系统，存在个人受让大宗股份没有及时锁定，让受让人第二天可以违规卖出的bug，这些个人面对系统bug，以及短期巨额的套利空间，能够忍住不操作的极少。大街上有成捆成捆的钞票，试问有几个人能忍住不捡？

比起处罚，散户更关注如何在后续，堵住相关漏洞。

这次四地证监局对不同的个人违规减持进行处罚，完成了对这种监管漏洞的暂时封堵以及交易跟踪，可是这又回到了前面那个问题，处罚太轻了，如何能堵住漏洞呢？

因为根据投资市场的规律和历史经验，无论如何修订规则，漏洞始终是存在的，这是客观规律。正所谓道高一尺魔高一丈，监管层堵上一个漏洞，只是逼迫这些人用其他更隐蔽手段，进行灰色地带套利获益而已。

就比如散户投资者持续质疑和控诉的韦尔股份，多次大笔受让股份捐赠给还在“纸面推进”的董事长名下宁波某高校。

虽然有慈善捐赠、支持教育等光环，但是各方依旧没有回应散户投资者对高位减持、避税等质疑。举出韦尔股份的例子，并不是也将它判定为大股东违规减持的案例之一，这个判定标准必须也只有监管层可以做出。

但是透过韦尔股份的案例，以及散户投资者的反馈，可以看出散户投资者对“违规减持花样翻新”的担忧。

要想减少类似的花样，除了堵住漏洞，更重要的还得有重重的处罚，才能让他们忌惮。

如果抢银行只需要退回钱，那么没几个人会不抢的。
