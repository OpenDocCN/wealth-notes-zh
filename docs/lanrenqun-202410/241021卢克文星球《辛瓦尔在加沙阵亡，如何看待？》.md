# 辛瓦尔在加沙阵亡，如何看待？

241021 文/卢克文工作室嘉宾 风雨如歌

整理：公众号懒人搜索，懒人专属群独享

懒人微信：lazyhelper

![](img/009df45e112b274ad4762fd5af39ccb0_0_0.png)

10月17日的消息，哈马斯领导人叶海亚·辛瓦尔，在加沙的南部拉法地区阵亡，这是几个月内，哈马斯身亡的第二位领导人了。

整个过程是这样的。

辛瓦尔这个人，向来有不太注重自身安全的缺点，喜欢到前线侦察，类似于李云龙，师长不当，喜欢当突击队长。

这一方面是个人性格使然，另一方面大概是他觉得，自己一个62岁的老头子，还有各种疾病，活不了多久了（辛瓦尔在以色列的监狱里蹲了20多年，被各种折磨，身体不好）。

他公开说过，与其身患新冠、心脏病或者交通意外死去，不如像战士那样死在战场上。所以他很喜欢带着几个人跑到前线去。但一次没事，不代表次次走运。

这一次，他就被发现了。

当时他带着四个人，一行合计五个人，躲在一栋楼里观察IDF，IDF的坦克发现了他们，朝建筑开了一炮，另外四个人当场身亡。

辛瓦尔幸运没死，但是身带重伤，右胳膊断了，只剩下左手还能活动。

他费尽力气，坐到了一处沙发上，随后，IDF 的一架小型无人机飞了进来，看到了坐在沙发上的、奄奄一息的辛瓦尔，辛瓦尔也看到了无人机。

于是他使出最后的力气，捡起一根木棍，砸向无人机。不远处的 IDF 狙击手开枪，辛瓦尔先是左腿中了一枪，然后左上角的额头又中了一枪。

辛瓦尔就此身亡。

IDF 起初并不知道这个人是辛瓦尔，过了一会打扫战场，发现这具尸体的面相和辛瓦尔很像，感到大吃一惊。

辛瓦尔的死，和纳斯鲁拉、哈尼亚有着明显的区别，后两者是被有预谋地暗杀，而前者是遭遇战中的意外，是误打误撞，一同死亡的，还有卡桑旅下属的汗尤尼斯旅指挥官。

辛瓦尔死后，哈立德·马沙尔被迅速任命为哈马斯的代理领导人。2004年，哈马斯的一号创始人亚辛被炸死后，就是马沙尔出任领导人，长达13年。

一直到 2017 年马沙尔才卸任，那十几年哈马斯过得也是非常艰难，现在他临危受命，再度出山，面临的危险和困难不会比第一次小。

在进一步展开之前，我们得先简单介绍一下，当前加沙地带的形势。

在大规模进攻惨败后，以色列的策略改为了围困，就是占据南部的拉法走廊，不让物资进出，还修了一堵厚厚的封锁墙，企图一点点耗死抵抗组织和加沙人民。

哈马斯和其他抵抗组织，当然知道这是要害，不断组织对拉法走廊的反击，这也是辛瓦尔会出现在这里的原因。

但最大的问题是，哈马斯手头上的武器威力不太够，正面刚 IDF 的装甲部队比较吃力。

哪怕是“亚辛-105”，威力也不足以摧毁坦克，虽然穿甲射流战斗部可以让车体内部燃烧，但坦克本身并没有被摧毁，只是被穿了一个小洞，毁坏了部分设备，修一下还能用。

至于车体内的燃烧，只要车上的人跑得快，一般不会死太多。

哈马斯想必很羡慕真主党，真主党手头的反坦克武器是正儿八经的反坦克导弹，“梅卡瓦”这种脆皮被命中会碎一地，不用修了。

和真主党正式开战十八天，IDF 就被摧毁了 20 辆“梅卡瓦”，是真正的摧毁，所以以军连一公里都推不动。

当然，也不是哈马斯不想弄反坦克导弹，实在是敌人的封锁力度大，物资供应困难，我们看辛瓦尔在牺牲的时候，只穿着一件没有插板的防弹衣就知道了。

正常来说，辛瓦尔作为一号人物，穿的防弹衣不可能没有插板（缺乏插板会令防御力大为下降），只能说，关键的军事物资是真的不足。

在这种情况下，有“亚辛-105”已经是极限了。于是尽管拉法走廊的“纳哈尔旅”和“吉瓦蒂旅”几个月来伤亡不小，却始终没能把敌人赶走。

马沙尔必须想办法，解决这个生死存亡的问题，而且外界不会有太明显的帮助，毕竟援助再多，不打通拉法走廊也运不进来。

哈马斯和一众抵抗组织，只能靠自己，这一关过得了就海阔天空，过不了，可能会逐步消亡。

虽然很难，但机会不是没有，哈马斯还是有一些有利条件的。

- 首先，IDF 在拉法走廊的部队也很疲惫了，前段时间还出现过集体请愿事件，许多基层官兵要求撤离，原因是伤亡大，受不了。有人说，IDF 不能轮换的吗？理论上可以，但实际上没啥部队可轮换，以色列陆军常备也就 12.6 万人，北线去了 5-10 万人，约旦河西岸的抵抗烈度越来越高，又部署了 3 万左右。加上国内的守军，IDF 能派到加沙的部队并不多。能出动第 162 装甲师和加沙师两个师继续围攻加沙，已经是极限了，而拉法走廊方向能分到两个旅，是真不少了。如果要新动员部队的话，对经济负面影响又很大。去年四季度，由于动员 36 万预备役，以色列 GDP 负增长了 20%，今年各种预备役陆续回家，但经济状况还是很差，增长基本在 0%上下，无法再承受大规模动员。所以说，哈马斯难，以色列也难，现在双方都只剩一口气，比的就是谁更能咬牙坚持。
- 其次，是加沙地带的抵抗武装保持着团结。要知道，加沙地带的抵抗组织不止哈马斯一家，还有杰哈德、人阵、民阵等大大小小十来支。正常情况下，条件这么艰苦，早就出现成规模的叛变了。然而事实却是，它们奇迹般地保持了团结，一致对外，没有出现大的叛徒。
- 三是地道体系还在运作。尽管拉法走廊下面的地道被 IDF 发现了不少，但还是有一些没有被发现的，按照 IDF 自己的说法，是发现了八成，那就还有两成没被发现。其实八成这个说法，肯定有夸大。但具体不管几成，有一部分地道还能运作是必然的，否则抵抗武装的后勤早就断了。

如果马沙尔在上任后，能利用这些有利条件，组织一次针对拉法走廊的攻势，类似于“阿克萨洪水”，重创拉法走廊的 IDF，让自己喘口气，局面或许会有大的改观。

这听起来非常难，但必须要做。

其实一年前，没人能想到今天，没人觉得哈马斯能和“强大的以色列”打上一年，而奇迹之所以会出现，关键在于领导者的身先士卒。

按照以色列的宣传，哈马斯领导人都是在卡塔尔的大别墅里享福的，但今天的事实很好地证明了，人家辛瓦尔并没有躲在卡塔尔的大别墅里。

人家一直呆在加沙，呆在最艰苦的前线，哪怕身带重伤，没了一只胳膊，哪怕手上只有木棍，也依然要用最后一丝的力气去战斗。

什么是战士，这就是战士。

只要这股子精神还在，别说一年，就算再打上一年，也不是不可能。而哈马斯并不需要消灭以色列，就像塔利班不需要正面消灭美军一样。

以色列也是强弩之末了，长期处于战争状态，它的经济早就受不了了，哈马斯和一众抵抗组织要做的就是咬紧牙关，拖下去。
