

# 从黎巴嫩南部撤军，以色列的“北伐”结束了？

10月28日，以色列从黎巴嫩南部撤出了四个师的部队，考虑到对黎巴嫩南部的进攻合计不过动用五个师，撤出了四个后，还在黎巴嫩境内的就只有一个第146预备役装甲师。

一个师是做不了太多事情的，何况还是个实力不怎么强的预备役师，这轮气势汹汹的北伐，基本上是结束了。那么问题来了，打了才不到一个月，怎么就停了呢？以色列取得了多少战果？后续会怎么应对呢？

我们无数次说过，真主党不好打，它是正规军，组织的严密程度和装备都比哈马斯强得多，黎巴嫩南部还是个经营多年的堡垒区。

进攻这样的地区，注定是要头大的，这一点别看以色列政府口头上说得很牛，其实他们私底下是知道的，毕竟双方打交道了这么多年。

在地面进攻开始时，以色列政府就说得明白，是“有限军事行动”。

在“阿克萨洪水”后，真主党为了策应哈马斯，在北线不断发起各种骚扰，今天来一发导弹，明天打个无人机，搅得以色列的北部没法睡觉。

当地至少有十万人逃离家园，以色列的目标是把真主党推离边境线，越远越好，建立一个缓冲区，好让逃离北部的人敢回去。

所以叫“有限军事行动”。

虽然叫“有限军事行动”，但以色列的投入一点也不少，合计有四个半师，包括第36装甲师、第98伞兵师、第99预备役步兵师的一部、第146预备役师和第91领土师。

有必要说一下，以色列的陆军战斗力分为三个档次：
- 第一档是装甲师、伞兵师这种常备部队，具备完整的野战能力；
- 第二档是预备役师，只有部分野战能力，战时无法作为独当一面的部队使用，只能给常备部队打下手、做辅助；
- 第三档是领土师，没有野战能力，纯粹是守备性质，进攻的时候只能当啦啦队。

整个IDF（以色列国防军）合计有9个师，也就是三个常备师、三个预备役师和三个领土师。

调了五个师去打真主党，其中两个常备师，可见十分重视。

但进攻的第一天就出了幺蛾子，连续遭遇伏击，损失不小，光是公开的死亡就有8个，五个是东线，也就是加利利手指方向“坚果”特战部队的。

有意思的是，此战过后，“坚果”的指挥官被撤职，新来的指挥官发现，士兵对于和真主党开战的抵触情绪很大，指挥不动大家。

整个“坚果”部队事实上不动了。在“坚果”的影响下，第98伞兵师整体出现了出工不出力的状况。

中线的进攻由第36装甲师主导，也在开战的头两天遭遇了重大伤亡，还被击毁了多辆坦克，从此变得小心翼翼，前进速度堪称龟速。

第36装甲师和第98伞兵师作为主力，是需要起到榜样作用的，当榜样起不到太大作用时，打辅助的预备役师和领土师也只能两手一摊了。

到10月下旬，IDF仅仅控制了一个村庄，这个村庄还不是拿下的，是整条村的村民都是基督徒，和真主党不对付，看到IDF来了就开城门带路。

总控制面积不到30平方公里，平摊到各个进攻方向上，还不到一公里的前进幅度，收获很小，伤亡却不小。

按照真主党的统计，进入黎巴嫩南部的IDF总伤亡750人，其中死亡55人，即使是IDF自己发的讣告，死亡人数也超过了30人。

且和加沙战事一样，阵亡者中的军官比例过高，如此看来，IDF在黎巴嫩南部的真实伤亡，大概率和真主党所说的差不多，也就是两个营的伤亡。

这个数字并不全面，因为只统计了进入黎巴嫩南部的伤亡人数，但真主党每天都会向以色列境内发射各种炮弹、无人机，造成的伤亡也是有的。

比如10月13日，真主党的一架无人机击中了以色列境内一处军事训练基地的食堂，当时正好是用餐时间，室内人员密集，死伤过百人，光救护车就来了50多台。

加上其他的零敲碎打，IDF和真主党开战不到一个月，总伤亡数字应该在千人左右。

要是仅仅一千人，那不至于要撤军，问题是IDF已经和哈马斯打了一年多，总伤亡1.5-2万，很多部队在加沙的伤亡就大，再被真主党来几下，就真的没心气了。

比如第1“戈拉尼”旅，它在“阿克萨洪水”当天就伤亡了300多人，后续在加沙打了三个月又伤亡了两到三个营，旅长、营长、连长、排长更是死了个遍，指挥系统和老兵折损严重，不得已才撤到北部修整。

结果到了黎巴嫩南部，第1“戈拉尼”旅在进攻第一天就被伏击，损失了一个排（40人），后续又损失了一个排，上面所说的食堂遇袭，挨打的也是它。

倒霉事还没完，伊拉克抵抗组织的另一架无人机，击中了“戈拉尼”旅的宿舍，又带走了一个班。

作为王牌的第1“戈拉尼”旅，其实早就没心气了，光是恢复士气就得一年半载，这个节骨眼上，你让他们去打比哈马斯强得多的真主党，那自然是出工不出力。

不光“戈拉尼”旅，其他部队也差不多，再打下去也没多大意义。

这不是单纯的军饷问题，是复杂的心理问题，哪怕临时提高工资也没用。正因为没多大把握，以色列官方才会说是“有限军事行动”。

10月25日，IDF的总参谋长哈列维中将，在视察前线后发表讲话，表示“黎巴嫩南部的军事行动可能彻底结束”，两天后，就传来了撤出四个师的消息。

留一个第146预备役师，是名义上表示“北伐”未结束，我还会回来的，但明眼人都知道，已经结束了。

而这轮“北伐”失败也证明了两点。

真主党虽然高层受到了重创，但基层的战斗力还是很强，这与真主党的分布式组织形态有关，高层很多时候并不会对基层进行具体指导，顶多做宏观调控。所以在高层动荡后，基层的战斗力并不会被清零。以色列想仅仅靠暗杀高层，来瓦解真主党，是不现实的。

有句话说得好，不是有了领袖才会抵抗，而是抵抗产生了领袖，纳斯鲁拉不过是黎巴嫩什叶派民意的代表，你把他炸死了，又能怎样？

靠以色列自己，是没希望摆平真主党的。

可如果不摆平真主党，北部的居民就不敢返回，这些人的吃喝拉撒都得以色列政府补贴，是一笔巨大的开支，唯一的办法还是得请美国下场。

最新的消息是，以色列政府正在和美国国家安全顾问沙利文谈判，以色列要求美国出面主导，从海陆空三个方面，全面封锁黎巴嫩。

是封锁整个黎巴嫩，不让物资进出，而不是仅仅封锁真主党的地盘。

毕竟真主党就在黎巴嫩，你如果只封锁它的地盘，如同不封锁逊尼派和基督徒的地盘，真主党还是能搞到各种物资，必须一块封锁。

拜登在白宫也不剩几天了，这么大、这么缺德的事不一定会答应，但懂王要是上台的话，答应的概率不小。

问题在于，黎巴嫩又不是个孤岛，它和叙利亚接壤的边境线很长，请问咋封锁？难道连叙利亚也一块封锁了？

不让物资进出？

没有那个能力知道吧。

不客气地说，所谓海陆空封锁黎巴嫩，就是个笑话，顶多在一些关键领域进行制裁，对身背一大堆制裁的真主党来说，早就习惯了。

虱子多了不痒。

未能趁着真主党高层被重创的有利时机，在地面扩大战果，等过段时间，真主党缓过劲来，以色列的麻烦就大了，其北部边境将再无安宁。

即便是距离边境比较远的中部地区，也会时不时来一发，整个以色列国家的战略态势，将持续恶化。
