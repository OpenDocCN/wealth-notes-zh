# 前言
> 这就像一个跷跷板，彼此是共生关系，用赶尽杀绝的方式干掉对方，这不是博弈，这是自杀!

今天所有的人都在谈论着美国的去工业化，而且是以嘲讽的口味。似乎美国人非常愚蠢，不知道完整的工业体系对一个国家综合实力的决定性影响。事实上，美国的去工业化是一种被迫无奈，而不仅仅源于资本的贪婪。

如果一个国家的税收高达90%，你会不会觉得这个国家疯了？你会不会认为在这样高的税率下，任何企业都无法正常经营？这就像和尚头上的虱子——明摆着！然而，美国曾经的税率就一度高达90%！是不是不可思议？但是你去查历史记录，这就是事实。

在苏联压过美国占据主导的时代，英国的工会力量和美国的工会力量，大到你无法想象。动不动就罢工，并要求加薪。在铁娘子上台以前的英国，各种福利制度是今天的英国人不敢想象的！直到今天，上了年纪的英国普通人都在怀念那个时代。所以他们非常痛恨撒切尔夫人，撒切尔夫人去世的时候，他们甚至上街庆祝。今天美国的主流媒体都是夸里根的，但经历过一个蓝领工人都可以养活全家并买得起汽车时代的红脖子群体恨死里根了！

里根和撒切尔其实就做了两件事：
- 把约束资本的所有限制全部清除。
- 搞垮了苏联！

前者让国内的金融资本取代产业资本获得了空前的发展，同时也让贫富差距迅速扩大，并形成不可逆的固化趋势。后者让蓬勃发展的金融资产找到了价值之锚——苏联解体之后，它的整个工业产值以及油气和矿产资源，为泛滥的美元和英镑提供了价值支撑。同时，还打开了中国的大门并找到了一个巨大的增长洼地。

这几个关键点都是环环相扣的，缺一不可。不管是打开中国的大门，还是苏联的解体，任何一环缺失，都不足以让放开限制的金融资本变成控制全球的美元体系。美元体系要成型，必须要伴随着产业、金融和贸易的全球化。在这个过程中，生产和贸易是基础，金融是让整个过程的资源配置更加高效的润滑剂。一旦没有了下面的基础，上面的那个就无法长久——当下的事实正在呈现这个结论。

许多人会告诉你美国和英国的去工业化是资本家为了追求更高利润的结果。这是其中的一个原因，但不是最核心的原因，只是一个顺势而为的原因。最核心的原因是什么呢？是产业工人动不动就罢工，让企业受不了。在苏联占据主导的阴影下，不管是英国还是美国都主动的向工会和工人让渡利益，以避免本国的工人群体在苏联的宣传下揭竿而起。当时的英国政府是极度软弱的，面对动不动就罢工的工会，一点办法都没有。美国如果有办法，就不至于会出台90%税率的法案！

在这种情况下，美国的企业主开始向国外转移工厂——没有了工厂，就不会有工会，没有了工会，工人就不会作为一股独立的势力存在！这个事情其实在70年代中期之后就开始了。亚洲四小龙承接的就是美国的产业转移，从劳动密集型开始——因为劳动密集型产业，需要的工人数量最多，也是工会力量最大的产业。所以他们率先向东南亚转移，随后又开始向中国大陆转移。这种产业转移，会转移设备、资金、技术和业务，唯独不会转移的就是工人！市场还是美国的市场，但工人却用的是别人的工人。成功接受这种转移的地方当时都处于威权状态，纪律性好且劳动力廉价，生产资料的成本低，而且还有税收优惠。如此一来，转移的企业主不仅可以保住自己的业务，甚至可以赚更多钱。最重要的是：再也不用看工会的脸色了！

任何博弈都需要有一个平衡，才能持续和长久。这就像一个跷跷板，彼此是共生关系，你把它搞没了，你也就无法存在了。

今天美国的红脖子们处境确实艰难，那是因为他们的父辈把树砍了，他们这一代就只能暴晒！从80年代中国的大门打开，产业转移就开始从东南亚四小龙向中国大陆这个更大的腹地迁移。它不仅能够像东南亚四小龙那样承接，还能消化和升级。而且人力资源、土地资源更便宜，政策倾斜也更猛。同时，还是一个巨大的潜在消费市场。所以，它不仅吸纳产业和技术，能够承接巨大的业务，对资金的需求量也非常大，而且投资回报率非常高。

如果说80年代的产业转移还只是春风化雨。那么整个90年代就是狂飙突进了！91年苏联解体，92年南巡提出加快改革开放的步伐。这并不是巧合，而是因为苏联解体之后，美国再无顾忌，对金融资本的各种约束只会更加宽松，而美元全球化的扩张必然伴随着生产和贸易的转移。在这个过程中，加快改革开放的速度和力度，就是为了顺应这个趋势，承接更多的产业、技术和生产的转移，并吸纳更多的投资。

我们确实恰到好处地适应了这个趋势，而且在这个过程中，用一种近乎变态的方式，约束着劳动者和基层组织，让我们忍受着各种压榨，也要优先满足外资的利益，从而让他们愿意把相关的产业及配套都转移到这个最有利于生产和活力的环境并逐渐完成定型。因为产业与产业之间形成的供应链会相互嵌套又相互支撑。嵌套的越多，综合成本就越低，综合效率就越高，作为一个整体的竞争力就越大。在这个过程中，需要外部的资金和技术持续输入，唯一能够让对方持续输入的动力就是投资的高收益！

在2015年之前吸纳的外部投资一直都在增长。2015年时投资回报率逐渐降低，生产成本逐渐增高。规模以上工业的综合利润率已经在1%左右盘旋，而银行的长期贷款利率远超这个数，外资开始逐渐撤离。我们开启了为期三年的用长期债换短期债的操作，就是为了让地方有钱运转！同时搞了一波降杠杆的操作。还搞了一波涨价去库存的操作。三波操作搞完，本来是准备迎接一个新的宽松起点。结果被19年的一场疫情意外打断了节奏，后续的一切都是在堵漏。目前各方面最大的努力目标就是回到2019年的水平!

从官方的统计数据中可以看出一个隐藏在数据中的事实：这几年的民间投资增长率为0，所有的增长都是官方投资拉动的!限于篇幅和其他原因，不能就此进行深入阐述。但有一个规律一直存在：水至清则无鱼，任何伴生关系的事物，所有的博弈都必须是在一个平衡的状态下。
