

# 也聊自由

里面举的奢侈品的例子挺有意思，奢侈品的溢价是别人的羡慕。换一个角度讲，只要我们不去羡慕，是不是他们的钱就白花了哈哈。

网上看人聊财务自由时，结论往往天差地远。

常见有人说个不高的数字，倒如此一世便吃穿无忧。

自然会有人横空出来反驳，说这点怎么够？“买套房就没了！”“买辆好车就没了！”“每年买个包包就没了！”“都不够每年出去旅游！”“孩子去海外读个书，回来房子都买不起了！”

大概有人将自由理解为无忧无虑，只求有一日终身吃穿无忧，不用悬心于柴米油盐日常用度，就能心满意足。

有人将自由理解为顺心如意、应有尽有：买得起心头好，享受到大热闹，颐指气使有人答应，才算不负一世。

所以这其实是两种自由：

“免于操心生存的自由”和“满足自己欲望的自由”，后者显然更困难些。欲望这事，几乎没有上限，这山望着那山高。

《红楼梦》四十三回，老太太给王熙凤过生日，“咱们大家好生乐一日。”庚辰双行夹批如是说：贾母犹云“好生乐一日”，可见逐日虽乐，皆还不称心也。所以世人不论贫富，各有愁肠，终不能时时遂心如意。

——大概，再是外人看来锦衣玉食荣华富贵大概够开心了，到底也不能“时时遂心如意”，到底也会“不称心”。

魏晋之间，有个大人物羊祜。亲戚堆里是蔡文姬、蔡邕、司马师、夏侯霸的女儿，富贵已极，位高权重，容貌风度名闻，身前身后功业卓著，甚至他的对手、东吴名将陆逊之子陆抗，也跟他惺惺相惜。人生堪称完美，然而他偏说：“天下不如意事恒十居七八。”

大概人力有限，而欲望无穷。

能免了吃穿用度的担忧，已属幸运；真要追求顺心如意，连羊祜和贾太太，都难以“时时遂心如意”呢。

所以又要回到“期望管理才是幸福之源”、“清心寡欲才能开心”的老道理吗？却也不是。

本来，有人追求柴米油盐，有人追求豪车游艇，各有各的自由。可以两不相犯嘛。

但为什么有人总要传递焦虑吹嘘欲望呢？

当然，其中有些是为了忽悠人以便发财，但还有一类人是这样的……

奢侈品行当有过个有趣的悖论：比如发宣传册给一百个人，也许其中只一人买得起，那剩下九十九份宣传册，不是明珠暗投浪费了吗？然而那九十九个拿了宣传册却买不起奢侈品，只能艳羡的旁观者，也是奢侈品的附加值。

“没这些知道这玩意多贵却又买不起的人们，显摆给谁看呢？”

毕竟有钱人并不是笨蛋，他们自然知道奢侈品的溢价；但奢侈品的意义本也不是产品质量，而是制造社交距离。“要让你一眼就知道我们不是一个阶层的。”

所以咯，有些人花钱是买自己的需求，有些人花钱是买他人的羡慕。

这事古已有之。项羽入咸阳分封天下后，还是决定东归彭城，还说了所谓：“富贵不归故乡，如衣绣夜行，谁知之者！”衣锦夜行这个成语，就这么来的。人嘛，成功了，总得显摆给人看吧？

然而在这个世上，各人有各人的生活目标。有人觉得粗茶淡饭了此一生也就罢了，也有人已经投入了巨大心力，以求获得他人的羡慕。

如果艳羡的目光并未如期而至，难免会觉得衣锦夜行。

我知道这点，是因为某次饭局，跟一位阿姨聊天。我性子疏懒，觉得饱食终日无所用心就是人生至乐了；那位阿姨却积极勤恳，要让全饭局的人都知道，她家的孩子骑马、弹琴、打高尔夫。大概是满桌人只顾吃，阿姨甚不快慰；看能跟我唠几句，就抓着我聊。等她觉得我是个可以推心置腹的人时，就公开表示了对满桌人的不满：

大概，饭搭子们对孩子的才艺表现出的艳羡程度，没达到阿姨的期望吧。

我劝阿姨说，毕竟也不是每个人都对骑马、弹琴、高尔夫有兴趣的……

阿姨颇为义愤填膺，觉得大家都对更高一级的爱好和趣味缺乏好奇心，实属无知无识，不求上进。说到最后，“晓不晓得我每个月要为这些花多少钱啊！”

我明白过来，赶紧奉承，“那可不是一般人家花得起的！”

阿姨懊恼欢欣地道：“是的呀！！！”

我有个不太厚道的揣测。

世上有许多人是真喜欢自己想要的东西，于是大踏步地朝自己的理想奋斗去了。

但也有许多宣扬焦虑、“这点钱买套房/买辆车/买包包/海外读书就没了”的诸位，不一定是想为难别人，也可能是业已踏上了追寻高尚生活与艳羡目光的不归路，于是希望把周围胸无大志只想平安度日的人们催醒。“我已经进了这坑，凭什么你们可以不用进？”

毕竟，“我努力那么久，投入那么多，已经那么累了；你们为什么只顾躺平？凭什么你们那么小富即安就能觉得开心，却不来羡慕赞美一下付出了那么多努力的我？”

昨天说这个时，有读者回了句黑神话梗，极好：

> “若不披上这袈裟 世人又怎知我尘缘已断 金海尽干？”
