STAR原则按照：
- situation 15%（说清楚时间、地点、人物、事件）。
- task 15%（自己的岗位和负责要做的事情，重点阐述这个任务的困难和挑战是什么）。
- action 40%（列出1、2、3、4点，每一点做了什么和为什么要这么做）。
- result 30%（事情的结果最好数据化，在这个事情中学到了什么，可以在以后改进后的东西）。
