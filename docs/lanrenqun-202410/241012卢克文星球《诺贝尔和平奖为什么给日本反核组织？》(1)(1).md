

# 诺贝尔和平奖为什么给日本反核组织？

241012 卢克文工作室嘉宾 漫游指南

整理：公众号懒人搜索，懒人专属群独享

懒人微信：lazyhelper

昨天，诺贝尔委员会宣布，把2024年的诺贝尔和平奖颁发给日本原子弹氢弹爆炸受害者团体协议会（后面简称：被团协）。

挪威诺贝尔委员会表示，日本被团协获奖是因其为实现无核武世界而努力，并通过亲历者的证词来表明决不能再次使用核武器。

听起来似乎很合理，但是再仔细一想就不不对劲了。

委员会只提了核爆对日本的伤害，却只字未提为什么日本会遭遇核爆。只表彰被团协对世界无核化做出的努力，却忽略了正是因为军国主义才让日本遭受了如此苦难。

那和平的敌人到底是核武器本身还是军国主义？给被团协和平奖这不是在给日本军国主义洗白吗？

再看看颁奖委员会的人员，就能发现端倪了。

评选诺贝尔和平奖的5人由挪威议会投票选出，5人的身份都是挪威议员。而其他奖项的评选都是该领域的顶尖专业学者。

5人中的托耶（Asle Toje）在从政前是一名外交政策学者，猜猜他现在在研究什么？是的，他正在研究核裁军和无核化。

5人中的新组长弗莱德内斯（Jørgen Watne Frydnes）是84年生人，是诺贝尔和平奖评选委员会历史上最年轻的组长。从政前是一个白左智库的CEO，现在是一个白左作家协会的主席。

你看看，让这帮人评奖，能评出什么好来？这就导致和平奖相比其他诺奖有着强烈的西方叙事倾向，其他奖项可以承认不同文化圈子的成就，但是和平奖只在西方的语境表彰他们心中的和平。

不光评奖的有问题，获奖的也不干净。

被团协的前身是47个广岛和长崎当地核爆受害者协会，这些协会合并在一起成立了被团协。

现在的被团协有两大诉求，一是世界范围内销毁核武器，二是要求日本政府赔偿受害者。

听起来这2个诉求没什么问题对吧？但是这两个诉求根本就不是当初那47个核爆受害者协会合并时提出的诉求。相反，这两个诉求是被团协在和日本政府斗智斗勇好多年后总结出来的最优解。

在90年代前，“被团协”基本是被无视的组织，因为他们除了反核，还反对帝国主义、反对美国的占领，有点反美色彩。这时候，当然不能让他们获奖。

但苏联解体后，全世界的左翼组织逐渐失去了统战价值，不断被改造，变成了白左，“被团协”也不例外，早就不是原来的人了，逐渐放弃了反美色彩，成了单纯的核组织。

而且重点是为日本二战的事翻案，也正是这样，才能获奖。

类似于被团协获奖的事情在今年已经不是第一次了，在今年8月初的广岛核爆纪念仪式上，主管裁军的联合国副秘书长，同时也是日本籍的中满泉，在讲话中全篇只提了核爆对日本的伤害，也是只字未提为什么会有核爆。

这种故意淡化日本二战侵略事实，以及故意淡化二战胜利的法理正当性，早就不是第一次发生，在未来也会再次发生。

在二战投降后的新日本，执政时间最长的政党是自民党。自民党顺着历史往上追溯，就是发动了侵略战争的立宪政友会，所以所谓的新日本不过是旧日本套了一层皮。

既然新日本和旧日本不过是换了层皮，那旧日本在二战作的恶，是新日本摆脱不掉的，也不愿意承认罪行，所以在编写教科书的时候，故意淡化了二战部分的历史。

日本人要想了解日本在二战中的作为，就必须得自己去挖掘。但是人的精力是有限的，就像不少国人并没有兴趣完整了解清朝历史，大部分日本人也不关心和他们生活无关的二战史。

历史教育是给日本本国人洗脑，那么对日本以外的外国人，日本政府通过文艺作品来有意洗白二战，把重心全落在日本普通民众在二战中遭受的苦难，但是只字不提为什么会有苦难，以及造成苦难的人有没有得到应有的惩罚。

日本NHK拍的纪录片《日本海军战败反省会》就是一个完美的例子。这部纪录片在日本以外，没什么名气，但在日本内部，可以说代表了主流叙事，拍的是战后一群旧日本海军军官聚在一起研究为什么日本战败。

其中的内容可以说，一点也不认为自己有错。

这群旧海军军人在研究后，得出的结论是，海军参谋和日本高层盲目自信、刚愎自用，完全不清楚能不能打赢美国就挑衅美国，从而导致日本战败。这里面只字未提日本军国主义，丝毫没说日本侵略中国和其他亚洲国家，只说日本战败是因为不自量力地惹了美国。

这还已经算是日本人自己拍的文艺作品中对二战的描述最为客观的内容了，相比于《男人们的大和》和《永远的零》这两部电影，纪录片里的海军军人至少是真的在反省战争。

《男人们的大和》和《永远的零》把重点落在了普通的日本军人身上，但是电影把片中的日本军人塑造成了保家卫国的英雄，而不是侵略别国的恶魔。

而日本自民党现在正在积极推进日本修宪，要去掉和平宪法中对日本放弃进攻权只保留自卫权的条款，让日本军队正常化。

为了完成修宪，自民党需要先淡化民众和世界心中日本在二战中的罪恶。所以就有了故意淡化历史教育和故意在错误方向讲述二战。这里我们且不说从自卫队到一支有打现代战争能力的军队，日本有多少课要补。至少自民党在舆论上的准备是做好了。

但是现在日本政府的洗白相对而言是小事，美国对日本洗白的纵容才是大事。很明显美国为了自己在东亚的利益，不惜掘自家祖坟，在二战这种大是大非的问题上，对日本军国主义让步。

电影《硫磺岛家书》就是一个完美的例子，这部电影也是把重点落在硫磺岛上的普通日本军人，也是只讲述日本人在二战中受的苦，却只字不提为什么会苦。

整部电影最重要的问题在于，导演和制片都是美国人。

只有演员是日本人。

换句话说，这是美国的意思，美国非但不反对，还鼓励日本翻案。

在全世界，特别是西方的历史教育中，日本在二战的罪行大都是一笔带过的，根源还是政治需求。

因为西方的政治需求，犹太人在二战中的遭遇被反复提及、反复强调，甚至每年都有关于犹太人被歧视或者犹太大屠杀的文艺作品跳到你我眼前，不断地强调犹太人在二战中的苦难经历。

而中国乃至整个亚洲在二战中遭受的苦难却不被世界所熟悉。现在，随着反华的需求日益迫切，放出日本军国主义，就成了新的拉拢日本的筹码。

不得不说，历史都是当代史这句话，是永恒不变的真理，任何人对于历史的态度，取决于现实的需求，你想伸张正义，是不可能乞求别人的，没人会在乎。

只有靠自己。

这是一场认知战场，一场话语权的战争，正如二战的胜利给了中国法理上的正当性一样，我们也只能通过赢下这场话语权的战争，才能让日本对中国造成的苦难，被世界所熟知。

“残暴的欢愉终有残暴的结局”，对历史的藐视和篡改，最终还是会报应回日本和美国身上。但是我们不能只依赖时间，自强，才是一切的根本。
