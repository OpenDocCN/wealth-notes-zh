

# 长征路上的超市：咪蒙是怎么重出江湖的？

240902 文/卢克文工作室嘉宾 流离
整理：公众号懒人搜索，懒人专属群分享
懒人微信：lazyhelper

国庆节期间，一部“红色奇幻”短剧火了，它就是在抖音爆红的《我在长征路上开超市》。

从9月27日抖音号“我在长征路上开超市”发布首个视频到写下本文的10月8日，短短十一天，它已经斩获了442万粉丝和4735万点赞，5集的系列短剧仅在抖音就达到了1.4亿播放。

只经历了一个国庆假期，这一网红短剧的热度就已经达到了现象级。

为啥它能这么火？原来上映之前，就有官媒提前做宣传了。

9月30日，“首都广电”官方微博发布的微短剧国庆档推荐片单上，《我在长征路上开超市》赫然位居榜首。

10月4日，人民文娱官方微博也发布了剧评，高度赞扬这部短剧，各大媒体纷纷转发，一时间《我在长征路上开超市》似乎成为了官媒点赞，网民喜爱的香饽饽。

然而，这次人民文娱显然翻车了，微博网友并不买账，评论区战火纷飞，喷得这条微博先是删除评论，后是开启评论精选，之后直接删除，闹了个草草收场。

《我在长征路上开超市》的评价在互联网上俨然已经割裂成了两个世界。一边是官媒宣传，粉丝狂赞，另一边则是被网友大加鞭笞，激烈批判。从时间上来看，赞扬的声音一开始很大，后面越来越小，而批判的声音则随着时间推移，占据上风。

那么，我们就回到这部短剧本身，看看它究竟哪里惹了争议。

这部剧改编自知乎盐选小说《苏梅梅的超市》。现代女大学生苏梅梅在“万界超市系统”的帮助下，让自己和自家超市回到了长征时期，为红军提供各式物资援助。

就像“给诸葛亮送无限泡面助力北伐”一样，将时空穿越和革命历史结合起来也是国内互联网的一个经典话题。回到过去给革命先烈送点什么，成为网友们经常想象的一个朴素爽点。

然而出品这部短剧的十月初五影视传媒公司，硬是把这么一个大有潜力的话题，做成了一盘没洗干净的九转大肠。

剧中红军战士用枪指着手无寸铁的女主，还自称“红军爷爷”；小战士拿起包子就吃，被阻止的理由不是“不拿群众一针一线”，而是怀疑包子有毒，先拿给女主试毒；冲锋时政委喊的是“都他娘的不能给老子怂，给我冲”……

这些事例，每一条拎出来都是严重违反我军“三大纪律八项注意”的。

这支对友好群众说话不和气，和超市进行不公平买卖，甚至满口粗话的“红军”，简直像一支换皮国民党军队，令很多观众深感愤怒。

评论和弹幕纷纷指斥：“这真的是红军？不要侮辱先烈！”“三大纪律八项注意是用来违反的吗？”

而各种强行制造误会的情节也让人怀疑主创者是否在偷偷抹黑：面对战斗，保护发电机的战士不说原因，只说了一句“你们打，我先跑了。”战斗后吩咐战士“给老子点个烟”的班长，片刻后才发现他已经重伤牺牲。明明可以用更好的方式呈现，它非要让人误会红军那么一下子。

我军一向倡导男女平等，剧中却流露出一种“女士优先”的思想，甚至还把生孩子和中枪拿来做疼痛对比。本来，生孩子和为革命流血都是伟光正的，没必要踩一捧一，但它偏要那么做，显然是故意的。

而这家十月初五影视传媒公司的法人马凌，正是当年微信公众号的“女权宗师”咪蒙。上次咪蒙出现在大众视野，还是在 2019 年的封杀风波中，此番重出江湖，她已经成功转型为短剧制作人。

最惊人的还是，从公众号到短剧的“华丽转身”，她到底是怎么做到的？2019 年，公众号被封杀，她的公司解散后一个月内，她和她的团队一直在开会研讨接下来的发展方向。

最终他们决定了新的发展方向：拍短视频。短视频的优势就是制作成本低，观看成本也低，更容易吸引日益浮躁，就图个爽的网民。虽然当时认定这个赛道是新的风口有赌的成分，但反正公众号做不得了，索性进军短视频。

赛道是新的，但是套路还可以用老的，伙伴也可以是老的。她联合了老合作伙伴张静思，在 2019 年 6 月成立了十月初五下属的 MCN 机构银色大地，来孵化、运营短视频账号。

其中咪蒙负责幕后操作，张静思则担任法人和执行董事。短视频账号的内容，依旧是集中在生活、情感和职场等领域，套路还是以紧跟热点，调动情绪为主。熟悉咪蒙的朋友都知道，当年这套她早就在公众号上玩烂了。不过转型总有阵痛期，短视频看似简单，想写好一个有流量的剧本并不简单，何况2019年对于短视频来说，风口尚未到来。

据咪蒙自己说，那一年她的公司亏损了1800万。但随后新冠疫情到来，让短视频赶上了一波意外的机遇。长时间的疫情管控，让无所事事的人们更钟情于短视频，实体经济受创也给了互联网经济更多发展空间。

经过了失败的磨炼，提早起步的银色大地，迎来了它大举扩张的时机。

2022年，银色大地的营收超1.6个亿，仅旗下的达人账号“四大天王”姜十七、乔七月、聂小舟和浩杰来了，一年的营收，基本就达到了这个数字。这“四大天王”在抖音粉丝加起来有7000万之多，成为了咪蒙的主力摇钱树。

但咪蒙并没有止步于此，她和张静思开始琢磨起了推出付费系列短剧。

这种短剧成本低，最多几十万，套路性强，专搞复仇、逆袭的爽点，免费桥段在高潮时戛然而止，还更能吸引观众付费。

去年十月，十月初五旗下的短剧团队“听花岛”推出了第一个爆款付费短剧《腹黑女佣》，24小时内，充值金额就达到了2300万元。

随后一个月，又一个爆款《黑莲花上位手册》诞生，虽然由于复仇情节用力过猛，几天后下架，但也产生了2000万的巨额收益。

在《我在长征路上开超市》之前，仅听花岛团队就创作了至少14部系列短剧，像《我在八零年代当后妈》《闪婚老伴是豪门》都是红极一时的大爆款，吸金无数。

复仇逆袭，扮猪吃老虎的套路屡见不鲜，早几百年前《灰姑娘》就是这么玩的，“歪嘴龙王”也成了热梗，但在女频赛道玩得这么风生水起的，不多。

而且咪蒙的团队也不是无脑复刻老套路，总能整点新花样。《腹黑女佣》《黑莲花上位手册》更多模仿了《甄嬛传》那样的自强大女主，《闪婚老伴是豪门》则切中中老年赛道，挖掘中老年群体的精神消费潜力。

不得不说，咪蒙修炼短视频多年，也算得道了。但她初次尝试离开舒适区，进军爱国红色赛道的《我在长征路上开超市》却是碰了个钉子。

人民群众的需求催生了“援共”题材，但不管你在虚构作品中给先烈们带了啥好东西，决不能犯历史虚无主义错误，没有他们的奋斗，哪有我们在这里？把现代主角塑造成从天而降的救世主，而不是强调当年红军的艰苦奋战和革命精神，就是一种“倒反天罡”。

像《我在长征路上开超市》这部带有现代人居高临下态度，歪曲塑造红军形象和商品拜物教的短剧，令人不快的原因就在这里。

革命历史题材，天然带有严肃性，更肩负着爱国主义、革命精神的传承任务，不是能随便戏说胡说的。当然，也不是说这一题材就要束之高阁，《地下交通站》《智取威虎山》都是大众认可的佳作，《开超市》拍不好，那是它又坏又菜。

就算有人携资本之能，媒体之威，试图瞎拍乱拍，这么多年的爱国教育红色教育，人民群众也不是白受的，想让大家丢掉脑子忘却历史，没门。
