# 引流精髓 | 给一半 | 留一半

心竺公考齐老师 提问：

抛一个问题，如果要做公务员考试培训的业务，很看好通过群，公号，知乎等渠道来吸引用户。

想知道下，怎么找到这些精准用户，怎么能够快速吸粉，微信公号怎么做。

回答：

核心就是把自己想象成用户，可能在什么地方出现，就去这些地方。

然后看他们需要什么，给一半，留一半，剩下的一半到你这来取。
