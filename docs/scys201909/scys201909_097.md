# 搬运的频道：如何二次利用、如何赚钱？

这个频道是群友发出来的，我一看，频道因为是搬运已经被取消获利了。

最近更新的视频是在 9 个月。

我觉得这样的频道这样浪费，非常可惜。

频道已经有 4.5 万的订阅了，而且是那种情景剧的订阅粉。

这个频道完全是可以二次利用起来的， 还可以赚到合理的钱；

如果是我怎么操作这个频道呢？

第一，我找内容产出方，是情景剧那种，双方谈一个合理的分钱比例。

第二，引流导粉或卖掉。这个订阅至少能卖 4500，但这么久不更新，卖出有点难。

第三，自产内容，这个方式要求蛮高的，自制短剧不是一般团队能做的了的。
