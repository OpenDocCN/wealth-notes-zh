**交易1：**
2012年4月1日，吉拉德决定投资200，000美元，帕克出资50，000美元，以购买新公司总共25，000股股票。股票的面值为1美元。

![](img/f4f37ad4c6bfa8d1dc6b823a6fcfcb53_108.png)

![](img/f4f37ad4c6bfa8d1dc6b823a6fcfcb53_110.png)

![](img/f4f37ad4c6bfa8d1dc6b823a6fcfcb53_112.png)

![](img/f4f37ad4c6bfa8d1dc6b823a6fcfcb53_114.png)

*   **分析：** 我们收到250，000美元现金，并发行了股权。
*   **现金** 增加了250，000美元。现金是资产，资产通过**借记**增加。因此，我们**借记**现金250，000美元。
*   **实收资本** 也增加了250，000美元，这属于股东权益。但我们需要将其分为两部分：面值和资本溢价。
    *   首先，**普通股（面值）** 增加了 25，000股 × $1 = $25，000。股东权益通过**贷记**增加，因此我们**贷记**普通股25，000美元。
    *   然后，我们**贷记** **资本溢价** 账户剩余的225，000美元，以确保借贷相等。
*   **日记账分录：**
    ```
    借：现金                    $250，000
        贷：普通股（面值）         $25，000
        贷：资本溢价              $225，000
    ```
*   **过账到T型账户：** 我们将这些金额过账到相应的T型账户中，并在数字旁标注交易编号以便追溯。

*   **分析：** 这笔交易**没有**日记账分录。因为这是丽贝卡·帕克**个人**借款，而不是文物寻宝公司借款。公司无需偿还这笔贷款。这体现了**会计主体概念**，即公司的账簿只应记录公司自身的交易，而非员工的个人交易。因此，帕克的个人贷款不会出现在公司的账簿上。

**交易3：**
4月2日，帕克聘请律师将企业注册为法人公司。由于组织结构相对简单，律师费仅为3，900美元。

![](img/f4f37ad4c6bfa8d1dc6b823a6fcfcb53_177.png)

![](img/f4f37ad4c6bfa8d1dc6b823a6fcfcb53_179.png)

![](img/f4f37ad4c6bfa8d1dc6b823a6fcfcb53_185.png)

*   **分析：** 我们支付了3，900美元现金作为律师费。
*   **现金** 减少了3，900美元。现金是资产，资产通过**贷记**减少。因此，我们**贷记**现金3，900美元。
*   我们得到了什么？这是一项开展业务的成本，因此应记为**律师费支出**。费用通过**借记**增加。因此，我们**借记**律师费支出3，900美元。
*   **日记账分录：**
    ```
    借：律师费支出        $3，900
        贷：现金               $3，900
    ```
*   **过账到T型账户：** 在现金T型账户的贷方（右侧）记录3，900美元，并新建一个律师费支出T型账户，在借方（左侧）记录3，900美元。

**交易4：**
为安置业务，帕克于4月7日以155，000美元的价格购买了费尔蒙特公园附近一家废弃的披萨店。该建筑老旧，需要翻新。购买文件将103，000美元分配至土地，52，000美元分配至建筑物。帕克用31，000美元现金和帝国银行提供的124，000美元抵押贷款支付了购房款。

![](img/f4f37ad4c6bfa8d1dc6b823a6fcfcb53_212.png)

*   **分析：** 这是一笔复杂的交易。
*   我们支付了31，000美元**现金**。现金减少，因此**贷记**现金31，000美元。
*   我们获得了**土地**（103，000美元）和**建筑物**（52，000美元）两项资产。资产增加通过**借记**。因此，**借记**土地103，000美元，**借记**建筑物52，000美元。
*   我们还承担了一笔**抵押贷款**（124，000美元），这是一项负债。负债增加通过**贷记**。因此，**贷记**应付抵押贷款124，000美元。
*   **日记账分录：**
    ```
    借：土地                    $103，000
    借：建筑物                  $52，000
        贷：现金                   $31，000
        贷：应付抵押贷款           $124，000
    ```
*   **过账到T型账户：** 更新现金账户（贷方），并新建土地、建筑物和应付抵押贷款的T型账户，记录相应金额。

**交易5：**
帕克认为进行一些翻修工程可将建筑物的使用寿命延长至25年，预计残值为10，000美元。她立即订购了价值33，000美元的翻修工程。工程于5月25日完成，当时她用现金支付了翻修费用。

*   **分析：** 我们暂时忽略关于残值和使用年限的信息，专注于5月25日支付现金的交易。
*   我们支付了33，000美元**现金**。现金减少，因此**贷记**现金33，000美元。
*   这笔支出增加了建筑物的价值，属于资本性改良。因此，我们**借记**建筑物33，000美元以增加其账面价值。
*   **日记账分录：**
    ```
    借：建筑物          $33，000
        贷：现金           $33，000
    ```
*   **过账到T型账户：** 在现金账户贷方和建筑物账户借方分别记录33，000美元。

*   **分析：** 我们再次暂时忽略关于使用年限和残值的信息。
*   我们支付了120，000美元**现金**购买金属探测器。现金减少，因此**贷记**现金120，000美元。
*   我们获得了**金属探测器**这项资产。资产增加通过**借记**。因此，**借记**金属探测器120，000美元。
*   **日记账分录：**
    ```
    借：金属探测器      $120，000
        贷：现金           $120，000
    ```
*   **过账到T型账户：** 在现金账户贷方记录120，000美元，并新建金属探测器T型账户，在借方记录120，000美元。
