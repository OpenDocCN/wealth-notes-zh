## 资产负债表方程：会计的基石

![](img/52a43a3ff7bbbcb26d34c02298b1536c_4.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_6.png)

学习一门新语言时，最困难的部分通常是掌握其复杂的语法规则。然而，学习会计语言有一个好消息：它只有一条核心的语法规则，那就是**资产负债表方程**，也称为会计恒等式。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_8.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_10.png)

这个方程的核心公式是：

![](img/52a43a3ff7bbbcb26d34c02298b1536c_12.png)

**资产 = 负债 + 股东权益**

![](img/52a43a3ff7bbbcb26d34c02298b1536c_14.png)

这个等式在任何时间点都必须成立。另一种理解方式是：公司的资源（资产）等于外部债权人和公司所有者对这些资源的所有权主张。

## 现实世界中的应用示例

![](img/52a43a3ff7bbbcb26d34c02298b1536c_16.png)

为了理解资产负债表方程在现实生活中的应用，我们可以考虑购买大宗商品的情景，比如买房或买车。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_18.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_20.png)

假设你想购买一套价值50万美元的房子，但你手头只有5万美元现金。那么，你需要向银行申请45万美元的抵押贷款。交易完成后，你的财务状况如下：
*   **资产**：价值50万美元的房子。
*   **负债**：45万美元的银行贷款（抵押贷款）。
*   **股东权益**：5万美元（你投入的现金，代表你对房子的所有权）。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_22.png)

此时，方程保持平衡：`50万资产 = 45万负债 + 5万股东权益`。

## 方程的核心特征：必须始终保持平衡

![](img/52a43a3ff7bbbcb26d34c02298b1536c_30.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_32.png)

资产负债表方程最重要的特征是它必须**始终保持平衡**。这引出了“复式记账法”的概念。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_34.png)

复式记账法要求，当你在方程的一侧增加或减少某个项目时，必须在另一侧进行相应的增加或减少，以保持整体平衡。因此，任何对资产负债表的改动都至少涉及两个分录。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_36.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_38.png)

接下来，我们将看到，两张资产负债表之间的变化，会通过利润表、股东权益变动表和现金流量表来总结。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_40.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_42.png)

## 利润与现金为何不同？

假设年初你的资产负债表显示：50万美元资产（房子），45万美元负债（贷款），5万美元权益。

在这一年中，假设你的房子升值到了100万美元。虽然实践中不能随意重估资产，但为了举例，我们假设你在年末将房子的价值从50万调整到了100万。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_66.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_68.png)

那么，年末的资产负债表变为：100万美元资产，45万美元负债（贷款未变），而你的权益则从5万美元增加到了55万美元。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_70.png)

现在，我们来分析解释两张资产负债表变动的报表：
*   **现金流量表**：房屋升值本身不涉及现金流动，因此不影响现金流量表。实际上，你的现金可能因为偿还贷款而减少了。
*   **利润表**：利润表会显示一笔50万美元的收益，这来自于你因房屋所有权而增加的权益。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_72.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_74.png)

## 完整的资产负债表方程

![](img/52a43a3ff7bbbcb26d34c02298b1536c_105.png)

接下来，我想展示我们将要讨论的所有内容如何融入这个资产负债表方程。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_107.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_109.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_111.png)

股东权益由两部分组成：
1.  **实收资本**：从股东那里筹集到的资金。
2.  **留存收益**：通过经营业务创造的收益。

留存收益的计算公式是：
**期末留存收益 = 期初留存收益 + 本期净利润 - 本期支付的股利**

![](img/52a43a3ff7bbbcb26d34c02298b1536c_113.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_114.png)

之所以称为“留存”收益，是因为它是净利润减去支付给股东的股利后，真正留在公司里的部分。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_116.png)

而净利润，正如之前视频所讲，是**收入减去费用**。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_118.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_120.png)

如果我们将所有这些代入资产负债表方程，就得到了一个完整的大方程：

![](img/52a43a3ff7bbbcb26d34c02298b1536c_122.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_124.png)

**资产 = 负债 + 实收资本 + 期初留存收益 + 收入 - 费用 - 本期股利**

![](img/52a43a3ff7bbbcb26d34c02298b1536c_126.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_127.png)

### 问题一：基础计算
已知：资产 = 100， 负债 = 50。股东权益是多少？

![](img/52a43a3ff7bbbcb26d34c02298b1536c_145.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_147.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_148.png)

**解答**：
根据方程 `资产 = 负债 + 股东权益`。
代入已知：`100 = 50 + 股东权益`。
因此，股东权益 = `100 - 50 = 50`。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_150.png)

### 问题二：变动分析
已知：负债增加100，股东权益不变。资产如何变动？

![](img/52a43a3ff7bbbcb26d34c02298b1536c_156.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_158.png)

**解答**：
考虑变动后的方程平衡。负债增加100，股东权益不变，等式右侧增加了100。
为了保持平衡，等式左侧的资产也必须增加100。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_160.png)

---

### 问题三：分解资产
已知：所有非现金资产 = 70，总负债 = 60，总股东权益 = 30。现金是多少？

**解答**：
将资产分解为现金和非现金资产，方程变为：`现金 + 非现金资产 = 负债 + 股东权益`。
代入已知：`现金 + 70 = 60 + 30` => `现金 + 70 = 90`。
因此，现金 = `90 - 70 = 20`。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_162.png)

### 问题四：注意信息缺失（陷阱题！）
已知：现金减少10，所有非现金资产增加15。负债如何变动？

![](img/52a43a3ff7bbbcb26d34c02298b1536c_168.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_170.png)

**解答**：
我们需要使用方程：`Δ现金 + Δ非现金资产 = Δ负债 + Δ股东权益`。
代入已知现金和非现金资产的变动：`(-10) + (+15) = Δ负债 + Δ股东权益` => `+5 = Δ负债 + Δ股东权益`。
由于我们不知道股东权益（Δ股东权益）的变动情况，因此**没有足够的信息**确定负债的变动。
如果已知股东权益不变，那么负债必须增加5才能平衡。但题目未给出该信息，所以无法确定。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_172.png)

### 问题五：留存收益与净利润
已知：留存收益增加100，股利 = 50。净利润是多少？

![](img/52a43a3ff7bbbcb26d34c02298b1536c_178.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_180.png)

**解答**：
根据留存收益变动公式：`Δ留存收益 = 净利润 - 股利`。
代入已知：`100 = 净利润 - 50`。
因此，净利润 = `100 + 50 = 150`。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_182.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_183.png)

### 问题六：收入的影响
已知：收入增加100，其他所有类别（负债、实收资本、期初留存收益、费用、股利）均不变，只有资产变动。资产如何变动？

![](img/52a43a3ff7bbbcb26d34c02298b1536c_189.png)

![](img/52a43a3ff7bbbcb26d34c02298b1536c_191.png)

**解答**：
使用完整方程：`资产 = 负债 + 实收资本 + 期初留存收益 + 收入 - 费用 - 股利`。
等式右侧只有收入增加了100，其他项不变，意味着右侧总值增加100。
为了保持等式平衡，左侧的资产也必须增加100。

![](img/52a43a3ff7bbbcb26d34c02298b1536c_193.png)

---

![](img/52a43a3ff7bbbcb26d34c02298b1536c_195.png)

### 问题七：费用的影响
已知：费用增加60，其他所有类别（负债、实收资本、期初留存收益、收入、股利、非现金资产）均不变，只有现金变动。现金如何变动？

**解答**：
使用完整方程，并将资产分解：`现金 + 非现金资产 = 负债 + 实收资本 + 期初留存收益 + 收入 - 费用 - 股利`。
等式右侧费用增加60，相当于右侧总值**减少**60（因为费用是减项）。
等式左侧非现金资产不变，为了保持平衡，现金必须**减少**60。

## 总结

本节课中，我们一起学习了会计的核心规则——资产负债表方程 `资产 = 负债 + 股东权益`。我们探讨了：
*   它的现实意义，即资源与索取权的平衡。
*   它必须始终保持平衡的特性，这引出了复式记账法。
*   它如何作为纽带，将资产负债表、利润表、现金流量表和股东权益变动表紧密联系在一起。
*   如何利用这个方程及其扩展形式，在已知大部分信息的情况下，推算出缺失的财务数据。

通过一系列练习题，我们初步体验了本课程中常见的解题思路：将已知信息代入方程或T型账户，然后解出未知数。掌握这个方程是理解所有后续会计概念的基础。下节课，我们将更深入地探讨资产、负债和股东权益的具体内容。
