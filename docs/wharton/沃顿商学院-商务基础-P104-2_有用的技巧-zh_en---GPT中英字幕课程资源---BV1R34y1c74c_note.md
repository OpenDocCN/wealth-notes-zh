在本节课中，我们将学习几种计算常见现金流现值和未来值的实用快捷方法。这些现金流模式在实践中经常出现，掌握其快速计算公式能极大简化我们的财务分析工作。

---

## 一、 年金（Annuity）的现值计算

上一节我们讨论了复利，即如何将现金流向前推移以计算其未来值。本节中，我们来看看一种特殊的现金流模式——年金。

年金是指一系列**金额相同**、**时间间隔相等**的**有限期**现金流。以下是年金的关键特征：
*   **金额相同**：每一笔现金流数额都相等。
*   **有限期**：现金流在某个时间点结束。
*   **间隔相等**：现金流之间的时间间隔（如每年、每月）是固定的。

年金在实践中有很多应用，例如保险公司销售的年金产品、住房抵押贷款、汽车租赁以及分期偿还贷款等。

如果我们想计算年金的现值，理论上可以对每一笔现金流分别进行折现后加总。但当期数很多时，这种方法非常繁琐。因此，我们有一个快捷公式：

**年金现值公式：**
`PV = CF * [1 - (1 + R)^(-T)] / R`
其中：
*   `PV` 是现值。
*   `CF` 是每期现金流。
*   `R` 是每期折现率。
*   `T` 是总期数。

**重要前提**：使用此公式时，第一笔现金流必须发生在**从现在起的一个周期后**（例如一年后）。如果第一笔现金流发生在今天，则需要单独处理。

---

### 年金计算示例

**问题**：如果年收益率为5%，为了能在未来20年内每年年末提取100美元，我今天需要存入多少钱？

**解答**：
1.  这是一个典型的年金问题：现金流为每年100美元，持续20年，折现率为5%。
2.  应用年金现值公式：
    `PV = 100 * [1 - (1 + 0.05)^(-20)] / 0.05`
3.  计算得出现值约为 **1,246.22美元**。这意味着今天存入约1,246.22美元，即可实现上述提款计划。

## 二、 增长型年金（Growing Annuity）的现值计算

了解了固定年金后，我们进一步探讨其一种变体——增长型年金。

增长型年金与年金类似，也是有限期、等间隔的现金流。不同之处在于，其每期现金流会以一个**固定增长率（G）** 持续增长。这种模式在现实中也很常见，例如随着职业生涯发展而增长的薪资、与收入增长同步增加的储蓄计划，以及公司项目中近似稳定增长的收入或费用流。

增长型年金的现值也有对应的快捷公式：

**增长型年金现值公式：**
`PV = CF1 * [1 - ((1 + G)/(1 + R))^T] / (R - G)`
其中：
*   `CF1` 是第一期末的现金流。
*   `G` 是每期现金流的固定增长率。
*   `R` 是每期折现率（要求 `R > G`）。
*   `T` 是总期数。

**同样，第一笔现金流需发生在第一期末。**

---

### 增长型年金计算示例

**问题**：如果年收益率为5%，为了能在今年末提取100美元，明年末提取102.5美元，后年末提取105.06美元，并以此规律（每年增长2.5%）持续提取19年，我今天需要存入多少钱？

**解答**：
1.  识别出这是一个增长型年金：首期现金流 `CF1 = 100`，增长率 `G = 2.5%`，折现率 `R = 5%`，期数 `T = 20`。
2.  应用增长型年金现值公式：
    `PV = 100 * [1 - ((1 + 0.025)/(1 + 0.05))^20] / (0.05 - 0.025)`
3.  计算得出现值约为 **1,529.69美元**。

## 三、 永续年金（Perpetuity）与增长型永续年金（Growing Perpetuity）

现在，让我们把目光投向无限期的现金流。首先是永续年金。

永续年金可视为年金的特例，即现金流**永远持续下去**。虽然听起来不现实，但在英国确实存在一种叫做“永续债券”的金融工具。其现值计算公式非常简单：

**永续年金现值公式：**
`PV = CF / R`

---

### 永续年金计算示例

**问题**：如果年收益率为5%，为了能每年年末提取100美元并永远持续下去，我今天需要存入多少钱？

**解答**：
1.  直接应用永续年金公式：`PV = 100 / 0.05`
2.  计算得出需要 **2,000美元**。直观理解是，在5%的收益率下，2,000美元本金每年产生的利息正好是100美元，因此可以无限期提取而不动用本金。

---

接下来是增长型永续年金，它是永续年金和增长型年金的结合体，即现金流**无限期**且以**固定增长率（G）** 增长。公司股利流常被近似看作增长型永续年金，因为理论上公司可以永续经营。

其现值计算公式为：

**增长型永续年金现值公式：**
`PV = CF1 / (R - G)`
其中 `CF1` 是第一期末的现金流，且要求 `R > G`。

---

### 增长型永续年金计算示例

**问题**：如果年收益率为5%，为了能在今年末提取100美元，之后每年提取额以2.5%的增长率永续增长，我今天需要存入多少钱？

![](img/62e9979090d09a58641c99c4c20872fc_3.png)

**解答**：
1.  应用增长型永续年金公式：`PV = 100 / (0.05 - 0.025)`
2.  计算得出需要 **4,000美元**。

---

## 四、 总结与重要提醒

本节课我们一起学习了四种常见现金流的现值计算快捷公式：
1.  **年金**：有限期、等额、等间隔现金流。公式：`PV = CF * [1 - (1 + R)^(-T)] / R`
2.  **增长型年金**：有限期、等间隔、以固定速率增长的现金流。公式：`PV = CF1 * [1 - ((1 + G)/(1 + R))^T] / (R - G)`
3.  **永续年金**：无限期、等额、等间隔现金流。公式：`PV = CF / R`
4.  **增长型永续年金**：无限期、等间隔、以固定速率增长的现金流。公式：`PV = CF1 / (R - G)`

这些公式的用途不仅限于计算现值，通过公式变形，我们还可以求解每期现金流（`CF`）、期数（`T`）、折现率（`R`）或增长率（`G`），这在解决实际问题时非常有用。

![](img/62e9979090d09a58641c99c4c20872fc_5.png)

**最后需要特别注意**：应用这些公式时必须确保现金流的模式严格符合其定义（如等间隔、第一笔现金流发生在第一期末等），切勿盲目套用。请通过习题来熟练掌握这些公式的运用。
