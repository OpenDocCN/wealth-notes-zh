# 沃顿商学院《实现个人和职业成功》课程笔记 P6：5_六种生活练习 🧩

![](img/2078ee3d537ea20d5e7290ec33f01253_1.png)

在本节课中，我们将跟随沃顿商学院的教授，通过一个名为“六种生活练习”的互动活动，来探索我们内心对于“成功”的潜在定义和价值观。这个练习旨在帮助我们反思个人对成功的理解。

---

在沃顿商学院教授成功学时，我们采用的方法之一是向学生提供一系列练习、思维谜题和小型课后任务。这些活动旨在促使他们思考自身的成功观、价值观、情感以及能力。

随后，当大家再次聚集时，我们会共同探讨每个人得出的不同答案。在本课程中，我们将为您提供完全相同的练习，并给予您完成它们的机会。

完成之后，我将返回课堂，与您分享我从这些练习中收集到的各类反馈。这些反馈来源于课堂、高级管理人员培训课程，以及成千上万参与过此类评估的在线学员。

通过这种方式，您将有机会对照他人的回答来衡量自己的答案，并从更广阔的视角进行思考。

那么，让我们从第一个练习开始。这是沃顿成功学课程第一堂课就会使用的练习，我称之为“六种生活练习”。这个练习非常简单。

以下是六个简短的段落，每个段落都描述了一个处于不同职业环境中的个体的故事。您需要阅读每一个段落。如果打印出来，这些内容大约占一张纸的正反两面。

阅读完这六个故事后，最后一步将为我们提供讨论的素材：我需要您根据您对成功的理解，将这六种生活从第1名（您认为最成功的生活）到第6名（您认为最不成功的生活）进行排序。

请注意，我们要求每种生活都有一个独立的排名，不允许出现并列名次。

因此，您的任务是：阅读这六个小故事，思考它们，然后进行排序。第1名代表最成功，第6名代表最不成功。

请带着您的答案进入下一环节，届时我们将探讨这个练习背后的一些背景信息。我相信，通过这个练习，您将开始理解它如何帮助您触及内心关于“成功”这个神奇词汇的一些潜在价值观。

现在，请花一点时间完成这个练习。稍后我将回来与您讨论结果。

---

**练习材料：六种生活**

（请仔细阅读以下六个虚构人物的生活描述，然后根据您对“成功”的定义进行排序。）

1.  **生活 A**：一位才华横溢的艺术家，一生贫困，作品在其去世后才被世人认可为杰作。
2.  **生活 B**：一位非常富有的商人，通过有时存疑的手段积累财富，家庭关系疏远，个人生活不快乐。
3.  **生活 C**：一位社区医生，收入中等，深受当地居民爱戴，家庭幸福，生活平静而满足。
4.  **生活 D**：一位著名的摇滚明星，享尽名利与追捧，但因吸毒过量在35岁时去世。
5.  **生活 E**：一位投身于慈善事业的社工，收入微薄，但觉得自己每天都在改变世界，内心充满平和与目标感。
6.  **生活 F**：一位公司中层经理，工作稳定但无趣，生活按部就班，没有大的成就也没有大的挫折，常感生活乏味。

---

**您的排序**（请将字母填入横线）：
最成功 1. ____ → 2. ____ → 3. ____ → 4. ____ → 5. ____ → 6. ____ 最不成功

在本节课中，我们一起学习了沃顿商学院的“六种生活练习”。这个练习通过让我们对不同虚拟人生进行排序，巧妙地揭示了我们潜意识里对成功要素（如财富、名声、家庭、贡献、快乐、德行等）的权重和偏好。请记住您的排序，它将是下一节我们深入探讨个人成功价值观的重要起点。
