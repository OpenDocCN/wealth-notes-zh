# 沃顿商学院笔记 第1课：核心商业框架 🏛️

在本节课中，我们将学习沃顿商学院课程中一个基础且核心的商业分析框架。这个框架帮助决策者系统地评估商业机会与挑战。

## 概述

任何商业决策都始于对现状的清晰理解。一个有效的分析框架能将复杂问题分解为可管理的部分。本节介绍的核心框架包含三个关键维度。

## 核心三维度分析框架

上一节我们概述了框架的重要性，本节中我们来看看构成这个框架的三个具体维度。以下是分析时需要依次考察的三个核心方面：

1.  **市场吸引力**
    *   评估目标市场的规模、增长率和盈利潜力。
    *   分析公式可简化为：**市场价值 = 客户数量 × 平均客户价值**。

2.  **竞争优势**
    *   检视企业能否在选定的市场中建立并维持比竞争对手更强的地位。
    *   这通常通过构建独特的“价值主张”或更低的成本结构来实现。

3.  **执行能力**
    *   判断企业是否拥有将战略转化为现实所需的资源、团队和运营流程。
    *   关键考量点包括团队能力、资金状况和技术储备。

## 框架的应用步骤

理解了框架的组成部分后，我们来看看如何系统地应用它。以下是使用该框架进行分析的标准步骤：

*   **第一步：数据收集**
    *   围绕三个维度，收集相关的市场数据、竞争对手信息和内部运营报告。
*   **第二步：逐一评估**
    *   对每个维度进行独立评分或定性评价，识别优势与劣势。
*   **第三步：综合判断**
    *   将三个维度的评估结果结合起来，形成整体性结论。通常，三者需同时具备正面条件，机会才值得追求。

> 迈克尔·波特曾指出：“战略的本质在于选择做什么和不做什么。” 本框架为这种选择提供了结构化思考的基础。

## 总结

本节课中我们一起学习了一个核心的商业分析框架。该框架通过**市场吸引力、竞争优势和执行能力**这三个维度，帮助初学者系统化地解构商业问题，为做出更理性的决策奠定基础。记住，结构化思考是商业分析的第一步。
