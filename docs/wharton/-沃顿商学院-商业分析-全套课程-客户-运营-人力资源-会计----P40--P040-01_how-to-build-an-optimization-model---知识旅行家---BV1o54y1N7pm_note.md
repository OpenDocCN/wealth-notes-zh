## 第一节：优化模型的核心要素 🔍

在第一周，我们学习了描述和预测不确定结果的方法，并接触了“报童问题”。为了在不确定性中做出最佳决策，我们需要掌握评估和选择行动方案的工具。从本周开始，我们将首先学习**优化工具包**，用于在低不确定性环境中进行选择。

让我们通过一个制造公司的例子来开始。我们将重点关注如何用决策变量、目标和约束来描述一个问题，并建立一个分析模型。

### 案例：Zooter公司的生产决策

假设一家名为Zooter的滑板车制造商，面临在两种主要产品（“Razor”和“Navajo”型号）之间分配有限资源的问题。
*   Navajo型号每单位利润为160美元。
*   Razor型号每单位利润为150美元。

生产每个型号需要消耗三种资源：车架制造、车轮与甲板组装、质量检测与包装。具体工时需求如下：

| 资源/型号 | Razor | Navajo | 下周可用总工时 |
| :--- | :--- | :--- | :--- |
| **车架制造** | 4小时 | 5小时 | 5610小时 |
| **车轮与甲板组装** | 1.5小时 | 2小时 | 2200小时 |
| **质量检测与包装** | 1小时 | 0.8小时 | 1200小时 |

Zooter公司需要决定下周每种滑板车各生产多少单位，才能在资源限制下实现利润最大化。

请注意，本例中所有数据（利润、工时）都是确定的单个数值，而非概率分布。这种**低不确定性**的假设，使得寻找最优解变得相对容易。

## 第二节：构建代数优化模型 📐

对任何问题的优化分析，都必须从识别**决策变量**开始。

### 1. 决策变量
在Zooter的例子中，公司必须决定每种型号的生产数量。因此，我们定义：
*   **R** = 下周计划生产的Razor滑板车数量。
*   **N** = 下周计划生产的Navajo滑板车数量。

一个具体的生产计划（例如 R=500, N=500）就是这些决策变量的一组取值。

### 2. 目标函数
**目标**是公司希望最大化或最小化的标准，例如利润或成本。在Zooter案例中，目标是**最大化总利润**。

一旦决策变量（R和N）的值确定，我们就能计算出总利润。因此，目标函数可以表示为决策变量的函数：
**总利润 = 150 * R + 160 * N**

### 3. 约束条件
**约束**是决策必须遵守的限制条件。对于Zooter来说，任何生产计划消耗的资源都不能超过可用资源。

以下是三个资源约束的代数表达式：
*   **车架制造工时约束**：`4*R + 5*N <= 5610`
*   **车轮与甲板组装工时约束**：`1.5*R + 2*N <= 2200`
*   **质量检测与包装工时约束**：`1*R + 0.8*N <= 1200`

此外，我们还有两个逻辑约束：
*   **非负约束**：生产数量不能为负数。`R >= 0`, `N >= 0`
*   **整数约束**：生产数量应为整数。`R, N 为整数`

### 模型总结
现在，我们可以将Zooter的优化问题总结为以下代数模型：
*   **最大化**：`150R + 160N`
*   **满足**：
    *   `4R + 5N <= 5610`
    *   `1.5R + 2N <= 2200`
    *   `1R + 0.8N <= 1200`
    *   `R, N >= 0`
    *   `R, N 为整数`

## 第三节：关于优化模型的重要概念 💡

上一节我们构建了Zooter问题的代数模型。本节中，我们来了解一些关于优化模型的通用概念。

### 模型的可处理性
优化模型可以有很多变量和约束，但通常只能有一个目标函数。如果公司有多个关注点（如利润、成本、利用率），通常选择一个作为目标，将其余的设为约束。

模型的数学形式决定了其求解难度：
*   **线性模型**：只包含常数、决策变量与常数的乘积、以及加减运算。这类模型最容易求解。
    *   例如：`150R + 160N`, `4R + 5N` 都是线性表达式。
*   **非线性模型**：包含决策变量的乘积、比率、幂次（如 `R*N`, `R/N`, `R^2`）等。这类模型通常更难求解。
*   **整数要求**：为变量添加必须取整数的要求（如Zooter案例），会显著增加求解复杂度，即使模型本身是线性的。

对于Zooter这样的小规模问题（2个变量，3个约束），复杂度影响不大。但对于变量和约束数量庞大的问题，选择正确的模型形式至关重要。

## 总结

本节课中，我们一起学习了在低不确定性环境下识别最佳决策的任务。我们通过Zooter公司的资源分配案例，掌握了优化模型的三个核心要素：**决策变量**、**目标函数**和**约束条件**。

我们了解到，即使在没有不确定性的情况下，由于可能的决策组合数量庞大，寻找最优解也可能很有挑战性，因此需要优化工具（如Excel求解器）的帮助。在借助软件求解之前，用代数语言清晰地表述问题非常有用，这有助于发现建模错误并创建结构清晰的电子表格模型。
