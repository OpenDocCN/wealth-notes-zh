# 沃顿商学院《实现个人和职业成功》课程笔记 P112：48_关于地位的总结 📝

在本节课中，我们将学习罗德·克雷默关于避免“天才综合症”的见解，并探讨成功与地位可能带来的潜在影响。

我们将以罗德·克雷默及其关于避免“天才综合症”文章中的更多内容来结束本次课程。

他的核心观点是：要预料到环境会改变你。

在讨论这一点时，他讲述了一个与他的MBA学生进行的练习故事。

他首先要求学生们写下他们退休时会获得的赞誉。

你将在职业生涯中取得什么成就？

这些来自顶尖商学院的MBA学生，期望取得巨大的成就。

因此，他们写下了非常详尽的成就清单。然后他说：“好的，很好。希望你们能做到。但实现所有这些目标会如何改变你们？”

他说，学生们的回答通常是“很少改变”或“根本不会改变”。

他们不期望自己的核心价值观会改变。他们预计为了实现这些目标，需要做出的权衡会非常少。

他认为这令人惊讶。基于他观察那些真正取得成就者的职业生涯。

他认为这种想法，至少可以说是天真的。他说：

“这仿佛在寻找成功的过程中，他们相信自己只会变成现在更大、更好的版本。”

克雷默引用了一句话，这可能是结束一堂关于地位潜在负面影响课程的最佳收尾。他说：“他们甚至无法想象自己会失宠。”

而这，当然，保证了他们中的一些人终将如此。

**本节课总结**

![](img/a325cf65a736b15034fba4b62d6506a2_5.png)

本节课中，我们一起学习了罗德·克雷默关于“天才综合症”的警示。核心在于认识到：**追求成功与地位的过程本身会深刻地改变一个人**，而对此缺乏预见和自省，可能会导致未来的失败。克雷默通过MBA学生的案例指出，天真地认为成功只会放大现有优点而无需付出价值观妥协的代价，是一种危险的盲点。真正的智慧在于预见变化，并时刻保持自省。
