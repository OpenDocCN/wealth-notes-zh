# 课程 28：积极倾听 👂

![](img/e234156cac76972a8c2e2dab1a019fbe_1.png)

在本节课中，我们将要学习“积极倾听”这一核心沟通技巧。我们将探讨如何通过积极倾听来获取信息、建立关系，并影响他人的情绪与感受。

上一节我们讨论了提问的艺术，本节中我们来看看如何将提问融入更广泛的“积极倾听”框架中。

## 什么是积极倾听？

积极倾听不仅是为了学习信息，更是为了建立人际关系。通过积极倾听，我们向他人及其想法表达关切与兴趣，并能够实际影响他人的情绪感受。

积极倾听的核心在于：**提问**、**展示兴趣**以及**深入探究**。这些行为协同作用，使我们能真正投入地聆听他人。

## 积极倾听的关键要素

以下是构成积极倾听的几个关键行为：

*   **提出开放式问题**：关键在于提出更广泛的开放式问题。例如：“发生了什么？”或“那是怎么发生的？”。我们邀请对方扩展答案，比如：“请再多告诉我一些。”
*   **减少自身发言**：我们常常认为表达自己的关键想法最重要。但在许多情况下，我们真正需要做的是倾听他人，以了解他们的需求，以及我们的想法如何与他们的需求互动。
*   **给予最小化鼓励**：这是积极倾听中“积极”部分的体现。我们可以通过语言和非语言方式展示兴趣。例如，使用“嗯”、“对”、“真的吗？”等简短词语表示我们在听，或通过点头表示赞同。这些行为表明我们正在专注倾听。
*   **进行镜像反馈**：通过重复对方最后几个字或他们话语的要点来展示兴趣。例如，重复某句话。这能让对方感觉到你真正投入并重视他们所说的内容。
*   **总结与寻求澄清**：我们需要进一步探究，可以总结对方说过的话，展示我们正在采纳他们的观点，然后寻求额外的澄清。例如：“让我确认一下是否理解了你的意思……”

## 如何实践积极倾听

将上述要素结合起来，就构成了积极的倾听实践。

我们总结对方的话语，然后推动获取更多信息。通过这样做，我们协同地：**提问以寻求详细阐述**，并**避免自己过多讲话**。

其中“留白”这部分颇具挑战。理念是，当我们留下沉默空间时，常常会制造一些不适感。因此，有时如果我们只是暂停不说话，人们会觉得沉默令人不安，从而有时会用额外的信息来填补它。这是一种常见的面试技巧，在谈判中也常用。人们会故意留白，而那个尴尬的停顿常常会被信息填满，有时甚至是对方原本不愿透露的信息。

然后，倾听的“积极”部分——最小化鼓励、镜像反馈（重复对方话语要点）以及后续跟进——就发挥作用了。我们进一步探究，总结并提出下一个问题。

## 积极倾听的好处

这种积极倾听能带来几个好处：一是让他人参与进来，展示关切；同时对我们自己而言，它不仅帮助我们专注于他人，还帮助我们获得他们的视角。当我们真正专注于积极倾听时，我们能更好地进行观点采择。

![](img/e234156cac76972a8c2e2dab1a019fbe_3.png)

本节课中我们一起学习了积极倾听的完整框架：从提出开放式问题、减少自身发言，到使用最小化鼓励和镜像反馈来积极展示关注，最后通过总结与提问来深入理解并建立融洽关系。掌握这一技巧，将极大提升你的沟通能力与影响力。
