# 课程一：为什么我要教授本课程 🎯

![](img/06dfdeaef6d522e31603356f879f14e9_1.png)

在本节课中，我们将探讨本课程开设的初衷与核心目标。我们将了解为何需要系统性地思考“成功”这一概念，以及本课程旨在提供的价值。

---

在我个人的生活和教育经历中，我发现并没有一个单一的地方可以让我学习如何思考成功。

成功在我们的文化中是一个巨大而有力的词汇。每个人都希望获得更多成功。

但没有人真正完全确定它意味着什么。大多数时候，当一切进展顺利时，你甚至不会想到它。你顺利前行，完成任务，并享受你所做的一切。

然后，某些事情发生了。你来到了生活中一个自然阶段的终点。

无论是学业的结束、你可能工作的公司的终结，还是你生活中一个时期的结束。你的配偶搬家，某些事情发生，你的生活被打乱了。

通常，正是在这种时候，人们会感到退缩，并且缺乏资源来仔细、系统、深思熟虑地思考他们接下来真正应该做什么，以使他们的生活达到一个更充实的境地。这就是我创建这门课程的原因。

我认为不同的见解会在不同的时间来到不同的人身边。

我要求我的学生每年写一篇期末论文，内容仅仅是他们关于个人对成功的看法以及他们打算如何实现这些看法的思考。

但我告诉他们，很有可能如果他们十年后写同一篇论文，他们可能会写出一篇不同的论文。在那个阶段，他们对自己是谁以及接下来应该做什么会有不同的想法，有不同的经验可以借鉴，与那些激励他们并为他们提供了以前没有的榜样的人建立了不同的关系。

因此，这门课程与其说是要给你答案，不如说是要给你一种思考问题的方式，以便你能找到自己的答案。

当然，这个问题是重大的。我用“成功”这个词是什么意思？很好。

那么，我期待在下一节课开始时再次见到你。

---

![](img/06dfdeaef6d522e31603356f879f14e9_3.png)

本节课中，我们一起学习了本课程的创立背景。我们了解到，成功是一个需要被系统思考的复杂概念，而本课程的目标是提供一个思考框架，而非标准答案，以帮助每个人在自己的生命旅程中探索和定义属于自己的成功。
