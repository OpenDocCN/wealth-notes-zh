

# 怎么和贵人大领导相处？

底层出身的孩子，会特别容易喜欢上“钱货两清”的商业逻辑，理想社会是人与人之间的关系是依靠可量化的合作价值决定的，而不是虚无缥缈的人情世故。

但必须要知道的是，商业逻辑只适用于短期协作。在人与人的长期协作中，没有人会“任人唯疏”，你与“资源分配者”的亲密度排位，往往就是资源分配的排位。

所以，每个人都有必要了解一些与“资源分配者”相处的基本原则，即使你不打算通过谄媚大佬上位，但也不应该因为人际关系这门课不合格，让人生失去一些可能性。

- 原则一：不要隐性占大佬便宜。

一个人，不论他如何相信“人人平等”那一套现代经书的说辞，只要他成为一个资源分配者，他就不可避免地产生“上位者心态”。这种心态的典型表现就是，把“平等相待”当做对下位者施恩的筹码。

原因也很简单，大佬是有权力选择“高高在上，享受上位者权威”这个选项的（至少他自己是这么认为的），所以他必然会把对下位者的“平等相待”当做自己的付出。也就是说在大佬看来，我“亲切随和”、“没有架子”是一种情分而不是本分。在这种心态之下，大佬的“平等相待”自然而然会包含古人的“礼贤下士”情结。

如果你以理所当然的姿态，面对大佬的“亲切随和”，那么就相当于让大佬失去了一个笼络下属的手段。代入大佬的视角，他必然会觉得自己被隐性占了便宜。

所以，当大佬主动和你“平等相待”的时候，你可以接着，但一定要以下位者的姿态接着，让大佬可以继续把他的“平等相待”当做筹码。

下位者的姿态就是，皇帝去看望生病的大臣，体恤大臣身体不适，表示不必行礼，大臣接受了皇帝的好意，但态度上诚惶诚恐。

当然，现代社会倒不必演“诚惶诚恐”那一套戏码，但对“上位”的善意表达基本的尊重还是必要的。

有些人会觉得，大佬被奉承惯了，我把他当做平等交流的朋友，他是不是会高看我一眼？

答案是：不会。

首先，每个人都有不可伸张的意志，前呼后拥的霸道总裁，在酒桌上大概率是要给副处赔笑脸的。“从来没人这么对过我”这种事只会在小说中出现。对于大佬来说，他有求于人的时候奉承一下更大的大佬，别人有求于他的时候再奉承他，这对心理健康来说是一种良性循环。

其次，就像老师对学生的小动作、小心思一目了然，大佬形形色色的人见多了，识人能力必然远超普通人，尤其是从底层摸爬滚打上来的大佬，你知道的所有关于攀附的套路，他大概率都见过甚至是亲自用过。而且，大佬不论多么“亲切随和”，真实的心气都会很高。如果你试图在大佬面前表演“我跟那些只会奉承你的人不一样”，那么从大佬的视角来看，你只不过是又一个“姿态高于才华，试图让我另眼相待的小孩”。

- 原则二：不要得罪大佬的“权力工具”。

谄媚者在职场上是一个很讨厌的存在，但在任何组织里，这种人都能混得很好。很多人会把原因归结为谄媚者为领导提供了大量的情绪价值，但真相并没有这么简单。

地位再高的大佬想要贯彻自己的意志，也需要下级的配合与服从，因为“活儿”在下级手上，配合 100 分还是配合 60 分，完全是不同的结果。

上级授予你的权力，只有你的下级愿意接受时，才是有效的。——巴纳德权威接受论

而谄媚者之所以能在职场上升迁更快，原因就是彻底抛弃个人意志，唯领导马首是瞻。理解的百分百执行，不理解的也百分百执行。对领导来说，这哪里是什么“谄媚小人”，分明是忠诚听话的小兄弟，自然会大大压缩对其进行服从性测试的时间，给予对方更多表现自己的机会。

就像《大明王朝 1566》里，嘉靖之所以重用严党，最重要的原因就是嘉靖的权力在清流那群人面前，打折率极其惊人。以清流那帮人的尿性，100 分的皇权，能有效执行 30 分就不错了。但是严党就不一样了，严党能最大程度贯彻嘉靖的个人意志，皇权能得到最大程度的彰显。

所以，从大佬的视角来看，百分百认可自己权威的下级，能极大地提升自己的权力有效率。这种人是亲信，是基本盘，是彰显自己权威的最佳工具，重要性要比能力 100 分但配合度只有 60 分的人高多了。

如果还不能理解，那就想象一下你是领导，你有一个下属，你画饼时他做捧哏，你布置任务时他第一个喊收到，你决策错误时他第一个替你开脱。这样的人你不会对他有任何负面想法，只会觉得遇到他是你的福气。

因此，大佬对这种人的包容度非常高，就像李达康对丁义珍，明知道对方有问题，但照用不误。如果你与处于这种生态位的人发生冲突，即便对方完全不占理，大佬也不太可能真的站在你这边。

- 原则三：对大佬的人生叙事保持尊重。

所谓人生叙事，就是一个人「为何至此的原因」。每个有点成就的人，对于自己的成功都有一个完整的叙事，这类叙事细节不同，但结构大同小异，也许是关于“白手起家”，也许是关于“杀不死我的终将使我强大”，但绝不是关于运气和出身。

比如东哥「带着全村人的希望去北京读大学」，郭德纲「一步一步地苦熬苦掖，终于我们也看见了花团锦簇，我们也知道了灯彩佳话」，俞敏洪「我曾走在崩溃边缘」，雷军「干了30多年，回头一望，全是漫长的挫折和煎熬」。

这种人生叙事是大佬们的精神源泉，关乎他们的自尊与自恋，他们会在各种场合反复提及，有心留意的话，拼凑这类人的人生叙事并不困难。

大家都知道，“劝酒文化”的本质是服从性测试。其实大佬的人生叙事，除了立人设的作用以外，也隐含了一丝“服从性测试”的味道。如果这是一道考卷，那么能升华大佬人生叙事的人可得 100 分，能对大佬的经历表达崇拜的人可得 80 分。

如果你做不到升华和崇拜，那么跟随众人附和一下还是有必要的。因为人在期待认同的时候会特别敏感，你的一丝丝不认同或者不屑一顾，万一被大佬察觉到，你们的缘分也就很难再进一步了。

多提一嘴，不只大佬们有人生叙事，普通人也会美化自己的经历。你会在每个老男人的酒局上，听到一个理想被磨灭，或者深情被辜负的故事。

> 那时我们有梦，关于文学，关于爱情，关于穿越世界的旅行。如今我们深夜饮酒，杯子碰到一起，都是梦破碎的声音。——北岛《波兰来客》

在以自己为主角的叙事中，「我」最深情，最无辜，最真诚，最迫不得已。但这种认同通常只能自己给自己，而且越是了解你的人，越不会给你这种认同。因为在彼此足够了解的情况下，以熟人的视角看待问题，很少有人会是完美受害者。

所以，每个人都会觉得自己不被理解，每个人都有难与人说的孤独。

如果你想获取这类人的友谊，那么只要对其表达充分的认同就够了，你将会收获无数“知己”。
