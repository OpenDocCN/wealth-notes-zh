

# 同盟军攻占腊戍，对中国影响几何？

7月25日，已经淡出中文热搜几个月的果敢民族民主同盟军，再次上了热搜。根据同盟军发布的公告，他们已经占领了腊戍城。

腊戍是啥地方？缅北重镇，北掸邦的首府！同盟军打下腊戍，不亚于诸葛亮拿下西川。这一切，对缅甸局势有什么影响，对中国利益又有什么影响呢？

## 一、战前态势与战役经过

这次同盟军的胜利，可以说是400年来，海外华人在缅甸最大的武力成就。400年间，无论是清军，还是中国远征军，或是中国知青组成的缅共人民军，都没能成功拿下腊戍。至此，这个地方成为所有缅北华人心中的意难平。谁能想到，在今天，竟然不费什么功夫就打下来了？

也难怪腊戍被占领会引发缅北全境震动，因为腊戍这个地方太重要了。腊戍位于缅甸南马河上游，东北距离果敢老街120公里，相比老街那样的穷乡僻壤，腊戍可以说是富庶多了。

论矿产，腊戍附近的包德温矿区为缅甸最大的铅、银、锌产区。

论交通，腊戍是中缅贸易通道的关键节点，是中缅公路的终点，也是曼德勒—腊戍铁路线的终点。

论人才，腊戍是掸邦教育集中所在地，有腊戍大学、腊戍计算机大学、腊戍科技大学等等，还有60多所华文学校，是缅北华人的文化中心。

论人力，这里人口80万，是果敢的4倍多，而且华人比例非常高，超过50%，和果敢理论上是一家（实际上肯定复杂得多）。

更关键的是，腊戍还是缅甸东北军区所在地，不仅囤积了大量重炮、武器弹药，还建设了“老象山”防线和十二堡，正规驻军6000人，民团数千，而且腊戍还有军工厂和飞机场，可以部署战斗机。

一开始很多人判断，自从去年12月同盟军与缅军达成协议，和平接收老街之后，一定会休养生息一段时间，把老街消化掉，再做其他打算。但万万没想到的是，仅仅半年时间，同盟军就发起了腊戍战役，打了20几天就一鼓而下。

那么，同盟军为什么要打原本在果敢实控疆域之外的腊戍呢？一方面，腊戍对果敢威胁太大，腊戍在缅军手里，果敢就没法和平。腊戍距离果敢也就120公里，去年的1027行动的时候，要不是同盟军在开战之初就炸了滚弄大桥导致99师无法支援，果敢之战的结果还不好说呢。尽管如此，缅军还是从腊戍机场起飞了很多战斗机去轰炸果敢，给同盟军造成了巨大伤亡。哪一天，敏昂莱腾出手来了，想收拾果敢的时候，腊戍就是最完美的前进基地。

另一方面，腊戍关乎果敢未来。果敢上次被缅甸偷家的历史教训教育了同盟军：单凭果敢县2700平方公里土地和20万人口，不足以自守。要想不被缅军控制，必须像佤邦那样，拥有1万平方公里以上土地、60万以上人口，才有底气自保。而整个腊戍地区，有11000平方公里土地、100万人口，无论是收税、发展工农业，还是征兵，都有足够的资源和发展空间。

所以，自从拿下果敢之后，同盟军就开始筹划进攻腊戍，只不过迫于北方大国的压力，没有实施。那么为啥这次又开打了呢？关键是缅甸军政府作死。从果敢撤出来后，缅军休整了半年，补充了大量弹药，觉得自己又行了，所以在6月23日，突袭了同盟军驻地芒章老寨，主动撕毁了停战协议。

站在北方大国的立场上，其实是希望缅北和平的，毕竟打起仗来影响中缅贸易，带来的难民危机也非常麻烦。所以中国就把缅军和FPNCC（七家民地武组成的谈判委员会）叫到昆明开会，你们好好谈一谈，看在我面子上，别打了。但是万万没想到，谈判刚开始，缅甸军方就大骂这几家民地武，要求克钦归还洋人街口岸、退出腾密公路沿线，要求果敢同盟军交回清水河口岸、退出联邦公路等等，否则大兵压境，灰飞烟灭。

俗话说，战场上得不到的东西，谈判桌上也别想得到。话不投机还谈个啥，直接掀桌子打吧。所以，7月3日，同盟军誓师，211、311、611三大主力旅对腊戍完成合围，开始了腊戍战役。半年不见，同盟军的战力又上了一个新台阶。

半年前，同盟军拿下果敢老街，其实不算是打下来的，是围下来的，缅军被断水断电断粮，熬不住了才投降。而这次倒是真正硬打下来的，比如在腊戍南北出入城口大佛塔41营联合指挥部、522营、AD桥附近、68团1堡5堡和507、912营地，都爆发过激烈的战斗。

同盟军是怎么打的呢？不得不说，同盟军很擅长学习，几乎完全复刻了瓦格纳在巴赫穆特的打法，化整为零，小组作战。先派无人机侦查，看看哪个楼房窗口堆有沙袋，然后用火箭筒、无后坐力炮定点打击；遇上角度不合适的，直接就用自杀无人机炸。对于外围的坚固据点，同盟军不知道从哪搞了一堆喷农药的植保无人机，载重都是几十公斤级的，拿这玩意扔炮弹威力巨大，一枚炮弹下去，一个据点就上天了。而缅军战术过于呆板，只知道死守阵地，缺乏主观能动性（一个月几百人民币，玩什么命啊），一直被动挨打，无法发起有效的反击。

本来在7月14日的时候，同盟军碍于北方压力，曾经宣布停火4天，但没想到缅军趁机加大对同盟军的轰炸力度，停火宣言失效，战火重开。转折点发生在7月21日，同盟军突袭了41营联合指挥部，击毙了正在指挥部的腊戍前线总指挥、东北军区总参谋长丹津温上校。群龙无首之下，缅军防线动摇，敏昂莱不得不启用东北军区司令索莱准将担任总指挥。

这哥们是原99师的师长，按道理来说，指挥能力没问题。但问题在于，这哥们是7月18日才上任的，就这么点时间，状况都没搞清，再加上腊戍的守军有一部分战斗意志不强，索莱也没什么办法。到了24日，同盟军又改变了战术，夜间大胆进行穿插分割。25日凌晨，611旅和211旅几乎同时攻进了东北军区司令部，摧毁了缅军的指挥中枢，缅军指挥体系彻底崩溃。

目前，一部分缅军已经投降，其中包括一名中校，还有的缅军在负隅顽抗，战斗正在激烈进行中。可能也是因为急于鼓舞士气，同盟军在还没有控制全城的情况下（特别是老象山防御体系还没拿下来），就匆匆忙忙发布了公告。事实上，要想清除腊戍的残敌，起码还要好几天时间。

## 攻占腊戍，有什么影响？

毫无疑问，这可能彻底颠覆缅北政治格局。同盟军原来算是“流寇”，根本上不得牌桌的。但现在呢？变成了一个有地盘的军政实体。如果能把地盘消化掉的话，那么毫无疑问，缅甸北部会出现一个以华人为主、人口比佤邦还多的华人地方政权。

丢掉如此大的城市，是缅甸建国以来从未有过的，对缅军的威信是一次粉碎性打击。那么有同盟军的先例，你以为其他的德昂军、克钦军、人民防卫军等等不会有样学样？加大攻势，兵锋直指密支那、曼德勒、马圭？如果缅军不全国总动员的话，那么很可能，缅甸政府军将被彻底赶出上缅甸，缅族在掸邦60年的经营将毁于一旦，而且引发重大地缘政治变革。

这对中国来说，是一个不大不小的麻烦。原因很简单，对中国来说，最看重的是什么？是保持缅北的和平稳定。只要和平稳定了，中国才能有效对缅贸易，才能从缅甸打破马六甲之困，才能不被缅甸事务牵扯应对东南的精力。而且，因为中缅石油管道、中缅天然气管道都要经过腊戍，战争打起来，谁能保证中国这条第四大能源生命线的安全？

说到底，中国不在乎缅北是谁掌控，只希望能够有一个稳定的政权，好做生意。所以，中国对缅甸方向的政策一贯是劝和促谈。去年12月底的时候的和平协议，其实也是中国从中调和的，实现了停战和果敢的和平交接。缅甸把皎漂港租借给中国，同意开建中缅铁路；中国呢？向同盟军等民地武施压，不要打大城市，作为对缅甸的回报。

但现在呢？缅甸军政府又菜又爱玩，撕毁协议开战，结果连腊戍都丢了。中国对这一格局变动很头疼，所以在同盟军开始攻打腊戍之后，中国就关闭了杨龙寨、清水河口岸，把果敢老街的电也给停了，这也说明了中国的态度。

但是反过来说，同盟军的胜利，对中国也不是一点好处都没有。我们都知道，不论缅甸的民选政府，还是军政府，都坚持大缅族主义，欺压境内的少数民族（包括华人）。至今缅北民地武控制区的居民，都办不了正式身份证，也没法出国，这也是民地武的群众基础之一。

你看看1027行动之前的果敢，是缅甸军政府控制的，结果黑灰产业泛滥，中国交涉好多次都没用。那干嘛不换一个听话的华人政权呢？就像佤邦和四特区一样，中国一个通缉令，那边就把人送来了。相比被屠杀的印尼华人、当二等公民的马来华人、被迫改名换姓的泰国华人，坚持武装斗争的缅甸华人，中国倒是有必要扶持一下的。最起码，作为牵制和敲打缅甸军政府的一枚棋子，也是不错的。

虽然关闭了口岸，但此后大概率会灵活对待既成事实。在同盟军作出一系列保证后（比如承诺不搞电诈，不破坏中缅油气管线等），默认同盟军对腊戍的占领。其实说起来，这也就是同盟军最初的打算，不管大国支持不支持，先打再说，反正大国只和胜利者谈判。

但是，同盟军占据腊戍后，并不意味着可以高枕无忧。
- **第一，**腊戍和果敢老街不一样，果敢老街是缅军2009年才抢过来的，丢了也就丢了。但腊戍呢？建国以来一直是缅甸军政府统治，丢了之后政府无法向国民交代，要么下台，要么打回来，没有其他路径。下一步，缅甸军政府必然会反扑，甚至不排除总动员，进入战争状态。考虑到政府军拥有飞机、坦克、装甲车等技术兵器，同盟军硬扛的话伤亡会很大，最佳方案就是用恭顺姿态换取大国的调停。
- **第二，**与其他民地武的关系。有人好奇，同盟军为啥那么能打？甚至怀疑同盟军背后有大国支持。但是其实吧，支持同盟军的并不是大国，而是佤邦。当年彭家声卷土重来的时候，鲍有祥（佤邦老大）就支持了他500支枪。2015年彭家声的儿子组织“探亲攻势”攻打老街失败后，之所以选择红岩乡落脚，也是因为这里是佤邦的边境，一旦缅军围剿就可以逃到佤邦去。1027行动中，同盟军的伤员，都是在佤邦医院治疗的，甚至一些技术兵种，干脆就是佤联军换了一身衣服而已。同盟军投桃报李，在解放果敢之后，同盟军就把很多打下来的地盘（比如户板地区），移交给了佤联军。但是，之前同盟军和佤邦可以合作无间，但从同盟军占领腊戍之后，无论人口还是地盘，果敢都和佤邦平起平坐了，而且因为其战绩带来的威势，也超过了佤邦。一山不容二虎，当二者实力相当的时候，很容易产生嫌隙。事实上，这种嫌隙已经产生了，7月11日的时候，本来同盟军已经快打下当阳县城了，但缅军却悄悄撤走了，佤联军却突然占领了当阳，导致同盟军内部对佤联军的摘桃子行为非常不满意。如果同盟军不把拿下腊戍的利益拿出来给佤联军分润的话，二者迟早有反目的那一天。
- **第三，**同盟军消化腊戍需要很长时间。占领一个城市，不是占下来就行的，而是要实施统治，在统治中获取利益。但问题在于，腊戍的华人并不会因为同盟军的同胞身份，而对他们抱有好感。几十年来，腊戍华人一直被采取同化政策，缅文教育、缅甸国家意识教育，导致这里的华人认可自己是缅甸人，而不是华人。同时，腊戍的华人是可以办理身份证、护照，可以出国旅游或者打工的。那么腊戍一旦被同盟军攻占，缅甸军政府就会把他们视同和果敢人一样的二等公民，这些华人的利益自然也会受损。人嘛，都是只注重自己利益，你说建立的“华邦”和我有啥关系？我只关心我自己过得好不好。同盟军想用民族大义来感化腊戍的华人，恐怕还有很多工作要做。所以，同盟军现在要做的，就是尽快稳定腊戍，消化，而不是继续开战。

不管怎么样，同盟军占领腊戍，必将给缅北带来一个新的变局，这场变局会如何发展，我们拭目以待。
