# 特朗普，与即将到来的动荡时代

罗文益

240715

有时候，历史就是这么偶然，就像特朗普挨这一枪，厘米的差距，打开的完全是两个平行宇宙。

如果这一枪没偏，特朗普真没了，那么共和党的党争就要开始了，像是之前已经惜败的尼基·黑莉，恐怕也有了卷土重来的机会；没了特朗普这样的扛把子人物，就像当年民主党失去了肯尼迪一样，剩下的没一个能服众的，今天的拜登哪怕再拉胯，也不是什么大问题，坐等看共和党养蛊就行。

但这一枪打偏了，美国的两党政治一下子进入了全新的局面，也就是两个党自身都变得更团结了。

其中，共和党因为特朗普生还，现在正在全面高潮，特朗普被当成了天选的“殉道者”。他之前的疯言疯语，也就是“民主党处心积虑迫害他”、他是“拜登最害怕的人”之类，现在全部得以验证。

再看拜登，民主党目前最大的分裂，就是现总统拜登和民主党本身的分裂；民主党的另一层分裂，则是巴以冲突以来，支持巴勒斯坦的和支持以色列的人之间的分裂，沿着这些分裂线，再加上6月28号拜登拉胯的电视辩论与难看的民调，目前民主党的最大议题，就是换掉拜登、换个人来参选。

可惜，特朗普挨了这一枪，对民主党是天大的坏事，对拜登却是绝对的好事，因为民主党现在肯定不敢临时换掉他了。

为啥呢？因为如果特朗普中了一枪，就把拜登给临时换掉了，那拜登不是给吓跑了吗？那民主党岂不是太小了？就像我前文说的那样，这一枪没打中特朗普，不等于是打中了拜登吗？

所以拜登现在绝对不能被换掉，不然等于是是给枪手背锅，证明了共和党的话，也就是：“正因为民主党以及拜登对特朗普的纳粹化渲染，特朗普才会在仇恨政治的大环境下遭受枪击”，所以拜登现在哪怕自己想走人，民主党也不会让他走了。

既然拜登走不了了，那民主党自然也就团结了，因为不管你闹得再大、意见再多，你也得让拜登选下去。

如此一来，就美国国内来说，接下来就是两党全面硬碰硬的局面——共和党团结了、民主党认命了，大家都没有了精神内耗，直接放开手脚搞大选了。

而从当前的民调来看，共和党是显著占优的，特朗普当选的概率还是蛮大的。

所以特朗普挨了这一枪，最直观的影响，就是大概率把他本人给送上台，美国再次右转，中美对抗跟现在不一样，又会进入到另一个全新的局面。这个全新的局面是什么？

对特朗普来说，他首先是一个主张“收缩战略”的人，这种收缩的底层逻辑，又在于美国为全球提供的公共产品（包括军事上对各地的驻军费用、北约美国出大头的军费、全球金融基础设施等）是否合算，正因为不合算，所以决定收缩。

所以，在他的任上，美国不仅和中国有贸易冲突，美国和印度、日韩以及欧洲之间，都存在大大小小的贸易战，并且在北约的问题上，也和北约诸国讨价还价，商量各自出钱的多少。

所以，对于当前俄乌之间的冲突，特朗普也自然会像他承诺的那样，不再给乌克兰钱了，他大概率会和俄罗斯达成共识，就像他自己说的那样：战争会在 24 小时以内停止。

其实大概率就是把乌克兰给卖了，以停止援助为威胁，让乌克兰割地认怂。

为什么这么说？拿最近北约唯一的逆行者匈牙利来说，它是北约里唯一一个不愿意制裁俄罗斯，并且还和俄罗斯主动接触的国家，基本上被认为是特朗普在欧洲的喉舌。

在最近大热的北约峰会上，匈牙利也是唯一一个不参与西方武器转让、训练乌克兰士兵的国家。其总理欧尔班，跟特朗普的关系也特好，专门跑到美国会见过特朗普，他提出的“乌克兰和平方案”，据说就是特朗普提供给他的。

7月10号，匈牙利的外交部长彼得·西雅尔多还公开表示，特朗普当选了，乌克兰才会迎来真正的和平。

对欧洲来说，这里面还有一个暗潮，就是欧洲本身的右转。欧尔班虽然表面只是匈牙利总理，很特立独行，但他的胆气主要来源于欧洲议会一个叫“欧洲爱国者”的新超国家党派联盟。

这个联盟属于欧洲议会里的新右翼，由欧尔班参与创立，目前在欧洲议会的支持者也都越来越多。

这个新右翼党派联盟，主张和俄罗斯保持联系，让乌克兰割肉，和特朗普的思路是比较贴合的。

一旦特朗普当选——特朗普目前在所有美国摇摆州的支持率都超过拜登，当选概率很大——那么欧洲的这个新右翼联盟，也就是“欧洲爱国者”，和特朗普就能实现对接，并改变俄乌战争的总体局势。

如此一来，最极端的一种情况，就是普京开始和特朗普接触，和中国若即若离。

对欧洲来说，它将正式分裂为两大势力，一部分是原来支持乌克兰的左翼与旧右翼，另一派，就是迎合特朗普的新右翼——目前新右翼的声音还比较小，是因为拜登在台上并且美国承担了主要的援乌费用，一旦美国不给钱了、欧洲也出不起钱，那么更加现实的、对俄罗斯友善的新右翼自然就会全面涌现。

对中国来说，一个分裂的欧洲，不一定是我们希望，因为很可能出现左翼在经济上反对我们、右翼在政治上反对我们，除非特朗普就像他自己说的那样，彻底专注美国事务、收缩得更加彻底。

再看我们现在的一大问题，是需要去产能，需要更稳定的国际贸易环境，虽然欧洲正在对中国的新能源车搞关税战，但总体还算可控，如果欧洲进一步分裂，那么中欧关税战的激烈程度可能在欧洲一部分保持不变、另一部分进一步加深。

再看中美之间的关系。

特朗普在战略收缩的同时，还有另一个目标，就是集中注意力和中国展开一对一的较量，这和拜登全面围堵中国的“小院高墙”战略是很不一样的，美国的资源可能会更加集中。

而如果美国的资源更加集中，那么未来就有两个爆点。

第一个是经济的，也就是中美之间更加猛烈的脱钩（特朗普已经扬言要把中国输入到美国商品的关税加到 60%），甚至可能演化为中国和印度之间这样的政治、经济全面双脱钩。这种脱钩什么效果？你现在要是申请去印度，基本上都不可能了，而中美之间的经济、科学、文化交流，可能会断崖式下落到改开前的水平。

第二个政治上的爆点，就是南海与台海区域。

之前有一些评论，说特朗普上台之后，会把台湾当做是一件“商品”进行交换，让中国大陆让出一部分利益，转手就把台湾给卖了。

这个看法有点道理，毕竟特朗普的战略确实是收缩，但我们不要忘记，特朗普的口号可不是收缩，而是“让美国再次伟大”，收缩只是“再次伟大”一个推论而已。

特朗普的“让美国再次伟大”，其实是首先针对美国自己，是要让美国的制造业回流、再次工业化，所以特朗普才会主张对中国收60%的关税——收这么高的关税，不就是为了让美国的本土制造业不被中国产的工业品冲击到吗？

所以，如果特朗普想要交易台湾，那么他希望中国能拿出的筹码是什么？那么大概率只有一个，就是**能让美国的制造业重新大回流**——不管是被迫还是自愿，他肯定都希望欧美的资本撤出中国，回到美国本土，在美国搞生产。

要怎么才能做到呢？其实也不难，就把今天制裁俄罗斯的剧本再演一遍即可。

所以，**特朗普有很强烈的愿望，去主动挑动台海的紧张局势，但是又不会让美国亲自下场，而是打一场在台海、南海地区的代理人战争，就像今天的俄乌局势一样，实现地区性的僵持。**

对特朗普来说，他最大的挑战是如何控制规模，因为中国肯定不会傻乎乎地入这个局，跟俄罗斯一样在自己家门口或者家里给自己放血，所以这场潜在的冲突，很容易从局部战争演变为新时代的、更大规模的全面冲突。

总的来说，特朗普上台之后，贸易战的深度肯定大幅加深，美国这个全球最大的消费机器，也将“关起门来自娱自乐”，中国的对外贸易，在今年年末可能会受到一次短暂的大冲击；其次，台海地区冲突的风险也会直接上升，甚至变得不可控。

很多人认为中美之间还会存在交易，可惜在特朗普眼里，中美断开而不是连接才是主要任务——连接才有交易，断开何谈交易呢？

当然，本文的推论只是机械性的，权力与利益在运转的过程中，还有两个现实的规律，第一个是“互相碰撞传导”，第二个则是“我预判了你的预判”。

如果特朗普重回舞台，按照这两个规律，可能还会发生另外一些不可预料的事，比如说，考虑到俄罗斯和美国接近，那么中国会不会和乌克兰接近呢？

目前中国的策略是中间偏俄，和欧洲暂时在台面上的左翼、旧右翼不对付，经济上中欧之间只能算是小摩擦，而一旦中国的立场转变为中间偏乌，那么连锁反应就是欧洲（支持俄罗斯的）新右翼成为了欧亚大陆上的孤岛，只能大概在中南欧的部分、作为美国的执行者而存活。

当然，这确实也符合特朗普的单边主义，对欧亚大陆实现了进一步的退出。

其次，既然特朗普归来已经是大势所趋，那么**很多国家可能也要提前大幅调整战略，以适应这个特朗普的时代**。

对特朗普来说，意识形态远不如实际的经济利益重要，所以美国在日韩等地的驻军问题，可能会转为其他更加现实的、更划算的援助。

比如说，特朗普和北朝鲜之间，会不会形成新的通道？如此一来，既可以收缩在韩国的驻军，还能在中国的周边埋一根钉子；同时，“MAGA”的收缩战略（以高关税保护美国本土制造业），还会大幅影响美国和韩国、越南之间的贸易，那么，韩国和越南的对中关系，是否会进一步加强？

如此一来，中国大陆在台海地区面临的压力，可能来自于俄罗斯、北朝鲜、日本与菲律宾——在冲突升级的过程里，它们是否愿意成为棋子？还是成为不可预测的因素？

总而言之，特朗普归来的概率很大，而且越来越大。

未来的几个月里，如果民主党没有祭出什么大招，那么特朗普将稳拿摇摆州的票，他再次君临美国，将会是大概率的事。

对美国来说，特朗普作为第二任期的总统，可能会毫不客气地、大张旗鼓地实现其政治理想，包括和世界更深地脱钩，筑起高墙保护美国本土的制造业，同时清算自他下台以来民主党对他的多次人身攻击。

特朗普比拜登有一个很大的优势，就是他基本上能掌控共和党，不像民主党，呈现出多个层面的撕裂、不完全受拜登控制，这意味着特朗普对美国政治的掌控达到空前，美国独行、并且出昏招的概率大幅提升。

同时，美国全方位的退出，对全球各地蠢蠢欲动的造反派，也是一大利好，全球将会更加动荡。

对于这个不稳定的局面，中国能提供什么呢？

“一带一路”作为一个偏向于经济的项目，是否要开始算政治账？

我们以经济为中心的时代，是否要政治化，以适应美国收缩的战略？

总之，无论国家，还是个人，都要开始准备迎接这场风暴了。
