

# 如何让女性认识到结婚生育的价值和意义?

来源：知乎回答
240708

整理：公众号懒人搜索，懒人专属群分享
懒人微信：lazyhelper

这个世界上一切为了服务集体而存在的意义，我通常都把它叫作思想诱奸。

社会有时候是很荒诞的，或者说部分人很荒诞，它极钟爱于设计一些宏大且虚无缥缈的主题灌注在那些新生儿身上。

它们也往往坚定不移地宣扬这种主题就是生而为人的最高意义，是能够体现生命价值的唯一途径。

但是呢，等集体需要的是另外一种服务时，它们便又会开始宣扬如何让女性意识到节育环的意义了。

这片土地上的女性时常以一种天然负债者的形象诞生在母亲的子宫里，结婚和生育多数时候都不是个体意志的体现，而是家庭与社会两种目光下垂涎三尺的回报。

似乎女性的生命是借来的，所以既得偿还家庭，也得偿还社会。

但偏偏它们不喜欢说这是要债，它们从古至今都极喜欢那些大义凛然的陈词滥调。

家庭会裹挟着亲情向懵懂的女性施压，并陈述扶养的不容易，以及自身的衰老和痛楚，搞得好像女性的出嫁与生育，就能够让年迈腐朽的两具躯壳重焕新生一般。

社会则像个猥琐的传销者那样靠近女性的耳朵，告诉她们在二十多岁的年纪步入妊娠反应是一种光荣的认可。

似乎只要这样，就可以拿到通往幸福的钥匙。

而那些拒绝的女性，则如同聊斋志异的孤魂野鬼那般被打上不正常的标签，然后在他人的描述里以某种凄惨阴怖的形象出现，以方便父母们因此告诫自己的孩子。

基于生育和婚姻的产业们倒是日夜不停地围猎着女性，它们殷勤和夸张的样子就仿佛个体们所有的付出都会迎来天大的回报。

但其实什么回报都不会有。

因为女性很多时候只是爱与家庭这个主题下堆砌的消耗品，就像被挤干净扔在一旁的染料剂。

大家用自己的血肉赫然铸成了一辆五彩缤纷的列车滚滚向前，那列车上插着无数的彩色旗帜迎风飘扬，少数人打开车窗探出脑袋唱着歌，多数人匍匐在冒着火花的齿轮里因疼痛咯吱作响。

这很正常，因为它们说时代总要在推动中前行。

这般光荣的事仿佛天然就不可以和物质需求挂钩。

于是它们绝口不提对于重男轻女的摒弃和打击，绝口不提对于生育的扶持与补偿。

就好像多了一个子儿都算是铜臭味的污秽，但往往拼了命从个体兜里掏钱的也是它们。

女性们的不幸偶尔也来自于那种更柔软的塑造性，关于意义与爱的塑造更容易渗透进她们的精神世界。

我在精神病院里时常觉得很怪诞的事情就是，我的男性病友们一般在妄想症的驱使下扮演着艺术家和哲学家，而我的女性病友们则常常蜷缩在角落里呼唤着爱，又或者假装被爱。

我察觉到了有某种不起眼但异常怪诞的教育存在其中。

那就是太多女性根本没有意识到爱可以朝向自己。

她们根深蒂固的认知和期盼，决定了爱必然是流通与牺牲，爱是我要为了谁而做什么，爱是一种发自内心的奉献。

世界上还有比把胎儿称为爱的结晶更为猎奇的象征吗，还有比爱他就给他生个孩子更为离奇的表达吗。

似乎如此这般，自我厌恶，缺爱，抑郁情绪，乃至于迷茫都将通通在爱的付出里得到弥补。

但其实都不需要。

因为爱也可以朝向自己。

今天我们在这里否决某一项荒唐的宣传时，就理应意识到对于个体而言真正的意义。

就是自由且自爱地度过一生。

而生命这种如同山中野草割了一茬又有一茬的东西，它没有什么意义，只不过草长莺飞时需要它，野火燎原时也需要它。

充充数而已。

我的自由来自于痛苦和幸福都不必被约束，我不需要遵循任何人的旨意，不需要去模仿群体运行的轨迹。

我可以穿着碎花裙子背着手走在悬崖边，因目睹今天的黄昏不合我心意那般火红便纵身跳崖，我还可以躺在瓦片叠成的屋顶上看着浩瀚的星空，在倦懒感和周遭的清风里浪费着时间。

我不必心急火燎地奔赴死亡，当然也没义务去完成所谓女性应该做的事。

在这样的一生里我为自己所有不源于他人的痛苦买单，而那些货真价实基于别人而造就的苦难，你也别怪我蹬鼻子上脸。

我会热烈且勇敢地爱上自己，并因这种爱重新滋养那些血肉上的空洞，哪怕它被称作是自私，我也愿意在这样的小窃喜中哼上一曲调子。

生命的延续也好，婚姻的缔结也罢。那都应该在爱和我的自由之中，而非之外。

这篇回答碰巧出现在了今天，那么当诸位看到这里的时候，我想应该送上自己的祝福。

我没有分男女性别的原因是，倘若阅读到这里的人是女性，那么倚靠着这些文字，我祝福你拥有相匹配的勇敢，因为只有勇敢和独立，才能迎来爱和自由。

而倘若阅读到这里的是男性，这些祝福其实同样没有偏差。

对于野草而言，从来只需要疯长。

不需要听话。
