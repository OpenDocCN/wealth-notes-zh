

# 孟加拉国大规模抗议，背后只是考公的问题吗？

近日，南亚孟加拉国发生了持续的国内动荡，上万大学生走上街头，与军警发生了激烈的冲突对峙，目前已经造成上百人死亡。

抗议的直接原因是上个月孟加拉宣布恢复六年前已经废除的公共服务部门岗位“配额制”政策，一石激起千层浪，结果引发了全国范围内的抗议性活动，至今仍没有降温的迹象。

那么，当年伴随着孟加拉独立建国就出台的“配额制”政策是怎么回事？这又与当下孟国内的政局走向有什么深刻联系呢？

## 一、 配额制的起源与演变

1971年底，伴随着孟加拉国独立战争（第三次印巴战争）的结束，孟加拉国正式迎来了独立，人民联盟主席谢赫·拉赫曼（现总理谢赫·哈西娜之父）出任开国总理。

当时，为了表彰在争取民族独立战争中浴血奋战的自由战士们的贡献，拉赫曼以政府令的形式出台了公务员岗位招录“配额制”，规定每年招聘的公务员岗位中，留出30%的名额直接内定给伤残退伍的老兵们，类似于咱们古代的“荫庇制”。

此外，基于统战的需要，国内其他各阶层群体均顺带获得了一定比例的配额，具体包括1%给身体有残疾的人群、5%给少数民族、10%给落后地区、10%给女性人群等，余下44%左右的岗位面向社会公开招录。

这一政策最初得到了民众的支持，毕竟老兵为国流血牺牲，女性又在战争中饱受摧残，为了笼络人心团结各阶层，出台政策安置照顾一下也说得过去，加上当时孟国内经济仍以传统农业为主，公职阶层与农民阶层、普通民众等利益冲突不大，大家都有口饭吃就行了，所以都相安无事，得过且过。

1997年，随着第一代抗战的老兵退休去世等原因，配额岗位的使用也减少了，孟加拉政府决定将这部分空缺配额的适用范围扩大到老兵的直系子女们，2010年又进一步扩大到了孙子辈，配额制正式实现了世袭罔替。

更有甚者，一些公务员岗位从老兵的妻儿，传给了孙子孙女，甚至重孙外侄等，这种现象已经损害了公平公正的原则。

随着时间的推移以及经济形势的发展，尤其是步入人口爆发的阶段之后，孟加拉国内产业能提供的优质就业岗位越来越少了，尤其是受过高等教育的大学生越来越多，而这批年轻人又是孟加拉大学体系下的精英，代表着孟加拉的未来。

你让这批人念完大学，继续回家务农？或者进厂缝衣服？上工地搬砖？他们当然不乐意了。自疫情发生以来，孟加拉国失业率便居高不下。数据显示，当下的失业人口高达近3000万，其中30岁以下年轻人占到了近1800万，每年有近60-80万应届和往届的大学生参加公务员选拔考试，但是竞争的岗位不过区区三千来个，想考公上岸，难于登天。

而看到与自己同龄的功勋阶级的后代凭着祖上的军功就能大摇大摆地走进政府，过上体面的公务员生活，不公的怒火已经悄然在各大学学生中蔓延开来。

2018年3月8日，孟加拉高等法院驳回了一项质疑配额制度合法性的申诉案件，学生们积压已久的不满终于前所未有地爆发为全国性抗议示威。

当时的那场抗议长达五个月，冲突造成了260多人受伤、44多人被捕，引发了国际社会的关注，迫使孟政府做出让步，在两年之内逐步放宽配额制，朝着公平公正的方向稳步推进改革。

然而到了今年6月，又是孟加拉高等法院的一纸裁定，毁掉了学生们好不容易争取来的成果。

原因就是一名“自由战士”的后代联合其他人就2018年政府取消配额制一事提出上诉，认为取消配额是违宪、非法和无效的，要求认定当初的政府命令不合法，高等法院审议后竟然予以支持，这就是通过司法途径事实上恢复了配额制，好不容易打破“家族传承”的公职世袭又回来了。

这次学生们直接要求取消功勋阶层的配额特权，只开放5%的比例给少数民族和残疾人士，其余的全部面向社会公开招录，给更多年轻人机会。

实际上，配额制实施这么多年来，经过孟加拉公用服务部门统计，近些年几次的公务员录取考试中功勋阶层及后代的实际招录人数根本用不完30%的配额比例，在这种情况下为了维持公职岗位的稀缺性和守住体制内的既得利益，招录部门就是不向社会公开，自家好不容易供养出一个大学生结果毕业即失业，而别人家孩子因为有个好爹好妈啥都不干就能身居高位。可想而知，民众对这种父死子继的制度会有多不满。

当然，不愿放开体制内多余的铁饭碗，背后还有执政阶层内部的政治动机。

## 二、 政治博弈与权力斗争

今年初，现在的执政党人民联盟刚在没有竞争对手的情况下赢得大选，在298个直选议席中赢得223个席位。但是大选投票率较往届相比依旧大幅下降，从2018年的80%下滑至今年的40%左右。

要说没有竞争对手，那是因为主要反对党孟加拉民族主义党，以“选举不公”“暴力横行”为理由抵制这场大选。

原因很简单，两党互不信任，都认为一方执政期间组织的选举有失公允，存在操纵选举的可能性，当然这不是现在才有，早在1994年、1996年就已有先例，过去的做法通常是当届政府期满后，由总统任命一个中立的临时政府来组织选举。

而这次哈西娜手腕强硬，丝毫不打算妥协，利用职权从去年底就开始调兵进城维稳，并且在选举前10天还禁止50人以上的政治集会，不给民族党组织造势冲刺的机会。

所以今年初的议会选举“毫无意外”的是人民联盟大获全胜，223席位里还有150个席位是在没有竞争对手的情况下自动当选的，形势处在下风的民族党只能号召支持者抵制大选，这也是本次投票率创下历史新低的主要原因。

两党缘何积怨颇深呢？

主要还是双方的政治理念大相径庭，包括国家意识形态，世俗化问题，在独立战争中发挥的作用等几个议题上。

比如人民联盟主张政治世俗化，实现政教分离，解散伊斯兰政党，民族党则认为90%信奉伊斯兰教的国民应该让世俗主义同伊斯兰文化相结合。

而在1971年独立战争时期，双方都认为自己的政党才真正发挥了中流砥柱的领导作用，互相否定对方的贡献。

于是乎，自上世纪90年代结束军政府统治恢复民主制度以来，两党把持下的孟国内政治局势开始陷入混乱。

双方之间不是抵制议会开会，就是组织工人罢工，不是绑架暗杀，就是政府镇压，下三滥的手段全都用上了，真的是能动手的绝对不会动嘴，结果导致严重的政治暴力事件频发。

在1991~1996年民族党执政期间，议会的400个工作日里，人民联盟不派议员参加，抵制了118天，并组织了416次罢工。

而到了1996~2001年人民联盟执政期间，议会的383个工作日里，民族党有样学样，抵制了156天，也组织了318次罢工，人盟政府为此还抓了5000多名民族党的活跃分子进行报复。

尤其是开头提到的人民联盟前主席，开国总理谢赫·拉赫曼就是被民族主义党前领导人卡莉达·齐亚的已故丈夫，前总统齐亚·拉赫曼发动的军事政变迫害致死，这可是杀父之仇啊，哈西娜怀恨在心立志报复。

1996年，哈西娜首次当选孟加拉国总理后，就开始追究当年政变参与者的责任。1998年，孟加拉国高等法院判处12名前政变领导人死刑。2010年，5名直接参与杀害谢赫·拉赫曼的政变军人被施以绞刑。

哈西娜自己也因为树敌太多，近几年多次遭遇到了暗杀（有记录19次），当然命硬也都挺过去了。

而对于民族党的齐亚，则在2008年，2018年两度因为贪污问题锒铛入狱。

而这次席卷全国学生抗议运动，则又是两党宫斗的一大高潮。

哈西娜指责是民族党旗下的学生组织——孟加拉民族学生党为主谋策划发动的，因为达卡警方在对一处学生党据点突击检查中发现了爆炸物和自制武器。年轻的学生们成为了政客之间隔空交火、相互攻击的工具。

而学生们则指责哈西娜任人唯亲，以权谋私。明明30%的配额用不完，多余的存量被哈西娜拿来犒赏自己的亲信，加上不公开录用信息，这里面很容易滋生腐败和暗箱操作。

面对学生的愤怒，哈西娜政府出动军方强力镇压甚至直接开枪的做法彻底激化了矛盾，导致这轮声势浩大的抗议运动已经从当初的反配额制，演化到了现在反哈西娜政府。

所以，此次试图恢复“自由战士”阶层的配额制，就是哈西娜凭借自己父亲开创的制度红利，借机更好地巩固人民联盟的基本盘，毕竟当年的抗战老兵及其子弟现如今大多为这个已经执政满15年的老牌政党的坚定支持者，只要进一步稳住体制内的政治精英，笼络好军人集团，就能确保大权不会旁落。

当前，迫于压力，孟最高法院上诉庭已经驳回了早前高等法院恢复配额制度的法令，开放93%的政府职位公开择优录取，只为“自由战士”其后代分配5%的职位，然而这项裁决能否让学生群体满意，还是未知数。

何况，即使这次哈西娜政府涉险过关，矛盾也只是被延后，并未得到彻底解决，她余下任期的施政恐怕是困难重重，步履维艰。

政治动荡，怕是接下来几年孟加拉国的常态了。
