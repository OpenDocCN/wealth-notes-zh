

# 拜登突然宣布退选，哈里斯能上位吗？

两个白头翁，一陨一落空，戴上皇冠者，女王坐最中。

这是美国“通灵女巫”朱迪·赫文莉，在2023年底作出的一个预言。

这位女巫是个传奇人物，她精通占星术、通灵术，曾经预言过苏联解体、911事件、英国女王去世、俄乌战争等一系列大事件，都得到了准确验证。甚至在成功预言苏联解体后，还被当时的美国总统亲自接见。所以如今80多岁的人了，还高朋满座，每天都有各种达官贵人和明星向她讨教各种预言。

虽然她显然是个神棍，但没想到形势的发展，似乎正在一步一步走近赫文莉的预言。

你看，“两个白头翁，一陨一落空”，7月13日那天，如果特朗普不是神来之笔扭了一下脑袋的话，恐怕也早已陨落了。而另一位白头翁呢？7月21日，拜登突然宣布，“终止寻求连任的竞选活动”，呼吁民主党支持自己的副手哈里斯，消息瞬间上了热搜。

那么，拜登为啥突然退选呢？哈里斯又能否印证“女王坐正中”的预言？

首先，拜登的退选，既让人出乎意料，也在意料之中。

说意料之中，是因为自从上次电视辩论之后，拜登把他的痴呆、语无伦次、思路不清毫无遮掩地展示在全世界面前，如果说此前民主党对拜登健康问题的遮掩还能掩耳盗铃的话，那么现在，几乎所有人都对拜登失望了，哪怕最坚定的民主党支持者也是如此。

说出乎意料，是拜登对退选的问题，一直很坚定，不退！

我们可以看看最近拜登的一系列表态，一开始，面对民主党支持者的退选呼声，拜登说自己不会退选，然后拜登强调，“除非被火车撞倒，否则不会退选”；当记者追问到底啥时候会考虑退选的时候，拜登又说，只有出现新的健康问题，才会促使他重新考虑寻求第二个任期的决定。

结果，没过两天，拜登就阳了。

然后美国各路媒体都纷纷传言，拜登家人开始讨论其退出竞选的计划，最快将于周末退选。

对于这种传言，先是拜登竞选团队出来辟谣说消息不实，“他老了，但他会赢”。

就在所有人都以为自己被美国媒体忽悠的时候，7月21日中午，拜登突然宣布：为了民主党和国家的最佳利益，他决定退出竞选，并专注于履行总统剩余任期内的职责。

这是闹哪样？既然要退选，前几天何必那么坚定？

显然事情没那么简单，如果拜登想退选，早就退了，不会拖到现在，那么现在自食其言选择退选，只有一个可能：被逼宫了。

7月初的时候，来自得州的民主党众议员劳埃德·道格特，就率先公开表示拜登应该让位。

明眼人都能看出来，这就是个小虾米，投石问路呢，拜登不为所动。

紧接着，逼宫一点一点加码，7月12日，24名前民主党国会议员发表了一封公开信，敦促拜登同意其他候选人竞争党内提名。

拜登还是不为所动，还发了一封公开信回怼。看到前议员不好使了，于是现任议员上场了。

参议员马丁·海因里希、参议院财务委员会主席、参议员谢罗德·布朗接连表示，拜登应该结束他的竞选活动，选择退位。这俩人可不是一般人，都是现任的民主党高层大佬，说明民主党高层动了换掉拜登的心思。但拜登觉得，自己还能抢救一下，因为自己还有两个关键的盟友，奥巴马和佩洛西。

毕竟，奥巴马曾提拔拜登为副总统，更助他在2020年入主白宫；佩洛西则协助拜登达成诸多立法成就，是在国会为拜登施政保驾护航的定海神针。

拜登对他俩寄予很大希望，因为5月份的时候，奥巴马还在为拜登的竞选活动站台，而佩洛西也公开说，“团结就是我们的力量”，呼吁所有民主党人紧紧团结在拜登身边，一起对抗特朗普。

但万万没想到，就是这样结下过深厚战斗友谊的老战友，竟然也叛变了！

先是奥巴马表态，拜登的胜选之路已经大不如前，拜登需要认真考虑参选的可行性。紧接着，资深众议员亚当·希夫也表示，拜登应该退选。

注意，亚当·希夫和佩洛西的关系非常密切，媒体普遍认为，他的言论背后有佩洛西支持，是佩洛西在传话，希望拜登能心领神会，大家最好不要撕破脸。

结果，拜登还是不理。事情到这里，佩洛西干脆也不装了，7月18日，佩洛西告诉一些议员，拜登“可能很快就会被说服退出总统竞选”。

奥巴马和佩洛西的表态被媒体报道之后，就营造了一种民主党高层都在劝说拜登退选的汹汹之势，给了拜登极大心理压力，拜登就算是再恼火，也必须认真考虑退选了。

这就是压垮骆驼的最后一根稻草，最终让拜登做出了退选决定。

拜登被这么多人连续逼宫，只说明了一个问题：在辩论之后，民主党内出现了一个致力于换掉党内候选人的“阴谋集团”，这个阴谋集团，包括不限于民主党高层、少壮派以及奥巴马、佩洛西这样的前盟友，而他们的目的，就是与拜登这个民主党的政治包袱进行切割，在最危急的时刻，死马当活马医，换个候选人拼一把。

但这个计划有个问题，从法理上来说，如果拜登自己不主动退选，阴谋集团是拿他没办法的，毕竟拜登已经赢了大多数州的党内初选，现在换人，各州的民主党代表干不干？别忘了，拜登虽然拉胯，但毕竟在政坛经营了这么多年，在各州还是有不少支持者的！

拜登自己不退咋办？那就想各种办法，发动一场“微型宫廷政变”，在短短一两周内，连续向拜登出了好几招。

- 第一招是“爆料”，阴谋集团安排了一个“白宫助手”向媒体爆料，八十多岁的拜登在六小时日光时间之外难以正常生活（上午10点到下午4点），其他时间，很容易心不在焉、失言或疲劳。这是暗示拜登身体严重不行了。
- 第二招是“儿子”，拜登的小儿子亨特·拜登在6月底，被陪审团一致认定三项重罪成立，或将面临最高长达25年的牢狱刑罚以及75万美元的巨额罚款。拜登想救儿子，只有总统特赦，然而一旦特赦自己儿子，很容易被攻击为徇私枉法。所以不排除民主党高层以特赦为条件，劝说拜登让位。就算民主党赢不了大选，没有特赦权，也可以和共和党做利益交换，换取特赦亨特。
- 第三招就是新冠了。拜登不是说，只有健康问题才会让自己考虑退选吗？那就让你健康出问题！于是，莫名其妙地，拜登就新冠了，距离他说出这话，也仅仅不到3天，这也太巧了好吗？如果拜登的新冠真的是有人故意让其染上的，性质就变了，称得上“弑君”。

面对这样花样百出的逼宫，拜登哪怕心里再不情愿，也只能老老实实退选了，毕竟，当所有人都背叛他的时候，他已经不再是总统，只是一个手无缚鸡之力的老人罢了。

## 2

拜登退选后，所有人都关注的是，谁会成为民主党新的候选人，去和如日中天的特朗普PK？

眼前的答案很明显，拜登推举了副总统哈里斯。

但问题在于，就算有了拜登的推荐，哈里斯能否上位，仍然是未知数。

为啥？因为这位大姐实在是太差了，比拜登还差！

副总统本来就是个吉祥物，平时基本没啥用，唯一作用就是做总统突然无法履职时的储备，以及参议院的一票（在选票一样时才起作用），当年拜登选她，无非就是她身上的buff足够多，可以拉女性、少数族裔、黑人群体的选票，而不是她真的有啥能力。

纵观当副总统这四年，哈里斯立下了什么功劳没有？

并没有。

大家对哈里斯的唯一印象，就是她在各种场合不停地“哈哈哈”，被称为“哈哈姐”。

对这样的一个“哈哈姐”，你说要让她成为拜登的接班人？真是想多了，民主党党内大把大把的人不服气。毕竟，虽然哈里斯是总统第一顺位继承人，却不是总统候选人第一顺位继承人啊！

## 那拜登为什么还要选哈里斯呢？

也许，这是拜登身边唯一一个没有背叛拜登的人了，只要她在，念在旧情的份上，和自己家族的各种利益，应该都能有保证，而且程序上也顺理成章，至于哈里斯到底能不能赢得党内支持，就看她自己的本事了。

但是目前，关于接班人的问题，目前民主党内明显出现了两派。

### 一派是地方势力。

拜登退选后，全美50个州的民主党主席举行了电话会议，各方均表示将全力支持哈里斯成为民主党总统候选人。同时，美国民主党全国委员会21日已提交一项修正案，对原来属于拜登的筹款委员会进行了名称的修改，并重新命名为“哈里斯胜利基金”和“哈里斯行动基金”，这意味着哈里斯马上就能用拜登的竞选资金。

但是需要注意的是，这些人支持哈里斯，并不代表指望哈里斯赢得大选。

为啥？因为他们很清楚，在现在的形势下，根基薄弱、政绩寥寥的哈里斯，根本不足以与特朗普竞争，那么既然如此，让哈里斯背这个败选的锅，再合适不过，其他党内候选人，可以毫无心理包袱地备战2028年大选。

第二派是阴谋集团。

前文说了，阴谋集团是觉得这次大选还值得抢救一下的，要不然干嘛逼宫拜登？无非是想换上来一个有实力的，去和特朗普PK。

在他们看来，哈里斯资历浅薄，显然不行。

所以，奥巴马在拜登退选后，赞赏了拜登的退选举动，但并没有像其他人那样，宣布支持哈里斯，而是说“相信民主党‘将能组织一个产生一位杰出候选人的过程’。”

而佩洛西则强调，不经公开竞争选出接替人选会赶跑选民，“这需要一个公开的过程”。意思是，党内总统候选人的确定，是全国代表大会决定的，拜登怎么能钦定呢？

这话也没毛病，所以，哈里斯要想成为总统候选人，可能还要走一个党内初选的程序，那就必然会面对阴谋集团推举的候选人的竞争。

## “阴谋集团”希望谁参选呢？

从目前的情况来看，他们希望加州州长纽森和奥巴马的老婆米歇尔，或者密歇根州州长惠特默和米歇尔搭配参选。

纽森和惠特默都是民主党中最有潜力的政治新星，本来就是专门为2028年美国大选培养的接班人，现在让他们提前参选，也问题不大。而米歇尔呢？是黑人，可以有效避免换掉哈里斯后带来的黑人选票流失。

不过问题在于，现在三人都曾表过态，不愿意参选（可能是怕得罪拜登支持者），但没关系，民主党有个“征召”模式，不管你愿不愿意，只要组织需要，就可以直接安排你当候选人竞争者，这样一来，就相当于民主党把拜登支持者的怒火给接过去了，不至于迁怒这些候选人。

如果是这样的话，被征召的候选人，就要抓紧了，此时距离民主党全国大会还有将近1个月时间，他们需要飞往全国各地开展党内竞选活动，尽量争取州代表和代表团的支持。

问题来了，如果无论是哈里斯，还是被征召的候选人，在代表们抵达芝加哥之前还没有达成共识咋办呢？那么佩洛西和奥巴马可能会推动改变8月份民主党全国代表大会的方式，改成“开放大会”。

开放大会指的是没有候选人获得明显多数的代表席位时，所有竞选者都要上台，说服代表们投他一票，一轮选不出来，就再投一轮，一直到投出候选人为止。在美国历史上的1924年，民主党曾经发生过经过103轮投票，才最终确定了妥协候选人的事情发生。

根据公开大会的规则，每个候选人都要获得至少300个代表的签名才行，而且同一个州的签名不能超过50个，这就必然会产生内幕交易、利益交换和抱团取暖。

相比根深蒂固的佩洛西和奥巴马来说，人脉微薄的哈里斯，显然不占优势。

不过，就算阴谋集团能利用这种手段干掉哈里斯，也同样会把民主党内部的矛盾公之于众，加速民主党的分裂。自从宾夕法尼亚那一发子弹开始，美国的这次选举已经不可控地走向混乱，而意外上位、野心勃勃的哈里斯，到底能不能实现“女王坐正中”的女巫预言？

就看她的造化了。
