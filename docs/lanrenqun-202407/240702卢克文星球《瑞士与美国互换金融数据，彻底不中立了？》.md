

# 瑞士与美国互换金融数据，彻底不中立了？

6月27日，瑞士和美国正式达成了“金融数据互换协议”。

这其实是一个爆炸性的新闻，但也正因为过于爆炸，舆论场反而风平浪静。

那么问题来了，这份协议到底是怎么回事呢？

我们知道，瑞士银行业一向以“保密严格”而著称，它们自己声称，不会向任何政府透露客户数据，包括瑞士政府。

世界上的许多人，他们的财产来源多少都带着灰色，不太见得光，对于财产保密有着强烈的需求，而瑞士的银行业，恰好能提供相应的服务。

除了保密制度，瑞士还是“中立国”，不会轻易挨揍，减少了因战争而被抢走财产的风险，这两点，让瑞士长期以来被部分富人和权贵所看重，并将大笔财产存放在瑞士。

不仅仅是中国富人，欧洲和美国的富人，也有许多选择将银子存在瑞士。尤其是美国，美国对本国居民是征收资本利得税的，反而对非居民不征收。

翻译过来就是，如果你能弄到一个非美国身份，就不用交这笔税，所以美国一些富豪绞尽脑汁地弄一个非美国身份，许多人都有双重甚至多重身份。

虽然美国在名义上，不承认双重国籍的合法性，但在实际中，操作办法多得是，想打擦边球并不难，这一点，无论哪个国家的富人，都是相同的。

同时，瑞士和美之间有投资优惠协议，也就是两国之间的投资，你给我税收优惠，我也给你税收优惠。

对美国许多富人来说，弄一个瑞士身份、注册一个瑞士公司，再以这个公司的名义回美国投资，吸引力非常大，不仅可以减免资本利得税，还可以享受投资上的税收优惠。

双重获利。

当这样操作的美国富人多了，美国国家的税收流失就显而易见了，所以金融危机后，奥巴马力推国会通过了《海外账户税收合规法案》（简称：FATCA）。

FATCA 的核心条款，就是“所有非美国外国金融机构有义务搜寻美国公民在该机构的金融活动记录，并向美国财政部报告这些人的资产和身份”。

简单来说，假设一个美国人在中国工商银行有金融活动，比如存款、投资啥的，那么工商银行就有义务向美国财政部提交报告，详细的报告。

赤裸裸且不要脸的长臂管辖。

稍微大一点的国家，根本就不会执行这种法案，因为事关主权。美国也知道，与其和大国纠缠，不如将主要精力放在了对瑞士等中小型经济体的施压上，要求他们向美国报告美国客户的金融数据。

说是美国客户，其实明眼人都知道，美国政府要的是全部数据，而不单单是美国客户的。

考虑到瑞士为世界上诸多富人服务，只要拿到了数据，美国政府就可以知道具体哪个国家哪个富豪/权贵，在瑞士有存款，存了多少。

有了这些数据，就能拿捏许多国家的统治阶层了。

而瑞士方面呢，一直不太配合。

虽然零零散散提供过一些数据，不过规模都不大，令美国很是不满。直到2013年，奥巴马威胁要制裁瑞士后，瑞士才和美国达成了“金融数据互换协议”的1.0版本。

根据 1.0 版本，名义上美国和瑞士能够互换数据，美国可以得到瑞士的金融数据了，先由美国财政部要提交申请，瑞士视具体情况决定是否配合。

结果就是，瑞士给的数据依旧不多。

毕竟瑞士是要做生意的，金融是瑞士的支柱产业，而保密性是瑞士金融的核心竞争力，数据都给了你，保密性全无，我还咋做生意?

如果世界局势没有重大变化的话，美国和瑞士的金融数据“互换”大概率就止步于 1.0 版本了。但是，俄乌战争这个大变量来了。

俄乌战争后，西方各国纷纷冻结俄罗斯资产，其中，瑞士冻结的俄罗斯资产在 350 亿美元左右。

当初打出这张牌的目的，是逼迫俄罗斯精英群体特别是富豪们，在巨额损失下迁怒于普京，从而发起颜色革命，从内部推墙。

然而没想到，此举效果不佳。

在极限制裁下，在资产被冻结的情况下，俄罗斯富豪们瑟瑟发抖，他们知道西方没有把自己当成“自己人”，失去选择的余地的他们只能支持普京，不管愿意与否。

瑞士因为冻结资产，打碎了“中立国”和“保密”这两块牌坊，而损失惨重，许多非俄罗斯客户纷纷从瑞士提款走人，导致瑞士规模第二大的银行--瑞信银行，在 2023 年爆雷。

瑞信的爆雷，一度让瑞士想缓一缓，希望挽回一些信誉，继续做生意。理想挺丰满，现实很骨感，都上贼船了，美国会允许你下去？

在美国现阶段的战略中，瑞士是极其重要的一环，毕竟有不少中国富人在瑞士有存款，这也不是秘密。

如果能拿到具体的数据，进可以冻结他们的资产，退可以用数据拿捏他们，时不时发布一些对他们不利的消息。

所以在俄乌战争后，美国加大了对瑞士的施压力度，软硬兼施，终于达成了 FATCA 的 2.0 版本，也就是文章开头说的，6 月 27 日的协议。

相比 1.0 版本，2.0 版本最大的变化就是不用再申请，不用再走流程，瑞士的金融数据将实时分享给美国。

名义上，双方的金融数据是互换，但谁都知道，瑞士是没那胆子问美国要数据的，就算问了美国也不会给，只有瑞士单方面输送数据的份。

这并不是瞎猜，而是有依据的。目前实行的 1.0 版本，名义上双方也是数据互换的，但美国财政部从未向瑞士传输过数据，只有瑞士向美国传输数据。

所以，2.0 版本也不会有什么意外。

2.0 版本一经达成，意味着两件过去不可能的事，已经成为了现实，一是瑞士事实放弃了银行业的保密制度，二是瑞士事实放弃了“中立国”地位。

俄乌战争后，冻结俄罗斯的资产时，还是犹抱琵琶半遮面，现在彻底不装了。

这对国内某些富人来说，恐怕是一件很难接受的事情，钱多钱少的还是小事，重要的是，2.0 版本的背后，是美帝要对他们动刀子了。

而过去几十年，他们信奉的价值观都认为，自己已经融入了“文明世界”，是“文明世界”的一员，是不会被“文明世界”打击的。

如果有，那一定是自身哪里做得不够好，惹恼了“文明世界”，加大反思力度即可。而“文明世界”是文明的，不会无缘无故打击自己，只要自己做得足够好，就不会被打击。

这种价值观尽管可笑，却在过去大行其道，殊不知，人家俄罗斯富豪好歹是个白皮，东正教严格来说也是基督教的一部分，都无法融入西方。

而某些人一身黄皮，拿什么来融入？
