

# 暗海沉沙——深海打捞员的诡异之旅，揭开尘封已久的深海异族之谜

《暗海沉沙》内容简介：

四个身份各异的年轻人古潼、花小七、刘一手和苏锐，被意外卷入了一场寻找海底“龙宫”的探险活动之中，他们怀揣不同的目的深入海底，发现了隐藏在海底深处的远古建筑群，看到了远古巨石阵、磁场诡异的远古金字塔等神秘历史遗迹，并遭遇了食人鱼、海蛇群、青鳞蛟龙、史前巨蟹等曾经远古生物的袭击，危险无处不在，谜团如影随形。在经历了一系列的冒险之后，他们终于揭开了“龙宫”内隐藏的“海底人”传说的真相和一个已经存在了万年之久的阴谋，同时四人也逐渐看清了自己的命运与海底龙宫之间的神秘联系。但是，潜入海底寻找“龙宫”真相的人并不只有他们，一股不明身份的J国杀手始终如影随形，似乎也在觊觎着海底深处那个巨大的秘密！

1990年夏季，北纬三十八度附近，东海麒麟岛海域。

“苍龙号”海洋监测船甲板上，六名设备作业人员通过一套特殊绳索装备，将一台海底探测仪缓缓送入水底。

指挥员白鲸拿着对讲机，不断重复着：“水下作业人员请注意，探测仪入水，预计五分钟到达九十五号礁石位置，祝你们好运……”

在距离海平面近一百五十米的海底深处，四名蛙人穿着防护潜水装备，看上去有些臃肿，但丝毫不影响他们顺利接住探测仪，并把仪器安放进一条礁石缝隙里面。

同时，一名叫古潼的蛙人呼叫水面：“报告指挥部，设备已经安放完毕，蛙人小组请求返回水面。”
水面指挥员道：“指挥部收到，你们可以回家了。”

退伍之前，古潼曾服役于海军陆战队蛙人大队，保家卫国多年，练就了一身的本领，两年前退伍，接着就被一家海洋环保机构聘请为海洋环境监测员，在很多重要的潜水作业当中，古潼都被任命为首席潜水员。

这一次，古潼的任务是把机构新采购的一台海底探测仪投放到海底。据说这次放置的海底探测仪非常先进，不但可以测绘出更加复杂的海底地貌，最主要的作用就是能够监测非常细微的海底环境变化。古潼热爱海洋，像这种有利于海洋环境保护的任务，他往往都是来者不拒。

十分钟后，四名蛙人依次浮出水面。古潼在向指挥员白鲸汇报完探测仪安放过程并确认无误之后，白鲸下令“苍龙号”开始满舵转向，加速驶离了该水域，准备与即将到来的“长远号”科考船汇合，并进行任务交接。

“长远号”是新型设备的提供方W公司的一艘数据采集船，按照合约，设备放置完毕，“长远号”将进入该水域进行相关数据采集。

可是，还没等“苍龙号”驶出多远，忽然间，远处的海面猛然蹿出一道非常奇特的水柱，突出海面至少三十米，几乎就是在同一时间，这道水柱凌空炸开，各种黑色的礁石碎片和来自海底深处的各种生物尸体，就如同天女散花一般四处散落，当过水兵的古潼瞬间就意识到了什么。

这是海底发生了爆炸才会出现的景象。

“怎么回事？”古潼心生疑惑，立即趴在船舷上去看水柱冒出的方向，发现爆炸地点竟然是放置探测仪的地方。

“苍龙号”上的人都愣住了，没有人知道发生了什么，十几秒之后，爆炸产生的冲击波到达了“苍龙号”所在的位置，尽管此时冲击波已经变得十分微弱，但还是差点把船掀翻。

“你们放置仪器的时候出什么纰漏了？”等船终于稳定下来，白鲸找到古潼问道。

古潼摸了一把被喷在脸上的海水，回道：“我们完全按照操作守则进行的，不可能是仪器出了问题！这么猛烈的爆炸，只有深水炸弹才能做到，这不是仪器的问题，你得问问W公司，提供给我们的到底是探测仪还是深水炸弹……”

白鲸想了想，觉得古潼说的有道理，不由陷入沉思。

“长远号”在一个小时后驶进了爆炸水域，两船汇合后，白鲸立即跟对方进行了信息交换，结果对方也不知道为什么会发生这样的事情。但是发生了这样的作业事故，双方都需要知道事故原因，特别是“长远号”，作为设备提供方，坚持就地展开事故调查。但是刚刚躲过一劫的“苍龙号”船体受损，而且有人受伤，无法陪同“长远号”进行作业，因此只好暂时返回陆地。

回到公司后，古潼立即找到白鲸，说自己认为这次的任务肯定有问题，引起那场爆炸的肯定是深水炸弹，并强烈要求报警。但是白鲸却阻止了古潼的这个念头，只是说W公司已经开始调查这次事故，而且机构方面也已经跟对方提出交涉，对方承诺一旦有了调查结果，会第一时间给予答复。

不知道为什么，古潼觉得白鲸在这件事情上的表现有些过于平静，不像是他平日里的风格。但是想来这也是没有办法的事情，他们都受雇于机构，即使发生了这样的事情，那也是机构跟W公司之间的事情，他们作为一线工作人员，除了等待调查结果，确实什么也做不了。

然而，就在事故发生的第三天，白鲸却突然找到古潼，并告诉他，当时进入爆炸水域进行事故调查的“长远号”出事了。

古潼追问发生了什么事情，白鲸就说，“长远号”在事故当天不但没有按时返回港口，而且还跟W公司失去了所有的联系。W公司在第二天就到相关部门申请了飞机和舰艇，对“长远号”展开了紧急搜索救援，然而在方圆几百里的范围内，搜救人员连“长远号”的影子都没看到。

但事情还不止如此，更加离奇的是，根据雷达系统显示，“长远号”最后发出的信号，确实是从爆炸海域发出来的，但根据相关部门的水下探测仪器显示，在麒麟岛以及附近的海域内，并没有发现任何沉船的痕迹。而且，古潼猜对了，引起海底爆炸的，确实是一颗威力巨大的深水炸弹，根据爆炸残留物初步推测，那应该是一颗鱼雷。

白鲸推测，在那条礁石缝隙更深的地方，有人先于他们一步，埋入了一颗鱼雷，目的不详。

古潼就问：“什么人干的，为什么要这样做？难道有人不希望我们在那里放置监测设备？”

白鲸摇头，回道：“有这种可能性，但微乎其微，因为想破坏那台设备的话，完全可以做到悄无声息，没有必要使用一颗鱼雷。

“还有一种可能。”古潼就道，“那颗鱼雷，也可能是历史遗留，只不过碰巧被我们遇到了，我在部队的时候听说过类似的说法。”

白鲸的表情还是很怪，好像心事很重的样子，喃喃说道：“我也希望这只是一种巧合……”

事情很快就在机构里传开，有人推测失踪的船只肯定是遇到了海盗，早就被海盗挟持到其他地方去了。有人就反对，说根据以往经验，麒麟岛海域常年有海军活动，完全是海盗的禁地，从来没有任何海盗组织在这片海域出现过，更何况海军动用了飞机扩大搜索范围好几百公里，盘查了所有的过往船只，都没有发现可疑迹象。

也有人说是不是遇到了骤起的风浪或者海底漩涡，毕竟这片海域刚刚发生了爆炸，不排除会引起小型海底地震和小型海啸的可能。但是这种可能性很快就被排除了，因为像这种近海环境的物理式变化，是很容易被监测到的，更何况搜救团队出动很及时，并没有发现所谓的风浪和其他类型的环境变化。

其他的一些说法就更加没头没尾了，一时间难以摆上桌面进行讨论。

也就是说，在找不到其他实际性的理论来解释这艘船去了哪里之前，这件事情暂时只能这么理解——“长远号”凭空消失了。

在连续搜救了七十二个小时无果之后，古潼所在的环保机构也出动了自己的轮船，让白鲸带队协助参加搜救工作，但是一个星期的时间一晃过去了，尽管在这段时间内，相关部门和W公司多次派出飞机和轮船不断扩大搜索范围，但依然没有找到“长远号”，事情似乎走入了绝境。

然而就在这个时候，古潼所在的单位内部，却忽然接到了一个陌生的电话，有一个声音还很稚嫩，而且自称花小七的渔民，在电话里跟环保机构的电话接线人员讲述了一个十分离奇的故事，并说他在几天前见过“长远号”。

可正是因为小渔民的故事太过于离奇了，在起初的时候，还没有人注意到它到底有多么重要，更没有人意识到，那个离奇的故事背后，实际上牵引着一个已经持续了很多年的巨大谜团，所以那通电话直接被列入了虚假信息的范畴，甚至连一份像样的通话记录都没有。

不过，电话的内容还是很快就传到了一个人的耳朵里，这个人正是跟“长远号”失踪事件有关的白鲸。他在听说了这件事情以后，同样使用电话反向联系到了渔民花小七。

在单位里，白鲸是个老资格，而且位高权重，但这是表象，没有人知道白鲸身份的特别之处，更没有人知道，他就是那个巨大谜团的秘密调查者之一，所以对于这样的信息，他觉得必须亲自过问仔细才行。对于很多人来说，这通电话的意义，只是有人提供了一份并不靠谱的线索，而对于他来说，这样的信息很有可能意味着，重新启动调查那个谜团的契机已经到来。

事实证明，白鲸的猜测是正确的。他是唯一一个认真听完了花小七所有故事的人，虽然那也仅仅是一通电话，但对于他来说，这就足够改变一些事情了。

花小七是海边的渔民，常年活动在麒麟岛海域以捕鱼为生，长大之后，他在一艘大型渔船上做了一名机械师，他所在的这艘“东方五号”渔船第一次看到失踪的“长远号”，是在“长远号”失踪后的第四天，也是这艘渔船连续没有打到像样海货的第三天。

那一天，船老大有些恼火，因为按照千吨级别的渔船来看，出海一次如果没有收获的话，他会损失很大一笔资金，因此他决定让渔船进入大海更远的地方试试风水。

按照航线，“东方五号”需要从麒麟岛的北面绕到海岛的南面，然后再继续向南直行，这样只需一天不到的航程，渔船就可以到达另外一片水产丰富的海域。

花小七作为一名机械师，只要渔船不出现机械故障，他平时的工作就是在机械舱睡觉，那一天，他在朦胧之间忽然听到船舱外面传来一阵骚动，好像是有人不断在喊着什么。

“那艘船就要追过来了，按照这样的速度差，很有可能会撞在一起……”

“我们会不会是被海盗盯上了，要不咱们赶紧跑吧……”

很快船老大的声音响了起来：“都别慌，我看这艘船的样子倒像是科考船，可能是出状况了，马上进行无线电通话，问问对方什么意思嘛！”

花小七被吵得心烦意乱，起身来到了渔船甲板，看到船老大和几名水手站在栏杆内侧，对着不远处的一艘体型更大的白色船体指指点点，所有人的表情都显得相当不自然。

“你们在嚷嚷什么？”花小七走上前去问道。

一个叫刘一手的船员看到花小七睡眼惺忪上来了，对他说道：“那艘船一直跟着我们，都快一天了，而且越来越近，谁知道是什么鬼幺蛾子。”

“哦？”花小七看着那艘船，一眼就认出那应该是海洋科考船一类的船只，他从小在海边长大，认识这种船，就说道，“有什么好担心的，这是科考船。”

船老大说道：“这走近了我也看得出那船是搞科学的，但是为什么一直跟着我们？试了好几次都甩不掉，他娘的就要撞过来了，这肯定有情况。”

花小七摇着头走进驾驶舱，拿起无线电开始调频，然后喊道：“我是‘东方五号’渔船，请六点钟航向的船只控制速度和距离，以免发生碰撞，如需帮助请及时沟通，收到请回复，收到请回复……”

连续喊了三遍之后，无线电里一直鸦雀无声，好像并没有人在听花小七在通信频道喊话，这不免让他隐约感觉到一丝不安，难不成那条船出事了？

花小七把无线电交给舵手，刚要走出驾驶舱去看那艘船的情况，结果还没等他走出舱门，就听到舵手忽然喊道：“有回音了！”

“对方说什么？”

“让我们跟着它，别的什么也没说。”

“继续通话，问明原因。”花小七心里嘀咕着，这他娘的是什么道理？

“又没有声音了。”舵手无奈地说道。

花小七跑出驾驶舱回到甲板，就看到那艘船竟然正在调整航向，往他们来的方向行驶了过去：“我去，来真的……”

花小七跟船老大说明了情况之后，船老大也十分不解，按理说这片海域才刚刚被解禁没几天，此时已经不在禁航范围内，怎么会发生这样的事情呢？

但是政府有规定，渔船必须要服从安排，不能在禁航区域捕鱼，在不能完全排除海军再次禁航的情况下，最好还是要服从安排，以免招来是非。尽管他心里还是有些困惑，但也来不及多想，于是下令让舵手一百八十度转向，跟着科考船一路返航，很快就再次来到了麒麟岛附近的水域。

然而，诡异的事情就在这时候发生了，而且来得毫无征兆。

包括花小七在内的“东方五号”甲板上所有的人都几乎同时看到，一直行驶在他们前面不到五百米的那艘巨大的科研船，竟然在到达麒麟岛海域之后，突然一下子就直接沉入了水里。

更加令人匪夷所思的是，那艘船在沉入水底的时候给人的感觉十分不舒服，具体形容出来的话，那就是，一艘悄无声息地行驶在海面的巨大科学考察船，整个船体忽然像被一张无形的巨网给罩住了，同时有一股力量在海面以下拉动这张网，使得整艘船以一种还在正常航行的姿态垂直地沉入了海底，没有惊起一丝波澜，也没有出现正常沉船时所产生的那种吸水漩涡，好像沉入水底的并不是一条实体的巨大船体，而是一艘大船的幽魂。

“我靠……”花小七亲眼看见了这一幕，跟渔船上其他人一样，完全呆傻掉了。

当船老大反应过来不能再继续前行的时候，“东方五号”渔船已经到了“长远号”沉没的位置附近，所有人都大气不敢出，唯恐自己的船也会出事，刚才“长远号”沉没的方式实在是太诡异了。

时间一分一秒地过去了，“东方五号”并没有出现任何不良迹象，慢慢地，在场所有人开始放松了警惕，大多数人开始七嘴八舌讨论起来，刚才那一幕，会不会就是传说中的集体幻觉呢？

传闻在航海历史上，因为船舶工作者的生活相当乏味，加上一些无法控制的意外时有发生，曾经就有一些轮船上的人出现过集体幻觉，例如长时间的食物短缺，长时间的营养不良，或者是遇到恶劣的自然天气和海市蜃楼一类的幻象等，都有可能会引发一些集体幻觉产生。

但是，“东方五号”是一艘渔船，而且刚刚出海，能想到的会出现集体幻觉的因素都不具备，这让本身就有些迷信的船工又开始怀疑他们是不是真的遇到幽灵船了。
