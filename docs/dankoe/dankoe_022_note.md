# 高效学习与记忆：如何记住你所学的每一件事

在本教程中，我们将探讨一种高效的学习与记忆方法。我们将从个人经验出发，分析有效学习的核心要素，并最终归纳出一套系统性的实践步骤。这套方法旨在帮助你从被动接收信息转变为主动构建知识，从而真正记住并应用所学的一切。

## 我是如何记住我所学的每一件事的

> 原文：[`thedankoe.com/letters/how-i-remember-everything-i-learn/`](https://thedankoe.com/letters/how-i-remember-everything-i-learn/)

如果你和我一样，喜欢深入研究深奥的概念，你可能会感到一种矛盾。你觉得自己永远无法像你欣赏的作者、演讲者或社交媒体上的影响者那样深刻、有洞察力或具有影响力。

我18岁时，因在大学停车场吸食大麻被捕。在那段低谷期，我阅读了埃克哈特·托利的《当下的力量》。我仔细阅读，练习冥想，并对“上帝”这个概念敞开心扉。这本书彻底改变了我，缓解了我对可能被定重罪的焦虑，帮助我冷静下来，对他人更少做出反应性行为。

我并没有记住那本书的所有内容，但我从中学到的东西，比在学校被要求阅读的任何其他书都要多十倍。

为什么我能记住那么多？

**1) 拥有明确的目标**

我非常重视设定目标。当我阅读那本书时，它完美地契合了我当时的生活困境。我处于人生低谷，面临法律麻烦和经济压力。因此，我带着强烈的**意图**去阅读，希望书中的教诲能改变我的心态。我愿意以极其开放的心态去做任何尝试。

**2) 怀有真正的好奇心**

我第一次从健身博主马特·奥格斯那里听说这本书。他是我当时的榜样，我信任他的推荐。这种信任激发了我对“灵性”话题的真正兴趣。好奇心迫使我放慢阅读速度，细致理解，特别是在难以理解的部分。

**3) 缩小“执行差距”**

> 一个人能培养的最伟大技能，就是缩短想法与执行之间的时间。
> 
> — DAN KOE (@thedankoe) [2022年5月5日](https://twitter.com/thedankoe/status/1522144576768823296?ref_src=twsrc%5Etfw)

阅读时，如果书中有需要练习的内容，我会立即放下书去实践。例如，尝试调动所有感官进行冥想练习。因为想法在脑海中还很新鲜，所以更容易判断练习是否有效。意识到它有效，会让这个想法在脑海中更加稳固。我获得了支持这个想法的**直接经验**。

拥有意图、好奇心和快速执行，是保留更多所学内容的绝佳方式。但我们可以做得更好。

## 🧱 你在建造什么？：项目式学习法

上一节我们探讨了有效学习的三个基础要素。本节中，我们来看看如何通过“建造”来十倍提升学习效果。

**实践**（名词）——*与理论相区别的实际操作*。

现代社会的问题是信息过载。我们喜欢囤积理论、教条和概念性理解，却缺乏直接经验。因为获得经验意味着要做让我们感到不适的事情，而我们已习惯了舒适。

真正的专家在大多数领域都很少需要指导。他们利用科学和理论达到中级水平，但进入高级阶段后，会认识到直觉和直接经验胜过一切。

那么，如何平衡实践与理论？

**答案是：成为一名建造者。**

也就是说，始终有一个与你所学理论相一致的项目正在进行。

当我学习Photoshop时，我一开始看了无数软件教程，结果陷入了“教程地狱”。当需要真正创造时，我大脑一片空白。后来，我改为边做项目边学习。如果我想学习建网站，我不会先盲目学习，而是选择一个建站工具，然后观看一个长教程，但**不**完全照搬。我会努力创建自己的网站，这种努力让我的大脑准备好“锁定”所学知识。

这种方法为何有助于记忆？

以下是其核心原理：

**1) 强制同步**

新奇感、模式识别以及相关的多巴胺对学习至关重要。当你的目标是构建一个项目时，你的思维会扩展，注意到那些有助于项目“生存”的事物。这就像《当下的力量》与我当时处境的关联性。这种关联性在意识中留下了印记，而发现新关联带来的多巴胺激发了立即执行的兴奋感。

如果你不集中注意力，你就不会“看见”。有一个项目能让你集中注意力，看到平时忽略的东西。

**2) 灵感行动法则**

> 我只在灵感来临时写作。幸运的是，它每天早上9点准时到来。 —— 威廉·福克纳

每个人都知道在灵感驱使下表现更好。拥有一个项目，你会自然而然地被驱使去学习与之相关的内容。如果你正在创业，你自然会去获取能激发创业灵感的资讯。但关键是，**要开始构建那个项目**。

**3) 培养专业知识**

我的建议是：**先行动，再学习**。

当你开始构建时，即使不知道自己在做什么，你也会遇到具体问题。这些问题会引发思考，并让你的大脑准备好学习。它们也为你提供了具体的研究方向。你不会盲目学习一堆没有上下文的信息，就像“脏增肌”一样，最终只会感到迟钝和疲惫。

不要盲目提问或学习。开始构建一些东西，然后针对出现的问题进行研究或寻求解答。重复这个过程。

## 🧠 你是项目：将一切联系起来

上一节我们介绍了通过项目构建来学习的方法。本节我们将更进一步，探讨如何通过“教学”和将“自我”作为终极项目，来最大化学习效果。

构建项目能帮助你保留更多知识。而教学能帮助你保留得更多。教学是将你在构建过程中获得的经验固化的另一种方式。

那么，何不将两者结合，从而创造一个充满意外发现、同步性和积极多巴胺的生活呢？

我们通过将**你**本身变成一个项目来实现这一点。这意味着追求全面的个人发展。个人发展不仅来自自我提升，也来自你对世界产生的影响，即你的事业、使命或你计划如何让世界变得更好。

这是一种自私与无私的结合：提升自己，然后帮助他人提升。记录你的旅程，以便他人能更快地成长。这就是“事业”的真正含义。

我已经帮助成千上万的人将他们自己转变为一个“单人企业”，让他们能够研究自己的兴趣，分享发现，并利用建造者和教师的双重天性获得创造性收入。

现在，让我们将所学的一切归纳为一个六步系统：

**1) 设定一个意图**

设定目标，创造目的，思考你的“为什么”。有意识地改善生活，实现潜力。这能将你的注意力引向未来，并鼓励你将精力投入到实现愿景的日常行动中。你所消费的一切信息开始为你提供动力。

**2) 安排创意时段**

![](img/6e7d52ebbef46fde94ded7511101a223.png)

每天安排30分钟进行创造性思考或散步。在这段时间里，你可以听与意图相关的有声书、思考未来、或让自己远离干扰。当你将注意力从外部转向内部时，大脑的“默认模式网络”就会启动，开始整合信息，将新的发现带入意识。

**3) 创建笔记系统**

记录任何能够**推动**你构建努力的信息、见解或发现。当你的心思专注于未来目标时，你的意识范围会扩大，开始从噪音中过滤出信号。就像你第一次注意到某款车后，会突然发现到处都是它。

**4) 构建你生活的每一个方面**

你必须明白你是在**构建**你的生活。爱上这个无限游戏。尝试所有感兴趣的事情，然后加倍投入你认为可以终身奋斗的事。这需要时间、实验和耐心。每天积极构建你的生活，从30-60分钟开始。如果你不花时间构建自己的生活，你就会被安排去构建别人的生活。

**5) 教授你所知**

构建项目是与目标生活保持一致、保留特定知识的绝佳方式。为了保留更多，引入**辅导效应**，即“教学相长”。当你将好奇心、项目构建和教学相结合时，你的理解和表达能力会得到极大提升。教授你的热情，可以在日常生活中开启新的意外发现水平。

**6) 将其变成你的人生事业**

> 你生活中可能发生的意外发现，你的**幸运面**，与你做你所热爱的事情的程度以及有效传达给的人数成正比。 —— 贾森·罗伯茨

工作是生活必要部分，但对许多人来说缺乏兴趣。如果你不想将生命的大部分时间消耗在讨厌的事情上，可以这样做：

+   研究你真正感兴趣的课题。
+   深入探索，不要分心。
+   理解：**持续的好奇心 + 兴奋 = 激情**
+   用我们讨论的方法维持那种激情。
+   记录发现，构建项目，并开始教授他人。

**产品最初是项目。** 如果你自己是**项目**，当你公开分享时，你就变成了人们愿意投资的**产品**。先将自己项目化，再将自己产品化。创业者推出的是MVP（最小可行产品）——一个提供足够价值以推出，并可随时间持续改进的项目。

**你是你生命的工作。** 你的最小可行产品在出生时就已存在。现在，推出，构建，教学。

---

在本教程中，我们一起学习了如何通过设定意图、保持好奇心、快速执行来奠定学习基础。进而，我们深入探讨了“项目式学习法”的核心，即通过构建具体项目来强制同步信息、激发灵感并培养专业知识。最后，我们升华到将“自我”作为终极项目的理念，通过记录、构建、教学并将其发展为事业，形成一个完整的学习、成长与价值创造的闭环。记住，高效学习的关键在于从被动的消费者转变为主动的建造者和分享者。
