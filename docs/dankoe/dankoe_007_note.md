# 心灵掌控之道：生活艰难，习惯它

> 原文链接：[`thedankoe.com/letters/life-is-hard-get-used-to-it-the-way-to-mental-mastery/`](https://thedankoe.com/letters/life-is-hard-get-used-to-it-the-way-to-mental-mastery/)

在本节课中，我们将要学习如何通过调整心态来应对生活中的固有困难。舒适感是暂时的，人们却常常将其误认为是永恒的归宿。有趣的是，我们越是固守在这个拒绝离开的心理舒适区，反而会感到越不舒服。通往真正舒适的唯一道路，是培养一种对生活中不可避免的不适感到自在的心态。

## 核心问题：逃避不适的倾向

当人们面对不适时，他们的自动反应是逃避，而非面对。我们可能不会逃离物理环境，但会试图逃回过去经历过的心理舒适区。这种逃避造成了内心的紧张。除非解决这种紧张，否则它会持续存在，比应有的时间更长。

这个问题的结果是导致分心、感觉“迷失”和陷入困境。我们不知道下一步该追求什么，因为我们缺乏不适感随着时间推移所能带来的清晰度。当我们处于这种“迷失”或混乱的心态时，我们无法超越最初的问题。原因在于，这个问题正亟待解决。

心灵决定了我们如何与现实互动。它容纳着我们的视角、感知和行为模式。大多数人持有狭隘的视角，这导致了对情况的负面看法，并决定了他们的无意识行为。我们的目标是以优雅和艺术的方式度过生活。这意味着，无论心灵如何试图将情况解释为困难，都要与当下合一。

当你做到以下几点时，你会意识到自己被灌输了一个谎言：传统道路并非通往美好生活的途径。
+   以开放心态对待生活，拓宽你的视角。
+   获得接受现状的力量。
+   享受进步带来的喜悦。

要做到这一点，你需要理解一些关键概念。

## 三种关键视角

期望与现实之间的差距创造了紧张，进而导致痛苦。哲学理论必须与实践相结合。知识需要以多种方式与现实接触，以验证其有效性。最好的哲学是建立在现实基础上的，并且是个人化的。

这里提供三种视角，帮助你认识到缩小以下差距的重要性：期望与现实、概念与经验、以及“是什么”与“不是什么”。

### 视角一：期望 VS 现实 🎯

期望是信念，而这些信念往往是无意识的，是在我们生活中被潜移默化地植入头脑的。期望是对未来的投射，是假设某事“应该”或“将会”发生。

期望将你的思维局限在一个可预测的未来，并阻止你敞开心扉接受其他可能性。从这种狭隘的心态出发，我们的视角是有限的。由于视角有限，我们对形势的认识也是有限的。由于认识不准确，我们常常做出不利于理想未来的选择。选择至关重要，它们像砖块一样构建着你的未来。

让我们通过一个例子来理解：

**例子：炎热的天气 vs. 桑拿**
当人们想到在炎热天气外出时，他们会抱怨，因为他们的期望将思维局限在负面结果上。当人们想到去桑拿时，这是有意的行为，通常与一个有意义的积极结果相关联。两种情况的现实（高温）是相同的，但期望不同，这导致了前一种情况中的痛苦。

**更实际的例子：创业**
以下是责任重大者想创业时常见的思维误区：
+   他们看到责任较轻的人在6-12个月内取得了商业成功。
+   他们没有考虑自己的具体情况，就抱持了同样的期望。
+   他们思考为什么自己不能做同样的事，并抱怨生活不公平（因为有工作、孩子、配偶）。

他们未能面对现实。事实上，他们在责任较轻时没有抓住机会创业，这没关系。过去无法改变，你越早接受现实，就能越早开始进步。一旦接受现实，他们就可以从一个新的、更可持续的角度来接近目标。他们可以寻找合适的导师、教育和策略，这同样能导向成功，只是时间跨度更长。另一种选择是成为期望的奴隶，永远达不到目标，明知可能却抱怨自己无法实现。

### 视角二：概念 VS 经验 🧠

概念只能从经验中创造出来。一个典型的误解例子是“上帝”、“宇宙”或任何你称之为更高力量的存在。宗教人士执着于“上帝”这个概念，但未能意识到它是一种**体验**。这个概念只是那种体验的象征性表示。

区别在于“知道”和“理解”。
+   **概念**使你能够**知道**。
+   **体验**使你能够**理解**。

知识是现实之上的一个层面，而理解是人类意识的内在方面。你可以知道如何打扫房间，也可以把房间打扫得很好，即使你并不想打扫。概念是必须与现实接触才能被体验的指南。现实就是体验本身，如果你在关注的话，这就是全部。人们喜欢概念化，因为这可以让他们不必前进到世界中去解决（内部和外部的）问题，而解决问题会带来更深刻、更不肤浅的情绪体验。

### 视角三：是什么 VS 不是什么 ⚖️

“就是这样”是你一生中会听到的最深刻的陈述之一，但很少有人意识到它的深度。在“是什么”和“不是什么”之间做出区分，是缩小期望与现实、概念与经验之间差距的绝佳方式。

“是什么”就是当下的瞬间，就在你面前的东西。
+   你手中哑铃的重量是**现实**。期望它应该很轻则**不是**。
+   为自己建立声誉的奋斗是**经验**。你心中对充实生活的概念则**不是**。

请注意，我并不是说期望、概念和“不是什么”是坏的。主要问题在于**消极的期望**、那些让你“知道”却阻止你行动的**概念**，以及无法与“是什么”合一的能力。

### 本节重点

在所有这些视角中，核心重点是：当生活艰难、令人不知所措或你对未来感到绝望时，是因为期望与现实之间出现了不协调。你头脑中的信念、想法与眼前展开的当下时刻不匹配。因为你给予了它们过多的关注，它们就优先于并掩盖了你的感官感知。一个是外在的建构，另一个是内在的真理。

上一节我们介绍了三种导致内心紧张的视角差距，本节中我们来看看如何通过提升思维水平来解决这些问题。

## 思维水平 🧗

> “没有任何问题能从创造它的同一意识水平上得到解决。” —— 阿尔伯特·爱因斯坦

问题被编码在系统之中，很少有人能看清它们的本质。问题引发了期望与现实之间的差距，只有当你能缩小这个差距时，问题才变得容易解决。但是，要解决问题并缩小差距，你必须达到一个新的思维水平。

在这里，我们将“思维”定义为你如何与现实互动。从这个意义上说，思维就是视角。将视角想象成一个包含所有期望、信念、想法、技能、经验、潜能等的领域。这个领域越宽广，你的思维水平就越高，就越难被单一的期望、信念或技能所束缚。

随着你解决每一个问题，生活感知起来的难度会降低，你的心灵层次会提升。你面临的问题的难度，是你当前个人发展阶段的最佳指标。你能解决的问题的难度，也决定了你能为他人解决的问题的难度。这意味着，通过承担不断增长的挑战来发展自己，你将获得更多经验。你面临的挑战将要求你多次缩小期望与现实之间的差距，每次这样做，你都会得到一个教训。这种经验可以转移到商业领域，因为产品解决的就是问题，所以你的经验越丰富，你的业务就可能越有利可图。

必须保持平衡，因为你不可能在连每天坚持走10公里的习惯都无法维持的情况下，去尝试攀登珠穆朗玛峰。

了解了思维水平的重要性后，接下来我们探讨达到新水平的具体方法：生活在你的能力边缘。

## 生活在你的边缘——心灵掌控之道 🚀

> “当你开始打开自己，生活在自己的极限边缘时，你最深层的目的将逐渐开始显现。在此期间，你将经历一层又一层的目的，每一层都越来越接近你最深层的完整目的。” —— 大卫·迪达

为了达到新的心灵层次，你必须遇到问题。你不能坐以待毙，反复做同样的事情而不考虑个人进化。你必须生活在技能和挑战的边缘。在解决问题的过程中，你提升心灵层次。为了解决问题，你必须提高技能水平。

以下是实现这一目标的四个步骤：

### 1) 追求一个具有挑战性的目标 🎯

这几乎是我每一篇通讯的核心教训。你需要一个每天都在为之努力的目标。一个能引导你走向理想未来的目标，即使你还不知道那个未来具体是什么。这个目标通常涉及健康的身体、充裕的财务和繁荣的人际关系（追求这些目标的结果往往是内心的平静，而不一定是完全实现它们）。当这个目标实现后，你必须设定一个新的目标（可能一开始并不明确），以防止自己陷入停滞。

### 2) 保持对当下困难的意识 👁️

你对于“挣扎”、“困难”或“艰难”这些词有什么样的判断、含义和联想？它们是否导致了对压力未来的强迫性思考？或者导致了对过去错误的回忆，从而引发恐惧以致无法行动？也可能是两者的混合，让你陷入焦虑的漩涡？请记住，这些只是你心中的期望、概念和“不是什么”。它们不是现实。现实本身就包括挣扎、困难和艰难。你无法逃避它们。这需要对当下时刻的彻底接受。

### 3) 达到感知阈值的临界点 💥

![感知阈值示意图](img/c21b44e18747ab061b5743d43b961671.png)

如果你遇到了写作瓶颈，如果你已经跑了30分钟的5公里，如果你正在进行最后一次365磅的深蹲，如果你犹豫是否发布第一篇内容，如果你已经冥想45分钟而大脑恳求你去查看通知……此时，思想和情感会涌入你的脑海，而“唯一的出路就是通过”。

你必须**训练**你的疼痛耐受能力，因为当你坚持足够长的时间，在任何情况下达到“跑步者高潮”般的状态时，魔法就会发生。

**感知阈值**是经过一段时间的挣扎后达到的临界点，**曾经困难的事情由于你对情况感知方式的转变而变得毫不费力**。
+   **期望**融入**现实**之中。
+   **概念**融入**经验**之中。
+   **不存在的东西**融入**存在**之中。

这可能需要5分钟，1小时，对于那些无法立即解决的问题，甚至可能需要数年。混沌是清晰的前提，而清晰是世界上最强大的燃料。

### 4) 达到新的思维层面 🌄

跨越感知阈值会迫使你提升到一个更高的思维层面，或者说，获得一个超越原先狭隘状态的更全面的视角。从这个新的层面来看，你可以准确地感知之前的问题，并看清它的本质。
+   “没有写作想法”变成了一个愚蠢的借口。
+   不完成最后一次（收益最大的）重复，变成了“我只是有点娇气，坚持下去。”
+   不发布你的第一篇内容，变成了“如果我这么做，它就会变成习惯，我就会一生陷入当前的状况，这与我的愿望相反。”

结果是深深的满足感。

## 总结

本节课中我们一起学习了心灵掌控之道。我们认识到，逃避不适是本能，但会导致紧张和停滞。通过审视并缩小**期望与现实**、**概念与经验**、**是什么与不是什么**之间的差距，我们可以更准确地认识世界。解决问题的关键在于提升**思维水平**，而提升的方法就是**生活在技能与挑战的边缘**。具体通过**追求挑战性目标**、**保持对困难的意识**、**坚持直到跨越感知阈值**，最终达到**新的思维层面**，从而以更清晰、更有力的方式面对生活。生活确实艰难，但通过习惯它、掌握它，我们便能以优雅和艺术的方式度过人生。
