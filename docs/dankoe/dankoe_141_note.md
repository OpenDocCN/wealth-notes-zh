# 多巴胺流行病：社会如何未能长期思考

在本节课中，我们将探讨现代社会面临的“多巴胺流行病”现象。我们将了解多巴胺的本质、社会技术基础如何塑造了我们的奖励系统，以及廉价多巴胺泛滥如何侵蚀了我们的长期思考能力。最后，我们将学习如何通过重塑数字环境和行为，从被动的消费者转变为主动的价值创造者，从而重新掌控自己的生活。

---

## 1：历史背景：从稀缺到泛滥 🕰️

为了理解我们正在经历的廉价多巴胺流行病，我们需要回顾一下历史。社会可以根据其技术经济基础进行分类，即社会用来满足基本需求的核心生产方式。主要有五种类型：

**觅食** – 以狩猎和采集为生。男性狩猎，女性采集。

**园艺** – 进行简单的种植。女性生产了大部分食物，社会崇拜的神祇也转变为女性形象。

**农业** – 使用动物拉犁耕作。由于耕作增加流产风险，女性无法继续承担主要耕作任务，因此社会角色转变，神祇形象也变回男性。80%的农业社会依赖男性奴隶劳动。

这标志着文化的巨大转变。因为耕作对女性在达尔文主义上不利，她们转变角色，开始承担更多“传统”职责，如照顾家庭和子女。由于犁的出现带来了食物丰裕，男性有了更多空闲时间来思考和研究新事物（书写和数学便是在这个时期发明的）。

在农业之后，最后两个基础是**工业**和**信息**。

值得注意的是，*智人*物种大约在200万年前出现，而现代*智人*大约在30万年前出现。工业时代始于约300年前（仅占人类进化史的千分之一），这意味着我们的大脑显然没有适应接下来发生的事情。

## 2：廉价多巴胺的崛起 📈

随着工业时代的发展，脂肪、糖和盐变得越来越不稀缺。我们可以在沿海地区交换盐，并能以很快的速度生产脂肪和糖。

在1920年至1950年间，情况继续变化。装配线和朝九晚五的工作制成为常态（这要归功于亨利·福特），人类的工作方式开始转变，进入我们今天所知的劳动力市场。简而言之，人类拥有了更多的金钱和潜力。

1950年后，快餐连锁店开始兴起。接着，信息时代到来。电视的黄金时代让信息传播得更广。

随后，互联网出现了。信息开始像野火一样增长和传播。快进到今天，我们有了谷歌、Facebook等巨头公司，它们为争夺顶峰而竞争。

起初，这一切并非全是坏事，但对权力的渴望导致技术在相当糟糕的激励机制下发展。一切都是新的，前所未有。谁能想到它会变得如此具有破坏性？

所有这些跨领域的巨头公司，不仅仅是互联网公司，都在试图通过任何可能的方式“赢得游戏”，即使这意味着牺牲客户的福祉。

### 脂肪、糖和盐的隐喻

人类不适应拥有丰富的、曾经稀缺的资源。我们适应的是生存。当找到稀缺成分时，多巴胺会涌入我们的大脑，发出信号，表明我们找到了有助于生存的东西。

一旦公司发现了这个机制，他们就可以测试并创造这些成分的组合，以最大化客户的愉悦感，并让他们不断回来寻求更多。这就是上瘾。

例如：四层汉堡，配上超大杯48盎司的奇佐查卢帕玉米卷苏打水。

互联网公司也效仿了。他们采用广告模式盈利，并很快发现，两极分化和煽动性的标题最能吸引注意力。现在，他们只需要向用户“喜欢”的内容提供更多，并立即呈现在屏幕上。于是，“为你推荐”页面诞生了！

在这个隐喻中：
+   **脂肪** = 恐慌主义
+   **糖** = 快速且诱人的承诺
+   **盐** = 绝对主义的选择

互联网公司测试这些组合，算法为我们提供尽可能多的信息，直到我们在精神上变得“肥胖”。引发恐惧的内容将多巴胺注入我们的大脑，并收窄我们的思维，因为我们“需要”这些信息以备后用。

## 3：多巴胺的本质：是“更多”，而非快乐 🧠

上一节我们看到了廉价多巴胺如何通过现代技术泛滥。本节中，我们来深入了解多巴胺本身的工作原理。

> 广义上说，称某物“重要”是另一种说法，即它与多巴胺有关。为什么？因为在众多功能中，多巴胺是任何可能帮助我们生存的事物的早期预警系统。——丹尼尔·Z·利伯曼

大多数人并不了解多巴胺。由于不理解，他们很难注意到不良行为并以有益自身发展的方式纠正它。

**多巴胺不是快乐分子，它是“更多”的分子。**

但多巴胺并非唯一根据我们行为和感知提供反馈的神经递质。神经递质主要有两种类型：

**1) 向下型化学物质或“当下”化学物质**
当你向下看时，你看到了什么？你个人空间内的一切。触手可及、你可以控制的东西。
与直接空间事物相关的“当下”化学物质让你能够*品味、享受、战斗或逃离*你所经历的事物。
这类物质包括：催产素、去甲肾上腺素、肾上腺素、血清素、内啡肽和内源性大麻素。

**2) 向上型化学物质：多巴胺**
向外看。仰望天空。上网浏览。看商店橱窗。回忆愉快的假期。
所有这些事物有什么共同点？*你尚未拥有的事物。*
多巴胺激励你去追求、控制和拥有你想要但尚未拥有的东西，因为它们可能（或曾经）对你的生存有用。例如：更多资源、更多性和后代、一匹马和一把剑用以穿越崎岖地形并在未知土地上发现更多。

## 4：问题核心与解决方向 ⚖️

理解了多巴胺的机制后，我们来看看它引发的问题。问题在于，我们被基于博弈论原则的海量信息所淹没，以至于失去了与现实的联系。

人际关系受损，因为一旦你因多巴胺驱动的动机而“拥有”了一个伴侣，你未能重新编程自己的习惯，在蜜月期过后去欣赏和品味这段关系。

在之前的讨论中，我们探讨了如何通过*存在、行动和成为*来重新与现实建立联系。

多巴胺是期待分子。因此，如果我们想享受已拥有的事物，就必须训练思维从未来导向转向持续的当下导向状态。

换句话说，*更多地成为*。沉浸在当下。培养正念习惯。去散步。凝视一片叶子，在这个小小的绿色物体中，发现生命交织网络带来的快乐。这可能是你缺失的核心习惯。

“多巴胺排毒”之所以有帮助，是因为你消除了所有你“没有”的东西，变得极其无聊。你唯一的选择就是对眼前的事物产生兴趣。像孩子一样看待一个全新的世界。

但是，你不必进行彻底的多巴胺排毒。你只需要改变对待、使用和利用那些正在摧毁大多数人生活的技术的态度。

## 5：全面解决方案：追求长期满足感 🎯

问题是这样的：多巴胺本身并非坏化学物质。它是成功与失败背后的驱动力。区别在于长期满足感与短期满足感。

这种差异是**巨大的**。
+   短期满足感——或称“廉价”多巴胺——导致**熵增**。
+   长期满足感——或称“获得性”多巴胺——导致**熵减**。

**熵增**，就我们所讨论的范畴而言，是逐渐陷入混乱、不确定和无序的过程。例如：焦虑、不堪重负、以及那种似乎只会恶化、无法理解世界的无法逃脱的负面思维泡沫，这通常是由于长期停留在事物表面导致的。

**熵减**，或者说逆转熵，是我们人类都面临的向上奋斗。它指的是维持、创造或改善秩序和确定性。例如：挑战如何带来满足感、心流体验以及对生活目标或控制的感知。

由于社交媒体是我们生活中如此庞大且几乎是必需的一部分，我们重点讨论它。

虽然社交媒体可能很快变得有毒，但很难忽视它是我们工作方式、未来工作形态、沟通方式以及与他人协调以改善社会的基本组成部分。当然，最后一部分正被过度利用。似乎没有人愿意将自己的思维判断交给那些希望最大化你在平台停留时间并利用你思维回路的公司。

这就引出了一个问题：成为定制算法的奴隶，它会给你带来短期满足的内容，让你处于压力和生存状态。恐惧和欲望的结合让你远离当下，导致熵增。你以为无害的帖子，实际上是由你同时阅读的50个短帖组合而成，这些帖子无声地在你的脑海中引发混乱。你退步了。你停滞不前。你将自己锁定在政治意识形态和静态观点的范式中，这些都不利于你在生活中取得任何有价值的成就。

如果社交媒体公司的激励机制不同，我们所有人的处境都会更好。但我们现在已深陷其中，明智的做法是不信任任何人，尤其是那些声称希望提高你生活质量的超级企业。这取决于你自己。

## 6：创造成功的新数字环境 🖥️

你听说过“破窗理论”吗？它非常引人深思。

简而言之，有一个实验研究两种不同环境：一种是有意打破窗户的环境，另一种看起来干净整洁。

在安装新窗户并清理区域的实验区，*犯罪率下降了约50%。*
在打破窗户并制造混乱的实验区，*犯罪率上升了。*

为什么会这样？因为你的环境在很大程度上决定了你的行为。

在互联网上，我们每天都在不断地将思想暴露在每天都有成千上万个“破窗”的区域。数字环境比物理环境更快地塑造你的思维，因为信息量更大。

这里有一个关键点：很少有人意识到，*是你自己选择置身于那个环境，作为一个有意识的个体，你可以创造一个新的环境*。

## 7：成为价值创造者：获取“获得性”多巴胺的生活 🚀

社交媒体是我们拥有的最伟大技术之一。它本身可以成为取得非传统成功的主要引擎和燃料。

想想看，社交媒体允许你：
+   **学习几乎任何东西**：挑战你的理解，寻求新视角，更接近真理。
+   **建立事业或独立收入来源**：你可以间接接触到数十亿人，*帮助他们*，以换取金钱或关注等回报，这些可以用来引导更多人走向长期满足的道路，产生连锁反应。
+   **寻找潜在的雇主、投资者、团队成员或人脉**：创造高价值内容现在可以为你完成大部分工作。

所有这些都是**中心化**的。它们需要时间和努力。它们通过提供有意义的输入和输出来逆转你习惯的熵和无序。它们是获取“获得性”多巴胺的途径。它们导致更大的复杂性和秩序，使你能够发现更多机会，解决更多问题，创造更多价值，并以有意义的方式获利。

由于社交媒体公司不会改变自己的激励机制，因此创建你自己的事业、你自己的秩序、你自己的环境以及你自己成功的催化剂，就取决于你了。

## 8：实践三步法：重塑你的数字生活 📝

以下是具体如何在三个步骤中做到这一点：

### 步骤一：筛选你的数字环境

第一个罪魁祸首是短内容。它可以用于快速获取见解，但很快就会变成一个充满梗图和毫无意义的兔子洞。

我们不再用那些主要动机不是成为《纽约时报》畅销书或创造最具争议性标题的书籍和文章来挑战我们的信念和理解。如果它们的主要意图是追求地位（这是主要意图，每个人都在以某种方式追求地位），那么它们传播的信息将据此定制。

*你在互联网上花费的大部分时间应该用于阅读那些能够挑战、扩展和培养你观点的长篇内容。*

多读长文章。阅读关于更复杂概念的内容。阅读那些*不*确认你信念、只为了让你感觉良好的内容。阅读很容易被误解为精神自慰，而实际上它远不止于此。我认为你未来的大部分成功都取决于这个单一的习惯：阅读更长的内容。

换句话说，消费那些有助于你发展的内容，而不是相反。
+   取消关注那些你*知道*对你成长无益的账户。
+   寻找你喜欢的长篇写作、视频和播客。
+   关注社交媒体上那些长篇创作者——他们也能写出更好的短内容来填补知识空白。
+   如果你愿意，注意那些人与谁互动，并关注他们。
+   算法将开始为你提供更好的内容。对于那些对你无益的内容，要无情地取消关注或屏蔽。

你越是精心管理你的数字环境，你就越能减少熵，促进中心化。

### 步骤二：专注于一个有意义的项目

> 去做一些伟大的事情，你的网络将立刻出现。——Naval Ravikant

现在你有了好的、中心化的输入，你需要好的、中心化的输出。

多巴胺不仅仅来自消费信息，因为信息并不仅限于你阅读和听到的内容。你的习惯……你的行为……会在积极或消极的方向上影响你的思维。你可以专注于一个构成你理想生活方式拼图的项目，或者你可以用导致混乱和失序的梗图和短暂快乐来填补你的无聊。

为什么要建立项目？因为它们是有序的。它们塑造你的思维。它们让你能够将你的输入——你消费的信息——转化为真实而有价值的东西。你可以从中赚取收入的东西。你可以回馈给世界的东西。

如果你没有要建立的项目，那么很可能你已经在做别人的项目了……一个老师或雇主分配给你的项目。你将花费8小时去实现他们的梦想，但不会花1小时去实现自己的梦想。是的，只需要1小时就可以完全重置你的心理健康。

### 步骤三：赢得你的控制与自由

现在，你该建立什么？我假设你是从零开始。你存款很少。你可能有一些技能。你有野心，但不知道从何开始建立自己的事业。

事实上，我们已经不再处于工业时代，我们现在处于信息时代。找份工作并投资你的储蓄*并不是唯一的选择*。

实际上，如果你没有注意到，人们正在失去对传统路径承诺保障他们未来的信任。这是最糟糕的选择之一，因为你的时间、精力和情感福祉都受老板的控制。

所有新的财富都在数字空间中产生。在思想空间。在创作者经济中。这是一个人们不受其人际关系或地理位置限制的地方，因为他们可以接触到任何有互联网连接的人。

如果你拥有正确的技能——写作、营销和构建——你所要做的就是让你的作品出现在那些认为你所建之物有价值的人面前。

所以：
+   创造你希望在世界上看到的内容。
+   建立你希望在世界上看到的产品。
+   创造能够拓展你思维并鼓励你在写作中成长的内容。通过建立受众来吸引志同道合的人加入你的兴趣圈。这些人是你最能帮助的人。
+   构建有助于他人发展和自我实现的产品。向你的受众推广你的产品——因为你不再有老板为你做这件事了——并赢得你的自由。

## 总结 ✨

在本节课中，我们一起学习了“多巴胺流行病”的成因与影响。我们从社会技术基础的历史演变出发，看到了脂肪、糖、盐以及现代信息如何被设计成让人上瘾的“廉价多巴胺”来源。我们深入了解了多巴胺作为“更多”分子的本质，以及它如何驱动我们的行为。

关键在于区分短期满足（导致熵增和混乱）与长期满足（导致熵减和秩序）。面对算法和数字环境的重重包围，解决方案不在于彻底逃离，而在于主动重塑。通过精心筛选数字环境输入、专注于有意义的个人项目、并利用技能创造价值，我们可以将社交媒体从消耗性的时间黑洞，转变为实现个人成长、建立事业和赢得自由的强大工具。最终，赢得对生活的控制权，取决于我们自己的选择和行动。
