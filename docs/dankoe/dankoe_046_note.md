# 生产力提升：第 1 课：终结拖延的 3 习惯晨间程序

在本节课中，我们将学习一个由三个简单习惯构成的晨间程序。这个程序旨在减少起床的阻力，为一天注入明确的目标感，从而有效对抗拖延，并建立持续一整天的积极能量。

我尝试过各种晨间程序。我曾尝试缓慢起床、冥想、阅读三十分钟以及其他所有类似的方法。但我总感觉自己落后了。我会对等待完成的任务感到紧张和焦虑。这种负面能量会持续一整天，让每一天都感觉不对劲。

经过多年的实践，我发现只需要三个简单的习惯。有三件明确的事情要做，能显著减少早晨起床的摩擦。一旦我在早晨确立了意图和目标，一切就变得大不相同。

## 3 习惯晨间程序

早晨是神圣的，它为一天剩下的时间定下了基调。你的晨间程序应该以产生能量为目标。我们希望这种能量能够持续一整天。

### 去散步

我们都知道身体跟随心灵，但心灵也可以跟随身体。它们相互影响。如果你想为接下来的时间产生不可阻挡的动力，就从活动身体开始。更好的做法是，到户外去活动。

> “清晨散步是一天中的祝福。” —— 亨利·大卫·梭罗

亨利确实有所发现。你不仅完成了一部分日常步数，还让皮肤直接暴露在阳光下。持续这样做，对维持健康的昼夜节律至关重要。这甚至可以帮助你建立更规律的睡眠模式——也许你根本不是天生的夜猫子！

你可以散步去咖啡馆，在公寓楼周围走走，或者去任何你喜欢的地方。在前一天晚上，将散步计划写下来，这样早晨就不会忘记。

### 10 分钟创意练习

接下来，让我们让动力的雪球越滚越大，让“今天算是废了”的想法变得不可能。这是我最喜欢的晨间习惯，已经培养了多年。它是日记、笔记和写作的结合。这有助于你更快地学习，锻炼创造力，并获得巨大的心理清晰度。如果你是内容创作者，这还能为你提供源源不断的工作灵感。

以下是具体步骤：

**步骤 1：获取灵感**

阅读一篇文章、一个博客帖子、一句喜欢的引言或一页书。你也可以在早晨散步时听有声书！注意：这些内容应与你的兴趣相符。这将为你接下来的时间提供灵感。

**步骤 2：记录兴奋点**

多巴胺帮助我们将信号与噪音区分开。在阅读或倾听时，写下那些吸引你注意并让你感到有点兴奋的内容。我通常在手机上的笔记应用中记录这些内容。

**步骤 3：撰写 3-5 点摘要**

你的总结应该包含什么？以下是建议：

+   你对这个主题的理解描述。
+   相关的个人经验和故事。
+   由此联想到的其他想法。

**给创作者的额外步骤：**

+   列出与该主题相关的一个常见问题。
+   列出解决该问题的好处。
+   列出克服它的步骤。

现在，你已经有了一个博客文章、社交媒体帖子、通讯或多条推文的提纲。这篇文章就是这样写成的。

**步骤 4：一句话概括核心思想**

这是发挥创意的部分。从你之前写下的所有内容中，提炼出一句总结性的话。发挥创意，尽量写出与你最喜欢的引言质量相当的句子。

我们为什么要这样做？

+   它能激发创意源泉。
+   它让你的大脑为模式识别做好准备。
+   它会向你证明，你能够进行原创且深刻的思考。

随着你写下越来越多的核心思想，这一点会变得越来越明显。

### 力量一小时

到目前为止，我们已经花了大约20分钟进行高效晨间活动。我们的身体和心灵都已准备好迎接挑战，进入心流状态。接下来是我喜欢称之为“力量一小时”的环节。

> “幸福是力量增强的感觉——阻力正在被克服。” —— 尼采

力量一小时是一个长达1小时的专注时间块，专门用于“建设”。没有干扰，只有一个优先事项，保持激光般的专注。可以把它想象成每天自动提升1%。

你在建设什么？个人自由。

+   经营你的副业。
+   写一篇文章、帖子或书。
+   创建社交媒体内容。
+   学习一项可销售的技能。
+   联系潜在客户。

你知道的，就是那些你总说要开始但还没做的事……

我怎么知道该做什么？以下是方法：

+   列出你对未来的愿景。
+   反向推导出10年目标。
+   然后设定1年目标。
+   接着设定月度目标。

现在，让你的力量一小时任务与这个月度目标对齐。那些能推动杠杆、关注基础、带来每日1%自动提升的行动，正是关键。这就是我创建[The Power Planner](https://shop.thedankoe.com/planer)的原因。

结束力量一小时后，你会对未来感到积极。在1小时内，你就在大多数人醒来之前赢得了这一天。这种感觉是无与伦比的。

**总结：**

本节课中，我们一起学习了终结拖延的3习惯晨间程序：

1.  **去散步**：活动身体并接受阳光直射。
2.  **10分钟创意练习**：训练你的创意肌肉。
3.  **力量一小时**：用1小时为自己构建更美好的未来。

如果持续实践，6个月后你回顾自己的进步，将会感到无比自豪。
