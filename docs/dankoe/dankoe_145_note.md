# 社会资本：新富的货币

> [`thedankoe.com/letters/social-capital-the-currency-of-the-new-rich/`](https://thedankoe.com/letters/social-capital-the-currency-of-the-new-rich/)

在本节课中，我们将要学习一种全新的财富形式——社会资本。我们将探讨它如何取代传统资金，成为新时代创造者、企业家和自由职业者实现个人与商业目标的关键杠杆。课程将详细拆解社会资本的核心构成、其重要性，并提供一套从零开始构建社会资本的具体方法。

## 社会资本：什么是社会资本？

许多人曾认为创办企业遥不可及。童年时期，我们接触的信息有限，往往只能模仿父母的行为模式。这导致我们逐渐形成了一些限制性信念，认为自己无法获得更多自由、金钱和满足感。

我们都有过那些从未付诸实践的想法。例如，“我们应该开发一个像Tinder但针对餐馆的应用程序”。然而，阻碍我们行动的原因似乎总是：没有资金、缺乏知识和技能、以及害怕处理复杂的商业事务。

我曾经也深陷这种思维。直到后来我才明白，启动事业真正需要的往往不是启动资金，而是**社会资本**。

那么，什么是社会资本？简单来说，**社会资本就是在那些愿意为你提供价值的人群中拥有的良好声誉**。

拥有了社会资本，你就拥有了杠杆。你可以成为一名成功的音乐家而无需唱片公司，成为一名成功的作家而无需出版商，或者成为一名成功的自由职业者而无需雇主。你可以启动任何业务，并拥有使其成功的技能和潜在客户。你不再依赖外部资金或投资，可以快速组建团队或网络。

更进一步看，许多创作者仅凭其个人品牌就能获得大公司的股权。例如，Marques Brownlee成为Ridge钱包的创意合作伙伴。初创公司也常向创作者提供股权，以换取其受众和快速成长，而无需付费广告。

因此，社会资本是下一代的货币，而非风险投资。虽然注意力常被称为新货币，但仅有注意力可能导致浅薄的生活。社会资本则能同时满足目标、利润与内心平和，因为它建立在你为世界提供真实价值的基础之上。

## 社会资本：2：社会资本的积极面与网络中的三种人

上一节我们介绍了社会资本的概念，本节中我们来看看它在数字时代的表现，并区分网络上的不同角色。

Netflix剧集《黑镜》中有一集名为《急转直下》，描绘了一个基于社会信用评分体系的社会。每个人的价值由他人评定的1-5星决定，这直接影响其生活、工作和社交。这听起来很极端，但它与当前的创作者经济有相似之处。

在网络上，我们大致可以区分三种人：

第一种是**影响者/名人**。他们拥有大量关注，能获得品牌合作和特殊待遇，是强大的营销渠道。

第二种是**消费者**。他们上网主要是为了消费内容——接受教育、娱乐或感到愤怒，而非创造价值。

但还有第三种人：**创作者**，或称**价值创造者**。

为了解释方便，我们将创作者定义为专注于创造价值的人。他们不沉迷于消费，也不仅仅追求虚荣的点赞数。他们关心的是影响力而非点赞数，是深度而非外观，是真相而非从众。他们通过自己创造的文字、产品、解决方案或代码来建立自身价值。

然而，创造价值需要一个前提：观众。如果一棵树在森林中倒下而无人听见，它是否发出了声音？同样，如果一个人创造了产品却无人知晓，它就无法产生影响力或收益。用公式表示就是：**1（你的创造）X 0（观众）= 0（影响力/收入）**。

那么，创作者拥有什么其他人没有的呢？答案是：构建在真实价值之上的社会资本。

## 社会资本：3：构建伦理社会资本的三大支柱

上一节我们区分了网络上的不同角色，本节我们将深入探讨，作为价值创造者，应如何正确构建社会资本。

社会资本不仅仅是积累粉丝。任何人都可以制作病毒式传播但无意义的短视频来吸引大众，但这除了短暂娱乐外，对人类贡献甚微。

对于作家、设计师、程序员或任何希望以热爱之事为生的人来说，必须建立坚实的社会资本。其核心公式是：

**社会资本 = 分销 + 声誉 + 价值**

以下是正确构建这三大支柱的方法：

### 分销

分销是指让你的作品被人看到的能力。主要有三种类型：
*   **自建分销**：例如建立粉丝群或邮件列表。
*   **借用分销**：例如他人引用你的书或分享你的帖子。
*   **购买分销**：例如投放付费广告或赞助邮件列表。

许多人无法从工作中获得收入，正是因为他们做了除吸引客户之外的一切。他们开发了应用却找不到用户，写了书却找不到读者。

专注于**分销**，是让你的创作触及他人的唯一途径，也是检验其价值的唯一方法。如果无人欣赏你的作品，这可能意味着你的技能尚未达到预期水平。这并非寻求外部认可，而是为了获得收入，从而能全职投入你的激情，否则生活压力会迫使你放弃。

每个人都应该从**自建**分销渠道开始。社交媒体和写作对任何有网络连接的人来说都是免费且可及的。只有在利用自建渠道获得收入后，才应考虑**购买**分销。

### 声誉

你可以基于外表或娱乐性建立庞大的观众群，但这不会持久。你吸引的是一群追求廉价多巴胺刺激的观众，他们看重你的外表而非思想，视你为可随时替换的商品。

价值创造者则专注于深度、创造力和解决复杂问题。他们通过传播清晰、有益的见解来获得长期满足，并用他们的**思想**赚钱，而非时间、外貌或体力劳动。思想是无限的，而其他资源则会随时间消逝。

没有声誉的分销会让你陷入更糟的境地：拥有庞大但肤浅的观众群，他们并不真正关心你的核心工作，也无法在未来为你提供支持。

### 价值

你可以整天写作、设计、创造，但这并不意味着它有价值或值得付费。如果无法获得报酬，你将很难在保持独立的同时维持你的激情。

价值的公式是：**价值 = (人们对你所做之事的关心程度) X (你所解决问题的规模)**

价值创造者不仅仅为了创造而创造。他们致力于解决真实、具体的人类问题，例如健康、财富、关系和幸福领域的障碍。他们从个人层面推动改进和进化。

你可以拥有分销和声誉，但如果不解决有价值的问题，你就是一个可替代的商品。不久之后，你会质疑工作的意义，追逐快钱，最终可能牺牲长期的事业，回到起点。

## 社会资本：4：如何从零开始建立社会资本

前面我们详细阐述了社会资本的三大支柱，本节我们将提供一套从零开始构建社会资本的具体行动指南。

首先需要明确：大多数人并没有足够的金钱作为捷径。他们看不到通往理想生活的道路，认为必须按部就班地工作、存钱、投资。结果，梦想被责任和消费文化埋葬，陷入早起压力工作、回家疲惫不堪的循环。

根本误解在于世界已经改变。我们不再处于工业时代，那时唯一的选择是上学、找工作。金钱不再是唯一的杠杆形式。

**社会资本本身就可以作为杠杆，帮你赚取想要的财富。而建立社会资本的核心，就是建立受众。**

以下是具体步骤及其重要性：

### 1) 学习一套相关且面向未来的技能组合

在建立受众的过程中，你会被迫学习一套对几乎任何人都有益的通用技能：
*   写作
*   说服
*   产品开发
*   市场营销与销售
*   平面设计
*   品牌建设
*   流量与转化

你不会在孤岛中学习这些技能，而是在现实世界中边做边学。市场会直接告诉你什么有价值。你无需许可即可开始，只需在网上建立个人品牌：创建资料、撰写内容、构建并销售产品或服务。

一旦你为自己取得了成果，就可以向他人或企业提供服务来获得收入。最棒的是，届时你已经拥有一个愿意为你付费的受众。此外，你可以围绕任何你感兴趣的技能（如哲学、健身、编程）来建立受众，不一定非得是商业或营销。

### 2) 从小处、易处、免费（或低成本）开始

上述步骤可能让人望而生畏，但关键在于聪明地启动。

以下是推荐的做法：
*   **从写作开始**：你不需要先学习视频或设计。写作能增强任何形式内容的核心。
*   **从一个短内容平台开始**：如X（原Twitter）、LinkedIn或Threads。你需要一个能被**发现**和**关注**的平台。
*   **从一个长内容平台开始**：推荐邮件通讯。这能让你保留一部分受众，并且内容可以转化为其他形式（如视频脚本）。

这三个步骤能强制你（1）快速获得反馈和结果，（2）学习真正重要的技能。如果你想写出吸引人的内容，就必须学习说服；如果你想建立邮件列表，就必须了解基本的营销。你不需要正式学习，只需尝试、失败、改进。

### 3) 建立声誉，传递价值

许多人“等待时机成熟再销售”，认为受众讨厌被推销。但我个人发展中最重要的一步，就是从一开始就尝试销售产品。

这给了我行动的动力和明确的方向。我知道，如果无法获得收入，就意味着我提供的价值尚未被市场认可。免费内容、写作或设计本身不足以深刻影响他人生活。真正的价值交换发生在他人愿意投资于能解决其问题的产品时。

回顾过去五年，我每个季度都会推出一个产品或服务（电子书、课程、咨询等）。我将所有的社会成长和 perceived success 归功于此。正是通过用我认为有影响力的产品或服务换取金钱，我才得以建立声誉、传递价值，并真正为互联网的价值生态做出贡献。

如果你对金钱抱有恐惧或偏见，认为谈钱不高尚，那么你将很难迈出这一步。金钱有多重视角，其中许多是极其积极的。

## 总结

本节课中，我们一起学习了社会资本这一新时代的关键货币。我们了解到，社会资本是建立在良好声誉、有效分销和真实价值基础上的杠杆，能帮助创作者和企业家摆脱对传统资金的依赖。

我们探讨了社会资本的核心公式 **社会资本 = 分销 + 声誉 + 价值**，并详细拆解了每个支柱的构建方法。最后，我们提供了一套从零开始的行动指南：学习未来技能组合、从小处低成本启动、并通过提供有价值的产品或服务来建立声誉。

建立社会资本，就是加入“新富”行列，用你的思想和创造的价值，换取自由、影响力与财富。
