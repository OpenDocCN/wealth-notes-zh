

# 决策层喜气洋洋，中美破冰还差诚意

本周国际会议活动非常集中。24日中国发展高层论坛2025年年会开幕，27日博鳌亚洲论坛2025年年会开幕，同日2025中关村论坛年会开幕，除了这些固定的大型交流活动，还有相对民间的活动，如27日中国公共外交协会举办的经济专题“临甲7号”沙龙活动。

为什么说决策层喜气洋洋呢？

在上述活动中，小镇观察到一个词出现频率大增，自最高决策层往下，不同领域不同人在各个场合纷纷使用“蔚然成风”这个词来描述中国的创新发展，如此普遍的集中使用，说明这个词一定来自于最高领导人，也确实如此。

“蔚然成风”意思是事物盛极一时，成为一种良好的风气。

2012年以前，在高层文件、讲话中极少出现，最近15年仅2011年政府工作报告曾出现“全民健身活动蔚然成风”，之后几年也是零零散散，比如2013年农村会议提到“让节约粮食在全社会蔚然成风”，2014年在北师大考察提到“让尊师重教蔚然成风”，之后还用在创新、调查研究等方面。但2017年之后就没有再使用这个词，直到2021年，在全国脱贫攻坚总结表彰大会上的讲话中，再次使用“蔚然成风”肯定过去五年脱贫攻坚的成就。

由此可见，“蔚然成风”在最高层用语中属于很高级别的肯定用词，一般情况根本不会使用。

小镇大概梳理了下，仅限前两号首长，讲话、文件、考察调研等各种情况都算上，从2013年到2017年五年间，一共也只出现了6次；但从2021年开始，出现频率越来越高，2022年到2024年三年间出现了11次。

尤其需要注意的是，从2023年9月开始，一号首长经常使用“蔚然成风”这个词，而且不像过去，分散指向不同领域，比如健身、法律、担当、学雷锋、妇幼、调查研究等，自2023年9月发表署名文章《推进中国式现代化需要处理好若干重大关系》之后，“蔚然成风”仅指向创新。

在《若干重大关系》一文中，提到“积极营造崇尚创新、鼓励创新、勇于创新的浓厚氛围，让创新在全社会蔚然成风”，这句话后来一再使用，包括2024年6月的科技大会讲话、11月署名文章《必须坚持守正创新》、12月署名文章《以中国式现代化全面推进强国建设、民族复兴伟业》。

而到了2025年，更多高层开始使用这个词，尤其这周。24日中国发展高层论坛开幕式李总致辞提到“绿色家电、新能源产业等绿色经济蔚然成风，展现了转型发展的巨大空间”，关注新兴产业这也是李总自上任以来一贯的风格；27日博鳌亚洲论坛开幕式，常务副总在致辞提到“发展新质生产力、推动科技创新蔚然成风”。

仅从这个词的使用上，就能感受到高层的信心相比前几年增强了许多。孤例不证，还可以观察3月28日高层论坛结束之后会见国际工商代表的讲话，一共谈了四点：
- 肯定外资企业作用
- 中国长期成为外资投资兴业的沃土
- 解决外资在华发展问题
- 为维护世界经济秩序共同努力

注意，这次没有像2024年那样单独会见美国工商界和战略学术界代表，讲话也更加以我为主、更加自信，比如讲话中有一句“阻挡别人的路，最终只会把自己的路堵住”，对外资更多是提要求，尤其要求外资要与中国一起共同维护世界经济秩序。

以上可以交叉验证决策层喜气洋洋、充满信心的判断。而这就关系到中美关系破冰进展。

美国一直在炒作中美元首会晤，先是说最早4月，然后又炒作6月生日峰会，但目前看别说4月了，6月的概率都很小。毕竟特朗普4月2日会拿出一揽子的对等关税政策，还有更多针对中国的政策有待实施，在这些手段没有落地且被验证无效以前，特朗普不会放弃极限施压，而中国也不可能在充满自信的时候，向美国妥协。

现在无非就是拼定力和实力。对于美国的沟通访问，见归见，但冷处理，毕竟美国人也不诚心。

比如这次戴恩斯访华，在访华前他还在芬太尼问题上公开对中国施压。戴恩斯访华时，根据公开报道，只单独见到了副外长，与何副总的会见，也是安排与淡马锡董事长一起，更高层级也仅仅是高层论坛结束后进行了集体会见。

这显然是在对戴恩斯访华进行冷处理，并没有如一些人期待的中国赶紧与美国缓和关系。现在情况很清晰了，决不能跟着特朗普的节奏走，想来访华那就来，但如果没有诚意，那也不过就是把礼数尽到，毕竟中美之间沟通总比没有沟通要好。

目前外媒在报道说1972年美国参议员访华，是中美破冰的开始，但是呢，破冰也是需要一个比较长的时间，还取决于双方是否达成共识、是否有诚意。懒人微信：lazyhelper

从最近看，中国的反制在精准中持续加码，过去一般不针对在华投资的外企，但最近打的就是美国外资。意思很明确，就如讲话中强调的“外资企业特别是跨国公司在维护世界经济秩序中发挥着重要作用，也承担着重要责任”，也就是说美国跨国公司们，就算不能阻止特朗普乱来，但起码不要助纣为虐、推波助澜，否则就是没有尽到责任，那生意也就别做了。

上一轮贸易战的敲打已经有效果了。当时2019年6月对联邦快递发起立案调查，而这次高层论坛召开前，联邦快递CEO芮思博拜会了沪粤两地政府及海关高层，而且是北京广东两地飞，在高层论坛上也是首个发言。

目前芮思博自2024年6月起还担任美中贸易全国委员会董事会主席。
