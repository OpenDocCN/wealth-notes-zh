

# 315晚会，为什么解决不了问题？

3月15日，一年一度的“打假春晚”315晚会再次开幕。

其实吧，哪怕不看晚会，我们也能大概猜出晚会曝光了哪些领域，然后黑心企业排队道歉，网友刷屏痛骂，热搜榜上挂满“塌房”品牌的名字，无数自媒体蹭热点的文章视频集中爆发，整个互联网都要热闹大概一周左右。

但问题在于，大家有没有感觉，315晚会，似乎越来越“不管用”了？

食品安全问题说了多少年了？可是总是还能死灰复燃。厂家的套路并不新鲜，但一阵喊打之后，就会再次出现。

那么问题来了，315晚会，为什么解决不了问题？

## 1. 315晚会的作用为何减退？

首先，我们要简单科普一下315晚会的历史。

315晚会始于1991年，由中央电视台、中国消费者报社、中华工商时报社与中国消费者协会联合举办。那一年的315晚会，直击消费者痛点，被曝光的假酒、劣质热水器，被打爆的热线电话，连夜成立的专案组，让315晚会成为那个中国消费者权益保护草莽年代里唯一的正道之光。

不得不说，315晚会是有其时代背景的。

90年代，中国正面临经济转型的阵痛，乱象频出，而市场监管发展又滞后，315晚会承担着“替代性监管”的使命。

但现在呢？时代变了，315晚会的作用，已经大大减退了。

一方面，投诉的渠道增多了，社交媒体异常发达，大家发现哪里不爽，直接抖音、小红书一发，运气好就能上热搜。

比如雪乡宰客、“职业闭店人”、演唱会“柱子票”、自动续费服务暗藏玄机、平台大数据“杀熟”、羽绒制品“以丝代绒”、“提灯定损”、“串串房”甲醛超标、“星期宠”等等，都是通过网络曝光，然后引起监管部门关注的。

说实话，过去大家是没得选，只能依赖315，而现在，随着各类投诉平台、社交网络的兴起，能够快速、有效的方式解决消费投诉问题，不用苦苦等待一年一度的“打假春晚”了。懒人微信：lazyhelper

另一方面，黑心商家应对315晚会，也越来越有经验了。

今年315晚会，有个很有意思的情况，晚会曝光了维修刺客啄木鸟，但晚会还没结束，啄木鸟的道歉信已经发出来了。

啄木鸟到底是找了品牌部的员工连夜写出来的？还是事先就准备好了？

大概率还是事先有准备。

事实上，在圈子里，如何应对315晚会，早已驾轻就熟而且已经有了成熟方案了。

对大企业来说，品牌部要么建立舆情监测系统，要么向第三方购买舆情监测服务，对舆情进行7×24小时的全网信息的采集和分类，可以对敏感重点信息进行“预警”，更早地发现负面，毕竟官方也是通过网络采集信息的，如果企业运气好，没准还能雇佣到和官方同一个数据源的舆情监测服务供应商。

得到预警后，就要进入到下一个阶段，评判曝光会不会对企业造成影响？会造成多大的影响？要不要采取行动？

一般来说，如果违规操作的收益足够大，那这些大企业就会想办法让官媒放弃曝光。

这个时候，就看企业的各显神通了，要么以纳税、就业的作用让地方政府部门劝说，要么以投标广告的方式“暗示”，要么就让一些公关公司出马搞定相关记者。不要低估那些公关公司的能量，绝对超过你的想象。

搞不定也没关系，那就提前准备好道歉信。

道歉信都是有模板的，也就是“三不”+“三表态”：

“三不”：不愤怒和指责，遮掩、避讳问题，不试图对抗媒体和大众，不用强硬和傲慢等方式表态，否则只会起到反效果。

“三表态”：真诚道歉、甩锅供应商、自己先赔付以彰显自己负责态度云云。

道歉信一般在舆情爆发4小时内先做第一次回应，表达自身态度，缓和舆论情绪，然后待2天舆论高峰期，连同解决方案一起再次发布回应，显示自身诚意，基本上就能把舆论风波消化无形。

那对于中小企业呢？可能没这个实力，但也有自己的应对手段。

比如，提前三个月启动“风险预检”，一些不合规的程序要尽量开始合规，尽可能消灭证据，晚会前一周物流系统设置“风险品临时仓”，将可疑产品转移至监管薄弱区域，晚会当天全员进入“静默模式”，等待舆情风向标。及时止损。如果曝光自己，也能把损失降到最小。

如果懂点传播的中小企业，还可以在发布道歉声明的同时，雇佣网络水军批量制造“良心企业”话题，或者通过科普短视频灌输“合规添加剂无害论”，将公众注意力转移至次要矛盾。

至于监管部门的调查和处罚，就更好应付了，无非也就罚点钱，只要保住人、保住厂，事后风头过去了，换个名字，生意照样做，甚至一个企业顶了雷，其他同类企业都安全了。

这其实反映了一个很严重的问题，315晚会只有“前哨”，没有“后效”啊。

说到底，晚会只是媒体监督，主办单位只有监督权，而没有执法权，结果就是曝光得热热闹闹，但问题整改了没有？整改的怎么样？315晚会不知道，也不追踪后效。

大家能看到的是，相关部门在曝光后第一时间赶到现场，然后查封、下架、处罚等等，看起来雷厉风行，但大众需要的是什么？不是“运动式治理”，而是“制度性防御”，这一点做了没有，谁也不知道。

咱们不能说这是一种“表演性处理”，但起码，消费者并没有真正告别“年年打假年年有假”的魔咒。

这里面，有复杂的原因。

一方面，有监管部门的力量不足问题。有的地方，小商品店铺、小饭馆、小厂多如牛毛，监管部门再监督，能24小时全盯着么？这不现实，只能抽检，但只要抽检，就一定有漏网之鱼，这条鱼被抓到了，就是大新闻。

所以，监管部门也很头疼：你们能不能搞一个线索通报机制啊。

另一方面，央媒和地方的出发点不一样。央媒需要的是“为民请命”的荣耀和流量，而地方，要的是税收和就业。在当前经济形势下，央媒可以不留情面地揭盖子，因为民生不归他们管，但地方，也是有成千上万的家庭要生活的。

还记得2011年，武安市被曝钢铁减产不力，“一问三不知”的马科长播出后沦为全国笑柄。但笑柄的背后，是马科长用装疯卖傻，保住了十几万钢铁工人的饭碗，事后还被提拔了。

相比于被曝光，各个地方更怕的是民生困境，所以在处罚相关曝光企业时，一般也都不会往死里整，而是留条活路，反正报道也不看后效，差不多得了。

利益出发点不一样，做事风格当然也就不一样。

315晚会真正应该做的是什么？曝光后，应该行使舆论监督权，推动有关部委立法，把舆论监督成果固化为制度，形成一条法规红线，这样，一方面不法商家在蠢蠢欲动之前，会考虑一下违法成本，如果成本过高就会知难而退。另一方面，也倒逼地方监管部门依法行事，不能再从轻处理。

可惜，现在这种机制并没有建立起来，结果就是315年年曝光，但换来的仅仅是一两天的热搜而已，就像一场漂亮的烟花，烟花闪耀过后，仍是一片暗淡。

## 2. 315晚会的局限与改革方向

比如房地产质量问题、烂尾楼，社会的怨气可比什么老坛酸菜要大多了，可是315曝光过么？没有啊。

现在房地产市场都成什么样了？非常需要全社会的信心，你再把矛头指向房地产，引发出来的社会问题谁来负责？

外资企业，也要尽量少曝光，毕竟现在美国正嚷嚷脱钩呢，中国现在需要的是稳外资，你把外资企业得罪了，他们撤出中国，谁来负责？

还有，一些民营大企业也尽量少曝光，毕竟民营企业老总很多都有工商联啊、政协之类的政治身份，惹了比较麻烦。

所以看来看去，还能曝光什么呢？

我们回顾一下，近些年曝光的最多的案例是什么就知道了——食品安全。

食品类企业多以民营小微企业为主，这些企业通常产值不高，对经济影响不大，其社会地位基本上也没有反抗能力，不用担心其反噬的问题，所以是315晚会最喜欢的开刀对象。

但问题在于，食品安全曝光那么多，对社会造成的恐慌情绪和不满情绪，谁来负责？

315晚会，对一些阴暗面的曝光尺度，拿捏得非常精准，既曝光现象，又点到为止，不做进一步深挖和扩大。

比如，315晚会曾经曝光过网络水军问题，这很难得。但问题在于，晚会却没有对网络水军背后的一些问题成因，提出疑问：

——海量身份证信息从何而来？
——电信运营商提供哪些手机号？如何绕过实名制？
——短视频平台运营商是否纵容水军？是否在享用水军的流量？

315晚会只指出了网络现象，却对背后的问题不愿提及，那么到底是水平问题，还是胆量问题？怕拔出萝卜带出泥？

- 第三，取证不严谨。

按道理来说，国家级媒体，应该严谨细致，每一项指控和取证都能经得住质疑和检验。

但问题在于，这几年的315晚会，频频出现曝光后却被质疑和打脸的情况。

比如，315晚会曝光无印良品食品来自日本福岛核辐射区，搞得人心惶惶，但随后就被上海国检局打脸，晚会是把公司注册地当成了食品产地，实际上的产地并不在国家限制进口区域内，闹了个大乌龙。

说实话，这种低级错误不应该出现的，这会极大损害晚会曝光的权威性。

- 第四，捂线索。

2022年，315晚会曝光部分老坛酸菜包防腐剂超标，腌渍过程中还用脚踩，在全国引发轩然大波。

可是媒体得到线索之后，并没有及时通知市场监管部门查处，而是自己调查，捂了几个月，一直等到315晚会才曝光。

那么问题来了，到底是晚会的流量重要，还是全国人民的健康重要？在线索被捂的这几个月内，全国有多少人吃了拿脚踩出来、防腐剂超标的老坛酸菜？

很多人评论，曾经自带使命的晚会，越来越像是一场为了流量及热点的表演秀了。

更关键的在于，晚会主办方本身也是一个盈利机构，是接广告的。既然要拿企业的钱，同时又担任着曝光企业的责任，能否保证自身曝光的公平和正义？会不会出现企业投了广告，就能拿到“免曝光金牌”的现象呢？

所以，当时代发生变化，而且晚会本身也存在各种问题的时候，也会有越来越多的商家去钻空子、逃避监督。结果，就变成了年年315，年年有问题的情况了。

也许，已经搞了将近40年的315晚会，真的该改革了。
