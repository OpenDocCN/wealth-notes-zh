

# 宠物行业，正在前所未有的“双重老龄化”

今天我们要说一个正在快速增长的机会：宠物行业。提起宠物行业，很多人可能会觉得这个话题有点老套。好像聊来聊去，不是狗窝就是猫粮。这个行业还能有什么新东西吗？

别着急，先问你个问题。你发现没有，现在市面上的宠物狗，腿好像越来越短了？除了有的品种本身就短腿，其他的狗好像腿也越来越短。你想过没有，为什么会这样呢？

再问你一个问题，你有没有发现，现在有很多宠物猫，变得越来越“狗里狗气”的？猫一贯带点野性，但最近几年很多宠物猫变得越来越温顺，性格越来越像狗。这又是为什么呢？

其实，前面两个现象都不是巧合，而是宠物行业刻意筛选培育的结果。没错，这一切都是人为干预的。为什么要做这样的改造？因为现在国内的宠物行业，正在发生一个非常关键的变化，叫“双重老龄化”。也就是，主人老龄化，宠物也在老龄化。

你看，对老年人来说，狗跑得太快追不上怎么办？假如狗的腿能短一点点，老人遛起狗来就友好多了。还有宠物猫，性子太野对老人也不友好，假如能把它们变得更温顺一点就好多了。

你看，这些趋势假如不明确指出，是不是很容易被忽视？

其实，宠物行业的发展可以分成三个阶段。

阶段一，是双重年轻化阶段。也就是，主人年轻，宠物也年轻。早几十年咱们国内就是这个状态。这个阶段催生了大量的狗粮猫粮需求，也造就了一波宠物创业热。但生命总是会衰老的，因此就会进入第二阶段。

阶段二，宠物单方面老龄化。宠物猫狗的寿命远比人类短，因此在主人老去之前，宠物会先老去。这也造就了宠物行业的一轮新需求，比如宠物医疗、宠物体检、宠物殡葬等等。

阶段三，是双重老龄化。也就是宠物和主人都老去。这个阶段是伴随着整个社会的老龄化同步发生的。

当然，前面三个阶段并不是整体的循序渐进，而是交替叠加。可能有的区域进入阶段三，有的地方同时叠加阶段二阶段三，等等。但总归，宠物行业从来都不是一潭死水，它每时每刻都在发生新变化。

那么，其中有哪些值得留意的趋势？又有哪些新机会呢？

今天，我们就请到了著名零售咨询顾问、消费行业专家，黄碧云老师。前段时间，黄碧云老师前往日本，专门对日本的宠物行业做了调研。我们都知道，日本是最早进入老龄化的社会之一，宠物行业也相对成熟，或许有一定的参考意义。同时，在调研过程中黄碧云老师也请教了国内的宠物媒体运营专家，宠业家的联合创始人雕兄，以及长居在日本的得到用户王淅老师。

接下来，我们就说说几位老师带来的观察。咱们主要说3个要点。

## 第一，人的居住空间变了，宠物行业也要跟着变。

首先，宠物行业有一个关键的消费群体，高校毕业生。很多年轻人毕业两到三年后，会觉得结婚尚早，但是独居又太孤单，他们就会开始养宠物。但这时，年轻人往往还没有买房，很可能是租房，这就导致宠物行业出现几个需求。

比如，共养宠物空间的需求，类似于宠物的幼儿园。毕竟，万一合租的人不想养宠物，那么把猫狗放在共养空间就是最好的选择。再比如，中国在养宠方面和日本有一个相似点，就是人宠共居空间较小。因此养大型犬就比较麻烦，这就进一步倒逼国内小型犬以及猫经济的发展。

再比如，人宠共生的家居改造服务。在日本的很多家装店里，都能看到为人宠共生设计的定制家具。有的是把柜子的底层做成狗窝，也有的是在桌子下面整合猫的穿梭架，还有直接把桌子腿做成猫抓板材质的。

换句话说，由于居住空间方面的限制，宠物产品的研发方向，不是宠物至上，而是人宠共生。

同时，还有一个值得注意的现象，现在很多年轻人，都属于养宠二代。也就是，他们的父母就养宠物，他们自己从小就生活在有宠物的环境里。大量的“00后”“10后”，都属于养宠二代。当这部分人开始赚钱独立生活，将会给宠物行业带来巨大的增量。

## 第二，针对人宠关系的情绪服务。

比如，给电子宠物办葬礼。没错，是电子宠物。在日本，因为老年人的体力精力有限，因此这几年就有很多人养机器宠物。像索尼的爱棒 Aibo 机器宠物狗就很受欢迎。机器狗这个产品本身并不新，但给机器狗办葬礼就很新鲜了。在 2015 年时，甚至还出现过机器狗的集体葬礼。当时爱棒有一批旧型号故障，因为已经停产，修不了，于是，主人们就给这批机器狗办了集体葬礼。地点在千叶县的光福寺，寺院的住持还严肃地念了超度经文。情绪价值相当到位。

再比如，制作宠物避难攻略。什么意思？日本是多地震国家，关于地震逃生的防灾减灾日历有很多。但是，这些都是针对人类，却没有针对宠物的。于是，有个民间爱宠协会就针对宠物做了防灾日历。上面附上专门的人宠避难点地址，还有宠物避难的要点。

这个宠物避难攻略有没有用另说，但至少情绪价值是给到了。宠物主人会觉得自己很对得起宠物，说白了，就是给了他们“爱他人”的感觉。而且注意，这个减灾日历非常适合定制化。宠物用品公司可以定制，作为赠品一起出售。

目前咱们国内也有不少有趣的尝试。比如，闲鱼上有个副业，叫“猫狗绝育坏人表演”。你看，主人带着宠物绝育，宠物可能会记仇吧？但是由别人带着去呢？宠物就会记别人的仇。更进一步说，假如在绝育结束时，主人突然出现，像英雄救美一样把宠物给带走，宠物是不是不仅不记仇，还会感激主人？

没错，所谓坏人表演，说白了就是替主人背锅，带着宠物做绝育。这个服务在闲鱼上的报价是单次50到100元，同时提供全程录像。

闲鱼上类似的服务还有很多，比如，宠物媒人、上门代遛狗、上门喂养宠物、负责找猫的猫咪侦探，等等。还有不少项目入选了2024福布斯中国副业100评选。

换句话说，人们养宠物的目的之一，是为了体验爱与被爱的感觉，而这方面的深度服务，将成为一个关键趋势。

## 第三，主人与宠物双重老龄化，带来的行业变化。

比如，针对宠物的“一老一小”服务。针对新生宠物，有宠物幼儿园。而随着宠物长大，还有配套的宠物保险。假如宠物去世，还有专门的宠物丧葬。在日本，一场宠物葬礼的费用大概在2万到15万日元之间，相当于900多到7000多元人民币之间，主要是根据宠物的体重浮动。

这主要是因为，日本有近80%的人住公寓，租房一族比例也不小，宠物死后不能随便埋葬，否则违法，加上社会道德方面的约束以及宠物主人的情感诉求，共同催生了日本宠物丧葬的需求。根据相关机构的调查，日本有66%的养宠人士会选择专业的宠物丧葬公司。目前宠物丧葬的市场规模在350亿日元左右，相当于17亿元人民币，并且还在增长中。

除了丧葬之外，最近日本还推出了老龄化宠物的终活服务。终活，就是为临终准备的相关活动，包括宠物的照护等等，以及假如主人老了，无力照顾宠物，还会有专门的法律服务人员替宠物寻找新监护人。

借用宠业家的联合创始人雕兄的话说，宠物是一个翻版的人类小世界，一个以毛孩子为主体的世界。因此，按照人的生活配置做宠物经济，是有很大的增长机会的。

而且别忘了，咱们中国的市场规模要大得多，前面任何一个趋势，在这里都会被放大很多倍。

好，前面说的，是关于宠物行业的机会。当然，你可能也发现了，宠物行业只是表象，这背后反映的是，人们对于精神生活的需求。

在日本的调研过程中，黄碧云老师也请教了《第四消费时代》的作者三浦展。三浦展说，日本目前正在到来的，将是第五消费时代。第五消费时代的本质是，人们在精神上经历高压之后，进入调节阶段。这时人们会更看重精神层面的幸福，而不是单纯的物质享受。一条个性化的街区、一间独特的咖啡馆、一间充满疗愈气息的小店，都是在回应这个需求。

当然，中国市场情况和日本有很多不同之处，包括市场规模，其中的结构化特征，等等。但在某种程度上，日本市场的趋势或许值得适当参考。在这里，也祝你能把握住其中的新机会。
