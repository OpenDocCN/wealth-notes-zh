

# 美国之音断粮停摆，背后真相是什么？

“美国之音”（VOA）电台哑巴了。

它是被特朗普给“饿”闭嘴的——3月14日，特朗普签署行政命令削减美国国际媒体署（USAGM）等六个联邦机构的资金开支至最低限度，美国国际媒体署就是“美国之音”的上级主管机构。

“美国之音”的1300多名员工被强制带薪休假，复工时间未知，总部大楼上锁，内容更新也停止。

“美国之音”自创办以来83年间，首次陷入了停摆状态，美国国际媒体署的其他外宣媒体“自由欧洲电台”“自由亚洲电台”等也是如此。

那么，特朗普为什么要收拾美国之音呢？

## I

VOA成立于1942年，最早是为抗击纳粹而成立的。二战结束后，它成为了冷战宣传工具，冷战后直到现在，又成为了对中国等几十个国家和地区搞意识形态宣传渗透的桥头堡。

### 断粮“美国之音”，是特朗普不想打对华舆论战了？

那是不可能的，关税战还打着呢，凭什么放弃舆论战？特朗普显然是想把外宣工具洗牌，再卷土重来。

话说特朗普就任之前，本来提名共和党人卡莉·莱克为“美国之音”新台长，不过由于特朗普提名的美国国际媒体署长官还在等候参议院同意权听证会，而“美国之音”台长也要由他任命，一时间特朗普还没法把莱克安排上去。

这是因为1999年克林顿在任期间，通过了一个《美国外交事务改革和重组法案》，将美国国际媒体署的前身美国广播理事会改成了独立机构，虽然领取政府预算，但组织相对独立。除了CEO本人由白宫任命，下辖媒体的人事任免都由CEO决定。

即使特朗普上个任期把它改名成美国国际媒体署，其独立性质仍然未变。

既然把自己人安排进“美国之音”有困难，干脆直接安排在它头上——2月28日，特朗普直接任命卡莉·莱克为USAGM特别顾问，帮助其“落实精简机构、其下辖网络及其拨款接受方所需的政策和策略。”

马斯克作为特朗普政府的“特别顾问”，地位有目共睹，卡莉·莱克也等于是特朗普在USAGM的代表。

她不当下属电台台长而是当母机构官员，表明特朗普想连USAGM本体一块翻个底朝天。

卡莉·莱克没让特朗普失望，真查出了一些黑料。

比如，USAGM斥资近2.5亿美元，在华盛顿宾夕法尼亚大道上租了一栋高层建筑，而它每年预算就接近10亿美元。全机构一季度预算的钱就租一栋大楼，怎么想都不正常，这栋大楼甚至没有广播设施用作电台播报，连办公楼都当不了。

花这么多钱租一栋无法办公的楼，猫腻能小的了？

USAGM还给了私人房地产经纪人900万美元佣金，背后怕是有什么肮脏交易，莱克宣布已取消了这栋大楼的15年租约。

贪污腐败见怪不怪，问题是，美国国际媒体署就像马斯克说的那样，每年烧掉纳税人10亿美元却不做正事。

首先，预算就这么10亿，史密斯专员拿多了，底下人的报销经费都成问题。

有这么个例子，“美国之音”为了搞英语教学，雇了个Jessica老师，中文名叫白洁（她微博还有号）。一开始说给18万，但预算不够了，于是台长好容易才把价格谈到14万，结果因为VOA内部扯皮，连14万都给不了，只能台长含泪垫钱。

人雇到了还没完，后面节目流量不够，还得找一堆人去刷点击率，这雇水军的钱VOA也出不起，只能找上级单位USAGM报销。

就这用人上的寒酸劲，怪不得“美国之音”这些媒体找不到啥人才。

更可笑的是，员工拿不到啥钱，美国国际媒体署却很舍得在买新闻上花钱。

美国国际媒体署和美联社、路透社和法新社有着5300万美元的合同，从后者那里花钱买新闻材料。莱克对此火冒三丈，有10亿美元预算，却连写新闻都写不明白，还花钱得跟别家买新闻，简直就是巨大草台班子。

买三大通讯社的新闻也就罢了，关键是，USAGM连假新闻它也买，甚至已经为此花了上亿美元——真新闻都不自己写，你指望这些职员会分辨新闻真假？

只能是眉毛胡子一把抓，能用就买来用。

这么个腐败无能的吞金机器，不给它按下待机键，实在对不起DOGE这座牌坊。

## 2

不过，美国国际媒体署和“美国之音”被断粮，原因不只是贪腐无能，

千万别把特朗普政府当成青天大老爷，还有政治恩怨在里面，不然也不会狠查它一波。

2020年4月，那时武汉才刚刚解封，“美国之音”也报道了此事。

结果，白宫连续两天针对“美国之音”公开批评，表示VOA简直是美国之耻，联邦政府每年拿将近2亿美元给VOA，是让它为美国人说话，而不是帮中国宣传。

VOA自然不服，时任台长阿曼达·贝内特称VOA是公费独立媒体，有法定自主权，就该全面报道问题，武汉已经控制住疫情解封了，咱不能装瞎不承认吧？

贝内特还拼命找补，说我们已经尽力在找角度黑中国了。

说了几万句中国的坏话，偶尔说句好话是为了平衡一下，假装中立。但在特朗普和白宫看来，忠诚不绝对，就是绝对不忠诚，VOA竟然吃饭砸锅，替中国人说话。

这是“美国之音”头一次得罪特朗普，特朗普意识到，这种独立媒体有点不好管，2021年卸任前的风波，更让他坚定了这个看法。

2021年1月11日，国务卿蓬佩奥在VOA总部演讲，称“美国之音”应该少报道美国负面新闻，多给美国正面宣传。这本来是特朗普下台前敲打VOA，没想到VOA白宫分社社长帕齐·维达库斯瓦拉哪壶不开提哪壶，直接拿之前国会骚乱事件提问，弄得蓬佩奥下不来台。

台长罗伯特·赖利当场怒斥维达库斯瓦拉：“这事该问吗？”

副台长罗宾斯也幸灾乐祸地对她说：“你完蛋了。”

蓬佩奥想训话“美国之音”，却被记者反将一军，当晚，维达库斯瓦拉就被降职处分。但是没过几天，她又官复原职，因为拜登上来了。

这下，轮到“识大体”的赖利和罗宾斯下岗走人了。

“美国之音”领导换成拜登的人，这几年反川言论不绝于口，特朗普也憋着口气：你们给我等着。

果然，特朗普2.0时代，VOA要被收拾了。

## 3

特朗普和“美国之音”的过节归根结底还是在利益和价值观冲突上，前者不满后者不仅不愿老实当自己喉舌，很多人还心向民主党，成为反川大本营；

而“美国之音”的精英记者们价值观上与MAGA分歧巨大，他们不愿为反移民、反DEI的特朗普服务。

前面说过，特朗普从“美国之音”的独立媒体身份中吸取了教训，第二次上任，他已经不再对它抱有什么“驯化”的希望，在宣传渠道上，彻底转向他喜爱的互联网自媒体。

众所周知，特朗普上个任期一直很喜欢“推特治国”。在被封之后，他还自己搞了个“真实社交”平台继续搞自媒体，如今马斯克把推特买来改成了X，他俩继续将自己的网红自媒体路线发扬光大，影响力水涨船高，全球吸粉无数。

反观“美国之音”作为传统广播媒体，虽然也试着紧跟电视、网络潮流，相继开启了卫视频道和网站，并在各大社交媒体建号运营，但效果实在不尽如人意。

这年头广播几乎没人听，电视看的人也少，影响力可谓日薄西山，“美国之音”网站和社交媒体的评论区也充满了冷嘲热讽，支持声音越发稀少。

原因一是互联网时代信息爆炸，大家不好糊弄了，在没有中间商的对账环节中，往往连一手消息都没有的“美国之音”编造、片面报道的新闻，轻易就被网友们戳穿，沦为笑柄。

二是“美国之音”等媒体真面目早已暴露，声名狼藉，被许多国家警惕甚至屏蔽，限制了宣传效果，甚至适得其反，就连它的英语听力练习节目，也因为负面宣传被许多外国学习者抛弃。

所以，特朗普、马斯克选择互联网自媒体可以说是水到渠成，在自己的社交网站搞宣传没成本，便于控评，宣传效果好，也不会引发他国警惕，能够拉拢更多年轻人。相比之下，独立性强难控制，且耗费大笔资金的传统媒体，效果差，名声臭，对特朗普他们来说，这种宣传工具是时候退休了。

美国的舆论战不会因为“美国之音”停摆而停止，它只是被更好的宣传工具替代，X和“真实社交”平台已经成为了特朗普政府的新武器，“鸟枪换炮”的美国舆论战，未来将更加隐蔽，更加危险。
