

# 古人为什么强调孝？

古代对孝的强调，本质是君王和庶民签的不平等条约。

现在把你拉回古代，给你换个身份。你现在是皇上了。

现在你面对着亿万苍生，你得叫他们给你的帝国服务吧。

黎民百姓也是听话，税银按时交了，公粮如期纳了，跟邻国掐架，他们脑袋别在裤腰带上战场了，大运河他们帮你挖了，长城他们给你凿了，甚至你还活蹦乱跳，他们连皇陵都帮你拾掇好了。

他们该给你做的，一件不落都给你办踏实了。

可办着办着，他们也老了，没劳动能力了，连自给自足都困难了。

现在问题就摆在你面前：

这群只能吃饭，不能干活的老庶民们，该作何处理？

按道理说，这些人把一辈子都奉献给了你和你帝国，现在应该是你回报他们的的时候了，起码一天三顿饭，你是应该管的。可你不管。

因为他们曾经给你奉献的钱粮，都叫你霍霍的差不多了，剩下点没霍霍的库存，你还打算留给你儿子霍霍呢。你可舍不得对子民搞反哺这一套。

是啊，吃到嘴的肉，谁舍得吐出去？

于是怎么处理这些耕不了地的老牛们，你开始动歪心眼子了。

历史上出现过挺多不人道的做法：

- 比方说六十还仓，人到六十岁，不问三七二十一，直接活埋。
- 比方说寄死窑，人上岁数了，挖个窑洞，给老头老太太搂里边，儿女每天去窑洞送一顿饭，顺便再垒一块砖，等砖头把窑洞封死，老人就在里边等饿死吧。
- 比方说上岁数之后把老人背到林子里，弃了变成野兽的口粮。

这些损招都真实出现在历史里，这是奉行孝道之前对待老人的方式，而且日韩欧洲多国都有类似的记录。

说白了就一句话：皇帝老儿用老了用旧了的人，直接干掉就好了。

是不是特不讲理。

而古代的皇上们也因此付出了代价。因为帝国里所有还年轻的人，一边抛弃父母的同时，也都看到了自己老去以后的下场，那么他们注定悲观，看不到活着的希望。

与其活到老被干掉，不如趁年轻干票大的！

当时的年轻人如是想。

于是乎，暴民流窜，盗贼四起，就成为了当时的社会现状。

你作为皇帝，看见这种景象是很焦虑的。

你给庶民养老吧，养不起。

你叫庶民弃老吧，不太平。

可如何是好？

急得你撩起龙袍，直挠屁股。

这时柱子后头转出来一大聪明谋士，说卑职献一计。

你说：但讲无妨。

谋士说了，咱们造个新词儿不就解决啦，这词儿叫孝。

您看，没有孝之前，养老的矛盾发生在陛下您和老人之间。

而创造了孝的概念之后，咱就拉进来一批新玩家，也就是年轻人。

您告诉这帮年轻人，得孝，孝才算是个高尚的人，纯粹的人，脱离低级趣味的人，谁不孝，就得叫人指脊梁骨骂街，连个媳妇都讨不到。

这么一来，您跟老人之间的债务，不用您还，而是他们的子女还，咱就告诉子女，父母生他们是对他们有恩，孝是用以报恩的。

如此这般，老人不用干掉了吧？年轻的能看见老人善终，自己活着带劲了吧？家里有了老人约束不会盲流子漫天飞了吧？您也不用急得抠屁股了吧？

这等于您欠了老人一辈子的债，他们子女帮您还了。

等子女老了，他们的子女又帮您还了，子子孙孙无穷匮也。

您收上来的税银，都是您自己的，打麻将玩商 K 您随意，下边的老人过的不如意，也是骂他们子女不孝，而不会跟您找邪茬了。

你眼睛一亮，说先生妙计。

谋士白折扇一摇，说卑职还没说完。

他说，这一计还有个天大的妙处，就在于每个老人，既然养老都依靠孝子了，那他打算过的舒坦点，是不是会特热衷于生孩子？孩子越多给他的养老保障越足啊！

所以，一旦孝的概念贯彻下去，您就擎好儿吧，用不了几年，您帝国里边保准人丁兴旺，这可能保您龙子龙孙千秋万代啊！

你一听此言，在理，当即抠着屁股就从龙椅上蹦起来了，说来人啊，赏，给朕重重的赏！

......就这么着，孝的概念被创造出来，且延续下去了。

这个概念几千年来被强调，给封建君王们养老人省钱省大发了，而且社会治安也好的超出预期。

因而，不谈孝，那么人老了之后的生计，就会考虑自己一辈子献给了谁，就找谁要饭吃。

压力在君王。

有了孝，那么人老了之后是伸手问子女要饭的，人老了之前是会不遗余力造小人的。

压力从君王转移到子女。

这就是开头说的不平等协议。

一个一生奉献给帝国的人，理应是君王养老，至少是君王和子女一起给养老。

可孝了之后，此人毕生对君王的奉献都变成了应该的，子女却承担了全部的养老义务。

所以你知道为什么古代评人的要点里，忠后边就紧跟着孝了吧？

孝就是君王给老年庶民的免责声明啊。

你看历代君王，出现过琳琅满目的各种混蛋，有花天酒地的，有穷兵黩武的，有不务正业炼丹做木工的。

但唯有不孝的君王，少之又少。

因为他们自己得以身作则，才能叫庶民有样学样，庶民学的越好，君王压力越小。

你拿着孝的古典概念，往当代全球看，也能有所思。

咱们都知道，现在越是发达的国家，越是生育率低，关于这个问题有很多种解释。

从孝的角度看，是能给出站得住脚的说法的。

国家越发达，社会福利体系越完备，无论是养老金还是医疗就越是到位。

那么生活在这种地区的人，他需要生一堆崽子给自己养老吗？

不必。

那么这些人就削弱了生育的最重要动机，他们不需要孝子贤孙伺候自己的老年生活，那他们年轻时肯定就可劲作，而不是把生几个孝子出来当做人生主线任务了。

北欧生育率低吧，他们官方要是立法说，从现在开始，北欧国家养老福利全停，必须指望孩子赡养来做养老保障。

保证北欧生育率直追阿拉伯。

从这你就能看见个反向指标。

越是强调孝的地方，大都是社会给老人保障越少的地方。

反之越是社会养老保障多的地方，越不强调孝的概念。

话说到这，有人得问了，说虎哥你是不是认为不该有孝的概念？

不是，我很认可孝，认为这是一种美德。

但这个美德，如果是从个人内心生发出来的，才是真货。

而被强调被逼着才拥有美德的人，那么谁强调给你这概念，谁才是既得利益者。
