

# 鹰酱的阳谋！

MAGA是一种类似于宗教的运动，之前的文章说过，它当下的所有行为都是为了巩固自己在内部的绝对地位，并且通过又拉又打的方式收服各路势力。真正能够打击并拿来当软柿子捏的也只有那些毫无根基且没有价值的棋子——譬如说，非法移民和通过LGBT占据德不配位地位和特权的抽象群体。

除了这些软柿子可以捏一下，其它的都需要拿利益去交换。内部没有蛋糕可分，就需要从外部搞蛋糕去满足这些人。内部即便捏这些软柿子，也挤不出太多的水，这点利益完全不够分。因此，在拉拢那些真正有实力的势力之时，打击是象征性的，只要对方认个错，服个输，就达到目的了。但接下来的奖励必须要是实实在在的。这些年都是千年的狐狸，别打算跟他们玩聊斋。必须是真金白银的纯粹利益输送才行。这些利益从哪里获取？答案是：外部！

鹰酱现在唯一的优势是什么？全世界所依赖的生产基地？科技的垄断地位？美元的支配体系？独步天下的军工？还是像灯塔一样充当不容置疑的真理一样的价值观？这些似乎都开始呈现出一种底气不足的疲态。鹰酱目前唯一的优势其实是它的消费市场！这个依然是全世界独一无二的体量和规模。虽然这个社会人均负债超过3万美金，但大家依然需要从这个市场挣取顺差，包括欧盟对美贸易也是顺差。中国和沙特对美贸易的顺差远远的超过其它国家。在每年的贸易中赚取美元最多的也就是这两个国家。兔子自从2015年以后，开始逐渐套现美债，把美元借给其它缺美元但有资源的国家，并且让对方还人民币。2024年10月份，兔子又进一步，开始在沙特发行美债，不仅把自己挣的美元借给其他缺美元有资源的国家，接下来会把沙特的美元也借过来再转手借给其他国家。此举会形成一个什么样的后果是显而易见的！

面对兔子这样的阳谋，鹰酱是没有办法的。早些天试图派航母过来做个样子，但被一个靶船就吓跑。这种力不从心，让武力威胁的策略都变得有点可笑。至于价值观上的引导优势因为自家内斗搞得乱七八糟，越来越对外缺乏号召力。鹰酱没有办法阻止美元加速回流自己家。当那些欠了美元债的国家，逐渐把自己的美元债还清之后，鹰酱之前在借款的时候附加给人家的那些盘剥条款，这些国家就可以逐渐不执行了！如果他们有资源却没有产业，兔子可以转移一些产能，投资一些产业，并且与之五五分成。他们就能逐渐的走出债务恶性循环：没钱活着，向鹰酱借高利贷，利滚利还附加恶性条款，变得更加糟糕，需要向鹰酱借更多的高利贷，接受更加恶劣的条款，并接受鹰酱安插的官方人员，进一步变得更糟糕！

这个恶性循环一旦被打破。西方不仅无法继续通过几乎白嫖的方式获取人家的资源并控制人家的经济命脉和内政外交。西方自己的生产生活成本，还会因此循环叠加的提升。这个是有先例的，尼日利亚就是最经典的案例———虽然搞的是法国，但模式都一样！美法之前能够在对方那里驻军，法国核电35%的铀矿几乎是以免费的方式从尼日利亚获取，而且人家的石油和其他矿产也完全控制在西方国家手上。现在则不然，驻军被赶走了，法国需要按照市场价购买他们的铀矿，成本大幅增加不说，供应能不能保障，现在由人家说了算。法国的核电不仅仅自己用，最大的出口国家是德国。也就是说，欧洲德法的工业生产所需电力的成本又增加。这是继油气能源成本增加之后进一步提高综合成本！

这种基础成本的叠加升高，会挤压工业品的利润和市场竞争力。除此之外，兔子在尼日利亚建了好几家石化炼油厂，并且还修了抵达邻国的铁路以及港口。以前尼日利亚的原油挖出来后，运到法国提炼，然后把成品油高价卖回尼日利亚！殖民者就是这么坏！！兔子帮助尼日利亚建立自己的炼油产业，提供工作岗位并培训人员。以后还会需要建立海外军事安保公司为他们提供安全保护并培训军事指挥及技术人员。它们本土提炼出来的原油通过港口运往兔子家换取人民币。然后购买兔子的商品，也购买兔子的武器。

这是一个成功的模型。一路走来真的非常艰辛，中间遭受过无数挑战。殖民者一直搞小动作。其实炼油厂早就建好了，但是没有路运出来。本来铺设了一个管道，通过右边的一个邻国出海。结果管道的路中间，出现了一群恐怖分子。谁资助的这些恐怖分子一目了然。许多博弈都在水下，所有为之付出努力的人都值得尊重和敬佩。大家都拼命骂中石油，但是中石油的职工在这些地方筚路蓝缕的开拓和百折不挠的坚持，才有了今天的结果——守得云开见月明！

当这个模型成功之后，许多地方就可以依葫芦画瓢的复制。这就是为什么秘鲁又搞了个港口。这可不仅仅是一个港口，会撬动周边所有国家的贸易。这些操作本质上就是一种更高难度的扶贫工作。我们在国内执行的扶贫工程，也是把一个一穷二白的地方，改变面貌，建立产业，把当地的人和资源融入到国内市场。在国外搞的这些，也是遵循这个脉络，只不过难度更高。因为这里面有更多不可控的因素，还有很多使坏的殖民者势力从中作梗。因为你的建设行为，会让它的掠夺难以为继。我们在老少边穷少数民族地区扶贫的时候，最大的阻力就来自于当地的宗教势力。在原有的模式下，谁获利最大，谁就会阻止这种变化！

此举的附带作用，会把美元从越来越多的未来国际贸易中排除出去。现有的范围可能难以排除，但是未来的增量如果大于现有的规模，美元的适应范围就会越来越小。这个模式就如同当年奥古斯都与罗马元老院的博弈——奥古斯都把当时罗马控制的成熟富裕地区都交给了罗马元老院，自己带兵把那些未控制地区都打下来，并且按照罗马的方式进行建设和治理。结果他打下来的地方和治理以后产生的财富规模，远远大于元老院控制的部分。最后元老院被他搞掉了，罗马从此进入帝国模式，奥古斯都成了罗马的开国皇帝——他的父亲凯撒因为与元老院的旧势力在存量中博弈，最终被刺杀。他跳出存量去开拓增量，以磅礴之势碾碎了元老院的旧秩序并成功的建立了罗马帝国。

不谋万世者，不足以谋一隅也！中华民族的谋略更关注长远——这是兔子的阳谋。与之同时，鹰酱的精英们也会进行自救和反制。MAGA的主张是发展生产力，让制造回归美国。它提出的是回归和发展，并没有说自己要从零开始重建。因为这不现实！如果你研究过制造业，就会发现这样一个规律：升级的制造业，绝对不是原来的那一批人。

鹰酱现在唯一能够挥舞的筹码就是它的消费市场。这是全世界都需要的！不管是成品还是原料最终都需要通过消费者来消耗。鹰酱拥有规模最大也最成熟的消费市场。它可以通过控制关税的方式来调节谁可以进入这个市场。许多人会说，它自己不能生产，还这样搞，难道不会造成通胀更厉害？当然会，但问题不大。如果你的收入能增加，通胀提高你其实是不在乎的！通胀之所以让人受不了，是因为底层人的收入没有增加，生活成本增加了！如果有一种办法，让人的收入也增加，这种通胀其实可以忍受。

什么办法可以这么做？提高关税本质上是增加联邦收入。如果拿这笔钱直接发福利呢？这个操作确实可以！先让大家忍受一年半载的新鲜期，到了快中期选举的时候直接大额福利发下去。对内可以收买人心，对外则可以产生另外一种效果：懂王对中国的商品普遍征收10%的关税，但是对加拿大和墨西哥却征收25%的关税。此举是不是有点让人摸不着头脑？刚开始我也没有反应过来，后来仔细一想就明白了：MAGA是不想从墨西哥和加拿大进口中国企业在当地组装的产品，而是想让这些企业直接搬到美国去生产！随后他可能会宣布一些优惠政策，一打一拉把这些已经从中国迁出的生产企业，直接逼入美国：你们要么从墨西哥撤回中国生产，然后忍受高关税。要么享受优惠条件，直接搬到一墙之隔的美国生产！

如果判断没错，随后懂王不仅会对墨西哥和加拿大，以及南美洲兔子转移产业的国家加征更高的关税。还会对包括东南亚，越南等地加征高关税。同样会对日韩和欧洲采取这一手。此举的真正目的应该是把从中日韩和欧洲转移到海外的产业都赶到美国去！胡萝卜加大棒，让这些已经转移出去和试图转移出去的企业被迫二选一！如果这些企业，真的在威逼利诱下选择转移到鹰酱家，接下来懂王就会让他们分享技术，并且把生产产品挣来的钱继续在当地投资，而不许转移出去！是不是有点熟悉的味道？对，印度就这么干过，只不过印度失败了。因为印度没有鹰酱那样规模和成熟度的优质消费市场。也没有鹰酱那样的影响力，威逼和利诱的力度都不够，最后就把自己搞成了外企的坟场！即便如此，小米还是不得不把从印度市场挣来的钱继续投资在他们家并间接培育其低端手机制造产业链。

面对鹰酱的这个阳谋，有办法破解吗？接下来兔子肯定会在涉及到电力消耗，以及对构建起相关产业链的关键环节进行控制。譬如说某种必需的原材料禁止出口。某种关键设备也会禁止出口。主动取消补贴降低某些大量需要的必需品的供应量让其成本暴涨，迟滞海外构建起相关生产体系的进度和规模。同时也要开拓其他消费市场，降低对鹰酱消费市场的依赖程度。
