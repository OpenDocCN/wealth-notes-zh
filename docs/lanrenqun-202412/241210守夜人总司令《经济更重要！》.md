

# 经济更重要！

阿德曾经一度众叛亲离到只剩官邸没有被攻陷。周边国家都想让他下台，从而能够夺取自己所需要的资源和土地实现战略缓冲。最后在俄罗斯的下注下才逐渐扭转态势，稳住了局面。经历过生死存亡之后，他一改之前的天真，开始正视自己所统治的这片土地上民众的现实状况，以及周边的现实环境。然而，在失去了对北部产粮区和油田的控制之后，阿德的财政每况愈下，不得不依靠俄罗斯和伊朗的施舍维持运转。

2016年在俄罗斯空天军的配合下叙军收复阿勒颇，库尔德人控制区也象征性地升起了叙的国旗——这种情况就如同1927年老蒋的北伐名义上统一一样：各地的诸侯依然割据一方自成体系，只不过换了面旗帜，昔日的军阀变成了官方的正规军。2017年阿德废除了全国动员令并裁撤和遣散了20多万忠于自己的政府军以节省开支——收编的军阀裁撤不了！准备把工作重心转向经济建设。然而，这个恶果在这几天显示出来了：遣散之后人员不足，武备松弛，对于俄乌战场上进化出来的无人机新战术没有任何应对之策，后面干脆直接放弃抵抗。

后来查了一下关键数据才明白为什么会变成这样：

- 1. 上一次力挽狂澜拼命抵抗的政府军士兵在稳定局势之后被遣散了。在打残的叙利亚社会当兵是他们的就业方式，此举等于被恶意裁员，士兵背后的家庭失去了经济来源。官方因为没钱，上次为国伤残和战死的士兵没有得到抚恤。虽然这是迫不得已的无奈之举，但从有家有口的士兵角度来看，这属于飞鸟尽良弓藏，狡兔死走狗烹！

在俄乌冲突中俄只从苦寒地区招募士兵并给予超出他们收入几十倍的抚恤金。对于普通士兵及其家属来说，钱给够了就会配合官方大谈牺牲精神。否则，他们根本就不在乎任何民族大义。底层民众唯一在乎的就是穿衣吃饭这种看得见摸得着的实实在在的利益。

难道他们不知道整个秩序崩溃后自己的生活只会更艰难？他们确实不知道，也不在乎。对他们而言，生活就是走一步看一步，真到了那一步再说！如果他们有前瞻性并能未雨绸缪，他们的生活处境就不会长期处于那种状况。这就是现实，也是人性。

任何秩序都必须要基于现实并尊重人性，否则就会变成一厢情愿。所有的抗日电视剧都把每个人描写的富有爱国心。如果真是这样，当年为什么伪军比鬼子还多，汉奸遍地都是？千万不要被影视剧骗了，现实中的人绝对不会呈现出那种理想状态。

- 2. 自2011年以来一直被西方经济制裁，在沙伊和解之前，甚至被整个中东国家孤立。本来就失去了对北部油田和小麦产区的控制。再加上内部动荡和外部的制裁。叙的汇率一路狂泻，从520：1变成了现在13000：1——贬值了200多倍！这意味着什么呢？从叙士兵的角度来说，这意味着上一次打战的时候，家里人每个月能拿到1万块钱，现在只能拿到5块钱！如果说1万块还能过得下去，那5块钱够生活吗？！

按照现在的汇率1万块钱都不足一美金。如果对面能从背后金主处搞到美金，直接给每个放下武器的叙士兵发10美金，给的比这边多10多倍，面对嗷嗷待哺的家人，普通士兵不会有太多的犹豫：拼死拼活打赢了也只能从政府军拿到1万，而且死了残了，还未必能拿到抚恤金，但直接放下武器就要拿到13万甚至更多！这就是普通士兵的现实处境。

当我查了汇率和阿德政权的收入数据之后，突然发现一切都合情合理了！

虽然叙会陷入更加的动荡不安，但对于普通民众来说，唯一在乎的就是眼前，至于以后怎么样没想那么多。对于那些长期生活在动荡不安的环境中的人来说，他们尤其不在乎长远——因为他们完全不确定明天是不是还能活着。不要用我们这种长期生活在一个充满确定性的环境中的思维和心态去揣测他们！

在11月24日发不出来的文章中提前说了接下来西方会在两个地方引爆：一个是中东，另一个是缅北。中东的爆点是库尔德人，缅北的爆点是克钦人。在所有的库尔德人中，叙利亚的库尔德人完全处于美国的控制之下，优先级更高。在这次事件发生之前，土和叙、俄及伊开了个会讨论黎以和平之后的考虑。土耳其要求叙军与自己一起打击库尔德人并且允许土在叙北部边境建立起隔离带。阿德拒绝了，随后就发生了这样的事！

现在主要各方都达到了自己的目的：土能在叙建立起更大的缓冲区，避免西方引爆库尔德人波及自己境内的1500万库尔德人。美切断了两条叙的国际公路，从此以后，伊再也没有办法通过陆地来支援包括哈马斯和黎巴嫩真主党以及叙什叶派武装。以色列可以趁机夺取整个戈兰高地及周边的粮食产区。看地形就明白：以处于低洼处而叙处于高地。一旦把自己的领土切入到高地就可以随时从高处进攻叙的南部城市。三方都达到了目的，伊的议员说伊在叙投入了300亿美金打水漂了。俄投入的更多，不仅损失了沉没成本，更重要的是：如果无法跟对面的势力谈妥继续保持阿拉维派大本营的西海岸地区作为割据势力存在。那俄罗斯在地中海唯一的海军基地就不得不撤离。从此以后，俄的海上力量全部被北约堵在家里了！这比损失的那些钱更要命——损失投入是过去的沉没成本，这可是一个大国的未来！所以，大毛不会善罢甘休，棋局还有后手，先等它糜烂。

经济非常重要，一旦没钱了，许多事都无能为力。如果阿德有钱，这事不会发生——当年的局势比现在危险的多，因为还有钱支撑将士用命，最终转危为安。这一次则完全不战而降。当年辛亥革命为什么一下子就成功了？因为清政府真的掏不出钱了！太平天国的时候，清政府虽然农业税少了，但通过关税每年能搞到2000多万两白银，与全国的农业税收入相当。双方杀的有来有回。到了辛亥革命的时候，百十人就能打下一座城市！
