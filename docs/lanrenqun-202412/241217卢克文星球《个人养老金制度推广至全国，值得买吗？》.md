

# 个人养老金制度推广至全国，值得买吗？

241217文/卢克文工作室嘉宾 壹壹壹

整理：公众号懒人搜索，懒人专属群独享

懒人微信:lazyhelper

就在前两天，人社部联合财政部、国家税务总局、金融监管总局和中国证监会一起发布了个通知，说从12月15日开始，个人养老金制度要在全国推广了。

那么问题来了，个人养老金具体是怎样的?到底值不值得买呢?

## 01

早在两年前，个人养老金制度就开始试水了，先在北京、上海、广州、深圳等36个城市搞了个试点。据人力资源社会保障部数据，截至11月底，已有7279万人开设了个人养老金账户。这次推广到全国就是国家鼓励咱们在社保之外，自己再存点钱，为退休做准备。

社保是强制交的，而个人养老金是自愿的，除了退休的人，只要你参加了基本养老保险，就能开个人养老金账户。开账户很简单，去银行或者下载银行APP线上搞定。你往这个账户里存钱，想啥时候存都行，存多存少看你自己，不过一年最多只能存1万2。钱放进去，可以买银行的理财产品、存款、基金或者保险，赚了亏了都是你自个儿的。

这个账户和普通存款账户最大的不同是，钱只能进不能出，得等到退休或者遇到特殊情况，比如生病、失业、出国定居等，才能提前取出来。具体怎么操作，还得等国家出台详细规则。等于是在帮你强制储蓄，国家用你的钱去投资，赚了钱给你分红，时间长了，账户里的钱就像滚雪球一样越来越多，专门留给你退休后用。

等到了时间你要取这笔钱，可以选择一次性全取出来，也可以每个月取一点，取完为止。

国家为了鼓励大家开个人养老金账户，给了税收优惠政策，你往账户里存钱，就能少交点税。如果你挣得比较多，比如每个月收入超过8万5（一年超过102万），那你就能享受到最大的税收优惠，一年最多能省下5400元的税。

得提醒大家一下，现在省下的税，只是暂时的。等你将来退休，开始领这个账户里的养老金时，还得交3%的税。如果你每个月的收入不到8000块，也就是说，你的个税税率不超过3%，那买个人养老金在节税上其实“占不到啥便宜”。

这么看来，要想开个人养老金账户，得先把每月收入提到8000块以上才行。换句话说，就是收入越高的人，买个人养老金越划算。

## 02

个人养老金制度的初衷是帮全民攒钱养老，但现在看来，这个制度对不同收入的人都照顾得不太周到。

对于收入高的人来说，尤其是那些年薪百万的大佬，一年存个万把块，意义不大。而对于收入低的人来说，现在打工赚的工资都达不到交税的门槛，反倒人老了该领养老金的时候，却要交3%的税，显然太不划算了。

而且，咱们国家大多数人的收入情况是，70%的人月收入不到5000元，88%的人月收入不到8000元。也就是说，全国有88%的打工人，都达不到交个人养老金的门槛。收入高的人，觉得个人养老金的收益没有吸引力，收入低的又觉得门槛太高。

所以，这个制度推广两年了，取得的效果只能说一般。虽然开户的人有7000万，户均不过800块钱，总规模在582亿元左右。不过，现在政策似乎又想让每个人都参与进来，那么，这个制度到底想要实现啥目标呢？

有分析说，个人养老金的主要客户是那些税率在10%以上的城镇人群。如果按照人均6000块的缴存金额来算，预计个人养老金的规模能达到1563亿一年。这笔钱很大一部分会通过国债、基金、保险流入股市，给资本市场注入流动性，帮助股市走出长牛、慢牛的局面。

这方面美国最有发言权，美国在1978年开始实施401K计划，把居民的养老金引入股市，美国股市也开始全面转向被动投资，结束了十几年的震荡市，开启了长牛的起点。

所以，不少人把这次的个人养老金制度叫做中国版的401K计划。

401(k)计划，是美国《国内税收法案》第401(k)节的内容。70年代末，美国也遇到了社保大锅饭不够吃的问题，于是鼓励职工自己存养老金，第401(k)节就是这么来的。具体操作是，打工人和雇主各出一部分钱，放到一个养老专户，等雇员退休后使用。

好处是雇主和打工人都能抵税。

计划在1979年确立，1981年制定了实施细则，90年代后迅速发展起来，并逐渐取代了传统的社会保障体系，成为美国很多雇主首选的社会保障计划。

根据美国劳工部的数据，到2023年，大约60%的美国企业为员工提供401k计划，超1亿的美国人手里都有401k账户，总资产达到6.7万亿美元。

401计划里的钱，67%投资进了美股，持有的股票市值占到了整个股市的三分之一。可以说是美国股市里的最大玩家。这些资金主要投资给了跟着大盘一起涨跌的指数基金，美国指数基金总规模约为5.9万亿美元，其中养老金投资约2.6万亿美元，占比近半。

401K计划给美国股市注入了稳定的长期资金，是长达20年慢牛行情的关键之一。这一计划也让养老金实现了保值增值，良性互动，让美国社保体系和股市的共同成长。

通过401K计划，美国人退休后的收入替代率提高到84%-106%。这意味着，如果你月入5000美金，退休后可以拿到4600美金，最高可能有5300美金，比工作时挣得还多。反观中国，社保的养老金替代率只有46%，等于如果你月入5000，退休后大概只能拿到2300块，生活质量严重受到影响。

如果个人养老金能像美国的401K计划，把钱投到股市里，而且股市逐步上涨，每年能赚不少，那就算是工资低到不用交税的人，也会愿意把钱放进去。因为哪怕以后要交点税，这笔投资的回报也是划算的。

## 03

长期来看，个人养老金投资确实能带来不错的收益，但风险也得自己承担。所以投资时得小心，确保资金安全。

现在银行都在抢客户，推出各种优惠，比如抽红包、送黄金。不过，咱们不能只看这些开户福利。别忘了，个人养老金规定每个人只能在一家银行开一个养老金账户。而且，每家银行提供的产品都不一样，得好好比较。

个人养老金的第一批试点产品主要是FOF基金，也就是投资基金的基金。这些FOF基金至少80%的资产都投在公募基金上。这样可以让咱们用不多的钱，参与到多个基金的投资中，提高了投资的分散性，降低了风险。

但FOF基金的手续费比较高，如第一批养老金产品里，199只FOF基金的管理费率，基本在0.31%以上，还存在双重收费的问题，买FOF基金要交手续费，FOF基金买其他基金还得再交手续费。

至于收益，受金融环境影响，FOF基金今年才开始回暖。很多人投资了两年多，亏了两年多，最近才开始盈利，算下来盈利大概1%，只能说一般般。

因此，这次为了让咱们有更多的投资选择，除了理财产品、存款、商业养老保险和公募基金，还把国债和指数基金加进了个人养老金的投资范围。证监会选了85只权益类指数基金加进个人养老金投资产品里，其中78只是跟踪宽基指数的，7只是跟踪红利指数的。

指数基金就是跟着某个指数走的基金，比如大家都知道的中证 A500 指数、创业板指。这种基金有两个好处：
- 第一，省心，不用花太多时间去研究公司和基金，就能跟上市场的平均表现；
- 第二，费率低，像某基金最近就把部分费率降低了，管理费从 0.5%降到 0.15%，托管费从 0.1%降到了 0.05%。比起 FOF 基金，手续费更划算。

红利指数则更关注那些长期持续分红、分红比例高的上市公司，这些公司大多经营稳定，现金流充足，在市场波动的时候股价依旧坚挺。而且，随着无风险利率下降和各类资产回报率降低，红利资产的 5%-6%的股息率会更有竞争力。

总的来说，这两类产品都属于适合养老金长期稳健增值的投资目标。

当然，不同的产品适合不同的投资者。
- 个人养老金里的储蓄、保险、国债比较稳健，预期收益相对较低，适合想要保本的投资者。
- 理财类产品预期收益和风险相对平衡，波动较小，适合追求一定投资收益的投资者。
- 基金类产品风险水平相对较高，预期收益也较高，适合喜欢冒险、有一定投资经验的投资者。

想要投资个人养老金的朋友，还是根据自己的需求，合理分配投资。
