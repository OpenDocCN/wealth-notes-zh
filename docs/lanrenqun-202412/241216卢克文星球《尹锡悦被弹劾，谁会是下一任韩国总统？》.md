

# 尹锡悦被弹劾，谁会是下一任韩国总统？

241216 文/卢克文工作室嘉宾 星海舰长

整理：公众号懒人搜索，懒人专属群独享

12月14日，尹锡悦弹劾案通过国会表决，进入宪法法院审理环节。很多人都觉得，这个人憎鬼厌的亲美亲日韩国总统，总算是要下台了，可喜可贺。

但是，弹劾案通过国会表决，还只是万里征途走完了第一步，后续还有很多流程要走。

所以，最后鹿死谁手，还未可知。

## 1

首先我们要思考一个问题，为什么第一轮弹劾中，国民力量党全体成员退席拒绝投票，一副坚决不支持弹劾的样子，然而才过了一个星期，就有12个议员反水了？

说实话，这和国民力量党的内斗有很大关系。

前几天的星球文章中，大概介绍了韩东勋的打算，是不急于让尹锡悦下台，而是通过手握弹劾权（共同民主党票数不够，弹劾是否成功取决于韩东勋），逼尹锡悦交出权力，让自己垂帘听政。

所以，才会出现韩东勋要求国民力量党议员集体退场，导致第一次弹劾流产。

但在这一周中，发生了太多事，震碎了很多人的三观，也改变了不少议员的态度。

比如著名媒体人、“世越”号调查记者金于俊（唯一一个在尹锡悦抓捕名单上的媒体界人士），参加了韩国国会举行的关于戒严现状的调查会议，根据他披露的消息，媒体才总算完成了尹锡悦戒严计划的完整拼图。

尹锡悦的“戒严剧本”，其实从今年9月份就开始了，尹锡悦安排自己的心腹金龙显，对韩国特战部队下令，对北边释放无人机，故意挑衅，北边如果激烈反制，甚至擦枪走火，尹锡悦就可以顺理成章进行戒严。

但是呢？将军根本不上套，只是炸毁了连接南北的通路，摆明了是一个防守姿态。强行搞戒严的话，讲不通。

于是尹锡悦就搞了个 PLAN B，先戒严，让韩国特种部队以实施戒严令为由，将韩国的一干政客们（包括李在明、韩东勋、禹元植、曹国、杨正哲等等）统统控制起来，然后送到“西冰库酒店”关起来，在押送过程中，会遇到假扮的“北方军队”营救，一方面坐实这些人“通北”的罪名，另一方面可以趁机把这些人都突出了。

关键在于，这些“北方军队”在逃亡的时候，还要与驻韩美军展开交火，射杀几名驻韩美军，引导美军报复性攻击。

这样一来，尹锡悦就可以恐吓国民，在战争的威胁下搞军事独裁。

如果只是搞了个戒严，恐怕很多国民力量党的议员会支持他，在右翼看来，搞戒严真的不是个啥大事。但如果尹锡悦的计划是挑起战争，那就触碰很多人的逆鳞了。毕竟，很多议员一家老小都在首尔，是真可能没命的。

虽然这一计划还没有得到最终证实，但很多国民力量党的议员都认为，不能让这个疯批总统再待在位子上了，否则谁知道还会干出啥魔幻事儿？

韩东勋就表示，不再反对弹劾尹锡悦。然而这一表态，在党内遭遇了不小的反对，比如党鞭权性东。权性东这个党鞭是刚当上的，前党鞭秋庆镐在12月7日因戒严的事情引咎辞职，然后12日权性东以72票对34票的压倒性优势当选新党鞭。

他是尹锡悦的铁杆亲信，大骂韩东勋是叛徒。

显然，哪怕尹锡悦干出了天怒人怨的事，在党内还是有很多“沉默的支持者”的。

韩东勋只能退一步，说“反对弹劾”仍然是全党立场，但不限制议员按照信念自行决定是否投票弹劾。韩东勋这话说的，意思还是鼓励弹劾的，这也是很多国民力量党议员选择反水的原因之一。

然而，韩东勋也马上遭遇了反噬，弹劾案通过后，张东赫、金旻圃、金在元、尹相直、秦钟午等五名国民力量党最高委员宣布辞职，国民力量党章程规定，如果四名或四名以上当选的最高委员辞职，最高委员会将被解散，该党的领导体制将转为紧急对策委员会制度，那么韩东勋就要下台了。

现在弹劾案通过，下一步的矛盾点，主要集中在两个方面。

第一是法官人数不够。

目前网上有说法，弹劾案最终裁决通过，需要现有6个法官都投赞成票，才能通过审核，只要有一个人投反对票都不行，其实这个说法是不完全准确的。

根据韩国《宪法法院法》法律第23条规定，“裁判部由7名以上法官出席审理案件”，而“做出法律违宪决定、弹劾决定、政党解散决定或宪法诉愿相关的引用决定”等，“必须有6名以上的法官赞成”。

那么问题来了，韩国宪法法院原本有9个法官，但是今年9月10月先后有3个法官退休，现在只有6个法官了，那么，6个法官能不能审弹劾案？目前有很大争议。如果不行，那就要增补法官。

如果6名法官可以弹劾，那共同民主党就紧张了，因为法官郑亨植是尹锡悦的人，也是现任的6名法官中，唯一一名由尹锡悦提名的法官（宣判李在镕无罪的就是他），为了拉拢郑亨植，尹锡悦此前还把郑亨植的大姨子朴善英，任命为真相和解委员长。

郑亨植有可能投反对票，导致弹劾案流产。

为了避免这种情况发生，共同民主党要做的，就是赶紧提名空缺的法官，凑够9个，增加通过弹劾的概率，这样就算郑亨植投反对票，也还有转圜的余地。

根据韩国法律，9名法官全部需要总统任命，其中3位由总统直接任命、3位由法院院长提名、3位是国会提名。巧了，现在缺编的3个名额，都是来自国会提名，这就给了共同民主党提名自己人的机会。

当然，考虑到国会的情况，共同民主党不可能把这三个名额都拿到，能拿到2个就不错了，共同民主党推荐了首尔西部地方法院院长郑启善和首尔西部地方法院部长法官马银赫，而国民力量党推荐了高等法院部长法官出身的赵汉昌。

如果被推荐的3人全部就任，那9名法官中，将是：保守派2人（郑亨植、赵汉昌），进步派4人（文亨培、李美善、郑启善、马银赫），中立派3人（金炯科、郑贞美、金福亨）。这样来看，弹劾案凑够6票还是比较容易的。

但问题在于，新提名这三个人，能不能得到总统任命？

因此第二个方面就是代理总统人选的问题了。

前面提到了，3个新法官人选需要总统任命，现在总统被弹劾停职，理论上应该由代理总统任命。

那么问题来了，来自国民力量党的代理总统韩德洙，明知道增补法官会提高共同民主党成功的概率，会批准任命吗？

事实上，这是有先例的，当年朴槿惠被弹劾时，代理总统黄教安就冻结了很多人事任命。

《宪法法院法》第38条规定：“因法官的缺位不可能出席7人时，其缺位期限不算在审判期限之内。”

也就是说，虽然宪法法院对弹劾案的审判期是180天，但只要韩德洙拖延一段时间不任命法官，共同民主党一点脾气都没有。

那共同民主党怎么办？只能威逼利诱。

此前，李在明指责韩德洙也是尹锡悦发动“紧急戒严”重要参与者，曾表示将推动弹劾韩德洙。根据韩国法律规定，如果国务总理也遭弹劾停职，将无法代行总统职权。

但是在12月15日，李在明在记者会上表示，将不再继续推动针对代行总统职权的国务总理韩德洙的弹劾程序。同时，李在明建议韩国执政党、在野党等各方建立合作机制，保障韩国政治民生安定，还要求韩国宪法法院应尽快进行尹锡悦总统弹劾程序，让韩国政局现在的“混乱最小化”。

显然，这是一种威胁：韩德洙你最好尽快任命法官，让宪法法院赶紧走审理程序，否则下一个弹劾的就是你。

如果韩德洙死硬就是不任命呢？

那共同民主党就会弹劾其参与“内乱罪”，取消其代理总统职务。按照顺位，代理总统职务应该由经济副总理兼企划财政部长官崔相穆接任。问题在于，崔相穆也不干净，他也在戒严前后参加了国务会议。

韩国警察厅特别调查团已开始对其调查，他也没资格代理总统了。

顺位再往下排，且没有参加戒严国务会议的，看来看去，就只有韩国科学技术情报通信部长李宗昊了，但他正因为儿子的论文舞弊丑闻焦头烂额呢，哪愿意趟这个浑水？

如果共同民主党这么搞下去，可能就会发现，国民力量党里面找不出人适合代理总统了。没人任命法官，对存心拖时间的国民力量党反而是有利的，所以，可能还是会回到妥协和交易的路线上来。

## 2

就算共同民主党增补了法官，尹锡悦就一定会被判决下台吗？

未必，因为历史上，是有过总统被弹劾后官复原职的先例的（卢武铉），尹锡悦更是准备困兽犹斗了。比如弹劾案通过后，他再次发表了电视讲话，表示“绝不放弃”。

同时，尹锡悦正在汉南洞官邸紧锣密鼓组建弹劾调查辩护团队，准备迎接即将到来的审查和辩论。注意，这个审查和辩论的过程，也是有门道的，特别是国民力量党不愿意把政权拱手送出的情况下，有很多工作可以做。

首先，尹锡悦可能将申请大量证人并亲自出庭辩论，以争取更多时间。

其次，尹锡悦因为是“内乱罪首犯”，很可能被警方逮捕起诉，但问题在于，根据韩国法律，如果弹劾案的指控与刑事诉讼涉及同一事由，宪法法院可选择暂停审理，这样一来，又可以把时间拖一阵。

只要把时间拖下去，拖到李在明二审（2025年2月）和三审（2025年5月），那么就有转机，只要法院审判李在明有罪，就可以以此作为交换的筹码。

不过，这只是战术上的操作，在战略上，大形势对尹锡悦还是很不利的。

目前，尹锡悦的支持率已经只有11%了，就算使出各种的花招，宪法法院可能也要考虑如此延宕时间，能不能向国民交代的问题，所以很可能不会如尹锡悦所愿，会加快审理。

毕竟卢武铉的审理才用了60多天，而尹锡悦的案情相对卢武铉要简单，所以很可能这次审理很快就能结束，赶在李在明二审前做出裁决。

所以，国民力量党必须现在就要做好一旦尹锡悦被裁决下台，大选怎么准备的问题。

首先是要确定候选人，来和如日中天的李在明PK。

从目前来看，国民力量党方面，有3个人能参与到总统角逐，分别是韩东勋、吴世勋、洪准杓。

韩东勋自然不必说，是戒严后国民力量党亮相最多的政治人物，本来支持率也很高。但问题在于，他的反复横跳，摇摆不定，让党内非常失望，觉得这个党首像变色龙一样，意志不坚定，跟着他混一不留神就被卖了。

所以韩东勋要想参选总统，就要先得到党内“尹派”的谅解。

而韩东勋党内最大的竞争对手，就是首尔市长吴世勋。

吴世勋算是国民力量党地方派头号人物，也明确表态说要参与总统竞选，而且表示自己不怕韩东勋，声称“韩代表一辈子主要在调查领域，很有经验。但我在执行第四任首尔市长职务等综合行政方面积累了经验。”

吴世勋这是在讥讽韩东勋没有从政经验，一个没有从政经验的尹锡悦已经把韩国搞成这个样子了，怎么能再上来一个检察官出身的？

而大邱市长洪准杓呢？是朴槿惠的旧部，早在2022年大选的时候，就参与了大选，结果在党内落败于尹锡悦。那么在尹锡悦下台后，洪准杓很可能会再次跳出来，继续参加竞选，圆过去没能圆的总统梦。

考虑到洪准杓和吴世勋都属于地方派，而且立场相近，不排除二人会达成同盟，互相支持，共同对付韩东勋。

相比国民力量党的四分五裂，共同民主党这边要团结一些，经过李在明这一番折腾，李在明在党内的权力前所未有地巩固了，而随着曹国涉嫌为子女升学舞弊案终审判决有罪，曹国的政治生命也没了，已经不能在进步派阵营对李在明造成威胁了。

而目前在共同民主党内，能和李在明PK的只有党鞭朴赞大，但问题在于，朴赞大本身就是“李在明派”的人，所以从现在来看，朴赞大不像是李在明的对手，而更像是李在明的备份，如果李在明真的被判刑，朴赞大就可以替补参选。

所以，在未来这四五个月的时间，国民力量党和共同民主党一定会围绕法官任命、总统代理权和党内候选人的问题开展一系列恶斗，再加上特朗普的上台，也可能给韩国的政坛带来新的变数，在这期间什么事都可能发生。

鹿死谁手，尚未可知。
