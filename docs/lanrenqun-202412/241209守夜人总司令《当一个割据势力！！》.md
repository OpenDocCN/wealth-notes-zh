

# 当一个割据势力

叙利亚自从丢失了北方的油田和粮食产区之后，财政一直枯萎。

别说民生建设这些年完全没有起色，就是军队也只能养四个师。在原有的战斗模式下，这4个装甲师加上俄军的空军配合，足以让各路军阀不敢造次。俄乌冲突中无人机的大放异彩，改变了这种力量对比。

一个无人机飞手就可以对付一队坦克，而且对方还没有还手之力。这是战场上的平衡被打破。除此之外，叙利亚完全靠俄罗斯和伊朗的拨款供养着。先是俄乌冲突，后是巴以冲突。把这两个金主都打得元气大伤。黎巴嫩真主党之所以愿意和谈，不仅仅是因为他们的指挥系统被打掉了，还包括他们没钱了。他们用来筹集资金的金融机构被美以联合封杀，筹集资金的能力越来越弱。现在把库存的资金打得差不多了！没有钱就无法组织起力量。

叙利亚的4个师都驻守在几个关键要地：譬如，守住西部低地的两个据点：哈马和霍姆斯，守住东北部通往伊拉克的国际公路的石油城市代尔祖尔。老虎师第25师的一部分在阿勒颇驻守。阿萨德弟弟的第四装甲师和第十七师在代尔祖尔。第11师在哈马，第25师的另一部分在大马士革北面的霍姆斯。大马士革本身是没有主力防守的！

老虎哈桑带着25师，从东南方向，沿着沙漠公路向西北进攻阿勒颇，本来打下了两个城镇，但在靠近阿勒颇国际机场的时候，开会的指挥部被对方的无人机击中！失去指挥的残部全部溃散，已向南撤回大马士革。在正面沿M5公路驻守哈马的第11师，在遭受叛军的无人机攻击配合坦克冲锋的攻势下，防线被叛军突破。叛军攻入城中，叙利亚第11师放弃了哈马向南溃逃到霍姆斯这个最后防线。微信 pep854

美军在东北角配合库尔德人进攻代尔祖尔，试图切断叙利亚到伊拉克再到伊朗的陆地通道，并出动了A-10对地攻击飞机。然而，阿萨德的弟弟带领的第四装甲师和第十七师打退了库尔德人的进攻。12月5号，美军发现东北方向无法得手，从东南方向，自己占据的边境据点出发，向西进攻大马士革，将自己的控制区域扩大了一倍。威胁着东北方向的代尔祖尔到西南方向的大马士革之间的沙漠公路。阿萨德的弟弟带领第4师和第十七师在美军切断这条沙漠公路之前，放弃了代尔祖尔向大马士革撤退。随后，美军控制的库尔德人，越过幼发拉底河，在没有经过战斗的情况下，占据了久攻不下的代尔祖尔。从此美军彻底切断了伊朗从陆地上增援叙利亚、巴勒斯坦和黎巴嫩的通道！美军的目标达成！

在叙利亚政府军放弃哈马退往霍姆斯之时，以色列正在集结部队试图向戈兰高地的剩余部分进军，同时策动了叙利亚南部的叛乱。土耳其资助和指挥的另一股叙利亚势力趁机占据了北方的一大块地方，并且攻击叙利亚西北方向的库尔德人武装。土耳其最初的目的是要建立起一条边境隔离带，现在有机会会进一步要扩大范围，会试图建立一个更大的缓冲区，从而把美军资助的库尔德人与自己境内的1500万库尔德人隔离开！

陆地通道被切断之后，伊朗目前使不上劲。伊朗的伊尔-76运输机非常少，而且F-4也没有办法护航。这两天以色列的F-16战机在叙利亚的天空驱赶伊朗的伊尔-76运输机，不让其在叙利亚的机场降落。以色列的目的是要把伊朗的势力从叙利亚彻底赶出去，美军在配合以色列的这个目标。伊朗凭借自己的力量没有反制的手段。

最后的核心就是俄罗斯的利益。阿萨德的力量不足以守住全国，阿萨德的资金更不足以维持全国的秩序。现在最务实的做法，就是把所有的力量集中到首都周边，守住西部靠海地区，当一个割据势力，等待时机，其他的地方不管了！

4个师如果摊开到全国，肯定任何地方都兵力不足。但要集中在周边，还是有一战之力的！而且俄罗斯的基地都在这一块，俄罗斯为了自己的利益，也不得不全力以赴，守住核心区。红色的这个地方聚集了叙利亚70%的人口，只要能守住这个地方就是一个实实在在的割据势力。接下来叙利亚政府军需要做的就是守住霍姆斯并反攻夺回哈马和赛拉米耶。然后与土耳其商谈，承认他所占据的缓冲区，以换取它的中立和援助。

俄罗斯在叙利亚目前控制的地方有重大利益——西边的两个港口，是俄罗斯进入地中海的支撑。除了这个港口，不管是俄的北方舰队，还是黑海舰队都被北约堵在家门口了！再失去这个，俄罗斯就彻底被堵在家里。接下来的局势关键有两点：

- 1. 与土耳其谈妥缓冲区。
- 2. 集中所有力量守住霍姆斯和大马士革周边核心区。

叙利亚不会成为第三次世界大战的前奏，参与博弈的各方达到自己的目的以后，也就不会继续投入。作为世界级玩家，他们都清楚——螳螂捕蝉黄雀在后！
