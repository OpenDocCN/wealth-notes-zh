

# 叙利亚变天的内幕，是一场政变？

241210 文/卢克文工作室嘉宾 风雨如歌

整理：公众号懒人搜索，懒人专属群独享

![](img/ea792fab86461298887b271b78f55a0c_0_0.png)

叙利亚短短12天时间，就完成了变天，政府崩溃，军队不知所踪，令世界惊愕。

个中原因，除了各方已经说烂了的经济问题外，越来越多的资料表明，这是一场精心策划的政变。

让我们先从开战的第一天说起。

11月27日，也就是以色列从黎巴嫩南部撤军的日期，盘踞在伊德利卜省的恐怖组织HTS（沙姆解放阵线），突然对政府军发起大规模进攻。

没错，政府军是被打了个措手不及，丢失了不少阵地，但起码在前几天，政府军的表现还是正常的。11月29日，政府军（SAA）进行反击，夺回了一半的丢失区域。

本来外界预计，也就这样了。

然而没想到，不正常的事情开始来了。11月30日，反击的SAA接到命令，要求他们撤退，还不是一次，是好几次强硬的命令。

一头雾水的SAA，就这样撤退了，HTS毫无阻碍地进入了阿勒颇市，开始大肆圈地。

意识到不对的巴沙尔，调动了王牌部队第25特种师（老虎部队）前去阿勒颇增援，第25特种师的主力到达郊外后，来自国防部的命令却非常强硬——不许进去。

于是，他们只能等在阿勒颇的东南郊，眼睁睁看着一切走向无可挽回。

也是从11月30日起，HTS的另一路兵力——仆从军TIP（突厥伊斯兰党）开始南下，沿途的SAA连连撤退，让出了汗谢洪、迈阿赖努阿曼等大批城镇，一直退到了哈马市。

TIP到达哈马市后，开始攻城，但是发现攻不下，SAA部分基层单位进行反击，在第25特种师一个留守团的带领下，协同其他单位和一些基督教民兵，向北夺回了几个城镇。

SAA在反击成功后，HTS调来增援部队，连续攻了几天还是没能攻下哈马市，就在外界以为，哈马市已经安全了的时候，那些反击的SAA单位，再次接到了不正常的命令。

上级非常恼火，把他们骂了一顿，严令他们撤退，此时HTS根本没有打进来。12月5日，SAA从哈马市撤退，HTS轻松拿下哈马市，接着一路狂追，追到了霍姆斯郊外。

距离霍姆斯城，只有5公里。

哈马市之后，便是霍姆斯市，霍姆斯市的丢失过程几乎是哈马市的翻版。

撤退到霍姆斯市后，SAA的部分单位、尤其是第25特种师的人员，不愿意再撤退，进行了激烈的抵抗，HTS的装备太差，就一些皮卡而已，所以打不进来。

双方僵持了两天，HTS的首领朱拉尼，亲自来到霍姆斯督战，依然没有太大进展。直到12月7日晚间，上级下达了一堆命令，让霍姆斯的守军立刻、马上给老子撤退。

霍姆斯遂丢失。

在霍姆斯激战时，南边也出现了同样诡异的事情。叙利亚西南部的德拉省，原本不剩下多少叛军，但在这一轮战事开启后，他们集结起来，重新进攻。

德拉省的叛军，也就是叙利亚自由军（FSA）并没有几个人，然而当地SAA接到的命令，也是清一色的“撤退”，于是德拉省拱手让给了FSA。FSA在拿下德拉省后，迅速北上，直取首都大马士革，沿途的SAA接到的命令，也是清一色的“撤退”。仅仅几天后的12月8日凌晨，FSA就打到了大马士革的郊区。

12月8日白天，FSA占领大马士革。

纵观整个过程，政府军不是完全没有抵抗意志，至少有部分精锐单位是愿意抵抗的，在他们的带领下，一度夺回了一些城镇。

HTS的阵亡数字并不低，光是发出明确讣告的就有一千人以上，加上受伤的，总伤亡起码有四五千，约30%是俄军空袭炸死的，剩下70%则是政府军和民兵的战果。

遗憾的是，每当基层有人想抵抗的时候，他们就会接到一道道要求“撤退”的命令。不仅是第25特种师，最精锐的第四装甲师也是如此。

政府军（SAA）有两支部队是最重要的：第25特种师和第四装甲师，他们在，政府军就在。

第25特种师的指挥官是苏海尔·哈桑，第四装甲师的指挥官是总统巴沙尔的表弟马希尔·巴沙尔。按照常规的部署，第四装甲师有一半兵力是在首都大马士革周边的。

然而这一轮战事开启后，第四装甲师的指挥官马希尔，表现极其反常，他竟然有时间玩社交媒体，随后，第四装甲师被他带着跑到了东部，远离首都，接着润到了伊拉克。

致使大马士革空虚。

在巴沙尔离开后，叙利亚总理和诸多部长，还有军方高层人员都没有离开，HTS和FSA进城后也没有攻击他们，继续让他们作为过渡政府运作。这就奇怪了。

他们为巴沙尔效力多年，和叛军的仇恨并不少，莫非一点也不怕报复？如果说巴沙尔政权是腐败的，捞了许多钱，那么下面这些总理、部长、军头们，就个个都是清廉的？难道不怕叛军以反腐为借口，收拾他们？

事实就是，叛军非但没有收拾他们，双方还在第一时间进行了合作。这又是一个奇怪的迹象。

总结这些信息，我们不禁发现，这更像是一场政变，不排除是将领和高官们联合起来，架空了巴沙尔，作为总统的巴沙尔，根本不知道发生了什么。

而其表弟马希尔和总理贾拉利的反常表现，更为“政变说”增添了可信度。

俄罗斯和伊朗最初没有反应过来，SAA让出了哈马市，俄罗斯政府还责问了巴沙尔，为什么你的军队一直撤离阵地，要知道，俄军的飞机轰炸还是挺努力的。

12月6日，两国才反应过来，大致明白是怎么回事了，只是为时已晚。

这场政变的主要幕后推手，大概率是土耳其，很多SAA的将领大概率早就被收买了，其实也正常，叙利亚经济崩溃后，哪怕是权贵富豪也过得不怎么样。

持有纸币吧，就叙镑的贬值幅度而言，早就是废纸了；

持有资产吧，天天打仗，还是贬值。如此乱世里，将领的收买成本相对低，即使是贵为总统表弟的马希尔，抵抗了那么多年，估计也是真的累了。

这也就解释了，为什么每当部分基层单位想要抵抗时，总是接到上级的撤退命令。以色列和美国可能起到了协助的作用，但土耳其才是主要的，埃尔多安都懒得掩饰。

12月6日，他公开表示支持HTS，是第一个这样做的外国领导人，而美国和以色列尽管支持HTS，但也不敢由领导人公开表态。

作为少数派，巴沙尔能撑那么多年，已经是奇迹了，然而奇迹不可能一直持续。后巴沙尔时代的叙利亚，是长治久安呢，还是像利比亚那样进入“吃鸡大赛”呢？

大概率是后者。

叙利亚的战略区位，相当于中东的十字路口，实在太重要了。

事实上，就在12月8日，也就是巴沙尔政权落下帷幕的第一天，新的战火就燃起了。

## 土耳其支持下的叙利亚国民军（SNA）

以色列军队则越过了边境，占领了叙利亚的库奈特拉市，是1974年来首次越境侵略。

还有占领大马士革的FSA和HTS，两家是不对付的，早晚要打起来。当原有的权力所有者倒下，权力真空出现时，不同的势力必然会卷起来，直到卷出一个胜利者为止。

考虑到叙利亚的地缘重要性，谁也不会轻易放手，意味着短时间内难有真正的胜利者，而叙利亚民众，唯有面对无休止的战火。
