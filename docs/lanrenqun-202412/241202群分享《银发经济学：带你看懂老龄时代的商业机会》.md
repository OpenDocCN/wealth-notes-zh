

# 银发经济学：带你看懂老龄时代的商业机会

241202

![](img/054160988a710eb867ddcd0c270bf5be_0_0.png)

今天我们聊聊银发经济。这个话题最近几年被提及得很多。但是，我最近收获了一个关于银发经济的，很不一样的视角。它来自日本学者大前研一的新书《银发经济学》。书里虽然说的是日本社会，但对我们理解银发经济现象，还是有所启发的。

过去提起银发经济，我们一般能想到两件事。一来，假如国家进入老龄化社会，那么为老年人提供好的商品和服务，肯定是个很重要的商业课题。二来，有了足够的商品和服务，老年人就能多消费，也能提升社会经济。

但是，这回大前研一提出了另外一个视角，也是这回《银发经济学》这本书的核心。说的是，只有老年人多消费，才能解决日本的少子化问题。

没错，大前研一认为，让老年人花钱不是目的，只是手段。整件事的最终目的，是要解决日本的少子化问题。

这个结论乍一听有点奇怪，老年人的消费，和年轻人生不生孩子，好像是两件事啊。它们是怎么搭上关系的呢？

在说大前研一的观点之前，我们得先交代一个背景。作为最早开始研究社会老龄化的学者之一，大前研一这几年的名气越来越大。比如早年的《低欲望社会》《M型社会》《无国界世界》等等，不仅书很畅销，更是为整个学界贡献了一个新名词。

但是，要特别注意的是，大前研一并不是那种单纯的学术型学者，他还跟日本政府保持着一种很奇妙的关系。要想理解大前研一的观点，我们就必须得明白这层关系。

一来，他曾经在日本政府担任过高级贸易官员，但离职之后，依然坚持不懈地给日本政府出主意。

二来，假如这些主意不被接受，他就会批评日本政府，从各级官员一路批评到首相，而且老人家还会把这些批评写在下一本书的前言里。比如上一本书跟这本书间隔三年，那好，他就把这三年积累下的对日本政府的吐槽，全都写在这本新书里。大前研一作品的前言假如集结在一起，差不多就是一本对日本政府的吐槽大合集。

三来，这老爷子尽管一直在批评政府，但另一方面他却在替政府培养人才。比如早在1994年的时候，大前研一就创立了一个叫一新塾的机构，塾，就是私塾的塾。一新塾学员的毕业论文，大前研一还会指导一番，然后报送给相关单位，敦促他们落实。当然，假如不落实，相关单位的负责人就可能出现在大前研一下一本书的前言里。从一新塾走出去的人，还有不少成了日本的众议员和地方议员。

换句话说，大前研一的一些书并不代表确定的结论，在一定程度上，这算是他与日本政府沟通的中间产物。书里面记录了他和政府沟通的过程，以及相关的调研和思考。

而这回，大前研一的最新思考产物，就是前面说的那个观点，只有老年人多花钱，才能解决日本的少子化问题。

那么，这个结论是怎么得出的呢？

咱们先从日本的现状说起。日本的少子化有多严重呢？1980年，日本的一人户家庭占家庭总数的19.8%，而2020年，这个数字是38%。没错，翻了一倍。大概相当于10个日本家庭里，有4个家庭都只有一个人。

日本还有另外一个统计指标，叫标准家庭，也就是夫妻二人加一个孩子的三口之家。这样的家庭有多少呢？是25%。根据日本国立社会保障人口问题研究所的推算，2040年，日本一人户家庭将达到40%，而标准家庭将减少到20%。

而且日本人现在不仅不愿意生孩子，连恋爱都不谈了。根据日本的《第16次出生动向基本调查》，在日本18到34岁的未婚群体中，有对象或者已订婚的男性占比只有21.1%，女性占比是27.8%。有同居经历的男性是6.4%，女性是8.2%。有性经验的男性占比53%，女性47.5%。而在这部分未婚男女中，有70%以上的人觉得单身生活更轻松，更自由，经济压力更小。

日本政府想没想过办法？当然想过。为了鼓励家庭育儿，只要是怀孕的日本女性，并且持有母子健康手册，就可以领取10万日元的补贴。这个补贴通过优惠券发放。10万日元，大概相当于4660元人民币。大前研一认为，这点补助是杯水车薪。

那么，怎么解决这个问题呢？你看，既然人们不想生孩子的理由是经济压力大，那么，能不能通过进一步减轻经济压力来促进生育呢？

关于这个问题，大前研一有个前部下叫茂木敏充。这个人最近经常上新闻，他曾经担任过日本经济产业大臣，前段时间还宣布要参选日本自民党总裁。茂木敏充曾经提出，日本政府应该借鉴N分N乘税收制度。

N分N乘税收制度起源于法国。而法国的生育率在发达国家里属于比较靠前的，2020年法国的总和生育率是1.83。而所谓N分N乘税收制度其实也不复杂，过去税收不是以个人为单位吗？而N分N乘税收制度，就是以家庭为课税对象，用家庭总收入除以家庭总人数，然后在这个基础上使用累进税率征税。累进税率，也就是多挣多缴，税率随着收入数额增加。

大前研一说，假如按照N分N乘税收制度征税，就意味着年轻人一旦成家生孩子，那么他的计税基数就可以被孩子平均一下，就变少了，一个家庭就可以少缴不少税。这多少能鼓励年轻人生育。

但相关的日本官员并没有采纳这个建议。理由是，按照这套制度，受益最大的其实是单职工家庭，而不是双职工家庭。而日本政府希望社会上的劳动者更多，双职工家庭更多。

当然，这个理由大前研一并不接受，他觉得税收制度还可以做更细化的改革。总之，经过一轮沟通，谁也没说服谁。其间大前研一还提了很多有点激进的建议，比如放开私生子的户籍制度，不要管孩子父亲是谁，只要在本国女性所生就应该给人家落户，等等。但结果又是谁也没说服谁。

怎么办？大前研一又开始算另外一笔账，叫做，日本孩子们的负债。也就是，一个孩子从出生开始，就背负的债务。大前研一说，日本的国家债务在2023年6月底，已经达到1276万亿零3155亿元日元。平摊到每个国民身上，债务超过1000万日元。同样，刚出生的孩子也背着这笔债。

这笔债是怎么欠下的呢？大前研一说，关键原因之一是日本政府持续推出助长负债的政策。比如，日本规定，国内汽油的平均价格不能高于每升170日元，假如高于这个价格，日本政府就要向石油总经销商发放补贴。但是，国家的补贴从哪来？从税收中来。因此，大前研一就说，这些持续的补贴政策变相加重了日本人的负债。假如一个孩子一出生就背负1000万日元的债务，人们是不是就越来越不愿意生孩子了？

换句话说，解决少子化问题的关键，是盘活日本经济。怎么盘活呢？必须得让这个国家最有钱的人开始花钱。谁最有钱？银发族。

日本的银发族有钱到什么程度？根据大前研一提供的数据，日本目前有2000万亿日元的个人金融资产，其中六成在银发族手里。2000万亿日元什么概念？世界上最大的原油出口国沙特阿拉伯，一年的原油出口额大概是20万亿日元，等于日本人的账户里沉睡的金融资产，相当于沙特阿拉伯100年的原油出口额。

而且在日本的社会文化里，是鼓励人们存钱的。据说平均每个日本人去世时，会留下3000万日元以上的现金，而且大都没有负债。人没了，钱没花了，这是日本的普遍现象。假如把日本的银发经济激活，意味着有1200万亿日元的金融资产流入市场，这对社会经济是个巨大的刺激。

那么，怎么让日本的老人花钱呢？对于这个问题，大前研一有这么几个思路还是挺新奇的。

- 第一，社会文化必须要变。没错，先不要想着赚老年人的钱，提供什么商品或者服务，第一步要先改变社会文化，改变他们的认知。比如，不要动不动就把老年人挂在嘴边，而是要称呼他们银发族。这两个称呼给人的印象不一样。老年人这三个字就让人觉得老态龙钟，应该在家待着养老，而银发族给人的感觉就不一样，隐含着一份活力。再比如，社会应该鼓励消费的文化。比如美国人在退休之后可能会在北纬37度以南的温暖地带买别墅。瑞典是高税收国家，很少有人存钱，人生的最后阶段由国家负责。而意大利人的文化是，假如死后还有剩余的财富，这就是耻辱，去世前必须把钱花光。总之，大前研一认为，要想改变行为，得先改变想法。要想改变想法，得先改变社会文化。

- 第二，大前研一说，每个人都应该在退休之前培养一些爱好，退休之后你才可以为这些爱好消费。这个说法乍一听很平常，但关键是大前研一很会说服人。比如，有这么个故事，说的是一个男的退休之后每天就宅在家里，妻子要给他做一日三餐。后来妻子忍无可忍，要求丈夫一周必须出去吃三次饭。结果这个丈夫出门后，愣是不知道该不知道去哪，找不到满意的餐厅，也没有别的爱好，于是只好坐地铁环线打发时间。

大前研一还给这样的老年男性起了个很扎心的称呼，叫，湿了的落叶。意思是，像湿了的落叶一样粘在家庭上，无法作贡献，同时又很难被清扫。要想摆脱这个状态，就得趁早培养爱好，退休后多多消费。

借用他的原话，银发经济不仅是个社会课题，它的本质是，我们每个人在老了以后，到底想活出怎样的人生。
