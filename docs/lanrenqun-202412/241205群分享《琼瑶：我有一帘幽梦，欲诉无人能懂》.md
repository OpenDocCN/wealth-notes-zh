

# 琼瑶：我有一帘幽梦，欲诉无人能懂

今天我们从...根据台湾媒体《ETtoday 新闻云》的报道，2024年12月4日，著名作家琼瑶被发现在台湾淡水的家中轻生，享年86岁。

琼瑶本名陈喆...寓意着琼瑶对爱情的美好向往与歌颂。

以前纪念一个人...但关于琼瑶，你可能很难想到更贴切的描述。

琼瑶对这代读者观众的影响有多大？1998年的《还珠格格》平均收视率45%。什么概念？这在当年是绝对的第一梯队。同年能超过它的只有央视春晚。

20世纪80年代初期，花城出版社取得了《琼瑶全集》的独家授权，琼瑶的作品开始以正版的方式面对内地读者。这套书据说至少卖出去10万套，加上盗版，印刷总量超过800万本。1985年，琼瑶与丈夫平鑫涛一起成立怡人传播公司，并且推出了第一部琼瑶电视剧《几度夕阳红》。从此琼瑶的影响力又上了一个台阶。

假如看口碑，琼瑶的作品大概算是黑粉体质的鼻祖，她的很多电视剧都是一推出就爆火，但爆火的同时口碑又有很大的分化，有人一路黑，有人一路粉，有人一边吐槽，又一边看得津津有味。但总归，不管你喜不喜欢，琼瑶都是当年的绝对顶流。

同时，琼瑶也是捧红明星最多的当代作家。琼瑶剧走出的明星包括，林青霞、刘雪华、秦汉、陈德容、林心如等等。还有人说，是琼瑶推动了言情剧的工业化。从创作、选角，再到后来的拍摄手法，是在琼瑶剧中摸索出了一套标准化的流程。

换句话说，琼瑶一度是中文世界里，影响面最广，读者最多，商业价值最大的言情作家。

接下来，我们就一起，再次读读琼瑶。

有人说，琼瑶最有代表性的故事，其实不是最早的《窗外》，也不是2019年的最后一部长篇小说《梅花英雄传》，她最有代表性的故事，其实是她自己这一辈子的经历。回顾琼瑶本人的经历和她的作品，你会发现，二者有着非常微妙的相似。

心理学家艾瑞克·伯恩曾经提出一个概念，叫人生脚本。什么意思？伯恩说，所谓人生脚本，就是你的潜意识为你的一生制定的计划。也就是，你冥冥之中已经决定，自己这一辈子要用什么样的方式度过。只不过这个决定发生在你的潜意识里，你可能自己都没有意识到。

作家王安忆曾经评价，在琼瑶的作品中，人物不必为衣食所忧，一心一意地纠葛于情感之中。而琼瑶本人的经历与这个评价最大的不同在于，她既有来自情感的千丝万缕的纠葛，又要面对来自现实的延绵不断的挑战。

提起琼瑶的一生，有两条非常明显的线索：三次死亡，两段婚姻。

咱们先从三次死亡说起。

琼瑶1938年出生在四川成都。1942年，四川作为抗战大后方，在很多人看来这是最安全的地方，于是大量的外省人进入四川，导致当地物产供不应求，物价飞涨。于是在这一年，琼瑶一家决定离开四川，前往祖籍所在地湖南。这一年琼瑶4岁，之后就一直在乱世中颠沛流离。6岁那年，琼瑶的两个弟弟在一次逃难中走失，家里的行李也丢了。据说穷途末路时，琼瑶的妈妈问她说，你要不要跟爸爸妈妈一块死？6岁的小孩对生死哪有概念？琼瑶就跟着爸妈一起投河。但毕竟再小的孩子也有本能反应，据说当水快要淹过头的时候，琼瑶突然大哭。这一哭惊动了爸妈，她的妈妈大喊，不死了，不死了。

这是琼瑶第一次接近死亡。

第二次，是上高中时，原因是成绩问题。据说琼瑶有一次考试只得了20分，母亲很失望。要知道，琼瑶还有个妹妹，属于考了98分都得大哭一场那种。这么一对比，琼瑶自己就更自卑了。而这个20分就像压倒骆驼的最后一根稻草。当天琼瑶找到母亲的一瓶安眠药，全部吞下。后来经过抢救，才捡回一条命。这是琼瑶第二次经历死亡。

第三次，是高考时。当时琼瑶迎来了人生中第一段重要的恋情，男方叫蒋仁，是琼瑶的国文老师，比琼瑶大整整25岁。那一年蒋仁刚刚离婚，两个人就走到了一起。当时蒋仁与琼瑶许下了四年之约，琼瑶考上大学，四年毕业后，假如到时琼瑶没有变心，两个人就结婚。后来琼瑶高考落榜，母亲知道她与蒋仁的恋情非常愤怒，一纸诉状把蒋仁告到了警局，理由是引诱未成年少女。那一年琼瑶再次吞安眠药，后来又被救了过来。这也是琼瑶第三次经历死亡。

而琼瑶的两段婚姻，第一段就发生在高考之后不久。琼瑶20岁那年，认识了一个台大外文系的高材生，庆筠。后来琼瑶回忆说，他虽然这么穷，却穷得满不在乎。他对物质的需求已接近于零，只是满脑子想写作。他这种傻劲，和他这份穷苦，都让我心中恻然。

琼瑶最终与庆筠结婚，其间日子过得很清苦，主要靠写作赚点生活费。但没成想，本来是科班出身的庆筠，迟迟没有写出名堂，反倒是把写作当成兴趣的琼瑶越写越好。这个反差也成了两个人频繁吵架的导火索之一。最终，两个人在1964年离婚。

到此，琼瑶的第一段婚姻终结。

但在写作期间，琼瑶结识了当时《皇冠》杂志的老板平鑫涛。平鑫涛认定琼瑶是难得一见的作家苗子，对她非常欣赏。一来二去的，两人就产生了感情。当时平鑫涛已经结婚，是有妇之夫。平鑫涛就跟琼瑶说，等他，等他的三个孩子都到了15岁，他就离婚跟琼瑶在一起。

终于，1979年5月，琼瑶与平鑫涛结婚。这场婚姻在当时还引起了不小的舆论风波。关于琼瑶本人的争议，以及她作品中爱情观的争议，也是在这一年到达了顶峰。

2017年，平鑫涛中风，留下了严重的后遗症。据说琼瑶曾经想过和丈夫一起安乐死，但被丈夫拒绝。同样在2017年，琼瑶出版《雪花飘落之前：我生命中最后的一课》，里面谈论了她对生离死别的看法。其间，当时79岁的琼瑶交代了自己的身后事，强调了两件事：绝不插管，绝不抢救。其中特别强调，不要在我的身上插尿管、呼吸管、鼻胃管，总之什么管子都绝对不行。琼瑶还引用了自己1989年出版的《追求活的尊严》，在自序中她说，当你最爱的人，生命将尽时，不是用各种管线，强留他的躯体，让他为你那自私的不舍，拖着逐渐变形的躯壳，躺在床上苟延残喘。我的同事，作家贾行家老师说，你可以把这些都看成是琼瑶在和生命的惯性抗衡。当然，这是琼瑶的选择，我们只是转述，不做评判。

2019年，平鑫涛去世，据说当时因为抢救插管的问题，平鑫涛的子女还与琼瑶发生过不小的争执，一度成为当时的热搜。

而之后，琼瑶再次出现在人们的视野里，就是昨天，她去世的消息。

同时，也是在这天，琼瑶的遗书被公布。在遗书的最后，琼瑶是这么说的。

年轻的你们，千万不要轻易放弃生命，一时的挫折打击，可能是美好生命中的磨炼。希望你们经得起磨炼，像我一样，活到八十六七岁，体力不支时，再来选择如何面对死亡。

而在遗书的开头，有几句话是这么写的。

不要哭，不要伤心，不要为我难过。我已经翩然的去了。翩然是我最喜欢的两个字，代表的是自主、自在、自由的飞翔，优美而轻盈，我摆脱了逐渐让我痛苦的躯壳，翩然的化为雪花飞去了。

这算是琼瑶留下的最后一句话，但其实，老人家很早之前就写下，但愿我生时有如火花，死时有如雪花。

这像不像是一场微妙的自洽？琼瑶用她的方式写了自己的作品，也写了自己的人生脚本。至于为什么这样写，外人也许无法了解。就像琼瑶在《一帘幽梦》里写的：我有一帘幽梦，不知与谁能共，多少秘密在其中，欲诉无人能懂。
