

# 新一轮汽车价格战到来，如何看待？

241202 文/卢克文工作室嘉宾 文如刀歌
整理：公众号懒人搜索，懒人专属群独享
懒人微信：lazyhelper

![](img/550c542a54857088648f6ddc15d75b1e_0_0.png)

过去几天，“比亚迪发函供应商，要求2025年降价10%”的贴文传遍全网，引发了很大的争议。

以往，新能源汽车打价格战，舆论默认是利好消费者的，大家乐见其成。可这一次，却引来了许多不满。不过，其实这不是事情的全貌。

就在比亚迪要求供应商降价10%的几天前，特斯拉刚刚官宣Model Y后轮驱动版及长续航全轮驱动版尾款减1万元，售价23.99万元起。如果再算上以旧换新补贴的2万元，特斯拉的这次降价力度非常大。

或许是习惯成自然，过去几年特斯拉引导了多次国内新能源车的降价潮，所以这次降价，舆论场居然一点水花都没有。

可在特斯拉官宣降价不久，就有车企在内部公开信中，将未来两年定义为“生死存亡期”。如果新势力的头部都在生死边缘，那排在后面的车企还能活几家？

简单来说，近期新能源车市场的动态是特斯拉先官宣降价，然后有“生死边缘”的判断，后续才有了比亚迪给供应商群发信息，要求降价10%。

很显然，比亚迪将供应商价格都打下去，是为了明年的降价空间。比亚迪的市场份额本来就是最大的，再降价，必然会逼迫其他车企不得不跟进，由此导致新一轮的价格战螺旋。

看到这里，许多朋友或许认识到了问题的症结，那不就是特斯拉的“鲶鱼效应”过了头导致的么？是不是回归过去一群车企围攻特斯拉，“禁止特斯拉降价”，就能把新能源车行业的内卷问题给解决呢？

这个问题我们还要深挖一层。

特斯拉并不是活雷锋，尽管很多网友喜欢特斯拉和马斯克，但作为商人和企业，赚取利润一定是第一需求。

能高价还保持竞争力，特斯拉是绝不会自我放血，持续降价的。

在特斯拉本轮降价之前的一个月，国内新能源车的圈子里，刚刚爆发了“六大派围攻特斯拉事件”。

今年九月，六家大型车企分别发售乐道L60、极氪7X、智己LS6、阿维塔07、智界R7、岚图知音六款新车。引发关注的是，这六款车从发布会到4S店宣传，产品性能与优势，对标的都是特斯拉的Model Y。

也就是说，国内大量新能源车企，一窝蜂地造出对标Model Y的车，然后用更低价格来主推性价比。

刚刚过去的十月，六款对标特斯拉的新车累计交付3.34万辆，少于特斯拉公布的Model Y在十月3.62万辆的销量。从总销量数据来看，国内车企似乎围剿失败，但是六大车型已经和特斯拉最畅销车型销量打平。

这个趋势已经足够威胁到特斯拉的后期订单。

六大车型之所以暂时还没有追平，一个很重要的原因，是国内这六款车型处在产能爬坡阶段，比如，蔚来推出的新车乐道L60目前的交付周期，最低也要一个半月左右。

如果各家车企产能爬坡完成，将交付周期缩减，对特斯拉的冲击就会更大。

所以事情的全貌，是先有多家国内新能源车对标围堵特斯拉，激烈竞争迫使特斯拉降价，特斯拉的降价，又逼迫其他车企如比亚迪、上汽等通过压缩供应商价格，来跟进明年的降价。

而比亚迪的降价，又回旋镖一样地传导到围攻特斯拉的这些车企头上，引发新一轮内卷，整个新能源车产业的“向下螺旋”就形成了闭环。

看清楚了么？这种新能源车产业的向下螺旋，并不是某个企业能够停止的，目前全网舆论似乎在逼迫比亚迪停止威逼供应商降价。可是，这一轮价格战的起点其实是特斯拉，特斯拉又推给其他围攻对标的车企。

所以才说，要改变新能源车的向下内卷螺旋，不是监管部门对某个企业发指令就能解决，而是整个行业发展模式到了不得不变的阶段。

那么，过去几年，新能源车的历次降价，似乎都能引起购车消费者的欢呼，为什么这一次降价会挨骂呢？

- 第一，是民众认识到过度内卷的刀刃，最终会砍在全民收入上。表面看来，新一轮降价，好像压缩的只是供应链上一些企业的10%利润，不用上纲上线到全民收入。但实际上，供应链的特性就是层层传导。整个供应链都降价10%，这些老板都是活雷锋，这10%都自己承担么？当然不是，他们最终会通过延长工人工作时间，增大绩效考核难度等手段，分摊给这条供应链的所有打工人。当前我国新能源车全产业链囊括了几千万工作人群，并且许多还是买房买车的消费主力，再降价，这些打工人的收入和消费肯定要下降，进而导致更大范围的恶性循环。同时，许多人共情供应商，是因为我国在产业升级，制造业扩产能的过程中，几乎所有行业的供应链下游企业，都在经历类似的内卷和盘剥。在整体全工业门类内卷之下，所有打工人，都面临更多工作强度，收入却降低的预期。借这次批判逼迫供应商降价，是许多网友希望改变那种单纯靠卷价格的不合理现状。

- 第二，对比胖东来模式，全民对“高端产业分配更多利润”失去信心。近期和比亚迪强迫供应商降价一同火爆的，是胖东来创始人于东来陷入“干涉员工彩礼”的风波。实际上，于东来在提出禁止彩礼的同时，也发表了对“各行业无限内卷的批判”，这部分内容都被各大媒体在彩礼的喧嚣中刻意隐藏了。实际上，最近于东来的表态，胖东来的模式以及所有员工的福利待遇和休假，让许多网友对过去十几年奉行的竞争路线产生了动摇。

胖东来是低端普通消费行业，没有门槛的商业超市行业；比亚迪、宁德时代、蔚来、特斯拉都是全球顶尖，是我们用几十年想要攀登的高端产业。

即使在商业超市领域，比起沃尔玛，甚至各省的省一级连锁超市，胖东来的规模都不大，排名不算靠前，更别说行业龙头。阿里、比亚迪、宁德时代等等，都是各行业的龙头企业，赚取高额利润。

可是对标胖东来与这些企业的员工工资与福利待遇，会发现天差地别。

过去我们信奉的竞争逻辑，是拿下高端产业链，成为行业第一，就能分配更多利益给老百姓。

可是胖东来火了之后，网友们看到的是传统行业不一定要做到龙头，只要愿意给员工分配一定的利益，不用内卷，员工也能活得比现在更好。

相反，不少企业已经做到全行业第一，也是世界未来发展方向这样的高端产业，是通过不断内卷拿到的全球话语权，但他们做的还是不断压缩供应链的利益和成本，最终导致全产业链的打工人，都没有高额回报。

这种对比才是最恐怖的，让网友觉得即便继续卷高端产业，继续去争全球第一，也是老板赚的多，打工人无限内卷，连收入增加、假期增多的诉求都不可能实现。

所以这次新一轮降价事件，已经不止是一个汽车新闻，而是让网友上升到分配路线和发展模式的讨论上。

要解决这个问题，或许真的该认真研究胖东来模式了。
