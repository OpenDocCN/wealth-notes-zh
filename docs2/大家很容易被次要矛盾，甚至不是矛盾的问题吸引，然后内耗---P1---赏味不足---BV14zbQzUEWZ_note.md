# 课程名称：识别核心矛盾，避免无效内耗 - 第1课

## 概述
在本节课中，我们将要学习如何识别生活中的核心矛盾与次要矛盾。很多人常常将时间和精力耗费在次要甚至无关紧要的问题上，导致自我内耗和效率低下。我们将通过具体案例，分析这种现象的成因，并学习如何将注意力聚焦在真正关键的问题上。

## 正文

很多人容易被次要矛盾，甚至根本不是矛盾的问题所吸引，然后开始内耗。

然而，很多人在浪费时间进行内耗时，自己并未察觉。这是一个潜移默化的问题。

首先，通知一下下周的活动安排。活动地点定在宁波。

同时，新增了两个讨论主题：一是产业园招商引资与孵化器的运作方法；二是产品或服务的定价策略。如需报名或了解详情，请私信联系。

一直以来，我想说的一句话是：很多时候，人们对于真正属于自己的问题反而不去思考，而对于不属于自己的问题却拼命思考，导致内耗。

这种现象我已见过多次。例如，今天之所以讨论这个主题，是因为早上在网上看到一个关于找工作的讨论。内容提到，现在很多HR会直接筛选掉简历中有职业空窗期的求职者。

我本以为接下来会开始讨论核心矛盾，结果整篇文章都在长篇大论地讨论职业空窗期本身。职业空窗期是问题吗？可以说是。但这是核心矛盾吗？当然不是。

当前求职市场的核心矛盾有两个：一是求职者数量远多于岗位数量；二是整体经济形势不如从前。这是两个最根本的问题。当一个岗位每天收到数百甚至上千份简历时，职业空窗期的问题早已被淹没。

你会发现，评论区和其他人都清一色地被带入了讨论职业空窗期的逻辑中。很多人开始反思自己有空窗期该怎么办，并因此焦虑，思考解决方案。

我们思考问题必须结合实际情况。职业空窗期并非核心矛盾。而整个市场人太多、岗位太少、经济下行，也不是我们个人能解决的。

如果你真的陷入了如何解决或抹平职业空窗期这个问题中，那就是在内耗。这相当于放着核心矛盾不解决，却去解决次要矛盾，是在浪费时间。

**核心矛盾公式：**
`核心矛盾 = 市场岗位供需失衡 + 宏观经济下行`
`次要矛盾 = 个人简历中的职业空窗期`

---

上一节我们分析了求职案例中混淆主次矛盾的问题，本节中我们来看看社交与合作中的类似情况。

其次，有时人们面对本属于自己的问题时，却非要寻找外部原因。

例如，至今不止一个人，甚至不止十个人对我说：陈老师，我社交了很久，认识很多人，但都无法促成合作。或者我感觉我认识的人不行，是不是我的圈层有问题？

然后你会发现这些人开始内耗，自我纠结。他们要么重复外出社交，要么不停地得出结论：是不是我的圈子不行、这些人不行、我找的方向不对。

我们一直强调，任何一个人或一种资源，如果你觉得它是垃圾，那是因为你不会用。任何人和资源都是可以利用的，区别仅在于你会不会用，以及这个资源是否出现在了正确的地方和正确的时间。也就是说，需要天时、地利、人和。

我一直认为，在社会上做事，无论你扮演什么角色，都必须有自己的“业务”。无论这个业务是真是假，是虚是实，这不重要，但你必须要有。因为只有这样，别人才知道你要做什么，才有可能与你进行更深层次的交流。

也就是说，当你与别人交流时，你不能只是闲聊。聊了半天，互相问“你是做什么业务的？”，然后就没有下文了。这样对方根本不知道你要干什么。

就像两个人相亲，也是一件事。但如果两个人认识了，互相介绍“我是厨师”，“我是老师”或“我是收废品的”，互相表达了职业后，请问这件事有什么意义呢？

你必须有一件具体的事情作为支点和撬动点，去社会上认识人才有意义。否则，交流就变成了“How are you? Fine, thank you. And you?” 然后就没有然后了。大家来回问好，没有意义。

**有效社交代码模型：**
```python
def 有效社交(个人, 对方):
    个人业务 = 个人.获取核心业务()
    对方业务 = 对方.获取核心业务()
    
    if 个人业务 is not None and 对方业务 is not None:
        # 存在业务支点，可探讨合作可能性
        合作可能性 = 评估合作潜力(个人业务, 对方业务)
        return 合作可能性
    else:
        # 缺乏业务支点，交流停留在表面
        return "无效社交：How are you? Fine, thank you. And you?"
```

---

上一节我们探讨了社交中缺乏核心支点的问题，本节中我们来看看在选择方向时如何抓住不变的核心。

第三，再比如，很多人跟我讨论方向选择。

例如AI、区块链、AR/VR、数据要素，包括以前的大数据等等。他们问我：陈老师，方向怎么选？我们应该如何看待这些方向？

我早就说过，方向、专业、岗位这些东西，都是随着社会和时间发展而变化的。你们自己回顾一下从2010年到现在，中国和全球每年的热点，都在变化，每两年都在变，而且这种变化不受我们个人控制。

把时间线拉长到十年甚至几十年，请问像我们这样的普通老百姓，一味地追逐这些变化的方向努力，有意义吗？至少对我而言，如果只是不停地跟着变，我会很累，而且我也没有能力一直跟上。即使跟上了，也可能赚不到钱。等我年纪大了，跟不上了怎么办？就成了炮灰，成了别人的垫脚石。

所以，你们问的关于这些方向的问题，很多都是次要矛盾。主要矛盾是什么？是所有机构、企业、协会、地方政府，究竟是如何利用这些方向去赚钱和盈利的。这才是主要矛盾，这才是不变的。

所以，这才是我要去探索、去社交、去做项目的核心矛盾和核心目的。到现在还有很多人在纠结“我要弄个学历”、“我要选个热门方向”，这他妈是核心矛盾吗？

当然，每次都有普罗大众跳出来说：陈老师，如果都去想着赚钱，事情没人做了呀。好的，这不又来了吗？请问事情有没有人做，是你我需要关心的吗？

每次这样说，就有人会说：陈老师，你格局太低了。不好意思，你要我格局高，可以的，你先给我……给我个什么？给我个能让我格局高的位置。我现在就是个普通人民群众，我格局能高到哪去？

**核心矛盾判断逻辑：**
```
IF 问题关于“追逐热门方向/学历”
THEN 这是次要矛盾
ELSE IF 问题关于“如何利用趋势盈利/创造价值”
THEN 这是核心矛盾
END IF
```

---

前面我们分析了几个具体场景，接下来要触及一些更根本的、可能令人不适的真相。

第四，很多时候都是如此。

接下来要说的可能有些扎心，请大家注意。

你赚不到钱的核心矛盾，在于你胆子不够大、见识不够广、想象力不够丰富、组织能力不够强、社交能力不够弱等等。当然也可能有很多别的原因。

但是，你的核心矛盾绝对不可能是你的学历和专业。绝对不可能。

你找不到对象，可能因为你没钱没房、颜值不够高、不是公务员，或者你的各项条件不符合对方或对方家庭的期望。可能也有各种别的原因。

但是，核心矛盾绝对不可能是“我觉得你是个好人，但是我们不太合适”。能明白吗？

同样的，当你去找一个层次高于你的人或机构合作时，最终合作不了，总是一场空。可能因为你没有强大的背书，可能因为你手上没有垄断性的专利或竞争力，也可能因为你没有对应的成功案例等等。

但是，核心矛盾绝对不可能是“我们觉得你很优秀，但是你现在这个业务方向，跟我们业务方向可能不是那么匹配”。能听懂吗？

虽然大家都懂，战略的重要性远大于战术，战略可以碾压战术。这个道理大家都懂。但问题是，你必须先有判断力去分辨，哪个是核心战略，哪个是无足轻重的细节。否则，你就是在浪费时间。

人生短短几十年，你整个过程中有精力去折腾、思维逻辑能跟上时代的，可能也就10年到20年。你哪有那么多时间可以浪费？

每次你会发现，很多人今天被小红书上的一个问题牵着走，明天被抖音上的另一个问题吸引，后天又关注B站上的新话题。每天都被牵着鼻子走，讨论的永远都不是核心矛盾，永远都在绕圈圈。

然后最后问：哎，那我怎么成功不了？我怎么得不到我想要的东西？你不想想你时间都花在什么地方了吗？

## 总结
本节课中，我们一起学习了如何识别并聚焦于核心矛盾，避免在次要矛盾甚至伪矛盾上浪费时间和精力。我们通过求职、社交、方向选择和自我归因等多个案例，分析了混淆主次矛盾的普遍现象及其危害。关键在于培养判断力，分清什么是个人无法改变的宏观趋势（如经济形势），什么是个人可以发力的核心支点（如自身业务和能力）。只有将有限的精力投入到真正关键的战略问题上，才能避免无效内耗，更有效地向目标前进。
