# 基本盘篇：没有赚钱的欲望，但有花钱的欲望 - P1 💰

在本节课中，我们将要学习一个关于“基本盘”的核心逻辑：许多人缺乏强烈的赚钱欲望，却拥有旺盛的消费欲望。我们将深入探讨这一现象的表现、成因及其背后的影响。

## 概述

本节内容将围绕“赚钱欲望缺失”与“消费欲望过剩”这一矛盾展开。我们将通过具体事例和分析，帮助你理解这一普遍存在的心理状态。

## 没有赚钱的欲望

上一节我们介绍了课程主题，本节中我们来看看“没有赚钱欲望”的具体表现。

说真的，很多人并没有真正的赚钱欲望。当被问及是否想赚钱时，大家都会回答“想”。但深入沟通后，我发现这种欲望几乎看不到。很多人一点也不着急，也没有真正的渴望。

而且，许多人的思维非常僵化。例如，我曾在微博分享早年只顾赚钱、在2014年付了首付的经历。当时赚钱仅仅是为了赚钱，并未具体想买什么。有人评论问“毕业几年买的房？”，我回答“3年”。对方表示惊讶，我进一步说明：我毕业月薪仅500元，还兼职做湾仔码头水饺试吃员，每月约200元，在双休日工作8天。我反问：以正常人的能力，再高的薪水，3年能付首付吗？对方回应：“哦，那你没打过工。”

这引出了关键点：**什么是赚钱的欲望？** 欲望意味着“我全都要”。如果选择躺平，那不叫欲望。真正的欲望是：打工的钱我要，自己找门路赚的钱我也要，只要时间允许，任何合法赚钱的方式我都要。然而，我接触的许多人并没有这种欲望。

基本盘给我的感觉是，很多人只是在“做样子”。这个“样子”是做给父母、老师、朋友、自己甚至整个世界看的，用以催眠所有人，让自己和别人都感觉“很想赚钱”、“很努力”，但实际行动上却停滞不前，徒有其表。

## 没有欲望与内心焦虑

没有赚钱欲望本身不是问题。这不是对错之分。问题在于，如果你没有赚钱的欲望，却同时感到焦虑、羡慕和攀比，这就会产生矛盾。

你既没有赚钱的动力，又每天陷入内耗和焦虑。何苦呢？有内耗的时间，不如去赚点钱。

我个人赚钱经历分为两个阶段：
1.  **年轻时纯粹追求金钱数字**：例如早期做政企咨询，从半天1000元，到一天1万、1.5万，最高时一天3万。追求数字增长带来类似游戏的成就感和爽感。
2.  **现在更关注探索与认知提升**：我仍然赚钱，也追求突破，但更关心合作项目中是否有我不了解的**商业流程**、**关系链**等新事物。我的欲望优先级发生了变化，探索未知变得更重要，金钱则成为第二优先级。但这两种状态都属于“欲望”的范畴。

## 欲望缺失的根源：长期PUA

我大胆猜测，大众缺乏欲望并非天生，而是过去二三十年长期遭受潜移默化的PUA结果。

这种PUA来自家庭、社会、舆论、网络、父母，甚至自己。它传递了一个强大的心理暗示：**你不行**。我问过很多人为什么不行，他们自己也不知道，只是觉得“就是不行”。

这就像某些城市对女孩的PUA：你必须本地读书、结婚、生子，出去就会被骗。但从逻辑上讲，人都没出去过，怎么就知道不行、会被骗呢？

我感觉许多人没有欲望，仿佛身处一个大型的“楚门世界”试验场，从小被灌输“你不行”的观念。

## 容易被激发的花钱欲望

花钱的欲望人人都有，但这里我想指出的是一种特定现象：赚钱的欲望很难被激发，但**花钱的欲望却太容易被激发**，也就是常说的冲动消费。

理论上，消费能力应根据个人整体财政和收入按比例决定。但现实中，很多人的消费是**过度消费**，本质就是冲动消费。

**冲动消费的核心公式**可以概括为：
```
花费 X 金额 → 获得 Y
```
但问题是，这个 **Y（获得物）到底是什么**，以及它**是否值 X（花费的金额）**，消费者在购买时往往根本不清楚。

例如，看直播电商时，脑子一热就各种买。买完后，这东西真的值这个价吗？你不知道。

## 总结与反思

本节课中我们一起学习了“缺乏赚钱欲望”与“旺盛消费欲望”并存的现象。

有的人会说，你讲的这两点都是人性的弱点。没错，但我看到的是，这些人性弱点被某些力量完全拿捏并无限放大，进而让你们养成潜意识的习惯：一方面觉得自己不行，另一方面又不停消费。

结果如何？这里不便明说，只能“懂得都懂”。
