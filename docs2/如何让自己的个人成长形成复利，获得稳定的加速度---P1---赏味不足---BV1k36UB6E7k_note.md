# 课程：如何让个人成长形成复利并获得稳定加速度 - P1 🚀

在本节课中，我们将探讨一个核心问题：如何让个人成长像复利一样累积，并实现稳定的加速发展。我们将分析成长复利与稳定加速度的本质，并提供一套清晰的行动框架。

## 概述：理解问题的核心

前两天的咨询中，一位伙伴提出了一个很好的问题。他的主题是：如何让自己的个人成长形成复利，并获得稳定的加速度。我们先来分析他的诉求。

他的问题包含两点。第一，个人成长从现在到未来，每一次成长之间应该有关联。就像上台阶，下一个台阶踩在当前台阶之上，再下一个又踩在上一个之上。必须有一个向上累积的趋势，才能形成复利。这与乔布斯早年在麻省理工演讲的观点一致。他说，人生中十年里做的事都是一个个点，我们需要把这些点连成线，才能产生更大的经济效益或助力。这是我理解的“复利”。

第二点是关于“稳定加速度”。他的意思应该是，随着时间的推移和自身的努力，能有一个相对稳定、持续上升的趋势。

## 反驳：成长不存在“稳定加速度”

当时听完他的问题，我先反驳了一个观点。在我的认知范围内，我不认为人类的成长存在“稳定的加速度”。

简单来说，就像我们常说的，每个人头顶都有无数层“象牙塔”。在我看来，只有当你突破一层象牙塔时，才算有真正的成长。例如，你突破了第一层——学历和社会对“打工”的束缚。我认为这是一种成长。之后，你开始了解资本、政治、高校如何运作，并参与一些浅层合作，这又是突破了一层象牙塔。

每次突破一层象牙塔，你就有成长。但如果不突破，无论你在同一层里如何努力，在我看来都只是同一级别的挣扎，是一种“过家家”行为。这不存在真正的成长。

因此，从我的角度看，只要不具备突破象牙塔的能力或现实，就不存在所谓的“稳定上涨”的成长。人类的成长往往依赖天时、地利、人和：时代的助力、顺势而为、个人契机以及各种机会。这些因素共同作用，可能让你实现一次大的突破。但只要你没有突破，你的生命线或成长线就是平的。至少我是这么认为的。

所以，成长并不存在一条所谓的、平滑上升的曲线。这是第一点。

## 核心：如何让成长形成复利

现在，我们回到核心问题：如何让成长形成复利。我们用白话文来解释一下。

乔布斯说要把点连成线。那么，我们怎样才能把点连成线呢？首先，这些点之间必须有连续性或相关性。我打个比方：假设你今天学的技能是钳工，明天去学炒菜，后天学短跑，大后天学国际象棋。本质上，这些技能点一开始相关性不大。当然，有些聪明的伙伴总能找到关联，但从我的角度看，它们确实相关性不大。

因此，你首先需要找到那些具有相关性的点。很多人会迷茫：我该怎么找呢？其实，个人成长要形成复利，最关键的是什么？想想看，我们做事情、做项目、打工，目的都是为了了解更多东西，形成更大的信息差，从而获取更多利益。

那么，我们真正要掌握和成长的是什么？其实是围绕**商业模式**和你**个人擅长的点**来成长。

## 行动框架：从商业模式出发

我们举个例子。假设你的商业模式是：通过向资本方介绍好项目，来赚取FA（财务顾问）费用或中介费。类似的模式还有：给外包公司或猎头介绍年薪数百万的高端打工人，从而赚取10%到20%的中介费。

你看，这是你的商业模式，你已经很清楚了。那么，围绕这个商业模式，你需要涉及哪些点呢？

以下是这个商业模式涉及的关键点：

1.  你需要认识优秀的猎头。
2.  你需要有能力认识顶尖的打工人。
3.  你需要有能力促成双方合作。
4.  你需要维护好双方关系。
5.  你需要有能力防止双方跳过你直接接触。
6.  在这个商业模型的衍生中，你还可以做很多事情，比如为候选人建立收费社群、做知识变现，或者涉足资本运作、企业服务等领域。

就像我们刚列出的这些点，它们都是在你确定商业模式后，未来成长过程中需要锻炼的点。每个点就像一个技能树，不可能一开始就是满级。例如，寻找候选人，你不可能一开始就找到年薪两三百万的，肯定是从二三十万、五六十万开始，逐步深入。

每个点都需要你循序渐进地学习和沉淀，不可能一步到位。

你会发现，只有非常清楚你的商业模式是什么，你才能分解出未来人生中应该关注哪些点，并知道如何深入挖掘这些点。这是在不可预见的未来道路上，让你知道如何沉淀，并让这些沉淀能够串联起来的**唯一可能性**。

剩下的可能性都像是在赌博——赌这些东西之间有关系，这完全是扯淡。

所以，关键又回到了原点：你必须先确定你的商业模式。

## 如何找到你的商业模式

这个商业模式不是凭空想出来的。它存在于地球上、这个国家、乃至全球，可能已经存在了二三十年、四五十年。你需要做的，是去了解那些已经存在的商业模式到底有哪些。

然后，你需要一个个去过滤、筛选，看看哪个最适合你手上的牌、你所在的行业和领域，以及你身上已有的技能点。你只能这么做。

## 总结：成长的路径与真相

当你明白并找到一个适合自己的商业模式后，你就会知道还有哪些点需要磨练。在磨练的过程中，你会认识很多人，了解更多事。而在这个过程中，时代给你的助力，以及你遇到的人能否带来更多利益，就是我们所说的“天时地利人和”。如果有，你就能实现一次较大的成长。这就是他所问问题的答案，至少在我看来是如此。

但从他问出这个问题来看，他和大多数人一样存在误区。第一，大家都希望成长形成复利，但根本不知道如何形成，靠的都是走一步看一步。这没有用，我告诉你们，这就是赌博。第二，大家都想获得稳定的加速度，但人的成长绝对不存在“稳定加速度”这个逻辑。

如果你想要稳定加速度，只有一种可能性：**躺平**。

躺平。打工本身都不算稳定加速度，打工本身算稳定减速度……哦不对，不能这么说，算是稳定的附加速度吧。

我不知道这样解释你们能不能明白。

但事实上，只有这一条路。至少在我的认知能力范围内，就这一条路，没有别的可能性。
